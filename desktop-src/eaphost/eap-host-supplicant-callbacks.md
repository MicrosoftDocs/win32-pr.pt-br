# <a name="eaphost-supplicant-callbacks"></a>Retornos de chamada do suplicante EAPHost

As funções de retorno de chamada da API suplicante do EAPHost são as seguintes.



| Callback                                           | Descrição                                                                                                                                                                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NotificationHandler**](/previous-versions/windows/desktop/api) | Um protótipo de retorno de chamada que notifica o suplicante de que há uma alteração na SoH (declaração de integridade) e a reautenticação de uma conexão de sistema NAP ( [proteção de acesso à rede](/windows/desktop/NAP/network-access-protection-start-page) ) é necessária. |
