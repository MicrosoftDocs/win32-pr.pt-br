# <a name="eaphost-peer-method-enumerations"></a>Enumerações do método de par EAPHost

As enumerações de API do método par do EAPHost são as seguintes.



| Enumeração                                                        | Descrição                                                                                                                |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**EapCredentialType**](/windows/desktop/api/eaptypes/ns-eaptypes-eapcredential)                         | Define os tipos de credencial EAP e as credenciais apropriadas.                                                          |
| [**EapPeerMethodResponseAction**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresponseaction) | Define o conjunto de ações que um autenticador EAP pode indicar a um método de ponto de protocolo EAP ou suplicante durante a autenticação.     |
| [**EapPeerMethodResultReason**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresultreason)     | Define o conjunto de resultados de uma sessão de autenticação EAP retornada por um método de autenticador EAP para um método EAP de mesmo nível. |
