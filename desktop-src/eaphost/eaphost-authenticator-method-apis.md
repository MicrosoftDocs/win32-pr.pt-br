# <a name="eaphost-authenticator-method-apis"></a>APIs do método de autenticador EAPHost

A documentação da API do método de autenticador EAPHost consiste nas seções a seguir.

-   [Funções de método de autenticador EAPHost](eap-host-authenticator-method-functions.md)
-   [Enumerações de método de autenticador EAPHost](eap-host-authenticator-method-enumerations.md)
-   [Estruturas de método do autenticador EAPHost](eap-host-authenticator-method-structures.md)

Observe que esses elementos devem ser implementados em uma DLL de conformidade pelo autor do método de autenticador EAP. A DLL com a implementação desses elementos de API é carregada e chamada pelo EAPHost do lado do servidor.
