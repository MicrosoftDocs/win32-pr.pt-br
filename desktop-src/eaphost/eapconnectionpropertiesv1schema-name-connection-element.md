# <a name="name-connection-element"></a>Elemento Name (Connection)

O elemento **Name (Connection)** captura o nome da conexão que está sendo definida, auxiliando na identificação de várias conexões.

``` syntax
<xs:element name="Name"
    type="string"
 />
```

O elemento **Name** é definido pelo elemento [**Connection**](eapconnectionpropertiesv1schema-connection-connections-element.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Vista\]<br/>       |
| Servidor mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Server 2008\]<br/> |
