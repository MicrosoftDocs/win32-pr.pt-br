# <a name="common-eaphost-api-enumerations"></a>Enumerações comuns da API EAPHost

As seguintes enumerações são comuns a todas as tecnologias de API do EAPHost.



| Enumeração                                                                  | Descrição                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**\_tipo de atributo EAP \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type)                           | Define o conjunto de tipos de atributo EAP possíveis disponíveis em uma entidade de autenticação.        |
| [**\_tipo de \_ campo de entrada de configuração EAP \_ \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_config_input_field_type)       | Define um conjunto de possíveis tipos de campo de entrada disponíveis ao consultar as credenciais do usuário. |
| [**\_tipo de \_ Propriedade do método EAP \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Define o conjunto de propriedades de método EAP possíveis.                                            |
| [**\_tipo de \_ valor da Propriedade do método EAP \_ \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Define o conjunto de tipos de dados possíveis para as propriedades do método.                             |
| [**EapCode**](/windows/win32/api/eapmethodtypes/ne-eapmethodtypes-eapcode)                                                   | Define o conjunto de tipos de pacotes EAP.                                                          |
