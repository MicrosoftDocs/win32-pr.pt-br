# <a name="about-eaphost"></a>Sobre o EAPHost

O EAPHost implementa a máquina de estado EAP (Extensible Authentication Protocol) e a estrutura do protocolo EAP, de acordo com a [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063).

O EAPHost dá suporte à NAP ( [proteção de acesso à rede](/windows/desktop/NAP/network-access-protection-start-page) ) baseada em EAP.
