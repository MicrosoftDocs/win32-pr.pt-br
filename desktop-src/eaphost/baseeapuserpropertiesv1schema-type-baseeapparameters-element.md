# <a name="type-baseeapparameters-element-for-user-properties"></a>Elemento Type (BaseEapParameters) para propriedades do usuário

O elemento **Type (BaseEapParameters)** define o elemento placeholder para o tipo de método selecionado e as credenciais específicas do método.

``` syntax
<xs:element name="Type"
    type="integer"
 />
```

O elemento **Type** é definido pelo tipo complexo [**BaseEapParameters**](baseeapuserpropertiesv1schema-baseeapparameters-complextype.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do vista\]<br/>       |
| Servidor mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do servidor 2008\]<br/> |
