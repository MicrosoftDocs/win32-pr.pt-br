# <a name="identity-element"></a>Elemento Identity

O elemento **Identity** captura a identidade usada para autenticação.

``` syntax
<xs:element name="Identity"
    type="string"
 />
```

## <a name="remarks"></a>Comentários

O elemento **Identity** é abstract; cada método deve definir e usar um elemento derivado nos documentos da instância.
