# <a name="credentialssource-eaptype-element"></a>Elemento CredentialName (EapType)

O elemento **CredentialName (EapType)** contém informações sobre o local do certificado EAP-TLS.

``` syntax
<xs:element name="CredentialsSource"
    type="CredentialsSourceParameters"
 />
```

O elemento **CredentialName** é definido pelo elemento [**EapType**](eaptlsconnectionpropertiesv1schema-eaptype-element.md) .

## <a name="remarks"></a>Comentários

O elemento **CredentialName** é opcional.
