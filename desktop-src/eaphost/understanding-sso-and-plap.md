# <a name="sso-and-plap"></a>SSO e PLAP

A API de único Sign-On (SSO) representa um conjunto de métodos usados para obter credenciais específicas do método EAP para uma conta de usuário ou computador de rede de maneira segura sem ter que gerar várias instâncias de interface do usuário.

> [!Note]  
> O termo PLAP (provedor de acesso de pré-logon) está obsoleto. Se "provedor de acesso de pré-logon" ou "PLAP" for encontrado no código ou na documentação, substitua-o por "logon único" ou "SSO", respectivamente.
