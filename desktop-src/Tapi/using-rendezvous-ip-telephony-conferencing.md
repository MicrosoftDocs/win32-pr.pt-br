# <a name="using-rendezvous-ip-telephony-conferencing"></a>Usando a conferência de telefonia IP da reunião

\[ Os controles e as interfaces da conferência de telefonia IP de reunião não estão disponíveis para uso no Windows Vista, no Windows Server 2008 e nas versões subsequentes do sistema operacional. A API do cliente RTC fornece funcionalidade semelhante.\]

A reunião consiste em três conjuntos básicos de interfaces:

-   Os [controles de diretório](directory-controls.md) fornecem um meio de descobrir, criar ou modificar as informações de conferência e de usuário em um servidor ILS ou NTDS.
-   Os [controles de blob de conferência](conference-blob-controls.md) permitem modificações específicas do provedor em uma conferência.
-   As [interfaces com multicast](multicast-com-interfaces.md) fornecem um meio de reservar endereços para uso por conferências multicast.
