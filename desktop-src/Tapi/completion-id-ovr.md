# <a name="completion-id"></a>ID de conclusão

O identificador de conclusão para a chamada de entrada é o resultado de uma solicitação de conclusão que é encerrada.

**TAPI 2. x:** Consulte [**lineGetCallInfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) (membro **DwCompletionID** da estrutura [**LINECALLINFO**](/windows/win32/api/tapi/ns-tapi-linecallinfo) apontada por *lpCallInfo*).

**TAPI 3. x:** Consulte [**ITCallInfo:: get \_ CallInfoLong**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfolong) (**membro \_ Completeid do cil** de [**CALLINFO \_ Long**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_long)).
