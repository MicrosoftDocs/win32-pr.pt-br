# <a name="tspi_linedropnoowner"></a>TSPI \_ lineDropNoOwner

A função **TSPI \_ lineDropNoOwner** está obsoleta. A TAPI não descartará mais automaticamente uma chamada se nenhum proprietário inicial puder ser identificado, a fim de acomodar o controle distribuído de chamadas em vários computadores cliente em um ambiente cliente/servidor.
