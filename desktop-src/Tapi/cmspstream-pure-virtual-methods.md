# <a name="cmspstream-pure-virtual-methods"></a>Métodos virtuais puros CMSPStream

Esses métodos devem ser substituídos por classes derivadas.



| Métodos virtuais puros CMSPStream        | Descrição                                                                                                                                                                                               |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**obter \_ nome**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_name) | Obtém o nome do fluxo. A classe de fluxo derivada deve fornecer um nome que permita que um usuário faça distinções significativas entre fluxos se isso for chamado para na interface do usuário do aplicativo. |
