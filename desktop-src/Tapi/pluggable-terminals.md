# <a name="pluggable-terminals"></a>Terminais conectáveis

Os terminais conectáveis permitem que um aplicativo crie terminais durante o tempo de execução. Os terminais são normalmente criados e controlados por um par de provedor de serviços de telefonia/provedor de serviços de mídia (TSP/MSP), e isso continua sendo o meio mais versátil e estável de manipular os terminais. No entanto, alguns aplicativos precisam usar terminais simples em situações em que um par de TSP/MSP apropriado não está disponível. Somente os gravadores de aplicativo que estão totalmente familiarizados com as operações de TSP/MSP devem usar terminais conectáveis.

Na TAPI versão 3,1, os terminais conectáveis são adicionados ao modelo TAPI. Os terminais conectáveis permitem que terceiros adicionem um novo objeto de terminal que qualquer MSP possa usar.

Consulte [registro de terminal conectável](pluggable-terminal-registration.md) e [gravando um terminal conectável](writing-a-pluggable-terminal.md) para obter mais informações.
