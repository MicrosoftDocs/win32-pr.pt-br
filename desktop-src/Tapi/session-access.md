# <a name="session-access"></a>Acesso à sessão

A referência principal para operações de sessão e informações está na principal visão geral deste SDK, em [controle de sessão](./session-control.md). Esta visão geral lista e define a terminologia envolvida e fornece saltos entre as páginas de referência TAPI 2,2 e TAPI 3 para elementos de programação relacionados.

O material a seguir aborda algumas informações que são relevantes apenas para TAPI 2,2 (TAPI/C).
