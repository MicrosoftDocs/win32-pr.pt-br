# <a name="tapi-21"></a>TAPI 2,1

A TAPI 2,1 fornece uma adição à especificação TAPI 2,0 anterior: funcionalidade de cliente/servidor. Com a TAPI 2,1 e posterior, o hardware de telefonia pode ser instalado em um servidor de telefonia que pode ser acessado de qualquer computador em uma rede. Um novo conjunto de funções permite que as DLLs de administração sejam gravadas para controlar quem tem acesso aos recursos remotos.

**Windows me/98/95:** A TAPI 2,1 fornece suporte para TSPs de 32 bits. Os binários do sistema TAPI 2,1, quando instalado, oferecem suporte:

-   32-bit TAPI 1,3, TAPI 1,4 e aplicativos TAPI 2. x
-   32-bit TAPI 2. x TSPs
-   aplicativos TAPI 1,3 e TAPI 1,4 de 16 bits
-   TAPI 1,3 de 16 bits e TAPI 1,4 TSPs
