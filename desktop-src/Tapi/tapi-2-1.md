# <a name="tapi-21"></a>TAPI 2.1

O TAPI 2.1 fornece uma adição à especificação anterior do TAPI 2.0: funcionalidade de cliente/servidor. Com o TAPI 2.1 e posterior, o hardware de telefonia pode ser instalado em um servidor de telefonia que pode ser acessado de qualquer computador em uma rede. Um novo conjunto de funções permite que as DLLs de administração sejam escritas para controlar quem tem acesso a recursos remotos.

**Windows Me/98/95:** O TAPI 2.1 dá suporte a TSPs de 32 bits. Os binários do sistema TAPI 2.1, quando instalados, são suportados:

-   Aplicativos TAPI 1.3, TAPI 1.4 e TAPI 2.x de 32 bits
-   TSPs TAPI 2.x de 32 bits
-   Aplicativos TAPI 1.3 e TAPI 1.4 de 16 bits
-   TAPI 1.3 e TAPI 1.4 TSPs de 16 bits
