# <a name="telephony-application-programming-interface-version-31"></a>Interface de Programação de Aplicativo de Telefonia versão 3.1

## <a name="purpose"></a>Finalidade

A TAPI (Interface de Programação de Aplicativos de Telefonia) da Microsoft versão 3.1 é uma API baseada em COMPONENT OBJECT MODEL (COM) que mescla telefonia IP e clássica.

## <a name="where-applicable"></a>Quando aplicável

Os possíveis aplicativos TAPI incluem:

-   Conferência de IP multimídia multicast com qualidade de serviço (QOS)
-   Chamadas de voz pela Internet usando o protocolo H.323
-   Aplicativos de call center capazes de acompanhar vários agentes
-   Chamadas de voz básicas na PSTN (Rede De Telefone Comutado Pública)
-   Controle PBX
-   Sistemas IVR (resposta de voz interativa)
-   Mensagem de voz

## <a name="developer-audience"></a>Público de desenvolvedores

Você pode escrever aplicativos habilitados para TAPI em várias linguagens, incluindo Java, Visual Basic e C/C++. A familiaridade com COM é necessária. A experiência de desenvolvimento com telecomunicações ou outros aplicativos de telefonia é útil, mas não é necessária.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

A TAPI versão 3.1 permite o desenvolvimento de aplicativos de comunicação para sistemas operacionais Windows Server 2003, Windows XP e Windows 2000.
