# <a name="assisted-telephony-services-reference"></a>Referência dos Serviços de Telefonia Assistida

As funções de Telefonia Assistida são listadas na tabela a seguir. As funções de Telefonia Assistida permitem que qualquer aplicativo estabeleça chamadas de voz e chamadas de mídia. Para obter informações em segundo plano, consulte [Visão geral da Telefonia](./assisted-telephony-overview.md) Assistida.

## <a name="assisted-telephony-services-functions"></a>Funções dos Serviços de Telefonia Assistidos



| Função                                             | Descrição                                                 |
|------------------------------------------------------|-------------------------------------------------------------|
| [**Tapirequestmakecall**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)   | Envia uma solicitação para fazer uma chamada de voz.                    |
| [**tapiRequestMediaCall**](tapirequestmediacall.md) | Obsoleto. Não use.                                       |
| [**tapiRequestDrop**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)           | Obsoleto. Não use.                                       |
| [**tapiGetLocationInfo**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)   | Retorna códigos que identificam o país/região e a cidade/área. |
