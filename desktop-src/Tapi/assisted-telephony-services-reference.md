# <a name="assisted-telephony-services-reference"></a>Referência de serviços de telefonia assistida

As funções de telefonia assistidas são listadas na tabela a seguir. As funções de telefonia assistidas permitem que qualquer aplicativo estabeleça chamadas de voz e chamadas de mídia. Para obter informações básicas, consulte Visão geral de [telefonia assistida](./assisted-telephony-overview.md) .

## <a name="assisted-telephony-services-functions"></a>Funções de serviços de telefonia assistida



| Função                                             | Descrição                                                 |
|------------------------------------------------------|-------------------------------------------------------------|
| [**tapiRequestMakeCall**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)   | Envia uma solicitação para fazer uma chamada de voz.                    |
| [**tapiRequestMediaCall**](tapirequestmediacall.md) | Obsoleto. Não use.                                       |
| [**tapiRequestDrop**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)           | Obsoleto. Não use.                                       |
| [**tapiGetLocationInfo**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)   | Retorna códigos que identificam o país/região e a cidade/área. |
