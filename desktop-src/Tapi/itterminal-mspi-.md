# <a name="itterminal-mspi"></a>ITTerminal (MSPI)

A [**interface ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) é a interface base para um [objeto terminal](terminal-object.md). Ele fornece métodos para obter informações como a classe de terminal e a mídia com suporte.

A interface [**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) é implementada por um MSP e não estará disponível se não houver nenhum provedor de serviços de mídia associado ao endereço. Confira **ITTerminal** na seção Interface do MSP para obter detalhes sobre essa interface.
