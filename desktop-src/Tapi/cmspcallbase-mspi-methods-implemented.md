# <a name="cmspcallbase-mspi-methods-implemented"></a>Métodos CMSPCallBase MSPI implementados



| Métodos auxiliares do CMSPCallBase MSPI                             | Descrição                                                                                                                                                                           |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-createstream)         | Cria o fluxo de mídia para uma nova chamada. Adquire o bloqueio na função.                                                                                                           |
| [**EnumerateStreams**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-enumeratestreams) | Enumera os fluxos disponíveis no momento. Adquire o bloqueio na função.                                                                                                            |
| [**obter \_ fluxos**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-get_streams)          | Cria uma coleção de fluxos disponíveis no momento. Fornecido para aplicativos cliente de automação, como aqueles escritos no Microsoft Visual Basic. Adquire o bloqueio na função. |
