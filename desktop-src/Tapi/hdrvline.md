# <a name="hdrvline"></a>HDRVLINE

O DataType **HDRVLINE** representa o identificador opaco de um provedor de serviços para uma estrutura de dados de linha. O provedor de serviços deve resolver um valor desse tipo em uma referência à instância de estrutura de dados apropriada.
