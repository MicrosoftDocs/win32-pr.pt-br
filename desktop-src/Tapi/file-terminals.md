# <a name="file-terminals"></a>Terminais de arquivo

No TAPI 3,1, o objeto de terminal é expandido para incluir um terminal de arquivo. Os terminais de arquivo fornecem um meio simples de executar operações comuns, por exemplo, gravar um fluxo de áudio ou fluxos em um arquivo ou reprodução subsequente de áudio gravado em um fluxo.

Cinco objetos básicos são usados para manipular terminais de arquivo:

-   Terminal de reprodução de arquivo
-   Terminal de gravação de arquivo
-   Faixa de reprodução de arquivo
-   Faixa de gravação de arquivo
-   Evento de arquivo

Os objetos de terminal de arquivo podem ser considerados como uma abstração para o armazenamento, que geralmente é um arquivo. As faixas correspondem aos fluxos de dados de áudio individuais nesse armazenamento, como quando a voz é registrada.

Para obter mais informações sobre terminais de arquivo, consulte:

-   [Terminal de reprodução de arquivos de terminal e gravação de arquivos](file-playback-terminal-and-file-recording-terminal.md)
-   [Acompanhar terminais](track-terminals.md)

Para obter mais informações e exemplos de código e listas de tarefas relacionadas a terminais de arquivo, consulte [usando terminais de arquivo](using-file-terminals.md).
