# <a name="cmspcallmultigraph-protected-helper-methods"></a>Métodos auxiliares protegidos por CMSPCallMultiGraph



| Métodos auxiliares protegidos por CMSPCallMultiGraph                             | Descrição                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**RegisterWaitEvent**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-registerwaitevent)       | Adiciona um evento de espera ao pool de threads.                                                     |
| [**UnregisterWaitEvent**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-unregisterwaitevent)   | Remove um evento de espera do pool de threads.                                                |
| [**InternalCreateStream**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-internalcreatestream) | Chamado por [**CreateStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-createstream) para criar um objeto de fluxo. |
