# <a name="itsubstream-mspi"></a>ITSubStream (MSPI)

A interface [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) estará disponível se um MSP estiver associado à chamada e implementar subfluxos. Os métodos dessa interface permitem que um aplicativo recupere informações em um subfluxo; para iniciar, pausar ou parar o Subfluxo; e para selecionar ou cancelar a seleção de terminais em um subfluxo.

A interface [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) é implementada por um msp. Consulte **ITSubStream** na seção de interface msp para obter detalhes sobre essa interface.
