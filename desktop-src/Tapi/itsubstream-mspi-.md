# <a name="itsubstream-mspi"></a>ITSubStream (MSPI)

A interface [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) estará disponível se um MSP estiver associado à chamada e implementar substreams. Os métodos dessa interface permitem que um aplicativo recupere informações em um substream; para iniciar, pausar ou parar o substream; e para selecionar ou desmarcar terminais em um substream.

A interface [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) é implementada por um MSP. Consulte **ITSubStream na** seção Interface MSP para obter detalhes sobre essa interface.
