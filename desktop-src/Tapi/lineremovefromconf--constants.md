# <a name="lineremovefromconf_-constants"></a>\_Constantes LINEREMOVEFROMCONF

As constantes escalares **LINEREMOVEFROMCONF \_** descrevem como as partes que participam de uma chamada de conferência podem ser removidas de uma chamada de conferência.

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**LINEREMOVEFROMCONF \_ qualquer**
</dt> <dd> <dl> <dt>



Qualquer parte participante pode ser removida da chamada de conferência.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**LINEREMOVEFROMCONF \_ último**
</dt> <dd> <dl> <dt>



Somente a parte mais recentemente adicionada pode ser removida da chamada de conferência


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**LINEREMOVEFROMCONF \_ nenhum**
</dt> <dd> <dl> <dt>



As partes não podem ser removidas da chamada de conferência.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Comentários

Sem extensibilidade. Todos os 32 bits são reservados.
