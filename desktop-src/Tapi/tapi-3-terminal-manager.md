# <a name="tapi-3-terminal-manager"></a>TAPI 3 Terminal Manager

O Gerenciador de Terminal tapi 3 é criado pelo Tapi3.dll a solicitação de um MSP e é usado pelo MSP para tarefas como a criação e a exclusão de terminais dinâmicos.

Os detalhes precisos de implementação para os terminais dinâmicos de um MSP estão além do escopo desta documentação. No entanto, muitos autores podem optar por usar alguns dos recursos do MST (Terminal de Streaming de Mídia). Esse terminal é baseado em DirectShow e filtros específicos a um determinado protocolo podem ser escritos e usados em conjunto com o MST.
