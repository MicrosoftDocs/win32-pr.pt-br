# <a name="tapi-3-terminal-manager"></a>Gerenciador de terminal TAPI 3

O Gerenciador de terminal TAPI 3 é criado por Tapi3.dll na solicitação de um MSP e é usado pelo MSP para tarefas como a criação e a exclusão de terminais dinâmicos.

Os detalhes de implementação preciso para os terminais dinâmicos de um MSP estão além do escopo desta documentação. No entanto, muitos autores podem optar por usar alguns dos recursos do terminal de streaming de mídia (MST). Esse terminal é baseado no DirectShow e filtros específicos de um determinado protocolo podem ser escritos e usados em conjunto com o MST.
