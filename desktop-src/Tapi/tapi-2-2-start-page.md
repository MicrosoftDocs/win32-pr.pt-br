# <a name="telephony-application-programming-interface-version-22"></a>Interface de programação de aplicativo de telefonia versão 2,2

## <a name="purpose"></a>Finalidade

A TAPI (interface de programação de aplicativos de telefonia) da Microsoft versão 2,2 (TAPI/C) permite a implementação de aplicativos de comunicação que vão desde o controle de modem básico até centros de chamadas com vários agentes e comutadores.

## <a name="where-applicable"></a>Quando aplicável

Os aplicativos TAPI 2,2 possíveis incluem:

-   Chamadas de voz básicas na rede telefônica pública comutada (PSTN)
-   Aplicativos de Call Center para acompanhamento de vários agentes
-   Controle de modem
-   Controle de PBX
-   Sistemas de reação a voz interativa (IVR)
-   Mensagem de voz
-   Controle de dispositivo de telefone detalhado

## <a name="developer-audience"></a>Público de desenvolvedores

A TAPI/C foi projetada para uso por programadores C/C++. A experiência de desenvolvimento com telecomunicações ou outros aplicativos de telefonia é útil, mas não é necessária.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

A TAPI versão 2,2 permite o desenvolvimento de aplicativos de comunicação para os sistemas operacionais Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows me, Windows 98 e Windows 95. Para obter mais informações sobre quais sistemas operacionais dão suporte a uma função específica, consulte a seção requisitos da documentação para essa função.
