# <a name="itstream-mspi"></a>ITStream (MSPI)

A interface [**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) estará disponível se um MSP estiver associado à chamada. Os métodos dessa interface permitem que um aplicativo recupere informações em um fluxo; para iniciar, pausar ou parar o fluxo; e para selecionar ou desmarcar terminais em um fluxo.

A interface [**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) é implementada por um MSP. Consulte **ITStream** na seção Interface MSP para obter detalhes sobre essa interface.
