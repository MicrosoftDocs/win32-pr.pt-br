# <a name="terminal"></a>Terminal

Um terminal é uma fonte ou um renderizador para um determinado endereço e tipo de mídia. O conceito de terminal é usado principalmente na TAPI 3 e é apenas muito importante para aplicativos TAPI 2. Para obter mais informações, consulte Visão geral do [objeto de terminal](terminal-object.md) TAPI 3.
