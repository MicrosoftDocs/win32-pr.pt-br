# <a name="h323-tsp"></a>H323 TSP

\[O H323 TSP não está disponível para uso no Windows Vista, no Windows Server 2008 e em versões subsequentes do sistema operacional. A API do cliente RTC fornece funcionalidade semelhante.\]

O H323 TSP (H323. TSP), quando emparelhado com o [H323 MSP](./h323-msp.md) (H323msp.dll), permite que um aplicativo faça e receba chamadas de vídeo e áudio usando o protocolo H. 323 para a telefonia IP.

Esse TSP dá suporte a valores de [**tipo de endereço**](./lineaddresstype--constants.md) de LINEADDRESSTYPE \_ PHONENUMBER, LINEADDRESSTYPE \_ IPAddress e LINEADDRESSTYPE \_ DomainName. O LINEADDRESSTYPE \_ DOmainname refere-se ao nome de domínio do computador.

O H323 MSP foi aprimorado no Windows 2000 SP1 para habilitar a comunicação entre clientes do H323 e do SDP.

O H323 TSP é instalado com o Windows 2000 e posterior.
