# <a name="hdrvphone"></a>HDRVPHONE

O **tipo de dados HDRVPHONE** representa o handle opaco de um provedor de serviços para uma estrutura de dados de telefone. O provedor de serviços deve resolver um valor desse tipo em uma referência à instância de estrutura de dados apropriada.
