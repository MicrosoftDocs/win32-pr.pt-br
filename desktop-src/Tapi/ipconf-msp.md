# <a name="ipconf-msp"></a>IPConf MSP

\[O IPConf MSP não está disponível para uso no Windows Vista, Windows Server 2008 e versões subsequentes do sistema operacional. A API do Cliente RTC fornece funcionalidade semelhante.\]

O IPConf MSP (Confmsp.dll), quando emparelhado com seu TSP (IPconf.tsp), fornece acesso de conferência a redes ip (protocolo IP), como a Internet e muitas intranets corporativas. Além dos métodos MSPI padrão, esse MSP expõe interfaces que permitem o controle sobre participantes individuais em uma conferência.

Consulte [Interfaces IPConf MSP](ipconf-msp-interfaces.md) para ver um resumo das interfaces específicas do provedor expostas por este MSP.

O IPConf MSP foi aprimorado no Windows 2000 SP1 para habilitar a comunicação entre clientes H323 e SDP. Agora, ele também expõe a interface [**IMulticastControl.**](imulticastcontrol.md)

O IPConf MSP é instalado com Windows 2000.
