# <a name="ipconf-msp"></a>IPConf MSP

\[O IPConf MSP não está disponível para uso no Windows Vista, no Windows Server 2008 e em versões subsequentes do sistema operacional. A API do cliente RTC fornece funcionalidade semelhante.\]

O IPConf MSP (Confmsp.dll), quando emparelhado com seu TSP (IPconf. TSP), fornece acesso de conferência a redes IP (Internet Protocol), como a Internet e muitas intranets corporativas. Além dos métodos MSPI padrão, esse MSP expõe interfaces que permitem o controle sobre os participantes individuais em uma conferência.

Consulte as [interfaces do IPConf MSP](ipconf-msp-interfaces.md) para obter um resumo das interfaces específicas do provedor expostas por este msp.

O IPConf MSP foi aprimorado no Windows 2000 SP1 para habilitar a comunicação entre os clientes do H323 e do SDP. Ele agora também expõe a interface [**IMulticastControl**](imulticastcontrol.md) .

O IPConf MSP é instalado com o Windows 2000.
