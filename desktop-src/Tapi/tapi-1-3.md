# <a name="tapi-13"></a>TAPI 1,3

A TAPI 1,3 (às vezes conhecida como TAPI 1, 3) foi a primeira versão lançada da especificação TAPI. Os binários do sistema TAPI 1,3 (Tapi.dll e outros) são encontrados no SDK da TAPI 1,0, que pode ser encontrado somente na assinatura do Microsoft Developer Network (MSDN) Level II. Os binários do sistema (encontrados no diretório redist) são redistribuíveis, mas são projetados para serem executados somente no Microsoft Windows 3,1 e no Windows para Workgroups 3,11. Eles não têm suporte e não funcionarão em nenhuma versão do Windows de 32 bits.

Os cabeçalhos e as bibliotecas que são usados para desenvolver um aplicativo TAPI 1,3 estão disponíveis somente no SDK do TAPI 1,0 e dão suporte apenas ao desenvolvimento de 16 bits. Na verdade, as bibliotecas TAPI 1,3 são as únicas bibliotecas TAPI de 16 bits disponíveis e devem ser usadas para todos os aplicativos de 16 bits, independentemente da plataforma de destino ou da versão de destino.

Os binários do sistema TAPI 1,3 dão suporte somente a Win16, TAPI 1,3 TSPs e aplicativos.
