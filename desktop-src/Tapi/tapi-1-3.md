# <a name="tapi-13"></a>TAPI 1.3

TAPI 1.3 (às vezes conhecido como TAPI 1.03) foi a primeira versão lançada da especificação tapi. Os binários do sistema TAPI 1.3 (Tapi.dll e outros) são encontrados no SDK da TAPI 1.0, que pode ser encontrado somente na assinatura do nível II da Rede Desenvolvedor Microsoft (MSDN). Os binários do sistema (encontrados no diretório REDIST) são redistribuíveis, mas foram projetados para ser executados somente no Microsoft Windows 3.1 e no Windows for Workgroups 3.11. Eles não têm suporte e não funcionarão em nenhuma versão de 32 bits Windows.

Os headers e bibliotecas usados para desenvolver um aplicativo TAPI 1.3 estão disponíveis apenas no SDK da TAPI 1.0 e são suportados apenas para desenvolvimento de 16 bits. Na verdade, as bibliotecas TAPI 1.3 são as únicas bibliotecas TAPI de 16 bits disponíveis e devem ser usadas para todos os aplicativos de 16 bits, independentemente da plataforma de destino ou da versão de destino.

Os binários do sistema TAPI 1.3 são suportados apenas por TSPs Win16, TAPI 1.3 e aplicativos.
