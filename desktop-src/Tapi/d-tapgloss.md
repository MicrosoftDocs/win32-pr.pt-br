# <a name="d-telephony-api"></a>D (API de telefonia)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) D [E](e-tapgloss.md) F G [H](h-tapgloss.md) [i](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.d_channel_tapgloss"></span><span id="TAPI2.D_CHANNEL_TAPGLOSS"></span>**Canal D**
</dt> <dd>

Um canal de dados da interface ISDN. É um dos três canais padrão em uma linha ISDN de taxa básica. Em uma linha do BRI-ISDN, um canal D pode transportar informações de sinalização e dados de pacotes de baixa velocidade a uma taxa de 16.000 bps. Em uma linha PRI-ISDN, o canal D é executado às 64.000 bps. Consulte [*interface de taxa básica (BRI-ISDN)*](b-tapgloss.md), [*canal B*](b-tapgloss.md), [*interface de taxa primária (PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.device_class_tapgloss"></span><span id="TAPI2.DEVICE_CLASS_TAPGLOSS"></span>**classe de dispositivo**
</dt> <dd>

Um grupo de dispositivos físicos ou drivers de dispositivo relacionados por meio dos quais os aplicativos enviam e recebem as informações ou os dados que compõem uma chamada. Para obter uma lista, consulte [classes de dispositivo TAPI](tapi-device-classes.md).

</dd> <dt>

<span id="tapi2.did_tapgloss"></span><span id="TAPI2.DID_TAPGLOSS"></span>**ACEITOU**
</dt> <dd>

Consulte a *discagem Direct Inward (DID)*.

</dd> <dt>

<span id="tapi2.direct_inward_dialing_did__tapgloss"></span><span id="TAPI2.DIRECT_INWARD_DIALING_DID__TAPGLOSS"></span>**discagem Direct Inward (DID)**
</dt> <dd>

Uma capacidade fornecida por praticamente todos os PBXs modernos que permitem que um chamador externo atinja uma extensão interna, discando um número de telefone padrão de sete dígitos sem precisar passar por um operador ou atendente. Consulte [*intercâmbio de ramificação privada (PBX)*](p-tapgloss.md).

</dd> </dl>
