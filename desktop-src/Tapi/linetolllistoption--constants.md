# <a name="linetolllistoption_-constants"></a>\_Constantes LINETOLLLISTOPTION

As constantes de sinalizador de bit **LINETOLLLISTOPTION \_** descrevem opções para manipular uma lista de chamadas.

<dl> <dt>

<span id="LINETOLLLISTOPTION_ADD"></span><span id="linetolllistoption_add"></span>**LINETOLLLISTOPTION \_ Adicionar**
</dt> <dd> <dl> <dt>



Um prefixo deve ser adicionado à lista de chamadas.


</dt> </dl> </dd> <dt>

<span id="LINETOLLLISTOPTION_REMOVE"></span><span id="linetolllistoption_remove"></span>**LINETOLLLISTOPTION \_ remover**
</dt> <dd> <dl> <dt>



Um prefixo deve ser removido da lista de chamadas.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Comentários

Sem extensibilidade. Todos os 32 bits são reservados.
