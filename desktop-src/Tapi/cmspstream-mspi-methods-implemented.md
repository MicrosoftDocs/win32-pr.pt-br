# <a name="cmspstream-mspi-methods-implemented"></a>Métodos MSPI do CMSPStream implementados



| Métodos MSPI do CMSPStream                                   | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**obter \_ MediaType**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_mediatype)          | Obtém o tipo de [**mídia do fluxo**](tapimediatype--constants.md).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**obter \_ Direção**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_direction)          | Obtém a direção [**do terminal do fluxo.**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_direction)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**SelectTerminal**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-selectterminal)         | Seleciona um [**objeto ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) no fluxo. Adquire o bloqueio na função. Esse método de classe base só sabe como manter a lista de terminais; Espera-se que a classe derivada substitua esse método para fornecer funcionalidade adicional (como parar e reiniciar o fluxo se um terminal for selecionado enquanto o fluxo estiver em execução). A implementação da classe base verifica se o ponteiro do terminal aponta para memória válida, obtém a interface [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol) do terminal, recupera o alça de endereço do terminal e verifica se ele corresponde ao alça de endereço do MSP, verifica se o terminal já existe na lista do fluxo, adiciona o terminal à lista de terminais do fluxo e AddRefs. Se algum dos testes falhar, ele retornará TAPI \_ E \_ INVALIDTERMINAL. |
| [**DesmarcarTerminal**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-unselectterminal)     | Desmarca o terminal do fluxo. Adquire o bloqueio na função. Localiza o terminal na lista de terminais do fluxo, remove-o da lista e o libera. Se o terminal não for encontrado, ele retornará TAPI \_ E \_ INVALIDTERMINAL.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**EnumerateTerminals**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-enumerateterminals) | Enumera os terminais selecionados no fluxo. É um instantâneo dos terminais neste ponto. Adquire o bloqueio na função.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**obter \_ terminais**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_terminals)          | Cria uma coleção de terminais associados ao fluxo atual. Fornecido para aplicativos cliente de Automação, como aqueles escritos no Microsoft Visual Basic. É um instantâneo dos terminais neste ponto. Adquire o bloqueio na função.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**StartStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-startstream)               | Inicia o fluxo. Chama **IMediaControl::Run** no grafo de filtro.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**PauseStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-pausestream)               | Pausa o fluxo. Chama **IMediaControl::P ause** no grafo de filtro.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**StopStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-stopstream)                 | Interrompe o fluxo. Chama **IMediaControl::Stop** no grafo de filtro.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
