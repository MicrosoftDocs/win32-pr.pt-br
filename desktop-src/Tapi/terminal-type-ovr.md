# <a name="terminal-type"></a>Tipo de terminal

O **tipo de terminal** identifica se um terminal é estático ou dinâmico. Por exemplo, um modem é um terminal estático enquanto a janela de vídeo em uma tela de computador é criada quando é necessária e destruída quando sua utilidade é ativada.

Esse conceito é usado principalmente por aplicativos TAPI 3.

**TAPI 3. x:** Consulte [**ITTerminal:: obter \_ terminaltype**](/windows/win32/api/tapi3if/nf-tapi3if-itterminal-get_terminaltype), [**\_ tipo de terminal**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_type).
