# <a name="displayable-address"></a>Endereço de exibição

O endereço exibível é uma cadeia de caracteres que apresenta o endereço em um formato que é adequado para exibição em uma interface do usuário.

**TAPI 2. x:** Consulte membros [**lineGetCallInfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo), **dwDisplayableAddressSize** e **dwDisplayableAddressOffset** de [**LINECALLINFO**](/windows/win32/api/tapi/ns-tapi-linecallinfo).

**TAPI 3. x:** Consulte [**ITCallInfo:: get \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfostring) ou [**ITCallInfo::p UT \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-put_callinfostring), chamado com o **membro \_ DISPLAYABLEADDRESS do CIS** da [**cadeia de \_ caracteres CALLINFO**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_string).
