# <a name="displayable-address"></a>Endereço exibivel

O endereço exibivel é uma cadeia de caracteres que apresenta o endereço em um formulário adequado para exibição em uma interface do usuário.

**TAPI 2.x:** Consulte [**os membros lineGetCallInfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo), **dwDisplayableAddressSize** e **dwDisplayableAddressOffset** de [**LINECALLINFO.**](/windows/win32/api/tapi/ns-tapi-linecallinfo)

**TAPI 3.x:** Consulte [**ITCallInfo::get \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfostring) ou [**ITCallInfo::p ut \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-put_callinfostring), chamado com o membro **CIS \_ DISPLAYABLEADDRESS** de [**CALLINFO \_ STRING**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_string).
