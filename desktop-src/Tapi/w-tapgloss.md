# <a name="w-telephony-api"></a>W (API de telefonia)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F G [H](h-tapgloss.md) [i](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) W X Y Z

<dl> <dt>

<span id="tapi2.web_telephony_tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_TAPGLOSS"></span>**Telefonia da Web**
</dt> <dd>

Uma plataforma de aplicativo que dá suporte a sites de acesso duplo (acessíveis por telefone ou navegador da Internet) e aplicativos IVR baseados em HTML. A plataforma inclui um mecanismo de tempo de execução integrado ao servidor, uma interface de administração baseada em COM para instalar e configurar aplicativos de telefonia da Web e acesso ao objeto de chamada TAPI 3. x e à API de script, para implementar operações de telefonia.

</dd> <dt>

<span id="tapi2.web_telephony_engine_webte__tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_ENGINE_WEBTE__TAPGLOSS"></span>**Mecanismo de telefonia da Web (WTE)**
</dt> <dd>

Fornece suporte para aplicativos IVR baseados em HTML e sites da Web de acesso duplo (um site que é acessível por telefone, bem como por meio de um navegador gráfico). Consulte *telefonia na Web*.

</dd> <dt>

<span id="tapi2.windows_open_services_architecture_wosa__tapgloss"></span><span id="TAPI2.WINDOWS_OPEN_SERVICES_ARCHITECTURE_WOSA__TAPGLOSS"></span>**WOSA (arquitetura de serviços abertos do Windows)**
</dt> <dd>

Arquitetura de software que fornece uma interface de nível de sistema única para conectar aplicativos de front-end com serviços de back-end. A telefonia do Windows faz parte do WOSA.

</dd> </dl>
