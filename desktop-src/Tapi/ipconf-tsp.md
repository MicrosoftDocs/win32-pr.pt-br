# <a name="ipconf-tsp"></a>IPConf TSP

\[O IPConf TSP não está disponível para uso no Windows Vista, no Windows Server 2008 e em versões subsequentes do sistema operacional. A API do cliente RTC fornece funcionalidade semelhante.\]

O IPConf TSP (IPConf. TSP), quando emparelhado com o [IPConf MSP](./ipconf-msp.md) (Confmsp.dll), fornece acesso de conferência a redes de protocolo de Internet, como a Internet e muitas intranets corporativas.

Este TSP dá suporte a um valor de [**tipo de endereço**](./lineaddresstype--constants.md) de \_ SDP LINEADDRESSTYPE.

O IPConf MSP foi aprimorado no Windows 2000 SP1 para habilitar a comunicação entre os clientes do H323 e do SDP.

O IPConf TSP é instalado com o Windows 2000 e posterior.
