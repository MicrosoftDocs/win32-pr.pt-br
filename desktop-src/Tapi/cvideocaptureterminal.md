# <a name="cvideocaptureterminal"></a>CVideoCaptureTerminal

O terminal **CVideoCaptureTerminal** é derivado de [CSingleFilterTerminal](csinglefilterterminal.md) e implementa um terminal de captura de vídeo estático usando um único filtro do DirectShow. Ele não dá suporte a certas interfaces avançadas de captura de vídeo aplicáveis somente à captura de vídeo WDM. A maioria dos MSPs não precisará substituir a implementação desse terminal.
