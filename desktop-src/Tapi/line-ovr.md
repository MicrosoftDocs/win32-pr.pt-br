# <a name="line"></a><span data-ttu-id="2e9a8-104">Linha</span><span class="sxs-lookup"><span data-stu-id="2e9a8-104">Line</span></span>

<span data-ttu-id="2e9a8-105">O conceito de linha evoluiu ao longo do tempo e foi parcialmente substituído pelos conceitos de endereço e terminal.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-105">The line concept has evolved over time and been partly superseded by the Address and Terminal concepts.</span></span> <span data-ttu-id="2e9a8-106">A TAPI 3 não usa diretamente o conceito de linha, mas a TAPI 2 continua a incorporar esse paradigma.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-106">TAPI 3 does not directly use the concept of line, but TAPI 2 continues to incorporate this paradigm.</span></span>

<span data-ttu-id="2e9a8-107">Um *dispositivo de linha* é um dispositivo físico, como uma placa de fax, um modem ou uma placa ISDN que está conectada a uma rede.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-107">A *line device* is a physical device such as a fax board, a modem, or an ISDN card that is connected to a network.</span></span> <span data-ttu-id="2e9a8-108">O dispositivo pode não estar fisicamente conectado ao computador no qual o aplicativo TAPI está sendo executado, como um pool de modems em um servidor.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-108">The device may not be physically connected to the computer on which the TAPI application is running, such as a modem pool on a server.</span></span> <span data-ttu-id="2e9a8-109">Os dispositivos de linha dão suporte a recursos de comunicação, permitindo que os aplicativos enviem informações ou recebam informações de uma rede.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-109">Line devices support communications capabilities by allowing applications to send information to or receive information from a network.</span></span> <span data-ttu-id="2e9a8-110">Um dispositivo de linha contém um conjunto de um ou mais canais homogêneos que podem ser usados para estabelecer chamadas.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-110">A line device contains a set of one or more homogeneous channels that can be used to establish calls.</span></span>

<span data-ttu-id="2e9a8-111">Em aplicativos TAPI 2. x, um dispositivo de linha é a representação lógica de um dispositivo de telefone físico.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-111">Within TAPI 2.x applications, a line device is the logical representation of a physical telephone device.</span></span> <span data-ttu-id="2e9a8-112">Embora a "linha" sempre denotasse algo com dois pontos de extremidade, é possível abstrair um dispositivo de linha para um único ponto, pois a TAPI o exibe apenas como um ponto de entrada para a linha que leva à mudança.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-112">Although "line" often connotes something with two endpoints, it is possible to abstract a line device to a single point because TAPI views it only as a point of entry to the line that leads to the switch.</span></span>

<span data-ttu-id="2e9a8-114">Embora as três linhas na ilustração anterior sejam compostas de hardware diferente e usadas para funções diferentes, elas são abstratas para o mesmo tipo de dispositivo e governadas pelas mesmas regras.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-114">Although the three lines in the preceding illustration are composed of different hardware and used for different functions, they are abstracted to the same device type and governed by the same rules.</span></span> <span data-ttu-id="2e9a8-115">O telefone não representa um dispositivo de telefone, mas um dispositivo de linha usado para chamadas de voz.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-115">The telephone represents not a phone device but a line device used for voice calls.</span></span> <span data-ttu-id="2e9a8-116">Ao usar esse dispositivo de linha para chamadas de entrada ou saída, o aplicativo também precisaria abrir e controlar uma instância da classe de dispositivo de telefone, que é descrita detalhadamente nas seções posteriores.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-116">When using this line device for incoming or outgoing calls, the application would also need to open and control an instance of the phone-device class, which is described in detail in later sections.</span></span>

<span data-ttu-id="2e9a8-117">A classe de dispositivo line é uma representação independente de dispositivo de um dispositivo de linha física, como um modem.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-117">The line device class is a device-independent representation of a physical line device, such as a modem.</span></span> <span data-ttu-id="2e9a8-118">Ele pode conter um ou mais canais de comunicação idênticos (usados para sinalização e/ou informações) entre o aplicativo e o comutador ou a rede.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-118">It can contain one or more identical communications channels (used for signaling and/or information) between the application and the switch or network.</span></span> <span data-ttu-id="2e9a8-119">Como os canais que pertencem a uma única linha têm recursos idênticos, eles são intercambiáveis.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-119">Because channels belonging to a single line have identical capabilities, they are interchangeable.</span></span> <span data-ttu-id="2e9a8-120">Em muitos casos (como com o POTS), um provedor de serviços modelará uma linha como tendo apenas um canal.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-120">In many cases (as with POTS), a service provider will model a line as having only one channel.</span></span> <span data-ttu-id="2e9a8-121">Outras tecnologias, como o ISDN, oferecem mais canais e o provedor de serviços devem tratá-las adequadamente.</span><span class="sxs-lookup"><span data-stu-id="2e9a8-121">Other technologies, like ISDN, offer more channels, and the service provider should treat them accordingly.</span></span>
