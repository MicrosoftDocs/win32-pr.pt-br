# <a name="ikeyframecontrolupdatepicture-method"></a>Método IKeyFrameControl:: UpdatePicture

\[O **UpdatePicture** não está disponível para uso no Windows Vista, no windows Server 2008 e em versões subsequentes do sistema operacional. A API do cliente RTC fornece funcionalidade semelhante.\]

O método **UpdatePicture** é chamado por um aplicativo para solicitar que o remetente do vídeo atualize a imagem. Sempre que esse método é chamado, uma solicitação é enviada ao remetente do vídeo. O fluxo deve estar ativo.
