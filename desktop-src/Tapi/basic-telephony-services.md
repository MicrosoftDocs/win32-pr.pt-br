# <a name="basic-telephony-services"></a>Serviços de telefonia básicos

Os serviços de telefonia básicos são um subconjunto mínimo da especificação de telefonia. Como todos os provedores de serviços devem dar suporte às funções de serviços de telefonia básicos, os aplicativos que usam apenas essas funções funcionarão com qualquer provedor de serviços TAPI. A funcionalidade contida na telefonia básica aproximadamente corresponde aos recursos de POTS.

Muitos programadores usarão apenas os serviços que a telefonia básica fornece. Outros, como aqueles que escrevem código para sistemas de telefone PBX, precisarão das funções de telefonia suplementar.

Para obter uma lista das funções de telefonia básica, consulte [referência rápida de função TAPI](./tapi-quick-function-reference.md).

Como o controle de dispositivos de telefone não é considerado para ser oferecido por todos os provedores de serviço, os serviços de dispositivo de telefone são considerados opcionais. Ou seja, eles não fazem parte da telefonia básica. Para obter uma lista de serviços de dispositivo de telefone, consulte [serviços de telefonia suplementares](supplementary-telephony-services.md); para obter mais informações sobre dispositivos de telefone, consulte [classes de dispositivo TAPI](./tapi-device-classes.md).
