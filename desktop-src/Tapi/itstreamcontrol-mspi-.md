# <a name="itstreamcontrol-mspi"></a>ITStreamControl (MSPI)

A interface [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) será exposta no objeto de chamada se houver um MSP associado. Os métodos dessa interface permitem que um aplicativo crie e remova fluxos.

A interface [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) é implementada por um msp. Consulte **ITStreamControl** na seção de interface msp para obter detalhes sobre essa interface.
