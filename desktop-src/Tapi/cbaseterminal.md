# <a name="cbaseterminal"></a>CBaseTerminal

**CBaseTerminal** é uma classe base de terminal aplicável a terminais estáticos e dinâmicos. Ele é completamente genérico, portanto, qualquer implementação de terminal pode derivar dessa classe direta ou indiretamente.

Definido em Mspterm. h.
