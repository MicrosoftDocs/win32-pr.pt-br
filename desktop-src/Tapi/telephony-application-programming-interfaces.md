# <a name="telephony-application-programming-interfaces"></a>Interfaces de programação de aplicativo de telefonia

As interfaces de programação de aplicativo de telefonia da Microsoft dão suporte ao desenvolvimento de aplicativos de comunicação para o Microsoft Windows. As interfaces de telefonia estão listadas na tabela a seguir.



| Interface                                                  | Descrição                                                                                                                                                                                                                                                                                                                                             |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [TAPI 2. x](./tapi-2-2-start-page.md)               | Uma API baseada em linguagem C que permite implementar aplicativos de comunicação que vão desde o controle de modem básico até centros de chamadas com vários agentes e comutadores.                                                                                                                                                                        |
| [TAPI 3. x](tapi-3-1-start-page.md)                        | Uma API baseada em COM que mescla a telefonia clássica e IP.                                                                                                                                                                                                                                                                                                   |
| [TSPI](./telephony-service-providers-start-page.md) | Um TSP (provedor de serviços de telefonia) é uma DLL (biblioteca de vínculo dinâmico) que dá suporte ao controle de dispositivo de comunicação por meio de um conjunto de funções de serviço exportadas. Um aplicativo TAPI usa comandos padronizados, a TAPI passa informações para o provedor de serviços de telefonia e o TSP lida com os comandos específicos que devem ser trocados pelo dispositivo. |
| [MSPI](media-service-providers-start-page.md)             | Um MSP (provedor de serviços de mídia) permite um controle considerável do aplicativo sobre a mídia para um mecanismo de transporte específico. Um MSP é sempre emparelhado com um provedor de serviços de telefonia (TSP).                                                                                                                                                         |
