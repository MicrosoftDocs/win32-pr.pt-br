# <a name="remote-tsp"></a>TSP remoto

O TSP Remoto (Remotesp.tsp) fornece acesso a recursos de comunicação não locais para o computador atual, como um PBX. Quando o TSP Remoto é configurado para apontar para um determinado servidor, os recursos aparecem para um aplicativo local como se eles existiam no computador do usuário.

Tcmsetup é um utilitário de linha de comando que permite a configuração de um TSP Remoto. Para obter mais informações sobre esse utilitário, consulte Windows ajuda ou o Windows Resource Kit.

O TSP Remoto é instalado com sistemas operacionais do Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Me, Windows 98 e Windows 95.
