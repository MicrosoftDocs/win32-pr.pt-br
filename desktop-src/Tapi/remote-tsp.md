# <a name="remote-tsp"></a>TSP remoto

O TSP (Remotesp. TSP) remoto fornece acesso a recursos de comunicação não locais para o computador atual, como um PBX. Quando o TSP remoto é configurado para apontar para um determinado servidor, os recursos aparecem para um aplicativo local como se existissem no computador do usuário.

Tcmsetup é um utilitário de linha de comando que permite a configuração de um TSP remoto. Para obter mais informações sobre esse utilitário, consulte a ajuda do Windows ou o Windows Resource Kit.

O TSP remoto é instalado com os sistemas operacionais Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows me, Windows 98 e Windows 95.
