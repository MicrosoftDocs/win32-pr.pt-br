# <a name="tapi-levels-of-service"></a>Níveis de serviço TAPI

Os serviços de telefonia TAPI são divididos em serviços de telefonia assistida e os serviços fornecidos pela API de telefonia completa. Em geral, a API de telefonia completa é usada para implementar aplicativos de telefonia avançados, e a telefonia assistida é usada para adicionar funcionalidade de telefonia mínima, mas útil, a aplicativos sem telefonia.
