# <a name="call-and-media-controls-reference"></a>Referência de controles de chamada e de mídia

O material de referência de controle de mídia e chamada contém informações sobre interfaces, métodos, enumeradores e tipos usados na configuração e no controle de chamada.

Você também pode desejar pesquisar o SDK (Kit de desenvolvimento de software) da plataforma para obter informações sobre o DirectShow e a Component Object Model (COM). A TAPI 3. x usa filtros do DirectShow e gerenciadores de filtro para controlar o fluxo de mídia após a configuração de conexão. O COM é o modelo de codificação básico para a TAPI 3. x, e a familiaridade com ele é assumida em todo este documento.
