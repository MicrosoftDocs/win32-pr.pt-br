# <a name="supplementary-phone-services"></a>Serviços de Telefone suplementares

Os serviços de telefone suplementares são um conjunto de serviços avançados além dos serviços de telefone básicos, compatíveis com dispositivos de telefone. Normalmente, esse conjunto inclui os seguintes recursos do dispositivo: [hookswitch,](/previous-versions/windows/desktop/legacy/ms725202(v=vs.85)) [display](/previous-versions/windows/desktop/legacy/ms725186(v=vs.85)), anel [distinto,](/previous-versions/windows/desktop/legacy/ms725264(v=vs.85)) [botões,](buttons.md) [lâmpadas distintas,](/previous-versions/windows/desktop/legacy/ms725210(v=vs.85))área de dados [e](/previous-versions/windows/desktop/legacy/ms725182(v=vs.85))a capacidade de verificar o status da [chamada.](/previous-versions/windows/desktop/legacy/ms725508(v=vs.85))
