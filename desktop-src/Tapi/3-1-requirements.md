# <a name="tapi-31-requirements"></a>Requisitos do TAPI 3.1

Veja a seguir os requisitos do sistema operacional para TAPI 3.x:

-   O TAPI versão 3.0 requer Windows 2000 ou posterior.
-   O TAPI versão 3.1 requer Windows sistemas operacionais server 2003 ou Windows XP.

Observe que o [controle de Conferência de IP](about-rendezvous-ip-telephony-conferencing.md) usando o Reunião requer acesso a pelo menos um servidor ILS ou NTDS. Se um servidor NTDS estiver em local, ele poderá ser usado para localizar um servidor ILS. Se nenhum servidor NTDS existir, o usuário final deverá saber o nome do servidor ILS.
