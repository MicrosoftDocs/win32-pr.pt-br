# <a name="tapi-31-requirements"></a>Requisitos de TAPI 3,1

A seguir estão os requisitos do sistema operacional para a TAPI 3. x:

-   A TAPI versão 3,0 requer o Windows 2000 ou posterior.
-   A TAPI versão 3,1 requer sistemas operacionais Windows Server 2003 ou Windows XP.

Observe que o [controle de conferência de IP](about-rendezvous-ip-telephony-conferencing.md) que usa a reunião requer acesso a pelo menos um servidor ILS ou NTDS. Se um servidor NTDS estiver em vigor, ele poderá ser usado para localizar um servidor ILS. Se não existir nenhum servidor NTDS, o usuário final deverá saber o nome do servidor ILS.
