# <a name="summaries-of-call-center-programming-elements"></a>Resumos dos elementos de programação do Call Center

A funcionalidade do Call Center é um recurso do TAPI 3. x. Os elementos de programação que aparecem no conjunto de dados TAPI 2,2 dão suporte a uma central de atendimento TAPI 3. x. Para obter mais informações sobre centrais de chamadas, consulte [sobre controles do Call Center](./about-call-center-controls.md).

Os resumos do elemento de programação TAPI 2,2 Call Center incluem:

-   [Funções do Call Center](call-center-functions.md)
-   [Estruturas do Call Center](call-center-structures.md)
-   [Mensagens do Call Center](call-center-messages.md)
-   [Constantes de Call Center](call-center-constants.md)
