# <a name="third-party-msp"></a>MSP de terceiros

O hardware de comunicações especializado normalmente chega com um ou mais TSPs e MSPs específicos para o dispositivo. A documentação do dispositivo deve indicar recursos e fornecer diretrizes de programação.
