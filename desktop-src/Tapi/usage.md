# <a name="usage"></a>Uso

Há dois casos gerais de uso da interface DLL da interface do usuário do TSP: quando o aplicativo chama uma função que foi projetada para gerar a interface do usuário e quando o provedor de serviços deve gerar espontaneamente a interface do usuário no decorrer do processamento de uma função que não está definida como geração da interface do usuário.
