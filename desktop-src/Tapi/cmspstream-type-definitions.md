# <a name="cmspstream-type-definitions"></a>Definições de tipo de CMSPStream

As definições a seguir fornecem um conjunto de Estados de fluxo que podem ser úteis para a classe derivada, embora seu uso não seja necessário. As classes derivadas podem estender convenientemente o conjunto de Estados disponíveis simplesmente definindo valores adicionais.

``` syntax
#define STRM_INITIAL            0x00000000
#define STRM_TERMINALSELECTED   0x00000001
#define STRM_CONFIGURED         0x00000002
#define STRM_RUNNING            0x00000004
#define STRM_PAUSED             0x00000008
#define STRM_STOPPED            0x00000010
```
