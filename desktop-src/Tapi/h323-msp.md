# <a name="h323-msp"></a>H323 MSP

\[o H323 MSP não está disponível para uso no Windows Vista, no Windows Server 2008 e em versões subsequentes do sistema operacional. A API do cliente RTC fornece funcionalidade semelhante.\]

O H323 MSP (H323msp.dll), quando emparelhado com seu TSP (H323. TSP), permite que um aplicativo faça e receba chamadas de vídeo e áudio usando o protocolo H. 323 para telefonia IP.

o H323 MSP foi aprimorado no Windows 2000 SP1 para habilitar a comunicação entre os clientes do H323 e do SDP. Agora, ele expõe as interfaces [**IH323LineEx**](ih323lineex.md) e [**IKeyFrameControl**](ikeyframecontrol.md) .

o H323 MSP é instalado com o Windows 2000.
