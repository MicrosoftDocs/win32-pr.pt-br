# <a name="tapi-object"></a>Objeto TAPI

O objeto TAPI representa todos os recursos de telefonia aos quais o computador local tem acesso, permitindo que um aplicativo Enumere todos os endereços locais e remotos. Um aplicativo TAPI 3 deve criar uma instância do objeto TAPI e, em seguida, inicializá-lo.

O exemplo de código [inicializar TAPI](initialize-tapi.md) mostra um exemplo de criação de objeto TAPI.

Consulte [interfaces de objeto TAPI](tapi-object-interfaces.md) para obter uma lista de todas as interfaces associadas ao objeto TAPI.
