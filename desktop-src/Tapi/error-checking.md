# <a name="error-checking"></a>Verificação de erros

No nível de TAPI, um aplicativo pode passar vários parâmetros diferentes, muitos dos quais podem ser inválidos. A TAPI verifica os parâmetros e retorna os erros para o aplicativo sem chamar o provedor de serviços. Cada descrição de função no nível de TSPI descreve os erros de parâmetro já testados. O provedor de serviços não precisa repetir esses testes, embora ele deva executar quaisquer testes de validade adicionais apropriados para a função. Títulos e descrições de testes de validade de parâmetro comuns que aparecem em muitas funções são listados na tabela a seguir.



| Teste de validade       | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|---------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Validade do ponteiro    | A TAPI já testou ponteiros para o armazenamento de dados para garantir que eles apontem para a memória legível ou gravável do tamanho apropriado para a operação. Além disso, para estruturas de dados de tamanho variados começando com um membro **dwTotalSize** , a estrutura de dados foi verificada para garantir que o tamanho total indicado esteja disponível.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Validade de tamanho fixo | Para estruturas de dados de tamanho variável, a estrutura de dados foi verificada, então o espaço para a parte de tamanho fixo da estrutura de dados e esse **dwTotalSize** é suficiente para a parte fixa.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Deslocamento/tamanho zerado  | Para estruturas de dados de tamanho variável, o "... **Deslocamento**"e"... **Tamanho**"campos que correspondem a partes que os conjuntos de provedores de serviços foram predefinidos com valores zero antes de o provedor de serviços ser chamado.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Validade do identificador     | A TAPI garante que os identificadores de linha, telefone e chamada (dos tipos definidos [HDRVLINE](hdrvline.md), [HDRVPHONE](hdrvphone.md)e HDRVCALL) sejam válidos. Ou seja, são valores que foram retornados sem erro como identificadores em [**TSPI \_ lineOpen**](/windows/win32/api/tspi/nf-tspi-tspi_lineopen), [**TSPI \_ phoneOpen**](/windows/win32/api/tspi/nf-tspi-tspi_phoneopen)ou um dos seguintes que inicia o tempo de vida de um identificador de chamada: [**TSPI \_ lineMakeCall**](/windows/win32/api/tspi/nf-tspi-tspi_linemakecall)<br/> [**TSPI \_ lineCompleteTransfer**](/windows/win32/api/tspi/nf-tspi-tspi_linecompletetransfer)<br/> [**TSPI \_ lineForward**](/windows/win32/api/tspi/nf-tspi-tspi_lineforward)<br/> [**TSPI \_ linePickup**](/windows/win32/api/tspi/nf-tspi-tspi_linepickup)<br/> [**TSPI \_ linePrepareAddToConference**](/windows/win32/api/tspi/nf-tspi-tspi_lineprepareaddtoconference)<br/> [**TSPI \_ lineSetupConference**](/windows/win32/api/tspi/nf-tspi-tspi_linesetupconference)<br/> [**TSPI \_ lineSetupTransfer**](/windows/win32/api/tspi/nf-tspi-tspi_linesetuptransfer)<br/> [**TSPI \_ lineUnpark**](/windows/win32/api/tspi/nf-tspi-tspi_lineunpark)<br/> [**Linha \_ Mensagens NEWCALL**](line-newcall.md)<br/> |
