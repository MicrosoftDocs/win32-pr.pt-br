# <a name="callhub-object-interfaces"></a>Interfaces de objeto CallHub

O [objeto CallHub](callhub-object.md) expõe métodos que recuperam informações relacionadas a participantes em uma chamada de várias partes.

As interfaces [**ITCallHubEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) e [**IEnumCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub) não são diretamente expostas no objeto CallHub, mas estão intimamente relacionadas a ela e estão listadas aqui para conveniência de referência.



| Interface                                | Descrição                                                           |
|------------------------------------------|-----------------------------------------------------------------------|
| [**ITCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub)           | Fornece métodos para recuperar informações relacionadas a um objeto CallHub. |
| [**ITCallHubEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) | Usado para adquirir informações relacionadas a eventos CallHub.                |
| [**IEnumCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub)     | Enumera [**ITCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub).                            |
