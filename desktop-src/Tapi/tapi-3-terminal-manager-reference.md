# <a name="tapi-3-terminal-manager-reference"></a>Referência do Gerenciador de terminal TAPI 3



| Interfaces do Gerenciador de terminal                      | Descrição                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | Fornece métodos para conectar e desconectar um terminal de um grafo de filtro e executar ou parar um renderizador.                                                                                 |
| [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | Fornece métodos para criar terminais dinâmicos.                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | Fornece métodos para recuperar informações sobre as classes de terminal conectável registradas no sistema atual; deriva da interface [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) . |
