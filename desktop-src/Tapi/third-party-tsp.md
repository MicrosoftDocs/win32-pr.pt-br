# <a name="third-party-tsp"></a>TSP de terceiros

Hardware de comunicação especializado, como modems de cabo ou de telefonia, normalmente chegam com um ou mais TSPs e MSPs específicos para o dispositivo. A documentação do dispositivo deve indicar recursos e fornecer diretrizes de programação.

Se um TSP herdado não tiver um MSP, mas usar dispositivos Wave, a TAPI associará o MSP de onda a esse TSP. Os TSPs herdados que não usam dispositivos Wave não serão atribuídos a um MSP associado, e as operações de controle de fluxo não estarão disponíveis.

Um TSP de terceiros é instalado com o dispositivo para o qual ele pretende dar suporte.
