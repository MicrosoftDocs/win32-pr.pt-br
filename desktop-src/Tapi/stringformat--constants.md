# <a name="stringformat_-constants"></a>\_Constantes StringFormat

As **\_ constantes StringFormat** descrevem formatos de cadeia de caracteres diferentes.

<dl> <dt>

<span id="STRINGFORMAT_ASCII"></span><span id="stringformat_ascii"></span>**STRINGFORMAT \_ ASCII**
</dt> <dd> <dl> <dt>



Especifica o formato de caractere ASCII padrão usando um byte por caractere.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_BINARY"></span><span id="stringformat_binary"></span>**STRINGFORMAT \_ binário**
</dt> <dd> <dl> <dt>



Esta é uma matriz de caracteres não assinados; pode ser usado para valores numéricos.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_DBCS"></span><span id="stringformat_dbcs"></span>**STRINGFORMAT \_ DBCS**
</dt> <dd> <dl> <dt>



Especifica o formato de caractere DBCS padrão usando um ou dois bytes por caractere.


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_UNICODE"></span><span id="stringformat_unicode"></span>**STRINGFORMAT \_ Unicode**
</dt> <dd> <dl> <dt>



Especifica o formato de caractere Unicode padrão usando dois bytes por caractere.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Comentários

Sem extensibilidade. Todos os 32 bits são reservados.
