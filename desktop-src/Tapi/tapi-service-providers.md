# <a name="tapi-service-providers"></a>Provedores de serviços TAPI

Os provedores de serviços TAPI são uma camada de abstração entre aplicativos TAPI e os protocolos de hardware e transporte subjacentes. Um [provedor de serviços de telefonia](telephony-service-providers-start-page.md) (TSP) fornece controles de chamada e um [provedor de serviços de mídia](./media-service-providers-start-page.md) (MSP) fornece controle detalhado sobre a mídia de uma chamada. Um par de TSP/MSP pode implementar os recursos necessários para aplicativos que variam de uma conferência de multimídia básica para multicast de IP.
