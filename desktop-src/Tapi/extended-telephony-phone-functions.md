# <a name="extended-telephony-phone-functions"></a>Funções de telefone de telefonia estendidas

Os serviços de telefone estendidos (ou Device-Specific serviços de telefone) incluem todas as extensões para a TAPI definida pelo provedor de serviços. A TAPI define um mecanismo que permite aos fornecedores do provedor de serviços estender a TAPI usando extensões específicas do dispositivo. A TAPI define apenas o mecanismo de extensão e, ao fazer isso, fornece acesso a extensões específicas do dispositivo. A TAPI não define seu comportamento, que é completamente definido pelo provedor de serviços.

A TAPI consiste em definições de constante de dados escalares e de sinalizador de bits, estruturas de dados, funções e mensagens. Os procedimentos são definidos para permitir que um fornecedor estenda a maioria deles, conforme descrito nos tópicos a seguir.

-   [Constantes de dados escalares](scalar-data-constants.md)
-   [Constantes de dados de sinalizador de bits](bit-flag-data-constants.md)
-   [Funções e mensagens](functions-and-messages.md)
