# <a name="whats-new-telephony-api"></a>Novidades (API de Telefonia)

O TAPI 3.1 adiciona controles detalhados de dispositivo de telefone e várias interfaces de terminal especializadas. A tabela a seguir lista as novas interfaces.



| nbbnInterface                                                                                  | Descrição                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IEnumPhone**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumphone)                                                               | Fornece métodos de enumeração para a interface [**ITPhone.**](/windows/desktop/api/tapi3if/nn-tapi3if-itphone)                                                                                                                                       |
| [**IEnumPluggableSuperclassInfo**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumpluggablesuperclassinfo)                           | Fornece métodos de enumeração para a interface [**ITPluggableTerminalSuperclassInfo.**](/windows/desktop/api/tapi3if/nn-tapi3if-itpluggableterminalsuperclassinfo)                                                                                   |
| [**IEnumPluggableTerminalClassInfo**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumpluggableterminalclassinfo)                     | Fornece métodos de enumeração para a interface [**ITPluggableTerminalClassInfo.**](/windows/desktop/api/tapi3if/nn-tapi3if-itpluggableterminalclassinfo)                                                                                             |
| [**ITASRTerminalEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itasrterminalevent)                                               | Recupera a descrição dos eventos de terminal de Reconhecimento Automático de Fala.                                                                                                                                       |
| [**ITAddress2**](/windows/desktop/api/tapi3if/nn-tapi3if-itaddress2)                                                               | Fornece métodos adicionais no objeto Address que dá suporte a dispositivos de telefone; deriva da interface [**ITAddress.**](/windows/desktop/api/tapi3if/nn-tapi3if-itaddress)                                                                         |
| [**ITAutomatedPhoneControl**](/windows/desktop/api/tapi3if/nn-tapi3if-itautomatedphonecontrol)                                     | Executa várias funções relacionadas ao telefone de alto nível, incluindo a habilitação e a configuração do controle automatizado dos toques e anéis de um telefone e a manipulação de chamadas automatizadas com base no estado de gancho de um telefone.          |
| [**ITBasicCallControl2**](/windows/desktop/api/tapi3if/nn-tapi3if-itbasiccallcontrol2)                                             | Estende a interface [**ITBasicCallControl**](/windows/desktop/api/tapi3if/nn-tapi3if-itbasiccallcontrol) fornecendo métodos para selecionar um terminal em uma chamada.                                                                                    |
| [**ITCallInfo2**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallinfo2)                                                             | Estende a interface [**ITCallInfo**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallinfo) fornecendo métodos para definir a filtragem de eventos por chamada.                                                                                          |
| [**ITCollection2**](/windows/desktop/api/Tapi3if/nn-tapi3if-itcollection2)                                                         | Estende a interface [**ITCollection**](/windows/desktop/api/tapi3if/nn-tapi3if-itcollection) fornecendo métodos adicionais para modificar a coleção.                                                                                         |
| [**ITCustomTone**](/windows/desktop/api/Tapi3if/nn-tapi3if-itcustomtone)                                                           | Fornece métodos que permitem controle detalhado sobre os tons personalizados possíveis com alguns conjuntos de telefone.                                                                                                                |
| [**ITDetectTone**](/windows/desktop/api/Tapi3if/nn-tapi3if-itdetecttone)                                                           | Fornece métodos que permitem que um aplicativo especifique ao Servidor TAPI os tom e as características de tom que causam o servidor a gerar um evento de tom.                                                      |
| [**ITDigitsGatheredEvent**](/windows/desktop/api/Tapi3if/nn-tapi3if-itdigitsgatheredevent)                                         | Fornece métodos para recuperar dados relacionados à solicitação de coleta de dígitos de um aplicativo.                                                                                                                           |
| [**ITFileTerminalEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itfileterminalevent)                                             | Recupera a descrição dos eventos do terminal de arquivo.                                                                                                                                                               |

| [**ITLegacyCallMediaControl2**](/windows/desktop/api/Tapi3if/nn-tapi3if-itlegacycallmediacontrol2)                                 | Estende a interface [**ITLegacyCallMediaControl**](/windows/desktop/api/tapi3if/nn-tapi3if-itlegacycallmediacontrol) fornecendo métodos adicionais para detecção e geração de tom.                                                            |
| [**ITMediaControl**](/windows/desktop/api/tapi3if/nn-tapi3if-itmediacontrol)                                                       | Inicia, para e pausa ações atuais, como uma reprodução.                                                                                                                                                   |
| [**ITMediaPlayback**](/windows/desktop/api/tapi3if/nn-tapi3if-itmediaplayback)                                                     | Fornece métodos específicos de reprodução que permitem que um aplicativo de definir e obter a lista de arquivos a reproduzir.                                                                                                           |
| [**ITMediaRecord**](/windows/desktop/api/tapi3if/nn-tapi3if-itmediarecord)                                                         | Fornece métodos específicos de gravação que permitem que um aplicativo de definir e obter os nomes dos arquivos a gravar.                                                                                                       |
| [**ITMultiTrackTerminal**](/windows/desktop/api/tapi3if/nn-tapi3if-itmultitrackterminal)                                           | Enumera, cria ou remove faixas em terminais multitrack.                                                                                                                                                  |
| [**ITPhone**](/windows/desktop/api/tapi3if/nn-tapi3if-itphone)                                                                     | Permite o acesso ao dispositivo de telefone em um nível comparável ao disponível com a TAPI 2. *API x* C.                                                                                                             |
| [**ITPhoneEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itphoneevent)                                                           | Recupera a descrição dos eventos de telefone.                                                                                                                                                                       |
| [**ITPluggableTerminalClassInfo**](/windows/desktop/api/tapi3if/nn-tapi3if-itpluggableterminalclassinfo)                           | Recupera informações sobre um terminal plug-able.                                                                                                                                                           |
| [**ITPluggableTerminalClassRegistration**](/windows/desktop/api/Termmgr/nn-termmgr-itpluggableterminalclassregistration)           | Cria, modifica ou exclui a entrada do Registro para um terminal que pode ser conectado.                                                                                                                                       |

| [**ITPluggableTerminalSuperclassRegistration**](/windows/desktop/api/Termmgr/nn-termmgr-itpluggableterminalsuperclassregistration) | Recupera e define informações sobre uma superclasse de terminal (nome e CLSID).                                                                                                                                     |
| [**ITScriptableAudioFormat**](/windows/desktop/api/tapi3if/nn-tapi3if-itscriptableaudioformat)                                     | Recupera o formato de áudio de ou define o formato de áudio para uma faixa.                                                                                                                                          |

Os MSPs tapi 3.1 devem expor essa interface em todos os terminais de áudio estáticos.                                              |
| [**ITTAPI2**](/windows/desktop/api/tapi3if/nn-tapi3if-ittapi2)                                                                     | Fornece métodos adicionais no objeto TAPI para dar suporte a dispositivos de telefone; deriva da interface [**ITTAPI.**](/windows/desktop/api/tapi3if/nn-tapi3if-ittapi)                                                                                    |
| [**ITTAPIObjectEvent2**](/windows/desktop/api/tapi3if/nn-tapi3if-ittapiobjectevent2)                                               | Estende a interface [**ITTAPIObjectEvent;**](/windows/desktop/api/tapi3if/nn-tapi3if-ittapiobjectevent) fornece um método que retorna um ponteiro para uma interface [**ITPhone**](/windows/desktop/api/tapi3if/nn-tapi3if-itphone) no objeto de telefone que causou o evento de objeto TAPI. |

| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2)                                               | Recupera informações sobre classes de terminal que podem ser plugáveis registradas no sistema atual; deriva da interface [**ITTerminalManager.**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)                                              |
| [**ITTerminalSupport2**](/windows/desktop/api/tapi3if/nn-tapi3if-itterminalsupport2)                                               | Recupera informações sobre classes de terminal e superclasses que podem ser plugáveis; deriva da interface [**ITTerminalSupport.**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport)                                                              |
