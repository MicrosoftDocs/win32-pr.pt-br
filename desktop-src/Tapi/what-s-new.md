# <a name="whats-new-telephony-api"></a>O que há de novo (API de telefonia)

A TAPI 3,1 adiciona controles de dispositivo de telefone detalhados e uma série de interfaces de terminal especializadas. A tabela a seguir lista as novas interfaces.



| nbbnInterface                                                                                  | Descrição                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IEnumPhone**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumphone)                                                               | Fornece métodos de enumeração para a interface [**ITPhone**](/windows/desktop/api/tapi3if/nn-tapi3if-itphone) .                                                                                                                                       |
| [**IEnumPluggableSuperclassInfo**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumpluggablesuperclassinfo)                           | Fornece métodos de enumeração para a interface [**ITPluggableTerminalSuperclassInfo**](/windows/desktop/api/tapi3if/nn-tapi3if-itpluggableterminalsuperclassinfo) .                                                                                   |

| [**ITASRTerminalEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itasrterminalevent)                                               | Recupera a descrição de eventos de terminal de reconhecimento automático de fala.                                                                                                                                       |
| [**ITAddress2**](/windows/desktop/api/tapi3if/nn-tapi3if-itaddress2)                                                               | Fornece métodos adicionais no objeto de endereço que dão suporte a dispositivos de telefone; deriva da interface [**ITAddress**](/windows/desktop/api/tapi3if/nn-tapi3if-itaddress) .                                                                         |

| [**ITDetectTone**](/windows/desktop/api/Tapi3if/nn-tapi3if-itdetecttone)                                                           | Fornece métodos que permitem que um aplicativo especifique ao servidor TAPI as características de tons e tons que fazem com que o servidor gere um evento de Tom.                                                      |
| [**ITDigitsGatheredEvent**](/windows/desktop/api/Tapi3if/nn-tapi3if-itdigitsgatheredevent)                                         | Fornece métodos para recuperar dados relacionados à solicitação de coleta de dígitos de um aplicativo.                                                                                                                           |

| [**ITLegacyCallMediaControl2**](/windows/desktop/api/Tapi3if/nn-tapi3if-itlegacycallmediacontrol2)                                 | Estende a interface [**ITLegacyCallMediaControl**](/windows/desktop/api/tapi3if/nn-tapi3if-itlegacycallmediacontrol) fornecendo métodos adicionais para detecção e geração de Tom.                                                            |

| [**ITMediaPlayback**](/windows/desktop/api/tapi3if/nn-tapi3if-itmediaplayback)                                                     | Fornece métodos específicos de reprodução que permitem que um aplicativo defina e obtenha a lista de arquivos a serem executados.                                                                                                           |
| [**ITMediaRecord**](/windows/desktop/api/tapi3if/nn-tapi3if-itmediarecord)                                                         | Fornece métodos específicos de gravação que permitem que um aplicativo defina e obtenha os nomes dos arquivos a serem registrados.                                                                                                       |

API *x* C.                                                                                                             |
