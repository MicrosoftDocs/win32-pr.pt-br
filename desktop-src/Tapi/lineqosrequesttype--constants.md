# <a name="lineqosrequesttype_-constants"></a>\_Constantes LINEQOSREQUESTTYPE

Essas constantes são usadas por um TSP para identificar o tipo de uma solicitação de QoS (qualidade de serviço) que está sendo feita. Atualmente, há suporte apenas para uma solicitação de nível de serviço. Consulte [ \_ constantes LINEQOSSERVICELEVEL](lineqosservicelevel--constants.md).

<span id="LINEQOSREQUESTTYPE_SERVICELEVEL"></span><span id="lineqosrequesttype_servicelevel"></span>**LINEQOSREQUESTTYPE de \_ nível**
</dt> <dd> <dl> <dt>

 0x00000001
</dt> <dt>
