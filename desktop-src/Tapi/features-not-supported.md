# <a name="features-not-supported"></a>Recursos sem suporte

-   A manipulação dinâmica de grafo será implementada pelo MSP derivado, se optar por fazê-lo.
-   Os subfluxos serão implementados pelo MSP derivado, se optar por fazê-lo.
-   Interfaces privadas, como participantes, controle de codec e assim por diante, não serão implementadas pelas classes base.
-   A comunicação do TSP-MSP será implementada pelo MSP derivado.
