# <a name="t-telephony-api"></a>T (API de telefonia)

<dl> <dt>

<span id="tapi2.t1_e1_tapgloss"></span><span id="TAPI2.T1_E1_TAPGLOSS"></span>**T1/E1**
</dt> <dd>

Um link de transmissão digital com capacidade de 1.544 Mbps. Nos EUA, esse link é chamado *T1*. Na União Europeia, ele é chamado de *E1*. Consulte [*megabits por segundo (Mbps)*](m-tapgloss.md).

</dd> <dt>

<span id="tapi2.tapi_tapgloss"></span><span id="TAPI2.TAPI_TAPGLOSS"></span>**TAPI**
</dt> <dd>

Consulte *TAPI (interface de programação de aplicativo de telefonia)*.

</dd> <dt>

<span id="tapi2.tapisrv_tapgloss"></span><span id="TAPI2.TAPISRV_TAPGLOSS"></span>**tapisrv**
</dt> <dd>

Um aplicativo de serviço de suporte que fornece um contexto de execução para a biblioteca de vínculo dinâmico TAPI para realizar tarefas como alocar memória ou acessar estruturas de dados que foram trocadas no disco.

</dd> <dt>

<span id="tapi2.tdd_tapgloss"></span><span id="TAPI2.TDD_TAPGLOSS"></span>**TDD**
</dt> <dd>

Consulte *dispositivos de telecomunicações para os surdos (TDD)*.

</dd> <dt>

<span id="tapi2.telecommucations_devices_for_the_deaf_tdd__tapgloss"></span><span id="TAPI2.TELECOMMUCATIONS_DEVICES_FOR_THE_DEAF_TDD__TAPGLOSS"></span>**Dispositivos Telecommucations para os surdos (TDD)**
</dt> <dd>

Um dispositivo de dados para a transmissão de sinais codificados. Normalmente, o dispositivo é um terminal de computador simples que consiste em um teclado, um acoplador acústico e uma exibição e é executado a 300 bauds.

</dd> <dt>

<span id="tapi2.telephony_tapgloss"></span><span id="TAPI2.TELEPHONY_TAPGLOSS"></span>**telefonia**
</dt> <dd>

A ciência de transmitir sinais de voz, de dados, de vídeo ou de imagem ao longo de uma distância usando a tecnologia que integra computadores à rede de telefone.

</dd> <dt>

<span id="tapi2.telephony_application_programming_interface_tapi__tapgloss"></span><span id="TAPI2.TELEPHONY_APPLICATION_PROGRAMMING_INTERFACE_TAPI__TAPGLOSS"></span>**TAPI (interface de programação de aplicativo de telefonia)**
</dt> <dd>

Conjunto de funções que permite a programação de dispositivos baseados em linha de telefone de maneira independente de dispositivo, fornecendo telefonia pessoal aos usuários. A TAPI dá suporte à transmissão de dados e de fala, permite uma variedade de dispositivos de terminal e dá suporte a tipos de conexão complexos e técnicas de gerenciamento de chamadas, como chamadas de conferência, espera de chamada e correio de voz. A TAPI permite todos os elementos de uso por telefone — da simples chamada de discagem e fala para a e-mail internacional — para ser controlada em aplicativos desenvolvidos para o Microsoft Windows. Consulte a [visão geral de telefonia da Microsoft](./microsoft-telephony-overview.md)

</dd> <dt>

<span id="tapi2.telephony_service_provider_interface_tspi__tapgloss"></span><span id="TAPI2.TELEPHONY_SERVICE_PROVIDER_INTERFACE_TSPI__TAPGLOSS"></span>**Interface do provedor de serviços de telefonia (TSPI)**
</dt> <dd>

Ferramenta para criar provedores de serviços para sistemas operacionais Microsoft Windows. TSPI define como a rede compartilha informações com a telefonia do Windows, que, por sua vez, conversa com a API, que se comunica com os aplicativos de telefonia do Windows. Para obter informações adicionais, consulte [TSPI (interface do provedor de serviços de telefonia)](./telephony-service-provider-interface-tspi-.md).

</dd> <dt>

<span id="tapi2.third_party_call_control_tapgloss"></span><span id="TAPI2.THIRD_PARTY_CALL_CONTROL_TAPGLOSS"></span>**controle de chamada de terceiros**
</dt> <dd>

Um modelo de controle de chamada de terceiros que permite que um aplicativo estabeleça ou responda a uma chamada entre duas partes.

</dd> <dt>

<span id="tapi2.time_to_live_ttl__tapgloss"></span><span id="TAPI2.TIME_TO_LIVE_TTL__TAPGLOSS"></span>**vida útil (TTL)**
</dt> <dd>

Um valor no intervalo de 0 a 255 que define o escopo no qual os pacotes multicast devem ser enviados por uma rede usando o protocolo IP. O escopo é definido em termos de como o destino de um pacote é local ou remoto. Cada roteador Decrementa o TTL por um. Quando o valor atinge um limite inferior predefinido, o roteador lança o pacote fora. Requisitos de MBONE (backbone multicast) atuais, disponíveis no site ftp://ftp.isi.edu/mbone/faq.txt, defina os seguintes escopos padrão: rede local, 1; site local, 15; região, 63; mundo, 127. Outras configurações podem ter significado local; por exemplo, 31 pode indicar todos os sites em uma determinada organização.

</dd> <dt>

<span id="tapi2.tspi_tapgloss"></span><span id="TAPI2.TSPI_TAPGLOSS"></span>**TSPI**
</dt> <dd>

Consulte [interface do provedor de serviços de telefonia (TSPI)](./telephony-service-provider-interface-tspi-.md).

</dd> <dt>

<span id="tapi2.ttl_tapgloss"></span><span id="TAPI2.TTL_TAPGLOSS"></span>**IGUAL**
</dt> <dd>

Consulte *TTL (vida útil)*.

</dd> <dt>

<span id="tapi2.twisted_pair_tapgloss"></span><span id="TAPI2.TWISTED_PAIR_TAPGLOSS"></span>**par trançado**
</dt> <dd>

Fiação de telefone normal. Cada fio de telefone é, na verdade, um par de fios.

</dd> </dl>
