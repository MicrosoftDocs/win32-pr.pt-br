# <a name="htapicall"></a>HTAPICALL

O **tipo de dados HTAPICALL** representa o handle opaco TAPI para uma estrutura de dados de chamada. A TAPI deve resolver um valor desse tipo em uma referência à instância de estrutura de dados apropriada. O provedor de serviços não deve tentar fazer referência a isso como se fosse um ponteiro, fazer suposições sobre seus valores ou interpretar sua representação de forma diferente de passar seu valor para TAPI em momentos apropriados.
