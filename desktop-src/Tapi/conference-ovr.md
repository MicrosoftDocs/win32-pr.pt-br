<span data-ttu-id="01f85-107">As sessões de conferência são sessões que incluem mais de duas partes simultaneamente.</span><span class="sxs-lookup"><span data-stu-id="01f85-107">Conference sessions are sessions that include more than two parties simultaneously.</span></span> <span data-ttu-id="01f85-108">Eles podem ser configurados usando uma ponte baseada no servidor externo ou uma ponte de conferência baseada em comutador.</span><span class="sxs-lookup"><span data-stu-id="01f85-108">They can be set up using either an external server-based bridge or a switch-based conference bridge.</span></span>

<span data-ttu-id="01f85-109">Em sessões de conferência baseadas em servidor, todas as partes participantes se conectam ao servidor, que combina os fluxos de mídia juntos e envia a cada participante a combinação.</span><span class="sxs-lookup"><span data-stu-id="01f85-109">In server-based conference sessions, all participating parties dial into the server, which mixes the media streams together and sends each participant the mix.</span></span> <span data-ttu-id="01f85-110">Pode não haver nenhuma noção de partes individuais na chamada à conferência, somente a partir de uma única chamada entre o aplicativo e o servidor de ponte.</span><span class="sxs-lookup"><span data-stu-id="01f85-110">There may be no notion of individual parties in the conference call, only that of a single call between the application and the bridge server.</span></span> <span data-ttu-id="01f85-111">Para a TAPI, esse tipo de chamada de conferência parece ser uma conexão um-para-um normal.</span><span class="sxs-lookup"><span data-stu-id="01f85-111">To TAPI, this type of conference call appears to be a normal one-to-one connection.</span></span>

<span data-ttu-id="01f85-123">Ao cancelar a sessão de consulta para a terceira parte de uma conferência ou ao remover a terceira parte de uma conferência estabelecida anteriormente, o provedor de serviços pode liberar a conferência e reverter a sessão para uma conexão normal de duas partes.</span><span class="sxs-lookup"><span data-stu-id="01f85-123">When canceling the consultation session to the third party for a conference or when removing the third party in a previously established conference, the service provider may release the conference and revert the session back to a normal two-party connection.</span></span> <span data-ttu-id="01f85-124">Se esse for o caso, a sessão de conferência fará a transição para o estado *ocioso* e a única sessão participante restante passará da conferência para o estado *conectado* .</span><span class="sxs-lookup"><span data-stu-id="01f85-124">If this is the case, the conference session will transition to the *idle* state, and the only remaining participating session will transition from the conferenced to the *connected* state.</span></span>

<span data-ttu-id="01f85-125">Nem todos os provedores de serviço dão suporte à conferência.</span><span class="sxs-lookup"><span data-stu-id="01f85-125">Not all service providers support conferencing.</span></span>

<span data-ttu-id="01f85-130">Os membros da conferência são removidos usando a função [**lineRemoveFromConference**](/windows/win32/api/tapi/nf-tapi-lineremovefromconference) , se o provedor de serviços oferecer suporte a ele.</span><span class="sxs-lookup"><span data-stu-id="01f85-130">Conference members are removed using the [**lineRemoveFromConference**](/windows/win32/api/tapi/nf-tapi-lineremovefromconference) function, if the service provider supports it.</span></span>

<span data-ttu-id="01f85-132">**TAPI 3. x:** O método [**ITBasicCallControl:: Conference**](/windows/desktop/api/tapi3if/nf-tapi3if-itbasiccallcontrol-conference) usa a sessão existente como entrada e cria um [objeto CallHub](callhub-object.md) , caso ainda não exista uma.</span><span class="sxs-lookup"><span data-stu-id="01f85-132">**TAPI 3.x:** The [**ITBasicCallControl::Conference**](/windows/desktop/api/tapi3if/nf-tapi3if-itbasiccallcontrol-conference) method takes the existing session as input and creates a [CallHub object](callhub-object.md) if one does not already exist.</span></span> <span data-ttu-id="01f85-133">O método [**ITBasicCallControl:: Finish**](/windows/desktop/api/tapi3if/nf-tapi3if-itbasiccallcontrol-finish) adiciona a chamada de consulta ao CallHub.</span><span class="sxs-lookup"><span data-stu-id="01f85-133">The [**ITBasicCallControl::Finish**](/windows/desktop/api/tapi3if/nf-tapi3if-itbasiccallcontrol-finish) method adds the consultation call to the CallHub.</span></span> <span data-ttu-id="01f85-134">Outras sessões de consulta podem ser criadas usando [**ITAddress:: createcall**](/windows/desktop/api/tapi3if/nf-tapi3if-itaddress-createcall)e adicionadas usando os métodos **Conference** e **Finish** .</span><span class="sxs-lookup"><span data-stu-id="01f85-134">Additional consultation sessions may be created using [**ITAddress::CreateCall**](/windows/desktop/api/tapi3if/nf-tapi3if-itaddress-createcall), and added using the **Conference** and **Finish** methods.</span></span>
