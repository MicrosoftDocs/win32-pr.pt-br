# <a name="wave-msp"></a>MSP do Wave

O Wave MSP (Wavemsp.dll) se enquadra em TSPs herdados escritos antes da introdução do MSPs à arquitetura de streaming baseada em MSP de 3 da TAPI. Alguns dos TSPs herdados expõem identificadores de dispositivo wave, que eram um meio para o aplicativo controlar o streaming no TAPI 2,1. Tapi3.dll usa o MSP de onda para qualquer TSP que não tenha seu próprio MSP, mas que tenha um dispositivo wave.

O Wave MSP é instalado com o Windows 2000.
