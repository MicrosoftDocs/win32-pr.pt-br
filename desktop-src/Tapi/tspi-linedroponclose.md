# <a name="tspi_linedroponclose"></a>TSPI \_ lineDropOnClose

A função **TSPI \_ lineDropOnClose** está obsoleta. A TAPI não descarta automaticamente uma chamada quando o único proprietário fecha seu identificador para o dispositivo de linha, a fim de acomodar o controle distribuído de chamadas em vários computadores cliente em um ambiente cliente/servidor.
