# <a name="basic-phone-services"></a>Serviços de telefone básicos

Os serviços de telefone básicos são o conjunto principal de serviços com suporte em todos os dispositivos de telefone. Como os serviços de telefone básicos são definidos pelo provedor de serviços, há muito variedade no conjunto principal.
