# <a name="ndis-proxy-tsp"></a>TSP do proxy NDIS

O proxy NDIS é um componente do modo kernel que expõe as miniportas NDIS como linhas TAPI. O proxy NDIS do TSP (Ndptsp. TSP) permite que a TAPI se comunique com o proxy NDIS.

O proxy NDIS do TSP é instalado com sistemas operacionais Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 e Windows 95.
