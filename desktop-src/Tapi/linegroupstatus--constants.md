# <a name="linegroupstatus_-constants"></a>Constantes LINEGROUPSTATUS \_

As **constantes LINEGROUPSTATUS \_ especificam** o status do grupo ACD.

</dt> <dd> <dl> <dt>

</dt> </dl> </dd> <dt>

<span id="LINEGROUPSTATUS_NEWGROUP"></span><span id="linegroupstatus_newgroup"></span>**LINEGROUPSTATUS \_ NEWGROUP**
</dt> <dd> <dl> <dt>

</dt> </dl> </dd> </dl>
