# <a name="itsubstreamcontrol-mspi"></a>ITSubStreamControl (MSPI)

A interface [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) será exposta no objeto de fluxo se houver um MSP associado e ele implementará subfluxos. Os métodos dessa interface permitem que um aplicativo crie e remova subfluxos.

A interface [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) é implementada por um msp. Consulte **ITSubStreamControl** na seção de interface msp para obter detalhes sobre essa interface.
