# <a name="itsubstreamcontrol-mspi"></a>ITSubStreamControl (MSPI)

A interface [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) será exposta no Objeto stream se houver um MSP associado e implementar substreams. Os métodos dessa interface permitem que um aplicativo crie e remova substreams.

A interface [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) é implementada por um MSP. Consulte **ITSubStreamControl na** seção Interface MSP para obter detalhes sobre essa interface.
