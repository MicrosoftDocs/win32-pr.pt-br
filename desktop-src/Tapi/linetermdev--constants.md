# <a name="linetermdev_-constants"></a>\_Constantes LINETERMDEV

As constantes de sinalizador de bit **LINETERMDEV \_** descrevem diferentes tipos de dispositivos de terminal.

<dl> <dt>

<span id="LINETERMDEV_HEADSET"></span><span id="linetermdev_headset"></span>**Headset de LINETERMDEV \_**
</dt> <dd> <dl> <dt>



O terminal é um headset.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_PHONE"></span><span id="linetermdev_phone"></span>**\_telefone LINETERMDEV**
</dt> <dd> <dl> <dt>



O terminal é um conjunto de telefones.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_SPEAKER"></span><span id="linetermdev_speaker"></span>**LINETERMDEV \_ palestrante**
</dt> <dd> <dl> <dt>



O terminal é um palestrante e um microfone externos.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Comentários

Sem extensibilidade. Todos os 32 bits são reservados.

Essas constantes são usadas para caracterizar o dispositivo de terminal de uma linha e ajudar um aplicativo a determinar a natureza de um dispositivo de terminal.
