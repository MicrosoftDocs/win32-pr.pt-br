# <a name="open-issues"></a>Problemas Abertos

-   Antes da versão final do Windows 2000, o caminho de desligamento de chamada pode ser ligeiramente alterado para facilitar o desligamento de uma chamada de uma operação assíncrona (ajuda a se proteger contra deadlocks para MSPs que dão suporte a vídeo).
-   Os métodos e membros da classe de thread de trabalho, as classes base de terminal e os terminais estáticos derivados são documentados apenas pelos próprios arquivos de cabeçalho.
