# <a name="raw-channel-access-msp"></a>Acesso ao canal bruto MSP

O MSP (Rcsmsp.dll) de acesso ao canal bruto faz o streaming via ksproxy para RCA, que permite o streaming em dispositivos NDIS (especificação de interface de driver de rede) orientados por conexão, como placas ATM e ISDN. O RCA MSP é criado pelo ndptsp, que é o TSP fornecido para adaptadores WAN NDIS. Com a versão 2000 do Windows, somente o streaming de áudio é implementado.

O acesso ao canal bruto do MSP é instalado com o Windows 2000.
