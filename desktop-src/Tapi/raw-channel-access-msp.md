# <a name="raw-channel-access-msp"></a>MSP de Acesso a Canal Bruto

O MSP (Acesso ao Canal Bruto) (Rcsmsp.dll) faz streaming via ksproxy para RCA, o que permite o streaming por dispositivos NDIS (Network Driver Interface Specification) orientados a conexão, como cartões ATM e ISDN. O RCA MSP é criado pelo ndptsp, que é o TSP fornecido para adaptadores NDIS WAN. Com a versão Windows 2000, somente o streaming de áudio é implementado.

O MSP de Acesso a Canal Bruto é instalado com Windows 2000.
