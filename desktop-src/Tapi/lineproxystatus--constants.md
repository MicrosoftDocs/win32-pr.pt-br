# <a name="lineproxystatus_-constants"></a>\_Constantes LINEPROXYSTATUS

As **\_ constantes LINEPROXYSTATUS** indicam o status do proxy em uma linha que o aplicativo abriu atualmente.

Consulte [**\_ constantes LINEPROXYREQUEST**](lineproxyrequest--constants.md) para obter uma lista e uma descrição de todos os possíveis valores de solicitação de proxy.

<dl> <dt>

<span id="LINEPROXYSTATUS_ALLOPENFORACD"></span><span id="lineproxystatus_allopenforacd"></span>**LINEPROXYSTATUS \_ ALLOPENFORACD**
</dt> <dd> <dl> <dt>



A linha agora tem proxies abertos para todos os tipos de solicitação de proxy necessários para operações de AD pela TAPI versão 3,0 e superior.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_CLOSE"></span><span id="lineproxystatus_close"></span>**LINEPROXYSTATUS \_ fechar**
</dt> <dd> <dl> <dt>



Um proxy foi fechado.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_OPEN"></span><span id="lineproxystatus_open"></span>**LINEPROXYSTATUS \_ abrir**
</dt> <dd> <dl> <dt>



Um novo proxy foi aberto.


</dt> </dl> </dd> </dl>
