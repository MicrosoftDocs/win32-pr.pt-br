# <a name="extending-data-structures-and-types"></a>Estendendo estruturas e tipos de dados

Um intervalo de valores é reservado para acomodar extensões futuras para o conjunto de funções TAPI básico e suplementar. Para obter informações sobre a quantidade pela qual uma estrutura de dados pode ser estendida, consulte [extensibilidade](extensibility.md). Para obter uma lista das funções usadas para estender a telefonia, consulte [referência de função rápida TAPI](./tapi-quick-function-reference.md).
