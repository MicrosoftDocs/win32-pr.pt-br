# <a name="hdrvcall"></a>Hdrvcall

O [**tipo de dados HDRVCALL**](hdrvline.md) representa o handle opaco de um provedor de serviços para uma estrutura de dados de chamada. O provedor de serviços deve resolver um valor desse tipo em uma referência à instância de estrutura de dados apropriada.
