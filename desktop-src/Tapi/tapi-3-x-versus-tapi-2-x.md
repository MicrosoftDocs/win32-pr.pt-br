# <a name="tapi-3x-vs-tapi-2x"></a>TAPI 3. x vs. TAPI 2. x

A API de telefonia baseada em COM, TAPI 3. x, está disponível a partir do Windows XP. A TAPI 3. x fornece ferramentas de desenvolvimento muito aprimoradas para o mundo moderno da programação de comunicação, em que uma "chamada" pode ser um fluxo de vídeo em uma rede baseada em IP, e não há nenhum conjunto de telefone envolvido na sessão. Em termos gerais, os criadores de aplicativos encontrarão a TAPI 3. x fornece um melhor suporte para implementar controles de comunicação flexíveis.

No entanto, a TAPI 2. x pode ser a escolha certa em algumas situações. Se forem necessárias modificações secundárias para um aplicativo herdado ou se houver plataformas herdadas para dar suporte, a conversão para a API COM pode não ser econômica.
