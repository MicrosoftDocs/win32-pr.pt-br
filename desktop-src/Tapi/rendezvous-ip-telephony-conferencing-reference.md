# <a name="rendezvous-ip-telephony-conferencing-reference"></a>Referência de conferência de telefonia IP de reunião

\[ Os controles e as interfaces da conferência de telefonia IP de reunião não estão disponíveis para uso no Windows Vista, no Windows Server 2008 e nas versões subsequentes do sistema operacional. A API do cliente RTC fornece funcionalidade semelhante.\]

A API de reunião da TAPI 3 é um grupo de interfaces COM (Component Object Model) que pode ser usado para anunciar e descobrir conferências de multimídia COM multipartes. O material a seguir fornece detalhes sobre os objetos, as interfaces e os métodos fornecidos.

A TAPI 3 usa filtros do DirectShow e gerenciadores de filtro para controlar o fluxo de mídia após a configuração de conexão. O COM é o modelo de codificação básico para a TAPI 3, e familiaridade com ele é assumida em todo este documento.
