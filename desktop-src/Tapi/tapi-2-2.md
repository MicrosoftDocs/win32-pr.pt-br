# <a name="tapi-22"></a>TAPI 2,2

A TAPI 2,2 fornece funcionalidade especializada de gerenciamento de Call Center. Em TAPI 2,1 e anterior, o software de Call Center exigia um mapeamento cuidadoso de linhas e endereços para a função de central de chamadas. As funções TAPI 2,1 são coordenadas com recursos de Call Center TAPI 3. x e tornam a criação de sistemas ad muito mais fácil.

**TAPI 2. x:** Para obter mais informações, consulte [controle do Call Center](call-center-control.md).

**TAPI 3. x:** Para obter mais informações, consulte [controles do Call Center](./about-call-center-controls.md).
