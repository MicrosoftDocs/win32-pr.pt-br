# <a name="tapi-quick-function-reference"></a>Referência de função rápida TAPI

O seguinte Resumo de hiperlinks das funções TAPI é organizado por categoria de operação. Para obter informações adicionais, consulte as visões gerais de [telefonia assistida](./assisted-telephony-overview.md) e [níveis de serviço](./tapi-levels-of-service.md).



| Seção de referência                                                                            | Descrição                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Referência de serviços de telefonia básicos](basic-telephony-services-reference.md)                 | Funções associadas a tarefas de telefonia de nível POTS, como [**lineMakeCall**](/windows/desktop/api/Tapi/nf-tapi-linemakecall). |
| [Referência de serviços de telefonia suplementares](supplementary-telephony-services-reference.md) | Funções com recursos mais avançados, como redirecionamento de chamada.                                       |
| [Referência de serviços de telefonia estendida](extended-telephony-services-reference.md)           | Funções que habilitam o acesso a funções específicas do provedor de serviço não definidas diretamente pela TAPI.   |
| [Referência de serviços de telefonia assistida](assisted-telephony-services-reference.md)           | Funções que estabelecem chamadas de voz e chamadas de mídia, como o uso de uma discagem telefônica.              |
