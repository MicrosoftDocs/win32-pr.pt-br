# <a name="extended-phone-services"></a>Serviços Telefone estendidos

Os Serviços Telefone Estendidos (ou serviços de telefone específicos do dispositivo) incluem todas as extensões para tSPI definidas pelo provedor de serviços. O TSPI define um mecanismo que permite que os fornecedores do provedor de serviços estendam o TSPI usando extensões específicas do dispositivo. O TSPI define apenas o mecanismo de extensão e, ao fazer isso, fornece acesso a extensões específicas do dispositivo, cujo comportamento é completamente definido pelo provedor de serviços.

O TSPI consiste em definições escalares e de constante de dados de sinalizador de bit, estruturas de dados, funções e mensagens de retorno de chamada. São definidos procedimentos que permitem que um fornecedor estenda a maioria deles, conforme descrito nos tópicos nesta seção.
