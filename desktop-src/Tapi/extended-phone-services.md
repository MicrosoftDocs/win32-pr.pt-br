# <a name="extended-phone-services"></a>serviços de Telefone estendidos

os serviços de Telefone estendidos (ou serviços de telefone específicos do dispositivo) incluem todas as extensões para TSPI definidas pelo provedor de serviços. TSPI define um mecanismo que permite aos fornecedores do provedor de serviços estender o TSPI usando extensões específicas do dispositivo. TSPI define apenas o mecanismo de extensão e, ao fazer isso, fornece acesso a extensões específicas do dispositivo, cujo comportamento é completamente definido pelo provedor de serviços.

O TSPI consiste em definições constantes escalares e de bits de dados, estruturas de dados, funções e mensagens de retorno de chamada. Os procedimentos são definidos para permitir que um fornecedor estenda a maioria deles, conforme descrito nos tópicos desta seção.
