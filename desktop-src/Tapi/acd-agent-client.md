# <a name="acd-agent-client"></a>Cliente do agente do AD

Um agente do Call Center é responsável por receber e processar chamadas recebidas e, às vezes, fazer chamadas de saída para clientes ou clientes potenciais. Um aplicativo cliente do agente ad fornece comunicações com o aplicativo de [proxy ad](acd-proxy.md) , fornecendo a ele informações como se o agente está disponível para chamadas e quais tipos de chamadas o agente pode manipular. O diagrama a seguir ilustra como os objetos TAPI parecem um agente.
