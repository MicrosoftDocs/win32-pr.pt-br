# <a name="linerequestmode_-constants"></a>\_Constantes LINEREQUESTMODE

As constantes de sinalizador de bit **LINEREQUESTMODE** descrevem diferentes tipos de solicitações de telefonia que podem ser feitas de um aplicativo para outro.

<dl> <dt>

<span id="LINEREQUESTMODE_DROP"></span><span id="linerequestmode_drop"></span>**\_descartar LINEREQUESTMODE**
</dt> <dd> <dl> <dt>



Uma solicitação [tapiRequestDrop](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop) .


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MAKECALL"></span><span id="linerequestmode_makecall"></span>**LINEREQUESTMODE \_ MAKECALL**
</dt> <dd> <dl> <dt>



Uma solicitação [tapiRequestMakeCall](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall) .


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MEDIACALL"></span><span id="linerequestmode_mediacall"></span>**LINEREQUESTMODE \_ MEDIACALL**
</dt> <dd> <dl> <dt>



Essa constante é obsoleta. Não use.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Comentários

Sem extensibilidade. Todos os 32 bits são reservados.
