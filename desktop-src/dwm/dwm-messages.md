# <a name="dwm-messages"></a>Mensagens DWM

Esta seção contém informações sobre as mensagens Gerenciador de Janelas da Área de Trabalho (DWM).

## <a name="in-this-section"></a>Nesta seção




| Tópico | Descrição | 
|-------|-------------|
| <a href="wm-dwmcolorizationcolorchanged.md"><strong>WM_DWMCOLORIZATIONCOLORCHANGED</strong></a><br /> | Informa a todas as janelas de nível superior que a cor da colorização foi alterada.<br /> | 
| <a href="wm-dwmcompositionchanged.md"><strong>WM_DWMCOMPOSITIONCHANGED</strong></a><br /> | Informa a todas as janelas de nível superior que a composição DWM foi habilitada ou desabilitada. <br /><blockquote>[!Note]<br />A partir Windows 8, a composição DWM está sempre habilitada, portanto, essa mensagem não é enviada independentemente das alterações no modo de vídeo.</blockquote><br /> | 
| <a href="wm-dwmncrenderingchanged.md"><strong>WM_DWMNCRENDERINGCHANGED</strong></a><br /> | Enviado quando a política de renderização de área não cliente foi alterada.<br /> | 
| <a href="wm-dwmsendiconiclivepreviewbitmap.md"><strong>WM_DWMSENDICONICLIVEPREVIEWBITMAP</strong></a><br /> | Instrui uma janela a fornecer um bitmap estático <em>a</em> ser usado como uma visualização ao vivo (também conhecida como uma <em>visualização de</em>Espiar ) dessa janela.<br /> | 
| <a href="wm-dwmsendiconicthumbnail.md"><strong>WM_DWMSENDICONICTHUMBNAIL</strong></a><br /> | Instrui uma janela a fornecer um bitmap estático a ser usado como uma representação em miniatura dessa janela.<br /> | 
| <a href="wm-dwmwindowmaximizedchange.md"><strong>WM_DWMWINDOWMAXIMIZEDCHANGE</strong></a><br /> | Enviado quando uma janela composta por DWM é maximizada.<br /> |
