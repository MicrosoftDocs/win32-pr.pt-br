# <a name="dwm-structures"></a>Estruturas do DWM

Esta seção contém informações sobre as estruturas do Gerenciador de Janelas da Área de Trabalho (DWM).

| Tópico                                                                     | Descrição                                                                                                                                               |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM \_ BLURBEHIND**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_blurbehind)<br/>                      | Especifica as propriedades de desfoque do DWM. Usado pela função [**DwmEnableBlurBehindWindow**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmenableblurbehindwindow) .<br/>                     |
| [**\_parâmetros presentes do DWM \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_present_parameters)<br/>     | Especifica os parâmetros de quadro de vídeo do DWM para composição de quadros. Usado pela função [**DwmSetPresentParameters**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetpresentparameters) .<br/>   |
| [**\_Propriedades de miniatura do DWM \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_thumbnail_properties)<br/> | Especifica as propriedades de miniatura do DWM. Usado pela função [**DwmUpdateThumbnailProperties**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmupdatethumbnailproperties) .<br/>                 |
| [**\_informações de tempo do DWM \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_timing_info)<br/>                   | Especifica informações de tempo de composição do DWM. Usado pela função [**DwmGetCompositionTimingInfo**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmgetcompositiontiminginfo) .<br/>         |
| [**MilMatrix3x2D**](/windows/desktop/api/Dwmapi/ns-dwmapi-milmatrix3x2d)<br/>                         | Especifica uma matriz 3x2 que descreve uma transformação. <br/>                                                                                            |
| [**taxa não assinada \_**](/windows/desktop/api/Dwmapi/ns-dwmapi-unsigned_ratio)<br/>                      | Define um tipo de dados usado pelas APIs DWM. Ele representa uma proporção genérica e é usado para diferentes finalidades e unidades, mesmo em uma única API.<br/> |
