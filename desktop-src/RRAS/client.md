# <a name="client"></a>Cliente

Um cliente do RTMv2 é uma instância de um protocolo de roteamento, ou um componente de gerenciamento que registra e interage com o Gerenciador de tabela de roteamento usando a API RTMv2. A menos que especificado de outra forma, qualquer referência a um cliente do Gerenciador de tabela de roteamento refere-se a um cliente do RTMv2.

Um cliente deve ser executado no mesmo processo que o Gerenciador do roteador.
