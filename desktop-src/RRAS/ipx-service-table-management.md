# <a name="ipx-service-table-management"></a>Gerenciamento de tabela de serviço IPX

Um protocolo de roteamento IPX que se registra para manipulação de serviço mantém uma tabela de bindery de serviço, conforme definido pelas especificações do SAP (Protocolo de Publicidade de Serviço) IPX da Novell. O protocolo de roteamento também fornece ao gerenciador de roteador acesso a essa tabela por meio das seguintes funções e estruturas.

-   [Funções de gerenciamento de tabela de serviço](service-table-management-functions.md)
-   [Estruturas de gerenciamento de tabela de serviço](service-table-management-structures.md)

Consulte a [**função RegisterProtocol**](/windows/desktop/api/Routprot/nc-routprot-pregister_protocol) para obter mais informações sobre como um protocolo de roteamento se registra para tratamento de serviço.
