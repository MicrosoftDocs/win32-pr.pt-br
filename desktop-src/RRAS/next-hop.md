# <a name="next-hop"></a>Próximo salto

Um próximo salto é o próximo roteador no caminho em direção a um destino. Os pacotes de uma fonte são encaminhados para um destino em uma base salto a salto.

O endereço do roteador que é o próximo salto em uma rota é usado para identificar exclusivamente os links em uma interface ponto a multiponto, onde todos os links compartilham o mesmo índice de interface.
