# <a name="view"></a>Visualizar

Uma exibição é um subconjunto da tabela de roteamento e contém um grupo de rotas relacionadas (por exemplo, rotas multicast). Atualmente, somente as exibições unicast e multicast têm suporte. As exibições às vezes são chamadas de RIBs (base de informações de roteamento).
