# <a name="view"></a>Visualizar

Uma exibição é um subconjunto da tabela de roteamento e contém um grupo de rotas relacionadas (por exemplo, rotas multicast). Atualmente, há suporte apenas para exibições unicast e multicast. Às vezes, as exibições são chamadas de RIBs (bases de informações de roteamento).
