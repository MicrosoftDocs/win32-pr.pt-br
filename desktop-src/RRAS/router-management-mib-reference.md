# <a name="router-management-mib-reference"></a>Referência do MIB de Gerenciamento de Roteador

Esta documentação descreve os elementos de referência usados na API MIB (Base de Informações de Gerenciamento) específica do gerenciamento de roteador. Use esses elementos de referência para consultar e definir os valores das variáveis MIB. Os gerenciadores de roteadores e os protocolos de roteamento que os serviços do gerenciador de roteador exportam as variáveis MIB:

-   [Funções MIB de gerenciamento de roteador](mib-functions.md)
-   [Constantes de transporte e protocolo](transport-and-protocol-constants.md)

As estruturas que definem variáveis MIB estão documentadas no tópico de rede da [API base](/previous-versions/windows/desktop/mib/management-information-base-reference) de informações de gerenciamento independentes de tecnologia.
