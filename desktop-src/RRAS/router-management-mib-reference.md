# <a name="router-management-mib-reference"></a>Referência de MIB de gerenciamento de roteador

Esta documentação descreve os elementos de referência usados na API da MIB (base de informações de gerenciamento) específica de gerenciamento de roteador. Use esses elementos de referência para consultar e definir os valores das variáveis MIB. Os gerenciadores de roteadores e os protocolos de roteamento dos quais os serviços do Gerenciador de roteadores exportam as variáveis MIB:

-   [Funções de MIB de gerenciamento de roteador](mib-functions.md)
-   [Constantes de transporte e protocolo](transport-and-protocol-constants.md)

As estruturas que definem variáveis MIB são documentadas no tópico rede de [API base de informações de gerenciamento](/previous-versions/windows/desktop/mib/management-information-base-reference) independente de tecnologia.
