<dl> <dt>

<span id="_rras_point_to_multipoint_interface_gly"></span><span id="_RRAS_POINT_TO_MULTIPOINT_INTERFACE_GLY"></span>**interface ponto a multiponto**
</dt> <dd>

A tecnologia do Gerenciador de grupos de multicast usa esse termo para significar uma interface que fornece comunicação entre uma única fonte e um único destino, como um link PPP.

</dd> <dt>

<span id="_rras_point_to_point_interface_gly"></span><span id="_RRAS_POINT_TO_POINT_INTERFACE_GLY"></span>**interface ponto a ponto**
</dt> <dd>

A tecnologia do Gerenciador de grupos multicast usa esse termo para significar uma interface que fornece comunicação entre um único host e vários destinos. As interfaces ponto a multiponto podem ser consideradas como uma coleção de links ponto a ponto com um único encerramento, como um link ATM.

</dd> </dl>
