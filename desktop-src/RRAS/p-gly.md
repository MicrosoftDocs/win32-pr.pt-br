# <a name="p-rras"></a>P (RRAS)

<dl> <dt>

<span id="_rras_point_to_multipoint_interface_gly"></span><span id="_RRAS_POINT_TO_MULTIPOINT_INTERFACE_GLY"></span>**interface ponto a ponto multiponto**
</dt> <dd>

A tecnologia Multicast Group Manager usa esse termo para significar uma interface que fornece comunicação entre uma única origem e um único destino, como um link PPP.

</dd> <dt>

<span id="_rras_point_to_point_interface_gly"></span><span id="_RRAS_POINT_TO_POINT_INTERFACE_GLY"></span>**interface ponto a ponto**
</dt> <dd>

A tecnologia Multicast Group Manager usa esse termo para significar uma interface que fornece comunicação entre um único host e vários destinos. As interfaces ponto a ponto podem ser pensadas como uma coleção de links ponto a ponto com uma única terminação, como um link do ATM.

</dd> </dl>
