# <a name="about-remote-access-service"></a>Sobre o serviço de acesso remoto

O RAS (serviço de acesso remoto) fornece recursos de acesso remoto a aplicativos cliente em computadores que executam o Windows.

Os aplicativos RAS podem executar as seguintes tarefas:

-   Exiba qualquer uma das caixas de diálogo comuns de RAS. Isso inclui a caixa de diálogo **rede dial-up** principal, a folha de propriedades do **Monitor de rede dial-up** e outras caixas de diálogo para criar, editar, copiar ou discar uma entrada de catálogo telefônico.
-   Inicie e encerre uma operação de conexão RAS usando as caixas de diálogo comuns ou as funções de discagem de nível baixo.
-   Crie, edite ou Copie entradas de catálogo telefônico usando as caixas de diálogo comuns ou as funções de catálogo telefônico de nível baixo.
-   Trabalhar com entradas no banco de dados de mapeamento de discagem automática do RAS. Esse banco de dados mapeia endereços de rede para a entrada do catálogo telefônico que pode estabelecer uma conexão com o endereço.
-   Obtenha informações de RAS, incluindo informações sobre conexões RAS existentes, informações sobre os dispositivos compatíveis com RAS configurados no computador local e notificações quando uma conexão RAS começa ou termina.

O Windows NT 4,0 também fornece suporte para administração de servidor RAS e extensões de terceiros para segurança de servidor RAS e gerenciamento de conexão. O Windows 95 não fornece suporte ao servidor RAS.
