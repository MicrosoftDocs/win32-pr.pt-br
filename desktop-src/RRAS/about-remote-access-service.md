# <a name="about-remote-access-service"></a>Sobre o Serviço de Acesso Remoto

O RAS (Serviço de Acesso Remoto) fornece recursos de acesso remoto para aplicativos cliente em computadores que executam Windows.

Os aplicativos RAS podem executar as seguintes tarefas:

-   Exibe qualquer uma das caixas de diálogo comuns do RAS. Isso inclui a caixa de **Sistema de Rede de Conexão Discada** principal, a folha de propriedades do **Sistema de Rede de Conexão Discada Monitor** e outras caixas de diálogo para criar, editar, copiar ou discar uma entrada de phone-book.
-   Inicie e termine uma operação de conexão RAS usando as caixas de diálogo comuns ou as funções de discagem de baixo nível.
-   Crie, edite ou copie entradas de phone-book usando as caixas de diálogo comuns ou as funções de livro telefônica de baixo nível.
-   Trabalhe com entradas no banco de dados de mapeamento automático ras. Esse banco de dados mapeia endereços de rede para a entrada de lista telefônica que pode estabelecer uma conexão com o endereço.
-   Obter informações ras, incluindo informações sobre conexões RAS existentes, informações sobre os dispositivos compatíveis com RAS configurados no computador local e notificações quando uma conexão RAS começa ou termina.

Windows NT 4.0 também fornece suporte para administração de servidor RAS e para extensões de terceiros para gerenciamento de conexão e segurança do servidor RAS. Windows 95 não oferece suporte ao servidor RAS.
