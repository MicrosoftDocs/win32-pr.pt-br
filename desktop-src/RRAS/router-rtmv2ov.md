# <a name="router-rras"></a>Roteador (RRAS)

Para os fins desta documentação, um roteador é um servidor Windows NT/Windows 2000 que está executando o RRAS (serviço de roteamento e acesso remoto) e o Gerenciador de tabela de roteamento.
