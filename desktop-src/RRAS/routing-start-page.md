# <a name="routing"></a>Roteamento

## <a name="purpose"></a>Finalidade

As APIs de Roteamento possibilitam criar aplicativos para administrar os recursos de roteamento do sistema operacional.

## <a name="where-applicable"></a>Quando aplicável

O roteamento possibilita que um computador funcione como um roteador de rede.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs de Roteamento são projetadas para uso por programadores C/C++. Os programadores também devem estar familiarizados com os conceitos de rede.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O roteamento é uma tecnologia baseada em servidor. Toda a funcionalidade de Roteamento é incorporada ao Windows 2000 Server e ao Windows Server 2003. Aplicativos de roteamento não podem ser executados Windows NT Workstation 4.0 ou em sistemas operacionais cliente, como Windows 95. Para obter informações mais específicas sobre quais sistemas operacionais suportam uma função específica, consulte as seções Requisitos na documentação.
