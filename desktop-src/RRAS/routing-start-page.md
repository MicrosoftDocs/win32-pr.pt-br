# <a name="routing"></a>Roteamento

## <a name="purpose"></a>Finalidade

As APIs de roteamento possibilitam a criação de aplicativos para administrar os recursos de roteamento do sistema operacional.

## <a name="where-applicable"></a>Quando aplicável

O roteamento torna possível que um computador funcione como um roteador de rede.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs de roteamento são projetadas para uso por programadores C/C++. Os programadores também devem estar familiarizados com os conceitos de rede.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O roteamento é uma tecnologia baseada em servidor. Toda a funcionalidade do roteamento é incorporada ao Windows 2000 Server e ao Windows Server 2003. Os aplicativos de roteamento não podem ser executados no Windows NT Workstation 4,0 ou em sistemas operacionais cliente, como o Windows 95. Para obter informações mais específicas sobre quais sistemas operacionais dão suporte a uma função específica, consulte as seções de requisitos na documentação do.
