# <a name="routing-protocols"></a>Protocolos de roteamento

## <a name="purpose"></a>Finalidade

Os desenvolvedores podem usar as APIs de Protocolo de Roteamento para implementar protocolos de roteamento.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs de Roteamento são projetadas para uso por programadores C/C++. Os programadores também devem estar familiarizados com os conceitos de rede.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Os protocolos de roteamento são uma tecnologia baseada em servidor. Toda a funcionalidade dos Protocolos de Roteamento é incorporada ao Windows Server 2008, Windows Server 2003 e Windows 2000 Server. Aplicativos de roteamento não podem ser executados em sistemas operacionais cliente, como Windows XP ou Windows Vista. Para obter informações mais específicas sobre quais sistemas operacionais suportam uma função específica, consulte as seções Requisitos na documentação.
