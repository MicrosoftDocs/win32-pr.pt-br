# <a name="routing-protocols"></a>Protocolos de roteamento

## <a name="purpose"></a>Finalidade

Os desenvolvedores podem usar as APIs do protocolo de roteamento para implementar protocolos de roteamento.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs de roteamento são projetadas para uso por programadores C/C++. Os programadores também devem estar familiarizados com os conceitos de rede.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Os protocolos de roteamento são uma tecnologia baseada em servidor. Toda a funcionalidade dos protocolos de roteamento é incorporada ao Windows Server 2008, ao Windows Server 2003 e ao Windows 2000 Server. Os aplicativos de roteamento não podem ser executados em sistemas operacionais cliente, como o Windows XP ou o Windows Vista. Para obter informações mais específicas sobre quais sistemas operacionais dão suporte a uma função específica, consulte as seções de requisitos na documentação do.
