# <a name="routing-protocols-request-for-comments"></a>Solicitação de comentários de protocolos de roteamento

As RFCs (solicitações de comentários) são uma série contínua de documentos que discutem as tecnologias relacionadas à Internet e à Internet. RFCs específicas são identificadas por número. Os RFCs a seguir são referenciados na seção RAS (Serviço de Acesso Remoto) do SDK (Kit de Desenvolvimento de Software de Plataforma). Para obter mais informações sobre RFCs e como localizar cópias de um RFC específico, consulte [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) – Protocolo Internet, Especificação do Protocolo DARPA do Programa de Internet, setembro de 1981
-   [RFC 1267](https://www.ietf.org/rfc/rfc1267.txt) – A Border Gateway Protocol 3 (BGP-3), outubro de 1991
-   [RFC 1268](https://www.ietf.org/rfc/rfc1268.txt) – Aplicativo do protocolo border gateway na Internet, outubro de 1991
-   [RFC 1354](https://www.ietf.org/rfc/rfc1354.txt) – Tabela de encaminhamento de IP MIB, julho de 1992
-   [RFC 1700](https://www.ietf.org/rfc/rfc1700.txt) – Números Atribuídos, outubro de 1994
-   [RFC 2246](https://www.ietf.org/rfc/rfc2246.txt) – Protocolo TLS, janeiro de 1999
-   [RFC 2715](https://www.ietf.org/rfc/rfc2715.txt) – Regras de interoperabilidade para protocolos de roteamento multicast, julho de 1998
