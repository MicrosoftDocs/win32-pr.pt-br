# <a name="address-family"></a>Família de endereços

As famílias de endereços incluem IPX (Internet Packet Exchange), IPv4 (Internet Protocol Versão 4) e IPv6 (Internet Protocol Version 6). As famílias de endereços também podem ser conhecidas como famílias de protocolo. Essas famílias de endereços são definidas [na RFC 1700](routing-protocols-request-for-comments.md).
