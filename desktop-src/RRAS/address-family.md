# <a name="address-family"></a>Família de endereços

As famílias de endereços incluem IPX (Internet Packet Exchange), IPv4 (Internet Protocol versão 4) e IPv6 (protocolo IP versão 6). As famílias de endereços também podem ser chamadas de famílias de protocolos. Essas famílias de endereços são definidas no [RFC 1700](routing-protocols-request-for-comments.md).
