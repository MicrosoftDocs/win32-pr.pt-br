# <a name="routing-table"></a>Tabela de roteamento

O Gerenciador de tabela de roteamento mantém uma tabela de roteamento distinta para cada família de endereços (como IPX e IPv4). A família de endereços IPX é suportada pela API RTMv1; O IPv4 tem suporte da API RTMv2.

A tabela de roteamento consiste em destinos, rotas e próximos saltos. Essas entradas definem uma rota para uma rede de destino.
