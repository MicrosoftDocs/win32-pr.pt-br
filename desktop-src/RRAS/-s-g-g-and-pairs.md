# <a name="s-g--g-and---pairs"></a>(s, g), ( \* , g) e ( \* , ) \* Pares

A notação (s, g) representa uma origem e um grupo específicos. Somente os dados da origem, s e do grupo, g, são solicitados.

A notação ( \* , g) representa uma origem curinga e um grupo específico. Todos os dados de qualquer origem para o grupo, g, foram solicitados.

A notação ( \* , ) representa uma \* origem curinga e um grupo de caracteres curinga. Todos os dados de todas as fontes e de todos os grupos foram solicitados.

Essas notações são usadas para descrever a adição e a remoção de associações de grupo.
