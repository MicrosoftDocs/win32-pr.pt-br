# <a name="s-g--g-and---pairs"></a>(s, g), ( \* , g) e ( \* , \* ) pares

A notação (s, g) representa uma origem e um grupo específicos. Somente os dados da origem, s e do grupo, g, são solicitados.

A notação ( \* , g) representa uma fonte curinga e um grupo específico. Todos os dados de qualquer fonte para o grupo, g, foram solicitados.

A notação ( \* , \* ) representa uma origem curinga e um grupo curinga. Todos os dados de todas as fontes e todos os grupos foram solicitados.

Essas notações são usadas para descrever a adição e a remoção de associações de grupo.
