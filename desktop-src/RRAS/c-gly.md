<dl> <dt>

<span id="_rras_cyclic_redundancy_check_crc__gly"></span><span id="_RRAS_CYCLIC_REDUNDANCY_CHECK_CRC__GLY"></span>**CRC (Verificação de Redundância Cíclica)**
</dt> <dd>

Um CRC é um valor numérico relacionado a um bloco de dados. Esse valor fornece informações que permitem que um aplicativo determine se os dados foram modificados. Um CRC é semelhante a uma verificação de verificação, mas é considerado um mecanismo de verificação de erros mais forte.

</dd> <dt>

<span id="_rras_compression_control_protocol_ccp__gly"></span><span id="_RRAS_COMPRESSION_CONTROL_PROTOCOL_CCP__GLY"></span>**Protocolo CCP**
</dt> <dd>

O protocolo CCP permite que dois computadores que se comuniquem por meio de PPP negociem algoritmos compatíveis para enviar e receber quadros PPP compactados. Os dois computadores não poderão usar o CCP até a fase de protocolo de controle de rede da conexão PPP.

Para obter mais informações, [consulte RFC 1962](remote-access-request-for-comments.md). Consulte também [**INFORMAÇÕES \_ CCP \_ PPP**](/windows/desktop/api/Mprapi/ns-mprapi-ppp_ccp_info) e [**RASPPPCCP**](/previous-versions/windows/desktop/legacy/aa377620(v=vs.85)).

</dd> </dl>
