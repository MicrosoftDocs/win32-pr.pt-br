# <a name="router-manager"></a>Gerenciador de roteador

o gerenciador de roteador é o componente de um roteador Windows NT/Windows 2000 que inicia e gerencia os diferentes componentes de roteamento. Os componentes incluem protocolos de roteamento, o Gerenciador de tabela de roteamento e o Gerenciador do grupo de multicast.
