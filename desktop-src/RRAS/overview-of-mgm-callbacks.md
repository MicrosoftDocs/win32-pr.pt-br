# <a name="overview-of-mgm-callbacks"></a>Visão geral de retornos de chamada do MGM

Há dois tipos de retornos de chamada na API MGM:

-   [Retornos de chamada do protocolo de roteamento](routing-protocol-callbacks.md)
-   [Habilitar e desabilitar retornos de chamada IGMP](igmp-enable-and-disable-callbacks.md)

Esses retornos de chamada, combinados com as APIs MGM, permitem a comunicação contínua entre protocolos de roteamento, IGMP e o Gerenciador do grupo de multicast.
