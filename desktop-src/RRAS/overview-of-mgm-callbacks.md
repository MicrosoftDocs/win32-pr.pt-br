# <a name="overview-of-mgm-callbacks"></a>Visão geral dos retornos de chamada do MGM

Há dois tipos de retornos de chamada na API do MGM:

-   [Retornos de chamada de protocolo de roteamento](routing-protocol-callbacks.md)
-   [Habilitar e desabilitar retornos de chamada do IGMP](igmp-enable-and-disable-callbacks.md)

Esses retornos de chamada, combinados com as APIs do MGM, permitem a comunicação contínua entre protocolos de roteamento, IGMP e o gerenciador de grupos multicast.
