# <a name="routing-table-manager-version-1-protocol-family-identifiers"></a>Identificadores da família de protocolos do Gerenciador de tabela de roteamento versão 1

**Windows Server 2003:** Essa API foi substituída pela API do [Gerenciador de tabela de roteamento versão 2](about-routing-table-manager-version-2.md) e não estará disponível além do Windows Server 2003. Novos aplicativos devem usar a API da versão 2 do Gerenciador de tabela de roteamento.

Os seguintes identificadores de transporte estão listados em RTM. h. Use esses identificadores com a API RTMv1.



| Constante                   | Descrição                        |
|----------------------------|------------------------------------|
| \_protocolo \_ IPX da família de protocolos RTM \_ | Identifica a família de endereços IPX. |
| \_ \_ IP da família de protocolo RTM \_  | Identifica a família de endereços IP.  |
