# <a name="adapters"></a>Adaptadores

Um adaptador representa o hardware que se conecta a um segmento de rede específico. Uma placa de LAN ligada é um exemplo de um adaptador. Da mesma forma, um computador com dois modems, cada um capaz de se conectar a uma rede remota, tem dois adaptadores, um para representar cada modem.
