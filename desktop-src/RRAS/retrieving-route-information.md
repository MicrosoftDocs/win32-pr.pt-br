# <a name="retrieving-route-information"></a>Recuperando informações de rota

Há três métodos usados para obter informações de rota do gerenciador de tabelas de roteamento.

-   Enumerando rotas (descritas em [Enumerando entradas de tabela de roteamento](enumerating-routing-table-entries.md)).
-   Procurando rotas específicas (descritas [em Encontrando informações específicas na tabela de roteamento](finding-specific-information-in-the-routing-table.md)).
-   Recuperando destinos alterados (descrito em [Recebendo notificação de alterações).](receiving-notification-of-changes.md)
