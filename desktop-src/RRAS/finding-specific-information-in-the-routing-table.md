# <a name="finding-specific-information-in-the-routing-table"></a>Encontrando informações específicas na tabela de roteamento

Os clientes devem ser capazes de localizar informações específicas na tabela de roteamento, em vez de serem necessários para enumerar todos os dados. A API RTMv2 fornece a capacidade de pesquisar destinos, rotas e próximos saltos específicos com base em determinados critérios.
