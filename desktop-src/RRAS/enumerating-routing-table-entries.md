# <a name="enumerating-routing-table-entries"></a>Enumerando entradas da tabela de roteamento

As funções de enumeração permitem que um cliente recupere informações sobre um tipo específico de objeto de tabela de roteamento (rotas, destinos e próximos saltos). Os protocolos de roteamento e os programas de Administração podem usar essas funções para localizar dados específicos.

Para obter um exemplo de código que mostra como usar essas funções, consulte [enumerar todos os destinos](enumerate-all-destinations.md) e [enumerar todas as rotas](enumerate-all-routes.md).
