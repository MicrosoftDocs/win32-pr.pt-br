# <a name="maintaining-client-specific-lists"></a>Mantendo Client-Specific listas

O RTMv2 fornece funções que permitem que os clientes criem uma lista privada de rotas armazenadas na tabela de roteamento. Usar essa lista pode ser mais eficiente do que enumerar rotas da tabela de roteamento.

Para ver o código de exemplo que mostra como usar essas funções, [consulte Usar uma Client-Specific de rotas .](use-a-client-specific-route-list.md)
