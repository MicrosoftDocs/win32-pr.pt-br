# <a name="maintaining-client-specific-lists"></a>Manutenção de listas de Client-Specific

O RTMv2 fornece funções que permitem que os clientes criem uma lista privada de rotas armazenadas na tabela de roteamento. O uso dessa lista pode ser mais eficiente do que enumerar rotas da tabela de roteamento.

Para obter um exemplo de código que mostra como usar essas funções, consulte [usar uma lista de rotas Client-Specific](use-a-client-specific-route-list.md).
