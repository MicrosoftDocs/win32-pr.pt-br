# <a name="span-idvspixengineeventcolumnidspaneventcolumnid-enumeration"></a><span id="vspixengine.eventcolumnid"></span>Enumeração EVENTCOLUMNID

Uma enumeração usada para indicar IDs de coluna bem conhecidas; Essas são IDs de coluna às quais todas as implementações devem dar suporte.

};

<span id="EventColumnID_EVENTTYPE"></span><span id="eventcolumnid_eventtype"></span><span id="EVENTCOLUMNID_EVENTTYPE"></span>**\_EVENTTYPE EventColumnID**  
Uma ID que corresponde à coluna de tipo de evento.

<span id="EventColumnID_EID"></span><span id="eventcolumnid_eid"></span><span id="EVENTCOLUMNID_EID"></span>**EventColumnID \_ Eid**  
Uma ID que corresponde à coluna de ID do evento.

<span id="EventColumnID_PARENTEID"></span><span id="eventcolumnid_parenteid"></span><span id="EVENTCOLUMNID_PARENTEID"></span>**EventColumnID \_ PARENTEID**  
Uma ID que corresponde à coluna de ID do evento pai.

<span id="EventColumnID_EVENTNAME"></span><span id="eventcolumnid_eventname"></span><span id="EVENTCOLUMNID_EVENTNAME"></span>**EventColumnID \_ EventName**  
Uma ID que corresponde à coluna de nome do evento.

<span id="EventColumnID_FRAME"></span><span id="eventcolumnid_frame"></span><span id="EVENTCOLUMNID_FRAME"></span>**\_Quadro EventColumnID**  
Uma ID que corresponde à coluna de quadro.

<span id="EventColumnID_EVENTSTRING"></span><span id="eventcolumnid_eventstring"></span><span id="EVENTCOLUMNID_EVENTSTRING"></span>**\_Eventostring do EventColumnID**  
Uma ID que corresponde à cadeia de caracteres completa a ser analisada para o evento.

<span id="EventColumnID_THISPTR"></span><span id="eventcolumnid_thisptr"></span><span id="EVENTCOLUMNID_THISPTR"></span>**EventColumnID \_ THISPTR**  
Uma ID que corresponde ao identificador de recurso para o ponteiro.

<span id="EventColumnID_STARTMOMENTHANDLE"></span><span id="eventcolumnid_startmomenthandle"></span><span id="EVENTCOLUMNID_STARTMOMENTHANDLE"></span>**EventColumnID \_ STARTMOMENTHANDLE**

<span id="EventColumnID_InfoQueueMessages"></span><span id="eventcolumnid_infoqueuemessages"></span><span id="EVENTCOLUMNID_INFOQUEUEMESSAGES"></span>**EventColumnID \_ InfoQueueMessages**

<span id="EventColumnID_ThreadId"></span><span id="eventcolumnid_threadid"></span><span id="EVENTCOLUMNID_THREADID"></span>**EventColumnID \_ ThreadID**
