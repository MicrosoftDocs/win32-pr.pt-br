# <a name="effect-compiler-tool"></a>Ferramenta de Effect-Compiler

FXC (fxc.exe) é uma ferramenta offline para compilar sombreadores [HLSL](/windows/desktop/direct3dhlsl/dx-graphics-hlsl) para todas as versões do Direct3D. A ferramenta está localizada em:

(*Raiz do SDK*) \\ Bandeja de utilitários \\ \\ x86\\
