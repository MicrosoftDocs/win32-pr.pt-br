# <a name="tree-view-control-window-styles"></a>Tree-View estilos de janela de controle

Esta seção lista os estilos de janela usados ao criar controles de exibição em árvore.



| Constante                                                                                                                                                                        | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TVS_CHECKBOXES"></span><span id="tvs_checkboxes"></span><dl> <dt>**\_caixas de seleção TVs**</dt> </dl>                | [Versão 4,70](common-control-versions.md). Habilita as caixas de seleção de itens em um controle de exibição de árvore. Uma caixa de seleção será exibida somente se uma imagem estiver associada ao item. Quando definido para esse estilo, o controle usa efetivamente [**DrawFrameControl**](/windows/desktop/api/winuser/nf-winuser-drawframecontrol) para criar e definir uma lista de imagens de estado contendo duas imagens. A imagem de estado 1 é a caixa desmarcada e a imagem de estado 2 é a caixa marcada. A definição da imagem de estado como zero remove totalmente a caixa de seleção. Para obter mais informações, consulte [trabalhando com índices de imagem de estado](tree-view-controls.md).<br/> [Versão 5,80](common-control-versions.md). Exibe uma caixa de seleção mesmo que nenhuma imagem esteja associada ao item.<br/> Depois que um controle de exibição de árvore é criado com esse estilo, o estilo não pode ser removido. Em vez disso, você deve destruir o controle e criar um novo em seu lugar. A destruição do controle de exibição de árvore não destrói a lista de imagens de estado da caixa de seleção. Você deve destruí-lo explicitamente. Obtenha o identificador para a lista de imagens de estado enviando o controle de exibição de árvore uma mensagem [**TVM \_ GetImageList**](tvm-getimagelist.md) . Em seguida, destrua a lista de imagens com [**ImageList \_ Destroy**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_destroy).<br/> Se você quiser usar esse estilo, deverá definir o \_ estilo das caixas de seleção TVs com [**SetWindowLong**](/windows/desktop/api/winuser/nf-winuser-setwindowlonga) depois de criar o controle TreeView e antes de preencher a árvore. Caso contrário, as caixas de seleção podem parecer desmarcadas, dependendo dos problemas de tempo.<br/> |

| <span id="TVS_EDITLABELS"></span><span id="tvs_editlabels"></span><dl> <dt>**TVS \_ EDITLABELS**</dt> </dl>                | Permite que o usuário edite os rótulos de itens de exibição em árvore. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="TVS_FULLROWSELECT"></span><span id="tvs_fullrowselect"></span><dl> <dt>**TVS \_ FULLROWSELECT**</dt> </dl>       | [Versão 4,71](common-control-versions.md). Habilita a seleção de linha completa no modo de exibição de árvore. A linha inteira do item selecionado é realçada e clicar em qualquer lugar na linha de um item faz com que ele seja selecionado. Esse estilo não pode ser usado em conjunto com o estilo **\_ HASLINES de TVs** . <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_HASBUTTONS"></span><span id="tvs_hasbuttons"></span><dl> <dt>**TVS \_ HASBUTTONS**</dt> </dl>                | Exibe os botões de mais (+) e menos (-) ao lado dos itens pai. O usuário clica nos botões para expandir ou recolher a lista de itens filho de um item pai. Para incluir botões com itens na raiz do modo de exibição de árvore, TVS \_ LINESATROOT também devem ser especificados. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_HASLINES"></span><span id="tvs_haslines"></span><dl> <dt>**TVS \_ HASLINES**</dt> </dl>                      | Usa linhas para mostrar a hierarquia de itens. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="TVS_INFOTIP"></span><span id="tvs_infotip"></span><dl> <dt>**INFOTIP de TVS \_**</dt> </dl>                         | [Versão 4,71](common-control-versions.md). Obtém informações de dica de ferramenta enviando a notificação [TVN \_ GETINFOTIP](tvn-getinfotip.md) . <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="TVS_LINESATROOT"></span><span id="tvs_linesatroot"></span><dl> <dt>**TVS \_ LINESATROOT**</dt> </dl>             | Usa linhas para vincular itens na raiz do controle de exibição de árvore. Esse valor será ignorado se TVS \_ HASLINES não for especificado também. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="TVS_NOHSCROLL"></span><span id="tvs_nohscroll"></span><dl> <dt>**TVS \_ NOHSCROLL**</dt> </dl>                   | [Versão 5,80](common-control-versions.md). Desabilita a rolagem horizontal no controle. O controle não exibirá nenhuma barra de rolagem horizontal. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="TVS_NONEVENHEIGHT"></span><span id="tvs_nonevenheight"></span><dl> <dt>**TVS \_ NONEVENHEIGHT**</dt> </dl>       | [Versão 4,71](common-control-versions.md) Define a altura dos itens para uma altura ímpar com a mensagem [**TVM \_ SETITEMHEIGHT**](tvm-setitemheight.md) . Por padrão, a altura dos itens deve ser um valor par. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

Desabilita a rolagem horizontal e vertical no controle. O controle não exibirá nenhuma barra de rolagem. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="TVS_NOTOOLTIPS"></span><span id="tvs_notooltips"></span><dl> <dt>**TVS \_ NOtooltips**</dt> </dl>                | [Versão 4,70](common-control-versions.md). Desabilita as dicas de ferramentas. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="TVS_RTLREADING"></span><span id="tvs_rtlreading"></span><dl> <dt>**TVS \_ RTLREADING**</dt> </dl>                | [Versão 4,70](common-control-versions.md). Faz com que o texto seja exibido da direita para a esquerda (RTL). Normalmente, o Windows exibe texto da esquerda para a direita (EPD). O Windows pode ser *espelhado* para exibir idiomas como hebraico ou árabe que lêem DPE. Normalmente, o texto de exibição em árvore é exibido na mesma direção que o texto em sua janela pai. Se TVS \_ RTLREADING estiverem definidas, o texto de exibição de árvore lerá na direção oposta do texto na janela pai. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_SHOWSELALWAYS"></span><span id="tvs_showselalways"></span><dl> <dt>**TVS \_ SHOWSELALWAYS**</dt> </dl>       | Faz com que um item selecionado permaneça selecionado quando o controle de exibição de árvore perde o foco. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| <span id="TVS_SINGLEEXPAND"></span><span id="tvs_singleexpand"></span><dl> <dt>**TVS \_ SINGLEEXPAND**</dt> </dl>          | [Versão 4,71](common-control-versions.md). Faz com que o item que está sendo selecionado se expanda e o item que está sendo desmarcado para recolher na seleção na exibição de árvore. Se o mouse for usado para clicar uma vez no item selecionado e esse item for fechado, ele será expandido. Se o usuário mantiver a tecla CTRL pressionada enquanto seleciona um item, o item que está sendo desmarcado não será recolhido. <br/> [Versão 5,80](common-control-versions.md). Faz com que o item que está sendo selecionado se expanda e o item que está sendo desmarcado para recolher na seleção na exibição de árvore. Se o usuário mantiver a tecla CTRL pressionada enquanto seleciona um item, o item que está sendo desmarcado não será recolhido.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

Habilita o rastreamento dinâmico em um controle de exibição de árvore. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
