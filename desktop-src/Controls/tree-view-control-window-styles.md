# <a name="tree-view-control-window-styles"></a>Tree-View de janela de controle

Esta seção lista os estilos de janela usados ao criar controles de exibição de árvore.

| <span id="TVS_CHECKBOXES"></span><span id="tvs_checkboxes"></span><dl> <dt>**CAIXAS DE \_ SELEÇÃO DE TVS**</dt> </dl>                | [Versão 4.70.](common-control-versions.md) Habilita caixas de seleção para itens em um controle de exibição de árvore. Uma caixa de seleção será exibida somente se uma imagem estiver associada ao item. Quando definido com esse estilo, o controle usa [**efetivamente DrawFrameControl**](/windows/desktop/api/winuser/nf-winuser-drawframecontrol) para criar e definir uma lista de imagens de estado que contém duas imagens. A imagem de estado 1 é a caixa desmarcada e a imagem de estado 2 é a caixa marcada. Definir a imagem de estado como zero remove completamente a caixa de seleção. Para obter mais informações, consulte [Trabalhando com índices de imagem de estado](tree-view-controls.md).<br/> [Versão 5.80.](common-control-versions.md) Exibe uma caixa de seleção mesmo que nenhuma imagem seja associada ao item.<br/> Depois que um controle de exibição de árvore é criado com esse estilo, o estilo não pode ser removido. Em vez disso, você deve destruir o controle e criar um novo em seu lugar. Destruir o controle de exibição de árvore não destrói a lista de imagens de estado da caixa de seleção. Você deve destruir explicitamente. Obter o alçamento para a lista de imagens de estado enviando ao controle de exibição de árvore uma mensagem [**TVM \_ GETIMAGELIST.**](tvm-getimagelist.md) Em seguida, destruir a lista de imagens [**com ImageList \_ Destroy**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_destroy).<br/> Se você quiser usar esse estilo, deverá definir o estilo TVS \_ CHECKBOXES com [**SetWindowLong**](/windows/desktop/api/winuser/nf-winuser-setwindowlonga) depois de criar o controle treeview e antes de preencher a árvore. Caso contrário, as caixas de seleção poderão aparecer desmarcadas, dependendo de problemas de tempo.<br/> |

| <span id="TVS_FULLROWSELECT"></span><span id="tvs_fullrowselect"></span><dl> <dt>**TVS \_ FULLROWSELECT**</dt> </dl>       | [Versão 4.71.](common-control-versions.md) Habilita a seleção de linha completa no modo de exibição de árvore. A linha inteira do item selecionado é realçada e clicar em qualquer lugar na linha de um item faz com que ele seja selecionado. Esse estilo não pode ser usado em conjunto com o estilo **\_ HASLINES da TVS.** <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

O usuário clica nos botões para expandir ou ressumentar a lista de itens filho de um item pai. Para incluir botões com itens na raiz do exibição de árvore, TVS \_ LINESATROOT também deve ser especificado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

| <span id="TVS_RTLREADING"></span><span id="tvs_rtlreading"></span><dl> <dt>**TVS \_ RTLREADING**</dt> </dl>                | [Versão 4.70.](common-control-versions.md) Faz com que o texto seja exibido da RTL (direita para esquerda). Normalmente, o texto de exibição de árvore é exibido na mesma direção que o texto em sua janela pai. Se TVS RTLREADING estiver definido, o texto de exibição de árvore será lido na direção oposta do \_ texto na janela pai. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

Se o usuário mantém a tecla CTRL ao selecionar um item, o item que está sendo deselecionado não será recolhido. <br/> [Versão 5.80.](common-control-versions.md) Faz com que o item que está sendo selecionado expanda e o item que está sendo deseleito seja resseleito após a seleção no exibição de árvore. Se o usuário mantém a tecla CTRL ao selecionar um item, o item que está sendo deselecionado não será recolhido.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
