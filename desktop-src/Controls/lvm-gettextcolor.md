# <a name="lvm_gettextcolor-message"></a>\_Mensagem GETTEXTCOLOR LVM

Recupera a cor do texto de um controle de exibição de lista. Você pode enviar essa mensagem explicitamente ou usando a macro [**\_ GetTextColor do ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_gettextcolor) .

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retorna a cor do texto.
