# <a name="tb_isbuttonhidden-message"></a>TB de \_ mensagem ISBUTTONHIDDEN

Determina se o botão especificado em uma barra de ferramentas está oculto.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador de comando do botão.

</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retornará zero se o botão estiver oculto ou nenhum outro.
