# <a name="sb_issimple-message"></a>Mensagem do SB \_ ISsimple

Verifica um controle de barra de status para determinar se ele está no modo simples.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Valor retornado

Retornará zero se o controle da barra de status estiver no modo simples ou nenhum outro.
