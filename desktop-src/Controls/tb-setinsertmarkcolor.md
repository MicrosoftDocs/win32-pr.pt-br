# <a name="tb_setinsertmarkcolor-message"></a>TB de \_ mensagem SETINSERTMARKCOLOR

Define a cor usada para desenhar a marca de inserção para a barra de ferramentas.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>

Valor **COLORREF** que contém a nova cor de marca de inserção.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retorna um valor **COLORREF** que contém a cor de marca de inserção anterior.
