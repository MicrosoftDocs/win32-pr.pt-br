# <a name="udm_getbase-message"></a>Mensagem do UDM \_ GETbase

Recupera a base da Radix atual (ou seja, a base 10 ou 16) para um controle acima-abaixo.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Valor retornado

O valor de retorno é o valor de base atual.
