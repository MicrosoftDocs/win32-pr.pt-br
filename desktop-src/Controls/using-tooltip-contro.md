# <a name="using-tooltip-controls"></a>Usando controles de dica de ferramenta

Esta seção contém exemplos que demonstram como criar diferentes tipos de dicas de ferramenta.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                    | Descrição                                                                                                                                                                                                                                                                      |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Como criar uma dica de ferramenta para um controle](create-a-tooltip-for-a-control.md)<br/>                   | A função de exemplo a seguir cria uma dica de ferramenta e a associa ao controle cuja ID de recurso é passada. <br/>                                                                                                                                                  |
| [Como criar uma dica de ferramenta para uma área retangular](create-a-tooltip-for-a-rectangular-area.md)<br/> | O exemplo a seguir demonstra como criar um controle de dica de ferramenta padrão para a área inteira do cliente de uma janela. <br/>                                                                                                                                                       |
| [Como implementar dicas de ferramentas de rastreamento](implement-tracking-tooltips.md)<br/>                         | As dicas de ferramentas de rastreamento permanecem visíveis até que sejam explicitamente fechadas pelo aplicativo e podem alterar a posição na tela dinamicamente. Eles têm suporte da [versão 4,70](common-control-versions.md) e posterior dos controles comuns. <br/>                         |
| [Como implementar dicas de ferramentas de várias linhas](implement-multiline-tooltips.md)<br/>                       | As dicas de ferramentas de várias linhas permitem que o texto seja exibido em mais de uma linha. <br/>                                                                                                                                                                                                 |
| [Como implementar dicas de ferramentas de balão](implement-balloon-tooltips.md)<br/>                           | As dicas de ferramentas de balão são semelhantes às dicas de ferramenta padrão, mas são exibidas em um "balão" estilo animado com uma haste apontando para a ferramenta. As dicas de ferramentas de balão podem ser de linha única ou de várias linhas. Eles são criados e manipulados quase da mesma forma que as dicas de ferramentas padrão. <br/> |
| [Como implementar dicas de ferramenta para ícones da barra de status](implement-tooltips-for-status-bar-icons.md)<br/> | Uma maneira não intrusiva de exibir uma mensagem explicativa para um ícone da barra de status é implementar uma dica de ferramenta. A dica de ferramenta desaparece quando clicada, mas você também pode especificar um valor de tempo limite. <br/>                                                                                |
| [Como implementar dicas de ferramentas de In-Place](implement-in-place-tooltips.md)<br/>                         | Dicas de ferramenta in-loco são usadas para exibir cadeias de caracteres de texto para objetos que foram recortados. Para obter uma ilustração, consulte [sobre os controles de dica de ferramenta](tooltip-controls.md). <br/>                                                                                                      |
