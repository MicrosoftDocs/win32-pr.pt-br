# <a name="window-classes-commctrlh"></a>Classes de janela (CommCtrl. h)

Esta seção lista os nomes de classe de janela fornecidos pela biblioteca de controle comum.



| Constante                                                                                                                                                                     | Descrição                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**\_classe Animate**</dt> </dl>                | Cria controles de animação. Esses controles exibem silenciosamente um clipe AVI (vídeo de áudio Intercalado).<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**\_classe DATETIMEPICK**</dt> </dl> | Cria controles de seletor de data e hora. Esses controles fornecem uma interface simples para trocar informações de data e hora com um usuário. <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**classe de tecla de atalho \_**</dt> </dl>                   | Cria controles de tecla quente. Esses controles facilitam para o usuário definir teclas de acesso. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**classe de LINK \_**</dt> </dl>                         | Cria controles Syslink. Esses controles são usados com links. <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**\_classe calendário mensal**</dt> </dl>             | Cria controles de calendário de mês. Esses controles fornecem uma maneira simples para um usuário selecionar uma data de uma interface familiar. <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**\_classe NATIVEFNTCTL**</dt> </dl> | Cria controles de fonte nativos. Esses controles são usados com fontes nativas. <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**\_classe Progress**</dt> </dl>             | Cria barras de progresso. Esses controles indicam o progresso de uma operação demorada.<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**REBARCLASSNAME**</dt> </dl>              | Cria controles Rebar. Esses controles atuam como um contêiner para janelas filhas. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**\_classes padrão**</dt> </dl>       | Cria controles de user32 intrínsecos, como botão, editar e barra de rolagem.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**STATUSCLASSNAME**</dt> </dl>           | Cria janelas de status. Esses controles exibem informações de status em uma janela horizontal.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**TOOLBARCLASSNAME**</dt> </dl>        | Cria barras de ferramentas. Esses controles contêm botões que executam comandos de menu. <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**classe TOOLTIPs \_**</dt> </dl>             | Cria controles de dica de ferramenta. Esses controles exibem uma pequena janela pop-up contendo uma linha de texto que descreve a finalidade de uma ferramenta em um aplicativo.<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**\_classe TrackBar**</dt> </dl>             | Cria trackbars. Esses controles permitem que o usuário selecione um intervalo de valores movendo um controle deslizante. <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**classe upDown \_**</dt> </dl>                   | Cria controles de cima para baixo. Esses controles combinam um par de setas com um controle de edição. Clicar nas setas incrementa ou Decrementa o valor no controle de edição. <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**\_botão WC**</dt> </dl>                            | Cria controles de botão. Esses controles permitem que um usuário inicie uma ação.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**\_ComboBox WC**</dt> </dl>                      | Cria controles ComboBox. Esses controles contêm uma lista suspensa da qual um único item pode ser selecionado. <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**WC \_ COMBOBOXEX**</dt> </dl>                | Cria controles ComboBoxEx. Esses controles fornecem uma extensão do controle de caixa de combinação que fornece suporte nativo para imagens de itens. <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**WC \_ Editar**</dt> </dl>                                  | Cria controles de edição. Esses controles contêm texto editável. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**\_cabeçalho WC**</dt> </dl>                            | Cria controles de cabeçalho. Esses controles exibem cabeçalhos na parte superior das colunas de informações e permitem que o usuário classifique as informações clicando nos títulos. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**caixa de \_ listagem WC**</dt> </dl>                         | Cria controles ListBox. Esses controles contêm listas de itens. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**\_IPAddress WC**</dt> </dl>                   | Cria controles de endereço IP. Esses controles são semelhantes a um controle de edição, mas permitem que você insira um endereço numérico no formato IP (Internet Protocol). <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**LINK do WC \_**</dt> </dl>                                  | Cria controles SysLink. Esses controles contêm links de hipertexto.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**\_ListView WC**</dt> </dl>                      | Cria controles de exibição de lista. Esses controles exibem uma coleção de itens, cada um consistindo em um ícone e um rótulo e fornecem várias maneiras de organizar os itens.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**WC \_ NATIVEFONTCTL**</dt> </dl>       | Cria controles de fonte nativos. Esses são controles invisíveis que funcionam em segundo plano para permitir que os controles predefinidos de uma caixa de diálogo exibam o idioma atual do sistema.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**WC \_ PAGESCROLLER**</dt> </dl>          | Cria controles de pager. Esses controles são usados para conter e rolar outra janela. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**WC \_ ScrollBar**</dt> </dl>                   | Cria controles de barra de rolagem. Esses controles permitem que o usuário Role o conteúdo de uma janela.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ estático**</dt> </dl>                            | Cria controles estáticos. Esses controles contêm texto não editável. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**\_TABCONTROL WC**</dt> </dl>                | Cria controles de guia. Esses controles definem várias páginas para a mesma área de uma janela ou caixa de diálogo. Cada página consiste em um conjunto de informações ou um grupo de controles que um aplicativo exibe quando o usuário seleciona a guia correspondente.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**WC \_ TreeView**</dt> </dl>                      | Cria controles de exibição em árvore. Esses controles exibem uma lista hierárquica de itens. Cada item consiste em um rótulo e um bitmap opcional.<br/>                                                                                                              |
