# <a name="general-control-reference"></a>Referência de controle geral

Esta seção contém informações de referência sobre elementos de programação que se aplicam a vários controles, não apenas a um controle específico. Há funções, macros, mensagens, notificações e estruturas com suporte da maioria dos controles. Por exemplo, a maioria dos controles usa as notificações de [ \_ foco de nm](nm-hover.md) para processar cliques do mouse.
