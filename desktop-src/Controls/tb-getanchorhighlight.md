# <a name="tb_getanchorhighlight-message"></a>TB de \_ mensagem GETANCHORHIGHLIGHT

Recupera a configuração de realce de âncora para uma barra de ferramentas.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Valor retornado

Retorna um valor booliano que indica se o realce de âncora está definido. Se esse valor for diferente de zero, o realce de âncora será habilitado. Se esse valor for zero, ele será desabilitado.
