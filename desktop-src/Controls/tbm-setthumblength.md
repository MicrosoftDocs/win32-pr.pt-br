# <a name="tbm_setthumblength-message"></a>\_Mensagem tbm SETTHUMBLENGTH

Define o comprimento do controle deslizante em um TrackBar. Essa mensagem será ignorada se o TrackBar não tiver o [**estilo \_ Cadeia TBS**](trackbar-control-styles.md) .

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Comprimento, em pixels, do controle deslizante.

</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Sem valor de retorno.
