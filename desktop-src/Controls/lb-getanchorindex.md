# <a name="lb_getanchorindex-message"></a>GETANCHORINDEX de mensagens de LB \_

Obtém o índice do item de âncora que é o item do qual uma seleção múltipla é iniciada. Uma seleção múltipla abrange todos os itens do item de âncora para o item de cursor.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Não usado; deve ser zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Não usado; deve ser zero.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

O valor de retorno é o índice do item de âncora.
