# <a name="static-control-styles"></a>Estilos de controle estático

## <a name="remarks"></a>Comentários

Para usar um estilo que trunca palavras ou cadeias de caracteres com reticências para texto com várias linhas, [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) ou [**DrawText**](/windows/desktop/api/winuser/nf-winuser-drawtext) podem ser usados em um controle de desenho proprietário com sinalizadores de formatação de **dt \_ justificação** e **dt \_** .
