# <a name="default-list-view-message-processing"></a>Processamento de List-View de mensagens padrão

Esta seção lista o processamento de mensagem de janela executado por um controle de exibição de lista.

## <a name="message-processing"></a>Processamento de mensagem

A tabela a seguir descreve o processamento de mensagem de janela executado por um controle de exibição de lista.



| Mensagem                  | Processamento realizado                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **caractere do WM \_**             | Pesquisa um item de exibição de lista que começa com o caractere especificado e, se o item for encontrado, seleciona e define o foco para o item. Vários caracteres recebidos dentro de um intervalo de tempo limite, atualmente fixos em um segundo, são concatenados e o controle de exibição de lista procura um item que começa com a cadeia de caracteres resultante.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **comando do WM \_**          | Processa os códigos de notificação [en \_ Update](en-update.md) e [en \_ KILLFOCUS](en-killfocus.md) e encaminha todas as outras notificações de controle de edição para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **criação do WM \_**           | Executa a inicialização adicional para o [**WM \_ NCCREATE**](/windows/desktop/winmsg/wm-nccreate). Se o estilo de janela [**LVS \_ SHAREIMAGELISTS**](list-view-window-styles.md) não for especificado, o controle List-View criará as listas de imagens icon e ícone pequeno neste ponto.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **destruição do WM \_**          | Libera recursos alocados durante [**a \_ criação do WM**](/windows/desktop/winmsg/wm-create).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **ERASEBKGND do WM \_**       | Apaga o plano de fundo da janela usando a cor da tela de fundo atual para o controle de exibição de lista. Se a cor do plano de fundo for o \_ valor CLR None, o controle List-View encaminhará a mensagem para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **GETDLGCODE do WM \_**       | Retorna uma combinação dos valores DLGC \_ WANTTAB e DLGC \_ WANTARROWS.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WM \_ GETfont**          | Retorna o identificador para a fonte do rótulo atual.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **HSCROLL do WM \_**          | Rola o controle de exibição de lista horizontalmente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **o WM \_ KEYDOWN**          | Processa as teclas de barra de espaços, ENTER e seta e envia um código de notificação [LVN \_ KEYDOWN](lvn-keydown.md) para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **KILLFOCUS do WM \_**        | Redesenha o item de lista focado, se houver, e envia um código de notificação [nm \_ KILLFOCUS](nm-killfocus.md) para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **LBUTTONDBLCLK do WM \_**    | Envia a janela pai um código de notificação [ \_ DBLCLK nm (exibição de lista)](nm-dblclk-list-view.md) .                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **LBUTTONDOWN do WM \_**      | Processado de maneiras diferentes, dependendo se uma operação de clicar ou arrastar está sendo iniciada. Para determinar qual operação está envolvida, o controle de exibição de lista entra em um loop de mensagem modal até que o botão seja liberado ou o mouse seja movido. No caso de um clique, o controle List-View pode alterar qual item tem o foco e quais itens são selecionados, levando em conta a posição do cursor, o estado das teclas SHIFT e CTRL e assim por diante. Em seguida, o controle List-View envia sua janela pai um código de notificação de [clique de nm \_ (exibição de lista)](nm-click-list-view.md) .<br/> Se o arrasto começar sobre um item, o controle List-View selecionará e definirá o foco para o item. Em seguida, ele envia um código de notificação [LVN \_ BEGINDRAG](lvn-begindrag.md) para a janela pai. A janela pai é responsável por realmente realizar a operação de arrastar.<br/> Se arrastar começar sobre a tela de fundo da janela, o controle List-View entrará em outro loop de mensagem modal, permitindo que o usuário formate um retângulo arrastando o mouse. Os itens dentro do retângulo são selecionados.<br/> |
| **NCCREATE do WM \_**         | Aloca e Inicializa uma estrutura de dados interna e, em seguida, chama a função [**DefWindowProc**](/windows/desktop/api/winuser/nf-winuser-defwindowproca) .                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **NCDESTROY do WM \_**        | Libera recursos alocados pelo controle de exibição de lista. A menos que o estilo [**LVS \_ SHAREIMAGELISTS**](list-view-window-styles.md) seja usado, isso inclui a exclusão de listas de imagens pequenas e de tamanho completo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **notificação do WM \_**           | Processa códigos de notificação de controle de cabeçalho.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **pintura do WM \_**            | Exibe todos os itens na região de atualização. Para itens de retorno de chamada, o controle primeiro envia um código de notificação [LVN \_ GETDISPINFO](lvn-getdispinfo.md) para a janela do proprietário para solicitar informações de exibição. Se o parâmetro *wParam* for não nulo, o controle assumirá que o valor é um hDC e pinta usando esse contexto de dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **RBUTTONDOWN do WM \_**      | Processado da mesma maneira que a mensagem do [**WM \_ LBUTTONDOWN**](/windows/desktop/inputdev/wm-lbuttondown) , exceto pelo fato de que o controle envia um código de notificação [nm \_ RCLICK (exibição de lista)](nm-rclick-list-view.md) (em vez de [nm, \_ clique (exibição de lista)](nm-click-list-view.md)) e um código de notificação [ \_ BEGINRDRAG LVN](lvn-beginrdrag.md) (em vez de [LVN \_ BEGINDRAG](lvn-begindrag.md)). Observe que o controle processa a mensagem [**de \_ RBUTTONUP do WM**](/windows/desktop/inputdev/wm-rbuttonup) correspondente e não a despacha. Portanto, os aplicativos não podem ver essa mensagem, mesmo por meio da subclasse do controle.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM \_ SETFOCUS**         | Redesenha o item de lista focado, se houver, e envia um código de notificação [nm \_ SETFOCUS](nm-setfocus.md) para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WM \_ SETfont**          | Salva o identificador de fonte especificado, encaminha a mensagem para a janela de cabeçalho, se houver, e repinta usando a nova fonte.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM \_ reemissão**        | Ativa ou desativa o redesenho.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **\_temporizador do WM**            | Inicia a edição de um rótulo de item. Se o usuário clicar no rótulo do item focado, o controle List-View definirá um temporizador em vez de entrar no modo de edição imediatamente. O timer possibilita que o controle de exibição de lista *não* entre no modo de edição se o usuário clicar duas vezes no rótulo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **VSCROLL do WM \_**          | Rola o controle de exibição de lista verticalmente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **WINDOWPOSCHANGED do WM \_** | Atualiza as barras de rolagem da janela. Se a exibição atual for ícone ou exibição de ícone pequeno e o estilo de [**\_ AutoOrganizar LVS**](list-view-window-styles.md) for especificado, o controle de exibição de lista também organizará os itens de lista.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WININICHANGE do WM \_**     | Processa alterações nas métricas do sistema.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
