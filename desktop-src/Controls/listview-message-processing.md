# <a name="default-list-view-message-processing"></a>Processamento List-View mensagem padrão

Esta seção lista o processamento de mensagem de janela executado por um controle de exibição de lista.

## <a name="message-processing"></a>Processamento de mensagem

A tabela a seguir descreve o processamento de mensagem de janela executado por um controle de exibição de lista.



| Mensagem                  | Processamento executado                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WM \_ CHAR**             | Pesquisa um item de exibição de lista que começa com o caractere especificado e, se o item for encontrado, seleciona e define o foco para o item. Vários caracteres recebidos em um intervalo de tempo-out, atualmente corrigidos em um segundo, são concatenados e o controle de exibição de lista pesquisa um item que começa com a cadeia de caracteres resultante.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **COMANDO \_ WM**          | Processa os [ \_ códigos de](en-update.md) notificação EN UPDATE e [EN \_ KILLFOCUS](en-killfocus.md) e encaminha todas as outras notificações de controle de edição para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **WM \_ CREATE**           | Executa a inicialização adicional para [**WM \_ NCCREATE.**](/windows/desktop/winmsg/wm-nccreate) Se o estilo da janela [**LVS \_ SHAREIMAGELISTS**](list-view-window-styles.md) não for especificado, o controle de exibição de lista criará o ícone e as listas de imagens de ícone pequeno neste ponto.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **WM \_ DESTROY**          | Libera os recursos alocados durante [**o WM \_ CREATE.**](/windows/desktop/winmsg/wm-create)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM \_ ERASEBKGND**       | Apaga a tela de fundo da janela usando a cor da tela de fundo atual para o controle de exibição de lista. Se a cor da tela de fundo for o valor CLR NONE, o controle de exibição de lista \_ encaminhará a mensagem para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **WM \_ GETDLGCODE**       | Retorna uma combinação dos valores \_ WANTTAB e DLGC \_ WANTARROWS do DLGC.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WM \_ GETFONT**          | Retorna o identificador para a fonte de rótulo atual.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **WM \_ HSCROLL**          | Rola o controle de exibição de lista horizontalmente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **WM \_ KEYDOWN**          | Processa as teclas SPACEBAR, ENTER e seta e envia um código de notificação [LVN \_ KEYDOWN](lvn-keydown.md) para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **WM \_ KILLFOCUS**        | Reintsimos o item de lista focalizado, se algum, e envia um código de notificação [ \_ KILLFOCUS NM](nm-killfocus.md) para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **WM \_ LBUTTONDBLCLK**    | Envia à janela pai um [código de notificação \_ DBLCLK (exibição de lista) NM.](nm-dblclk-list-view.md)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **WM \_ LBUTTONDOWN**      | Processado de maneiras diferentes, dependendo se uma operação de clique ou arrastar está sendo iniciada. Para determinar qual operação está envolvida, o controle de exibição de lista entra em um loop de mensagem modal até que o botão seja liberado ou o mouse seja movido. No caso de um clique, o controle de exibição de lista pode alterar qual item tem o foco e quais itens são selecionados, levando em conta a posição do cursor, o estado das teclas SHIFT e CTRL e assim por diante. Em seguida, o controle de exibição de lista envia à janela pai um código de notificação [NM \_ CLICK (exibição de](nm-click-list-view.md) lista).<br/> Se arrastar começa em um item, o controle de exibição de lista seleciona e define o foco para o item. Em seguida, ele envia um [código de notificação \_ BEGINDRAG LVN](lvn-begindrag.md) para a janela pai. A janela pai é responsável por realmente realizar a operação de arrastar.<br/> Se o arrastar começar pela janela em segundo plano, o controle de exibição de lista entrará em outro loop de mensagem modal, permitindo que o usuário forme um retângulo arrastando o mouse. Itens dentro do retângulo são selecionados.<br/> |
| **WM \_ NCCREATE**         | Aloca e inicializa uma estrutura de dados interna e, em seguida, chama a [**função DefWindowProc.**](/windows/desktop/api/winuser/nf-winuser-defwindowproca)                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| **WM \_ NCDESTROY**        | Libera os recursos alocados pelo controle de exibição de lista. A menos que [**o estilo LVS \_ SHAREIMAGELISTS**](list-view-window-styles.md) seja usado, isso inclui a exclusão das listas de imagens de tamanho completo e pequeno.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **WM \_ NOTIFY**           | Processa códigos de notificação de controle de header.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WM \_ PAINT**            | Exibe todos os itens na região de atualização. Para itens de retorno de chamada, o controle primeiro envia um código de notificação [ \_ GETDISPINFO LVN](lvn-getdispinfo.md) para a janela do proprietário para solicitar informações de exibição. Se o *parâmetro wParam* for não NULL, o controle assumirá que o valor é um HDC e pintará usando esse contexto de dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **WM \_ RBUTTONDOWN**      | Processado da mesma maneira que a mensagem [**WM \_ LBUTTONDOWN,**](/windows/desktop/inputdev/wm-lbuttondown) exceto que o controle envia um código de notificação [NM \_ RCLICK (exibição](nm-rclick-list-view.md) de lista) (em vez de [NM CLICK \_ (exibição](nm-click-list-view.md)de lista) ) e um código de notificação [LVN \_ BEGINRDRAG](lvn-beginrdrag.md) (em vez [de LVN \_ BEGINDRAG).](lvn-begindrag.md) Observe que o controle processa a mensagem [**WM \_ RBUTTONUP**](/windows/desktop/inputdev/wm-rbuttonup) correspondente e não a expedi. Portanto, os aplicativos não podem ver essa mensagem, mesmo com a subclasse do controle .                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM \_ SETFOCUS**         | Reintsimos o item de lista focalizado, se algum, e envia um código de notificação [ \_ SETFOCUS NM](nm-setfocus.md) para a janela pai.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WM \_ SETFONT**          | Salva o alça de fonte especificado, encaminha a mensagem para a janela de header, se for o caso, e reintsifique usando a nova fonte.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM \_ SETREDRAW**        | Liga ou desliga o redesenho.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **TEMPORIZADOR \_ DE WM**            | Inicia a edição de um rótulo de item. Se o usuário clicar no rótulo do item focalizado, o controle de exibição de lista define um temporizador em vez de entrar no modo de edição imediatamente. O temporizador possibilita que o controle  de exibição de lista não entre no modo de edição se o usuário clicar duas vezes no rótulo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| **WM \_ VSCROLL**          | Rola o controle de exibição de lista verticalmente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **WM \_ WINDOWPOSCHANGED** | Atualiza as barras de rolagem da janela. Se a exibição atual for ícone ou exibição de ícone pequeno e o estilo [**LVS \_ AUTOARRANGE**](list-view-window-styles.md) for especificado, o controle de exibição de lista também organizará os itens de lista.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WM \_ WININICHANGE**     | Processa alterações nas métricas do sistema.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
