# <a name="cbem_getcombocontrol-message"></a>\_Mensagem CBEM GETCOMBOCONTROL

Obtém o identificador para o controle da caixa de combinação filho.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Valor retornado

Retorna o identificador para o controle de caixa de combinação dentro do controle ComboBoxEx.
