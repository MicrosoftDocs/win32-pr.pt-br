# <a name="lb_setcolumnwidth-message"></a>SETCOLUMNWIDTH de mensagens de LB \_

Define a largura, em pixels, de todas as colunas em uma caixa de listagem de várias colunas.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Especifica a largura, em pixels, de todas as colunas.

</dd> <dt>

*lParam* 
</dt> <dd>

Este parâmetro não é usado.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Essa mensagem não retorna um valor.
