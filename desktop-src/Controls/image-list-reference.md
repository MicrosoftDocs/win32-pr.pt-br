# <a name="image-lists"></a>Listas de imagens

Esta seção contém informações sobre os elementos de programação usados com listas de imagens.

### <a name="overviews"></a>Visões gerais



| Tópico                          | Sumário                                                                                                            |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [Listas de imagens](image-lists.md) | Uma lista de imagens é uma coleção de imagens do mesmo tamanho, cada uma das quais pode ser referenciada por seu índice.<br/> |
