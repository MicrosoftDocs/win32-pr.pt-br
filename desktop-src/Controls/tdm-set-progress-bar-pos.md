# <a name="tdm_set_progress_bar_pos-message"></a>\_Mensagem de \_ PDV da barra de progresso do conjunto TDM \_ \_

Define a posição da barra de progresso em uma caixa de diálogo de tarefa.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* \[ no\]
</dt> <dd>

Um **int** que especifica a nova posição.

</dd> <dt>

*lParam* \[ no\]
</dt> <dd>

Deve ser zero.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retorna a posição anterior.
