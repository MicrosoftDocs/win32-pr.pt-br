# <a name="rb_enddrag-message"></a>Mensagem de fim de operação RB \_

Encerra a operação de arrastar e soltar do controle rebar. Esta mensagem não faz com que uma notificação [RBN \_ endarraste](rbn-enddrag.md) seja enviada.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

O valor retornado para esta mensagem não é usado.
