# <a name="tree-view"></a>Modo de exibição de árvore

Esta seção contém informações sobre os elementos de programação usados com controles de exibição de árvore.

### <a name="overviews"></a>Visões gerais



| Tópico                                              | Sumário                                                                                                                                                                                    |
|----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre controles de Tree-View](tree-view-controls.md) | Um controle de exibição de árvore é uma janela que exibe uma lista hierárquica de itens, como os cabeçalhos de um documento, as entradas em um índice ou os arquivos e diretórios em um disco.<br/> |
| [Usando controles de Tree-View](using-treeview.md)     | Esta seção contém detalhes de implementação e código de exemplo para trabalhar com controles de exibição de árvore.<br/>                                                                               |
