# <a name="udm_setrange32-message"></a>\_Mensagem de SETRANGE32 UDM

Define o intervalo de 32 bits de um controle acima-abaixo.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Valor inteiro assinado que representa o novo limite inferior do intervalo de controle acima-abaixo.

</dd> <dt>

*lParam* 
</dt> <dd>

Valor inteiro assinado que representa o novo limite superior do intervalo de controle acima-abaixo.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

O valor retornado para esta mensagem não é usado.
