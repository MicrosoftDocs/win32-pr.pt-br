# <a name="status-bar-styles"></a>Estilos da barra de status

Esta seção lista os estilos, além dos estilos de janela padrão, suportados pelos controles de *barra de status* .



| Constante                                                                                                                                                         | Descrição                                                                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SBARS_SIZEGRIP"></span><span id="sbars_sizegrip"></span><dl> <dt>**SBARS \_ SIZEGRIP**</dt> </dl> | O controle da barra de status incluirá uma alça de dimensionamento na extremidade direita da barra de status. Uma alça de dimensionamento é semelhante a uma borda de dimensionamento; é uma área retangular que o usuário pode clicar e arrastar para redimensionar a janela pai. <br/> |
| <span id="SBT_TOOLTIPS"></span><span id="sbt_tooltips"></span><dl> <dt>**\_dicas de ferramenta SBT**</dt> </dl>       | [Versão 4,71.](common-control-versions.md) Use este estilo para habilitar dicas de ferramentas.<br/>                                                                                                                                             |
| <span id="SBARS_TOOLTIPS"></span><span id="sbars_tooltips"></span><dl> <dt>**\_dicas de ferramenta SBARS**</dt> </dl> | [Versão 5,80.](common-control-versions.md) Idêntico ao SBT \_ tooltips. Use este sinalizador para as versões 5, 0 ou posteriores. <br/>                                                                                                         |
