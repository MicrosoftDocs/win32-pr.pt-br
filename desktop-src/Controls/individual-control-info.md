Os tópicos nesta seção discutem cada controle separadamente. As informações de cada controle contêm visões gerais que descrevem o controle e como usá-lo, bem como elementos de programação que são usados especificamente com esse controle.

| [Botão](buttons.md)                                              | Um *botão* é um controle no qual o usuário pode clicar para fornecer entrada para um aplicativo. | [Lista de imagens](image-list-reference.md)                             | Uma *lista de imagens* é uma coleção de imagens do mesmo tamanho, cada uma das quais pode ser referenciada por seu índice. | [Rebar](rebar-control-reference.md)                               | Um *controle rebar* atua como um contêiner para janelas filhas. Esta seção contém informações sobre os elementos de programação usados com controles de edição avançados.<br/>                                                                 |
| [Barra de rolagem](scroll-bars.md)                                      | Uma *barra de rolagem* permite que o usuário Coloque em exibição as partes de um objeto que se estendem além das bordas de uma janela. | [SysLink](syslink-control-entry.md)                               | Um *controle Syslink* é uma janela que renderiza o texto marcado para cima e notifica o aplicativo quando o usuário clica em um link inserido. Esta seção contém informações sobre os elementos de programação usados com controles de guia.<br/>                                                                      |

Esta seção contém informações sobre os elementos de programação usados com as caixas de diálogo de tarefas.<br/> |

Esta seção contém informações sobre os elementos de programação usados com trackbars.<br/>                  |
