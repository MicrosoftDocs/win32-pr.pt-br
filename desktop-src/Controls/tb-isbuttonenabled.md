# <a name="tb_isbuttonenabled-message"></a>TB de \_ mensagem ISBUTTONENABLED

Determina se o botão especificado em uma barra de ferramentas está habilitado.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador de comando do botão.

</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Valor retornado

Retornará zero se o botão estiver habilitado ou zero caso contrário.
