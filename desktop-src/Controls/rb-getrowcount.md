# <a name="rb_getrowcount-message"></a>\_Mensagem RB GETrowcount

Recupera o número de linhas de faixas em um controle rebar.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retorna um valor **uint** que representa o número de linhas de banda no controle.
