# <a name="rebar-sample"></a>Exemplo de barra de rebar

Este tópico descreve o exemplo de código de exemplo rebar. Ele contém as seções a seguir:

-   [Descrição](#description)
-   [Requisitos mínimos](#minimum-requirements)
-   [Baixando o exemplo](#downloading-the-sample)
-   [Compilando o exemplo](#building-the-sample)
-   [Tópicos relacionados](#related-topics)

## <a name="description"></a>Descrição

O Exemplo de barra de rebar mostra como implementar um controle comum de barra rebar simples em um aplicativo. Este exemplo cria um controle de barra de rebar que tem duas faixas. Uma contém uma caixa de combinação e a outra contém um botão de push.

## <a name="minimum-requirements"></a>Requisitos mínimos



| Produto          | Versão                               |
|------------------|---------------------------------------|
| DLL              | comctl32.dll versão 4.71             |
| Sistema operacional | Windows 95 com Internet Explorer 4.0 |



 

## <a name="downloading-the-sample"></a>Baixando o exemplo

O Exemplo de Barra de Rebar é instalado como parte [do SDK (Software Development Kit) do Windows](https://msdn.microsoft.com/windows/bb980924.aspx) e está disponível no local a seguir.



| Localização    | Caminho/URL                                                                                              |
|-------------|-------------------------------------------------------------------------------------------------------|
| SDK do Windows | %Arquivos de Programas% \\ SDKs da Microsoft Windows número de versão Exemplos de barras comuns \\ de \\ \[ \] \\ \\ controles \\ \\ \\ winui |



 

## <a name="building-the-sample"></a>Compilando o exemplo

Para criar o exemplo usando o prompt de comando:

1.  Abra a janela Prompt de Comando e navegue até o diretório do projeto.
2.  Digite `msbuild [project file]`.

Para criar o exemplo usando Visual Studio:

1.  Abra Windows Explorer e navegue até o diretório do projeto.
2.  Clique duas vezes no ícone do arquivo .vcproj para abrir o projeto Visual Studio.
3.  No menu **Build,** selecione **Criar Solução** para criar a solução.
