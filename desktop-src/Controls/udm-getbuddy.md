# <a name="udm_getbuddy-message"></a>Mensagem GETBUDDY do UDM \_

Recupera o alça para a janela atual do parceiro.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Valor retornado

O valor de retorno é o alça para a janela de parceiro atual.
