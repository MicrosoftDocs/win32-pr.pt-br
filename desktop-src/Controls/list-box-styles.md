# <a name="list-box-styles"></a>Estilos de caixa de listagem

Para criar uma caixa de listagem usando a função [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) ou [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , use a classe ListBox, as constantes de estilo de janela apropriadas e as seguintes constantes de estilo para definir a caixa de listagem. Depois que o controle tiver sido criado, esses estilos não poderão ser modificados, exceto quando observado.



| Constante                                                                                                                                                                              | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**\_ComboBox kg**</dt> </dl>                            | Notifica uma caixa de listagem de que ela faz parte de uma caixa de combinação. Isso permite a coordenação entre os dois controles para que eles apresentem uma interface do usuário unificada. A própria caixa de combinação deve definir esse estilo. Se o estilo for definido por qualquer coisa, exceto a caixa de combinação, a caixa de listagem se considerará incorretamente como um filho de uma caixa de combinação e ocorrerá uma falha.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**LBS \_ DISABLENOSCROLL**</dt> </dl>       | Mostra uma barra de rolagem horizontal ou vertical desabilitada quando a caixa de listagem não contém itens suficientes para rolar. Se você não especificar esse estilo, a barra de rolagem ficará oculta quando a caixa de listagem não contiver itens suficientes. Esse estilo deve ser usado com o estilo [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles) ou [**WS \_ HSCROLL**](/windows/desktop/winmsg/window-styles) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**LBS \_ EXTENDEDSEL**</dt> </dl>                   | Permite que vários itens sejam selecionados usando a tecla SHIFT e as combinações de teclas especiais e do mouse.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**LBS \_ HASSTRINGS**</dt> </dl>                      | Especifica que uma caixa de listagem contém itens que consistem em cadeias de caracteres. A caixa de listagem mantém a memória e os endereços das cadeias de caracteres para que o aplicativo possa usar a mensagem [**\_ gettext do lb**](lb-gettext.md) para recuperar o texto de um item específico. Por padrão, todas as caixas de listagem, exceto as caixas de listagem desenhadas pelo proprietário, têm esse estilo. Você pode criar uma caixa de listagem desenhada pelo proprietário com ou sem esse estilo.<br/> Para as caixas de listagem desenhadas pelo proprietário sem esse estilo, a mensagem de [**\_ gettext do lb**](lb-gettext.md) recupera o valor associado a um item (os dados do item). <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**\_várias colunas lbs**</dt> </dl>                   | Especifica uma caixa de listagem de várias colunas que é rolada horizontalmente. A caixa de listagem calcula automaticamente a largura das colunas ou um aplicativo pode definir a largura usando a mensagem SETCOLUMNWIDTH de [**lb \_**](lb-setcolumnwidth.md) . Se uma caixa de listagem tiver o estilo de **lbs \_ OwnerDrawFixed** , um aplicativo poderá definir a largura quando a caixa de listagem enviar a mensagem do [**WM \_ MEASUREITEM**](wm-measureitem.md) . <br/> Uma caixa de listagem com o estilo de **\_ várias colunas lbs** não pode rolar verticalmente, ignora todas as mensagens [**\_ VSCROLL do WM**](wm-vscroll.md) recebidas. <br/> Os estilos de **\_ OwnerDrawVariable** de **\_ várias colunas lbs** e lbs não podem ser combinados. Se ambos forem especificados, **lbs \_ OwnerDrawVariable** será ignorado.<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**LBS \_ MULTIPLESEL**</dt> </dl>                   | Ativa ou desativa a seleção de cadeia de caracteres cada vez que o usuário clica ou clica duas vezes em uma cadeia de caracteres na caixa de listagem. O usuário pode selecionar qualquer número de cadeias de caracteres.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**LBS \_ NOdata**</dt> </dl>                                  | Especifica uma caixa de listagem sem dados. Especifique esse estilo quando a contagem de itens na caixa de listagem exceder 1000. Uma caixa de listagem sem dados também deve ter o estilo de **lbs \_ OwnerDrawFixed** , mas não deve ter o estilo de **\_ HASSTRINGS** de **\_ classificação lbs** ou lbs. <br/> Uma caixa de listagem sem dados é semelhante a uma caixa de listagem desenhada pelo proprietário, exceto que ela não contém dados de cadeia de caracteres ou de bitmap para um item. Comandos para adicionar, inserir ou excluir um item sempre ignoram qualquer dado de item especificado; as solicitações para localizar uma cadeia de caracteres dentro da caixa de listagem sempre falham. O sistema envia a mensagem do [**WM \_ DRAWITEM**](wm-drawitem.md) para a janela do proprietário quando um item deve ser desenhado. O membro **ItemId** da estrutura [**DRAWITEMSTRUCT**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) passada com a mensagem **\_ DRAWITEM do WM** especifica o número de linha do item a ser desenhado. Uma caixa de listagem sem dados não envia uma mensagem do [**WM \_ DELETEITEM**](wm-deleteitem.md) .<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**LBS \_ NOINTEGRALHEIGHT**</dt> </dl>    | Especifica que o tamanho da caixa de listagem é exatamente o tamanho especificado pelo aplicativo quando ele criou a caixa de listagem. Normalmente, o sistema dimensiona uma caixa de listagem para que a caixa de listagem não exiba itens parciais.<br/> Para caixas de listagem com o estilo de **lbs \_ OwnerDrawVariable** , o estilo de **lbs \_ NOINTEGRALHEIGHT** é sempre imposto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**LBS \_ redesenhar**</dt> </dl>                            | Especifica que a aparência da caixa de listagem não é atualizada quando são feitas alterações.<br/> Para alterar o estado de redesenho do controle, use a mensagem de [**\_ reemissão do WM**](/windows/desktop/gdi/wm-setredraw) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**nariz de LBS \_**</dt> </dl>                                     | Especifica que a caixa de listagem contém itens que podem ser exibidos, mas não selecionados. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**LBS- \_ notificação**</dt> </dl>                                  | Faz com que a caixa de listagem envie um código de notificação para a janela pai sempre que o usuário clicar em um item da caixa de listagem ([lbn \_ SELCHANGE](lbn-selchange.md)), clicar duas vezes em um item ([lbn \_ DBLCLK](lbn-dblclk.md)) ou cancelar a seleção ([lbn \_ ](lbn-selcancel.md)SELCANCEL). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**LBS \_ OwnerDrawFixed**</dt> </dl>          | Especifica que o proprietário da caixa de listagem é responsável por desenhar seu conteúdo e que os itens na caixa de listagem têm a mesma altura. A janela do proprietário recebe uma mensagem do [**WM \_ MEASUREITEM**](wm-measureitem.md) quando a caixa de listagem é criada e uma mensagem do [**WM \_ DRAWITEM**](wm-drawitem.md) quando um aspecto visual da caixa de listagem é alterado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**LBS \_ OwnerDrawVariable**</dt> </dl> | Especifica que o proprietário da caixa de listagem é responsável por desenhar seu conteúdo e que os itens na caixa de listagem são variáveis em altura. A janela do proprietário recebe uma mensagem do [**WM \_ MEASUREITEM**](wm-measureitem.md) para cada item na caixa quando a caixa de listagem é criada e uma mensagem do [**WM \_ DRAWITEM**](wm-drawitem.md) quando um aspecto visual da caixa de listagem é alterado.<br/> Esse estilo faz com que o estilo de **lbs \_ NOINTEGRALHEIGHT** seja habilitado.<br/> Esse estilo será ignorado se o estilo de **\_ várias colunas de lbs** for especificado. <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**classificação de LBS \_**</dt> </dl>                                        | Classifica as cadeias de caracteres na caixa de listagem em ordem alfabética.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**LBS \_ padrão**</dt> </dl>                            | Classifica as cadeias de caracteres na caixa de listagem em ordem alfabética. A janela pai recebe um código de notificação sempre que o usuário clica em um item da caixa de listagem, clica duas vezes em um item ou cancela a seleção. A caixa de listagem tem uma barra de rolagem vertical e tem bordas em todos os lados. Esse estilo combina os estilos de VSCROLL de **\_ notificação lbs**, **lbs \_ Sort**, [**WS \_**](/windows/desktop/winmsg/window-styles)e [**WS \_ Border**](/windows/desktop/winmsg/window-styles) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**LBS \_ USETABSTOPS**</dt> </dl>                   | Permite que uma caixa de listagem reconheça e expanda caracteres de tabulação ao desenhar suas cadeias. Você pode usar a mensagem do [**lb \_ SetTabStops**](lb-settabstops.md) para especificar posições de parada de tabulação. As posições de guia padrão são unidades de modelo de caixa de diálogo 32 separadas. As unidades de modelo de caixa de diálogo são as unidades independentes de dispositivo usadas em modelos de caixas de diálogo. Para converter medidas de unidades de modelo de caixa de diálogo em unidades de tela (pixels), use a função [**MapDialogRect**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**LBS \_ WANTKEYBOARDINPUT**</dt> </dl> | Especifica que o proprietário da caixa de listagem recebe mensagens do [**WM \_ VKEYTOITEM**](wm-vkeytoitem.md) sempre que o usuário pressiona uma tecla e a caixa de listagem tem o foco de entrada. Isso permite que um aplicativo execute processamento especial na entrada do teclado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
