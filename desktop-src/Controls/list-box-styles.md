# <a name="list-box-styles"></a>Estilos de caixa de listagem

Para criar uma caixa de listagem usando a função [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) ou [**CreateWindowEx,**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) use a classe LISTBOX, as constantes de estilo de janela apropriadas e as constantes de estilo a seguir para definir a caixa de listagem. Depois que o controle tiver sido criado, esses estilos não poderão ser modificados, exceto conforme o que foi feito.



| Constante                                                                                                                                                                              | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**LBS \_ COMBOBOX**</dt> </dl>                            | Notifica uma caixa de listagem de que ela faz parte de uma caixa de combinação. Isso permite a coordenação entre os dois controles para que eles apresentem uma interface do usuário unificada. A própria caixa de combinação deve definir esse estilo. Se o estilo for definido por qualquer coisa, menos a caixa de combinação, a caixa de listagem se considerará incorretamente como um filho de uma caixa de combinação e ocorrerá uma falha.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**LBS \_ DISABLENOSCROLL**</dt> </dl>       | Mostra uma barra de rolagem horizontal ou vertical desabilitada quando a caixa de listagem não contém itens suficientes para rolar. Se você não especificar esse estilo, a barra de rolagem será ocultada quando a caixa de listagem não contém itens suficientes. Esse estilo deve ser usado com o [**estilo \_ WS VSCROLL**](/windows/desktop/winmsg/window-styles) ou [**WS \_ HSCROLL.**](/windows/desktop/winmsg/window-styles)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**LBS \_ EXTENDEDSEL**</dt> </dl>                   | Permite que vários itens sejam selecionados usando a tecla SHIFT e o mouse ou combinações de teclas especiais.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**LBS \_ HASSTRINGS**</dt> </dl>                      | Especifica que uma caixa de listagem contém itens que consistem em cadeias de caracteres. A caixa de listagem mantém a memória e os endereços das cadeias de caracteres para que o aplicativo possa usar a mensagem [**\_ LB GETTEXT**](lb-gettext.md) para recuperar o texto de um item específico. Por padrão, todas as caixas de listagem, exceto caixas de listagem desenhadas pelo proprietário, têm esse estilo. Você pode criar uma caixa de listagem desenhada pelo proprietário com ou sem esse estilo.<br/> Para caixas de listagem desenhadas pelo proprietário sem esse estilo, a mensagem [**\_ GETTEXT**](lb-gettext.md) do LB recupera o valor associado a um item (os dados do item). <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**LBS \_ MULTICOLUMN**</dt> </dl>                   | Especifica uma caixa de listagem de várias colunas que é rolada horizontalmente. A caixa de listagem calcula automaticamente a largura das colunas ou um aplicativo pode definir a largura usando a [**mensagem LB \_ SETCOLUMNWIDTH.**](lb-setcolumnwidth.md) Se uma caixa de listagem tiver o estilo **LBS \_ OWNERDRAWFIXED,** um aplicativo poderá definir a largura quando a caixa de listagem enviar a mensagem [**WM \_ MEASUREITEM.**](wm-measureitem.md) <br/> Uma caixa de listagem com o estilo **\_ LBS MULTICOLUMN** não pode rolar verticalmente, ela ignora as mensagens [**WM \_ VSCROLL**](wm-vscroll.md) recebidas. <br/> Os **estilos LBS \_ MULTICOLUMN** e **LBS \_ OWNERDRAWVARIABLE** não podem ser combinados. Se ambos são especificados, **LBS \_ OWNERDRAWVARIABLE** é ignorado.<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**LBS \_ MULTIPLESEL**</dt> </dl>                   | Liga ou desliga a seleção de cadeia de caracteres sempre que o usuário clica ou clica duas vezes em uma cadeia de caracteres na caixa de listagem. O usuário pode selecionar qualquer número de cadeias de caracteres.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**LBS \_ NODATA**</dt> </dl>                                  | Especifica uma caixa de listagem sem dados. Especifique esse estilo quando a contagem de itens na caixa de listagem exceder um mil. Uma caixa de listagem sem dados também deve ter o estilo **LBS \_ OWNERDRAWFIXED,** mas não deve ter o estilo **LBS \_ SORT** ou **\_ LBS HASSTRINGS.** <br/> Uma caixa de listagem sem dados é semelhante a uma caixa de listagem desenhada pelo proprietário, exceto que ela não contém dados de cadeia de caracteres ou bitmap para um item. Comandos para adicionar, inserir ou excluir um item sempre ignoram todos os dados de item especificados; solicitações para encontrar uma cadeia de caracteres dentro da caixa de listagem sempre falham. O sistema envia a [**mensagem WM \_ DRAWITEM**](wm-drawitem.md) para a janela do proprietário quando um item deve ser desenhado. O **membro itemID** da estrutura [**DRAWITEMSTRUCT**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) passada com a mensagem **WM \_ DRAWITEM** especifica o número de linha do item a ser desenhado. Uma caixa de listagem sem dados não envia uma mensagem [**WM \_ DELETEITEM.**](wm-deleteitem.md)<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**LBS \_ NOINTEGRALHEIGHT**</dt> </dl>    | Especifica que o tamanho da caixa de listagem é exatamente o tamanho especificado pelo aplicativo quando ele criou a caixa de listagem. Normalmente, o sistema tamanhou uma caixa de listagem para que a caixa de listagem não exibe itens parciais.<br/> Para caixas de listagem com o estilo **\_ LBS OWNERDRAWVARIABLE,** o estilo **\_ NOINTEGRALHEIGHT lbS** é sempre imposto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**LBS \_ NOREDRAW**</dt> </dl>                            | Especifica que a aparência da caixa de listagem não é atualizada quando as alterações são feitas.<br/> Para alterar o estado de redesenhar do controle, use a [**mensagem WM \_ SETREDRAW.**](/windows/desktop/gdi/wm-setredraw)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**LBS \_ NOSEL**</dt> </dl>                                     | Especifica que a caixa de listagem contém itens que podem ser exibidos, mas não selecionados. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**LBS \_ NOTIFY**</dt> </dl>                                  | Faz com que a caixa de listagem envie um código de notificação para a janela pai sempre que o usuário clicar em um item de caixa de listagem ([LBN \_ SELCHANGE](lbn-selchange.md)), clicar duas vezes em um item ([LBN \_ DBLCLK](lbn-dblclk.md)) ou cancelar a seleção ([LBN \_ SELCANCEL](lbn-selcancel.md)). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**LBS \_ OWNERDRAWFIXED**</dt> </dl>          | Especifica que o proprietário da caixa de listagem é responsável por desenhar seu conteúdo e que os itens na caixa de listagem têm a mesma altura. A janela proprietário recebe uma mensagem [**WM \_ MEASUREITEM**](wm-measureitem.md) quando a caixa de listagem é criada e uma mensagem [**WM \_ DRAWITEM**](wm-drawitem.md) quando um aspecto visual da caixa de listagem é alterado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**LBS \_ OWNERDRAWVARIABLE**</dt> </dl> | Especifica que o proprietário da caixa de listagem é responsável por desenhar seu conteúdo e que os itens na caixa de listagem são variáveis de altura. A janela proprietário recebe uma mensagem [**WM \_ MEASUREITEM**](wm-measureitem.md) para cada item na caixa quando a caixa de listagem é criada e uma mensagem [**WM \_ DRAWITEM**](wm-drawitem.md) quando um aspecto visual da caixa de listagem é alterado.<br/> Esse estilo faz com que **o \_ estilo NOINTEGRALHEIGHT lbS** seja habilitado.<br/> Esse estilo será ignorado se o estilo **\_ LBS MULTICOLUMN** for especificado. <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**CLASSIFICAÇÃO DE \_ LBS**</dt> </dl>                                        | Classifica cadeias de caracteres na caixa de listagem em ordem alfabética.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**LBS \_ STANDARD**</dt> </dl>                            | Classifica cadeias de caracteres na caixa de listagem em ordem alfabética. A janela pai recebe um código de notificação sempre que o usuário clica em um item de caixa de listagem, clica duas vezes em um item ou cancela a seleção. A caixa de listagem tem uma barra de rolagem vertical e tem bordas em todos os lados. Esse estilo combina os estilos **LBS \_ NOTIFY,** **LBS \_ SORT,** [**\_ WS VSCROLL**](/windows/desktop/winmsg/window-styles)e [**WS \_ BORDER.**](/windows/desktop/winmsg/window-styles)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**LBS \_ USETABSTOPS**</dt> </dl>                   | Permite que uma caixa de listagem reconheça e expanda caracteres de tabulação ao desenhar suas cadeias de caracteres. Você pode usar a [**mensagem \_ LB SETTABSTOPS**](lb-settabstops.md) para especificar posições de parada de tabulação. As posições de tabulação padrão têm 32 unidades de modelo de caixa de diálogo separadas. Unidades de modelo de caixa de diálogo são as unidades independentes de dispositivo usadas em modelos de caixa de diálogo. Para converter medidas de unidades de modelo de caixa de diálogo em unidades de tela (pixels), use a [**função MapDialogRect.**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**LBS \_ WANTKEYBOARDINPUT**</dt> </dl> | Especifica que o proprietário da caixa de listagem recebe mensagens [**\_ WM VKEYTOITEM**](wm-vkeytoitem.md) sempre que o usuário pressiona uma tecla e a caixa de listagem tem o foco de entrada. Isso permite que um aplicativo execute processamento especial na entrada do teclado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
