## <a name="remarks"></a>Comentários

> [!Note]  
> Comctl32.dll versão 6 não é redistribuível, mas está incluído no Windows. Para usar Comctl32.dll versão 6, especifique-a em um manifesto. Para obter mais informações sobre manifestos, consulte [Habilitando estilos visuais.](cookbook-overview.md)

 

Um item fica *quente* quando o mouse passa o mouse sobre ele. Se o acompanhamento a quente estiver habilitado, os itens ativos serão realçadas, mas o usuário ainda deverá clicar no item pelo menos uma vez para selecioná-lo. *Ativar um* item faz com que o controle de exibição de lista envie uma [notificação LVN \_ ITEMACTIVATE.](lvn-itemactivate.md) Um item pode ser *selecionado* quando ele está em um estado no qual um único clique o selecionará.

Se o **estilo LVS \_ EX \_ ONECLICKACTIVATE** estiver definido, será necessário apenas um clique para selecionar qualquer item, de modo que todos os itens possam ser selecionados. Se o **estilo LVS \_ EX \_ TWOCLICKACTIVATE** estiver definido, serão necessários dois cliques para selecionar um item. Com esse estilo, o item pode ser selecionado somente depois de ter sido clicado uma vez.

Definir o **estilo \_ \_ UNDERLINEHOT do LVS EX** fará com que o texto de qualquer item quente que possa ser selecionado seja sublinhado. Definir o **estilo \_ sublinhado \_ LVS EX** fará com que o texto de itens que não estão ativos, mas que podem ser selecionados, seja sublinhado.
