## <a name="remarks"></a>Comentários

> [!Note]  
> O Comctl32.dll versão 6 não é redistribuível, mas está incluído no Windows. Para usar Comctl32.dll versão 6, especifique-o em um manifesto. Para obter mais informações sobre manifestos, consulte [habilitando estilos visuais](cookbook-overview.md).

 

Um item fica *quente* quando o mouse passa sobre ele. Se o acompanhamento de quente estiver habilitado, os itens ativos serão realçados, mas o usuário ainda deverá clicar no item pelo menos uma vez para selecioná-lo. *Ativar* um item faz com que o controle de exibição de lista envie uma notificação de [LVN \_](lvn-itemactivate.md) . Um item pode ser *selecionado* quando estiver em um estado no qual um único clique o selecionará.

Se o estilo **LVS \_ ex \_ ONECLICKACTIVATE** for definido, apenas um clique será necessário para selecionar qualquer item, para que todos os itens possam ser selecionados. Se o estilo **LVS \_ ex \_ TWOCLICKACTIVATE** for definido, dois cliques serão necessários para selecionar um item. Com esse estilo, o item pode ser selecionado somente depois que ele tiver sido clicado uma vez.

Definir o estilo **LVS \_ ex \_ UNDERLINEHOT** fará com que o texto de qualquer item ativo que possa ser selecionado seja sublinhado. Definir o estilo **LVS \_ ex \_ UNDERLINECOLD** fará com que o texto dos itens que não são quentes, mas pode ser selecionado, seja sublinhado.
