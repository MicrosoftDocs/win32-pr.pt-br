# <a name="visual-styles"></a>Estilos visuais

## <a name="purpose"></a>Finalidade

Os *estilos visuais* alteram a aparência de controles comuns com base no tema escolhido pelo usuário. Antes do Windows 8, você deve configurar especificamente seu aplicativo para usar estilos visuais; caso contrário, os controles comuns do aplicativo sempre são renderizados no estilo associado ao tema clássico do Windows, independentemente do tema atualmente selecionado. No Windows 8, os estilos visuais não podem ser desativados, o modo clássico do Windows não existe mais e o modo de alto contraste foi modificado para funcionar com estilos visuais.

Esta seção fornece uma visão geral dos estilos visuais e explica como configurar seu aplicativo para usá-los.

## <a name="developer-audience"></a>Público de desenvolvedores

Os estilos visuais são projetados para uso por desenvolvedores de C/C++ e designers de interface do usuário. Em geral, os desenvolvedores precisam de um nível moderado de compreensão sobre conceitos de programação de interface do usuário, programação de API do Windows e Unicode.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Windows Vista e sistemas operacionais posteriores.

> [!Note]  
> Não há suporte para estilos visuais no modo de cor 256.
