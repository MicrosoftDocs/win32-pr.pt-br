# <a name="rb_settooltips-message"></a>\_Mensagem de GETtooltips de RB

Associa um controle ToolTip ao controle rebar.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador para o controle de dica de ferramenta a ser definido.

</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

O valor retornado para esta mensagem não é usado.
