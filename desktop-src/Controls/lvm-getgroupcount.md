# <a name="lvm_getgroupcount-message"></a>\_Mensagem GETGROUPCOUNT LVM

Obtém o número de grupos.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* \[ fora\]
</dt> <dd>

Não usado. Deve ser 0.

</dd> <dt>

*lParam* 
</dt> <dd>

Não usado. Deve ser 0.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retorna o número de grupos.
