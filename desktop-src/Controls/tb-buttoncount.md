# <a name="tb_buttoncount-message"></a>TB de \_ mensagem BUTTONCOUNT

Recupera uma contagem dos botões atualmente na barra de ferramentas.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retorna a contagem dos botões.
