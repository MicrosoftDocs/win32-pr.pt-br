# <a name="visual-styles-reference"></a>Referência de estilos visuais

Esta seção descreve os seguintes elementos de API usados com [estilos visuais](themes-overview.md).

### <a name="functions"></a>Funções



| Tópico                                                                                  | Sumário                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BeginBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedanimation)                               | Inicia uma operação de animação em buffer. A animação consiste em um esmaecimento cruzado entre o conteúdo de dois buffers durante um período de tempo especificado. <br/>                                                                     |
| [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint)                                       | Inicia uma operação de pintura em buffer.<br/>                                                                                                                                                                                      |
| [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback)                                   | Notifica o sistema para enviar comentários sobre uma janela de destino afetada pelos gestos de panorâmica. <br/>                                                                                                                               |
| [**BufferedPaintClear**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintclear)                                       | Limpa um retângulo especificado no buffer para ARGB = {0,0,0,0} .<br/>                                                                                                                                                         |
| [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit)                                         | Inicializar a pintura em buffer para o thread atual.<br/>                                                                                                                                                                    |
| [**BufferedPaintRenderAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintrenderanimation)                   | Pinta o próximo quadro de uma animação de pintura em buffer.<br/>                                                                                                                                                                    |
| [**BufferedPaintSetAlpha**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintsetalpha)                                 | Define o alfa para um valor especificado em um determinado retângulo. O alfa controla a quantidade de transparência aplicada durante a mesclagem com o buffer no contexto do dispositivo de destino (DC) de destino. <br/>                         |
| [**BufferedPaintStopAllAnimations**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintstopallanimations)               | Interrompe todas as animações em buffer para a janela especificada.<br/>                                                                                                                                                                     |
| [**BufferedPaintUnInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintuninit)                                     | Fecha a pintura em buffer para o thread atual. Chamado uma vez para cada chamada para [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit) depois que as chamadas para [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint) não são mais necessárias.<br/> |
| [**CloseThemeData**](/windows/desktop/api/Uxtheme/nf-uxtheme-closethemedata)                                               | Fecha o identificador de dados do tema.<br/>                                                                                                                                                                                           |
| [**DrawThemeBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackground)                                     | Desenha a borda e o preenchimento definidos pelo estilo visual para a parte de controle especificada.<br/>                                                                                                                                   |
| [**DrawThemeBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackgroundex)                                 | Desenha a imagem de plano de fundo definida pelo estilo visual para a parte de controle especificada.<br/>                                                                                                                                  |
| [**DrawThemeEdge**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeedge)                                                 | Desenha uma ou mais bordas definidas pelo estilo visual de um retângulo.<br/>                                                                                                                                                     |
| [**DrawThemeIcon**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeicon)                                                 | Desenha uma imagem de uma lista de imagens com o efeito de ícone definido pelo estilo visual.<br/>                                                                                                                                     |
| [**DrawThemeParentBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackground)                         | Desenha a parte de um controle pai que é coberto por um controle filho parcialmente transparente ou com mistura de alfa.<br/>                                                                                                           |
| [**DrawThemeParentBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackgroundex)                     | Usado por controles filho parcialmente transparentes ou alfanuméricos para desenhar a parte de seu pai na frente de onde eles aparecem. Envia uma mensagem do WM \_ ERASEBKGND seguida por um WM \_ fileclient.<br/>                             |
| [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext)                                                 | Desenha o texto usando a cor e a fonte definidos pelo estilo visual.<br/>                                                                                                                                                        |
| [**DrawThemeTextEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetextex)                                             | Desenha o texto usando a cor e a fonte definidos pelo estilo visual. Estende o [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) permitindo opções de formato de texto adicionais.<br/>                                                             |
| [**EnableThemeDialogTexture**](/windows/desktop/api/Uxtheme/nf-uxtheme-enablethemedialogtexture)                           | Habilita ou desabilita o estilo visual do plano de fundo de uma janela de caixa de diálogo.<br/>                                                                                                                                                   |
| [**EnableTheming**](/windows/desktop/api/Uxtheme/nf-uxtheme-enabletheming)                                                 | Habilita ou desabilita os estilos visuais para o usuário atual nas sessões atuais e posteriores.<br/>                                                                                                                               |
| [**EndBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedanimation)                                   | Renderiza o primeiro quadro de uma operação de animação em buffer e inicia o temporizador de animação.<br/>                                                                                                                               |
| [**EndBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedpaint)                                           | Conclui uma operação de pintura em buffer e libera o identificador de pintura armazenado em buffer associado.<br/>                                                                                                                                    |
| [**EndPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-endpanningfeedback)                                       | Encerra qualquer animação existente que estava em processo ou configurada por [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback) e [**UpdatePanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-updatepanningfeedback). <br/>                                    |
| [**GetBufferedPaintBits**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintbits)                                   | Recupera um ponteiro para o bitmap de buffer se o buffer for um bitmap independente de dispositivo (DIB).<br/>                                                                                                                            |
| [**GetBufferedPaintDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintdc)                                       | Obtém o controlador de domínio de pintura. Esse é o mesmo valor recuperado por [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint).<br/>                                                                                                                |
| [**GetBufferedPaintTargetDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetdc)                           | Recupera o controlador de domínio de destino. <br/>                                                                                                                                                                                               |
| [**GetBufferedPaintTargetRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetrect)                       | Recupera o retângulo de destino especificado por BeginBufferedPaint.<br/>                                                                                                                                                         |
| [**GetCurrentThemeName**](/windows/desktop/api/Uxtheme/nf-uxtheme-getcurrentthemename)                                     | Recupera o nome do estilo visual atual e, opcionalmente, recupera o nome do esquema de cores e o nome do tamanho.<br/>                                                                                                           |
| [**GetThemeAppProperties**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeappproperties)                                 | Recupera os sinalizadores de propriedade que controlam como os estilos visuais são aplicados no aplicativo atual.<br/>                                                                                                                     |
| [**GetThemeBackgroundContentRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundcontentrect)                 | Recupera o tamanho da área de conteúdo para o plano de fundo definido pelo estilo visual.<br/>                                                                                                                                  |
| [**GetThemeBackgroundExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundextent)                           | Calcula o tamanho e o local do plano de fundo, definidos pelo estilo visual, considerando a área de conteúdo.<br/>                                                                                                                |
| [**GetThemeBackgroundRegion**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundregion)                           | Computa a região para um plano de fundo regular ou parcialmente transparente que é limitado por um retângulo especificado.<br/>                                                                                                         |
| [**GetThemeBitmap**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebitmap)                                               | Recupera o bitmap associado a um tema, parte, estado e Propriedade específicos.<br/>                                                                                                                                     |
| [**GetThemeBool**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebool)                                                   | Recupera o valor de uma propriedade **bool** da seção SysMetrics dos dados do tema.<br/>                                                                                                                                   |
| [**GetThemeColor**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemecolor)                                                 | Recupera o valor de uma Propriedade Color.<br/>                                                                                                                                                                                |
| [**GetThemeDocumentationProperty**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemedocumentationproperty)                 | Recupera o valor de uma propriedade Theme da seção de documentação do arquivo de tema especificado.<br/>                                                                                                                    |
| [**GetThemeEnumValue**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeenumvalue)                                         | Recupera o valor de uma propriedade de tipo enumerado.<br/>                                                                                                                                                                     |
| [**GetThemeFilename**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemefilename)                                           | Recupera o valor de uma propriedade FileName.<br/>                                                                                                                                                                             |
| [**GetThemeFont**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemefont)                                                   | Recupera o valor de uma propriedade de fonte. <br/>                                                                                                                                                                                |
| [**GetThemeInt**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeint)                                                     | Recupera o valor de uma propriedade **int** .<br/>                                                                                                                                                                             |
| [**GetThemeIntList**](/windows/desktop/api/UxTheme/nf-uxtheme-getthemeintlist)                                             | Recupera uma lista de dados **int** de um estilo visual.<br/>                                                                                                                                                                   |
| [**GetThemeMargins**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthememargins)                                             | Recupera o valor de uma propriedade [**Margins**](/windows/desktop/api/Uxtheme/ns-uxtheme-margins) .<br/>                                                                                                                                                           |
| [**GetThemeMetric**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthememetric)                                               | Recupera o valor de uma propriedade de métrica.<br/>                                                                                                                                                                               |
| [**GetThemePartSize**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemepartsize)                                           | Calcula o tamanho original da parte definida por um estilo visual.<br/>                                                                                                                                                     |
| [**GetThemePosition**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeposition)                                           | Recupera o valor de uma propriedade Position.<br/>                                                                                                                                                                             |
| [**GetThemePropertyOrigin**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemepropertyorigin)                               | Recupera o local da definição de propriedade de tema para uma propriedade. <br/>                                                                                                                                                |
| [**GetThemeRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemerect)                                                   | Recupera o valor de uma propriedade [**Rect**](/previous-versions//dd162897(v=vs.85)) .<br/>                                                                                                                                                                 |
| [**GetThemeStream**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemestream)                                               | Recupera um fluxo de dados correspondente a um tema especificado, a partir de uma parte, estado e Propriedade especificados.<br/>                                                                                                        |
| [**Getthemestring**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemestring)                                               | Recupera o valor de uma propriedade de cadeia de caracteres.<br/>                                                                                                                                                                               |
| [**GetThemeSysBool**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysbool)                                             | Recupera o valor booliano de uma métrica do sistema.<br/>                                                                                                                                                                         |
| [**GetThemeSysColor**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesyscolor)                                           | Recupera o valor de uma cor do sistema.<br/>                                                                                                                                                                                  |
| [**GetThemeSysColorBrush**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesyscolorbrush)                                 | Recupera um pincel de cor do sistema.<br/>                                                                                                                                                                                         |
| [**GetThemeSysFont**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysfont)                                             | Recupera o [**LOGFONT**](/windows/win32/api/wingdi/ns-wingdi-logfonta) de uma fonte do sistema.<br/>                                                                                                                                                              |
| [**GetThemeSysInt**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysint)                                               | Recupera o valor de um sistema **int**. <br/>                                                                                                                                                                               |
| [**GetThemeSysSize**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesyssize)                                             | Recupera o valor de uma métrica de tamanho do sistema dos dados do tema.<br/>                                                                                                                                                            |
| [**GetThemeSysString**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysstring)                                         | Recupera o valor de uma cadeia de caracteres do sistema. <br/>                                                                                                                                                                                |
| [**GetThemeTextExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemetextextent)                                       | Calcula o tamanho e o local do texto especificado quando renderizado na fonte do estilo visual.<br/>                                                                                                                          |
| [**GetThemeTextMetrics**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemetextmetrics)                                     | Recupera informações sobre a fonte especificada por um estilo visual para uma parte específica.<br/>                                                                                                                                 |

| [**IsThemePartDefined**](/windows/desktop/api/Uxtheme/nf-uxtheme-isthemepartdefined)                                       | Recupera se um estilo visual tem parâmetros definidos para a parte e o estado especificados.<br/>                                                                                                                               |
