# <a name="using-edit-controls"></a>Usando editar controles

Esta seção fornece informações e código de exemplo para implementar controles de edição.

Controles de edição normalmente são usados em caixas de diálogo, mas você também pode usá-los na área do cliente de uma janela padrão. Controles de edição de linha única são úteis para recuperar uma única cadeia de caracteres do usuário. Controles de edição multilinha permitem que você implemente a maioria dos recursos de um processador de palavras simples em seu aplicativo.

Os tópicos a seguir são discutidos.
