# <a name="using-edit-controls"></a>Usando controles de edição

Esta seção fornece informações e código de exemplo para implementação de controles de edição.

Os controles de edição normalmente são usados em caixas de diálogo, mas você também pode usá-los na área cliente de uma janela padrão. Os controles de edição de linha única são úteis para recuperar uma única cadeia de caracteres do usuário. Os controles de edição de várias linhas permitem que você implemente a maioria dos recursos de um processador de texto simples em seu aplicativo.

Os tópicos a seguir são discutidos.
