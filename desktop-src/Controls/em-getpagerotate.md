# <a name="em_getpagerotate-message"></a>\_Mensagem em GETPAGEROTATE

\[**Em \_ O GETPAGEROTATE** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.\]

Obtém o layout de texto de um controle de edição rico da Microsoft.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Não usado; deve ser zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Não usado; deve ser zero.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Obtém o layout de texto atual. Para obter uma lista de possíveis valores de layout de texto, consulte em [**\_ SETPAGEROTATE**](em-setpagerotate.md).
