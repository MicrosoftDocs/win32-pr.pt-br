# <a name="ttm_settiptextcolor-message"></a>\_Mensagem TTM SETTIPTEXTCOLOR

Define a cor do texto em uma janela de dica de ferramenta.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Nova cor do texto.

</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

O valor retornado para esta mensagem não é usado.

## <a name="remarks"></a>Comentários

Quando os estilos visuais são habilitados, essa mensagem não tem nenhum efeito.
