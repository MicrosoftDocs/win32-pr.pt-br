## <a name="remarks"></a>Comentários

Você usa esses valores para especificar um índice de imagem dentro de uma lista de imagens padrão que foi carregada com a [**mensagem \_ LOADIMAGES**](tb-loadimages.md) de TB. Os valores de índice correspondem a imagens dentro dos bitmaps padrão usados por controles comuns.

Cada um dos títulos a seguir contém uma lista de imagens dos ícones grandes, uma lista de imagens dos ícones pequenos e uma tabela que corresponde à constante de cada ícone com seu valor de índice.
