# <a name="em_isime-message"></a>\_Mensagem em ISIME

Determine com a localidade de entrada atual de um controle de edição rico é uma localidade do leste asiático.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Não usado; deve ser zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Não usado; deve ser zero.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retornará **true** se for uma localidade do leste asiático. Caso contrário, retornará **false**.
