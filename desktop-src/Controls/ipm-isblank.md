# <a name="ipm_isblank-message"></a>\_Mensagem de IsBlank IPM

Determina se todos os campos no controle de endereço IP estão em branco.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retornará zero se todos os campos estiverem em branco ou nenhum outro.
