# <a name="syslink"></a>SysLink

Esta seção contém informações sobre os elementos de programação usados com controles SysLink.

### <a name="overviews"></a>Visões gerais



| Tópico                                    | Sumário                                                                                       |
|------------------------------------------|------------------------------------------------------------------------------------------------|
| [Controles SysLink](syslink-overview.md) | O controle SysLink fornece uma maneira conveniente de inserir links de hipertexto em uma janela.<br/> |



 

### <a name="messages"></a>Mensagens



| Tópico                                           | Sumário                                                                             |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**\_GETIDEALHEIGHT LM**](lm-getidealheight.md) | Recupera a altura preferida de um link para a largura atual do controle.<br/> |
| [**\_GETIDEALSIZE LM**](lm-getidealsize.md)     | Recupera a altura preferida de um link para a largura atual do controle.<br/> |
| [**\_GETITEM LM**](lm-getitem.md)               | Recupera os Estados e atributos de um item.<br/>                           |
| [**HITTEST do LM \_**](lm-hittest.md)               | Determina se o usuário clicou no link especificado.<br/>                   |
| [**\_SETITEM LM**](lm-setitem.md)               | Define os Estados e atributos de um item.<br/>                                |

| Tópico                                | Sumário                                                                                                                                                                           |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LHITTESTINFO**](/windows/win32/api/commctrl/ns-commctrl-lhittestinfo) | Usado para obter informações sobre o link correspondente a um determinado local. <br/>                                                                                              |
| [**LITEm**](/windows/win32/api/commctrl/ns-commctrl-litem)               | Usado para definir e recuperar informações sobre um item de link.<br/>                                                                                                                 |
| [**NMLINK**](/windows/win32/api/commctrl/ns-commctrl-nmlink)             | O [**NMLINK**](/windows/win32/api/commctrl/ns-commctrl-nmlink) contém informações de notificação. Envie essa estrutura com as mensagens de [ \_ retorno](nm-return.md) do [nm \_ Click](nm-click-syslink.md) ou nm.<br/> |
