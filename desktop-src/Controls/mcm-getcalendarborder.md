# <a name="mcm_getcalendarborder-message"></a>Mensagem MCM \_ GETCALENDARBORDER

Obtém o tamanho da borda, em pixels. Você pode enviar essa mensagem explicitamente ou usando a macro [**\_ MonthCal GetCurrentView.**](/windows/desktop/api/Commctrl/nf-commctrl-monthcal_getcurrentview)

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Deve ser zero.

</dd> <dt>

*lParam* 
</dt> <dd>

Deve ser zero.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Tamanho da borda, em pixels.
