# <a name="tb_gettextrows-message"></a>TB de \_ mensagem GETTEXTROWS

Recupera o número máximo de linhas de texto que podem ser exibidas em um botão da barra de ferramentas.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Valor retornado

Retorna um valor INT que representa o número máximo de linhas de texto que o controle exibirá para um botão.
