# <a name="tbm_getrangemin-message"></a>\_Mensagem tbm GETRANGEMIN

Recupera a posição mínima do controle deslizante em um TrackBar.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>Deve ser zero.</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retorna um valor de 32 bits que especifica a posição mínima no intervalo de TrackBar de mínimo para posições de controle deslizante máximo.
