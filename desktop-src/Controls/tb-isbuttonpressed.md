# <a name="tb_isbuttonpressed-message"></a>TB de \_ mensagem ISBUTTONPRESSED

Determina se o botão especificado em uma barra de ferramentas é pressionado.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador de comando do botão.

</dd> <dt>

*lParam* 
</dt> <dd>Deve ser zero.</dd> </dl>

## <a name="return-value"></a>Retornar valor

Retornará zero se o botão for pressionado ou zero caso contrário.
