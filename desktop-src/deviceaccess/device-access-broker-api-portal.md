# <a name="device-access-api"></a>API de acesso do dispositivo

## <a name="purpose"></a>Finalidade

Você pode usar a API de acesso do dispositivo para gravar aplicativos de dispositivo da Windows Store para dispositivos especializados que usam drivers personalizados. A API fornece métodos para enviar códigos de controle para se comunicar com o driver personalizado do dispositivo.

## <a name="developer-audience"></a>Público de desenvolvedores

A API de acesso ao dispositivo é projetada para os desenvolvedores de OEM (fornecedor independente de hardware) e OEMs que estão familiarizados com C++ e Component Object Model (COM).
