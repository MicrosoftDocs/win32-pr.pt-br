# <a name="create-a-windows-runtime-component-for-device-access"></a>criar um componente Windows Runtime para acesso ao dispositivo

você pode criar um componente Windows Runtime para encapsular o componente que acessa seu driver. em seguida, você pode escrever um aplicativo de dispositivo de Windows Store para seu dispositivo em C# ou JavaScript.

## <a name="instructions"></a>Instruções

Consulte o [exemplo de acesso de driver personalizado](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/411c271e537727d737a53fa2cbe99eaecac00cc0/Official%20Windows%20Platform%20Sample/Custom%20driver%20access%20sample). este exemplo implementa um componente Windows Runtime para a funcionalidade providingthe de um dispositivo especializado e usa a funcionalidade em um aplicativo de dispositivo de armazenamento Windows.
