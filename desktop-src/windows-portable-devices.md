# <a name="windows-portable-devices"></a>Windows Dispositivos portáteis

Windows Os dispositivos portáteis (WPD) permitem que os computadores se comuniquem com dispositivos de mídia e armazenamento anexados. O WPD fornece uma maneira flexível e robusta para que os computadores se comuniquem com players de música, dispositivos de armazenamento, celulares, câmeras e muitos outros tipos de dispositivos conectados. esse sistema substitui Windows mídia Gerenciador de Dispositivos e Windows aquisição de imagem.

Aplicativos criados em WPD podem explorar um dispositivo, enviar e receber conteúdo e até mesmo controlar o dispositivo, por exemplo, tirar uma foto ou enviar uma mensagem de texto. O sistema foi projetado para ser flexível, de modo que muitos tipos de dispositivos possam ser explorados e extensíveis para que os desenvolvedores de driver possam definir propriedades e comandos personalizados para dispositivos personalizados.

você pode escrever aplicativos Windows e aplicativos Web com WPD. você usa a Interface de programação de aplicativo WPD para criar Windows aplicativos. esses aplicativos podem ser escritos em C++, C \# .net ou Visual Basic .net. Você usa o modelo de objeto de automação WPD para escrever aplicativos Web. esses aplicativos são escritos em JScript e HTML.

a Interface de programação de aplicativo WPD tem suporte nos sistemas operacionais Windows 7, Windows Vista e Windows XP. o modelo de objeto de automação WPD só tem suporte no Windows 7.

para obter mais informações sobre como criar um aplicativo de Windows wpd, consulte a documentação e os exemplos da [Interface de programação de aplicativos wpd](./wpd_sdk/wpd-application-programming-interface.md) .

Para obter mais informações sobre como criar um aplicativo de automação WPD, consulte a documentação e os exemplos do [modelo de objeto de automação WPD](/previous-versions/windows/desktop/legacy/dd389295(v=vs.85)) .
