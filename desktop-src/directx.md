# <a name="directx-graphics-and-gaming"></a>Gráficos e jogos do DirectX

Esse conteúdo se concentra no uso do DirectX em um aplicativo Win32. Para obter informações sobre como usar o DirectX em um aplicativo UWP, consulte [o guia Windows 10 desenvolvimento de jogos (UWP).](/windows/uwp/gaming/e2e)

> [!TIP]
> Para ver descrições de e links para componentes do DirectX no desenvolvimento ativo, consulte a postagem no blog Página de aterrissagem [do DirectX](https://devblogs.microsoft.com/directx/landing-page/).
