# <a name="directx-graphics-and-gaming"></a>Gráficos e jogos do DirectX

Esse conteúdo se concentra em usar o DirectX em um aplicativo Win32. Para obter informações sobre como usar o DirectX em um aplicativo UWP, consulte o [Guia de desenvolvimento de jogos do Windows 10 (UWP)](/windows/uwp/gaming/e2e).

O DirectX Graphics inclui suporte para gráficos 2D e 3D de alto desempenho. |
