# <a name="restart-manager-callback-functions"></a>Funções de retorno de chamada do Gerenciador de Reinicialização

A API do Gerenciador de Reinicialização usa as funções de retorno de chamada na tabela a seguir.



| Função de retorno da chamada                                               | Descrição                                                                                                                                                            |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RM \_ WRITE \_ STATUS \_ CALLBACK**](/windows/win32/api/restartmanager/nc-restartmanager-rm_write_status_callback) | O aplicativo que iniciou a sessão do Gerenciador de Reinicialização pode passar um ponteiro para essa função para as funções do Gerenciador de Reinicialização para receber uma porcentagem de conclusão. |
