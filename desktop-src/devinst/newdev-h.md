# <a name="newdevh"></a>Newdev. h

Esta seção contém tópicos de referência para o cabeçalho Newdev. h.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                     | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DiInstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | A função **DiInstallDevice** instala um driver especificado pré-instalado no [repositório de drivers](/windows-hardware/drivers/install/driver-store) em um dispositivo especificado que está presente no sistema.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**DiInstallDriver**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | A função **DiInstallDriver** pré-instala um driver no [repositório de drivers](/windows-hardware/drivers/install/driver-store) e instala o driver em dispositivos presentes no sistema ao qual o driver dá suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DiRollbackDriver**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | A função **DiRollbackDriver** reverte o driver que está instalado em um dispositivo especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**DiShowUpdateDevice**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | A função **DiShowUpdateDevice** exibe o assistente de atualização de hardware para um dispositivo especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | A função [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) desinstala um dispositivo e remove seu nó de dispositivo ([Devnode](/windows-hardware/drivers/)) do sistema. Isso é diferente de usar [**SetupDiCallClassInstaller**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) com o código de [**\_ remoção de dif**](/windows-hardware/drivers/install/dif-remove) , pois ele tenta desinstalar o nó do dispositivo, além de devnodes filhos que estão presentes no momento da chamada.<br/> Antes do Windows 8, todos os dispositivos filho que não estão presentes no momento da chamada não serão desinstalados. No entanto, a partir do Windows 8, todos os dispositivos filho que não estão presentes no momento da chamada serão desinstalados.<br/> |
| [**UpdateDriverForPlugAndPlayDevices**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | Dado um arquivo INF e uma [ID de hardware](/windows-hardware/drivers/install/hardware-ids), a função **UpdateDriverForPlugAndPlayDevices** instala drivers atualizados para dispositivos que correspondem à ID de hardware. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
