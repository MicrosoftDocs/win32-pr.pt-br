# <a name="newdevh"></a>Newdev.h

Esta seção contém tópicos de referência para o header Newdev.h.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                     | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DiInstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diinstalldevice)<br/>                                     | A **função DiInstallDevice** instala um driver especificado que é pré-instalado no armazenamento de [driver](/windows-hardware/drivers/install/driver-store) em um dispositivo especificado que está presente no sistema.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**DiInstallDriver**](/windows/desktop/api/Newdev/nf-newdev-diinstalldrivera)<br/>                                     | A **função DiInstallDriver** pré-instala um driver no armazenamento de [driver](/windows-hardware/drivers/install/driver-store) e, em seguida, instala o driver em dispositivos presentes no sistema que o driver dá suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**DiRollbackDriver**](/windows/desktop/api/Newdev/nf-newdev-dirollbackdriver)<br/>                                   | A **função DiRollbackDriver** rola de volta o driver que está instalado em um dispositivo especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**DiShowUpdateDevice**](/windows/desktop/api/Newdev/nf-newdev-dishowupdatedevice)<br/>                               | A **função DiShowUpdateDevice** exibe o assistente de Atualização de Hardware para um dispositivo especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice)<br/>                                 | A [**função DiUninstallDevice**](/windows/desktop/api/Newdev/nf-newdev-diuninstalldevice) desinstala um dispositivo e remove seu nó de dispositivo ([devnode](/windows-hardware/drivers/)) do sistema. Isso é diferente de usar [**SetupDiCallClassInstaller**](/windows/desktop/api/Setupapi/nf-setupapi-setupdicallclassinstaller) com o código [**DIF \_ REMOVE**](/windows-hardware/drivers/install/dif-remove) porque ele tenta desinstalar o nó do dispositivo, além dos devnodes filho que estão presentes no momento da chamada.<br/> Antes Windows 8 dispositivos filho que não estão presentes no momento da chamada não serão desinstalados. No entanto, começando com Windows 8, todos os dispositivos filho que não estão presentes no momento da chamada serão desinstalados.<br/> |
| [**UpdateDriverForPlugAndPlayDevices**](/windows/desktop/api/Newdev/nf-newdev-updatedriverforplugandplaydevicesa)<br/> | Considerando um arquivo INF e uma ID de [hardware,](/windows-hardware/drivers/install/hardware-ids)a função **UpdateDriverForPlugAndPlayDevices** instala drivers atualizados para dispositivos que corresponderem à ID de hardware. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
