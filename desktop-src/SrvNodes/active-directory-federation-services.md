# <a name="active-directory-federation-services"></a>Serviços de Federação do Active Directory

O Serviço de Federação de Active Directory (AD FS) é um serviço de autenticação e autorização que é executado no Windows Server. O AD FS fornece aos usuários acesso de logon único entre ambientes corporativos e aplicativos Web.
