# <a name="active-directory-federation-services"></a>Serviços de Federação do Active Directory

O active directory Serviço de Federação (AD FS) é um serviço de autenticação e autorização executado no Windows Server. AD FS fornece aos usuários acesso de login único entre ambientes empresariais e aplicativos Web.
