# <a name="microsoft-rights-management-sdks"></a>Microsoft Rights Management SDKs

Três gerações do SDK do Rights Management agora estão disponíveis: O [SDK do Rights Management 4.2,](/previous-versions/windows/desktop/msipcthin2/active-directory-rights-management-services-multi-platform-thin-client-sdk-portal)o [SDK do Microsoft Rights Management 2.1](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal) e o [SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)do Active Directory Rights Management Services, bem como uma API de script, a API de Script do [Active Directory Rights Management Services,](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)que permite a administração personalizada de um servidor RMS.

RMS SDK 4.2 é um conjunto de ferramentas simplificado e de última geração que fornece uma experiência de desenvolvimento leve para habilenciar seus aplicativos de dispositivo Android, iOS e Mac OS X com proteção de informações por meio dos serviços microsoft Rights Management.

Em vez disso, [use o Microsoft Rights Management Services SDK 2.1,](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal)que aproveita a funcionalidade exposta pelo cliente no Msipc.dll.
