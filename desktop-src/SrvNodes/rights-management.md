# <a name="microsoft-rights-management-sdks"></a>SDKs do Microsoft Rights Management

Três gerações do SDK do Rights Management agora estão disponíveis: [Rights Management sdk 4,2](/previous-versions/windows/desktop/msipcthin2/active-directory-rights-management-services-multi-platform-thin-client-sdk-portal), [Microsoft Rights Management sdk 2,1](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal) e [Active Directory Rights Management Services SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal), bem como uma API de script, [Active Directory Rights Management Services API de script](/previous-versions/windows/desktop/adrms_script/adrms-script-portal), que permite a administração personalizada de um servidor RMS.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[Exemplos de código e ferramentas](rms-code-samples-and-tools.md)
</dt> <dd>

Uma coleção de exemplos de código do RMS fornecidos pela Microsoft e ferramentas de suporte do desenvolvedor em todos os sistemas operacionais com suporte; Android, iOS/OS X, Windows Phone e área de trabalho do Windows.

</dd> <dd>

O RMS SDK 4,2 é um conjunto de ferramentas simplificado e de última geração que fornece uma experiência de desenvolvimento leve para habilitar seus aplicativos de dispositivo Android, iOS e Mac OS X com proteção de informações por meio dos serviços de Rights Management da Microsoft.

</dd> <dd>

O RMS SDK 2,1 é uma poderosa oferta de SDK para desenvolvedores de aplicativos do Windows desktop e provedores de soluções baseados em servidor para habilitar seus produtos com o Rights Management.

</dd> <dt>

[Active Directory Rights Management Services SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)
</dt> <dd>

> [!Note]  
> AD RMS SDK aproveitando a funcionalidade exposta pelo cliente no Msdrm.dll está disponível para uso no Windows Server 2012, Windows 8, Windows Server 2008 R2, Windows 7, Windows Server 2008 e Windows Vista. Ele poderá ser alterado ou ficar indisponível em versões subsequentes. Em vez disso, use o [Microsoft Rights Management Services SDK 2,1](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal), que aproveita a funcionalidade exposta pelo cliente no Msipc.dll.

 

</dd> <dt>

[API de script do Active Directory Rights Management Services](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)
</dt> <dd>

A API de script do Windows Active Directory Rights Management Services (AD RMS) pode ser usada para criar scripts para administrar uma instalação do AD RMS.

</dd> </dl>
