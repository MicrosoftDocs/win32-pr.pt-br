# <a name="side-by-side-assembly-interfaces"></a>Interfaces de assembly lado a lado

A API do assembly lado a lado usa as seguintes interfaces.



| Interface                                        | Descrição                                                                                                                                            |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache)         | Usado para instalar, desinstalar ou consultar um assembly lado a lado.                                                                                          |
| [**IAssemblyCacheItem**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycacheitem) | Usado para instalar assemblies lado a lado no armazenamento lado a lado usando uma instalação baseada em fluxo.                                                 |
| [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname)           | Representa um nome de assembly lado a lado. O nome do assembly lado a lado consiste em um conjunto de pares de nome-valor que descrevem o assembly lado a lado. |
