# <a name="side-by-side-assembly-api"></a>API do assembly lado a lado

A API do assembly lado a lado instala, remove ou consulta assemblies lado a lado do armazenamento de assembly lado a lado do sistema. Isso também é chamado de armazenamento winSxS no Windows XP e da loja de componentes da CSI (infraestrutura de manutenção de componentes), começando com o Windows Vista e o Windows Server 2008.

As partes da API do assembly lado a lado são descritas nas seções a seguir:

-   [Enumerações de assembly lado a lado](enumerations.md)
-   [Funções de assembly lado a lado](side-by-side-assembly-functions.md)
-   [Interfaces de assembly lado a lado](side-by-side-assembly-interfaces.md)
-   [Estruturas de assembly lado a lado](structures.md)
