# <a name="g-isolated-applications-and-side-by-side-assemblies"></a>G (aplicativos isolados e assemblies lado a lado)

[A](a-sbscs-gly.md) B C [D](d-sbscs-gly.md) E F G H [i](i-sbscs-gly.md) J K L [M](m-sbscs-gly.md) N O [P](p-sbscs-gly.md) Q [R](r-sbscs-gly.md) [S](s-sbscs-gly.md) T U V W X Y Z

<dl> <dt>

<span id="_win32_global_application_configuration_gly"></span><span id="_WIN32_GLOBAL_APPLICATION_CONFIGURATION_GLY"></span>**configuração de aplicativo global**
</dt> <dd>

Especificação que todos os aplicativos no sistema usam a mesma versão do assembly lado a lado. A configuração global é especificada em uma base por assembly em um arquivo de manifesto de assembly global instalado na pasta WinSxS. A configuração global pode ser usada quando um desenvolvedor ou administrador de assembly precisar configurar todos os aplicativos no sistema para usar uma versão de assembly mais recente. Nesse caso, o novo assembly precisa ser compatível com versões anteriores. Uma configuração por aplicativo substitui a configuração de aplicativo padrão ou global para aplicativos específicos.

</dd> <dt>

<span id="_win32_global_assembly_manifest_gly"></span><span id="_WIN32_GLOBAL_ASSEMBLY_MANIFEST_GLY"></span>**manifesto de assembly global**
</dt> <dd>

Arquivo que define a configuração de aplicativo global de um assembly lado a lado. Os arquivos de manifesto de assembly global são instalados no repositório de assemblies do sistema na pasta WinSxS.

</dd> </dl>
