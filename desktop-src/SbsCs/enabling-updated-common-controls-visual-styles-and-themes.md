# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>Habilitando controles comuns, estilos visuais e temas atualizados

Os desenvolvedores de aplicativos executados no Windows XP podem adicionar estilos e temas visuais à interface do usuário de seus aplicativos, incluindo manifestos que descrevem a dependência do aplicativo na versão COMCTL32.DLL 6,0. Um aplicativo requer a funcionalidade dos controles comuns atualizados para usar temas e estilos visuais.

Para obter mais informações sobre como usar estilos visuais, consulte usando os estilos visuais do Windows XP no [Guia do programador do Shell](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85)). O assembly lado a lado que contém COMCTL32.DLL versão 6,0 é um dos assemblies lado a lado da [Microsoft com suporte](supported-microsoft-side-by-side-assemblies.md).
