# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>Habilitando controles comuns, estilos visuais e temas atualizados

Os desenvolvedores de aplicativos que são executados no Windows XP podem adicionar estilos visuais e temas à interface do usuário de seus aplicativos, incluindo manifestos que descrevem a dependência do aplicativo COMCTL32.DLL versão 6.0. Um aplicativo requer a funcionalidade dos controles comuns atualizados para usar temas e estilos visuais.

Para obter mais informações sobre como usar estilos visuais, consulte Using Windows XP Visual Styles in the [Shell Programmer's Guide](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85)). O assembly lado a lado que COMCTL32.DLL versão 6.0 é um dos assemblies lado a [lado da Microsoft com suporte.](supported-microsoft-side-by-side-assemblies.md)
