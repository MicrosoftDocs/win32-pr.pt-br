# <a name="default-configuration"></a>Configuração padrão

As versões de [assembly especificadas](assembly-manifests.md) nos manifestos do assembly e nos [manifestos](application-manifests.md) do aplicativo compõem a configuração padrão que o aplicativo deve usar quando o aplicativo é executado. Os aplicativos e administradores podem substituir [a](publisher-configuration.md) configuração de assembly padrão instalando uma configuração de editor ou configuração por aplicativo no Windows Server 2003.
