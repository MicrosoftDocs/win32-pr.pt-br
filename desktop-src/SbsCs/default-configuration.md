# <a name="default-configuration"></a>Configuração padrão

As versões de assembly especificadas em [manifestos de assembly](assembly-manifests.md) e [manifestos de aplicativo](application-manifests.md) incluem a configuração padrão que o aplicativo deve usar quando o aplicativo é executado. Os aplicativos e administradores podem substituir a configuração de assembly padrão instalando uma [configuração de Publicador](publisher-configuration.md) ou [configuração por aplicativo](per-application-configuration.md) no Windows Server 2003.
