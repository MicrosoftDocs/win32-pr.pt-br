# <a name="side-by-side-assembly-development-tools"></a>Ferramentas de desenvolvimento lado a lado do assembly

As ferramentas a seguir estão disponíveis apenas no SDK (Software Development Kit) do Microsoft Windows.



| Utilitário                                | Descrição                                                   |
|----------------------------------------|---------------------------------------------------------------|
| [Manifestchk.vbs](manifestchk-vbs.md) | Uma ferramenta de validação de manifesto.                                   |
| [Mt.exe](mt-exe.md)                   | Uma ferramenta de compilação de manifesto que cria arquivos e catálogos assinados. |
| [Pktextract.exe](pktextract-exe.md)   | Extrai tokens de chave pública de arquivos de certificado.            |
