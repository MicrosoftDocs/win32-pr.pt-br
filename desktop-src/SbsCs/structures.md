# <a name="side-by-side-assembly-structures"></a>Estruturas de assembly lado a lado

A API de assembly lado a lado usa as estruturas a seguir.



| Estrutura                                                       | Descrição                                                                            |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**INFORMAÇÕES DO \_ ASSEMBLY**](/windows/desktop/api/Winsxs/ns-winsxs-assembly_info)                        | Contém informações sobre um assembly no armazenamento de assembly lado a lado do sistema.    |
| [**REFERÊNCIA DE INSTALAÇÃO DO FUSION \_ \_**](/windows/win32/api/winsxs/ns-winsxs-fusion_install_reference) | Contém informações sobre o aplicativo que faz referência ao assembly lado a lado. |
