# <a name="supported-microsoft-side-by-side-assemblies"></a>Assemblies lado a lado da Microsoft com suporte

Veja a seguir os assemblies lado a lado com suporte da Microsoft. Windows Os Service Packs podem incluir versões compatíveis desses assemblies com correções de segurança ou de bugs.

Você pode pesquisar os redistribuíveis disponíveis no [centro de download da Microsoft](https://www.microsoft.com/Downloads/).

## <a name="windows-server-2003"></a>Windows Server 2003

esses assemblies lado a lado têm suporte da Microsoft a partir do Windows Server 2003.

-   WinHTTP 5,1
-   Assemblies de automação de isolamento da Microsoft que permitem o uso de assemblies lado a lado com scripts.
-   Controles comuns do Shell versão 6,0 (Comctl32.dll)
-   GDI Plus versão 1,0 (GDIplus.dll)
-   Visual C++ bibliotecas de tempo de execução versão 6,0

## <a name="windows-xp"></a>Windows XP

esses assemblies lado a lado foram suportados pela Microsoft a partir do Windows XP.

-   Controles comuns do Shell versão 6,0 (Comctl32.dll)
-   GDI Plus versão 1,0 (GDIplus.dll)
-   Visual C++ bibliotecas de tempo de execução versão 6,0
