# <a name="side-by-side-assembly-functions"></a>Funções de assembly lado a lado

A API do assembly lado a lado usa as seguintes funções para acessar as interfaces de assembly lado a lado.



| Função                                                     | Descrição                                                                    |
|--------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**CreateAssemblyCache**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblycache)           | Obtém uma instância da interface [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache) . |
| [**CreateAssemblyNameObject**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblynameobject) | Obtém uma instância da interface [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname) .   |
