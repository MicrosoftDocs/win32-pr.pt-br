# <a name="installing-side-by-side-assemblies"></a>Instalando assemblies lado a lado

Você pode instalar assemblies lado a lado como [assemblies compartilhados](/windows/desktop/Msi/shared-assemblies) ou como [assemblies privados](/windows/desktop/Msi/private-assemblies). Para obter mais informações, consulte [instalando assemblies lado a lado como assemblies privados](installing-side-by-side-assemblies-as-private-assemblies.md) e [instalando assemblies lado a lado como assemblies compartilhados](installing-side-by-side-assemblies-as-shared-assemblies.md).

Observe que:

-   Os assemblies instalados no cache de assembly global por uma instalação usando o contexto de [instalação](/windows/desktop/Msi/installation-context) por usuário não são protegidos pela proteção de arquivo do Windows.
-   Os assemblies que são instalados no cache de assembly global por uma instalação usando o contexto de [instalação](/windows/desktop/Msi/installation-context) por máquina são protegidos pela proteção de arquivo do Windows.
