# <a name="installing-side-by-side-assemblies"></a>Instalando assemblies lado a lado

Você pode instalar assemblies lado a lado como [assemblies compartilhados](/windows/desktop/Msi/shared-assemblies) ou [como assemblies privados](/windows/desktop/Msi/private-assemblies). Para obter mais informações, consulte Instalando assemblies lado a lado como [assemblies](installing-side-by-side-assemblies-as-private-assemblies.md) privados e Instalando assemblies lado a [lado como assemblies compartilhados.](installing-side-by-side-assemblies-as-shared-assemblies.md)

Observe que:

-   Os assemblies instalados no cache de assembly global [](/windows/desktop/Msi/installation-context) por uma instalação usando o contexto de instalação por usuário não são protegidos pelo Windows Proteção de Arquivos.
-   Os assemblies instalados no cache de assembly global por [](/windows/desktop/Msi/installation-context) uma instalação usando o contexto de instalação por computador são protegidos por Windows Proteção de Arquivos.
