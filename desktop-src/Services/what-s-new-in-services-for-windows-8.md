# <a name="whats-new-in-services-for-windows-8"></a>Novidades nos Serviços para Windows 8

Windows 8 e Windows Server 2012 incluem os novos recursos a seguir para serviços.

## <a name="new-service-functions"></a>Novas funções de serviço



| Função                                                                            | Descrição                                                                    |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**QueryServiceDynamicInformation**](/windows/desktop/api/Winsvc/nf-winsvc-queryservicedynamicinformation)<br/> | Recupera informações dinâmicas relacionadas ao início do serviço atual.<br/> |

| [**ITEM DE \_ DADOS ESPECÍFICO DO GATILHO DE \_ \_ \_ SERVIÇO**](/windows/desktop/api/winsvc/ns-winsvc-service_trigger_specific_data_item)<br/> | ADICIONADO NÍVEL DE TIPO DE DADOS DO GATILHO **\_ DE \_ \_ \_ SERVIÇO**, **\_ \_ \_ \_ PALAVRA-CHAVE \_ ANY DO** TIPO DE DADOS DO GATILHO DE SERVIÇO E **PALAVRA-CHAVE ALL DO TIPO \_ DE DADOS DO GATILHO DE \_ \_ \_ \_ SERVIÇO**.<br/> |
