# <a name="guids-of-user-interface-elements"></a>GUIDs dos elementos da interface do usuário

A tabela a seguir lista o identificador de classe, expresso como um GUID, para objetos de interface do usuário implementados por um serviço de diretório.



| Identificador de classe              | Descrição                                                                                                                                                                                                                                                                 |
|-------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_DSFOLDERPROPERTIES CLSID** | Reservado. Declarado em DSClient. h.                                                                                                                                                                                                                                           |
| **\_DSPROPERTYPAGES CLSID**    | Esse objeto fornece as interfaces [**IShellExtInit**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellextinit), [**IShellPropSheetExt**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellpropsheetext)e [**IContextMenu**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-icontextmenu) para uso com objetos de serviço de diretório fornecidos pelo sistema. Declarado em DSClient. h. |
