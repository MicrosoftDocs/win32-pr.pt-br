# <a name="guids-of-user-interface-elements"></a>GUIDs de Interface do Usuário elementos

A tabela a seguir lista o identificador de classe, expresso como um GUID, para objetos de interface do usuário implementados por um serviço de diretório.



| Identificador de Classe              | Descrição                                                                                                                                                                                                                                                                 |
|-------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **CLSID \_ DsFolderProperties** | Reservado. Declarado em Dsclient.h.                                                                                                                                                                                                                                           |
| **CLSID \_ DsPropertyPages**    | Esse objeto fornece as interfaces [**IShellExtInit**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellextinit), [**IShellPropSheetExt**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ishellpropsheetext)e [**IContextMenu**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-icontextmenu) para uso com objetos de serviço de diretório fornecidos pelo sistema. Declarado em Dsclient.h. |
