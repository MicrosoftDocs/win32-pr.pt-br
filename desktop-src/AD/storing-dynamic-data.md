# <a name="storing-dynamic-data"></a>Armazenando Dados Dinâmicos

A maioria dos dados armazenados usando Active Directory Domain Services é estático. Por exemplo, os objetos de diretório que representam usuários, grupos, impressoras, serviços de rede e assim por diante, normalmente são de longa duração e alterados com pouca frequência.

No Windows Server 2003 e versões posteriores do Windows, Active Directory Domain Services fornecem suporte para os seguintes recursos que permitem que os aplicativos usem o diretório para armazenar dados dinâmicos:

-   [Objetos dinâmicos](dynamic-objects.md)
-   [Classes auxiliares dinâmicas](dynamic-auxiliary-classes.md)
