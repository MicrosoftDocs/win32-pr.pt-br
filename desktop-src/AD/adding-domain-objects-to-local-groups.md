<span data-ttu-id="f5532-113">**Para adicionar um objeto de domínio a um grupo local**</span><span class="sxs-lookup"><span data-stu-id="f5532-113">**To add a domain object to a local group**</span></span>

1.  <span data-ttu-id="f5532-114">Associe-se à interface [**IADsContainer**](/windows/desktop/api/iads/nn-iads-iadscontainer) do computador que contém o grupo local ao qual adicionar um membro.</span><span class="sxs-lookup"><span data-stu-id="f5532-114">Bind to the [**IADsContainer**](/windows/desktop/api/iads/nn-iads-iadscontainer) interface of the computer that contains the local group to add a member to.</span></span> <span data-ttu-id="f5532-115">A associação deve ser executada usando uma conta que tenha direitos suficientes para acessar esse computador.</span><span class="sxs-lookup"><span data-stu-id="f5532-115">The binding must be performed using an account that has sufficient rights to access that computer.</span></span> <span data-ttu-id="f5532-116">A cadeia de vinculação deve assumir o formato "WinNT:// <computer name> , Computer", em que " &lt; 5.  <span data-ttu-id="f5532-123">Adicione o usuário ou grupo ao grupo local com o método [**IADs.
