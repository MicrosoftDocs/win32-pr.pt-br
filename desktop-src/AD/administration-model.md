# <a name="administration-model"></a>Modelo de administração

Os usuários autorizados executam a administração no Active Directory Domain Services. Um usuário é autorizado por uma autoridade mais alta para executar um conjunto especificado de ações em um conjunto especificado de objetos e classes de objeto em algumas subárvores identificadas do diretório. Isso é chamado de *administração delegada*. A administração delegada permite um controle muito refinado sobre quem pode fazer o que e habilitar a delegação de autoridade sem conceder privilégios elevados.
