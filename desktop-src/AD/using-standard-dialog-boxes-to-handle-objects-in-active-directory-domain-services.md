# <a name="using-standard-dialog-boxes-to-handle-objects-in-active-directory-domain-services"></a>Usando caixas de diálogo padrão para manipular objetos no Active Directory Domain Services

Windows fornece uma biblioteca de caixa de diálogo comum para operações comuns, como abrir arquivo, procurar arquivo e assim por diante. a partir do Windows 2000, o sistema fornece caixas de diálogo que podem ser usadas para operações comuns de interface do usuário no Active Directory Domain Services.

As caixas de diálogo a seguir estão disponíveis para uso com Active Directory Domain Services:

-   [Seletor de objetos de diretório](directory-object-picker.md)
-   [Navegador de domínio](domain-browser.md)
-   [Navegador de contêineres](container-browser.md)
-   [Active Directory folhas de propriedades de usuários e computadores](active-directory-users-and-computers-property-sheets.md)

Além disso, os assistentes de criação de Active Directory Domain Services também podem ser chamados de um aplicativo. Para obter mais informações, consulte [invocando assistentes de criação do seu aplicativo](invoking-creation-wizards-from-your-application.md).
