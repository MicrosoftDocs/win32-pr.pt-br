# <a name="cfstr_dsobjectnames"></a>CFSTR \_ DSOBJECTNAMES

<dl> <dt>

<span id="CFSTR_DSOBJECTNAMES"></span><span id="cfstr_dsobjectnames"></span>**CFSTR \_ DSOBJECTNAMES**
</dt> <dd> <dl> <dt>

"DsObjectNames"
</dt> <dt>



O formato de área de transferência **CFSTR \_ DSOBJECTNAMES** é suportado pelo [**IDataObject**](/windows/win32/api/objidl/nn-objidl-idataobject) fornecido pelo método [**ICommonQuery:: OpenQueryWindow**](/windows/win32/api/cmnquery/nf-cmnquery-icommonquery-openquerywindow) . O formato de área de transferência **CFSTR \_ DSOBJECTNAMES** fornece um identificador de memória global (**HGLOBAL**) que contém a estrutura [**DSOBJECTNAMES**](/windows/desktop/api/Dsclient/ns-dsclient-dsobjectnames) .


</dt> </dl> </dd> </dl>
