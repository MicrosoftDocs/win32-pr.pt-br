# <a name="choosing-a-syntax"></a>Escolhendo uma sintaxe

Há 23 sintaxes definidas em Active Directory Domain Services. Este tópico contém uma lista de sintaxes recomendadas a serem usadas ao definir um novo atributo. Para obter mais informações, consulte [sintaxes para atributos em Active Directory Domain Services](syntaxes-for-attributes-in-active-directory-domain-services.md).

A tabela a seguir fornece uma lista de recomendações.



| Dados a serem armazenados no atributo      | Sintaxe a ser usada                                                      | Comentário                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
|---------------------------------|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Dados binários                     | [**Cadeia de caracteres (octeto)**](/windows/desktop/ADSchema/s-string-octet)                       | Use para armazenar dados binários. Esta é uma matriz de bytes.                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Dados binários com uma referência DN | [**Objeto (DN-binário)**](/windows/desktop/ADSchema/s-object-dn-binary)               | Contém um valor binário e um DN (nome distinto). O servidor de Active Directory mantém o DN atualizado.                                                                                                                                                                                                                                                                                                                                                                                                     |
| Boolean                         | [**Boolean**](/windows/desktop/ADSchema/s-boolean)                                  | Use para valores Boolianos.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Referência de DN                    | [**Objeto (DS-DN)**](/windows/desktop/ADSchema/s-object-ds-dn)                       | Use para armazenar nomes distintos que você deseja manter atualizados pelo servidor de Active Directory. Quando um atributo de sintaxe DN é criado com um DN válido, o servidor trata o atributo como uma referência ao objeto representado pelo DN que foi definido. Se o objeto referenciado for renomeado ou movido, o servidor garantirá que o atributo reflita a alteração. Se o atributo for redefinido com um novo DN, o atributo será referenciado ao objeto representado pelo novo DN.                                |
| Inteiro                         | [**Inteiro**](/windows/desktop/ADSchema/s-integer)                                  | Use para números inteiros.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Inteiro grande (valores de 64 bits)   | [**LargeInteger**](/windows/desktop/ADSchema/s-largeinteger)                        | Use para valores de 64 bits.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| DN vinculado                       | [**Objeto (DS-DN)**](/windows/desktop/ADSchema/s-object-ds-dn)                       | Essa sintaxe de cadeia de caracteres pode ser usada para DNs vinculado. Links de retorno devem ser do DN de sintaxe. Os links de encaminhamento podem ser de DN de sintaxe, bem como um objeto ( [**cadeia de caracteres DN)**](/windows/desktop/ADSchema/s-object-dn-string), [**objeto (DN-binário)**](/windows/desktop/ADSchema/s-object-dn-binary), [**objeto (ponto de acesso)**](/windows/desktop/ADSchema/s-object-access-point)ou [**objeto (ou-nome)**](/windows/desktop/ADSchema/s-object-or-name). Atributos vinculados devem ter uma **LinkId** definida. Consulte a descrição de **LinkId** em Propriedades de [**esquema de atributo**](/windows/desktop/ADSchema/c-attributeschema) . |
| Descritor de Segurança             | [**Cadeia de caracteres (NT-SEC-DESC)**](/windows/desktop/ADSchema/s-string-nt-sec-desc)           | Cadeia de caracteres de octeto que contém um descritor de segurança.                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Identificador de segurança (SID)       | [**Cadeia de caracteres (SID)**](/windows/desktop/ADSchema/s-string-sid)                           | Cadeia de caracteres de octeto que contém um SID (identificador de segurança). Use esta sintaxe para armazenar somente valores de SID.                                                                                                                                                                                                                                                                                                                                                                                                                  |
| String                          | [**Cadeia de caracteres (Unicode)**](/windows/desktop/ADSchema/s-string-unicode)                   | Use para a maioria dos atributos de cadeia de caracteres. Ele dá suporte ao conjunto de caracteres Unicode. Quando o servidor de Active Directory executa comparações com atributos dessa sintaxe (como avaliar uma consulta), ele executa comparações que não diferenciam maiúsculas de minúsculas. Use as outras sintaxes de cadeia de caracteres ([**seqüência de caracteres (IA5)**](/windows/desktop/ADSchema/s-string-ia5), [**cadeia de caracteres (numérica)**](/windows/desktop/ADSchema/s-string-numeric)e assim por diante) para armazenar cadeias de caracteres que devem conter apenas os conjuntos de caractere específicos com suporte da sintaxe.<br/>                          |
| Dados de cadeia de caracteres com uma referência DN | [**Objeto (DN-String)**](/windows/desktop/ADSchema/s-object-dn-string)               | Cadeia de caracteres que contém um valor de cadeia de caracteres e um DN (nome distinto). O servidor de Active Directory mantém o DN atualizado.                                                                                                                                                                                                                                                                                                                                                                                            |
| Hora                            | [**Cadeia de caracteres (em tempo geral)**](/windows/desktop/ADSchema/s-string-generalized-time) | Use a sintaxe da [**cadeia de caracteres (de tempo generalizado)**](/windows/desktop/ADSchema/s-string-generalized-time) para armazenar valores de tempo em vez da sintaxe da [**cadeia de caracteres (UTC-time)**](/windows/desktop/ADSchema/s-string-utc-time) porque a cadeia de **caracteres (tempo generalizado)** usa quatro caracteres para o ano e a **cadeia de caracteres (UTC-time)** usa apenas dois.                                                                                                                                                                                                                 |
