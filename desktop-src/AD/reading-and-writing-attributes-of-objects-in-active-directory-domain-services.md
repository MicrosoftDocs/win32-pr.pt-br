# <a name="reading-and-writing-attributes-of-objects-in-active-directory-domain-services"></a>Lendo e escrevendo atributos de objetos em Active Directory Domain Services

Os objetos Active Directory Domain Services têm atributos que contêm dados sobre o objeto específico. A maneira como os atributos são acessados e modificados depende da tecnologia de programação que está sendo usada. Para obter mais informações sobre como ler e modificar atributos Active Directory Domain Services com uma tecnologia de programação específica, consulte os tópicos listados a seguir.



| Tecnologia de programação                                                                       | Para obter mais informações                                                                        |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [Active Directory Service Interfaces](/windows/desktop/ADSI/active-directory-service-interfaces-adsi)         | [Acessando e manipulando dados com ADSI](/windows/desktop/ADSI/accessing-and-manipulating-data-with-adsi) |
| [Lightweight Directory Access Protocol](/previous-versions/windows/desktop/ldap/lightweight-directory-access-protocol-ldap-api) | [Modificando uma entrada de diretório](/previous-versions/windows/desktop/ldap/modifying-a-directory-entry)                             |
| [System.DirectoryServices](/dotnet/api/system.directoryservices) | [Propriedades do objeto Directory](https://msdn.microsoft.com/library/ms180858(v=VS.80).aspx)                         |



 

Para obter mais informações sobre atributos, consulte [Determinando um tipo de atributo](determining-an-attribute-type.md).
