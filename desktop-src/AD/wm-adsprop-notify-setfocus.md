# <a name="wm_adsprop_notify_setfocus-message"></a>Mensagem WM \_ ADSPROP \_ NOTIFY \_ SETFOCUS

A **mensagem WM \_ ADSPROP NOTIFY \_ \_ SETFOCUS** é usada internamente pelo objeto de notificação.


```C++
WM_ADSPROP_NOTIFY_SETFOCUS

    WPARAM wParam
    LPARAM lParam
    
```

<dl> <dt>

*Hwnd* 
</dt> <dd>

Não usado.

</dt> <dd>

Não usado.

</dd> <dt>

</dt> <dd>
