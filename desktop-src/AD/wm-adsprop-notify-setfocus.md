# <a name="wm_adsprop_notify_setfocus-message"></a>\_Mensagem de \_ notificação \_ SETFOCUS do WM ADSPROP

A mensagem do **WM \_ ADSPROP \_ Notify \_ SETFOCUS** é usada internamente pelo objeto de notificação.


```C++
WM_ADSPROP_NOTIFY_SETFOCUS

    WPARAM wParam
    LPARAM lParam
    
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*HWND* 
</dt> <dd>

Não usado.

</dd> <dt>

*wParam* 
</dt> <dd>

Não usado.

</dd> <dt>

*lParam* 
</dt> <dd>

Não usado.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Esta mensagem não tem nenhum valor de retorno.
