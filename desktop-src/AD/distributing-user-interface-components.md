# <a name="distributing-user-interface-components"></a>Distribuindo componentes da interface do usuário

Os componentes COM, aplicativos e/ou arquivos usados por uma extensão de interface do usuário podem ser distribuídos criando um pacote de Windows Installer e usando uma política de grupo para distribuir o pacote. Para obter mais informações sobre como criar o pacote do instalador, consulte [Windows Installer](/windows/desktop/Msi/windows-installer-portal). Para obter mais informações sobre políticas de grupo, consulte [política de grupo](/previous-versions/windows/desktop/Policy/group-policy-start-page).
