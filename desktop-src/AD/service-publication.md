# <a name="service-publication"></a>Publicação de serviço

Os serviços se anunciam usando objetos armazenados no Active Directory Domain Services. Isso é conhecido como *publicação de serviço*. Os clientes consultam o diretório para localizar serviços. Isso é chamado *de reunião do serviço cliente*. Esta seção aborda os tipos de objetos de diretório usados para publicação de serviço e explica como eles são usados para a reunião do serviço de cliente.

Esta seção aborda os seguintes tópicos:

-   [Uma visão geral da publicação de serviço](about-service-publication.md)
-   [Problemas de segurança para publicação de serviço](security-issues-for-service-publication.md)
-   [Objetos de ponto de conexão](connection-points.md)
-   [Publicando com pontos de conexão de serviço (SCPs)](publishing-with-service-connection-points.md)
-   [Quais informações armazenar em um ponto de conexão de serviço](service-connection-point-properties.md)
-   [Onde criar um ponto de conexão de serviço](where-to-create-a-service-connection-point.md)
-   [Como publicar serviços replicáveis, baseados em host e de banco de dados usando pontos de conexão de serviço](service-connection-points-for-replicated-host-based-and-database-services.md)
-   [Criando e mantendo um ponto de conexão de serviço](creating-and-maintaining-a-service-connection-point.md)
-   [Como um cliente consulta um SCP e o usa para associar a uma instância de serviço](how-clients-find-and-use-a-service-connection-point.md)
-   [Usando as APIs do serviço de nomes RPC (RpcNs) para publicar um serviço RPC](publishing-with-the-rpc-name-service-rpcns.md)
-   [RnR (registro e resolução do Windows Sockets) para publicar um serviço do Windows Sockets](publishing-with-windows-sockets-registration-and-resolution.md)
-   [Publicação de serviços baseados em COM no repositório de classe COM+](publishing-com-services.md)

Para obter mais informações sobre como os serviços e clientes se autenticam, consulte [autenticação mútua usando o Kerberos](mutual-authentication-using-kerberos.md). Para obter mais informações sobre contextos de segurança de serviço e contas de logon, consulte [contas de logon de serviço](service-logon-accounts.md).
