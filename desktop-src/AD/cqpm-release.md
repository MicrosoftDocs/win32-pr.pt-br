# <a name="cqpm_release-message"></a>Mensagem de liberação do CQPM \_

A mensagem de **\_ liberação CQPM** é enviada para a função de retorno de chamada [**CQPageProc**](/windows/desktop/api/Cmnquery/nc-cmnquery-lpcqpageproc) de uma página de extensão de formulário de consulta quando a página está prestes a ser descarregada.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*wParam* 
</dt> <dd>

Não usado.

</dd> <dt>

*lParam* 
</dt> <dd>

Não usado.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

O valor de retorno para essa mensagem é ignorado.
