# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Decidindo quais atributos recuperar para cada objeto encontrado

Se você pesquisar objetos de uma classe específica, parece lógico que você recupere atributos que realmente existem nos objetos dessa classe. Siga as mesmas regras discutidas em [Localizar uma lista de atributos a consultar.](finding-a-list-of-attributes-to-query.md) Esteja ciente de que atributos construídos podem ser recuperados na consulta, mas atributos construídos não podem ser usados em filtros.
