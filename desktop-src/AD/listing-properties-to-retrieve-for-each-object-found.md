# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Decidindo quais atributos recuperar para cada objeto encontrado

Se você Pesquisar objetos de uma determinada classe, parece lógico que você recuperou os atributos que realmente existem nos objetos dessa classe. Siga as mesmas regras discutidas em [encontrando uma lista de atributos a serem consultados](finding-a-list-of-attributes-to-query.md). Lembre-se de que os atributos construídos podem ser recuperados na consulta, mas atributos construídos não podem ser usados em filtros.
