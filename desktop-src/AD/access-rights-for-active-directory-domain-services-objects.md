# <a name="access-rights-for-active-directory-domain-services-objects"></a>Direitos de acesso para objetos de Active Directory Domain Services

Todos os objetos de diretório usam o mesmo conjunto de direitos de acesso predefinidos que correspondem às operações de diretório comuns. Esses direitos são descritos no tipo de enumeração Enumeração de [**\_ \_ direitos de ADS**](/windows/win32/api/iads/ne-iads-ads_rights_enum) .

Além disso, você pode definir direitos de acesso privados (controlar direitos de acesso) para operações que vão além daquelas cobertas pelos direitos predefinidos. Consulte [controlar direitos de acesso](control-access-rights.md).
