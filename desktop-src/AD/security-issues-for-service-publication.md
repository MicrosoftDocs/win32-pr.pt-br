# <a name="security-issues-for-service-publication"></a><span data-ttu-id="73b83-106">Problemas de segurança para publicação de serviço</span><span class="sxs-lookup"><span data-stu-id="73b83-106">Security Issues for Service Publication</span></span>

<span data-ttu-id="73b83-107">O sistema restringe a capacidade de criar, modificar ou excluir objetos de ponto de conexão.</span><span class="sxs-lookup"><span data-stu-id="73b83-107">The system restricts the ability to create, modify, or delete connection point objects.</span></span> <span data-ttu-id="73b83-108">Esteja atento e manipule essas restrições ao publicar um serviço.</span><span class="sxs-lookup"><span data-stu-id="73b83-108">Be aware of, and handle, these restrictions when you publish a service.</span></span>

<span data-ttu-id="73b83-109">Os clientes devem ser capazes de confiar nos dados publicados em um objeto de ponto de conexão no diretório.</span><span class="sxs-lookup"><span data-stu-id="73b83-109">Clients must be able to trust the data published in a connection point object in the directory.</span></span> <span data-ttu-id="73b83-110">Por esse motivo, a permissão para criar um objeto de ponto de conexão normalmente é restrita a usuários privilegiados, como administradores de domínio.</span><span class="sxs-lookup"><span data-stu-id="73b83-110">For this reason, permission to create a connection point object is typically restricted to privileged users, such as domain administrators.</span></span> <span data-ttu-id="73b83-111">Isso impede que usuários não autorizados de clientes decepcionante criem pontos de conexão inválidos para serviços bem conhecidos.</span><span class="sxs-lookup"><span data-stu-id="73b83-111">This prevents unauthorized users from deceiving clients by creating invalid connection points for well-known services.</span></span>

<span data-ttu-id="73b83-112">Os serviços não devem ser executados com privilégios de administrador de domínio.</span><span class="sxs-lookup"><span data-stu-id="73b83-112">Services must not run with domain administrator privileges.</span></span> <span data-ttu-id="73b83-113">Isso significa que um serviço normalmente não pode criar seu próprio ponto de conexão.</span><span class="sxs-lookup"><span data-stu-id="73b83-113">This means that a service typically cannot create its own connection point.</span></span> <span data-ttu-id="73b83-114">Em vez disso, você fornece um aplicativo de instalação ou configuração de serviço que cria o ponto de conexão.</span><span class="sxs-lookup"><span data-stu-id="73b83-114">Instead, you provide a service installation or configuration application that creates the connection point.</span></span> <span data-ttu-id="73b83-115">Esse instalador deve ser executado por um usuário com os privilégios necessários.</span><span class="sxs-lookup"><span data-stu-id="73b83-115">This installer must be run by a user with the necessary privileges.</span></span>

<span data-ttu-id="73b83-116">Embora um serviço normalmente não possa criar seu ponto de conexão, ele deve ser capaz de atualizar as propriedades do ponto de conexão em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="73b83-116">Although a service cannot typically create its connection point, it must be able to update the connection point properties at run time.</span></span> <span data-ttu-id="73b83-117">As propriedades do ponto de conexão contêm os dados de associação usados pelos clientes para se conectarem ao serviço.</span><span class="sxs-lookup"><span data-stu-id="73b83-117">The connection point properties contain the binding data used by clients to connect to the service.</span></span> <span data-ttu-id="73b83-118">Se os dados de associação forem alterados, o serviço deverá atualizar o ponto de conexão; caso contrário, os clientes não poderão usar o serviço.</span><span class="sxs-lookup"><span data-stu-id="73b83-118">If the binding data changes, the service must update the connection point; otherwise, clients cannot use the service.</span></span> <span data-ttu-id="73b83-119">Isso significa que o instalador também deve modificar o descritor de segurança no objeto de ponto de conexão para permitir que o serviço leia e grave as propriedades apropriadas em tempo de execução.</span><span class="sxs-lookup"><span data-stu-id="73b83-119">This means that the installer must also modify the security descriptor on the connection point object to enable the service to read and write the appropriate properties at run time.</span></span> <span data-ttu-id="73b83-120">Para obter mais informações e um exemplo de código, consulte [habilitando a conta de serviço para acessar as propriedades do SCP](enabling-service-account-to-access-scp-properties.md).</span><span class="sxs-lookup"><span data-stu-id="73b83-120">For more information and a code example, see [Enabling Service Account to Access SCP Properties](enabling-service-account-to-access-scp-properties.md).</span></span>

<span data-ttu-id="73b83-121">Um serviço em execução na conta LocalSystem pode criar um ponto de conexão como um objeto filho em seu próprio objeto de computador no diretório.</span><span class="sxs-lookup"><span data-stu-id="73b83-121">A service running under the LocalSystem account can create a connection point as a child object under its own computer object in the directory.</span></span> <span data-ttu-id="73b83-122">Esse serviço é uma exceção à regra de serviços que não cria seus próprios pontos de conexão.</span><span class="sxs-lookup"><span data-stu-id="73b83-122">Such a service is an exception to the rule of services not creating their own connection points.</span></span> <span data-ttu-id="73b83-123">Um serviço LocalSystem também tem permissão para modificar as propriedades de objetos de ponto de conexão em seu próprio objeto de computador.</span><span class="sxs-lookup"><span data-stu-id="73b83-123">A LocalSystem service also has permission to modify the properties of connection point objects under its own computer object.</span></span> <span data-ttu-id="73b83-124">Lembre-se de que um serviço deve ser executado na conta LocalSystem somente se for absolutamente necessário.</span><span class="sxs-lookup"><span data-stu-id="73b83-124">Be aware that a service should run under the LocalSystem account only if absolutely necessary.</span></span> <span data-ttu-id="73b83-125">Para obter mais informações, consulte [diretrizes para selecionar uma conta de logon de serviço](guidelines-for-selecting-a-service-logon-account.md).</span><span class="sxs-lookup"><span data-stu-id="73b83-125">For more information, see [Guidelines for Selecting a Service Logon Account](guidelines-for-selecting-a-service-logon-account.md).</span></span>

<span data-ttu-id="73b83-126">O aplicativo que cria um objeto de ponto de conexão ou qualquer objeto deve ter permissões criar filho para a classe de objeto a ser criada no contêiner em que o objeto será criado.</span><span class="sxs-lookup"><span data-stu-id="73b83-126">The application that creates a connection point object, or any object, must have create child permissions for the object class to be created in the container where the object will be created.</span></span> <span data-ttu-id="73b83-127">Para remover um objeto, o processo que executa a operação deve ter permissões de excluir filho para a classe de objeto a ser excluída no contêiner que contém o objeto ou ter permissões de exclusão no próprio objeto.</span><span class="sxs-lookup"><span data-stu-id="73b83-127">To remove an object, the process performing the operation must have delete child permissions for the object class to be deleted on the container holding the object, or have delete permissions on the object itself.</span></span> <span data-ttu-id="73b83-128">Para atualizar um ponto de conexão, o processo que executa a operação deve ter acesso de gravação às propriedades a serem atualizadas no objeto.</span><span class="sxs-lookup"><span data-stu-id="73b83-128">To update a connection point the process performing the operation must have write-access to the properties to be updated on the object.</span></span>
