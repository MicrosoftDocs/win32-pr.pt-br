# <a name="inheritance-ad-ds"></a>Herança (AD DS)

A [*herança*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) permite que uma determinada Ace seja propagada do contêiner onde ela foi aplicada a todos os filhos do contêiner. A herança pode ser combinada com a delegação para conceder direitos administrativos a uma subárvore inteira do diretório em uma única operação.
