# <a name="inheritance-ad-ds"></a>Herança (AD DS)

[*A*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) herança permite que uma determinada ACE seja propagada do contêiner em que ele foi aplicado a todos os filhos do contêiner. A herança pode ser combinada com a delegação para conceder direitos administrativos a uma subárvore inteira do diretório em uma única operação.
