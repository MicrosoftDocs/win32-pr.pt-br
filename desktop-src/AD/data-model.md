# <a name="data-model"></a>Modelo de dados

O modelo de dados para Active Directory Domain Services é derivado do modelo de dados X. 500. O diretório contém objetos que representam as coisas de várias classificações, descritas por atributos. O universo de objetos que podem ser armazenados no diretório é definido no esquema. Para cada classe de objeto, o esquema define quais atributos uma instância da classe deve ter, quais atributos adicionais ele pode ter e qual classe de objeto pode ser pai da classe de objeto atual.
