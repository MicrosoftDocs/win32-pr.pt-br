# <a name="supported-installation-mechanisms"></a>Mecanismos de instalação com suporte

Active Directory Domain Services dá suporte a quatro mecanismos para estender o esquema de Active Directory. Você pode estender o esquema de Active Directory usando

-   [Scripts LDIF](ldif-scripts.md) (preferencial)
-   [Scripts de valores separados por vírgulas (CSV)](comma-separated-value-csv-scripts.md)
-   [Extensão programática](programmatic-extension.md)
-   e [estendendo a interface do usuário para objetos de diretório](extending-the-user-interface-for-directory-objects.md).
