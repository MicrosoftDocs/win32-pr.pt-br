# <a name="searching-in-active-directory-domain-services"></a>Pesquisando no Active Directory Domain Services

A pesquisa é um recurso importante do Active Directory Domain Services. A pesquisa permite que você encontre objetos no diretório com base em critérios de seleção (consulta) e recupere as propriedades especificadas para os objetos encontrados.

Pesquisar em Active Directory Domain Services é uma questão de localizar um controlador de domínio (DC), a associação ao objeto em que a pesquisa deve começar no diretório, enviar uma consulta e processar os resultados.
