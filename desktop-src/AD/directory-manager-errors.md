# <a name="directory-manager-errors"></a>Erros do Gerenciador de diretórios

Este tópico contém os valores de retorno para erros do Gerenciador de diretórios de funções no Active Directory Domain Services.



| Valor                    | Descrição                                              |
|--------------------------|----------------------------------------------------------|
| **hrPMRecDeleted**       | O registro foi excluído.                                  |
| **hrRemainingVersions**  | Há um trabalho ocioso restante.                            |
| **hrFLDKeyTooBig**       | A chave foi truncada porque tem mais de 255 bytes. |
| **hrFLDTooManySegments** | Há muitos segmentos de chave.                         |
| **hrFLDNullKey**         | A chave é **nula**.                                     |
