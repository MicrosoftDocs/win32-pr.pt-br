# <a name="active-directory-schema-ad-ds"></a>Esquema de Active Directory (AD DS)

O esquema de Active Directory contém definições formais de cada classe de objeto que podem ser criadas em uma floresta Active Directory. O esquema também contém definições formais de cada atributo que podem existir em um objeto Active Directory.

Os tópicos que fornecem uma visão geral do esquema de Active Directory incluem:

-   [Implementação de esquema](schema-implementation.md)
-   [Características das classes de objeto](characteristics-of-object-classes.md)
-   [Características dos atributos](characteristics-of-attributes.md)
-   [O esquema abstrato](the-abstract-schema.md)

Para obter mais informações sobre programação de esquema, incluindo a leitura do esquema e a definição de novas classes e atributos no esquema, consulte [estendendo o esquema](extending-the-schema.md).

Para obter mais informações, consulte a [referência de Active Directory Domain Services](active-directory-domain-services-reference.md). Para obter as páginas de referência para as classes de esquema predefinidas, atributos e sintaxes de atributo, consulte a [referência de esquema de Active Directory](/windows/desktop/ADSchema/active-directory-schema).
