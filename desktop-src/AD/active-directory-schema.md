# <a name="active-directory-schema-ad-ds"></a>Esquema do Active Directory (AD DS)

O esquema do Active Directory contém definições formais de cada classe de objeto que pode ser criada em uma floresta do Active Directory. O esquema também contém definições formais de cada atributo que pode existir em um objeto do Active Directory.

Os tópicos que fornecem uma visão geral do esquema do Active Directory incluem:

-   [Implementação de esquema](schema-implementation.md)
-   [Características das classes Object](characteristics-of-object-classes.md)
-   [Características de atributos](characteristics-of-attributes.md)
-   [O esquema abstrato](the-abstract-schema.md)

Para obter mais informações sobre programação de esquema, incluindo a leitura do esquema e a definição de novas classes e atributos no esquema, consulte [Estendendo o esquema](extending-the-schema.md).

Para obter mais informações, consulte a [referência Active Directory Domain Services dados](active-directory-domain-services-reference.md). Para as páginas de referência para as classes de esquema predefinidas, atributos e sintaxes de atributo, consulte a Referência [de esquema do Active Directory](/windows/desktop/ADSchema/active-directory-schema).
