# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Gerenciando usuários em servidores membros e no Windows 2000 Professional

Em servidores membros e no Windows 2000 Professional, há um banco de dados de segurança local. Esse banco de dados de segurança local pode conter seus próprios usuários locais cujo escopo é apenas o computador em que eles são criados. Ao gerenciar esses tipos de usuários em estações de trabalho e servidores membros, você usa o provedor WinNT.

Ao gerenciar usuários em um domínio do Windows 2000 usando ADSI, você usa o provedor LDAP. Ao gerenciar usuários em estações de trabalho e servidores membros, você usa o provedor WinNT.
