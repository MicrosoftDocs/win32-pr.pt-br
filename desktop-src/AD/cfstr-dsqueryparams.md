# <a name="cfstr_dsqueryparams"></a>CFSTR \_ DSQUERYPARAMS

<dl> <dt>

<span id="CFSTR_DSQUERYPARAMS"></span><span id="cfstr_dsqueryparams"></span>**CFSTR \_ DSQUERYPARAMS**
</dt> <dd> <dl> <dt>

"DsQueryparameters"
</dt> <dt>



O formato de área de transferência **CFSTR \_ DSQUERYPARAMS** é suportado pelo [**IDataObject**](/windows/win32/api/objidl/nn-objidl-idataobject) fornecido pelo método [**ICommonQuery:: OpenQueryWindow**](/windows/win32/api/cmnquery/nf-cmnquery-icommonquery-openquerywindow) . O formato de área de transferência **CFSTR \_ DSQUERYPARAMS** fornece um identificador de memória global (**HGLOBAL**) que contém uma estrutura [**DSQUERYPARAMS**](/windows/desktop/api/Dsquery/ns-dsquery-dsqueryparams) .
