# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>Classes de provedor WMI no Active Directory Domain Services

Active Directory Domain Services ter as seguintes classes para uso com script WMI:

-   [**MSAD \_ DomainController**](msad-domaincontroller.md)
-   [**MSAD \_ NamingContext**](msad-namingcontext.md)
-   [**MSAD \_ ReplCursor**](msad-replcursor.md)
-   [**MSAD \_ ReplNeighbor**](msad-replneighbor.md)
-   [**MSAD \_ ReplPendingOp**](msad-replpendingop.md)

O próprio provedor é representado pela classe [**ReplicationProvider1**](replicationprovider1.md) , que é instância com o nome "ReplProv1".
