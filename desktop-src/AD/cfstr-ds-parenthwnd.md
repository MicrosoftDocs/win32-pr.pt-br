# <a name="cfstr_ds_parenthwnd"></a>CFSTR \_ DS \_ PARENTHWND

<dl> <dt>

<span id="CFSTR_DS_PARENTHWND"></span><span id="cfstr_ds_parenthwnd"></span>**CFSTR \_ DS \_ PARENTHWND**
</dt> <dd> <dl> <dt>

"DsAdminParentHwndClipFormat"
</dt> <dt>



O formato de área de transferência **CFSTR \_ DS \_ PARENTHWND** fornece um identificador de memória global (**HGLOBAL**) que contém o identificador de janela (**HWND**) da janela principal do MMC.

> [!Note]  
> O valor do formato da área de transferência não está definido em um arquivo de cabeçalho publicado. Para usar esse valor de formato da área de transferência, defina-o no formato exato mostrado.
