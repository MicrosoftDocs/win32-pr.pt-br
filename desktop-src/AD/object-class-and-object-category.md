<span data-ttu-id="44a1c-104">Cada instância de uma classe de objeto tem uma propriedade [**objectClass**](/windows/desktop/ADSchema/a-objectclass) com valores múltiplos que identifica a classe da qual o objeto é uma instância, bem como todas as superclasses estruturais ou abstratas das quais essa classe é derivada.</span><span class="sxs-lookup"><span data-stu-id="44a1c-104">Each instance of an object class has a multi-valued [**objectClass**](/windows/desktop/ADSchema/a-objectclass) property that identifies the class of which the object is an instance, as well as all structural or abstract superclasses from which that class is derived.</span></span> <span data-ttu-id="44a1c-105">Assim, a propriedade **objectClass** de um objeto de usuário identificaria as classes [**Top**](/windows/desktop/ADSchema/c-top), [**Person**](/windows/desktop/ADSchema/c-person), [**OrganizationalPerson**](/windows/desktop/ADSchema/c-organizationalperson)e [**User**](/windows/desktop/ADSchema/c-user) .</span><span class="sxs-lookup"><span data-stu-id="44a1c-105">Thus, the **objectClass** property of a user object would identify the [**top**](/windows/desktop/ADSchema/c-top), [**person**](/windows/desktop/ADSchema/c-person), [**organizationalPerson**](/windows/desktop/ADSchema/c-organizationalperson), and [**user**](/windows/desktop/ADSchema/c-user) classes.</span></span> <span data-ttu-id="44a1c-106">A propriedade **objectClass** não inclui classes auxiliares na lista.</span><span class="sxs-lookup"><span data-stu-id="44a1c-106">The **objectClass** property does not include auxiliary classes in the list.</span></span> <span data-ttu-id="44a1c-107">O sistema define o valor de **objectClass** quando a instância do objeto é criada e não pode ser alterada.</span><span class="sxs-lookup"><span data-stu-id="44a1c-107">The system sets the **objectClass** value when the object instance is created and it cannot be changed.</span></span>

<span data-ttu-id="44a1c-111">Para obter mais informações e um exemplo de código que recupera a propriedade [**objectClass**](/windows/desktop/ADSchema/a-objectclass) de um objeto, consulte [recuperando o atributo objectClass](retrieving-the-objectclass-property.md).</span><span class="sxs-lookup"><span data-stu-id="44a1c-111">For more information, and a code example that retrieves an object's [**objectClass**](/windows/desktop/ADSchema/a-objectclass) property, see [Retrieving the objectClass Attribute](retrieving-the-objectclass-property.md).</span></span>
