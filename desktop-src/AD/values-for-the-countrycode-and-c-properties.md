# <a name="values-for-the-countrycode-and-c-properties"></a>Valores para as propriedades countryCode e c

Os valores das propriedades **CountryCode** e **c** são obtidos do ISO Standard 3166. Esse padrão está disponível no site [organização internacional de normalização](https://www.iso.org/) . O valor da propriedade **CountryCode** é o código numérico do país da ISO 3166 e o valor da propriedade **c** é a designação de país/região de duas letras do ISO 3166.
