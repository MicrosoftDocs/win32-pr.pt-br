# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Obtendo um OID raiz de uma autoridade de registro de nome ISO

A maneira preferida de obter um identificador de objeto raiz (OID) é solicitar um de uma autoridade de registro de nome ISO (International Standards Organization). Esta é uma ação única; Quando você tiver obtido um OID raiz, o espaço OID que ele define será seu e você mesmo poderá administrá-lo.

Normalmente, há uma taxa associada ao registro de um nome de organização e ao recebimento de um OID raiz. As políticas de registro e taxas variam de acordo com o país/região. No Estados Unidos, o NRA ISO é o American National Standards Institute (ANSI). Para obter mais informações sobre o procedimento de registro ANSI e o agendamento de taxa, consulte https://web.ansi.org . O ISO mantém uma lista de organizações membro em https://www.iso.ch . Se você estiver fora do Estados Unidos, deverá entrar em contato com a organização membro ISO do seu país/região para obter informações de registro de nome.

Independentemente da fonte usada para obter o OID, se você pretende estender o esquema de Active Directory e deseja aplicar para o logotipo certificado para Windows, você deve registrar seu OID com a Microsoft. Para obter mais informações sobre como registrar seu OID com a Microsoft, consulte [obtendo um identificador de objeto da Microsoft](obtaining-an-object-identifier-from-microsoft.md).
