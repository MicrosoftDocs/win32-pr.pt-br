# <a name="cfstr_ds_propsheetconfig"></a>CFSTR \_ DS \_ PROPSHEETCONFIG

<dl> <dt>

<span id="CFSTR_DS_PROPSHEETCONFIG"></span><span id="cfstr_ds_propsheetconfig"></span>**CFSTR \_ DS \_ PROPSHEETCONFIG**
</dt> <dd> <dl> <dt>

"DsPropSheetCfgClipFormat"
</dt> <dt>



O formato de área de transferência **CFSTR \_ DS \_ PROPSHEETCONFIG** fornece um identificador de memória global (**HGLOBAL**) que contém uma estrutura [**PROPSHEETCFG**](propsheetcfg.md) .

> [!Note]  
> O valor do formato da área de transferência não está definido em um arquivo de cabeçalho publicado. Para usar esse valor de formato da área de transferência, você mesmo deve defini-lo no formato exato mostrado.
