# <a name="system-errors-in-active-directory-domain-services"></a>Erros do sistema no Active Directory Domain Services

Este tópico contém valores de retorno de erro do sistema em funções do Active Directory Domain Services.



| Valor                             | Descrição                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **hrFileClose**<br/>        | Não é possível fechar o arquivo DOS.<br/>                              |
| **hrOutOfThreads**<br/>     | Não é possível iniciar um thread porque não há nenhum disponível.<br/> |
| **hrTooManyIO**<br/>        | O sistema está ocupado porque há muitas e/SS.<br/>        |
| **hrBFNotSynchronous**<br/> | A página de buffer foi removida.<br/>                          |
| **hrBFPageNotFound**<br/>   | Não é possível localizar a página.<br/>                                   |
| **hrBFInUse**<br/>          | Não é possível abandonar o buffer.<br/>                              |
