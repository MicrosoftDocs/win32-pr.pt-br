# <a name="user-address-book-attributes"></a>Atributos do catálogo de endereços do usuário

Os atributos do catálogo de endereços são usados para fornecer identificação e informações suplementares para um usuário. Seu conteúdo é definido pelo usuário e pela organização do usuário. Esses atributos podem ser exibidos e gerenciados pelo snap-in Active Directory usuário e computadores ou o WAB (catálogo de endereços do Windows).



| Atributo                         | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **c**                             | O país/região no endereço do usuário. O país/região é representado como o código de país/região de dois caracteres com base no ISO-3166. Para obter os códigos válidos, consulte [valores para CountryCode](values-for-the-countrycode-and-c-properties.md).<br/>                                                                                                                                                                                        |
| **co**                            | O país/região em que o usuário está localizado.                                                                                                                                                                                                                                                                                                                                                                                              |
| **registra**                         | Um comentário. Essa cadeia de caracteres pode ser uma cadeia de caracteres nula.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **department**                    | O nome do departamento no qual o usuário trabalha.                                                                                                                                                                                                                                                                                                                                                                                          |
| **descrição**                   | A descrição a ser exibida para o usuário.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **displayName**                   | O nome exibido no catálogo de endereços para um usuário específico. Normalmente, essa é a combinação do nome do usuário, do início do meio e do sobrenome.                                                                                                                                                                                                                                                                                        |
| **directReports**                 | A lista de usuários que se reportam diretamente ao usuário. Os usuários listados como relatórios são aqueles que têm o atributo **Manager** definido para esse usuário. Cada item na lista é uma referência vinculada ao objeto que representa o usuário; Portanto, o servidor de Active Directory atualiza automaticamente esse atributo quando o atributo **gerente** de um usuário adiciona ou remove esse usuário como um gerente. Os itens são representados como nomes distintos. |
| **facsimileTelephoneNumber**      | O número de telefone do aparelho de fax comercial do usuário.                                                                                                                                                                                                                                                                                                                                                                                      |
| **givenName**                     | O nome (primeiro nome) do usuário.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **homePhone**                     | O número de telefone residencial principal do usuário.                                                                                                                                                                                                                                                                                                                                                                                               |
| **iniciais**                      | As iniciais de partes do nome completo do usuário. Isso pode ser usado como a inicial do meio no catálogo de endereços do Windows.                                                                                                                                                                                                                                                                                                                           |
| **ipPhone**                       | Usado por telefonia.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **l**                             | A localidade, como cidade ou cidade, no endereço do usuário.                                                                                                                                                                                                                                                                                                                                                                                |
| **managedObjects**                | A lista de objetos que são gerenciados pelo usuário. Os objetos listados são aqueles que têm o atributo **ManagedBy** definido para esse usuário. Cada item na lista é uma referência vinculada ao objeto gerenciado; Portanto, o servidor de Active Directory atualiza automaticamente o atributo **managedObjects** quando o atributo **ManagedBy** de um objeto adiciona ou remove esse usuário como seu gerente. Os itens são representados como nomes distintos.  |
| **Manager**                       | O usuário que é o gerente do usuário. O objeto de usuário do gerente contém um atributo **DirectReports** que contém referências a todos os objetos de usuário que têm seu atributo **Manager** definido como o objeto de usuário do gerente.                                                                                                                                                                                                                     |
| **Serviço Móvel**                        | O número do telefone celular principal do usuário.                                                                                                                                                                                                                                                                                                                                                                                           |
| **otherFacsimileTelephoneNumber** | A lista de números de telefone de máquinas de fax alternativas para o usuário.                                                                                                                                                                                                                                                                                                                                                                         |
| **otherIpPhone**                  | Usado por telefonia.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **otherMobile**                   | A lista de números de telefone celular alternativos para o usuário.                                                                                                                                                                                                                                                                                                                                                                                |
| **otherPager**                    | A lista de números de telefone de pager alternativos para o usuário.                                                                                                                                                                                                                                                                                                                                                                                   |
| **otherTelephone**                | A lista de números de telefone comercial alternativos para o usuário.                                                                                                                                                                                                                                                                                                                                                                                |
| **pager**                         | O número de telefone do pager principal para o usuário.                                                                                                                                                                                                                                                                                                                                                                                              |
| **physicalDeliveryOfficeName**    | A localização do escritório na sede da empresa do usuário.                                                                                                                                                                                                                                                                                                                                                                                          |
| **postalAddress**                 | O endereço postal do usuário.                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **postalCode**                    | O CEP do endereço postal do usuário. O CEP é específico do país/região do usuário. No Estados Unidos da América, esse atributo contém o código ZIP.                                                                                                                                                                                                                                                               |
| **postOfficeBox**                 | O número ou o identificador da caixa de postagem do usuário.                                                                                                                                                                                                                                                                                                                                                                                       |
| **sn**                            | O sobrenome do usuário (nome de família ou último nome).                                                                                                                                                                                                                                                                                                                                                                                                |
| **St**                            | O estado ou a província no endereço do usuário.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **streetAddress**                 | O endereço da sede da empresa do usuário.                                                                                                                                                                                                                                                                                                                                                                                           |
| **telephoneNumber**               | O número de telefone principal do sede da empresa do usuário.                                                                                                                                                                                                                                                                                                                                                                                 |
| **title**                         | O cargo do usuário. Esse atributo é comumente usado para indicar o cargo formal, como programador sênior, em vez de classe de ocupação, como programador. Normalmente, ele não é usado para títulos de "sufixo", como esq. ou DDS. Exemplos: diretor de gerenciamento, programador II, professor associado e líder de desenvolvimento.<br/>                                                                                                    |
| **url**                           | A lista de URLs para as páginas da Web alternativas do usuário.                                                                                                                                                                                                                                                                                                                                                                                           |
| **wWWHomePage**                   | A URL da página da Web principal do usuário.                                                                                                                                                                                                                                                                                                                                                                                                       |
