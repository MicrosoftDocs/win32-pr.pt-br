# <a name="domain-object-user-interface-mapping"></a>Mapeamento de interface do usuário de objeto de domínio

Este tópico discute as folhas de propriedades de objeto de domínio no snap-in Active Directory usuários e computadores.

## <a name="general-property-sheet"></a>Folha de propriedades geral

A tabela a seguir mostra os rótulos de interface do usuário na folha de propriedades **geral** .



| Rótulo da interface do usuário                       | Active Directory atributo |
|--------------------------------|----------------------------|
| Nome de domínio (anterior ao Windows 2000) | DC\*                       |
| Descrição                    | descrição                |



 

\*O atributo DC fornece acesso ao primeiro dos componentes de domínio que não é necessariamente o mesmo que o DN NetBIOS.
