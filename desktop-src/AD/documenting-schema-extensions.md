# <a name="documenting-schema-extensions"></a>Documentando extensões de esquema

Além de fornecer um procedimento de instalação separado para extensões de esquema, a natureza das extensões de esquema deve ser claramente documentada. A documentação deve conter:

-   Uma lista de novas classes e atributos com uma explicação de cada um deles
-   Uma lista de classes padrão que são estendidas com novos atributos e uma lista de atributos
-   Uma lista de novas páginas de propriedades com uma explicação de como usar cada uma delas
-   Uma explicação de como o serviço usa as extensões de esquema
