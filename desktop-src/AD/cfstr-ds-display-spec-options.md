# <a name="cfstr_ds_display_spec_options"></a>\_Opções de \_ especificação de exibição \_ \_ do CFSTR DS

<dl> <dt>

<span id="CFSTR_DS_DISPLAY_SPEC_OPTIONS"></span><span id="cfstr_ds_display_spec_options"></span>**\_Opções de \_ especificação de exibição \_ \_ do CFSTR DS**
</dt> <dd> <dl> <dt>

"DsDisplaySpecOptions"
</dt> <dt>



O Active Directory usuários e computadores, o Active Directory sites e serviços e os snap-ins Active Directory domínios e relações de confiança dão suporte ao formato de área de transferência **CFSTR \_ DS \_ Display \_ spec \_ Options** . O formato da área de transferência **Opções de especificação de exibição do CFSTR \_ DS \_ \_ \_** fornece um **HGLOBAL** que contém uma estrutura [**DSDISPLAYSPECOPTIONS**](/windows/desktop/api/Dsclient/ns-dsclient-dsdisplayspecoptions) . O **DSDISPLAYSPECOPTIONS** contém dados de configuração para uso pela extensão.


</dt> </dl> </dd> <dt>

<span id="CFSTR_DSDISPLAYSPECOPTIONS"></span><span id="cfstr_dsdisplayspecoptions"></span>**CFSTR \_ DSDISPLAYSPECOPTIONS**
</dt> <dd> <dl> <dt>



O formato de área de transferência **CFSTR \_ DSDISPLAYSPECOPTIONS** é idêntico ao formato de área de transferência **Opções de especificação de exibição do CFSTR \_ \_ \_ \_ DS** .


</dt> </dl> </dd> </dl>
