A interface do usuário para Active Directory Domain Services inclui folhas de propriedades e outras interfaces de usuário para configurar e recuperar os atributos de classes usadas com frequência. Os tópicos nesta seção contêm tabelas que mapeiam rótulos que aparecem nas interfaces do usuário com os atributos correspondentes no esquema de Active Directory.
