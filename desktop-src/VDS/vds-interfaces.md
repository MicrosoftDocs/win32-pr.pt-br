# <a name="vds-interfaces"></a>Interfaces VDS

\[A partir do Windows 8 e do Windows Server 2012, a interface com do [serviço de disco virtual](virtual-disk-service-portal.md) é substituída pela [API de gerenciamento de armazenamento do Windows](/previous-versions/windows/desktop/stormgmt/windows-storage-management-api-portal).\]

Os objetos VDS (serviço de disco virtual) fornecem interfaces que expõem métodos para consultar, configurar e manter dispositivos de armazenamento.



| Interface                                                            | Descrição                                                                                                                                                                                           |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IEnumVdsObject**](/windows/desktop/api/Vds/nn-vds-ienumvdsobject)                             | Enumera por meio de um conjunto de objetos VDS de um determinado tipo.                                                                                                                                              |
| [**IVdsAdmin**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsadmin)                                       | Registra um provedor com VDS.                                                                                                                                                                        |
| [**IVdsAdvancedDisk**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk)                         | Cria e exclui partições e modifica atributos de partição.                                                                                                                                    |
| [**IVdsAdvancedDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk2)                       | Fornece um método para alterar os tipos de partição.                                                                                                                                                          |
| [**IVdsAdviseSink**](/windows/desktop/api/Vds/nn-vds-ivdsadvisesink)                             | Recebe notificações do VDS.                                                                                                                                                                           |
| [**IVdsAsync**](/windows/desktop/api/Vds/nn-vds-ivdsasync)                                       | Gerencia operações assíncronas.                                                                                                                                                                      |
| [**IVdsController**](/windows/desktop/api/Vds/nn-vds-ivdscontroller)                             | Expõe métodos para executar operações de consulta e configuração em um controlador.                                                                                                                    |
| [**IVdsControllerControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollercontrollerport) | Fornece um método para enumerar portas do controlador para uma classe que implementa a interface [**IVdsController**](/windows/desktop/api/Vds/nn-vds-ivdscontroller) .                                                                      |
| [**IVdsControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollerport)                     | Fornece métodos para executar operações de consulta e configuração em uma porta do controlador.                                                                                                              |
| [**IVdsCreatePartitionEx**](/windows/desktop/api/Vds/nn-vds-ivdscreatepartitionex)               | Cria uma partição em um disco básico.                                                                                                                                                                  |
| [**IVdsDisk**](/windows/desktop/api/Vds/nn-vds-ivdsdisk)                                         | Consulta e configura discos básicos e dinâmicos.                                                                                                                                                       |
| [**IVdsDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsdisk2)                                       | Fornece um método para definir o modo SAN de um disco para offline ou online.                                                                                                                                 |
| [**IVdsDiskPartitionMF**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf)                   | Fornece métodos para executar operações de gerenciamento do sistema de arquivos em partições.                                                                                                                          |
| [**IVdsDiskPartitionMF2**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf2)                 | Fornece um método para formatar uma partição com opções de formatação adicionais.                                                                                                                           |
| [**IVdsDrive**](/windows/desktop/api/Vds/nn-vds-ivdsdrive)                                       | Fornece métodos para executar operações de consulta e configuração em uma unidade.                                                                                                                        |
| [**IVdsDrive2**](/windows/desktop/api/Vds/nn-vds-ivdsdrive2)                                     | Fornece um método para consultar as propriedades de uma unidade.                                                                                                                                             |
| [**IVdsHbaPort**](/windows/desktop/api/Vds/nn-vds-ivdshbaport)                                   | Fornece métodos para consultar e interagir com portas HBA no sistema local.                                                                                                                            |
| [**IVdsHwProvider**](/windows/desktop/api/Vds/nn-vds-ivdshwprovider)                             | Fornece métodos para executar operações de consulta, reenumeração e atualização em um provedor de hardware.                                                                                                  |
| [**IVdsHwProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivate)               | Fornece um método que habilita o VDS a determinar se o provedor de hardware possui um LUN especificado.                                                                                                   |
| [**IVdsHwProviderPrivateMpio**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivatempio)       | Fornece um método que define o status dos caminhos provenientes de uma porta HBA específica para o provedor.                                                                                               |
| [**IVdsHwProviderStoragePools**](/windows/desktop/api/Vds/nn-vds-ivdshwproviderstoragepools)     | Fornece métodos para criar LUNs em um [pool de armazenamento](storage-pool-object.md) e enumerar os pools de armazenamento gerenciados por um provedor de hardware.                                                          |
| [**IVdsHwProviderType**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype)                     | Fornece um método para recuperar o tipo de provedor de hardware.                                                                                                                                          |
| [**IVdsHwProviderType2**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype2)                   | Esta interface não está implementada. Use [**IVdsHwProviderType**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype) em vez disso.                                                                                                      |
| [**IVdsIscsiInitiatorAdapter**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatoradapter)       | Fornece métodos para consultar e interagir com adaptadores de iniciador iSCSI no sistema local.                                                                                                             |
| [**IVdsIscsiInitiatorPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatorportal)         | Fornece métodos para consultar e interagir com portais do iniciador iSCSI no sistema local.                                                                                                              |
| [**IVdsIscsiPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportal)                           | Fornece métodos para executar operações de consulta e configuração em um portal iSCSI.                                                                                                                |
| [**IVdsIscsiPortalGroup**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportalgroup)                 | Fornece métodos para executar serviços de consulta e configuração em um grupo de portal iSCSI.                                                                                                            |
| [**IVdsIscsiPortalLocal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportallocal)                 | Fornece métodos para definir chaves pré-compartilhadas IPSEC específicas do iniciador local em um portal iSCSI.                                                                                                       |
| [**IVdsIscsiTarget**](/windows/desktop/api/Vds/nn-vds-ivdsiscsitarget)                           | Fornece métodos para executar operações de consulta e configuração em um destino iSCSI.                                                                                                                |
| [**IVdsLun**](/windows/desktop/api/Vds/nn-vds-ivdslun)                                           | Fornece métodos para executar operações de consulta e configuração em um LUN (número de unidade lógica).                                                                                                    |
| [**IVdsLun2**](/windows/desktop/api/Vds/nn-vds-ivdslun2)                                         | Fornece métodos para aplicar e consultar dicas de LUN (número de unidade lógica).                                                                                                                           |
| [**IVdsLunControllerPorts**](/windows/desktop/api/Vds/nn-vds-ivdsluncontrollerports)             | Fornece métodos para executar operações de configuração de porta do controlador em um LUN.                                                                                                                    |
| [**IVdsLunIscsi**](/windows/desktop/api/Vds/nn-vds-ivdsluniscsi)                                 | Fornece métodos para executar operações de consulta e configuração em um LUN iSCSI.                                                                                                                   |
| [**IVdsLunMpio**](/windows/desktop/api/Vds/nn-vds-ivdslunmpio)                                   | Fornece métodos para executar operações de consulta e configuração em um LUN com extensões de MPIO.                                                                                                     |
| [**IVdsLunNaming**](/windows/desktop/api/Vds/nn-vds-ivdslunnaming)                               | Fornece um método para nomear LUNs para uma classe que implementa a interface [**IVdsLun**](/windows/desktop/api/Vds/nn-vds-ivdslun) .                                                                                                     |
| [**IVdsLunNumber**](/windows/desktop/api/Vds/nn-vds-ivdslunnumber)                               | Fornece um método para consultar o número de LUN para um LUN.                                                                                                                                                  |
| [**IVdsLunPlex**](/windows/desktop/api/Vds/nn-vds-ivdslunplex)                                   | Fornece métodos para executar operações de consulta e configuração em um plex de LUN.                                                                                                                     |
| [**IVdsMaintenance**](/windows/desktop/api/Vds/nn-vds-ivdsmaintenance)                           | Fornece métodos para executar operações de manutenção em um subsistema, controlador ou unidade.                                                                                                          |
| [**IVdsOpenVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsopenvdisk)                               | Define métodos para gerenciar um disco virtual.                                                                                                                                                          |
| [**IVdsPack**](/windows/desktop/api/Vds/nn-vds-ivdspack)                                         | Consulta e configura pacotes que contêm discos e cria volumes.                                                                                                                                   |
| [**IVdsPack2**](/windows/desktop/api/Vds/nn-vds-ivdspack2)                                       | Fornece um método para criar volumes alinhados em um pacote.                                                                                                                                                |
| [**IVdsProvider**](/windows/desktop/api/Vds/nn-vds-ivdsprovider)                                 | Retorna as propriedades de um provedor de hardware ou de software.                                                                                                                                                 |
| [**IVdsProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsproviderprivate)                   | Fornece métodos para habilitar o VDS para executar operações diversas em objetos de provedor.                                                                                                               |
| [**IVdsProviderSupport**](/windows/desktop/api/Vds/nn-vds-ivdsprovidersupport)                   | Fornece um método para indicar quais versões das interfaces VDS têm suporte no provedor.                                                                                                      |
| [**IVdsRemovable**](/windows/desktop/api/Vds/nn-vds-ivdsremovable)                               | Consulta e ejeta discos removíveis, como CD-ROMs.                                                                                                                                                  |
| [**IVdsService**](/windows/desktop/api/Vds/nn-vds-ivdsservice)                                   | Fornece métodos de nível de serviço para trabalhar com VDS.                                                                                                                                                  |
| [**IVdsServiceHba**](/windows/desktop/api/Vds/nn-vds-ivdsservicehba)                             | Fornece um método para consultar portas HBA no sistema local.                                                                                                                                             |
| [**IVdsServiceIscsi**](/windows/desktop/api/Vds/nn-vds-ivdsserviceiscsi)                         | Fornece métodos para a interface com o serviço iniciador local.                                                                                                                                       |
| [**IVdsServiceLoader**](/windows/desktop/api/Vds/nn-vds-ivdsserviceloader)                       | Inicia o VDS.                                                                                                                                                                                         |
| [**IVdsServiceUninstallDisk**](/windows/desktop/api/Vds/nn-vds-ivdsserviceuninstalldisk)         | Fornece métodos para desinstalar discos básicos e dinâmicos.                                                                                                                                                |
| [**IVdsStoragePool**](/windows/desktop/api/Vds/nn-vds-ivdsstoragepool)                           | Fornece métodos para consultar informações e enumerar objetos relacionados para um [pool de armazenamento](storage-pool-object.md).                                                                                    |
| [**IVdsSubSystem**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem)                               | Fornece métodos para executar operações de consulta e configuração em um subsistema.                                                                                                                    |
| [**IVdsSubSystem2**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem2)                             | Fornece métodos para executar operações de configuração e consulta em um subsistema usando as estruturas PROP2 do [**VDS \_ HINTS2**](/windows/desktop/api/Vds/ns-vds-vds_hints2) e [**VDS \_ \_ subsistema \_**](/windows/desktop/api/Vds/ns-vds-vds_sub_system_prop2) . |
| [**IVdsSubSystemImportTarget**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemimporttarget)       | Fornece métodos para consultar e configurar o destino de importação padrão do VSS para o subsistema.                                                                                                              |
| [**IVdsSubSystemInterconnect**](/windows/desktop/api/Vds/nn-vds-ivdssubsysteminterconnect)       | Fornece um método para consultar os tipos de interconexão que são suportados por um subsistema.                                                                                                                  |
| [**IVdsSubSystemIscsi**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemiscsi)                     | Fornece métodos para consultar e configurar destinos e portais iSCSI em um subsistema.                                                                                                                     |
| [**IVdsSubSystemNaming**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemnaming)                   | Fornece um método para nomear subsistemas para uma classe que implementa a interface [**IVdsSubSystem**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem) .                                                                                   |
| [**IVdsSwProvider**](/windows/desktop/api/Vds/nn-vds-ivdsswprovider)                             | Executa operações de provedor de software.                                                                                                                                                                |
| [**IVdsVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsvdisk)                                       | Define métodos para gerenciar um disco virtual.                                                                                                                                                          |
| [**IVdsVdProvider**](/windows/desktop/api/Vds/nn-vds-ivdsvdprovider)                             | Define métodos para criar e gerenciar discos virtuais.                                                                                                                                              |
| [**IVdsVolume**](/windows/desktop/api/Vds/nn-vds-ivdsvolume)                                     | Cria e exclui plexes de volume e modifica os atributos de volume.                                                                                                                                    |
| [**IVdsVolume2**](/windows/desktop/api/Vds/nn-vds-ivdsvolume2)                                   | Fornece um método para retornar informações de propriedade de volume, incluindo os GUIDs de volume.                                                                                                              |
| [**IVdsVolumeMF**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf)                                 | Executa o caminho de acesso e as operações do sistema de arquivos em objetos de volume.                                                                                                                                    |
| [**IVdsVolumeMF2**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf2)                               | Fornece métodos para executar operações adicionais de gerenciamento do sistema de arquivos no objeto de volume.                                                                                                        |
| [**IVdsVolumeMF3**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf3)                               | Fornece métodos para executar operações adicionais de gerenciamento do sistema de arquivos no objeto de volume.                                                                                                        |
| [**IVdsVolumeOnline**](/windows/desktop/api/Vds/nn-vds-ivdsvolumeonline)                         | Fornece um método para colocar volumes únicos online.                                                                                                                                                     |
| [**IVdsVolumePlex**](/windows/desktop/api/Vds/nn-vds-ivdsvolumeplex)                             | Consulta o volume que o contém e repara extensões inadequadas.                                                                                                                                                |
| [**IVdsVolumeShrink**](/windows/desktop/api/Vds/nn-vds-ivdsvolumeshrink)                         | Fornece métodos para dar suporte à redução de volume.                                                                                                                                                         |
