<span data-ttu-id="08e68-107">O DFS dá suporte a namespaces de DFS *autônomos* , aqueles com um servidor host e namespaces *baseados em domínio* que têm vários servidores host e alta disponibilidade.</span><span class="sxs-lookup"><span data-stu-id="08e68-107">DFS supports *stand-alone* DFS namespaces, those with one host server, and *domain-based* namespaces that have multiple host servers and high availability.</span></span> <span data-ttu-id="08e68-108">Os *dados de topologia* do DFS para Namespaces baseados em domínio são armazenados em Active Directory.</span><span class="sxs-lookup"><span data-stu-id="08e68-108">The DFS *topology data* for domain-based namespaces is stored in Active Directory.</span></span> <span data-ttu-id="08e68-109">Os dados incluem a raiz DFS, os links DFS e os destinos DFS.</span><span class="sxs-lookup"><span data-stu-id="08e68-109">The data includes the DFS root, DFS links, and DFS targets.</span></span>
