# <a name="cloud-filter-functions"></a>Funções de filtro de nuvem

As funções a seguir são usadas na criação e manutenção de diretórios e arquivos de espaço reservado.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                  | Descrição                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CfCloseHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfclosehandle)<br/>                                                 | Fecha o identificador de arquivo ou diretório retornado por [**CfOpenFileWithOplock**](/windows/desktop/api/cfapi/nf-cfapi-cfopenfilewithoplock). Isso não deve ser usado com identificadores de arquivo padrão do Win32, somente em identificadores usados em CfApi. h.<br/> |
| [**CfConnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfconnectsyncroot)<br/>                                         | Inicia a comunicação bidirecional entre um provedor de sincronização e a API de filtro de sincronização.<br/>                                                                                                                   |
| [**CfConvertToPlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfconverttoplaceholder)<br/>                               | Converte um arquivo/diretório normal em um arquivo/diretório de espaço reservado.<br/>                                                                                                                                         |
| [**CfCreatePlaceholders**](/windows/desktop/api/cfapi/nf-cfapi-cfcreateplaceholders)<br/>                                   | Cria um ou mais novos arquivos de espaço reservado ou diretórios em uma árvore raiz de sincronização.<br/>                                                                                                                          |
| [**CfDisconnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfdisconnectsyncroot)<br/>                                   | Desconecta um canal de comunicação criado pelo [**CfConnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfconnectsyncroot).<br/>                                                                                                       |
| [**CfExecute**](/windows/desktop/api/cfapi/nf-cfapi-cfexecute)<br/>                                                         | O ponto de entrada principal para todas as operações de espaço reservado com base na chave de conexão. Ele se destina a ser usado por um provedor de sincronização para responder a vários retornos de chamada da plataforma.<br/>                                 |
| [**CfGetCorrelationVector**](/windows/desktop/api/cfapi/nf-cfapi-cfgetcorrelationvector)<br/>                               | Permite que o provedor de sincronização consulte o vetor de correlação atual para um determinado arquivo de espaço reservado.<br/>                                                                                                            |
| [**CfGetPlaceholderInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderinfo)<br/>                                   | Obtém várias características de um arquivo ou pasta de espaço reservado.<br/>                                                                                                                                             |
| [**CfGetPlaceholderRangeInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderrangeinfo)<br/>                         | Obtém informações de intervalo sobre um arquivo ou pasta de espaço reservado.<br/>                                                                                                                                                |
| [**CfGetPlaceholderStateFromAttributeTag**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderstatefromattributetag)<br/> | Obtém um conjunto de Estados de espaço reservado com base nos valores *FileAttributes* e *ReparseTag* do arquivo.<br/>                                                                                                       |
| [**CfGetPlaceholderStateFromFileInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderstatefromfileinfo)<br/>         | Obtém um conjunto de Estados de espaço reservado com base nas várias informações do arquivo.<br/>                                                                                                                            |
| [**CfGetPlaceholderStateFromFindData**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplaceholderstatefromfinddata)<br/>         | Obtém um conjunto de Estados de espaço reservado com base na estrutura de dados de localização do WIN32 \_ \_ .<br/>                                                                                                                                |
| [**CfGetPlatformInfo**](/windows/desktop/api/cfapi/nf-cfapi-cfgetplatforminfo)<br/>                                         | Obtém as informações de versão da plataforma.<br/>                                                                                                                                                                    |
| [**CfGetSyncRootInfoByHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfgetsyncrootinfobyhandle)<br/>                         | Obtém várias características da raiz de sincronização que contém um determinado arquivo especificado por um identificador de arquivo.<br/>                                                                                                         |
| [**CfGetSyncRootInfoByPath**](/windows/desktop/api/cfapi/nf-cfapi-cfgetsyncrootinfobypath)<br/>                             | Obtém várias informações de raiz de sincronização dadas um arquivo sob a raiz de sincronização.<br/>                                                                                                                                      |
| [**CfGetTransferKey**](/windows/desktop/api/cfapi/nf-cfapi-cfgettransferkey)<br/>                                           | Inicia uma transferência de dados para um arquivo ou pasta de espaço reservado.<br/>                                                                                                                                           |
| [**CfGetWin32HandleFromProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfgetwin32handlefromprotectedhandle)<br/>     | Converte um identificador protegido em um identificador do Win32 para que ele possa ser usado com todas as APIs do Win32 baseadas em identificador. <br/>                                                                                                   |
| [**CfHydratePlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfhydrateplaceholder)<br/>                                   | Hidratam um arquivo de espaço reservado garantindo que o intervalo de bytes especificado esteja presente no disco no espaço reservado. Isso é válido somente para arquivos.<br/>                                                                |
| [**CfOpenFileWithOplock**](/windows/desktop/api/cfapi/nf-cfapi-cfopenfilewithoplock)<br/>                                   | Abre um identificador opaco assíncrono para um arquivo ou diretório (para arquivos normais e de espaço reservado) e configura um oplock apropriado com base nos sinalizadores abertos.<br/>                                             |
| [**CfQuerySyncProviderStatus**](/windows/desktop/api/cfapi/nf-cfapi-cfquerysyncproviderstatus)<br/>                         | Consulta um provedor de sincronização para obter o status do provedor.<br/>                                                                                                                                                |
| [**CfReferenceProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfreferenceprotectedhandle)<br/>                       | Permite que o chamador referencie um identificador protegido para um identificador do Win32 que pode ser usado com APIs do Win32 não CfApi. <br/>                                                                                         |
| [**CfRegisterSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfregistersyncroot)<br/>                                       | Executa um registro de raiz de sincronização de uma vez.<br/>                                                                                                                                                               |
| [**CfReleaseProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfreleaseprotectedhandle)<br/>                           | Libera um identificador protegido referenciado por [**CfReferenceProtectedHandle**](/windows/desktop/api/cfapi/nf-cfapi-cfreferenceprotectedhandle).<br/>                                                                                          |
| [**CfReleaseTransferKey**](/windows/desktop/api/cfapi/nf-cfapi-cfreleasetransferkey)<br/>                                   | Libera uma chave de transferência obtida por [**CfGetTransferKey**](/windows/desktop/api/cfapi/nf-cfapi-cfgettransferkey).<br/>                                                                                                                    |
| [**CfReportProviderProgress**](/windows/desktop/api/cfapi/nf-cfapi-cfreportproviderprogress)<br/>                           | Permite que um provedor de sincronização relate o progresso fora de banda.<br/>                                                                                                                                                    |
| [**CfReportSyncStatus**](/windows/desktop/api/cfapi/nf-cfapi-cfreportsyncstatus)<br/>                                       | Permite que um provedor de sincronização Notifique a plataforma de seu status em uma raiz de sincronização especificada sem precisar se conectar a uma chamada para [**CfConnectSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfconnectsyncroot) primeiro. <br/>                 |
| [**CfRevertPlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfrevertplaceholder)<br/>                                     | Reverte um espaço reservado para um arquivo regular, eliminando todas as características especiais, como a marca de nova análise, a identidade do arquivo, etc.<br/>                                                                 |
| [**CfSetCorrelationVector**](/windows/desktop/api/cfapi/nf-cfapi-cfsetcorrelationvector)<br/>                               | Permite que um provedor de sincronização instrua a plataforma a usar um vetor de correlação específico para fins de telemetria em um arquivo de espaço reservado. Isso é opcional.<br/>                                                      |
| [**CfSetInSyncState**](/windows/desktop/api/cfapi/nf-cfapi-cfsetinsyncstate)<br/>                                           | Define o estado em sincronização para um arquivo ou pasta de espaço reservado.<br/>                                                                                                                                                  |
| [**CfSetPinState**](/windows/desktop/api/cfapi/nf-cfapi-cfsetpinstate)<br/>                                                 | Isso define o estado do PIN de um espaço reservado, usado para representar uma intenção do usuário. Qualquer aplicativo (não apenas o provedor de sincronização) pode chamar essa função.<br/>                                                         |
| [**CfUnregisterSyncRoot**](/windows/desktop/api/cfapi/nf-cfapi-cfunregistersyncroot)<br/>                                   | Cancela o registro de uma raiz de sincronização previamente registrada.<br/>                                                                                                                                                            |
| [**CfUpdatePlaceholder**](/windows/desktop/api/cfapi/nf-cfapi-cfupdateplaceholder)<br/>                                     | Atualiza as características do arquivo ou diretório de espaço reservado.<br/>                                                                                                                                             |
| [**CfUpdateSyncProviderStatus**](/windows/desktop/api/cfapi/nf-cfapi-cfupdatesyncproviderstatus)<br/>                       | Atualiza o status atual do provedor de sincronização.<br/>                                                                                                                                                          |
