# <a name="cim_replacementset-class"></a>Classe de substituição de CIM \_

A classe de conjunto de **\_ substituição CIM** agrega elementos físicos que devem ser substituídos juntos. Por exemplo, ao substituir um cartão de memória, os chips de memória do componente também podem ser removidos e substituídos. Ou, essa associação pode ser usada para substituir ou atualizar um conjunto de chips de memória.

> [!IMPORTANT]
> As classes DMTF (Distributed Management Task Force) CIM (modelo CIM) são as classes pai nas quais as classes WMI são criadas. Atualmente, o WMI dá suporte apenas aos [esquemas de versão do CIM 2. x](https://dmtf.org/standards/cim/schemas).

 

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[Abstract, UUID("{FAF76B6C-798C-11D2-AAD1-006008C78BC7}"), AMENDMENT]
class CIM_ReplacementSet
{
  string Description;
  string Name;
};
```
