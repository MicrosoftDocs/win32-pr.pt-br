# <a name="wmi-service-management-classes"></a>Classes de gerenciamento de serviço WMI

As classes de gerenciamento de serviço WMI são usadas para gerenciar o serviço WMI em si e não o sistema de computador ou a rede corporativa. O gerenciamento do WMI abrange a configuração do serviço WMI e o gerenciamento de operações WMI.

A categoria Gerenciamento de Serviços WMI inclui as seguintes subcategorias de classes:

-   [Classes de configuração WMI](#wmi-configuration-classes)
-   [Classes de gerenciamento WMI](#wmi-management-classes)

## <a name="wmi-configuration-classes"></a>Classes de configuração WMI

A classe De configuração WMI deriva métodos que configuram o serviço WMI.

A seguir está uma tabela de classes de configuração WMI.



| Classe                                                             | Descrição                                                                     |
|-------------------------------------------------------------------|---------------------------------------------------------------------------------|
| [**Método \_ Win32ParameterClass**](win32-methodparameterclass.md) | Abstrata, classe base que implementa parâmetros de método derivados dessa classe. |



 

## <a name="wmi-management-classes"></a>Classes de gerenciamento WMI

As classes de gerenciamento WMI representam parâmetros operacionais para o serviço WMI.

A seguir está uma tabela de classes de gerenciamento WMI.



| Classe                                                       | Descrição                                                                                   |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**Win32 \_ WMISetting**](win32-wmisetting.md)               | Contém os parâmetros operacionais para o serviço WMI.                                      |
| [**Win32 \_ WMIElementSetting**](win32-wmielementsetting.md) | Associação entre um serviço em execução no Windows e as configurações de WMI que ele pode usar. |
