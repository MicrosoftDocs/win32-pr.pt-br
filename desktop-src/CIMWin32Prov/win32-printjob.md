<span data-ttu-id="58e11-324">O exemplo de código Perl a seguir descreve como recuperar estatísticas de trabalhos de impressora de instâncias do **Win32 \_ PrintJob**.</span><span class="sxs-lookup"><span data-stu-id="58e11-324">The following Perl code sample describes how to retrieve printer jobs stats from instances of **Win32\_PrintJob**.</span></span>


```
use strict;
use Win32::OLE;

close (STDERR);

my ($PrintJobset, $PrintJob);
eval {$PrintJobset = Win32::OLE->GetObject("winmgmts:{impersonationLevel=impersonate}")->
 InstancesOf ("Win32_PrintJob") };
if (!$@ && defined $PrintJobset)
{
 if ($PrintJobset->{Count} == 0 ) 
 {
  print "\nNo print jobs!\n";
 }

 foreach $PrintJob (in $PrintJobset)
 {
  print $PrintJob->{Name} , "\n";
  print $PrintJob->{JobId} , "\n";
  print $PrintJob->{Status} , "\n";
  print $PrintJob->{TotalPages} , "\n";
 }
}
else
{
 print Win32::OLE->LastError, "\n";
}
```
