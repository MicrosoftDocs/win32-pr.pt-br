# <a name="win32_comclass-class"></a>\_Classe COMClass Win32

A [classe WMI](/windows/desktop/WmiSdk/retrieving-a-class) **\_ COMClass** abstrato do Win32 representa as propriedades de um componente de Component Object Model (com).

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[abstract, UUID("{0F73ED50-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMClass : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
