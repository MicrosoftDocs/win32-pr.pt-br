# <a name="win32_comsetting-class"></a>Classe de configuração do Win32 \_

A [classe WMI](/windows/desktop/WmiSdk/retrieving-a-class) abstrata de **\_ configuração do Win32** representa as configurações associadas a um componente de Component Object Model (com) ou a um aplicativo com.

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[abstract, UUID("{E5D8A560-F6C0-11d2-B35E-00105A1F8569}"), AMENDMENT]
class Win32_COMSetting : CIM_Setting
{
  string Caption;
  string Description;
  string SettingID;
};
```
