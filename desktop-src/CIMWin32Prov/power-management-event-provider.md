# <a name="power-management-event-provider"></a>Provedor de eventos de gerenciamento de energia

o provedor de gerenciamento de energia permite que o WMI modele o Windows protocolos de gerenciamento de energia. Como um provedor de eventos, o provedor de gerenciamento de energia fornece informações para a classe [**Win32 \_ PowerManagementEvent**](win32-powermanagementevent.md) para descrever os eventos de gerenciamento de energia que resultam de alterações de estado de energia. Essas alterações de estado são associadas ao APM (gerenciamento avançado de energia) ou aos protocolos de gerenciamento de sistema da interface de energia e configuração avançada (ACPI). O nome da instância [**\_ \_ Win32Provider**](/windows/desktop/WmiSdk/--win32provider) é " \_ provedor de eventos de gerenciamento de energia MS \_ \_ \_ ".
