# <a name="win32_dcomapplication-class"></a>Classe \_ DCOMApplication win32

A **classe \_ WMI DCOMApplication win32** representa as propriedades de um aplicativo DCOM. [](/windows/desktop/WmiSdk/retrieving-a-class)

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[Dynamic, Provider("CIMWin32"), UUID("{0F73ED52-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_DCOMApplication : Win32_COMApplication
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
  string   AppID;
};
```
