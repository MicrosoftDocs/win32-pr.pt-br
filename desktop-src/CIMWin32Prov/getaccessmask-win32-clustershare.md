# <a name="getaccessmask-method-of-the-win32_clustershare-class"></a>Método GetAccessMask da classe ClusterShare do Win32 \_

Retorna um **bitmap uint32** com os direitos de acesso ao compartilhamento mantido pelo usuário ou grupo em cujo nome a instância é retornada.

## <a name="syntax"></a>Sintaxe


```mof
uint32 GetAccessMask();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Direitos de acesso ao compartilhamento mantido pelo usuário ou grupo.
