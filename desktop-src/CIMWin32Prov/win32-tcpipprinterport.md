# <a name="win32_tcpipprinterport-class"></a>\_Classe Win32 TCPIPPrinterPort

A [classe WMI](../wmisdk/retrieving-a-class.md) **\_ TCPIPPrinterPort do Win32** representa um ponto de acesso do serviço TCP/IP.

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
class Win32_TCPIPPrinterPort : CIM_ServiceAccessPoint
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Status;
  string   CreationClassName;
  string   Name;
  string   SystemCreationClassName;
  string   SystemName;
  uint32   Type;
  boolean  ByteCount;
  string   HostAddress;
  uint32   PortNumber;
  uint32   Protocol;
  string   Queue;
  string   SNMPCommunity;
  uint32   SNMPDevIndex;
  boolean  SNMPEnabled;
};
```

## <a name="remarks"></a>Comentários

A classe **Win32 \_ TCPIPPrinterPort** é derivada de [**\_ ServiceAccessPoint CIM**](cim-serviceaccesspoint.md) que deriva de [**CIM \_ LogicalElement**](cim-logicalelement.md).

O privilégio **SeLoadDriverPrivilege** é necessário para excluir uma instância dessa classe WMI. O trecho de script a seguir demonstra como estabelecer uma conexão com o WMI que usa esse privilégio.

`Set objWMIService = GetObject("winmgmts:{impersonationLevel=impersonate, (LoadDriver)}")`

## <a name="examples"></a>Exemplos

O exemplo do PowerShell a seguir remove uma impressora e a porta de impressora TCPIP associada.


```PowerShell
function Remove-PrinterAndPort{
    Param( $printername )
   $printer=gwmi win32_Printer -filter "name='HPDJ600'"
   $printer.Delete()
   $port=gwmi win32_tcpipprinterport -filter "name='$($printer.portname)'" -enableall
   $port.Delete()
}
```
