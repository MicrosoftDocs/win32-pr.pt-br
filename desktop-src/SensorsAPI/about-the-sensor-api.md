# <a name="about-the-sensor-api"></a>Sobre a API do sensor

A API do sensor fornece sua funcionalidade por meio de um conjunto de interfaces COM. A API do sensor expõe algumas dessas interfaces, o que significa que você pode chamar métodos nelas. Outras interfaces são definidas pela API para que você possa implementar os métodos necessários para que a API do sensor forneça notificações de eventos ao seu programa.
