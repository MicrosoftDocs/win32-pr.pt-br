# <a name="about-the-sensor-api"></a>Sobre a API do sensor

A API do Sensor fornece sua funcionalidade por meio de um conjunto de interfaces COM. A API do Sensor expõe algumas dessas interfaces, o que significa que você pode chamar métodos neles. Outras interfaces são definidas pela API para que você possa implementar os métodos que a API do Sensor precisa para fornecer notificações de eventos ao programa.
