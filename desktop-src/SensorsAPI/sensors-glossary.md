# <a name="sensor-api-glossary"></a>Glossário da API do sensor

<dl> <dt>

<span id="winsensors.sensors_glossary_ambient_light_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_AMBIENT_LIGHT_SENSOR"></span>**sensor de luz ambiente**
</dt> <dd>

Um sensor que detecta o nível de iluminação atual.

</dd> <dt>

<span id="winsensors.sensors_glossary_geodetic_reference_system"></span><span id="WINSENSORS.SENSORS_GLOSSARY_GEODETIC_REFERENCE_SYSTEM"></span>**sistema de referência do geodésico**
</dt> <dd>

A combinação de um elipsoide, que especifica o tamanho e a forma da terra, e um ponto base do qual a latitude e a longitude de todos os outros pontos são referenciadas.

</dd> <dt>

<span id="winsensors.sensors_glossary_light-aware_user_interface__ui_"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LIGHT-AWARE_USER_INTERFACE__UI_"></span>**interface do usuário com reconhecimento claro (IU)**
</dt> <dd>

Um programa que usa dados de sensor de luz ambiente para otimizar sua interface do usuário e conteúdo para condições de iluminação específicas.

</dd> <dt>

<span id="winsensors.sensors_glossary_logical_id"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LOGICAL_ID"></span>**ID lógica**
</dt> <dd>

Um GUID (identificador global exclusivo) que identifica um sensor lógico específico. Uma ID lógica é necessária toda vez que você se conecta a um sensor lógico.

</dd> <dt>

<span id="winsensors.sensors_glossary_logical_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LOGICAL_SENSOR"></span>**sensor lógico**
</dt> <dd>

Um sensor que fornece dados sem usar um dispositivo de hardware. Por exemplo, um sensor lógico pode fornecer dados sobre o local atual do usuário usando um serviço que pesquisa um endereço IP em uma tabela.

</dd> <dt>

<span id="winsensors.sensors_glossary_lux_value"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LUX_VALUE"></span>**valor de Lux**
</dt> <dd>

A quantidade de luz, em lumens por metro quadrado, que ilumina uma superfície. Os sensores de luz ambiente fornecem dados em valores Lux.

</dd> <dt>

<span id="winsensors.sensors_glossary_reference_ellipsoid"></span><span id="WINSENSORS.SENSORS_GLOSSARY_REFERENCE_ELLIPSOID"></span>**Elipsoide de referência**
</dt> <dd>

Um elipsoide associado a um sistema de referência geodésico, cuja superfície aproxima o tamanho e a forma da terra.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR"></span>**sensores**
</dt> <dd>

Hardware ou software que pode detectar eventos externos ou condições ambientais. Por exemplo, um receptor de GPS (Global Positioning System) é um sensor que pode determinar a localização física atual de um usuário.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_category"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_CATEGORY"></span>**Categoria do sensor**
</dt> <dd>

Uma ampla classe de dispositivos de sensor. Por exemplo, um sensor em um controlador de jogo de vídeo que detecta a posição e o movimento da mão de um jogador pertence à categoria de orientação dos sensores.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_collection"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_COLLECTION"></span>**coleção de sensores**
</dt> <dd>

Vários sensores que podem ser trabalhados como um grupo. Por exemplo, uma coleção de sensores pode incluir todos os sensores de movimento disponíveis e um método poderia ser chamado para recuperar essa coleção.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_data_report"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_DATA_REPORT"></span>**relatório de dados do sensor**
</dt> <dd>

Um relatório que contém valores de campo de dados que são gerados por um sensor.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_device"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_DEVICE"></span>**dispositivo de sensor**
</dt> <dd>

Um dispositivo de hardware que pode detectar eventos externos ou condições ambientais. Um único dispositivo de hardware pode conter vários sensores.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_manager"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_MANAGER"></span>**Gerenciador de sensores**
</dt> <dd>

Um objeto de programação que mantém uma lista de sensores disponíveis e fornece acesso a eles.

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_type"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_TYPE"></span>**tipo de sensor**
</dt> <dd>

Um tipo específico de sensor, como um acelerômetro tridimensional.

</dd> </dl>
