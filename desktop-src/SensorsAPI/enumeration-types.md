# <a name="enumeration-types"></a>Tipos de enumeração

A API do sensor define os tipos de enumeração a seguir.



| Enumeração                                                                | Descrição                                      |
|----------------------------------------------------------------------------|--------------------------------------------------|
| [**SensorConnectionType**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorconnectiontype) | Define os tipos de conexões de dispositivo de sensor.      |
| [**Sensorstate**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorstate)                   | Define possíveis estados operacionais para sensores. |
