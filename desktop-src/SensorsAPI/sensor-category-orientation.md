# <a name="sensor_category_orientation"></a>\_orientação da categoria do sensor \_

A \_ categoria orientação da categoria do sensor \_ contém sensores que fornecem informações sobre a orientação física. Os comparações fornecem orientação de navegação, como as baseadas no norte magnético. Inclinometers medir a inclinação ou a elevação. Sensores de distância medem a proximidade de algum objeto com o sensor.
