# <a name="optimizing-the-user-experience"></a>Otimizando a experiência do usuário

Ao implementar uma interface de usuário com reconhecimento claro, você deve considerar cuidadosamente como os usuários podem reagir ao comportamento do programa. Evite transições dissonante ou alterações frequentes no conteúdo do programa e na interface do usuário. Transições suaves e graduais que ocorrem apenas conforme a necessidade são as melhores. O ideal é que seu programa seja testado com usuários em condições de iluminação e cenários de usuário do mundo real. Por fim, considere a criação de um mecanismo que permita que os usuários alterem manualmente as otimizações leves do programa ou desabilitem a funcionalidade.
