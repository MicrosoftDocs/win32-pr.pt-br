# <a name="com-interfaces-sensor-api"></a>Interfaces COM (API do sensor)

A API do sensor expõe as seguintes interfaces COM.



| Interface                                              | Descrição                                                                                                      |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**ILocationPermissions**](/windows/desktop/api/sensorsapi/nn-sensorsapi-ilocationpermissions)   | Fornece o status da configuração do sistema que permite aos usuários alterar as configurações de local.                         |
| [**ILogicalSensorManager**](/previous-versions/windows/desktop/legacy/dd318934(v=vs.85)) | Fornece métodos usados por autores de sensor lógico para gerenciar conexões de sensor lógico.                            |
| [**ISensor**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensor)                             | Representa um sensor.                                                                                             |
| [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection)         | Representa uma coleção de sensores.                                                                              |
| [**ISensorDataReport**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensordatareport)         | Representa um relatório de dados de sensor.                                                                                 |
| [**ISensorEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorevents)                 | A interface de retorno de chamada para receber eventos de sensor.                                                              |
| [**ISensorManager**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanager)               | Fornece métodos para descobrir e recuperar sensores disponíveis e um método para solicitar eventos do Gerenciador de sensor. |
| [**ISensorManagerEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanagerevents)   | A interface de retorno de chamada para receber eventos do Gerenciador de sensor.                                                      |
