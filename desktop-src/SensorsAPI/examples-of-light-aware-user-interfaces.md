# <a name="examples-of-light-aware-user-interfaces"></a>Exemplos de interfaces Light-Aware usuário

Agora que você entende alguns princípios básicos para otimizar sua interface do usuário para diferentes condições de iluminação, você pode ver alguns exemplos de como essa otimização faz a diferença ao exibir o conteúdo ao ar livre. As imagens a seguir são comparações lado a lado de laptops em linha direta.

As imagens a seguir comparam a interface do usuário com light-aware com a interface do usuário normal em um laptop na área de trabalho direta. A primeira fotografia mostra a aparência do conteúdo normalmente ao ar livre quando um laptop está usando suas configurações de exibição de uso de bateria. A segunda fotografia mostra a combinação de Brilho Adaptável e interface do usuário com luminosidade e como esses recursos podem trabalhar juntos para aumentar a leitura da tela.

![interface do usuário normal com brilho de 40%](images/ui-unaware.png)![interface do usuário com 100% de brilho](images/ui-lightaware.png)

As fotografias a seguir mostram um programa de navegação como visto ao ar livre com reconhecimento de luz desligado e ligado. O conteúdo interno não é acessível ao ar livre, enquanto o conteúdo externo é facilmente legível. Observe também como as reflexões são minimizadas quando um esquema de cores preto em branco é usado, conforme mostrado na segunda fotografia.

![interface do usuário de navegação normal](images/nav-photo-normal.png)![interface do usuário de navegação com luminária](images/nav-photo-contrast.png)
