# <a name="the-sensor-collection-object"></a>O objeto de coleção do sensor

O objeto de coleção de sensor contém um grupo de objetos de sensor.

Às vezes, você desejará trabalhar com sensores como um grupo. Por exemplo, talvez você queira usar o Gerenciador de sensor para descobrir todos os sensores de movimento disponíveis. O objeto de coleção de sensor fornece métodos para trabalhar com um grupo de objetos de sensor. Você pode usar a interface [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) para chamar métodos que contam, adicionam, removem e recuperam sensores na coleção.
