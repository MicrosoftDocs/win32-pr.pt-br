## <a name="purpose"></a><span data-ttu-id="dcfdc-104">Finalidade</span><span class="sxs-lookup"><span data-stu-id="dcfdc-104">Purpose</span></span>

<span data-ttu-id="dcfdc-105">O Windows 7 inclui suporte nativo para sensores, que são dispositivos que podem medir fenômenos físicos, como temperatura ou localização.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-105">Windows 7 includes native support for sensors, which are devices that can measure physical phenomena such as temperature or location.</span></span> <span data-ttu-id="dcfdc-106">Esta documentação descreve a API do sensor, que permite que os aplicativos obtenham e usem dados de sensores de forma padronizada.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-106">This documentation describes the Sensor API, which enables applications to get and use data from sensors in a standardized way.</span></span>

<span data-ttu-id="dcfdc-107">Como seres humanos, nós confiamos em nossos sentidos para nos fornecer informações sobre o mundo em torno dos EUA.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-107">As humans, we rely on our senses to provide us with information about the world around us.</span></span> <span data-ttu-id="dcfdc-108">Quando criamos máquinas para executar algum de nossos trabalhos, adicionamos mecanismos de sensor para que os computadores possam responder adequadamente às condições em constante mudança.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-108">When we create machines to take on some of our work, we add sensor mechanisms so the machines can respond appropriately to changing conditions.</span></span>

<span data-ttu-id="dcfdc-109">Por exemplo, os mecanismos de automóvel normalmente usam uma variedade de sensores.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-109">For example, automobile engines typically use a variety of sensors.</span></span> <span data-ttu-id="dcfdc-110">Esses sensores são monitorados por um computador integrado que ajusta continuamente as configurações, como o tempo do mecanismo, para maximizar o poder e a eficiência.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-110">These sensors are monitored by an onboard computer that continuously adjusts settings, such as engine timing, to maximize power and efficiency.</span></span> <span data-ttu-id="dcfdc-111">Uma televisão pode usar um sensor de luz ambiente para ajustar o brilho da imagem para corresponder às condições de sala em constante mudança.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-111">A television may use an ambient light sensor to adjust the brightness of the picture to match changing room conditions.</span></span> <span data-ttu-id="dcfdc-112">Até mesmo algo tão simples quanto um botão doorbell atua como um sensor rudimentar para detectar uma presença humana na porta.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-112">Even something as simple as a doorbell button acts as a rudimentary sensor to detect a human presence at the door.</span></span>

<span data-ttu-id="dcfdc-113">Embora o doorbell puramente mecânico atenda à sua finalidade, as informações fornecidas por sensores complexos se tornam muito mais poderosas quando combinadas com o software.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-113">While the purely mechanical doorbell fulfills its purpose, the information provided by complex sensors becomes far more powerful when it is combined with software.</span></span> <span data-ttu-id="dcfdc-114">Os sensores modernos podem fornecer muitos dados com muita rapidez e, em uma variedade de formatos, de forma que o software fornece um mecanismo natural para fazer sentido dos dados do sensor.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-114">Modern sensors can provide a lot of data very quickly, and in a variety of formats, so software provides a natural mechanism for making sense of sensor data.</span></span>

<span data-ttu-id="dcfdc-115">Hoje, os desenvolvedores de software podem escrever programas que usam sensores, mas a falta de padronização torna a programação de sensores uma tarefa árdua.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-115">Today, software developers can write programs that use sensors, but a lack of standardization makes programming for sensors an arduous task.</span></span> <span data-ttu-id="dcfdc-116">Depois que um programa baseado em sensor é concluído, geralmente é sempre dependente de um determinado tipo de hardware.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-116">After a sensor-based program is completed, it is usually forever dependent on a particular type of hardware.</span></span> <span data-ttu-id="dcfdc-117">Usar uma ou mais soluções verticais para habilitar a implantação de um sistema baseado em software limitou a integração de sensores com hardware de computador e, até agora, os computadores baseados em Windows não têm nenhuma exceção.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-117">Using one or more vertical solutions to enable deployment of a software-based system has limited the integration of sensors with computer hardware and, until now, Windows-based computers have been no exception.</span></span>

<span data-ttu-id="dcfdc-118">O Windows 7 inclui suporte nativo para sensores, expandido por uma nova plataforma de desenvolvimento para trabalhar com sensores, incluindo sensores de local, como dispositivos GPS.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-118">Windows 7 includes native support for sensors, expanded by a new development platform for working with sensors, including location sensors, such as GPS devices.</span></span> <span data-ttu-id="dcfdc-119">A plataforma de sensor e localização do Windows fornece uma maneira padrão para que os fabricantes de dispositivos exponham dispositivos de sensor a desenvolvedores e consumidores de software, fornecendo aos desenvolvedores uma API (interface de programação de aplicativo) padronizada para trabalhar com sensores e dados de sensor.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-119">The Windows Sensor and Location platform provides a standard way for device manufacturers to expose sensor devices to software developers and consumers, while providing developers with a standardized application programming interface (API) for working with sensors and sensor data.</span></span>

<span data-ttu-id="dcfdc-120">Sensores são dispositivos ou mecanismos que podem medir fenômenos físicos, fornecer dados descritivos ou fornecer informações sobre o estado de um objeto ou ambiente físico.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-120">Sensors are devices or mechanisms that can measure physical phenomena, provide descriptive data, or provide information about the state of a physical object or environment.</span></span> <span data-ttu-id="dcfdc-121">Os computadores podem fazer uso de sensores internos, sensores conectados por meio de conexões com ou sem fio, ou sensores que fornecem dados por meio de uma rede ou da Internet.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-121">Computers can make use of built-in sensors, sensors that are connected through wired or wireless connections, or sensors that provide data through a network or the Internet.</span></span>

<span data-ttu-id="dcfdc-122">A API do sensor fornece uma maneira padrão de acessar de forma programática os dados fornecidos pelos sensores.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-122">The Sensor API provides a standard way to programmatically access data that sensors provide.</span></span> <span data-ttu-id="dcfdc-123">A API do sensor padroniza:</span><span class="sxs-lookup"><span data-stu-id="dcfdc-123">The Sensor API standardizes:</span></span>

-   <span data-ttu-id="dcfdc-124">Categorias, tipos e propriedades do sensor.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-124">Sensor categories, types, and properties.</span></span>
-   <span data-ttu-id="dcfdc-125">Formatos de dados para tipos de sensor padrão.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-125">Data formats for standard sensor types.</span></span>
-   <span data-ttu-id="dcfdc-126">Interfaces COM para trabalhar com sensores e coleções de sensores.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-126">COM interfaces for working with sensors and collections of sensors.</span></span>
-   <span data-ttu-id="dcfdc-127">Mecanismos de eventos para recebimento assíncrono de dados de sensor.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-127">Event mechanisms for asynchronously receiving sensor data.</span></span>

<span data-ttu-id="dcfdc-128">A API do sensor também permite que você defina categorias, tipos, propriedades, formatos de dados e eventos de sensor personalizados.</span><span class="sxs-lookup"><span data-stu-id="dcfdc-128">The Sensor API also enables you to define custom sensor categories, types, properties, data formats, and events.</span></span>
