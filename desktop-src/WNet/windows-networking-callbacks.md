# <a name="windows-networking-callbacks"></a>Windows Retornos de chamada de rede

Os protótipos de função a seguir são usados em Windows Rede.



| Protótipo da função                                 | Descrição                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Protótipo para uma função que obtém um caminho de perfil do usuário.                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Protótipo de uma função que retorna as informações de política de conta de usuário associadas a um processo Windows específico. |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Protótipo de uma função que reconcilia um perfil local com seu perfil global correspondente.                        |
