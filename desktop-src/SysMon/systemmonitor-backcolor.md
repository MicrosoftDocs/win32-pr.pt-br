# <a name="systemmonitorbackcolor-property"></a>Propriedade SystemMonitor. BackColor

Recupera ou define a cor do plano de fundo das exibições de gráfico e relatório.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property BackColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valor da propriedade

Cor do plano de fundo das exibições de gráfico e relatório.

## <a name="remarks"></a>Comentários

Essa é uma propriedade de ambiente. O valor dessa propriedade é determinado pelo contêiner. Definir o valor dessa propriedade pode afetar a ilusão do controle e do contêiner sendo um único aplicativo.
