# <a name="systemmonitorcopy-method"></a>Método SystemMonitor:: Copy

Copia as configurações de Propriedade do controle, a lista de contadores e os dados do contador para a área de transferência como um objeto HTML.

## <a name="syntax"></a>Sintaxe


```VB
Sub Copy()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Somente uma cópia pode existir na área de transferência.
