# <a name="systemmonitorshowhorizontalgrid-property"></a>Propriedade SystemMonitor. ShowHorizontalGrid

Recupera ou define um valor que determina se as linhas de grade horizontais são exibidas no grafo.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property ShowHorizontalGrid As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Verdadeiro indica que as linhas de grade horizontais são exibidas no grafo; caso contrário, false. O valor padrão dessa propriedade é false.
