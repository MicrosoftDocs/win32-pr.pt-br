# <a name="systemmonitorcounters-property"></a>Propriedade SystemMonitor. Counters

Recupera os **contadores** que contêm uma coleção de objetos de [**coitem**](counteritem.md) .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property Counters As ICounters
```



## <a name="property-value"></a>Valor da propriedade

Um objeto **Counters** que contém uma coleção de objetos de [**coitem**](counteritem.md) .

## <a name="remarks"></a>Comentários

Essa é a propriedade padrão do objeto [**SystemMonitor**](systemmonitor.md) .
