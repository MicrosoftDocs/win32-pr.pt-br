# <a name="logfilescount-property"></a>Propriedade LogFiles. Count

Número de instâncias de [**LogFileItem**](logfileitem.md) na coleção.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property Count As Long
```



## <a name="property-value"></a>Valor da propriedade

Número de arquivos de log na coleção.
