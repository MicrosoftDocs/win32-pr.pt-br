# <a name="systemmonitorreset-method"></a>Método SystemMonitor:: Reset

Remove todos os objetos de [**comitem**](counteritem.md) do objeto de coleção de **contadores** e limpa as exibições de gráfico e relatório.

## <a name="syntax"></a>Sintaxe


```VB
Sub Reset()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
