# <a name="systemmonitordisplayproperties-method"></a>Método SystemMonitor::D isplayProperties

Exibe a caixa **de diálogo Graph Propriedades.**

## <a name="syntax"></a>Sintaxe


```VB
Sub DisplayProperties()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Essa caixa de diálogo permite que o usuário modifique as propriedades do controle Monitor do Sistema.
