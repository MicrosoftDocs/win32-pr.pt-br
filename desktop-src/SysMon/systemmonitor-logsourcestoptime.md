# <a name="systemmonitorlogsourcestoptime-property"></a>Propriedade SystemMonitor.LogSourceStopTime

Recupera o carimbo de data/hora do valor do contador mais recente de um contador em sua coleção de contadores que foi registrado nos arquivos de log.

## <a name="syntax"></a>Syntax


```VB
Property LogSourceStopTime As Date
```



## <a name="property-value"></a>Valor da propriedade

Carimbo de data/hora do valor do contador mais recente de um contador em sua coleção de contadores que foi registrado nos arquivos de log.
