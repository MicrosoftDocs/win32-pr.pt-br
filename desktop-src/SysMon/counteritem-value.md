# <a name="counteritemvalue-property"></a>Propriedade MyItem. Value

Recupera o último valor de amostra do contador.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property Value As Double
```



## <a name="property-value"></a>Valor da propriedade

Último valor de amostra do contador. O valor será-1 se ocorrer um erro.

## <a name="remarks"></a>Comentários

Essa é a propriedade padrão do objeto [**MYITEM**](counteritem.md) .
