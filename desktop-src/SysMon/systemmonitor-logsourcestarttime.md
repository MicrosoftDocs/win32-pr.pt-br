# <a name="systemmonitorlogsourcestarttime-property"></a>Propriedade SystemMonitor. LogSourceStartTime

Recupera o carimbo de data/hora do valor do contador mais antigo de um contador em sua coleção de contadores que foi registrada nos arquivos de log.

## <a name="syntax"></a>Syntax


```VB
Property LogSourceStartTime As Date
```



## <a name="property-value"></a>Valor da propriedade

Carimbo de data/hora do valor do contador mais antigo de um contador em sua coleção de contadores que foi registrado nos arquivos de log.
