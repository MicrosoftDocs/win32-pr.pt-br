# <a name="systemmonitorforecolor-property"></a>Propriedade SystemMonitor. ForeColor

Recupera ou define a cor do texto que aparece no controle.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property ForeColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valor da propriedade

Cor do texto que aparece no controle

## <a name="remarks"></a>Comentários

Essa é uma propriedade de ambiente. O valor dessa propriedade é determinado pelo contêiner. Definir o valor dessa propriedade pode afetar a ilusão do controle e do contêiner sendo um único aplicativo.
