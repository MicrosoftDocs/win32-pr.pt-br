# <a name="sysmon-classes"></a>Classes SYSMON

Você pode usar as classes a seguir para configurar o controle do monitor do sistema (SYSMON).



| Nome da classe                             | Descrição                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Item**](counteritem.md)     | Fornece informações de Propriedade do item de contador.                                                                  |
| [**Contadores**](counters.md)           | Fornece os métodos para gerenciar uma coleção de objetos de [**coitem**](counteritem.md) .                |
| [**LogFileItem**](logfileitem.md)     | Fornece uma propriedade para recuperar o caminho de um único arquivo de log acessado pelo controle do monitor do sistema.     |
| [**arquivos de log**](logfiles.md)           | Fornece os métodos e as propriedades para gerenciar uma coleção de objetos [**LogFileItem**](logfileitem.md) . |
| [**SystemMonitor**](systemmonitor.md) | Fornece os métodos e as propriedades usadas para configurar o controle do monitor do sistema.                            |
