# <a name="systemmonitorbackcolorctl-property"></a>Propriedade SystemMonitor. BackColorCtl

Recupera ou define a cor do plano de fundo do controle.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property BackColorCtl As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valor da propriedade

Cor do plano de fundo do controle.
