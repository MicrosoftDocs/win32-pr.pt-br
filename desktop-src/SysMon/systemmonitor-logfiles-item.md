# <a name="logfilesitem-property"></a>Propriedade LogFiles. Item

Recupera a instância [**LogFileItem**](logfileitem.md) especificada da coleção.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property Item( _
  ByVal index As Object _
) As LogFileItem
```



## <a name="property-value"></a>Valor da propriedade

Instância de [**LogFileItem**](logfileitem.md) que corresponde ao valor de índice especificado.

## <a name="remarks"></a>Comentários

Essa é a propriedade padrão do objeto [**LogFiles**](systemmonitor-logfiles.md) .
