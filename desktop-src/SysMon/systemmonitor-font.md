# <a name="systemmonitorfont-property"></a>Propriedade SystemMonitor.Font

Recupera ou define a fonte usada no controle .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property Font As stdole.IFontDisp
```



## <a name="property-value"></a>Valor da propriedade

Fonte usada para exibir texto no controle .

## <a name="remarks"></a>Comentários

Essa é uma propriedade de ambiente. O valor dessa propriedade é determinado pelo contêiner. Definir o valor dessa propriedade pode afetar a ilusão de que o controle e o contêiner são um único aplicativo.
