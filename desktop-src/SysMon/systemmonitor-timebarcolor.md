# <a name="systemmonitortimebarcolor-property"></a>Propriedade SystemMonitor. TimeBarColor

Recupera ou define a cor da barra de tempo. A barra de tempo é a barra vertical que se move pela janela do grafo para indicar a passagem de cada intervalo de amostragem na exibição de gráfico de linha.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property TimeBarColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valor da propriedade

Cor da barra de tempo.
