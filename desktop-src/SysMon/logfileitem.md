# <a name="logfileitem-object"></a>Objeto LogFileItem

Use essa classe para recuperar o caminho de um arquivo de log acessado pelo controle do monitor do sistema.

Para recuperar esse objeto, chame [**LogFiles. Item**](systemmonitor-logfiles-item.md).

## <a name="members"></a>Membros

O objeto **LogFileItem** tem estes tipos de membros:

-   [Propriedades](#properties)

### <a name="properties"></a>Propriedades

O objeto **LogFileItem** tem essas propriedades.



| Propriedade                                    | Descrição                                    |
|:--------------------------------------------|:-----------------------------------------------|
| [**Caminho**](logfileitem-path.md)<br/> | Recupera o caminho do arquivo de log.<br/> |
