# <a name="systemmonitorshowlegend-property"></a>Propriedade SystemMonitor. textlegend

Recupera ou define um valor que determina se a legenda é exibida. A legenda lista os contadores e mostra a cor usada para o gráfico de cada contador.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property ShowLegend As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Verdadeiro indica que a legenda é exibida no controle; caso contrário, false. O valor padrão para essa propriedade é true.
