# <a name="systemmonitoronsamplecollected-event"></a>Evento SystemMonitor. OnSampleCollected

Notifica você quando os valores de exemplo para os contadores foram coletados.

## <a name="syntax"></a>Sintaxe


```VB
SystemMonitor.OnSampleCollected()
```



## <a name="parameters"></a>Parâmetros

Este evento não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse evento não retorna um valor.
