# <a name="systemmonitorshowverticalgrid-property"></a>Propriedade SystemMonitor. ShowVerticalGrid

Recupera ou define um valor que determina se as linhas de grade verticais são exibidas no grafo.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property ShowVerticalGrid As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Verdadeiro indica que as linhas de grade verticais são exibidas no grafo; caso contrário, false. O valor padrão dessa propriedade é false.
