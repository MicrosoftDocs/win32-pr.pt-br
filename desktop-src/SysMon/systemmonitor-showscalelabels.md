# <a name="systemmonitorshowscalelabels-property"></a>Propriedade SystemMonitor. ShowScaleLabels

Recupera ou define um valor que determina se os rótulos de escala são exibidos no eixo vertical do grafo.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property ShowScaleLabels As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Verdadeiro indica que os rótulos de escala são exibidos no eixo vertical do grafo; caso contrário, false. O valor padrão para essa propriedade é true.
