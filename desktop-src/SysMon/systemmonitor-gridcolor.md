# <a name="systemmonitorgridcolor-property"></a>Propriedade SystemMonitor. GridColor

Recupera ou define a cor das linhas de grade usadas no grafo.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Property GridColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Valor da propriedade

Cor das linhas de grade horizontal e vertical usadas em um grafo se as propriedades [**ShowHorizontalGrid**](systemmonitor-showhorizontalgrid.md) ou [**ShowVerticalGrid**](systemmonitor-showverticalgrid.md) estiverem selecionadas.
