# <a name="tasknamedvaluepairvalue-property"></a>Propriedade TaskNamedValuePair. Value

Para scripts, Obtém ou define o valor associado a um nome em um par de nome-valor.

## <a name="syntax"></a>Syntax


```VB
TaskNamedValuePair.Value As String
```



## <a name="property-value"></a>Valor da propriedade

O valor que está associado a um nome em um par de nome-valor.
