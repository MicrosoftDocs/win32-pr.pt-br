# <a name="taskvariablessetoutput-method"></a>Método TaskVariables. SetOutput

Para scripts, define as variáveis de saída para uma tarefa. Este método não está implementado.

## <a name="syntax"></a>Sintaxe


```VB
TaskVariables.SetOutput( _
  ByVal input _
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*entrada* \[ do no\]
</dt> <dd>

As variáveis de saída para uma tarefa.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
