# <a name="taskfoldergetfolders-property"></a>Propriedade TaskFolder. GetFolders

Para scripts, o obtém todas as subpastas na pasta.

## <a name="syntax"></a>Syntax


```VB
TaskFolder.GetFolders( _
  ByVal flags, _
  ByRef folders _
)
```
