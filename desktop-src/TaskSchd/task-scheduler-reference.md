# <a name="task-scheduler-reference"></a>Agendador de Tarefas referência

Esta seção descreve as APIs, os objetos de script e o esquema XML fornecidos pelo Agendador de Tarefas.

Os tópicos da API incluem uma descrição da API, a sintaxe de declaração da API, comentários que descrevem condições especiais para a API e um bloco de requisitos que descreve qual plataforma, arquivos de header e bibliotecas são necessários.

Os tópicos de objeto de script incluem uma descrição do objeto, a sintaxe de declaração do objeto, comentários que descrevem condições especiais para o objeto e um bloco de requisitos que descreve quais bibliotecas de plataforma e tipo são necessárias.

Os tópicos de esquema XML incluem uma descrição das informações de elemento, pai, filho e atributo e comentários que descrevem condições especiais.
