# <a name="runningtaskcurrentaction-property"></a>Propriedade RunningTask. CurrentAction

Para scripts, obtém o nome da ação atual que a tarefa em execução está executando.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
RunningTask.CurrentAction As String
```



## <a name="property-value"></a>Valor da propriedade

O nome da ação atual que a tarefa em execução está executando.
