# <a name="emailactioncc-property"></a>Propriedade EmailAction.Cc

\[Não há mais suporte para este objeto. Use IExecAction com o cmdlet [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) do PowerShell como uma solução alternativa.\]

Para scripts, Obtém ou define o endereço de email ou endereços que você deseja CC na mensagem de email.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Sintaxe


```VB
EmailAction.Cc As String
```



## <a name="property-value"></a>Valor da propriedade

O endereço de email ou endereços que você deseja CC na mensagem de email.
