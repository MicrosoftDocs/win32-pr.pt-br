# <a name="taskvariablesgetcontext-method"></a>Método TaskVariables. GetContext

Para scripts, usado para compartilhar o contexto entre diferentes etapas e tarefas que estão na mesma instância de trabalho. Este método não está implementado.

## <a name="syntax"></a>Sintaxe


```VB
TaskVariables.GetContext( _
  ByRef context _
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*contexto* \[ do fora\]
</dt> <dd>

O contexto usado para compartilhar o contexto entre diferentes etapas e tarefas que estão na mesma instância de trabalho.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
