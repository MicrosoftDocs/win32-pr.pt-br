# <a name="starting-an-executable-weekly"></a>Iniciando um executável semanalmente

Escrever uma tarefa que inicia um executável semanalmente é feito definindo um gatilho semanal e uma ação executável.

## <a name="weekly-triggers"></a>Gatilhos semanais

Os gatilhos semanais usam o limite inicial para ativar o gatilho e para especificar a hora do dia em que a tarefa é executada. Quando o gatilho é ativado, o gatilho também especifica quando a tarefa é executada durante a semana e se a tarefa é executada todas as semanas, a cada semana, a cada terceira semana e assim por diante.
