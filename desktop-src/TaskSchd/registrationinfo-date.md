# <a name="registrationinfodate-property"></a>Propriedade RegistrationInfo. Date

Para scripts, Obtém ou define a data e a hora em que a tarefa é registrada.

## <a name="syntax"></a>Syntax


```VB
RegistrationInfo.Date As String
```



## <a name="property-value"></a>Valor da propriedade

A data de registro da tarefa.

## <a name="remarks"></a>Comentários

Ao ler ou gravar XML para uma tarefa, a data de registro é especificada usando o elemento [**Date**](taskschedulerschema-date-registrationinfotype-element.md) do esquema de Agendador de tarefas.
