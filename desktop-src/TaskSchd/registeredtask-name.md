# <a name="registeredtaskname-property"></a>Propriedade RegisteredTask.Name

Para scripts, obtém o nome da tarefa registrada.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Name As String
```



## <a name="property-value"></a>Valor da propriedade

O nome da tarefa registrada.
