# <a name="taskserviceconnected-property"></a>Propriedade TaskService. Connected

Para scripts, obtém um valor booliano que indica se você está conectado ao serviço de Agendador de Tarefas.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
TaskService.Connected
```



## <a name="property-value"></a>Valor da propriedade

Um valor booliano que indica se você está conectado ao serviço de Agendador de Tarefas.
