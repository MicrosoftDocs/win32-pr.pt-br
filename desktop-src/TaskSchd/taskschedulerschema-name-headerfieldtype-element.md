# <a name="name-headerfieldtype-element"></a>Elemento Name (headerFieldType)

Contém o nome de um campo de header em uma mensagem de email.

``` syntax
<xs:element name="Name"
    type="nonEmptyString"
 />
```

O **elemento Name** é definido pelo tipo complexo [**headerFieldType.**](taskschedulerschema-headerfieldtype-complextype.md)

## <a name="parent-element"></a>Elemento pai



| Elemento                                                                         | Derivado de                                                               | Descrição                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contém um campo para um header em uma mensagem de email. <br/> |
