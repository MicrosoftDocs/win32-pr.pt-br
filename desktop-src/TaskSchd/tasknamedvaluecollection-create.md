# <a name="tasknamedvaluecollectioncreate-method"></a>Método TaskNamedValueCollection. Create

Para scripts, cria um par de nome-valor na coleção.

## <a name="syntax"></a>Sintaxe


```VB
TaskNamedValueCollection.Create( _
  ByVal name, _
  ByVal value, _
  ByRef pair _
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*nome* \[ do no\]
</dt> <dd>

O nome que está associado a um valor em um par de nome-valor.

</dd> <dt>

*valor* \[ do no\]
</dt> <dd>

O valor que está associado a um nome em um par de nome-valor.

</dd> <dt>

*par* \[ fora\]
</dt> <dd>

O par nome-valor que é criado na coleção.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
