# <a name="runningtaskinstanceguid-property"></a>Propriedade RunningTask. InstanceGuid

Para scripts, obtém o identificador GUID para esta instância da tarefa.

## <a name="syntax"></a>Syntax


```VB
RunningTask.InstanceGuid As string
```



## <a name="property-value"></a>Valor da propriedade

O identificador de GUID para esta instância da tarefa. Um identificador é gerado pelo serviço de Agendador de Tarefas cada vez que a tarefa é executada.
