# <a name="taskserviceconnecteddomain-property"></a>Propriedade TaskService. ConnectedDomain

Para scripts, obtém o nome do domínio ao qual o computador [**TargetServer**](taskservice-targetserver.md) está conectado.

## <a name="syntax"></a>Syntax


```VB
TaskService.ConnectedDomain As String
```



## <a name="property-value"></a>Valor da propriedade

O domínio ao qual o computador [**TargetServer**](taskservice-targetserver.md) está conectado.
