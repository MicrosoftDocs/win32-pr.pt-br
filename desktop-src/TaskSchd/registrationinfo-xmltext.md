# <a name="registrationinfoxmltext-property"></a>RegistrationInfo.Xmlpropriedade Text

Para scripts, obtém ou define uma versão formatada em XML das informações de registro para a tarefa.

## <a name="syntax"></a>Syntax


```VB
RegistrationInfo.XmlText As String
```



## <a name="property-value"></a>Valor da propriedade

Uma versão formatada em XML das informações de registro de tarefa.
