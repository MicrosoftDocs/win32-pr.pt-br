# <a name="registrationinfoxmltext-property"></a>Propriedade de texto de RegistrationInfo.Xml

Para scripts, Obtém ou define uma versão formatada em XML das informações de registro da tarefa.

## <a name="syntax"></a>Syntax


```VB
RegistrationInfo.XmlText As String
```



## <a name="property-value"></a>Valor da propriedade

Uma versão formatada em XML das informações de registro da tarefa.
