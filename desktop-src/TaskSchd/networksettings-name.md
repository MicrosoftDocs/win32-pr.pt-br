# <a name="networksettingsname-property"></a>NetworkSettings.Name propriedade

Para scripts, obtém ou define o nome de um perfil de rede. O nome é usado para fins de exibição.

Essa propriedade é leitura/gravação.

## <a name="syntax"></a>Syntax


```VB
NetworkSettings.Name As String
```



## <a name="property-value"></a>Valor da propriedade

O nome de um perfil de rede.
