# <a name="emailactionto-property"></a>Propriedade EmailAction.To

\[Não há mais suporte para este objeto. Use IExecAction com o cmdlet [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) do PowerShell como uma solução alternativa.\]

Para scripts, Obtém ou define o endereço de email ou endereços para os quais você deseja enviar o email.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```VB
EmailAction.To As String
```



## <a name="property-value"></a>Valor da propriedade

O endereço de email ou endereços para os quais você deseja enviar o email.
