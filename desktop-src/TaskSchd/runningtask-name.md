# <a name="runningtaskname-property"></a>RunningTask.Name propriedade

Para scripts, obtém o nome da tarefa.

## <a name="syntax"></a>Syntax


```VB
RunningTask.Name As string
```



## <a name="property-value"></a>Valor da propriedade

O nome da tarefa.
