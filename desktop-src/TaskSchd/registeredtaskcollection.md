# <a name="registeredtaskcollection-object"></a>Objeto RegisteredTaskCollection

Objeto de script que contém todas as tarefas registradas.

## <a name="members"></a>Membros

O objeto **RegisteredTaskCollection** tem estes tipos de membros:

-   [Propriedades](#properties)

### <a name="properties"></a>Propriedades

O objeto **RegisteredTaskCollection** tem essas propriedades.



| Propriedade                                                   | Tipo de acesso          | Descrição                                                        |
|:-----------------------------------------------------------|:---------------------|:-------------------------------------------------------------------|
| [**Contar**](registeredtaskcollection-count.md)<br/> | Somente leitura<br/> | Obtém o número de tarefas registradas na coleção.<br/>  |
| [**Item**](registeredtaskcollection-item.md)<br/>   | Somente leitura<br/> | Obtém a tarefa registrada especificada da coleção.<br/> |
