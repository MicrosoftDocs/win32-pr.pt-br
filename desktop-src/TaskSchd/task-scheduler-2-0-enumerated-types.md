# <a name="task-scheduler-20-enumerated-types"></a>Agendador de Tarefas tipos enumerados 2.0

Descreve os tipos enumerados que definem as constantes usadas pelas APIs Agendador de Tarefas 2.0.


Os seguintes tipos enumerados são introduzidos no Agendador de Tarefas 2.0.



| Enumeração                                                                  | Descrição                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**TIPO \_ DE AÇÃO DE \_ TAREFA**](/windows/desktop/api/taskschd/ne-taskschd-task_action_type)                               | Define o tipo de ações que uma tarefa pode executar.                                                             |
| [**COMPATIBILIDADE \_ DE TAREFAS**](/windows/desktop/api/taskschd/ne-taskschd-task_compatibility)                            | Define com quais versões do Agendador de Tarefas ou do comando AT com as quais a tarefa é compatível.                      |
| [**CRIAÇÃO DE \_ TAREFA**](/windows/desktop/api/taskschd/ne-taskschd-task_creation)                                       | Define como o serviço Agendador de Tarefas cria, atualiza ou desabilita a tarefa.                                   |
| [**SINALIZADORES \_ DE ENUM \_ TASK**](/windows/desktop/api/taskschd/ne-taskschd-task_enum_flags)                                 | Define como o Agendador de Tarefas enumera por meio de tarefas registradas.                                              |
| [**POLÍTICA \_ DE INSTÂNCIAS DE \_ TAREFA**](/windows/desktop/api/taskschd/ne-taskschd-task_instances_policy)                     | Define como o Agendador de Tarefas lida com instâncias existentes da tarefa quando ela inicia uma nova instância da tarefa. |
| [**TIPO \_ DE LOGON DE TAREFA**](/windows/desktop/api/taskschd/ne-taskschd-task_logon_type)                                  | Define qual técnica de logon é necessária para executar uma tarefa.                                                          |
| [**TIPO \_ TASK PROCESSTOKENSID \_**](/windows/win32/api/taskschd/ne-taskschd-task_processtokensid_type)             | Define os tipos de SID de processo que podem ser usados por tarefas.                                                      |
| [**SINALIZADORES \_ \_ DE EXECUTAR TAREFA**](/windows/desktop/api/taskschd/ne-taskschd-task_run_flags)                                   | Define como uma tarefa é executado.                                                                                       |
| [**TIPO \_ DE \_ RUNLEVEL DE TAREFA**](/windows/win32/api/taskschd/ne-taskschd-task_runlevel_type)                           | Define sinalizadores de elevação DE LUA que especificam com qual nível de privilégio a tarefa será executado.                         |
| [**TIPO \_ DE ALTERAÇÃO DE ESTADO DA SESSÃO DE \_ \_ \_ TAREFA**](/windows/desktop/api/taskschd/ne-taskschd-task_session_state_change_type) | Define quais tipos de alteração de estado de sessão do Servidor de Terminal você pode usar para disparar uma tarefa a ser iniciada.               |
| [**ESTADO \_ DA TAREFA**](/windows/desktop/api/taskschd/ne-taskschd-task_state)                                            | Define os diferentes estados em que uma tarefa registrada pode estar.                                                   |
| [**TIPO \_ DE GATILHO DE \_ TAREFA2**](/windows/desktop/api/taskschd/ne-taskschd-task_trigger_type2)                                  | Define o tipo de gatilhos que podem ser usados por tarefas.                                                          |
