# <a name="emailactionfrom-property"></a>Propriedade emailaction. from

\[Não há mais suporte para este objeto. Use IExecAction com o cmdlet [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) do PowerShell como uma solução alternativa.\]

Para scripts, Obtém ou define o endereço de email do qual você deseja enviar o email.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```VB
EmailAction.From As String
```



## <a name="property-value"></a>Valor da propriedade

O endereço de email do qual você deseja enviar o email.
