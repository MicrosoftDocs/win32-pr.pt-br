# <a name="actionid-property"></a>Propriedade Action.Id

Para scripts, Obtém ou define o identificador da ação.

## <a name="syntax"></a>Syntax


```VB
Action.Id As String
```



## <a name="property-value"></a>Valor da propriedade

O identificador definido pelo usuário para a ação. Esse identificador é usado pelo Agendador de Tarefas para fins de log.

## <a name="remarks"></a>Comentários

Para obter informações sobre como as ações e tarefas funcionam em conjunto, consulte [ações da tarefa](task-actions.md).
