# <a name="actionid-property"></a>Action.Id propriedade

Para scripts, obtém ou define o identificador da ação.

## <a name="syntax"></a>Syntax


```VB
Action.Id As String
```



## <a name="property-value"></a>Valor da propriedade

O identificador definido pelo usuário para a ação. Esse identificador é usado pelo Agendador de Tarefas para fins de registro em log.

## <a name="remarks"></a>Comentários

Para obter informações sobre como as ações e tarefas funcionam em conjunto, consulte [Ações de tarefa](task-actions.md).
