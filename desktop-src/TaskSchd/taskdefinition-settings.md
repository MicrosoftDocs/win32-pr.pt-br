# <a name="taskdefinitionsettings-property"></a>Propriedade TaskDefinition. Settings

Para scripts, Obtém ou define as configurações que definem como o serviço de Agendador de Tarefas executa a tarefa.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Settings As TaskSettings
```



## <a name="property-value"></a>Valor da propriedade

As configurações que definem como o serviço de Agendador de Tarefas executa a tarefa.
