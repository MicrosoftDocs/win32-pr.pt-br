# <a name="registeredtaskpath-property"></a>Propriedade RegisteredTask. Path

Para scripts, obtém o caminho para onde a tarefa registrada é armazenada.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Path As String
```



## <a name="property-value"></a>Valor da propriedade

O caminho para onde a tarefa registrada é armazenada.
