# <a name="triggercollectioncount-property"></a>Propriedade Triggers. Count

Para scripts, obtém o número de gatilhos na coleção.

## <a name="syntax"></a>Syntax


```VB
TriggerCollection.Count As long
```



## <a name="property-value"></a>Valor da propriedade

O número de gatilhos na coleção.
