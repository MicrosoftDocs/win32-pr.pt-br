# <a name="registeredtaskcollectioncount-property"></a>Propriedade RegisteredTaskCollection. Count

Para scripts, obtém o número de tarefas registradas na coleção.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
RegisteredTaskCollection.Count As long
```



## <a name="property-value"></a>Valor da propriedade

O número de tarefas registradas na coleção.
