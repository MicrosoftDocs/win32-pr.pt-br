# <a name="runningtaskcollectioncount-property"></a>Propriedade RunningTaskCollection. Count

Para scripts, obtém o número de tarefas em execução na coleção.

## <a name="syntax"></a>Sintaxe


```VB
RunningTaskCollection.Count As long
```



## <a name="property-value"></a>Valor da propriedade

O número de tarefas em execução na coleção.
