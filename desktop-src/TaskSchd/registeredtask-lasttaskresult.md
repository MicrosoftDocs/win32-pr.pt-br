# <a name="registeredtasklasttaskresult-property"></a>Propriedade RegisteredTask. LastTaskResult

Para scripts, obtém os resultados retornados na última vez em que a tarefa registrada foi executada.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.LastTaskResult As String
```



## <a name="property-value"></a>Valor da propriedade

Os resultados retornados na última vez em que a tarefa registrada foi executada.
