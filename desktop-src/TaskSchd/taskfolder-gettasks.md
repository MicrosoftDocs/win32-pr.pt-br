# <a name="taskfoldergettasks-property"></a>Propriedade TaskFolder.GetTasks

Para scripts, obtém todas as tarefas na pasta .

## <a name="syntax"></a>Syntax


```VB
TaskFolder.GetTasks( _
  ByVal flags, _
  ByRef ppTasks _
)
```
