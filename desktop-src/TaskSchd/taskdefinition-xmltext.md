# <a name="taskdefinitionxmltext-property"></a>Propriedade de texto de TaskDefinition.Xml

Para scripts, Obtém ou define a definição formatada como XML da tarefa.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.XmlText As String
```



## <a name="property-value"></a>Valor da propriedade

A definição formatada em XML da tarefa.

## <a name="remarks"></a>Comentários

O XML de uma tarefa é definido pelo [esquema de Agendador de tarefas](task-scheduler-schema.md). Para obter um exemplo de XML de tarefa, consulte [exemplo de gatilho diário (XML)](daily-trigger-example--xml-.md).
