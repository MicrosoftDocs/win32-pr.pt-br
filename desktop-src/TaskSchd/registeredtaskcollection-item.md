# <a name="registeredtaskcollectionitem-property"></a>Propriedade RegisteredTaskCollection. Item

Para scripts, obtém a tarefa registrada especificada da coleção.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
RegisteredTaskCollection.Item( _
  ByVal Index _
) As RegisteredTask
```



## <a name="property-value"></a>Valor da propriedade

Um objeto [**RegisteredTask**](registeredtask.md) que contém a tarefa registrada.
