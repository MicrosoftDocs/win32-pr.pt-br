# <a name="actioncollectioncount-property"></a>Propriedade ActionCollection. Count

Para scripts, obtém o número de ações na coleção.

## <a name="syntax"></a>Syntax


```VB
ActionCollection.Count As long
```



## <a name="property-value"></a>Valor da propriedade

O número de ações na coleção. A coleção pode conter até 32 ações.
