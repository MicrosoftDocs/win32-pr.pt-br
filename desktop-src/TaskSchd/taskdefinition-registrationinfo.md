# <a name="taskdefinitionregistrationinfo-property"></a>Propriedade TaskDefinition.RegistrationInfo

Para scripts, obtém ou define as informações de registro usadas para descrever uma tarefa, como a descrição da tarefa, o autor da tarefa e a data em que a tarefa é registrada.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Valor da propriedade

As informações de registro usadas para descrever uma tarefa, como a descrição da tarefa, o autor da tarefa e a data em que a tarefa é registrada.
