# <a name="taskfoldername-property"></a>Propriedade TaskFolder.Name

Para scripts, obtém o nome que é usado para identificar a pasta que contém uma tarefa.

## <a name="syntax"></a>Syntax


```VB
TaskFolder.Name As String
```



## <a name="property-value"></a>Valor da propriedade

O nome usado para identificar a pasta.
