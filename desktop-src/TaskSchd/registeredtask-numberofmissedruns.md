# <a name="registeredtasknumberofmissedruns-property"></a>Propriedade RegisteredTask.NumberOfMissedRuns

Para scripts, obtém o número de vezes que a tarefa registrada perdeu uma executar agendada.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.NumberOfMissedRuns As Integer
```



## <a name="property-value"></a>Valor da propriedade

O número de vezes que a tarefa registrada perdeu uma executar agendada.
