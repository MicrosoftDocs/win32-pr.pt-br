# <a name="networksettingsid-property"></a>Propriedade NetworkSettings.Id

Para scripts, Obtém ou define um valor de GUID que identifica um perfil de rede.

## <a name="syntax"></a>Syntax


```VB
NetworkSettings.Id As String
```



## <a name="property-value"></a>Valor da propriedade

Um valor de GUID que identifica um perfil de rede.
