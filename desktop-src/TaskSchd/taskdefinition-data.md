# <a name="taskdefinitiondata-property"></a>Propriedade TaskDefinition. Data

Para scripts, Obtém ou define os dados associados à tarefa. Esses dados são ignorados pelo serviço de Agendador de Tarefas, mas são usados por terceiros que desejam estender o formato de tarefa.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Valor da propriedade

Os dados associados à tarefa.
