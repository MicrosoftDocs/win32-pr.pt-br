# <a name="tasksettingsxmltext-property"></a>Propriedade de texto de TaskSettings.Xml

Para scripts, Obtém ou define uma definição formatada em XML das configurações de tarefa.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Sintaxe


```VB
TaskSettings.XmlText As String
```



## <a name="property-value"></a>Valor da propriedade

Uma definição formatada em XML das configurações de tarefa.
