# <a name="monthlydowtriggerrunonlastweekofmonth-property"></a>Propriedade MonthlyDOWTrigger. RunOnLastWeekOfMonth

Para scripts, Obtém ou define um valor booliano que indica que a tarefa é executada na última semana do mês.

## <a name="syntax"></a>Syntax


```VB
MonthlyDOWTrigger.RunOnLastWeekOfMonth As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Verdadeiro indica que a tarefa é executada na última semana do mês; caso contrário, false.
