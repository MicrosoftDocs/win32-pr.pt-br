# <a name="repetitionpatternstopatdurationend-property"></a>Propriedade RepetitionPattern. StopAtDurationEnd

Para scripts, Obtém ou define um valor booliano que indica se uma instância em execução da tarefa é interrompida no final da duração do padrão de repetição.

## <a name="syntax"></a>Syntax


```VB
RepetitionPattern.StopAtDurationEnd As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Um valor booliano que indica se uma instância em execução da tarefa é interrompida no final da duração do padrão de repetição.

## <a name="remarks"></a>Comentários

Ao ler ou gravar XML para uma tarefa, essas informações são especificadas no elemento [**StopAtDurationEnd**](taskschedulerschema-stopatdurationend-repetitiontype-element.md) do esquema de Agendador de tarefas.
