# <a name="emailactionheaderfields-property"></a>Propriedade emailaction. HeaderFields

\[Não há mais suporte para este objeto. Use IExecAction com o cmdlet [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) do PowerShell como uma solução alternativa.\]

Para scripts, Obtém ou define as informações de cabeçalho no email que você deseja enviar.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```VB
EmailAction.HeaderFields As TaskNamedValueCollection
```



## <a name="property-value"></a>Valor da propriedade

As informações de cabeçalho no email que você deseja enviar.
