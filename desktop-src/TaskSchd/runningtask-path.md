# <a name="runningtaskpath-property"></a>Propriedade RunningTask. Path

Para scripts, obtém o caminho para onde a tarefa é armazenada.

## <a name="syntax"></a>Syntax


```VB
RunningTask.Path As string
```



## <a name="property-value"></a>Valor da propriedade

O caminho para onde a tarefa é armazenada.
