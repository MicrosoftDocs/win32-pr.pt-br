# <a name="emailactionbcc-property"></a>Propriedade EmailAction.Bcc

\[Não há mais suporte para esse objeto. Use IExecAction com o cmdlet [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) do PowerShell como uma solução alternativa.\]

Para scripts, obtém ou define o endereço de email ou os endereços que você deseja receber na mensagem de email.

Essa propriedade é leitura/gravação.

## <a name="syntax"></a>Syntax


```VB
EmailAction.Bcc As String
```



## <a name="property-value"></a>Valor da propriedade

O endereço de email ou endereços que você deseja receber na mensagem de email.
