# <a name="emailactionbcc-property"></a>Propriedade emailaction. Bcc

\[Não há mais suporte para este objeto. Use IExecAction com o cmdlet [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) do PowerShell como uma solução alternativa.\]

Para scripts, Obtém ou define o endereço de email ou endereços que você deseja CCO na mensagem de email.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```VB
EmailAction.Bcc As String
```



## <a name="property-value"></a>Valor da propriedade

O endereço de email ou os endereços que você deseja CCO na mensagem de email.
