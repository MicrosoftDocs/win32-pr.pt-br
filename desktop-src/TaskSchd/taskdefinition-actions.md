# <a name="taskdefinitionactions-property"></a>Propriedade TaskDefinition. Actions

Para scripts, Obtém ou define uma coleção de ações que são executadas pela tarefa.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Actions As ActionCollection
```



## <a name="property-value"></a>Valor da propriedade

Uma coleção de ações que são executadas pela tarefa.
