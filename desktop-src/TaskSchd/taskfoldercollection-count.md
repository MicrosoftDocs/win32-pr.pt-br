# <a name="taskfoldercollectioncount-property"></a>Propriedade TaskFolderCollection. Count

Para scripts, obtém o número de pastas na coleção.

## <a name="syntax"></a>Syntax


```VB
TaskFolderCollection.Count As long
```



## <a name="property-value"></a>Valor da propriedade

O número de gatilhos na coleção.
