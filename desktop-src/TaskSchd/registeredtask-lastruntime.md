# <a name="registeredtasklastruntime-property"></a>Propriedade RegisteredTask. LastRunTime

Para scripts, obtém a hora em que a tarefa registrada foi executada pela última vez.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.LastRunTime As String
```



## <a name="property-value"></a>Valor da propriedade

A hora em que a tarefa registrada foi executada pela última vez.
