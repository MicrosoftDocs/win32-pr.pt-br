# <a name="taskfoldercollectionitem-property"></a>Propriedade TaskFolderCollection. Item

Para scripts, obtém a pasta especificada da coleção.

## <a name="syntax"></a>Syntax


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## <a name="property-value"></a>Valor da propriedade

Um objeto [**TaskFolder**](taskfolder.md) que representa a pasta solicitada.
