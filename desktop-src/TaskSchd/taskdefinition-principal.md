# <a name="taskdefinitionprincipal-property"></a>Propriedade TaskDefinition. principal

Para scripts, Obtém ou define a entidade para a tarefa que fornece as credenciais de segurança para a tarefa.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Principal As Principal
```



## <a name="property-value"></a>Valor da propriedade

A entidade para a tarefa que fornece as credenciais de segurança para a tarefa.
