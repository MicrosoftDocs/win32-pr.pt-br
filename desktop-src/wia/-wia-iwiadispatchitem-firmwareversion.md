# <a name="itemfirmwareversion-property"></a>Propriedade Item. FirmwareVersion

Recupera a versão do firmware do dispositivo. Essa propriedade só se aplica a itens do tipo dispositivo (itens raiz). Somente leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Item.FirmwareVersion
```



## <a name="property-value"></a>Valor da propriedade

Cadeia de caracteres que recebe a versão do firmware.
