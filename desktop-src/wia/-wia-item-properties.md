# <a name="item-properties"></a>Propriedades do item

As propriedades de item são propriedades que controlam a função de um item de aquisição de imagem do Windows (WIA). Essas propriedades são acessadas por meio de qualquer item em uma árvore que representa um dispositivo. Há três tipos de propriedades de item: comum, scanner e câmera.

Para obter informações de referência para essas propriedades, consulte:

-   [**Constantes de propriedade de item WIA comuns**](-wia-wiaitempropcommonitem.md)
-   [**Constantes da Propriedade Item WIA da câmera**](-wia-wiaitempropcameraitem.md)
-   [**Constantes da propriedade item do scanner WIA**](-wia-wiaitempropscanneritem.md)
