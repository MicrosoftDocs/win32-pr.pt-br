# <a name="item-properties"></a>Propriedades do item

As propriedades do item são propriedades que controlam a função de um Windows WIA (Aquisição de Imagem). Essas propriedades são acessadas por meio de qualquer item em uma árvore que representa um dispositivo. Há três tipos de propriedades de item: comum, scanner e câmera.

Para obter informações de referência para essas propriedades, consulte:

-   [**Constantes comuns de propriedade de item WIA**](-wia-wiaitempropcommonitem.md)
-   [**Constantes de propriedade de item WIA da câmera**](-wia-wiaitempropcameraitem.md)
-   [**Constantes de propriedade de item WIA do scanner**](-wia-wiaitempropscanneritem.md)
