# <a name="wia-event-identifiers"></a>Identificadores de evento WIA

Os identificadores de evento WIA são identificadores globais exclusivos (GUID) definidos pelo sistema de aquisição de imagem do Windows (WIA).

Um aplicativo WIA pode assinar esses eventos não persistentes usando [**IwiaDevMgr2:: RegisterEventCallbackInterface**](-wia-iwiadevmgr2-registereventcallbackinterface.md) para receber notificações de alteração de status do dispositivo do dispositivo WIA ao qual a sessão atual do aplicativo WIA está conectada. O mini-driver WIA deve garantir que \_ \_ o status de manuseio de documento do DPS do WIA \_ \_ seja atualizado corretamente antes que o driver sinalize um desses eventos.

Estes são os GUIDs de evento WIA válidos: 

| GUID                             | Significado                                                                                                                                                                                                                                                                                                                               |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_es de \_ cancelamento de evento WIA \_           | O aplicativo tenta cancelar uma transferência de dados.                                                                                                                                                                                                                                                                                   |
| \_cobertura de evento WIA \_ \_ fechada        | Todas as capas do scanner são fechadas. Se houver várias capas de scanner abertas, o minidriver WIA deverá sinalizar esse evento somente quando a última capa tiver sido fechada. Esse recurso está disponível a partir do Windows 8.                                                                                                                      |
| \_cobertura de evento WIA \_ \_ aberta          | Uma cobertura de scanner foi aberta pelo operador. O dispositivo não pode verificar no momento. O minidriver WIA deve sinalizar esse evento apenas uma vez se várias capas de scanner forem abertas em sucessão. Esse recurso está disponível a partir do Windows 8.                                                                                        |
| \_dispositivo de evento WIA \_ \_ conectado    | Um dispositivo de hardware WIA foi conectado ao computador do usuário ou iniciado.                                                                                                                                                                                                                                                                |
| \_dispositivo de evento WIA \_ \_ desconectado | Um dispositivo de hardware WIA foi desconectado do computador do usuário ou foi interrompido.                                                                                                                                                                                                                                                           |
| o \_ dispositivo de evento WIA \_ \_ não \_ está pronto   | O dispositivo entrou no estado não pronto. O aplicativo deve ler o \_ \_ \_ \_ status de manuseio de documento do DPS do WIA para obter mais informações. Esse recurso está disponível a partir do Windows 8.                                                                                                                                                         |
| \_dispositivo de evento WIA \_ \_ pronto        | O dispositivo entrou no estado pronto. O estado anterior não pronto sinalizado com o \_ dispositivo de evento WIA \_ \_ não \_ está pronto foi limpo. Esse recurso está disponível a partir do Windows 8.                                                                                                                                                             |
| \_alimentador de eventos WIA \_ \_ esvaziado      | O operador removeu todos os documentos da bandeja de entrada do alimentador e o alimentador está vazio. O dispositivo não pode verificar a partir do Feeder no momento. Importante: o minidriver não deve sinalizar esse evento quando o alimentador se tornar vazio como resultado de um trabalho de verificação executado por um aplicativo. Esse recurso está disponível a partir do Windows 8. |
| \_alimentador de eventos WIA \_ \_ carregado       | O operador colocou documentos na bandeja de entrada do alimentador. O alimentador é carregado com papel. O aplicativo pode iniciar um trabalho de verificação do Feeder no momento. Esse recurso está disponível a partir do Windows 8.                                                                                                                     |
| \_tampa de mesa do evento WIA \_ \_ \_ fechada | A tampa da mesa foi fechada pelo operador. O aplicativo pode iniciar um trabalho de verificação a partir do superfície no momento. Esse recurso está disponível a partir do Windows 8.                                                                                                                                                                 |
| \_tampa de mesa do evento WIA \_ \_ \_ aberta   | A tampa de mesa foi aberta pelo operador. O dispositivo não pode examinar a superfície no momento. Esse recurso está disponível a partir do Windows 8.                                                                                                                                                                                  |
| \_item de evento WIA \_ \_ criado        | Um item foi adicionado à árvore de objetos [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) ou [**IWiaItem2**](-wia-iwiaitem2.md) para um dispositivo WIA.                                                                                                                                                                                                   |
| \_item de evento WIA \_ \_ excluído        | Um item foi excluído da árvore de objetos [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) ou [**IWiaItem2**](-wia-iwiaitem2.md) para um dispositivo WIA.                                                                                                                                                                                               |
| retomada de energia do \_ evento WIA \_ \_        | A energia foi restaurada para o dispositivo ou computador.                                                                                                                                                                                                                                                                                    |
| \_suspensão de \_ energia do evento WIA \_       | A energia foi suspensa para o dispositivo ou computador.                                                                                                                                                                                                                                                                                   |
| \_imagem de \_ email de verificação de eventos WIA \_ \_   | Uma verificação foi iniciada com a intenção de enviar por email a imagem resultante.                                                                                                                                                                                                                                                                   |
| \_imagem de \_ fax de verificação de eventos WIA \_ \_     | Uma verificação foi iniciada com a intenção de enviar por fax a imagem resultante.                                                                                                                                                                                                                                                                      |
| \_imagem do \_ filme de verificação de eventos WIA \_ \_    | Uma imagem foi examinada a partir de um adaptador de filme.                                                                                                                                                                                                                                                                                             |
| \_imagem de \_ verificação de evento WIA \_          | Uma imagem foi verificada.                                                                                                                                                                                                                                                                                                                 |
| \_IMAGE2 de \_ verificação de eventos WIA \_         | Uma imagem foi verificada. Esse evento permite que os scanners com mais de um botão de verificação direcionem os botões para diferentes aplicativos.                                                                                                                                                                                                      |
| \_IMAGE3 de \_ verificação de eventos WIA \_         | Uma imagem foi verificada. Esse evento permite que os scanners com mais de um botão de verificação direcionem os botões para diferentes aplicativos.                                                                                                                                                                                                      |
| \_IMAGE4 de \_ verificação de eventos WIA \_         | Uma imagem foi verificada. Esse evento permite que os scanners com mais de um botão de verificação direcionem os botões para diferentes aplicativos.                                                                                                                                                                                                      |
| \_imagem de \_ OCR de verificação de eventos WIA \_ \_     | Uma verificação foi iniciada com a intenção de converter a imagem resultante com reconhecimento óptico de caracteres (OCR).                                                                                                                                                                                                                         |
| \_imagem de \_ impressão de verificação de eventos WIA \_ \_   | Uma verificação foi iniciada com a intenção de imprimir a imagem resultante.                                                                                                                                                                                                                                                                    |
| \_ \_ proxy STI do evento WIA \_           | Destinado apenas para uso interno. Não use.                                                                                                                                                                                                                                                                                           |
| \_armazenamento de eventos WIA \_ \_ criado     | Uma unidade de armazenamento, como um cartão flash, foi adicionada ao dispositivo. O aplicativo deve recriar a árvore de itens que representa esse dispositivo.                                                                                                                                                                                            |
| \_armazenamento de eventos WIA \_ \_ excluído     | Uma unidade de armazenamento, como um cartão flash, foi removida do dispositivo. O aplicativo deve recriar a árvore de itens que representa esse dispositivo.                                                                                                                                                                                        |
| \_árvore de eventos WIA \_ \_ atualizada        | A árvore de itens WIA foi atualizada.                                                                                                                                                                                                                                                                                                        |
| \_volume do evento WIA \_ \_ inserido     | Um dispositivo de armazenamento, como um cartão flash, foi inserido no dispositivo.                                                                                                                                                                                                                                                                 |
