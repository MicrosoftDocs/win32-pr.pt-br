# <a name="deviceinfomanufacturer-property"></a>Propriedade DeviceInfo. manufacturer

Recupera o nome do fabricante deste dispositivo de hardware.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = DeviceInfo.Manufacturer
```



## <a name="property-value"></a>Valor da propriedade

Cadeia de caracteres que recebe o nome.
