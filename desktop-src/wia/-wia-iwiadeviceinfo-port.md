# <a name="deviceinfoport-property"></a>Propriedade DeviceInfo. Port

Recupera a porta à qual o dispositivo de hardware da aquisição de imagem do Windows (WIA) está conectado.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = DeviceInfo.Port
```



## <a name="property-value"></a>Valor da propriedade

Cadeia de caracteres que recebe a porta.
