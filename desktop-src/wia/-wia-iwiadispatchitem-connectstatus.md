# <a name="itemconnectstatus-property"></a>Propriedade Item. ConnectStatus

Recupera o status de conexão do dispositivo. Essa propriedade só se aplica a itens do tipo dispositivo (itens raiz). Os valores possíveis são "Connected", "Disconnected" ou **NULL** (se essa propriedade não se aplicar ao item). Somente leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Item.ConnectStatus
```



## <a name="property-value"></a>Valor da propriedade

Cadeia de caracteres que recebe o status da conexão.
