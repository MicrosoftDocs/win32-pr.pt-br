# <a name="itemtakepicture-method"></a>Método Item.TakePicture

O **método TakePicture** do [**objeto Item**](-wia-item.md) faz com que um dispositivo de câmera digital tire uma foto e retorna um objeto **Item** que representa a imagem resultante. Esse método se aplica somente a dispositivos de câmera digital.

## <a name="syntax"></a>Sintaxe


```JScript
retVal = Item.TakePicture()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **IWiaDispatchItem**

Um [**objeto Item**](-wia-item.md) que representa a imagem que esse método cria.
