# <a name="syntax-conventions"></a>Convenções de sintaxe

A tabela a seguir lista os termos usados em instruções de sintaxe para a referência do modelo de script da aquisição de imagens do Windows (WIA). 

| Termo                      | Descrição                                                    |
|---------------------------|----------------------------------------------------------------|
| *Filename*                | O caminho e o nome de um arquivo.                                   |
| *objDeviceInfo*           | Um objeto [**DeviceInfo**](-wia-deviceinfo.md) .                |
| *objDeviceInfoCollection* | Uma coleção de objetos [**DeviceInfo**](-wia-deviceinfo.md) . |
| *objItem*                 | Um objeto [**Item**](-wia-item.md) .                           |
| *objItemCollection*       | Uma coleção de objetos de [**Item**](-wia-item.md) .             |
| *objWia*                  | Um objeto [**WIA**](-wia-wia.md) .                              |
| *PropID*                  | Uma constante de propriedade WIA.                                       |
| *var*                     | Um valor de **variante** .                                           |
