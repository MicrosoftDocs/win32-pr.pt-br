# <a name="using-wia-video"></a>Usando o vídeo WIA

O WIA fornece a interface [**IWiaVideo**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) para permitir que os aplicativos exibam vídeo de streaming em uma janela, capturem imagens ainda do fluxo de vídeo e salvem essas imagens como arquivos JPEG.

as seções a seguir explicam como usar os recursos de vídeo WIA (aquisição de imagem Windows):

-   [Visualizando vídeo](-wia-previewing-video.md)
-   [Capturando uma imagem ainda de um fluxo de vídeo](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> o WIA não oferece suporte a dispositivos de vídeo no Windows Server 2003, Windows Vista ou posterior. para essas versões do Windows, use [DirectShow](/previous-versions//ms783323(v=vs.85)) para adquirir imagens do vídeo.
