# <a name="itemthumbheight-property"></a>Propriedade Item. ThumbHeight

A altura, em pixels, da imagem em miniatura. Essa propriedade retornará-1 se este item não oferecer suporte a miniaturas. Somente leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Item.ThumbHeight
```



## <a name="property-value"></a>Valor da propriedade

Variável que recebe a altura da miniatura.
