# <a name="itemthumbwidth-property"></a>Propriedade Item. ThumbWidth

A largura, em pixels, da imagem em miniatura. Essa propriedade retornará-1 se este item não oferecer suporte a miniaturas. Somente leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Item.ThumbWidth
```



## <a name="property-value"></a>Valor da propriedade

Variável que recebe a largura.
