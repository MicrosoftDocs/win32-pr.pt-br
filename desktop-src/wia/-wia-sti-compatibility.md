# <a name="sti-compatibility"></a>Compatibilidade COM OLS

A Windows WIA (Aquisição de Imagem) do Windows é criada com base na base estabelecida pela ARQUITETURA de Imagem Ainda (WILL). Os drivers de dispositivo WIA estarão em conformidade com o modelo DE USD (Driver de Modo de Usuário) do MOTION. Embora a finalidade original do USD seja dar suporte a fontes de dados DOIS e outras APIs, os drivers WIA são suportados pela nova interface de aplicativo WIA.
