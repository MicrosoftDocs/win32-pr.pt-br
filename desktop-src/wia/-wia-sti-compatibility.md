# <a name="sti-compatibility"></a>Compatibilidade do STI

A arquitetura da aquisição de imagens do Windows (WIA) baseia-se na Fundação estabelecida pela STI (arquitetura de imagem ainda). Os drivers de dispositivo WIA serão compatíveis com o modelo de USD (driver de modo de usuário) do STI. Embora a finalidade original de USD seja dar suporte a fontes de dados TWAIN e outras APIs, os drivers WIA dão suporte à nova interface de aplicativo WIA.
