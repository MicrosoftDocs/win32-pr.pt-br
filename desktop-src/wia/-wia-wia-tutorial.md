# <a name="wia-tutorial"></a>Tutorial do WIA

Este tutorial fornece instruções passo a passo para usar a WIA (aquisição de imagem do Windows) em aplicativos do mundo real.

O tutorial apresenta o código dos aplicativos de exemplo incluídos no SDK. Ele se destina a demonstrar como os desenvolvedores podem implementar a principal funcionalidade do WIA e não é um plano gráfico para criar um aplicativo completo.

O tutorial se concentra nas seguintes tarefas:

-   [Criando um Gerenciador de Dispositivos WIA](-wia-creating-a-wia-device-manager.md)
-   [Enumerando dispositivos do sistema](-wia-enumerating-system-devices.md)
-   [Lendo propriedades do dispositivo](-wia-reading-device-properties.md)
-   [Criando um dispositivo](-wia-creating-a-device.md)
-   [Enumerando itens](-wia-enumerating-items.md)
-   [Transferindo dados de imagem](-wia-transferring-image-data.md)
-   [Registrar-se em eventos](-wia-registering-for-events.md)
-   [Capturando uma imagem ainda do streaming de vídeo](-wia-capturing-a-still-image-from-streaming-video.md)
