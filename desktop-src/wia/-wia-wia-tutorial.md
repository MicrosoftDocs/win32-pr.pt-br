# <a name="wia-tutorial"></a>WIA Tutorial

Este tutorial fornece instruções passo a passo para usar Windows WIA (Aquisição de Imagem) em aplicativos do mundo real.

O tutorial apresenta o código dos aplicativos de exemplo incluídos no SDK. Ele se destina a demonstrar como os desenvolvedores podem implementar a funcionalidade principal do WIA e não é um blueprint para criar um aplicativo completo.

O tutorial se concentra nas seguintes tarefas:

-   [Criando um wi-Gerenciador de Dispositivos](-wia-creating-a-wia-device-manager.md)
-   [Enumerando dispositivos do sistema](-wia-enumerating-system-devices.md)
-   [Lendo propriedades do dispositivo](-wia-reading-device-properties.md)
-   [Criando um dispositivo](-wia-creating-a-device.md)
-   [Enumerando itens](-wia-enumerating-items.md)
-   [Transferindo dados de imagem](-wia-transferring-image-data.md)
-   [Registrando para eventos](-wia-registering-for-events.md)
-   [Capturando uma imagem ainda do vídeo de streaming](-wia-capturing-a-still-image-from-streaming-video.md)
