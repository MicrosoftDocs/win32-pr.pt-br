# <a name="itempictureheight-property"></a>Propriedade Item. PictureHeight

A altura, em pixels, das imagens produzidas por essa câmera digital. Aplica-se somente a câmeras digitais. Somente leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Item.PictureHeight
```



## <a name="property-value"></a>Valor da propriedade

Variável que recebe a altura.
