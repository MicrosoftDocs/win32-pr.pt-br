# <a name="deviceinfotype-property"></a>Propriedade DeviceInfo. Type

recupera o tipo de dispositivo de hardware de aquisição de imagem de Windows (WIA). Os valores possíveis são:

-   DigitalCamera
-   Scanner
-   StreamingVideo
-   Padrão

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = DeviceInfo.Type
```



## <a name="property-value"></a>Valor da propriedade

Cadeia de caracteres que recebe o dispositivo.
