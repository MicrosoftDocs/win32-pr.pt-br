# <a name="wia-property-constants"></a>Constantes da propriedade WIA

Windows as constantes da propriedade de aquisição de imagem (WIA) incluem os seguintes grupos de constantes com suporte pelo Windows Millennium Edition (Windows Me), Windows XP e posterior.

-   [**Constantes da propriedade de informações do dispositivo**](-wia-wiadeviceinfoprop.md)
-   [**Constantes de propriedade de dispositivo comum**](-wia-wiaitempropcommondevice.md)
-   [**Constantes de propriedade de dispositivo do scanner**](-wia-wiaitempropscannerdevice.md)
-   [**Constantes de propriedade de item WIA comuns**](-wia-wiaitempropcommonitem.md)
-   [**Constantes da propriedade item do scanner WIA**](-wia-wiaitempropscanneritem.md)

os seguintes grupos de constantes têm suporte do Windows Me, Windows XP e Windows Server 2003; mas o Windows Vista e versões posteriores não têm suporte.

-   [**Constantes de Propriedade do dispositivo Camera**](-wia-wiaitempropcameradevice.md)
-   [**Constantes da Propriedade Item WIA da câmera**](-wia-wiaitempropcameraitem.md)

o seguinte grupo de constantes é suportado pelo Windows Me e Windows XP; mas não tem suporte do Windows Server 2003 e posterior.

-   [**Constantes de propriedade de dispositivo WIA de vídeo**](-wia-wiaitempropvideodevice.md)
