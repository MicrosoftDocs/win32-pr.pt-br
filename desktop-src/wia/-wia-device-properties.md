# <a name="device-properties-wia"></a>Propriedades do dispositivo (WIA)

As propriedades do dispositivo são propriedades que controlam a função do dispositivo inteiro. Essas propriedades só podem ser acessadas por meio do item raiz da árvore que representa um dispositivo. Há três tipos de propriedades de dispositivo: comum, scanner e câmera.

Para obter informações de referência, consulte:

-   [**Constantes de propriedade de dispositivo comum**](-wia-wiaitempropcommondevice.md)
-   [**Constantes de Propriedade do dispositivo Camera**](-wia-wiaitempropcameradevice.md)
-   [**Constantes de propriedade de dispositivo do scanner**](-wia-wiaitempropscannerdevice.md)
