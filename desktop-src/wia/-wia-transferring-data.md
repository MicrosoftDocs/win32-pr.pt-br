# <a name="transferring-data"></a>Transferindo dados

> [!Note]  
> Este sistema de scripts foi substituído pela camada de automação da WIA (aquisição de imagem do Windows). Consulte [camada de automação de aquisição de imagem do Windows](/previous-versions/windows/desktop/wiaaut/-wiaaut-startpage).

 

Use o método de [**transferência**](-wia-iwiadispatchitem-transfer.md) de um objeto [**Item**](-wia-item.md) para transferir dados de imagem ou de áudio de um dispositivo para um arquivo ou para a área de transferência.

Passe um caminho e um nome de arquivo como o primeiro parâmetro para salvar os dados no disco ou passe a cadeia de caracteres "Clipboard" para transferir o item para a área de transferência.
