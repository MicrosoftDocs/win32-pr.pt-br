# <a name="transferring-data"></a>Transferindo dados

> [!Note]  
> Esse sistema de scripts foi substituído pela camada de automação wia (aquisição de imagem) Windows imagem. Consulte [Windows camada de automação de aquisição de imagem.](/previous-versions/windows/desktop/wiaaut/-wiaaut-startpage)

 

Use o [**método Transfer**](-wia-iwiadispatchitem-transfer.md) de um [**objeto Item**](-wia-item.md) para transferir dados de imagem ou áudio de um dispositivo para um arquivo ou para a área de transferência.

Passe um caminho e um nome de arquivo como o primeiro parâmetro para salvar os dados no disco ou passe a cadeia de caracteres "área de transferência" para transferir o item para a área de transferência.
