# <a name="additional-wia-properties"></a>Propriedades de WIA adicionais

Os dispositivos WIA podem dar suporte a propriedades de item adicionais, dependendo da configuração de hardware. As propriedades sem suporte têm um VARTYPE de VT \_ vazio.
