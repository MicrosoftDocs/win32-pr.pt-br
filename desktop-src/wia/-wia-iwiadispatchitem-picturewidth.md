# <a name="itempicturewidth-property"></a>Propriedade Item. PictureWidth

A largura, em pixels, das imagens produzidas por essa câmera digital. Aplica-se somente a câmeras digitais. Somente leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Item.PictureWidth
```



## <a name="property-value"></a>Valor da propriedade

Variável que recebe a largura.
