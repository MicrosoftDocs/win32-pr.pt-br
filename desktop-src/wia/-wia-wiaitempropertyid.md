# <a name="wiaitempropertyid"></a>WiaItemPropertyId

A maioria das constantes da propriedade de aquisição de imagem do Windows (WIA) é agrupada em um tipo de dados enumerado, WiaItemPropertyId para autores de script.

A tabela a seguir apresenta o mapeamento entre as convenções de nomenclatura usadas em script e C++. Por exemplo, no script, o prefixo "CameraDevice" mapeia para o prefixo "DPC de WIA \_ " para a constante de C++ correspondente. Em um exemplo mais específico, a propriedade "CameraDeviceFlashMode" é equivalente à constante "modo de \_ flash DPC WIA \_ \_ " C++. Consulte os tópicos constantes de propriedade correspondente para obter descrições de cada propriedade. 

| Prefixo do script  | Tópico do C++ (prefixo)                                                                    |
|----------------|---------------------------------------------------------------------------------------|
| Dispositivo         | [**Constantes de propriedade de dispositivo comum ( \_ DPA WIA)**](-wia-wiaitempropcommondevice.md)   |
| CameraDevice   | [**Constantes de Propriedade do dispositivo Camera (DPC do WIA \_ )**](-wia-wiaitempropcameradevice.md)   |
| ScannerDevice  | [**Constantes de propriedade de dispositivo do scanner (DPS do WIA \_ )**](-wia-wiaitempropscannerdevice.md) |
| VideoDevice    | [**Constantes de propriedade de dispositivo WIA de vídeo (WIA \_ DPV)**](-wia-wiaitempropvideodevice.md) |
| Picture        | [**Constantes de propriedade de item WIA comuns (WIA \_ IPA)**](-wia-wiaitempropcommonitem.md)   |
| CameraPicture  | [**Constantes da Propriedade Item WIA da câmera ( \_ IPC WIA)**](-wia-wiaitempropcameraitem.md)   |
| ScannerPicture | [**Constantes de propriedade de item do scanner WIA ( \_ IPS WIA)**](-wia-wiaitempropscanneritem.md) |
