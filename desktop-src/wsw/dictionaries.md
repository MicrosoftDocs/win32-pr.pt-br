# <a name="dictionaries"></a>Dicionários

Os dicionários permitem a geração de tokens de dados de nó comuns, como nomes de elementos/atributos, namespaces e texto comum em um formato XML binário.


Um [**>de dicionário**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_dictionary) pode ser especificado como parte do tipo de [**cadeia de \_ \_ caracteres XML do WS**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_string) . Todas as cadeias de caracteres em um único **dicionário>** devem ser exclusivas.

As IDs do dicionário têm suporte para todas as codificações (não apenas o formato XML binário).

A função a seguir é usada com dicionários:

-   [**WsGetDictionary**](/windows/desktop/api/WebServices/nf-webservices-wsgetdictionary)
