# <a name="encoding-windows-web-services"></a>codificação (Windows Web Services)

A codificação é uma transformação entre XML e um formato de conexão linear. A decodificação é a transformação inversa.


Uma codificação é identificada usando uma constante de [**\_ codificação WS**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) .
