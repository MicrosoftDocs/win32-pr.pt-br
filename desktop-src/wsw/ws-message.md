# <a name="ws_message"></a>\_mensagem WS

O tipo opaco usado para fazer referência a um objeto de [mensagem](message.md) .


```C++
typedef struct _WS_MESSAGE WS_MESSAGE;
```



## <a name="remarks"></a>Comentários

Este objeto não é thread-safe. Para obter mais informações, consulte [segurança do thread](thread-safety.md).
