# <a name="prefetch-debug-apis-for-windows-store-apps"></a>APIs de depuração de pré-busca para aplicativos da Windows Store

As APIs a seguir são usadas para depurar e testar o comportamento de pré-busca de conteúdo em um aplicativo da Windows Store que implementa a classe [**ContentPrefetcher**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher) .



| Interface                                                              | Descrição                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IContentPrefetcherTaskTrigger**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | Use essa interface para verificar se um pacote do aplicativo da Windows Store instalado está registrado para essa tarefa em segundo plano e disparar manualmente suas operações de pré-busca de conteúdo.<br/> |
