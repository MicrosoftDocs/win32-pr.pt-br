# <a name="windows-web-services-api"></a>API de serviços Web do Windows

## <a name="purpose"></a>Finalidade

WWSAPI é uma implementação de código nativo do SOAP que fornece a funcionalidade de comunicação de rede principal ao dar suporte a um conjunto de WS- \* and. NET- \* Family of Protocols. O WWSAPI foi projetado para ser usado por componentes/aplicativos que se enquadram em uma das seguintes categorias:

-   Mandato de código nativo
-   Exigir dependências mínimas
-   Exigir tempo de inicialização mínimo
-   Ambientes com restrição de memória

## <a name="developer-audience"></a>Público de desenvolvedores

A API de serviços Web do Windows (WWSAPI) oferece a API (interface de programação de aplicativo) C/C++ para a criação de serviços Web e clientes baseados em SOAP. Uma compreensão básica dos serviços Web e do protocolo associado a eles é importante para usar essa API.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

A API de serviços Web do Windows (WWSAPI) é um componente de sistema operacional do Windows 7 e do Windows Server 2008 R2 ou versões posteriores do Microsoft Windows.
