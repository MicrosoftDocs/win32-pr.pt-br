<dl> <dt>

<span id="_security_x.200_gly"></span><span id="_SECURITY_X.200_GLY"></span>**X. 200**
</dt> <dd>

Um padrão de interconexão de sistemas abertos (OSI).

</dd> <dt>

<span id="_security_x.208_gly"></span><span id="_SECURITY_X.208_GLY"></span>**X. 208**
</dt> <dd>

O padrão ITU que define o ASN. 1.

</dd> <dt>

<span id="_security_x.209_gly"></span><span id="_SECURITY_X.209_GLY"></span>**X. 209**
</dt> <dd>

O padrão ITU que define as codificações do BER para objetos ASN. 1.

Consulte também [*regras básicas de codificação*](b-gly.md).

</dd> <dt>

<span id="_security_x.400_gly"></span><span id="_SECURITY_X.400_GLY"></span>**X. 400**
</dt> <dd>

O padrão ITU que define os tokens de mensagem.

</dd> <dt>

<span id="_security_x.500_gly"></span><span id="_SECURITY_X.500_GLY"></span>**X. 500**
</dt> <dd>

O padrão ITU que define a estrutura de um dicionário global.

</dd> <dt>

<span id="_security_x.509_gly"></span><span id="_SECURITY_X.509_GLY"></span>**X. 509**
</dt> <dd>

Um padrão reconhecido internacionalmente para certificados que define suas partes necessárias.

</dd> <dt>

<span id="_security_x509_asn_encoding_gly"></span><span id="_SECURITY_X509_ASN_ENCODING_GLY"></span>**\_Codificação ASN \_ X509**
</dt> <dd>

Tipo de codificação que especifica a codificação de certificado. Os tipos de codificação de certificado são armazenados na palavra de ordem inferior de um **DWORD** (o valor é: 0x00000001). Esse tipo de codificação é funcionalmente o mesmo que \_ a \_ codificação ASN cript.

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**XOR**
</dt> <dd>

Exclusivo **ou**. Uma operação booliana que retorna true somente se um de seus operandos for true e o outro for false. Se ambos os operandos forem os mesmos (true ou false), a operação resultará em false.

</dd> </dl>
