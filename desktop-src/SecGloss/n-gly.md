<dl> <dt>

<span id="_security_national_institute_of_standards_and_technology_gly"></span><span id="_SECURITY_NATIONAL_INSTITUTE_OF_STANDARDS_AND_TECHNOLOGY_GLY"></span>**Instituto Nacional de padrões e tecnologia**
</dt> <dd>

NIST Uma divisão do departamento de comércio de Estados Unidos que publica padrões oficiais para sistemas de computador do setor privado e do governo. Esses padrões são publicados como publicações FIPS (Federal Information Processing Standards). Em 1987, o NIST foi direcionado para definir padrões para garantir a segurança de informações confidenciais, mas não classificadas, em sistemas de computador do governo.

</dd> <dt>

<span id="_security_negotiate_gly"></span><span id="_SECURITY_NEGOTIATE_GLY"></span>**Inicia**
</dt> <dd>

Um SSP ( [*provedor de suporte de segurança*](s-gly.md) ) que atua como uma camada de aplicativo entre a interface do provedor de suporte de [*segurança*](s-gly.md) (SSPI) e os outros SSPs. Quando um aplicativo chama o SSPI para fazer logon em uma rede, ele pode especificar um SSP para processar a solicitação. Se o aplicativo especificar Negotiate, Negotiate analisará a solicitação e escolherá o melhor SSP para lidar com a solicitação com base na política de segurança configurada pelo cliente.

</dd> <dt>

<span id="_security_nist_gly"></span><span id="_SECURITY_NIST_GLY"></span>**NIST**
</dt> <dd>

Consulte *National Institute of Standards and Technology*.

</dd> <dt>

<span id="_security_nonce_gly"></span><span id="_SECURITY_NONCE_GLY"></span>**momentos**
</dt> <dd>

Um valor gerado aleatoriamente usado para anular ataques de "reprodução".

</dd> <dt>

<span id="_security_nonrepudiation_gly"></span><span id="_SECURITY_NONREPUDIATION_GLY"></span>**Não-repúdio**
</dt> <dd>

A capacidade de identificar os usuários que executaram determinadas ações, portanto, irrefutably o contador de quaisquer tentativas feitas por um usuário para negar a responsabilidade. Por exemplo, um sistema pode registrar em log a ID de um usuário sempre que um arquivo for excluído.

</dd> </dl>
