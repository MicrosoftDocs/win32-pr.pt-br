# <a name="parental-controls"></a>Controles dos pais

## <a name="purpose"></a>Finalidade

Como parte dos esforços de segurança da Microsoft, o Windows fornece a funcionalidade de controles dos pais para monitorar e limitar a exposição de usuários de computadores selecionados a perigos e conteúdo inadequados online.

A tecnologia de controles dos pais no Windows destina-se a auxiliar os pais ou tutores na garantia de acesso aos materiais apropriados por idade ou nível de maturidade para aqueles sob sua guardião. Ele fornece uma infraestrutura extensível, além de recursos internos.

## <a name="developer-audience"></a>Público de desenvolvedores

ISVs (fornecedores independentes de software) de produtos que potencialmente expõem os usuários a riscos de segurança ou que estão trabalhando atualmente para proteger os usuários desses riscos são incentivados a integrar os controles dos pais.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Essa funcionalidade foi introduzida no Windows Vista. A funcionalidade de controles dos pais só está disponível em sistemas operacionais de consumidor selecionados; Ele não foi projetado para uso com versões do Windows Server. Não há suporte para essa funcionalidade no Windows 10.
