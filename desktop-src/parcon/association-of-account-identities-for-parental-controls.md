# <a name="association-of-account-identities-for-parental-controls"></a>Associação de identidades de conta para controles dos pais

Conforme discutido na Introdução, o uso efetivo da funcionalidade de configuração, registro em log e restrições fornecida pela infraestrutura de controles dos pais do Windows Vista depende Windows contas que estão sendo configuradas como usuários padrão. É recomendável que os fornecedores que fornecem acesso online (como os suites isp) criem e mantenham uma associação de nomes de usuário online permitidos pelo pai ou guardião para cada conta Windows controlada. Dessa forma, ignorar controles parentais baseados em nuvem simplesmente usando identidades online alternativas é reduzido.
