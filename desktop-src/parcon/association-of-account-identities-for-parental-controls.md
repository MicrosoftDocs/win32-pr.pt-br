# <a name="association-of-account-identities-for-parental-controls"></a>Associação de identidades de conta para controles dos pais

Conforme discutido na introdução, o uso efetivo da funcionalidade de configuração, registro em log e restrições fornecida pela infraestrutura de controles dos pais do Windows Vista depende das contas do Windows que estão sendo configuradas como usuários padrão. É recomendável que os fornecedores que fornecem acesso online (como pacotes de ISP) criem e mantenham uma associação de nomes de usuário online permitidos pelo pai ou guardião para cada conta do Windows controlada. Dessa forma, ignorar os controles dos pais baseados em nuvem simplesmente usando identidades online alternativas é reduzido.
