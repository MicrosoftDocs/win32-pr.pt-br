# <a name="parental-controls-top-level-user-interface"></a>Controles dos pais Top-Level interface do usuário

O forte link entre a segurança da família e as contas de usuário do Windows tem impulsionado uma categoria combinada que aparece como **contas de usuário e segurança de família** no painel de controle para SKUs orientadas ao consumidor do Windows Vista.

> [!Note]  
> A categoria será exibida como **contas de usuário** quando um computador com recurso de ingresso no domínio for ingressado.

 

Se uma conta de usuário padrão ainda não tiver sido configurada para o usuário controlado, um link no painel de controle fornecerá um fluxo de trabalho simples para adicionar uma nova conta controlada de forma responsável.

Depois que uma conta é criada, o painel de controle dos controles pai expõe a funcionalidade voltada para o usuário de nível superior para o usuário controlado. Elementos

-   As restrições gerais de controles dos pais ativam ou desligam os botões de opção.
-   Relatórios de atividade independentes ativados ou desativados botões de opção de controle.
-   Uma seção de configurações com o link de restrições da Web implementada pela Microsoft é mostrada condicionalmente e os principais tipos de restrições implementadas pela Microsoft disponíveis são mostrados.
-   Uma área mais configurações que aparece se as extensões de interface do usuário são registradas pela Microsoft ou por terceiros.
-   Uma área de status de resumo mostrando o nome de usuário e o bloco, o link exibir relatórios de atividade e o estado das configurações de controles dos pais. Se ativado, a exibição incluirá o nível de configuração do filtro de conteúdo da Web (se o filtro da Microsoft estiver ativo) ou o nome do filtro se for substituído por uma extensão de terceiros e o estado das configurações offline do usuário.

O botão de opção Habilitar controles pai geral é definido inicialmente como off. Quando ativado por um administrador, os relatórios de atividades também estarão ativados por padrão, mas poderão ser desativados independentemente. Assim como acontece com quase todas as configurações nos controles dos pais, a configuração pode ser executada programaticamente por meio das APIs disponíveis.
