# <a name="parental-controls-activity-reporting"></a>Relatórios de atividade de controles dos pais

O relatório de atividades é considerado uma questão essencial nos problemas de engenharia social. Aplicativos ou filtros com reconhecimento são solicitados a registrar atividades conforme especificado na seção usando APIs de controles pais.

Clicar no link relatórios de atividade abre uma página do Visualizador de log mostrando todas as entradas de log de um intervalo máximo de uma semana (limitado por limites de tamanho de log), com árvore de categoria e painéis de dados de formato de coluna.

A caixa Resumo no canto superior direito do painel controles pais indica o status atual do relatório de atividades.

Os eventos de atividade também são visíveis no Visualizador de Eventos com um nome de Microsoft.com/Windows/ParentalControls nos logs do aplicativo.

Essa funcionalidade depende da nova funcionalidade de log de eventos no Windows Vista, usando as APIs de publicação fornecidas.
