# <a name="about-parental-controls"></a>Sobre os controles dos pais

O recurso controles dos pais inclui o seguinte:

-   Funcionalidade da interface central do usuário integrada com o painel de controle.
-   Implementações de restrições selecionadas.
-   Uma infraestrutura no sistema operacional que fornece configurações para políticas gerais e restrições implementadas e que expõe os recursos de extensibilidade direcionados.
