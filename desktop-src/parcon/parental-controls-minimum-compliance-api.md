# <a name="parental-controls-minimum-compliance-api"></a>Controle dos pais API de conformidade mínima

A API de conformidade mínima implementa métodos de interface simples para que os aplicativos obtenham informações básicas de estado em uma determinada conta de usuário controlada de forma responsável, suas configurações de restrições da Web e configurações de restrições de jogos.

-   [**IWindowsParentalControls**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**IWPCGamesSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**IWPCSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**IWPCWebSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
