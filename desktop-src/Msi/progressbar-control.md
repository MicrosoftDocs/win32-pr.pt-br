# <a name="progressbar-control"></a>Controle ProgressBar

O controle ProgressBar exibe um gráfico de barras que muda de comprimento enquanto recebe mensagens de progresso. Esse controle assina o ControlEvent, de [toprogress](setprogress-controlevent.md) . Ele pode assinar um ControlEvent, chamado após a ação monitorada.

Para obter informações relacionadas, consulte [criando um controle ProgressBar](authoring-a-progressbar-control.md)e [adicionando ações personalizadas ao ProgressBar](adding-custom-actions-to-the-progressbar.md).

## <a name="control-attributes"></a>Atributos de controle

Você pode usar os atributos a seguir com este controle. Para alterar o valor de um atributo usando um evento, assine o controle para um ControlEvent, na [tabela EventMappings](eventmapping-table.md) e liste o identificador do atributo na coluna atributo. Insira o identificador do ControlEvent, na coluna de evento.
