# <a name="msidisableembeddedui"></a>MsiDisableEmbeddedUI

Se essa [política do sistema](system-policy.md) por máquina for definida como 1, a capacidade de usar a interface do usuário inserida será desabilitada no computador. O valor padrão é 0, o que habilita o recurso de manipuladores de interface do usuário inserido.

Para desabilitar a interface do usuário inserida para um único pacote, você pode usar a propriedade [**MSIDISABLEEEUI**](msidisableeeui.md) .

**[Windows Installer 4,0 e anteriores](not-supported-in-windows-installer-4-0.md):** Sem suporte. Essa função está disponível a partir do Windows Installer 4,5.

## <a name="registry-key"></a>Chave do Registro

**HKEY \_ \_** Políticas de \\ **software** do computador local \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo de Dados

**REG \_ DWORD**
