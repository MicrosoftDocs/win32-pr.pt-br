# <a name="sysmodal-dialog-style-bit"></a>Bit de estilo de caixa de diálogo SysModal

Se esse bit de estilo for definido, a caixa de diálogo irá parar todos os outros aplicativos e nenhum outro aplicativo poderá assumir o foco. Esse estado permanece até que a caixa de diálogo SysModal seja ignorada.

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                          |
|---------|-------------|-----------------------------------|
| 8       | 0x00000008  | **msidbDialogAttributesSysModal** |
