# <a name="determining-whether-a-column-is-a-primary-or-external-key"></a>Determinando se uma coluna é uma chave primária ou externa

Você pode determinar se uma coluna específica é uma chave primária ou uma chave externa da tabela consultando a página de referência da tabela. Uma lista de links para as páginas de referência de todas as tabelas de banco de dados de instalação está localizada em Tabelas [de Banco de Dados](database-tables.md). Cada uma das páginas de referência identifica colunas que são uma chave primária ou uma chave externa.

As colunas de banco de dados que são chaves externas levam o nome da coluna de chave primária com um caractere de sublinhado adicionado. Por exemplo, uma chave externa para a coluna Arquivo da tabela Arquivo é sempre nomeada Arquivo \_ .

Para exibir diagramas de relação de entidade que ilustram relações de tabela, consulte Grupo de Tabelas Principais [,](core-tables-group.md)Grupo de Tabelas de Arquivos [,](file-tables-group.md) [Grupo](registry-tables-group.md)de Tabelas do Registro [,](system-tables-group.md)Grupo de Tabelas do Sistema ou [Esquema Interface do Usuário](user-interface-schema.md).
