# <a name="determining-whether-a-column-is-a-primary-or-external-key"></a>Determinando se uma coluna é uma chave primária ou externa

Você pode determinar se uma coluna específica é uma chave primária ou uma chave externa da tabela consultando a página de referência da tabela. Uma lista de links para as páginas de referência de todas as tabelas de banco de dados de instalação está localizada em [tabelas de banco de dados](database-tables.md). Cada uma das páginas de referência identifica colunas que são uma chave primária ou uma chave externa.

As colunas de banco de dados que são chaves externas usam o nome da coluna de chave primária com um caractere de sublinhado adicionado. Por exemplo, uma chave externa para a coluna arquivo da tabela de arquivos é sempre denominada arquivo \_ .

Para exibir diagramas de relação de entidade que ilustram relações de tabela, consulte [grupo de tabelas principais](core-tables-group.md), grupo tabelas de [arquivos](file-tables-group.md), [grupo tabelas de registro](registry-tables-group.md), [grupo tabelas do sistema](system-tables-group.md)ou esquema de interface do [usuário](user-interface-schema.md).
