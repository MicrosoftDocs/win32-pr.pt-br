# <a name="machine-policies"></a>Políticas de computador

As políticas de computador a seguir podem ser configuradas em:

**HKEY \_ Instalador \_ do** Microsoft machine \\ **software** \\ **policies** \\  \\ **microsoft Windows** \\ 



| Política                                                                                    | Tipo de dados de valor           | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [AlwaysInstallElevated](alwaysinstallelevated.md)<br/>                             | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1 e o valor do usuário correspondente também estiver definido, o instalador sempre será instalado com privilégios elevados.<br/> Caso contrário, o instalador usa privilégios elevados para instalar aplicativos gerenciados e usa o nível de privilégio do usuário atual para aplicativos não gerenciados.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [AllowLockdownBrowse](allowlockdownbrowse.md)<br/>                                 | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os usuários não administrativos poderão procurar novas fontes durante a execução de uma instalação com privilégios elevados. O padrão é que somente os administradores possam procurar fontes durante uma instalação com privilégios elevados. Definir essa política também permite que usuários não administrativos executem programas em privilégios localSystem durante uma instalação com privilégios elevados.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [AllowLockdownMedia](allowlockdownmedia.md)<br/>                                   | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os usuários não administrativos poderão usar fontes de mídia, como um CD-ROM, durante a execução de uma instalação com privilégios elevados. O padrão é que somente os administradores podem usar fontes de mídia durante uma instalação com privilégios elevados. Definir essa política também permite que usuários não administrativos executem programas em privilégios localSystem durante uma instalação com privilégios elevados.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [AllowLockdownPatch](allowlockdownpatch.md)<br/>                                   | **REG \_ DWORD**<br/> | Se esse valor de política de sistema por computador não for definido, somente os administradores poderão corrigir os produtos existentes que foram instalados com privilégios elevados. Se esse valor de política for definido como 1, os usuários não administrativos poderão, em alguns casos, aplicar patches aos produtos durante a execução de uma instalação usando privilégios elevados. Com o conjunto de políticas, o patch pode instalar [atualizações secundárias](minor-upgrades.md) durante a execução de uma instalação usando privilégios elevados; o patch não pode instalar [atualizações principais.](major-upgrades.md) Definir essa política também permite que usuários não administrativos executem programas em privilégios localSystem durante uma instalação com privilégios elevados.<br/>                                                                                                                                                                                                                                                                                      |
| [Depurar](debug.md)<br/>                                                             | **REG \_ DWORD**<br/> | Se esse valor de política existir e for definido como 1, o instalador grava mensagens comuns de depuração no depurador usando a [**função OutputDebugString.**](/windows/win32/api/debugapi/nf-debugapi-outputdebugstringa) Se esse valor existir e for definido como 2, o instalador grava todas as mensagens de depuração válidas no depurador usando a **função OutputDebugString.**<br/> Essa política é apenas para fins de depuração e pode não ter suporte em versões futuras do Windows Installer.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [DisableAutomaticApplicationShutdown](disableautomaticapplicationshutdown.md)<br/> | **REG \_ DWORD**<br/> | Se esse valor de política existir e estiver definido como 1, Windows Instalador não interagirá com o [Gerenciador](../rstmgr/restart-manager-portal.md) de Reinicialização, mas usará a [funcionalidade FilesInUse Dialog.](filesinuse-dialog.md)<br/> **[Windows Instalador 3.1 e anterior:](not-supported-in-windows-installer-version-3-1.md)** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [DisableBrowse](disablebrowse.md)<br/>                                             | **REG \_ DWORD**<br/> | Se esse valor de política existir e estiver definido como 1, os usuários serão impedidos de navegar para localizar as fontes do instalador. A **caixa de combinação Usar recurso** para entrada direta está bloqueada e o **botão** Procurar está desabilitado. Para obter mais informações sobre a navegação de origem, consulte [Resiliência de origem.](source-resiliency.md)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [DisableFlyWeightPatching](disableflyweightpatching.md)<br/>                       | **REG \_ DWORD**<br/> | Se esse valor de política de sistema por computador for definido como 1, todas as opções de Otimização [de](patch-optimization.md) Patch serão desligadas durante a instalação.<br/> **Windows Instalador 2.0:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [DisableLUAPatching](disableluapatching.md)<br/>                                   | **REG \_ DWORD**<br/> | Se esse valor de política de sistema por computador for definido como 1, o instalador impedirá que não administradores usem a aplicação de patch de CONTA com privilégios mínimos (LUA) em qualquer aplicativo instalado no computador. Quando esse valor não estiver definido ou 0, os não administradores poderão aplicar patches LUA ao aplicativo habilitado para LUA.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [DisableMSI](disablemsi.md)<br/>                                                   | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 0, estiver ausente ou qualquer número diferente de 1 ou 2, o efeito no instalador do Windows dependerá do sistema operacional. No Windows Server 2003, o Windows Instalador está habilitado para aplicativos gerenciados e desabilitado para instalação de aplicativos não gerenciados. No Windows XP, o Windows Instalador está habilitado para todos os aplicativos.<br/> Se esse valor de política for definido como 0, Windows Instalador será habilitado para todos os aplicativos. Todas as operações de instalação são permitidas.<br/> Se esse valor de política for definido como 1, Windows Instalador será desabilitado para aplicativos não gerenciados, mas ainda está habilitado para aplicativos gerenciados. Instalações não elevadas por usuário são bloqueadas. As instalação por computador e elevadas por usuário são permitidas.<br/> Se esse valor de política for definido como 2, Windows Instalador estará sempre desabilitado para todos os aplicativos. Nenhuma instalação é permitida, incluindo reparos, reinstalações ou instalações sob demanda.<br/> |
| [DisablePatch](disablepatch.md)<br/>                                               | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, o instalador não aplicará patches. Essa política pode ser usada para fornecer segurança em ambientes em que a adoção de patch deve ser restrita.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [DisablePatchUninstall](disablepatchuninstall.md)<br/>                             | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os patches não poderão ser removidos do computador por um usuário ou um administrador. O Windows instalador de dados ainda pode remover patches que não são mais aplicáveis a um produto.<br/> **Windows Instalador 2.0:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [DisableRollback](disablerollback.md)<br/>                                         | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, o instalador não armazenará arquivos de retorção durante a instalação, desabilitando a reação da instalação. Por padrão, a reação está habilitada. Os administradores são aconselhados a não usar essa política, a menos que seja absolutamente essencial.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [DisableSharedComponent](disablesharedcomponent.md)<br/>                           | **REG \_ DWORD**<br/> | Se essa política [](system-policy.md) de sistema por computador for definida como 1, nenhum pacote no sistema obtém a funcionalidade de componente compartilhado habilitada pelo atributo **msidbComponentAttributesShared** na tabela [Componente](component-table.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [DisableUserInstalls](disableuserinstalls.md)<br/>                                 | **REG \_ DWORD**<br/> | Se esse valor de política não estiver definido, o instalador pesquisa produtos no Registro na seguinte ordem: produtos gerenciados registrados como por usuário, produtos não gerenciados registrados como por usuário e, por fim, produtos registrados como por computador.<br/> Se esse valor de política for definido como 1, o instalador ignorará todos os produtos registrados como por usuário e procurará apenas produtos registrados como por computador. Uma tentativa de executar uma instalação por usuário faz com que o instalador exibe uma mensagem de erro e interrompe a instalação.<br/>                                                                                                                                                                                                                                                                                                                                                              |
| [EnforceUpgradeComponentRules](enforceupgradecomponentrules.md)<br/>               | **REG \_ DWORD**<br/> | De definir esse valor de política como [](small-updates.md) 1 [](minor-upgrades.md) para aplicar regras de componente de atualização durante pequenas atualizações e pequenas atualizações de todos os produtos no computador.<br/> **Windows Instalador 2.0:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [EnableAdminTSRemote](enableadmintsremote.md)<br/>                                 | **REG \_ DWORD**<br/> | Definir essa política permite que os administradores executem instalações de uma sessão de cliente de um servidor que executa o serviço de função servidor terminal.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [EnableUserControl](enableusercontrol.md)<br/>                                     | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, o instalador poderá passar todas as propriedades públicas para o lado do servidor durante uma instalação gerenciada usando privilégios elevados. [](public-properties.md) Definir essa política tem o mesmo efeito que definir a [**propriedade EnableUserControl.**](-enableusercontrol.md)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [LimitSystemRestoreCheckpointing](limitsystemrestorecheckpointing.md)<br/>         | **REG \_ DWORD**<br/> | Essa política desliga a criação de pontos de verificação Windows Instalador.<br/> Se o valor da política for definido como 0 ou ausente, o Windows Instalador faz o ponto de verificação normal para instalar ou desinstalar.<br/> Se o valor da política for definido como 1, Windows Instalador não criará nenhum ponto de verificação.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [Logging](logging.md)<br/>                                                         | **REG \_ SZ**<br/>    | Esse valor de política será usado somente se o registro em log não tiver sido habilitado pela opção de linha de comando "/L" ou [**MsiEnableLog**](/windows/desktop/api/Msi/nf-msi-msienableloga). Se uma política for definida nesse caso, um arquivo de log será criado no diretório temporário com o nome aleatório: MSI \* . Log. Especifique o modo de registro em log definindo o valor da política como uma cadeia de caracteres. Use os mesmos caracteres para especificar a política de modo de registro em log, conforme usado pela opção de linha de comando "/L". Para obter mais informações, consulte [Opções de linha de comando](command-line-options.md). Observe que você não pode usar "+" e \* "" para a política.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [MaxPatchCacheSize](maxpatchcachesize.md)<br/>                                     | **REG \_ DWORD**<br/> | Se esse valor de política for definido como um valor maior que 0, Windows Instalador salvará versões antigas de arquivos com patch em um cache. De definir o valor como o percentual máximo de espaço em disco que pode ser usado para o cache de arquivos. Por exemplo, um valor de 15 e define o máximo como 15%. De definido como 0 para não salvar nenhum arquivo. Quando essa política não está definida, o padrão é 10%.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [MsiDisableEmbeddedUI](msidisableembeddedui.md)<br/>                               | **REG \_ DWORD**<br/> | Para desabilitar manipuladores de interface do usuário inseridos no computador, de definir esse valor de política como 1.<br/> **[Windows Instalador 4.0 e anterior:](not-supported-in-windows-installer-4-0.md)** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [SafeForScripting](safeforscripting.md)<br/>                                       | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os usuários não serão solicitados quando os scripts usarem a automação do instalador em uma página da Web. Isso pode ser útil para ferramentas baseadas na Web, mas pode permitir instalações silenciosas de aplicativos sem conhecimento ou consentimento do usuário.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [TransformsSeure policy](transformssecure-policy.md)<br/>                         | **REG \_ DWORD**<br/> | Definir o valor da política [TransformsSecure](transformssecure-policy.md) como 1 informa ao instalador que as transformaçãos devem ser armazenadas em cache localmente no computador do usuário em um local em que o usuário não tenha acesso de gravação.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [DisableLoggingFromPackage](disableloggingfrompackage.md)<br/>                     | **REG \_ DWORD**<br/> | De definir esse valor de política como 1 para desabilitar o registro em log especificado para o pacote pela [**propriedade MsiLogging**](msilogging.md) para todos os usuários do computador.<br/> **Windows Instalador 3.1 e anterior:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [WinHttpAutoLogonLevel](winhttpautologonlevel.md)<br/>                             | **REG \_ SZ**<br/>    | A política de logon automático (logon automático) determina quando é aceitável incluir as credenciais padrão em uma solicitação ao servidor. **Windows 8 e Windows Server 2012:** Essa política requer Windows instalador em execução no Windows 8 ou Windows Server 2012 e não está disponível em todas as versões anteriores do Windows.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
