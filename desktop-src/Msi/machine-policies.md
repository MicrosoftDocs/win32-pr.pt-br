# <a name="machine-policies"></a>Políticas do computador

As seguintes políticas de computador podem ser configuradas em:

**HKEY \_ \_** Políticas de \\ **software** do computador local \\  \\ **Microsoft** \\ **Windows** \\ **Installer**



| Política                                                                                    | Tipo de dados de valor           | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [AlwaysInstallElevated](alwaysinstallelevated.md)<br/>                             | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1 e o valor de usuário correspondente também for definido, o instalador sempre será instalado com privilégios elevados.<br/> Caso contrário, o instalador usa privilégios elevados para instalar aplicativos gerenciados e usa o nível de privilégio do usuário atual para aplicativos não gerenciados.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [AllowLockdownBrowse](allowlockdownbrowse.md)<br/>                                 | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os usuários não administrativos poderão procurar novas fontes durante a execução de uma instalação em privilégios elevados. O padrão é que somente os administradores possam procurar fontes durante uma instalação elevada. A definição dessa política também permite que usuários não administrativos executem programas em privilégios LocalSystem durante uma instalação elevada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [AllowLockdownMedia](allowlockdownmedia.md)<br/>                                   | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os usuários não administrativos poderão usar fontes de mídia, como um CD-ROM, enquanto executam uma instalação em privilégios elevados. O padrão é que somente os administradores podem usar as fontes de mídia durante uma instalação elevada. A definição dessa política também permite que usuários não administrativos executem programas em privilégios LocalSystem durante uma instalação elevada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [AllowLockdownPatch](allowlockdownpatch.md)<br/>                                   | **REG \_ DWORD**<br/> | Se esse valor de política do sistema por máquina não estiver definido, somente os administradores poderão corrigir os produtos existentes que foram instalados com privilégios elevados. Se esse valor de política for definido como 1, os usuários não administrativos poderão, em alguns casos, aplicar patches aos produtos ao executarem uma instalação usando privilégios elevados. Com a política definida, o patch pode instalar [pequenas atualizações](minor-upgrades.md) ao executar uma instalação usando privilégios elevados; o patch não pode instalar as [principais atualizações](major-upgrades.md). A definição dessa política também permite que usuários não administrativos executem programas em privilégios LocalSystem durante uma instalação elevada.<br/>                                                                                                                                                                                                                                                                                      |
| [Depurar](debug.md)<br/>                                                             | **REG \_ DWORD**<br/> | Se esse valor de política existir e for definido como 1, o instalador gravará mensagens de depuração comuns no depurador usando a função [**OutputDebugString**](/windows/win32/api/debugapi/nf-debugapi-outputdebugstringa) . Se esse valor existir e for definido como 2, o instalador gravará todas as mensagens de depuração válidas no depurador usando a função **OutputDebugString** .<br/> Essa política destina-se apenas para fins de depuração e pode não ter suporte em versões futuras do Windows Installer.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [DisableAutomaticApplicationShutdown](disableautomaticapplicationshutdown.md)<br/> | **REG \_ DWORD**<br/> | Se esse valor de política existir e estiver definido como 1, Windows Installer não interagirá com o [Gerenciador de reinicialização](../rstmgr/restart-manager-portal.md) , mas usará a funcionalidade de [caixa de diálogo FilesInUse](filesinuse-dialog.md) .<br/> **[Windows Installer 3,1 e anteriores](not-supported-in-windows-installer-version-3-1.md):** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [DisableBrowse](disablebrowse.md)<br/>                                             | **REG \_ DWORD**<br/> | Se esse valor de política existir e estiver definido como 1, os usuários serão impedidos de procurar para localizar fontes de instalador. O **recurso usar da** caixa de combinação para entrada direta está bloqueado e o botão **procurar** está desabilitado. Para obter mais informações sobre a navegação de origem, consulte [resiliência de origem](source-resiliency.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [DisableFlyWeightPatching](disableflyweightpatching.md)<br/>                       | **REG \_ DWORD**<br/> | Se esse valor de política do sistema por máquina for definido como 1, todas as opções de [otimização de patch](patch-optimization.md) serão desativadas durante a instalação.<br/> **Windows Installer 2,0:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [DisableLUAPatching](disableluapatching.md)<br/>                                   | **REG \_ DWORD**<br/> | Se esse valor de política do sistema por máquina for definido como 1, o instalador impedirá que não administradores usem a aplicação de patches de LUA (conta com privilégios mínimos) para qualquer aplicativo instalado no computador. Quando esse valor não é definido ou 0, não administradores podem aplicar patches de LUA ao aplicativo habilitado para LUA.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [DisableMSI](disablemsi.md)<br/>                                                   | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 0, estiver ausente ou qualquer número diferente de 1 ou 2, o efeito na Windows Installer dependerá do sistema operacional. No Windows Server 2003, o Windows Installer é habilitado para aplicativos gerenciados e desabilitado para instalações de aplicativos não gerenciados. No Windows XP, a Windows Installer está habilitada para todos os aplicativos.<br/> Se esse valor de política for definido como 0, Windows Installer será habilitado para todos os aplicativos. Todas as operações de instalação são permitidas.<br/> Se esse valor de política for definido como 1, o Windows Installer será desabilitado para aplicativos não gerenciados, mas ainda estará habilitado para aplicativos gerenciados. As instalações não elevadas por usuário são bloqueadas. Instalações elevadas por usuário e por máquina são permitidas.<br/> Se esse valor de política for definido como 2, Windows Installer será sempre desabilitada para todos os aplicativos. Nenhuma instalação é permitida, incluindo reparos, reinstalação ou instalações sob demanda.<br/> |
| [DisablePatch](disablepatch.md)<br/>                                               | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, o instalador não aplicará patches. Essa política pode ser usada para fornecer segurança em ambientes em que a aplicação de patch deve ser restrita.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [DisablePatchUninstall](disablepatchuninstall.md)<br/>                             | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os patches não poderão ser removidos do computador por um usuário ou um administrador. O Windows Installer ainda pode remover patches que não são mais aplicáveis a um produto.<br/> **Windows Installer 2,0:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [DisableRollback](disablerollback.md)<br/>                                         | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, o instalador não armazenará arquivos de reversão durante a instalação, desabilitando a reversão da instalação. Por padrão, a reversão está habilitada. Os administradores são aconselhados a não usar essa política, a menos que seja absolutamente essencial.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [DisableSharedComponent](disablesharedcomponent.md)<br/>                           | **REG \_ DWORD**<br/> | Se essa [política do sistema](system-policy.md) por máquina estiver definida como 1, nenhum pacote no sistema obterá a funcionalidade de componente compartilhado habilitada pelo atributo **msidbComponentAttributesShared** na [tabela de componentes](component-table.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [DisableUserInstalls](disableuserinstalls.md)<br/>                                 | **REG \_ DWORD**<br/> | Se esse valor de política não for definido, o instalador pesquisará o registro em busca de produtos na seguinte ordem: produtos gerenciados que são registrados como produtos por usuário e não gerenciados que são registrados como por usuário e, por fim, produtos que são registrados como por máquina.<br/> Se esse valor de política for definido como 1, o instalador ignorará todos os produtos registrados como por usuário e procurará apenas por produtos que estejam registrados como por máquina. Uma tentativa de executar uma instalação por usuário faz com que o instalador exiba uma mensagem de erro e pare a instalação.<br/>                                                                                                                                                                                                                                                                                                                                                              |
| [EnforceUpgradeComponentRules](enforceupgradecomponentrules.md)<br/>               | **REG \_ DWORD**<br/> | Defina esse valor de política como 1 para aplicar as regras de componente de atualização durante [pequenas atualizações](small-updates.md) e pequenas [atualizações](minor-upgrades.md) de todos os produtos no computador.<br/> **Windows Installer 2,0:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [EnableAdminTSRemote](enableadmintsremote.md)<br/>                                 | **REG \_ DWORD**<br/> | Definir essa política permite que os administradores executem instalações de uma sessão de cliente de um servidor que executa o serviço de função Terminal Server.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [EnableUserControl](enableusercontrol.md)<br/>                                     | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, o instalador poderá passar todas as [Propriedades públicas](public-properties.md) para o lado do servidor durante uma instalação gerenciada usando privilégios elevados. Definir essa política tem o mesmo efeito que definir a propriedade [**EnableUserControl**](-enableusercontrol.md) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [LimitSystemRestoreCheckpointing](limitsystemrestorecheckpointing.md)<br/>         | **REG \_ DWORD**<br/> | Essa política desativa a criação de pontos de verificação por Windows Installer.<br/> Se o valor da política for definido como 0 ou ausente, Windows Installer o ponto de verificação normal para instalação ou desinstalação.<br/> Se o valor da política for definido como 1, Windows Installer não criará nenhum ponto de verificação.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [Logging](logging.md)<br/>                                                         | **REG \_ sz**<br/>    | Esse valor de política será usado somente se o registro em log não tiver sido habilitado pela opção de linha de comando "/L" ou [**MsiEnableLog**](/windows/desktop/api/Msi/nf-msi-msienableloga). Se uma política for definida nesse caso, um arquivo de log será criado no diretório Temp com o nome aleatório: MSI \* . Façam. Especifique o modo de log definindo o valor da política como uma cadeia de caracteres. Use os mesmos caracteres para especificar a política do modo de log, conforme usado pela opção de linha de comando "/L". Para obter mais informações, consulte [Opções de linha de comando](command-line-options.md). Observe que você não pode usar "+" e " \* " para a política.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [MaxPatchCacheSize](maxpatchcachesize.md)<br/>                                     | **REG \_ DWORD**<br/> | Se esse valor de política for definido como um valor maior que 0, Windows Installer salvar versões antigas de arquivos com patch em um cache. Defina o valor para a porcentagem máxima de espaço em disco que pode ser usada para o cache de arquivos. Por exemplo, um valor de 15 e define o máximo como 15%. Defina como 0 para não salvar nenhum arquivo. Quando essa política não é definida, o padrão é 10%.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [MsiDisableEmbeddedUI](msidisableembeddedui.md)<br/>                               | **REG \_ DWORD**<br/> | Para desabilitar os manipuladores de interface do usuário inseridos no computador, defina esse valor de política como 1.<br/> **[Windows Installer 4,0 e anteriores](not-supported-in-windows-installer-4-0.md):** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [SafeForScripting](safeforscripting.md)<br/>                                       | **REG \_ DWORD**<br/> | Se esse valor de política for definido como 1, os usuários não serão solicitados quando os scripts usarem a automação do instalador em uma página da Web. Isso pode ser útil para ferramentas baseadas na Web, mas pode permitir instalações silenciosas de aplicativos sem conhecimento ou consentimento do usuário.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [Política de TransformsSecure](transformssecure-policy.md)<br/>                         | **REG \_ DWORD**<br/> | Definir o valor da [política TransformsSecure](transformssecure-policy.md) como 1 informa ao instalador que as transformações devem ser armazenadas em cache localmente no computador do usuário em um local em que o usuário não tem acesso de gravação.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [DisableLoggingFromPackage](disableloggingfrompackage.md)<br/>                     | **REG \_ DWORD**<br/> | Defina esse valor de política como 1 para desabilitar o log especificado para o pacote pela propriedade [**MsiLogging**](msilogging.md) para todos os usuários do computador.<br/> **Windows Installer 3,1 e anteriores:** Sem suporte.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [WinHttpAutoLogonLevel](winhttpautologonlevel.md)<br/>                             | **REG \_ sz**<br/>    | A política de logon automático (logon automático) determina quando é aceitável incluir as credenciais padrão em uma solicitação ao servidor. **Windows 8 e Windows Server 2012:** Essa política requer Windows Installer em execução no Windows 8 ou no Windows Server 2012 e não está disponível em todas as versões anteriores do Windows.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
