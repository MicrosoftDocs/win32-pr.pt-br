# <a name="ice93"></a>ICE93

ICE93 emitirá um aviso se uma ação personalizada usar o mesmo nome que uma ação padrão. Para obter uma lista de todos os nomes de ação padrão, consulte [referência de ações padrão](standard-actions-reference.md).

## <a name="result"></a>Resultado

ICE93 posta o aviso a seguir.



| Aviso de ICE93                                                      | Descrição                                                                                                                    |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| A ação personalizada ' \[ 1 \] ' usa o mesmo nome que uma ação padrão. | Há um identificador na coluna ação da tabela CustomAction que é o nome de uma Windows Installer ação padrão. |
