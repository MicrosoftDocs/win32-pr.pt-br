# <a name="billboardname-control-attribute"></a>Atributo de controle do muralname

Esse atributo retorna o nome do mural atualmente em execução ou define e exibe uma mensagem por nome. Esse atributo é usado no modo de visualização.

## <a name="valid-controls"></a>Controles válidos

[Mensagem](billboard-control.md)

## <a name="associated-bit-flags"></a>Sinalizadores de bits associados

Este atributo não usa sinalizadores de bit.

## <a name="remarks"></a>Comentários

Consulte [atributos de controle](control-attributes.md) e o controle que você precisa criar sob [controles](controls.md).
