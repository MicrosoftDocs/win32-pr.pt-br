# <a name="billboardname-control-attribute"></a>Atributo de controleName

Esse atributo retorna o nome do sets do atualmente em execução ou define e exibe um visor por nome. Esse atributo é usado no modo de visualização.

## <a name="valid-controls"></a>Controles válidos

[Outdoor](billboard-control.md)

## <a name="associated-bit-flags"></a>Sinalizadores de bit associados

Esse atributo não usa sinalizadores de bit.

## <a name="remarks"></a>Comentários

Consulte [Atributos de controle](control-attributes.md) e o controle que você precisa criar em [Controles](controls.md).
