# <a name="codepage-summary-property"></a>Propriedade de resumo da página de código

A propriedade de **Resumo CodePage** é o valor numérico da página de código ANSI usada para qualquer cadeia de caracteres armazenada nas informações de resumo. Observe que essa não é a mesma página de código para cadeias de caracteres no banco de dados de instalação. A propriedade de **Resumo CodePage** é usada para converter as cadeias de caracteres nas informações resumidas em Unicode ao chamar as funções da API Unicode. A propriedade de **Resumo CodePage** deve ser definida antes de qualquer propriedade de cadeia de caracteres ser definida nas informações de resumo.
