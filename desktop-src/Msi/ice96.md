# <a name="ice96"></a>ICE96

ICE96 verifica se a [ação PublishFeatures](publishfeatures-action.md) e a [ação PublishProduct](publishproduct-action.md) são inseridas na [tabela AdvtExecuteSequence](advtexecutesequence-table.md). Um pacote não pode ser anunciado sem essas ações.

## <a name="result"></a>Resultado

ICE96 posta os seguintes avisos.



| Aviso de ICE96                                                            | Descrição                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| A ação PublishFeatures é necessária na tabela AdvtExecuteSequence. | Não é possível anunciar um pacote sem a ação PublishFeatures incluída na tabela AdvtExecuteSequence. |
| A ação PublishProduct é necessária na tabela AdvtExecuteSequence.  | Não é possível anunciar um pacote sem a ação PublishProduct incluída na tabela AdvtExecuteSequence.  |
