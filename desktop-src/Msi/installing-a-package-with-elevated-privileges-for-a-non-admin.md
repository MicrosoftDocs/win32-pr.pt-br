    <span data-ttu-id="dabb0-106">**Windows Vista:** Necessário.</span><span class="sxs-lookup"><span data-stu-id="dabb0-106">**Windows Vista:** Required.</span></span>

    <span data-ttu-id="dabb0-121">Se o aplicativo estiver instalado por computador enquanto a política [AlwaysInstallElevated](alwaysinstallelevated.md) estiver definida, o produto será tratado como gerenciado.</span><span class="sxs-lookup"><span data-stu-id="dabb0-121">If the application is installed per-machine while the [AlwaysInstallElevated](alwaysinstallelevated.md) policy is set, the product is treated as managed.</span></span> <span data-ttu-id="dabb0-122">Nesse caso, o aplicativo ainda poderá executar um reparo com privilégios elevados se a política for removida.</span><span class="sxs-lookup"><span data-stu-id="dabb0-122">In this case, the application can still perform a repair with elevated privileges if the policy is removed.</span></span> <span data-ttu-id="dabb0-123">Além disso, se o aplicativo for instalado por usuário enquanto a política AlwaysInstallElevated estiver definida, o aplicativo não poderá executar um reparo se a política for removida.</span><span class="sxs-lookup"><span data-stu-id="dabb0-123">Also, if the application is installed per-user while the AlwaysInstallElevated policy is set, the application is unable to perform a repair if the policy is removed.</span></span>
