# <a name="properties-windows-installer"></a>Propriedades (Windows Installer)

As propriedades são variáveis globais que Windows Installer usa durante uma instalação. As seções a seguir descrevem as propriedades usadas pelo instalador:

-   [Sobre propriedades](about-properties.md)
-   [Usando propriedades](using-properties.md)
-   [Referência de propriedade](property-reference.md)

O termo "Propriedade" também se refere a um atributo de um objeto de automação. Consulte [interface de automação](automation-interface.md).
