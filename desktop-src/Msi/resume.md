# <a name="resume-property"></a>Retomar Propriedade

A propriedade **resume** é definida ao retomar uma instalação suspensa. Essa propriedade pode determinar qual texto deve ser exibido na interface do usuário. Por exemplo, quando não estiver no modo de retomada, o usuário poderá ser solicitado "você está pronto para instalar?" no modo de retomada, o usuário pode ser perguntado "você está pronto para concluir a instalação?".

## <a name="remarks"></a>Comentários

Para compatibilidade com versões anteriores, o instalador também dá suporte à **retomada** de nome para essa propriedade.
