# <a name="enableusercontrol-property"></a>Propriedade EnableUserControl

Se a propriedade **EnableUserControl** for definida como 1, o instalador poderá passar todas as [Propriedades públicas](public-properties.md) para o lado do servidor durante uma instalação gerenciada usando privilégios elevados. Definir essa propriedade tem o mesmo efeito que definir a [](enableusercontrol.md) [política do sistema](system-policy.md)EnableUserControl.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versão<br/> | Windows instalador 5,0 em Windows Server 2012, Windows 8, Windows Server 2008 R2 ou Windows 7. Windows Installer 4,0 ou Windows Installer 4,5 no Windows Server 2008 ou Windows Vista. Windows instalador no Windows Server 2003, Windows XP e Windows 2000. consulte os [requisitos de Run-Time Windows Installer](windows-installer-portal.md) para obter informações sobre o service pack mínimo Windows exigido por uma versão Windows Installer.<br/> |
