# <a name="executable-files"></a>Arquivos executáveis

Uma ação personalizada pode iniciar um arquivo executável armazenado no computador do usuário ou contido no pacote de instalação.

Os seguintes tipos de ação personalizada iniciam um arquivo executável.



| Tipo de ação personalizada                                 | Descrição                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [Tipo de ação personalizada 2](custom-action-type-2.md)   | Arquivo EXE armazenado em um fluxo de tabela binária.           |
| [Tipo de ação personalizada 18](custom-action-type-18.md) | Arquivo EXE instalado com um produto.                  |
| [Tipo de ação personalizada 50](custom-action-type-50.md) | Arquivo EXE com um caminho especificado por um valor de propriedade. |
| [Tipo de ação personalizada 34](custom-action-type-34.md) | Arquivo EXE com um caminho fazendo referência a um diretório.       |



 

Consulte a [lista Resumo da seção de todos os tipos de ação personalizada](summary-list-of-all-custom-action-types.md) para obter um resumo de todos os tipos de ações personalizadas e como eles são codificados na [tabela CustomAction](customaction-table.md).
