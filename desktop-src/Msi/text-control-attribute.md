# <a name="text-control-attribute"></a>Atributo de controle de texto

Esse atributo é a cadeia de texto exibida no controle. Em configuração, se o campo 0 do registro não for nulo, o registro será formatado usando FormatText. Se o campo 0 for nulo, o primeiro campo do registro definirá o texto. Ao obter o valor é sempre retornado no primeiro campo. Para alguns controles, esse texto pode não estar visível. No Windows, a tecla aceleradora para um controle é definida colocando um "&" na frente do caractere desejado nessa cadeia de caracteres.

## <a name="valid-controls"></a>Controles válidos

Todos os controles, exceto o [controle linha](line-control.md).

## <a name="associated-bit-flags"></a>Sinalizadores de bits associados

Nenhum.
