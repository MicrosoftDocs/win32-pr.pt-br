# <a name="regpath"></a>RegPath

O tipo de dados RegPath é uma cadeia de caracteres de texto que contém um caminho de registro. Os caminhos do registro podem incluir propriedades, como com o tipo de dados [formatado](formatted.md) . Um RegPath não pode começar ou terminar com uma barra invertida ( \\ ). A chave de \[ \# arquivo \] e a \[ chave de $Component \] podem ser inseridas no caminho ou precedido por outros caracteres.
