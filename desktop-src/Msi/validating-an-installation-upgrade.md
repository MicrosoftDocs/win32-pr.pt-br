# <a name="validating-an-installation-upgrade"></a>Validando uma atualização de instalação

Sempre que fizer alterações no pacote, os autores de pacotes de atualização sempre devem executar a validação em seus pacotes antes de tentar instalar o pacote pela primeira vez e executar novamente a validação. Execute a validação em um pacote de atualização da mesma maneira que para um pacote de instalação. Para obter detalhes, consulte [Validando um banco de dados de instalação](validating-an-installation-database.md).

Isso conclui o pacote de atualização de exemplo. Para instalar a atualização, primeiro instale MNP2000.msi e, em seguida, instale o MNP2001.msi. Quando você desinstala o MNP2001, os produtos original e de atualização são removidos do seu computador.
