# <a name="validating-an-installation-upgrade"></a>Validando uma atualização de instalação

Sempre que houver alterações no pacote, os autores de pacotes de atualização sempre deverão executar a validação em seus pacotes antes de tentar instalar o pacote pela primeira vez e executar a validação de novo. Execute a validação em um pacote de atualização da mesma maneira que para um pacote de instalação. Para obter detalhes, consulte [Validando um banco de dados de instalação](validating-an-installation-database.md).

Isso conclui o pacote de atualização de exemplo. Para instalar a atualização, primeiro instale o MNP2000.msi e, em seguida, instale MNP2001.msi. Quando você desinstala o MNP2001, os produtos originais e de atualização são removidos do computador.
