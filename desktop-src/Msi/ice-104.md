# <a name="ice-104"></a>ICE 104

A ICE 104 verifica se apenas uma das duas tabelas: [MsiLockPermissionsEx](msilockpermissionsex-table.md) e [LockPermissions](lockpermissions-table.md) está presente no pacote.

A ICE 104 verifica a sintaxe usada nos campos lockobject, Table e SDDLText na tabela [MsiLockPermissionsEx](msilockpermissionsex-table.md) . A ICE 104 não verifica se o valor no campo SDDLText na tabela MsiLockPermissionsEx é um descritor de segurança válido.

**[Windows Installer 4,5 ou anterior](not-supported-in-windows-installer-4-5.md):** Sem suporte. Esta ICE está disponível a partir do Windows Installer 5,0.

## <a name="result"></a>Resultado

O ICE 104 posta os erros a seguir.



| Erro de ICE104                                                                                                                            | Descrição                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| As tabelas LockPermissions e MsiLockPermissionsEx existem no banco de dados. Apenas uma dessas tabelas deve estar presente.                      | Um pacote não pode conter as tabelas [MsiLockPermissionsEx](msilockpermissionsex-table.md) e [LockPermissions](lockpermissions-table.md) .                                       |
| Não foi possível encontrar o item ' \[ 2 \] ' na tabela ' \[ 3 \] ', que é referenciada na entrada da tabela MsiLockPermissionsEx (MsiLockPermissionsEx =) \[ 1\] | Windows Installer não consegue localizar e proteger um item especificado na tabela [MsiLockPermissionsEx](msilockpermissionsex-table.md) .                                                  |
| Na entrada da tabela MsiLockPermissionsEx (MsiLockPermissionsEx =) \[ 1 \] , SDDLText =% s parece ser um valor de FormattedSDDLText inválido  | O valor no campo SDDLText da tabela [MsiLockPermissionsEx](msilockpermissionsex-table.md) deve ser um tipo de dados [**FormattedSDDLText**](formattedsddltext.md) válido. |
