Esta seção destina-se a desenvolvedores que estão escrevendo seus próprios programas de instalação e que querem saber mais sobre as tabelas de banco de dados Windows Installer.

Para obter informações sobre automação e acesso a bibliotecas C++, consulte [Sobre a Interface de Automação](about-the-automation-interface.md).

Para obter informações sobre como usar a automação e criar [**o objeto do instalador**](installer-object.md), consulte [Usando a Interface de Automação](using-the-automation-interface.md).

Para o material de referência para objetos do instalador, consulte Referência [da Interface de Automação](automation-interface-reference.md).

Para obter informações sobre o WMI, consulte [Using Windows Installer with WMI](using-windows-installer-with-wmi.md).
