# <a name="32-bit-windows-installer-packages"></a>pacotes de Windows Installer de 32 bits

um pacote de 32 bits consiste em apenas [componentes de Windows Installer](windows-installer-components.md) de 32 bits e deve ter o valor "Intel" inserido no campo plataforma da propriedade [**resumo do modelo**](template-summary.md) .

para obter mais informações, consulte [Windows Installer em sistemas operacionais de 64 bits](windows-installer-on-64-bit-operating-systems.md) e [pacotes de Windows Installer de 64 bits](64-bit-windows-installer-packages.md).
