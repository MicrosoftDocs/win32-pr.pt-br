# <a name="admintoolsfolder-property"></a>Propriedade AdminToolsFolder

A propriedade **AdminToolsFolder** contém o caminho completo para o diretório do sistema de arquivos que armazena as ferramentas administrativas. O MMC (console de gerenciamento Microsoft) salva consoles personalizados nesse diretório e faz roaming com o usuário. Se a propriedade [**AllUsers**](allusers.md) for definida, essa propriedade apontará para o diretório do sistema de arquivos que contém as ferramentas administrativas para todos os usuários do computador.
