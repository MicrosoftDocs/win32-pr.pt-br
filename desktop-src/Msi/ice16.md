# <a name="ice16"></a>ICE16

ICE16 valida que o valor da propriedade [**ProductName**](productname.md) na [tabela de propriedades](property-table.md) não tem mais de 63 caracteres de comprimento. Não existe limite no comprimento da chave do registro para DisplayName.

## <a name="result"></a>Resultado

ICE16 posta uma mensagem de erro se o conjunto de NomeDoProduto na tabela de propriedades tiver mais de 63 caracteres.
