# <a name="merge-module-ice-reference"></a>Referência de ICE do módulo de mesclagem

A lista a seguir fornece links para cada ICE do módulo de mesclagem.



| ICEM                 | Breve descrição do ICE do módulo de mesclagem                                                                                                                                                                                                                    |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ICEM01](icem01.md) | Valida se o mecanismo ICE está funcionando.                                                                                                                                                                                                             |
| [ICEM02](icem02.md) | Valida que todas as exclusões de módulo e dependências se relacionam com o módulo atual.                                                                                                                                                                      |
| [ICEM03](icem03.md) | Valida que todas as ações no módulo são ações básicas ou derivam de uma ação base.                                                                                                                                                           |
| [ICEM04](icem04.md) | Verifica se as tabelas vazias necessárias do módulo de mesclagem estão realmente vazias.                                                                                                                                                                                 |
| [ICEM05](icem05.md) | Verifica se há associações inválidas com componentes.                                                                                                                                                                                                         |
| [ICEM06](icem06.md) | Verifica se há referências inválidas a recursos pelo módulo.                                                                                                                                                                                                 |
| [ICEM07](icem07.md) | Valida que a ordem dos arquivos nas tabelas de sequência e na correspondência do arquivo [MergeModule.CABinet](mergemodule-cabinet.md) .                                                                                                                               |
| [ICEM08](icem08.md) | Garante que um módulo não exclua algo de que depende.                                                                                                                                                                                          |
| [ICEM09](icem09.md) | Verifica se o módulo de mesclagem manipula com segurança os diretórios predefinidos.                                                                                                                                                                                    |
| [ICEM10](icem10.md) | Verifica se um módulo de mesclagem não contém propriedades não permitidas na [tabela de propriedades](property-table.md).                                                                                                                                         |
| [ICEM11](icem11.md) | Verifica se um módulo de mesclagem configurável lista a [Tabela ModuleConfiguration](moduleconfiguration-table.md) e a [tabela ModuleSubstitution](modulesubstitution-table.md) na [tabela ModuleIgnoreTable](moduleignoretable-table.md) do módulo. |
| [ICEM12](icem12.md) | Verifica isso em uma tabela ModuleSequence, as ações padrão têm números de sequência e as ações personalizadas têm os valores Baseaction e After.                                                                                                                     |
| [ICEM13](icem13.md) | Verifica se a política do Publicador e os assemblies de configuração não estão incluídos no módulo de mesclagem.                                                                                                                                                        |
| [ICEM14](icem14.md) | Valida a coluna Value da [tabela ModuleSubstitution](modulesubstitution-table.md).                                                                                                                                                              |
