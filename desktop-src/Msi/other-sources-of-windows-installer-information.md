## <a name="orca-documentation"></a>Documentação do Orca

O [Orca](orca-exe.md) é um editor de arquivo GUI. msi fornecido com o SDK do Windows Installer. Ele fornece acesso completo às tabelas de banco de dados Windows Installer.

Um conjunto de arquivos de ajuda é fornecido com o Orca. Para obter a documentação orca.exe e orca, você deve baixar e instalar o SDK do Windows Installer, que é fornecido como um arquivo Orca.msi. Depois de instalar o [SDK (Software Development Kit) do Microsoft Windows](https://developer.microsoft.com/windows/downloads/windows-10-sdk/), clique duas vezes no arquivo Orca.msi para instalar o Orca.

> [!Note]  
> Não é possível baixar o Orca separadamente.

 

Embora a orca forneça acesso a todos os recursos do Windows Installer, ela não se destina a substituir um ambiente de criação de pacote totalmente em destaque. Em muitos casos, é mais fácil criar uma instalação de Windows Installer para um aplicativo usando uma das ferramentas de criação de pacote comercial disponíveis de fornecedores de software independentes.

## <a name="windows-installer-software-vendors"></a>Windows Installer fornecedores de software

Você pode entrar em contato com um fornecedor independente de software sobre as ferramentas para criar um pacote de Windows Installer para um aplicativo. Essas ferramentas podem fornecer um ambiente de criação de pacotes que pode ser mais fácil de usar do que as ferramentas fornecidas no SDK do Windows Installer.
