# <a name="propertyname-control-attribute"></a>Atributo de controle PropertyName

Este é o nome da propriedade associada a este controle. Se o bit [indireto](indirect-control-attribute.md) não estiver definido, o controle exibirá ou alterará o valor da propriedade com esse nome. Esse atributo é especificado na coluna propriedade da tabela de [controle](control-table.md). Esse atributo retorna NULL para controles Inert.

## <a name="valid-controls"></a>Controles válidos

Todos os controles.

## <a name="associated-bit-flags"></a>Sinalizadores de bits associados

Este atributo não usa sinalizadores de bit.

## <a name="remarks"></a>Comentários

Consulte [atributos de controle](control-attributes.md) e o controle que você precisa criar sob [controles](controls.md).
