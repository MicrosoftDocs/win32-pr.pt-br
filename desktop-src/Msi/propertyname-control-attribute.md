# <a name="propertyname-control-attribute"></a>Atributo de controle PropertyName

Esse é o nome da propriedade associada a esse controle. Se o bit [Indireto](indirect-control-attribute.md) não estiver definido, o controle exibirá ou altera o valor da propriedade que tem esse nome. Esse atributo é especificado na coluna Propriedade da [tabela Control](control-table.md). Esse atributo retorna Null para controles inertes.

## <a name="valid-controls"></a>Controles válidos

Todos os controles.

## <a name="associated-bit-flags"></a>Sinalizadores de bit associados

Esse atributo não usa sinalizadores de bit.

## <a name="remarks"></a>Comentários

Consulte [Atributos de controle](control-attributes.md) e o controle que você precisa criar em [Controles](controls.md).
