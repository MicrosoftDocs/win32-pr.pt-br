# <a name="applying-merge-modules"></a>Aplicando módulos de mesclagem

Os módulos de mesclagem devem ser mesclados em um pacote de instalação usando uma ferramenta de mesclagem. A melhor alternativa é obter uma ferramenta de mesclagem distribuída livremente ou comprar uma das ferramentas de mesclagem a serem disponibilizadas por fornecedores de software independentes. Por exemplo, você pode usar a funcionalidade fornecida pelo Mergemod.dll.

Os módulos de mesclagem em vários idiomas podem ser abertos em idiomas diferentes e mesclados em um destino para produzir uma versão de idioma diferente do pacote. Para obter mais informações, consulte:

-   [Usando a automação para mesclar um módulo de mesclagem em um banco de dados](using-automation-to-merge-a-merge-module-into-a-database.md)
-   [Usando a API para mesclar um módulo de mesclagem em um banco de dados](using-the-api-to-merge-a-merge-module-into-a-database.md)
-   [Abrindo um módulo de mesclagem Multiple-Language em um idioma específico](opening-a-multiple-language-merge-module-in-a-specific-language.md)
-   [Mesclar um módulo de vários idiomas no mesmo pacote várias vezes](merging-a-multiple-language-module-into-the-same-package-multiple-times.md)
-   [Conectando um módulo de mesclagem a vários recursos](connecting-a-merge-module-to-multiple-features.md)
