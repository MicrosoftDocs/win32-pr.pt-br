# <a name="installerversion-property"></a>Propriedade Installer. Version

A propriedade **version** do objeto [**instalador**](installer-object.md) é uma propriedade somente leitura que é a representação de cadeia de caracteres da versão atual do Windows Installer. A cadeia de caracteres é retornada no formulário a seguir.

*principal*. *secundária*. *Compilar*. *Atualizar* o

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Installer.Version
```



## <a name="property-value"></a>Valor da propriedade

|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
