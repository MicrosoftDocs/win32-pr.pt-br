# <a name="icem01"></a>ICEM01

ICEM01 valida que o mecanismo ICE está funcionando. Essa ICE usa a propriedade [**time**](time.md) para obter a hora e retorna a hora do sistema ou nenhuma.

O módulo de mesclagem ICEs é armazenado em um arquivo. Cub do módulo de mesclagem chamado Mergemod. Cub e não no arquivo. Cub que contém a ICEs usada para a validação do pacote.

## <a name="result"></a>Resultado

ICEM01 posta uma mensagem fornecendo a hora em que o instalador chamou ICEM01. Essa ICE nunca deve retornar um erro.
