# <a name="system16folder-property"></a>Propriedade System16Folder

Windows O instalador define a **propriedade System16Folder** como o caminho completo para a pasta para DLLs do sistema de 16 bits. Observe que a **propriedade System16Folder** não está definida para Windows de 64 bits porque esse sistema não dá suporte a aplicativos ou componentes de 16 bits.

Windows Instalador 4.0 ou Windows Instalador 4.5 no Windows Server 2008 ou Windows Vista. Windows Instalador no Windows Server 2003 ou Windows XP.
