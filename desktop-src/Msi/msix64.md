# <a name="msix64-property"></a>Propriedade Msix64

A propriedade **Msix64** só será definida se estiver em execução em um processador x64. O valor é definido pelo instalador para o nível numérico do processador. Os valores são os mesmos do campo **wProcessorLevel** da estrutura de [**\_ informações do sistema**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info) .

## <a name="remarks"></a>Comentários

**Windows Installer 2,0 e Windows Installer 3,0:** Sem suporte. A propriedade **Msix64** está disponível a partir do Windows Installer 3,1.
