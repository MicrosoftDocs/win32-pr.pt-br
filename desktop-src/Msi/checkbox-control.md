# <a name="checkbox-control"></a>Controle de CheckBox

Esse controle CheckBox \_ é uma caixa de seleção de dois estados. Para associar um inteiro ou uma propriedade de cadeia de caracteres a esse controle, insira o nome da propriedade na coluna Propriedade da [tabela Controle](control-table.md). O estado selecionado da caixa define a propriedade como o valor especificado na coluna Valor da tabela [CheckBox](checkbox-table.md) ou como o valor inicial da propriedade especificada na [tabela Property](property-table.md). Se a propriedade não tiver nenhum valor inicial, o estado verificado a define como 1. O estado não selecionado define a propriedade como null.

Os controles CheckBox só podem ser usados para publicar controles [AddLocal ControlEvent](addlocal-controlevent.md), [AddSource ControlEvent](addsource-controlevent.md), [Remove ControlEvent](remove-controlevent.md), [DoAction ControlEvent](doaction-controlevent.md)ou [SetProperty ControlEvent.](setproperty-controlevent.md)

## <a name="remarks"></a>Comentários

Esse controle pode ser criado com a classe BUTTON usando a [**função CreateWindowEx.**](/windows/win32/api/winuser/nf-winuser-createwindowexa) Ele tem os **estilos BS \_ CHECKBOX**, **WS \_ TABSTOP,** **WS \_ GROUP,** **WS \_ CHILD** e **BS \_ MULTILINE.**
