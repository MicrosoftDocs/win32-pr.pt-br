# <a name="checkbox-control"></a>Controle de CheckBox

Esse \_ controle CheckBox é uma caixa de seleção de dois Estados. Para associar uma propriedade de inteiro ou de cadeia de caracteres a esse controle, insira o nome da propriedade na coluna propriedade da [tabela de controle](control-table.md). O estado selecionado da caixa define a propriedade para o valor especificado na coluna valor da tabela de caixa de [seleção](checkbox-table.md) ou para o valor inicial da propriedade especificada na [tabela de propriedades](property-table.md). Se a propriedade não tiver nenhum valor inicial, o estado selecionado a definirá como 1. O estado não selecionado define a propriedade como NULL.

Os controles de caixa de seleção só podem ser usados para publicar os controles [ADDLOCAL ControlEvent,](addlocal-controlevent.md), [addsource ControlEvent,](addsource-controlevent.md), [Remove ControlEvent,](remove-controlevent.md), [DoAction ControlEvent,](doaction-controlevent.md)ou [SetProperty ControlEvent,](setproperty-controlevent.md) .

## <a name="remarks"></a>Comentários

Esse controle pode ser criado a partir da classe BUTTON usando a função [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Ele tem os estilos de **\_ caixa de seleção BS**, **WS \_ TabStop**, **WS \_ Group**, **WS \_ Child** e **BS \_ multilinha** .
