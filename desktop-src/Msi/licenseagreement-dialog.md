# <a name="licenseagreement-dialog"></a>Caixa de diálogo LicenseAgreement

Essa caixa de diálogo modal exibe o contrato de licença para o usuário. Normalmente, a caixa de diálogo exibe o texto do contrato usando um [controle ScrollableText](scrollabletext-control.md) e um par de botões de opção que permitem que o usuário aceite ou rejeite o contrato. Por motivos legais, é aconselhável que nenhum botão seja apresentado ao usuário como já selecionado por padrão. Isso força o usuário a fazer uma escolha ativa. Os autores geralmente usam um [controle de botão de opção](radiobuttongroup-control.md) para essa finalidade. No entanto, observe que o foco em uma caixa de diálogo não é movido para um controle de grupo de botões de opção até que um dos botões no grupo tenha sido selecionado.
