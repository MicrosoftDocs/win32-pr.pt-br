# <a name="released-versions-tools-and-redistributables"></a>Versões, ferramentas e redistribuíveis liberados

os tópicos a seguir fornecem mais informações sobre versões lançadas, ferramentas e redistribuíveis do Windows Installer.

-   [SDK do Windows components for Windows Installer developers](platform-sdk-components-for-windows-installer-developers.md) explica onde obter os componentes redistribuíveis do instalador, a documentação, a ferramenta de validação de banco de dados, o editor de tabela de banco de dados, o esquema de banco, as ferramentas de desenvolvimento, as ferramentas VBScript, o produto de exemplo
-   [Windows Installer redistribuíveis](windows-installer-redistributables.md) fornece informações sobre onde obter os componentes redistribuíveis do instalador.
-   [versões lançadas do Windows Installer](released-versions-of-windows-installer.md) lista as versões lançadas do instalador que foram enviadas com sistemas operacionais diferentes.
-   [Windows Installer ferramentas de desenvolvimento](windows-installer-development-tools.md) contêm páginas de referência para usar as ferramentas de desenvolvimento do instalador.
-   [Instmsi.exe](instmsi-exe.md) descreve qual Instmsi é compatível em qual sistema operacional. InstMsi é o pacote redistribuível para instalar o instalador.
