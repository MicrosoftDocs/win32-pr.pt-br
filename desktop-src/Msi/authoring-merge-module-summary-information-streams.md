# <a name="authoring-merge-module-summary-information-streams"></a>Criando fluxos de informações de resumo do módulo de mesclagem

Ao criar um módulo de mesclagem, o fluxo de informações de resumo é necessário para incluir essas propriedades de informações de resumo necessárias.

Cada módulo de mesclagem precisa ter um fluxo de informações resumido de módulo de mesclagem com as seguintes propriedades de informações de Resumo:

-   [**Resumo do modelo**](template-summary.md)
-   [**Resumo do número de revisão**](revision-number-summary.md)
-   [**Resumo de contagem de páginas**](page-count-summary.md)
-   [**Resumo de contagem de palavras**](word-count-summary.md)

Para obter uma lista completa das propriedades de informações de resumo em módulos de mesclagem, consulte [mesclar referência de fluxo de informações de resumo do módulo](merge-module-summary-information-stream-reference.md).
