# <a name="authoring-merge-module-summary-information-streams"></a>Informações de resumo do módulo de mesclagem de Fluxos

Ao autorizar um módulo de mesclagem, o fluxo de informações resumidas é necessário para incluir essas propriedades de informações resumidas necessárias.

Cada módulo de mesclagem é necessário para ter um Fluxo de Informações de Resumo do Módulo de Mesclagem com as seguintes propriedades de informações resumidas:

-   [**Resumo do modelo**](template-summary.md)
-   [**Resumo do número de revisão**](revision-number-summary.md)
-   [**Resumo da contagem de páginas**](page-count-summary.md)
-   [**Resumo da contagem de palavras**](word-count-summary.md)

Para obter uma lista completa das propriedades de informações resumidas em módulos de mesclagem, consulte Referência do fluxo de informações de resumo do módulo [de mesclagem.](merge-module-summary-information-stream-reference.md)
