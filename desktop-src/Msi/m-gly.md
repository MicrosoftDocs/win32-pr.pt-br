<dl> <dt>

<span id="_msi_managed_application_gly"></span><span id="_MSI_MANAGED_APPLICATION_GLY"></span>**aplicativo gerenciado**
</dt> <dd>

Os privilégios do sistema, também conhecidos como privilégios elevados, são usados para instalar o aplicativo. Um aplicativo será gerenciado em um sistema se for uma instalação por computador no Windows 2000 ou se o aplicativo for atribuído ou publicado usando Política de Grupo no Windows 2000.

</dd> <dt>

<span id="_msi_merge_using_windows_installer_gly"></span><span id="_MSI_MERGE_USING_WINDOWS_INSTALLER_GLY"></span>**Mesclagem**
</dt> <dd>

Combina dois bancos de dados de instalação para formar um banco de dados individual. Para obter mais informações, consulte [Mesclando bancos de dados](merging-databases.md).

</dd> <dt>

<span id="_msi_merge_module_gly"></span><span id="_MSI_MERGE_MODULE_GLY"></span>**módulo merge**
</dt> <dd>

Banco de dados mesclado com um pacote de instalação para fornecer novos componentes a um aplicativo. Módulos de mesclagem não podem ser instalados sozinhos. Para obter mais informações, consulte [Mesclar módulos](merge-modules.md).

</dd> <dt>

<span id="_msi_microsoft_windows_installer_gly"></span><span id="_MSI_MICROSOFT_WINDOWS_INSTALLER_GLY"></span>**Microsoft Windows Installer**
</dt> <dd>

Serviço de instalador do lado do cliente para gerenciar a instalação de aplicativos em plataformas de 32 bits. O aplicativo deve ser encapsulado em um [*pacote*](p-gly.md). Para obter mais informações, consulte [About Windows Installer](about-windows-installer.md).

</dd> <dt>

<span id="_msi_.msi_file_gly"></span><span id="_MSI_.MSI_FILE_GLY"></span>**.msi arquivo**
</dt> <dd>

Arquivo de armazenamento estruturado com COM que contém as instruções e os dados necessários para instalar um aplicativo. Cada [*pacote*](p-gly.md) contém pelo menos um .msi arquivo. O .msi arquivo contém o banco de [](s-gly.md)dados do instalador [*,*](i-gly.md)um fluxo de informações de resumo e, possivelmente, uma ou mais [*transformaçãos*](t-gly.md) e [*arquivos de origem internos*](i-gly.md). Para obter mais informações, [consulte Windows extensões de arquivo do instalador.](windows-installer-file-extensions.md)

</dd> <dt>

<span id="_msi_.msm_file_gly"></span><span id="_MSI_.MSM_FILE_GLY"></span>**Arquivo .msm**
</dt> <dd>

[Módulo mesclar](merge-modules.md). Para obter mais informações, [consulte Windows extensões de arquivo do instalador.](windows-installer-file-extensions.md)

</dd> <dt>

<span id="_msi_.msp_file"></span><span id="_MSI_.MSP_FILE"></span>**Arquivo .msp**
</dt> <dd>

[Pacote de patch](patch-packages.md). Para obter mais informações, [consulte Windows extensões de arquivo do instalador.](windows-installer-file-extensions.md)

</dd> <dt>

<span id="_msi_.mst_file_gly"></span><span id="_MSI_.MST_FILE_GLY"></span>**Arquivo .mst**
</dt> <dd>

Transformar pacote. Para obter mais informações, consulte [Merges and Transforms](merges-and-transforms.md). Para obter mais informações, [consulte Windows extensões de arquivo do instalador.](windows-installer-file-extensions.md)

</dd> </dl>
