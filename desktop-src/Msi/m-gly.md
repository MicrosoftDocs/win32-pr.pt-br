# <a name="m-windows-installer"></a>M (Windows Installer)

<dl> <dt>

<span id="_msi_managed_application_gly"></span><span id="_MSI_MANAGED_APPLICATION_GLY"></span>**aplicativo gerenciado**
</dt> <dd>

Os privilégios do sistema, também conhecidos como privilégios elevados, são usados para instalar o aplicativo. Um aplicativo será gerenciado em um sistema se for uma instalação por computador no Windows 2000 ou se o aplicativo for atribuído ou publicado usando Política de Grupo no Windows 2000.

</dd> <dt>

<span id="_msi_merge_using_windows_installer_gly"></span><span id="_MSI_MERGE_USING_WINDOWS_INSTALLER_GLY"></span>**Mescle**
</dt> <dd>

Combina dois bancos de dados de instalação para formar um único banco. Para obter mais informações, consulte [mesclando bancos de dados](merging-databases.md).

</dd> <dt>

<span id="_msi_merge_module_gly"></span><span id="_MSI_MERGE_MODULE_GLY"></span>**Mesclar módulo**
</dt> <dd>

Banco de dados mesclado com um pacote de instalação para entregar novos componentes a um aplicativo. Os módulos de mesclagem não podem ser instalados sozinhas. Para obter mais informações, consulte [Mesclar módulos](merge-modules.md).

</dd> <dt>

<span id="_msi_microsoft_windows_installer_gly"></span><span id="_MSI_MICROSOFT_WINDOWS_INSTALLER_GLY"></span>**Microsoft Windows Installer**
</dt> <dd>

Serviço de instalação do lado do cliente para gerenciar a instalação de aplicativos em plataformas de 32 bits. O aplicativo deve ser encapsulado em um [*pacote*](p-gly.md). Para obter mais informações, consulte [About Windows Installer](about-windows-installer.md).

</dd> <dt>

<span id="_msi_.msi_file_gly"></span><span id="_MSI_.MSI_FILE_GLY"></span>**arquivo. msi**
</dt> <dd>

Arquivo de armazenamento estruturado COM que contém as instruções e os dados necessários para instalar um aplicativo. Cada [*pacote*](p-gly.md) contém pelo menos um arquivo. msi. O arquivo. msi contém o [*banco de dados do instalador*](i-gly.md), um [*fluxo de informações de resumo*](s-gly.md)e, possivelmente, uma ou mais [*transformações*](t-gly.md) e [*arquivos de origem internos*](i-gly.md). Para obter mais informações, consulte [Windows Installer extensões de arquivo](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msm_file_gly"></span><span id="_MSI_.MSM_FILE_GLY"></span>**arquivo. msm**
</dt> <dd>

[Módulo de mesclagem](merge-modules.md). Para obter mais informações, consulte [Windows Installer extensões de arquivo](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msp_file"></span><span id="_MSI_.MSP_FILE"></span>**arquivo. msp**
</dt> <dd>

[Pacote de patch](patch-packages.md). Para obter mais informações, consulte [Windows Installer extensões de arquivo](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.mst_file_gly"></span><span id="_MSI_.MST_FILE_GLY"></span>**arquivo. MST**
</dt> <dd>

Transforme o pacote. Para obter mais informações, consulte [mesclagens e transformações](merges-and-transforms.md). Para obter mais informações, consulte [Windows Installer extensões de arquivo](windows-installer-file-extensions.md).

</dd> </dl>
