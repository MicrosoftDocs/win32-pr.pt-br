# <a name="ttcsupport-property"></a>Propriedade TTCSupport

O instalador define a propriedade **TTCSupport** para indicar se o sistema operacional dá suporte a arquivos. TTC (conjuntos de fontes True Type).

## <a name="remarks"></a>Comentários

essa propriedade será definida se o instalador detectar Windows 2000 ou qualquer um dos seguintes sistemas operacionais:

-   Japão-932
-   China-936
-   Taiwan-950
-   Coreia-949
-   Rae de Hong Kong-950
