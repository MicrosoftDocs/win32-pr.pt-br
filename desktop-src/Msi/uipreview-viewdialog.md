# <a name="uipreviewviewdialog-method"></a>Método UIPreview. ViewDialog

O método **ViewDialog** do objeto [**UIPreview**](uipreview-object.md) exibe uma caixa de diálogo de interface do usuário criada no banco de dados atual.

## <a name="syntax"></a>Sintaxe


```JScript
UIPreview.ViewDialog(
  dialog
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*'* 
</dt> <dd>

Nome necessário da caixa de diálogo, diferencia maiúsculas de minúsculas e a chave primária da tabela do banco de dados de diálogo.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
