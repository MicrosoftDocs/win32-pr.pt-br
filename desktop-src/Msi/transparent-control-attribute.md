# <a name="transparent-control-attribute"></a>Atributo de controle transparente

Se o bit de controle transparente for definido em um controle de texto, o controle será exibido de forma transparente com o plano de fundo mostrando o controle em que não há nenhum caractere. Se esse bit não estiver definido, o controle de texto será opaco.

## <a name="valid-controls"></a>Controles válidos

[Text](text-control.md)

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                              |
|---------|-------------|---------------------------------------|
| 65536   | 0x00010000  | **msidbControlAttributesTransparent** |



 

## <a name="remarks"></a>Comentários
