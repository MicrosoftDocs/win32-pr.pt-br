# <a name="replacedinusefiles-property"></a>Propriedade ReplacedInUseFiles

A propriedade **ReplacedInUseFiles** será definida se o instalador gravar em um arquivo que está sendo mantido em uso. Essa propriedade é usada por ações personalizadas para detectar que uma reinicialização é necessária para concluir a instalação.

Definido durante as ações [InstallExecute](installexecute-action.md) e [InstallFinalize](installfinalize-action.md) .
