# <a name="mergemodulecabinet"></a>MergeModule.CABinet

Cada arquivo fornecido pelo módulo de mesclagem deve ser armazenado dentro de um arquivo de gabinete inserido como um fluxo no armazenamento estruturado do módulo de mesclagem. Em um módulo de mesclagem padrão, o nome desse gabinete é sempre: MergeModule.CABinet. Para obter mais informações, consulte [gerando MergeModule.CABarquivos de gabinete inet](generating-mergemodule-cabinet-cabinet-files.md).
