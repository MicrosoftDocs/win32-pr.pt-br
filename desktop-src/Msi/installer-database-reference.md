# <a name="installer-database-reference"></a>Referência de banco de dados do instalador

Para adicionar a funcionalidade do instalador ao seu aplicativo, você pode usar as tabelas de banco de dados Windows Installer e as funções de banco de dados do instalador. Os tópicos nesta seção descrevem cada uma das funções de banco de dados e tabelas de banco de dados.

Para exibir uma lista de todas as tabelas de banco de dados de instalação, consulte [Tabelas de banco de dados](database-tables.md).

Para exibir uma lista das funções usadas para trabalhar com informações em bancos de dados de instalação, consulte Funções de banco [de dados](database-functions.md).
