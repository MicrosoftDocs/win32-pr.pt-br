# <a name="installer-database-reference"></a>Referência do banco de dados do instalador

Para adicionar a funcionalidade do instalador ao seu aplicativo, você pode usar as tabelas de banco de dados Windows Installer e as funções de banco de dados do instalador. Os tópicos nesta seção descrevem cada uma das funções de banco de dados e das tabelas de banco de dados.

Para exibir uma lista de todas as tabelas de banco de dados de instalação, consulte [tabelas de banco de dados](database-tables.md).

Para exibir uma lista das funções que são usadas para trabalhar com informações em bancos de dados de instalação, consulte [Database Functions](database-functions.md).
