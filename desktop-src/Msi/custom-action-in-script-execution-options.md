# <a name="custom-action-in-script-execution-options"></a>Opções de execução de In-Script de ação personalizada

Você pode usar os seguintes sinalizadores de opção para especificar a execução em script de ações personalizadas. Essas opções copiam o código de ação para o script de execução, reversão ou confirmação. Para definir uma opção, adicione o valor nessa tabela ao valor no campo tipo da [tabela CustomAction](customaction-table.md).

Observe que o **msidbCustomActionTypeInScript** deve ser incluído em cada uma dessas opções.
