# <a name="using-isolated-components"></a>Usando componentes isolados

Usando [componentes isolados](isolated-components.md), os autores de pacotes de instalação podem especificar que o instalador Copie os arquivos compartilhados de um aplicativo diretamente na pasta do aplicativo em vez de em um local compartilhado. Esse conjunto particular de arquivos é usado exclusivamente pelo aplicativo.

Para obter mais informações, consulte [instalando um componente com em um local privado](installing-a-com-component-to-a-private-location.md), [instalando um componente que não seja com em um local privado](installing-a-non-com-component-to-a-private-location.md), [tornar um componente com em um pacote existente](make-a-com-component-in-an-existing-package-private.md) ou [tornar um componente não-com em um pacote existente privado](make-a-non-com-component-in-an-existing-package-private.md).
