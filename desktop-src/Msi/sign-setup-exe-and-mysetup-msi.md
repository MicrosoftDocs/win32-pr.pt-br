# <a name="sign-setupexe-and-mysetupmsi"></a>Assinar Setup.exe e MySetup.msi

Antes de posicionar Setup.exe e MySetup.msi no servidor Web, você deve assinar os arquivos com seu certificado digital e a chave privada, MyCert. cer e MyCert. pvk, usando o utilitário [SignTool](../seccrypto/signtool.md) . Para obter mais informações sobre como usar o utilitário SignTool, consulte [referência de ferramentas de CryptoAPI](../seccrypto/cryptoapi-tools-reference.md) no SDK (Software Development Kit) do Microsoft Windows.
