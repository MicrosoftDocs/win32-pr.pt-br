# <a name="disablerollback-property"></a>Propriedade DISABLEROLLBACK

Definir a propriedade **DISABLEROLLBACK** desabilita a opção de [reversão](rollback-installation.md) para a configuração atual. Defina essa propriedade como 1 para impedir que o instalador gere um script de reversão e salve cópias de arquivos excluídos durante a instalação.

## <a name="remarks"></a>Comentários

Por padrão, a reversão está habilitada.
