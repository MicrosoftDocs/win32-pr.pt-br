# <a name="template"></a>Modelo

O tipo de dados de modelo é uma cadeia de caracteres de texto que pode conter propriedades que são colocadas entre colchetes \[ \] . O tipo de modelo permite todos os formatos de tipo [formatado](formatted.md) , além de \[ *1* , \] em que *1* é um número.
