# <a name="template"></a>Modelo

O tipo de dados Template é uma cadeia de caracteres de texto que pode conter propriedades que estão entre colchetes \[ \] . O tipo de modelo permite todos os [formatos de](formatted.md) tipo formatado, além \[ *de 1* \] em que *1* é um número.
