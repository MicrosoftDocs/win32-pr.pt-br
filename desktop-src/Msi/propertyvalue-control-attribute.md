# <a name="propertyvalue-control-attribute"></a>Atributo de controle PropertyValue

Esse atributo é o valor atual da propriedade exibida ou alterada por esse controle. Se o bit [Indireto](indirect-control-attribute.md) não estiver definido, esse será o valor de PropertyName. Se o bit de atributo Indireto estiver definido, esse será o valor de IndirectPropertyName. Se o atributo mudar, o controle refletirá o novo valor como uma cadeia de caracteres ou inteiro, dependendo do atributo [Integer](integer-control-attribute.md) do controle.

## <a name="valid-controls"></a>Controles válidos

Todos os controles ativos.

## <a name="associated-bit-flags"></a>Sinalizadores de bit associados

Esse atributo não usa sinalizadores de bit.

## <a name="remarks"></a>Comentários

Consulte [Atributos de controle](control-attributes.md) e o controle que você precisa criar em [Controles](controls.md).
