# <a name="nocompanyname-property"></a>Propriedade nocompanyname

Defina a propriedade **NOcompanyname** como 1 para suprimir a configuração automática da propriedade [**CompanyName**](companyname.md) pelo instalador. Essa propriedade é usada por aplicativos que precisam coletar o nome da empresa na primeira execução.

## <a name="default-value"></a>Valor padrão

Não definido. O instalador define a propriedade [**CompanyName**](companyname.md) automaticamente usando valores do registro.
