# <a name="nocompanyname-property"></a>Propriedade NOCOMPANYNAME

De definir **a propriedade NOCOMPANYNAME** como 1 para suprimir a configuração automática da [**propriedade COMPANYNAME**](companyname.md) pelo instalador. Essa propriedade é usada por aplicativos que precisam coletar o nome da empresa na primeira vez.

## <a name="default-value"></a>Valor padrão

Não definido. O instalador define a [**propriedade COMPANYNAME**](companyname.md) automaticamente usando valores do Registro.
