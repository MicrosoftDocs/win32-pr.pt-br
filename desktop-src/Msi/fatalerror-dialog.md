# <a name="fatalerror-dialog"></a>Caixa de diálogo FatalError

Essa caixa de diálogo modal será exibida no final de uma instalação, se a instalação for encerrada devido a um erro fatal. Essa caixa de diálogo normalmente contém apenas algum texto e um botão **OK** . O nome dessa caixa de diálogo deve aparecer na [tabela AdminUISequence](adminuisequence-table.md) e na [tabela InstallUISequence](installuisequence-table.md) com-3 como o número de sequência.
