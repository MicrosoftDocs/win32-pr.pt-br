# <a name="logaction-property"></a>Propriedade LOGACTION

A propriedade **LOGACTION** é uma lista de nomes de ação separados somente por ponto e vírgula e sem espaços. Os nomes de ação nessa propriedade não diferenciam maiúsculas de minúsculas e a lista pode terminar com um ponto-e-vírgula.

## <a name="remarks"></a>Comentários

O instalador Registra mensagens de dados de ação para as ações que estão na lista. A propriedade não tem nenhum efeito sobre outros tipos de registro em log.
