# <a name="errortype-property"></a>Propriedade Error. Type

A propriedade **tipo** somente leitura retorna um dos valores de msmErrorType, indicando o tipo de erro representado por esse objeto.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Error.Type
```



## <a name="property-value"></a>Valor da propriedade

## <a name="c"></a>C++

Consulte a função [**obter \_ tipo de função**](/windows/win32/api/mergemod/nf-mergemod-imsmerror-get_type) (objeto de erro).
