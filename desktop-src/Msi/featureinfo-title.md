# <a name="featureinfotitle-property"></a>Propriedade FeatureInfo. title

A propriedade **title** é uma propriedade somente leitura que retorna o título do recurso na coluna título da [tabela de recursos](feature-table.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = FeatureInfo.Title
```



## <a name="property-value"></a>Valor da propriedade
