# <a name="summaryinfopropertycount-property"></a>Propriedade SummaryInfo. PropertyCount

A propriedade **PropertyCount** do objeto [**SummaryInfo**](summaryinfo-object.md) é uma propriedade somente leitura que indica o número atual de valores de propriedade no objeto de informações de resumo. Ele leva em conta as propriedades que foram adicionadas, excluídas ou substituídas.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = SummaryInfo.PropertyCount
```
