# <a name="windows-installer-file-name-extensions"></a>Extensões de nome de arquivo Windows Installer

As seguintes extensões de nome de arquivo são usadas com o Windows Installer.



| Extensão | Descrição                                |
|-----------|--------------------------------------------|
| .msi      | Windows Installer banco de dados.                |
| . msm      | Windows Installer módulo de mesclagem.            |
| .msp      | Windows Installer patch.                   |
| .mst      | Transformação de Windows Installer.               |
| . IDT      | Tabela de banco de dados Windows Installer exportada. |
| . Cub      | Módulo de validação.                         |
| . PCP      | Windows Installer arquivo de criação de patch      |
