# <a name="msicheckcrcs-property"></a>Propriedade MSICHECKCRCS

A CRC (verificação de redundância cíclica) está disponível com Windows Installer versão 2,0 e posterior. O instalador fará um CRC nos arquivos somente se a propriedade **MSICHECKCRCS** estiver definida. Para obter mais informações, consulte [verificação de CRC durante uma instalação](crc-checking-during-an-installation.md).

Windows Installer 4,0 ou Windows Installer 4,5 no Windows Server 2008 ou no Windows Vista. Windows Installer no Windows Server 2003 ou no Windows XP.
