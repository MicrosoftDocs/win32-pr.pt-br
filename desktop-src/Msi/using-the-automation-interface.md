# <a name="using-the-automation-interface"></a>Usando a interface de automação

Para acessar a automação, a DLL deve ser autoregistrada. O ProgId COM usado para criar o [**objeto do instalador**](installer-object.md) é **WindowsInstaller. Installer**.

Para obter informações sobre automação e acesso a bibliotecas C++, consulte [sobre a interface de automação](about-the-automation-interface.md).

para obter material de referência para os objetos Windows Installer, consulte [referência da Interface de automação](automation-interface-reference.md).
