# <a name="terminalserver-property"></a>Propriedade TerminalServer

O instalador define a propriedade **TerminalServer** quando o sistema é um servidor com serviços de terminal.

## <a name="default-value"></a>Valor padrão

Nenhum. A propriedade é indefinida quando não está em execução nos serviços de terminal.

## <a name="remarks"></a>Comentários

a propriedade [**RemoteAdminTS**](remoteadmints.md) só pode ser definida no Microsoft Windows 2000 ou posterior. a propriedade **TerminalServer** pode ser definida em Windows 2000.
