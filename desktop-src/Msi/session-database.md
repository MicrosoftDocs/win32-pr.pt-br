# <a name="sessiondatabase-property"></a>Propriedade Session.Database

A **propriedade Database** do objeto Session é [**uma**](session-object.md) propriedade somente leitura que retorna o banco de dados para a sessão de instalação atual como um [**objeto Database.**](database-object.md)

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Session.Database
```
