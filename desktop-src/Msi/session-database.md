# <a name="sessiondatabase-property"></a>Propriedade Session. Database

A propriedade **Database** do objeto [**Session**](session-object.md) é uma propriedade somente leitura que retorna o banco de dados para a sessão de instalação atual como um objeto de [**banco de dados**](database-object.md) .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Session.Database
```



## <a name="property-value"></a>Valor da propriedade
