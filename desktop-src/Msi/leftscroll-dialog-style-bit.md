# <a name="leftscroll-dialog-style-bit"></a>Bit de estilo de caixa de diálogo LeftScroll

Se esse bit de estilo for definido, a barra de rolagem estará localizada no lado esquerdo da caixa de diálogo.

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                            |
|---------|-------------|-------------------------------------|
| 512     | 0x00000200  | **msidbDialogAttributesLeftScroll** |
