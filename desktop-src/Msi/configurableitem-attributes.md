# <a name="configurableitemattributes-property"></a>Propriedade ConfigurableItem. Attributes

A propriedade **Attributes** retorna o valor da coluna Attributes da [Tabela ModuleConfiguration](moduleconfiguration-table.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = ConfigurableItem.Attributes
```



## <a name="property-value"></a>Valor da propriedade

## <a name="c"></a>C++

Confira a função [**obter \_ atributos**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_attributes) .
