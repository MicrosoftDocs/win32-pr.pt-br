# <a name="ice01"></a>ICE01

ICE01 valida que o mecanismo ICE está funcionando. Essa ICE usa a propriedade [**time**](time.md) para obter a hora e retorna a hora do sistema ou nenhuma.

## <a name="result"></a>Resultado

ICE01 posta uma mensagem fornecendo a hora em que o instalador chamou ICE01. Essa ICE nunca deve retornar um erro.
