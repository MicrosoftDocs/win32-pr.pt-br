# <a name="ice01"></a>ICE01

ICE01 valida se o mecanismo ICE está funcionando. Esse ICE usa a [**propriedade Time**](time.md) para obter a hora e retorna a hora do sistema ou Nenhum.

## <a name="result"></a>Resultado

ICE01 posta uma mensagem dando a hora em que o instalador chamou ICE01. Esse ICE nunca deve retornar um erro.
