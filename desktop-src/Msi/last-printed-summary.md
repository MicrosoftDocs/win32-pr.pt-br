# <a name="last-printed-summary-property"></a>Última propriedade de resumo impressa

Em um pacote de instalação, a **última Propriedade Resumo impressa** pode ser definida como a data e a hora durante uma [instalação administrativa](administrative-installation.md) para registro quando a imagem administrativa foi criada. Para instalações não administrativas, essa propriedade é a mesma que a propriedade de [**Resumo de data/hora de criação**](create-time-date-summary.md) .

Em uma transformação, essa propriedade deve ser nula.

Em um pacote de patch, essa propriedade deve ser NULL.
