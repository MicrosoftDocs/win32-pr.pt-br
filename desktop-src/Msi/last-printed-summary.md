# <a name="last-printed-summary-property"></a>Propriedade Resumo da Última Impressa

Em um pacote de instalação, [a](administrative-installation.md) propriedade **Resumo** da Última Impressa pode ser definida como a data e a hora durante uma instalação administrativa para registrar quando a imagem administrativa foi criada. Para instalações não administrativas, essa propriedade é a mesma que a propriedade Criar Resumo [**de Data/Hora.**](create-time-date-summary.md)

Em uma transformação, essa propriedade deve ser nula.

Em um pacote de patch, essa propriedade deve ser nula.
