# <a name="windows-installer-error-messages-for-developers"></a>Windows Installer mensagens de erro (para desenvolvedores)

> [!NOTE]  
> Este artigo destina-se a desenvolvedores de software que estão usando Windows Installer para criar pacotes do instalador para seus aplicativos. Se você for um usuário com dificuldade em seu computador durante ou após a instalação ou desinstalação de um aplicativo, entre em contato com o suporte ao cliente para o software que você está tentando instalar ou remover. Se você sentir que precisa de suporte para um produto da Microsoft, acesse nosso site de suporte técnico em [support.Microsoft.com](https://support.microsoft.com).

Os códigos de erro detalhados neste tópico são retornados pelo Windows Installer e têm códigos de erro de 1000 ou mais. Os códigos de erro numerados 1000 a 1999 são os erros de remessa e devem ser criados na [tabela de erros](error-table.md). Os códigos de erro numerados maiores que 2000 são erros internos e não têm cadeias de caracteres criadas, mas podem ocorrer se o pacote de instalação tiver sido criado incorretamente. Para códigos de erro específicos das funções Windows Installer **MsiExec.exe** e **InstMsi.exe**, consulte [MsiExec.exe e InstMsi.exe mensagens de erro](error-codes.md). Para obter uma lista de códigos de erro reservados, consulte [tabela de erros](error-table.md). Você pode pesquisar a Internet ou o [site de suporte da Microsoft](https://support.microsoft.com/) para obter soluções para muitas das mensagens na tabela a seguir.

Os desenvolvedores de pacotes de instalação também podem testar a consistência interna de seus pacotes usando avaliadores de consistência internos. Para obter mais informações, consulte [avaliadores de consistência internos-ICES](internal-consistency-evaluators-ices.md).
