# <a name="error-message-custom-actions"></a>Ações personalizadas da mensagem de erro

Esse tipo de ação personalizada exibe uma mensagem de erro especificada e retorna uma falha, encerrando a instalação. A mensagem de erro a ser exibida pode ser fornecida como uma cadeia de caracteres ou como um índice na [tabela de erros](error-table.md).



| Tipo de ação personalizada                                 | Descrição                                                |
|----------------------------------------------------|------------------------------------------------------------|
| [Tipo de ação personalizada 19](custom-action-type-19.md) | Encerra a instalação e exibe uma mensagem de erro. |



 

Consulte a [lista Resumo de todos os tipos de ação personalizada](summary-list-of-all-custom-action-types.md) para obter um resumo de como codificar cada um dos tipos de ação personalizada na [tabela CustomAction](customaction-table.md).
