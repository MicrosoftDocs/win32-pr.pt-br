-   [Caminho](path.md)

-   [AnyPath](anypath.md)
-   [DefaultDir](defaultdir.md)
-   [RegPath](regpath.md)
-   [Binário](formatted.md)
-   [FormattedSDDLText](formattedsddltext.md)
-   [Modelo](template.md)
-   [Condição](condition.md)
-   [GUID](guid.md)
-   [Versão](version.md)
-   [Idioma](language.md)
-   [Binary](binary.md)
-   [CustomSource](customsource.md)
-   [Gabinete](cabinet.md)
