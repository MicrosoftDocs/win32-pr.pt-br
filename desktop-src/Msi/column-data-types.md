-   [DefaultDir](defaultdir.md)
-   [RegPath](regpath.md)

-   [Condição](condition.md)
-   [GUID](guid.md)
-   [Versão](version.md)
-   [Idioma](language.md)
-   [Binary](binary.md)
-   [CustomSource](customsource.md)
-   [Gabinete](cabinet.md)
