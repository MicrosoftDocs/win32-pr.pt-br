# <a name="wildcardfilename"></a>WildCardFilename

O tipo de dados WildCardFilename é um [nome de arquivo](filename.md) que também pode conter os caracteres curinga "?" para qualquer caractere único ou " \* " para zero ou mais ocorrências de qualquer caractere.
