# <a name="moduleadvtuisequence-table"></a>Tabela ModuleAdvtUISequence

Não crie esta tabela. O Windows Installer não usa a [tabela AdvtUISequence](advtuisequence-table.md). A tabela AdvtUISequence não deve existir ou deve ser deixada em branco no banco de dados de instalação. Além disso, não crie a tabela ModuleAdvtUISequence.
