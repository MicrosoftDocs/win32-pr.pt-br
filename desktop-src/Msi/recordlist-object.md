# <a name="recordlist-object"></a>Objeto RecordList

O **objeto RecordList** é uma coleção de [**objetos Record.**](record-object.md) Você deve verificar se o **objeto RecordList** existe e não está vazio antes de referenciar suas propriedades.

## <a name="members"></a>Membros

O **objeto RecordList** tem estes tipos de membros:

-   [Propriedades](#properties)

### <a name="properties"></a>Propriedades

O **objeto RecordList** tem essas propriedades.



| Propriedade                                     | Descrição                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**Contagem**](recordlist-count.md)<br/> | Retorna o número de itens no **objeto RecordList.**<br/> |
| [**Item**](recordlist-item.md)<br/>   | Retorna um registro em uma coleção **de objetos RecordList.**<br/>   |
