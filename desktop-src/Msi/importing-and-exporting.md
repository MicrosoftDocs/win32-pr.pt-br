# <a name="importing-and-exporting"></a>Importando e exportando

Você pode usar o instalador para importar um arquivo de texto para um banco de dados ativo, bem como para exportar um arquivo de banco de dados para um arquivo de texto. Isso pode ser útil para sistemas de controle do código-fonte baseados em texto.

Para importar um arquivo de texto para um banco de dados ativo, chame a [**função MsiDatabaseImport.**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseimporta)

Para exportar um arquivo de banco de dados para um arquivo de texto, chame a [**função MsiDatabaseExport.**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseexporta)
