# <a name="userexit-dialog"></a>Caixa de diálogo UserExit

Essa caixa de diálogo modal é exibida no final de uma instalação que foi encerrada pelo usuário. Essa caixa de diálogo normalmente contém apenas algum texto e um botão **OK** . O nome dessa caixa de diálogo deve aparecer na [tabela AdminUISequence](adminuisequence-table.md) e na [tabela InstallUISequence](installuisequence-table.md) com-2 como o número de sequência.
