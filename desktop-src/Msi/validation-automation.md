# <a name="validation-automation"></a>Automação de Validação

Você pode usar o Evalcom2.dll para implementar operações de validação para pacotes de instalação e módulos de mesclagem.

As seções a seguir desta documentação contêm informações sobre como usar Windows do Instalador de dentro das ferramentas de autor:

-   [Usando Evalcom2](using-evalcom2.md)
-   [EvalCom2 Interfaces](evalcom2-interfaces.md)

É recomendável que Evalcom2.dll ser instalado usando o Windows Instalador. O pacote Orca.msi fornecido com o SDK (Software Development Kit) do Microsoft Windows para o Windows XP com Service Pack 2 (SP2) ou mais Evalcom2.dll. A ID do componente que contém a interface COM é {53315473-B8D6-470A-9951-D9AE5C11FC91}.

Para obter informações sobre como obter o SDK do Windows, consulte [Componentes do SDK](platform-sdk-components-for-windows-installer-developers.md)do Windows para desenvolvedores Windows instalador.
