# <a name="validation-automation"></a>Automação de validação

Você pode usar o Evalcom2.dll para implementar operações de validação para pacotes de instalação e módulos de mesclagem.

As seções a seguir desta documentação contêm informações sobre como usar Windows Installer validação de dentro de ferramentas de criação:

-   [Usando Evalcom2](using-evalcom2.md)
-   [Interfaces EvalCom2](evalcom2-interfaces.md)

É recomendável que Evalcom2.dll seja instalado usando o Windows Installer. O pacote de Orca.msi fornecido com o SDK (Kit de desenvolvimento de software) do Microsoft Windows para Windows XP com Service Pack 2 (SP2) ou instalações posteriores Evalcom2.dll. A ID do componente que contém a interface COM é {53315473-B8D6-470A-9951-D9AE5C11FC91}.

Para obter informações sobre como obter a SDK do Windows, consulte [SDK do Windows components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md).
