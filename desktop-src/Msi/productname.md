# <a name="productname-property"></a>Propriedade ProductName

A propriedade **ProductName** contém o nome do aplicativo que está sendo instalado. Isso é usado somente para fins de exibição. Anunciado como uma propriedade de produto.

Essa propriedade é necessária.

Essa propriedade pode ser alterada por uma transformação.

## <a name="remarks"></a>Comentários

A propriedade **ProductName** não pode ter mais de 63 caracteres de comprimento. Não existe limite no comprimento da chave do registro para DisplayName.
