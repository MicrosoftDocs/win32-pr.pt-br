# <a name="condition"></a>Condição

O tipo de dados Condition é uma cadeia de caracteres de texto que contém uma instrução condicional válida que pode ser avaliada como true ou false. Para obter informações sobre a sintaxe de instruções condicionais, consulte [sintaxe de instrução condicional](conditional-statement-syntax.md).
