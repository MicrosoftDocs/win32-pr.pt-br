# <a name="updating-directory-structure-for-an-upgrade"></a>Atualizando a estrutura de diretório para uma atualização

Use o Orca do editor de banco de dados fornecido com o SDK ou outro editor para abrir a tabela [Diretório](directory-table.md) no MNP2001.msi. A estrutura de diretório da atualização de exemplo é a mesma que a estrutura de diretório do produto original e nenhuma alteração é necessária para a tabela de diretório original descrita em [Especificando](specifying-directory-structure.md)a estrutura de diretório .
