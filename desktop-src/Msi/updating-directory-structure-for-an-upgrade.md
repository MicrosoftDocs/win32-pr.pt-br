# <a name="updating-directory-structure-for-an-upgrade"></a>Atualizando a estrutura de diretório para uma atualização

Use o editor de banco de dados Orca fornecido com o SDK ou outro editor para abrir a [tabela de diretórios](directory-table.md) no MNP2001.msi. A estrutura de diretório da atualização de exemplo é igual à estrutura de diretório do produto original e nenhuma alteração é necessária para a tabela de diretório original descrita em [especificando a estrutura de diretório](specifying-directory-structure.md).
