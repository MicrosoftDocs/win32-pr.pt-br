# <a name="position-control-attribute"></a>Atributo de controle de posição

Esse atributo determina a posição do controle na caixa de diálogo. Os quatro campos do atributo são: esquerda, superior, largura e altura. Insira a largura, a altura e as coordenadas do canto esquerdo do controle nas colunas Width, Height, X e Y da [tabela Control](control-table.md). Use [unidades do instalador](installer-units.md) para unidades de comprimento e distância.

## <a name="valid-controls"></a>Controles válidos

Todos os controles ativos.

## <a name="associated-bit-flags"></a>Sinalizadores de bit associados

Esse atributo não usa sinalizadores de bit.

## <a name="remarks"></a>Comentários

Consulte [Atributos de controle](control-attributes.md) e o controle que você precisa criar em [Controles](controls.md).

Uma unidade do instalador é igual a um décimo segundo da altura do tamanho da fonte MS Sans Serif de 10 pontos.
