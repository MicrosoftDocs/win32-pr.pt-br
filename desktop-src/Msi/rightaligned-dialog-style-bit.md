# <a name="rightaligned-dialog-style-bit"></a>Bit de estilo de caixa de diálogo RightAligned

Se esse bit de estilo for definido, o texto será alinhado no lado direito da caixa de diálogo.

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                              |
|---------|-------------|---------------------------------------|
| 256     | 0x00000100  | **msidbDialogAttributesRightAligned** |
