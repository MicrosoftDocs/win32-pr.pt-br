<dl> <dt>

<span id="_msi_reduced_ui_gly"></span><span id="_MSI_REDUCED_UI_GLY"></span>**interface do usuário reduzida**
</dt> <dd>

Nível dos recursos internos da [*interface do usuário*](i-gly.md) do instalador. Exibe somente caixas de diálogo sem janela restrita. Para obter mais informações, consulte [níveis de interface do usuário](user-interface-levels.md).

</dd> <dt>

<span id="_msi_reinstall_gly"></span><span id="_MSI_REINSTALL_GLY"></span>**Install**
</dt> <dd>

Reinstalação parcial ou completa de um aplicativo instalado. Normalmente, feito para reparar um aplicativo se algum arquivo ou entrada de registro tiver sido corrompido ou estiver ausente. Para obter mais informações, consulte [reinstalando um recurso ou aplicativo](reinstalling-a-feature-or-application.md).

</dd> <dt>

<span id="_msi_resiliency_gly"></span><span id="_MSI_RESILIENCY_GLY"></span>**resiliência**
</dt> <dd>

Capacidade do aplicativo de recuperação normal a partir de erro sem gerar mensagens de erro. Para obter mais informações, consulte [resiliência](resiliency.md).

</dd> <dt>

<span id="_msi_rollback_gly"></span><span id="_MSI_ROLLBACK_GLY"></span>**versão**
</dt> <dd>

A restauração automática do estado original do computador deve ocorrer falha na instalação. Para obter mais informações, consulte [Rollback Installation (Windows Installer)](rollback-installation.md).

</dd> </dl>
