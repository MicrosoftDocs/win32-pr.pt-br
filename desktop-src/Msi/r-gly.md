<dl> <dt>

<span id="_msi_reduced_ui_gly"></span><span id="_MSI_REDUCED_UI_GLY"></span>**interface do usuário reduzida**
</dt> <dd>

Nível dos recursos de interface do usuário interno [*do instalador.*](i-gly.md) Exibe apenas caixas de diálogo sem modo de autor. Para obter mais informações, consulte [Interface do Usuário níveis](user-interface-levels.md).

</dd> <dt>

<span id="_msi_reinstall_gly"></span><span id="_MSI_REINSTALL_GLY"></span>**Reinstalar**
</dt> <dd>

Reinstalação parcial ou completa de um aplicativo instalado. Normalmente feito para reparar um aplicativo se algum arquivo ou entrada do Registro tiver se corrompido ou estiver ausente. Para obter mais informações, [consulte Reinstalando um recurso ou aplicativo](reinstalling-a-feature-or-application.md).

</dd> <dt>

<span id="_msi_resiliency_gly"></span><span id="_MSI_RESILIENCY_GLY"></span>**Resiliência**
</dt> <dd>

Capacidade do aplicativo de recuperação normalmente de erro sem gerar mensagens de erro. Para obter mais informações, consulte [Resiliência.](resiliency.md)

</dd> <dt>

<span id="_msi_rollback_gly"></span><span id="_MSI_ROLLBACK_GLY"></span>**Reversão**
</dt> <dd>

Restauração automática do estado original do computador caso a instalação falhe. Para obter mais informações, consulte [Instalação de re rollback (Windows Installer)](rollback-installation.md).

</dd> </dl>
