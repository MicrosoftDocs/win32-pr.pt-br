# <a name="installer-units"></a>Unidades do instalador

uma Windows Installer unidade de interface de usuário é aproximadamente igual a um décimo (1/12) da altura do tamanho de fonte MS Sans Serif de 10 pontos. O tamanho relativo de caixas de diálogo, controles e fontes pode ser alterado dependendo do tamanho da fonte usado na interface do usuário. Para garantir a exibição correta da interface do usuário, os desenvolvedores de instalação devem sempre testar seus aplicativos usando os tamanhos de fonte reais.
