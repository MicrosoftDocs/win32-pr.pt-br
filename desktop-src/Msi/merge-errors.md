# <a name="mergeerrors-property"></a>Propriedade Merge.Errors

A propriedade **Erros** somente leitura do objeto [**Merge**](merge-object.md) retorna uma coleção de objetos [**Error**](error-object.md) que é o conjunto atual de erros.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Merge.Errors
```



## <a name="property-value"></a>Valor da propriedade

## <a name="remarks"></a>Comentários

A recuperação não é destrutiva. Várias instâncias da coleção de erros podem ser recuperadas chamando repetidamente esse método.

### <a name="c"></a>C++

Consulte [**Obter \_ Função de**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-get_errors) Erros.
