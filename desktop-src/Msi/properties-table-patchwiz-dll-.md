# <a name="properties-table-patchwizdll"></a>Tabela de propriedades (Patchwiz.dll)

A Tabela de Propriedades contém configurações globais para o pacote de patch. A Tabela de Propriedades é necessária no banco de dados de criação de patch (arquivo .pcp) e é usada pela [função UiCreatePatchPackageEx.](uicreatepatchpackageex--patchwiz-dll-.md)

A Tabela de Propriedades tem as seguintes colunas.



| Coluna | Tipo | Chave | Nullable |
|--------|------|-----|----------|
| Name   | texto | Y   | N        |
| Valor  | texto |     | Y        |



 

## <a name="columns"></a>Colunas

<dl> <dt>

<span id="Name"></span><span id="name"></span><span id="NAME"></span>Nome
</dt> <dd>

O nome de uma propriedade de criação de patch.

</dd> <dt>

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Valor
</dt> <dd>

O valor da propriedade de criação de patch.

</dd> </dl>
