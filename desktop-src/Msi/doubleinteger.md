# <a name="doubleinteger"></a>DoubleInteger

O tipo de dados DoubleInteger é um valor inteiro de quatro bytes. A menos que seja restrito de outra forma, o intervalo de valores válidos é de – 2.147.483.647 a + 2.147.483.647.
