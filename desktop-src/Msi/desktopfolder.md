# <a name="desktopfolder-property"></a>Propriedade DesktopFolder

O instalador define a **propriedade DesktopFolder** como o caminho completo da pasta Área de Trabalho do usuário atual. Se existir um perfil "Todos os Usuários" e a [**propriedade ALLUSERS**](allusers.md) estiver definida, essa propriedade será definida como a pasta no perfil "Todos os Usuários".

Windows Instalador 4.0 ou Windows Instalador 4.5 no Windows Server 2008 ou Windows Vista. Windows Instalador no Windows Server 2003 ou Windows XP. Consulte o [Windows instalador Run-Time para](windows-installer-portal.md) obter informações sobre o Windows service pack mínimo exigido por uma versão Windows Installer.<br/> |
