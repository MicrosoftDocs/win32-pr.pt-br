# <a name="desktopfolder-property"></a>Propriedade DesktopFolder

O instalador define a propriedade **DesktopFolder** como o caminho completo da pasta da área de trabalho do usuário atual. Se um perfil "todos os usuários" existir e a propriedade [**AllUsers**](allusers.md) estiver definida, essa propriedade será definida como a pasta no perfil "todos os usuários".
