# <a name="user-interface-wizard-behavior"></a>Comportamento do assistente de interface do usuário

Para criar uma interface do usuário simples, os desenvolvedores de pacotes de instalação podem utilizar um design de interface do usuário que siga o comportamento do assistente de interface.

O termo comportamento do assistente de interface do usuário significa que cada caixa de diálogo em uma sequência contém um **próximo botão>>** , no qual o usuário clica para ir para a próxima caixa de diálogo depois de inserir dados ou configurar informações na caixa de diálogo atual. Se o usuário decidir voltar e alterar as informações inseridas em uma caixa de diálogo anterior, cada caixa de diálogo conterá um **<<botão anterior** em que o usuário clica para voltar. No final da sequência do assistente, o usuário clica em um botão **concluir** para iniciar o processo de instalação.

A interface do usuário descrita no exemplo da interface do usuário adere ao comportamento do assistente de interface do utilizador. Consulte [um exemplo de instalação](an-installation-example.md).
