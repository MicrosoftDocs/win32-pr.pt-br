# <a name="recordcleardata-method"></a>Método record. ClearData

O método **ClearData** do objeto [**Record**](record-object.md) limpa os dados em todos os campos, definindo-os como NULL. Todos os objetos armazenados nos campos são liberados.

## <a name="syntax"></a>Sintaxe


```JScript
Record.ClearData()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
