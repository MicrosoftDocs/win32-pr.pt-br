# <a name="intel-property"></a>Propriedade Intel

A **propriedade Intel** é definida pelo instalador Windows para o nível numérico do processador ao ser executado em um processador Intel. Os valores são os mesmos que o *campo wProcessorLevel* da [**estrutura SYSTEM \_ INFO.**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info) Ao executar em um processador x64, o instalador Windows define a propriedade **Intel** para refletir o processador x86 emulado pelo computador x64, conforme relatado pelo sistema operacional.
