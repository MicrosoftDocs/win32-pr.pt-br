# <a name="comments-summary-property"></a>Propriedade Resumo de Comentários

A **propriedade Resumo de** Comentários transmite a finalidade geral do pacote de instalação, transformação ou pacote de patch.

Um autor de um pacote de instalação, transformação ou pacote de patch deve definir o valor da propriedade **Resumo** de Comentários como um dos seguintes valores:

<dl> "Esse banco de dados do instalador contém a lógica e os dados necessários para instalar <*de>."*  
"Essa transformação contém a lógica e os dados necessários para instalar <*de>."*  
"Esse patch contém a lógica e os dados necessários para instalar <*de>."*  
</dl>

em que <*produto*> é o nome do produto.

## <a name="requirements"></a>Requisitos
