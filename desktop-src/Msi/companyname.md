# <a name="companyname-property"></a>Propriedade COMPANYNAME

A **propriedade COMPANYNAME** é a organização ou a empresa do usuário que está executando a instalação.

## <a name="default-value"></a>Valor padrão

Um valor padrão pode ser colocado na [tabela Propriedade](property-table.md). Se a **propriedade COMPANYNAME** não estiver definida, o instalador a definirá automaticamente usando valores do Registro.

## <a name="remarks"></a>Comentários

De definir [**a propriedade NOCOMPANYNAME**](nocompanyname.md) para suprimir a configuração automática de **COMPANYNAME.**
