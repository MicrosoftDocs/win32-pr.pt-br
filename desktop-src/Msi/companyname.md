# <a name="companyname-property"></a>Propriedade COMPANYNAME

A propriedade **CompanyName** é a organização ou a empresa do usuário que está executando a instalação.

## <a name="default-value"></a>Valor padrão

Um valor padrão pode ser colocado na [tabela de propriedades](property-table.md). Se a propriedade **CompanyName** não for definida, o instalador a definirá automaticamente usando valores do registro.

## <a name="remarks"></a>Comentários

Defina a propriedade [**NOcompanyname**](nocompanyname.md) para suprimir a configuração automática de **CompanyName**.
