# <a name="defaultdir"></a>DefaultDir

O tipo de dados DefaultDir é uma cadeia de caracteres de texto que contém um [nome de arquivo](filename.md) válido ou um [identificador](identifier.md)válido. Isso é usado somente na [tabela de diretórios](directory-table.md). Ele deve ser um identificador se o diretório for um diretório raiz. Se o diretório for não raiz, esse valor deverá ser um nome de arquivo ou um par filename: filename. Observe que "." é permitido como um nome de arquivo e tem um significado especial na tabela de diretórios.
