# <a name="ice-103"></a>ICE 103

ICE 103 verifica os eventos de controle [MsiPrint](msiprint-controlevent.md) e [MsiLaunchApp](msilaunchapp-controlevent.md) .

O ICE 103 verifica se o evento de controle [MsiPrint](msiprint-controlevent.md) é usado apenas em uma caixa de diálogo com um controle [ScrollableText](scrollabletext-control.md) .

**[Windows Installer 4,5 ou anterior](not-supported-in-windows-installer-4-5.md):** Sem suporte. Esta ICE está disponível a partir do Windows Installer 5,0.
