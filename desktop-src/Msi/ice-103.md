# <a name="ice-103"></a>ICE 103

O ICE 103 verifica os eventos de controle [MsiPrint](msiprint-controlevent.md) e [MsiLaunchApp.](msilaunchapp-controlevent.md)

ICE 103 verifica se o evento de controle [MsiPrint](msiprint-controlevent.md) é usado somente em uma caixa de diálogo com um [controle ScrollableText.](scrollabletext-control.md)

**[Windows Instalador 4.5 ou anterior:](not-supported-in-windows-installer-4-5.md)** Sem suporte. Este ICE está disponível a partir do Windows Installer 5.0.
