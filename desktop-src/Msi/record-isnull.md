# <a name="recordisnull-property"></a>Propriedade Record. IsNull

A propriedade **IsNull** do objeto [**Record**](record-object.md) é uma propriedade somente leitura que retornará true se o campo indicado for nulo e false se o campo contiver dados.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Record.IsNull
```



## <a name="property-value"></a>Valor da propriedade

Campo obrigatório número do valor dentro do registro, baseado em 1.
