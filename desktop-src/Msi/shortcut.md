# <a name="shortcut"></a>Atalho

O tipo de dados de atalho geralmente é usado na coluna de destino da [tabela de atalho](shortcut-table.md). Se ele contiver colchetes ( \[ \] ), o destino do atalho será avaliado como uma cadeia de caracteres formatada. Caso contrário, o atalho será avaliado como um [identificador](identifier.md) e deverá ser uma chave estrangeira válida na [tabela de recursos](feature-table.md).
