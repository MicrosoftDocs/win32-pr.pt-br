# <a name="create-timedate-summary-property"></a>Criar propriedade de Resumo de data/hora

A propriedade de **Resumo data/hora de criação** transmite a hora e a data em que um autor criou o pacote de instalação, a transformação ou o pacote de patch.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versão<br/> | Windows Installer 5,0 no Windows Server 2012, Windows 8, Windows Server 2008 R2 ou Windows 7. Windows Installer 4,0 ou Windows Installer 4,5 no Windows Server 2008 ou no Windows Vista. Windows Installer no Windows Server 2003 ou no Windows XP<br/> |
