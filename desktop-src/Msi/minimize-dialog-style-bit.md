# <a name="minimize-dialog-style-bit"></a>Minimizar o bit do estilo de caixa de diálogo

Se esse bit for definido, a caixa de diálogo poderá ser minimizada. Esse bit é ignorado para caixas de diálogo modais, que não podem ser minimizadas.

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                          |
|---------|-------------|-----------------------------------|
| 4       | 0x00000004  | **msidbDialogAttributesMinimize** |
