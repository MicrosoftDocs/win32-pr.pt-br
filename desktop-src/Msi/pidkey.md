# <a name="pidkey-property"></a>Propriedade PIDKEY

A propriedade **PIDKEY** contém a parte da ID do produto inserida pelo usuário. A [ação ValidateProductID](validateproductid-action.md) valida a ID do produto inserida comparando a propriedade **PIDKEY** com a propriedade [**PIDTemplate**](pidtemplate.md) .

A propriedade **PIDKEY** pode ser definida na linha de comando ou por meio da interface do usuário. Essa propriedade contém apenas a parte da ID do produto que é inserida pelo usuário.
