# <a name="ice33"></a>ICE33

O ICE33 processa entradas na [tabela do registro](registry-table.md) e pode emitir um aviso para cada entrada de tabela que registra classes, extensões de nome de arquivo, ProgIDs, verbos de Shell, APPIDs de servidor remoto, tipos MIME ou TypeLibs. Os avisos são obsoletos e podem ser ignorados.
