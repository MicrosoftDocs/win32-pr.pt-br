# <a name="ice33"></a>ICE33

O ICE33 processa [](registry-table.md) entradas na tabela do Registro e pode emitir um aviso para cada entrada de tabela que registra Classes, Extensões de Nome de Arquivo, ProgIDs, Verbos do Shell, AppIDs do Servidor Remoto, tipos MIME ou Typelibs. Os avisos são obsoletos e podem ser ignorados.
