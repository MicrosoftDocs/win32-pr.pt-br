# <a name="databaseenableuipreview-method"></a>Método Database. EnableUIPreview

O método **EnableUIPreview** do objeto de [**banco de dados**](database-object.md) facilita a criação de caixas de diálogo e de mural, fornecendo o suporte necessário para exibir caixas de diálogo de interface do usuário armazenadas no banco de dados do instalador. O método inicia o modo de visualização retornando um objeto de **banco de dados** de visualização. O modo de visualização termina quando o objeto de visualização é liberado.

## <a name="syntax"></a>Sintaxe


```JScript
Database.EnableUIPreview()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
