# <a name="ice-101"></a>ICE 101

Verifica se nenhum valor na coluna recurso da tabela de [recursos](feature-table.md) excede um comprimento máximo de 38 caracteres.

**[Windows Installer 4,5 ou anterior](not-supported-in-windows-installer-4-5.md):** Sem suporte. Esta ICE está disponível a partir do Windows Installer 5,0.
