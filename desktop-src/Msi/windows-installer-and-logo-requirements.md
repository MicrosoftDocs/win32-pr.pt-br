# <a name="windows-installer-and-logo-requirements"></a>Requisitos de Windows Installer e logotipo

O logotipo certificado para Microsoft Windows identifica os produtos que foram verificados por meio de testes independentes para atender à especificação do aplicativo para Windows. Essa especificação foi desenvolvida pela Microsoft em cooperação com clientes e outros desenvolvedores para fornecer um mapa de estrada para a criação de aplicativos confiáveis e gerenciáveis. Fornecedores de software que estão em conformidade com a especificação se qualificam para o logotipo certificado para Microsoft Windows e, em seguida, licenciam o logotipo para uso em seus pacotes de produtos, propaganda, material de apoio e outros materiais de marketing.

Para obter mais informações sobre a certificação do Windows Vista, do Windows XP e do Windows Server 2008 para seus aplicativos, consulte a [Microsoft Partner Network]( https://partner.microsoft.com).

Para obter mais informações sobre a certificação do Windows Server 2003 e do Windows 2000 para seus aplicativos, consulte [https://msdn.microsoft.com/certification](https://www.microsoft.com/windowsserver2003/partners/isvs/cfw.mspx) .

Se você estiver criando um pacote de instalação para seu aplicativo, poderá aproveitar os recursos do Microsoft Windows Installer para ajudar a atender a vários dos requisitos de certificação. A tabela a seguir fornece links para informações na documentação Windows Installer pertinentes para alguns dos requisitos de certificação. Observe que nem todos os requisitos de certificação discutidos na especificação do aplicativo dependem Windows Installer.
