# <a name="installerpatchinfo-property"></a>Propriedade Installer.PatchInfo

A propriedade **PatchInfo** somente leitura do objeto [**Installer**](installer-object.md) retorna informações sobre um patch.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Installer.PatchInfo
```



## <a name="property-value"></a>Valor da propriedade
