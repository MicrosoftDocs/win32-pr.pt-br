# <a name="programfiles64folder-property"></a>Propriedade ProgramFiles64Folder

O instalador define a propriedade **ProgramFiles64Folder** como o caminho completo da pasta predefinida de arquivos de programas de 64 bits. A propriedade [**ProgramFilesFolder**](programfilesfolder.md) existente é definida como a pasta correspondente de 32 bits.

## <a name="remarks"></a>Comentários

O instalador define essa propriedade. por exemplo, no Windows de 64 bits, o valor pode ser C: \\ arquivos de programas. Essa propriedade não é usada em Windows de 32 bits.
