# <a name="recordlistcount-property"></a>Propriedade recordlist. Count

A propriedade **Count** é uma propriedade somente leitura que retorna o número de itens no objeto [**recordlist**](recordlist-object.md) .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = RecordList.Count
```



## <a name="property-value"></a>Valor da propriedade

## <a name="remarks"></a>Comentários

O cliente deve verificar se o objeto [**recordlist**](recordlist-object.md) existe e não está vazio antes de fazer referência à propriedade Count.
