# <a name="dependencyversion-property"></a>Propriedade Dependency. Version

A propriedade **versão** somente leitura retorna a versão do módulo exigida pela mesclagem atual.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Dependency.Version
```



## <a name="property-value"></a>Valor da propriedade

## <a name="c"></a>C++

Confira a função [**obter \_ versão**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_version) .
