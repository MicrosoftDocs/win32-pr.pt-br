# <a name="version9x-property"></a>Propriedade Version9X

A **propriedade Version9X** fornece o número de versão para versões 9x Windows sistemas operacionais.

O valor dessa propriedade é um inteiro: MajorVersion \* 100 + MinorVersion. A propriedade permanecerá indefinida se o sistema operacional não for uma versão 9x. Para obter mais informações, consulte [Valores de propriedade do sistema operacional](operating-system-property-values.md).

## <a name="remarks"></a>Comentários

As expressões de condição podem testar a versão usando o nome da propriedade ou podem verificar a versão usando um operador de comparação.

Todos os nomes de propriedade são sensíveis a minúsculas. Observe que o X no nome **Version9X** está em letras maiúsculas.
