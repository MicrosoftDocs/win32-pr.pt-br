# <a name="publishproduct-action"></a>Ação PublishProduct

A ação PublishProduct gerencia o anúncio das informações do produto com o sistema. Essa ação publicará o produto se o produto estiver no modo de anúncio ou se algum recurso estiver sendo instalado ou reinstalado.

## <a name="sequence-restrictions"></a>Restrições de sequência

A ação PublishProduct deve vir após a ação [PublishFeatures](publishfeatures-action.md) .

## <a name="actiondata-messages"></a>Mensagens ActionData



| Campo | Descrição dos dados da ação        |
|-------|-----------------------------------|
| \[1\] | Identificador do produto anunciado. |
