# <a name="msiexecexe-and-instmsiexe-error-messages-for-developers"></a>MsiExec.exe e InstMsi.exe mensagens de erro (para desenvolvedores)

> [!NOTE]  
> este artigo destina-se a desenvolvedores de software que estão usando Windows Installer para criar pacotes do instalador para seus aplicativos. Se você for um usuário com dificuldade em seu computador durante ou após a instalação ou desinstalação de um aplicativo, entre em contato com o suporte ao cliente para o software que você está tentando instalar ou remover. Se você sentir que precisa de suporte para um produto da Microsoft, acesse nosso site de suporte técnico em [support.Microsoft.com](https://support.microsoft.com).

esses códigos de erro são retornados pelas funções Windows Installer **MsiExec.exe** e **InstMsi.exe**. Observe que qualquer erro em Winerror. h (como \_ dados inválidos de erro \_ , incluído aqui) também pode ser retornado. para obter mais códigos de erro retornados pelo Windows Installer, consulte [Windows Installer mensagens de erro](windows-installer-error-messages.md).
 
> O erro de erros de erro \_ , \_ a \_ reinicialização bem-sucedida do erro \_ foi iniciada e a \_ \_ reinicialização bem-sucedida do erro \_ necessária é um indício de sucesso. Se \_ \_ a reinicialização de êxito \_ de erro necessária for retornada, a instalação foi concluída com êxito, mas uma reinicialização é necessária para concluir a operação de instalação.
