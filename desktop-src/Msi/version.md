# <a name="version-windows-installer"></a>Versão (Windows Installer)

O tipo de dados Version é uma cadeia de texto que contém uma cadeia de caracteres de versão válida. Uma cadeia de caracteres de versão tem o formato

``` syntax
xxxxx.xxxxx.xxxxx.xxxxx
```

em que *x* é um dígito.

A cadeia de caracteres de versão máxima aceitável é 65535.65535.65535.65535.

Veja a seguir exemplos de cadeias de caracteres de versão válidas:

-   1
-   1,0
-   1.00
-   10,00
-   1.00.1
-   1.0.1
-   1.00.10
-   1.00.100
-   1.0.1000.0
