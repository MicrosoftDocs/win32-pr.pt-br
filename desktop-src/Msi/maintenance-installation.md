# <a name="maintenance-installation"></a>Instalação de manutenção

Depois que um aplicativo tiver sido instalado com êxito, talvez o usuário queira retornar à instalação para adicionar ou remover recursos. Isso é conhecido como uma instalação de manutenção.

As [transformações](transforms.md) que foram aplicadas a uma instalação devem estar disponíveis durante as instalações de manutenção subsequentes.
