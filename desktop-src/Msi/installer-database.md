# <a name="installer-database"></a>Banco de dados do instalador

Esse material é basicamente para desenvolvedores que desejam escrever ferramentas que criam novos pacotes do instalador e que precisam de informações detalhadas sobre o banco de dados relacional do instalador. Os pacotes completos do instalador são adotados por desenvolvedores de aplicativos como um meio eficiente de gerenciar a instalação de seus aplicativos. O conhecimento do banco de dados de instalação também é útil para desenvolvedores e administradores de aplicativos que desejam modificar os pacotes do instalador e os processos de instalação existentes usando ações personalizadas. Para obter informações gerais sobre Windows Installer e tecnologia de gerenciamento de componentes, consulte [About Windows Installer](about-windows-installer.md).

O banco de dados do instalador contém todas as informações necessárias para a instalação de um aplicativo. Portanto, os desenvolvedores de ferramentas para a criação de pacotes de instalação e desenvolvedores que criam pacotes de instalação sem usar essas ferramentas precisam entender o banco de dados do instalador. As principais áreas de funcionalidade do banco de dados do instalador são apresentadas nos seguintes tópicos:

[Sobre o banco de dados do instalador](about-the-installer-database.md)

[Usando o banco de dados do instalador](using-the-installer-database.md)

[Referência do banco de dados do instalador](installer-database-reference.md)
