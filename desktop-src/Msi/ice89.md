# <a name="ice89"></a>ICE89

ICE89 valida que o valor na coluna Progid Parent na tabela ProgId é uma chave estrangeira válida na coluna ProgId na tabela \_ ProgId. [](progid-table.md) Cada progId pai deve ter um registro na tabela ProgId.

## <a name="result"></a>Resultado

ICE89 posta os seguintes erros.



| Erro ICE89                                                           | Descrição                                                                |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------|
| O ProgId \_ Parent ' \[ 1 ' na tabela \] ProgId não é um ProgId válido. | Há um pai ProgId especificado que não está listado na tabela ProgId. |
