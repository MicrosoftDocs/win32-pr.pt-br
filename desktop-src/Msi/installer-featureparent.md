# <a name="installerfeatureparent-property"></a>Propriedade Installer. FeatureParent

A propriedade **FeatureParent** especifica o recurso pai de um recurso. Uma cadeia de caracteres vazia para essa propriedade indica que o recurso não tem pai e é um item raiz.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Installer.FeatureParent
```



## <a name="property-value"></a>Valor da propriedade
