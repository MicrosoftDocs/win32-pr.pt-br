# <a name="sequence-property"></a>Propriedade SEQUENCE

A propriedade **Sequence** especifica uma tabela que tem o mesmo esquema que a tabela InstallExecuteSequence, que é uma ação, condição e colunas de sequência. Essa propriedade é usada pela [ação de sequência](sequence-action.md).

## <a name="remarks"></a>Comentários

A ação de sequência executa as ações na tabela na ordem especificada pela coluna sequência da tabela.
