# <a name="msiwin32assemblysupport-property"></a>Propriedade MsiWin32AssemblySupport

A propriedade **MsiWin32AssemblySupport** indica se o computador oferece suporte a assemblies do Win32. Em sistemas que dão suporte a assemblies Win32, o instalador define o valor de **MsiWin32AssemblySupport** para a versão de arquivo do sxs.dll. O instalador não definirá essa propriedade se o sistema operacional não oferecer suporte a assemblies Win32. Para obter mais informações, consulte [assemblies](assemblies.md).
