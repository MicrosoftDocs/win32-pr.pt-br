# <a name="sessionproductproperty-property"></a>Propriedade Session. Productproperty

A propriedade **productproperty** do objeto [**Session**](session-object.md) é uma propriedade somente leitura que recupera as propriedades do produto do banco de dados do produto.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Session.ProductProperty
```



## <a name="property-value"></a>Valor da propriedade

Nome necessário diferenciar maiúsculas de minúsculas de uma propriedade.
