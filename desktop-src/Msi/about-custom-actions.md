# <a name="about-custom-actions"></a>Sobre ações personalizadas

Embora as ações padrão sejam suficientes para executar uma instalação na maioria dos casos, as ações personalizadas permitem que o autor de um pacote de instalação estenda os recursos das ações padrão, incluindo executáveis, bibliotecas de vínculo dinâmico e script. Ações personalizadas não podem ser usadas nas tabelas de sequência usadas para anúncio, na tabela [AdvtUISequence](advtuisequence-table.md) e na [tabela AdvtExecuteSequence](advtexecutesequence-table.md).
