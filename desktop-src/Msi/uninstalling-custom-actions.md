# <a name="uninstalling-custom-actions"></a>Desinstalando ações personalizadas

Os desenvolvedores de pacotes de instalação que fazem uso de ações personalizadas devem determinar em cada caso como condiçãor que a ação personalizada funcione corretamente quando o pacote for desinstalado. Cabe ao autor do pacote decidir se a ação personalizada deve ser executada na desinstalação, não executar na desinstalação ou executar uma ação personalizada alternativa na desinstalação.
