# <a name="productid-property"></a>Propriedade ProductID

A propriedade **ProductID** é definida como o **ProductID** completo após uma validação bem-sucedida.

Esse valor é exibido pela interface do usuário e usado pela [ação RegisterProduct](registerproduct-action.md).

Após uma validação bem-sucedida, essa propriedade é definida pela [ação ValidateProductID](validateproductid-action.md) ou [ValidateProductID ControlEvent,](validateproductid-controlevent.md).

Observe que a interface do usuário de exemplo específica fornecida com o SDK como Uisample.msi requer um valor para **ProductID**. Se você usar essa interface do usuário, mas não quiser usar **ProductID** para validar identificadores de produtos, defina a propriedade **ProductID** como "None" na [tabela de propriedades](property-table.md).
