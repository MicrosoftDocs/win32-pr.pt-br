# <a name="ice-105"></a>ICE 105

Valida o pacote em relação a um conjunto de requisitos para instalação no contexto por usuário. A ICE 105 também pode validar pacotes de uso duplo. Um pacote de uso duplo permite que um usuário selecione se deseja instalar o aplicativo no contexto por usuário ou no contexto por máquina. Para obter informações sobre como desenvolver um pacote de uso duplo, consulte [criação de pacote único](single-package-authoring.md).

O ICE 105 executa a seguinte validação do pacote.

-   Verifica se a tabela [CustomAction](customaction-table.md) não contém ações personalizadas que foram marcadas para execução com privilégios elevados. Para obter mais informações sobre ações personalizadas elevadas, consulte [segurança de ação personalizada](custom-action-security.md).
-   Verifica se a tabela de [diretório](directory-table.md) não inclui nenhuma das propriedades de pasta do sistema a seguir.
    -   [**AdminToolsFolder**](admintoolsfolder.md)
    -   [**CommonAppDataFolder**](commonappdatafolder.md)
    -   [**FontsFolder**](fontsfolder.md)
    -   [**System16Folder**](system16folder.md)
    -   [**System64Folder**](system64folder.md)
    -   [**SystemFolder**](systemfolder.md)
    -   [**TempFolder**](tempfolder.md)
    -   [**WindowsFolder**](windowsfolder.md)
    -   [**WindowsVolume**](windowsvolume.md)
-   Verifica se o pacote não instala um Common Language Runtime assembly para o cache de assembly global (GAC). Para obter mais informações sobre como instalar assemblies no cache de assembly global, consulte [adicionando assemblies a um pacote](adding-assemblies-to-a-package.md) e [instalação de assemblies do Common Language Runtime](installation-of-common-language-runtime-assemblies.md).
-   Verifica a tabela [ODBCDataSource](odbcdatasource-table.md) para verificar se o pacote não instala nenhuma fonte de dados.
-   Verifica a tabela de [instalação](serviceinstall-table.md) para verificar se o pacote não instala nenhum serviço.
-   Verifica se a tabela [do registro](registry-table.md) não grava entradas na chave do **\_ \_ computador local hKey** .

**[Windows Installer 4,5 ou anterior](not-supported-in-windows-installer-4-5.md):** Sem suporte. Esta ICE está disponível a partir do Windows Installer 5,0.
