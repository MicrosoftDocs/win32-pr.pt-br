# <a name="configurableitemdefaultvalue-property"></a>Propriedade ConfigurableItem. DefaultValue

A propriedade **DefaultValue** do objeto [**ConfigurableItem**](configurableitem-object.md) retorna o valor da coluna DefaultValue da [Tabela ModuleConfiguration](moduleconfiguration-table.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = ConfigurableItem.DefaultValue
```
