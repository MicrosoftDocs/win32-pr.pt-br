# <a name="configurableitemhelpkeyword-property"></a>Propriedade ConfigurableItem. HelpKeyword

A propriedade **HelpKeyword** retorna o valor da coluna HelpKeyword da [Tabela ModuleConfiguration](moduleconfiguration-table.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## <a name="property-value"></a>Valor da propriedade

## <a name="c"></a>C++

Consulte [**obter \_**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helpkeyword) função de HelpKeyword.
