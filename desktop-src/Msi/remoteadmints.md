# <a name="remoteadmints-property"></a>Propriedade RemoteAdminTS

O instalador define a propriedade **RemoteAdminTS** quando o sistema é configurado para administração remota usando os serviços de terminal do Windows. Observe que, nesse caso, o instalador não define a propriedade [**TerminalServer**](terminalserver.md) .

## <a name="default-value"></a>Valor padrão

Nenhum. A propriedade é indefinida quando não está sendo executada em um servidor de administração remota com serviços de terminal.
