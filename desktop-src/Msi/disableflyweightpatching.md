# <a name="disableflyweightpatching"></a>DisableFlyWeightPatching

Se essa política do sistema por máquina for definida como 1 (uma), todas as opções de [otimização de patch](patch-optimization.md) serão desativadas durante a instalação.

As opções de [otimização de patch](patch-optimization.md) e a política DisableFlyWeightPatching estão disponíveis a partir do Windows Installer versão 3,0.

## <a name="registry-key"></a>Chave do Registro

**HKEY \_ \_** Políticas de \\ **software** do computador local \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo de Dados

**REG \_ DWORD**
