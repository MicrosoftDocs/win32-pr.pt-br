# <a name="nousername-property"></a>Propriedade NOUSERNAME

Defina a propriedade **NOusername** como 1 para suprimir o instalador da configuração da propriedade [**username**](username.md) . Essa propriedade é usada por aplicativos que precisam coletar o nome de usuário na primeira execução.

## <a name="default-value"></a>Valor padrão

Não definido. O instalador define a propriedade [**username**](username.md) automaticamente usando valores do registro.
