# <a name="pushbutton-control"></a>Controle de supressão

O controle de supressão exibe um botão de ação básico, também conhecido como um botão de comando.

Os controles de pressão não podem ser usados para publicar um [IgnoreChange ControlEvent,](ignorechange-controlevent.md), [SelectionDescription ControlEvent,](selectiondescription-controlevent.md), [SelectionSize ControlEvent,](selectionsize-controlevent.md), [SelectionPath ControlEvent,](selectionpath-controlevent.md), [SelectionPathOn ControlEvent,](selectionpathon-controlevent.md), [selectionaction ControlEvent,](selectionaction-controlevent.md), [SelectionNoItems ControlEvent,](selectionnoitems-controlevent.md), [ActionText ControlEvent,](actiontext-controlevent.md), [ActionData ControlEvent,](actiondata-controlevent.md), [setProgress ControlEvent,](setprogress-controlevent.md)ou [timecontinueing ControlEvent,](timeremaining-controlevent.md).

Você pode usar os atributos a seguir com este controle. Para alterar o valor de um atributo usando um evento, assine o controle para um ControlEvent, na [tabela EventMappings](eventmapping-table.md) e liste o identificador do atributo na coluna atributo. Insira o identificador do ControlEvent, na coluna de evento.

## <a name="remarks"></a>Comentários

Esse controle pode ser criado a partir da classe BUTTON usando a função [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Ele tem os estilos de **\_ grupo** de **\_ linhas BS**, **WS \_ Child**, **WS \_ TabStop** e WS.
