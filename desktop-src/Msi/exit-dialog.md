# <a name="exit-dialog"></a>Sair da caixa de diálogo

A caixa de diálogo sair é exibida no final de uma instalação bem-sucedida. Essa caixa de diálogo normalmente contém apenas algum texto e um botão **OK** . O nome dessa caixa de diálogo deve aparecer na [tabela AdminUISequence](adminuisequence-table.md) e na [tabela InstallUISequence](installuisequence-table.md) com-1 como o número de sequência. Essa é uma caixa de diálogo modal.
