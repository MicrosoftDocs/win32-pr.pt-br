# <a name="creating-application-summary-property"></a>Criando a propriedade de resumo do aplicativo

A propriedade de **Resumo do aplicativo de criação** transmite qual aplicativo criou o banco de dados do instalador. Em geral, o valor dessa propriedade de resumo é o nome do software usado para criar esse banco de dados.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versão<br/> | Windows Installer 5,0 no Windows Server 2012, Windows 8, Windows Server 2008 R2 ou Windows 7. Windows Installer 4,0 ou Windows Installer 4,5 no Windows Server 2008 ou no Windows Vista. Windows Installer no Windows Server 2003 ou no Windows XP<br/> |
