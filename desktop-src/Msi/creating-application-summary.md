# <a name="creating-application-summary-property"></a>Criando a propriedade Resumo do Aplicativo

A **propriedade Criando Resumo do Aplicativo** transmite qual aplicativo criou o banco de dados do instalador. Em geral, o valor dessa propriedade de resumo é o nome do software que é usado para a autor desse banco de dados.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versão<br/> | Windows Instalador 5.0 no Windows Server 2012, Windows 8, Windows Server 2008 R2 ou Windows 7. Windows Instalador 4.0 ou Windows Instalador 4.5 no Windows Server 2008 ou Windows Vista. Windows Instalador no Windows Server 2003 ou Windows XP<br/> |
