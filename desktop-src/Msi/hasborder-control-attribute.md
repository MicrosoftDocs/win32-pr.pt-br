# <a name="hasborder-control-attribute"></a>Atributo de controle HasBorder

Se esse bit estiver definido, [o RadioButtonGroup](radiobuttongroup-control.md) terá texto e uma borda exibida em torno dele.

Se o bit de estilo não estiver definido, a borda não será exibida e nenhum texto será exibido no grupo.

## <a name="valid-controls"></a>Controles válidos

[RadioButtonGroup](radiobuttongroup-control.md)

## <a name="valid"></a>Válido



| Decimal  | Hexadecimal | Constante                            |
|----------|-------------|-------------------------------------|
| 16777216 | 0x01000000  | **msidbControlAttributesHasBorder** |



 

## <a name="remarks"></a>Comentários

Para definir esse atributo em um controle, inclua os bits HasBorder na coluna Atributos do registro do controle na [tabela Control](control-table.md).

Consulte [Atributos de controle](control-attributes.md) e o controle que você precisa criar em [Controles](controls.md).
