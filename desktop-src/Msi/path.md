# <a name="path"></a>Caminho

O tipo de dados Path é uma cadeia de caracteres de texto que contém um caminho completo válido.

A cadeia de caracteres também pode conter um nome de propriedade entre colchetes \[ \] . Nesse caso, o nome da propriedade, incluindo os colchetes, é substituído na cadeia de caracteres pelo valor da propriedade .

Exemplos:

-   Caminho UNC: \\ \\ compartilhamento de \\ servidor
-   Unidade local: c: \\ temp
-   Com um nome de propriedade: \[ TEMP \] \\ DRIVE
