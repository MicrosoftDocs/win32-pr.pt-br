# <a name="formatted-text-custom-actions"></a>Ações personalizadas de texto formatado

Esse tipo de ação personalizada define uma propriedade ou um diretório de instalação a partir de uma cadeia de caracteres de texto formatada. Os parâmetros nessa cadeia de caracteres podem ser Propriedades, variáveis de ambiente, caminhos de arquivo ou caminhos de diretório de componentes. Isso é útil para passar informações para ações personalizadas subsequentes.

Os seguintes tipos básicos de ação personalizada definem uma propriedade ou um diretório. Para obter mais informações, consulte o tipo de ação personalizada específico.



| Tipo de ação personalizada                                 | Descrição                                    |
|----------------------------------------------------|------------------------------------------------|
| [Tipo de ação personalizada 51](custom-action-type-51.md) | Define uma propriedade com uma cadeia de caracteres de texto formatada.  |
| [Tipo de ação personalizada 35](custom-action-type-35.md) | Define um diretório com uma cadeia de caracteres de texto formatada. |



 

Consulte a [lista Resumo da seção de todos os tipos de ação personalizada](summary-list-of-all-custom-action-types.md) para obter um resumo de todos os tipos de ações personalizadas e como eles são codificados na [tabela CustomAction](customaction-table.md).
