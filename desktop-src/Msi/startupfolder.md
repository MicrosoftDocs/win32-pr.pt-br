# <a name="startupfolder-property"></a>Propriedade StartupFolder

O instalador define a propriedade **startupfolder** como o caminho completo da pasta de inicialização. Por padrão, essa propriedade é definida como a pasta para o usuário atual. Se um perfil "todos os usuários" existir e a propriedade [**AllUsers**](allusers.md) estiver definida, essa propriedade será definida como a pasta no perfil "todos os usuários".
