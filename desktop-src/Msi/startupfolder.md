# <a name="startupfolder-property"></a>Propriedade StartupFolder

O instalador define a **propriedade StartupFolder** como o caminho completo da pasta Inicialização. Por padrão, essa propriedade é definida como a pasta do usuário atual. Se existir um perfil "Todos os Usuários" e a [**propriedade ALLUSERS**](allusers.md) estiver definida, essa propriedade será definida como a pasta no perfil "Todos os Usuários".
