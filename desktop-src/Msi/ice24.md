# <a name="ice24"></a>ICE24

ICE24 valida as seguintes propriedades na [tabela Property](property-table.md):

-   Que a [**propriedade ProductCode**](productcode.md) é um tipo de [dados GUID](guid.md) válido.
-   Que a [**propriedade ProductVersion**](productversion.md) é uma versão válida do produto.
-   Que a [**propriedade ProductLanguage**](productlanguage.md) é um tipo de dados [de Linguagem](language.md) válido.

## <a name="result"></a>Resultado

ICE24 postará uma mensagem de erro se qualquer uma dessas propriedades não estiver na forma de um tipo de dados válido.
