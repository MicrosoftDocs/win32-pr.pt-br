# <a name="user-interface-windows-installer"></a>Interface do usuário (Windows Installer)

Windows Installer fornece uma interface do usuário (IU) completa para a instalação de um aplicativo ou produto. A interface do usuário apresenta ao usuário as opções disponíveis para configurar a instalação e obtém informações do usuário sobre o processo de instalação pendente.

[Sobre a interface do usuário](about-the-user-interface.md) descreve a funcionalidade da interface do usuário do instalador. [Usar a interface do usuário](using-the-user-interface.md) descreve o uso da interface de usuário interna do instalador.

As informações de referência sobre a caixa de diálogo interna do instalador e os estilos e as opções de controle são apresentadas na [referência da interface do usuário](user-interface-reference.md).
