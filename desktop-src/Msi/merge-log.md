# <a name="mergelog-method"></a>Método Merge. log

O método **log** do objeto de [**mesclagem**](merge-object.md) grava uma cadeia de texto no arquivo de log aberto no momento.

## <a name="syntax"></a>Sintaxe


```JScript
Merge.Log(
  Message
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Message* 
</dt> <dd>

A cadeia de texto a ser gravada no arquivo de log.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.

## <a name="c"></a>C++

Consulte a função [**log**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-log) .
