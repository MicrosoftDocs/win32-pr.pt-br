# <a name="mergelog-method"></a>Método Merge.Log

O **método Log** do objeto Merge [**grava**](merge-object.md) uma cadeia de caracteres de texto no arquivo de log aberto no momento.

## <a name="syntax"></a>Sintaxe


```JScript
Merge.Log(
  Message
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Message* 
</dt> <dd>

A cadeia de caracteres de texto a ser escrita no arquivo de log.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
