# <a name="rtlro-dialog-style-bit"></a>Bit de estilo de caixa de diálogo RTLRO

Se esse bit de estilo for definido, o texto na caixa de diálogo será exibido na ordem de leitura da direita para a esquerda.

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **msidbDialogAttributesRTLRO** |
