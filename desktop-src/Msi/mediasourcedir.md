# <a name="mediasourcedir-property"></a>Propriedade MediaSourceDir

O instalador define a propriedade **MediaSourceDir** como 1 quando a instalação usa uma origem localizada em mídia, como um CD-ROM. Essa propriedade não é definida quando a instalação usa uma origem localizada em um local de rede. Por exemplo, o [controle SelectionTree](selectiontree-control.md) usa **MediaSourceDir** para determinar se a instalação está sendo executada da origem ou executada a partir de um local de rede.
