# <a name="restrictions-on-property-names"></a>Restrições em nomes de propriedade

Todos os nomes de propriedade devem seguir essas restrições.

-   Um nome de propriedade é [um Identificador](identifier.md), que é uma cadeia de caracteres de texto que deve começar com uma letra ou um sublinhado. Identificadores e nomes de propriedade podem conter letras, numerais, sublinhados ou pontos; mas não pode começar com um numeral ou ponto.
-   [Os nomes de](public-properties.md) propriedade pública não podem conter letras minúsculas.
-   [Os nomes de](private-properties.md) propriedade privada devem conter algumas letras minúsculas.
-   Os nomes de propriedade prefixados com % representam variáveis de ambiente do sistema e do usuário. Eles nunca são inseridos na [tabela Propriedade](property-table.md). As configurações permanentes de variáveis de ambiente só podem ser modificadas usando a [Tabela de Ambiente](environment-table.md).
