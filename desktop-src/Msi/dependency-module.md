# <a name="dependencymodule-property"></a>Propriedade Dependency. Module

A propriedade de **módulo** somente leitura do [**objeto de dependência**](dependency-object.md) retorna a ModuleID do módulo exigido pela cadeia de caracteres atual na forma de um **BSTR**. A ModuleID é a mesma forma usada na [tabela ModuleSignature](modulesignature-table.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Dependency.Module
```



## <a name="property-value"></a>Valor da propriedade

## <a name="c"></a>C++

Consulte o método do [**\_ \_ módulo obter IMsmDependency**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_module) .
