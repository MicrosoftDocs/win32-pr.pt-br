# <a name="listbox-control"></a>Controle ListBox

O controle ListBox é uma caixa de listagem comum que permite ao usuário fazer uma única seleção de uma lista de valores predeterminados. Os valores possíveis são lidos na [tabela ListBox](listbox-table.md). Você pode associar uma cadeia de caracteres ou propriedade de inteiro inserindo o nome da propriedade na coluna propriedade da [tabela de controle](control-table.md).

## <a name="control-attributes"></a>Atributos de controle

Você pode usar os atributos a seguir com este controle. Para alterar o valor de um atributo usando um evento, assine o controle para um ControlEvent, na [tabela EventMappings](eventmapping-table.md) e liste o identificador do atributo na coluna atributo. Insira o identificador do ControlEvent, na coluna de evento.

## <a name="remarks"></a>Comentários

Esse controle pode ser criado a partir da classe LISTBOX usando a função [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Ele tem os estilos **\_ filho** WS **\_ TabStop**, **WS \_ Group** e WS. Se o bit do estilo de controle classificado estiver ativado, o controle será criado com os estilos de VSCROLL de **\_ notificação lbs**, **WS- \_** e **WS \_** . caso contrário, o controle será criado com o estilo **\_ padrão de lbs** .
