# <a name="windows-sdk-components-for-windows-installer-developers"></a>SDK do Windows componentes para desenvolvedores de Windows Installer

Os componentes do Windows Installer Software Development Kit estão incluídos no SDK (Software Development Kit) do Microsoft Windows. A SDK do Windows inclui componentes, documentação, ferramenta de validação de banco de dados do instalador, editor de tabela de banco de dados, esquema de banco de dados, ferramentas de desenvolvimento, ferramentas de Visual Basic Scripting Edition (VBScript), produtos de exemplo e exemplos de código.

Para obter a página de download do SDK do Windows, consulte [Microsoft Windows Software Development Kit (SDK)](https://developer.microsoft.com/windows/downloads/windows-10-sdk).
