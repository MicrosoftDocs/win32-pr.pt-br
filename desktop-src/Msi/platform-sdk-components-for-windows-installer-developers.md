# <a name="windows-sdk-components-for-windows-installer-developers"></a>Windows componentes do SDK para desenvolvedores de Windows Installer

os componentes do Windows Installer software development kit estão incluídos no Microsoft Windows software development kit (SDK). a SDK do Windows inclui componentes, documentação, ferramenta de validação de banco de dados do instalador, editor de tabela de banco de dados, esquema de banco de dados, ferramentas de desenvolvimento, ferramentas do Visual Basic scripting Edition (VBScript), exemplos de produtos e exemplos de código do redistribuível.

para obter a página de download do SDK do Windows, consulte [Microsoft Windows Software Development Kit (SDK)](https://developer.microsoft.com/windows/downloads/windows-10-sdk).
