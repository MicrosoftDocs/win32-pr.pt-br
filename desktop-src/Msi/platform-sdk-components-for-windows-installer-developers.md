# <a name="windows-sdk-components-for-windows-installer-developers"></a>Windows Componentes do SDK para desenvolvedores Windows instalador

Os componentes do Windows Software Development Kit do Instalador estão incluídos no SDK (Software Development Kit) do Microsoft Windows. O SDK do Windows inclui componentes redistribuíveis, documentação, ferramenta de validação de banco de dados do instalador, editor de tabela de banco de dados, esquema de banco de dados, ferramentas de desenvolvimento, ferramentas de script do Visual Basic (VBScript), exemplos de produto e exemplos de código.

Para a página de download do SDK do Windows, consulte [Microsoft Windows SDK (Software Development Kit).](https://developer.microsoft.com/windows/downloads/windows-10-sdk)
