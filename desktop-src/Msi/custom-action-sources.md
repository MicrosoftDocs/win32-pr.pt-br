# <a name="custom-action-sources"></a>Origens de ação personalizadas

Uma fonte de ação personalizada pode existir em várias formas diferentes. A tabela a seguir lista os tipos de ações personalizadas que podem ter cada forma de origem. Consulte o tópico para obter o tipo de ação personalizada básica para obter mais informações.



| Origem da ação personalizada                                           | Tipos de ação personalizada associados                                                                                                                                                                                 |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ação personalizada armazenada na tabela binária.                      | [Tipo de ação personalizada 1](custom-action-type-1.md), [tipo de ação personalizada 2](custom-action-type-2.md), [tipo de ação personalizada 5](custom-action-type-5.md), [tipo de ação personalizada 6](custom-action-type-6.md)         |
| Ação personalizada copiada durante a instalação.                      | [Tipo de ação personalizada 17](custom-action-type-17.md), [tipo de ação personalizada 18](custom-action-type-18.md), [tipo de ação personalizada 21](custom-action-type-21.md), [tipo de ação personalizada 22](custom-action-type-22.md) |
| Ação personalizada fazendo referência a um diretório.                         | [Tipo de ação personalizada 34](custom-action-type-34.md)                                                                                                                                                             |
| Ação personalizada que faz referência a uma propriedade.                          | [Tipo de ação personalizada 50](custom-action-type-50.md), [tipo de ação personalizada 53](custom-action-type-53.md), [tipo de ação personalizada 54](custom-action-type-54.md)                                                     |
| Ação personalizada armazenada como código de script literal na tabela de banco de dados. | [Tipo de ação personalizada 37](custom-action-type-37.md), [tipo de ação personalizada 38](custom-action-type-38.md)                                                                                                         |
| Ação personalizada exibindo uma mensagem de erro.                     | [Tipo de ação personalizada 19](custom-action-type-19.md)                                                                                                                                                             |



 

Confira a [lista resumida de todos os tipos de ação personalizada](summary-list-of-all-custom-action-types.md) para obter um resumo de todos os tipos de ações personalizadas e como as fontes são especificadas na [tabela CustomAction](customaction-table.md).
