# <a name="setting-public-property-values-on-the-command-line"></a>Definindo valores de propriedade pública na linha de comando

Para definir uma propriedade pública como um valor de cadeia de caracteres literal, inclua a cadeia de caracteres entre aspas.

PROPERTY = "String"

As aspas só serão necessárias se a cadeia de caracteres contiver espaços. Para limpar uma propriedade pública em uma linha de comando (defini-la como NULL), defina o valor da propriedade como uma cadeia de caracteres vazia. Nesse caso, são necessárias aspas.

PROPRIEDADE = "".
