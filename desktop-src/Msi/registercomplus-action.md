# <a name="registercomplus-action"></a>Ação RegisterComPlus

A ação RegisterComPlus registra aplicativos COM+.

## <a name="sequence-restrictions"></a>Restrições de sequência

A ação RegisterComPlus deve seguir a [ação InstallFiles](installfiles-action.md) e [a ação UnregisterComPlus](unregistercomplus-action.md).

## <a name="actiondata-messages"></a>Mensagens ActionData



| Campo | Descrição dos dados de ação              |
|-------|-----------------------------------------|
| \[1\] | ID do aplicativo COM+. |
