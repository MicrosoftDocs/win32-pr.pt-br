# <a name="table-names"></a>Nomes de tabela

Os nomes de tabela são limitados a 31 caracteres. Se um nome de tabela contiver um ponto (uma prática que é permitida, mas não recomendado), o nome deverá ser colocado entre acento grave \` (valor ASCII 96), por exemplo, \` filename. ext \` . Essa sintaxe de escape impede conflitos entre o nome da tabela e as palavras reservadas, como palavras-chave SQL. Para obter mais informações, consulte [sintaxe SQL](sql-syntax.md).
