# <a name="windows-installer-features"></a>Recursos de Windows Installer

Um recurso faz parte da funcionalidade total do aplicativo que um usuário reconhece e pode decidir instalar de forma independente. Por exemplo, um recurso pode ser um verificador ortográfico, um dicionário de sinônimos ou um conjunto de clip-arts. As relações hierárquicas de recursos pai e filho normalmente existem, de modo que, se um recurso filho estiver instalado, o recurso pai também será instalado automaticamente.
