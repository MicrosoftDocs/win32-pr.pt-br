# <a name="initialization-of-property-values"></a>Inicialização de valores de propriedade

É recomendável que todas as propriedades a serem usadas pela instalação sejam inseridas na [tabela de propriedades](property-table.md) com um valor inicial. O instalador define as propriedades para esses valores no início da instalação. As propriedades para as quais um espaço em branco é um valor aceitável e as propriedades internas do instalador não precisam ser inicializadas.
