# <a name="username-property"></a>Propriedade USERNAME

A propriedade **username** é um usuário executando a instalação.

## <a name="default-value"></a>Valor padrão

O valor padrão pode ser colocado na tabela de propriedades. Se a propriedade **username** não for definida, ela será definida automaticamente usando valores do registro.

## <a name="remarks"></a>Comentários

Defina a propriedade [**NOusername**](nousername.md) para suprimir a configuração automática de **nome de usuário**.
