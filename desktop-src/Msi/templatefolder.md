# <a name="templatefolder-property"></a>Propriedade TemplateFolder

O instalador define a propriedade **TemplateFolder** como o caminho completo para a pasta de modelo do usuário atual.

somente no Windows 2000, se a propriedade [**ALLUSERS**](allusers.md) estiver definida, essa propriedade apontará para um diretório do sistema de arquivos que contém os modelos disponíveis para todos os usuários.
