# <a name="templatefolder-property"></a>Propriedade TemplateFolder

O instalador define a propriedade **TemplateFolder** como o caminho completo para a pasta de modelo do usuário atual.

Se a propriedade [**AllUsers**](allusers.md) for definida, essa propriedade apontará para um diretório do sistema de arquivos que contém os modelos disponíveis para todos os usuários.
