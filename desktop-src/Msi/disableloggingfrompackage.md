# <a name="disableloggingfrompackage"></a>DisableLoggingFromPackage

O valor dessa política [](system-policy.md) de sistema por computador é definido como "1" para desabilitar o log especificado para o pacote pela propriedade [**MsiLogging**](msilogging.md) para todos os usuários do computador.

A [**propriedade MsiLogging**](msilogging.md) e a política DisableLoggingFromPackage Windows Installer 4.0.

## <a name="registry-key"></a>Chave do Registro

**HKEY \_ Instalador \_ do** Microsoft machine \\ **software** \\ **policies** \\  \\ **microsoft Windows** \\ 

## <a name="data-type"></a>Tipo de Dados

**REG \_ DWORD**
