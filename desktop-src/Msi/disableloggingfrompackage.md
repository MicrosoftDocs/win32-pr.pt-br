# <a name="disableloggingfrompackage"></a>DisableLoggingFromPackage

O valor dessa [política do sistema](system-policy.md) por máquina é definido como "1" para desabilitar o log especificado para o pacote pela propriedade [**MsiLogging**](msilogging.md) para todos os usuários do computador.

A propriedade [**MsiLogging**](msilogging.md) e a política DisableLoggingFromPackage exigem Windows Installer 4,0.

## <a name="registry-key"></a>Chave do Registro

**HKEY \_ \_** Políticas de \\ **software** do computador local \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo de Dados

**REG \_ DWORD**
