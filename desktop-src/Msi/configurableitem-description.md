# <a name="configurableitemdescription-property"></a>Propriedade ConfigurableItem. Description

A propriedade **Description** retorna o valor da coluna Description da [Tabela ModuleConfiguration](moduleconfiguration-table.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = ConfigurableItem.Description
```



## <a name="property-value"></a>Valor da propriedade

## <a name="c"></a>C++

Consulte a função [**obter \_ Descrição**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_description)
