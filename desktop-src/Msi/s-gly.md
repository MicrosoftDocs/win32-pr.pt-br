# <a name="s-windows-installer"></a>S (Windows Instalador)

<dl> <dt>

<span id="_msi_sequence_tables_gly"></span><span id="_MSI_SEQUENCE_TABLES_GLY"></span>**tabelas de sequência**
</dt> <dd>

Tabelas no [*banco de dados do instalador*](i-gly.md) que listam ações sequencialmente. Para obter mais informações, consulte [Usando uma tabela de sequência](using-a-sequence-table.md).

</dd> <dt>

<span id="_msi_source_list_gly"></span><span id="_MSI_SOURCE_LIST_GLY"></span>**lista de origem**
</dt> <dd>

Especifica os locais em que o instalador deve pesquisar arquivos [*de origem externos.*](e-gly.md)

</dd> <dt>

<span id="_msi_standard_action_gly"></span><span id="_MSI_STANDARD_ACTION_GLY"></span>**ação padrão**
</dt> <dd>

Ação integrada [*do*](a-gly.md) Windows Instalador. Para obter mais informações, consulte [Ações padrão](standard-actions.md).

</dd> <dt>

<span id="_msi_standard_user_gly"></span><span id="_MSI_STANDARD_USER_GLY"></span>**usuário padrão**
</dt> <dd>

Usuário sem privilégios de administrador.

</dd> <dt>

<span id="_msi_structured_query_language_gly"></span><span id="_MSI_STRUCTURED_QUERY_LANGUAGE_GLY"></span>**linguagem SQL (SQL)**
</dt> <dd>

Linguagem usada para processar dados em um banco de dados relacional. Você pode usar SQL para fazer consultas no banco de dados do instalador. Para obter mais informações, consulte [Trabalhando com consultas](working-with-queries.md).

</dd> <dt>

<span id="_msi_summary_information_stream_gly"></span><span id="_MSI_SUMMARY_INFORMATION_STREAM_GLY"></span>**fluxo de informações resumidas**
</dt> <dd>

Armazenamento local no arquivo [*.msi para*](m-gly.md) obter informações que podem ser exibidas com o Microsoft Windows Explorer. Para obter mais informações, consulte [Sobre o fluxo de informações de resumo.](about-the-summary-information-stream.md)

</dd> <dt>

<span id="_msi_synchronous_execution_gly"></span><span id="_MSI_SYNCHRONOUS_EXECUTION_GLY"></span>**execução síncrona**
</dt> <dd>

Quando o instalador aguarda a conclusão do thread separado de uma ação personalizada antes de continuar o thread da instalação atual. Para obter mais informações, consulte Ações personalizadas síncronas e [assíncronas](synchronous-and-asynchronous-custom-actions.md)

</dd> <dt>

<span id="_msi_system_policy_using_windows_installer__gly"></span><span id="_MSI_SYSTEM_POLICY_USING_WINDOWS_INSTALLER__GLY"></span>**política do sistema**
</dt> <dd>

Usado pelos administradores do sistema para controlar as configurações de usuário e computador de um único local em uma rede. As políticas do sistema propagam as configurações do Registro para um grande número de computadores sem exigir que o administrador tenha conhecimento detalhado do Registro. A política do sistema do instalador pode ser configurada usando o GPE (Editor Política de Grupo), incluído no Windows 2000. Para obter mais informações, consulte [Política do sistema.](system-policy.md)

</dd> </dl>
