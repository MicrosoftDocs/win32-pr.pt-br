# <a name="featureinfodescription-property"></a>Propriedade FeatureInfo. Description

A propriedade **Description** é uma propriedade somente leitura que retorna a descrição do recurso na coluna Descrição da [tabela de recursos](feature-table.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = FeatureInfo.Description
```



## <a name="property-value"></a>Valor da propriedade
