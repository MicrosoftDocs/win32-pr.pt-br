# <a name="title-summary-property"></a>Propriedade de resumo do título

A propriedade **Resumo do título** descreve brevemente o tipo do pacote do instalador. Frases como "banco de dados de instalação" ou "transformação" ou "patch" podem ser usadas para essa propriedade.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versão<br/> | Windows Installer 5,0 no Windows Server 2012, Windows 8, Windows Server 2008 R2 ou Windows 7. Windows Installer 4,0 ou Windows Installer 4,5 no Windows Server 2008 ou no Windows Vista. Windows Installer no Windows Server 2003 ou no Windows XP<br/> |
