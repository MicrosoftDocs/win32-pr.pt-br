# <a name="ice85"></a>ICE85

ICE85 valida que a coluna SourceName da [tabela MoveFile](movefile-table.md) é um nome de arquivo longo válido. Este campo pode conter caracteres curinga ( \* e?).

## <a name="result"></a>Resultado

ICE85 posta os erros a seguir.



| Erro de ICE85                                                      | Descrição                                                              |
|------------------------------------------------------------------|--------------------------------------------------------------------------|
| O SourceName ' \[ 2 \] ' encontrado na tabela MoveFile tem formato inadequado. | O campo SourceName na tabela MoveFile não é um nome de arquivo longo válido |
