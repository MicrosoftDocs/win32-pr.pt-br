# <a name="identifier"></a>Identificador

O tipo de dados Identificador é uma cadeia de caracteres de texto. Os identificadores podem conter os caracteres ASCII A-Z (a-z), dígitos, sublinhados ( ) ou \_ pontos (.). No entanto, cada identificador deve começar com uma letra ou um sublinhado.
