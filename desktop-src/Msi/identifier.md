# <a name="identifier"></a>Identificador

O tipo de dados do identificador é uma cadeia de texto. Os identificadores podem conter os caracteres ASCII de a-Z (a-z), dígitos, sublinhados ( \_ ) ou pontos (.). No entanto, cada identificador deve começar com uma letra ou um sublinhado.
