# <a name="keepmodeless-dialog-style-bit"></a>Bit de estilo de caixa de diálogo KeepModeless

Normalmente, quando esse bit não está definido e uma caixa de diálogo é criada por meio de DoAction, todas as outras caixas de diálogo (normalmente sem modo) são destruídas. Se esse bit estiver definido, as outras caixas de diálogo permanecerão ativas quando essa caixa de diálogo for criada.

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                              |
|---------|-------------|---------------------------------------|
| 16      | 0x00000010  | **msidbDialogAttributesKeepModeless** |
