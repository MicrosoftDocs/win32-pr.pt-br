# <a name="msidisableluapatching-property"></a>Propriedade MSIDISABLELUAPATCHING

Defina a propriedade **MSIDISABLELUAPATCHING** como 1 na linha de comando ou na tabela de [Propriedades](property-table.md) para evitar a aplicação de patches com privilégios mínimos de um aplicativo. Para evitar a aplicação de patches com privilégios mínimos de todos os aplicativos no computador, defina a política [DisableLUAPatching](disableluapatching.md) como 1. Para obter informações sobre patches de conta de usuário com privilégios mínimos, consulte [aplicação de patch de UAC (controle de conta de usuário)](user-account-control--uac--patching.md).

| Requisito | Valor |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versão<br/> | Windows Installer 5,0 no Windows Server 2012, Windows 8, Windows Server 2008 R2 ou Windows 7. Windows Installer 4,0 ou Windows Installer 4,5 no Windows Server 2008 ou no Windows Vista. Windows Installer 3,0 ou posterior no Windows Server 2003 ou no Windows XP. Consulte os [requisitos de Run-Time Windows Installer](windows-installer-portal.md) para obter informações sobre a Service Pack mínima do Windows exigida por uma versão Windows Installer.<br/> |
