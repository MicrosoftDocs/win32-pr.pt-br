# <a name="diskcost-dialog"></a>Caixa de diálogo DiskCost

Uma caixa de diálogo DiskCost normalmente aparece quando a propriedade [**OutOfDiskSpace**](outofdiskspace.md) está definida. Isso indica que não há espaço em disco suficiente para a instalação selecionada.

Essa caixa de diálogo também pode ser ativada no [diálogo de seleção](selection-dialog.md) para fornecer ao usuário uma visão geral do espaço em disco necessário para a instalação.

Uma caixa de diálogo DiskCost é uma caixa de diálogo modal. Normalmente, ele contém um botão de [pressão](pushbutton-control.md) que retorna o usuário para a caixa de diálogo anterior e um [controle VolumeCostList](volumecostlist-control.md).
