# <a name="object-model-for-mergemoddll-version-20"></a>Modelo de objeto para Mergemod.dll versão 2,0

O modelo de objeto para Mergemod.dll versão 2,0 é organizado da seguinte maneira.

![modelo de objeto para mergemod.dll versão 2,0](images/mergobj2.png)

O [objeto de mesclagem](merge-object.md) é o objeto principal do modelo. O [**objeto GetFiles**](getfiles-object.md) é um objeto secundário. As dependências são coleções de [**objetos de dependência**](dependency-object.md). Erros são coleções de [**objetos de erro**](error-object.md). ConfigurableItems são coleções do [**objeto ConfigurableItem**](configurableitem-object.md).
