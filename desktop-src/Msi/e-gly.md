# <a name="e-windows-installer"></a>E (Windows Instalador)

[A](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) E [F](f-gly.md) [G](g-gly.md) H [I](i-gly.md) J K L [M](m-gly.md) N [O](o-gly.md) [P](p-gly.md) [Q](q-gly.md) [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) [U](u-gly.md) [V](v-gly.md) W X Y Z

<dl> <dt>

<span id="_msi_elevated_gly"></span><span id="_MSI_ELEVATED_GLY"></span>**Elevado**
</dt> <dd>

As ações executadas com privilégios do sistema são chamadas de elevadas.

</dd> <dt>

<span id="_msi_execution_phase_gly"></span><span id="_MSI_EXECUTION_PHASE_GLY"></span>**fase de execução**
</dt> <dd>

Quando o instalador executa um script de ações do instalador. No Microsoft Windows 2000, esse processo é executado pelo serviço do instalador. Em um aplicativo gerenciado, o script é executado com privilégios do sistema. Para obter mais informações, consulte [Mecanismo de instalação](installation-mechanism.md).

</dd> <dt>

<span id="_msi_execution_script_gly"></span><span id="_MSI_EXECUTION_SCRIPT_GLY"></span>**script de execução**
</dt> <dd>

Ações do instalador para uma instalação. O script de execução é gerado durante a fase [*de aquisição*](a-gly.md) da instalação e executado durante a [*fase de execução*](/windows). Para obter mais informações, consulte [Mecanismo de instalação](installation-mechanism.md).

</dd> <dt>

<span id="_msi_external_source_files_gly"></span><span id="_MSI_EXTERNAL_SOURCE_FILES_GLY"></span>**arquivos de origem externos**
</dt> <dd>

Arquivos de origem do aplicativo que está sendo instalado que são armazenados fora do [*arquivo.msi .*](m-gly.md) Vários arquivos de origem externos podem ser compactados e armazenados juntos dentro de um arquivo [*de gabinete*](c-gly.md) incluído no [*pacote*](p-gly.md).

</dd> <dt>

<span id="_msi_external_user_interface_gly"></span><span id="_MSI_EXTERNAL_USER_INTERFACE_GLY"></span>**interface do usuário externa**
</dt> <dd>

Interface do usuário fornecida pelo autor de um pacote de instalação. Ele não usa os recursos internos da interface do usuário do instalador. Para obter mais informações, [consulte Sobre o Interface do Usuário](about-the-user-interface.md).

</dd> </dl>
