# <a name="featureinfoattributes-property"></a>Propriedade FeatureInfo. Attributes

A propriedade **Attributes** é uma propriedade de leitura/gravação que retorna o valor do recurso na coluna atributos da [tabela de recursos](feature-table.md).

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```JScript
propVal = FeatureInfo.Attributes
FeatureInfo.Attributes = propVal 
```
