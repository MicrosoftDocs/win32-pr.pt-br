# <a name="mergecloselog-method"></a>Método Merge. CloseLog

O método **CloseLog** do objeto de [**mesclagem**](merge-object.md) fecha o arquivo de log atual.

## <a name="syntax"></a>Sintaxe


```JScript
Merge.CloseLog()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.

## <a name="c"></a>C++

Consulte a função [**CloseLog**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-closelog) .
