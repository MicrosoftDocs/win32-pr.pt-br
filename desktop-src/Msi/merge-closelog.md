# <a name="mergecloselog-method"></a>Método Merge.CloseLog

O **método CloseLog** do objeto [**Merge**](merge-object.md) fecha o arquivo de log atual.

## <a name="syntax"></a>Sintaxe


```JScript
Merge.CloseLog()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.

Consulte [**Função CloseLog.**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-closelog)
