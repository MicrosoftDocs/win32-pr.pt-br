# <a name="restrictedusercontrol-property"></a>Propriedade RestrictedUserControl

A propriedade **RestrictedUserControl** deve ser definida como 1 se for restringir quais [Propriedades públicas](public-properties.md) podem ser enviadas para o lado do servidor. Os autores podem usar essa propriedade na instrução condicional para fazer a execução de caixas de diálogo ou ações dependendo se houver restrições em propriedades públicas.
