# <a name="filesinuse-dialog"></a>Caixa de diálogo FilesInUse

Uma caixa de diálogo FilesInUse alerta o usuário sobre os processos em execução no momento que devem ser substituídos ou excluídos pela instalação. Isso dá ao usuário a oportunidade de desligar esses processos e evitar a reinicialização do computador para concluir a substituição ou a exclusão desses arquivos. Consulte [reinicializações do sistema](system-reboots.md).

Essa caixa de diálogo será criada conforme exigido pela [ação InstallValidate](installvalidate-action.md).
