# <a name="sessionlanguage-property"></a>Propriedade Session. Language

A propriedade **Language** do objeto [**Session**](session-object.md) é uma propriedade somente leitura que representa a ID de idioma numérico usada pela sessão de instalação atual.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
propVal = Session.Language
```



## <a name="property-value"></a>Valor da propriedade
