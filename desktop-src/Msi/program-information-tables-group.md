# <a name="program-information-tables-group"></a>Grupo de tabelas de informações do programa

O grupo tabelas de informações do programa contém informações importantes que são usadas em toda a instalação.

-   A [tabela de propriedades](property-table.md) fornece um meio para especificar todas as [Propriedades](properties.md) de uma instalação do.
-   A [tabela binária](binary-table.md) contém os dados binários de itens como bitmaps, animações e ícones. A tabela binária também é usada para armazenar dados para ações personalizadas. Esta tabela pode conter um bitmap para um mural, o ícone do seu programa ou a forma executável de uma ação personalizada.
-   A [tabela de erros](error-table.md) é usada para pesquisar modelos de formatação de mensagem de erro ao processar erros com um código de erro definido. Isso é para o caso usual em que não há nenhum modelo de formatação definido. O instalador tem seu próprio mecanismo de processamento de erro. Os erros são passados como registros.
-   [Tabela de atalho](shortcut-table.md) a tabela de atalho contém as informações de que o aplicativo precisa para criar atalhos no computador do usuário.
-   A [tabela ReserveCost](reservecost-table.md) contém o espaço em disco necessário para que cada componente funcione corretamente.
