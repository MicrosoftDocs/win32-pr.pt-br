# <a name="program-information-tables-group"></a>Grupo de Tabelas de Informações do Programa

O grupo Tabelas de Informações do Programa contém informações importantes usadas em toda a instalação.

-   A [tabela Propriedade](property-table.md) fornece um meio para especificar todas as [propriedades](properties.md) de uma instalação.
-   A [tabela Binária](binary-table.md) contém os dados binários para itens como bitmaps, animações e ícones. A tabela binária também é usada para armazenar dados para ações personalizadas. Essa tabela pode conter um bitmap para um widget, o ícone do programa ou a forma executável de uma ação personalizada.
-   A [tabela Erro](error-table.md) é usada para procurar modelos de formatação de mensagem de erro ao processar erros com um código de erro definido. Isso ocorre para o caso comum em que não há nenhum conjunto de modelos de formatação. O instalador tem seu próprio mecanismo de processamento de erros. Os erros são passados como registros.
-   [A tabela de](shortcut-table.md) atalhos da tabela de atalho contém as informações que o aplicativo precisa para criar Atalhos no computador do usuário.
-   A [tabela ReserveCost contém](reservecost-table.md) o espaço em disco necessário para que cada componente funcione corretamente.
