# <a name="summary-information-stream"></a>Fluxo de informações de resumo

O material nesta seção destina-se a desenvolvedores que estão escrevendo seus próprios programas de instalação que desejam saber mais sobre Windows Installer tabelas de banco de dados. Para obter informações gerais, consulte [About Windows Installer](about-windows-installer.md).

O fluxo de informações de resumo é usado pelo instalador para duas finalidades. Primeiro, ele contém informações sobre o pacote que pode ser exibido por meio do Microsoft Windows Explorer. Essas informações podem ser acessadas por meio da interface **IStream** . Em segundo lugar, ele contém propriedades que são usadas pelo instalador para instalar o pacote. Os tópicos a seguir fornecem informações sobre como usar o fluxo de informações de resumo com o instalador:

-   [Sobre o fluxo de informações de resumo](about-the-summary-information-stream.md)
-   [Usando o fluxo de informações de resumo](using-the-summary-information-stream.md)
-   [Referência de fluxo de informações de resumo](summary-information-stream-reference.md)
