# <a name="summary-information-stream"></a>Fluxo de informações resumidas

O material desta seção destina-se a desenvolvedores que estão escrevendo seus próprios programas de instalação que querem saber mais sobre as tabelas de banco de dados Windows Instalador. Para obter informações gerais, [consulte About Windows Installer](about-windows-installer.md).

O fluxo de informações de resumo é usado pelo instalador para duas finalidades. Primeiro, ele contém informações sobre o pacote que pode ser exibido por meio do Microsoft Windows Explorer. Essas informações podem ser acessadas por meio da interface **IStream.** Em segundo lugar, ele contém propriedades que são usadas pelo instalador para instalar o pacote. Os tópicos a seguir fornecem informações sobre como usar o fluxo de informações resumidas com o instalador:

-   [Sobre o fluxo de informações de resumo](about-the-summary-information-stream.md)
-   [Usando o fluxo de informações de resumo](using-the-summary-information-stream.md)
-   [Referência de fluxo de informações de resumo](summary-information-stream-reference.md)
