# <a name="programmenufolder-property"></a>Propriedade ProgramMenuFolder

O instalador define a propriedade **ProgramMenuFolder** como o caminho completo da pasta de menu do programa para o usuário atual. Se um perfil "todos os usuários" existir e a propriedade [**AllUsers**](allusers.md) estiver definida, essa propriedade será definida como a pasta no perfil "todos os usuários".
