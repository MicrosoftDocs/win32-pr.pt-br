# <a name="programmenufolder-property"></a>Propriedade ProgramMenuFolder

O instalador define a **propriedade ProgramMenuFolder** como o caminho completo da pasta Menu do Programa para o usuário atual. Se existir um perfil "Todos os Usuários" e a [**propriedade ALLUSERS**](allusers.md) estiver definida, essa propriedade será definida como a pasta no perfil "Todos os Usuários".
