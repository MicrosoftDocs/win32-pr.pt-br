# <a name="managing-bluetooth-devices-and-services"></a>Gerenciando dispositivos e serviços Bluetooth

Esta seção descreve como usar a API do Bluetooth para controlar diretamente os dispositivos e serviços Bluetooth.

Para obter informações sobre como usar a interface do Windows Sockets para programar Bluetooth no Windows, consulte [suporte do Windows Sockets para Bluetooth](windows-sockets-support-for-bluetooth.md).

> [!Note]  
> O Bluetooth não impede que os recursos de gerenciamento de energia interrompam as transmissões de Bluetooth. Os aplicativos habilitados para Bluetooth podem monitorar mensagens de energia para evitar tal interrupção. Para obter mais informações, consulte [usando o gerenciamento de energia](/windows/desktop/Power/using-power-management).
