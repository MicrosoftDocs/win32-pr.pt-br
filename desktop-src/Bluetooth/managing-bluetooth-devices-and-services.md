# <a name="managing-bluetooth-devices-and-services"></a>gerenciando dispositivos e serviços Bluetooth

esta seção descreve como usar a API Bluetooth para controlar diretamente Bluetooth dispositivos e serviços.

para obter informações sobre como usar a interface de soquetes Windows para programar Bluetooth no Windows, consulte [suporte a Windows sockets para Bluetooth](windows-sockets-support-for-bluetooth.md).

> [!Note]  
> Bluetooth não impede que os recursos de gerenciamento de energia interrompam Bluetooth transmissões. os aplicativos habilitados para Bluetooth podem monitorar mensagens de energia para evitar tal interrupção. Para obter mais informações, consulte [usando o gerenciamento de energia](/windows/desktop/Power/using-power-management).
