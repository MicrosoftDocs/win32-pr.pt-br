# <a name="bluetooth-and-wsaqueryset"></a>Bluetooth e WSAQUERYSET

A estrutura [**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) é usada em operações, incluindo consulta de dispositivo, consulta de serviço e configuração do serviço.

Os tópicos a seguir descrevem o uso da estrutura WSAQUERYSET em cada uma dessas operações:

-   [Bluetooth e WSAQUERYSET para a consulta do dispositivo](bluetooth-and-wsaqueryset-for-device-inquiry.md)
-   [Bluetooth e WSAQUERYSET para consulta de serviço](bluetooth-and-wsaqueryset-for-service-inquiry.md)
-   [Bluetooth e WSAQUERYSET para o serviço de conjunto](bluetooth-and-wsaqueryset-for-set-service.md)

para obter mais informações e uma referência de [**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) completa ou para outros elementos de soquetes de Windows, consulte a seção Windows sockets do SDK (Software Development Kit) da plataforma.
