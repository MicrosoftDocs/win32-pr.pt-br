# <a name="bluetooth-and-wsaqueryset"></a>Bluetooth e WSAQUERYSET

A estrutura [**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) é usada em operações, incluindo consulta de dispositivo, consulta de serviço e configuração do serviço.

Os tópicos a seguir descrevem o uso da estrutura WSAQUERYSET em cada uma dessas operações:

-   [Bluetooth e WSAQUERYSET para consulta de dispositivo](bluetooth-and-wsaqueryset-for-device-inquiry.md)
-   [Bluetooth e WSAQUERYSET para consulta de serviço](bluetooth-and-wsaqueryset-for-service-inquiry.md)
-   [Bluetooth e WSAQUERYSET para o serviço set](bluetooth-and-wsaqueryset-for-set-service.md)

Para obter mais informações e uma referência [**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) completa ou para outros elementos do Windows Sockets, consulte a seção Windows Sockets do SDK (Software Development Kit) da plataforma.
