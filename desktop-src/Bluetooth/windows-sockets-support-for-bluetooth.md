# <a name="windows-sockets-support-for-bluetooth"></a>Suporte do Windows Sockets para Bluetooth

Esta seção descreve como usar a interface Windows Sockets com Bluetooth funcionalidade introduzida no Windows Vista. Para obter mais detalhes sobre as APIs que possibilitam isso, consulte Programação Bluetooth com soquetes [Windows](bluetooth-programming-with-windows-sockets.md) e o exemplo [Bluetooth de conexão.](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Official%20Windows%20Platform%20Sample/Bluetooth%20connection%20sample)

Para obter informações sobre como gerenciar dispositivos diretamente usando interfaces de Bluetooth não soquetes, consulte [Managing Bluetooth Devices and Services](managing-bluetooth-devices-and-services.md).
