# <a name="windows-sockets-support-for-bluetooth"></a>Suporte do Windows Sockets para Bluetooth

Esta seção descreve como usar a interface do Windows Sockets com a funcionalidade do Bluetooth introduzida no Windows Vista. Para obter mais detalhes sobre as APIs que tornam isso possível, consulte [programação Bluetooth com o Windows Sockets](bluetooth-programming-with-windows-sockets.md) e o [exemplo de conexão Bluetooth](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Official%20Windows%20Platform%20Sample/Bluetooth%20connection%20sample).

Para obter informações sobre como gerenciar dispositivos diretamente usando interfaces Bluetooth sem soquetes, consulte [Gerenciando dispositivos e serviços Bluetooth](managing-bluetooth-devices-and-services.md).
