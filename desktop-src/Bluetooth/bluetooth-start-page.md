# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>Finalidade

Bluetooth é um protocolo padrão do setor que permite a conectividade sem fio para computadores, dispositivos portáteis, celulares e outros dispositivos.

## <a name="where-applicable"></a>Quando aplicável

a interface de programação de aplicativo Bluetooth permite que os desenvolvedores usem o conhecimento de programação de rede existente para desenvolver ou portar aplicativos rapidamente.

## <a name="developer-audience"></a>Público de desenvolvedores

o Bluetooth foi projetado para ser usado por programadores C/C++. alguns recursos Bluetooth estão disponíveis com Windows soquetes. é necessário ter familiaridade com o Microsoft Windows networking e a programação Windows sockets.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

o suporte do Microsoft Bluetooth começa com o Windows XP com Service Pack 1 (SP1).

o suporte para o Bluetooth 2,1 é oferecido no Windows Vista SP2 e no Windows 7.
