# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>Finalidade

O Bluetooth é um protocolo padrão do setor que permite a conectividade sem fio para computadores, dispositivos portáteis, telefones móveis e outros dispositivos.

## <a name="where-applicable"></a>Quando aplicável

A interface de programação de aplicativo Bluetooth permite que os desenvolvedores usem o conhecimento de programação de rede existente para desenvolver rapidamente ou por portabilidade de aplicativos.

## <a name="developer-audience"></a>Público de desenvolvedores

O Bluetooth foi projetado para uso por programadores C/C++. Alguns recursos de Bluetooth estão disponíveis com o Windows Sockets. É necessária familiaridade com a programação de rede do Microsoft Windows e Windows Sockets.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O suporte a Bluetooth da Microsoft começa com o Windows XP com o Service Pack 1 (SP1).

O suporte para Bluetooth 2.1 é oferecido no Windows Vista SP2 e Windows 7.
