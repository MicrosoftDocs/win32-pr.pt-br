# <a name="bluetooth-and-getpeername"></a>Bluetooth e getemparelhaname

A função [**getpeername**](/windows/desktop/api/winsock/nf-winsock-getpeername) é usada para recuperar o endereço Bluetooth do dispositivo Bluetooth par. A função **getpeername** só é válida para soquetes Bluetooth conectados.

Para obter mais informações, consulte [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
