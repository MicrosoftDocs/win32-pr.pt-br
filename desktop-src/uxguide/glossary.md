## <a name=""></a><a name="alink">A</a>

**Caixa sobre**

Uma caixa de diálogo que fornece informações gerais do programa, como identificação de versão, direitos autorais, contratos de licenciamento e maneiras de acessar o suporte técnico.

**acima da dobra**

Uma metáfora de layout de tela tirada do jornal jornalismo. O conteúdo acima da dobra do jornal deve ser particularmente interessante para estimular vendas. Da mesma forma, no layout da tela, o conteúdo mais importante deve estar visível sem rolagem. Os usuários devem ser motivados a passar o tempo para rolar o conteúdo que encontrar inicialmente "acima da dobra".

**chave de acesso**

Uma chave alfanumérica que, quando combinada com a tecla Alt, ativa um controle. As chaves de acesso são indicadas por meio do sublinhado de um dos caracteres no rótulo do controle. Por exemplo, pressionar ALT + O ativa um controle cujo rótulo é "aberto" e cuja chave de acesso atribuída é "O". Chaves de acesso não diferenciam maiúsculas de minúsculas. O efeito de ativar um controle depende do tipo de controle.

Em contraste com as teclas de atalho, que se destinam principalmente a usuários avançados, as chaves de acesso são projetadas para melhorar a acessibilidade. Como eles são documentados diretamente dentro da interface do usuário, eles nem sempre podem ser atribuídos de forma consistente e não devem ser memorizados.

**monitor ativo**

O monitor em que o programa ativo está em execução.

**barra de endereços**

Um elemento de navegação, geralmente exibido na parte superior de uma janela, que exibe e permite que os usuários alterem, sua localização atual. Consulte também: [barra de navegação estrutural](#b).

**preço**

Propriedades visuais de um objeto que indicam como ele pode ser usado ou acionado. Por exemplo, os botões de comando têm a aparência visual dos botões do mundo real, sugerindo enviar ou clicar.

**aplicativo**

Um programa usado para executar um conjunto relacionado de tarefas de usuário; geralmente relativamente complexo e sofisticado. Consulte também: [programa](#p).

**chave do aplicativo**

Uma tecla de teclado com um gráfico de menu de contexto. Essa chave é usada para exibir o menu de contexto do item selecionado.

**menu do aplicativo**

Um controle que apresenta um menu de comandos que envolvem fazer algo com um documento ou espaço de trabalho, como comandos relacionados a arquivos.

Exibe o menu de contexto da seleção atual (o mesmo que pressionar SHIFT + F10).

**aplicativos com tema**

Usar técnicas visuais relacionadas, como controles personalizados, para criar uma aparência ou marca exclusiva para um aplicativo.

**taxa de proporção**

Uma expressão da relação entre a largura de um objeto e sua altura. Por exemplo, a televisão de alta definição usa uma taxa de proporção de 16:9.

**preenchimento automático**

Um tipo de lista usada em caixas de texto e listas suspensas editáveis nas quais a entrada provável pode ser extrapolada e preenchida automaticamente de tendo sido inserida anteriormente. Os usuários digitam uma quantidade mínima de texto para preencher a lista de preenchimento automático.

**sair automaticamente**

Uma caixa de texto na qual o foco de entrada se move automaticamente para a próxima caixa de texto relacionada assim que um usuário digita o último caractere.

## <a name="b"></a>B

**balão**

Um controle comum do Windows que informa os usuários sobre um problema não crítico ou uma condição especial.

**barra de navegação estrutural**

Um elemento de navegação, geralmente exibido na parte superior de uma janela, que exibe e permite que os usuários alterem, sua localização atual. "Breadcrumb" refere-se à interrupção do local atual em uma série de links separados por setas com as quais os usuários podem interagir diretamente. Em vez disso, use a barra de endereços. Consulte também: [barra de endereços](#alink).

## <a name="c"></a>C

**caixa de seleção**

Um controle comum do Windows que permite aos usuários decidir entre opções claramente diferentes, como ativar ou desativar uma opção.

**visível**

Um pequeno controle ou botão que indica que há mais itens do que podem ser exibidos no espaço alocado. Os usuários clicam na divisa para ver os itens adicionais.

**janela filho**

Uma janela, como um controle ou painel, que está contida completamente dentro de outra janela, chamada de janela pai. Consulte também: [janela pai](#p), [janela de propriedade](#o).

**desmarcado**

Em uma caixa de seleção, indica que a opção não está definida. Consulte também: [estado misto](#m) [selecionado](#s).

**caixa de combinação**

Um controle comum do Windows que combina as características de uma lista suspensa ou caixa de listagem padrão e uma caixa de texto editável. Consulte também: [caixa de listagem](#l), [lista suspensa](#d).

**área de comando**

A área em uma janela em que os botões de confirmação estão localizados. Normalmente, caixas de diálogo e assistentes têm uma área de comando. Consulte também: botão confirmar.

**botão de comando**

Um controle comum do Windows que permite que os usuários iniciem uma ação imediatamente.

**link de comando**

Um controle usado para fazer uma escolha entre um conjunto de opções relacionadas mutuamente exclusivas. Em seu estado normal, os links de comando têm uma aparência leve semelhante a hiperlinks, mas seu comportamento é mais semelhante aos botões de comando.

**botão confirmar**

Um botão de comando usado para confirmar uma tarefa, prosseguir para a próxima etapa em uma tarefa de várias etapas ou cancelar uma tarefa. Consulte também: área de comando.

**página de confirmação**

Um tipo de página de assistente na qual os usuários se confirmam para executar a tarefa. Depois de fazer isso, a tarefa não pode ser desfeita clicando nos botões voltar ou cancelar.

**página conclusão**

Uma página de assistente usada para indicar o final de um assistente. Às vezes, usado em vez de páginas de parabenização. Consulte também: página Parabéns.

**página de parabéns**

Uma página de assistente usada para indicar o final de um assistente. Essas páginas não são mais recomendadas. Os assistentes são concluídos com mais eficiência com uma página de confirmação ou, se necessário, uma página de acompanhamento ou conclusão. Consulte também: página confirmar, página conclusão, [página de acompanhamento](#f).

**IU de consentimento**

Uma caixa de diálogo usada pelo controle de conta de usuário (UAC) que permite que os administradores protegidos elevem seus privilégios temporariamente.

**Constraint**

Em controles que envolvem a entrada do usuário, como caixas de texto, as restrições de entrada são uma maneira valiosa de evitar erros. Por exemplo, se a única entrada válida para um controle específico for numérica, o controle poderá usar as restrições de valor apropriadas para impor esse requisito.

**área de conteúdo**

A parte das superfícies da interface do usuário, como caixas de diálogo, itens do painel de controle e assistentes, dedicado à apresentação de opções, fornecimento de informações e descrição de controles. Diferenciado da área de comando, do painel de tarefas e da área de navegação.

**guia contextual**

Uma guia que contém uma coleção de comandos que são relevantes somente quando o usuário seleciona um tipo de objeto específico. Consulte também: [Ribbon](#r).

**Painel de Controle**

Um programa do Windows que coleta e exibe para usuários os recursos de nível de sistema do computador, incluindo instalação e configuração de hardware e software. No painel de controle, os usuários podem clicar em itens individuais para configurar recursos no nível do sistema e executar tarefas relacionadas. Consulte também: item do painel de controle.

**item do painel de controle**

Um recurso individual disponível no painel de controle. Por exemplo, programas e facilidade de acesso são dois itens do painel de controle.

**IU da credencial**

Uma caixa de diálogo usada pelo controle de conta de usuário (UAC) que permite aos usuários padrão solicitar a elevação temporária de seus privilégios.

**drasticamente**

O grau mais alto de gravidade. Por exemplo, em mensagens de erro e de aviso, circunstâncias críticas podem envolver perda de dados, perda de privacidade ou perda de integridade do sistema.

**ícone personalizado**

Uma representação de ilustração exclusiva de um programa (em oposição a um ícone de sistema do Windows).

**visuais personalizados**

Elementos gráficos, animações, ícones e outros elementos visuais especialmente desenvolvidos para um programa.

## <a name="d"></a>D

**link ou botão de comando padrão**

O botão de comando ou link que é invocado quando os usuários pressionam a tecla Enter. O botão de comando padrão ou o link é atribuído pelo desenvolvedor, mas qualquer botão de comando ou link se torna o padrão quando os usuários a guiam.

**monitor padrão**

O monitor com o menu Iniciar, a barra de tarefas e a área de notificação.

**modelo de confirmação atrasada**

O modelo de confirmação usado pelas páginas spoke do item do painel de controle, em que as alterações não são feitas até explicitamente confirmadas pelos usuários clicando em um botão de confirmação. Assim, os usuários podem abandonar uma tarefa, navegando para fora usando o botão voltar, fechar ou a barra de endereços. Consulte também: [modelo de confirmação imediata](#i).

**placa**

A área de trabalho na tela fornecida pelo Windows, análoga a um desktop físico. Consulte também: [área de trabalho](#w).

**comando destrutivo**

Uma ação que tem um efeito difundido e não pode ser desfeita facilmente, ou não é imediatamente perceptível.

**painel de detalhes**

O painel na parte inferior de uma janela do Windows Explorer que exibe detalhes (se houver) sobre os itens selecionados; caso contrário, ele exibirá detalhes sobre a pasta. Por exemplo, a Galeria de fotos do Windows exibe o nome da imagem, o tipo de arquivo, a data obtida, as marcas, a classificação, as dimensões e o tamanho do arquivo. Consulte também: [painel de visualização](#p).

**caixa de diálogo**

Uma janela secundária que permite aos usuários executar um comando, pergunta aos usuários uma pergunta ou fornece aos usuários informações ou comentários sobre o progresso.

**iniciador de caixa de diálogo**

Em uma faixa de, um botão na parte inferior de alguns grupos que abre uma caixa de diálogo com recursos relacionados ao grupo. Consulte também: [Ribbon](#r).

**unidade de diálogo**

Uma unidade de caixa de diálogo (DLU) é a medida independente de dispositivo a ser usada para layout com base na fonte atual do sistema.

**manipulação direta**

Interação direta entre o usuário e os objetos na interface do usuário (como ícones, controles e elementos de navegação). O mouse e o toque são métodos comuns de manipulação direta.

**janela encaixada**

Uma janela que aparece em um local fixo na borda de sua janela do proprietário. Consulte também: [janela flutuante](#f).

**seta suspensa**

A seta associada a listas suspensas, caixas de combinação, botões de divisão e botões de menu, indicando que os usuários podem exibir a lista associada clicando na seta.

**lista suspensa**

Um controle comum do Windows que permite aos usuários selecionar entre uma lista de valores mutuamente exclusivos. Ao contrário de uma caixa de listagem, essa lista de opções disponíveis normalmente é oculta.

## <a name="e"></a>E

**resolução efetiva**

A resolução física de um monitor normalizado pela configuração de DPI (pontos por polegada) atual. A 96 dpi, a resolução efetiva é igual à resolução física, mas em outras DPIs, a resolução efetiva deve ser dimensionada proporcionalmente. Em geral, a resolução efetiva pode ser calculada usando a seguinte equação:

Resolução efetiva = resolução física x (96/configuração de DPI atual)

Consulte também: [pixels relativos](#r), [resolução física](#p).

**administrador elevado**

No controle de conta de usuário, os administradores elevados têm seus privilégios de administrador. Sem elevar, os administradores são executados em seu estado com menos privilégios. A caixa de diálogo da interface do usuário de consentimento é usada para elevar os administradores ao status elevado somente quando necessário. Consulte também: [administrador protegido](#p), [usuário padrão](#s).

**Dica de ferramenta aprimorada**

Uma janela pop-up que explica de forma concisa o comando que está sendo apontado. Como Tooltips regulares, as dicas de ferramentas aprimoradas podem fornecer a tecla de atalho para o comando. Mas, ao contrário das dicas de ferramenta normais, elas também podem fornecer informações complementares, gráficos e um indicador de que a ajuda está disponível. Eles também podem usar Rich Text e separadores. Consulte também: [dica de ferramenta](#t).

**error**

Um estado em que ocorreu um problema. Consulte também: [aviso](#w).

**títulos expansíveis**

Um padrão de divisa de divulgação progressiva em que um título pode ser expandido ou recolhido para revelar ou ocultar um grupo de itens. Consulte também: [divulgação progressiva](#p).

**seleção estendida**

Em exibições de lista e caixas de listagem, um modo de seleção múltipla em que a seleção de um único item pode ser estendida arrastando ou com SHIFT + clique ou CTRL + clique para selecionar grupos de valores contíguos ou não adjacentes, respectivamente. Consulte também: [seleção múltipla](#m).

## <a name="f"></a>F

**Flick**

Um traço rápido e reto de um dedo ou caneta em uma tela. Um movimento é reconhecido como um gesto e interpretado como uma navegação ou um comando de edição.

**janela flutuante**

Uma janela que pode aparecer em qualquer lugar na tela que o usuário desejar. Consulte também: [janela encaixada](#d).

**menu**

Uma janela pop-up que mostra temporariamente mais informações. Na área de trabalho do Windows, os submenus são exibidos clicando em um gadget e descartados clicando em qualquer lugar fora do submenu. Você pode usar submenus nos Estados encaixado e flutuante.

**página de acompanhamento**

Uma página de assistente usada para apresentar as tarefas relacionadas que os usuários provavelmente farão como acompanhamento. Às vezes, usado em vez de páginas de parabenização.

**la**

Um conjunto de atributos para caracteres de texto.

**tela inteira**

Uma janela maximizada que não tem um quadro.

## <a name="g"></a>G

**gadget**

Um mini-aplicativo simples hospedado na área de trabalho do usuário. Consulte também: [barra lateral](#s).

**galeria**

Uma lista de comandos ou opções apresentada graficamente. Uma galeria baseada em resultados ilustra o efeito dos comandos ou opções em vez dos próprios comandos. Pode ser rotulado ou agrupado. Por exemplo, as opções de formatação podem ser apresentadas em uma galeria de miniaturas.

**Gesture**

Um movimento rápido de um dedo ou caneta em uma tela que o computador interpreta como um comando, em vez de um movimento de mouse, gravação ou desenho.

**página de introdução**

Uma página de assistente opcional que descreve os pré-requisitos para executar o assistente com êxito ou explica a finalidade do assistente.

**Pare**

Uma opção de quadro de janela caracterizada por translucence, ajudando os usuários a se concentrarem no conteúdo e na funcionalidade, e não na interface em torno dela.

**ícones**

Um termo genérico usado para fazer referência a qualquer grafo ou imagem simbólica. Setas, divisas e marcadores são glifos comumente usados no Windows.

**caixa de grupo**

Um controle comum do Windows que mostra as relações entre um conjunto de controles relacionados.

## <a name="h"></a>H

**reconhecimento de manuscrito**

Software que converte tinta em texto.

**Ajuda**

Assistência ao usuário de uma natureza mais detalhada do que está disponível na interface do usuário principal. Normalmente acessados em um menu ou clicando em um link ou ícone de ajuda, esse conteúdo pode usar uma variedade de formas, incluindo procedimentos passo a passo, texto conceitual ou tutoriais guiados com mais base visualmente.

**modo de alto contraste**

Uma configuração de vídeo especial que fornece contraste extremo para elementos visuais de primeiro e segundo plano (preto em branco ou branco em preto). Particularmente útil para acessibilidade.

**página de Hub**

Em itens do painel de controle, uma página de Hub apresenta opções de alto nível, como as tarefas usadas com mais frequência (como em páginas de Hub baseadas em tarefas) ou os objetos disponíveis (assim como as páginas de Hub baseadas em objeto). Os usuários podem navegar para páginas do spoke para executar tarefas específicas. Consulte também: [página do spoke](#s).

**página de Hub híbrido**

Em itens do painel de controle, uma página de Hub híbrido é uma página de Hub que também tem algumas propriedades ou comandos diretamente. As páginas de Hub híbrido são altamente recomendadas quando os usuários têm mais probabilidade de usar o item do painel de controle para acessar essas propriedades e comandos.

## <a name="i"></a>I

**modelo de confirmação imediata**

O modelo de confirmação usado por páginas de Hub híbrido em que as alterações entram em vigor assim que os usuários as fazem. Botões de confirmação não são usados neste modelo. Consulte também: [modelo de confirmação atrasada](#d).

**mensagem in-loco**

Uma mensagem que aparece no contexto da superfície da interface do usuário atual, em vez de uma janela separada. Diferentemente das janelas separadas, as mensagens in-loco exigem o espaço de tela ou o layout dinâmico disponível.

**caixa de diálogo indireta**

Uma caixa de diálogo exibida fora do contexto, seja como um resultado indireto de uma tarefa ou como resultado de um problema com um sistema ou processo em segundo plano.

**interface do usuário indutivo**

Uma interface do usuário que divide uma tarefa complexa em etapas simples, facilmente explicadas e claramente declaradas com uma finalidade clara.

**InfoTip**

Uma pequena janela pop-up que descreve de forma concisa o objeto que está sendo apontado, como descrições de controles da barra de ferramentas, ícones, elementos gráficos, links, objetos do Windows Explorer, itens de menu iniciar e botões da barra de tarefas. Infotips são uma forma de divulgação progressiva, eliminando sempre a necessidade de ter texto descritivo na tela.

**entrada**

A saída bruta de uma caneta. Essa tinta digital pode ser mantida da mesma forma que é escrita ou pode ser convertida em texto usando o software de reconhecimento de manuscrito.

**linha**

Posicionamento de links ou mensagens diretamente no contexto de sua interface do usuário relacionada. Por exemplo, um link embutido ocorre dentro de outro texto em vez de separadamente.

**foco de entrada**

O local em que o usuário está atualmente direcionando a entrada. Observe que, apenas porque um local na interface do usuário é realçado não significa necessariamente que esse local tem o foco de entrada.

**cópia**

Uma sessão de programa. Por exemplo, o Windows Internet Explorer permite que os usuários executem várias instâncias do programa, pois os usuários podem ter várias sessões independentes em execução por vez. As configurações podem ser salvas entre as sessões do programa. Consulte também: [persistência](#p).

## <a name="j"></a>J

## <a name="k"></a>K

**KeyTip**

Em uma faixa de faixas, o mecanismo usado para exibir as chaves de acesso. As chaves de acesso aparecem na forma de uma pequena dica sobre cada comando ou grupo, em oposição às letras sublinhadas normalmente usadas para exibir as chaves de acesso. Consulte também: [chave de acesso](#alink).

## <a name="l"></a>L

**modo paisagem**

Uma opção de apresentação que orienta um objeto para ser mais largo do que é alto. Consulte também: [modo retrato](#p).

**conta de usuário com privilégios mínimos**

Uma conta de usuário que normalmente é executada com privilégios mínimos. Consulte também: [controle de conta de usuário](#u).

**caixa de listagem**

Um controle comum do Windows que permite que os usuários selecionem um conjunto de valores apresentados em uma lista, que, ao contrário de uma lista suspensa, está sempre visível. Dá suporte a seleções únicas ou múltiplas.

**exibição de lista**

Um controle comum do Windows que permite aos usuários exibir e interagir com uma coleção de objetos de dados, usando uma única seleção ou seleção múltipla.

**Visualização dinâmica**

Uma técnica de visualização que mostra o efeito de um comando imediatamente na seleção ou no foco sem que o usuário confirme a ação. Por exemplo, as opções de formatação, como temas, fontes e cores, se beneficiam das visualizações dinâmicas, mostrando os usuários o efeito com o mínimo de esforço.

**Localizar**

O processo de adaptação de software para diferentes países, idiomas, culturas ou mercados.

**arquivo de log**

Um repositório baseado em arquivo para obter informações sobre vários tipos de atividade em um sistema de computador. Os administradores geralmente consultam arquivos de log; Normalmente, os usuários comuns não fazem isso.

## <a name="m"></a>M

**instrução principal**

Texto exibido em destaque que explica de forma concisa o que fazer na janela ou na página. A instrução deve ser uma instrução específica, uma direção imperativa ou uma pergunta. Boas instruções principais comunicam o objetivo do usuário em vez de se concentrar apenas na manipulação da interface do usuário.

**ambiente gerenciado**

Um ambiente de computador em rede gerenciado por um departamento de ti ou por um provedor de terceiros, em vez de usuários individuais. Os administradores podem otimizar o desempenho e aplicar atualizações do sistema operacional e do aplicativo, entre outras tarefas.

**manusei**

Um tipo de interação por toque na qual a entrada corresponde diretamente a como o objeto que está sendo tocado reagiria naturalmente à ação no mundo real.

**tirar**

Para exibir uma janela em seu tamanho maior. Consulte também: minimizar, [restaurar janela](#r).

**AdicionarMenu**

Uma lista de comandos ou opções disponíveis para usuários no contexto atual.

**caixa de mensagem**

Uma janela secundária que é exibida para informar um usuário sobre uma condição específica.

**mini-barra de ferramentas**

Uma barra de ferramentas contextual exibida ao focalizar.

**maximiza**

Para ocultar uma janela. Consulte também: maximizar, [janela restaurada](#r).

**Estado misto**

Para caixas de seleção que se aplicam a um grupo de itens, um estado misto indica que alguns dos itens estão selecionados e outros são apagados.

**janelas**

Interação restritiva ou limitada devido à operação em um modo. A modal geralmente descreve uma janela secundária que restringe a interação de um usuário com a janela do proprietário. Consulte também: sem janela restrita.

**sem modo**

Interação não restritiva ou não limitada. Modeleva frequentemente descreve uma janela secundária que não restringe a interação de um usuário com a janela do proprietário. Consulte também: modal.

**seleção múltipla**

A capacidade de os usuários escolherem mais de um objeto em uma lista ou árvore.

## <a name="n"></a>N

**evento não crítico do sistema**

Um tipo de evento do sistema que não requer atenção imediata, geralmente relacionado ao status do sistema. Consulte também: [crítico](#c).

**Notification**

Informações de uma natureza não crítica que é exibida brevemente para o usuário; uma notificação assume a forma de um balão de um ícone na área de notificação da barra de tarefas.

## <a name="o"></a>O

**aceitar**

A capacidade de os usuários selecionarem recursos opcionais explicitamente. Menos invasivo para usuários do que a aceitação, especialmente para recursos relacionados à privacidade e ao marketing, porque não há uma continuação dos desejos dos usuários. Consulte também: recusar, opções.

**recusar**

A capacidade dos usuários de remover recursos que eles não desejam desmarcando sua seleção. Mais invasivo para usuários do que aceitar, especialmente para recursos relacionados a privacidade e marketing, porque há uma suposição dos desejos dos usuários. Consulte também: aceitar opções.

**options**

Opções disponíveis para usuários para personalizar um programa. Por exemplo, uma caixa de diálogo Opções permite que os usuários exibam e alterem opções de programa. Consulte também: [Propriedades](#p).

**interface do usuário fora do contexto**

Qualquer interface do usuário exibida em uma janela pop-up que não esteja diretamente relacionada à atividade atual do usuário. Por exemplo, as notificações e a IU de consentimento para o controle de acesso do usuário são IU fora do contexto.

**janela de propriedade**

Uma janela secundária usada para executar uma tarefa auxiliar. Não é uma janela de nível superior (portanto, não é exibida na barra de tarefas); em vez disso, ele é "de propriedade" de sua janela do proprietário. Por exemplo, a maioria das caixas de diálogo são janelas de propriedade. Consulte também: [janela filho](#c), janela do proprietário.

**controle de proprietário**

A origem de uma dica, balão ou submenu. Por exemplo, uma caixa de texto com restrições de entrada pode exibir um balão para permitir que o usuário conheça essas limitações. Nesse caso, a caixa de texto é considerada o controle de proprietário.

**janela do proprietário**

Uma janela da qual uma janela de propriedade é originada. Aparece abaixo da janela de propriedade na ordem Z. Consulte também: janela de propriedade, [janela pai](#p), [ordem Z](#z).

## <a name="p"></a>P

**page**

Uma unidade básica de navegação para a interface do usuário baseada em tarefas, como assistentes, folhas de propriedades, itens do painel de controle e sites da Web. Os usuários executam tarefas navegando de uma página para uma página em uma única janela de host. Consulte também: fluxo de página, [janela](#w).

**fluxo de página**

Uma coleção de páginas em que os usuários executam uma tarefa. Consulte também: página, [tarefa](#t), [Assistente](#w), [painel de controle](#c).

**controle de espaço de página**

Permite que os usuários exibam e interajam com uma coleção hierarquicamente organizada de objetos. Os controles de espaço de página são como controles de árvore, mas têm uma aparência visual ligeiramente diferente. Eles são usados principalmente pelo Windows Explorer.

**janela Paleta**

Uma janela secundária sem-modo que exibe uma barra de ferramentas ou outras opções, como cores, padrões, fontes ou atributos de fonte.

**quadra**

Para mover uma cena, como um mapa ou foto, em duas dimensões arrastando-a diretamente. Isso difere da rolagem de duas maneiras: o conteúdo rolado geralmente tem uma dimensão predominante e, muitas vezes, rola apenas junto a essa dimensão; e a rolagem de conteúdo é exibida de forma convencional com as barras de rolagem que o usuário arrasta na direção oposta do movimento de rolagem.

**painel**

Uma área retangular dentro de uma janela que os usuários podem ser capazes de mover, redimensionar, ocultar ou fechar. Os painéis são sempre encaixados no lado da janela pai. Eles podem ser adjacentes a outros painéis, mas nunca se sobrepõem. Desencaixar um painel converte-o em uma janela filho. Consulte também: [janela](#w).

**janela pai**

O contêiner de janelas filhas (como controles ou painéis). Consulte também: [janela do proprietário](#o).

**pena**

Uma caneta usada para apontar, gestos, entrada de texto simples e manuscrito de forma livre. As canetas têm uma dica tranqüila e suave que dá suporte a anotações, gravações ou desenhos precisos em tinta. Eles também podem ter um botão de caneta opcional (usado para executar cliques com o botão direito do mouse) e borracha (usado para apagar a tinta).

**persistência**

O princípio de que o estado ou as propriedades de um objeto é preservado automaticamente.

**personalização**

Personalização de uma experiência básica que é crucial para a identificação pessoal do usuário com um programa. Por outro lado, as opções e as propriedades comuns não são cruciais para a identificação pessoal do usuário com um programa.

**Personas**

Descrições detalhadas de pessoas imaginários. As pessoas são construídas de dados bem compreendidos e altamente especificados sobre pessoas reais.

**resolução física**

Os pixels horizontal e vertical que podem ser exibidos pelo hardware de um monitor de computador.

**botão de grupo pop-up**

Em uma faixa de opções, um botão de menu que consolida todos os comandos do em um grupo. Usado para exibir as faixas de faixa em espaços pequenos.

**modo retrato**

Uma opção de apresentação que orienta um objeto para ser mais alto do que o necessário. Consulte também: [modo paisagem](#l).

**das**

Não use. Use [as opções](#o) ou propriedades em vez disso.

**versão prévia**

Uma representação do que os usuários verão quando selecionam uma opção. As visualizações podem ser exibidas estaticamente como parte da opção, ou mediante solicitação com um botão Visualizar ou aplicar.

**painel de visualização**

Um painel de janela usado para mostrar visualizações e outros dados sobre objetos selecionados.

**comando primário**

Uma ação central que atende ao objetivo principal de uma janela. Por exemplo, Print é um comando primário para uma caixa de diálogo de impressão. Consulte também: [comando secundário](#s).

**barra de ferramentas primária**

Uma coleção de comandos projetados para ser abrangente o suficiente para impedir o uso de uma barra de menus. Consulte também: [barra de ferramentas suplementar](#s).

**janela principal**

Uma janela principal não tem janela de proprietário e é exibida na barra de tarefas. As janelas principais do programa são sempre janelas primárias. Consulte também: [janela secundária](#s).

**Program**

Uma sequência de instruções que podem ser executadas por um computador. Os tipos comuns de programas incluem aplicativos de produtividade, aplicativos de consumidor, jogos, quiosques e utilitários.

**barra de progresso**

Um controle comum do Windows que exibe o progresso de uma operação específica como uma barra gráfica.

**divulgação progressiva**

Uma técnica de permitir que os usuários exibam informações menos usadas (normalmente, dados, opções ou comandos), conforme necessário. Por exemplo, se mais opções às vezes forem necessárias, os usuários poderão expô-las no contexto clicando em um botão de divisa.

**escalonamento progressivo**

Uma sequência na qual a interface do usuário costumava informar os usuários se torna cada vez mais invasiva à medida que o evento se torna mais crítico. Por exemplo, uma notificação pode ser usada para um evento que os usuários possam ignorar com segurança primeiro. À medida que a situação se torna crítica, uma interface do usuário mais invasiva, como uma caixa de diálogo modal, deve ser usada.

**prompt**

Um rótulo ou uma instrução curta colocada dentro de uma caixa de texto ou lista suspensa editável como seu valor padrão. Ao contrário do texto estático, os prompts desaparecem quando os usuários digitam algo no controle ou ele obtém o foco de entrada.

**properties**

As configurações de um objeto que os usuários podem alterar, como o nome de um arquivo e o status somente leitura, bem como os atributos de um objeto que os usuários não podem alterar diretamente, como o tamanho de um arquivo e a data de criação. Normalmente, as propriedades definem o estado, o valor ou a aparência de um objeto.

**administrador protegido**

No controle de conta de usuário, um administrador em execução em seu estado com privilégios mínimos. Consulte também: [administrador elevado](#e), [usuário padrão](#s).

## <a name="q"></a>Q

**Barra de ferramentas de acesso rápido**

Uma barra de ferramentas pequena e personalizável que exibe comandos usados com frequência.

**Barra de início rápido**

Um ponto de acesso direto na área de trabalho do Windows, localizado ao lado do botão Iniciar, preenchido com ícones para programas da escolha do usuário. Removido do Windows 7.

## <a name="r"></a>R

**botão de opção**

Um controle comum do Windows que permite aos usuários selecionar entre um conjunto de escolhas mutuamente exclusivas e relacionadas.

**pixels relativos**

Uma métrica independente de dispositivo que é igual a um pixel físico a 96 DPI (pontos por polegada), mas dimensionado proporcionalmente em outros DPIs. Consulte também: [resolução efetiva](#e).

**janela restaurada**

Uma janela visível, em tela parcial, não maximizada nem minimizada. Consulte também: [maximizar](#m), minimizar.

**3D**

Um contêiner com guias de comandos e opções, localizado na parte superior de uma janela ou área de trabalho, e que tem um local e uma altura fixos. As faixas de faixa geralmente têm um menu de aplicativo e uma barra de ferramentas de acesso rápido. Consulte também: [menu](#m), [barra de ferramentas](#t).

**ação arriscada**

Uma qualidade de ação do usuário que pode ter consequências negativas e não pode ser facilmente desfeita. Ações arriscadas incluem ações que podem prejudicar a segurança de um computador, afetar o acesso a um computador ou resultar em perda indesejada de dados.

## <a name="s"></a>S

**caminho de verificação**

A rota que os usuários provavelmente terão ao examinar para localizar itens em uma janela. Particularmente importante se os usuários não estiverem envolvidos na leitura de texto de imersão.

**leitor de tela**

Uma tecnologia assistencial que permite aos usuários com deficiências visuais interpretar e navegar por uma interface do usuário transformando visuais em áudio. Assim, texto, controles, menus, barras de ferramentas, elementos gráficos e outros elementos da tela são falados pela voz em computador do leitor de tela.

**barra de rolagem**

Um controle que permite aos usuários rolar o conteúdo de uma janela, tanto vertical quanto horizontalmente.

**comando secundário**

Uma ação periférica que, embora útil, não é essencial para a finalidade da janela. Por exemplo, localizar impressora ou instalar impressora são comandos secundários para uma caixa de diálogo de impressão. Consulte também: [comando primário](#p).

**janela secundária**

Uma janela que tem uma janela do proprietário e, consequentemente, não é exibida na barra de tarefas. Consulte também: [janela principal](#p).

**proteger área de trabalho**

Um ambiente protegido que é isolado de programas em execução no sistema, usado para aumentar a segurança de tarefas altamente seguras, como logon, alterações de senha e interface do usuário de elevação do UAC. Consulte também: [controle de conta de usuário](#u).

**blindagem de segurança**

Um ícone de escudo usado para identidade visual de segurança.

**selecionado**

Escolhido pelo usuário para executar uma operação; taco.

**capitalização de estilo de sentença**

Para capitalização em estilo de sentença:

-   Sempre coloque em maiúscula a primeira palavra de uma nova frase.
-   Não coloque maiúsculas na palavra após dois-pontos, a menos que a palavra seja um substantivo adequado, ou o texto após os dois-pontos é uma frase completa.
-   Não coloque a palavra em maiúscula após um travessão, a menos que seja um substantivo adequado, mesmo que o texto após o traço seja uma frase completa.
-   Sempre coloque em maiúscula a primeira palavra de uma nova frase após qualquer pontuação final. Reescreva as frases que começam com uma palavra com letras minúsculas que diferencia maiúsculas de minúsculas.

**configurações**

Valores específicos que foram escolhidos (pelo usuário ou por padrão) para configurar um programa ou objeto.

**tecla de atalho**

Chaves ou combinações de teclas que os usuários podem pressionar para acessar rapidamente as ações executadas com frequência. CTRL + combinações de letras e teclas de função (F1 a F12) geralmente são as melhores opções para teclas de atalho. Por definição, uma tecla de atalho é o equivalente de teclado da funcionalidade que tem suporte adequado em qualquer lugar na interface. Portanto, evite usar uma tecla de atalho como a única maneira de acessar uma operação específica.

Ao contrário das chaves de acesso, que são projetadas para melhorar a acessibilidade, as teclas de atalho são projetadas principalmente para usuários avançados. Como eles não estão documentados diretamente na própria interface do usuário (embora possam estar documentados em menus e dicas de ferramenta da barra de ferramentas), eles têm a intenção de serem memorizados e, portanto, devem ser atribuídos consistentemente em aplicativos e em aplicativos diferentes.

**Barra lateral**

Uma região no lado da área de trabalho do usuário usada para exibir gadgets no Windows Vista. Consulte também: [gadget](#g).

**erro de ponto único**

Um erro de entrada do usuário relacionado a um único controle. Por exemplo, a inserção de um número de cartão de crédito incorreto é um erro de ponto único, enquanto que um logon incorreto é um erro de ponto duplo, porque o nome de usuário ou a senha pode ser o problema.

**classificação**

Um controle comum do Windows que exibe e define um valor de um intervalo contínuo de valores possíveis, como o brilho ou o volume.

**experiência especial**

Em programas, experiências especiais estão relacionadas à função principal do programa, algo exclusivo sobre o programa ou, de outra forma, fazer uma conexão emocional com os usuários. Por exemplo, reproduzir um áudio ou vídeo é uma experiência especial para um player de mídia.

**caixa de rotação**

A combinação de uma caixa de texto e seu controle de rotação associado. Os usuários clicam na seta para cima ou para baixo de uma caixa de rotação para aumentar ou diminuir um valor numérico. Ao contrário dos controles Slider, que são usados para quantidades relativas, as caixas de rotação são usadas somente para valores numéricos conhecidos e exatos.

**controle de rotação**

Um controle em que os usuários clicam para alterar valores. Controles de rotação usam setas para cima e para baixo para aumentar ou diminuir o valor.

**tela inicial**

A imagem da tela de transição que aparece como um programa está no processo de inicialização.

**botão de divisão**

Um botão de comando bipartite que inclui um botão pequeno com um triângulo apontando para baixo na parte mais à direita do botão principal. Os usuários clicam no triângulo para exibir as variações de um comando em um menu suspenso. Consulte também: [botão de comando](#c).

**página do spoke**

Em itens do painel de controle, as páginas do spoke são o local em que os usuários executam tarefas. Dois tipos de páginas spoke são páginas de tarefas e páginas de formulário: as páginas de tarefas apresentam uma tarefa ou uma etapa em uma tarefa com uma instrução principal baseada em tarefa específica; as páginas de formulário apresentam uma coleção de propriedades e tarefas relacionadas com base em uma instrução principal geral. Consulte também: [página de Hub](#h).

**usuário padrão**

No controle de conta de usuário, os usuários padrão têm os privilégios mínimos no computador e devem solicitar permissão de um administrador no computador para executar tarefas administrativas. Em contraste com os administradores protegidos, os usuários padrão não podem se elevar. Consulte também: [administrador elevado](#e), [administrador protegido](#p).

**texto estático**

Texto da interface do usuário que não faz parte de um controle interativo. Inclui rótulos, instruções principais, instruções complementares e explicações complementares.

**instruções complementares**

Uma forma opcional de texto de interface do usuário que adiciona informações, detalhes ou contexto à instrução principal. Consulte também: [instrução principal](#m).

**barra de ferramentas suplementar**

Uma coleção de comandos projetados para trabalhar em conjunto com uma barra de menus. Consulte também: [barra de ferramentas primária](#p).

**cor do sistema**

Uma cor definida pelo Windows para uma finalidade específica, acessada usando a API (interface de programação de aplicativo) do GetSysColor. Por exemplo, janela de cores \_ define a cor do plano de fundo da janela e a cor \_ WINDOWTEXT define a cor do texto da janela. As cores do sistema não são tão ricas quanto as cores do tema. Consulte também: [cor do tema](#t).

**menu do sistema**

Uma coleção de comandos de janela básicos, como mover, dimensionar, maximizar, minimizar e fechar, disponível no ícone do programa na barra de título ou clicando com o botão direito do mouse em um botão da barra de tarefas.

## <a name="t"></a>T

**caixa de diálogo com guias**

Uma caixa de diálogo que contém informações relacionadas em páginas rotuladas separadas (guias). Ao contrário das folhas de propriedades, que geralmente contêm guias, as caixas de diálogo com guias não são usadas para exibir as propriedades de um objeto. Consulte também: [Propriedades](#p).

**agenda**

Uma unidade de atividade do usuário, geralmente representada por uma única superfície de interface do usuário (como uma caixa de diálogo) ou uma sequência de páginas (como um assistente).

**caixa de diálogo tarefa**

Uma caixa de diálogo implementada usando a API da caixa de diálogo de tarefa. Requer o Windows Vista ou posterior.

**fluxo de tarefas**

Uma sequência de páginas que ajuda os usuários a executarem uma tarefa, seja em um assistente, Explorer ou navegador.

**link de tarefa**

Um link usado para iniciar uma tarefa, em contraste com os links que navegam para outras páginas ou janelas, escolha opções ou exiba a ajuda.

**painel de tarefas**

Um tipo de interface do usuário semelhante a uma caixa de diálogo, exceto que é apresentada dentro de um painel de janela em vez de uma janela separada. Como resultado, os painéis de tarefas têm uma ideia mais direta e contextual do que as caixas de diálogo. Um painel de tarefas pode conter um menu para fornecer ao usuário um pequeno conjunto de comandos relacionados ao modo de programa ou objeto selecionado.

**na**

O ponto de acesso para executar programas que têm uma presença na área de trabalho. Os usuários interagem com controles chamados de botões da barra de tarefas para mostrar, ocultar e minimizar janelas de programas.

**caixa de texto**

Um controle projetado especificamente para entrada textual; permite que os usuários exibam, insiram ou editem texto ou números.

**cor do tema**

Uma cor definida pelo Windows para uma finalidade específica, acessada usando a API GetThemeColor juntamente com partes, Estados e cores. Por exemplo, a parte do Windows define uma FillColor e uma TextColor. As cores do tema são mais sofisticadas do que as cores do sistema, mas exigem que o serviço tema esteja em execução. Consulte também: [cor do sistema](#s).

**capitalização de estilo de título**

Para capitalização de estilo de título:

-   Capitalizar todos os substantivos, verbos (incluindo is e outras formas de serem), advérbios (incluindo e quando), adjetivos (incluindo isso e isso) e prosubstantivos (incluindo seu).
-   Aproveite a primeira e a última palavra, independentemente de suas partes de fala (por exemplo, o texto a ser procurado).
-   Colocar em maiúsculas as preposições que fazem parte de uma frase verbal (por exemplo, fazendo backup do disco).
-   Não coloque os artigos em maiúsculas (a, a, a), a menos que o artigo seja a primeira palavra no título.
-   Não coloque em maiúsculas as conjunçãos de coordenadas (e, mas, para, nem ou), a menos que a conjunção seja a primeira palavra no título.
-   Não coloque as preposições em maiúsculas de quatro ou menos letras, a menos que a preposição seja a primeira palavra no título.
-   Não coloque em maiúsculas em uma frase infinita (por exemplo, como formatar seu disco rígido), a menos que a frase seja a primeira palavra no título.
-   Coloque em maiúscula a segunda palavra em palavras compostas se for um substantivo ou um adjetivo apropriado, "e-Word" ou as palavras têm o peso igual (por exemplo, comércio eletrônico, referência cruzada, software anterior da Microsoft, acesso de leitura/gravação, tempo de execução). Não coloque a segunda palavra em maiúscula, se ela for outra parte da fala, como uma preposição ou outra palavra secundária (por exemplo, suplemento, como fazer).
-   Aproveite a interface do usuário e os termos da interface de programação de aplicativos que normalmente você não colocaria em maiúsculas, a menos que eles diferenciam maiúsculas de minúsculas (por exemplo, o comando fdisk). Siga a capitalização tradicional de palavras-chave e outros termos especiais em linguagens de programação (por exemplo, a função printf, usando as diretivas uniforme e ALIGN).
-   Colocar somente a primeira palavra de cada título de coluna.

**barra**

Uma apresentação gráfica de comandos otimizados para acesso eficiente.

**dessa**

Uma pequena janela pop-up que rotula o controle sem rótulo que está sendo apontado, como controles de barra de ferramentas ou botões de comando sem rótulo.

**Tom**

Interação direta com uma exibição de computador usando um dedo.

## <a name="u"></a>U

**estudo de usabilidade**

Uma técnica de pesquisa que ajuda você a melhorar sua experiência de usuário testando seu design de interface do usuário e coletando comentários de usuários de destino reais. Estudos de usabilidade podem variar desde técnicas formais em configurações, como laboratórios de usabilidade, para as técnicas informais em configurações como o próprio escritório do usuário. Mas as constantes desses estudos são: capturando informações dos participantes; avaliando essas informações para tendências e padrões significativos; e, por fim, implementar alterações lógicas que resolvem os problemas identificados no estudo.

**Controle de conta de usuário**

Com o controle de conta de usuário (ou o UAC, anteriormente conhecido como "conta de usuário com privilégios mínimos," ou LUA) habilitado, os administradores interativos normalmente são executados com privilégios mínimos de usuário, mas podem se elevar automaticamente para executar tarefas administrativas, fornecendo consentimento explícito com a interface do usuário de consentimento. Essas tarefas administrativas incluem a instalação de software e drivers, a alteração de configurações de todo o sistema, a exibição ou a alteração de outras contas de usuário e a execução de ferramentas administrativas.

**Escudo de controle de conta de usuário**

Um ícone de escudo usado para indicar que um comando ou opção precisa de elevação para o controle de conta de usuário.

**problema de entrada do usuário**

Um erro resultante da entrada do usuário. Normalmente, os problemas de entrada do usuário não são críticos, pois eles devem ser corrigidos antes de continuar.

**cenário do usuário**

Uma descrição de uma meta de usuário, um problema ou uma tarefa em um conjunto específico de circunstâncias.

## <a name="v"></a>V

## <a name="w"></a>W

**warning**

Uma mensagem que descreve uma condição que pode causar um problema no futuro. Avisos não são erros ou perguntas. No Windows Vista e posterior, as mensagens de aviso são normalmente exibidas em caixas de diálogo de tarefas, incluem uma instrução Main clara e concisa, e geralmente incluem um ícone de aviso padrão para reforço visual do texto.

**Página inicial**

A primeira página de um assistente, usada para explicar a finalidade do assistente. As páginas de boas-vindas não são mais recomendadas. Os usuários têm uma experiência mais eficiente sem essas páginas.

**Window**

Uma área retangular em uma tela de computador em que os programas e o conteúdo são exibidos. Uma janela pode ser movida, redimensionada, minimizada ou fechada; Ele pode sobrepor outras janelas. O encaixe de uma janela filho a converte em um painel. Consulte também: [painel](#p).

**Tecla de logotipo do Windows**

Uma tecla modificadora com o logotipo do Windows. Essa chave é usada para vários atalhos do Windows e é reservada para uso do Windows. Por exemplo, pressionar a tecla do logotipo do Windows exibe ou oculta o menu Iniciar do Windows.

**wireframes**

Uma interface do usuário modelo que mostra a funcionalidade e o layout de uma janela, mas não sua aparência concluída. Um wireframe usa apenas segmentos de linha, controles e texto, sem cor, elementos gráficos complexos ou o uso de temas.

**para**

Uma sequência de páginas que orienta os usuários por meio de uma tarefa de várias etapas executada com pouca frequência. Os assistentes efetivos reduzem o conhecimento necessário para executar a tarefa em comparação com as UIs alternativas.

**área de trabalho**

A área na tela onde os usuários podem realizar seu trabalho, bem como armazenar programas, documentos e seus atalhos. Consulte também: [Desktop](#d).
