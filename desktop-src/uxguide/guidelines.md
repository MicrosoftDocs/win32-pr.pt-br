# <a name="guidelines"></a>Diretrizes

> [!NOTE]
> Este guia de design foi criado para o Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte da orientação ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais](/windows/uwp/design/).

Essas seções compreendem as diretrizes de experiência do usuário detalhadas para aplicativos de área de trabalho baseados no Windows.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                     | Descrição                                                                                                                                                                                                                                                                                      |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Controles](controls.md)<br/>                       | Os controles são elementos da interface do usuário com os quais seus usuários interagem na área principal da janela do seu aplicativo. Consulte exemplos visuais de controles em aplicativos de área de trabalho baseados no Windows e obtenha links para diretrizes para cada controle.<br/>                                                                                   |
| [Comandos](commands.md)<br/>                       | Os comandos são ações que os usuários podem executar ao usar seu aplicativo. Conheça as diretrizes para adicionar comandos aos menus, às faixas de ferramentas e às barras de ferramentas do seu aplicativo.<br/>                                                                                                                                        |
| [Text](text.md)<br/>                               | O texto inclui qualquer texto que os usuários possam ver em seu aplicativo. Examine essas diretrizes sobre o uso de texto, estilo e Tom da interface do usuário para seus aplicativos. <br/>                                                                                                                                                      |
| [Mensagens](messages.md)<br/>                       | As mensagens são qualquer tipo de mensagem que os usuários precisam ou desejam ver à medida que usam seu aplicativo. Saiba como apresentar erros, avisos, confirmações e notificações em seu aplicativo.<br/>                                                                                                                  |
| [Interação](interaction.md)<br/>                 | A interação é a variedade de maneiras como os usuários interagem com seu aplicativo, incluindo toque, teclado, mouse e assim por diante. Use essas diretrizes para criar experiências intuitivas e informativas que são otimizadas para toque, mas funcionam consistentemente entre dispositivos de entrada.<br/>                           |
| [Windows](windows.md)<br/>                         | As janelas são as principais "telas" ou as superfícies de interface do usuário do seu aplicativo de área de trabalho, incluindo o próprio Windows e pop-ups, caixas de diálogo e assistentes principais. Siga estas diretrizes ao decidir qual superfície usar e a melhor maneira de usá-las.<br/>                                                         |
| [Visuais](visuals.md)<br/>                         | Os visuais incluem os elementos visuais que não são os controles. Essas diretrizes ajudam você a tomar decisões sobre layout, fontes, cor, ícones e assim por diante em seu aplicativo.<br/>                                                                                                                       |
| [Experiências](experiences.md)<br/>                 | As experiências são as experiências comuns e os casos de uso para todos os aplicativos, como configurar, executar primeiro e imprimir. Saiba mais sobre as práticas recomendadas para a criação dessas experiências e a comunicação da marca do aplicativo.<br/>                                                                              |
| [Ambiente do Windows](windows-environment.md)<br/> | O ambiente do Windows é a área de trabalho na tela fornecida pelo Windows, análoga a um desktop físico e os pontos de extensão principais do sistema operacional. Saiba como aproveitar a área de trabalho, a barra de tarefas, a área de notificação, os painéis de controle, a ajuda e o controle de conta de usuário para seu aplicativo.<br/> |
