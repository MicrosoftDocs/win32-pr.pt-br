# <a name="guidelines"></a>Diretrizes

> [!NOTE]
> este guia de design foi criado para o Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte da orientação ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais](/windows/uwp/design/).

essas seções compreendem as diretrizes de experiência do usuário detalhadas para aplicativos de área de trabalho baseados em Windows.

| [Text](text.md)<br/>                               | O texto inclui qualquer texto que os usuários possam ver em seu aplicativo. | [Mensagens](messages.md)<br/>                       | As mensagens são qualquer tipo de mensagem que os usuários precisam ou desejam ver à medida que usam seu aplicativo. | [Windows](windows.md)<br/>                         | Windows são as principais "telas" ou as superfícies de interface do usuário do seu aplicativo de desktop, incluindo o próprio Windows e pop-ups, caixas de diálogo e assistentes principais. | [Visuais](visuals.md)<br/>                         | Os visuais incluem os elementos visuais que não são os controles.
