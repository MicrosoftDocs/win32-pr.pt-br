# <a name="messages-design-basics"></a>Mensagens (noções básicas de design)

> [!NOTE]
> Este guia de design foi criado para Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte das diretrizes ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais.](/windows/uwp/design/)

As mensagens são qualquer tipo de mensagem que os usuários precisam ou querem ver enquanto usam seu aplicativo. Saiba como apresentar erros, avisos, confirmações e notificações em seu aplicativo.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                        | Descrição                                                                                                                                                                                                                                                                                                                                     |
|----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Mensagens de erro](mess-error.md)<br/>  | Uma mensagem de erro alerta os usuários de um problema que já ocorreu. Por outro lado, uma mensagem de aviso alerta os usuários de uma condição que pode causar um problema no futuro. As mensagens de erro podem ser apresentadas usando caixas de diálogo modais, mensagens in-locar, notificações ou balão.<br/>                                                  |
| [Mensagens de aviso](mess-warn.md)<br/> | Uma mensagem de aviso é uma caixa de diálogo modal, mensagem in-place, notificação ou balão que alerta o usuário de uma condição que pode causar um problema no futuro.<br/>                                                                                                                                                                    |
| [Confirmações](mess-confirm.md)<br/> | Uma confirmação é uma caixa de diálogo modal que pergunta se o usuário deseja continuar com uma ação.<br/>                                                                                                                                                                                                                                          |
| [Notificações](mess-notif.md)<br/>   | Uma notificação informa os usuários sobre eventos que não estão relacionados à atividade do usuário atual, exibindo brevemente um balão de um ícone na área de notificação. A notificação pode resultar de uma ação do usuário ou evento significativo do sistema ou pode oferecer informações potencialmente úteis do Microsoft Windows ou de um aplicativo.<br/> |
