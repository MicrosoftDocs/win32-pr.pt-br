# <a name="experiences"></a>Experiências

> [!NOTE]
> Este guia de design foi criado para o Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte da orientação ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais](/windows/uwp/design/).

As experiências são as experiências comuns e os casos de uso para todos os aplicativos, como configurar, executar primeiro e imprimir. Saiba mais sobre as práticas recomendadas para a criação dessas experiências e a comunicação da marca do aplicativo.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                | Descrição                                                                                                                                                                                                                        |
|------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Identidade visual do software](exper-branding.md)<br/>   | A identidade visual é o posicionamento emocional de um produto, conforme percebido por seus clientes. A identidade visual bem-sucedida requer a criação de Skillful de uma imagem de produto e não é obtida apenas por meio de logotipos de produto e esquemas de cores. <br/> |
| [Instalação](exper-setup.md)<br/>                  | Os usuários não gostam de instalar o software, portanto, as experiências de instalação modernas precisam ser simples, eficientes e sem problemas. <br/>                                                                                                     |
| [Primeira experiência](exper-first-exper.md)<br/> | Na primeira experiência ideal, os usuários instalam seu programa e o utilizam de forma produtiva, sem responder a várias perguntas ou aprender uma série de coisas. <br/>                                                   |
| [Impressão](exper-printing.md)<br/>            | A impressão é a experiência do usuário em papel. É fácil de ignorar, mas é uma parte importante da experiência geral do usuário. <br/>                                                                                           |
