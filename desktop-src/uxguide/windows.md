# <a name="windows-design-basics"></a>Windows (noções básicas de design)

> [!NOTE]
> Este guia de design foi criado para Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte das diretrizes ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais.](/windows/uwp/design/)

Windows são as principais "telas" ou superfícies de interface do usuário do seu aplicativo da área de trabalho, incluindo as janelas principais e pop-ups, caixas de diálogo e assistentes. Siga estas diretrizes ao decidir qual superfície usar e a melhor maneira de usá-las.
