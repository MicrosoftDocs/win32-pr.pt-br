# <a name="windows-environment"></a>Ambiente do Windows

> [!NOTE]
> Este guia de design foi criado para o Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte da orientação ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais](/windows/uwp/design/).

O ambiente do Windows é a área de trabalho na tela fornecida pelo Windows, análoga a um desktop físico e os pontos de extensão principais do sistema operacional. Saiba como aproveitar a área de trabalho, a barra de tarefas, a área de notificação, os painéis de controle, a ajuda e o controle de conta de usuário para seu aplicativo.
