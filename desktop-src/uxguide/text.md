# <a name="text-design-basics"></a>Texto (noções básicas de Design)

> [!NOTE]
> Este guia de design foi criado para o Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte da orientação ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais](/windows/uwp/design/).

O texto inclui qualquer texto que os usuários possam ver em seu aplicativo. Examine essas diretrizes sobre o uso de texto, estilo e Tom da interface do usuário para seus aplicativos.

O Tom destina-se a criar uma resposta ou uma emoção específica no leitor; Ele cria uma personalidade que pode envolver ou repelir os usuários.<br/> |
