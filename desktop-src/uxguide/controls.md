# <a name="controls-design-basics"></a>Controles (noções básicas de design)

> [!NOTE]
> Este guia de design foi criado para Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte das diretrizes ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais.](/windows/uwp/design/)

Os controles são elementos de interface do usuário com os que os usuários interagem na área da janela principal do aplicativo. Confira exemplos visuais de controles Windows aplicativos da área de trabalho baseados em aplicativos e obter links para diretrizes para cada controle.



|          Exemplos                                                                                                                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ![Ballon](images/controls-image1.png)<br/> [Os balão](ctrl-balloons.md) informam os usuários sobre um problema não crítico ou uma condição especial em um controle.<br/>                                                                                                                                                                                                                 |
| ![Checkbox](images/controls-image2.png)<br/> [As caixas de](ctrl-check-boxes.md) seleção permitem que os usuários façam uma decisão entre duas ou mais opções claramente diferentes.<br/>                                                                                                                                                                                                     |
| ![botão de comando](images/controls-image3.png)<br/> [Os botões de comando](ctrl-command-buttons.md) permitem que os usuários executem uma ação imediata.<br/>                                                                                                                                                                                                                         |
| ![link de comando](images/controls-image4.png)<br/> [Os links de](ctrl-command-links.md) comando permitem que os usuários façam uma escolha entre um conjunto de opções mutuamente exclusivas e relacionadas.<br/>                                                                                                                                                                                          |
| ![caixa de combinação e lista de combinação](images/controls-image5.png)<br/> [As listas listadas e as caixas de combinação](/windows/desktop/uxguide/ctrl-drop) permitem que os usuários façam uma escolha entre uma lista de valores mutuamente exclusivos.<br/>                                                                                                                                                                           |
| ![caixa group](images/controls-image6.png)<br/> [As caixas de](ctrl-group-boxes.md) grupo permitem que os usuários vejam relações entre um conjunto de controles relacionados.<br/>                                                                                                                                                                                                                |
| ![Link](images/controls-image7.png)<br/> [Os links](ctrl-links.md) permitem que os usuários naveguem para outra página, janela ou tópico de Ajuda; exibir uma definição; iniciar um comando; ou escolha uma opção.<br/>                                                                                                                                                                    |
| ![caixa de listagem](images/controls-image8.png)<br/> [As caixas de](ctrl-list-boxes.md) listagem permitem que os usuários selecionem em um conjunto de valores apresentados em uma lista que está sempre visível. Com uma caixa de listagem de seleção única, os usuários selecionam um item em uma lista de valores mutuamente exclusivos. Com uma caixa de listagem de seleção múltipla, os usuários selecionam zero ou mais itens em uma lista de valores.<br/> |
| ![exibição de lista](images/controls-image9.png)<br/> [Exibições de](ctrl-list-views.md) lista permitem que os usuários vejam e interajam com uma coleção de objetos de dados, usando seleção única ou seleção múltipla.<br/>                                                                                                                                                           |
| ![Notificação](images/controls-image10.png)<br/> [As notificações](mess-notif.md) informam os usuários sobre eventos que não estão relacionados à atividade do usuário atual.<br/>                                                                                                                                                                                                          |
| ![barra de progresso](images/controls-image11.png)<br/> [As barras de](progress-bars.md) progresso permitem que os usuários acompanhem o progresso de uma operação demorada.<br/>                                                                                                                                                                                                                    |
| ![divulgação progressiva](images/controls-image12.png)<br/> [Controles de divulgação](ctrl-progressive-disclosure-controls.md) progressiva permitem que os usuários mostrem ou o hideem informações adicionais, incluindo dados, opções ou comandos.<br/>                                                                                                                                   |
| ![botão de rádio](images/controls-image13.png)<br/> [Os botões de opção](ctrl-radio-buttons.md) permitem que os usuários façam uma escolha entre um conjunto de opções mutuamente exclusivas e relacionadas.<br/>                                                                                                                                                                                         |
| ![caixa de pesquisa](images/controls-image14.png)<br/> [As caixas de](ctrl-search-boxes.md) pesquisa fornecem aos usuários uma maneira de localizar objetos específicos ou texto rapidamente.<br/>                                                                                                                                                                                                              |
| ![Controle deslizante](images/controls-image15.png)<br/> [Os controles](ctrl-sliders.md) deslizantes permitem que os usuários escolham entre um intervalo contínuo de valores.<br/>                                                                                                                                                                                                                                   |
| ![Spin](images/controls-image16.png)<br/> [Controles de](ctrl-spin-controls.md) rotação permitem que os usuários alterem incrementalmente o valor dentro de sua caixa de texto numérica associada.<br/>                                                                                                                                                                                            |
| ![barra de status](images/controls-image17.png)<br/> [As barras de](ctrl-status-bars.md) status exibem informações sobre o estado da janela atual, tarefas em segundo plano ou outras informações contextuais.<br/>                                                                                                                                                                  |
| ![Guia](images/controls-image18.png)<br/> [As guias](ctrl-tabs.md) apresentam aos usuários informações relacionadas em páginas rotuladas separadas.<br/>                                                                                                                                                                                                                                   |
| ![caixa de texto](images/controls-image19.png)<br/> [As caixas de](ctrl-text-boxes.md) texto permitem que os usuários exibem, insiram ou editem um texto ou valor numérico.<br/>                                                                                                                                                                                                                    |
| ![Tooltip](images/controls-image20.png)<br/> [Dicas de ferramenta](ctrl-tooltips-and-infotips.md) rotulam um controle sem rótulo.<br/>                                                                                                                                                                                                                                                |
| ![Infotip](images/controls-image21.png)<br/> [As infotips](ctrl-tooltips-and-infotips.md) descrevem um objeto para o qual o usuário está apontando.<br/>                                                                                                                                                                                                                          |
| ![treeview](images/controls-image22.png)<br/> [As exibições de](ctrl-tree-views.md) árvore permitem que os usuários vejam e interajam com uma coleção hierárquica de objetos, usando seleção única ou seleção múltipla.<br/>                                                                                                                                        |
