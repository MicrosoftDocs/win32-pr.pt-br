# <a name="controls-design-basics"></a>Controles (noções básicas de Design)

> [!NOTE]
> Este guia de design foi criado para o Windows 7 e não foi atualizado para versões mais recentes do Windows. Grande parte da orientação ainda se aplica em princípio, mas a apresentação e os exemplos não refletem nossas [diretrizes de design atuais](/windows/uwp/design/).

Os controles são elementos da interface do usuário com os quais seus usuários interagem na área principal da janela do seu aplicativo. Consulte exemplos visuais de controles em aplicativos de área de trabalho baseados no Windows e obtenha links para diretrizes para cada controle.



|          Exemplos                                                                                                                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ![ballon](images/controls-image1.png)<br/> Os [balões](ctrl-balloons.md) informam os usuários de um problema não crítico ou uma condição especial em um controle.<br/>                                                                                                                                                                                                                 |
| ![verificação](images/controls-image2.png)<br/> As [caixas de seleção](ctrl-check-boxes.md) permitem que os usuários tomem uma decisão entre duas ou mais opções claramente diferentes.<br/>                                                                                                                                                                                                     |
| ![botão de comando](images/controls-image3.png)<br/> Os [botões de comando](ctrl-command-buttons.md) permitem que os usuários executem uma ação imediata.<br/>                                                                                                                                                                                                                         |
| ![link de comando](images/controls-image4.png)<br/> Os [links de comando](ctrl-command-links.md) permitem que os usuários façam uma escolha entre um conjunto de escolhas mutuamente exclusivas e relacionadas.<br/>                                                                                                                                                                                          |
| ![lista suspensa e caixa de combinação](images/controls-image5.png)<br/> As [listas suspensas e as caixas de combinação](/windows/desktop/uxguide/ctrl-drop) permitem que os usuários façam uma escolha entre uma lista de valores mutuamente exclusivos.<br/>                                                                                                                                                                           |
| ![caixa de grupo](images/controls-image6.png)<br/> As [caixas de grupo](ctrl-group-boxes.md) permitem que os usuários vejam as relações entre um conjunto de controles relacionados.<br/>                                                                                                                                                                                                                |
| ![criar](images/controls-image7.png)<br/> Os [links](ctrl-links.md) permitem que os usuários naveguem para outra página, janela ou tópico da ajuda; exibir uma definição; iniciar um comando; ou escolha uma opção.<br/>                                                                                                                                                                    |
| ![caixa de listagem](images/controls-image8.png)<br/> As [caixas de listagem](ctrl-list-boxes.md) permitem que os usuários selecionem um conjunto de valores apresentados em uma lista que está sempre visível. Com uma caixa de listagem de seleção única, os usuários selecionam um item de uma lista de valores mutuamente exclusivos. Com uma caixa de listagem de seleção múltipla, os usuários selecionam zero ou mais itens de uma lista de valores.<br/> |
| ![exibição de lista](images/controls-image9.png)<br/> As [exibições de lista](ctrl-list-views.md) permitem que os usuários exibam e interajam com uma coleção de objetos de dados, usando uma única seleção ou seleção múltipla.<br/>                                                                                                                                                           |
| ![Notification](images/controls-image10.png)<br/> As [notificações](mess-notif.md) informam os usuários de eventos que não estão relacionados à atividade do usuário atual.<br/>                                                                                                                                                                                                          |
| ![barra de progresso](images/controls-image11.png)<br/> As [barras de progresso](progress-bars.md) permitem que os usuários sigam o andamento de uma operação demorada.<br/>                                                                                                                                                                                                                    |
| ![divulgação progressiva](images/controls-image12.png)<br/> Os [controles de divulgação progressiva](ctrl-progressive-disclosure-controls.md) permitem que os usuários mostrem ou ocultem informações adicionais, incluindo dados, opções ou comandos.<br/>                                                                                                                                   |
| ![botão de opção](images/controls-image13.png)<br/> Os [botões de opção](ctrl-radio-buttons.md) permitem que os usuários façam uma escolha entre um conjunto de escolhas mutuamente exclusivas e relacionadas.<br/>                                                                                                                                                                                         |
| ![caixa de pesquisa](images/controls-image14.png)<br/> As [caixas de pesquisa](ctrl-search-boxes.md) fornecem aos usuários uma maneira de localizar objetos ou textos específicos rapidamente.<br/>                                                                                                                                                                                                              |
| ![classificação](images/controls-image15.png)<br/> Os [controles deslizantes](ctrl-sliders.md) permitem que os usuários escolham um intervalo contínuo de valores.<br/>                                                                                                                                                                                                                                   |
| ![ativa](images/controls-image16.png)<br/> [Controles de rotação](ctrl-spin-controls.md) permitem que os usuários mudem incrementalmente o valor dentro de sua caixa de texto numérica associada.<br/>                                                                                                                                                                                            |
| ![barra de status](images/controls-image17.png)<br/> [Barras de status](ctrl-status-bars.md) exibem informações sobre o estado da janela atual, tarefas em segundo plano ou outras informações contextuais.<br/>                                                                                                                                                                  |
| ![abas](images/controls-image18.png)<br/> As [guias](ctrl-tabs.md) apresentam aos usuários informações relacionadas em páginas rotuladas separadas.<br/>                                                                                                                                                                                                                                   |
| ![caixa de texto](images/controls-image19.png)<br/> As [caixas de texto](ctrl-text-boxes.md) permitem que os usuários exibam, insiram ou editem um texto ou valor numérico.<br/>                                                                                                                                                                                                                    |
| ![dessa](images/controls-image20.png)<br/> [Dicas de ferramentas](ctrl-tooltips-and-infotips.md) rotulam um controle sem rótulo.<br/>                                                                                                                                                                                                                                                |
| ![InfoTip](images/controls-image21.png)<br/> [Infotips](ctrl-tooltips-and-infotips.md) descreve um objeto ao qual o usuário está apontando.<br/>                                                                                                                                                                                                                          |
| ![treeview](images/controls-image22.png)<br/> As [exibições de árvore](ctrl-tree-views.md) permitem que os usuários exibam e interajam com uma coleção hierarquicamente organizada de objetos, usando uma única seleção ou seleção múltipla.<br/>                                                                                                                                        |
