# <a name="multimedia-audio"></a>Áudio de multimídia

O Windows fornece funções que permitem que um aplicativo adicione serviços de áudio. As funções de áudio de multimídia foram introduzidas no Windows de 16 bits. Muitos de seus recursos foram substituídos por recursos do DirectX. Para obter mais informações, consulte a documentação do DirectX.

Os tópicos a seguir são discutidos na documentação de áudio de multimídia.

-   [Gerenciador de compactação de áudio](audio-compression-manager.md)
-   [Mixers de áudio](audio-mixers.md)
-   [MIDI (interface digital de instrumento musical)](musical-instrument-digital-interface--midi.md)
-   [Áudio de onda](waveform-audio.md)

O áudio também pode ser reproduzido usando a interface de controle de mídia. Para obter mais informações, consulte [MCI](mci.md).
