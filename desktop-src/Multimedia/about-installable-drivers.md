# <a name="about-installable-drivers"></a>Sobre drivers instaláveis

Um driver instanciável é uma DLL (biblioteca de vínculo dinâmico) que fornece uma interface padrão por meio da qual aplicativos baseados em Windows e DLLs se comunicam com o driver e gerenciam o driver. Drivers instaláveis são usados mais comumente para como drivers de dispositivo multimídia, mas drivers instanciados também podem ser usados para outras finalidades e são particularmente úteis em situações que exigem uma interface padrão e controle de várias instâncias.
