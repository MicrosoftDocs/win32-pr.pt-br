# <a name="windows-multimedia"></a>Multimídia do Windows

## <a name="purpose"></a>Finalidade

O suporte a multimídia do Microsoft Windows permite que os aplicativos usem som e vídeo.

## <a name="where-applicable"></a>Quando aplicável

O multimídia do Windows pode ser usado em todos os aplicativos baseados no Windows.

## <a name="developer-audience"></a>Público de desenvolvedores

O multimídia do Windows foi projetado para ser usado por programadores C/C++. É necessário ter familiaridade com a interface gráfica do usuário do Windows e com a arquitetura controlada por mensagem.
