# <a name="windows-multimedia"></a>Windows Multimídia

## <a name="purpose"></a>Finalidade

O suporte Windows multimídia da Microsoft permite que os aplicativos usem som e vídeo.

## <a name="where-applicable"></a>Quando aplicável

Windows multimídia pode ser usada em todos os Windows baseados em aplicativos.

## <a name="developer-audience"></a>Público de desenvolvedores

Windows multimídia foi projetada para uso por programadores C/C++. Familiaridade com a interface Windows gráfica do usuário e a arquitetura controlada por mensagens é necessária.
