# <a name="the-wait-notify-and-test-flags"></a>Os sinalizadores de espera, notificação e teste

A maioria dos comandos MCI inclui sinalizadores que modificam o comando. Os sinalizadores "espera" ( \_ espera de MCI) e "notificar" ( \_ notificação MCI) são comuns a todos os comandos. O sinalizador "teste" ( \_ teste MCI) está disponível para dispositivos de vídeo digital e VCR. Os tópicos a seguir descrevem o uso desses sinalizadores.

-   [O sinalizador de espera](the-wait-flag.md)
-   [O sinalizador de notificação](the-notify-flag.md)
-   [O sinalizador de teste](the-test-flag.md)
