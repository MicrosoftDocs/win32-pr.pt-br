# <a name="level-1-optical-storage"></a>Armazenamento óptico de nível 1

O dispositivo de armazenamento óptico de desempenho mínimo é uma unidade de CD-ROM que atende aos seguintes critérios:

-   Uma taxa de transferência de dados sustentada de 150 quilobytes por segundo
-   Um uso de largura de banda de CPU de 40% ou menos ao manter uma taxa de transferência de dados sustentada de 150 quilobytes por segundo
-   Um tempo médio de busca de 1 segundo ou menos
