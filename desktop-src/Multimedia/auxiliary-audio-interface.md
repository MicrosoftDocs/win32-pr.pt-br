# <a name="auxiliary-audio-interface"></a>Interface de áudio auxiliar

Dispositivos de áudio auxiliares são dispositivos de áudio cuja saída é misturada com os dispositivos MIDI e waveform-audio output em um computador multimídia. Um exemplo de um dispositivo de áudio auxiliar é a saída de áudio cd de uma unidade CD-ROM.

-   [Consultando dispositivos de áudio auxiliares](querying-auxiliary-audio-devices.md)
-   [Alterando o volume de dispositivos de áudio auxiliares](changing-the-volume-of-auxiliary-audio-devices.md)
