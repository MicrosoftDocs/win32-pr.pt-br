# <a name="auxiliary-audio-interface"></a>Interface de áudio auxiliar

Os dispositivos de áudio auxiliares são dispositivos de áudio cuja saída é misturada com os dispositivos de saída MIDI e Wave-Audio em um computador multimídia. Um exemplo de um dispositivo de áudio auxiliar é a saída de áudio de CD de uma unidade de CD-ROM.

-   [Consultando dispositivos de áudio auxiliares](querying-auxiliary-audio-devices.md)
-   [Alterando o volume de dispositivos de áudio auxiliares](changing-the-volume-of-auxiliary-audio-devices.md)
