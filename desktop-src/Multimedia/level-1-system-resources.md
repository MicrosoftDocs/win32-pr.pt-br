# <a name="level-1-system-resources"></a>Recursos do sistema de nível 1

A configuração mínima para um PC atender à especificação de Nível 1 inclui os seguintes itens:

-   Um microprocessador 386SX com 2 MB (megabytes) de RAM (memória de acesso aleatório)
-   Uma unidade de disquete de alta densidade de 3,5 polegadas
-   Uma unidade de disco rígido com pelo menos 30 MB de espaço em disco
-   Um monitor de cores com uma resolução de exibição de 640 por 480 pixels com 16 cores
-   Software do sistema que oferece compatibilidade binária com o microsoft Windows do sistema operacional versão 3.0
