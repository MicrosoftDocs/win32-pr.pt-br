# <a name="video-for-windows"></a>Vídeo para Windows

O Video for Windows (VFW) fornece funções que permitem que um aplicativo processe dados de vídeo. O VFW foi introduzido no Windows de 16 bits. Muitos de seus recursos foram substituídos por recursos do DirectX. Para obter mais informações, consulte a documentação do DirectX.

Os tópicos a seguir são discutidos na documentação do vídeo para Windows.

-   [Funções e macros do AVIFile](avifile-functions-and-macros.md)
-   [Gerenciador de compactação de vídeo](video-compression-manager.md)
-   [Captura de vídeo](video-capture.md)
-   [Manipuladores de arquivo e fluxo personalizados](custom-file-and-stream-handlers.md)
-   [DrawDib](drawdib.md)
