# <a name="video-for-windows"></a>Vídeo para Windows

o vídeo para Windows (VFW) fornece funções que permitem que um aplicativo processe dados de vídeo. O VFW foi introduzido em Windows de 16 bits. Muitos de seus recursos foram substituídos por recursos do DirectX. Para obter mais informações, consulte a documentação do DirectX.
