# <a name="mciwnd-window-class"></a>Classe de janela MCIWnd

MCIWnd é uma classe de janela para controlar dispositivos multimídia. Uma biblioteca de funções, mensagens e macros associadas ao MCIWnd fornece um método simples para adicionar recursos de reprodução de multimídia ou de gravação aos seus aplicativos.
