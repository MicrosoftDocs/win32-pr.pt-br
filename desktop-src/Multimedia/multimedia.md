# <a name="multimedia"></a>Multimídia

Este glossário contém definições para termos usados na documentação de multimídia do Windows.

<dl> <dt>

<span id="_win32_adaptive_differential_pulse_code_modulation_adpcm_gloss"></span><span id="_WIN32_ADAPTIVE_DIFFERENTIAL_PULSE_CODE_MODULATION_ADPCM_GLOSS"></span>**Modulação de código de pulso diferencial adaptável (ADPCM)**
</dt> <dd>

Uma técnica de compactação de áudio.

</dd> <dt>

<span id="_win32_adpcm_gloss"></span><span id="_WIN32_ADPCM_GLOSS"></span>**ADPCM**
</dt> <dd>

Consulte [modulação diferencial de código de pulso (ADPCM)](/windows).

</dd> <dt>

<span id="_win32_break_key_gloss"></span><span id="_WIN32_BREAK_KEY_GLOSS"></span>**chave de interrupção**
</dt> <dd>

Na interface de controle de mídia (MCI), um pressionamento de tecla que interrompe uma operação de espera. Por padrão, o MCI define essa chave como CTRL + BREAK. Um aplicativo pode redefinir essa chave usando a mensagem de \_ comando de interrupção MCI.

</dd> <dt>

<span id="_win32_cd_rom_extended_architecture_cd_xa_gloss"></span><span id="_WIN32_CD_ROM_EXTENDED_ARCHITECTURE_CD_XA_GLOSS"></span>**Arquitetura estendida de CD-ROM (CD-XA)**
</dt> <dd>

Uma extensão do padrão de CD-ROM que fornece armazenamento de dados de áudio compactados junto com outros dados em um CD. Esse padrão também define a maneira como os dados são lidos de um disco. Os sinais de áudio são combinados com dados de texto e gráficos em uma única faixa para que possam ser lidos praticamente ao mesmo tempo.

</dd> <dt>

<span id="_win32_channel_gloss"></span><span id="_WIN32_CHANNEL_GLOSS"></span>**canal**
</dt> <dd>

Um método, fornecido pela MIDI (interface digital de instrumento musical), para enviar mensagens para um dispositivo individual dentro de uma instalação de MIDI. Há 16 números de canal de MIDI. Os dispositivos em uma instalação de MIDI podem ser direcionados para responder somente a mensagens marcadas com um número de canal específico para o dispositivo.

</dd> <dt>

<span id="_win32_channel_map_gloss"></span><span id="_WIN32_CHANNEL_MAP_GLOSS"></span>**mapa de canal**
</dt> <dd>

Um mapa de canal, fornecido pelo mapeador de MIDI, que pode redirecionar mensagens MIDI (interface digital de instrumento musical) de um canal para outro. Consulte também [mapeador de Midi](/windows), [MIDI (interface digital de instrumento musical)](/windows).

</dd> <dt>

<span id="_win32_chunk_gloss"></span><span id="_WIN32_CHUNK_GLOSS"></span>**parte**
</dt> <dd>

O bloco de construção básico de um arquivo de formato de arquivo de intercâmbio de recursos (RIFF), que consiste em um identificador (chamado de identificador de parte), uma variável de tamanho de bloco e uma área de dados de parte de tamanho variável.

</dd> <dt>

<span id="_win32_command_message_gloss"></span><span id="_WIN32_COMMAND_MESSAGE_GLOSS"></span>**mensagem de comando**
</dt> <dd>

Na interface de controle de mídia (MCI), uma constante simbólica que representa um comando exclusivo para um dispositivo MCI. As mensagens de comando têm estruturas de dados associadas que fornecem informações que um dispositivo exige para realizar uma solicitação.

</dd> <dt>

<span id="_win32_command_string_gloss"></span><span id="_WIN32_COMMAND_STRING_GLOSS"></span>**cadeia de caracteres de comando**
</dt> <dd>

Na interface de controle de mídia (MCI), uma cadeia de caracteres terminada em nulo que representa um comando para um dispositivo MCI. A cadeia de texto contém todas as informações de que um dispositivo MCI precisa para realizar uma solicitação. O MCI analisa a cadeia de texto e a converte em uma mensagem de comando equivalente e na estrutura de dados que ela envia para um driver de dispositivo MCI.

</dd> <dt>

<span id="_win32_compact_disc_digital_audio_cd_da_gloss"></span><span id="_WIN32_COMPACT_DISC_DIGITAL_AUDIO_CD_DA_GLOSS"></span>CD **– áudio digital (CD-da)**
</dt> <dd>

Um formato de armazenamento de dados óptico que fornece o armazenamento de até 73 minutos de dados de áudio digital de alta qualidade em um CD. Também conhecido como áudio de livro vermelho.

</dd> <dt>

<span id="_win32_compact_disc_read_only_memory_cd_rom_gloss"></span><span id="_WIN32_COMPACT_DISC_READ_ONLY_MEMORY_CD_ROM_GLOSS"></span>**disco compacto – memória somente leitura (CD-ROM)**
</dt> <dd>

Uma tecnologia de armazenamento de dados óptico que permite que grandes quantidades de dados sejam armazenadas em um CD.

</dd> <dt>

<span id="_win32_compound_device_gloss"></span><span id="_WIN32_COMPOUND_DEVICE_GLOSS"></span>**dispositivo composto**
</dt> <dd>

Um dispositivo MCI (interface de controle de mídia) que requer um elemento de dispositivo, geralmente um arquivo de dados. Um exemplo de um dispositivo composto é o driver de áudio MCI de onda. Consulte também [elemento Device](/windows).

</dd> <dt>

<span id="_win32_control_change_gloss"></span><span id="_WIN32_CONTROL_CHANGE_GLOSS"></span>**alteração de controle**
</dt> <dd>

Consulte [controle MIDI – alterar mensagem](/windows).

</dd> <dt>

<span id="_win32_device_element_gloss"></span><span id="_WIN32_DEVICE_ELEMENT_GLOSS"></span>**elemento de dispositivo**
</dt> <dd>

Dados necessários para a operação dos dispositivos compostos da interface de controle de mídia (MCI). O elemento de dispositivo geralmente é um arquivo de dados de entrada ou saída.

</dd> <dt>

<span id="_win32_division_type_gloss"></span><span id="_WIN32_DIVISION_TYPE_GLOSS"></span>**tipo de divisão**
</dt> <dd>

A técnica usada para representar o tempo entre os eventos MIDI (interface digital de instrumentos musicais) em um sequenciador MIDI.

</dd> <dt>

<span id="_win32_file_element_gloss"></span><span id="_WIN32_FILE_ELEMENT_GLOSS"></span>**elemento file**
</dt> <dd>

Um arquivo completo contido em um arquivo composto de formato de arquivo de intercâmbio de recursos (RIFF).

</dd> <dt>

<span id="_win32_fm_synthesizer_gloss"></span><span id="_WIN32_FM_SYNTHESIZER_GLOSS"></span>**Sintetizador FM**
</dt> <dd>

Consulte [sintetizador de freqüência de modulação (FM)](/windows).

</dd> <dt>

<span id="_win32_fourcc_four_character_code_gloss"></span><span id="_WIN32_FOURCC_FOUR_CHARACTER_CODE_GLOSS"></span>**FOURCC (código de quatro caracteres)**
</dt> <dd>

Um código usado para identificar as partes do formato de arquivo de intercâmbio de recursos (RIFF). Um FOURCC é uma quantidade de 32 bits representada como uma sequência de um a quatro caracteres alfanuméricos ASCII, preenchidos à direita com caracteres em branco.

</dd> <dt>

<span id="_win32_frequency_modulation_fm_synthesizer_gloss"></span><span id="_WIN32_FREQUENCY_MODULATION_FM_SYNTHESIZER_GLOSS"></span>**sintetizador FM (modulação de frequência)**
</dt> <dd>

Um sintetizador que cria sons combinando a saída de osciladores digital usando uma técnica de modulação de frequência.

</dd> <dt>

<span id="_win32_general_midi_gloss"></span><span id="_WIN32_GENERAL_MIDI_GLOSS"></span>**MIDI geral**
</dt> <dd>

Uma especificação do sintetizador criada pela MMA (Associação de fabricantes de MIDI) que define uma configuração comum e um conjunto de recursos para sintetizadores de MIDI (interface digital) de instrumentos musicais do consumidor.

</dd> <dt>

<span id="_win32_hms_time_format_gloss"></span><span id="_WIN32_HMS_TIME_FORMAT_GLOSS"></span>**Formato de hora de HMS**
</dt> <dd>

Um formato de hora usado pela MCI (interface de controle de mídia) para expressar tempo em horas, minutos e segundos. O formato de hora HMS é usado principalmente por dispositivos VIDEODISC.

</dd> <dt>

<span id="_win32_ima_gloss"></span><span id="_WIN32_IMA_GLOSS"></span>**IMA**
</dt> <dd>

Consulte [ima (Associação de multimídia interativa)](/windows) e [ima (Associação de Midi internacional)](/windows).

</dd> <dt>

<span id="_win32_interactive_multimedia_association_ima_gloss"></span><span id="_WIN32_INTERACTIVE_MULTIMEDIA_ASSOCIATION_IMA_GLOSS"></span>**Ima (Associação de multimídia interativa)**
</dt> <dd>

Uma associação profissional de empresas, instituições e indivíduos envolvidos na produção e no uso de tecnologia de multimídia interativa.

</dd> <dt>

<span id="_win32_international_midi_association_ima_gloss"></span><span id="_WIN32_INTERNATIONAL_MIDI_ASSOCIATION_IMA_GLOSS"></span>**Ima (Associação de Midi internacional)**
</dt> <dd>

A organização sem fins lucrativos que circula informações sobre a especificação MIDI (interface digital de instrumento musical).

</dd> <dt>

<span id="_win32_list_chunk_gloss"></span><span id="_WIN32_LIST_CHUNK_GLOSS"></span>**Parte da lista**
</dt> <dd>

Um bloco RIFF (formato de arquivo de intercâmbio de recursos) com um identificador de parte da lista. Partes de lista contêm uma série de subpartes.

</dd> <dt>

<span id="_win32_list_type_gloss"></span><span id="_WIN32_LIST_TYPE_GLOSS"></span>**tipo de lista**
</dt> <dd>

Um FOURCC (código de quatro caracteres) que identifica o tipo de dados contidos em uma parte do formato RIFF (Data Interchange File Format) com um identificador de parte da lista. Por exemplo, uma parte de lista com um tipo de informação de lista contém uma lista de informações sobre um arquivo, como a data de criação e o autor.

</dd> <dt>

<span id="_win32_mci_gloss"></span><span id="_WIN32_MCI_GLOSS"></span>**MCI**
</dt> <dd>

Consulte [interface de controle de mídia](/windows).

</dd> <dt>

<span id="_win32_media_control_interface_mci_gloss"></span><span id="_WIN32_MEDIA_CONTROL_INTERFACE_MCI_GLOSS"></span>**MCI (interface de controle de mídia)**
</dt> <dd>

Software de controle de alto nível que fornece uma interface independente de dispositivo para dispositivos multimídia e arquivos de mídia. O MCI inclui uma interface de mensagem de comando e uma interface de cadeia de caracteres de comando.

</dd> <dt>

<span id="_win32_midi_gloss"></span><span id="_WIN32_MIDI_GLOSS"></span>**MIDI**
</dt> <dd>

Consulte [interface digital do instrumento musical](/windows).

</dd> <dt>

<span id="_win32_midi_control_change_message_gloss"></span><span id="_WIN32_MIDI_CONTROL_CHANGE_MESSAGE_GLOSS"></span>**Controle MIDI – alterar mensagem**
</dt> <dd>

Uma mensagem MIDI (interface digital de instrumento musical) enviada a um sintetizador para alterar diferentes configurações de controle do sintetizador. Um exemplo de uma mensagem de alteração de controle é a mensagem do controlador de volume, que altera o volume de um canal MIDI específico.

</dd> <dt>

<span id="_win32_midi_manufacturers_association_mma_gloss"></span><span id="_WIN32_MIDI_MANUFACTURERS_ASSOCIATION_MMA_GLOSS"></span>**Associação de fabricantes de Midi (MMA)**
</dt> <dd>

Uma organização coletiva composta por fabricantes de instrumento MIDI (interface digital de instrumentos musicais) e empresas de software MIDI. O MMA funciona com o Comitê de MIDI padrão para manter a especificação de MIDI.

</dd> <dt>

<span id="_win32_midi_mapper_gloss"></span><span id="_WIN32_MIDI_MAPPER_GLOSS"></span>**Mapeador de Midi**
</dt> <dd>

O software de sistemas Windows que modifica mensagens de saída MIDI (interface digital de instrumento musical) e redireciona-as para um dispositivo de saída MIDI usando valores armazenados em um mapa de instalação MIDI. O mapeador de MIDI pode alterar o canal de destino e o dispositivo de saída para uma mensagem, bem como modificar mensagens de alteração de programa, valores de volume e valores de chave.

</dd> <dt>

<span id="_win32_midi_mapping_gloss"></span><span id="_WIN32_MIDI_MAPPING_GLOSS"></span>**Mapeamento de Midi**
</dt> <dd>

O processo de tradução e redirecionamento de mensagens MIDI (interface digital de instrumento musical) de acordo com os dados definidos em uma configuração de mapa de MIDI.

</dd> <dt>

<span id="_win32_midi_program_change_message_gloss"></span><span id="_WIN32_MIDI_PROGRAM_CHANGE_MESSAGE_GLOSS"></span>**Programa MIDI – alterar mensagem**
</dt> <dd>

Uma mensagem MIDI (interface digital de instrumento musical) enviada a um sintetizador para alterar o patch em um canal MIDI específico.

</dd> <dt>

<span id="_win32_midi_sequence_gloss"></span><span id="_WIN32_MIDI_SEQUENCE_GLOSS"></span>**Sequência de Midi**
</dt> <dd>

Dados MIDI (interface digital de instrumento musical) que podem ser reproduzidos por um sequenciador MIDI.

</dd> <dt>

<span id="_win32_midi_sequencer_gloss"></span><span id="_WIN32_MIDI_SEQUENCER_GLOSS"></span>**Sequenciador MIDI**
</dt> <dd>

Um programa que cria ou desempenha músicas armazenadas como arquivos MIDI (interface digital de instrumento musical). Quando um sequenciador reproduz arquivos MIDI, ele envia dados de MIDI do arquivo para um sintetizador MIDI, que produz os sons. O Windows fornece um sequenciador MIDI, acessível por meio da MCI (interface de controle de mídia), que reproduz arquivos MIDI. Consulte também [MCI (interface de controle de mídia)](/windows).

</dd> <dt>

<span id="_win32_midi_setup_map_gloss"></span><span id="_WIN32_MIDI_SETUP_MAP_GLOSS"></span>**Mapa de instalação MIDI**
</dt> <dd>

Um conjunto completo de dados para o Mapeador MIDI (interface digital de instrumento musical) a ser usado ao redirecionar mensagens MIDI. Somente um mapa de instalação pode estar em vigor em um determinado momento, mas o usuário pode ter vários mapas de instalação disponíveis e pode escolher entre eles usando a opção do painel de controle do mapeador de MIDI.

</dd> <dt>

<span id="_win32_mma_gloss"></span><span id="_WIN32_MMA_GLOSS"></span>**MMA**
</dt> <dd>

Consulte [Associação de fabricantes de Midi](/windows).

</dd> <dt>

<span id="_win32_msf_time_format_gloss"></span><span id="_WIN32_MSF_TIME_FORMAT_GLOSS"></span>**Formato de hora do MSF**
</dt> <dd>

Um formato de hora usado pela MCI (interface de controle de mídia) para expressar tempo em minutos, segundos e quadros. O número de quadros em um segundo depende do tipo de dispositivo que está sendo usado; os dispositivos de áudio de CD usam 75 quadros por segundo. O formato de tempo do MSF é usado principalmente por dispositivos de áudio de CD.

</dd> <dt>

<span id="_win32_musical_instrument_digital_interface_midi_gloss"></span><span id="_WIN32_MUSICAL_INSTRUMENT_DIGITAL_INTERFACE_MIDI_GLOSS"></span>**MIDI (interface digital de instrumento musical)**
</dt> <dd>

Um protocolo padrão para comunicação entre instrumentos musicais e computadores.

</dd> <dt>

<span id="_win32_parts_per_quarter_note_ppqn_gloss"></span><span id="_WIN32_PARTS_PER_QUARTER_NOTE_PPQN_GLOSS"></span>**partes por trimestre observação (PPQN)**
</dt> <dd>

Um formato de hora usado para sequências MIDI (interface digital de instrumento musical). PPQN é o formato de hora mais comum usado com arquivos MIDI padrão.

</dd> <dt>

<span id="_win32_patch_gloss"></span><span id="_WIN32_PATCH_GLOSS"></span>**patch** do
</dt> <dd>

Uma configuração específica de um sintetizador de MIDI (interface digital de instrumento musical) que resulta em um som específico, geralmente um som simulando um instrumento musical específico. Os patches também são chamados de programas. Uma mensagem de alteração de programa MIDI altera a configuração de patch em um sintetizador. Patch também refere-se à conexão ou conexões entre dispositivos MIDI. Consulte também [MIDI (interface digital de instrumento musical)](/windows).

</dd> <dt>

<span id="_win32_patch_caching_gloss"></span><span id="_WIN32_PATCH_CACHING_GLOSS"></span>**cache de patch**
</dt> <dd>

Uma técnica que permite que alguns drivers de dispositivo do sintetizador MIDI (interface digital de instrumentos digitais) para pré-carregar seus dados de patch, reduzindo o atraso entre o momento em que o sintetizador recebe uma mensagem de alteração de programa MIDI e o momento em que ele desempenha uma observação usando o novo patch. O cache de patch também garante que os patches necessários estejam disponíveis (o sintetizador pode carregar apenas um subconjunto de seus patches). Consulte também [MIDI (interface digital de instrumento musical)](/windows), [patch](/windows).

</dd> <dt>

<span id="_win32_pitch_scale_factor_gloss"></span><span id="_WIN32_PITCH_SCALE_FACTOR_GLOSS"></span>**fator de escala de densidade**
</dt> <dd>

Em áudio de onda, o valor pelo qual um driver de áudio de onda escala a densidade. Um fator de escala de dois resulta em um aumento de Octave em pitch. O dimensionamento de densidade requer hardware especializado. A taxa de reprodução e a taxa de amostragem não são alteradas.

</dd> <dt>

<span id="_win32_playback_rate_scale_factor_gloss"></span><span id="_WIN32_PLAYBACK_RATE_SCALE_FACTOR_GLOSS"></span>**fator de escala da taxa de reprodução**
</dt> <dd>

Em áudio de forma de onda, o valor pelo qual o driver de áudio de forma de onda dimensiona a taxa de reprodução. O dimensionamento da reprodução é realizado por meio de software; a taxa de amostra não é alterada, mas o driver interpola-se ignorando ou sintetizando amostras. Por exemplo, se a taxa de reprodução for alterada por um fator de dois, o driver ignorará todas as outras amostras.

</dd> <dt>

<span id="_win32_polyphony_gloss"></span><span id="_WIN32_POLYPHONY_GLOSS"></span>**Polyphony**
</dt> <dd>

O número máximo de observações que um dispositivo de saída MIDI (interface digital de instrumento musical) pode reproduzir simultaneamente.

</dd> <dt>

<span id="_win32_ppqn_gloss"></span><span id="_WIN32_PPQN_GLOSS"></span>**PPQN**
</dt> <dd>

Consulte as [partes por trimestre observação](/windows).

</dd> <dt>

<span id="_win32_preimaging_gloss"></span><span id="_WIN32_PREIMAGING_GLOSS"></span>**imagem**
</dt> <dd>

O processo de criação de um quadro de filme em um buffer de memória antes de ser exibido.

</dd> <dt>

<span id="_win32_red_book_audio_gloss"></span><span id="_WIN32_RED_BOOK_AUDIO_GLOSS"></span>**Áudio de livro vermelho**
</dt> <dd>

Consulte [Compact Disc-áudio digital (CD-da)](/windows).

</dd> <dt>

<span id="_win32_resource_interchange_file_format_riff_gloss"></span><span id="_WIN32_RESOURCE_INTERCHANGE_FILE_FORMAT_RIFF_GLOSS"></span>**Formato de arquivo de intercâmbio de recursos (riff)**
</dt> <dd>

Uma especificação de arquivo marcado usada para definir formatos padrão para arquivos multimídia. A estrutura de arquivos marcados ajuda a evitar problemas de compatibilidade que geralmente ocorrem quando as definições de formato de arquivo mudam ao longo do tempo. Como cada parte dos dados no arquivo é identificada por um cabeçalho padrão, um aplicativo que não reconhece um determinado elemento de dados pode ignorar as informações desconhecidas. Consulte também [formato de arquivo marcado](/windows).

</dd> <dt>

<span id="_win32_riff_gloss"></span><span id="_WIN32_RIFF_GLOSS"></span>**Riff**
</dt> <dd>

Consulte [formato de arquivo de intercâmbio de recursos (riff)](/windows).

</dd> <dt>

<span id="_win32_riff_chunk_gloss"></span><span id="_WIN32_RIFF_CHUNK_GLOSS"></span>**Parte do riff**
</dt> <dd>

Uma parte com o [formato de arquivo de intercâmbio de recursos (riff)](/windows) do identificador de partes que inclui um código de identificação e zero ou mais subpartes, o conteúdo do qual depende do tipo de formulário.

</dd> <dt>

<span id="_win32_riff_file_gloss"></span><span id="_WIN32_RIFF_FILE_GLOSS"></span>**Arquivo riff**
</dt> <dd>

Um arquivo cujo formato está em conformidade com um dos formulários de [formato de arquivo de intercâmbio de recursos (riff)](/windows) publicados. Exemplos de arquivos RIFF incluem arquivos WAVE para dados de áudio de forma de onda, arquivos RMID para sequências de MIDI (interface digital de instrumento musical) e arquivos RDIB para bitmaps independentes de dispositivo.

</dd> <dt>

<span id="_win32_riff_form_gloss"></span><span id="_WIN32_RIFF_FORM_GLOSS"></span>**Formulário riff**
</dt> <dd>

Uma especificação de formato de arquivo baseada no padrão [riff (formato de arquivo de intercâmbio de recursos)](/windows) .

</dd> <dt>

<span id="_win32_sample_gloss"></span><span id="_WIN32_SAMPLE_GLOSS"></span>**exemplo** de
</dt> <dd>

Um pedaço discreto de dados de formato de onda representados por um único valor numérico. A amostragem é o processo de converter dados analógicos em dados digitais, tirando amostras da onda analógica em intervalos regulares.

</dd> <dt>

<span id="_win32_sampling_rate_gloss"></span><span id="_WIN32_SAMPLING_RATE_GLOSS"></span>**taxa de amostragem**

A taxa em que um driver de áudio de forma de onda executa conversão de áudio para digital ou digital para áudio. Para o CD – áudio digital (CD-DA), a taxa de amostragem é 44,1 kHz. Consulte também CD [-áudio digital](/windows)

</dd> <dt>

<span id="_win32_sequence_gloss"></span><span id="_WIN32_SEQUENCE_GLOSS"></span>**sequência** de
</dt> <dd>

Um dispositivo MCI (interface de controle de mídia) que não requer um elemento de dispositivo (arquivo de dados) para reprodução. O driver de áudio do CD do MCI é um exemplo de um dispositivo simples.

</dd> <dt>

<span id="_win32_smpte_gloss"></span><span id="_WIN32_SMPTE_GLOSS"></span>**SMPTE**

O tipo de divisão SMPTE especifica o valor de quadros por segundo correspondente a uma determinada hora SMPTE. Por exemplo, um horário SMPTE de uma hora, 30 minutos, 24 segundos e 15 quadros será útil apenas se o valor de quadros por segundo ou o tipo de divisão SMPTE for conhecido.

</dd> <dt>

Um evento MIDI (interface digital de instrumento musical) que designa a hora SMPTE na qual a reprodução de um arquivo MIDI é iniciada. Os deslocamentos SMPTE são usados somente com arquivos MIDI usando o tipo de divisão SMPTE.

</dd> <dt>

<span id="_win32_smpte_time_gloss"></span><span id="_WIN32_SMPTE_TIME_GLOSS"></span>**Hora SMPTE**
</dt> <dd>

Uma representação padrão de tempo desenvolvida para os setores de vídeo e filme. O tempo SMPTE é usado com áudio MIDI (interface digital de instrumento musical) porque muitas pessoas usam MIDI para pontuar filmes e vídeos. O tempo SMPTE é um formato de hora absoluto expresso em horas, minutos, segundos e quadros. Os tipos de divisão Standard SMPTE são 24, 25 e 30 quadros por segundo.

</dd> <dt>

<span id="_win32_society_of_motion_picture_and_television_engineers_smpte_gloss"></span><span id="_WIN32_SOCIETY_OF_MOTION_PICTURE_AND_TELEVISION_ENGINEERS_SMPTE_GLOSS"></span>**Sociedade de imagem de movimento e engenheiros de televisão (SMPTE)**

</dd> <dt>

<span id="_win32_system_exclusive_data_gloss"></span><span id="_WIN32_SYSTEM_EXCLUSIVE_DATA_GLOSS"></span>**dados exclusivos do sistema**

Os fabricantes de dispositivos MIDI podem usar dados exclusivos do sistema para definir mensagens personalizadas que podem ser trocadas entre seus dispositivos MIDI. (A especificação MIDI padrão define apenas uma estrutura para mensagens exclusivas do sistema.) Consulte também [interface digital de instrumento musical](/windows).

A velocidade na qual um arquivo MIDI (interface digital de instrumento musical) é reproduzido. O tempo é medido em batidas por minuto (BPM); o tempo de MIDI típico é de 120 BPM. Consulte também [interface digital de instrumento musical](/windows).

Um limite alto reduz o número de mensagens de joystick enviadas a um aplicativo, mas também reduz a sensibilidade do joystick.

</dd> <dt>

<span id="_win32_time_stamp_gloss"></span><span id="_WIN32_TIME_STAMP_GLOSS"></span>**carimbo de data/hora**

<span id="_win32_tmsf_time_format_gloss"></span><span id="_WIN32_TMSF_TIME_FORMAT_GLOSS"></span>**Formato de hora de TMSF**

Um formato de hora usado pela MCI (interface de controle de mídia) para expressar tempo em faixas, minutos, segundos e quadros. O número de quadros em um segundo depende do tipo de dispositivo que está sendo usado; os dispositivos de áudio de CD usam 75 quadros por segundo. </dd> <dt>

<span id="_win32_track_gloss"></span><span id="_WIN32_TRACK_GLOSS"></span>**acompanhar**
</dt> <dd>

Uma sequência de som em um disco compacto – disco de áudio digital (CD-DA). Com um arquivo MIDI (interface digital de instrumento musical), as informações podem ser separadas em faixas, definidas pelo criador do arquivo. As faixas de arquivos MIDI podem corresponder a canais de MIDI ou podem corresponder a partes de uma música (como melodia ou Chorus); uma faixa de CD-DA geralmente corresponde a uma música. Consulte também CD [-áudio digital](/windows).
