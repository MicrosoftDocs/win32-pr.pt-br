# <a name="single-image-compression"></a>Single-Image compactação

Você pode usar a [**função ICImageCompress**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) para compactar uma única imagem. Essa função retorna um handle do DIB (bitmap independente de dispositivo compactado). O DIB compactado é empacotado usando o formato \_ CF DIB.
