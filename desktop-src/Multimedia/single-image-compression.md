# <a name="single-image-compression"></a>Compactação de Single-Image

Você pode usar a função [**ICImageCompress**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) para compactar uma única imagem. Essa função retorna um identificador do bitmap independente de dispositivo compactado (DIB). O DIB compactado é compactado usando o \_ formato DIB do CF.
