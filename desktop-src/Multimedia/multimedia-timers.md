# <a name="multimedia-timers"></a>Temporizadores de multimídia

Os serviços de timer de multimídia permitem que um aplicativo agende eventos de timer periódicos, ou seja, o aplicativo pode solicitar e receber mensagens de temporizador em intervalos especificados pelo aplicativo.
