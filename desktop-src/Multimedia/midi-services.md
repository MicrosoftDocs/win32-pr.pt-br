# <a name="midi-services"></a>Serviços MIDI

A maioria dos aplicativos poderá usar o sequenciador MIDI MCI ou buffers de fluxo (e a função [**midiStreamOut**](/windows/win32/api/mmeapi/nf-mmeapi-midistreamout) ) para implementar toda a funcionalidade de Midi de que precisam. Os desenvolvedores de MIDI sérios, aqueles que produzem ferramentas de criação de MIDI ou de sequenciamento, podem usar uma combinação dos recursos de fluxo e dos serviços de MIDI ou usar apenas os serviços de MIDI. Os tópicos a seguir fornecem informações gerais sobre como usar os serviços MIDI.

-   [Consultando dispositivos MIDI](querying-midi-devices.md)
-   [Abrindo e fechando drivers de dispositivo](opening-and-closing-device-drivers.md)
-   [Alocando e preparando blocos de dados MIDI](allocating-and-preparing-midi-data-blocks.md)
-   [Gerenciando blocos de dados MIDI](managing-midi-data-blocks.md)
-   [Solicitando formatos de hora](requesting-time-formats.md)
-   [Tratamento de erros com funções MIDI](handling-errors-with-midi-functions.md)
