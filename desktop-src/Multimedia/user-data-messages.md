# <a name="user-data-messages"></a>Mensagens de dados do usuário

Você pode associar dados a uma janela de captura usando a [**Cap do WM \_ \_ obter \_ \_ dados do usuário**](wm-cap-get-user-data.md) e a Cap do WM definir mensagens de [](/windows/desktop/api/Vfw/nf-vfw-capgetuserdata) [**\_ \_ \_ \_ dados do usuário**](wm-cap-set-user-data.md) (ou as macros capGetUserData e [**capSetUserData**](/windows/desktop/api/Vfw/nf-vfw-capsetuserdata) ). Você pode recuperar um valor de dados **longo** usando a mensagem do WM \_ Cap \_ obter \_ dados do usuário \_ e definir um valor de dados **longo** usando a mensagem de dados do usuário do WM \_ Cap \_ set \_ \_ .
