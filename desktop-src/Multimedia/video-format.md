# <a name="video-format"></a>Formato de vídeo

Você pode recuperar a estrutura que especifica o formato de vídeo ou o tamanho dessa estrutura enviando a mensagem [**WM \_ CAP GET \_ \_ VIDEOFORMAT**](wm-cap-get-videoformat.md) (ou as macros [**capGetVideoFormat**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformat) e [**capGetVideoFormatSize)**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformatsize) para uma janela de captura. Você pode definir o formato dos dados de vídeo capturados enviando a mensagem [**WM \_ CAP SET \_ \_ VIDEOFORMAT**](wm-cap-set-videoformat.md) (ou a [**macro capSetVideoFormat)**](/windows/desktop/api/Vfw/nf-vfw-capsetvideoformat) para uma janela de captura.
