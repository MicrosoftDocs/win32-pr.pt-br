# <a name="joysticks"></a>Joysticks

Esta visão geral descreve as funções e as mensagens que dão suporte a joysticks, bem como outros dispositivos de entrada auxiliares que controlam posições em um sistema de coordenadas absoluto, como uma tela sensível ao toque, digitalizando o Tablet e a caneta clara. Recursos estendidos também fornecem suporte para pedais de leme, manchetes e outros dispositivos que usam até seis eixos de movimento, ponto de vista e botões de 32.

-   [Sobre os joysticks](about-joysticks.md)
-   [Usando joysticks](using-joysticks.md)
-   [Referência de joystick](joystick-reference.md)

A API do joystick multimídia foi substituída pelo DirectInput, que tem mais recursos para os dispositivos de entrada e de comentários de força de hoje. Para obter mais informações, consulte a documentação do DirectX.
