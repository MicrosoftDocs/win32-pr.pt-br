# <a name="joysticks"></a>Joysticks

Esta visão geral descreve as funções e mensagens que suportam os pixels, bem como outros dispositivos de entrada auxiliares que acompanham posições em um sistema de coordenadas absoluto, como uma tela sensível ao toque, tablet de digitalização e caneta clara. As funcionalidades estendidas também oferecem suporte para botões de voo, voos e outros dispositivos que usam até seis eixos de movimento, um capacete de ponto de exibição e 32 botões.

-   [Sobre o Ltda.](about-joysticks.md)
-   [Usando o Joysticks](using-joysticks.md)
-   [Referência do Conector](joystick-reference.md)

A API de multimídia do player foi superada pelo DirectInput, que tem mais funcionalidades para os dispositivos de entrada e forçar comentários atuais. Para obter mais informações, consulte a documentação do DirectX.
