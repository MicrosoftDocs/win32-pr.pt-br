# <a name="level-2-optical-storage"></a>Armazenamento óptico de nível 2

O dispositivo de armazenamento óptico de desempenho mínimo é uma unidade de CD-ROM de velocidade dupla que atende aos seguintes critérios:

-   Uma taxa de transferência de dados sustentada de 300 quilobytes por segundo
-   Um uso de largura de banda de CPU de 40% ou menos ao manter uma taxa de transferência de dados sustentada de 150 quilobytes por segundo, ou uma largura de banda de CPU de 60% ou menos quando manter uma taxa de transferência de dados sustentada de 300 quilobytes por segundo
-   Um tempo médio de busca de 400 milissegundos ou menos
-   Uma classificação de tempo médio de 10.000 horas-entre-falhas
-   Preparado para CD-ROM XA (modo 1 com capacidade, modo 2 formato 1, compatível, modo 2)
-   Compatível com várias sessões
-   Driver MSCDEX-2,2 ou equivalente que implementa as funções de áudio estendidas

A largura de banda de CPU recomendada deve ser alcançada usando um tamanho de bloco de leitura de pelo menos 16K e um tempo de avanço de não mais do que o tempo necessário para carregar um bloco de leitura de dados no buffer de CD-ROM.
