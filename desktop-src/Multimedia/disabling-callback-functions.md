# <a name="disabling-callback-functions"></a>Desabilitando funções de retorno de chamada

Você pode desabilitar permanentemente ou temporariamente qualquer uma das funções de retorno de chamada especificando **NULL** no lugar da função de retorno de chamada ao enviar a mensagem apropriada para definir uma função de retorno de chamada.
