# <a name="vcm-services"></a>Serviços VCMs

Em geral, um aplicativo usa VCM para executar as seguintes tarefas:

-   Localize, abra ou instale um compactador ou descompactador.
-   Configurar ou obter informações de configuração sobre o compressor ou o descompactador.
-   Use uma série de funções para compactar, descompactar ou desenhar os dados.

As funções e macros da biblioteca DrawDib executam essas tarefas implicitamente e podem fornecer a maneira mais conveniente de usar o VCM. Para obter mais informações sobre a biblioteca DrawDib, consulte [DrawDib](drawdib.md).
