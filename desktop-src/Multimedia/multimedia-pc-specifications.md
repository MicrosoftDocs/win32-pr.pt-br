# <a name="multimedia-pc-specifications"></a>Especificações de PC multimídia

O MPC (Multimídia PC) Marketing Council desenvolveu duas especificações para incentivar a adoção de recursos multimídia. A especificação nível 1, desenvolvida em 1990, fornece uma definição de linha de base de computação multimídia em funcionalidade, componentes de hardware e componentes de software. A especificação nível 2, emitida em 1993, se baseia na primeira especificação e se concentra em recursos multimídia aprimorados desde que a primeira especificação foi emitida.

Este apêndice resume as especificações.

-   [Especificação de nível 1](level-1-specification.md)
-   [Especificação de nível 2](level-2-specification.md)
