# <a name="multimedia-input"></a>Entrada de multimídia

Windows fornece funções que permitem que um aplicativo receba entrada multimídia. As funções de entrada de multimídia foram introduzidas em Windows de 16 bits. Muitos de seus recursos foram substituídos por recursos do DirectX. Para obter mais informações, consulte a documentação do DirectX.

Os tópicos a seguir são discutidos na documentação de entrada de multimídia.

-   [E/s de arquivo multimídia](multimedia-file-i-o.md)
-   [Joysticks](joysticks.md)
-   [Temporizadores de multimídia](multimedia-timers.md)
