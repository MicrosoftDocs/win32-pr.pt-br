# <a name="multimedia-file-io"></a>E/s de arquivo multimídia

Os serviços de e/s de arquivo de multimídia fornecem mais funcionalidade do que os serviços do sistema operacional padrão, incluindo suporte para e/s armazenada em buffer, arquivos RIFF (Resource Interchange File Format), arquivos de memória e sistemas de armazenamento personalizados. Além disso, os serviços de e/s de arquivo de multimídia são otimizados para aplicativos sensíveis ao desempenho.
