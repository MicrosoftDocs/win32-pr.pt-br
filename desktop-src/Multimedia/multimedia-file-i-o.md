# <a name="multimedia-file-io"></a>E/S de arquivo multimídia

Os serviços de E/S de arquivo multimídia fornecem mais funcionalidade do que os serviços padrão do sistema operacional, incluindo suporte para E/S em buffer, arquivos DEDL (formato de arquivo de intercâmbio de recursos), arquivos de memória e sistemas de armazenamento personalizados. Além disso, os serviços de E/S de arquivo multimídia são otimizados para aplicativos sensíveis ao desempenho.
