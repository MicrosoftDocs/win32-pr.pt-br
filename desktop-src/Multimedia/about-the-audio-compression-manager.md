# <a name="about-the-audio-compression-manager"></a>Sobre o Gerenciador de Compactação de Áudio

O gerenciador de compactação de áudio adiciona suporte no nível do sistema para os seguintes serviços:

-   Compactação e descompactação de áudio em tempo de run time transparente
-   Seleção de formato de dados waveform-audio
-   Seleção de filtro de dados waveform-audio
-   Conversão de formato de dados waveform-audio
-   Filtragem de dados waveform-audio
