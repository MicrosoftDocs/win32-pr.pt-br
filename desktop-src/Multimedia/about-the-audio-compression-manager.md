# <a name="about-the-audio-compression-manager"></a>Sobre o Gerenciador de compactação de áudio

O Gerenciador de compactação de áudio adiciona suporte de nível de sistema para os seguintes serviços:

-   Compactação e descompactação de áudio em tempo de execução transparente
-   Forma de onda-seleção de formato de dados de áudio
-   Formato de onda-seleção de filtro de dados de áudio
-   Forma de onda-conversão de formato de dados de áudio
-   Formato de onda-filtragem de dados de áudio
