# <a name="digital-video-errors"></a>Erros de Digital-Video

Os seguintes valores de retorno adicionais são definidos para dispositivos de vídeo digital:



| Valor                           | Significado                                                         |
|---------------------------------|-----------------------------------------------------------------|
| MCIAVI \_ NomeDoProduto             | Vídeo                                                           |
| MCIERR \_ AVI \_ AUDIOERROR         | Erro desconhecido ao tentar reproduzir áudio.                   |
| MCIERR \_ AVI \_ BADPALETTE         | Não é possível alternar para a nova paleta.                                |
| MCIERR \_ AVI \_ CANTPLAYFULLSCREEN | Este arquivo AVI não pode ser reproduzido no modo de tela inteira.             |
| MCIERR \_ AVI \_ DISPLAYERROR       | Erro desconhecido ao tentar exibir o vídeo.                |
| MCIERR \_ AVI \_ nocompactador       | Não é possível localizar o compressor instalável necessário para reproduzir este arquivo.   |
| MCIERR \_ AVI \_ NODISPDIB          | 256 modo VGA de cor não disponível.                               |
| MCIERR \_ AVI não \_ intercalado     | Este arquivo AVI não é intercalado.                               |
| MCIERR \_ AVI \_ OLDAVIFORMAT       | Este arquivo AVI está em um formato obsoleto.                         |
| MCIERR \_ AVI \_ TOOBIGFORVGA       | Este arquivo AVI é muito grande para ser reproduzido no modo VGA selecionado. |
