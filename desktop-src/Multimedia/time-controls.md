# <a name="time-controls"></a>Controles de tempo

Os controles de tempo permitem que o usuário insira dados relacionados a tempo, como um atraso de eco ou reverberation. Os dados de tempo são expressos como inteiros positivos. Os tipos de controles de tempo incluem o seguinte:



| Control     | Descrição                                                                                            |
|-------------|--------------------------------------------------------------------------------------------------------|
| Microssegundos | Dá suporte a dados de tempo expressos em microssegundos. O intervalo de valores aceitáveis é de 0 a (232 – 1). |
| Milissegundos | Dá suporte a dados de tempo expressos em milissegundos. O intervalo de valores aceitáveis é de 0 a (232 – 1). |
