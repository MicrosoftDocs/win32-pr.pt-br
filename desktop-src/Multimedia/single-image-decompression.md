# <a name="single-image-decompression"></a>Descompactação de Single-Image

Você pode usar a função [**ICImageDecompress**](/windows/desktop/api/Vfw/nf-vfw-icimagedecompress) para descompactar uma única imagem. Essa função retorna um identificador do DIB descompactado. O DIB descompactado é armazenado no \_ formato DIB do CF.
