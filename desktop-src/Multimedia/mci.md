# <a name="mci"></a>MCI

A MCI (interface de controle de mídia) fornece comandos padrão para a reprodução de dispositivos de multimídia e a gravação de arquivos de recursos de multimídia. Esses comandos são uma interface genérica para quase todos os tipos de dispositivos de multimídia.

-   [Usando mensagens de comando do MCI](using-mci-command-messages.md)
-   [Referência de MCI](mci-reference.md)
