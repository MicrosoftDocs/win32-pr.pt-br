# <a name="joystick-position"></a>Posição do Conector

Você pode consultar um botão para obter informações de posição e botão usando a [**funçãogetPos.**](/windows/win32/api/joystickapi/nf-joystickapi-joygetpos) Por exemplo, um aplicativo pode consultar o basal em busca de valores de posição de linha de base. A folha de Painel de Controle de propriedades do Painel de Controle Usa essa técnica ao calibrar o joystick.

Você também pode consultar um ltda ou outro dispositivo que tenha recursos estendidos usando a [**funçãogetGetPosEx.**](/windows/win32/api/joystickapi/nf-joystickapi-joygetposex)
