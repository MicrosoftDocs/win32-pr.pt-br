<span data-ttu-id="63ad8-116">Um aplicativo pode localizar e abrir um compactador ou descompactador instalado usando as funções [**ICLocate**](/windows/desktop/api/Vfw/nf-vfw-iclocate) e [**ICOpen**](/windows/desktop/api/Vfw/nf-vfw-icopen) .</span><span class="sxs-lookup"><span data-stu-id="63ad8-116">An application can locate and open an installed compressor or decompressor by using the [**ICLocate**](/windows/desktop/api/Vfw/nf-vfw-iclocate) and [**ICOpen**](/windows/desktop/api/Vfw/nf-vfw-icopen) functions.</span></span> <span data-ttu-id="63ad8-117">Quando um aplicativo termina de usar um compressor ou um descompactador, ele o fecha usando a função [**ICClose**](/windows/desktop/api/Vfw/nf-vfw-icclose) .</span><span class="sxs-lookup"><span data-stu-id="63ad8-117">When an application finishes using a compressor or decompressor, it closes it by using the [**ICClose**](/windows/desktop/api/Vfw/nf-vfw-icclose) function.</span></span>

<span data-ttu-id="63ad8-118">Um aplicativo pode remover a entrada do registro para um compactador ou descompactador instalado usando a função [**ICRemove**](/windows/desktop/api/Vfw/nf-vfw-icremove) .</span><span class="sxs-lookup"><span data-stu-id="63ad8-118">An application can remove the registry entry for an installed compressor or decompressor by using the [**ICRemove**](/windows/desktop/api/Vfw/nf-vfw-icremove) function.</span></span> <span data-ttu-id="63ad8-119">Essa função remove a entrada do registro de um compressor ou descompactador que não está carregado atualmente na memória.</span><span class="sxs-lookup"><span data-stu-id="63ad8-119">This function removes the registry entry of a compressor or decompressor that is not currently loaded in memory.</span></span>

> <span data-ttu-id="63ad8-127">Um aplicativo que instala uma função como um compactador ou descompactador deve remover a função antes que o aplicativo seja fechado para que outros aplicativos não tentem usar a função.</span><span class="sxs-lookup"><span data-stu-id="63ad8-127">An application installing a function as a compressor or decompressor must remove the function before the application is closed so other applications do not try to use the function.</span></span> <span data-ttu-id="63ad8-128">Ao remover uma função, o aplicativo deve identificá-la com o código de quatro caracteres usado para instalá-la.</span><span class="sxs-lookup"><span data-stu-id="63ad8-128">When removing a function, the application must identify it with the four-character code used to install it.</span></span>
