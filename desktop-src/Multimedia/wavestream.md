# <a name="wavestream"></a>WaveStream

Os aplicativos usam as funções de retorno de chamada do WaveStream durante a captura de streaming para processar buffers de áudio. A janela de captura chama uma função de retorno de chamada de WaveStream logo antes de gravar cada buffer de áudio no disco.
