# <a name="frame-windows-multimedia"></a>Quadro (Windows Multimídia)

Uma janela de captura usa mensagens de notificação de retorno de chamada de quadro para notificar seu aplicativo quando um novo quadro de vídeo está disponível. A janela de captura habilita essas notificações de retorno de chamada somente se a taxa de visualização for menor que zero e a captura de streaming não estiver em andamento.
