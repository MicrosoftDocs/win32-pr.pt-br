# <a name="frame-windows-multimedia"></a>Quadro (multimídia do Windows)

Uma janela de captura usa mensagens de notificação de retorno de chamada de quadro para notificar seu aplicativo quando um novo quadro de vídeo está disponível. A janela de captura só habilitará essas notificações de retorno de chamada se a taxa de visualização for diferente de zero e a captura de streaming não estiver em andamento.
