# <a name="musical-instrument-digital-interface-midi"></a>MIDI (Instrument Digital Interface)

O MIDI (Instrument Digital Interface) é um protocolo e um conjunto de comandos para armazenar e transmitir informações sobre música. Os dispositivos de saída MIDI interpretam essas informações e as usam para sintetizar música.

-   [Usando MIDI](using-midi.md)
