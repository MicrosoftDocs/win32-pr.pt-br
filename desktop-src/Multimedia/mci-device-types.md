# <a name="mci-device-types"></a>Tipos de dispositivo MCI

Os valores a seguir identificam dispositivos em estruturas e mensagens MCI:



| Valor                         | Significado                       |
|-------------------------------|-------------------------------|
| \_ID de todos os \_ dispositivos MCI \_          | Qualquer dispositivo                    |
| \_animação MCI DEVTYPE \_       | Animação – dispositivo de reprodução     |
| \_áudio MCI DEVTYPE \_ CD \_       | Dispositivo de áudio de CD               |
| \_dat DEVTYPE \_ MCI             | Dispositivo de fita digital-audio     |
| \_ \_ vídeo digital DEVTYPE \_ MCI  | Dispositivo de reprodução de vídeo digital |
| \_DEVTYPE MCI \_           | Dispositivo indefinido              |
| sobreposição de \_ DEVTYPE MCI \_         | Dispositivo de sobreposição de vídeo          |
| \_scanner MCI DEVTYPE \_         | Dispositivo de scanner                |
| \_ \_ sequenciador DEVTYPE MCI       | Dispositivo sequenciador MIDI         |
| \_VCR DEVTYPE \_ MCI             | Gravador de vídeo-fita       |
| \_VIDEODISC MCI DEVTYPE \_       | VIDEODISC Player              |
| \_áudio MCI DEVTYPE \_ Wave \_ | Formato de onda-dispositivo de áudio         |
