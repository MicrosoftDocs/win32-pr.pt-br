# <a name="mci-device-types"></a>Tipos de dispositivo MCI

Os seguintes valores identificam dispositivos em estruturas e mensagens MCI:



| Valor                         | Significado                       |
|-------------------------------|-------------------------------|
| MCI \_ ALL \_ DEVICE \_ ID          | Qualquer dispositivo                    |
| ANIMAÇÃO \_ MCI DEVTYPE \_       | Dispositivo de reprodução de animação     |
| MCI \_ DEVTYPE \_ CD \_ AUDIO       | Dispositivo de áudio de CD               |
| MCI \_ DEVTYPE \_ DAT             | Dispositivo de fita de áudio digital     |
| VÍDEO DIGITAL MCI \_ DEVTYPE \_ \_  | Dispositivo de reprodução de vídeo digital |
| MCI \_ DEVTYPE \_ OTHER           | Dispositivo indefinido              |
| SOBREPOSIÇÃO \_ DE MCI DEVTYPE \_         | Dispositivo de sobreposição de vídeo          |
| MCI \_ DEVTYPE \_ SCANNER         | Dispositivo de scanner                |
| MCI \_ DEVTYPE \_ SEQUENCER       | Dispositivo de sequenciador MIDI         |
| MCI \_ DEVTYPE \_ VCR             | Gravador de gravador de vídeo       |
| MCI \_ DEVTYPE \_ VIDEODISC       | Player do Videodisc              |
| MCI \_ DEVTYPE \_ WAVEFORM \_ AUDIO | Dispositivo waveform-audio         |
