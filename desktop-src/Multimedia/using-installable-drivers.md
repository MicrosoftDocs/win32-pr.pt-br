# <a name="using-installable-drivers"></a>Usando drivers instaláveis

Você usa drivers instaláveis para fornecer aos aplicativos ou DLLs uma maneira padrão de acessar um dispositivo ou um conjunto de rotinas úteis. As seções a seguir mostram como criar um driver instalável usando uma função [DriverProc](/windows/win32/api/mmiscapi/nc-mmiscapi-driverproc) e como abrir um driver instalável e direcioná-lo para realizar tarefas úteis.
