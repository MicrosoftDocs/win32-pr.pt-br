# <a name="video-compression-manager"></a>Gerenciador de compactação de vídeo

O VCM (Gerenciador de compactação de vídeo) fornece acesso à interface usada por compactadores instaláveis para lidar com dados em tempo real. Os aplicativos podem usar os compactadores instaláveis para executar as seguintes tarefas:

-   Compactar e descompactar dados de vídeo.
-   Envie um renderizador de dados de vídeo compactado e faça com que ele o desenhe na tela.
-   Compactar, descompactar ou desenhar dados com renderizadores definidos pelo aplicativo.
-   Use renderizadores para manipular texto e dados personalizados.
