# <a name="using-joysticks"></a>Usando joysticks

Esta seção contém exemplos que demonstram como executar as seguintes tarefas:

-   [Obtendo os recursos do driver](getting-the-driver-capabilities.md)
-   [Capturando entrada do joystick](capturing-joystick-input.md)
-   [Processando mensagens do joystick](processing-joystick-messages.md)

Os exemplos são extraídos de um aplicativo de joystick simples que recupera informações de posição e de estado de botão dos serviços de joystick, reproduz os recursos de wave-áudio e pinta os buracos de marcadores na tela quando um usuário pressiona os botões do joystick.
