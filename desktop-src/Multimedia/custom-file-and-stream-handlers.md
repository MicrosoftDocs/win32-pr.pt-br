# <a name="custom-file-and-stream-handlers"></a>Manipuladores de arquivo e fluxo personalizados

Manipuladores de arquivo e fluxo são drivers que fornecem interfaces consistentes a um aplicativo que controla dados de multimídia. Os manipuladores de arquivo e de fluxo incluídos no sistema operacional usam dados de vídeo e de formato de onda de áudio armazenados em arquivos AVI (áudio-video intercalado) e Wave-Audio.

Você pode escrever manipuladores para permitir que seu aplicativo grave ou acesse dados de multimídia de outra fonte, como um arquivo usando um formato proprietário, um arquivo AVI que foi expandido para conter fluxos de dados adicionais ou um manipulador que gera seus próprios dados de multimídia. Se você tiver um formato de arquivo personalizado para dados AVI que deseja usar com as [funções e macros do avifile](avifile-functions-and-macros.md), precisará escrever um manipulador personalizado.
