# <a name="mci-devices"></a>Dispositivos MCI

Cada dispositivo multimídia MCI dá suporte a um conjunto principal de comandos MCI de uma maneira que faça sentido para esse dispositivo. Por exemplo, o [**comando play**](play.md) ([**MCI \_ PLAY**](mci-play.md)) faz com que o dispositivo aberto reproduza um arquivo ou uma faixa, independentemente do tipo de dados com o qual o dispositivo funciona. Os tópicos a seguir discutem dispositivos MCI e como eles respondem aos comandos MCI padrão.

-   [Controle de dispositivo](device-control.md)
-   [Reprodução e posicionamento](playback-and-positioning.md)
-   [Tipos de dispositivo](device-types.md)
-   [Nomes de dispositivo](device-names.md)
-   [Suporte de driver para comandos MCI](driver-support-for-mci-commands.md)
-   [Comportamento padrão de drivers](default-behavior-of-drivers.md)
