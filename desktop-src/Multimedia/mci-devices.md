# <a name="mci-devices"></a>Dispositivos MCI

Cada dispositivo de multimídia MCI dá suporte a um conjunto principal de comandos MCI de uma maneira que faz sentido para esse dispositivo. Por exemplo, o comando [**Play**](play.md) ([**MCI \_ Play**](mci-play.md)) faz com que o dispositivo aberto reproduza um arquivo ou uma faixa, independentemente do tipo de dados com o qual o dispositivo funciona. Os tópicos a seguir abordam os dispositivos MCI e como eles respondem aos comandos MCI padrão.

-   [Controle de dispositivo](device-control.md)
-   [Reprodução e posicionamento](playback-and-positioning.md)
-   [Tipos de dispositivo](device-types.md)
-   [Nomes de dispositivo](device-names.md)
-   [Suporte de driver para comandos MCI](driver-support-for-mci-commands.md)
-   [Comportamento padrão de drivers](default-behavior-of-drivers.md)
