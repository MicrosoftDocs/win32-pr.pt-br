# <a name="multimedia-playback"></a>Reprodução de multimídia

A função [**MCIWndCreate**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) fornece os meios para controlar uma janela MCIWnd e o dispositivo associado a ela. Em geral, essa função registra a classe de janela MCIWnd e cria uma janela MCIWnd para usar serviços MCI. Esta seção descreve como usar o **MCIWndCreate** para executar as seguintes tarefas:

-   [Adicionando User-Controlled reprodução](adding-user-controlled-playback.md)
-   [Automatizando a reprodução](automating-playback.md)
-   [Usando estilos de janela para alterar a janela MCIWnd](using-window-styles-to-change-the-mciwnd-window.md)
-   [Permitindo que o usuário especifique dispositivos e arquivos](allowing-the-user-to-specify-devices-and-files.md)
