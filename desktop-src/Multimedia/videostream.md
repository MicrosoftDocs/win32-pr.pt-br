# <a name="videostream"></a>Videostream

Os aplicativos podem usar funções de retorno de chamada videostream durante a captura de fluxo para processar um quadro de vídeo capturado. A janela de captura chama uma função de retorno de chamada videostream logo antes de gravar cada quadro capturado no disco.
