# <a name="videostream"></a>Fluxo de vídeo

Os aplicativos podem usar funções de retorno de chamada de videostream durante a captura de streaming para processar um quadro de vídeo capturado. A janela de captura chama uma função de retorno de chamada de videostream logo antes de escrever cada quadro capturado no disco.
