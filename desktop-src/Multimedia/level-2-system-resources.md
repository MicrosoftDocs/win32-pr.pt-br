# <a name="level-2-system-resources"></a>Recursos do sistema de nível 2

A configuração mínima para um PC para atender à especificação de nível 2 inclui os seguintes itens:

-   Um microprocessador 486SX de 25 MHz com 4 MB de RAM
-   Uma unidade de disquete de alta densidade de 3,5 polegadas
-   Uma unidade de disco rígido com pelo menos 160 MB de espaço em disco
-   Um teclado 101-Key com um conector DIN padrão ou um que fornece funcionalidade idêntica usando combinações de teclas
-   Um mouse de dois botões com um conector serial ou de barramento
-   Uma porta MIDI (interface digital de instrumentos musicais) que inclui MIDI out, MIDI in e MIDI thru, e que tem suporte de interrupção para a transferência de entrada e FIFO
-   Uma porta de joystick (Game) digital ou analógica de estilo IBM
-   Um monitor de cores com uma resolução de vídeo de 640 por 480 pixels com 65.536 cores
-   Software de sistema que oferece compatibilidade binária com o Windows 3,0 ou o Windows 3,1

Além disso, a meta de desempenho recomendada para o adaptador de vídeo (VGA +) é de 1,2 milhões pixels por segundo usando 40% ou menos da largura de banda da CPU. Os bitmaps independentes de dispositivo (DIBs) usados para medir essa meta de desempenho têm profundidades de cor de 1, 4 e 8 bits e podem usar a codificação de comprimento de execução ou ser não codificados. Um segundo método para medir o desempenho do vídeo é fornecer imagens de vídeo digital de 256 cores, 320 por 240 pixel em 15 quadros por segundo.
