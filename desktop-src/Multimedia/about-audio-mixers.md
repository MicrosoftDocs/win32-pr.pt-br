# <a name="about-audio-mixers"></a>Sobre os mixers de áudio

Os serviços de mixer de áudio controlam o roteamento de linhas de áudio para um dispositivo de destino para reprodução ou gravação. Esses serviços também podem controlar o volume e outros efeitos. Muitas das técnicas necessárias para usar esses serviços são semelhantes às de dispositivos de áudio abordados em outras visões gerais de multimídia.
