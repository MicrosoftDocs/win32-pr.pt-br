# <a name="basic-capture-options"></a>Opções básicas de captura

Ao modificar um ou mais dos parâmetros de captura definidos na estrutura [**CAPTUREPARMS**](/windows/win32/api/vfw/ns-vfw-captureparms) , você pode executar as seguintes tarefas:

-   Altere a taxa de captura do quadro.
-   Especifique o teclado ou controle de mouse para encerrar uma sessão de captura.
-   Especifique uma duração para uma sessão de captura.
