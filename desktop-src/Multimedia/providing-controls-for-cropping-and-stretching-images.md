# <a name="providing-controls-for-cropping-and-stretching-images"></a><span data-ttu-id="d141b-107">Fornecendo controles para cortar e alongar imagens</span><span class="sxs-lookup"><span data-stu-id="d141b-107">Providing Controls for Cropping and Stretching Images</span></span>

<span data-ttu-id="d141b-108">O MCIWnd permite cortar e alongar imagens de um clipe de vídeo.</span><span class="sxs-lookup"><span data-stu-id="d141b-108">MCIWnd allows you to crop and stretch images of a video clip.</span></span> <span data-ttu-id="d141b-109">Para entender esses recursos, você precisa entender as relações entre o *tamanho do quadro*, o *retângulo de origem*, o retângulo de *destino* e a *área de reprodução*.</span><span class="sxs-lookup"><span data-stu-id="d141b-109">To understand these features, you need to understand the relationships between *frame size*, *source rectangle*, *destination rectangle*, and *playback area*.</span></span>

<span data-ttu-id="d141b-110">Um clipe de vídeo consiste em vários quadros, cada um contendo uma imagem.</span><span class="sxs-lookup"><span data-stu-id="d141b-110">A video clip consists of several frames, each containing one image.</span></span> <span data-ttu-id="d141b-111">O tamanho do quadro de um videoclipe é o tamanho da imagem no quadro atual.</span><span class="sxs-lookup"><span data-stu-id="d141b-111">The frame size of a video clip is the size of the image in the current frame.</span></span> <span data-ttu-id="d141b-112">Normalmente, um videoclipe tem um tamanho de quadro porque todas as imagens no clipe têm o mesmo tamanho.</span><span class="sxs-lookup"><span data-stu-id="d141b-112">Typically, a video clip has one frame size because all the images in the clip are the same size.</span></span>

<span data-ttu-id="d141b-113">O retângulo de origem é uma área retangular que sobrepõe os quadros de um clipe de vídeo.</span><span class="sxs-lookup"><span data-stu-id="d141b-113">The source rectangle is a rectangular area that overlays the frames of a video clip.</span></span> <span data-ttu-id="d141b-114">O retângulo de origem define a parte de cada quadro exibido durante a reprodução.</span><span class="sxs-lookup"><span data-stu-id="d141b-114">The source rectangle defines the portion of each frame that is displayed during playback.</span></span> <span data-ttu-id="d141b-115">Quando um clipe de vídeo é carregado com MCIWnd, o retângulo de origem é inicializado com as mesmas dimensões e posição que o quadro inicial do clipe de vídeo.</span><span class="sxs-lookup"><span data-stu-id="d141b-115">When a video clip is loaded with MCIWnd, the source rectangle is initialized with the same dimensions and position as the initial frame of the video clip.</span></span>

<span data-ttu-id="d141b-116">O retângulo de destino é uma área retangular que define uma janela de reprodução virtual.</span><span class="sxs-lookup"><span data-stu-id="d141b-116">The destination rectangle is a rectangular area that defines a virtual playback window.</span></span> <span data-ttu-id="d141b-117">O retângulo de destino recebe os dados da imagem do retângulo de origem para cada quadro do clipe de vídeo.</span><span class="sxs-lookup"><span data-stu-id="d141b-117">The destination rectangle receives the image data from the source rectangle for each frame of the video clip.</span></span> <span data-ttu-id="d141b-118">Quando as dimensões do retângulo de origem e destino são diferentes, o MCIWnd ajusta os dados de imagem horizontal e verticalmente conforme necessário para preencher o retângulo de destino.</span><span class="sxs-lookup"><span data-stu-id="d141b-118">When the source and destination rectangle dimensions are different, MCIWnd adjusts the image data horizontally and vertically as needed to fill the destination rectangle.</span></span> <span data-ttu-id="d141b-119">Quando um clipe de vídeo é carregado com MCIWnd, o retângulo de destino é inicializado com as mesmas dimensões e posição que o quadro inicial do clipe de vídeo.</span><span class="sxs-lookup"><span data-stu-id="d141b-119">When a video clip is loaded with MCIWnd, the destination rectangle is initialized with the same dimensions and position as the initial frame of the video clip.</span></span>

<span data-ttu-id="d141b-123">Você pode cortar um clipe de vídeo usando as macros [**MCIWndGetSource**](/windows/desktop/api/Vfw/nf-vfw-mciwndgetsource) e [**MCIWndPutSource**](/windows/desktop/api/Vfw/nf-vfw-mciwndputsource) para alterar o retângulo de origem.</span><span class="sxs-lookup"><span data-stu-id="d141b-123">You can crop a video clip by using the [**MCIWndGetSource**](/windows/desktop/api/Vfw/nf-vfw-mciwndgetsource) and [**MCIWndPutSource**](/windows/desktop/api/Vfw/nf-vfw-mciwndputsource) macros to alter the source rectangle.</span></span> <span data-ttu-id="d141b-124">Cortar uma imagem determina apenas qual parte dos quadros é exibida durante a reprodução; Ele não altera o conteúdo do arquivo que está sendo reproduzido.</span><span class="sxs-lookup"><span data-stu-id="d141b-124">Cropping an image determines only which portion of the frames are displayed during playback; it does not alter the content of the file being played.</span></span> <span data-ttu-id="d141b-125">Antes de cortar uma imagem, você pode recuperar o tamanho atual do retângulo de origem usando **MCIWndGetSource**.</span><span class="sxs-lookup"><span data-stu-id="d141b-125">Before you crop an image, you can retrieve the current size of the source rectangle by using **MCIWndGetSource**.</span></span> <span data-ttu-id="d141b-126">Depois que o novo tamanho e o local do retângulo de origem forem calculados, você poderá definir os limites de corte do retângulo de origem usando **MCIWndPutSource**.</span><span class="sxs-lookup"><span data-stu-id="d141b-126">After the new size and location of the source rectangle are calculated, you can set the cropping boundaries of the source rectangle by using **MCIWndPutSource**.</span></span>

<span data-ttu-id="d141b-127">Você pode alongar um clipe de vídeo usando as macros [**MCIWndGetDest**](/windows/desktop/api/Vfw/nf-vfw-mciwndgetdest) e [**MCIWndPutDest**](/windows/desktop/api/Vfw/nf-vfw-mciwndputdest) para alterar o retângulo de destino.</span><span class="sxs-lookup"><span data-stu-id="d141b-127">You can stretch a video clip by using the [**MCIWndGetDest**](/windows/desktop/api/Vfw/nf-vfw-mciwndgetdest) and [**MCIWndPutDest**](/windows/desktop/api/Vfw/nf-vfw-mciwndputdest) macros to alter the destination rectangle.</span></span> <span data-ttu-id="d141b-128">Ao alongar um clipe de vídeo, você aumenta ou reduz o tamanho do quadro de um clipe de vídeo verticalmente, horizontalmente ou em ambas as direções.</span><span class="sxs-lookup"><span data-stu-id="d141b-128">When you stretch a video clip, you lengthen or shorten the frame size of a video clip vertically, horizontally, or in both directions.</span></span> <span data-ttu-id="d141b-129">Antes de alongar uma imagem, você pode recuperar o tamanho atual e o local do retângulo de destino usando **MCIWndGetDest**.</span><span class="sxs-lookup"><span data-stu-id="d141b-129">Before you stretch an image, you can retrieve the current size and location of the destination rectangle by using **MCIWndGetDest**.</span></span> <span data-ttu-id="d141b-130">A macro **MCIWndPutDest** permite redefinir o retângulo de destino.</span><span class="sxs-lookup"><span data-stu-id="d141b-130">The **MCIWndPutDest** macro allows you to redefine the destination rectangle.</span></span> <span data-ttu-id="d141b-131">O alongamento pode distorcer a imagem durante a reprodução, mas não altera o conteúdo do arquivo que está sendo reproduzido.</span><span class="sxs-lookup"><span data-stu-id="d141b-131">Stretching can distort the image during playback, but it does not alter the content of the file being played.</span></span>

<span data-ttu-id="d141b-132">Se o tamanho do retângulo de destino se tornar maior que a área de reprodução, você poderá especificar qual parte da área de reprodução exibirá o clipe de vídeo usando **MCIWndPutDest**.</span><span class="sxs-lookup"><span data-stu-id="d141b-132">If the size of the destination rectangle becomes larger than the playback area, you can specify which portion of the playback area will display the video clip by using **MCIWndPutDest**.</span></span>

> <span data-ttu-id="d141b-133">A macro [**MCIWndPutDest**](/windows/desktop/api/Vfw/nf-vfw-mciwndputdest) não altera o tamanho da área de reprodução.</span><span class="sxs-lookup"><span data-stu-id="d141b-133">The [**MCIWndPutDest**](/windows/desktop/api/Vfw/nf-vfw-mciwndputdest) macro does not change the size of the playback area.</span></span> <span data-ttu-id="d141b-134">Para alongar a janela MCIWnd junto com o retângulo de destino, você precisa saber o tamanho atual da janela MCIWnd e emitir novas dimensões de janela com base no retângulo de destino.</span><span class="sxs-lookup"><span data-stu-id="d141b-134">To stretch the MCIWnd window along with the destination rectangle, you need to know the current size of the MCIWnd window and issue new window dimensions based on the destination rectangle.</span></span> <span data-ttu-id="d141b-135">Você pode recuperar as dimensões da janela MCIWnd usando a função [GetWindowRect](/windows/win32/api/winuser/nf-winuser-getwindowrect) e redimensionar a janela do MCIWnd usando a função [SetWindowPos](/windows/win32/api/winuser/nf-winuser-setwindowpos) .</span><span class="sxs-lookup"><span data-stu-id="d141b-135">You can retrieve the MCIWnd window dimensions by using the [GetWindowRect](/windows/win32/api/winuser/nf-winuser-getwindowrect) function and resize the MCIWnd window by using the [SetWindowPos](/windows/win32/api/winuser/nf-winuser-setwindowpos) function.</span></span>
