# <a name="about-mci"></a>Sobre a MCI

A MCI fornece aplicativos com funcionalidades independentes de dispositivo para controlar periféricos visuais e de áudio. Seu aplicativo pode usar a MCI para controlar qualquer dispositivo multimídia com suporte, incluindo dispositivos waveform-audio, sequenciadores MIDI, dispositivos de áudio de CD e dispositivos de vídeo digital (reprodução de vídeo).
