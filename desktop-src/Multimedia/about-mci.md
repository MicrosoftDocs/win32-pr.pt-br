# <a name="about-mci"></a>Sobre o MCI

O MCI fornece aplicativos com recursos independentes de dispositivo para controlar o áudio e os periféricos visuais. Seu aplicativo pode usar o MCI para controlar qualquer dispositivo multimídia com suporte, incluindo dispositivos de wave-áudio, sequenciais MIDI, dispositivos de áudio de CD e dispositivos de vídeo digital (reprodução de vídeo).
