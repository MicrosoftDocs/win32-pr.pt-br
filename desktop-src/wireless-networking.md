# <a name="wireless-networking"></a>Redes sem Fio

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                          | Descrição                                                                                                                                                                                                                                                                                                     |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Bluetooth](./bluetooth/bluetooth-start-page.md)<br/>                         | O Bluetooth é um protocolo padrão do setor que permite a conectividade sem fio para computadores, dispositivos portáteis, telefones celulares e outros dispositivos.<br/>                                                                                                                                                     |
| [Associação de dados de infravermelho (IrDA)](/previous-versions/windows/desktop/irda/irda-start-page)<br/>            | A IrDA (Associação de dados de infravermelho) é um pacote de protocolo projetado para fornecer conectividade sem fio e de linha de visão entre os dispositivos. <br/>                                                                                                                                                                      |
| [Banda larga móvel](./mbn/mobile-broadband-networks-portal.md)<br/>            | A API de banda larga móvel é usada para implementar a conectividade com redes celulares. Os aplicativos não devem se comunicar diretamente com esses dispositivos de banda larga móvel. Em vez disso, eles devem usar a API de banda larga móvel. <br/>                                                                                  |
| [WiFi nativo](./nativewifi/portal.md)<br/>                                         | O componente de configuração automática WiFi nativo configura, conecta-se a e desconecta-se de redes sem fio. O WiFi nativo pode armazenar perfis nas redes com as quais interage na forma de documentos XML. <br/>                                                                                   |
| [Windows Connect agora](./wcn/portal.md)<br/>                                   | O WCN (Windows Connect Now) permite dispositivos móveis e incorporados, APs (pontos de acesso) 802,11 e computadores para se conectar com segurança e trocar configurações entre si. O WCN foi projetado para o usuário doméstico ou de pequena empresa, fornecendo um comprometimento razoável entre a facilidade de uso e a segurança robusta. <br/> |
| [Gerenciador de Conexão do Windows](./wcm/windows-connection-manager-portal.md)<br/> | O WCM (Gerenciador de conexões do Windows) permite a criação e a configuração do software do Gerenciador de conexões.<br/>                                                                                                                                                                                              |
