# <a name="idcompositionrotatetransform3dsetaxisz-methods"></a>Métodos IDCompositionRotateTransform3D:: SetAxisZ

Altera ou anima o valor da propriedade AxisZ de uma transformação de rotação 3D. A propriedade AxisZ especifica a coordenada z para o vetor de rotação de eixo. O valor padrão é 1.0.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                       | Descrição                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisz(float))                                     | Altera o valor da propriedade AxisZ.<br/>  |
| [**SetAxisZ (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisz(idcompositionanimation)) | Anima o valor da propriedade AxisZ.<br/> |
