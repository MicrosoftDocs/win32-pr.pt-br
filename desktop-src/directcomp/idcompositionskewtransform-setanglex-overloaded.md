# <a name="idcompositionskewtransformsetanglex-methods"></a>Métodos IDCompositionSkewTransform::SetAngleX

Altera ou anima o valor da propriedade AngleX de uma transformação de distorção 2D. A propriedade AngleX especifica o ângulo de rotação, em graus. O valor padrão é zero.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                     | Descrição                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetAngleX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(float))                                     | Altera o valor da propriedade AngleX.<br/>  |
| [**SetAngleX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(idcompositionanimation)) | Anima o valor da propriedade AngleX.<br/> |
