# <a name="idcompositionrotatetransform3dsetaxisx-methods"></a>Métodos IDCompositionRotateTransform3D:: SetAxisX

Altera ou anima o valor da propriedade as de uma transformação de rotação 3D. A propriedade as especifica a coordenada x para o vetor de rotação do eixo. O valor padrão é zero.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                       | Descrição                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | Altera o valor da propriedade as.<br/>  |
| [**SetAxisX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | Anima o valor da propriedade as.<br/> |
