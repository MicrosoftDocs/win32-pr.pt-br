# <a name="idcompositionrotatetransform3dsetaxisx-methods"></a>Métodos IDCompositionRotateTransform3D::SetAxisX

Altera ou anima o valor da propriedade AxisX de uma transformação de rotação 3D. A propriedade AxisX especifica a coordenada x para o vetor de eixo de rotação. O valor padrão é zero.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                       | Descrição                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | Altera o valor da propriedade AxisX.<br/>  |
| [**SetAxisX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | Anima o valor da propriedade AxisX.<br/> |
