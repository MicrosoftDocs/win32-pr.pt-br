# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>Métodos IDCompositionRotateTransform3D:: setaxisy

Altera ou anima o valor da propriedade AxisY de uma transformação de rotação. A propriedade AxisY especifica a coordenada y para o vetor de rotação do eixo. O valor padrão é zero.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                       | Descrição                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Setaxis (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | Altera o valor da propriedade AxisY.<br/>  |
| [**Setaxisy (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | Anima o valor da propriedade AxisY.<br/> |
