# <a name="idcompositionmatrixtransformsetmatrixelement-methods"></a>Métodos IDCompositionMatrixTransform:: setmatrixelement

Altera ou anima o valor de um elemento da matriz dessa transformação 2D.

| Método                                                                                                                               | Descrição                                                                      |
|:-------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**Setmatrixelement (int, int, float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionmatrixtransform-setmatrixelement(int_int_float))                                           | Altera o valor de um elemento da matriz desta transformação 2D.<br/>  |
| [**Setmatrixelement (int, int, IDCompositionAnimation \* )**](idcompositionmatrixtransform-setmatrixelement-idcompositionanimation.md) | Anima o valor de um elemento da matriz dessa transformação 2D.<br/> |
