# <a name="directcomposition-enumerations"></a>Enumerações DirectComposition

Esta seção descreve as enumerações fornecidas pela API do Microsoft DirectComposition.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                  | Descrição                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**visibilidade do DCOMPOSITION \_ BACKface \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Especifica a visibilidade do backface a ser aplicada a um Visual. <br/>                                                                                                                            |
| [**\_modo de \_ interpolação de bitmap DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Especifica o modo de interpolação a ser usado quando um bitmap é composto com qualquer transformação em que os pixels no bitmap não alinham exatamente um para um com pixels na tela. <br/>         |
| [**\_modo de borda DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Especifica o modo de borda a ser usado ao compor um bitmap ou aplicar um clipe com qualquer transformação, de modo que as bordas do bitmap ou do clipe não sejam alinhadas ao eixo com coordenadas de inteiros. <br/> |
| [**\_modo composto \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | O modo a ser usado para misturar o conteúdo de bitmap de um Visual com o destino de renderização.<br/>                                                                                                          |
| [**\_modo de OPACIDADE DCOMPOSITION \_**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Especifica como o valor de opacidade efetivo de um Visual é aplicado ao conteúdo e aos filhos do Visual s.<br/>                                                                                  |
| [**\_modo de transformação DCOMPOSITION \_**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Especifica o estilo de transformação a ser aplicado a um Visual. <br/>                                                                                                                                |
