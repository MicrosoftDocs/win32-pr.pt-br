# <a name="directcomposition-enumerations"></a>Enumerações DirectComposition

Esta seção descreve as enumerações fornecidas pela API do Microsoft DirectComposition.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                  | Descrição                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**VISIBILIDADE DE \_ BACKFACE DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Especifica a visibilidade de backface a ser aplicada a um visual. <br/>                                                                                                                            |
| [**MODO DE INTERPOLAÇÃO \_ DE BITMAP \_ DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Especifica o modo de interpolação a ser usado quando um bitmap é composto por qualquer transformação em que os pixels no bitmap não se alinham exatamente um para um com pixels na tela. <br/>         |
| [**MODO DE BORDA \_ \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Especifica o modo de borda a ser usado ao compor um bitmap ou aplicar um clipe com qualquer transformação, de modo que as bordas do bitmap ou clipe não estejam alinhadas ao eixo com coordenadas de inteiro. <br/> |
| [**MODO COMPOSTO \_ \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | O modo a ser usado para combinar o conteúdo de bitmap de um visual com o destino de renderização.<br/>                                                                                                          |
| [**MODO DE \_ \_ OPACIDADE DCOMPOSITION**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Especifica como o valor de opacidade efetivo de um visual é aplicado ao conteúdo e aos filhos desse visual.<br/>                                                                                  |
| [**MODO DE TRANSFORMAÇÃO \_ \_ DCOMPOSITION**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Especifica o estilo de transformação a ser aplicado a um visual. <br/>                                                                                                                                |
