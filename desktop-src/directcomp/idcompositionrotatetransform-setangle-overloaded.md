# <a name="idcompositionrotatetransformsetangle-methods"></a>Métodos IDCompositionRotateTransform:: setangle

Altera ou anima o valor da propriedade Angle de uma transformação Rotation. A propriedade Angle especifica o ângulo de rotação, em graus. O valor padrão é zero.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                     | Descrição                                          |
|:-----------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Setangle (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(float))                                     | Altera o valor da propriedade Angle.<br/>  |
| [**Setangle (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(idcompositionanimation)) | Anima o valor da propriedade Angle.<br/> |
