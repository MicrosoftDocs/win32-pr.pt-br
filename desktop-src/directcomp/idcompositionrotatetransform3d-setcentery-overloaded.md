# <a name="idcompositionrotatetransform3dsetcentery-methods"></a>Métodos IDCompositionRotateTransform3D:: setcentery

Altera ou anima o valor da propriedade CenterY de uma transformação de rotação 3D. A propriedade Centername especifica a coordenada y do ponto sobre o qual a rotação é executada. O valor padrão é zero.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                           | Descrição                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Setcentery (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcentery(float))                                     | Altera o valor da propriedade CenterY.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcentery(idcompositionanimation)) | Anima o valor da propriedade CenterY.<br/> |
