# <a name="directcomposition-interfaces"></a>Interfaces DirectComposition

Esta seção descreve as interfaces fornecidas pela API do Microsoft DirectComposition.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                               | Descrição                                                                                                                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDCompositionAffineTransform2DEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect)<br/>     | O efeito composto aritmético é usado para combinar duas imagens usando uma soma ponderada de pixels das imagens de entrada. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionAnimation**](/windows/desktop/api/DcompAnimation/nn-dcompanimation-idcompositionanimation)<br/>                                 | Representa uma função para animar uma ou mais propriedades de um ou mais objetos DirectComposition.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionArithmeticCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionarithmeticcompositeeffect)<br/> | O efeito composto aritmético é usado para combinar duas imagens usando uma soma ponderada de pixels das imagens de entrada.<br/>                                                                                                                                                                                                                                                                              |
| [**IDCompositionBlendEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionblendeffect)<br/>                             | O Efeito Blend é usado para combinar duas imagens. <br/>                                                                                                                                                                                                                                                                                                                                                 |
| [**IDCompositionBrightnessEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionbrightnesseffect)<br/>                   | O efeito de brilho controla o brilho da imagem.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionColorMatrixEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect)<br/>                 | O efeito da matriz de cores altera os valores RGBA de um bitmap.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncompositeeffect)<br/>                     | O efeito composto é usado para combinar duas ou mais imagens. Esse efeito tem 13 modos compostos diferentes. O efeito composto aceita duas ou mais entradas. Quando você especifica duas imagens, o destino é a primeira entrada (índice 0) e a origem é a segunda entrada (índice 1). Se você especificar mais de 2 entradas, as imagens serão compostas começando com a primeira entrada e a segunda e assim por diante. <br/> |
| [**IDCompositionClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionclip)<br/>                                           | Representa um objeto de clipe que é usado para restringir a renderização de uma subárvore visual a uma área retangular. <br/>                                                                                                                                                                                                                                                                                    |
| [**IDCompositionDesktopDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondesktopdevice)<br/>                         | Um aplicativo deve usar a interface IDCompositionDesktopDevice para usar DirectComposition em um aplicativo da área de trabalho Win32. Essa interface permite que o aplicativo conecte uma árvore visual a uma janela e hospede janelas filho em camadas para composição<br/>                                                                                                                                |
| [**IDCompositionDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice)<br/>                                       | Serve como uma fábrica para todos os outros objetos DirectComposition e fornece métodos para controlar a composição transacional.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice2**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice2)<br/>                                     | Serve como uma fábrica para todos os outros objetos DirectComposition e fornece métodos para controlar a composição transacional.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice3**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice3)<br/>                                     | Serve como uma fábrica para todos os outros objetos DirectComposition e fornece métodos para controlar a composição transacional. <br/>                                                                                                                                                                                                                                                                        |
| [**IDCompositionDeviceDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevicedebug)<br/>                             | Fornece acesso a recursos de renderização que ajudam com a depuração de aplicativos e ajuste de desempenho. Essa interface pode ser consultada na interface do dispositivo DirectComposition.<br/>                                                                                                                                                                                                                  |
| [**IDCompositionEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffect)<br/>                                       | Representa um efeito de bitmap que modifica a rasterização da subárvore de um visual. <br/>                                                                                                                                                                                                                                                                                                             |
| [**IDCompositionEffectGroup**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffectgroup)<br/>                             | Representa um grupo de efeitos de bitmap que são aplicados juntos para modificar a rasterização da subárvore de um visual. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionFilterEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionfiltereffect)<br/>                           | Representa um efeito de filtro.<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionFloodEffect**](/previous-versions/windows/desktop/legacy/dn919732(v=vs.85))<br/>                             | O efeito de inundação é usado para gerar um bitmap com base na cor e no valor alfa especificados. Você pode usar esse efeito quando quiser uma cor específica como uma entrada para um efeito, como uma cor da tela de fundo. <br/>                                                                                                                                                                                        |
| [**IDCompositionGaussianBlurEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiongaussianblureffect)<br/>               |                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IDCompositionRotationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionhuerotationeffect)<br/>                 | O efeito de rotação de matiz altera o matiz de uma imagem aplicando uma matriz de cores com base no ângulo de rotação. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionLinearTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionlineartransfereffect)<br/>           | O efeito de transferência linear é usado para mapear as intensidades de cor de uma imagem usando uma função linear criada com uma lista de valores que você fornece para cada canal. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionMatrixTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform)<br/>                     | Representa uma transformação 2D arbitrária de afinidade definida por uma matriz 3 por 2.<br/>                                                                                                                                                                                                                                                                                                                   |
| [**IDCompositionMatrixTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform3d)<br/>                 | Representa uma transformação 3D arbitrária definida por uma matriz 4 por 4.<br/>                                                                                                                                                                                                                                                                                                                          |
| [**IDCompositionRectangleClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrectangleclip)<br/>                         | Representa um objeto de clipe que restringe a renderização de uma subárvore visual à região retangular especificada. Opcionalmente, o objeto clip pode ter cantos arredondados especificados.<br/>                                                                                                                                                                                                                 |
| [**IDCompositionRotateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform)<br/>                     | Representa uma transformação 2D que afeta a rotação de um visual em torno do eixo z. O sistema de coordenadas é girado em torno do ponto central especificado. <br/>                                                                                                                                                                                                                                   |
| [**IDCompositionSaturationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsaturationeffect)<br/>                   | Esse efeito é usado para alterar a saturação de uma imagem. O efeito de saturação é uma especialização do efeito da matriz de cores. <br/>                                                                                                                                                                                                                                                                |
| [**IDCompositionRotateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform3d)<br/>                 | Representa uma transformação 3D que afeta a rotação de um visual ao longo de um eixo arbitrário no espaço 3D. O sistema de coordenadas é girado em torno do ponto central especificado. <br/>                                                                                                                                                                                                                 |
| [**IDCompositionScaleTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform)<br/>                       | Representa uma transformação 2D que afeta a escala de um visual ao longo do eixo x e do eixo y. O sistema de coordenadas é dimensionado do ponto central especificado. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionScaleTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform3d)<br/>                   | Representa um efeito de transformação 3D que afeta a escala de um visual ao longo do eixo x, eixo y e eixo z. O sistema de coordenadas é dimensionado do ponto central especificado. <br/>                                                                                                                                                                                                               |
| [**IDCompositionSkewTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionskewtransform)<br/>                         | Representa uma transformação 2D que afeta a distorção de um visual ao longo do eixo x e do eixo y. O sistema de coordenadas é distorcedo em torno do ponto central especificado. <br/>                                                                                                                                                                                                                              |
| [**IDCompositionShadowEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionshadoweffect)<br/>                           | O efeito de sombra é usado para gerar uma sombra do canal alfa de uma imagem. A sombra é mais opaca para valores alfa mais altos e mais transparente para valores alfa inferiores. Você pode definir a quantidade de desfoque e a cor da sombra. <br/>                                                                                                                                                 |
| [**IDCompositionSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurface)<br/>                                     | Representa um bitmap físico que pode ser associado a um visual para composição em uma árvore visual. Essa interface também pode ser usada para atualizar o conteúdo do bitmap. <br/>                                                                                                                                                                                                                            |
| [**IDCompositionTableTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontabletransfereffect)<br/>             | O efeito de transferência de tabela é usado para mapear as intensidades de cor de uma imagem usando uma função de transferência criada por meio da interpolação de uma lista de valores que você fornece.<br/>                                                                                                                                                                                                                                  |
| [**IDCompositionSurfaceFactory**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurfacefactory)<br/>                       | Cria objetos de superfície e de superfície virtual associados a um dispositivo de renderização fornecido pelo aplicativo.<br/>                                                                                                                                                                                                                                                                                          |
| [**IDCompositionTarget**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontarget)<br/>                                       | Representa uma associação entre uma árvore visual DirectComposition e um destino sobre o qual a árvore visual deve ser composta. <br/>                                                                                                                                                                                                                                                            |
| [**IDCompositionTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform)<br/>                                 | Representa uma transformação 2D que pode ser usada para modificar o espaço de coordenadas de uma subárvore visual. <br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform3d)<br/>                             | Representa um efeito de transformação 3D que pode ser usado para modificar a rasterização de uma subárvore visual. <br/>                                                                                                                                                                                                                                                                                       |
| [**IDCompositionTranslateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform)<br/>               | Representa uma transformação 2D que afeta apenas o deslocamento de um visual ao longo do eixo x e do eixo y.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTurbulenceEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionturbulenceeffect)<br/>                   | O efeito de redução é usado para gerar um bitmap com base na função de ruído Perlin. O efeito de insuporte não tem nenhuma imagem de entrada. <br/>                                                                                                                                                                                                                                                              |
| [**IDCompositionTranslateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform3d)<br/>           | Representa uma transformação 3D que afeta o deslocamento de um visual ao longo do eixo x, eixo y e eixo z. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVirtualSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvirtualsurface)<br/>                       | Representa um bitmap alocado esparsamente que pode ser associado a um visual para composição em uma árvore visual.<br/>                                                                                                                                                                                                                                                                                  |
| [**IDCompositionVisual**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual)<br/>                                       | Representa um visual DirectComposition. <br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IDCompositionVisual2**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual2)<br/>                                     | Representa um visual DirectComposition em uma árvore visual.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisual3**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual3)<br/>                                     | Representa um visual DirectComposition em uma árvore visual.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisualDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisualdebug)<br/>                             | Representa um visual de depuração.<br/>                                                                                                                                                                                                                                                                                                                                                                     |
