# <a name="directcomposition-interfaces"></a>Interfaces DirectComposition

Esta seção descreve as interfaces fornecidas pela API do Microsoft DirectComposition.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                               | Descrição                                                                                                                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDCompositionAffineTransform2DEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect)<br/>     | O efeito composto aritmético é usado para combinar 2 imagens usando uma soma ponderada de pixels das imagens de entrada. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionAnimation**](/windows/desktop/api/DcompAnimation/nn-dcompanimation-idcompositionanimation)<br/>                                 | Representa uma função para animar uma ou mais propriedades de um ou mais objetos DirectComposition.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionArithmeticCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionarithmeticcompositeeffect)<br/> | O efeito composto aritmético é usado para combinar 2 imagens usando uma soma ponderada de pixels das imagens de entrada.<br/>                                                                                                                                                                                                                                                                              |
| [**IDCompositionBlendEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionblendeffect)<br/>                             | O efeito de Blend é usado para combinar 2 imagens. <br/>                                                                                                                                                                                                                                                                                                                                                 |
| [**IDCompositionBrightnessEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionbrightnesseffect)<br/>                   | O efeito de brilho controla o brilho da imagem.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionColorMatrixEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect)<br/>                 | O efeito da matriz de cores altera os valores de RGBA de um bitmap.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncompositeeffect)<br/>                     | O efeito composto é usado para combinar duas ou mais imagens. Esse efeito tem 13 modos compostos diferentes. O efeito composto aceita duas ou mais entradas. Quando você especifica 2 imagens, o destino é a primeira entrada (índice 0) e a origem é a segunda entrada (índice 1). Se você especificar mais de duas entradas, as imagens serão compostas começando com a primeira entrada e a segunda, e assim por diante. <br/> |
| [**IDCompositionClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionclip)<br/>                                           | Representa um objeto de clipe que é usado para restringir a renderização de uma subárvore visual a uma área retangular. <br/>                                                                                                                                                                                                                                                                                    |
| [**IDCompositionDesktopDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondesktopdevice)<br/>                         | Um aplicativo deve usar a interface IDCompositionDesktopDevice para usar o DirectComposition em um aplicativo de área de trabalho Win32. Essa interface permite que o aplicativo Conecte uma árvore visual a uma janela e hospede janelas filho em camadas para composição<br/>                                                                                                                                |
| [**IDCompositionDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice)<br/>                                       | Serve como um alocador para todos os outros objetos DirectComposition e fornece métodos para controlar a composição transacional.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice2**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice2)<br/>                                     | Serve como um alocador para todos os outros objetos DirectComposition e fornece métodos para controlar a composição transacional.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice3**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice3)<br/>                                     | Serve como um alocador para todos os outros objetos DirectComposition e fornece métodos para controlar a composição transacional. <br/>                                                                                                                                                                                                                                                                        |
| [**IDCompositionDeviceDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevicedebug)<br/>                             | Fornece acesso a recursos de renderização que ajudam com a depuração de aplicativos e o ajuste de desempenho. Essa interface pode ser consultada a partir da interface do dispositivo DirectComposition.<br/>                                                                                                                                                                                                                  |
| [**IDCompositionEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffect)<br/>                                       | Representa um efeito de bitmap que modifica a rasterização da subárvore de um Visual. <br/>                                                                                                                                                                                                                                                                                                             |
| [**IDCompositionEffectGroup**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffectgroup)<br/>                             | Representa um grupo de efeitos de bitmap que são aplicados em conjunto para modificar a rasterização da subárvore de um Visual. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionFilterEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionfiltereffect)<br/>                           | Representa um efeito de filtro.<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionFloodEffect**](/previous-versions/windows/desktop/legacy/dn919732(v=vs.85))<br/>                             | O efeito de inundação é usado para gerar um bitmap com base na cor especificada e no valor alfa. Você pode usar esse efeito quando desejar uma cor específica como uma entrada para um efeito, como uma cor de fundo. <br/>                                                                                                                                                                                        |
| [**IDCompositionGaussianBlurEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiongaussianblureffect)<br/>               |                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IDCompositionHueRotationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionhuerotationeffect)<br/>                 | O efeito de rotação de matiz altera o matiz de uma imagem aplicando uma matriz de cores com base no ângulo de rotação. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionLinearTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionlineartransfereffect)<br/>           | O efeito de transferência linear é usado para mapear as intensidades de cor de uma imagem usando uma função linear criada a partir de uma lista de valores que você fornece para cada canal. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionMatrixTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform)<br/>                     | Representa uma transformação 2D afim arbitrária definida por uma matriz 3-por-2.<br/>                                                                                                                                                                                                                                                                                                                   |
| [**IDCompositionMatrixTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform3d)<br/>                 | Representa uma transformação 3D arbitrária definida por uma matriz de 4 por 4.<br/>                                                                                                                                                                                                                                                                                                                          |
| [**IDCompositionRectangleClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrectangleclip)<br/>                         | Representa um objeto de clipe que restringe a renderização de uma subárvore Visual à região retangular especificada. Opcionalmente, o objeto de clipe pode ter cantos arredondados especificados.<br/>                                                                                                                                                                                                                 |
| [**IDCompositionRotateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform)<br/>                     | Representa uma transformação 2D que afeta a rotação de um Visual em volta do eixo z. O sistema de coordenadas é girado em volta do ponto central especificado. <br/>                                                                                                                                                                                                                                   |
| [**IDCompositionSaturationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsaturationeffect)<br/>                   | Esse efeito é usado para alterar a saturação de uma imagem. O efeito de saturação é uma especialização do efeito da matriz de cores. <br/>                                                                                                                                                                                                                                                                |
| [**IDCompositionRotateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform3d)<br/>                 | Representa uma transformação 3D que afeta a rotação de um Visual ao longo de um eixo arbitrário no espaço 3D. O sistema de coordenadas é girado em volta do ponto central especificado. <br/>                                                                                                                                                                                                                 |
| [**IDCompositionScaleTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform)<br/>                       | Representa uma transformação 2D que afeta a escala de um Visual ao longo do eixo x e eixo y. O sistema de coordenadas é dimensionado a partir do ponto central especificado. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionScaleTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform3d)<br/>                   | Representa um efeito de transformação 3D que afeta a escala de um Visual ao longo do eixo x, eixo y e eixo z. O sistema de coordenadas é dimensionado a partir do ponto central especificado. <br/>                                                                                                                                                                                                               |
| [**IDCompositionSkewTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionskewtransform)<br/>                         | Representa uma transformação 2D que afeta a distorção de um Visual ao longo do eixo x e eixo y. O sistema de coordenadas está distorcido em relação ao ponto central especificado. <br/>                                                                                                                                                                                                                              |
| [**IDCompositionShadowEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionshadoweffect)<br/>                           | O efeito de sombra é usado para gerar uma sombra do canal alfa de uma imagem. A sombra é mais opaca para valores Alfa mais altos e mais transparente para valores Alfa inferiores. Você pode definir a quantidade de desfoque e a cor da sombra. <br/>                                                                                                                                                 |
| [**IDCompositionSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurface)<br/>                                     | Representa um bitmap físico que pode ser associado a um Visual para composição em uma árvore visual. Essa interface também pode ser usada para atualizar o conteúdo do bitmap. <br/>                                                                                                                                                                                                                            |
| [**IDCompositionTableTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontabletransfereffect)<br/>             | O efeito de transferência de tabela é usado para mapear as intensidades de cor de uma imagem usando uma função de transferência criada por meio da interpolação de uma lista de valores que você fornece.<br/>                                                                                                                                                                                                                                  |
| [**IDCompositionSurfaceFactory**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurfacefactory)<br/>                       | Cria objetos Surface e Surface virtual associados a um dispositivo de renderização fornecido pelo aplicativo.<br/>                                                                                                                                                                                                                                                                                          |
| [**IDCompositionTarget**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontarget)<br/>                                       | Representa uma associação entre uma árvore visual DirectComposition e um destino na parte superior da qual a árvore visual deve ser composta. <br/>                                                                                                                                                                                                                                                            |
| [**IDCompositionTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform)<br/>                                 | Representa uma transformação 2D que pode ser usada para modificar o espaço de coordenadas de uma subárvore Visual. <br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform3d)<br/>                             | Representa um efeito de transformação 3D que pode ser usado para modificar a rasterização de uma subárvore Visual. <br/>                                                                                                                                                                                                                                                                                       |
| [**IDCompositionTranslateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform)<br/>               | Representa uma transformação 2D que afeta apenas o deslocamento de um Visual ao longo do eixo x e eixo y.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTurbulenceEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionturbulenceeffect)<br/>                   | O efeito turbulência é usado para gerar um bitmap com base na função de ruído Perl. O efeito turbulência não tem nenhuma imagem de entrada. <br/>                                                                                                                                                                                                                                                              |
| [**IDCompositionTranslateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform3d)<br/>           | Representa uma transformação 3D que afeta o deslocamento de um Visual ao longo do eixo x, eixo y e eixo z. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVirtualSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvirtualsurface)<br/>                       | Representa um bitmap alocado de uma abgrosseira que pode ser associado a um Visual para composição em uma árvore visual.<br/>                                                                                                                                                                                                                                                                                  |
| [**IDCompositionVisual**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual)<br/>                                       | Representa um Visual DirectComposition. <br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IDCompositionVisual2**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual2)<br/>                                     | Representa um Visual DirectComposition em uma árvore visual.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisual3**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual3)<br/>                                     | Representa um Visual DirectComposition em uma árvore visual.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisualDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisualdebug)<br/>                             | Representa um Visual de depuração.<br/>                                                                                                                                                                                                                                                                                                                                                                     |
