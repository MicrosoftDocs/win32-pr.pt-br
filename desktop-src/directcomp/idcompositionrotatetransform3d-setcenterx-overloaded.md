# <a name="idcompositionrotatetransform3dsetcenterx-methods"></a>Métodos IDCompositionRotateTransform3D:: SetCenterX

Altera ou anima o valor da propriedade CenterX de uma transformação de rotação 3D. A propriedade CenterX especifica a coordenada x do ponto sobre o qual a rotação é executada. O valor padrão é zero.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                           | Descrição                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(float))                                     | Altera o valor da propriedade CenterX.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(idcompositionanimation)) | Anima o valor da propriedade CenterX.<br/> |
