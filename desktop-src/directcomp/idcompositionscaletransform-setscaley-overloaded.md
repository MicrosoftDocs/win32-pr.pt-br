# <a name="idcompositionscaletransformsetscaley-methods"></a>Métodos IDCompositionScaleTransform:: setscaley

Altera ou anima o valor da propriedade ScaleY de uma transformação de escala 2D.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                      | Descrição                                           |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Setscaley (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscaley(float))                                     | Altera o valor da propriedade ScaleY.<br/>  |
| [**Setscaley (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscaley(idcompositionanimation)) | Anima o valor da propriedade ScaleY.<br/> |
