# <a name="grouping-api-enumerations"></a>Agrupando enumerações de API

A API de Agrupação usa as seguintes enumerações:



| Enumeração                                                                        | Descrição                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ESQUEMA DE \_ AUTENTICAÇÃO \_ DE GRUPO \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Define o conjunto de possíveis esquemas de autenticação que podem ser usados para autenticar pares que ingressaram em um grupo par.                                                                                                                                           |
| [**TIPO \_ DE EVENTO PEER GROUP \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Contém os tipos de evento de par específicos que podem ocorrer dentro de um grupo par.                                                                                                                                                                                |
| [**SINALIZADORES \_ DE \_ \_ CREDENCIAL DE PROBLEMA \_ DE GRUPO PAR**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Especifica se as credenciais do usuário são armazenadas em um grupo.                                                                                                                                                                                                  |
| [**SINALIZADORES \_ DE PROPRIEDADE DE GRUPO \_ \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Especifica as configurações de associação de grupo par.                                                                                                                                                                                                                 |
| [**\_STATUS DO GRUPO \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Indica se o grupo par tem ou não conexões presentes.                                                                                                                                                                                          |
| [**TIPO \_ DE ALTERAÇÃO DE MEMBRO \_ \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Define o conjunto de possíveis estados de presença e associação de grupo par para um par.                                                                                                                                                                         |
| [**SINALIZADORES \_ \_ DE MEMBRO PAR**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Permite que um aplicativo especifique se todos os membros ou apenas os membros presentes devem ser enumerados quando a função [**PeerGroupEnumMembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) é chamada ou para indicar se um membro está presente ou não dentro do grupo par. |
