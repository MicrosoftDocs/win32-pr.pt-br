# <a name="grouping-api-enumerations"></a>Agrupamento de enumerações de API

A API de agrupamento usa as seguintes enumerações:



| Enumeração                                                                        | Descrição                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_esquema de \_ autenticação de grupo de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Define o conjunto de possíveis esquemas de autenticação que podem ser usados para autenticar os colegas que ingressam em um grupo de pares.                                                                                                                                           |
| [**\_tipo de \_ evento de grupo de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Contém os tipos de evento de pares específicos que podem ocorrer em um grupo de pares.                                                                                                                                                                                |
| [**\_sinalizadores de \_ credenciais de problema do grupo de pares \_ \_**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Especifica se as credenciais do usuário são armazenadas em um grupo.                                                                                                                                                                                                  |
| [**\_sinalizadores de \_ Propriedade do grupo de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Especifica as configurações de associação de grupo par.                                                                                                                                                                                                                 |
| [**\_status do grupo de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Indica se o grupo de pares tem ou não as conexões presentes.                                                                                                                                                                                          |
| [**\_tipo de \_ alteração de membro de par \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Define o conjunto de possíveis membros de grupo par e Estados de presença para um par.                                                                                                                                                                         |
| [**\_sinalizadores de membro de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Permite que um aplicativo especifique se todos os membros ou apenas os presentes devem ser enumerados quando a função [**PeerGroupEnumMembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) é chamada ou para indicar se um membro está ou não presente no grupo de pares. |
