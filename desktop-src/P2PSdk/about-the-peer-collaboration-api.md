# <a name="about-the-peer-collaboration-api"></a>Sobre a API de colaboração de pares

A infraestrutura de colaboração de pares fornece uma estrutura baseada em rede de mesmo nível para atividades de colaboração sem servidor, como relacionamento de jogos de rede, conferência e outras atividades interativas de vários participantes.

A infraestrutura de colaboração de pares fornece APIs que simplificam o processo pelo qual os aplicativos podem controlar a presença de pares sem um servidor, enviar convites para os participantes, descobrir colegas na mesma sub-rede ou LAN e gerenciar contatos.

Além disso, essa infraestrutura também permite que os pares descubram outros pares na mesma sub-rede ou rede local e permite a interação entre esses pares.

A presença sem servidor é um recurso da infraestrutura de colaboração de pares disponível para todos os aplicativos do Windows Vista. Qualquer aplicativo que usa o componente de presença sem servidor da infraestrutura de colaboração poderá exibir corretamente a presença do usuário para qualquer aplicativo que o esteja assistindo. Os usuários também podem publicar informações detalhadas sobre sua presença e publicar objetos de dados registrados com o par.
