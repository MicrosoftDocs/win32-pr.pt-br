# <a name="graphing-api-constants"></a>Constantes de API de grafo

A API de Grafo usa as seguintes constantes:



| Constante                                                          | Descrição                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**TIPO \_ DE EVENTO PEER GRAPH \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | Especifica os tipos de evento par para os quais o aplicativo deve ser notificado. |
| [**SINALIZADORES \_ \_ DE PROPRIEDADE DO GRAFO \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | Especifica as propriedades de um grafo par.                             |
| [**ESCOPO \_ DO \_ GRAFO PAR**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | Especifica o escopo de rede de um grafo par.                      |
| [**SINALIZADORES \_ DE \_ STATUS DO \_ GRAFO PAR**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | Especifica o status atual de um nó dentro do grafo par.     |
| [**TIPO DE \_ \_ ALTERAÇÃO DE NÓ \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | Especifica os tipos de alterações de status do grafo de nó par.            |
