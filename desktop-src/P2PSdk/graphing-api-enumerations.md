# <a name="graphing-api-constants"></a>Constantes de API de gráfico

A API de gráfico usa as seguintes constantes:



| Constante                                                          | Descrição                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**\_tipo de \_ evento de grafo par \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | Especifica os tipos de evento de mesmo nível para os quais o aplicativo será notificado. |
| [**\_sinalizadores de \_ Propriedade do grafo par \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | Especifica as propriedades de um grafo par.                             |
| [**\_escopo do grafo par \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | Especifica o escopo de rede de um grafo par.                      |
| [**\_sinalizadores de \_ status do grafo par \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | Especifica o status atual de um nó dentro do grafo par.     |
| [**\_tipo de \_ alteração de nó par \_**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | Especifica os tipos de alterações de status de grafo de nó par.            |
