# <a name="graphing-api-structures"></a>Estruturas de API de gráfico

A API de gráfico usa as seguintes estruturas:



| Estrutura                                                                 | Descrição                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_dados de \_ alteração do nó de evento de pares \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_node_change_data)    | Contém um ponteiro para os dados se um evento de **alteração de nó de \_ evento de grafo \_ \_ \_ par** for disparado.                  |
| [**\_ \_ dados sincronizados do evento de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_synchronized_data)   | Indica o tipo de registro que foi sincronizado.                                                      |
| [**\_dados de \_ evento de grafo de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_data)                 | Contém dados associados a um evento de mesmo nível.                                                                   |
| [**\_registro de \_ evento de grafo par \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_registration) | Especifica para quais eventos de ponto um aplicativo requer notificações.                                        |
| [**\_Propriedades do grafo de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_properties)                  | Contém dados sobre a política de um grafo par, ID, escopo e outras informações.                             |
| [**\_informações de nó par \_**](/windows/desktop/api/P2P/ns-p2p-peer_node_info)                                | Contém informações específicas para um determinado nó em um grafo par.                                   |
| [**\_interface de segurança par \_**](/windows/desktop/api/P2P/ns-p2p-peer_security_interface)              | Especifica as interfaces de segurança que chamadas para APIs de grafo de pares usam para validar, proteger e liberar registros. |
