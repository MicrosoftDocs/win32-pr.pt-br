# <a name="peer-distribution"></a>Distribuição de pares

A API de Distribuição de Pares, que dá suporte ao recurso Cache de Branch no Windows 7 e no Windows Server 2008 R2, oferece um conjunto de APIs de plataforma que podem aumentar a capacidade de resposta da rede de aplicativos centralizados quando acessados de escritórios remotos, dando aos usuários nesses escritórios a experiência de trabalhar em sua rede local. Os recursos oferecidos por esse conjunto de APIs também podem ajudar a reduzir a utilização geral da WAN (rede de ampla área).

Implementar esses recursos não requer nenhuma infraestrutura pré-existente. Fazer melhorias no desempenho de redes remotas é feito simplesmente implantando o Windows 7 em computadores cliente, implantando o Windows Server 2008 R2 em computadores servidores e habilitando o recurso cache de branch.

As APIs de Distribuição de Pares funcionam perfeitamente junto com tecnologias de segurança de rede, incluindo SSL, Assinatura SMB e IPsec de ponta a ponta. Você pode usar essa API para reduzir a utilização da largura de banda de rede e melhorar o desempenho do aplicativo mesmo que o conteúdo seja criptografado.
