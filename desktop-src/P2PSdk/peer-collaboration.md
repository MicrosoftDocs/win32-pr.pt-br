# <a name="peer-collaboration"></a>Colaboração de pares

A infraestrutura de colaboração de pares fornece um conjunto de interfaces de programação de aplicativo que podem ser usadas em conjunto com a infraestrutura de pares para habilitar aplicativos de rede sem servidor de colaboração, como mensagens instantâneas e relacionamento de jogos de vários jogadores.
