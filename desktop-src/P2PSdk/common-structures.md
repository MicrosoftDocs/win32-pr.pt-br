# <a name="common-structures"></a>Estruturas comuns

A infraestrutura de mesmo nível usa as seguintes estruturas comuns.



| Estrutura                                                                          | Descrição                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**Endereço de mesmo nível \_**](/windows/desktop/api/P2P/ns-p2p-peer_address)                                              | Especifica as informações sobre o endereço IP.                                                                                     |
| [**\_informações de conexão de mesmo nível \_**](/windows/desktop/api/P2P/ns-p2p-peer_connection_info)                             | Contém informações sobre uma conexão. Essa estrutura é retornada quando você está enumerando o gráfico de pares ou as conexões de agrupamento. |
| [**dados de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_data)                                                    | Contém dados binários.                                                                                                               |
| [**\_dados de \_ alteração de conexão de evento de pares \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_connection_change_data) | Contém informações atualizadas que incluem alterações em uma conexão vizinha ou direta.                                              |
| [**\_dados de \_ entrada do evento de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_incoming_data)                    | Contém informações atualizadas que incluem alterações de dados que um nó recebe de uma conexão vizinha ou direta.                       |
| [**\_dados de \_ alteração de registro de evento de pares \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_record_change_data)         | Contém informações atualizadas que um aplicativo solicita para alterações de dados em um registro ou tipo de registro.                              |
| [**registro de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_record)                                                | Contém o objeto de registro usado por um aplicativo.                                                                                |
| [**\_dados de versão de mesmo nível \_**](/windows/desktop/api/P2P/ns-p2p-peer_version_data)                                   | Contém as informações de versão sobre o gráfico de pares e as APIs de agrupamento.                                                         |
