# <a name="collaboration-api-structures"></a>Estruturas de API de colaboração

As estruturas a seguir dão suporte às funções de colaboração de pares.

| Estrutura                                                                                      | Descrição                                                                                                                                                                                        |
|------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_aplicativo par**](/windows/desktop/api/P2P/ns-p2p-peer_application)                                                  | Contém dados que descrevem um aplicativo de software instalado localmente ou um componente que pode ser registrado e compartilhado com contatos confiáveis em uma rede de colaboração de mesmo nível.                        |
| [**\_informações de \_ registro do aplicativo par \_**](/windows/desktop/api/P2P/ns-p2p-peer_application_registration_info)            | Contém informações de aplicativo par para registro com o computador local.                                                                                                                    |
| [**\_informações de \_ inicialização do aplicativo par \_**](/windows/desktop/api/P2P/ns-p2p-peer_app_launch_info)                                        | Contém as informações de inicialização do aplicativo de mesmo nível fornecidas por um contato em uma solicitação de convite de mesmo nível anterior.                                                                              |
| [**PEER_COLLAB_EVENT_DATA**](/windows/win32/api/p2p/ns-p2p-peer_collab_event_data-r1)                                    | Contém dados de variante para cada evento de rede de colaboração de ponto possível gerado em um par.                                                                                                         |
| [**\_registro de \_ evento \_ collab de pares**](/windows/desktop/api/P2P/ns-p2p-peer_collab_event_registration)                    | Contém os dados usados por um par para se registrar para eventos de rede de colaboração de pares específicos.                                                                                                       |
| [**\_contato par**](/windows/desktop/api/P2P/ns-p2p-peer_contact)                                                          | Contém informações sobre um contato específico.                                                                                                                                                     |
| [**ponto de extremidade de par \_**](/windows/desktop/api/P2P/ns-p2p-peer_endpoint)                                                        | Contém o endereço e o nome amigável de um ponto de extremidade par.                                                                                                                                         |
| [**COLEGAS de \_ pessoas \_ ao meu redor \_**](/windows/desktop/api/P2P/ns-p2p-peer_people_near_me)                                          | Contém informações sobre um par na mesma sub-rede lógica ou virtual.                                                                                                                           |
| [**\_dados de \_ alteração do aplicativo de evento de pares \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_application_changed_data)         | Contém informações retornadas quando um aplicativo de ponto de extremidade de evento de par \_ \_ \_ \_ alterou ou evento de emparelhamento \_ \_ meu \_ \_ evento de alteração do aplicativo é gerado em um par que participa de uma rede de colaboração de mesmo nível. |
| [**dados de ponto de extremidade de evento de par \_ \_ \_ alterados \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_endpoint_changed_data)               | Contém informações retornadas quando um ponto de extremidade de evento de par \_ \_ \_ alterado ou evento de ponto \_ \_ meu \_ ponto de extremidade \_ alterado é gerado em um par que participa de uma rede de colaboração de mesmo nível.                 |
| [**\_dados de \_ alteração do objeto de evento de pares \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_object_changed_data)                   | Contém informações retornadas quando um objeto de ponto de extremidade de evento de par \_ \_ \_ \_ alterado ou evento de par \_ \_ meu \_ \_ evento de alteração de objeto é gerado em um par que participa de uma rede de colaboração de mesmo nível.           |
| [**o \_ evento de pares \_ pessoas \_ ao meu redor \_ \_ alterou \_ dados**](/windows/desktop/api/P2P/ns-p2p-peer_event_people_near_me_changed_data) | Contém informações retornadas quando um evento de evento de pares de \_ \_ pessoas \_ ao meu redor \_ \_ alterado é gerado em um par que participa de uma rede de colaboração de pares específica de sub-rede.                               |
| [**\_ \_ dados alterados de presença de evento de \_ pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_presence_changed_data)               | Contém informações retornadas quando uma \_ presença de ponto de extremidade de evento de emparelhamento \_ \_ \_ foi alterada ou evento de emparelhamento \_ \_ meu \_ evento de \_ alteração de presença é gerado em um par que participa de uma rede de colaboração de mesmo nível.       |
| [**dados de status da solicitação de evento de pares \_ \_ \_ \_ alterados \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_request_status_changed_data)  | Contém informações retornadas quando um \_ evento de status de solicitação de evento de pares \_ \_ \_ alterado é gerado em um par que participa de uma rede de colaboração de mesmo nível.                                                |
| [**\_ \_ dados alterados da inspeção de eventos \_ de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_watchlist_changed_data)             | Contém informações retornadas quando um \_ evento de alteração da inspeção de eventos de mesmo nível \_ \_ é gerado em um par que participa de uma rede de colaboração de mesmo nível.                                                      |
| [**\_convite par**](/windows/desktop/api/P2P/ns-p2p-peer_invitation)                                                    | Contém uma solicitação para iniciar ou ingressar em uma atividade de colaboração de pares.                                                                                                                              |
| [**\_resposta de convite par \_**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_response)                                 | Uma resposta a um convite para ingressar em uma atividade de colaboração de pares.                                                                                                                                 |
| [**\_objeto par**](/windows/desktop/api/P2P/ns-p2p-peer_object)                                                            | Contém informações de tempo de execução específicas do aplicativo que podem ser compartilhadas com contatos confiáveis em uma rede de colaboração de mesmo nível.                                                                   |
| [**\_informações de presença de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_presence_info)                                             | Contém informações específicas de presença de pares.                                                                                                                                                       |
