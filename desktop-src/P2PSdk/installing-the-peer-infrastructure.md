# <a name="installing-the-peer-infrastructure"></a>Instalando a infraestrutura de pares

A infraestrutura de rede de mesmo nível do Windows é uma plataforma de desenvolvedor para criar aplicativos de mesmo nível para computadores que executam o Windows XP com Service Pack 1 (SP1) com o Advanced Networking Pack, o Windows XP com Service Pack 2 (SP2), o Windows Vista e o Windows Server 2008.

Computadores que executam o Windows Vista ou o Windows Server 2008 já têm a infraestrutura de rede de pares do Windows instalada. Para computadores que executam o Windows XP com SP2, as etapas a seguir são necessárias para instalar a rede de mesmo nível do Windows:

-   Clique em **Iniciar**, clique em **Painel de Controle** e clique em **Adicionar ou Remover Programas**.
-   Clique em **Adicionar ou remover componentes do Windows**.
-   Em **componentes**, clique em **serviços de rede** (mas não marque sua caixa de seleção) e, em seguida, clique em **detalhes**.
-   Marque a caixa de seleção **par** e clique em **OK**.
-   Clique em **Avançar** e siga as instruções no assistente.

Os computadores que executam o Windows XP com SP1 exigem a instalação do [pacote de rede avançado](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4), um download gratuito, para habilitar a rede de pares.
