# <a name="people-near-me-apis"></a>APIs Pessoas ao meu Redor de Pessoas ao meu Redor

Qualquer aplicativo capaz [de Pessoas ao meu Redor](about-people-near-me.md) (PNM) pode usar as seguintes APIs win32:

### <a name="contacts-api"></a>API de Contatos

A API contatos é usada para acessar as informações de contato do usuário conectado, recuperar informações sobre contatos armazenados anteriormente ou armazenar as informações de contato e o certificado digital de um novo contato.

### <a name="people-near-me-api"></a>API do Pessoas ao meu Redor

A API pnm é usada para encontrar usuários próximos, seus interesses anunciados, quaisquer objetos arbitrários que eles optem por publicar e aplicativos anunciados.

### <a name="publication-api"></a>API de Publicação

A API de Publicação é usada para interagir com contatos remotos. A [camada de Sinalização](windows-peer-components-for-people-near-me.md) de Pares usada pelo Gerenciador de Publicações também é usada pelo módulo PNM para estabelecer conexões.

### <a name="invitation-api"></a>API de convite

A API de Convite é usada por um aplicativo de colaboração para convidar pares específicos para uma atividade de colaboração.

### <a name="application-registration-api"></a>API de Registro de Aplicativo

A API de Registro de Aplicativo é usada pelo instalador de um aplicativo de colaboração para registrar o aplicativo. Um aplicativo com capacidade para PNM solicitará ao usuário durante o processo de instalação para determinar se o aplicativo deve ser disponibilizado para os usuários na sub-rede.
