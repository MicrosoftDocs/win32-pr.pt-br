# <a name="people-near-me-apis"></a>APIs do pessoas ao meu redor

Qualquer aplicativo capaz de [pessoas ao meu redor](about-people-near-me.md) (PNM) pode usar as seguintes APIs do Win32:

### <a name="contacts-api"></a>API de contatos

A API de contatos é usada para acessar as informações de contato para o usuário conectado, recuperar informações sobre contatos armazenados anteriormente ou armazenar as informações de contato e o certificado digital de um novo contato.

### <a name="people-near-me-api"></a>API de pessoas ao meu redor

A API do PNM é usada para localizar os usuários próximos, seus interesses anunciados, quaisquer objetos arbitrários que escolherem para publicar e aplicativos anunciados.

### <a name="publication-api"></a>API de publicação

A API de publicação é usada para interagir com contatos remotos. A camada de [sinalização de par](windows-peer-components-for-people-near-me.md) usada pelo Gerenciador de publicação também é usada pelo módulo do PNM para estabelecer conexões.

### <a name="invitation-api"></a>API de convite

A API de convite é usada por um aplicativo de colaboração para convidar pares específicos para uma atividade de colaboração.

### <a name="application-registration-api"></a>API de registro de aplicativo

A API de registro do aplicativo é usada pelo instalador de um aplicativo de colaboração para registrar o aplicativo. Um aplicativo compatível com o PNM solicitará ao usuário durante o processo de instalação para determinar se o aplicativo deve ser disponibilizado para os usuários na sub-rede.
