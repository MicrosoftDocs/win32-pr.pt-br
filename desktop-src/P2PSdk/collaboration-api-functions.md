# <a name="collaboration-api-functions"></a>Funções da API de colaboração

A infraestrutura de colaboração do par dá suporte às funções a seguir.



| Função                                                                                       | Descrição                                                                                                                                                                                     |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PeerCollabAddContact**](/windows/desktop/api/P2P/nf-p2p-peercollabaddcontact)                                           | Adiciona um contato à lista de contatos de um par.                                                                                                                                                   |
| [**PeerCollabAsyncInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinvitecontact)                           | Envia um convite para um contato de par confiável para ingressar na atividade de colaboração de pares do remetente em uma conexão segura.                                                                       |
| [**PeerCollabAsyncInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinviteendpoint)                         | Envia um convite para um ponto de extremidade de par especificado para ingressar na atividade de colaboração de pares do remetente. A disponibilidade da resposta ao convite é atualizada por meio de um evento assíncrono. |
| [**PeerCollabCancelInvitation**](/windows/desktop/api/P2P/nf-p2p-peercollabcancelinvitation)                               | Cancela um convite enviado anteriormente pelo chamador para um contato.                                                                                                                               |
| [**PeerCollabCloseHandle**](/windows/desktop/api/P2P/nf-p2p-peercollabclosehandle)                                         | Fecha o identificador para um convite de atividade de colaboração de pares.                                                                                                                                  |
| [**PeerCollabDeleteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabdeletecontact)                                     | Exclui um contato do par atual.                                                                                                                                                        |
| [**PeerCollabDeleteEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteendpointdata)                           | Exclui os dados de ponto de extremidade de par no nó de par de chamada que corresponde aos dados de ponto de extremidade fornecidos.                                                                                                |
| [**PeerCollabDeleteObject**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteobject)                                       | Exclui um objeto par do ponto de extremidade de chamada.                                                                                                                                                |
| [**PeerCollabEnumApplications**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplications)                               | Retorna o identificador para uma enumeração que contém os recursos registrados para os pontos de extremidade de um par específico.                                                                                |
| [**PeerCollabEnumApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplicationregistrationinfo) | Obtém o identificador de enumeração usado para recuperar informações de aplicativo par.                                                                                                                   |
| [**PeerCollabEnumContacts**](/windows/desktop/api/P2P/nf-p2p-peercollabenumcontacts)                                       | Retorna um identificador para um conjunto enumerado que contém todos os contatos de rede de colaboração de pares disponíveis atualmente no par de chamadas.                                                     |
| [**PeerCollabEnumEndpoints**](/windows/desktop/api/P2P/nf-p2p-peercollabenumendpoints)                                     | Retorna o identificador para uma enumeração que contém os pontos de extremidade associados a um contato par específico.                                                                                       |
| [**PeerCollabEnumObjects**](/windows/desktop/api/P2P/nf-p2p-peercollabenumobjects)                                         | Retorna o identificador para uma enumeração que contém os objetos pares associados ao ponto de extremidade de um par específico.                                                                                 |
| [**PeerCollabEnumPeopleNearMe**](/windows/desktop/api/P2P/nf-p2p-peercollabenumpeoplenearme)                               | Retorna um identificador para um conjunto enumerado que contém todos os pontos de extremidade da rede de colaboração de pares "pessoas ao meu redor" disponíveis atualmente na sub-rede do ponto de chamada.                     |
| [**PeerCollabExportContact**](/windows/desktop/api/P2P/nf-p2p-peercollabexportcontact)                                     | Exporta os dados de contato associados a um nome de par para um buffer de cadeia de caracteres de dados XML de contato.                                                                                                       |
| [**PeerCollabGetAppLaunchInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplaunchinfo)                               | Obtém as informações de inicialização do aplicativo par, incluindo o nome do contato, o ponto de extremidade do par e a solicitação de convite.                                                                     |
| [**PeerCollabGetApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplicationregistrationinfo)   | Obtém informações específicas de registro do aplicativo.                                                                                                                                          |
| [**PeerCollabGetContact**](/windows/desktop/api/P2P/nf-p2p-peercollabgetcontact)                                           | Obtém as informações de um contato par específico, dado o nome do par do contato.                                                                                                         |
| [**PeerCollabGetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabgetendpointname)                                 | Recupera o nome do ponto de extremidade atual do par de chamada definido anteriormente por uma chamada para PeerCollabSetEndpointName.                                                                           |
| [**PeerCollabGetEventData**](/windows/desktop/api/P2P/nf-p2p-peercollabgeteventdata)                                       | Obtém os dados associados a um evento de colaboração de ponto gerado no par.                                                                                                                 |
| [**PeerCollabGetInvitationResponse**](/windows/desktop/api/P2P/nf-p2p-peercollabgetinvitationresponse)                     | Obtém a resposta do ponto convidado anteriormente para ingressar em uma atividade de colaboração de pares.                                                                                                        |
| [**PeerCollabGetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetpresenceinfo)                                 | Recupera as informações de presença do ponto de extremidade associado a um contato específico.                                                                                                         |
| [**PeerCollabGetSigninOptions**](/windows/desktop/api/P2P/nf-p2p-peercollabgetsigninoptions)                               | Obtém as opções de presença da rede de colaboração de pares conectado no momento do par.                                                                                                               |
| [**PeerCollabInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabinvitecontact)                                     | Envia um convite para unir uma atividade de colaboração de pares a um contato confiável. Essa chamada é síncrona e, se for bem-sucedida, obterá uma resposta do contato.                               |
| [**PeerCollabInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabinviteendpoint)                                   | Envia um convite para um ponto de extremidade de par especificado para ingressar na atividade de colaboração de pares do remetente. Essa chamada é síncrona e, se for bem-sucedida, obterá uma resposta do ponto de extremidade do par.      |
| [**PeerCollabParseContact**](/windows/desktop/api/P2P/nf-p2p-peercollabparsecontact)                                       | Analisa um buffer de cadeia de caracteres Unicode que contém dados XML de contato em uma estrutura de dados de [**\_ contato par**](/windows/desktop/api/P2P/ns-p2p-peer_contact) .                                                                         |
| [**PeerCollabQueryContactData**](/windows/desktop/api/P2P/nf-p2p-peercollabquerycontactdata)                               | Recupera as informações de contato para o ponto de extremidade de par fornecido.                                                                                                                               |
| [**PeerCollabRefreshEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabrefreshendpointdata)                         | Atualiza o nó par de chamadas com novos dados de ponto de extremidade.                                                                                                                                           |
| [**PeerCollabRegisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterapplication)                         | Registra um aplicativo com o computador local para que ele possa ser iniciado em uma atividade de colaboração de pares.                                                                                   |
| [**PeerCollabRegisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterevent)                                     | Registra um aplicativo com a infraestrutura de colaboração de pares para receber retornos de chamada para eventos de colaboração de pares específicos.                                                                |
| [**PeerCollabSetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabsetendpointname)                                 | Define o nome do ponto de extremidade atual usado pelo aplicativo par.                                                                                                                             |
| [**PeerCollabSetObject**](/windows/desktop/api/P2P/nf-p2p-peercollabsetobject)                                             | Cria ou atualiza um objeto de dados de mesmo nível usado em uma rede de colaboração de pares.                                                                                                                     |
| [**PeerCollabSetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabsetpresenceinfo)                                 | Atualiza as informações de presença do chamador para todos os contatos que o estão assistindo.                                                                                                                          |
| [**PeerCollabSignIn**](/windows/desktop/api/P2P/nf-p2p-peercollabsignin)                                                   | Assina o ponto em uma Internet hospedada (presença sem servidor) ou sub-rede ("pessoas ao meu redor") provedor de presença da rede de colaboração de pares.                                                          |
| [**PeerCollabSignOut**](/windows/desktop/api/P2P/nf-p2p-peercollabsignout)                                                 | Assina um par de um tipo específico de provedor de presença de rede de colaboração de pares.                                                                                                            |
| [**PeerCollabShutdown**](/windows/desktop/api/P2P/nf-p2p-peercollabshutdown)                                               | Desliga a infraestrutura de colaboração de pares e libera todos os recursos associados a ela.                                                                                                 |
| [**PeerCollabStartup**](/windows/desktop/api/P2P/nf-p2p-peercollabstartup)                                                 | Inicializa a infraestrutura de colaboração de pares.                                                                                                                                              |
| [**PeerCollabSubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabsubscribeendpointdata)                     | Cria uma assinatura para um ponto de extremidade disponível.                                                                                                                                                |
| [**PeerCollabUnregisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterapplication)                     | Cancela o registro de aplicativos específicos de um par do computador local.                                                                                                                        |
| [**PeerCollabUnregisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterevent)                                 | Cancela o registro de um aplicativo de notificação sobre eventos de colaboração de pares específicos.                                                                                                          |
| [**PeerCollabUnsubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabunsubscribeendpointdata)                 | Remove uma assinatura para um ponto de extremidade criado com PeerCollabSubscribeEndpointData.                                                                                                             |
| [**PeerCollabUpdateContact**](/windows/desktop/api/P2P/nf-p2p-peercollabupdatecontact)                                     | Atualiza um par que participa de uma rede de colaboração de pares com novas informações em um contato de mesmo nível.                                                                                            |
