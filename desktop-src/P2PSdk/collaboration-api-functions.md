# <a name="collaboration-api-functions"></a>Funções da API de Colaboração

A Infraestrutura de Colaboração Ponto a Ponto dá suporte às funções a seguir.



| Função                                                                                       | Descrição                                                                                                                                                                                     |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PeerCollabAddContact**](/windows/desktop/api/P2P/nf-p2p-peercollabaddcontact)                                           | Adiciona um contato à lista de contatos de um par.                                                                                                                                                   |
| [**PeerCollabAsyncInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinvitecontact)                           | Envia um convite para um contato de par confiável para ingressar na atividade de Colaboração Ponto a Ponto do remetente em uma conexão segura.                                                                       |
| [**PeerCollabAsyncInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinviteendpoint)                         | Envia um convite para um ponto de extremidade de par especificado para ingressar na atividade de Colaboração Ponto a Ponto do remetente. A disponibilidade da resposta ao convite é atualizada por meio de um evento assíncrono. |
| [**PeerCollabCancelInvitation**](/windows/desktop/api/P2P/nf-p2p-peercollabcancelinvitation)                               | Cancela um convite enviado anteriormente pelo chamador para um contato.                                                                                                                               |
| [**PeerCollabCloseHandle**](/windows/desktop/api/P2P/nf-p2p-peercollabclosehandle)                                         | Fecha o handle para um convite de atividade de Colaboração Ponto a Ponto.                                                                                                                                  |
| [**PeerCollabDeleteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabdeletecontact)                                     | Exclui um contato do par atual.                                                                                                                                                        |
| [**PeerCollabDeleteEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteendpointdata)                           | Exclui os dados do ponto de extremidade par no nó par de chamada que corresponde aos dados de ponto de extremidade fornecidos.                                                                                                |
| [**PeerCollabDeleteObject**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteobject)                                       | Exclui um objeto par do ponto de extremidade de chamada.                                                                                                                                                |
| [**PeerCollabEnumApplications**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplications)                               | Retorna o handle para uma enumeração que contém os recursos registrados em pontos de extremidade de um par específico.                                                                                |
| [**PeerCollabEnumApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplicationregistrationinfo) | Obtém o alça de enumeração usado para recuperar informações de aplicativo par.                                                                                                                   |
| [**PeerCollabEnumContacts**](/windows/desktop/api/P2P/nf-p2p-peercollabenumcontacts)                                       | Retorna um alça para um conjunto enumerado que contém todos os contatos de rede de colaboração par disponíveis atualmente no par de chamada.                                                     |
| [**PeerCollabEnumEndpoints**](/windows/desktop/api/P2P/nf-p2p-peercollabenumendpoints)                                     | Retorna o handle para uma enumeração que contém os pontos de extremidade associados a um contato par específico.                                                                                       |
| [**PeerCollabEnumObjects**](/windows/desktop/api/P2P/nf-p2p-peercollabenumobjects)                                         | Retorna o alça para uma enumeração que contém os objetos pares associados ao ponto de extremidade de um par específico.                                                                                 |
| [**PeerCollabEnumPeopleNearMe**](/windows/desktop/api/P2P/nf-p2p-peercollabenumpeoplenearme)                               | Retorna um handle para um conjunto enumerado que contém todos os pontos de extremidade de "pessoas próximas" da rede de colaboração par disponíveis atualmente na sub-rede do par de chamada.                     |
| [**PeerCollabExportContact**](/windows/desktop/api/P2P/nf-p2p-peercollabexportcontact)                                     | Exporta os dados de contato associados a um nome par para um buffer de cadeia de caracteres de dados XML de contato.                                                                                                       |
| [**PeerCollabGetAppLaunchInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplaunchinfo)                               | Obtém as informações de início do aplicativo par, incluindo o nome do contato, o ponto de extremidade par e a solicitação de convite.                                                                     |
| [**PeerCollabGetApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplicationregistrationinfo)   | Obtém informações específicas de registro de aplicativo.                                                                                                                                          |
| [**PeerCollabGetContact**](/windows/desktop/api/P2P/nf-p2p-peercollabgetcontact)                                           | Obtém as informações de um contato par específico, considerando o nome do par do contato.                                                                                                         |
| [**PeerCollabGetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabgetendpointname)                                 | Recupera o nome do ponto de extremidade atual do par de chamada definido anteriormente por uma chamada para PeerCollabSetEndpointName.                                                                           |
| [**PeerCollabGetEventData**](/windows/desktop/api/P2P/nf-p2p-peercollabgeteventdata)                                       | Obtém os dados associados a um evento de Colaboração Ponto a Ponto gerado no par.                                                                                                                 |
| [**PeerCollabGetInvitationResponse**](/windows/desktop/api/P2P/nf-p2p-peercollabgetinvitationresponse)                     | Obtém a resposta do par convidado anteriormente para ingressar em uma atividade de Colaboração Ponto a Ponto.                                                                                                        |
| [**PeerCollabGetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetpresenceinfo)                                 | Recupera as informações de presença para o ponto de extremidade associado a um contato específico.                                                                                                         |
| [**PeerCollabGetSigninOptions**](/windows/desktop/api/P2P/nf-p2p-peercollabgetsigninoptions)                               | Obtém as opções atuais de presença de rede de colaboração ponto a ponto do par.                                                                                                               |
| [**PeerCollabInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabinvitecontact)                                     | Envia um convite para ingressar em uma atividade de Colaboração Ponto a Ponto para um contato confiável. Essa chamada é síncrona e, se bem-sucedida, obtém uma resposta do contato.                               |
| [**PeerCollabInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabinviteendpoint)                                   | Envia um convite para um ponto de extremidade de par especificado para ingressar na atividade de Colaboração Ponto a Ponto do remetente. Essa chamada é síncrona e, se bem-sucedida, obtém uma resposta do ponto de extremidade par.      |
| [**PeerCollabParseContact**](/windows/desktop/api/P2P/nf-p2p-peercollabparsecontact)                                       | Analisar um buffer de cadeia de caracteres Unicode que contém dados XML de contato em uma estrutura [**de dados PEER \_ CONTACT.**](/windows/desktop/api/P2P/ns-p2p-peer_contact)                                                                         |
| [**PeerCollabQueryContactData**](/windows/desktop/api/P2P/nf-p2p-peercollabquerycontactdata)                               | Recupera as informações de contato para o ponto de extremidade par fornecido.                                                                                                                               |
| [**PeerCollabRefreshEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabrefreshendpointdata)                         | Atualiza o nó par de chamada com novos dados de ponto de extremidade.                                                                                                                                           |
| [**PeerCollabRegisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterapplication)                         | Registra um aplicativo no computador local para que ele possa ser lançado em uma atividade de colaboração ponto a ponto.                                                                                   |
| [**PeerCollabRegisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterevent)                                     | Registra um aplicativo com a infraestrutura de Colaboração Ponto a Ponto para receber retornos de chamada para eventos específicos de Colaboração Ponto a Ponto.                                                                |
| [**PeerCollabSetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabsetendpointname)                                 | Define o nome do ponto de extremidade atual usado pelo aplicativo par.                                                                                                                             |
| [**PeerCollabSetObject**](/windows/desktop/api/P2P/nf-p2p-peercollabsetobject)                                             | Cria ou atualiza um objeto de dados par usado em uma rede de colaboração par.                                                                                                                     |
| [**PeerCollabSetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabsetpresenceinfo)                                 | Atualiza as informações de presença do chamador para todos os contatos que as observam.                                                                                                                          |
| [**PeerCollabSignIn**](/windows/desktop/api/P2P/nf-p2p-peercollabsignin)                                                   | Assina o par em uma Internet hospedada (presença sem servidor) ou sub-rede ("Pessoas ao meu Redor") provedor de presença de rede de colaboração par.                                                          |
| [**PeerCollabSignOut**](/windows/desktop/api/P2P/nf-p2p-peercollabsignout)                                                 | Sai um par de um tipo específico de provedor de presença de rede de colaboração par.                                                                                                            |
| [**PeerCollabShutdown**](/windows/desktop/api/P2P/nf-p2p-peercollabshutdown)                                               | Desliga a infraestrutura de Colaboração Ponto a Ponto e libera todos os recursos associados a ela.                                                                                                 |
| [**PeerCollabStartup**](/windows/desktop/api/P2P/nf-p2p-peercollabstartup)                                                 | Inicializa a infraestrutura de Colaboração Ponto a Ponto.                                                                                                                                              |
| [**PeerCollabSubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabsubscribeendpointdata)                     | Cria uma assinatura para um ponto de extremidade disponível.                                                                                                                                                |
| [**PeerCollabUnregisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterapplication)                     | Unregisters the specific applications of a peer from the local computer.                                                                                                                        |
| [**PeerCollabUnregisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterevent)                                 | Desregula um aplicativo da notificação sobre eventos específicos de Colaboração Ponto a Ponto.                                                                                                          |
| [**PeerCollabUnsubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabunsubscribeendpointdata)                 | Remove uma assinatura de um ponto de extremidade criado com PeerCollabSubscribeEndpointData.                                                                                                             |
| [**PeerCollabUpdateContact**](/windows/desktop/api/P2P/nf-p2p-peercollabupdatecontact)                                     | Atualiza um par que participa de uma rede de colaboração par com novas informações em um contato de par.                                                                                            |
