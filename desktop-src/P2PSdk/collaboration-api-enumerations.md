# <a name="collaboration-api-enumerations"></a>Enumerações da API de Colaboração

As enumerações a seguir suportam as funções de Infraestrutura de Colaboração Ponto a Ponto.



| Enumeração                                                                         | Descrição                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TIPO DE \_ REGISTRO \_ DE APLICATIVO \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Define o conjunto de sinalizadores de registro de aplicativo par.                                                                                                                                                                                                                       |
| [**TIPO DE \_ ALTERAÇÃO \_ DE PAR**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Define o conjunto de alterações que podem ser executadas em um objeto par, ponto de extremidade ou aplicativo.                                                                                                                                                                                  |
| [**SINALIZADORES \_ DE \_ SINALIZAÇÃO PAR**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Define o conjunto de comportamentos de publicação de presença par disponíveis quando o par entrar em uma rede de colaboração par.                                                                                                                                                      |
| [**PERMISSÃO PEER \_ WATCH \_**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Define o conjunto de permissões que representam a presença de um contato para um par de observação.                                                                                                                                                                                        |
| [**TIPO \_ DE EVENTO PEER COLLAB \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Define o conjunto de eventos que podem ser gerados em um par pela infraestrutura de evento de rede de colaboração par.                                                                                                                                                                |
| [**STATUS \_ DE PRESENÇA DE \_ PARES**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Define o conjunto de possíveis configurações de status de presença disponíveis para um par que participa de uma rede de colaboração par. Essas configurações podem ser definidas por um ponto de extremidade de rede de colaboração par para indicar o nível atual de participação do contato par com seus watchers. |
| [**TIPO DE \_ RESPOSTA \_ DE CONVITE \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Define o conjunto de respostas recebidas a um convite para ingressar em uma atividade de Colaboração Ponto a Ponto.                                                                                                                                                                                 |
| [**ESCOPO DA \_ PUBLICAÇÃO \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Define o conjunto de escopos para a publicação de dados ou objetos pares.                                                                                                                                                                                                        |
