# <a name="collaboration-api-enumerations"></a>Enumerações de API de colaboração

As enumerações a seguir dão suporte às funções de infraestrutura de colaboração de pares.



| Enumeração                                                                         | Descrição                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_tipo de \_ registro de aplicativo par \_**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Define o conjunto de sinalizadores de registro de aplicativo par.                                                                                                                                                                                                                       |
| [**\_tipo de alteração de par \_**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Define o conjunto de alterações que podem ser executadas em um objeto par, ponto de extremidade ou aplicativo.                                                                                                                                                                                  |
| [**sinalizadores de entrada de pares \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Define o conjunto de comportamentos de publicação de presença de pares disponível quando o par entra em uma rede de colaboração de mesmo nível.                                                                                                                                                      |
| [**\_permissão de relógio de emparelhamento \_**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Define o conjunto de permissões que representa a presença de um contato para um ponto de observação.                                                                                                                                                                                        |
| [**\_tipo de \_ evento \_ collab de pares**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Define o conjunto de eventos que podem ser gerados em um par pela infraestrutura de eventos da rede de colaboração de pares.                                                                                                                                                                |
| [**\_status de presença de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Define o conjunto de possíveis configurações de status de presença disponíveis para um ponto que participa de uma rede de colaboração de mesmo nível. Essas configurações podem ser definidas por um ponto de extremidade de rede de colaboração de pares para indicar o nível de participação atual do contato de pares para seus inspetores. |
| [**\_tipo de \_ resposta de convite par \_**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Define o conjunto de respostas recebidas para um convite para ingressar em uma atividade de colaboração de pares.                                                                                                                                                                                 |
| [**\_escopo de publicação par \_**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Define o conjunto de escopos para a publicação de dados ou objetos pares.                                                                                                                                                                                                        |
