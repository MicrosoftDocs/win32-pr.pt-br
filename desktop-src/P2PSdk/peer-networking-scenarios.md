# <a name="peer-networking-scenarios"></a>Cenários de rede par

A infraestrutura ponto a ponto oferece várias opções para o desenvolvimento de aplicativos de rede ponto a ponto. Este tópico contém informações sobre os seguintes cenários de rede ponto a ponto:

-   Aplicativos colaborativos
-   Mensagens instantâneas ou aplicativos de chat
-   Aplicativos de distribuição de conteúdo

## <a name="collaborative-applications"></a>Aplicativos colaborativos

Você pode usar a Infraestrutura Ponto a Ponto para criar aplicativos colaborativos para que os usuários compartilhem ideias interativamente entre si, independentemente de eles estar na mesma sala de conferência ou na Internet. Por exemplo, você pode criar um aplicativo compartilhado do tipo quadro de branco.

> [!Note]  
> A infraestrutura ponto a ponto não dá suporte ao streaming de áudio ou vídeo.

 

## <a name="instant-messaging-or-chat-applications"></a>Mensagens instantâneas ou aplicativos de chat

Aplicativos de chat e mensagens instantâneas que usam servidores centrais podem ser atualizados para usar a infraestrutura ponto a ponto. Por exemplo, um aplicativo de chat pode usar a infraestrutura ponto a ponto como uma alternativa a um servidor centralizado. A infraestrutura ponto a ponto melhora a escalabilidade e permite que os usuários participem de atividades de chat e mensagens sem um servidor.

## <a name="content-distribution-applications"></a>Aplicativos de distribuição de conteúdo

Você pode usar a infraestrutura ponto a ponto como um mecanismo de distribuição para conteúdo. Por exemplo, você pode distribuir um boletim informativo para pares que assinam um serviço. Como os dados na infraestrutura ponto a ponto são sincronizados, os assinantes recebem o boletim informativo mesmo quando o publicador não está online.
