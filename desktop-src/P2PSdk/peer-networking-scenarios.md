# <a name="peer-networking-scenarios"></a>Cenários de rede de pares

A infraestrutura ponto a ponto oferece muitas opções para o desenvolvimento de aplicativos de rede ponto a ponto. Este tópico contém informações sobre os seguintes cenários de rede ponto a ponto:

-   Aplicativos de colaboração
-   Mensagens instantâneas ou aplicativos de chat
-   Aplicativos de distribuição de conteúdo

## <a name="collaborative-applications"></a>Aplicativos de colaboração

Você pode usar a infraestrutura ponto a ponto para criar aplicativos colaborativos para que os usuários compartilhem ideias interativamente entre si, estejam eles na mesma sala de conferências ou na Internet. Por exemplo, você pode criar um aplicativo semelhante ao quadro de comunicações compartilhado.

> [!Note]  
> A infraestrutura ponto a ponto não oferece suporte a streaming de áudio ou vídeo.

 

## <a name="instant-messaging-or-chat-applications"></a>Mensagens instantâneas ou aplicativos de chat

Mensagens instantâneas e aplicativos de chat que usam servidores centrais podem ser atualizados para usar a infraestrutura ponto a ponto. Por exemplo, um aplicativo de chat pode usar a infraestrutura ponto a ponto como uma alternativa para um servidor centralizado. A infraestrutura ponto a ponto melhora a escalabilidade e permite que os usuários participem de atividades de bate-papo e mensagens sem um servidor.

## <a name="content-distribution-applications"></a>Aplicativos de distribuição de conteúdo

Você pode usar a infraestrutura ponto a ponto como um mecanismo de distribuição para conteúdo. Por exemplo, você pode distribuir um boletim informativo para os colegas que assinam um serviço. Como os dados na infraestrutura ponto a ponto são sincronizados, os assinantes recebem o boletim informativo mesmo quando o Publicador não está online.
