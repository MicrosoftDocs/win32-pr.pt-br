# <a name="why-use-people-near-me"></a>Por que usar pessoas ao meu redor?

Alguns exemplos de cenários que [as pessoas ao meu redor](about-people-near-me.md) (PNM) habilitam são os seguintes:

-   Inicie o chat com uma pessoa em uma sala de conferência durante uma reunião. Use um aplicativo de chat para descobrir dinamicamente outros usuários na sala, selecione um usuário específico e inicie uma sessão de bate-papo para colaborar ou trocar informações.
-   A API do PNM permite o compartilhamento de documentos com todos em sua sala de conferência. Embora as unidades flash USB (barramento serial universal) (UFDs) facilitem muito a transferência de documentos entre computadores em uma configuração fechada, como uma reunião, ela ainda leva tempo e esforço para conectar o UFD em cada computador e copiar o documento para uma pasta local. Com aplicativos capazes de PNM, você pode selecionar os usuários na sala com os quais deseja compartilhar ou enviar um documento.
-   Transmita as informações na área de trabalho de um computador para o computador de um colega para compartilhamento de recursos.
-   Inicie um jogo com alguém durante um trânsito em um terminal de aeroporto que dê suporte à conectividade sem fio. A API do PNM permite que um aplicativo Anuncie seu interesse em participar de um jogo, descubra outros no aeroporto que também estão interessados em jogar o jogo e inicie uma sessão de jogos.
-   Anuncie interesses em uma conferência do setor e descubra os de outros que usam conectividade sem fio. Após a descoberta, os usuários podem bater papo ou configurar reuniões improvisadas para trocar ideias.
