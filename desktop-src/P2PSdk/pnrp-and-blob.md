# <a name="pnrp-and-blob"></a>PNRP e BLOB

O PNRP usa a estrutura de [**blob**](winsock-nsp-reference-links.md) para passar dados para a estrutura **WSAQUERYSET** durante chamadas para várias funções. Para uso com o PNRP, a estrutura de **blob** aponta para uma estrutura [**PNRPINFO**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpinfo_v1) ou uma estrutura [**PNRPCLOUDINFO**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpcloudinfo) .
