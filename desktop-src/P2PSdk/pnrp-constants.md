# <a name="pnrp-constants"></a>Constantes PNRP

A API do Provedor de Namespace PNRP usa as constantes a seguir.



| Constante                                                 | Descrição                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**SINALIZADORES DE \_ \_ NUVEM PNRP**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Especifica a validade de um nome de nuvem.                    |
| [**ESTADO DE \_ NUVEM \_ PNRP**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Especifica os diferentes estados em que uma nuvem PNRP pode estar.     |
| [**CRITÉRIOS DE \_ RESOLUÇÃO DE \_ PNRP**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Especifica os critérios que o PNRP usa para resolver pesquisas. |
