# <a name="pnrp-constants"></a>Constantes PNRP

A API do provedor de namespace do PNRP usa as seguintes constantes.



| Constante                                                 | Descrição                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**\_sinalizadores de nuvem PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Especifica a validade de um nome de nuvem.                    |
| [**\_estado de nuvem PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Especifica os Estados diferentes em que uma nuvem PNRP pode estar.     |
| [**\_critérios de resolução PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Especifica os critérios que o PNRP usa para resolver pesquisas. |
