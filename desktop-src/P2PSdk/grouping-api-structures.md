# <a name="grouping-api-structures"></a>Agrupando estruturas de API

A API de Agrupamento usa as estruturas a seguir.



| Estrutura                                                                  | Descrição                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**INFORMAÇÕES DE \_ CREDENCIAL \_ DE PAR**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Define as informações usadas para obter e emitir as credenciais de segurança de um par.                                   |
| [**DADOS DE \_ ALTERAÇÃO DE MEMBRO DE EVENTO \_ \_ \_ PAR**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Contém dados que descrevem uma alteração no status de um membro do grupo par.                                   |
| [**DADOS DE \_ EVENTO \_ DE GRUPO \_ PAR**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Contém informações sobre um evento de grupo par específico que ocorreu.                                     |
| [**REGISTRO \_ DE EVENTO DE GRUPO \_ \_ PAR**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Define o evento de grupo par específico para o qual um membro pode se registrar.                                            |
| [**PROPRIEDADES \_ DO GRUPO DE \_ PARES**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Contém dados sobre a política de associação de um grupo par.                                                    |
| [**INFORMAÇÕES DE \_ CONVITE \_ PAR**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Define informações sobre um convite para ingressar em um grupo par. Os convites são representados como cadeias de caracteres Unicode. |
| [**MEMBRO \_ PAR**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Contém informações que descrevem um membro de um grupo par.                                                 |
