# <a name="grouping-api-structures"></a>Agrupando estruturas de API

A API de agrupamento usa as seguintes estruturas.



| Estrutura                                                                  | Descrição                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_informações de credenciais de par \_**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Define as informações usadas para obter e emitir as credenciais de segurança de um par.                                   |
| [**\_dados de \_ alteração de membro de evento de pares \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Contém dados que descrevem uma alteração no status de um membro do grupo de pares.                                   |
| [**\_dados de \_ evento do grupo de pares \_**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Contém informações sobre um evento de grupo de pares específico que ocorreu.                                     |
| [**\_registro de \_ evento do grupo de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Define o evento de grupo par específico ao qual um membro pode se registrar.                                            |
| [**\_Propriedades do grupo de pares \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Contém dados sobre a política de associação de um grupo de pares.                                                    |
| [**\_informações de convite de mesmo nível \_**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Define informações sobre um convite para ingressar em um grupo de pares. Os convites são representados como cadeias de caracteres Unicode. |
| [**\_membro par**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Contém informações que descrevem um membro de um grupo de pares.                                                 |
