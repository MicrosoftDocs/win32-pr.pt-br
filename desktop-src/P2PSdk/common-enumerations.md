# <a name="common-constants"></a>Constantes comuns

A Infraestrutura par usa as constantes comuns a seguir.



| Constante                                                      | Descrição                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**SINALIZADORES \_ DE \_ CONEXÃO PAR**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Especifica os tipos de conexões que um par pode ter.      |
| [**STATUS \_ DA CONEXÃO PONTO A \_ PONTO**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Especifica o status de uma conexão direta ou de vizinho par. |
| [**TIPO DE \_ \_ ALTERAÇÃO DE REGISTRO \_ PAR**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Especifica as alterações que podem ocorrer em um registro.             |
| [**SINALIZADORES \_ \_ DE REGISTRO PAR**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Especifica um conjunto de sinalizadores para comportamentos de registro par.           |
