# <a name="common-constants"></a>Constantes comuns

A infraestrutura de mesmo nível usa as seguintes constantes comuns.



| Constante                                                      | Descrição                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------|
| [**\_sinalizadores de conexão de mesmo nível \_**](/windows/desktop/api/P2P/ne-p2p-peer_connection_flags)      | Especifica os tipos de conexões que um par pode ter.      |
| [**\_status de conexão de mesmo nível \_**](/windows/desktop/api/P2P/ne-p2p-peer_connection_status)    | Especifica o status de uma conexão vizinha ou direta. |
| [**\_tipo de \_ alteração de registro de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_change_type) | Especifica as alterações que podem ocorrer em um registro.             |
| [**\_sinalizadores de registro de pares \_**](/windows/desktop/api/P2P/ne-p2p-peer_record_flags)              | Especifica um conjunto de sinalizadores para comportamentos de registro de pares.           |
