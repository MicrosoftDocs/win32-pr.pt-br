# <a name="mobile-broadband"></a>Banda Larga Móvel

## <a name="purpose"></a>Finalidade

A API de banda larga móvel é usada para implementar a conectividade com redes celulares. Os aplicativos não devem se comunicar diretamente com esses dispositivos de banda larga móvel. Em vez disso, eles devem usar a API de banda larga móvel.

## <a name="developer-audience"></a>Público de desenvolvedores

A API de banda larga móvel foi projetada para desenvolvedores de C++.

Também há uma interface para desenvolvedores do .NET usando C \# . Para obter detalhes, leia este [White Paper](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx).

A API de banda larga móvel pode ser usada por aplicativos de terceiros que precisam controlar e gerenciar interfaces de banda larga móvel.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O suporte a banda larga móvel começa com o Windows 7.
