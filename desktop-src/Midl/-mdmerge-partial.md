# <a name="partial-switch"></a>comutador/partial

A opção **/partial** especifica que os tipos não resolvidos são resolvidos em relação aos arquivos de metadados listados pela opção [**/Metadata \_ dir**](-mdmerge-metadata-dir.md) .

``` syntax
mdmerge /partial 
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.

## <a name="remarks"></a>Comentários

## <a name="examples"></a>Exemplos

**mdmerge /partial**
