# <a name="metadata_dir-switch-midlrt"></a>opção/metadata_dir (MIDLRT)

A opção **/Metadata \_ dir** especifica um ou mais diretórios que contêm arquivos de metadados de plataforma.

``` syntax
midlrt /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Opções de comutação

<dl> <dt>

*diretório de metadados \_* 
</dt> <dd>

Especifica um ou mais diretórios que contêm arquivos de metadados de plataforma.

</dd> </dl>

## <a name="remarks"></a>Comentários

Use essa opção para especificar o local do arquivo de metadados principal para Windows, chamado Windows. winmd.

## <a name="examples"></a>Exemplos

**midlrt/Metadata \_ dir dir1 dir2**
