# <a name="cc-compiler-considerations"></a>C/C++-considerações do compilador

O compilador MIDL deve interoperar com o compilador C e o pré-processador C. MIDL não aceita a sintaxe C++ na entrada. O arquivo. h gerado tem definições de estilo C e C++ para interfaces. Os tópicos a seguir descrevem os requisitos para o compilador do C.

-   [C-problemas de empacotamento do compilador](c-compiler-packing-issues.md)
-   [C-definições de compilador para proxy/stubs](c-compiler-definitions-for-proxy-stubs.md)
