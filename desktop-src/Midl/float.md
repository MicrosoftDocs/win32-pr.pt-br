# <a name="float-attribute"></a>atributo float

A palavra-chave **float** designa um número de ponto flutuante de 32 bits.

``` syntax
float identifier-name;
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*nome-do-identificador* 
</dt> <dd>

Especifica um identificador MIDL válido. Os identificadores MIDL válidos consistem em até 31 caracteres alfanuméricos e/ou de sublinhado e devem começar com um caractere alfabético ou de sublinhado.

</dd> </dl>

## <a name="remarks"></a>Comentários

O tipo **float** é um dos tipos base da IDL (Interface Definition Language). O tipo **float** pode aparecer como um especificador de tipo em declarações de [**typedef**](typedef.md) , declarações gerais e declaradores de função (como um especificador de tipo de retorno de função e um especificador de tipo de parâmetro). Para o contexto no qual os especificadores de tipo aparecem, consulte [arquivo de definição de interface (IDL)](interface-definition-idl-file.md).

O tipo **float** não pode aparecer em declarações [**const**](const.md) .
