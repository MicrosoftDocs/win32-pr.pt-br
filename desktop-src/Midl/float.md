# <a name="float-attribute"></a>Atributo float

A **palavra-chave float** designa um número de ponto flutuante de 32 bits.

``` syntax
float identifier-name;
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*identifier-name* 
</dt> <dd>

Especifica um identificador MIDL válido. Os identificadores MIDL válidos consistem em até 31 caracteres alfanuméricos e/ou sublinhados e devem começar com um caractere alfabético ou sublinhado.

</dd> </dl>

## <a name="remarks"></a>Comentários

O **tipo float** é um dos tipos base da IDL (linguagem de definição de interface). O **tipo float** pode aparecer como um especificador de tipo em declarações [**typedef,**](typedef.md) declarações gerais e declaradores de função (como um especificador de tipo de retorno de função e um especificador de tipo de parâmetro). Para o contexto no qual os especificadores de tipo aparecem, consulte [Arquivo de definição de interface (IDL).](interface-definition-idl-file.md)

O **tipo float** não pode aparecer em declarações [**const.**](const.md)
