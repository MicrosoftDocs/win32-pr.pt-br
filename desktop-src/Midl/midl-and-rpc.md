# <a name="midl-and-rpc"></a>MIDL e RPC

Ao desenvolver um aplicativo RPC, você usa um editor de texto para criar um arquivo IDL que define a interface entre o cliente e o servidor. Você usa o compilador MIDL para gerar arquivos de origem que os programas de cliente e servidor exigem. Os tópicos a seguir apresentam informações sobre o uso do compilador MIDL para aplicativos RPC:

-   [Arquivos gerados para uma interface RPC](files-generated-for-an-rpc-interface.md)
-   [Direcionando stubs para plataformas específicas de 32 bits ou 64 bits](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md)

Para obter mais informações, consulte [criando aplicativos RPC](/windows/desktop/Rpc/building-rpc-applications).
