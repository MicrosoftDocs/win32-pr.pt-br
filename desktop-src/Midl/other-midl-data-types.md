# <a name="other-midl-data-types"></a>Outros tipos de dados MIDL

MIDL reconhece vários tipos de dados COM e Automation, como **BSTR,** **VARIANT** e **SAFEARRAY,** e os alças COM, como **HMETAFILE \_ PICT**, **HEN GUIDAFILE,** **HMETAFILE,** **HBITMAP,** **HPALETTE** e **HGLOBAL.** Para obter mais informações, incluindo os nomes dos arquivos de importação necessários para usar esses tipos, consulte [Marshaling de tipos](marshaling-ole-data-types.md)de dados OLE .
