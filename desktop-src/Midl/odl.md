# <a name="odl-attribute"></a>atributo odl

MKTYPLIB exigiu o atributo **\[ ODL \]** em interfaces ODL. O compilador MIDL não requer o atributo **\[ ODL \]** ; ele é reconhecido apenas por compatibilidade com arquivos **ODL** mais antigos.

> [!Note]  
> A ferramenta de Mktyplib.exe está obsoleta. Em vez disso, use o compilador MIDL.
