# <a name="odl-attribute"></a>atributo odl

MKTYPLIB exigia o atributo **\[ odl \]** em interfaces ODL. O compilador MIDL não requer o atributo **\[ odl; \]** ele é reconhecido apenas para compatibilidade com arquivos **ODL** mais antigos.

> [!Note]  
> A Mktyplib.exe de dados está obsoleta. Em vez disso, use o compilador MIDL.
