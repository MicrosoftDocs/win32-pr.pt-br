# <a name="nomd-switch"></a>comutador/nomd

A opção **/nomd** especifica que nenhum arquivo de metadados (. winmd) é gerado.

``` syntax
midlrt /nomd
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.

## <a name="remarks"></a>Comentários

## <a name="examples"></a>Exemplos

**midlrt /nomd**
