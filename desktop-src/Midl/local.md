## <a name="remarks"></a><span data-ttu-id="60397-123">Comentários</span><span class="sxs-lookup"><span data-stu-id="60397-123">Remarks</span></span>

<span data-ttu-id="60397-127">Para uma interface RPC, o atributo **\[ local \]** não pode ser usado ao mesmo tempo que o **\[** atributo [**UUID**](uuid.md) **\]** .</span><span class="sxs-lookup"><span data-stu-id="60397-127">For an RPC interface, the **\[local\]** attribute cannot be used at the same time as the **\[**[**uuid**](uuid.md)**\]** attribute.</span></span> <span data-ttu-id="60397-128">O **\[ UUID \]** ou o **\[ local \]** devem estar presentes no cabeçalho da interface e o que você escolher deve ocorrer exatamente uma vez.</span><span class="sxs-lookup"><span data-stu-id="60397-128">Either **\[uuid\]** or **\[local\]** must be present in the interface header, and the one you choose must occur exactly once.</span></span>

<span data-ttu-id="60397-129">Para uma interface COM (identificada pelo **\[** atributo de interface de [**objeto**](object.md) **\]** ), a lista de atributos de interface pode incluir o atributo **\[ local \]** , embora o **\[** atributo [**UUID**](uuid.md) **\]** esteja presente.</span><span class="sxs-lookup"><span data-stu-id="60397-129">For a COM interface (identified by the **\[**[**object**](object.md)**\]** interface attribute), the interface attribute list can include the **\[local\]** attribute even though the **\[**[**uuid**](uuid.md)**\]** attribute is present.</span></span>

<span data-ttu-id="60397-130">Quando usado em uma função individual, o atributo **\[ local \]** designa um procedimento local para o qual nenhum stub é gerado.</span><span class="sxs-lookup"><span data-stu-id="60397-130">When used in an individual function, the **\[local\]** attribute designates a local procedure for which no stubs are generated.</span></span> <span data-ttu-id="60397-131">Usar o **\[ local \]** como um atributo de função é uma extensão da Microsoft para o DCE IDL.</span><span class="sxs-lookup"><span data-stu-id="60397-131">Using **\[local\]** as a function attribute is a Microsoft extension to DCE IDL.</span></span> <span data-ttu-id="60397-132">Portanto, esse atributo não estará disponível quando você compilar usando a opção MIDL [**/OSF**](-osf.md)</span><span class="sxs-lookup"><span data-stu-id="60397-132">Therefore this attribute is not available when you compile using the MIDL [**/osf**](-osf.md) switch.</span></span>

<span data-ttu-id="60397-133">Observe que uma interface sem atributos pode ser importada para um arquivo IDL base.</span><span class="sxs-lookup"><span data-stu-id="60397-133">Note that an interface without attributes can be imported into a base IDL file.</span></span> <span data-ttu-id="60397-134">No entanto, a interface deve conter apenas tipos de texto sem procedimentos.</span><span class="sxs-lookup"><span data-stu-id="60397-134">However, the interface must contain only datatypes with no procedures.</span></span> <span data-ttu-id="60397-135">Se até um procedimento estiver contido na interface, um atributo local ou UUID deverá ser especificado.</span><span class="sxs-lookup"><span data-stu-id="60397-135">If even one procedure is contained in the interface, a local or UUID attribute must be specified.</span></span>

## <a name="examples"></a><span data-ttu-id="60397-136">Exemplos</span><span class="sxs-lookup"><span data-stu-id="60397-136">Examples</span></span>

``` syntax

    local

{ 
    void MyLocalProc(void);
} 
 
/* IDL file #2 */

    object, 
    uuid(12345678-1234-1234-123456789ABC), 
    local
]

    void MyLocalObjectProc(void);
} 
 
/* IDL file #3 */

    uuid(87654321-1234-1234-123456789ABC)
]

{ 
    [local] void MyLocalProc(void); 
    HRESULT MyRemoteProc([in] short sParam); 
}
```
