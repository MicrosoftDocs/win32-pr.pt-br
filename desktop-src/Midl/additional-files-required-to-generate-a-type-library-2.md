# <a name="additional-files-required-to-generate-a-type-library"></a>Arquivos adicionais necessários para gerar uma biblioteca de tipos

Para compilar um arquivo IDL que contém uma instrução de biblioteca, os arquivos DLL de automação OLE e OLE devem estar no seu sistema. Esses arquivos são instalados automaticamente.
