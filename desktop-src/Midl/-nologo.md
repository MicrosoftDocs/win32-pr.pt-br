# <a name="nologo-switch"></a>Opção /nologo

A **opção /nologo** desabilita a exibição da faixa de direitos autorais.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Opções de opção

Essa opção não tem parâmetros.
