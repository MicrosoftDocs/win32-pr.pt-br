# <a name="amd64-switch"></a>comutador/AMD64

A opção **/AMD64** instrui o compilador MIDL a gerar arquivos stub, ou um arquivo de biblioteca de tipos, para um ambiente avançado micro dispositivos de 64 bits.

``` syntax
midl /amd64
```

## <a name="remarks"></a>Comentários

> [!Note]  
> A opção **/x64** descrita pela ajuda da linha de comando MIDL.exe foi preterida. **/AMD64** deve ser usado em seu lugar.
