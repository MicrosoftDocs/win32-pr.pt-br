# <a name="no_cpp-nocpp"></a>/// \_ /nocpp

A opção **/do \_ cpp** especifica que o compilador MIDL não chama o pré-processador de C para pré-processar o arquivo IDL.

A opção de **\_ cpp de///** a tem precedência sobre os comutadores [**/cpp \_ cmd**](-cpp-cmd.md) e [**/cpp \_ opt**](-cpp-opt.md) .

## <a name="switch-options"></a>Opções de comutação

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>Exemplo

**MIDL/ \_ cpp nome de arquivo. idl**
