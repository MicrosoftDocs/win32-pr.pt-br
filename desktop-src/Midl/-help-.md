# <a name="help--switch"></a>opção/help (/?)

A opção **/Help** (**/?**) instrui o compilador a exibir uma mensagem de uso detalhando todas as opções de linha de comando MIDL disponíveis.

A opção [**/Confirm**](-confirm.md) exibe as configurações de opção do compilador MIDL selecionadas pelo usuário.

``` syntax
midl /help
midl /?
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.

## <a name="examples"></a>Exemplos

**MIDL/Help**

**MIDL/?**
