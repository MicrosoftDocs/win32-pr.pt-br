# <a name="no_warn-switch"></a>opção de aviso de/// \_

A opção de **\_ aviso** de alteração INSTRUI o compilador MIDL a suprimir mensagens de aviso.

``` syntax
midl /no_warn
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.

## <a name="remarks"></a>Comentários

O uso da opção **/ \_ avisar** é equivalente a **/W0**.

## <a name="examples"></a>Exemplos

**MIDL/ \_ Warn filename. idl**

**MIDL/W0 filename. idl**
