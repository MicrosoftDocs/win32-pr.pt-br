# <a name="ia64-switch"></a>comutador/IA64

A opção **/IA64** instrui o compilador MIDL a gerar arquivos stub, ou um arquivo de biblioteca de tipos, para um ambiente de arquitetura Intel de 64 bits.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Comentários

> [!Note]  
> Não é possível usar MIDL.exe para produzir um TLB de 64 bits no Windows 2000.
