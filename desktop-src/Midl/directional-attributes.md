# <a name="directional-attributes"></a>Atributos direcionais

Aplique atributos direcionais a parâmetros para indicar a direção em que os dados estão sendo passados. Ambos os atributos podem ser aplicados a um único parâmetro, mas isso aumenta a sobrecarga associada a uma chamada.



| Atributo              | Uso                                                                 |
|------------------------|-----------------------------------------------------------------------|
| [**no**](in.md)       | O parâmetro está sendo passado do chamador para a função chamada. |
| [**fora**](out-idl.md) | O parâmetro está sendo passado da função chamada para o chamador. |
