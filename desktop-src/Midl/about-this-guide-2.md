# <a name="about-this-guide"></a>Sobre este guia

Esta seção é uma referência para o linguagem IDL da Microsoft (MIDL) e um guia do usuário para o compilador MIDL, que gera arquivos de linguagem C de arquivos IDL.

A seção [usando o compilador MIDL](using-the-midl-compiler-2.md) lista os requisitos para o compilador C e antes do processador com o qual MIDL deve interoperar. Esta seção também descreve os arquivos que o compilador MIDL gera para stubs RPC, stubs DCOM e bibliotecas de tipos de automação OLE.

A seção [definições de interface e bibliotecas de tipos](interface-definitions-and-type-libraries.md) fornece informações sobre como criar arquivos de definição de interface (IDL) e de configuração de aplicativo (ACF) para MIDL, incluindo páginas de referência para atributos IDL e ACF. Também há informações sobre como importar arquivos e bibliotecas de tipos e importar arquivos de cabeçalho do sistema.

A seção de [referência de Command-Line MIDL](midl-command-line-reference.md) contém informações de referência para cada opção de linha de comando e opções de comutador reconhecidas pelo compilador MIDL.

A seção [referência de linguagem de MIDL](midl-language-reference.md) contém uma entrada de referência para cada palavra-chave na linguagem IDL da Microsoft.

A seção [erros e avisos do compilador MIDL](midl-compiler-errors-and-warnings.md) lista as mensagens de erro e de aviso que o compilador MIDL pode gerar.
