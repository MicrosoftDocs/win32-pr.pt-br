# <a name="enum_class-switch"></a>\_opção de classe/enum

A opção de **\_ classe/enum** habilita o uso da construção de classe de **Enumeração** C++.

``` syntax
midlrt /enum_class 
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.
