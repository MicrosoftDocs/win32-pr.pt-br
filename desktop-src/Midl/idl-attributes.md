# <a name="idl-attributes"></a>Atributos IDL

Os atributos são palavras-chave que especificam as características de uma interface e dos dados e métodos dentro dessa interface. O conjunto de atributos que se aplica a uma única interface é colocado entre colchetes (**\[ \]**). Os tópicos a seguir apresentam as categorias de atributos que podem ser usados em um arquivo IDL.

-   [Atributos de campo](/windows/desktop/Rpc/field-attributes)
-   [Atributos de alias e empacotamento](aliasing-and-marshaling-attributes.md)
-   [Atributos assíncronos](asynchronous-attributes.md)
-   [Atributos de matriz e de Sized-Pointer](array-and-sized-pointer-attributes.md)
-   [Atributos de tipo de dados](data-type-attributes.md)
-   [Atributos direcionais](directional-attributes.md)
-   [Atributos de chamada de função](function-call-attributes.md)
-   [Atributos de cabeçalho de interface](interface-header-attributes.md)
-   [Atributos de desempenho](performance-attributes.md)
-   [Atributos de tipo de ponteiro](pointer-type-attributes.md)
-   [Atributos de estrutura e União](structure-and-union-attributes.md)
-   [Atributos da biblioteca de tipos](type-library-attributes.md)
