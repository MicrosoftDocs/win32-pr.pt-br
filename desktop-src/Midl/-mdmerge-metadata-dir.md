# <a name="metadata_dir-switch-mdmerge"></a>opção/metadata_dir (MDMERGE)

A opção **/Metadata \_ dir** especifica o diretório que contém os arquivos de metadados da plataforma.

``` syntax
mdmerge /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Opções de comutação

<dl> <dt>

*diretório de metadados \_* 
</dt> <dd>

Especifica o diretório que contém os arquivos de metadados de plataforma.

</dd> </dl>

## <a name="remarks"></a>Comentários

Use essa opção para especificar o local do arquivo de metadados principal para Windows, chamado Windows. winmd.

## <a name="examples"></a>Exemplos

**mdmerge/Metadata \_ dir dir1**
