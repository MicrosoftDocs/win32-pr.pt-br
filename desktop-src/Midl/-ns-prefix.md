# <a name="ns_prefix-switch"></a>\_opção de prefixo/NS

A opção de **\_ prefixo/NS** precede o prefixo "Abi" para todos os tipos.

``` syntax
midlrt /ns_prefix 
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.
