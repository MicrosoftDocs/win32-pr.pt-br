# <a name="interface-definition-idl-file"></a>Arquivo de definição de interface (IDL)

Por convenção, o arquivo que contém as definições de interface e biblioteca de tipos é chamado de arquivo IDL e tem uma extensão de nome de arquivo. idl. Na realidade, o compilador MIDL analisará um arquivo de definição de interface, independentemente de sua extensão. Uma interface é identificada pela [**interface**](interface.md)de palavra-chave.

Cada interface consiste em um cabeçalho e um corpo. O cabeçalho da interface contém atributos que se aplicam à interface inteira. O corpo da interface contém as definições de interface restantes. Para obter uma visão geral de interfaces e arquivos IDL, consulte [o arquivo IDL (Interface Definition Language)](/windows/desktop/Rpc/the-interface-definition-language-idl-file). Para obter um resumo dos atributos que podem ser usados em um cabeçalho de interface, consulte [atributos IDL](idl-attributes.md).
