# <a name="o-switch-mdmerge"></a>opção/o (MDMERGE)

A opção **/o** especifica o diretório de saída para arquivos de metadados compostos.

``` syntax
mdmerge /o output_directory
```

## <a name="switch-options"></a>Opções de comutação

<dl> <dt>

*diretório de saída \_* 
</dt> <dd>

Especifica o diretório que contém os arquivos de metadados mesclados.

</dd> </dl>
