# <a name="confirm-switch"></a>/confirm switch

A **opção /confirm** instrui o compilador a exibir todas as opções do compilador MIDL sem processar a IDL de entrada e os arquivos ACF opcionais.

``` syntax
midl /confirm
```

## <a name="switch-options"></a>Opções de opção

Essa opção não tem parâmetros.

## <a name="examples"></a>Exemplos

**midl /confirm**

**midl /confirm @response.rsp filename.idl**
