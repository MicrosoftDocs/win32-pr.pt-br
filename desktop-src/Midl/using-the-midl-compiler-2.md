Os tópicos a seguir descrevem os requisitos do compilador C MIDL e do pré-processador do C, as bibliotecas de links que fazem parte do produto RPC (chamada de procedimento remoto) e os arquivos gerados pelo MIDL para interfaces DCOM, interfaces RPC e interfaces de biblioteca de tipos de automação OLE.

-   [Invocando o compilador MIDL](invoking-the-midl-compiler.md)
-   [Arquivos de resposta](response-files.md)
-   [Requisitos de pré-processador do C para MIDL](c-preprocessor-requirements-for-midl.md)
-   [C/C++-considerações do compilador](c-c-compiler-considerations.md)
-   [Usando a \_ \_ constante predefinida MIDL](using-the---midl-predefined-constant.md)
-   [MIDL e RPC](midl-and-rpc.md)
-   [MIDL e COM](midl-and-com.md)
-   [MIDL e ODL](midl-and-odl.md)

Para obter mais informações sobre os arquivos que compõem o produto RPC, consulte [instalando o ambiente de programação RPC](/windows/desktop/Rpc/installing-the-rpc-programming-environment).
