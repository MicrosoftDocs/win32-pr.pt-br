# <a name="win32-switch"></a>comutador/Win32

A opção **/Win32** instrui o compilador MIDL a gerar arquivos stub, ou um arquivo de biblioteca de tipos, para um ambiente de 32 bits.

``` syntax
midl /win32
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.

## <a name="remarks"></a>Comentários

A opção **/Win32** é equivalente a definir a opção [**/env**](-env.md) como Win32.
