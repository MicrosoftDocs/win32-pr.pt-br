# <a name="midl-enumerated-types"></a>Tipos enumerados de MIDL

A Declaração [**enum**](enum.md) não é convertida pelo compilador MIDL em instruções de **\# definição** , como é por alguns compiladores do DCE. Ele é reproduzido como uma declaração de **Enumeração** de linguagem C no arquivo de cabeçalho gerado.
