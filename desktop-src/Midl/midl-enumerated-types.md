# <a name="midl-enumerated-types"></a>Tipos enumerados MIDL

A [**declaração de enum**](enum.md) não é convertida pelo compilador MIDL em instruções **\# define,** como é feito por alguns compiladores DCE. Ele é reproduzido como uma declaração de **enum** da linguagem C no arquivo de header gerado.
