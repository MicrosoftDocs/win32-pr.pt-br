# <a name="tlb-switch"></a>opção/tlb

A opção **/tlb** especifica um nome de arquivo para a biblioteca de tipos gerada pelo compilador MIDL.

``` syntax
midl /tlb filename
```

## <a name="switch-options"></a>Opções de comutação

<dl> <dt>

*nome do arquivo* 
</dt> <dd>

Especifica o nome do arquivo de biblioteca de tipos de saída (TLB). Por padrão, se a opção **/tlb** não for usada, o arquivo tlb terá o mesmo nome que o arquivo IDL, com a extensão. TLB.

</dd> </dl>

## <a name="examples"></a>Exemplos

**MIDL/tlb newname. tlb**
