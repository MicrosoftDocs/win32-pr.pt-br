# <a name="mktyplib203-switch"></a>comutador/mktyplib203

A opção **/mktyplib203** força o compilador MIDL a analisar o arquivo de entrada.

``` syntax
midl /mktyplib203
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.

## <a name="remarks"></a>Comentários

Para usar a opção **/mktyplib203** , o arquivo de entrada deve seguir a sintaxe ODL exatamente, o que significa que você não pode inserir nenhuma instrução fora do bloco de biblioteca. Exemplos

**MIDL/mktyplib203 myoldodl. odl**

**MIDL/mktyplib203 oldhabit. idl**
