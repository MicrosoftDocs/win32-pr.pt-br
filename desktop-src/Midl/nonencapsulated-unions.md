# <a name="nonencapsulated-unions"></a>Uniões não encapsuladas

Uma União não encapsulada é uma União que não está contida em uma estrutura com seu discriminante. A União não encapsulada é indicada pela presença do **\[ \_ tipo \] de opção** de atributo de tipo e a opção de atributo de campo **\[ \_ é \]**.

Para obter uma descrição da forma de uma União não encapsulada, consulte [**Union**](union.md).

A forma de uniões deve ser a mesma entre plataformas para garantir a interconectividade.

Para obter mais informações, consulte [uniões encapsuladas](encapsulated-unions.md), [**Union**](union.md), **\[** [**switch \_**](switch-is.md) **\]** e **\[** [**\_ Type switch**](switch-type.md) **\]** .
