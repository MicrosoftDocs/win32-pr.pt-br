# <a name="nomidl-switch"></a>comutador/nomidl

A opção **/nomidl** suprime a execução de MIDL.EXE após o processamento do arquivo IDL Windows Runtime.

``` syntax
midlrt /nomidl 
```

## <a name="switch-options"></a>Opções de comutação

Essa opção não tem parâmetros.

## <a name="remarks"></a>Comentários

## <a name="examples"></a>Exemplos

**midlrt /nomidl**

| Servidor<br/> | Windows Server 2012<br/> |
