# <a name="counter-counterset-element"></a>Elemento counter (counterSet)

Identifica um contador em um conjunto de contadores que o provedor fornece.

``` syntax
<xs:element name="counter"
    type="man:counter"
 />
```

O **elemento** counter é definido pelo [**tipo complexo counterSet.**](performance-counters-counterset-complex-type.md)
