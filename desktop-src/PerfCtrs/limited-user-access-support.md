# <a name="limited-user-access-support"></a>Suporte limitado ao acesso do usuário

Somente o administrador do computador ou usuários no Grupo de Usuários de Logs de Desempenho pode registrar dados do contador. Os usuários no grupo Administrador só poderão registrar dados do contador se a ferramenta usada para registrar dados do contador for iniciada em uma janela do Prompt de Comando aberta com Executar **como administrador...**. Todos os usuários em sessões interativas de logon podem exibir dados do contador. No entanto, os usuários em sessões de logon não interativas devem estar no grupo Usuários de Monitoramento de Desempenho para exibir dados do contador.

**Windows XP:** O Administrador ou os usuários no grupo Administrador podem registrar e exibir dados do contador sem restrição.
