# <a name="limited-user-access-support"></a>Suporte limitado ao acesso do usuário

Somente o administrador do computador ou dos usuários no grupo de usuários de logs de desempenho pode registrar dados do contador. Os usuários no grupo administrador podem registrar dados do contador somente se a ferramenta que eles usam para registrar dados do contador for iniciada em uma janela de prompt de comando aberta com **Executar como administrador...**. Todos os usuários em sessões de logon interativas podem exibir dados do contador. No entanto, os usuários em sessões de logon não interativas devem estar no grupo usuários de monitoramento de desempenho para exibir dados do contador.

**Windows XP:** O administrador ou os usuários no grupo administrador podem registrar e exibir dados do contador sem restrição.
