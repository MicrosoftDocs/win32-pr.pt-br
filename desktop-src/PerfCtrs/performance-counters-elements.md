# <a name="performance-counters-elements"></a>Elementos de contadores de desempenho

O esquema de contadores de desempenho define os seguintes elementos:

-   [**contador (CounterSet)**](performance-counters-counter--counterset--element.md)
-   [**Atributo myAttribute (metaatributos)**](performance-counters-counterattribute--counterattributes--element.md)
-   [**metaatributos (contador)**](performance-counters-counterattributes--counter--element.md)
-   [**contadores**](performance-counters-counters-element.md)
-   [**CounterSet (provedor)**](performance-counters-counterset--provider--element.md)
-   [**provedor (contadores)**](performance-counters-provider--counters--element.md)
-   [**struct (Structs)**](performance-counters-struct--structs--element.md)
-   [**estruturas (CounterSet)**](performance-counters-structs--counterset--element.md)

Esta seção contém os nomes dos elementos que você usa em seu manifesto; no entanto, para obter os detalhes de cada elemento, consulte o tipo complexo que contém o elemento. O elemento raiz da seção de contadores de desempenho do manifesto é o elemento [**Counters**](performance-counters-counters-element.md) .
