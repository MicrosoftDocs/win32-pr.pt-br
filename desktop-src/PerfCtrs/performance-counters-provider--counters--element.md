# <a name="provider-counters-element"></a>Elemento Provider (Counters)

Identifica o provedor e os contadores que ele fornece.

``` syntax
<xs:element name="provider"
    type="man:provider"
 />
```

O elemento **Provider** é definido pelo tipo complexo [**Counters**](performance-counters-counters-complex-type.md) .

| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Vista\]<br/>       |
| Servidor mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Server 2008\]<br/> |
