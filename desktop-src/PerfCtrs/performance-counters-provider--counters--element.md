# <a name="provider-counters-element"></a>Elemento Provider (Counters)

Identifica o provedor e os contadores que ele fornece.

``` syntax
<xs:element name="provider"
    type="man:provider"
 />
```

O elemento **Provider** é definido pelo tipo complexo [**Counters**](performance-counters-counters-complex-type.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do vista\]<br/>       |
| Servidor mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do servidor 2008\]<br/> |
