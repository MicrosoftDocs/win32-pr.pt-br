# <a name="performance-counters"></a>Contadores de desempenho

Os contadores de desempenho do Windows fornecem uma camada de abstração de alto nível que fornece uma interface consistente para coletar vários tipos de dados do sistema, como uso de CPU, memória e disco. Os administradores do sistema geralmente usam contadores de desempenho para monitorar sistemas quanto a problemas de desempenho ou comportamento. Os desenvolvedores de software geralmente usam contadores de desempenho para examinar o uso de recursos de seus programas.
