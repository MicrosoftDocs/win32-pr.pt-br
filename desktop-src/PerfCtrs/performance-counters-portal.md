# <a name="performance-counters"></a>Contadores de desempenho

Windows Os Contadores de Desempenho fornecem uma camada de abstração de alto nível que fornece uma interface consistente para coletar vários tipos de dados do sistema, como CPU, memória e uso de disco. Os administradores do sistema geralmente usam contadores de desempenho para monitorar sistemas em busca de problemas de desempenho ou comportamento. Os desenvolvedores de software geralmente usam contadores de desempenho para examinar o uso de recursos de seus programas.
