# <a name="performance-data-helper-error-codes"></a><span data-ttu-id="11429-103">Códigos de erro de auxiliar de dados de desempenho</span><span class="sxs-lookup"><span data-stu-id="11429-103">Performance Data Helper Error Codes</span></span>

<span data-ttu-id="11429-104">Todas as funções PDH (auxiliar de dados de desempenho) retornam um valor do tipo **PDH \_ status**.</span><span class="sxs-lookup"><span data-stu-id="11429-104">All Performance Data Helper (PDH) functions return a value of type **PDH\_STATUS**.</span></span> <span data-ttu-id="11429-105">Se a função for realizada com sucesso, o valor de retorno será `ERROR_SUCCESS` .</span><span class="sxs-lookup"><span data-stu-id="11429-105">If the function succeeds, the return value is `ERROR_SUCCESS`.</span></span> <span data-ttu-id="11429-106">Caso contrário, a função retorna um código de [erro do sistema](/windows/desktop/Debug/system-error-codes) ou um código de erro PDH.</span><span class="sxs-lookup"><span data-stu-id="11429-106">Otherwise, the function returns a [system error code](/windows/desktop/Debug/system-error-codes) or a PDH error code.</span></span> <span data-ttu-id="11429-107">Para recuperar o texto de descrição do erro em seu aplicativo, use a função [**FormatMessage**](/windows/desktop/api/winbase/nf-winbase-formatmessage) , conforme mostrado no exemplo a seguir.</span><span class="sxs-lookup"><span data-stu-id="11429-107">To retrieve the description text for the error in your application, use the [**FormatMessage**](/windows/desktop/api/winbase/nf-winbase-formatmessage) function as shown in the following example.</span></span>

```C++
#include <windows.h>
#include <stdio.h>
#include <pdhmsg.h>

void main(void)
{
    HANDLE hPdhLibrary = NULL;
    LPWSTR pMessage = NULL;
    DWORD_PTR pArgs[] = { (DWORD_PTR)L"<collectionname>" };
    DWORD dwErrorCode = PDH_PLA_ERROR_ALREADY_EXISTS;

    hPdhLibrary = LoadLibrary(L"pdh.dll");
    if (NULL == hPdhLibrary)
    {
        wprintf(L"LoadLibrary failed with %lu\n", GetLastError());
        return;
    }

    if (!FormatMessage(FORMAT_MESSAGE_FROM_HMODULE
                       FORMAT_MESSAGE_ALLOCATE_BUFFER
                       FORMAT_MESSAGE_IGNORE_INSERTS,
                       hPdhLibrary,
                       dwErrorCode,
                       0,
                       (LPWSTR)&pMessage,
                       0,
                       NULL))
    {
        wprintf(L"Format message failed with 0x%x\n", GetLastError());
        return;
    }

    wprintf(L"Formatted message: %ls\n", pMessage);
    LocalFree(pMessage);
}
```
