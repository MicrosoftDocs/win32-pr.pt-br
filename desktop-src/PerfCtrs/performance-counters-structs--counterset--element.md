# <a name="structs-counterset-element"></a>Elemento structs (CounterSet)

Uma lista de estruturas que contêm os valores dos contadores no conjunto de contadores pai.

``` syntax
<xs:element name="structs"
    type="man:structs"
 />
```

O elemento **structs** é definido pelo tipo complexo [**CounterSet**](performance-counters-counterset-complex-type.md) .
