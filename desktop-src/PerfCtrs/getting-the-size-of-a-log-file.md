# <a name="getting-the-size-of-a-log-file"></a>Obtendo o tamanho de um arquivo de log

[**PdhGetLogFileSize**](/windows/desktop/api/Pdh/nf-pdh-pdhgetlogfilesize) retorna o tamanho, em bytes, do arquivo de log especificado. Use essa função quando precisar alocar memória para um buffer que receberá o conteúdo do arquivo de log.
