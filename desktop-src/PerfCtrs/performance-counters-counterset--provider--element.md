# <a name="counterset-provider-element"></a>Elemento CounterSet (provedor)

Contém uma lista de contadores que estão logicamente relacionados.

``` syntax
<xs:element name="counterSet"
    type="man:counterSet"
 />
```

O elemento **CounterSet** é definido pelo tipo complexo do [**provedor**](performance-counters-provider-complex-type.md) .

| Requisito | Valor |
