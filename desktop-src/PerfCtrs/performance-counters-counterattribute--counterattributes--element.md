# <a name="counterattribute-counterattributes-element"></a>Elemento MyAttribute (comattributes)

Identifica um atributo de um contador que descreve como o contador deve ser exibido.

``` syntax
<xs:element name="counterAttribute"
    type="man:counterAttribute"
 />
```

O elemento **comattribute** é definido pelo tipo complexo de [**atributo**](performance-counters-counterattributes-complex-type.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Vista\]<br/>       |
| Servidor mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Server 2008\]<br/> |
