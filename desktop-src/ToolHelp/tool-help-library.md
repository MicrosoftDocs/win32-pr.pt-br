# <a name="tool-help-library"></a>Biblioteca de ajuda da ferramenta

As funções fornecidas pela biblioteca de ajuda da ferramenta facilitam a obtenção de informações sobre os aplicativos em execução no momento. Essas funções foram projetadas para simplificar a criação de ferramentas, especificamente depuradores.
