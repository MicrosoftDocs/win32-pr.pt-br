# <a name="winhttp-interfaces"></a>WinHTTP Interfaces

[O Microsoft Windows Http Services (WinHTTP)](about-winhttp.md) expõe as duas interfaces a seguir:

-   [**IWinHttpRequest**](iwinhttprequest-interface.md) implementa métodos WinHTTP que não envolvem eventos.
-   [**IWinHttpRequestEvents**](iwinhttprequestevents-interface.md) implementa eventos WinHTTP e métodos baseados em eventos.
