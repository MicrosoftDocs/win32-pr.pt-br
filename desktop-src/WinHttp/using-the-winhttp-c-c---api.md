# <a name="using-the-winhttp-cc-api"></a>Usando a API do WinHTTP C/C++

Esta seção contém informações específicas sobre como usar a interface do WinHTTP C/C++.

-   As [sessões WinHTTP](winhttp-sessions-overview.md) descrevem o fluxo básico de um aplicativo WinHTTP, incluindo a ordem em que as funções são chamadas.
-   Os [identificadores do HINTERNET no WinHTTP](hinternet-handles-in-winhttp.md) descrevem os vários tipos de identificadores, as convenções de nomenclatura para esses identificadores e sua estrutura hierárquica.
-   Os [URLs (Uniform Resource Locators) no WinHTTP](uniform-resource-locators--urls--in-winhttp.md) descrevem como o WinHTTP usa URLs e fornece um exemplo de código para ilustrar o uso das funções fornecidas.
-   A [autenticação no WinHTTP](authentication-in-winhttp.md) descreve o suporte WinHTTP para autenticação de servidor e proxy de sessões http.
-   A [autenticação do Passport no WinHTTP](passport-authentication-in-winhttp.md) descreve o suporte WinHTTP para autenticação do Passport 1,4.
-   O [SSL no WinHTTP](ssl-in-winhttp.md) descreve o suporte a WinHTTP para o protocolo protocolo SSL (SSL).
-   [Usar o WinHTTP como um assembly lado a lado](using-winhttp-as-a-side-by-side-assembly.md) mostra um manifesto de aplicativo de exemplo que pode ser usado para vincular ao WinHTTP.
-   A [manipulação de cookies no WinHTTP](cookie-handling-in-winhttp.md) descreve várias opções que os aplicativos cliente do WinHTTP podem usar para lidar com cookies.
-   O [tratamento de erros no WinHTTP](error-handling-in-winhttp.md) descreve como definir opções de WinHTTP programaticamente de seu aplicativo C/C++.
-   O [suporte ao WinHTTP AutoProxy](winhttp-autoproxy-support.md) descreve as diferentes maneiras pelas quais as funções da API do WinHTTP relatam erros.
