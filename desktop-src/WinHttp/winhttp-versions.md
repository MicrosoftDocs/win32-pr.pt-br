# <a name="winhttp-versions"></a>Versões do WinHTTP

Os usuários do Microsoft Windows HTTP Services (WinHTTP) devem usar a versão mais recente da tecnologia, a versão 5,1. A versão 5,0 não é mais suportada.

## <a name="version-51"></a>Versão 5.1

O WinHTTP 5,1 oferece melhorias na versão 5,0; para obter mais informações sobre novos recursos, consulte [What ' s New in WinHTTP 5,1](what-s-new-in-winhttp-5-1.md).

Com a versão 5,1, o WinHTTP é um componente de sistema operacional dos seguintes sistemas operacionais:

-   Windows 2000, Service Pack 3 e posterior (exceto o Datacenter Server)
-   Windows XP com Service Pack 1 (SP1) e posterior
-   Windows Server 2003 com Service Pack 1 (SP1) e posterior

Para o Windows Server 2003 com SP1, o WinHTTP é um assembly lado a lado do sistema. Para obter mais informações, consulte [assemblies lado a lado](/windows/desktop/SbsCs/about-side-by-side-assemblies-).

Não há suporte para a redistribuição do WinHTTP 5,1 fora desses sistemas operacionais.

## <a name="version-50"></a>Versão 5.0

O WinHTTP 5,0 redistribuível, "winhttp5.dll", é executado no Windows 2000 ou posterior ou no Windows NT 4,0 com o Microsoft Internet Explorer 5, 1 ou posterior.

> [!IMPORTANT]
> O download do WinHTTP 5,0 não está mais disponível. A partir de 1º de outubro de 2004, a Microsoft removeu o download do SDK do WinHTTP 5,0 do MSDN e terminou o suporte do produto para a versão 5,0.
