<span id="INTERNET_DEFAULT_PORT"></span><span id="internet_default_port"></span>**\_porta padrão da Internet \_**
</dt> <dd> <dl> <dt>

0
</dt> <dt>



A porta padrão da Internet para o protocolo especificado.


</dt> </dl> </dd> <dt>

</dt> <dd> <dl> <dt>

80
</dt> <dt>



A porta HTTP da Internet padrão.


</dt> </dl> </dd> <dt>

<span id="INTERNET_DEFAULT_HTTPS_PORT"></span><span id="internet_default_https_port"></span>**\_ \_ porta https padrão da Internet \_**
</dt> <dd> <dl> <dt>

</dt> <dt>



A porta HTTPS padrão da Internet.


</dt> </dl> </dd> </dl>
