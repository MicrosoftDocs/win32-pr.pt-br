<span data-ttu-id="2f209-104">Nem todas as funções da API do WinHTTP relatam erros da mesma maneira.</span><span class="sxs-lookup"><span data-stu-id="2f209-104">Not all WinHTTP API functions report errors in the same way.</span></span>

<span data-ttu-id="2f209-105">Algumas funções, como [**WinHttpSetTimeouts**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpsettimeouts), retornam um **bool** que indica falha quando **false**.</span><span class="sxs-lookup"><span data-stu-id="2f209-105">Some functions, such as [**WinHttpSetTimeouts**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpsettimeouts), return a **BOOL** that indicates failure when **FALSE**.</span></span> <span data-ttu-id="2f209-106">Se **false** for retornado, os chamadores interessados no erro devem chamar [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span><span class="sxs-lookup"><span data-stu-id="2f209-106">If **FALSE** is returned, callers interested in the error should call [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span></span> <span data-ttu-id="2f209-107">Se **GetLastError** for chamado quando a função êxito na (retornou qualquer coisa, exceto **false**), o valor retornado será imprevisível e poderá mudar entre as versões do Windows, Service Packs ou até mesmo entre as chamadas para a mesma função.</span><span class="sxs-lookup"><span data-stu-id="2f209-107">If **GetLastError** is called when the function succeded (returned anything but **FALSE**), the returned value is unpredictable and may change between Windows versions, Service Packs, or even between calls to the same function.</span></span>

<span data-ttu-id="2f209-108">Algumas funções, como [**WinHttpConnect**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpconnect), retornam um pseudodispositivo [HINTERNET](hinternet-handles-in-winhttp.md) .</span><span class="sxs-lookup"><span data-stu-id="2f209-108">Some functions, such as [**WinHttpConnect**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpconnect), return an [HINTERNET](hinternet-handles-in-winhttp.md) pseudo handle.</span></span> <span data-ttu-id="2f209-109">Essas funções são exatamente as mesmas, exceto que a falha é indicada retornando **NULL**.</span><span class="sxs-lookup"><span data-stu-id="2f209-109">These functions are exactly the same, except failure is indicated by returning **NULL**.</span></span> <span data-ttu-id="2f209-110">Se **NULL** for retornado, os chamadores interessados no erro devem chamar [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span><span class="sxs-lookup"><span data-stu-id="2f209-110">If **NULL** is returned, callers interested in the error should call [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).</span></span> <span data-ttu-id="2f209-111">Se **GetLastError** for chamado quando a função êxito na (retornou qualquer coisa, exceto **NULL**), o valor retornado será imprevisível e poderá mudar entre as versões do Windows, Service Packs ou até mesmo entre as chamadas para a mesma função.</span><span class="sxs-lookup"><span data-stu-id="2f209-111">If **GetLastError** is called when the function succeded (returned anything but **NULL**), the returned value is unpredictable and may change between Windows versions, Service Packs, or even between calls to the same function.</span></span>

<span data-ttu-id="2f209-112">Algumas funções, como [**WinHttpGetProxyResult**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpgetproxyresult), retornam um código de erro **DWORD** e não há necessidade de chamar outras funções para obter mais informações sobre o erro.</span><span class="sxs-lookup"><span data-stu-id="2f209-112">Some functions, such as [**WinHttpGetProxyResult**](/windows/desktop/api/Winhttp/nf-winhttp-winhttpgetproxyresult), return a **DWORD** error code and there is no need to call any other functions for more error information.</span></span> <span data-ttu-id="2f209-113">Para essas funções, [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) não deve ser chamado.</span><span class="sxs-lookup"><span data-stu-id="2f209-113">For these functions, [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) should not be called.</span></span> <span data-ttu-id="2f209-114">Se **GetLastError** for chamado, independentemente do êxito ou da falha da função, o valor retornado será imprevisível e poderá mudar entre as versões do Windows, Service Packs ou até mesmo entre as chamadas para a mesma função.</span><span class="sxs-lookup"><span data-stu-id="2f209-114">If **GetLastError** is called, regardless of the success or failure of the function, the returned value is unpredictable and may change between Windows versions, Service Packs, or even between calls to the same function.</span></span>
