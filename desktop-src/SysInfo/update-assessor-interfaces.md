# <a name="update-assessor-interfaces"></a>Atualizar interfaces assessor

As interfaces a seguir são usadas pelo provedor de acessador de atualização.



| Função                               | Descrição                                                                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [**IWaaSAssessor**](/windows/desktop/api/waasapi/nn-waasapi-iwaasassessor) | Obtém a avaliação de atualização do sistema operacional comparando a compilação mais recente da Microsoft com relação à compilação em execução no dispositivo atual. |
