# <a name="object-categories"></a>Categorias de objeto

O sistema fornece três categorias de objetos: usuário, GDI (graphics device interface) e kernel. O sistema usa objetos de usuário para dar suporte ao gerenciamento de janelas, objetos GDI para dar suporte a elementos gráficos e objetos de kernel para dar suporte ao gerenciamento de memória, à execução de processos e às comunicações entre processos (IPC). Para obter informações sobre como criar e usar um objeto específico, consulte a visão geral associada.
