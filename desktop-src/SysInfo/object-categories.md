# <a name="object-categories"></a>Categorias de objeto

O sistema fornece três categorias de objetos: usuário, interface gráfica de dispositivo (GDI) e kernel. O sistema usa objetos de usuário para oferecer suporte ao gerenciamento de janelas, objetos GDI para dar suporte a gráficos e objetos kernel para dar suporte ao gerenciamento de memória, execução de processos e IPC (comunicação entre processos). Para obter informações sobre como criar e usar um objeto específico, consulte a visão geral associada.
