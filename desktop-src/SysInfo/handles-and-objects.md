# <a name="handles-and-objects"></a>Identificadores e objetos

Um *objeto* é uma estrutura de dados que representa um recurso do sistema, como um arquivo, um thread ou uma imagem gráfica. Um aplicativo não pode acessar diretamente os dados do objeto ou o recurso do sistema que um objeto representa. Em vez disso, um aplicativo deve obter um *identificador* de objeto, que pode ser usado para examinar ou modificar o recurso do sistema. Cada identificador tem uma entrada em uma tabela mantida internamente. Essas entradas contêm os endereços dos recursos e os meios para identificar o tipo de recurso.
