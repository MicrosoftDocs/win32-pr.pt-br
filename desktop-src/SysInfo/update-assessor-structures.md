# <a name="update-assessor-structures"></a>Atualizar estruturas assessor

As funções a seguir são usadas pelo provedor de acessador de atualização.



| Função                                         | Descrição                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**OSUpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment) | Define o quão atualizado o sistema operacional em um dispositivo de destino é.                |
| [**UpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment)   | Contém informações que avaliam o quão atualizado é um sistema operacional instalado. |
