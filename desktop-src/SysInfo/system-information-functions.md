# <a name="system-information-functions"></a>Informações do Sistema Funções

As funções a seguir são usadas para recuperar ou definir informações do sistema.



| Função                                                                     | Descrição                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CeipIsOptedIn**](/windows/desktop/api/Windowsceip/nf-windowsceip-ceipisoptedin)                                       | Verifica se o usuário optou pela coleta de dados do SQM como parte do Programa de Aperfeiçoamento da Experiência do Usuário (CeIP).                                                                                                       |
| [**DnsHostnameToComputerName**](/windows/desktop/api/Winbase/nf-winbase-dnshostnametocomputernamea)               | Converte um nome DNS em um nome NetBIOS.                                                                                                                                                                                            |
| [**EnumSystemFirmwareTables**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-enumsystemfirmwaretables)                 | Enumera todas as tabelas de firmware do sistema do tipo especificado.                                                                                                                                                                      |
| [**ExpandEnvironmentStrings**](/windows/win32/api/processenv/nf-processenv-expandenvironmentstringsa)                 | Substitui cadeias de caracteres de variável de ambiente pelos valores definidos.                                                                                                                                                                  |
| [**GetComputerName**](/windows/desktop/api/Winbase/nf-winbase-getcomputernamea)                                   | Recupera o nome NetBIOS do computador local.                                                                                                                                                                                 |
| [**GetComputerNameEx**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getcomputernameexa)                               | Recupera o nome NetBIOS ou DNS do computador local.                                                                                                                                                                          |
| [**GetComputerObjectName**](/windows/desktop/api/Secext/nf-secext-getcomputerobjectnamea)                       | Recupera o nome do computador local em um formato especificado.                                                                                                                                                                        |
| [**GetCurrentHwProfile**](/windows/desktop/api/Winbase/nf-winbase-getcurrenthwprofilea)                           | Recupera o perfil de hardware atual para o computador local.                                                                                                                                                                    |
| [**GetFirmwareEnvironmentVariable**](/windows/desktop/api/Winbase/nf-winbase-getfirmwareenvironmentvariablea)     | Recupera o valor da variável de ambiente de firmware especificada de NVRAM.                                                                                                                                                    |
| [**GetFirmwareEnvironmentVariableEx**](/windows/desktop/api/Winbase/nf-winbase-getfirmwareenvironmentvariableexa) | Recupera o valor da variável de ambiente de firmware especificada e seus atributos.                                                                                                                                            |
| [**GetFirmwareType**](/windows/desktop/api/Winbase/nf-winbase-getfirmwaretype)                                   | Recupera o tipo de firmware do computador local.                                                                                                                                                                                |
| [**GetIntegratedDisplaySize**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getintegrateddisplaysize)                 | Recupera a melhor estimativa do tamanho diagonal da tela integrado, em polegadas.                                                                                                                                               |
| [**GetNativeSystemInfo**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getnativesysteminfo)                           | Recupera informações sobre o sistema atual para um aplicativo em execução em WOW64.                                                                                                                                            |
| [**GetProductInfo**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getproductinfo)                                     | Recupera o tipo de produto para o sistema operacional no computador local e mapeia o tipo para os tipos de produto com suporte pelo sistema operacional especificado.                                                                    |
| [**Getsystemdirectory**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsystemdirectorya)                             | Recupera o caminho do diretório do sistema.                                                                                                                                                                                       |
| [**GetSystemFirmwareTable**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsystemfirmwaretable)                     | Recupera a tabela de firmware especificada do provedor de tabela de firmware.                                                                                                                                                          |
| [**Getsysteminfo**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsysteminfo)                                       | Recupera informações sobre o sistema atual.                                                                                                                                                                                   |
| [**GetSystemRegistryQuota**](/windows/desktop/api/Winbase/nf-winbase-getsystemregistryquota)                     | Recupera o tamanho atual do Registro e o tamanho máximo que o Registro tem permissão para obter no sistema.                                                                                                             |
| [**GetSystemWindowsDirectory**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getsystemwindowsdirectorya)               | Recupera o caminho do diretório Windows compartilhado em um sistema de vários usuários.                                                                                                                                                        |
| [**GetSystemWow64Directory**](/windows/win32/api/wow64apiset/nf-wow64apiset-getsystemwow64directorya)                   | Recupera o caminho do diretório do sistema usado pelo WOW64.                                                                                                                                                                         |
| [**GetSystemWow64Directory2**](/windows/desktop/api/wow64apiset/nf-wow64apiset-getsystemwow64directory2a)                 | Recupera o caminho do diretório do sistema usado pelo WOW64, usando o tipo de computador de arquivo de imagem especificado.                                                                                                                            |
| [**Getusername**](/windows/desktop/api/Winbase/nf-winbase-getusernamea)                                           | Recupera o nome de usuário do thread atual.                                                                                                                                                                                    |
| [**GetUserNameEx**](/windows/desktop/api/Secext/nf-secext-getusernameexa)                                       | Recupera o nome do usuário ou de outra entidade de segurança associada ao thread de chamada. Você pode especificar o formato do nome retornado.                                                                                   |
| [**Getversion**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getversion)                                             | Recupera o número de versão do sistema operacional atual.                                                                                                                                                                     |
| [**Getversionex**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getversionexa)                                         | Recupera informações sobre o sistema operacional atual.                                                                                                                                                                         |
| [**Getwindowsdirectory**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-getwindowsdirectorya)                           | Recupera o caminho do diretório Windows.                                                                                                                                                                                      |
| [**IsNativeVhdBoot**](/windows/desktop/api/Winbase/nf-winbase-isnativevhdboot)                                   | Indica se o sistema operacional foi inicializado de um contêiner VHD.                                                                                                                                                                              |
| [**IsWow64GuestMachineSupported**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64guestmachinesupported)         |                                                                                                                                                                                                                                   |
| [**IsProcessorFeaturePresent**](/windows/win32/api/processthreadsapi/nf-processthreadsapi-isprocessorfeaturepresent)               | Determina se há suporte para um recurso de processador no computador atual.                                                                                                                                                      |
| [**IsWow64Message**](/windows/desktop/api/winuser/nf-winuser-iswow64message)                                    | Determina se a última mensagem lida da fila do processo atual foi originada de um processo WOW64.                                                                                                                   |
| [**IsWow64Process**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process)                                    | Determina se um processo está em execução em WOW64.                                                                                                                                                                              |
| [**Queryperformancecounter**](/windows/win32/api/profileapi/nf-profileapi-queryperformancecounter)                   | Recupera o valor atual do contador de desempenho de alta resolução.                                                                                                                                                           |
| [**QueryPerformanceFrequency**](/windows/win32/api/profileapi/nf-profileapi-queryperformancefrequency)               | Recupera a frequência do contador de desempenho de alta resolução.                                                                                                                                                               |
| [**RtlGetSuiteMask**](rtlgetsuitemask.md)                                   | Recupera uma máscara de bit que identifica os pacote de produtos disponíveis no sistema. Se essa função for chamada em um aplicativo executado no contexto de um silo de servidor, a máscara de pacote para o silo do servidor será recuperada. |
| [**SetComputerName**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-setcomputernamea)                                   | Define um novo nome NetBIOS para o computador local.                                                                                                                                                                                   |
| [**SetComputerNameEx**](/windows/win32/api/sysinfoapi/nf-sysinfoapi-setcomputernameexa)                               | Define um novo nome NetBIOS ou DNS para o computador local.                                                                                                                                                                            |
| [**SetFirmwareEnvironmentVariable**](/windows/desktop/api/Winbase/nf-winbase-setfirmwareenvironmentvariablea)     | Define o valor da variável de ambiente de firmware especificada.                                                                                                                                                                    |
| [**SetFirmwareEnvironmentVariableEx**](/windows/desktop/api/Winbase/nf-winbase-setfirmwareenvironmentvariableexa) | Define o valor da variável de ambiente de firmware especificada e seus atributos.                                                                                                                                                 |
| [**TranslateName**](/windows/desktop/api/Secext/nf-secext-translatenamea)                                       | Converte um nome de objeto de serviço de diretório de um formato em outro.                                                                                                                                                              |
| [**VerifyVersionInfo**](/windows/desktop/api/Winbase/nf-winbase-verifyversioninfoa)                               | Compara um conjunto de requisitos de versão com os valores do sistema operacional atual.                                                                                                                                            |
| [**VerSetConditionMask**](/windows/desktop/api/Winnt/nf-winnt-versetconditionmask)                           | Cria a máscara de condição para a [**função VerifyVersionInfo.**](/windows/desktop/api/Winbase/nf-winbase-verifyversioninfoa)                                                                                                                                        |
| [**Wow64DisableWow64FsRedirection**](/windows/desktop/api/wow64apiset/nf-wow64apiset-wow64disablewow64fsredirection)    | Desabilita o redirecionamento do sistema de arquivos para o thread de chamada.                                                                                                                                                                          |
| [**Wow64EnableWow64FsRedirection**](/windows/desktop/api/winbase/nf-winbase-wow64enablewow64fsredirection)      | Habilita ou desabilita o redirecionamento do sistema de arquivos para o thread de chamada.                                                                                                                                                               |
| [**Wow64RevertWow64FsRedirection**](/windows/desktop/api/wow64apiset/nf-wow64apiset-wow64revertwow64fsredirection)      | Restaura o redirecionamento do sistema de arquivos para o thread de chamada.                                                                                                                                                                          |
