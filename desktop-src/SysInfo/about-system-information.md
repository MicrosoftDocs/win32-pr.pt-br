# <a name="about-system-information"></a>Sobre as informações do sistema

As funções descritas nesta visão geral recuperam uma variedade de dados. As informações de configuração do sistema descrevem o hardware do computador, o nome do computador e o nome de usuário. As informações de configuração do sistema operacional descrevem a versão do sistema operacional, os diretórios-chave e as variáveis de ambiente. Os parâmetros do sistema descrevem os atributos do sistema. As métricas do sistema fornecem as dimensões dos elementos na área de exibição.

As seções a seguir descrevem como obter esses tipos de informação:

-   [Configuração de hardware](hardware-configuration.md)
-   [Nomes de computador](computer-names.md)
-   [Versão do sistema operacional](operating-system-version.md)
-   [Configuração do sistema operacional](operating-system-configuration.md)
