# <a name="update-assessor-enumerations"></a>Atualizar enumerações assessor

As enumerações a seguir são usadas pelo provedor de acessador de atualização.



| Função                                                 | Descrição                                                                    |
|----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**UpdateAssessmentStatus**](updateassessmentstatus.md) | Descreve o quão atualizado o sistema operacional em um dispositivo é.                                |
| [**UpdateImpactLevel**](updateimpactlevel.md)           | Indica um impacto alto, médio ou baixo de um dispositivo que executa um sistema operacional desatualizado. |
