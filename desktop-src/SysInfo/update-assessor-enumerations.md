# <a name="update-assessor-enumerations"></a>Atualizar enumerações de avaliador

As enumerações a seguir são usadas pelo provedor do Acessador de Atualizações.



| Função                                                 | Descrição                                                                    |
|----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**UpdateAssessmentStatus**](updateassessmentstatus.md) | Descreve como o sistema operacional em um dispositivo está atualizado.                                |
| [**UpdateImpactLevel**](updateimpactlevel.md)           | Indica um impacto alto, médio ou baixo de um dispositivo executando um sistema operacional des date. |
