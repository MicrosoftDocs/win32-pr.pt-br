# <a name="handle-and-object-functions"></a>Funções de identificador e objeto

As funções a seguir são usadas com identificadores e objetos.



| Função                                             | Descrição                                                                                  |
|------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**CloseHandle**](/windows/win32/api/handleapi/nf-handleapi-closehandle)                   | Fecha um identificador de objeto aberto.                                                                |
| [**CompareObjectHandles**](/windows/desktop/api/handleapi/nf-handleapi-compareobjecthandles) | Compara dois identificadores de objeto para determinar se eles se referem ao mesmo objeto de kernel subjacente. |
| [**DuplicateHandle**](/windows/win32/api/handleapi/nf-handleapi-duplicatehandle)           | Duplica um identificador de objeto.                                                                 |
| [**GetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-gethandleinformation) | Recupera certas propriedades de um identificador de objeto.                                            |
| [**SetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-sethandleinformation) | Define certas propriedades de um identificador de objeto.                                                 |
