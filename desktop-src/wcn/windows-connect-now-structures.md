# <a name="windows-connect-now-structures"></a>Estruturas do Windows Connect Now

As estruturas a seguir dão suporte às funções da API do Windows Connect Now.



| Estrutura                                                                             | Descrição                                                                                                                                            |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**tipo do valor do WCN tipo de \_ \_ \_ \_ dispositivo primário \_**](/windows/win32/api/wcntypes/ns-wcntypes-wcn_value_type_primary_device_type) | Contém dados enumerados que identificam o tipo de um dispositivo por categoria, subcategoria e uma OUI (ID da organização) específica do fabricante.<br/> |
| [**\_especificação de \_ extensão de fornecedor do WCN \_**](/windows/win32/api/wcndevice/ns-wcndevice-wcn_vendor_extension_spec)                     | Contém dados que definem uma extensão de fornecedor. <br/>                                                                                             |
