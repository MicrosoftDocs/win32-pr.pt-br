# <a name="windows-connect-now-interfaces"></a>Interfaces do Windows Connect Now

A [API do Windows Connect Now](portal.md) (WCNAPI) define e utiliza as seguintes interfaces.



| Interface                                      | Descrição                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | Receba uma notificação de êxito ou falha quando uma sessão do Windows Connect Now for concluída.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configura o dispositivo e inicia a sessão.<br/>                                                |
