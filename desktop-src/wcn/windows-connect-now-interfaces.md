# <a name="windows-connect-now-interfaces"></a>Conexão Fácil do Windows Interface

a [API de Conexão Fácil do Windows](portal.md) (WCNAPI) define e utiliza as interfaces a seguir.



| Interface                                      | Descrição                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | receba uma notificação de êxito ou de falha quando uma sessão do Conexão Fácil do Windows conexão for concluída.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configura o dispositivo e inicia a sessão.<br/>                                                |
