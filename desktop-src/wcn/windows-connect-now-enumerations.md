# <a name="windows-connect-now-enumerations"></a>Enumerações agora do Windows Connect

Agora, o Windows Connect define os seguintes tipos de enumeração.



| Enumeração                                                                                             | Descrição                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**\_tipo de atributo WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Define os vários tipos de buffer de atributo definidos para Wi-Fi configuração protegida.<br/>                                      |
| [**\_tipo de senha do WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Define a autenticação que será usada em uma sessão do WPS. <br/>                                                    |
| [**\_status da sessão WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Define o status de uma sessão do WPS.<br/>                                                                               |
| [**\_estado de \_ associação do tipo de valor WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Define os Estados de associação possíveis de uma estação sem fio durante uma solicitação de descoberta.<br/>                          |
| [**tipo de valor do WCN \_ \_ \_ booliano**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Define os valores usados para representar as condições true/false encontradas durante a instalação e a associação do dispositivo.<br/>            |
| [**\_tipo de \_ autenticação do tipo de valor WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Define os métodos de autenticação com suporte pelo registro (ponto de acesso ou estação).<br/>                            |
| [**\_métodos de \_ configuração do tipo de valor WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Define os métodos de configuração com suporte no registro ou registrador.<br/>                                          |
| [**\_erro de \_ configuração do tipo de valor WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Define os possíveis valores de erro retornados a um dispositivo ao tentar configurar o para o e associá-lo ao WLAN.<br/> |
| [**\_ID de \_ senha do dispositivo do tipo de valor WCN \_ \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Define os valores de senha possíveis.<br/>                                                                              |
| [**\_tipo de \_ criptografia do tipo de valor WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Define os tipos de criptografia de WLAN com suporte.<br/>                                                                       |
| [**\_ \_ faixas RF do tipo de valor WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Define a banda de frequência de rádio na qual um registro envia solicitações de descoberta.<br/>                                    |
| [**\_versão do \_ tipo de valor do WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Define a versão com suporte da WPS (configuração protegida do Wi-Fi).<br/>                                                      |
| [**tipo de valor do WCN \_ \_ \_ estado da \_ \_ \_ instalação protegida \_ do Wi-Fi**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Define valores que indicam se um dispositivo está configurado.<br/>                                                            |
