# <a name="windows-connect-now-enumerations"></a>Conexão Fácil do Windows Enumerações

Conexão Fácil do Windows define os seguintes tipos de enumeração.



| Enumeração                                                                                             | Descrição                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**TIPO DE ATRIBUTO WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Define os vários tipos de buffer de atributo definidos para Wi-Fi Configuração Protegida.<br/>                                      |
| [**TIPO DE SENHA WCN \_ \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Define a autenticação que será usada em uma sessão WPS. <br/>                                                    |
| [**STATUS DA \_ SESSÃO WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Define o status de uma sessão WPS.<br/>                                                                               |
| [**ESTADO DE ASSOCIAÇÃO \_ DO \_ TIPO DE VALOR \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Define os possíveis estados de associação de uma estação sem fio durante uma solicitação de descoberta.<br/>                          |
| [**TIPO DE VALOR WCN \_ \_ \_ BOOLEAN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Define os valores usados para representar condições true/false encontradas durante a configuração e a associação do dispositivo.<br/>            |
| [**TIPO DE \_ AUTENTICAÇÃO \_ DE TIPO DE \_ VALOR \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Define os métodos de autenticação com suporte pelo Enrollee (ponto de acesso ou estação).<br/>                            |
| [**MÉTODOS DE \_ CONFIGURAÇÃO \_ DE TIPO \_ \_ DE VALOR WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Define os métodos de configuração com suporte pelo Enrollee ou pelo Registrador.<br/>                                          |
| [**ERRO DE \_ CONFIGURAÇÃO \_ DO TIPO DE VALOR \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Define possíveis valores de erro retornados a um dispositivo durante a tentativa de configurar e associar ao WLAN.<br/> |
| [**ID DA SENHA DO DISPOSITIVO DO TIPO DE VALOR WCN \_ \_ \_ \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Define os valores de senha possíveis.<br/>                                                                              |
| [**TIPO DE \_ CRIPTOGRAFIA DE TIPO DE \_ \_ VALOR \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Define os tipos de criptografia WLAN com suporte.<br/>                                                                       |
| [**FAIXAS DE \_ RF DO TIPO DE VALOR \_ \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Define a faixa de frequência de rádio na qual um inscrito envia solicitações de descoberta.<br/>                                    |
| [**VERSÃO DO TIPO \_ DE \_ VALOR WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Define a versão com suporte do WPS (Wi-Fi Configuração Protegida).<br/>                                                      |
| [**ESTADO DE INSTALAÇÃO PROTEGIDO POR \_ \_ WI \_ \_ FI \_ \_ \_ DO TIPO DE VALOR WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Define valores que indicam se um dispositivo está configurado.<br/>                                                            |
