O CIM \_ Studio e outras ferramentas administrativas do WMI podem ser baixados das [Ferramentas administrativas do WMI](https://www.microsoft.com/downloads/details.aspx?FamilyId=6430F853-1120-48DB-8CC5-F2ABDC3ED314).

As ferramentas WMI incluem:

-   WMI CIM Studio

    A ferramenta CIM Studio permite que você trabalhe com classes e namespaces WMI no repositório CIM. Também é uma ferramenta importante para o desenvolvimento do provedor de WMI, pois ajuda na criação da classe WMI inicial no repositório do CIM.

    O WMI CIM Studio fornece a capacidade de:

    -   Conecte-se a um sistema escolhido e procure o repositório CIM em qualquer namespace disponível.
    -   Pesquise classes por nome, descrições ou nomes de propriedade.
    -   Examine as propriedades, os métodos e as associações relacionadas a uma determinada classe.
    -   Consulte as instâncias disponíveis para uma determinada classe.
    -   Execute consultas na linguagem WQL.
    -   Gerar um arquivo MOF com base em classes selecionadas.
    -   Compile um arquivo MOF que você deseja carregar no repositório CIM.

-   Pesquisador de objetos WMI

    O pesquisador de objetos permite que você exiba os objetos WMI presentes no seu computador. A raiz da hierarquia é a instância de [**\_ ComputerSystem do Win32**](/windows/desktop/CIMWin32Prov/win32-computersystem) que representa o sistema de computador. Usando o pesquisador de objetos, você pode editar valores de propriedade e qualificadores e métodos de execução.

-   Registro de eventos WMI

    A ferramenta de registro de eventos permite criar, exibir e modificar consumidores de eventos, filtros e temporizadores para um namespace especificado. É uma ferramenta para desenvolvedores e administradores de sistema.

-   Visualizador de Eventos WMI

    Você pode usar o Visualizador de Eventos para exibir a ocorrência de eventos WMI configurados no registro de eventos WMI.

Você pode baixar o PowerShell RC1 e a documentação em [https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx](https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx) .
