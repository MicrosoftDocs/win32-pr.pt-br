# <a name="accessing-other-operating-system-features-with-wmi"></a>Acessando outros recursos do sistema operacional com o WMI

O WMI tem extensões ou provedores que permitem que você acesse outros recursos do sistema operacional, como Active Directory, ODBC e SNMP. Você também pode acessar dados de drivers de dispositivo.

As extensões WMI não estão disponíveis em todos os sistemas operacionais. Para obter mais informações, consulte [disponibilidade do sistema operacional de componentes WMI](operating-system-availability-of-wmi-components.md).

Os tópicos a seguir fornecem informações sobre como acessar outros recursos do sistema operacional:

-   [Acessando dados de drivers de dispositivo](accessing-data-from-device-drivers.md)
-   [Acessando Active Directory](accessing-active-directory.md)
-   [Acessando dispositivos SNMP](accessing-snmp-devices.md)
