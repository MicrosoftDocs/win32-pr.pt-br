# <a name="modifying-an-instance-property"></a>Modificando uma propriedade de instância

Depois de criar ou localizar uma instância de uma classe, você provavelmente desejará modificar a instância. Por exemplo, talvez você queira alterar o nome do nome do sistema armazenado na instância local da classe de [**\_ ComputerSystem do Win32**](/windows/desktop/CIMWin32Prov/win32-computersystem) . A maneira mais comum de atualizar uma instância é atualizar a instância inteira. Para obter mais informações, consulte [atualizando uma instância inteira](updating-an-entire-instance.md). A segunda opção é atualizar apenas parte da instância. Para obter mais informações, consulte [atualizando parte de uma instância](updating-part-of-an-instance.md).

Para obter mais informações sobre como trabalhar com classes, consulte [manipulando informações de classe e instância](manipulating-class-and-instance-information.md).
