# <a name="application-inventory-classes"></a>Classes de inventário de aplicativos

## <a name="purpose"></a>Finalidade

as classes WMI para o inventário de aplicativos habilitam a descoberta dos aplicativos Win32 instalados e Windows aplicativos de armazenamento em um sistema Windows.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                  | Descrição                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_InstalledWin32Program Win32**](win32-installedwin32program.md)<br/>         | A classe [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) representa um aplicativo Win32 instalado.<br/>                                                                                                                            |
| [**\_InstalledStoreProgram Win32**](win32-installedstoreprogram.md)<br/>         | a classe [**Win32 \_ InstalledStoreProgram**](win32-installedstoreprogram.md) representa um aplicativo de repositório Windows instalado.<br/>                                                                                                                    |
| [**\_InstalledProgramFramework Win32**](win32-installedprogramframework.md)<br/> | A classe [**Win32 \_ InstalledProgramFramework**](win32-installedprogramframework.md) representa uma estrutura tecnológica na qual uma instância do [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) é compilada ou usada no tempo de execução.<br/> |



 

## <a name="developer-audience"></a>Público de desenvolvedores

as classes WMI para o inventário de aplicativos são criadas para uso por desenvolvedores de soluções de gerenciamento de Enterprise e por ferramentas de administração locais ou remotas que precisam gerar um inventário de aplicativos em um sistema.
