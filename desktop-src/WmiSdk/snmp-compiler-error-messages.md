# <a name="snmp-compiler-error-messages"></a>Mensagens de erro do compilador SNMP

Esta seção descreve as várias condições de erro que podem ocorrer quando o compilador do módulo de informações de SNMP (Simple Network Management Protocol) analisa um módulo de MIB (base de informações de gerenciamento).

As mensagens de erro têm três níveis de severidade:

-   Erros fatais
-   Warnings
-   Mensagens de informações

O nível de severidade é mostrado entre colchetes angulares no início da descrição da mensagem de erro. Veja a seguir uma lista de erros do provedor SNMP WMI:

-   [Erros 281 a 290](errors-281-through-290.md)

-   [Erros 1011 a 1020](errors-1011-through-1020.md)
