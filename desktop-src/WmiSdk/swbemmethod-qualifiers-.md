# <a name="swbemmethodqualifiers_-property"></a>Propriedade SWbemMethod. Qualifiers \_

A propriedade **Qualifiers \_** do objeto [**SWbemMethod**](swbemmethod.md) retorna um objeto [**SWbemQualifierSet**](swbemqualifierset.md) que é a coleção de qualificadores para esse método. Esta propriedade é somente para leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
SWbemMethod.Qualifiers_ As Object
```
