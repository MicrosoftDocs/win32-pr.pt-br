# <a name="the-performance-monitoring-provider"></a>O provedor de monitoramento de desempenho

Os dados de bibliotecas de desempenho estão disponíveis como classes de contador de desempenho [WMI](/windows/desktop/CIMWin32Prov/performance-counter-classes) por meio do provedor [WMIPerfClass](wmiperfclass-provider.md) e do [provedor WMIPerfInst.](wmiperfinst-provider.md)
