# <a name="__systemclass-class"></a>\_\_Classe SystemClass

A classe de sistema abstrata **\_ \_ SystemClass** é a classe base da qual a maioria das classes de sistema deriva. Não é possível criar instâncias dessa classe.

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[abstract]
class __SystemClass
{
};
```

## <a name="members"></a>Membros

A classe **\_ \_ SystemClass** não define nenhum membro.
