# <a name="m-wmi"></a>M (WMI)

[A](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) G [H](gloss-h.md) [i](gloss-i.md) J [K](gloss-k.md) [L](gloss-l.md) M [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V [W](gloss-w.md) X Y Z

<dl> <dt>

<span id="wmi.gloss_managed_object"></span><span id="WMI.GLOSS_MANAGED_OBJECT"></span>**objeto gerenciado**
</dt> <dd>

Um componente de hardware ou de sistema de software que é representado como uma instância de uma classe WMI.

</dd> <dt>

<span id="wmi.gloss_managed_object_format"></span><span id="WMI.GLOSS_MANAGED_OBJECT_FORMAT"></span>**Formato MOF (MOF)**
</dt> <dd>

Uma linguagem baseada na IDL (linguagem de definição de interface) que descreve as informações de gerenciamento.

</dd> <dt>

<span id="wmi.gloss_management_information_base"></span><span id="WMI.GLOSS_MANAGEMENT_INFORMATION_BASE"></span>**MIB (base de informações de gerenciamento)**
</dt> <dd>

Um banco de dados que armazena uma coleção de representações de objetos de rede.

</dd> <dt>

<span id="wmi.gloss_metamodel"></span><span id="WMI.GLOSS_METAMODEL"></span>**MetaModel**
</dt> <dd>

Um componente CIM que descreve as entidades e relações que representam *objetos gerenciados*. Por exemplo, classes, instâncias e associações são incluídas no metamodelo.

</dd> <dt>

<span id="wmi.gloss_metaschema"></span><span id="WMI.GLOSS_METASCHEMA"></span>**Metaesquema**
</dt> <dd>

O [*esquema*](gloss-s.md) do *metamodelo*.

</dd> <dt>

<span id="wmi.gloss_method_provider"></span><span id="WMI.GLOSS_METHOD_PROVIDER"></span>**provedor de método**
</dt> <dd>

Um provedor que permite o acesso de WMI aos métodos de uma classe. Consulte também [escrevendo um provedor de método](writing-a-method-provider.md).

</dd> <dt>

<span id="wmi.gloss_mib"></span><span id="WMI.GLOSS_MIB"></span>**MIB**
</dt> <dd>

Consulte *base de informações de gerenciamento (MIB)*.

</dd> <dt>

<span id="wmi.gloss_mof"></span><span id="WMI.GLOSS_MOF"></span>**MOF**
</dt> <dd>

Consulte *formato MOF (MOF)*.

</dd> <dt>

<span id="wmi.gloss_mof_file"></span><span id="WMI.GLOSS_MOF_FILE"></span>**Arquivo MOF**
</dt> <dd>

Um arquivo de texto que contém definições de classes e instâncias especificadas usando a linguagem *formato MOF (MOF)* .

</dd> </dl>
