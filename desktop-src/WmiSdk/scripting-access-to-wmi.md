# <a name="scripting-access-to-wmi"></a>Script de acesso ao WMI

Os scripts podem acessar todas as classes WMI para objetos de hardware e software. Os scripts do WSH (Windows Script Host) podem executar operações em objetos do sistema de arquivos, manipular impressoras de rede ou alterar variáveis de ambiente. Você pode encontrar uma variedade de tarefas de administrador e uma breve orientação sobre como realizá-las no WMI em [tarefas do WMI para scripts e aplicativos](wmi-tasks-for-scripts-and-applications.md). Para obter mais informações e exemplos, consulte o [repositório de scripts](https://www.microsoft.com/technet/scriptcenter/scripts/default.mspx)do TechNet ScriptCenter.

Se você for novo no script ou em scripts específicos do WMI, consulte a seção do TechNet ScriptCenter [introdução](https://www.microsoft.com/technet/scriptcenter/hubs/start.mspx).

Com a [API de script para WMI](scripting-api-for-wmi.md), você pode desenvolver scripts rápidos e simples ou aplicativos complexos. O script oferece o mesmo recurso de obter informações ou configurar a maioria dos objetos em uma empresa como você teria por meio de um aplicativo C++ ou C#. Para obter mais informações, consulte [scripts no WMI](/windows/desktop/WmiSdk/creating-a-wmi-script).

Não é possível gravar um [*provedor WMI*](gloss-p.md) no script. Para obter mais informações, consulte [fornecendo dados ao WMI](providing-data-to-wmi.md).

Os scripts do WMI podem ser escritos em qualquer linguagem de script que possa interagir com objetos ActiveX.

O Windows PowerShell fornece um ambiente simples para a administração e script do WMI. Para obter mais informações sobre o PowerShell, consulte [introdução com o Windows PowerShell](/powershell/scripting/getting-started/getting-started-with-windows-powershell?view=powershell-7&preserve-view=true).

Os scripts de ADSI (Active Directory Service Interfaces) fornecem acesso a objetos Active Directory Domain Services (AD DS). Os scripts do WSH e do ADSI acessam objetos e permitem que os procedimentos não estejam disponíveis por meio de arquivos em lotes.
