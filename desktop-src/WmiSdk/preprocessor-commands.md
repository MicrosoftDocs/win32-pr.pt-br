# <a name="preprocessor-commands"></a>Comandos de pré-processador

Um comando de pré-processador é uma linha de código colocada dentro de um arquivo MOF que controla a maneira como o compilador MOF compila o código. Há dois comandos de pré-processador usados na compilação de arquivos MOF:

-   [\#incluir](-include.md)
-   [\#pragma](-pragma.md)
