# <a name="mapping-active-directory-to-wmi"></a>Mapeando o Active Directory para o WMI

O Provedor de Serviços de Diretório espelha classes e instâncias do Active Directory no namespace LDAP (WMI Lightweight Directory Access Protocol). Devido às diferenças fundamentais nos dois ambientes, você não pode simplesmente renomear as classes e instâncias do Active Directory e esperar acessar corretamente essas classes e instâncias no WMI. Em vez disso, o provedor de Serviços de Diretório segue as regras de nomen entre as classes e as instâncias do Active Directory.

Os tópicos a seguir fornecem informações sobre como mapear classes e instâncias do Active Directory:

-   [Mapeando classes do Active Directory](mapping-active-directory-classes.md)
-   [Mapeando instâncias do Active Directory](mapping-active-directory-instances.md)

> [!Note]  
> Para obter mais informações sobre o suporte e a instalação desse componente em um sistema operacional específico, consulte Disponibilidade do sistema operacional [de componentes WMI](operating-system-availability-of-wmi-components.md).
