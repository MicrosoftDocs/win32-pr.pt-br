# <a name="trap-type-macro"></a>TRAP-TYPE Macro

Para uniformidade, cada instância da macro TRAP-TYPE é mapeada primeiro para uma instância da macro [NOTIFICATION-TYPE](notification-type-macro.md) usando o mapeamento definido no [RFC 1908](https://www.ietf.org/rfc/rfc1908.txt).

> [!Note]  
> Para obter mais informações sobre como instalar o provedor, consulte [Configurando o ambiente WMI SNMP](setting-up-the-wmi-snmp-environment.md).
