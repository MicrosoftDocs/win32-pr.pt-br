# <a name="trap-type-macro"></a>Macro do tipo INTERCEPTAção

Para a uniformidade, cada instância da macro do tipo TRAP é mapeada primeiro para uma instância da macro do [tipo notificação](notification-type-macro.md) usando o mapeamento definido em [RFC 1908](https://www.ietf.org/rfc/rfc1908.txt).

> [!Note]  
> Para obter mais informações sobre como instalar o provedor, consulte [Configurando o ambiente SNMP WMI](setting-up-the-wmi-snmp-environment.md).
