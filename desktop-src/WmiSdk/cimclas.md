# <a name="cim-classes-wmi"></a>Classes CIM (WMI)

As classes do DMTF (Distributed Management Task Force) modelo CIM (CIM) são as classes pai nas quais as classes WMI são criadas. A tabela a seguir lista as categorias de classes CIM.

> [!IMPORTANT]
> As classes DMTF (Distributed Management Task Force) CIM (modelo CIM) são as classes pai nas quais as classes WMI são criadas. Atualmente, o WMI dá suporte apenas aos [esquemas de versão do CIM 2. x](https://dmtf.org/standards/cim/schemas). Para obter mais informações sobre o CIM, consulte [modelo CIM](common-information-model.md).
