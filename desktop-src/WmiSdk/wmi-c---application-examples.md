# <a name="wmi-c-application-examples"></a>Exemplos de aplicativos WMI C++

Os exemplos de aplicativo WMI nesta seção são escritos em C++. eles demonstram uma variedade de tarefas que podem ser concluídas usando componentes WMI e oferecem uma alternativa em relação ao uso de scripts Visual Basic. Cada aplicativo é separado em uma série de etapas de forma semelhante, para que as seções de código de exemplos diferentes possam ser facilmente combinadas para formar aplicativos personalizados.

A tabela a seguir lista os exemplos de C++ nesta seção.



| Exemplo                                                                                                                                  | Descrição                                                                                                                                                |
|------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Exemplo: obtendo dados WMI do computador local](example--getting-wmi-data-from-the-local-computer.md)                               | Este exemplo conecta-se ao namespace WMI no computador local e obtém dados de uma consulta no computador local. Os dados são coletados semisynchronously. |
| [Exemplo: obtendo dados do WMI do computador local de forma assíncrona](example--getting-wmi-data-from-the-local-computer-asynchronously.md) | Este exemplo conecta-se ao namespace WMI no computador local e obtém dados de uma consulta no computador local. Os dados são coletados de forma assíncrona.    |
| [Exemplo: obtendo dados WMI de um computador remoto](example--getting-wmi-data-from-a-remote-computer.md)                                 | Este exemplo conecta-se ao namespace do WMI em um computador remoto e obtém dados de uma consulta no computador remoto. Os dados são coletados semisynchronously. |
| [Exemplo: chamando um método de provedor](example--calling-a-provider-method.md)                                                             | Este exemplo conecta-se ao namespace WMI no computador local e chama um método no WMI. O método é executado de forma síncrona.                          |
| [Exemplo: recebendo notificações de eventos por meio do WMI](example--receiving-event-notifications-through-wmi-.md)                            | Este exemplo conecta-se ao namespace WMI no computador local e recebe um evento do computador local. O evento é recebido semisynchronously.   |
