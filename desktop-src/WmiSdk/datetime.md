# <a name="datetime"></a>DATETIME

O tipo de dados DATETIME é uma cadeia de caracteres VT BSTR de comprimento fixo \_ que representa uma data e hora e um intervalo específicos. Para obter mais informações sobre DATETIME como uma data, consulte [formato de data e hora](date-and-time-format.md). Para obter mais informações sobre DATETIME como um intervalo, consulte [formato do intervalo](interval-format.md).
