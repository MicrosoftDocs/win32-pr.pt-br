# <a name="datetime"></a>DATETIME

O tipo de dados DATETIME é uma cadeia de caracteres BSTR VT de comprimento fixo que representa uma data e hora ou \_ intervalo específicos. Para obter mais informações sobre DATETIME como uma data, consulte [Formato de data e hora.](date-and-time-format.md) Para obter mais informações sobre DATETIME como um intervalo, consulte [Formato de intervalo](interval-format.md).
