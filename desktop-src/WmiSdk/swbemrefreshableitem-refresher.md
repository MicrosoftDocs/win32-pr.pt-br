# <a name="swbemrefreshableitemrefresher-property"></a>SWbemRefreshableItem. Propriedade do atualizador

A propriedade **SWbemRefreshableItem. upnovator** representa o objeto pai [**SWbemRefresher**](swbemrefresher.md) que contém o objeto [**SWbemRefreshableItem**](swbemrefreshableitem.md) .

Para obter uma explicação dessa sintaxe, consulte [convenções de documento para a API de script](document-conventions-for-the-scripting-api.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
SWbemRefreshableItem.Refresher As Object
```
