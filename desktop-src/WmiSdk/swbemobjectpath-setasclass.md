# <a name="swbemobjectpathsetasclass-property"></a>Propriedade SWbemObjectPath. SetAsClass

O método **SetAsClass** do objeto [**SWbemObjectPath**](swbemobjectpath.md) força o caminho para endereçar uma classe WMI.

Essa propriedade é somente gravação.

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.SetAsClass
```



## <a name="property-value"></a>Valor da propriedade

## <a name="error-codes"></a>Códigos do Erro

Depois de obter ou definir a propriedade **SetAsClass** , o objeto **Err** poderá conter o código de erro abaixo.

<dl> <dt>

**wbemErrFailed** -2147749889 (0x80041001)
</dt> <dd>

Erro não especificado.

</dd> </dl>
