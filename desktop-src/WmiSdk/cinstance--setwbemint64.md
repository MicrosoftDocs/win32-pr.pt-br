# <a name="cinstancesetwbemint64-methods"></a>Métodos CInstance::SetWBEMINT64

\[A [**classe CInstance**](/windows/win32/api/instance/nl-instance-cinstance) faz parte do WMI Provider Framework, que agora é considerado no estado final e nenhum desenvolvimento, aprimoramentos ou atualizações adicionais estará disponível para problemas não relacionados à segurança que afetam essas bibliotecas. As [APIs de MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) devem ser usadas para todo o novo desenvolvimento.\]

O [**método CInstance::GetWBEMINT64**](cinstance-getwbemint64.md) define uma propriedade de inteiro de 64 bits.

| Método                                                                                               | Descrição                                |

| [**SetWBEMINT64(LPCWSTR, const LONGLONG&)**](/windows/win32/api/instance/nf-instance-cinstance-setwbemint64(lpcwstr_constlonglong))    | Define uma propriedade de inteiro de 64 bits.<br/> |
| [**SetWBEMINT64(LPCWSTR, const WBEMINT64&)**](/windows/win32/api/instance/nf-instance-cinstance-setwbemint64(lpcwstr_constulonglong)) | Define uma propriedade de inteiro de 64 bits.<br/> |
| [**SetWBEMINT64(LPCWSTR, const ULONG&)**](/windows/win32/api/instance/nf-instance-cinstance-setwbemint64(lpcwstr_constulonglong))  | Define uma propriedade de inteiro de 64 bits.<br/> |
