# <a name="policy-provider-classes"></a>Classes de provedor de política

O provedor de política do WMI fornece extensões para a diretiva de grupo por meio de classes, permitindo refinamentos na aplicação da política. A política é definida usando o linguagem WQL (WQL) e reside no Active Directory para facilitar a implantação.

A tabela a seguir lista as classes para o provedor de política do WMI.



| Classe                                               | Descrição                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------|
| [**Provedores de MSFT \_**](/previous-versions/windows/desktop/wmisystemprov/msft-providers) | Expõe a configuração relacionada às instâncias do provedor.             |
| [**\_Regra MSFT**](/previous-versions/windows/desktop/policmanprov/msft-rule)                | Define uma única regra dentro de um escopo de gerenciamento (SOM).         |
| [**\_SOMFILTER MSFT**](/previous-versions/windows/desktop/policmanprov/msft-somfilter)      | Fornece uma lista de regras que são avaliadas em um computador de destino. |
