# <a name="__securityrelatedclass-class"></a>\_\_Classe SecurityRelatedClass

A classe de sistema abstrata **\_ \_ SecurityRelatedClass** serve como uma classe pai para todos os tipos de classes de segurança. Nenhuma instância dessa classe já foi criada e todas as classes de segurança devem derivar dessa classe base.

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[abstract]
class __SecurityRelatedClass
{
};
```

## <a name="members"></a>Membros

A classe **\_ \_ SecurityRelatedClass** não define nenhum membro.
