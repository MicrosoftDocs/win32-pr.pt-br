# <a name="operating-system-availability-of-wmi-components"></a>Disponibilidade do sistema operacional de componentes WMI

o WMI é instalado por padrão em todas as plataformas de área de trabalho e servidor Windows. No entanto, alguns provedores WMI podem ou não ser instalados, dependendo da versão e da configuração do sistema operacional. por exemplo, o provedor SNMP não está habilitado por padrão e o provedor de Windows Installer (provedor de MSI) não é instalado por padrão em sistemas operacionais de 64 bits. Para obter mais informações sobre a disponibilidade de um provedor específico, consulte [provedores WMI](/windows/desktop/SrvNodes/wmi-mi-omi-providers).
