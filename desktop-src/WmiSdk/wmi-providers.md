# <a name="wmi-providers"></a>Provedores de WMI

A tabela a seguir lista os provedores WMI do sistema operacional. Outros provedores e suas classes podem existir em seu computador instalado por produtos da Microsoft ou por software de terceiros. Para obter mais informações sobre como escrever seu próprio provedor, consulte [usando o WMI](using-wmi.md).

Os provedores WMI enviam mensagens de erro e de aviso ao log de eventos. Para obter mais informações, consulte [**mensagens do provedor WMI**](wmi-event-messages.md).
