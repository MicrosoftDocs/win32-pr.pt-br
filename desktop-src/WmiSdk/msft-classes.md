# <a name="msft-classes"></a>Classes MSFT

As classes MSFT são específicas à implementação do [*modelo CIM*](gloss-c.md) (CIM) do Windows.

Os objetos podem ser divididos em várias categorias principais:

-   [Classes de provedor de política](policy-provider-classes.md)
-   [Classes de solução de problemas do WMI](wmi-troubleshooting-classes.md)

As classes MSFT a seguir são definidas no System. MOF, mas não são implementadas pelo WMI.

-   \_WMICOREEVENT MSFT
-   \_WMICORELOGOFFEVENT MSFT
-   \_WMICORELOGONEVENT MSFT
-   \_WMICOREOBJECT MSFT
-   \_WMICORESTATUS MSFT
-   \_WMICOREUSER MSFT
-   \_WMITASKFAILURE MSFT
