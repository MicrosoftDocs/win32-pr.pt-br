# <a name="swbemprivilegedisplayname-property"></a>Propriedade SWbemPrivilege. DisplayName

A propriedade **DisplayName** de um objeto [**SWbemPrivilege**](swbemprivilege.md) é uma cadeia de caracteres que exibe uma descrição de um objeto **SWbemPrivilege** . Por exemplo, o privilégio que controla se um usuário pode executar o método de desligamento de um objeto tem a cadeia de caracteres "desligar o sistema" na propriedade **SWbemPrivilege. DisplayName** . Esta propriedade é somente para leitura.

Para obter uma explicação dessa sintaxe, consulte [convenções de documento para a API de script](document-conventions-for-the-scripting-api.md).

Esta propriedade é somente para leitura.

```VB
SWbemPrivilege.DisplayName As
```
