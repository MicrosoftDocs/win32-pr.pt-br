# <a name="date-and-time-format"></a>Formato de data e hora

O WMI usa os formatos de data e hora definidos pela especificação de CIM ([DMTF.org](https://www.dmtf.org/)) modelo CIM (Distributed Management Task Force).

O formato de **\_ data e hora CIM** é implementado no Microsoft [formato MOF (MOF)](managed-object-format--mof-.md) pelo tipo de dados MOF [DateTime](datetime.md) . Os formatos de data e hora também podem expressar um intervalo de tempo.

Para obter mais informações sobre formatos WMI, consulte [CIM \_ DateTime](cim-datetime.md) e [formato de intervalo](interval-format.md).

Para obter mais informações sobre como converter de e para o formato de data e hora do WMI, consulte [**SWbemDateTime**](swbemdatetime.md) e o artigo sobre o TechNet ScriptCenter [sobre o tempo (Ah e sobre datas também)](https://www.microsoft.com/technet/technetmag/issues/2006/07/ScriptingGuy/default.aspx).
