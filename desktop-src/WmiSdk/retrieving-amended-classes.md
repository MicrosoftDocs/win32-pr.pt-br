# <a name="retrieving-amended-classes"></a>Recuperando classes corrigidas

Você pode recuperar e modificar uma classe corrigida especificando uma localidade e definindo um sinalizador que indica que você deseja informações localizadas. O procedimento usado para recuperar definições de classe localizadas depende de como você se conecta ao WMI. Para obter mais informações, consulte [recuperando classes corrigidas usando a API de script para WMI](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) e [recuperando classes corrigidas usando a API com para o WMI](retrieving-amended-classes-using-the-com-api-for-wmi.md).
