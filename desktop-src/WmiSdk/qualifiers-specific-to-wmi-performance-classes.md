# <a name="qualifiers-specific-to-wmi-performance-classes"></a>Qualificadores específicos para classes de desempenho WMI

As classes de contador de desempenho brutos, derivadas do [**Win32 \_ PerfRawData**](/windows/desktop/CIMWin32Prov/win32-perfrawdata), contêm valores não calculados que servem como entrada para classes do [**Win32 \_ PerfFormattedData**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) , bem como cálculos do monitor do sistema.

Os qualificadores de contador de desempenho especificam informações sobre o objeto de desempenho e os contadores para esse objeto. Os objetos de desempenho são mapeados para [classes de contador de desempenho](/windows/desktop/CIMWin32Prov/performance-counter-classes) WMI e contadores de desempenho são representados pelas propriedades da classe WMI.

-   [Qualificadores de classe para classes de contador de desempenho](class-qualifiers-for-performance-counter-classes.md)
-   [Qualificadores de propriedade para classes de contador de desempenho formatadas](property-qualifiers-for-performance-counter-classes.md)
