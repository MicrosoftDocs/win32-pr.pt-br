# <a name="retrieving-wmi-class-or-instance-data"></a>Recuperando dados de instância ou classe WMI

Os principais objetos de dados no WMI são classes e instâncias. A tabela a seguir lista as diferentes técnicas disponíveis para recuperar dados de uma instância ou classe WMI.



| Tópico                                                                   | Descrição                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Recuperando uma classe WMI](retrieving-a-class.md)                        | Recuperando informações sobre os parâmetros gerais de um objeto.                    |
| [Recuperando uma instância do WMI](retrieving-an-instance.md)                 | Recuperando informações que descrevem um único objeto individual em sua empresa. |
| [Recuperando parte de uma instância do WMI](retrieving-part-of-an-instance.md) | Recuperando uma ou mais partes de uma instância, mas não a instância inteira.         |
| [Acessando um qualificador WMI](accessing-a-qualifier.md)                  | Acessando mais informações contidas em um qualificador.                              |



 

Para obter mais informações, consulte [manipulando informações de classe e instância](manipulating-class-and-instance-information.md).
