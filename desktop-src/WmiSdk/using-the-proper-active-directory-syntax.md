# <a name="using-the-proper-active-directory-syntax"></a>Usando a sintaxe de Active Directory apropriada

Forneça o namespace LDAP (Lightweight Directory Access Protocol) e os caminhos da interface de serviços de Active Directory (ADSI) ao chamar o [provedor de Active Directory](/previous-versions/windows/desktop/dsprov/active-directory-provider) por meio do WMI.

Os tópicos a seguir descrevem o namespace LDAP e o caminho ADSI:

-   [Descrevendo o namespace LDAP](describing-the-ldap-namespace.md)

    Um caminho de namespace descreve o local de um namespace remoto. Use o caminho do namespace para se conectar remotamente a um namespace Active Directory.

-   [Descrevendo o caminho ADSI](describing-the-adsi-path.md)

    O caminho ADSI descreve o local de um objeto usando a API ADSI. Use o caminho ADSI nos parâmetros que o provedor passa para Active Directory.

> [!Note]  
> Para obter mais informações sobre o suporte e a instalação desse componente em um sistema operacional específico, consulte [disponibilidade do sistema operacional de componentes WMI](operating-system-availability-of-wmi-components.md).
