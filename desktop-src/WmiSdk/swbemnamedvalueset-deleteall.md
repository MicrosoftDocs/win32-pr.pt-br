# <a name="swbemnamedvaluesetdeleteall-method"></a>Método SWbemNamedValueSet. DeleteAll

O método **DeleteAll** do objeto [**SWbemNamedValueSet**](swbemnamedvalueset.md) remove todos os valores nomeados da coleção, esvaziando-os.

## <a name="syntax"></a>Sintaxe


```VB
SWbemNamedValueSet.DeleteAll()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.

## <a name="error-codes"></a>Códigos do Erro

Após a conclusão do método **DeleteAll** , o objeto **Err** pode conter o código de erro abaixo.

<dl> <dt>

**wbemErrFailed** -2147749889 (0x80041001)
</dt> <dd>

Erro não especificado.

</dd> </dl>
