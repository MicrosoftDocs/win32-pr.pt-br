# <a name="w-wmi"></a>W (WMI)

[A](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) G [H](gloss-h.md) [i](gloss-i.md) J [K](gloss-k.md) [L](gloss-l.md) [M](gloss-m.md) [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V W X Y Z

<dl> <dt>

<span id="wmi.gloss_win32_schema"></span><span id="WMI.GLOSS_WIN32_SCHEMA"></span>**Esquema Win32**
</dt> <dd>

uma extensão da Microsoft para o [*esquema CIM*](gloss-c.md) que contém definições de classes e instâncias para representar [*objetos gerenciados*](gloss-m.md) que existem em um ambiente de Windows típico.

</dd> <dt>

<span id="wmi.gloss_wmi_class"></span><span id="WMI.GLOSS_WMI_CLASS"></span>**Classe WMI**
</dt> <dd>

Um modelo para um tipo de [*objeto gerenciado*](gloss-m.md). As classes WMI definem as unidades básicas de gerenciamento.

</dd> <dt>

<span id="wmi.gloss_wmi_class_provider"></span><span id="WMI.GLOSS_WMI_CLASS_PROVIDER"></span>**Provedor de classes WMI**
</dt> <dd>

Um servidor COM que fornece definições de classe. Os provedores de classe implementam métodos na interface [**IWbemServices**](/windows/desktop/api/WbemCli/nn-wbemcli-iwbemservices) para dar suporte à recuperação de dados, modificação, exclusão, enumeração e processamento de consulta.

</dd> <dt>

<span id="wmi.gloss_wmi_control"></span><span id="WMI.GLOSS_WMI_CONTROL"></span>**Controle WMI**
</dt> <dd>

Um snap-in do MMC (console de gerenciamento Microsoft) que permite que um usuário defina as configurações do WMI em um computador local ou remoto.

</dd> <dt>

<span id="wmi.gloss_wmi_for_windows_driver_model"></span><span id="WMI.GLOSS_WMI_FOR_WINDOWS_DRIVER_MODEL"></span>**WMI para Windows Driver Model (WDM)**
</dt> <dd>

Um componente de nível baixo que fornece acesso entre dispositivos WMI e de hardware.

</dd> <dt>

<span id="wmi.gloss_wmi_odbc_adapter"></span><span id="WMI.GLOSS_WMI_ODBC_ADAPTER"></span>**Adaptador ODBC do WMI**
</dt> <dd>

uma interface que permite que um aplicativo obtenha informações do gerenciamento de Windows usando *linguagem WQL*. Isso fornece uma API padrão que permite que aplicativos baseados em ODBC usem os dados no [*repositório CIM*](gloss-c.md) como se fosse um banco de dado.

</dd> <dt>

<span id="wmi.gloss_wmi_query_language"></span><span id="WMI.GLOSS_WMI_QUERY_LANGUAGE"></span>**Linguagem WQL (WQL)**
</dt> <dd>

um subconjunto da linguagem de consulta estruturada (SQL) com extensões para dar suporte à notificação de eventos do wmi e outros recursos específicos do wmi.

</dd> <dt>

<span id="wmi.gloss_wmi_method"></span><span id="WMI.GLOSS_WMI_METHOD"></span>**Método WMI**
</dt> <dd>

Um método implementado pela [API com](com-api-for-wmi.md) do WMI ou pela [API de script](scripting-api-for-wmi.md). Dois exemplos são o método COM [**IWbemServices:: ExecMethod**](/windows/desktop/api/WbemCli/nf-wbemcli-iwbemservices-execmethod) e o método de API de script semelhante [**SWbemServices.ExecMethod**](swbemservices-execmethod.md).

</dd> <dt>

<span id="wmi.gloss_wmi_repository"></span><span id="WMI.GLOSS_WMI_REPOSITORY"></span>**Repositório do WMI**
</dt> <dd>

Uma área de armazenamento central gerenciada pelo [*Gerenciador de objetos CIM*](gloss-c.md).

</dd> <dt>

<span id="wmi.gloss_wql"></span><span id="WMI.GLOSS_WQL"></span>**WQL**
</dt> <dd>

Consulte *linguagem WQL (WQL)*.

</dd> </dl>
