# <a name="swbemmethodsetcount-property"></a>Propriedade SWbemMethodSet. Count

A propriedade **Count** do objeto [**SWbemMethodSet**](swbemmethodset.md) define o número de itens na coleção. Esta propriedade é somente para leitura.

Para obter uma explicação dessa sintaxe, consulte [convenções de documento para a API de script](document-conventions-for-the-scripting-api.md).

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
SWbemMethodSet.Count As Integer
```
