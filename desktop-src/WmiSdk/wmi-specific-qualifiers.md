# <a name="wmi-specific-qualifiers"></a>Qualificadores de WMI-Specific

Além dos qualificadores CIM, o WMI fornece vários qualificadores adicionais para refinar ainda mais o provisionamento de instâncias e operações de classe.

Veja a seguir uma lista de qualificadores adicionais que o WMI fornece:

-   [Qualificadores WMI padrão](standard-wmi-qualifiers.md)
-   [Qualificadores específicos para o provedor SNMP](qualifiers-specific-to-the-snmp-provider.md)
-   [Qualificadores específicos para o provedor de exibição](qualifiers-specific-to-the-view-provider.md)
-   [Qualificadores específicos para o provedor WDM](qualifiers-specific-to-the-wdm-provider.md)
-   [Qualificadores específicos para classes de desempenho WMI](qualifiers-specific-to-wmi-performance-classes.md)

Além dos qualificadores, os tipos que descrevem o comportamento dos qualificadores podem ser aplicados aos vários qualificadores. Para obter mais informações, consulte [qualificadores de tipos](qualifier-flavors.md).
