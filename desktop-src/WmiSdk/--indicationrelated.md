# <a name="__indicationrelated-class"></a>\_\_Classe IndicationRelated

A classe de sistema abstrata **\_ \_ IndicationRelated** serve como uma classe pai para todas as classes relacionadas a eventos. A classe não contém propriedades.

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas. As propriedades são listadas em ordem alfabética, não em ordem MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[abstract]
class __IndicationRelated : __SystemClass
{
};
```

## <a name="members"></a>Membros

A classe **\_ \_ IndicationRelated** não define nenhum membro.

## <a name="remarks"></a>Comentários

A classe **\_ \_ IndicationRelated** é derivada de [**\_ \_ SystemClass**](--systemclass.md), que não tem propriedades.
