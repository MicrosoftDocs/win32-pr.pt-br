# <a name="cooker_range"></a>INTERVALO DO \_ COOKER

A fórmula de tipo de contador COOKER RANGE mostra a diferença entre os valores Min e Max para um conjunto de observações brutas de uma propriedade em uma \_ instância do [**Win32 \_ PerfRawData.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) [](cooker-min.md) [](cooker-max.md) Esse tipo de contador é definido somente no WMI e não está disponível para as tecnologias de monitoramento de desempenho, como Contadores de Desempenho [versão 6.0](/windows/desktop/PerfCtrs/performance-counters-portal).

Para obter mais informações sobre a fórmula de tipo de contador, consulte [Tipos de contador](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obter mais informações sobre scripts e provedores de alto desempenho, consulte Tipos de contador de desempenho [WMI](wmi-performance-counter-types.md).
