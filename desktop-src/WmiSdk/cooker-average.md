# <a name="cooker_average"></a>média de COOKER \_

A fórmula do tipo de contador de média de COOKER \_ fornece o ponto médio estatístico dos dados para um contador. A fórmula para esse tipo de contador soma as observações repetidas de uma propriedade em uma instância [**\_ PerfRawData do Win32**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) e divide a soma pelo número de observações. Esse tipo de contador é definido somente dentro do WMI e não está disponível para as tecnologias de monitoramento de desempenho, como [contadores de desempenho versão 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Para obter mais informações sobre a fórmula de tipo de contador, consulte [tipos de contador](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obter mais informações sobre provedores de alto desempenho e scripts do WMI, consulte [tipos de contador de desempenho WMI](wmi-performance-counter-types.md).
