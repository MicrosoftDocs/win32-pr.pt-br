<span id="wmi.gloss_namespace"></span><span id="WMI.GLOSS_NAMESPACE"></span>**namespace**
</dt> <dd>

Uma categorização de classes e instâncias usadas para controlar seu escopo e visibilidade. Os namespaces não são locais físicos. Eles são mais como bancos de dados lógicos que contêm classes e instâncias específicas. Os namespaces são representados pela classe de sistema de [**\_ \_ namespace**](--namespace.md) ou por uma classe derivada dele.
