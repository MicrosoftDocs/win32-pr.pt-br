# <a name="creating-an-instance"></a>Criando uma instância

Uma instância é um caso de uma classe que descreve um objeto específico em sua empresa. Por exemplo, se você tiver duas unidades de CD-ROM em uma plataforma de área de trabalho, poderá usar duas instâncias da classe [**Win32 \_ CDROMDrive**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) para descrever ambas. Como uma instância representa uma propriedade real de sua empresa, é mais provável que você crie mais instâncias do que as classes. Cada classe da qual uma instância é criada tem uma ou mais propriedades de chave. Você não pode criar uma instância com mais de 256 chaves.

Para obter mais informações sobre como criar instâncias, consulte os seguintes tópicos:

-   [Criando e declarando uma instância usando C++](creating-and-declaring-an-instance-using-c-.md)
-   [Criando uma instância usando o MOF](creating-an-instance-using-mof.md)
-   [Criando um objeto usando o VBScript](creating-an-object-using-vbscript.md)
