# <a name="wmi-events"></a>Eventos WMI

Os eventos podem ser relatados por WMI ou provedores. O WMI usa o ETW [(Rastreamento](/windows/desktop/ETW/event-tracing-portal) de Eventos). Para obter mais informações, consulte [Rastreamento de atividade WMI](tracing-wmi-activity.md).

> A documentação a seguir foi projetada para desenvolvedores e administradores de IT. Se você estiver tentando resolver uma mensagem de erro WMI em seu sistema base, consulte o [Suporte da Microsoft](https://support.microsoft.com/) site.
