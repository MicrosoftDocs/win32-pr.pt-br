# <a name="swbempropertyqualifiers_-property"></a>Propriedade SWbemProperty. Qualifiers \_

A propriedade **Qualifiers \_** do objeto [**SWbemProperty**](swbemproperty.md) retorna um objeto [**SWbemQualifierSet**](swbemqualifierset.md) que é uma coleção de qualificadores para essa propriedade WMI. Esta propriedade é somente para leitura.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
SWbemProperty.Qualifiers_ As Object
```
