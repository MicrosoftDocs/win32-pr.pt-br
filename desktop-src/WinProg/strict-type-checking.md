# <a name="strict-type-checking"></a>Verificação de tipo estrito

O arquivo de cabeçalho Windows. h contém definições, macros e estruturas para ajudá-lo a escrever o código-fonte portátil entre as versões do Windows. Alguns desses recursos são habilitados quando você define o símbolo estrito ao compilar seu aplicativo. Os tópicos a seguir explicam as vantagens de recursos ESTRITOs e como usá-los afeta a escrita de código.

-   [Habilitando STRICT](enabling-strict.md)
-   [Desabilitando estrito](disabling-strict.md)
-   [Conformidade estrita](strict-compliance.md)
