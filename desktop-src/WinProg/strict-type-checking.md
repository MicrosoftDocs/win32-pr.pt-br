# <a name="strict-type-checking"></a>Verificação de tipo STRICT

O arquivo de Windows.h contém definições, macros e estruturas para ajudá-lo a escrever código-fonte portátil entre versões do Windows. Alguns desses recursos são habilitados quando você define o símbolo STRICT ao criar seu aplicativo. Os tópicos a seguir explicam as vantagens dos recursos STRICT e como usá-los afeta a escrita de código.

-   [Habilitando STRICT](enabling-strict.md)
-   [Desabilitando STRICT](disabling-strict.md)
-   [Conformidade ESTRITA](strict-compliance.md)
