# <a name="device-drivers"></a>Drivers de dispositivo

Esta documentação é para aplicativos de modo de usuário que desejam usar o ETW para rastreamento de eventos. Esta documentação não é para drivers de dispositivo que são executados no modo kernel. Para aqueles que criam drivers de dispositivo e aplicativos em modo kernel que desejam usar o ETW como um depurador de baixo impacto para solucionar problemas relacionados a intervalos, consulte [rastreamento de eventos para Windows](/windows-hardware/drivers/devtest/event-tracing-for-windows--etw-), [rastreamento de software do WPP](/windows-hardware/drivers/devtest/wpp-software-tracing)e rastreamento de [eventos WMI](/windows-hardware/drivers/kernel/wmi-event-tracing) no Windows Driver Kit (WDK) para obter informações sobre como instrumentar seu driver.
