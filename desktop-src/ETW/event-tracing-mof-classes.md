# <a name="event-tracing-mof-classes"></a>Classes MOF de Rastreamento de Eventos

As classes são definidas no \\ \\ namespace wmi raiz. O **qualificador EventVersion** para os eventos de kernel associa a classe a um sistema operacional específico. Por exemplo, a versão zero está associada ao Windows 2000, versão um com o Windows XP e o Windows Server 2003 e a versão dois com Windows Vista.

As classes MOF a seguir definem as classes de evento ETW.

-   [**Evento \_ Perdido**](lost-event.md)
-   [**SystemTrace do MSNT \_**](msnt-systemtrace.md)
-   [**RT \_ LostEvent**](rt-lostevent.md)
