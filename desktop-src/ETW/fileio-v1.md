# <a name="fileio_v1-class"></a>\_Classe FileIo v1

Essa classe é a classe pai para eventos de e/s de arquivo.

A sintaxe a seguir é simplificada do código MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[Guid("{90cbdc39-4a3e-11d1-84f4-0000f80464e3}"), EventVersion(1)]
class FileIo_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Membros

A classe **FileIo \_ v1** não define nenhum membro.
