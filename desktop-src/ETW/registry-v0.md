# <a name="registry_v0-class"></a>\_Classe V0 do registro

Essa classe é a classe pai para eventos de registro.

``` syntax
[Guid("{ae53722e-c863-11d2-8659-00c04fa321a1}"), EventVersion(0)]
class Registry_V0 : MSNT_SystemTrace
{
};
```
