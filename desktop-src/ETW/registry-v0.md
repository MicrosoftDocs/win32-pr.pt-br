# <a name="registry_v0-class"></a>Classe \_ Registry V0

Essa classe é a classe pai para eventos do Registro.

## <a name="syntax"></a>Sintaxe

``` syntax
[Guid("{ae53722e-c863-11d2-8659-00c04fa321a1}"), EventVersion(0)]

{
```
