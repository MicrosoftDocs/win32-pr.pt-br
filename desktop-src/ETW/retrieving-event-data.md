# <a name="retrieving-event-data"></a>Recuperando dados de evento

Para consumir dados específicos do evento, o consumidor deve saber o formato dos dados do evento. Isso não será um problema se você estiver consumindo eventos de seu próprio provedor porque sabe o formato dos dados. No entanto, para consumir um evento de qualquer provedor, o provedor deve publicar o layout dos dados do evento. Para ter uma visão geral de como os metadados de evento operam, bem como publicar os metadados do evento para que outras pessoas possam usá-los, consulte Visão geral dos metadados [de evento](event-metadata-overview.md).

Para consumir eventos no Windows Vista que foram publicados usando um manifesto, MOF ou arquivos TMF, bem como eventos TraceLogging, consulte Recuperando dados de evento usando [TDH](retrieving-event-data-using-tdh.md).

Para consumir eventos anteriores ao Windows Vista que foram publicados usando MOF, consulte Recuperando dados [de evento usando MOF](retrieving-event-data-using-mof.md).
