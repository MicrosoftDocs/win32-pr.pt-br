# <a name="retrieving-event-data"></a>Recuperando dados de evento

Para consumir dados específicos do evento, o consumidor deve saber o formato dos dados do evento. Isso não é um problema se você estiver consumindo eventos de seu próprio provedor, pois sabe o formato dos dados. No entanto, para consumir um evento de qualquer provedor, o provedor deve publicar o layout dos dados do evento. Para obter uma visão geral de como os metadados de eventos operam e como publicar seus metadados de eventos para que outros possam usá-los, consulte [visão geral de metadados de evento](event-metadata-overview.md).

Para consumir eventos no Windows Vista que foram publicados usando arquivos manifest, MOF ou TMF, bem como eventos TraceLogging, consulte [recuperando dados de evento usando TDH](retrieving-event-data-using-tdh.md).

Para consumir eventos anteriores ao Windows Vista que foram publicados usando o MOF, consulte [recuperando dados de evento usando o MOF](retrieving-event-data-using-mof.md).
