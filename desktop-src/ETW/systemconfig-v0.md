# <a name="systemconfig_v0-class"></a>\_Classe SystemConfig V0

Essa classe é a classe pai para eventos de configuração de hardware.

A sintaxe a seguir é simplificada do código MOF.

## <a name="syntax"></a>Sintaxe

``` syntax
[Guid("{01853a65-418f-4f36-aefc-dc0f1d2fd235}"), EventVersion(0)]
class SystemConfig_V0 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Membros

A classe **SystemConfig \_ V0** não define nenhum membro.

## <a name="remarks"></a>Comentários

Para eventos de configuração de hardware no Windows XP, consulte a classe [**HWConfig**](hwconfig.md) .
