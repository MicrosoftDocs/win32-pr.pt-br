# <a name="event-tracing-samples"></a>Exemplos de rastreamento de eventos

O SDK (Software Development Kit) da plataforma contém exemplos completos de rastreamento de eventos. Esses exemplos estão localizados no diretório Samples \\ WinBase \\ EventTrace. A raiz desse caminho é o diretório de instalação base do PSDK. Você pode baixar o PSDK do [https://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5](https://msdn.microsoft.com/windowsserver/bb980924.aspx) .

A tabela a seguir descreve os exemplos contidos no diretório Samples \\ WinBase \\ EventTrace.



| Amostra   | Descrição                                                                                                                            |
|----------|----------------------------------------------------------------------------------------------------------------------------------------|
| TraceDmp | Um consumidor de rastreamento de eventos. Ele decodifica os dados de evento usando as informações de formato obtidas do WMI e gera os dados em um arquivo de .csv. |
| TraceDp  | Usa a API do provedor de rastreamento de eventos para fornecer dados de rastreamento de eventos para o agente de log ou um consumidor.                                             |
| TraceLog | Usa a API do controlador de rastreamento de eventos para gerenciar sessões de registro em log.                                                                        |



 

Para obter descrições detalhadas, consulte o arquivo Readme.txt incluído com estes exemplos.
