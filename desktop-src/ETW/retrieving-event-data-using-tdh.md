# <a name="retrieving-event-data-using-tdh"></a>Recuperando dados de evento usando TDH

Para consumir dados específicos do evento, o consumidor deve saber o formato dos dados do evento. Se o provedor usava arquivos de manifesto, MOF ou TMF para publicar o formato dos dados do evento, você poderá usar as funções do auxiliar de dados de rastreamento (TDH) para analisar os dados do evento. as funções TDH estão disponíveis a partir do Windows Vista.

Os tópicos a seguir mostram como usar o TDH para recuperar dados de eventos e metadados.

-   [Usando TdhFormatProperty para consumir dados de evento](using-tdhformatproperty-to-consume-event-data.md)
-   [Usando TdhGetProperty para consumir dados de evento](using-tdhgetproperty-to-consume-event-data.md)
-   [Recuperando metadados de evento](retrieving-event-metadata.md)
-   [Enumerar provedores](enumerating-providers.md)
