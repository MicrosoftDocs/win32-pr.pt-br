# <a name="ole-moniker-implementations"></a>Implementações de moniker OLE

O OLE fornece implementações de vários monikers para diferentes situações, conforme descrito nos tópicos a seguir:

-   [Monikers de arquivo](file-monikers.md)
-   [Monikers compostos](composite-monikers.md)
-   [Monikers de item](item-monikers.md)
-   [Anti-moniker](anti-monikers.md)
-   [Monikers de ponteiro](pointer-monikers.md)
-   [Monikers de classe](class-monikers.md)
-   [Monikers assíncronos](asynchronous-monikers.md)
-   [Identificadores de URL](url-monikers.md)

Os monikers File, Composite e item são os monikers usados com mais frequência, pois podem ser usados para fazer praticamente qualquer objeto em qualquer local. Os monikers e os monikers de ponteiro são usados principalmente dentro do OLE, mas têm algum aplicativo na implementação de monikers personalizados.
