# <a name="ole-moniker-implementations"></a>Implementações do OLE Moniker

O OLE fornece implementações de vários monikers para diferentes situações, conforme descrito nos tópicos a seguir:

-   [Monikers de arquivo](file-monikers.md)
-   [Monikers compostos](composite-monikers.md)
-   [Item Monikers](item-monikers.md)
-   [Anti monikers](anti-monikers.md)
-   [Monikers de ponteiro](pointer-monikers.md)
-   [Monikers de classe](class-monikers.md)
-   [Monikers assíncronos](asynchronous-monikers.md)
-   [URL Monikers](url-monikers.md)

Os monikers de arquivo, composição e item são os monikers usados com mais frequência, pois podem ser usados para fazer quase qualquer objeto em qualquer local. Os monikers antimoniker e ponteiro são usados principalmente dentro do OLE, mas têm algum aplicativo para implementar monikers personalizados.
