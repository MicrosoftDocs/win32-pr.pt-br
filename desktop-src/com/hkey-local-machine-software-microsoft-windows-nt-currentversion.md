# <a name="hkey_local_machinesoftwaremicrosoftwindows-ntcurrentversion"></a>HKEY \_ local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion

As subchaves e os valores do registro associados à chave **HKEY \_ local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion** contêm informações relacionadas à funcionalidade de depuração de RPC do com.



| Subchave                                                 | Descrição                                         |
|--------------------------------------------------------|-----------------------------------------------------|
| [**DebugObjectRPCEnabled**](debugobjectrpcenabled.md) | Opções de configuração para depuração remota de RPC COM. |
