# <a name="ipropertybag-and-ipersistpropertybag"></a>IPropertyBag e IPersistPropertyBag

[**IPropertyBag**](/windows/win32/api/oaidl/nn-oaidl-ipropertybag) e [**IPersistPropertyBag**](/windows/win32/api/ocidl/nn-ocidl-ipersistpropertybag) otimizam salvar como mecanismos de texto e, portanto, são recomendados para contêineres de controle ActiveX que implementam um mecanismo de salvar como texto. **IPropertyBag** é implementado por um contêiner e é aproximadamente análogo ao [**IStream.**](/windows/desktop/api/objidl/nn-objidl-istream) **IPersistPropertyBag** é implementado por controles e é aproximadamente análogo a [**IPersistStream.**](/windows/desktop/api/ObjIdl/nn-objidl-ipersiststream)
