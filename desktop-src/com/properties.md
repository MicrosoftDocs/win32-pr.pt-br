# <a name="exposing-properties-through-idispatch"></a>Expondo Propriedades por meio de IDispatch

Embora a maioria dos controles tenha propriedades, os controles não são necessários para expor nenhuma propriedade e, portanto, o controle não requer **IDispatch**. Se o controle tiver Propriedades, não haverá requisitos para quais propriedades um controle deve expor.
