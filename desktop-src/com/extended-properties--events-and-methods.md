# <a name="extended-properties-events-and-methods"></a>Propriedades, eventos e métodos estendidos

Os contêineres de controle ActiveX não são necessários para dar suporte a controles estendidos. No entanto, se o contêiner de controle oferecer suporte a propriedades estendidas, ele deverá dar suporte ao seguinte conjunto mínimo:

-   Visible
-   Pai
-   Padrão
-   Cancelar

Atualmente, propriedades estendidas, eventos e métodos não têm DISPIDs padrão.
