# <a name="appid-appid-key"></a>AppID (chave AppID)

Identifica o GUID AppID que corresponde ao executável nomeado.

## <a name="registry-entry"></a>Entrada do Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\AppID
   <Executable_name>
      AppID = {AppID_GUID}
```
