# <a name="registry-hierarchy"></a>Hierarquia do registro

O registro é estruturado como uma hierarquia de chaves, subchaves e valores nomeados ou valores padrão únicos. As subchaves podem ter várias subchaves e valores. As chaves são nomeadas por cadeias de caracteres delimitadas por barra invertida. Cada chave no registro pode ter um ou mais valores, que podem conter cadeias de caracteres, valores integrais ou dados binários.
