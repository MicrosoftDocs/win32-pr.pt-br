# <a name="visual-basic-object-browser"></a>Visual Basic Pesquisador de Objetos

Depois que um objeto COM tiver sido adicionado ao projeto como uma referência ou componente, você poderá usar o pesquisador de objetos Visual Basic para exibir informações da biblioteca de tipos para esse objeto Visual Basic.

Para exibir um objeto COM no Pesquisador de Objetos:

-   No menu **Exibir** , clique em **Pesquisador de Objetos**.

O lado esquerdo da janela Pesquisador de Objetos exibe todas as classes de referências e componentes adicionados ao projeto atual. O lado direito contém os métodos, as propriedades e os eventos dessas classes.

Você pode filtrar quais classes de objeto COM são exibidas selecionando uma biblioteca de objetos na lista suspensa seleção de biblioteca.

Para obter mais informações, [consulte Navegando ActiveX de tipo de](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) componente na documentação do Visual Basic.

Em alguns casos, o Visual Basic Object Browser parece não mostrar um tipo de retorno para uma função ou propriedade. Nenhum tipo de retorno aparece quando o tipo de retorno é o tipo padrão, **Variant**. Se o painel de descrição do Pesquisador de Objetos descrever o elemento como uma função ou propriedade e um tipo de retorno não for mostrado, a instrução de tipo de retorno será "Como Variante".
