# <a name="visual-basic-object-browser"></a>Visual Basic Pesquisador de objetos

depois que um objeto COM tiver sido adicionado ao seu projeto como uma referência ou um componente, você poderá usar o pesquisador de objetos Visual Basic para exibir informações da biblioteca de tipos para esse objeto em Visual Basic.

Para exibir um objeto COM no Pesquisador de objetos:

-   No menu **Exibir** , clique em **pesquisador de objetos**.

O lado esquerdo da janela pesquisador de objetos exibe todas as classes de referências e componentes adicionados ao projeto atual. O lado direito contém os métodos, as propriedades e os eventos dessas classes.

Você pode filtrar quais classes de objeto COM são exibidas selecionando uma biblioteca de objetos na lista suspensa seleção de biblioteca.

para obter mais informações, consulte [procurando ActiveX bibliotecas de tipo de componente](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) na documentação do Visual Basic.

em alguns casos, o pesquisador de objetos Visual Basic parece não mostrar um tipo de retorno para uma função ou propriedade. Nenhum tipo de retorno é exibido quando o tipo de retorno é o tipo padrão, **Variant**. Se o painel Descrição do pesquisador de objetos descrever o elemento como uma função ou propriedade e um tipo de retorno não for mostrado, a instrução de tipo de retorno será "como variante".
