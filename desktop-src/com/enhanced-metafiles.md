# <a name="enhanced-metafiles"></a>Metadados aprimorados

Não surpreendentemente, os metadados aprimorados fornecem mais funcionalidade do que os metadados padrão; O uso de metadados aprimorados geralmente simplifica o código de renderização. Um DC de metadados aprimorado é usado exatamente da mesma maneira que um DC de metadados padrão. O OLE dá suporte a metadados aprimorados e inclui compatibilidade com metadados padrão.

ActiveX contêineres de controle devem usar metadados aprimorados em vez de metadados padrão.
