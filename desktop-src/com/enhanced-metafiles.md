# <a name="enhanced-metafiles"></a>Metarquivos aprimorados

Não surpreendentemente, os metaarquivos aprimorados fornecem mais funcionalidade do que os metaarquivos padrão; o uso de metaarquivos aprimorados geralmente simplifica o código de renderização. Um DC de metarquivo avançado é usado exatamente da mesma forma que um DC de metarquivo padrão. O OLE dá suporte a metaarquivos avançados e inclui compatibilidade com versões anteriores com metarquivos padrão.

Os contêineres de controle ActiveX devem usar metarquivos avançados em vez de metaarquivos padrão.
