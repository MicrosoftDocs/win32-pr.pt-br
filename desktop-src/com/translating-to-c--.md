# <a name="translating-to-c"></a>Traduzindo para C++

Visual C++ fornece várias ferramentas para ajudá-lo a adicionar objetos COM ao seu projeto. Você pode adicionar um ActiveX ao projeto, usar o Assistente de Classe MFC ou executar o compilador MIDL em um arquivo IDL.

Se você estiver usando um ambiente de desenvolvimento C++ diferente Visual C++, consulte a documentação desse produto para obter informações sobre ferramentas de conversão COM.

Depois de adicionar os arquivos de stub ou de inclusão do objeto COM ao seu projeto, seu aplicativo poderá usar as classes do objeto COM.

Visual C++ também fornece o Visualizador de Objetos OLE-COM, que você pode usar para exibir informações de biblioteca de tipos compiladas. O visualizador de objetos apresenta as classes COM na sintaxe C++.
