# <a name="translating-to-c"></a>Convertendo para C++

O Visual C++ fornece várias ferramentas para ajudá-lo a adicionar objetos COM ao seu projeto. Você pode adicionar um controle ActiveX ao seu projeto, usar o assistente de classe do MFC ou executar o compilador MIDL em um arquivo IDL.

Se você estiver usando um ambiente de desenvolvimento C++ diferente de Visual C++, consulte a documentação do produto para obter informações sobre as ferramentas de conversão COM.

Depois de adicionar os arquivos de inclusão ou stub do objeto COM ao seu projeto, seu aplicativo poderá usar as classes do objeto COM.

Visual C++ também fornece o Visualizador de objetos OLE-COM, que você pode usar para exibir informações de biblioteca de tipos compiladas. O Visualizador de objetos apresenta as classes COM na sintaxe do C++.
