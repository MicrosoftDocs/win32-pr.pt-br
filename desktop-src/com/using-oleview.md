# <a name="using-oleview"></a>Usando OleView

Enquanto o editor do registro mostra uma exibição não processada do registro e é útil para aprender a organização do registro, usar o editor do registro pode ser demorado.

O OleView fornece uma exibição de nível superior das informações contidas no registro. Em vez de examinar uma longa lista de CLSIDs, OleView apresenta controles de árvore com nomes amigáveis.
