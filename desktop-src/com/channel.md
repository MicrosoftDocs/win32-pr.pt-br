# <a name="channel-com"></a>Canal (COM)

O canal tem a responsabilidade de transmitir todas as mensagens entre o cliente e o objeto no limite do processo. O canal foi projetado para funcionar de forma transparente com tipos de canal diferentes, é compatível com o RPC padrão uso DCE e dá suporte a aplicativos únicos e multithread.
