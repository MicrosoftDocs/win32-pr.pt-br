# <a name="translating-to-java"></a>Traduzindo para Java

O sistema de desenvolvimento do Microsoft Visual J++ para Java fornece duas ferramentas que você pode usar para gerar informações de biblioteca de tipos para um objeto COM na sintaxe Java, o Assistente de Biblioteca de Tipos Java e a ferramenta de linha de comando JActiveX. Usando qualquer uma dessas ferramentas, você pode gerar arquivos de classe Java para um objeto COM.

No Visual J++ versão 6.0, a ferramenta JActiveX substitui a ferramenta de linha de comando JavaTLB. Para obter mais informações, consulte a documentação do Visual J++.

Se você estiver usando um ambiente de desenvolvimento Java diferente do Visual J++, consulte a documentação do produto para obter informações sobre ferramentas de conversão COM.

Depois de gerar os arquivos de classe para um componente, você pode adicionar as classes de componente ao seu aplicativo Java usando a **instrução import.**

Em seguida, você pode criar e usar as classes do objeto em seu aplicativo Java. Para obter mais informações sobre a instrução de **importação java,** consulte uma referência na linguagem de programação Java.

Para obter mais informações, consulte estes tópicos:

-   [Traduzindo para Java do C++](translating-to-java-from-c--.md)
-   [Traduzindo para Java de Visual Basic](translating-to-java-from-visual-basic.md)
-   [Ferramenta de Command-Line JActiveX](jactivex-command-line-tool.md)
-   [Assistente de Biblioteca de Tipos Java](java-type-library-wizard.md)
