# <a name="insertable-progid-key"></a>Insertável (chave ProgID)

Indica que essa classe é insertável em contêineres OLE 2.

## <a name="registry-entry"></a>Entrada do Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes
   {ProgID}
      Insertable
```

## <a name="remarks"></a>Comentários

Essa é uma entrada necessária para objetos que podem ser inseridos em contêineres OLE 2.
