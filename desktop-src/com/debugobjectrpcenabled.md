# <a name="debugobjectrpcenabled"></a>DebugObjectRPCEnabled

O **DebugObjectRPCEnabled** habilita a depuração com remota via RPC. Se essa chave não estiver definida no servidor, a depuração remota será desabilitada.

## <a name="registry-key"></a>Chave do Registro

**HKEY \_ LOCAL \_ MACHINE \\ SOFTWARE \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled**



| Chave do Registro               | Descrição                       |
|----------------------------|-----------------------------------|
| [**AeDebug**](aedebug.md) | Armazena opções específicas do depurador. |
