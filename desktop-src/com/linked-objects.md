# <a name="linked-objects"></a>Objetos vinculados

Quando um link para um objeto é inserido em um documento composto, os dados de origem, ou a *origem do link*, continuam a residir onde ele foi inicialmente criado, geralmente em outro documento. O documento composto contém apenas uma referência, ou um *link*, para os dados reais armazenados na fonte do link, juntamente com informações sobre como apresentar esses dados ao usuário.

Ativar um link executa o aplicativo de servidor da origem do link, que o usuário precisa para editar ou manipular os dados do link. A vinculação mantém o tamanho de um documento composto pequeno. Ele também é útil quando a fonte de dados é mantida por outra pessoa e deve ser compartilhada entre vários usuários. Se a pessoa que mantém a origem do link alterar os dados, a alteração será atualizada automaticamente em todos os documentos que contiverem um link para esses dados. Além de criar links simples, os usuários podem aninhar links e combinar objetos vinculados e incorporados para criar documentos complexos.
