# <a name="notifications-com"></a>Notificações (COM)

As notificações são retornos de chamada gerados por um objeto quando ele detecta uma alteração em seu nome, estado, dados ou apresentação. Os contêineres e outros clientes exigem que as notificações respondam adequadamente a essas alterações. Um contêiner registra-se para receber notificações Configurando uma conexão de consultoria com um objeto de interesse. Outros clientes interessados podem fazer o mesmo. O contêiner também cria um coletor de consultoria para receber as notificações. Usando a conexão estabelecida pelo contêiner, um objeto que está enfrentando uma alteração notifica o coletor de consultoria. Ao receber uma notificação, o contêiner pega qualquer ação que tenha sido definida para o tipo de alteração ocorrida.

Para mais informações, consulte os seguintes tópicos:
