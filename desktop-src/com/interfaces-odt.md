# <a name="interfaces-ole-and-data-transfer"></a>Interfaces (OLE e transferência de dados)

As interfaces a seguir são usadas para implementar documentos compostos e executar a transferência de dados entre aplicativos.



| Interface                                                          | Descrição                                                                                                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Iadvisesink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)                                 | Permite que contêineres e outros objetos recebam notificações de alterações de dados, exibição de alterações e alterações de documento composto que ocorrem em objetos de interesse.                                                                                                                                                                                              |
| [**IAdviseSink2**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink2)                               | Uma extensão para [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) que adiciona um método para manipular uma alteração no moniker de um objeto vinculado.                                                                                                                                                                                                                               |
| [**IAdviseSinkEx**](/windows/desktop/api/OCIdl/nn-ocidl-iadvisesinkex)                             | Uma extensão para [**IAdviseSink que**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink) adiciona a capacidade de notificar o sink das alterações no status de exibição de um objeto.                                                                                                                                                                                                                        |
| [**IDataAdviseHolder**](/windows/desktop/api/ObjIdl/nn-objidl-idataadviseholder)                     | Cria e gerencia conexões de consultoria entre um objeto de dados e um ou mais sinks de consultoria.                                                                                                                                                                                                                                                              |
| [**Idataobject**](/windows/desktop/api/ObjIdl/nn-objidl-idataobject)                                 | Habilita a transferência de dados e a notificação de alterações nos dados.                                                                                                                                                                                                                                                                                                |
| [**IDropSource**](/windows/desktop/api/OleIdl/nn-oleidl-idropsource)                                 | Uma das interfaces que você deve implementar para fornecer operações do tipo "arrastar e soltar" em seu aplicativo.                                                                                                                                                                                                                                                         |
| [**IDropSourceNotify**](/windows/desktop/api/OleIdl/nn-oleidl-idropsourcenotify)                     | Recebe notificações quando um usuário arrasta o mouse para dentro ou para fora de uma janela de destino de soltar potencial.                                                                                                                                                                                                                                                         |
| [**Idroptarget**](/windows/desktop/api/OleIdl/nn-oleidl-idroptarget)                                 | Uma das interfaces que você deve implementar para fornecer operações do tipo "arrastar e soltar" em seu aplicativo.                                                                                                                                                                                                                                                         |
| [**IEnterpriseDropTarget**](/windows/desktop/api/OleIdl/nn-oleidl-ienterprisedroptarget)             | Quando implementada pelo aplicativo de destino drop, essa interface fornece ao mecanismo de arrastar e soltar OLE a capacidade de determinar se o aplicativo de destino de soltar pretende avaliar a política de proteção empresarial e fornece ao mecanismo de arrastar e soltar OLE uma maneira de fornecer a ID corporativa do aplicativo de origem de soltar para o aplicativo de destino de soltar. |
| [**Ienumformatetc**](/windows/desktop/api/ObjIdl/nn-objidl-ienumformatetc)                           | Enumera as estruturas [**FORMATETC**](/windows/win32/api/objidl/ns-objidl-formatetc) que definem os formatos e a mídia com suporte por um determinado objeto de dados.                                                                                                                                                                                                                              |
| [**Ienummoniker**](/windows/desktop/api/ObjIdl/nn-objidl-ienummoniker)                               | Enumera os monikers em uma tabela de monikers.                                                                                                                                                                                                                                                                                                           |
| [**IEnumOleDocumentViews**](/windows/desktop/api/DocObj/nn-docobj-ienumoledocumentviews)             | Enumera as exibições com suporte de um objeto de documento.                                                                                                                                                                                                                                                                                                      |
| [**IEnumOleUndoUnits**](/windows/desktop/api/OCIdl/nn-ocidl-ienumoleundounits)                     | Enumera as unidades de desfazer na pilha desfazer ou refazer.                                                                                                                                                                                                                                                                                                      |
| [**Ienumoleverb**](/windows/desktop/api/OleIdl/nn-oleidl-ienumoleverb)                               | Enumera os diferentes verbos disponíveis para um objeto em ordem crescente de número de verbo.                                                                                                                                                                                                                                                                 |
| [**Ienumstatdata**](/windows/desktop/api/ObjIdl/nn-objidl-ienumstatdata)                             | Enumera as informações de conexão de consultoria para um objeto de dados.                                                                                                                                                                                                                                                                                         |
| [**Iobjectwithsite**](/windows/desktop/api/OCIdl/nn-ocidl-iobjectwithsite)                         | Fornece uma maneira simples de dar suporte à comunicação entre um objeto e seu site no contêiner.                                                                                                                                                                                                                                                           |
| [**Ioleadviseholder**](/windows/desktop/api/OleIdl/nn-oleidl-ioleadviseholder)                       | Gerencia conexões de consultoria e notificações compostas de documentos em um servidor de objetos.                                                                                                                                                                                                                                                                     |
| [**IOleCache**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache)                                     | Fornece controle dos dados de apresentação armazenados em cache dentro de um objeto .                                                                                                                                                                                                                                                                             |
| [**IOleCache2**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache2)                                   | Permite que os clientes de objeto atualizem seletivamente cada cache criado com [**IOleCache::Cache**](/windows/desktop/api/OleIdl/nf-oleidl-iolecache-cache).                                                                                                                                                                                                                                |
| [**IOleCacheControl**](/windows/desktop/api/OleIdl/nn-oleidl-iolecachecontrol)                       | Fornece a manutenção adequada de caches.                                                                                                                                                                                                                                                                                                                    |
| [**Ioleclientsite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite)                           | Fornece um objeto inserido com informações sobre o local e a extensão de seu site de exibição, seu moniker, sua interface do usuário e outros recursos fornecidos por seu contêiner.                                                                                                                                                                           |
| [**IOleCommandTarget**](/windows/desktop/api/DocObj/nn-docobj-iolecommandtarget)                     | Permite que objetos e seus contêineres enviem comandos entre si.                                                                                                                                                                                                                                                                                  |
| [**Iolecontainer**](/windows/desktop/api/OleIdl/nn-oleidl-iolecontainer)                             | Enumera objetos em um documento composto ou bloquear um contêiner no estado de execução.                                                                                                                                                                                                                                                                       |
| [**Ioledocument**](/windows/desktop/api/DocObj/nn-docobj-ioledocument)                               | Permite que um objeto de documento comunique aos contêineres sua capacidade de criar exibições de seus dados.                                                                                                                                                                                                                                                           |
| [**Ioledocumentsite**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentsite)                       | Permite que um documento que foi implementado como um objeto de documento ignore a sequência de ativação normal para objetos in-locar ativos e instrua diretamente seu site cliente a ativá-lo como um objeto de documento.                                                                                                                                      |
| [**Ioledocumentview**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentview)                       | Permite que um contêiner se comunique com cada exibição com suporte por um objeto de documento.                                                                                                                                                                                                                                                                         |
| [**Ioleinplaceactiveobject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceactiveobject)         | Fornece um canal direto de comunicação entre um objeto in-place e a janela de quadro mais externa do aplicativo associado e a janela do documento dentro do aplicativo que contém o objeto inserido.                                                                                                                                          |
| [**Ioleinplaceframe**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceframe)                       | Controla a janela de quadro de nível superior do contêiner.                                                                                                                                                                                                                                                                                                          |
| [**Ioleinplaceobject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceobject)                     | Gerencia a ativação e desativação de objetos in-place e determina quanto do objeto in-place deve estar visível.                                                                                                                                                                                                                            |
| [**IOleInPlaceObjectWindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplaceobjectwindowless) | Permite que um objeto sem janela processe mensagens de janela e participe das operações de arrastar e soltar.                                                                                                                                                                                                                                                       |
| [**Ioleinplacesite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite)                         | Gerencia a interação entre o contêiner e o site do cliente in-place do objeto.                                                                                                                                                                                                                                                                          |
| [**Ioleinplacesiteex**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesiteex)                     | Fornece um conjunto adicional de métodos de notificação de ativação e desativação que permitem que um objeto evite a exibição desnecessária na tela quando o objeto é ativado e desativado.                                                                                                                                                            |
| [**Ioleinplacesitewindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesitewindowless)     | Permite que um objeto sem janela processe mensagens de janela, participe de operações de arrastar e soltar e executar operações de desenho.                                                                                                                                                                                                                       |
| [**Ioleinplaceuiwindow**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceuiwindow)                 | Negocia o espaço de borda na janela do documento ou do quadro.                                                                                                                                                                                                                                                                                                  |
| [**Iolelink**](/windows/desktop/api/OleIdl/nn-oleidl-iolelink)                                       | Permite que um objeto vinculado forneça ao contêiner funções referentes à vinculação.                                                                                                                                                                                                                                                                    |
| [**Ioleobject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject)                                   | Permite que um objeto inserido forneça funcionalidade básica para o contêiner e se comunique com ele.                                                                                                                                                                                                                                                       |
| [**IOleParentUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleparentundounit)                   | Permite que as unidades de desfazer contenham unidades de desfazer filho.                                                                                                                                                                                                                                                                                                           |
| [**IOleUILinkContainer**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera)                 | Caixas de diálogo comuns OLE usam essa interface para gerenciar as propriedades dos links de um contêiner.                                                                                                                                                                                                                                                               |
| [**IOleUILinkInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkinfoa)                           | Uma extensão para a interface [**IOleUILinkContainer**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera) que permite que os contêineres deem suporte à página **Link** da caixa **de** diálogo Propriedades do Objeto .                                                                                                                                                                            |
| [**IOleUIObjInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuiobjinfoa)                             | Fornece informações usadas  pelas páginas  **Geral** e Exibição da caixa de diálogo Propriedades do Objeto , que exibem informações sobre o tamanho, o local, o tipo e o nome do objeto. Ele também permite que o objeto seja convertido por meio da caixa **de diálogo** Converter.                                                                                          |
| [**IOleUndoManager**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundomanager)                         | Permite que os contêineres implementem operações de desfazer e refazer de vários níveis para ações que ocorrem em controles contidos.                                                                                                                                                                                                                                    |
| [**IOleUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundounit)                               | O gerenciador de desfazer chama os métodos nessa interface para executar ações de desfazer e obter cadeias de caracteres que podem ser exibidas na interface do usuário para descrever a ação de desfazer.                                                                                                                                                                                  |
| [**Iolewindow**](/windows/desktop/api/OleIdl/nn-oleidl-iolewindow)                                   | Permite que um aplicativo obtenha o handle para as várias janelas que participam da ativação in-locar e também para entrar e sair do modo de ajuda contextitivo.                                                                                                                                                                                       |
| [**Iviewobject**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject)                                 | Permite que um objeto seja exibido diretamente sem passar um objeto de dados para o chamador. Além disso, essa interface pode criar e gerenciar uma conexão com um sink de consultoria para que o chamador possa ser notificado das alterações no objeto de exibição.                                                                                                                 |
| [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2)                               | Uma extensão para a interface [**IViewObject**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject) que retorna o tamanho do desenho para uma determinada exibição de um objeto.                                                                                                                                                                                                                      |
| [**Iviewobjectex**](/windows/desktop/api/OCIdl/nn-ocidl-iviewobjectex)                             | Uma extensão para a interface [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2) que fornece suporte para desenho sem cintilação, teste de acerto e controle de replicação.                                                                                                                                                                                                       |
