# <a name="appid-clsid-key"></a>AppID (chave CLSID)

Associa um AppID a um CLSID.

## <a name="registry-entry"></a>Entrada do Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      AppID = {AppID_GUID}
```

## <a name="remarks"></a>Comentários
