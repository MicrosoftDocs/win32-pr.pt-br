# <a name="exposing-methods-through-idispatch"></a>Expondo métodos por meio de IDispatch

Embora a maioria dos controles exponha e dê suporte a vários métodos, os controles não são necessários para expor ou dar suporte a qualquer método e, portanto, o controle não requer **IDispatch**. Se o controle tiver qualquer método, não haverá requisitos para quais métodos um controle deve expor.
