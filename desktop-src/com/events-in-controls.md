# <a name="events-in-controls"></a>Eventos em controles

Embora a maioria dos controles exponha e dispare vários eventos, os controles não são necessários para expor ou acionar eventos e, portanto, o controle não requer [**IConnectionPointContainer**](/windows/desktop/api/OCIdl/nn-ocidl-iconnectionpointcontainer). Se o controle tiver algum evento, não haverá requisitos para os eventos que um controle deve expor.
