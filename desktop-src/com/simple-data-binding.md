# <a name="simple-data-binding"></a>Vinculação de dados simples

A arquitetura de controles ActiveX define um mecanismo de vinculação de dados, no qual um controle ActiveX pode especificar que uma ou mais de suas propriedades sejam vinculáveis. Na maioria dos casos, um controle vinculado a dados não deve exigir absolutamente a vinculação de dados, para que ele pudesse ser inserido em um contêiner que não ofereça suporte à vinculação de dados. Obviamente, nessa situação, a funcionalidade do controle pode ser reduzida.
