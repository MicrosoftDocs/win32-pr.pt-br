# <a name="hkey_local_machinesoftwareclasses"></a>HKEY \_ local \_ Machine \\ software \\ classes

As subchaves e os valores de registro associados à chave **HKEY \_ local \_ Machine \\ \\ classes de software** contêm informações sobre um aplicativo que é necessário para oferecer suporte à funcionalidade com. Essas informações incluem tópicos como formatos de dados com suporte, informações de compatibilidade, identificadores programáticos, DCOM e controles.



| Subchave                                                                         | Description                                                                                                       |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**AppID**](appid-key.md)                                                     | Opções de configuração para aplicativos baseados em COM.                                                                 |
| [**CLSID**](clsid-key-hklm.md)                                                | Opções de configuração para classes COM.                                                                            |
| [**><\_ extensão de arquivo**](-file-extension--key.md)                        | Associa uma extensão de nome de arquivo a um ProgID.                                                                   |
| [**FileType**](filetype-key.md)                                               | Usado por [**GetClassFile**](/windows/desktop/api/Objbase/nf-objbase-getclassfile) para corresponder padrões em vários bytes de arquivo em um arquivo não composto. |
| [**Interface**](interface-key.md)                                             | Associa um nome de interface a uma ID de interface (IID).                                                          |
| [**&lt;ProgID&gt;**](-progid--key.md)                                         | Identifica uma classe. Observe que o ProgID não tem garantia de ser globalmente exclusivo, ao contrário de um CLSID.                 |
| [**<ProgID independente de versão>**](-version-independent-progid--key.md) | Usado para determinar a versão mais recente de um aplicativo de objeto.                                                    |
