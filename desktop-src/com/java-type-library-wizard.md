# <a name="java-type-library-wizard"></a>Assistente de Biblioteca de Tipos Java

O Assistente de Biblioteca de Tipos Java é um componente do Visual J++ 5.0. Esse assistente gera arquivos de classe e informações de biblioteca de tipos Java para um objeto COM.

**Para gerar arquivos de classe Java e informações de biblioteca de tipos**

1.  No menu **Ferramentas,** clique em **Assistente de Biblioteca de Tipos Java**. Uma lista dos componentes COM instalados é exibida.
2.  Clique para selecionar as caixas de seleção ao lado dos itens que você deseja converter.
3.  Clique em **OK**. O assistente de biblioteca de tipos gera arquivos de classe para os componentes selecionados. Ele **imprime** a instrução de importação necessária para inserir cada componente em um aplicativo Java ou um applet e um arquivo de texto de resumo que contém as informações da biblioteca de tipos Java para o objeto .
