# <a name="java-type-library-wizard"></a>Assistente de biblioteca de tipos Java

O assistente da biblioteca de tipos Java é um componente do Visual J++ 5,0. Este assistente gera arquivos de classe e informações da biblioteca de tipos Java para um objeto COM.

**Para gerar arquivos de classe Java e informações de biblioteca de tipos**

1.  No menu **ferramentas** , clique em **Assistente de biblioteca de tipos Java**. É exibida uma lista dos componentes COM instalados.
2.  Clique para marcar as caixas de seleção ao lado dos itens que você deseja converter.
3.  Clique em **OK**. O assistente de biblioteca de tipos gera arquivos de classe para os componentes selecionados. Ele imprime a instrução de **importação** necessária para inserir cada componente em um aplicativo ou applet Java e um arquivo de texto de resumo que contém as informações da biblioteca de tipos do Java para o objeto.
