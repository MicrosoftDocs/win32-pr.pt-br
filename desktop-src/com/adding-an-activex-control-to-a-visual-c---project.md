# <a name="adding-an-activex-control-to-a-visual-c-project"></a>Adicionando um ActiveX controle a um Visual C++ Project

O procedimento a seguir adiciona arquivos de classe de expedição e de ActiveX para um Visual C++ projeto.

**Para adicionar um ActiveX a um Visual C++ Project**

1.  No menu **Project,** clique **em Adicionar ao Project**. Um menu de atalho é exibido.
2.  Clique **em Componentes e Controles**. A **caixa de diálogo Galeria de Componentes** e Controles é exibida.
3.  Clique no componente para adicionar ao seu projeto. Visual C++ exibe uma caixa de diálogo da qual você pode selecionar um subconjunto das classes do componente para adicionar ao projeto.
4.  Clique para selecionar as caixas de seleção das classes que você deseja adicionar e clique em **OK.**

Visual C++ gera arquivos de classe e de header de expedição para o componente e os adiciona ao seu projeto.
