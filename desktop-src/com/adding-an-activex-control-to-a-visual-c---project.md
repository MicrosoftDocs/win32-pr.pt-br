# <a name="adding-an-activex-control-to-a-visual-c-project"></a>Adicionando um controle ActiveX a um projeto Visual C++

O procedimento a seguir adiciona a classe de expedição e os arquivos de cabeçalho para um controle ActiveX a um projeto Visual C++.

**Para adicionar um controle ActiveX a um projeto Visual C++**

1.  No menu **projeto** , clique em **Adicionar ao projeto**. Um menu de atalho é exibido.
2.  Clique em **componentes e controles**. A caixa de diálogo **Galeria de componentes e controles** é exibida.
3.  Clique no componente a ser adicionado ao seu projeto. Visual C++ exibe uma caixa de diálogo na qual você pode selecionar um subconjunto das classes do componente para adicionar ao seu projeto.
4.  Clique para marcar as caixas de seleção das classes que você deseja adicionar e clique em **OK**.

Visual C++ gera arquivos de cabeçalho e classe de expedição para o componente e os adiciona ao seu projeto.
