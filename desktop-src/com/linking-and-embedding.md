# <a name="linking-and-embedding"></a>vincular e inserir

Os usuários podem criar dois tipos de objetos de documento composto: *vinculado* ou *inserido*. A diferença entre os dois tipos reside em como e onde os dados de origem do objeto são armazenados. O local em que o objeto reside afeta, por sua vez, a portabilidade e os métodos de ativação do objeto, como as atualizações de dados são executadas e o tamanho e a estrutura de seu contêiner.

-   [Objetos vinculados](linked-objects.md)
-   [Objetos inseridos](embedded-objects.md)
