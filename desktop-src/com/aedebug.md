# <a name="aedebug"></a>AeDebug

O **AeDebug** define opções de depuração remota para com.

## <a name="registry-key"></a>Chave do Registro

**HKEY \_ local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled \\ AeDebug**



| Valor do registro               | Descrição                                           |
|------------------------------|-------------------------------------------------------|
| [**Auto**](auto.md)         | Determina se o depurador é iniciado automaticamente. |
| [**Depurador**](debugger.md) | Armazena o nome do depurador.                             |
