# <a name="ambient-properties"></a>Propriedades de ambiente

No mínimo, os contêineres ActiveX Control devem dar suporte às seguintes propriedades de ambiente usando os dispids padrão.



| Propriedade Ambient             | Dispid          | Comentários/condições                                                      |
|------------------------------|-----------------|--------------------------------------------------------------------------|
| LocaleID<br/>          | -705<br/> |   <br/>                                                          |
| Usermode<br/>          | -709<br/> | Para contêineres que têm ambientes de usuário e de executar diferentes usuários.<br/> |
| DisplayAsDefault <br/> | -713<br/> | Para os contêineres em que um botão padrão é relevante.<br/>      |
