# <a name="ambient-properties"></a>Propriedades de ambiente

No mínimo, os contêineres de controle ActiveX devem dar suporte às seguintes propriedades de ambiente usando os DISPIDs padrão.



| Propriedade de ambiente             | DISPID          | Comentários/condições                                                      |
|------------------------------|-----------------|--------------------------------------------------------------------------|
| LocaleID<br/>          | -705<br/> |   <br/>                                                          |
| Modo<br/>          | -709<br/> | Para contêineres que têm ambientes diferentes de usuário e de execução.<br/> |
| DisplayAsDefault <br/> | -713<br/> | Para os contêineres onde um botão padrão é relevante.<br/>      |
