# <a name="autotreatas"></a>Tratados autotratações

Define automaticamente o CLSID para a chave [**Treats**](treatas.md) para o valor especificado.

## <a name="registry-entry"></a>Entrada do Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      AutoTreatAs = value
```

## <a name="remarks"></a>Comentários

Este é um **valor \_ sz de reg** que especifica o identificador de classe.
