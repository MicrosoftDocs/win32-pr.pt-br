# <a name="translating-to-javascript"></a>Convertendo para JavaScript

O JavaScript é uma linguagem de script criada pelo Netscape. Ele foi originalmente chamado de LiveScript. Apesar das semelhanças em nome e sintaxe, o JavaScript não está relacionado à linguagem de programação Java. O Java é uma linguagem de programação completa desenvolvida pela Sun Microsystems, Inc.

O JavaScript é compatível com uma ampla variedade de navegadores da Web, incluindo o Internet Explorer e o Netscape Navigator.

Além dos recursos de linguagem fornecidos pelo JavaScript, o host de script também pode expor um modelo de objeto que seu script pode acessar. Por exemplo, o Internet Explorer expõe um modelo de objeto que permite que os scripts interajam e controlem o navegador de forma programática. Para obter mais informações sobre como trabalhar com esses modelos de objeto, consulte a documentação em seu host de script.

Para obter mais informações sobre JavaScript, consulte uma referência de JavaScript.
