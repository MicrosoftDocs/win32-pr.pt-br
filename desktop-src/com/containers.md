# <a name="containers"></a>Contêineres

Um contêiner de controle ActiveX é um contêiner OLE que dá suporte aos seguintes recursos adicionais:

-   Objetos inseridos de servidores em processo
-   Ativação in-loco
-   OLEMISC \_ ACTIVATEWHENVISIBLE
-   Manipulação de eventos

Os contêineres de controle ActiveX devem fornecer suporte a todos esses recursos.
