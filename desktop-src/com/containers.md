# <a name="containers"></a>Contêineres

Um contêiner ActiveX Control é um contêiner OLE que dá suporte aos seguintes recursos adicionais:

-   Objetos inseridos de servidores em processo
-   Ativação in-in-place
-   OLEMISC \_ ACTIVATEWHENVISIBLE
-   Manipulação de eventos

ActiveX Os Contêineres de Controle devem dar suporte a todos esses recursos.
