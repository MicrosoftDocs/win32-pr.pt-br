# <a name="donotaddallapplicationpackagestorestrictions"></a>DoNotAddAllApplicationPackagesToRestrictions

Determina se o RPCSS anexa automaticamente um SID DE TODOS OS PACOTES DE APLICATIVOs às \_ \_ restrições COM por padrão.

## <a name="registry-entry"></a>Entrada do Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Ole
   DoNotAddAllApplicationPackagesToRestrictions = value
```

## <a name="remarks"></a>Comentários

Esse é um **valor \_ REG DWORD.**



| Valor | Descrição                                                                               |
|-------|-------------------------------------------------------------------------------------------|
| 0     | Desabilita Windows Store após a migração do Windows 7 para Windows 8.                   |
| 1     | Não adiciona o SID TODOS \_ OS PACOTES \_ DE APLICATIVOs às restrições COM. Esse é o padrão. |
