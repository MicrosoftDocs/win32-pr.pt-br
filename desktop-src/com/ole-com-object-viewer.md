# <a name="ole-com-object-viewer"></a>Visualizador de objetos OLE-COM

O Visualizador de objetos OLE-COM, Oleview.exe, é um aplicativo fornecido com Visual C++ que exibe os objetos COM instalados em seu computador e as interfaces às quais eles dão suporte. Você pode usar esse visualizador de objetos para exibir bibliotecas de tipos.

Para exibir a biblioteca de tipos de um objeto COM:

1.  No menu **arquivo** do Visualizador de objetos, clique em **Exibir TypeLib**. Será exibida uma caixa de diálogo **aberta** .
2.  Especifique o arquivo de biblioteca de tipos que você deseja abrir e clique em **OK**.

O Visualizador de objetos exibe as interfaces do objeto.

Para obter mais informações, consulte [usando o Visualizador de objetos OLE/com](/previous-versions/d0kh9f4c(v=vs.140)) na documentação do Visual C++.
