# <a name="ole-com-object-viewer"></a>Visualizador de Objetos OLE-COM

O visualizador de objetos OLE-COM, Oleview.exe, é um aplicativo fornecido com Visual C++ que exibe os objetos COM instalados em seu computador e as interfaces que eles suportam. Você pode usar esse visualizador de objetos para exibir bibliotecas de tipos.

Para exibir a biblioteca de tipos de um objeto COM:

1.  No menu Arquivo do visualizador **de** objetos, clique em **Exibir TypeLib.** Uma **caixa de** diálogo Abrir é exibida.
2.  Especifique o arquivo de biblioteca de tipos que você deseja abrir e clique em **OK.**

O visualizador de objetos exibe as interfaces do objeto.

Para obter mais informações, consulte Usando o Visualizador de [Objetos OLE/COM](/previous-versions/d0kh9f4c(v=vs.140)) na documentação Visual C++ dados.
