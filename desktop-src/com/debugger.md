# <a name="debugger"></a>Depurador

Armazena o nome do depurador preferencial a ser usado com a depuração COM remota via RPC.

## <a name="registry-entry"></a>Entrada do Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DebugObjectRPCEnabled\AeDebug
   Debugger = value
```

## <a name="remarks"></a>Comentários

Este é um valor de **\_ sz do reg** .
