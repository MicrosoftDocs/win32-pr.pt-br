# <a name="ws_group-and-ws_tabstop-flags-in-controls"></a>\_Sinalizadores WS Group e WS \_ TabStop nos controles

Um controle não deve usar os \_ sinalizadores WS Group e WS \_ TabStop internamente; alguns contêineres dependem desses sinalizadores para gerenciar o manuseio de teclado.
