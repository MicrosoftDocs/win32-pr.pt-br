# <a name="property-pages"></a>Páginas de propriedade

O suporte para páginas de propriedades e navegação por propriedade é altamente recomendado, mas não é necessário. Se um controle implementar páginas de propriedades, essas páginas deverão estar de acordo com um dos tamanhos padrão: 250x62 ou DLUs (unidades de diálogo 250x110).
