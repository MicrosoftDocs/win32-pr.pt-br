# <a name="using-the-containers-functionality"></a>Usando a funcionalidade do contêiner

as seções anteriores descreveram o suporte necessário do lado do chamador que um controle de ActiveX deve ter para acessar determinados recursos de seu contêiner. A tabela a seguir descreve o uso do controle de interfaces do lado do contêiner e quando esse uso ocorreria.



| Interface                                                     | Objeto de contêiner    | Uso                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------|---------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite)<br/>           | Site<br/>     | Controles que implementam [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject) chamam métodos [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite) como parte do protocolo de incorporação OLE padrão, especificamente os métodos [**SaveObject**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-saveobject), [**mostreobject**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-showobject), [**OnShowWindow**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-onshowwindow) (somente se houver suporte para um estado de ativação de janela separada), [**RequestNewObjectLayout**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-requestnewobjectlayout)e GetContainer (se a comunicação com outros controles for desejada). O método [**GetMoniker**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-getmoniker) só é usado quando o controle pode ser vinculado externamente, ou seja, o controle não está marcado com OLEMISC \_ CANTLINKINSIDE.<br/> |
| [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite)<br/>         | Site<br/>     | Controles que têm uma ativação in-loco e, possivelmente um estado ativo da interface do usuário, chamarão os métodos [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite) (geralmente todos eles com a exceção de [**ContextSensitiveHelp**](/windows/desktop/api/OleIdl/nf-oleidl-iolewindow-contextsensitivehelp)) como parte do protocolo de ativação in-loco no local padrão.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)<br/>                 | Site<br/>     | O controle [**chama OnDataChange**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-ondatachange) se o controle der suporte a [**IDataObject**](/windows/desktop/api/ObjIdl/nn-objidl-idataobject), [**OnViewChange**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onviewchange) se o controle der suporte a [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2), [**fechamento**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onclose), [**OnSave**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onsave)e rerename se o controle der suporte a [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject). [](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onrename)<br/>                                                                                                                                                                                                                                                                                                           |
| [**IOleControlSite**](/windows/desktop/api/OCIdl/nn-ocidl-iolecontrolsite)<br/>         | Site<br/>     | Se houver suporte, o controle chamará [**OnControlInfoChanged**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-oncontrolinfochanged) quando os mnemônicos forem alterados, [**LockInPlaceActive**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-lockinplaceactive) e [**TransformCoords**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-transformcoords) se os eventos forem disparados (o último método só será usado se as coordenadas forem passadas como argumentos de evento), [**onfocus**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-onfocus) e [**TranslateAccelerator**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-translateaccelerator) se o controle tiver um estado ativo da interface do usuário e [**GetExtendedControl**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-getextendedcontrol) se o controle quiser examinar as propriedades de controle estendido (Propriedade do contêiner).<br/>                                                                       |
| **IDispatch** (Propriedades de ambiente)<br/>                 | Site<br/>     | Usado para acessar as propriedades de ambiente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IPropertyNotifySink**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink)<br/> | Varia<br/>   | Um controle deve gerar [**OnChanged**](/windows/desktop/api/OCIdl/nf-ocidl-ipropertynotifysink-onchanged) e [**OnRequestEdit**](/windows/desktop/api/OCIdl/nf-ocidl-ipropertynotifysink-onrequestedit) para qualquer propriedade de controle marcada como \[ [**ligável**](/windows/desktop/Midl/bindable) \] e \[ [**Request**](/windows/desktop/Midl/requestedit) \] , respectivamente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Outras interfaces do coletor de eventos<br/>                        | Varia<br/>   | Um controle que tem interfaces de saída diferentes de [**IPropertyNotifySink**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink) será direcionado a outros ponteiros de interface da IID correta para as implementações de [**IConnectionPoint:: Advise**](/windows/desktop/api/OCIdl/nf-ocidl-iconnectionpoint-advise) do controle (que geralmente são encontradas em subobjetos do controle). Um controle sempre sabe como chamar suas próprias interfaces de evento porque o controle define essas interfaces.<br/>                                                                                                                                                                                                                                                                                                              |
| [**IOleInPlaceFrame**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceframe)<br/>       | Quadro<br/>    | Usado quando um controle tem um estado ativo de interface do usuário in-loco que requer ferramentas de nível de quadro ou itens de menu.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IOleInPlaceUIWindow**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceuiwindow)<br/> | Documento<br/> | Usado somente quando um controle tem um estado ativo da interface do usuário in-loco que requer ferramentas de interface do usuário no nível do documento ou do painel. Isso é raro.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
