# <a name="using-the-containers-functionality"></a>Usando a funcionalidade do contêiner

As seções anteriores descrevem alguns dos suportes necessários do lado do chamador que um controle ActiveX precisa ter para acessar determinados recursos de seu contêiner. A tabela a seguir descreve o uso de um controle de interfaces do lado do contêiner e quando esse uso ocorreria.



| Interface                                                     | Objeto Container    | Uso                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------|---------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Ioleclientsite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite)<br/>           | Site<br/>     | Os controles que implementam métodos [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject) chamam [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite) como parte do protocolo de incorporação OLE padrão, especificamente os métodos [**SaveObject**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-saveobject), [**ShowObject**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-showobject), [**OnShowWindow**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-onshowwindow) (somente se há suporte para um estado de ativação de janela separada), [**RequestNewObjectLayout**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-requestnewobjectlayout)e GetContainer (se a comunicação com outros controles for desejada). O [**método GetMoniker**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-getmoniker) só é usado quando o controle pode ser vinculado externamente, ou seja, o controle não está marcado com OLEMISC \_ CANTLINKINSIDE.<br/> |
| [**Ioleinplacesite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite)<br/>         | Site<br/>     | Os controles que têm um estado ativo in-loquente e possivelmente um estado ativo da interface do usuário chamarão métodos [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite) (geralmente todos eles com exceção de [**ContextSensitiveHelp**](/windows/desktop/api/OleIdl/nf-oleidl-iolewindow-contextsensitivehelp)) como parte do protocolo de ativação in-loca OLE padrão.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**Iadvisesink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)<br/>                 | Site<br/>     | O controle chamará [**OnDataChange**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-ondatachange) se o controle for compatível com [**IDataObject**](/windows/desktop/api/ObjIdl/nn-objidl-idataobject), [**OnViewChange**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onviewchange) se o controle for compatível com [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2)e [**OnClose,**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onclose) [**OnSave**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onsave)e [**OnRename**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onrename) se o controle for compatível com [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject).<br/>                                                                                                                                                                                                                                                                                                           |
| [**Iolecontrolsite**](/windows/desktop/api/OCIdl/nn-ocidl-iolecontrolsite)<br/>         | Site<br/>     | Se tiver suporte, o controle chamará [**OnControlInfoChanged**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-oncontrolinfochanged) quando mnemonics mudarem, [**LockInPlaceActive**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-lockinplaceactive) e [**TransformCoords**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-transformcoords) se os eventos são acionados (o último método só será usado se as coordenadas são passadas como argumentos de evento), [**OnFocus**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-onfocus) e [**TranslateAccelerator**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-translateaccelerator) se o controle tiver um estado ativo da interface do usuário e [**GetExtendedControl**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-getextendedcontrol) se o controle quiser ver as propriedades de controle estendido (de propriedade do contêiner).<br/>                                                                       |
| **IDispatch** (propriedades de ambiente)<br/>                 | Site<br/>     | Usado para acessar propriedades de ambiente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**Ipropertynotifysink**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink)<br/> | Varia<br/>   | Um controle deve gerar [**OnChanged**](/windows/desktop/api/OCIdl/nf-ocidl-ipropertynotifysink-onchanged) e [**OnRequestEdit**](/windows/desktop/api/OCIdl/nf-ocidl-ipropertynotifysink-onrequestedit) para todas as propriedades de controle marcadas como a vincáveis e \[ [](/windows/desktop/Midl/bindable) \] \[ [**solicitar**](/windows/desktop/Midl/requestedit) \] , respectivamente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Outras interfaces do sink de eventos<br/>                        | Varia<br/>   | Um controle que tem interfaces de saída diferentes de [**IPropertyNotifySink**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink) receberá outros ponteiros de interface do IID correto para as implementações [**IConnectionPoint::Advise**](/windows/desktop/api/OCIdl/nf-ocidl-iconnectionpoint-advise) do controle (que geralmente são encontradas em subpropriedades do controle). Um controle sempre sabe como chamar suas próprias interfaces de evento porque o controle define essas interfaces.<br/>                                                                                                                                                                                                                                                                                                              |
| [**Ioleinplaceframe**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceframe)<br/>       | Quadro<br/>    | Usado quando um controle tem um estado ativo da interface do usuário in-locar que requer ferramentas no nível do quadro ou itens de menu.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**Ioleinplaceuiwindow**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceuiwindow)<br/> | Documento<br/> | Usado somente quando um controle tem um estado ativo de interface do usuário in-loque requer ferramentas de interface do usuário no nível do documento ou no nível do painel. Isso é raro.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
