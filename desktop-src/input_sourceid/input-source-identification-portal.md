# <a name="input-source-identification"></a>Identificação da fonte de entrada

## <a name="purpose"></a>Finalidade

Os tópicos desta seção fornecem uma visão geral do suporte à identificação da fonte de entrada no Windows 8. A identificação de fonte de entrada permite que os aplicativos detectem com precisão e confiança uma fonte de entrada, identifiquem o tipo de entrada e forneçam uma experiência de usuário mais apropriada para o tipo de entrada.

A coleção de tópicos fornecida aqui descreve como seu aplicativo pode obter essas informações e melhor usá-la para otimizar a experiência do usuário para o cliente.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs de identificação de fonte de entrada são projetadas para desenvolvedores que estão criando estruturas de interface do usuário que fornecem uma experiência de usuários consistente e otimizada para toque em aplicativos de área de trabalho.
