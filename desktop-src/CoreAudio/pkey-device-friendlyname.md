# <a name="pkey_device_friendlyname"></a>Nome Amigável \_ do Dispositivo PKEY \_

A **propriedade PKEY \_ Device \_ FriendlyName** contém o nome amigável do dispositivo de ponto de extremidade (por exemplo, "Alto-falantes (Adaptador de Áudio XYZ)").

O **membro vt** da estrutura **PROPVARIANT** é definido como VT \_ LPWSTR.

O **membro pwszVal** da estrutura **PROPVARIANT** aponta para uma cadeia de caracteres largos terminada em nulo que contém o nome amigável do dispositivo de ponto de extremidade.
