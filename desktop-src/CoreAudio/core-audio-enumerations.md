# <a name="core-audio-enumerations"></a>Enumerações de áudio de núcleo

esta seção descreve as enumerações usadas pelas APIs de áudio principal no Windows Vista e posterior.



| Enumeração                                                                   | Descrição                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**\_AUDCLNT \_ BUFFERFLAGS**](/windows/win32/api/audioclient/ne-audioclient-_audclnt_bufferflags)                        | Sinalizadores de status para um buffer de ponto de extremidade de áudio.                                                         |
| [**AUDCLNT \_ ShareMode**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audclnt_sharemode)                               | O modo de compartilhamento para um fluxo.                                                                     |
| [**AUDCLNT \_ streamoptions**](/windows/desktop/api/audioclient/ne-audioclient-audclnt_streamoptions)                       | Define os valores que descrevem as características de um fluxo de áudio.                               |
| [**\_categoria de fluxo de áudio \_**](/windows/desktop/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)                      | Especifica a categoria de um fluxo de áudio.                                                         |
| [**AudioSessionState**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audiosessionstate)                                | O estado de uma sessão de áudio.                                                                     |
| [**ConnectorType**](/windows/win32/api/devicetopology/ne-devicetopology-connectortype)                                        | O tipo de conector em uma topologia de dispositivo.                                                        |
| [**Flow**](/windows/win32/api/devicetopology/ne-devicetopology-dataflow)                                                  | A direção do fluxo de dados de um fluxo de áudio.                                                        |
| [**EDataFlow**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-edataflow)                                                | A direção na qual os dados de áudio fluem entre um dispositivo de ponto de extremidade de áudio e um aplicativo cliente. |
| [**EndpointFormFactor**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-endpointformfactor)                              | Os atributos físicos gerais de um dispositivo de ponto de extremidade de áudio.                                       |
| [**ERole**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-erole)                                                        | A função do sistema de um dispositivo de ponto de extremidade de áudio.                                                       |
| [**\_ConnectionType do coletor KSJACK \_**](/windows/win32/api/devicetopology/ne-devicetopology-ksjack_sink_connectiontype)<br/> | O tipo de conexão.<br/>                                                                 |
| [**Parte**](/windows/win32/api/devicetopology/ne-devicetopology-parttype)                                                  | O tipo de parte de uma parte (conector ou subunidade) em uma topologia de dispositivo.                               |
