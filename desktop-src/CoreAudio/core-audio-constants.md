# <a name="core-audio-constants"></a>Principais constantes de áudio

Esta seção descreve as constantes usadas pelas principais APIs de áudio no Windows Vista e versões posteriores.

| Constante                                                                            | Descrição                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ xxx**](audclnt-sessionflags-xxx-constants.md)<br/> | Sinalizadores para a sessão de áudio que é aberta durante a inicialização do fluxo.<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ xxx**](audclnt-streamflags-xxx-constants.md)              | Sinalizadores de inicialização para um fluxo de áudio.                                           |
| [**Estado do dispositivo \_ \_ xxx**](device-state-xxx-constants.md)                            | Sinalizadores de estado para um dispositivo de ponto de extremidade de áudio.                                           |
| [**PONTO de extremidade \_ suporte de hardware \_ \_ xxx**](endpoint-hardware-support-xxx-constants.md) | Sinalizadores de suporte de hardware para um dispositivo de ponto de extremidade de áudio.                                |
