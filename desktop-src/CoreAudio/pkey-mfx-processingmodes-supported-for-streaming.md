# <a name="pkey_mfx_processingmodes_supported_for_streaming-property"></a>PKEY \_ MFX \_ ProcessingModes \_ com suporte \_ para a \_ Propriedade streaming

Lista os modos de processamento de sinal com suporte no APO de efeito de modo.

Tipo de PROPVARIANT (VT) **: \_** \| **\_ LPWSTR** de vetor VT de VT

## <a name="remarks"></a>Comentários

Essa lista inclui apenas os modos de processamento de sinais em que o APO realmente processa o sinal de áudio durante o streaming.

Esta lista não inclui nenhum modo de processamento de sinal com suporte do APO para fins de descoberta.
