# <a name="core-audio-functions"></a>Funções de áudio principais

Esta seção descreve as funções que são usadas pelas APIs de áudio principais que começam no Windows 8.



| Função                                                           | Descrição                                                                                                                                                                         |
|--------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActivateAudioInterfaceAsync**](/windows/desktop/api/mmdeviceapi/nf-mmdeviceapi-activateaudiointerfaceasync) | permite que os aplicativos da Windows Store acessem interfaces COM (Component Object Model) preexistentes na família [WASAPI](wasapi.md) .<br/> Aplica-se começando com Windows 8.<br/> |
