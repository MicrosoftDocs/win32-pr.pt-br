# <a name="pkey_deviceinterface_friendlyname"></a>PKEY \_ DeviceInterface \_ FriendlyName

A **propriedade \_ \_ FriendlyName de PKEY DeviceInterface** contém o nome amigável do adaptador de áudio ao qual o dispositivo de ponto de extremidade está anexado (por exemplo, "adaptador de áudio XYZ").

O membro **VT** da estrutura **PROPVARIANT** é definido como VT \_ LPWSTR.

O membro **pwszVal** da estrutura **PROPVARIANT** aponta para uma cadeia de caracteres largos terminada em nulo que contém o nome amigável.
