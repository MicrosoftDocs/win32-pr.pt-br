# <a name="about-the-winsnmp-20-addendum"></a>Sobre o adendo do WinSNMP 2,0

O adendo do WinSNMP versão 2,0 é uma atualização para a especificação de API do WinSNMP Manager, versão 1.1 a. (Neste momento, nenhuma especificação define a versão 2,0 da API WinSNMP.) O adendo foi desenvolvido em conjunto com a cooperação, a entrada e o suporte de várias empresas, associações e indivíduos.

A especificação do WinSNMP 1.1 a define a interface de programação de aplicativo do WinSNMP Manager para o desenvolvimento de aplicativos de gerenciamento de rede baseados em SNMP. No adendo do WinSNMP versão 2,0, a interface é renomeada como a API WinSNMP. Isso ocorre porque o WinSNMP 2,0 dá suporte às operações SNMP do agente e do Gerenciador. Você deve estar familiarizado com os dois documentos se estiver programando no ambiente WinSNMP.

A implementação do Microsoft WinSNMP é compatível com o WinSNMP 2,0. Ele dá suporte a todas as funções e operações definidas na especificação WinSNMP 1.1 a e no adendo de WinSNMP 2,0.
