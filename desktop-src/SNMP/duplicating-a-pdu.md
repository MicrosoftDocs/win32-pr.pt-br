# <a name="duplicating-a-pdu"></a>Duplicando uma PDU

A [**função SnmpDuplicatePdu**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpduplicatepdu) duplica uma PDU, alocando qualquer memória necessária. Para liberar recursos alocados por **SnmpDuplicatePdu para** uma nova PDU, um aplicativo WinSNMP deve chamar a [**função SnmpFreePdu.**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpfreepdu)
