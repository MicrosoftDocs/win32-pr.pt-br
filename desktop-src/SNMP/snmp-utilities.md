# <a name="snmp-utilities"></a>Utilitários SNMP

A tabela a seguir lista os utilitários SNMP que estão disponíveis no Microsoft Windows Resource Kit.



| Nome de arquivo     | Descrição                                                                             |
|--------------|-----------------------------------------------------------------------------------------|
| EVNTCMD.EXE  | Um aplicativo de linha de comando para configurar o tradutor de eventos SNMP                    |
| EVNTWIN.EXE  | Um aplicativo que fornece uma interface do usuário para configurar o tradutor de eventos SNMP |
| MIBCC.EXE    | O compilador de MIB SNMP                                                                   |
| SNMPUTIL.EXE | Um exemplo de aplicativo de console do Gerenciador SNMP                                               |



 

O provedor SNMP no SDK do Instrumentação de Gerenciamento do Windows (WMI) permite que aplicativos cliente acessem informações SNMP dinâmicas e estáticas por meio do WMI. Para obter mais informações, consulte [SNMP Provider](/windows/desktop/WmiSdk/snmp-provider) na documentação do SDK do WMI.
