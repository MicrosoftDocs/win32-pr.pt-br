# <a name="configuring-snmp-agent-information"></a>Configurando informações do agente SNMP

As informações do agente SNMP permitem que você especifique comentários sobre o usuário e o local físico do computador e para indicar os tipos de serviço a serem relatados. Os tipos de serviço que podem ser relatados são baseados na configuração do computador.
