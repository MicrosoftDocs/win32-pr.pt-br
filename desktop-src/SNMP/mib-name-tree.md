# <a name="mib-name-tree"></a>Árvore de nomes MIB

O espaço de nome para os identificadores de objeto MIB é hierárquico. Ele é estruturado para que cada objeto gerenciável possa ser atribuído a um nome globalmente exclusivo.

A autoridade para partes do espaço de nome é atribuída a organizações individuais. Isso permite que as organizações atribuam nomes sem consultar uma autoridade de Internet para cada atribuição. Por exemplo, o espaço de nome atribuído à Microsoft é 1.3.6.1.4.1.311, que é definido em MSFT. MIB. A Microsoft tem a autoridade de atribuir nomes a objetos em qualquer lugar abaixo desse espaço de nome.

O identificador de objeto na hierarquia é escrito como uma sequência de subidentificadores que começam na raiz e terminam no objeto. Os subidentificadores são separados por um ponto.
