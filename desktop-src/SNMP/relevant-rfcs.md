# <a name="relevant-rfcs"></a>RFCs relevantes

Os padrões de TCP/IP são definidos em RFCs (Solicitações de Comentários), que são publicadas pela IETF (Internet Engineering Task Force). A tabela a seguir lista os RFCs relevantes para o SNMP.



| Número RFC | Título                                                                                                           |
|------------|-----------------------------------------------------------------------------------------------------------------|
| 1155       | "Estrutura e identificação de informações de gerenciamento para Internets baseadas em TCP/IP." Ele define o SMI. Mib.        |
| 1157       | "Um protocolo SNMP simples". Ele define o próprio SNMP.                                          |
| 1213       | "Base de informações de gerenciamento para gerenciamento de rede de internets baseadas em TCP/IP: MIB-II". Ele define MIB \_ II.MIB. |
| 1901       | "Introdução ao SNMPv2 Community baseado em Community"                                                                        |
| 1902       | "Estrutura de informações de gerenciamento para a versão 2 do protocolo SNMPv2)"          |
| 1903       | "Convenções textuais para a versão 2 do protocolo SNMPv2)"                          |
| 1904       | "Instruções de conformidade para a versão 2 do protocolo SNMPv2)"                       |
| 1905       | "Operações de protocolo para a versão 2 do protocolo SNMPv2)"                          |
| 1906       | "Mapeamentos de transporte para a versão 2 do protocolo SNMPv2)"                           |
| 1907       | "Base de informações de gerenciamento para a versão 2 do protocolo SNMPv2)"                  |
| 1908       | "Coexistência entre a versão 1 e a versão 2 da Estrutura de Gerenciamento de Rede padrão da Internet"             |
| 2089       | "V2ToV1 Mapeando SNMPv2 para SNMPv1 dentro de um agente SNMP bilíngico"                                              |
