# <a name="about-snmp-versions"></a>Sobre as versões SNMP

A estrutura de gerenciamento de rede padrão da Internet original, descrita nas RFCs 1155, 1157 e 1213, é chamada de estrutura de SNMPv1 (SNMP versão 1). As partes relevantes da estrutura proposta para a versão 2C do SNMPv2C (Simple Network Management Protocol) são descritas nas RFCs 1901 a 1908.

A API WinSNMP dá suporte à funcionalidade de protocolo SNMP descrita nos RFCs relevantes da Internet. O WinSNMP não impõe nenhuma restrição sobre o uso do SNMPv1 ou do SNMPv2C por aplicativos de WinSNMP.

Uma entidade de gerenciamento pode dar suporte a uma versão diferente do SNMP do que a que o aplicativo WinSNMP dá suporte. A implementação do Microsoft WinSNMP executa as traduções apropriadas de SNMPv1 para SNMPv2C de acordo com a RFC relevante.
