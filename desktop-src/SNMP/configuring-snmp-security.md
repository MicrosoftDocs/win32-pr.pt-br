# <a name="configuring-snmp-security"></a>Configurando a segurança SNMP

Os recursos de segurança do SNMP permitem que você especifique as comunidades e os hosts dos quais um computador aceita solicitações, bem como o tipo de operações a serem aceitas dos computadores pertencentes a uma comunidade. O SNMP também permite que você especifique se deseja enviar uma interceptação de autenticação quando uma comunidade ou um host não autorizado solicitar informações.

para obter mais informações, consulte o Windows Resource Kit e a documentação online do serviço SNMP.

**Windows Server 2003:** Por padrão, os nomes de comunidade não são configurados e o serviço SNMP é configurado para aceitar solicitações SNMP somente do nome de host "localhost".
