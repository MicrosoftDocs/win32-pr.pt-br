# <a name="winsnmp-sessions"></a>Sessões de WinSNMP

Uma *sessão* é a unidade básica de gerenciamento de recursos e comunicações entre um aplicativo de chamada WinSNMP e a implementação do Microsoft WinSNMP. É recomendável que um aplicativo use sessões para organizar suas operações e reduzir a demanda pelos recursos da implementação.

Para obter mais informações, consulte [abrindo e fechando uma sessão WinSNMP](opening-and-closing-a-winsnmp-session.md).
