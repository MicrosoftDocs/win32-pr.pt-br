# <a name="winsnmp-sessions"></a>Sessões winSNMP

Uma *sessão* é a unidade básica de gerenciamento de recursos e comunicações entre um aplicativo WinSNMP de chamada e a implementação do Microsoft WinSNMP. É recomendável que um aplicativo use sessões para organizar suas operações e reduzir a demanda nos recursos da implementação.

Para obter mais informações, [consulte Abrindo e fechando uma sessão WinSNMP](opening-and-closing-a-winsnmp-session.md).
