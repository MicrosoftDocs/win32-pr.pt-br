# <a name="rfcs-relevant-to-winsnmp"></a>RFCs relevantes para o WinSNMP

Os padrões de TCP/IP são definidos em solicitações de comentários (RFCs), que são publicadas pelo Internet Engineering Task Force (IETF). As RFCs relevantes para os recursos de WinSNMP são listadas na tabela a seguir.



| Número da RFC | Título                                                                                                  |
|------------|--------------------------------------------------------------------------------------------------------|
| 1155       | "Estrutura e identificação de informações de gerenciamento para Internets baseadas em TCP/IP"                    |
| 1157       | "Um Simple Network Management Protocol (SNMP)"                                                          |
| 1213       | "Base de informações de gerenciamento para gerenciamento de rede de Internets baseadas em TCP/IP: MIB-II"                 |
| 1901       | "Introdução ao SNMPv2 baseado na Comunidade"                                                               |
| 1902       | "Estrutura de informações de gerenciamento para a versão 2 do protocolo SNMPv2 (Simple Network Management Protocol)" |
| 1903       | "Convenções textuais para a versão 2 do protocolo SNMPv2 (Simple Network Management Protocol)"                 |
| 1904       | "Instruções de conformidade para a versão 2 do protocolo SNMPv2 (Simple Network Management Protocol)"              |
| 1905       | "Operações de protocolo para a versão 2 do protocolo SNMPv2 (Simple Network Management Protocol)"                 |
| 1906       | "Mapeamentos de transporte para a versão 2 do protocolo SNMPv2 (Simple Network Management Protocol)"                  |
| 1907       | "Base de informações de gerenciamento para a versão 2 do protocolo SNMPv2 (Simple Network Management Protocol)"         |
| 1908       | "Coexistência entre a versão 1 e a versão 2 da estrutura de gerenciamento de rede padrão da Internet"    |
| 2089       | "V2ToV1 de mapeamento do SNMPv2 no SNMPv1 em um agente SNMP de bi-multilíngue"                                     |
