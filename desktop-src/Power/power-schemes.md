# <a name="power-schemes"></a>Esquemas de energia

Um esquema de energia é uma coleção de configurações que controla o uso de energia do seu computador. Você pode usar os esquemas de energia para reduzir o consumo de energia de dispositivos individuais ou todo o sistema. Você pode configurar recursos opcionais, como suporte para hibernação, a quantidade de tempo que o sistema deve ficar ocioso antes de desligar a exibição ou o estado de suspensão usado quando o timer de ociosidade do sistema expira.

Há várias maneiras de criar esquemas de energia: o aplicativo opções de energia no painel de controle, a ferramenta de linha de comando PowerCfg.exe e a API do esquema de energia. Para obter mais informações sobre opções de energia ou PowerCfg.exe, consulte a ajuda incluída no sistema operacional.

Os tópicos a seguir descrevem o suporte do esquema de energia fornecido pela API do esquema de energia:

-   [Gerenciamento de esquema de energia](managing-power-schemes.md)
-   [Substituições do administrador](administrator-overrides.md)
