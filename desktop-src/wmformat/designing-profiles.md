# <a name="designing-profiles"></a>Criando perfis

A implementação de um perfil não é muito difícil, mas muitas vezes é difícil decidir quais configurações melhor atendem às suas necessidades. Os perfis que você usar afetarão diretamente a experiência de reprodução de seus usuários.

As seções a seguir discutem os fatores envolvidos na criação de perfis.



| Seção                                                                                    | Descrição                                                                                                     |
|--------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Selecionando recursos de perfil](selecting-profile-features.md)                               | Discute os vários recursos de perfil e quando usá-los.                                                    |
| [Escolhendo um método de codificação](choosing-an-encoding-method.md)                             | Discute os métodos de codificação disponíveis usando os codecs de áudio e vídeo do Windows Media e quando usá-los.  |
| [Selecionando taxas de bits](selecting-bit-rates.md)                                             | Discute o processo de selecionar as taxas de bits certas para os fluxos em seu perfil.                         |
| [Trabalhando com modelos de conformidade do dispositivo](working-with-device-conformance-templates.md) | Discute os novos modelos de conformidade do dispositivo e como usá-los para personalizar o conteúdo para o público-alvo. |
