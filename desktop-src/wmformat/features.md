# <a name="features"></a>Recursos

O Windows SDK de Formato de Mídia fornece muitas opções para lidar com arquivos ASF e outros arquivos usando o Formato de Sistemas Avançados. Muitos desses recursos não são facilmente classificados em uma das principais funções do SDK. Esta seção descreve os recursos do SDK Windows Formato de Mídia de uma perspectiva de programação. Cada tópico de recurso fornece uma visão geral do recurso e fornece links para tópicos relacionados nesta documentação.



| Seção                                                                      | Descrição                                                                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Recursos de arquivo ASF](asf-file-features.md)                                   | Descreve os recursos inerentes ao ASF (Advanced Systems Format). Esses recursos geralmente são implementados por meio do perfil usado para criar um arquivo. |
| [Recursos do Codec](codec-features.md)                                         | Descreve os recursos dos codecs Windows Media que têm suporte direto pelos objetos do SDK Windows Formato de Mídia                                    |
| [Recursos de escrita de arquivo](file-writing-features.md)                           | Descreve os recursos que suportam a escrita de arquivo.                                                                                                                |
| [Recursos de leitura de arquivo](file-reading-features.md)                           | Descreve os recursos que suportam a leitura de arquivo.                                                                                                                |
| [Recursos de metadados](metadata-features.md)                                   | Descreve os recursos que suportam metadados, que são informações sobre o arquivo e seu conteúdo.                                                               |
| [Recursos Rights Management digital](digital-rights-management-features.md) | Descreve a funcionalidade de gerenciamento de direitos digitais integrada ao SDK Windows Formato de Mídia.                                                      |
