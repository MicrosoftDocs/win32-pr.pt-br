# <a name="features"></a>Recursos

O SDK do Windows Media Format fornece muitas opções para lidar com arquivos ASF e outros arquivos usando o formato de sistemas avançados. Muitos desses recursos não são facilmente classificados em uma das principais funções do SDK. Esta seção descreve os recursos do Windows Media Format SDK de uma perspectiva de programação. Cada tópico de recurso fornece uma visão geral do recurso e fornece links para tópicos relacionados nesta documentação.



| Seção                                                                      | Descrição                                                                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Recursos de arquivo ASF](asf-file-features.md)                                   | Descreve os recursos que são inerentes ao ASF (Advanced Systems Format). Normalmente, esses recursos são implementados por meio do perfil usado para criar um arquivo. |
| [Recursos do codec](codec-features.md)                                         | Descreve os recursos dos codecs de mídia do Windows que são diretamente suportados pelos objetos do SDK do Windows Media Format                                    |
| [Recursos de gravação de arquivo](file-writing-features.md)                           | Descreve os recursos que dão suporte à gravação de arquivos.                                                                                                                |
| [Recursos de leitura de arquivo](file-reading-features.md)                           | Descreve os recursos que dão suporte à leitura de arquivos.                                                                                                                |
| [Recursos de metadados](metadata-features.md)                                   | Descreve os recursos que dão suporte a metadados, que são informações sobre o arquivo e seu conteúdo.                                                               |
| [Recursos de Rights Management digital](digital-rights-management-features.md) | Descreve a funcionalidade de gerenciamento de direitos digitais integrada ao SDK do Windows Media Format.                                                      |
