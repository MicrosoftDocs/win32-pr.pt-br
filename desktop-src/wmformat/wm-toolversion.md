# <a name="wmtoolversion"></a>WM/ToolVersion

O **atributo WM/ToolVersion** contém o número de versão do aplicativo usado para criar o arquivo. Se você usar esse atributo, também deverá incluir o nome do aplicativo usando [**WM/ToolName.**](wm-toolname.md)

## <a name="global-constant"></a>Constante global

g \_ wszWMToolVersion

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**
