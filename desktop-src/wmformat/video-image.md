# <a name="video-image"></a>Imagem de vídeo

O codec do Windows Media Video 9 Image V2 permite que você crie arquivos de vídeo do Windows Media cujo conteúdo consiste em uma série de imagens ainda, como fotos digitais. As imagens no fluxo de imagem de vídeo podem ser manipuladas para produzir o deslocamento e os zooms, bem como um número de transições. Os usuários finais podem usar essa capacidade para criar e compartilhar apresentações de slides simples que também podem conter uma trilha sonora de música ou uma narração por voz. Devido à compactação superior do codec de imagem de vídeo do Windows Media, um arquivo que contém várias fotos ainda é pequeno o suficiente para enviar um email convenientemente pela Internet.

Um fluxo de imagem de vídeo tem seu próprio tipo de mídia e uma estrutura especializada que define as várias transições possíveis ao mover de uma imagem para a próxima. Você pode deslocar e aplicar zoom na exibição das imagens, combinando-as juntas. O arquivo resultante pode ser executado como qualquer outro arquivo de vídeo do Windows Media.
