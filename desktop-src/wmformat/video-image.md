# <a name="video-image"></a>Imagem de vídeo

O codec Windows Media Video 9 Image v2 permite que você crie arquivos de vídeo de mídia Windows cujo conteúdo consiste em uma série de imagens ainda, como fotografias digitais. As imagens no fluxo de Imagem de Vídeo podem ser manipuladas para produzir painéis e zooms, bem como várias transições. Os usuários finais podem usar essa funcionalidade para criar e compartilhar slides simples que também podem conter uma narração de voz ou música. Devido à compactação superior do codec Windows Media Video Image, um arquivo que contém várias fotografias ainda é pequeno o suficiente para enviar emails convenientemente pela Internet.

Um fluxo de Imagem de Vídeo tem seu próprio tipo de mídia e uma estrutura especializada que define as várias transições possíveis ao mover de uma imagem para a próxima. Você pode panorcar e ampliar a exibição das imagens enquanto as combina. O arquivo resultante pode ser interpretado como qualquer outro arquivo de vídeo Windows Mídia.
