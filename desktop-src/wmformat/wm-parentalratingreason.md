# <a name="wmparentalratingreason"></a>WM/ParentalRatingReason

O atributo **WM/ParentalRatingReason** contém os motivos para a classificação dos pais atribuída ao conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMParentalRatingReason

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Esse atributo é usado para difundir televisão e outros conteúdos de vídeo que são atribuídos a uma classificação de pais.
