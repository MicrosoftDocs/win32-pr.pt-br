# <a name="wmparentalratingreason"></a>WM/ParentRatingReason

O **atributo WM/ParentRatingReason** contém os motivos para a classificação dos pais atribuída ao conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMParentalRatingReason

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentários

Esse atributo é usado para transmissão de tv e outro conteúdo de vídeo que recebe uma classificação dos pais.
