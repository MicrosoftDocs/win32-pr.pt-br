# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Formato de mídia 9.5 SDK x64 Edition

Esta documentação se aplica ao novo SDK x64 Edition do Windows Media Format 9.5, bem como às versões anteriores de 32 bits do SDK de Formato de Mídia Windows.

A lista a seguir resume as alterações no SDK para a versão baseada em x64:

-   Não há suporte para DRM. Os arquivos relacionados ao DRM (incluindo exemplos) foram removidos da instalação do SDK.
-   Windows A Gerenciador de Dispositivos mídia não faz parte da instalação do SDK baseado em x64.
-   Os exemplos e os headers que suportam Windows interfaces de codec de áudio e vídeo de mídia não fazem parte da instalação do SDK baseado em x64.
-   Os perfis de sistema localizados não fazem parte da instalação do SDK baseado em x64.

Observações em toda a seção Referência de Programação desta documentação chamam recursos que não têm suporte no SDK x64 Windows Media Format 9.5.
