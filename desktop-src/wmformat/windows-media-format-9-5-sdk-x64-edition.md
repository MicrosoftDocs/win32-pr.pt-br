# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media Format 9,5 SDK x64 Edition

Esta documentação se aplica ao novo Windows Media Format 9,5 SDK x64 Edition, bem como às versões anteriores de 32 bits do SDK do Windows Media Format.

A lista a seguir resume as alterações no SDK para a versão baseada em x64:

-   Não há suporte para DRM. Os arquivos relacionados ao DRM (incluindo exemplos) foram removidos da instalação do SDK.
-   O Windows Media Gerenciador de Dispositivos não faz parte da instalação do SDK baseado em x64.
-   Os exemplos e cabeçalhos que dão suporte às interfaces do codec de áudio e vídeo do Windows Media não fazem parte da instalação do SDK baseado em x64.
-   Os perfis de sistema localizados não fazem parte da instalação do SDK baseado em x64.

As observações em toda a seção de referência de programação desta documentação chamam recursos que não têm suporte no Windows Media Format 9,5 SDK x64 Edition.
