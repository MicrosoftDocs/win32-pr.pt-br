# <a name="drm_actionallowed"></a>\_ACTIONALLOWED DRM

A propriedade **DRM \_ ActionAllowed** é a cadeia de caracteres base para as outras constantes de cadeia de caracteres da propriedade "ActionAllowed", como DRM \_ ActionAllowed \_ CopyToCD, porque a propriedade "ação permitida" não é extensível, essa cadeia de caracteres base não é usada para aplicativos.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ ActionAllowed

## <a name="data-type"></a>Tipo de Dados

Nenhum tipo de dados.

## <a name="remarks"></a>Comentários

As propriedades "ActionAllowed" não são extensíveis no momento.
