# <a name="wmmediaispremiere"></a>WM/MediaIsPremiere

O atributo **WM/MediaIsPremiere** indica se o conteúdo é o primeiro de uma série.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsPremiere

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ bool**

## <a name="remarks"></a>Comentários

Esse atributo é usado para difusão de televisão.
