# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Exibindo atributos DRM no editor de metadados

O editor de metadados expõe a interface IWMDRMEditor, que permite a edição de aplicativos para examinar determinados atributos de um cabeçalho DRM em um arquivo ASF protegido, e os atributos na licença de conteúdo, se houver, mesmo que o aplicativo não tenha a biblioteca estática específica do DRM (wmstubdrm. lib) que seja necessária para aplicativos do DRM Player e do gravador totalmente habilitados.

> [!Note]  
> O DRM não é compatível com a versão baseada em x64 deste SDK.
