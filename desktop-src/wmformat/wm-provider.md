# <a name="wmprovider"></a>WM/Provedor

O **atributo WM/Provider** contém o nome do provedor de conteúdo de metadados. Os metadados geralmente são obtidos por meio de um serviço.

## <a name="global-constant"></a>Constante global

g \_ wszWMProvider

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

| Tipo de arquivo | Valor de exemplo |
|-----------|---------------|
| Tudo       | "AMG"         |
