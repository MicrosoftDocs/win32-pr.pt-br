# <a name="drm-export"></a>Exportação de DRM

Windows a funcionalidade de exportação de DRM de mídia permite que você crie aplicativos licenciados que descriptografam o conteúdo protegido por Windows mídia DRM e criptografe novamente em um esquema de proteção de conteúdo de saída (CPS) especificado explicitamente pelo emissor de licença associado. O emissor da licença autoriza explicitamente a exportação de conteúdo para um CPS específico por meio de uma lista de inclusão.

> [!Note]  
> Para acessar a funcionalidade de exportação, você deve se inscrever para um [certificado de aplicativo de exportação](export-application-certificate.md) da Microsoft.

 

Windows A funcionalidade de exportação de DRM de mídia é explicada nas seções a seguir.



| Seção                                                                                      | Descrição                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Exportar certificado de aplicativo](export-application-certificate.md)                         | Descreve um certificado de aplicativo de exportação.                                                        |
| [Autorização explícita e listas de inclusão](explicit-authorization-and-inclusion-lists.md) | Explica o processo de autorização explícita e como as inclusões de lista funcionam.                      |
| [Exportando conteúdo compactado](exporting-compressed-content.md)                             | descreve como exportar conteúdo compactado de áudio Windows mídia protegida por DRM ou conteúdo de vídeo. |
