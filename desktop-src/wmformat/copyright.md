# <a name="copyright"></a>Direitos autorais

O atributo de **direitos autorais** contém uma mensagem de direitos autorais para o conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMCopyright

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="example"></a>Exemplo



| Tipo de arquivo | Valor de exemplo                          |
|-----------|----------------------------------------|
| Qualquer       | "Copyright (C) Microsoft Corporation". |



 

## <a name="remarks"></a>Comentários

O texto de uma mensagem de direitos autorais não está em conformidade com nenhum padrão. Empresas individuais podem usar qualquer formato apropriado.

Esse atributo se aplica a todo o arquivo.
