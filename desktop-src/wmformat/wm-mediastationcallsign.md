# <a name="wmmediastationcallsign"></a>WM/MediaStationCallSign

O atributo **WM/MediaStationCallSign** contém o sinal de chamada da estação que transmite o conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaStationCallSign

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Esse atributo é usado para difusão de televisão e rádio.
