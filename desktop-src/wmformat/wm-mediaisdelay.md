# <a name="wmmediaisdelay"></a>WM/MediaIsDelay

O atributo **WM/MediaIsDelay** indica se o conteúdo foi difundido como um atraso de fita.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsDelay

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ bool**

## <a name="remarks"></a>Comentários

Esse atributo é usado para difusão de televisão.
