# <a name="drm_actionallowed_playlistburn"></a>\_PlaylistBurn ACTIONALLOWED \_ DRM

O atributo **DRM \_ ActionAllowed \_ PlaylistBurn** indica se o conteúdo pode ser copiado para o CD como parte de uma lista de reprodução.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ ActionAllowed \_ PlaylistBurn

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ bool**

## <a name="remarks"></a>Comentários

Esta é uma propriedade somente leitura que é recuperada usando [**IWMDRMReader:: GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
