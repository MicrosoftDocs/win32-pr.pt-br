# <a name="description-windows-media-format-11-sdk"></a>Descrição (Windows SDK de Formato de Mídia 11)

O **atributo Description** contém uma descrição do conteúdo do arquivo.

## <a name="global-constant"></a>Constante global

g \_ wszWMDescription

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentários

Esse é um atributo de nível de arquivo.

Para arquivos ID3v1 MP3, esse atributo é limitado a 30 caracteres.
