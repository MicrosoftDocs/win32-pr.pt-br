# <a name="description-windows-media-format-11-sdk"></a>Descrição (SDK do Windows Media Format 11)

O atributo de **Descrição** contém uma descrição do conteúdo do arquivo.

## <a name="global-constant"></a>Constante global

g \_ wszWMDescription

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Esse é um atributo em nível de arquivo.

Para arquivos MP3 ID3v1, esse atributo é limitado a 30 caracteres.
