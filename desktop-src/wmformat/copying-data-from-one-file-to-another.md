# <a name="copying-data-from-one-file-to-another"></a>Copiando dados de um arquivo para outro

No nível mais básico, copiar um fluxo de um arquivo ASF para outro é bastante simples. No entanto, há problemas a considerar ao trabalhar com fluxos de vários arquivos de entrada ou ao copiar fluxos que você descompacta e codifica.

As seções a seguir descrevem a cópia de fluxos.



| Seção                                                                                              | Descripiton                                                                                    |
|------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [Copiar Fluxos sem descompactar os dados](copying-streams-without-decompressing-the-data.md) | Descreve como copiar fluxos usando exemplos compactados para preservar a qualidade do conteúdo. |
| [Copiando Fluxos usando exemplos descompactados](copying-streams-using-decompressed-samples.md)         | Descreve as dificuldades na cópia de fluxos usando exemplos descompactados.                      |
