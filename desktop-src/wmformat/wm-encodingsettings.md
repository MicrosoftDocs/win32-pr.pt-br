# <a name="wmencodingsettings"></a>WM/EncodingSettings

O **atributo WM/EncodingSettings** contém as configurações usadas para codificar o conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMEncodingSettings

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentários

Não há nenhum formato padrão para indicar configurações de codificação. No entanto, é preferível descrever as configurações de codificação em inglês em vez de incluir um despejo das configurações.
