# <a name="wmdirector"></a>WM/Director

O **atributo WM/Director** contém o nome do diretor.

## <a name="global-constant"></a>Constante global

g \_ wszWMDirector

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentários

Normalmente, os arquivos de vídeo têm diretores, os arquivos de áudio têm maior probabilidade de usar [**WM/Producer**](wm-producer.md).
