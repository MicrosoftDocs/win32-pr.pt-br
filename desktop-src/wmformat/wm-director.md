# <a name="wmdirector"></a>WM/diretor

O atributo **WM/Director** contém o nome do diretor.

## <a name="global-constant"></a>Constante global

g \_ wszWMDirector

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Normalmente, os arquivos de vídeo têm directors, os arquivos de áudio têm mais probabilidade de usar o [**WM/Producer**](wm-producer.md).
