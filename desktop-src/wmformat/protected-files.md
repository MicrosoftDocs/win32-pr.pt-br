# <a name="protected-files"></a>Arquivos protegidos

Quando um arquivo é protegido usando o Windows Media DRM, a seção de dados do arquivo é criptografada. O processo de proteção de um arquivo também é chamado de empacotamento do arquivo. As APIs estendidas do cliente DRM do Windows Media não permitem que você empacote os arquivos. Você pode empacotar arquivos usando os objetos do SDK do Windows Media Rights Manager ou do Windows Media Format SDK.

Um usuário não pode acessar os dados em um arquivo protegido sem primeiro obter uma licença para esse arquivo.
