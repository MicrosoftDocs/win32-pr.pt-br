# <a name="watermark-category-types"></a>Tipos de categoria de marca d' água

você pode usar a função de enumeração DMO, **DMOEnum**, para obter informações sobre as DMOs de marca d' água que estão instaladas em um computador. o SDK do formato de mídia Windows define guids que identificam as categorias de DMO de marca d' água. para obter mais informações sobre o **DMOEnum**, consulte a documentação do DirectShow no MSDN.

A tabela a seguir lista os identificadores para categorias de marca d' água.

| \_ \_ marca d' água de vídeo WMT DMOCATEGORY \_ | 187cc922-8efc-4404-9daf-63f4830df1bc |
