# <a name="watermark-category-types"></a>Tipos de categoria de marca d' água

Você pode usar a função de enumeração DMO, **DMOEnum**, para obter informações sobre as DMOs de marca d' água que estão instaladas em um computador. O SDK do Windows Media Format define GUIDs que identificam as categorias da marca d' água. Para obter mais informações sobre o **DMOEnum**, consulte a documentação do DIRECTSHOW no msdn.

A tabela a seguir lista os identificadores para categorias de marca d' água.



| Constante de categoria da marca d' água | GUID                                 |
|------------------------------------|--------------------------------------|
| \_ \_ marca d' água de áudio WMT DMOCATEGORY \_ | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| \_ \_ marca d' água de vídeo WMT DMOCATEGORY \_ | 187cc922-8efc-4404-9daf-63f4830df1bc |
