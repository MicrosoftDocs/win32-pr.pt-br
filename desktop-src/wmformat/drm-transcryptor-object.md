# <a name="drm-transcryptor-object"></a>Objeto de transscriptografador DRM

O objeto de transscriptografador DRM converte arquivos ASF protegidos por DRM em fluxos de dados prontos para serem enviados para dispositivos de rede que usam Windows DRM de Mídia 10 para Dispositivos de Rede.

O objeto de transscriptografador DRM é criado pela função [**WMCreateDRMTranscryptor,**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-wmcreatedrmtranscryptor) que define um ponteiro para a interface [**IWMDRMTranscryptor.**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmdrmtranscryptor) **IUnknown** e **IWMDRMTranscryptor são** as únicas interfaces com suporte neste objeto.
