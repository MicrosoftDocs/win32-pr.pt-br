# <a name="file-streams"></a>Fluxos de arquivo

Um fluxo de arquivos contém um ou mais arquivos de dados. Ao transferir informações do arquivo como um fluxo, você deve anexar o nome do arquivo a cada exemplo no fluxo. Para obter informações sobre como anexar informações relacionadas a exemplos, consulte [extensões de unidade de dados](data-unit-extensions.md).
