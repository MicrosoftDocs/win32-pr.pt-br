# <a name="wmaudiofileurl"></a>WM/AudioFileURL

O **atributo WM/AudioFileURL** contém o endereço de uma página da Web oficial com informações sobre o arquivo no qual esse atributo é exibido. Por exemplo, uma música pode ter um link de volta para a página do álbum no site do artista.

## <a name="global-constant"></a>Constante global

g \_ wszWMAudioFileURL

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**
