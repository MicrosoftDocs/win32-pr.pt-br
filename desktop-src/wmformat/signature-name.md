# <a name="signature_name"></a>Nome da assinatura \_

O **atributo \_ nome da assinatura** contém o nome no certificado usado para assinar o arquivo. Esse atributo será válido somente se o atributo [**for \_ confiável**](is-trusted.md) estiver definido como true.

## <a name="global-constant"></a>Constante global

nome do g \_ wszWMSignature \_

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Este é um atributo codificado.

Este atributo não pode ser duplicado no nível do arquivo. Se esse atributo for usado para um fluxo individual, ele será tratado como metadados personalizados e não transmitirá seu significado normal para os objetos do SDK do Windows Media Format.
