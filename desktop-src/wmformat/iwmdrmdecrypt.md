# <a name="iwmdrmdecrypt-interface"></a>Interface IWMDRMDecrypt

A interface **IWMDRMDecrypt** fornece um método para descriptografar o conteúdo.

## <a name="members"></a>Membros

A interface **IWMDRMDecrypt** herda da interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMDRMDecrypt** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IWMDRMDecrypt** tem esses métodos.



| Método                                   | Descrição                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Descriptografar**](iwmdrmdecrypt-decrypt.md) | Descriptografa um buffer de dados no local.<br/> |
