# <a name="numberofframes"></a>NumberOfFrames

O atributo **NumberOfFrames** contém o número de quadros em um fluxo de vídeo.

## <a name="global-constant"></a>Constante global

g \_ wszWMNumberOfFrames

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ QWORD**

## <a name="remarks"></a>Comentários

O gravador adiciona esse valor a um fluxo de vídeo no momento da codificação. **NumberOfFrames** não é somente leitura. No entanto, você só deve alterar o valor se editar o arquivo, alterando o número de quadros em um fluxo de vídeo.
