# <a name="wmalbumcoverurl"></a>WM/AlbumCoverURL

O atributo **WM/AlbumCoverURL** contém o endereço de uma página da Web que contém uma imagem da cobertura do álbum e informações sobre o álbum.

## <a name="global-constant"></a>Constante global

g \_ wszWMAlbumCoverURL

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**
