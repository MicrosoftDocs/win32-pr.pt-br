# <a name="codec-features"></a>Recursos do codec

o SDK do formato de mídia Windows é fornecido com vários codecs de áudio e vídeo. Você pode usar os codecs fornecidos para compactar e descompactar conteúdo de acordo com uma variedade de necessidades. O codec usado pelo gravador para compactar dados é especificado por informações de configuração de fluxo no perfil. As informações do perfil são armazenadas no cabeçalho do arquivo criado pelo gravador. Em seguida, quando o arquivo é aberto pelo leitor ou leitor síncrono, as informações de perfil no cabeçalho identificam o codec necessário para descompactar os dados.

Os recursos a seguir são discutidos nesta seção.

-   [Codecs com suporte](supported-codecs.md)
-   [Codificação de taxa de bits constante (CBR)](constant-bit-rate--cbr--encoding.md)
-   [Codificação de taxa de bits variável (VBR)](variable-bit-rate--vbr--encoding.md)
-   [Codificação de duas passagens](two-pass-encoding.md)
-   [Suporte de áudio de alta resolução](high-resolution-audio-support.md)
-   [Áudio de atraso baixo](low-delay-audio.md)
-   [Saída de áudio S/PDIF](s-pdif-audio-output.md)
-   [Imagem de vídeo](video-image.md)
-   [Modelos de conformidade do dispositivo](device-conformance-templates.md)
-   [Configurações de complexidade de vídeo](video-complexity-settings.md)
-   [Interpolação de quadros](frame-interpolation.md)
