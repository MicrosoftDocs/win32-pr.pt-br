# <a name="codec-features"></a>Recursos do Codec

O Windows SDK de Formato de Mídia é entregue com vários codecs de áudio e vídeo. Você pode usar os codecs fornecidos para compactar e descompactar o conteúdo para atender a uma variedade de necessidades. O codec usado pelo autor para compactar dados é especificado pelas informações de configuração de fluxo no perfil. As informações do perfil são armazenadas no header do arquivo criado pelo autor. Em seguida, quando o arquivo é aberto pelo leitor ou pelo leitor síncrono, as informações de perfil no header identificam o codec necessário para descompactar os dados.

Os recursos a seguir são discutidos nesta seção.

-   [Suporte a áudio de alta resolução](high-resolution-audio-support.md)
-   [Áudio de baixo atraso](low-delay-audio.md)
-   [Saída de áudio S/PDIF](s-pdif-audio-output.md)
-   [Imagem de vídeo](video-image.md)

-   [Problemas de complexidade de Configurações](video-complexity-settings.md)
