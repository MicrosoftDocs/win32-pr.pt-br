# <a name="wmt_videoimage_transition_cross_fade"></a>\_ \_ \_ esmaecimento cruzado de transição WMT VIDEOIMAGE \_

A transição de esmaecimento cruzado não tem nenhum efeito especial; as características do esmaecimento (dissolver) são determinadas pelos membros **fPrevBlendCoef** e **FCurrBlendCoef** da estrutura [**WMT \_ VIDEOIMAGE \_ SAMPLE2**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2) .

## <a name="parameters"></a>Parâmetros

Essa transição não usa parâmetros.

## <a name="remarks"></a>Comentários

Definir essa transição é o mesmo que definir o membro **dwEffectType** da estrutura **WMT \_ VIDEOIMAGE \_ SAMPLE2** como 0.
