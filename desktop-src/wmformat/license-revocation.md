<span data-ttu-id="9c555-112">O Windows Media Format SDK fornece métodos pelos quais um aplicativo pode interagir com um site do emissor da licença para revogar uma licença de conteúdo.</span><span class="sxs-lookup"><span data-stu-id="9c555-112">The Windows Media Format SDK provides methods by which an application can interact with a license issuer's Web site to revoke a content license.</span></span>
