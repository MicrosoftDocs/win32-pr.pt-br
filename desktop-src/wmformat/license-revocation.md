# <a name="license-revocation-windows-media-format-sdk"></a>Revogação de licença (SDK do Windows Media Format)

O Windows Media Format SDK fornece métodos pelos quais um aplicativo pode interagir com um site do emissor da licença para revogar uma licença de conteúdo.

Um cenário típico para revogação de licença é quando um usuário dá a um computador antigo para outra pessoa. Nesse caso, o usuário original pode ter muitas licenças para o conteúdo protegido armazenado no computador, embora esse usuário não use mais o computador. A revogação de licença permite que o usuário original remova as licenças do computador.

O processo de revogação de licença envolve as mensagens que estão sendo passadas entre o aplicativo cliente e um aplicativo baseado na Web criado pelo emissor da licença.

**Observação** O DRM não é compatível com a versão baseada em x64 deste SDK.
