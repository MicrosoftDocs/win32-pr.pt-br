# <a name="advanced-systems-format-asf-files"></a>Arquivos ASF (Advanced Systems Format)

Todos os objetos do SDK do Windows Media Format funcionam com arquivos que estão em conformidade com a estrutura de arquivos ASF (Advanced Systems Format). Esses arquivos são chamados de arquivos ASF nesta documentação. Para obter mais informações sobre a estrutura de arquivos ASF, consulte [visão geral do formato ASF](overview-of-the-asf-format.md).

Às vezes, o termo *arquivo do Windows Media* é usado para descrever os arquivos criados usando esse SDK ou outras tecnologias de mídia do Windows. Não é preciso chamar um arquivo ASF de um arquivo do Windows Media, a menos que ele contenha apenas o conteúdo codificado usando os codecs de mídia do Windows. Para obter mais informações, consulte as diretrizes para o uso das extensões de arquivo. WMA e. wmv na seção [diretrizes de extensão de nome de arquivo](file-name-extension-guidelines.md) .
