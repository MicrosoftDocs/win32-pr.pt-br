# <a name="advanced-systems-format-asf-files"></a>Arquivos ASF (Advanced Systems Format)

todos os objetos do SDK do formato de mídia Windows funcionam com arquivos que estão em conformidade com a estrutura de arquivos ASF (Advanced Systems Format). Esses arquivos são chamados de arquivos ASF nesta documentação. Para obter mais informações sobre a estrutura de arquivos ASF, consulte [visão geral do formato ASF](overview-of-the-asf-format.md).

às vezes, o termo *Windows arquivo de mídia* é usado para descrever os arquivos criados usando esse SDK ou outras tecnologias de mídia Windows. não é preciso chamar um arquivo ASF com um Windows arquivo de mídia, a menos que ele contenha apenas o conteúdo codificado usando codecs de mídia Windows. Para obter mais informações, consulte as diretrizes para o uso das extensões de arquivo. WMA e. wmv na seção [diretrizes de extensão de nome de arquivo](file-name-extension-guidelines.md) .
