# <a name="microsoft-windows-media-drm-client-enumeration-types"></a>tipos de enumeração de cliente Microsoft Windows Media DRM

a tabela a seguir descreve as enumerações que são compatíveis com as APIs estendidas do Microsoft Windows Media DRM Client.



| Enumeração                                                                      | Descrição                                                                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_resultados de \_ \_ consulta permitidos \_ da ação DRM**](drm-action-allowed-query-results.md) | Usado pela interface [**IWMDRMLicenseQuery:: QueryActionAllowed**](iwmdrmlicensequery-queryactionallowed.md) para especificar o motivo pelo qual uma ação não é permitida. |
| [**\_tipo de \_ dados attr DRM**](drm-attr-datatype.md)                                 | Define os tipos de dados usados para propriedades e atributos DRM.                                                                                                |
| [**\_tipo de criptografia DRM \_**](drm-crypto-type.md)                                     | Define os tipos de algoritmos criptográficos que podem ser usados para criptografar conteúdo.                                                                                |
| [**\_status do http DRM \_**](drmdrm-http-status.md)                                  | Define um intervalo de Estados HTTP para uma solicitação de DRM.                                                                                                             |
| [**\_status de individualização de DRM \_**](drmdrm-individualization-status.md)        | Define os Estados válidos para individualização de DRM.                                                                                                           |
| [**\_categoria de \_ estado da licença DRM \_**](drmdrm-license-state-category.md)           | Especifica o tipo de restrição de licença que é descrito por uma estrutura de [**\_ dados de \_ estado \_ de licença DRM**](drmdrm-license-state-data.md) .                    |
| [**STATUS do MSDRM \_**](msdrm-status.md)                                            | Define as condições de status do subsistema DRM.                                                                                                              |
| [**\_tipo de política WMDRMNET \_**](wmdrmnet-policy-type.md)                           | lista os tipos de políticas que estão disponíveis para Windows mídia DRM para operações de dispositivos de rede.                                                          |
| [**direitos de WMT \_**](drm-wmt-rights.md)                                            | Define os direitos que podem ser especificados em uma licença DRM.                                                                                                    |
