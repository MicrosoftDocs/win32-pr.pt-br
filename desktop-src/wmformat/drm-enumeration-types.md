# <a name="microsoft-windows-media-drm-client-enumeration-types"></a>Tipos de enumeração de cliente DRM da Mídia do Microsoft Windows

A tabela a seguir descreve as enumerações com suporte nas APIs Estendidas do Cliente DRM da Mídia do Microsoft Windows.



| Enumeração                                                                      | Descrição                                                                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**A AÇÃO DRM \_ \_ PERMITIU RESULTADOS DA \_ \_ CONSULTA**](drm-action-allowed-query-results.md) | Usado pela interface [**IWMDRMLicenseQuery::QueryActionAllowed**](iwmdrmlicensequery-queryactionallowed.md) para especificar o motivo pelo qual uma ação não é permitida. |
| [**DRM \_ ATTR \_ DATATYPE**](drm-attr-datatype.md)                                 | Define os tipos de dados usados para propriedades e atributos drm.                                                                                                |
| [**TIPO DE CRIPTOGRAFIA DRM \_ \_**](drm-crypto-type.md)                                     | Define os tipos de algoritmo criptográfico que podem ser usados para criptografar conteúdo.                                                                                |
| [**STATUS \_ HTTP DO DRM \_**](drmdrm-http-status.md)                                  | Define um intervalo de estados HTTP para uma solicitação DRM.                                                                                                             |
| [**STATUS DE INDIVIDUALIZAÇÃO DO DRM \_ \_**](drmdrm-individualization-status.md)        | Define os estados válidos para individualização de DRM.                                                                                                           |
| [**CATEGORIA DE ESTADO \_ DA \_ LICENÇA DRM \_**](drmdrm-license-state-category.md)           | Especifica o tipo de restrição de licença descrito por uma estrutura [**DE DADOS DE ESTADO DE LICENÇA \_ \_ \_ DRM.**](drmdrm-license-state-data.md)                    |
| [**STATUS DO MSDRM \_**](msdrm-status.md)                                            | Define condições de status para o subsistema DRM.                                                                                                              |
| [**TIPO DE POLÍTICA WMDRMNET \_ \_**](wmdrmnet-policy-type.md)                           | Lista os tipos de políticas disponíveis para operações de DRM de Mídia do Windows para Dispositivos de Rede.                                                          |
| [**DIREITOS DO \_ WMT**](drm-wmt-rights.md)                                            | Define os direitos que podem ser especificados em uma licença drm.                                                                                                    |
