# <a name="getting-information-from-licenses-in-the-local-license-store"></a>Obtendo informações de licenças no repositório de licenças local

As APIs estendidas do cliente DRM do Windows Media fornecem várias maneiras de obter informações sobre os direitos concedidos por licenças no repositório de licenças local. Os tópicos a seguir descrevem como obter esses vários tipos de informações.



| Tópico                                                                                                  | Descrição                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Consultando informações de direitos simples](querying-for-simple-rights-information.md)                   | Descreve como obter informações simples sobre se ações específicas são permitidas para conteúdo protegido. Por exemplo, um aplicativo pode reproduzir um arquivo de mídia.                    |
| [Consultando informações detalhadas sobre direitos](querying-for-detailed-rights-information.md)               | Descreve como obter informações agregadas detalhadas sobre os direitos concedidos por todas as licenças no repositório de licenças local para uma parte do conteúdo protegido usando a ID da chave. |
| [Enumerando licenças no repositório de licenças local](enumerating-licenses-in-the-local-license-store.md) | Descreve como obter licenças individuais do armazenamento de licença local.                                                                                                          |
