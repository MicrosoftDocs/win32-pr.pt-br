# <a name="getting-information-from-licenses-in-the-local-license-store"></a>Obter informações de licenças no armazenamento de licenças local

As WINDOWS APIs Estendidas do Cliente DRM de Mídia fornecem várias maneiras de obter informações sobre os direitos concedidos por licenças no armazenamento de licenças local. Os tópicos a seguir descrevem como obter esses vários tipos de informações.



| Tópico                                                                                                  | Descrição                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Consultando informações de direitos simples](querying-for-simple-rights-information.md)                   | Descreve como obter informações simples sobre se ações específicas são permitidas para conteúdo protegido. Por exemplo, um aplicativo pode reproduzir um arquivo de mídia.                    |
| [Consultando informações detalhadas sobre direitos](querying-for-detailed-rights-information.md)               | Descreve como obter informações agregadas detalhadas sobre os direitos concedidos por todas as licenças no armazenamento de licenças local para um conteúdo protegido usando a ID da chave. |
| [Enumerando licenças no armazenamento de licenças local](enumerating-licenses-in-the-local-license-store.md) | Descreve como obter licenças individuais do armazenamento de licenças local.                                                                                                          |
