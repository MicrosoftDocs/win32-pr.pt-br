# <a name="iwmdrmencrypt-interface"></a>Interface IWMDRMEncrypt

A interface **IWMDRMEncrypt** fornece um método para criptografar dados no local.

## <a name="members"></a>Membros

A interface **IWMDRMEncrypt** herda da interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMDRMEncrypt** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IWMDRMEncrypt** tem esses métodos.



| Método                                   | Descrição                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Encrypt**](iwmdrmencrypt-encrypt.md) | Criptografa um buffer de dados no local.<br/> |
