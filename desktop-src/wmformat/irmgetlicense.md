# <a name="irmgetlicense-interface"></a>Interface IRMGetLicense

\[**IRMGetLicense** está disponível para uso nos sistemas operacionais especificados na seção Requisitos. Ele poderá ser alterado ou ficar indisponível em versões subsequentes. Em vez disso, [use Microsoft PlayReady](https://www.microsoft.com/PlayReady/). \]

A interface **IRMGetLicense** é implementada em uma biblioteca em tempo de execução instalada como parte do SDK Windows Formato de Mídia. Ele permite a aquisição de licença do lado do cliente. Como esse método é usado por emissores de licença por meio de aplicativos baseados na Web, ele está documentado na documentação do SDK Windows Media Rights Manager.

## <a name="members"></a>Membros

A interface **IRMGetLicense** herda da interface [**IUnknown,**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) mas não tem membros adicionais.
