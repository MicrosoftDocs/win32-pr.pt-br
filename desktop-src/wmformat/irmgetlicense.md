# <a name="irmgetlicense-interface"></a>Interface IRMGetLicense

\[O **IRMGetLicense** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Ele poderá ser alterado ou ficar indisponível em versões subsequentes. Em vez disso, use [o Microsoft PlayReady](https://www.microsoft.com/PlayReady/). \]

A interface **IRMGetLicense** é implementada em uma biblioteca de tempo de execução que é instalada como parte do SDK do Windows Media Format. Ele habilita a aquisição de licença do lado do cliente. Como esse método é usado por emissores de licença por meio de aplicativos baseados na Web, ele está documentado na documentação do SDK do Windows Media Rights Manager.

## <a name="members"></a>Membros

A interface **IRMGetLicense** herda da interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) , mas não tem membros adicionais.
