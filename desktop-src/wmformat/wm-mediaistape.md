# <a name="wmmediaistape"></a>WM/MediaIsTape

O atributo **WM/MediaIsTape** indica se o conteúdo foi difundido de uma fita gravada anteriormente.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsTape

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ bool**

## <a name="remarks"></a>Comentários

Esse atributo é usado para difusão de televisão.
