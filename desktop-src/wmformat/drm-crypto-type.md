# <a name="drm_crypto_type-enumeration"></a>\_Enumeração DE TIPO DE CRIPTOGRAFIA DRM \_

A **enumeração \_ TIPO DE CRIPTOGRAFIA \_ DRM** define os tipos de algoritmo criptográfico que podem ser usados para criptografar conteúdo.

## <a name="syntax"></a>Syntax


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>Constantes

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**TIPO \_ DE \_ CRIPTOGRAFIA MCE**
</dt> <dd>

Especifica o modo de contador de 128 bits criptografia AES criptografia (AES).

</dd> </dl>
