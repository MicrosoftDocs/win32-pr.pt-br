# <a name="drm_crypto_type-enumeration"></a>Enumeração de tipo de \_ criptografia DRM \_

A enumeração do **\_ \_ tipo de criptografia DRM** define os tipos de algoritmos criptográficos que podem ser usados para criptografar conteúdo.

## <a name="syntax"></a>Syntax


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>Constantes

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**tipo de criptografia \_ \_ MCE**
</dt> <dd>

Especifica a criptografia de criptografia AES de modo de contador de 128 bits (AES).

</dd> </dl>
