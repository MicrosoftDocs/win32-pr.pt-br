# <a name="wmvideowidth"></a>WM/VideoWidth

O **atributo WM/VideoWidth** contém a largura, em pixels, de um quadro de vídeo em um fluxo.

## <a name="global-constant"></a>Constante global

g \_ wszWMVideoWidth

## <a name="data-type"></a>Tipo de Dados

**WMT \_ TYPE \_ DWORD**
