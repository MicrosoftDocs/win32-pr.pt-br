# <a name="rights-constants"></a>Constantes de direitos

As constantes listadas na tabela a seguir são usadas para identificar ações de DRM e para criar listas de ações.



| Constante                                        | Descrição                                                                                                                                                                                                                                                    |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| marca de ação da g \_ wszWMDRM \_ \_                    | Define o nome do elemento XML para uma lista de ações.                                                                                                                                                                                                               |
| \_marca de \_ ação g wszWMDRM \_                        | Define o nome do elemento XML para uma entrada em uma lista de ações.                                                                                                                                                                                                   |
| \_ \_ reprodução à direita g wszWMDRM \_                    | Define a cadeia de caracteres para o direito de reprodução de conteúdo.                                                                                                                                                                                                              |
| \_ \_ copiar à direita do g wszWMDRM \_                        | Define a cadeia de caracteres para o direito de copiar o conteúdo.                                                                                                                                                                                                              |
| \_gravação da playlist g wszWMDRM \_ Right \_ \_              | Define a cadeia de caracteres para o direito de gravar o conteúdo no CD como parte de uma lista de reprodução.                                                                                                                                                                                  |
| g \_ wszWMDRM \_ direita \_ criar \_ imagem em miniatura \_    | Define a cadeia de caracteres para o direito de criar uma imagem em miniatura do conteúdo do vídeo.                                                                                                                                                                               |
| \_ \_ \_ copiar \_ para o \_ CD do g wszWMDRM                | Define a cadeia de caracteres para o direito de copiar o conteúdo para um CD. As novas licenças não devem usar esse direito. Em vez disso, todos os direitos que concedem permissão para copiar conteúdo devem ser cobertos pelo direito de cópia e pelo direito de gravação de playlist.                                     |
| \_ \_ copiar à direita do g wszWMDRM \_ para o \_ \_ \_ dispositivo SDMI      | Define a cadeia de caracteres para o direito de copiar conteúdo para um dispositivo que esteja de acordo com o SDMI (Secure Digital Music Initiative). As novas licenças não devem usar esse direito. Em vez disso, todos os direitos que concedem permissão para copiar conteúdo devem ser cobertos pelo direito de cópia. |
| \_ \_ copiar à direita do g wszWMDRM \_ para um \_ \_ \_ dispositivo não SDMI \_ | Define a cadeia de caracteres para o direito de copiar para um dispositivo que não está de acordo com a SDMI (iniciativa de música digital) segura. As novas licenças não devem usar esse direito. Em vez disso, todos os direitos que concedem permissão para copiar conteúdo devem ser cobertos pelo direito de cópia. |
| \_ \_ backup do g wszWMDRM direito \_                      | Define a cadeia de caracteres para o direito de fazer backup da licença.                                                                                                                                                                                                        |
| g \_ wszWMDRM \_ de \_ colaboração à \_ direita         | Define a cadeia de caracteres para o direito de reprodução de conteúdo em uma rede como parte de uma lista de reprodução compartilhada.                                                                                                                                                                  |
