# <a name="wmpeakbitrate"></a>WM/PeakBitrate

O **atributo WM/PeakBitrate** contém a taxa de bits de pico de um fluxo de VBR (taxa de bits variável) em bits por segundo.

## <a name="global-constant"></a>Constante global

g \_ wszWMPeakBitrate
