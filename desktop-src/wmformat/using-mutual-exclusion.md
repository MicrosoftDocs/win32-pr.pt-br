# <a name="using-mutual-exclusion"></a>Usando a exclusão mútua

Você pode usar objetos de exclusão mútua para especificar grupos de fluxos, apenas um dos quais será entregue de cada vez. A exclusão mútua pode ser usada para lidar com uma variedade de cenários, ambos nativos ao SDK e personalizados.

As seções a seguir descrevem como trabalhar com tipos diferentes de exclusão mútua.



| Seção                                                                                  | Descrição                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [Usando a exclusão mútua de taxa de bits múltiplas](using-multiple-bit-rate-mutual-exclusion.md) | Descreve como usar a exclusão mútua para codificar uma entrada em vários fluxos com diferentes configurações de taxa de bits. |
| [Usando tipos de exclusão mútua personalizada](using-custom-mutual-exclusion-types.md)           | Discute a exclusão mútua personalizada.                                                                                |
