# <a name="microsoft-secure-audio-path-deprecated"></a>Caminho de áudio seguro da Microsoft (preterido)

Esta página documenta um recurso que terá suporte com uma solução técnica diferente em versões futuras do Windows.

Os criadores de conteúdo e os distribuidores podem especificar em uma licença de DRM que um arquivo de áudio só pode ser reproduzido em um sistema com componentes do Microsoft Secure áudio Path (SAP). O caminho de áudio seguro fornece um grau muito mais alto de proteção para o conteúdo de áudio, tornando praticamente impossível para aplicativos ou drivers de áudio não confiáveis acessarem os bits de áudio não criptografados. O caminho de áudio seguro tem suporte no Microsoft Windows® me e no Windows XP. Ele protege música digital no kernel do sistema operacional. Além disso, a lei de direitos autorais do Digital Millennium torna a burlação de medidas antipirataria no software um crime.

O caminho de áudio seguro é ativado e implementado completamente automaticamente pelo componente DRM da Microsoft quando uma licença DRM requer SAP. Para aplicativos executados no Windows® XP Service Pack 1, você pode habilitar a criptografia do SAP para qualquer conteúdo de áudio, fora do contexto da solução de DRM da Microsoft, usando o SDK de caminho de áudio seguro. Para obter mais informações sobre o SDK de caminho de áudio seguro, consulte o [site da Microsoft](/documentation/).
