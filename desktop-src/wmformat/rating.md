# <a name="rating"></a>Classificação

O atributo Classificação contém a classificação do conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMRating

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentários

Esse é um atributo de nível de arquivo.

Implementações individuais em arquivos existentes usam escalas e notações diferentes. Por isso, você deve evitar usar esse atributo; não há nenhuma padronização da qual desenvolver interfaces de usuário consistentes.
