# <a name="wmmediacredits"></a>WM/MediaCredits

O **atributo WM/MediaCredits** contém os créditos para mídia de difusão.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaCredits

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentários

Esse atributo é usado para transmissão de tv ou rádio.
