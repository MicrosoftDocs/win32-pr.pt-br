# <a name="wmproviderstyle"></a>WM/Provedorstyle

O atributo **WM/providerstyle** contém o estilo do arquivo, conforme atribuído pelo provedor de conteúdo de metadados.

## <a name="global-constant"></a>Constante global

g \_ wszWMProviderStyle

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Esse atributo é essencialmente uma segunda designação de gênero. Como classificadores diferentes têm noções diferentes dos gêneros associados ao conteúdo, o estilo do provedor pode ser diferente do gênero atribuído pelo criador de conteúdo.
