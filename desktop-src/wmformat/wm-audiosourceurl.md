# <a name="wmaudiosourceurl"></a>WM/AudioSourceURL

O **atributo WM/AudioSourceURL** contém o endereço de uma página oficial da Web da mídia de origem. Por exemplo, uma música de uma apresentação pode ter um link para a página oficial do filme.

## <a name="global-constant"></a>Constante global

g \_ wszWMAudioSourceURL

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**
