# <a name="wmaudiosourceurl"></a>WM/AudioSourceURL

O atributo **WM/AudioSourceURL** contém o endereço de uma página da Web oficial da mídia de origem. Por exemplo, uma música de uma trilha sonora pode ter um link para a página oficial do filme.

## <a name="global-constant"></a>Constante global

g \_ wszWMAudioSourceURL

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**
