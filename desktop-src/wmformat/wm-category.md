# <a name="wmcategory"></a>WM/categoria

O atributo **WM/Category** contém a categoria do conteúdo. Não há especificações para categorias. Os usuários devem ser incentivados a inserir suas próprias categorias para organizar seu conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMCategory

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="example"></a>Exemplo



| Tipo de arquivo | Valor de exemplo   |
|-----------|-----------------|
| Áudio     | "Impulsionando música" |
| Vídeo     | "Vídeos caseiros"   |
