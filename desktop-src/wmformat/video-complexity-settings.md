# <a name="video-complexity-settings"></a>Configurações de complexidade de vídeo

Alguns codecs de vídeo dão suporte a vários *níveis de complexidade*. O nível de complexidade usado para codificar um fluxo não afeta diretamente a taxa de bits desse fluxo, mas pode afetar a qualidade. Em que taxa de bits é uma medida do tamanho dos exemplos compactados no arquivo ASF, o nível de complexidade é uma medida da capacidade de processamento necessária para reconstruir os dados compactados. Portanto, o nível de complexidade é ditado mais pelo poder de processamento da plataforma de reprodução do que pela largura de banda disponível.

para obter informações sobre como trabalhar com níveis de complexidade, consulte [configurando Fluxos de vídeo](configuring-video-streams.md).
