# <a name="wmtoolname"></a>WM/ToolName

O atributo **WM/ToolName** contém o nome do aplicativo usado para criar o arquivo. Você também pode especificar a versão do aplicativo usando o [**WM/ToolVersion**](wm-toolversion.md).

## <a name="global-constant"></a>Constante global

g \_ wszWMToolName

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**
