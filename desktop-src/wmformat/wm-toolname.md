# <a name="wmtoolname"></a>WM/ToolName

O **atributo WM/ToolName** contém o nome do aplicativo usado para criar o arquivo. Você também pode especificar a versão do aplicativo usando [**WM/ToolVersion.**](wm-toolversion.md)

## <a name="global-constant"></a>Constante global

g \_ wszWMToolName

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**
