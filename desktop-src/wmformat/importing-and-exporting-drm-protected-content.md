# <a name="importing-and-exporting-drm-protected-content"></a>Importando e exportando conteúdo protegido por DRM

Uma das principais funções das novas APIs estendidas do cliente drm de Windows mídia é habilitar a importação e a exportação de conteúdo protegido por DRM para sistemas de gerenciamento de direitos de terceiros.

O principal desafio de importar e exportar dados protegidos em seu aplicativo é compartilhar robustamente exemplos de dados ASF entre o sistema e o subsistema DRM. Quando seu aplicativo lida com exemplos de dados, ele precisa garantir que determinados padrões de robustez sejam atendidos para evitar que o mecanismo de proteção seja contornado. Esses padrões são descritos nas regras de robustez e conformidade que você concorda ao licenciar o DRM.
