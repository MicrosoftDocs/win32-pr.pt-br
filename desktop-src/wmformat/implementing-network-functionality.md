# <a name="implementing-network-functionality"></a>Implementando a funcionalidade de rede

esta seção descreve como usar os recursos de rede disponíveis por meio do SDK do formato de mídia Windows. Um aplicativo pode usar esses recursos para ler fluxos ASF de uma rede, transmitir fluxos ASF para uma rede ou enviar fluxos ASF para um ponto de publicação em um servidor que executa o Windows Media Services.

As seções a seguir descrevem os recursos de rede deste SDK.



| Seção                                                                    | Descrição                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Visão geral das interfaces de rede](overview-of-networking-interfaces.md) | Nomes e descreve cada interface que dá suporte a métodos de rede.                                                 |
| [Lendo dados ASF por meio de uma rede](reading-asf-data-over-a-network.md)     | Descreve como reproduzir arquivos de uma fonte de rede e como definir as configurações de rede no objeto leitor.    |
| [Enviando dados ASF por meio de uma rede](sending-asf-data-over-a-network.md)     | descreve como transmitir dados asf por uma rede ou enviar dados asf para um ponto de publicação em um servidor de mídia Windows. |
| [Configurações de rede padrão](default-networking-settings.md)             | Fornece uma referência rápida para as configurações padrão usadas pelo SDK.                                                 |
