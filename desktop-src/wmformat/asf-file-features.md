# <a name="asf-file-features"></a>Recursos de arquivo ASF

A principal finalidade do SDK de Formato de Mídia do Windows é fornecer suporte para encapsular dados de mídia digital em arquivos ASF (Advanced Systems Format) e entregar a mídia a um aplicativo cliente. Os cenários de entrega podem variar muito de aplicativo para aplicativo, mas todos usam a estrutura de arquivo ASF entre a autorização e a entrega. Os arquivos ASF estão em conformidade com uma estrutura bem definida, mas muito flexível. Para obter mais informações sobre a estrutura do arquivo ASF, consulte [Visão geral do formato ASF](overview-of-the-asf-format.md).

Informações detalhadas sobre os dados em um arquivo ASF são fornecidas na especificação do ASF, que você pode baixar do [site da Microsoft.](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc)

O Windows SDK de Formato de Mídia oferece suporte para os recursos da especificação do ASF principalmente por meio do perfil usado para criar um arquivo. Para obter mais informações sobre perfis, consulte [Perfis](profiles.md).
