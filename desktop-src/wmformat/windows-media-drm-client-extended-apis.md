# <a name="windows-media-drm-client-extended-apis"></a>Windows APIs estendidas do cliente DRM de mídia

\[O Windows drm de mídia de mídia foi preterido e não deve ser usado. Use [Microsoft PlayReady](/windows/uwp/audio-video-camera/playready-client-sdk) em vez disso.\]

Esta documentação descreve as APIs (Interfaces de Programação de Aplicativos Estendidos) do DRM (Microsoft Windows Media Digital Rights Management). As APIs estendidas do cliente drm de Windows mídia incluem objetos que podem ser usados para gerenciar Windows operações drm (Rights Management de mídia digital) em um computador cliente.

O foco principal dessas APIs é o gerenciamento de licenças para conteúdo de mídia digital protegido. Além disso, as APIs podem ser usadas para atualizar os componentes drm no computador cliente e para criar aplicativos que transmitem conteúdo usando Windows DRM de Mídia para Dispositivos de Rede.

Essas APIs constituem o equivalente do lado do cliente ao SDK Windows Media Rights Manager. Quando Windows Media Rights Manager é usado para criar um serviços online que protege arquivos e emitir licenças, as APIs estendidas do cliente drm de mídia do Windows são usadas para criar aplicativos que consomem esse conteúdo.
