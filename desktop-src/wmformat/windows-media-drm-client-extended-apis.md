# <a name="windows-media-drm-client-extended-apis"></a>APIs estendidas do cliente DRM do Windows Media

\[O recurso DRM do Windows Media foi preterido e não deve ser usado. Em vez disso, use [o Microsoft PlayReady](/windows/uwp/audio-video-camera/playready-client-sdk) .\]

Esta documentação descreve as APIs (interfaces de programação de aplicativo) do Microsoft Windows Media Digital Rights Management (DRM) cliente estendidas. As APIs estendidas do cliente DRM do Windows Media incluem objetos que podem ser usados para gerenciar as operações do Windows Media Digital Rights Management (DRM) em um computador cliente.

O foco principal dessas APIs é o gerenciamento de licenças para conteúdo de mídia digital protegido. Além disso, as APIs podem ser usadas para atualizar os componentes DRM no computador cliente e para criar aplicativos que transmitam conteúdo usando o Windows Media DRM para dispositivos de rede.

Essas APIs constituem a contraparte do lado do cliente para o SDK do Windows Media Rights Manager. Onde o Windows Media Rights Manager é usado para criar serviços online que protegem arquivos e emitem licenças, as APIs estendidas do cliente DRM do Windows Media são usadas para criar aplicativos que consomem esse conteúdo.
