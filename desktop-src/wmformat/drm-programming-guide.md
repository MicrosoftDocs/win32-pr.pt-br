|----------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|

| [Obtendo informações de licenças no repositório de licenças local](getting-information-from-licenses-in-the-local-license-store.md) | Descreve como obter informações sobre direitos de conteúdo protegido.                                       |

| [Importação de DRM](drm-import.md)                                                                                                     | Descreve como importar conteúdo protegido pelo Windows Media DRM.                                            |
