# <a name="microsoft-windows-media-drm-client-programming-guide"></a>Guia de programação de cliente drm de Windows mídia da Microsoft

|----------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|

| [Obter informações de licenças no armazenamento de licenças local](getting-information-from-licenses-in-the-local-license-store.md) | Descreve como obter informações sobre direitos para conteúdo protegido.                                       |
