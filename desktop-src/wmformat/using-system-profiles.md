# <a name="using-system-profiles"></a>Usando perfis de sistema

Os perfis de sistema incluídos no Windows Media Format SDK foram projetados para abranger as necessidades básicas do conteúdo de mídia digital composto por áudio e vídeo. Devido à grande variedade de recursos disponíveis e ao aumento relacionado em possíveis cenários de mídia digital, os perfis de sistema foram enfatizados. O conjunto mais recente de perfis de sistema faz uso dos codecs de mídia do Windows versão 8 que foram lançados com o Windows XP. Apesar de não oferecer suporte à versão mais recente dos codecs, os perfis de sistema tornam-se um excelente ponto de partida para a criação de perfis para atender às suas necessidades.
