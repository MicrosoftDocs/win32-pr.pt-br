# <a name="using-system-profiles"></a>Usando perfis do sistema

Os perfis do sistema incluídos no SDK Windows Media Format foram projetados para abranger as necessidades básicas do conteúdo de mídia digital composto por áudio e vídeo. Devido à grande variedade de recursos agora disponíveis e ao aumento relacionado em possíveis cenários de mídia digital, os perfis do sistema foram desemphasized. O conjunto mais recente de perfis do sistema usa os codecs de mídia Windows versão 8 que foram lançados com Windows XP. Apesar de não dar suporte à versão mais recente dos codecs, os perfis do sistema fazem um excelente ponto de partida para a criação de perfis para atender às suas necessidades.
