# <a name="wmlyrics"></a>WM/letras de música

O atributo **WM/letras de música** contém as letras de músicas como uma cadeia de caracteres simples.

## <a name="global-constant"></a>Constante global

g \_ wszWMLyrics

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Para armazenamento mais estruturado de letras de música, use o atributo [**\_ sincronizado do WM/música**](wm-lyrics-synchronised.md) .
