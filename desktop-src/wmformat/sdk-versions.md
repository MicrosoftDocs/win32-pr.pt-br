# <a name="sdk-versions"></a>Versões do SDK

Esta documentação aborda o SDK do Windows Media Format 7 por meio do SDK do Windows Media Format 11. Os métodos, as estruturas, as funções e as enumerações individuais incluem requisitos de versão em seus tópicos de referência.

O Windows Media Format 11 SDK tem suporte apenas no Windows Vista e no Windows XP.

O Windows Media Format 9,5 SDK tem suporte apenas no Windows XP. Se você escrever um aplicativo com base no SDK 9,5 do Windows Media Format, seu aplicativo não será executado em nenhum outro sistema operacional.

Alguns recursos do SDK do Windows Media Format 9 Series também exigem o Windows XP. Os tópicos de referência individuais relacionados a esses recursos chamam esse requisito.
