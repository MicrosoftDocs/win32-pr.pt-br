<span data-ttu-id="06dac-107">O ASF (Advanced Systems Format) é um formato de arquivo extensível projetado principalmente para armazenar e reproduzir fluxos de mídia digital sincronizados e transmiti-los por redes.</span><span class="sxs-lookup"><span data-stu-id="06dac-107">The Advanced Systems Format (ASF) is an extensible file format designed primarily for storing and playing synchronized digital media streams and transmitting them over networks.</span></span> <span data-ttu-id="06dac-108">O ASF é o formato de contêiner para o Windows Media Audio e o conteúdo baseado em vídeo do Windows Media.</span><span class="sxs-lookup"><span data-stu-id="06dac-108">ASF is the container format for Windows Media Audio and Windows Media Video-based content.</span></span> <span data-ttu-id="06dac-109">A extensão WMA ou WMV é usada para especificar um arquivo ASF que contém conteúdo codificado com os codecs de vídeo do Windows Media Audio e/ou Windows Media.</span><span class="sxs-lookup"><span data-stu-id="06dac-109">The extension wma or wmv is used to specify an ASF file that contains content encoded with the Windows Media Audio and/or Windows Media Video codecs.</span></span> <span data-ttu-id="06dac-110">O Windows Media Format SDK pode ser usado para criar e ler arquivos de mídia do Windows, bem como arquivos ASF que contêm outros tipos de dados compactados ou descompactados.</span><span class="sxs-lookup"><span data-stu-id="06dac-110">The Windows Media Format SDK can be used to create and read Windows Media files, as well as ASF files that contain other types of compressed or uncompressed data.</span></span>

<span data-ttu-id="06dac-111">Esta seção fornece uma descrição geral do formato ASF como informações de segundo plano.</span><span class="sxs-lookup"><span data-stu-id="06dac-111">This section provides a general description of the ASF format as background information.</span></span> <span data-ttu-id="06dac-112">Como os objetos Reader e Writer lidam com todas as tarefas de formatação e análise de arquivo de nível baixo, não é necessário ter uma compreensão detalhada do ASF antes de usar esse SDK para criar arquivos ASF.</span><span class="sxs-lookup"><span data-stu-id="06dac-112">Because the reader and writer objects handle all low-level file parsing and formatting tasks, it is not necessary to have a detailed understanding of ASF before using this SDK to create ASF files.</span></span> <span data-ttu-id="06dac-113">A especificação ASF completa pode ser encontrada no [site da Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).</span><span class="sxs-lookup"><span data-stu-id="06dac-113">The complete ASF specification can be found on the [Microsoft Web site](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).</span></span>

<span data-ttu-id="06dac-114">Os principais objetivos do formato ASF são:</span><span class="sxs-lookup"><span data-stu-id="06dac-114">The primary goals of the ASF format are:</span></span>

-   <span data-ttu-id="06dac-115">Para dar suporte à reprodução eficiente de servidores de mídia, servidores HTTP e dispositivos de armazenamento local.</span><span class="sxs-lookup"><span data-stu-id="06dac-115">To support efficient playback from media servers, HTTP servers, and local storage devices.</span></span>
-   <span data-ttu-id="06dac-116">Para dar suporte a tipos de mídia escalonáveis, como áudio e vídeo.</span><span class="sxs-lookup"><span data-stu-id="06dac-116">To support scalable media types such as audio and video.</span></span>

-   <span data-ttu-id="06dac-119">Para ser independente de qualquer sistema de composição de multimídia, sistema operacional de computador ou protocolo de comunicação de dados específico.</span><span class="sxs-lookup"><span data-stu-id="06dac-119">To be independent of any particular multimedia composition system, computer operating system, or data communications protocol.</span></span>

<span data-ttu-id="06dac-120">Um arquivo ASF pode conter vários fluxos independentes ou dependentes, incluindo vários fluxos de áudio para áudio de vários canais ou fluxos de vídeo de taxa de bits múltiplos adequados para transmissão por diferentes larguras de banda.</span><span class="sxs-lookup"><span data-stu-id="06dac-120">An ASF file can contain multiple independent or dependent streams, including multiple audio streams for multichannel audio, or multiple bit rate video streams suitable for transmission over different bandwidths.</span></span> <span data-ttu-id="06dac-121">Os fluxos podem estar em qualquer formato compactado ou descompactado; <span data-ttu-id="06dac-125">Um arquivo ASF é organizado em seções chamadas "objetos".</span><span class="sxs-lookup"><span data-stu-id="06dac-125">An ASF file is organized into sections called "objects."</span></span> <span data-ttu-id="06dac-126">Há três objetos de nível superior, um objeto de cabeçalho e um objeto de dados (ambos necessários), além de um objeto de índice codecs usados.</span><span class="sxs-lookup"><span data-stu-id="06dac-127">The Header object contains general information about the file, such as file size, number of streams, error correction methods, and codecs used.</span></span> <span data-ttu-id="06dac-128">Os metadados também são armazenados aqui.</span><span class="sxs-lookup"><span data-stu-id="06dac-128">Metadata is also stored here.</span></span> <span data-ttu-id="06dac-129">O objeto de cabeçalho é o único objeto de nível superior que pode conter outros objetos.</span><span class="sxs-lookup"><span class="sxs-lookup"><span data-stu-id="06dac-132">The index associated with each key frame can be a presentation time, a video frame number, or a reference time stamp.</span></span>

<span data-ttu-id="06dac-133">Cada objeto de nível superior ou de nível inferior começa com um GUID (identificador global exclusivo) e um valor de tamanho.</span><span class="sxs-lookup"><span data-stu-id="06dac-133">Each top-level or lower-level object begins with a globally unique identifier (GUID) and a size value.</span></span> <span data-ttu-id="06dac-134">Esses números permitem que o leitor de arquivos analise as informações em locais apropriados em objetos identificáveis.</span><span class="sxs-lookup"><span data-stu-id="06dac-134">These numbers allow the file reader to parse the information at appropriate places into identifiable objects.</span></span> <span data-ttu-id="06dac-135">Devido a esses GUIDs, os objetos de nível inferior podem ser enviados em qualquer ordem e ainda serem reconhecidos.</span><span class="sxs-lookup"><span data-stu-id="06dac-135">Because of these GUIDs, lower-level objects can be sent in any order and still be recognized.</span></span> <span data-ttu-id="06dac-136">O formato ASF foi projetado para superar a recepção de dados imprecisa.</span><span class="sxs-lookup"><span data-stu-id="06dac-136">The ASF format is designed to overcome inaccurate data reception.</span></span> <span data-ttu-id="06dac-137">Um arquivo ASF parcialmente baixado ainda pode ser lido, contanto que ele contenha o objeto de cabeçalho e pelo menos um objeto de dados.</span><span class="sxs-lookup"><span data-stu-id="06dac-137">A partially downloaded ASF file can still be read, as long as it contains the Header object and at least one Data object.</span></span>
