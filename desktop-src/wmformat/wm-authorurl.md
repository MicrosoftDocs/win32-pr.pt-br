# <a name="wmauthorurl"></a>WM/AuthorURL

O atributo **WM/AuthorURL** contém o endereço do site do autor.

## <a name="global-constant"></a>Constante global

g \_ wszWMAuthorURL

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

O endereço especificado por esse atributo deve corresponder ao artista ou autor em um atributo [**autor**](author.md) no arquivo.
