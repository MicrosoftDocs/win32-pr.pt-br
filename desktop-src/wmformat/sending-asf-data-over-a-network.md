# <a name="sending-asf-data-over-a-network"></a>Enviando dados ASF por meio de uma rede

O SDK do Windows Media Format dá suporte a duas maneiras diferentes de enviar dados ASF por uma rede:

-   O aplicativo pode transmitir os dados ASF usando o protocolo HTTP.
-   O aplicativo pode enviar por push os dados do ASF para um ponto de publicação em um servidor do Windows Media. Em seguida, o servidor distribui os dados.

Os tópicos a seguir descrevem esses recursos:

-   [Transmissão de dados ASF](broadcasting-asf-data.md)
-   [Enviando dados ASF para um ponto de publicação](sending-asf-data-to-a-publishing-point.md)
