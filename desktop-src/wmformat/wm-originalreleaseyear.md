# <a name="wmoriginalreleaseyear"></a>WM/OriginalReleaseYear

O atributo **WM/OriginalReleaseYear** contém o ano durante o qual o conteúdo foi lançado pela primeira vez.

## <a name="global-constant"></a>Constante global

g \_ wszWMOriginalReleaseYear

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="example"></a>Exemplo



| Tipo de arquivo | Valor de exemplo |
|-----------|---------------|
| Tudo       | "2002"        |
