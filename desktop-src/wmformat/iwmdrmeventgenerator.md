# <a name="iwmdrmeventgenerator-interface"></a>Interface IWMDRMEventGenerator

A interface **IWMDRMEventGenerator** é uma extensão da interface **IMFMediaEventGenerator** que fornece um método para cancelar operações assíncronas.

## <a name="members"></a>Membros

A interface **IWMDRMEventGenerator** herda da interface [**IUnknown.**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) **IWMDRMEventGenerator** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IWMDRMEventGenerator** tem esses métodos.



| Método                                                                    | Descrição                                   |
|:--------------------------------------------------------------------------|:----------------------------------------------|
| [**CancelAsyncOperation**](iwmdrmeventgenerator-cancelasyncoperation.md) | Cancela uma operação assíncrona.<br/> |
