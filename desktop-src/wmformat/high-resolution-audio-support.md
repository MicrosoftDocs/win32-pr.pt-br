# <a name="high-resolution-audio-support"></a>Suporte a áudio High-Resolution

o codec Windows media audio 9 Professional e o codec Windows media audio 9 lossless dão suporte a áudio de alta resolução. Áudio de alta resolução é definido como um formato de áudio com mais de 16 bits por amostra. O número de bits que são usados para um exemplo de áudio individual afeta a fidelidade da onda de som codificada.

como os formatos de som codificados com PCM foram usados em sistemas operacionais Windows por muito tempo, algumas das funções de Windows são antigas e não conseguem processar áudio PCM de alta resolução. Como resultado, os formatos PCM de alta resolução são representados usando estruturas diferentes de áudio padrão. Para obter mais informações, consulte [trabalhando com High-Resolution áudio PCM](working-with-high-resolution-pcm-audio.md).
