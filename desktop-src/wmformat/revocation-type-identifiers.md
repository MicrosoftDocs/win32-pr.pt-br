# <a name="revocation-type-identifiers"></a>Identificadores de tipo de revogação

As constantes listadas na tabela a seguir definem valores guid que identificam diferentes tipos de revogação.



| Constante                         | Descrição                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| APLICATIVO DE TIPO \_ DE REVOGAÇÃO \_ WMDRM \_     | Identifica informações de revogação referentes a certificados de aplicativo.                           |
| DISPOSITIVO DE TIPO \_ DE REVOGAÇÃO \_ WMDRM \_  | Identifica informações de revogação referentes a certificados de dispositivo.                                |
| WMDRM \_ REVOCATION \_ TYPE \_ CARDEA  | Identifica informações de revogação referentes a certificados Windows DRM de Mídia para Dispositivos de Rede. |
| REVINFO DO TIPO DE \_ \_ \_ REVOGAÇÃO WMDRM | Não documentado nesta versão.                                                                     |
| TIPO DE REVOGAÇÃO WMDRM \_ \_ \_ GRL     | Não documentado nesta versão.                                                                     |
| TIPO DE REVOGAÇÃO WMDRM \_ \_ \_ HDCP    | Não documentado nesta versão.                                                                     |
