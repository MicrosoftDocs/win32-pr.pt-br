# <a name="revocation-type-identifiers"></a>Identificadores de tipo de revogação

As constantes listadas na tabela a seguir definem valores GUID que identificam diferentes tipos de revogação.



| Constante                         | Descrição                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| \_aplicativo de tipo de revogação WMDRM \_ \_     | Identifica informações de revogação pertencentes a certificados de aplicativo.                           |
| \_dispositivo de tipo de revogação WMDRM \_ \_  | Identifica informações de revogação pertencentes a certificados de dispositivo.                                |
| \_tipo de revogação do WMDRM \_ \_ CARDEA  | Identifica as informações de revogação pertencentes ao Windows Media DRM para certificados de dispositivos de rede. |
| \_tipo de revogação do WMDRM \_ \_ REVINFO | Não documentado nesta versão.                                                                     |
| \_tipo de revogação do WMDRM \_ \_     | Não documentado nesta versão.                                                                     |
| tipo de REVOGAção de WMDRM de \_ \_ \_ HDCP    | Não documentado nesta versão.                                                                     |
