# <a name="drm-import"></a>Importação de DRM

as seções a seguir explicam como converter o conteúdo de mídia de um CPS (sistema de proteção de conteúdo) de terceiros para Windows DRM de mídia. Esse processo é conhecido como importação de DRM e consiste basicamente nas seguintes etapas:

1.  Criando o arquivo ASF.
2.  Criando a licença.
3.  Opcionalmente, excluindo a licença.

A importação de DRM é explicada nas seções a seguir.



| Seção                                                                              | Descrição                                                                          |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [Importando a licença e o material da chave](importing-license-and-key-material.md)         | descreve como emitir licenças localmente e importá-las para Windows Media DRM.      |
| [Verificando revogação de certificado](checking-certificate-revocation.md)               | Descreve como verificar a revogação de certificado.                                       |
| [Criando uma licença do XMR](building-an-xmr-license.md)                               | Descreve como criar uma licença do XMR e passá-la para o subsistema DRM.             |
| [Criando e inicializando um gravador de DRM](creating-and-initializing-a-drm-writer.md) | Descreve como inicializar um objeto do gravador ASF para se preparar para a importação de DRM.          |
| [Criptografando e importando amostras de mídia](encrypting-and-importing-media-samples.md) | descreve como criptografar e importar amostras de mídia individuais em Windows media DRM. |
| [Exclusão de licenças](license-deletion.md)                                             | Descreve como excluir licenças criadas localmente.                                    |
