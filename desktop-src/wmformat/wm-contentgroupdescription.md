# <a name="wmcontentgroupdescription"></a>WM/ContentGroupDescription

O atributo **WM/ContentGroupDescription** contém uma descrição do grupo de conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMContentGroupDescription

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

Os grupos de conteúdo são coleções de mídias às quais os arquivos individuais podem pertencer. Por exemplo, um arquivo de áudio pode ser uma música que é uma faixa em um CD que faz parte de um conjunto em caixa. Da mesma forma, um videoclipe pode fazer parte de uma série.
