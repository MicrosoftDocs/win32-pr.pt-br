# <a name="drm_actionallowed_playback"></a>Ação do \_ DRMProduções \_ permitidos

O **atributo \_ ActionAllowed \_ Playback** do DRM indica se a reprodução do conteúdo é permitida.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ ActionAllowed \_ Playback

## <a name="data-type"></a>Tipo de Dados

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Comentários

Essa é uma propriedade somente leitura que é recuperada usando [**IWMDRMReader::GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
