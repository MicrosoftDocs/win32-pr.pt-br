# <a name="wmlanguage"></a>WM/linguagem

O atributo **WM/Language** contém o idioma do fluxo.

## <a name="global-constant"></a>Constante global

g \_ wszWMLanguage

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**

## <a name="remarks"></a>Comentários

A cadeia de caracteres de linguagem usada deve ser compatível com RFC 1766. Para obter mais informações, consulte [cadeias de caracteres de linguagem](language-strings.md).

### <a name="example"></a>Exemplo



| Tipo de arquivo | Valor de exemplo |
|-----------|---------------|
| Todos       | "en-US"       |
