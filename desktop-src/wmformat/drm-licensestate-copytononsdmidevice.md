# <a name="drm_licensestate_copytononsdmidevice"></a>CopyToNonSDMIDevice de licença do DRM \_ \_

A **propriedade \_ \_ CopyToNonSDMIDevice de licença do DRM** contém uma estrutura de [**\_ \_ \_ dados de estado da licença do WM**](/previous-versions/windows/desktop/legacy/dd757942(v=vs.85)) que contém detalhes sobre como esse direito foi aplicado ao conteúdo.
