# <a name="wmmediaissap"></a>WM/MediaIsSAP

O atributo **WM/MediaIsSAP** indica se o conteúdo inclui um segundo programa de áudio (SAP).

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsSAP

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ bool**

## <a name="remarks"></a>Comentários

Esse atributo é usado para difusão de televisão.

O SAP geralmente identifica programas de televisão que incluem uma trilha sonora alternativa em um idioma diferente.
