# <a name="configuring-screen-capture-streams"></a>Configurando a captura de tela Fluxos

Fluxos que usam o codec Windows Media® Vídeo 9 são configurados por aplicativos da mesma maneira que os fluxos de vídeo normais. No entanto, se você definir o nível de complexidade do vídeo como 0, o autor ignorará qualquer valor de qualidade de vídeo definido com [**IWMVideoMediaProps::SetQuality**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmvideomediaprops-setquality). Para obter mais informações, consulte [Getting Good Results with the Windows Media Video 9 Screen Codec](getting-good-results-with-the-windows-media-video-9-screen-codec.md).
