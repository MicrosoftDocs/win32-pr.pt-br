# <a name="bannerimagedata"></a>BannerImageData

O atributo **BannerImageData** contém uma imagem de faixa para fins de identidade visual. O tipo de imagem que esse atributo contém é descrito pelo atributo [**BannerImageType**](bannerimagetype.md) .

## <a name="global-constant"></a>Constante global

g \_ wszWMBannerImageData

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ binário**

## <a name="remarks"></a>Comentários

A imagem armazenada nesse atributo pode estar no formato bitmap, JPEG ou GIF.
