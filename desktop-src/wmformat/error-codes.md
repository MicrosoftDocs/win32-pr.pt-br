# <a name="error-codes-windows-media-format-11-sdk"></a>Códigos de erro (SDK do Windows Media Format 11)

A Microsoft definiu os seguintes códigos de erro e êxito para uso no Windows Media Format SDK. Esses códigos são um subconjunto de uma coleção maior que pode ser exibida nos arquivos de cabeçalho modelers. h (códigos ASF) e NSError. h (outros códigos) incluídos no Windows Media Format SDK. Códigos de erro comuns adicionais podem ser exibidos em Winerror. h.
