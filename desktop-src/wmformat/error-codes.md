# <a name="error-codes-windows-media-format-11-sdk"></a>Códigos de erro (Windows SDK de formato de mídia 11)

A Microsoft definiu os seguintes códigos de erro e êxito para uso no SDK Windows Formato de Mídia. Esses códigos são um subconjunto de uma coleção maior que pode ser exibido nos arquivos de header asferr.h (códigos ASF) e nserror.h (outros códigos) incluídos no SDK de Formato de Mídia Windows. Códigos de erro comuns adicionais podem ser exibidos em Winerror.h.
