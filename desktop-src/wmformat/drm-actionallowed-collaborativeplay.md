# <a name="drm_actionallowed_collaborativeplay"></a>\_CollaborativePlay ACTIONALLOWED \_ DRM

O atributo **DRM \_ ActionAllowed \_ CollaborativePlay** indica se a licença permite que o conteúdo seja reproduzido de forma colaborativa.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ ActionAllowed \_ CollaborativePlay

## <a name="data-type"></a>Tipo de Dados

**WMT \_ tipo \_ bool**

## <a name="remarks"></a>Comentários

O direito de diversão de colaboração aplica-se à reprodução de conteúdo protegido como parte de uma sessão colaborativa usando serviços ponto a ponto. Por exemplo, os usuários do MSN Messenger 2004 podem reproduzir conteúdo protegido em uma sessão MusicMix. Esse direito só pode ser usado para contribuir com um arquivo protegido para uma única sessão ao mesmo tempo, e todos os usuários que participam da sessão devem estar online para renderizar o conteúdo.

Esta é uma propriedade somente leitura que é recuperada usando [**IWMDRMReader:: GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
