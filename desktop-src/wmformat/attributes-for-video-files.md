# <a name="attributes-for-video-files"></a>Atributos para arquivos de vídeo

Esta seção lista os atributos comumente usados para arquivos de vídeo. É recomendável que você defina atributos para arquivos de acordo com essas listas para garantir que seus arquivos sejam totalmente compatíveis com uma ampla variedade de aplicativos de reprodução. Os atributos nesta seção são listados em três categorias: primário, secundário e terciário.

Os atributos primários transmitem as informações mais básicas sobre um arquivo. Se você estiver criando arquivos de vídeo para distribuição, esse será o conjunto mínimo de atributos que você deve usar.

Os atributos secundários contêm metadados comuns que são importantes, mas não são universais para todos os arquivos de vídeo.

Os atributos terciários devem ser incluídos conforme necessário, mas não são essenciais para descrever o arquivo.

-   [**WM/ContentDistributor**](wm-contentdistributor.md)

-   [**WM/MediaClassPrimaryID**](wm-mediaprimaryid.md)
