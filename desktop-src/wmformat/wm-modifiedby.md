# <a name="wmmodifiedby"></a>WM/ModifiedBy

O atributo **WM/ModifiedBy** contém o nome de uma pessoa ou grupo que modificou o arquivo.

## <a name="global-constant"></a>Constante global

g \_ wszWMModifiedBy
