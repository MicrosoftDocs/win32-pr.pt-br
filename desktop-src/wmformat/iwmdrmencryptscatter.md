# <a name="iwmdrmencryptscatter-interface"></a>Interface IWMDRMEncryptScatter

A interface **IWMDRMEncryptScatter** desembaralha e criptografa os dados.

A interface **IWMDRMEncryptScatter** herda da interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMDRMEncryptScatter** também tem estes tipos de membros:

-   [Métodos](#methods)

|:----------------------------------------------------------------------|:-----------------------------------------------------------------------|
