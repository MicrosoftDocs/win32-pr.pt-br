# <a name="wmpromotionurl"></a>WM/PromotionURL

O atributo **WM/PromotionURL** contém o endereço de um site da Web que oferece uma promoção relacionada ao conteúdo.

## <a name="global-constant"></a>Constante global

g \_ wszWMPromotionURL

## <a name="data-type"></a>Tipo de Dados

**Cadeia de caracteres do \_ tipo WMT \_**
