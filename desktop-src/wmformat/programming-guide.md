# <a name="windows-media-format-sdk-programming-guide"></a>Guia de programação do Windows Media Format SDK

O guia de programação destina-se a ajudá-lo a aprender a usar o SDK (Software Development Kit) do Microsoft Windows Media Format para criar aplicativos que funcionam com arquivos usando o ASF (Advanced Systems Format).

Você pode usar o Windows Media Format SDK para criar aplicativos que gravam fluxos de mídia digital e ler fluxos de arquivos ASF. Esse SDK também dá suporte à edição de metadados em arquivos ASF. Além disso, esse SDK pode ser usado para ler e manipular metadados em arquivos MP3.

As seções a seguir explicam detalhadamente as etapas necessárias para gravar, ler e editar arquivos ASF usando esse SDK.
