# <a name="iwmsecurebuffer-interface"></a>Interface IWMSecureBuffer

A interface **IWMSecureBuffer** criptografa e descriptografa os ponteiros de dados.

## <a name="members"></a>Membros

A interface **IWMSecureBuffer** herda da interface [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IWMSecureBuffer** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IWMSecureBuffer** tem esses métodos.



| Método                                     | Descrição                         |
|:-------------------------------------------|:------------------------------------|
| [**Descriptografar**](iwmsecurebuffer-decrypt.md) | Descriptografa um ponteiro de dados.<br/> |
| [**Encrypt**](iwmsecurebuffer-encrypt.md) | Criptografa um ponteiro de dados.<br/> |
