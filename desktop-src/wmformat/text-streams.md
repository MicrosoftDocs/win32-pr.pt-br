# <a name="text-streams"></a>Fluxos de texto

Fluxos de texto contêm texto simples armazenado em exemplos. Seu aplicativo pode usar texto de um fluxo de texto de qualquer maneira que você escolher. Por exemplo, talvez você queira criar um aplicativo de tutorial de idioma que apresente palavras em um idioma estrangeiro. Você pode incluir cada palavra como um exemplo de fluxo de texto no momento da apresentação que corresponde à palavra falada em um fluxo de áudio. Em seguida, seu aplicativo pode renderizar o texto quando o exemplo é entregue.
