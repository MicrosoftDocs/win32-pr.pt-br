# <a name="windows-media-drm-10-for-network-devices"></a>Windows Media DRM 10 para dispositivos de rede

Seu aplicativo pode transmitir conteúdo protegido para receptores de mídia digital em uma rede usando o Windows Media DRM 10 para dispositivos de rede. Esse recurso é um protocolo para comunicação segura com dispositivos de rede.

O SDK do Windows Media Format dá suporte ao Windows Media DRM 10 para dispositivos de rede, mantendo um banco de dados de dispositivos de rede, validando a proximidade da rede desses dispositivos e convertendo o conteúdo protegido por DRM em um formato de transporte protegido.

> [!Note]  
> O DRM não é compatível com a versão baseada em x64 deste SDK.
