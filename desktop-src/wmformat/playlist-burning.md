# <a name="playlist-burning"></a>Gravação de playlist

As licenças do Windows Media DRM 10 podem dar suporte à cópia de listas de reprodução de arquivos protegidos para CD de áudio Red Book. Os emissores de licença podem limitar o número de vezes que o conteúdo pode ser copiado para o CD em uma lista de reprodução por um número total de vezes e pelo número de vezes que uma lista de reprodução específica pode ser copiada. Para obter mais informações sobre direitos de gravação de playlist em licenças, consulte a documentação do SDK do Windows Media Rights Manager 10.

O Windows Media Format SDK fornece a capacidade de verificar rapidamente todos os arquivos em uma lista de reprodução para garantir que eles sejam licenciados para a gravação da playlist.

**Observação** O DRM não é compatível com a versão baseada em x64 deste SDK.
