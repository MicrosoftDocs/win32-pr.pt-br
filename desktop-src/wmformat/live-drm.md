# <a name="live-drm"></a>DRM ao vivo

a partir do Windows media 9 Series, o SDK do formato de mídia do Windows pode ser usado para gravar arquivos protegidos por DRM. Isso permite cenários como o DRM ao vivo, no qual a proteção de DRM é aplicada a um arquivo ou fluxo à medida que ele está sendo codificado, em vez de como uma etapa de pós-processamento após a conclusão da codificação. O DRM ao vivo permite que os proprietários de conteúdo protejam transmissões ao vivo, como concertos de pagamento por exibição ou eventos esportivos. O DRM ao vivo também representa uma maneira simplificada ou simplificada de criar arquivos protegidos. Você pode usar o DRM ao vivo para aplicar qualquer versão da proteção de DRM. Um arquivo protegido criado usando o DRM ao vivo é exatamente o mesmo que um arquivo protegido como uma etapa de pós-processamento.

**Observação** O DRM não é compatível com a versão baseada em x64 deste SDK.
