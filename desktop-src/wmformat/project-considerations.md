# <a name="project-considerations"></a>Considerações sobre o projeto

Você deve garantir que o usuário final possa executar corretamente os aplicativos criados com o Windows Media Format SDK. Esta seção descreve duas coisas que você deve considerar ao distribuir aplicativos, extensões de nome de arquivo e redistribuição de software.

Você deve usar as extensões de nome de arquivo corretas para todos os arquivos criados com seus aplicativos. Usar extensões de nome de arquivo apropriadas torna mais fácil para os usuários finais reconhecer arquivos ASF e evitar confusão ao decodificar arquivos.

Você deve fornecer os pacotes redistribuíveis necessários para executar seus aplicativos. Sem os redistribuíveis corretos, os usuários não poderão usar seus aplicativos.

As seções a seguir descrevem as extensões de nome de arquivo e a redistribuição de software em detalhes.

| [Redistribuição de software](software-redistribution.md)                       | Descreve os redistribuíveis do Windows Media Format SDK e como incluí-los com seus aplicativos. |
