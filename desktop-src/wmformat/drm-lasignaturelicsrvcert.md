# <a name="drm_lasignaturelicsrvcert"></a>DRM \_ LASignatureLicSrvCert

O **atributo DRM \_ LASignatureLicSrvCert** contém o certificado que verifica o certificado contido em [**DRM \_ LASignatureCert.**](drm-lasignaturecert.md)

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ LASignatureLicSrvCert

## <a name="data-type"></a>Tipo de Dados

**CADEIA DE CARACTERES DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentários

Essa propriedade pode ser definida com o [**método IWMDRMWriter::SetDRMAttribute.**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) Ele não está acessível ao objeto de leitor.
