# <a name="the-application-configuration-file-acf"></a>O arquivo de configuração de aplicativo (ACF)

O arquivo de configuração de aplicativo (ACF) tem duas partes: um [cabeçalho de interface](the-acf-header.md), semelhante ao cabeçalho da interface no arquivo IDL e um [corpo](the-acf-body.md), que contém atributos de configuração que se aplicam a tipos e funções definidos no corpo da interface do arquivo IDL. Eles são discutidos nos seguintes tópicos:

-   [O cabeçalho ACF](the-acf-header.md)
-   [O corpo de ACF](the-acf-body.md)
