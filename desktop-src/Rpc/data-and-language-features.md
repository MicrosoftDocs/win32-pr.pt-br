# <a name="data-and-language-features"></a>Recursos de dados e linguagem

O linguagem IDL da Microsoft (MIDL) fornece o conjunto de recursos que estendem a linguagem de programação C para dar suporte a chamadas de procedimento remoto. MIDL não é uma variação de C; é uma linguagem formal fortemente tipada por meio da qual você pode controlar os dados transmitidos por uma rede. O MIDL foi projetado para que os desenvolvedores familiarizados com o C possam aprender rapidamente.

Este tópico aborda três recursos de linguagem:

-   [Tipagem forte](strong-typing.md)
-   [Atributos direcional (parâmetro)](directional-parameter-attributes.md)
-   [Representação de dados](data-representation.md)
