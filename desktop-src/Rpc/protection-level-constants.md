# <a name="protection-level-constants"></a>Constantes no nível de proteção

A especificação RPC osF/DCE renomeou as constantes de nível de autenticação. Agora, eles são chamados de constantes de nível de proteção. O Microsoft RPC dá suporte a nomes novos e antigos. Para obter mais informações, consulte [Constantes de nível de autenticação](authentication-level-constants.md).
