# <a name="protection-level-constants"></a>Constantes de nível de proteção

A especificação de RPC uso/DCE renomeou as constantes de nível de autenticação. Agora, eles são chamados de constantes de nível de proteção. O Microsoft RPC dá suporte aos nomes antigo e novo. Para obter mais informações, consulte [constantes de nível de autenticação](authentication-level-constants.md).
