# <a name="authentication-services"></a>Serviços de Autenticação

A interface de provedor de suporte de segurança (SSPI) fornece os serviços de autenticação subjacentes para RPC. Portanto, quando o aplicativo especifica um serviço de autenticação, ele seleciona um SSP. Para obter uma lista dos SSPs que o SPPI dá suporte atualmente, consulte [constantes de serviço de autenticação](authentication-service-constants.md). Para obter mais informações, consulte [interface do provedor de suporte de segurança (SSPI)](security-support-provider-interface-sspi-.md).

> [!Note]  
> Os termos provedor de segurança e serviço de autenticação são intercambiáveis.
