# <a name="authentication-services"></a>Serviços de Autenticação

A SSPI (Interface do Provedor de Suporte de Segurança) fornece os serviços de autenticação subjacentes para RPC. Portanto, quando seu aplicativo especifica um serviço de autenticação, ele seleciona um SSP. Para ver uma lista dos SSPs compatíveis com o SPPI no momento, consulte Constantes de serviço [de autenticação.](authentication-service-constants.md) Para obter mais informações, consulte [SSPI (Interface](security-support-provider-interface-sspi-.md)do Provedor de Suporte de Segurança).

> [!Note]  
> Os termos Provedor de Segurança e Serviço de Autenticação são intercambiáveis.
