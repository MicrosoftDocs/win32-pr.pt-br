# <a name="i-rpc"></a>I (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H i J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_idempotent_glos"></span><span id="_RPC_IDEMPOTENT_GLOS"></span>**idempotente**
</dt> <dd>

Chamada de procedimento remoto que não altera um estado e retorna as mesmas informações cada vez que é chamada com os mesmos parâmetros de entrada.

</dd> <dt>

<span id="_rpc_input_parameter_glos"></span><span id="_RPC_INPUT_PARAMETER_GLOS"></span>**parâmetro de entrada**
</dt> <dd>

Parâmetro que contém dados que o aplicativo cliente está transmitindo para o aplicativo de servidor. O atributo in designa um parâmetro **de** entrada.

</dd> <dt>

<span id="_rpc_idl_glos"></span><span id="_RPC_IDL_GLOS"></span>**Linguagem de definição de interface (IDL)**
</dt> <dd>

USO-idioma padrão da DCE para especificar a interface para chamadas de procedimento remoto. Consulte também [*MIDL*](m-glos.md).

</dd> <dt>

<span id="_rpc_ip_glos"></span><span id="_RPC_IP_GLOS"></span>**Protocolo IP**
</dt> <dd>

Protocolo de comunicação sem conexão de camada de rede. Consulte também [*Datagram*](d-glos.md).

</dd> <dt>

<span id="_rpc_ipx_glos"></span><span id="_RPC_IPX_GLOS"></span>**IPX (Internet Packet Exchange)**
</dt> <dd>

Protocolo de comunicação Novell NetWare que usa soquetes de [*datagrama*](d-glos.md) para rotear pacotes de informações em redes locais e redes de longa distância.

</dd> <dt>

<span id="_rpc_intranet_glos"></span><span id="_RPC_INTRANET_GLOS"></span>**intranet**
</dt> <dd>

Rede privada que usa produtos e tecnologias da Internet (por exemplo, servidores Web), mas não está disponível para usuários externos da Internet.

</dd> </dl>
