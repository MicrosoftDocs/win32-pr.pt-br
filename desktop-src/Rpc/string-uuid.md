# <a name="string-uuid"></a>UUID de cadeia de caracteres

Um UUID de cadeia de caracteres contém a representação de matriz de caracteres de um UUID. Um UUID de cadeia de caracteres é composto por vários campos de caracteres hexadecimais. Cada membro tem um comprimento fixo e os campos são separados pelo caractere de hífen. Por exemplo:

`989C6E5C-2CC1-11CA-A044-08002B1BB4F5`

Ao fornecer um UUID de cadeia de caracteres como um parâmetro de entrada para uma função de tempo de execução RPC, insira os caracteres hexadecimais alfabéticos como caracteres maiúsculos ou minúsculos. No entanto, quando uma função de tempo de execução retorna um UUID de cadeia de caracteres, os caracteres hexadecimais sempre são minúsculos.
