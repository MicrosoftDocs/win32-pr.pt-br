# <a name="rpc_authz_handle"></a>RPC \_ AUTHZ \_ HANDLE

O **tipo de dados HANDLE \_ AUTHZ \_ RPC** declara um identificador de autorização. Esse handle aponta para as informações de privilégios para o aplicativo cliente que fez a chamada de procedimento remoto.


```C++
typedef void* RPC_AUTHZ_HANDLE;
```
