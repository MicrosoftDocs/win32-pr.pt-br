# <a name="rpc_authz_handle"></a>identificador do RPC \_ AUTHZ \_

O tipo de dados de **\_ \_ identificador de RPC AUTHZ** declara um identificador de autorização. Esse identificador aponta para as informações de privilégios do aplicativo cliente que fez a chamada de procedimento remoto.


```C++
typedef void* RPC_AUTHZ_HANDLE;
```
