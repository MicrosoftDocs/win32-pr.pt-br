# <a name="u-rpc"></a>U (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H [i](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) U [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_udp_glos"></span><span id="_RPC_UDP_GLOS"></span>**UDP (User Datagram Protocol)**
</dt> <dd>

O transporte de rede que usa soquetes de [*datagrama*](d-glos.md) sem conexão e é colocado em camadas sobre o IP (protocolo de Internet [*)*](i-glos.md).

</dd> <dt>

<span id="_rpc_unbind_glos"></span><span id="_RPC_UNBIND_GLOS"></span>**desassociar**
</dt> <dd>

Processo de encerrar a conexão lógica entre um cliente e um servidor.

</dd> <dt>

<span id="_rpc_unique_pointer_glos"></span><span id="_RPC_UNIQUE_POINTER_GLOS"></span>**ponteiro exclusivo**
</dt> <dd>

Ponteiro que pode ser nulo ou apontar para dados existentes, cujo valor pode ser alterado durante uma chamada de procedimento remoto. Um ponteiro exclusivo não pode receber um alias. O \[ atributo [Unique](/windows/desktop/Midl/unique) \] designa um ponteiro exclusivo. Consulte também [*ponteiro completo*](f-glos.md), [*ponteiro de referência*](r-glos.md).

</dd> <dt>

<span id="_rpc_uuid_glos"></span><span id="_RPC_UUID_GLOS"></span>**UUID (identificador exclusivo universal)**
</dt> <dd>

(ou [**GUID**](/windows/win32/api/guiddef/ns-guiddef-guid)) 128-valor de bits usado na comunicação entre processos para identificar entidades como interfaces de cliente e de servidor, vetores de ponto de entrada de Gerenciador e objetos RPC. Consulte também [Uuidgen](/windows).

</dd> <dt>

<span id="_rpc_unmarshaling_glos"></span><span id="_RPC_UNMARSHALING_GLOS"></span>**desempacotamento**
</dt> <dd>

Processo de desempacotamento de parâmetros que foram enviados entre limites de processo.

</dd> <dt>

<span id="_rpc_uuidgen_glos"></span><span id="_RPC_UUIDGEN_GLOS"></span>**uuidgen**
</dt> <dd>

Programa utilitário, fornecido com o SDK (Software Development Kit) da plataforma, que usa um valor de tempo e a ID de placa de rede do seu computador para gerar [UUIDs](/windows) que são garantidos como exclusivos.

</dd> </dl>
