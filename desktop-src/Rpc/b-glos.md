<dl> <dt>

<span id="_rpc_bind_glos"></span><span id="_RPC_BIND_GLOS"></span>**associa**
</dt> <dd>

Processo pelo qual uma conexão lógica é estabelecida entre um cliente e um servidor para direcionar uma chamada de procedimento remoto para esse servidor.

</dd> <dt>

<span id="_rpc_binding_glos"></span><span id="_RPC_BINDING_GLOS"></span>**vinculação**
</dt> <dd>

Relação entre um cliente e um servidor estabelecido em uma sequência de protocolo específica para um sistema host e ponto de extremidade específicos. Também é usado como uma forma abreviada de [identificador de ligação](/windows).

</dd> <dt>

<span id="_rpc_binding_handle_glos"></span><span id="_RPC_BINDING_HANDLE_GLOS"></span>**identificador de associação**
</dt> <dd>

Estrutura de dados que representa a conexão lógica entre um cliente e um servidor.

</dd> <dt>

<span id="_rpc_binding_vector_glos"></span><span id="_RPC_BINDING_VECTOR_GLOS"></span>**vetor de associação**
</dt> <dd>

Estrutura de dados RPC que contém uma lista de [identificadores de associação](/windows) sobre os quais um aplicativo de servidor pode receber chamadas de procedimento remoto.

</dd> </dl>
