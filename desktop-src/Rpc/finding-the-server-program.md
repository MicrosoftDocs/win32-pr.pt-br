# <a name="finding-the-server-program"></a>Localizando o programa do servidor

Depois que o tempo de execução RPC do lado do cliente se conecta a um sistema host do servidor solicitado no identificador de associação, a biblioteca de tempo de execução RPC do cliente encontra o processo do servidor. Para fazer isso, ele consulta o mapa do ponto de extremidade no sistema host do servidor. O mapa do ponto de extremidade contém informações sobre o ponto de extremidade que o servidor está ouvindo.
