# <a name="using-more-than-one-context-handle-per-call"></a>Usando mais de um identificador de contexto por chamada

A capacidade de usar matrizes de identificadores de contexto como parâmetros é disponibilizada no Windows Vista e em sistemas operacionais posteriores. No entanto, esse recurso deve ser usado com cuidado dentro de um aplicativo. Por exemplo, em uma situação em que um identificador de contexto está sendo usado em várias chamadas, ele se torna cada vez mais difícil de controlar seu uso.
