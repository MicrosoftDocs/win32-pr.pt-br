# <a name="using-more-than-one-context-handle-per-call"></a>Usando mais de um alça de contexto por chamada

A capacidade de usar matrizes de alças de contexto como parâmetros é disponibilizada no Windows Vista e sistemas operacionais posteriores. No entanto, esse recurso deve ser usado cuidadosamente dentro de um aplicativo. Por exemplo, em uma situação em que um alça de contexto está sendo usado em várias chamadas, torna-se cada vez mais difícil controlar seu uso.
