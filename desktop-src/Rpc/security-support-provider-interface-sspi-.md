# <a name="security-support-provider-interface-sspi"></a>SSPI (Interface do Provedor de Suporte de Segurança)

Em conjunto com seus sistemas operacionais, a Microsoft oferece a SSPI (Interface do Provedor de Suporte de Segurança). O SSPI fornece uma interface universal e padrão do setor para aplicativos distribuídos seguros.
