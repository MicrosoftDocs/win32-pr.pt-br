# <a name="security-support-provider-interface-sspi"></a>SSPI (interface do provedor de suporte de segurança)

Em conjunto com seus sistemas operacionais, a Microsoft oferece a interface de provedor de suporte de segurança (SSPI). O SSPI fornece uma interface padrão do setor universal para aplicativos distribuídos seguros.

Esta seção descreve o SSPI nos seguintes tópicos:

-   [Gravando um cliente SSPI autenticado](writing-an-authenticated-sspi-client.md)
-   [Gravando um servidor SSPI autenticado](writing-an-authenticated-sspi-server.md)
