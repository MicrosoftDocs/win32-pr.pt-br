# <a name="client-context-reset"></a>Redefinição de contexto de cliente

Quando o servidor fica indisponível, o aplicativo cliente pode liberar seus dados de contexto chamando a função RPC [**RpcSsDestroyClientContext**](/windows/desktop/api/Rpcndr/nf-rpcndr-rpcssdestroyclientcontext).
