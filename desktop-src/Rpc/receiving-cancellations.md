# <a name="receiving-cancellations"></a>Recebendo cancelamentos

O aplicativo de servidor pode chamar [**RpcServerTestCancel**](/windows/desktop/api/Rpcdce/nf-rpcdce-rpcservertestcancel) com o alçamento de associação da chamada em questão para descobrir se o cliente solicitou o cancelamento da chamada assíncrona. Ele retornará RPC \_ S OK se o cliente tiver cancelado a \_ chamada.
