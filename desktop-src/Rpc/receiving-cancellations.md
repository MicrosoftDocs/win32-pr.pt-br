# <a name="receiving-cancellations"></a>Cancelando o recebimento

O aplicativo de servidor pode chamar [**RpcServerTestCancel**](/windows/desktop/api/Rpcdce/nf-rpcdce-rpcservertestcancel) com o identificador de associação da chamada em questão para descobrir se o cliente solicitou o cancelamento da chamada assíncrona. Ele retornará RPC \_ S \_ OK se o cliente cancelou a chamada.
