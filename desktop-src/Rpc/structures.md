# <a name="structs-rpc"></a>Structs (RPC)

A semântica C normal se aplica aos campos de tipos base. Campos de tipos mais complexos, como ponteiros, matrizes e outros tipos construídos, podem ser modificados por atributos de tipo [**ou \_ campo**](field-attributes.md). Para obter mais informações, consulte [**struct**](/windows/desktop/Midl/struct).
