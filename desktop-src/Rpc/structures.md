# <a name="structs-rpc"></a>Structs (RPC)

A semântica C normal se aplica aos campos de tipos base. Os campos de tipos mais complexos, como ponteiros, matrizes e outros tipos construídos, podem ser modificados por [**\_ atributos**](field-attributes.md)de tipo ou campo. Para obter mais informações, consulte [**struct**](/windows/desktop/Midl/struct).
