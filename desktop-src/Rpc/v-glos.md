<dl> <dt>

<span id="_rpc_varying_array_glos"></span><span id="_RPC_VARYING_ARRAY_GLOS"></span>**matriz variável**
</dt> <dd>

Matriz cujo intervalo de elementos transmitidos é determinado em tempo de executar por outro parâmetro, estrutura ou expressão. Consulte também [*matriz compatível e*](c-glos.md) [*matriz aberta.*](o-glos.md)

</dd> <dt>

<span id="_rpc_version_number_glos"></span><span id="_RPC_VERSION_NUMBER_GLOS"></span>**número de versão**
</dt> <dd>

Dois números, separados por um ponto decimal, que identificam a versão de uma interface. Para ser compatível, o número de versão principal (o número à esquerda do ponto decimal) deve ser o mesmo para cliente e servidor, e o número de versão secundária do servidor deve ser maior ou igual ao número de versão secundária do cliente.

</dd> </dl>
