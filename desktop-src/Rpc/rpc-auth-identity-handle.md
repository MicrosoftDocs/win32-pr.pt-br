# <a name="rpc_auth_identity_handle"></a>RPC \_ AUTH \_ IDENTITY \_ HANDLE

O tipo de dados **RPC \_ AUTH \_ IDENTITY \_ HANDLE** declara um identificador que aponta para uma estrutura que contém as credenciais de autenticação e autorização do cliente especificadas para chamadas de procedimento remoto.


```C++
typedef void* RPC_AUTH_IDENTITY_HANDLE;
```
