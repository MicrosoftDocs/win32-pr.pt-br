</dd> <dt>

<span id="_rpc_tunneling_glos"></span><span id="_RPC_TUNNELING_GLOS"></span>**Encapsulamento**
</dt> <dd>

Protocolo TCP/IP para transmitir dados de um aplicativo de servidor de conteúdo para um roteador de difusão.

</dd> <dt>

<span id="_rpc_type_serialization_glos"></span><span id="_RPC_TYPE_SERIALIZATION_GLOS"></span>**serialização de tipo**
</dt> <dd>

Serialização [*de dados*](s-glos.md) que usa rotinas geradas por MIDL para tamanho, codificação e decodificar objetos de um tipo especificado. O aplicativo cliente chama essas rotinas para serializar os dados. A serialização de tipo é realizada aplicando os atributos de codificação e decodificar a um único tipo de dados ou a uma \[ [](/windows/desktop/Midl/encode) \] \[ [](/windows/desktop/Midl/decode) \] interface, no arquivo ACF. Consulte também [*serialização de procedimento*](p-glos.md).

</dd> </dl>
