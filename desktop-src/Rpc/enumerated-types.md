# <a name="enumerated-types-rpc"></a>Tipos enumerados (RPC)

A Declaração [**enum**](/windows/desktop/Midl/enum) não é convertida em instruções de **\# definição** , pois é por alguns compiladores da DCE, mas é reproduzida como uma declaração de **Enumeração** de linguagem C no arquivo de cabeçalho gerado.
