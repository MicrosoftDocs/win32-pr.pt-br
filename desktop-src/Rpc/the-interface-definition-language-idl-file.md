# <a name="the-interface-definition-language-idl-file"></a>O arquivo IDL (Linguagem de Definição de Interface)

Um arquivo IDL contém uma ou mais definições de interface. Cada definição de interface é composta por um header de interface e um corpo de interface. O título da interface é demarcado por colchetes. O corpo da interface está contido entre colchetes. Isso é ilustrado na interface de exemplo a seguir:

``` syntax
[
  /*Interface attributes go here. */
]
interface INTERFACENAME
{
  /*The interface body goes here. */
}
```

Esta seção fornece uma visão geral dos componentes de uma interface. Ele é organizado nos seguintes tópicos:

-   [O header da interface IDL](the-idl-interface-header.md)
-   [O corpo da interface IDL](the-idl-interface-body.md)
