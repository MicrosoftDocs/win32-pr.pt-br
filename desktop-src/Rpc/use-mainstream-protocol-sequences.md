# <a name="use-mainstream-protocol-sequences"></a>Usar sequências de protocolo principais

Use [**ncacn \_ IP \_ TCP**](/windows/desktop/Midl/ncacn-ip-tcp), [**ncacn \_ http**](/windows/desktop/Midl/ncacn-http) ou [**ncacn \_ NP**](/windows/desktop/Midl/ncacn-np) para chamadas RPC remotas. Use [**Ncalrpc**](/windows/desktop/Midl/ncalrpc) para chamadas RPC locais. Esses protocolos são amplamente usados e recebem a mais análise e teste.
