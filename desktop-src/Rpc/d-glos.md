# <a name="d-rpc"></a>D (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) D [E](e-glos.md) [F](f-glos.md) G H [i](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_datagram_glos"></span><span id="_RPC_DATAGRAM_GLOS"></span>**datagrama**
</dt> <dd>

Protocolo de comunicação ou transporte no qual os pacotes de dados são roteados de forma independente. Eles podem seguir rotas diferentes e chegar em uma ordem diferente da qual foram enviados. [*UDP*](u-glos.md) e [*IPX*](i-glos.md) são exemplos de camada de transporte – protocolos de datagrama. Consulte também [*orientado a conexão*](c-glos.md).

</dd> <dt>

<span id="_rpc_discriminant_glos"></span><span id="_RPC_DISCRIMINANT_GLOS"></span>**discriminante**
</dt> <dd>

Variável que especifica os tipos de dados que podem ser armazenados em uma União.

</dd> <dt>

<span id="_rpc_discriminated_union_glos"></span><span id="_RPC_DISCRIMINATED_UNION_GLOS"></span>**união discriminada**
</dt> <dd>

(ou *registro de variante*) Union que inclui um discriminador como parte da estrutura de dados para que o tipo de dados válido atualmente seja transmitido junto com a União. Consulte também [*União encapsulada*](e-glos.md), [*União não encapsulada*](n-glos.md).

</dd> <dt>

<span id="_rpc_dce_glos"></span><span id="_RPC_DCE_GLOS"></span>**Ambiente de computação distribuída (DCE)**
</dt> <dd>

Especificação da Open Software Foundation para um conjunto de serviços integrados, incluindo chamadas de procedimento remoto, sistemas de arquivos distribuídos e serviços de segurança. O padrão RPC uso-DCE é a base para o Microsoft RPC.

</dd> <dt>

<span id="_rpc_dynamic_endpoint_glos"></span><span id="_RPC_DYNAMIC_ENDPOINT_GLOS"></span>**ponto de extremidade dinâmico**
</dt> <dd>

Específico de rede – endereço de servidor que é solicitado e atribuído em tempo de execução. Confira também [*ponto de extremidade bem conhecido*](w-glos.md).

</dd> <dt>

<span id="_rpc_dynamic_identity_tracking_glos"></span><span id="_RPC_DYNAMIC_IDENTITY_TRACKING_GLOS"></span>**rastreamento de identidade dinâmico**
</dt> <dd>

Especificação que a biblioteca de tempo de execução de RPC usará as credenciais do thread de chamada, em vez do identificador de associação, para autenticação sempre que o cliente chamar um procedimento remoto. Consulte também [*rastreamento de identidade estático*](s-glos.md).

</dd> </dl>
