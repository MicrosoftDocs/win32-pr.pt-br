# <a name="configuring-the-security-server"></a>Configurando o servidor de segurança

O procedimento a seguir fornece detalhes sobre como configurar o servidor de segurança para o Microsoft RPC.

**Para configurar o servidor de segurança para RPC**

1.  Inicie o painel de controle.
2.  No painel de controle, clique no ícone **redes** . A caixa de diálogo **configurações de rede** é exibida.
3.  Na lista **software de rede instalado** , selecione **configuração de RPC**. A caixa de diálogo **configuração de RPC** é exibida.
4.  Na lista **provedor de serviços de segurança** , selecione em um ou mais provedores de segurança.
5.  Clique em **OK**.
