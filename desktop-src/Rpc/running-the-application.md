# <a name="running-the-application"></a>Executando o aplicativo

Para executar o aplicativo em um único computador, abra duas janelas de console. Na primeira janela, digite

**Hellos**

e, na segunda janela, digite

**helloc**

Para experimentar diferentes sequências de protocolo, pontos de extremidade e outras opções, crie o aplicativo hello de exemplo dos arquivos de origem no diretório RPC Hello no SDK (Kit de Desenvolvimento de Software de \\ Plataforma).

> [!Note]  
> **A Restrição de Interface RPC**  pode fazer com que esse aplicativo falhe com um erro de acesso negado no Windows XP com Service Pack 2 (SP2) e versões posteriores do Windows. Para obter mais informações, consulte [RPC Interface Restriction](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA).
