# <a name="running-the-application"></a>Executando o aplicativo

Para executar o aplicativo em um único computador, abra duas janelas de console. Na primeira janela, digite

**Saudações**

e na segunda janela, digite

**helloc**

Para experimentar diferentes sequências de protocolo, pontos de extremidade e outras opções, crie o aplicativo de exemplo Hello dos arquivos de origem no \\ diretório de saudação RPC no SDK (Software Development Kit) da plataforma.

> [!Note]  
> A **restrição de interface RPC** pode fazer com que esse aplicativo falhe com um erro de **acesso negado** no Windows XP com Service Pack 2 (SP2) e versões posteriores do Windows. Para obter mais informações, consulte [restrição de interface RPC](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA).
