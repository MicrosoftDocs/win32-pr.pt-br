# <a name="arrays-and-rpc"></a>Matrizes e RPC

As linguagens de programação C e C++ fornecem, essencialmente, dois tipos de matrizes: unidimensional e multidimensional. O RPC permite que os desenvolvedores especifiquem tipos de matriz adicionais usando atributos MIDL para descrever as características das matrizes em aplicativos distribuídos.

Esta seção descreve os tipos de matrizes disponíveis em RPC. Ele também aborda os atributos MIDL que os desenvolvedores podem usar para descrever matrizes unidimensionais e multidimensionais. Este material é apresentado nos seguintes tópicos:

-   [Tipos de matrizes](kinds-of-arrays.md)
-   [Atributos de matriz](array-attributes.md)
-   [Matrizes multidimensionais](multidimensional-arrays.md)
