# <a name="network-list-manager"></a>Gerenciador de lista de rede

## <a name="purpose"></a>Finalidade

A API do Gerenciador de lista de rede permite que os aplicativos recuperem uma lista de conexões de rede disponíveis. Os aplicativos podem filtrar redes, com base em atributos e assinaturas, e escolher as redes mais adequadas para sua tarefa. A infraestrutura do Gerenciador de lista de rede notifica os aplicativos sobre as alterações no ambiente de rede, permitindo que os aplicativos atualizem dinamicamente as conexões de rede.

## <a name="developer-audience"></a>Público de desenvolvedores

O Gerenciador de lista de rede foi projetado para ser usado por programadores COM.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

A API do Gerenciador de lista de rede tem suporte a partir do Windows Vista.
