# <a name="nlm-enumerations"></a>Enumerações NLM

A seguir estão as enumerações NLM.

| Enumeração                                                                 | Descrição                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**\_custo de conexão do NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | Especifica um conjunto de níveis de custo e sinalizadores de custo com suporte nas APIs de custo do Windows 8.            |
| [**\_alteração da \_ propriedade de conexão NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | Define as alterações feitas nas propriedades de uma conexão de rede.                            |
| [**conectividade do NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | Fornece notificação sempre que os parâmetros relacionados à conectividade forem alterados.               |
| [**\_tipo de domínio NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | Especifica o tipo de domínio de uma rede.                                                    |
| [**\_rede de enumeração NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | Especifica que tipos de redes são enumeradas.                                           |
| [**\_conexão de Internet NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | Define um conjunto de sinalizadores que fornecem dados adicionais para conectividade de rede IPv4 ou IPv6. |
| [**\_classe de rede NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | Define um conjunto de sinalizadores que especificam se uma rede foi identificada.                      |
| [**\_categoria de rede NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | Especifica o tipo de categoria de uma rede.                                                  |
| [**\_alteração de \_ propriedade de rede NLM \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | Define as alterações feitas nas propriedades de uma rede.                                       |
