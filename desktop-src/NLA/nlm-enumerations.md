# <a name="nlm-enumerations"></a>Enumerações NLM

A seguir estão as enumerações NLM.

| Enumeração                                                                 | Descrição                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**CUSTO DE CONEXÃO NLM \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | Especifica um conjunto de níveis de custo e sinalizadores de custo com suporte nas APIs Windows 8 Custo.            |
| [**ALTERAÇÃO DA PROPRIEDADE DE CONEXÃO NLM \_ \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | Define as alterações feitas nas propriedades de uma conexão de rede.                            |
| [**CONECTIVIDADE \_ NLM**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | Fornece notificação sempre que os parâmetros relacionados à conectividade foram alterados.               |
| [**TIPO DE DOMÍNIO NLM \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | Especifica o tipo de domínio de uma rede.                                                    |
| [**NLM \_ ENUM \_ NETWORK**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | Especifica quais tipos de redes são enumerados.                                           |
| [**CONECTIVIDADE COM A INTERNET NLM \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | Define um conjunto de sinalizadores que fornecem dados adicionais para conectividade de rede IPv4 ou IPv6. |
| [**CLASSE DE REDE NLM \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | Define um conjunto de sinalizadores que especificam se uma rede foi identificada.                      |
| [**CATEGORIA DE REDE NLM \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | Especifica o tipo de categoria de uma rede.                                                  |
| [**ALTERAÇÃO DE PROPRIEDADE DE REDE NLM \_ \_ \_**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | Define as alterações feitas nas propriedades de uma rede.                                       |
