# <a name="nlm-structures"></a>Estruturas NLM

As estruturas a seguir dão suporte à API do Gerenciador de lista de rede:



| Estrutura                                                           | Descrição                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**informações de perfil do NLM \_ simuladas \_ \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_simulated_profile_info) | Representa as informações de perfil a serem usadas durante a simulação de custo.    |
| [**NLM \_ SOCKADDR**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_sockaddr)                               | Representa o endereço de destino IPv4/IPv6.                        |
| [**\_dados de uso do NLM \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_usage_data)                          | Representa informações que indicam o uso de dados de um plano de dados. |
| [**\_status do \_ plano de dados NLM \_**](https://www.bing.com/search?q=**NLM\_DATA\_PLAN\_STATUS**)             | Representa o status atual de um plano de dados.                        |
