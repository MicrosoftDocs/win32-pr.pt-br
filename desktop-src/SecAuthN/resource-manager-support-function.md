# <a name="resource-manager-support-function"></a>Função Resource Manager suporte do Resource Manager

A função a seguir libera a memória alocada por meio do uso do designador de comprimento SCARD AUTOALLOCATE, simplificando o uso das outras funções \_ [*do gerenciador*](../secgloss/r-gly.md) de recursos.



| Tópico                                      | Descrição                                                     |
|--------------------------------------------|-----------------------------------------------------------------|
| [**SCardFreeMemory**](/windows/desktop/api/Winscard/nf-winscard-scardfreememory) | Memória de liberação retornada por meio do uso de SCARD \_ AUTOALLOCATE. |
