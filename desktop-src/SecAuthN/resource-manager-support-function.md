# <a name="resource-manager-support-function"></a>Função de suporte do Gerenciador de recursos

A função a seguir libera a memória alocada por meio do uso do \_ designador de comprimento de AUTOalocação scartar, simplificando o uso das outras funções do [*Resource Manager*](../secgloss/r-gly.md) .



| Tópico                                      | Descrição                                                     |
|--------------------------------------------|-----------------------------------------------------------------|
| [**SCardFreeMemory**](/windows/desktop/api/Winscard/nf-winscard-scardfreememory) | Memória de liberação retornada por meio do uso de SCARTAR \_ autoallocate. |
