# <a name="administrative-functions"></a>Funções administrativas

as seguintes funções administrativas permitem que um provedor de rede adote uma ação específica da rede para exibir e manipular diretórios específicos da rede, em outras palavras, diretórios mapeados para protocolos não Windows.



| Função                                         | Descrição                                                                                                                                                   |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NPGetDirectoryType**](/windows/desktop/api/Npapi/nf-npapi-npgetdirectorytype) | Chamado pelo Gerenciador de arquivos para determinar o tipo de um diretório de rede.                                                                                          |
| [**NPDirectoryNotify**](/windows/desktop/api/Npapi/nf-npapi-npdirectorynotify)   | Chamado pelo Gerenciador de arquivos para notificar o provedor de rede sobre operações de diretório. Essa função pode ser usada para executar um comportamento especial para determinados diretórios. |
