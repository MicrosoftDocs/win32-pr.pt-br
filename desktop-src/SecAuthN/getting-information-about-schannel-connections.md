# <a name="getting-information-about-schannel-connections"></a>Obtendo informações sobre conexões Schannel

Os tópicos a seguir descrevem as funções que retornam informações sobre as credenciais de Schannel especificadas e o [*contexto de segurança*](/windows/desktop/SecGloss/s-gly) que protege uma conexão de cliente/servidor.



| Tópico                                                                                                      | Descrição                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Consultando os atributos de um contexto Schannel](querying-the-attributes-of-an-schannel-context.md)       | Descreve como obter as configurações de Schannel especificadas ao criar o contexto de segurança que protege uma conexão Schannel.<br/>                                                                                               |
| [Consultando os atributos de uma credencial Schannel](querying-the-attributes-of-an-schannel-credential.md) | Descreve como obter as configurações de Schannel especificadas ao selecionar a credencial usada para criar uma conexão. As configurações são válidas para todas as conexões (contextos de segurança) criadas usando a credencial especificada.<br/> |
