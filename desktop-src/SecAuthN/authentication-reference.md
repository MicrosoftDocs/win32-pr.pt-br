# <a name="authentication-reference"></a>Referência de autenticação

As páginas de referência de autenticação contêm descrições detalhadas das funções de autenticação da Microsoft, interfaces, objetos, estruturas e outros elementos de programação. Esses tópicos fornecem informações sobre os elementos de programação usados para estabelecer a identidade de usuários e serviços. Depois que uma identidade é estabelecida, ela pode ser usada para tentar acessar um único computador ou os recursos em uma rede de computador.

As páginas de referência são divididas nos grupos a seguir.



| Seção                                                          | Descrição                                                                          |
|------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [Constantes de autenticação](authentication-constants.md)         | Constantes usadas por elementos de programação de autenticação.<br/>                    |
| [Tipos de dados de autenticação](authentication-data-types.md)       | Tipos de dados usados por elementos de programação de autenticação.<br/>                   |
| [Enumerações de autenticação](authentication-enumerations.md)   | Enumerações usadas por elementos de programação de autenticação.<br/>                 |
| [Funções de autenticação](authentication-functions.md)         | Funções usadas com o estabelecimento e a verificação da identidade do usuário.<br/>  |
| [Interfaces de autenticação](authentication-interfaces.md)       | Interfaces usadas com o estabelecimento e a verificação da identidade do usuário.<br/> |
| [Chaves do registro de autenticação](authentication-registry-keys.md) | Chaves do registro usadas por elementos de programação de autenticação.<br/>                |
| [Valores de retorno de autenticação](authentication-return-values.md) | Valores de retorno usados por elementos de programação de autenticação.<br/>                |
| [Estruturas de autenticação](authentication-structures.md)       | Estruturas usadas com funções, interfaces e objetos de autenticação.<br/>   |
