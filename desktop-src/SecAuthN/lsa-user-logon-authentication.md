# <a name="lsa-user-logon-authentication"></a>Autenticação de logon do usuário LSA

A LSA manipula o logon e a autenticação do usuário no computador local e, se o pacote de autenticação que processa a solicitação de logon der suporte à autenticação de passagem, o LSA também poderá fazer logon de usuários em outros computadores na rede. O LSA fornece acesso a pacotes de autenticação para os SSPs (provedores de suporte de segurança). Um dos dois tipos de autenticação a seguir é executado:

-   [Autenticação interativa](interactive-authentication.md)
-   [Autenticação não interativa](noninteractive-authentication.md)
