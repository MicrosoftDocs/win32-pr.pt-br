# <a name="secure-channel"></a>Canal de Segurança

O canal seguro, também conhecido como [*Schannel*](../secgloss/s-gly.md), é um SSP ( [*provedor de suporte de segurança*](../secgloss/s-gly.md) ) que contém um conjunto de protocolos de segurança que fornecem autenticação de identidade e comunicação segura e privada por meio de criptografia.

O Schannel é usado principalmente para aplicativos de Internet que exigem comunicações HTTP (Hypertext Transfer Protocol) seguras.

-   [Credenciais do Schannel](schannel-credentials.md)
-   [Protocolo de segurança da camada de transporte](transport-layer-security-protocol.md)

-   [Códigos de erro Schannel para alertas TLS e SSL](schannel-error-codes-for-tls-and-ssl-alerts.md)
