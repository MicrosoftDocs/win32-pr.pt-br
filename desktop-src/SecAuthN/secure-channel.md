# <a name="secure-channel"></a>Canal de Segurança

O Secure Channel, também conhecido como [*Schannel,*](../secgloss/s-gly.md)é um SSP [*(provedor*](../secgloss/s-gly.md) de suporte de segurança) que contém um conjunto de protocolos de segurança que fornecem autenticação de identidade e comunicação privada segura por meio da criptografia.

O Schannel é usado principalmente para aplicativos da Internet que exigem comunicações HTTP (Protocolo de Transferência de Hipertexto) seguras.
