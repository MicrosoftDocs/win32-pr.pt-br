# <a name="querycontextattributes-general-function"></a>Função QueryContextAttributes (geral)

Permite que um aplicativo de transporte consulte um [*pacote de segurança*](../secgloss/s-gly.md) para determinados [*atributos*](../secgloss/a-gly.md#_security_attribute_gly) de um contexto de [*segurança*](../secgloss/s-gly.md).

Para obter informações sobre como usar essa função com um SSP ( [*provedor de suporte de segurança*](../secgloss/s-gly.md) ) específico, consulte os tópicos a seguir.



| Tópico                                                                            | Descrição                                                                                                                                          |
|----------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**QueryContextAttributes (CredSSP)**](querycontextattributes--credssp.md)     | Permite que um aplicativo de transporte consulte o provedor de suporte de segurança de credencial (CredSSP) para determinados atributos de um [*contexto de segurança*](../secgloss/s-gly.md).<br/> |
| [**QueryContextAttributes (resumo)**](querycontextattributes--digest.md)       | Permite que um aplicativo de transporte consulte o pacote de segurança de resumo para determinados atributos de um [*contexto de segurança*](../secgloss/s-gly.md).<br/>                        |
| [**QueryContextAttributes (Kerberos)**](querycontextattributes--kerberos.md)   | Permite que um aplicativo de transporte consulte o pacote de segurança Kerberos para determinados atributos de um [*contexto de segurança*](../secgloss/s-gly.md).<br/>                      |
| [**QueryContextAttributes (negociar)**](querycontextattributes--negotiate.md) | Permite que um aplicativo de transporte consulte o pacote de segurança Negotiate para determinados atributos de um [*contexto de segurança*](../secgloss/s-gly.md).<br/>                     |
| [**QueryContextAttributes (NTLM)**](querycontextattributes--ntlm.md)           | Permite que um aplicativo de transporte consulte o pacote de segurança NTLM para determinados atributos de um [*contexto de segurança*](../secgloss/s-gly.md).<br/>                          |
| [**QueryContextAttributes (Schannel)**](querycontextattributes--schannel.md)   | Permite que um aplicativo de transporte consulte o pacote de segurança Schannel para determinados atributos de um [*contexto de segurança*](../secgloss/s-gly.md).<br/>                      |



 

## <a name="syntax"></a>Sintaxe


```C++
SECURITY_STATUS SEC_ENTRY QueryContextAttributes(
  _In_  PCtxtHandle phContext,
  _In_  ULONG       ulAttribute,
  _Out_ PVOID       pBuffer
);
```

## <a name="return-value"></a>Valor retornado

Se a função for bem-sucedida, o valor de retorno será SEC \_ E \_ OK.

Se a função falhar, o valor de retorno será um código de erro diferentes de zero.

## <a name="remarks"></a>Comentários

A estrutura apontada pelo parâmetro *pBuffer* varia dependendo do atributo que está sendo consultado. O chamador deve alocar a própria estrutura *pBuffer,* mas o SSP aloca qualquer memória necessária para manter membros de tamanho variável da estrutura *pBuffer.* A memória alocada pelo SSP pode ser liberada chamando a [**função FreeContextBuffer.**](/windows/win32/api/sspi/nf-sspi-freecontextbuffer)

Depois que o valor CONTEXT DO CERTIFICADO REMOTO SECPKG ATTR ou SECPKG ATTR LOCAL CERT CONTEXT tiver sido lido, o membro hCertStore será definido como um lidar com um repositório de certificados que contém os \_ \_ \_ \_ \_ \_ \_ \_ certificados intermediários, se algum.  Além disso, o aplicativo é responsável por [**chamar CertFreeCertificateContext para**](/windows/win32/api/wincrypt/nf-wincrypt-certfreecertificatecontext) liberar a memória usada pelo contexto do certificado.
