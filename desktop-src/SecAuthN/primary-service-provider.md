# <a name="primary-service-provider"></a>Provedor de serviços primário

Cada [*cartão inteligente*](../secgloss/s-gly.md) conhecido pelo sistema pode ter uma interface com registrada listada como seu provedor de serviços primário. Quando você acessa um *cartão inteligente* por meio do com, esse [*provedor de serviços primário*](../secgloss/p-gly.md) fornece as interfaces de controle para o cartão. Isso permite que os serviços de cartão sejam expostos a uma ampla gama de ambientes de programação, incluindo Java e o sistema de desenvolvimento Microsoft Visual Basic.
