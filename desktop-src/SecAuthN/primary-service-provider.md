# <a name="primary-service-provider"></a>Provedor de Serviços Primários

Cada [*cartão inteligente*](../secgloss/s-gly.md) conhecido pelo sistema pode ter uma interface COM registrada listada como seu provedor de serviços primário. Quando você acessa um *cartão inteligente por* meio do COM, esse provedor de [*serviços*](../secgloss/p-gly.md) primário fornece as interfaces de controle para o cartão. Isso permite que os serviços de cartão sejam expostos a uma ampla variedade de ambientes de programação, incluindo Java e o sistema de Visual Basic microsoft.
