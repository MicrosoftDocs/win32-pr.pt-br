# <a name="authentication-data-types"></a>Tipos de dados de autenticação

## <a name="lsa-data-types"></a>Tipos de dados LSA

A [*autoridade de segurança local*](/windows/desktop/SecGloss/l-gly) (LSA) usa o seguinte tipo de dados.



| Tipo de dados                                            | Descrição                                                                                                              |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_solicitação de cliente PLSA \_**](plsa-client-request.md) | Um tipo de dados opaco usado internamente pela LSA para manter as informações do cliente relacionadas a solicitações individuais.<br/> |



 

## <a name="sspi-data-types-and-values"></a>Tipos de dados e valores SSPI

O [SSPI](sspi.md) usa os seguintes dados e tipos enumerados.



| Tipo de dados ou valor                         | Descrição                                                                      |
|--------------------------------------------|----------------------------------------------------------------------------------|
| [Identificadores SSPI](sspi-handles.md)           | SSPI identifica tipos e ponteiros para esses identificadores.<br/>                      |
| [Códigos de status SSPI](sspi-status-codes.md) | Códigos de status retornados em aplicativos SSPI.<br/>                           |
| [Estampa](timestamp.md)                 | Tipo definido para manter informações sobre a validade do tempo de tokens de segurança.<br/> |
