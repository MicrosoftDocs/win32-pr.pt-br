# <a name="user-name-functions"></a>Funções de nome de usuário

As funções de nome de usuário incluem uma única função, [**NPGetUser**](/windows/desktop/api/Npapi/nf-npapi-npgetuser). Essa função recupera o nome de usuário padrão atual ou o nome de usuário usado para estabelecer a conexão de rede.
