# <a name="lsa-authentication"></a>Autenticação LSA

Autenticação LSA descreve as partes da [*autoridade de segurança local*](../secgloss/l-gly.md) (LSA) que os aplicativos podem usar para autenticar e fazer logon de usuários no sistema local. Ele também descreve como criar e chamar pacotes de autenticação e pacotes de segurança.

As funções de autenticação LSA permitem que você escreva um pacote de autenticação, um pacote de subautenticação ou um provedor de suporte de segurança combinado/pacote de autenticação (SSP/AP).

Os tópicos a seguir fornecem mais informações sobre a autenticação LSA:

-   [Modelo de autenticação LSA](lsa-authentication-model.md)
-   [Sessões de logon LSA](lsa-logon-sessions.md)
-   [Autenticação de logon do usuário LSA](lsa-user-logon-authentication.md)
-   [Pacotes de autenticação](authentication-packages.md)

Para obter informações sobre as funções de autenticação de LSA, consulte [referência de autenticação](authentication-reference.md).

Para obter informações sobre as tecnologias relacionadas, consulte [política de LSA](../secmgmt/lsa-policy.md) e [Winlogon e provedores de credenciais](winlogon-and-credential-providers.md).
