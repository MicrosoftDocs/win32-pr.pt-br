# <a name="microsoft-digest-ssp"></a>Microsoft Digest SSP

O Microsoft Digest é um SSP ( [*provedor de suporte de segurança*](../secgloss/s-gly.md) ) que implementa o protocolo de acesso Digest, um protocolo de autenticação leve para partes envolvidas em comunicações baseadas no protocolo http ou na SASL (camada de segurança de autenticação simples).

O Microsoft Digest fornece um mecanismo de resposta de desafio simples para autenticar clientes. Este SSP destina-se a uso por aplicativos cliente/servidor usando comunicações baseadas em HTTP ou SASL.

> [!IMPORTANT]
> Não há suporte para autenticação Digest em florestas.
