# <a name="tutorial-for-authenticating-web-pages"></a>Tutorial para autenticação de páginas da Web

Este tutorial destaca os diferentes aspectos aos quais os desenvolvedores da Web devem se preocupar ao criar páginas para o fluxo de autorização da Web para o Windows 8. Esta seção mostra um fluxo de autorização de duas páginas.

**Objetivo:** O objetivo do exemplo é não ser prescritiva sobre o layout ou o design visual que, com certeza, cada provedor terá um ponto de vista exclusivo, mas para destacar algumas áreas que valem a pena investir em ter uma experiência de estilo de design da Microsoft de primeira classe para o Windows 8.

## <a name="prerequisites"></a>Pré-requisitos

Para usar as APIs do agente de autenticação da Web, você precisa:

-   Windows 8 (somente x86 ou AMD64)
-   O Microsoft Serviços de Informações da Internet (IIS) deve ser instalado no painel de controle em "programas e recursos" e usando a opção "ativar ou desativar recursos do Windows".

**Tempo total para conclusão:** 2 minutos.
