# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Especificações de IETF para Microsoft Digest SSP

Autenticação HTTP: a Autenticação de Acesso Básico e Digest (RFC 2617) está disponível em [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) .

O uso do Digest como um rascunho da Internet do mecanismo SASL está disponível em [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) .

Para obter informações adicionais sobre SASL, consulte também a especificação Simple Authentication and Security Layer (RFC 2222) disponível em [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
