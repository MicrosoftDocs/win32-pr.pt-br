# <a name="msv1_0_getuserinfo_response"></a>\_Resposta de \_ GetUserInfo MSV1 0 \_

A \_ estrutura de \_ resposta GetUserInfo MSV1 0 \_ está obsoleta.

Para recuperar informações sobre o usuário da estação de trabalho, consulte a função [**NetWkstaUserGetInfo**](/windows/win32/api/lmwksta/nf-lmwksta-netwkstausergetinfo) .
