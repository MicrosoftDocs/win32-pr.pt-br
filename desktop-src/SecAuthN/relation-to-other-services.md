# <a name="relation-to-other-services"></a>Relação com outros serviços

Outras partes do Microsoft Internet Security Framework usam o [*subsistema de cartão*](../secgloss/s-gly.md)inteligente , conforme mostrado na ilustração a seguir. (Devido às restrições de exportação dos EUA, um CSP [*(provedor*](../secgloss/c-gly.md) de serviços criptográficos) que usa um SCSP para comunicar solicitações relacionadas à criptografia deve assinar e verificar o SCSP.)

![relação da estrutura de segurança da Internet da Microsoft com o subsistema de cartão inteligente](images/smartovr3.png)

Para obter mais informações sobre a função do subsistema de cartão inteligente no Microsoft Internet Security Framework, consulte [https://www.microsoft.com/whdc/device/input/smartcard/default.mspx](https://www.microsoft.com/whdc/device/input/smartcard/default.mspx) .
