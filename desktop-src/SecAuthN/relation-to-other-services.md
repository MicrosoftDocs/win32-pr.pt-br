# <a name="relation-to-other-services"></a>Relação com outros serviços

Outras partes do Microsoft Internet Security Framework usam o [*subsistema de cartão inteligente*](../secgloss/s-gly.md), como mostrado na ilustração a seguir. (Devido às restrições de exportação dos EUA, um CSP ( [*provedor de serviços de criptografia*](../secgloss/c-gly.md) ) que usa um SCSP para comunicar solicitações relacionadas a criptografia deve assinar e verificar o SCSP.)

![relação do Microsoft Internet Security Framework com o subsistema de cartão inteligente](images/smartovr3.png)

Para obter mais informações sobre a função do subsistema de cartão inteligente no Microsoft Internet Security Framework, consulte [https://www.microsoft.com/whdc/device/input/smartcard/default.mspx](https://www.microsoft.com/whdc/device/input/smartcard/default.mspx) .
