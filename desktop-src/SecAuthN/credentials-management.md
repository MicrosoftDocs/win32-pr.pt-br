# <a name="credentials-management"></a>Gerenciamento de credenciais

Os desenvolvedores que escrevem para o Windows podem usar a API de gerenciamento de credenciais, incluindo funções de interface do usuário de gerenciamento de credenciais para obter e gerenciar informações de credenciais, como nomes de usuário e senhas. Essas funções solicitam que as informações da conta do Windows sejam usadas em vez das credenciais estabelecidas durante o logon. Essas solicitações normalmente ocorrem quando as credenciais de logon não têm permissões exigidas pelo aplicativo.

As funções de IU de gerenciamento de credenciais fornecem interfaces com a aparência da interface do usuário do Windows. Essas funções incluem opções personalizáveis que adicionam informações do usuário ao armazenamento de credenciais do usuário.

Os tópicos a seguir fornecem mais informações sobre a API de gerenciamento de credenciais:

-   [Tipos de credenciais](kinds-of-credentials.md)
-   [Formatos de nome de usuário](user-name-formats.md)

Para obter informações sobre as funções na API de gerenciamento de credenciais, consulte [referência de autenticação](authentication-reference.md).
