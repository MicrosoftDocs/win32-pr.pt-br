# <a name="credentials-management"></a>Gerenciamento de credenciais

Os desenvolvedores que escrevem para Windows Interface do Usuário podem usar o API de Gerenciamento de credenciais, incluindo funções de UI (gerenciamento de credenciais) para obter e gerenciar informações de credenciais, como nomes de usuário e senhas. Essas funções solicitam Windows de conta a serem usadas em vez das credenciais estabelecidas durante o logo on. Essas solicitações normalmente ocorrem quando as credenciais de logon não têm permissões exigidas pelo aplicativo.

As funções de interface do usuário do Gerenciamento de Credenciais fornecem interfaces com a aparência da Windows do usuário. Essas funções incluem opções personalizáveis que adicionam informações do usuário ao armazenamento de credenciais do usuário.

Os tópicos a seguir fornecem mais informações sobre as credenciais API de Gerenciamento:

-   [Tipos de credenciais](kinds-of-credentials.md)
-   [Formatos de nome de usuário](user-name-formats.md)

Para obter informações sobre funções na credencial API de Gerenciamento, consulte Referência [de autenticação](authentication-reference.md).
