# <a name="winlogon-notification-package-reference"></a>Referência do pacote de notificação do Winlogon

Os tópicos a seguir fornecem informações de referência para pacotes de notificação do [*Winlogon*](../secgloss/w-gly.md) :

-   [Entradas de registro](registry-entries.md) lista os valores de chave e as configurações que devem ser inseridas no registro do sistema para registrar um pacote de notificação e suas funções de manipulador de eventos.
-   O [protótipo de função do manipulador de eventos](event-handler-function-prototype.md) detalha o protótipo para funções de manipulador de eventos. Cada função de manipulador de eventos em seu pacote de notificação deve corresponder a este protótipo.
