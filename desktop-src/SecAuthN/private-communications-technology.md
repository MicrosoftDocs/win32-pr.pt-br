# <a name="private-communications-technology"></a>Tecnologia de comunicações privadas

A PCT (Tecnologia de Comunicações Privadas) foi superada pelo [*protocolo SSL*](../secgloss/s-gly.md) 3.0 e pelo Protocolo de Segurança da Camada [de Transporte](transport-layer-security-protocol.md).

O Schannel dá suporte ao PCT 1.0 apenas para compatibilidade com versões versões atrás; ele não deve ser usado para novo desenvolvimento. O PCT dá suporte à maioria dos pacote de criptografia com suporte do SSL 2.0.
