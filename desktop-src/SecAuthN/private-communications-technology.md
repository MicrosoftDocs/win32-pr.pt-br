# <a name="private-communications-technology"></a>Tecnologia de comunicação privada

A tecnologia de comunicação privada (PCT) foi substituída pelo [*protocolo SSL*](../secgloss/s-gly.md) 3,0 e pelo [protocolo de segurança da camada de transporte](transport-layer-security-protocol.md).

O Schannel dá suporte apenas a PCT 1,0 para compatibilidade com versões anteriores; Ele não deve ser usado para um novo desenvolvimento. O PCT oferece suporte à maioria dos conjuntos de codificação com suporte do SSL 2,0.
