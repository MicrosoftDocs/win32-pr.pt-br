# <a name="winlogon-notification-packages"></a>Pacotes de notificação do Winlogon

[*Pacotes de notificação do Winlogon*](../secgloss/w-gly.md) são DLLs que recebem e lidam com eventos gerados pelo Winlogon. Você pode implementar esse pacote de notificação para monitorar e responder a eventos do Winlogon. Isso é útil para aplicativos que precisam executar processamento adicional [](../secgloss/s-gly.md) durante o logon ou logoff ou manter informações de estado que devem ser atualizadas quando ocorrem eventos winlogon.

Para obter mais informações sobre Winlogon e DRAAs, consulte [Winlogon e LTD.](winlogon-and-gina.md)
