# <a name="winlogon-notification-packages"></a>Pacotes de notificação do Winlogon

Os pacotes de notificação do [*Winlogon*](../secgloss/w-gly.md) são DLLs que recebem e manipulam eventos gerados pelo winlogon. Você pode implementar um pacote de notificação desse tipo para monitorar e responder a eventos do Winlogon. Isso é útil para aplicativos que precisam executar processamento adicional durante o logon ou logoff, ou manter informações de [*estado*](../secgloss/s-gly.md) que devem ser atualizadas quando eventos do Winlogon ocorrem.

Para obter mais informações sobre Winlogon e GINAs, consulte [Winlogon e Gina](winlogon-and-gina.md).
