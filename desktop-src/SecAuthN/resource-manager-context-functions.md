# <a name="resource-manager-context-functions"></a>Funções de contexto do Gerenciador de recursos

As funções a seguir estabelecem e liberam o [*contexto do Resource Manager*](../secgloss/r-gly.md) que é usado pelas funções de gerenciamento de banco de dados e consulta de banco de dados.



| Tópico                                                  | Descrição                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishContext**](/windows/desktop/api/Winscard/nf-winscard-scardestablishcontext) | Estabelece um contexto para acessar o [*banco de dados do cartão inteligente*](../secgloss/s-gly.md). |
| [**SCardReleaseContext**](/windows/desktop/api/Winscard/nf-winscard-scardreleasecontext)     | Fecha um contexto estabelecido.                                                                                                                       |
