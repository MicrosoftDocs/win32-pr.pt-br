# <a name="sspi"></a>SSPI

Esta seção apresenta informações sobre a [*SSPI (Interface*](../secgloss/s-gly.md) do Provedor de Suporte de Segurança). A SSPI permite que um aplicativo use os vários modelos de segurança disponíveis em um computador ou na rede sem alterar a interface do sistema de segurança. Os tópicos incluem o seguinte:

-   [Modelo SSPI](sspi-model.md)
-   [Opções de SSPI para aplicativos distribuídos](sspi-options-for-distributed-applications.md)
-   [Semântica de contexto SSPI](sspi-context-semantics.md)
-   [Pacotes SSP fornecidos pela Microsoft](ssp-packages-provided-by-microsoft.md)
-   [Pacotes de segurança personalizados](custom-security-packages.md)
-   [Informações de erro estendidas](extended-error-information.md)
