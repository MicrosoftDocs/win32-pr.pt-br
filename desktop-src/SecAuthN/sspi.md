# <a name="sspi"></a>SSPI

Esta seção apresenta informações sobre a [*interface de provedor de suporte de segurança*](../secgloss/s-gly.md) (SSPI). A SSPI permite que um aplicativo use os vários modelos de segurança disponíveis em um computador ou na rede sem alterar a interface do sistema de segurança. Os tópicos incluem o seguinte:

-   [Modelo SSPI](sspi-model.md)
-   [Opções de SSPI para aplicativos distribuídos](sspi-options-for-distributed-applications.md)
-   [Semântica de contexto SSPI](sspi-context-semantics.md)
-   [Pacotes do SSP fornecidos pela Microsoft](ssp-packages-provided-by-microsoft.md)
-   [Pacotes de segurança personalizados](custom-security-packages.md)
-   [Informações de erro estendidas](extended-error-information.md)
