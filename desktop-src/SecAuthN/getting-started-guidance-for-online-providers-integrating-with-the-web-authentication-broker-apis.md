# <a name="getting-started-guidance-for-online-providers-integrating-with-the-web-authentication-broker-apis"></a>Diretrizes de introdução para provedores online integração com as APIs do agente de autenticação da Web

Esta seção orienta os desenvolvedores da Web e os provedores online para a criação de páginas da Web adaptadas para as APIs do agente de autenticação da Web do Windows 8. Ele fornece as recomendações visuais e interativas para uma experiência de usuário de ponta a ponta da página da Web, além de como habilitar os recursos de SSO (logon único).
