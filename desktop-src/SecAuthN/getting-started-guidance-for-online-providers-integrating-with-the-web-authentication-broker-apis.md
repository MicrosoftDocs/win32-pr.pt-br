# <a name="getting-started-guidance-for-online-providers-integrating-with-the-web-authentication-broker-apis"></a>Diretrizes de início para provedores online que se integram às APIs do Agente de Autenticação da Web

Esta seção orienta os desenvolvedores web e provedores online para a criação de páginas da Web personalizadas para as APIs Windows 8 Web Authentication Broker. Ele fornece as recomendações visuais e interativas para uma experiência do usuário de ponta a ponta da página da Web, bem como como habilitar recursos de SSO (logout único).
