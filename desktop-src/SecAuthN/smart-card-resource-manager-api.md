# <a name="smart-card-resource-manager-api"></a>API do Gerenciador de recursos de cartão inteligente

A [*API do Gerenciador de recursos*](../secgloss/r-gly.md) fornece a funcionalidade a seguir.



| Tópico                                                                                               | Descrição                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Funções de consulta de banco de dados de cartão inteligente](smart-card-database-query-functions.md)<br/>           | Consulte o banco de dados do cartão inteligente.<br/>                                                                                                                                                                                                                                   |
| [Funções de gerenciamento de banco de dados de cartão inteligente](smart-card-database-management-functions.md)<br/> | Gerencie o banco de dados do cartão inteligente.<br/>                                                                                                                                                                                                                                  |
| [Funções de contexto do Gerenciador de recursos](resource-manager-context-functions.md)<br/>             | Gerencie o contexto das operações de banco de dados do Gerenciador de recursos.<br/>                                                                                                                                                                                                |
| [Função de suporte do Gerenciador de recursos](resource-manager-support-function.md)<br/>               | Liberar memória alocada.<br/>                                                                                                                                                                                                                                        |
| [Funções de controle de cartão inteligente](smart-card-tracking-functions.md)<br/>                       | Acompanhe cartões inteligentes dentro de leitores.<br/>                                                                                                                                                                                                                                |
| [Funções de acesso de cartão inteligente e leitor](smart-card-and-reader-access-functions.md)<br/>     | Conecte-se e comunique-se com um cartão inteligente, incluindo a transferência de dados usando [*T = 0*](../secgloss/t-gly.md), [*T = 1*](../secgloss/t-gly.md)e protocolos brutos.<br/> |
| [Funções de acesso de cartão direto](direct-card-access-functions.md)<br/>                         | Comunique-se com cartões que podem não estar em conformidade com as especificações ISO 7816.<br/>                                                                                                                                                                                      |
