# <a name="subauthentication-packages"></a>Pacotes de subautenticação

Os [*pacotes de autenticação*](../secgloss/a-gly.md) fornecidos com o Windows dão suporte à personalização usando [*pacotes de subautenticação*](../secgloss/s-gly.md). Um pacote de subautenticação é uma DLL que complementa ou substitui parte dos critérios de autenticação e validação usados pelo pacote de autenticação principal. Por exemplo, um servidor específico pode fornecer um pacote de subautenticação que valida a senha de um usuário com um algoritmo diferente ou especifica restrições de estação de trabalho em um formato diferente.

Para obter mais informações sobre pacotes de subautenticação, consulte o exemplo de subautenticação instalado com o SDK (Software Development Kit) da plataforma.
