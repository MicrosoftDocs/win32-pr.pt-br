<span data-ttu-id="64592-104">O modelo de autenticação da [*autoridade de segurança local*](../secgloss/l-gly.md) (LSA) tem os seguintes recursos:</span><span class="sxs-lookup"><span data-stu-id="64592-104">The [*Local Security Authority*](../secgloss/l-gly.md) (LSA) authentication model has the following features:</span></span>
