# <a name="memory-management-structures"></a>Estruturas de gerenciamento de memória

As estruturas a seguir são usadas com o gerenciamento de memória.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**\_informações de \_ destino da chamada cfg \_**](-cfg-call-target-info.md) | Representa informações sobre destinos de chamada para a proteção de fluxo de controle (CFG). |
| [**\_informações de otimização de \_ recursos de heap \_**](/windows/desktop/api/winnt/ns-winnt-heap_optimize_resources_information) | Especifica os sinalizadores para uma operação HeapOptimizeResources iniciada com [**HeapSetInformation**](/windows/desktop/api/HeapApi/nf-heapapi-heapsetinformation). |
| [**\_requisitos de endereço do mem \_**](/windows/desktop/api/winnt/ns-winnt-mem_address_requirements) | Especifica um endereço base mais baixo e mais alto e alinhamento como parte de um parâmetro estendido para uma função que gerencia a memória virtual. |
| [**\_parâmetro estendido mem \_**](/windows/desktop/api/winnt/ns-winnt-mem_extended_parameter) | Representa um parâmetro estendido para uma função que gerencia a memória virtual. |
| [**\_informações básicas de memória \_**](/windows/desktop/api/winnt/ns-winnt-memory_basic_information) | Contém informações sobre um intervalo de páginas no espaço de endereço virtual de um processo. |
| [**MEMORYSTATUS**](/windows/desktop/api/WinBase/ns-winbase-memorystatus) | Contém informações sobre o estado atual de memória física e virtual. |
| [**MEMORYSTATUSEX**](/windows/desktop/api/sysinfoapi/ns-sysinfoapi-memorystatusex) | Contém informações sobre o estado atual de memória física e virtual, incluindo memória estendida. |
| [**\_entrada de heap de processo \_**](/windows/desktop/api/minwinbase/ns-minwinbase-process_heap_entry) | Contém informações sobre um elemento heap. |
| [**\_Entrada de \_ intervalo de memória do Win32 \_**](/windows/desktop/api/memoryapi/ns-memoryapi-win32_memory_range_entry) | Especifica um intervalo de memória. |
| [**\_Informações de \_ região de memória do Win32 \_**](/windows/desktop/api/memoryapi/ns-memoryapi-win32_memory_region_information) | Contém informações sobre uma região de memória. |
| [**AtlThunkData \_ t**](/previous-versions/windows/desktop/legacy/mt805050(v=vs.85)) | Uma estrutura de dados opaca que representa uma conversão ATL. |
