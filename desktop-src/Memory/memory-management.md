# <a name="memory-management-memory-management"></a>Gerenciamento de memória (gerenciamento de memória)

O Gerenciador de memória implementa a memória virtual, fornece um conjunto principal de serviços, como arquivos de memória mapeada, memória de cópia de gravação, suporte a memória grande e suporte subjacente para o Gerenciador de cache.
