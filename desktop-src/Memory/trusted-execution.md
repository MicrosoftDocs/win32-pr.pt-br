# <a name="trusted-execution"></a>Execução confiável

Enclaves são usados para criar ambientes de execução confiáveis. Um enclave é uma região isolada de código e dados dentro do espaço de endereço para um aplicativo. Somente o código executado no enclave pode acessar dados no mesmo enclave.
