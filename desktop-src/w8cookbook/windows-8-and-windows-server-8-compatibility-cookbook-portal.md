# <a name="windows-and-windows-server-compatibility-cookbook-windows-10-windows-8-windows-81-windows-server-2012-and-windows-server-2012-r2"></a>guia de compatibilidade do Windows e Windows Server: Windows 10, Windows 8, Windows 8.1, Windows Server 2012 e Windows Server 2012 R2

## <a name="purpose"></a>Finalidade

No *Guia*, fornecemos informações sobre alterações nos sistemas operacionais Windows 10, Windows 8, Windows 8.1, Windows Server 2012 e Windows Server 2012 R2. Também fornecemos diretrizes para você verificar a compatibilidade de seus programas existentes e planejados com os novos sistemas operacionais. Vamos supor que você está familiarizado com as versões anteriores do Windows.

O conteúdo se aplica a:

-   Windows 10
-   Windows 8.1
-   Windows Server 2012 R2
-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2
