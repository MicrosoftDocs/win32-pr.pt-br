# <a name="windows-app-certification-kit"></a>Kit de Certificação de Aplicativos Windows

## <a name="platforms"></a>Plataformas

**Clientes –** Windows 7 \| Windows 8 \| Windows 8.1  


## <a name="description"></a>Descrição

Windows O Kit de Certificação de Aplicativos 3.1 está incluído no SDK Windows para Windows 8.1. O kit pode ser usado para pré-certificar aplicativos da Windows Store para Windows 8 e Windows 8.1 antes da integração, bem como para os aplicativos Windows 7, Windows 8 e Windows 8.1 Desktop App Certification Programs e Windows 8.1 Telefone.

O Kit de Certificação de Aplicativos do Windows 3.1 foi atualizado para fornecer aos desenvolvedores uma experiência de usuário perfeita, incluindo a execução de testes em paralelo para economizar tempo geral, seleção seletiva de teste e relatórios aprimorados, para citar algumas das melhorias.

**Observação:** Quando solicitado a fazer uma atualização, atualize sua versão do Windows Kit de Certificação de Aplicativos para se beneficiar dos recursos mais recentes incluídos no kit. Quando seu aplicativo é testado em relação à versão mais recente do Windows Kit de Certificação de Aplicativos, ele tem a garantia de melhor conformidade com as políticas da Windows Store e os programas de certificação da área de trabalho.
