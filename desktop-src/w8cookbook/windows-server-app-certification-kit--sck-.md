# <a name="microsoft-platform-ready-test-tool"></a>Ferramenta de teste pronto para plataforma Microsoft

## <a name="platform"></a>Plataforma

 **Servidores** – windows Server 2012 \| Windows Server 2012 R2  

## <a name="description"></a>Descrição

A ferramenta de teste Microsoft Platform Ready (MPR) Test é usada para a preparação do aplicativo da plataforma e para validar a conformidade com os requisitos de certificação para aplicativos do Windows Server 2012 e do Windows Server 2012 R2.

A Microsoft incentiva enfaticamente a incorporação da ferramenta de teste de MPR aos processos de criação e teste de software, garantindo assim a prontidão da plataforma antes que os aplicativos sejam lançados no mercado. A ferramenta de teste de MPR também é uma ferramenta recomendada para testar aplicativos de linha de negócios e para avaliar os produtos de servidor para a compatibilidade da plataforma antes de tomar decisões de compra.

## <a name="requirements"></a>Requisitos

A ferramenta de teste de MPR inclui testes automatizados para o:

-   Windows Installer
-   Configuração e funcionalidade primária
-   Drivers e segurança
-   Práticas Recomendadas

O teste automático e o envio de resultados para a maioria dos aplicativos de servidor devem levar 2 horas ou menos; aplicativos mais complexos podem levar cerca de 4 horas.

Todos os drivers devem passar separadamente no teste de certificação de hardware do Windows e ser assinados pela Microsoft para o Windows Server 2012.

## <a name="usage"></a>Uso

Para testar seus aplicativos:

1.  Instale a configuração mínima mais recente do Windows Server 2012 (GUI do servidor completo, interface mínima do servidor ou núcleo do servidor) conforme necessário para seu aplicativo.
2.  Examine os requisitos de certificação.
3.  Em um sistema limpo, execute a ferramenta de teste MPR no modo de interface do usuário completa ou no modo de linha de comando.
4.  Conclua o processo de teste autoguiado.
5.  Examine os resultados e corrija seu aplicativo conforme necessário.
6.  Entre e carregue os resultados bem-sucedidos no portal pronto para a plataforma Microsoft para obter uma listagem no catálogo do Windows Server e no uso do logotipo.
