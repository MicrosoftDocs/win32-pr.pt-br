# <a name="bluetooth-drivers"></a>Drivers Bluetooth

Os clientes de IHVs e corporativos devem estar cientes de que os drivers Bluetooth e os pacotes de perfil de terceiros não serão migrados para o Windows 10 como parte da atualização do Windows 7, Windows 8 ou Windows 8.1. Consulte [Este artigo](/windows-hardware/design/component-guidelines/bluetooth) para obter mais informações sobre as alterações de Bluetooth para o Windows 10.
