# <a name="undocumented-apis"></a>APIs não documentadas

Seus aplicativos não devem chamar APIs do Windows não documentadas ou depender da exportação de um arquivo específico do Windows ou de chaves do registro. Isso pode levar a problemas de funcionalidade interrompida, perda de dados e segurança em potencial. Se a funcionalidade que o seu aplicativo requer não estiver disponível, será uma oportunidade de fornecer comentários por meio dos canais de comentários usuais e por meio do programa Windows Insider.
