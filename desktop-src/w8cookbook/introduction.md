# <a name="introduction-compatibility-cookbook-for-windows"></a>Introdução (recookbook de compatibilidade para Windows)

O Windows 10, o Windows 8, o Windows 8.1 e o Windows Server 2012 R2 apresentam a tecnologia de sistema operacional mais recente e as plataformas de desenvolvimento de software para uso por desenvolvedores de aplicativos e empresas em todo o mundo. Como parte de aprimorar ainda mais a segurança, a confiabilidade, o desempenho e a experiência do usuário do Windows, a Microsoft introduziu muitos recursos novos, aprimorou os recursos existentes e removeu outros.

Embora o objetivo do Windows 10, Windows 8, Windows 8.1 e Windows Server 2012 R2 seja compatível com a maioria dos seus respectivos aplicativos escritos para sistemas operacionais lançados anteriormente, algumas quebras de compatibilidade são inevitáveis devido a inovações, segurança rígida e maior confiabilidade. Em geral, a compatibilidade dos novos sistemas operacionais com aplicativos existentes é alta.

Este *manual* é dividido em três seções principais, Windows 10, Windows 8.1 e Windows 8:

-   Windows 10
-   Windows 8.1 e Windows Server 2012 R2
-   Windows 8 e Windows Server 2012

Os tópicos a seguir são novos com esta versão do *manual* ou foram atualizados:

-   Inicialização do arquivo de imagem do Windows (WimBoot)
-   Kit de Certificação de Aplicativos Windows

Convidamos você a conferir esses tópicos para aprender a otimizar seus aplicativos e aproveitar o que essa versão mais recente do Windows tem a oferecer.

As edições do Windows Compatibility Cookbook que abordam versões anteriores dos sistemas operacionais Windows e Windows Server podem ser encontradas no [Windows Vista e no Windows server 2008](/previous-versions/bb757005(v=msdn.10)) e no Windows [7 e no Windows Server 2008 R2 Application Quality Cookbook](/windows/desktop/Win7AppQual/windows-7-application-quality-cookbook)
