# <a name="stackedby-argument-windows-search"></a>Argumento STACKEDBY (Windows Search)

O **argumento stackedby** especifica a coluna de propriedade pela qual empilhar os resultados. Você pode empilhar por qualquer propriedade válida do sistema de propriedades.

Este tópico é organizado da seguinte forma:

-   [Exemplo](#example)
-   [Tópicos relacionados](#related-topics)

## <a name="example"></a>Exemplo


```
 search-ms:query=vacation&stackedby=System.Author 
```
