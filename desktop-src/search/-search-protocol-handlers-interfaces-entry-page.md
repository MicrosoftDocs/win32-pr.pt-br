# <a name="protocol-handlers-interfaces"></a>Interfaces de manipuladores de protocolo

esta seção de referência de pesquisa Windows consiste em interfaces de manipulador de protocolo.


## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                   | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IItemPropertyBag**](iitempropertybag.md)<br/>                                 | Define métodos para obter informações sobre as propriedades de um item de pesquisa. essa interface tem suporte apenas no Windows XP e no Windows Server 2003 e não deve mais ser usada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus)<br/>                               | fornece métodos para verificar o bloqueio oportunista usado pelo Microsoft Windows Desktop Search (WDS) em itens durante a indexação. Se outro processo bloquear o arquivo de forma incompatível, o WDS perderá seu bloqueio e permitirá que o outro processo tenha o arquivo. Esse mecanismo permite que o WDS seja executado em segundo plano. Consequentemente, o WDS precisa verificar seus bloqueios para garantir que outro processo não tenha precedência enquanto o WDS indexa o item. <br/> Um objeto [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) de terceiros pode implementar essa interface se o armazenamento de dados subjacente fornecer um mecanismo para controlar o acesso simultâneo aos itens. Se essa interface for exposta por **IUrlAccessor**, o WDS verificará o [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus) durante a indexação de itens desse repositório. <br/> |
| [**IProtocolHandlerServices**](/previous-versions/windows/desktop/legacy/bb231447(v=vs.85))<br/>         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProtocolHandlerSite**](/windows/desktop/api/Searchapi/nn-searchapi-iprotocolhandlersite)<br/>                 | Fornece métodos para um objeto [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) do manipulador de protocolo para consultar o daemon de filtro para o filtro apropriado para o item de URL. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**ISearchProtocol**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol)<br/>                           | Fornece métodos para invocar, inicializar e gerenciar objetos [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . Os métodos nessa interface são chamados pelo host de protocolo ao processar URLs do gatherer. <br/> O manipulador de protocolo implementa o protocolo para acessar uma fonte de conteúdo em seu formato nativo. Use essa interface para implementar um manipulador de protocolo personalizado para expandir as fontes de dados que podem ser indexadas. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocol2**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol2)<br/>                         | Fornece métodos para invocar, inicializar e gerenciar objetos [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . Os métodos nessa interface são chamados pelo host de protocolo ao processar URLs do gatherer. <br/> O manipulador de protocolo implementa o protocolo para acessar uma fonte de conteúdo em seu formato nativo. Use essa interface para implementar um manipulador de protocolo personalizado para expandir as fontes de dados que podem ser indexadas. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocolThreadContext**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocolthreadcontext)<br/> | Essa interface opcional permite que o manipulador de protocolo execute uma ação no thread usado para filtragem no host de protocolo. Quando o host de protocolo é iniciado, ele inicializa primeiro todos os manipuladores de protocolo e, em seguida, cria os threads de filtragem. Os métodos nessa interface permitem que os manipuladores de protocolo gerenciem seus recursos que são usados por um thread de filtragem.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISearchProtocolUI**](-search-isearchprotocolui.md)<br/>                       | Fornece um método para invocar objetos [**ISearchItem**](-search-isearchitem.md) . Os métodos nessa interface são chamados pelo host de protocolo ao processar URLs do gatherer. O manipulador de protocolo implementa o protocolo para acessar uma fonte de conteúdo em seu formato nativo, e essa interface implementa um manipulador de protocolo personalizado para expandir as fontes de dados que podem ser indexadas.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor)<br/>                                 | Fornece métodos para processar um item individual em uma fonte de conteúdo cuja URL é fornecida pelo gatherer para o host de filtro. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2)<br/>                               | Estende a funcionalidade da interface [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3)<br/>                               | Estende a funcionalidade da interface [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2) com o método [**IUrlAccessor3:: GetImpersonationSidBlobs**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor3-getimpersonationsidblobs) para identificar identificadores de segurança do usuário (SIDs) para uma URL especificada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IUrlAccessor4**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor4)<br/>                               | Estende a funcionalidade da interface [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3) com o método [**IUrlAccessor4:: ShouldIndexItemContent**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor4-shouldindexitemcontent) que identifica se o conteúdo do item deve ser indexado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
