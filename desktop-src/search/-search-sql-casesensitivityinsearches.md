# <a name="case-sensitivity-in-searches"></a>Distinção de maiúsculas e minúsculas em pesquisas

as consultas do Microsoft Windows Search não diferenciam maiúsculas de minúsculas. Isso inclui palavras-chave de consulta (SELECT é idêntico a Select, Select e sELect) e termos de pesquisa. Se um identificador ou literal usar caracteres Unicode que têm semântica de mapeamento de maiúsculas e minúsculas, o mecanismo de pesquisa considerará todos os casos equivalentes.
