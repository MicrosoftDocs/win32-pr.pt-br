# <a name="managing-the-index"></a>Gerenciar o índice

O Windows Search permite que você gerencie o índice do Windows Search com três componentes principais: Gerenciador de pesquisa, Gerenciador de catálogo e Gerenciador de escopo de rastreamento.

Esta seção é organizada da seguinte maneira:

- [Interfaces para gerenciar o índice](interfaces-for-managing-the-index.md)
- [Usando o Gerenciador de pesquisa](-search-3x-wds-mngidx-searchmanager.md)
- [Usando o Gerenciador de catálogo](-search-3x-wds-mngidx-catalog-manager.md)
- [Usando o Gerenciador de escopo de rastreamento](-search-3x-wds-extidx-csm.md)
