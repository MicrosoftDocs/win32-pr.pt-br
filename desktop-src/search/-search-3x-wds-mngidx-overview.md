# <a name="managing-the-index"></a>Gerenciar o índice

Windows a pesquisa permite que você gerencie o índice de pesquisa Windows com três componentes principais: gerenciador de pesquisa, gerenciador de catálogo e gerenciador de escopo de rastreamento.

Esta seção é organizada da seguinte maneira:

- [Interfaces para gerenciar o índice](interfaces-for-managing-the-index.md)
