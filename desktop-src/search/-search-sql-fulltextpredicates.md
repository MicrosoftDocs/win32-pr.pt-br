# <a name="full-text-predicates"></a>Predicados de Full-Text

a linguagem de consulta do Microsoft Windows search dá suporte a dois predicados de pesquisa de texto completo. O predicado CONTAINS executa comparações em colunas que contêm texto. A cláusula CONTAINS pode executar a correspondência em palavras ou frases únicas, com base na proximidade dos termos de pesquisa. Em comparação, o predicado FREETEXT é ajustado para corresponder ao significado das frases de pesquisa em relação às colunas de texto.

Esta seção contém os seguintes tópicos:

-   [Predicado CONTAINS](-search-sql-contains.md)
-   [Predicado FREETEXT](-search-sql-freetext.md)
