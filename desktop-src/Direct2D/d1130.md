# <a name="d1130-dxgi-format-disallowed-by-caps"></a>D1130: formato DXGI não permitido por Caps

O dispositivo não tem a capacidade de dar suporte ao formato DXGI fornecido.






 

## <a name="possible-causes"></a>Possíveis causas

Para operações Direct2D que são executadas em hardware e usam um formato de superfície, o formato da superfície deve ser suportado pelo nível de recurso do hardware. Por exemplo, se o hardware for do nível de recurso 9, ele não poderá dar suporte a formatos 16bpp ou formatos SRGB, como o \_ formato dxgi \_ B8G8R8A8 \_ UNORM \_ sRGB.

## <a name="possible-fixes"></a>Possíveis correções

Consulte [níveis de recurso do Direct3D](/previous-versions/windows/apps/hh994923(v=win.10)) para obter orientação sobre as diferenças entre os níveis de recurso e seus formatos com suporte. Considere o uso do [**método ID3D11Device:: CheckFormatSupport**](/windows/desktop/api/d3d11/nf-d3d11-id3d11device-checkformatsupport) para verificar se o formato necessário é suportado pelo hardware antes de usá-lo com Direct2D.
