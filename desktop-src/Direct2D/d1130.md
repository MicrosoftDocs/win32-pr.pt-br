# <a name="d1130-dxgi-format-disallowed-by-caps"></a>D1130: Formato DXGI não permitido por caps

O dispositivo não tem a capacidade de dar suporte ao formato DXGI determinado.






 

## <a name="possible-causes"></a>Possíveis causas

Para Direct2D operações executadas no hardware e usar um formato de superfície, o formato de superfície deve ter suporte pelo nível de recurso do hardware. Por exemplo, se o hardware for de nível de recurso 9, ele poderá não dar suporte a formatos 16bpp ou formatos SRGB, como DXGI \_ FORMAT \_ B8G8R8A8 \_ UNORM \_ SRGB.

## <a name="possible-fixes"></a>Possíveis correções

Confira [Níveis de recursos do Direct3D](/previous-versions/windows/apps/hh994923(v=win.10)) para obter diretrizes sobre as diferenças entre os níveis de recurso e seus formatos com suporte. Considere usar o método [**ID3D11Device::CheckFormatSupport**](/windows/desktop/api/d3d11/nf-d3d11-id3d11device-checkformatsupport) para verificar se o formato necessário é compatível com o hardware antes de usá-lo com Direct2D.
