# <a name="d1154-invalid-bitmap-input-format"></a>D1154: Formato de entrada de bitmap inválido

Essa operação não é compatível com o formato de pixel do bitmap.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D dá suporte a formatos de superfície somente alfa, como DXGI \_ FORMAT \_ A8 UNORM, mas eles só podem ser usados como pincéis de opacidade ou para \_ mascaramento. Se um aplicativo cria um pincel de bitmap cujo bitmap tem um formato somente alfa, ele não deve tentar usar o pincel com operações que exigem um pincel de cor. ID2D1RenderTarget::D rawLine e ID2D1RenderTarget::FillRectangle, por exemplo, use um pincel de cor.

## <a name="possible-fixes"></a>Possíveis correções

Para resolver essa mensagem, verifique se bitmaps com formatos somente alfa são usados apenas para pincéis de opacidade ou mascaramento. Se um bitmap deve conter informações de cor, crie-o usando um formato de superfície que inclui canais de cores.
