# <a name="d1199-changed-command-list-resource"></a>D1199: recurso de lista de comandos alterada

Uma imagem desenhada em uma lista de comandos acabou de ser alterada. Isso pode produzir consequências indesejadas. A lista de comandos agora conterá a imagem alterada em vez do conteúdo da imagem no momento em que ela foi desenhada.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo adicionou uma imagem a uma lista de comandos e depois alterou a imagem.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o comportamento resultante é o esperado. O comportamento nesse caso, em que uma lista de comandos é o destino, produzirá resultados diferentes do que se um bitmap fosse definido como o destino. Se o comportamento não for o esperado, talvez o aplicativo queira clonar a imagem antes de atualizá-la.
