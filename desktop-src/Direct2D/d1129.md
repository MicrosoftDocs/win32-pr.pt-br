# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129: formato DXGI incompatível com o modo alfa

A combinação do modo alfa e do formato DXGI fornecido não é compatível entre si.





 

## <a name="possible-causes"></a>Possíveis causas

Para operações Direct2D que usam um formato de superfície e modo alfa, há casos em que o formato de superfície e o modo alfa não concordam. Por exemplo, um formato de superfície como o \_ formato dxgi \_ a8 \_ UNORM não faz sentido ser usado com ignore Alpha, pois ele é renderizado apenas para um canal alfa. Além disso, o alfa reto não tem suporte em alguns \_ formatos dxgi, mas o alfa é premultiplicado.

## <a name="possible-fixes"></a>Possíveis correções

Consulte [formatos de pixel com suporte e modos alfa](supported-pixel-formats-and-alpha-modes.md) para obter orientação sobre quais combinações de formato têm suporte e quais não são. Modifique o aplicativo para usar uma combinação de formato com suporte.
