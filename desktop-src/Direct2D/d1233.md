# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233: BeginDraw inválido do erro anterior

BeginDraw foi chamado, mas será ignorado porque o contexto do dispositivo está em um estado inválido.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo chamado BeginDraw, mas o contexto do dispositivo estava em um estado inválido de uma operação anterior.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o aplicativo responde adequadamente aos erros retornados por Flush ou EndDraw, recriando seu dispositivo conforme necessário. Se um comando fez com que um contexto de dispositivo estivesse em um estado inválido, o Flush ou EndDraw correspondente retornará um erro.
