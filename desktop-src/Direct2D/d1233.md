# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233: BeginDraw inválido de erro anterior

BeginDraw foi chamado, mas será ignorado porque o contexto do dispositivo está em um estado inválido.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo chamou BeginDraw, mas o contexto do dispositivo estava em um estado inválido de uma operação anterior.

## <a name="possible-fixes"></a>Possíveis correções

Certifique-se de que o aplicativo responda adequadamente aos erros retornados por flush ou EndDraw, recriando seu dispositivo conforme necessário. Se um comando tiver feito com que um contexto de dispositivo esteja em um estado inválido, o flush ou EndDraw correspondente retornará um erro.
