# <a name="d1160-cycle-in-graph"></a>D1160: ciclo no grafo

Foi feita uma tentativa de desenhar um grafo de imagem com um ciclo nele.






 

## <a name="possible-causes"></a>Possíveis causas

O Direct2D permite que os aplicativos construam grafos de efeito para flexibilidade no comportamento de efeito. No entanto, os aplicativos não devem criar um grafo de efeito que contenha um ciclo. Por exemplo, considere um gráfico de efeito contendo os nós A, B e C: é inválido vincular a a B, então, B a C, então C a a também. Direct2D identificará grafos de efeito cíclico e relatará um erro; a camada de depuração produzirá essa mensagem.

## <a name="possible-fixes"></a>Possíveis correções

Para evitar essa mensagem, Evite criar grafos de efeito que tenham um ciclo. Os grafos de efeito de design efetivamente, incluindo pelo menos um nó que atua apenas como saída.
