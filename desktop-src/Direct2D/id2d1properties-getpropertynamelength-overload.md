# <a name="id2d1propertiesgetpropertynamelength-methods"></a>Métodos ID2D1Properties:: GetPropertyNameLength

Obtém o número de caracteres para o nome de propriedade fornecido.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                         | Descrição                                                                                        |
|:-------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**GetPropertyNameLength (U)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getpropertynamelength(u))     | Obtém o número de caracteres para o nome de propriedade fornecido. Essa é uma sobrecarga de modelo.<br/> |
| [**GetPropertyNameLength (UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getpropertynamelength(uint32)) | Obtém o número de caracteres para o nome de propriedade fornecido.<br/>                              |
