# <a name="d1173-texture-cube-interop-unsupported"></a>D1173: interoperabilidade de cubo de textura sem suporte

As texturas sinalizadas como D3D11RESOURCE \_ misc \_ TEXTURECUBE não podem ser usadas pelo Direct2D.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo tentou fazer a interoperabilidade de uma textura Direct3D com a \_ opção D3D11RESOURCE misc \_ TEXTURECUBE especificada.

## <a name="possible-fixes"></a>Possíveis correções

Evite as texturas interoperacionais que têm o especificador TEXTURECUBE. Em vez disso, reorganize o design de renderização do aplicativo ou considere interoperar um intermediário.
