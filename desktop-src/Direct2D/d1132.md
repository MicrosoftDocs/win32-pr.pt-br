# <a name="d1132-dxgi-format-not-hwnd-or-dc-compatible"></a>D1132: formato DXGI não é compatível com HWND ou DC

O formato DXGI fornecido não é válido para este método de criação de destino de renderização.





 

## <a name="possible-causes"></a>Possíveis causas

A criação de rendertargets de HWND ou de DC usando ID2D1Factory:: CreateHwndRenderTarget ou ID2D1Factory:: CreateDCRenderTarget introduz restrições adicionais no formato de superfície do destino. Em particular, esses destinos devem ser do tipo DXGI \_ Format \_ R8G8B8A8 \_ UNORM ou dxgi \_ Format \_ B8G8R8A8 \_ UNORM.

## <a name="possible-fixes"></a>Possíveis correções

Para resolver esse problema, verifique se o formato especificado para o renderTarget do controlador de domínio ou HWND do aplicativo é um dos formatos mencionados acima. Se o aplicativo não exigir necessariamente esses tipos de destinos, considere a renderização para uma textura, suas opções de formato de superfície para compatibilidade, uma vez que um destino pode ser menos restritivo.
