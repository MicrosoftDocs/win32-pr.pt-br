# <a name="d2d1_rect_u"></a>D2D1 \_ Rect \_ U

Representa um retângulo definido pelas coordenadas do canto superior esquerdo (esquerda, superior) e as coordenadas do canto inferior direito (direita, inferior).


```C++
typedef D2D_RECT_U D2D1_RECT_U;
```



## <a name="remarks"></a>Comentários

**D2d1 \_ RECT \_ u** é um novo nome para a estrutura [**U D2D \_ Rect \_**](/windows/desktop/api/dcommon/ns-dcommon-d2d_rect_u) já definida.
