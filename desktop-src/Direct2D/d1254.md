# <a name="d1254-operation-on-single-threaded-domain"></a>D1254: Operação em um único domínio threaded

Uma chamada para [**ID2D1Multithread::Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) ou [**ID2D1Multithread::Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) foi feita em um domínio de fábrica de thread único.

Isso poderá ocorrer se o aplicativo tiver criado inadvertidamente a fábrica D2D com [**D2D1 \_ FACTORY TYPE MULTI \_ \_ \_ THREADED**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type). Nesse caso, a chamada será ignorada.
