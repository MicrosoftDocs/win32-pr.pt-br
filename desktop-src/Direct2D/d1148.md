# <a name="d1148-invalid-operation-on-closed-print-control"></a>D1148: operação inválida no controle de impressão fechado

Direct2D Print: [**AddPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage) ou [**Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) é chamado depois que [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) foi fechado






 

## <a name="possible-causes"></a>Possíveis causas

Essa mensagem pode ocorrer se um aplicativo chamar ID2D1PrintControl:: AddPage em um controle de impressão que foi fechado. O Direct2D considera que os objetos de controle de impressão sejam inalteráveis depois de fechados, de maneira semelhante à das listas de comandos e geometrias de caminho.

## <a name="possible-fixes"></a>Possíveis correções

Certifique-se de que o aplicativo não tente adicionar páginas a um controle de impressão fechado. Modifique a lógica do aplicativo para que todas as páginas necessárias sejam enviadas a um controle de impressão antes de serem fechadas.
