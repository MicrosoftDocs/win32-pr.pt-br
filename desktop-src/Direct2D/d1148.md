# <a name="d1148-invalid-operation-on-closed-print-control"></a>D1148: Operação inválida no controle de impressão fechado

Direct2D Impressão: [**AddPage ou**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage) [**Close é**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) chamado depois [**que ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) foi fechado






 

## <a name="possible-causes"></a>Possíveis causas

Essa mensagem poderá ocorrer se um aplicativo chamar ID2D1PrintControl::AddPage em um controle de impressão que foi fechado. Direct2D considera objetos de controle de impressão inalteráveis depois que eles são fechados, de maneira semelhante a geometrias de caminho e listas de comandos.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o aplicativo não tenta adicionar páginas a um controle de impressão fechado. Modifique a lógica do aplicativo para que todas as páginas necessárias sejam enviadas a um controle de impressão antes de ser fechada.
