# <a name="d2d1_matrix_4x4_f"></a>D2D1 \_ matriz \_ 4x4 \_ F

Representa uma matriz de 4 por 4.


```C++
typedef D2D_MATRIX_4X4_F D2D1_MATRIX_4X4_F;
```



## <a name="remarks"></a>Comentários

**D2d1 \_ MATRIZ \_ 4x4** é um novo nome para a estrutura [**D2D \_ Matrix \_ 4x4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x4_f) . Para obter uma lista de campos fornecidos pela matriz, consulte **D2D \_ Matrix \_ 4x4 \_ F**.
