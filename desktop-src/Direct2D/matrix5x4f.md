# <a name="matrix5x4f-class"></a>Classe Matrix5x4F

A classe **Matrix5x4F** representa uma matriz 5 por 4 e fornece métodos de conveniência para a criação de matrizes.

A classe **Matrix5x4F** herda da [**\_ matriz d2d1 \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f). **Matrix5x4F** também tem estes tipos de membros:
