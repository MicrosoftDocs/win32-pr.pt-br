# <a name="d1235-target-rendered-to-from-multiple-dcs"></a>D1235: destino processado para de vários DCs

Foi feita uma tentativa de renderização em uma imagem enquanto ela ainda estava sendo processada por um contexto de dispositivo anterior. EndDraw deve ser chamado nesse contexto de dispositivo antes do processamento com o contexto do dispositivo atual. Quando a camada de depuração é usada, o endereço desse contexto de dispositivo também é de saída.
