# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: o estilo de traço personalizado requer matriz de traços

A criação de um estilo de traço com DASH \_ STYLE CUSTOM requer uma matriz de \_ traços.






 

## <a name="possible-causes"></a>Possíveis causas

Quando o aplicativo tentou criar um ID2D1Style, ele usou o atributo PERSONALIZADO D2D1 DASH STYLE, mas forneceu uma matriz de traços \_ \_ \_ NULL.

## <a name="possible-fixes"></a>Possíveis correções

Se o aplicativo exigir traços personalizados, verifique se uma matriz de traços válida foi especificada. Consulte CreateStrkeStyle para ver um exemplo. Se um estilo de traço integrado for aceitável, deixe a matriz de traços como NULL e use um especificador como D2D1 \_ DASH \_ STYLE \_ DASH.
