# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: o estilo de tracejado personalizado requer a matriz Dash

A criação de um estilo de traço com \_ estilo tracejado \_ personalizado requer uma matriz Dash.






 

## <a name="possible-causes"></a>Possíveis causas

Quando o aplicativo tentou criar um ID2D1StrokeStyle, ele usou o \_ \_ atributo personalizado de estilo tracejado d2d1 \_ , mas fornecia uma matriz NULL Dash.

## <a name="possible-fixes"></a>Possíveis correções

Se o aplicativo exigir traços personalizados, verifique se uma matriz Dash válida está especificada. Consulte CreateStrokeStyle para obter um exemplo. Se um estilo de tracejado interno for aceitável, deixe a matriz Dash como nula e use um especificador, como D2D1 \_ Dash de \_ estilo tracejado \_ .
