# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229: D1229: chamada imersiva para GetDesktopDpi

ID2D1Factory:: GetDesktopDpi foi chamado em um aplicativo da Windows Store. Isso não é recomendável. Em vez disso, DisplayProperties:: LogicalDpi deve ser usado.






 

## <a name="possible-causes"></a>Possíveis causas

ID2D1Factory:: GetDesktopDpi foi chamado em um aplicativo da Windows Store.

## <a name="possible-fixes"></a>Possíveis correções

displayproperties:: LogicalDpi deve ser usado para obter o DPI em um aplicativo da Windows Store.
