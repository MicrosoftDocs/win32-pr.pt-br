# <a name="d1158-pixels-unit-mode-unsupported-with-print"></a>D1158: modo de unidade de pixels sem suporte com Print

Direct2D Print: o comando setunitmode (PIXELS) sem suporte estava presente na lista de comandos enviada para impressão.





 

## <a name="possible-causes"></a>Possíveis causas

Direct2D Print não permite um modo de unidade de PIXELS. O modo de unidade padrão é DIPS.

## <a name="possible-fixes"></a>Possíveis correções

Para resolver essa mensagem, evite definir o modo de unidade do contexto do dispositivo como PIXELS ao criar conteúdo imprimível. Se for necessário aplicar uma transformação de escala global que afeta todas as operações de desenho primitivo, considere se uma chamada equivalente a ID2D1RenderTarget:: SetDpi ou ID2D1RenderTarget:: SetTransform é suficiente.
