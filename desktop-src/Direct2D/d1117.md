# <a name="d1117-bitmap-is-not-a-target"></a>D1117: bitmap não é um destino

Falha de SetTarget porque o bitmap não tem a opção de destino.






 

## <a name="possible-causes"></a>Possíveis causas

Para definir um ID2D1Bitmap como um destino, ele deve ser criado usando o \_ sinalizador de destino de opções de bitmap d2d1 \_ \_ . Esse sinalizador está disponível como um campo de \_ PROPERTIES1 de bitmap d2d1 \_ usado com ID2D1DeviceContext:: CreateBitmap.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o aplicativo cria seu bitmap de destino usando D2D1 \_ Opções de bitmap de \_ \_ destino. Se ele criar o bitmap de destino usando ID2D1RenderTarget:: CreateBitmap, modifique-o para usar ID2D1DeviceContext:: CreateBitmap. Quando o sinalizador de destino for especificado, o Direct2D também verificará se os atributos do Direct3D da textura são compatíveis com seu uso como um destino.
