# <a name="d1163-print-bitmap-too-big"></a>D1163: bitmap de impressão muito grande

O bitmap é muito grande para impressão, considere dividi-lo em blocos menores.






 

## <a name="possible-causes"></a>Possíveis causas

A impressão Direct2D impõe um limite para os tamanhos de bitmaps imprimíveis. Esse limite é de 65535 unidades dependentes do dispositivo. Se o aplicativo incluir um bitmap que exceda esse tamanho em conteúdo imprimível, o Direct2D relatará um erro e a camada de depuração produzirá essa mensagem.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se os bitmaps usados em operações de impressão não excedem o limite de tamanho da impressão Direct2D. Considere particionar o bitmap em vários blocos.
