# <a name="d1163-print-bitmap-too-big"></a>D1163: bitmap de impressão muito grande

O bitmap é muito grande para impressão, considere dividi-lo em blocos menores.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D Print impõe um limite para os tamanhos de bitmaps imprimíveis. Esse limite é de 65535 unidades dependentes do dispositivo. se o aplicativo incluir um bitmap que exceda esse tamanho em conteúdo imprimível, Direct2D relatará um erro e a camada de depuração produzirá essa mensagem.

## <a name="possible-fixes"></a>Possíveis correções

verifique se os bitmaps usados nas operações de impressão não excedem Direct2D limite de tamanho da impressão. Considere particionar o bitmap em vários blocos.
