# <a name="d1250-inaccessible-source"></a>D1250: fonte inacessível

Foi detectada uma tentativa de desenhar a partir de um bitmap Inacessível.






 

## <a name="possible-causes"></a>Possíveis causas

Foi detectada uma tentativa de desenhar a partir de uma fonte inacessível. Um motivo pelo qual uma fonte pode ser inacessible é se ela foi criada a partir de uma textura compartilhada e a textura não foi adquirida.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se todos os bitmaps estão acessíveis antes de desenhá-los com Direct2D.
