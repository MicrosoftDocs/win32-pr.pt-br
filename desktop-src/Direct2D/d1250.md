# <a name="d1250-inaccessible-source"></a>D1250: Fonte inacessível

Foi detectada uma tentativa de desenhar de um bitmap inacessível.






 

## <a name="possible-causes"></a>Possíveis causas

Foi detectada uma tentativa de desenhar de uma fonte inacessível. Um motivo pelo qual uma fonte pode ser inacessível é se ela foi criada com uma textura compartilhada e a textura não foi adquirida.

## <a name="possible-fixes"></a>Possíveis correções

Certifique-se de que todos os bitmaps sejam acessíveis antes de desenhar com Direct2D.
