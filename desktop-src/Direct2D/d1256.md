# <a name="d1256-invalid-analysis-effect-usage"></a>D1256: Uso inválido do efeito de análise

Um efeito de análise foi usado incorretamente. Isso pode ocorrer ao usar um efeito de análise como a entrada para outro efeito ou um pincel de imagem, desenhando-o em uma lista de comandos ou armazenando em cache sua saída.
