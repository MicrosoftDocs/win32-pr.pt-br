# <a name="d1256-invalid-analysis-effect-usage"></a>D1256: uso de efeito de análise inválido

Um efeito de análise foi usado incorretamente. Isso pode ocorrer ao usar um efeito de análise como a entrada para outro efeito ou um pincel de imagem, desenhá-lo para uma lista de comandos ou armazenar em cache sua saída.
