# <a name="d1124-dxgi-format-unknown-with-null-target-bitmap"></a>D1124: formato DXGI desconhecido com bitmap de destino nulo

\_O formato dxgi \_ desconhecido não é permitido como um parâmetro para essa API quando o destino é definido como nulo ou é uma lista de comandos.






 

## <a name="possible-causes"></a>Possíveis causas

Para operações como ID2D1RenderTarget:: CreateCompatibleRenderTarget, o destino de renderização de origem deve ter um bitmap de destino atribuído. Sem um bitmap de destino, Direct2D não pode determinar qual formato de superfície o novo recurso de destino de renderização terá.

## <a name="possible-fixes"></a>Possíveis correções

Para resolver isso, verifique se o ID2D1DeviceContext correspondente tem um bitmap de destino atribuído usando ID2D1DeviceContext:: SetTarget. Ou, se o aplicativo simplesmente precisar da capacidade de renderizar para duas superfícies de destino diferentes, considere criar um ID2D1Bitmap e defini-lo como o destino do contexto do dispositivo em vez de usar ID2D1RenderTarget:: CreateCompatibleRenderTarget.
