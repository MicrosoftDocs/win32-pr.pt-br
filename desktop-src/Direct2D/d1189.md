# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: opção de bitmap de leitura de CPU foi usada incorretamente

\_ \_ As opções de bitmap d2d1 \_ leitura da CPU \_ devem ser combinadas com \_ Opções de bitmap d2d1 \_ \_ não podem ser \_ desenhadas






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou uma combinação incompatível dos \_ sinalizadores de opções de bitmap d2d1 \_ .

## <a name="possible-fixes"></a>Possíveis correções

Remova um dos sinalizadores para criar um conjunto válido de opções de bitmap D2D1 \_ \_ .
