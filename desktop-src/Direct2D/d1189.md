# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: A opção bitmap de leitura de CPU foi usada incorretamente

D2D1 OPÇÕES DE BITMAP A LEITURA DA CPU deve ser combinada \_ \_ com OPÇÕES \_ \_ D2D1 \_ BITMAP NÃO PODE \_ \_ \_ DESENHAR






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou uma combinação incompatível dos sinalizadores D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possíveis correções

Remova um dos sinalizadores para criar um conjunto válido de OPÇÕES D2D1 \_ \_ BITMAP.
