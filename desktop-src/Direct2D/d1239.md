# <a name="d1239-source-transform-bitmap-invalid-call"></a>D1239: chamada de bitmap de transformação de origem inválida

O bitmap passado para ID2D1SourceTransform::D RAW não oferece suporte a esse método.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo chamou um método sem suporte em um bitmap passado para ID2D1SourceTransform::D RAW.

## <a name="possible-fixes"></a>Possíveis correções

Para atualizar esse tipo de bitmap, use ID2D1Bitmap1:: Map.
