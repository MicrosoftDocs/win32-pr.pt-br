# <a name="d1151-unsupported-array-selection"></a>D1151: seleção de matriz sem suporte

O IDXGISurface2 fornecido pertence a uma matriz, mas não há suporte para fatias de matriz nesse nível de dispositivo (D3D9).






 

## <a name="possible-causes"></a>Possíveis causas

As texturas do Interopped Direct3D podem ser arranjadas; Direct2D só funcionará em uma fatia de matriz de cada vez. Outras fatias de matriz da textura pai não serão afetadas ou renderizadas por Direct2D. No entanto, os dispositivos de nível de recurso 9 não dão suporte à seleção de fatia de matriz e não podem usar esse recurso de interoperabilidade.

## <a name="possible-fixes"></a>Possíveis correções

Para aplicativos que devem ser executados corretamente em dispositivos de nível de recurso 9, considere modificar a lógica de renderização para evitar a necessidade de interoperar texturas em matrizes. Se possível, interopere um intermediário em vez disso. Para minimizar os efeitos de trabalhar em volta dessa limitação, o aplicativo pode optar por implementá-lo como um caminho de fallback específico de nível de recurso 9.
