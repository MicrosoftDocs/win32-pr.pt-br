# <a name="d1227-print-rasterization-event"></a>D1227: Evento de rasterização de impressão

Essa mensagem é emitida de Direct2D impressão. Indica que uma das seguintes operações causou a necessidade de rasterização:

-   Uso de uma camada D2D1 \_ LAYER \_ OPTIONS1 \_ INITIALIZE \_ FROM \_ BACKGROUND.
-   Uso do modo de combinação primitivo SOURCE COPY ou qualquer modo de combinação diferente de SOURCE OVER.
-   Uso de determinados modos compostos, como BOUNDED \_ SOURCE \_ COPY, DESTINATION \_ OUT, SOURCE \_ ATOP, XOR, PLUS ou INVERT.
-   Uso de [**DrawGdiMetafile.**](id2d1devicecontext-drawgdimetafile-overload.md)
-   Uso de uma transformação 4x4 (ou perspectiva) em uma [**chamada DrawBitmap.**](id2d1devicecontext-drawbitmap-overload.md)

Para obter um desempenho ideal, essas operações devem ser evitadas, se possível.
