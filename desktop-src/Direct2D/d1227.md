# <a name="d1227-print-rasterization-event"></a>D1227: evento de rasterização de impressão

Essa mensagem é emitida de Direct2D Print. Isso indica que uma das seguintes operações causou uma necessidade de rasterização:

-   Uso de uma \_ camada d2d1 \_ OPTIONS1 \_ inicializar \_ da \_ camada de plano de fundo.
-   Uso da cópia de origem do modo de mesclagem primitiva ou qualquer modo de mesclagem diferente de origem.
-   Uso de determinados modos compostos, como \_ \_ cópia de origem limitada, destino \_ de saída, origem sobre \_ , Xor, mais ou inverter.
-   Uso de [**DrawGdiMetafile**](id2d1devicecontext-drawgdimetafile-overload.md).
-   Uso de uma transformação 4x4 (ou, Perspective) em uma chamada [**DrawBitmap**](id2d1devicecontext-drawbitmap-overload.md) .

Para obter um desempenho ideal, essas operações devem ser evitadas, se possível.
