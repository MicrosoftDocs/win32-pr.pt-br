# <a name="d1200-pushlayer-with-multisampled-target"></a>D1200: PushLayer com Destino Multisampled

Pushlayer/Poplayer foi usado com um destino multisampled. A chamada ainda será emitida, mas o resultado renderizado pode não representar o padrão multisample esperado.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo interoperou uma textura Direct3D multisampled como um destino Direct2D e chamou PushLayer no destino.

## <a name="possible-fixes"></a>Possíveis correções

Evite interoperar texturas multisampladas se a funcionalidade da camada for importante. O uso de camadas ou clipes com esses tipos de recursos pode incorrer em uma penalidade de desempenho ou resultados de renderização inesperados. Se possível, use um meio alternativo de controlar a opacidade ou mascaramento em vez de camadas. Ou use um intermediário em vez de um recurso multisampled. Se o recurso multisampled for destinado ao conteúdo 2D de suavização, considere se Direct2D suavização por primitivo é suficiente.
