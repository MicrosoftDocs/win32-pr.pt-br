# <a name="d1200-pushlayer-with-multisampled-target"></a>D1200: PushLayer com destino de uma amostra

Pushlayer/Poplayer foi usado com um destino de uma amostra. A chamada ainda será emitida, mas o resultado renderizado poderá não representar o padrão de multiamostração esperado.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo interoperau uma textura multiamostrada do Direct3D como um destino Direct2D e chamou PushLayer no destino.

## <a name="possible-fixes"></a>Possíveis correções

Evite a interoperação de texturas com várias amostras se a funcionalidade da camada for importante. Usar camadas ou clipes com esses tipos de recursos pode incorrer em uma penalidade de desempenho ou resultados de renderização inesperados. Se possível, use um meio alternativo de controlar a opacidade ou mascaramento em vez de camadas. Ou então, use um intermediário em vez de um recurso de uma amostra. Se o recurso de multiamostrar for destinada ao conteúdo 2D de suavização de alias, considere se a suavização interna por primitivo Direct2D's é suficiente.
