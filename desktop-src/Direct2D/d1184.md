# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: CreateWicBitmapRenderTarget incompatível com o tipo de destino HW

É inválido passar o sinalizador D2D1 \_ RENDER TARGET TYPE HARDWARE para \_ \_ \_ CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Possíveis causas

Quando um aplicativo chamado ID2D1Factory::CreateWicBitmapRenderTarget, o campo de tipo do argumento D2D1 RENDER TARGET PROPERTIES era \_ \_ \_ D2D1 \_ D2D1 \_ RENDER TARGET TYPE \_ \_ \_ HARDWARE.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o tipo de destino usado durante a criação de um destino de renderização interocalado por WIC é SOFTWARE ou DEFAULT. Direct2D normalmente reserva a semântica HARDWARE para descrever a interopabilidade do Direct3D.
