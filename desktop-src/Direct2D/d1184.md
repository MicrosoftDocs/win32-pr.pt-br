# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: CreateWicBitmapRenderTarget incompatível com o tipo de destino de HW

É inválido passar o sinalizador D2D1 processar o \_ \_ \_ tipo de destino \_ hardware para CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Possíveis causas

Quando um aplicativo chamado ID2D1Factory:: CreateWicBitmapRenderTarget, o campo de tipo do \_ argumento de propriedades de destino de RENDERIZAÇÃO d2d1 \_ \_ era d2d1 \_ d2d1 de tipo de destino de \_ renderização \_ \_ \_ .

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o tipo de destino usado ao criar um destino de renderização WIC-interopped é SOFTWARE ou padrão. Direct2D normalmente reserva a semântica de HARDWARE para descrever a interoperabilidade do Direct3D.
