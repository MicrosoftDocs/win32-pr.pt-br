# <a name="direct2d-samples"></a>Exemplos de Direct2D

Os exemplos a seguir demonstram a API Direct2D dados.

## <a name="windows-10-and-microsoft-visual-studio-2015-samples"></a>Windows 10 exemplos Microsoft Visual Studio 2015

| Amostra | Descrição |
|-|-|
| [Amostra de efeitos de imagem personalizados do Direct2D](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DCustomEffects) | Este exemplo demonstra como implementar efeitos [personalizados usando](custom-effects.md) pixel HLSL, vértice e sombreadores de computação. |
| [Amostra de malha de gradiente do Direct2D](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DGradientMesh) | Este exemplo demonstra como insinuar e renderizar uma malha de gradiente Direct2D. Esses exemplos usam o método auxiliar fornecido pelo Direct2D para criar uma malha que consiste em dois patches tensores que compartilham um lado. |
| [Amostra de ajuste de fotos do Direct2D](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/D2DPhotoAdjustment) | Este exemplo demonstra como criar um visualizador de fotos e um editor usando Direct2D e Direct2D Efeitos. |
