# <a name="d1201-pushaxisalignedclip-with-antialiased-multisampled-target"></a>D1201: PushAxisAlignedClip com destino multisampled antialiased

PushAxisAlignedClip foi usado com um destino multisampled. A chamada ainda será emitida, mas poderão surgir artefatos se os limites do retângulo de clipe não estão alinhados a pixels. Além disso, o resultado renderizado pode não representar o padrão multisample esperado.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo interoperou uma textura Direct3D multisampled como um destino Direct2D e chamou PushAxisAlignedClip no destino.

## <a name="possible-fixes"></a>Possíveis correções

Evite interoperar texturas multisampled se a funcionalidade de clipe for importante. O uso de camadas ou clipes com esses tipos de recursos pode incorrer em uma penalidade de desempenho ou resultados de renderização inesperados. Isso é particularmente verdadeiro se os limites do clipe não estão alinhados a pixels. Se possível, use um meio alternativo de mascarar conteúdo em vez de um clipe, como uma tesoura Direct3D. Ou use um intermediário em vez de um recurso multisampled. Se o recurso multisampled for destinado ao conteúdo 2D de suavização, considere se Direct2D suavização por primitivo é suficiente.
