# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: modo alfa desconhecido não permitido

O \_ modo alfa d2d1 \_ \_ desconhecido não é permitido como um parâmetro para essa API.



|             |         |
|-------------|---------|
| Nível de erro | Aviso |



 

## <a name="possible-causes"></a>Possíveis causas

O \_ \_ especificador desconhecido de modo alfa é permitido em alguns contextos, como uma descrição de formato para ID2D1RenderTarget:: CreateCompatibleRenderTarget. No entanto, da mesma forma que o \_ formato dxgi \_ é desconhecido, Direct2D geralmente requer que os formatos sejam totalmente especificados.

## <a name="possible-fixes"></a>Possíveis correções

Ao decidir sobre um especificador de formato, considere o tratamento da superfície alfa para usar um \_ modo alfa diferente do \_ modo alfa \_ desconhecido. Use \_ \_ o modo alfa d2d1 \_ ignore se a superfície não usar alfa. Consulte comentários no [**\_ \_ modo alfa d2d1**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
