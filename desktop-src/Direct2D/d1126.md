# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: Modo alfa desconhecido não permitido

D2D1 \_ MODO ALFA DESCONHECIDO não é permitido como um parâmetro para essa \_ \_ API.





 

## <a name="possible-causes"></a>Possíveis causas

O especificador ALPHA MODE UNKNOWN é permitido em alguns contextos, como uma descrição de formato para \_ \_ ID2D1RenderTarget::CreateCompatibleRenderTarget. No entanto, da mesma forma que o FORMATO DXGI DESCONHECIDO, Direct2D geralmente requer \_ \_ que os formatos sejam totalmente especificados.

## <a name="possible-fixes"></a>Possíveis correções

Ao decidir sobre um especificador de formato, considere o tratamento da superfície de alfa para usar um MODO ALFA diferente \_ de MODO \_ ALFA \_ DESCONHECIDO. Use O MODO ALFA D2D1 \_ IGNORE se a superfície não usar \_ \_ alfa. Consulte Comentários no [**MODO \_ ALFA \_ D2D1**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
