# <a name="d1240-source-tranform-still-mapped"></a>D1240: fonte transformar ainda mapeada

O bitmap passado para ID2D1SourceTransform::D RAW nunca foi não mapeado.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo chamou ID2D1Bitmap1:: Map, mas não chamou ID2D1Bitmap1:: remapear em um bitmap passado para ID2D1SourceTransform::D RAW.

## <a name="possible-fixes"></a>Possíveis correções

Certifique-se de que o desmapeamento seja chamado sempre que o mapa for chamado.
