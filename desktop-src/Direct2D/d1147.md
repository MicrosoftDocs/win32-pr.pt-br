# <a name="d1147-print-control-closed-without-addpage"></a>D1147: controle de impressão fechado sem AddPage

Direct2D Print: fechar é chamado sem chamadas AddPage anteriores






 

## <a name="possible-causes"></a>Possíveis causas

Essa mensagem pode ocorrer se um [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) for fechado sem nenhuma invocação anterior para [**ID2D1PrintControl:: AddPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage). Direct2D Print considera um controle de impressão sem que as páginas sejam erradas e o comando [**ID2D1PrintControl:: Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) correspondente retornará um erro.

## <a name="possible-fixes"></a>Possíveis correções

Para endereçar a mensagem, adicione uma página ao controle de impressão antes de fechá-la. Se o aplicativo não precisar imprimir nenhum conteúdo, evite fechar o controle de impressão ou criá-lo.
