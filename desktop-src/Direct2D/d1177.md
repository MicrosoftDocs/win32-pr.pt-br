# <a name="d1177-unsupported-wic-pixel-format"></a>D1177: formato de pixel do WIC sem suporte

O formato de pixel do WIC fornecido não tem suporte do Direct2D. Em particular, Direct2D não dá suporte a bitmaps de WIC que usam alfa reta, a menos que a profundidade seja 8bpp.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo tentou interoperar um recurso de WIC cujo formato não é suportado pelo Direct2D. Normalmente, esses formatos são aqueles com Straight-Alpha e cuja profundidade não é de 8 bits por plano.

## <a name="possible-fixes"></a>Possíveis correções

Considere criar o recurso de WIC usando um formato alternativo para interoperabilidade com Direct2D ou convertê-lo.
