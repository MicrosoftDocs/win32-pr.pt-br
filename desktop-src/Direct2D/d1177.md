# <a name="d1177-unsupported-wic-pixel-format"></a>D1177: Formato de pixel WIC sem suporte

Não há suporte para o formato de pixel WIC fornecido pelo Direct2D. Em particular, Direct2D dá suporte a bitmaps WIC que usam alfa reto, a menos que a profundidade seja 8bpp.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo tentou interoperar um recurso WIC cujo formato não é suportado pelo Direct2D. Normalmente, esses formatos são aqueles com alfa reto e cuja profundidade não é de 8 bits por plano.

## <a name="possible-fixes"></a>Possíveis correções

Considere criar o recurso WIC usando um formato alternativo para interoperabilidade com Direct2D ou convertê-lo.
