# <a name="d1152-command-list-closed-twice"></a>D1152: lista de comandos fechada duas vezes

Foi feita uma tentativa de fechar uma lista de comandos pela segunda vez.






 

## <a name="possible-causes"></a>Possíveis causas

Depois de preencher uma lista de comandos, ela só precisa ser fechada uma vez. Direct2D considera as chamadas subsequentes para ID2D1CommandList:: Close para ser um erro.

## <a name="possible-fixes"></a>Possíveis correções

Para evitar essa mensagem, certifique-se de que uma lista de comandos seja fechada apenas uma vez. Se um aplicativo usar retornos de chamada ou caminhos de código recursivos para interagir com listas de comandos, verifique se seu comportamento está correto.
