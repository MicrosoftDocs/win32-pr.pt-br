# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: A opção de mapa de gravação requer bitmap que pode ser escrito pela CPU

D2D1 \_ MAP OPTIONS WRITE só é válido em um bitmap que \_ pode ser escrito pela \_ CPU






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá enviar essa mensagem se um aplicativo chamar ID2D1Bitmap1::Map com o atributo WRITE DE OPÇÕES DE MAPA D2D1, mas o bitmap não for escrito pela \_ \_ \_ CPU.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo exigir acesso de gravação usando ID2D1Bitmap1::Map, verifique se o recurso Direct3D subjacente tem o atributo D3D11 \_ CPU \_ ACCESS \_ WRITE.
