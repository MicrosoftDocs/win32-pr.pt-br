# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: a opção de mapa de gravação requer bitmap gravável de CPU

\_A gravação de opções de mapa d2d1 \_ \_ só é válida em um bitmap gravável de CPU






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá gerar essa mensagem se um aplicativo chamar ID2D1Bitmap1:: MAP com o \_ atributo de \_ gravação opções de mapa d2d1 \_ , mas o bitmap não for gravável de CPU.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo exigir acesso de gravação usando ID2D1Bitmap1:: Map, verifique se o recurso do Direct3D subjacente tem o \_ atributo de gravação de acesso da CPU D3D11 \_ \_ .
