# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183: CreateWicBitmapRenderTarget com nível de recurso incompatível

É inválido passar um nível de recurso Direct2D diferente do padrão de \_ nível de recurso d2d1 \_ \_ para CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Possíveis causas

Quando um aplicativo chamado ID2D1Factory:: CreateWicBitmapRenderTarget, o campo minLevel do argumento de \_ Propriedades de destino de RENDERIZAÇÃO d2d1 \_ \_ não era o \_ padrão de nível de recurso d2d1 \_ \_ .

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o nível de recurso usado ao criar um destino de renderização WIC-interopped é o padrão. A interoperabilidade com o WIC envolve recursos somente de software; a semântica de nível de recurso é relevante para o comportamento de hardware.
