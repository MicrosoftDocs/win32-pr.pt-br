# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: a opção Descartar Mapa requer textura dinâmica

D2D1 \_ MAP OPTIONS DISCARD só é válido com um \_ \_ bitmap criado com uma textura dinâmica.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá enviar essa mensagem se um aplicativo chamar ID2D1Bitmap1::Map com o atributo D2D1 MAP OPTIONS DISCARD, mas o recurso Direct3D subjacente para o bitmap não for atribuído com \_ \_ \_ D3D11 \_ USAGE \_ DYNAMIC.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo exigir acesso a um bitmap usando ID2D1Bitmap1::Map, verifique se o recurso Direct3D subjacente tem o atributo D3D11 \_ USAGE \_ DYNAMIC.
