# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: opção de mapeamento de descarte requer textura dinâmica

\_As opções de mapa d2d1 \_ \_ descartar são válidas somente com um bitmap criado a partir de uma textura dinâmica.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá gerar essa mensagem se um aplicativo chamar ID2D1Bitmap1:: MAP com o \_ atributo de \_ descartar opções do mapa d2d1 \_ , mas o recurso do Direct3D subjacente para o bitmap não for atribuído com o uso de D3D11 \_ \_ dinâmico.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo exigir acesso a um bitmap usando ID2D1Bitmap1:: Map, verifique se o recurso do Direct3D subjacente tem o \_ atributo dinâmico de uso de D3D11 \_ .
