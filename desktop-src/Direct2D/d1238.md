# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: SRGB de Canal Único de Transformação de Origem

ID2D1SourceTransform não permite que D2D1 BUFFER PRECISION SRGB seja combinado com \_ \_ \_ D2D1 \_ CHANNEL \_ DEPTH \_ 1.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou D2D1 BUFFER PRECISION SRGB e \_ \_ \_ D2D1 \_ CHANNEL DEPTH \_ \_ 1 em um ID2D1SourceTransform.

## <a name="possible-fixes"></a>Possíveis correções

D2D1 \_ CHANNEL DEPTH DEFAULT ou \_ \_ D2D1 \_ CHANNEL DEPTH \_ \_ 4 devem ser usados em vez disso.
