# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: origem de transformação de canal único SRGB

ID2D1SourceTransform não permite que \_ \_ a precisão de buffer d2d1 \_ de sRGB seja combinada com a profundidade de canal de d2d1 \_ \_ \_ 1.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou a \_ precisão de buffer d2d1 \_ \_ sRGB e a profundidade de \_ canal d2d1 \_ \_ 1 em um ID2D1SourceTransform.

## <a name="possible-fixes"></a>Possíveis correções

O \_ \_ \_ padrão de profundidade de canal d2d1 ou a profundidade de canal d2d1 \_ \_ \_ 4 deve ser usado em vez disso.
