# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191: a opção de bitmap compatível com GDI foi usada incorretamente

\_ \_ As opções de bitmap d2d1 \_ \_ compatíveis com GDI devem ser combinadas com o destino de opções de \_ bitmap d2d1 \_ \_






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou uma combinação incompatível dos \_ sinalizadores de opções de bitmap d2d1 \_ .

## <a name="possible-fixes"></a>Possíveis correções

Adicione \_ o destino de opções de bitmap d2d1 \_ \_ para os \_ sinalizadores de opções de bitmap d2d1 \_ .
