# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191: A opção de bitmap compatível com GDI foi usada incorretamente

D2D1 \_ OPÇÕES DE BITMAP \_ \_ GDI COMPATIBLE devem ser combinadas com DESTINO DE OPÇÕES \_ DE \_ BITMAP D2D1 \_ \_






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou uma combinação incompatível dos sinalizadores D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possíveis correções

Adicione D2D1 \_ BITMAP \_ OPTIONS TARGET aos \_ sinalizadores D2D1 \_ BITMAP \_ OPTIONS.
