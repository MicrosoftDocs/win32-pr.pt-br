# <a name="d1133-buffer-precision-unknown-not-allowed"></a>D1133: precisão de buffer desconhecida não permitida

\_A precisão de buffer d2d1 \_ \_ desconhecida não é permitida como um parâmetro para essa API.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D permite que um aplicativo especifique uma precisão de buffer desejada para algumas operações. Por exemplo, a precisão usada para o pipeline de geração de imagens é especificada usando ID2D1DeviceContext:: SetRenderingControls. O \_ especificador de precisão de buffer \_ desconhecido permite que um aplicativo tenha sido anterior a essa configuração e Direct2D usará um valor padrão, mas esse especificador não é permitido para algumas operações. Em particular, pode não ser permitido durante a criação de uma coleção de parada de gradiente.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o \_ especificador de precisão de buffer \_ desconhecido não é usado com ID2D1DeviceContext:: CreateGradientStopCollection. Usar um valor de precisão conhecido; Considere o uso da precisão do formato de superfície de destino para a operação.
