# <a name="d1198-copy-to-read-only-bitmap"></a>D1198: copiar para Read-Only bitmap

Foi feita uma tentativa de copiar dados para um bitmap somente leitura ou imutável.






 

## <a name="possible-causes"></a>Possíveis causas

Foi feita uma tentativa de copiar dados para um bitmap somente leitura ou imutável.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se os bitmaps gravados em não são criados como imutáveis.
