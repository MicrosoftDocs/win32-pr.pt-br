# <a name="d1202-copyfromrendertarget-with-multisampled-source"></a>D1202: CopyfromRenderTarget com fonte multiamostrada

CopyFromRenderTarget foi usado com um renderTarget multiamostrado. A chamada ainda será emitida, mas o resultado copiado poderá não representar o padrão de multiamostração esperado.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo interoperau uma textura de multiamostrada do Direct3D como um destino Direct2D e chamou CopyFromRenderTarget no destino

## <a name="possible-fixes"></a>Possíveis correções

Evite a interoperação de texturas de multiamostras como destinos se a funcionalidade CopyFromRenderTarget for importante. O resultado copiado pode não representar o padrão de multiamostração esperado.
