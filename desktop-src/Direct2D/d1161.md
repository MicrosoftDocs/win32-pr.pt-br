# <a name="d1161-copyfrommemory-requires-cpu-readable-bitmap"></a>D1161: CopyFromMemory requer bitmap legível da CPU

Este método não é permitido em um bitmap criado com [**\_ Opções de bitmap d2d1 \_ \_ \_ leitura da CPU**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_bitmap_options).






 

## <a name="possible-causes"></a>Possíveis causas

Essa mensagem pode ocorrer durante uma chamada para [**ID2D1Bitmap:: CopyFromMemory**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory). Se o bitmap tiver sido criado usando \_ Opções de bitmap d2d1 \_ \_ \_ de leitura de CPU, Direct2D relatará um erro e a camada de depuração produzirá essa mensagem.

## <a name="possible-fixes"></a>Possíveis correções

Se for necessário que um aplicativo chame [**ID2D1Bitmap:: CopyFromMemory**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory), certifique-se de que o bitmap não seja criado como legível pela CPU. Se não for possível alterar os atributos de superfície do bitmap, considere se há uma operação de renderização equivalente para o resultado desejado.
