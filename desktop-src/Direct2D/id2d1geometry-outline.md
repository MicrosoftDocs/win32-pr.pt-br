## <a name="examples"></a><span data-ttu-id="2578e-125">Exemplos</span><span class="sxs-lookup"><span data-stu-id="2578e-125">Examples</span></span>

<span data-ttu-id="2578e-126">O código a seguir mostra como usar a **estrutura de tópicos** para construir uma geometria equivalente sem interseções automáticas.</span><span class="sxs-lookup"><span data-stu-id="2578e-126">The following code shows how to use **Outline** to construct an equivalent geometry with no self-intersections.</span></span> <span data-ttu-id="2578e-127">Ele usa a tolerância de nivelamento padrão e, portanto, não deve ser usado com geometrias muito pequenas.</span><span class="sxs-lookup"><span data-stu-id="2578e-127">It uses the default flattening tolerance and hence should not be used with very small geometries.</span></span>


```C++
HRESULT D2DOutline(
    ID2D1Geometry *pGeometry,
    ID2D1Geometry **ppGeometry
    )
{
    HRESULT hr;
    ID2D1Factory *pFactory = NULL;
    pGeometry->GetFactory(&pFactory);

    ID2D1PathGeometry *pPathGeometry = NULL;
    hr = pFactory->CreatePathGeometry(&pPathGeometry);

    if (SUCCEEDED(hr))
    {
        ID2D1GeometrySink *pSink = NULL;
        hr = pPathGeometry->Open(&pSink);

        if (SUCCEEDED(hr))
        {
            hr = pGeometry->Outline(NULL, pSink);

            if (SUCCEEDED(hr))
            {
                hr = pSink->Close();

                if (SUCCEEDED(hr))
                {
                    *ppGeometry = pPathGeometry;
                    (*ppGeometry)->AddRef();
                }
            }
            pSink->Release();
        }
        pPathGeometry->Release();
    }

    pFactory->Release();

    return hr;
}
```
