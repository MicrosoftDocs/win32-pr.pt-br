# <a name="d1176-negative-dpi"></a>D1176: DPI negativo

É um erro especificar um DPI negativo ao criar um rendertarget.






 

## <a name="possible-causes"></a>Possíveis causas

Um aplicativo tentou especificar um DPI negativo, seja por meio do uso de ID2D1RenderTarget::SetDpi ou criando um destino com uma propriedade DPI negativa.

## <a name="possible-fixes"></a>Possíveis correções

Escolha valores válidos e não negativos para DPI ao criar ou modificar destinos de renderização. Se um DPI padrão for aceitável, evite chamar SetDpi ou especificar uma opção de DPI ao criar um destino. Especificar um DPI de 0 é equivalente a usar um DPI padrão.
