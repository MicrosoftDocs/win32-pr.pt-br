# <a name="d1176-negative-dpi"></a>D1176: DPI negativa

É um erro especificar um DPI negativo ao criar um renderTarget.






 

## <a name="possible-causes"></a>Possíveis causas

Um aplicativo tentou especificar um DPI negativo, usando o uso de ID2D1RenderTarget:: SetDpi ou criando um destino com uma propriedade DPI negativa.

## <a name="possible-fixes"></a>Possíveis correções

Escolha valores válidos e não negativos para DPI ao criar ou modificar destinos de renderização. Se um DPI padrão for aceitável, evite chamar SetDpi ou especificar uma opção de DPI ao criar um destino. A especificação de um DPI 0 é equivalente ao uso de um DPI padrão.
