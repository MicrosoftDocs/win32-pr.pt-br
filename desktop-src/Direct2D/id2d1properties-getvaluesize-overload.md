# <a name="id2d1propertiesgetvaluesize-methods"></a>Métodos ID2D1Properties:: GetValues

Obtém o tamanho do valor da propriedade em bytes, usando o índice de propriedade.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                       | Descrição                                                                                                     |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**GetValues (U)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvaluesize(u))     | Obtém o tamanho do valor da propriedade em bytes, usando o índice de propriedade. Essa é uma sobrecarga de modelo.<br/> |
| [**GetValues (UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvaluesize(uint32)) | Obtém o tamanho do valor da propriedade em bytes, usando o índice de propriedade.<br/>                              |
