# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203: CopyFromBitmap com fonte multiamostrada

CopyFromBitmap foi usado com um bitmap de origem com uma amostra. A chamada ainda será emitida, mas o resultado copiado poderá não representar o padrão de multiamostração esperado.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo interoperau uma textura multiamostrada do Direct3D como um bitmap Direct2D e chamou CopyFromBitmap no bitmap.

## <a name="possible-fixes"></a>Possíveis correções

Evite a interoperação de texturas com multiamostras como bitmaps se a funcionalidade CopyFromBitmap for importante. O resultado copiado pode não representar o padrão de multiamostração esperado.
