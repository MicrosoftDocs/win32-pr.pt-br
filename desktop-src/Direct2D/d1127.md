# <a name="d1127-alpha-mode-mismatch"></a>D1127: incompatibilidade de modo alfa

O modo alfa passado para essa API não é compatível com o formato de pixel da \[ *superfície* \] .

## <a name="placeholders"></a>Espaços reservados

<dl> <dt>

<span id="surface"></span><span id="SURFACE"></span>*tona*
</dt> <dd>

A superfície do Direct3D que não oferece suporte ao modo alfa.

</dd> </dl> 

|             |             |
|-------------|-------------|
| Nível de erro | Informações do |



 

## <a name="possible-causes"></a>Possíveis causas

Operações como ID2D1RenderTarget:: CreateBitmapFromWicBitmap ou ID2D1RenderTarget:: CreateDxgiSurfaceRendertarget exigem uma superfície e um modo alfa de superfície especificado. Direct2D verifica se o modo alfa especificado é compatível com a superfície. Se não for compatível, o Direct2D produzirá essa mensagem.

## <a name="possible-fixes"></a>Possíveis correções

Modifique o aplicativo para usar um \_ modo alfa que corresponda ou seja compatível com o modo alfa da superfície subjacente. Consulte [formatos de pixel com suporte e modos alfa](supported-pixel-formats-and-alpha-modes.md) para obter diretrizes adicionais sobre os modos alfa expostos por Direct2D.
