# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153: as listas de comandos fechadas não podem ser definidas como destinos

Foi feita uma tentativa de definir uma lista de comandos fechada em um destino.






 

## <a name="possible-causes"></a>Possíveis causas

Depois que uma lista de comandos é fechada, um aplicativo não deve tentar defini-la como um destino. Direct2D considera que essa chamada para ID2D1DeviceContext:: SetTarget é um erro porque o conteúdo de uma lista de comandos não pode ser alterado depois de ser fechado. Definir uma lista de comandos como um destino indica uma intenção de renderá-la.

## <a name="possible-fixes"></a>Possíveis correções

Para endereçar essa mensagem, não chame ID2D1DeviceContext:: SetTarget com uma lista de comandos que foi fechada. Se for necessário usar uma lista de comandos com conteúdo novo e diferente, crie uma nova lista de comandos.
