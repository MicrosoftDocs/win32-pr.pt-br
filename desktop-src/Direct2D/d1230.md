# <a name="d1230-unknown-buffer-precision-support"></a>D1230: suporte à precisão do buffer desconhecido

Uma chamada para IsBufferPrecisionSupported foi feita com \_ precisão de buffer d2d1 \_ \_ desconhecida. Essa API retorna false, embora \_ \_ a precisão de buffer d2d1 \_ desconhecida possa ser válida como uma entrada para algumas APIs.






 

## <a name="possible-causes"></a>Possíveis causas

Uma chamada para IsBufferPrecisionSupported foi feita com \_ precisão de buffer d2d1 \_ \_ desconhecida.

## <a name="possible-fixes"></a>Possíveis correções

Determine se \_ a precisão de buffer d2d1 \_ \_ desconhecida é uma precisão de buffer válida para uma determinada API lendo a documentação para essa API em vez de tentar determinar isso programaticamente.
