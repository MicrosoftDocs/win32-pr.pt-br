# <a name="d1230-unknown-buffer-precision-support"></a>D1230: Suporte de precisão de buffer desconhecido

Uma chamada para IsBufferPrecisionSupported foi feita com D2D1 \_ BUFFER \_ PRECISION \_ UNKNOWN. Essa API retorna false, embora D2D1 BUFFER PRECISION UNKNOWN possa ser válido como \_ uma entrada para algumas \_ \_ APIs.






 

## <a name="possible-causes"></a>Possíveis causas

Uma chamada para IsBufferPrecisionSupported foi feita com D2D1 \_ BUFFER \_ PRECISION \_ UNKNOWN.

## <a name="possible-fixes"></a>Possíveis correções

Determine se D2D1 BUFFER PRECISION UNKNOWN é uma precisão de buffer válida para uma determinada API lendo a documentação dessa API em vez de tentar \_ \_ determinar isso \_ programaticamente.
