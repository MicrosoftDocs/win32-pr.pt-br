# <a name="d1190-cannotdraw-bitmap-flag-was-used-incorrectly"></a>D1190: Não é possível redesenhar sinalizador de bitmap usado incorretamente

D2D1 OPÇÕES DE BITMAP NÃO PODE DESENHAR DEVEM ser combinadas com OPÇÕES \_ \_ \_ \_ D2D1 \_ OPÇÕES DE \_ \_ BITMAP CPU READ ou \_ D2D1 \_ BITMAP OPTIONS \_ \_ TARGET





 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou uma combinação incompatível dos sinalizadores D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possíveis correções

Adicionar OPÇÕES DE BITMAP D2D1 LEITURA DA \_ CPU ou DESTINO DE OPÇÕES DE \_ \_ \_ BITMAP D2D1 \_ às OPÇÕES \_ \_ D2D1 \_ BITMAP \_
