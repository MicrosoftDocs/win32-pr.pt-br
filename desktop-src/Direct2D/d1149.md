# <a name="d1149-print-detected-an-invalid-unicode-character"></a>D1149: Print detectou um caractere Unicode inválido

Direct2D Print: caractere Unicode inválido detectado na \_ Descrição de execução de GLIFO DWRITE \_ \_ . cadeia de caracteres. Ele foi substituído por espaço (0x20).






 

## <a name="possible-causes"></a>Possíveis causas

A impressão Direct2D considera caracteres inválidos anotados na [documentação Unicode](https://unicode.org/faq/utf_bom.html#utf16-7), bem como determinados caracteres ASCII fora do intervalo visível, para que não sejam imprimíveis. O uso desses caracteres não causa um erro. Em vez disso, o Direct2D Print substituirá esses caracteres por espaços.

## <a name="possible-fixes"></a>Possíveis correções

Para evitar essa notificação, não use caracteres Unicode malformados ou ASCII não imprimível ao desenhar texto para capacidade de impressão.
