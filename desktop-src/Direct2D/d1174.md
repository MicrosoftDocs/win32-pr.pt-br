# <a name="d1174-pushlayer-to-command-list-will-ignore-layer-object"></a>D1174: PushLayer para a lista de comandos ignorará o objeto de camada

Ao enviar por push uma camada (não nula) para uma lista de comandos, o objeto da camada será ignorado.



|             |         |
|-------------|---------|
| Nível de erro | Aviso |



 

## <a name="possible-causes"></a>Possíveis causas

Quando um comando PushLayer usa um objeto ID2D1Layer e o destino é uma lista de comandos, o Direct2D se comportará como se nenhuma camada fosse especificada e não imporá limites fixo na camada se a lista de comandos for renderizada posteriormente.

## <a name="possible-fixes"></a>Possíveis correções

Os aplicativos que antecipam os comandos de streaming para uma lista de comandos devem considerar o uso de objetos de camada nula, ou então estar ciente de que as configurações de restrição de tamanho especificadas no objeto ID2D1Layer não serão respeitadas.
