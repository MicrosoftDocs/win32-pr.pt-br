# <a name="d1138-invalid-command-list"></a>D1138: lista de comandos inválida

Uma lista de comandos foi referenciada inválida (ocorreu um erro anteriormente durante a população da lista de comandos).






 

## <a name="possible-causes"></a>Possíveis causas

Se ocorrerem erros durante a população de uma lista de comandos, um aplicativo não poderá usá-lo para operações de desenho ou streaming. Quando a lista de comandos estiver fechada, ID2D1CommandList:: Close retornará um código de erro para indicar que a lista de comandos não foi populada corretamente. Geralmente, as operações ID2D1RenderTarget:: EndDraw ou flush usadas durante a população retornarão códigos de erro também, fornecendo informações adicionais. Os problemas que causam um erro durante ID2D1RenderTarget:: EndDraw, como a tentativa de operações de desenho que usam recursos de outro domínio ou uma geometria de caminho não fechada, normalmente impedirão que uma lista de comandos seja fechada com êxito.

## <a name="possible-fixes"></a>Possíveis correções

Para garantir que as listas de comandos sejam preenchidas corretamente, solucione erros que ocorram em ID2D1CommandList:: Close e ID2D1RenderTarget:: EndDraw ou flush durante a população da lista de comandos. Certifique-se de que o aplicativo não ignore o código de retorno de ID2D1CommandList:: Close.
