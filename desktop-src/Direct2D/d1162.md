# <a name="d1162-fillmesh-requires-aliased-rendering"></a>D1162: FillMesh requer renderização com alias

FillMesh requer que o modo AntiAlias seja definido como \_ alias do modo ANTIALIAS d2d1 \_ \_ .






 

## <a name="possible-causes"></a>Possíveis causas

Quando um aplicativo chama ID2D1RenderTarget:: FillMesh, o modo AntiAlias do renderTarget envolvido deve ser o \_ modo de ANTIALIAS d2d1 com \_ \_ alias. O comportamento padrão de um destino de renderização é usar a suavização, portanto, é necessário chamar ID2D1RenderTarget:: SetAntialiasMode antes de preencher malhas.

## <a name="possible-fixes"></a>Possíveis correções

Use ID2D1RenderTarget:: SetAntialiasMode para garantir que um renderTarget use a renderização com alias ao preencher malhas. Se as operações subsequentes exigirem a suavização, defina o modo de antialias do destino de volta para o \_ modo de ANTIALIAS d2d1 \_ \_ por \_ primitiva. Verifique se as chamadas para ID2D1RenderTarget:: SetAntialiasMode e ID2D1RenderTarget:: FillMesh são emitidas no mesmo destino.
