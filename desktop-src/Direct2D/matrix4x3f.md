# <a name="matrix4x3f-class"></a>Classe Matrix4x3F

A classe **Matrix4x3F** representa uma matriz de 4 por 3 e fornece métodos de conveniência para a criação de matrizes.

## <a name="members"></a>Membros

A classe **Matrix4x3F** herda da [**\_ matriz d2d1 \_ 4x3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f). **Matrix4x3F** também tem estes tipos de membros:

-   [Construtores](#constructors)

### <a name="constructors"></a>Construtores

A classe **Matrix4x3F** tem esses construtores.



| Construtor                                                                                                                                                                                           | Descrição                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix4x3F()**](matrix4x3f-matrix4x3f--.md)                                                                                                                                                       | Instancia uma nova instância de uma classe **Matrix4x3F** não inicializada.<br/>                                                   |
| [**Matrix4x3F (FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, float, FLOAT, float, float, FLOAT, FLOAT) (FLOAT, FLOAT, float, float, FLOAT, float, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT)**](matrix4x3f-matrix4x3f-floats-.md) | Instancia uma nova instância de uma classe **Matrix4x3F** que é inicializada com todos os valores de matriz de ponto flutuante.<br/> |
