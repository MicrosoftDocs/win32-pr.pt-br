# <a name="d1009-unexpected-hardware-creation-error"></a>D1009: erro de criação de hardware inesperado

Um erro de erro inesperado \[  \] foi encontrado ao tentar criar um destino Direct3D.

<dl> <dt>

</dd> </dl> 

|             |         |
|-------------|---------|
| Nível de erro | Aviso |



 

## <a name="possible-causes"></a>Possíveis causas

-   O destino foi maior que o tamanho máximo do bitmap.
-   Memória de vídeo insuficiente, pois o destino de renderização é criado.
-   Não foi possível criar um dispositivo Direct3D.
-   O aplicativo está sendo executado em IA64.
