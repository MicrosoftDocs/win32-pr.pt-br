# <a name="d1231-unknown-dxgi-format-support"></a>D1231: suporte a formato DXGI desconhecido

Uma chamada para IsDxgiFormatSupported foi feita com \_ formato dxgi \_ desconhecido. Essa API retorna false, embora \_ o formato dxgi \_ desconhecido possa ser válido como uma entrada para algumas APIs.






 

## <a name="possible-causes"></a>Possíveis causas

Uma chamada para IsDxgiFormatSupported foi feita com \_ formato dxgi \_ desconhecido.

## <a name="possible-fixes"></a>Possíveis correções

Determine se \_ o formato dxgi \_ desconhecido é uma precisão de buffer válida para uma determinada API lendo a documentação para essa API em vez de tentar determinar isso programaticamente.
