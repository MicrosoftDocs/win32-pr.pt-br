# <a name="direct2d-helper-classes"></a>Direct2D classes auxiliares

O namespace D2D1 define as classes a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                       | Descrição                                                                                                                              |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**ColorF**](/windows/win32/api/d2d1helper/nl-d2d1helper-colorf)<br/>         | Define um conjunto de constantes que representa cores conhecidas e fornece métodos de conveniência para instanciar cores diferentes.<br/>  |
| [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)<br/> | A classe [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f) representa uma matriz 3 por 2 e fornece métodos de conveniência para a criação de matrizes.<br/> |
| [**Matrix4x3F**](matrix4x3f.md)<br/> | A classe [**Matrix4x3F**](matrix4x3f.md) representa uma matriz de 4 por 3 e fornece métodos de conveniência para a criação de matrizes.<br/> |
| [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)<br/> | A classe [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f) representa uma matriz 4 por 4 e fornece métodos de conveniência para a criação de matrizes.<br/> |
| [**Matrix5x4F**](matrix5x4f.md)<br/> | A classe [**Matrix5x4F**](matrix5x4f.md) representa uma matriz 5 por 4 e fornece métodos de conveniência para a criação de matrizes.<br/> |
