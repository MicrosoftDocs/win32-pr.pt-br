# <a name="d2d1_point_2u"></a>D2D1 \_ POINT \_ 2U

Representa um par de coordenadas x e y no espaço bidimensional.


```C++
typedef D2D_POINT_2U D2D1_POINT_2U;
```



## <a name="remarks"></a>Comentários

Os pontos Direct2D são representados pelas estruturas [**D2D1 \_ POINT \_ 2F**](d2d1-point-2f.md) ou **D2D1 \_ POINT \_ 2U.** Ambos contêm um par de coordenadas x e y no espaço bidimensional. A **estrutura D2D1 \_ POINT \_ 2F** armazena as coordenadas como valores **FLOAT** e a estrutura **D2D1 \_ POINT \_ 2U** as armazena como **valores UINT32.**

**D2D1 \_ POINT \_ 2U** é um novo nome para o tipo já definido [**D2D \_ POINT \_ 2U.**](/windows/desktop/api/dcommon/ns-dcommon-d2d_point_2u)
