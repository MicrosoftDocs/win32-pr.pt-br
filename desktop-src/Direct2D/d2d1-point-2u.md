# <a name="d2d1_point_2u"></a>D2D1 \_ ponto \_ 2U

Representa um par de coordenadas x e y em espaço bidimensional.


```C++
typedef D2D_POINT_2U D2D1_POINT_2U;
```



## <a name="remarks"></a>Comentários

Os pontos em Direct2D são representados pelas estruturas do [**d2d1 \_ Point \_ 2F**](d2d1-point-2f.md) ou do **d2d1 \_ Point \_ 2U** . Ambos contêm uma coordenada x e um par de coordenadas y em espaço bidimensional. A estrutura **d2d1 do \_ ponto \_ 2F** armazena as coordenadas como valores **float** e a **estrutura \_ \_ 2U do ponto d2d1** armazena-as como valores **UINT32** .

**D2d1 \_ O ponto \_ 2U** é um novo nome para o tipo já definido [**D2D \_ Point \_ 2U**](/windows/desktop/api/dcommon/ns-dcommon-d2d_point_2u).
