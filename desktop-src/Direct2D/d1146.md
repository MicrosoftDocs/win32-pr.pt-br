# <a name="d1146-invalid-render-controls"></a>D1146: controles de renderização inválidos

Os controles de renderização especificados tinham parâmetros inválidos e serão ignorados.






 

## <a name="possible-causes"></a>Possíveis causas

Essa mensagem pode ocorrer durante ID2D1DeviceContext:: SetRenderingControls se o aplicativo especificar um valor de precisão de buffer inválido ou um tamanho de bloco igual a zero.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o aplicativo usa um valor de precisão de buffer válido. Consulte [**\_ \_ precisão do buffer d2d1**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision). Verifique se o aplicativo especifica um tamanho de bloco diferente de zero.
