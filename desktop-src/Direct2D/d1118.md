# <a name="d1118-target-not-a-bitmap"></a>D1118: Destino não um bitmap

Falha em uma operação de desenho porque o destino é NULL ou uma lista de comandos.





 

## <a name="possible-causes"></a>Possíveis causas

Uma causa provável dessa mensagem é que o aplicativo chamado GetDC ou ReleaseDC enquanto o destino do contexto do dispositivo foi definido como NULL ou como uma lista de comandos.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo usar GetDC ou ReleaseDC exposto por IGdiInteropRenderTarget, verifique se o ID2D1DeviceContext correspondente tem um bitmap válido definido como seu destino.
