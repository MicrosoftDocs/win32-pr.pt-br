# <a name="d1118-target-not-a-bitmap"></a>D1118: destino não é um bitmap

Uma operação de desenho falhou porque o destino é nulo ou uma lista de comandos.





 

## <a name="possible-causes"></a>Possíveis causas

Uma causa provável dessa mensagem é que o aplicativo chamado GetDC ou ReleaseDC enquanto o destino do seu contexto de dispositivo foi definido como nulo ou para uma lista de comandos.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo usar GetDC ou ReleaseDC exposto pelo IGdiInteropRenderTarget, verifique se o ID2D1DeviceContext correspondente tem um bitmap válido definido como seu destino.
