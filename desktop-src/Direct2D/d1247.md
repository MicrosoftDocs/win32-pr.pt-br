# <a name="d1247-dynamic-copy-from-memory"></a>D1247: Cópia dinâmica da memória

CopyFromMemory não pode copiar a memória para um bitmap criado com uma textura dinâmica.





 

## <a name="possible-causes"></a>Possíveis causas

CopyFromMemory foi chamado em um bitmap que foi criado como D3D11 \_ USAGE \_ DYNAMIC.

## <a name="possible-fixes"></a>Possíveis correções

Em vez disso, use Map e Unmap para esse tipo de bitmap.
