# <a name="d1247-dynamic-copy-from-memory"></a>D1247: cópia dinâmica da memória

CopyFromMemory não pode copiar memória para um bitmap criado a partir de uma textura dinâmica.





 

## <a name="possible-causes"></a>Possíveis causas

CopyFromMemory foi chamado em um bitmap que foi criado como D3D11 de \_ uso \_ dinâmico.

## <a name="possible-fixes"></a>Possíveis correções

Use mapear e desmapear em vez disso para esse tipo de bitmap.
