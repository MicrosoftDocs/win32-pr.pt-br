# <a name="direct2d-helper-structures"></a>estruturas auxiliares Direct2D

O namespace D2D1 define as estruturas a seguir.



| Nome                                            | Descrição                                                                                                   |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**Tipo de TypeTraits &lt;&gt;**](/windows/win32/api/d2d1_1helper/ns-d2d1_1helper-typetraits) | Contém implementações de **Point**, **size** e **Rect** que armazenam seus dados usando o tipo especificado. |
