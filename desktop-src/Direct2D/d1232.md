# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232:32 bpp RGBA causa conversão

Considere usar BGR em vez de RGB para formatos de 32 bpp para que o efeito de BitmapSource possa evitar conversões de formato adicionais.
