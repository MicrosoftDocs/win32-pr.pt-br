# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166: a opção Descartar Mapa requer a opção de mapa de gravação

D2D1 \_ MAP OPTIONS DISCARD deve ser combinado com \_ \_ D2D1 \_ MAP OPTIONS \_ \_ WRITE






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá enviar essa mensagem se um aplicativo chamar ID2D1Bitmap1::Map com D2D1 MAP OPTIONS DISCARD, mas não \_ \_ \_ D2D1 \_ MAP \_ OPTIONS \_ WRITE.

## <a name="possible-fixes"></a>Possíveis correções

A opção DISCARD requer WRITE para ser uma operação útil para um aplicativo. Consulte D2D1 \_ OPÇÕES DE MAPA para ver uma \_ descrição das opções de mapa. Ao chamar ID2D1Bitmap1::Map, considere qual tipo de acesso o aplicativo precisa.
