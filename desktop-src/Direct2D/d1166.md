# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166: opção de mapeamento de descarte requer opção de mapa de gravação

\_ \_ Descartar opções de mapa d2d1 \_ deve ser combinado com a \_ gravação de opções de mapa d2d1 \_ \_






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá gerar essa mensagem se um aplicativo chamar ID2D1Bitmap1:: MAP com opções de mapa do D2D1 \_ \_ \_ descartar, mas não d2d1 \_ Opções de mapeamento \_ \_ gravar.

## <a name="possible-fixes"></a>Possíveis correções

A opção descartar requer gravação para ser uma operação útil para um aplicativo. Consulte \_ Opções de mapa d2d1 \_ para obter uma descrição das opções de mapa. Ao chamar ID2D1Bitmap1:: Map, considere o tipo de acesso que o aplicativo precisa.
