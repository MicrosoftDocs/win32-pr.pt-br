# <a name="d1116-unexpected-hardware-creation-error"></a>D1116: Erro inesperado de criação de hardware

Um código de \[ *erro de erro inesperado* foi encontrado ao tentar criar um Destino \] Direct3D.

## <a name="possible-causes"></a>Possíveis causas

Essa mensagem ocorre quando Direct2D não pôde produzir um recurso de destino de hardware com os parâmetros especificados pelo aplicativo. Uma causa provável é que o aplicativo solicitou um destino de hardware no qual o Direct2D deve assumir a propriedade da alocação da textura de destino, como por meio de [**ID2D1Factory::CreateHwndRenderTarget**](/windows/win32/api/d2d1/nf-d2d1-id2d1factory-createhwndrendertarget(constd2d1_render_target_properties__constd2d1_hwnd_render_target_properties__id2d1hwndrendertarget))e o tamanho solicitado era muito grande. Cada nível de recurso do Direct3D impõe um limite para o nivelamento de textura. Consulte [Níveis de recursos do Direct3D.](/previous-versions/windows/apps/hh994923(v=win.10))

## <a name="possible-fixes"></a>Possíveis correções

Ao alocar um destino, verifique se o tamanho de destino não excede o tamanho máximo permitido pelo nível do recurso. Para solucionar problemas de alocação de destino mais bem, considere modificar o aplicativo para alocar a textura de destino em si e verificar se a alocação foi bem-sucedida. Para usar diretamente essa textura como um destino, interopere-a como [**um ID2D1Bitmap**](/windows/win32/api/d2d1/nn-d2d1-id2d1bitmap) e use **o ID2D1Bitmap** como o argumento para [**ID2D1DeviceContext::SetTarget**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1devicecontext-settarget).
