# <a name="id2d1devicecreateprintcontrol-methods"></a>Métodos ID2D1Device::CreatePrintControl

Cria um [**objeto ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) que converte Direct2D [primitivos](/windows/desktop/direct2d/direct2d-portal) armazenados em [**ID2D1CommandList**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1commandlist) em uma representação de página fixa. Em seguida, o subsistema de impressão consome os primitivos.
