# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131: formato DXGI não compatível com GDI

Uma opção compatível com GDI foi passada para essa API, mas o formato DXGI não é compatível com GDI.






 

## <a name="possible-causes"></a>Possíveis causas

O uso do \_ \_ sinalizador compatível com GDI de opções de bitmap d2d1 \_ \_ durante a criação de bitmap permite a compatibilidade do bitmap com a renderização GDI, mas apresenta restrições adicionais no formato de superfície do bitmap. Em geral, a renderização GDI é compatível com os formatos de formato DXGI \_ \_ B8G8R8A8 \_ UNORM e dxgi \_ \_ B8G8R8A8 \_ UNORM \_ sRGB.

## <a name="possible-fixes"></a>Possíveis correções

Para resolver esse problema, verifique se o formato especificado para um bitmap compatível com GDI é um formato BGRA de quatro canais, conforme especificado acima. Se o bitmap não exigir interoperabilidade com GDI, omita o \_ especificador compatível com GDI.
