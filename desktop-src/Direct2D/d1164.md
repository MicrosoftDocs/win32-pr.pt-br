# <a name="d1164-invalid-map-option"></a>D1164: opção de mapa inválida

\_ \_ Opções de mapa d2d1 \_ nenhum não é um parâmetro válido para mapear






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá gerar essa mensagem se um aplicativo chamar ID2D1Bitmap1:: MAP com a opção D2D1 do \_ map \_ opções \_ None.

## <a name="possible-fixes"></a>Possíveis correções

Ao chamar ID2D1Bitmap1:: Map, use uma opção de mapa válida. Consulte \_ Opções de mapa d2d1 \_ para obter uma descrição das opções de mapa; considere se o aplicativo precisa de acesso de leitura ou gravação.
