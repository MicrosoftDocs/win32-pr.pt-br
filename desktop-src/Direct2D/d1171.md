# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: Chamada redundante para mapear bitmap

Foi feita uma tentativa de mapear um bitmap que já está mapeado.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D pode enviar essa mensagem e relatar um erro se um aplicativo tentar mapear um bitmap que já foi mapeado usando ID2D1Bitmap1::Map.

## <a name="possible-fixes"></a>Possíveis correções

Da mesma forma que ID2D1RenderTarget::EndDraw deve seguir chamadas para ID2D1RenderTarget::BeginDraw, verifique se ID2D1Bitmap1::Unmap segue chamadas para ID2D1Bitmap1::Map. Se um aplicativo usar retornos de chamada ou caminhos de código recursivos para mapear bitmaps, verifique se seu comportamento está correto.
