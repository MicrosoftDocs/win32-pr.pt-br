# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: chamada redundante para bitmap de mapa

Foi feita uma tentativa de mapear um bitmap que já está mapeado.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D pode gerar essa mensagem e relatar um erro se um aplicativo tentar mapear um bitmap que já foi mapeado usando ID2D1Bitmap1:: Map.

## <a name="possible-fixes"></a>Possíveis correções

Da mesma forma que ID2D1RenderTarget:: EndDraw deve seguir as chamadas para ID2D1RenderTarget:: BeginDraw, verifique se ID2D1Bitmap1:: mapeamento segue as chamadas para ID2D1Bitmap1:: Map. Se um aplicativo usar retornos de chamada ou caminhos de código recursivo para mapear bitmaps, verifique se seu comportamento está correto.
