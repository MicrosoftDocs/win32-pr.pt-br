# <a name="d1186-dc-render-target-requires-binddc"></a>D1186: o destino de renderização de DC requer BindDC

É inválido usar um destino de renderização de DC antes de chamar seu método BindDC.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo criou um ID2D1DCRenderTarget e o usou para chamar BeginDraw, mas ainda não chamou seu método BindDC. Ou, BindDC resultou em um erro.

## <a name="possible-fixes"></a>Possíveis correções

Antes de usar um destino de renderização DC-interopped, os aplicativos devem chamar BindDC para associar o destino a um contexto de dispositivo GDI. O destino não é considerado válido até que um contexto de dispositivo GDI tenha sido atribuído. Consulte ID2D1DCRenderTarget:: BindDC para obter um exemplo. Se o aplicativo já chamar BindDC, certifique-se de que BindDC não retornou um erro.
