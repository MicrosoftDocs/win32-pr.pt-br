# <a name="d1196-a8-shared-bitmap"></a>D1196: bitmap compartilhado a8

Uma superfície DXGI do formato DXGI \_ \_ UNORM a8 \_ não tem suporte nesta API.





 

## <a name="possible-causes"></a>Possíveis causas

\_ \_ \_ Não há suporte para o formato dxgi A8 UNORM como um formato de bitmap de interoperabilidade de dxgi, mesmo que ele tenha suporte em outras APIs.

## <a name="possible-fixes"></a>Possíveis correções

Crie um bitmap de interoperabilidade DXGI de um formato diferente.
