# <a name="interoperability"></a>Interoperabilidade

Os tópicos nesta seção descrevem como o Direct2D interopera com outros sistemas, como GDI, Direct3D e WIC.

> [!Note]  
> A partir do Windows 8, o Direct2D fornece a interface [**ID2D1DeviceContext**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1devicecontext) , que fornece uma maneira simples de interoperar. Consulte o tópico [dispositivos e contextos de dispositivo](devices-and-device-contexts.md) para obter mais informações.
