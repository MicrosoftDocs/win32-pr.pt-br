# <a name="interoperability"></a>Interoperabilidade

Os tópicos nesta seção descrevem como Direct2D interopera com outros sistemas, como GDI, Direct3D e WIC.

> [!Note]  
> Começando com Windows 8, Direct2D fornece a interface [**ID2D1DeviceContext,**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1devicecontext) que fornece uma maneira simples de interoperar. Consulte o [tópico Dispositivos e Contextos de Dispositivo](devices-and-device-contexts.md) para obter mais informações.
