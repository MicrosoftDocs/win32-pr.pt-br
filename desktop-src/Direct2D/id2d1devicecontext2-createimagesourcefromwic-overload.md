# <a name="id2d1devicecontext2createimagesourcefromwic-methods"></a>Métodos ID2D1DeviceContext2:: CreateImageSourceFromWic

Cria um objeto de origem de imagem de uma origem de bitmap do WIC, ao mesmo tempo em que preenche toda a memória de pixel na origem da imagem. A imagem é carregada e armazenada ao usar uma quantidade mínima de memória.
