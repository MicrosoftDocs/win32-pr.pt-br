# <a name="id2d1devicecontextfillopacitymask-methods"></a>Métodos ID2D1DeviceContext:: FillOpacityMask

Preencha usando o canal alfa do bitmap de máscara de opacidade fornecido. A opacidade do pincel será modulada pela máscara. O modo de suavização de destino de renderização deve ser definido como alias.
