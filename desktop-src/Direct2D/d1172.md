# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: inválido para desmapear bitmap não mapeado

Foi feita uma tentativa de cancelar o mapeamento de um bitmap que não está mapeado.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D pode gerar essa mensagem e relatar um erro se um aplicativo tentar cancelar o mapeamento de um bitmap que ainda não foi mapeado usando ID2D1Bitmap1:: Map.

## <a name="possible-fixes"></a>Possíveis correções

Da mesma forma que ID2D1RenderTarget:: EndDraw deve seguir as chamadas para ID2D1RenderTarget:: BeginDraw, verifique se ID2D1Bitmap1:: mapeamento segue as chamadas para ID2D1Bitmap1:: Map. Não chame ID2D1Bitmap1:: remapear várias vezes para cancelar o mapeamento de um bitmap.
