# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: Inválido para desmapear bitmap não mapeado

Foi feita uma tentativa de desmmapar um bitmap que não está mapeado.






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá enviar essa mensagem e relatar um erro se um aplicativo tentar desmapear um bitmap que ainda não foi mapeado usando ID2D1Bitmap1::Map.

## <a name="possible-fixes"></a>Possíveis correções

Da mesma forma que ID2D1RenderTarget::EndDraw deve seguir chamadas para ID2D1RenderTarget::BeginDraw, verifique se ID2D1Bitmap1::Unmap segue chamadas para ID2D1Bitmap1::Map. Não chame ID2D1Bitmap1::Unmap várias vezes para desmapear um bitmap.
