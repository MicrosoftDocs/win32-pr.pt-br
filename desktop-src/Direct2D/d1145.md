# <a name="d1145-print-control-dpi-incorrect"></a>D1145: DPI de controle de impressão incorreto

Direct2D Print: Propriedades de controle de impressão \_ \_ . o valor de rasterDPI é inválido.






 

## <a name="possible-causes"></a>Possíveis causas

Durante a criação de um controle de impressão, essa mensagem indica que o aplicativo especificou um DPI inválido; em particular, um DPI menor que 1,0. Embora Direct2D não considere isso como um erro, o Direct2D Print se comportará como se o aplicativo especificasse um DPI de 150,0.

## <a name="possible-fixes"></a>Possíveis correções

Para resolver essa mensagem, use um DPI de 1,0 ou superior ao criar um controle de impressão.
