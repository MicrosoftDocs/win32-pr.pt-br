# <a name="d1188-cpu-read-and-target-bitmap-options-are-incompatible"></a>D1188: opções de bitmap de leitura e de destino da CPU são incompatíveis

\_ \_ \_ \_ A leitura de CPU de opções de bitmap d2d1 não é compatível com o \_ destino de opções de bitmap d2d1 \_ \_ .






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou uma combinação incompatível dos \_ sinalizadores de opções de bitmap d2d1 \_ .

## <a name="possible-fixes"></a>Possíveis correções

Remova as \_ opções \_ de bitmap d2d1 \_ CPU \_ Read ou d2d1 \_ \_ Opções de bitmap \_ de destino dos sinalizadores. Se o aplicativo precisar de ambas as opções, ele deverá criar dois bitmaps separados.
