# <a name="d1188-cpu-read-and-target-bitmap-options-are-incompatible"></a>D1188: As opções de bitmap de leitura e destino da CPU são incompatíveis

D2D1 OPÇÕES DE BITMAP A LEITURA da CPU não é compatível com O DESTINO DAS OPÇÕES \_ DE \_ \_ \_ BITMAP D2D1. \_ \_ \_






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo especificou uma combinação incompatível dos sinalizadores D2D1 \_ BITMAP \_ OPTIONS.

## <a name="possible-fixes"></a>Possíveis correções

Remova OPÇÕES DE BITMAP D2D1 LEITURA da CPU ou DESTINO DE OPÇÕES \_ \_ DE \_ \_ BITMAP D2D1 \_ dos \_ \_ sinalizadores. Se o aplicativo precisar de ambas as opções, ele deverá criar dois bitmaps separados.
