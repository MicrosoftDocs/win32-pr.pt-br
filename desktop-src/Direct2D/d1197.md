# <a name="d1197-wrong-d3d-device"></a>D1197: dispositivo D3D incorreto

A superfície DXGI passada para essa API é do dispositivo errado.





 

## <a name="possible-causes"></a>Possíveis causas

A superfície DXGI passada para essa API é do dispositivo errado.

## <a name="possible-fixes"></a>Possíveis correções

Verifique se as superfícies DXGI têm o mesmo dispositivo D3D que o contexto do dispositivo D2D.
