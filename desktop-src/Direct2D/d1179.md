# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: a coleção de parada de gradiente zero é interrompida

A criação de um ID2D1GradientStopCollection requer pelo menos uma parada de gradiente.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo chamou ID2D1RenderTarget:: CreateGradientStopCollection, mas especificou uma contagem de parada de gradiente de 0.

## <a name="possible-fixes"></a>Possíveis correções

Ao criar um ID2D1GradientStopCollection, o aplicativo deve especificar pelo menos um limite de gradiente com sua cor e posição. Consulte ID2D1RenderTarget:: CreateGradientStopCollection para obter um exemplo.
