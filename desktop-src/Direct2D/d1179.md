# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: Gradient Stop Collection Zero Stops

A criação de uma ID2D1GradientStopCollection requer pelo menos uma parada de gradiente.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo chamado ID2D1RenderTarget::CreateGradientStopCollection, mas especificou uma contagem de pontos de gradiente de 0.

## <a name="possible-fixes"></a>Possíveis correções

Ao criar uma ID2D1GradientStopCollection, o aplicativo deve especificar pelo menos uma parada de gradiente com sua cor e posição. Consulte ID2D1RenderTarget::CreateGradientStopCollection para ver um exemplo.
