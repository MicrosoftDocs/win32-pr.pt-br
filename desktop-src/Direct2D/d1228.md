# <a name="d1228-not-original-target"></a>D1228: Destino não original

Uma chamada para ID2D1HwndRenderTarget::Resize ou ID2D1DCRenderTarget::BindDC foi feita enquanto um destino diferente do original foi definido no contexto do dispositivo.
