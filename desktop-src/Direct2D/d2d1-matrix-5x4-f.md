# <a name="d2d1_matrix_5x4_f"></a>D2D1 \_ MATRIX \_ 5X4 \_ F

Representa uma matriz 5 por 4.


```C++
typedef D2D_MATRIX_5X4_F D2D1_MATRIX_5X4_F;
```



## <a name="remarks"></a>Comentários

**D2D1 \_ MATRIX \_ 5X4** é um novo nome para a estrutura [**D2D \_ MATRIX \_ 5X4 \_ F.**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f) Para ver uma lista de campos fornecidos pela matriz, [**consulte D2D \_ MATRIX \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f).
