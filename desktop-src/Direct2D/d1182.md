# <a name="d1182-operation-on-unclosed-path-geometry"></a>D1182: operação em geometria de caminho não fechada

Ocorreu uma operação em uma geometria de caminho que não foi fechada.






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo começou a popular um ID2D1PathGeometry, mas não chamou ID2D1SimplifiedGeometrySink:: Close em seu coletor populando. Ou, o aplicativo chamado Close, mas um valor de erro foi retornado.

## <a name="possible-fixes"></a>Possíveis correções

Certifique-se de que todas as geometrias de caminho sejam preenchidas corretamente chamando Close no coletor populando e garantindo que Close não retornou um erro. Consulte a referência para ID2D1PathGeometry para obter um exemplo de como criar e preencher um objeto de geometria de caminho.
