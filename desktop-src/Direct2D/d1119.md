# <a name="d1119-bitmap-bound-as-target"></a>D1119: bitmap associado como destino

Uma operação de desenho falhou porque o bitmap de destino e um bitmap de origem eram os mesmos.






 

## <a name="possible-causes"></a>Possíveis causas

Uma causa provável dessa mensagem é que o aplicativo tentou uma operação de desenho na qual o mesmo bitmap foi usado tanto para a origem quanto para o destino. Semanticamente, isso é possível para texturas interoperadas do Direct3D com \_ o \_ recurso BIND Shader e o destino de processamento de ligação \_ \_ .

## <a name="possible-fixes"></a>Possíveis correções

Certifique-se de que o aplicativo não tente usar seu bitmap de destino como a origem de uma operação de desenho, incluindo efeitos. Em alguns casos, essa mensagem pode indicar que o aplicativo definiu o destino do seu contexto de dispositivo para um bitmap diferente do pretendido.
