# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: a opção ler mapa requer bitmap legível da CPU

\_ \_ As opções de mapa d2d1 \_ lidas só são válidas em um bitmap legível por CPU






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá gerar essa mensagem se um aplicativo chamar ID2D1Bitmap1:: MAP com o \_ atributo de leitura de opções do mapa d2d1 \_ \_ , mas o bitmap não estiver legível para a CPU.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo exigir acesso de leitura usando ID2D1Bitmap1:: Map, crie o bitmap usando a \_ opção de leitura de CPU de opções de bitmap d2d1 \_ \_ \_ .
