# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: a opção Ler Mapa requer bitmap acessível pela CPU

D2D1 \_ OPÇÕES DE MAPA READ só é válido em um \_ \_ bitmap acessível pela CPU






 

## <a name="possible-causes"></a>Possíveis causas

Direct2D poderá enviar essa mensagem se um aplicativo chamar ID2D1Bitmap1::Map com o atributo READ OPÇÕES DE MAPA D2D1, mas o bitmap não estiver acessível para \_ \_ \_ CPU.

## <a name="possible-fixes"></a>Possíveis correções

Se um aplicativo exigir acesso de leitura usando ID2D1Bitmap1::Map, crie o bitmap usando a opção D2D1 \_ OPÇÕES DE BITMAP \_ DE CPU \_ \_ READ.
