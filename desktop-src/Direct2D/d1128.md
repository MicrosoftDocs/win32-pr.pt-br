# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: formato DXGI sem suporte em nenhuma API

O formato DXGI passado para essa API não tem suporte do Direct2D.



|             |         |
|-------------|---------|
| Nível de erro | Aviso |



 

## <a name="possible-causes"></a>Possíveis causas

Alguns formatos de superfície não são compatíveis com operações Direct2D, independentemente de como são usados. Essa mensagem indica que o aplicativo deve escolher outro formato de superfície.

## <a name="possible-fixes"></a>Possíveis correções

Consulte [formatos de pixel com suporte e modos alfa](supported-pixel-formats-and-alpha-modes.md) para obter orientação sobre quais formatos têm suporte e quais não são. Se o aplicativo precisar ser processado em um único canal e estiver tentando usar um formato somente de canal vermelho, o que não tem suporte, considere alterá-lo para renderizar para um formato somente de canal alfa.
