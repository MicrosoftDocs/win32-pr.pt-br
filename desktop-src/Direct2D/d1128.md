# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: Formato DXGI sem suporte em nenhuma API

O formato DXGI passado para essa API não é suportado pelo Direct2D.





 

## <a name="possible-causes"></a>Possíveis causas

Alguns formatos de superfície não são compatíveis com Direct2D operações, independentemente de como elas são usadas. Essa mensagem indica que o aplicativo deve escolher outro formato de superfície.

## <a name="possible-fixes"></a>Possíveis correções

Consulte [Formatos de pixel com suporte](supported-pixel-formats-and-alpha-modes.md) e modos alfa para obter diretrizes sobre quais formatos têm suporte e quais não têm suporte. Se o aplicativo deve renderizar em um único canal e estiver tentando usar um formato somente de canal vermelho que não tem suporte, considere a possibilidade de alterar para renderizar para um formato somente de canal alfa.
