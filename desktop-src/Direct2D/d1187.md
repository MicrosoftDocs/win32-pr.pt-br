# <a name="d1187-settarget-called-on-outstanding-dc"></a>D1187: SetTarget chamado no controlador de domínio pendente

[**SetTarget**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1devicecontext-settarget) foi chamado entre um par [**GetDC**](/windows/win32/api/d2d1/nf-d2d1-id2d1gdiinteroprendertarget-getdc) / [**ReleaseDC**](/windows/win32/api/d2d1/nf-d2d1-id2d1gdiinteroprendertarget-releasedc) .






 

## <a name="possible-causes"></a>Possíveis causas

O aplicativo obteve um [**ID2D1GdiInteropRenderTarget**](/windows/win32/api/d2d1/nn-d2d1-id2d1gdiinteroprendertarget) e chamou seu método [**GetDC**](/windows/win32/api/d2d1/nf-d2d1-id2d1gdiinteroprendertarget-getdc) e, em seguida, chamou [**flush**](/windows/win32/api/d2d1/nf-d2d1-id2d1rendertarget-flush) ou [**EndDraw**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1commandsink-enddraw) antes de chamar [**ReleaseDC**](/windows/win32/api/d2d1/nf-d2d1-id2d1gdiinteroprendertarget-releasedc).

## <a name="possible-fixes"></a>Possíveis correções

Verifique se o aplicativo chama [**ReleaseDC**](/windows/win32/api/d2d1/nf-d2d1-id2d1gdiinteroprendertarget-releasedc) antes de chamar [**flush**](/windows/win32/api/d2d1/nf-d2d1-id2d1rendertarget-flush) ou [**EndDraw**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1commandsink-enddraw), e se cada chamada para **ReleaseDC** é correspondida com uma chamada anterior para [**GetDC**](/windows/win32/api/d2d1/nf-d2d1-id2d1gdiinteroprendertarget-getdc). Direct2D não tentará operar em um recurso de destino GDI se o contexto do dispositivo GDI estiver sendo usado no momento pelo aplicativo.
