# <a name="viewtitle"></a>Exibir. title

O atributo **title** especifica ou recupera o título da **exibição**.

``` syntax
        elementID.title
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** especificada em tempo de design e somente leitura depois. Pode ter até 255 caracteres de comprimento e não tem valor padrão.

## <a name="remarks"></a>Comentários

O título especificado ou recuperado é o nome de exibição do tema.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
