# <a name="working-with-the-library"></a>Trabalhando com a biblioteca

A biblioteca é um banco de dados de informações sobre o conteúdo de mídia armazenado no computador do usuário ou que foi reproduzido lá. Algumas das informações são exibidas no recurso biblioteca no Windows Media Player. Um conjunto maior de informações pode ser acessado programaticamente.

Os itens de mídia podem ser organizados em listas de reprodução. As listas de reprodução podem ser criadas por usuários ou por meio de código. O modelo de objeto do Windows Media Player fornece objetos e métodos que podem ser usados para manipular itens de mídia individuais e listas de reprodução.

As seções a seguir explicam técnicas para trabalhar com a biblioteca do:

-   [Acesso à biblioteca](library-access.md)
-   [Gerenciando itens de mídia](managing-media-items.md)
-   [Atributos de item de mídia](media-item-attributes.md)
-   [Gerenciando listas de reprodução](managing-playlists.md)
