# <a name="onresize"></a>OnResize

O manipulador de eventos **OnResize** manipula um evento que ocorre quando um controle é redimensionado.

``` syntax
onresize
```

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------|
| Versão<br/> | Windows Media Player versão 70 ou posterior<br/> |
