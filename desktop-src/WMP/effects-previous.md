# <a name="effectsprevious"></a>EFEITOS. anterior

O método **anterior** exibe a predefinição de visualização anterior, movendo para a última predefinição da visualização anterior, se necessário.

``` syntax
        elementID.previous()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Se a predefinição atual for a primeira na série de todas as visualizações disponíveis, a última predefinição da última visualização será feita atual.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
