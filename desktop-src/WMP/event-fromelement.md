# <a name="eventfromelement"></a>evento. FromElement

O atributo **FromElement** recupera o elemento do qual veio o evento. Se nenhum elemento passar um evento, o valor será **NULL**.

``` syntax
event.fromElement
```

## <a name="possible-values"></a>Valores possíveis

Este atributo é um objeto somente leitura.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
