# <a name="why-make-skins"></a>Por que fazer as capas?

Quando alguém usa um programa em um ambiente de computação gráfica, como o Microsoft Windows, os componentes visuais com os quais o usuário interage formam a interface do usuário. Uma das finalidades do Microsoft Windows é fornecer uma interface de usuário padrão para que todos os programas operem de uma maneira semelhante e familiar. Por exemplo, a Microsoft recomenda, na experiência do usuário do Microsoft Windows, que todos os programas forneçam um botão **fechar** no canto superior direito da janela principal do programa.

O Windows Media Player fornece os recursos para criar sua própria interface do usuário. Se você quiser colocar o botão **fechar** no meio da tela, poderá fazer isso. Talvez você não goste da aparência do botão **fechar** (parece um X dentro de uma caixa); Se você quiser que ele se pareça com um Skull e crossbones, você pode criar uma interface do usuário onde o botão **fechar** é exatamente isso! O Windows Media Player fornece todas as ferramentas de que você precisa para criar uma interface do usuário personalizada para reproduzir música e vídeo: botões, barras deslizantes, janelas de vídeo, janelas de visualização, barras de equalização e assim por diante.

Há várias boas razões pelas quais você talvez queira criar sua própria interface do usuário para o Windows Media Player. Uma razão é que você talvez queira adicionar funcionalidade que ainda não esteja no Windows Media Player. Por exemplo, talvez você queira criar um jogador que reproduza música de listas de reprodução com base na hora do dia, para que você tenha uma pedra na manhã e lentamente o jazz na noite. Ou talvez você queira criar uma capa com um grande botão vermelho que irá parar a música rapidamente. O Windows Media Player não vem com um "reproduzir a mesma música repetidamente até que meu impublicável fique louco", mas se você quiser um, poderá criá-lo.

Outro motivo para criar uma capa é fazer uma aparência de marca para o Windows Media Player. Se você estiver distribuindo música do seu site e usar um logotipo específico, talvez queira criar uma capa que use seu logotipo para lembrar as pessoas sobre seu site. Se você tiver uma banda de rock, poderá criar uma capa com imagens da sua banda.

E outro motivo para fazer as capas é fazer algo exclusivo que possa melhorar sua área de trabalho. Quando seus amigos chegarem e perguntarem o que é o programa interessante na tela, você pode dizer que você mesmo o fez. Você pode até mesmo tirar uma foto de seu cachorro, examiná-lo em seu computador, adicionar alguns botões e criar uma capa na qual clicar no nariz de um cão começa a música e clicar na parte final para. Você pode criar diferentes capas para diferentes tipos de música ou ter uma aparência diferente para todos os dias da semana.
