# <a name="why-make-skins"></a>Por que fazer capas?

Quando alguém usa um programa em um ambiente de computação gráfica, como o Microsoft Windows, os componentes visuais com os quais o usuário interage com a interface do usuário. Uma das finalidades do Microsoft Windows é fornecer uma interface do usuário padrão para que todos os programas operem de maneira semelhante e familiar. Por exemplo, **a** Microsoft recomenda, na Experiência do Usuário do Microsoft Windows, que cada programa forneça um botão Fechar no canto superior direito da janela principal do programa.

Windows Media Player fornece os recursos para criar sua própria interface do usuário. Se você quiser colocar o **botão** Fechar no meio da tela, poderá fazer isso. Talvez você não gosta da aparência do **botão** Fechar (ele se parece com um X dentro de uma caixa); se você quiser que ele se pareça com um rosto e um crossbones, poderá fazer uma interface do usuário em que o **botão** Fechar é exatamente isso! Windows Media Player oferece todas as ferramentas necessárias para fazer uma interface do usuário personalizada para reprodução de música e vídeo: botões, barras deslizantes, janelas de vídeo, janelas de visualização, barras de equalização e assim por diante.

Há vários bons motivos pelos quais você pode querer criar sua própria interface do usuário para Windows Media Player. Um motivo é que talvez você queira adicionar funcionalidades que ainda não estão no Windows Media Player. Por exemplo, talvez você queira criar um player que reproduz música de playlists baseadas na hora do dia, para que você tenha uma rocha animada pela manhã e um jazz lento à noite. Ou talvez você queira fazer uma capa com um botão vermelho grande que interromperá a música rapidamente. Windows Media Player vem com um botão "tocar a mesma música uma e outra vez até que meu coração se desesgoe", mas se você quiser uma, poderá criar.

Outro motivo para criar uma capa é fazer com que uma marca procure por Windows Media Player. Se você estiver distribuindo música do seu site e usar um logotipo específico, talvez você queira criar uma capa que use seu logotipo para lembrar as pessoas sobre seu site. Se você tiver uma banda de rock, poderá fazer uma capa com imagens da sua banda.

E outro motivo para fazer capas é tornar algo exclusivo que possa a sua área de trabalho. Quando seus amigos aparecerem e perguntarem o que é esse programa interessante na tela, você pode dizer que você fez isso por conta própria. Você pode até mesmo tirar uma foto do seu cachorro, digitalizar no computador, adicionar alguns botões e criar uma capa na qual clicar no nariz do cachorro inicia a música e clicar na parte final a interrompe. Você pode criar capas diferentes para diferentes tipos de música ou ter uma capa diferente para cada dia da semana.
