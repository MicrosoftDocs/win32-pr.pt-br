# <a name="why-make-skins"></a><span data-ttu-id="8cdba-108">Por que fazer as capas?</span><span class="sxs-lookup"><span data-stu-id="8cdba-108">Why Make Skins?</span></span>

<span data-ttu-id="8cdba-109">Quando alguém usa um programa em um ambiente de computação gráfica, como o Microsoft Windows, os componentes visuais com os quais o usuário interage formam a interface do usuário.</span><span class="sxs-lookup"><span data-stu-id="8cdba-109">When someone uses a program in a graphical computing environment such as Microsoft Windows, the visual components with which the user interacts make up the user interface.</span></span> <span data-ttu-id="8cdba-110">Uma das finalidades do Microsoft Windows é fornecer uma interface de usuário padrão para que todos os programas operem de uma maneira semelhante e familiar.</span><span class="sxs-lookup"><span data-stu-id="8cdba-110">One of the purposes of Microsoft Windows is to provide a standard user interface so that all programs will operate in a similar, familiar way.</span></span> <span data-ttu-id="8cdba-111">Por exemplo, a Microsoft recomenda, na experiência do usuário do Microsoft Windows, que todos os programas forneçam um botão **fechar** no canto superior direito da janela principal do programa.</span><span class="sxs-lookup"><span data-stu-id="8cdba-111">For example, Microsoft recommends, in the Microsoft Windows User Experience, that every program provide a **Close** button in the upper-right corner of the main window of the program.</span></span>

<span data-ttu-id="8cdba-112">O Windows Media Player fornece os recursos para criar sua própria interface do usuário.</span><span class="sxs-lookup"><span data-stu-id="8cdba-112">Windows Media Player provides the capabilities for creating your own user interface.</span></span> <span data-ttu-id="8cdba-113">Se você quiser colocar o botão **fechar** no meio da tela, poderá fazer isso.</span><span class="sxs-lookup"><span data-stu-id="8cdba-113">If you want to put the **Close** button in the middle of the screen, you can do that.</span></span> <span data-ttu-id="8cdba-114">Talvez você não goste da aparência do botão **fechar** (parece um X dentro de uma caixa); Se você quiser que ele se pareça com um Skull e crossbones, você pode criar uma interface do usuário onde o botão **fechar** é exatamente isso!</span><span class="sxs-lookup"><span data-stu-id="8cdba-114">Perhaps you do not like the way the **Close** button looks (it looks like an X inside a box); if you want it to look like a skull and crossbones, you can make a user interface where the **Close** button is just that!</span></span> <span data-ttu-id="8cdba-115">O Windows Media Player fornece todas as ferramentas de que você precisa para criar uma interface do usuário personalizada para reproduzir música e vídeo: botões, barras deslizantes, janelas de vídeo, janelas de visualização, barras de equalização e assim por diante.</span><span class="sxs-lookup"><span data-stu-id="8cdba-115">Windows Media Player gives you all the tools you need to make a custom user interface for playing music and video: buttons, slider bars, video windows, visualization windows, equalization bars, and so on.</span></span>

<span data-ttu-id="8cdba-121">Outro motivo para criar uma capa é fazer uma aparência de marca para o Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="8cdba-121">Another reason for creating a skin is to make a branded look for Windows Media Player.</span></span> <span data-ttu-id="8cdba-122">Se você estiver distribuindo música do seu site e usar um logotipo específico, talvez queira criar uma capa que use seu logotipo para lembrar as pessoas sobre seu site.</span><span class="sxs-lookup"><span data-stu-id="8cdba-122">If you are distributing music from your website and use a particular logo, you might want to design a skin that uses your logo to remind people about your site.</span></span> <span data-ttu-id="8cdba-123">Se você tiver uma banda de rock, poderá criar uma capa com imagens da sua banda.</span><span class="sxs-lookup"><span data-stu-id="8cdba-123">If you have a rock band, you can make a skin with pictures of your band on it.</span></span>

<span data-ttu-id="8cdba-124">E outro motivo para fazer as capas é fazer algo exclusivo que possa melhorar sua área de trabalho.</span><span class="sxs-lookup"><span data-stu-id="8cdba-124">And another reason to make skins is to make something unique that can dress up your desktop.</span></span> <span data-ttu-id="8cdba-125">Quando seus amigos chegarem e perguntarem o que é o programa interessante na tela, você pode dizer que você mesmo o fez.</span><span class="sxs-lookup"><span data-stu-id="8cdba-125">When your friends come over and ask you what that cool program on your screen is, you can say you made it yourself.</span></span> <span data-ttu-id="8cdba-126">Você pode até mesmo tirar uma foto de seu cachorro, examiná-lo em seu computador, adicionar alguns botões e criar uma capa na qual clicar no nariz de um cão começa a música e clicar na parte final para.</span><span class="sxs-lookup"><span data-stu-id="8cdba-126">You can even take a picture of your dog, scan it into your computer, add some buttons, and create a skin in which clicking on your dog's nose starts the music and clicking on the tail stops it.</span></span> <span data-ttu-id="8cdba-127">Você pode criar diferentes capas para diferentes tipos de música ou ter uma aparência diferente para todos os dias da semana.</span><span class="sxs-lookup"><span data-stu-id="8cdba-127">You can create different skins for different kinds of music or have a different skin for every day of the week.</span></span>
