# <a name="sliderbackgroundhoverimage"></a>SLIDER. backgroundHoverImage

O atributo **backgroundHoverImage** especifica ou recupera a imagem de plano de fundo do controle deslizante que aparece ao passar o mouse sobre ele.

``` syntax
        elementID.backgroundHoverImage
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém o nome de um arquivo de imagem.

## <a name="remarks"></a>Comentários

Esse atributo é opcional. Se não for especificado, o **backgroundImage** será usado.

Os formatos com suporte são BMP, JPG, PNG e GIF (não incluindo GIFs animados).
