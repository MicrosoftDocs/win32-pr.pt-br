# <a name="skin-reference"></a>Referência de capa

Essa referência é dividida nas seções a seguir.



| Seção                        | Descrição                                                                     |
|--------------------------------|---------------------------------------------------------------------------------|
| [Arte do álbum](album-art.md)     | Informações sobre como incluir arte de álbum em capas.                                 |
| [Bitmaps](bitmaps.md)         | Informações sobre os arquivos Art usados na criação de capa.                          |
| [Botões](buttons.md)         | Informações sobre os controles de pressão disponíveis em capas.                       |

Para saber a ordem na qual eles devem aparecer em um arquivo de definição de capa, consulte [criando um arquivo de definição de capa](creating-a-skin-definition-file.md). Detalhes sobre a composição do arquivo de definição de capa são fornecidos na seção [sobre capas](about-skins-mobile.md) .
