# <a name="mutebutton"></a>MUTEBUTTON

Este é um botão predefinido com os valores padrão a seguir.

``` syntax
onclick="jscript:player.settings.mute=down;"
upToolTip="Mute"
downToolTip="Sound"
down="wmpprop:player.settings.mute"
sticky="true"
```

## <a name="remarks"></a>Comentários

Isso cria um controle de **botão** que fará o mudo e o mudo do áudio. As dicas de ferramenta são localizadas. Todas as propriedades desse **botão** podem ser substituídas especificando-as explicitamente.
