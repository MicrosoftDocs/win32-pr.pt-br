# <a name="videosettings-element"></a>Elemento VIDEOSETTINGS

O elemento **VIDEOSETTINGS** fornece uma maneira de modificar várias configurações de vídeo, usando os atributos e o método listados aqui.

O elemento **VIDEOSETTINGS** dá suporte aos seguintes atributos.



| Atributo                                  | Descrição                                                 |
|--------------------------------------------|-------------------------------------------------------------|
| [lo](videosettings-brightness.md) | Especifica ou recupera a configuração de brilho do vídeo. |
| [contraste](videosettings-contrast.md)     | Especifica ou recupera a configuração de contraste do vídeo.   |
| [Hue](videosettings-hue.md)               | Especifica ou recupera a configuração de matiz do vídeo.        |
| [saturação](videosettings-saturation.md) | Especifica ou recupera a configuração de saturação do vídeo. |



 

O elemento **VIDEOSETTINGS** dá suporte ao método a seguir.



| Método                           | Descrição                                    |
|----------------------------------|------------------------------------------------|
| [reset](videosettings-reset.md) | Redefine todos os atributos para seus valores padrão. |



 

O elemento **VIDEOSETTINGS** pode implementar o [atributo \_ onChange](attribute-onchange.md) manipuladores de eventos.

> [!Note]  
> Esse elemento requer o Windows Media Player para Windows XP ou posterior.
