# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Incorporação do controle Windows Media Player em uma página da Web

Ao inserir o controle Windows Media Player em uma página da Web, você pode usar uma abordagem básica ou uma abordagem mais elaborado.

Na abordagem básica, você usa o controle simplesmente para reproduzir música em segundo plano ou para adicionar uma exibição de vídeo à sua página.

Na abordagem mais elaborado, você torna sua página interativa adicionando código de script para manipular o controle por meio do modelo de objeto Player. Isso permite que você forneça uma interface do usuário de seu próprio design ou responda dinamicamente a comandos ou eventos de script, como transições de mídia.

Para obter mais informações, [consulte Usando o controle Windows Media Player em uma página da Web](using-the-windows-media-player-control-in-a-web-page.md).
