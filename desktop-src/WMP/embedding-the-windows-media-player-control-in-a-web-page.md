# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Inserindo o controle do Windows Media Player em uma página da Web

Ao inserir o controle do Windows Media Player em uma página da Web, você pode adotar uma abordagem básica ou uma abordagem mais elaborada.

Na abordagem básica, você usa o controle simplesmente para reproduzir música em segundo plano ou para adicionar uma exibição de vídeo à sua página.

Na abordagem mais elaborada, você torna sua página interativa, adicionando código de script para manipular o controle por meio do modelo de objeto do Player. Isso permite que você forneça uma interface do usuário de seu próprio design ou responda dinamicamente a comandos ou eventos de script, como transições de mídia.

Para obter mais informações, consulte [usando o controle do Windows Media Player em uma página da Web](using-the-windows-media-player-control-in-a-web-page.md).
