# <a name="equalizersettingssplinetension"></a>EQUALIZERSETTINGS.splineTension

O **atributo splineTension** especifica ou recupera a tensão de spline para o controle de equalizador.

``` syntax
        elementID.splineTension
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um número de leitura/gravação **(** **float**) que varia de 0,0 (sem tensão) a 10,0 (tensão total) com um valor padrão de 3,0.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player série 9 ou posterior<br/> |
