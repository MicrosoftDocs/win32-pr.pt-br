# <a name="listboxsorted"></a>Caixa de listagem. classificada

O atributo **ordenado** especifica ou recupera um valor que indica se o controle da caixa de listagem está classificado em ordem alfabética. Só pode ser definido em tempo de design.

``` syntax
        elementID.sorted
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                                                 |
|-------|-------------------------------------------------------------|
| true  | O controle de caixa de listagem é classificado alfabeticamente.              |
| false | Padrão. O controle caixa de listagem não está classificado em ordem alfabética. |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
