# <a name="themeversion"></a>TEMA. versão

O atributo **version** especifica ou recupera o número de versão do Windows Media Player para o qual a capa foi criada.

``` syntax
theme.version
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** (**float**). A especificação de um valor para esse atributo em tempo de design ou tempo de execução não tem efeito. O valor é sempre 1,0.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
