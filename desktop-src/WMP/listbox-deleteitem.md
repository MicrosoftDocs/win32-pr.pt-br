# <a name="listboxdeleteitem"></a>LISTBOX. deleteItem

O método **deleteItem** exclui o item de controle de caixa de listagem no índice especificado.

``` syntax
        elementID.deleteItem(index)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*index*
</dt> <dd>

**Número** (**longo**) que contém o índice do item a ser excluído.

</dd> </dl>

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

As linhas abaixo da linha excluída passarão por um valor de índice.
