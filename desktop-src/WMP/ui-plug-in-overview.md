<span data-ttu-id="31e2e-112">Os plug-ins de interface do usuário podem ser instalados, desinstalados e configurados usando a guia plug-ins da caixa de diálogo opções no Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="31e2e-112">UI plug-ins can be installed, uninstalled, and configured using the Plug-ins tab of the Options dialog in Windows Media Player.</span></span>
