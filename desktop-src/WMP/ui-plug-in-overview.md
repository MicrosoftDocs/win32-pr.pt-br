# <a name="ui-plug-in-overview"></a>Visão geral do plug-in da interface do usuário

Os plug-ins de interface do usuário podem ser instalados, desinstalados e configurados usando a guia Plug-ins da caixa de diálogo Opções no Windows Media Player.

Há cinco tipos de plug-ins de interface do usuário com suporte pelo Windows Media Player. Três desses tipos correspondem a diferentes áreas do **painel Agora Em** jogo do modo completo do jogador. Os outros dois tipos são para plug-ins que são exibidos em uma janela separada e para plug-ins que não têm exibição (exceto talvez uma página de propriedades).

Os plug-ins de interface do usuário são descarregados quando são fechados ou desabilitados. Plug-ins de interface do  usuário que aparecem no painel Agora Em Reprodução também são descarregados quando  o usuário escolhe outro plug-in do mesmo tipo ou alterna para fora do painel Agora Em reprodução. Quando o usuário alterna para o modo de capa, todos os plug-ins permanecem carregados, mas não são exibidos. Se os plug-ins separados de janela ou segundo plano estão em execução quando Windows Media Player fechado, eles serão recarregados automaticamente na próxima vez que o player for iniciado.
