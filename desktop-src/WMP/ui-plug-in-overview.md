# <a name="ui-plug-in-overview"></a>Visão geral do plug-in da interface do usuário

Os plug-ins de interface do usuário podem ser instalados, desinstalados e configurados usando a guia plug-ins da caixa de diálogo opções no Windows Media Player.

Há cinco tipos de plug-ins de interface do usuário com suporte do Windows Media Player. Três desses tipos correspondem a áreas diferentes do painel **agora em execução** do modo completo do Player. Os outros dois tipos são para plug-ins que são exibidos em uma janela separada e para plug-ins que não têm exibição (exceto talvez uma página de propriedades).

Os plug-ins de interface do usuário são descarregados quando são fechados ou desabilitados. Plug-ins de interface do usuário que aparecem no painel **agora** são descarregados quando o usuário escolhe outro plug-in do mesmo tipo ou alterna para fora do painel que está **sendo executado agora** . Quando o usuário alterna para o modo de capa, todos os plug-ins permanecem carregados, mas não são exibidos. Se plug-ins separados de janela ou de segundo plano estiverem em execução quando o Windows Media Player for fechado, eles serão recarregados automaticamente na próxima vez em que o player for iniciado.
