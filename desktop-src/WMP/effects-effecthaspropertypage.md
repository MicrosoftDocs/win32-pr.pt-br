# <a name="effectseffecthaspropertypage"></a>EFFECTs. effectHasPropertyPage

O atributo **effectHasPropertyPage** recupera um valor que indica se a visualização atual tem uma página de propriedades.

``` syntax
        elementID.effectHasPropertyPage
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** somente leitura.



| Valor | Descrição                                  |
|-------|----------------------------------------------|
| true  | A visualização tem uma página de propriedades.           |
| false | A visualização não tem uma página de propriedades. |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
