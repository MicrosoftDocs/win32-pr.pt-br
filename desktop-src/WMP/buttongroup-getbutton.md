# <a name="buttongroupgetbutton"></a>BOTÃO. getbutton

O método **getbutton** recupera o **buttonelement** com o índice especificado.

``` syntax
        elementID.getButton(button)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*Button*
</dt> <dd>

**Número** (**longo**) que contém o índice do **buttonelement** a ser recuperado.

</dd> </dl>

## <a name="return-value"></a>Valor Retornado

Esse método retorna um objeto correspondente a um elemento **buttonelement** .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
