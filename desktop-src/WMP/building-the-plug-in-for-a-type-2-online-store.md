# <a name="building-the-plug-in-for-a-type-2-online-store"></a>Criando o plug-in para uma loja online tipo 2

Você pode criar um plug-in de loja online do tipo 2 usando o assistente de plug-in da loja online. O assistente de projeto está incluído no SDK do Windows Media Player, que faz parte do SDK do Windows. O assistente é um suplemento para Microsoft Visual Studio que cria um projeto C++ e gera um código de exemplo que você pode usar como um ponto de partida para desenvolver sua própria loja online.

Os tópicos a seguir explicam como usar o assistente de plug-in da loja online:

-   [Instalando o assistente de projeto](installing-the-project-wizard.md)
-   [Criando um projeto de exemplo](creating-a-sample-project.md)
-   [Compilando o projeto de exemplo](compiling-the-sample-project.md)
-   [Executando o projeto de exemplo](running-the-sample-project.md)
-   [Personalizando o projeto de exemplo](customizing-the-sample-project.md)
