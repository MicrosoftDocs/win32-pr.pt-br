# <a name="axwindowsmediaplayerctlenabled-property"></a>Propriedade AxWindowsMediaPlayer. Ctlenabled

A propriedade Ctlenabled Obtém ou define um valor que indica se o controle do Windows Media Player está habilitado.

## <a name="syntax"></a>Syntax


```CSharp
public System.Boolean Ctlenabled {get; set;}
```


```VB

Public Property Ctlenabled As System.Boolean
```





## <a name="property-value"></a>Valor da propriedade

Um valor System. Boolean que indica se o controle do Windows Media Player está habilitado. O valor padrão é true.

## <a name="remarks"></a>Comentários

Se a propriedade **Ctlenabled** for definida como false, o Windows Media Player ocultará os controles de usuário durante a reprodução de tela inteira.
