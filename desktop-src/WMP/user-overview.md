# <a name="user-overview"></a>Visão geral do usuário

Para o usuário, as visualizações são imagens que dançaram no tempo para a música. Eles fornecem diversão adicional para aprimorar experiências musicais. Quanto mais divertido for uma visualização, melhor!

Uma visualização típica aparece conforme mostrado na ilustração a seguir.

![visualização típica](images/vizanim.gif)

## <a name="standard-visualizations"></a>Visualizações padrão

O Windows Media Player inclui várias visualizações, como as seguintes:

-   Ambience
-   Plenoptic
-   Pico

## <a name="standard-presets"></a>Predefinições padrão

Além disso, cada visualização vem com predefinições específicas que alteram como a visualização é exibida. Por exemplo, a visualização de pico vem em duas predefinições:

-   Pico
-   Amoeba

Ambos os efeitos fazem parte da mesma visualização, mas têm predefinições diferentes. Uma predefinição é um conjunto de valores para os atributos de uma visualização específica.

## <a name="skin-attributes"></a>Atributos de capa

Se você criar capas para o Windows Media Player, poderá exibir as visualizações padrão em suas capas usando o elemento **Effects** . Para obter mais detalhes sobre como usar visualizações em capas, consulte [capas do Windows Media Player](windows-media-player-skins.md).
