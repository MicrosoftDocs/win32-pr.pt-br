# <a name="equalizersettingsnormalizationaverage"></a>EQUALIZERSETTINGS.normalizationAverage

O atributo **normalizationAverage** recupera o valor de normalização média.

``` syntax
        elementID.normalizationAverage
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** somente leitura (**float**).

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
