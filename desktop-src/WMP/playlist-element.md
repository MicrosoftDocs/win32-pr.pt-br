# <a name="playlist-element"></a>Elemento PLAYLIST

O elemento **playlist** fornece uma maneira de organizar itens de mídia em uma lista para facilitar a manipulação usando os seguintes atributos e métodos. Elementos de **playlist** predefinidos também são fornecidos para sua conveniência. Colunas personalizadas podem ser especificadas para uma lista de reprodução, incluindo elementos de **coluna** como filhos do elemento **playlist** .

O elemento **playlist** dá suporte aos seguintes atributos.



| Atributo                                                                                 | Descrição                                                                                                                                |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [allowColumnSorting](playlist-allowcolumnsorting.md)                                     | Especifica ou recupera um valor que indica se a classificação em cabeçalhos de coluna é permitida.                                                    |
| [allowItemEditing](playlist-allowitemediting.md)                                         | Especifica ou recupera um valor que indica se os itens em uma lista de reprodução dão suporte à edição in-loco.                                            |
| [backgroundColor](playlist-backgroundcolor.md)                                           | Especifica ou recupera a cor do plano de fundo.                                                                                               |
| [backgroundImage](playlist-backgroundimage.md)                                           | Especifica ou recupera a imagem de plano de fundo.                                                                                               |
| [checkboxesVisible](playlist-checkboxesvisible.md)                                       | Especifica ou recupera um valor que indica se caixas de seleção estão visíveis.                                                                  |
| [columnCount](playlist-columncount.md)                                                   | Recupera o número de colunas mostradas.                                                                                                     |
| [columnOrder](playlist-columnorder.md)                                                   | Especifica ou recupera a ordem das colunas da lista de reprodução.                                                                                  |
| [Columns](playlist-columns.md)                                                           | Define as colunas que aparecem no elemento **playlist** .                                                                               |
| [columnsVisible](playlist-columnsvisible.md)                                             | Especifica ou recupera um valor que indica se as colunas são mostradas.                                                                       |
| [Copia](playlist-copying.md)                                                           | Recupera um valor que indica se o elemento **playlist** está no ato de cópia.                                                    |
| [disabledItemColor](playlist-disableditemcolor.md)                                       | Especifica ou recupera a cor de uma faixa de CD desabilitada ou de conteúdo online quando offline.                                                 |
| [dropDownBackgroundImage](playlist-dropdownbackgroundimage.md)                           | Especifica ou recupera o nome da imagem exibida no plano de fundo da lista suspensa.                                        |
| [dropDownImage](playlist-dropdownimage.md)                                               | Especifica ou recupera o nome da imagem usada para o botão de lista suspensa que é exibido na borda direita da lista suspensa. |
| [dropDownList](playlist-dropdownlist.md)                                                 | Especifica ou recupera um valor que indica quais elementos aparecem na lista suspensa para uma determinada instância do elemento **playlist** .   |

| [dropDownVisible](playlist-dropdownvisible.md)                                           | Especifica ou recupera um valor que indica se o seletor de lista suspensa de elementos de **playlist** está visível.                                  |
| [editButtonVisible](playlist-editbuttonvisible.md)                                       | Especifica ou recupera um valor que indica se o botão de edição do elemento de **playlist** está visível.                                         |
| [foregroundColor](playlist-foregroundcolor.md)                                           | Especifica ou recupera a cor de primeiro plano.                                                                                               |
| [hueShift](playlist-hueshift.md)                                                         | Especifica ou recupera a quantidade pela qual o matiz das imagens suspensas é deslocado.                                                     |
| [itemCount](playlist-itemcount.md)                                                       | Recupera o número de itens atualmente exibidos no elemento **playlist** .                                                             |
| [itemErrorColor](playlist-itemerrorcolor.md)                                             | Especifica ou recupera a cor de realce que indica um item de playlist que tem uma condição de erro.                                     |
| [Arquivo de mídia](playlist-itemmedia.md)                                                       | Recupera o objeto de **mídia** correspondente ao índice fornecido no elemento **playlist** .                                               |
| [itemPlayingBackgroundColor](playlist-itemplayingbackgroundcolor.md)                     | Especifica ou recupera a cor do plano de fundo do item da playlist em execução no momento.                                                        |
| [itemPlayingColor](playlist-itemplayingcolor.md)                                         | Especifica ou recupera a cor de realce que indica o item em execução no momento na lista de reprodução.                                      |
| [de exibição](playlist-itemplaylist.md)                                                 | Recupera a lista de reprodução para o item de mídia que é exibido no índice fornecido no elemento **playlist** .                                |
| [itemSelectedBackgroundColor](playlist-itemselectedbackgroundcolor.md)                   | Especifica ou recupera um valor que indica a cor do plano de fundo de um item selecionado na lista de reprodução.                                         |
| [itemSelectedBackgroundFocusLostColor](playlist-itemselectedbackgroundfocuslostcolor.md) | Especifica ou recupera um valor que indica a cor do texto de um item selecionado na lista de reprodução.                                               |

| [statusTextColor](playlist-statustextcolor.md)                                           | Especifica ou recupera um valor que indica a cor do texto de status.                                                                    |
| [toolbarVisible](playlist-toolbarvisible.md)                                             | Especifica ou recupera um valor que indica se a barra de ferramentas da lista de reprodução é exibida.                                                           |



 

O elemento **playlist** dá suporte aos seguintes métodos.



| Método                                                              | Descrição                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [abortCopy](playlist-abortcopy.md)                                 | Cancela uma operação de cópia.                                                                                                 |
| [addSelectedToPlaylist](playlist-addselectedtoplaylist.md)         | Adiciona o item selecionado à lista de reprodução.                                                                                   |
| [copy](playlist-copy.md)                                           | Inicia uma operação de cópia do CD.                                                                                      |
| [deleteSelected](playlist-deleteselected.md)                       | Exclui o item selecionado da lista de reprodução.                                                                              |
| [deleteSelectedFromLibrary](playlist-deleteselectedfromlibrary.md) | Exclui o item selecionado da lista de reprodução e da biblioteca.                                                         |
| [getNextCheckedItem](playlist-getnextcheckeditem.md)               | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado.                               |
| [getNextCheckedItem2](playlist-getnextcheckeditem2.md)             | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado. Funciona com listas de reprodução aninhadas.  |
| [getNextSelectedItem](playlist-getnextselecteditem.md)             | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado.                              |
| [getNextSelectedItem2](playlist-getnextselecteditem2.md)           | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado. Funciona com listas de reprodução aninhadas. |
| [moveSelectedDown](playlist-moveselecteddown.md)                   | Move o item selecionado uma posição para baixo na lista.                                                                    |
| [moveSelectedUp](playlist-moveselectedup.md)                       | Move o item selecionado uma posição para cima na lista.                                                                      |
| [setcheckstate](playlist-setcheckedstate.md)                     | Especifica que o item indexado na playlist está marcado.                                                               |
| [setCheckedState2](playlist-setcheckedstate2.md)                   | Define o estado de ativação do item com o índice especificado no elemento **playlist** . Funciona com listas de reprodução aninhadas.     |
| [setColumnResizeMode](playlist-setcolumnresizemode.md)             | Especifica como os tamanhos de coluna indexados em si.                                                                            |
| [setColumnWidth](playlist-setcolumnwidth.md)                       | Especifica a largura da coluna e altera o modo de redimensionamento da coluna para "wmpcrmFixed".                                    |
| [setselecionostate](playlist-setselectedstate.md)                   | Especifica que o item indexado na lista de reprodução está selecionado.                                                              |
| [setSelectedState2](playlist-setselectedstate2.md)                 | Define o estado selecionado do item com o índice especificado no elemento **playlist** . Funciona com listas de reprodução aninhadas.    |
| [sortColumn](playlist-sortcolumn.md)                               | Classifica os dados na coluna especificada.                                                                                   |



 

O elemento **playlist** dá suporte aos atributos de ambiente e pode implementar os manipuladores de eventos de ambiente, exceto quando indicado. Para obter mais informações, consulte [atributos de ambiente](ambient-attributes.md) e [manipuladores de eventos de ambiente](ambient-event-handlers.md).

As listas de reprodução predefinidas são elementos de **playlist** normais com várias configurações de atributo comuns especificadas por padrão. As listas de reprodução predefinidas a seguir estão disponíveis.



| PLAYLIST predefinida                      | Descrição                                                       |
|------------------------------------------|-------------------------------------------------------------------|
| [DROPDOWNPLAYLIST](dropdownplaylist.md) | Uma lista suspensa de **reprodução** sem itens visíveis.                   |
| [ITEMSPLAYLIST](itemsplaylist.md)       | Uma lista suspensa de **reprodução** sem itens ou cabeçalhos de coluna visíveis. |
