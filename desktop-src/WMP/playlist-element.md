# <a name="playlist-element"></a>Elemento PLAYLIST

O **elemento PLAYLIST** fornece uma maneira de organizar itens de mídia em uma lista para facilitar a manipulação usando os seguintes atributos e métodos. Elementos **PLAYLIST predefinidos** também são fornecidos para conveniência. Colunas personalizadas podem ser especificadas para uma playlist incluindo elementos **COLUMN** como filhos do **elemento PLAYLIST.**

O **elemento PLAYLIST** dá suporte aos atributos a seguir.



| Atributo                                                                                 | Descrição                                                                                                                                |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [allowColumnSorting](playlist-allowcolumnsorting.md)                                     | Especifica ou recupera um valor que indica se a classificação em headers de coluna é permitida.                                                    |
| [allowItemEditing](playlist-allowitemediting.md)                                         | Especifica ou recupera um valor que indica se os itens em uma playlist são suportados para edição in-locar.                                            |
| [Backgroundcolor](playlist-backgroundcolor.md)                                           | Especifica ou recupera a cor da tela de fundo.                                                                                               |
| [backgroundImage](playlist-backgroundimage.md)                                           | Especifica ou recupera a imagem de plano de fundo.                                                                                               |
| [checkboxesVisible](playlist-checkboxesvisible.md)                                       | Especifica ou recupera um valor que indica se as caixas de seleção estão visíveis.                                                                  |
| [ColumnCount](playlist-columncount.md)                                                   | Recupera o número de colunas mostradas.                                                                                                     |
| [columnOrder](playlist-columnorder.md)                                                   | Especifica ou recupera a ordem das colunas da playlist.                                                                                  |
| [Colunas](playlist-columns.md)                                                           | Define as colunas que aparecem no elemento **PLAYLIST.**                                                                               |
| [columnsVisible](playlist-columnsvisible.md)                                             | Especifica ou recupera um valor que indica se as colunas são mostradas.                                                                       |
| [Copiar](playlist-copying.md)                                                           | Recupera um valor que indica se o **elemento PLAYLIST** está no ato de copiar.                                                    |
| [disabledItemColor](playlist-disableditemcolor.md)                                       | Especifica ou recupera a cor de uma faixa de CD desabilitada ou de conteúdo online quando offline.                                                 |
| [dropDownBackgroundImage](playlist-dropdownbackgroundimage.md)                           | Especifica ou recupera o nome da imagem que é exibida na tela de fundo da lista lista listada.                                        |
| [dropDownImage](playlist-dropdownimage.md)                                               | Especifica ou recupera o nome da imagem usada para o botão de lista de listas listadas que é exibido na borda direita da lista lista listada. |
| [Dropdownlist](playlist-dropdownlist.md)                                                 | Especifica ou recupera um valor que indica quais elementos aparecem na listada para uma determinada instância do elemento **PLAYLIST.**   |
| [dropDownToolTip](playlist-dropdowntooltip.md)                                           | Especifica ou recupera a Dica de Ferramenta mostrada quando o usuário passar o mouse sobre o menu suspenso do elemento **PLAYLIST.**                                |
| [dropDownVisible](playlist-dropdownvisible.md)                                           | Especifica ou recupera um valor que indica se o **seletor de** lista de opções do elemento PLAYLIST está visível.                                  |
| [editButtonVisible](playlist-editbuttonvisible.md)                                       | Especifica ou recupera um valor que indica se o botão de edição do **elemento PLAYLIST** está visível.                                         |
| [Foregroundcolor](playlist-foregroundcolor.md)                                           | Especifica ou recupera a cor de primeiro plano.                                                                                               |
| [hueShift](playlist-hueshift.md)                                                         | Especifica ou recupera a quantidade pela qual o matiz das imagens listadas é deslocado.                                                     |
| [Itemcount](playlist-itemcount.md)                                                       | Recupera o número de itens exibidos atualmente no elemento **PLAYLIST.**                                                             |
| [itemErrorColor](playlist-itemerrorcolor.md)                                             | Especifica ou recupera a cor de realçada que indica um item de playlist que tem uma condição de erro.                                     |
| [itemMedia](playlist-itemmedia.md)                                                       | Recupera o **objeto Media** correspondente ao índice determinado no elemento **PLAYLIST.**                                               |
| [itemPlayingBackgroundColor](playlist-itemplayingbackgroundcolor.md)                     | Especifica ou recupera a cor da tela de fundo do item de playlist atualmente em reprodução.                                                        |
| [itemPlayingColor](playlist-itemplayingcolor.md)                                         | Especifica ou recupera a cor de realçada que indica o item atualmente em reprodução na playlist.                                      |
| [itemPlaylist](playlist-itemplaylist.md)                                                 | Recupera a playlist do item de mídia exibido no índice determinado no elemento **PLAYLIST.**                                |
| [itemSelectedBackgroundColor](playlist-itemselectedbackgroundcolor.md)                   | Especifica ou recupera um valor que indica a cor da tela de fundo de um item selecionado na playlist.                                         |
| [itemSelectedBackgroundFocusLostColor](playlist-itemselectedbackgroundfocuslostcolor.md) | Especifica ou recupera um valor que indica a cor do texto de um item selecionado na playlist.                                               |
| [itemSelectedColor](playlist-itemselectedcolor.md)                                       | Especifica ou recupera um valor que indica a cor do texto de um item selecionado na playlist.                                               |

| [Lista](playlist-playlist.md)                                                         | Especifica ou recupera o objeto **Playlist ao** qual o elemento **PLAYLIST** fornece uma interface.                                    |

| [Saturação](playlist-saturation.md)                                                     | Especifica ou recupera o valor de saturação das imagens listadas.                                                                       |
| [statusColor](playlist-statuscolor.md)                                                   | Especifica ou recupera a cor da linha de status no elemento **PLAYLIST.**                                                           |
| [statusTextColor](playlist-statustextcolor.md)                                           | Especifica ou recupera um valor que indica a cor do texto de status.                                                                    |
| [Toolbarvisible](playlist-toolbarvisible.md)                                             | Especifica ou recupera um valor que indica se a barra de ferramentas da playlist é exibida.                                                           |



 

O **elemento PLAYLIST** dá suporte aos métodos a seguir.



| Método                                                              | Descrição                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [abortCopy](playlist-abortcopy.md)                                 | Cancela uma operação de cópia.                                                                                                 |
| [addSelectedToPlaylist](playlist-addselectedtoplaylist.md)         | Adiciona o item selecionado à playlist.                                                                                   |
| [copy](playlist-copy.md)                                           | Inicia uma operação de cópia do CD.                                                                                      |
| [deleteSelected](playlist-deleteselected.md)                       | Exclui o item selecionado da playlist.                                                                              |
| [deleteSelectedFromLibrary](playlist-deleteselectedfromlibrary.md) | Exclui o item selecionado da lista de reprodução e da biblioteca.                                                         |
| [getNextCheckedItem](playlist-getnextcheckeditem.md)               | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado.                               |
| [getNextCheckedItem2](playlist-getnextcheckeditem2.md)             | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado. Funciona com listas de reprodução aninhadas.  |
| [getNextSelectedItem](playlist-getnextselecteditem.md)             | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado.                              |
| [getNextSelectedItem2](playlist-getnextselecteditem2.md)           | Recupera o índice do próximo item selecionado na lista de reprodução após o índice especificado. Funciona com listas de reprodução aninhadas. |
| [moveSelectedDown](playlist-moveselecteddown.md)                   | Move o item selecionado uma posição para baixo na lista.                                                                    |
| [moveSelectedUp](playlist-moveselectedup.md)                       | Move o item selecionado uma posição para cima na lista.                                                                      |
| [setcheckstate](playlist-setcheckedstate.md)                     | Especifica que o item indexado na playlist está marcado.                                                               |
| [setCheckedState2](playlist-setcheckedstate2.md)                   | Define o estado de ativação do item com o índice especificado no elemento **playlist** . Funciona com listas de reprodução aninhadas.     |
| [setColumnResizeMode](playlist-setcolumnresizemode.md)             | Especifica como os tamanhos de coluna indexados em si.                                                                            |
| [setColumnWidth](playlist-setcolumnwidth.md)                       | Especifica a largura da coluna e altera o modo de redimensionamento da coluna para "wmpcrmFixed".                                    |
| [setselecionostate](playlist-setselectedstate.md)                   | Especifica que o item indexado na lista de reprodução está selecionado.                                                              |
| [setSelectedState2](playlist-setselectedstate2.md)                 | Define o estado selecionado do item com o índice especificado no elemento **playlist** . Funciona com listas de reprodução aninhadas.    |
| [sortColumn](playlist-sortcolumn.md)                               | Classifica os dados na coluna especificada.                                                                                   |



 

O elemento **playlist** dá suporte aos atributos de ambiente e pode implementar os manipuladores de eventos de ambiente, exceto quando indicado. Para obter mais informações, consulte [atributos de ambiente](ambient-attributes.md) e [manipuladores de eventos de ambiente](ambient-event-handlers.md).

As listas de reprodução predefinidas são elementos de **playlist** normais com várias configurações de atributo comuns especificadas por padrão. As listas de reprodução predefinidas a seguir estão disponíveis.



| PLAYLIST predefinida                      | Descrição                                                       |
|------------------------------------------|-------------------------------------------------------------------|
| [DROPDOWNPLAYLIST](dropdownplaylist.md) | Uma lista suspensa de **reprodução** sem itens visíveis.                   |
| [ITEMSPLAYLIST](itemsplaylist.md)       | Uma lista suspensa de **reprodução** sem itens ou cabeçalhos de coluna visíveis. |
