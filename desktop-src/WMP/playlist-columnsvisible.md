# <a name="playlistcolumnsvisible"></a>PLAYLIST. columnsVisible

O atributo **columnsVisible** especifica ou recupera um valor que indica se os cabeçalhos de coluna são mostrados.

``` syntax
        elementID.columnsVisible
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                        |
|-------|------------------------------------|
| true  | Padrão. Os cabeçalhos de coluna são mostrados. |
| false | Os cabeçalhos de coluna não são mostrados.      |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
