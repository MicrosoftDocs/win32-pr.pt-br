# <a name="playerapplicationplayerdocked"></a>PLAYERAPPLICATION.playerDocked

O atributo **playerDocked** recupera um valor que indica se o Windows Media Player está em um estado encaixado.

``` syntax
        elementID.playerDocked
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** somente leitura.



| Valor | Descrição                       |
|-------|-----------------------------------|
| True  | O Windows Media Player está encaixado.   |
| Falso | O Windows Media Player está desencaixado. |



 

## <a name="remarks"></a>Comentários

Esse atributo é usado somente quando você faz a comunicação remota do controle do Windows Media Player.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
