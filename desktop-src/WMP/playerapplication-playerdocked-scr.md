# <a name="playerapplicationplayerdocked"></a>PLAYERAPPLICATION.playerDocked

o atributo **playerDocked** recupera um valor que indica se Windows Media Player está em um estado encaixado.

``` syntax
        elementID.playerDocked
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** somente leitura.



| Valor | Descrição                       |
|-------|-----------------------------------|
| True  | Windows Media Player está encaixada.   |
| Falso | Windows Media Player está desencaixado. |



 

## <a name="remarks"></a>Comentários

esse atributo é usado somente quando você faz a comunicação remota do controle de Windows Media Player.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
