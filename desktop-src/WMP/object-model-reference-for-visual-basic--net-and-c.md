# <a name="object-model-reference-for-visual-basic-net-and-c"></a>Referência de modelo de objeto para Visual Basic .NET e C #

> [!Note]  
> O uso do controle ActiveX do Windows Media Player com Visual Basic .NET e C# tem suporte apenas para o SDK do Windows Media Player 9 Series e posterior.

 

Esta seção de referência fornece documentação para o objeto **AxWindowsMediaPlayer** e para interfaces para o controle ActiveX do Windows Media Player. Essa funcionalidade está disponível quando você está usando o controle ActiveX do Windows Media Player por meio da interoperabilidade COM em uma solução do Visual Studio .NET 2002 ou posterior.

Para ver uma demonstração do uso do controle do Windows Media Player com o .NET Framework, consulte os exemplos de Visual Basic e C# incluídos neste SDK.

As seções a seguir documentam os objetos e interfaces do Visual Basic .NET e C# fornecidos pelo assembly de interoperabilidade COM que o Visual Studio cria.

> [!Note]  
> A funcionalidade que requer comunicação remota do controle do Windows Media Player não tem suporte para programação .NET.

| [Interfaces para Visual Basic .NET e C #](interfaces-for-visual-basic--net-and-c.md) | Lista as interfaces disponíveis para programadores do .NET que usam o controle ActiveX do Windows Media Player.                           |
