# <a name="object-model-reference-for-visual-basic-net-and-c"></a>referência de modelo de objeto para Visual Basic .net e C #

> [!Note]  
> o uso do controle de ActiveX Windows Media Player com Visual Basic .net e C# tem suporte apenas para o SDK do Windows Media Player 9 Series e posterior.

 

esta seção de referência fornece documentação para o objeto **AxWindowsMediaPlayer** e para interfaces para o controle de ActiveX de Windows Media Player. essa funcionalidade está disponível quando você está usando o controle de ActiveX Windows Media Player por meio da interoperabilidade COM em uma solução Visual Studio .net 2002 ou posterior.

para ver uma demonstração do uso do controle de Windows Media Player com o .NET Framework, consulte os exemplos de Visual Basic e C# incluídos neste SDK.

as seções a seguir documentam os objetos e interfaces do Visual Basic .net e C# fornecidos pelo assembly de interoperabilidade COM que o Visual Studio cria.

> [!Note]  
> a funcionalidade que exige comunicação remota do controle de Windows Media Player não tem suporte para programação .net.

 



| Seção                                                                                | Descrição                                                                                                                    |
|----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [Objeto AxWindowsMediaPlayer (VB e C#)](axwindowsmediaplayer-object--vb-and-c.md)  | liste as propriedades, os métodos e os eventos disponíveis para os programadores do .net para o objeto raiz do controle de Windows Media Player. |
| [Interfaces para Visual Basic .net e C #](interfaces-for-visual-basic--net-and-c.md) | lista as interfaces disponíveis para programadores do .net que usam o controle de ActiveX de Windows Media Player.                           |
