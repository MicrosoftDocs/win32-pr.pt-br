# <a name="listboxmultiselect"></a>Caixa de listagem. seleção

O atributo de **seleção** múltipla especifica ou recupera um valor que indica se o usuário pode selecionar várias linhas. Só pode ser definido em tempo de design.

``` syntax
        elementID.multiSelect
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** especificado em tempo de design e somente leitura depois.



| Valor | Descrição                                         |
|-------|-----------------------------------------------------|
| true  | O usuário pode selecionar várias linhas.                 |
| false | Padrão. O usuário só pode selecionar linhas individuais. |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
