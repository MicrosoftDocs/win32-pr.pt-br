# <a name="listboxmultiselect"></a>LISTBOX.multiSelect

O **atributo multiSelect** especifica ou recupera um valor que indica se o usuário pode selecionar várias linhas. Só pode ser definido em tempo de design.

``` syntax
        elementID.multiSelect
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliana** especificado em tempo de design e somente leitura depois disso.



| Valor | Descrição                                         |
|-------|-----------------------------------------------------|
| true  | O usuário pode selecionar várias linhas.                 |
| false | Padrão. O usuário só pode selecionar linhas individuais. |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
