# <a name="windows-media-player"></a>Windows Media Player

O SDK do Windows Media Player fornece recursos que afetam o comportamento do Windows Media Player e do Windows Media Player para Windows Mobile.

As seções a seguir detalham os recursos do SDK que se aplicam ao Windows Media Player em geral.



| Seção                                                                                                        | Descrição                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Configurações do registro](registry-settings.md)                                                                     | Os valores de detalhes que podem ser alterados no registro do usuário para permitir que o Windows Media Player reconheça extensões de nome de arquivo personalizadas.                                    |
| [Parâmetros de linha de comando](command-line-parameters.md)                                                         | Detalha o conjunto de parâmetros de linha de comando que especificam como o Windows Media Player se comporta quando é iniciado.                                                              |
| [Saídas de áudio](audio-outputs.md)                                                                             | Descreve como o Windows Media Player e o controle ActiveX do Windows Media Player escolhem dispositivos de saída de áudio padrão.                                                  |
| [Redistribuindo o software Windows Media Player](redistributing-windows-media-player-software.md)               | Fornece informações sobre a redistribuição do Windows Media Player.                                                                                                       |
| [Codecs de publicação do Windows Media Player](publishing-codecs-for-windows-media-player.md)                   | Não há mais suporte para a publicação do seu codec no site do WMPlugins.                                                                                                |
| [Redimensionando a caixa de diálogo de aquisição de licença](resizing-the-license-acquisition-dialog-box.md)                 | Descreve como modificar um atributo de arquivo de mídia do Windows para especificar um tamanho para a caixa de diálogo de aquisição de licença do Windows Media Player 10 ou posterior.                     |
| [Extensões de dispositivo para transferência de metadados acelerada](device-extensions-for-accelerated-metadata-transfer.md) | Descreve como o Windows Media Player recupera informações de dispositivos portáteis sobre alterações que ocorrem em itens de conteúdo específicos entre sessões de sincronização. |
| [Extensões de dispositivo para relatórios de conteúdo adquirido](device-extensions-for-reporting-acquired-content.md)       | Descreve como o Windows Media Player pode recuperar uma lista de novos conteúdos adquiridos por um dispositivo portátil entre sessões de sincronização.                                 |
| [Extensões de dispositivo para preferências de objeto de playlist](device-extensions-for-playlist-object-preferences.md)     | Descreve como um dispositivo portátil pode especificar quais tipos de objeto de lista de reprodução o Windows Media Player deve copiar para o dispositivo durante a sincronização automática.                |
| [Códigos de controle de e/s de dispositivo](device-i-o-control-codes.md)                                                       | Lista códigos de controle de e/s de dispositivo usados pelo Windows Media Player 10 ou posterior para se comunicar com dispositivos portáteis por meio do SDK do Windows Media Gerenciador de Dispositivos.            |
| [Suporte a imagens personalizadas para dispositivos](custom-image-support-for-devices.md)                                       | Descreve dois arquivos de imagem que os fabricantes de dispositivos portáteis podem criar para personalizar a identidade visual no Windows Media Player 10 ou posterior.                                    |
