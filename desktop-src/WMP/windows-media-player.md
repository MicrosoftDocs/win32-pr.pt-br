# <a name="windows-media-player"></a>Windows Media Player

o SDK do Windows Media Player fornece recursos que afetam o comportamento de Windows Media Player e Windows Media Player para Windows Mobile.

as seções a seguir detalham os recursos do SDK que se aplicam ao Windows Media Player em geral.



| Seção                                                                                                        | Descrição                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Configurações do Registro](registry-settings.md)                                                                     | os valores de detalhes que você pode alterar no registro do usuário para permitir que Windows Media Player reconheçam extensões de nome de arquivo personalizado.                                    |
| [Parâmetros de linha de comando](command-line-parameters.md)                                                         | detalha o conjunto de parâmetros de linha de comando que especificam como o Windows Media Player se comporta quando é iniciado.                                                              |
| [Saídas de áudio](audio-outputs.md)                                                                             | descreve como Windows Media Player e o Windows Media Player controle de ActiveX escolher dispositivos de saída de áudio padrão.                                                  |
| [redistribuindo Windows Media Player Software](redistributing-windows-media-player-software.md)               | Fornece informações sobre a redistribuição de Windows Media Player.                                                                                                       |
| [Publicando codecs para Windows Media Player](publishing-codecs-for-windows-media-player.md)                   | Não há mais suporte para a publicação do seu codec no site do WMPlugins.                                                                                                |
| [Redimensionando a caixa de diálogo de aquisição de licença](resizing-the-license-acquisition-dialog-box.md)                 | descreve como modificar um atributo de arquivo de mídia Windows para especificar um tamanho para a caixa de diálogo de aquisição de licença Windows Media Player 10 ou posterior.                     |
| [Extensões de dispositivo para transferência de metadados acelerada](device-extensions-for-accelerated-metadata-transfer.md) | descreve como Windows Media Player recupera informações de dispositivos portáteis sobre alterações que ocorrem em itens de conteúdo específicos entre sessões de sincronização. |
| [Extensões de dispositivo para relatórios de conteúdo adquirido](device-extensions-for-reporting-acquired-content.md)       | descreve como Windows Media Player pode recuperar uma lista de novos conteúdos adquiridos por um dispositivo portátil entre sessões de sincronização.                                 |
| [Extensões de dispositivo para preferências de objeto de playlist](device-extensions-for-playlist-object-preferences.md)     | descreve como um dispositivo portátil pode especificar quais tipos de objeto de playlist Windows Media Player devem ser copiados para o dispositivo durante a sincronização automática.                |
| [Códigos de controle de e/s de dispositivo](device-i-o-control-codes.md)                                                       | lista códigos de controle de e/s de dispositivo usados pelo Windows Media Player 10 ou posterior para se comunicar com dispositivos portáteis por meio do SDK do Windows Media Gerenciador de Dispositivos.            |
| [Suporte a imagens personalizadas para dispositivos](custom-image-support-for-devices.md)                                       | descreve dois arquivos de imagem que os fabricantes de dispositivos portáteis podem criar para personalizar a identidade visual no Windows Media Player 10 ou posterior.                                    |
