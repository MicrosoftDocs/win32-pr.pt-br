# <a name="playlistitemselectedbackgroundcolor"></a>PLAYLIST. itemSelectedBackgroundColor

O atributo **itemSelectedBackgroundColor** especifica ou recupera um valor que indica a cor do plano de fundo de um item selecionado na lista de reprodução.

``` syntax
        elementID.itemSelectedBackgroundColor
          
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que representa uma cor. O valor padrão é a cor de realce do Windows.

| Requisito | Valor |
|--------------------|------------------------------------|
| Versão<br/> | Windows Media Player 10<br/> |
