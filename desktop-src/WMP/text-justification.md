# <a name="textjustification"></a>TEXTO. justificativa

O atributo de **justificativa** especifica ou recupera o alinhamento do texto dentro do controle de texto.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Valores possíveis

Este atributo é uma **cadeia de caracteres** de leitura/gravação.



| Valor  | Descrição                                                   |
|--------|---------------------------------------------------------------|
| Esquerda   | Padrão. Alinha o texto à esquerda do controle de texto.     |
| Direita  | Alinha o texto à direita do controle de texto.             |
| Centro | Alinha o texto ao centro horizontal do controle de texto. |
