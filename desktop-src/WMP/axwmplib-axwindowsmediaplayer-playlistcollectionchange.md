# <a name="playlistcollectionchange-event-of-the-axwindowsmediaplayer-object"></a>Evento PlaylistCollectionChange do objeto AxWindowsMediaPlayer

O evento PlaylistCollectionChange ocorre quando algo é alterado na coleção de playlist.

``` syntax
[C#]
private void player_PlaylistCollectionChange(
  object sender,
  System.EventArgs e
)

[Visual Basic]
Private Sub player_PlaylistCollectionChange(
  sender As Object,
  e As System.EventArgs
) Handles player.PlaylistCollectionChange
```

## <a name="event-data"></a>Dados de evento

Este evento não contém dados.
