# <a name="playlistitemplayingbackgroundcolor"></a>PLAYLIST. itemPlayingBackgroundColor

O atributo **itemPlayingBackgroundColor** especifica ou recupera a cor do plano de fundo do item da playlist em execução no momento.

``` syntax
        elementID.itemPlayingBackgroundColor
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém qualquer valor de cor do Microsoft Internet Explorer. Ele tem um valor padrão de " \# 222222".

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
