# <a name="wmmood-attribute"></a>Atributo WM/humor

O atributo **WM/humor** é um nome de categoria para o humor do conteúdo.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de áudio](audio-item-attributes.md)
-   [Atributos de arquivo de mídia do Windows comumente usados](commonly-used-windows-media-file-attributes.md)

## <a name="remarks"></a>Comentários

Esse atributo é armazenado somente no arquivo de mídia digital.

**Humor** é um alias para este atributo.

A constante do Windows Media Format SDK para esse atributo é g \_ wszWMMood.

Para determinar se você pode alterar o valor desse atributo, use o método [Media. isReadOnlyItem](media-isreadonlyitem.md) .

|--------------------|---------------------------------------------------|
