# <a name="videomaintainaspectratio"></a>VÍDEO. maintainAspectRatio

O atributo **maintainAspectRatio** especifica ou recupera um valor que indica se o vídeo manterá sua taxa de proporção ao tentar se ajustar dentro da largura e altura definidas para o controle.

``` syntax
        elementID.maintainAspectRatio
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                                              |
|-------|----------------------------------------------------------|
| true  | Padrão. O vídeo mantém sua taxa de proporção ao redimensionar. |
| false | O vídeo não mantém sua taxa de proporção ao redimensionar.  |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
