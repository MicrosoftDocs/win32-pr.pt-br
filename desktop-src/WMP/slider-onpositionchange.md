# <a name="slideronpositionchange"></a>SLIDER. onPositionChange

O manipulador de eventos **onPositionChange** manipula um evento que ocorre quando a posição do controle deslizante é alterada como resultado do usuário clicar ou arrastar.

``` syntax
onPositionChange
```

## <a name="remarks"></a>Comentários

Se a posição do controle deslizante for alterada como resultado do atributo **Value** que está sendo modificado no script, esse evento não será acionado. Para acomodar essa possibilidade, implemente o valor do manipulador de eventos **\_ onChange** .
