# <a name="windows-media-metafile-elements-reference"></a>Referência de elementos de metarquivo do Windows Media

Esta seção contém a documentação de referência para todos os elementos do metarquivo do Windows Media para aplicativos do lado do cliente. As entradas de referência incluem definições dos elementos, seus atributos e valores e condições especiais relacionadas a cada elemento.

Os seguintes elementos de metarquivo têm suporte para aplicativos do lado do cliente.



| Elemento                                           | Descrição                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [RESUME](abstract-element.md)                  | Contém o texto que descreve o elemento **ASX**, **banner** ou **entry** associado.                                 |
| [ASX](asx-element.md)                            | Define um arquivo como um metarquivo do Windows Media.                                                                            |
| [AUTORIZAÇÃO](author-element.md)                      | Contém o nome do autor de um clipe de mídia ou um metarquivo do Windows Media.                                           |
| [TIMBRADO](banner-element.md)                      | Especifica uma URL para um elemento gráfico que aparece no painel de exibição do Windows Media Player.                               |
| [POLYBASE](base-element.md)                          | Especifica uma cadeia de caracteres que é anexada à frente das URLs enviadas ao Windows Media Player.                                 |
| [Comentários](comments.md)                          | Especifica a sintaxe XML para comentários ( <!--...--> ).                                                            |
| [INTERNACIONAIS](copyright-element.md)                | Contém as informações de direitos autorais de um elemento **ASX** ou de **entrada** .                                                |
| [DURATION](duration-element.md)                  | Especifica o período de tempo durante o qual o Windows Media Player renderiza um fluxo.                                                    |
| [MARCA de fim](endmarker-element.md)                | Especifica um marcador no qual o Windows Media Player para de renderizar o fluxo.                                           |
| [INICIAL](entry-element.md)                        | Especifica o caminho para um clipe de mídia.                                                                                   |
| [ENTRYREF](entryref-element.md)                  | Links para os elementos de **entrada** em um metarquivo do Windows Media externo que tem uma extensão. ASX,. Wax,. wvx ou. WMX. |
| [CIRCUNSTÂNCIA](event-element.md)                        | Define um comportamento ou uma ação realizada pelo Windows Media Player quando ele recebe um comando de script rotulado como um evento.      |
| [LOGURL](logurl-element.md)                      | Instrui o Windows Media Player a enviar quaisquer dados de log para a URL especificada.                                            |
| [MOREINFO](moreinfo-element.md)                  | Especifica uma URL para um site, endereço de email ou comando de script associado a um show, clip ou banner.               |
| [PARAM](param-element.md)                        | Especifica o valor de um parâmetro personalizado associado a uma lista de reprodução ou a um elemento de uma lista de reprodução.                      |
| [PREVIEWDURATION](previewduration-element.md)    | Especifica o período de tempo que um clipe é reproduzido no modo de visualização.                                                         |
| [REFERÊNCIA](ref-element.md)                            | Especifica uma URL para uma parte do conteúdo de mídia digital.                                                                  |
| [Pete](repeat-element.md)                      | Especifica o número de vezes que o Windows Media Player repete um ou mais elementos de **entrada** ou **ENTRYREF** .             |
| [CAPA (preterida)](skin-element--deprecated.md) | Especifica uma URL para uma capa inserida.                                                                                   |
| [STARTMARKER](startmarker-element.md)            | Especifica um marcador no qual o Windows Media Player começa a renderizar o fluxo.                                          |
| [STARTTIME](starttime-element.md)                | Especifica uma hora na qual o Windows Media Player começará a renderizar o fluxo.                                        |
| [TÍTULO](title-element--metafile.md)              | Contém o título de um elemento **ASX** ou de **entrada** .                                                                 |
