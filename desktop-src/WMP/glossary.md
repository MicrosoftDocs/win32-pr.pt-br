# <a name="glossary-windows-media-player-sdk"></a>Glossário (SDK do Windows Media Player)

<dl> <dt>

<span id="projectid515.ambient"></span><span id="PROJECTID515.AMBIENT"></span>**ambiente**
</dt> <dd>

Com suporte em mais de um elemento Skin.

</dd> <dt>

<span id="projectid515.analog"></span><span id="PROJECTID515.ANALOG"></span>**analógico**
</dt> <dd>

O formato tradicional no qual áudio e vídeo são transmitidos usando um sinal Wave ou analógico. Um sinal analógico pode não funcionar com alto-falantes digitais; os computadores usam sinais digitais.

</dd> <dt>

<span id="projectid515.anchor_window"></span><span id="PROJECTID515.ANCHOR_WINDOW"></span>**janela de ancoragem**
</dt> <dd>

Uma pequena janela que pode aparecer no canto inferior direito da tela quando o Windows Media Player está no modo de capa. Você pode usar a janela para retornar ao modo completo e acessar outros comandos.

</dd> <dt>

<span id="projectid515.announcement"></span><span id="PROJECTID515.ANNOUNCEMENT"></span>**anúncio**
</dt> <dd>

Um metarquivo do Windows Media que fornece a um player as informações necessárias para receber conteúdo. Os arquivos de anúncio contêm scripts de linguagem XML (XML).

</dd> <dt>

<span id="projectid515.aspect_ratio"></span><span id="PROJECTID515.ASPECT_RATIO"></span>**taxa de proporção**
</dt> <dd>

A razão entre a largura de uma imagem e sua altura.

</dd> <dt>

<span id="projectid515.attribute"></span><span id="PROJECTID515.ATTRIBUTE"></span>**atributo**
</dt> <dd>

Um par de dados de nome-valor.

</dd> <dt>

<span id="projectid515.bandwidth"></span><span id="PROJECTID515.BANDWIDTH"></span>**largura de banda**
</dt> <dd>

A capacidade de transferência de dados de um meio de transmissão.

</dd> <dt>

<span id="projectid515.bit_rate"></span><span id="PROJECTID515.BIT_RATE"></span>**taxa de bits**
</dt> <dd>

O número de bits transferidos por unidade de tempo, normalmente expressos em bits por segundo.

</dd> <dt>

<span id="projectid515.broadcast"></span><span id="PROJECTID515.BROADCAST"></span>**difusão**
</dt> <dd>

Um método pelo qual um cliente recebe um fluxo. Durante uma conexão de difusão, os clientes não podem controlar o fluxo. Esse é o oposto de uma apresentação sob demanda.

</dd> <dt>

<span id="projectid515.broadcast_multicast"></span><span id="PROJECTID515.BROADCAST_MULTICAST"></span>difusão **multicast**
</dt> <dd>

Entrega de um fluxo por um servidor do Windows Media para muitos clientes. Da perspectiva do cliente, uma difusão é uma experiência sem conexão, pois o cliente nunca se conecta a um servidor do Windows Media.

</dd> <dt>

<span id="projectid515.buffer"></span><span id="PROJECTID515.BUFFER"></span>**buffer**
</dt> <dd>

Uma área de memória reservada para manter dados temporariamente antes que esses dados sejam usados por um dispositivo ou aplicativo receptor. O armazenamento em buffer protege contra a interrupção do fluxo de dados.

</dd> <dt>

<span id="projectid515.caption"></span><span id="PROJECTID515.CAPTION"></span>**legenda**
</dt> <dd>

Texto que acompanha imagens ou vídeos, como uma descrição suplementar ou uma transcrição de palavras faladas.

</dd> <dt>

<span id="projectid515.cd"></span><span id="PROJECTID515.CD"></span>**CD**
</dt> <dd>

Consulte a definição de: CD (CD)

</dd> <dt>

<span id="projectid515.client"></span><span id="PROJECTID515.CLIENT"></span>**cliente** do
</dt> <dd>

Qualquer computador ou programa que se conecte ou solicite os serviços do, outro computador ou programa. O cliente também pode se referir ao software que permite que o computador ou programa estabeleça a conexão.

</dd> <dt>

<span id="projectid515.clipping_image"></span><span id="PROJECTID515.CLIPPING_IMAGE"></span>**imagem de recorte**
</dt> <dd>

Uma imagem que não é exibida, mas, em vez disso, define a região visível de outra imagem.

</dd> <dt>

<span id="projectid515.closed_disc"></span><span id="PROJECTID515.CLOSED_DISC"></span>**disco fechado**
</dt> <dd>

Um CD ou DVD que passou por um processo durante a gravação, o que torna o conteúdo do disco visível para a reprodução de programas e dispositivos. Depois que um disco é fechado, nenhum dado adicional pode ser gravado nele. Alguns dispositivos de reprodução, como alguns aparelhos portáteis, domésticos e de CD e DVD players, não podem reproduzir discos que não foram fechados.

</dd> <dt>

<span id="projectid515.codec"></span><span id="PROJECTID515.CODEC"></span>**codec** do
</dt> <dd>

Uma abreviação de compressor/descompactador. Software ou hardware usado para compactar e descompactar mídia digital.

</dd> <dt>

<span id="projectid515.compact_disc__cd_"></span><span id="PROJECTID515.COMPACT_DISC__CD_"></span>CDs **(CD)**
</dt> <dd>

Um meio de armazenamento óptico para dados digitais.

</dd> <dt>

<span id="projectid515.compression"></span><span id="PROJECTID515.COMPRESSION"></span>**compactação**
</dt> <dd>

Um processo para remover dados redundantes de um arquivo de mídia digital ou fluxo para reduzir seu tamanho ou a largura de banda usada.

</dd> <dt>

<span id="projectid515.content"></span><span id="PROJECTID515.CONTENT"></span>**conteúdo** do
</dt> <dd>

Áudio, vídeo, imagens, texto ou qualquer outra informação contida em um fluxo ou arquivo de mídia digital.

</dd> <dt>

<span id="projectid515.down_image"></span><span id="PROJECTID515.DOWN_IMAGE"></span>**imagem inativa**
</dt> <dd>

Uma imagem que é exibida quando o botão do mouse está na posição para baixo sobre um controle.

</dd> <dt>

<span id="projectid515.download_def"></span><span id="PROJECTID515.DOWNLOAD_DEF"></span>**baixar**
</dt> <dd>

Para copiar um arquivo de um computador para outro usando um modem ou rede.

</dd> <dt>

<span id="projectid515.element"></span><span id="PROJECTID515.ELEMENT"></span>**elemento**
</dt> <dd>

Uma unidade sintática fundamental em linguagens de marcação, como HTML ou XML. Os elementos são delimitados por marcas de início e marcas de fim. Elementos vazios são definidos usando uma marca de elemento vazio.

</dd> <dt>

<span id="projectid515.encode"></span><span id="PROJECTID515.ENCODE"></span>**codificar**
</dt> <dd>

Para converter o conteúdo de áudio e vídeo em um formato digital especificado, geralmente envolvendo tecnologia de compactação ou tecnologia de criptografia.

</dd> <dt>

<span id="projectid515.error_correction"></span><span id="PROJECTID515.ERROR_CORRECTION"></span>**correção de erro**
</dt> <dd>

No Windows Media Player, um processo para garantir que os dados de áudio digital sejam lidos da unidade de CD-ROM com precisão durante a reprodução ou cópia. O uso da correção de erro pode impedir ruídos indesejáveis que não fazem parte do material original.

</dd> <dt>

<span id="projectid515.extensible_markup_language__xml_"></span><span id="PROJECTID515.EXTENSIBLE_MARKUP_LANGUAGE__XML_"></span>**Linguagem XML (XML)**
</dt> <dd>

Uma linguagem de marcação que fornece um formato para descrever dados estruturados. XML é uma especificação de World Wide Web Consortium (W3C) e é um subconjunto da linguagem SGML.

</dd> <dt>

<span id="projectid515.file_format"></span><span id="PROJECTID515.FILE_FORMAT"></span>**formato de arquivo**
</dt> <dd>

A estrutura ou organização dos dados em um arquivo. O formato de arquivo geralmente é indicado pela extensão de nome de arquivo.

</dd> <dt>

<span id="projectid515.file_name_extension"></span><span id="PROJECTID515.FILE_NAME_EXTENSION"></span>**extensão de nome de arquivo**
</dt> <dd>

Um conjunto de caracteres adicionado ao final de um nome de arquivo que identifica o tipo ou o formato do arquivo.

</dd> <dt>

<span id="projectid515.file_type"></span><span id="PROJECTID515.FILE_TYPE"></span>**tipo de arquivo**
</dt> <dd>

O formato de um arquivo, normalmente indicado por sua extensão de nome de arquivo. O tipo de arquivo indica em qual programa o arquivo foi criado e pode ser aberto com.

</dd> <dt>

<span id="projectid515.frame_rate"></span><span id="PROJECTID515.FRAME_RATE"></span>**taxa de quadros**
</dt> <dd>

O número de quadros de vídeo exibidos por segundo. Taxas de quadros mais altas geralmente produzem movimento mais suave na imagem.

</dd> <dt>

<span id="projectid515.full_mode"></span><span id="PROJECTID515.FULL_MODE"></span>**modo completo**
</dt> <dd>

O estado operacional padrão do Windows Media Player do qual todos os seus recursos estão disponíveis. O Player também pode aparecer no modo de capa, no modo de mini Player ou no modo de tela inteira.

</dd> <dt>

<span id="projectid515.global_attribute"></span><span id="PROJECTID515.GLOBAL_ATTRIBUTE"></span>**atributo global**
</dt> <dd>

Um atributo associado a um arquivo de definição de capa como um todo, que pode ser acessado de qualquer elemento.

</dd> <dt>

<span id="projectid515.hover_image"></span><span id="PROJECTID515.HOVER_IMAGE"></span>**imagem em foco**
</dt> <dd>

Uma imagem que é exibida para um controle sempre que o ponteiro do mouse passa sobre ele.

</dd> <dt>

<span id="projectid515.http"></span><span id="PROJECTID515.HTTP"></span>**Http**
</dt> <dd>

Consulte a definição para: protocolo HTTP (Hypertext Transfer Protocol)

</dd> <dt>

<span id="projectid515.hypertext_transfer_protocol__http_"></span><span id="PROJECTID515.HYPERTEXT_TRANSFER_PROTOCOL__HTTP_"></span>**Protocolo http (Hypertext Transfer Protocol)**
</dt> <dd>

Um conjunto padrão de regras que os computadores usam para transferir informações pela Internet.

</dd> <dt>

<span id="projectid515.intelligent_streaming"></span><span id="PROJECTID515.INTELLIGENT_STREAMING"></span>**streaming inteligente**
</dt> <dd>

Um tipo de streaming que detecta condições de rede e ajusta as propriedades de um vídeo ou fluxo de áudio para maximizar a qualidade.

</dd> <dt>

<span id="projectid515.library"></span><span id="PROJECTID515.LIBRARY"></span>**biblioteca** do
</dt> <dd>

No Windows Media Player, um banco de dados que contém informações sobre arquivos de mídia digital em um computador, uma unidade de rede ou a Internet.

</dd> <dt>

<span id="projectid515.licensed_file"></span><span id="PROJECTID515.LICENSED_FILE"></span>**arquivo licenciado**
</dt> <dd>

Um arquivo de mídia do Windows que tem uma licença associada que define como o arquivo pode ser reproduzido. As restrições indicadas na licença variam de acordo com o criador da licença. Quando uma faixa de CD é copiada usando o Windows Media Player, uma licença pode ser atribuída ao arquivo recém-criado. Sob essa licença, o arquivo só pode ser reproduzido no computador em que o arquivo foi criado.

</dd> <dt>

<span id="projectid515.lossy_compression"></span><span id="PROJECTID515.LOSSY_COMPRESSION"></span>**compactação com perdas**
</dt> <dd>

Um processo para compactação de dados em que as informações consideradas desnecessárias é removido e não pode ser recuperado após a descompactação. Normalmente usado com áudio e dados visuais em que uma pequena degradação da qualidade é aceitável.

</dd> <dt>

<span id="projectid515.mapping_image"></span><span id="PROJECTID515.MAPPING_IMAGE"></span>**imagem de mapeamento**
</dt> <dd>

Uma imagem que não é exibida, mas, em vez disso, define regiões significativas em outra imagem com o uso de cores.

</dd> <dt>

<span id="projectid515.media_usage_rights"></span><span id="PROJECTID515.MEDIA_USAGE_RIGHTS"></span>**direitos de uso de mídia**
</dt> <dd>

Permissões para usar um arquivo protegido baseado no Windows Media de uma maneira específica. Por exemplo, um provedor de conteúdo, como uma loja de música online, pode conceder a você o direito de reproduzir uma música em seu computador (um direito de jogo), gravar a música em um CD de áudio (um direito de gravação) ou sincronizar a música com um dispositivo portátil (um direito de sincronização). O provedor de conteúdo determina os termos dos direitos (por exemplo, quais direitos ele concede a você e por quanto tempo) e o componente DRM (Windows Media Digital Rights Management) impõe esses direitos. Também conhecido como licença.

</dd> <dt>

<span id="projectid515.metadata"></span><span id="PROJECTID515.METADATA"></span>**metadados** do
</dt> <dd>

Dados sobre dados. Título, assunto, autor e tamanho são exemplos de metadados de um arquivo.

</dd> <dt>

<span id="projectid515.metafile_playlist"></span><span id="PROJECTID515.METAFILE_PLAYLIST"></span>**playlist de metarquivo**
</dt> <dd>

Um metarquivo do Windows Media que contém informações sobre uma série de itens de mídia digital.

</dd> <dt>

<span id="projectid515.method"></span><span id="PROJECTID515.METHOD"></span>**método**
</dt> <dd>

Uma função de programação que está relacionada a um objeto específico e pode ser chamada para modificar as propriedades do objeto ou para obter um efeito específico para o qual o objeto é responsável. Os métodos podem ser passados com partes de informações chamadas de parâmetros para modificar o comportamento resultante. Frequentemente, o resultado de uma chamada de método é o retorno de uma parte diferente das informações que podem ser usadas por outra parte do programa.

</dd> <dt>

<span id="projectid515.microsoft_media_server__mms__protocol"></span><span id="PROJECTID515.MICROSOFT_MEDIA_SERVER__MMS__PROTOCOL"></span>**Protocolo MMS (Microsoft Media Server)**
</dt> <dd>

Um protocolo proprietário usando UDP ou TCP para fornecer conteúdo como um fluxo unicast.

</dd> <dt>

<span id="projectid515.midi"></span><span id="PROJECTID515.MIDI"></span>**MIDI**
</dt> <dd>

Consulte a definição para: MIDI (interface digital de instrumento musical)

</dd> <dt>

<span id="projectid515.mime"></span><span id="PROJECTID515.MIME"></span>**MIME**
</dt> <dd>

Consulte a definição para: MIME (Multipurpose Internet Mail Extension)

</dd> <dt>

<span id="projectid515.mms_protocol"></span><span id="PROJECTID515.MMS_PROTOCOL"></span>**Protocolo MMS**
</dt> <dd>

Consulte a definição de: protocolo MMS (Microsoft Media Server)

</dd> <dt>

<span id="projectid515.moving_picture_experts_group__mpeg_"></span><span id="PROJECTID515.MOVING_PICTURE_EXPERTS_GROUP__MPEG_"></span>**Grupo MPEG (MPEG)**
</dt> <dd>

O Comitê que cria padrões internacionais para codificação de informações visuais de áudio para um formato digital e compactado. O acrônimo MPEG é acrescentado ao início das especificações individuais desenvolvidas pelo Comitê. Por exemplo, MPEG-2 refere-se ao ISO/IEC-11172 padrão.

</dd> <dt>

<span id="projectid515.mpeg"></span><span id="PROJECTID515.MPEG"></span>**MPEG**
</dt> <dd>

Consulte a definição para: grupo MPEG (MPEG)

</dd> <dt>

<span id="projectid515.multicast"></span><span id="PROJECTID515.MULTICAST"></span>**multicast**
</dt> <dd>

Um método de distribuição de conteúdo no qual um único fluxo é transmitido de um servidor de mídia para vários clientes. Os clientes não têm conexão com o servidor. Em vez disso, o servidor envia uma única cópia do fluxo pela rede para roteadores habilitados para multicast, que replicam os dados. Os clientes podem então receber o fluxo monitorando um endereço IP e uma porta multicast específicos.

</dd> <dt>

<span id="projectid515.multipurpose_internet_mail_extension__mime_"></span><span id="PROJECTID515.MULTIPURPOSE_INTERNET_MAIL_EXTENSION__MIME_"></span>**MIME (Multipurpose Internet Mail Extension)**
</dt> <dd>

Um padrão que estende o protocolo SMTP para codificar arquivos de dados não ASCII, como vídeo, som e arquivos binários para anexo a email da Internet.

</dd> <dt>

<span id="projectid515.musical_instrument_digital_interface__midi_"></span><span id="PROJECTID515.MUSICAL_INSTRUMENT_DIGITAL_INTERFACE__MIDI_"></span>**MIDI (interface digital de instrumento musical)**
</dt> <dd>

Uma especificação da MMA (Associação de fabricantes de MIDI). A especificação define um protocolo para descrever dados de música, como observação sobre e anotar mensagens; um formato de arquivo para armazenar dados de música, chamados MIDI padrão; e uma interface de hardware padrão.

</dd> <dt>

<span id="projectid515.object"></span><span id="PROJECTID515.OBJECT"></span>**objeto** do
</dt> <dd>

Uma coleção de propriedades e métodos que atendem a uma finalidade específica e são tratados como uma unidade.

</dd> <dt>

<span id="projectid515.online_store"></span><span id="PROJECTID515.ONLINE_STORE"></span>**loja online**
</dt> <dd>

No Windows Media Player, um site que oferece conteúdo de mídia digital por assinatura ou para compra.

</dd> <dt>

<span id="projectid515.packet"></span><span id="PROJECTID515.PACKET"></span>do **pacote**
</dt> <dd>

Uma unidade de informações transmitidas de um computador ou dispositivo para outro em uma rede.

</dd> <dt>

<span id="projectid515.parameter"></span><span id="PROJECTID515.PARAMETER"></span>**parâmetro** do
</dt> <dd>

Uma informação que é passada para um método a fim de atingir um resultado específico.

</dd> <dt>

<span id="projectid515.partnership"></span><span id="PROJECTID515.PARTNERSHIP"></span>**parceria**
</dt> <dd>

Consulte a definição para: parceria de sincronização

</dd> <dt>

<span id="projectid515.playlist"></span><span id="PROJECTID515.PLAYLIST"></span>**lista de reprodução**
</dt> <dd>

Uma lista de conteúdo de mídia digital.

</dd> <dt>

<span id="projectid515.port"></span><span id="PROJECTID515.PORT"></span>**porta** do
</dt> <dd>

Uma interface baseada em hardware ou software usada para transferir informações entre um computador e outros dispositivos. As portas de hardware são conexões físicas visíveis fora do computador. As portas de software são os gateways numerados em programas que os programas de software usam para trocar informações.

</dd> <dt>

<span id="projectid515.portable_device"></span><span id="PROJECTID515.PORTABLE_DEVICE"></span>**dispositivo portátil**
</dt> <dd>

Um dispositivo eletrônico móvel que pode trocar arquivos ou outros dados com um computador ou dispositivo. Exemplos de dispositivos portáteis incluem Pocket PCs, players de música digital portáteis e smartphones.

</dd> <dt>

<span id="projectid515.preset"></span><span id="PROJECTID515.PRESET"></span>**predefinição**
</dt> <dd>

Uma configuração predefinida. Para o Windows Media Player, há predefinições definidas para as configurações do equalizador e para a maioria das visualizações.

</dd> <dt>

<span id="projectid515.progress_bar"></span><span id="PROJECTID515.PROGRESS_BAR"></span>**barra de progresso**
</dt> <dd>

Um indicador do tipo medidor em uma interface gráfica do usuário que mostra o progresso de um processo, como um download de arquivo ou uma instalação.

</dd> <dt>

<span id="projectid515.property"></span><span id="PROJECTID515.PROPERTY"></span>**Propriedade**
</dt> <dd>

Uma informação que especifica um detalhe específico de um objeto.

</dd> <dt>

<span id="projectid515.protocol"></span><span id="PROJECTID515.PROTOCOL"></span>**protocolo** do
</dt> <dd>

Um conjunto padrão de formatos e procedimentos que permitem que computadores troquem informações.

</dd> <dt>

<span id="projectid515.proxy_server"></span><span id="PROJECTID515.PROXY_SERVER"></span>**servidor proxy**
</dt> <dd>

Um servidor localizado em uma rede entre o software cliente, como um navegador da Web e outro servidor. Ele intercepta todas as solicitações ao servidor para determinar se ele pode atendê-las por conta própria. Caso contrário, ele encaminha a solicitação para outro servidor.

</dd> <dt>

<span id="projectid515.publishing_point"></span><span id="PROJECTID515.PUBLISHING_POINT"></span>**ponto de publicação**
</dt> <dd>

Um local de memória organizado que é identificado por um nome em um servidor do Windows Media. O nome é parte da URL usada por um cliente ao solicitar o conteúdo do servidor. Consulte também: ponto de publicação de difusão, ponto de publicação sob demanda.

</dd> <dt>

<span id="projectid515.sami"></span><span id="PROJECTID515.SAMI"></span>**Sami**
</dt> <dd>

Consulte a definição para: SAMI (intercâmbio de mídia acessível) sincronizado

</dd> <dt>

<span id="projectid515.skin"></span><span id="PROJECTID515.SKIN"></span>**aparência**
</dt> <dd>

Uma interface do usuário que fornece uma aparência alternativa e uma funcionalidade personalizada para software como o Windows Media Player.

</dd> <dt>

<span id="projectid515.skin_definition_file"></span><span id="PROJECTID515.SKIN_DEFINITION_FILE"></span>**arquivo de definição de capa**
</dt> <dd>

Um documento XML que especifica os elementos em uma capa, juntamente com suas relações e funcionalidade. Um arquivo de definição de capa tem uma extensão de nome de arquivo. WMS.

</dd> <dt>

<span id="projectid515.skin_mode"></span><span id="PROJECTID515.SKIN_MODE"></span>**modo de capa**
</dt> <dd>

Um estado operacional do Windows Media Player no qual sua interface do usuário é exibida como uma capa.

</dd> <dt>

<span id="projectid515.slider"></span><span id="PROJECTID515.SLIDER"></span>**controle deslizante**
</dt> <dd>

Um controle dentro de uma interface gráfica do usuário na qual um indicador chamado Thumb pode ser movido ao longo de um caminho contínuo para várias posições diferentes.

</dd> <dt>

<span id="projectid515.sticky"></span><span id="PROJECTID515.STICKY"></span>**adesivo**
</dt> <dd>

Uma característica de um controle de botão dentro de uma interface gráfica do usuário na qual o botão, quando clicado, permanece em uma posição inativa até que seja clicada novamente. Um botão adesivo também é conhecido como alternância.

</dd> <dt>

<span id="projectid515.stream"></span><span id="PROJECTID515.STREAM"></span>**fluxo**
</dt> <dd>

Mídia digital que está no processo de ser entregue em um fluxo contínuo em uma rede.

</dd> <dt>

<span id="projectid515.streaming"></span><span id="PROJECTID515.STREAMING"></span>**streaming**
</dt> <dd>

Um método de entrega de mídia digital em uma rede em um fluxo contínuo. A mídia digital é reproduzida pelo software cliente conforme ele é recebido. Normalmente, o streaming torna desnecessário que os usuários baixem um arquivo antes de reproduzi-lo.

</dd> <dt>

<span id="projectid515.subscription_service"></span><span id="PROJECTID515.SUBSCRIPTION_SERVICE"></span>**serviço de assinatura**
</dt> <dd>

Um serviço oferecido por uma loja online que permite alugar arquivos baseados no Windows Media por uma taxa recorrente, após a qual você pode reproduzir ou sincronizar os arquivos, dependendo do serviço.

</dd> <dt>
