# <a name="navigation-for-type-2-online-stores"></a>Navegação para lojas online do tipo 2

As páginas da Web hospedadas pelo Windows Media Player 10 ou posterior, como aquelas nos painéis de tarefas do serviço de loja online, podem usar o método [external. NavigateTaskPaneURL](external-navigatetaskpaneurl.md) para controlar a navegação no Windows Media Player. A navegação pode ocorrer das seguintes maneiras:

-   Entre os painéis de tarefas de serviço. Consulte [navegando entre os painéis de tarefas de serviço](navigating-between-service-task-panes.md)
-   De recursos como exibição do centro de informações, páginas da Web do HTMLView e a página de informações do álbum para os painéis de tarefas de serviço. Consulte [navegação nos recursos do Windows Media Player](navigating-from-windows-media-player-features.md).
-   Entre lojas online. Consulte [navegando para outras lojas online](navigating-to-other-online-stores.md).
