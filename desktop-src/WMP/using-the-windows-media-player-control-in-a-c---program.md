# <a name="using-the-windows-media-player-control-in-a-c-program"></a>Usando o controle do Windows Media Player em um programa C++

> [!Note]  
> O uso do C++ para inserir o controle do Windows Media Player tem suporte no Windows Media Player 9 Series ou posterior.

 

Há várias maneiras diferentes de usar o controle do Windows Media Player em um programa C++. Você pode criar uma instância do controle em um aplicativo de console ou pode inserir o controle em um aplicativo do Windows. Além disso, você pode implementar interfaces que permitem executar um controle de player incorporado no modo remoto. Você pode personalizar a interface do usuário de um controle inserido aplicando um arquivo de definição de capa.

Essas informações são descritas nos tópicos a seguir.



| Tópico                                                                                                                                      | Descrição                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Usando o controle do Windows Media Player em um aplicativo de console](using-the-windows-media-player-control-in-a-console-application.md)     | Descreve um aplicativo de console C++ simples que instancia o controle do Windows Media Player para exibir a versão.                                                       |
| [Hospedando o controle do Windows Media Player em um aplicativo do Windows](hosting-the-windows-media-player-control-in-a-windows-application.md) | Descreve como usar a janela do host ActiveX do ATL para inserir o controle do Windows Media Player em um programa do Windows.                                                            |
| [Estabelecer comunicação remota do controle do Windows Media Player](remoting-the-windows-media-player-control.md)                                                 | Descreve como inserir o controle do Windows Media Player em um programa C++ no modo remoto, o que permite que os usuários desencaixem o controle para alternar para o modo completo do Player. |
| [Manipulando eventos em C++](handling-events-in-c.md)                                                                                         | Descreve como receber notificações de eventos do Windows Media Player.                                                                                                     |
| [Usando capas com o controle do Windows Media Player](using-skins-with-the-windows-media-player-control.md)                                 | Descreve como aplicar um arquivo de capa a um controle do Windows Media Player incorporado em um programa C++.                                                                             |



 

> [!Note]  
> Você pode inserir o controle móvel do Windows Media Player 10 em um aplicativo Windows CE. As técnicas que você usa para fazer isso são semelhantes às usadas com o controle Windows Media Player da área de trabalho. No entanto, há diferenças entre ATL para Windows e ATL para Windows CE. Esta documentação descreve as diferenças entre essas implementações, quando apropriado.
