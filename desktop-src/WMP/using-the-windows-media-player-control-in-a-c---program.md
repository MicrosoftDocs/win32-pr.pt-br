# <a name="using-the-windows-media-player-control-in-a-c-program"></a>Usando o Windows Media Player controle em um programa C++

> [!Note]  
> O uso do C++ para inserir o Windows Media Player controle tem suporte para Windows Media Player Série 9 ou posterior.

 

Há várias maneiras diferentes de usar o controle Windows Media Player em um programa C++. Você pode criar uma instância do controle em um aplicativo de console ou pode inserir o controle em um Windows aplicativo. Além disso, você pode implementar interfaces que permitem executar um controle Player inserido no modo remoto. Você pode personalizar a interface do usuário de um controle inserido aplicando um arquivo de definição de capa.

Essas informações são descritas nos tópicos a seguir.



| Tópico                                                                                                                                      | Descrição                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Usando o controle Windows Media Player em um aplicativo de console](using-the-windows-media-player-control-in-a-console-application.md)     | Descreve um aplicativo de console C++ simples que instania o controle Windows Media Player para exibir a versão.                                                       |
| [Hospedando o Windows Media Player controle em um Windows aplicativo](hosting-the-windows-media-player-control-in-a-windows-application.md) | Descreve como usar a janela de host do ActiveX ATL para inserir o controle Windows Media Player em um Windows de dados.                                                            |
| [Estabelecer comunicação remota do controle do Windows Media Player](remoting-the-windows-media-player-control.md)                                                 | Descreve como inserir o controle Windows Media Player em um programa C++ no modo remoto, o que permite que os usuários desencaixem o controle para alternar para o modo completo do Player. |
| [Manipulando eventos em C++](handling-events-in-c.md)                                                                                         | Descreve como receber notificações de eventos de Windows Media Player.                                                                                                     |
| [Usando capas com o Windows Media Player controle](using-skins-with-the-windows-media-player-control.md)                                 | Descreve como aplicar um arquivo de capa a um controle Windows Media Player inserido em um programa C++.                                                                             |



 

> [!Note]  
> Você pode inserir o controle Windows Media Player 10 Mobile em um Windows CE aplicativo. As técnicas usadas para fazer isso são semelhantes às usadas com o controle de Windows Media Player desktop. No entanto, há diferenças entre a ATL para Windows e a ATL para Windows CE. Esta documentação descreve as diferenças entre essas implementações, quando apropriado.
