# <a name="listboxdismiss"></a>Caixa de listagem. ignorar

O método **ignore** oculta o controle.

``` syntax
        elementID.dismiss()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Esse método é usado principalmente com o elemento **Popup** , que representa um controle de caixa de listagem que é exibido somente quando necessário.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
