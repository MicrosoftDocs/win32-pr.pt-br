# <a name="equalizersettingspresetcount"></a>EQUALIZERSETTINGS.presetCount

O atributo **presetCount** recupera o número de predefinições disponíveis.

``` syntax
        elementID.presetCount
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** somente leitura (**Long**).

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
