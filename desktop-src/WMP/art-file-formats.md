# <a name="art-file-formats"></a>Formatos de arquivo artístico

Os seguintes formatos de arquivo artístico são reconhecidos pelo Windows Media Player para capas:



| Formatar                                  | Extensão   | Descrição                                                                                        |
|-----------------------------------------|-------------|----------------------------------------------------------------------------------------------------|
| Bitmap                                  | .bmp        | Imagens de bitmap são recomendadas porque oferecem o máximo de controle sobre a imagem e as cores exatas. |
| GIF       | .gif        | Formato de imagem compactado usado para páginas da Web. Há suporte para GIFs animados.                            |
| JPEG | .jpeg, .jpg | Formato de imagem compactado usado para páginas da Web.                                                         |
| PNG         | .png        | Formato de imagem compactado usado para páginas da Web.                                                         |



 

Se você usar um dos formatos de arquivo compactados que define uma cor como transparente para um navegador da Web, não defina uma cor como transparente no arquivo de imagem. Use uma cor visível para representar áreas transparentes em sua imagem e, em seguida, defina essa cor como transparente no arquivo de definição de capa. Por exemplo, se você criar um arquivo GIF com algumas áreas transparentes, elas não serão transparentes em sua imagem final e você não poderá usar a cor definida como transparente no arquivo GIF para a cor de transparência em sua capa.
