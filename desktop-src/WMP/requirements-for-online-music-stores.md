# <a name="requirements-for-online-music-stores"></a>Requisitos para lojas de música online

Todos os requisitos que sua loja online deve atender são detalhados no Contrato de Listagem de Serviço de Mídia Online para Windows Media Player. Para garantir que sua loja online fornece uma ótima experiência do usuário, realçamos alguns requisitos importantes sobre personalização e reprodução.

-   A página inicial do armazenamento exposta na interface do Windows Media Player usuário deve estar funcional sem exigir que o usuário faça logoff.
-   Anúncios pop-up que iniciam uma janela separada do navegador são proibidos.
-   Toda a experiência de navegação, compra e reprodução deve ser renderizada na interface do usuário do Windows Media Player e não em uma janela separada do navegador pop-up ou em um player de mídia inserido.
-   A instalação systray de um aplicativo e processos em segundo plano que são executados continuamente são proibidas.
-   Ações discretas, como logon, cobrança, compra de CDs físicos ou outros recursos relacionados fora do serviço de mídia digital primário, podem ser vinculadas a uma instância do navegador fora do player para o recurso de mídia não digital.
-   As licenças fornecidas pelo armazenamento devem permanecer válidas, independentemente da conexão do armazenamento com Windows Media Player. Para obter detalhes, consulte a seção Termo e Término do Contrato de Listagem do Serviço de Mídia Online para Windows Media Player.

De tempos em tempos, a Microsoft pode atualizar os requisitos de qualificação e de entrada e postá-los no [site Windows Media](https://www.microsoft.com/windows/windowsmedia/default.mspx) ( https://www.microsoft.com/windows/windowsmedia/default.mspx) .
