# <a name="requirements-for-online-music-stores"></a>Requisitos para lojas de música online

Todos os requisitos que sua loja online deve atender são detalhados no contrato de listagem do serviço de mídia online do Windows Media Player. Para garantir que sua loja online forneça uma excelente experiência do usuário, destacamos alguns requisitos importantes em relação à personalização e à reprodução.

-   A página inicial do repositório exposto na interface do usuário do Windows Media Player deve ser funcional sem exigir que o usuário faça logon.
-   Os anúncios pop-up que iniciam uma janela separada do navegador são proibidos.
-   Toda a experiência de procura, compra e reprodução deve ser processada na interface do usuário do Windows Media Player e não em uma janela de navegador pop-up separada ou um player de mídia incorporado.
-   A instalação do systray de um aplicativo e processos em segundo plano que são executados continuamente é proibida.
-   Ações discretas, como logon, cobrança, compra de CDs físicos ou outros recursos relacionados fora do serviço de mídia digital primário, podem ser vinculadas a uma instância do navegador fora do Player para o recurso de mídia não digital.
-   As licenças fornecidas pelo repositório devem permanecer válidas independentemente da conexão do repositório com o Windows Media Player. Para obter detalhes, consulte a seção termo e terminação do contrato de listagem do serviço de mídia online do Windows Media Player.

De tempos em tempos, a Microsoft pode atualizar os requisitos de qualificação e integração e postá-los no [site do Windows Media](https://www.microsoft.com/windows/windowsmedia/default.mspx) ( https://www.microsoft.com/windows/windowsmedia/default.mspx) .
