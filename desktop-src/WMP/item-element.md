# <a name="item-element"></a>Elemento ITEM

O elemento **Item** representa um item em uma caixa de listagem ou controles pop-up. Elementos de **Item** são sempre filhos de elementos **ListBox** ou **Popup** .

O elemento **Item** dá suporte aos seguintes atributos.



| Atributo               | Descrição                                                                  |
|-------------------------|------------------------------------------------------------------------------|
| [value](item-value.md) | Especifica ou recupera o texto que é exibido para uma caixa de listagem ou um item pop-up. |



 

> [!Note]  
> Esse elemento requer o Windows Media Player para Windows XP ou posterior.
