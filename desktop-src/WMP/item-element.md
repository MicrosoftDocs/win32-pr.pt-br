# <a name="item-element"></a>Elemento ITEM

O **elemento ITEM** representa um item em uma caixa de listagem ou controles pop-up. **Elementos ITEM** são sempre filhos de **elementos LISTBOX** **ou POPUP.**

O **elemento ITEM** dá suporte aos atributos a seguir.



| Atributo               | Descrição                                                                  |
|-------------------------|------------------------------------------------------------------------------|
| [value](item-value.md) | Especifica ou recupera o texto exibido para uma caixa de listagem ou item pop-up. |



 

> [!Note]  
> Esse elemento requer Windows Media Player para Windows XP ou posterior.
