# <a name="supported-protocols-and-file-types"></a>Protocolos e tipos de arquivo com suporte

Windows Media Player dá suporte a vários protocolos e muitos tipos de arquivo, todos eles podem ser usados ao especificar valores de URL no modelo de objeto Player para propriedades como *Player*. **URL** ou nas listas Windows de metadados de mídia. Além disso, os protocolos com suporte podem ser usados ao especificar valores de protocolo com os métodos relacionados ao proxy do **objeto Network.**

No momento, há suporte para os seguintes formatos de arquivo de mídia digital para reprodução Windows Media Player:

-   Advanced Systems Format (ASF)
-   AIF
-   Aifc
-   Aiff
-   AU
-   AVI
-   MID
-   MPE
-   Mpeg
-   MPG
-   MPv2
-   MP2
-   MP3
-   M1v
-   Snd
-   WAV
-   Windows Arquivos de mídia com uma extensão de nome de arquivo .wm
-   Áudio do Windows Media (WMA)
-   Vídeo do Windows Media (WMV)

No momento, os protocolos a seguir têm suporte Windows Media Player.



| Protocolo | Descrição                                                                        |
|----------|------------------------------------------------------------------------------------|
| HTTP     | Protocolo de transferência de hipertexto. Inclui HTTP com cache rápido e multicast.          |
| RTSP     | Protocolo de streaming em tempo real. Inclui o RTSP com cache rápido.                       |
| Rtspu    | RTSP usado com o protocolo UDP. Inclui RTSPU com cache rápido        |
| Rtspt    | RTSP usado com o protocolo TCP. Inclui RTSPT com cache rápido |
| mms      | Protocolo do Microsoft Media Server.                                                   |
| MMSU     | MMS usado com UDP.                                                                 |
| MMST     | MMS usado com TCP.                                                                 |
| WMPCD    | Um protocolo usado pelo Windows Media Player para fornecer acesso a CDs.                  |
| WMPDVD   | Um protocolo usado pelo Windows Media Player para fornecer acesso a DVDs.                 |
