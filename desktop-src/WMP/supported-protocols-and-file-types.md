# <a name="supported-protocols-and-file-types"></a>Tipos de arquivos e protocolos com suporte

O Windows Media Player dá suporte a vários protocolos e vários tipos de arquivo, que podem ser usados ao especificar valores de URL no modelo de objeto do Player para propriedades como o *Player*. **URL** ou nas listas de reprodução do metarquivo de mídia do Windows. Além disso, os protocolos com suporte podem ser usados ao especificar valores de protocolo com os métodos relacionados ao proxy do objeto de **rede** .

Os seguintes formatos de arquivo de mídia digital atualmente têm suporte para reprodução pelo Windows Media Player:

-   Advanced Systems Format (ASF)
-   AIF
-   AIFC
-   AIFF
-   AU
-   AVI
-   MID
-   MPE
-   FUNCIONALIDADES
-   MPG
-   MPv2
-   MP2
-   MP3
-   M1V
-   SND
-   WAV
-   Arquivos de mídia do Windows com extensão de nome de arquivo. WM
-   Áudio do Windows Media (WMA)
-   Vídeo do Windows Media (WMV)

Os protocolos a seguir têm suporte no momento pelo Windows Media Player.



| Protocolo | Descrição                                                                        |
|----------|------------------------------------------------------------------------------------|
| HTTP     | Protocolo de transferência de hipertexto. Inclui HTTP com cache rápido e multicast.          |
| RTSP     | Protocolo de streaming em tempo real. Inclui RTSP com cache rápido.                       |
| RTSPu    | RTSP usado com UDP (User Datagram Protocol). Inclui RTSPu com cache rápido        |
| RTSPt    | RTSP usado com TCP (protocolo de controle de transmissão). Inclui RTSPt com cache rápido |
| mms      | Protocolo Microsoft Media Server.                                                   |
| MMSU     | MMS usado com UDP.                                                                 |
| MMST     | MMS usado com TCP.                                                                 |
| WMPCD    | Um protocolo usado pelo Windows Media Player para fornecer acesso ao CDs.                  |
| WMPDVD   | Um protocolo usado pelo Windows Media Player para fornecer acesso a DVDs.                 |
