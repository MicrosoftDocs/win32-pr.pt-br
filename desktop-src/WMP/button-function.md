# <a name="button-function"></a>Função de botão

A tabela a seguir mostra valores válidos para a função Button. Não é necessário definir valores para funções que você não usará em sua capa.



| Valor           | Descrição                                                                                                                                                  |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Reproduzir            | Reproduz a seleção atual na lista de reprodução.                                                                                                                 |
| Pausar           | Pausa a reprodução do item de mídia atual.                                                                                                                   |
| Stop            | Interrompe a reprodução do item de mídia atual.                                                                                                                    |
| Prev            | Move para o item anterior na lista de reprodução.                                                                                                                  |
| Avançar            | Move para o próximo item na lista de reprodução.                                                                                                                      |
| Shuffle         | Embaralha a playlist aleatoriamente.                                                                                                                              |
| RepeatPlaylist  | Repete a lista de reprodução indefinidamente.                                                                                                                           |
| Mute            | Silencia a saída de áudio.                                                                                                                                   |
| PlayPause       | Alterna entre o estado de reprodução e o estado de pausa.                                                                                                          |
| Info            | Fornece informações sobre o item de mídia que está sendo executado. Para as capas do Windows Media Player 10 Mobile ou posteriores, essa função não é usada.                     |
| FullScreen      | Redimensiona o vídeo para o tamanho máximo da tela do dispositivo.                                                                                                 |
| WindowsMedia\*  | Abre o Pocket Internet Explorer e carrega o site do Windowsmedia.com.                                                                                       |
| PlayPauseStop\* | Além de alternar entre o estado de reprodução e o estado de pausa, ele também permite que os usuários interrompam uma transmissão ao vivo quando a funcionalidade de pausa não está disponível. |
| VolumeUp\*      | Aumenta a saída de áudio.                                                                                                                                  |
| VolumeDown\*    | Diminui a saída de áudio.                                                                                                                                  |



 

\*Disponível para Windows Media Player 10 Mobile ou posterior.
