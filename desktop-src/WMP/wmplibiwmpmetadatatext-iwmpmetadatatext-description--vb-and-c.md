# <a name="iwmpmetadatatextdescription-property"></a>IWMPMetadataText: Propriedade descrição de:D

A `Description` propriedade Obtém uma descrição do texto de metadados.

## <a name="syntax"></a>Syntax


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## <a name="property-value"></a>Valor da propriedade

Um **System. String** que é a descrição.
