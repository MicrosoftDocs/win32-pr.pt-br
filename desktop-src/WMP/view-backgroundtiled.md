# <a name="viewbackgroundtiled"></a>Exibir. backgroundTiled

O atributo **backgroundTiled** especifica ou recupera um valor que indica se a imagem de plano de fundo do **modo de exibição** ou da **subexibição** é colocada em ladrilho.

``` syntax
        elementID.backgroundTiled
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                                        |
|-------|----------------------------------------------------|
| true  | A imagem é repetida horizontal e verticalmente. |
| false | Padrão. A imagem não é repetida.                |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
