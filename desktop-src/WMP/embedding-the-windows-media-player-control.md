# <a name="embedding-the-windows-media-player-control"></a>Incorporação do controle Windows Media Player dados

Além de inserir o controle Windows Media Player em uma página da Web, conforme descrito na seção anterior, você pode inscrever em documentos do Microsoft Office e em programas personalizados que você cria usando uma das várias linguagens de programação, incluindo Microsoft Visual Basic, C++ e C#.

As seções a seguir fornecem informações sobre como usar o controle em vários ambientes de programação.



| Seção                                                                                                                                  | Descrição                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Incorporação do controle Windows Media Player em uma página da Web](embedding-the-windows-media-player-control-in-a-web-page.md)                 | Esta seção descreve o uso do controle em uma página da Web.                                                  |
| [Incorporação do controle Windows Media Player em um documento de Office](embedding-the-windows-media-player-control-in-an-office-document.md) | Esta seção descreve o uso do controle em um Microsoft Office documento.                                |
| [Incorporação do controle Windows Media Player em um programa personalizado](embedding-the-windows-media-player-control-in-a-custom-program.md)     | Esta seção descreve o uso do controle em um Windows baseado em Visual Basic escrito em Visual Basic ou C++. |
