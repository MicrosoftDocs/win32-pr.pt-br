# <a name="embedding-the-windows-media-player-control"></a>Inserindo o controle do Windows Media Player

Além de inserir o controle do Windows Media Player em uma página da Web, conforme descrito na seção anterior, você pode incorporá-lo em Microsoft Office documentos e em programas personalizados que você cria usando uma das várias linguagens de programação, incluindo Microsoft Visual Basic, C++ e C#.

As seções a seguir fornecem informações sobre como usar o controle em vários ambientes de programação.



| Seção                                                                                                                                  | Descrição                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Inserindo o controle do Windows Media Player em uma página da Web](embedding-the-windows-media-player-control-in-a-web-page.md)                 | Esta seção descreve como usar o controle em uma página da Web.                                                  |
| [Inserindo o controle do Windows Media Player em um documento do Office](embedding-the-windows-media-player-control-in-an-office-document.md) | Esta seção descreve como usar o controle em um Microsoft Office documento.                                |
| [Inserindo o controle do Windows Media Player em um programa personalizado](embedding-the-windows-media-player-control-in-a-custom-program.md)     | Esta seção descreve como usar o controle em um aplicativo baseado em Windows escrito em Visual Basic ou C++. |
