# <a name="text-section"></a>Seção de texto

Nenhuma seção de texto é usada nesta capa, mas este é um exemplo do formato que você deve usar ao implementar uma seção de texto:


```C++
[ Text ]

//  <Item>       <Location>     <Align>  <Font>          <Color>
//  ------       ----------     -------  ------          -------

```
