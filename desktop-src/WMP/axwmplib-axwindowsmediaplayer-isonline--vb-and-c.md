# <a name="axwindowsmediaplayerisonline-property"></a>Propriedade AxWindowsMediaPlayer. IsOnline

A propriedade IsOnline Obtém um valor que indica se o usuário está conectado a uma rede.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public System.Boolean isOnline {get;}
```


```VB

Public ReadOnly Property isOnline As System.Boolean
```





## <a name="property-value"></a>Valor da propriedade

Um valor System. Boolean que indica se o usuário está conectado a uma rede.
