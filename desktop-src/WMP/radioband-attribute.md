# <a name="radioband-attribute"></a>Atributo RadioBand

O atributo **RadioBand** é a banda de frequência da estação de rádio.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de rádio](radio-item-attributes.md)

## <a name="remarks"></a>Comentários

Esse atributo é armazenado somente na biblioteca do.

Esse atributo tem um dos seguintes valores: "FM", "AM" ou "net".

Para determinar se você pode alterar o valor desse atributo, use o método [Media. isReadOnlyItem](media-isreadonlyitem.md) .

|--------------------|-----------------------------------------------|
