# <a name="button-location"></a>Localização do botão

Você deve definir o local para cada imagem de botão. Para fazer isso, você fornece quatro inteiros positivos separados por vírgulas. Os números definem as coordenadas do canto superior esquerdo, a largura e a altura de um botão, em pixels.

Se você estiver usando botões de região (PushHit, ToggleHit e 2PushHit), ainda deverá inserir esses quatro números. Os valores são usados para determinar os locais das imagens secundárias, como Enviada por Pushed e Disabled.

> [!Note]  
> Os tipos de botão são preterido em capas para Windows Media Player 10 Mobile ou posterior.
