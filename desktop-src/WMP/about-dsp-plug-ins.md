# <a name="about-dsp-plug-ins"></a>Sobre plug-ins do DSP

O Microsoft Windows Media Player fornece a arquitetura para conectar o software complementar que executa funções de processamento de sinal digital (DSP) nos fluxos de áudio ou vídeo durante a reprodução. Esses programas complementares são chamados de plug-ins do DSP.

As seções a seguir fornecem uma visão geral conceitual dos plug-ins do DSP do Windows Media Player.



| Seção                                                              | Descrição                                                                                            |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [Visão geral do usuário de plug-in do DSP](dsp-plug-in-user-overview.md)           | Descreve os plug-ins do DSP da perspectiva do usuário final.                                                |
| [Visão geral do desenvolvedor de plug-in do DSP](dsp-plug-in-developer-overview.md) | Fornece informações gerais que você precisa saber antes de criar um plug-in do DSP para o Windows Media Player. |
| [Criando um plug-in do DSP](building-a-dsp-plug-in.md)                 | Explica como usar o assistente de plug-in do Windows Media Player para criar um plug-in de DSP de exemplo.            |
| [Implementando seu código do DSP](implementing-your-dsp-code.md)         | Discute considerações para criar plug-ins de áudio e vídeo de DSP personalizados para o Windows Media Player.    |
