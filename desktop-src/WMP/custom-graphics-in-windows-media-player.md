# <a name="custom-graphics-in-windows-media-player"></a>Gráficos personalizados no Windows Media Player

Com o script do metarquivo do Windows Media, você pode adicionar elementos gráficos ao Windows Media Player à medida que o fluxo é reproduzido. Há um tipo básico de elementos gráficos que você pode adicionar.



| Tipo   | Descrição                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| TIMBRADO | Usando o elemento **banner** , você pode inserir uma imagem de faixa (32 pixels de altura por 194 pixels de largura) no Windows Media Player na parte inferior da área de exibição do vídeo. Você pode usar um elemento filho para adicionar um hiperlink à faixa. |
