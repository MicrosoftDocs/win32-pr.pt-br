# <a name="playlisteditbuttonvisible"></a>PLAYLIST. editButtonVisible

O atributo **editButtonVisible** especifica ou recupera um valor que indica se o botão de edição da playlist está visível.

``` syntax
        elementID.editButtonVisible
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                                       |
|-------|---------------------------------------------------|
| true  | O botão Editar da lista de reprodução está visível.              |
| false | Padrão. O botão Editar da lista de reprodução não está visível. |



 

## <a name="remarks"></a>Comentários

Quando você definir esse atributo como true, o botão Editar da lista de reprodução será exibido no canto inferior esquerdo do elemento **playlist** . Clicar nesse botão exibe um menu de opções que permitem ao usuário editar, limpar, classificar, abrir, salvar ou copiar uma lista de reprodução.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
