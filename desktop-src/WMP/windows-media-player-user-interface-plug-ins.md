# <a name="windows-media-player-user-interface-plug-ins"></a>Windows Media Player Plug-ins de interface do usuário

o Microsoft Windows Media Player fornece uma arquitetura que permite ao usuário instalar e ativar os programas de plug-in que adicionam a funcionalidade de interface do usuário ao modo completo do Player. Um plug-in de interface do usuário típico pode permitir que um usuário compre o CD que contém a faixa que está sendo executada ou acesse informações adicionais sobre o artista. esta seção do Windows Media Player Software Development Kit (SDK) fornece as informações de programação necessárias para criar seu próprio plug-in de interface do usuário.

A documentação de plug-in da interface do usuário é dividida em três seções:



| Seção                                                                                            | Descrição                                                                                                                                   |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre plug-ins de interface do usuário](about-user-interface-plug-ins.md)                                 | Fornece uma visão geral da arquitetura usada para plug-ins de interface do usuário. Leia esta seção para aprender os conceitos gerais envolvidos nessa tecnologia. |
| [Guia de programação de plug-ins de interface do usuário](user-interface-plug-ins-programming-guide.md)         | Explica o que você precisa fazer para criar um plug-in de interface do usuário. Esta seção contém código de exemplo e procedimentos passo a passo.                          |
| [Referência de programação de plug-ins de interface do usuário](user-interface-plug-ins-programming-reference.md) | fornece uma referência detalhada para a interface COM e métodos com suporte pelo SDK Windows Media Player para plug-ins de interface do usuário.                    |



 

> [!Note]  
> o Windows Media Player 10 Mobile fornece o mesmo modelo de plug-in que o Windows Media Player da área de trabalho. Esse modelo de plug-in permite que você use plug-ins em segundo plano para monitorar eventos, exibir o status das propriedades e assim por diante. o guia de programação nesta seção descreve como criar um plug-in de interface do usuário em segundo plano para Windows Media Player 10 Mobile usando o assistente de plug-in de Windows Media Player de área de trabalho.
