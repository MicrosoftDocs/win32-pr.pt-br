# <a name="onmouseover"></a>onmouseover

O manipulador de eventos **onmouseover** manipula um evento que ocorre quando o usuário coloca o ponteiro pela primeira vez sobre o elemento.

``` syntax
onmouseover
```

## <a name="remarks"></a>Comentários

Este manipulador de eventos não é aplicável ao elemento **playlist** .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------|
| Versão<br/> | Windows Media Player versão 70 ou posterior<br/> |
