# <a name="windows-media-playlists"></a>Windows Listas de reprodução de mídia

essa referência documenta Windows playlists de mídia. uma lista de reprodução de mídia do Windows (WPL) é um arquivo que usa os elementos documentados nesta referência para definir uma lista de reprodução dinâmica "inteligente" de itens de mídia ou um conjunto estático de itens de mídia.

Esta referência contém as seções a seguir.

|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
