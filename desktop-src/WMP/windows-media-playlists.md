# <a name="windows-media-playlists"></a>Playlists do Windows Media

Esta referência documenta as listas de reprodução do Windows Media. Uma lista de reprodução de mídia do Windows (WPL) é um arquivo que usa os elementos documentados nesta referência para definir uma lista de reprodução dinâmica "inteligente" de itens de mídia ou um conjunto estático de itens de mídia.

Esta referência contém as seções a seguir.



| Seção                                                                                    | Descrição                                                                                                            |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [Referência de elementos da playlist do Windows Media](windows-media-playlist-elements-reference.md) | Explica em detalhes cada um dos elementos da lista de reprodução, seus atributos e valores e condições especiais relacionadas a cada um. |
