# <a name="iwmpcdromburnburnplaylist-property"></a>Propriedade IWMPCdromBurn:: burnPlaylist

A propriedade **burnPlaylist** Obtém a lista de reprodução atual para gravar no CD.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public IWMPPlaylist burnPlaylist {get;}
```


```VB

Public ReadOnly Property burnPlaylist As IWMPPlaylist
```





## <a name="property-value"></a>Valor da propriedade

A interface **WMPLib. IWMPPlaylist** da lista de reprodução a ser gravada.
