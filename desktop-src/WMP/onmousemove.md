# <a name="onmousemove"></a>OnMouseMove

O manipulador de eventos **OnMouseMove** manipula um evento que ocorre quando o usuário move o ponteiro do mouse enquanto ele está sobre um elemento.

``` syntax
onmousemove
```

## <a name="remarks"></a>Comentários

Esse manipulador de eventos não é aplicável ao elemento **playlist** , o elemento **Video** quando o *vídeo*. **sem janela** é falso ou o elemento **Effects** quando há *efeitos*. a **janela** é verdadeira.
