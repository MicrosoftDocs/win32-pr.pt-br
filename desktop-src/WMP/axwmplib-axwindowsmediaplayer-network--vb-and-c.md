# <a name="axwindowsmediaplayernetwork-property"></a>Propriedade AxWindowsMediaPlayer. Network

A propriedade Network Obtém uma interface IWMPNetwork que fornece acesso a informações de rede e configurações.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public IWMPNetwork network {get;}
```


```VB

Public ReadOnly Property network As IWMPNetwork
```





## <a name="property-value"></a>Valor da propriedade

A interface WMPLib. IWMPNetwork.
