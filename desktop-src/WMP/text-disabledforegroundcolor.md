# <a name="textdisabledforegroundcolor"></a>TEXT. disabledForegroundColor

O atributo **disabledForegroundColor** especifica ou recupera a cor de texto usada para o controle de texto quando ele está desabilitado.

``` syntax
        elementID.disabledForegroundColor
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém qualquer valor de cor do Microsoft Internet Explorer.

## <a name="remarks"></a>Comentários

Se **disabledForegroundColor** não for especificado, **foregroundColor** será usado.

Consulte o atributo [Value](text-value.md) para ver um exemplo ilustrando como os atributos do elemento **Text** são usados.
