# <a name="windows-media-player-mobile-functionality"></a>Windows Media Player Funcionalidade móvel

As seguintes ações estão disponíveis para o usuário da sua capa:

-   Botões ao tocar ou clicar
-   Trackbars deslizante
-   Lendo texto de rolagem
-   Assistindo ao vídeo
-   Exibindo arte do álbum
-   Classificando itens de mídia

As seções a seguir descrevem os tipos de elementos que você pode criar para habilitar essas ações do usuário:

-   [Funções de botão](button-functions.md)
-   [Tipos de TrackBar](trackbar-types.md)
-   [Tipos de exibição de texto](text-display-types.md)
-   [A exibição do vídeo](the-video-display.md)
-   [Orientation](orientation.md)
