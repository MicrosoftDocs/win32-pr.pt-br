# <a name="rules-for-each-line"></a>Regras para cada linha

Cada definição no arquivo de definição de capa deve estar em uma linha separada. Você deve inserir o texto em linhas usando as seguintes regras:

-   Cada linha deve terminar com um retorno de parágrafo. Algumas linhas podem ser mais longas do que o tamanho da linha visível do seu editor de texto, mas não quebram as linhas. Talvez você queira ativar a quebra de linha para ter certeza de que suas linhas não são interrompidas.
-   Use somente caracteres visíveis, conforme definido na referência de capa.
-   Uma linha consiste em um ou mais itens, separados por espaços.
-   Você pode inserir quantos espaços entre os itens desejar para facilitar a leitura.
-   Não use nenhum caractere de tabulação em qualquer lugar no arquivo.
-   As linhas que começam com duas barras de avanço consecutivas (//) são ignoradas. Use-os para comentários. As linhas em branco também são ignoradas.
-   Inclua espaços entre colchetes e o nome da seção.
