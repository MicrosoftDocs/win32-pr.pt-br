# <a name="equalizersettingspresettitle"></a>EQUALIZERSETTINGS.presetTitle

O método **presetTitle** recupera o nome da predefinição do equalizador com o índice especificado.

``` syntax
        elementID.presetTitle(index)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*index*
</dt> <dd>

**Número** (**longo**) que contém o índice predefinido.

</dd> </dl>

## <a name="return-value"></a>Valor Retornado

Esse método retorna uma **cadeia de caracteres**.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
