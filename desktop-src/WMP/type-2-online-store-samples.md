# <a name="type-2-online-store-samples"></a>Tipo 2 amostras de loja online

o SDK do Windows Media Player inclui os seguintes exemplos de lojas online:

-   **Repositórios online de exemplo.**
-   **Um plug-in de loja online tipo 2.** este exemplo é gerado por um assistente de Visual Studio projeto, que cria um projeto C++ que você pode usar como um ponto de partida para seu próprio componente. O uso do assistente garante que seu componente seja exclusivo. Para obter mais informações, consulte [tipo 2 plug-in de loja online](type-2-online-store-plug-in.md).
-   **Uma página da Web da loja online.** esta página da web de exemplo mostra uma página da web que Windows Media Player 10 ou posterior pode hospedar em um painel de tarefas de serviço.
-   **Especificando o atributo ContentDistributor.** este código C++ de exemplo mostra como usar o SDK do formato de mídia Windows para adicionar o atributo **ContentDistributor** ao cabeçalho de um arquivo de mídia Windows. esse é o atributo que Windows Media Player 10 ou posteriores usa para identificar provedores de loja online individuais. Para obter mais informações, consulte [adicionando o atributo ContentDistributor](adding-the-contentdistributor-attribute.md).

**Exibindo as lojas online de exemplo do tipo 2**

A Microsoft forneceu duas lojas online de exemplo de dois tipos para você exibir. Uma é uma loja de comércio de exemplo e a outra é uma loja de música de exemplo. As lojas online de exemplo são implementações simples que mostram alguns dos recursos básicos disponíveis para uso em suas próprias lojas online. Você pode exibir o código-fonte para as páginas da Web de exemplo da loja online.

Como os exemplos estão disponíveis na Internet, eles serão atualizados com recursos adicionais de tempos em tempos.

Para exibir as lojas online de exemplo do tipo 2, configure o registro conforme descrito em [chaves e entradas do registro para uma loja online tipo 2](registry-keys-and-entries-for-a-type-2-online-store.md). A tabela a seguir fornece as chaves para os repositórios de exemplo do tipo 2.



| Tipo de armazenamento      | Chave  | Versões do Player que dão suporte ao repositório de exemplo |
|-----------------|------|-----------------------------------------------|
| Tipo 2 música    | 8540 | 10 e 11                                     |
| Comércio do tipo 2 | 8541 | 10                                            |
