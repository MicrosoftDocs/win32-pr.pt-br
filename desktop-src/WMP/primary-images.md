# <a name="primary-images"></a>Imagens primárias

Você desejará criar imagens primárias para sua capa. A imagem primária é composta de uma imagem de plano de fundo definida pelo elemento **View** e quaisquer outras imagens que vão com elementos específicos. Por exemplo, se você criar um elemento de **botão** , deverá incluir a arte desse botão e certificar-se de que ele tem uma ordem z maior do que a imagem de plano de fundo.
