# <a name="creating-web-based-presentations"></a>Criando Web-Based apresentações

o controle Windows Media Player permite que você crie facilmente apresentações de apresentação de slides baseadas na Web que combinam áudio e vídeo com HTML. Ao adicionar comandos de script de tipo de URL aos seus arquivos de mídia, você pode fazer com que as páginas da Web especificadas sejam exibidas em um quadro de navegador da Web especificado em momentos específicos durante a reprodução de mídia digital.

Windows Media Player também oferece streaming de mídia avançada para permitir a entrega eficiente de dados HTML em uma rede dentro de um único fluxo de mídia Windows ou arquivo. O Player e o servidor lidam com a entrega tranqüila e oportuna de áudio, vídeo e HTML para o navegador. Assim como nas apresentações da Web que não usam streaming de mídia avançada, os comandos de script inseridos renderizam a apresentação em um quadro de navegador especificado em horários especificados.

Um layout típico para apresentações baseadas na Web usa dois quadros. um quadro contém uma página da web que incorpora o controle de Windows Media Player e exibe a parte de vídeo de uma apresentação. O outro quadro exibe as páginas da Web que são alteradas em vários momentos conforme o vídeo é reproduzido.

se a mídia digital que acompanha suas páginas da web contiver somente áudio, você poderá ocultar o quadro que contém o controle de Windows Media Player definindo sua largura como zero. Dessa forma, o áudio pode tocar sem interrupções em segundo plano enquanto as páginas da Web são exibidas na janela completa do navegador.

As seções a seguir descrevem técnicas comuns para habilitar apresentações baseadas na Web:

-   [Inversão de URL](url-flipping.md)
-   [Streaming de mídia avançada](rich-media-streaming.md)
