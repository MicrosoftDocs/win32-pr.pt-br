# <a name="listbox-element"></a>Elemento LISTBOX

O elemento **ListBox** fornece uma maneira para os usuários selecionarem itens de uma lista. Esses itens podem ser especificados usando elementos de **Item** como filhos do elemento **ListBox** . Se você precisar de um controle de caixa de listagem que seja exibido somente quando necessário, use o elemento **Popup** , que é idêntico ao elemento **ListBox** , exceto pelo valor padrão do atributo **Popup** , que determina seu comportamento de exibição.

O elemento **ListBox** dá suporte aos seguintes atributos.



| Atributo                                        | Descrição                                                                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [backgroundColor](listbox-backgroundcolor.md)   | Especifica ou recupera a cor do plano de fundo no controle da caixa de listagem.                                                  |
| [borda](listbox-border.md)                     | Especifica ou recupera um valor que indica se o controle da caixa de listagem tem uma borda. Só pode ser definido em tempo de design.  |
| [firstVisibleItem](listbox-firstvisibleitem.md) | Especifica ou recupera o índice da primeira linha visível no controle de caixa de listagem.                                   |
| [focusItem](listbox-focusitem.md)               | Especifica ou recupera a linha que contém o foco.                                                                  |
| [fontFace](listbox-fontface.md)                 | Especifica ou recupera a fonte do controle de caixa de listagem.                                                             |
| [fontSize](listbox-fontsize.md)                 | Especifica ou recupera o tamanho da fonte para o controle da caixa de listagem.                                                        |
| [fontStyle](listbox-fontstyle.md)               | Especifica ou recupera o estilo da fonte para o controle da caixa de listagem.                                                       |
| [foregroundColor](listbox-foregroundcolor.md)   | Especifica ou recupera a cor do texto no controle da caixa de listagem.                                                        |
| [itemCount](listbox-itemcount.md)               | Recupera o número de itens no controle caixa de listagem.                                                                |
| [Seleção múltipla](listbox-multiselect.md)           | Especifica ou recupera um valor que indica se o usuário pode selecionar várias linhas. Só pode ser definido em tempo de design. |
| [Pop-up](listbox-popup.md)                       | Especifica um valor que indica se o elemento representa um controle de caixa de listagem ou pop-up.                              |
| [readOnly](listbox-readonly.md)                 | Especifica ou recupera um valor que indica se o texto é somente leitura ou se o texto pode ser selecionado pelo usuário.              |
| [selectedItem](listbox-selecteditem.md)         | Especifica ou recupera o índice do item selecionado no controle da caixa de listagem.                                        |
| [organizados](listbox-sorted.md)                     | Especifica ou recupera um valor que indica se o controle da caixa de listagem está classificado em ordem alfabética.                      |



 

O elemento **ListBox** dá suporte aos seguintes métodos.



| Método                                                 | Descrição                                                                                                           |
|--------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [appendItem](listbox-appenditem.md)                   | Insere um novo texto no final do controle da caixa de listagem.                                                                  |
| [deleteAll](listbox-deleteall.md)                     | Exclui todos os itens do controle caixa de listagem.                                                                          |
| [deleteItem](listbox-deleteitem.md)                   | Exclui o item de controle de caixa de listagem no índice especificado.                                                             |
| [desativar](listbox-dismiss.md)                         | Oculta o controle.                                                                                                    |
| [findItem](listbox-finditem.md)                       | Pesquisa uma determinada cadeia de caracteres começando com o item após o índice de item especificado.                                |
| [getItem](listbox-getitem.md)                         | Recupera o texto do item com o índice especificado.                                                             |
| [getNextSelectedItem](listbox-getnextselecteditem.md) | Recupera o próximo item selecionado no controle caixa de listagem, começando no item após aquele com o índice especificado. |
| [insertItem](listbox-insertitem.md)                   | Insere o texto especificado no índice especificado no controle da caixa de listagem.                                            |
| [replaceItem](listbox-replaceitem.md)                 | Substitui o texto no índice especificado pelo texto especificado.                                                     |
| [setselecionostate](listbox-setselectedstate.md)       | Seleciona ou anula a seleção do item com o índice especificado.                                                               |
| [show](listbox-show.md)                               | Exibe o controle.                                                                                                 |



 

> [!Note]  
> Esse elemento requer o Windows Media Player para Windows XP ou posterior.
