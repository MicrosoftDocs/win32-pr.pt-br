# <a name="structures-for-type-1-online-stores"></a>Estruturas para lojas online do tipo 1

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

A estrutura a seguir está disponível para lojas online do tipo 1 usando código C++.



| Estrutura                                    | Descrição                                 |
|----------------------------------------------|---------------------------------------------|
| [WMPContextMenuInfo](/previous-versions/windows/desktop/api/contentpartner/ns-contentpartner-wmpcontextmenuinfo) | Contém dados sobre um comando de menu de contexto. |
