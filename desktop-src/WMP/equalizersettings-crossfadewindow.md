# <a name="equalizersettingscrossfadewindow"></a>EQUALIZERSETTINGS.crossFadeWindow

O atributo **crossFadeWindow** especifica ou recupera a quantidade de sobreposição de esmaecimento cruzado em milissegundos.

``` syntax
        elementID.crossFadeWindow
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**) que varia de 0 a 10.000 com um valor padrão de 250.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
