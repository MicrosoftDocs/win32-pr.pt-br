# <a name="displaying-web-pages-in-windows-media-player"></a>Exibindo páginas da Web no Windows Media Player

O recurso HTMLView do Windows Media Player permite que os provedores de conteúdo da Internet (ICPs) usem o recurso **agora em execução** do player de modo completo para exibir o conteúdo baseado na Web enquanto o conteúdo de mídia digital está sendo reproduzido. (Não há suporte para HTMLView pelo controle do Windows Media Player.) No passado, o ICPs tinha que exibir anúncios em janelas separadas do navegador da Web (chamado de "propaganda pop-up") ou teve de adicionar players de mídia digital personalizados às páginas da sua página. Com o HTMLView, o ICPs pode fornecer uma experiência personalizada e integrada criada especificamente para o Windows Media Player 9 Series e o Windows Media Player 10, garantindo que o conteúdo de mídia digital ainda reproduza em versões anteriores do Windows Media Player.

-   [Criando uma apresentação do HTMLView](creating-an-htmlview-presentation.md)
-   [Adicionando um controle do Windows Media Player inserido](adding-an-embedded-windows-media-player-control.md)
-   [Garantindo que o Windows Media Player Abra o conteúdo do HTMLView](ensuring-that-windows-media-player-opens-the-htmlview-content.md)
