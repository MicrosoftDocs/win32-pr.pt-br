# <a name="displaying-web-pages-in-windows-media-player"></a>Exibindo páginas da Web no Windows Media Player

o recurso de Windows Media Player HTMLView permite que os provedores de conteúdo da Internet (ICPs) usem o recurso **agora em execução** do Player de modo completo para exibir o conteúdo baseado na Web enquanto o conteúdo de mídia digital está sendo reproduzido. (o controle Windows Media Player não dá suporte para HTMLView.) No passado, o ICPs tinha que exibir anúncios em janelas separadas do navegador da Web (chamado de "propaganda pop-up") ou teve de adicionar players de mídia digital personalizados às páginas da sua página. com o HTMLView, o ICPs pode fornecer uma experiência personalizada e integrada criada especificamente para o Windows Media Player 9 Series e o Windows Media Player 10, garantindo que o conteúdo da mídia digital ainda reproduza em versões anteriores do Windows Media Player.

Esta seção fornece informações sobre como usar o HTMLView, criar conteúdo do HTMLView e resolver possíveis problemas que você pode encontrar ao usar o recurso HTMLView nos seguintes tópicos:

-   [Criando uma apresentação do HTMLView](creating-an-htmlview-presentation.md)
-   [adicionando um controle de Windows Media Player inserido](adding-an-embedded-windows-media-player-control.md)
-   [garantindo que Windows Media Player abra o conteúdo do HTMLView](ensuring-that-windows-media-player-opens-the-htmlview-content.md)
-   [Problemas de página da Web](web-page-issues.md)
