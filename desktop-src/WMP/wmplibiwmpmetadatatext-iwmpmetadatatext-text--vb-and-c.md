# <a name="iwmpmetadatatexttext-property"></a>Propriedade IWMPMetadataText:: Text

A `text` propriedade Obtém o texto de metadados.

## <a name="syntax"></a>Syntax


```CSharp
public System.String text {get; set;}
```


```VB

Public ReadOnly Property text As System.String
```





## <a name="property-value"></a>Valor da propriedade

Um **System. String** que é o texto de metadados.

## <a name="remarks"></a>Comentários

Antes de usar essa propriedade, você deve ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
