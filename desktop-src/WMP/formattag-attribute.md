# <a name="formattag-attribute"></a>Atributo FormatTag

O atributo **FormatTag** é a marca de formato do codec de áudio.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de áudio](audio-item-attributes.md)
-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

O valor é uma representação de cadeia de caracteres do valor da marca de formato decimal. Por exemplo, para o codec do Windows Media Audio 9, o atributo **FormatTag** conteria a cadeia de caracteres "354".
