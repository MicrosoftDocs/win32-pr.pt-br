# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>inserindo o controle de Windows Media Player em um documento Office

a inserção do controle de Windows Media Player em um documento de Office é uma maneira fácil de adicionar conteúdo de mídia digital interativo e dinâmico a um documento estático de outra forma. por exemplo, você pode criar uma planilha em Microsoft Excel e inserir um vídeo de "cabeça de conversa" resumindo um relatório ou pode criar um documento de Microsoft Word e inserir uma pequena animação ilustrando um ponto no texto. se você não gostar da interface do usuário fornecida pelo controle, poderá usar o Microsoft Visual Basic for Applications (VBA) para fornecer uma interface do usuário personalizada.

inserir o controle de Windows Media Player em uma apresentação de slides do Microsoft PowerPoint® permite complementar os efeitos dinâmicos já fornecidos por esse programa. quando você salva a apresentação como uma página da web, o controle de Windows Media Player é inserido no HTML, conforme descrito em [usando o controle de Windows Media Player em uma página](using-the-windows-media-player-control-in-a-web-page.md). Isso permite que você adicione um código de script adicional que você pode ter desenvolvido para outras páginas da Web.

outra maneira de gerar facilmente páginas da web que incorporam o controle de Windows Media Player é com o® do Microsoft FrontPage. o FrontPage permite que você crie rapidamente uma página, adicione o controle de Windows Media Player a partir de um menu e configure-o em uma caixa de diálogo de propriedades. novamente, o HTML resultante incorpora o controle de Windows Media Player, exatamente como descrito anteriormente.
