# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Inserindo o controle do Windows Media Player em um documento do Office

A inserção do controle do Windows Media Player em um documento do Office é uma maneira fácil de adicionar conteúdo dinâmico e interativo de mídia digital a um documento estático de outra forma. Por exemplo, você pode criar uma planilha no Microsoft Excel e inserir um vídeo de "cabeça de fala" Resumindo um relatório ou pode criar um documento do Microsoft Word e inserir uma breve animação ilustrando um ponto no texto. Se você não gostar da interface do usuário fornecida pelo controle, poderá usar o Microsoft Visual Basic for Applications (VBA) para fornecer uma interface do usuário personalizada.

A inserção do controle do Windows Media Player em uma apresentação de slides do Microsoft PowerPoint® permite complementar os efeitos dinâmicos já fornecidos pelo programa. Quando você salva a apresentação como uma página da Web, o controle do Windows Media Player é inserido no HTML, conforme descrito em [usando o controle do Windows Media Player em uma página da Web](using-the-windows-media-player-control-in-a-web-page.md). Isso permite que você adicione um código de script adicional que você pode ter desenvolvido para outras páginas da Web.

Outra maneira de gerar facilmente páginas da Web que incorporam o controle do Windows Media Player é com o Microsoft FrontPage®. O FrontPage permite criar rapidamente uma página, adicionar o controle do Windows Media Player a partir de um menu e configurá-lo em uma caixa de diálogo de propriedades. Novamente, o HTML resultante incorpora o controle do Windows Media Player da mesma forma descrita anteriormente.
