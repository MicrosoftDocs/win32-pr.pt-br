# <a name="axwindowsmediaplayererror-property"></a>Propriedade AxWindowsMediaPlayer. Error

A Propriedade Error Obtém a interface IWMPError que fornece acesso a uma coleção de interfaces IWMPErrorItem.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public IWMPError Error {get;}
```


```VB

Public ReadOnly Property Error As IWMPError
```





## <a name="property-value"></a>Valor da propriedade

A interface WMPLib. IWMPError.
