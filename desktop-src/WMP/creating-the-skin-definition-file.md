# <a name="creating-the-skin-definition-file"></a>Criando o arquivo de definição de capa

O arquivo de definição de capa é um arquivo de texto simples. Você pode criar arquivos de definição de capa em qualquer editor de texto que salve arquivos de texto sem formatação ASCII, como o bloco de notas da Microsoft, Microsoft Visual InterDev® ou Microsoft Visual Studio. Como você estará escrevendo XML, não HTML, talvez não queira usar um editor para HTML porque ele pode adicionar códigos extras que você não deseja ou pode informá-lo sobre erros que você não tem. Provavelmente, você vai querer evitar processadores de texto como o Microsoft Word; Embora o Word possa salvar arquivos de texto sem formatação, talvez você não queira se lembrar de salvar como texto toda vez que salvar.

As seções a seguir irão guiá-lo na criação de um arquivo de definição de capa:

-   [Estrutura XML do arquivo de definição de aparência](skin-definition-file-xml-structure.md)
-   [Iniciar com tema e exibição](start-with-theme-and-view.md)
-   [Adicionando um botão](adding-buttongroup.md)
-   [Adicionando o botão de reprodução](adding-the-play-buttonelement.md)
-   [Adicionando o botão fechar](adding-the-close-buttonelement.md)
-   [Código completo para aparência simples](complete-code-for-simple-skin.md)
