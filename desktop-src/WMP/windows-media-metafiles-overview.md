# <a name="windows-media-metafiles-overview"></a>Windows Visão geral dos metaarquivos de mídia

a parte mais importante do uso bem-sucedido Windows metarquivos de mídia está usando a sintaxe correta para os elementos de metarquivo. os erros de sintaxe em um Windows metarquivo de mídia podem fazer com que qualquer um dos únicos atributos seja ignorado, para que o metarquivo não seja reconhecido como válido e não funcione.

quase tão importante é a ordem na qual os elementos aparecem em um metarquivo de mídia Windows. Os atributos de alguns elementos substituem temporariamente os atributos de elementos semelhantes em seções diferentes do metarquivo. Há uma [ordem de precedência](order-of-precedence.md)definida.

Windows as listas de reprodução do metarquivo de mídia são Windows arquivos de mídia que fornecem informações que Windows Media Player usa para receber fluxos unicast, fluxos multicast e outras mídias com suporte de uma intranet ou da Internet. Uma lista de reprodução de metarquivo é basicamente um atalho para conteúdo de mídia. Uma lista de reprodução de metarquivo pode ser enviada como email, usada como uma referência de link em uma página da Web, criada dinamicamente usando páginas de Active Server (ASP) ou existir como um arquivo autônomo em uma unidade de disco local. uma lista de reprodução de metarquivo pode fazer referência a outra lista de reprodução de metarquivo, uma página ASP ou um Windows arquivo de estação de mídia (com uma extensão de nome de arquivo. nsc). um arquivo. nsc é usado para definir uma Windows estação de mídia para Windows Media Player. O processo básico de manipulação é o mesmo para cada caso.
