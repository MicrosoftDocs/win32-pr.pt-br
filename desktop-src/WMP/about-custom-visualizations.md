# <a name="about-custom-visualizations"></a>Sobre visualizações personalizadas

O Windows Media Player fornece uma interface para criar visualizações personalizadas. Esta seção fornece uma visão geral da arquitetura de visualizações, informações sobre como elas se relacionam ao Windows Media Player e os detalhes técnicos que você precisará para criar visualizações personalizadas.

Observe que vários locais no código, você verá o efeito de palavra. As visualizações são efeitos.

As visualizações personalizadas são descritas mais detalhadamente nos tópicos a seguir.



| Tópico                                                              | Descrição                                                                                |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [Visão geral do usuário](user-overview.md)                                 | Descreve as visualizações da perspectiva de um usuário.                                          |
| [Visão geral do desenvolvedor](developer-overview.md)                       | Descreve as visualizações de uma perspectiva do desenvolvedor.                                     |
| [Criando uma visualização](building-a-visualization.md)           | Descreve como criar um projeto de visualização.                                           |
| [Implementando seu código](implementing-your-code.md)               | Descreve como modificar o código de exemplo gerado pelo assistente de plug-in de visualização.     |
| [Instalando visualizações](installing-visualizations.md)         | Descreve como instalar uma visualização no computador de um usuário.                             |
| [Enviando sua visualização](submitting-your-visualization.md) | Não há mais suporte para a distribuição de sua visualização concluída no site do WMPlugins. |
