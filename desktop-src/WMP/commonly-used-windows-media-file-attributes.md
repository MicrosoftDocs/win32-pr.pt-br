# <a name="commonly-used-windows-media-file-attributes"></a>Atributos de arquivo Windows mídia comumente usados

Windows Os arquivos de mídia podem conter qualquer número de atributos, incluindo atributos definidos pelo usuário.

Se o arquivo tiver sido adicionado à biblioteca, atributos adicionais associados ao tipo de mídia também poderão estar disponíveis. Consulte [Itens de áudio](audio-item-attributes.md) ou itens de [vídeo](video-item-attributes.md) para obter esses atributos adicionais.

Para obter mais informações sobre os atributos de arquivo Windows Media, consulte as Diretrizes Windows uso de [metadados de mídia.](/previous-versions/ms867702(v=msdn.10))

A lista a seguir contém atributos comumente usados.

-   [Atributo Author](author-attribute.md)
-   [Atributo title](title-attribute.md)
-   [Atributo WM/AlbumArtist](wm-albumartist-attribute.md)
-   [Atributo WM/ContentDistributor](wm-contentdistributor-attribute.md)
-   [Atributo WM/Genre](wm-genre-attribute.md)
-   [Atributo WM/MCDI](wm-mcdi-attribute.md)
-   [Atributo WM/MediaClassPrimaryID](wm-mediaclassprimaryid-attribute.md)
-   [Atributo WM/MediaClassSecondaryID](wm-mediaclasssecondaryid-attribute.md)
-   [Atributo WM/Provider](wm-provider-attribute.md)
-   [Atributo WM/TrackNumber](wm-tracknumber-attribute.md)
