# <a name="commonly-used-windows-media-file-attributes"></a>Atributos de arquivo de mídia do Windows comumente usados

Os arquivos de mídia do Windows podem conter qualquer número de atributos, incluindo atributos definidos pelo usuário.

Se o arquivo tiver sido adicionado à biblioteca, os atributos adicionais associados ao tipo de mídia também poderão estar disponíveis. Consulte [itens de áudio](audio-item-attributes.md) ou [itens de vídeo](video-item-attributes.md) para esses atributos adicionais.

Para obter mais informações sobre os atributos de arquivo de mídia do Windows, consulte as [diretrizes de uso de metadados do Windows Media](/previous-versions/ms867702(v=msdn.10)).

A lista a seguir contém atributos usados com frequência.

-   [Atributo de autor](author-attribute.md)
-   [Atributo de título](title-attribute.md)
-   [Atributo WM/AlbumArtist](wm-albumartist-attribute.md)
-   [Atributo WM/ContentDistributor](wm-contentdistributor-attribute.md)
-   [Atributo do WM/gênero](wm-genre-attribute.md)
-   [Atributo WM/MCDI](wm-mcdi-attribute.md)
-   [Atributo WM/MediaClassPrimaryID](wm-mediaclassprimaryid-attribute.md)
-   [Atributo WM/MediaClassSecondaryID](wm-mediaclasssecondaryid-attribute.md)
-   [Atributo do WM/provedor](wm-provider-attribute.md)
-   [Atributo WM/TrackNumber](wm-tracknumber-attribute.md)
