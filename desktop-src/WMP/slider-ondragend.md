# <a name="sliderondragend"></a>SLIDER. onDragEnd

O manipulador de eventos **onDragEnd** manipula um evento que ocorre quando o botão esquerdo do mouse é liberado após uma operação de arrastar.

``` syntax
onDragEnd
```

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
