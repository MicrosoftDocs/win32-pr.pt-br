# <a name="about-skins"></a>Sobre as capas

Com o Windows Media Player, os usuários podem escolher entre uma variedade de capas padrão, cada uma fornecendo uma experiência visual adicional que aprimora o prazer de escuta e exibição. O Windows Media Player vem com várias capas para escolher, mas é relativamente fácil criar e distribuir capas personalizadas.

As capas são apenas coleções de um ou mais arquivos de arte de computador, organizadas por um arquivo de texto, que diz ao Windows Media Player como usar esses arquivos para exibir uma capa. Você pode usar uma variedade de programas para modificar arte existente ou criar arte simples. Por exemplo, usando as ferramentas de desenho e o clip-art fornecido com o Microsoft PowerPoint, você pode criar um trabalho artístico apenas arrastando e soltando. Da mesma forma, você não precisa ser um programador para criar o arquivo de texto usado para definir uma capa individual.

Um dos motivos pelos quais as capas são relativamente fáceis de criar é que você pode examinar as folhas de arte individuais e o código-fonte de qualquer capa e ver como cada uma delas é reunida. Várias capas de exemplo simples são fornecidas com esse SDK, bem como uma referência para elementos e atributos de pele.
