<span data-ttu-id="4dad8-108">Com o Windows Media Player, os usuários podem escolher entre uma variedade de capas padrão, cada uma fornecendo uma experiência visual adicional que aprimora o prazer de escuta e exibição.</span><span class="sxs-lookup"><span data-stu-id="4dad8-108">With Windows Media Player, users can choose from a variety of standard skins, each one providing an additional visual experience that enhances listening and viewing pleasure.</span></span> <span data-ttu-id="4dad8-109">O Windows Media Player vem com várias capas para escolher, mas é relativamente fácil criar e distribuir capas personalizadas.</span><span class="sxs-lookup"><span data-stu-id="4dad8-109">Windows Media Player comes with several skins to choose from, but it is relatively easy to create and distribute custom skins.</span></span>

<span data-ttu-id="4dad8-110">As capas são apenas coleções de um ou mais arquivos de arte de computador, organizadas por um arquivo de texto, que diz ao Windows Media Player como usar esses arquivos para exibir uma capa.</span><span class="sxs-lookup"><span data-stu-id="4dad8-110">Skins are simply collections of one or more computer art files, organized by a text file, which tells Windows Media Player how to use these files to display a skin.</span></span> <span data-ttu-id="4dad8-111">Você pode usar uma variedade de programas para modificar arte existente ou criar arte simples.</span><span class="sxs-lookup"><span data-stu-id="4dad8-111">You can use a variety of programs to modify existing art or create simple art of your own.</span></span> <span data-ttu-id="4dad8-112">Por exemplo, usando as ferramentas de desenho e o clip-art fornecido com o Microsoft PowerPoint, você pode criar um trabalho artístico apenas arrastando e soltando.</span><span class="sxs-lookup"><span data-stu-id="4dad8-112">For example, using the drawing tools and clip art supplied with Microsoft PowerPoint, you can create artwork by just dragging and dropping.</span></span> <span data-ttu-id="4dad8-113">Da mesma forma, você não precisa ser um programador para criar o arquivo de texto usado para definir uma capa individual.</span><span class="sxs-lookup"><span data-stu-id="4dad8-113">Similarly, you do not need to be a programmer to create the text file used to define an individual skin.</span></span>

<span data-ttu-id="4dad8-114">Um dos motivos pelos quais as capas são relativamente fáceis de criar é que você pode examinar as folhas de arte individuais e o código-fonte de qualquer capa e ver como cada uma delas é reunida.</span><span class="sxs-lookup"><span data-stu-id="4dad8-114">One of the reasons that skins are relatively easy to create is that you can look at the individual pieces of art and the source code for any skin and see how each skin is put together.</span></span> <span data-ttu-id="4dad8-115">Várias capas de exemplo simples são fornecidas com esse SDK, bem como uma referência para elementos e atributos de pele.</span><span class="sxs-lookup"><span data-stu-id="4dad8-115">Several simple example skins are provided with this SDK, as well as a reference for skin elements and attributes.</span></span>
