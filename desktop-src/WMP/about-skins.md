# <a name="about-skins"></a>Sobre capas

Com Windows Media Player, os usuários podem escolher entre uma variedade de capas padrão, cada uma fornecendo uma experiência visual adicional que aprimora a escuta e a exibição de comentários. Windows Media Player vem com várias capas para escolher, mas é relativamente fácil criar e distribuir capas personalizadas.

As capas são simplesmente coleções de um ou mais arquivos de arte de computador, organizados por um arquivo de texto, que Windows Media Player como usar esses arquivos para exibir uma capa. Você pode usar uma variedade de programas para modificar a arte existente ou criar sua própria arte. Por exemplo, usando as ferramentas de desenho e a clip-art fornecidas com o Microsoft PowerPoint, você pode criar arte apenas arrastando e soltando. Da mesma forma, você não precisa ser um programador para criar o arquivo de texto usado para definir uma capa individual.

Um dos motivos pelos quais as capas são relativamente fáceis de criar é que você pode ver as partes de arte individuais e o código-fonte de qualquer capa e ver como cada capa é colocada junto. Várias capas de exemplo simples são fornecidas com esse SDK, bem como uma referência para elementos e atributos de capa.
