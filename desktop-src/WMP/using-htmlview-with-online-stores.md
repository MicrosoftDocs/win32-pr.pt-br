# <a name="using-htmlview-with-online-stores"></a>Usando HTMLView com lojas online

Windows Media Player solicita aos usuários permissão para exibir conteúdo HTMLView em **Agora Em Reprodução.** As lojas online podem desabilitar esse prompt especificando um valor de URL base no **elemento HTMLView** do documento ServiceInfo. Quando Windows Media Player uma playlist que especifica o conteúdo htmlView a ser exibido, o Player compara a URL base do documento ServiceInfo do armazenamento online ativo atual com a URL base do conteúdo HTMLView. Se eles corresponderem, Windows Media Player exibirá o conteúdo htmlView sem solicitar ao usuário.
