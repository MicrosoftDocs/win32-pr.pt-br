# <a name="using-htmlview-with-online-stores"></a>Usando o HTMLView com lojas online

O Windows Media Player solicita aos usuários a permissão para exibir o conteúdo do HTMLView no **momento em execução**. As lojas online podem desabilitar esse prompt especificando um valor de URL base no elemento **HtmlView** do documento serviceInfo. Quando o Windows Media Player abre uma lista de reprodução que especifica o conteúdo do HTMLView a ser exibido, o Player compara a URL base do documento do serviceInfo do repositório online ativo atual com a URL base do conteúdo do HTMLView. Se eles corresponderem, o Windows Media Player exibirá o conteúdo do HTMLView sem avisar o usuário.
