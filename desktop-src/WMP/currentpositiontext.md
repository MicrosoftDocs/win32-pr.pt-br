# <a name="currentpositiontext"></a>CURRENTPOSITIONTEXT

Esse é um elemento **TEXT** predefinido com os seguintes valores padrão.

``` syntax
value="wmpprop:player.controls.currentPositionString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Comentários

Isso criará um **elemento TEXT** que exibirá a hora atual da mídia que está sendo tocada. Todas as propriedades desse **elemento TEXT** podem ser substituídos especificando-as explicitamente.
