# <a name="currentpositiontext"></a>CURRENTPOSITIONTEXT

Este é um elemento de **texto** predefinido com os seguintes valores padrão.

``` syntax
value="wmpprop:player.controls.currentPositionString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Comentários

Isso criará um elemento de **texto** que exibirá a hora atual da mídia que está sendo reproduzida. Todas as propriedades desse elemento de **texto** podem ser substituídas especificando-as explicitamente.
