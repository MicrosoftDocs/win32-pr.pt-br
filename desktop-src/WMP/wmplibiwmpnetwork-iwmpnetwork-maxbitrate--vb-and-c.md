# <a name="iwmpnetworkmaxbitrate-property"></a>Propriedade IWMPNetwork::maxBitRate

A **propriedade maxBitRate** obtém a taxa máxima de bits de vídeo possível.

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 maxBitRate {get; set;}
```


```VB

Public ReadOnly Property maxBitRate As System.Int32
```





## <a name="property-value"></a>Valor da propriedade

Um **System.Int32 que** é a taxa de bits máxima.
