# <a name="playlistbackgroundcolor"></a>PLAYLIST. backgroundColor

O atributo **backgroundColor** especifica ou recupera a cor do plano de fundo.

``` syntax
        elementID.backgroundColor
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém qualquer valor de cor do Microsoft Internet Explorer. Ele tem um valor padrão igual à cor da janela do sistema Windows.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
