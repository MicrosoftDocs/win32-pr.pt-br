# <a name="playlistitemplayingcolor"></a>PLAYLIST. itemPlayingColor

O atributo **itemPlayingColor** especifica ou recupera a cor de realce que indica o item em execução no momento na lista de reprodução.

``` syntax
        elementID.itemPlayingColor
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém qualquer valor de cor do Microsoft Internet Explorer. Ele tem um valor padrão de " \# 00FF00".

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
