# <a name="pausebutton"></a>PAUSEBUTTON

Este é um **botão** predefinido com os valores padrão a seguir.

``` syntax
onclick="jscript:player.controls.pause()" 
upToolTip="Pause"
cursor="system"
enabled="wmpenabled:player.controls.pause"
```

## <a name="remarks"></a>Comentários

Todas as propriedades desse **botão** podem ser substituídas especificando-as explicitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
