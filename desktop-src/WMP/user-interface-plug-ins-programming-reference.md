# <a name="user-interface-plug-ins-programming-reference"></a>Referência de programação de plug-ins de interface do usuário

o SDK (Software Development Kit) do Microsoft Windows Media Player oferece suporte a uma interface para plug-ins de interface do usuário. A seção a seguir documenta isso em detalhes.



| Seção                                                  | Descrição                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [WMPNotifyPluginAddRemove](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | uma função independente usada para notificar Windows Media Player que um plug-in foi instalado ou desinstalado. |
| [Interface IWMPPluginUI](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Uma interface para plug-ins de interface do usuário.                                                                                  |
| [Sinalizadores de registro](registration-flags.md)             | Uma lista de sinalizadores de configurações que afetam o comportamento dos plug-ins de interface do usuário.                                             |
