# <a name="writing-code"></a>Escrevendo código

O poder real das capas vem de ser capaz de responder a eventos. Você quer que sua capa seja capaz de fazer algo quando o usuário clica em um botão, e você também deseja responder a alterações que ocorrem no Windows Media Player, como o progresso do arquivo de mídia digital ou do fluxo que está sendo executado.

As seções a seguir abordam a escrita de código para capas:



| Seção                                | Descrição                                 |
|----------------------------------------|---------------------------------------------|
| [Manipulando eventos](handling-events.md) | Explica como tratar eventos no código de capa. |
| [Usando o JScript](using-jscript.md)     | Explica como escrever código de script em capas. |
