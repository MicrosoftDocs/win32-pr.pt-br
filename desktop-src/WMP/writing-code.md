# <a name="writing-code"></a>Escrevendo código

O poder real das capas vem da capacidade de responder a eventos. Você deseja que sua capa possa fazer algo quando o usuário clicar em um botão e também responder às alterações que ocorrem ao Windows Media Player, como o progresso do arquivo de mídia digital ou fluxo que está sendo transmitido.

As seções a seguir discutem a escrita de código para capas:



| Seção                                | Descrição                                 |
|----------------------------------------|---------------------------------------------|
| [Manipulando eventos](handling-events.md) | Explica como manipular eventos no código de capa. |
| [Usando JScript](using-jscript.md)     | Explica como escrever código de script em capas. |
