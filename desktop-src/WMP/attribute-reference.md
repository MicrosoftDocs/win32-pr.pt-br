# <a name="attribute-reference"></a>Referência de atributo

O Windows Media Player referência de atributo documenta atributos de metadados que são de interesse para desenvolvedores Windows Media Player SDK. A tabela a seguir lista esses atributos em várias categorias.



| Categoria                                                                                       | Descrição                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Referência alfabética](alphabetical-attribute-reference.md)                                 | Fornece uma lista alfabética de atributos predefinidos.                                                                  |
| [Itens de áudio](audio-item-attributes.md)                                                       | Lista os atributos que estão disponíveis para um item de áudio na biblioteca.                                                    |
| [CD Playlists](cd-playlist-attributes.md)                                                     | Lista os atributos que estão disponíveis para um CD.                                                                            |
| [Faixas de CD](cd-track-attributes.md)                                                           | Lista os atributos que estão disponíveis para uma faixa de CD.                                                                      |
| [Atributos de arquivo Windows mídia comumente usados](commonly-used-windows-media-file-attributes.md) | Lista os atributos que estão disponíveis para um arquivo de mídia digital.                                                            |
| [DVDs](dvd-attributes.md)                                                                     | Lista os atributos que estão disponíveis para um DVD.                                                                           |
| [Arquivos de música](music-file-attributes.md)                                                       | Lista atributos que estão disponíveis para um arquivo de música que não está na biblioteca.                                         |
| [Outros itens](other-item-attributes.md)                                                       | Lista os atributos que estão disponíveis para itens na biblioteca que não são áudio, vídeo, playlist, foto ou itens de rádio. |
| [Itens de foto](photo-item-attributes.md)                                                       | Lista os atributos que estão disponíveis para um item de foto na biblioteca.                                                     |
| [Playlists](playlist-attributes-ref.md)                                                       | Lista os atributos disponíveis para playlists recuperadas como itens de mídia da biblioteca.                             |
| [Itens de rádio](radio-item-attributes.md)                                                       | Lista os atributos que estão disponíveis para um item de rádio na biblioteca.                                                     |
| [Itens de vídeo](video-item-attributes.md)                                                       | Lista os atributos que estão disponíveis para um item de vídeo na biblioteca.                                                     |



 

Para obter mais informações sobre os atributos de metadados, consulte as Diretrizes Windows uso de [metadados de mídia.](/previous-versions/ms867702(v=msdn.10))
