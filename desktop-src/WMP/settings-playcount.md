# <a name="settingsplaycount"></a>Settings. playCount

A propriedade **playCount** especifica ou recupera o número de vezes que um item de mídia será reproduzido.

## <a name="syntax"></a>Syntax

Player. Settings. playCount

## <a name="possible-values"></a>Valores possíveis

Essa propriedade é um **número** de leitura/gravação (**longo**) com um valor mínimo de um e um valor padrão de um.

**Windows Media Player 10 Mobile**: não há suporte para essa propriedade.
