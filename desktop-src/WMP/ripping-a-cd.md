# <a name="ripping-a-cd"></a>Copiando um CD

há duas maneiras de copiar um CD usando o controle de Windows Media Player. o Windows Media Player 9 pode copiar CDs usando **IWMPPlayerServices:: setTaskPane**. o Windows Media Player 11 apresenta a interface **IWMPCdromRip** para copiar CDs. Essa interface fornece mais funcionalidade do que **IWMPPlayerServices:: setTaskPane**, e é recomendável que você use o **IWMPCdromRip** se ele estiver disponível.

As seções a seguir descrevem como copiar um CD usando Windows Media Player.

-   [Copiando usando a interface IWMPCdromRip](ripping-by-using-the-iwmpcdromrip-interface.md)
-   [Copiando por meio de IWMPPlayerServices:: setTaskPane](ripping-by-using-iwmpplayerservices--settaskpane.md)
