# <a name="onclick"></a>OnClick

O manipulador de eventos **onclick** manipula um evento que ocorre quando o usuário clica no elemento.

``` syntax
onclick
```

## <a name="remarks"></a>Comentários

Esse manipulador de eventos não é aplicável ao elemento **playlist** , o elemento **Popup** , o elemento **Video** , o elemento **ListBox** , quando *Video*. **sem janela** é falso ou o elemento **Effects** quando há *efeitos*. a **janela** é verdadeira.
