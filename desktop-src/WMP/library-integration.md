<span data-ttu-id="889bc-113">A interface do usuário do Windows Media Player é organizada em áreas de recursos, chamadas de *painéis de tarefas*, que encapsulam os vários recursos de alto nível do programa.</span><span class="sxs-lookup"><span data-stu-id="889bc-113">The Windows Media Player user interface is organized into feature areas, called *task panes*, which encapsulate the various high-level features of the program.</span></span> <span data-ttu-id="889bc-114">Isso inclui os painéis de tarefas **biblioteca**, **sincronização** e **gravação** (entre outros).</span><span class="sxs-lookup"><span data-stu-id="889bc-114">These include the **Library**, **Sync**, and **Burn** task panes (among others).</span></span> <span data-ttu-id="889bc-115">O painel de tarefas **biblioteca** permite que os usuários trabalhem com a biblioteca; o painel de tarefas **sincronizar** permite que os usuários sincronizem arquivos de mídia digital para um dispositivo portátil; e o painel de tarefas **gravar** permite que os usuários gravem arquivos de mídia digital em um CD ou DVD.</span><span class="sxs-lookup"><span data-stu-id="889bc-115">The **Library** task pane enables users to work with the library; the **Sync** task pane enables users to synchronize digital media files to a portable device; and the **Burn** task pane enables users to burn digital media files to a CD or DVD.</span></span>

> [!Note]  
> <span data-ttu-id="889bc-116">O painel de tarefas da **biblioteca** é, às vezes, chamado de painel de tarefas **procurar** .</span><span class="sxs-lookup"><span data-stu-id="889bc-116">The **Library** task pane is sometimes called the **Browse** task pane.</span></span>

<span data-ttu-id="889bc-121">Cada um desses três painéis de tarefas é organizado em três partes principais.</span><span class="sxs-lookup"><span data-stu-id="889bc-121">Each of these three task panes <span data-ttu-id="889bc-129">Quando um catálogo de loja online é integrado à biblioteca, a loja online aparece como uma categoria de nível superior, ou *nó*, no controle de exibição de árvore de biblioteca.</span><span class="sxs-lookup"><span data-stu-id="889bc-129">When an online store catalog is integrated with the library, the online store appears as a top-level category, or *node*, in the library tree-view control.</span></span> <span data-ttu-id="889bc-130">(Somente um catálogo de loja online fica visível para o usuário de cada vez.) <span data-ttu-id="889bc-132">O nó de armazenamento online de nível superior tem um conjunto de nós filho fornecidos pelo Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="889bc-132">The top-level online store node has a set of child nodes provided by Windows Media Player.</span></span> <span data-ttu-id="889bc-133">Por exemplo, o nó de armazenamento online de nível superior tem nós filho de rádio, artista e álbum, entre outros.</span><span class="sxs-lookup"><span data-stu-id="889bc-133">For example, the top-level online store node has Radio, Artist, and Album child nodes, among others.</span></span> <span data-ttu-id="889bc-134">O nó de armazenamento online de nível superior também pode ter até oito nós filho personalizados fornecidos pela loja online.</span><span class="sxs-lookup"><span data-stu-id="889bc-134">The top-level online store node can also have up to eight custom child nodes provided by the online store.</span></span> <span data-ttu-id="889bc-135">O Windows Media Player cria um nó filho personalizado para qualquer lista que tenha um identificador de lista no intervalo de 0 a 7.</span><span class="sxs-lookup"><span data-stu-id="889bc-135">Windows Media Player creates a custom child node for any list that has a list identifier in the range 0 through 7.</span></span> <span data-ttu-id="889bc-136">A loja online especifica o identificador de uma lista no arquivo de [list.csv](list-csv.md) que faz parte do catálogo do repositório.</span><span class="sxs-lookup"><span data-stu-id="889bc-136">The online store specifies the identifier of a list in the [list.csv](list-csv.md) file that is part of the store's catalog.</span></span>

<span data-ttu-id="889bc-138">À medida que o usuário navega pelo catálogo, o Windows Media Player faz chamadas para **IWMPContentPartner:: GetItemInfo** para recuperar metadados do plug-in de parceiro de conteúdo sobre os itens de música que o usuário seleciona.</span><span class="sxs-lookup"><span data-stu-id="889bc-138">As the user navigates through the catalog, Windows Media Player makes calls to **IWMPContentPartner::GetItemInfo** to retrieve metadata from the content partner plug-in about the music items the user selects.</span></span> <span data-ttu-id="889bc-139">Esses metadados fornecem informações ao Player para que o Player possa exibir detalhes sobre os itens do catálogo.</span><span class="sxs-lookup"><span data-stu-id="889bc-139">This metadata provides information to the Player so that the Player can display details about the catalog items.</span></span> <span data-ttu-id="889bc-140">Por exemplo, se o usuário selecionar um álbum, o Windows Media Player recuperará a URL de arte do álbum para que o usuário possa ver a arte da capa do álbum.</span><span class="sxs-lookup"><span data-stu-id="889bc-140">For example, if the user selects an album, Windows Media Player retrieves the album art URL so that the user can see the album cover art.</span></span>
