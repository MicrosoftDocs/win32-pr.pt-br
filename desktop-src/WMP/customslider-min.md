# <a name="customslidermin"></a>QUEDISTRLIDER.min

O **atributo min** especifica ou recupera o valor mínimo do intervalo definido pelo controle deslizante personalizado.

``` syntax
        elementID.min
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um  número de leitura/gravação (**float**) com um valor padrão de zero.

## <a name="remarks"></a>Comentários

O valor de **min** deve ser menor que **o máximo.**

## <a name="examples"></a>Exemplos

Consulte o [atributo positionImage](customslider-positionimage.md) para ver um exemplo que ilustra como os atributos do **elemento ATTRIBUTESLIDER** são usados.
