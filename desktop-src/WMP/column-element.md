# <a name="column-element"></a>Elemento COLUMN

O elemento **Column** representa uma coluna em um controle playlist. Os elementos de **coluna** são sempre filhos dos elementos de **playlist** .

O elemento **Column** dá suporte aos seguintes atributos.



| Atributo                                       | Descrição                                                        |
|-------------------------------------------------|--------------------------------------------------------------------|
| [columnName](column-columnname.md)             | Especifica ou recupera um nome de coluna no controle **playlist** .  |
| [columnID](column-columnid.md)                 | Especifica ou recupera uma ID de coluna no controle **playlist** .    |
| [columnResizeMode](column-columnresizemode.md) | Especifica ou recupera o modo de redimensionamento para esta coluna.            |
| [columnWidth](column-columnwidth.md)           | Especifica ou recupera uma largura de coluna no controle **playlist** . |
