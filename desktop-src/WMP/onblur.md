# <a name="onblur"></a>Onblur

O **manipulador de eventos onblur** lida com um evento que ocorre quando o elemento perde o foco do teclado.

``` syntax
onblur
```

## <a name="remarks"></a>Comentários

O **manipulador de eventos onblur** não é aplicável ao elemento **EFFECTS** ou ao **elemento POPUP.**

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------|
| Versão<br/> | Windows Media Player versão 70 ou posterior<br/> |
