# <a name="submitting-your-visualization"></a>Enviando sua visualização

Não há mais suporte para o envio da visualização por meio do site WMPlugins.

## <a name="test-your-visualization"></a>Testar sua visualização

Talvez você queira tentar instalar sua visualização em vários máquinas para ter certeza de que tudo funciona da maneira que você deseja. Experimente cada predefinição para garantir que seu código funcione corretamente.

## <a name="sign-your-visualization"></a>Assinar sua visualização

Você deve enviar sua visualização para assinatura de código para que o usuário possa decidir se deve confiar na visualização antes de instalá-la. Para obter informações sobre assinatura de código, consulte [Introdução à assinatura de código.](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85))

## <a name="distributing-your-visualization"></a>Distribuindo sua visualização

Embora o site WMPlugins não seja mais compatível com a distribuição de sua visualização, você ainda pode distribuir sua visualização de um site de sua escolha.
