# <a name="initial-lines"></a>Linhas iniciais

Cada arquivo de definição de capa para o Windows Media Player Mobile deve começar com uma linha que especifica o número de versão do arquivo de capa. A linha a seguir funciona para o Windows Media Player 9 Series para Windows Mobile 2003 ou posterior:


```C++
[Pocket WMP Skin File v9.0.1]

```
