# <a name="onmousedown"></a>OnMouseDown

O manipulador de eventos **OnMouseDown** manipula um evento que ocorre quando o usuário clica em um botão do mouse.

``` syntax
onmousedown
```

## <a name="remarks"></a>Comentários

Esse manipulador de eventos não é aplicável ao elemento **playlist** , o elemento **Video** quando o *vídeo*. **sem janela** é falso ou o elemento **Effects** quando há *efeitos*. a **janela** é verdadeira.
