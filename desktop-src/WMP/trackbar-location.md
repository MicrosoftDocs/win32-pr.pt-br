# <a name="trackbar-location"></a>Local TrackBar

Você deve definir o local do TrackBar que deseja usar. Você deve fornecer quatro inteiros positivos, separados por vírgulas, que definem a esquerda, a parte superior, a largura e a altura do TrackBar, em pixels, em relação à imagem de plano de fundo.


```C++
180,46,50,30 

```
