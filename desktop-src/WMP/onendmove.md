# <a name="onendmove"></a>onendmove

O manipulador de eventos **onendmove** manipula um evento que ocorre quando um elemento conclui uma operação MoveTo.

``` syntax
onendmove
```

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------|
| Versão<br/> | Windows Media Player versão 70 ou posterior<br/> |
