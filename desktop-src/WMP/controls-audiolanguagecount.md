# <a name="controlsaudiolanguagecount"></a>Controls. audioLanguageCount

A propriedade **audioLanguageCount** recupera a contagem de idiomas de áudio com suporte.

``` syntax
player.controls.audioLanguageCount
      
```

## <a name="possible-values"></a>Valores possíveis

Essa propriedade é um **número** somente leitura (**Long**).

## <a name="remarks"></a>Comentários

Para conteúdo baseado no Windows Media, as propriedades e os métodos relacionados à seleção de idioma dão suporte apenas a uma única saída.

**Windows Media Player 10 Mobile:** Essa propriedade sempre retorna 1.
