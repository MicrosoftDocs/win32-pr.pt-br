# <a name="wmvideoframerate-attribute"></a>Atributo WM/VideoFrameRate

O atributo **WM/VideoFrameRate** é um valor decimal que especifica a taxa de quadros (em quadros por segundo) para um arquivo de vídeo.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

A constante do Windows Media Format SDK para esse atributo é g \_ wszWMVideoFrameRate.

Para determinar se você pode alterar o valor desse atributo, use o método [Media. isReadOnlyItem](media-isreadonlyitem.md) .
