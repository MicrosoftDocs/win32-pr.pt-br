# <a name="handling-events"></a>Tratando eventos

além do código XML que você escreve para inicializar atributos para sua capa, o código principal que você escreve é JScript código para manipular eventos. Os eventos são externos ou internos; ou seja, eles são provenientes do usuário ou de Windows Media Player.

As seções a seguir fornecem detalhes sobre como manipular eventos:

-   [Eventos externos](external-events.md)
-   [Eventos internos](internal-events.md)
-   [Gravando código de evento](writing-event-code.md)
-   [Eventos secundários](secondary-events.md)
