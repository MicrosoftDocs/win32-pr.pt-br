# <a name="playlistcollectionsetdeleted-method"></a>Método playlistcollection. setdeleted

O método **Setdeled** move uma lista de reprodução para a pasta itens excluídos.

## <a name="syntax"></a>Sintaxe


```JScript
PlaylistCollection.setDeleted(
  playlist,
  true
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*lista de reprodução* \[ no\]
</dt> <dd>

O objeto de **playlist** a ser movido.

</dd> <dt>

*verdadeiro* \[ no\]
</dt> <dd>

Sempre Especifique esse valor.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

**Windows Media Player 10 Mobile**: não há suporte para esse método.
