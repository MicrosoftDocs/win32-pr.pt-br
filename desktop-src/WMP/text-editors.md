# <a name="text-editors"></a>Editores de Texto

Você deve usar um editor de texto que não inclua nenhum caractere de formatação. Um editor de texto simples como Bloco de notas é recomendado, embora ambientes de desenvolvimento, como Microsoft Visual Studio ofereçam recursos avançados. Você não deve usar um editor como Microsoft Word, porque o formato de arquivo padrão do Word salva códigos de formatação extras que não serão usáveis. Se você usar um processador de palavras como o Word, lembre-se de salvar seu trabalho como texto não formatado sempre que salvar o arquivo.

Provavelmente, você deve desativar o quebra-texto se o editor de texto permitir, e talvez você queira ativar a capacidade de exibir todos os caracteres para garantir que você não adicione acidentalmente nenhuma guia ou outros caracteres de controle. Lembre-se de que incluir caracteres de tabulação no arquivo de definição de capa fará com que a capa falhe.
