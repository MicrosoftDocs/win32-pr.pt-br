# <a name="text-editors"></a>Editores de texto

Você deve usar um editor de texto que não inclua nenhum caractere de formatação. Um editor de texto simples, como o bloco de notas, é recomendado, embora ambientes de desenvolvimento, como Microsoft Visual Studio, ofereçam recursos avançados. Você não deve usar um editor como o Microsoft Word, pois o formato de arquivo padrão do Word salva códigos de formatação extras que não serão utilizáveis. Se você usar um processador de texto como o Word, deverá se lembrar de salvar seu trabalho como texto não formatado toda vez que salvar o arquivo.

Você provavelmente deverá desativar a quebra automática de texto se o editor de textos permitir, e talvez você queira ativar a capacidade de exibir todos os caracteres para certificar-se de não adicionar nenhuma Tabulação ou outro caractere de controle acidentalmente. Lembre-se de que a inclusão de caracteres de tabulação no arquivo de definição de capa fará com que a aparência falhe.
