<span data-ttu-id="30035-109">Quando o Windows Media Player habilita um plug-in do DSP, ele pode criar e destruir muitas instâncias do objeto de plug-in durante o curso de uma sessão.</span><span class="sxs-lookup"><span data-stu-id="30035-109">When Windows Media Player enables a DSP plug-in, it may create and destroy many instances of the plug-in object during the course of a session.</span></span> <span data-ttu-id="30035-110">O plug-in precisa de uma maneira de manter seus valores de propriedade entre instâncias.</span><span class="sxs-lookup"><span data-stu-id="30035-110">The plug-in needs a way to persist its property values between instances.</span></span> <span data-ttu-id="30035-111">O código de exemplo gerado pelo assistente de plug-in do Windows Media Player armazena esses valores no registro e os recupera quando a página de propriedades é invocada ou quando uma nova instância do plug-in é criada.</span><span class="sxs-lookup"><span data-stu-id="30035-111">The sample code generated by the Windows Media Player Plug-in Wizard stores these values in the registry and retrieves them when the property page is invoked or when a new instance of the plug-in is created.</span></span>

<span data-ttu-id="30035-112">O código de exemplo padrão em Echo. h inclui duas constantes que armazenam o caminho de registro padrão e a cadeia de caracteres de nome do fator de escala.</span><span class="sxs-lookup"><span data-stu-id="30035-112">The default sample code in Echo.h includes two constants that store the default registry path and the scale factor name string.</span></span> <span data-ttu-id="30035-113">Você deve manter a variável que especifica o caminho, mas excluir a linha que especifica o nome do registro do fator de escala.</span><span class="sxs-lookup"><span data-stu-id="30035-113">You should keep the variable that specifies the path, but delete the line that specifies the scale factor registry name.</span></span> <span data-ttu-id="30035-114">Em seguida, adicione o código a seguir para definir constantes para o tempo de atraso e os nomes de propriedade de combinação úmida no registro.</span><span class="sxs-lookup"><span data-stu-id="30035-114">Then, add the following code to define constants for the delay time and wet mix property names in the registry.</span></span> <span data-ttu-id="30035-115">A seção concluído deve aparecer da seguinte maneira:</span><span class="sxs-lookup"><span data-stu-id="30035-115">The finished section should appear as follows:</span></span>

const TCHAR kszPrefsRegKey[] = _T("Software\\Echo\\DSP Plugin");
const TCHAR kszPrefsDelayTime[] = _T("DelayTime");
const TCHAR kszPrefsWetmix[] = _T("Wetmix");
