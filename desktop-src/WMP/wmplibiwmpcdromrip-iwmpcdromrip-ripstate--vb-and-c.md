# <a name="iwmpcdromripripstate-property"></a>Propriedade IWMPCdromRip:: ripState

A propriedade **ripState** Obtém um valor de enumeração que indica o estado atual do processo de cópia de CDs.

## <a name="syntax"></a>Syntax


```CSharp
public WMPRipState ripState {get; set;}
```


```VB

Public ReadOnly Property ripState As WMPRipState
```





## <a name="property-value"></a>Valor da propriedade

Um **WMPLib. WMPRipState** que é um valor da enumeração **WMPRipState** que indica o estado atual.
