# <a name="debugging-the-page"></a>Depurando a página

Quando o Windows Media Player hospeda sua página da Web da loja online, ele desabilita a capacidade de abrir caixas de diálogo da página da Web. Para depurar o código da página da Web, você deve adicionar o código que imprime temporariamente a saída para um elemento HTML na página da Web, como um DIV ou um elemento de texto.
