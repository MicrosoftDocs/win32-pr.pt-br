# <a name="playlisthueshift"></a>PLAYLIST. hueShift

O atributo **hueShift** especifica ou recupera a quantidade pela qual o matiz das imagens suspensas é deslocado.

``` syntax
        elementID.hueShift
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**float**) com um valor que varia de 0,0 a 360,0 com um valor padrão de 0,0.

## <a name="remarks"></a>Comentários

Esse atributo altera o valor de matiz das imagens especificadas pelos atributos **dropDownBackgroundImage** e **dropDownImage** se eles tiverem sido especificados e se referirem a imagens bmp de 8 bits.
