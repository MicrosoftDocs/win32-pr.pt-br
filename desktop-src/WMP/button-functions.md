# <a name="button-functions"></a>Funções de botão

Você deseja fornecer ao usuário botões para controlar a seleção e a reprodução de itens de mídia. Usando os botões apropriados, você pode reproduzir, pausar ou parar o item de mídia, mover para itens diferentes na playlist, desligar o som ou fornecer informações sobre o item de mídia que está sendo reproduzindo.

Você não precisa fornecer todas as funções de botão na sua capa, mas a melhor experiência do usuário provavelmente incluirá pelo menos as funções de botão PlayPause, Stop ou PlayPauseStop, Prev e Next.

As seguintes funções de botão estão disponíveis:

-   Reproduzir
-   Pausar
-   Stop
-   Prev
-   Avançar
-   Shuffle
-   Info
-   RepeatPlaylist
-   Mute
-   PlayPause
-   FullScreen
-   WindowsMedia\*
-   PlayPauseStop\*
-   VolumeUp\*
-   VolumeDown\*

\*Disponível para o Windows Media Player 10 Mobile ou posterior.

Para obter mais informações sobre funções de botão, consulte a [seção Botões](buttons.md) da Referência de Capa.
