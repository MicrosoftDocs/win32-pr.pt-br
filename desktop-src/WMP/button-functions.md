# <a name="button-functions"></a>Funções de botão

Você desejará fornecer botões ao usuário para controlar a seleção e a reprodução de itens de mídia. Usando os botões apropriados, você pode reproduzir, pausar ou parar o item de mídia, mover para itens diferentes na lista de reprodução, ativar mudo do som ou fornecer informações sobre o item de mídia que está sendo executado.

Você não precisa fornecer todas as funções de botão em sua capa, mas a melhor experiência de usuário provavelmente incluirá pelo menos as funções de botão PlayPause, Stop ou PlayPauseStop, anterior e Next.

As seguintes funções de botão estão disponíveis:

-   Reproduzir
-   Pausar
-   Stop
-   Prev
-   Avançar
-   Shuffle
-   Info
-   RepeatPlaylist
-   Mute
-   PlayPause
-   FullScreen
-   WindowsMedia\*
-   PlayPauseStop\*
-   VolumeUp\*
-   VolumeDown\*

\*Disponível para Windows Media Player 10 Mobile ou posterior.

Para obter mais informações sobre funções de botão, consulte a seção [Buttons](buttons.md) da referência de Skin.
