# <a name="skin-definition-file-mobile"></a>Arquivo de definição de capa (móvel)

O arquivo de definição de capa é onde você coloca todas as partes juntas. Você define as imagens a serem usadas, especifica os elementos da interface do usuário que deseja que o usuário veja e digita as funções, coordenadas, nomes de arquivo e assim por diante.

As seções a seguir fornecem detalhes sobre como criar o arquivo de definição de capa:

-   [Criando um arquivo de definição de capa](creating-a-skin-definition-file.md)
-   [Editores de texto](text-editors.md)
-   [Regras para cada linha](rules-for-each-line.md)
-   [Elementos necessários](required-elements.md)
