# <a name="equalizersettingscurrentspeakername"></a>EQUALIZERSETTINGS.currentSpeakerName

O atributo **currentSpeakerName** recupera o nome da configuração do alto-falante atual.

``` syntax
        elementID.currentSpeakerName
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** somente leitura que contém um dos valores a seguir.



| Valor           | Descrição                              |
|-----------------|------------------------------------------|
| Fone      | Os alto-falantes atuais são fones de ouvido.     |
| Alto-falantes normais | Os alto-falantes atuais são do tamanho normal. |
| Alto-falantes grandes  | Os alto-falantes atuais são grandes.          |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
