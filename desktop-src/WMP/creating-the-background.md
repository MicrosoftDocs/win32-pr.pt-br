# <a name="creating-the-background"></a>Criando o plano de fundo

Agora você está pronto para criar a imagem de plano de fundo. Mantenha a camada de retângulo visível, mas mantenha-a atrás da camada que você está desenhando e use-a como um guia para criar as imagens para sua capa. Você desejará montar cada elemento em uma ou mais camadas para facilitar a cópia para outro arquivo mais tarde. Por exemplo, o botão PlayPause precisará ser copiado e recolorido no arquivo em pausa, no arquivo desabilitado e no arquivo de região. Quando você move a arte para o arquivo de região, certifique-se de que a arte que você copiar não tenha efeitos de borda complexos, como sombras ou bisels.

Quando terminar de criar a imagem desejada, torne visível todas as camadas que você deseja usar e oculte as outras (como esboço e retângulo). Em seguida, use o comando salvar uma cópia para salvar o trabalho artístico em um arquivo. bmp.

O diagrama a seguir é a imagem de plano de fundo.

![arquivo em segundo plano](images/ceswmbak.png)

Bisels, entalhes e sombras foram usados para criar os efeitos visuais para os vários elementos. Observe que não há nenhum botão para a imagem do volume Thumb. Ele será criado mais tarde.

O Photoshop tem vários efeitos visuais que você pode aplicar a imagens, facilitando a criação de um trabalho artístico mais interessante. Por exemplo, o botão PlayPause foi criado com um círculo simples. Em seguida, um bisel e um alto-relevo com efeito de sombra foi criado. A seta sólida era um triângulo simples criado com três cliques da ferramenta de caneta. Em seguida, a seta recebeu uma aparência em relevo usando um efeito. Ao usar um programa como o Photoshop, você não precisa ser capaz de desenhar. Você pode apenas criar imagens simples e aplicar efeitos. Você também pode importar suas próprias fotos ou trabalho artístico digitalizado.
