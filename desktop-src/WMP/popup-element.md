# <a name="popup-element"></a>Elemento POPUP

O elemento **Popup** fornece uma maneira para os usuários selecionarem itens de uma lista. Esses itens podem ser especificados usando elementos de **Item** como filhos do elemento **Popup** . O elemento **Popup** é idêntico ao elemento **ListBox** , exceto que o valor padrão do atributo **Popup** é true. Quando **popUp** é definido como true, o controle é exibido somente quando o método **show** é chamado e sua largura é definida como a largura do conteúdo (o atributo de **largura** de ambiente é ignorado). O controle é ocultado automaticamente quando o usuário seleciona um item da lista ou quando o método **ignore** é chamado.

> [!Note]  
> Esse elemento requer o Windows Media Player para Windows XP ou posterior.
