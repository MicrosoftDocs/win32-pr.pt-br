# <a name="eventscreenheight"></a>event.screenHeight

O **atributo screenHeight** recupera a altura do tamanho da tela disponível em pixels.

``` syntax
event.screenHeight
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um Número somente **leitura** (**long).**

## <a name="remarks"></a>Comentários

Isso é útil para determinar a quantidade de espaço que há no monitor. Se houver dois monitores, ele calculará o espaço para ambos os monitores.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player série 9 ou posterior<br/> |
