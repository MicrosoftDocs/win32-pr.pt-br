# <a name="columncolumnname"></a>COLUNA. columnName

O atributo **ColumnName** especifica ou recupera um nome de coluna no controle **playlist** .

``` syntax
        elementID.columnName
```

## <a name="possible-values"></a>Valores possíveis

Este atributo é uma **cadeia de caracteres** de leitura/gravação.

## <a name="remarks"></a>Comentários

O nome da coluna é o nome amigável que aparece no cabeçalho da coluna.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
