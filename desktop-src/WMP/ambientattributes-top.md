# <a name="ambientattributestop"></a>AmbientAttributes.top

O atributo **Top** especifica ou recupera a coordenada superior do controle.

``` syntax
        elementID.top
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**) que representa a distância em pixels do controle até a borda superior da **exibição** pai ou da **subexibição**. Ele tem um valor padrão de zero. Números negativos são permitidos; nesse caso, a borda superior da **exibição** ou da **subexibição** corta o controle.
