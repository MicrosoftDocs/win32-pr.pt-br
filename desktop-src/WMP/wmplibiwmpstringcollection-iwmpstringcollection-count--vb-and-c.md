# <a name="iwmpstringcollectioncount-property"></a>Propriedade IWMPStringCollection::count

A **propriedade count** obtém o número de itens na coleção de cadeias de caracteres.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public System.Int32 count {get;}
```


```VB

Public ReadOnly Property count As System.Int32
```





## <a name="property-value"></a>Valor da propriedade

Um **System.Int32 que** é o número de itens na coleção de cadeias de caracteres.

## <a name="remarks"></a>Comentários

Antes de usar essa propriedade, você deve ter acesso de leitura à biblioteca. Para obter mais informações, consulte [Acesso à biblioteca.](library-access.md)
