# <a name="ambientattributesright"></a>Ambienteattributes. à direita

O atributo **Right** especifica ou recupera a coordenada direita do controle.

``` syntax
        elementID.right
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**Long**) que representa a distância em pixels do controle para a borda direita da **exibição** pai ou da **subexibição**. O comportamento para valores negativos ou quando [ambienteattributes. Width](ambientattributes-width.md) não é especificado, é indefinido.
