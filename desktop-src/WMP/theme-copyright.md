# <a name="themecopyright"></a>THEME.copyright

O **atributo de** direitos autorais especifica ou recupera a cadeia de caracteres de direitos autorais para a capa.

``` syntax
theme.copyright
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma cadeia de caracteres de **leitura/gravação** sem valor padrão.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7.0 ou posterior<br/> |
