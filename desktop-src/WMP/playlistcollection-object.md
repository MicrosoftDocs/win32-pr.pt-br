# <a name="playlistcollection-object"></a>Objeto playlistcollection

O objeto **playlistcollection** fornece uma maneira de organizar suas listas de reprodução.

O objeto **playlistcollection** dá suporte aos métodos a seguir.



| Método                                                  | Descrição                                                                                                              |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [getAll](playlistcollection-getall.md)                 | Recupera um objeto [PlaylistArray](playlistarray-object.md) que contém todas as listas de reprodução na biblioteca.                |
| [getByName](playlistcollection-getbyname.md)           | Recupera um objeto [PlaylistArray](playlistarray-object.md) contendo listas de reprodução com o nome especificado, se existir. |
| [importPlaylist](playlistcollection-importplaylist.md) | Adiciona uma lista de reprodução estática à biblioteca.                                                                                   |
| [isDeleted](playlistcollection-isdeleted.md)           | Recupera um valor que indica se a playlist especificada está na pasta itens excluídos.                              |
| [newPlaylist](playlistcollection-newplaylist.md)       | Cria uma nova lista de reprodução na biblioteca.                                                                                   |
| [remove](playlistcollection-remove.md)                 | Remove uma lista de reprodução da biblioteca.                                                                                     |
| [setdeled](playlistcollection-setdeleted.md)         | Move uma lista de reprodução para a pasta itens excluídos.                                                                            |



 

O objeto **playlistcollection** é acessado por meio da propriedade a seguir.



| Objeto                      | Propriedade                                            |
|-----------------------------|-----------------------------------------------------|
| [Jogador](player-object.md) | [playlistcollection](player-playlistcollection.md) |
