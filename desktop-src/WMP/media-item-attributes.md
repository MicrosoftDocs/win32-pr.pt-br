# <a name="media-item-attributes"></a>Atributos de item de mídia

ao examinar a biblioteca em Windows Media Player, você normalmente vê uma grande quantidade de informações sobre um item de mídia. Além do nome de uma faixa de áudio, por exemplo, você provavelmente verá o nome do álbum no qual a faixa está, os nomes do artista e do Composer, o gênero da música e assim por diante.

no modelo de objeto Windows Media Player, esses itens de metadados são chamados de atributos. O modelo de objeto inclui propriedades e métodos que você pode usar para consultar itens de mídia e listas de reprodução para atributos. Você pode exibir valores de atributo em sua interface do usuário ou o seu código pode executar ações com base no valor de um atributo.
