# <a name="externalpluginrunning"></a>External. pluginRunning

> [!Note]  
> Este tópico descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

A propriedade **pluginRunning** recupera um valor que indica se o plug-in do repositório online está em execução.

## <a name="syntax"></a>Syntax

``` syntax
window.external.pluginRunning
```

## <a name="possible-values"></a>Valores possíveis

Esta propriedade é um **booliano** somente leitura. **Verdadeiro** indica que o plug-in está em execução.
