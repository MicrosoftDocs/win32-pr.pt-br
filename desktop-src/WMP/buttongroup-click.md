# <a name="buttongroupclick"></a>BOTÃO. clique

O método **Click** chama o manipulador de eventos **onclick** definido para o **buttonelement** com o índice especificado.

``` syntax
        elementID.click(button)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*Button*
</dt> <dd>

**Número** (**longo**) que contém o índice do **botãoelement** com o manipulador de eventos **onclick** a ser chamado.

</dd> </dl>

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Use esse método para fornecer um meio alternativo de executar o código associado a um botão contido no **botão**.
