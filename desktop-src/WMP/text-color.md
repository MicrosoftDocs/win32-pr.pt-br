# <a name="text-color"></a>Cor do texto

O valor de cor do tipo deve ser especificado. O valor deve ser especificado como três inteiros positivos separados por vírgulas. Os valores representam a quantidade de vermelho, verde e azul para uma cor de bitmap e varia de 0 a 255.
