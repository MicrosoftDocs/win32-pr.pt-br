# <a name="listboxborder"></a>Caixa de listagem. borda

O atributo **Border** especifica ou recupera um valor que indica se o controle da caixa de listagem tem uma borda.

``` syntax
        elementID.border
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um valor **booliano** .



| Valor | Descrição                                  |
|-------|----------------------------------------------|
| true  | Padrão. O controle de caixa de listagem tem uma borda.  |
| false | O controle de caixa de listagem não tem uma borda. |



 

## <a name="remarks"></a>Comentários

A borda aparece como uma borda de baixo pixel em largura de dois pixels em torno do controle caixa de listagem com cores do sistema.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
