# <a name="iwmpmetadatapicturemimetype-property"></a>Propriedade IWMPMetadataPicture:: mimeType

A `mimeType` propriedade Obtém o tipo MIME da imagem representada pelo atributo de metadados.

## <a name="syntax"></a>Syntax


```CSharp
public System.String mimeType {get; set;}
```


```VB

Public ReadOnly Property mimeType As System.String
```





## <a name="property-value"></a>Valor da propriedade

Um **System. String** que é o tipo MIME da imagem.

## <a name="remarks"></a>Comentários

Antes de usar essa propriedade, você deve ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
