# <a name="windows-media-player-sdk-enumeration-types"></a>Windows Media Player Tipos de enumeração do SDK

O Windows Media Player SDK dá suporte aos seguintes tipos de enumeração.



| Tipo de enumeração                                                             | Descrição                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [WMP Ltdaformat](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnformat)                                           | Define os possíveis tipos de CDs para gravação.                                                                         |
| [WMPState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnstate)                                             | Define os possíveis estados operacionais de Windows Media Player à medida que ele ativas em um CD.                                      |
| [WMPDeviceStatus](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpdevicestatus)                                       | Define os valores possíveis para o status atual de um dispositivo.                                                        |
| [WMPFolderScanState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpfolderscanstate)                                 | Define os possíveis estados operacionais de Windows Media Player enquanto monitora pastas de arquivos para conteúdo de mídia digital. |
| [WMPLibraryType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmplibrarytype)                                         | Define os tipos de biblioteca possíveis aos quais Windows Media Player pode se conectar.                                          |
| [WMPOpenState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpopenstate)                                             | Define os possíveis estados operacionais de Windows Media Player à medida que abre um arquivo de mídia digital.                      |
| [WMPPlaylistChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaylistchangeeventtype)                 | Define os tipos de alterações que podem ser feitas em uma playlist.                                                           |
| [WMPPlayState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaystate)                                             | Define os possíveis estados operacionais de Windows Media Player como ele reproduz um arquivo de mídia digital.                      |
| [WMPRipState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpripstate)                                               | Define os possíveis estados operacionais de Windows Media Player à medida que ele desdova um CD.                                       |
| [WMPStringCollectionChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpstringcollectionchangeeventtype) | Define os tipos de alterações que podem ocorrer em uma coleção de cadeias de caracteres.                                                    |
| [WMPSyncState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpsyncstate)                                             | Define os possíveis estados operacionais de Windows Media Player à medida que sincroniza mídia digital com um dispositivo.          |
