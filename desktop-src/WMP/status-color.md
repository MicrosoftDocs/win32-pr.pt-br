# <a name="status-color"></a>Cor do status

O valor de cor do texto de exibição de status deve ser especificado. O valor deve ser três inteiros positivos separados por vírgulas. Os valores representam a quantidade de vermelho, verde e azul para uma cor de texto e variam de 0 a 255.
