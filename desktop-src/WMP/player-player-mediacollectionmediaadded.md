# <a name="playermediacollectionmediaadded-event"></a>Evento Player. MediaCollectionMediaAdded

O evento MediaCollectionMediaAdded ocorre quando um item de mídia é adicionado à biblioteca local.

## <a name="syntax"></a>Sintaxe


```JScript
Player.MediaCollectionMediaAdded(
  pdispMedia
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*pdispMedia* 
</dt> <dd>

Objeto de **mídia** que foi adicionado.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Esse evento não retorna um valor.

## <a name="remarks"></a>Comentários

Esse evento ocorre apenas para a biblioteca local.

**Windows Media Player 10 Mobile:** Não há suporte para esse evento.
