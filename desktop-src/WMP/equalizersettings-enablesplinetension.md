# <a name="equalizersettingsenablesplinetension"></a>EQUALIZERSETTINGS.enableSplineTension

O atributo **enableSplineTension** especifica ou recupera um valor que indica se a tensão de spline está habilitada.

``` syntax
        elementID.enableSplineTension
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação com um valor padrão de true.

## <a name="remarks"></a>Comentários

Habilitar a tensão de spline permite que o usuário ajuste as larguras de banda do equalizador mais suavemente, para que não haja saltos grandes entre os níveis de frequência.
