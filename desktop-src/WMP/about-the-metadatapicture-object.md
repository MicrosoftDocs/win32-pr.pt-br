# <a name="about-the-metadatapicture-object"></a>Sobre o objeto MetadataPicture

O atributo de metadados do **WM/Picture** fornece informações detalhadas sobre imagens de metadados, como arte do álbum. O objeto **MetadataPicture** fornece uma maneira de recuperar os dados individuais relacionados à imagem. Você pode considerar o atributo de metadados do **WM/Picture** como um atributo composto. Quando você usa *mídia*. **getItemInfoByType** para recuperar informações sobre uma imagem de metadados, você recupera um objeto **MetadataPicture** que tem propriedades que contêm os dados individuais.
