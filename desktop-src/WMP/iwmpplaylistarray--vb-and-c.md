# <a name="iwmpplaylistarray-vb-and-c-interface"></a>Interface IWMPPlaylistArray (VB e C#)

Fornece uma propriedade e um método para acessar uma coleção de interfaces **IWMPPlaylist** por número de índice.

A interface **IWMPPlaylistArray** expõe a propriedade a seguir.

## <a name="members"></a>Membros

A interface **IWMPPlaylistArray (VB e C#)** tem estes tipos de membros:

-   [Métodos](#methods)
-   [Propriedades](#properties)

### <a name="methods"></a>Métodos

A interface **IWMPPlaylistArray (VB e C#)** tem esses métodos.



| Método                                                                   | Descrição                                                                                        |
|:-------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**Item**](wmplibiwmpplaylistarray-iwmpplaylistarray-item--vb-and-c.md) | Retorna uma interface **IWMPPlaylist** que representa a playlist no índice especificado.<br/> |



 

### <a name="properties"></a>Propriedades

A interface **IWMPPlaylistArray (VB e C#)** tem essas propriedades.



| Propriedade                                                                              | Tipo de acesso          | Descrição                                                   |
|:--------------------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------|
| [**contar**](wmplibiwmpplaylistarray-iwmpplaylistarray-count--vb-and-c.md)<br/> | Somente leitura<br/> | Obtém o número de listas de reprodução na matriz de playlist<br/> |



 

Obtenha uma interface **IWMPPlaylistArray** usando os métodos a seguir.



| Interface                                                      | Método                                                                                                                                                                               |
|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [IWMPPlaylistCollection](iwmpplaylistcollection--vb-and-c.md) | [**getAll**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-getall--vb-and-c.md), [ **getByName**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-getbyname--vb-and-c.md) |
