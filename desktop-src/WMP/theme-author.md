# <a name="themeauthor"></a>THEME.author

O **atributo** author especifica ou recupera o nome do autor da capa.

``` syntax
theme.author
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma cadeia de caracteres de **leitura/gravação** sem valor padrão.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7.0 ou posterior<br/> |
