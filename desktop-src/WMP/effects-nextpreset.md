# <a name="effectsnextpreset"></a>EFFECTs. nextPreset

O método **nextPreset** exibe a próxima predefinição da visualização atual.

``` syntax
        elementID.nextPreset()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Se a predefinição atual for a última na lista para a visualização atual, a primeira predefinição será tornada atual.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
