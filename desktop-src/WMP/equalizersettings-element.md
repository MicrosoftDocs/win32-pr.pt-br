# <a name="equalizersettings-element"></a>Elemento EQUALIZERSETTINGS

O **elemento EQUALIZERSETTINGS** fornece uma maneira de manipular o equalizador gráfico e outras configurações de áudio Windows Media Player usando os atributos e o método listados aqui.

O **elemento EQUALIZERSETTINGS** dá suporte aos seguintes atributos.



| Atributo                                                          | Descrição                                                                                             |
|--------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [bandas](equalizersettings-bands.md)                               | Recupera o número de faixas de frequência com suporte.                                                      |
| [Ignorar](equalizersettings-bypass.md)                             | Especifica ou recupera um valor que indica se o filtro de equalizador é ignorado no grafo de filtro. |
| [Crossfade](equalizersettings-crossfade.md)                       | Especifica ou recupera um valor que indica se o esmaeçamento cruzado está habilitado.                                |
| [crossFadeWindow](equalizersettings-crossfadewindow.md)           | Especifica ou recupera a quantidade de sobreposição de esmaece cruzado em milissegundos.                                |
| [currentPreset](equalizersettings-currentpreset.md)               | Especifica ou recupera o índice da predefinição atual.                                                 |
| [currentPresetTitle](equalizersettings-currentpresettitle.md)     | Recupera o título da predefinição atual.                                                              |
| [currentSpeakerName](equalizersettings-currentspeakername.md)     | Recupera o nome da configuração atual do locutor.                                                      |
| [enableSplineTension](equalizersettings-enablesplinetension.md)   | Especifica ou recupera um valor que indica se a tensão spline está habilitada ou desabilitada.                |
| [enhancedAudio](equalizersettings-enhancedaudio.md)               | Especifica ou recupera um valor que indica se o áudio aprimorado está ligado.                          |
| [gainLevels](equalizersettings-gainlevels.md)                     | Especifica ou recupera o nível de ganho da faixa correspondente ao índice fornecido.                  |
| [gainLevel1](equalizersettings-gainlevel1.md)                     | Especifica ou recupera o nível de ganho da banda 1.                                                        |
| [gainLevel2](equalizersettings-gainlevel2.md)                     | Especifica ou recupera o nível de ganho da banda 2.                                                        |
| [gainLevel3](equalizersettings-gainlevel3.md)                     | Especifica ou recupera o nível de ganho da banda 3.                                                        |
| [gainLevel4](equalizersettings-gainlevel4.md)                     | Especifica ou recupera o nível de ganho da banda 4.                                                        |
| [gainLevel5](equalizersettings-gainlevel5.md)                     | Especifica ou recupera o nível de ganho da banda 5.                                                        |
| [gainLevel6](equalizersettings-gainlevel6.md)                     | Especifica ou recupera o nível de ganho da banda 6.                                                        |
| [gainLevel7](equalizersettings-gainlevel7.md)                     | Especifica ou recupera o nível de ganho da banda 7.                                                        |
| [gainLevel8](equalizersettings-gainlevel8.md)                     | Especifica ou recupera o nível de ganho da banda 8.                                                        |
| [gainLevel9](equalizersettings-gainlevel9.md)                     | Especifica ou recupera o nível de ganho da banda 9.                                                        |
| [gainLevel10](equalizersettings-gainlevel10.md)                   | Especifica ou recupera o nível de ganho da banda 10.                                                       |
| [Normalização](equalizersettings-normalization.md)               | Especifica ou recupera um valor que indica se a normalização está habilitada.                             |
| [normalizationAverage](equalizersettings-normalizationaverage.md) | Recupera o valor médio de normalização.                                                              |
| [normalizationPeak](equalizersettings-normalizationpeak.md)       | Recupera o valor de normalização de pico.                                                                 |
| [presetCount](equalizersettings-presetcount.md)                   | Recupera o número de predefinições disponíveis.                                                              |
| [speakerSize](equalizersettings-speakersize.md)                   | Especifica ou recupera o índice do tamanho atual do locutor.                                           |
| [splineTension](equalizersettings-splinetension.md)               | Especifica ou recupera a tensão de spline para o controle do equalizador.                                    |
| [truBassLevel](equalizersettings-trubasslevel.md)                 | Especifica ou recupera o nível truBass do SRS.                                                           |
| [wowLevel](equalizersettings-wowlevel.md)                         | Especifica ou recupera o nível de efeito SRS WOW.                                                        |



 

O **elemento EQUALIZERSETTINGS** dá suporte aos métodos a seguir.



| Método                                                 | Descrição                                                          |
|--------------------------------------------------------|----------------------------------------------------------------------|
| [nextPreset](equalizersettings-nextpreset.md)         | Aplica a próxima predefinição de equalizador.                                   |
| [presetTitle](equalizersettings-presettitle.md)       | Recupera o nome da predefinição do equalizador com o índice especificado. |
| [previousPreset](equalizersettings-previouspreset.md) | Aplica a predefinição de equalizador anterior.                               |
| [reset](equalizersettings-reset.md)                   | Redefine os níveis de ganho de todas as faixas para zero decibéis.                |



 

O **elemento EQUALIZERSETTINGS** pode implementar os [ \_ manipuladores de](attribute-onchange.md) eventos de intercâmbio de atributo.
