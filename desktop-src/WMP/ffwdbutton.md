# <a name="ffwdbutton"></a>FFWDBUTTON

Este é um **botão** predefinido com os valores padrão a seguir.


```C++
onclick="jscript:player.controls.fastForward()"
upToolTip="Fast Forward"
cursor="system"
enabled="wmpenabled:player.controls.fastForward"

```



## <a name="remarks"></a>Comentários

Todas as propriedades desse **botão** podem ser substituídas especificando-as explicitamente.
