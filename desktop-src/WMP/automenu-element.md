# <a name="automenu-element"></a>Elemento Automenu

O elemento **Automenu** fornece uma maneira de exibir o **painel de acesso rápido** em uma capa. Esse é um menu que aparece no modo completo do Player quando você clica na seta à direita do botão em **execução** . Ele fornece acesso imediato a arquivos de mídia digital organizados de várias maneiras, como em listas de reprodução ou por álbum, artista ou gênero. Use os atributos **esquerdo** e **superior** para especificar o local no qual o menu deve aparecer quando o método **show** for chamado.

O elemento **Automenu** oferece suporte ao método a seguir.



| Método                    | Descrição                          |
|---------------------------|--------------------------------------|
| [show](automenu-show.md) | Exibe o **painel de acesso rápido**. |



 

O elemento **Automenu** dá suporte aos seguintes atributos de ambiente: esquerda e superior.

> [!Note]  
> este elemento requer o Windows Media Player 9 Series ou posterior.
