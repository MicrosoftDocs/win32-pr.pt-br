# <a name="windows-media-download-packages-deprecated"></a>Pacotes de download do Windows Media (preterido)

Esta página documenta um recurso que pode não estar disponível em versões futuras do Windows Media Player e no SDK do Windows Media Player.

Os pacotes de download do Windows Media combinam bordas do Windows Media Player, informações da playlist e conteúdo multimídia em um único arquivo que pode ser baixado com uma extensão de nome de arquivo. WMD. Um pacote de download do Windows Media pode incluir um álbum inteiro de vídeos de música que também exibe publicidade na forma de marcas gráficas e links para um site varejista de música online.

Os usuários podem baixar um pacote de download do Windows Media de um site simplesmente clicando em um link. Depois que o pacote é baixado no computador do usuário, o Windows Media Player extrai os arquivos dentro do pacote automaticamente e, em seguida, adiciona a lista de reprodução empacotada à caixa suspensa listas de reprodução, adiciona o conteúdo à biblioteca, exibe a capa de borda no painel de **execução** completa do Windows Media Player e, em seguida, reproduz o item inicial na lista de reprodução.

Os pacotes de download do Windows Media oferecem os seguintes benefícios:

-   Um único clique faz downloads, extrai e cataloga vários arquivos para o computador do usuário.
-   O conteúdo é reproduzido imediatamente após ser baixado.
-   As bordas personalizadas podem exibir informações de publicidade e identidade visual.
-   As listas de reprodução empacotadas são catalogadas na biblioteca.
-   O redirecionamento de site pode ocorrer de dentro do Windows Media Player para um site relacionado.
-   Há uma variedade de controles de borda interativos disponíveis.
-   Vários tipos de arquivos de áudio e vídeo têm suporte para empacotamento.

Todos esses benefícios podem ser usados juntos. Por exemplo, um único pacote de download do Windows Media pode dar aos usuários a oportunidade de exibir as letras de músicas à medida que elas são reproduzidas, exibir um vídeo que acompanha as músicas, exibir informações de publicidade sobre o distribuidor de registros, exibir a arte da capa do álbum, visitar um site do ventilador e catalogar o conteúdo na biblioteca.
