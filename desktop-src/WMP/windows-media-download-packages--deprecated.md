# <a name="windows-media-download-packages-deprecated"></a>Windows Pacotes de download de mídia (preterido)

esta página documenta um recurso que pode estar indisponível em versões futuras do Windows Media Player e o SDK do Windows Media Player.

Windows os pacotes de Download de mídia combinam Windows Media Player bordas, informações de playlist e conteúdo multimídia em um único arquivo que pode ser baixado com uma extensão de nome de arquivo. wmd. um pacote de Download de mídia Windows pode incluir um álbum inteiro de vídeos de música que também exibe publicidade na forma de marcas gráficas e links para um site varejista de música online.

os usuários podem baixar um pacote de download de mídia Windows de um site simplesmente clicando em um link. depois que o pacote é baixado para o computador do usuário, Windows Media Player extrai os arquivos dentro do pacote automaticamente e, em seguida, adiciona a lista de reprodução empacotada à caixa suspensa listas de reprodução, adiciona o conteúdo à biblioteca, exibe a capa de borda no painel em **execução** do modo completo Windows Media Player e, em seguida, reproduz o item inicial na lista de reprodução.

Windows Os pacotes de download de mídia fornecem os seguintes benefícios:

-   Um único clique faz downloads, extrai e cataloga vários arquivos para o computador do usuário.
-   O conteúdo é reproduzido imediatamente após ser baixado.
-   As bordas personalizadas podem exibir informações de publicidade e identidade visual.
-   As listas de reprodução empacotadas são catalogadas na biblioteca.
-   o redirecionamento de site pode ocorrer de dentro do Windows Media Player para um site relacionado.
-   Há uma variedade de controles de borda interativos disponíveis.
-   Vários tipos de arquivos de áudio e vídeo têm suporte para empacotamento.

Todos esses benefícios podem ser usados juntos. por exemplo, um único pacote de Download de mídia Windows pode dar aos usuários a oportunidade de exibir as letras de músicas à medida que elas são reproduzidas, exibir um vídeo que acompanha as músicas, exibir informações de publicidade sobre o distribuidor de registros, exibir a arte da capa do álbum, visitar um site do ventilador e catalogar o conteúdo na biblioteca.
