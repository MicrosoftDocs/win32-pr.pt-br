# <a name="playing-digital-media-from-online-store-web-pages"></a>Reproduzindo mídia digital de páginas da Web da loja online

Você pode reproduzir conteúdo de mídia digital de suas páginas da Web da loja online. Basta inserir o controle do Windows Media Player conforme descrito em [usando o controle do Windows Media Player em uma página da Web](using-the-windows-media-player-control-in-a-web-page.md). Quando você faz isso em um painel de tarefas de serviço, o controle incorporado se conecta ao programa do Windows Media Player de forma semelhante ao modo remoto (Confira [comunicação remota do controle do Windows Media Player](remoting-the-windows-media-player-control.md)). Isso significa que os usuários podem usar os controles de transporte do player de modo completo para controlar a reprodução de mídia digital na página da Web.

Recomendamos que você defina **UIMODE** como "nenhum" ou "invisível" ao inserir o controle de jogador em uma página da Web do painel de tarefas de serviço.
