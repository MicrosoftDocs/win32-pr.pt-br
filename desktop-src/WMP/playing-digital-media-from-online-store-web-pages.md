# <a name="playing-digital-media-from-online-store-web-pages"></a>Como tocar mídia digital de páginas da Web da Loja Online

Você pode reproduzir conteúdo de mídia digital em suas páginas da Web da loja online. Basta inserir o controle Windows Media Player como descrito em [Usando o controle Windows Media Player em uma página da Web](using-the-windows-media-player-control-in-a-web-page.md). Quando você faz isso em um painel de tarefas de serviço, o controle inserido se conecta ao programa Windows Media Player de forma semelhante ao modo remoto (consulte Como fazer a [remotação](remoting-the-windows-media-player-control.md)do controle Windows Media Player ). Isso significa que os usuários podem usar os controles de transporte do Player de modo completo para controlar a reprodução de mídia digital na página da Web.

É recomendável definir **uiMode** como "nenhum" ou "invisível" ao incorporar o controle Player em uma página da Web do painel de tarefas de serviço.
