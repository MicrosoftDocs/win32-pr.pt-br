# <a name="designing-the-interface"></a>Projetando a interface

Depois que as funções foram escolhidas, a interface pode ser projetada. Uma interface simples foi escolhida para corresponder à funcionalidade. Os símbolos para os controles foram escolhidos de controles VCR padrão.

A imagem a seguir mostra a aparência da interface.

![interface de exemplo](images/ceswmful.png)

-   **Botão PlayPause ou PlayPauseStop.** O usuário vai tocar mais nisso, portanto, você pode considerar um botão maior. O canto superior direito torna um bom local que o usuário pode encontrar rapidamente. Uma seta sólida é usada para indicar a reprodução e duas barras verticais (não mostradas aqui) serão usadas para indicar pausa.
    > [!Note]  
    > O botão PlayPauseStop só pode ser usado ao criar uma capa para Windows Media Player 10 Mobile ou posterior.

     

-   **Botão Parar.** Para tornar isso fácil de encontrar, ele é colocado no canto superior esquerdo. Um quadrado é usado para indicar a parada. Se você estiver criando uma capa para Windows Media Player 10 Mobile ou posterior, o botão PlayPauseStop já fornece essa funcionalidade.
-   **Botões Next e Prev.** Como eles não serão usados com a mesma frequência, eles são colocados à esquerda. O Próximo está acima do botão Prev porque as pessoas provavelmente querem avançar em uma playlist. Os símbolos de seta dupla são usados porque são semelhantes em função a um controle de avanço rápido.
-   **Barra de faixa de volume.** Isso é colocado na parte inferior da tela e é uma linha simples com um botão de miniatura sobre ela.
-   **Marque a caixa de texto.** Isso é colocado sob o botão PlayPause ou PlayPauseStop para que seja fácil de ver.

Talvez você queira esboçar isso primeiro e experimentar o posicionamento de cada elemento de interface do usuário. O design mostrado aqui foi escolhido para simplificar e facilitar o uso.
