# <a name="designing-the-interface"></a>Criando a interface

Depois que as funções forem escolhidas, a interface poderá ser projetada. Uma interface simples foi escolhida para corresponder à funcionalidade. Os símbolos para os controles foram escolhidos dos controles VCR padrão.

A imagem a seguir mostra como será a aparência da interface.

![interface de exemplo](images/ceswmful.png)

-   **Botão PlayPause ou PlayPauseStop.** O usuário ficará mais tocando, então você pode considerar um botão maior. O canto superior direito faz um bom local que o usuário pode encontrar rapidamente. Uma seta sólida é usada para indicar que a reprodução e duas barras verticais (não mostradas aqui) serão usadas para indicar pausa.
    > [!Note]  
    > O botão PlayPauseStop só pode ser usado ao criar uma capa para o Windows Media Player 10 Mobile ou posterior.

     

-   **Botão parar.** Para facilitar a localização, ele é colocado no canto superior esquerdo. Um quadrado é usado para indicar a parada. Se você estiver criando uma capa para o Windows Media Player 10 Mobile ou posterior, o botão PlayPauseStop já fornecerá essa funcionalidade.
-   **Botões próximo e anterior.** Como eles não serão usados com frequência, eles são colocados à esquerda. A seguir está acima do botão anterior, pois as pessoas provavelmente desejarão avançar em uma lista de reprodução. Os símbolos de seta dupla são usados porque são semelhantes na função a um controle de avanço rápido.
-   **Volume TrackBar.** Isso é colocado na parte inferior da tela e é uma linha simples com um botão de polegar sobre ela.
-   **Caixa de texto do letreiro.** Isso é colocado sob o botão PlayPause ou PlayPauseStop para que seja fácil de ver.

Você pode querer esboçar isso primeiro e experimentar o posicionamento de cada elemento da interface do usuário. O design mostrado aqui foi escolhido para simplificar e facilitar o uso.
