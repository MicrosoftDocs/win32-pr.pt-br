# <a name="customsliderondragend"></a>QUEDISTRLIDER.onDragEnd

O **manipulador de eventos onDragEnd** lida com um evento que ocorre quando o botão esquerdo do mouse é liberado após uma operação de arrastar.

``` syntax
onDragEnd
```
