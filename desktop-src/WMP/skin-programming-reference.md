# <a name="skin-programming-reference"></a>Referência de programação de capa

A referência de programação de capa documenta os seguintes elementos e seus atributos, métodos e eventos associados.

Os elementos e atributos nesta seção exigem o Windows Media Player 7,0 ou posterior, salvo indicação em contrário.



| Elemento                                                  | Descrição                                                                         |
|----------------------------------------------------------|-------------------------------------------------------------------------------------|
| [Atributos de ambiente](ambient-attributes.md)             | Atributos que se aplicam a todos os elementos de capa com exceções indicadas.                   |
| [Manipuladores de eventos de ambiente](ambient-event-handlers.md)     | Manipuladores de eventos que podem ser implementados pela maioria dos elementos de capa.                       |
| [Atributos de evento de ambiente](ambient-event-attributes.md) | Atributos que detalham o estado do Windows Media Player quando um evento é acionado.      |
| [Automenu](automenu-element.md)                         | Fornece uma maneira de exibir o painel de acesso rápido em uma capa.                         |
| [BUTTON](button-element.md)                             | Um botão autônomo.                                                                |
| [BOTÃO](buttonelement-element.md)               | Um botão dentro de um grupo de botões.                                                     |
| [BUTTONGROUP](buttongroup-element.md)                   | Um grupo de elementos Button.                                                         |
| [COLUMN](column-element.md)                             | Representa uma coluna em um controle de lista de reprodução.                                      |
| [CONTROLES](controls-element.md)                         | Fornece acesso ao objeto [Controls](controls-object.md) de dentro de uma capa.   |
| [CUSTOMSLIDER](customslider-element.md)                 | Um controle deslizante personalizável.                                                      |
| [EDIÇÃO](editbox-element.md)                           | Fornece uma maneira para os usuários digitarem texto em uma capa.                               |
| [Effect](effects-element.md)                           | Um elemento que contém e controla uma coleção de efeitos.                      |
| [EQUALIZERSETTINGS](equalizersettings-element.md)       | Um elemento que permite a manipulação do equalizador gráfico.                          |
| [ITEM](item-element.md)                                 | Representa um item em um controle de caixa de listagem ou de pop-up.                                 |
| [CAIXAS](listbox-element.md)                           | Fornece uma maneira para os usuários selecionarem itens de uma lista.                               |
| [PLAYER](player-element.md)                             | Fornece acesso ao objeto do [Player](player-object.md) de dentro de uma capa.       |
| [7.1](playlist-element.md)                         | Um elemento para controlar a aparência de uma lista de reprodução em uma capa.              |
| [POP-up](popup-element.md)                               | Fornece uma maneira para os usuários selecionarem itens de uma lista.                               |
| [PROGRESSBAR](progressbar-element.md)                   | Fornece uma maneira de exibir informações de progresso em um controle horizontal ou vertical. |
| [CONFIGURAÇÕES](settings-element.md)                         | Fornece acesso ao objeto de [configurações](settings-object.md) de dentro de uma capa.   |
| [CLASSIFICAÇÃO](slider-element.md)                             | Um controle deslizante.                                                                   |
| [SUBEXIBIÇÃO](subview-element.md)                           | Subseções dentro de uma exibição que pode ser movida ou ocultada.                              |
| [TEXTO](text-element.md)                                 | Um controle que contém texto.                                                          |
| [-](theme-element.md)                               | O elemento principal que identifica um arquivo de capa.                                           |
| [MONITOR](video-element.md)                               | Um elemento para especificar a aparência de uma janela de vídeo.                         |
| [VIDEOSETTINGS](videosettings-element.md)               | Um elemento que permite o controle de várias configurações de vídeo.                              |
| [VIEW](view-element.md)                                 | Especifica a aparência da interface do usuário para cada categoria de mídia.       |
| [Diversos](miscellaneous.md)                       | Atributos especializados e outros tópicos diversos para uso na criação de capas.    |
