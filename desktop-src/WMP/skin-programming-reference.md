# <a name="skin-programming-reference"></a>Referência de programação de capa

A Referência de Programação de Capa documenta os elementos a seguir e seus atributos, métodos e eventos associados.

Os elementos e atributos nesta seção exigem Windows Media Player 7.0 ou posterior, a menos que seja notado de outra forma.



| Elemento                                                  | Descrição                                                                         |
|----------------------------------------------------------|-------------------------------------------------------------------------------------|
| [Atributos de ambiente](ambient-attributes.md)             | Atributos que se aplicam a todos os elementos de capa com exceções notadas.                   |
| [Manipuladores de eventos de ambiente](ambient-event-handlers.md)     | Manipuladores de eventos que podem ser implementados pela maioria dos elementos de capa.                       |
| [Atributos de evento de ambiente](ambient-event-attributes.md) | Atributos que detalham o estado de Windows Media Player quando um evento é acionado.      |
| [AUTOMENU](automenu-element.md)                         | Fornece uma maneira de exibir o Guia Painel de Acesso em uma capa.                         |
| [Botão](button-element.md)                             | Um botão autônomo.                                                                |
| [BUTTONELEMENT](buttonelement-element.md)               | Um botão dentro de um grupo de botões.                                                     |
| [Buttongroup](buttongroup-element.md)                   | Um grupo de elementos de botão.                                                         |
| [COLUMN](column-element.md)                             | Representa uma coluna dentro de um controle de playlist.                                      |
| [Controles](controls-element.md)                         | Fornece acesso ao [objeto Controls](controls-object.md) de dentro de uma capa.   |
| [SÃOLIDER](customslider-element.md)                 | Um controle deslizante personalizável.                                                      |
| [Editbox](editbox-element.md)                           | Fornece uma maneira para os usuários inserirem texto dentro de uma capa.                               |
| [Efeitos](effects-element.md)                           | Um elemento que contém e controla uma coleção de efeitos.                      |
| [EQUALIZERSETTINGS](equalizersettings-element.md)       | Um elemento que permite a manipulação do equalizador gráfico.                          |
| [Item](item-element.md)                                 | Representa um item em uma caixa de listagem ou controle pop-up.                                 |
| [Listbox](listbox-element.md)                           | Fornece uma maneira para os usuários selecionarem itens de uma lista.                               |
| [Jogador](player-element.md)                             | Fornece acesso ao [objeto Player](player-object.md) de dentro de uma capa.       |
| [Lista](playlist-element.md)                         | Um elemento para controlar a aparência de uma playlist dentro de uma capa.              |
| [Popup](popup-element.md)                               | Fornece uma maneira para os usuários selecionarem itens de uma lista.                               |
| [Progressbar](progressbar-element.md)                   | Fornece uma maneira de exibir informações de progresso em um controle horizontal ou vertical. |
| [CONFIGURAÇÕES](settings-element.md)                         | Fornece acesso ao objeto [Configurações](settings-object.md) de dentro de uma capa.   |
| [Controle deslizante](slider-element.md)                             | Um controle deslizante.                                                                   |
| [SUBVIEW](subview-element.md)                           | Subseções dentro de uma exibição que podem ser movidas ou ocultadas.                              |
| [Texto](text-element.md)                                 | Um controle que contém texto.                                                          |
| [Tema](theme-element.md)                               | O elemento principal que identifica um arquivo de capa.                                           |
| [Vídeo](video-element.md)                               | Um elemento para especificar a aparência de uma janela de vídeo.                         |
| [VIDEOSETTINGS](videosettings-element.md)               | Um elemento que permite o controle de várias configurações de vídeo.                              |
| [VIEW](view-element.md)                                 | Especifica a aparência da interface do usuário para cada categoria de mídia.       |
| [Diversos](miscellaneous.md)                       | Atributos especializados e outros tópicos diversos para uso na criação de capas.    |
