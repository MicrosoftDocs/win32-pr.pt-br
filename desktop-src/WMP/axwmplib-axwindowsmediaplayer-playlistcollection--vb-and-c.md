# <a name="axwindowsmediaplayerplaylistcollection-property"></a>Propriedade AxWindowsMediaPlayer. playlistcollection

A propriedade playlistcollection Obtém uma interface **IWMPPlaylistCollection** .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public IWMPPlaylistCollection playlistCollection {get;}
```


```VB

Public ReadOnly Property playlistCollection As IWMPPlaylistCollection
```





## <a name="property-value"></a>Valor da propriedade

O WMPLib. Interface **IWMPPlaylistCollection** .

## <a name="remarks"></a>Comentários

Para recuperar o valor dessa propriedade, é necessário ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
