# <a name="playlistisidentical-method"></a>Método playlist. isidêntico

O método **isidêntico** recupera um valor que indica se o objeto de **playlist** fornecido é idêntico ao atual.

## <a name="syntax"></a>Sintaxe


```JScript
bRetVal = Playlist.isIdentical(
  playlist
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*lista de reprodução* \[ no\]
</dt> <dd>

Objeto de **playlist** a ser comparado com o atual.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Esse método retorna um **valor booleano**.

| Requisito | Valor |
|--------------------|------------------------------------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior.<br/>                              |
