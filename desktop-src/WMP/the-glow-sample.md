# <a name="the-glow-sample"></a>O exemplo de brilho

Esta seção descreve como criar uma visualização simples chamada brilho. Três predefinições são usadas: vermelho, verde e azul. O assistente de visualização personalizada foi executado e o nome brilho foi escolhido.

O brilho simplesmente muda a cor no tempo para a música. Os movimentos de brilho porque a potência de áudio da onda às vezes cruza zero no momento do instantâneo. Este exemplo é escolhido porque é um dos exemplos mais simples possíveis.
