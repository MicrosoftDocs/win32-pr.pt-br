# <a name="installing-visualizations"></a>Instalando visualizações

Você deve fornecer um processo de instalação para o usuário da sua visualização. Você também deve fornecer um processo de desinstalação para o usuário. a versão atual do Windows Media Player não instala visualizações da interface do usuário.

## <a name="installing-to-the-visualization-folder"></a>Instalando na pasta de visualização

é recomendável que você instale todas as visualizações na subpasta visualizações da pasta em que o Windows Media Player está instalado.

## <a name="registering-your-visualization"></a>Registrando sua visualização

As visualizações são DLLs COM e seguem todas as regras normais de instalação e remoção. Você pode usar regsvr32.exe ou outras ferramentas de instalação para registrar sua visualização.
