# <a name="itemvalue"></a>ITEM. Value

O atributo **Value** especifica ou recupera o texto que é exibido para uma caixa de listagem ou um item pop-up.

``` syntax
        elementID.value
```

## <a name="possible-values"></a>Valores possíveis

Este atributo é uma **cadeia de caracteres** de leitura/gravação.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
