<span data-ttu-id="31b44-110">Para o usuário, os plug-ins do DSP são recursos complementares que, de alguma forma, alteram o modo como o usuário passa o áudio ou vídeo que o Windows Media Player está reproduzindo.</span><span class="sxs-lookup"><span data-stu-id="31b44-110">To the user, DSP plug-ins are add-on features that somehow change the way the user experiences the audio or video that Windows Media Player is playing.</span></span> <span data-ttu-id="31b44-111">O resultado de um plug-in do DSP pode ser algo óbvio, como as cores na imagem de vídeo sendo revertida, de modo que parece um negativo fotográfico ou algo muito sutil, como um efeito de reverberação leve no áudio para adicionar uma sensação de spaciousness ao som.</span><span class="sxs-lookup"><span data-stu-id="31b44-111">The result of a DSP plug-in could be something obvious, like the colors in the video image being reversed so it looks like a photographic negative, or something very subtle, like a light reverb effect in the audio to add a feeling of spaciousness to the sound.</span></span>
