# <a name="dsp-plug-in-user-overview"></a>Visão geral do usuário de plug-in do DSP

para o usuário, os plug-ins do DSP são recursos complementares que, de alguma forma, alteram a maneira como o usuário experimenta o áudio ou vídeo que Windows Media Player está tocando. O resultado de um plug-in do DSP pode ser algo óbvio, como as cores na imagem de vídeo sendo revertida, de modo que parece um negativo fotográfico ou algo muito sutil, como um efeito de reverberação leve no áudio para adicionar uma sensação de spaciousness ao som.

Os usuários podem instalar e desinstalar plug-ins DSP usando a guia **plug-ins** da caixa de diálogo opções no Windows Media Player. Os usuários podem habilitar e desabilitar plug-ins instalados usando as caixas de seleção fornecidas na guia **plug-ins** . plug-ins do DSP podem implementar uma página de propriedades, que os usuários também podem acessar na guia **plug-** ins. Uma página de propriedades fornece ao usuário uma maneira de alterar as configurações que afetam a maneira como um plug-in do DSP funciona. Por exemplo, uma página de propriedades pode incluir um controle deslizante que permite ao usuário alterar o tempo de decaimento de um efeito de reverberação.
