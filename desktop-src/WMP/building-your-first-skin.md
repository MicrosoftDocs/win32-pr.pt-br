# <a name="building-your-first-skin"></a>Criando sua primeira capa

conforme explicado mais detalhadamente [sobre as capas](about-skins.md), uma capa é definida em um arquivo de definição de capa e requer um ou mais arquivos de arte de suporte, arquivos do Microsoft JScript e arquivos de mídia digital. Este tutorial para uma capa muito simples requer apenas o arquivo de definição de capa e três arquivos de arte. Você passará por quatro etapas descritas nos tópicos a seguir:

-   [Criando o arquivo de arte principal](creating-the-primary-art-file.md)
-   [Criando o arquivo de mapeamento](creating-the-mapping-file.md)
-   [Criando a imagem em foco](creating-the-hover-image.md)
-   [Criando o arquivo de definição de capa](creating-the-skin-definition-file.md)
