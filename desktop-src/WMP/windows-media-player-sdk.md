# <a name="windows-media-player-sdk"></a>Windows Media Player SDK

esta documentação descreve o SDK (Software Development Kit) do Microsoft Windows Media Player. o SDK do Windows Media Player é um dos componentes do SDK do Microsoft Windows. outros componentes de mídia incluem o sdk do Microsoft Media Foundation, o microsoft Windows media Format sdk e o sdk do microsoft Windows Media Services.

o SDK do Windows Media Player documenta as tecnologias de programação que podem ser usadas para estender os recursos do Windows Media Player e Windows Media Player Mobile. Essas tecnologias estão documentadas nas seguintes seções:



| Seção                                                                      | Descrição                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [sobre o SDK do Windows Media Player](about-the-windows-media-player-sdk.md) | Esta seção fornece detalhes sobre como encontrar informações específicas no SDK. Ele inclui uma seção sobre novos recursos e informações sobre como usar os exemplos incluídos no SDK.                                                                                                                                     |
| [Windows Media Player Modelo de objeto](windows-media-player-object-model.md)   | o controle de Windows Media Player da microsoft é um controle de ActiveX da microsoft usado para adicionar recursos de reprodução de mídia digital a páginas da web. Ele fornece uma interface de programação para renderizar fluxos e arquivos de mídia digital.                                                                                                     |
| [Windows Media Player Skin](windows-media-player-skins.md)                 | As capas são uma tecnologia baseada em XML usada para personalizar a interface do usuário do Windows Media Player. você também pode usar Windows Media Player aparências móveis para personalizar a interface do usuário do Windows Media Player mobile.                                                                                                                |
| [Windows Media Player Plug-ins](windows-media-player-plug-ins.md)           | Plug-ins são objetos que estendem Windows Media Player funcionalidade de várias maneiras. Os tipos de plug-in incluem visualizações personalizadas, plug-ins de interface do usuário, plug-ins de DSP e plug-ins de renderização. Windows Media Player O suporte ao plug-in móvel também é descrito nesta seção.                                              |
| [Windows Metaarquivos de mídia](windows-media-metafiles.md)                       | Windows Os metaarquivos de mídia são documentos XML que fornecem informações sobre um item de mídia e sua apresentação. Os metaarquivos podem ser usados para organizar itens de mídia em listas de reprodução que podem incluir funcionalidade para comutação de fluxo contínuo, inserção de anúncios e outros recursos.                                                          |
| [Windows Listas de reprodução de mídia](windows-media-playlists.md)                       | As listas de reprodução são arquivos que usam elementos XML para definir uma lista de reprodução dinâmica "inteligente" de itens de mídia ou um conjunto estático de itens de mídia.                                                                                                                                                                                              |
| [Windows Media Player Lojas online](windows-media-player-online-stores.md) | o Windows Media Player fornece funcionalidade que permite que os provedores de conteúdo de mídia digital integrem seus serviços com o Windows Media Player. A integração entre o Player e um repositório de mídia digital online permite ao usuário localizar conteúdo, baixar e gerenciar arquivos, reproduzir conteúdo e copiar conteúdo para CDs ou dispositivos. |
| [Windows Media Player](windows-media-player.md)                             | alguns recursos do SDK se aplicam ao Windows Media Player, ao controle de ActiveX de Windows Media Player e Windows Media Player dispositivos móveis. Esta seção fornece informações sobre esses recursos.                                                                                                                                        |
| [Glossário](glossary.md)                                                     | Esta seção contém definições de termos usados em todo o SDK.                                                                                                                                                                                                                                                               |



 

> [!Note]  
> a instalação do SDK do Windows não instala Windows Media Player ou Windows Media Player Mobile. você deve ter Windows Media Player instalado para usar o material neste SDK. Windows Media Player o mobile é uma parte do Windows mobile, que está disponível somente em um Pocket PC ou Smartphone fornecido por um fabricante de dispositivo ou operadora móvel.
