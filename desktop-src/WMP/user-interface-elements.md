# <a name="user-interface-elements"></a>Elementos da interface do usuário

Embora você possa escolher entre uma ampla gama de funcionalidades para controlar mídia digital, listas de reprodução e volume, cabe a você fornecer ao usuário de sua capa uma maneira de interagir com essa funcionalidade. Por exemplo, se você quiser que o usuário inicie a reprodução de um item de mídia, deverá fornecer um elemento de botão de interface do usuário que fará com que o item de mídia seja tocado quando o usuário tocar na imagem do seu botão.

Nenhum elemento é necessário, mas se você não tiver alguns botões para reproduzir e parar o item de mídia, não ocorrerá muito. Também é uma boa ideia fornecer um controle de volume.

Os elementos da interface do usuário correspondem aos tipos de funcionalidade usados para controlar a seleção e a reprodução de mídia digital.

Os elementos a seguir são fornecidos.



| Elemento                    | Descrição                                                                                                                                                                                                                                                             |
|----------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Botões](buttons.md)     | Muitos tipos de botão são fornecidos. Os botões podem ser usados para ativar ou desativar funções ou alternar entre duas funções.                                                                                                                                                        |
| [Trackbars](trackbars.md) | Você pode fornecer controle linear por meio de trackbars. Você desejará usar o trackbars para alterar o volume ou a posição de reprodução em um item de mídia.                                                                                                                         |
| [Text](text.md)           | As caixas de texto podem ser usadas para exibir texto para o usuário. Um bom uso de uma caixa de texto é exibir o nome da música em execução no momento.                                                                                                                                      |
| [Marquee](marquee.md)     | Um letreiro é uma caixa de texto de rolagem que você pode usar para exibir texto de mais de uma categoria de texto. Por exemplo, você pode exibir a lista de reprodução, o número de faixas e a reprodução de números de faixa. Use um letreiro quando você não tiver muito espaço e tiver muito a ser exibido. |
| [Status](status.md)       | Você pode fornecer uma exibição de status que pode ser usada para exibir automaticamente mensagens de texto que informam ao usuário sobre o estado atual do Windows Media Player. Por exemplo, a exibição de status informará ao usuário quando um item de mídia estiver em buffer.                     |
| [Vídeo](video.md)         | Você pode fornecer uma área de exibição de vídeo retangular em sua capa se quiser reproduzir vídeo.                                                                                                                                                                                |
| [Classificações](ratings.md)     | Você pode exibir a classificação associada a qualquer conteúdo baseado em áudio do Windows Media que esteja sendo executado no momento.                                                                                                                                                             |
