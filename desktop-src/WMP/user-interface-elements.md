# <a name="user-interface-elements"></a>Elementos da interface do usuário

Embora você possa escolher entre uma ampla variedade de funcionalidades para controlar mídia digital, playlists e volume, é responsabilidade fornecer ao usuário da sua capa uma maneira de interagir com essa funcionalidade. Por exemplo, se você quiser que o usuário comece a reproduzir um item de mídia, deverá fornecer um elemento de botão de interface do usuário que fará com que o item de mídia seja reproduzir quando o usuário tocar na imagem do botão.

Nenhum elemento é necessário, mas se você não tiver alguns botões para reproduzir e parar o item de mídia, pouca coisa acontecerá. Também é uma boa ideia fornecer um controle de volume.

Os elementos de interface do usuário correspondem aos tipos de funcionalidade usados para controlar a seleção e a reprodução de mídia digital.

Os elementos a seguir são fornecidos.



| Elemento                    | Descrição                                                                                                                                                                                                                                                             |
|----------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Botões](buttons.md)     | Muitos tipos de botão são fornecidos. Os botões podem ser usados para ativar ou desativar funções ou alternar entre duas funções.                                                                                                                                                        |
| [Trackbars](trackbars.md) | Você pode fornecer controle linear por meio de barras de controle. Você deve usar barras de controle para alterar o volume ou a posição de reprodução em um item de mídia.                                                                                                                         |
| [Text](text.md)           | As caixas de texto podem ser usadas para exibir texto para o usuário. Um bom uso de uma caixa de texto é exibir o nome da música atualmente em reprodução.                                                                                                                                      |
| [Marquee](marquee.md)     | Um letreiro é uma caixa de texto de rolagem que você pode usar para exibir texto de mais de uma categoria de texto. Por exemplo, você pode exibir a playlist, o número de faixas e o número da faixa em reprodução. Use um letreiro quando você não tiver muito espaço e tiver muito a exibir. |
| [Status](status.md)       | Você pode fornecer uma exibição de status que pode ser usada para exibir automaticamente mensagens de texto que informam ao usuário sobre o estado atual do Windows Media Player. Por exemplo, a exibição de status informará o usuário quando um item de mídia estiver em buffer.                     |
| [Vídeo](video.md)         | Você pode fornecer uma área de exibição de vídeo retangular na sua capa se quiser reproduzir vídeo.                                                                                                                                                                                |
| [Classificações](ratings.md)     | Você pode exibir a classificação associada a qualquer Windows conteúdo baseado em áudio de mídia que está sendo exibido no momento.                                                                                                                                                             |
