# <a name="viewontimer"></a>Exibir. OnTimer

O evento **OnTimer** manipula eventos de temporizador.

``` syntax
ontimer
```

## <a name="remarks"></a>Comentários

O atributo **IntervaloDoCronômetro** especifica o intervalo no qual os eventos de temporizador serão acionados.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
