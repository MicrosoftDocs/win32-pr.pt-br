# <a name="durationtext"></a>DURATIONTEXT

Este é um elemento de **texto** predefinido com os seguintes valores padrão.

``` syntax
value="wmpprop:player.currentMedia.DurationString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Comentários

Isso criará um elemento de **texto** que exibirá a duração da mídia que está sendo reproduzida. Todas as propriedades desse elemento de **texto** podem ser substituídas especificando-as explicitamente.
