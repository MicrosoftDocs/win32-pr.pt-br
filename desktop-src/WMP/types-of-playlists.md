# <a name="types-of-playlists"></a>Tipos de playlists

Windows As playlists de metadados de mídia, simplesmente conhecidas como playlists neste documento, têm extensões .chrome, .wvx ou .asx. Listas de reprodução com essas extensões são usadas para acessar Windows mídia com extensões de nome de arquivo .wma, .wmv e .asf, respectivamente.
