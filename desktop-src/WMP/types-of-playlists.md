# <a name="types-of-playlists"></a>Tipos de listas de reprodução

As listas de reprodução do metarquivo do Windows Media, simplesmente chamadas de listas de reprodução neste documento, têm extensões. Wax,. wvx ou. asx. As listas de reprodução com essas extensões são usadas para acessar arquivos de mídia do Windows com extensões de nome de arquivo. WMA,. wmv e. ASF, respectivamente.
