# <a name="viewminwidth"></a>VIEW.minWidth

O **atributo minWidth** especifica ou recupera a largura mínima em pixels do **VIEW** ao reizing.

``` syntax
        elementID.minWidth
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um  número de leitura/gravação (**longo**) com um valor de zero ou maior. Ele tem um valor padrão de zero, o que significa que não há nenhuma restrição na largura mínima do **VIEW.**

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7.0 ou posterior<br/> |
