# <a name="pixelaspectratiox"></a>PixelAspectRatioX

O atributo **PixelAspectRatioX** é a parte horizontal da taxa de proporção de um vídeo.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

Para um vídeo com uma taxa de proporção de 16:9, **PixelAspectRatioX** seria igual a "16".
