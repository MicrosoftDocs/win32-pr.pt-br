# <a name="playerplaylistcollectionchange-event"></a>Evento Player. PlaylistCollectionChange

O evento **PlaylistCollectionChange** ocorre quando algo é alterado na coleção de playlist.

## <a name="syntax"></a>Sintaxe


```JScript
Player.PlaylistCollectionChange()
```



## <a name="parameters"></a>Parâmetros

Este evento não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse evento não retorna um valor.

## <a name="remarks"></a>Comentários

**Windows Media Player 10 Mobile:** Não há suporte para esse evento.
