# <a name="iwmpstringcollection-vb-and-c-interface"></a>Interface IWMPStringCollection (VB e C#)

Fornece uma propriedade e um método para acessar uma coleção de cadeias de caracteres por número de índice.

A interface **IWMPStringCollection** expõe a propriedade a seguir.

## <a name="members"></a>Membros

A interface **IWMPStringCollection (VB e C#)** tem estes tipos de membros:

-   [Métodos](#methods)
-   [Propriedades](#properties)

### <a name="methods"></a>Métodos

A interface **IWMPStringCollection (VB e C#)** tem esses métodos.



| Método                                                                         | Descrição                                           |
|:-------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Item**](wmplibiwmpstringcollection-iwmpstringcollection-item--vb-and-c.md) | Retorna a cadeia de caracteres no índice especificado.<br/> |



 

### <a name="properties"></a>Propriedades

A interface **IWMPStringCollection (VB e C#)** tem essas propriedades.



| Propriedade                                                                                    | Tipo de acesso          | Descrição                                                   |
|:--------------------------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------|
| [**count**](wmplibiwmpstringcollection-iwmpstringcollection-count--vb-and-c.md)<br/> | Somente leitura<br/> | Obtém o número de itens na coleção de cadeias de caracteres.<br/> |



 

Obtenha uma interface **IWMPStringCollection** usando os métodos a seguir.



| Objeto                                                   | Propriedade                                                                                 |
|----------------------------------------------------------|------------------------------------------------------------------------------------------|
| [IWMPMediaCollection](iwmpmediacollection--vb-and-c.md) | [**getAttributeStringCollection**](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpmediacollection-getattributestringcollection) |
