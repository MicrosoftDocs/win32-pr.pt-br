# <a name="playermediacollection"></a>Player. mediacollection

A propriedade **mediacollection** recupera o objeto **mediacollection** .

## <a name="syntax"></a>Syntax

*Player* . **mediacollection**

## <a name="possible-values"></a>Valores possíveis

Esta propriedade é um objeto **mediacollection** somente leitura.

## <a name="remarks"></a>Comentários

Para recuperar o valor dessa propriedade, é necessário ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).

## <a name="requirements"></a>Requisitos

|--------------------|------------------------------------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior.<br/>                              |
