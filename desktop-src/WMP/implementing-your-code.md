# <a name="implementing-your-code"></a>Implementando seu código

Depois de concluir o processo de compilação, você provavelmente vai querer melhorar o código gerado pelo assistente. Caso contrário, sua visualização será muito parecida com as barras. O que você precisa fazer é alterar a implementação da função **render** e, possivelmente, a função **RenderWindowed** .

As seções a seguir fornecem detalhes sobre como modificar o código de exemplo gerado pelo assistente de plug-in:

-   [Implementando renderização](implementing-render.md)
-   [Implementando outras funções](implementing-other-functions.md)
-   [Predefinições](presets.md)
