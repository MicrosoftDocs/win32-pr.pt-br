# <a name="dsp-plug-in-enumeration-types"></a>Tipos de enumeração de plug-in do DSP

O SDK do Windows Media Player implementa o seguinte tipo de enumeração para criar plug-ins DSP.



| Tipo de enumeração                                        | Descrição                                                                                    |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [WMPPlugin \_ Caps](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpplugin_caps)                   | Usado com **IWMPPlugin:: GetCaps** para indicar se o plug-in pode ser convertido entre os formatos. |
| [WMPServices \_ streamstate](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpservices_streamstate) | Indica se o fluxo está parado, em pausa ou em execução no momento.                     |
