# <a name="minimizebutton"></a>MINIMIZEBUTTON

Este é um botão predefinido com os valores padrão a seguir.

``` syntax
onclick="jscript:view.minimize();"
upToolTip="Minimize"
```

## <a name="remarks"></a>Comentários

Isso criará um controle de **botão** que minimizará o Player. As dicas de ferramenta são localizadas. Todas as propriedades desse **botão** podem ser substituídas especificando-as explicitamente.
