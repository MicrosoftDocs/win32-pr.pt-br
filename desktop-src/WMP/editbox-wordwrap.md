# <a name="editboxwordwrap"></a>Paminhar. wordWrap

O atributo **WordWrap** especifica ou recupera um valor que indica se a quebra automática de texto está habilitada.

``` syntax
        elementID.wordWrap
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação com um valor padrão de true.

## <a name="remarks"></a>Comentários

Esse atributo é útil somente quando **editstyle** é definido como "Multiline".

Se a quebra automática de palavra estiver desabilitada e o texto não couber no controle, o texto será truncado.
