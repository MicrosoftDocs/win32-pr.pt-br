# <a name="button-placement"></a>Posicionamento do botão

Embora você possa desenhar botões para qualquer forma, eles são tratados internamente como tendo um limite externo retangular. Por esse motivo, evite sobreposição de coordenadas de botão porque podem ocorrer resultados inesperados.
