# <a name="ondblclick"></a>AoClicarDuasVezes

O manipulador de eventos **AoClicarDuasVezes** manipula um evento que ocorre quando o usuário clica duas vezes no elemento.

``` syntax
ondblclick
```

## <a name="remarks"></a>Comentários

Esse manipulador de eventos não é aplicável ao elemento **playlist** , o elemento **Popup** , o elemento **Video** quando o *vídeo*. **sem janela** é falso ou o elemento **Effects** quando há *efeitos*. a **janela** é verdadeira.
