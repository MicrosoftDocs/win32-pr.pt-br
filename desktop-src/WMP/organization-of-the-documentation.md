# <a name="organization-of-the-documentation"></a>Organização da documentação

O Windows Media Player SDK documenta uma ampla variedade de tecnologias relacionadas a Windows Media Player. Para facilitar a localização das informações necessárias, a documentação é dividida em seções que ajudam você a localizar rapidamente tópicos sobre um recurso ou tecnologia específico.

Quando apropriado, as seções principais incluem três subseções: Sobre, Guia de Programação e Referência. A seção Sobre descreve o recurso ou tecnologia específico para ajudá-lo a entender como usá-lo. A seção Guia de Programação adiciona tutoriais que o aprofundam na tecnologia com instruções passo a passo. A seção Referência detalha as interfaces, objetos, métodos, propriedades, eventos, elementos e atributos disponíveis para seu uso.
