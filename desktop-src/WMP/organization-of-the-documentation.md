# <a name="organization-of-the-documentation"></a>Organização da documentação

O SDK do Windows Media Player documenta uma ampla variedade de tecnologias relacionadas ao Windows Media Player. Para facilitar a localização das informações necessárias, a documentação é dividida em seções que ajudam você a localizar rapidamente tópicos sobre um determinado recurso ou tecnologia.

Quando apropriado, as seções principais incluem três subseções: sobre, guia de programação e referência. A seção sobre descreve o recurso ou a tecnologia específica para ajudá-lo a entender como você pode usá-lo. A seção guia de programação adiciona tutoriais que levam você mais profundamente à tecnologia com instruções passo a passo. A seção de referência detalha as interfaces, objetos, métodos, propriedades, eventos, elementos e atributos que estão disponíveis para seu uso.
