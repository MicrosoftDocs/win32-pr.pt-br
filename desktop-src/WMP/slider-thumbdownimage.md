# <a name="sliderthumbdownimage"></a>SLIDER. thumbDownImage

O atributo **thumbDownImage** especifica ou recupera a imagem que representa o estado inoperante do Thumb.

``` syntax
        elementID.thumbDownImage
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém o nome de um arquivo de imagem.

## <a name="remarks"></a>Comentários

O **thumbDownImage** é opcional e é a imagem usada para representar o estado inoperante do **thumbImage**. Se um **thumbDownImage** não for fornecido, o controle deslizante usará o **thumbImage**.

Os formatos com suporte são BMP, JPG, PNG e GIF (não incluindo GIFs animados).
