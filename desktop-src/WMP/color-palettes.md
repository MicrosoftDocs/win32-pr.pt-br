# <a name="color-palettes-windows-media-player-sdk"></a>Paletas de cores (SDK do Windows Media Player)

Dispositivos diferentes têm paletas de cores diferentes. Provavelmente, você desejará criar diferentes capas para cada paleta de cores. As seguintes paletas de cores são recomendadas:

-   16 bits
-   8 bits
-   escala de cinza de 2 bits
