# <a name="color-palettes-windows-media-player-sdk"></a>Paletas de cores (Windows Media Player SDK)

Dispositivos diferentes têm paletas de cores diferentes. Você provavelmente vai querer criar capas diferentes para cada paleta de cores. As seguintes paletas de cores são recomendadas:

-   16 bits
-   8 bits
-   Escala de cinza de 2 bits
