# <a name="columncolumnwidth"></a>COLUNA. columnWidth

O atributo **ColumnWidth** especifica ou recupera uma largura de coluna no controle **playlist** .

``` syntax
        elementID.columnWidth
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**Long**) que representa a largura da coluna em pixels.

## <a name="remarks"></a>Comentários

A propriedade **columnResizeMode** deve ser definida como "fixed" para que essa propriedade funcione.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
