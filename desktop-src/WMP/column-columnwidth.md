# <a name="columncolumnwidth"></a>COLUMN.columnWidth

O **atributo columnWidth** especifica ou recupera uma largura de coluna no **controle PLAYLIST.**

``` syntax
        elementID.columnWidth
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um  número de leitura/gravação (**long**) que representa a largura da coluna em pixels.

## <a name="remarks"></a>Comentários

A **propriedade columnResizeMode** deve ser definida como "fixa" para que essa propriedade funcione.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player série 9 ou posterior<br/> |
