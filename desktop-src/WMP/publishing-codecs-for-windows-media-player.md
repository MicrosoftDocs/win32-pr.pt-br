# <a name="publishing-codecs-for-windows-media-player"></a>Codecs de publicação do Windows Media Player

Não há mais suporte para o site WMPlugins para publicar seu codec e torná-lo detectável pelos usuários do Windows Media Player. Em vez disso, você pode publicar seu codec em um site de sua escolha.
