# <a name="publishing-codecs-for-windows-media-player"></a>Publicando codecs para Windows Media Player

Não há mais suporte para o site WMPlugins para publicar seu codec e fazer com que ele seja Windows Media Player usuários. Em vez disso, você pode publicar seu codec em um site de sua escolha.
