# <a name="sample-text-section"></a>Seção de texto de exemplo

As linhas a seguir mostram uma seção de texto típica de um arquivo de definição de capa:


```C++
//  <Type>       <Location>     <Align>  <Font>          <Color>

//  ------       ----------     -------  ------          -------

    Time         180,46,50,30   Right    Tahoma,16,N     255,255,255

    Playlist     3,46,175,20    Left     Tahoma,10,N     255,255,255

    Track#       10,68,40,20    Left     Tahoma,14,N     255,255,255

```



O exemplo anterior define três caixas de exibição de texto que mostram a hora atual dentro do item de mídia, a playlist atual e o número de faixa atual.
