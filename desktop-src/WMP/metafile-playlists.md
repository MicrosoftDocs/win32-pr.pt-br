# <a name="metafile-playlists"></a>Playlists de metadados

Windows As playlists de metadados de mídia são usadas para organizar o conteúdo do fluxo e a apresentação. Uma playlist pode simplesmente redirecionar Windows Media Player para o conteúdo a ser transmitido, mas geralmente fornecerá uma lista de arquivos a serem tocadas sequencialmente. Uma playlist mais complexa pode conter repetições, inserções comerciais e informações sobre os itens na lista.

Outra vantagem das playlists é que, em vez de reproduzir um fluxo, parar, iniciar o próximo fluxo e, em seguida, aguardar que ele termine o buffer, Windows Media Services e Windows Media Player trabalhem juntos para reproduzir os clipes um após o outro com tempo de buffer mínimo ou interrupção.
