# <a name="metafile-playlists"></a>Playlists de metarquivo

As listas de reprodução do metarquivo do Windows Media são usadas para organizar o conteúdo e a apresentação do fluxo. Uma playlist pode simplesmente redirecionar o Windows Media Player para o conteúdo a ser transmitido, mas, mais comumente, fornecerá uma lista de arquivos a serem executados em sequência. Uma lista de reprodução mais complexa pode conter repetições, inserções comerciais e informações sobre os itens na lista.

Outra vantagem das listas de reprodução é que, em vez de reproduzir um fluxo, parar, iniciar o próximo fluxo e, em seguida, esperar que ele termine o armazenamento em buffer, o Windows Media Services e o Windows Media Player trabalham juntos para reproduzir os clipes um após o outro com tempo mínimo de buffer ou interrupção.
