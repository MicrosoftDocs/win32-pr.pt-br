# <a name="windows-media-playlist-elements-reference"></a>Referência de elementos da playlist do Windows Media

Esta seção contém a documentação de referência para todos os elementos da lista de reprodução do Windows Media. As entradas de referência incluem definições dos elementos, seus atributos e valores e condições especiais relacionadas a cada elemento.

Há suporte para os seguintes elementos de playlist.



| Elemento                                    | Descrição                                                                                                                                                                                          |
|--------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [argument](argument-element.md)           | Contém uma parte de uma cadeia de caracteres de condição.                                                                                                                                                          |
| [body](body-element.md)                   | Contém os elementos que definem o conteúdo de uma lista de reprodução.                                                                                                                                        |
| [filter](filter-element.md)               | Contém elementos que limitam o tamanho de uma lista de reprodução, a duração de uma lista de reprodução ou o número de itens de mídia em uma lista de reprodução.                                                                                 |
| [fragmento](fragment-element.md)           | Especifica uma condição da consulta que seleciona itens da biblioteca.                                                                                                                            |
| [principal](head-element.md)                   | Contém metadados que se aplicam a toda a lista de reprodução.                                                                                                                                               |
| [media](media-element.md)                 | Especifica um dos itens de mídia em uma lista de reprodução.                                                                                                                                                      |
| [volume](meta-element.md)                   | Especifica os metadados que se aplicam a toda a lista de reprodução.                                                                                                                                              |
| [queryset](queryset-element.md)           | Contém elementos que definem quais itens de mídia serão selecionados da biblioteca.                                                                                                                   |
| [Seq](seq-element.md)                     | Contém elementos que definem uma lista de reprodução estática ou elementos que definem uma lista de reprodução inteligente.                                                                                                            |
| [smartPlaylist](smartplaylist-element.md) | Contém a parte definida dinamicamente de uma playlist.                                                                                                                                              |
| [SMIL](smil-element.md)                   | O elemento **SMIL** é sempre o elemento de nível superior em um arquivo de lista de reprodução de mídia do Windows (WPL). Ele especifica que o arquivo usa sintaxe e gramática de SMIL (linguagem de integração multimídia sincronizada). |
| [sourceFilter](sourcefilter-element.md)   | Contém elementos que selecionam itens de uma biblioteca.                                                                                                                                                  |
| [title](title-element--wpl.md)            | Especifica um título para a lista de reprodução.                                                                                                                                                                  |
