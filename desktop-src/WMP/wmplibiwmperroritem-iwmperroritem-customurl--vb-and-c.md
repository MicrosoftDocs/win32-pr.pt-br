# <a name="iwmperroritemcustomurl-property"></a>Propriedade IWMPErrorItem:: customUrl

A propriedade **customUrl** Obtém a URL de um site que exibe informações específicas sobre a falha de download do codec.

## <a name="syntax"></a>Syntax


```CSharp
public System.String customUrl {get; set;}
```


```VB

Public ReadOnly Property customUrl As System.String
```





## <a name="property-value"></a>Valor da propriedade

Um **System. String** que é a URL personalizada.
