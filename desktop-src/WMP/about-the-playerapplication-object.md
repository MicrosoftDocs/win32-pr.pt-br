# <a name="about-the-playerapplication-object"></a>Sobre o objeto PlayerApplication

O objeto **PlayerApplication** é usado para Windows Media Player de comunicação remota. ele fornece a funcionalidade necessária para alternar entre um controle de Windows Media Player remoto e o modo completo do Player.

a comunicação remota permite que um controle de Windows Media Player inserido em um aplicativo C++ use o mesmo mecanismo de reprodução que o Player. Isso significa que, normalmente, você usará o objeto **PlayerApplication** no código C++ usando as interfaces com. há um caso especial, no entanto, onde você pode inserir o controle de Windows Media Player que exibe uma capa Windows Media Player como sua interface do usuário. nesse caso, **PlayerApplication** pode ser programado usando JScript no código de capa.
