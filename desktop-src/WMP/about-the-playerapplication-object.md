# <a name="about-the-playerapplication-object"></a>Sobre o objeto PlayerApplication

O objeto **PlayerApplication** é usado para o Windows Media Player de comunicação remota. Ele fornece a funcionalidade necessária para alternar entre um controle remoto do Windows Media Player e o modo completo do Player.

A comunicação remota permite que um controle do Windows Media Player incorporado em um aplicativo C++ use o mesmo mecanismo de reprodução que o Player. Isso significa que, normalmente, você usará o objeto **PlayerApplication** no código C++ usando as interfaces com. Há um caso especial, no entanto, onde você pode inserir o controle do Windows Media Player que exibe uma capa do Windows Media Player como sua interface do usuário. Nesse caso, o **PlayerApplication** pode ser programado usando o JScript no código de capa.
