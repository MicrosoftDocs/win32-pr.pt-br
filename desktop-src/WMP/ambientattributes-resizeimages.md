# <a name="ambientattributesresizeimages"></a>Ambienteattributes. resizeImages

O atributo **resizeImages** especifica ou recupera um valor que indica se as imagens contidas no controle são redimensionadas automaticamente quando o tamanho do controle é alterado.

``` syntax
        elementID.resizeImages
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                                           |
|-------|-------------------------------------------------------|
| true  | As imagens contidas no controle são redimensionadas automaticamente. |
| false | As imagens contidas no controle têm um tamanho fixo.    |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------|
| Versão<br/> | Windows Media Player 11<br/> |
