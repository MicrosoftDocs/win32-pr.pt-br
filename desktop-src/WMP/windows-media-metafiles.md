# <a name="windows-media-metafiles"></a>Metaarquivos do Windows Media

Essa referência documenta os metaarquivos do Windows Media, que usam as extensões de nome de arquivo. Wax,. WVX,. WMX e. asx. Ele contém as seções visão geral e guia de programação e uma seção de referência completa sobre marcas de elemento de metarquivo, seus atributos e valores e condições especiais relacionadas a cada elemento.

Um metarquivo é um arquivo que contém informações sobre outros arquivos. Um metarquivo pode ser usado para listar um grupo de arquivos de conteúdo de mídia que devem ser reproduzidos na ordem. As listas de reprodução do metarquivo do Windows Media, simplesmente chamadas de listas de reprodução neste documento de referência, são um dos recursos mais poderosos das tecnologias de mídia do Microsoft Windows. As listas de reprodução permitem que você controle e personalize seu conteúdo de mídia. Por exemplo, com as listas de reprodução, você pode agendar conteúdo para reproduzir sucessivamente ou inserir anúncios ou clipes de interesse especiais em uma apresentação. Uma vantagem adicional das listas de reprodução é que, em vez de reproduzir um fluxo, parar, iniciar o próximo fluxo e, em seguida, esperar que ele termine o armazenamento em buffer, o Windows Media Services e o Windows Media Player trabalham juntos para reproduzir os clipes um após o outro com tempo mínimo de buffer ou interrupção entre eles.

As tecnologias de mídia do Windows e outros produtos da Internet fornecem as ferramentas para entender os dados demográficos do usuário e personalizar dinamicamente uma transmissão ou mensagem para usuários individuais.
