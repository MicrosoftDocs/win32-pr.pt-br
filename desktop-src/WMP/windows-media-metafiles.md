# <a name="windows-media-metafiles"></a>Windows Metadados de mídia

Essa referência documenta Windows metadados de mídia, que usam as extensões de nome de arquivo .file, .wvx, .wmx e .asx. Ele contém seções de visão geral e guia de programação e uma seção de referência completa sobre marcas de elemento de metadados, seus atributos e valores e condições especiais relacionadas a cada elemento.

Um metadado é um arquivo que contém informações sobre outros arquivos. Um metadado pode ser usado para listar um grupo de arquivos de conteúdo de mídia que devem ser interpretados na ordem. Windows Playlists de metadados de mídia, simplesmente conhecidas como playlists neste documento de referência, são um dos recursos mais poderosos do Microsoft Windows Media Technologies. As playlists permitem controlar e personalizar o conteúdo da mídia. Por exemplo, com playlists, você pode agendar o conteúdo para reproduzir em sucessão ou inserir clipes de publicidade ou de interesse especial em uma apresentação. Uma vantagem ainda maior das playlists é que, em vez de reproduzir um fluxo, parar, iniciar o próximo fluxo e, em seguida, esperar que ele termine o buffer, Windows Media Services e Windows Media Player trabalhem juntos para reproduzir os clipes um após o outro com tempo de buffer mínimo ou interrupção entre eles.

Windows As Tecnologias de Mídia e outros produtos da Internet fornecem as ferramentas para entender os dados demográficos do usuário e personalizar dinamicamente uma difusão ou uma mensagem para usuários individuais.
