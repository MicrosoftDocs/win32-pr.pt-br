# <a name="the-video-display"></a>A exibição do vídeo

Você pode optar por incluir uma área de exibição de vídeo em sua capa. O vídeo é exibido em uma região retangular, que é especificada na seção [vídeo](video.md) do arquivo de definição de capa.
