# <a name="art-file-formats-mobile"></a>Formatos de arquivo de arte (móvel)

Arquivos de arte para capas podem ser imagens de bitmap (.bmp) ou GIF (.gif). Também há suporte para imagens JPEG (.jpg) e PNG (.png). Não inclua espaços ou vírgulas no nome do arquivo de arte.

Se você estiver sincronizando arquivos com um dispositivo portátil, certifique-se de que a sincronização não altere a extensão de nome de arquivo nem converta os arquivos de imagem em outro formato.

Muitos programas de desenho podem ser usados para criar bitmaps e arquivos GIF ou para converter arte em outros formatos em bitmaps ou arquivos GIF. Por exemplo, você pode usar Microsoft Paint, que vem com o Microsoft Windows, para desenhos simples.

Você deseja cortar as áreas em branco de suas imagens em torno das bordas para reduzir o tamanho do arquivo. Você pode usar os deslocamentos na seção Bitmaps para compensar a remoção de espaços em branco nos lados esquerdo e superior. Você também pode cortar as bordas em branco inferior e direita das imagens porque elas não são necessárias para medidas. Os únicos arquivos que você não deve cortar são as imagens Background, SeekThumb e VolumeThumb.
