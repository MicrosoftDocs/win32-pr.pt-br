# <a name="art-file-formats-mobile"></a>Formatos de arquivo artístico (móvel)

Arquivos de arte para capas podem ser imagens bitmap (.bmp) ou GIF (.gif). Também há suporte para imagens JPEG (.jpg) e PNG (.png). Não inclua espaços ou vírgulas no nome do arquivo de arte.

Se você estiver sincronizando arquivos para um dispositivo portátil, certifique-se de que a sincronização não altere a extensão de nome de arquivo ou converta os arquivos de imagem em outro formato.

Muitos programas de desenho podem ser usados para criar bitmaps e arquivos GIF, ou para converter arte em outros formatos em bitmaps ou arquivos GIF. por exemplo, você pode usar Microsoft Paint, que vem com o Microsoft Windows, para desenhos simples.

Você desejará cortar as áreas em branco das imagens em volta das bordas para reduzir o tamanho do arquivo. Você pode usar os deslocamentos na seção bitmaps para compensar a remoção de espaços em branco nos lados esquerdo e superior. Você também pode cortar as bordas em branco inferior e direita das imagens, pois elas não são necessárias para as medições. Os únicos arquivos que você não deve cortar são as imagens background, SeekThumb e VolumeThumb.
