# <a name="about-compatibility"></a>Sobre a compatibilidade

Nem todas as versões do Windows Media Player Mobile funcionarão em todos os dispositivos. Além disso, algumas versões são pré-instaladas no dispositivo, enquanto outras estão disponíveis para download e instalação no site da Microsoft. Versões posteriores podem oferecer recursos de capa que versões anteriores não oferecem suporte.
