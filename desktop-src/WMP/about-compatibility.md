# <a name="about-compatibility"></a>Sobre a compatibilidade

Nem todas as versões do Windows Media Player Mobile funcionarão em todos os dispositivos. Além disso, algumas versões são pré-instaladas no dispositivo, enquanto outras versões estão disponíveis para download e instalação no site da Microsoft. Versões posteriores podem oferecer recursos de capa que não dão suporte a versões anteriores.

A tabela a seguir detalha a compatibilidade do Windows Media Player Mobile com vários dispositivos.

| Windows Media Player 1,1          | **PCs de Palm-tamanho:** Dispositivos de classe E-10 *x* da Casio Cassiopeia<br/> Compaq Aero 21 *XX* e Aero 15 *XX*<br/> HP jornada 430<br/>                                      | Versão da Web.                                                     | Somente áudio.                                       |
| Windows Media Player 1,2          | **PCs de mão:** HP jornada 820, HP jornada 680/690/720<br/> Vadem Clio C-1050<br/> Sharp Telios HC-VJ1c/HC-AJ2<br/> Dispositivos NEC Mobile Pro 780/880<br/> | Versão da Web (HPC 3,0).
