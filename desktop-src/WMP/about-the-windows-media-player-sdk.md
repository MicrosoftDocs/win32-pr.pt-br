# <a name="about-the-windows-media-player-sdk"></a>Sobre o SDK do Windows Media Player

O SDK (Software Development Kit) do Microsoft Windows Media Player fornece informações e ferramentas para personalizar o Windows Media Player e o Windows Media Player Mobile e usar o controle ActiveX do Windows Media Player e o controle ActiveX móvel do Windows Media Player. Esta documentação também fornece informações sobre como usar os metaarquivos do Windows Media.

O suporte para personalizar o Windows Media Player é fornecido por:

-   Capas do Windows Media Player. As capas permitem que você personalize a interface do usuário do Player e aprimore sua funcionalidade usando XML.
-   Plug-ins do Windows Media Player. O Windows Media Player inclui suporte para plug-ins que criam efeitos de visualização, que executam tarefas de DSP (processamento de sinal digital), que adicionam elementos personalizados de interface do usuário ao Player de modo completo e que renderizam fluxos de dados personalizados em arquivos de mídia digital criados usando o formato de arquivo ASF.
-   Lojas online do Windows Media Player. As lojas online são empresas comerciais que integram seus serviços à interface do usuário do Windows Media Player para oferecer conteúdo de mídia digital para aluguel ou venda.

Há suporte para a inserção do controle do Windows Media Player em várias tecnologias, incluindo:

-   HTML em navegadores da Web. Há suporte para os navegadores Microsoft Internet Explorer, Firefox e Netscape Navigator versão 7,1.
-   Programas criados com o sistema de desenvolvimento do Microsoft Visual C++®.
-   Programas criados com o Microsoft Visual Basic® 6,0.
-   Programas criados usando o .NET Framework, incluindo programas escritos nas linguagens de programação Visual Basic e C#.
-   Microsoft Office.

O controle móvel do Windows Media Player tem suporte nas seguintes tecnologias:

-   HTML no Microsoft Pocket Internet Explorer.
-   Programas criados com o sistema de desenvolvimento Microsoft eMbedded Visual C++ 4,0.
-   Programas baseados em Active Template Library (ATL) 3,0 ou superior.
