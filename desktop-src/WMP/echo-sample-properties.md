# <a name="echo-sample-properties"></a>Propriedades de exemplo de eco

O exemplo Echo expõe duas propriedades: o tempo de atraso e o nível de efeito (mixagem úmida). O valor do nível de sinal seco (mistura seca) sempre é derivado do valor de mistura úmida. Você precisa modificar o código existente e adicionar um novo código para tornar essas propriedades acessíveis.

As seções a seguir explicam como modificar o código de propriedades:

-   [Como funcionam as propriedades](how-properties-work.md)
-   [Variáveis para armazenar propriedades](variables-to-store-properties.md)
-   [Modificando a propriedade Scale](modifying-the-scale-property.md)
-   [Adicionando a propriedade de combinação úmida](adding-the-wet-mix-property.md)
