# <a name="download-manager"></a>Gerenciador de download

o gerenciador de Download do Microsoft Windows Media Player permite que o tipo 2 lojas online controle o processo de download de conteúdo de dentro de um Windows Media Player painel de tarefas da loja online. Um painel de tarefas da loja online hospeda uma página da Web que é exibida quando o usuário opta por usar uma loja online.

Esta introdução ao Gerenciador de download contém as seções a seguir.
