# <a name="download-manager"></a>Gerenciador de Download

O Gerenciador de Download do Microsoft Windows Media Player permite que as lojas online do tipo 2 controlem o processo de download de conteúdo de dentro de um Windows Media Player de tarefas da loja online. Um painel de tarefas da loja online hospeda uma página da Web que é exibida quando o usuário opta por usar uma loja online.

Esta introdução ao Gerenciador de Download contém as seções a seguir.
