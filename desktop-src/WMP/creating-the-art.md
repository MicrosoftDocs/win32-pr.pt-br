# <a name="creating-the-art"></a>Criando a arte

A arte para essa capa foi criada usando o Adobe Photoshop. Outros programas de edição de gráficos, como o Jasc Paint Shop Pro ou o Microsoft Paint, podem ter sido usados. O Photoshop foi escolhido como um exemplo de um aplicativo de edição de gráficos que dá suporte a camadas, o que é útil para criar vários arquivos de bitmap que serão alinhados entre si.

As etapas a seguir foram usadas para criar a arte. Muitos outros procedimentos e ferramentas de arte podem ser usados; Essa é apenas uma maneira de fazer isso. O importante a ser lembrado é que você deseja terminar com os arquivos de bitmap específicos de que precisa para sua capa.

-   [Fazer um esboço](make-a-sketch.md)
-   [Definir os retângulos](define-the-rectangles.md)
-   [Criando o plano de fundo](creating-the-background.md)
-   [Criando o arquivo de região](creating-the-region-file.md)
-   [Criando o arquivo enviado por push](creating-the-pushed-file.md)
-   [Criando o arquivo desabilitado](creating-the-disabled-file.md)
-   [Criando o super arquivo](creating-the-super-file.md)
-   [Criando o arquivo VolumeThumb](creating-the-volumethumb-file.md)
