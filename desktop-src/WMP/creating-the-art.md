# <a name="creating-the-art"></a>Criando a arte

A arte para essa capa foi criada usando o Adobe Adobe. Outros programas de edição de gráficos, como o Jasc Paint Shop Pro ou Microsoft Paint, podem ter sido usados. O Telegram foi escolhido como um exemplo de um aplicativo de edição de gráficos que dá suporte a camadas, que são úteis para criar vários arquivos de bitmap que serão alinhados entre si.

As etapas a seguir foram usadas para criar a arte. Muitos outros procedimentos de arte e ferramentas podem ser usados; essa é apenas uma maneira de fazer isso. O importante a ter em mente é que você deseja acabar com os arquivos de bitmap específicos necessários para sua capa.

-   [Definir os retângulos](define-the-rectangles.md)
-   [Criando a plano de fundo](creating-the-background.md)
-   [Criando o arquivo de região](creating-the-region-file.md)
-   [Criando o arquivo pressionado](creating-the-pushed-file.md)
-   [Criando o arquivo desabilitado](creating-the-disabled-file.md)
-   [Criando o superar arquivo](creating-the-super-file.md)
