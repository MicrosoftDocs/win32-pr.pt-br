# <a name="iwmperroritem2-vb-and-c-interface"></a>Interface IWMPErrorItem2 (VB e C#)

Fornece uma propriedade para obter um código de condição de erro.

Além das propriedades herdadas de **IWMPErrorItem**, a interface **IWMPErrorItem2** expõe a propriedade a seguir.



| Propriedade                                                                 | Descrição                                          |
|--------------------------------------------------------------------------|------------------------------------------------------|
| [problema](wmplibiwmperroritem2-iwmperroritem2-condition--vb-and-c.md) | Obtém um valor que indica a condição do erro. |



 

Obtenha uma interface **IWMPErrorItem2** convertendo o valor retornado pela propriedade **Item** (em C#, o **método get \_ Item**) da interface **IWMPErrorItem** .

## <a name="members"></a>Membros

A interface **IWMPErrorItem2 (VB e C#)** não define nenhum membro.
