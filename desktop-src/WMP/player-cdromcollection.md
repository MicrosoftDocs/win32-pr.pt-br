# <a name="playercdromcollection"></a>Player. cdromCollection

A propriedade cdromCollection recupera o objeto **cdromCollection** .

## <a name="syntax"></a>Syntax

*Player* . **cdromCollection**

## <a name="possible-values"></a>Valores possíveis

Esta propriedade é um objeto **cdromCollection** somente leitura.

## <a name="remarks"></a>Comentários

Para recuperar o valor dessa propriedade, é necessário ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
