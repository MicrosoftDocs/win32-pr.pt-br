# <a name="equalizersettingsenhancedaudio"></a>EQUALIZERSETTINGS.enhancedAudio

O **atributo enhancedAudio** especifica ou recupera um valor que indica se o áudio aprimorado está ligado.

``` syntax
        elementID.enhancedAudio
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um booliana **de leitura/gravação.**



| Descrição | Valor                                  |
|-------------|----------------------------------------|
| true        | O áudio aprimorado está ligado.           |
| false       | Padrão. O áudio aprimorado está desligado. |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player série 9 ou posterior<br/> |
