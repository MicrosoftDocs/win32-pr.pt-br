# <a name="playerapplication-element"></a>Elemento PLAYERAPPLICATION

O elemento **PLAYERAPPLICATION** pode ser usado por determinadas capas para acessar o objeto **PLAYERAPPLICATION** . Esse elemento fornece funcionalidade semelhante ao atributo global **playerApplication** . esse elemento só é usado por capas que são exibidas por aplicativos C++ que incorporam uma instância remota do controle de Windows Media Player.

O elemento **PLAYERAPPLICATION** dá suporte aos seguintes atributos.



| Atributo                                              | Descrição                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | recupera um valor que indica se o vídeo pode ser exibido por meio do controle de Windows Media Player remoto. |
| [playerDocked](playerapplication-playerdocked-scr.md) | recupera um valor que indica se Windows Media Player está em um estado encaixado.                          |
