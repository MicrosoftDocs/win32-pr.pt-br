# <a name="playerapplication-element"></a>Elemento PLAYERAPPLICATION

O elemento **PLAYERAPPLICATION** pode ser usado por determinadas capas para acessar o objeto **PLAYERAPPLICATION** . Esse elemento fornece funcionalidade semelhante ao atributo global **playerApplication** . Esse elemento só é usado por capas que são exibidas por aplicativos C++ que incorporam uma instância remota do controle do Windows Media Player.

O elemento **PLAYERAPPLICATION** dá suporte aos seguintes atributos.



| Atributo                                              | Descrição                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | Recupera um valor que indica se o vídeo pode ser exibido por meio do controle remoto do Windows Media Player. |
| [playerDocked](playerapplication-playerdocked-scr.md) | Recupera um valor que indica se o Windows Media Player está em um estado encaixado.                          |
