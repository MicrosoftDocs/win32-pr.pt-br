# <a name="effectspreviouspreset"></a>EFFECTs. previousPreset

O método **previousPreset** exibe a predefinição anterior da visualização atual.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Se a predefinição atual for a primeira na lista para a visualização atual, a última predefinição será tornada atual.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
