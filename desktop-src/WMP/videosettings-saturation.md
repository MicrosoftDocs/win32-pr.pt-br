# <a name="videosettingssaturation"></a>VIDEOSETTINGS. saturação

O atributo **saturação** especifica ou recupera a configuração de saturação do vídeo.

``` syntax
        elementID.saturation
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**) com um valor que varia de-127 a + 127 e um valor padrão de 0.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------------------|
| Versão<br/> | versão Windows Media Player para Windows XP ou posterior<br/> |
