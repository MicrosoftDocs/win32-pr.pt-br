# <a name="iwmperroritem-vb-and-c-interface"></a>Interface IWMPErrorItem (VB e C#)

Fornece acesso a informações de erro.

A interface **IWMPErrorItem** expõe as propriedades a seguir.



| Propriedade                                                                             | Descrição                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [customUrl](wmplibiwmperroritem-iwmperroritem-customurl--vb-and-c.md)               | Obtém a URL de um site que exibe informações específicas sobre a falha de download do codec. |
| [errorCode](wmplibiwmperroritem-iwmperroritem-errorcode--vb-and-c.md)               | Obtém o código de erro atual.                                                               |
| [errorContext](wmplibiwmperroritem-iwmperroritem-errorcontext--vb-and-c.md)         | Obtém um valor que indica o contexto do erro.                                          |
| [errorDescription](wmplibiwmperroritem-iwmperroritem-errordescription--vb-and-c.md) | Obtém uma descrição do erro.                                                           |
| [soluciona](wmplibiwmperroritem-iwmperroritem-remedy--vb-and-c.md)                     | Reservado para uso futuro.                                                                   |



 

Obtenha uma interface **IWMPErrorItem** usando a propriedade a seguir.



| Interface                            | Propriedade                                                                   |
|--------------------------------------|----------------------------------------------------------------------------|
| [IWMPError](iwmperror--vb-and-c.md) | [Item](iwmperror-item--vb-and-c.md) (em C#, use o método **Get \_ Item** ) |



 

## <a name="members"></a>Membros

A interface **IWMPErrorItem (VB e C#)** não define nenhum membro.
