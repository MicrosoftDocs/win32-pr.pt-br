# <a name="listboxfocusitem"></a>LISTBOX. focusItem

O atributo **focusItem** especifica ou recupera a linha que contém o foco.

``` syntax
        elementID.focusItem
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**).

## <a name="remarks"></a>Comentários

O item de foco e o item selecionado podem ser diferentes. Um item pode ser selecionado por meio de script, mas outro item pode receber o foco por meio de script. O item com foco terá um retângulo pontilhado em volta e será selecionado se a tecla ENTER for pressionada.
