# <a name="repeatbutton"></a>REPEATBUTTON

Este é um **botão** predefinido com os valores padrão a seguir.

``` syntax
onclick="jscript:player.settings.setMode('loop',down);"
upToolTip="Turn Repeat On"
downToolTip="Turn Repeat Off"
down="wmpprop:player.settings.getMode('loop')"
sticky="true"
```

## <a name="remarks"></a>Comentários

Isso cria um controle de **botão** que alterna para repetir e desativar. As dicas de ferramenta são localizadas. Todas as propriedades desse **botão** podem ser substituídas especificando-as explicitamente.
