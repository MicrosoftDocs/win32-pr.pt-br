# <a name="statustext"></a>STATUSTEXT

Este é um elemento de **texto** predefinido com os seguintes valores padrão.

``` syntax
value="wmpprop:player.status"
tabstop="true"
```

## <a name="remarks"></a>Comentários

Isso criará um elemento de **texto** que exibirá o status de reprodução. Todas as propriedades desse elemento de **texto** podem ser substituídas especificando-as explicitamente.

|--------------------|----------------------------------------------|
| Versão<br/> | Windows Media Player 7,0 ou posterior<br/> |
