# <a name="running-the-sample-project"></a>Executando o projeto de exemplo

O componente COM de exemplo demonstra a funcionalidade fornecida pelos métodos de **IWMPSubscriptionServices** exibindo uma caixa de diálogo sempre que o Windows Media Player chama um dos métodos. Para ver isso acontecer, primeiro você deve adicionar algum conteúdo à biblioteca que contém um atributo **ContentDistributor** que corresponde ao seu serviço. Consulte [adicionando o atributo ContentDistributor](adding-the-contentdistributor-attribute.md). Em seguida, use o Windows Media Player 10 ou posterior para abrir sua loja online, reproduzir o conteúdo premium e copiar o conteúdo premium para um dispositivo. O Windows Media Player criará uma instância do componente COM de exemplo e chamará os métodos quando apropriado.
