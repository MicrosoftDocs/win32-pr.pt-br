# <a name="iwmplibraryname-property"></a>Propriedade IWMPLibrary::name

A **propriedade** name obtém o nome de exibição da biblioteca atual.

## <a name="syntax"></a>Sintaxe


```CSharp
public System.String name {get; set;}
```


```VB

Public ReadOnly Property name As System.String
```





## <a name="property-value"></a>Valor da propriedade

Um **System.String** que é o nome da biblioteca atual.
