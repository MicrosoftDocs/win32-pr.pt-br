# <a name="mapping-images"></a>Mapeando imagens

Imagens de mapeamento são usadas para controles específicos para especificar quais regiões responderão aos cliques do mouse e determinar quais controles recebem quais eventos. A imagem de mapeamento deve ter a mesma altura e largura que a imagem primária e deve ser criada para que as imagens em ambos os arquivos sejam alinhadas exatamente. Por esse motivo, um programa de edição de gráficos que dá suporte a camadas é recomendado porque você pode ter certeza de que suas imagens estão perfeitamente alinhadas.

Controles diferentes exigem diferentes tipos de mapeamento. Um **elemento ButtonElement** precisa de um mapa que tenha uma cor diferente para cada botão. As áreas coloridas no arquivo de mapeamento devem corresponder às áreas dos botões que você deseja mapear.
