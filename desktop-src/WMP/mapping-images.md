# <a name="mapping-images"></a>Mapeando imagens

As imagens de mapeamento são usadas para controles específicos para especificar quais regiões responderão a cliques do mouse e para determinar quais controles recebem quais eventos. A imagem de mapeamento deve ter a mesma altura e largura que a imagem primária e deve ser criada para que as imagens em ambos os arquivos sejam exatamente alinhadas. Por esse motivo, é recomendável um programa de edição de gráficos que dá suporte a camadas, pois você pode ter certeza de que suas imagens estão perfeitamente alinhadas.

Controles diferentes exigem tipos diferentes de mapeamento. Um elemento **buttonelement** precisa de um mapa que tenha uma cor diferente para cada botão. As áreas coloridas no arquivo de mapeamento devem corresponder às áreas dos botões que você deseja mapear.
