# <a name="mapping-images"></a><span data-ttu-id="6d333-110">Mapeando imagens</span><span class="sxs-lookup"><span data-stu-id="6d333-110">Mapping Images</span></span>

<span data-ttu-id="6d333-111">As imagens de mapeamento são usadas para controles específicos para especificar quais regiões responderão a cliques do mouse e para determinar quais controles recebem quais eventos.</span><span class="sxs-lookup"><span data-stu-id="6d333-111">Mapping images are used for specific controls to specify which regions will respond to mouse clicks and to determine which controls receive which events.</span></span> <span data-ttu-id="6d333-112">A imagem de mapeamento deve ter a mesma altura e largura que a imagem primária e deve ser criada para que as imagens em ambos os arquivos sejam exatamente alinhadas.</span><span class="sxs-lookup"><span data-stu-id="6d333-112">The mapping image must be the same height and width as the primary image and should be created so that the images in both files line up exactly.</span></span> <span data-ttu-id="6d333-113">Por esse motivo, é recomendável um programa de edição de gráficos que dá suporte a camadas, pois você pode ter certeza de que suas imagens estão perfeitamente alinhadas.</span><span class="sxs-lookup"><span data-stu-id="6d333-113">For this reason, a graphics editing program that supports layers is recommended because you can be certain your images are perfectly aligned.</span></span>
