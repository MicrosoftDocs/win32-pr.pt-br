# <a name="_wmpocxevents-interface"></a>\_Interface WMPOCXEvents

A interface **\_ WMPOCXEvents** fornece os mesmos eventos que as interfaces **IWMPEvents**, **IWMPEvents2**, **IWMPEvents3** e **IWMPEvents4** , mas herda de **IDispatch** em vez de **IUnknown**. Isso permite que você receba notificações de eventos do Windows Media Player por meio de um coletor de eventos **IDispatch** .
