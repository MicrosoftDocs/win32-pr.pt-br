# <a name="playlistname"></a>Playlist.name

A `name` propriedade especifica ou recupera o nome da lista de reprodução.

## <a name="syntax"></a>Syntax

*Player*. *currentPlaylist*. **nome** do

## <a name="possible-values"></a>Valores possíveis

Esta propriedade é uma **cadeia de caracteres** de leitura/gravação.

## <a name="remarks"></a>Comentários

Para recuperar o valor dessa propriedade, é necessário ter acesso de leitura à biblioteca. Para especificar o valor, o acesso completo é necessário. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).

**Windows Media Player 10 Mobile:** Esta propriedade é somente leitura.
