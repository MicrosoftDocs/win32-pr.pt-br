# <a name="viewfocusobjectid"></a>Exibir. focusObjectID

O atributo **focusObjectID** especifica ou recupera um valor que indica qual elemento tem o foco do teclado. Só pode ser definido em tempo de execução.

``` syntax
        elementID.focusObjectID
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que especifica a **ID** do elemento que tem o foco. Não tem nenhum valor padrão.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
