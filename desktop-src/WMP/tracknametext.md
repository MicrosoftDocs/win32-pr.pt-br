# <a name="tracknametext"></a>TRACKNAMETEXT

Este é um elemento de **texto** predefinido com os seguintes valores padrão.

``` syntax
value="wmpprop:player.currentMedia.name"
tabstop="true"
```

## <a name="remarks"></a>Comentários

Isso criará um elemento de **texto** que exibirá o nome da mídia atual. Todas as propriedades desse elemento de **texto** podem ser substituídas especificando-as explicitamente.
