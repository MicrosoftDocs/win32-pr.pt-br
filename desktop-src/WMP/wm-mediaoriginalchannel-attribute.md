# <a name="wmmediaoriginalchannel-attribute"></a>Atributo WM/MediaOriginalChannel

O atributo **WM/MediaOriginalChannel** especifica o canal no qual uma determinada apresentação foi transmitida pela primeira vez.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

A constante do Windows Media Format SDK para esse atributo é g \_ wszWMMediaOriginalChannel.

Para determinar se você pode alterar o valor desse atributo, use o método [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------|
| Versão<br/> | Windows Media Player 10 ou posterior<br/> |
