# <a name="about-the-player-object"></a>Sobre o objeto do Player

O objeto **Player** é o objeto principal para o controle do Windows Media Player. Todos os outros objetos relacionados são conectados a esse objeto por meio de propriedades específicas que retornam o objeto. Por exemplo, o objeto **Settings** é acessado por meio da propriedade **Settings** . O objeto **Player** fornece métodos, propriedades e eventos relacionados à funcionalidade principal do Windows Media Player.

Como essa referência também será usada para a programação de capas, a ID do objeto do **Player** será "Player" para obter exemplos de sintaxe.

O uso do atributo global do **Player** em um arquivo de definição de capa dá acesso ao objeto **Player** para uso no script. Por meio do objeto **Player** , todos os outros objetos no controle do Windows Media Player também se tornam acessíveis para scripts. Os eventos do objeto **Player** e a propriedade **URL** também podem ser especificados em tempo de design usando o elemento de capa do [jogador](player-element.md) . Para obter mais informações, consulte as [capas do Windows Media Player](windows-media-player-skins.md).
