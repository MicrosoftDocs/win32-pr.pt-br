# <a name="about-the-player-object"></a>Sobre o objeto player

O **objeto** Player é o objeto principal para o Windows Media Player controle. Todos os outros objetos relacionados são conectados a esse objeto por meio de propriedades específicas que retornam o objeto . Por exemplo, o **Configurações** objeto é acessado por meio da **propriedade settings.** O **objeto** Player fornece métodos, propriedades e eventos relacionados à funcionalidade principal do Windows Media Player.

Como essa referência também deve ser usada para programação de capas, a ID do objeto **Player** será "player" para exemplos de sintaxe.

O uso **do atributo** global player em um arquivo de definição de capa fornece acesso ao **objeto Player** para uso em scripts. Por meio **do objeto Player,** todos os outros objetos no controle Windows Media Player se tornam acessíveis para scripts também. Os eventos do objeto **Player** e da **propriedade URL** também podem ser especificados em tempo de design usando o elemento de capa [PLAYER.](player-element.md) Para obter mais informações, [consulte Windows Media Player skins](windows-media-player-skins.md).
