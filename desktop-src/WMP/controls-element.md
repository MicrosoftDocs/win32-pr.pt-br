# <a name="controls-element"></a>Elemento CONTROLS

O **elemento CONTROLS** permite que os seguintes atributos do objeto **Controls** sejam especificados em uma capa.



| Atributo                                                           | Descrição                                                                                     |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [currentAudioLanguage](controls-currentaudiolanguage.md)           | Especifica ou recupera o LCID (identificador de localidade) da linguagem de áudio para reprodução.         |
| [currentAudioLanguageIndex](controls-currentaudiolanguageindex.md) | Especifica ou recupera o índice que corresponde à linguagem de áudio para reprodução.           |
| [Currentitem](controls-currentitem.md)                             | Especifica ou recupera o item de mídia atual.                                                  |
| [Currentmarker](controls-currentmarker.md)                         | Especifica ou recupera o número do marcador atual.                                               |
| [Currentposition](controls-currentposition.md)                     | Especifica ou recupera a posição atual do clipe em segundos desde o início.          |
| [currentPositionTimecode](controls-currentpositiontimecode.md)     | Especifica ou recupera a posição atual no item de mídia atual usando um formato de código de hora. |
