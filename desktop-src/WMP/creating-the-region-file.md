# <a name="creating-the-region-file"></a>Criando o arquivo de região

Você deseja criar um arquivo Region que mostra quais áreas respondem aos toques. Você pode simplesmente copiar as camadas base de cada elemento para uma nova camada e colori-las para corresponder aos números de cores que você usará no arquivo de definição de capa. Certifique-se de que as imagens que você cria nessa camada sejam sólidas e que nenhum efeito seja aplicado. É provável que você queira anotar os números de cores de cada imagem. Você pode obter os números de cores do Seletor de Cor. Você deseja os números RGB, que são três números decimais. Cada número varia de 0 a 255. Por exemplo, um vermelho sólido seria 255, 0, 0.

> [!Note]  
> Os arquivos de região não são usados no Windows Media Player 10 Mobile porque não há suporte para tipos de botão no Windows Media Player 10 Mobile ou posterior.

![arquivo de região](images/ceswmreg.png)

Há apenas quatro imagens nesse arquivo porque somente os botões PlayPause, Stop, Next e Prev são botões de tipo de clique.
