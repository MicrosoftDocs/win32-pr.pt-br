# <a name="creating-the-region-file"></a>Criando o arquivo de região

Você desejará criar um arquivo de região que mostre quais áreas respondem a toques. Você pode simplesmente copiar as camadas de base de cada elemento para uma nova camada e Colorá-las para corresponder aos números de cor que serão usados no arquivo de definição de capa. Certifique-se de que as imagens criadas nesta camada sejam sólidas e que nenhum efeito seja aplicado. Provavelmente, você desejará anotar os números de cor para cada imagem. Você pode obter os números de cor do seletor de cores do Photoshop. Você deseja os números RGB, que são três números decimais. Cada número varia de 0 a 255. Por exemplo, um vermelho sólido seria 255, 0, 0.

> [!Note]  
> Os arquivos de região não são usados nas capas móveis do Windows Media Player 10 porque os tipos de botão não têm suporte no Windows Media Player 10 Mobile ou posterior.

 

A imagem a seguir é o arquivo de região.

![arquivo de região](images/ceswmreg.png)

Há apenas quatro imagens neste arquivo porque apenas os botões PlayPause, stop, Next e Ant são botões de tipo de clique.
