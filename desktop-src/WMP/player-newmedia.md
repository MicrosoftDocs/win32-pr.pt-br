# <a name="playernewmedia-method"></a>Método Player. newMedia

O método **newMedia** cria um novo objeto de **mídia** .

## <a name="syntax"></a>Sintaxe


```JScript
retVal = Player.newMedia(
  URL
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*URL* \[ do no\]
</dt> <dd>

**Cadeia de caracteres** que contém a URL do arquivo de mídia digital com o qual criar o objeto de **mídia** .

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Esse método retorna um objeto de **mídia** .

## <a name="remarks"></a>Comentários

O parâmetro de *URL* não deve ser uma cadeia de caracteres vazia ou nula.
