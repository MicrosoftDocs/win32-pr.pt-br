# <a name="mediacollectioncreatequery-method"></a>Método mediacollection. createQuery

O método **CreateQuery** cria um novo objeto de **consulta** .

## <a name="syntax"></a>Sintaxe


```JScript
retVal = MediaCollection.createQuery()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método retorna um novo objeto de **consulta** vazio.

## <a name="remarks"></a>Comentários

A criação de um novo objeto de **consulta** é a primeira etapa para criar uma consulta composta.
