# <a name="jscript-files"></a>Arquivos JScript

Os arquivos JScript são carregados com o atributo **scriptfile** do elemento **View** . Eles devem ser arquivos de texto e devem usar a extensão de nome de arquivo. js. Se você tiver um arquivo JScript que tenha o mesmo nome que o arquivo de definição de capa, o arquivo JScript será carregado ao mesmo tempo que o arquivo de definição de capa. Por exemplo, se você tiver um arquivo de definição de capa chamado Laure. WMS e tiver um arquivo JScript chamado laure.js, o arquivo de laure.js será carregado automaticamente.

Você pode usar o JScript para criar uma funcionalidade elaborada por trás da sua capa. Ao criar funções no JScript, você pode fazer quase qualquer coisa imaginar com capas. Por exemplo, você pode usar uma lista de reprodução diferente para cada dia da semana, mas sempre ter a mesma na sexta-feira.

Consulte [usando o JScript](using-jscript.md) para obter mais informações sobre como usar o JScript com capas. Observe que Visual Basic Scripting Edition (VBScript), que pode ser usado ao inserir o controle do Windows Media Player em uma página da Web, não tem suporte para uso com capas.
