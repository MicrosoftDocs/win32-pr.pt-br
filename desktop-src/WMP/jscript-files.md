# <a name="jscript-files"></a>JScript Arquivos

JScript arquivos são carregados com o **atributo scriptFile** do **elemento VIEW.** Eles devem ser arquivos de texto e devem usar a extensão de nome de arquivo .js. Se você tiver um arquivo JScript que tenha o mesmo nome que o arquivo de definição de capa, o arquivo JScript será carregado ao mesmo tempo que o arquivo de definição de capa. Por exemplo, se você tiver um arquivo de definição de capa chamado laure.wms e tiver um arquivo JScript chamado laure.js, o arquivo laure.js será carregado automaticamente.

Você pode usar JScript para criar funcionalidades elaborados por trás da sua capa. Ao criar funções no JScript, você pode fazer quase tudo o que é possível com capas. Por exemplo, você pode usar uma playlist diferente para cada dia da semana, mas sempre terá a mesma na sexta-feira.

Consulte [Usando JScript](using-jscript.md) para obter mais informações sobre como usar JScript com capas. Observe que Visual Basic VBScript (Scripting Edition), que pode ser usado ao incorporar o controle Windows Media Player em uma página da Web, não tem suporte para uso com capas.
