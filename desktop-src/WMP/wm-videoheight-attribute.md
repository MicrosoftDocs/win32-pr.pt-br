# <a name="wmvideoheight-attribute"></a>Atributo WM/VideoHeight

O atributo **WM/VideoHeight** especifica a altura da foto ou do vídeo em pixels.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de foto](photo-item-attributes.md)
-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

Este é o atributo somente leitura.

A constante do Windows Media Format SDK para esse atributo é g \_ wszWMVideoHeight.

Para determinar se você pode alterar o valor desse atributo, use o método [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------|
| Versão<br/> | Windows Media Player 10 ou posterior<br/> |
