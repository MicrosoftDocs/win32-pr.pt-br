# <a name="downloadmanagercreatedownloadcollection-method"></a>Método DownloadManager. createbaixecollection

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

O método **Createdownloadscollection** cria uma coleção de download.

## <a name="syntax"></a>Sintaxe


```JScript
retVal = DownloadManager.createDownloadCollection()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método retorna um objeto **downloadcollection** .
