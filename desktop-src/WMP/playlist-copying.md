# <a name="playlistcopying"></a>PLAYLIST. cópia

O atributo de **cópia** recupera um valor que indica se o elemento **playlist** está no ato de cópia.

``` syntax
        elementID.copying
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** somente leitura.



| Valor | Descrição                              |
|-------|------------------------------------------|
| true  | O elemento **playlist** está sendo copiado.     |
| false | O elemento **playlist** não está sendo copiado. |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
