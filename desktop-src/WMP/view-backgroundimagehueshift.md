# <a name="viewbackgroundimagehueshift"></a>Exibir. backgroundImageHueShift

O atributo **backgroundImageHueShift** especifica ou recupera a quantidade pela qual o matiz da imagem de plano de fundo é deslocado.

``` syntax
        elementID.backgroundImageHueShift
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**float**) com um valor que varia de 0,0 a 360,0 com um valor padrão de 0,0.

## <a name="remarks"></a>Comentários

Esse atributo altera o valor de matiz das imagens especificadas pelo atributo **backgroundImage** se ela tiver sido especificada e se refere a uma imagem BMP de 8 bits.
