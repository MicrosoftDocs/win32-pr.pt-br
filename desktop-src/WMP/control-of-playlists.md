# <a name="control-of-playlists"></a>Controle de listas de reprodução

O controle das listas de reprodução é mantido pelo cliente. As listas de reprodução são analisadas pelo Windows Media Player, que executa ações com base nos elementos da lista de reprodução. O cliente tem controle da experiência de streaming.
