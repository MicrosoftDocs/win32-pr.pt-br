# <a name="about-the-closedcaption-object"></a>Sobre o objeto ClosedCaption

O objeto **ClosedCaption** rege a interface de legendas para Windows Media Player. O objeto **ClosedCaption** é obtido por meio da propriedade **ClosedCaption** do objeto **Player** . Por exemplo, para obter o valor da propriedade **SAMIFileName** , digite o seguinte:


```C++
myfile = player.closedcaption.SAMIFileName;

```
