# <a name="videotooltip"></a>VIDEO. toolTip

O atributo **ToolTip** especifica ou recupera o texto da dica de ferramenta para a janela de vídeo.

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>Valores possíveis

Este atributo é uma **cadeia de caracteres** de leitura/gravação, que não deve exceder 1024 caracteres de comprimento. Não tem nenhum valor padrão.

## <a name="remarks"></a>Comentários

Quando esse atributo é definido como "" (cadeia de caracteres vazia), nenhuma dica de ferramenta é exibida.
