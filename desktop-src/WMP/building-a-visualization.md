# <a name="building-a-visualization"></a>Criando uma visualização

você pode criar uma visualização personalizada usando Visual Studio junto com o assistente de plug-in de Windows Media Player. Para obter informações sobre como configurar o assistente e criar um plug-in, consulte [criando um plug-in](building-a-plug-in.md).

Ao executar o assistente, você terá a opção de vários tipos diferentes de plug-ins. Escolha visualização.

depois de concluir o assistente e criar seu plug-in, inicie uma música no Windows Media Player e clique nas visualizações. As predefinições que são criadas automaticamente para você são chamadas de barras *ProjectName* e do *ProjectName* Wave, em que *ProjectName* é o nome do seu projeto.
