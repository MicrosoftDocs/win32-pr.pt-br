# <a name="playlistdropdownvisible"></a>PLAYLIST. dropDownVisible

O atributo **dropDownVisible** especifica ou recupera um valor que indica se a lista suspensa de playlist está visível.

``` syntax
        elementID.dropDownVisible
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                             |
|-------|-----------------------------------------|
| true  | Padrão. A lista suspensa é visível. |
| false | A lista suspensa não está visível.      |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
