# <a name="effectsnexteffect"></a>EFFECTs. nextEffect

O método **nextEffect** exibe a primeira predefinição da próxima visualização, ignorando as predefinições intermediárias.

``` syntax
        elementID.nextEffect()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Esse método exibe a primeira predefinição da próxima visualização na ordem de criação. Se a visualização atual for a última na ordem de criação e, se **allowAll** for false, a primeira visualização será feita atual.
