# <a name="totalduration-attribute"></a>Atributo TotalDuration

O atributo **TotalDuration** é a duração da reprodução do item.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de áudio](audio-item-attributes.md)
-   [Playlists de CD](cd-playlist-attributes.md)
-   [Faixas de CD](cd-track-attributes.md)

## <a name="remarks"></a>Comentários

Esse atributo é armazenado somente no cache da biblioteca.

Para determinar se você pode alterar o valor desse atributo, use o método [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------------------------------------------------------------|
| Versão<br/> | O Windows Media Player 9 Series Windows Media Player 10 ou posterior não oferece suporte a esse atributo<br/> |
