# <a name="using-the-windows-media-player-control-in-a-net-framework-solution"></a>usando o controle de Windows Media Player em uma solução de .NET Framework

o controle de ActiveX Windows Media Player 9 Series ou posterior dá suporte à inserção em aplicativos .NET Framework. as seções a seguir fornecem informações específicas para usar o controle de Windows Media Player com o .NET Framework.



| Seção                                                                                                                                                      | Descrição                                                                                                                  |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [usando o controle de Windows Media Player com Microsoft Visual Studio](using-the-windows-media-player-control-with-microsoft-visual-studio.md)               | fornece detalhes sobre como adicionar o Windows Media Player 9 Series ou o controle posterior a uma solução Visual Studio.                |
| [inserindo o controle de Windows Media Player em uma solução C#](embedding-the-windows-media-player-control-in-a-c--solution.md)                              | fornece instruções para criar um aplicativo C# simples que usa o controle Windows Media Player.                      |
| [inserindo o controle de Windows Media Player em uma solução Visual Basic .net](embedding-the-windows-media-player-control-in-a-visual-basic--net-solution.md) | fornece instruções para criar um aplicativo simples Visual Basic .net que usa o controle Windows Media Player.        |
| [criando o controle de Windows Media Player programaticamente](creating-the-windows-media-player-control-programmatically.md)                                 | descreve como criar um controle de Windows Media Player em um aplicativo sem usar a caixa de ferramentas Visual Studio.            |
