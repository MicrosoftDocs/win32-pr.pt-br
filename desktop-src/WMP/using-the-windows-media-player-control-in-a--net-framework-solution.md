# <a name="using-the-windows-media-player-control-in-a-net-framework-solution"></a>Usando o controle do Windows Media Player em uma solução de .NET Framework

O controle ActiveX do Windows Media Player 9 Series ou posterior dá suporte à inserção em aplicativos .NET Framework. As seções a seguir fornecem informações específicas sobre como usar o controle do Windows Media Player com o .NET Framework.



| Seção                                                                                                                                                      | Descrição                                                                                                                  |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [Usando o controle do Windows Media Player com Microsoft Visual Studio](using-the-windows-media-player-control-with-microsoft-visual-studio.md)               | Fornece detalhes sobre como adicionar o controle do Windows Media Player 9 Series ou posterior a uma solução do Visual Studio.                |
| [Inserindo o controle do Windows Media Player em uma solução C#](embedding-the-windows-media-player-control-in-a-c--solution.md)                              | Fornece instruções para criar um aplicativo C# simples que usa o controle do Windows Media Player.                      |
| [Inserindo o controle do Windows Media Player em uma solução Visual Basic .NET](embedding-the-windows-media-player-control-in-a-visual-basic--net-solution.md) | Fornece instruções para criar um aplicativo simples Visual Basic .NET que usa o controle do Windows Media Player.        |
| [Criando o controle do Windows Media Player programaticamente](creating-the-windows-media-player-control-programmatically.md)                                 | Descreve como criar um controle do Windows Media Player em um aplicativo sem usar a caixa de ferramentas do Visual Studio.            |
| [Sobre tipos de dados de .NET Framework](about--net-framework-data-types.md)                                                                                       | Fornece as informações necessárias para converter a referência de modelo de objeto orientado a script em .NET Framework tipos de dados base. |
