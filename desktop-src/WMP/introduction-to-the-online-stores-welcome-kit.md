# <a name="introduction-to-the-online-stores-welcome-kit"></a>Introdução ao kit de boas-vindas de lojas online

O kit de boas-vindas de lojas online descreve o processo de colocar um Digital Media Store Online integrado no Windows Media Player. O Windows Media Player dá suporte a dois tipos de lojas online: tipo 1 e tipo 2. Para obter informações sobre a diferença entre o tipo 1 e os repositórios do tipo 2, consulte [armazenamentos online do Windows Media Player](windows-media-player-online-stores.md).

O processo de integração descrito nas seções a seguir se aplica a repositórios tipo 1 e tipo 2, exceto por algumas diferenças, que são observadas.

## <a name="eligibility"></a>Qualificação

Para estar qualificado a ter um armazenamento de música online integrado ao Windows Media Player, sua empresa deve atender aos requisitos de integração e requisitos de manutenção fornecidos no contrato de lojas online. Os tipos de requisitos no contrato incluem, mas não se limitam a, o seguinte.

-   Sua empresa deve ter direitos para distribuir conteúdo online por meio do Windows Media Player.
-   Sua empresa deve concordar em tornar um determinado número de faixas disponíveis para os usuários do Windows Media Player.
-   Sua empresa deve fornecer um certo nível de serviço ininterrupto.

Alguns requisitos específicos do contrato são destacados em [requisitos para lojas de música online](requirements-for-online-music-stores.md).

## <a name="contacting-microsoft"></a>Entrando em contato com a Microsoft

Se você tiver dúvidas sobre lojas online e o processo de integração, entre em contato com a equipe virtual dos serviços do Windows Media Player em mpsvctm@microsoft.com .
