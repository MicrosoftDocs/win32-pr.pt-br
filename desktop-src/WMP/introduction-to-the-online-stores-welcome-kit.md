# <a name="introduction-to-the-online-stores-welcome-kit"></a>Introdução ao Kit de Boas-vindas das Lojas Online

O Kit de Boas-vindas de Lojas Online descreve o processo de colocar um armazenamento de mídia digital online no quadro para Windows Media Player. Windows Media Player dá suporte a dois tipos de lojas online: tipo 1 e tipo 2. Para obter informações sobre a diferença entre as lojas do tipo 1 e do tipo 2, [consulte Windows Media Player Online Stores](windows-media-player-online-stores.md).

O processo de entrada descrito nas seções a seguir se aplica aos armazenamentos do tipo 1 e do tipo 2, exceto por algumas diferenças, que são notadas.

## <a name="eligibility"></a>Qualificação

Para se qualificar para ter uma loja de música online integrada ao Windows Media Player, sua empresa deve atender aos requisitos de integração e aos requisitos de manutenção determinados no contrato de lojas online. Os tipos de requisitos no contrato incluem, mas não estão limitados a, o seguinte.

-   Sua empresa deve ter os direitos de distribuir conteúdo online por meio Windows Media Player.
-   Sua empresa deve concordar em disponibilizar um determinado número de faixas para os usuários de Windows Media Player.
-   Sua empresa deve fornecer um determinado nível de serviço ininterrupto.

Alguns requisitos específicos do contrato são realçadas em [Requisitos para lojas de música online.](requirements-for-online-music-stores.md)

## <a name="contacting-microsoft"></a>Entrar em contato com a Microsoft

Se você tiver dúvidas sobre lojas online e o processo de entrada, entre em contato com a equipe virtual Windows Media Player Services em mpsvctm@microsoft.com .
