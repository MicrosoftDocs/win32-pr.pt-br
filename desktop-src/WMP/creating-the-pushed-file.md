# <a name="creating-the-pushed-file"></a>Criando o arquivo enviado por push

Você pode pegar os elementos que criou e reutilizá-los para criar as outras imagens necessárias. Ao combinar camadas e colorir os resultados, você pode criar um bitmap enviado por push que contém as imagens que você deseja ver quando o usuário envia um botão por push. Esse arquivo também contém a imagem secundária para o estado de pausa do botão PlayPause.

A imagem a seguir é o arquivo enviado por push.

![arquivo enviado por push](images/ceswmpus.png)

Os dois botões à direita são para o estado normal pausado e o estado de pausa por push do botão PlayPause. A cor dos botões é amarela, exceto pelo estado normal pausado do botão PlayPause, como um contraste bom para a cor verde usada para o estado normal de todos os botões.
