# <a name="onkeydown"></a>AoApertarTecla

O manipulador de eventos **OnKeyDown** manipula um evento que ocorre quando uma tecla é pressionada.

``` syntax
onkeydown
```

## <a name="remarks"></a>Comentários

Esse manipulador de eventos não é aplicável ao elemento **playlist** , ao elemento **Effects** ou ao elemento **Video** quando o *vídeo*. **sem janela** é false.
