# <a name="ambientattributesbottom"></a>AmbientAttributes.bottom

O **atributo** inferior especifica ou recupera a coordenada inferior do controle.

``` syntax
        elementID.bottom
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um número de leitura/gravação **(** **longo**) que representa a distância em pixels do controle até a borda inferior do **VIEW pai** ou **SUBVIEW.** Ele tem um valor padrão de zero. O comportamento de valores negativos ou quando [AmbientAttributes.height](ambientattributes-height.md) não é especificado, é indefinido.
