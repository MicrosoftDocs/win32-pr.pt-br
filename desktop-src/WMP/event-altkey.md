# <a name="eventaltkey"></a>evento. cotkey

O atributo **entkey** recupera um valor que indica se a tecla Alt estava inoperante quando o evento ocorreu.

``` syntax
event.altKey
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** somente leitura.



| Valor | Descrição                                     |
|-------|-------------------------------------------------|
| true  | Indica que a tecla ALT estava na posição para baixo. |
| false | Indica que a tecla ALT estava na posição para cima.   |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
