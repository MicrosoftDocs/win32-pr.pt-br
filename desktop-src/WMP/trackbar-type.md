# <a name="trackbar-type"></a>Tipo de barra de faixa

Você deve definir o tipo de barra de faixa que deseja usar. A tabela a seguir mostra os tipos válidos.



| Tipo   | Descrição                                                                                                                                                                                                                                                                                                     |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Seek   | Controla a posição de reprodução. Se o valor for zero, o item de mídia será no início. O valor aumenta até que o final do item de mídia seja atingido. O usuário pode deslizar a imagem do thumb para alterar o valor e, enquanto o item de mídia é reproduzindo, a imagem do thumb é movimentada para corresponder à posição de reprodução atual. |
| Volume | Controla o volume de áudio. O valor varia de 0 a 100. O usuário pode deslizar a imagem digital para alterar o volume. Se o volume mudar, a imagem de miniatura será moveda para corresponder ao novo volume.                                                                                                                   |
