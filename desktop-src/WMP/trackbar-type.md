# <a name="trackbar-type"></a>Tipo de TrackBar

Você deve definir o tipo de TrackBar que deseja usar. A tabela a seguir mostra os tipos que são válidos.



| Tipo   | Descrição                                                                                                                                                                                                                                                                                                     |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Seek   | Controla a posição de reprodução. Se o valor for zero, o item de mídia estará no início. O valor aumenta até que o final do item de mídia seja atingido. O usuário pode deslizar a imagem em miniatura para alterar o valor e, enquanto o item de mídia é reproduzido, a imagem em miniatura é movida para corresponder à posição de reprodução atual. |
| Volume | Controla o volume de áudio. O valor varia de 0 a 100. O usuário pode deslizar a imagem em miniatura para alterar o volume. Se o volume for alterado, a imagem Thumb será movida para corresponder ao novo volume.                                                                                                                   |
