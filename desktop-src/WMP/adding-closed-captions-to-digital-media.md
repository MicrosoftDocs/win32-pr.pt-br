# <a name="adding-closed-captions-to-digital-media"></a>Adicionando legendas ocultas à mídia digital

O SAMI (Synchronized Media Interchange) é um formato de arquivo projetado para fornecer texto sincronizado, como legendas, legendas ou descrições de áudio com conteúdo de mídia digital. Integrado em um navegador da Web usando o modelo de objeto do Windows Media Player, o SAMI promove a acessibilidade. Usando SAMI, você pode criar conteúdo de mídia avançado que atinja um público maior e mais diversificado.

Além das fontes padrão, o SAMI pode dar suporte a outros estilos de texto, como diferentes cores, tamanhos ou linguagens para auxiliar uma variedade de usuários. O SAMI pode ser particularmente útil para indivíduos com deficiência visual ou aqueles surdos ou com deficiência auditiva. O formato SAMI também pode ajudar em propósitos educacionais, como ensinar alunos a uma segunda linguagem.

Este tópico se concentra na incorporação do SAMI com o modelo de objeto de controle ActiveX do Windows Media Player. Os arquivos SAMI existem independentemente dos arquivos de mídia digital e não dependem de um formato de áudio ou vídeo específico para funcionar. Como os arquivos são separados, o controle do Windows Media Player irá localizar, analisar, sincronizar e renderizar cada arquivo no computador do cliente. Isso proporciona flexibilidade e funcionalidade adicionais porque permite a edição de arquivos SAMI individuais, a incorporação do arquivo SAMI com formatos de mídia digital diferentes e o armazenamento de arquivos SAMI em diferentes locais de servidor.
