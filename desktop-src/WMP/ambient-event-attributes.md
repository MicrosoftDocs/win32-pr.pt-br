# <a name="ambient-event-attributes"></a>Atributos de evento de ambiente

Os atributos a seguir são comuns a todos os eventos de capa. Eles são acessados com a palavra-chave **Event** dentro de um manipulador de eventos para o elemento.



| Atributo                              | Descrição                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [altKey](event-altkey.md)             | Recupera um valor que indica se a tecla ALT estava inoperante quando o evento ocorreu.                           |
| [Button](event-button.md)             | Recupera um valor que indica quais botões do mouse estavam inativos quando o evento ocorreu.                          |
| [clientX](event-clientx.md)           | Recupera a coordenada x do ponteiro do mouse em relação à região do cliente da janela do aplicativo. |
| [cliente](event-clienty.md)           | Recupera a coordenada y do ponteiro do mouse em relação à região do cliente da janela do aplicativo. |
| [ctrlKey](event-ctrlkey.md)           | Recupera um valor que indica se a tecla CTRL estava inoperante quando o evento ocorreu.                          |
| [deelement](event-fromelement.md)   | Recupera o elemento do qual veio o evento.                                                                   |
| [KeyCode](event-keycode.md)           | Recupera o código de chave ASCII se uma chave foi pressionada quando o evento ocorreu.                                   |
| [offsetX](event-offsetx.md)           | Recupera a coordenada x do ponteiro do mouse em relação ao elemento que aciona o evento.                |
| [offsetY](event-offsety.md)           | Recupera a coordenada y do ponteiro do mouse em relação ao elemento que aciona o evento.                |
| [screenHeight](event-screenheight.md) | Recupera a altura do tamanho de tela disponível em pixels.                                                 |
| [screenWidth](event-screenwidth.md)   | Recupera a largura do tamanho de tela disponível em pixels.                                                  |
| [screenX](event-screenx.md)           | Recupera a coordenada x absoluta do ponteiro do mouse em relação à tela.                         |
| [tela](event-screeny.md)           | Recupera a coordenada y absoluta do ponteiro do mouse em relação à tela.                         |
| [shiftKey](event-shiftkey.md)         | Recupera um valor que indica se a tecla SHIFT estava inoperante quando o evento ocorreu.                         |
| [srcElement](event-srcelement.md)     | Recupera o elemento que disparou o evento.                                                                  |
| [elemento](event-toelement.md)       | Recupera o elemento ao qual o foco do teclado foi movido.                                                           |
| [x](event-x.md)                       | Recupera a coordenada x do ponteiro do mouse em relação à janela do aplicativo.                      |
| [y](event-y.md)                       | Recupera a coordenada y do ponteiro do mouse em relação à janela do aplicativo.                      |
