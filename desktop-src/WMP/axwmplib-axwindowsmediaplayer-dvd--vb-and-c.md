# <a name="axwindowsmediaplayerdvd-property"></a>Propriedade AxWindowsMediaPlayer. DVD

A propriedade DVD Obtém uma interface IWMPDVD que fornece propriedades e métodos para trabalhar com DVDs.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public IWMPDVD dvd {get;}
```


```VB

Public ReadOnly Property dvd As IWMPDVD
```





## <a name="property-value"></a>Valor da propriedade

A interface WMPLib. IWMPDVD.
