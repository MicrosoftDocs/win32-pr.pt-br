# <a name="textfontsize"></a>TEXT. fontSize

O atributo **FontSize** especifica ou recupera o tamanho da fonte para o controle de texto.

``` syntax
        elementID.fontSize
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**) que especifica o tamanho da fonte em pontos. Ele tem um valor padrão de 10.

Consulte o atributo [Value](text-value.md) para ver um exemplo ilustrando como os atributos do elemento **Text** são usados.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
