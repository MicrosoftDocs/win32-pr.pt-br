# <a name="about-the-stringcollection-object"></a>Sobre o objeto StringCollection

O objeto **StringCollection** governa coleções de cadeias de caracteres, que são usadas para fornecer recursos de manipulação de cadeia de caracteres para o objeto **mediacollection** . O objeto **StringCollection** pode ser recuperado por meio do método **GetAttributeStringCollection** do objeto **mediacollection** . Você pode usar isso para trabalhar com coleções de cadeias de caracteres que representam os valores de vários atributos de item de mídia.
