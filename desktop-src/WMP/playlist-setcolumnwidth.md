# <a name="playlistsetcolumnwidth"></a>PLAYLIST. setColumnWidth

O método **setColumnWidth** especifica a largura da coluna e altera o modo de redimensionamento da coluna para "fixed".

``` syntax
        elementID.setColumnWidth(column, width)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="column"></span><span id="COLUMN"></span>*pilha*
</dt> <dd>

**Número** (**longo**) indicando o índice da coluna que está sendo alterada.

</dd> <dt>

<span id="width"></span><span id="WIDTH"></span>*Largura*
</dt> <dd>

**Número** (**longo**) indicando a nova largura em pixels.

</dd> </dl>

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.
