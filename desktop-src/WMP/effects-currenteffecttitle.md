# <a name="effectscurrenteffecttitle"></a>EFFECTs. currentEffectTitle

O atributo **currentEffectTitle** recupera o título de exibição da visualização atual.

``` syntax
        elementID.currentEffectTitle
```

## <a name="possible-values"></a>Valores possíveis

Este atributo é uma **cadeia de caracteres** somente leitura.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
