# <a name="creating-the-volumethumb-file"></a>Criando o arquivo VolumeThumb

O arquivo VolumeThumb contém a imagem Thumb que será usada com um volume TrackBar.

A imagem a seguir é o arquivo VolumeThumb.

![arquivo volumethumb](images/ceswmvol.png)

Ele contém as duas imagens Thumb. A outra à esquerda é o estado normal e a outra à direita é o estado enviado por push. Não há nenhum estado desabilitado. Se um TrackBar estiver desabilitado, a imagem de fundo desabilitada do TrackBar será exibida no arquivo super.
