# <a name="equalizersettingsnormalizationpeak"></a>EQUALIZERSETTINGS.normalizationPeak

O atributo **normalizationPeak** recupera o valor de normalização de pico.

``` syntax
        elementID.normalizationPeak
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** somente leitura (**float**).

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
