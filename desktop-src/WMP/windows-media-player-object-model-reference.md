# <a name="windows-media-player-object-model-reference"></a>Windows Media Player Referência de modelo de objeto

as seções de referência do modelo de objeto Windows Media Player contêm informações detalhadas sobre o modelo de objeto de controle de ActiveX de Windows Media Player. o modelo de objeto fornece interfaces que permitem que os desenvolvedores adicionem Windows Media Player funcionalidade a páginas da web, programas C++ e aplicativos .net.

a referência de modelo de objeto para scripts é apresentada em um estilo projetado para uso com linguagens de script como o Microsoft JScript e documenta o modelo de objeto em termos de objetos com suas propriedades, métodos e eventos associados.

a referência de modelo de objeto para C++ fornece documentação para interfaces COM que os programadores de c++ podem usar para se comunicar com o controle de ActiveX Windows Media Player ou com o Windows Media Player ActiveX controle móvel. a maioria dessas interfaces é implementada pelo controle de Windows Media Player e chamada pelo aplicativo C++. no entanto, algumas das interfaces (por exemplo, **IWMPEvents** e **IWMPRemoteMediaServices**) são implementadas pelo aplicativo C++ e chamadas pelo controle de Windows Media Player.

a referência de modelo de objeto para Visual Basic .net e C# fornece documentação para as propriedades, métodos e eventos do objeto **AxWindowsMediaPlayer** e para interfaces para o controle de ActiveX de Windows Media Player que estão disponíveis por meio da interoperabilidade COM em uma solução codificada no Visual Studio .net 2002 ou posterior.

a referência de modelo de objeto Windows Media Player contém as seções a seguir.



| Seção                                                                                                        | Descrição                                                                                                                                                                                              |
|----------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Referência de modelo de objeto para scripts](object-model-reference-for-scripting.md)                               | Fornece documentação de referência para as propriedades, métodos e eventos que o modelo de objeto disponibiliza para as linguagens de script. essas informações também são úteis para a programação Visual Basic 6,0. |
| [Referência de modelo de objeto para C++](object-model-reference-for-c.md)                                             | Fornece documentação de referência para interfaces C++.                                                                                                                                                     |
| [referência de modelo de objeto para Visual Basic .net e C #](object-model-reference-for-visual-basic--net-and-c.md) | fornece documentação de referência para objetos e interfaces disponíveis para Visual Basic programadores .net e C# por meio da interoperabilidade COM.                                                             |
| [Referência de atributo](attribute-reference.md)                                                                 | fornece documentação de referência para atributos de metadados que são de interesse para Windows Media Player desenvolvedores de SDK.                                                                                    |
| [Tipos de enumeração](enumeration-types.md)                                                                     | Fornece documentação de referência para os tipos de enumeração que são usados pelas interfaces C++ e .NET.                                                                                                 |
