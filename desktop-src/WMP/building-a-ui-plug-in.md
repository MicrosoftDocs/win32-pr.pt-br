# <a name="building-a-ui-plug-in"></a>Criando um plug-in de interface do usuário

Você pode criar um plug-in de interface do usuário usando o Visual Studio e o assistente de plug-in do Windows Media Player. Para obter informações sobre como configurar o assistente e criar um plug-in, consulte [criando um plug-in](building-a-plug-in.md).

Ao executar o assistente, você terá a opção de vários tipos diferentes de plug-ins. Escolha plug-in de interface do usuário.

O assistente cria um plug-in de interface do usuário que você pode usar como ponto de partida. O tópico a seguir explica como personalizar o plug-in para suas próprias finalidades:

-   [Personalizando o plug-in da interface do usuário](customizing-the-ui-plug-in.md)
