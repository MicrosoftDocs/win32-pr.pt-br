# <a name="separate-window-plug-ins"></a>Plug-ins de janela separados

Plug-ins separados da interface do usuário da janela são mostrados em suas próprias janelas. Essas janelas podem ser corrigidas em tamanho ou podem se tornar redimensionáveis. Várias janelas de plug-in de interface do usuário separadas podem ser abertas ao mesmo tempo.

Plug-ins de interface do usuário de janela separados são úteis quando você deseja que as informações ou os controles fornecidos permaneçam visíveis quando o usuário sai do painel em **execução** ou carrega outro plug-in de interface do usuário. quaisquer plug-ins de interface do usuário da janela separados que estejam em execução quando Windows Media Player for fechado serão recarregados e exibidos no mesmo tamanho e na mesma posição quando o Player for iniciado novamente.
