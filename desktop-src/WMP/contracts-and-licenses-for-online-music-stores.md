# <a name="contracts-and-licenses-for-online-music-stores"></a>Contratos e licenças para lojas de música online

A tabela a seguir mostra quem deve assinar quais tipos de contratos antes que uma loja online possa ser incluída Windows Media Player.




| O que você está tentando fazer? | quem é você? | Contrato ou licenças que você precisará assinar | 
|----------------------------|--------------|------------------------------------------|
| Crie uma loja online com marca que está listada em Windows Media Player. A marca da loja online é vista pelo consumidor. | O proprietário da marca da loja online que tem direitos de distribuição para o conteúdo, desenvolveu a loja para Windows Media Player e é responsável por gerenciar o servidor de licença (se aplicável) e atividades de atendimento ao cliente, como cobrança e suporte. | O proprietário da marca assina:<br /><ul><li>Contrato de Listagem de Serviço de Mídia Online para Windows Media Player</li><li>Windows Licença do Media Rights Manager (se estiver usando DRM)</li><li>Windows Licença do SDK de Formato de Mídia</li><li>Windows Media Player Contrato de redistribuição (para redistribuir Windows Media Player por meio de CD, site ou computador pessoal e definir o armazenamento como o serviço ativo inicial) (opcional)</li></ul> | 
| Ofereça uma loja online com marca que está listada em Windows Media Player.<br /> A marca da loja online é vista pelo consumidor. <br /> | O proprietário da marca da loja online que contrai com um ou mais provedores de serviços para criar e operar uma loja online exclusiva.<br /> Essas atividades podem incluir:<br /><ul><li>Desenvolver os recursos de interface do usuário/UX e a funcionalidade que atendem aos requisitos técnicos e Windows Media Player SDK</li><li>Operando o servidor de licença (se aplicável)</li><li>Fornecendo atividades de atendimento ao cliente, como cobrança e suporte.</li></ul> | O proprietário da marca assina:<br /><ul><li>Contrato de Listagem de Serviço de Mídia Online para Windows Media Player</li><li>Windows Media Player Contrato de redistribuição (para redistribuir o Player por meio de CD, site ou computador pessoal e definir o armazenamento como o serviço ativo inicial) (opcional)</li></ul>Sinal de provedor(s) de serviço:<br /><ul><li>Windows Licença do Media Rights Manager (se estiver usando DRM)</li><li>Windows Licença do SDK de Formato de Mídia</li></ul><blockquote>[!Note]<br />O operador do servidor de licença é responsável por assinar a licença Windows Media Rights Manager.</blockquote><br /> | 
| Ofereça serviços de rótulo branco turnkey para proprietários de marca e liste cada loja exclusiva no Windows Media Player.<br /> A marca online Store é vista pelo consumidor. <br /> | Provedor de serviços que cria e opera lojas online para proprietários de marca.<br /> Essas atividades podem incluir:<br /><ul><li>Desenvolver recursos e funcionalidades de interface do usuário/UX que atendem Windows Media Player SDK e requisitos técnicos</li><li>Operando o servidor de licença (se aplicável)</li><li>Fornecendo atividades de atendimento ao cliente, como cobrança e suporte.</li></ul> | Sinais do provedor de serviços:<br /><ul><li>Contrato de Listagem de Serviço de Mídia Online para Windows Media Player</li><li>Windows Licença do Media Rights Manager (se estiver usando DRM)</li><li>Windows Licença do SDK de Formato de Mídia</li></ul>Sinais de proprietário da marca:<br /><ul><li>Exibe um do Contrato de Listagem do Serviço de Mídia Online para Windows Media Player que o provedor de serviços assinou.</li></ul><blockquote>[!Note]<br />O operador do servidor de licença é responsável por assinar a licença Windows Media Rights Manager.</blockquote><br /> | 
| Ofereça uma loja online com identidade visual e liste-a Windows Media Player.<br /> É de sua conta e da loja online determinar qual marca resultante está listada Windows Media Player e vista pelo consumidor.<br /> | Terceiros (como um PC OEM) que negocia com uma loja online existente. | Sinais de terceiros (como um PC OEM) :<br /><ul><li>Windows Media Player Contrato de redistribuição (para redistribuir o Player por meio de CD, site ou computador pessoal e definir o armazenamento como o serviço ativo inicial)</li></ul>Sinais de proprietário da marca:<br /><ul><li>Contrato de Listagem de Serviço de Mídia Online para Windows Media Player</li><li>Windows Licença do Media Rights Manager (se estiver usando DRM)</li><li>Windows Licença do Media Rights Manager (se estiver usando DRM)</li></ul><blockquote>[!Note]<br />Esses contratos podem ter sido assinados pelo proprietário da marca da loja online ou seu provedor de serviços anteriormente.</blockquote><br /> |
