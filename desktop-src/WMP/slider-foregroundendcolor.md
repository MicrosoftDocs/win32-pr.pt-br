# <a name="sliderforegroundendcolor"></a>SLIDER. foregroundEndColor

O atributo **foregroundEndColor** especifica ou recupera a cor final do primeiro plano do controle deslizante.

``` syntax
        elementID.foregroundEndColor
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém qualquer valor de cor do Microsoft Internet Explorer. Não tem nenhum valor padrão.

## <a name="remarks"></a>Comentários

O **foregroundEndColor** é usado em conjunto com um **foregroundColor**. O efeito criado é um esmaecimento de gradiente do **foregroundColor** para o **foregroundEndColor**.
