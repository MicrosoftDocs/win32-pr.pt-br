# <a name="using-windows-media-player-with-netscape-71"></a>Uso do Windows Media Player com o Netscape 7.1

O navegador da Web Netscape 7.1 fornece suporte nativo para controles Windows Media Player ActiveX usando o componente Gecko ActiveX. Isso significa que você pode inserir o controle Windows Media Player em suas páginas da Web para Netscape 7.1 usando a mesma sintaxe que você usaria para o Microsoft Internet Explorer. O Netscape 7.1 também dá suporte ao script do controle e ao tratamento de eventos de maneira semelhante.

Se você quiser usar o controle Windows Media Player no Netscape, é recomendável usar o Netscape 7.1.

Para obter mais informações sobre como usar o Windows Media Player com o Netscape 7.1, consulte o [site do Netscape.](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape)

Para obter mais informações sobre o componente ActiveX Gecko, consulte o [StreamingMedia.com site](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

O download completo desse SDK inclui um exemplo que demonstra a detecção do navegador da Web. Para obter mais informações sobre exemplos incluídos com o SDK, consulte [Exemplos](samples.md).
