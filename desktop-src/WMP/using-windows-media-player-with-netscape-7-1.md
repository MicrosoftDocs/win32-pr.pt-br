# <a name="using-windows-media-player-with-netscape-71"></a>Uso do Windows Media Player com o Netscape 7.1

O navegador da Web Netscape 7,1 fornece suporte nativo para controles ActiveX do Windows Media Player usando o componente ActiveX do Gecko. Isso significa que você pode inserir o controle do Windows Media Player em suas páginas da Web para o Netscape 7,1 usando a mesma sintaxe que usaria para o Microsoft Internet Explorer. O Netscape 7,1 também dá suporte a scripts de controle e manipulação de eventos de maneira semelhante.

Se você quiser usar o controle do Windows Media Player no Netscape, é recomendável usar o Netscape 7,1.

Para obter mais informações sobre como usar o controle do Windows Media Player com o Netscape 7,1, consulte o [site do Netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Para obter mais informações sobre o componente do Gecko ActiveX, consulte o [site do StreamingMedia.com](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

O download completo deste SDK inclui um exemplo que demonstra a detecção do navegador da Web. Para obter mais informações sobre exemplos incluídos no SDK, consulte [Samples](samples.md).
