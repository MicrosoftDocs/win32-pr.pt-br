# <a name="text-type"></a>Tipo de texto

Você deve definir o tipo de caixa de exibição de texto que deseja usar. Os tipos válidos a seguir são mostrados na tabela a seguir.



| Tipo        | Descrição                                                                                                                                                                                                                                                                         |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Hora        | Posição de reprodução atual no item de mídia em minutos e segundos.                                                                                                                                                                                                                 |
| 7.1    | Nome da playlist atual.                                                                                                                                                                                                                                                       |
| Rastrear\#     | Número do item atual na lista de reprodução.                                                                                                                                                                                                                                         |
| \#Controla    | Número de itens na playlist atual.                                                                                                                                                                                                                                            |
| Título       | Título do item atual na playlist atual. Em determinadas condições, esse tipo pode não ser exibido corretamente. Como alternativa, use título como parte de um letreiro.                                                                                                                 |
| Autor      | Autor do item atual na playlist atual. Em determinadas condições, esse tipo pode não ser exibido corretamente. Como alternativa, use autor como parte de um letreiro.                                                                                                               |
| Direitos autorais   | Dados de direitos autorais do item atual na playlist atual. Em determinadas condições, esse tipo pode não ser exibido corretamente. Como alternativa, use direitos autorais como parte de um letreiro.                                                                                                    |
| Nome de arquivo    | Nome de arquivo do item atual na playlist atual.                                                                                                                                                                                                                              |
| FilenameExt | A extensão de nome de arquivo do item atual na playlist atual. Os únicos valores com suporte são "WMA" e "wav".                                                                                                                                                                 |
| Bitrate     | A taxa de bits codificada (em Kbps) do item de mídia atual.                                                                                                                                                                                                                           |
| Frequência   | A frequência de amostra (em KHz) do item de mídia atual.                                                                                                                                                                                                                           |
| Status\*    | O artista, o álbum, o gênero e a taxa de bits do item de áudio atual durante a reprodução, ou a taxa de bits e as legendas (se aplicável) do item de vídeo atual. Quando um item de mídia não está sendo reproduzido, o tipo de texto status exibe o estado de execução atual (por exemplo, parado ou pausado). |



 

\*Disponível para Windows Media Player 10 Mobile ou posterior.
