# <a name="about-type-2-online-stores"></a>Sobre as lojas online do tipo 2

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

O Windows Media Player dá suporte a dois tipos de lojas online: type1 e tipo 2. Os armazenamentos do tipo 1 são compatíveis com o Windows Media Player 11. Os repositórios do tipo 2 têm suporte do Windows Media Player 10 e Windows Media Player 11. Os repositórios do tipo 2 fornecem um nível básico de integração com a interface do usuário do jogador e os repositórios do tipo 1 fornecem uma integração mais profunda. Para obter mais informações sobre a diferença entre o tipo 1 e os repositórios do tipo 2, consulte [armazenamentos online do Windows Media Player](windows-media-player-online-stores.md).
