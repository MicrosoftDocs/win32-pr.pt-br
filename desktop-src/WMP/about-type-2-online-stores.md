# <a name="about-type-2-online-stores"></a>Sobre as lojas online do tipo 2

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

Windows Media Player dá suporte a dois tipos de lojas online: type1 e tipo 2. os armazenamentos do tipo 1 têm suporte no Windows Media Player 11. os repositórios do tipo 2 têm suporte no Windows Media Player 10 e Windows Media Player 11. Os repositórios do tipo 2 fornecem um nível básico de integração com a interface do usuário do jogador e os repositórios do tipo 1 fornecem uma integração mais profunda. para obter mais informações sobre a diferença entre o tipo 1 e os repositórios do tipo 2, consulte [Windows Media Player lojas Online](windows-media-player-online-stores.md).
