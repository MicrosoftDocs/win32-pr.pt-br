# <a name="define-the-rectangles"></a>Definir os retângulos

Como nenhum elemento de interface do usuário pode se sobrepor, é útil definir os retângulos dentro dos quais todos os elementos serão desenhados. Além disso, a criação de retângulos será útil quando você precisar anotar as coordenadas.

Ative a grade do Photoshop com linhas de grade a cada dez pixels. Ative o ajuste de grade para facilitar o desenho de retângulo.

Crie uma nova camada e desenhe retângulos. Com a camada de esboço abaixo, você sabe o local geral onde deseja que o elemento de interface do usuário seja, mas agora é possível ajustar o local exato. Você sempre pode alterar as coisas mais tarde, mas isso ajudará a ficar o mais próximo possível agora. A ilustração a seguir mostra vários retângulos em uma camada.

![camada de retângulo](images/ceswmrek.png)

Agora que os retângulos estão definidos, você pode desativar a camada de esboço.
