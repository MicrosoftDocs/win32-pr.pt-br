# <a name="pixelaspectratioy"></a>PixelAspectRatioY

O **atributo PixelAspectRatioY** é a parte vertical da taxa de proporção de um vídeo.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

Para um vídeo com uma taxa de proporção de 16:9, **PixelAspectRatioY** seria igual a "9".
