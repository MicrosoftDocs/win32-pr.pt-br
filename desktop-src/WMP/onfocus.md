# <a name="onfocus"></a>enfoco

O manipulador de eventos **onfocus** manipula um evento que ocorre quando o elemento recebe o foco do teclado.

``` syntax
onfocus
```

## <a name="remarks"></a>Comentários

O manipulador de eventos **onfocus** não é aplicável ao elemento **Effects** ou ao elemento **Popup** .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------|
| Versão<br/> | Windows Media Player versão 70 ou posterior<br/> |
