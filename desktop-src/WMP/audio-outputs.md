# <a name="audio-outputs"></a>Saídas de áudio

O painel de controle do Windows XP permite que o usuário atribua o nome dispositivo DirectSound padrão a um dispositivo de saída de áudio específico. No Windows Vista, o painel de controle permite que o usuário atribua cada um dos três nomes (eConsole, eMultimedia e eCommunications) a um dispositivo de saída de áudio. Todos esses nomes podem ser atribuídos a dispositivos diferentes ou vários nomes podem ser atribuídos ao mesmo dispositivo. A tabela a seguir mostra como Windows Media Player e o controle Windows Media Player ActiveX escolher dispositivos de saída de áudio padrão.



|       &nbsp;                               | Windows XP                                                                                                                                                                                | Windows Vista                                                                                                                                                              |
|--------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Windows Media Player                 | Por padrão, o Player usa o dispositivo de áudio designado como Dispositivo DirectSound Padrão. O Player fornece uma caixa de diálogo que permite ao usuário alternar o Player para um dispositivo de áudio diferente. | Por padrão, o Player usa o dispositivo de áudio designado como eMultimedia. O Player fornece uma caixa de diálogo que permite ao usuário alternar o Player para um dispositivo de áudio diferente. |
| Windows Media Player controle ActiveX | Por padrão, o controle Player usa o dispositivo de áudio designado como Dispositivo DirectSound Padrão. O dispositivo de saída de áudio não pode ser alterado programaticamente.                                | Por padrão, o controle Player usa o dispositivo de áudio designado como eMultimedia. O dispositivo de saída de áudio não pode ser alterado programaticamente.                                |
