# <a name="audio-outputs"></a>Saídas de áudio

O painel de controle do Windows XP permite que o usuário atribua o nome dispositivo DirectSound padrão a um dispositivo de saída de áudio específico. No Windows Vista, o painel de controle permite que o usuário atribua cada um dos três nomes (eConsole, eMultimedia e eCommunications) a um dispositivo de saída de áudio. Esses nomes podem ser atribuídos a diferentes dispositivos, ou vários nomes podem ser atribuídos ao mesmo dispositivo. A tabela a seguir mostra como o Windows Media Player e o controle ActiveX do Windows Media Player escolhem dispositivos de saída de áudio padrão.



|                                      | Windows XP                                                                                                                                                                                | Windows Vista                                                                                                                                                              |
|--------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Windows Media Player                 | Por padrão, o Player usa o dispositivo de áudio designado como dispositivo DirectSound padrão. O Player fornece uma caixa de diálogo que permite que o usuário alterne o Player para um dispositivo de áudio diferente. | Por padrão, o Player usa o dispositivo de áudio designado como eMultimedia. O Player fornece uma caixa de diálogo que permite que o usuário alterne o Player para um dispositivo de áudio diferente. |
| Controle ActiveX do Windows Media Player | Por padrão, o controle de jogador usa o dispositivo de áudio designado como dispositivo DirectSound padrão. O dispositivo de saída de áudio não pode ser alterado programaticamente.                                | Por padrão, o controle de jogador usa o dispositivo de áudio designado como eMultimedia. O dispositivo de saída de áudio não pode ser alterado programaticamente.                                |
