# <a name="playlistindex-attribute"></a>Atributo PlaylistIndex

O atributo **PlaylistIndex** é o índice do item de mídia em sua lista de reprodução pai.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de áudio](audio-item-attributes.md)
-   [Faixas de CD](cd-track-attributes.md)
-   [Outros itens](other-item-attributes.md)
-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

Esse atributo é um atributo temporário. Ele não é armazenado na biblioteca ou no arquivo de mídia digital.

Itens de mídia podem ser elementos de playlist. Esse atributo é o índice de base zero que representa a posição do item de mídia na lista de reprodução. Se o item de mídia não fizer parte de uma playlist, o valor desse atributo será uma cadeia de caracteres vazia ("").
