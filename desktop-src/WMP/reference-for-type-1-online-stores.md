# <a name="reference-for-type-1-online-stores"></a>Referência para lojas online do tipo 1

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

O Windows Media Player 11 fornece um compilador de catálogo, um objeto e várias interfaces que dão suporte a lojas online do tipo 1. As seções a seguir documentam esses detalhes.



| Seção                                                                                    | Descrição                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Compilador de catálogo para repositórios online do tipo 1](catalog-compiler-for-type-1-online-stores.md) | Fornece material de referência para o compilador que uma loja online usa para compilar seu catálogo de música.                                                                                                                               |
| [Interface IWMPContentContainer](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | Fornece páginas de referência para métodos que o plug-in do repositório online chama para inspecionar um contêiner de conteúdo. Um contêiner de conteúdo representa um conjunto de itens de mídia a serem baixados ou comprados. Implementado pelo Windows Media Player. |
| [Interface IWMPContentContainerList](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | Fornece páginas de referência para métodos que o plug-in do repositório online chama para inspecionar uma lista de contêineres de conteúdo. Implementado pelo Windows Media Player.                                                                           |
| [Interface IWMPContentPartner](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | Fornece páginas de referência para métodos que o Windows Media Player chama para obter informações da loja online e notificar a loja online sobre as ações do usuário. Implementado pelo plug-in da loja online.                  |
| [Interface IWMPContentPartnerCallback](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | Fornece páginas de referência para métodos que o plug-in do repositório online chama para notificar o Windows Media Player sobre a conclusão de operações assíncronas. Implementado pelo Windows Media Player.                                    |
| [Objeto externo para repositórios online do tipo 1](external-object-for-type-1-online-stores.md)   | Fornece páginas de referência para propriedades, métodos e eventos usados pelo script em páginas da Web fornecidas pela loja online.                                                                                                              |
| [Enumerações para repositórios online do tipo 1](enumerations-for-type-1-online-stores.md)         | Fornece páginas de referência para as enumerações usadas na comunicação entre o Windows Media Player e o plug-in do loja online.                                                                                               |
| [Estruturas para lojas online do tipo 1](structures-for-type-1-online-stores.md)             | Fornece páginas de referência para as estruturas usadas na comunicação entre o Windows Media Player e o plug-in do loja online.                                                                                                 |
