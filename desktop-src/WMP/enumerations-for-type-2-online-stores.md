# <a name="enumerations-for-type-2-online-stores"></a>Enumerações para lojas online do tipo 2

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

O tipo de enumeração a seguir está disponível para lojas online usando código C++.



| Enumeração                                                    | Descrição                                         |
|----------------------------------------------------------------|-----------------------------------------------------|
| [WMPSubscriptionServiceEvent](/previous-versions/windows/desktop/api/subscriptionservices/ne-subscriptionservices-wmpsubscriptionserviceevent) | Define os tipos de eventos de serviço que podem ocorrer. |
