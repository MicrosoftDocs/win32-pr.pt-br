# <a name="marquee-color"></a>Cor do letreiro

O valor de cor do tipo deve ser especificado. O valor deve ser especificado usando três inteiros positivos separados por vírgulas. Os valores representam a quantidade de vermelho, verde e azul para uma cor de bitmap e os intervalos de 0 a 255.
