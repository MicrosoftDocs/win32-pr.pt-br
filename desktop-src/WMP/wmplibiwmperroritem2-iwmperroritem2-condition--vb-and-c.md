# <a name="iwmperroritem2condition-property"></a>Propriedade IWMPErrorItem2:: Condition

A propriedade **Condition** Obtém um valor que indica a condição do erro.

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 condition {get; set;}
```


```VB

Public ReadOnly Property condition As System.Int32
```





## <a name="property-value"></a>Valor da propriedade

Um **System. Int32** que é o código de condição.

## <a name="remarks"></a>Comentários

O código de condição é um valor que é usado pela Microsoft para fornecer informações adicionais para a equipe de suporte técnico.
