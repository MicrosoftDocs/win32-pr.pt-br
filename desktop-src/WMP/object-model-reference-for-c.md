# <a name="object-model-reference-for-c"></a>Referência de modelo de objeto para C++

> [!Note]  
> O uso do controle ActiveX do Windows Media Player com C++ tem suporte para o Windows Media Player 9 Series ou posterior.

 

A referência de modelo de objeto para C++ fornece documentação para interfaces móveis do Windows Media Player e tipos de enumeração do Windows Media Player.

Se você quiser usar comunicação remota ou capas com o controle do Windows Media Player, também precisará aprender sobre as interfaces **IWMPPlayerServices** e **IWMPRemoteMediaServices** , que foram projetadas para serem usadas com o C++.

Para ver uma demonstração de inserção do controle do Windows Media Player em um programa C++, consulte os exemplos de C++ incluídos neste SDK.

As seções a seguir documentam as interfaces e enumerações C++ fornecidas pelo modelo de objeto do Windows Media Player.



| Seção                                    | Descrição                                                                                                     |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Interfaces](interfaces.md)               | Lista as interfaces e os métodos disponíveis para programadores do C++ que usam o controle ActiveX do Windows Media Player. |
| [Tipos de enumeração](enumeration-types.md) | Lista os tipos de enumeração disponíveis no código C++.                                                              |
