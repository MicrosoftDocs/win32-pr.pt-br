# <a name="object-model-reference-for-c"></a>Referência de modelo de objeto para C++

> [!Note]  
> O uso Windows Media Player ActiveX controle com C++ é suportado para Windows Media Player Série 9 ou posterior.

 

A Referência de Modelo de Objeto para C++ fornece documentação para Windows Media Player e Windows Media Player interfaces móveis e tipos de enumeração.

Se você quiser usar remoting ou capas com o controle Windows Media Player, também precisará saber mais sobre as interfaces **IWMPPlayerServices** e **IWMPRemoteMediaServices,** que são projetadas para serem usadas com C++.

Para ver uma demonstração de como Windows Media Player controle de Windows Media Player em um programa C++, confira os exemplos do C++ incluídos neste SDK.

As seções a seguir documentam as interfaces e enumerações do C++ fornecidas pelo modelo Windows Media Player objeto.



| Seção                                    | Descrição                                                                                                     |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Interfaces](interfaces.md)               | Lista as interfaces e métodos disponíveis para programadores C++ que usam o controle Windows Media Player ActiveX aplicativo. |
| [Tipos de enumeração](enumeration-types.md) | Lista os tipos de enumeração disponíveis no código C++.                                                              |
