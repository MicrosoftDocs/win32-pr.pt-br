# <a name="managing-playlists"></a>Gerenciando playlists

Um **objeto Playlist** representa um conjunto de itens de mídia. Você usa propriedades e métodos do objeto para manipular os itens de mídia que ele contém e para recuperar metadados sobre a playlist. Para gerenciar as próprias playlists, use o objeto **PlaylistCollection** e o **objeto MediaCollection.**

Os tópicos a seguir descrevem como gerenciar playlists usando o modelo Windows Media Player objeto de controle:

-   [Atributos de playlist](playlist-attributes.md)
-   [Playlists e itens de mídia](playlists-and-media-items.md)
-   [Playlists e o objeto PlaylistCollection](playlists-and-the-playlistcollection-object.md)
-   [Playlists estáticas e automáticas](static-and-auto-playlists.md)
-   [Playlists e o objeto MediaCollection](playlists-and-the-mediacollection-object.md)
