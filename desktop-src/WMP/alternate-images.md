# <a name="alternate-images"></a>Imagens alternativas

Imagens alternativas são imagens que não são vistas quando a capa é inicialmente exibida, mas torna-se visível em resposta a eventos de mouse. Por exemplo, você pode configurar a capa para que, quando um mouse passar sobre uma área específica, a arte nessa área seja alterada. Isso pode informar aos usuários visualmente que eles podem clicar em uma área para fazer algo. Você também pode usar uma imagem alternativa quando o usuário clica em uma área para dar comentários visuais de que o clique foi recebido pelo Windows Media Player.
