# <a name="ambient-event-handlers"></a>Manipuladores de eventos de ambiente

Os manipuladores de eventos a seguir podem ser implementados para a maioria dos elementos de capa. Os atributos de evento  de ambiente acessados com a palavra-chave de evento podem ser usados dentro de um manipulador de eventos para determinar o estado do teclado e do mouse no momento do evento.



| Manipulador de eventos                                   | Descrição                                                                                                                                                                                                                   |
|-------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*atributo* \_ Onchange](attribute-onchange.md) | Quando um atributo de capa altera o valor, ocorre um evento que pode ser manipulado por um manipulador de eventos. O nome do manipulador de eventos é o nome do atributo seguido por um sublinhado e "onchange", como "value \_ onchange". |
| [Onblur](onblur.md)                            | Trata um evento que ocorre quando o elemento perde o foco do teclado.                                                                                                                                                       |
| [Onclick](onclick.md)                          | Trata um evento que ocorre quando o usuário clica no elemento .                                                                                                                                                                |
| [ondblclick](ondblclick.md)                    | Trata um evento que ocorre quando o usuário clica duas vezes no elemento.                                                                                                                                                         |
| [onendalphablend](onendalphablend.md)          | Trata um evento que ocorre quando um elemento conclui uma **operação alphaBlendTo.**                                                                                                                                         |
| [onendmove](onendmove.md)                      | Trata um evento que ocorre quando um elemento conclui uma **operação moveTo.**                                                                                                                                                |
| [Onfocus](onfocus.md)                          | Trata um evento que ocorre quando o elemento recebe o foco do teclado.                                                                                                                                                    |
| [Onkeydown](onkeydown.md)                      | Trata um evento que ocorre quando uma tecla é pressionada.                                                                                                                                                                           |
| [Onkeypress](onkeypress.md)                    | Trata um evento que ocorre quando o usuário pressiona uma tecla alfanumérico.                                                                                                                                                       |
| [Onkeyup](onkeyup.md)                          | Trata um evento que ocorre quando uma chave é liberada.                                                                                                                                                                          |
| [Onmousedown](onmousedown.md)                  | Trata um evento que ocorre quando o usuário clica em um botão do mouse.                                                                                                                                                             |
| [Onmousemove](onmousemove.md)                  | Trata um evento que ocorre quando o usuário move o ponteiro do mouse enquanto ele está sobre um elemento.                                                                                                                               |
| [Onmouseout](onmouseout.md)                    | Trata um evento que ocorre quando o usuário move o ponteiro para fora do elemento.                                                                                                                                                 |
| [Onmouseover](onmouseover.md)                  | Trata um evento que ocorre quando o usuário coloca o ponteiro sobre o elemento pela primeira vez.                                                                                                                                         |
| [Onmouseup](onmouseup.md)                      | Trata um evento que ocorre quando o usuário libera um botão do mouse enquanto o ponteiro está sobre o elemento.                                                                                                                     |
| [Onresize](onresize.md)                        | Trata um evento que ocorre quando um controle é resize.                                                                                                                                                                          |
