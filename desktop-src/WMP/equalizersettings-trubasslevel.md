# <a name="equalizersettingstrubasslevel"></a>EQUALIZERSETTINGS.truBassLevel

O atributo **truBassLevel** especifica ou recupera o nível de TRUBASS do SRS.

``` syntax
        elementID.truBassLevel
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**) que varia de 0 a 100 com um valor padrão de 50.

## <a name="remarks"></a>Comentários

TruBass é um efeito que melhora o som dos níveis de graves da faixa de áudio. Esse atributo será ignorado se **enhancedAudio** estiver definido como false.
