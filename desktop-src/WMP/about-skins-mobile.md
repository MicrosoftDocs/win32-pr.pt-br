# <a name="about-skins-mobile"></a>Sobre capas (móvel)

Uma capa é uma interface do usuário personalizada para Windows Media Player. Você pode criar seus próprios botões para iniciar e parar a reprodução de mídia digital, adicionar controles deslizantes para alterar a posição de volume e reprodução no item de mídia e fornecer ao usuário informações de texto, como o nome de uma música. O melhor de tudo é que você pode adicionar sua própria arte ou usar a arte existente para criar uma aparência exclusiva para sua aparência.

As etapas básicas para criar uma capa são:

1.  **Projete** sua interface do usuário. Decida quais botões, caixas de texto e barras de controle você deseja fornecer ao usuário para que ele possa controlar as funções escolhidas na etapa 1. Talvez você queira esboçar suas ideias para ver como elas se ajustam ao tamanho da imagem com o qual você trabalhará.
2.  **Crie** a arte que você deseja mostrar ao usuário. Isso consistirá em várias imagens que mostram as imagens de plano de fundo, outras imagens que talvez você queira exibir e mapear imagens se você usar botões de região.
3.  **Escreva** o arquivo de definição de capa que vincula as funções de mídia digital, a interface do usuário e as imagens. Você deve seguir determinadas regras para inserir os dados de texto no arquivo, mas pode ver a aparência padrão como um guia.

Você não precisa seguir essas etapas na ordem determinada, mas um bom design vem de estar ciente de todas as possibilidades e cuidar de todos os detalhes.
