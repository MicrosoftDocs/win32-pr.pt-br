# <a name="about-skins-mobile"></a>Sobre as capas (móveis)

Uma capa é uma interface do usuário personalizada para o Windows Media Player. Você pode criar seus próprios botões para iniciar e parar a reprodução de mídia digital, adicionar controles deslizantes para alterar a posição do volume e da reprodução no item de mídia e fornecer ao usuário informações de texto, como o nome de uma música. O melhor de tudo é que você pode adicionar seu próprio trabalho artístico ou usar uma arte existente para criar uma aparência exclusiva para sua capa.

As etapas básicas na criação de uma capa são:

1.  **Crie** sua interface do usuário. Decida quais botões, caixas de texto e trackbars você deseja fornecer ao usuário para que eles possam controlar as funções escolhidas na etapa 1. Talvez você queira esboçar suas ideias para ver como elas se enquadram no tamanho da imagem com a qual você trabalhará.
2.  **Crie** a arte que você deseja mostrar ao usuário. Isso consistirá em várias imagens que mostram as imagens de plano de fundo, outras imagens que você talvez queira exibir e mapeando imagens se usar botões de região.
3.  **Escreva** o arquivo de definição de capa que vinculará as funções de mídia digital, a interface do usuário e as imagens. Você deve seguir determinadas regras para inserir os dados de texto no arquivo, mas pode examinar a aparência padrão como um guia.

Você não precisa seguir essas etapas na ordem fornecida, mas um bom design será informado sobre todas as possibilidades e cuidar de todos os detalhes.
