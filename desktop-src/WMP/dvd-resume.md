# <a name="dvdresume-method"></a>Método DVD. resume

O método **resume** retorna para o modo de reprodução no modo de menu na mesma posição de título que quando o menu foi invocado.

## <a name="syntax"></a>Sintaxe


```JScript
DVD.resume()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

**Windows Media Player 10 Mobile:** Esse método sempre tem sucesso, mas não executa a operação pretendida.
