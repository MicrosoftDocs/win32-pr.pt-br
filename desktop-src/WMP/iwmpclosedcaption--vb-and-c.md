# <a name="iwmpclosedcaption-vb-and-c-interface"></a>Interface IWMPClosedCaption (VB e C#)

Fornece uma maneira de incluir legendas com um arquivo de mídia digital. O texto da legenda está em um arquivo de intercâmbio de mídia acessível (SAMI) sincronizado.

## <a name="members"></a>Membros

A interface **IWMPClosedCaption (VB e C#)** não define nenhum membro.

A interface **IWMPClosedCaption** expõe as propriedades a seguir.



| Propriedade                                                                             | Descrição                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [captioningId](wmplibiwmpclosedcaption-iwmpclosedcaption-captioningid--vb-and-c.md) | Obtém ou define o nome do elemento HTML que exibe a legenda.                       |
| [SAMIFileName](wmplibiwmpclosedcaption-iwmpclosedcaption-samifilename--vb-and-c.md) | Obtém ou define o nome do arquivo que contém as informações necessárias para a legendagem oculta. |
| [SAMILang](wmplibiwmpclosedcaption-iwmpclosedcaption-samilang--vb-and-c.md)         | Obtém ou define o idioma exibido para Legendagem oculta.                                 |
| [SAMIstyle](wmplibiwmpclosedcaption-iwmpclosedcaption-samistyle--vb-and-c.md)       | Obtém ou define o estilo de legendagem oculta.                                                  |



 

Obtenha uma interface **IWMPClosedCaption** usando a propriedade a seguir.



| Objeto                                                            | Propriedade                                                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------|
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [closedCaption](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) |
