# <a name="iwmpcdromburnburnstate-property"></a>Propriedade IWMPCdromBurn:: burnstate

A propriedade **burnstate** Obtém um valor de enumeração que indica o estado de gravação atual.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```CSharp
public WMPBurnState burnState {get;}
```


```VB

Public ReadOnly Property burnState As WMPBurnState
```





## <a name="property-value"></a>Valor da propriedade

Um **WMPLib. WMPBurnState** que é um valor da enumeração **WMPBurnState** que indica o estado atual.
