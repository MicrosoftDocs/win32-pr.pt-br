# <a name="iwmpplaylistarraycount-property"></a>Propriedade IWMPPlaylistArray::count

A **propriedade count** obtém o número de playlists na matriz de playlists.

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 count {get; set;}
```


```VB

Public ReadOnly Property count As System.Int32
```





## <a name="property-value"></a>Valor da propriedade

Um **System.Int32** que é o número de playlists na matriz.

## <a name="remarks"></a>Comentários

Antes de usar essa propriedade, você deve ter acesso de leitura à biblioteca. Para obter mais informações, consulte [Acesso à biblioteca.](library-access.md)
