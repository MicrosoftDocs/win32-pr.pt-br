# <a name="windows-media-player-conversion-plug-ins"></a>Windows Media Player Plug-ins de conversão

O Microsoft Windows Media Player fornece uma arquitetura para programas de plug-in que convertem formatos de arquivo de mídia digital.

A documentação do plug-in de conversão é dividida nas seções a seguir.



| Seção                                                                                    | Descrição                                                                                                                                           |
|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre plug-ins de conversão](about-conversion-plug-ins.md)                                 | Fornece uma visão geral da arquitetura usada para plug-ins de conversão. Leia esta seção para aprender os conceitos gerais envolvidos com essa tecnologia. |
| [Referência de programação de plug-ins de conversão](conversion-plug-ins-programming-reference.md) | Fornece uma referência detalhada para a interface COM com suporte pelo SDK Windows Media Player para plug-ins de conversão.                                |
