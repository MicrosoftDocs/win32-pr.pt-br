# <a name="text-alignment"></a>Alinhamento de texto

Você deve definir o alinhamento da caixa de exibição de texto que deseja usar. Os valores mostrados na tabela a seguir são válidos.



| Valor  | Descrição                |
|--------|----------------------------|
| Esquerda   | O texto é alinhado à esquerda.  |
| Centro | O texto é centralizado.      |
| Direita  | O texto é alinhado à direita. |
