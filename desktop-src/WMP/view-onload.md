# <a name="viewonload"></a>Exibir. OnLoad

O manipulador de eventos **OnLoad** manipula um evento que ocorre quando a **exibição** é exibida pela primeira vez.

``` syntax
onload
```

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
