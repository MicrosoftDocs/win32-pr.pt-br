# <a name="eventshiftkey"></a>Event. shiftKey

O atributo **shiftKey** recupera um valor que indica se a tecla Shift estava inoperante quando o evento ocorreu.

``` syntax
event.shiftKey
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** somente leitura.



| Valor | Descrição                                       |
|-------|---------------------------------------------------|
| true  | Indica que a tecla SHIFT estava na posição inativa. |
| false | Indica que a tecla SHIFT estava na posição para cima.   |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
