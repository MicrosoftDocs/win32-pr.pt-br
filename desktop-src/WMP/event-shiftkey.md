# <a name="eventshiftkey"></a>event.shiftKey

O **atributo shiftKey** recupera um valor que indica se a tecla SHIFT estava inomeada quando o evento ocorreu.

``` syntax
event.shiftKey
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliana somente leitura.**



| Valor | Descrição                                       |
|-------|---------------------------------------------------|
| true  | Indica que a tecla SHIFT estava na posição para baixo. |
| false | Indica que a tecla SHIFT estava na posição para cima.   |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7.0 ou posterior<br/> |
