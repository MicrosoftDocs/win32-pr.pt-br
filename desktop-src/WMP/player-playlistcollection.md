# <a name="playerplaylistcollection"></a>Player. playlistcollection

A propriedade **playlistcollection** recupera o objeto **playlistcollection** .

## <a name="syntax"></a>Syntax

*Player* . **playlistcollection**

## <a name="possible-values"></a>Valores possíveis

Esta propriedade é um objeto **playlistcollection** somente leitura.

## <a name="remarks"></a>Comentários

Para recuperar o valor dessa propriedade, é necessário ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
