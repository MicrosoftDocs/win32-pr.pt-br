# <a name="videosettingsreset"></a>VIDEOSETTINGS. Reset

O método de **redefinição** redefine todos os atributos para seus valores padrão de zero.

``` syntax
        elementID.reset()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-----------------------------------------------------------------|
| Versão<br/> | Versão do Windows Media Player para Windows XP ou posterior<br/> |
