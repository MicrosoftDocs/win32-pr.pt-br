# <a name="videoonvideoend"></a>VÍDEO. onvideoend

O manipulador de eventos **onvideoend** manipula um evento que ocorre quando o vídeo para de renderizar e é descarregado.

``` syntax

onvideoend 
```

| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
