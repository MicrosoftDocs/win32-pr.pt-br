# <a name="playerisonline"></a>Player. IsOnline

A propriedade **IsOnline** recupera um valor que indica se o usuário está conectado a uma rede.

## <a name="syntax"></a>Syntax

*Player* . **IsOnline**

## <a name="possible-values"></a>Valores possíveis

Esta propriedade é um **booliano** somente leitura.



| Valor | Descrição                             |
|-------|-----------------------------------------|
| true  | O usuário está conectado a uma rede.     |
| false | O usuário não está conectado a uma rede. |



 

## <a name="remarks"></a>Comentários

**Windows Media Player 10 Mobile:** Essa propriedade sempre retorna true.
