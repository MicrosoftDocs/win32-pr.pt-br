# <a name="display-area-plug-ins-deprecated"></a>Exibir plug-ins de área (preteridos)

Esta página documenta um recurso que pode não estar disponível em versões futuras do Windows Media Player e no SDK do Windows Media Player.

Os plug-ins de interface do usuário da área de exibição são mostrados na área principal do painel em **execução** , abaixo das informações do artista e do controle e à esquerda da área da playlist. Somente um plug-in de área de exibição pode ser habilitado por vez. Quando um plug-in de área de exibição está habilitado, o Player não pode entrar no modo de tela inteira.

Os plug-ins de área de exibição são úteis para exibir grandes quantidades de informações. Por exemplo, um plug-in de área de exibição pode recuperar informações sobre o item de mídia em execução no momento e exibir artigos, entrevistas e revisões relacionados. Um plug-in de área de exibição pode hospedar uma janela do navegador Microsoft Internet Explorer para exibir essas informações usando HTML.

> [!Note]  
> Uma janela do navegador hospedado pode inserir a versão do ActiveX do Windows Media Player, mas isso não é recomendado porque o controle inserido não pode se comunicar com o Player autônomo que a contém.

 

> [!Note]  
> O Windows Media Player fecha qualquer plug-in de área de exibição aberta quando o usuário reproduz conteúdo que contém vídeo.
