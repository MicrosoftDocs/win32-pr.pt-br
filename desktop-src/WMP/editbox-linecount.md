# <a name="editboxlinecount"></a>Admy. lineCount

O atributo **LineCount** recupera o número de linhas no controle caixa de edição.

``` syntax
        elementID.lineCount
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** somente leitura (**Long**).

## <a name="remarks"></a>Comentários

Esse atributo é útil somente quando **editstyle** é definido como "Multiline".

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
