# <a name="onkeypress"></a>OnKeyPress

O manipulador de eventos **OnKeyPress** manipula um evento que ocorre quando uma tecla alfanumérica é pressionada.

``` syntax
onkeypress
```

## <a name="remarks"></a>Comentários

Esse manipulador de eventos não é aplicável ao elemento **playlist** , ao elemento **Effects** ou ao elemento **Video** quando o *vídeo*. **sem janela** é false.
