# <a name="dsp-plug-in-developer-overview"></a>Visão geral do desenvolvedor de plug-in do DSP

Da perspectiva do desenvolvedor, plug-ins do DSP são programas de software que recebem dados de áudio ou vídeo fornecidos pelo Windows Media Player logo antes do estágio de renderização, depois processam esses dados e retornam os dados para o Windows Media Player para renderização.

As seções a seguir fornecem informações gerais sobre plug-ins do DSP do Windows Media Player:

-   [Empacotamento de plug-in do DSP](dsp-plug-in-packaging.md)

-   [Entrada e saída de dados](data-input-and-output.md)
-   [Negociação de formato](format-negotiation.md)
-   [Fornecendo uma interface do usuário](providing-a-user-interface.md)
-   [Assistente de plug-in do DSP](dsp-plug-in-wizard.md)
-   [Interfaces necessárias](required-interfaces.md)
-   [Registrando plug-ins do DSP](registering-dsp-plug-ins.md)
-   [Atualizando plug-ins de DSP existentes](updating-existing-dsp-plug-ins.md)
