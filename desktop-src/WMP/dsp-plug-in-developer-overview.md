# <a name="dsp-plug-in-developer-overview"></a>Visão geral do desenvolvedor do plug-in DSP

Da perspectiva do desenvolvedor, os plug-ins DSP são programas de software que recebem dados de áudio ou vídeo fornecidos pelo Windows Media Player logo antes do estágio de renderização e, em seguida, processam esses dados e retornam os dados para o Windows Media Player para renderização.

As seções a seguir fornecem informações gerais sobre Windows Media Player plug-ins DSP:

-   [Empacotamento de plug-in do DSP](dsp-plug-in-packaging.md)

-   [Entrada e saída de dados](data-input-and-output.md)
-   [Negociação de formato](format-negotiation.md)
-   [Fornecendo um Interface do Usuário](providing-a-user-interface.md)
-   [Assistente de Plug-in do DSP](dsp-plug-in-wizard.md)
-   [Interfaces necessárias](required-interfaces.md)
-   [Registrando plug-ins DSP](registering-dsp-plug-ins.md)
