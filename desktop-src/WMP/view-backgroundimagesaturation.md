# <a name="viewbackgroundimagesaturation"></a>Exibir. backgroundImageSaturation

O atributo **backgroundImageSaturation** especifica ou recupera o valor de saturação da imagem de plano de fundo.

``` syntax
        elementID.backgroundImageSaturation
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**float**) com um valor que varia de 0,0 a 2,0 com um valor padrão de 1,0.

## <a name="remarks"></a>Comentários

Esse atributo altera o valor de saturação das imagens especificadas pelo atributo **backgroundImage** se ela tiver sido especificada e se refere a uma imagem BMP de 8 bits.
