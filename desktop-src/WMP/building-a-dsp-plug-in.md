# <a name="building-a-dsp-plug-in"></a>Criando um plug-in do DSP

Você pode criar um plug-in do DSP usando o Visual Studio e o assistente de plug-in do Windows Media Player. Para obter informações sobre como configurar o assistente e criar um plug-in, consulte [criando um plug-in](building-a-plug-in.md).

Ao executar o assistente, você terá a opção de vários tipos diferentes de plug-ins. Escolha **plug-in DSP**.

Os tópicos a seguir descrevem os recursos dos plug-ins de áudio e vídeo DSP de exemplo que você pode criar usando o assistente.

-   [Sobre o plug-in de DSP de áudio de exemplo](about-the-sample-audio-dsp-plug-in.md)
-   [Sobre o plug-in de DSP de vídeo de exemplo](about-the-sample-video-dsp-plug-in.md)
