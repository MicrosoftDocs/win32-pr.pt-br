# <a name="playlistitemselectedfocuslostcolor"></a>PLAYLIST.itemSelectedFocusLostColor

O **atributo itemSelectedFocusLostColor** especifica ou recupera um valor que indica a cor do texto de um item selecionado na playlist quando a playlist perde o foco.

``` syntax
        elementID.itemSelectedFocusLostColor
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma cadeia de caracteres de **leitura/gravação** que representa uma cor. O valor padrão é a Windows de texto do botão.
