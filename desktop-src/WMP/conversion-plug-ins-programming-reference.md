# <a name="conversion-plug-ins-programming-reference"></a>Referência de programação de plug-ins de conversão

o SDK (Software Development Kit) do Microsoft Windows Media Player oferece suporte a uma interface para plug-ins de conversão que convertem um formato de arquivo de mídia digital em outro. O SDK também especifica os requisitos para registrar formatos de arquivo para conversão por um plug-in específico. As seções a seguir documentam esses detalhes.



| Seção                                                                | Descrição                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registrando plug-ins de conversão](registering-conversion-plug-ins.md) | Descreve como registrar formatos de arquivo de mídia digital para conversão.                                                                                                                                  |
| [Interface IWMPConvert](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | fornece métodos para habilitar os plug-ins de conversão de Windows Media Player para converter arquivos de mídia digital criados usando tecnologias não fornecidas pela Microsoft no formato ASF (Advanced Systems Format). |
