# <a name="wmp-sdk-samples"></a>Amostras do SDK do WMP

O SDK do Windows Media Player inclui uma variedade de exemplos que demonstram muitas das técnicas de programação descritas neste documento.

-   Exemplos de script demonstram como inserir o controle ActiveX do Player em uma página da Web. Um exemplo de página da Web completa demonstra como detectar a versão atual do controle ActiveX do Player.
-   Um exemplo de capas demonstra como alterar a aparência visual e o comportamento do Windows Media Player.
-   Um assistente para Microsoft Visual C++ pode criar uma amostra funcional exclusiva de vários tipos de plug-in: um plug-in de visualização, um plug-in de interface do usuário, um plug-in de renderização e um plug-in de processamento de sinal digital.
-   Exemplos de lojas online incluem um assistente para Microsoft Visual C++ que pode criar um exemplo de trabalho exclusivo de um objeto COM da loja online.
-   Exemplos de Microsoft Visual C++ demonstram a inserção do controle ActiveX do Windows Media Player, a comunicação remota do controle ActiveX do Player e o trabalho com a biblioteca.
-   Um exemplo em C# demonstra como usar o controle ActiveX do Windows Media Player com o .NET Framework.
-   Um exemplo Visual Basic .NET demonstra o uso do controle ActiveX do Windows Media Player com o .NET Framework.

Para usar os exemplos, você deve instalar o SDK do Windows, que inclui o SDK do Windows Media Player. Para baixar o SDK do Windows, vá para a página de [SDK do Windows](https://msdn.microsoft.com/windows/bb980924.aspx) no centro de desenvolvimento do Windows.
