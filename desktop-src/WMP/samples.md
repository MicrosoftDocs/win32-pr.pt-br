# <a name="wmp-sdk-samples"></a>Exemplos de SDK do WMP

O Windows Media Player SDK inclui uma variedade de exemplos que demonstram muitas das técnicas de programação descritas neste documento.

-   Exemplos de script demonstram como inserir o controle player ActiveX em uma página da Web. Um exemplo de página da Web completo demonstra como detectar a versão atual do controle player ActiveX.
-   Um exemplo de capas demonstra como alterar a aparência visual e o comportamento de Windows Media Player.
-   Um assistente para Microsoft Visual C++ pode criar um exemplo de trabalho exclusivo de vários tipos de plug-in: um plug-in de visualização, um plug-in de interface do usuário, um plug-in de renderização e um plug-in de processamento de sinal digital.
-   Exemplos de lojas online incluem um assistente para Microsoft Visual C++ que pode criar um exemplo de trabalho exclusivo de um objeto COM da loja online.
-   Exemplos para Microsoft Visual C++ demonstram a incorporação do controle Windows Media Player ActiveX, a ActiveX player e o trabalho com a biblioteca.
-   Um exemplo de C# demonstra o uso do controle Windows Media Player ActiveX com o .NET Framework.
-   Um Visual Basic .NET demonstra o uso do controle Windows Media Player ActiveX com o .NET Framework.

Para usar os exemplos, você deve instalar o SDK do Windows, que inclui o SDK do Windows Media Player. Para baixar o SDK Windows, acesse [a página Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) do Windows Developer Center.
