# <a name="sliderthumbhoverimage"></a>SLIDER. thumbHoverImage

O atributo **thumbHoverImage** especifica ou recupera a imagem do Thumb que aparece ao passar o mouse sobre ele.

``` syntax
        elementID.thumbHoverImage
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém o nome de um arquivo de imagem.

## <a name="remarks"></a>Comentários

O **thumbHoverImage** é opcional. Se não for fornecido, o controle deslizante não alterará a aparência quando o mouse passar sobre ele.

Os formatos com suporte são BMP, JPG, PNG e GIF (não incluindo GIFs animados).
