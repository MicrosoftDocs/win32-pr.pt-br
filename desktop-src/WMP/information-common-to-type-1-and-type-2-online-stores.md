# <a name="information-common-to-type-1-and-type-2-online-stores"></a>Informações comuns às lojas online do tipo 1 e do tipo 2

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

Windows Media Player dá suporte a dois tipos de lojas online: tipo 1 e tipo 2. Os armazenamentos do tipo 1 têm suporte Windows Media Player 11. Os armazenamentos do tipo 2 têm suporte Windows Media Player 10 e Windows Media Player 11. Os armazenamentos do tipo 2 fornecem um nível básico de integração com a interface do usuário do Player e os armazenamentos do Tipo 1 fornecem uma integração mais profunda. Para obter mais informações sobre a diferença entre as lojas do tipo 1 e do tipo 2, [consulte Windows Media Player Online Stores](windows-media-player-online-stores.md).
