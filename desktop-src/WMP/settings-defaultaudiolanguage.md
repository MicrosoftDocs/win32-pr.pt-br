# <a name="settingsdefaultaudiolanguage"></a>Configurações. defaultAudioLanguage

A propriedade **defaultAudioLanguage** recupera o LCID do idioma de áudio padrão especificado em Windows Media Player.

## <a name="syntax"></a>Syntax

Player. Settings. defaultAudioLanguage

## <a name="possible-values"></a>Valores possíveis

Essa propriedade é um **número** somente leitura (**Long**).

## <a name="remarks"></a>Comentários

Um LCID identifica exclusivamente um dialeto de idioma específico, chamado de localidade.
