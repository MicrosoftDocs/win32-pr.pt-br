# <a name="windows-media-player-dsp-plug-ins"></a>Windows Media Player Plug-ins do DSP

o Microsoft Windows Media Player fornece uma arquitetura que permite ao usuário instalar e ativar programas de plug-in que adicionam funcionalidade de processamento de sinal digital (DSP). um plug-in do DSP é um objeto de mídia DirectX da Microsoft (DMO) ou uma Media Foundation transformação (MFT) que se conecta ao Player usando interfaces COM. Um plug-in típico do DSP pode ser um equalizador de áudio ou um controle de tonalidade de vídeo. Esta seção do SDK fornece as informações de programação necessárias para criar seu próprio plug-in do DSP.

A documentação de plug-in do DSP é dividida nas três seções a seguir.



| Seção                                                                      | Descrição                                                                                                                                     |
|------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre plug-ins do DSP](about-dsp-plug-ins.md)                                 | Fornece uma visão geral da arquitetura usada para plug-ins do DSP. Leia esta seção para aprender os conceitos gerais envolvidos nessa tecnologia.  |
| [Guia de programação de plug-ins do DSP](dsp-plug-ins-programming-guide.md)         | Explica o que você precisa fazer para criar um plug-in do DSP. Esta seção contém código de exemplo e procedimentos passo a passo.                           |
| [Referência de programação de plug-ins do DSP](dsp-plug-ins-programming-reference.md) | fornece uma referência detalhada para as interfaces COM, os métodos e os tipos enumerados com suporte pelo SDK Windows Media Player para plug-ins do DSP. |
