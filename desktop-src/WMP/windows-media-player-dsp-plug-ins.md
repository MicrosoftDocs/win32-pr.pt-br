# <a name="windows-media-player-dsp-plug-ins"></a>Plug-ins de DSP do Windows Media Player

O Microsoft Windows Media Player fornece uma arquitetura que permite ao usuário instalar e ativar programas de plug-in que adicionam funcionalidade de DSP (processamento de sinal digital). Um plug-in do DSP é um Microsoft DirectX Media Object (DMO) ou uma Media Foundation Transform (MFT) que se conecta ao Player usando interfaces COM. Um plug-in típico do DSP pode ser um equalizador de áudio ou um controle de tonalidade de vídeo. Esta seção do SDK fornece as informações de programação necessárias para criar seu próprio plug-in do DSP.

A documentação de plug-in do DSP é dividida nas três seções a seguir.

| [Sobre plug-ins do DSP](about-dsp-plug-ins.md)                                 | Fornece uma visão geral da arquitetura usada para plug-ins do DSP. Leia esta seção para aprender os conceitos gerais envolvidos nessa tecnologia.  |
| [Guia de programação de plug-ins do DSP](dsp-plug-ins-programming-guide.md)         | Explica o que você precisa fazer para criar um plug-in do DSP. Esta seção contém código de exemplo e procedimentos passo a passo.                           |
| [Referência de programação de plug-ins do DSP](dsp-plug-ins-programming-reference.md) | Fornece uma referência detalhada para as interfaces COM, os métodos e os tipos enumerados suportados pelo SDK do Windows Media Player para plug-ins do DSP. |
