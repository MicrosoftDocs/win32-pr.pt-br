# <a name="the-constructor"></a>O Construtor

O construtor que o assistente fornece armazena um ponteiro para a classe de plug-in principal em uma variável de membro. O assistente usa essa referência para obter acesso a variáveis de membro declaradas na classe CSearch. Nenhuma alteração de implementação é necessária.
