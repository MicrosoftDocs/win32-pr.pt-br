# <a name="progressbar-element"></a>Elemento PROGRESSBAR

O **elemento PROGRESSBAR** fornece uma maneira de exibir informações de progresso em um controle horizontal ou vertical. Esse elemento é idêntico ao **elemento SLIDER.** Uma barra de progresso normalmente é um controle deslizante que não é interativo, mas não há nada que a impeça de ser usada interativamente. Esse elemento é fornecido exclusivamente como uma marca de conveniência para deixar a nãointeratividade clara em seu código, se você optar por usá-la dessa maneira. Ele não tem nenhum outro valor.
