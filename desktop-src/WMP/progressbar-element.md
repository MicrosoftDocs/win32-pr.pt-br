# <a name="progressbar-element"></a>Elemento PROGRESSBAR

O elemento **PROGRESSBAR** fornece uma maneira de exibir informações de progresso em um controle horizontal ou vertical. Esse elemento é idêntico ao elemento **Slider** . Uma barra de progresso normalmente é um controle deslizante que não é interativo, mas não há nada que impeça que ele seja usado interativamente. Esse elemento é fornecido unicamente como uma marca de conveniência para tornar o noninteractivity claro em seu código, se você optar por usá-lo dessa forma. Ele não tem nenhum outro valor.
