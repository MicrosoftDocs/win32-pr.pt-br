# <a name="iwmpdvdresume-method"></a>Método IWMPDVD:: resume

O método **resume** retorna para o modo de reprodução no modo de menu, retomando na mesma posição que o menu foi invocado.

## <a name="syntax"></a>Sintaxe


```CSharp
public void resume();
```


```VB

Public Sub resume()
Implements IWMPDVD.resume
```





## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
