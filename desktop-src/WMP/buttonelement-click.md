# <a name="buttonelementclick"></a>BOTÃO. clique em

O método **Click** chama o manipulador de eventos **onclick** definido para o **buttonelement**.

``` syntax
        elementID.click()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Use esse método para fornecer um meio alternativo de executar o código associado a um botão.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
