# <a name="art-files-mobile"></a>Arquivos de arte (móvel)

Você precisará criar arquivos de arte para sua capa. Há vários tipos de arquivos de arte que você pode criar e eles devem estar em um formato de arquivo artístico específico.

A tabela a seguir descreve sete tipos de arquivos de arte que você pode criar para sua capa.



| Tipo de arquivo Art                            | Descrição                                                                                                |
|------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Arquivo em segundo plano](background-file.md)   | Obrigatórios. O plano de fundo é a imagem na qual todas as imagens de botão são sobrepostas                        |
| [Arquivo desabilitado](disabled-file.md)       | Obrigatórios. Imagens usadas para indicar que os botões de envio por push não terão nenhum efeito.                                |
| [Arquivo enviado por push](pushed-file.md)           | Obrigatórios. Define as imagens que são exibidas quando o usuário envia botões.                              |
| [Arquivos de região](region-files.md)         | Define os bitmaps que usam áreas de cores para representar os locais de resposta de toque para os botões de tipo de acesso      |
| [Super arquivos](super-files.md)           | Define as imagens desabilitadas para trackbars.                                                                 |
| [Arquivo SeekThumb](seekthumb-file.md)     | Define uma imagem a ser usada em conjunto com um TrackBar para indicar a posição atual no item de mídia. |
| [Arquivo VolumeThumb](volumethumb-file.md) | Define uma imagem a ser usada em conjunto com um TrackBar para indicar a posição de controle de volume.            |



 

As seções a seguir fornecem mais informações sobre arquivos de arte.

-   [Formatos de arquivo artístico](art-file-formats-mobile.md)
-   [Paletas de cores](color-palettes.md)
-   [Tamanhos de imagem](image-sizes.md)
-   [Posicionamento do botão](button-placement.md)

Para obter mais informações sobre tipos de arquivos de arte e como identificá-los no arquivo de definição de capa, consulte a seção [bitmaps](bitmaps.md) da referência de capa.
