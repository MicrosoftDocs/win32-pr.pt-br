# <a name="modifiedby-attribute"></a>Atributo ModifiedBy

\[Não há mais suporte para este atributo. O valor é sempre vazio ou "usuário". \]

O atributo **ModifiedBy** é uma cadeia de caracteres que indica se os atributos do item foram modificados pelo usuário.

## <a name="applies-to"></a>Aplica-se A

-   [DVDs](dvd-attributes.md)

## <a name="remarks"></a>Comentários

Esse atributo é armazenado somente no arquivo de mídia digital.

Para determinar se você pode alterar o valor desse atributo, use o método [**Media. isReadOnlyItem**](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versão<br/> | Windows Media Player 9 Series ou posterior<br/> |
