# <a name="type-2-online-stores"></a>Tipo 2 lojas online

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

O Windows Media Player 11 dá suporte às lojas online tipo 1 e tipo 2. Os armazenamentos do tipo 1 são mais profundamente integrados à interface do usuário do Player do que os repositórios do tipo 2. O Windows Media Player 10 dá suporte às lojas do tipo 2, mas não ao tipo 1. Esta documentação, que se aplica somente a repositórios do tipo 2, é dividida nas seções a seguir.



| Seção                                                                                      | Descrição                                                                                                                            |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre as lojas online do tipo 2](about-type-2-online-stores.md)                                 | Fornece uma visão geral das lojas online do tipo 2 e informações gerais sobre o plug-in e os exemplos.                                |
| [Guia de programação para lojas online do tipo 2](programming-guide-for-type-2-online-stores.md) | Explica como criar um plug-in para uma loja online tipo 2 e fornece informações sobre páginas da Web associadas e o Gerenciador de download. |
| [Referência para lojas online do tipo 2](reference-for-type-2-online-stores.md)                 | Fornece referência detalhada para as interfaces, objetos, enumerações e entradas do registro usadas por lojas online do tipo 2.              |
