# <a name="enumerations-for-type-1-online-stores"></a>Enumerações para lojas online do tipo 1

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

Os seguintes tipos de enumeração estão disponíveis para lojas online do tipo 1 usando código C++.



| Enumeração                                            | Descrição                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [WMPAccountType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpaccounttype)                   | Enumeração que define os tipos de conta para um armazenamento online.                                              |
| [WMPCallbackNotification](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpcallbacknotification) | Enumeração que define estados para uso pela função de retorno de chamada **IWMPContentPartnerCallback::Notify.** |
| [WMPPartnerNotification](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmppartnernotification)   | Enumeração que define os estados operacionais de uma loja online.                                          |
| [WMPStreamingType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpstreamingtype)               | Enumeração que especifica o tipo de mídia de streaming.                                                  |
| [WMPTaskType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype)                         | Enumeração que representa Windows Media Player de tarefas.                                             |
| [WMPTemplateSize](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptemplatesize)                 | Enumeração que representa tamanhos de modelo HTML.                                                         |
| [WMPTransactionType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptransactiontype)           | Enumeração que representa um tipo de transação.                                                          |
