# <a name="coordinates"></a>Coordenadas

Isso define as coordenadas x e y da imagem em relação ao arquivo de plano de fundo. São necessários dois inteiros positivos, separados por uma vírgula. Há dois tipos de coordenadas que você deve usar, dependendo do tipo de imagem que está definindo. A tabela a seguir mostra os valores de coordenadas permitidos para tipos de bitmap.



| Tipo de bitmap               | Valores de coordenadas                                                             |
|---------------------------|-------------------------------------------------------------------------------|
| Plano de fundo ou super       | 0, 0                                                                           |
| Desabilitado, Push ou região | O canto superior esquerdo da imagem em relação à imagem de plano de fundo, em pixels. |
