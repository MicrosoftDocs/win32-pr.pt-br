# <a name="using-the-echo-sample-dsp-plug-in"></a>Usando o plug-in DSP de exemplo Echo

Quando você cria uma compilação sem erros do plug-in do DSP de eco do Windows Media Player, você pode usá-lo no Windows Media Player para ouvir o efeito. Quando você inicia o Windows Media Player, o plug-in deve ser habilitado por padrão. Se você reproduzir áudio ou vídeo com áudio, deverá ouvir um eco de um segundo no som que é reproduzido no mesmo volume que o som original.

Você pode experimentar diferentes tempos de atraso e níveis de efeitos alterando os valores na página de propriedades de plug-in de eco. Experimente o efeito em diferentes tipos de material de origem, como música ou fala. Um tempo de atraso de 200 ms com um nível de efeito de 30 resultará em um eco com espaçamento rígido que parece o efeito usado nas gravações antecipadas e no rolo de registros. Os níveis de efeitos maiores que 50 resultarão no menor sinal original do volume do que o eco, que é chamado de efeito de eco prévio.
