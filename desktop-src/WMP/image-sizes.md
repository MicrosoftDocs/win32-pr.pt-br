# <a name="image-sizes"></a>Tamanhos de imagem

Você deve levar em consideração o tamanho da área de exibição ao criar um trabalho artístico para capas. A tabela a seguir mostra os tamanhos de área de exibição para diferentes configurações de tela.



| DPI (modo)                    | Largura em pixels | Altura em pixels |
|-------------------------------|-----------------|------------------|
| 96 (retrato)                 | 240             | 320              |
| 96 (paisagem)                | 320             | 240              |
| 192 (retrato)                | 480             | 640              |
| 192 (paisagem)               | 640             | 220              |
| 96 (quadrado)                   | 240             | 240              |
| 192 (quadrado)                  | 480             | 480              |
| 96 (retrato para Smartphone)  | 176             | 220              |
| 131 (retrato para Smartphone) | 240             | 320              |



 

> [!Note]  
> as versões anteriores à Windows Media Player 9 Series para Windows Mobile 2003 dão suporte apenas a capas de modo retrato de 96 DPI. o Windows Media Player 10 Mobile ou posterior dá suporte a capas de modo retrato de 131 DPI em dispositivos Smartphone, e esse suporte deve continuar em versões futuras do Windows Media Player Mobile.
