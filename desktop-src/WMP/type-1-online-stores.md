# <a name="type-1-online-stores"></a>Tipo 1 Lojas Online

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

Windows Media Player 11 dá suporte a lojas online do tipo 1 e do tipo 2. Os armazenamentos do tipo 1 são mais profundamente integrados à interface do usuário do Player do que os armazenamentos do tipo 2. Esta documentação, que se aplica somente a lojas do tipo 1, é dividida nas seções a seguir.



| Seção                                                                                      | Descrição                                                                                                                                      |
|----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre as Lojas Online do Tipo 1](about-type-1-online-stores.md)                                 | Discute os principais recursos das lojas online do tipo 1, incluindo catálogos baixáveis, integração de bibliotecas e páginas de descoberta.               |
| [Guia de programação para lojas online do tipo 1](programming-guide-for-type-1-online-stores.md) | Fornece diretrizes sobre como criar um plug-in de loja online do tipo 1.                                                                                     |
| [Referência para lojas online do tipo 1](reference-for-type-1-online-stores.md)                 | Fornece referência detalhada para as interfaces, objetos, enumerações, estruturas, constantes e entradas do Registro usadas por lojas online do tipo 1. |
