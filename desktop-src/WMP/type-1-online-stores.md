# <a name="type-1-online-stores"></a>Tipos 1 lojas online

> [!Note]  
> Esta seção descreve a funcionalidade projetada para uso por lojas online. Não há suporte para o uso dessa funcionalidade fora do contexto de uma loja online.

 

O Windows Media Player 11 dá suporte às lojas online tipo 1 e tipo 2. Os armazenamentos do tipo 1 são mais profundamente integrados à interface do usuário do Player do que os repositórios do tipo 2. Esta documentação, que se aplica somente a repositórios do tipo 1, é dividida nas seções a seguir.



| Seção                                                                                      | Descrição                                                                                                                                      |
|----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre o tipo 1 lojas online](about-type-1-online-stores.md)                                 | Discute os principais recursos do tipo 1 lojas online, incluindo catálogos baixáveis, integração de biblioteca e páginas de descoberta.               |
| [Guia de programação para lojas online do tipo 1](programming-guide-for-type-1-online-stores.md) | Fornece orientação sobre como criar um plug-in de loja online tipo 1.                                                                                     |
| [Referência para lojas online do tipo 1](reference-for-type-1-online-stores.md)                 | Fornece referência detalhada para as interfaces, objetos, enumerações, estruturas, constantes e entradas do registro usadas por lojas online do tipo 1. |
