# <a name="purpose-of-a-metafile-playlist"></a>Finalidade de uma lista de reprodução de metarquivo

A finalidade básica de uma playlist de metarquivo é redirecionar o conteúdo de mídia de streaming para fora dos navegadores para o Windows Media Player, pois a maioria dos navegadores tentará baixar o conteúdo em vez de transmiti-lo. Uma lista de reprodução de metarquivo fornece informações que o Windows Media Player usa para receber fluxos, como o Uniform Resource Locator (URL). A lista de reprodução do metarquivo pode fornecer o caminho para arquivos de mídia pela Internet, uma intranet ou apenas uma mídia armazenada em uma rede ou unidade local. Com o processamento Redirecionado para o Windows Media Player, você tem maior controle sobre a mídia de streaming.

As listas de reprodução de metarquivo são usadas para controlar a experiência de streaming. Ao combinar arquivos de mídia diferentes em um único fluxo de conteúdo, você pode controlar e personalizar sua apresentação.
