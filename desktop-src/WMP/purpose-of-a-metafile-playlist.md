# <a name="purpose-of-a-metafile-playlist"></a>Finalidade de uma playlist de metadados

A finalidade básica de uma playlist de metadados é redirecionar o conteúdo de mídia de streaming para fora dos navegadores Windows Media Player porque a maioria dos navegadores tentará baixar o conteúdo em vez de transmiti-lo. Uma playlist de metadados fornece informações Windows Media Player usa para receber fluxos, como a URL (Uniform Resource Locator). A playlist de metarquivo pode fornecer o caminho para arquivos de mídia pela Internet, uma intranet ou apenas mídia armazenada em uma rede ou unidade local. Com o processamento redirecionado para Windows Media Player, você tem maior controle sobre a mídia de streaming.

Playlists de metadados são usadas para controlar a experiência de streaming. Combinando arquivos de mídia diferentes em um único fluxo de conteúdo, você pode controlar e personalizar sua apresentação.
