# <a name="prevbutton"></a>PREVBUTTON

Este é um **BUTTON predefinido** com os seguintes valores padrão.

``` syntax
onclick="jscript:player.controls.previous()"
upToolTip="Previous"
cursor="system"
enabled="wmpenabled:player.controls.previous"
```

## <a name="remarks"></a>Comentários

Todas as propriedades desse BUTTON podem ser substituídos especificando-as explicitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7.0 ou posterior<br/> |
