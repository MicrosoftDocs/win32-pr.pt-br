# <a name="prevbutton"></a>PREVBUTTON

Este é um **botão** predefinido com os valores padrão a seguir.

``` syntax
onclick="jscript:player.controls.previous()"
upToolTip="Previous"
cursor="system"
enabled="wmpenabled:player.controls.previous"
```

## <a name="remarks"></a>Comentários

Todas as propriedades desse botão podem ser substituídas especificando-as explicitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
