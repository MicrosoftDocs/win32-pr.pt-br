# <a name="about-windows-media-metafiles"></a>Sobre Windows metadados de mídia

Os metadados Windows Media da Microsoft são arquivos de texto que fornecem informações sobre um fluxo de arquivos e sua apresentação. Windows Os metarquivos de mídia são baseados na sintaxe linguagem XML (XML) e podem ser codificados no formato ANSI ou UNICODE (UTF-8). Eles são feitos de vários elementos com suas marcas e atributos associados. Cada elemento em um Windows de mídia define uma determinada configuração ou ação em Windows Media Player. Alguns elementos devem estar localizados em uma posição específica no arquivo em relação a outros elementos. Alguns elementos têm marcas e atributos necessários que devem ser definidos no metadado Windows Mídia.

Windows Os pacotes de Download de Mídia fornecem uma maneira de combinar Windows Media Player bordas, informações de playlist e conteúdo multimídia em um único arquivo que pode ser baixado.
