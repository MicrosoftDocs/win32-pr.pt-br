# <a name="about-windows-media-metafiles"></a>Sobre os metaarquivos de mídia do Windows

Os metaarquivos do Microsoft Windows Media são arquivos de texto que fornecem informações sobre um fluxo de arquivos e sua apresentação. Os metaarquivos do Windows Media são baseados na sintaxe linguagem XML (XML) e podem ser codificados em formato ANSI ou UNICODE (UTF-8). Eles são compostos de vários elementos com suas marcas e atributos associados. Cada elemento em um metarquivo do Windows Media define uma configuração ou ação específica no Windows Media Player. Alguns elementos devem estar localizados em uma posição específica no arquivo em relação a outros elementos. Alguns elementos têm marcas e atributos necessários que devem ser definidos no metarquivo do Windows Media.

Os pacotes de download do Windows Media fornecem uma maneira de combinar bordas do Windows Media Player, informações de playlist e conteúdo multimídia em um único arquivo que pode ser baixado.
