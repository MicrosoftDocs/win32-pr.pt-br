# <a name="registry-settings"></a>Configurações do Registro

o Windows Media Player 9 Series ou posterior usa o registro para armazenar determinadas configurações. você pode fazer alterações nessas configurações para alterar o comportamento de Windows Media Player e o controle de Windows Media Player.

As seções a seguir detalham as configurações de registro com suporte.



| Seção                                                                                                        | Descrição                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Configurações de registro de extensão de nome de arquivo](file-name-extension-registry-settings.md)                             | Descreve como registrar uma extensão de nome de arquivo personalizado.                                                                                                               |
| [Configurações de registro de esquema personalizado](custom-scheme-registry-settings.md)                                         | Descreve como registrar um esquema personalizado.                                                                                                                            |
| [Configurações do registro de monitoramento de pasta](folder-monitoring-registry-settings.md)                                 | Descreve como registrar pastas de arquivos a serem monitoradas pelo Windows Media Player.                                                                                       |
| [Configurações do registro de porta do Firewall](firewall-port-registry-settings.md)                                         | descreve uma entrada de registro que Windows Media Player define para especificar se os firewalls devem deixar as portas abertas para o compartilhamento de biblioteca.                                   |
| [Configuração do registro de aquisição silenciosa](silent-acquisition-registry-setting.md)                                 | descreve uma entrada de registro que o Windows Media Player usa para determinar se os direitos de uso devem ser baixados automaticamente quando o usuário reproduz ou sincroniza o conteúdo protegido. |
| [Registrando dependência de aplicativo](registering-application-dependency.md)                                   | Fornece informações sobre como registrar seu aplicativo para garantir que os arquivos de tempo de execução corretos existam no computador do usuário.                                         |
| [Entradas de registro para acompanhar o progresso da instalação](registry-entries-for-tracking-installation-progress.md) | descreve as entradas do registro que os programas de instalação podem usar para acompanhar o progresso da instalação do Windows Media Player.                                                    |
