# <a name="listboxforegroundcolor"></a>LISTBOX. foregroundColor

O atributo **foregroundColor** especifica ou recupera a cor do texto no controle caixa de listagem.

``` syntax
        elementID.foregroundColor
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que contém qualquer valor de cor do Microsoft Internet Explorer. ele tem um valor padrão igual ao Windows cor do texto.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
