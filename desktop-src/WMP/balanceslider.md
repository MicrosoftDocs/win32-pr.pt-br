# <a name="balanceslider"></a>BALANCESLIDER

Este é um **controle deslizante** predefinido com os valores padrão a seguir.

``` syntax
toolTip="Balance"
max="100"
min="-100"
value="wmpprop:player.settings.balance"
value_onchange="jscript:player.settings.balance=value;"
```

## <a name="remarks"></a>Comentários

Isso cria um controle **deslizante** que define o equilíbrio de áudio. As dicas de ferramenta são localizadas. Todas as propriedades desse **controle deslizante** podem ser substituídas especificando-as explicitamente.
