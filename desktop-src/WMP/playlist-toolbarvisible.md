# <a name="playlisttoolbarvisible"></a>PLAYLIST. toolbarVisible

O atributo **toolbarVisible** especifica ou recupera um valor que indica se a barra de ferramentas de playlist é exibida.

``` syntax
        elementID.toolbarVisible
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação. O valor padrão é false.

## <a name="remarks"></a>Comentários

A barra de ferramentas da lista de reprodução fornece elementos de interface do usuário que facilitam o trabalho com listas de reprodução.
