# <a name="eventtoelement"></a>evento. ToElement

O atributo **ToElement** recupera o elemento ao qual o foco do teclado foi movido. Este atributo só se aplica ao evento **onblur** ; para todos os outros eventos, seu valor é **NULL**.

``` syntax
event.toElement
```

## <a name="possible-values"></a>Valores possíveis

Este atributo é um objeto somente leitura.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
