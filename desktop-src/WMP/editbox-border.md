# <a name="editboxborder"></a>Área de edição. Border

O atributo **Border** especifica ou recupera um valor que indica se o controle da caixa de edição tem uma borda. Só pode ser definido em tempo de design.

``` syntax
        elementID.border
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** especificado em tempo de design e somente leitura depois. Ele tem um valor padrão de true.

## <a name="remarks"></a>Comentários

A borda aparece como uma borda de baixo pixel em largura de dois pixels em torno do controle caixa de edição com cores do sistema.
