# <a name="playlistcheckboxesvisible"></a>PLAYLIST. checkboxesVisible

O atributo **checkboxesVisible** especifica ou recupera um valor que indica se as caixas de seleção estão visíveis.

``` syntax
        elementID.checkboxesVisible
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **booliano** de leitura/gravação.



| Valor | Descrição                           |
|-------|---------------------------------------|
| true  | As caixas de seleção estão visíveis.              |
| false | Padrão. As caixas de seleção não estão visíveis. |



 

## <a name="remarks"></a>Comentários

As caixas de seleção aparecem na coluna da extrema esquerda.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
