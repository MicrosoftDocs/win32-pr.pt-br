# <a name="creating-the-disabled-file"></a>Criando o arquivo desabilitado

O arquivo desabilitado é semelhante ao arquivo enviado, exceto que outra cor (vermelho) é usada para mostrar quando algo está desabilitado.

A imagem a seguir é o arquivo desabilitado.

![arquivo desabilitado](images/ceswmdis.png)

Observe que arquivos como este podem ser cortados nos lados direito e inferior. O tamanho real do bitmap não é importante, apenas que os elementos se alinham com o bitmap de região para teste de clique, conforme medido nos cantos superiores à esquerda (incluindo deslocamentos).
