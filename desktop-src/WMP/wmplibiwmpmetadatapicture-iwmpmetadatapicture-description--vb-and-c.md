# <a name="iwmpmetadatapicturedescription-property"></a>IWMPMetadataPicture: Propriedade descrição de:D

A `Description` propriedade Obtém a descrição da imagem representada pelo atributo de metadados.

## <a name="syntax"></a>Sintaxe


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## <a name="property-value"></a>Valor da propriedade

Um **System. String** que é a descrição da imagem.

## <a name="remarks"></a>Comentários

Antes de usar essa propriedade, você deve ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
