# <a name="listboxsetselectedstate"></a>LISTBOX. setselecionostate

O método **Setselecionostate** seleciona ou desmarca o item com o índice especificado.

``` syntax
        elementID.setSelectedState(index, selected)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*index*
</dt> <dd>

**Número** (**longo**) que contém o índice do item a ser selecionado ou desmarcado.

</dd> <dt>

<span id="selected"></span><span id="SELECTED"></span>*Selecione*
</dt> <dd>

Valor **booliano** que indica se o item deve ser selecionado (true) ou não selecionado (false).

</dd> </dl>

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Isso é usado para selecionar ou desmarcar várias linhas.
