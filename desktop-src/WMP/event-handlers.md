# <a name="event-handlers"></a>Manipuladores de eventos

O Microsoft JScript é usado para processar eventos no arquivo de definição de capa. Consulte [manipulação de eventos](handling-events.md) para obter mais informações sobre manipuladores de eventos.

Você pode ter mais de uma linha de código em um manipulador de eventos, mas deve-se ter cuidado para não exceder o comprimento da linha que o JScript permite. Separe as linhas por ponto e vírgula.


```C++
onclick = "JScript: player.URL = 'https://proseware.com/cool.wma' ; myText.value = 'Playing'; "

```
