# <a name="editboxtextlimit"></a>DefaultLimit

O atributo **textlimit** especifica ou recupera o número máximo de caracteres que o usuário pode digitar no controle caixa de edição.

``` syntax
        elementID.textLimit
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**) com um valor padrão de 0, o que indica que não há limite.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
