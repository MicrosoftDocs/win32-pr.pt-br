# <a name="editboxtextlimit"></a>EDITBOX.textLimit

O **atributo textLimit** especifica ou recupera o número máximo de caracteres que o usuário pode digitar no controle de caixa de edição.

``` syntax
        elementID.textLimit
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um  número de leitura/gravação (**longo**) com um valor padrão de 0, que indica nenhum limite.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versão<br/> | Windows Media Player para Windows XP ou posterior<br/> |
