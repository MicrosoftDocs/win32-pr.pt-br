# <a name="ambientattributesid"></a>AmbientAttributes.id

O atributo **ID** especifica ou recupera o identificador de um controle. Esse atributo dará suporte a qualquer nome permitido pelo Microsoft Visual Basic for Applications. Só pode ser definido em tempo de design.

``` syntax
        elementID.id
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** especificada em tempo de design e somente leitura depois. Ele tem um valor padrão de

Número de \_ *ElementType* \_  sem nome

em que *ElementType* é o nome do tipo de controle, e *num* é um número que corresponde à posição do controle na sequência de todos os controles sem nome no tema. Esse número é redefinido para zero para cada novo tema.
