# <a name="ambientattributesid"></a>AmbientAttributes.id

O **atributo id** especifica ou recupera o identificador de um controle . Esse atributo dará suporte a qualquer nome permitido pela Microsoft Visual Basic for Applications. Só pode ser definido em tempo de design.

``` syntax
        elementID.id
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **Cadeia de** caracteres especificada em tempo de design e somente leitura depois disso. Ele tem um valor padrão de

Elementtype num \_ *sem* \_ *nome*

em *que elementtype* é o nome do tipo de controle e *num* é um número correspondente à posição do controle na sequência de todos os controles sem nome no Tema. Esse número é redefinido como zero para cada novo Tema.
