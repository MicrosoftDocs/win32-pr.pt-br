# <a name="metadata-area-plug-ins-deprecated"></a>Plug-ins de área de metadados (preteridos)

Esta página documenta um recurso que pode não estar disponível em versões futuras do Windows Media Player e no SDK do Windows Media Player.

Os plug-ins de interface do usuário da área de metadados são mostrados na área de metadados do painel **agora em execução** à direita da área Visual principal e do painel de ferramentas e abaixo da exibição de playlist (se estiver habilitada). Somente um plug-in de área de metadados pode ser habilitado por vez.

Plug-ins de área de metadados são úteis para exibir informações concisas sobre o item de mídia em execução no momento, controles simples ou hiperlinks para informações adicionais.
