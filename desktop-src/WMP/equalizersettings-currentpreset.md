# <a name="equalizersettingscurrentpreset"></a>EQUALIZERSETTINGS.currentPreset

O atributo **currentPreset** especifica ou recupera o índice da predefinição atual.

``` syntax
        elementID.currentPreset
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**).

## <a name="remarks"></a>Comentários

Se esse atributo não for especificado, o valor anterior será retido.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
