# <a name="stringcollectioncount"></a>Cadeia de caracteres. Count

A propriedade **Count** recupera o número de itens na coleção de cadeias de caracteres.

## <a name="syntax"></a>Syntax

*Player*. *mediacollection*. **getAttributeStringCollection**( *atributo*, *MediaType*). **contagem** de

## <a name="possible-values"></a>Valores possíveis

Essa propriedade é um **número** somente leitura (**Long**).

## <a name="remarks"></a>Comentários

Para recuperar o valor dessa propriedade, é necessário ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
