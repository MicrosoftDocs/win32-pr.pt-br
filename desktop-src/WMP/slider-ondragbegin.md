# <a name="sliderondragbegin"></a>SLIDER. onDragBegin

O manipulador de eventos **onDragBegin** manipula um evento que ocorre quando o usuário clica e mantém o botão esquerdo do mouse pressionado e começa a arrastar o mouse.

``` syntax
onDragBegin
```

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
