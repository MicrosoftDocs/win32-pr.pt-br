# <a name="dropdownplaylist"></a>DROPDOWNPLAYLIST

Este é um elemento de **playlist** predefinido com os seguintes valores padrão.

``` syntax
playlistItemsVisible="false"
```

## <a name="remarks"></a>Comentários

Isso criará uma lista de **reprodução** de lista suspensa simples sem itens exibidos. Todas as propriedades desta **playlist** podem ser substituídas especificando-as explicitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|----------------------------------------------|
| Versão<br/> | Windows Media Player 7,0 ou posterior<br/> |
