# <a name="viewmaxheight"></a>Exibir. maxHeight

O atributo **MaxHeight** especifica ou recupera a altura máxima em pixels da **exibição** ao redimensionar.

``` syntax
        elementID.maxHeight
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é um **número** de leitura/gravação (**longo**) com um valor igual ou maior que zero. Ele tem um valor padrão de zero, o que significa que não há nenhuma restrição na altura máxima da **exibição**.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
