# <a name="playercontrols"></a>Player. controles

A propriedade **Controls** recupera o objeto Controls.

## <a name="syntax"></a>Syntax

## <a name="possible-values"></a>Valores possíveis

Esta propriedade é um objeto de **controles** somente leitura.
