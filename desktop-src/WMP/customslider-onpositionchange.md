# <a name="customslideronpositionchange"></a>CUSTOMSLIDER.onPositionChange

O manipulador de eventos **onPositionChange** manipula um evento que ocorre quando a posição do controle deslizante é alterada como resultado do usuário clicar ou arrastar.

``` syntax
onPositionChange
```

## <a name="remarks"></a>Comentários

Se a posição do controle deslizante personalizado for alterada como resultado do atributo de **valor** que está sendo modificado no script, esse evento não será acionado. Para acomodar essa possibilidade, implemente o valor do manipulador de eventos **\_ onChange** .
