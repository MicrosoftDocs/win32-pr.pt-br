# <a name="onkeyup"></a>OnKeyUp

O manipulador de eventos **onkeyup** manipula um evento que ocorre quando uma chave é liberada.

``` syntax
onkeyup
```

## <a name="remarks"></a>Comentários

Esse manipulador de eventos não é aplicável ao elemento **playlist** , ao elemento **Effects** ou ao elemento **Video** quando o *vídeo*. **sem janela** é false.
