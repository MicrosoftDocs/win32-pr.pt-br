# <a name="background-file"></a>Arquivo em segundo plano

Cada capa precisa de um arquivo em segundo plano, que contém a imagem que o usuário verá quando a capa é exibida pela primeira vez. Esse arquivo é definido como o arquivo de plano de fundo no arquivo de definição de capa.

A imagem a seguir é um arquivo de plano de fundo típico.

![arquivo em segundo plano](images/cesdkbak.png)

Esta é a imagem de plano de fundo que os usuários verão a princípio. No entanto, eles também verão as duas imagens de Thumb nos locais apropriados no trackbars para Seek (acima) e o volume (abaixo). Além disso, todos os botões desabilitados serão exibidos usando suas imagens desabilitadas.
