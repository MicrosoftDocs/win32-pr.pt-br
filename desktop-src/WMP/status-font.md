# <a name="status-font"></a>Fonte de status

Você deve definir a fonte usada pela exibição de status que deseja usar. A fonte é definida por três valores, separados por vírgulas, representando a face, o tamanho e o peso da fonte.

**Valores de face de tipo**

Qualquer nome de face de tipo poderá ser usado se for provável que ele seja instalado no computador do usuário. Se uma face de tipo não for encontrada no computador, uma alternativa será selecionada pelo sistema operacional. A tabela a seguir mostra as face de tipo que geralmente são encontradas Windows dispositivos baseados no Mobile 2003.



| Face de tipos       | Descrição              |
|----------------|--------------------------|
| Tahoma         | Uma face de tipo sans-serif.   |
| Lucida Console | Uma face de tipo serif quadrada. |



 

**Valores de tamanho**

Esse é o tamanho da face de tipo em pontos. Qualquer valor inteiro positivo é válido, embora sejam recomendados números entre 10 e 18. Tamanhos menores que 10 podem ser difíceis de ler e tamanhos acima de 18 podem não deixar espaço suficiente para exibir mais de algumas letras por vez.

**Valores de peso**

Os únicos valores permitidos são mostrados na tabela a seguir.



| Valor | Descrição |
|-------|-------------|
| B     | Negrito        |
| N     | Normal      |
