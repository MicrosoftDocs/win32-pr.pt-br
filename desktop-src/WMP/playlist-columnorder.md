# <a name="playlistcolumnorder"></a>LISTA de reprodução. columnOrder

O atributo **columnOrder** especifica ou recupera a ordem das colunas da lista de reprodução.

``` syntax
        elementID.columnOrder
```

## <a name="possible-values"></a>Valores possíveis

Esse atributo é uma **cadeia de caracteres** de leitura/gravação que especifica uma lista delimitada por ponto-e-vírgula de índices de coluna de playlist. O valor padrão é "0; 1; 2; 3". Os espaços e vírgulas à esquerda e à direita não devem estar presentes.

|--------------------|------------------------------------------------------|
