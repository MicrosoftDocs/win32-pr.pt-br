# <a name="object-model-migration-guide"></a>Guia de migração do modelo de objeto

o Windows Media Player 7 introduziu um novo modelo de objeto que oferece um novo conjunto avançado de funcionalidades que você pode usar para aprimorar suas páginas da web. As versões subsequentes estenderam o modelo de objeto versão 7 com propriedades, métodos e eventos adicionais. no entanto, o novo modelo de objeto difere substancialmente do modelo de objeto Windows Media Player 6,4, portanto, você precisa entender como atualizar seu código existente se quiser dar suporte a Windows Media Player 7 e posteriores em seus projetos.

para obter um exemplo que demonstra como detectar a versão do Windows Media Player do usuário e o navegador atual, consulte o exemplo chamado "detecção" instalado com esse SDK. Para obter mais informações sobre exemplos, consulte [exemplos](samples.md).

este guia refere-se ao novo modelo de objeto como o modelo de objeto Windows Media Player 7 ou posterior. para obter detalhes sobre qual funcionalidade está disponível em cada versão do Windows Media Player, consulte [referência de modelo de objeto para scripts](object-model-reference-for-scripting.md).
