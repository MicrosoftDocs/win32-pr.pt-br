# <a name="iwmpplaylistname-property"></a>Propriedade IWMPPlaylist:: Name

A propriedade **Name** Obtém ou define o nome da lista de reprodução.

## <a name="syntax"></a>Syntax


```CSharp
public System.String name {get; set;}
```


```VB

Public Property name As System.String
```





## <a name="property-value"></a>Valor da propriedade

Um **System. String** que é o nome da playlist.

## <a name="remarks"></a>Comentários

Antes de usar essa propriedade, você deve ter acesso de leitura à biblioteca. Para obter mais informações, consulte [acesso à biblioteca](library-access.md).
