# <a name="effectsnext"></a>EFEITOS. Next

O **próximo** método exibirá a próxima predefinição de visualização, movendo para a próxima visualização, se necessário.

``` syntax
        elementID.next()
```

## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor Retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Se a predefinição atual for a última na série de todas as visualizações disponíveis, a primeira predefinição da primeira visualização será feita atual.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versão<br/> | Windows Media Player versão 7,0 ou posterior<br/> |
