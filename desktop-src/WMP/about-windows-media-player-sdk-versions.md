# <a name="about-windows-media-player-sdk-versions"></a>sobre as versões Windows Media Player SDK

a versão atual do SDK do Windows Media Player é a versão 12. o sdk do Windows Media Player 12 se baseia nos recursos introduzidos nas versões anteriores do sdk do Windows Media Player, começando com a versão 7. as informações contidas nesta documentação incluem os recursos dos seguintes Windows Media Player versões do SDK:

-   SDK do Windows Media Player 7
-   SDK do Windows Media Player 7,1
-   SDK do Windows Media Player 9 Series
-   SDK do Windows Media Player 10
-   SDK do Windows Media Player 11
-   SDK do Windows Media Player 12

Cada versão do SDK está associada a uma versão do Windows Media Player. por exemplo, o SDK do Windows Media Player 10 introduziu novos recursos com suporte do Windows Media Player 10. observe que nenhuma atualização para o SDK Windows Media Player foi lançada com o Windows Media Player para Windows XP.

Em toda a documentação, os requisitos de controle de versão são fornecidos para cada recurso, quando apropriado. se nenhuma informação de versão específica for fornecida, o recurso foi introduzido no SDK do Windows Media Player 7 e terá suporte em todas as versões subsequentes.

para obter informações sobre como atualizar projetos com base no SDK do Windows Media Player 6,4 para usar o modelo de objeto do Windows Media Player 7 ou posterior, consulte [guia de migração de modelo de objeto](object-model-migration-guide.md).
