# <a name="dvd-object"></a>Objeto de DVD

O objeto **DVD** fornece propriedades e métodos para trabalhar com DVDs.

O objeto **DVD** dá suporte às propriedades a seguir.



| Propriedade                           | Descrição                                                                                        |
|------------------------------------|----------------------------------------------------------------------------------------------------|
| [controlador](dvd-domain.md)           | Recupera o domínio atual do DVD.                                                           |
| [isAvailable](dvd-isavailable.md) | Recupera se um tipo especificado de informação está disponível ou se uma determinada ação pode ser executada. |



 

O objeto **DVD** dá suporte aos seguintes métodos.



| Método                         | Descrição                                                                                          |
|--------------------------------|------------------------------------------------------------------------------------------------------|
| [Voltar](dvd-back.md)           | Altera a exibição de um submenu para seu menu pai.                                               |
| [Volte](dvd-resume.md)       | Altera para o modo de reprodução no modo de menu, retomando na mesma posição em que o menu foi invocado. |
| [titleMenu](dvd-titlemenu.md) | Interrompe a reprodução e exibe o menu de título.                                                          |
| [topMenu](dvd-topmenu.md)     | Interrompe a reprodução e exibe o menu raiz.                                                           |



 

O objeto **DVD** é acessado por meio da propriedade a seguir.



| Objeto                      | Propriedade              |
|-----------------------------|-----------------------|
| [Jogador](player-object.md) | [DVD](player-dvd.md) |
