# <a name="wmpvideo"></a>WMPVIDEO

Este é um elemento de **vídeo** predefinido com os seguintes valores padrão.

``` syntax
backgroundColor="black"
horizontalAlignment="stretch"
verticalAlignment="stretch"
```

## <a name="remarks"></a>Comentários

Isso criará um elemento de **vídeo** que ampliará a janela de vídeo quando a capa for redimensionada. O vídeo digital exibido nesta janela será alongado para se ajustar ao espaço disponível ou será reduzido ou reduzido de acordo com as configurações no menu **Exibir** do Player, em **tamanho do vídeo**. O menu **Exibir** é exibido no modo completo do Player ou quando o atributo **View. TitleBar** é definido como true em uma capa.

Todas as propriedades deste elemento de **vídeo** podem ser substituídas especificando-as explicitamente.
