# <a name="status-alignment"></a>Alinhamento de status

Você deve definir o alinhamento da exibição de status que deseja usar. A tabela a seguir mostra os valores válidos.



| Valor  | Descrição                                                                       |
|--------|-----------------------------------------------------------------------------------|
| Esquerda   | O texto é alinhado à esquerda (alinhado com a margem esquerda da exibição de status).   |
| Centro | O texto é centralizado na exibição de status.                                       |
| Direita  | O texto é alinhado à direita (alinhado com a margem direita da exibição de status). |
