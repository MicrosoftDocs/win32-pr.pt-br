# <a name="radioformat-attribute"></a>Atributo RadioFormat

O atributo **RadioFormat** é uma descrição do tipo de conteúdo fornecido pela estação de rádio.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de rádio](radio-item-attributes.md)

## <a name="remarks"></a>Comentários
