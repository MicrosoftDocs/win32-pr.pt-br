# <a name="status-location"></a>Local do status

Você deve definir o local da exibição de status que deseja usar. Você deve fornecer quatro inteiros positivos, separados por vírgulas, que definem a esquerda, a parte superior, a largura e a altura da exibição de status, em pixels, em relação à imagem de plano de fundo.


```C++
45,193,120,13 

```
