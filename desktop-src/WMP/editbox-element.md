# <a name="editbox-element"></a>Elemento admy

O **elemento** adenter fornece uma maneira para os usuários digitarem texto em uma capa.

O **elemento** admy oferece suporte aos seguintes atributos.



| Atributo                                      | Descrição                                                                                                   |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [backgroundColor](editbox-backgroundcolor.md) | Especifica ou recupera a cor do plano de fundo do controle caixa de edição.                                         |
| [borda](editbox-border.md)                   | Especifica ou recupera um valor que indica se o controle da caixa de edição tem uma borda.                          |
| [editstyle](editbox-editstyle.md)             | Especifica ou recupera o estilo do controle caixa de edição.                                                     |
| [fontFace](editbox-fontface.md)               | Especifica ou recupera a fonte usada para texto no controle da caixa de edição.                                        |
| [fontSize](editbox-fontsize.md)               | Especifica ou recupera o tamanho da fonte para o controle caixa de edição.                                                |
| [fontStyle](editbox-fontstyle.md)             | Especifica ou recupera o estilo da fonte para o controle caixa de edição.                                               |
| [foregroundColor](editbox-foregroundcolor.md) | Especifica ou recupera a cor do texto no controle da caixa de edição.                                                |
| [elabora](editbox-justification.md)     | Especifica ou recupera o alinhamento do texto dentro do controle da caixa de edição.                                 |
| [lineCount](editbox-linecount.md)             | Recupera o número de linhas no controle da caixa de edição.                                                        |
| [readOnly](editbox-readonly.md)               | Especifica ou recupera um valor que indica se o texto no controle da caixa de edição é somente leitura ou pode ser editado. |
| [textlimit](editbox-textlimit.md)             | Especifica ou recupera o número máximo de caracteres que o usuário pode digitar no controle caixa de edição.       |
| [value](editbox-value.md)                     | Especifica ou recupera o texto que é exibido no controle da caixa de edição.                                    |
| [Quebra](editbox-wordwrap.md)               | Especifica ou recupera um valor que indica se a quebra automática de texto está habilitada no controle da caixa de edição.               |



 

O **elemento** admy oferece suporte aos métodos a seguir.



| Método                                             | Descrição                                                                                         |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [getline](editbox-getline.md)                     | Recupera o texto da linha com o índice especificado.                                           |
| [getLineFromChar](editbox-getlinefromchar.md)     | Recupera o índice de linha para o índice de caracteres especificado.                                         |
| [getLineIndex](editbox-getlineindex.md)           | Recupera o índice de caracteres para o índice de linha especificado.                                         |
| [getSelectionEnd](editbox-getselectionend.md)     | Recupera a posição final do texto selecionado no controle caixa de edição.                         |
| [getSelectionStart](editbox-getselectionstart.md) | Recupera a posição inicial do texto selecionado no controle da caixa de edição.                       |
| [replaceSelection](editbox-replaceselection.md)   | Substitui a seleção atual pelo texto especificado.                                             |
| [setseleções](editbox-setselection.md)           | Seleciona o texto no controle da caixa de edição do índice inicial especificado para o índice final especificado. |



 

O **elemento** admy oferece suporte aos atributos de ambiente e pode implementar todos os manipuladores de eventos de ambiente com a exceção de **onclick**. Para obter mais informações, consulte [atributos de ambiente](ambient-attributes.md) e [manipuladores de eventos de ambiente](ambient-event-handlers.md).

> [!Note]  
> Esse elemento requer o Windows Media Player para Windows XP ou posterior.
