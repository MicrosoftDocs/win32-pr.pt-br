# <a name="windows-media-player-custom-visualizations"></a>Windows Media Player Visualizações personalizadas

O Microsoft Windows Media Player fornece visualizações que permitem que o usuário veja imagens visuais sincronizadas com o som do conteúdo de mídia enquanto ele é reproduzido. Várias visualizações padrão são incluídas com Windows Media Player, incluindo Picos, Barras e DotPlane. Esta seção do SDK fornece informações de programação que permitirão que você crie suas próprias visualizações.

A documentação de visualização personalizada é dividida em três seções:



| Seção                                                                                      | Descrição                                                                                                                                                          |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre visualizações personalizadas](about-custom-visualizations.md)                               | Discute a arquitetura de visualizações personalizadas em termos abstratos. Você deve ler esta seção para entender como as visualizações se relacionam com Windows Media Player. |
| [Guia de Programação de Visualização Personalizada](custom-visualization-programming-guide.md)         | Explica o que você precisa fazer para criar uma visualização personalizada. Esta seção contém dicas e dicas úteis que ajudarão você a criar visualizações interessantes.     |
| [Referência de programação de visualização personalizada](custom-visualization-programming-reference.md) | Fornece uma referência para a interface que você deve implementar, bem como outras informações de referência técnica.                                                          |
