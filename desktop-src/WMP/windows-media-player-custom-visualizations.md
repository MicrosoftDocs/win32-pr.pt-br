# <a name="windows-media-player-custom-visualizations"></a>Visualizações personalizadas do Windows Media Player

O Microsoft Windows Media Player fornece visualizações que permitem ao usuário ver imagens visuais que são sincronizadas com o som do conteúdo de mídia conforme elas são reproduzidas. Várias visualizações padrão estão incluídas no Windows Media Player, incluindo picos, barras e DotPlane. Esta seção do SDK fornece informações de programação que lhe permitirão criar suas próprias visualizações.

A documentação de visualização personalizada é dividida em três seções:



| Seção                                                                                      | Descrição                                                                                                                                                          |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Sobre visualizações personalizadas](about-custom-visualizations.md)                               | Discute a arquitetura de visualizações personalizadas em termos abstratos. Você deve ler esta seção para entender como as visualizações estão relacionadas ao Windows Media Player. |
| [Guia de programação de visualização personalizada](custom-visualization-programming-guide.md)         | Explica o que você precisa fazer para criar uma visualização personalizada. Esta seção contém dicas úteis e dicas que o ajudarão a criar visualizações interessantes.     |
| [Referência de programação de visualização personalizada](custom-visualization-programming-reference.md) | Fornece uma referência para a interface que você deve implementar, bem como outras informações de referência técnica.                                                          |
