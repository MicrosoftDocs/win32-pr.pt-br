# <a name="skin-files"></a>Arquivos de capa

Quando você vê uma capa, está vendo apenas a superfície. Atrás da superfície estão vários arquivos que comem a capa completa. Tecnicamente, uma capa é um grupo de arquivos, com cada arquivo contendo um tipo específico de informações. Aqui estão os tipos de arquivos que você pode usar para fazer uma capa completa.



| Tipo de arquivo                                        | Descrição                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Arquivo de definição de capa](skin-definition-file.md) | O arquivo mestre que define como os outros arquivos serão usados. Esse é um arquivo de texto e tem uma extensão de nome de arquivo .wms.                                                    |
| [Arquivos de arte](art-files.md)                       | Arquivos de arte que contêm os elementos gráficos da sua capa. Isso inclui arquivos bitmap, GIF, JPEG e PNG.                                                                 |
| [JScript Arquivos](jscript-files.md)               | Se você quiser criar respostas mais complicadas para eventos, poderá criar arquivos de script usando o Microsoft JScript. Esses são arquivos de texto e têm uma extensão .js nome de arquivo. |
