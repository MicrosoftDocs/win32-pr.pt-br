# <a name="videobitrate-attribute"></a>Atributo VideoBitrate

O atributo **VideoBitrate** é a taxa de bits para o fluxo de vídeo em um arquivo de vídeo, em bits por segundo.

## <a name="applies-to"></a>Aplica-se A

-   [Itens de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentários

Para determinar se você pode alterar o valor desse atributo, use o método [Media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------|
| Versão<br/> | Windows Media Player 10 ou posterior<br/> |
