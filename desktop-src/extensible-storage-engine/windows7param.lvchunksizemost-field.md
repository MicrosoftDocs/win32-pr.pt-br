# <a name="windows7paramlvchunksizemost-field"></a>Campo Windows7Param. LVChunkSizeMost

Esse parâmetro é usado para recuperar o tamanho da parte de dados de valor longo (BLOB). A configuração e a recuperação de dados em múltiplos desse tamanho aumenta a eficiência.
