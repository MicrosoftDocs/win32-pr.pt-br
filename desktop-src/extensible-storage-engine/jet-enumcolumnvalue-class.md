# <a name="jet_enumcolumnvalue-class"></a>JET_ENUMCOLUMNVALUE classe

Enumera os valores de coluna de um registro usando a função JetEnumerateColumns. [JetEnumerateColumns(JET_SESID, JET_TABLEID, Int32, \[ \] , Int32, \[ \] , JET_PFNREALLOC, IntPtr, Int32, EnumerateColumnsGrbit)](./api.jetenumeratecolumns-method.md) retorna uma matriz de estruturas JET_ENUMCOLUMNVALUE. A matriz é retornada na memória que foi alocada usando o retorno de chamada fornecido para essa função.
