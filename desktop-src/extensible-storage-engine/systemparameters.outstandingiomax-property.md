# <a name="systemparametersoutstandingiomax-property"></a>Propriedade SystemParameters. OutstandingIOMax

Esse parâmetro controla quantas e/SS de arquivo de banco de dados podem ser enfileiradas por disco no sistema operacional do host ao mesmo tempo. Um valor maior para esse parâmetro pode ajudar significativamente o desempenho de um aplicativo de banco de dados grande.
