# <a name="vistaparamkeymost-field"></a>Campo VistaParam. keymost

Esse parâmetro somente leitura indica o comprimento máximo de chave de índice permitido que pode ser selecionado para o tamanho da página do banco de dados atual (conforme configurado por [DatabasePageSize](./jet-param-enumeration.md)).
