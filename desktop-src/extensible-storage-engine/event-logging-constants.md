# <a name="event-logging-constants"></a>Constantes de registro em log de eventos


_**Aplica-se a:** Windows | Windows Servidor_

## <a name="event-logging-constants"></a>Constantes de registro em log de eventos

As constantes a seguir indicam o nível de detalhes para mensagens de log de eventos para o [JET_ParamEventLoggingLevel](./event-log-parameters.md) do sistema.


| <p>Constante/valor</p> | <p>Descrição</p> | 
|-----------------------|--------------------|
| <p>JET_EventLoggingDisable<br />0</p> | <p>Desabilita o log de eventos.</p> | 
| <p>JET_EventLoggingLevelMax<br />100</p> | <p>Define o nível máximo a ser usado para logs de eventos, que atualmente tem 100 caracteres.</p> | 



### <a name="requirements"></a>Requisitos


| Requisito | Valor |
|------------|----------|
| <p><strong>Cliente</strong></p> | <p>Requer Windows Vista, Windows XP ou Windows 2000 Professional.</p> | 
| <p><strong>Servidor</strong></p> | <p>Requer Windows Server 2008, Windows Server 2003 ou Windows 2000 Server.</p> | 
| <p><strong>Cabeçalho</strong></p> | <p>Declarado em Esent.h.</p> |
