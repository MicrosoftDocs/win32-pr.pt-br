# <a name="event-logging-constants"></a>Constantes de log de eventos


_**Aplica-se a:** Windows | Windows Server_

## <a name="event-logging-constants"></a>Constantes de log de eventos

As constantes a seguir indicam o nível de detalhe para mensagens de log de eventos para o [JET_ParamEventLoggingLevel](./event-log-parameters.md) parâmetro do sistema.

<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th><p>Constante/valor</p></th>
<th><p>Descrição</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><p>JET_EventLoggingDisable<br />
0</p></td>
<td><p>Desabilita o log de eventos.</p></td>
</tr>
<tr class="even">
<td><p>JET_EventLoggingLevelMax<br />
100</p></td>
<td><p>Define o nível máximo a ser usado para logs de eventos, que são, no momento, 100 caracteres.</p></td>
</tr>
</tbody>
</table>
