# <a name="event-logging-constants"></a>Constantes de log de eventos


_**Aplica-se a:** Windows | Windows Servidor_

## <a name="event-logging-constants"></a>Constantes de log de eventos

As constantes a seguir indicam o nível de detalhe para mensagens de log de eventos para o [JET_ParamEventLoggingLevel](./event-log-parameters.md) parâmetro do sistema.


| <p>Constante/valor</p> | <p>Descrição</p> | 
|-----------------------|--------------------|
| <p>JET_EventLoggingDisable<br />0</p> | <p>Desabilita o log de eventos.</p> | 
| <p>JET_EventLoggingLevelMax<br />100</p> | <p>Define o nível máximo a ser usado para logs de eventos, que são, no momento, 100 caracteres.</p> |
