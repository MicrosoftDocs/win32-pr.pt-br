# <a name="instanceparameterscircularlog-property"></a>Propriedade instanceparameters. CircularLog

Obtém ou define um valor que indica se o log circular está ativado. Quando o log circular está desativado, todos os arquivos de log de transações gerados são mantidos no disco até que não sejam mais necessários porque um backup completo do banco de dados foi executado. Quando o log circular está ativado, somente os arquivos de log de transações mais jovens do que o ponto de verificação atual são mantidos no disco. O benefício desse modo é que os backups não são necessários para desativar arquivos de log de transação antigos.
