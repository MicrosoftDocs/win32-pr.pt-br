# <a name="apijetdupcursor-method"></a>Método API. JetDupCursor

Duplica um cursor Open e retorna um identificador para o cursor duplicado. Se o cursor que foi duplicado era um cursor somente leitura, o cursor duplicado também é um cursor somente leitura. Qualquer estado relacionado à construção de uma chave de pesquisa ou à atualização de um registro não é copiado para o cursor duplicado. Além disso, o local do cursor original não é duplicado no cursor duplicado. O cursor duplicado sempre é aberto no índice clusterizado e seu local está sempre na primeira linha da tabela.
