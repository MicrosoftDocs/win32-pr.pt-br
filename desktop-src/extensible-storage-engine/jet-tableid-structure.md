# <a name="jet_tableid-structure"></a>Estrutura de JET_TABLEID

Um JET_TABLEID contém um identificador para o cursor do banco de dados a ser usado para uma chamada para a API do JET. Um cursor só pode ser usado com a sessão que foi usada para abrir esse cursor.
