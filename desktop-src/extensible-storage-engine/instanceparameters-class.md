# <a name="instanceparameters-class"></a>Classe instanceparameters

Essa classe fornece propriedades para definir e obter parâmetros do sistema em uma instância de ESENT. Essa classe fornece propriedades estáticas para definir e obter parâmetros de sistema de ESENT por instância.
