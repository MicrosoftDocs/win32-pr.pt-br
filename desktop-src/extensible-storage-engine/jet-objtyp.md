# <a name="jet_objtyp"></a>JET_OBJTYP


_**Aplica-se a:** Windows | Windows Servidor_

## <a name="jet_objtyp"></a>JET_OBJTYP

O **JET_OBJTYP** de constantes representa o tipo de um objeto de banco de dados. Atualmente, há suporte apenas para tabelas.
