# <a name="jet_conditionalcolumn-class"></a>Classe JET_CONDITIONALCOLUMN

Define como a indexação condicional é executada para um determinado índice. Um índice condicional contém uma entrada de índice somente para as linhas que correspondem à condição especificada. No entanto, a coluna condicional não faz parte da chave do índice, ela controla apenas a presença da entrada de índice.
