# <a name="apijetgetcursorinfo-method"></a>Método API. JetGetCursorInfo

Determine se uma atualização do registro atual de um cursor resultará em um conflito de gravação, com base no status de atualização atual do registro. É possível que um conflito de gravação será retornado, mesmo se JetGetCursorInfo retornar com êxito. Porque outra sessão pode atualizar o registro antes que a sessão atual seja capaz de atualizar o mesmo registro.
