# <a name="apijetsetindexrange-method"></a>Método API. JetSetIndexRange

Limita temporariamente o conjunto de entradas de índice que o cursor pode percorrer usando [JetMove (JET_SESID, JET_TABLEID, Int32, MoveGrbit)](./api.jetmove-method-jet-sesid-jet-tableid-int32-movegrbit-.md) para aqueles que começam da entrada de índice atual e terminam na entrada de índice que corresponde aos critérios de pesquisa especificados pela chave de pesquisa no cursor e nos critérios de associação especificados. Uma chave de pesquisa deve ter sido construída anteriormente usando [JetMakeKey (JET_SESID, JET_TABLEID, \[ \] , Int32, MakeKeyGrbit)](./api.jetmakekey-method.md). Consulte também [TrySetIndexRange (JET_SESID, JET_TABLEID, SetIndexRangeGrbit)](./api.trysetindexrange-method.md).
