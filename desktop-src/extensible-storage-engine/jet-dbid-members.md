# <a name="jet_dbid-members"></a>JET_DBID membros

Incluir membros protegidos  
Incluir membros herdados  

Um JET_DBID contém o handle para o banco de dados. Um handle de banco de dados é usado para gerenciar o esquema de um banco de dados. Ele também pode ser usado para gerenciar as tabelas dentro desse banco de dados.

O [JET_DBID](./jet-dbid-structure.md) tipo expõe os membros a seguir.
