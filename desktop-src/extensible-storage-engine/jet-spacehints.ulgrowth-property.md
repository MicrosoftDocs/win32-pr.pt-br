# <a name="jet_spacehintsulgrowth-property"></a>Propriedade JET_SPACEHINTS. ulGrowth

Obtém ou define o crescimento percentual do último crescimento ou do tamanho inicial (possivelmente arredondado para o tamanho de alocação mais próximo do JET nativo). Os valores válidos são 0 e \[ 100, 50000).
