# <a name="error-handling-constants"></a>Constantes de tratamento de erro


_**Aplica-se a:** Windows | Windows Servidor_

## <a name="error-handling-constants"></a>Constantes de tratamento de erro

As constantes a seguir são usadas para definir o intervalo para os [JET_paramExceptionAction](./error-handling-parameters.md) do sistema.


| <p>Constante/valor</p> | <p>Descrição</p> | 
|-----------------------|--------------------|
| <p>JET_ExceptionMsgBox<br />0x0001</p> | <p>Exibe uma caixa de mensagem quando ocorre uma exceção.</p> | 
| <p>JET_ExceptionNone<br />0x0002</p> | <p>Não faz nada quando ocorre uma exceção.</p> | 



### <a name="requirements"></a>Requisitos


| Requisito | Valor |
|------------|----------|
| <p><strong>Cliente</strong></p> | <p>Requer Windows Vista, Windows XP ou Windows 2000 Professional.</p> | 
| <p><strong>Servidor</strong></p> | <p>Requer Windows Server 2008, Windows Server 2003 ou Windows 2000 Server.</p> | 
| <p><strong>Cabeçalho</strong></p> | <p>Declarado em Esent.h.</p> |
