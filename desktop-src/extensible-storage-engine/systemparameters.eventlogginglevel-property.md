# <a name="systemparameterseventlogginglevel-property"></a>Propriedade SystemParameters. EventLoggingLevel

Obtém ou define o nível de detalhe das mensagens de log de eventos emitidas para o log de eventos pelo mecanismo de banco de dados. Números mais altos resultarão em mensagens de log de eventos mais detalhadas.
