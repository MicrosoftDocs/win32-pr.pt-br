# <a name="using-extensible-storage-engine"></a>Usando o mecanismo de armazenamento extensível


_**Aplica-se a:** Windows | Windows Server_

## <a name="using-extensible-storage-engine"></a>Usando o mecanismo de armazenamento extensível

Esta seção contém informações gerais sobre como usar as seguintes partes da API do ESE.

  - [Colunas](./columns.md)

  - [Indexação na tabela](./indexing-in-the-table.md)

  - [Criando bancos de dados](./creating-databases.md)

  - [Transações](./transactions.md)

  - [Erros do ESE](./extensible-storage-engine-errors.md)

  - [Arquivos ESE](./extensible-storage-engine-files.md)

Os arquivos de origem do programa devem incluir o arquivo de cabeçalho ESENT. h para acessar protótipos de função e definições de estrutura para a API do mecanismo de armazenamento extensível. Os desenvolvedores podem usar o arquivo de biblioteca ESENT. lib para criar aplicativos que usam a API do mecanismo de armazenamento extensível. Em tempo de execução, os aplicativos são vinculados ao esent.dll.
