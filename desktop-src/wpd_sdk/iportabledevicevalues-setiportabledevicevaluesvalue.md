# <a name="iportabledevicevaluessetiportabledevicevaluesvalue-method"></a>Método IPortableDeviceValues:: SetIPortableDeviceValuesValue

O método **SetIPortableDeviceValuesValue** adiciona um novo valor **IPORTABLEDEVICEVALUES** (tipo VT \_ desconhecido) ou substitui um existente.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT SetIPortableDeviceValuesValue(
  [in] REFPROPERTYKEY        key,
  [in] IPortableDeviceValues *pValue
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*chave* \[ no\]
</dt> <dd>

Um **REFPROPERTYKEY** que especifica o item a ser criado ou substituído.

</dd> <dt>

*valores* \[ no\]
</dt> <dd>

Ponteiro para uma interface **IPortableDeviceValues** que especifica o novo valor. O SDK copia uma referência para a interface enviada e chama **AddRef** nela.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

O método retorna um **HRESULT**. Os possíveis valores incluem, mas sem limitação, aqueles na tabela a seguir.



| Código de retorno                                                                          | Descrição                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**S \_ OK**</dt> </dl> | O método foi bem-sucedido.<br/> |



 

## <a name="remarks"></a>Comentários

Se um valor existente tiver a mesma chave especificada pelo parâmetro de *chave* , ele substituirá o valor existente sem nenhum aviso. A memória de chave existente é liberada adequadamente.
