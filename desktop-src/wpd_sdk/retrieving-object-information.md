# <a name="retrieving-object-information"></a>Recuperando Informações do objeto

Além de recuperar as informações do dispositivo, a API WPD fornece um conjunto de interfaces e métodos que você pode usar para recuperar informações de um determinado objeto. Em geral, a recuperação de informações do objeto pode ser dividida em duas categorias, conforme descrito na tabela a seguir.



| Dados do dispositivo                                                                                    | Descrição                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Identificador de Objeto](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | Você pode recuperar o identificador de objeto que é mapeado para um determinado identificador exclusivo persistente. |
| [Identificadores de objeto funcional](retrieving-the-functional-object-identifiers-for-a-device.md) | Cada categoria funcional dá suporte a um ou mais objetos funcionais.                         |
