# <a name="wpd-constants"></a>Constantes WPD

Windows Os dispositivos portáteis definem os seguintes tipos de constantes.



| Tipo de constante                                              | Descrição                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes gerais](general-constants.md)                 | Constantes de uso geral para dispositivos e aplicativos.                                                                                                                                                                                       |
| [Constantes de erro](error-constants.md)                     | Valores de erro retornados por métodos.                                                                                                                                                                                                         |
| [Constantes de evento](event-constants.md)                     | tipos de evento definidos por Windows dispositivos portáteis. Os tipos de evento incluem objeto adicionado, objeto removido, objeto atualizado, objeto movido, objeto copiado, transferência de objeto solicitada, redefinição de dispositivo, dispositivo removido e recursos de dispositivo atualizados. |
| [GUIDs de tipo de objeto](object-type-guids.md)                 | tipos de objeto definidos por Windows dispositivos portáteis. Os tipos de objeto incluem áudio, email e objetos de imagem.                                                                                                                                  |
| [GUIDs de formato de objeto](object-format-guids.md)             | tipos de formato de arquivo definidos por Windows dispositivos portáteis. Os tipos de formato incluem WMA, WMV e JPEG.                                                                                                                                           |
| [GUIDs de tipo de recurso](resource-type-guids.md)             | tipos de recursos definidos por Windows dispositivos portáteis. Os tipos de recursos incluem dados de imagem de áudio e miniatura.                                                                                                                                |
| [Propriedades e atributos](properties-and-attributes.md) | propriedades e atributos definidos por Windows dispositivos portáteis.                                                                                                                                                                            |
