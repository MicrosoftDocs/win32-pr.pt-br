# <a name="wpd-constants"></a>Constantes WPD

Os Dispositivos Portáteis do Windows definem os seguintes tipos constantes.



| Tipo constante                                              | Descrição                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes gerais](general-constants.md)                 | Constantes de uso geral para dispositivos e aplicativos.                                                                                                                                                                                       |
| [Constantes de erro](error-constants.md)                     | Valores de erro retornados por métodos.                                                                                                                                                                                                         |
| [Constantes de evento](event-constants.md)                     | Tipos de evento definidos por Dispositivos Portáteis do Windows. Os tipos de evento incluem objeto adicionado, objeto removido, objeto atualizado, objeto movido, objeto copiado, transferência de objeto solicitada, redefinição de dispositivo, dispositivo removido e funcionalidades do dispositivo atualizadas. |
| [GUIDs de tipo de objeto](object-type-guids.md)                 | Tipos de objeto definidos por Dispositivos Portáteis do Windows. Os tipos de objeto incluem objetos de áudio, email e imagem.                                                                                                                                  |
| [GUIDs de formato de objeto](object-format-guids.md)             | Tipos de formato de arquivo definidos por Dispositivos Portáteis do Windows. Os tipos de formato incluem WMA, WMV e JPEG.                                                                                                                                           |
| [GUIDs de tipo de recurso](resource-type-guids.md)             | Tipos de recursos definidos por Dispositivos Portáteis do Windows. Os tipos de recursos incluem dados de imagem de áudio e miniatura.                                                                                                                                |
| [Propriedades e atributos](properties-and-attributes.md) | Propriedades e atributos definidos por Dispositivos Portáteis do Windows.                                                                                                                                                                            |
