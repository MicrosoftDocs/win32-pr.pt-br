# <a name="retrieving-device-information"></a>Recuperando Informações do dispositivo

A API WPD fornece um rico conjunto de interfaces e métodos que você pode usar para recuperar informações de um determinado dispositivo. Em geral, os dados do dispositivo podem ser divididos em três categorias, conforme descrito na tabela a seguir.



| Dados do dispositivo                                                                             | Descrição                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Categorias funcionais](retrieving-the-functional-categories-supported-by-a-device.md) | As categorias funcionais descrevem as operações que um dispositivo é capaz de executar. |
| [Tipos de conteúdo](retrieving-the-content-types-supported-by-a-device.md)                 | Os tipos de conteúdo indicam quais tipos de conteúdo um dispositivo é capaz de renderizar.         |
| [Eventos](retrieving-the-events-supported-by-a-device.md)                               | Alguns dispositivos são capazes de acionar eventos que seu aplicativo pode acompanhar e responder. |
