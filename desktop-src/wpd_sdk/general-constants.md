# <a name="general-constants"></a>Constantes gerais

Os dispositivos portáteis do Windows definem as seguintes constantes.



| Constante                                                                                                                                                                             | Descrição                                                                        |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------|
| <span id="WPD_MESSAGE"></span><span id="wpd_message"></span><dl> <dt>**\_mensagem WPD**</dt> </dl>                              | Os drivers usam essa constante para verificar se um comando é um comando WPD.<br/>    |
| <span id="WPD_DEVICE_OBJECT_ID"></span><span id="wpd_device_object_id"></span><dl> <dt>**\_ID de \_ objeto do dispositivo WPD \_**</dt> </dl> | Um valor constante que funciona como o identificador para o dispositivo especificado.<br/> |
