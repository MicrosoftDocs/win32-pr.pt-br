# <a name="properties-and-attributes"></a>Propriedades e atributos

As propriedades descrevem objetos, propriedades, recursos e dispositivos. Os atributos são um tipo especial de propriedade que descreve outras propriedades ou recursos.

As propriedades e os atributos são definidos como estruturas **PROPERTYKEY** com duas partes: um GUID de categoria e uma ID exclusiva para essa categoria. As propriedades que descrevem as coisas relacionadas compartilham um GUID comum; Esse GUID é sua categoria de propriedade. Uma categoria pode descrever uma ou mais propriedades, cada uma com uma ID exclusiva. A Microsoft define um conjunto de categorias de propriedade; Você pode usá-los para suas próprias propriedades ou criar novas categorias por conta própria.

Os tópicos a seguir listam os atributos e as propriedades que são definidos por dispositivos portáteis do Windows.
