# <a name="properties-and-attributes"></a>Propriedades e atributos

As propriedades descrevem objetos, propriedades, recursos e dispositivos. Atributos são um tipo especial de propriedade que descreve outras propriedades ou recursos.

Propriedades e atributos são definidos como estruturas **PROPERTYKEY** com duas partes: um GUID de categoria e uma ID exclusiva para essa categoria. Propriedades que descrevem coisas relacionadas compartilham um GUID comum; esse GUID é sua categoria de propriedade. Uma categoria pode descrever uma ou mais propriedades, cada uma com uma ID exclusiva. A Microsoft define um conjunto de categorias de propriedade; você pode usá-los para suas próprias propriedades ou criar novas categorias por conta própria.
