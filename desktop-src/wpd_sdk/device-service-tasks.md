# <a name="device-service-tasks"></a>Tarefas do serviço de dispositivo

A API WPD dá suporte a uma série de tarefas de serviços de dispositivo. Essas tarefas incluem o seguinte:

-   Enumerando serviços de dispositivo.
-   Abrindo um serviço de dispositivo.
-   Enumerando o conteúdo em um serviço de dispositivo.
-   Recuperando eventos de serviço com suporte.
-   Recuperando formatos de serviço com suporte.
-   Recuperando métodos de serviço com suporte.
-   Recuperando serviços abstratos com suporte.
-   Recuperando propriedades de objeto de conteúdo.
-   Gravando Propriedades de objeto de conteúdo.
-   Invocar métodos de serviço de forma síncrona.
-   Invocar métodos de serviço de forma assíncrona.

Essas operações são descritas nas seções a seguir e são demonstradas no aplicativo WpdServicesApiSample. Além disso, a API WPD também dá suporte à transferência de conteúdo, à movimentação de conteúdo e à exclusão de conteúdo de um serviço de dispositivo.
