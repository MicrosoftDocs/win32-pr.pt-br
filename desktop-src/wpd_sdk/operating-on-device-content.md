# <a name="operating-on-device-content"></a>Operando no conteúdo do dispositivo

A API WPD dá suporte a várias operações em dispositivos. Essas operações incluem o seguinte:

-   Transferência de conteúdo de um PC para um dispositivo conectado.
-   Transferência de conteúdo de um dispositivo conectado para um PC.
-   Movendo o conteúdo em um dispositivo conectado.
-   Excluindo o conteúdo de um dispositivo conectado.
-   Adicionar um recurso a um objeto em um dispositivo conectado.

Essas operações são descritas nas seções a seguir.
