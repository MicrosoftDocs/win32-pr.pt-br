# <a name="wpd-programming-guide"></a>Guia de programação WPD

A fonte principal para este guia de programação é um exemplo que acompanha o SDK (Software Development Kit) do Windows Portable Devices (WPD). Este exemplo, chamado WpdApiSample.exe, consiste em sete módulos. cpp. Seis desses módulos contêm o código que demonstra 26 tarefas que um aplicativo pode realizar usando a API (interface de programação de aplicativo) WPD.

As exceções acima são os tópicos que descrevem: tarefas do serviço de dispositivo, o menu de contexto e as extensões de folha de propriedades; esses tópicos não foram extraídos do WpdApiSample. As tarefas do serviço de dispositivo são baseadas no exemplo chamado WpdServiceApiSample. O menu de contexto e as extensões de folha de propriedades são trechos extraídos de aplicativos que não são fornecidos com o SDK do Windows.

As seções a seguir concentram-se nas tarefas realizadas por esses exemplos e explicam como cada um é realizado usando as interfaces WPD e vários métodos encontrados nessas interfaces.
