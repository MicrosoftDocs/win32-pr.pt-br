# <a name="wpd-programming-guide"></a>Guia de programação wpd

A origem principal deste guia de programação é um exemplo que acompanha o SDK (Kit de Desenvolvimento de Software) WPD (Dispositivos Portáteis do Windows). Este exemplo, chamado WpdApiSample.exe, consiste em sete módulos .cpp. Seis desses módulos contêm o código que demonstra 26 tarefas que um aplicativo pode realizar usando a API (interface de programação de aplicativo) WPD.

As exceções ao acima são os tópicos que descrevem: tarefas de serviço do dispositivo, o menu de contexto e as extensões da folha de propriedades; esses tópicos não foram retirados de WpdApiSample. As tarefas de serviço do dispositivo são baseadas no exemplo chamado WpdServiceApiSample. O menu de contexto e as extensões de folha de propriedades são snippets retirados de aplicativos que não são encaminhados com o SDK do Windows.

As seções a seguir se concentram nas tarefas realizadas por esses exemplos e explicam como cada uma é realizada usando as interfaces WPD e vários métodos encontrados nessas interfaces.
