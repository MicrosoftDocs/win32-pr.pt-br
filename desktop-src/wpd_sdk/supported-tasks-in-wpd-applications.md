# <a name="supported-tasks-in-wpd-applications"></a>Tarefas com suporte em aplicativos WPD

As interfaces de aplicativo WPD oferecem suporte às seguintes tarefas:

-   Conectar-se a um dispositivo
-   Enumerar dispositivos anexados
-   Examinar os recursos de um dispositivo
-   Explorar o conteúdo de um dispositivo
-   Transferir conteúdo para ou criar conteúdo em um dispositivo
-   Editar conteúdo no dispositivo
-   Modificar objetos em um dispositivo
-   Controlar o dispositivo
-   Monitorar eventos de dispositivo

Para obter mais informações sobre cada tarefa, bem como detalhes sobre como programar uma determinada tarefa, consulte o [Guia de programação de dispositivos portáteis do Windows](programming-guide.md).
