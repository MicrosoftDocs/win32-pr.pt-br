# <a name="application-overview"></a>Visão geral do aplicativo

Windows Os dispositivos portáteis (WPD) fornecem uma infraestrutura que permite o acesso do PC a dispositivos multifuncionais que armazenam várias mídias e conteúdos que não são de mídia. Essa infraestrutura permite que os dispositivos Convergent e inicialmente se concentre em dispositivos de armazenamento inteligente, como players de mídia portáteis, câmeras digitais ainda e celulares.

O WPD consiste em uma DDI (interface de driver de dispositivo) e em um conjunto de APIs (interfaces de programação de aplicativo). esta visão geral descreve como você pode aproveitar essas APIs em seu aplicativo Windows.

A infraestrutura WPD oferece representação de dispositivo altamente extensível e mecanismos de controle. Isso permite que os provedores de soluções verticais usem a API WPD para criar experiências de usuário aprimoradas que estão fora do conjunto padronizado WPD. Exemplos disso incluem software fornecido pelo fornecedor que executa atualizações de firmware ou diagnóstico de dispositivo (para suporte remoto).
