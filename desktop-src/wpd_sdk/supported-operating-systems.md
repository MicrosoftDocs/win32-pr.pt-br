# <a name="supported-operating-systems"></a>Sistemas operacionais com suporte

Os dispositivos portáteis do Windows têm suporte nos sistemas operacionais Windows 7, Windows Vista e Windows XP. Seu aplicativo será compilado e vinculado a esses sistemas operacionais usando as bibliotecas e cabeçalhos fornecidos.
