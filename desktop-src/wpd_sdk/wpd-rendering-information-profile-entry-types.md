# <a name="wpd_rendering_information_profile_entry_types-enumeration"></a>Enumeração WPD \_ RENDERING INFORMATION PROFILE ENTRY \_ \_ \_ \_ TYPES

O **tipo de enumeração WPD \_ RENDERING INFORMATION PROFILE ENTRY \_ \_ \_ \_ TYPES** indica se a entrada do perfil de informações de renderização corresponde a um Objeto ou um Recurso.

## <a name="syntax"></a>Syntax


```C++
typedef enum SMS_MESSAGE_TYPES { 
  WPD_RENDERING_INFORMATION_PROFILE_ENTRY_TYPE_OBJECT    = 0,
  WPD_RENDERING_INFORMATION_PROFILE_ENTRY_TYPE_RESOURCE  = 1
} ;
```
