## <a name="syntax"></a>Syntax


```C++
typedef enum SMS_MESSAGE_TYPES { 
  WPD_RENDERING_INFORMATION_PROFILE_ENTRY_TYPE_OBJECT    = 0,
  WPD_RENDERING_INFORMATION_PROFILE_ENTRY_TYPE_RESOURCE  = 1
```
