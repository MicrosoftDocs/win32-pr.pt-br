# <a name="wpd-samples"></a>Exemplos de WPD

O SDK do Windows fornece dois exemplos: [WpdApiSample](wpdapisample-sample-application.md) e [WpdServiceApiSample.](wpdapisample-sample-service-application.md)

O WpdApiSample é um aplicativo de área de trabalho de linha de comando que permite que o usuário execute várias funções, como: enumerar dispositivos conectados, explorar os objetos nesses dispositivos e enumerar propriedades de objeto.

O WpdServiceApiSample é um aplicativo de área de trabalho de linha de comando que dá suporte a um serviço de dispositivo Contatos e permite que o usuário execute funções como: enumerar serviços de dispositivo conectados, recuperar propriedades e formatos para um determinado serviço e invocar métodos nesse serviço.
