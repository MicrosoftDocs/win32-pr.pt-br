# <a name="wpd-samples"></a>Amostras de WPD

O SDK do Windows fornece dois exemplos: [WpdApiSample](wpdapisample-sample-application.md) e [WpdServiceApiSample](wpdapisample-sample-service-application.md).

O WpdApiSample é um aplicativo de área de trabalho de linha de comando que permite ao usuário executar várias funções, como: enumerar dispositivos conectados, explorar os objetos nesses dispositivos e enumerar Propriedades de objeto.

O WpdServiceApiSample é um aplicativo de área de trabalho de linha de comando que dá suporte a um serviço de dispositivo de contatos e permite ao usuário executar funções como: enumerar serviços de dispositivo conectados, recuperar propriedades e formatos para um determinado serviço e invocar métodos nesse serviço.
