# <a name="still-image-properties"></a>Propriedades da imagem ainda

Os dispositivos portáteis do Windows oferecem suporte às seguintes propriedades de imagem ainda.



| Propriedade                                                                                                                                                       | VarType        | Descrição                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_artista de \_ imagem \_ ainda mais WPD**                                                                                                                                  | **LPWStr do VT \_** | O nome do proprietário/usuário do dispositivo.                                                                                                                                                                                                                                                                                    |
| <span id="wpd_still_image_burst_interval"></span><span id="WPD_STILL_IMAGE_BURST_INTERVAL"></span>**\_intervalo de \_ intermitência de imagem ainda \_ mais WPD \_**                       | **\_UI4 VT**    | Em uma captura de imagem de modo intermitente, o tempo de espera entre capturas de imagem, em milissegundos.                                                                                                                                                                                                                                       |
| <span id="wpd_still_image_burst_number"></span><span id="WPD_STILL_IMAGE_BURST_NUMBER"></span>**\_número de \_ intermitência de imagem ainda \_ WPD \_**                             | **\_UI4 VT**    | Em uma captura de imagem de modo intermitente, o número de imagens a serem capturadas.                                                                                                                                                                                                                                                              |
| **\_atraso de \_ captura de imagem \_ \_ do WPD ainda**                                                                                                                          | **\_UI4 VT**    | Especifica o atraso de tempo entre o comando para capturar uma imagem e a captura real, em milissegundos.                                                                                                                                                                                                                    |
| **\_formato de \_ captura de imagem \_ \_ do WPD ainda**                                                                                                                         | **CLSID do VT \_**  | O formato a ser usado para capturar a imagem. Esse é um dos [valores de GUID de formato](object-format-guids.md) que é definido por dispositivos portáteis do Windows.                                                                                                                                                                        |
| <span id="wpd_still_image_capture_mode"></span><span id="WPD_STILL_IMAGE_CAPTURE_MODE"></span>**\_modo de \_ captura de imagem ainda \_ WPD \_**                             | **\_UI4 VT**    | Um enumerador de [**\_ \_ modos de captura WPD**](wpd-capture-modes.md) que especifica o modo de captura a ser usado para capturar uma imagem.                                                                                                                                                                                                    |
| **\_resolução de \_ captura de imagem \_ \_ do WPD ainda**                                                                                                                     | **LPWStr do VT \_** | Uma cadeia de caracteres que especifica o tamanho da imagem a ser capturada. O tamanho da imagem é especificado em pixels, no formato "*largura* x *altura*". Por exemplo: "800x600".                                                                                                                                                                          |
| **\_configuração de \_ \_ compactação de imagem \_ do WPD ainda**                                                                                                                    | **\_UI8 VT**    | Descreve as configurações de compactação de imagem que são usadas durante a captura de uma imagem. Isso é específico do dispositivo.                                                                                                                                                                                                                     |
| **a WPD ainda é o \_ \_ contraste da imagem \_**                                                                                                                                | **\_UI4 VT**    | Um número que especifica a configuração de contraste a ser aplicada durante a captura de uma imagem, sendo que zero é o menor contraste e números maiores, indicando maior contraste.                                                                                                                                                            |
| **\_ \_ \_ zoom digital da imagem \_ do WPD ainda**                                                                                                                           | **\_UI4 VT**    | Um número que especifica a quantidade de zoom digital a ser usado ao capturar uma imagem. O número é 1 ou superior, escalado por um fator de 10. Um valor de 1 (representando 10) indica que não há zoom digital; um valor de 2 (representando 20) indica um zoom 2x (1/4 da imagem vista por um zoom 1x será capturado).                  |
| **\_modo de \_ efeito de imagem ainda \_ mais WPD \_**                                                                                                                            | **\_UI4 VT**    | Um enumerador de [**\_ \_ modos de efeito WPD**](wpd-effect-modes.md) que especifica um efeito de imagem a ser usado durante a captura de uma imagem.                                                                                                                                                                                                   |
| **o WPD ainda é uma \_ compensação de desvio de \_ exposição de imagem \_ \_ \_**                                                                                                            | **\_UI4 VT**    | Um número que especifica as unidades de APEX para ajustar a exposição. As unidades são paradas x 1.000. A especificação de zero indica o padrão de fábrica.                                                                                                                                                                                       |
| **\_índice de \_ exposição de imagem \_ \_ do WPD ainda**                                                                                                                         | **\_UI4 VT**    | Um número que especifica a velocidade do filme ISO a ser emulado ao capturar uma imagem. Um valor de 0xFFFF indica que esse valor deve ser definido automaticamente.                                                                                                                                                                      |
| **\_modo de \_ medição de exposição de imagem \_ WPD \_ e ainda \_**                                                                                                                | **\_UI4 VT**    | Um enumerador de [**\_ modos de \_ medição \_ de exposição WPD**](wpd-exposure-metering-modes.md) que especifica o modo de medição a ser usado durante a captura de uma imagem.                                                                                                                                                                          |
| <span id="wpd_still_image_exposure_program_mode"></span><span id="WPD_STILL_IMAGE_EXPOSURE_PROGRAM_MODE"></span>**\_modo de \_ programa de exposição de imagem \_ \_ WPD \_** | **\_UI4 VT**    | Um enumerador de [**\_ modos de \_ programa \_ de exposição WPD**](wpd-exposure-program-modes.md) que especifica o programa de exposição a ser usado durante a captura de uma imagem.                                                                                                                                                                         |
| **\_tempo de \_ exposição da imagem \_ \_ do WPD ainda**                                                                                                                          | **\_UI4 VT**    | A velocidade do obturador a ser usada ao capturar uma imagem, em segundos x 10.000.                                                                                                                                                                                                                                                       |
| **\_modo de \_ flash de imagem ainda \_ mais WPD \_**                                                                                                                             | **\_UI4 VT**    | Um enumerador de [**\_ \_ modos flash WPD**](wpd-flash-modes.md) que especifica o modo flash a ser usado durante a captura de uma imagem.                                                                                                                                                                                                      |
| **\_FNUMBER de \_ imagens WPD ainda \_**                                                                                                                                 | **\_UI4 VT**    | Um número que especifica a abertura da lente durante a captura de uma imagem. O número é o número de parada f 100 (portanto, 14 indicaria f/1.4). Essa propriedade normalmente é válida somente quando [o \_ \_ modo de \_ \_ programa de \_ exposição de imagem WPD ainda](/windows) está definido como prioridade manual ou de abertura.<br/> |
| **\_ \_ comprimento focal da imagem ainda \_ WPD \_**                                                                                                                           | **\_UI4 VT**    | O comprimento focal, em milímetros x 100, a ser usado durante a captura de uma imagem. Por exemplo, 35mm seria 3500.                                                                                                                                                                                                                 |
| **a \_ \_ distância do \_ foco da imagem WPD ainda \_**                                                                                                                         | **\_UI4 VT**    | A distância de foco que um dispositivo deve usar, em milímetros. Um valor de 0xFFFF indica foco infinito (mais de 655 metros).                                                                                                                                                                                              |
| **\_modo de \_ medição de foco de imagem \_ WPD \_ \_**                                                                                                                   | **\_UI4 VT**    | Um enumerador de [**\_ modos de \_ medição \_ de foco WPD**](wpd-focus-metering-modes.md) que especifica qual modo de foco um dispositivo de captura de imagem deve usar ao decidir como se concentrar em um assunto.                                                                                                                                     |
| **o \_ \_ modo de \_ foco de imagem \_ do WPD ainda**                                                                                                                             | **\_UI4 VT**    | Um enumerador de [**\_ \_ modos de foco WPD**](wpd-focus-modes.md) que especifica o modo de foco a ser usado durante a captura de uma imagem.                                                                                                                                                                                                      |
| **a imagem de WPD ainda tem um \_ \_ \_ \_ lucro RGB**                                                                                                                               | **LPWStr do VT \_** | Uma cadeia de caracteres terminada em nulo que especifica o lucro de RGB a ser usado ao capturar uma imagem. A cadeia de caracteres é de três números delimitados por dois-pontos sem espaços intermediários, no formato "*R*:*G*:*B*".                                                                                                                                    |
| **a \_ \_ nitidez da imagem \_ do WPD ainda**                                                                                                                               | **\_UI4 VT**    | Um número que especifica a nitidez a ser usada ao capturar uma imagem, onde zero é o número menos nítido e maior indicam maior nitidez.                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_interval"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_INTERVAL"></span>**\_intervalo de \_ TIMELAPSE de imagens WPD ainda \_ \_**           | **\_UI4 VT**    | O tempo de atraso entre capturas de imagem em uma captura de imagem de lapso de tempo, em milissegundos.                                                                                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_number"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_NUMBER"></span>**\_TIMELAPSE WPD \_ ainda \_ \_ número de imagem**                 | **\_UI4 VT**    | O número de imagens a serem capturadas em uma captura de imagem de modo de lapso de tempo.                                                                                                                                                                                                                                                          |
| **\_URL de \_ upload de imagem \_ \_ do WPD ainda**                                                                                                                             | **LPWStr do VT \_** | Especifica uma URL na qual o dispositivo deve enviar a imagem após sua captura.                                                                                                                                                                                                                                                 |
| **\_proporção de \_ branco de imagem ainda \_ WPD \_**                                                                                                                          | **\_UI4 VT**    | Um valor enumerado [**\_ \_ \_ das configurações de balanço de branco WPD**](wpd-white-balance-settings.md) que especifica as configurações de balanço de branco a serem usadas durante a captura de uma imagem.                                                                                                                                                             |
