# <a name="driver-interfaces"></a>Interfaces de driver

As interfaces a seguir são usadas somente por um driver de dispositivos portáteis do Windows.



| Interface                                | Descrição                                                                                                                                            |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWpdSerializer**](iwpdserializer.md) | Usado pelo driver de dispositivo para serializar as interfaces **IPortableDeviceValues** de e para os buffers de dados brutos usados para se comunicar com o aplicativo. |
