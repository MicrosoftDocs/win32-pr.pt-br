# <a name="object-properties"></a>Propriedades de objeto

Os dispositivos portáteis do Windows oferecem suporte às seguintes propriedades de objeto.



| Propriedade                                                                                                                                    | VarType         | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------------------------------------------------------------------|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_referências de \_ retorno de objeto WPD \_**                                                                                                           | **VT \_ desconhecido** | Um [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) do tipo VT \_ LPWSTR que indica uma lista de ObjectIDs. Quando um objeto é referenciado por um objeto de contêiner, por exemplo, um objeto de áudio referenciado por uma lista de reprodução, o objeto referenciado usa essa propriedade para fazer referência ao seu objeto de contêiner. O objeto de áudio pode referir-se ao objeto playlist neste exemplo.                                                                                                                   |
| **o \_ objeto WPD \_ pode \_ excluir**                                                                                                                | **BOOL do VT \_**    | Um valor booliano que especifica se o objeto fornecido pode ser excluído.                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **\_ID de \_ \_ objeto funcional \_ do contêiner de objetos WPD \_**                                                                                          | **LPWStr do VT \_**  | A ID de objeto do objeto funcional mais próximo que contém esse objeto. Por exemplo, um arquivo dentro de um objeto funcional de armazenamento terá essa propriedade definida como a ID do objeto funcional de armazenamento.                                                                                                                                                                                                                                                                                                                        |
| <span id="wpd_object_content_type"></span><span id="WPD_OBJECT_CONTENT_TYPE"></span>**\_tipo de \_ conteúdo de objeto WPD \_**                          | **CLSID do VT \_**   | Um **GUID** que identifica o tipo genérico desse objeto, por exemplo, um documento ou email. Pode ser um [tipo de objeto definido por dispositivos portáteis do Windows](requirements-for-objects.md)ou um tipo de conteúdo de driver personalizado. O objeto de dispositivo é o único objeto que não relata essa propriedade.                                                                                                                                                                                                                        |
| <span id="wpd_object_date_authored"></span><span id="WPD_OBJECT_DATE_AUTHORED"></span>**\_data do objeto WPD \_ \_ criado**                       | **Data de VT \_**    | Um valor que especifica a data e a hora em que o conteúdo foi criado. Isso pode não ser igual à data de criação do arquivo. Por exemplo, um arquivo de música tem uma data de criação de quando a música foi registrada, mas uma data de criação de quando o arquivo WMA foi realmente criado no dispositivo.                                                                                                                                                                                                                                |
| <span id="wpd_object_date_created"></span><span id="WPD_OBJECT_DATE_CREATED"></span>**\_data do objeto WPD \_ \_ criado**                          | **Data de VT \_**    | Um valor que especifica a data e a hora em que o objeto foi criado no dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="wpd_object_date_modified"></span><span id="WPD_OBJECT_DATE_MODIFIED"></span>**\_data do objeto WPD \_ \_ modificado**                       | **Data de VT \_**    | Um valor que especifica a data e a hora em que o objeto foi modificado no dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="wpd_object_format"></span><span id="WPD_OBJECT_FORMAT"></span>**\_formato de objeto WPD \_**                                             | **CLSID do VT \_**   | Um **GUID** que identifica o formato dos dados do objeto. Isso pode ser um [formato definido por dispositivos portáteis do Windows](object-format-guids.md) ou um formato de driver personalizado.                                                                                                                                                                                                                                                                                                                                                            |
| **o \_ objeto WPD \_ gera \_ miniaturas \_ do \_ recurso**                                                                                        | **BOOL do VT \_**    | Um valor booliano que especifica se a imagem em miniatura desse objeto deve ser criada a partir dos dados de recurso padrão. Isso fornece uma maneira de objetos sem um recurso de miniaturas fornecer uma experiência de navegação mais amigável. O uso desse sinalizador pode afetar a primeira resposta de exibição, já que o aplicativo deve recuperar e calcular uma imagem em miniatura do dispositivo; é mais eficiente fornecer um recurso de imagem em miniatura separado, se possível.<br/>                                                  |
| **\_nome de \_ exibição do local de dica de objeto WPD \_ \_ \_**                                                                                              | **LPWStr do VT \_**  | Se o objeto fornecido aparecer como um local de dica, essa propriedade indicará o nome específico da dica a ser exibido, em vez do nome do objeto. Os drivers podem especificar dicas de local para vários tipos de conteúdo. Eles podem ser considerados como atalhos para objetos de pasta de nível superior que contêm objetos dos tipos especificados. Os clientes que usam essas dicas de local podem exibir um nome para esses atalhos que diferem do nome do objeto da pasta. Se essa propriedade não existir, o **nome do \_ objeto \_ WPD** normalmente será usado. |
| <span id="wpd_object_id"></span><span id="WPD_OBJECT_ID"></span>**\_ID de objeto WPD \_**                                                         | **LPWStr do VT \_**  | Uma ID de cadeia de caracteres que identifica exclusivamente o objeto no dispositivo. Essa ID não precisa ser armazenada entre as sessões. Se essa propriedade for exclusiva e persistente, o driver poderá definir a **\_ \_ \_ \_ ID exclusiva persistente do objeto WPD** e a **\_ \_ ID de objeto WPD** com o mesmo valor.<br/>                                                                                                                                                                                                                                                 |
| <span id="wpd_object_is_drm_protected"></span><span id="WPD_OBJECT_IS_DRM_PROTECTED"></span>**o \_ objeto \_ WPD \_ é \_ protegido por DRM**             | **BOOL do VT \_**    | Um valor booliano que especifica se os dados de mídia são protegidos por DRM. Se não estiver presente, isso será considerado falso.                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="wpd_object_ishidden"></span><span id="WPD_OBJECT_ISHIDDEN"></span>**objeto WPD- \_ \_ oculto**                                       | **BOOL do VT \_**    | Um valor booliano que especifica se o objeto deve ser oculto. Se não estiver presente, supõe-se que o objeto não esteja oculto.                                                                                                                                                                                                                                                                                                                                                                                                     |
| **o \_ objeto WPD \_ pode \_ excluir**                                                                                                                | **BOOL do VT \_**    | Um valor booliano que especifica se o objeto pode ser excluído.                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="wpd_object_issystem"></span><span id="WPD_OBJECT_ISSYSTEM"></span>**IsSystem de \_ objetos WPD \_**                                       | **BOOL do VT \_**    | Um valor booliano que especifica se o objeto representa dados do sistema (como um arquivo do sistema). Se não estiver presente, supõe-se que o objeto não seja um objeto do sistema.                                                                                                                                                                                                                                                                                                                                                              |
| <span id="wpd_object_keywords"></span><span id="WPD_OBJECT_KEYWORDS"></span>**\_ \_ palavras-chave do objeto WPD**                                       | **LPWStr do VT \_**  | Cadeia de caracteres que contém uma lista de palavras-chave delimitadas por espaço associadas a este objeto.                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **\_localidade do \_ idioma do objeto WPD \_**                                                                                                           | **LPWStr do VT \_**  | Uma cadeia de caracteres que dentifies o idioma usado por um determinado objeto. Se vários idiomas estiverem contidos nesse objeto, ele deverá identificar o idioma principal. Essa propriedade pode conter um código de idioma, conforme definido no ISO-639, como: "en". Ele também pode conter um código de país de idioma, que consiste em um código de idioma de dois ou três caracteres, conforme definido no padrão ISO-639, seguido por um hífen, seguido por um código de país, conforme definido no ISO-3166, como: "en-US".                                |
| <span id="wpd_object_name"></span><span id="WPD_OBJECT_NAME"></span>**\_nome do objeto WPD \_**                                                   | **LPWStr do VT \_**  | O nome de exibição do objeto.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="wpd_object_non_consumable"></span><span id="WPD_OBJECT_NON_CONSUMABLE"></span>**\_objeto WPD \_ não \_ consumível**                    | **BOOL do VT \_**    | Um valor booliano que especifica se esse objeto deve ser compreendido ou simplesmente armazenado pelo dispositivo. Se essa propriedade não estiver presente, todos os dados serão considerados para consumo.                                                                                                                                                                                                                                                                                                                            |
| <span id="wpd_object_original_file_name"></span><span id="WPD_OBJECT_ORIGINAL_FILE_NAME"></span>**\_nome de \_ \_ arquivo original do objeto \_ WPD**       | **LPWStr do VT \_**  | Um nome de cadeia de caracteres para o arquivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="wpd_object_parent_id"></span><span id="WPD_OBJECT_PARENT_ID"></span>**\_ \_ ID pai do objeto WPD \_**                                   | **LPWStr do VT \_**  | A ID de objeto do objeto pai. O único objeto que pode retornar uma cadeia de caracteres vazia para esse valor é o objeto de dispositivo raiz. Para modificar essa propriedade, chame [**IPortableDevice:: SendCommand**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledevice-sendcommand) (**a \_ \_ \_ movimentação de armazenamento de comando WPD**).<br/>                                                                                                                                                                                                                                                    |
| <span id="wpd_object_persistent_unique_id"></span><span id="WPD_OBJECT_PERSISTENT_UNIQUE_ID"></span>**\_ \_ \_ ID exclusiva persistente do objeto WPD \_** | **LPWStr do VT \_**  | Uma ID de cadeia de caracteres que identifica exclusivamente o objeto no dispositivo, semelhante **à \_ \_ ID de objeto WPD**, mas deve ser armazenada entre as sessões. Se a **\_ \_ ID de objeto** do identificador de objeto WPD for exclusiva e persistente, o driver poderá definir a **\_ \_ \_ \_ ID exclusiva persistente do objeto WPD** e a **\_ \_ ID de objeto WPD** com o mesmo valor.<br/>                                                                                                                                                                          |
| <span id="wpd_object_references"></span><span id="WPD_OBJECT_REFERENCES"></span>**\_referências de objetos WPD \_**                                 | **VT \_ desconhecido** | Um [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) que contém uma coleção de IDs de objeto do do VT que \_ identifica os objetos referenciados. Isso só será necessário se o objeto for um objeto de referência, como uma pasta ou uma lista de reprodução.                                                                                                                                                                                                                                                            |
| <span id="wpd_object_size"></span><span id="WPD_OBJECT_SIZE"></span>**\_tamanho do objeto WPD \_**                                                   | **\_UI8 VT**     | O tamanho dos dados do recurso de objeto.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="wpd_object_sync_id"></span><span id="WPD_OBJECT_SYNC_ID"></span>**\_ID de \_ sincronização do objeto WPD \_**                                         | **LPWStr do VT \_**  | Uma cadeia de caracteres opaca criada por um cliente para reter o estado entre sessões sem reter um catálogo de conteúdo de dispositivo conectado.                                                                                                                                                                                                                                                                                                                                                                                                |
