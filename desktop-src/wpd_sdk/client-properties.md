# <a name="client-properties"></a>Propriedades do cliente

Os dispositivos portáteis do Windows oferecem suporte às seguintes propriedades de cliente.



| Propriedade                                          | VarType                   | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|---------------------------------------------------|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ acesso desejado ao cliente WPD \_**                  | **\_UI4 VT**               | Um valor que especifica que o cliente requer permissões de acesso exclusivas, como \_ somente leitura genérica. Se essa propriedade não for definida, WPD assumirá \_ \| \_ a gravação genérica de leitura genérica, que é o padrão recomendado.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **\_ \_ versão principal do cliente WPD \_**                   | **\_UI4 VT**               | Um número que especifica a versão principal do cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_ \_ versão secundária do cliente WPD \_**                   | **\_UI4 VT**               | Um número que especifica a versão secundária do cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_nome do cliente WPD \_**                             | **LPWStr do VT \_**            | Uma cadeia de caracteres de nome para o aplicativo cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **\_revisão de cliente WPD \_**                         | **\_UI4 VT**               | Um número que especifica a revisão (ou o número de compilação) do cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ \_ qualidade \_ de serviço do Client \_ Security WPD**   | **\_UI4 VT**               | Um valor que especifica a qualidade de serviço de segurança para a conexão com o driver. Isso está relacionado aos sinalizadores de qualidade de serviço de segurança para **CreateFile**.<br/> Por exemplo, esses sinalizadores permitem ou proíbem que um driver represente o cliente. \_A identificação de segurança será assumida se esse valor não for fornecido. Normalmente, os aplicativos devem definir esse valor como \_ representação de segurança, já que alguns drivers exigirão representação para funcionar. No entanto, os aplicativos executados sob requisitos de segurança estritos com um subconjunto conhecido de dispositivos podem querer restringir a capacidade do driver de identificar ou representar o chamador. Observe que a redução do nível de representação para qualquer coisa inferior à representação de segurança \_ pode resultar em alguns drivers com funcionalidade reduzida.<br/>                                         |
| **\_modo de \_ compartilhamento de cliente WPD \_**                      | **\_UI4 VT**               | Um valor que especifica o modo de compartilhamento que o cliente está solicitando do driver. Os valores possíveis são os mesmos para **CreateFile**, por exemplo, leitura de \_ compartilhamento de arquivos \_ , gravação de compartilhamento de arquivos \_ \_ e assim por diante.<br/> \_ \_ \| \_ \_ A gravação do compartilhamento de arquivos de leitura do compartilhamento de arquivos será assumida se o cliente não especificar esse parâmetro. Esse é o padrão recomendado. No entanto, onde um modo de compartilhamento diferente é necessário, por exemplo, \_ \_ leitura de compartilhamento de arquivos para solicitar somente compartilhamento de leitura, esse parâmetro é necessário.<br/>                                                                                                                                                                                                                                                                                                                                                          |
| **\_ \_ \_ chave privada do aplicativo \_ de cliente WPD \_** | **\_UI1 de vetor \_ \| VT \_ VT** | Um valor que especifica a chave privada do aplicativo DRM do Windows Media do cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **\_certificado de \_ aplicativo de WMDRM cliente \_ WPD \_**  | **\_UI1 de vetor \_ \| VT \_ VT** | Um valor que especifica o certificado DRM do Windows Media do cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **\_cookie de \_ evento de cliente WPD \_**                    | **LPWStr do VT \_**            | Um cookie fornecido pelo cliente retornado pelo driver em eventos postados como um resultado direto de operações emitidas por esse cliente. Essa é uma propriedade opcional que o driver pode fornecer como um parâmetro de evento. Os clientes normalmente usam esse valor para otimizar a manipulação de eventos. Por exemplo, se o cliente excluir um objeto, seu manipulador de eventos poderá optar por ignorar o evento resultante, sabendo que ele já realizou a ação apropriada (por exemplo, atualizou a exibição no thread que emitiu a exclusão).                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ tamanho mínimo de \_ buffer de resultados do \_ cliente WPD \_**   | **\_UI4 VT**               | Especifica o tamanho mínimo do buffer (em bytes) para enviar comandos para o driver. Os clientes podem usar essa opção para recomendar o tamanho mínimo do buffer interno que é usado para enviar comandos para o driver. Isso pode resultar em uma superfície de memória menor para clientes que só enviam comandos com tamanhos de dados pequenos, como obter ou definir propriedades. O driver pode substituir esse tamanho de buffer definindo **PortableDeviceResultsBufferMin** em sua seção de parâmetros de dispositivo. Esse tamanho de buffer deve ser 1024 ou superior. Se nem o cliente nem o driver definir o tamanho do buffer, o padrão será 1 MB.                                                                                                                                                                                                                                                  |
| **\_ \_ fechamento manual do cliente WPD \_ \_ ao \_ desconectar**    | **BOOL do VT \_**              | Uma opção avançada para clientes que chamam manualmente **IPortableDevice:: Close** ou **IPortableDeviceService:: Close** para cada objeto desconectado, em vez de depender da API para fazer isso. FALSE será considerado se o cliente não especificar esse parâmetro. Esse é o padrão recomendado se o cliente não estiver escutando mensagens do **WM \_ DEVICECHANGE** . Os clientes que especificam essa opção normalmente escutam mensagens do **WM \_ DEVICECHANGE** para o dispositivo e fecham manualmente as instâncias **IPortableDevice** e **IPortableDeviceService** na desconexão do dispositivo. Os clientes que escolhem essa opção devem gerenciar cuidadosamente o tempo de vida de cada instância de **IPortableDevice** e **IPortableDeviceService** liberando esses objetos quando o dispositivo estiver desconectado. Não fazer isso resultará em vazamentos nos processos de WUDFHost.exe para o driver. |
