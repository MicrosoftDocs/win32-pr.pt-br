# <a name="selecting-a-device-wpd-api"></a>Selecionando um dispositivo (API WPD)

Depois que seu aplicativo enumerar os dispositivos conectados, talvez você queira dar ao usuário a capacidade de selecionar um dispositivo específico e estabelecer uma conexão para operações posteriores. No aplicativo de exemplo, `ChooseDevice` a função no módulo DeviceEnumeration.cpp realiza essas tarefas.

A função usa o código de enumeração do `ChooseDevice` dispositivo que já vimos para enumerar os dispositivos disponíveis. (Consulte a seção [Enumerando dispositivos](enumerating-devices.md) para obter mais informações.)
