# <a name="selecting-a-device-wpd-api"></a>Selecionando um dispositivo (API WPD)

Depois que o aplicativo enumera os dispositivos conectados, talvez você queira conceder ao usuário a capacidade de selecionar um dispositivo específico e estabelecer uma conexão para outras operações. No aplicativo de exemplo, a `ChooseDevice` função no módulo DeviceEnumeration. cpp realiza essas tarefas.

A `ChooseDevice` função usa o código de enumeração do dispositivo que já vimos para enumerar os dispositivos disponíveis. (Consulte a seção [enumerando dispositivos](enumerating-devices.md) para obter mais informações.)
