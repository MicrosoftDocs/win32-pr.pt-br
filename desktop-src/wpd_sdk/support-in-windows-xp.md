# <a name="support-in-windows-xp"></a>Suporte no Windows XP

Para que seu aplicativo seja executado no Windows XP, o usuário precisará de uma cópia do runtime Windows Formato de Mídia em seu computador. Você pode redistribuir esse runtime com seu aplicativo WPD. (O runtime é encontrado em uma pasta de redistribuíveis no SDK Windows Formato de Mídia.)
