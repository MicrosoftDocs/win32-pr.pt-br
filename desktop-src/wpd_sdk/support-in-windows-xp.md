# <a name="support-in-windows-xp"></a>Suporte no Windows XP

Para que seu aplicativo seja executado no Windows XP, o usuário precisará de uma cópia do tempo de execução de formato do Windows Media em seu computador. Você pode redistribuir esse tempo de execução com seu aplicativo WPD. (O tempo de execução é encontrado em uma pasta de redistribuíveis no Windows Media Format SDK.)
