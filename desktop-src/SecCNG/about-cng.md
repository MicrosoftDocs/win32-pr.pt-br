# <a name="about-cng"></a>Sobre a CNG

Cryptography API: CNG (próxima geração) é a substituição de longo prazo para o [*CryptoAPI*](/windows/desktop/SecGloss/c-gly). A CNG foi projetada para ser extensível em muitos níveis e a criptografia não é um comportamento diferente.

## <a name="features"></a>Recursos

Para obter informações sobre os recursos da CNG, consulte [recursos da CNG](cng-features.md).

## <a name="cryptographic-primitives"></a>Primitivos criptográficos

Para obter mais informações sobre os primitivos criptográficos na CNG, consulte [primitivos de criptografia](cryptographic-primitives.md).

## <a name="key-storage-and-retrieval"></a>Armazenamento e recuperação de chave

para obter mais informações sobre o armazenamento de chaves e a recuperação na CNG, consulte [key Armazenamento and retrievity](key-storage-and-retrieval.md).

## <a name="key-import-and-export"></a>Importação e exportação de chave

Para obter mais informações sobre como importar e exportar chaves, consulte [importação e exportação de chave](key-import-and-export.md).

## <a name="data-protection-api-next-generation"></a>API de proteção de dados: próxima geração

A CNG fornece um conjunto de APIs que podem ser usadas para criptografar e descriptografar facilmente o conteúdo para entidades de autorização em vários computadores. Para obter mais informações, consulte [CNG DPAPI](cng-dpapi.md).
