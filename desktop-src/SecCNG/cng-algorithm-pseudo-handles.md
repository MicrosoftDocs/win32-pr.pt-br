# <a name="cng-algorithm-pseudo-handles"></a>Pseudo-identificadores de algoritmo CNG

## <a name="remarks"></a>Comentários

A partir do Windows 10, a CNG fornece identificadores de algoritmo predefinidos para muitos algoritmos. Você pode usar esses identificadores em qualquer situação que exija um identificador de algoritmo. No entanto, qualquer chamada para [**BCryptSetProperty**](/windows/desktop/api/Bcrypt/nf-bcrypt-bcryptsetproperty) falhará, pois o identificador de algoritmo é compartilhado e não pode ser modificado. Além disso, esses identificadores não podem ser usados em IRQL = DISPATCH no modo kernel.
