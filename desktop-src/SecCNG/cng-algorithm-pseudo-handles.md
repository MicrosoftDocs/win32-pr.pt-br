# <a name="cng-algorithm-pseudo-handles"></a>Pseudo-alças do algoritmo CNG

## <a name="remarks"></a>Comentários

A partir Windows 10, o CNG fornece alças de algoritmo pré-definidas para muitos algoritmos. Você pode usar esses alças em qualquer situação que exija um alça de algoritmo. No entanto, qualquer chamada para [**BCryptSetProperty**](/windows/desktop/api/Bcrypt/nf-bcrypt-bcryptsetproperty) falha quando o alça de algoritmo é compartilhado e não pode ser modificado. Além disso, esses alças não podem ser usados em IRQL=DISPATCH no modo kernel.
