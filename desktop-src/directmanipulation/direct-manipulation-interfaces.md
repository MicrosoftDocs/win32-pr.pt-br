# <a name="direct-manipulation-interfaces"></a>Interfaces de manipulação direta

Os tópicos contidos nesta seção fornecem as especificações de referência para interfaces [de Manipulação](direct-manipulation-portal.md) Direta.

> [!Note]  
> Ao implementar um [objeto de Manipulação](direct-manipulation-portal.md) Direta, verifique se a implementação [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) dá suporte à multithreading por meio da contagem de referências thread-safe. Para obter mais informações, [**consulte InterlockedIncrement**](/windows/win32/api/winnt/nf-winnt-interlockedincrement) e [**InterlockedDecrement**](/windows/win32/api/winnt/nf-winnt-interlockeddecrement).
