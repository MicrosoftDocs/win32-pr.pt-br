<span id="upnp.n_1_gly"></span><span id="UPNP.N_1_GLY"></span>**dispositivo não em execução**
</dt> <dd>

Um dispositivo não em execução é um dispositivo para o qual o código é carregado somente quando uma solicitação para esse dispositivo é recebida. Geralmente, esses são dispositivos para os quais as descrições de serviço são conhecidas antes do registro.

Um exemplo típico de dispositivos não em execução é um dispositivo soft, como um dispositivo de player de mídia.
