# <a name="device-host-programming-considerations"></a>Considerações sobre programação do host do dispositivo

Uma URL de apresentação vazia na descrição do dispositivo não é permitida. Se o dispositivo não tiver uma URL de apresentação, remova as marcas de URL de apresentação da descrição do dispositivo.
