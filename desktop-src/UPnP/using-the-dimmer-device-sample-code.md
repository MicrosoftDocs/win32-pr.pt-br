# <a name="using-the-dimmer-device-sample-code"></a>Usando o código de exemplo do dispositivo DIMM

O SDK (Software Development Kit) da plataforma inclui o código de exemplo do dispositivo DIMM. Este código de exemplo demonstra um dispositivo simples baseado em UPnP implementado usando a API de host do dispositivo com tecnologia UPnP.

O \_ diretório DimmerService do DCO contém a implementação para o Device. dll. O diretório RegisterDevice contém a implementação para registrar o Device. dll com o serviço de registrador UPnP no computador. O diretório StepByStep contém projetos do VC 6.0 que correspondem ao "guia passo a passo" contido no arquivo de Readme.txt. O "guia passo a passo" é para os desenvolvedores que não têm a experiência ATL/COM anterior.
