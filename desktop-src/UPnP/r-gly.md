# <a name="r-upnp-apis"></a>R (APIs UPnP)

Contém termos relacionados a UPnP que começam com a letra R.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) i J K L M [N](n-gly.md) O [P](p-gly.md) Q R [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**diretório de recursos**
</dt> <dd>

O diretório de recursos é um diretório local para o dispositivo em que arquivos, como arquivos de ícone, estão localizados.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**dispositivo raiz**
</dt> <dd>

Um dispositivo raiz é o dispositivo mais alto em uma árvore de dispositivos. Por exemplo, se você estiver acessando um dispositivo de combinação de videocassete/TV chamado estação de exibição, há três dispositivos: a estação de visualização, o VCR e a TV. A estação de visualização é considerada o dispositivo raiz.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**dispositivo em execução**
</dt> <dd>

Um dispositivo em execução é um dispositivo que já tinha seu código carregado antes do registro.

Esses são dispositivos para os quais uma descrição de serviço não é conhecida antes do registro.

Exemplos desses dispositivos são pontes, como uma ponte de SCP UPnP. Essa ponte registra a execução de dispositivos com o host do dispositivo.

</dd> </dl>
