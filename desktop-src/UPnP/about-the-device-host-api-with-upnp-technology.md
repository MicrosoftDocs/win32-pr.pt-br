# <a name="about-the-device-host-api"></a>Sobre a API de Host do Dispositivo

A API de Host do Dispositivo com tecnologia UPnP é uma estrutura para implementar a funcionalidade de dispositivo baseada em UPnP na Windows plataforma. Os desenvolvedores que estão criando dispositivos usando a API de [](h-gly.md)Host do Dispositivo com a tecnologia UPnP (conhecida como dispositivos hospedados) precisam implementar apenas a funcionalidade principal do dispositivo. Os desenvolvedores podem contar com o host do dispositivo para lidar com os detalhes específicos do UPnP de descoberta, descrição, controle, eventos e apresentação. O host do dispositivo valida os dados de entrada de clientes baseados em UPnP e formatifica todos os dados de saída de dispositivos hospedados de acordo com a arquitetura do dispositivo UPnP.

As seções a seguir explicam, em geral, como funciona o host do dispositivo UPnP:

-   [Registrando um dispositivo hospedado com o host do dispositivo](registering-a-hosted-device-with-the-device-host.md)
-   [Provedores de dispositivos](device-providers.md)
-   [Eventos](eventing.md)
-   [Apresentação](presentation.md)
