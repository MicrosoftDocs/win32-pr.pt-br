# <a name="about-the-device-host-api"></a>Sobre a API de host do dispositivo

A API de host de dispositivo com tecnologia UPnP é uma estrutura para implementar a funcionalidade de dispositivo baseada em UPnP na plataforma Windows. Os desenvolvedores que estão criando dispositivos usando a API de host do dispositivo com a tecnologia UPnP (conhecida como [*dispositivos hospedados*](h-gly.md)) só precisam implementar a funcionalidade principal do dispositivo. Os desenvolvedores podem contar com o host do dispositivo para lidar com os detalhes específicos do UPnP de descoberta, descrição, controle, eventos e apresentações. O host do dispositivo valida os dados de entrada de clientes baseados em UPnP e formata todos os dados de saída de dispositivos hospedados de acordo com a arquitetura do dispositivo UPnP.

-   [Registrando um dispositivo hospedado no host do dispositivo](registering-a-hosted-device-with-the-device-host.md)
-   [Provedores de dispositivo](device-providers.md)
-   [Eventos](eventing.md)
-   [Apresentação](presentation.md)
