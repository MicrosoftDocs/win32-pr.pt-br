# <a name="about-the-control-point-api"></a>Sobre a API do ponto de controle

A tecnologia UPnP permite a rede dinâmica de dispositivos inteligentes, PCs e outros pontos de controle. A rede dinâmica está disponível em uma variedade de mídias físicas. A implementação da API do ponto de controle com a tecnologia UPnP consiste em um conjunto de objetos COM que são usados para gravar aplicativos e páginas da Web que localizam e controlam dispositivos baseados em UPnP. Por exemplo, um VCR compatível com UPnP pode ser controlado de uma página da Web que foi especificamente projetada para controlá-lo. Para estender esse conceito, os desenvolvedores podem controlar vários dispositivos de um local central, remoto ou de mesmo nível.

Aplicativos criados com a API do ponto de controle com tecnologia UPnP de scripts inseridos em páginas HTML até programas completos em C++. A API do ponto de controle com tecnologia UPnP dá suporte ao sistema de desenvolvimento Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) e C++.

As seções a seguir explicam, em geral, como a API do ponto de controle com a tecnologia UPnP funciona:

-   [Noções básicas do ponto de controle](control-point-basics.md)
-   [Localizando dispositivos](finding-devices.md)
-   [Descrevendo dispositivos](describing-devices.md)
-   [Controlando dispositivos](controlling-devices.md)

Todos os exemplos em toda a documentação da API do ponto de controle com tecnologia UPnP referem-se a dispositivos fictícios, tipos de dispositivo e cadeias de caracteres.
