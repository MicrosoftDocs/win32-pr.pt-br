# <a name="about-the-control-point-api"></a>Sobre a API do Ponto de Controle

A tecnologia UPnP permite a rede dinâmica de dispositivos inteligentes, computadores e outros pontos de controle. A rede dinâmica está disponível em uma variedade de mídias físicas. A implementação da Microsoft da API do Ponto de Controle com a tecnologia UPnP consiste em um conjunto de objetos COM que são usados para escrever aplicativos e páginas da Web que encontram e controlam dispositivos baseados em UPnP. Por exemplo, um VCR com capacidade de UPnP pode ser controlado de uma página da Web projetada especificamente para controlá-lo. Para estender esse conceito, os desenvolvedores podem controlar muitos dispositivos de um local central, remoto ou par.

Os aplicativos criados com a API do Ponto de Controle com a tecnologia UPnP variam de scripts inseridos em páginas HTML a programas C++ completos. A API do Ponto de Controle com a tecnologia UPnP dá suporte ao sistema de desenvolvimento do Microsoft Visual Basic, ao Microsoft Visual Basic VBScript (Scripting Edition) e ao C++.

As seções a seguir explicam, em geral, como funciona a API do Ponto de Controle com a tecnologia UPnP:

-   [Noções básicas do ponto de controle](control-point-basics.md)
-   [Localizar dispositivos](finding-devices.md)
-   [Descrevendo dispositivos](describing-devices.md)
-   [Controlando dispositivos](controlling-devices.md)

Todos os exemplos em toda a API do Ponto de Controle com a documentação de tecnologia UPnP referem-se a dispositivos fictícios, tipos de dispositivos e cadeias de caracteres.
