# <a name="control-point-programming-considerations"></a>Considerações de programação de ponto de controle

Os desenvolvedores que criam aplicativos baseados em UPnP (ou pontos de controle) devem usar esses aplicativos de um \_ cliente coinit APARTMENTTHREADED. Há problemas conhecidos ao usar a API do ponto de controle de um \_ cliente de vários threads coinit em execução sob estresse.
