# <a name="device-host-api-reference"></a>Referência da API do host do dispositivo

As interfaces a seguir fazem parte da API de host do dispositivo com tecnologia UPnP. O host do dispositivo com a tecnologia UPnP dá suporte ao Microsoft Visual Basic e ao C++.

Essa API não fornece suporte para o Microsoft Visual Basic Scripting Edition (VBScript).



| Interface                                                  | Descrição                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**IUPnPDeviceControl**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdevicecontrol)           | Usado pelo host do dispositivo para gerenciar dispositivos e serviços relacionados.                                     |
| [**IUPnPDeviceProvider**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdeviceprovider)         | Usado pelo host do dispositivo para iniciar e parar provedores de dispositivos.                                         |
| [**IUPnPEventSink**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsink)                   | Usado por dispositivos para enviar notificações de eventos para o host do dispositivo.                                     |
| [**IUPnPEventSource**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsource)               | Usado pelo host do dispositivo para assinar ou cancelar a assinatura de eventos fornecidos pelo serviço hospedado.       |
| [**IUPnPRegistrar**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpregistrar)                   | Usado por dispositivos para registrar com o host do dispositivo.                                                   |
| [**IUPnPRemoteEndpointInfo**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpremoteendpointinfo) | Usado por dispositivos para obter informações sobre um solicitante (isto é, um ponto de controle) e a solicitação. |
| [**IUPnPReregistrar**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpreregistrar)               | Usado pelos dispositivos para registrar novamente no host do dispositivo.                                                |
