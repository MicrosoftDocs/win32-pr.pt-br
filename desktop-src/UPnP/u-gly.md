# <a name="u-upnp-apis"></a>U (APIs UPnP)

Contém termos relacionados a UPnP que começam com a letra U.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) i J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T U V W X Y Z

<dl> <dt>

<span id="upnp.u_1_gly"></span><span id="UPNP.U_1_GLY"></span>**UDN**
</dt> <dd>

Significa nome de dispositivo exclusivo. Um UDN identifica exclusivamente um dispositivo; Ele se baseia no UUID (ID exclusiva universal).

</dd> <dt>

<span id="upnp.u_2_gly"></span><span id="UPNP.U_2_GLY"></span>**UTL**
</dt> <dd>

Significa linguagem de modelo UPnP. A linguagem de modelo UPnP é usada para especificar descrições de dispositivo ou de serviço.

</dd> </dl>
