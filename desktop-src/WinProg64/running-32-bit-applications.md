<span data-ttu-id="3ea12-113">O sistema isola os aplicativos de 32 bits de aplicativos de 64 bits, o que inclui a prevenção de colisões de arquivo e registro.</span><span class="sxs-lookup"><span data-stu-id="3ea12-113">The system isolates 32-bit applications from 64-bit applications, which includes preventing file and registry collisions.</span></span> <span data-ttu-id="3ea12-114">Há suporte para aplicativos de console, GUI e serviço.</span><span class="sxs-lookup"><span data-stu-id="3ea12-114">Console, GUI, and service applications are supported.</span></span> <span data-ttu-id="3ea12-115">O sistema fornece interoperabilidade entre o limite de 32/64 para cenários como recortar e colar e COM.</span><span class="sxs-lookup"><span data-stu-id="3ea12-115">The system provides interoperability across the 32/64 boundary for scenarios such as cut and paste and COM.</span></span> <span data-ttu-id="3ea12-116">No entanto, os processos de 32 bits não podem carregar DLLs de 64 bits para execução, e processos de 64 bits não podem carregar DLLs de 32 bits para execução.</span><span class="sxs-lookup"><span data-stu-id="3ea12-116">However, 32-bit processes cannot load 64-bit DLLs for execution, and 64-bit processes cannot load 32-bit DLLs for execution.</span></span> <span data-ttu-id="3ea12-117">Essa restrição não se aplica a DLLs carregadas como arquivos de dados ou arquivos de recurso de imagem; para obter mais informações, consulte [**LoadLibraryEx**](/windows/desktop/api/libloaderapi/nf-libloaderapi-loadlibraryexa).</span><span class="sxs-lookup"><span data-stu-id="3ea12-117">This restriction does not apply to DLLs loaded as data files or image resource files; for more information, see [**LoadLibraryEx**](/windows/desktop/api/libloaderapi/nf-libloaderapi-loadlibraryexa).</span></span>

<span data-ttu-id="3ea12-118">Um aplicativo de 32 bits pode detectar se ele está em execução no WOW64 chamando a função [**IsWow64Process**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process) (use [**IsWow64Process2**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process2) se estiver direcionando para o Windows 10).</span><span class="sxs-lookup"><span data-stu-id="3ea12-118">A 32-bit application can detect whether it is running under WOW64 by calling the [**IsWow64Process**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process) function (use [**IsWow64Process2**](/windows/desktop/api/wow64apiset/nf-wow64apiset-iswow64process2) if targeting Windows 10).</span></span> <span data-ttu-id="3ea12-119">O aplicativo pode obter informações adicionais sobre o processador usando a função [**GetNativeSystemInfo**](/windows/desktop/api/sysinfoapi/nf-sysinfoapi-getnativesysteminfo) .</span><span class="sxs-lookup"><span data-stu-id="3ea12-119">The application can obtain additional information about the processor by using the [**GetNativeSystemInfo**](/windows/desktop/api/sysinfoapi/nf-sysinfoapi-getnativesysteminfo) function.</span></span>

<span data-ttu-id="3ea12-120">Observe que o Windows de 64 bits não oferece suporte à execução de aplicativos baseados no Windows de 16 bits.</span><span class="sxs-lookup"><span data-stu-id="3ea12-120">Note that 64-bit Windows does not support running 16-bit Windows-based applications.</span></span> <span data-ttu-id="3ea12-121">O principal motivo é que os identificadores têm 32 bits significativos no Windows de 64 bits.</span><span class="sxs-lookup"><span data-stu-id="3ea12-121">The primary reason is that handles have 32 significant bits on 64-bit Windows.</span></span> <span data-ttu-id="3ea12-122">Portanto, os identificadores não podem ser truncados e passados para aplicativos de 16 bits sem perda de dados.</span><span class="sxs-lookup"><span data-stu-id="3ea12-122">Therefore, handles cannot be truncated and passed to 16-bit applications without loss of data.</span></span> <span data-ttu-id="3ea12-123">As tentativas de iniciar aplicativos de 16 bits falham com o seguinte erro: **erro de \_ \_ \_ formato exe inválido**.</span><span class="sxs-lookup"><span data-stu-id="3ea12-123">Attempts to launch 16-bit applications fail with the following error: **ERROR\_BAD\_EXE\_FORMAT**.</span></span>
