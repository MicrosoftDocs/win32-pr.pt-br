# <a name="getting-ready-for-64-bit-windows"></a>Preparando-se para o Windows de 64 bits

Um objetivo importante da versão de 64 bits do Windows é tornar mais fácil para os desenvolvedores usarem uma base de código-fonte única para seus aplicativos de 32 bits e de 64 bits. Esta visão geral descreve como fazer com que o código-fonte dê suporte a computação de 32 bits e 64 bits. A familiaridade com [os tipos de dados do Windows](/windows/desktop/WinProg/windows-data-types) existentes ajudará.
