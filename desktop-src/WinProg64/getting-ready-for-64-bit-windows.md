# <a name="getting-ready-for-64-bit-windows"></a>Preparando-se para 64 bits Windows

Uma meta fundamental da versão de 64 bits do Windows é facilitar para os desenvolvedores usarem uma única base de código-fonte para seus aplicativos de 32 bits e 64 bits. Esta visão geral descreve como fazer seu código-fonte dar suporte à computação de 32 bits e 64 bits. Familiaridade com os tipos [de Windows existentes](/windows/desktop/WinProg/windows-data-types) ajudará.
