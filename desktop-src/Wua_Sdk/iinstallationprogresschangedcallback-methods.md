# <a name="iinstallationprogresschangedcallback-methods"></a>Métodos IInstallationProgressChangedCallback

A interface [**IInstallationProgressChangedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallback) define o método a seguir.



| Método                                                        | Descrição                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Invoke**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallback-invoke) | Lida com a notificação da alteração do progresso de uma instalação ou desinstalação assíncrona iniciada por uma chamada para o método [**IUpdateInstaller.BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall) ou [**IUpdateInstaller.BeginUninstall.**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) |
