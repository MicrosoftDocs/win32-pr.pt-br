# <a name="iupdatehistoryentry2-properties"></a>Propriedades de IUpdateHistoryEntry2

A interface [**IUpdateHistoryEntry2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry2) define a propriedade a seguir.



| Propriedade                                              | Descrição                                                            |
|-------------------------------------------------------|------------------------------------------------------------------------|
| [**Categorias**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentry2-get_categories) | Obtém uma coleção das categorias de atualização às quais uma atualização pertence. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a seguinte interface:

-   [**IUpdateHistoryEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry)
