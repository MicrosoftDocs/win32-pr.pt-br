# <a name="iupdateservicemanager-properties"></a>Propriedades de IUpdateServiceManager

A interface [**IUpdateServiceManager**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager) define a propriedade a seguir.



| Propriedade                                           | Descrição                                                                                                        |
|----------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**Serviços**](/windows/win32/api/wuapi/nf-wuapi-iupdateservicemanager-get_services) | Obtém um [**IUpdateServiceCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) dos serviços que estão registrados com o WUA. |
