# <a name="iupdatesearcher3-methods"></a>Métodos IUpdateSearcher3

A interface [**IUpdateSearcher3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher3) define o método a seguir



| Método                                              | Descrição                                                                                                                                    |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SearchScope**](/previous-versions/windows/desktop/ee694850(v=vs.85)) | Obtém e define um valor de enumeração [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope) que indica o escopo das atualizações que são retornadas pela pesquisa. |



 

Para obter informações sobre os membros herdados por essa interface, consulte as interfaces a seguir.

-   [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
-   [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2)
