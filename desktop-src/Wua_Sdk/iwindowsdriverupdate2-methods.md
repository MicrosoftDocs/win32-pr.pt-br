# <a name="iwindowsdriverupdate2-methods"></a>Métodos IWindowsDriverUpdate2

A interface [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) define o método a seguir.



| Método                                                   | Descrição                                       |
|----------------------------------------------------------|---------------------------------------------------|
| [**CopyToCache**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-copytocache) | Copia os binários de atualização externa para uma atualização. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
