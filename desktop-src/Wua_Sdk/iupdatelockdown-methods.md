# <a name="iupdatelockdown-methods"></a>Métodos IUpdateLockdown

A interface [**IUpdateLockdown**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatelockdown) define o método a seguir.



| Método                                       | Descrição                                                                               |
|----------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Bloqueado**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatelockdown-lockdown) | Restringe o acesso aos métodos e às propriedades do objeto que implementa esse método. |
