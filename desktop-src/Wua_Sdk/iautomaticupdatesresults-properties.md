# <a name="iautomaticupdatesresults-properties"></a>Propriedades de IAutomaticUpdatesResults

A interface [**IAutomaticUpdatesResults**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatesresults) define as propriedades a seguir.



| Propriedade                                                                                    | Descrição                                                                                                                                             |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LastInstallationSuccessDate**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastinstallationsuccessdate) | Obtém a última data e hora universal coordenada (UTC) quando Atualizações Automáticas instalou com êxito todas as atualizações, mesmo que tenham ocorrido algumas falhas. |
| [**LastSearchSuccessDate**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastsearchsuccessdate)             | Obtém a última data e hora universal coordenada (UTC) quando AutomaticUpdates pesquisou com êxito as atualizações.                                   |
