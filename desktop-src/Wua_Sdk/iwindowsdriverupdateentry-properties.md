# <a name="iwindowsdriverupdateentry-properties"></a>Propriedades de IWindowsDriverUpdateEntry

A interface [**IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) define as propriedades a seguir.



| Propriedade                                                                     | Descrição                                                                                                          |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_deviceproblemnumber) | Obtém o número do problema do dispositivo que corresponde à atualização de driver do Windows.                                    |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_devicestatus)               | Obtém o status do dispositivo que corresponde à atualização de driver do Windows.                                            |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverclass)                 | Obtém a classe da atualização de driver do Windows.                                                                         |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverhardwareid)       | Obtém o hardware ou o identificador compatível que a atualização de driver do Windows deve corresponder para ser instalável. |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermanufacturer)   | Obtém o nome invariável de idioma do fabricante da atualização de driver do Windows.                                   |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermodel)                 | Obtém o nome do modelo invariável de idioma do dispositivo para o qual a atualização de driver do Windows se destina.                |
| [**Driverprovider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverprovider)           | Obtém o nome invariável de idioma do provedor da atualização de driver do Windows.                                       |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driververdate)             | Obtém a data da versão do driver da atualização do driver do Windows.                                                           |
