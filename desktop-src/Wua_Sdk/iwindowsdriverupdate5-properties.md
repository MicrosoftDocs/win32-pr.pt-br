# <a name="iwindowsdriverupdate5-properties"></a>Propriedades de IWindowsDriverUpdate5

A interface [**IWindowsDriverUpdate5**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate5) define as propriedades a seguir.



| Propriedade                                                     | Descrição                                                                                                                                               |
|--------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Download autobaixado**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autodownload)   | Obtém um valor [**Autodownloadmode**](/windows/win32/api/wuapi/ne-wuapi-autodownloadmode) que indica o modo de download automático da atualização.                                          |
| [**Seleção automarcada**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autoselection) | Obtém um valor de [**Autoseleçãomode**](/windows/win32/api/wuapi/ne-wuapi-autoselectionmode) que indica o modo de seleção automática de uma atualização no painel de controle de Windows Update. |



 

Para obter informações sobre os membros herdados por essa interface, consulte as interfaces a seguir.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
-   [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4)
