# <a name="iupdatesession2-properties"></a>Propriedades de IUpdateSession2

A interface [**IUpdateSession2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession2) define a propriedade a seguir.



| Propriedade                                         | Descrição                                                                   |
|--------------------------------------------------|-------------------------------------------------------------------------------|
| [**UserLocale**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession2-get_userlocale) | Obtém e define a localidade preferencial para a qual as informações de atualização são recuperadas. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession)
