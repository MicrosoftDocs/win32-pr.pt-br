# <a name="iupdatedownloadcontentcollection-properties"></a>Propriedades de IUpdateDownloadContentCollection

A interface **IUpdateDownloadContentCollection** define as propriedades a seguir.



| Propriedade                                                       | Descrição                                                                                                                             |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get__newenum) | Obtém uma interface [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que é usada para enumerar a coleção.                    |
| [**Contar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_count)        | Obtém o número de elementos na coleção.                                                                                          |
| [**Item**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_item)          | Obtém o conteúdo de download de uma atualização de uma interface [**IUpdateDownloadContentCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadcontentcollection) . |
