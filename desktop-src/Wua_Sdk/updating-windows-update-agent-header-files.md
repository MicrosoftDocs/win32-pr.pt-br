# <a name="updating-windows-update-agent-header-files"></a>Atualizando arquivos de cabeçalho do agente Windows Update

As APIs do agente de Windows Update eram anteriormente um download separado. Agora eles estão incluídos como parte do SDK do Windows. Para baixar o SDK do Windows vá para [esta página](https://developer.microsoft.com/windows/downloads/windows-10-sdk).

Para obter mais informações sobre como atualizar o WUA, consulte [atualizando o agente de Windows Update](updating-the-windows-update-agent.md).
