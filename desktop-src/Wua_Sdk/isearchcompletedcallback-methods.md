# <a name="isearchcompletedcallback-methods"></a>Métodos ISearchCompletedCallback

A interface [**ISearchCompletedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-isearchcompletedcallback) define o método a seguir.



| Método                                            | Descrição                                                                                                                                                                     |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Chame**](/windows/desktop/api/Wuapi/nf-wuapi-isearchcompletedcallback-invoke) | Manipula a notificação de conclusão de uma pesquisa assíncrona que é iniciada chamando o método [**IUpdateSearcher. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) . |
