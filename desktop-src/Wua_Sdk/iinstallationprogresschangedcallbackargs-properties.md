# <a name="iinstallationprogresschangedcallbackargs-properties"></a>Propriedades de IInstallationProgressChangedCallbackArgs

A interface [**IInstallationProgressChangedCallbackArgs**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallbackargs) define a propriedade a seguir.



| Propriedade                                                              | Descrição                                                                                                                        |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Progresso**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallbackargs-get_progress) | Obtém uma interface que contém o progresso da instalação assíncrona ou da desinstalação no momento em que o retorno de chamada foi feito. |
