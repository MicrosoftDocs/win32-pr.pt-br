# <a name="istringcollection-methods"></a>Métodos IStringCollection

A interface [**IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) define os métodos a seguir.



| Método                                         | Descrição                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Adicionar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Adiciona um item à coleção.                                |
| [**Limpar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Remove todos os elementos da coleção.                  |
| [**Copiar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Cria uma cópia de leitura/gravação profunda da coleção.              |
| [**Inserir**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Insere um item na coleção na posição especificada. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Remove o item no índice especificado da coleção.   |
