# <a name="istringcollection-methods"></a>Métodos isastringcollection

A interface [**isastringcollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) define os métodos a seguir.



| Método                                         | Descrição                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Agrega**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Adiciona um item à coleção.                                |
| [**Formatação**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Remove todos os elementos da coleção.                  |
| [**Copiar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Cria uma cópia de leitura/gravação profunda da coleção.              |
| [**Inserido**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Insere um item na coleção na posição especificada. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Remove o item no índice especificado da coleção.   |
