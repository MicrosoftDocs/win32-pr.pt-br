# <a name="iupdate4-properties"></a>Propriedades de IUpdate4

A interface [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4) define a propriedade a seguir.



| Propriedade                            | Descrição                                                            |
|-------------------------------------|------------------------------------------------------------------------|
| [**Peruser**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser) | Obtém um valor booliano que indica se esta é uma atualização por usuário. |



 

Para obter informações sobre os membros herdados por essa interface, consulte as seguintes interfaces:

-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
