# <a name="iupdateidentity-properties"></a>Propriedades de IUpdateIdentity

A interface [**IUpdateIdentity**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateidentity) define as propriedades a seguir.



| Propriedade                                                 | Descrição                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| [**RevisionNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_revisionnumber) | Obtém o número de revisão de uma atualização.                 |
| [**UpdateID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_updateid)             | Obtém o identificador independente de revisão de uma atualização. |
