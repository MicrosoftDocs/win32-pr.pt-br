# <a name="iwebproxy-methods"></a>Métodos IWebProxy

A interface [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) define os métodos a seguir.



| Método                                                                         | Descrição                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**PromptForCredentials**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Solicita ao usuário a senha a ser usada para autenticação de proxy.  |
| [**PromptForCredentialsFromHwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Solicita ao usuário a senha a ser usada para autenticação de proxy.  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Define a senha a ser enviada ao servidor proxy para autenticação. |
