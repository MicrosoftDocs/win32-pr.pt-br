# <a name="isearchjob-properties"></a>Propriedades de ISearchJob

A interface [**ISearchJob**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) define as propriedades a seguir.



| Propriedade                                      | Descrição                                                                                                                                                 |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AsyncState**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_asyncstate)   | Obtém o objeto de estado específico do chamador que é passado para o método [**IUpdateSearch. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) .                         |
| [**IsCompleted**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_iscompleted) | Obtém um valor booliano que indica se a chamada para o método [**IUpdateSearch. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) é completamente processada. |
