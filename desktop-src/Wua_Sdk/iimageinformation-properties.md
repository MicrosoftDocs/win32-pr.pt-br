# <a name="iimageinformation-properties"></a>Propriedades de IImageInformation

A interface [**IImageInformation**](/windows/desktop/api/Wuapi/nn-wuapi-iimageinformation) define as propriedades a seguir.



| Propriedade                                     | Descrição                              |
|----------------------------------------------|------------------------------------------|
| [**AltText**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_alttext) | Obtém o texto alternativo para a imagem.   |
| [**Altura**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_height)   | Obtém a altura da imagem, em pixels. |
| [**Fonte**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_source)   | Obtém o local de origem da imagem.   |
| [**Largura**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_width)     | Obtém a largura da imagem, em pixels.  |
