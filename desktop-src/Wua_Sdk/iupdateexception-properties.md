# <a name="iupdateexception-properties"></a>Propriedades de IUpdateException

A interface [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) define as propriedades a seguir.



| Propriedade                                    | Descrição                                                |
|---------------------------------------------|------------------------------------------------------------|
| [**Contexto**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_context) | Obtém o contexto da exceção.                         |
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_hresult) | obtém o código **HRESULT** baseado em Windows para a exceção. |
| [**Mensagem**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_message) | Obtém uma mensagem que descreve a exceção atual.       |
