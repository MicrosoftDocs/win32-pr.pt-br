# <a name="isearchresult-properties"></a>Propriedades de ISearchResult

A interface [**ISearchResult**](/windows/desktop/api/Wuapi/nn-wuapi-isearchresult) define as propriedades a seguir.



| Propriedade                                               | Descrição                                                                                                   |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**Resultcode**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_resultcode)         | Obtém [**uma enumeração OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) que especifica o resultado de uma pesquisa. |
| [**RootCategories**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_rootcategories) | Obtém uma coleção de interface das categorias raiz que estão disponíveis no computador no momento.             |
| [**Atualizações**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_updates)               | Obtém uma coleção de interface das atualizações resultantes de uma pesquisa.                                        |
| [**Avisos**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_warnings)             | Obtém uma coleção dos avisos resultantes de uma pesquisa.                                                  |
