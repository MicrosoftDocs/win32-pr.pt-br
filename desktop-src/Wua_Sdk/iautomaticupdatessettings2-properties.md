# <a name="iautomaticupdatessettings2-properties"></a>Propriedades de IAutomaticUpdatesSettings2

A interface [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2) define a propriedade a seguir.



| Propriedade                                                                                  | Descrição                                                                                                                                                         |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) | Obtém e define um valor booliano que indica se deve incluir atualizações opcionais ou recomendadas quando uma pesquisa de atualizações e instalação de atualizações for executada. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
