# <a name="iupdateinstaller-methods"></a>Métodos IUpdateInstaller

A interface [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller) define os métodos a seguir.



| Método                                                    | Descrição                                                                          |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall)     | Inicia uma instalação assíncrona das atualizações.                                  |
| [**BeginUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) | Inicia uma desinstalação assíncrona das atualizações.                                |
| [**Fim da instalação**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-endinstall)         | Conclui uma instalação assíncrona das atualizações.                               |
| [**Fim da desinstalação**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-enduninstall)     | Conclui uma desinstalação assíncrona das atualizações.                             |
| [**Instalar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-install)               | Inicia uma instalação síncrona das atualizações.                                    |
| [**RunWizard**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-runwizard)           | Inicia um assistente que orienta o usuário local por meio das etapas para instalar as atualizações. |
| [**Desinstalar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-uninstall)           | Inicia uma desinstalação síncrona das atualizações.                                  |
