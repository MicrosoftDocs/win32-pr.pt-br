# <a name="iupdateendpointauthprovider-methods"></a>Métodos IUpdateEndpointAuthProvider

A interface [**IUpdateEndpointAuthProvider**](iupdateendpointauthprovider.md) define os métodos a seguir.



| Método                                                                                             | Descrição                                                                         |
|----------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**GetEndpointToken**](iupdateendpointauthprovider-getendpointtoken.md)                           | Solicite um token para o ponto de extremidade do serviço usando as credenciais especificadas.    |
| [**GetPreferredEndpointTokenType**](iupdateendpointauthprovider-getpreferredendpointtokentype.md) | Retorna o tipo preferencial de token de autenticação para o ponto de extremidade do serviço. |
