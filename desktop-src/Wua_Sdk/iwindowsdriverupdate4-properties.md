# <a name="iwindowsdriverupdate4-properties"></a>Propriedades de IWindowsDriverUpdate4

A interface [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4) define as propriedades a seguir.



| Propriedade                                                                               | Descrição                                                                 |
|----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**Peruser**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_peruser)                                       | Obtém um valor booliano que indica se uma atualização é uma atualização por usuário. |
| [**WindowsDriverUpdateEntries**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_windowsdriverupdateentries) | Obtém as entradas de atualização de driver que são aplicáveis para a atualização.          |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
