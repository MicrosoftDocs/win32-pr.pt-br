# <a name="istringcollection-properties"></a>Propriedades de IStringCollection

A interface [**IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) define as propriedades a seguir.



| Propriedade                                        | Descrição                                                                                                                     |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get__newenum) | Obtém uma interface [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que pode ser usada para enumerar a coleção. |
| [**Contagem**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_count)        | Obtém o número de elementos na coleção.                                                                                  |
| [**Item**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_item)          | Obtém ou define uma cadeia de caracteres na coleção.                                                                                        |
| [**ReadOnly**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_readonly)  | Obtém um valor booliana que indica se a coleção é somente leitura.                                                        |
