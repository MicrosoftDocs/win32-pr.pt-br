# <a name="iautomaticupdates-properties"></a>Propriedades de IAutomaticUpdates

A interface [**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) define as propriedades a seguir.



| Propriedade                                                   | Descrição                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**Habilitado**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | Obtém um valor booliano que indica se todos os componentes que Atualizações Automáticas requer estão disponíveis. |
| [**Configurações**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | Obtém as definições de configuração para Atualizações Automáticas.                                                        |
