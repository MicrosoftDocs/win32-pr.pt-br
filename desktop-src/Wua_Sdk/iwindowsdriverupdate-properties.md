# <a name="iwindowsdriverupdate-properties"></a>Propriedades de IWindowsDriverUpdate

A interface [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) define as propriedades a seguir.



| Propriedade                                                                                                    | Descrição                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[**Endereço** DeviceProblemNumber](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Obtém o número do problema do dispositivo que corresponde à atualização de driver do Windows.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Obtém o status do dispositivo que corresponde à atualização de driver do Windows.                             |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Obtém a classe da atualização de driver do Windows.                                                          |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Obtém a ID de hardware ou a ID compatível à qual a atualização de driver do Windows deve corresponder para ser instalada.    |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Obtém o nome invariável de idioma do fabricante da atualização de driver do Windows.                    |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Obtém o nome do modelo invariável de idioma do dispositivo para o qual a atualização de driver do Windows se destina. |
| [**Driverprovider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Obtém o nome invariável de idioma do provedor da atualização de driver do Windows.                        |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Obtém a data da versão do driver da atualização do driver do Windows.                                            |
