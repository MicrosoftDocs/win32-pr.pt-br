# <a name="iwindowsdriverupdate-properties"></a>Propriedades de IWindowsDriverUpdate

A interface [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) define as propriedades a seguir.



| Propriedade                                                                                                    | Descrição                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**Endereço DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Obtém o número do problema do dispositivo que corresponde à atualização Windows driver.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Obtém o status do dispositivo que corresponde à atualização Windows driver.                             |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Obtém a classe da atualização Windows driver.                                                          |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Obtém a ID de hardware ou a ID compatível que a atualização Windows driver de hardware deve corresponder para ser instalável.    |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Obtém o nome invariável de idioma do fabricante da atualização Windows driver.                    |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Obtém o nome do modelo invariável de idioma do dispositivo para o qual a atualização Windows driver é destinada. |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Obtém o nome invariável de idioma do provedor da atualização Windows driver.                        |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Obtém a data de versão do driver da Windows do driver.                                            |
