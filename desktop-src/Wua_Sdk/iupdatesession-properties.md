# <a name="iupdatesession-properties"></a>Propriedades de IUpdateSession

A interface [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession) define as propriedades a seguir.



| Propriedade                                                          | Descrição                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**ClientApplicationID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_clientapplicationid) | Obtém e define o aplicativo cliente atual.                                |
| [**ReadOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_readonly)                       | Obtém um valor booliano que indica se o objeto de sessão é somente leitura. |
| [**Atualizar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_webproxy)                       | Obtém e define as configurações de proxy que são usadas para acessar o servidor.         |
