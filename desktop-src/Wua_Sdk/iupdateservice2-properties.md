# <a name="iupdateservice2-properties"></a>Propriedades de IUpdateService2

A interface **IUpdateService2** define a propriedade a seguir.



| Propriedade                                                         | Descrição                                                                                                                                                                         |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IsDefaultAUService**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice2-get_isdefaultauservice) | Obtém um valor booliano que indica se o serviço está registrado com Atualizações Automáticas e se o serviço está sendo usado atualmente pelo Atualizações Automáticas como o serviço padrão. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice)
