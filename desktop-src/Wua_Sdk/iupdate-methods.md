# <a name="iupdate-methods"></a>Métodos IUpdate

A interface [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate) define os métodos a seguir.



| Método                                         | Descrição                                                                                      |
|------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**AcceptEula**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-accepteula)       | aceita os termos de licença para Software Microsoft associados ao Windows Update.<br/> |
| [**CopyFromCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-copyfromcache) | Copia o conteúdo de uma atualização para um caminho especificado.<br/>                                 |
