# <a name="iinstallationresult-methods"></a>Métodos IInstallationResult

A interface [**IInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationresult) define o método a seguir.



| Método                                                         | Descrição                                                                                                                                        |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationresult-getupdateresult) | Retorna uma interface [**IUpdateInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstallationresult) que contém os resultados da instalação para uma atualização especificada. |
