# <a name="iupdatesearcher2-properties"></a>Propriedades de IUpdateSearcher2

A interface [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2) define a propriedade a seguir.



| Propriedade                                                                  | Descrição                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**IgnoreDownloadPriority**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher2-get_ignoredownloadpriority) | Obtém e define um valor booliano que indica se a prioridade de download deve ser ignorada. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
