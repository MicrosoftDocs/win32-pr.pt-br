# <a name="iautomaticupdatessettings3-properties"></a>Propriedades de IAutomaticUpdatesSettings3

A interface [**IAutomaticUpdatesSettings3**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings3) define as propriedades a seguir.



| Propriedade                                                                                  | Descrição                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FeaturedUpdatesEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_featuredupdatesenabled)       | Obtém e define um valor booliano que indica se as notificações para atualizações em destaque devem ser exibidas.                                             |
| [**NonAdministratorsElevated**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_nonadministratorselevated) | Obtém e define um valor booliano que indica se não administradores podem executar algumas ações relacionadas à atualização sem aprovação do administrador. |



 

Para obter informações sobre os membros herdados por essa interface, consulte as interfaces a seguir.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
-   [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)
