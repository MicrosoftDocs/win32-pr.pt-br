# <a name="iwindowsdriverupdate3-properties"></a>Propriedades de IWindowsDriverUpdate3

A interface **IWindowsDriverUpdate3** define a propriedade a seguir.



| Propriedade                                               | Descrição                                                                                                             |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate3-get_browseonly) | Obtém um valor booliano que indica se uma atualização pode ser descoberta apenas navegando pelas atualizações disponíveis. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
