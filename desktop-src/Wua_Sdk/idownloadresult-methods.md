# <a name="idownloadresult-methods"></a>Métodos IDownloadResult

A interface [**IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) define o método a seguir.



| Método                                                     | Descrição                                                                                                                              |
|------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-getupdateresult) | Retorna uma interface [**IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) que contém informações de download para a atualização especificada. |
