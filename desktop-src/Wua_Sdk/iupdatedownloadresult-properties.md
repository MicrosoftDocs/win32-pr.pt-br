# <a name="iupdatedownloadresult-properties"></a>Propriedades de IUpdateDownloadResult

A interface [**IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) define as propriedades a seguir.



| Propriedade                                               | Descrição                                                                                                                          |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_hresult)       | Obtém o valor de **HRESULT** da exceção, se houver, que é gerado durante uma operação em uma atualização.                                       |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_resultcode) | Obtém um valor de enumeração [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) que especifica o resultado de uma operação em uma atualização. |
