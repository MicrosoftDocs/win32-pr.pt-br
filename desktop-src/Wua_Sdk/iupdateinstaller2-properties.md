# <a name="iupdateinstaller2-properties"></a>Propriedades de IUpdateInstaller2

A interface [**IUpdateInstaller2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller2) define a propriedade a seguir.



| Propriedade                                           | Descrição                                                                                                                       |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**ForceQuiet**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller2-get_forcequiet) | obtém e define um valor booliano que indica se Windows Installer é forçado a instalar as atualizações sem interação do usuário. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)
