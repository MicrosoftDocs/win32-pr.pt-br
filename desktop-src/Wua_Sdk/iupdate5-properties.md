# <a name="iupdate5-properties"></a>Propriedades de IUpdate5

A interface [**IUpdate5**](/windows/win32/api/wuapi/nn-wuapi-iupdate5) define as propriedades a seguir.



| Propriedade                                        | Descrição                                                                                            |
|-------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Download autobaixado**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autodownload)   | Obtém um valor que indica o modo de download automático da atualização.                                         |
| [**Seleção automarcada**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autoselection) | Obtém um valor que indica o modo de seleção automática de Update no painel de controle de Windows Update. |



 

Para obter informações sobre os membros herdados por essa interface, consulte as seguintes interfaces:

-   [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4)
-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
