# <a name="idownloadresult-properties"></a>Propriedades de IDownloadResult

A interface [**IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) define as propriedades a seguir.



| Propriedade                                         | Descrição                                                                                                     |
|--------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_hresult)       | Obtém o número do código de exceção se um número de código de exceção é gerado durante um download.                         |
| [**Resultcode**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_resultcode) | Obtém [**uma enumeração OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) que especifica o resultado de um download. |
