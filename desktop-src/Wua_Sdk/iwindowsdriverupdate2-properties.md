# <a name="iwindowsdriverupdate2-properties"></a>Propriedades de IWindowsDriverUpdate2

A interface [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) define as propriedades a seguir.



| Propriedade                                                       | Descrição                                                                                                             |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_cveids)                 | Contém uma coleção de identificadores de CVE (vulnerabilidades e exposições) comuns que estão associados a uma atualização. |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_ispresent)           | Obtém um valor booliano que indica se uma atualização está instalada no computador.                                     |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_rebootrequired) | Obtém um valor booliano que indica se o computador deve ser reiniciado após a instalação ou desinstalação de uma atualização.    |



 

Para obter informações sobre os membros herdados por essa interface, consulte a interface a seguir.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
