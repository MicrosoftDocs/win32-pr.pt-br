# <a name="iupdate2-properties"></a>Propriedades de IUpdate2

A [**interface IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) define as propriedades a seguir.



| Propriedade                                          | Descrição                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Obtém uma coleção de IDs de CVE (vulnerabilidades e exposições comuns) associadas à atualização.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Obtém um valor booliana que indica se uma atualização está presente em um computador.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Obtém um valor boolano que indica se uma reinicialização do sistema é necessária em um computador para concluir a instalação ou a desinstalação de uma atualização. |



 

Para obter informações sobre os membros herdados por essa interface, consulte a seguinte interface:

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
