# <a name="other-sources-of-windows-update-agent-information"></a>Outras fontes de informações do Windows Update Agent

O SDK da [API do agente do Windows Update](portal-client.md)contém as descrições mais completas e atuais da API. É a principal fonte do material de referência do desenvolvedor para o WUA.

## <a name="software-development-kit-sdk-for-windows-update-agent-wua"></a>SDK (Software Development Kit) para o WUA (Windows Update Agent)

Os recursos a seguir descrevem como implantar e administrar Windows Server Update Services e WUA:

-   [Windows Server Update Services](/documentation/)

    Você pode baixar o SDK do Windows Server Update Services e examinar outros documentos que descrevem como instalar, implantar e manter os serviços do Windows Update.

-   [Download do MSDN e centro de código](/powerapps/developer/common-data-service/org-service/subscribe-sdk-assembly-updates-using-nuget)

    Você pode baixar atualizações para os exemplos, cabeçalhos e bibliotecas do SDK do WUA. Os downloads no centro de download e de código do MSDN são organizados por tópico. Você também pode encontrar downloads no MSDN, o Microsoft Developer Network, usando a ferramenta de pesquisa do [centro de download da Microsoft](https://go.microsoft.com/fwlink/p/?linkid=83468) .

## <a name="windows-update-agent-wua-newsgroups"></a>Grupos de notícias do WUA (agente de Windows Update)

Você pode ler e postar informações em um grupo de notícias que discute o WUA ou Pesquisar Postagens de grupos de notícias arquivadas.

O MSDN fornece acesso aos seguintes grupos de notícias públicos:

-   [Grupos de notícias públicos da Microsoft](https://go.microsoft.com/fwlink/p/?linkid=84395)

    Este grupo de notícias é orientado para profissionais de ti com perguntas sobre implantação e personalização.

-   [Grupos de notícias do MSDN](https://go.microsoft.com/fwlink/p/?linkid=83943)

    Este grupo de notícias é orientado para os desenvolvedores com perguntas sobre o desenvolvimento de aplicativos e de infraestrutura.
