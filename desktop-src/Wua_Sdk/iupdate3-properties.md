# <a name="iupdate3-properties"></a>Propriedades de IUpdate3

A interface [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3) define a propriedade a seguir.



| Propriedade                                  | Descrição                                                                                                             |
|-------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate3-get_browseonly) | Obtém um valor booliano que indica se uma atualização pode ser descoberta apenas navegando pelas atualizações disponíveis. |



 

Para obter informações sobre os membros herdados por essa interface, consulte as seguintes interfaces:

-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
