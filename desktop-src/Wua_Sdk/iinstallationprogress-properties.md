# <a name="iinstallationprogress-properties"></a>Propriedades de IInstallationProgress

A interface [**IInstallationProgress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) define as propriedades a seguir.



| Propriedade                                                                                   | Descrição                                                                                                                                        |
|--------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CurrentUpdateIndex**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdateindex)                     | Obtém um valor de índice com base em zero que especifica a atualização atualmente instalada ou desinstalada quando várias atualizações foram selecionadas. |
| [**CurrentUpdatePercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdatepercentcomplete) | Obtém a distância em que o processo de instalação ou desinstalação da atualização atual foi progredido, como uma porcentagem.                                    |
| [**PercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_percentcomplete)                           | Obtém a distância em que o processo geral de instalação ou desinstalação foi progredido, como uma porcentagem.                                                   |
