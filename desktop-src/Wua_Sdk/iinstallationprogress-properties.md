# <a name="iinstallationprogress-properties"></a>Propriedades de IInstallationProgress

A interface [**IInstallationProgress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) define as propriedades a seguir.



| Propriedade                                                                                   | Descrição                                                                                                                                        |
|--------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CurrentUpdateIndex**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdateindex)                     | Obtém um valor de índice baseado em zero que especifica a atualização que está sendo instalada ou desinstalada no momento quando várias atualizações foram selecionadas. |
| [**CurrentUpdatePercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdatepercentcomplete) | Obtém até que ponto o processo de instalação ou desinstalação para a atualização atual progrediu, como um percentual.                                    |
| [**PercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_percentcomplete)                           | Obtém até que ponto o processo geral de instalação ou desinstalação avançou, como um percentual.                                                   |
