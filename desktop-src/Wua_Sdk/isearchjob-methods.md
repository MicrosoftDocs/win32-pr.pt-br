# <a name="isearchjob-methods"></a>Métodos ISearchJob

A interface [**ISearchJob**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) define os métodos a seguir.



| Método                                          | Descrição                                                                          |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**Eliminação**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-cleanup)           | Aguarda a conclusão de uma operação assíncrona e, em seguida, libera todos os retornos de chamada. |
| [**RequestAbort**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-requestabort) | Faz uma solicitação para cancelar uma pesquisa assíncrona.                                    |
