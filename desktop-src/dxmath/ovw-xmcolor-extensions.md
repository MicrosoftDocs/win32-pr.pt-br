# <a name="xmcolor-extensions"></a>Extensões XMCOLOR

[**XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) descreve um vetor de cor ARGB (Azul Vermelho Vermelho Alfa) de 32 bits, em que cada canal de cor é especificado por 8 bits.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
