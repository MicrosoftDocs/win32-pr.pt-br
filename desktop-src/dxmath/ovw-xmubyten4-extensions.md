# <a name="xmubyten4-extensions"></a>Extensões XMUBYTEN4

[**XMUBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmubyten4) descreve um vetor 3D para armazenar valores normalizados não assinados como inteiros não assinados de 8 bits (1 byte).

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
