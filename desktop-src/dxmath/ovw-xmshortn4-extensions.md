# <a name="xmshortn4-extensions"></a>Extensões XMSHORTN4

[**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) descreve o vetor 4D para armazenar valores normalizados assinados como inteiros de 16 bits assinados, (tipo `int16_t` ).

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
