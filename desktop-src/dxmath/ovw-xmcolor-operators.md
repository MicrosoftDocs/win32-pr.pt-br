# <a name="xmcolor-operators"></a>Operadores XMCOLOR

A estrutura [**XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                            | Descrição                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmcolor-operatoruint32_t)<br/> | Retorna uma instância do `uint32_t` que contém os componentes da `XMCOLOR` instância em um formato empacotado. <br/> |
| [**operador =**](xmcolor-operator-eq.md)<br/>               | Atribui dados de componente de vetor à instância atual do `XMCOLOR` .<br/>                                        |
