# <a name="xmfloat3-constructors"></a>Construtores XMFLOAT3

Construtores expostos por `XMFLOAT3` .

Constrói instâncias da estrutura [**XMFLOAT3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) .

> [!Note]  
> Os construtores listados aqui só estão disponíveis em C++.

 

### <a name="overload-list"></a>Lista de sobrecargas




| Construtor | Descrição | 
|-------------|-------------|
| [<strong>XMFLOAT3 ()</strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat3-xmfloat3(constfloat)) | Construtor padrão para <code>XMFLOAT3</code> .<br /> Construtor padrão para [<strong>XMFLOAT3</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat3).<br /><blockquote>[!Note]<br />Esse construtor só está disponível em C++.</blockquote><br /> | 
| [<strong>XMFLOAT3 (const float *)</strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat3a-xmfloat3a(constfloat)) | Inicializa uma nova instância de <code>XMFLOAT3</code> de um argumento de matriz de três elementos <code>float</code> .<br /> Esse construtor inicializa uma nova instância de [<strong>XMFLOAT3</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) de um de um argumento de matriz de três elementos <code>float</code> .<br /><blockquote>[!Note]<br />Esse construtor só está disponível em C++.</blockquote><br /> | 
| [<strong>XMFLOAT3 (float, float, float)</strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat3-xmfloat3(float_float_float)) | Inicializa uma nova instância do de <code>XMFLOAT3</code> três <code>float</code> argumentos.<br /> Esse construtor inicializa uma nova instância de [<strong>XMFLOAT3</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) de um dos três <code>float</code> argumentos.<br /><blockquote>[!Note]<br />Esse construtor só está disponível em C++.</blockquote><br /> |
