# <a name="operator--operators"></a>operadores de operador \*

Operadores de multiplicação

### <a name="overload-list"></a>Lista de sobrecargas




| Operador | Descrição | 
|----------|-------------|
| <a href="/previous-versions/windows/desktop/legacy/ee421390(v=vs.85)"><strong>XMVECTOR:: Operator * (float, XMVECTOR)</strong></a> | Multiplique um valor de ponto flutuante por uma instância do <code>XMVECTOR</code> , retornando o resultado de uma nova instância do <code>XMVECTOR</code> .<br /> O <code>operator *</code> multiplica um valor de ponto flutuante em relação a cada componente de uma instância do <a href="xmvector-data-type.md"><strong>tipo de dados XMVECTOR</strong></a>, retornando uma nova <code>XMVECTOR</code> instância cujos componentes contêm o resultado. <br /><blockquote>[!Note]<br />Esse operador só está disponível em C++.</blockquote><br /> | 
| <a href="/previous-versions/windows/desktop/legacy/ee421391(v=vs.85)"><strong>XMVECTOR:: Operator * (XMVECTOR, float)</strong></a> | Multiplique uma instância de <code>XMVECTOR</code> por um valor de ponto flutuante, retornando o resultado de uma nova instância do <code>XMVECTOR</code> .<br /> O <code>operator *</code> multiplica cada componente de uma instância do <a href="xmvector-data-type.md"><strong>tipo de dados XMVECTOR</strong></a> por um valor de ponto flutuante, retornando uma nova <code>XMVECTOR</code> instância cujos componentes contêm o resultado. <br /><blockquote>[!Note]<br />Esse operador só está disponível em C++.</blockquote><br /> | 
| <a href="/previous-versions/windows/desktop/legacy/ee421392(v=vs.85)"><strong>XMVECTOR:: Operator * (XMVECTOR, XMVECTOR)</strong></a> | Multiplica uma instância de <code>XMVECTOR</code> por uma segunda instância, retornando o resultado em uma terceira instância. <br /> O <code>operator *</code> multiplica cada componente de uma instância do <a href="xmvector-data-type.md"><strong>tipo de dados XMVECTOR</strong></a> pelo componente correspondente em uma segunda instância do <code>XMVECTOR</code> , retornando uma nova <code>XMVECTOR</code> instância que contém o resultado. <br /><blockquote>[!Note]<br />Esse operador só está disponível em C++.</blockquote><br /> |
