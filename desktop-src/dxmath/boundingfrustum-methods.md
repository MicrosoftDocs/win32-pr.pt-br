# <a name="boundingfrustum-methods"></a>Métodos BoundingFrustum



| Método                                                                  | Descrição                                                                                                    |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**Terá**](boundingfrustum-contains.md)<br/>                 | Testa se o [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) contém um objeto especificado.<br/>           |
| [**Intersecciona esses**](boundingfrustum-intersects.md)<br/>             | Testa o [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para interseção com outro objeto.<br/>          |
| [**Transform**](boundingfrustum-transform.md)<br/>               | Transforma o [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                                          |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Testa se o [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) está contido no frustum especificado.<br/> |
| [**CreateFromMatrix**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-createfrommatrix)<br/> | Cria um [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) a partir da matriz de projeção especificada.<br/>          |
| [**Getvértices**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-getcorners)<br/>             | Obtém os cantos que constituem o [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                          |
| [**Getplans**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-getplanes)<br/>               | Obtém os planos que compõem o [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                           |
| [**atribuição de op \_**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-operator-assign(boundingfrustum__))<br/>      | Copia valores de outro [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                              |
