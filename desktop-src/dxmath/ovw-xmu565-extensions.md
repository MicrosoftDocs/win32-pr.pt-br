# <a name="xmu565-extensions"></a>Extensões XMU565

[**XMU565**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) descreve um vetor 4D com componentes x-e z-bit sem sinal de 5 bits e um componente y sem sinal de 6 bits.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
