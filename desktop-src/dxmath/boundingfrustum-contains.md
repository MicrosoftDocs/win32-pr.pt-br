# <a name="boundingfrustumcontains-methods"></a>Métodos BoundingFrustum.Contains

Testa se [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contém um objeto especificado.

### <a name="overload-list"></a>Lista de sobrecargas

| Método | Descrição |
|-|-|
| [**BoundingFrustum::Contains (const BoundingBox&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingbox_)) | Testa se [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contém o [**BoundingBox especificado.**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) |
| [**BoundingFrustum::Contains (const BoundingFrustum&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingfrustum_)) | Testa se [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contém **o BoundingFrustum especificado.** |
| [**BoundingFrustum::Contains (const BoundingOrientedBox&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingorientedbox_)) | Testa se [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contém o [**BoundingOrientedBox especificado.**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingorientedbox) |
| [**BoundingFrustum::Contains (const BoundingSphere&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingsphere_)) | Testa se [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contém o [**BoundingSphere especificado.**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingsphere) |
| [**BoundingFrustum::Contains (FXMVECTOR)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains) | Testa se [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contém o ponto especificado. |
| [**BoundingFrustum::Contains (FXMVECTOR, FXMVECTOR, FXMVECTOR)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(fxmvector_fxmvector_fxmvector)) | Testa se [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contém o triângulo especificado. |
