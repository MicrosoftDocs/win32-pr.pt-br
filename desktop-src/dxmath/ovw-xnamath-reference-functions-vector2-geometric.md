# <a name="directxmath-library-2d-vector-geometric-functions"></a>Funções geométricas de vetor 2D da Biblioteca DirectXMath

Lista as funções geométricas de vetor 2D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                 | Descrição                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector2AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormals)<br/>       | Calcula o ângulo radiano entre dois vetores 2D normalizados.<br/>       |
| [**XMVector2AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormalsest)<br/> | Estima o ângulo radiano entre dois vetores 2D normalizados.<br/>      |
| [**XMVector2AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweenvectors)<br/>       | Calcula o ângulo radiano entre dois vetores 2D.<br/>                  |
| [**XMVector2ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplength)<br/>                       | Fixa o comprimento de um vetor 2D para um determinado intervalo.<br/>                 |
| [**XMVector2ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplengthv)<br/>                     | Fixa o comprimento de um vetor 2D para um determinado intervalo.<br/>                 |
| [**XMVector2Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector2cross)<br/>                                   | Calcula o produto cruzado 2D.<br/>                                     |
| [**XMVector2Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector2dot)<br/>                                       | Calcula o produto de ponto entre vetores 2D.<br/>                       |
| [**XMVector2InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector2inbounds)<br/>                             | Testa se os componentes de um vetor 2D estão dentro dos limites definidos.<br/> |
| [**XMVector2IntersectLine**](/windows/win32/api/directxmath/nf-directxmath-xmvector2intersectline)<br/>                   | Localiza a interseção de duas linhas.<br/>                               |
| [**XMVector2Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector2length)<br/>                                 | Calcula o comprimento de um vetor 2D.<br/>                                |
| [**XMVector2LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthest)<br/>                           | Estima o comprimento de um vetor 2D.<br/>                               |
| [**XMVector2LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthsq)<br/>                             | Calcula o quadrado do comprimento de um vetor 2D.<br/>                  |
| [**XMVector2LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector2linepointdistance)<br/>           | Calcula a distância mínima entre uma linha e um ponto.<br/>          |
| [**XMVector2Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalize)<br/>                           | Retorna a versão normalizada de um vetor 2D.<br/>                     |
| [**XMVector2NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalizeest)<br/>                     | Estima a versão normalizada de um vetor 2D.<br/>                   |
| [**XMVector2Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2orthogonal)<br/>                         | Calcula um vetor em um vetor 2D.<br/>                    |
| [**XMVector2ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallength)<br/>             | Calcula o recíproco do comprimento de um vetor 2D.<br/>              |
| [**XMVector2ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallengthest)<br/>       | Estima o recíproco do comprimento de um vetor 2D.<br/>             |
| [**XMVector2Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reflect)<br/>                               | Reflete um vetor 2D de incidente em um vetor normal 2D.<br/>          |
| [**XMVector2Rector**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refract)<br/>                               | Reafrata um vetor 2D de incidente em um vetor normal 2D.<br/>          |
| [**XMVector2RectorV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refractv)<br/>                             | Reafrata um vetor 2D de incidente em um vetor normal 2D.<br/>          |
