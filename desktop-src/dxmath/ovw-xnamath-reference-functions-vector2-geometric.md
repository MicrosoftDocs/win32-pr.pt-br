# <a name="directxmath-library-2d-vector-geometric-functions"></a>Funções geométricas de vetor 2D de biblioteca DirectXMath

Lista as funções geométricas de vetor 2D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                 | Descrição                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector2AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormals)<br/>       | Computa o ângulo de radiano entre dois vetores 2D normalizados.<br/>       |
| [**XMVector2AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweennormalsest)<br/> | Estima o ângulo de radiano entre dois vetores 2D normalizados.<br/>      |
| [**XMVector2AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector2anglebetweenvectors)<br/>       | Computa o ângulo de radiano entre dois vetores 2D.<br/>                  |
| [**XMVector2ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplength)<br/>                       | Coloca o comprimento de um vetor 2D para um determinado intervalo.<br/>                 |
| [**XMVector2ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2clamplengthv)<br/>                     | Coloca o comprimento de um vetor 2D para um determinado intervalo.<br/>                 |
| [**XMVector2Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector2cross)<br/>                                   | Computa o produto 2D cruzado.<br/>                                     |
| [**XMVector2Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector2dot)<br/>                                       | Computa o produto de ponto entre os vetores 2D.<br/>                       |
| [**XMVector2InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector2inbounds)<br/>                             | Testa se os componentes de um vetor 2D estão dentro dos limites definidos.<br/> |
| [**XMVector2IntersectLine**](/windows/win32/api/directxmath/nf-directxmath-xmvector2intersectline)<br/>                   | Localiza a interseção de duas linhas.<br/>                               |
| [**XMVector2Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector2length)<br/>                                 | Calcula o comprimento de um vetor 2D.<br/>                                |
| [**XMVector2LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthest)<br/>                           | Estima o comprimento de um vetor 2D.<br/>                               |
| [**XMVector2LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lengthsq)<br/>                             | Computa o quadrado do comprimento de um vetor 2D.<br/>                  |
| [**XMVector2LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector2linepointdistance)<br/>           | Computa a distância mínima entre uma linha e um ponto.<br/>          |
| [**XMVector2Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalize)<br/>                           | Retorna a versão normalizada de um vetor 2D.<br/>                     |
| [**XMVector2NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2normalizeest)<br/>                     | Estima a versão normalizada de um vetor 2D.<br/>                   |
| [**XMVector2Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2orthogonal)<br/>                         | Computa um vetor perpendicular a um vetor 2D.<br/>                    |
| [**XMVector2ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallength)<br/>             | Computa o recíproco do comprimento de um vetor 2D.<br/>              |
| [**XMVector2ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reciprocallengthest)<br/>       | Estima o recíproco do comprimento de um vetor 2D.<br/>             |
| [**XMVector2Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector2reflect)<br/>                               | Reflete um vetor 2D de incidente em um vetor normal 2D.<br/>          |
| [**XMVector2Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refract)<br/>                               | Refracts um vetor 2D de incidente em um vetor normal 2D.<br/>          |
| [**XMVector2RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector2refractv)<br/>                             | Refracts um vetor 2D de incidente em um vetor normal 2D.<br/>          |
