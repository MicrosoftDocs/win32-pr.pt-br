# <a name="directxmath-library-2d-vector-transformation-functions"></a>Funções de transformação de vetor 2D da biblioteca DirectXMath

Lista as funções de transformação de vetor 2D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                               | Descrição                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transform)<br/>                         | Transforma um vetor 2D por uma matriz.<br/>                                                                                                   |
| [**XMVector2TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoord)<br/>               | Transforma um vetor 2D por uma determinada matriz, projetando o resultado de volta em w = 1.<br/>                                                      |
| [**XMVector2TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoordstream)<br/>   | Transforma um fluxo de vetores 2D por uma determinada matriz, projetando os vetores resultantes, de modo que suas coordenadas w sejam iguais a 1,0.<br/> |
| [**XMVector2TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormal)<br/>             | Transforma o vetor 2D normal pela matriz especificada.<br/>                                                                                  |
| [**XMVector2TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormalstream)<br/> | Transforma um fluxo de vetores normais de 2D por uma determinada matriz.<br/>                                                                           |
| [**XMVector2TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformstream)<br/>             | Transforma um fluxo de vetores 2D por uma determinada matriz.<br/>                                                                                  |
