# <a name="operator--operators"></a>operador + operadores

Operadores de adição.

### <a name="overload-list"></a>Lista de sobrecargas




| Operador | Descrição | 
|----------|-------------|
| <a href="/previous-versions/windows/desktop/legacy/ee421395(v=vs.85)"><strong>XMVECTOR:: Operator + (XMVECTOR)</strong></a> | Desempenho de uma operação de identidade em uma <code>XMVECTOR</code> instância.<br /> O <code>operator +</code> usa uma instância do <a href="xmvector-data-type.md"><strong>tipo de dados XMVECTOR</strong></a> e retorna uma nova instância do <code>XMVECTOR</code> , com um operador Identity aplicado a cada componente.<br /><blockquote>[!Note]<br />Esse operador só está disponível em C++.</blockquote><br /> | 
| <a href="/previous-versions/windows/desktop/legacy/ee421396(v=vs.85)"><strong>XMVECTOR:: Operator + (XMVECTOR, XMVECTOR)</strong></a> | Adiciona duas instâncias do <code>XMVECTOR</code> , retornando o resultado em uma nova instância.<br /> O <code>operator +</code> adiciona cada componente de duas instâncias do <a href="xmvector-data-type.md"><strong>tipo de dados XMVECTOR</strong></a>e, em seguida, retorna o resultado em uma nova <code>XMVECTOR</code> instância. <br /><blockquote>[!Note]<br />Esse operador só está disponível em C++.</blockquote><br /> |
