# <a name="directxmath-3d-vector-transformation-functions"></a>Funções de transformação de vetor 3D DirectXMath

Lista as funções de transformação de vetor 3D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                               | Descrição                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | Gira um vetor 3D usando o inverso de um Quaternion.<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | Projetar um vetor 3D do espaço do objeto no espaço da tela.<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | Projeta um fluxo de vetores 3D do espaço do objeto no espaço da tela.<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | Gira um vetor 3D usando um Quaternion.<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | Transforma um vetor 3D por uma matriz.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | Transforma um vetor 3D por uma determinada matriz, projetando o resultado de volta em w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | Transforma um fluxo de vetores 3D por uma determinada matriz, projetando os vetores resultantes, de modo que suas coordenadas w sejam iguais a 1,0.<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | Transforma o vetor 3D normal pela matriz especificada.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | Transforma um fluxo de vetores normais 3D por uma determinada matriz.<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | Transforma um fluxo de vetores 3D por uma determinada matriz.<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | Projeta um vetor 3D do espaço da tela no espaço do objeto.<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | Transforma um fluxo de vetores 3D do espaço da tela para o espaço do objeto.<br/>                                                                  |
