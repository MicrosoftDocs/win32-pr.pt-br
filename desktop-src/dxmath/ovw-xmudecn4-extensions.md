# <a name="xmudecn4-extensions"></a>Extensões XMUDECN4

[**XMUDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudecn4) Um vetor de 4D para armazenar valores normalizados e não assinados como um componente x-, y-e z-unsinaled de 10 bits e um componentes de 2 bits sem sinal.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
