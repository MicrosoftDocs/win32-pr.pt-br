# <a name="xmvector-operators"></a>Operadores XMVECTOR

Os operadores a seguir são expostos pela estrutura [**XMVECTOR**](xmvector-data-type.md) .

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                    | Descrição                                                                                                                                 |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**operador + =**](/previous-versions/windows/desktop/legacy/ee421394(v=vs.85))<br/>  | Adiciona um valor de ponto flutuante a uma `XMVECTOR` instância e retorna uma referência à instância atualizada. <br/>                         |
| [**operador-=**](/previous-versions/windows/desktop/legacy/ee421383(v=vs.85))<br/>    | Subtrai um valor de ponto flutuante da instância atual do `XMVECTOR` , retornando o resultado na instância atual atualizada. <br/> |
| [**operador \** _](xmvector-operator-mul.md)<br/>    | Operadores de multiplicação<br/>                                                                                                         |
| [*operador \* =* _*](xmvector-operator-muleq.md)<br/> | Operadores de atribuição de multiplicação<br/>                                                                                              |
| [**operador**](xmvector-operator-div.md)<br/>     | Operador de divisão.<br/>                                                                                                               |
| [**operador/=**](xmvector-operator-diveq.md)<br/>  | Operador de atribuição de divisão.<br/>                                                                                                    |
| [**operador**](xmvector-operator--.md)<br/>       | Operadores de destração e de negação<br/>                                                                                               |
| [**operador +**](xmvector-operator-pls.md)<br/>     | Operadores de adição.<br/>                                                                                                              |
