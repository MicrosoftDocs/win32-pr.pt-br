# <a name="xmushort2-extensions"></a>Extensões XMUSHORT2

[**XMUSHORT2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort2) descreve o vetor a2D para armazenar valores normalizados e não assinados como inteiros não assinados de 16 bits, (tipo `uint16_t` ).

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
