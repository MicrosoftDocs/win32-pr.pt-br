# <a name="geometric-vector-functions"></a>Funções de vetor geométricas

Lista as funções de vetor geométricos.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                           | Descrição                                                                                                                                                                                                      |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorBaryCentric**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentric)<br/>   | Retorna um ponto nas coordenadas de barycentric, usando os vetores de posição especificados.<br/>                                                                                                                     |
| [**XMVectorBaryCentricV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentricv)<br/> | Retorna um ponto nas coordenadas de barycentric, usando os vetores de posição especificados.<br/>                                                                                                                     |
| [**XMVectorCatmullRom**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullrom)<br/>     | Executa uma interpolação de Catmull-Rom, usando os vetores de posição especificados.<br/>                                                                                                                           |
| [**XMVectorCatmullRomV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullromv)<br/>   | Executa uma interpolação de Catmull-Rom, usando os vetores de posição especificados.<br/>                                                                                                                           |
| [**XMVectorHermite**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermite)<br/>           | Executa uma interpolação de spline Hermite, usando os vetores especificados.<br/>                                                                                                                                 |
| [**XMVectorHermiteV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermitev)<br/>         | Executa uma interpolação de spline Hermite, usando os vetores especificados.<br/>                                                                                                                                 |
| [**XMVectorInBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinbounds)<br/>         | Testa se os componentes de um determinado vetor estão dentro dos limites definidos.<br/>                                                                                                                                 |
| [**XMVectorInBoundsR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinboundsr)<br/>       | Testa se os componentes de um determinado vetor estão dentro de determinados limites e define um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVectorLerp**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerp)<br/>                 | Executa uma interpolação linear entre dois vetores.<br/>                                                                                                                                                  |
| [**XMVectorLerpV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerpv)<br/>               | Executa uma interpolação linear entre dois vetores.<br/>                                                                                                                                                  |
