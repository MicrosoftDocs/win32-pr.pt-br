# <a name="xmfloat4x4-operators"></a>Operadores XMFLOAT4X4

A estrutura [**XMFLOAT4X4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x4) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                      | Descrição                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4x4-operator-assign)<br/>      | Atribui os dados do componente de vetor de uma instância do `XMFLOAT4X4` à instância atual do `XMFLOAT4X4` . <br/> |
| [**operador ()**](xmfloat4x4-operator-parens.md)<br/> | Acessa elementos de matriz específicos referenciados por linha e coluna da instância atual do `XMFLOAT4X4` . <br/>    |
