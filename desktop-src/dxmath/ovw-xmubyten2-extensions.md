# <a name="xmubyten2-extensions"></a>Extensões XMUBYTEN2

[**XMUBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmubyten2) descreve um vetor 2D para armazenar valores normalizados não assinados como inteiros não assinados de 8 bits (1 byte).

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
