# <a name="directxmath-library-4d-vector-comparison-functions"></a>Funções de comparação de vetor 4D da Biblioteca DirectXMath

Lista as funções de comparação de vetor 4D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                   | Descrição                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector4Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equal)<br/>                     | Testa se dois vetores 4D são iguais.<br/>                                                                                                                                                                                                        |
| [**XMVector4EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalint)<br/>               | Testa se dois vetores 4D são iguais, tratando cada componente como um inteiro sem sinal.<br/>                                                                                                                                                        |
| [**XMVector4EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalintr)<br/>             | Testa se dois vetores 4D são iguais, tratando cada componente como um inteiro sem sinal. Além disso, essa função retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector4EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4equalr)<br/>                   | Testa se dois vetores 4D são iguais. Além disso, essa função retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector4Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greater)<br/>                 | Testa se um vetor 4D é maior que outro vetor 4D.<br/>                                                                                                                                                                                 |
| [**XMVector4GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequal)<br/>   | Testa se um vetor 4D é maior que ou igual a outro vetor 4D.<br/>                                                                                                                                                                     |
| [**XMVector4GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterorequalr)<br/> | Testa se um vetor 4D é maior ou igual a outro vetor 4D e retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector4GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector4greaterr)<br/>               | Testa se um vetor 4D é maior que outro vetor 4D e retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector4IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isinfinite)<br/>           | Testa se qualquer componente de um vetor 4D é infinito positivo ou negativo.<br/>                                                                                                                                                                   |
| [**XMVector4IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector4isnan)<br/>                     | Testa se qualquer componente de um vetor 4D é um NaN.<br/>                                                                                                                                                                                           |
| [**XMVector4Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector4less)<br/>                       | Testa se um vetor 4D é menor que outro vetor 4D.<br/>                                                                                                                                                                                    |
| [**XMVector4LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lessorequal)<br/>         | Testa se um vetor 4D é menor que ou igual a outro vetor 4D.<br/>                                                                                                                                                                        |
| [**XMVector4NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4nearequal)<br/>             | Testa se um vetor 4D está próximo a outro vetor 4D.<br/>                                                                                                                                                                                         |
| [**XMVector4NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequal)<br/>               | Testa se dois vetores 4D não são iguais.<br/>                                                                                                                                                                                                    |
| [**XMVector4NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector4notequalint)<br/>         | Teste se dois vetores 4D não são iguais, tratando cada componente como um inteiro sem sinal.<br/>                                                                                                                                                     |
