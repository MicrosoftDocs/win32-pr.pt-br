# <a name="xmmatrix-extensions"></a>Extensões XMMATRIX

[**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) descreve uma matriz de 4 \* 4 alinhada em um limite de 16 bytes que é mapeado para quatro registros de vetor de hardware.

> [!Note]  
> Os recursos listados aqui só estão disponíveis em C++.
