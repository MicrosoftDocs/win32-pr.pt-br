# <a name="xmfloat3pk-extensions"></a>Extensões XMFLOAT3PK

Um vetor 3D com componentes X e Y armazenados como um número de ponto flutuante de 11 bits e o componente Z armazenado como um valor de ponto flutuante de 10 bits.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
