# <a name="directxmath-library-quaternion-functions"></a>Funções de Quaternion da biblioteca DirectXMath

Lista as funções de Quaternion fornecidas pelo DirectXMath.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                       | Descrição                                                                                                     |
|-------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**XMQuaternionBaryCentric**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionbarycentric)<br/>                                       | Retorna um ponto em coordenadas barycentric, usando os quaternions especificados.<br/>                         |
| [**XMQuaternionBaryCentricV**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionbarycentricv)<br/>                                     | Retorna um ponto em coordenadas barycentric, usando os quaternions especificados.<br/>                         |
| [**XMQuaternionConjugate**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionconjugate)<br/>                                           | Computa o conjugado de um Quaternion.<br/>                                                              |
| [**XMQuaternionDot**](/windows/win32/api/directxmath/nf-directxmath-xmquaterniondot)<br/>                                                       | Computa o produto de ponto de dois quaternions.<br/>                                                         |
| [**XMQuaternionEqual**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionequal)<br/>                                                   | Testa se dois quaternions são iguais.<br/>                                                             |
| [**XMQuaternionExp**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionexp)<br/>                                                       | Computa o exponencial de um determinado Quaternion puro.<br/>                                                 |
| [**XMQuaternionIdentity**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionidentity)<br/>                                             | Retorna o quaternion da identidade.<br/>                                                                     |
| [**XMQuaternionInverse**](/windows/win32/api/directxmath/nf-directxmath-xmquaternioninverse)<br/>                                               | Computa o inverso de um Quaternion.<br/>                                                                |
| [**XMQuaternionIsIdentity**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionisidentity)<br/>                                         | Testa se um Quaternion específico é o quaternion da identidade.<br/>                                      |
| [**XMQuaternionIsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionisinfinite)<br/>                                         | Testar se qualquer componente de um Quaternion é um infinito positivo ou negativo.<br/>                  |
| [**XMQuaternionIsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionisnan)<br/>                                                   | Testar se qualquer componente de um Quaternion é um NaN.<br/>                                                 |
| [**XMQuaternionLength**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionlength)<br/>                                                 | Computa a magnitude de um Quaternion.<br/>                                                              |
| [**XMQuaternionLengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionlengthsq)<br/>                                             | Computa o quadrado da magnitude de um Quaternion.<br/>                                                |
| [**XMQuaternionLn**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionln)<br/>                                                         | Computa o logaritmo natural de uma determinada unidade Quaternion.<br/>                                           |
| [**XMQuaternionMultiply**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionmultiply)<br/>                                             | Computa o produto de dois quaternions.<br/>                                                             |
| [**XMQuaternionNormalize**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionnormalize)<br/>                                           | Normaliza um Quaternion.<br/>                                                                             |
| [**XMQuaternionNormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionnormalizeest)<br/>                                     | Estima a versão normalizada de um Quaternion.<br/>                                                    |
| [**XMQuaternionNotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionnotequal)<br/>                                             | Testa se dois quaternions não são iguais.<br/>                                                         |
| [**XMQuaternionReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionreciprocallength)<br/>                             | Computa o recíproco da magnitude de um Quaternion.<br/>                                            |
| [**XMQuaternionRotationAxis**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationaxis)<br/>                                     | Computa um Quaternion de rotação sobre um eixo.<br/>                                                        |
| [**XMQuaternionRotationMatrix**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationmatrix)<br/>                                 | Computa um Quaternion de rotação de uma matriz de rotação.<br/>                                               |
| [**XMQuaternionRotationNormal**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationnormal)<br/>                                 | Computa o quaternion de rotação sobre um vetor normal.<br/>                                              |
| [**XMQuaternionRotationRollPitchYaw**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationrollpitchyaw)<br/>                     | Computa um Quaternion de rotação com base nos ângulos de pitch, guinada e roll (Euler).<br/>                     |
| [**XMQuaternionRotationRollPitchYawFromVector**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionrotationrollpitchyawfromvector)<br/> | Computa um Quaternion de rotação com base em um vetor que contém os ângulos de Euler (Pitch, guinada e roll).<br/> |
| [**XMQuaternionSlerp**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionslerp)<br/>                                                   | Interpola entre dois quaternions de unidade, usando interpolação linear esférica.<br/>                     |
| [**XMQuaternionSlerpV**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionslerpv)<br/>                                                 | Interpola entre dois quaternions de unidade, usando interpolação linear esférica.<br/>                     |
| [**XMQuaternionSquad**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionsquad)<br/>                                                   | Interpola entre quatro quaternions de unidade, usando interpolação de Quadrangle esférica.<br/>                |
| [**XMQuaternionSquadSetup**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionsquadsetup)<br/>                                         | Fornece endereços de pontos de controle de instalação para interpolação de Quadrangle esférica.<br/>                   |
| [**XMQuaternionSquadV**](/windows/win32/api/directxmath/nf-directxmath-xmquaternionsquadv)<br/>                                                 | Interpola entre quatro quaternions de unidade, usando interpolação de Quadrangle esférica.<br/>                |
| [**XMQuaternionToAxisAngle**](/windows/win32/api/directxmath/nf-directxmath-xmquaterniontoaxisangle)<br/>                                       | Computa um eixo e um ângulo de rotação sobre esse eixo para um determinado Quaternion.<br/>                       |
