# <a name="xmushortn4-extensions"></a>Extensões XMUSHORTN4

[**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) descreve um vetor 4D para armazenar valores normalizados não assinados como inteiros de 16 bits assinados, (tipo `uint16_t` ).

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
