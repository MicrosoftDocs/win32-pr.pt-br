# <a name="xmudec4-extensions"></a>Extensões XMUDEC4

[**XMUDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudec4)descreve um vetor 4D com componentes x-, y-e z representados como valores inteiros sem sinal de 10 bits e o componente w como um valor inteiro sem sinal de 2 bits.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
