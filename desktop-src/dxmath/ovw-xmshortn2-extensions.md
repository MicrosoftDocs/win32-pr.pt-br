# <a name="xmshortn2-extensions"></a>Extensões XMSHORTN2

[**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) descreve um vetor 2D para armazenar valores normalizados assinados como inteiros de 16 bits assinados, (tipo `int16_t` ).

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
