# <a name="xmdec4-extensions"></a>Extensões XMDEC4

[**XMDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdec4)descreve um vetor 4D com componentes x-, y-e z representados como valores inteiros com sinal de 10 bits e o componente w como um valor inteiro com sinal de 2 bits.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
