# <a name="directxmath-library-3d-vector-geometric-functions"></a>Funções geométricas de vetor 3D da Biblioteca DirectXMath

Lista as funções geométricas de vetor 3D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                 | Descrição                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormals)<br/>       | Calcula o ângulo radiano entre dois vetores 3D normalizados.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormalsest)<br/> | Estima o ângulo radiano entre dois vetores 3D normalizados.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweenvectors)<br/>       | Calcula o ângulo radiano entre dois vetores 3D.<br/>                                                                      |
| [**XMVector3ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplength)<br/>                       | Fixa o comprimento de um vetor 3D para um determinado intervalo.<br/>                                                                     |
| [**XMVector3ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplengthv)<br/>                     | Fixa o comprimento de um vetor 3D para um determinado intervalo.<br/>                                                                     |
| [**XMVector3ComponentsFromNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3componentsfromnormal)<br/>     | Usando um vetor normal de referência, o divide um vetor 3D em componentes paralelos e com o normal.<br/> |
| [**XMVector3Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector3cross)<br/>                                   | Calcula o produto cruzado entre dois vetores 3D.<br/>                                                                     |
| [**XMVector3Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector3dot)<br/>                                       | Calcula o produto de ponto entre vetores 3D.<br/>                                                                           |
| [**XMVector3InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inbounds)<br/>                             | Testa se os componentes de um vetor 3D estão dentro dos limites definidos.<br/>                                                     |
| [**XMVector3Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector3length)<br/>                                 | Calcula o comprimento de um vetor 3D.<br/>                                                                                    |
| [**XMVector3LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthest)<br/>                           | Estima o comprimento de um vetor 3D.<br/>                                                                                   |
| [**XMVector3LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthsq)<br/>                             | Calcula o quadrado do comprimento de um vetor 3D.<br/>                                                                      |
| [**XMVector3LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector3linepointdistance)<br/>           | Calcula a distância mínima entre uma linha e um ponto.<br/>                                                              |
| [**XMVector3Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalize)<br/>                           | Retorna a versão normalizada de um vetor 3D.<br/>                                                                         |
| [**XMVector3NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalizeest)<br/>                     | Estima a versão normalizada de um vetor 3D.<br/>                                                                       |
| [**XMVector3Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3orthogonal)<br/>                         | Calcula um vetor em um vetor 3D.<br/>                                                                        |
| [**XMVector3ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallength)<br/>             | Calcula o recíproco do comprimento de um vetor 3D.<br/>                                                                  |
| [**XMVector3ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallengthest)<br/>       | Estima o recíproco do comprimento de um vetor 3D.<br/>                                                                 |
| [**XMVector3Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reflect)<br/>                               | Reflete um vetor 3D de incidente em um vetor normal 3D.<br/>                                                              |
| [**XMVector3Rector**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refract)<br/>                               | Reafrata um vetor 3D de incidente em um vetor normal 3D.<br/>                                                              |
| [**XMVector3RectorV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refractv)<br/>                             | Reafrata um vetor 3D de incidente em um vetor normal 3D.<br/>                                                              |
