# <a name="directxmath-library-3d-vector-geometric-functions"></a>Funções geométricas de vetor 3D da biblioteca DirectXMath

Lista as funções geométricas de vetor 3D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                 | Descrição                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormals)<br/>       | Computa o ângulo de radiano entre dois vetores 3D normalizados.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormalsest)<br/> | Estima o ângulo de radiano entre dois vetores 3D normalizados.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweenvectors)<br/>       | Computa o ângulo em radiano entre dois vetores 3D.<br/>                                                                      |
| [**XMVector3ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplength)<br/>                       | Coloca o comprimento de um vetor 3D para um determinado intervalo.<br/>                                                                     |
| [**XMVector3ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplengthv)<br/>                     | Coloca o comprimento de um vetor 3D para um determinado intervalo.<br/>                                                                     |
| [**XMVector3ComponentsFromNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3componentsfromnormal)<br/>     | Usando um vetor normal de referência, o divide um vetor 3D em componentes que são paralelos e perpendiculares ao normal.<br/> |
| [**XMVector3Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector3cross)<br/>                                   | Computa o produto cruzado entre dois vetores 3D.<br/>                                                                     |
| [**XMVector3Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector3dot)<br/>                                       | Computa o produto de ponto entre os vetores 3D.<br/>                                                                           |
| [**XMVector3InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inbounds)<br/>                             | Testa se os componentes de um vetor 3D estão dentro dos limites definidos.<br/>                                                     |
| [**XMVector3Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector3length)<br/>                                 | Computa o comprimento de um vetor 3D.<br/>                                                                                    |
| [**XMVector3LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthest)<br/>                           | Estima o comprimento de um vetor 3D.<br/>                                                                                   |
| [**XMVector3LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthsq)<br/>                             | Computa o quadrado do comprimento de um vetor 3D.<br/>                                                                      |
| [**XMVector3LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector3linepointdistance)<br/>           | Computa a distância mínima entre uma linha e um ponto.<br/>                                                              |
| [**XMVector3Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalize)<br/>                           | Retorna a versão normalizada de um vetor 3D.<br/>                                                                         |
| [**XMVector3NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalizeest)<br/>                     | Estima a versão normalizada de um vetor 3D.<br/>                                                                       |
| [**XMVector3Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3orthogonal)<br/>                         | Computa um vetor perpendicular a um vetor 3D.<br/>                                                                        |
| [**XMVector3ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallength)<br/>             | Computa o recíproco do comprimento de um vetor 3D.<br/>                                                                  |
| [**XMVector3ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallengthest)<br/>       | Estima o recíproco do comprimento de um vetor 3D.<br/>                                                                 |
| [**XMVector3Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reflect)<br/>                               | Reflete um vetor 3D de incidente em um vetor normal 3D.<br/>                                                              |
| [**XMVector3Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refract)<br/>                               | Refracts um vetor 3D de incidente em um vetor normal 3D.<br/>                                                              |
| [**XMVector3RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refractv)<br/>                             | Refracts um vetor 3D de incidente em um vetor normal 3D.<br/>                                                              |
