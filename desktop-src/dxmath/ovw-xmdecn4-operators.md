# <a name="xmdecn4-operators"></a>Operadores XMDECN4

A [**estrutura XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                            | Descrição                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdecn4-operatoruint32_t)<br/> | Retorna uma instância de `uint32_t` que contém os componentes da instância em `XMDECN4` um formato empacotado. <br/> |
| [**operator =**](xmdecn4-operator-eq.md)<br/>               | Atribui dados de componente vetor à instância atual do `XMDECN4` . <br/>                                       |
