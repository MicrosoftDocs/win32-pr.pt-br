# <a name="xmxdec4-operators"></a>Operadores XMXDEC4

A estrutura [**XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                            | Descrição                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdec4-operatoruint32_t)<br/> | Retorna uma instância do `uint32_t` que contém os componentes da `XMXDEC4` instância em um formato empacotado. <br/> |
| [**operador =**](xmxdec4-operator-eq.md)<br/>               | Atribui dados de componente de vetor à instância atual do `XMXDEC4` . <br/>                                       |
