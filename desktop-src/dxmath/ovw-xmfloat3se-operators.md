# <a name="xmfloat3se-operators"></a>Operadores XMFLOAT3SE

A estrutura [**XMFLOAT3SE**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3se) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                               | Descrição                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3se-operatoruint32_t)<br/> | Retorna uma instância do `uint32_t` que contém os componentes da `XMFLOAT3SE` instância em um formato empacotado.<br/> |
| [**operador =**](xmfloat3se-operator-eq.md)<br/>               | Atribui dados de componente de vetor à instância atual do `XMFLOAT3SE` .<br/>                                       |
