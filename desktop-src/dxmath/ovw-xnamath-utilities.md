# <a name="directxmath-library-utility-functions"></a>Funções do utilitário de biblioteca DirectXMath

Lista as funções de utilitário fornecidas pelo DirectXMath.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                       | Descrição                                                                                                                |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**XMComparisonAllFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallfalse)<br/>             | Testa o valor de comparação para determinar se todos os componentes comparados são falsos.<br/>                            |
| [**XMComparisonAllInBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallinbounds)<br/>       | Testa o valor de comparação para determinar se todos os componentes comparados estão dentro dos limites definidos.<br/>                |
| [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>               | Testa o valor de comparação para determinar se todos os componentes comparados são verdadeiros.<br/>                             |
| [**XMComparisonAnyFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyfalse)<br/>             | Testa o valor de comparação para determinar se algum dos componentes comparados é falso.<br/>                            |
| [**XMComparisonAnyOutOfBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyoutofbounds)<br/> | Testa o valor de comparação para determinar se algum dos componentes comparados está fora dos limites do conjunto.<br/>           |
| [**XMComparisonAnyTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanytrue)<br/>               | Testa o valor de comparação para determinar se algum dos componentes comparados são verdadeiros.<br/>                             |
| [**XMComparisonMixed**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonmixed)<br/>                   | Testa o valor de comparação para determinar se os componentes comparados tinham resultados mistos, alguns verdadeiros e alguns falsos.<br/> |
| [**XMFresnelTerm**](/windows/win32/api/directxmath/nf-directxmath-xmfresnelterm)<br/>                           | Calcula o termo Fresnel para uma luz não polarizada.<br/>                                                              |
| [**XMVerifyCPUSupport**](/windows/win32/api/directxmath/nf-directxmath-xmverifycpusupport)<br/>                 | Indica se a biblioteca DirectXMath dá suporte à plataforma atual.<br/>                                             |
