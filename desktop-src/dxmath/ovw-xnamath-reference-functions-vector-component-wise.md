# <a name="component-wise-vector-functions"></a>Funções de vetor com componente

Lista as funções de vetor por componente.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                             | Descrição                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorInsert**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinsert)<br/>               | Gira um vetor à esquerda por um determinado número de componentes de 32 bits e insere os elementos selecionados desse resultado em outro vetor.<br/> |
| [**XMVectorMergeXY**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergexy)<br/>             | Cria um novo vetor combinando os componentes x e y de dois vetores.<br/>                                                      |
| [**XMVectorMergeZW**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergezw)<br/>             | Cria um novo vetor combinando os componentes z e w de dois vetores.<br/>                                                      |
| [**XMVectorPermute**](/windows/win32/api/directxmath/nf-directxmath-xmvectorpermute)<br/>             | Permuta mudo dos componentes de dois vetores para criar um novo vetor.<br/>                                                                |
| [**XMVectorRotateLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateleft)<br/>       | Gira o vetor à esquerda por um determinado número de elementos de 32 bits.<br/>                                                                 |
| [**XMVectorRotateRight**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateright)<br/>     | Gira o vetor à direita por um determinado número de elementos de 32 bits.<br/>                                                                |
| [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>               | Executa uma seleção por componente entre dois vetores de entrada e retorna o vetor resultante.<br/>                                |
| [**XMVectorSelectControl**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselectcontrol)<br/> | Define um vetor de controle para uso em [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect).<br/>                                                 |
| [**XMVectorShiftLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorshiftleft)<br/>         | Desloca um vetor à esquerda por um determinado número de elementos de 32 bits, preenchendo os elementos vagados com elementos de um segundo vetor.<br/>   |
| [**XMVectorSplatW**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatw)<br/>               | Replica o componente w de um vetor para todos os componentes.<br/>                                                              |
| [**XMVectorSplatX**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatx)<br/>               | Replica o componente x de um vetor para todos os componentes.<br/>                                                              |
| [**XMVectorSplatY**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplaty)<br/>               | Replica o componente y de um vetor para todos os componentes.<br/>                                                              |
| [**XMVectorSplatZ**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatz)<br/>               | Replica o componente z de um vetor para todos os componentes.<br/>                                                              |
| [**XMVectorSwizzle**](/windows/win32/api/directxmath/nf-directxmath-xmvectorswizzle)<br/>             | Swizzles um vetor.<br/>                                                                                                            |
