# <a name="xmu555-operators"></a>Operadores XMU555

A estrutura [**XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                           | Descrição                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operador UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu555-operatoruint16_t)<br/> | Retorna uma instância do `uint16_t` que contém os componentes da `XMU555` instância em um formato empacotado. <br/> |
| [**operador =**](xmu555-operator-eq.md)<br/>               | Atribui dados de componente de vetor à instância atual do `XMU555` . <br/>                                       |
