# <a name="directxmath-library-types"></a>Tipos de Biblioteca DirectXMath

A Biblioteca DirectXMath fornece várias estruturas e tipos definidos para encapsular dados para dar suporte à facilidade de uso, otimização e portabilidade.

A lista a seguir inclui estruturas atualmente parte da Biblioteca DirectXMath e disponíveis por meio do header DirectXMath.h.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                             | Descrição                                                                                                                                                                       |
|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Tipo de dados HALF**](half-data-type.md)<br/>               | Um alias para **uint16 \_** não empacotado com um número de ponto flutuante de 16 bits que consiste em um bit de sinal, um expoente com desvio de 5 bits e uma mantissa de 10 bits.<br/>                         |
| [**Tipo de dados XMVECTOR**](xmvector-data-type.md)<br/>       | Um tipo portátil usado para representar um vetor de quatro componentes inteiros ou de ponto flutuante de 32 bits, cada um alinhado de forma ideal e mapeado para um registro de vetor de hardware.<br/>       |
| [**Tipo de dados XMVECTORF32**](xmvectorf32-data-type.md)<br/> | Um tipo portátil opaco para dar suporte ao uso da sintaxe do inicializador C/C++ para carregar valores de ponto flutuante em uma instância do [**tipo XMVECTOR.**](xmvector-data-type.md)<br/> |
| [**Tipo de dados XMVECTORI32**](xmvectori32-data-type.md)<br/> | Um tipo portátil opaco para dar suporte ao uso da sintaxe do inicializador C/C++ para carregar valores inteiros em uma instância do [**tipo XMVECTOR.**](xmvector-data-type.md)<br/>        |
| [**Tipo de dados XMVECTORU32**](xmvectoru32-data-type.md)<br/> | Um tipo portátil opaco para dar suporte ao uso da sintaxe do inicializador C/C++ para carregar valores uint32 t em uma instância do \_ tipo XMVECTOR.<br/>                                    |
| [**Tipo de dados XMVECTORU8**](xmvectoru8-data-type.md)<br/>   | Um tipo portátil opaco para dar suporte ao uso da sintaxe do inicializador C/C++ para carregar valores uint8 t em uma instância do \_ tipo XMVECTOR.<br/>                                     |
