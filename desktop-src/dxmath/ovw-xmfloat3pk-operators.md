# <a name="xmfloat3pk-operators"></a>Operadores XMFLOAT3PK

A estrutura [**XMFLOAT3PK**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                               | Descrição                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-operatoruint32_t)<br/> | Retorna uma instância do `uint32_t` que contém os componentes da `XMFLOAT3PK` instância em um formato empacotado.<br/> |
| [**operador =**](xmfloat3pk-operator-eq.md)<br/>               | Atribui dados de componente de vetor à instância atual do `XMFLOAT3PK` .<br/>                                       |
