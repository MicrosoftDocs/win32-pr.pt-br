# <a name="xmfloat4-extensions"></a>Extensões XMFLOAT4

[**XMFLOAT4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4) descreve uma estrutura que contém um vetor 4D que consiste em quatro valores de ponto flutuante de precisão simples.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
