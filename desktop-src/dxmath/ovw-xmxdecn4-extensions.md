# <a name="xmxdecn4-extensions"></a>Extensões XMXDECN4

[**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) define um vetor de 4D para armazenar valores normalizados assinados como 10 bits assinados x-, y-e z-Components e valores normalizados não assinados como um componente w-unsigned de 2 bits.

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
