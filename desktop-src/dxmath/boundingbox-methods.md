# <a name="boundingbox-methods"></a>Métodos BoundingBox



| Método                                                              | Descrição                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Terá**](boundingbox-contains.md)<br/>                 | Testa se o BoundingBox contém um objeto especificado.<br/>                                  |
| [**CreateFromPoints**](boundingbox-createfrompoints.md)<br/> | Cria um BoundingBox de pontos.<br/>                                                          |
| [**Intersecciona esses**](boundingbox-intersects.md)<br/>             | Testa o BoundingBox para interseção com outro objeto.<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | Transforma o BoundingBox.<br/>                                                                 |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Testa se o [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) está contido no frustum especificado.<br/> |
| [**CreateFromSphere**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createfromsphere)<br/> | Cria um BoundingBox grande o suficiente para conter um BoundingSphere especificado.<br/>               |
| [**Createmerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createmerged)<br/>         | Cria um BoundingBox grande o suficiente para conter dois instâncias de BoundBox especificados.<br/>             |
| [**Getvértices**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-getcorners)<br/>             | Recupera os cantos do BoundingBox.<br/>                                                   |
| [**atribuição de op \_**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-operator-assign)<br/>      | Copia valores de outro BoundingBox.<br/>                                                     |
