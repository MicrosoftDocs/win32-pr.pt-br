# <a name="xmudec4-operators"></a>Operadores XMUDEC4

A estrutura [**XMUDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudec4) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                            | Descrição                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmudec4-operatoruint32_t)<br/> | Retorna uma instância do `uint32_t` que contém os componentes da `XMUDEC4` instância em um formato empacotado. <br/> |
| [**operador =**](xmudec4-operator-eq.md)<br/>               | Atribui dados de componente de vetor à instância atual do `XMUDEC4` . <br/>                                       |
