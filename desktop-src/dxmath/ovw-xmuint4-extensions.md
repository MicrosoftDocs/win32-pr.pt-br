# <a name="xmuint4-extensions"></a>Extensões XMUINT4

[**XMUINT4**](/windows/win32/api/directxmath/ns-directxmath-xmuint4) descreve uma estrutura que contém um vetor 4D que consiste em quatro valores inteiros sem sinal.

> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
