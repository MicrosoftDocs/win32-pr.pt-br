# <a name="xmbyten4-extensions"></a>Extensões XMBYTEN4

[**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) descreve um vetor 3D para armazenar valores normalizados e assinados como inteiros assinados de 8 bits (1 byte).

> [!Note]  
> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
