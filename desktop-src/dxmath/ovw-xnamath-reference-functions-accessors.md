# <a name="directxmath-library-vector-accessor-functions"></a>Funções de acessador de vetor da biblioteca DirectXMath

Lista as funções de acessador de vetor fornecidas pela biblioteca DirectXMath.

Os acessadores de vetor DirectXMath permitem que os desenvolvedores escrevam código que obtém e define componentes individuais de um vetor de maneira portátil e ideal.
