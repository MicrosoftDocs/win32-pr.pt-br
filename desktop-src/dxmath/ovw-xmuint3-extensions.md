# <a name="xmuint3-extensions"></a>Extensões XMUINT3

[**XMUINT3**](/windows/win32/api/directxmath/ns-directxmath-xmuint3) descreve uma estrutura que contém um vetor 3D que consiste em três valores inteiros sem sinal.

> Os recursos listados aqui só estão disponíveis ao desenvolver com C++.
