# <a name="directxmath-library-scalar-functions"></a>Funções escalares da biblioteca DirectXMath

Lista as funções escalares fornecidas pelo DirectXMath.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                     | Descrição                                                          |
|-----------------------------------------------------------|----------------------------------------------------------------------|
| [**XMScalarACos**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracos)<br/>           | Computa o arco cosseno de um número de ponto flutuante.<br/>        |
| [**XMScalarACosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracosest)<br/>     | Estima o arco cosseno de um número de ponto flutuante.<br/>       |
| [**XMScalarASin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasin)<br/>           | Computa o arco seno de um número de ponto flutuante.<br/>          |
| [**XMScalarASinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasinest)<br/>     | Estima o arco seno de um número de ponto flutuante.<br/>         |
| [**XMScalarCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcos)<br/>             | Computa o cosseno de um ângulo radiano.<br/>                    |
| [**XMScalarCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcosest)<br/>       | Estima o cosseno de um ângulo radiano.<br/>                   |
| [**XMScalarModAngle**](/windows/win32/api/directxmath/nf-directxmath-xmscalarmodangle)<br/>   | Computa um ângulo entre-XM \_ PI e XM \_ PI.<br/>             |
| [**XMScalarNearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmscalarnearequal)<br/> | Determina se dois valores de ponto flutuante são quase iguais.<br/> |
| [**XMScalarSin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsin)<br/>             | Computa o seno de um ângulo radiano.<br/>                      |
| [**XMScalarSinCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincos)<br/>       | Computa o seno e o cosseno de um ângulo em radiano.<br/>      |
| [**XMScalarSinCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincosest)<br/> | Calcula o seno e o cosseno de um ângulo em radiano.<br/>     |
| [**XMScalarSinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsinest)<br/>       | Estima o seno de um ângulo radiano.<br/>                     |
