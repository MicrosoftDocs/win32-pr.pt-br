# <a name="directxmath-library-3d-vector-comparison-functions"></a>Funções de comparação de vetor 3D da biblioteca DirectXMath

Lista as funções de comparação de vetor 3D.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                   | Descrição                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equal)<br/>                     | Testa se dois vetores 3D são iguais.<br/>                                                                                                                                                                                                        |
| [**XMVector3EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalint)<br/>               | Testa se dois vetores 3D são iguais, tratando cada componente como um inteiro sem sinal.<br/>                                                                                                                                                        |
| [**XMVector3EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalintr)<br/>             | Testa se dois vetores 3D são iguais, tratando cada componente como um inteiro sem sinal. Além disso, essa função retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector3EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalr)<br/>                   | Testa se dois vetores 3D são iguais. Além disso, essa função retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector3Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greater)<br/>                 | Testa se um vetor 3D é maior que outro vetor 3D.<br/>                                                                                                                                                                                 |
| [**XMVector3GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequal)<br/>   | Testa se um vetor 3D é maior que ou igual a outro vetor 3D.<br/>                                                                                                                                                                     |
| [**XMVector3GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequalr)<br/> | Testa se um vetor 3D é maior que ou igual a outro vetor 3D e retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector3GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterr)<br/>               | Testa se um vetor 3D é maior que outro vetor 3D e retorna um valor de comparação que pode ser examinado usando funções como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector3IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isinfinite)<br/>           | Testa se algum componente de um vetor 3D é infinito positivo ou negativo.<br/>                                                                                                                                                                   |
| [**XMVector3IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isnan)<br/>                     | Testa se qualquer componente de um vetor 3D é um NaN.<br/>                                                                                                                                                                                           |
| [**XMVector3Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector3less)<br/>                       | Testa se um vetor 3D é menor que outro vetor 3D.<br/>                                                                                                                                                                                    |
| [**XMVector3LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lessorequal)<br/>         | Testa se um vetor 3D é menor que ou igual a outro vetor 3D.<br/>                                                                                                                                                                        |
| [**XMVector3NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3nearequal)<br/>             | Testa se um vetor 3D está próximo a outro vetor 3D.<br/>                                                                                                                                                                                         |
| [**XMVector3NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequal)<br/>               | Testa se dois vetores 3D não são iguais.<br/>                                                                                                                                                                                                    |
| [**XMVector3NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequalint)<br/>         | Teste se dois vetores 3D não são iguais, tratando cada componente como um inteiro sem sinal.<br/>                                                                                                                                                     |
