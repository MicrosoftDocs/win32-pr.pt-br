# <a name="directxmath-programming-guide"></a>Guia de programação do DirectXMath

O DirectXMath fornece uma solução matemática otimizada para o Windows.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                             | Descrição                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Introdução](pg-xnamath-getting-started.md)<br/>      | A biblioteca DirectXMath implementa uma interface ideal e portátil para operações de algebra aritméticos e lineares em vetores de ponto flutuante de precisão simples (2D, 3D e 4D) ou matrizes (3 × 3 e 4 × 4). <br/>                                                    |
| [Novidades](pg-xnamath-whatsnew.md)<br/>                  | A biblioteca DirectXMath é baseada na [biblioteca de matemática do XNA C++ SIMD versão 2, 4](https://walbourn.github.io/). Aqui, descrevemos como o DirectXMath difere do XNA Math e como as versões do DirectXMath são diferentes. <br/> |
| [Migração de código](pg-xnamath-migration.md)<br/>             | Esta visão geral descreve as alterações necessárias para migrar o código existente usando a biblioteca matemática do XNA para a biblioteca DirectXMath.<br/>                                                                                                                                 |
| [Trabalhando com D3DXMath](pg-xnamath-migration-d3dx.md)<br/> | D3DXMath é uma biblioteca auxiliar matemática para aplicativos Direct3D. <br/>                                                                                                                                                                                                |
| [Otimização de código](pg-xnamath-optimizing.md)<br/>         | Este tópico descreve as considerações e as estratégias de otimização com a biblioteca DirectXMath.<br/>                                                                                                                                                                |
| [Elementos internos da biblioteca](pg-xnamath-internals.md)<br/>          | Este tópico descreve o design interno da biblioteca DirectXMath.<br/>                                                                                                                                                                                         |
