# <a name="directxmath-library-template-functions"></a>Funções de modelo de biblioteca DirectXMath

Lista as funções de modelo fornecidas pelo DirectXMath.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                  | Descrição                                                                                                                                                                                                                               |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMMax**](xmmax-template.md)<br/>                             | Compara duas instâncias de tipo de dados numéricos ou duas instâncias de um objeto que dá suporte a uma sobrecarga de < e retorna a maior das duas instâncias. O tipo de dados dos argumentos e o valor de retorno são os mesmos.<br/>  |
| [**XMMin**](xmmin-template.md)<br/>                             | Compara duas instâncias de tipo de dados numéricos ou duas instâncias de um objeto que oferece suporte a uma sobrecarga de < e retorna a menor das duas instâncias. O tipo de dados dos argumentos e o valor de retorno são os mesmos.<br/> |
| [**XMVectorPermute**](xmvectorpermute-template.md)<br/>         | Permuta mudo dos componentes de dois vetores para criar um novo vetor.<br/>                                                                                                                                                                 |
| [**XMVectorRotateLeft**](xmvectorrotateleft-template.md)<br/>   | Gira o vetor à esquerda por um determinado número de elementos de 32 bits.<br/>                                                                                                                                                                  |
| [**XMVectorRotateRight**](xmvectorrotateright-template.md)<br/> | Gira o vetor à direita por um determinado número de elementos de 32 bits.<br/>                                                                                                                                                                 |
| [**XMVectorShiftLeft**](xmvectorshiftleft-template.md)<br/>     | Desloca um vetor à esquerda por um determinado número de elementos de 32 bits, preenchendo os elementos vagados com elementos de um segundo vetor.<br/>                                                                                                    |
| [**XMVectorSwizzle**](xmvectorswizzle-template.md)<br/>         | Swizzles um vetor.<br/>                                                                                                                                                                                                             |
| [**XMVectorInsert**](xmvectorinsert-template.md)<br/>           | Gira um vetor à esquerda por um determinado número de componentes de 32 bits e insere os elementos selecionados desse resultado em outro vetor.<br/>                                                                                                  |
