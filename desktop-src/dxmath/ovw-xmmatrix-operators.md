# <a name="xmmatrix-operators"></a>Operadores XMMATRIX

A [**estrutura XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                    | Descrição                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Operador \***](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Executa uma multiplicação de matriz da instância atual do `XMMATRIX` por outra instância do `XMMATRIX` .<br/>                                                                          |
| [**Operador \*=**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Executa uma multiplicação de matriz da instância atual do por outra instância do e retorna uma referência à `XMMATRIX` `XMMATRIX` instância atual, que foi atualizada. <br/> |
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Atribui os dados de matriz de uma instância de `XMMATRIX` à instância atual do e retorna um para a instância `XMMATRIX` `reference` atual. <br/>                                    |
| [**operator ()**](xmmatrix-operator-parens.md)<br/> | Acessa elementos de matriz específicos referenciados por linha e coluna da instância atual do `XMMATRIX` . <br/>                                                                           |
