# <a name="xmmatrix-operators"></a>Operadores XMMATRIX

A estrutura [**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) expõe os operadores a seguir.

> [!Note]  
> Os operadores listados aqui só estão disponíveis em C++.

 

## <a name="in-this-section"></a>Nesta seção



| Métodos                                                    | Descrição                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**operador \** _](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Executa uma multiplicação de matriz da instância atual do `XMMATRIX` por outra instância do `XMMATRIX` .<br/>                                                                          |
| [*operador \* =* _*](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Executa uma multiplicação de matriz da instância atual do `XMMATRIX` por outra instância do `XMMATRIX` e retorna uma referência à instância atual, que foi atualizada. <br/> |
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Atribui os dados de matriz de uma instância do `XMMATRIX` à instância atual do `XMMATRIX` e retorna um `reference` para a instância atual. <br/>                                    |
| [**operador ()**](xmmatrix-operator-parens.md)<br/> | Acessa elementos de matriz específicos referenciados por linha e coluna da instância atual do `XMMATRIX` . <br/>                                                                           |
