# <a name="name-lanpolicy-element"></a>Elemento Name (LANPolicy)

O elemento Name (LANPolicy) contém o nome de uma política de LAN com fio.

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

O elemento **Name** é definido pelo elemento [**LANPolicy**](lan-policyschema-lanpolicy-element.md) .

## <a name="remarks"></a>Comentários

Os nomes não diferenciam maiúsculas de minúsculas.

|-------------------------------------|------------------------------------------------------|

| Servidor mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do servidor 2008\]<br/> |
