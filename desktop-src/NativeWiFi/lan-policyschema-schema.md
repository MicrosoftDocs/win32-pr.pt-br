# <a name="lan_policy-schema"></a>Esquema de política de LAN \_

O esquema de política de LAN \_ define um perfil de política com fio (LAN).

O elemento raiz de um perfil de política de LAN é o elemento [**LANPolicy**](lan-policyschema-lanpolicy-element.md) . Cada perfil tem exatamente um elemento raiz. Todos os \_ elementos da diretiva de LAN estão no namespace `https://www.microsoft.com/networking/LAN/policy/v1` .
