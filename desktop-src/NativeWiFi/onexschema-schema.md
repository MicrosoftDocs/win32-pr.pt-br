# <a name="onex-schema"></a>Esquema OneX

O esquema OneX define elementos de configuração 802.1 X. Todos os elementos de esquema de OneX se aplicam a perfis com e sem fio. Para obter uma lista de elementos definidos, consulte [elementos de esquema de Onex](onexschema-elements.md).

O elemento raiz de um perfil 802.1 X é o elemento [**Onex**](onexschema-onex-element.md) . Cada perfil terá exatamente um elemento raiz. O elemento **Onex** está no `https://www.microsoft.com/networking/OneX/v1` namespace.

Para exibir perfis de exemplo para redes sem fio que incluem elementos de configuração do 802.1 X, consulte os exemplos de perfil a seguir:

-   [Exemplo de perfil Bootstrap](bootstrap-profile-sample.md)
-   [Exemplo de perfil FIPS](fips-profile-sample.md)
-   [Exemplo de perfil de Sign-On único](single-sign-on-profile-sample.md)
-   [Exemplo de perfil de WPA-Enterprise com PEAP-MSCHAPv2](wpa-enterprise-with-peap-mschapv2-profile-sample.md)
-   [Exemplo de perfil de WPA-Enterprise com TLS](wpa-enterprise-with-tls-profile-sample.md)
-   [Exemplo de perfil WPA2-Enterprise com PEAP-MSCHAPv2](wpa2-enterprise-with-peap-mschapv2-profile-sample.md)
-   [Exemplo de perfil WPA2-Enterprise com TLS](wpa2-enterprise-with-tls-profile-sample.md)

Para exibir perfis de exemplo para redes com fio que incluem elementos de configuração do 802.1 X, consulte os exemplos de perfil a seguir:
