# <a name="using-native-wifi"></a>Usando WiFi nativo

Esta seção descreve os procedimentos e as técnicas de programação empregadas com WiFi nativo. Ele inclui técnicas básicas de programação de WiFi nativo, como [exemplo de API Wi-Fi nativo](native-wifi-api-sample.md), bem como técnicas avançadas úteis para desenvolvedores NativeWifi experientes.

Os tópicos a seguir estão nesta seção:

-   [APIs de interface do usuário sem fio](wireless-user-interface-apis.md)
-   [Usando o Wi-Fi funções diretas](using-the-wi-fi-direct-api.md)
-   [Usando rede hospedada e compartilhamento de conexão com a Internet](using-hosted-network-and-internet-connection-sharing.md)
-   [Exemplo de API WiFi nativa](native-wifi-api-sample.md)
-   [Exemplo de rede hospedada sem fio](wireless-hosted-network-sample.md)
-   [Exemplos de perfil com fio](wired-profile-samples.md)
-   [Exemplos de perfil sem fio](wireless-profile-samples.md)

O tópico [APIs da interface do usuário sem fio](wireless-user-interface-apis.md) mostra alguns códigos usados para iniciar as interfaces do usuário do gerenciamento de rede sem fio.

As listas de tópicos [usando o Wi-Fi Direct Functions](using-the-wi-fi-direct-api.md) do Wi-Fi Direct Functions e discutem como as funções são usadas.

O tópico [usando rede hospedada e compartilhamento de conexão com a Internet](using-hosted-network-and-internet-connection-sharing.md) aborda como usar a rede hospedada sem fio e o ICS (compartilhamento de conexão com a Internet).

O SDK (Software Development Kit) do Microsoft Windows inclui dois exemplos de código que demonstram o uso da API WiFi nativa. Além disso, os perfis de exemplo estão disponíveis para demonstrar o uso de vários esquemas de perfil Wi-Fi nativos.

Os tópicos a seguir têm mais informações sobre os exemplos:

-   [Exemplo de API WiFi nativa](native-wifi-api-sample.md)
-   [Exemplo de rede hospedada sem fio](wireless-hosted-network-sample.md)
-   [Exemplos de perfil com fio](wired-profile-samples.md)
-   [Exemplos de perfil sem fio](wireless-profile-samples.md)
