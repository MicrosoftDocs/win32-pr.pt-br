# <a name="lan_policy-schema-elements"></a>Elementos do esquema de política de LAN \_

Um perfil de política com fio (LAN) contém os seguintes elementos de esquema. Todos os elementos nomeados estão no namespace `https://www.microsoft.com/networking/LAN/policy/v1` .

A lista a seguir mostra os elementos definidos na ordem em que os elementos aparecem em um perfil. A ordem dos elementos é imposta. Nem todos os elementos estão em todos os perfis, pois alguns elementos são opcionais.

Essa lista não mostra todos os elementos possíveis que podem aparecer em um perfil, pois elementos podem ser adicionados em **xs: qualquer** ponto de inserção.

-   [**LANPolicy**](lan-policyschema-lanpolicy-element.md)
    -   [**nome (LANPolicy)**](lan-policyschema-name-lanpolicy-element.md)
    -   [**Descrição (LANPolicy)**](lan-policyschema-description-lanpolicy-element.md)
    -   [**globalFlags (LANPolicy)**](lan-policyschema-globalflags-lanpolicy-element.md)
        -   [**enableAutoConfig (globalFlags)**](lan-policyschema-enableautoconfig-globalflags-element.md)
    -   [**ProfileList (LANPolicy)**](lan-policyschema-profilelist-lanpolicy-element.md)
