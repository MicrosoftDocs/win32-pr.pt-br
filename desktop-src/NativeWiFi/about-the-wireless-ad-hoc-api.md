# <a name="about-the-wireless-ad-hoc-api"></a>Sobre a API ad hoc sem fio

A API ad hoc sem fio expõe interfaces para enumerar e se conectar a redes ad hoc 802,11. Essas interfaces fornecem uma abordagem simplificada orientada a objeto para o gerenciamento de rede ad hoc. Enumeradores padrão são fornecidos para redes e interfaces. Esses enumeradores podem ser filtrados para enumerar somente redes ad hoc criadas pelo seu aplicativo.

> [!Note]  
> O modo ad hoc pode não estar disponível em versões futuras do Windows. A partir do Windows 8.1 e do Windows Server 2012 R2, use [Wi-Fi Direct](about-the-wi-fi-direct-api.md) .

 

Os desenvolvedores que usam a API ad hoc devem estar cientes de que as redes e os perfis criados usando a API ad hoc podem ser manipulados usando a API Wi-Fi nativa sem notificar os ouvintes de notificação de API ad hoc relevantes.

A API ad hoc sem fio é descrita detalhadamente na [referência ad hoc sem fio](wireless-ad-hoc-reference.md).

> [!Note]  
> A API ad hoc sem fio só tem suporte no Windows Vista e em versões posteriores do sistema operacional.
