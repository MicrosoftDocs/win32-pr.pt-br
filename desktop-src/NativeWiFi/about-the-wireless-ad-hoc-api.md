# <a name="about-the-wireless-ad-hoc-api"></a>Sobre a API Ad Hoc sem fio

A API Ad Hoc sem fio expõe interfaces para enumerar e conectar-se a redes ad hoc 802.11. Essas interfaces fornecem uma abordagem simplificada orientada a objeto para o gerenciamento de rede ad hoc. Enumeradores padrão são fornecidos para redes e interfaces. Esses enumeradores podem ser filtrados para enumerar apenas redes ad hoc criadas pelo seu aplicativo.

> [!Note]  
> O modo ad hoc pode não estar disponível em versões futuras do Windows. Começando com Windows 8.1 e Windows Server 2012 R2, use [Wi-Fi Direct](about-the-wi-fi-direct-api.md) em vez disso.

 

Os desenvolvedores que usam a API Ad Hoc devem estar cientes de que redes e perfis criados usando a API Ad Hoc podem ser manipulados usando a API Wifi Nativa sem notificar os ouvintes de notificação de API Ad Hoc relevantes.

A API Ad Hoc sem fio é descrita em detalhes na Referência [Ad Hoc sem fio.](wireless-ad-hoc-reference.md)

> [!Note]  
> A API Ad Hoc sem fio só tem suporte no Windows Vista e versões posteriores do sistema operacional.
