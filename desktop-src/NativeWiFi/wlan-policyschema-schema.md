# <a name="wlan_policy-schema"></a>Esquema de política de WLAN \_

O esquema de política de WLAN \_ define um perfil de política de WLAN usado pelo serviço de configuração automática WiFi nativo.

O elemento raiz de um perfil de política de WLAN é o elemento [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md) . Cada perfil terá exatamente um elemento raiz. Todos os \_ elementos da diretiva de WLAN estão no namespace `https://www.microsoft.com/networking/WLAN/policy/v1` .

-   [Elementos de policySchema da WLAN \_](wlan-policyschema-elements.md)
-   [\_Tipos simples de POLICYSCHEMA WLAN](wlan-policyschema-simple-types.md)
-   [\_Tipos complexos de WLAN policySchema](wlan-policyschema-complex-types.md)
