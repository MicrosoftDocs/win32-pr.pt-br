# <a name="wlan_policy-schema"></a>Esquema de política \_ WLAN

O esquema de \_ política WLAN define um perfil de política WLAN usado pelo serviço Native Wifi AutoConfig.

O elemento raiz de um perfil de política WLAN é o [**elemento WLANPolicy.**](wlan-policyschema-wlanpolicy-element.md) Cada perfil terá exatamente um elemento raiz. Todos os elementos de \_ política WLAN estão no namespace `https://www.microsoft.com/networking/WLAN/policy/v1` .

-   [WLAN \_ policySchema Elements](wlan-policyschema-elements.md)
-   [Política \_ WLANSchema Simple Types](wlan-policyschema-simple-types.md)
-   [Política WLAN \_ Tipos complexos de esquemas](wlan-policyschema-complex-types.md)
