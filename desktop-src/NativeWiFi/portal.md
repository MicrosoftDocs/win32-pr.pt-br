# <a name="native-wifi"></a>WiFi nativo

## <a name="purpose"></a>Finalidade

O componente de configuração automática WiFi nativo configura, conecta-se a e desconecta-se de redes sem fio. O WiFi nativo pode armazenar perfis nas redes com as quais interage na forma de documentos XML.

## <a name="developer-audience"></a>Público de desenvolvedores

A API Wi-Fi nativa foi projetada para desenvolvedores de C/C++. Os programadores devem estar familiarizados com a terminologia e os conceitos de rede sem fio.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O componente WiFi nativo requer clientes que executam o Windows Vista, o Windows XP com Service Pack 3 (SP3) ou a API de LAN sem fio para Windows XP com Service Pack 2 (SP2).
