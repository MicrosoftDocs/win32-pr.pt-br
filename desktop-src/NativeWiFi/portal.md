# <a name="native-wifi"></a>Wi-Fi nativo

## <a name="purpose"></a>Finalidade

O componente de configuração automática do Native Wifi configura, conecta-se e desconecta-se de redes sem fio. O Wifi nativo pode armazenar perfis nas redes com as que ele interage na forma de documentos XML.

## <a name="developer-audience"></a>Público de desenvolvedores

A API Wi-Fi Nativa foi projetada para desenvolvedores de C/C++. Os programadores devem estar familiarizados com conceitos e terminologia de rede sem fio.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O componente Wi-Fi Nativo requer clientes que executam Windows Vista, Windows XP com Service Pack 3 (SP3) ou API de LAN sem fio para Windows XP com Service Pack 2 (SP2).
