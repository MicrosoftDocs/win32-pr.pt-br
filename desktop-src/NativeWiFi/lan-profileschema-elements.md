# <a name="lan_profile-schema-elements"></a>Elementos de \_ esquema do perfil lan

Um perfil de rede com fio contém os seguintes elementos de esquema. Todos os elementos nomeados estão no namespace `https://www.microsoft.com/networking/LAN/profile/v1` .

A lista a seguir mostra os elementos definidos na ordem em que os elementos aparecem em um perfil. A ordenação de elementos é imposta. Essa lista não mostra todos os elementos possíveis que podem aparecer em um perfil, pois os elementos podem ser **adicionados em xs:any pontos** de inserção.

-   [**LANProfile**](lan-profileschema-lanprofile-element.md)
    -   [**MSM (LANProfile)**](lan-profileschema-msm-lanprofile-element.md)
        -   [**security (MSM)**](lan-profileschema-security-msm-element.md)
            -   [**OneXEnforced (segurança)**](lan-profileschema-onexenforced-security-element.md)
            -   [**OneXEnabled (segurança)**](lan-profileschema-onexenabled-security-element.md)
