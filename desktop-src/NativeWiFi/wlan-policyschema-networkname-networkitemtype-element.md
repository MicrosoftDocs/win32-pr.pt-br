# <a name="networkname-networkitemtype-element"></a>Elemento NetworkName (networkitemtype)

O elemento NetworkName (networkitemtype) especifica o SSID (identificador de conjunto de serviços) de uma rede sem fio.

``` syntax
<xs:element name="networkName"
    type="networkNameType"
 />
```

O elemento **NetworkName** é definido pelo tipo complexo [**networkitemtype**](wlan-policyschema-networkitemtype-complextype.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Vista\]<br/>       |
| Servidor mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Server 2008\]<br/> |
