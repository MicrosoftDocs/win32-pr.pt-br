# <a name="name-wlanpolicy-element"></a>Elemento Name (WLANPolicy)

O elemento Name (WLANPolicy) contém o nome de uma política de LAN sem fio.

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

O elemento **Name** é definido pelo elemento [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md) .

## <a name="remarks"></a>Comentários

Os nomes não diferenciam maiúsculas de minúsculas.
