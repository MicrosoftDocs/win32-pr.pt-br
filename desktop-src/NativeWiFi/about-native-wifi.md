# <a name="about-native-wifi"></a>Sobre WiFi nativo

As funções nativas da API (interface de programação de aplicativo) WiFi têm duas finalidades: para gerenciar perfis de rede sem fio e para gerenciar conexões de rede sem fio. Os elementos da API são expostos pelo ACM (módulo de configuração automática). Os elementos da API de desconexão e de conexão expostos podem ser usados para substituir a lógica de configuração automática.
