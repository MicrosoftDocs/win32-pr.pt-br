# <a name="network-allowlist-element"></a>Elemento de rede (permitirlist)

O elemento de rede (permitir alist) define uma rede permitida. Um computador deve ter permissão para se conectar a uma rede permitida.

``` syntax
<xs:element name="network"
    type="networkItemType"
 />
```

O elemento **Network** é definido pelo elemento [**allowlist**](wlan-policyschema-allowlist-networkfilter-element.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Vista\]<br/>       |
| Servidor mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Server 2008\]<br/> |
