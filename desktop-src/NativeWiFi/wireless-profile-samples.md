# <a name="wireless-profile-samples"></a>Exemplos de perfil sem fio

Os tópicos a seguir mostram os perfis XML de exemplo para várias configurações sem fio. Esses perfis XML usam [ \_ elementos de esquema de perfil WLAN](wlan-profileschema-elements.md) e elementos de [esquema Onex](onexschema-elements.md).

Você pode adicionar um perfil sem fio ao repositório de perfis programaticamente chamando [**WlanSetProfile**](/windows/desktop/api/wlanapi/nf-wlanapi-wlansetprofile). Você também pode adicionar um perfil ao repositório de perfis usando o utilitário de linha de comando **netsh** . Para obter mais informações, consulte [comandos netsh para rede local sem fio (WLAN)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10)).
