# <a name="wireless-zero-configuration-structures"></a>Estruturas de configuração sem fio zero

\[Não há mais suporte para a interface de programação de configuração zero sem fio a partir do Windows Vista e do Windows Server 2008. Em vez disso, use a [API Wi-Fi nativa](native-wifi-reference.md), que fornece funcionalidade semelhante. Para obter mais informações, consulte [sobre a API Wi-Fi nativa](about-the-native-wifi-api.md).\]

As estruturas a seguir permitem que os desenvolvedores de software usem a funcionalidade fornecida pelo módulo de configuração zero sem fio.

-   [**parâmetros de EAPOL \_ INTF \_**](eapol-intf-params.md)
-   [**\_entrada INTF**](intf-entry.md)
-   [**\_entrada de chave INTF \_**](intf-key-entry.md)
-   [**\_tabela de chave INTFS \_**](intfs-key-table.md)
