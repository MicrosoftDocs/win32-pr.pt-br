# <a name="creating-a-basic-debugger"></a>Criando um depurador básico

[Depurar um processo em execução](debugging-a-running-process.md) descreve como anexar o depurador a um processo em execução. [Escrever o Loop Principal](writing-the-debugger-s-main-loop.md) do Depurador descreve as funções usadas no loop principal de execução do depurador.

Embora a maioria das funções de depuração seja usada para criar um depurador, várias funções são projetadas para uso no processo que está sendo depurado. Para obter mais informações, consulte [Comunicando-se com o depurador](communicating-with-the-debugger.md).
