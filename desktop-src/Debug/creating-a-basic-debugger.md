# <a name="creating-a-basic-debugger"></a>Criando um depurador básico

A [depuração de um processo em execução](debugging-a-running-process.md) descreve a anexação do depurador a um processo em execução. [A gravação do loop principal do depurador](writing-the-debugger-s-main-loop.md) descreve as funções usadas no loop principal de execução do depurador.

Embora a maioria das funções de depuração sejam usadas para criar um depurador, várias funções são projetadas para uso no processo que está sendo depurado. Para obter mais informações, consulte [comunicando-se com o depurador](communicating-with-the-debugger.md).
