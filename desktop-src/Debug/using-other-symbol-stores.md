# <a name="using-other-symbol-stores"></a>Usando outros armazenamentos de símbolo

É possível escrever seu próprio programa de criação de armazenamento de símbolos, em vez de usar o SymStore.

Como as transações SymStore são todas registradas em arquivos de texto em formato CSV, você pode aproveitar todos os arquivos de log SymStore existentes para uso em seu próprio programa de banco de dados.

Se você planeja usar o programa SymSrv fornecido com as ferramentas de depuração para Windows, é recomendável usar o SymStore também. As atualizações desses dois programas sempre serão lançadas em conjunto e, portanto, suas versões sempre serão correspondentes.
