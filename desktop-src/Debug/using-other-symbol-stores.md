# <a name="using-other-symbol-stores"></a>Usando outros armazenamentos de símbolos

É possível escrever seu próprio programa de criação de repositório de símbolos, em vez de usar SymStore.

Como todas as transações SymStore são registradas em arquivos de texto no formato CSV, você pode aproveitar os arquivos de log SymStore existentes para uso em seu próprio programa de banco de dados.

Se você planeja usar o programa SymSrv fornecido com as Ferramentas de Depuração para Windows, é recomendável usar o SymStore também. As atualizações desses dois programas sempre serão lançadas juntas e, portanto, suas versões sempre corresponderão.
