# <a name="fatal-application-exit"></a>Saída fatal do aplicativo

A função **FatalAppExit** exibe uma caixa de mensagem e encerra o aplicativo quando o usuário fecha a caixa de mensagem. Essa função deve ser usada apenas como último recurso, porque ela pode não liberar a memória ou os arquivos pertencentes ao aplicativo.
