# <a name="debug-support-from-process-thread-and-exception-functions"></a>Suporte de depuração de funções de processo, thread e exceção

Algumas funções essenciais à depuração são, na verdade, classificadas como funções de processamento, thread ou tratamento de exceção. Para obter uma descrição de como iniciar um processo que será depurado, consulte [processar funções](process-functions-for-debugging.md). Para obter informações sobre como examinar e manipular o contexto e a execução de um thread, consulte [funções de thread](thread-functions-for-debugging.md). Para obter uma descrição de como um depurador deve tratar exceções, consulte [funções de tratamento](exception-handling-functions-for-debugging.md)de exceções.
