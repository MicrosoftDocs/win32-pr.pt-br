# <a name="image-help-library"></a>Biblioteca de Ajuda da Imagem

Esta visão geral descreve o conjunto de funções fornecido pela DLL ImageHlp. Essas funções permitem que você trabalhe com uma imagem PE (executável portátil).

-   [Sobre ImageHlp](about-imagehlp.md)
-   [Referência de ImageHlp](imagehlp-reference.md)

Você não pode redistribuir a DLL ImageHlp incluída no sistema operacional. Um subconjunto das funções está incluído na Biblioteca de Ajuda [de Depuração](debug-help-library.md), que é restrivível.
