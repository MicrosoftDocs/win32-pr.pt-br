# <a name="image-help-library"></a>Biblioteca de ajuda de imagem

Esta visão geral descreve o conjunto de funções fornecido pela DLL do ImageHlp. Essas funções permitem que você trabalhe com uma imagem executável portátil (PE).

-   [Sobre o ImageHlp](about-imagehlp.md)
-   [Referência do ImageHlp](imagehlp-reference.md)

Não é possível redistribuir a DLL do ImageHlp que está incluída com o sistema operacional. Um subconjunto das funções é incluído na [biblioteca de ajuda de depuração](debug-help-library.md), que é restributable.
