# <a name="debug-help-library"></a>Biblioteca de Ajuda de Depuração

Esta visão geral descreve o conjunto de funções fornecido pela biblioteca de ajuda de depuração, DbgHelp. Ele contém um conjunto de rotinas de suporte de depuração que permitem trabalhar com imagens executáveis no formato PE (executável portátil).

A documentação do DbgHelp é a seguinte:

- [Sobre DbgHelp](about-dbghelp.md)
- [Usando DbgHelp](using-dbghelp.md)
- [Referência de DbgHelp](dbghelp-reference.md)

Para obter a versão mais recente do DbgHelp.dll, acesse [https://developer.microsoft.com/windows/downloads/windows-10-sdk](https://developer.microsoft.com/windows/downloads/windows-10-sdk) e baixe Ferramentas de Depuração para Windows.

Para uma descrição do formato PE, baixe a especificação do seguinte local: [https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx](https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx) .

Para obter informações sobre como procurar informações encontradas em arquivos .pdb, consulte o SDK de Acesso à [Interface de Depuração](/visualstudio/debugger/debug-interface-access/debug-interface-access-sdk?view=vs-2015).
