# <a name="error-handling-error-handling"></a>Tratamento de erro (tratamento de erro)

Os aplicativos bem escritos incluem o código de tratamento de erros que permite que eles se recuperem normalmente de erros inesperados. Quando ocorre um erro, o aplicativo pode precisar solicitar a intervenção do usuário ou pode ser capaz de se recuperar por conta própria. Em casos extremos, o aplicativo pode desconectar o usuário ou desligar o sistema.

-   [Sobre o tratamento de erros](about-error-handling.md)
-   [Usando o tratamento de erros](using-error-handling.md)
-   [Referência de tratamento de erro](error-handling-reference.md)

Para obter informações sobre manipulação de exceção, consulte [manipulação de exceção estruturada](structured-exception-handling.md).
