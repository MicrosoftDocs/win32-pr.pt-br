# <a name="intel-avx"></a>Intel AVX

## <a name="purpose"></a>Finalidade

AVX (extensões de vetor avançado) da Intel é uma extensão de vetor de ponto flutuante de 256 bits da arquitetura Intel. Ele inclui extensões para os conjuntos de instruções e de registro.

A Microsoft desenvolveu alguns aprimoramentos de API, como o XState functions, que permitem que os aplicativos acessem e manipulem informações e estado do recurso do processador estendido, incluindo o Intel AVX.

## <a name="developer-audience"></a>Público de desenvolvedores

O Intel AVX foi projetado para uso por aplicativos que são intensivamente de computação de ponto flutuante e podem ser vetorizados. Aplicativos de exemplo incluem processamento de áudio e codecs de áudio, aplicativos de edição de imagens e vídeos, análise de serviços financeiros e software de modelagem, além de software de fabricação e engenharia.
