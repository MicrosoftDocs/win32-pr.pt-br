# <a name="symbol-handling"></a>Manipulação de símbolos

As funções de manipulador de símbolo fornecem aos aplicativos acesso fácil e portátil às informações de depuração simbólicas de uma imagem executável. Essas funções são usadas exclusivamente para garantir o acesso a informações simbólicas. Isso é necessário porque essas funções isolam o aplicativo do formato de símbolo.

Para mais informações, consulte os seguintes tópicos:

-   [Caminhos de símbolo](symbol-paths.md)
-   [Carregamento de símbolos](symbol-loading.md)
-   [Carregamento de símbolo adiado](deferred-symbol-loading.md)
-   [Nomes de símbolos decorados](decorated-symbol-names.md)
-   [Localizando símbolos](finding-symbols.md)
-   [Símbolos públicos, globais e locais](public--global--and-local-symbols.md)
-   [Limpeza do manipulador de símbolos](symbol-handler-cleanup.md)
