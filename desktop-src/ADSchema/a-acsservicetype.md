# <a name="acs-service-type-attribute"></a>ACS-atributo de tipo de serviço

O tipo de serviço do ACS. Carga controlada ou largura de banda garantida.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-tipo de serviço                     |
| LDAP-Display-Name | aCSServiceType                       |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.762               |
| System-ID-GUID    | 7f56127f-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
