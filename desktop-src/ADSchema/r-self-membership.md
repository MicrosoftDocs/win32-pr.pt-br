# <a name="self-membership-validated-writes"></a>Self-Membership de gravações validadas

Permissão de gravação validada para habilitar a atualização da Associação de um grupo em termos de adição/remoção de uma conta própria.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Self-Membership                      |
| Display-Name | Adicionar/remover si mesmo como membro            |
| GUID de direitos  | bf9679c0-0de6-11d0-a285-00aa003049e2 |
