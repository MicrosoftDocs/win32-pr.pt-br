# <a name="self-membership-validated-writes"></a>Self-Membership gravações validadas

Permissão de gravação validada para habilitar a associação de atualização de um grupo em termos de adição/remoção da própria conta.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Self-Membership                      |
| Display-Name | Adicionar/remover auto como membro            |
| RIGHTS-GUID  | bf9679c0-0de6-11d0-a285-00aa003049e2 |
