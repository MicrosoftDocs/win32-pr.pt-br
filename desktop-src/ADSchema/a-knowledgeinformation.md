# <a name="knowledge-information-attribute"></a>Knowledge-Information atributo

Especifica uma descrição acumulada e legível do conhecimento do Mastered Directory por um agente de sistema de diretório específico.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Knowledge-Information                       |
| LDAP-Display-Name | knowledgeInformation                        |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 2.5.4.2                                     |
| System-ID-GUID    | 1677581f-47f3-11d1-a9c3-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md) |
