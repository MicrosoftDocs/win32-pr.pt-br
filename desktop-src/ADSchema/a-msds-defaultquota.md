# <a name="ms-ds-default-quota-attribute"></a>ms-DS-default-atributo quota

A cota padrão que será aplicada a uma entidade de segurança que cria um objeto no NC se não existir nenhuma especificação de cota que cubra a entidade de segurança.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ms-DS-padrão-quota                  |
| LDAP-Display-Name | msDS-defaultquota                    |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-ID-GUID    | 6818f726-674b-441b-8a3a-f40596374cea |
| Syntax            | [**Enumeração**](s-enumeration.md) |
