# <a name="ms-ds-never-reveal-group-attribute"></a>atributo ms-DS-neve-Reveal-Group

Usado com RODCs para definir quais usuários, computadores e grupos não têm permissão para ter suas senhas em cache em um RODC.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Never-Reveal-Group                |
| LDAP-Display-Name | msDS-NeverRevealGroup                   |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1926                 |
| System-ID-GUID    | 15585999-fd49-4D66-b25d-eeb96aba8174    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
