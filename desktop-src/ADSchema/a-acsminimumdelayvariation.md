# <a name="acs-minimum-delay-variation-attribute"></a>ACS-atributo de variação de atraso mínimo

O atributo **ACS-Minimum-Delay-Variance** é somente para uso interno. Com base em RFC2210.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-mínimo-variação de atraso          |
| LDAP-Display-Name | aCSMinimumDelayVariation             |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1317              |
| System-ID-GUID    | 9c65329b-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
