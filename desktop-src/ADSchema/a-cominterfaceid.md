# <a name="com-interfaceid-attribute"></a>COM-InterfaceID atributo

Esse atributo armazena a lista de interfaces que são implementadas neste pacote de aplicativos.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-InterfaceID                                                                  |
| LDAP-Display-Name | cominterfaceid                                                                   |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Qualquer pessoa pode atualizar esse objeto com base na segurança do objeto que está sendo criado. |
| Frequência de atualização  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.20                                                            |
| System-ID-GUID    | bf96793c-0de6-11d0-a285-00aa003049e2                                             |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                      |
