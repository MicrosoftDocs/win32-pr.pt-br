# <a name="bridgehead-server-list-bl-attribute"></a>Atributo bridgehead-Server-List-BL

A lista de servidores que são bridgeheads para replicação.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Bridgehead-Server-List-BL               |
| LDAP-Display-Name | bridgeheadServerListBL                  |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Esse valor é definido pelo sistema.        |
| Frequência de atualização  | Sempre que um site é configurado.               |
| Attribute-Id      | 1.2.840.113556.1.4.820                  |
| System-ID-GUID    | d50c2cdb-8951-11d1-aebc-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
