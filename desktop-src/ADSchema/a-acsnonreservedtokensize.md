# <a name="acs-non-reserved-token-size-attribute"></a>ACS-não reservado-atributo de tamanho de token

O atributo **ACS-non-reserved-token-size** é somente para uso interno. Com base em RFC2814.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-não reservado-tamanho de token          |
| LDAP-Display-Name | aCSNonReservedTokenSize              |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1319              |
| System-ID-GUID    | a916d7c9-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
