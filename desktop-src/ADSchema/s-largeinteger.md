# <a name="largeinteger-syntax"></a>Sintaxe de LargeInteger

Representa um valor inteiro com sinal de 64 bits.



| Entrada | Valor |
|--------------|------------------------------------------------------------------------------------|
| Nome         | LargeInteger                                                                       |
| ID da sintaxe    | 2.5.5.16                                                                           |
| ID DO OM        | 65                                                                                 |
| Tipo de MAPI    | \-                                                                                 |
| Tipo de ADS     | \_inteiro grande \_ ADSTYPE                                                            |
| Tipo de variante | expedição de VT \_                                                                       |
| Tipo SDS     | Um objeto COM que pode ser convertido em um [**IADsLargeInteger**](/windows/desktop/api/iads/nn-iads-iadslargeinteger). |
