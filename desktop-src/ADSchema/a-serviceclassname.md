# <a name="service-class-name-attribute"></a>Atributo de nome de classe de serviço

O nome da cadeia de caracteres do serviço que um ponto de administração representa.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Nome da classe de serviço                          |
| LDAP-Display-Name | serviceClassName                            |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.509                      |
| System-ID-GUID    | b7b1311d-b82e-11d0-afee-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
