# <a name="ms-wmi-intflags1-attribute"></a>atributo ms-WMI-intFlags1

O atributo **MS-WMI-intFlags1** é reservado para uso interno.

|-------------------|--------------------------------------|
| CN                | MS-WMI-intFlags1                     |
| LDAP-Display-Name | msWMI-intFlags1                      |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1678              |
| System-ID-GUID    | 18e006b9-6445-48e3-9dcf-b5ecfbc4df8e |
