# <a name="ds-ui-shell-maximum-attribute"></a>DS-UI-Shell-máximo de atributos

Este é o número máximo padrão de objetos a serem mostrados em um contêiner pela interface do usuário do Shell.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | DS-UI-Shell-máximo                  |
| LDAP-Display-Name | dSUIShellMaximum                     |
| Tamanho              | \-                                   |
| Privilégio de atualização  | Administrador de domínio                 |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1345              |
| System-ID-GUID    | fcca766a-6f91-11d2-9905-0000f87a57d4 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
