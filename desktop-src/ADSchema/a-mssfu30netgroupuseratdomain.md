# <a name="mssfu-30-netgroup-user-at-domain-attribute"></a>msSFU-30-netgroup-atributo usuário-at-domínio

Contém parte do mapa do netgroup que representa cadeias de caracteres computadas, como " user@domain ".



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | msSFU-30-netgroup-usuário-em-domínio     |
| LDAP-Display-Name | msSFU30NetgroupUserAtDomain          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.349          |
| System-ID-GUID    | a9e84eed-e630-4b67-b4b3-cad2a82d345e |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
