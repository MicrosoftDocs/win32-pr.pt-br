# <a name="frs-time-last-command-attribute"></a>Atributo de tempo de FRS-último-comando

A hora em que o último comando foi executado.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-tempo-último comando                                         |
| LDAP-Display-Name | fRSTimeLastCommand                                            |
| Tamanho              | \-                                                            |
| Privilégio de atualização  | \-                                                            |
| Frequência de atualização  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.880                                        |
| System-ID-GUID    | 2a132583-9373-11d1-aebc-0000f80367c1                          |
| Syntax            | [**Cadeia de caracteres (em tempo geral)**](s-string-generalized-time.md) |
