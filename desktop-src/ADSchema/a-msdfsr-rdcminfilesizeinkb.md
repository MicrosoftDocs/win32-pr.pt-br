# <a name="ms-dfsr-rdcminfilesizeinkb-attribute"></a>atributo ms-DFSR-RdcMinFileSizeInKb

Contém o tamanho mínimo do arquivo, em kilobytes, ao qual aplicar a RDC.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-RdcMinFileSizeInKb           |
| LDAP-Display-Name | msDFSR-RdcMinFileSizeInKb            |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.20           |
| System-ID-GUID    | f402a330-ace5-4dc1-8cc9-74d900bf8ae0 |
| Syntax            | [**Intervalo**](s-interval.md)       |
