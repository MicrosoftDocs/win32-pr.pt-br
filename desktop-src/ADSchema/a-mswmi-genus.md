# <a name="ms-wmi-genus-attribute"></a>atributo ms-WMI-genus

Identifica o tipo de objeto de uma codificação.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-genus                                                    |
| LDAP-Display-Name | msWMI-Genus                                                     |
| Tamanho              | 4 bytes. 1 indica uma classe. 2 indica uma instância.          |
| Privilégio de atualização  | Administrador de domínio                                            |
| Frequência de atualização  | Sempre que uma classe que contém o atributo é adicionada ou modificada. |
| Attribute-Id      | 1.2.840.113556.1.4.1677                                         |
| System-ID-GUID    | 50c8673a-8f56-4614-9308-9e1340fb9af3                            |
| Syntax            | [**Enumeração**](s-enumeration.md)                            |
