# <a name="common-name-attribute"></a>Common-Name atributo

O nome que representa um objeto. Usado para executar pesquisas.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| LDAP-Display-Name | cn                                          |
| Tamanho              | O tamanho médio é de 64 bytes.                   |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | Somente quando um novo objeto é criado.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-ID-GUID    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
