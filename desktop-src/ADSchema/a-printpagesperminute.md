# <a name="print-pages-per-minute-attribute"></a>Imprimir páginas por atributo por minuto

Taxa de impressão fornecida pelo driver em páginas por minuto.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Imprimir páginas por minuto               |
| LDAP-Display-Name | printPagesPerMinute                  |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.631               |
| System-ID-GUID    | 19405b97-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
