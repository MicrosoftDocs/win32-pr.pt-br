# <a name="ms-sql-alias-attribute"></a>Atributo MS-SQL-alias

Um alias usado para se conectar ao banco de dados. O padrão é definido como alias.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-alias                                |
| LDAP-Display-Name | mS-SQL-alias                                |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1395                     |
| System-ID-GUID    | e0c6baae-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
