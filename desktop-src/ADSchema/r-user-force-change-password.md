# <a name="user-force-change-password-extended-right"></a>Usuário-Force-alteração-senha estendida à direita

Permite redefinir uma senha em uma conta de usuário.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Usuário-forçar alteração-senha           |
| Display-Name | Redefinir senha                       |
| GUID de direitos  | 00299570-246d-11d0-a768-00aa006e0529 |
