# <a name="gpc-wql-filter-attribute"></a>Atributo GPC-WQL-Filter

Usado para armazenar uma cadeia de caracteres que contém um GUID para o filtro e um caminho de namespace do WMI.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-WQL-filtro                                                                  |
| LDAP-Display-Name | gPCWQLFilter                                                                    |
| Tamanho              | \-                                                                              |
| Privilégio de atualização  | Administrador de domínio                                                            |
| Frequência de atualização  | Somente quando o administrador altera a propriedade de filtro na interface do usuário do Política de Grupo. |
| Attribute-Id      | 1.2.840.113556.1.4.1694                                                         |
| System-ID-GUID    | 7bd4c7a6-1add-4436-8c04-3999a880154c                                            |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                     |
