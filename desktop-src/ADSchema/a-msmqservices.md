| CN                | MSMQ-Services                        |
| Ldap-Display-Name | Msmqservices                         |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.950               |
| System-Id-Guid    | 9a0dc33d-c100-11d1-football5-0080c76670c0 |
