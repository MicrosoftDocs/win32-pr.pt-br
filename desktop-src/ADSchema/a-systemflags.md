# <a name="system-flags-attribute"></a>System-Flags atributo

Um valor inteiro que contém sinalizadores que definem propriedades adicionais da classe. Consulte Observações.



| Entrada | Valor |
|-------------------|------------------------------------------------|
| CN                | System-Flags                                   |
| LDAP-Display-Name | systemFlags                                    |
| Tamanho              | \-                                             |
| Privilégio de atualização  | Esse valor é definido pelo administrador do esquema. |
| Frequência de atualização  | Sempre que o estado do objeto for alterado.      |
| Attribute-Id      | 1.2.840.113556.1.4.375                         |
| System-ID-GUID    | e0fa1e62-9b45-11d0-afdd-00c04fd930c9           |
| Syntax            | [**Enumeração**](s-enumeration.md)           |

## <a name="remarks"></a>Comentários

Esse atributo pode ser zero ou uma combinação de um ou mais dos valores a seguir.



| Valor                   | Descrição                                                                                                                                                                                                                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Quando aplicado a um atributo, o atributo não será replicado. Quando aplicado a um objeto de [**referência cruzada**](c-crossref.md) , o contexto de nomenclatura está em NTDS.<br/>                                                                                                                          |
| 2 (0x00000002)          | Quando aplicado a um atributo, o atributo será replicado para o catálogo global. Quando aplicado a um objeto de [**referência cruzada**](c-crossref.md) , o contexto de nomenclatura é um domínio.<br/>                                                                                                       |
| 4 (0x00000004)          | Quando aplicado a um atributo, o atributo é construído.                                                                                                                                                                                                                                     |
| 16 (0x00000010)         | Quando definido, indica que o objeto é um objeto da categoria 1. Um objeto Category 1 é uma classe ou um atributo que é incluído no esquema base incluído no sistema.                                                                                                                                |
| 33554432 (0x02000000)   | O objeto não é movido para o contêiner objetos excluídos quando ele é excluído. Ele será excluído imediatamente.                                                                                                                                                                                    |
| 67108864 (0x04000000)   | O objeto não pode ser movido.                                                                                                                                                                                                                                                                     |
| 134217728 (0x08000000)  | O objeto não pode ser renomeado.                                                                                                                                                                                                                                                                   |
| 268435456 (0x10000000)  | Para objetos na partição de configuração, se esse sinalizador for definido, o objeto poderá ser movido com restrições; caso contrário, o objeto não poderá ser movido. Por padrão, esse sinalizador não é definido em novos objetos criados na partição de configuração. Esse sinalizador só pode ser definido durante a criação do objeto. |
| 536870912 (0x20000000)  | Para objetos na partição de configuração, se esse sinalizador for definido, o objeto poderá ser movido; caso contrário, o objeto não poderá ser movido. Por padrão, esse sinalizador não é definido em novos objetos criados na partição de configuração. Esse sinalizador só pode ser definido durante a criação do objeto.                   |
| 1073741824 (0x40000000) | Para objetos na partição de configuração, se esse sinalizador for definido, o objeto poderá ser renomeado; caso contrário, o objeto não poderá ser renomeado. Por padrão, esse sinalizador não é definido em novos objetos criados na partição de configuração. Esse sinalizador só pode ser definido durante a criação do objeto.               |
| 2147483648 (0x80000000) | O objeto não pode ser excluído.                                                                                                                                                                                                                                                                   |
