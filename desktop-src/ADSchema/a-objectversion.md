# <a name="object-version-attribute"></a>Object-Version atributo

Isso pode ser usado para armazenar um número de versão para o objeto.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------|
| CN                | Object-Version                                                 |
| LDAP-Display-Name | objectVersion                                                  |
| Tamanho              | 4 bytes                                                        |
| Privilégio de atualização  | O designer do objeto definiria esse valor.               |
| Frequência de atualização  | Esse valor deve ser incrementado toda vez que o objeto for alterado. |
| Attribute-Id      | 1.2.840.113556.1.2.76                                          |
| System-ID-GUID    | 16775848-47f3-11d1-a9c3-0000f80367c1                           |
| Syntax            | [**Enumeração**](s-enumeration.md)                           |
