# <a name="msmq-send-extended-right"></a>msmq-Send estendido direito

| CN           | msmq-Send                            |
| Display-Name | Enviar Mensagem                         |
| RIGHTS-GUID  | 06bd3202-df3e-11d1-9c86-006008764d0e |
