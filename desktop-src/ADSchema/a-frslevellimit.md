# <a name="frs-level-limit-attribute"></a>Atributo de limite de nível de FRS

Limite a profundidade da árvore de diretórios a ser replicada para a replicação de arquivos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Limite de nível de FRS                      |
| LDAP-Display-Name | fRSLevelLimit                        |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.534               |
| System-ID-GUID    | 5245801e-ca6a-11d0-afff-0000f80367c1 |
| Sintaxe            | [**Enumeração**](s-enumeration.md) |
