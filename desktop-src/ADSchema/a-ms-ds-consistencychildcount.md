# <a name="ms-ds-consistency-child-count-attribute"></a>Atributo MS-DS-Consistency-filho-Count

Esse atributo é usado para verificar a consistência entre o diretório e outro objeto, banco de dados ou aplicativo, comparando uma contagem de objetos filho.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-DS-Consistency-filho-Count        |
| LDAP-Display-Name | mS-DS-ConsistencyChildCount          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1361              |
| System-ID-GUID    | 178b7bc2-b63a-11d2-90e1-00c04fd91ab1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
