# <a name="phone-pager-primary-attribute"></a>O atributo Phone-pager-primário

O número do pager principal.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telefone-pager-primário                                                              |
| LDAP-Display-Name | pager                                                                            |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta.                                           |
| Frequência de atualização  | Quando o registro do usuário é criado e sempre que o número de telefone precisa ser alterado. |
| Attribute-Id      | 0.9.2342.19200300.100.1.42                                                       |
| System-ID-GUID    | f0f8ffa6-1191-11d0-a060-00aa006c33ed                                             |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                      |
