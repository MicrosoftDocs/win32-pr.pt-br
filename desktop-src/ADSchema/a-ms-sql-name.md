# <a name="ms-sql-name-attribute"></a>Atributo MS-SQL-Name

O nome de uma instância de SQL Server. O padrão é MSSQLSERVER.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Name                                 |
| LDAP-Display-Name | mS-SQL-Name                                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | Na configuração do sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-ID-GUID    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
