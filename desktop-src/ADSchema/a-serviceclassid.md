# <a name="service-class-id-attribute"></a>Atributo de ID de classe de serviço

O GUID da classe de serviço.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | ID de classe de serviço                                      |
| LDAP-Display-Name | objectclassid                                        |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.122                                |
| System-ID-GUID    | bf967a35-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
