# <a name="stringnt-sec-desc-syntax"></a>Sintaxe de cadeia de caracteres (NT-SEC-DESC)

Uma cadeia de caracteres de octeto que contém um descritor de segurança do Windows NT ou do Windows 2000.



| Entrada | Valor |
|--------------|------------------------------------------------------------------------------------------------|
| Nome         | Cadeia de caracteres (NT-SEC-DESC)                                                                            |
| ID da sintaxe    | 2.5.5.15                                                                                       |
| ID DO OM        | 66                                                                                             |
| Tipo de MAPI    | BINARY                                                                                         |
| Tipo de ADS     | \_ \_ descritor de segurança do ADSTYPE NT \_                                                              |
| Tipo de variante | expedição de VT \_                                                                                   |
| Tipo SDS     | Um objeto COM que pode ser convertido em um [**IADsSecurityDescriptor**](/windows/desktop/api/iads/nn-iads-iadssecuritydescriptor). |
