A lista de sufixos permitidos para o atributo dNSHostName em objetos de computador.

|-------------------|---------------------------------------------|
| CN                | ms-DS-allowed-sufixos-DNS                  |
| LDAP-Display-Name | msDS-AllowedDNSSuffixes                     |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1710                     |
| System-ID-GUID    | 8469441b-9ac4-4e45-8205-bd219dbf672d        |
