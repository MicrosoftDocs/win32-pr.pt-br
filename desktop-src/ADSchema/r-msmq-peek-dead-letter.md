# <a name="msmq-peek-dead-letter-extended-right"></a>MSMQ-Peek-Dead-letra estendida à direita

Permite exibir mensagens na fila de mensagem mortas.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | MSMQ-Peek-mensagens mortas                |
| Display-Name | Inspecionar mensagens mortas                     |
| GUID de direitos  | 4b6e08c1-df3c-11d1-9c86-006008764d0e |
