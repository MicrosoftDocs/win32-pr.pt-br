# <a name="ldap-admin-limits-attribute"></a>Atributo LDAP-admin-Limits

Contém um conjunto de pares atributo-valor que define os limites administrativos do servidor LDAP.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | LDAP-admin-Limits                           |
| LDAP-Display-Name | lDAPAdminLimits                             |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.843                      |
| System-ID-GUID    | 7359a352-90f7-11d1-aebc-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
