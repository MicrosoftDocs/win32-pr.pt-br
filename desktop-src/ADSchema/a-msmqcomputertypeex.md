# <a name="msmq-computer-type-ex-attribute"></a>Atributo MSMQ-tipo de computador-ex

Uma cadeia de caracteres que descreve a versão do sistema operacional e a versão do MSMQ.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MSMQ-tipo de computador-ex                       |
| LDAP-Display-Name | mSMQComputerTypeEx                          |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1417                     |
| System-ID-GUID    | 18120de8-f4c4-4341-bd95-32eb5bcf7c80        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
