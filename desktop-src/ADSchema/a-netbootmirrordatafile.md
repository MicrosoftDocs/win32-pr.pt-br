O atributo **netboot-espelho-Data-File** é reservado para uso interno.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Netboot-espelho-Data-File                    |
| LDAP-Display-Name | netbootMirrorDataFile                       |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1241                     |
