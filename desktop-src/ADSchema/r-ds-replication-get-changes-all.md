# <a name="ds-replication-get-changes-all-extended-right"></a>DS-Replication-Get-Changes-todos os direitos estendidos

Controle o direito de acesso que permite a replicação de dados de domínio secretos.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | DS-Replication-Get-Changes-All       |
| Display-Name | Replicar todas as alterações de diretório    |
| GUID de direitos  | 1131f6ad-9c07-11d1-f79f-00c04fc2dcd2 |
