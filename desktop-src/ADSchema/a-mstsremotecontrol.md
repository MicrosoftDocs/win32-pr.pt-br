# <a name="ms-ts-remote-control-attribute"></a>atributo MS-TS-Remote-Control

O controle remoto dos serviços de terminal especifica se a observação remota ou o controle remoto da sessão dos serviços de terminal do usuário devem ser permitidos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-TS-controle remoto                 |
| LDAP-Display-Name | msTSRemoteControl                    |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1980              |
| System-ID-GUID    | 15177226-8642-468b-8c48-03ddfd004982 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
