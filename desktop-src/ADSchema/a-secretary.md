# <a name="secretary-attribute"></a>atributo Secretary

Contém o nome distinto da secretária de uma conta.

|-------------------|-----------------------------------------|
| CN                | secretário                               |
| LDAP-Display-Name | secretário                               |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.21              |
| System-ID-GUID    | 01072d9a-98ad-4a53-9744-e83e287278fb    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
