# <a name="acs-max-aggregate-peak-rate-per-user-attribute"></a>ACS-Max-Aggregate-taxa de pico por atributo de usuário

A taxa máxima de pico que qualquer usuário pode ter para todos os fluxos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Aggregate-pico de taxa por usuário |
| LDAP-Display-Name | aCSMaxAggregatePeakRatePerUser       |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.897               |
| System-ID-GUID    | f072230c-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
