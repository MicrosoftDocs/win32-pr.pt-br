# <a name="governs-id-attribute"></a>Governa-atributo de ID

A ID de objeto exclusiva da classe definida por este objeto de Class-Schema.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------|
| CN                | Regis-ID                                                      |
| LDAP-Display-Name | governsID                                                       |
| Tamanho              | \-                                                              |
| Privilégio de atualização  | \-                                                              |
| Frequência de atualização  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.22                                           |
| System-ID-GUID    | bf96797d-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**Cadeia de caracteres (identificador de objeto)**](s-string-object-identifier.md) |
