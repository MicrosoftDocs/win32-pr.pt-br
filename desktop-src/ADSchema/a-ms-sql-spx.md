# <a name="ms-sql-spx-attribute"></a>Atributo MS-SQL-SPX

O ponto de conexão do SPX.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-SPX                                  |
| LDAP-Display-Name | mS-SQL-SPX                                  |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | Na inicialização do sistema.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1376                     |
| System-ID-GUID    | 86b08004-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
