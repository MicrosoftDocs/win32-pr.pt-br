# <a name="acs-non-reserved-tx-size-attribute"></a>ACS – atributo non-reserved-TX-size

O tamanho do Bucket de token que um aplicativo pode usar antes que uma reserva esteja em vigor.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-não reservado-TX-tamanho             |
| LDAP-Display-Name | aCSNonReservedTxSize                 |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.898               |
| System-ID-GUID    | f072230d-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
