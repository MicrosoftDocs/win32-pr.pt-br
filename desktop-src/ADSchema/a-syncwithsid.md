# <a name="sync-with-sid-attribute"></a>Atributo Sync-with-SID

Para a sincronização de diretiva local/objeto de grupo do SAM Builtin, esse é o grupo local ao qual um objeto corresponde.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Sincronizar-com-SID                        |
| LDAP-Display-Name | syncWithSID                          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.667               |
| System-ID-GUID    | 037651e5-441d-11d1-a9c3-0000f80367c1 |
| Syntax            | [**Cadeia de caracteres (SID)**](s-string-sid.md)  |
