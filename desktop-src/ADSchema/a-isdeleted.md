# <a name="is-deleted-attribute"></a>Is-Deleted atributo

Se **for true**, esse objeto foi marcado para exclusão e não poderá ser instanciado. Depois que o período de marca para exclusão tiver expirado, ele será removido do sistema.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Is-Deleted                           |
| LDAP-Display-Name | isDeleted                            |
| Tamanho              | \-                                   |
| Privilégio de atualização  | Administrador de domínio                 |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.48                |
| System-ID-GUID    | bf96798f-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
