# <a name="ms-wmi-normalizedclass-attribute"></a>atributo ms-WMI-NormalizedClass

O nome de uma classe de política principal do WMI.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-NormalizedClass                      |
| LDAP-Display-Name | msWMI-NormalizedClass                       |
| Tamanho              | Menos de vinte caracteres.                |
| Privilégio de atualização  | Administrador de Política de Grupo                  |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1640                     |
| System-ID-GUID    | eaba628f-eb8e-4fe9-83fc-693be695559b        |
| Sintaxe            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
