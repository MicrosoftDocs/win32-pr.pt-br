# <a name="print-duplex-supported-attribute"></a>Print-duplex-atributo com suporte

Indica o tipo de suporte duplex que uma impressora tem.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Impressão-duplex-com suporte                                |
| LDAP-Display-Name | printDuplexSupported                                  |
| Tamanho              | 4 bytes. Duplex com suporte = 2. Simplex somente = 1 ou 0. |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1311                               |
| System-ID-GUID    | 281416cc-1968-11d0-a28f-00aa003049e2                  |
| Syntax            | [**Boolean**](s-boolean.md)                          |
