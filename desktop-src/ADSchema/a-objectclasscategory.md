# <a name="object-class-category-attribute"></a>Atributo de categoria de classe de objeto

Esse atributo contém o tipo de classe, como abstract, auxiliar ou Structured.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------------------|
| CN                | Categoria de classe de objeto                                                           |
| LDAP-Display-Name | objectClassCategory                                                             |
| Tamanho              | 4 bytes. Estrutural 1, abstrata 2, auxiliar 3. A classe 88, 0 não deve ser usada. |
| Privilégio de atualização  | O designer do objeto definiria esse valor.                                |
| Frequência de atualização  | Esse valor nunca deve ser alterado.                                                 |
| Attribute-Id      | 1.2.840.113556.1.2.370                                                          |
| System-ID-GUID    | bf9679e6-0de6-11d0-a285-00aa003049e2                                            |
| Syntax            | [**Enumeração**](s-enumeration.md)                                            |
