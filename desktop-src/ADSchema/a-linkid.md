# <a name="link-id-attribute"></a>Atributo de ID de link

Um inteiro que indica que o atributo é um atributo vinculado. Um inteiro par é um link para frente e um inteiro ímpar é um link para trás.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ID do link                              |
| LDAP-Display-Name | linkID                               |
| Tamanho              | \-                                   |
| Privilégio de atualização  | Administrador de esquema                 |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-ID-GUID    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
