# <a name="dit-content-rules-attribute"></a>DIT-atributo Content-Rules

Isso especifica o conteúdo permitido de entradas de uma determinada classe de objeto estrutural usando a identificação de um conjunto opcional de classes de objeto auxiliar e os atributos obrigatórios, opcionais e impedidos. Atributos coletivos são incluídos em DIT-Content-Rules, conforme definido no RFC 2251.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | DIT-Content-Rules                           |
| LDAP-Display-Name | dITContentRules                             |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | Sempre que uma nova classe é criada.            |
| Attribute-Id      | 2.5.21.2                                    |
| System-ID-GUID    | 9a7ad946-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
