# <a name="last-logoff-attribute"></a>Last-Logoff atributo

Esse atributo não é usado.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Last-Logoff                          |
| LDAP-Display-Name | lastLogoff                           |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Cada vez que o usuário faz logoff.         |
| Attribute-Id      | 1.2.840.113556.1.4.51                |
| System-ID-GUID    | bf967996-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
