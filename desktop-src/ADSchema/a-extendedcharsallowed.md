# <a name="extended-chars-allowed-attribute"></a>Caracteres estendidos-atributo permitido

Indica se caracteres estendidos são permitidos no valor deste atributo. Aplica-se somente aos atributos de cadeia de caracteres IA5, Numeric, imprimível e Teletex.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Caracteres estendidos-permitidos               |
| LDAP-Display-Name | extendedCharsAllowed                 |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.380               |
| System-ID-GUID    | bf967966-0de6-11d0-a285-00aa003049e2 |
| Sintaxe            | [**Boolean**](s-boolean.md)         |
