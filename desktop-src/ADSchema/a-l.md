# <a name="locality-name-attribute"></a>Locality-Name atributo

Representa o nome de uma localidade, como um município ou uma cidade.



| Entrada | Valor |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| LDAP-Display-Name | l                                                                            |
| Tamanho              | \-                                                                           |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta.                                       |
| Frequência de atualização  | Quando o registro do usuário é criado e sempre que a localidade precisa ser alterada. |
| Attribute-Id      | 2.5.4.7                                                                      |
| System-ID-GUID    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                  |
