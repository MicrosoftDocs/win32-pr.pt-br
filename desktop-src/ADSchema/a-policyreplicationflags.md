# <a name="policy-replication-flags-attribute"></a>Política-replicação-atributo de sinalizadores

Determina quais propriedades LSA são replicadas para os clientes.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Política-replicação-sinalizadores             |
| LDAP-Display-Name | policyReplicationFlags               |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.633               |
| System-ID-GUID    | 19405b96-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
