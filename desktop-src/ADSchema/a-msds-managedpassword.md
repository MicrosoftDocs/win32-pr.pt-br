# <a name="ms-ds-managedpassword-attribute"></a>atributo ms-DS-ManagedPassword

Esse atributo construído retorna um blob que contém o texto não criptografado anterior e a senha atual, TimeUntilEpochExpires e TimeUntilNextScheduledUpdate para um grupo MSA.
