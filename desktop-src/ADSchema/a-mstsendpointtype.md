# <a name="ms-ts-endpoint-type-attribute"></a>atributo MS-TS-Endpoint-Type

Esse atributo define se o computador é um computador físico ou uma máquina virtual.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-TS-Endpoint-Type                  |
| LDAP-Display-Name | msTSEndpointType                     |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2071              |
| System-ID-GUID    | 377ade80-e2d8-46c5-9bcd-6d9dec93b35e |
| Syntax            | [**Enumeração**](s-enumeration.md) |
