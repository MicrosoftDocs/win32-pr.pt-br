# <a name="ms-sql-informationdirectory-attribute"></a>Atributo MS-SQL-InformationDirectory

O diretório informativo para a instância atual do SQL Server.



| Entrada | Valor |
|-------------------|------------------------------------------|
| CN                | MS-SQL-InformationDirectory              |
| LDAP-Display-Name | mS-SQL-InformationDirectory              |
| Tamanho              | \-                                       |
| Privilégio de atualização  | Administrador de domínio                     |
| Frequência de atualização  | Na inicialização do sistema ou quando o AD é atualizado. |
| Attribute-Id      | 1.2.840.113556.1.4.1392                  |
| System-ID-GUID    | d0aedb2e-ccee-11d2-9993-0000f87a57d4     |
| Syntax            | [**Boolean**](s-boolean.md)             |
