# <a name="ms-ds-entry-time-to-die-attribute"></a>atributo ms-DS-entry-time-to-Die

Mantém o tempo de expiração absoluto de um objeto dinâmico no diretório.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------|
| CN                | ms-DS-entry-time-to-Die                                       |
| LDAP-Display-Name | msDS-entry-time-to-Die                                        |
| Tamanho              | \-                                                            |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                              |
| Frequência de atualização  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-ID-GUID    | e1e9bad7-c6dd-4101-a843-794cec85b038                          |
| Syntax            | [**Cadeia de caracteres (em tempo geral)**](s-string-generalized-time.md) |
