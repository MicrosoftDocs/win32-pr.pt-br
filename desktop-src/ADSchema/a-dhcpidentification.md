|-------------------|---------------------------------------------|
| CN                | Identificação de DHCP                         |
| LDAP-Display-Name | dhcpIdentification                          |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |

| Attribute-Id      | 1.2.840.113556.1.4.701                      |
