# <a name="managed-by-attribute"></a>Managed-By atributo

O nome distinto do usuário atribuído para gerenciar este objeto.



| Entrada | Valor |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Managed-By                                                                          |
| LDAP-Display-Name | managedBy                                                                           |
| Tamanho              | \-                                                                                  |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta.                                              |
| Frequência de atualização  | Quando o registro do usuário é criado e sempre que os objetos gerenciados precisam ser alterados. |
| Attribute-Id      | 1.2.840.113556.1.4.653                                                              |
| System-ID-GUID    | 0296c120-40da-11d1-a9c0-0000f80367c1                                                |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md)                                             |
