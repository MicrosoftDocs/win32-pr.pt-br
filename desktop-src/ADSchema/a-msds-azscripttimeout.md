# <a name="ms-ds-az-script-timeout-attribute"></a>atributo ms-DS-AZ-script-Timeout

O tempo máximo, em milissegundos, para aguardar um script concluir a auditoria de uma política específica.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-script-Timeout                 |
| LDAP-Display-Name | msDS-AzScriptTimeout                    |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Administrador do AzRoles                           |
| Frequência de atualização  | Durante a inicialização ou a alteração da política. |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| System-ID-GUID    | 87d0fb41-2c8b-41f6-b972-11fdfd50d6b0    |
| Syntax            | [**Enumeração**](s-enumeration.md)    |
