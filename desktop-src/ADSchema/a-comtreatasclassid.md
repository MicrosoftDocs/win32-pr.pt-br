# <a name="com-treat-as-class-id-attribute"></a>Atributo COM-tratá-as-Class-ID

Treat-As CLSID de GUID de cadeia de caracteres para a classe de host.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | COM-tratar como ID de classe                       |
| LDAP-Display-Name | cOMTreatAsClassId                           |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.251                      |
| System-ID-GUID    | 281416db-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
