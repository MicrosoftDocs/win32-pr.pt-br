# <a name="omt-indx-guid-attribute"></a>Atributo OMT-INDX-GUID

O identificador de índice para uma entrada de tabela link-Track-Object-move.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | OMT-INDX-GUID                                         |
| LDAP-Display-Name | oMTIndxGuid                                           |
| Tamanho              | 16 bytes                                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                      |
| Frequência de atualização  | Sempre que o link de um arquivo é alterado.                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| System-ID-GUID    | 1f0075fa-7e40-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
