a descrição do banco de dados SQL Server.

|-------------------|--------------------------------------------------|
| CN                | MS-SQL-descrição                               |
| LDAP-Display-Name | mS-SQL-descrição                               |
| Tamanho              | \-                                               |
| Privilégio de atualização  | Administrador de domínio                             |
| Frequência de atualização  | Quando o usuário decide alterar a descrição. |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| System-ID-GUID    | 8386603c-ccef-11d2-9993-0000f87a57d4             |
