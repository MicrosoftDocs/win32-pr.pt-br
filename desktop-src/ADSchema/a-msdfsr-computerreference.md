# <a name="ms-dfsr-computerreference-attribute"></a>atributo ms-DFSR-ComputerReference

Contém um link de encaminhamento para um objeto de [**computador**](c-computer.md) .



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | MS-DFSR-ComputerReference               |
| LDAP-Display-Name | msDFSR-ComputerReference                |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.101             |
| System-ID-GUID    | 6c7b5785-3d21-41bf-8a8a-627941544d5a    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
