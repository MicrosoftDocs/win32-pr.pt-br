# <a name="teletex-terminal-identifier-attribute"></a>Teletex-atributo de identificador de terminal

Especifica o identificador de terminal Teletex e, opcionalmente, parâmetros, para um terminal Teletex associado a um objeto.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Teletex-identificador de terminal                           |
| LDAP-Display-Name | teletexTerminalIdentifier                             |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 2.5.4.22                                              |
| System-ID-GUID    | bf967a4a-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
