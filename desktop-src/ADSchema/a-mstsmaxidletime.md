# <a name="ms-ts-max-idle-time-attribute"></a>MS-TS-Max-ociosidade-atributo de tempo

Tempo ocioso máximo da sessão dos serviços de terminal é a quantidade máxima de tempo, em minutos, que a sessão dos serviços de terminal pode permanecer ociosa.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-TS-Max-Idle-Time                  |
| LDAP-Display-Name | msTSMaxIdleTime                      |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1983              |
| System-ID-GUID    | ff739e9c-6bb7-460e-b221-e250f3de0f95 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
