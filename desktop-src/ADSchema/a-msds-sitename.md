# <a name="ms-ds-sitename-attribute"></a>atributo ms-DS-SiteName

Lista o nome do site que corresponde ao controlador de domínio.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-SiteName                              |
| LDAP-Display-Name | msDS-SiteName                               |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1961                     |
| System-ID-GUID    | 98a7f36d-3595-448a-9e6f-6b8965baed9c        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
