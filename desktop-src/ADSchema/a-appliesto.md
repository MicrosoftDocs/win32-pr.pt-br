# <a name="applies-to-attribute"></a>Applies-To atributo

Contém a lista de classes de objeto às quais o direito estendido se aplica. Na lista, uma classe de objeto é representada pela propriedade schemaIDGUID para seu objeto schemaClass.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Applies-To                                  |
| LDAP-Display-Name | appliesTo                                   |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.341                      |
| System-ID-GUID    | 8297931d-86d3-11d0-afda-00c04fd930c9        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
