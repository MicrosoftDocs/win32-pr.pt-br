# <a name="ds-install-replica-extended-right"></a>DS-install-réplica estendida à direita

Direito estendido necessário para fazer uma instalação de réplica.

|--------------|--------------------------------------|
| CN           | DS-instalar-réplica                   |
| Display-Name | Adicionar/remover réplica no domínio         |
| GUID de direitos  | 9923a32a-3607-11d2-b9be-0000f87a36b2 |
