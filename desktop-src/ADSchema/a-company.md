# <a name="company-attribute"></a>Atributo da empresa

O nome da empresa do usuário.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Empresa                                                                          |
| LDAP-Display-Name | company                                                                          |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Qualquer pessoa pode atualizar esse objeto com base na segurança do objeto que está sendo criado. |
| Frequência de atualização  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.2.146                                                           |
| System-ID-GUID    | f0f8ff88-1191-11d0-a060-00aa006c33ed                                             |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                      |
