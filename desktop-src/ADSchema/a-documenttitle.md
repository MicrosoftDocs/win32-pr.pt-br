# <a name="documenttitle-attribute"></a>atributo documentTitle

O tipo de atributo documentTitle especifica o título de um documento.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | documentTitle                               |
| LDAP-Display-Name | documentTitle                               |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.12                  |
| System-ID-GUID    | de265a9c-ff2c-47b9-91dc-6e6fe2c43062        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
