# <a name="domain-cross-ref-attribute"></a>Atributo de referência cruzada de domínio

Essa é uma referência de um objeto de domínio confiável para o objeto de referência cruzada do domínio confiável.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Domínio-Cruz-referência                        |
| LDAP-Display-Name | domainCrossRef                          |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.472                  |
| System-ID-GUID    | b000ea7b-a086-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
