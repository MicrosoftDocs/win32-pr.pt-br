# <a name="frs-root-security-attribute"></a>FRS-raiz-atributo de segurança

O descritor de segurança da raiz do conjunto de réplicas para replicação de arquivo.



| Entrada | Valor |
|-------------------|-----------------------------------------------------|
| CN                | FRS-raiz-segurança                                   |
| LDAP-Display-Name | fRSRootSecurity                                     |
| Tamanho              | \-                                                  |
| Privilégio de atualização  | \-                                                  |
| Frequência de atualização  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| System-ID-GUID    | 5245801f-ca6a-11d0-afff-0000f80367c1                |
| Sintaxe            | [**Cadeia de caracteres (NT-SEC-DESC)**](s-string-nt-sec-desc.md) |
