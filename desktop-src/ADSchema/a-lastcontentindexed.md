# <a name="last-content-indexed-attribute"></a>Atributo indexado do último conteúdo

A hora em que este volume foi indexado pelo último conteúdo.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Último conteúdo indexado                 |
| LDAP-Display-Name | lastContentIndexed                   |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.50                |
| System-ID-GUID    | bf967995-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
