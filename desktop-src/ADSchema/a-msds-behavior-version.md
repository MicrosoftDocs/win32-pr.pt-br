# <a name="ms-ds-behavior-version-attribute"></a>Atributo ms-DS-Behavior-Version

Esse atributo é usado para acompanhar a versão de comportamento do domínio ou da floresta. É um número de aumento monotônica que é usado para habilitar determinados recursos do Active Directory.



| Entrada | Valor |
|-------------------|------------------------------------------------|
| CN                | ms-DS-Behavior-Version                         |
| Ldap-Display-Name | msDS-Behavior-Version                          |
| Tamanho              | \-                                             |
| Privilégio de atualização  | Esse valor é definido pelo Administrador de Domínio. |
| Frequência de atualização  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1459                        |
| System-Id-Guid    | d31a8757-2447-4545-8081-3bb610cacbf2           |
| Sintaxe            | [**Enumeração**](s-enumeration.md)           |
