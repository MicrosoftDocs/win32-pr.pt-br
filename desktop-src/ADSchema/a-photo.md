# <a name="photo-attribute"></a>atributo de foto

Um objeto codificado no fax G3, conforme explicado na recomendação T. 4, com um wrapper ASN. 1 para torná-lo compatível com um X. 400 BodyPart conforme definido em X. 420.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | foto                                                 |
| LDAP-Display-Name | foto                                                 |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| System-ID-GUID    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
