# <a name="localization-display-id-attribute"></a>Localização-exibição-atributo de ID

Usado para indexar no arquivo Extrts.mc para obter o displayName localizado para os objetos para fins de interface do usuário.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Localização-exibição-ID              |
| LDAP-Display-Name | localizationDisplayId                |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | Administrador de esquema                 |
| Frequência de atualização  | Uma ID nunca deve ser alterada.       |
| Attribute-Id      | 1.2.840.113556.1.4.1353              |
| System-ID-GUID    | a746f0d1-78d0-11d2-9916-0000f87a57d4 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
