# <a name="preferred-ou-attribute"></a>Atributo de UO preferencial

A unidade organizacional a ser mostrada por padrão na área de trabalho do usuário.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Preferencial-UO                            |
| LDAP-Display-Name | preferredOU                             |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.97                   |
| System-ID-GUID    | bf9679ff-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
