# <a name="netboot-new-machine-naming-policy-attribute"></a>netboot-novo-Machine-atributo de política de nomenclatura

Indica o esquema de nomenclatura que as contas de novo computador cliente usarão.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | netboot-novo-Machine-nome-da-política           |
| LDAP-Display-Name | netbootNewMachineNamingPolicy               |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.855                      |
| System-ID-GUID    | 0738307c-91df-11d1-aebc-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
