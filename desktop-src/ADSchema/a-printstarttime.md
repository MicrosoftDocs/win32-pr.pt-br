# <a name="print-start-time-attribute"></a>Imprimir-atributo de hora de início

O tempo que uma fila de impressão começa a atender aos trabalhos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Imprimir-hora de início                     |
| LDAP-Display-Name | imstart                       |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.233               |
| System-ID-GUID    | 281416c9-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
