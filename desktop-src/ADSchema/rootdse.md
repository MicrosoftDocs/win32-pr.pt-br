# <a name="rootdse-ad-schema"></a>RootDSE (esquema do AD)

No LDAP 3,0, o rootDSE é definido como a raiz da árvore de dados do diretório em um servidor de diretório. O rootDSE não faz parte de nenhum namespace. A finalidade do rootDSE é fornecer dados sobre o servidor de diretório. Para obter mais informações sobre o rootDSE, consulte [associação sem servidor e RootDSE](/windows/desktop/AD/serverless-binding-and-rootdse) na documentação do SDK do Active Directory.

rootDSE contém os seguintes atributos. Todos os atributos são de valor único, salvo indicação em contrário.



| Atributo                                    | Sintaxe                                                 | Descrição                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **configurationNamingContext**<br/>    | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o nome distinto para o contêiner de configuração.<br/>                                                                                                                                                                                                                                                                                                                                            |
| **currentTime**<br/>                   | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém a hora atual definida neste servidor de diretório no formato de tempo universal coordenado.<br/>                                                                                                                                                                                                                                                                                                                |
| **defaultNamingContext**<br/>          | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o nome distinto para o domínio do qual esse servidor de diretório é membro.<br/>                                                                                                                                                                                                                                                                                                                  |
| **dnsHostName**<br/>                   | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o endereço DNS para este servidor de diretório.<br/>                                                                                                                                                                                                                                                                                                                                                         |
| **domainControllerFunctionality**<br/> | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Indica o nível funcional do controlador de domínio. Isso pode ser um dos valores a seguir.<br/> "0"-modo do Windows 2000<br/> "2"-modo do Windows Server 2003<br/> "3"-modo do Windows Server 2008<br/>                                                                                                                                                                                    |
| **domainFunctionality**<br/>           | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Indica o nível funcional do domínio. Isso pode ser um dos valores a seguir.<br/> "0"-modo de domínio do Windows 2000<br/> "1"-modo de domínio provisório do Windows Server 2003<br/> "2"-modo de domínio do Windows Server 2003<br/> "3"-modo de domínio do Windows Server 2008<br/> "4"-modo de domínio do Windows Server 2008 R2<br/>                                                             |
| **dsServiceName**<br/>                 | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o nome distinto do objeto de configurações NTDS para este servidor de diretório.<br/>                                                                                                                                                                                                                                                                                                                      |
| **forestFunctionality**<br/>           | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Indica o nível funcional da floresta. Isso pode ser um dos valores a seguir.<br/> "0"-modo de floresta do Windows 2000<br/> "1"-modo de floresta provisória do Windows Server 2003<br/> "2"-modo de floresta do Windows Server 2003<br/> "3"-modo de floresta do Windows Server 2008<br/> "4"-modo de floresta do Windows Server 2008 R2<br/>                                                             |
| **highestCommittedUSN**<br/>           | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o USN (número de sequência de atualização) mais alto neste servidor de diretório. Usado pela replicação de diretório.<br/>                                                                                                                                                                                                                                                                                                  |
| **isGlobalCatalogReady**<br/>          | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Indica se o catálogo global está totalmente operacional. Contém "TRUE" ou "FALSE".<br/>                                                                                                                                                                                                                                                                                                                    |
| **IsSynchronized**<br/>                | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Indica se o servidor de diretório está totalmente sincronizado. Contém "TRUE" ou "FALSE".<br/>                                                                                                                                                                                                                                                                                                                 |
| **ldapServiceName**<br/>               | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o SPN (nome da entidade de serviço) do servidor LDAP. Usado para autenticação mútua.<br/>                                                                                                                                                                                                                                                                                                              |
| **namingContexts**<br/>                | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Um atributo de valores múltiplos que contém os nomes distintos para todos os contextos de nomenclatura armazenados nesse servidor de diretório. Por padrão, um controlador de domínio do Windows 2000 contém pelo menos três contextos de nomenclatura: esquema, configuração e outro para o domínio do qual o servidor é membro.<br/>                                                                                                             |
| **rootDomainNamingContext**<br/>       | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o nome distinto para o primeiro domínio na floresta que contém o domínio do qual esse servidor de diretório é membro.<br/>                                                                                                                                                                                                                                                                     |
| **schemaNamingContext**<br/>           | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o nome distinto para o contêiner de esquema.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| **serverName**<br/>                    | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o nome distinto para o objeto de servidor para esse servidor de diretório no contêiner de configuração.<br/>                                                                                                                                                                                                                                                                                             |
| **subschemaSubentry**<br/>             | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém o nome distinto para o objeto de [**subesquema**](c-subschema.md) . O objeto de **subesquema** contém propriedades que expõem os atributos com suporte (na propriedade [**attributetypes**](a-attributetypes.md) ) e classes (na propriedade [**ObjectClasses**](a-objectclasses.md) ).<br/> A propriedade e o subesquema **subschemaSubentry** são definidos no LDAP 3,0 (consulte RFC 2251).<br/> |
| **supportedCapabilities**<br/>         | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Um atributo de valores múltiplos que contém os recursos com suporte neste servidor de diretório.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedControl**<br/>              | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Um atributo de valores múltiplos que contém os OIDs para controles de extensão com suporte neste servidor de diretório. Consulte a tabela abaixo para obter uma lista dos possíveis OIDs de controle.<br/>                                                                                                                                                                                                                                  |
| **supportedLDAPPolicies**<br/>         | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Um atributo de valores múltiplos que contém os nomes das políticas de gerenciamento de LDAP com suporte.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedLDAPVersion**<br/>          | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Um atributo de valores múltiplos que contém as versões LDAP (especificado pelo número de versão principal) com suporte neste servidor de diretório.<br/>                                                                                                                                                                                                                                                                         |
| **supportedSASLMechanisms**<br/>       | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md)<br/> | Contém os mecanismos de segurança com suporte para negociação SASL (consulte RFCs LDAP). Por padrão, o GSSAPI tem suporte.<br/>                                                                                                                                                                                                                                                                                           |



 

Active Directory dá suporte aos OIDs de controle a seguir no atributo **supportedControl** . Para obter mais informações, consulte [**LDAPControl**](/previous-versions/windows/desktop/api/winldap/ns-winldap-ldapcontrola) and [**LDAP \_ Search \_ init \_ Page**](/previous-versions/windows/desktop/api/winldap/nf-winldap-ldap_search_init_page).
