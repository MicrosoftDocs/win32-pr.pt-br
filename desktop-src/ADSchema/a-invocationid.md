# <a name="invocation-id-attribute"></a>Invocation-Id atributo

Usado para identificar exclusivamente cada diretório do Microsoft Exchange Server na organização.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Invocation-Id                                         |
| LDAP-Display-Name | invocationId                                          |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | Quando o Exchange Server está instalado.                |
| Attribute-Id      | 1.2.840.113556.1.2.115                                |
| System-ID-GUID    | bf96798e-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
