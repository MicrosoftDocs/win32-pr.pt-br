# <a name="ms-ds-filter-containers-attribute"></a>atributo ms-DS-Filter-containers

Uma cadeia de caracteres com valores múltiplos que contém os nomes das classes usadas para determinar quais tipos de contêiner devem ser mostrados pelo snap-in Active Directory usuários e computadores ao filtrar.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Filter-containers                     |
| LDAP-Display-Name | msDS-FilterContainers                       |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | Quando a lista de tipos de contêiner é alterada.   |
| Attribute-Id      | 1.2.840.113556.1.4.1703                     |
| System-ID-GUID    | fb00dcdf-ac37-483a-9c12-ac53a6603033        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
