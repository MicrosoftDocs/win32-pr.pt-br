# <a name="system-poss-superiors-attribute"></a>Atributo System-Poss-Superiors

A lista de classes que podem conter essa classe. Essa lista só pode ser modificada pelo sistema.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------------------|
| CN                | System-Poss-Superiors                                                       |
| Ldap-Display-Name | systemPossSuperiors                                                         |
| Tamanho              | \-                                                                          |
| Privilégio de atualização  | Administrador de esquema                                                        |
| Frequência de atualização  | Quando a classe é criada ou um novo superior possível é adicionado à classe . |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-Id-Guid    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)             |
