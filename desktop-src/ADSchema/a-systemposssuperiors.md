# <a name="system-poss-superiors-attribute"></a>System-poss-superior atributo

A lista de classes que podem conter esta classe. Essa lista só pode ser modificada pelo sistema.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------------------|
| CN                | Sistema-poss-superiors                                                       |
| LDAP-Display-Name | systemPossSuperiors                                                         |
| Tamanho              | \-                                                                          |
| Privilégio de atualização  | Administrador de esquema                                                        |
| Frequência de atualização  | Quando a classe é criada ou um novo superior possível é adicionado à classe. |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-ID-GUID    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**Cadeia de caracteres (identificador de objeto)**](s-string-object-identifier.md)             |
