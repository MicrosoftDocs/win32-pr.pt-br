# <a name="ms-sql-lastbackupdate-attribute"></a>Atributo MS-SQL-LastBackupDate

A última data em que foi feito o backup do banco de dados.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastBackupDate                       |
| LDAP-Display-Name | mS-SQL-LastBackupDate                       |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | Quando é feito backup do banco de dados.             |
| Attribute-Id      | 1.2.840.113556.1.4.1398                     |
| System-ID-GUID    | f2b6abca-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
