# <a name="initial-auth-incoming-attribute"></a>Atributo Initial-auth-Outcoming

Contém informações sobre uma solicitação de autenticação de entrada inicial por um cliente para este servidor. Essa solicitação é enviada por esse servidor para o servidor de autenticação para o domínio.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Inicial-autenticação-entrada                       |
| LDAP-Display-Name | initialAuthIncoming                         |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.539                      |
| System-ID-GUID    | 52458023-ca6a-11D0-afff-0000f80367c1        |
| Sintaxe            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
