# <a name="retired-repl-dsa-signatures-attribute"></a>Atributo removida-repl-DSA-Signatures

Controla as identidades de replicação DS anteriores de um determinado DC.



| Entrada | Valor |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Desativado-repl-DSA-Signatures                                                         |
| LDAP-Display-Name | retiredReplDSASignatures                                                            |
| Tamanho              | O comprimento é proporcional ao número de vezes que o DC foi restaurado do backup. |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                                                    |
| Frequência de atualização  | \-                                                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.673                                                              |
| System-ID-GUID    | 7bfdcb7f-4807-11d1-a9c3-0000f80367c1                                                |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md)                               |
