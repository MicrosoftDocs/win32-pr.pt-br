# <a name="category-id-attribute"></a>Category-Id atributo

| LDAP-Display-Name | categoryId                                            |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.322                                |
