# <a name="acs-rsvp-account-files-location-attribute"></a>ACS-RSVP-Account-files-atributo Location

O local do diretório dos arquivos de conta RSVP. O padrão é o subdiretório "System32" localizado no diretório do sistema padrão.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ACS-RSVP-Account-files-Location             |
| LDAP-Display-Name | aCSRSVPAccountFilesLocation                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.900                      |
| System-ID-GUID    | f072230f-aef5-11d1-bdcf-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
