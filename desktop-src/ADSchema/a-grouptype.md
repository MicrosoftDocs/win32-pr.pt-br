# <a name="group-type-attribute"></a>Group-Type atributo

Contém um conjunto de sinalizadores que definem o tipo e o escopo de um objeto de grupo. Para obter os valores possíveis para esse atributo, consulte comentários.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Group-Type                           |
| LDAP-Display-Name | groupType                            |
| Tamanho              | \-                                   |
| Privilégio de atualização  | Administrador de domínio                 |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.750               |
| System-ID-GUID    | 9a9a021e-4a5b-11d1-a9c3-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |

## <a name="remarks"></a>Comentários

Esse atributo pode ser zero ou uma combinação de um ou mais dos valores a seguir.



| Valor                   | Descrição                                                                                  |
|-------------------------|----------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Especifica um grupo que é criado pelo sistema.                                             |
| 2 (0x00000002)          | Especifica um grupo com escopo global.                                                         |
| 4 (0x00000004)          | Especifica um grupo com escopo de domínio local.                                                   |
| 8 (0x00000008)          | Especifica um grupo com escopo universal.                                                      |
| 16 (0x00000010)         | Especifica um \_ grupo básico do aplicativo para o Gerenciador de autorização do Windows Server.                      |
| 32 (0x00000020)         | Especifica um \_ grupo de consulta de aplicativo para o Gerenciador de autorização do Windows Server.                      |
| 2147483648 (0x80000000) | Especifica um grupo de segurança. Se esse sinalizador não for definido, o grupo será um grupo de distribuição. |



 

Para obter mais informações sobre o tipo de grupo e o escopo, consulte os tópicos [tipos](/previous-versions/windows/it-pro/windows-server-2003/cc781446(v=ws.10)) de grupo e [escopo de grupo](/previous-versions/windows/it-pro/windows-server-2003/cc755692(v=ws.10)) no Microsoft TechNet.
