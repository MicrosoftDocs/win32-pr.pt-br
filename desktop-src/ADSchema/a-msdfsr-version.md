# <a name="ms-dfsr-version-attribute"></a>atributo ms-DFSR-Version

Contém o número de versão do serviço de replicação do Sistema de Arquivos Distribuído (DFS).



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-versão                             |
| LDAP-Display-Name | msDFSR-versão                              |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.1                   |
| System-ID-GUID    | 1a861408-38c3-49ea-ba75-85481a77c655        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
