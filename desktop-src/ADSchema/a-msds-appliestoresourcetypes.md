# <a name="ms-ds-applies-to-resource-types-attribute"></a>atributo ms-DS-aplica-se ao tipo de recurso

Para uma propriedade de recurso, esse atributo indica a quais tipos de recursos essa propriedade de recurso se aplica.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-aplica-se a tipos de recurso             |
| LDAP-Display-Name | msDS-AppliesToResourceTypes                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2195                     |
| System-ID-GUID    | 693f2006-5764-3d4a-8439-58f04aab4b59        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
