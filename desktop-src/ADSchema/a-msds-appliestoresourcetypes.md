# <a name="ms-ds-applies-to-resource-types-attribute"></a>Atributo ms-DS-Applies-To-Resource-Types

Para uma propriedade de recurso, esse atributo indica a quais tipos de recurso essa propriedade de recurso se aplica.

| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
