# <a name="address-book-roots2-attribute"></a>Atributo de catálogo de endereços-Roots2

Usado pelo Exchange. Exchange configura árvores de contêineres de catálogo de endereços para aparecerem no catálogo de endereços MAPI. esse atributo no objeto de configuração Exchange lista as raízes das árvores de contêiner do catálogo de endereços.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Catálogo de endereços-Roots2                     |
| LDAP-Display-Name | addressBookRoots2                       |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2046                 |
| System-ID-GUID    | 508ca374-a511-4e4e-9f4f-856f61a6b7e4    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
