# <a name="address-book-roots2-attribute"></a>Atributo Address-Book-Roots2

Usado por Exchange. Exchange configura árvores de contêineres do livro de endereços para aparecer no livro de endereços MAPI. Esse atributo no objeto Exchange Config lista as raízes das árvores de contêiner do livro de endereços.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Address-Book-Roots2                     |
| Ldap-Display-Name | addressBookRoots2                       |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2046                 |
| System-Id-Guid    | 508ca374-a511-4e4e-9f4f-856f61a6b7e4    |
| Sintaxe            | [**Object(DS-DN)**](s-object-ds-dn.md) |
