# <a name="print-keep-printed-jobs-attribute"></a>Imprimir-atributo de impressão-manter-impresso

**True** se os trabalhos impressos forem mantidos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Imprimir-manter-impresso-trabalhos              |
| LDAP-Display-Name | printKeepPrintedJobs                 |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.275               |
| System-ID-GUID    | ba305f6d-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Boolean**](s-boolean.md)         |
