# <a name="ms-ds-object-reference-attribute"></a>atributo de referência ms-DS-Object

Um link para o objeto que usa os dados armazenados no objeto que contém esse atributo.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Object-Reference                  |
| LDAP-Display-Name | msDS-ObjectReference                    |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1840                 |
| System-ID-GUID    | 638ec2e8-22e7-409c-85d2-11b21bee72de    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
