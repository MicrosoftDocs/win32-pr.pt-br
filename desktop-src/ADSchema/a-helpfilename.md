# <a name="help-file-name-attribute"></a>Ajuda-atributo de nome de arquivo

Esse atributo foi usado para o Exchange 4,0. Ele continha o nome que deve ser usado para o arquivo quando o provedor baixasse os dados de ajuda para um computador cliente. Ele não é usado para nenhuma outra versão do Exchange.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Ajuda-nome do arquivo                              |
| LDAP-Display-Name | Arquivodeajudaname                                |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Isso é usado pelo sistema.                 |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.327                      |
| System-ID-GUID    | 5fd424a9-1262-11d0-a060-00aa006c33ed        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
