# <a name="force-logoff-attribute"></a>Force-Logoff atributo

Usado na computação do tempo de lançamento no SamIGetAccountRestrictions. Tempo de logoff menos forçar logoff é igual a tempo de lançamento.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Force-Logoff                         |
| LDAP-Display-Name | forceLogoff                          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.39                |
| System-ID-GUID    | bf967977-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
