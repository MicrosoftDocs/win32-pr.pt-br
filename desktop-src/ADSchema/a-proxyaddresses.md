# <a name="proxy-addresses-attribute"></a>Proxy-Addresses atributo

Um endereço de proxy é o endereço pelo qual um objeto de destinatário do Microsoft Exchange Server é reconhecido em um sistema de email externo. Os endereços proxy são necessários para todos os objetos de destinatário, como destinatários personalizados e listas de distribuição.



| Entrada | Valor |
|-------------------|------------------------------------------------------------------------------------------------------|
| CN                | Proxy-Addresses                                                                                      |
| LDAP-Display-Name | proxyAddresses                                                                                       |
| Tamanho              | \-                                                                                                   |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                                                                     |
| Frequência de atualização  | Criado por uma DLL fornecida com o objeto Address-Type Directory quando o tipo de endereço é instalado. |
| Attribute-Id      | 1.2.840.113556.1.2.210                                                                               |
| System-ID-GUID    | bf967a06-0de6-11d0-a285-00aa003049e2                                                                 |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                                          |
