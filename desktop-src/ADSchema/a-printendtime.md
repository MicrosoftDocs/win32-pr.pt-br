# <a name="print-end-time-attribute"></a>Imprimir-atributo de hora de término

O tempo que uma fila de impressão interrompe os trabalhos de manutenção.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Impressão – hora de término                       |
| LDAP-Display-Name | imendtime                         |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.234               |
| System-ID-GUID    | 281416ca-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
