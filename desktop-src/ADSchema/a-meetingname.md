# <a name="meetingname-attribute"></a>atributo meetingname

O nome da reunião.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | reuniãoname                                                                      |
| LDAP-Display-Name | reuniãoname                                                                      |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Qualquer pessoa pode atualizar esse objeto com base na segurança do objeto que está sendo criado. |
