# <a name="default-hiding-value-attribute"></a>Padrão-Ocultar atributo-Value

Um valor booliano que especifica a configuração padrão da propriedade showInAdvancedViewOnly de novas instâncias dessa classe.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Padrão-ocultando valor                 |
| LDAP-Display-Name | defaultocultarvalue                   |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.518               |
| System-ID-GUID    | b7b13116-b82e-11d0-afee-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
