# <a name="lsa-modified-count-attribute"></a>LSA-modificado-atributo de contagem

O atributo **LSA-Modified-Count** é usado para dar suporte à replicação em domínios do Windows NT 4,0.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | LSA-modificado-contagem                   |
| LDAP-Display-Name | lSAModifiedCount                     |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.67                |
| System-ID-GUID    | bf9679ae-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
