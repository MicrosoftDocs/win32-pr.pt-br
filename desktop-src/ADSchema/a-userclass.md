# <a name="userclass-attribute"></a>atributo userclass

O tipo de atributo userclass especifica uma categoria de usuário do computador.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | userclass                                   |
| LDAP-Display-Name | userclass                                   |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.8                   |
| System-ID-GUID    | 11732a8a-e14d-4cc5-b92f-d93f51c6d8e4        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
