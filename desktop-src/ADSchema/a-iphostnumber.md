# <a name="iphostnumber-attribute"></a>atributo ipHostNumber

Contém o endereço IP do host na notação decimal pontilhada, omitindo os zeros à esquerda.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ipHostNumber                         |
| LDAP-Display-Name | ipHostNumber                         |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.19                     |
| System-ID-GUID    | de8bb721-85dc-4fde-b687-9657688e667e |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
