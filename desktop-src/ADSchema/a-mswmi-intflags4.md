# <a name="ms-wmi-intflags4-attribute"></a>atributo ms-WMI-intFlags4

O atributo **MS-WMI-intFlags4** é reservado para uso interno.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-WMI-intFlags4                     |
| LDAP-Display-Name | msWMI-intFlags4                      |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1681              |
| System-ID-GUID    | bd74a7ac-c493-4c9c-bdfa-5c7b119ca6b2 |
| Sintaxe            | [**Enumeração**](s-enumeration.md) |
