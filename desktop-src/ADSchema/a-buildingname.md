# <a name="buildingname-attribute"></a>atributo buildName

O nome da compilação em que uma organização ou unidade organizacional é baseada.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | buildName                                |
| LDAP-Display-Name | buildName                                |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.48                  |
| System-ID-GUID    | f87fa54b-b2c5-4fd7-88c0-daccb21d93c5        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
