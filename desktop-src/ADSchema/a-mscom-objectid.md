# <a name="ms-com-objectid-attribute"></a>atributo ms-COM-ObjectId

GUID usado para identificar uma partição COM+ ou um Gptset COM+.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | MS-COM-ObjectId                                       |
| LDAP-Display-Name | msCOM-ObjectId                                        |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1428                               |
| System-ID-GUID    | 430f678b-889f-41f2-9843-203b5a65572f                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
