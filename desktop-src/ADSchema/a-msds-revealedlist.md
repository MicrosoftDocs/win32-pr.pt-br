# <a name="ms-ds-revealed-list-attribute"></a>atributo ms-DS-revelate-List

Identifica entidades de segurança cujas senhas de conta do computador atual foram replicadas para o RODC.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | ms-DS-revelado-lista                                   |
| LDAP-Display-Name | msDS-revelalist                                     |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1940                               |
| System-ID-GUID    | cbdad11c-7fec-387b-6219-3a0627d9af81                  |
| Syntax            | [**Objeto (ponto de acesso)**](s-object-access-point.md) |
