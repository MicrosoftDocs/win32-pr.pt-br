# <a name="ms-ts-default-to-main-printer-attribute"></a>MS-TS-default-to-Main-atributo Printer

O padrão dos serviços de terminal para a impressora principal especifica se a impressão deve ser impressa automaticamente na impressora padrão do cliente.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-TS-default-to-principal-Printer        |
| LDAP-Display-Name | msTSDefaultToMainPrinter             |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1988              |
| System-ID-GUID    | c0ffe2bd-cacf-4dc7-88d5-61e9e95766f6 |
| Syntax            | [**Boolean**](s-boolean.md)         |
