# <a name="ms-sql-size-attribute"></a>Atributo MS-SQL-size

O tamanho, em quilobytes, do banco de dados.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-SQL-size                          |
| LDAP-Display-Name | mS-SQL-size                          |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Quando o banco de dados muda de tamanho.      |
| Attribute-Id      | 1.2.840.113556.1.4.1396              |
| System-ID-GUID    | e9098084-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Intervalo**](s-interval.md)       |
