# <a name="next-level-store-attribute"></a>Atributo de repositório de próximo nível

Esse atributo indica o próximo repositório de classe a ser pesquisado.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Armazenamento de nível mais próximo                        |
| LDAP-Display-Name | nextLevelStore                          |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.214                  |
| System-ID-GUID    | bf9679da-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
