# <a name="run-protect-admin-groups-task-extended-right"></a>Execute-Protect-admin-groups-Task Extended Right

Controlar direitos de acesso autorizando operações de modificação de rootDSE em um PDC FSMO para disparar a operação de proteção do AdminSDHolder.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Executar-Protect-admin-groups-Task        |
| Display-Name | Executar tarefa proteger grupos de administradores        |
| GUID de direitos  | 7726b9d5-a4b4-4288-a6b2-dce952e80a7f |
