# <a name="run-protect-admin-groups-task-extended-right"></a>Direito estendido Run-Protect-Admin-Groups-Task

Controle os direitos de acesso que autorizam operações de modificação rootDSE em um FSMO do PDC para disparar a operação de proteção AdminSDHolder.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Run-Protect-Admin-Groups-Task        |
| Display-Name | Tarefa Executar Proteger Grupos de Administradores        |
| RIGHTS-GUID  | 7726b9d5-a4b4-4288-a6b2-dce952e80a7f |
