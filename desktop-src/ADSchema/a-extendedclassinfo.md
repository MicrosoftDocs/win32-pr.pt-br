# <a name="extended-class-info-attribute"></a>Atributo de informações de classe estendida

Uma propriedade com valores múltiplos que contém cadeias de caracteres que representam informações adicionais para cada classe. Cada valor contém o governsID, lDAPDisplayName e schemaIDGUID da classe.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Informações de classe estendida                         |
| LDAP-Display-Name | extendedClassInfo                           |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.908                      |
| System-ID-GUID    | 9a7ad948-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
