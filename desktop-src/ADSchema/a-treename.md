# <a name="tree-name-attribute"></a>Tree-Name atributo

Nome DNS do domínio na raiz de uma árvore.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Tree-Name                                   |
| LDAP-Display-Name | treename                                    |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.660                      |
| System-ID-GUID    | 28630ebd-41d5-11d1-a9c1-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
