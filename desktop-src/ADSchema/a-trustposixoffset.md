# <a name="trust-posix-offset-attribute"></a>Atributo Trust-POSIX-offset

O deslocamento POSIX (interface do sistema operacional portátil) para o domínio confiável.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Trust-POSIX-offset                   |
| LDAP-Display-Name | trustPosixOffset                     |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Quando uma nova relação de confiança é criada.         |
| Attribute-Id      | 1.2.840.113556.1.4.134               |
| System-ID-GUID    | bf967a5e-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
