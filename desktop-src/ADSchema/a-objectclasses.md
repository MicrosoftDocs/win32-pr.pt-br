# <a name="object-classes-attribute"></a>Object-Classes atributo

Uma propriedade de valores múltiplos que contém cadeias de caracteres que representam cada classe no esquema. Cada valor contém governsID, lDAPDisplayName, mustContain, mayContain e assim por diante.



| Entrada | Valor |
|-------------------|--------------------------------------------------|
| CN                | Object-Classes                                   |
| LDAP-Display-Name | objectclasss                                    |
| Tamanho              | Cerca de 20 bytes em média por nome de classe.        |
| Privilégio de atualização  | O designer do objeto definiria esse valor. |
| Frequência de atualização  | Sempre que o design da classe é alterado.        |
| Attribute-Id      | 2.5.21.6                                         |
| System-ID-GUID    | 9a7ad94b-ca53-11d1-bbd0-0080c76670c0             |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)      |
