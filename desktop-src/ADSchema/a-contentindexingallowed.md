# <a name="content-indexing-allowed-attribute"></a>Indexação de conteúdo-atributo permitido

Indica se o objeto de volume pode ser indexado com conteúdo.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Indexação de conteúdo-permitida             |
| LDAP-Display-Name | contentIndexingAllowed               |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.24                |
| System-ID-GUID    | bf967943-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
