# <a name="com-clsid-attribute"></a>Atributo COM-CLSID

Esse atributo contém o GUID associado a esta classe de objeto.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | COM-CLSID                                   |
| LDAP-Display-Name | comclsid                                    |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.249                      |
| System-ID-GUID    | 281416d9-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
