# <a name="ipnetworknumber-attribute"></a>atributo ipNetworkNumber

Contém um número de rede IP na notação decimal pontilhada, omitindo os zeros à esquerda.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ipNetworkNumber                      |
| LDAP-Display-Name | ipNetworkNumber                      |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| System-ID-GUID    | 4e3854f4-3087-42a4-a813-bb0c528958d3 |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
