# <a name="ms-wmi-id-attribute"></a>atributo ms-WMI-ID

Uma ID exclusiva para uma instância de objeto. Indexado no catálogo global.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-ID                                   |
| LDAP-Display-Name | msWMI-ID                                    |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1627                     |
| System-ID-GUID    | 9339a803-94b8-47f7-9123-a853b9ff7e45        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
