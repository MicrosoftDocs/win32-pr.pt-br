# <a name="netboot-new-machine-ou-attribute"></a>netboot – atributo New-Machine-UO

Indica onde a nova conta de computador cliente será criada.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | netboot – nova máquina-UO                  |
| LDAP-Display-Name | netbootNewMachineOU                     |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Esse valor é definido pelo sistema.        |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.856                  |
| System-ID-GUID    | 0738307d-91df-11d1-aebc-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
