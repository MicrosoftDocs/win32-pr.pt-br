# <a name="pki-default-csps-attribute"></a>Atributo PKI-default-CSPs

A lista de provedores de serviços de criptografia para o modelo de certificado.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | PKI-default-CSPs                            |
| LDAP-Display-Name | pKIDefaultCSPs                              |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1334                     |
| System-ID-GUID    | 1ef6336e-3b9e-11d2-90cc-00c04fd91ab1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
