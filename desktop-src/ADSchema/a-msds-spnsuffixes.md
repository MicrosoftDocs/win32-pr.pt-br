# <a name="ms-ds-spn-suffixes-attribute"></a>atributo ms-DS-SPN-Suffixs

Esse atributo descreve os sufixos dos nomes de host DNS usados pelos servidores na floresta. Esses sufixos DNS serão compartilhados com outras florestas que têm confiança entre florestas com essa floresta.



| Entrada | Valor |
|-------------------|----------------------------------------------|
| CN                | ms-DS-SPN-sufixos                           |
| LDAP-Display-Name | msDS-SPNSuffixes                             |
| Tamanho              | 255 bytes                                    |
| Privilégio de atualização  | Administrador de domínio                         |
| Frequência de atualização  | Quando os servidores na floresta obtiverem um novo sufixo. |
| Attribute-Id      | 1.2.840.113556.1.4.1715                      |
| System-ID-GUID    | 789ee1eb-8c8e-4e4c-8cec-79b31b7617b5         |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)  |
