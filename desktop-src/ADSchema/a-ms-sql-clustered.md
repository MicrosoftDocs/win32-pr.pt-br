# <a name="ms-sql-clustered-attribute"></a>Atributo MS-SQL-Clustered

True se o servidor estiver clusterizado.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-SQL-clusterizado                     |
| LDAP-Display-Name | mS-SQL-clusterizado                     |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Na inicialização do sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1373              |
| System-ID-GUID    | 7778bd90-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Boolean**](s-boolean.md)         |
