# <a name="msi-file-list-attribute"></a>MSI-atributo de lista de arquivos

Esse atributo contém uma lista de arquivos do Microsoft Installer, como o arquivo MSI base (. msi) e arquivos de transformação MST (. MST).



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MSI-lista de arquivos                               |
| LDAP-Display-Name | msiFileList                                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.671                      |
| System-ID-GUID    | 7bfdcb7d-4807-11d1-a9c3-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
