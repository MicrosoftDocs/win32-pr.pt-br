# <a name="owner-attribute"></a>Atributo de proprietário

O nome distinto de um objeto que tem a propriedade de um objeto.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Proprietário                                   |
| LDAP-Display-Name | proprietário                                   |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 2.5.4.32                                |
| System-ID-GUID    | bf9679f3-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
