# <a name="short-server-name-attribute"></a>Atributo de nome de servidor curto

Nome do servidor compatível com anterior ao Windows 2000 para servidores de impressão.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Nome de servidor curto                           |
| LDAP-Display-Name | shortServerName                             |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1209                     |
| System-ID-GUID    | 45b01501-c419-11d1-bbc9-0080c76670c0        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
