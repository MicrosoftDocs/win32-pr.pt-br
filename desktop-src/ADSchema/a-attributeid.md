# <a name="attribute-id-attribute"></a>Atributo de ID de atributo

O OID X. 500 exclusivo para identificar um atributo.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------|
| CN                | ID do atributo                                                    |
| LDAP-Display-Name | attributeID                                                     |
| Tamanho              | \-                                                              |
| Privilégio de atualização  | Administrador de esquema                                            |
| Frequência de atualização  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.30                                           |
| System-ID-GUID    | bf967922-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**Cadeia de caracteres (identificador de objeto)**](s-string-object-identifier.md) |
