# <a name="flat-name-attribute"></a>Flat-Name atributo

Para domínios do Windows NT, o nome simples é o nome NetBIOS. Para links com domínios não Windows NT, o nome simples é o nome de identificação desse domínio ou é **nulo**.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Flat-Name                                   |
| LDAP-Display-Name | flatname                                    |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.511                      |
| System-ID-GUID    | b7b13117-b82e-11d0-afee-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
