# <a name="labeleduri-attribute"></a>atributo labeledURI

Um Uniform Resource Identifier seguido por um rótulo. O rótulo é usado para descrever o recurso para o qual o URI aponta e destina-se como um nome de exibição acessível por humanos.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | labeledURI                                  |
| Ldap-Display-Name | labeledURI                                  |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.3.6.1.4.1.250.1.57                        |
| System-Id-Guid    | c569bb46-c680-44bc-a273-e6c227d71b45        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
