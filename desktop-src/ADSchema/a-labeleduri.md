# <a name="labeleduri-attribute"></a>atributo labeledURI

Um Uniform Resource Identifier seguido por um rótulo. O rótulo é usado para descrever o recurso para o qual os pontos de URI e se destina como um nome de exibição que é legível por humanos.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | labeledURI                                  |
| LDAP-Display-Name | labeledURI                                  |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.3.6.1.4.1.250.1.57                        |
| System-ID-GUID    | c569bb46-c680-44bc-a273-e6c227d71b45        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
