# <a name="print-rate-attribute"></a>Print-Rate atributo

Taxa de impressão fornecida pelo driver.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Print-Rate                           |
| LDAP-Display-Name | printRate                            |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.285               |
| System-ID-GUID    | ba305f77-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
