# <a name="ms-ds-nc-repl-inbound-neighbors-attribute"></a>ms-DS-NC-repl-atributo de entrada-vizinhos

Parceiros de replicação para esta partição. Esse servidor obtém dados de replicação desses outros servidores, que atuam como fontes.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-repl-Neighbors-Bounds             |
| LDAP-Display-Name | msDS-NCReplInboundNeighbors                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1705                     |
| System-ID-GUID    | 9edba85a-3e9e-431b-9b1a-a5b6e9eda796        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
