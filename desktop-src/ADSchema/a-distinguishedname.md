# <a name="obj-dist-name-attribute"></a>Atributo Obj-Dist-Name

O mesmo que o Nome Diferenciado de um objeto . Usado por Exchange.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Obj-Dist-Name                           |
| Ldap-Display-Name | distinguishedName                       |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Esse valor é definido pelo sistema.        |
| Frequência de atualização  | Sempre que um objeto é criado ou movido. |
| Attribute-Id      | 2.5.4.49                                |
| System-Id-Guid    | bf9679e4-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
