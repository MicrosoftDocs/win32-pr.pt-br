# <a name="acs-enable-rsvp-message-logging-attribute"></a>ACS-habilitar-RSVP-atributo de log de mensagem

True se o log RSVP estiver habilitado.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-Enable-RSVP-log de mensagens      |
| LDAP-Display-Name | aCSEnableRSVPMessageLogging          |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.768               |
| System-ID-GUID    | 7f561285-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
