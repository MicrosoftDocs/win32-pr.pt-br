# <a name="print-form-name-attribute"></a>Atributo de nome de formulário de impressão

O nome do formulário atualmente carregado.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Nome do formulário de impressão                             |
| LDAP-Display-Name | Formato de transformação                               |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.235                      |
| System-ID-GUID    | 281416cb-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
