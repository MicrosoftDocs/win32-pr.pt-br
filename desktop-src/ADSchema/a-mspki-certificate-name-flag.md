# <a name="ms-pki-certificate-name-flag-attribute"></a>MS-PKI-Certificate-Name-atributo de sinalizador

Contém os sinalizadores relacionados à construção do nome da entidade em um certificado emitido.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Certificate-Name-Flag                                                                      |
| LDAP-Display-Name | msPKI-nome-do-certificado-sinalizador                                                                       |
| Tamanho              | 4 bytes                                                                                           |
| Privilégio de atualização  | Administrador de domínio                                                                              |
| Frequência de atualização  | Quando o modelo de certificado (MS-PKI-Certificate-template) é editado, criado ou clonado. |
| Attribute-Id      | 1.2.840.113556.1.4.1432                                                                           |
| System-ID-GUID    | ea1dddc4-60ff-416e-8cc0-17cee534bce7                                                              |
| Syntax            | [**Enumeração**](s-enumeration.md)                                                              |
