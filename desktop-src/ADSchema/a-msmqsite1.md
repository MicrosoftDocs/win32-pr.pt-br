# <a name="msmq-site-1-attribute"></a>Atributo MSMQ-site-1

O DN do primeiro site de um par que está conectado.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | MSMQ-site-1                             |
| LDAP-Display-Name | mSMQSite1                               |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | Cada vez que uma linha de site do MSMQ é criada. |
| Attribute-Id      | 1.2.840.113556.1.4.943                  |
| System-ID-GUID    | 9a0dc337-c100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
