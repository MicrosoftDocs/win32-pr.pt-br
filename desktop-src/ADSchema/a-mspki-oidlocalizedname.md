# <a name="ms-pki-oid-localizedname-attribute"></a>atributo ms-PKI-OID-Traduziname

Contém uma lista de nomes de exibição usados para descrever um OID por localidade.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-PKI-OID-Traduziname                    |
| LDAP-Display-Name | msPKI-OIDLocalizedName                      |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1712                     |
| System-ID-GUID    | 7d59a816-bb05-4a72-971f-5c1331f67559        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
