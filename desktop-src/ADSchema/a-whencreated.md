# <a name="when-created-attribute"></a>When-Created atributo

A data em que este objeto foi criado. Esse valor é replicado e está no catálogo global.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------|
| CN                | When-Created                                                  |
| LDAP-Display-Name | whenCreated                                                   |
| Tamanho              | 8 bytes                                                       |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                              |
| Frequência de atualização  | Quando o objeto é criado.                                   |
| Attribute-Id      | 1.2.840.113556.1.2.2                                          |
| System-ID-GUID    | bf967a78-0de6-11d0-a285-00aa003049e2                          |
| Syntax            | [**Cadeia de caracteres (em tempo geral)**](s-string-generalized-time.md) |
