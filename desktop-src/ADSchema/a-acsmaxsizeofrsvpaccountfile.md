# <a name="acs-max-size-of-rsvp-account-file-attribute"></a>ACS-Max-Size-of-RSVP-atributo-Account-File

O tamanho máximo, em bytes, de um arquivo de conta RSVP.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Size-of-RSVP-Account-File    |
| LDAP-Display-Name | aCSMaxSizeOfRSVPAccountFile          |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.902               |
| System-ID-GUID    | f0722311-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
