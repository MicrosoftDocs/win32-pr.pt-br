# <a name="msmq-dependent-client-service-attribute"></a>Atributo de serviço de cliente dependente do MSMQ

Indica se este servidor fornece serviços de clientes dependentes.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Serviço de cliente dependente do MSMQ        |
| LDAP-Display-Name | mSMQDependentClientService           |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1239              |
| System-ID-GUID    | 2df90d83-009f-11d2-aa4c-00c04fd7d83a |
| Syntax            | [**Boolean**](s-boolean.md)         |
