# <a name="ms-com-partitionlink-attribute"></a>atributo ms-COM-PartitionLink

Um link usado para associar um particionador COM+ com suas partições COM+ de constituintes.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | MS-COM-PartitionLink                    |
| LDAP-Display-Name | msCOM-PartitionLink                     |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1423                 |
| System-ID-GUID    | 09abac62-043f-4702-ac2b-6ca15eee5754    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
