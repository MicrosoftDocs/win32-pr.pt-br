# <a name="ms-sql-allowanonymoussubscription-attribute"></a>Atributo MS-SQL-AllowAnonymousSubscription

True se a publicação permitir que assinantes anônimos se assinem.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-SQL-AllowAnonymousSubscription    |
| LDAP-Display-Name | mS-SQL-AllowAnonymousSubscription    |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Quando a replicação é configurada.           |
| Attribute-Id      | 1.2.840.113556.1.4.1394              |
| System-ID-GUID    | db77be4a-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Boolean**](s-boolean.md)         |
