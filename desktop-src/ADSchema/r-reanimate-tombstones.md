# <a name="reanimate-tombstones-extended-right"></a>Reanimate-Tombstones estendido à direita

Controle direito de acesso que permite que elementos de esquema excluídos sejam restaurados.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Reanimate-Tombstones                 |
| Display-Name | Reanimar marcas para exclusão                 |
| GUID de direitos  | 45ec5156-db7e-47bb-b53f-dbeb2d03c40f |
