# <a name="ipsec-name-attribute"></a>Ipsec-Name atributo

O atributo de **nome de IPSec** é somente para uso interno.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Ipsec-Name                                  |
| LDAP-Display-Name | ipsecname                                   |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.620                      |
| System-ID-GUID    | b40ff81c-427a-11d1-a9c2-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
