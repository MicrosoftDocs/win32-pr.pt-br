# <a name="max-renew-age-attribute"></a>Atributo Max-Renew-age

Esse atributo determina o período de tempo, em dias, durante o qual o tíquete de concessão de tíquete (TGT) de um usuário pode ser renovado para fins de autenticação Kerberos. A configuração padrão é 7 dias no objeto de Política de Grupo de domínio padrão (GPO).



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Max-Renew-age                        |
| LDAP-Display-Name | maxRenewAge                          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.75                |
| System-ID-GUID    | bf9679bc-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
