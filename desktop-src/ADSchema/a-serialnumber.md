| CN                | Serial-Number                        |
| LDAP-Display-Name | serialNumber                         |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 2.5.4.5                              |
