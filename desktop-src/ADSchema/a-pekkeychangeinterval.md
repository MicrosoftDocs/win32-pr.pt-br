# <a name="pek-key-change-interval-attribute"></a>Atributo PEK-Key-ALTER-Interval

Intervalo de alteração da chave de criptografia de senha.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | PEK-chave-alterar intervalo              |
| LDAP-Display-Name | pekKeyChangeInterval                 |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.866               |
| System-ID-GUID    | 07383084-91df-11d1-aebc-0000f80367c1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
