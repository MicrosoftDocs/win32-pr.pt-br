# <a name="stringoctet-syntax"></a>Sintaxe da cadeia de caracteres (octeto)

Uma cadeia de caracteres que representa uma matriz de bytes. Essa sintaxe é usada para armazenar dados binários.



| Entrada | Valor |
|--------------|-------------------------------------------------------------------|
| Nome         | Cadeia de caracteres (octeto)                                                     |
| ID da sintaxe    | 2.5.5.10                                                          |
| ID DO OM        | 4                                                                 |
| Tipo de MAPI    | BINARY                                                            |
| Tipo de ADS     | Cadeia de caracteres de \_ OCTETO ADSTYPE \_                                            |
| Tipo de variante | \_Matriz VT UI1 \| VT \_                                              |
| Tipo SDS     | [System.Byte\[\]](/dotnet/api/system.byte) |
