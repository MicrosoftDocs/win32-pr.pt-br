# <a name="unexpire-password-extended-right"></a>Unexpire-Password estendido à direita

Direito de acesso de controle estendido que permite que um usuário Restaure uma senha expirada para um objeto de usuário.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Unexpire-Password                    |
| Display-Name | Senha de desexpiração                    |
| GUID de direitos  | ccc2dc7d-a6ad-4a7a-8846-c04e3cc53501 |
