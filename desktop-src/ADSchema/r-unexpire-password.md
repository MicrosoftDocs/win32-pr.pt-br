# <a name="unexpire-password-extended-right"></a>Unexpire-Password direito estendido

Direito de acesso de controle estendido que permite que um usuário restaure uma senha expirada para um objeto de usuário.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Unexpire-Password                    |
| Display-Name | Nãoexpireção de senha                    |
| RIGHTS-GUID  | ccc2dc7d-a6ad-4a7a-8846-c04e3cc53501 |
