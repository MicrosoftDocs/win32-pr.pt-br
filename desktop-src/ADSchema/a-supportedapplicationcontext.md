# <a name="supported-application-context-attribute"></a>Atributo de contexto de aplicativo com suporte

Especifica os identificadores de objeto dos contextos de aplicativo aos quais um aplicativo OSI dá suporte.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Com suporte-contexto de aplicativo                         |
| LDAP-Display-Name | supportedApplicationContext                           |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 2.5.4.30                                              |
| System-ID-GUID    | 1677588f-47f3-11d1-a9c3-0000f80367c1                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
