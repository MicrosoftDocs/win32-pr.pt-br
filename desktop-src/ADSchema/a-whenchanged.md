# <a name="when-changed-attribute"></a>When-Changed atributo

A data em que esse objeto foi alterado pela última vez. Esse valor não é replicado e existe no catálogo global.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------|
| CN                | When-Changed                                                  |
| LDAP-Display-Name | whenChanged                                                   |
| Tamanho              | 8 bytes                                                       |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                              |
| Frequência de atualização  | Cada vez que o objeto é alterado.                              |
| Attribute-Id      | 1.2.840.113556.1.2.3                                          |
| System-ID-GUID    | bf967a77-0de6-11d0-a285-00aa003049e2                          |
| Syntax            | [**Cadeia de caracteres (em tempo geral)**](s-string-generalized-time.md) |
