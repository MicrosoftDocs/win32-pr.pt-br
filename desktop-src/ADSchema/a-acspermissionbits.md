# <a name="acs-permission-bits-attribute"></a>ACS-atributo de permissão-bits

Permite a origem do fluxo de multicast para um determinado usuário.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-permissão-bits                  |
| LDAP-Display-Name | aCSPermissionBits                    |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.765               |
| System-ID-GUID    | 7f561282-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
