# <a name="dns-record-attribute"></a>Dns-Record atributo

Usado para armazenar registros de recursos DNS binários em objetos DNS.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Dns-Record                                            |
| LDAP-Display-Name | dnsRecord                                             |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                      |
| Frequência de atualização  | Sempre que os registros de recurso DNS forem alterados.                 |
| Attribute-Id      | 1.2.840.113556.1.4.382                                |
| System-ID-GUID    | e0fa1e69-9b45-11d0-afdd-00c04fd930c9                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
