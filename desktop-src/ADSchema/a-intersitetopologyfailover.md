# <a name="inter-site-topology-failover-attribute"></a>Atributo entre sites-topologia-failover

Indica quanto tempo deve ser transcorrido desde a última atividade para o gerador de topologia entre sites ser considerada inativa.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Failover entre sites-topologia         |
| LDAP-Display-Name | interSiteTopologyFailover            |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1248              |
| System-ID-GUID    | b7c69e60-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
