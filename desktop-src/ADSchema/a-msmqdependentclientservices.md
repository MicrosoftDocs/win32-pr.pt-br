# <a name="msmq-dependent-client-services-attribute"></a>Atributo dependente do MSMQ-cliente-serviços

Indica se o MSMQ instalado neste computador fornece serviços de cliente dependentes do MSMQ.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Serviços de cliente dependentes do MSMQ       |
| LDAP-Display-Name | mSMQDependentClientServices          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1226              |
| System-ID-GUID    | 2df90d76-009f-11d2-aa4c-00c04fd7d83a |
| Syntax            | [**Boolean**](s-boolean.md)         |
