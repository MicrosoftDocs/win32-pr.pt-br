# <a name="frs-root-path-attribute"></a>Atributo FRS-raiz-Path

Caminho para a raiz da árvore do sistema de arquivos replicado.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | FRS-raiz-caminho                               |
| LDAP-Display-Name | fRSRootPath                                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.487                      |
| System-ID-GUID    | 1be8f174-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
