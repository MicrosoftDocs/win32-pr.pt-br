# <a name="uniqueidentifier-attribute"></a>Atributo uniqueIdentifier

O tipo de atributo uniqueIdentifier especifica um "identificador exclusivo" para um objeto representado no Diretório.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Uniqueidentifier                            |
| Ldap-Display-Name | Uniqueidentifier                            |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.44                  |
| System-Id-Guid    | ba0184c7-38c5-4bed-a526-75421470580c        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
