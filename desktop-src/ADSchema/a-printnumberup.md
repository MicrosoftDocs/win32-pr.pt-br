# <a name="print-number-up-attribute"></a>Imprimir-atributo de numeração

O número de imagens de página por folha.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Imprimir-numeração                      |
| LDAP-Display-Name | printNumberUp                        |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.290               |
| System-ID-GUID    | 3bcbfcf4-4d3d-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
