# <a name="presentation-address-attribute"></a>Presentation-Address atributo

Especifica um endereço de apresentação associado a um objeto que representa uma entidade de aplicativo OSI.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------------|
| CN                | Presentation-Address                                                  |
| LDAP-Display-Name | presentationAddress                                                   |
| Tamanho              | \-                                                                    |
| Privilégio de atualização  | \-                                                                    |
| Frequência de atualização  | \-                                                                    |
| Attribute-Id      | 2.5.4.29                                                              |
| System-ID-GUID    | a8df744b-c5ea-11d1-bbcb-0080c76670c0                                  |
| Syntax            | [**Objeto (endereço de apresentação)**](s-object-presentation-address.md) |
