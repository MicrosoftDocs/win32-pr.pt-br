# <a name="ms-net-ieee-8023-gp-policydata-attribute"></a>atributo ms-net-IEEE-8023-GP-PolicyData

| LDAP-Display-Name | MS-net-IEEE-8023-GP-PolicyData              |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
