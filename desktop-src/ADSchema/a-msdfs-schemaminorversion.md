# <a name="ms-dfs-schema-minor-version-attribute"></a>MS-DFS-Schema-atributo de versão secundária

Contém a versão secundária do esquema de metadados do DFS.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-DFS-Schema – minor-version          |
| LDAP-Display-Name | msDFS-SchemaMinorVersion             |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2031              |
| System-ID-GUID    | fef9a725-e8f1-43ab-bd86-6a0115ce9e38 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
