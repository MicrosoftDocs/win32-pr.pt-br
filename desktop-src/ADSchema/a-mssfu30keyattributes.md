# <a name="mssfu-30-key-attributes-attribute"></a>atributo msSFU-30-Key-Attributes

Contém os nomes dos atributos que o servidor NIS usa como chaves para um mapa de pesquisa.

|-------------------|---------------------------------------------|
| CN                | msSFU-30-chave-atributos                     |
| LDAP-Display-Name | msSFU30KeyAttributes                        |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.301                 |
| System-ID-GUID    | 32ecd698-ce9e-4894-a134-7ad76b082e83        |
| Sintaxe            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
