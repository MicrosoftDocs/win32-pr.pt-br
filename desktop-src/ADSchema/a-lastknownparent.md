# <a name="last-known-parent-attribute"></a>Último atributo pai conhecido

O DN (nome distinto) do último pai conhecido de um objeto órfão.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Último-pai conhecido                       |
| LDAP-Display-Name | lastKnownParent                         |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Esse valor é definido pelo sistema.        |
| Frequência de atualização  | Sempre que um objeto estiver órfão.         |
| Attribute-Id      | 1.2.840.113556.1.4.781                  |
| System-ID-GUID    | 52ab8670-5709-11d1-a9c6-0000f80367c1    |
| Sintaxe            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
