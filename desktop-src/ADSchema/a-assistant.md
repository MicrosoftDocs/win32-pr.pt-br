# <a name="assistant-attribute"></a>Atributo do assistente

O nome distinto do assistente administrativo de um usuário.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Assistente                               |
| LDAP-Display-Name | assistente                               |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.652                  |
| System-ID-GUID    | 0296c11c-40da-11d1-a9c0-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
