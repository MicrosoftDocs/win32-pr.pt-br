# <a name="object-guid-attribute"></a>Object-Guid atributo

O identificador exclusivo de um objeto.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------|
| CN                | Object-Guid                                                         |
| LDAP-Display-Name | objectGUID                                                          |
| Tamanho              | 16 bytes                                                            |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                                    |
| Frequência de atualização  | Esse valor é definido quando o objeto é criado e não pode ser alterado. |
| Attribute-Id      | 1.2.840.113556.1.4.2                                                |
| System-ID-GUID    | bf9679e7-0de6-11d0-a285-00aa003049e2                                |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md)               |
