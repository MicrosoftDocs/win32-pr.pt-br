# <a name="generate-rsop-logging-extended-right"></a>Gerar-RSoP-registro em log estendido à direita

O usuário que tem os direitos em uma UO ou domínio poderá gerar dados RSoP no modo de log para os usuários ou computadores dentro da UO.



| Entrada | Valor |
|--------------|--------------------------------------------|
| CN           | Gerar-RSoP-log                      |
| Display-Name | Gerar Conjunto de Políticas Resultante (registro em log) |
| GUID de direitos  | b7b1b3de-ab09-4242-9e30-9980e5d322f7       |
