# <a name="ms-dfsr-schedule-attribute"></a>MS-DFSR-atributo Schedule

Contém o agendamento de replicação para o serviço de replicação do Sistema de Arquivos Distribuído (DFS).



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFSR-Schedule                                      |
| LDAP-Display-Name | msDFSR-agenda                                       |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.14                            |
| System-ID-GUID    | 4699f15f-a71f-48e2-9ff5-5897c0759205                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
