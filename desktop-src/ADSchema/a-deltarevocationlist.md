# <a name="delta-revocation-list-attribute"></a>Atributo Delta-Revocation-List

Lista de certificados que foram revogados desde a última atualização delta.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Delta-Revocation-List                                 |
| Ldap-Display-Name | deltaRevocationList                                   |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 2.5.4.53                                              |
| System-Id-Guid    | 167757b5-47f3-11d1-a9c3-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
