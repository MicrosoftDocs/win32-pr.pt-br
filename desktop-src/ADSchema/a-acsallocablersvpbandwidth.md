# <a name="acs-allocable-rsvp-bandwidth-attribute"></a>ACS-allocable-RSVP-atributo de largura de banda

A largura de banda máxima que pode ser reservada.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-allocable-RSVP-largura de banda         |
| LDAP-Display-Name | aCSAllocableRSVPBandwidth            |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.766               |
| System-ID-GUID    | 7f561283-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
