# <a name="membership-property-set"></a>Conjunto de propriedades de associação

Conjunto de propriedades que contém os atributos de usuário que descrevem as informações de associação de grupo.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Associação                           |
| Display-Name | Associação de grupo                     |
| GUID de direitos  | bc0ac240-79a9-11d0-9020-00c04fc2d4cf |
