# <a name="version-number-lo-attribute"></a>Versão-o atributo Number-lo

Um número de versão secundária de uso geral.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Versão-número-lo                    |
| LDAP-Display-Name | versionNumberLo                      |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.329               |
| System-ID-GUID    | 7d6c0e9b-7e20-11d0-afd6-00c04fd930c9 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
