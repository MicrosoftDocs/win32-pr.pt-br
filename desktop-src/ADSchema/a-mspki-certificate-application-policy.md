# <a name="ms-pki-certificate-application-policy-attribute"></a>atributo de política de aplicativo MS-PKI-Certificate

O OID da política de aplicativo em um certificado.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Certificate-Application-Policy                                                      |
| LDAP-Display-Name | msPKI-certificado-aplicativo-política                                                       |
| Tamanho              | \-                                                                                         |
| Privilégio de atualização  | Administrador corporativo                                                                   |
| Frequência de atualização  | Sempre que um novo modelo é criado ou os atributos de modelos existentes são editados. |
| Attribute-Id      | 1.2.840.113556.1.4.1674                                                                    |
| System-ID-GUID    | dbd90548-aa37-4202-9966-8c537ba5ce32                                                       |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                                |
