# <a name="attribute-security-guid-attribute"></a>Atributo -Security-GUID atributo

O GUID a ser usado para aplicar credenciais de segurança a um conjunto de objetos .



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Attribute-Security-GUID                               |
| Ldap-Display-Name | attributeSecurityGUID                                 |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | Administrador de esquema                                  |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.149                                |
| System-Id-Guid    | bf967924-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
