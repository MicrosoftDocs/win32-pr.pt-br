# <a name="rdn-att-id-attribute"></a>Atributo RDN-ATT-ID

O RDN para o atributo que é usado para nomear uma classe.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------|
| CN                | RDN-ATT-ID                                                      |
| LDAP-Display-Name | rDNAttID                                                        |
| Tamanho              | \-                                                              |
| Privilégio de atualização  | Administrador de esquema                                            |
| Frequência de atualização  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.26                                           |
| System-ID-GUID    | bf967a0f-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**Cadeia de caracteres (identificador de objeto)**](s-string-object-identifier.md) |
