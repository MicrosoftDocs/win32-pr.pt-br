# <a name="ms-ds-nc-repl-outbound-neighbors-attribute"></a>atributo ms-DS-NC-repl-Outbound-Neighbors

Parceiros de replicação para esta partição. Esse servidor envia dados de replicação para esses outros servidores, que atuam como destinos. Esse servidor notificará esses outros servidores quando novos dados estiverem disponíveis.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-repl-Bound-Neighbors            |
| LDAP-Display-Name | msDS-NCReplOutboundNeighbors                |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1706                     |
| System-ID-GUID    | 855f2ef5-a1c5-4cc4-ba6d-32522848b61f        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
