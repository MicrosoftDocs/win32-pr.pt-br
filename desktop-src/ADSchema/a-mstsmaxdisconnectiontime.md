# <a name="ms-ts-max-disconnection-time-attribute"></a>MS-TS-Max-desconnect-Attribute-time

O tempo máximo de desconexão da sessão dos serviços de terminal é a quantidade máxima de tempo, em minutos, em que uma sessão desconectada dos serviços de terminal permanece ativa no servidor de terminal.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-TS-Max-Disconnection-time         |
| LDAP-Display-Name | msTSMaxDisconnectionTime             |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1981              |
| System-ID-GUID    | 326f7089-53d8-4784-b814-46d8535110d2 |
