# <a name="acs-non-reserved-min-policed-size-attribute"></a>ACS-atributo não reservado-min-Police-size

O atributo **ACS-non-reservado-min-Police-size** é somente para uso interno. Com base em RFC2814.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-não reservada-min-Tamanho-de-polícia    |
| LDAP-Display-Name | aCSNonReservedMinPolicedSize         |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1321              |
| System-ID-GUID    | b6873917-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
