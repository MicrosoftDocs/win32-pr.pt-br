# <a name="ms-ds-revealed-users-attribute"></a>ms-DS-revelados-atributo de usuários

Usado com RODCs para identificar os objetos de usuário cujos segredos foram divulgados para esse RODC.



| Entrada | Valor |
|-------------------|-------------------------------------------------|
| CN                | ms-DS-revelado-usuários                            |
| LDAP-Display-Name | msDS-RevealedUsers                              |
| Tamanho              | \-                                              |
| Privilégio de atualização  | \-                                              |
| Frequência de atualização  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1924                         |
| System-ID-GUID    | 185c7821-3749-443a-bd6a-288899071adb            |
| Sintaxe            | [**Objeto (DN-binário)**](s-object-dn-binary.md) |
