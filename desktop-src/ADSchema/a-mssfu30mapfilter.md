# <a name="mssfu-30-map-filter-attribute"></a>msSFU-30-mapear atributo de filtro

Contém uma cadeia de caracteres usada para filtrar dados em um mapa. Pode conter chaves de mapa, o nome de domínio e outros tipos de dados.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | msSFU-30-MAP-Filter                         |
| LDAP-Display-Name | msSFU30MapFilter                            |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.306                 |
| System-ID-GUID    | b7b16e01-024f-4e23-ad0d-71f1a406b684        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
