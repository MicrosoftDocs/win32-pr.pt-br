# <a name="send-as-extended-right"></a>Send-As estendido à direita

Exchange direito: permite enviar email como a caixa de correio.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Send-As                              |
| Display-Name | Enviar como                              |
| GUID de direitos  | ab721a54-1e2f-11d0-9819-00aa0040529b |
