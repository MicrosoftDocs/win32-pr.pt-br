# <a name="ms-ds-quota-effective-attribute"></a>ms-DS-atributo de cota efetiva

A cota efetiva para uma entidade de segurança computada das cotas atribuídas para uma partição de diretório.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ms-DS-cota-efetiva                |
| LDAP-Display-Name | msDS-QuotaEffective                  |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1848              |
| System-ID-GUID    | 6655b152-101c-48b4-b347-e1fcebc60157 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
