# <a name="generate-rsop-planning-extended-right"></a>Gerar-RSoP-planejamento estendido à direita

O usuário que tem os direitos em uma UO ou domínio poderá gerar dados RSoP do modo de planejamento para os usuários ou computadores dentro da UO.



| Entrada | Valor |
|--------------|---------------------------------------------|
| CN           | Gerar-RSoP-Planning                      |
| Display-Name | Gerar Conjunto de Políticas Resultante (planejamento) |
| GUID de direitos  | b7b1b3dd-ab09-4242-9e30-9980e5d322f7        |
