# <a name="machine-wide-policy-attribute"></a>Atributo de política de todo o computador

Usado para replicar a política extensível do usuário para os clientes.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Política de todo o computador                                   |
| LDAP-Display-Name | machineWidePolicy                                     |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.459                                |
| System-ID-GUID    | 80a67e4f-9f22-11d0-afdd-00c04fd930c9                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
