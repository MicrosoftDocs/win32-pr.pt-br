# <a name="nc-name-attribute"></a>NC-Name atributo

O nome distinto do contexto de nomenclatura para o objeto.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | NC-Name                                 |
| LDAP-Display-Name | nCName                                  |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Administrador de domínio                    |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.16                   |
| System-ID-GUID    | bf9679d6-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
