# <a name="ds-replication-monitor-topology-extended-right"></a>DS-Replication-Monitor-topologia estendida à direita

Direito de acesso de controle estendido que permite a leitura de dados de monitoramento de replicação, como status de replicação e metadados de objeto.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | DS-Replication-Monitor-Topology      |
| Display-Name | Monitorar Active Directory replicação |
| GUID de direitos  | f98340fb-7c5b-4cdb-a00b-2ebdfa115a96 |
