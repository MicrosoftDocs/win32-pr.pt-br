# <a name="ms-ds-preferred-gc-site-attribute"></a>ms-DS-Preferred-GC-atributo de site

O atributo **MS-DS-Preferred-GC-site** é usado pelo gerente de contas de segurança para a expansão de grupo durante a avaliação do token.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Preferred-GC-site                 |
| LDAP-Display-Name | msDS-Preferred-GC-site                  |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Administrador de domínio                    |
| Frequência de atualização  | A critério do administrador.      |
| Attribute-Id      | 1.2.840.113556.1.4.1444                 |
| System-ID-GUID    | d921b50a-0ab2-42cd-87f6-09cf83a91854    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
