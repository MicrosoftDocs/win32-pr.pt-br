# <a name="ms-ds-authenticatedat-dc-attribute"></a>atributo ms-DS-AuthenticatedAt-DC

Link de encaminhamento para [**MS-DS-AuthenticatedTo-Accountlist**](a-msds-authenticatedtoaccountlist.md). Identifica a qual DC um usuário autenticou.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AuthenticatedAt-DC                |
| LDAP-Display-Name | msDS-AuthenticatedAtDC                  |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1958                 |
| System-ID-GUID    | 3e1ee99c-6604-4489-89d9-84798a89515a    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
