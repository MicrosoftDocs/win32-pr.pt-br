# <a name="frs-staging-path-attribute"></a>FRS-atributo de preparo-Path

O caminho para a área de preparo de replicação de arquivo.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | FRS – caminho de preparo                            |
| LDAP-Display-Name | fRSStagingPath                              |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.488                      |
| System-ID-GUID    | 1be8f175-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
