# <a name="ms-ieee-80211-id-attribute"></a>atributo ms-IEEE-80211-ID

Um identificador usado para um objeto de política sem fio no AD.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-ID                                                                 |
| LDAP-Display-Name | msieee80211-ID                                                                   |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Administrador de domínio                                                             |
| Frequência de atualização  | Cada vez que um administrador de domínio altera a diretiva de rede sem fio para um domínio ou unidade organizacional. |
| Attribute-Id      | 1.2.840.113556.1.4.1823                                                          |
| System-ID-GUID    | 7f73ef75-14c9-4c23-81de-dd07a06f9e8b                                             |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                      |
