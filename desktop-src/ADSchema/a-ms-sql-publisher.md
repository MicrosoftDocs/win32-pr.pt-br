# <a name="ms-sql-publisher-attribute"></a>Atributo MS-SQL-Publisher

O nome do banco de dados Publicador para replicação.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Publisher                            |
| LDAP-Display-Name | mS-SQL-Publisher                            |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | Quando a replicação é configurada.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| System-ID-GUID    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
