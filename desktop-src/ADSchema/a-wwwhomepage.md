# <a name="www-home-page-attribute"></a>Atributo WWW-Home-Page

Uma página da Web que é a página de aterrissagem principal de um site.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-Home-Page                                                               |
| LDAP-Display-Name | wWWHomePage                                                                 |
| Tamanho              | \-                                                                          |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta.                                      |
| Frequência de atualização  | Quando o registro do usuário é criado e sempre que a página da Web precisa ser alterada. |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| System-ID-GUID    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                 |
