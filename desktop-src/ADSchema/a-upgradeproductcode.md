# <a name="upgrade-product-code-attribute"></a>Atualizar-atributo de código do produto

Esse atributo contém o código do produto de outros pacotes, como aplicativos, que pode ser atualizado por esse pacote ou que pode atualizar este pacote.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Atualização – código do produto                                  |
| LDAP-Display-Name | upgradeProductCode                                    |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| System-ID-GUID    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
