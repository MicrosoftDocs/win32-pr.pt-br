# <a name="operating-system-service-pack-attribute"></a>Atributo Operating System-Service-Pack

A cadeia de caracteres de ID de service pack do sistema operacional (por exemplo, SP3).



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Sistema operacional-Service-Pack               |
| LDAP-Display-Name | operatingSystemServicePack                  |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.365                      |
| System-ID-GUID    | 3e978927-8c01-11d0-afda-00c04fd930c9        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
