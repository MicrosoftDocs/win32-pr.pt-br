# <a name="default-class-store-attribute"></a>Atributo de repositório de classe padrão

O repositório de classe padrão para um determinado usuário.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Armazenamento de classe padrão                     |
| LDAP-Display-Name | defaultClassStore                       |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.213                  |
| System-ID-GUID    | bf967948-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
