# <a name="acs-non-reserved-max-sdu-size-attribute"></a>ACS-não reservado-atributo Max-SDU-size

O atributo **ACS-non-reserved-Max-SDU-size** é somente para uso interno. Com base em RFC2814.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-não reservada-Max-SDU-size        |
| LDAP-Display-Name | aCSNonReservedMaxSDUSize             |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1320              |
| System-ID-GUID    | aec2cfe3-3b90-11d2-90cc-00c04fd91ab1 |
| Sintaxe            | [**Intervalo**](s-interval.md)       |
