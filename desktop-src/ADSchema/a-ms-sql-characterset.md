# <a name="ms-sql-characterset-attribute"></a>Atributo MS-SQL-CharacterSet

O conjunto de caracteres da instância atual do SQL Server.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-SQL-CharacterSet                  |
| LDAP-Display-Name | mS-SQL-CharacterSet                  |
| Tamanho              | \-                                   |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Na inicialização do sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1370              |
| System-ID-GUID    | 696177a6-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
