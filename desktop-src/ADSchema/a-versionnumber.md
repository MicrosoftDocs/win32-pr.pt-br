# <a name="version-number-attribute"></a>Version-Number atributo

Um número de versão de uso geral.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Version-Number                       |
| LDAP-Display-Name | versionNumber                        |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.141               |
| System-ID-GUID    | bf967a76-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
