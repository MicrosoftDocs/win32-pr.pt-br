# <a name="picture-attribute"></a>Atributo de imagem

Uma imagem do usuário. É recomendável um formato eficiente de espaço como JPEG ou GIF.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Picture                                               |
| LDAP-Display-Name | thumbnailPhoto                                        |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 2.16.840.1.113730.3.1.35                              |
| System-ID-GUID    | 8d3bca50-1d7e-11d0-a081-00aa006c33ed                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
