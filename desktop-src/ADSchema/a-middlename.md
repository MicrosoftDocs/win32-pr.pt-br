# <a name="other-name-attribute"></a>Other-Name atributo

Nomes adicionais para um usuário. Por exemplo, nome do meio, patronymic, matronymic ou outros.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Other-Name                                  |
| LDAP-Display-Name | middleName                                  |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 2.16.840.1.113730.3.1.34                    |
| System-ID-GUID    | bf9679f2-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
