# <a name="frs-computer-reference-attribute"></a>FRS-computador-atributo de referência

Referência a um objeto de computador do membro do conjunto de réplicas.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | FRS-referência de computador                  |
| LDAP-Display-Name | frsComputerReference                    |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.869                  |
| System-ID-GUID    | 2a132578-9373-11d1-aebc-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
