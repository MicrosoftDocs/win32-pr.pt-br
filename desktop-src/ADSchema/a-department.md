# <a name="department-attribute"></a>Atributo Department

Contém o nome do departamento no qual o usuário trabalha.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Departamento                                  |
| LDAP-Display-Name | department                                  |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta.      |
| Frequência de atualização  | Sempre que o departamento precisar ser alterado.    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| System-ID-GUID    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
