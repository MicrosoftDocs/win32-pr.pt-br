# <a name="site-guid-attribute"></a>Atributo site-GUID

O identificador exclusivo de um site.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Site-GUID                                             |
| LDAP-Display-Name | siteGUID                                              |
| Tamanho              | 16 bytes                                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                      |
| Frequência de atualização  | Sempre que um novo site é criado.                       |
| Attribute-Id      | 1.2.840.113556.1.4.362                                |
| System-ID-GUID    | 3e978924-8c01-11d0-afda-00c04fd930c9                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
