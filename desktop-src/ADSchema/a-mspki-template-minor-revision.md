# <a name="ms-pki-template-minor-revision-attribute"></a>Atributo ms-PKI-Template-Minor-Revision

Mantém o controle dos atributos na alteração de classe. No entanto, isso não disparará o registro automático.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Template-Minor-Revision                                                                    |
| Ldap-Display-Name | msPKI-Template-Minor-Revision                                                                     |
| Tamanho              | 4 bytes                                                                                           |
| Privilégio de atualização  | Administrador de domínio                                                                              |
| Frequência de atualização  | Quando o objeto de modelo de certificado (ms-PKI-Certificate-Template) é editado, criado ou clonado. |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| System-Id-Guid    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Syntax            | [**Enumeração**](s-enumeration.md)                                                              |
