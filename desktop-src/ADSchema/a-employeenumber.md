# <a name="employee-number-attribute"></a>Employee-Number atributo

O número atribuído a um funcionário que não seja a ID.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Employee-Number                             |
| LDAP-Display-Name | employeeNumber                              |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.610                      |
| System-ID-GUID    | a8df73ef-c5ea-11d1-bbcb-0080c76670c0        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
