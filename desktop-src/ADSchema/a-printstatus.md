# <a name="print-status-attribute"></a>Print-Status atributo

Status do spooler de impressão. Atualmente não utilizado.

|-------------------|---------------------------------------------|
| CN                | Print-Status                                |
| LDAP-Display-Name | restatus                                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.273                      |
