# <a name="validated-spn-validated-writes"></a>Validado-gravações validadas de SPN

| Entrada | Valor |
|--------------|-------------------------------------------|
| CN           | Validado-SPN                             |
| Display-Name | Gravação validada no nome da entidade de serviço |
| GUID de direitos  | f3a64788-5306-11d1-a9c5-0000f80367c1      |
