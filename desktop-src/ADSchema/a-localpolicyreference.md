# <a name="local-policy-reference-attribute"></a>Atributo de referência de política local

Nome distinto de um objeto de política local do qual um objeto de política copia.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Referência de política local                  |
| LDAP-Display-Name | localPolicyReference                    |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.457                  |
| System-ID-GUID    | 80a67e4d-9f22-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
