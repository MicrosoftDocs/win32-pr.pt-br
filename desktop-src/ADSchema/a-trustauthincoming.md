# <a name="trust-auth-incoming-attribute"></a>Atributo Trust-auth-transcoming

Informações de autenticação para a parte de entrada de uma relação de confiança.

|-------------------|-------------------------------------------------------|
| CN                | Trust-auth-entrada                                   |
| LDAP-Display-Name | trustAuthIncoming                                     |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.129                                |
