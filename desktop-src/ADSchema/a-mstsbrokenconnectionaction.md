# <a name="ms-ts-broken-connection-action-attribute"></a>MS-TS-atributo de ação de conexão interrompida

A ação conexão interrompida da sessão dos serviços de terminal especifica a ação a ser tomada quando um limite de sessão dos serviços de terminal for atingido.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-TS-ação de conexão interrompida       |
| LDAP-Display-Name | msTSBrokenConnectionAction           |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1985              |
| System-ID-GUID    | 1cf41bba-5604-463e-94d6-1a1287b72ca3 |
| Syntax            | [**Boolean**](s-boolean.md)         |
