# <a name="acs-dsbm-refresh-attribute"></a>ACS-DSBM-atualizar atributo

Esse atributo contém o valor de timer de intervalo que determina quando o DSBM (Gerenciador de largura de banda de sub-rede) designado envia uma mensagem de atualização (estou \_ \_ DSBM) para todos os gerenciadores de largura de banda de sub-rede em um domínio.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-atualizar                     |
| LDAP-Display-Name | aCSDSBMRefresh                       |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.777               |
| System-ID-GUID    | 1cb3559f-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
