# <a name="netboot-current-client-count-attribute"></a>netboot-atributo atual-de-cliente-contagem

O atributo **netboot – Current-Client-Count** é reservado para uso interno.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | netboot-atual-contagem de clientes         |
| LDAP-Display-Name | netbootCurrentClientCount            |
| Tamanho              | \-                                   |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.852               |
| System-ID-GUID    | 07383079-91df-11d1-aebc-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
