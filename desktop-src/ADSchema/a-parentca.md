# <a name="parent-ca-attribute"></a>Atributo pai-CA

O nome distinto de um objeto de autoridade de certificação (CA) para uma autoridade de certificação pai.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Pai-AC                               |
| LDAP-Display-Name | parentCA                                |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.557                  |
| System-ID-GUID    | 5245801b-ca6a-11d0-afff-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
