# <a name="text-encoded-or-address-attribute"></a>Atributo OR-Address codificado em texto

Esse atributo é usado para dar suporte a endereços X.400 em um formato de texto.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Text-Encoded-OR-Address                     |
| Ldap-Display-Name | textEncodedORAddress                        |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.2                   |
| System-Id-Guid    | a8df7489-c5ea-11d1-ctb-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
