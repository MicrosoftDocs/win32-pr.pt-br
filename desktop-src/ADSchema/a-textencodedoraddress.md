# <a name="text-encoded-or-address-attribute"></a>Atributo Text-encoded-ou-address

Esse atributo é usado para dar suporte a endereços X. 400 em um formato de texto.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Texto-codificado ou endereço                     |
| LDAP-Display-Name | textEncodedORAddress                        |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.2                   |
| System-ID-GUID    | a8df7489-c5ea-11d1-bbcb-0080c76670c0        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
