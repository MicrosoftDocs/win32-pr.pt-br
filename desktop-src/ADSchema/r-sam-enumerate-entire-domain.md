# <a name="sam-enumerate-entire-domain-extended-right"></a>SAM-enumerar-total-domínio estendido à direita

Esse direito de acesso de controle pode ser usado para restringir quem pode ter permissão para usar uma API de nível inferior, como NetQueryDisplayInformation e netuser/GroupEnum, e enumerar o domínio inteiro.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | SAM-enumerar-inteiro-domínio          |
| Display-Name | Enumerar todo o domínio SAM          |
| GUID de direitos  | 91d67418-0135-4acc-8d79-c08e857cfbec |
