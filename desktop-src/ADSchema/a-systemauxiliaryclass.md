# <a name="system-auxiliary-class-attribute"></a>Atributo System-Auxiliary-Class

Uma lista de classes auxiliares que não podem ser modificadas pelo usuário.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------|
| CN                | Classe auxiliar do sistema                                             |
| Ldap-Display-Name | systemIliaryClass                                               |
| Tamanho              | \-                                                                 |
| Privilégio de atualização  | Administrador de esquema                                               |
| Frequência de atualização  | Quando a classe é criada ou uma nova classe auxiliar é adicionada a ela. |
| Attribute-Id      | 1.2.840.113556.1.4.198                                             |
| System-Id-Guid    | bf967a43-0de6-11d0-a285-00aa003049e2                               |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)    |
