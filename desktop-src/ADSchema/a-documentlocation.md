# <a name="documentlocation-attribute"></a>atributo documentLocation

O tipo de atributo documentLocation especifica o local do documento original.

|-------------------|---------------------------------------------|
| CN                | documentLocation                            |
| LDAP-Display-Name | documentLocation                            |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
