# <a name="ms-ds-az-ldap-query-attribute"></a>Atributo ms-DS-Az-LDAP-Query

Uma cadeia de caracteres que define a consulta LDAP (comprimento máximo 4096) que determina a associação de um objeto de usuário ao grupo.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Az-LDAP-Query                         |
| Ldap-Display-Name | msDS-AzLDAPQuery                            |
| Tamanho              | 4096 caracteres                             |
| Privilégio de atualização  | Administrador do AzRoles                               |
| Frequência de atualização  | Durante a inicialização ou alteração da política.     |
| Attribute-Id      | 1.2.840.113556.1.4.1792                     |
| System-Id-Guid    | 5e53368b-fc94-45c8-9d7d-daf31ee7112d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
