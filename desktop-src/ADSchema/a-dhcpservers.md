# <a name="dhcp-servers-attribute"></a>atributo DHCP-Servers

Contém uma lista de servidores que são autorizados na empresa.



| Entrada | Valor |
|-------------------|--------------------------------------------------------|
| CN                | Servidores DHCP                                           |
| LDAP-Display-Name | dhcpServers                                            |
| Tamanho              | \-                                                     |
| Privilégio de atualização  | Administrador de domínio.                                  |
| Frequência de atualização  | Quando um novo servidor é adicionado ou removido da floresta. |
| Attribute-Id      | 1.2.840.113556.1.4.704                                 |
| System-ID-GUID    | 963d2745-48be-11d1-a9c3-0000f80367c1                   |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)                    |
