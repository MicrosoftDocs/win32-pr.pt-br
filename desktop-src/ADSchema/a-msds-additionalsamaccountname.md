# <a name="ms-ds-additional-sam-account-name-attribute"></a>ms-DS-Additional-Sam-atributo-Account-Name

Esse atributo é usado para armazenar os nomes de conta SAM que correspondem aos nomes de host DNS em MS-DS-Additional-DNS-Host-Name.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-adicional-Sam-Account-Name           |
| LDAP-Display-Name | msDS-AdditionalSamAccountName               |
| Tamanho              | Menos de 20 bytes.                         |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | Quando o DC é renomeado.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1718                     |
| System-ID-GUID    | 975571df-a4d5-429a-9f59-cdc6581d91e6        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
