# <a name="ms-pki-oid-attribute-attribute"></a>atributo ms-PKI-OID-Attribute

O atributo para o OID da empresa.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-Attribute                                                                       |
| LDAP-Display-Name | msPKI-OID-Attribute                                                                        |
| Tamanho              | 4 bytes                                                                                    |
| Privilégio de atualização  | Administrador corporativo                                                                   |
| Frequência de atualização  | Sempre que um novo modelo é criado ou os atributos de modelos existentes são editados. |
| Attribute-Id      | 1.2.840.113556.1.4.1671                                                                    |
| System-ID-GUID    | 8c9e1288-5028-4f4f-a704-76d026f246ef                                                       |
| Syntax            | [**Enumeração**](s-enumeration.md)                                                       |
