# <a name="cost-attribute"></a>Atributo de custo

Contém o custo relativo para rotear mensagens por meio de um conector de site específico.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Custo                                 |
| LDAP-Display-Name | cost                                 |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.135               |
| System-ID-GUID    | bf967944-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
