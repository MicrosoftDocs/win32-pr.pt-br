# <a name="usn-last-obj-rem-attribute"></a>Atributo USN-Last-obj-REM

Contém o USN (número de sequência de atualização) do último objeto que não é do sistema que foi removido de um servidor.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | USN-Last-obj-REM                     |
| LDAP-Display-Name | uSNLastObjRem                        |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Sempre que um objeto de diretório é alterado. |
| Attribute-Id      | 1.2.840.113556.1.2.121               |
| System-ID-GUID    | bf967a73-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
