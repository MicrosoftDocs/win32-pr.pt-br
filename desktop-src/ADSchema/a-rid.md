# <a name="rid-attribute"></a>Atributo Rid

O Identificador relativo de um objeto .



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Livrar                                         |
| Ldap-Display-Name | rid                                         |
| Tamanho              | 4 bytes                                     |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | Quando um objeto que precisa de um RID é criado. |
| Attribute-Id      | 1.2.840.113556.1.4.153                      |
| System-Id-Guid    | bf967a22-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**Enumeração**](s-enumeration.md)        |
