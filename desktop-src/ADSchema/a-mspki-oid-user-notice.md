# <a name="ms-pki-oid-user-notice-attribute"></a>atributo ms-PKI-OID-User-Notice

O usuário observa o OID da política do emissor corporativo.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-User-Notice                                                                     |
| LDAP-Display-Name | msPKI-OID-User-Notice                                                                      |
| Tamanho              | \-                                                                                         |
| Privilégio de atualização  | Administrador corporativo                                                                   |
| Frequência de atualização  | Sempre que um novo modelo é criado ou os atributos de modelos existentes são editados. |
| Attribute-Id      | 1.2.840.113556.1.4.1673                                                                    |
| System-ID-GUID    | 04c4da7a-e114-4e69-88de-e293f2d3b395                                                       |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                                |
