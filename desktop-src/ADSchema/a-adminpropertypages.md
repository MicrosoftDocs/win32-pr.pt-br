# <a name="admin-property-pages-attribute"></a>Atributo admin-Property-Pages

O número do pedido e o GUID das páginas de propriedades de um objeto a ser exibido nas telas de administração Active Directory. Para obter mais informações, consulte o documento, estendendo a interface do usuário para objetos de diretório.



| Entrada | Valor |
|-------------------|------------------------------------------------|
| CN                | Admin-Propriedades-páginas                           |
| LDAP-Display-Name | adminPropertyPages                             |
| Tamanho              | \-                                             |
| Privilégio de atualização  | Administrador de domínio ou desenvolvedor de aplicativos. |
| Frequência de atualização  | Sempre que uma página de propriedades é adicionada ou removida.  |
| Attribute-Id      | 1.2.840.113556.1.4.562                         |
| System-ID-GUID    | 52458038-ca6a-11D0-afff-0000f80367c1           |
| Sintaxe            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)    |
