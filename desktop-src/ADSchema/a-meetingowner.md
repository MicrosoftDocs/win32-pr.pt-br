# <a name="meetingowner-attribute"></a>atributo meetingOwner

A pessoa responsável pela reunião.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | meetingOwner                                                                     |
| LDAP-Display-Name | meetingOwner                                                                     |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Qualquer pessoa pode atualizar esse objeto com base na segurança do objeto que está sendo criado. |
| Frequência de atualização  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.579                                                           |
| System-ID-GUID    | 11b6cc88-48c4-11d1-a9c3-0000f80367c1                                             |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                      |
