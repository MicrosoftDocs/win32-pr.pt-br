# <a name="ms-ds-has-master-ncs-attribute"></a>atributo ms-DS-tem-Master-NCs

Contém os contextos de nomenclatura mantidos por um controlador de domínio. Esse atributo é substituído por-Master-NCs.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-tem-mestre-NCs                    |
| LDAP-Display-Name | msDS-hasMasterNCs                       |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Esse valor é definido pelo sistema         |
| Frequência de atualização  | Cada vez que um novo NC é criado.          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| System-ID-GUID    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
