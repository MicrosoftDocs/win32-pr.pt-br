Link para trás do grupo de aplicativos do membro ou usuário para Az-Role objetos que se vinculam a ele.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Members-for-AZ-role-BL            |
| LDAP-Display-Name | msDS-MembersForAzRoleBL                 |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | Esse valor é definido pelo sistema.        |
| Attribute-Id      | 1.2.840.113556.1.4.1807                 |
| System-ID-GUID    | ececcd20-a7e0-4688-9ccf-02ece5e287f5    |
