# <a name="ms-ds-external-key-attribute"></a>atributo ms-DS-external-Key

Uma cadeia de caracteres que identifica um objeto em um repositório externo, como um registro em um banco de dados.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-external-Key                          |
| LDAP-Display-Name | msDS-ExternalKey                            |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1833                     |
| System-ID-GUID    | b92fd528-38ac-40d4-818d-0433380837c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
