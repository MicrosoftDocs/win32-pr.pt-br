# <a name="macaddress-attribute"></a>atributo macAddress

Contém o endereço MAC na notação de hexadecimal máxima separada por dois-pontos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | macAddress                           |
| LDAP-Display-Name | macAddress                           |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.22                     |
| System-ID-GUID    | e6a522dd-9770-43e1-89de-1de5044328f7 |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
