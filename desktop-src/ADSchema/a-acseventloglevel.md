# <a name="acs-event-log-level-attribute"></a>ACS-atributo de nível de log de evento

Nível de log RSVP.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ACS-nível de log de eventos                     |
| LDAP-Display-Name | aCSEventLogLevel                        |
| Tamanho              | 4 bytes pode ter os valores 0, 1, 2 e 3. |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.769                  |
| System-ID-GUID    | 7f561286-5301-11d1-a9c5-0000f80367c1    |
| Syntax            | [**Enumeração**](s-enumeration.md)    |
