# <a name="lm-pwd-history-attribute"></a>Atributo LM-pwd-History

O histórico de senha do usuário no formato unidirecional do LM (LAN Manager) (). O LM OWF é usado para compatibilidade com o LAN Manager 2. *x* clients, Windows 95 e Windows 98.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | LM-pwd-History                                        |
| LDAP-Display-Name | lmPwdHistory                                          |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | Administrador de domínio                                  |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.160                                |
| System-ID-GUID    | bf96799d-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
