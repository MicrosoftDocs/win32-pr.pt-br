# <a name="ms-ds-az-operation-id-attribute"></a>atributo ms-DS-AZ-Operation-ID

Uma ID específica do aplicativo que torna a operação exclusiva para o aplicativo.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-AZ-Operation-ID                   |
| LDAP-Display-Name | msDS-AzOperationID                      |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Administrador do AzRoles                           |
| Frequência de atualização  | Durante a inicialização ou a alteração da política. |
| Attribute-Id      | 1.2.840.113556.1.4.1800                 |
| System-ID-GUID    | a5f3b553-5d76-4cbe-ba3f-4312152cab18    |
| Syntax            | [**Enumeração**](s-enumeration.md)    |
