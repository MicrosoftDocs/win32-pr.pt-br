# <a name="address-syntax-attribute"></a>Address-Syntax atributo

Uma gramática para codificar as propriedades da tabela de exibição como uma cadeia de caracteres.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Address-Syntax                                        |
| LDAP-Display-Name | addressSyntax                                         |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.255                                |
| System-ID-GUID    | 5fd42463-1262-11d0-a060-00aa006c33ed                  |
| Sintaxe            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
