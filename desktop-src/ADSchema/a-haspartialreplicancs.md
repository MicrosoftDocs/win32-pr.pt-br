# <a name="has-partial-replica-ncs-attribute"></a>Atributo tem-parcial-Replica-NCs

Irmão para tem-Master-NCs. O-partial-Replica-NCs reflete o nome distinto para todos os NCs de outro domínio que foram replicados em um catálogo global.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | De-parcial-réplica-NCs                 |
| LDAP-Display-Name | hasPartialReplicaNCs                    |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.15                   |
| System-ID-GUID    | bf967981-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
