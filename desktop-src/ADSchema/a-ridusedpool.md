# <a name="rid-used-pool-attribute"></a>RID-usado-atributo de pool

Os pools RID que foram usados por um controlador de domínio.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | RID-usado-pool                        |
| LDAP-Display-Name | rIDUsedPool                          |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Sempre que um pool RID é esvaziado.      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| System-ID-GUID    | 6617188b-8f3c-11d0-afda-00c04fd930c9 |
| Syntax            | [**Intervalo**](s-interval.md)       |
