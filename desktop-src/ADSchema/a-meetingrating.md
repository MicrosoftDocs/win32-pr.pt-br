# <a name="meetingrating-attribute"></a>atributo meetingRating

A plataforma para a classificação de seleção de conteúdo da Internet de uma reunião.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | meetingRating                               |
| LDAP-Display-Name | meetingRating                               |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.584                      |
| System-ID-GUID    | 11b6cc8d-48c4-11d1-a9c3-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
