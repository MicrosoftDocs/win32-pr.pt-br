# <a name="ms-ds-replication-notify-subsequent-dsa-delay-attribute"></a>atributo ms-DS-Replication-Notify-subseqüente-DSA-Delay

Esse atributo controla o atraso em tempo entre a notificação de cada parceiro de réplica subsequente para um NC.



| Entrada | Valor |
|-------------------|-----------------------------------------------|
| CN                | ms-DS-Replication-Notify-subseqüente-DSA-Delay |
| LDAP-Display-Name | msDS-Replication-Notify-subseqüente-DSA-Delay  |
| Tamanho              | \-                                            |
| Privilégio de atualização  | Esse valor é definido pelo sistema.              |
| Frequência de atualização  | \-                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1664                       |
| System-ID-GUID    | d63db385-dd92-4b52-b1d8-0d3ecc0e86b6          |
| Syntax            | [**Enumeração**](s-enumeration.md)          |
