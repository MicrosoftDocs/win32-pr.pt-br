| CN                | PKI-crítico-Extensions                     |
| LDAP-Display-Name | pKICriticalExtensions                       |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1330                     |
