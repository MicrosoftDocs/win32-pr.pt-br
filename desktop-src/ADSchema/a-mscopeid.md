# <a name="mscope-id-attribute"></a>Mscope-Id atributo

Indica que há um escopo de multicast no servidor DHCP especificado.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Mscope-Id                            |
| LDAP-Display-Name | mscopeId                             |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.716               |
| System-ID-GUID    | 963d2751-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
