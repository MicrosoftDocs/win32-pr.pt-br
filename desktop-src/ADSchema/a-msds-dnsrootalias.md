# <a name="ms-ds-dnsrootalias-attribute"></a>atributo ms-DS-DnsRootAlias

Usado para armazenar o alias de domínio.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-DnsRootAlias                          |
| LDAP-Display-Name | msDS-DnsRootAlias                           |
| Tamanho              | Tamanho máximo de 255 bytes.                     |
| Privilégio de atualização  | Esse valor é definido pelo sistema.            |
| Frequência de atualização  | Somente durante a reestruturação do domínio.             |
| Attribute-Id      | 1.2.840.113556.1.4.1719                     |
| System-ID-GUID    | 2143acca-eead-4d29-b591-85fa49ce9173        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
