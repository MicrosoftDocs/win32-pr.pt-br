# <a name="boolean-syntax"></a>Sintaxe booliana

Representa um valor booliano.



| Entrada | Valor |
|--------------|--------------------------------------------------------------------------|
| Nome         | Boolean                                                                  |
| ID da sintaxe    | 2.5.5.8                                                                  |
| ID DO OM        | 1                                                                        |
| Tipo de MAPI    | BOOLEAN                                                                  |
| Tipo de ADS     | ADSTYPE \_ booliano                                                         |
| Tipo de variante | BOOL do VT \_                                                                 |
| Tipo SDS     | [System.Boolean](/dotnet/api/system.boolean) |



## <a name="remarks"></a>Comentários

Para consultas que incluem atributos de sintaxe booliana em um filtro, especifique "TRUE" ou "FALSE" (por exemplo, "myboolattr = TRUE").
