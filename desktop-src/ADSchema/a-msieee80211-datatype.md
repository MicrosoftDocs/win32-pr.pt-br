# <a name="ms-ieee-80211-data-type-attribute"></a>MS-IEEE-80211-atributo de tipo de dados

Tipo de dados usado internamente para BLOB de msieee80211 de dados.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-Data-Type                                                          |
| LDAP-Display-Name | msieee80211-tipo de dados                                                             |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Administrador de domínio                                                             |
| Frequência de atualização  | Cada vez que um administrador de domínio altera a diretiva de rede sem fio para um domínio ou unidade organizacional. |
| Attribute-Id      | 1.2.840.113556.1.4.1822                                                          |
| System-ID-GUID    | 6558b180-35da-4efe-beed-521f8f48cafb                                             |
| Syntax            | [**Enumeração**](s-enumeration.md)                                             |
