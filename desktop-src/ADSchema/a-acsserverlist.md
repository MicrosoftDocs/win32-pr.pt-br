# <a name="acs-server-list-attribute"></a>ACS-atributo de lista de servidores

Contém os nomes DNS de servidores do Windows NT que têm permissão para executar o ACS.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ACS-Server-List                             |
| LDAP-Display-Name | aCSServerList                               |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador de domínio                        |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1312                     |
| System-ID-GUID    | 7cbd59a5-3b90-11d2-90cc-00c04fd91ab1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
