# <a name="pwd-last-set-attribute"></a>Nome do atributo pwd-Last-Set

A data e a hora em que a senha desta conta foi alterada pela última vez. Esse valor é armazenado como um inteiro grande que representa o número de intervalos de 100 nanossegundos desde 1º de janeiro de 1601 (UTC). Se esse valor for definido como 0 e o atributo de [**controle de conta de usuário**](a-useraccountcontrol.md) não contiver o sinalizador de **senha da UF não \_ \_ expirar \_** , o usuário deverá definir a senha no próximo logon.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Pwd-último-conjunto                         |
| LDAP-Display-Name | pwdLastSet                           |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | Esse valor é definido pelo sistema.     |
| Frequência de atualização  | Cada vez que a senha é alterada.   |
| Attribute-Id      | 1.2.840.113556.1.4.96                |
| System-ID-GUID    | bf967a0a-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |

## <a name="remarks"></a>Comentários

A parte superior desse inteiro grande corresponde ao membro **dwHighDateTime** da estrutura [**FILETIME**](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) e a parte inferior corresponde ao membro **dwLowDateTime** da estrutura **FILETIME** .
