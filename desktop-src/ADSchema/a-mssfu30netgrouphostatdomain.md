# <a name="mssfu-30-netgroup-host-at-domain-attribute"></a>msSFU-30-netgroup – atributo host-at-Domain

Contém parte do mapa do netgroup que representa cadeias de caracteres computadas, como " host@domain ".

|-------------------|--------------------------------------|
| CN                | msSFU-30-netgroup – host em domínio     |
| LDAP-Display-Name | msSFU30NetgroupHostAtDomain          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.348          |
| System-ID-GUID    | 97d2bf65-0466-4852-a25a-ec20f57ee36c |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
