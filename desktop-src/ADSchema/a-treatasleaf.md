# <a name="treat-as-leaf-attribute"></a>Atributo tratá-as-folha

Os especificadores de exibição com esse sinalizador definido como **true** forçam a exibição da classe relacionada como uma classe folha, mesmo que ela tenha filhos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Tratar como folha                        |
| LDAP-Display-Name | treatAsLeaf                          |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.806               |
| System-ID-GUID    | 8fd044e3-771f-11d1-aeae-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
