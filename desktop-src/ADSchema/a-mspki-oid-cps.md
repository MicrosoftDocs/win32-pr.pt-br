# <a name="ms-pki-oid-cps-attribute"></a>atributo ms-PKI-OID-CPS

O CPS (declaração de política de certificado) para o OID da política do emissor empresarial.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-CPS                                                                             |
| LDAP-Display-Name | msPKI-OID-CPS                                                                              |
| Tamanho              | \-                                                                                         |
| Privilégio de atualização  | Administrador corporativo                                                                   |
| Frequência de atualização  | Sempre que um novo modelo é criado ou os atributos de modelos existentes são editados. |
| Attribute-Id      | 1.2.840.113556.1.4.1672                                                                    |
| System-ID-GUID    | 5f49940e-a79f-4a51-bb6f-3d446a54dc6b                                                       |
