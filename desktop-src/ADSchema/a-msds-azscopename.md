# <a name="ms-ds-az-scope-name-attribute"></a>atributo ms-DS-AZ-Scope-Name

Uma cadeia de caracteres que identifica exclusivamente um objeto de escopo.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Scope-Name                         |
| LDAP-Display-Name | msDS-AzScopeName                            |
| Tamanho              | \-                                          |
| Privilégio de atualização  | Administrador do AzRoles                               |
| Frequência de atualização  | Durante a inicialização ou a alteração da política.     |
| Attribute-Id      | 1.2.840.113556.1.4.1799                     |
| System-ID-GUID    | 515a6b06-2617-4173-8099-d5605df043c6        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
