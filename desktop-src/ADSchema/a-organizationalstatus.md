# <a name="organizationalstatus-attribute"></a>atributo organizationalStatus

O tipo de atributo organizationalStatus especifica uma categoria pela qual uma pessoa geralmente é referida em uma organização.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | organizationalStatus                        |
| LDAP-Display-Name | organizationalStatus                        |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.45                  |
| System-ID-GUID    | 28596019-7349-4d2f-adff-5a629961f942        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
