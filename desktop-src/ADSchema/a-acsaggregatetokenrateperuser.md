# <a name="acs-aggregate-token-rate-per-user-attribute"></a>ACS-Aggregate-token-taxa por atributo de usuário

A taxa máxima de token que qualquer usuário pode ter para todos os fluxos.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-Aggregate-token-taxa por usuário    |
| LDAP-Display-Name | aCSAggregateTokenRatePerUser         |
| Tamanho              | 8 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.760               |
| System-ID-GUID    | 7f56127d-5301-11d1-a9c5-0000f80367c1 |
| Sintaxe            | [**Intervalo**](s-interval.md)       |
