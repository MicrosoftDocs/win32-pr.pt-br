# <a name="user-logon-property-set"></a>User-Logon de propriedades

Conjunto de propriedades que contém atributos de usuário que descrevem informações de logon do usuário.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | User-Logon                           |
| Display-Name | Informações de logon                    |
| RIGHTS-GUID  | 5f202010-79a5-11d0-9020-00c04fc2d4cf |
