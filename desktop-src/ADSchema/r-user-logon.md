# <a name="user-logon-property-set"></a>Conjunto de propriedades User-Logon

Conjunto de propriedades que contém os atributos de usuário que descrevem as informações de logon do usuário.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | User-Logon                           |
| Display-Name | Informações de logon                    |
| GUID de direitos  | 5f202010-79a5-11d0-9020-00c04fc2d4cf |
