# <a name="ms-dfsr-options-attribute"></a>atributo ms-DFSR-Options

Contém as opções de objeto de replicação do Sistema de Arquivos Distribuído (DFS).

|-------------------|--------------------------------------|
| CN                | MS-DFSR-opções                      |
| LDAP-Display-Name | msDFSR-opções                       |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.17           |
| System-ID-GUID    | d6d67084-C720-417D-8647-b696237a114c |
| Syntax            | [**Enumeração**](s-enumeration.md) |
