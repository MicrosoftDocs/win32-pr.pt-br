# <a name="domain-password-property-set"></a>Conjunto de propriedades Domain-Password

Conjunto de propriedades que contém todos os atributos relacionados à idade de bloqueio e senha na conta de usuário.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Domain-Password                      |
| Display-Name | Políticas de bloqueio de & de senha de domínio   |
| GUID de direitos  | c7407360-20bf-11d0-a768-00aa006e0529 |
