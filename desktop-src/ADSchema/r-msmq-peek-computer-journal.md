# <a name="msmq-peek-computer-journal-extended-right"></a>MSMQ-Peek-Computer-Journal Extended Right

Permite exibir mensagens na fila de diário do computador.

|--------------|--------------------------------------|
| CN           | MSMQ-Peek-Computer-Journal           |
| Display-Name | Inspecionar diário do computador                |
| GUID de direitos  | 4b6e08c3-df3c-11d1-9c86-006008764d0e |
