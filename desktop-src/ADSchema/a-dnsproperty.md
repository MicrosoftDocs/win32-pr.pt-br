# <a name="dns-property-attribute"></a>DNS-Property atributo

Usado para armazenar configurações binárias (Propriedades) em objetos de zona DNS.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| LDAP-Display-Name | dNSProperty                                           |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | Esse valor é definido pelo sistema.                      |
| Frequência de atualização  | Sempre que os registros de zona DNS são alterados.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| System-ID-GUID    | 675a15fe-3b70-11d2-90cc-00c04fd91ab1                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
