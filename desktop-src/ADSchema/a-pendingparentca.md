# <a name="pending-parent-ca-attribute"></a>Atributo pendente-pai-CA

Referência às autoridades de certificação que emitiram os certificados pendentes para esta autoridade de certificação.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Pendente-pai-AC                       |
| LDAP-Display-Name | pendingParentCA                         |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.695                  |
| System-ID-GUID    | 963d273e-48be-11d1-a9c3-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
