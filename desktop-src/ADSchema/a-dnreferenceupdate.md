# <a name="dn-reference-update-attribute"></a>Atributo DN-Reference-Update

Se um objeto for renomeado, esse atributo será usado para rastrear todos os nomes anteriores e atuais que foram atribuídos a um objeto para que os objetos vinculados ainda possam encontrá-lo.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | DN-referência-atualização                     |
| LDAP-Display-Name | dNReferenceUpdate                       |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Isso é definido pelo sistema.              |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1242                 |
| System-ID-GUID    | 2df90d86-009f-11d2-aa4c-00c04fd7d83a    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
