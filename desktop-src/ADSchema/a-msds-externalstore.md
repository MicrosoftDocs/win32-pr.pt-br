# <a name="ms-ds-external-store-attribute"></a>atributo ms-DS-external-Store

Uma cadeia de caracteres que identifica o local de um repositório externo, como um banco de dados.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-external-Store                        |
| LDAP-Display-Name | msDS-ExternalStore                          |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1834                     |
| System-ID-GUID    | 604877cd-9cdb-47c7-b03d-3daadb044910        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
