# <a name="ds-ui-admin-maximum-attribute"></a>DS-UI-admin-atributo máximo

Esse é o número máximo padrão de objetos a serem mostrados em um contêiner pela interface do usuário do administrador.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | DS-UI-admin-máximo                  |
| LDAP-Display-Name | dSUIAdminMaximum                     |
| Tamanho              | \-                                   |
| Privilégio de atualização  | Administrador de domínio                 |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1344              |
| System-ID-GUID    | ee8d0ae0-6f91-11d2-9905-0000f87a57d4 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
