# <a name="ms-tapi-ip-address-attribute"></a>atributo ms-TAPI-IP-address

Endereços IP de um computador cliente TAPI no qual um usuário está conectado. Esse atributo pode ser usado como um atributo genérico para manter os endereços IP do computador.



| Entrada | Valor |
|-------------------|------------------------------------------------------------|
| CN                | MS-TAPI-IP-address                                         |
| LDAP-Display-Name | msTAPI-IpAddress                                           |
| Tamanho              | Cada endereço IP é armazenado como uma cadeia de caracteres em uma notação. B. C. D. |
| Privilégio de atualização  | Nenhum privilégio especial é necessário.                            |
| Frequência de atualização  | Normalmente muito baixo.                                        |
| Attribute-Id      | 1.2.840.113556.1.4.1701                                    |
| System-ID-GUID    | efd7d7f7-178e-4767-87fa-f8a16b840544                       |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                |
