| Ldap-Display-Name | printRateUnit                               |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.286                      |
| System-Id-Guid    | ba305f78-47e3-11d0-a1a6-00c04fd930c9        |
