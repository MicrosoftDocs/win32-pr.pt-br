# <a name="print-stapling-supported-attribute"></a>Grampeamento de impressão-atributo com suporte

**True** se a impressora oferecer suporte a grampeamento. Fornecido pelo driver.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Grampeamento de impressão-com suporte             |
| LDAP-Display-Name | printStaplingSupported               |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.281               |
| System-ID-GUID    | ba305f73-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Boolean**](s-boolean.md)         |
