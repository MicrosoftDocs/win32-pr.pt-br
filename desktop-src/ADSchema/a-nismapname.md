# <a name="nismapname-attribute"></a>atributo nisMapName

Contém o nome do mapa ao qual o objeto pertence.

|-------------------|--------------------------------------|
| CN                | nisMapName                           |
| LDAP-Display-Name | nisMapName                           |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.26                     |
| System-ID-GUID    | 969d3c79-0e9a-4d95-b0ac-bdde7ff8f3a1 |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
