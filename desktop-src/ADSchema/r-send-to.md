# <a name="send-to-extended-right"></a>Send-To estendido à direita

|--------------|--------------------------------------|
| CN           | Send-To                              |
| Display-Name | Enviar para                              |
| GUID de direitos  | ab721a55-1e2f-11d0-9819-00aa0040529b |
