# <a name="current-parent-ca-attribute"></a>Atributo Current-Parent-CA

Referência às autoridades de certificação que emitiram os certificados atuais para uma autoridade de certificação.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Current-Parent-CA                       |
| LDAP-Display-Name | currentParentCA                         |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.696                  |
| System-ID-GUID    | 963d273f-48be-11d1-a9c3-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
