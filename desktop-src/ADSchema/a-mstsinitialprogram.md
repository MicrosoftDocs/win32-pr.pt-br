# <a name="ms-ts-initial-program-attribute"></a>Atributo ms-TS-Initial-Program

O Programa Inicial da Sessão dos Serviços de Terminal especifica o caminho e o nome do arquivo do aplicativo que o usuário deseja iniciar automaticamente quando o usuário faz logon no Servidor do Terminal.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-TS-Initial-Program                       |
| Ldap-Display-Name | msTSInitialProgram                          |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1990                     |
| System-Id-Guid    | 9201ac6f-1d69-4dfb-802e-d95510109599        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
