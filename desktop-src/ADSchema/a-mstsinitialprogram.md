# <a name="ms-ts-initial-program-attribute"></a>atributo MS-TS-Initial-Program

Programa inicial da sessão dos serviços de terminal especifica o caminho e o nome do arquivo do aplicativo que o usuário deseja iniciar automaticamente quando o usuário faz logon no Terminal Server.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-TS-inicial-Program                       |
| LDAP-Display-Name | msTSInitialProgram                          |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1990                     |
| System-ID-GUID    | 9201ac6f-1d69-4dfb-802e-d95510109599        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
