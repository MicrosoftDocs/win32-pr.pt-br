# <a name="default-object-category-attribute"></a>Atributo de categoria de objeto-padrão

A categoria de objeto a ser usada para o objeto se um não for especificado.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Categoria de objeto-padrão                 |
| LDAP-Display-Name | defaultObjectCategory                   |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Administrador de esquema                    |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.783                  |
| System-ID-GUID    | 26d97367-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
