# <a name="ms-ds-has-domain-ncs-attribute"></a>atributo ms-DS-is-Domain-NCs

Informações de replicação do serviço de diretório que detalham os contextos de nomeação de domínio presentes em um servidor específico.



| Entrada | Valor |
|-------------------|-----------------------------------------------------|
| CN                | ms-DS-is-Domain-NCs                                |
| LDAP-Display-Name | msDS-HasDomainNCs                                   |
| Tamanho              | \-                                                  |
| Privilégio de atualização  | Esse valor é definido pelo sistema                     |
| Frequência de atualização  | Ele é definido pelo DCPromo e nunca é modificado depois. |
| Attribute-Id      | 1.2.840.113556.1.4.1820                             |
| System-ID-GUID    | 6f17e347-a842-4498-b8b3-15e007da4fed                |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md)             |
