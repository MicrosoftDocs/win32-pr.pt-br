A lista de nomes distintos de entidades de segurança concedeu esse privilégio.

| CN                | Privilege-Holder                        |
| LDAP-Display-Name | privilegeHolder                         |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.637                  |
