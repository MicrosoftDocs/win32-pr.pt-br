# <a name="user-shared-folder-other-attribute"></a>Usuário-compartilhado-pasta-outro atributo

Especifica um caminho UNC para a pasta de documentos compartilhados adicional do usuário. O caminho deve ser um caminho UNC de rede do **\\\\** diretório de compartilhamento do _servidor_ de formulário *_\\_*  *_\\_* . Esse valor pode ser uma cadeia de caracteres nula.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Usuário-compartilhado-pasta-outro                                                          |
| LDAP-Display-Name | userSharedFolderOther                                                             |
| Tamanho              | \-                                                                                |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta.                                            |
| Frequência de atualização  | Quando o registro do usuário é criado e sempre que a pasta compartilhada precisa ser alterada. |
| Attribute-Id      | 1.2.840.113556.1.4.752                                                            |
| System-ID-GUID    | 9a9a0220-4a5b-11d1-a9c3-0000f80367c1                                              |
| Sintaxe            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md)                                       |
