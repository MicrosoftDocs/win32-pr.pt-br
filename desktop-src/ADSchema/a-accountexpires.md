# <a name="account-expires-attribute"></a>Account-Expires atributo

A data em que a conta expira. Esse valor representa o número de intervalos de 100 a nanossegundos desde 1º de janeiro de 1601 (UTC). Um valor de 0 ou 0x7FFFFFFFFFFFFFFF (9223372036854775807) indica que a conta nunca expira.



| Entrada | Valor |
|-------------------|------------------------------------------------------------------------|
| CN                | Account-Expires                                                        |
| LDAP-Display-Name | accountExpires                                                         |
| Tamanho              | 8 bytes                                                                |
| Privilégio de atualização  | O administrador de domínio define esse atributo.                          |
| Frequência de atualização  | Sempre que a data de expiração anterior expirar e precisar ser atualizada. |
| Attribute-Id      | 1.2.840.113556.1.4.159                                                 |
| System-ID-GUID    | bf967915-0de6-11d0-a285-00aa003049e2                                   |
| Syntax            | [**Intervalo**](s-interval.md)                                         |

## <a name="remarks"></a>Comentários

A parte superior desse inteiro grande corresponde ao membro **dwHighDateTime** da estrutura [**FILETIME**](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) e a parte inferior corresponde ao membro **dwLowDateTime** da estrutura **FILETIME** .
