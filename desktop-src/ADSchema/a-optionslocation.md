# <a name="options-location-attribute"></a>Options-Location atributo

Para o DHCP, o local de opções contém o DN para sites alternativos que contêm as informações de opções.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Options-Location                     |
| LDAP-Display-Name | optionsLocation                      |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.713               |
| System-ID-GUID    | 963d274e-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
