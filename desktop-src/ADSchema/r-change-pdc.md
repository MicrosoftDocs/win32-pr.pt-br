# <a name="change-pdc-extended-right"></a>Change-PDC estendido à direita

Direito estendido necessário para alterar o proprietário da função FSMO do emulador do controlador de domínio primário (PDC).



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Alterar-PDC                           |
| Display-Name | Alterar PDC                           |
| GUID de direitos  | bae50096-4752-11d1-9052-00c04fc2d4cf |
