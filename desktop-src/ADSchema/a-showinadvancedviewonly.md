# <a name="show-in-advanced-view-only-attribute"></a>Mostrar-in-avançado-atributo somente exibição

**True** se esse atributo for para ser visível no modo avançado da interface do usuário.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | Mostrar-in-avançado-somente exibição                                                              |
| LDAP-Display-Name | showInAdvancedViewOnly                                                                  |
| Tamanho              | 4 bytes                                                                                 |
| Privilégio de atualização  | Administrador de domínio                                                                    |
| Frequência de atualização  | Quando o objeto é criado ou quando os aspectos de exibição do atributo precisam ser alterados. |
| Attribute-Id      | 1.2.840.113556.1.2.169                                                                  |
| System-ID-GUID    | bf967984-0de6-11d0-a285-00aa003049e2                                                    |
| Syntax            | [**Boolean**](s-boolean.md)                                                            |
