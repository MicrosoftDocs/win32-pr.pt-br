# <a name="rpc-ns-entry-flags-attribute"></a>atributo RPC-NS-entry-flags

Sinalizador para indicar que a entrada RPC NS foi criada explicitamente.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | RPC-NS-entry-flags                   |
| LDAP-Display-Name | rpcNsEntryFlags                      |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.754               |
| System-ID-GUID    | 80212841-4bdc-11d1-a9c4-0000f80367c1 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
