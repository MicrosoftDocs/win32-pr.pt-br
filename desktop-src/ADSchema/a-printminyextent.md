| CN                | Impressão-min-Y-extensão                   |
| LDAP-Display-Name | printMinYExtent                      |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.280               |
