# <a name="uidnumber-attribute"></a>atributo uidNumber

Contém um inteiro que identifica exclusivamente um usuário em um domínio administrativo.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | uidNumber                            |
| LDAP-Display-Name | uidNumber                            |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.0                      |
| System-ID-GUID    | 850fcc8f-9c6b-47e1-b671-7c654be4d5b3 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
