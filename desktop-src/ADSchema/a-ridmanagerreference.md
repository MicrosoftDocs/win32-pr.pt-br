# <a name="rid-manager-reference-attribute"></a>RID-Manager-atributo de referência

O nome distinto para o Gerenciador de RID de um objeto.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | RID-Manager-referência                   |
| LDAP-Display-Name | rIDManagerReference                     |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Esse valor é definido pelo sistema.        |
| Frequência de atualização  | Sempre que o Gerenciador de RID for alterado.       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| System-ID-GUID    | 66171886-8f3c-11d0-afda-00c04fd930c9    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
