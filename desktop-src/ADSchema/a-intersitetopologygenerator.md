# <a name="inter-site-topology-generator-attribute"></a>Atributo Inter-Site-Topology-Generator

Esse atributo será usado para dar suporte ao failover para o computador designado como aquele que é executado Knowledge Consistency Checker geração de topologia entre sites em um determinado site.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Gerador de topologia entre sites           |
| LDAP-Display-Name | interSiteTopologyGenerator              |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1246                 |
| System-ID-GUID    | b7c69e5e-2cc7-11d2-854e-00a0c983f608    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
