# <a name="msmq-site-name-attribute"></a>Atributo MSMQ-site-Name

| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MSMQ-nome do site                              |
| LDAP-Display-Name | mSMQSiteName                                |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.965                      |
| System-ID-GUID    | ffadb4b2-de39-11d1-90a5-00c04fd91ab1        |
| Syntax            | [**Cadeia de caracteres (Teletex)**](s-string-teletex.md) |
