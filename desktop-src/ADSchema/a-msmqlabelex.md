# <a name="msmq-label-ex-attribute"></a>Atributo MSMQ-Label-ex

Uma cadeia de caracteres que descreve o tipo de serviço fornecido pela fila.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Label-ex                               |
| LDAP-Display-Name | mSMQLabelEx                                 |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1415                     |
| System-ID-GUID    | 4580ad25-d407-48d2-ad24-43e6e56793d7        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
