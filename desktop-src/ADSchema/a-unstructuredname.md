# <a name="unstructuredname-attribute"></a>atributo unstructurename

O nome DNS do roteador (por exemplo: router1.fabrikam.com).



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | unstructurename                     |
| LDAP-Display-Name | unstructurename                     |
| Tamanho              | \-                                   |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113549.1.9.2                 |
| System-ID-GUID    | 9c8ef177-41cf-45c9-9673-7716c0c8901b |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)  |
