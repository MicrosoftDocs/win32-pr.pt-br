# <a name="phone-mobile-other-attribute"></a>Telefone-outro atributo de dispositivo móvel

Uma lista de números de telefone celular alternativos.



| Entrada | Valor |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telefone-celular-outro                                                               |
| LDAP-Display-Name | otherMobile                                                                      |
| Tamanho              | \-                                                                               |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta.                                           |
| Frequência de atualização  | Quando o registro do usuário é criado e sempre que o número de telefone precisa ser alterado. |
| Attribute-Id      | 1.2.840.113556.1.4.647                                                           |
| System-ID-GUID    | 0296c11e-40da-11d1-a9c0-0000f80367c1                                             |
