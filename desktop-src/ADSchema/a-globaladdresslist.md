# <a name="global-address-list-attribute"></a>Atributo de lista de endereços global

Esse atributo é usado em um contêiner do Microsoft Exchange para armazenar o nome distinto de uma GAL (lista de endereços global) recém-criada. Esse atributo deve ter uma entrada antes que você possa habilitar clientes Messaging Application Programming Interface (MAPI) para usar uma GAL.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Lista de endereços global                     |
| LDAP-Display-Name | globalAddressList                       |
| Tamanho              | \-                                      |
| Privilégio de atualização  | \-                                      |
| Frequência de atualização  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1245                 |
| System-ID-GUID    | f754c748-06f4-11d2-aa53-00c04fd7d83a    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
