# <a name="ms-ds-az-application-data-attribute"></a>atributo ms-DS-AZ-Application-Data

Uma cadeia de caracteres que é usada por aplicativos individuais para armazenar quaisquer informações que possam precisar.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-AZ-Application-Data                   |
| LDAP-Display-Name | msDS-AzApplicationData                      |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1819                     |
| System-ID-GUID    | 503fc3e8-1cc6-461a-99a3-9eee04f402a7        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
