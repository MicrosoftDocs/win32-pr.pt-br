# <a name="msmq-owner-id-attribute"></a>Atributo MSMQ-Owner-ID

O atributo **MSMQ-Owner-ID** contém informações de modo misto do MSMQ.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Owner-ID                                         |
| LDAP-Display-Name | mSMQOwnerID                                           |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.925                                |
| System-ID-GUID    | 9a0dc328-c100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
