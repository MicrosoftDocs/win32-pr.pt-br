# <a name="migrate-sid-history-extended-right"></a>Migrar-SID-History estendido à direita

Direito estendido que permite que um usuário migre a SID-History sem privilégios de administrador.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Migrar-SID-History                  |
| Display-Name | Migrar Histórico do SID                  |
| GUID de direitos  | ba33815a-4f93-4c76-87f3-57574bff8109 |
