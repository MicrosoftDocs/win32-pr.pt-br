# <a name="version-number-hi-attribute"></a>Número de versão – atributo Hi

Um número de versão principal de uso geral.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Versão-número-Olá                    |
| LDAP-Display-Name | versionNumberHi                      |
| Tamanho              | 4 bytes                              |
| Privilégio de atualização  | \-                                   |
| Frequência de atualização  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.328               |
| System-ID-GUID    | 7d6c0e9a-7e20-11d0-afd6-00c04fd930c9 |
| Syntax            | [**Enumeração**](s-enumeration.md) |
