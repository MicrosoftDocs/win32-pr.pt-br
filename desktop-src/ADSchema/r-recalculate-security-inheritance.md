# <a name="recalculate-security-inheritance-extended-right"></a>Recalcular-segurança-herança estendida à direita

Direito estendido necessário para forçar o DS a recalcular a herança de ACL em um contexto de nomenclatura.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Recalcular-segurança-herança     |
| Display-Name | Recalcular a herança de segurança     |
| GUID de direitos  | 62dd28a8-7f46-11d2-b9ad-00c04f79f805 |
