|-------------------|---------------------------------------------|
| CN                | FRS-controle-Data-criação                   |
| LDAP-Display-Name | fRSControlDataCreation                      |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.871                      |
