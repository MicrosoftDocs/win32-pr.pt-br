# <a name="display-name-printable-attribute"></a>Nome de exibição-atributo imprimível

O nome de exibição imprimível de um objeto. O nome de exibição imprimível geralmente é a combinação do nome do usuário, do início do meio e do sobrenome.



| Entrada | Valor |
|-------------------|----------------------------------------|
| CN                | Nome de exibição-imprimível                 |
| LDAP-Display-Name | displayNamePrintable                   |
| Tamanho              | \-                                     |
| Privilégio de atualização  | Administrador de domínio ou proprietário da conta. |
| Frequência de atualização  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-ID-GUID    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Syntax            | [**Cadeia de caracteres (IA5)**](s-string-ia5.md)    |
