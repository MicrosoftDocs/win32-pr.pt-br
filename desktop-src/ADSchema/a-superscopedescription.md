# <a name="super-scope-description-attribute"></a>Atributo de descrição de super escopo

Esse atributo fornece uma descrição para um superescopo.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Super escopo-descrição                     |
| LDAP-Display-Name | superScopeDescription                       |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.711                      |
| System-ID-GUID    | 963d274c-48be-11d1-a9c3-0000f80367c1        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
