# <a name="do-garbage-collection-extended-right"></a>Do-lixo-coleta estendida à direita

Direito estendido para forçar o serviço de diretório a fazer a coleta de lixo. Direito de controle para forçar o serviço de diretório a fazer a coleta de lixo.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Coleta de lixo                |
| Display-Name | Fazer coleta de lixo                |
| GUID de direitos  | fec364e0-0a98-11d1-adbb-00c04fd8d5cd |
