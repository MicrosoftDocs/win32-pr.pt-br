# <a name="transport-type-attribute"></a>Transport-Type atributo

O nome distinto para um tipo de transporte que está sendo usado para conectar sites. Esse valor pode apontar para um transporte IP ou SMTP.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | Transport-Type                          |
| LDAP-Display-Name | transportType                           |
| Tamanho              | \-                                      |
| Privilégio de atualização  | Esse valor é definido pelo sistema.        |
| Frequência de atualização  | Ao conectar sites.                  |
| Attribute-Id      | 1.2.840.113556.1.4.791                  |
| System-ID-GUID    | 26d97374-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Objeto (DS-DN)**](s-object-ds-dn.md) |
