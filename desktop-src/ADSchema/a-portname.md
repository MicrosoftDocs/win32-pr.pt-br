# <a name="port-name-attribute"></a>Port-Name atributo

Lista de nomes de porta. Por exemplo, para portas de impressora ou de comunicação.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Port-Name                                   |
| LDAP-Display-Name | portName                                    |
| Tamanho              | \-                                          |
| Privilégio de atualização  | \-                                          |
| Frequência de atualização  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.228                      |
| System-ID-GUID    | 281416c4-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**Cadeia de caracteres (Unicode)**](s-string-unicode.md) |
