# <a name="domain-wide-policy-attribute"></a>Atributo de diretiva de todo o domínio

Isso é para que a política extensível do usuário seja replicada para os clientes.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Política de todo o domínio                                    |
| LDAP-Display-Name | domainWidePolicy                                      |
| Tamanho              | \-                                                    |
| Privilégio de atualização  | \-                                                    |
| Frequência de atualização  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.421                                |
| System-ID-GUID    | 80a67e29-9f22-11d0-afdd-00c04fd930c9                  |
| Syntax            | [**Objeto (link de réplica)**](s-object-replica-link.md) |
