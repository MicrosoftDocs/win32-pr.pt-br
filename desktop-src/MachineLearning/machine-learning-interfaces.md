# <a name="windows-machine-learning-interfaces"></a>Interfaces de Machine Learning do Windows

Os tópicos a seguir descrevem as interfaces COM do SDK do Windows Machine Learning.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                             | Descrição                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWinMLEvaluationContext**](/windows/win32/api/winml/nn-winml-iwinmlevaluationcontext)<br/> | Representa o contexto para associar entradas e saídas a um modelo WinML.<br/>                                                                                                             |
| [**IWinMLModel**](/windows/win32/api/winml/nn-winml-iwinmlmodel)<br/>                         | Representa um modelo de Machine Learning do Windows com metadados correspondentes; inclui descrições de modelo (nome, autor, controle de versão, etc.), bem como entradas e saídas esperadas. <br/> |
| [**IWinMLRuntime**](/windows/win32/api/winml/nn-winml-iwinmlruntime)<br/>                     | Representa o tempo de execução para carregar e avaliar um modelo WinML.<br/>                                                                                                                      |
| [**IWinMLRuntimeFactory**](/windows/win32/api/winml/nn-winml-iwinmlruntimefactory)<br/>       | Representa a fábrica que cria o tempo de execução WinML para carregamento e avaliação de modelo.<br/>                                                                                         |
