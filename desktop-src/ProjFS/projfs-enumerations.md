# <a name="projfs-enumerations"></a>Enumerações ProjFS

As enumerações a seguir são declaradas em projectedfslib. h.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**PRJ_CALLBACK_DATA_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_callback_data_flags) | Sinalizadores que controlam o que é retornado na enumeração. |
| [**PRJ_COMPLETE_COMMAND_TYPE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_complete_command_type) | Especifica tipos de comando. |
| [**PRJ_FILE_STATE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_file_state) | O estado de um item. |
| [**PRJ_NOTIFICATION**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notification) | Um valor de notificação especificado ao enviar a notificação em um retorno de chamada. |
| [**PRJ_NOTIFY_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notify_types) | Tipos de notificações que descrevem uma alteração no arquivo ou na pasta. |
| [**PRJ_PLACEHOLDER_ID**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_placeholder_id) | Define o comprimento de um identificador de espaço reservado. |
| [**PRJ_STARTVIRTUALIZING_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_startvirtualizing_flags) | Sinalizadores a serem fornecidos ao iniciar uma instância de virtualização. |
| [**PRJ_UPDATE_FAILURE_CAUSES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_failure_causes) | Descrições da razão pela qual uma atualização falhou. |
| [**PRJ_UPDATE_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_types) | Sinalizadores para especificar se as atualizações serão permitidas, dado o estado de um arquivo ou diretório no disco. |
