# <a name="event-logging-reference"></a>Referência de log de eventos

A referência do Log de Eventos é dividida nas seguintes seções:

-   [Funções de registro em log de eventos](event-logging-functions.md)
-   [Estruturas de registro em log de eventos](event-logging-structures.md)

> [!Note]  
> A API de Log de Eventos foi projetada para aplicativos executados no Windows Server 2003, Windows XP ou Windows sistema operacional 2000. No Windows Vista, a infraestrutura de log de eventos foi reprojetada. Os aplicativos projetados para executar no Windows Vista ou sistemas operacionais posteriores devem usar Windows Log de [Eventos](/windows/desktop/WES/windows-event-log) para registrar eventos.
