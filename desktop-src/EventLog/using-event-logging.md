# <a name="using-event-logging"></a>Usando o log de eventos

Os aplicativos podem usar a API de log de eventos para relatar e exibir eventos. Para obter detalhes sobre eventos de relatório, consulte [Reporting Events](reporting-an-event.md).

Para exibir eventos que foram gravados em arquivos de log, consulte [consultando mensagens de origem do evento](querying-for-event-source-messages.md) e [recebendo notificação de eventos](receiving-event-notification.md). Você também pode usar o Visualizador de Eventos para exibir eventos.
