# <a name="access-permission-flags"></a>Sinalizadores de permissão de acesso

Sinalizadores de permissão de acesso especificam o tipo de acesso que um aplicativo cliente tem a um objeto aberto: leitura, gravação ou leitura/gravação. A permissão de leitura permite que um aplicativo Leia o conteúdo de um fluxo, mas não grave alterações no fluxo. A permissão de gravação permite que um aplicativo chame uma função que confirme as alterações no armazenamento de um objeto, mas não leia o conteúdo do fluxo. A permissão de leitura/gravação permite que um aplicativo Leia os fluxos do objeto ou grave no armazenamento do objeto.
