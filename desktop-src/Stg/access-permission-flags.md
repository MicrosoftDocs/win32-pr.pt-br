# <a name="access-permission-flags"></a>Sinalizadores de permissão de acesso

Sinalizadores de permissão de acesso especificam o tipo de acesso que um aplicativo cliente tem a um objeto aberto: leitura, gravação ou leitura/gravação. A permissão de leitura permite que um aplicativo leia o conteúdo de um fluxo, mas não escreva alterações no fluxo. A permissão de gravação permite que um aplicativo chame uma função que confirma as alterações no armazenamento de um objeto, mas não para ler o conteúdo do fluxo. A permissão de leitura/gravação permite que um aplicativo leia os fluxos do objeto ou escreva no armazenamento do objeto.
