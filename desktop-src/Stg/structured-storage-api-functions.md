# <a name="structured-storage-api-functions"></a>Funções de API de armazenamento estruturado

Algumas das funções de API para o armazenamento estruturado são funções auxiliares que executam uma sequência de chamadas para outras funções de API e métodos de interface. Você pode usar essas funções auxiliares como pequenos cortes.

Outras funções de API fornecem acesso à implementação do COM de armazenamento estruturado, arquivos compostos.

Ainda outro conjunto de funções de API permite que você converta objetos OLE 1 em armazenamento estruturado. Você pode usar essas funções para determinar se uma classe de objeto é proveniente de OLE 1 e converter objetos entre os formatos de armazenamento COM OLE 1 e atual.

Por fim, há funções de API para converter e emular outros objetos. Essas funções fornecem serviços para um aplicativo de servidor trabalhar com dados de outro aplicativo. Os dados podem ser convertidos para o formato nativo do aplicativo de servidor lendo os dados de seu formato original, mas gravando-os no formato nativo do aplicativo de objeto. Ou os dados podem permanecer em seu formato original enquanto o aplicativo de servidor lê e grava os dados em seu formato original.

Para obter mais informações, consulte a seção a seguir [modos de acesso estruturado de armazenamento](structured-storage-access-modes.md).
