# <a name="extensions"></a>Extensões

As extensões do conjunto de propriedades, conforme definido no formato original do conjunto de propriedades COM, foram removidas e não têm suporte, exceto aquelas nos [conjuntos de propriedades DocumentSummaryInformation e UserDefined](the-documentsummaryinformation-and-userdefined-property-sets.md).
