# <a name="reserved"></a>Reservado

O valor do tipo de dados **DWORD** é reservado para uso futuro. Os gravadores dos conjuntos de propriedades devem definir esse valor como 1; os leitores dos conjuntos de propriedades devem garantir que esse valor seja pelo menos 1. Uma exceção a essa diretriz é que, para [os conjuntos de propriedades DocumentSummaryInformation e UserDefined](the-documentsummaryinformation-and-userdefined-property-sets.md), esse valor pode ser 2.
