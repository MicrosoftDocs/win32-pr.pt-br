# <a name="byte-order-indicator"></a>Indicador de Byte-Order

O indicador de ordem de byte é um tipo de dados do **Word** e deve sempre conter o valor 0xFFFE. Isso é o mesmo que o indicador de ordem de byte Unicode. Esse valor é sempre gravado na ordem de bytes da Intel e, portanto, aparece no arquivo ou no fluxo como 0xFE, 0xFF.
