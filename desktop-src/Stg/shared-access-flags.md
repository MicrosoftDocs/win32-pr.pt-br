# <a name="shared-access-flags"></a>Sinalizadores de acesso compartilhado

Os sinalizadores de acesso compartilhado especificam o grau de acesso que outros aplicativos podem ter com um objeto que seu aplicativo está abrindo. Você pode negar acesso de leitura, negar acesso de gravação ou negar todo o acesso. Você também pode especificar explicitamente que nenhum tipo de acesso seja negado.
