# <a name="storage-creation-flags"></a>Armazenamento Sinalizadores de criação

Armazenamento de criação especificam o que o COM deve fazer se um objeto de armazenamento, fluxo ou lockbytes existente tiver o mesmo nome que um novo objeto de armazenamento ou fluxo que você está criando. O padrão é retornar uma mensagem de erro e não criar o novo objeto. Você pode usar apenas um desses sinalizadores em uma determinada chamada de criação.
