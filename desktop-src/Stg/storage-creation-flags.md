# <a name="storage-creation-flags"></a>Sinalizadores de criação de armazenamento

Os sinalizadores de criação de armazenamento especificam o que o deve fazer se um objeto de armazenamento, fluxo ou LockBytes existente tiver o mesmo nome que um novo objeto de armazenamento ou de fluxo que você está criando. O padrão é retornar uma mensagem de erro e não criar o novo objeto. Você pode usar apenas um desses sinalizadores em uma determinada chamada de criação.
