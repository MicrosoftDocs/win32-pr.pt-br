# <a name="structured-storage-fundamentals"></a>Conceitos Armazenamento estruturadas

A Armazenamento estruturada fornece o equivalente a um sistema de arquivos completo para armazenar objetos por meio dos elementos listados na tabela a seguir.



| Elemento                                                                  | Descrição                                                                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaces Armazenamento estruturadas](structured-storage-interfaces.md)       | As interfaces [**IStorage**](/windows/desktop/api/Objidl/nn-objidl-istorage), [**IStream,**](/windows/desktop/api/Objidl/nn-objidl-istream) [**ILockBytes**](/windows/desktop/api/Objidl/nn-objidl-ilockbytes)e [**IRootStorage,**](/windows/desktop/api/Objidl/nn-objidl-irootstorage) juntamente com um conjunto de interfaces relacionadas —[**IPersistStorage,**](/windows/win32/api/objidl/nn-objidl-ipersiststorage) [**IPersist,**](/windows/win32/api/objidl/nn-objidl-ipersist) [**IPersistFile**](/windows/win32/api/objidl/nn-objidl-ipersistfile)e [**IPersistStream.**](/windows/win32/api/objidl/nn-objidl-ipersiststream) |
| [Funções de API Armazenamento estruturadas](structured-storage-api-functions.md) | APIs auxiliares que facilitam uma implementação de armazenamento estruturado, bem como um segundo conjunto de APIs para arquivos compostos; que é a implementação COM das interfaces de armazenamento estruturadas. O COM também fornece um conjunto de estruturas de suporte e valores de enumeração usados para organizar parâmetros para métodos de interface e APIs.                              |
| [Modos Armazenamento de acesso estruturados](structured-storage-access-modes.md)   | Um conjunto de modos de acesso para regulamentar o acesso a arquivos compostos.                                                                                                                                                                                                                                                                                                 |
