# <a name="enumerations-windows-update-agent"></a>enumerações (agente de Windows Update)

enumerações em Armazenamento estruturados são tipos de dados que consistem em uma sequência de valores nomeados em uma determinada ordem. Um tipo de dados enumerado permite que um membro de uma classe de objetos seja descrito em linguagem natural. As enumerações são mais úteis no contexto em que todos os valores possíveis de uma variável são conhecidos e a expressão dessa variável na linguagem de programação pode, portanto, ser expressa em palavras de idioma natural.

as enumerações Armazenamento estruturadas são descritas nas seções a seguir.

-   [**LOCKTYPE**](/windows/win32/api/objidl/ne-objidl-locktype)
-   [**STATFLAG**](/windows/win32/api/wtypes/ne-wtypes-statflag)
-   [**STGC**](/windows/win32/api/wtypes/ne-wtypes-stgc)
-   [**STGFMT**](/previous-versions/windows/desktop/legacy/aa380330(v=vs.85))
-   [**STGMOVE**](/windows/win32/api/wtypes/ne-wtypes-stgmove)
-   [**STGTY**](/windows/win32/api/objidl/ne-objidl-stgty)
-   [**pesquisa de fluxo \_**](/windows/win32/api/objidl/ne-objidl-stream_seek)
