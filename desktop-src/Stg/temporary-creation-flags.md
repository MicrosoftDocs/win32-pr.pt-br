# <a name="temporary-creation-flags"></a>Sinalizadores de criação temporários

O sinalizador de criação temporária indica que o arquivo subjacente deve ser destruído automaticamente quando o objeto de armazenamento raiz é liberado. Esse recurso é mais útil para a criação de arquivos temporários.
