# <a name="dictionary"></a>Dicionário

[**IPropertyStorage::WritePropertyNames**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writepropertynames) é implementado usando o dicionário [ID da propriedade 0](/windows/desktop/Stg/reserved-property-identifiers). A ID da propriedade 0 não está acessível por [**meio de IPropertyStorage::ReadMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-readmultiple) ou [**IPropertyStorage::WriteMultiple.**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writemultiple)
