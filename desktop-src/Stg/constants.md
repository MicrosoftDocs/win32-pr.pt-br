# <a name="constants-structured-storage"></a>Constantes (armazenamento estruturado)

As constantes, no armazenamento estruturado, são uma sequência de valores nomeados. As constantes permitem que os objetos sejam descritos em linguagem natural.

As constantes de armazenamento estruturado são descritas nas seções a seguir:

-   [Constantes PROPSETFLAG](propsetflag-constants.md)
-   [Constantes STGM](stgm-constants.md)
