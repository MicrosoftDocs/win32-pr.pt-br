# <a name="count-of-properties"></a>Contagem de propriedades

Esse tipo de dados **DWORD** fornece uma contagem dos valores de propriedade na seção. Um conjunto de propriedades pode conter qualquer número de valores de propriedade. Os leitores devem ser capazes de lidar com o caso em que não há nenhuma propriedade.
