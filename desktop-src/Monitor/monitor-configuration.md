# <a name="monitor-configuration-monitor-configuration"></a>Configuração do monitor (configuração do monitor)

Você pode usar as funções de configuração do monitor para obter informações de um monitor e alterar as configurações de um monitor. Você pode usar essas funções para:

-   Alterar as configurações de geometria de um monitor, como a largura e a altura da exibição.
-   Altere as configurações de imagem, como brilho e contraste.
-   Redefinir as configurações de um monitor para seus padrões de fábrica.
-   Degauss o monitor.

Internamente, as funções de configuração do monitor usam a interface de comando exibir canal de dados (DDC/CI) para enviar comandos para o monitor.

As funções de configuração do monitor são documentadas nas seções a seguir:

-   [Sobre a configuração do monitor](about-monitor-configuration.md)
-   [Usando a configuração do monitor](using-monitor-configuration.md)
-   [Referência de configuração do monitor](monitor-configuration-reference.md)
