# <a name="monitor-configuration-enumeration-types"></a>Monitorar tipos de enumeração de configuração

As enumerações a seguir são usadas pelas funções de configuração do monitor.



| Tipo de enumeração                                                    | Descrição                                                                                      |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**\_temperatura de cor de MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_color_temperature)              | Descreve a temperatura de cor do monitor.                                                         |
| [**\_tipo de \_ tecnologia de vídeo MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_display_technology_type) | Identifica as tecnologias de exibição do monitor.                                                         |
| [**\_tipo de unidade MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_drive_type)                            | Especifica se é para definir ou obter a unidade vermelha, verde ou azul de um monitor.                           |
| [**\_tipo de lucro MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_gain_type)                              | Especifica se deve ser obtido ou definido o lucro vermelho, verde ou azul de um monitor.                            |
| [**\_tipo de posição MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_position_type)                      | Especifica se é para obter ou definir a posição vertical ou horizontal da área de exibição de um monitor. |
| [**\_tipo de tamanho de MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_size_type)                              | Especifica se é para obter ou definir a largura ou a altura da área de exibição de um monitor.                 |
| [**tipo de código do MC \_ VCP \_ \_**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ne-lowlevelmonitorconfigurationapi-mc_vcp_code_type)                     | Descreve um tipo de código VCP (painel de controle virtual).                                               |
