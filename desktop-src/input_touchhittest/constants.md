# <a name="touch-hit-testing-constants"></a>Constantes de teste de toque

Os tópicos nesta seção fornecem as especificações de referência para constantes usadas pelas APIs [de Teste de Toque.](touchhittest-reference.md)

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
| --- | --- |
| [**Comportamentos de teste de toque de toque**](hit-testing-behaviors.md) | As constantes a seguir são usadas por aplicativos ou estruturas de interface do usuário para identificar como as mensagens para teste de toque são processadas por janelas registradas por meio da [**função RegisterTouchHitTestingWindow.**](/windows/win32/api/winuser/nf-winuser-registertouchhittestingwindow) |
| [**Pontuações de teste de toque de toque**](hit-testing-scores.md)       | As constantes a seguir identificam as pontuações de teste de acerto possíveis para um objeto, em relação a outros objetos que intersectam a área de contato de toque. |
