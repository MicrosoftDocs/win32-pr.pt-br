# <a name="touch-hit-testing-constants"></a>Constantes de teste de clique de toque

Os tópicos nesta seção fornecem as especificações de referência para constantes usadas pelas APIs de [teste de colisão de toque](touchhittest-reference.md) .

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
| --- | --- |
| [**Comportamentos de teste de colisão de toque**](hit-testing-behaviors.md) | As constantes a seguir são usadas por aplicativos ou estruturas de interface do usuário para identificar como as mensagens de teste de toque são processadas pelo Windows que são registradas por meio da função [**RegisterTouchHitTestingWindow**](/windows/win32/api/winuser/nf-winuser-registertouchhittestingwindow) . |
| [**Pontuações de teste de colisão de toque**](hit-testing-scores.md)       | As constantes a seguir identificam as possíveis pontuações de teste de clique para um objeto, em relação a outros objetos que interseccionam a área de contato de toque. |
