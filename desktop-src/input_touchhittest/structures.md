# <a name="touch-hit-testing-structures"></a>Estruturas de teste de colisão de toque

Os tópicos nesta seção fornecem as especificações de referência para estruturas de [teste de colisão de toque](touch-hit-testing-portal.md) .

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | Contém informações sobre a área de contato de toque relatada pelo digitalizador de toque.<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | Contém a pontuação de teste de clique que indica se o objeto é o destino provável da área de contato de toque, em relação a outros objetos que interseccionam a área de contato de toque.<br/> |
