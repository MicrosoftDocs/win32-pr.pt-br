# <a name="general-purpose-functions-cryptography"></a>Funções de Uso Geral (criptografia)

As seguintes funções de exemplo de finalidade geral executam várias tarefas auxiliares nos programas de exemplo:



| Função                               | Descrição                                                              |
|----------------------------------------|--------------------------------------------------------------------------|
| [**ByteToStr**](bytetostr.md)         | Converte uma matriz de bytes em uma cadeia de caracteres.                  |
| [**GetSignerCert**](getsignercert.md) | Recupera um certificado de um repositório que tem uma extensão de assinatura de chave. |
| [**MyHandleError**](myhandleerror.md) | Grava uma mensagem de erro e sai de um programa.                             |
