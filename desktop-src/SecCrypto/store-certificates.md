# <a name="storecertificates-property"></a>Propriedade Store. Certificates

\[A propriedade **Certificates** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Em vez disso, use a [**classe X509Store**](/previous-versions/windows/embedded/hh424027(v=msdn.10)) no namespace [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) .\]

A propriedade **Certificates** recupera a coleção que inclui todos os certificados no repositório. Essa é a propriedade padrão.

## <a name="syntax"></a>Syntax


```VB
Store.Certificates As Certificates
```



## <a name="property-value"></a>Valor da propriedade

A coleção de certificados no repositório.
