# <a name="publickeylength-property"></a>Propriedade PublicKey. Length

\[A propriedade **Length** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Em vez disso, use a [**Propriedade X509Certificate2. PublicKey**](/dotnet/api/system.security.cryptography.x509certificates.x509certificate2.publickey?view=netcore-3.1) no namespace [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) .\]

A propriedade **Length** recupera o comprimento da chave pública em bits.

## <a name="syntax"></a>Syntax


```VB
PublicKey.Length As Long
```



## <a name="property-value"></a>Valor da propriedade

O comprimento da chave pública em bits.
