# <a name="noticenumberscount-property"></a>Propriedade NoticeNumbers. Count

\[A propriedade **Count** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Para obter mais informações, consulte [**qualificador**](qualifier.md).\]

A propriedade **Count** recupera o número de objetos de [**extensão**](extension.md) na coleção.

## <a name="syntax"></a>Syntax


```VB
NoticeNumbers.Count As Long
```



## <a name="property-value"></a>Valor da propriedade

O número de objetos de [**extensão**](extension.md) na coleção.
