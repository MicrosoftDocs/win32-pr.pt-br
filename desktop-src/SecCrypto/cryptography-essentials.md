# <a name="cryptography-essentials"></a>Cryptography Essentials

O Microsoft CryptoAPI, a API (interface de programação de aplicativos) e o CAPICOM fornecem serviços que permitem aos desenvolvedores adicionar segurança com base [*na criptografia*](../secgloss/c-gly.md) aos aplicativos. O CryptoAPI inclui a funcionalidade de codificação e decodificação do ASN.1 [*(Abstract Syntax Notation One),*](../secgloss/a-gly.md) criptografia e descriptografia de dados, autenticação usando [*certificados*](../secgloss/d-gly.md)digitais e gerenciamento de certificado digital usando armazenamentos de certificados. CryptoAPI e CAPICOM são suporte para criptografia de chave simétrica e PKI.

O Cryptography Essentials contém os seguintes tópicos:

-   [Criptografia, CryptoAPI e CAPICOM](cryptography--cryptoapi--and-capicom.md)
-   [Chaves criptográficas](cryptographic-keys.md)
-   [Dados codificados e decodificados](encoded-and-decoded-data.md)
-   [Criptografia e descriptografia de dados](data-encryption-and-decryption.md)
-   [Hashes e assinaturas digitais](hashes-and-digital-signatures.md)
-   [Carimbo de data/hora de assinaturas Authenticode](time-stamping-authenticode-signatures.md)
-   [Certificados Digitais](digital-certificates.md)
-   [Mensagens digitais](digital-messages.md)
