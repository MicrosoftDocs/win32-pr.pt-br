# <a name="oidscount-property"></a>Propriedade OIDs. Count

\[A propriedade **Count** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Em vez disso, use a [**classe OidCollection**](/dotnet/api/system.security.cryptography.oidcollection?view=netcore-3.1) no namespace [**System. Security. Cryptography**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

A propriedade **Count** recupera o número de OIDs na coleção.

## <a name="syntax"></a>Syntax


```VB
OIDs.Count As Long
```



## <a name="property-value"></a>Valor da propriedade

Número de objetos [**OID**](oid.md) na coleção.
