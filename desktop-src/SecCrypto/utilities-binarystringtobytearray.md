# <a name="utilitiesbinarystringtobytearray-method"></a>Método Utilities. BinaryStringToByteArray

\[O método **BinaryStringToByteArray** está disponível para uso nos sistemas operacionais especificados na seção requisitos.\]

O método **BinaryStringToByteArray** converte uma cadeia de caracteres de pacote binário em uma matriz de bytes.

## <a name="syntax"></a>Sintaxe


```VB
Utilities.BinaryStringToByteArray( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Binarystring* \[ no\]
</dt> <dd>

A cadeia de caracteres de pacote binário a ser convertida em uma matriz de bytes.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

A matriz que contém o resultado da conversão.
