# <a name="prov_fortezza"></a>PROV \_ FORTEZZA

O tipo de provedor PROV FORTEZZA contém um conjunto de protocolos criptográficos e \_ algoritmos pertencentes ao NIST [*(National Institute of Standards and Technology).*](../secgloss/n-gly.md)

## <a name="algorithms-supported"></a>Algoritmos com suporte

Para ver descrições de cada um desses algoritmos, confira o glossário.



| Finalidade      | Algoritmos compatíveis                                                                                      |
|--------------|-----------------------------------------------------------------------------------------------------------|
| Chave Exchange | [*Kea*](../secgloss/k-gly.md)         |
| Assinatura    | [*Dss*](../secgloss/d-gly.md) |
| Criptografia   | [*Skipjack*](../secgloss/s-gly.md)                                |
| Hash      | [*SHA*](../secgloss/s-gly.md)           |
