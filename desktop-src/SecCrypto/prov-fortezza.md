# <a name="prov_fortezza"></a>PROV \_ Fortezza

O \_ tipo de provedor do Prov Fortezza contém um conjunto de protocolos criptográficos e algoritmos pertencentes ao [*National Institute of Standards and Technology*](../secgloss/n-gly.md) (NIST).

## <a name="algorithms-supported"></a>Algoritmos com suporte

Para obter descrições de cada um desses algoritmos, consulte o Glossário.



| Finalidade      | Algoritmos compatíveis                                                                                      |
|--------------|-----------------------------------------------------------------------------------------------------------|
| Troca de chaves | [*KEA*](../secgloss/k-gly.md)         |
| Assinatura    | [*DSS*](../secgloss/d-gly.md) |
| Criptografia   | [*Skipjack*](../secgloss/s-gly.md)                                |
| Hash      | [*SHA*](../secgloss/s-gly.md)           |

Mais informações estão disponíveis diretamente do [*National Institute of Standards and Technology*](../secgloss/n-gly.md) (NIST).
