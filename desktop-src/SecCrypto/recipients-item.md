# <a name="recipientsitem-property"></a>Propriedade Recipients. Item

\[A propriedade **Item** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Em vez disso, use a [**classe CmsRecipientCollection**](/dotnet/api/system.security.cryptography.pkcs.cmsrecipientcollection?view=dotnet-plat-ext-3.1&preserve-view=true) no namespace [**System. Security. Cryptography. Pkcs**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

A propriedade **Item** recupera um objeto da coleção [**Recipients**](recipients.md) . Essa é a propriedade padrão.

## <a name="syntax"></a>Syntax


```VB
Recipients.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>Valor da propriedade

Uma variante que representa o item indexado na coleção de [**destinatários**](recipients.md) .
