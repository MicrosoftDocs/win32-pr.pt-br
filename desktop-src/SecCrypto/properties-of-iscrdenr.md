# <a name="properties-of-iscrdenr"></a>Propriedades de ISCrdEnr

As propriedades introduzidas por ISCrdEnr estão listadas na tabela a seguir.



| Propriedade                              | Descrição                                                                                                                                                                                              |
|---------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CSPCount**](iscrdenr-cspcount.md) | Especifica o número de [*provedores de serviços de criptografia*](../secgloss/c-gly.md) (CSP). Esta propriedade é somente para leitura. |
| [**CSPName**](iscrdenr-cspname.md)   | Nome do CSP. Esta propriedade é de leitura/gravação.                                                                                                                                                            |
