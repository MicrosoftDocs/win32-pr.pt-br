# <a name="methods-of-ienroll4"></a>Métodos de IEnroll4

Os métodos a seguir são definidos pela interface [**IEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll4) ou são métodos que a interface **IEnroll4** herda de [**IEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll) e [**IEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll2). Os métodos de acesso à propriedade não são mostrados aqui. Para ver as propriedades de **IEnroll4**, consulte [Propriedades de IEnroll4](properties-of-ienroll4.md).



| Método                                                                                                | Descrição                                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**acceptFilePKCS7WStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-acceptfilepkcs7wstr)                                           | Aceita e processa uma \# mensagem PKCS 7 contendo um certificado e, em seguida, armazena a mensagem em um arquivo.                                                                                                                                                                                                 |
| [**acceptFileResponseWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-acceptfileresponsewstr)                                     | Aceita a entrega das credenciais emitidas em resposta a uma chamada anterior para [**createFileRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createfilerequestwstr) e coloca as credenciais no repositório apropriado.                                                                                                     |
| [**acceptPKCS7Blob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-acceptpkcs7blob)                                                   | Aceita e processa uma \# mensagem PKCS 7 contendo um certificado. | [**acceptResponseBlob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-acceptresponseblob)                                             | Aceita a entrega das credenciais emitidas em resposta a uma chamada anterior para [**IEnroll4:: createRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createrequestwstr) e coloca as credenciais no repositório apropriado.                                                                                                   |
| [**addAttributeToRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addattributetorequestwstr)                               | Adiciona um atributo à solicitação de certificado.                                                                                                                                                                                                                                                         |
| [**AddAuthenticatedAttributesToPKCS7Request**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-addauthenticatedattributestopkcs7request) | Adiciona atributos autenticados a uma \# solicitação de certificado PKCS 7.                                                                                                                                                                                                                                      |
| [**addBlobPropertyToCertificateWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addblobpropertytocertificatewstr)                 | Adiciona uma propriedade de BLOB a um certificado.                                                                                                                                                                                                                                                                |
| [**AddCertTypeToRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-addcerttypetorequestwstr)                                 | Adiciona um modelo de certificado a uma solicitação (usada para dar suporte à AC (autoridade de certificação) corporativa).                                                                                                                                                                                               |
| [**AddCertTypeToRequestWStrEx**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addcerttypetorequestwstrex)                             | Adiciona um modelo de certificado (ou "tipo de certificado") a uma solicitação.                                                                                                                                                                                                                                     |
| [**AddExtensionsToRequest**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-addextensionstorequest)                                     | Adiciona extensões à solicitação de certificado.                                                                                                                                                                                                                                                           |
| [**addExtensionToRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-addextensiontorequestwstr)                               | Adiciona uma extensão à solicitação.                                                                                                                                                                                                                                                                     |
| [**addNameValuePairToRequestWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll4-addnamevaluepairtorequestwstr)                       | Adiciona um par de cadeia de caracteres de valor de nome à solicitação.                                                                                                                                                                                                                                                         |
| [**AddNameValuePairToSignatureWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-addnamevaluepairtosignaturewstr)                   | Adiciona o par de nome e valor de um atributo à solicitação. Cabe à CA interpretar o significado do par nome-valor.                                                                                                                                                                      |
| [**binaryBlobToString**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-binaryblobtostring)                                             | Converte um BLOB de dados binários em uma cadeia de caracteres.                                                                                                                                                                                                                                                              |

| [**createFilePKCS10WStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-createfilepkcs10wstr)                                         | Cria uma solicitação de certificado PKCS 10 codificada em base64 \# e a salva em um arquivo.                                                                                                                                                                                                                        |
| [**createFileRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createfilerequestwstr)                                       | Cria uma \# solicitação de certificado PKCS 10, uma \# solicitação PKCS 7 ou uma solicitação de certificado CMC completa e a armazena em um arquivo.                                                                                                                                                                               |

| [**createPKCS10WStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-createpkcs10wstr)                                                 | Cria uma solicitação de certificado PKCS 10 codificada em base64 \# .                                                                                                                                                                                                                                               |

| [**createRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-createrequestwstr)                                               | Cria uma \# solicitação de certificado PKCS 10, PKCS \# 7 ou formato CMC completo e a armazena em uma cadeia de caracteres.                                                                                                                                                                                                      |
| [**EnumAlgs**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-enumalgs)                                                                 | Recupera as IDs de algoritmos criptográficos em uma determinada classe de algoritmo que são compatíveis com o CSP atual.                                                                                                                                                                                       |
| [**enumContainersWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-enumcontainerswstr)                                             | Recupera os nomes dos contêineres para o CSP especificado pela propriedade [**ProviderNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providernamewstr) .                                                                                                                                                                    |
| [**enumPendingRequestWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-enumpendingrequestwstr)                                     | Enumera as solicitações de certificado pendentes e recupera uma propriedade especificada de cada uma.                                                                                                                                                                                                                 |
| [**enumProvidersWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-enumproviderswstr)                                               | Recupera os nomes dos CSPs (provedores de serviços de criptografia) disponíveis especificados pela propriedade [**ProviderType**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providertype) .                                                                                                                                                  |
| [**freeRequestInfoBlob**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-freerequestinfoblob)                                           | Exclui um contexto de certificado.                                                                                                                                                                                                                                                                        |

| [**GetKeyLen**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-getkeylen)                                                               | Recupera os comprimentos mínimo e máximo da chave para a assinatura e as chaves de troca.                                                                                                                                                                                                                    |
| [**GetKeyLenEx**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-getkeylenex)                                                           | Recupera informações de tamanho para a assinatura e as chaves de troca.                                                                                                                                                                                                                                       |
| [**getMyStore**](/windows/win32/api/xenroll/nf-xenroll-ienroll-getmystore)                                                             | Reservado para uso futuro.                                                                                                                                                                                                                                                                              |

| [**getProviderTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-getprovidertypewstr)                                           | Recupera o tipo do CSP especificado.                                                                                                                                                                                                                                                              |

| [**GetSupportedKeySpec**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-getsupportedkeyspec)                                           | Recupera informações sobre o suporte do CSP para assinatura ou chaves do Exchange.                                                                                                                                                                                                                     |
