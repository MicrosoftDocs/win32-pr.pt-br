# <a name="required-libraries"></a>Bibliotecas necessárias

Vincular funções de CryptoAPI requer a disponibilidade de crypt32. lib. Os programas que usam algumas das funções também podem exigir advapi32. lib. Ao usar o compilador Visual C++, esses arquivos devem ser listados entre os módulos objeto/biblioteca.
