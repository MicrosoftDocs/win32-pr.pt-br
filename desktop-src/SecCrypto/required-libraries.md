# <a name="required-libraries"></a>Bibliotecas necessárias

Vincular funções CryptoAPI requer a disponibilidade de Crypt32.lib. Programas que usam algumas das funções também podem exigir Advapi32.lib. Ao usar o Visual C++, esses arquivos devem ser listados entre os módulos de objeto/biblioteca.
