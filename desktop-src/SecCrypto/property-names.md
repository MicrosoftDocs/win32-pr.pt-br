# <a name="property-names"></a>Nomes de propriedade

Um nome de propriedade de [*solicitação de certificado*](../secgloss/c-gly.md) pode ser o nome "amigável" de uma das propriedades comuns que os serviços de certificados compreendem ou um OID ( [*identificador de objeto*](../secgloss/o-gly.md) ). Para obter mais informações sobre nomes de propriedade de solicitação, consulte [Propriedades de nome](name-properties.md).
