# <a name="signed-data-additions"></a>Adições de dados assinados

As seguintes alterações foram feitas em recursos de dados assinados:

-   O [*conteúdo interno*](../secgloss/i-gly.md) não data é encapsulado em uma cadeia de caracteres de octeto.
-   O provisionamento é feito para versões de mensagens diferentes.
-   Os assinantes podem ser identificados pelo emissor e pelo número de série, como no PKCS \# 7 1,5 ou pelo identificador de chave da entidade.
-   Vários assinantes são permitidos.
-   As assinaturas RSA, DSA e ECDSA são permitidas.
-   algoritmos baseados em ECC (criptografia de curva elíptica) e AES exigem o Windows Vista ou posterior.
