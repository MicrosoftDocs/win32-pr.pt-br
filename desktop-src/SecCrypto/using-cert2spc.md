# <a name="using-cert2spc"></a>Usando Cert2SPC

o comando a seguir encapsula um certificado [*X. 509*](../secgloss/x-gly.md) , *mycert*. cer, em um Software PKCS \# 7 [*Publisher certificate*](../secgloss/s-gly.md) (SPC), chamado *mycert*. SPC. O SPC criado deve ser usado apenas para fins de teste. Um SPC usado para realmente assinar o código a ser distribuído para o público deve ser obtido em GTE, VeriSign, Inc. ou outra [*autoridade de certificação*](../secgloss/c-gly.md) (CA) confiável.
