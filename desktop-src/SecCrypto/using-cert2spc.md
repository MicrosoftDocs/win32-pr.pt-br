SPC. O SPC criado deve ser usado apenas para fins de teste. Um SPC usado para realmente assinar o código a ser distribuído para o público deve ser obtido em GTE, VeriSign, Inc. ou outra [*autoridade de certificação*](../secgloss/c-gly.md) (CA) confiável.
