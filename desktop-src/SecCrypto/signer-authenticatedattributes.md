# <a name="signerauthenticatedattributes-property"></a>Propriedade signatário. AuthenticatedAttributes

\[A propriedade **AuthenticatedAttributes** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Em vez disso, use a [**classe CmsSigner**](/dotnet/api/system.security.cryptography.pkcs.cmssigner?view=dotnet-plat-ext-3.1&preserve-view=true) no namespace [**System. Security. Cryptography. Pkcs**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

A propriedade **AuthenticatedAttributes** recupera a coleção de atributos autenticados.

## <a name="syntax"></a>Syntax


```VB
Signer.AuthenticatedAttributes As Attributes
```



## <a name="property-value"></a>Valor da propriedade

Uma coleção de [**atributos**](attributes.md) que representa os atributos autenticados do signatário.
