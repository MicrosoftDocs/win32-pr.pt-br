# <a name="active-directory"></a>Active Directory

Os serviços de certificados oferecem suporte ao uso de Active Directory. No caso da AC ( [*autoridade de certificação*](../secgloss/c-gly.md) ) corporativa, os serviços de certificados exigem o uso do Active Directory. A autoridade de certificação corporativa publica certificados de usuário e [*listas de certificados revogados*](../secgloss/c-gly.md) (CRL) para o Active Directory. Uma autoridade de certificação autônoma pode ser administrada para executar as mesmas ações, desde que o Active Directory esteja disponível para a autoridade de certificação autônoma e um módulo de política personalizado seja usado.
