# <a name="methods-of-iocsppropertycollection"></a>Métodos de IOCSPPropertyCollection

Os métodos a seguir são definidos pela interface [**IOCSPPropertyCollection.**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) Os métodos de acesso à propriedade não são mostrados aqui. Para ver as propriedades de **IOCSPPropertyCollection,** consulte [Propriedades de IOCSPPropertyCollection](properties-of-iocsppropertycollection.md)



| Método                                                                                      | Descrição                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Createproperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-createproperty)                     | Cria uma nova propriedade e a adiciona a um conjunto de propriedades.                                    |
| [**DeleteProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-deleteproperty)                     | Remove uma propriedade nomeada de um conjunto de propriedades.                                             |
| [**GetAllProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-getallproperties)                 | Obtém todas as propriedades em um conjunto de propriedades.                                                    |
| [**InitializeFromProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties) | Cria um conjunto de propriedades com base nas propriedades contidas em uma configuração de servidor existente. |
