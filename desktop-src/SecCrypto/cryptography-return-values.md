# <a name="cryptography-return-values"></a>Valores de retorno de criptografia

Os valores de retorno a seguir são usados por funções de criptografia e definidos em Wincrypt. h. Os valores de retorno de criptografia são categorizados de acordo com o uso da seguinte maneira.



| Valor retornado                                                                       | Descrição                                                                                                |
|------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [ASN. 1 codificação/decodificação de valores de retorno](asn-1-encoding-decoding-return-values.md) | Constantes predefinidas que são retornadas por funções que fazem a codificação ou decodificação do ASN. 1.                    |
| [Valores de retorno de certificado e de confiança](certificate-and-trust-return-values.md)     | Constantes predefinidas que são retornadas quando o status de validade e de confiança de um certificado são determinados. |



 

Para obter uma lista de valores de retorno de CAPICOM, consulte [**\_ \_ código de erro CAPICOM**](capicom-error-code.md).
