# <a name="properties-of-iocspcaconfigurationcollection"></a>Propriedades de IOCSPCAConfigurationCollection

As propriedades a seguir são definidas pela interface [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) .



| Propriedade                                                               | Descrição                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get__newenum)    | Obtém um enumerador para o conjunto de configuração.                                                                                                                                            |
| [**Contar**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_count)           | Obtém o número de configurações de [*autoridade de certificação*](../secgloss/c-gly.md) (CA) no conjunto de configurações. |
| [**Item**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_item)             | Obtém uma configuração de autoridade de certificação identificada pelo índice no conjunto de configuração.                                                                                                                    |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_itembyname) | Obtém uma configuração de autoridade de certificação identificada pelo nome no conjunto de configuração.                                                                                                                     |
