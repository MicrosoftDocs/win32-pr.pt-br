# <a name="protocol-flags"></a>Sinalizadores de protocolo

A seguir estão constantes predefinidos para protocolos de criptografia. Esses valores são definidos em Wincrypt.h.



| Constante/valor                                                                                                                                                                                                                            | Descrição                                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| <span id="CRYPT_FLAG_IPSEC"></span><span id="crypt_flag_ipsec"></span><dl> <dt>**CRYPT \_ SINALIZADOR \_ IPSEC**</dt> <dt>0x0010</dt> </dl>       | Protocolo IPsec.<br/>               |
| <span id="CRYPT_FLAG_PCT1"></span><span id="crypt_flag_pct1"></span><dl> <dt>**CRYPT \_ SINALIZADOR \_ PCT1**</dt> <dt>0x0001</dt> </dl>          | Protocolo PCT (transporte de comunicações privadas) versão 1.<br/> |
| <span id="CRYPT_FLAG_SIGNING"></span><span id="crypt_flag_signing"></span><dl> <dt>**CRYPT \_ ASSINATURA \_ DE**</dt> <dt>SINALIZADOR 0X0020</dt> </dl> | Protocolo de assinatura.<br/>                                          |
| <span id="CRYPT_FLAG_SSL2"></span><span id="crypt_flag_ssl2"></span><dl> <dt>**CRYPT \_ SINALIZADOR \_ SSL2**</dt> <dt>0x0002</dt> </dl>          | Protocolo SSL (Secure Sockets Layer) versão 2.<br/>             |
| <span id="CRYPT_FLAG_SSL3"></span><span id="crypt_flag_ssl3"></span><dl> <dt>**CRYPT \_ SINALIZADOR \_ SSL3**</dt> <dt>0x0004</dt> </dl>          | Protocolo SSL versão 3.<br/>                                    |
| <span id="CRYPT_FLAG_TLS1"></span><span id="crypt_flag_tls1"></span><dl> <dt>**CRYPT \_ SINALIZADOR \_ TLS1**</dt> <dt>0x0008</dt> </dl>          | Protocolo TLS versão 1.<br/>         |
