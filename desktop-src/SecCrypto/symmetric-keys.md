# <a name="symmetric-keys"></a>Chaves simétricas

Uma [*chave simétrica*](../secgloss/s-gly.md) é uma chave única que é usada com as operações de um esquema de [*criptografia simétrica*](../secgloss/s-gly.md) . Por exemplo, tal chave pode ser usada em um algoritmo de criptografia simétrica para criptografia e descriptografia.

Os algoritmos simétricos mais conhecidos são mais rápidos do que os [*algoritmos de chave pública*](../secgloss/p-gly.md)mais conhecidos. Portanto, eles são preferidos ao processar grandes quantidades de dados. Alguns dos algoritmos simétricos mais comuns são o [*RC4*](../secgloss/r-gly.md), o Triple DES ([*padrão de criptografia de dados*](../secgloss/d-gly.md)) e o [*criptografia AES*](../secgloss/a-gly.md) (AES).

Como grande parte da atividade que envolve chaves simétricas está relacionada para mantê-las secretas, é importante manter o número de pessoas que possuem uma chave simétrica específica o menor possível. Recomendamos que apenas uma ou duas pessoas possuam uma chave simétrica.
