# <a name="error-handling-cryptography"></a>Tratamento de erro (criptografia)

Em todos os exemplos C que acompanham a documentação de cada função, a função de exemplo [**MyHandleError**](myhandleerror.md) é usada para imprimir uma mensagem de erro e sair de um procedimento. Em situações reais, procedimentos de tratamento de erro mais extensos podem ser necessários. Para obter mais informações, consulte [Limitações do programa de exemplo](example-program-limitations.md).
