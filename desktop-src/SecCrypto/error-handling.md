# <a name="error-handling-cryptography"></a>Tratamento de erro (criptografia)

Em todos os exemplos de C que acompanham a documentação de cada função, a função de exemplo [**MyHandleError**](myhandleerror.md) é usada para imprimir uma mensagem de erro e sair de um procedimento. Em situações reais, podem ser necessários procedimentos de tratamento de erros mais extensivos. Para obter mais informações, consulte [limitações do programa de exemplo](example-program-limitations.md).
