# <a name="properties-of-iocspadmin"></a>Propriedades de IOCSPAdmin

As propriedades a seguir são definidas pela interface [**IOCSPAdmin**](/windows/desktop/api/certadm/nn-certadm-iocspadmin) .



| Propriedade                                                                                 | Descrição                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) | Obtém uma instância de um objeto [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) . Esse objeto representa o conjunto de certificados para o qual um serviço Respondente pode manipular solicitações de status. |
| [**OCSPServiceProperties**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspserviceproperties)                 | Obtém uma instância de um objeto [**OCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) . Esse objeto representa os atributos de um serviço Respondente.                                                                            |
