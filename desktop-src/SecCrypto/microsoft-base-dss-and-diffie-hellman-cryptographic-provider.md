# <a name="microsoft-base-dss--diffie-hellman-cryptographic-provider"></a>Microsoft Base DSS & Diffie-Hellman provedor criptográfico

O Microsoft Base DSS e o provedor de criptografia Diffie-Hellman oferece suporte à troca de chaves Diffie-Hellman (D-H) (um derivativo DES de 40 bits), hash de SHA, assinatura de dados DSS e verificação de assinatura DSS. Ele pode ser exportado para outros países/regiões.

|----------------|----------------------------|

| Nome do provedor: | **Prov DH do MS \_ Def \_ DSS \_ \_** |
