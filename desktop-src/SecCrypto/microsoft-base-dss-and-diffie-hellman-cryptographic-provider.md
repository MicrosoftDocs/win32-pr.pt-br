# <a name="microsoft-base-dss--diffie-hellman-cryptographic-provider"></a>Provedor criptográfico microsoft base DSS & Diffie-Hellman criptografia

O Microsoft Base DSS e o provedor criptográfico Diffie-Hellman dá suporte à troca de chaves de Diffie-Hellman (D-H) (um derivado de DES de 40 bits), hash SHA, assinatura de dados DSS e verificação de assinatura DSS. Ele pode ser exportado para outros países/regiões.
