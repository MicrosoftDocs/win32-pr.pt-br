# <a name="prov_rsa_sig"></a>PROV \_ RSA \_ SIG

O \_ tipo de \_ provedor de SIG RSA do Prov é um subconjunto do Prov \_ RSA \_ completo. Ele dá suporte apenas a funções e algoritmos necessários para [*hashes*](../secgloss/h-gly.md) e [*assinaturas digitais*](../secgloss/d-gly.md).

## <a name="algorithms-supported"></a>Algoritmos com suporte

Para obter descrições de cada um desses algoritmos, consulte o Glossário.



| Finalidade      | Algoritmos compatíveis                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Exchange de chave | Nenhum                                                                                                                              |
| Assinatura    | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Criptografia   | Nenhum                                                                                                                              |
| Hash      | [](../secgloss/m-gly.md)[*Sha* MD5](../secgloss/s-gly.md)<br/> |
