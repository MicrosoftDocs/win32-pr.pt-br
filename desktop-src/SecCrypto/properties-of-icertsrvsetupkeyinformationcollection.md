# <a name="properties-of-icertsrvsetupkeyinformationcollection"></a>Propriedades de ICertSrvSetupKeyInformationCollection

As propriedades a seguir são definidas pela interface [**ICertSrvSetupKeyInformationCollection**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformationcollection) .



| Propriedade                                                            | Descrição                                                                                                              |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get__newenum) | Obtém um enumerador para a coleção.                                                                                   |
| [**Contar**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_count)        | Obtém o número de objetos [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) na coleção.         |
| [**Item**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_item)          | Obtém um objeto [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) identificado pelo índice na coleção. |
