# <a name="keyusageisdecipheronlyenabled-property"></a>Propriedade KeyUsage.IsDecipherOnlyEnabled

\[A **propriedade IsDecipherOnlyEnabled** está disponível para uso nos sistemas operacionais especificados na seção Requisitos. Em vez disso, [**use a classe X509EnhancedKeyUsageExtension**](/dotnet/api/system.security.cryptography.x509certificates.x509enhancedkeyusageextension?view=netcore-3.1) no namespace [**System.Security.Cryptography.X509Certificates.**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1)\]

A **propriedade IsDecipherOnlyEnabled** recupera um valor booliana que indica se o bit de desarmãoOnly está definido.

## <a name="syntax"></a>Syntax


```VB
KeyUsage.IsDecipherOnlyEnabled As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Se **for true,** o bit de desadecidaOnly será definido.
