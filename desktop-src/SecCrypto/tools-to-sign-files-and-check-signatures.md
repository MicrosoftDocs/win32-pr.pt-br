# <a name="tools-to-sign-files-and-check-signatures"></a>Ferramentas para assinar arquivos e verificar assinaturas

As ferramentas de CryptoAPI são as seguintes ferramentas que você pode usar para assinar arquivos digitalmente, para confirmar assinaturas em arquivos e para criar arquivos de catálogo.



| Ferramenta                     | Comentários                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MakeCat](makecat.md)   | Cria um arquivo de catálogo não assinado. Esse arquivo de catálogo é usado posteriormente para comparar os [*hashes*](../secgloss/h-gly.md) no arquivo de catálogo com os hashes de arquivos individuais, o que ajuda a provar que os arquivos originais estão livres de adulteração. |
| [SetReg](setreg.md)     | Define as chaves do registro que controlam o processo de verificação de certificado.                                                                                                                                                                                                              |
| [SignTool](signtool.md) | Assina digitalmente arquivos e confirma assinaturas digitais de arquivos.                                                                                                                                                                                                                    |
