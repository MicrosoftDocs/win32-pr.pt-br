# <a name="properties-of-imscepsetup"></a>Propriedades de IMSCEPSetup

As propriedades a seguir são definidas pela interface [**IMSCEPSetup**](/windows/desktop/api/Casetup/nn-casetup-imscepsetup) .



| Propriedade                                                 | Descrição                                                                                                                                              |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSCEPErrorId**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorid)         | Obtém a ID de informações de erro adicionais relacionadas a uma especificação de NDES com falha. Qualquer chamada de método no objeto pai redefine essa propriedade.          |
| [**MSCEPErrorString**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorstring) | Obtém os dados de cadeia de caracteres para obter informações de erro adicionais relacionadas a uma especificação de NDES com falha. Qualquer chamada de método no objeto pai redefine essa propriedade. |
