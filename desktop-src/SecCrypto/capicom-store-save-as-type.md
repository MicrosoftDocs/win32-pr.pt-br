# <a name="capicom_store_save_as_type-enumeration"></a>Enumeração CAPICOM \_ STORE \_ SAVE AS \_ \_ TYPE

O tipo de enumeração **CAPICOM \_ STORE SAVE AS \_ \_ \_ TYPE** indica a codificação de um armazenamento [*de certificados.*](../secgloss/c-gly.md)

## <a name="members"></a>Membros



| Membro                                   | DESCRIÇÃO                                         | Valor |
|------------------------------------------|-----------------------------------------------------|-------|
| **CAPICOM \_ STORE \_ SAVE \_ AS \_ SERIALIZED** | O armazenamento é salvo em formato serializado.<br/> | 0     |
| **CAPICOM \_ STORE \_ SAVE \_ AS \_ PKCS7**      | O armazenamento é salvo no formato PKCS \# 7.<br/>   | 1     |
