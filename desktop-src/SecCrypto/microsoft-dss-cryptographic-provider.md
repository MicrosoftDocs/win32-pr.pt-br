# <a name="microsoft-dss-cryptographic-provider"></a>Provedor criptográfico do Microsoft DSS

O Provedor criptográfico do Microsoft DSS dá suporte à verificação de hash, assinatura de dados e assinatura usando os algoritmos SHA (Secure [*Hash Algorithm)*](../secgloss/s-gly.md) e DSS [*(Digital Signature Standard).*](../secgloss/d-gly.md) Ele pode ser exportado fora do Estados Unidos e do Canadá.



|                   | Valor               |
|-------------------|---------------------|
| **Tipo de provedor** | PROV \_ DSS           |
| **Nome do provedor** | MS \_ DEF \_ DSS \_ PROV  |
