# <a name="microsoft-dss-cryptographic-provider"></a>Provedor criptográfico do Microsoft DSS

O provedor criptográfico do Microsoft DSS dá suporte a hashing, assinatura de dados e verificação de assinatura usando os algoritmos Sha ( [*algoritmo de hash seguro*](../secgloss/s-gly.md) ) e DSS ( [*padrão de assinatura digital*](../secgloss/d-gly.md) ). Ele pode ser exportado fora do Estados Unidos e do Canadá.

|----------------|------------------------|
| Tipo de provedor: | **PROV \_ DSS**          |
| Nome do provedor: | **Prov de DSS de MS \_ Def \_ \_** |
