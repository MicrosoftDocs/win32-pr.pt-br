# <a name="microsoft-strong-cryptographic-provider"></a>Microsoft Strong Cryptographic Provider

O provedor de criptografia forte da Microsoft é usado como o CSP ( [*provedor de serviços de criptografia*](../secgloss/c-gly.md) completa) padrão da RSA. Ele dá suporte a todos os algoritmos do provedor criptográfico aprimorado da Microsoft e a todos os mesmos [*comprimentos de chave*](../secgloss/k-gly.md). Para obter informações sobre os tamanhos e algoritmos de chave padrão, consulte Microsoft Base Cryptographic Provider. Para obter informações sobre comprimentos de chave mais longos e algoritmos disponíveis, consulte a tabela com o [provedor criptográfico aprimorado da Microsoft](microsoft-enhanced-cryptographic-provider.md).



|                   | Valor             |
|-------------------|-------------------|
| **Tipo de provedor** | PROV \_ RSA \_ completo   |
| **Nome do provedor** | \_Prov de alta segurança da MS \_  |
