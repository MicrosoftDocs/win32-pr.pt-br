# <a name="prov_rsa_full"></a>PROV \_ RSA \_ completo

O \_ tipo de \_ provedor completo RSA Prov oferece suporte a [*assinaturas digitais*](../secgloss/d-gly.md) e criptografia de dados. Ele é considerado um CSP de uso geral. O algoritmo de chave pública RSA é usado para todas as operações de chave pública.

## <a name="algorithms-supported"></a>Algoritmos com suporte

Para obter descrições de cada um desses algoritmos, consulte o Glossário.



| Finalidade      | Algoritmos compatíveis                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Troca de chaves | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Assinatura    | RSA                                                                                                                               |
| Criptografia   | [](../secgloss/r-gly.md)[*RC4* RC2](../secgloss/r-gly.md)<br/> |
| Hash      | [](../secgloss/m-gly.md)[*Sha* MD5](../secgloss/s-gly.md)<br/> |
