# <a name="oidname-property"></a>OIDs. Propriedade Name

\[A propriedade **Name** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Em vez disso, use a [**classe Oid**](/dotnet/api/system.security.cryptography.oid?view=netcore-3.1) no namespace [**System. Security. Cryptography**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

A propriedade **Name** define ou recupera o nome definido pelo CAPICOM para o identificador. Essa é a propriedade padrão.

## <a name="syntax"></a>Syntax


```VB
OID.Name As CAPICOM_OID
```
