# <a name="digital-messages"></a>Mensagens digitais

Uma mensagem será definida como todos os dados que foram codificados. As mensagens são dados codificados para transmissão para outra pessoa ou entidade ou dados codificados recebidos de outra pessoa ou entidade. No ambiente de criptografia, as mensagens geralmente são assinadas, criptografadas ou ambas. O termo mensagens geralmente inclui certificados.

No CryptoAPI versão 2.0, as funções de manipulação de mensagens são [*divididas*](../secgloss/l-gly.md) em funções de mensagem de baixo nível e funções de [*mensagem simplificadas.*](../secgloss/s-gly.md)
