# <a name="digital-messages"></a>Mensagens digitais

Uma mensagem será definida como qualquer dado que tenha sido codificado. As mensagens são codificadas em dados para transmissão para outra pessoa ou entidade ou dados codificados recebidos de outra pessoa ou entidade. No ambiente de criptografia, as mensagens geralmente são assinadas, criptografadas ou ambas. O termo mensagens geralmente inclui certificados.

No CryptoAPI versão 2,0, as funções de manipulação de mensagens são divididas em [*funções de mensagens de nível baixo*](../secgloss/l-gly.md) e [*funções de mensagens simplificadas*](../secgloss/s-gly.md).
