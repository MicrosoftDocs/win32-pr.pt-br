# <a name="capicom_signed_data_verify_flag-enumeration"></a>\_Enumeração de \_ \_ sinalizador de verificação de dados ASSINADOs CAPICOM \_

A **enumeração \_ \_ sinalizador de \_ verificação \_ de dados assinados de capicor** indica o que é verificado quando uma [*assinatura digital*](../secgloss/d-gly.md) é verificada.

## <a name="members"></a>Membros



| Membro                                           | DESCRIÇÃO                                                                                                 | Valor |
|--------------------------------------------------|-------------------------------------------------------------------------------------------------------------|-------|
| **CAPICOM \_ verificar \_ \_ somente assinatura**             | Somente a assinatura é verificada.<br/>                                                                   | 0     |
| **CAPICOM \_ verificar \_ assinatura \_ e \_ certificado** | A assinatura e a validade do certificado usado para criar a assinatura são verificadas.<br/> | 1     |
