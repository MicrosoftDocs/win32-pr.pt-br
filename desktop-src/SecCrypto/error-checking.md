# <a name="error-checking-cryptography"></a>Verificação de erro (criptografia)

Para C++, o valor de retorno é sempre um **HRESULT** e indica se a chamada do método foi bem-sucedida ou falhou. Se a chamada falhar, o valor de retorno indicará o motivo da falha.
