# <a name="cryptography-constants"></a>Constantes de criptografia

As constantes a seguir são usadas por funções de criptografia e definidas em Wincrypt.h. As constantes de criptografia são categorizadas de acordo com o uso da seguinte forma.



| Constante                                                                                                           | Descrição                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes para CryptEncodeObject e CryptDecodeObject](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Constantes predefinidos usadas com operações de codificação e decodificação de ASN.1 (Abstract [*Syntax Notation One).*](../secgloss/a-gly.md) |
| [Constantes para extensões do Netscape](constants-for-netscape-extensions.md)                                         | Constantes predefinidos usadas para extensões de certificado Netscape. Esta seção fornece detalhes sobre o uso dessas extensões com operações de codificação e decodificar ASN.1.                                      |
| [Nomes de provedor criptográfico](cryptographic-provider-names.md)                                                   | Constantes de cadeia de caracteres predefinida para os nomes de CSPs (provedores de serviços [*criptográficos)*](../secgloss/c-gly.md) comuns.   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | Uma constante numérica que especifica o número máximo de caracteres que alguns dos provedores criptográficos da Microsoft usarão ao obter a ID de usuário.                                                  |
| [**Constantes de Caching privada**](private-key-caching-constants.md)                                             | Constantes que representam entradas do Registro que controlam o cache [*de chaves privadas*](../secgloss/p-gly.md) por um CSP.                                                |
| [Sinalizadores de protocolo](protocol-flags.md)                                                                               | Constantes predefinidos para protocolos de criptografia.                                                                                                                                                            |
