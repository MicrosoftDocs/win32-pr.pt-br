# <a name="cryptography-constants"></a>Constantes de criptografia

As constantes a seguir são usadas por funções de criptografia e definidas em Wincrypt. h. As constantes de criptografia são categorizadas de acordo com o uso da seguinte maneira.



| Constante                                                                                                           | Descrição                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes para CryptEncodeObject e CryptDecodeObject](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Constantes predefinidas usadas com operações de codificação e decodificação de [*uma*](../secgloss/a-gly.md) (ASN. 1). |
| [Constantes para extensões do Netscape](constants-for-netscape-extensions.md)                                         | Constantes predefinidas usadas para extensões de certificado do Netscape. Esta seção fornece detalhes sobre o uso dessas extensões com as operações de codificação e decodificação ASN. 1.                                      |
| [Nomes de provedor criptográfico](cryptographic-provider-names.md)                                                   | Constantes de cadeia de caracteres predefinidas para os nomes de CSPs ( [*provedores de serviço de criptografia*](../secgloss/c-gly.md) ) comuns.   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | Uma constante numérica que especifica o número máximo de caracteres que alguns dos provedores de criptografia da Microsoft usarão ao obter a ID de usuário.                                                  |
| [**Constantes de cache de chave privada**](private-key-caching-constants.md)                                             | Constantes que representam entradas de registro que controlam o cache de [*chave privada*](../secgloss/p-gly.md) por um CSP.                                                |
| [Sinalizadores de protocolo](protocol-flags.md)                                                                               | Constantes predefinidas para protocolos de criptografia.                                                                                                                                                            |
