# <a name="utilitiesbinarytohex-method"></a>Método Utilities. BinaryToHex

\[O método **BinaryToHex** está disponível para uso nos sistemas operacionais especificados na seção requisitos.\]

O método **BinaryToHex** converte uma cadeia de caracteres de pacote binário em uma cadeia de caracteres hexadecimal.

## <a name="syntax"></a>Sintaxe


```VB
Utilities.BinaryToHex( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Binarystring* \[ no\]
</dt> <dd>

A cadeia de caracteres de pacote binário a ser convertida em uma cadeia de caracteres hexadecimal.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Uma cadeia de caracteres hexadecimal equivalente à cadeia de caracteres especificada compactada em binário.
