# <a name="properties-of-ienroll4"></a>Propriedades de IEnroll4

As propriedades a seguir são definidas pela interface [**IEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll4) ou são propriedades que a interface **IEnroll4** herda de [**IEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll) e [**IEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll2).



| Propriedade                                                                              | Descrição                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoreflags)                                         | Define ou recupera um sinalizador que controla o repositório de certificados quando ele é aberto.                                                                                                                                                                                                                                         |
| [**CAStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr)                                   | Define ou recupera o nome da loja em que todos os certificados não "raiz" e "MY" não são mantidos.                                                                                                                                                                                                                        |
| [**CAStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoretypewstr)                                   | Define ou recupera o tipo de armazenamento a ser usado para o repositório especificado pela propriedade [**CAStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr) .                                                                                                                                                                                 |
| [**ClientId**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_clientid)                                                 | Define ou recupera o atributo de solicitação de ID do cliente.                                                                                                                                                                                                                                                                      |
| [**ContainerNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_containernamewstr)                               | Define ou recupera o nome do [*contêiner de chave*](../secgloss/k-gly.md) a ser usado.                                                                                                                                                                                     |
| [**DeleteRequestCert**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_deleterequestcert)                               | Define ou recupera um indicador booliano que controla se os certificados fictícios no repositório de solicitações são excluídos.                                                                                                                                                                                                        |
| [**EnableSMIMECapabilities**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-get_enablesmimecapabilities)                   | Define ou recupera um valor booliano que controla se o PKCS \# 10 conterá um atributo assinado para recursos de S/MIME ( [*Internet Mail Extensions) seguros/multipropósitos*](../secgloss/s-gly.md) . |
| [**EnableT61DNEncoding**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_enablet61dnencoding)                           | Define ou recupera um valor booliano que controla se o nome distinto na solicitação é codificado como uma cadeia de caracteres T61 em vez de uma cadeia de caracteres [*Unicode*](../secgloss/u-gly.md) .                                                                                                |
| [**GenKeyFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_genkeyflags)                                           | Define ou recupera os valores passados para [**CryptGenKey**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgenkey) quando a solicitação de certificado é gerada.                                                                                                                                                                                                    |
| [**HashAlgID**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_hashalgid)                                               | Define ou recupera o algoritmo de hash usado ao assinar uma \# solicitação de certificado PKCS 10.                                                                                                                                                                                                                                 |
| [**HashAlgorithmWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_hashalgorithmwstr)                               | Define ou recupera apenas o algoritmo de hash de assinatura usado para assinar o PKCS \# 10.                                                                                                                                                                                                                                         |
| [**IncludeSubjectKeyID**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_includesubjectkeyid)                           | Especifica se a extensão de ID de chave de entidade é adicionada à solicitação de certificado.                                                                                                                                                                                                                                     |
| [**KeySpec**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_keyspec)                                                   | Define ou recupera o tipo de chave gerada.                                                                                                                                                                                                                                                                            |
| [**LimitExchangeKeyToEncipherment**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_limitexchangekeytoencipherment)     | Define ou recupera um valor booliano que controla se uma \_ solicitação de troca de chaves contém [*assinatura digital*](../secgloss/d-gly.md) e usos de chave de não repúdio.                                                                                       |
| [**MyStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystoreflags)                                         | Define ou recupera o local do registro usado para o meu repositório.                                                                                                                                                                                                                                                          |
| [**MyStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr)                                   | Define ou recupera o nome do repositório onde os certificados com chaves privadas vinculadas são mantidos.                                                                                                                                                                                                                           |
| [**MyStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystoretypewstr)                                   | Define ou recupera o tipo de repositório especificado pela propriedade [**MyStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr) .                                                                                                                                                                                                      |
| [**ProviderFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providerflags)                                       | Define ou recupera o tipo de CSP.                                                                                                                                                                                                                                                                                         |
| [**ProviderNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providernamewstr)                                 | Define ou recupera o nome do CSP a ser usado.                                                                                                                                                                                                                                                                           |
| [**ProviderType**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providertype)                                         | Define ou recupera o tipo de provedor.                                                                                                                                                                                                                                                                                 |
| [**PVKFileNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_pvkfilenamewstr)                                   | Define ou recupera o nome do arquivo que conterá chaves exportadas.                                                                                                                                                                                                                                                 |
| [**RenewalCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_renewalcertificate)                             | Especifica o contexto do certificado para o certificado de renovação.                                                                                                                                                                                                                                                          |
| [**RequestStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_requeststoreflags)                               | Define ou recupera o local do registro usado para o repositório de solicitações.                                                                                                                                                                                                                                                     |
| [**RequestStoreNameWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststorenamewstr)                         | Define ou recupera o nome do repositório que contém o certificado fictício. Esse certificado fictício, juntamente com as chaves privadas adicionadas, permanece no repositório de solicitações até que uma autoridade de certificação processe a solicitação e responda com um PKCS \# 7.                                                                |
| [**RequestStoreTypeWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststoretypewstr)                         | Define ou recupera o tipo de armazenamento a ser usado para o repositório especificado pela propriedade [**RequestStoreNameWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststorenamewstr) .                                                                                                                                                                       |
| [**ReuseHardwareKeyIfUnableToGenNew**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_reusehardwarekeyifunabletogennew) | Define ou recupera um valor booliano que determina a ação realizada pelo objeto de controle de registro de certificado se for encontrado um erro ao gerar uma nova chave.                                                                                                                                                   |
| [**RootStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstoreflags)                                     | Define ou recupera o local do registro usado para o repositório raiz.                                                                                                                                                                                                                                                        |
| [**RootStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr)                               | Define ou recupera o nome do repositório raiz onde todos os certificados raiz autoassinados intrinsecamente confiáveis são mantidos.                                                                                                                                                                                                    |
| [**RootStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstoretypewstr)                               | Define ou recupera o tipo de armazenamento a ser usado para o repositório especificado pela propriedade [**RootStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr) .                                                                                                                                                                             |
| [**SPCFileNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_spcfilenamewstr)                                   | Define ou recupera o nome do arquivo para gravar o PKCS 7 codificado na base64 resultante \# , conforme retornado da autoridade de certificação.                                                                                                                                                                                     |
| [**ThumbPrintWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_thumbprintwstr)                                     | Define ou recupera um [*hash*](../secgloss/h-gly.md) dos dados do certificado.                                                                                                                                                                                                             |
| [**UseExistingKeySet**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_useexistingkeyset)                               | Define ou recupera um valor booliano que indica se as chaves existentes devem ser usadas.                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_writecerttocsp)                                     | Define ou recupera um valor booliano que indica se um certificado deve ser gravado no CSP.                                                                                                                                                                                                                    |
| [**WriteCertToUserDS**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_writecerttouserds)                               | Define ou recupera um valor booliano que controla se o certificado é gravado no armazenamento de Active Directory do usuário.                                                                                                                                                                                                |
