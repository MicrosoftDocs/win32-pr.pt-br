# <a name="methods-of-iocspcaconfigurationcollection"></a>Métodos de IOCSPCAConfigurationCollection

Os métodos a seguir são definidos pela interface [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) . Os métodos de acesso à propriedade não são mostrados aqui. Para ver as propriedades de **IOCSPCAConfigurationCollection**, consulte [Propriedades de IOCSPCAConfigurationCollection](properties-of-iocspcaconfigurationcollection.md)



| Método                                                                                       | Descrição                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Cria uma nova configuração de AC ( [*autoridade de certificação*](../secgloss/c-gly.md) ) e a adiciona ao conjunto de configuração. |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Remove uma configuração de autoridade de certificação nomeada do conjunto de configuração.                                                                                                                                   |
