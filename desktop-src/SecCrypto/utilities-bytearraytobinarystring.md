# <a name="utilitiesbytearraytobinarystring-method"></a>Método Utilities. ByteArrayToBinaryString

\[O método **ByteArrayToBinaryString** está disponível para uso nos sistemas operacionais especificados na seção requisitos.\]

O método **ByteArrayToBinaryString** converte uma matriz de bytes em uma cadeia de caracteres de pacote binário.

## <a name="syntax"></a>Sintaxe


```VB
Utilities.ByteArrayToBinaryString( _
  ByVal varByteArray _
)
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*varByteArray* \[ no\]
</dt> <dd>

A matriz de bytes a ser convertida em uma cadeia de caracteres de pacote binário.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

A cadeia de caracteres compactados em binário que contém o resultado da conversão.
