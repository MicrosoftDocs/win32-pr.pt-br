# <a name="capicom_attribute-enumeration"></a>Enumeração DE ATRIBUTO CAPICOM \_

O **tipo de \_ enumeração CAPICOM ATTRIBUTE** define o tipo de atributo associado a uma [*assinatura*](../secgloss/d-gly.md).

## <a name="members"></a>Membros



| Membro                                                       | DESCRIÇÃO                                                                | Valor |
|--------------------------------------------------------------|----------------------------------------------------------------------------|-------|
| **HORA DE ASSINATURA \_ DE \_ ATRIBUTO AUTENTICADO \_ CAPICOM \_**         | O atributo contém a hora em que a assinatura foi criada.<br/> | 0     |
| **NOME DO DOCUMENTO \_ DE \_ ATRIBUTO AUTENTICADO \_ CAPICOM \_**        | O atributo contém o nome do documento assinado.<br/>         | 1     |
| **DESCRIÇÃO DO \_ DOCUMENTO DE \_ ATRIBUTO AUTENTICADO \_ CAPICOM \_** | O atributo contém uma descrição do documento assinado.<br/>    | 2     |



## <a name="remarks"></a>Comentários

O **tipo de \_ enumeração CAPICOM ATTRIBUTE** é usado pela propriedade [**Attribute.Name.**](attribute-name.md)
