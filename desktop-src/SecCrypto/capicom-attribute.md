# <a name="capicom_attribute-enumeration"></a>Enumeração de atributo CAPICOM \_

O tipo de enumeração de **\_ atributo capicor** define o tipo de atributo associado a uma [*assinatura*](../secgloss/d-gly.md).

## <a name="members"></a>Membros



| Membro                                                       | DESCRIÇÃO                                                                | Valor |
|--------------------------------------------------------------|----------------------------------------------------------------------------|-------|
| **tempo de assinatura do atributo capicod \_ autenticado \_ \_ \_**         | O atributo contém a hora em que a assinatura foi criada.<br/> | 0     |
| **\_nome do documento de atributos autenticados do CApicom \_ \_ \_**        | O atributo contém o nome do documento assinado.<br/>         | 1     |
| **\_Descrição do documento de atributos autenticados CApicom \_ \_ \_** | O atributo contém uma descrição do documento assinado.<br/>    | 2     |



## <a name="remarks"></a>Comentários

O tipo de enumeração de **\_ atributo capicor** é usado pela propriedade [**Attribute.Name**](attribute-name.md) .
