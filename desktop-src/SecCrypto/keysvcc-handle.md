# <a name="keysvcc_handle"></a>KEYSVCC \_ HANDLE

O **tipo de dados KEYSVCC \_ HANDLE** define um identificador de serviço de chave. Um **handle KEYSVCC \_ HANDLE** é usado pelas [**funções RKeyOpenKeyService**](rkeyopenkeyservice.md) e [**RKeyCloseKeyService.**](rkeyclosekeyservice.md)


```C++
typedef void* KEYSVCC_HANDLE;
```
