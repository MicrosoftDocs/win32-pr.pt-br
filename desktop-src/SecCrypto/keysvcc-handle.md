# <a name="keysvcc_handle"></a>identificador de KEYSVCC \_

O tipo de dados do **\_ identificador KEYSVCC** define um identificador de serviço de chave. Um identificador de **\_ identificador KEYSVCC** é usado pelas funções [**RKeyOpenKeyService**](rkeyopenkeyservice.md) e [**RKeyCloseKeyService**](rkeyclosekeyservice.md) .


```C++
typedef void* KEYSVCC_HANDLE;
```
