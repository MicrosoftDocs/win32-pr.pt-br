# <a name="hcryptprov"></a>HCRYPTPROV

O tipo de dados **HCRYPTPROV** é usado para representar identificadores para CSPs. Esses identificadores são usados para indicar qual módulo CSP executa operações específicas.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
