# <a name="hcryptprov"></a>Hcryptprov

O **tipo de dados HCRYPTPROV** é usado para representar identificador para CSPs. Esses alças são usados para indicar qual módulo do CSP executa operações específicas.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
