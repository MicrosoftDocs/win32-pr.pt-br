# <a name="properties-of-icertsrvsetup"></a>Propriedades de ICertSrvSetup

As propriedades a seguir são definidas pela interface [**ICertSrvSetup**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetup) .



| Propriedade                                             | Descrição                                                                                                                                                                                                  |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAErrorId**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorid)         | Obtém a ID de informações de erro adicionais relacionadas a uma especificação de AC ( [*autoridade de certificação*](../secgloss/c-gly.md) ) com falha. |
| [**CAErrorString**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorstring) | Obtém os dados de cadeia de caracteres para obter informações de erro adicionais relacionadas a uma especificação de autoridade de certificação com falha.                                                                                                                  |
