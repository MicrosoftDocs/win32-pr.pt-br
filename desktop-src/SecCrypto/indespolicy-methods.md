# <a name="indespolicy-methods"></a>Métodos INDESPolicy

A interface [**INDESPolicy**](/windows/desktop/api/certpol/nn-certpol-indespolicy) dá suporte aos métodos a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                        | Descrição                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Método GenerateChallenge**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Executa a decisão de política para emitir uma senha de desafio para o cliente SCEP.<br/> |
| [**Método Initialize**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Inicializa o módulo de política de NDES.<br/>                                                    |
| [**Notificar método**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Notifica o plug-in do status da transação da solicitação de certificado SCEP.<br/>        |
| [**Método Uninitialize**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Desinicializa o módulo de política de NDES.<br/>                                                  |
| [**Método VerifyRequest**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Verifica a solicitação de certificado NDES para envio à AC.<br/>                        |
