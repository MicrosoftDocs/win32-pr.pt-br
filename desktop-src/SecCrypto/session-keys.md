# <a name="session-keys"></a>Chaves de sessão

[*As chaves de sessão*](../secgloss/s-gly.md) são chaves que são geradas para serem usadas em uma única sessão de comunicação. As chaves de sessão são frequentemente alteradas e são descartadas quando não são mais necessárias. Normalmente, uma [*chave simétrica*](../secgloss/s-gly.md) é usada como a chave de sessão.

As chaves de sessão são voláteis. Os aplicativos podem salvar essas chaves para uso posterior ou para transmissão para outros usuários. para obter mais informações, consulte [chave de criptografia Armazenamento e Exchange](cryptographic-key-storage-and-exchange.md).
