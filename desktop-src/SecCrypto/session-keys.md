# <a name="session-keys"></a>Chaves de sessão

[*As chaves de sessão*](../secgloss/s-gly.md) são chaves que são geradas para serem usadas em uma única sessão de comunicação. As chaves de sessão são frequentemente alteradas e são descartadas quando não são mais necessárias. Por exemplo, o TLS usa uma chave de sessão separada para cada conexão e S/MIME usam uma chave de sessão separada para cada mensagem de email. Normalmente, uma [*chave simétrica*](../secgloss/s-gly.md) é usada como a chave de sessão.

As chaves de sessão são voláteis. Os aplicativos podem salvar essas chaves para uso posterior ou para transmissão para outros usuários. Para obter mais informações, consulte [armazenamento de chaves de criptografia e troca](cryptographic-key-storage-and-exchange.md).
