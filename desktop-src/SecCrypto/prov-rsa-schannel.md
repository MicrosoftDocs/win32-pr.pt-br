# <a name="prov_rsa_schannel"></a>PROV \_ RSA \_ Schannel

O \_ tipo de provedor do RSA Schannel do Prov \_ oferece suporte aos protocolos RSA e [*Schannel*](../secgloss/s-gly.md) .

## <a name="algorithms-supported"></a>Algoritmos com suporte

Para obter descrições de cada um desses algoritmos, consulte o Glossário.



| Finalidade      | Algoritmos compatíveis                                                                                                                                                                                                                                                                                                          |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Troca de chaves | [*RSA*](../secgloss/r-gly.md)                                                                                                                                                                                                                                                                   |
| Assinatura    | RSA                                                                                                                                                                                                                                                                                                                           |
| Criptografia   | Qualquer PROV \_ RSA \_ Schannel CSP deve implementar um ou mais dos seguintes algoritmos: [ *RC4*](../secgloss/r-gly.md)<br/> [*DES*](../secgloss/d-gly.md)<br/> [*DES triplo*](../secgloss/t-gly.md)<br/> |
| Hash      | [](../secgloss/m-gly.md)[*Sha* MD5](../secgloss/s-gly.md)<br/>                                                                                                                                                                                             |
