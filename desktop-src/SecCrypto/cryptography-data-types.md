# <a name="cryptography-data-types"></a>Tipos de dados de criptografia

Os tipos de dados a seguir são usados por funções, interfaces e objetos de criptografia.



| Tipo de dados                                                                      | Descrição                                                                                                                                                                                      |
|--------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ID de ALG \_**](alg-id.md)                                                      | Especifica os identificadores de algoritmo.                                                                                                                                                                 |
| [**\_resposta OCSP do servidor HCERT \_ \_**](hcert-server-ocsp-response.md)            | Representa um identificador para uma resposta OCSP associada a uma cadeia de certificados de servidor.                                                                                                              |
| [**HCRYPTHASH**](hcrypthash.md)                                               | Especifica identificadores para um [*objeto de hash*](../secgloss/h-gly.md).                                                                                      |
| [**HCRYPTKEY**](hcryptkey.md)                                                 | Especifica identificadores para chaves de criptografia.                                                                                                                                                         |
| [**HCRYPTOIDFUNCADDR**](hcryptoidfuncaddr.md)                                 | Representa um identificador para uma função que pode ser instalada usando um OID ( [*identificador de objeto*](../secgloss/o-gly.md) ).                 |
| [**HCRYPTOIDFUNCSET**](hcryptoidfuncset.md)                                   | Representa um identificador para um conjunto de funções que podem ser instaladas usando um OID.                                                                                                                 |
| [**HCRYPTPROV \_ herdado**](hcryptprov-legacy.md)                                | Usado para substituir o tipo de dados [**HCRYPTPROV**](hcryptprov.md) em que o tipo de dados **HCRYPTPROV** não é mais usado.                                                                             |
| [**\_identificador de chave HCRYPTPROV ou \_ NCRYPT \_ \_**](hcryptprov-or-ncrypt-key-handle.md) | Especifica um identificador para um [*provedor de serviços de criptografia*](../secgloss/c-gly.md) (CSP) ou CSP CNG do CryptoAPI. |
| [**HCRYPTPROV**](hcryptprov.md)                                               | Especifica um identificador para um CSP CryptoAPI.                                                                                                                                                           |
| [**identificador de KEYSVCC \_**](keysvcc-handle.md)                                      | Especifica identificadores para o serviço de chave.                                                                                                                                                            |
