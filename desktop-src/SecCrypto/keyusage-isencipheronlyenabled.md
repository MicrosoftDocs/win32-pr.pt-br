# <a name="keyusageisencipheronlyenabled-property"></a>Propriedade KeyUsage. IsEncipherOnlyEnabled

\[A propriedade **IsEncipherOnlyEnabled** está disponível para uso nos sistemas operacionais especificados na seção requisitos. Em vez disso, use a [**classe X509EnhancedKeyUsageExtension**](/dotnet/api/system.security.cryptography.x509certificates.x509enhancedkeyusageextension?view=netcore-3.1) no namespace [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) .\]

A propriedade **IsEncipherOnlyEnabled** recupera um valor booliano que indica se o bit encipherOnly está definido.

## <a name="syntax"></a>Syntax


```VB
KeyUsage.IsEncipherOnlyEnabled As Boolean
```



## <a name="property-value"></a>Valor da propriedade

Se for **true**, o bit encipherOnly será definido.
