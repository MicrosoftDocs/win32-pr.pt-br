# <a name="certificates-and-messages"></a>Certificados e mensagens

As funções criptográficas base usam [*certificados*](../secgloss/c-gly.md), e [*funções de mensagens de nível baixo*](../secgloss/l-gly.md) e funções de [*mensagens simplificadas*](../secgloss/s-gly.md) também as usam. Para enviar ou receber mensagens que incorporam certificados, use funções de mensagens de nível baixo.

Para obter exemplos de como usar certificados com as funções de mensagem de baixo nível, consulte [exemplo c programa: codificando e decodificando uma mensagem de assinatura](example-c-program-encoding-and-decoding-a-countersigned-message.md), [exemplo c programa: codificando e decodificando uma mensagem com hash](example-c-program-encoding-and-decoding-a-hashed-message.md).

Para obter exemplos de como usar certificados com as funções de mensagem simplificadas, consulte o [programa de exemplo C: usando CryptEncryptMessage e CryptDecryptMessage](example-c-program-using-cryptencryptmessage-and-cryptdecryptmessage.md).

Para obter informações sobre funções de mensagens de nível baixo, consulte [mensagens de nível inferior](low-level-messages.md).

Para obter informações sobre funções de mensagens simplificadas, consulte [mensagens simplificadas](simplified-messages.md).
