# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ herdado

O tipo de dados **\_ herdado HCRYPTPROV** é usado para substituir o tipo de dados [**HCRYPTPROV**](hcryptprov.md) em que o tipo de dados **HCRYPTPROV** não é mais usado. Os parâmetros de função e as estruturas de dados do tipo **HCRYPTPROV \_ herdado** devem ser definidos como **NULL**.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
