# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ HERDADO

O **tipo de dados HCRYPTPROV \_ LEGACY** é usado para substituir o tipo de dados [**HCRYPTPROV**](hcryptprov.md) em que o tipo de dados **HCRYPTPROV** não é mais usado. Parâmetros de função e estruturas de dados do tipo **HCRYPTPROV \_ LEGACY** devem ser definidos como **NULL.**


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
