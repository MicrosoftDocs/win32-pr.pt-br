# <a name="using-capicom"></a>Usando CAPICOM

\[CAPICOM é um componente somente de 32 bits que está disponível para uso nos seguintes sistemas operacionais: Windows Server 2008, Windows Vista e Windows XP. Em vez disso, use o .NET Framework para implementar recursos de segurança. Para obter mais informações, [consulte Alternativas ao uso de CAPICOM.](alternatives-to-using-capicom.md)\]

Esta seção inclui cenários que usam procedimentos CAPICOM.

> [!Note]  
> A criação de [*assinaturas*](../secgloss/d-gly.md) digitais e mensagens não envelopadas com CAPICOM é feita usando a criptografia PKI (Infraestrutura de Chave Pública) e só poderá ser feita se o signator ou o usuário descriptografar uma mensagem envelopada tiver acesso a um certificado com uma chave privada disponível e associada. Para descriptografar uma mensagem envelografada, um certificado com acesso à chave privada deve estar no meu armazenamento.
