# <a name="cryptxml"></a>CryptXML

O CryptXML fornece um conjunto de baixo nível de APIs nativas que implementam a criação e a verificação de assinatura digital XML com base na recomendação W3C de Sintaxe e Processamento de Assinatura XML (Segunda Edição). Os desenvolvedores precisam estar familiarizados com esse padrão para usar efetivamente as APIs de assinatura digital cryptXML.

CryptXML contém os seguintes tópicos:

-   [Visão geral da assinatura digital XML](xml-digital-signature-overview.md)
-   [Funcionalidade da API de Assinatura Digital XML](xml-digital-signature-api-functionality.md)
-   [Algoritmos criptográficos de assinatura digital XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Extensões criptográficas de assinatura digital XML](xml-digital-signature-cryptographic-extensions.md)
-   [Considerações sobre segurança de assinatura digital XML](xml-digital-signature-security-considerations.md)
