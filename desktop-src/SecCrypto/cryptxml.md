# <a name="cryptxml"></a>CryptXML

O CryptXML fornece um conjunto de nível baixo de APIs nativas que implementam a criação e a verificação da assinatura digital XML com base na recomendação do W3C de sintaxe e processamento (segunda edição) de assinatura XML. Os desenvolvedores precisam estar familiarizados com esse padrão para usar efetivamente as APIs de assinatura digital do CryptXML.

CryptXML contém os seguintes tópicos:

-   [Visão geral da assinatura digital XML](xml-digital-signature-overview.md)
-   [Funcionalidade de API de assinatura digital XML](xml-digital-signature-api-functionality.md)
-   [Algoritmos criptográficos de assinatura digital XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Extensões criptográficas de assinatura digital XML](xml-digital-signature-cryptographic-extensions.md)
-   [Considerações de segurança de assinatura digital XML](xml-digital-signature-security-considerations.md)
