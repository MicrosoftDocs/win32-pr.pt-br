# <a name="using-a-cert_info-data-structure"></a>Usando uma estrutura de dados de informações de certificado \_

[**Certificado \_**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_context) As [**\_ informações**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_info) de contexto e certificado são as estruturas de dados principais que contêm as informações em um certificado. Quando um certificado é usado, ele pode precisar ser decodificado. Para obter mais informações, consulte [decodificando uma \_ estrutura de informações de certificado](decoding-a-cert-info-structure.md). As informações podem ser adicionadas a uma estrutura **de \_ informações de certificado** decodificada e, em seguida, a estrutura pode ser codificada novamente. Para obter detalhes, consulte [codificando uma \_ estrutura de informações de certificado](encoding-a-cert-info-structure.md).
