# <a name="hardware-cryptography"></a>Criptografia de hardware

Os serviços de certificados oferecem suporte à criptografia relacionada a hardware, como a emissão de certificados armazenados em [*cartões inteligentes*](../secgloss/s-gly.md). A AC ( [*autoridade de certificação*](../secgloss/c-gly.md) ) corporativa dá suporte à emissão de certificados de cartão inteligente para logon de domínio no sistema operacional. Além disso, a autoridade de certificação corporativa e a autoridade de certificação autônoma dão suporte à emissão de outros tipos de certificados de cartão inteligente.

Para obter detalhes, consulte a documentação do produto. Para obter informações de desenvolvedor sobre cartões inteligentes, consulte [autenticação de cartão inteligente](../secauthn/smart-card-authentication.md). Para obter informações sobre o controle que permite que um administrador solicite certificados de cartão inteligente em nome dos usuários, consulte [controle de registro de cartão inteligente](/previous-versions/windows/desktop/secsmart/smart-card-enrollment-control-usage-scenario).
