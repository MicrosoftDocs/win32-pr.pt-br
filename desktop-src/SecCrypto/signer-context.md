# <a name="signer_context-structure"></a>Estrutura de contexto do signatário \_

A estrutura de **\_ contexto do signatário** contém um [*blob*](../secgloss/b-gly.md)assinado.

> [!Note]  
> Essa estrutura não está definida em nenhum arquivo de cabeçalho. Para usar essa estrutura, você deve defini-la por conta própria, conforme mostrado neste tópico.

 

## <a name="syntax"></a>Sintaxe


```C++
typedef struct _SIGNER_CONTEXT {
  DWORD cbSize;
  DWORD cbBlob;
  BYTE  *pbBlob;
} SIGNER_CONTEXT, *PSIGNER_CONTEXT;
```



## <a name="members"></a>Membros

<dl> <dt>

**cbSize**
</dt> <dd>

O tamanho, em bytes, da estrutura.

</dd> <dt>

**cbBlob**
</dt> <dd>

O tamanho, em bytes, do membro **pbBlob** .

</dd> <dt>

**pbBlob**
</dt> <dd>

Um ponteiro para o BLOB assinado.

</dd> </dl>
