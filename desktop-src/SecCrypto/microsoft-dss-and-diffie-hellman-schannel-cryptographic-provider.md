# <a name="microsoft-dss-and-diffie-hellmanschannel-cryptographic-provider"></a>Provedor criptográfico do Microsoft DSS e Diffie-Hellman/Schannel

O provedor criptográfico do Microsoft DSS e [*Diffie-Hellman*](../secgloss/d-gly.md) / [*Schannel*](../secgloss/s-gly.md) oferece suporte a hash, assinatura de dados com DSS, geração de chaves Diffie-Hellman (d-h), troca de chaves de d-h e exportação de uma chave d-h. Esse CSP dá suporte à derivação de chave para os protocolos SSL3 e TLS1. Ele pode ser exportado para outros países/regiões.

|----------------|---------------------------------|
| Tipo de provedor: | **PROV \_ DH \_ Schannel**          |
| Nome do provedor: | **Prov Schannel do MS \_ Def \_ DH \_ \_** |
