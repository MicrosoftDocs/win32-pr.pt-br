# <a name="microsoft-dss-and-diffie-hellmanschannel-cryptographic-provider"></a>Provedor criptográfico Microsoft DSS e Diffie-Hellman/Schannel

O Provedor criptográfico microsoft DSS e [*Diffie-Hellman*](../secgloss/d-gly.md)Schannel dá suporte a hash, assinatura de dados com DSS, geração de chaves D-H (Diffie-Hellman), troca de chaves D-H e exportação de / [](../secgloss/s-gly.md) uma chave D-H. Esse CSP dá suporte à derivação de chave para os protocolos SSL3 e TLS1. Ele pode ser exportado para outros países/regiões.



|                   | Valor                       |
|-------------------|-----------------------------|
| **Tipo de provedor** | PROV \_ DH \_ SCHANNEL          |
| **Nome do provedor** | MS \_ DEF \_ DH \_ SCHANNEL \_ PROV |
