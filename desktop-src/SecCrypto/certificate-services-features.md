# <a name="certificate-services-features"></a>Recursos dos serviços de certificados

Ele tem vários recursos que o tornam valioso para as organizações que não optam por autoridades de certificação externas e que precisam de uma ferramenta flexível que possa ser adaptada às necessidades da organização.
