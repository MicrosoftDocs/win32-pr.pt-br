# <a name="systemidentityusername"></a>System.Identity.UserName

Nome de usuário para uma identidade.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Identity.UserName
   shellPKey = PKEY_Identity_UserName
   formatID = C4322503-78CA-49C6-9ACC-A68E2AFD7B6B
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
