# <a name="systemdevicesaepcontainerid"></a>System.Devices.Aep.ContainerId

ID do contêiner pai do ponto de extremidade de associação de dispositivo

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.Aep.ContainerId
   shellPKey = PKEY_Devices_Aep_ContainerId
   formatID = E7C3FB29-CAA7-4F47-8C8B-BE59B330D4C5
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
