# <a name="systemcontactdisplayotherphonenumbers"></a>System. Contact. DisplayOtherPhoneNumbers

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1

```
propertyDescription
   name = System.Contact.DisplayOtherPhoneNumbers
   shellPKey = PKEY_Contact_DisplayOtherPhoneNumbers
   formatID = 03089873-8EE8-4191-BD60-D31F72B7900B
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
