# <a name="systemphotodigitalzoom"></a>System. Photo. DigitalZoom

A taxa de zoom digital quando a imagem foi capturada. Ler da câmera nas informações de EXIF (arquivo de imagem intercambiável) do arquivo. Essa propriedade é calculada de [System. Photo. DigitalZoomNumerator](./props-system-photo-digitalzoomnumerator.md) e [System. Photo. DigitalZoomDenominator](./props-system-photo-digitalzoomdenominator.md). Se o numerador do valor registrado for 0, um zoom digital não será usado.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.DigitalZoom
   shellPKey = PKEY_Photo_DigitalZoom
   formatID = F85BF840-A925-4BC2-B0C4-8E36B598679E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
