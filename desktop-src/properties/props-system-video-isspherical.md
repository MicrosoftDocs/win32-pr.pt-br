# <a name="systemvideoisspherical"></a>System. vídeo. isesférico

Indica se o arquivo de mídia tem um fluxo de vídeo esférico.

## <a name="windows-10-version-1703"></a>Windows 10, versão 1703

```
propertyDescription
   name = System.Video.IsSpherical
   shellPKey = PKEY_Video_IsSpherical
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
