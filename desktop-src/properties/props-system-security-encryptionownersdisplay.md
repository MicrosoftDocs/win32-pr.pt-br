# <a name="systemsecurityencryptionownersdisplay"></a>System. Security. EncryptionOwnersDisplay

Propriedade do arquivo 

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511

```
propertyDescription
   name = System.Security.EncryptionOwnersDisplay
   shellPKey = PKEY_Security_EncryptionOwnersDisplay
   formatID = DE621B8F-E125-43A3-A32D-5665446D632A
   propID = 25
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
