# <a name="systemdeviceswifidirectislegacydevice"></a>System. Devices. WiFiDirect. IsLegacyDevice

Indica se Wi-Fi dispositivo direto é um dispositivo herdado

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsLegacyDevice
   shellPKey = PKEY_Devices_WiFiDirect_IsLegacyDevice
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
