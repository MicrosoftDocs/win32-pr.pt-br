# <a name="systemdeviceinterfacehidversionnumber"></a>System.DeviceInterface.Hid.VersionNumber

Número de versão do dispositivo HID.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Hid.VersionNumber
   shellPKey = PKEY_DeviceInterface_Hid_VersionNumber
   formatID = CBF38310-4A17-4310-A1EB-247F0B67593B
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
