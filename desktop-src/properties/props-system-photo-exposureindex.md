# <a name="systemphotoexposureindex"></a>System. Photo. ExposureIndex

Indica o índice de exposição selecionado na câmera ou no dispositivo de entrada no momento em que a foto foi tirada. Calculado de PKEY \_ Photo \_ EXPOSUREINDEXNUMERATOR e PKEY \_ Photo \_ ExposureIndexDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureIndex
   shellPKey = PKEY_Photo_ExposureIndex
   formatID = 967B5AF8-995A-46ED-9E11-35B3C5B9782D
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
