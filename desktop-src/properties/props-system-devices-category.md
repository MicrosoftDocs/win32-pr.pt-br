# <a name="systemdevicescategory"></a>System.Devices.Category

Forma singular da categoria do dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Category
   shellPKey = PKEY_Devices_Category
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 91
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="windows-7"></a>Windows 7

```
propertyDescription
   name = System.Devices.Category
   shellPKey = PKEY_Devices_Category_Desc_Singular
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 91
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
