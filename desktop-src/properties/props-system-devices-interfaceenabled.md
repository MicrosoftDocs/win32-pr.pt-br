# <a name="systemdevicesinterfaceenabled"></a>System.Devices.InterfaceEnabled

Indica se a interface está habilitada ou não.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.InterfaceEnabled
   shellPKey = PKEY_Devices_InterfaceEnabled
   formatID = 026E516E-B814-414B-83CD-856D6FEF4822
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
