# <a name="systemcontactcompanymaintelephone"></a>System.Contact.CompanyMainTelephone

Indica o número de telefone principal da empresa do contato.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.CompanyMainTelephone
   shellPKey = PKEY_Contact_CompanyMainTelephone
   formatID = 8589E481-6040-473D-B171-7FA89C2708ED
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
