# <a name="systemdevicesmicrophonearraygeometry"></a>System.Devices.MicrophoneArray.Geometry

Dados de geometria para a matriz de microfone.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1

```
propertyDescription
   name = System.Devices.MicrophoneArray.Geometry
   shellPKey = PKEY_Devices_MicrophoneArray_Geometry
   formatID = A1829EA2-27EB-459E-935D-B2FAD7B07762
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = true
```

Os valores PKEY são definidos em Propkey.h.
