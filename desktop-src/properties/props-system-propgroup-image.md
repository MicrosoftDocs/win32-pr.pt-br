# <a name="systempropgroupimage"></a>System. propy. Image

O separador de grupo de propriedades usado em listas de propriedades para separar arquivos de imagem de outros tipos. Não use essa propriedade para obter ou definir valores. Ele é destinado apenas como um marcador.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Image
   shellPKey = PKEY_PropGroup_Image
   formatID = E3690A87-0FA8-4A2A-9A9F-FCE8827055AC
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
