# <a name="systemcalendarshowtimeastext"></a>System.Calendar.ShowTimeAsText

A forma amigável de [System.Calendar.ShowTimeAs.](./props-system-calendar-showtimeas.md) Esse valor não se destina a ser analisado programaticamente.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.ShowTimeAsText
   shellPKey = PKEY_Calendar_ShowTimeAsText
   formatID = 53DA57CF-62C0-45C4-81DE-7610BCEFD7F5
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
