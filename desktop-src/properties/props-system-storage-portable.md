# <a name="systemstorageportable"></a>Sistema. Armazenamento. Portátil

Indica se a unidade para o armazenamento é portátil.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Storage.Portable
   shellPKey = PKEY_Storage_Portable
   formatID = 4D1EBEE8-0803-4774-9842-B77DB50265E9
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
