# <a name="systemsearchautosummary"></a>System. Search. autosummary

Um resumo do sistema de pesquisa automatizado do conteúdo de texto completo de um documento, exibido na exibição de resultados da pesquisa no **Gerenciador de pesquisa**. Use essa propriedade somente ao consumir os valores em um aplicativo, não ao fornecer valores a um manipulador de propriedades.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.AutoSummary
   shellPKey = PKEY_Search_AutoSummary
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
