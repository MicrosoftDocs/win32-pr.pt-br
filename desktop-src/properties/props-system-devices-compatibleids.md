# <a name="systemdevicescompatibleids"></a>System.Devices.CompatibleIds

IDs compatíveis.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.CompatibleIds
   shellPKey = PKEY_Devices_CompatibleIds
   formatID = A45C254E-DF1C-4EFD-8020-67D146A850E0
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
