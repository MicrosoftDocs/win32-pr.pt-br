# <a name="systemphotomakernoteoffset"></a>System. Photo. MakerNoteOffset

O deslocamento da nota do fabricante especificada em [System. Photo. MakerNote](./props-system-photo-makernote.md). Consulte a especificação do EXIF (exchangable Image File) 2,2 para obter mais informações. Esta propriedade não deve ser exibida no Shell, mas está disponível programaticamente para aplicativos.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MakerNoteOffset
   shellPKey = PKEY_Photo_MakerNoteOffset
   formatID = 813F4124-34E6-4D17-AB3E-6B1F3C2247A1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
