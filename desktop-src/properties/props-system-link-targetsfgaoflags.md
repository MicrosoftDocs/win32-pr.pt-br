# <a name="systemlinktargetsfgaoflags"></a>System. link. TargetSFGAOFlags

Os sinalizadores [**IShellFolder:: GetAttributesOf**](/windows/win32/api/shobjidl_core/nf-shobjidl_core-ishellfolder-getattributesof) para o destino de um link, com \_ atributos SFGAO PKEYSFGAOMASK mascarados.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Link.TargetSFGAOFlags
   shellPKey = PKEY_Link_TargetSFGAOFlags
   formatID = B9B4B3FC-2B51-4A42-B5D8-324146AFCF25
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Link.TargetSFGAOFlags
   shellPKey = PKEY_Link_TargetSFGAOFlags
   formatID = B9B4B3FC-2B51-4A42-B5D8-324146AFCF25
   propID = 8
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
