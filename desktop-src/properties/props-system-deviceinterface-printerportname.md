# <a name="systemdeviceinterfaceprinterportname"></a>System.DeviceInterface.PrinterPortName

A porta em que a impressora está localizada.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DeviceInterface.PrinterPortName
   shellPKey = PKEY_DeviceInterface_PrinterPortName
   formatID = EEC7B761-6F94-41B1-949F-C729720DD13C
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
