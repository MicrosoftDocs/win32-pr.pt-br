# <a name="windows-property-system"></a>Windows Sistema de propriedades

## <a name="purpose"></a>Finalidade

O Windows property System é um sistema extensível de leitura/gravação de definições de dados que fornece uma maneira uniforme de expressar metadados sobre itens do Shell. O Windows propriedade no Windows Vista e posterior permite armazenar e recuperar metadados para itens do Shell. Um item do Shell é qualquer parte do conteúdo, como um arquivo, pasta, email ou contato. Uma propriedade é uma parte individual dos metadados associados a um item do Shell.

## <a name="developer-audience"></a>Público de desenvolvedores

Antes de começar a ler a documentação Windows do SDK do Sistema de Propriedades do Windows, você deve ter uma compreensão fundamental do seguinte:

-   COM (Component Object Model)
-   Programação de Namespace do Shell

Para ver uma introdução ao COM, consulte [Conceitos básicos do COM.](../com/com-fundamentals.md) Para ver uma introdução à programação do Namespace do Shell, [consulte Ponto de Partida com o Namespace do Shell](../shell/namespace-intro.md).

Para usos do sistema Windows propriedade, consulte Visão geral do sistema de [propriedades: cenários de desenvolvimento.](property-system-overview.md)

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O ambiente de tempo de Windows com suporte para usar o sistema de propriedades Windows é Windows Vista ou posterior e o SDK (Software Development Kit) do Windows.
