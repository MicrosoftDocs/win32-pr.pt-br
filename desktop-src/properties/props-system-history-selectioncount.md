# <a name="systemhistoryselectioncount"></a>System. History. SelectionCount

A contagem de instâncias que o usuário selecionou o item.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1

```
propertyDescription
   name = System.History.SelectionCount
   shellPKey = PKEY_History_SelectionCount
   formatID = 1CE0D6BC-536C-4600-B0DD-7E0C66B350D5
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
