# <a name="systemcontacthomeaddresspostofficebox"></a>System. Contact. HomeAddressPostOfficeBox

Endereço residencial caixa postal número do contato.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.HomeAddressPostOfficeBox
   shellPKey = PKEY_Contact_HomeAddressPostOfficeBox
   formatID = 7B9F6399-0A3F-4B12-89BD-4ADC51C918AF
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
