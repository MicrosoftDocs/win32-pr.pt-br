# <a name="systemdevicesdiscoverymethod"></a>System.Devices.DiscoveryMethod

Indica o transporte ou a conexão física na qual o dispositivo é descoberto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.DiscoveryMethod
   shellPKey = PKEY_Devices_DiscoveryMethod
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 52
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
