# <a name="systemdevicesdnssddomain"></a>System. Devices. dnssd. Domain

Parte do domínio do nome da instância do serviço DNS-SD. (por exemplo, ". local" em "MyService. \_ http. \_ TCP. local ")

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.Dnssd.Domain
   shellPKey = PKEY_Devices_Dnssd_Domain
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
