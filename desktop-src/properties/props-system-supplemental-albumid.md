# <a name="systemsupplementalalbumid"></a>System. suplementar. albumid

Contém os identificadores dos álbuns dos quais o item é membro. Pode ser usado em conjunto com o item de álbum nas APIs do indexador de conteúdo para notificar outros aplicativos sobre álbuns de imagem que o usuário criou ou que os aplicativos já foram criados.

## <a name="windows-10-version-1703"></a>Windows 10, versão 1703

```
propertyDescription
   name = System.Supplemental.AlbumID
   shellPKey = PKEY_Supplemental_AlbumID
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
