# <a name="systemvolumeisroot"></a>System.Volume.IsRoot

Um valor booliana que indica se um volume é um volume raiz.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Volume.IsRoot
   shellPKey = PKEY_Volume_IsRoot
   formatID = 9B174B35-40FF-11D2-A27E-00C04FC30871
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
