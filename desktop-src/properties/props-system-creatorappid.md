# <a name="systemcreatorappid"></a>System.CreatorAppId

A AppId do aplicativo que criou esse arquivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.CreatorAppId
   shellPKey = PKEY_CreatorAppId
   formatID = C2EA046E-033C-4E91-BD5B-D4942F6BBE49
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
