# <a name="systemitemnamedisplaywithoutextension"></a>System. ItemNameDisplayWithoutExtension

Isso é semelhante a System. ItemNameDisplay, exceto pelo fato de que ele nunca inclui uma extensão de arquivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.ItemNameDisplayWithoutExtension
   shellPKey = PKEY_ItemNameDisplayWithoutExtension
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 24
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
