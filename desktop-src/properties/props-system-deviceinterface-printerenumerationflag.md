# <a name="systemdeviceinterfaceprinterenumerationflag"></a>System.DeviceInterface.PrinterEnumerationFlag

Sinalizador de Enumeração de Impressora de informações da impressora.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.DeviceInterface.PrinterEnumerationFlag
   shellPKey = PKEY_DeviceInterface_PrinterEnumerationFlag
   formatID = A00742A1-CD8C-4B37-95AB-70755587767A
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
