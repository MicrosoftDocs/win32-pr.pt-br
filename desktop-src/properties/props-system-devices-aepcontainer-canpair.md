# <a name="systemdevicesaepcontainercanpair"></a>System.Devices.AepContainer.CanPair

Se um dos pontos de extremidade de associação de dispositivo filho pode ser emparelhado com o sistema ou não.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.AepContainer.CanPair
   shellPKey = PKEY_Devices_AepContainer_CanPair
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
