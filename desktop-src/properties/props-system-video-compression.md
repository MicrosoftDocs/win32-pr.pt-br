# <a name="systemvideocompression"></a>System. vídeo. Compression

Especifica o formato de compactação de vídeo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Video.Compression
   shellPKey = PKEY_Video_Compression
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 10
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

O valor dessa propriedade é uma representação de cadeia de caracteres de um GUID na forma canônica: "{xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx}". O GUID especifica o formato de vídeo. Para obter os valores possíveis, consulte [GUIDs de subtipo de vídeo](../medfound/video-subtype-guids.md).

Os valores de PKEY são definidos em Propkey. h.
