# <a name="systemdevicesaepserviceserviceclassid"></a>System.Devices.AepService.ServiceClassId

Identidade do serviço que este serviço de ponto de extremidade de associação de dispositivo representa

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.AepService.ServiceClassId
   shellPKey = PKEY_Devices_AepService_ServiceClassId
   formatID = 71724756-3E74-4432-9B59-E7B2F668A593
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
