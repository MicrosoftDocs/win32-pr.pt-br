# <a name="systemdevicesdevicehasproblem"></a>System.Devices.DeviceHasProblem

O dispositivo tem um problema.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.DeviceHasProblem
   shellPKey = PKEY_Devices_DeviceHasProblem
   formatID = 540B947E-8B40-45BC-A8A2-6A0B894CBDA2
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
