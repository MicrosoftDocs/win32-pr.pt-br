# <a name="systemdevicesaepcontainersupportslimiteddiscovery"></a>System. Devices. AepContainer. SupportsLimitedDiscovery

Indica se o contêiner de ponto de extremidade de associação de dispositivo dá suporte à descoberta limitada

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, versão 1703, Windows 10, versão 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsLimitedDiscovery
   shellPKey = PKEY_Devices_AepContainer_SupportsLimitedDiscovery
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
