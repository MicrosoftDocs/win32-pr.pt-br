# <a name="systemdevicescategoryids"></a>System.Devices.CategoryIds

Indica a categoria bruta real

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.CategoryIds
   shellPKey = PKEY_Devices_CategoryIds
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 90
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
