# <a name="systemtitlesortoverride"></a>System.TitleSortOverride

Esse valor de cadeia de caracteres opcional permite a substituição da ordem de classificação padrão de System.Title. Isso é muito importante para a classificação adequada de arquivos de música em japonês que não podem ser ordenados phoneticalmente (a ordenação esperada pelo usuário) sem esse campo. Ele também pode ser usado para personalizar a classificação em cenários que não são do Leste da Ásia, como permitir que o usuário remova artigos para fins de classificação.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.TitleSortOverride
   shellPKey = PKEY_TitleSortOverride
   formatID = F0F7984D-222E-4AD2-82AB-1DD8EA40E57E
   propID = 300
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
