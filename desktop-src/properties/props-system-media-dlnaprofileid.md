# <a name="systemmediadlnaprofileid"></a>System. Media. DlnaProfileID

A ID do perfil DLNA para conteúdo de mídia, definida por padrões de DLNA.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Media.DlnaProfileID
   shellPKey = PKEY_Media_DlnaProfileID
   formatID = CFA31B45-525D-4998-BB44-3F7D81542FA4
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
