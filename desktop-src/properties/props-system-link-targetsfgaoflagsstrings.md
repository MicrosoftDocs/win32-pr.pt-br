# <a name="systemlinktargetsfgaoflagsstrings"></a>System. link. TargetSFGAOFlagsStrings

Expressa os sinalizadores SFGAO de um link como valores de cadeia de caracteres e é usado como uma otimização de consulta. Consulte PKEY \_ shell \_ SFGAOFlagsStrings para obter os possíveis valores dessa propriedade.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Link.TargetSFGAOFlagsStrings
   shellPKey = PKEY_Link_TargetSFGAOFlagsStrings
   formatID = D6942081-D53B-443D-AD47-5E059D9CD27A
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
