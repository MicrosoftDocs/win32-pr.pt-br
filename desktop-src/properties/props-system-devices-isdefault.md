# <a name="systemdevicesisdefault"></a>System.Devices.IsDefault

Se essa propriedade for true, o dispositivo será o dispositivo padrão.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.IsDefault
   shellPKey = PKEY_Devices_IsDefault
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 86
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Default
            value = #TRUE#
            text = Default
            defineToken = ISDEFAULTDEVICE_DEFAULTDEVICE
         enum
            name = NotDefault
            value = #FALSE#
            text = Not Default
            defineToken = ISDEFAULTDEVICE_NOTDEFAULTDEVICE
```

## <a name="windows-7"></a>Windows 7

```
propertyDescription
   name = System.Devices.IsDefault
   shellPKey = PKEY_Devices_IsDefaultDevice
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 86
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Default
            value = #TRUE#
            text = Default
            defineToken = ISDEFAULTDEVICE_DEFAULTDEVICE
         enum
            name = NotDefault
            value = #FALSE#
            text = Not Default
            defineToken = ISDEFAULTDEVICE_NOTDEFAULTDEVICE
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
