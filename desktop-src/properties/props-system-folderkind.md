# <a name="systemfolderkind"></a>System.FolderKind

Essa propriedade representa os tipos de conteúdo armazenados nessa pasta especificada pelo provedor de armazenamento. Cada tipo de pasta deve ser um dos valores conhecidos especificados por System.Kind definitionSystem.FolderKind é uma propriedade somente leitura, deve ser atualizada apenas pelo provedor de armazenamento.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1

```
propertyDescription
   name = System.FolderKind
   shellPKey = PKEY_FolderKind
   formatID = FCEFF153-E839-4CF3-A9E7-EA22832094B8
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
