# <a name="systemdeviceswifidirectinterfaceguid"></a>System. Devices. WiFiDirect. InterfaceGuid

GUID de interface direta do Wi-Fi

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.InterfaceGuid
   shellPKey = PKEY_Devices_WiFiDirect_InterfaceGuid
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
