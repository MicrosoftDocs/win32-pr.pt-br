# <a name="systemsearchrank"></a>Sistema. Search. Rank

Classificação de relevância da linha, com um intervalo de 0-1000. Números maiores significam correspondências melhores. Somente tempo de consulta; Não definido no esquema de pesquisa. Essa propriedade é recuperável, mas não é pesquisável.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.Rank
   shellPKey = PKEY_Search_Rank
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
