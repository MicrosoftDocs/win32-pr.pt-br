# <a name="systemphotoflashmanufacturer"></a>System. Photo. FlashManufacturer

Uma cadeia de caracteres que indica o fabricante do flash usado para tirar a foto. Pode ficar em branco ou não estar presente.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashManufacturer
   shellPKey = PKEY_Photo_FlashManufacturer
   formatID = AABAF6C9-E0C5-4719-8585-57B103E584FE
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
