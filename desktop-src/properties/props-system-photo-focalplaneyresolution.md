# <a name="systemphotofocalplaneyresolution"></a>System. Photo. FocalPlaneYResolution

Indica o número de pixels na altura da imagem (direção Y) por FocalPlaneResolutionUnit no plano focal da câmera. Calculado de PKEY \_ Photo \_ FOCALPLANEYRESOLUTIONNUMERATOR e PKEY \_ Photo \_ FocalPlaneYResolutionDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalPlaneYResolution
   shellPKey = PKEY_Photo_FocalPlaneYResolution
   formatID = 4FFFE4D0-914F-4AC4-8D6F-C9C61DE169B1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
