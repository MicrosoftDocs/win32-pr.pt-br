# <a name="systemproplistnonpersonal"></a>Sistema. PropList. não Personal

A lista de propriedades é considerada ' não pessoal '. O sistema deixará essas propriedades inalteradas quando instruído a remover todas as propriedades não pessoais de um determinado arquivo. Registre-se sob o regvalue de "não-Person".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.NonPersonal
   shellPKey = PKEY_PropList_NonPersonal
   formatID = 49D1091F-082E-493F-B23F-D2308AA9668C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
