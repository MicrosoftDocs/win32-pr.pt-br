# <a name="systemdeviceinterfacebluetoothflags"></a>System. DeviceInterface. Bluetooth. Flags

Sinalizadores de dispositivo Bluetooth.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.Flags
   shellPKey = PKEY_DeviceInterface_Bluetooth_Flags
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
