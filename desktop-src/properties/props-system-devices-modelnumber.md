# <a name="systemdevicesmodelnumber"></a>System.Devices.ModelNumber

Número do modelo do dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.ModelNumber
   shellPKey = PKEY_Devices_ModelNumber
   formatID = 656A3BB3-ECC0-43FD-8477-4AE0404A96CD
   propID = 8195
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```
