# <a name="systemidentitykeyprovidername"></a>System.Identity.KeyProviderName

Nome do provedor de chave de identidade

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.KeyProviderName
   shellPKey = PKEY_Identity_KeyProviderName
   formatID = A26F4AFC-7346-4299-BE47-EB1AE613139F
   propID = 16
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
