# <a name="systemcalendaroptionalattendeeaddresses"></a>System.Calendar.OptionalAttendeeAddresses

Endereços de todos os participantes opcionais.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.OptionalAttendeeAddresses
   shellPKey = PKEY_Calendar_OptionalAttendeeAddresses
   formatID = D55BAE5A-3892-417A-A649-C6AC5AAAEAB3
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
