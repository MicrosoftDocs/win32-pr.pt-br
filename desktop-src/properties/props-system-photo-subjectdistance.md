# <a name="systemphotosubjectdistance"></a>System. Photo. SubjectDistance

A distância para o assunto em metros. Calculado de PKEY \_ Photo \_ SUBJECTDISTANCENUMERATOR e PKEY \_ Photo \_ SubjectDistanceDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.SubjectDistance
   shellPKey = PKEY_Photo_SubjectDistance
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 37382
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
