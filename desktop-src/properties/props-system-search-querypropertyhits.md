# <a name="systemsearchquerypropertyhits"></a>System. Search. QueryPropertyHits

Contém a lista de propriedades correspondentes de uma consulta.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1

```
propertyDescription
   name = System.Search.QueryPropertyHits
   shellPKey = PKEY_Search_QueryPropertyHits
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 21
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
