# <a name="systemdevicesaepservicebluetoothgattservicedevice"></a>System. Devices. AepService. Bluetooth. GattService. Device

Define o endereço do dispositivo Bluetooth para consulta.

## <a name="windows-10-version-1511"></a>Windows 10, versão 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.Device
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_Device
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
