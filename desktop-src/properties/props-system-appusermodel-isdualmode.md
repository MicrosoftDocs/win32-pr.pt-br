# <a name="systemappusermodelisdualmode"></a>System. AppUserModel. isdualmode

Indica que um aplicativo dá suporte a modos de área de trabalho dupla e de imersão. no Windows 8, essa propriedade só é aplicável a navegadores da web.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.AppUserModel.IsDualMode
   shellPKey = PKEY_AppUserModel_IsDualMode
   formatID = 9F4C2855-9F79-4B39-A8D0-E1D42DE1D5F3
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = false
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
