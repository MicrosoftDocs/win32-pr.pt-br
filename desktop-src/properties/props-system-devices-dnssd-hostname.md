# <a name="systemdevicesdnssdhostname"></a>System. Devices. dnssd. HostName

O nome DNS do dispositivo está hospedando o serviço.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.Dnssd.HostName
   shellPKey = PKEY_Devices_Dnssd_HostName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
