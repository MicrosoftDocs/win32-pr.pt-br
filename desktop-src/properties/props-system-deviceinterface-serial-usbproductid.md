# <a name="systemdeviceinterfaceserialusbproductid"></a>System. DeviceInterface. serial. UsbProductId

ID do produto USB do dispositivo serial

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.DeviceInterface.Serial.UsbProductId
   shellPKey = PKEY_DeviceInterface_Serial_UsbProductId
   formatID = 4C6BF15C-4C03-4AAC-91F5-64C0F852BCF4
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
