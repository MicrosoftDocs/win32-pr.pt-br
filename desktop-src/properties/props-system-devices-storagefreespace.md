# <a name="systemdevicesstoragefreespace"></a>System. Devices. StorageFreeSpace

Total de espaço livre no dispositivo de armazenamento.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.StorageFreeSpace
   shellPKey = PKEY_Devices_StorageFreeSpace
   formatID = 49CD1F76-5626-4B17-A4E8-18B4AA1A2213
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = StorageFreeSpace
            minValue = 0
            setValue = 0
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
