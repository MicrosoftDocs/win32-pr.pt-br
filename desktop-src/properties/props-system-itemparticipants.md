# <a name="systemitemparticipants"></a>System. os participantes

A lista genérica de pessoas associadas e contribuindo com um item. Por exemplo, essa é a combinação de pessoas na lista para, lista de CC e o remetente de uma mensagem de email.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ItemParticipants
   shellPKey = PKEY_ItemParticipants
   formatID = D4D0AA16-9948-41A4-AA85-D97FF9646993
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
