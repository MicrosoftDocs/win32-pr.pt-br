# <a name="systemmediathumbnaillargeuri"></a>System.Media.ThumbnailLargeUri

URI da representação em miniatura grande do item de mídia.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Media.ThumbnailLargeUri
   shellPKey = PKEY_Media_ThumbnailLargeUri
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 48
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
