# <a name="systemrecordedtvchannelnumber"></a>System.RecordedTV.ChannelNumber

Os canais de TV gravados. Por exemplo, 42, 5, 53.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.RecordedTV.ChannelNumber
   shellPKey = PKEY_RecordedTV_ChannelNumber
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.RecordedTV.ChannelNumber
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
