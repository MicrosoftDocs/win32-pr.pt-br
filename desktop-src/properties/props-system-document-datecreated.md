# <a name="systemdocumentdatecreated"></a>System.Document. DateCreated

Indica a data e a hora em que um documento foi criado. Essas informações são armazenadas no documento, não obtidas do sistema de arquivos.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Document.DateCreated
   shellPKey = PKEY_Document_DateCreated
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Document.DateCreated
   shellPKey = PKEY_Document_DateCreated
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 12
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
