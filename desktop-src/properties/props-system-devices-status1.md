Primeira linha do status do dispositivo.

```
propertyDescription
   name = System.Devices.Status1
   shellPKey = PKEY_Devices_Status1
   formatID = D08DD4C0-3A9E-462E-8290-7B636B2576B9
   propID = 257
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```
