# <a name="systemparentalratingreason"></a>System. ParentalRatingReason

Explica as classificações de arquivo. Por exemplo, "violência, linguagem fracasso" para uma classificação de R no sistema de classificação MPAA.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRatingReason
   shellPKey = PKEY_ParentalRatingReason
   formatID = 10984E0A-F9F2-4321-B7EF-BAF195AF4319
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
