# <a name="systemphotogaincontrolnumerator"></a>System. Photo. GainControlNumerator

O numerador do PKEY \_ Photo \_ GainControl.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.GainControlNumerator
   shellPKey = PKEY_Photo_GainControlNumerator
   formatID = 8E8ECF7C-B7B8-4EB8-A63F-0EE715C96F9E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```
