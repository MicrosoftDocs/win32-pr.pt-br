# <a name="systemparentalratingsorganization"></a>System. ParentalRatingsOrganization

O nome da organização cujo sistema de classificação é usado para [System. ParentalRating](./props-system-parentalrating.md).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRatingsOrganization
   shellPKey = PKEY_ParentalRatingsOrganization
   formatID = A7FE0840-1344-46F0-8D37-52ED712A4BF9
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
