# <a name="systemappusermodeltoastactivatorclsid"></a>System. AppUserModel. ToastActivatorCLSID

Usado para Cocriar uma interface INotificationActivationCallback para notificar sobre ativações do sistema.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.AppUserModel.ToastActivatorCLSID
   shellPKey = PKEY_AppUserModel_ToastActivatorCLSID
   formatID = 9F4C2855-9F79-4B39-A8D0-E1D42DE1D5F3
   propID = 26
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = false
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
