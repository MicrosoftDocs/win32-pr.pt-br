# <a name="systemdataobjectformat"></a>System.DataObjectFormat

O formato do objeto de dados. Um valor de cadeia de caracteres que é o nome do formato da área de transferência.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.DataObjectFormat
   shellPKey = PKEY_DataObjectFormat
   formatID = 1E81A3F8-A30F-4247-B9EE-1D0368A9425C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
