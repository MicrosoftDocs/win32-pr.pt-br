# <a name="property-system-code-samples"></a>Exemplos de código do sistema de propriedades

Os exemplos de código a seguir demonstram como usar o sistema de Windows propriedade.

| Tópico            | Sumário                     |
|-------------------|---------------------------------------------------|
| [Exemplo de manipulador de propriedades ideal](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/IdealPropertyHandler)         | Demonstra a implementação de um manipulador de propriedades ideal para um formato de arquivo OleDoc (.docfile-ms) ou um formato OpenMetadata fictício (.openmetadata-ms) que dá suporte à leitura e à escrita de propriedades e esquema personalizado. |
| [Exemplo de manipulador de propriedades de playlist](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/PlaylistPropertyHandler)   | Demonstra a implementação de um manipulador de propriedades para os formatos de arquivo de playlist .wpl e .zpl.                                                                                                                          |
| [Exemplo de edição de propriedade](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/PropertyEdit)                          | Demonstra como usar as APIs do sistema Windows propriedade para ler e gravar valores de propriedade de arquivo.                                                                                                                            |
| [Exemplo de esquema de propriedade](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform/propertyschemas)                      | Demonstra como registrar e não registrar, propriedades personalizadas e como obter informações detalhadas sobre esquemas de propriedade personalizados que já estão registrados no sistema.                                               |
| [Exemplo de manipulador de propriedades recipe](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration/RecipePropertyHandler) | Demonstra a implementação de um manipulador de propriedades para arquivos de receita de formato personalizado (.recipe).                                                                                                                             |

Para obter mais exemplos relacionados, consulte o seguinte:

-   [Windows Exemplos de aplicativo do Shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appplatform)
-   [Windows Exemplos de extensibilidade do shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/shellextensibility)
-   [Windows Exemplos de integração do Shell](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/winui/shell/appshellintegration)
