# <a name="systemimageresolutionunit"></a>System. Image. ResolutionUnit

Indica as unidades de resolução. Usado para imagens com uma taxa de proporção não quadrada, mas sem dimensões absolutas significativas. 1 = nenhuma unidade absoluta de medida. 2 = polegadas. 3 = centímetros. O valor padrão é 2 (polegadas).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.ResolutionUnit
   shellPKey = PKEY_Image_ResolutionUnit
   formatID = 19B51FA6-1F92-4A5C-AB48-7DF0ABD67444
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Int16
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
