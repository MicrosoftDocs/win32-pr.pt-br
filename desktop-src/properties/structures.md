# <a name="structures-windows-property-system"></a>Estruturas (Windows de propriedades)

Esta seção descreve as estruturas Windows sistema de propriedades.



| Tópico                                | Sumário                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) | Especifica o identificador FMTID/PID que identifica programaticamente uma propriedade. Substitui [**SHCOLUMNID.**](../shell/objects.md)<br/> |
| [**PROPPRG**](/windows/desktop/api/Shlobj_core/ns-shlobj_core-propprg)           | Essa estrutura contém informações de um arquivo .pif. Ele é usado pelo [**PifMgr \_ GetProperties.**](/windows/desktop/api/Shlobj_core/nf-shlobj_core-pifmgr_getproperties)<br/>       |
