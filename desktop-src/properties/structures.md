# <a name="structures-windows-property-system"></a>Estruturas (sistema de propriedades do Windows)

Esta seção descreve as estruturas do sistema de propriedades do Windows.



| Tópico                                | Sumário                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) | Especifica o identificador FMTID/PID que identifica de forma programática uma propriedade. Substitui [**SHCOLUMNID**](../shell/objects.md).<br/> |
| [**PROPPRG**](/windows/desktop/api/Shlobj_core/ns-shlobj_core-propprg)           | Essa estrutura contém informações de um arquivo. pif. Ele é usado por [**PifMgr \_ GetProperties**](/windows/desktop/api/Shlobj_core/nf-shlobj_core-pifmgr_getproperties).<br/>       |
