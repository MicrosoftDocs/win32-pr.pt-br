# <a name="systemdevicesconnected"></a>System.Devices.Connected

Estado de conexão do dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Connected
   shellPKey = PKEY_Devices_Connected
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 55
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-7"></a>Windows 7

```
propertyDescription
   name = System.Devices.Connected
   shellPKey = PKEY_Devices_IsConnected
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 55
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
