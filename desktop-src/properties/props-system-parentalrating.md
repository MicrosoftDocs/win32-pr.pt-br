# <a name="systemparentalrating"></a>System. ParentalRating

A classificação dos pais armazenada em um formato normalmente determinado pela organização chamada em [System. ParentalRatingsOrganization](./props-system-parentalratingsorganization.md).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRating
   shellPKey = PKEY_ParentalRating
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 21
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.

Essa cadeia de caracteres é exibida diretamente para o usuário e, portanto, palavras amigáveis devem ser usadas. Por exemplo, "restrito por R" ao basear as classificações no sistema de classificação da MPAA (Associação de imagem de movimento da América).
