# <a name="systemproplistextendedtileinfo"></a>System. PropList. ExtendedTileInfo

A lista de propriedades a ser mostrada na ListView em blocos estendidos. Registre-se sob o regvalue de "ExtendedTileInfo".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.ExtendedTileInfo
   shellPKey = PKEY_PropList_ExtendedTileInfo
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
