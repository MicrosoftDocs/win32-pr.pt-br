# <a name="systemdevicesaepservicefriendlyname"></a>System.Devices.AepService.FriendlyName

Nome Amigável do Serviço de Ponto de Extremidade de Associação de Dispositivo

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.AepService.FriendlyName
   shellPKey = PKEY_Devices_AepService_FriendlyName
   formatID = 71724756-3E74-4432-9B59-E7B2F668A593
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```
