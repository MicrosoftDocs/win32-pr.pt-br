# <a name="systemdevicesaepbluetoothleisconnectable"></a>System.Devices.Aep. Bluetooth. Le.IsConnectable

Se o Bluetooth LE está anunciando um anúncio conectável no momento.

## <a name="windows-10-version-1703"></a>Windows 10, versão 1703

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Le.IsConnectable
   shellPKey = PKEY_Devices_Aep_Bluetooth_Le_IsConnectable
   formatID = 995EF0B0-7EB3-4A8B-B9CE-068BB3F4AF69
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
