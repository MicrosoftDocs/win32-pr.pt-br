# <a name="systemphotoorientationtext"></a>System. Photo. OrientationText

A forma amigável de [System. Photo. Orientation](./props-system-photo-orientation.md). Não destinado a ser analisado programaticamente.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.OrientationText
   shellPKey = PKEY_Photo_OrientationText
   formatID = A9EA193C-C511-498A-A06B-58E2776DCC28
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
