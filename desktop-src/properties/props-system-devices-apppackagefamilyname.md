# <a name="systemdevicesapppackagefamilyname"></a>System.Devices.AppPackageFamilyName

O nome da família de pacotes registrado como o aplicativo para este dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.AppPackageFamilyName
   shellPKey = PKEY_Devices_AppPackageFamilyName
   formatID = 51236583-0C4A-4FE8-B81F-166AEC13F510
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
