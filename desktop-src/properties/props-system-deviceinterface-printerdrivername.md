# <a name="systemdeviceinterfaceprinterdrivername"></a>System.DeviceInterface.PrinterDriverName

O nome do arquivo do driver de impressora.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DeviceInterface.PrinterDriverName
   shellPKey = PKEY_DeviceInterface_PrinterDriverName
   formatID = AFC47170-14F5-498C-8F30-B0D19BE449C6
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
