# <a name="systemdevicesaepcontainersupportsnetworking"></a>System.Devices.AepContainer.SupportsNetworking

Indica se o contêiner de ponto de extremidade de associação de dispositivo dá suporte a supportsNetworking

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, versão 1703, Windows 10, versão 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsNetworking
   shellPKey = PKEY_Devices_AepContainer_SupportsNetworking
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
