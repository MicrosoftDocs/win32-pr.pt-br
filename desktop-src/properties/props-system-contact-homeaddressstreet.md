# <a name="systemcontacthomeaddressstreet"></a>System.Contact.HomeAddressDress

Nome da rua do endereço residencial do contato.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.HomeAddressStreet
   shellPKey = PKEY_Contact_HomeAddressStreet
   formatID = 0ADEF160-DB3F-4308-9A21-06237B16FA2A
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
