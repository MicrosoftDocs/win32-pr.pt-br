# <a name="systemdeviceinterfacewinusbusbproductid"></a>System. DeviceInterface. WinUsb. UsbProductId

ID do produto do descritor de dispositivo USB do dispositivo USB.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbProductId
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbProductId
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
