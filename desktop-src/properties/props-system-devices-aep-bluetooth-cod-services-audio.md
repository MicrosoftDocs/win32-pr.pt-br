# <a name="systemdevicesaepbluetoothcodservicesaudio"></a>System. Devices. AEP. Bluetooth. Cod. Services. Audio

Bluetooth classe de áudio do serviço de dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Services.Audio
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Services_Audio
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
