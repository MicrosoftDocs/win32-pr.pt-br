# <a name="systemcategory"></a>System. Category

Preterido. A categoria que pode ser atribuída a um item como um documento ou arquivo. Essa propriedade é herdada de propriedades de documento OLE e é preterida para o Windows Vista. Palavras-chave devem ser usadas em vez disso. O código mais antigo trata essa propriedade como VT \_ LPSTR.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Category
   shellPKey = PKEY_Category
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
