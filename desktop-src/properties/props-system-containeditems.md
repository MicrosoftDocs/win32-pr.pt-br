# <a name="systemcontaineditems"></a>System.ContainedItems

Uma lista do tipo de conteúdo no item. Esse valor é representado como uma matriz de vetor de GUIDs, em que cada GUID representa um determinado tipo, como URLs ou anexos.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ContainedItems
   shellPKey = PKEY_ContainedItems
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 29
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue Guid
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
