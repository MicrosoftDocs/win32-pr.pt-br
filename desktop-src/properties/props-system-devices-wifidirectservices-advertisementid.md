# <a name="systemdeviceswifidirectservicesadvertisementid"></a>System. Devices. WiFiDirectServices. Advertisementfileid

ID do anúncio do Wi-Fi Direct Services

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.WiFiDirectServices.AdvertisementId
   shellPKey = PKEY_Devices_WiFiDirectServices_AdvertisementId
   formatID = 31B37743-7C5E-4005-93E6-E953F92B82E9
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
