# <a name="systemlayoutpatterncontentviewmodeforbrowse"></a>System.LayoutPattern.ContentViewModeForBrowse

Identifica o padrão de layout que o modo de exibição de conteúdo deve aplicar a esse item no contexto de navegação.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.LayoutPattern.ContentViewModeForBrowse
   shellPKey = PKEY_LayoutPattern_ContentViewModeForBrowse
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 500
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Alpha
            value = alpha
            text
            defineToken = LAYOUTPATTERN_CVMFB_ALPHA
         enum
            name = Beta
            value = beta
            text
            defineToken = LAYOUTPATTERN_CVMFB_BETA
         enum
            name = Gamma
            value = gamma
            text
            defineToken = LAYOUTPATTERN_CVMFB_GAMMA
         enum
            name = Delta
            value = delta
            text
            defineToken = LAYOUTPATTERN_CVMFB_DELTA
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
