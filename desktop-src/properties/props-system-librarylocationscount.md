# <a name="systemlibrarylocationscount"></a>System. LibraryLocationsCount

Indica o número de locais de biblioteca.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.LibraryLocationsCount
   shellPKey = PKEY_LibraryLocationsCount
   formatID = 908696C7-8F87-44F2-80ED-A8C1C6894575
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
