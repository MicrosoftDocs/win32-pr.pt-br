# <a name="systemstorageproviderfilechecksum"></a>System. StorageProviderFileChecksum

A soma de verificação computada pelo provedor de armazenamento para o arquivo. Os arquivos com o mesmo valor de soma de verificação terão o mesmo conteúdo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.StorageProviderFileChecksum
   shellPKey = PKEY_StorageProviderFileChecksum
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = false
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
