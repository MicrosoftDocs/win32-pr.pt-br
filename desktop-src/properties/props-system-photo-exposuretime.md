# <a name="systemphotoexposuretime"></a>Sistema. Photo. Exposiçãotime

O tempo de exposição para a foto, em segundos, como lido das informações de EXIF (arquivo de imagem intercambiável). Essa propriedade é calculada de [System. Photo. ExposureTimeNumerator](./props-system-photo-exposuretimenumerator.md) e [System. Photo. ExposureTimeDenominator](./props-system-photo-exposuretimedenominator.md).

A seguir está uma lista de possíveis valores obtidos da especificação EXIF 2,2.

-   30
-   15
-   8
-   4
-   2
-   1
-   1/2
-   1/4
-   1/8
-   1/15
-   1/30
-   1/60
-   1/125
-   1/250
-   1/500
-   1/1000
-   1/2000

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureTime
   shellPKey = PKEY_Photo_ExposureTime
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 33434
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
