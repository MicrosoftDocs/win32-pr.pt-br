# <a name="systemdevicesdnssdservicename"></a>System.Devices.Dnssd.ServiceName

Parte do tipo de serviço do nome da instância de serviço DNS-SD. (por exemplo, " \_ http. \_ tcp" em "myservice. \_ http. \_ tcp.local")

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.Dnssd.ServiceName
   shellPKey = PKEY_Devices_Dnssd_ServiceName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
