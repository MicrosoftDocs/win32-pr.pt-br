# <a name="systemphotoexposurebias"></a>System. Photo. ExposureBias

A quantidade de tendência de exposição usada na foto, como lida da câmera. Essa propriedade é calculada de [System. Photo. ExposureBiasNumerator](./props-system-photo-exposurebiasnumerator.md) e [System. Photo. ExposureBiasDenominator](./props-system-photo-exposurebiasdenominator.md). A unidade é o valor de APEX. Normalmente, ele varia de-99,99 a 99,99.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureBias
   shellPKey = PKEY_Photo_ExposureBias
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 37380
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
