# <a name="systemgpstrackref"></a>System.GPS.TrackRef

Indica a referência para a direção do movimento do receptor de GPS (por exemplo, direção verdadeira, direção magnética).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.TrackRef
   shellPKey = PKEY_GPS_TrackRef
   formatID = 35DBE6FE-44C3-4400-AAAE-D2C799C407E8
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
