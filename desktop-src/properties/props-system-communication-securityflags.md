# <a name="systemcommunicationsecurityflags"></a>System. Communication. SecurityFlags

Sinalizadores de segurança associados ao item para indicar se o item está criptografado, assinado ou habilitado para DRM.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.SecurityFlags
   shellPKey = PKEY_Communication_SecurityFlags
   formatID = 8619A4B6-9F4D-4429-8C0F-B996CA59E335
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
