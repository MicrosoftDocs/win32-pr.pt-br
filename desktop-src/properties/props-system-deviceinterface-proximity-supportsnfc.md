# <a name="systemdeviceinterfaceproximitysupportsnfc"></a>System.DeviceInterface.Proximity.SupportsNfc

Indica se o dispositivo dá suporte a comunicações NFC (NDEF).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.DeviceInterface.Proximity.SupportsNfc
   shellPKey = PKEY_DeviceInterface_Proximity_SupportsNfc
   formatID = FB3842CD-9E2A-4F83-8FCC-4B0761139AE9
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
