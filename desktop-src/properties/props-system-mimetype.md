# <a name="systemmimetype"></a>System. MIMEType

O tipo MIME.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.MIMEType
   shellPKey = PKEY_MIMEType
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 5
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.

Por exemplo, um arquivo EML teria um valor [System. MIMEType]() de `message/rfc822` .

Para arquivos de mídia que são compatíveis com vários tipos MIME, essa propriedade pode ter uma tendência de indicar um tipo MIME adequado para a DLNA (rede de vida digital).
