# <a name="systemdevicesstatus"></a>System. Devices. status

Matriz de cadeias de caracteres de status do dispositivo

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Status
   shellPKey = PKEY_Devices_Status
   formatID = D08DD4C0-3A9E-462E-8290-7B636B2576B9
   propID = 259
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
