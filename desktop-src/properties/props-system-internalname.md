# <a name="systeminternalname"></a>System. InternalName

O nome de um arquivo. exe ou. dll como armazenado em uma seção de recurso dentro desse arquivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.InternalName
   shellPKey = PKEY_InternalName
   formatID = 0CEF7D53-FA64-11D1-A203-0000F81FEDEE
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.

Historicamente, essas informações eram geralmente exibidas na página da folha de propriedades da **versão** XPÂ do Windows. No Windows Vista, as informações foram movidas para a guia **detalhes** . No entanto, essa propriedade específica não era publicada no Windows Vista até o Service Pack 1 (SP1).
