# <a name="systemappzoneidentifier"></a>System. AppZoneIdentifier

Marca do contêiner do aplicativo. O identificador de zona, conforme determinado pelo último gravador do arquivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.AppZoneIdentifier
   shellPKey = PKEY_AppZoneIdentifier
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 102
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = false
```

## <a name="remarks"></a>Comentários

Os valores de PKEY são definidos em Propkey. h.
