# <a name="systemlinktargeturlhostname"></a>System. link. TargetUrlHostName

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Link.TargetUrlHostName
   shellPKey = PKEY_Link_TargetUrlHostName
   formatID = 8A2F99F9-3C37-465D-A8D7-69777A246D0C
   propID = 5
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

Os valores de PKEY são definidos em Propkey. h.
