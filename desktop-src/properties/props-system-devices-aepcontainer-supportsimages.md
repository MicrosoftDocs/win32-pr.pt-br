# <a name="systemdevicesaepcontainersupportsimages"></a>System.Devices.AepContainer.SupportsImages

Indica se o contêiner de ponto de extremidade de associação de dispositivo dá suporte à seleção de imagem

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versão 1703, Windows 10, versão 1607, Windows 10, versão 1511, Windows 10, versão 1507

```
propertyDescription
   name = System.Devices.AepContainer.SupportsImages
   shellPKey = PKEY_Devices_AepContainer_SupportsImages
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentários

Os valores PKEY são definidos em Propkey.h.
