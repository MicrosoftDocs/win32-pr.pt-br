# <a name="dmo-structures"></a>Estruturas de DMO

Esta seção descreve as estruturas do objeto de mídia do DirectX (DMO).



| Estrutura                                                   | Descrição                                                 |
|-------------------------------------------------------------|-------------------------------------------------------------|
| [**\_tipo de mídia DMO \_**](/previous-versions/windows/desktop/api/mediaobj/ns-mediaobj-dmo_media_type)                  | Descreve o formato dos dados usados por um fluxo em um DMO. |
| [**\_buffer de \_ dados de saída de DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ns-mediaobj-dmo_output_data_buffer) | Descreve um buffer de saída usado por um DMO.                   |
| [**\_MediaType parcial de DMO \_**](/previous-versions/windows/desktop/api/Dmoreg/ns-dmoreg-dmo_partial_mediatype)    | Descreve parcialmente um tipo de mídia usado por um DMO.             |
| [**\_segmento de envelope MP \_**](/previous-versions/windows/desktop/api/Medparam/ns-medparam-mp_envelope_segment)        | Define um segmento de envelope usado por um parâmetro de mídia.      |
| [**\_PARAMINFO MP**](/previous-versions/windows/desktop/api/Medparam/ns-medparam-mp_paraminfo)                       | Contém informações sobre um parâmetro de mídia.               |
