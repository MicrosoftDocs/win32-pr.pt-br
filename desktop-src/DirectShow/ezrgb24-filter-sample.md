# <a name="ezrgb24-filter-sample"></a>Exemplo de filtro EZRGB24

## <a name="description"></a>Descrição

O filtro EZRGB24 é um filtro de transformação de processamento de imagem. Esse filtro executa vários efeitos de vídeo individuais, incluindo efeitos vermelhos, verdes, azuis, escuros, XOR, desfoque, cinza e emboss.

Este exemplo mostra como usar várias classes DirectShow base, incluindo [**CTransformFilter,**](ctransformfilter.md) [**CPersistStream**](cpersiststream.md)e [**CBasePropertyPage.**](cbasepropertypage.md)

## <a name="downloading-the-sample"></a>Baixando o exemplo

Para baixar os DirectShow exemplos do SDK do DirectShow, instale a versão mais recente [do SDK Windows .](https://msdn.microsoft.com/windowsvista/bb980924.aspx)

Este exemplo é instalado no seguinte caminho: Multimídia de Exemplos Raiz do *\[ SDK \]* \\ DirectShow \\ \\ \\ Filtros \\ EZRGB24.
