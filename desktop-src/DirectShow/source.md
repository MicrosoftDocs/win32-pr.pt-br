# <a name="source"></a>Fonte

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O objeto de origem representa um clipe de origem na linha do tempo. Para criar esse objeto, chame o [**método IAMTimeline::CreateEmptyNode.**](iamtimeline-createemptynode.md)

O objeto de origem expõe as seguintes interfaces:

-   [**IAMTimelineEffectable**](iamtimelineeffectable.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineSrc**](iamtimelinesrc.md)
-   [**IAMTimelineSplittable**](iamtimelinesplittable.md)
