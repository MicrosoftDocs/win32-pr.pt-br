# <a name="avencvideosourceisbw-property"></a>Propriedade AVEncVideoSourceIsBW

Especifica se o vídeo é monocromático (preto e branco) ou contém a cor.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoSourceIsBW**

## <a name="property-value"></a>Valor da propriedade

Se o valor for **Variant \_ true**, o vídeo será monocromático. Caso contrário, o vídeo conterá cor.
