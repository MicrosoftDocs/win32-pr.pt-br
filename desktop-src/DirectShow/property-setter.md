# <a name="property-setter"></a>Setter da propriedade

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O objeto setter de propriedade define as propriedades **IDispatch** em um objeto Timeline. Para criar esse objeto, chame **CoCreateInstance**. O identificador de classe é CLSID \_ PropertySetter.

O objeto setter de propriedade expõe as seguintes interfaces:

-   [**IAMSetErrorLog**](iamseterrorlog.md)
-   [**IPropertySetter**](ipropertysetter.md)
