# <a name="playchapterintitle"></a>PlayChapterInTitle

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `PlayChapterInTitle` evento é enviado quando o `PlayChapterInTitle` comando tiver sido habilitado ou desabilitado.

``` syntax
PlayChapterInTitle(bEnabled)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="bEnabled"></span><span id="benabled"></span><span id="BENABLED"></span>*bEnabled*
</dt> <dd>

Especifica se a operação está habilitada ou desabilitada como um booliano.

</dd> </dl>
