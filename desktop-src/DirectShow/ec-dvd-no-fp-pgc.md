# <a name="ec_dvd_no_fp_pgc"></a>DVD do EC \_ \_ sem \_ FP \_ PGC

Sinaliza que o disco DVD não tem um FP \_ PGC (primeira cadeia de programa de reprodução) e que o DVD Navigator não carregará automaticamente nenhum PGC e iniciará a reprodução.

## <a name="remarks"></a>Comentários

Esse evento é gerado a partir do primeiro domínio de reprodução.
