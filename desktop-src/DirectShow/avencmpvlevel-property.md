# <a name="avencmpvlevel-property"></a>Propriedade AVEncMPVLevel

Especifica o nível MPEG-2. Essa propriedade se aplica a codificadores de vídeo MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVLevel**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncMPVLevel**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvlevel) .
