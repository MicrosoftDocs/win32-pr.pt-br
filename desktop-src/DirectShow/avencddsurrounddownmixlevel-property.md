# <a name="avencddsurrounddownmixlevel-property"></a>Propriedade AVEncDDSurroundDownMixLevel

Especifica o nível downmix Surround, em dB (decib), em um fluxo de áudio Dolby Digital. Essa propriedade se aplica a codificadores de áudio Dolby Digital.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDSurroundDownMixLevel**
