# <a name="mapper-sample"></a>Exemplo de mapeador

## <a name="description"></a>Descrição

Este exemplo demonstra como usar o método [**IFilterMapper2:: EnumMatchingFilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) para localizar filtros no registro.

A interface do usuário do aplicativo de exemplo contém controles que correspondem aos parâmetros do método [**EnumMatchingFilters**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) , como o valor de mérito mínimo, as categorias de PIN e os tipos de mídia. Selecione as opções de pesquisa desejadas e clique no botão **Pesquisar** para enumerar os filtros que correspondem a essas opções.

## <a name="downloading-the-sample"></a>Baixando o exemplo

Para baixar os exemplos do SDK do DirectShow, instale a versão mais recente do [SDK do Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este exemplo é instalado no seguinte caminho: \[ exemplos de raiz do SDK \] \\ \\ multimídia multi Mapper do Multimedia \\ \\ \\ .
