# <a name="avencmpaprivateuserbit-property"></a>Propriedade AVEncMPAPrivateUserBit

Define o valor do bit de usuário privado no fluxo MPEG-1. Essa propriedade se aplica a codificadores de áudio MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPAPrivateUserBit**

## <a name="property-value"></a>Valor da propriedade

Essa propriedade pode ter o valor 0 ou 1.
