# <a name="coareftimeoperator-double-method"></a>Método Double COARefTime. Operator

Esse operador converte o tempo de referência em um valor **duplo** .

## <a name="syntax"></a>Sintaxe


```C++
double operator double();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna o tempo de referência em segundos.
