# <a name="avencddsurround3dbattenuation-property"></a>Propriedade AVEncDDSurround3dBAttenuation

Especifica se os canais surround em um fluxo de áudio Dolby Digital são atenuados em 3 decibéis (dB) antes da codificação. Essa propriedade se aplica aos codificadores Dolby Digital Audio.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDSurround3dBAttenuation**
