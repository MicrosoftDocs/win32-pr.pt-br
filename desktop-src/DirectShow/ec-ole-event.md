# <a name="ec_ole_event"></a>EVENTO \_ EC OLE \_

Um filtro está passando uma cadeia de caracteres de texto para o aplicativo.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*Lparam1*
</dt> <dd>

(**BSTR**) Cadeia de caracteres do **tipo BSTR.**

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

(**BSTR**) Cadeia de caracteres do **tipo BSTR.**

</dd> </dl>

## <a name="default-action"></a>Ação Padrão

Nenhum.

## <a name="remarks"></a>Comentários

Por convenção, o primeiro parâmetro contém informações de tipo (por exemplo, "Texto") e o segundo parâmetro contém a cadeia de caracteres de texto.
