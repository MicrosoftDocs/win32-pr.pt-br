# <a name="creating-asf-files-in-directshow-directshow"></a>Criando arquivos ASF no DirectShow (DirectShow)

Esta seção descreve como usar o filtro do [gravador ASF](wm-asf-writer-filter.md) do DirectShow WM para criar arquivos no formato ASF (Advanced Systems Format). O ASF é um formato de contêiner que pode conter qualquer tipo de dados compactados ou descompactados. Os arquivos de formato de mídia do Windows são arquivos ASF que usam determinados codecs, conforme especificado no SDK (Software Development Kit) do Windows Media Format. Esses arquivos usam as extensões ". WMA" para arquivos de áudio do Windows Media e ". wmv" para arquivos de vídeo do Windows Media.

Este tópico contém as seguintes seções:

-   [Configurando o gravador ASF](configuring-the-asf-writer.md)
-   [Criando gráficos de filtro para gravar arquivos ASF](building-filter-graphs-to-write-asf-files.md)
-   [Configurar perfis e outras propriedades de arquivo ASF](configuring-profiles-and-other-asf-file-properties.md)
