# <a name="creating-asf-files-in-directshow-directshow"></a>Criando arquivos ASF em DirectShow (DirectShow)

Esta seção descreve como usar o filtro DirectShow [Wm ASF Writer](wm-asf-writer-filter.md) para criar arquivos no ASF (Advanced Systems Format). ASF é um formato de contêiner que pode conter qualquer tipo de dados compactados ou descompactados. Windows Arquivos de Formato de Mídia são arquivos ASF que usam determinados codecs, conforme especificado no SDK (Kit de Desenvolvimento de Software) Windows Formato de Mídia. Esses arquivos usam as extensões ".wma" para Windows de Áudio de Mídia e ".wmv" para Windows de Vídeo de Mídia.
