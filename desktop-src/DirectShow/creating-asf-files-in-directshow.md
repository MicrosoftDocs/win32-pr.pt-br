# <a name="creating-asf-files-in-directshow-directshow"></a>Criando arquivos ASF no DirectShow (DirectShow)

Esta seção descreve como usar o filtro DirectShow [WM ASF Writer](wm-asf-writer-filter.md) para criar arquivos no ASF (Advanced Systems Format). ASF é um formato de contêiner que pode conter qualquer tipo de dados compactados ou descompactados. Windows Media Format arquivos são arquivos ASF que usam determinados codecs, conforme especificado no SDK (Software Development Kit) do Windows Media Format. Esses arquivos usam as extensões ".wma" para arquivos de Áudio de Mídia do Windows e ".wmv" para arquivos de Vídeo de Mídia do Windows.

-   [Configurar perfis e outras propriedades de arquivo ASF](configuring-profiles-and-other-asf-file-properties.md)
