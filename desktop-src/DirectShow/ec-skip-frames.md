# <a name="ec_skip_frames"></a>\_ignorar \_ quadros do EC

Enviado pelo navegador de DVD quando ele precisa ignorar quadros.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Número de quadros a serem ignorados.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Reservado.

</dd> </dl>

## <a name="default-action"></a>Ação Padrão

Esse evento não é enviado para o aplicativo. Os aplicativos não podem substituir a ação padrão desse evento.
