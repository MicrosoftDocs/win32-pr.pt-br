# <a name="dmo-functions"></a>Funções de DMO

Esta seção descreve as funções do Microsoft DirectX Media Object (DMO).



| Função                                             | Descrição                                                   |
|------------------------------------------------------|---------------------------------------------------------------|
| [**DMOEnum**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoenum)                           | Enumera DMOs listados no registro.                       |
| [**DMOGetName**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogetname)                     | Recupera o nome de um DMO do registro.                |
| [**DMOGetTypes**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogettypes)                   | Recupera os tipos de mídia registrados para um DMO.               |
| [**DMORegister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoregister)                   | Registra um DMO.                                              |
| [**DMOUnregister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmounregister)               | Cancela o registro de um DMO.                                            |
| [**MoCopyMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocopymediatype)           | Copia uma estrutura de tipo de mídia para outra.                   |
| [**MoCreateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocreatemediatype)       | Aloca uma nova estrutura de tipo de mídia.                         |
| [**MoDeleteMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-modeletemediatype)       | Exclui uma estrutura de tipo de mídia que foi alocada anteriormente. |
| [**MoDuplicateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moduplicatemediatype) | Duplica uma estrutura de tipo de mídia.                            |
| [**MoFreeMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mofreemediatype)           | Libera os membros alocados de uma estrutura de tipo de mídia.        |
| [**MoInitMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moinitmediatype)           | Inicializa uma estrutura de tipo de mídia.                           |
