# <a name="dmo-enumerated-types"></a>Tipos enumerados de DMO

Esta seção descreve os tipos enumerados do Microsoft DirectX Media Object (DMO).



| Tipo enumerado                                                            | Descrição                                                                  |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_sinalizadores de enumeração de DMO \_**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_enum_flags)                                 | Define sinalizadores que especificam critérios de pesquisa ao enumerar DMOs.            |
| [**\_sinalizadores do \_ buffer de dados de entrada do DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_data_buffer_flags)     | Define sinalizadores que descrevem um buffer de entrada.                                 |
| [**\_sinalizadores de \_ informações do fluxo de entrada DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_stream_info_flags)     | Define sinalizadores que descrevem um fluxo de entrada.                                 |
| [**\_sinalizadores do \_ buffer de dados de saída DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_data_buffer_flags)   | Define sinalizadores que descrevem um buffer de saída.                                |
| [**\_sinalizadores de \_ informações do fluxo de saída do DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_stream_info_flags)   | Define sinalizadores que descrevem um fluxo de saída.                                |
| [**\_sinalizadores de \_ saída do processo DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_process_output_flags)            | Define sinalizadores que especificam solicitações de processamento de saída.                       |
| [**\_sinalizadores de registro de DMO \_**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_register_flags)                         | Define sinalizadores que especificam informações de registro para um DMO.                   |
| [**\_sinalizadores de \_ tipo \_ set de DMO**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_set_type_flags)                        | Define sinalizadores para definir o tipo de mídia em um fluxo.                        |
| [**\_sinalizadores de \_ fluxo de saída de vídeo DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_video_output_stream_flags) | Define sinalizadores que descrevem os recursos solicitados para otimizações de vídeo.      |
| [**\_tipo de curva MP \_**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_curve_type)                                   | Define a curva que um parâmetro de mídia segue dentro de um segmento de envelope. |
| [**tipo de MP \_**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_type)                                                | Especifica o tipo de dados para um parâmetro.                                     |
