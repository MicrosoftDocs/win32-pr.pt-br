# <a name="external-device-control-interfaces"></a>Interfaces de controle de dispositivo externo

Essas interfaces dão suporte ao controle de aplicativo sobre dispositivos externos, como camcorders DV.



| Interface                                            | Descrição                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Controle uma câmera.                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Controle um dispositivo externo, como um videocassete ou camcorder.                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Controlar funções de transporte de VCR, como reproduzir, pausar, gravar, avançar e retroceder. |
| [**IAMTimecodeDisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Defina as propriedades em um dispositivo que exibe o código de logon SMPTE.                                |
| [**IAMTimecodeGenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Defina as propriedades em um dispositivo que gera um código de meio SMPTE.                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Leia o código de erro SMPTE de um dispositivo externo.                                            |
