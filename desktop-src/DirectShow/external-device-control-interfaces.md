# <a name="external-device-control-interfaces"></a>Interfaces de controle de dispositivo externas

Essas interfaces são compatíveis com o controle de aplicativos sobre dispositivos externos, como gravações DV.



| Interface                                            | Descrição                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**IAMCameraControl**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Controlar uma câmera.                                                                       |
| [**IAMExtDevice**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Controlar um dispositivo externo, como um VCR ou uma câmera.                                  |
| [**IAMExtTransport**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Controlar funções de transporte de VCR, como reproduzir, pausar, registrar, avançar e retroceder. |
| [**IAMTimecodeDisplay**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Definir propriedades em um dispositivo que exibe o código de data/hora SMPTE.                                |
| [**IAMTimecodeGenerator**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Definir propriedades em um dispositivo que gera o código de data/hora SMPTE.                               |
| [**IAMTimecodeReader**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Leia o código de tempo SMPTE de um dispositivo externo.                                            |
