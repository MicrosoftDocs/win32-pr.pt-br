# <a name="avencddchannelbwlowpassfilter-property"></a>Propriedade AVEncDDChannelBWLowPassFilter

Especifica se um filtro de passagem baixa é aplicado aos principais canais de entrada em um fluxo de áudio Dolby Digital. Essa propriedade se aplica aos codificadores Dolby Digital Audio.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDChannelBWLowPassFilter**
