# <a name="stream_time"></a>tempo de fluxo \_

> [!Note]  
> Essa API está preterida. Novos aplicativos não devem usá-lo.

 

o \_ tipo de dados TIME de fluxo é usado para expressar os tempos de referência no Microsoft DirectShow streaming de multimídia. As unidades são 100 nanossegundos.


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>Comentários

esse tipo de dados é equivalente ao tipo de dados de [**\_ tempo de referência**](reference-time.md) DirectShow.
