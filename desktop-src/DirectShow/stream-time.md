# <a name="stream_time"></a>tempo de fluxo \_

> [!Note]  
> Essa API está preterida. Novos aplicativos não devem usá-lo.

 

O \_ tipo de dados time de fluxo é usado para expressar os tempos de referência no streaming multimídia do Microsoft DirectShow. As unidades são 100 nanossegundos.


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>Comentários

Esse tipo de dados é equivalente ao tipo de dados de [**\_ tempo de referência**](reference-time.md) do DirectShow.
