# <a name="timeline-element"></a>Elemento Timeline

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `timeline` elemento define a linha do tempo. Esse elemento é o nó raiz no arquivo XML.

## <a name="attributes"></a>Atributos

[**defaultfx**](defaultfx-attribute.md), [**defaulttrans**](defaulttrans-attribute.md), [**enablefx**](enablefx-attribute.md), [**enabletrans**](enabletrans-attribute.md), [**taxa de quadros**](framerate-attribute.md)

## <a name="parentchild-information"></a>Informações de pai/filho



| Label | Valor |
|----------|-------------------------------------------------------|
| Pai   | Nenhum. Esse elemento deve ser o nó raiz no arquivo. |
| Children | [**Group**](group-element.md)                        |



 

## <a name="examples"></a>Exemplos


```XML
<timeline> </timeline>
```
