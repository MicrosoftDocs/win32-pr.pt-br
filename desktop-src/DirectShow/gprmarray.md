# <a name="gprmarray"></a>GPRMARRAY

O tipo de dados **GPRMARRAY** contém uma matriz de valores de GPRM (registro geral de parâmetro) em DVD.


```C++
typedef DVD_REGISTER GPRMARRAY [16];
```
