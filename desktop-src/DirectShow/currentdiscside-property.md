# <a name="currentdiscside-property"></a>Propriedade CurrentDiscSide

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `CurrentDiscSide` propriedade recupera o lado atual do disco.

``` syntax
[ iCurSide = ] MSWebDVD.CurrentDiscSide
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro que representa o lado atual; Side A = 1 e o lado B = 2.

## <a name="remarks"></a>Comentários

Esta propriedade é somente leitura sem valor padrão.
