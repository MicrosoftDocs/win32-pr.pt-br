# <a name="cmediasamplem_end-member"></a>CMediaSample:: m \_ end member

Hora de término da amostra. Esse valor será válido somente se a variável de membro [**CMediaSample:: m \_ dwFlags**](cmediasample-m-dwflags.md) contiver o sinalizador am de \_ exemplo \_ STOPVALID.

## <a name="syntax"></a>Syntax


```C++
REFERENCE_TIME m_End;
```
