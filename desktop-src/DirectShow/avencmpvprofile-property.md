# <a name="avencmpvprofile-property"></a>Propriedade AVEncMPVProfile

Especifica o perfil MPEG-2. Essa propriedade se aplica a codificadores de vídeo MPEG.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVProfile**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da [**enumeração eAVEncMPVProfile.**](/windows/win32/api/codecapi/ne-codecapi-eavencmpvprofile)
