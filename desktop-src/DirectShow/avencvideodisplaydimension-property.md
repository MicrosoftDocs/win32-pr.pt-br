# <a name="avencvideodisplaydimension-property"></a>Propriedade AVEncVideoDisplayDimension

Especifica o tamanho do fluxo de vídeo quando ele é decodificado.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoDisplayDimension**

## <a name="property-value"></a>Valor da propriedade

Os 16 bits superiores do valor contêm a largura, em pixels. Os 16 bits inferiores contêm a altura, em pixels.
