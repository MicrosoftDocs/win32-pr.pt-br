# <a name="cbasewindowgetmemoryhdc-method"></a>Método CBaseWindow. GetMemoryHDC

O `GetMemoryHDC` método recupera um identificador para o contexto de dispositivo de memória (DC).

## <a name="syntax"></a>Sintaxe


```C++
virtual HDC GetMemoryHDC();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna um identificador para o controlador de domínio de memória.
