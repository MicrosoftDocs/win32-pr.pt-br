# <a name="cbasewindowgetmemoryhdc-method"></a>Método CBaseWindow.GetMemoryHDC

O `GetMemoryHDC` método recupera um handle para o DC (contexto do dispositivo de memória).

## <a name="syntax"></a>Sintaxe


```C++
virtual HDC GetMemoryHDC();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Retorna um alça para o DC de memória.
