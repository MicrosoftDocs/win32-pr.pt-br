# <a name="tooltip-property"></a>Propriedade ToolTip

> [!Note]  
> esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `ToolTip` propriedade define o texto para a dica de ferramenta que aparece quando o ponteiro do mouse está sobre o retângulo de vídeo **MSWebDVD** .

``` syntax
[ sToolTip = ] MSWebDVD.ToolTip
```

## <a name="return-value"></a>Valor Retornado

Retorna o texto da dica de ferramenta como uma cadeia de caracteres.

## <a name="remarks"></a>Comentários

Esta propriedade é de leitura/gravação sem valor padrão. Se a cadeia de caracteres estiver vazia, nenhuma dica de ferramenta será exibida.
