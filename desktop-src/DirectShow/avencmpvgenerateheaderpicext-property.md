# <a name="avencmpvgenerateheaderpicext-property"></a>Propriedade AVEncMPVGenerateHeaderPicExt

Especifica se o codificador gera cabeçalhos de extensão de imagem. Essa propriedade se aplica a codificadores de vídeo MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVGenerateHeaderPicExt**

## <a name="remarks"></a>Comentários

Se o valor for **Variant \_ true**, o codificador gerará cabeçalhos de extensão de imagem.
