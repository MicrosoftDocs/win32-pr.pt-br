# <a name="avencmpvgenerateheaderpicext-property"></a>Propriedade AVEncMPVGenerateHeaderPicExt

Especifica se o codificador gera os headers de extensão de imagem. Essa propriedade se aplica a codificadores de vídeo MPEG.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVGenerateHeaderPicExt**

## <a name="remarks"></a>Comentários

Se o valor for **VARIANT \_ TRUE,** o codificador gerará os headers de extensão de imagem.
