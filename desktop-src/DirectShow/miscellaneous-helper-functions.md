# <a name="miscellaneous-helper-functions"></a>Funções auxiliares diversas

Veja a seguir funções auxiliares diversas.



| Função                                                   | Descrição                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**EqualPins**](equalpins.md)                             | Verifica se dois Pins estão no mesmo objeto.                                                                    |
| [**Int64x32Div32**](int64x32div32.md)                     | Implementa a fórmula `((a*b)+rnd)/c` em que `a` é um valor de 64 bits e `b` , `c` e `rnd` são valores de 32 bits. |
| [**Isequalobject**](isequalobject.md)                     | Verifica se duas interfaces estão no mesmo objeto.                                                              |
| [**llMulDiv**](llmuldiv.md)                               | Implementa a fórmula `((a*b)+rnd)/c` em que cada termo é um valor de 64 bits.                                     |
| [**WaitDispatchingMessages**](waitdispatchingmessages.md) | Aguarda a sinalização de um objeto, ao mesmo tempo que distribui mensagens de janela.                                        |
