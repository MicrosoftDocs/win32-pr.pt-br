# <a name="cbaserendererhavecurrentsample-method"></a>Método CBaseRenderer. HaveCurrentSample

O `HaveCurrentSample` método determina se o filtro tem um exemplo.

## <a name="syntax"></a>Sintaxe


```C++
virtual BOOL HaveCurrentSample();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Retornará **true** se o filtro tiver um exemplo, ou **false** caso contrário.
