# <a name="avencdddchighpassfilter-property"></a>Propriedade AVEncDDDCHighPassFilter

Especifica se um filtro de passagem alta com bloqueio de DC é aplicado a um fluxo de áudio Dolby Digital. Essa propriedade se aplica a codificadores de áudio Dolby Digital.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDDCHighPassFilter**
