# <a name="avencdddchighpassfilter-property"></a>Propriedade AVEncDDDCHighPassFilter

Especifica se um filtro de alta passagem de bloqueio de DC é aplicado a um fluxo de áudio Dolby Digital. Essa propriedade se aplica aos codificadores Dolby Digital Audio.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDDCHighPassFilter**
