# <a name="cbasecontrolvideoonupdaterectangles-method"></a>Método CBaseControlVideo. OnUpdateRectangles

O `OnUpdateRectangles` método é chamado quando o retângulo de origem ou de destino é alterado.

## <a name="syntax"></a>Sintaxe


```C++
virtual HRESULT OnUpdateRectangles();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna NOERROR.
