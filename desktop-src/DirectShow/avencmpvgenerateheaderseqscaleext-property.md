# <a name="avencmpvgenerateheaderseqscaleext-property"></a>Propriedade AVEncMPVGenerateHeaderSeqScaleExt

Especifica se o codificador gera cabeçalhos de extensão escalonáveis de sequência. Essa propriedade se aplica a codificadores de vídeo MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVGenerateHeaderSeqScaleExt**

## <a name="remarks"></a>Comentários

Se o valor for **Variant \_ true**, o codificador gerará cabeçalhos de extensão escalonáveis de sequência.
