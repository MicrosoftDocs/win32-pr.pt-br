# <a name="filter-mapper"></a>Mapeador de filtro

O mapeador de filtro pesquisa o registro em busca de filtros registrados. o filtro Graph Manager usa esse componente para criar gráficos de filtro. Os aplicativos podem usá-lo para localizar filtros que correspondem aos critérios de pesquisa especificados. Crie esse objeto chamando **CoCreateInstance**.




| Rótulo | Valor |
|--------|-------|
| Identificador de classe | CLSID_FilterMapper2 | 
| Interfaces | <ul><li><a href="/windows/desktop/api/Strmif/nn-strmif-ifiltermapper2"><strong>IFilterMapper2</strong></a></li><li><a href="iamfilterdata.md"><strong>IAMFilterData</strong></a> (preterido)</li><li><a href="/windows/desktop/api/Strmif/nn-strmif-ifiltermapper3"><strong>IFilterMapper3</strong></a> (preterido)</li></ul> |
