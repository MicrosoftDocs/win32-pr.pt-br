# <a name="filter-mapper"></a>Mapeador de filtro

O mapeador de filtro pesquisa o registro em busca de filtros registrados. O Gerenciador de gráfico de filtro usa esse componente para criar gráficos de filtro. Os aplicativos podem usá-lo para localizar filtros que correspondem aos critérios de pesquisa especificados. Crie esse objeto chamando **CoCreateInstance**.
