# <a name="cbaserendererisstreaming-method"></a>Método CBaseRenderer. isstreaming

O `IsStreaming` método consulta se o filtro está transmitindo dados.

## <a name="syntax"></a>Sintaxe


```C++
BOOL IsStreaming();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna o sinalizador [**\_ bStreaming CBaseRenderer:: m**](cbaserenderer-m-bstreaming.md) .
