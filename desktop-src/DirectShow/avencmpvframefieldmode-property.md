# <a name="avencmpvframefieldmode-property"></a>Propriedade AVEncMPVFrameFieldMode

Especifica se o codificador produz campos codificados ou quadros codificados. Essa propriedade se aplica a codificadores de vídeo MPEG.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVFrameFieldMode**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncMPVFrameFieldMode.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvframefieldmode)
