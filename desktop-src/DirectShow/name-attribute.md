# <a name="name-attribute"></a>Atributo de nome

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `name` atributo especifica o nome de um objeto.

## <a name="possible-values"></a>Valores possíveis

Pode ser qualquer valor.

## <a name="applies-to"></a>Aplica-se A

[**grupo**](group-element.md), [ **param**](param-element.md)

## <a name="remarks"></a>Comentários

Para o elemento **Group** , esse atributo especifica um nome definido pelo aplicativo para o grupo.

Para o elemento **param** , esse atributo especifica o nome da propriedade.
