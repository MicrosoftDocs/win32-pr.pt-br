# <a name="capture-method"></a>Método Capture

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O método captura uma imagem ainda do quadro de vídeo quando o objeto `Capture` MSWebDVD está no modo sem janelas.

``` syntax
MSWebDVD.Capture()
```

## <a name="remarks"></a>Comentários

Esse método captura o quadro atual da imagem DVD-Video e o colará em uma janela da qual o usuário pode salvar ou editar a imagem. O objeto MSWebDVD deve estar no modo sem janelas para que esse método seja bem-sucedido.
