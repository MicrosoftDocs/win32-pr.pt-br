# <a name="capture-method"></a>Método Capture

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `Capture` método captura uma imagem ainda do quadro de vídeo quando o objeto MSWebDVD está no modo sem janela.

``` syntax
MSWebDVD.Capture()
```

## <a name="remarks"></a>Comentários

Esse método captura o quadro atual da imagem de DVD-Video e cola-o em uma janela da qual o usuário pode salvar ou editar a imagem. O objeto MSWebDVD deve estar no modo sem janela para que esse método tenha sucesso.
