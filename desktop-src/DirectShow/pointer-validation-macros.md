# <a name="pointer-validation-macros"></a>Macros de validação de ponteiro

O Microsoft DirectShow fornece várias macros para validar ponteiros.



| Macro                                                | Descrição                                                                   |
|------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Ponto de verificação**](checkpointer.md)                 | Verifica se um ponteiro é **nulo**.                                         |
| [**ValidateReadPtr**](validatereadptr.md)           | Verifica se o processo de chamada tem acesso de leitura a um bloco de memória.          |
| [**ValidateReadWritePtr**](validatereadwriteptr.md) | Verifica se o processo de chamada tem acesso de leitura/gravação a um bloco de memória.    |
| [**ValidateStringPtr**](validatestringptr.md)       | Verifica se o processo de chamada tem acesso de leitura a uma cadeia de caracteres.                |
| [**ValidateStringPtrA**](validatestringptra.md)     | Verifica se o processo de chamada tem acesso de leitura a uma cadeia de caracteres ANSI.          |
| [**ValidateStringPtrW**](validatestringptrw.md)     | Verifica se o processo de chamada tem acesso de leitura a uma cadeia de caracteres largos. |
| [**ValidateWritePtr**](validatewriteptr.md)         | Verifica se o processo de chamada tem acesso de gravação a um bloco de memória.         |
