> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

Sem suporte.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT GetGroupIBelongTo(
   IAMTimelineGroup **ppGroup
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*ppGroup* 
</dt> <dd>

Reservado.

</dd> </dl>
