# <a name="using-the-dmotest-application"></a>Usando o aplicativo DMOTest

> [!Note]  
> O aplicativo DMOTest não está mais disponível no SDK.

 

o utilitário DMOTest pode ajudar a verificar se um DMO atende às especificações de DMO. se você desenvolver uma DMO, deverá incluir o aplicativo de teste de DMO como parte do teste.

O DMOTest é instalado no \\ diretório bin DXUtils, no diretório raiz do SDK. Para usar o DMOTest, primeiro você deve gerar alguns dados de teste para o DMO. o SDK inclui um filtro DirectShow para essa finalidade. Detalhes sobre o filtro e o aplicativo DMOTest podem ser encontrados na ajuda do aplicativo DMOTest. No aplicativo, escolha **conteúdo** no menu **ajuda** . A ajuda inclui instruções passo a passo para gerar dados de teste e executar testes.
