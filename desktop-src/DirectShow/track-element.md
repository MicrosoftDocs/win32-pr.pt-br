# <a name="track-element"></a>Elemento Track

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `track` elemento define um objeto Track.

## <a name="attributes"></a>Atributos

[**Lock**](lock-attribute.md), [**mudo**](mute-attribute.md), [**UserData**](userdata-attribute.md), [**userid**](userid-attribute.md), [**username**](username-attribute.md)

## <a name="parentchild-information"></a>Informações de pai/filho



| Rótulo | Valor |
|----------|----------------------------------------------------------------------------------------------------------|
| Pai   | [**composto**](composite-element.md), [ **grupo**](group-element.md)                                   |
| Children | [**clip**](clip-element.md), [**efeito**](effect-element.md), [**transição**](transition-element.md) |
