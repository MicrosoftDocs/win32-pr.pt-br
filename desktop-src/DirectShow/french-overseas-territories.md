# <a name="french-overseas-territories"></a>Territórios franceses do exterior

Para obter uma explicação das tabelas a seguir, consulte [tabelas e atribuições](tables-and-assignments.md).

Os índices de frequência na tabela de transmissão de territórios franceses do exterior são adequados para uso como números de canal; os índices de frequência de cabo não são adequados para uso como números de canal.

### <a name="french-overseas-territories-broadcast-assignments"></a>Atribuições de difusão de territórios franceses do exterior


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```



### <a name="french-overseas-territories-cable-assignments"></a>Atribuições de cabos para territórios franceses do exterior


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```
