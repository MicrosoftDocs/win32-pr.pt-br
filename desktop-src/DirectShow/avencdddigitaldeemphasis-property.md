# <a name="avencdddigitaldeemphasis-property"></a>Propriedade AVEncDDDigitalDeemphasis

Especifica se a des ênfase digital está habilitada em um fluxo de áudio Dolby Digital. Essa propriedade se aplica a codificadores de áudio Dolby Digital.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID da propriedade
