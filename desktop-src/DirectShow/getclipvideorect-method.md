# <a name="getclipvideorect-method"></a>Método GetClipVideoRect

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `GetClipVideoRect` método recupera o retângulo de recorte definido atualmente para a exibição de vídeo.

``` syntax
[ oClipRect = ] MSWebDVD.GetClipVideoRect()
```

## <a name="return-value"></a>Valor Retornado

Retorna um objeto [DVDRect](dvdrect-object.md) que define o retângulo.
