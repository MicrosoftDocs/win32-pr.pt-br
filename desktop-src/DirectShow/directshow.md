# <a name="directshow"></a>DirectShow

A API (interface de programação de aplicativo) do Microsoft DirectShow é uma arquitetura de streaming de mídia para o Microsoft Windows. Usando o DirectShow, seus aplicativos podem executar reprodução ou captura de vídeo e áudio de alta qualidade.

Os cabeçalhos, as bibliotecas, as ferramentas do SDK e os exemplos do DirectShow estão disponíveis na [SDK do Windows](https://msdn.microsoft.com/windows/aa904949.aspx).

> [!Note]  
> As versões anteriores do SDK do DirectShow foram incluídas no SDK do DirectX. A última versão do SDK do DirectX para incluir o DirectShow foi a atualização do SDK do DirectX 9,0-(fevereiro de 2005) extras. Após essa versão, o DirectShow foi movido para a SDK do Windows. Para obter a versão mais recente dos cabeçalhos, das bibliotecas e dos exemplos do DirectShow, baixe o SDK do Windows.
