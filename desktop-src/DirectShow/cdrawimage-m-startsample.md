# <a name="cdrawimagem_startsample-member"></a>Membro de CDrawImage:: m \_ StartSample

A variável de membro **m \_ StartSample** especifica a hora de início do exemplo mais recente.

## <a name="syntax"></a>Sintaxe


```C++
CRefTime m_StartSample;
```



## <a name="remarks"></a>Comentários

O valor só é válido dentro do método [**CDrawImage::D isplaysampletimes**](cdrawimage-displaysampletimes.md) .
