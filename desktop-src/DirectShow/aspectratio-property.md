# <a name="aspectratio-property"></a>Propriedade AspectRatio

> [!Note]  
> esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `AspectRatio` propriedade recupera a taxa de proporção do fluxo de vídeo atual como criado no disco.

``` syntax
[  nAspectRatio = ] MSWebDVD.AspectRatio 
```

## <a name="return-value"></a>Valor Retornado

Retorna um número que representa a taxa de proporção do fluxo de vídeo atual, como criado no disco (1,33 para uma taxa de proporção de 4:3 ou 1,78 para uma taxa de proporção de 16:9).

## <a name="remarks"></a>Comentários

Esta propriedade é somente leitura sem valor padrão.
