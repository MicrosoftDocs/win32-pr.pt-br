# <a name="avencaudiomapdestchanneln-property"></a>Propriedade AVEncAudioMapDestChannelN

Especifica qual canal de áudio é mapeado para o canal *N* no fluxo de áudio codificado.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**Consulte comentários.**

## <a name="remarks"></a>Comentários

Este tópico de referência descreve um conjunto de propriedades relacionadas. Cada propriedade mapeia um canal de entrada para um canal de saída especificado. As propriedades a seguir são definidas.



| GUID da propriedade                            | Descrição             |
|------------------------------------------|-------------------------|
| **CODECAPI \_ AVEncAudioMapDestChannel0**  | Canal de destino 0.  |
| **CODECAPI \_ AVEncAudioMapDestChannel1**  | Canal de destino 1.  |
| **CODECAPI \_ AVEncAudioMapDestChannel2**  | Canal 2 de destino.  |
| **CODECAPI \_ AVEncAudioMapDestChannel3**  | Canal de destino 3.  |
| **CODECAPI \_ AVEncAudioMapDestChannel4**  | Canal de destino 4.  |
| **CODECAPI \_ AVEncAudioMapDestChannel5**  | Canal de destino 5.  |
| **CODECAPI \_ AVEncAudioMapDestChannel6**  | Canal de destino 6.  |
| **CODECAPI \_ AVEncAudioMapDestChannel7**  | Canal de destino 7.  |
| **CODECAPI \_ AVEncAudioMapDestChannel8**  | Canal de destino 8.  |
| **CODECAPI \_ AVEncAudioMapDestChannel9**  | Canal de destino 9.  |
| **CODECAPI \_ AVEncAudioMapDestChannel10** | Canal de destino 10. |
| **CODECAPI \_ AVEncAudioMapDestChannel11** | Canal de destino 11. |
| **CODECAPI \_ AVEncAudioMapDestChannel12** | Canal de destino 12. |
| **CODECAPI \_ AVEncAudioMapDestChannel13** | Canal de destino 13. |
| **CODECAPI \_ AVEncAudioMapDestChannel14** | Canal de destino 14. |
| **CODECAPI \_ AVEncAudioMapDestChannel15** | Canal de destino 15. |
