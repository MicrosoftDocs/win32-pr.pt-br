# <a name="system-device-enumerator"></a>Enumerador de dispositivo do sistema

O enumerador de dispositivo do sistema enumera filtros e dispositivos de hardware instalados no sistema. Os aplicativos podem usar esse componente para localizar filtros e dispositivos em uma determinada categoria. Crie esse objeto chamando **CoCreateInstance**.



| Rótulo | Valor |
|------------------|------------------------------------------|
| Identificador de classe | \_SYSTEMDEVICEENUM CLSID                  |
| Interfaces       | [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) |
