# <a name="cbasepropertypagem_bdirty-member"></a>Membro de CBasePropertyPage:: m \_ bDirty

O sinalizador **m \_ bDirty** indica se alguma das propriedades foi alterada. Defina esse sinalizador como **true** se o usuário alterar qualquer propriedade por meio dos controles da caixa de diálogo.

## <a name="syntax"></a>Syntax


```C++
BOOL m_bDirty;
```
