# <a name="changecurrentaudiostream"></a>ChangeCurrentAudioStream

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `ChangeCurrentAudioStream` evento é enviado quando o disco habilita ou desabilita a alteração do fluxo de áudio.

``` syntax
ChangeCurrentAudioStream(bEnabled)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="bEnabled"></span><span id="benabled"></span><span id="BENABLED"></span>*bEnabled*
</dt> <dd>

Especifica se a operação está habilitada ou desabilitada como booliana.

</dd> </dl>
