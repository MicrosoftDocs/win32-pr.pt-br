# <a name="cbasepropertypagem_dialogid-member"></a>Membro da caixa de diálogo CBasePropertyPage:: m \_

A `m_DialogId` variável de membro especifica o identificador de recurso para a caixa de diálogo. Essa variável é inicializada no método do construtor.

## <a name="syntax"></a>Syntax


```C++
int m_DialogId;
```
