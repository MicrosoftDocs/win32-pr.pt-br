# <a name="avdecvideopixelaspectratio-property"></a>Propriedade AVDecVideoPixelAspectRatio

Especifica a taxa de proporção de pixel do fluxo de vídeo decodificado.

Esta propriedade é somente para leitura.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVDecVideoPixelAspectRatio**

## <a name="remarks"></a>Comentários

Os 16 bits superiores do valor contêm a largura e os 16 bits inferiores contêm a altura.
