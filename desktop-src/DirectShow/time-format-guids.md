# <a name="time-format-guids"></a>GUIDs de formato de hora

Os seguintes identificadores globalmente exclusivos (GUIDs) definem formatos de hora diferentes.



| GUID                                                                                                                                                                                       | Descrição                                       |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------|
| <span id="TIME_FORMAT_NONE"></span><span id="time_format_none"></span><dl> <dt>**formato de hora \_ \_ nenhum**</dt> </dl>                    | Nenhum formato.<br/>                             |
| <span id="TIME_FORMAT_FRAME"></span><span id="time_format_frame"></span><dl> <dt>**\_quadro de formato de hora \_**</dt> </dl>                 | Quadros de vídeo.<br/>                          |
| <span id="TIME_FORMAT_SAMPLE"></span><span id="time_format_sample"></span><dl> <dt>**\_exemplo de formato de hora \_**</dt> </dl>              | Exemplos no fluxo.<br/>                 |
| <span id="TIME_FORMAT_FIELD"></span><span id="time_format_field"></span><dl> <dt>**\_campo de formato de hora \_**</dt> </dl>                 | Campos de vídeo entrelaçados.<br/>               |
| <span id="TIME_FORMAT_BYTE"></span><span id="time_format_byte"></span><dl> <dt>**\_byte de formato de hora \_**</dt> </dl>                    | Deslocamento de byte dentro do fluxo.<br/>         |
| <span id="TIME_FORMAT_MEDIA_TIME"></span><span id="time_format_media_time"></span><dl> <dt>**\_tempo de \_ mídia do formato de hora \_**</dt> </dl> | Tempo de referência (unidades de 100 a nanossegundos).<br/> |
