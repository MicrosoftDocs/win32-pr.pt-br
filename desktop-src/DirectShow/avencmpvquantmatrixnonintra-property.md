# <a name="avencmpvquantmatrixnonintra-property"></a>Propriedade AVEncMPVQuantMatrixNonIntra

Especifica a matriz de quantização Luma para não-intra macroblocos. Essa propriedade se aplica a codificadores de vídeo MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**BSTR** (**VT \_ BSTR**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVQuantMatrixNonIntra**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é uma cadeia de caracteres que contém os coeficientes de 64 8 bits codificados como uma cadeia de caracteres de valores hexadecimais (128 caracteres).
