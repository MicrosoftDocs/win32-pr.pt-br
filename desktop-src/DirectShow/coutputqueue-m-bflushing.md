# <a name="coutputqueuem_bflushing-member"></a>Membro de COutputQueue:: m \_ bFlushing

Sinalizador que especifica se o objeto está executando uma operação de liberação. Se **for true**, o objeto iniciará a liberação.

## <a name="syntax"></a>Syntax


```C++
BOOL m_bFlushing;
```
