# <a name="ctransinplacefiltertypesmatch-method"></a>Método CTransInPlaceFilter. TypesMatch

O `TypesMatch` método determina se o tipo de mídia de entrada corresponde ao tipo de mídia de saída.

## <a name="syntax"></a>Sintaxe


```C++
BOOL TypesMatch();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retornará **true** se os tipos de mídia de entrada e saída forem iguais ou **false** caso contrário.
