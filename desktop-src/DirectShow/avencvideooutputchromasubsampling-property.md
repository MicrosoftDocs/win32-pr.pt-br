# <a name="avencvideooutputchromasubsampling-property"></a>Propriedade AVEncVideoOutputChromaSubsampling

Especifica o croma localizando para o vídeo codificado. Croma localizando define as posições dos exemplos de croma em relação aos exemplos de Luma.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoOutputChromaSubsampling**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um bit a bit ou dos sinalizadores da enumeração [**eAVEncVideoChromaSubsampling**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideochromasubsampling) .
