# <a name="y-property"></a>Propriedade y

> [!Note]  
> esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `y` propriedade define ou recupera a posição y do canto superior esquerdo do retângulo.

``` syntax
[ iYPos = ] DVDRect.y
```

## <a name="return-value"></a>Valor Retornado

Retorna a posição Y do canto superior esquerdo do retângulo como um inteiro.

## <a name="remarks"></a>Comentários

Esta propriedade é de leitura/gravação sem valor padrão.
