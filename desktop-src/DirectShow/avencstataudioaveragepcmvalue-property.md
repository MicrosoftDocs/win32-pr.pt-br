# <a name="avencstataudioaveragepcmvalue-property"></a>Propriedade AVEncStatAudioAveragePCMValue

Retorna o nível médio de volume do conteúdo de áudio.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncStatAudioAveragePCMValue**

## <a name="remarks"></a>Comentários

Essa propriedade estará disponível depois que a codificação for concluída.
