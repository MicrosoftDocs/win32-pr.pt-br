# <a name="avencvideooutputcolortransfermatrix-property"></a>Propriedade AVEncVideoOutputColorTransferMatrix

Especifica a matriz de conversão do espaço de cores Y'Cb'Cr' para o espaço de cores R'G'B' para o vídeo codificado.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoOutputColorTransferMatrix**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncVideoColorTransferMatrix.**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolortransfermatrix)
