# <a name="currenttitle-property"></a>Propriedade CurrentTitle

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `CurrentTitle` propriedade recupera o número do título em reprodução no momento.

``` syntax
[ iCurTitle = ] MSWebDVD.CurrentTitle
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro de 1 a 99 que representa o título do disco atual.

## <a name="remarks"></a>Comentários

Essa propriedade é somente leitura sem valor padrão.
