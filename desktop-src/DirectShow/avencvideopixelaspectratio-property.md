# <a name="avencvideopixelaspectratio-property"></a>Propriedade AVEncVideoPixelAspectRatio

Especifica a taxa de proporção de pixel.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoPixelAspectRatio**

## <a name="property-value"></a>Valor da propriedade

Os 16 bits superiores do valor contêm a largura e os 16 bits inferiores contêm a altura.
