> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

Não há suporte.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT SetSubObject(
   IUnknown *newVal
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*newVal* 
</dt> <dd>

</dd> </dl>
