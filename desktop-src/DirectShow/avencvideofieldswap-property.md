# <a name="avencvideofieldswap-property"></a>Propriedade AVEncVideoFieldSwap

Inverte a ordem dos campos entrelaçados no vídeo de origem.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoFieldSwap**

## <a name="remarks"></a>Comentários

Se o valor for **Variant \_ true**, o codificador trocará a ordem do campo.
