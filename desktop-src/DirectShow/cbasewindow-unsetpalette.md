# <a name="cbasewindowunsetpalette-method"></a>Método CBaseWindow. UnsetPalette

O `UnsetPalette` método exclui a paleta atual da janela e restaura a paleta padrão do sistema.

## <a name="syntax"></a>Sintaxe


```C++
void UnsetPalette();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
