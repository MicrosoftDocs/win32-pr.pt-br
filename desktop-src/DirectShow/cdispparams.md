# <a name="cdispparams-class"></a>Classe CDispParams

![hierarquia de classe cdispparams](images/cutil11.png)

A `CDispParams` classe implementa a estrutura **DISPPARAMS** usada na automação como uma classe base C++. O método **IDispatch:: Invoke** usa a estrutura OLE **DISPPARAMS** para conter os argumentos passados para qualquer método ou propriedade.

|------------------------------------------------|--------------------------------------|
