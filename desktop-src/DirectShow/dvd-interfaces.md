# <a name="dvd-interfaces"></a>DVD Interfaces

Essas interfaces são suportadas pela reprodução e navegação de DVD.



| Interface                                    | Descrição                                                                                            |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**IAMDecoderCaps**](/windows/desktop/api/Strmif/nn-strmif-iamdecodercaps)     | Implementado por decodificadores para indicar vários recursos.                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Bloqueie [o Filtro de Navegador de DVD](dvd-navigator-filter.md) até que um comando seja iniciado ou concluído.        |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Navegue e DVD-Video títulos, incluindo títulos de campeonato.                                          |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Crie um grafo de filtro para DVD-Video reprodução do software e do hardware disponíveis no sistema.    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Consulte vários atributos de um disco de DVD ou o estado do Navegador de DVD.                               |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Salve o estado da sessão do usuário no disco, incluindo o local de reprodução, o nível dos pais e a região. |
