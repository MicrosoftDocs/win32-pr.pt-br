# <a name="dvd-interfaces"></a>Interfaces de DVD

Essas interfaces dão suporte à reprodução e à navegação em DVD.



| Interface                                    | Descrição                                                                                            |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**IAMDecoderCaps**](/windows/desktop/api/Strmif/nn-strmif-iamdecodercaps)     | Implementado por decodificadores para indicar vários recursos.                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Bloquear o [filtro de navegador de DVD](dvd-navigator-filter.md) até que um comando seja iniciado ou concluído.        |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Navegue e jogue DVD-Video títulos, incluindo títulos de karaokê.                                          |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Crie um gráfico de filtro para reprodução de DVD-Video do software e hardware disponíveis no sistema.    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Consultar vários atributos de um disco de DVD ou o estado do navegador de DVD.                               |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Salve o estado da sessão do usuário em disco, incluindo o local de reprodução, o nível do pai e a região. |
