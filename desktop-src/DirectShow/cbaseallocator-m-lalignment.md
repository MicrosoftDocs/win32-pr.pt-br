# <a name="cbaseallocatorm_lalignment-member"></a>Membro de CBaseAllocator:: m \_ lAlignment

Alinhamento de cada buffer. O endereço de cada buffer deve ser um múltiplo par desse valor. O prefixo deve ser calculado para o alinhamento; consulte [**CBaseAllocator:: m \_ lPrefix**](cbaseallocator-m-lprefix.md).

## <a name="syntax"></a>Syntax


```C++
long m_lAlignment;
```
