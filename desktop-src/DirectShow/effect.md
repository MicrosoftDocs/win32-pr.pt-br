# <a name="effect"></a>Efeito

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O objeto Effect produz um efeito de vídeo ou áudio. Para criar esse objeto, chame o método [**IAMTimeline:: CreateEmptyNode**](iamtimeline-createemptynode.md) .

O objeto Effect expõe as seguintes interfaces:

-   [**IAMTimelineEffect**](iamtimelineeffect.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineSplittable**](iamtimelinesplittable.md)
-   **IPropertyPage**
