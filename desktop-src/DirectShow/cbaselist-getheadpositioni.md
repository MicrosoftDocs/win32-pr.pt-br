# <a name="cbaselistgetheadpositioni-method"></a>Método CBaseList. GetHeadPositionI

O `GetHeadPositionI` método recupera a posição do primeiro item na lista.

## <a name="syntax"></a>Sintaxe


```C++
POSITION GetHeadPositionI();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Retorna um valor de posição ou **NULL** se a lista estiver vazia.
