# <a name="avencvideosourcefilmcontent-property"></a>Propriedade AVEncVideoSourceFilmContent

Especifica se a origem original do vídeo de entrada foi filme ou vídeo.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoSourceFilmContent**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncVideoFilmContent**](/windows/win32/api/codecapi/ne-codecapi-eavencvideofilmcontent) .
