# <a name="height-property-directshow"></a>Propriedade Height (DirectShow)

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `Height` propriedade define ou recupera a altura do retângulo.

``` syntax
[ iHeight = ] DVDRect.Height
```

## <a name="return-value"></a>Valor Retornado

Retorna a altura, em pixels, do retângulo como um Inteiro.

## <a name="remarks"></a>Comentários

Essa propriedade é de leitura/gravação sem valor padrão.
