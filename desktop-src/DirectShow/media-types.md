# <a name="media-types-directshow"></a>Tipos de mídia (DirectShow)

Microsoft® DirectShow® usa a estrutura [**AM \_ MEDIA \_ TYPE**](/windows/win32/api/strmif/ns-strmif-am_media_type) para descrever exemplos de mídia. Essa estrutura inclui membros **guid** para o tipo principal, subtipo e tipo de formato. Esta seção resume os GUIDs de tipo principal e subtipo definidos para DirectShow. Filtros de terceiros podem definir seus próprios GUIDs de tipo de mídia.

-   [**Tipos principais**](major-types.md)
-   [**Subtipos de áudio**](audio-subtypes.md)
-   [**Tipos de mídia BDA**](bda-media-types.md)
-   [**Tipos de mídia de DVD**](dvd-media-types.md)
-   [**Tipos de mídia da linha 21**](line-21-media-types.md)
-   [**Tipos de Mídia MPEG-1**](mpeg-1-media-types.md)
-   [Tipos de mídia MPEG-2](mpeg-2-media-types.md)
-   [**Subtipos de fluxo**](stream-subtypes.md)
-   [Tipos de mídia da VBI](vbi-media-types.md)
-   [Subtipos de vídeo](video-subtypes.md)
