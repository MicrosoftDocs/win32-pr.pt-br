# <a name="media-types-directshow"></a>Tipos de mídia (DirectShow)

O Microsoft® DirectShow® usa a estrutura [**am \_ Media \_ Type**](/windows/win32/api/strmif/ns-strmif-am_media_type) para descrever amostras de mídia. Essa estrutura inclui membros de **GUID** para tipo principal, subtipo e tipo de formato. Esta seção resume o tipo principal e os GUIDs de subtipo que são definidos para o DirectShow. Filtros de terceiros podem definir seus próprios GUIDs de tipo de mídia.

-   [**Tipos principais**](major-types.md)
-   [**Subtipos de áudio**](audio-subtypes.md)
-   [**Tipos de mídia BDA**](bda-media-types.md)
-   [**Tipos de mídia de DVD**](dvd-media-types.md)
-   [**Linha 21 tipos de mídia**](line-21-media-types.md)
-   [**Tipos de mídia MPEG-1**](mpeg-1-media-types.md)
-   [Tipos de mídia MPEG-2](mpeg-2-media-types.md)
-   [**Subtipos de fluxo**](stream-subtypes.md)
-   [Tipos de mídia VBI](vbi-media-types.md)
-   [Subtipos de vídeo](video-subtypes.md)
