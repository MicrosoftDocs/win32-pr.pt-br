# <a name="multimedia-streaming"></a>Streaming de multimídia

> [!Note]  
> Essas APIs são preteridas. Os aplicativos devem usar o filtro de [**apoio de exemplo**](sample-grabber-filter.md) ou implementar um filtro personalizado para obter dados de um grafo de filtro do DirectShow.

 

O streaming de multimídia é um conjunto de APIs que permite que um aplicativo obtenha dados de áudio e vídeo de um grafo de filtro do DirectShow sem gravar um filtro do DirectShow personalizado.
