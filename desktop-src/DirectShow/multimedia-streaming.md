# <a name="multimedia-streaming"></a>Streaming multimídia

> [!Note]  
> Essas APIs foram preterida. Os aplicativos devem usar [**o filtro Grabber de**](sample-grabber-filter.md) exemplo ou implementar um filtro personalizado para obter dados de um DirectShow de filtro.

 

O Streaming multimídia é um conjunto de APIs que permitem que um aplicativo receba dados de áudio e vídeo de um grafo de filtro DirectShow sem escrever um filtro DirectShow personalizado.
