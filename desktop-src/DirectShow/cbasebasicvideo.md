# <a name="cbasebasicvideo-class"></a>Classe CBaseBasicVideo

![hierarquia de classe CBaseBasicVideo](images/wctrl02.png)

A `CBaseBasicVideo` classe implementa os métodos **IDispatch** da interface [**IBasicVideo**](/windows/desktop/api/Control/nn-control-ibasicvideo) . Os métodos **IBasicVideo** restantes são deixados como métodos virtuais puros e devem ser implementados por uma classe derivada.

Os métodos **IDispatch** nessa classe são implementações padrão e não são descritos detalhadamente aqui.
