# <a name="cmediatypecmediatype-constructor-mtypeh---no-parameters"></a>Construtor CMediaType. CMediaType (mtype. h)-sem parâmetros

Método de construtor.

## <a name="syntax"></a>Sintaxe


```C++
CMediaType();
```



## <a name="parameters"></a>Parâmetros

Este construtor não tem parâmetros.

## <a name="remarks"></a>Comentários

O construtor chama o método [**CMediaType:: InitMediaType**](cmediatype-initmediatype.md) para inicializar o tipo de mídia.
