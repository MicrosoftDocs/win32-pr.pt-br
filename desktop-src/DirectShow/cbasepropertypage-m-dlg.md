# <a name="cbasepropertypagem_dlg-member"></a>Membro CBasePropertyPage:: m \_ Dlg

A `m_Dlg` variável de membro contém um identificador para a janela da caixa de diálogo. Essa variável de membro é inicializada quando a janela de diálogo processa a mensagem do WM \_ INITDIALOG.

## <a name="syntax"></a>Syntax


```C++
HWND m_Dlg;
```
