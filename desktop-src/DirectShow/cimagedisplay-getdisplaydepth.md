# <a name="cimagedisplaygetdisplaydepth-method"></a>Método CImageDisplay. GetDisplayDepth

O `GetDisplayDepth` método recupera a profundidade de bits do modo de exibição atual.

## <a name="syntax"></a>Sintaxe


```C++
WORD GetDisplayDepth();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna o número de bits por pixel.
