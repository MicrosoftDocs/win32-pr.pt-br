# <a name="directshow-error-and-success-codes"></a>DirectShow Códigos de erro e êxito

Este tópico lista os códigos de retorno **HRESULT** que são definidos para DirectShow. observe que DirectShow APIs podem retornar outros valores **HRESULT** não mostrados aqui. Para obter valores de **HRESULT** adicionais, consulte [códigos de erro com](/windows/desktop/com/com-error-codes).

a tabela a seguir contém erros específicos de DirectShow e códigos de êxito em ordem numérica. Códigos de retorno começando com VFW \_ S \_ indicam êxito. Uma operação pode ter êxito parcial e ainda ter um valor de retorno bem-sucedido. Por exemplo, **VFW \_ S \_ alguns \_ dados \_ ignorados** indica que um arquivo foi renderizado ignorando algumas de suas configurações de propriedade. Códigos de retorno que começam com VFW \_ E \_ indicam um erro.

Esses valores são definidos no arquivo de cabeçalho Vfwmsgs. h.
