# <a name="avencstataudioaveragebps-property"></a>Propriedade AVEncStatAudioAverageBPS

Retorna a média de bits por segundo do áudio codificado.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncStatAudioAverageBPS**

## <a name="remarks"></a>Comentários

Essa propriedade estará disponível após a conclusão da codificação.

Esta propriedade se aplica somente à codificação de taxa de bits variável (VBR) com base em qualidade
