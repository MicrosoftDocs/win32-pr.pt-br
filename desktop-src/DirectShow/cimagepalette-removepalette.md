# <a name="cimagepaletteremovepalette-method"></a>Método CImagePalette. RemovePalette

O `RemovePalette` método exclui a paleta lógica existente e chama [**CBaseWindow:: UnsetPalette**](cbasewindow-unsetpalette.md) no objeto **CBaseWindow** .

## <a name="syntax"></a>Sintaxe


```C++
HRESULT RemovePalette();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna S \_ OK.
