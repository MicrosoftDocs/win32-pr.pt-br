# <a name="capture-graph-builder"></a>Capture Graph Builder

O Capture Graph Builder é um objeto auxiliar para a criação de grafos de captura de áudio e vídeo. Os aplicativos de captura e edição podem usar esse componente para construir grafos de filtro. Crie esse objeto chamando **CoCreateInstance**.



| Rótulo | Valor |
|------------------|--------------------------------------------------------|
| Identificador de Classe | CLSID \_ CaptureGraphBuilder2                            |
| Interfaces       | [**Icapturegraphbuilder2**](/windows/desktop/api/Strmif/nn-strmif-icapturegraphbuilder2) |
