# <a name="capture-graph-builder"></a>Construtor do grafo de captura

O construtor do grafo de captura é um objeto auxiliar para a criação de gráficos de captura de áudio e vídeo. Os aplicativos de captura e de edição podem usar esse componente para construir gráficos de filtro. Crie esse objeto chamando **CoCreateInstance**.



| Label | Valor |
|------------------|--------------------------------------------------------|
| Identificador de classe | \_CAPTUREGRAPHBUILDER2 CLSID                            |
| Interfaces       | [**ICaptureGraphBuilder2**](/windows/desktop/api/Strmif/nn-strmif-icapturegraphbuilder2) |
