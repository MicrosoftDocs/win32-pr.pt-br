# <a name="ctransformfilterm_bsampleskipped-member"></a>Membro de CTransformFilter:: m \_ bSampleSkipped

Sinalizador que indica se a amostra mais recente foi descartada. Se o método [**Receive**](ctransformfilter-receive.md) descartar um exemplo, ele definirá o valor como **true**.

## <a name="syntax"></a>Syntax


```C++
BOOL m_bSampleSkipped;
```
