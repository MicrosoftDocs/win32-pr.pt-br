# <a name="avencmpvuseconcealmentmotionvectors-property"></a>Propriedade AVEncMPVUseConcealmentMotionVectors

Especifica se o codificador usa vetores de movimento de ocultação. Essa propriedade se aplica a codificadores de vídeo MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVUseConcealmentMotionVectors**

## <a name="remarks"></a>Comentários

Se o valor for **Variant \_ true**, o codificador usará os vetores de movimento de ocultação.
