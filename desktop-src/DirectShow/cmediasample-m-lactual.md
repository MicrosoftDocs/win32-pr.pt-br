# <a name="cmediasamplem_lactual-member"></a>Membro de CMediaSample:: m \_ lActual

Comprimento dos dados válidos no buffer, em bytes. O valor deve ser igual ou menor que o tamanho do buffer, especificado pela variável de membro [**CMediaSample:: m \_ cbBuffer**](cmediasample-m-cbbuffer.md) .

## <a name="syntax"></a>Syntax


```C++
LONG m_lActual;
```
