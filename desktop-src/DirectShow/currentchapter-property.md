# <a name="currentchapter-property"></a>Propriedade CurrentChapter

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `CurrentChapter` propriedade recupera o número do capítulo em execução no momento.

``` syntax
[ iCurChapter = ] MSWebDVD.CurrentChapter
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro que representa o capítulo atual no título atual.

## <a name="remarks"></a>Comentários

Esta propriedade é somente leitura sem valor padrão.
