# <a name="dll-setup-functions"></a>Funções de instalação de DLL

Essas funções registram um filtro.



| Função                                                         | Descrição                                        |
|------------------------------------------------------------------|----------------------------------------------------|
| [**AMovieDllRegisterServer**](amoviedllregisterserver.md)       | Obsoleto.                                          |
| [**AMovieDllRegisterServer2**](amoviedllregisterserver2.md)     | Registra e não registra filtros.                 |
| [**AMovieDllUnregisterServer**](amoviedllunregisterserver.md)   | Obsoleto.                                          |
| [**AMovieSetupRegisterFilter**](amoviesetupregisterfilter.md)   | Obsoleto.                                          |
| [**AMovieSetupRegisterFilter2**](amoviesetupregisterfilter2.md) | Registra o valor, os pinos e os tipos de mídia de um filtro. |
