### <a name="frequency-tables"></a>Tabelas de frequência

Em todo o mundo, há um conjunto finito e conhecido de padrões de frequência de difusão. Esses padrões foram identificados por seu país/região de origem, mas, em muitos casos, vários países/regiões compartilham o mesmo padrão. O filtro Tv Tuner define uma tabela para cada padrão.

No entanto, não há nenhum conjunto finito e conhecido de padrões de frequência de cabo. Os provedores de cabos em alguns países/regiões são livres para transmitir programação em frequências de sua escolha. Frequências de vários padrões geralmente são misturadas. Na maioria dos casos, essa prática se correlaciona diretamente com o país/região de origem para cada um dos padrões de frequência. O filtro TV Tuner define tabelas para os poucos padrões de cabo (e países/regiões) que têm alocações de frequência de cabo confiáveis. Para todos os outros, ele usa a [Tabela de Frequência Unicavel](unicable-frequency-table.md).

As tabelas de frequência nas seções a seguir são extraídas textualmente das definições de arquivo de recurso que foram usadas para criar as tabelas no filtro Definador de TV. Cada tabela começa com sua ID de recurso. Você pode usar essas IDs de recurso para fazer referência cruzada da tabela Atribuições de [País/Região](country-region-assignments.md) para determinar quais tabelas estão em uso para cada país/região ou localidade.

As tabelas de frequência incluem os números teóricos de canal superior e inferior (ou índices de frequência) para cada código de país/região. Esse intervalo pode ser estendido usando o Registro; Para obter mais informações, consulte [Substituições de frequência](frequency-overrides.md).
