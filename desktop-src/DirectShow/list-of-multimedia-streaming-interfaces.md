# <a name="list-of-multimedia-streaming-interfaces"></a>Lista de interfaces de streaming de multimídia

Este artigo inclui as seções a seguir.

-   [Interfaces de streaming de multimídia base](base-multimedia-streaming-interfaces.md)
-   [Interfaces de streaming do DirectDraw](directdraw-streaming-interfaces.md)
-   [Interfaces de streaming de áudio](audio-streaming-interfaces.md)
