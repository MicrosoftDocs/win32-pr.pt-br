# <a name="tv-audio-filter"></a>Filtro de áudio de TV

O filtro de áudio de TV fornece controle de decodificação de áudio de televisão, seleção de estéreo ou monoaural e seleção de programa de áudio secundário (SAP).



|                                          |                                                                                                                                |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Filtrar interfaces                        | [**IAMTVAudio**](/windows/desktop/api/Strmif/nn-strmif-iamtvaudio), **IPersistPropertyBag**, **ISpecifyPropertyPages**, [**IKsPropertySet**](ikspropertyset.md) |
| Tipos de mídia de pino de entrada                    | AnalogAudio de MEDIATYPE \_                                                                                                         |
| Interfaces de pino de entrada                     | **IKsPropertySet**                                                                                                             |
| Tipos de mídia do pino de saída                   | AnalogAudio de MEDIATYPE \_                                                                                                         |
| Interfaces de pino de saída                    | **IKsPropertySet**                                                                                                             |
| CLSID do filtro                             | \_TVAUDIOFILTER CLSID                                                                                                           |
| CLSID de página de propriedades                      | \_KSCATEGORY \_ TVAUDIO                                                                                                        |
| [Núcleo](merit.md)                       | MÉRITO \_ \_ não \_ use                                                                                                            |
| [Categoria do filtro](filter-categories.md) | \_AUDIOINPUTDEVICECATEGORY CLSID                                                                                                |
