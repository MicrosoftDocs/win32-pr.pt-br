# <a name="tooltipmaxwidth-property"></a>Propriedade ToolTipMaxWidth

> [!Note]  
> esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `ToolTipMaxWidth` propriedade define ou recupera a largura máxima da dica de ferramenta associada ao objeto **MSWebDVD** .

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>Valor Retornado

Retorna a largura máxima para a dica de ferramenta do aplicativo como um inteiro.

## <a name="remarks"></a>Comentários

Essa propriedade é de leitura/gravação com um valor padrão de 200 pixels.
