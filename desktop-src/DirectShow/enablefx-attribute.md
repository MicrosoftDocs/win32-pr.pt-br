# <a name="enablefx-attribute"></a>Atributo enablefx

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `enablefx` atributo especifica se os efeitos estão habilitados. Se o valor for **true**, o mecanismo de renderização processará efeitos. Caso contrário, isso não acontece. O valor padrão é **true**.

## <a name="possible-values"></a>Valores possíveis

Os valores a seguir são definidos como TRUE: y, Y, t, T, 1. Os valores a seguir são definidos como FALSE: n, N, f, F, 0 (zero).

## <a name="applies-to"></a>Aplica-se A

[**linha do tempo**](timeline-element.md)
