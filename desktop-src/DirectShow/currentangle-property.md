# <a name="currentangle-property"></a>Propriedade CurrentAngle

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `CurrentAngle` propriedade define ou recupera o ângulo atual em um bloco de ângulo.

``` syntax
[ iCurAngle = ] MSWebDVD.CurrentAngle
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro que representa o ângulo. Deve ser um valor de 1 a 9.

## <a name="remarks"></a>Comentários

Esta propriedade é de leitura/gravação sem valor padrão.
