# <a name="am_rate_simpleratechange-property"></a>Propriedade de SimpleRateChange da \_ taxa de am \_

Essa propriedade é usada para enviar alterações de taxa ao decodificador. O tipo de dados para essa propriedade é uma estrutura **am \_ SimpleRateChange** , que fornece a nova taxa de reprodução e a hora da apresentação quando a nova taxa entra em vigor.



| Label | Valor |
|-------------------|-----------------------------------------------------|
| GUID do Conjunto de Propriedades | \_KSPROPSETID \_ TSRateChange                       |
| ID da propriedade       | \_SimpleRateChange de taxa de am \_                          |
| Tipo de Dados         | [**AM \_ SimpleRateChange**](/previous-versions/windows/desktop/api/Dvdmedia/ns-dvdmedia-am_simpleratechange) |
