# <a name="getdelaytime-method"></a>Método GetDelayTime

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `GetDelayTime` método recupera o tempo de atraso para a dica de ferramenta associada ao objeto **MSWebDVD** .

``` syntax
[ iDelay = ] MSWebDVD.GetDelayTime()
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro que indica o tempo de atraso da dica de ferramenta, em milissegundos.
