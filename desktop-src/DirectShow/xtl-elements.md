# <a name="xtl-elements"></a>Elementos XTL

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O formato XTL define os elementos a seguir.



| Elemento                                  | Descrição                                                                    |
|------------------------------------------|--------------------------------------------------------------------------------|
| [**no**](at-element.md)                 | Especifica uma configuração baseada em tempo para um elemento **param** .                        |
| [**clip**](clip-element.md)             | Especifica um arquivo de origem.                                                       |
| [**Spot**](composite-element.md)   | Define uma composição.                                                         |
| [**funciona**](effect-element.md)         | Define um efeito.                                                             |
| [**group**](group-element.md)           | Define um grupo (áudio ou vídeo).                                              |
| [**linear**](linear-element.md)         | Especifica uma configuração baseada em tempo para um elemento **param** .                        |
| [**param**](param-element.md)           | Especifica o valor de uma propriedade em uma transição, efeito ou outro subobjeto. |
| [**linha do tempo**](timeline-element.md)     | Define a linha do tempo.                                                          |
| [**controles**](track-element.md)           | Define uma faixa.                                                               |
| [**transição**](transition-element.md) | Define uma transição.                                                          |
