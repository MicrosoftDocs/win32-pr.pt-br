# <a name="cbasepropertypagem_hwnd-member"></a>Membro HWND CBasePropertyPage:: m \_

A `m_hwnd` variável de membro contém um identificador para a janela da caixa de diálogo. Essa variável de membro é inicializada depois que o objeto cria a janela de diálogo, quando a função **CreateDialogParam** retorna.

## <a name="syntax"></a>Syntax


```C++
HWND m_hwnd;
```
