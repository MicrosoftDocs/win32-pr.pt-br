# <a name="dmo-demo-sample"></a>DMO Exemplo de demonstração

## <a name="description"></a>Descrição

Este aplicativo de exemplo demonstra como usar o [directX media objects](directx-media-objects.md) (DMO) em um aplicativo. Ele transmite dados de áudio de um arquivo WAV por meio de um efeito de áudio DMO para um buffer DirectSound.

## <a name="usage"></a>Uso

Clique no **botão Arquivo de** Som para carregar um arquivo de som. Use a caixa de combinação listada para selecionar um efeito de áudio DMO. Clique no **botão Reproduzir** para ouvir o arquivo de som reproduzir com o efeito de áudio.

## <a name="downloading-the-sample"></a>Baixando o exemplo

Para baixar os exemplos DirectShow SDK, instale a versão mais recente do [SDK Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este exemplo é instalado no seguinte caminho: Multimídia de Exemplos Raiz do *\[ SDK \]* \\ DirectShow DMO \\ \\ \\ \\ DMODemo.
