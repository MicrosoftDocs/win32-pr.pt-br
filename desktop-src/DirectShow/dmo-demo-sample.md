# <a name="dmo-demo-sample"></a>Exemplo de demonstração de DMO

## <a name="description"></a>Description

Este aplicativo de exemplo demonstra como usar os [objetos de mídia do DirectX](directx-media-objects.md) (DMO) em um aplicativo. Ele transmite dados de áudio de um arquivo WAV por meio de um efeito de áudio DMO para um buffer do DirectSound.

## <a name="usage"></a>Uso

Clique no botão **arquivo de som** para carregar um arquivo de som. Use a caixa de combinação suspensa para selecionar um efeito de áudio DMO. Clique no botão **reproduzir** para ouvir o arquivo de som tocado com o efeito de áudio.

## <a name="downloading-the-sample"></a>Baixando o exemplo

Para baixar os exemplos do SDK do DirectShow, instale a versão mais recente do [SDK do Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este exemplo é instalado sob o seguinte caminho: exemplos de *\[ raiz \] do SDK* \\ \\ multimídia \\ DirectShow \\ DMO \\ DMODemo.
