# <a name="coutputqueuefreesamples-method"></a>Método COutputQueue. FreeSamples

O `FreeSamples` método libera Todas as amostras pendentes.

## <a name="syntax"></a>Sintaxe


```C++
void FreeSamples();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Esse método remove todas as amostras pendentes da fila e da matriz de exemplo.
