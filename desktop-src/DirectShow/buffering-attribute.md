# <a name="buffering-attribute"></a>Atributo de buffer

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `buffering` atributo especifica o número de quadros a serem processados antecipadamente durante a visualização.

## <a name="possible-values"></a>Valores possíveis

O valor deve ser um número maior que 1. O valor padrão é 30.

[**group**](group-element.md)
