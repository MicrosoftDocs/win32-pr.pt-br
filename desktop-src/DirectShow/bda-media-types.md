# <a name="bda-media-types"></a>Tipos de mídia BDA

As tabelas a seguir descrevem os tipos de mídia usados para filtros de televisão digital BDA.

## <a name="antenna"></a>Antena

Esse tipo de mídia é usado para a conexão de PIN entre o filtro do provedor de rede e o filtro de sintonizador.



|             |                                  |
|-------------|----------------------------------|
| Tipo principal  | \_ \_ antena BDA do tipo KSDATAFORMAT \_ |
| Subtype     | MEDIASUBTYPE \_ nenhum               |
| Tipo de formato | Formatar \_ nenhum                     |



 

## <a name="intermediate-frequency-if-signal"></a>Sinal de frequência intermediária (IF)

Esse tipo de mídia representa o sinal de se que é enviado do sintonizador para o demodulador.



|             |                                     |
|-------------|-------------------------------------|
| Tipo principal  | KSDATAFORMAT \_ tipo \_ BDA \_ If \_ Signal |
| Subtype     | MEDIASUBTYPE \_ nenhum                  |
| Tipo de formato | Formatar \_ nenhum                        |
