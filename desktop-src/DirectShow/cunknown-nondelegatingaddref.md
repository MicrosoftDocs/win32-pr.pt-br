# <a name="cunknownnondelegatingaddref-method"></a>Método CUnknown. NonDelegatingAddRef

O `NonDelegatingAddRef` método incrementa a contagem de referência no objeto. Esse método implementa o método **INonDelegatingUnknown:: NonDelegatingAddRef** .

## <a name="syntax"></a>Sintaxe


```C++
ULONG NonDelegatingAddRef();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna a contagem de referência.
