# <a name="getnumberofchapters-method"></a>Método GetNumberOfChapters

> [!Note]  
> esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `GetNumberOfChapters` método recupera o número de capítulos no título especificado.

``` syntax
[iChapters = ] MSWebDVD.GetNumberOfChapters(iTitle)
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*iTitle*
</dt> <dd>

Especifica o título como um inteiro.

</dd> </dl>

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro de 1 a 999 indicando o número de capítulos.
