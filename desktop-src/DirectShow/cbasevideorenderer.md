# <a name="cbasevideorenderer-class"></a>Classe CBaseVideoRenderer

![Hierarquia de classes cbasevideorenderer](images/rbase03.png)

A `CBaseVideoRenderer` classe base é usada para criar filtros de renderização de vídeo.



| Membros de Dados Protegidos                                                                 | Descrição                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| m \_ bDrawLateFrames                                                                     | Sinalizar para sinalizar que nenhum quadro deve ser descartado. Somente depuração. Isso destrói a sincronização.                                                                                                                                          |
| m \_ bSupplierHandlingQuality                                                            | **TRUE** indica que as mensagens de controle de qualidade estão sendo manipuladas. Isso permite que o renderista saiba esperar até o mais tarde possível para soltar quadros em si e exibir o próximo quadro muito cedo depois que o fornecedor tiver descartado um quadro. |
| m \_ cFramesDrawn                                                                        | Número total de quadros que foram desenhados desde o início do streaming.                                                                                                                                                                 |
| m \_ cFramesDropped                                                                      | Quadros cumulativos que foram descartados no renderdor desde o início do streaming. Os quadros também podem ser descartados upstream sem que o renderista os reconheça.                                                                         |
| m \_ idDecision                                                                          | ID da MSR \_ para o código de decisão de [**ShouldDrawSampleNow.**](cbasevideorenderer-shoulddrawsamplenow.md)                                                                                                                              |
| m \_ idDuration                                                                          | ID da \_ MSR durante um quadro.                                                                                                                                                                                                 |
| m \_ idFrameAccuracy                                                                     | Identificador do log de desempenho para o tempo em milissegundos em que o quadro estava atrasado.                                                                                                                                                     |
| m \_ idFrameAvg                                                                          | Identificador de log de desempenho para o tempo médio do quadro usado para sincronização e controle de qualidade.                                                                                                                          |
| m \_ idQualityRate                                                                       | ID da \_ MSR para a taxa de qualidade solicitada.                                                                                                                                                                                              |
| m \_ idQualityTime                                                                       | ID da MSR \_ para o tempo de qualidade solicitado.                                                                                                                                                                                              |
| m \_ idRenderAvg                                                                         | Identificador de log de desempenho para o tempo médio do renderista registrado.                                                                                                                                                                   |
| m \_ idSchLateTime                                                                       | ID da MSR \_ para o atraso do quadro quando agendado.                                                                                                                                                                                   |
| m \_ idSendQuality                                                                       | ID da MSR \_ para cronometragem das notificações (nãousadas).                                                                                                                                                                                       |
| m \_ idTimeStamp                                                                         | ID da MSR \_ para um carimbo de data/hora do quadro.                                                                                                                                                                                                      |
| m \_ idWait                                                                              | Identificador do log de desempenho para o tempo de espera registrado (nãousado).                                                                                                                                                                      |
| m \_ idWaitReal                                                                          | Identificador do log de desempenho para o tempo de espera real.                                                                                                                                                                                   |
| m \_ iSumFrameTime                                                                       | Soma dos tempos de interframe; necessário para a página de propriedades.                                                                                                                                                                           |
| m \_ iSumSqAcc                                                                           | Soma dos quadrados das precisões (em milissegundos) necessários para a página de propriedades.                                                                                                                                                 |
| m \_ iSumSqFrameTime                                                                     | Soma dos quadrados de tempos entre períodos; necessário para a página de propriedades.                                                                                                                                                                |
| m \_ iTotAcc                                                                             | Soma das precisões (em milissegundos) necessárias para a página de propriedades.                                                                                                                                                                |
| m \_ nNormal                                                                             | Número de quadros consecutivos desenhados no horário agendado. Um número negativo indica que um quadro acabou de ser descartado pelo renderdor.                                                                                          |
| m \_ trDuration                                                                          | Duração do último quadro (diferença entre as horas de início e término).                                                                                                                                                             |
| m \_ trEarliness                                                                         | A que horas um quadro pode ser tocado quando um quadro acaba de ser descartado.                                                                                                                                                        |
| m \_ trFrame                                                                             | Tempo registrado mais recentemente entre quadros. Usado em medidas estatísticas.                                                                                                                                                        |
| m \_ trFrameAvg                                                                          | Tempo de interframe médio em unidades de tempo de referência.                                                                                                                                                                                     |
| m \_ trLastDraw                                                                          | Hora do quadro anterior. Usado para referências de tempo entre períodos.                                                                                                                                                                         |
| m \_ trLate                                                                              | Tempo que o quadro atual estava atrasado. Usado em medidas estatísticas.                                                                                                                                                    |
| m \_ trRenderAvg                                                                         | Tempo que os quadros estão levando para executar a transferência de bloco de bits.                                                                                                                                                                       |
| m \_ trRenderLast                                                                        | Hora da última transferência de bloco de bits do quadro.                                                                                                                                                                                          |
| m \_ trRenderStart                                                                       | Hora em que a transferência de bloco de bits foi iniciada. Usado para obter **m \_ trRenderLast.**                                                                                                                                                                |
| m \_ trThrottle                                                                          | Período a ser inserido após a renderização de cada quadro, normalmente usado quando a qualidade do áudio foi aumentada e o desempenho de vídeo deve ser reduzido para permitir isso.                                                                             |
| m \_ trWaitAvg                                                                           | Tempo médio de espera em unidades de tempo de referência.                                                                                                                                                                                           |
| m \_ tStreamingStart                                                                     | Usado para estatísticas de página de propriedades. Representa a hora de início do processo de streaming atual ou o processo de streaming anterior, se não estiver transmitindo no momento.                                                                          |
