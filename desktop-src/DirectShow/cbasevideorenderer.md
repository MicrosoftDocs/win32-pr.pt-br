# <a name="cbasevideorenderer-class"></a>Classe CBaseVideoRenderer

![hierarquia de classe CBaseVideoRenderer](images/rbase03.png)

A `CBaseVideoRenderer` classe base é usada para criar filtros de renderização de vídeo.



| Membros de Dados Protegidos                                                                 | Descrição                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_bDrawLateFrames m                                                                     | Sinalizador para sinalizar que nenhum quadro deve ser removido. Somente depuração. Isso destrói a sincronização.                                                                                                                                          |
| \_bSupplierHandlingQuality m                                                            | **Verdadeiro** indica que as mensagens de controle de qualidade estão sendo tratadas. Isso permite que o renderizador saiba aguardar até o mais tarde possível para descartar os quadros em si e exibir o próximo quadro logo após o fornecedor ter descartado um quadro. |
| \_cFramesDrawn m                                                                        | Número total de quadros que foram desenhados desde o início do streaming.                                                                                                                                                                 |
| \_cFramesDropped m                                                                      | Quadros cumulativos que foram descartados no renderizador desde o início do streaming. Os quadros também podem ser retirados upstream sem que o renderizador os reconheça.                                                                         |
| \_idDecision m                                                                          | ID de MSR \_ para o código de decisão de [**ShouldDrawSampleNow**](cbasevideorenderer-shoulddrawsamplenow.md).                                                                                                                              |
| \_idDuration m                                                                          | ID de MSR \_ para a duração de um quadro.                                                                                                                                                                                                 |
| \_idFrameAccuracy m                                                                     | Identificador de log de desempenho para o tempo em milissegundos em que o quadro estava atrasado.                                                                                                                                                     |
| \_idFrameAvg m                                                                          | Identificador de log de desempenho para o tempo médio de quadro usado para sincronização e controle de qualidade.                                                                                                                          |
| \_idQualityRate m                                                                       | \_ID de MSR para a taxa de qualidade solicitada.                                                                                                                                                                                              |
| \_idQualityTime m                                                                       | \_ID de MSR para o tempo de qualidade solicitado.                                                                                                                                                                                              |
| \_idRenderAvg m                                                                         | Identificador do log de desempenho para o tempo médio do processador registrado.                                                                                                                                                                   |
| \_idSchLateTime m                                                                       | \_ID de MSR para saber a tarde em que o quadro foi agendado.                                                                                                                                                                                   |
| \_idSendQuality m                                                                       | \_ID de MSR para cronometrar as notificações (não usado).                                                                                                                                                                                       |
| \_idTimeStamp m                                                                         | \_ID de MSR para um carimbo de data/hora do quadro.                                                                                                                                                                                                      |
| \_idWait m                                                                              | Identificador de log de desempenho para o tempo de espera gravado (não usado).                                                                                                                                                                      |
| \_idWaitReal m                                                                          | Identificador de log de desempenho para o tempo de espera verdadeiro.                                                                                                                                                                                   |
| \_iSumFrameTime m                                                                       | Soma dos horários de interestruturação; necessário para a página de propriedades.                                                                                                                                                                           |
| \_iSumSqAcc m                                                                           | Soma dos quadrados da imprecisões (em milissegundos) necessários para a página de propriedades.                                                                                                                                                 |
| \_iSumSqFrameTime m                                                                     | Soma dos quadrados de períodos de interestruturação; necessário para a página de propriedades.                                                                                                                                                                |
| \_iTotAcc m                                                                             | Soma da imprecisões (em milissegundos) necessária para a página de propriedades.                                                                                                                                                                |
| \_nNormal m                                                                             | Número de quadros consecutivos desenhados no horário agendado. Um número negativo indica que um quadro acabou de ser Descartado pelo renderizador.                                                                                          |
| \_trDuration m                                                                          | Duração do último quadro (diferença entre as horas de início e término).                                                                                                                                                             |
| \_trEarliness m                                                                         | Como o início de um quadro pode ser reproduzido quando um quadro acabou de ser Descartado.                                                                                                                                                        |
| \_trFrame m                                                                             | Tempo gravado mais recentemente entre quadros. Usado em medidas estatísticas.                                                                                                                                                        |
| \_trFrameAvg m                                                                          | Tempo médio entre quadros em unidades de tempo de referência.                                                                                                                                                                                     |
| \_trLastDraw m                                                                          | Hora do quadro anterior. Usado para referências de tempo entre quadros.                                                                                                                                                                         |
| \_trLate m                                                                              | Quantidade de tempo em que o quadro atual estava atrasado. Usado em medidas estatísticas.                                                                                                                                                    |
| \_trRenderAvg m                                                                         | Tempo que os quadros estão realizando para executar a transferência de bloco de bits.                                                                                                                                                                       |
| \_trRenderLast m                                                                        | Hora para a última transferência de bloco de bits do quadro.                                                                                                                                                                                          |
| \_trRenderStart m                                                                       | Hora em que a transferência de bloco de bits foi iniciada. Usado para obter **m \_ trRenderLast**.                                                                                                                                                                |
| \_trThrottle m                                                                          | Período a ser inserido após a renderização de cada quadro, normalmente usado quando a qualidade de áudio aumenta e o desempenho do vídeo deve ser reduzido para permitir isso.                                                                             |
| \_trWaitAvg m                                                                           | Tempo de espera médio nas unidades de tempo de referência.                                                                                                                                                                                           |
| \_tStreamingStart m                                                                     | Usado para estatísticas de página de propriedades. Representa a hora de início do processo de streaming atual ou o processo de streaming anterior, se não estiver transmitindo no momento.                                                                          |
