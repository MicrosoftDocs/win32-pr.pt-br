# <a name="avencstatvideooutputframerate-property"></a>Propriedade AVEncStatVideoOutputFrameRate

Retorna a taxa média de quadros do conteúdo do vídeo, em quadros por segundo.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncStatVideoOutputFrameRate**

## <a name="remarks"></a>Comentários

Essa propriedade estará disponível após a conclusão da codificação.
