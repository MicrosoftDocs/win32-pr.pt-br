# <a name="cimagepalettem_pdrawimage-member"></a>Membro de CImagePalette:: m \_ pDrawImage

A `m_pDrawImage` variável de membro é um ponteiro para o objeto [**CDrawImage**](cdrawimage.md) que desenha a imagem de vídeo.

## <a name="syntax"></a>Syntax


```C++
CDrawImage *m_pDrawImage;
```
