# <a name="avencvideoinputcolorlighting-property"></a>Propriedade AVEncVideoInputColorLighting

Especifica as condições de iluminação pretendidas para exibir o vídeo de entrada.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoInputColorLighting**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncVideoColorLighting**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting) .
