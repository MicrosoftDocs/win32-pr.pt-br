# <a name="directshow-objects"></a>Objetos do DirectShow

Este tópico contém entradas de referência para os objetos COM principais incluídos no DirectShow, além de filtros. Para obter uma lista de filtros incluídos com o DirectShow, consulte [filtros do DirectShow](directshow-filters.md).



| Objeto                                                   | Descrição                                                      |
|----------------------------------------------------------|------------------------------------------------------------------|
| [Construtor do grafo de captura](capture-graph-builder.md)       | Cria gráficos de captura de vídeo.                                     |
| [Construtor de gráficos de DVD](dvd-graph-builder.md)               | Cria grafos de reprodução de DVD.                                      |
| [Gerenciador de gráfico de filtro](filter-graph-manager.md)         | Cria e controla grafos de filtro.                               |
| [Mapeador de filtro](filter-mapper.md)                       | Pesquisa os filtros registrados no registro.                    |
| [Conjunto de propriedades de mídia](media-property-bag.md)             | Define ou recupera partes de informações e de DISP em arquivos AVI.             |
| [Alocador de memória](memory-allocator.md)                 | Aloca buffers para exemplos de mídia.                             |
| [Relógio de referência do sistema](system-reference-clock.md)     | Implementa um relógio de referência.                                    |
| [Enumerador de dispositivo do sistema](system-device-enumerator.md) | Enumera filtros e dispositivos de hardware instalados no sistema. |
