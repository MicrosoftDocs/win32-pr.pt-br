# <a name="enumerating-objects-in-a-filter-graph"></a>Enumerando objetos em um filtro Graph

Um aplicativo pode precisar localizar um filtro específico no grafo de filtro ou até mesmo um pin específico em um filtro. Por exemplo, ele pode usar uma interface que um filtro específico expõe. Ou pode construir um grafo de filtro especializado e precisar chamar métodos em pinos individuais para conectar os filtros. Para essa finalidade, DirectShow fornece vários métodos para enumerar objetos em um grafo de filtro.

Os enumeradores discutidos nesta seção seguem o formulário padrão usado pelas interfaces de enumeração COM. Para obter mais informações, consulte o tópico "IEnumXXXX" no SDK da plataforma. Para obter informações sobre como enumerar filtros registrados no computador do usuário, mas ainda não estão no grafo de filtro, consulte Enumerando dispositivos [e filtros.](enumerating-devices-and-filters.md)

-   [Enumerando filtros](enumerating-filters.md)
-   [Enumerando pinos](enumerating-pins.md)
-   [Enumerando tipos de mídia](enumerating-media-types.md)
