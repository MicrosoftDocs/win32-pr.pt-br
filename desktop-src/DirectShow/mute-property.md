# <a name="mute-property"></a>Propriedade de mudo

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `Mute` propriedade ativa ou desativa a saída do fluxo de áudio.

``` syntax
[ bMute = ] MSWebDVD.Mute
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor booliano que indica se o fluxo de áudio deve ser mudo.

## <a name="remarks"></a>Comentários

Essa propriedade é de leitura/gravação com um valor padrão de false.
