# <a name="avencddservice-property"></a>Propriedade AVEncDDService

Especifica o serviço de áudio contido em um fluxo de áudio Dolby Digital. Essa propriedade se aplica aos codificadores Dolby Digital Audio.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDService**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncDDService**](/windows/desktop/api/codecapi/ne-codecapi-eavencddservice) .

## <a name="remarks"></a>Comentários

Esta propriedade é de leitura/gravação.
