# <a name="mswebdvd-object"></a>Objeto MSWebDVD

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

> [!Note]  
> Essa API está preterida. Para obter informações sobre reprodução e navegação em DVD no DirectShow, consulte [aplicativos de DVD](dvd-applications.md).

 

Os métodos, as propriedades e os eventos do objeto MSWebDVD permitem que um aplicativo controle todos os aspectos de navegação e reprodução de DVD-Video e recupere informações do disco. O objeto MSWebDVD não executa o trabalho de navegação real; em vez disso, ele passa os comandos para o filtro de [navegador de DVD](dvd-navigator-filter.md) , um componente de® do Microsoft® DirectShow que lê DVD-Video discos.

Os métodos e as propriedades do MSWebDVD agem sobre o estado atual do navegador de DVD ou sobre as informações em um disco, ou ambos. Para armazenar e recuperar vários tipos de informações específicas do aplicativo no registro, como as preferências do usuário para os níveis pai e o idioma padrão, use os métodos do objeto [MSDVDAdm](msdvdadm-object.md) . Use a propriedade [**DVDAdm**](dvdadm-property.md) para acessar esse objeto.

> [!Note]  
> A partir do DirectX 9.0 b, o objeto é carregado somente em zonas confiáveis. Ele não será carregado em zonas não confiáveis.
