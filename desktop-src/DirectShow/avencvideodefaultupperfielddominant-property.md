# <a name="avencvideodefaultupperfielddominant-property"></a>Propriedade AVEncVideoDefaultUpperFieldDominant

Especifica qual campo é exibido primeiro.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoDefaultUpperFieldDominant**

## <a name="remarks"></a>Comentários

Se o valor for **Variant \_ true**, o campo superior será exibido primeiro. Caso contrário, o campo inferior será exibido primeiro.
