# <a name="defaultfx-attribute"></a>Atributo defaultfx

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `defaultfx` atributo especifica o identificador de classe (CLSID) de um efeito padrão para a linha do tempo.

## <a name="possible-values"></a>Valores possíveis

Deve ser uma cadeia de caracteres com o formato {00000000-0000-0000-0000-000000000000} .

## <a name="applies-to"></a>Aplica-se A

[**linha do tempo**](timeline-element.md)
