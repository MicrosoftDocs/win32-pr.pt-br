# <a name="cdrawimagem_endsample-member"></a>Membro de exemplo endCDrawImage:: m \_

A `m_EndSample` variável de membro especifica a hora de parada do exemplo mais recente.

## <a name="syntax"></a>Sintaxe


```C++
CRefTime m_EndSample;
```



## <a name="remarks"></a>Comentários

O valor só é válido dentro do método [**CDrawImage::D isplaysampletimes**](cdrawimage-displaysampletimes.md) .
