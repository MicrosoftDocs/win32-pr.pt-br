# <a name="ccritsecm_lockcount-member"></a>Membro de CCritSec:: m \_ lockCount

Número de bloqueios pendentes neste objeto.

## <a name="syntax"></a>Sintaxe


```C++
DWORD m_lockCount;
```



## <a name="remarks"></a>Comentários

Essa variável de membro é definida somente na versão de depuração da classe base. A [seção crítica Depurando funções funções](critical-section-debugging-functions.md) usam esse membro.
