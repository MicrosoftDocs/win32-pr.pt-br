# <a name="avencvideoheaderhours-property"></a>Propriedade AVEncVideoHeaderHours

Especifica o número de hora inicial no cabeçalho do grupo de imagens (GOP).

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoHeaderHours**

## <a name="property-value"></a>Valor da propriedade

O intervalo válido dessa propriedade é de 0 a 23, inclusive.
