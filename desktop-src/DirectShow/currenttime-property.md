# <a name="currenttime-property"></a>Propriedade CurrentTime

> [!Note]  
> esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `CurrentTime` propriedade recupera o tempo de reprodução atual.

``` syntax
[ sCurTime = ] MSWebDVD.CurrentTime
```

## <a name="return-value"></a>Valor Retornado

Retorna uma **cadeia de caracteres** de 11 caracteres que representa o tempo de reprodução atual no formato "*hh: mm: SS: FF*" (horas, minutos, segundos, quadros).

## <a name="remarks"></a>Comentários

Esta propriedade é somente leitura sem valor padrão.
