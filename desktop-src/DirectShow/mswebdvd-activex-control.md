# <a name="mswebdvd-activex-control"></a>Controle ActiveX MSWebDVD

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

> [!Note]  
> Este componente foi preterido. Para obter informações sobre reprodução e navegação em DVD no DirectShow, consulte [aplicativos de DVD](dvd-applications.md).

 

O controle de® do Microsoft® MSWebDVD ActiveX pode ser usado para criar um aplicativo de DVD baseado em script. Ele expõe os objetos e eventos a seguir para um aplicativo.

-   [MSWebDVD](mswebdvd-object.md)
-   [MSDVDAdm](msdvdadm-object.md)
-   [DVDRect](dvdrect-object.md)
-   [Eventos de MSWebDVD](mswebdvd-events.md)
