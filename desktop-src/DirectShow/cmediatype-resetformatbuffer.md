# <a name="cmediatyperesetformatbuffer-method"></a>Método CMediaType. ResetFormatBuffer

O `ResetFormatBuffer` método exclui o bloco de formato. O método define os membros **cbFormat** e **pbFormat** da estrutura **do \_ \_ tipo de mídia am** como **NULL**.

## <a name="syntax"></a>Sintaxe


```C++
void ResetFormatBuffer();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
