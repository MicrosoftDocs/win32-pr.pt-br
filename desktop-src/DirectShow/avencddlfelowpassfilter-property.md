# <a name="avencddlfelowpassfilter-property"></a>Propriedade AVEncDDLFELowPassFilter

Especifica se um filtro de passagem baixa é aplicado ao canal de baixo efeito de frequência (LFE) em um fluxo de áudio Dolby Digital. Essa propriedade se aplica aos codificadores Dolby Digital Audio.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDLFELowPassFilter**
