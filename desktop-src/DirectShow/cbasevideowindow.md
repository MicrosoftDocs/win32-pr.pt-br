# <a name="cbasevideowindow-class"></a>Classe CBaseVideoWindow

![hierarquia de classe CBaseVideoWindow](images/wctrl01.png)

A `CBaseVideoWindow` classe implementa os métodos **IDispatch** da interface [**IVideoWindow**](/windows/desktop/api/Control/nn-control-ivideowindow) . Os métodos **IVideoWindow** restantes são deixados como métodos virtuais puros e devem ser implementados por uma classe derivada.

Os métodos **IDispatch** nessa classe são implementações padrão e não são descritos detalhadamente aqui.
