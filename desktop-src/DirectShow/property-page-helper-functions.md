# <a name="property-page-helper-functions"></a>Funções auxiliares da página de propriedades

O arquivo de header Videoctl.h nas classes base microsoft® DirectShow® fornece funções para ajudar com implementações de página de propriedades.



| Função                                                 | Descrição                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**GetDialogSize**](getdialogsize.md)                   | Recupera o tamanho de uma caixa de diálogo de recurso em pixels de tela.                          |
| [**StringFromResource**](stringfromresource.md)         | Carrega uma cadeia de caracteres de um arquivo de recurso com o identificador de recurso determinado.                |
| [**WideStringFromResource**](widestringfromresource.md) | Carrega uma cadeia de caracteres largos de um arquivo de recurso com o identificador de recurso determinado. |
