# <a name="avencstataudiopeakpcmvalue-property"></a>Propriedade AVEncStatAudioPeakPCMValue

Retorna o nível de volume mais alto que estava presente no conteúdo de áudio.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncStatAudioPeakPCMValue**

## <a name="remarks"></a>Comentários

Essa propriedade estará disponível após a conclusão da codificação.

Esta propriedade se aplica somente à codificação de taxa de bits variável (VBR) com base em qualidade
