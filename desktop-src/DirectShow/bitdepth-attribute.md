# <a name="bitdepth-attribute"></a>Atributo bitdepth

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `bitdepth` atributo especifica a profundidade de bits do vídeo de saída, em bits por pixel.

## <a name="possible-values"></a>Valores possíveis

Deve ser 16, 24 ou 32. O valor padrão é 16.

## <a name="applies-to"></a>Aplica-se A

[**group**](group-element.md)

## <a name="remarks"></a>Comentários

Defina esse atributo somente se o atributo de tipo for `video` .
