# <a name="defaultmenulanguage-property"></a>Propriedade DefaultMenuLanguage

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `DefaultMenuLanguage` propriedade recupera o idioma do menu padrão.

``` syntax
[ iLang = ] MSWebDVD.DefaultMenuLanguage
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor LCID que contém a ID de idioma primário para o idioma de áudio padrão.

## <a name="remarks"></a>Comentários

Essa propriedade é leitura/gravação sem valor padrão.
