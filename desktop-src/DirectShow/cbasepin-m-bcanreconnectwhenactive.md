# <a name="cbasepinm_bcanreconnectwhenactive-member"></a>Membro de CBasePin:: m \_ bCanReconnectWhenActive

Sinalizador que indica se o PIN dá suporte à reconexão dinâmica.

## <a name="syntax"></a>Sintaxe


```C++
bool m_bCanReconnectWhenActive;
```



## <a name="remarks"></a>Comentários

Esse sinalizador assume como padrão **false**.
