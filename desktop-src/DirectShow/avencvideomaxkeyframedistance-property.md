# <a name="avencvideomaxkeyframedistance-property"></a>Propriedade AVEncVideoMaxKeyframeDistance

Especifica o número máximo de quadros entre quadros-chave.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoMaxKeyframeDistance**

## <a name="property-value"></a>Valor da propriedade

Essa propriedade é retornada como um intervalo de valores. Para obter o intervalo com suporte, chame [**ICodecAPI:: GetParameterRange**](/windows/desktop/api/Strmif/nf-strmif-icodecapi-getparameterrange).
