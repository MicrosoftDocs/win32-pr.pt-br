# <a name="avaudiosamplerate-property"></a>Propriedade AVAudioSampleRate

Obtém a taxa de amostragem do fluxo de bits de áudio, em amostras por segundo.

Esta propriedade é somente para leitura.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVAudioSampleRate**

## <a name="property-value"></a>Valor da propriedade

O conjunto de valores possíveis depende da implementação do decodificador e do fluxo de bits de áudio.
