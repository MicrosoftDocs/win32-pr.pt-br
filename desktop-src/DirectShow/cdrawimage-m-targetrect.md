# <a name="cdrawimagem_targetrect-member"></a>Membro de CDrawImage:: m \_ TargetRect

A `m_TargetRect` variável de membro especifica o retângulo de destino para o desenho. O valor inicial é um retângulo vazio, indicando que a imagem deve ser desenhada para toda a janela de destino.

## <a name="syntax"></a>Syntax


```C++
RECT m_TargetRect;
```
