# <a name="avencmpacodingmode-property"></a>Propriedade AVEncMPACodingMode

Especifica o modo de codificação de áudio MPEG-1/2. Essa propriedade se aplica a codificadores de áudio MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPACodingMode**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncMPACodingMode**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpacodingmode) .
