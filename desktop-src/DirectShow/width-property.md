# <a name="width-property-directshow"></a>Propriedade Width (DirectShow)

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `Width` propriedade define ou recupera a largura do retângulo.

``` syntax
[ iWidth = ] DVDRect.Width
```

## <a name="return-value"></a>Valor Retornado

Retorna a largura, em pixels, do retângulo como um inteiro.

## <a name="remarks"></a>Comentários

Esta propriedade é de leitura/gravação sem valor padrão.
