# <a name="avddsurroundmode-property"></a>Propriedade AVDDSurroundMode

Especifica se o áudio é codificado no Dolby Surround.

Esta propriedade é somente para leitura.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVDDSurroundMode**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVDDSurroundMode.**](/windows/desktop/api/codecapi/ne-codecapi-eavddsurroundmode)
