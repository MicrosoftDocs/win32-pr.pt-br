# <a name="cmediatypeistemporalcompressed-method"></a>Método CMediaType. IsTemporalCompressed

O `IsTemporalCompressed` método determina se o fluxo usa a compactação temporal.

## <a name="syntax"></a>Sintaxe


```C++
BOOL IsTemporalCompressed() const;
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna o valor do membro **bTemporalCompression** .
