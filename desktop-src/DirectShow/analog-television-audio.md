# <a name="analog-television-audio"></a>Áudio de televisão analógica

Com a televisão analógica, o áudio é proveniente do sintonizador de TV. Diferentes cartões sintonizadores devem ser configurados de forma diferente. Alguns cartões usam um conector externo para corrigir o áudio para a linha da placa de som. Outras usam um conector interno para corrigir o áudio para a entrada de CD da placa de som.

Os tópicos a seguir descrevem como Visualizar e capturar áudio de TV no DirectShow:

-   [Visualizando áudio de TV](previewing-tv-audio.md)
-   [Capturando áudio de TV](capturing-tv-audio.md)
