# <a name="src-attribute"></a>Atributo src

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `src` atributo especifica o nome do caminho de um arquivo de mídia. O arquivo pode ser qualquer tipo com suporte no DirectShow ou uma imagem ainda. Os formatos com suporte para imagens do ainda são bitmap, bitmap independente de dispositivo, GIF e JPEG.

## <a name="possible-values"></a>Valores possíveis

Deve ser uma cadeia de caracteres que especifica um nome de caminho.
