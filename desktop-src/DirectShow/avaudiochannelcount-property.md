# <a name="avaudiochannelcount-property"></a>Propriedade AVAudioChannelCount

Obtém o número de canais no fluxo de bits de áudio.

Esta propriedade é somente para leitura.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVAudioChannelCount**

## <a name="remarks"></a>Comentários

O número de canais inclui o canal de baixo efeito de frequência (LFE), se presente.
