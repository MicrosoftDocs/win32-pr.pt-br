# <a name="cutscene-sample"></a>Exemplo de CutScene

## <a name="description"></a>Descrição

O exemplo CutScene reproduz um arquivo de vídeo no modo de tela inteira até que o usuário digite ESC, ENTER ou a barra de espaço.

Execute este exemplo na linha de comando, com o nome de um arquivo de vídeo como um argumento de linha de comando.

## <a name="downloading-the-sample"></a>Baixando o exemplo

Para baixar os DirectShow exemplos do SDK, instale a versão mais recente do [SDK Windows .](https://msdn.microsoft.com/windowsvista/bb980924.aspx)

Este exemplo é instalado no seguinte caminho: *\[ SDK \] Root* \\ Samples Multimedia DirectShow Players \\ \\ \\ \\ CutScene.
