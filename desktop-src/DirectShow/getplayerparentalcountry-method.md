# <a name="getplayerparentalcountry-method"></a>Método GetPlayerParentalCountry

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `GetPlayerParentalCountry` método recupera o país/região atual conforme definido no objeto **MSWebDVD** .

``` syntax
[ iCountryCode = ] MSWebDVD.GetPlayerParentalCountry()
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro com o código de país/região.
