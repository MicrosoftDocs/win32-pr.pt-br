# <a name="directshow-structures"></a>Estruturas do DirectShow

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O DirectShow Editing Services define as estruturas a seguir.



| Estrutura                             | Descrição                                          |
|---------------------------------------|------------------------------------------------------|
| [**PARAM DE SÃO \_ PAULO**](dexter-param.md) | Especifica uma propriedade em uma transição ou efeito.      |
| [**VALOR \_ DE**](dexter-value.md) | Especifica o valor de uma propriedade.                   |
| [**SCompFmt0**](scompfmt0.md)        | Contém informações de formato para recompactação inteligente. |
