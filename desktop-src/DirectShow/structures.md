# <a name="directshow-structures"></a>Estruturas do DirectShow

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

Os serviços de edição do DirectShow definem as seguintes estruturas.



| Estrutura                             | Descrição                                          |
|---------------------------------------|------------------------------------------------------|
| [**\_parâmetro Dexter**](dexter-param.md) | Especifica uma propriedade em uma transição ou efeito.      |
| [**valor de DEXTER \_**](dexter-value.md) | Especifica o valor de uma propriedade.                   |
| [**SCompFmt0**](scompfmt0.md)        | Contém informações de formato para a recompactação inteligente. |
