# <a name="ec_dvd_disc_ejected"></a>\_disco DVD do EC \_ \_ ejetado

Sinaliza que um disco DVD foi ejetado.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Zero.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Zero.

</dd> </dl>

## <a name="remarks"></a>Comentários

A reprodução é interrompida automaticamente quando um disco é ejetado. O aplicativo não precisa executar nenhuma ação especial em resposta a esse evento.

Esse evento é gerado em todos os domínios.
