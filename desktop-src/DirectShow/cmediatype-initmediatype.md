# <a name="cmediatypeinitmediatype-method"></a>CMediaType.Inimétodo tMediaType

O `InitMediaType` método inicializa o tipo de mídia.

## <a name="syntax"></a>Sintaxe


```C++
void InitMediaType();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

Esse método Zera a memória do objeto, define a propriedade de tamanho de amostra fixo como **true** e define o tamanho da amostra como 1.
