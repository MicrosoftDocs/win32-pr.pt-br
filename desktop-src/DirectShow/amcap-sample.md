# <a name="amcap-sample"></a>Exemplo de AmCap

## <a name="description"></a>Descrição

Aplicativo de captura de vídeo.

Este aplicativo de exemplo demonstra as seguintes tarefas relacionadas à captura de áudio e vídeo:

-   Capturar para um arquivo
-   Visualização dinâmica
-   Alocação do arquivo de captura
-   Exibição de páginas de propriedades do dispositivo
-   Enumeração de dispositivo
-   Controle de fluxo

O AMCap dá suporte à entrada de fluxo de programa MPEG-2, por exemplo, de sintonizadores de TV analógicos que transmitem conteúdo MPEG-2. Um decodificador MPEG-2 compatível com DirectShow é necessário para decodificar os fluxos.

## <a name="downloading-the-sample"></a>Baixando o exemplo

Para baixar os exemplos do SDK do DirectShow, instale a versão mais recente do [SDK do Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este exemplo é instalado no seguinte caminho: exemplo de \[ *raiz do SDK* \] \\ \\ multimídia \\ DirectShow de \\ captura \\ AMCAP.
