# <a name="cmediatypeformatlength-method"></a>Método CMediaType.FormatLength

O `FormatLength` método recupera o comprimento do bloco de formato.

## <a name="syntax"></a>Sintaxe


```C++
ULONG FormatLength() const;
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Retorna o **membro cbFormat.**
