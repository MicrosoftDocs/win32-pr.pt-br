# <a name="directshow-data-types"></a>DirectShow Tipos de dados

DirectShow fornece as definições de tipo a seguir.



| Tipo de Dados                                 | Descrição                                                                                       |
|-------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**registro de DVD \_**](dvd-register.md)     | Contém um valor para um GPRM (registro geral de parâmetro de DVD) ou SPRM (registro de parâmetro do sistema). |
| [**GPRMARRAY**](gprmarray.md)            | Contém uma matriz de valores GPRM.                                                                 |
| [**OAEVENT**](oaevent.md)                | Define um identificador de eventos compatível com a automação.                                                    |
| [**OAFilterState**](oafilterstate.md)    | Define o estado do gráfico de filtro.                                                            |
| [**OAHWND**](oahwnd.md)                  | Define um identificador de janela compatível com a automação.                                                   |
| [**hora de referência \_**](reference-time.md) | Define o tipo de dados para o tempo de referência.                                                         |
| [**REFTIME**](reftime.md)                | Define o tipo de dados usado para expressar tempo em segundos.                                            |
| [**SPRMARRAY**](sprmarray.md)            | Contém uma matriz de valores SPRM.                                                                 |
