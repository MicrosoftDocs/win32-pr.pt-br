# <a name="group-directshow"></a>Grupo (DirectShow)

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O objeto de grupo representa uma composição de nível superior. Cada grupo define um fluxo de áudio ou vídeo no projeto final. Para criar esse objeto, chame o método [**IAMTimeline:: CreateEmptyNode**](iamtimeline-createemptynode.md) .

O objeto de grupo expõe as seguintes interfaces:

-   [**IAMTimelineComp**](iamtimelinecomp.md)
-   [**IAMTimelineGroup**](iamtimelinegroup.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
