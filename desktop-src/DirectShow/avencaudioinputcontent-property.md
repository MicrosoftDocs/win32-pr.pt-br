# <a name="avencaudioinputcontent-property"></a>Propriedade AVEncAudioInputContent

Especifica se o conteúdo de áudio contém música ou voz.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncAudioInputContent**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncAudioInputContent.**](/windows/win32/api/codecapi/ne-codecapi-eavencaudioinputcontent)
