# <a name="avencmuxoutputstreamtype-property"></a>Propriedade AVEncMuxOutputStreamType

Especifica o tipo de fluxo de saída produzido por um multiplexador.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMuxOutputStreamType**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncMuxOutput**](/windows/win32/api/codecapi/ne-codecapi-eavencmuxoutput) .
