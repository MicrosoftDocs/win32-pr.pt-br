# <a name="cenummediatypesreset-method"></a>Método CEnumMediaTypes. Reset

O `Reset` método redefine a sequência de enumeração para o início. Esse método implementa o método [**IEnumMediaTypes:: Reset**](/windows/desktop/api/Strmif/nf-strmif-ienummediatypes-reset) .

## <a name="syntax"></a>Sintaxe


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna S \_ OK.
