# <a name="backcolor-property-directshow"></a>Propriedade BackColor (DirectShow)

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `BackColor` propriedade define ou recupera a cor das barras que aparecem ao contrário das bordas do retângulo de vídeo quando a taxa de proporção do vídeo nativo não é a mesma da área de exibição do objeto.

``` syntax
[ iBackColor = ] MSWebDVD.BackColor
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor inteiro que representa os valores RGB da cor de fundo.

## <a name="remarks"></a>Comentários

Essa propriedade é de leitura/gravação com um valor padrão de off-Black (0x100010).
