# <a name="wavdest-filter-sample"></a>Exemplo de filtro WavDest

## <a name="description"></a>Descrição

O filtro WavDes grava um fluxo de áudio em um arquivo WAV. Ele usa um fluxo de áudio único como entrada e seu PIN de saída deve estar conectado ao filtro do [gravador de arquivo](file-writer-filter.md) . Este filtro de exemplo é baseado na classe [**CTransformFilter**](ctransformfilter.md) .

## <a name="downloading-the-sample"></a>Baixando o exemplo

para baixar os exemplos do SDK do DirectShow, instale a versão mais recente do [SDK do Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

este exemplo é instalado no seguinte caminho: exemplos de *\[ raiz \] do SDK* \\ \\ multimídia \\ DirectShow \\ filtros \\ WavDest.
