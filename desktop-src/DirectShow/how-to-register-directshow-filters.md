Este artigo descreve como fazer o registro automático de um filtro do Microsoft DirectShow. Ele contém as seções a seguir:

-   [Layout das chaves do registro](layout-of-the-registry-keys.md)
-   [Declarando informações de filtro](declaring-filter-information.md)
-   [Declarando o modelo de fábrica](declaring-the-factory-template.md)
-   [Implementando DllRegisterServer](implementing-dllregisterserver.md)
-   [Diretrizes para o registro de filtros](guidelines-for-registering-filters.md)

Este artigo não descreve como criar uma DLL para um filtro do DirectShow. Para obter informações sobre como criar DLLs, consulte [como criar uma DLL de filtro do DirectShow](how-to-create-a-dll.md).
