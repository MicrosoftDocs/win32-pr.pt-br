# <a name="ec_dvd_beginnavigationcommands"></a>\_BeginNavigationCommands de DVD do EC \_

Enviado quando um conjunto de comandos de navegação de DVD está sendo iniciado.

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Um valor da enumeração [**\_ NavCmdType de DVD**](/windows/win32/api/strmif/ne-strmif-dvd_navcmdtype) .

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Zero.

</dd> </dl>

## <a name="remarks"></a>Comentários

Esse evento é desabilitado por padrão. Para habilitar esse evento, chame [**IDvdControl2:: SetOption**](/windows/desktop/api/Strmif/nf-strmif-idvdcontrol2-setoption) e defina a opção **DVD \_ EnableLoggingEvents** como **true**.
