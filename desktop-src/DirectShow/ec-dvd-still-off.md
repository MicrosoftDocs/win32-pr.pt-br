# <a name="ec_dvd_still_off"></a>o \_ DVD do EC \_ ainda está \_ desativado

Sinaliza o final de qualquer ainda (PGC, Cell ou VOBU).

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Zero.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Zero.

</dd> </dl>

## <a name="remarks"></a>Comentários

Esse evento indica que qualquer um atualmente ativo ainda foi lançado.

Esse evento é gerado em todos os domínios.
