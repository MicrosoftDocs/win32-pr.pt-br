# <a name="defaultaudiolanguage-property"></a>Propriedade DefaultAudioLanguage

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `DefaultAudioLanguage` propriedade recupera o idioma de áudio padrão.

``` syntax
[ iLang = ] MSWebDVD.DefaultAudioLanguage
```

## <a name="return-value"></a>Valor Retornado

Retorna um valor LCID que contém a ID de idioma principal para o idioma de áudio padrão.

## <a name="remarks"></a>Comentários

Esta propriedade é somente leitura sem valor padrão.
