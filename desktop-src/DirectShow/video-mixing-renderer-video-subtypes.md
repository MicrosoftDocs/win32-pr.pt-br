# <a name="video-mixing-renderer-video-subtypes"></a>Subtipos de vídeo de renderização de combinação de vídeo

Os subtipos a seguir são definidos para uso com o filtro VMR (Video Mixing Renderer).

## <a name="vmr-7-subtypes"></a>Subtipos VMR-7



| Subtype                              | Descrição                                      |
|--------------------------------------|--------------------------------------------------|
| MEDIASUBTYPE \_ RGB32 \_ D3D \_ DX7 \_ RT    | Destino de renderização RGB de 32 bits.                        |
| MEDIASUBTYPE \_ RGB16 \_ D3D \_ DX7 \_ RT    | Destino de renderização RGB de 16 bits.                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ DX7 \_ RT   | Destino de renderização ARGB de 32 bits.                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ DX7 \_ RT | Destino de renderização ARGB4444. Para elementos gráficos de subpicture. |
| MEDIASUBTYPE \_ ARGB1555 \_ D3D \_ DX7 \_ RT | Destino de renderização ARGB1555. Para elementos gráficos de subpicture. |



 

## <a name="vmr-9-subtypes"></a>Subtipos VMR-9



| Subtype                              | Descrição                                      |
|--------------------------------------|--------------------------------------------------|
| MEDIASUBTYPE \_ RGB32 \_ D3D \_ DX9 \_ RT    | Destino de renderização RGB de 32 bits.                        |
| MEDIASUBTYPE \_ RGB16 \_ D3D \_ DX9 \_ RT    | Destino de renderização RGB de 16 bits.                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ DX9 \_ RT   | Destino de renderização ARGB de 32 bits.                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ DX9 \_ RT | Destino de renderização ARGB4444. Para elementos gráficos de subpicture. |
| MEDIASUBTYPE \_ ARGB1555 \_ D3D \_ DX9 \_ RT | Destino de renderização ARGB1555. Para elementos gráficos de subpicture. |
