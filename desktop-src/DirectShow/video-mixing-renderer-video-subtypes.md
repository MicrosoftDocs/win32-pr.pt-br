# <a name="video-mixing-renderer-video-subtypes"></a>Subtipos de vídeo do processador de mixagem de vídeo

Os seguintes subtipos são definidos para uso com o filtro de processador de mixagem de vídeo (VMR).

## <a name="vmr-7-subtypes"></a>Subtipos do VMR-7



| Subtype                              | Descrição                                      |
|--------------------------------------|--------------------------------------------------|
| MEDIASUBTYPE \_ RGB32 \_ D3D \_ DX7 \_ RT    | destino de renderização RGB de 32 bits.                        |
| MEDIASUBTYPE \_ RGB16 \_ D3D \_ DX7 \_ RT    | destino de renderização RGB de 16 bits.                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ DX7 \_ RT   | destino de renderização ARGB de 32 bits.                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ DX7 \_ RT | Destino de renderização ARGB4444. Para gráficos de subimagem. |
| MEDIASUBTYPE \_ ARGB1555 \_ D3D \_ DX7 \_ RT | Destino de renderização ARGB1555. Para gráficos de subimagem. |



 

## <a name="vmr-9-subtypes"></a>Subtipos do VMR-9



| Subtype                              | Descrição                                      |
|--------------------------------------|--------------------------------------------------|
| MEDIASUBTYPE \_ RGB32 \_ D3D \_ DX9 \_ RT    | destino de renderização RGB de 32 bits.                        |
| MEDIASUBTYPE \_ RGB16 \_ D3D \_ DX9 \_ RT    | destino de renderização RGB de 16 bits.                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ DX9 \_ RT   | destino de renderização ARGB de 32 bits.                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ DX9 \_ RT | Destino de renderização ARGB4444. Para gráficos de subimagem. |
| MEDIASUBTYPE \_ ARGB1555 \_ D3D \_ DX9 \_ RT | Destino de renderização ARGB1555. Para gráficos de subimagem. |
