# <a name="avencmpvqscaletype-property"></a>Propriedade AVEncMPVQScaleType

Especifica se a escala de quantizador é linear ou não linear. Essa propriedade se aplica a codificadores de vídeo MPEG.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncMPVQScaleType**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncMPVQScaleType**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvqscaletype) .
