# <a name="avencddlorosurroundmixlvl_x10-property"></a>Propriedade avEncDDLoRoSurroundMixLvl \_ x10

Especifica a mudança de nível que é aplicada aos canais Surround para o downmixing Lo/Ro. Essa propriedade se aplica a codificadores de áudio Dolby Digital.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDLoRoSurroundMixLvl \_ x10**

## <a name="property-value"></a>Valor da propriedade

O valor é especificado como decibéis (dB) x 10.
