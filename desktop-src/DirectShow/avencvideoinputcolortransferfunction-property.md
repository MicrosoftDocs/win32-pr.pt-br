# <a name="avencvideoinputcolortransferfunction-property"></a>Propriedade AVEncVideoInputColorTransferFunction

Especifica a função de conversão de RGB para R'G'B ' para vídeo de entrada.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoInputColorTransferFunction**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncVideoColorTransferFunction**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolortransferfunction) .
