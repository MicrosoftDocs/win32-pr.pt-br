Não há suporte.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT SetSourceConnectCallback(
   IGrfCache *pCallback
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*pCallback* 
</dt> <dd>

Reservado.

</dd> </dl>
