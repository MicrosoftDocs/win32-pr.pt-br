# <a name="cutsonly-attribute"></a>Atributo cutsonly

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `cutsonly` atributo especifica se uma transição é renderizada como um recorte. Se o valor for **true**, a transição será uma recorte instantânea. Caso contrário, a transição é renderizada normalmente. O valor padrão é **false**.

## <a name="possible-values"></a>Valores possíveis

Os valores a seguir são definidos como TRUE: y, Y, t, T, 1. Os valores a seguir são definidos como FALSE: n, N, f, F, 0 (zero).

## <a name="applies-to"></a>Aplica-se A

[**transição**](transition-element.md)
