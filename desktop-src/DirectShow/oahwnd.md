# <a name="oahwnd"></a>OAHWND

O `OAHWND` tipo de dados define um identificador de janela que pode ser usado por clientes de automação OLE, como o Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAHWND;
```
