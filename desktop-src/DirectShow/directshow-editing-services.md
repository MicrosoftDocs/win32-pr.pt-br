# <a name="directshow-editing-services"></a>DirectShow Editando serviços

\[Essa API não tem suporte e pode ser alterada ou não estar disponível no futuro.\]

DirectShow A edição de serviços (DES) é uma API (interface de programação de aplicativo) que simplifica muito as tarefas envolvidas na edição de vídeo. o DES é criado sobre a arquitetura de DirectShow básica. ele abstrai grande parte da complexidade do DirectShow e fornece um conjunto de interfaces projetadas especificamente para manipular projetos de edição de vídeo. como desenvolvedor de aplicativos, você obtém os benefícios de DirectShow dentro de uma estrutura muito mais adequada para a criação de aplicativos de edição de vídeo.
