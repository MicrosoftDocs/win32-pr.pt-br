# <a name="directshow-editing-services"></a>Serviços de edição do DirectShow

\[Essa API não tem suporte e pode ser alterada ou não estar disponível no futuro.\]

Os serviços de edição do DirectShow (DES) são uma API (interface de programação de aplicativo) que simplifica muito as tarefas envolvidas na edição de vídeo. O DES é criado sobre a arquitetura básica do DirectShow. Ele abstrai grande parte da complexidade do DirectShow e fornece um conjunto de interfaces projetadas especificamente para manipular projetos de edição de vídeo. Como desenvolvedor de aplicativos, você obtém os benefícios do DirectShow dentro de uma estrutura muito mais adequada para a criação de aplicativos de edição de vídeo.
