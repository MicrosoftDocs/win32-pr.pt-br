# <a name="creftimemillisecs-method"></a>Método CReftime. milissegundos

O `Millisecs` método converte o tempo de referência em milissegundos.

## <a name="syntax"></a>Sintaxe


```C++
LONG Millisecs();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna o tempo de referência em milissegundos.
