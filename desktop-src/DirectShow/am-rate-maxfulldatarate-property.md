# <a name="am_rate_maxfulldatarate-property"></a>Propriedade de MaxFullDataRate da \_ taxa de am \_

Retorna a taxa máxima na qual um decodificador pode processar dados. O valor dessa propriedade é a velocidade de reprodução máxima do decodificador x 10000. Por exemplo, se a velocidade máxima de encaminhamento for 2,0, o valor dessa propriedade será 20000.

O tipo de dados para essa propriedade **é \_ MaxFullDataRate**, que é um `typedef` para **longo**.

Esta propriedade é somente para leitura.



|                   |                                    |
|-------------------|------------------------------------|
| GUID do Conjunto de Propriedades | \_KSPROPSETID \_ TSRateChange      |
| ID da propriedade       | \_MaxFullDataRate de taxa de am \_          |
| Tipo de Dados         | **Am \_ MaxFullDataRate** (**longo**) |
