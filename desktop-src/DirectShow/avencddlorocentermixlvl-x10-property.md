# <a name="avencddlorocentermixlvl_x10-property"></a>\_Propriedade AVEncDDLoRoCenterMixLvl x

Especifica a mudança de nível que é aplicada ao canal central para downmixing de lo/ro. Essa propriedade se aplica aos codificadores Dolby Digital Audio.

Esta propriedade é de leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncDDLoRoCenterMixLvl \_ x**

## <a name="property-value"></a>Valor da propriedade

O valor é especificado como decibéis (dB) x 10.
