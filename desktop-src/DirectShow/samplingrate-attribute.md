# <a name="samplingrate-attribute"></a>Atributo samplingrate

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `samplingrate` atributo especifica a taxa de amostragem do áudio de saída, em Hz.

## <a name="possible-values"></a>Valores possíveis

O valor deve ser 8000, 11025, 22050, 32000, 44100 ou 48000. O valor padrão é 44100.

## <a name="applies-to"></a>Aplica-se A

[**group**](group-element.md)

## <a name="remarks"></a>Comentários

Defina esse atributo somente se o atributo de **tipo** for `audio` .
