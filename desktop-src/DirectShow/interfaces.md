# <a name="directshow-interfaces"></a>Interfaces do DirectShow

Este tópico contém informações de referência para as interfaces principais do Microsoft® DirectShow®. Para fornecer amplo suporte a vários tipos de hardware, formatos de arquivo e operações de fluxo, o implementa um grande número de interfaces. Alguns são usados por aplicativos; outras são usadas internamente por filtros e outros componentes do DirectShow.

O tópico a seguir contém uma lista alfabética de todas as interfaces do código DirectShow.

-   [Lista alfabética de interfaces do DirectShow](alphabetical-list-of-directshow-interfaces.md)

Qualquer aplicativo específico usará apenas um pequeno subconjunto dessas interfaces. Os tópicos a seguir listam interfaces por categoria funcional. Nem todas as interfaces estão listadas nessas páginas, e algumas interfaces aparecem em mais de uma página.

-   [Interfaces para criação de gráficos de filtro](interfaces-for-building-filter-graphs.md)
-   [Interfaces para controlar um grafo de filtro](interfaces-for-controlling-a-filter-graph.md)
-   [Interfaces de renderização e captura de áudio](audio-capture-and-rendering-interfaces.md)
-   [Interfaces de DVD](dvd-interfaces.md)
-   [Interfaces de captura de vídeo](video-capture-interfaces.md)
-   [Codificação de arquivos e interfaces de decodificação](file-encoding-and-decoding-interfaces.md)
-   [Interfaces para renderização e sobreposição de vídeo](interfaces-for-video-rendering-and-overlay.md)
-   [Interfaces de controle de dispositivo externo](external-device-control-interfaces.md)
