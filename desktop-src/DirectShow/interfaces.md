# <a name="directshow-interfaces"></a>DirectShow interfaces

Este tópico contém informações de referência para as interfaces ® DirectShow® principais da Microsoft. Para fornecer suporte amplo para vários tipos de hardware, formatos de arquivo e operações de fluxo, implementa um grande número de interfaces. Alguns são usados por aplicativos; outros são usados internamente por filtros e outros DirectShow componentes.

O tópico a seguir contém uma lista alfabética de todas as interfaces DirectShow código.

-   [Lista alfabética de DirectShow interfaces](alphabetical-list-of-directshow-interfaces.md)

Qualquer aplicativo específico usará apenas um pequeno subconjunto dessas interfaces. Os tópicos a seguir listam interfaces por categoria funcional. Nem todas as interfaces são listadas nessas páginas e alguma interface aparece em mais de uma página.

-   [Interfaces para criar grafos de filtro](interfaces-for-building-filter-graphs.md)
-   [Interfaces para controlar um filtro Graph](interfaces-for-controlling-a-filter-graph.md)
-   [Interfaces de captura e renderização de áudio](audio-capture-and-rendering-interfaces.md)
-   [DVD Interfaces](dvd-interfaces.md)
-   [Interfaces de Captura de Vídeo](video-capture-interfaces.md)
-   [Interfaces de codificação e decodificação de arquivo](file-encoding-and-decoding-interfaces.md)
-   [Interfaces para renderização e sobreposição de vídeo](interfaces-for-video-rendering-and-overlay.md)
-   [Interfaces de controle de dispositivo externas](external-device-control-interfaces.md)
