# <a name="cbasecontrolwindowput_visible-method"></a>Método visível CBaseControlWindow. put \_

O `put_Visible` método faz com que mostre ou oculte a janela.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT put_Visible(
   long Visible
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Visível* 
</dt> <dd>

Sinalizador booliano de automação (0 significa que a janela está oculta, 1 significa que a janela é mostrada).

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Retorna um valor **HRESULT** .
