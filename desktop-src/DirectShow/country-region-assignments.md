# <a name="countryregion-assignments"></a>Atribuições de país/região

Os códigos de chamada de país padrão ITU (International Telecommunication Union) são passados para o método [**IAMTuner::p UT \_ CountryCode**](/windows/desktop/api/Strmif/nf-strmif-iamtuner-put_countrycode) para selecionar mapeamentos de frequência de televisão padrão.

Por exemplo, passar 20, o código de chamada ITU-T para o Egito, selecionará frequências de televisão padrão para o Egito.

Para obter uma lista de códigos de chamada de país, consulte a recomendação atual do ITU-T de códigos de país atribuídos.
