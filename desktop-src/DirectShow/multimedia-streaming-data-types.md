# <a name="multimedia-streaming-data-types"></a>Tipos de dados de streaming de multimídia

> [!Note]  
> Essa API está preterida. Novos aplicativos não devem usá-lo.

 

As interfaces de streaming multimídia do Microsoft DirectShow usam os seguintes tipos de dados.



| Tipo de dados                           | Descrição                                            |
|-------------------------------------|--------------------------------------------------------|
| [**MSPID**](mspid.md)              | Identifica a finalidade de um fluxo de mídia.              |
| [**tempo de fluxo \_**](stream-time.md) | Define o tipo de dados usado para expressar tempos de referência. |
