# <a name="overview-of-the-timeline-components"></a>Visão geral dos componentes da linha do tempo

\[Essa API não tem suporte e pode ser alterada ou não disponível no futuro.\]

A linha do [tempo DirectShow DES (Serviços](directshow-editing-services.md) de Edição) é um componente COM. Os objetos dentro da linha do tempo também são componentes COM. Este artigo fornece uma visão geral das interfaces expostas pelos vários objetos.

Este artigo contém os seguintes tópicos:

-   [A linha do tempo](the-timeline.md)
-   [Objetos de linha do tempo](timeline-objects.md)
-   [Subobjetos](subobjects.md)

Para obter mais informações sobre linhas do tempo, consulte [O modelo de linha do tempo](the-timeline-model.md). Para ver uma lista de componentes no DES, [consulte DirectShow Objetos de Serviços de Edição](directshow-editing-services-objects.md).
