# <a name="avencvideoinputchromaresolution-property"></a>Propriedade AVEncVideoInputChromaResolution

Especifica a resolução de chroma do vídeo de entrada.

Essa propriedade é leitura/gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoInputChromaResolution**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncVideoChromaResolution.**](/windows/win32/api/codecapi/ne-codecapi-eavencvideochromaresolution)
