# <a name="defaultsubpicturelanguage-property"></a>Propriedade DefaultSubpictureLanguage

> [!Note]  
> esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

A `DefaultSubpictureLanguage` propriedade recupera o idioma padrão da subimagem.

``` syntax
[ iLang = ] MSWebDVD.DefaultSubpictureLanguage
```

## <a name="return-value"></a>Valor Retornado

Returrn um valor LCID que contém a ID de idioma principal para o idioma de áudio padrão.

## <a name="remarks"></a>Comentários

Esta propriedade é somente leitura sem valor padrão.
