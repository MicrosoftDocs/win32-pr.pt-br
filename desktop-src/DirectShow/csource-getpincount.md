# <a name="csourcegetpincount-method"></a>Método CSource. GetPinCount

O `GetPinCount` método recupera o número de Pins no filtro. Esse método implementa o método puramente virtual [**CBaseFilter:: GetPinCount**](cbasefilter-getpincount.md) .

## <a name="syntax"></a>Sintaxe


```C++
int GetPinCount();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Retorna o número de Pins neste filtro.
