# <a name="oafilterstate"></a>OAFilterState

O `OAFilterState` tipo de dados define o estado do grafo de filtro. O valor é um membro da enumeração [**de \_ estado do filtro**](/windows/win32/api/strmif/ne-strmif-filter_state) . Esse tipo de dados é definido para clientes de automação OLE, como o Microsoft Visual Basic 6,0.


```C++
typedef long OAFilterState;
```
