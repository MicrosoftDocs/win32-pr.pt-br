# <a name="userdata-attribute"></a>Atributo UserData

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O `userdata` atributo especifica dados persistentes definidos pelo aplicativo para um objeto.

## <a name="possible-values"></a>Valores possíveis

O valor deve ser um número de hexedecimal com um número par de dígitos. Os valores de a – F devem estar em letras maiúsculas. Não use um prefixo como 0x. Exemplo: 123ABC.

## <a name="applies-to"></a>Aplica-se A

[**clip**](clip-element.md), [**composto**](composite-element.md), [**efeito**](effect-element.md), [**grupo**](group-element.md), [**linha do tempo**](timeline-element.md), [**transição**](transition-element.md)
