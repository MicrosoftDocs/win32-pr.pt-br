# <a name="loading-and-previewing-a-project"></a>Carregando e visualizando um Project

\[Essa API não tem suporte e pode ser alterada ou não estar disponível no futuro.\]

este tópico descreve como carregar e visualizar um arquivo de projeto com os [serviços de edição de DirectShow](directshow-editing-services.md) (DES). Ele apresenta um aplicativo de exemplo que carrega um arquivo de projeto XML em uma linha do tempo vazia e renderiza a linha do tempo para visualização. Você pode modificar o aplicativo de exemplo para trabalhar com qualquer arquivo de projeto XML de exemplo que acompanha esse SDK.

Este artigo contém os tópicos a seguir.

-   [carregando um arquivo de Project](loading-a-project-file.md)
-   [Visualizando o Project](previewing-the-project.md)
-   [visualizando um Project: código de exemplo](previewing-a-project--example-code.md)
