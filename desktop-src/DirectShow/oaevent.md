# <a name="oaevent"></a>OAEVENT

O tipo de dados **OAEVENT** define um identificador de evento que pode ser usado por clientes de automação OLE, como o Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAEVENT;
```
