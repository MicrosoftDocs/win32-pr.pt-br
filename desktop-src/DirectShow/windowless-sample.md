# <a name="windowless-sample"></a>Exemplo sem janela

## <a name="description"></a>Descrição

Este exemplo demonstra o modo sem janela no filtro do processador de mixagem de vídeo 9 (VMR-9).

Você pode capturar a imagem da janela de vídeo clicando com o botão direito do mouse na imagem ou selecionando **capturar quadro atual** no menu **imagem** .

## <a name="downloading-the-sample"></a>Baixando o exemplo

para baixar os exemplos do SDK do DirectShow, instale a versão mais recente do [SDK do Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx).
