# <a name="avenccommonpassstart-property"></a>Propriedade AVEncCommonPassStart

Inicia a primeira passagem de codificação.

Essa propriedade é somente gravação.

## <a name="data-type"></a>Tipo de dados

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncCommonPassStart**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é a passagem de codificação para iniciar. Para obter o intervalo de valores com suporte, consulte a propriedade [**AVEncCommonMultipassMode**](avenccommonmultipassmode-property.md) do codificador.
