# <a name="cbaserendererisendofstream-method"></a>Método CBaseRenderer. IsEndOfStream

O `IsEndOfStream` método consulta se a notificação de fim de fluxo foi recebida.

## <a name="syntax"></a>Sintaxe


```C++
BOOL IsEndOfStream();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Retorna o sinalizador [**\_ bEOS CBaseRenderer:: m**](cbaserenderer-m-beos.md) .
