# <a name="media-locator"></a>Localizador de mídia

> [!Note]  
> \[Preterido. Essa API pode ser removida de versões futuras do Windows.\]

 

O objeto do localizador de mídia localiza os arquivos associados aos objetos de origem. Use esse objeto para validar as fontes na linha do tempo. Para criar esse objeto, chame **CoCreateInstance**. O identificador de classe é CLSID \_ MediaLocator.

O objeto de localizador de mídia expõe a seguinte interface:

-   [**IMediaLocator**](imedialocator.md)
