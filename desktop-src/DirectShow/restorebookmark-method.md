# <a name="restorebookmark-method"></a>Método RestoreBookmark

> [!Note]  
> Esse componente está disponível para uso nos sistemas operacionais Microsoft Windows 2000, Windows XP e Windows Server 2003. Ele poderá ser alterado ou ficar indisponível em versões subsequentes.

 

O `RestoreBookmark` método move o objeto **MSWebDVD** para o ponto no disco conforme especificado no indicador atual, com todas as configurações de áudio, vídeo e subimagem restauradas.

``` syntax
MSWebDVD.RestoreBookmark()
```

## <a name="return-value"></a>Valor Retornado

Sem valor de retorno.
