# <a name="dvd-graph-builder"></a>Construtor de gráficos de DVD

O construtor de gráficos de DVD é um objeto auxiliar para a criação de grafos de reprodução de DVD. Os aplicativos podem usar esse componente para criar gráficos de filtro para reprodução e navegação em DVD. Crie esse objeto chamando **CoCreateInstance**.



|                  |                                              |
|------------------|----------------------------------------------|
| Identificador de classe | \_DVDGRAPHBUILDER CLSID                       |
| Interfaces       | [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
