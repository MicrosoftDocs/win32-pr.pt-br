# <a name="dvd-graph-builder"></a>construtor de Graph de DVD

o construtor de Graph de dvd é um objeto auxiliar para a criação de grafos de reprodução de DVD. Os aplicativos podem usar esse componente para criar gráficos de filtro para reprodução e navegação em DVD. Crie esse objeto chamando **CoCreateInstance**.



| Rótulo | Valor |
|------------------|----------------------------------------------|
| Identificador de classe | \_DVDGRAPHBUILDER CLSID                       |
| Interfaces       | [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
