# <a name="dvd-graph-builder"></a>DVD Graph Builder

O DVD Graph Builder é um objeto auxiliar para a criação de grafos de reprodução de DVD. Os aplicativos podem usar esse componente para criar grafos de filtro para reprodução e navegação de DVD. Crie esse objeto chamando **CoCreateInstance**.



| Rótulo | Valor |
|------------------|----------------------------------------------|
| Identificador de Classe | CLSID \_ DvdGraphBuilder                       |
| Interfaces       | [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
