# <a name="cbaselistremoveall-method"></a>Método CBaseList. RemoveAll

O `RemoveAll` método remove todos os nós da lista.

## <a name="syntax"></a>Sintaxe


```C++
void RemoveAll();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

É responsabilidade do chamador excluir os itens contidos em todos os nós.
