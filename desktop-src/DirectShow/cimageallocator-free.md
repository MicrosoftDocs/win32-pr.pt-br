# <a name="cimageallocatorfree-method"></a>Método CImageAllocator. Free

O `Free` método libera toda a memória do buffer. Esse método implementa o método puramente virtual [**CBaseAllocator:: Free**](cbaseallocator-free.md) .

## <a name="syntax"></a>Sintaxe


```C++
void Free();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
