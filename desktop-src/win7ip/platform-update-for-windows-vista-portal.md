# <a name="platform-update-for-windows-vista"></a>Atualização da plataforma para Windows Vista

## <a name="purpose"></a>Finalidade

A atualização da plataforma para o Windows Vista e a atualização da plataforma para Windows Server 2008 são atualizações do sistema operacional do usuário final que dão suporte ao uso de tecnologias selecionadas do Windows 7 em versões anteriores do sistema operacional Windows. As atualizações incluem um conjunto de bibliotecas de tempo de execução que permitem que os desenvolvedores de aplicativos direcionem versões atuais, o Windows 7 e o Windows Server 2008 R2, bem como versões anteriores, o Windows Vista e o Windows Server 2008.

## <a name="developer-audience"></a>Público de desenvolvedores

A API com suporte da atualização da plataforma para Windows Vista e a atualização da plataforma para Windows Server 2008 foram projetadas para uso por desenvolvedores experientes de C/C++.

Habilidades recomendadas:

-   Programação COM
-   Programação da API do Windows

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Para o Windows Vista, os usuários finais obtêm a atualização da plataforma para o Windows Vista da Windows Update. Para o Windows Server 2008, os usuários finais obtêm a atualização de plataforma para o Windows Server 2008 do Windows Update.
