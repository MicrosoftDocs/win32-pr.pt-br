# <a name="platform-update-for-windows-vista"></a>Atualização de plataforma para Windows Vista

## <a name="purpose"></a>Finalidade

A Atualização de Plataforma para Windows Vista e a Atualização da Plataforma para o Windows Server 2008 são atualizações do sistema operacional do usuário final que suportam o uso de tecnologias do Windows 7 selecionadas em versões anteriores do sistema operacional Windows. As atualizações incluem um conjunto de bibliotecas de runtime que permitem aos desenvolvedores de aplicativos direcionar versões atuais, Windows 7 e Windows Server 2008 R2, bem como versões anteriores, Windows Vista e Windows Server 2008.

## <a name="developer-audience"></a>Público de desenvolvedores

A API com suporte da Atualização de Plataforma para Windows Vista e a Atualização da Plataforma para Windows Server 2008 foi projetada para uso por desenvolvedores experientes em C/C++.

Proficiências recomendadas:

-   Programação COM
-   Windows Programação de API

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Por Windows Vista, os usuários finais obtém a Atualização da Plataforma para Windows Vista Windows Atualização. Para Windows Server 2008, os usuários finais obtém a Atualização da Plataforma para Windows Server 2008 Windows Atualização.
