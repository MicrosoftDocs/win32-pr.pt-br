# <a name="regions-windows-gdi"></a>Regiões (Windows GDI)

Uma *região* é um retângulo, polígono ou elipse (ou uma combinação de duas ou mais dessas formas) que pode ser preenchida, pintada, invertida, emoldurada e usada para executar testes de clique (testando o local do cursor).
