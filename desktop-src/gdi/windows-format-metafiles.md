# <a name="windows-format-metafiles"></a>Windows-Format metadados

Os metarquivos de formato Windows Microsoft são limitados em seus recursos e raramente devem ser usados. As funções Windows de formato de Windows têm suporte para manter a compatibilidade com compatibilidade com aplicativos que foram escritos para ser executados como aplicativos Windows baseados em 16 bits. Em vez disso, você deve usar as funções de formato aprimorado.

Um *Windows de formato Windows* é usado por aplicativos baseados em Windows de 16 bits. O formato consiste em um header e uma matriz de registros de metarquivo.

Veja a seguir as limitações desse formato:

-   Um Windows de formato de Windows é dependente do aplicativo e do dispositivo. As alterações nos modos de mapeamento do aplicativo ou na resolução do dispositivo afetam a aparência dos metarquivos criados nesse formato.
-   Um metarquivo de formato Windows não contém um header abrangente que descreve as dimensões originais da imagem, a resolução do dispositivo no qual a imagem foi criada, uma descrição de texto opcional ou uma paleta opcional.
-   Um Windows de formato de Windows não dá suporte às novas funções de curva, caminho e transformação. Consulte a lista de funções com suporte na tabela a seguir.
-   Alguns Windows de metarquivo de formato de arquivo não podem ser dimensionados.
-   O contexto do dispositivo de metarquivo associado a um metarquivo de formato Windows não pode ser consultado (ou seja, um aplicativo não pode recuperar dados de resolução de dispositivo, métricas de fonte e assim por diante).

Para converter um metarquivo de formato Windows em um metarquivo de formato avançado, chame a função [**GetMetaFileBitsEx**](/windows/desktop/api/Wingdi/nf-wingdi-getmetafilebitsex) para recuperar os dados do metarquivo de formato Windows e, em seguida, chame a [**função SetWinMetaFileBits**](/windows/desktop/api/Wingdi/nf-wingdi-setwinmetafilebits) para converter esses dados em um metarquivo de formato avançado. Para converter um registro de formato avançado em um Windows de formato de Windows, chame a [**função GetWinMetaFileBits.**](/windows/desktop/api/Wingdi/nf-wingdi-getwinmetafilebits)
