# <a name="polyline-records"></a>Registros de polilinha

Os registros de [polilinha](/windows/desktop/api/Wingdi/nf-wingdi-polyline) são uma série de pontos; as linhas desenhadas entre os pontos descrevem a estrutura de tópicos do caractere. Um registro de polilinha começa com o último ponto no registro anterior (ou, para o primeiro registro na delimitação, o ponto de partida). Cada ponto no registro está no contorno de glifo e pode ser conectado simplesmente usando linhas retas.
