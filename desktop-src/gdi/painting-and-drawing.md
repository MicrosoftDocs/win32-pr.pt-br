# <a name="painting-and-drawing"></a>Pintura e desenho

Esta visão geral descreve como o sistema gerencia a saída na tela e explica o que os aplicativos devem fazer para desenhar em uma janela. Em particular, esta visão geral descreve os *contextos de dispositivo de exibição* (ou, mais simples, *exibe DCS*) e como prepará-los e usá-los. A visão geral não explica como usar funções de interface gráfica de dispositivo (GDI) para gerar a saída, nem explica como imprimir.
