# <a name="rectangle-functions"></a>Funções de retângulo

As funções a seguir são usadas com retângulos.



| Função                               | Descrição                                                                                                                                   |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**CopyRect**](/windows/desktop/api/Winuser/nf-winuser-copyrect)           | Copia as coordenadas de um retângulo para outro.                                                                                           |
| [**EqualRect**](/windows/desktop/api/Winuser/nf-winuser-equalrect)         | Determina se os dois retângulos especificados são iguais comparando as coordenadas de seus cantos superior esquerdo e inferior direito.           |
| [**InflateRect**](/windows/desktop/api/Winuser/nf-winuser-inflaterect)     | Aumenta ou diminui a largura e a altura do retângulo especificado.                                                                       |
| [**IntersectRect**](/windows/desktop/api/Winuser/nf-winuser-intersectrect) | Calcula a interseção de dois retângulos de origem e coloca as coordenadas do retângulo de interseção no retângulo de destino. |
| [**IsRectEmpty**](/windows/desktop/api/Winuser/nf-winuser-isrectempty)     | Determina se o retângulo especificado está vazio.                                                                                          |
| [**OffsetRect**](/windows/desktop/api/Winuser/nf-winuser-offsetrect)       | Move o retângulo especificado pelos deslocamentos especificados.                                                                                       |
| [**Ptinrect**](/windows/desktop/api/Winuser/nf-winuser-ptinrect)           | Determina se o ponto especificado está dentro do retângulo especificado.                                                                   |
| [**Setrect**](/windows/desktop/api/Winuser/nf-winuser-setrect)             | Define as coordenadas do retângulo especificado.                                                                                              |
| [**SetRectEmpty**](/windows/desktop/api/Winuser/nf-winuser-setrectempty)   | Cria um retângulo vazio no qual todas as coordenadas são definidas como zero.                                                                          |
| [**Subtractrect**](/windows/desktop/api/Winuser/nf-winuser-subtractrect)   | Determina as coordenadas de um retângulo formado subtraindo um retângulo de outro.                                                   |
| [**UnionRect**](/windows/desktop/api/Winuser/nf-winuser-unionrect)         | Cria a união de dois retângulos.                                                                                                          |
