# <a name="about-coordinate-spaces-and-transformations"></a>Sobre espaços de coordenadas e transformações

Os espaços de coordenadas e as transformações são usados pelos seguintes tipos de aplicativos:

-   Aplicativos de publicação de área de trabalho (para as partes "zoom" de uma página ou para exibir páginas adjacentes em uma janela).
-   Aplicativos de projeto auxiliados por computador (CAD) (para girar objetos, dimensionar desenhos ou criar exibições de perspectiva).
-   Aplicativos de planilha (para mover e dimensionar grafos).
