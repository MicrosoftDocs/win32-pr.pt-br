# <a name="filled-shapes"></a>Formas preenchidas

*Formas preenchidas* são formas geométricas que são descritas usando a caneta atual e preenchidas usando o pincel atual. Há cinco formas preenchidas:

-   Ellipse
-   Chord
-   Pizza
-   Polygon
-   Retângulo

Esta visão geral descreve as formas preenchidas.
