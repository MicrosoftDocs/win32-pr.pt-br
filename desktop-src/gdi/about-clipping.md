# <a name="about-clipping"></a>Sobre recorte

O recorte é usado por aplicativos de várias maneiras. Aplicativos de planilha e processamento de texto cortam a entrada do teclado para impedir que ele apareça nas margens de uma página ou planilha. Imagens CAD (design auxiliada por computador) e de desenho de aplicativos do clipe para impedir que ele substitua as bordas de um desenho ou imagem.

Uma *região de recorte* é uma região com bordas que são linhas retas ou curvas. Um *caminho de clipe* é uma região com bordas que são linhas retas, curvas Bézier ou combinações de ambos. Para obter mais informações sobre regiões, consulte [regiões](regions.md). Para obter mais informações sobre caminhos, consulte [caminhos](paths.md).

-   [Caminhos de clipes](clip-paths.md)
