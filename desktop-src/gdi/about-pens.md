# <a name="about-pens"></a>Sobre as canetas

Há dois tipos de caneta: superficial e geométrica. Uma *caneta superficial* é usada com aplicativos que exigem linhas de largura fixa e linhas que são rapidamente desenhadas. Um aplicativo CAD, por exemplo, usa uma caneta superficial para gerar linhas ocultas, de seção, de centro e de dimensão que estão entre .015 e .022 polegadas de largura, independentemente do fator de escala. Uma *caneta geométrica* é usada com aplicativos que exigem linhas escalonáveis, linhas com estilos de extremidade ou junção exclusivos e linhas que são mais largas que um único pixel. Um aplicativo de planilha, por exemplo, usa uma caneta geométrica para definir cada uma das barras em um gráfico de barras como uma linha larga.
