# <a name="about-pens"></a>Sobre canetas

Há dois tipos de caneta: superficial e geométrica. Uma *caneta cosmética* é usada com aplicativos que exigem linhas de largura fixa e linhas que são desenhadas rapidamente. Um aplicativo CAD, por exemplo, usa uma caneta superficial para gerar linhas ocultas, de seção, central e de dimensão que têm entre 0,015 e 0,022 polegadas de largura, independentemente do fator de escala. Uma *caneta geométrica é* usada com aplicativos que exigem linhas escalonáveis, linhas com estilos de fim ou junção exclusivos e linhas mais largas do que um único pixel. Um aplicativo de planilha, por exemplo, usa uma caneta geométrica para definir cada uma das barras em um grafo de barras como uma linha larga.
