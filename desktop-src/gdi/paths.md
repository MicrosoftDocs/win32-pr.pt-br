# <a name="paths-windows-gdi"></a>Caminhos (GDI do Windows)

Um *caminho* é uma ou mais figuras (ou formas) que são preenchidas, contornadas ou ambas preenchidas e contornadas. Os caminhos são usados em aplicativos de desenho e pintura. Os aplicativos de projeto auxiliados por computador (CAD) usam caminhos para criar regiões de recorte exclusivas, desenhar contornos de formas irregulares e preencher os interiores de formas irregulares. Uma forma irregular é uma forma composta por linhas retas e curvas Bézier. (As formas regulares incluem elipses, círculos, retângulos e polígonos.)
