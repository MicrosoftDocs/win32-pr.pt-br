# <a name="paths-windows-gdi"></a>Caminhos (Windows GDI)

Um *caminho* é uma ou mais figuras (ou formas) preenchidas, descritas ou preenchidas e descritas. Os caminhos são usados em aplicativos de desenho e pintura. Os aplicativos CAD (design auxiliado por computador) usam caminhos para criar regiões de recorte exclusivas, desenhar contornos de formas irregulares e preencher os interiores de formas irregulares. Uma forma irregular é uma forma composta de curvas Bézier e linhas retas. (As formas regulares incluem re elipses, círculos, retângulos e polígonos.)
