# <a name="pen-end-cap"></a>Ponta da caneta

O atributo de extremidade final especifica a forma de uma caneta geométrica: ARRED, Square ou Flat. A ilustração a seguir mostra linhas paralelas desenhadas usando cada tipo de Cap de extremidade.

![ilustração mostrando três linhas horizontais, cada uma com um tipo diferente de extremidade final](images/cspen-04.png)

As extremidades arredondadas e extremidades quadradas ultrapassam os pontos inicial e final de uma linha desenhada com uma caneta geométrica; o limite de extremidade simples não.
