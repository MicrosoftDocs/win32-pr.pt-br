# <a name="lines-and-curves"></a>Linhas e curvas

Linhas e curvas são usadas para desenhar a saída de gráficos em dispositivos de varredura. Conforme discutido nesta visão geral, uma *linha* é um conjunto de pixels realçados em uma exibição rasterizada (ou um conjunto de pontos em uma página impressa) identificado por dois pontos: um ponto inicial e um ponto final. Uma *curva regular* é um conjunto de pixels realçados em uma exibição de varredura (ou pontos em uma página impressa) que define o perímetro (ou parte do perímetro) de uma seção de cone. Uma *curva irregular* é um conjunto de pixels que define uma curva que não se ajusta ao perímetro de uma seção de cone.
