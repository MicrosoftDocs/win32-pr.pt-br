# <a name="brushes-windows-gdi"></a>Pincéis (Windows GDI)

Um *pincel* é uma ferramenta gráfica que os aplicativos usam para pintar o interior de polígonos, re elipses e caminhos. Os aplicativos de desenho usam pincéis para pintar formas; aplicativos de processamento de palavras usam pincéis para pintar regras; Aplicativos CAD (design auxiliado por computador) usam pincéis para pintar os interiores de exibições entre seções; Os aplicativos de planilha e usam pincéis para pintar as seções de gráficos de pizza e as barras em grafos de barras.
