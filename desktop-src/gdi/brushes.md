# <a name="brushes-windows-gdi"></a>Pincéis (GDI do Windows)

Um *pincel* é uma ferramenta gráfica que os aplicativos usam para pintar o interior de polígonos, elipses e caminhos. Os aplicativos de desenho usam pincéis para pintar formas; aplicativos de processamento de texto usam pincéis para regras de pintura; os aplicativos de projeto auxiliados por computador (CAD) usam pincéis para pintar os interiores de exibições entre seções; e os aplicativos de planilha usam pincéis para pintar as seções de gráficos de pizza e as barras em gráficos de barras.
