# <a name="pens"></a>Canetas

Uma caneta é uma ferramenta gráfica que um aplicativo pode usar para desenhar linhas e curvas. Os aplicativos de desenho usam canetas para desenhar linhas à mão livre, linhas retas e curvas. Os aplicativos CAD (Computer-Aided Design) usam canetas para desenhar linhas visíveis, linhas ocultas, linhas de seção, linhas centrais e assim por diante. Aplicativos de processamento de texto e publicação de área de trabalho usam canetas para desenhar bordas e regras. Os aplicativos de planilha usam as canetas para designar tendências em gráficos e gráficos de barras de tópicos e gráfico de pizza.
