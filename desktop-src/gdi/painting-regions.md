# <a name="painting-regions"></a>Regiões de pintura

Um aplicativo preenche o interior de uma região usando o pincel atualmente selecionado em um contexto de dispositivo pela [**função PaintRgn.**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) Essa função usa os modos de preenchimento de polígono atuais (alternativo e sinuoso).
