# <a name="coordinate-spaces-and-transformations"></a>Coordenar espaços e transformações

Os aplicativos usam espaços de coordenadas e transformações para dimensionar, girar, traduzir, distorcer e refletir a saída dos gráficos. Um *espaço de coordenadas* é um espaço planar que localiza objetos bidimensionais usando dois eixos de referência que são perpendiculares entre si. Há quatro espaços de coordenadas: mundo, página, dispositivo e dispositivo físico (área de cliente, área de trabalho ou página de papel de impressora).

Uma *transformação* é um algoritmo que altera ("transforma") o tamanho, a orientação e a forma dos objetos. As transformações também transferem um objeto gráfico de um espaço de coordenadas para outro. Por fim, o objeto aparece no dispositivo físico, que geralmente é uma tela ou impressora.
