# <a name="device-space-to-physical-device-transformation"></a>Device-Space para Physical-Device transformação

A transformação espaço do dispositivo para dispositivo físico é exclusiva em vários aspectos. Por exemplo, ele é limitado à tradução e é controlado pelo sistema. A única finalidade dessa transformação é garantir que a origem do espaço do dispositivo seja mapeada para o ponto adequado no dispositivo físico. Não há funções para definir essa transformação, nem nenhuma função para recuperar dados relacionados.
