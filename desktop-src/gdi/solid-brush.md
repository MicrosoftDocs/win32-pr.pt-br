# <a name="solid-brush"></a>Pincel sólido

Um *pincel sólido* é um pincel lógico que contém 64 pixels da mesma cor. Um aplicativo pode criar um pincel lógico sólido chamando a função [**CreateSolidBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createsolidbrush) , especificando a cor do pincel necessário. Depois de criar o pincel sólido, o aplicativo pode selecioná-lo em seu contexto de dispositivo e usá-lo para pintar formas preenchidas.
