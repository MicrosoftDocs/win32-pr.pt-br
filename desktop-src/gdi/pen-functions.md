# <a name="pen-functions-windows-gdi"></a>Funções de caneta (GDI do Windows)

As funções a seguir são usadas com canetas.



| Função                                       | Descrição                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Cria uma caneta lógica que tem o estilo, a largura e a cor especificados.                                  |
| [**CreatePenIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Cria uma caneta de superficial lógica que tem o estilo, a largura e a cor especificados em uma estrutura.          |
| [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Cria uma caneta de superficial ou geométrica lógica que tem os atributos de estilo, largura e pincel especificados. |
| [**SetDCPenColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Define a cor da caneta do contexto do dispositivo atual.                                                             |
