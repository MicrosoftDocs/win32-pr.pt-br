# <a name="pen-functions-windows-gdi"></a>Funções de caneta (Windows GDI)

As funções a seguir são usadas com canetas.



| Função                                       | Descrição                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Createpen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Cria uma caneta lógica que tem o estilo, a largura e a cor especificados.                                  |
| [**Createpenindirect**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Cria uma caneta lógica cosmética que tem o estilo, a largura e a cor especificados em uma estrutura.          |
| [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Cria uma caneta lógica superficial ou geométrica que tem os atributos de estilo, largura e pincel especificados. |
| [**SetDCPenColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Define a cor da caneta de contexto do dispositivo atual.                                                             |
