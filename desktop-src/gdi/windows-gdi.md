# <a name="windows-gdi"></a>GDI do Windows

## <a name="purpose"></a>Finalidade

a interface de dispositivo gráfico do Microsoft Windows (GDI) permite que os aplicativos usem gráficos e texto formatado na tela de vídeo e na impressora. os aplicativos baseados em Windows não acessam diretamente o hardware de gráficos. Em vez disso, a GDI interage com drivers de dispositivo em nome dos aplicativos.

## <a name="where-applicable"></a>Quando aplicável

o GDI pode ser usado em todos os aplicativos baseados em Windows.

## <a name="developer-audience"></a>Público de desenvolvedores

Essa API foi projetada para uso por programadores C/C++. é necessário ter familiaridade com a [arquitetura orientada a mensagens](../learnwin32/window-messages.md) Windows.
