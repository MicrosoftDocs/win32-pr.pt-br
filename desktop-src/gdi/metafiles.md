# <a name="metafiles-windows-gdi"></a>Metaarquivos (GDI do Windows)

Um *metarquivo* é uma coleção de estruturas que armazenam uma imagem em um formato independente de dispositivo. A independência do dispositivo é o único recurso que define os metaarquivos, além dos bitmaps. Ao contrário de um bitmap, um metarquivo garante a independência do dispositivo. Há uma desvantagem para os metaarquivos no entanto, eles geralmente são desenhados mais lentamente do que os bitmaps. Portanto, se um aplicativo exigir um desenho rápido e a independência do dispositivo não for um problema, ele deverá usar bitmaps em vez de metaarquivos.

-   [Sobre os metaarquivos](about-metafiles.md)
-   [Usando os metaarquivos](using-metafiles.md)
-   [Referência de metarquivo](metafile-reference.md)

Para obter informações sobre bitmaps, consulte [bitmaps](bitmaps.md).
