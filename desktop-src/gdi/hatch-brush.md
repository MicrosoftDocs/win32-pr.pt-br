# <a name="hatch-brush"></a>Pincel de hachura

Há seis pincéis de hachura lógica predefinidos mantidos pela GDI. Os retângulos a seguir foram pintados usando os seis pincéis de hachura predefinidos.

![ilustração mostrando seis caixas, uma preenchida por cada um dos seis pincéis de hachura](images/hatchbrush.png)

Um aplicativo pode criar um pincel de hachura chamando a função [**CreateHatchBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createhatchbrush) , especificando um dos seis estilos de hachura.
