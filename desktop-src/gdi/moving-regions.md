# <a name="moving-regions"></a>Movendo regiões

Um aplicativo move uma região chamando a função [**OffsetRgn**](/windows/desktop/api/Wingdi/nf-wingdi-offsetrgn) . Os deslocamentos fornecidos ao longo do eixo x e eixo y determinam o número de unidades lógicas a serem movidas para a esquerda ou para a direita e para cima ou para baixo.
