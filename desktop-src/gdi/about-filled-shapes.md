# <a name="about-filled-shapes"></a>Sobre as formas preenchidas

Os aplicativos usam formas preenchidas para uma variedade de tarefas. Os aplicativos de planilha, por exemplo, usam formas preenchidas para construir gráficos e grafos, e os aplicativos de desenho e pintura usam formas preenchidas para permitir que o usuário desenhe figuras e ilustrações.
