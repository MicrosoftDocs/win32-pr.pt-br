# <a name="multiple-monitor-considerations-for-older-programs"></a>Várias considerações de monitor para programas mais antigos

Em geral, os aplicativos mais antigos não precisam de alterações para trabalhar em vários sistemas de monitor. Para a maioria, o sistema parecerá ter apenas uma exibição. As métricas do sistema refletem a exibição principal e os aplicativos em tela inteira aparecem no primário. O sistema lida com a minimização, maximização, menus e caixas de diálogo.

No entanto, alguns programas terão problemas. Alguns que podem ter problemas são aplicativos de controle remoto, aqueles que têm acesso direto ao hardware e aqueles que têm corrigido o GDI ou o driver de vídeo. Nesses casos, o usuário pode ser solicitado a desabilitar qualquer monitor além do monitor primário.
