# <a name="about-rectangles"></a>Sobre retângulos

Os retângulos são usados para a região de recorte de cursor, a parte inválida da área do cliente, uma área para exibir texto formatado ou a área de rolagem. Seus aplicativos também podem usar retângulos para preencher, emoldurar ou inverter uma parte da área do cliente com um determinado pincel e para recuperar as coordenadas de uma janela ou da área do cliente da janela.

-   [Coordenadas de retângulo](rectangle-coordinates.md)
-   [Operações de retângulo](rectangle-operations.md)
