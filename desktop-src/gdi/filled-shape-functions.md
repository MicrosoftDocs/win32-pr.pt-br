# <a name="filled-shape-functions"></a>Funções de forma preenchida

As funções a seguir são usadas com formas preenchidas.



| Função                           | Descrição                                                     |
|------------------------------------|-----------------------------------------------------------------|
| [**Chord**](/windows/desktop/api/Wingdi/nf-wingdi-chord)             | Desenha uma área limitada por uma elipse e um segmento de linha.         |
| [**Ellipse**](/windows/desktop/api/Wingdi/nf-wingdi-ellipse)         | Desenha uma elipse.                                               |
| [**FillRect**](/windows/desktop/api/Winuser/nf-winuser-fillrect)       | Preenche um retângulo usando um pincel.                                |
| [**FrameRect**](/windows/desktop/api/Winuser/nf-winuser-framerect)     | Desenha uma borda em torno de um retângulo usando um pincel.                |
| [**InvertRect**](/windows/desktop/api/Winuser/nf-winuser-invertrect)   | Inverte os valores de cor dos pixels em um retângulo.          |
| [**Pizza**](/windows/desktop/api/Wingdi/nf-wingdi-pie)                 | Desenha uma fatia com formato de pizza limitada por uma elipse e duas radials. |
| [**Polígono**](/windows/desktop/api/Wingdi/nf-wingdi-polygon)         | Desenha um polígono.                                                |
| [**Polipolygon**](/windows/desktop/api/Wingdi/nf-wingdi-polypolygon) | Desenha uma série de polígonos fechados.                              |
| [**Nele**](/windows/desktop/api/Wingdi/nf-wingdi-rectangle)     | Desenha um retângulo.                                              |
| [**RoundRect**](/windows/desktop/api/Wingdi/nf-wingdi-roundrect)     | Desenha um retângulo com cantos arredondados.                         |
