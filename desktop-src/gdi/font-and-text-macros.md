# <a name="font-and-text-macros"></a>Macros de fonte e texto

As macros a seguir são usadas com fontes e texto.



| Macro                            | Descrição                                            |
|----------------------------------|--------------------------------------------------------|
| [**DeleteFont**](/windows/desktop/api/Windowsx/nf-windowsx-deletefont) | Exclui uma fonte especificada.                              |
| [**SelectFont**](/windows/desktop/api/Windowsx/nf-windowsx-selectfont) | Seleciona uma fonte no DC (contexto de dispositivo) especificado. |
