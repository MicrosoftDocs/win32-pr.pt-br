# <a name="device-context-types"></a>Tipos de contexto do dispositivo

Há quatro tipos de DCs: exibição, impressora, memória (ou compatível) e informações. Cada tipo atende a uma finalidade específica, conforme descrito na tabela a seguir.



| Contexto de dispositivo                                 | Descrição                                          |
|------------------------------------------------|------------------------------------------------------|
| [Exibição](display-device-contexts.md)         | Dá suporte a operações de desenho em uma exibição de vídeo.      |
| [Impressora](printer-device-contexts.md)         | Dá suporte a operações de desenho em uma impressora ou plotadora. |
| [Memória](memory-device-contexts.md)           | Dá suporte a operações de desenho em um bitmap.             |
| [Informações](information-device-contexts.md) | Dá suporte à recuperação de dados do dispositivo.               |
