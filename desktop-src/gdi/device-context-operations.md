# <a name="device-context-operations"></a>Operações de contexto do dispositivo

Um aplicativo pode executar as seguintes operações em um contexto de dispositivo:

-   Enumere objetos gráficos existentes.
-   Selecione novos objetos gráficos.
-   Excluir objetos gráficos existentes.
-   Salve os objetos gráficos atuais, seus atributos e os modos gráficos.
-   Restaure os objetos gráficos salvos anteriormente, seus atributos e os modos gráficos.

Além disso, um aplicativo pode usar um contexto de dispositivo para:

-   Determine como a saída de gráficos é traduzida.
-   Cancelar operações demoradas de desenho (iniciadas por um thread em um aplicativo multithread).
-   Redefina uma impressora para um estado específico.
