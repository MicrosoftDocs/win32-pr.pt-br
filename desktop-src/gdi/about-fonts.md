# <a name="about-fonts"></a>Sobre fontes

Para uma pessoa treinada no mecanismo de composição de manuscrito ou familiar com a tipografia padrão, alguns dos termos tipográficos usados nesta visão geral podem ser incomuns. A maioria das diferenças entre a tipografia e o texto padrão no Microsoft Windows refletem alterações na tecnologia. Os termos tipográficos originais eram baseados na composição de metal, enquanto os termos usados aqui, que aparecem como nomes de membro para as estruturas de saída de fonte e texto, refletem uma nova tecnologia baseada na saída da impressora laser e na composição executada em um computador pessoal usando o software de publicação na área de trabalho.
