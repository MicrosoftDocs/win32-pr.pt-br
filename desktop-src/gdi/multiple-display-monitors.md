# <a name="multiple-display-monitors"></a>Vários monitores de exibição

Vários monitores de exibição são um conjunto de recursos relacionados que permitem que os aplicativos façam uso de vários dispositivos de vídeo ao mesmo tempo. Há duas maneiras de usar vários monitores: como uma grande área de trabalho ou uma série de exibições independentes. Quando usado como um grande desktop, os monitores criam mais espaço na tela para aplicativos. Isso é útil sempre que você precisa maximizar o espaço de trabalho na tela e é especialmente útil na publicação de área de trabalho, no desenvolvimento da Web ou na edição de vídeo. Quando usado como exibições independentes, os monitores podem ser usados para jogos, depuração de aplicativos de tela inteira, ensino ou apresentações.
