# <a name="multiple-display-monitors"></a>Vários Monitores de Exibição

Vários Monitores de Exibição é um conjunto de recursos relacionados que permitem que os aplicativos usem vários dispositivos de exibição ao mesmo tempo. Há duas maneiras pelas quais vários monitores podem ser usados: como uma área de trabalho grande ou como um número de exibições independentes. Quando usado como uma área de trabalho grande, os monitores criam mais espaço na tela para aplicativos. Isso é útil sempre que você precisa maximizar seu workspace na tela e é especialmente útil na publicação da área de trabalho, no desenvolvimento na Web ou na edição de vídeo. Quando usados como exibições independentes, os monitores podem ser usados para jogos, depuração de aplicativos de tela inteira, ensino ou apresentação.
