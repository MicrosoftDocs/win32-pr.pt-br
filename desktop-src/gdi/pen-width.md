# <a name="pen-width"></a>Largura da caneta

O atributo width especifica uma largura de caneta superficial nas unidades do dispositivo. No entanto, quando usado com uma caneta geométrica, ele especifica a largura da caneta em unidades lógicas. Para obter mais informações sobre unidades de dispositivo, consulte [espaços de coordenadas e transformações](coordinate-spaces-and-transformations.md).

Atualmente, o sistema limita a largura de canetas superficiais a um único pixel; no entanto, as versões futuras podem remover essa limitação.
