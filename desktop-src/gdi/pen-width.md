# <a name="pen-width"></a>Largura da caneta

O atributo width especifica uma largura de caneta cosmética em unidades de dispositivo. No entanto, quando usada com uma caneta geométrica, ela especifica a largura da caneta em unidades lógicas. Para obter mais informações sobre unidades de dispositivo, consulte [Espaços e transformações de coordenadas.](coordinate-spaces-and-transformations.md)

Atualmente, o sistema limita a largura das canetas cosméticas a um único pixel; no entanto, versões futuras podem remover essa limitação.
