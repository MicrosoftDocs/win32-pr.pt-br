# <a name="pen-join"></a>Junção de caneta

O atributo de junção especifica como as extremidades de duas linhas geométricas são Unidas: chanfrado, com mitra ou arredondado. A ilustração a seguir mostra pares de linhas conectadas desenhadas usando cada tipo de junção.
