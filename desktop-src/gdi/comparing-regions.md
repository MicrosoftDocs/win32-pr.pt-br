# <a name="comparing-regions"></a>Comparando regiões

Um aplicativo compara duas regiões para determinar se elas são idênticas chamando a função [**EqualRgn**](/windows/desktop/api/Wingdi/nf-wingdi-equalrgn) . EqualRgn considera duas regiões idênticas se elas são iguais em tamanho e forma.
