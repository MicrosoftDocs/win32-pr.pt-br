# <a name="logical-brush-types"></a>Tipos de pincel lógicos

Há quatro tipos de pincéis lógicos: [sólido,](solid-brush.md) [estoque,](stock-brush.md) [hatch](hatch-brush.md)e [padrão.](pattern-brush.md) Esses pincéis são mostrados na ilustração a seguir.

![ilustração mostrando quatro círculos, cada um preenchido por um tipo de pincel diferente: sólido, estoque, hatch e padrão](images/csbru-02.png)

Os tipos de estoque e de hatch têm vários pincéis predefinidos.

A [**função CreateBrushIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createbrushindirect) cria um pincel lógico com um estilo, cor e padrão especificados.
