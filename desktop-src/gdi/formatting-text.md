# <a name="formatting-text-windows-gdi"></a>Formatação de texto (Windows GDI)

As funções de formatação podem ser divididas em três categorias: aquelas que recuperam ou configuram os atributos de [formatação](text-formatting-attributes.md) de texto para um contexto de dispositivo, aquelas que recuperam larguras de caractere e aquelas que recuperam alturas e larguras de cadeia de [caracteres.](character-widths.md) [](string-widths-and-heights.md)
