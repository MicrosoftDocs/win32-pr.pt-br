# <a name="formatting-text-windows-gdi"></a>Formatando texto (GDI do Windows)

As funções de formatação podem ser divididas em três categorias: as que recuperam ou definem os [atributos de formatação de texto](text-formatting-attributes.md) para um contexto de dispositivo, aquelas que recuperam [larguras de caracteres](character-widths.md)e aquelas que recuperam [larguras e alturas de cadeia de caracteres](string-widths-and-heights.md).
