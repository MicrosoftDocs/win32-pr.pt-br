# <a name="page-space-to-device-space-transformations"></a>Page-Space Device-Space transformações

A transformação espaço em disco para dispositivo-espaço determina o modo de mapeamento para todos os gráficos de saída associados a um determinado DC. Um *modo de mapeamento* é uma transformação de dimensionamento que especifica o tamanho das unidades usadas para operações de desenho. O modo de mapeamento também pode executar a tradução. Em alguns casos, o modo de mapeamento altera a orientação do eixo x e eixo y no espaço do dispositivo. Os tópicos a seguir descrevem os modos de mapeamento.

-   [Modos de mapeamento e traduções](mapping-modes-and-translations.md)
-   [Modos de mapeamento predefinidos](predefined-mapping-modes.md)
-   [Modos de mapeamento definidos pelo aplicativo](application-defined-mapping-modes.md)
