# <a name="font-families"></a>Famílias de fontes

Uma *família de fontes* é um conjunto de fontes com largura de traço comum e características de serif. Há cinco famílias de fontes. Uma sexta família permite que um aplicativo use a fonte padrão. A tabela a seguir descreve as famílias de fontes.



| Família de fontes | Descrição                                                                                                                                   |
|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| Decorativa  | Especifica uma fonte de responsabilidade. Um exemplo é inglês antigo.                                                                                          |
| Dontcare    | Especifica um nome de família genérico. Esse nome é usado quando as informações sobre uma fonte não existem ou não importam. A fonte padrão é usada. |
| Moderna      | Especifica uma fonte monospace com ou sem serifs. As fontes monospace geralmente são modernas; os exemplos incluem Que, Elite e Courier New.         |
| Romano       | Especifica uma fonte proporcional com serifs. Um exemplo é Times New Roman.                                                                     |
| Script      | Especifica uma fonte projetada para se parecer com manuscrito; os exemplos incluem Script e Cursive.                                              |
| Suíço       | Especifica uma fonte proporcional sem serifs. Um exemplo é Arial.                                                                            |



 

Essas famílias de fontes correspondem às constantes encontradas no arquivo Wingdi.h: FF \_ DECORATIVE, FF \_ DONTCARE, FF \_ MODERN, FF \_ ROMAN, FF SCRIPT e \_ \_ FF? Um aplicativo usa essas constantes quando cria uma fonte, seleciona uma fonte ou recupera informações sobre uma fonte.

As fontes em uma família são diferenciadas por tamanho (10 pontos, 24 pontos e assim por diante) e estilo (regular, itálico e assim por diante).
