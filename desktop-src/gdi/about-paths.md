# <a name="about-paths"></a>Sobre caminhos

Um caminho é um dos objetos associados a um contexto de dispositivo (DC). No entanto, ao contrário dos objetos padrão (a caneta, o pincel e a fonte) que fazem parte de qualquer novo DC, não há nenhum caminho padrão. Para obter mais informações sobre DCs, consulte [contextos de dispositivo](device-contexts.md).
