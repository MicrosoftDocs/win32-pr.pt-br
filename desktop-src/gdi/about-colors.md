# <a name="about-colors"></a>Sobre cores

A cor pode ser usada para comunicar ideias, mostrar relações entre itens e melhorar o recurso e a qualidade da saída. Windows permite que os aplicativos descubram os recursos de cores de determinados dispositivos e escolham entre as cores disponíveis aquelas que melhor se adequam às suas necessidades.

Esta visão geral fornece informações sobre os seguintes tópicos:

-   [Noções básicas de cor](color-basics.md)
-   [Paletas de cores](color-palettes.md)

Embora não seja descrito nesta visão geral, a correspondência de cores da imagem é um recurso importante do gerenciamento de cores que ajuda a garantir que as imagens de cores tenham a mesma aparência, sejam exibidas na tela ou impressas em papel. Para obter mais informações, consulte [Windows Color System](/previous-versions//dd372446(v=vs.85)).
