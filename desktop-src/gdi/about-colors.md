# <a name="about-colors"></a>Sobre cores

A cor pode ser usada para comunicar ideias, mostrar relações entre itens e melhorar a aparência e a qualidade da saída. O Windows permite que os aplicativos descubram os recursos de cores de determinados dispositivos e escolha entre as cores disponíveis as que melhor atendam às suas necessidades.

Esta visão geral fornece informações sobre os seguintes tópicos:

-   [Noções básicas de cor](color-basics.md)
-   [Paletas de cores](color-palettes.md)

Embora não esteja descrito nesta visão geral, a correspondência de cores da imagem é um recurso importante do gerenciamento de cores que ajuda a garantir que as imagens coloridas pareçam as mesmas, sejam exibidas na tela ou impressas em papel. Para obter mais informações, consulte [sistema de cores do Windows](/previous-versions//dd372446(v=vs.85)).
