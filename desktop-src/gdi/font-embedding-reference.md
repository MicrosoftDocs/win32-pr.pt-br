# <a name="font-embedding-reference"></a>Referência de incorporação de fonte

A biblioteca de serviços de inserção de fontes fornece um mecanismo para agrupar fontes TrueType e OpenType da Microsoft em um documento ou arquivo. Normalmente, um documento que contém fontes inseridas precisa dessas fontes para renderizar o documento em outro computador. A inserção de uma fonte garante que uma fonte especificada em um arquivo estará presente no computador que está recebendo o arquivo. No entanto, algumas fontes não podem ser movidas para outros computadores devido a problemas de direitos autorais que limitam a distribuição.

Os elementos a seguir são usados com serviços de inserção de fontes.

-   [Funções de incorporação de fontes](font-embedding-functions.md)
-   [Estruturas de incorporação de fontes](font-embedding-services-structures.md)
-   [Mensagens de erro de incorporação de fontes](font-embedding-function-error-messages.md)
