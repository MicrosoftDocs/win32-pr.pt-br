# <a name="upload-limits"></a>Limites de carregamento

Para limitar o tamanho dos carregamentos, defina a propriedade de extensão do IIS [**BITSMaximumUploadSize**](bits-iis-extension-properties.md) . No IIS 7, você também pode precisar modificar o atributo **maxAllowedContentLength** . Por padrão, o limite de carregamento do IIS 7 é 30 milhões bytes. Para obter detalhes e informações sobre como alterar o padrão do IIS, consulte [KB942074](https://support.microsoft.com//kb/942074).
