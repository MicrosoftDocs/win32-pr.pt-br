# <a name="qmgr-enumerations"></a>Enumerações QMGR

\[As enumerações do Gerenciador de filas (QMGR) estão disponíveis para uso nos sistemas operacionais especificados na seção de requisitos de um tópico. Eles podem ter sido alterados ou não estarem disponíveis em versões subsequentes. Em vez disso, use as [enumerações de bits](bits-enumerations.md).\]

As [interfaces](qmgr-interfaces.md) do Gerenciador de filas (QMGR) usam a seguinte enumeração.



| Enumeração                               | Descrição                                                                              |
|-------------------------------------------|------------------------------------------------------------------------------------------|
| [**GROUPPROP**](/windows/desktop/api/Qmgr/ne-qmgr-groupprop)<br/> | Define os valores constantes para recuperar e definir valores de propriedade de grupo.<br/> |
