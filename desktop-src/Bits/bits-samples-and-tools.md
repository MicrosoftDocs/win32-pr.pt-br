# <a name="bits-samples-and-tools"></a>Exemplos e ferramentas de BITS

A seção a seguir contém exemplos de C++ passo a passo para Serviço de Transferência Inteligente em Segundo Plano (BITS).



| Seção                                                            | Finalidade                                                                                      |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [Exemplos de aplicativo C++ bits](bits-c---application-examples.md) | Exemplos do C++ que demonstram uma variedade de tarefas que podem ser concluídas usando recursos bits. |



 

Além disso, o BITS inclui os exemplos a seguir Windows SDK. Os exemplos estão localizados em *InstallDirectory* \\ Samples \\ Web. Cada exemplo inclui um Readme.txt que explica como instalar e executar o exemplo. Você pode baixar o Windows SDK do [Microsoft Windows Software Development Kit](https://msdn.microsoft.com/windowsserver/bb980924.aspx). BITS está incluído no SDK principal.

> [!Note]  
> Windows Vista: os exemplos nesta página estão incluídos no [SDK](https://www.microsoft.com/download/details.aspx?id=30998)do Microsoft Windows para Windows Vista . No entanto, o exemplo ClientCert não está incluído.

 



| Amostra       | Finalidade                                                                                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BITS \_ IE     | Mostra como usar as [interfaces](bits-interfaces.md) BITS para baixar arquivos de um servidor.                                                                                                                                             |
| Controles     | Mostra como usar as [interfaces](bits-interfaces.md) BITS para controlar e configurar opções de BITS, incluindo o uso de alguns dos métodos de administração de cache par.<br/> **BITS 4.0:** O exemplo controles foi preterido.<br/> |
| Downloads    | Mostra como usar as [interfaces](bits-interfaces.md) BITS para baixar arquivos de um servidor.                                                                                                                                             |
| HTTPHeaders  | Mostra como usar as interfaces BITS para modificar [cabeçalhos](bits-interfaces.md) HTTP.                                                                                                                                                      |
| PeerCaching  | Mostra como usar as [interfaces](bits-interfaces.md) BITS para habilitar um trabalho a ser baixado de um par.<br/> **BITS 4.0:** O exemplo peerCaching foi preterido.<br/>                                                           |
| Uploads      | Mostra como usar as [interfaces](bits-interfaces.md) BITS para carregar um arquivo.                                                                                                                                                            |
| UploadSample | Mostra como usar as [interfaces](bits-interfaces.md) BITS para carregar um arquivo em um servidor e receber uma resposta.                                                                                                                            |
