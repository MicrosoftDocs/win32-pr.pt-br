# <a name="determining-the-status-of-a-job"></a>Determinando o status de um trabalho

O status de um trabalho refere-se a alterações no estado de um trabalho (por exemplo, o estado do trabalho muda de BG, \_ transferindo o estado do trabalho \_ para o \_ BG \_ \_ estado do trabalho \_ transferido) e o progresso na transferência dos arquivos. Para capturar as alterações no status de um trabalho, use as seguintes opções:

-   [Sondar o status do trabalho](polling-for-the-status-of-the-job.md).
-   [Registre-se para receber a notificação de eventos](registering-a-com-callback.md).
-   [Registre-se para executar um programa](registering-to-execute-a-program.md) com base em um evento.

Para obter informações sobre o progresso de um trabalho, consulte [determinando o progresso de um trabalho](determining-the-progress-of-a-job.md).
