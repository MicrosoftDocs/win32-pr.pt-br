# <a name="bits-adsi-extension"></a>Extensão ADSI do BITS

O servidor BITS fornece as seguintes interfaces de extensão ADSI para habilitar carregamentos de BITS.



| Interface                                                        | Descrição                                                                                                                                    |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup)               | Use para habilitar e desabilitar carregamentos de BITS em um diretório virtual.                                                                                 |
| [**IBITSExtensionSetupFactory**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetupfactory) | Use para recuperar um ponteiro para uma interface [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup) em um programa de instalação que instala o servidor bits. |
