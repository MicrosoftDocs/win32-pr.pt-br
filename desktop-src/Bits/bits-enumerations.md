# <a name="bits-enumerations"></a>Enumerações de BITS

As [interfaces do serviço de transferência inteligente em segundo plano (bits)](bits-interfaces.md) usam as seguintes enumerações.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**BG_AUTH_SCHEME**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_scheme) | Define constantes que especificam o esquema de autenticação a ser usado quando um proxy ou servidor solicita autenticação de usuário. |
| [**BG_AUTH_TARGET**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_target) | Define constantes que especificam se as credenciais são usadas para solicitações de autenticação de usuário do proxy ou do servidor. |
| [**BG_CERT_STORE_LOCATION**](/windows/win32/api/bits2_5/ne-bits2_5-bg_cert_store_location) | Define constantes que especificam o local do repositório de certificados. |
| [**BG_ERROR_CONTEXT**](/windows/win32/api/bits/ne-bits-bg_error_context) | Define constantes que especificam o contexto no qual o erro ocorreu. |
| [**BG_JOB_PRIORITY**](/windows/win32/api/bits/ne-bits-bg_job_priority) | Define constantes que especificam o nível de prioridade de um trabalho.  |
| [**BG_JOB_PROXY_USAGE**](/windows/win32/api/bits/ne-bits-bg_job_proxy_usage) | Define constantes que especificam qual proxy usar para transferências de arquivos. Você pode definir configurações de proxy diferentes para cada trabalho. |
| [**BG_JOB_STATE**](/windows/win32/api/bits/ne-bits-bg_job_state) | Define constantes que especificam os diferentes Estados de um trabalho. |
| [**BG_JOB_TYPE**](/windows/win32/api/bits/ne-bits-bg_job_type) | Define constantes que especificam o tipo de trabalho de transferência, como download. |
| [**BITS_COST_STATE**](./bits-cost-state.md) | Define constantes que especificam o estado de custo do BITS. |
| [**BITS_FILE_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_file_property_id) | Define constantes que especificam valores de ID correspondentes às propriedades do arquivo de cópia em segundo plano. Consulte [IBackgroundCopyFile5:: GetProperty](../delivery_optimization/ibackgroundcopyfile5-getproperty.md) e [SetProperty](../delivery_optimization/ibackgroundcopyfile5-setproperty.md). |
| [**BITS_JOB_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_property_id) | Define constantes que especificam a ID da propriedade para o trabalho BITS. |
| [**BITS_JOB_TRANSFER_POLICY**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_transfer_policy) | Define constantes que especificam valores de ID correspondentes às propriedades do BITS. |
