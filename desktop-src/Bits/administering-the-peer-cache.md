# <a name="administering-the-peer-cache"></a><span data-ttu-id="20d1b-103">Administrando o cache de pares</span><span class="sxs-lookup"><span data-stu-id="20d1b-103">Administering the Peer Cache</span></span>

> [!Note]  
> <span data-ttu-id="20d1b-104">A partir do Windows 7, o modelo de cache de pares Serviço de Transferência Inteligente em Segundo Plano (BITS) 3,0 é preterido.</span><span class="sxs-lookup"><span data-stu-id="20d1b-104">Starting with Windows 7, the Background Intelligent Transfer Service (BITS) 3.0 peer caching model is deprecated.</span></span> <span data-ttu-id="20d1b-105">Se o BITS 4,0 estiver instalado, o modelo de cache de pares de BITS 3,0 estará indisponível.</span><span class="sxs-lookup"><span data-stu-id="20d1b-105">If BITS 4.0 is installed, the BITS 3.0 peer caching model is unavailable.</span></span>

 

<span data-ttu-id="20d1b-106">Para melhorar o desempenho do download, o BITS permite que você baixe o conteúdo de computadores pares.</span><span class="sxs-lookup"><span data-stu-id="20d1b-106">To improve download performance, BITS lets you download content from peer computers.</span></span> <span data-ttu-id="20d1b-107">Para habilitar esse recurso, o administrador deve habilitar a configuração da política de grupo EnablePeerCaching.</span><span class="sxs-lookup"><span data-stu-id="20d1b-107">To enable this feature, the administrator must enable the EnablePeerCaching group policy setting.</span></span> <span data-ttu-id="20d1b-108">Se habilitada, o par pode baixar conteúdo de pares e fornecer conteúdo a pares.</span><span class="sxs-lookup"><span data-stu-id="20d1b-108">If enabled, the peer can download content from peers and serve content to peers.</span></span> <span data-ttu-id="20d1b-109">O administrador também pode usar as configurações de política DisablePeerCachingClient e DisablePeerCachingServer para evitar o download de conteúdo de um par ou o fornecimento de conteúdo a pares, respectivamente.</span><span class="sxs-lookup"><span data-stu-id="20d1b-109">The administrator can also use the DisablePeerCachingClient and DisablePeerCachingServer policy settings to prevent downloading content from a peer or serving content to peers, respectively.</span></span>

<span data-ttu-id="20d1b-110">Se as configurações de política de grupo não estiverem configuradas, um aplicativo poderá chamar o método [**IBitsPeerCacheAdministration:: SetConfigurationFlags**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-setconfigurationflags) para definir a preferência de cache de sistemas pares para o computador.</span><span class="sxs-lookup"><span data-stu-id="20d1b-110">If the group policy settings are not configured, an application can call the [**IBitsPeerCacheAdministration::SetConfigurationFlags**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-setconfigurationflags) method to set the peer caching preference for the computer.</span></span> <span data-ttu-id="20d1b-111">Observe que essas preferências são substituídas pelas configurações da política de grupo se elas forem definidas mais tarde.</span><span class="sxs-lookup"><span data-stu-id="20d1b-111">Note that these preferences are overridden by the group policy settings if they are set later.</span></span> <span data-ttu-id="20d1b-112">Para determinar se o computador habilita o cache de pares, chame o método [**IBitsPeerCacheAdministration:: GetConfigurationFlags**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-getconfigurationflags) .</span><span class="sxs-lookup"><span data-stu-id="20d1b-112">To determine if the computer enables peer caching, call the [**IBitsPeerCacheAdministration::GetConfigurationFlags**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-getconfigurationflags) method.</span></span>

<span data-ttu-id="20d1b-113">Se o cache de pares estiver habilitado, o BITS só armazenará em cache o conteúdo de um trabalho se o trabalho permitir explicitamente que seu conteúdo seja armazenado em cache.</span><span class="sxs-lookup"><span data-stu-id="20d1b-113">If peer caching is enabled, BITS will only cache the contents of a job if the job explicitly allows its content to be cached.</span></span> <span data-ttu-id="20d1b-114">O BITS também baixará apenas o conteúdo de um par se o trabalho permitir explicitamente.</span><span class="sxs-lookup"><span data-stu-id="20d1b-114">BITS will also only download content from a peer if the job explicitly allows it.</span></span> <span data-ttu-id="20d1b-115">Para habilitar o cache de sistemas pares para um trabalho, chame o método [**IBackgroundCopyJob4:: SetPeerCachingFlags**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibackgroundcopyjob4-setpeercachingflags) .</span><span class="sxs-lookup"><span data-stu-id="20d1b-115">To enable peer caching for a job, call the [**IBackgroundCopyJob4::SetPeerCachingFlags**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibackgroundcopyjob4-setpeercachingflags) method.</span></span>

<span data-ttu-id="20d1b-116">Além de usar Política de Grupo ou a interface [**IBitsPeerCacheAdministration**](/windows/desktop/api/Bits3_0/nn-bits3_0-ibitspeercacheadministration) para habilitar o cache de pares, você também pode usar qualquer um dos métodos para alterar o tamanho do cache padrão e o período de tempo que um arquivo não acessado permanece no cache.</span><span class="sxs-lookup"><span data-stu-id="20d1b-116">In addition to using Group Policy or the [**IBitsPeerCacheAdministration**](/windows/desktop/api/Bits3_0/nn-bits3_0-ibitspeercacheadministration) interface to enable peer caching, you can also use either method to change the default cache size and length of time that a non-accessed file remains in the cache.</span></span> <span data-ttu-id="20d1b-117">Para alterar os padrões usando a interface **IBitsPeerCacheAdministration** , chame os métodos [**SetMaximumCacheSize**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-setmaximumcachesize) e [**SetMaximumContentAge**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-setmaximumcontentage) .</span><span class="sxs-lookup"><span data-stu-id="20d1b-117">To change the defaults using the **IBitsPeerCacheAdministration** interface, call the [**SetMaximumCacheSize**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-setmaximumcachesize) and [**SetMaximumContentAge**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-setmaximumcontentage) methods.</span></span> <span data-ttu-id="20d1b-118">Como esses métodos definem as configurações de preferência, eles são substituídos pelas configurações da política de grupo.</span><span class="sxs-lookup"><span data-stu-id="20d1b-118">Since these methods set the preference settings, they are overridden by the group policy settings.</span></span>

<span data-ttu-id="20d1b-119">Para listar os pares dos quais o BITS tentará baixar o conteúdo, chame o método [**IBitsPeerCacheAdministration:: EnumPeers**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-enumpeers) .</span><span class="sxs-lookup"><span data-stu-id="20d1b-119">To list the peers from whom BITS will try to download content, call the [**IBitsPeerCacheAdministration::EnumPeers**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-enumpeers) method.</span></span>

<span data-ttu-id="20d1b-120">Para listar os arquivos no cache que o BITS vai atender aos pares, chame o método [**IBitsPeerCacheAdministration:: EnumRecords**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-enumrecords) .</span><span class="sxs-lookup"><span data-stu-id="20d1b-120">To list the files in the cache that BITS will serve to peers, call the [**IBitsPeerCacheAdministration::EnumRecords**](/windows/desktop/api/Bits3_0/nf-bits3_0-ibitspeercacheadministration-enumrecords) method.</span></span>

<span data-ttu-id="20d1b-121">Você nunca deve gerenciar o cache de sistemas pares com relação à descoberta de pares ou exclusão de registros de cache.</span><span class="sxs-lookup"><span data-stu-id="20d1b-121">You should never have to manage the peer cache with regards to discovering peers or deleting cache records.</span></span> <span data-ttu-id="20d1b-122">Essa funcionalidade foi incluída na interface [**IBitsPeerCacheAdministration**](/windows/desktop/api/Bits3_0/nn-bits3_0-ibitspeercacheadministration) para fins de integridade.</span><span class="sxs-lookup"><span data-stu-id="20d1b-122">This functionality was included in the [**IBitsPeerCacheAdministration**](/windows/desktop/api/Bits3_0/nn-bits3_0-ibitspeercacheadministration) interface for completeness.</span></span>
