# <a name="bits-c-application-examples"></a>Exemplos de aplicativo C++ bits

Os exemplos Serviço de Transferência Inteligente em Segundo Plano aplicativo bits (BITS) nesta seção são escritos em C++. Eles demonstram uma variedade de tarefas que podem ser concluídas usando recursos bits. Cada aplicativo é separado em uma série de etapas.

A tabela a seguir lista os exemplos do C++ nesta seção.



| Exemplo                                                                                                                                                            | Descrição                                                                                                                                                                                                                                                                                |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Exemplo: Classes comuns](common-classes.md)                                                                                                                      | Este exemplo contém classes comuns que são usadas para os exemplos nesta seção. Essas classes comuns incluem uma classe de retorno de chamada, uma classe de erro genérica e uma classe auxiliar de aquisição de recursos para a função COM [CoInitializeEx.](/windows/win32/api/combaseapi/nf-combaseapi-coinitializeex) |
| [Exemplo: especificando credenciais de autenticação de servidor para um trabalho de transferência de BITS](example-specifying-server-authentication-credentials-for-a-bits-transfer-job-.md) | Este exemplo obtém o esquema de autorização e as credenciais do usuário, cria um trabalho de transferência bits, define as credenciais para o trabalho e monitora o status do trabalho.                                                                                                               |
| [Exemplo: usando a codificação de autenticação SSPI com BITS.](example-using-sspi-authentication-encoding-with-bits.md)                                                 | Este exemplo demonstra o uso da autenticação SSPI (Interface do Provedor de Suporte de Segurança) e bits para obter as credenciais de um usuário, codificar as credenciais e definir as credenciais codificadas em um trabalho de transferência de BITS.                                                                |
| [Exemplo: Adicionando um token auxiliar a um trabalho de transferência de BITS](example-adding-a-helper-token-to-a-bits-transfer-job-.md)                                                 | Este exemplo cria um trabalho de transferência de BITS e adiciona um segundo conjunto de credenciais ao trabalho.                                                                                                                                                                                                  |
