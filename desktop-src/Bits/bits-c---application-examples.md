# <a name="bits-c-application-examples"></a>Exemplos de aplicativo do BITS C++

Os exemplos de aplicativos Serviço de Transferência Inteligente em Segundo Plano (BITS) nesta seção são escritos em C++. Eles demonstram uma variedade de tarefas que podem ser concluídas usando recursos do BITS. Cada aplicativo é separado em uma série de etapas.

A tabela a seguir lista os exemplos de C++ nesta seção.



| Exemplo                                                                                                                                                            | Descrição                                                                                                                                                                                                                                                                                |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Exemplo: classes comuns](common-classes.md)                                                                                                                      | Este exemplo contém classes comuns que são usadas para os exemplos nesta seção. Essas classes comuns incluem uma classe de retorno de chamada, uma classe de erro genérica e uma classe auxiliar de aquisição de recursos para a função de [COINITIALIZEEX](/windows/win32/api/combaseapi/nf-combaseapi-coinitializeex) com. |
| [Exemplo: especificando credenciais de autenticação de servidor para um trabalho de transferência de BITS](example-specifying-server-authentication-credentials-for-a-bits-transfer-job-.md) | Este exemplo obtém o esquema de autorização e as credenciais do usuário, cria um trabalho de transferência de BITS, define as credenciais para o trabalho e monitora o status do trabalho.                                                                                                               |
| [Exemplo: usando a codificação de autenticação SSPI com BITS.](example-using-sspi-authentication-encoding-with-bits.md)                                                 | Este exemplo demonstra o uso da autenticação do SSPI (Security Support Provider interface) e BITS para obter as credenciais de um usuário, codificar as credenciais e definir as credenciais codificadas em um trabalho de transferência de BITS.                                                                |
| [Exemplo: adicionando um token auxiliar a um trabalho de transferência de BITS](example-adding-a-helper-token-to-a-bits-transfer-job-.md)                                                 | Este exemplo cria um trabalho de transferência de BITS e adiciona um segundo conjunto de credenciais ao trabalho.                                                                                                                                                                                                  |
