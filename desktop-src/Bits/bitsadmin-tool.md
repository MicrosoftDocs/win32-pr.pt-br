# <a name="bitsadmin-tool"></a>Ferramenta BITSAdmin

BITSAdmin é uma ferramenta de linha de comando que você pode usar para criar trabalhos de download ou upload e monitorar seu progresso. Para ver uma documentação completa e abrangente da ferramenta e de todos os seus comandos, confira [exemplos de bitsadmin](/windows-server/administration/windows-commands/bitsadmin) e [bitsadmin](/windows-server/administration/windows-commands/bitsadmin-examples) no centro de Windows de PRO.

Por exemplo, para listar todos os trabalhos bits, você pode emitir o comando `bitsadmin /list /verbose` . Aqui está uma captura de tela da saída de exemplo para esse comando.
