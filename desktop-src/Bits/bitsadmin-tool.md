# <a name="bitsadmin-tool"></a>Ferramenta BITSAdmin

BITSAdmin é uma ferramenta de linha de comando que você pode usar para criar tarefas de download ou de upload e monitorar seu progresso. Para obter a documentação completa e abrangente da ferramenta e de todos os seus comandos, consulte [exemplos](/windows-server/administration/windows-commands/bitsadmin-examples) de [Bitsadmin](/windows-server/administration/windows-commands/bitsadmin) e Bitsadmin no centro de profissionais de ti do Windows.

Por exemplo, para listar todos os trabalhos do BITS, você pode emitir o comando `bitsadmin /list /verbose` . Aqui está uma captura de tela de exemplo de saída para esse comando.
