# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Usando a interface do usuário do IIS para configurar o diretório virtual

Para usar a interface do usuário do IIS para  configurar o diretório virtual para uploads de BITS, selecione Propriedades no menu de contexto do diretório virtual. Pressione a **guia Extensão bits** para acessar a página de propriedades bits (a extensão do servidor BITS ISAPI deve ser instalada). Altere os valores de propriedade padrão conforme apropriado e habilita o diretório para uploads; você não pode carregar arquivos até que o diretório seja habilitado. Para obter detalhes sobre como definir os valores de propriedade, consulte a Ajuda online.
