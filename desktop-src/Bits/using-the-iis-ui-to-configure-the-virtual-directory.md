# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Usando a interface do usuário do IIS para configurar o diretório virtual

Para usar a interface do usuário do IIS para configurar o diretório virtual para carregamentos do BITS, selecione **Propriedades** no menu de contexto do diretório virtual. Pressione a guia **extensão do bits** para acessar a página Propriedades do bits (a extensão do servidor bits ISAPI deve estar instalada). Altere os valores de propriedade padrão conforme apropriado e habilite o diretório para carregamentos; Você não pode carregar arquivos até que o diretório esteja habilitado. Para obter detalhes sobre como definir os valores de propriedade, consulte a ajuda online.
