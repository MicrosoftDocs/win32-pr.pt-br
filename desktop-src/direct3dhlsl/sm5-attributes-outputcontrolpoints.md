# <a name="outputcontrolpoints"></a>outputcontrolpoints

Define o número de pontos de controle de saída (por thread) que serão criados no sombreador de chassi.


```
outputcontrolpoints(X)
```



## <a name="remarks"></a>Comentários

Esse será o número de vezes que a função principal será executada.

Esse atributo tem suporte nos seguintes tipos de sombreadores:



| Vértice | Casco | Domínio | Geometria | Pixel | Computação |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
