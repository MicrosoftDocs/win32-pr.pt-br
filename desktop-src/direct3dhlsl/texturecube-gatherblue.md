# <a name="texturecubegatherblue-methods"></a>Métodos TextureCube::GatherBlue

Retorna os componentes azuis dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                             | Descrição                                                                                                                                      |
|:-------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherBlue(S,float,uint)**](tcube-gatherblue-s-float-uint-.md)  | Retorna os componentes azuis dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
