# <a name="texture3dsamplegrad-methods"></a>Métodos Texture3D::SampleGrad

Amostra uma textura usando um gradiente para influenciar a maneira como o local de exemplo é calculado.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                       | Descrição                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad(S,float,float,float,int)**](dx-graphics-hlsl-to-samplegrad.md)                                | Amostra uma textura, usando um gradiente para influenciar a maneira como o local de exemplo é calculado.<br/>                                                                                                             |
| [**SampleGrad(S,float,float,float,int,float)**](t3d-samplegrad-s-float-float-float-int-float-.md)           | Amostra uma textura, usando um gradiente para influenciar a maneira como o local de exemplo é calculado, com um valor opcional para fixar valores lod (nível de detalhes) de exemplo.<br/>                                     |
| [**SampleGrad(S,float,float,float,int, float,uint)**](t3d-samplegrad-s-float-float-float-int-float-uint-.md) | Amostra uma textura, usando um gradiente para influenciar a maneira como o local de exemplo é calculado, com um valor opcional para fixar valores lod (nível de detalhes) de exemplo. Retorna o status sobre a operação.<br/> |
