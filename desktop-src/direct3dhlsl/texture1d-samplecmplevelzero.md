# <a name="texture1dsamplecmplevelzero-methods"></a>Métodos Texture1D::SampleCmpLevelZero

Amostra um [**Texture1D apenas**](sm5-object-texture1d.md) no nível 0 do mipmap e compara o resultado com um valor de comparação.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                               | Descrição                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero(S, float,float,int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)              | Amostra apenas uma textura no nível 0 do mipmap e compara o resultado com um valor de comparação.<br/>                                     |
| [**SampleCmpLevelZero(S, float,float,int,uint)**](t1d-samplecmplevelzero-s-float-float-int-uint-.md) | Amostra apenas uma textura no nível 0 do mipmap e compara o resultado com um valor de comparação. Retorna o status sobre a operação.<br/> |
