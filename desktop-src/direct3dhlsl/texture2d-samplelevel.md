# <a name="texture2dsamplelevel-methods"></a>Métodos Texture2D::SampleLevel

Amostra um [**Texture2D**](sm5-object-texture2d.md) no nível de mipmap especificado.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                             | Descrição                                                                                        |
|:-----------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)          | Amostra uma textura no nível de mipmap especificado.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](samplelevel-s-float-float-int-uint-.md) | Amostra uma textura no nível de mipmap especificado e retorna o status sobre a operação.<br/> |
