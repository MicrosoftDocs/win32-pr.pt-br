# <a name="texturecubearraygatheralpha-methods"></a>Métodos TextureCubeArray::GatherAlpha

Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                    | Descrição                                                                                                                                      |
|:--------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherAlpha(S, float,uint)**](tcubearray-gatheralpha-s-float-uint-.md) | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
