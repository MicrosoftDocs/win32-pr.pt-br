# <a name="sv_stencilref"></a>\_STENCILREF VA

\_STENCILREF VA representa o valor de referência do estêncil do sombreador de pixel atual.

## <a name="type"></a>Tipo



| Tipo     |
|------|
| uint |



 

## <a name="remarks"></a>Comentários
