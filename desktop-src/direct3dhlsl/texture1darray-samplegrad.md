# <a name="texture1darraysamplegrad-methods"></a>Métodos Texture1DArray:: SampleGrad

Amostra uma textura usando um gradiente para influenciar a maneira como o local de exemplo é calculado.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                            | Descrição                                                                                                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad (S, float, float, float, int)**](dx-graphics-hlsl-to-samplegrad.md)                                     | Exemplifica uma textura, usando um gradiente para influenciar a maneira como o local de exemplo é calculado.<br/>                                                                                                             |
| [**SampleGrad (S, float, float, float, int, float)**](t1darray-samplegrad-s-float-float-float-int-float-.md)           | Exemplifica uma textura, usando um gradiente para influenciar a maneira como o local de exemplo é calculado, com um valor opcional para fixe os valores de nível de detalhe (LOD) de exemplo para.<br/>                                     |
| [**SampleGrad (S, float, float, float, int, float, uint)**](t1darray-samplegrad-s-float-float-float-int-float-uint-.md) | Exemplifica uma textura, usando um gradiente para influenciar a maneira como o local de exemplo é calculado, com um valor opcional para fixe os valores de nível de detalhe (LOD) de exemplo para. Retorna o status sobre a operação.<br/> |
