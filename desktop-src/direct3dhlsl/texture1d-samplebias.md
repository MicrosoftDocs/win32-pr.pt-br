# <a name="texture1dsamplebias-methods"></a>Métodos Texture1D:: SampleBias

Amostras de um [**Texture1D**](sm5-object-texture1d.md), depois de aplicar o valor de tendência ao nível de mipmap.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                           | Descrição                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias (S, float, float, int)**](dx-graphics-hlsl-to-samplebias.md)                          | Amostra uma textura, depois de aplicar o valor de tendência ao nível de mipmap.<br/>                                                                                                             |
| [**SampleBias (S, float, float, int, float)**](t1d-samplebias-s-float-float-int-float-.md)           | Amostra uma textura, depois de aplicar o valor de tendência ao nível de mipmap, com um valor opcional para fixe de exemplo de valores de LOD (nível de detalhe) para.<br/>                                     |
| [**SampleBias (S, float, float, int, float, uint)**](t1d-samplebias-s-float-float-int-float-uint-.md) | Amostra uma textura, depois de aplicar o valor de tendência ao nível de mipmap, com um valor opcional para fixe de exemplo de valores de LOD (nível de detalhe) para. Retorna o status sobre a operação.<br/> |
