# <a name="samplecmplevelzero-directx-hlsl-texture-object"></a>SampleCmpLevelZero (objeto de textura DirectX HLSL)

Obtém uma textura e compara o resultado com um valor de comparação. Essa função é idêntica a chamar [**SampleCmp**](dx-graphics-hlsl-to-samplecmp.md) somente no nível de mipmap 0.
