# <a name="hlsl-specified-root-signature"></a>HLSL assinatura de raiz especificada

Uma [assinatura de raiz](/windows/desktop/direct3d12/root-signatures) (uma tabela de chave de recursos e outros elementos para D3D12) pode ser especificada em HLSL como uma cadeia de caracteres.

Consulte os documentos do D3D12 sobre como [especificar assinaturas raiz em HLSL](/windows/desktop/direct3d12/specifying-root-signatures-in-hlsl) para obter detalhes completos e código de exemplo.
