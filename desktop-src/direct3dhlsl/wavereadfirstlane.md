# <a name="wavereadlanefirst-function"></a>Função WaveReadLaneFirst

Retorna o valor da expressão para a pista ativa da onda atual com o menor índice.

## <a name="syntax"></a>Sintaxe

``` syntax
<type> WaveReadLaneFirst(
   <type> expr
);
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*expr* 
</dt> <dd>

A expressão a ser avaliada.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

O valor resultante é uniforme em toda a onda.

## <a name="remarks"></a>Comentários

Essa função tem suporte do modelo de sombreador 6,0 em todos os estágios de sombreador.
