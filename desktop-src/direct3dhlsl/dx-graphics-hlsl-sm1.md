# <a name="shader-model-1"></a>Modelo de sombreador 1

O Shader Model 1 foi o primeiro modelo de sombreador criado no DirectX. Ele introduziu os sombreadores de vértices e de pixel na primeira implementação do pipeline programável.

O suporte para instruções de sombreador de pixel (ps_1_x) foi preterido. Para compilar ps_1_x sombreadores como sombreadores de ps_2_0, consulte <a href="/windows/desktop/direct3dtools/dx-graphics-tools-fxc-using">compilando o modelo de sombreador 1</a>.</li></ul> | | Conjunto de registros | <ul><li><a href="dx9-graphics-reference-asm-vs-registers-vs-1-1.md">Registros de sombreador de vértice</a></li></ul> | | Sombreador de vértice máx | 128 instruções | | Perfis de sombreador | vs_1_1 |
