# <a name="shader-model-51"></a>Modelo do sombreador 5,1

Esta seção contém as páginas de referência para o modelo 5,1 do sombreador HLSL, introduzidas com D3D12.

O Shader Model 5,1 é funcionalmente muito semelhante ao [modelo de sombreador 5](d3d11-graphics-reference-sm5.md), a principal alteração é mais flexibilidade na seleção de recursos, permitindo a indexação de matrizes de descritores de dentro de um sombreador.



| Recurso                   | Funcionalidade                                                                |
|---------------------------|---------------------------------------------------------------------------|
| Conjunto de instruções           | [**Funções intrínsecas HLSL**](dx-graphics-hlsl-intrinsic-functions.md)  |
| Sombreador máximo do vértice         | Nenhuma restrição                                                            |
| Sombreador máximo de pixel          | Nenhuma restrição                                                            |
| Novos perfis de sombreador adicionados | cs \_ 5 \_ 1, DS \_ 5 \_ 1, GS \_ 5 \_ 1, HS \_ 5 \_ 1, PS \_ 5 \_ 1, vs \_ 5 \_ 1, rootsig \_ 1 \_ 0 |
