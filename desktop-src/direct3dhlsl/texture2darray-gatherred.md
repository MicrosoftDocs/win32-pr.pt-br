# <a name="texture2darraygatherred-methods"></a>Métodos Texture2DArray::GatherRed

Retorna os componentes vermelhos de quatro valores de texel de [**Texture2DArray**](sm5-object-texture2darray.md)que seriam usados em uma operação de filtragem bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                      | Descrição                                                                                                                                     |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed(S,float,int)**](sm5-object-texture2darray-gatherred.md)                                        | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherRed(S,float,int,uint)**](t2darray-gatherred-s-float-int-uint-.md)                                  | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
| [**GatherRed(S,float,int2, int2, int2, int2)**](t2darray-gatherred-s-float-int2-int2-int2-int2-.md)            | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherRed(S,float,int2, int2, int2, int2,uint)**](t2darray-gatherred-s-float-int2-int2-int2-int2-uint-.md)  | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
