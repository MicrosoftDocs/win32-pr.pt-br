# <a name="texture2dgatherblue-methods"></a>Métodos Texture2D:: GatherBlue

Amostra um [**Texture2D**](sm5-object-texture2d.md) e retorna o componente azul.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                  | Descrição                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherBlue (S, float, int)**](sm5-object-texture2d-gatherblue.md)                                       | Retorna os componentes azuis dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherBlue (S, float, int, uint)**](t2d-gatherblue-s-float-int-uint-.md)                                 | Retorna os componentes azuis dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
| [**GatherBlue (S, float, Int2, Int2, Int2, Int2)**](t2d-gatherblue-s-float-int2-int2-int2-int2-.md)           | Retorna os componentes azuis dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherBlue (S, float, Int2, Int2, Int2, Int2, uint)**](t2d-gatherblue-s-float-int2-int2-int2-int2-uint-.md) | Retorna os componentes azuis dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
