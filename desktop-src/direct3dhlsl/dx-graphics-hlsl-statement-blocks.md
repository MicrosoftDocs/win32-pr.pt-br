# <a name="statement-blocks"></a>Blocos de instrução

Um bloco de instruções contém uma ou mais instruções; cada instrução é composta por [expressões](dx-graphics-hlsl-expressions.md) e [operadores](dx-graphics-hlsl-operators.md).


```
{
   statement 1;
   statement 2;
   ...
   statement n;
}
```



Um bloco de instruções também indica subescopo. Variáveis declaradas dentro de um bloco de instruções são reconhecidas apenas dentro do bloco.
