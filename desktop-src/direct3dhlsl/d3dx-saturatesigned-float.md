# <a name="d3dx_saturatesigned_float-function"></a>Função FLOAT SaturateSigned D3DX \_ \_

Recupera um valor saturado assinado do FLOAT determinado.

## <a name="syntax"></a>Sintaxe

``` syntax
 D3DX_SaturateSigned_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*\_V* 
</dt> <dd>

O valor a saturar.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

O valor saturado assinado.
