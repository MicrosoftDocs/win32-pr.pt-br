# <a name="texturecubearraygathercmp-methods"></a>Métodos TextureCubeArray:: GatherCmp

Para quatro valores Texel de um [**TextureCubeArray**](texturecubearray.md) que seriam usados em uma operação de filtragem de bi-linear, o retorna a comparação contra um valor de comparação.

Consulte a documentação sobre [gather4_c](./gather4-c--sm5---asm-.md) para obter mais informações descrevendo a instrução DXBC subjacente.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                            | Descrição                                                                                                      |
|:----------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**GatherCmp (S, float, float, uint)**](tcubearray-gathercmp-s-float-float-uint-.md) | Exemplos e compara uma textura e retorna todos os quatro componentes juntamente com o status sobre a operação.<br/> |
