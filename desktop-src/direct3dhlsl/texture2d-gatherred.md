# <a name="texture2dgatherred-methods"></a>Métodos Texture2D:: GatherRed

Retorna os componentes vermelhos dos quatro valores Texel de um [**Texture2D**](sm5-object-texture2d.md)que seriam usados em uma operação de filtragem de bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                  | Descrição                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed (S, float, int)**](sm5-object-texture2d-gatherred.md)                                         | Retorna os componentes vermelhos dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherRed (S, float, int, uint)**](t2d-gatherred-s-float-int-uint-.md)                                   | Retorna os componentes vermelhos dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
| [**GatherRed (S, float, Int2, Int2, Int2, Int2)**](t2d-gatherred-s-float-int2-int2-int2-int2-.md)             | Retorna os componentes vermelhos dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherRed (S, float, Int2, Int2, Int2, Int2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md) | Retorna os componentes vermelhos dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
