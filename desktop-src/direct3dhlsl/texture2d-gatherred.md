# <a name="texture2dgatherred-methods"></a>Métodos Texture2D::GatherRed

Retorna os componentes vermelhos dos quatro valores de texel de [**texture2D**](sm5-object-texture2d.md)que seriam usados em uma operação de filtragem bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                  | Descrição                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed(S,float,int)**](sm5-object-texture2d-gatherred.md)                                         | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherRed(S,float,int,uint)**](t2d-gatherred-s-float-int-uint-.md)                                   | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
| [**GatherRed(S,float,int2, int2, int2, int2)**](t2d-gatherred-s-float-int2-int2-int2-int2-.md)             | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherRed(S,float,int2, int2, int2, int2,uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md) | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
