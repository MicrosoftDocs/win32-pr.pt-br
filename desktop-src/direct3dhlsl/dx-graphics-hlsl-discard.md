# <a name="discard-statement"></a>Instrução de descarte

Não gere o resultado do pixel atual.



|          |
|----------|
| carte |



 

## <a name="parameters"></a>Parâmetros

Nenhum

## <a name="remarks"></a>Comentários

Esta instrução só pode ser chamada de um sombreador de pixel; Não há suporte para ele em um sombreador de geometria ou em um sombreador de vértice.
