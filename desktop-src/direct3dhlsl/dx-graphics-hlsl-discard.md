# <a name="discard-statement"></a>Instrução discard

Não esmiuem o resultado do pixel atual.

descartar;



 

## <a name="parameters"></a>Parâmetros

Nenhum

## <a name="remarks"></a>Comentários

Essa instrução só pode ser chamada de um sombreador de pixel; Não há suporte para ele em um sombreador de geometria ou em um sombreador de vértice.
