# <a name="texture2dgathergreen-methods"></a>Métodos Texture2D:: GatherGreen

Amostra um [**Texture2D**](sm5-object-texture2d.md) e retorna o componente verde.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                     | Descrição                                                                                                                                       |
|:-----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherGreen (S, float, int)**](sm5-object-texture2d-gathergreen.md)                                        | Retorna os componentes verdes dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherGreen (S, float, int, uint)**](t2d-gathergreen-s-float-int-uint-.md)                                  | Retorna os componentes verdes dos quatro valores de Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
| [**GatherGreen (S, float, Int2, Int2, Int2, Int2)**](t2d-gathergreen-s-float-float-int2-int2-int2-int2-.md)      | Retorna os componentes verdes dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherGreen (S, float, Int2, Int2, Int2, Int2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md)  | Retorna os componentes verdes dos quatro valores de Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
