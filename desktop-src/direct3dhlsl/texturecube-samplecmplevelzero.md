# <a name="texturecubesamplecmplevelzero-methods"></a>Métodos TextureCube:: SampleCmpLevelZero

Amostras de uma textura no nível de mipmap 0 somente, usando um valor de comparação para rejeitar amostras.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                         | Descrição                                                                                                                            |
|:-----------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, float, float)**](dx-graphics-hlsl-to-samplecmplevelzero.md)            | Faz a amostragem de uma textura no nível de mipmap 0 somente e compara o resultado a um valor de comparação.<br/>                                     |
| [**SampleCmpLevelZero (S, float, float, uint)**](tcube-samplecmplevelzero-s-float-float-uint-.md) | Faz a amostragem de uma textura no nível de mipmap 0 somente e compara o resultado a um valor de comparação. Retorna o status sobre a operação.<br/> |
