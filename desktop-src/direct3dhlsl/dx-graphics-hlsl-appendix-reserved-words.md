# <a name="reserved-words"></a>Palavras reservadas

|-------------------------------------|
| auto                                |
| caso, catch                         |
| conversão Char, Class, const \_            |
| padrão, excluir, \_ conversão dinâmica      |
| enum                                |
| explicit                            |
| friend                              |
| goto                                |
| long                                |
| mutable                             |
| novo                                 |
| operador                            |
| privado, protegido, público          |
| reinterpretar \_ conversão                   |
| conversão estática curta, assinada, sizeof \_ |
| modelo, isso, throw               |
| tentar, TypeName                       |
| União, não assinado                     |
| using                               |
| virtual                             |
