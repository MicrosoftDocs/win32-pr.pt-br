# <a name="d3dx_int_to_float-function"></a>D3DX \_ int \_ para a \_ função float

Converte um valor INT em FLOAT.

## <a name="syntax"></a>Sintaxe

``` syntax
FLOAT D3DX_INT_to_FLOAT(
   INT _V,
   FLOAT _Scale
);
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*\_L* 
</dt> <dd>

O valor v.

</dd> <dt>

*\_Escala* 
</dt> <dd>

O valor de escala.

</dd> </dl>

## <a name="return-value"></a>Retornar valor
