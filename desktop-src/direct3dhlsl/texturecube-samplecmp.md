|:-----------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S, float, float)**](dx-graphics-hlsl-to-samplecmp.md)                        | Amostra uma textura, usando um valor de comparação para rejeitar amostras.<br/>                                                                                                             |
| [**SampleCmp (S, float, float, float)**](tcube-samplecmp-s-float-float-float-.md)           | Amostras de uma textura, usando um valor de comparação para rejeitar amostras, com um valor opcional para fixe os valores de nível de detalhe (LOD) de exemplo para.<br/>                                     |
| [**SampleCmp (S, float, float, float, uint)**](tcube-samplecmp-s-float-float-float-uint-.md) | Amostras de uma textura, usando um valor de comparação para rejeitar amostras, com um valor opcional para fixe os valores de nível de detalhe (LOD) de exemplo para.
