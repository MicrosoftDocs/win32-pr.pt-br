# <a name="texturecubearraygatherred-methods"></a>Métodos TextureCubeArray::GatherRed

Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                | Descrição                                                                                                                                     |
|:----------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed(S,float,uint)**](tcubearray-gatherred-s-float-uint-.md)  | Retorna os componentes vermelhos dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
