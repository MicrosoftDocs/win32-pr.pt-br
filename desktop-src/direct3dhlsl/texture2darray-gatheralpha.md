# <a name="texture2darraygatheralpha-methods"></a>Métodos Texture2DArray::GatherAlpha

Retorna os componentes alfa de quatro valores de texel de [**Texture2DArray**](sm5-object-texture2darray.md)que seriam usados em uma operação de filtragem bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                          | Descrição                                                                                                                                       |
|:----------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherAlpha(S, float,int)**](sm5-object-texture2darray-gatheralpha.md)                                        | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherAlpha(S,float,int,uint)**](t2darray-gatheralpha-s-float-int-uint-.md)                                  | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
| [**GatherAlpha(S,float,int2, int2, int2,int2)**](t2darray-gatheralpha-s-float-int2-int2-int2-int2-.md)            | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherAlpha(S,float,int2, int2, int2, int2,uint)**](t2darray-gatheralpha-s-float-int2-int2-int2-int2-uint-.md)  | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
