# <a name="texturecubegathergreen-methods"></a>Métodos TextureCube:: GatherGreen

Retorna os componentes verdes dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                               | Descrição                                                                                                                                       |
|:---------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherGreen (S, float, uint)**](tcube-gathergreen-s-float-uint-.md)  | Retorna os componentes verdes dos quatro valores de Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
