# <a name="data-types-hlsl"></a>Tipos de dados (HLSL)

O HLSL dá suporte a muitos tipos de dados intrínsecos diferentes. Esta tabela mostra quais tipos usar para definir variáveis de sombreador.



| Usar esse tipo intrínseco                                                                                                                         | Para definir essa variável de sombreador                                            |
|-------------------------------------------------------------------------------------------------------------------------|--------------------------------------------|
| [Escalar](dx-graphics-hlsl-scalar.md)                                                                                   | Escalar de um componente                       |
| [Vector](dx-graphics-hlsl-vector.md), [Matrix](dx-graphics-hlsl-matrix.md)                                            | Matriz ou vetor de vários componentes        |
| [Amostra, textura](dx-graphics-hlsl-sampler.md) [ou](dx-graphics-hlsl-texture.md) [buffer](dx-graphics-hlsl-buffer.md)   | Objeto sampler, textura ou buffer         |
| [Struct](dx-graphics-hlsl-struct.md), [Definido pelo Usuário](dx-graphics-hlsl-user-defined.md)                                | Estrutura personalizada ou typedef                |
| Array                                                                                   | Expressões escalares literais declaradas que contêm a maioria dos outros tipos                       |
| [Objeto State](dx-graphics-hlsl-state-object.md) | Representações HLSL de objetos de estado |


 

Para [ajudá-lo](dx-graphics-hlsl-per-component-math.md) a entender melhor como usar vetores e matrizes em HLSL, talvez você queira ler essas informações em segundo plano sobre como o HLSL usa matemática por componente.
