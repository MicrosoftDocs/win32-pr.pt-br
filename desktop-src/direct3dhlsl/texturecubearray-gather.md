# <a name="texturecubearraygather-methods"></a>Métodos TextureCubeArray:: Reúna

Retorna os componentes vermelhos de quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.

Consulte a documentação em [gather4](./gather4--sm5---asm-.md) para obter mais informações descrevendo a instrução DXBC subjacente.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                          | Descrição                                                                                                                                |
|:----------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Coleta (S, float)**](dx-graphics-hlsl-to-gather.md)           | Retorna os componentes vermelhos de quatro valores de Texel (somente componente vermelho) que seriam usados em uma operação de filtragem de bi-linear. <br/>           |
| [**Coletar (S, float, uint)**](tcubearray-gather-s-float-uint-.md) | Retorna os componentes vermelhos de quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
