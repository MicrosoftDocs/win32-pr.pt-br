# <a name="debugging-shaders-in-visual-studio"></a>Depurando sombreadores no Visual Studio

A ferramenta mais recente para depuração de sombreadores agora é fornecida como um recurso no Microsoft Visual Studio, chamado depurador de gráficos do Visual Studio. Essa nova ferramenta é uma substituição para a ferramenta [PIX](/windows/desktop/directx-sdk--august-2009-) para Windows. O depurador de gráficos do Visual Studio aprimorou muito a usabilidade, o suporte para o Windows 8 e o Direct3D 11,1 e a integração com recursos tradicionais do Visual Studio, como pilhas de chamadas e janelas de depuração para depuração [HLSL](dx-graphics-hlsl.md) . Para obter mais informações sobre esse novo recurso, consulte [depuração de elementos gráficos do DirectX](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015).
