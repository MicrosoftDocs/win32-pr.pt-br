# <a name="debugging-shaders-in-visual-studio"></a>Depurando sombreadores no Visual Studio

A ferramenta mais recente para depurar sombreadores agora é Microsoft Visual Studio, chamada Visual Studio Depurador de Gráficos. Essa nova ferramenta é uma substituição para [a FERRAMENTA DE WINDOWS.](/windows/desktop/directx-sdk--august-2009-) Visual Studio O Depurador de Gráficos melhorou bastante a usabilidade, o suporte para o Windows 8 e o Direct3D 11.1 e a integração com recursos tradicionais do Visual Studio, como pilhas de chamada e janelas de depuração para depuração [HLSL.](dx-graphics-hlsl.md) Para obter mais informações sobre esse novo recurso, consulte [Depurando elementos gráficos do DirectX.](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015)
