# <a name="patchconstantfunc"></a>patchconstantfunc

Define a função para computação de dados constantes de patch.


```
patchconstantfunc("function_name")
```



## <a name="remarks"></a>Comentários

*\_ nome da função* é o nome de uma função separada que gera os dados de constante de patch.

Esse atributo tem suporte nos seguintes tipos de sombreadores:



| Vértice | Envoltória | Domínio | Geometria | 16x16 | Computação |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
