# <a name="texture2dsamplebias-methods"></a>Métodos Texture2D::SampleBias

Amostra um [**Texture2D**](sm5-object-texture2d.md), depois de aplicar o valor de desvio ao nível de mipmap.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                       | Descrição                                                                                                                                                                                  |
|:---------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias(S,float,float,int)**](dx-graphics-hlsl-to-samplebias.md)                      | Amostra uma textura, depois de aplicar o valor de desvio ao nível de mipmap.<br/>                                                                                                             |
| [**SampleBias(S,float,float,int,float)**](samplebias-s-float-float-int-float-.md)           | Amostra uma textura, depois de aplicar o valor de desvio ao nível de mipmap, com um valor opcional para fixar valores lod (nível de detalhes) de exemplo.<br/>                                     |
| [**SampleBias(S,float,float,int, float,uint)**](samplebias-s-float-float-int-float-uint-.md) | Amostra uma textura, depois de aplicar o valor de desvio ao nível de mipmap, com um valor opcional para fixar valores lod (nível de detalhes) de exemplo. Retorna o status sobre a operação.<br/> |
