# <a name="d3dx_float_to_int-function"></a>D3DX \_ flutuar \_ para \_ função int

Converte um valor FLOAT em INT.

## <a name="syntax"></a>Sintaxe

``` syntax
INT D3DX_FLOAT_to_INT(
   FLOAT _V,
   FLOAT _Scale
);
```
