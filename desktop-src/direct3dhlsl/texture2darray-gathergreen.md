# <a name="texture2darraygathergreen-methods"></a>Métodos Texture2DArray:: GatherGreen

Retorna os componentes verdes dos quatro valores de Texel de um [**Texture2DArray**](sm5-object-texture2darray.md)que seriam usados em uma operação de filtragem de bi linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                          | Descrição                                                                                                                                       |
|:----------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherGreen (S, float, int)**](sm5-object-texture2darray-gathergreen.md)                                        | Retorna os componentes verdes dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherGreen (S, float, int, uint)**](t2darray-gathergreen-s-float-int-uint-.md)                                  | Retorna os componentes verdes dos quatro valores de Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
| [**GatherGreen (S, float, Int2, Int2, Int2, Int2)**](t2darray-gathergreen-s-float-int2-int2-int2-int2-.md)            | Retorna os componentes verdes dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.<br/>                                 |
| [**GatherGreen (S, float, Int2, Int2, Int2, Int2, uint)**](t2darray-gathergreen-s-float-int2-int2-int2-int2-uint-.md)  | Retorna os componentes verdes dos quatro valores de Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
