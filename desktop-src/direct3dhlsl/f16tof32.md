# <a name="f16tof32-function"></a>Função f16tof32

Converte o float16 armazenado na metade baixa do uint em um float.

## <a name="syntax"></a>Sintaxe

``` syntax
float f16tof32(
  in uint value
);
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*value* \[ Em\]
</dt> <dd>

Tipo: **uint**

O valor de entrada.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Tipo: **float**

O valor convertido.

## <a name="remarks"></a>Comentários

As seguintes versões sobrecarregadas também estão disponíveis:

``` syntax
float2 f16tof32(uint2 value);
float3 f16tof32(uint3 value);
float4 f16tof32(uint4 value);
```

### <a name="minimum-shader-model"></a>Modelo de sombreador mínimo

Essa função tem suporte nos modelos de sombreador a seguir.



| Modelo de Sombreador                                                        | Com suporte |
|---------------------------------------------------------------------|-----------|
| [Modelo de sombreador 4](dx-graphics-hlsl-sm4.md) e modelos de sombreador superior | sim       |



 

Embora essa função seja disponível no Modelo de Sombreador [4](dx-graphics-hlsl-sm4.md) e modelos de sombreador superior, pois ela é emulada em 4.0 e 4.1, ela é menos perfomant nesses modelos de sombreador do que no Modelo de Sombreador [5.](d3d11-graphics-reference-sm5.md)

Essa função tem suporte nos seguintes tipos de sombreadores:



| Vértice | Casco | Domínio | Geometry | Pixel | Computação |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
