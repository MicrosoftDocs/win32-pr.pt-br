# <a name="texturecubegatherred-methods"></a>Métodos TextureCube:: GatherRed

Retorna os componentes vermelhos dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                           | Descrição                                                                                                                                     |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed (S, float, uint)**](tcube-gatherred-s-float-uint-.md)  | Retorna os componentes vermelhos dos quatro valores Texel que seriam usados em uma operação de filtragem de bi-linear, juntamente com o status de mapeamento de bloco.<br/> |
