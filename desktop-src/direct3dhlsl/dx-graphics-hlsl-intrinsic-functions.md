## <a name="component-and-template-types"></a>Componentes e tipos de modelo

As declarações de função intrínsecas HLSL usam tipos de componentes e tipos de modelo para argumentos de parâmetro de entrada e valores de retorno. Os tipos disponíveis são listados na tabela a seguir.



| Esses tipos de modelo | Descrição                                           | Suporte a esses tipos de dados                                        |
|----------------------|-------------------------------------------------------|-----------------------------------------------------------------|
| matriz               | até 16 componentes, dependendo da declaração      | [Tipos básicos de HLSL](dx-graphics-hlsl-data-types.md)             |
| objeto               | objeto de amostra                                        | *amostra*, *sampler1D*, *sampler2D*, *sampler3D*, *samplerCUBE* |
| escalar               | 1 componente                                           | [Tipos básicos de HLSL](dx-graphics-hlsl-data-types.md)             |
| vector               | 1 componente no mínimo, 4 componentes máximo (inclusivo) | [Tipos básicos de HLSL](dx-graphics-hlsl-data-types.md)             |
