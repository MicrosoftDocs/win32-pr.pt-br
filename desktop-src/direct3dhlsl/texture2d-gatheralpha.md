# <a name="texture2dgatheralpha-methods"></a>Métodos Texture2D::GatherAlpha

Amostra um [**Texture2D**](sm5-object-texture2d.md) e retorna o componente alfa.

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                     | Descrição                                                                                                                                       |
|:-----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherAlpha(S, float,int)**](sm5-object-texture2d-gatheralpha.md)                                        | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherAlpha(S,float,int,uint)**](t2d-gatheralpha-s-float-int-uint-.md)                                  | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
| [**GatherAlpha(S,float,int2, int2, int2,int2)**](t2d-gatheralpha-s-float-int2-int2-int2-int2-.md)            | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear.<br/>                                 |
| [**GatherAlpha(S,float,int2, int2, int2, int2,uint)**](t2d-gatheralpha-s-float-int2-int2-int2-int2-uint-.md)  | Retorna os componentes alfa dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
