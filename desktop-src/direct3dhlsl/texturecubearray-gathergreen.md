# <a name="texturecubearraygathergreen-methods"></a>Métodos TextureCubeArray::GatherGreen

Amostra uma textura e retorna o componente verde.

| Método                                                                    | Descrição                                                                                                                                       |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherGreen(S,float,uint)**](tcubearray-gathergreen-s-float-uint-.md)  | Retorna os componentes verdes dos quatro valores de texel que seriam usados em uma operação de filtragem bi-linear, juntamente com o status de mapeamento de peças.<br/> |
