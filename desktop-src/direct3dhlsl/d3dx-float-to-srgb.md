# <a name="d3dx_float_to_srgb-function"></a>D3DX \_ flutuar \_ para a \_ função sRGB

Converte um valor FLOAT em um SRGB.

## <a name="syntax"></a>Sintaxe

``` syntax
FLOAT D3DX_FLOAT_to_SRGB(
   FLOAT val
);
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Val* 
</dt> <dd>

Valor FLOAT a ser convertido.

</dd> </dl>

## <a name="return-value"></a>Retornar valor
