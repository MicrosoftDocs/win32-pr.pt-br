# <a name="dcl_hs_max_tessfactor-sm5---asm"></a>DCL \_ HS \_ Max \_ tessfactor (SM5-ASM)

Declare o maxTessFactor para o patch.



| DCL \_ HS \_ Max \_ tessfactor n |
|----------------------------|



 



| Item                                                   | Descrição                          |
|--------------------------------------------------------|--------------------------------------|
| <span id="n"></span><span id="N"></span>*p*<br/> | \[no \] maxTessFactor.<br/> |



 

## <a name="remarks"></a>Comentários

MaxTessFactor é um valor de float32 no intervalo de {1,0... 64,0}.

Essa instrução se aplica aos seguintes estágios de sombreador:



| Vértice | Envoltória | Domínio | Geometria | 16x16 | Computação |
|--------|------|--------|----------|-------|---------|
|        | X    |        |          |       |         |
