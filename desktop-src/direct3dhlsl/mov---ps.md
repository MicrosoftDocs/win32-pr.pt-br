# <a name="mov---ps"></a>mov - ps

Mover dados entre registros.

## <a name="syntax"></a>Syntax



| mov dst, src |
|--------------|



 

onde

-   dst é o registro de destino.
-   src é um registro de origem.

## <a name="remarks"></a>Comentários
