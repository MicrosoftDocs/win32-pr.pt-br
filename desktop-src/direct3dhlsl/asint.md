# <a name="asint-function"></a>função asent

Interpreta o padrão de bit de um valor de entrada como um inteiro. Para obter mais informações sobre a função intrínseca de **asent** , consulte [**Asent (DirectX HLSL)**](dx-graphics-hlsl-asint.md).

## <a name="syntax"></a>Sintaxe

``` syntax
int asint(
  in  value
);
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

*valor* \[ do no\]
</dt> <dd>

O valor de entrada.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Tipo: **int**

A entrada interpretada como um inteiro.
