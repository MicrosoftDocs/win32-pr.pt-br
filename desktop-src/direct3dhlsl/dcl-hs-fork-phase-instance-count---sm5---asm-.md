| dcl \_ hs \_ fork phase instance count \_ \_ \_ {1...max 32-bit UINT} |
|-------------------------------------------------------------|



 



| Item                                                   | Descrição                           |
|--------------------------------------------------------|---------------------------------------|
| <span id="N"></span><span id="n"></span>*N*<br/> | \[em \] A contagem de instâncias.<br/> |
