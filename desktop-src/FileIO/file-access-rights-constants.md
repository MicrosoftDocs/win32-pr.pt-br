# <a name="file-access-rights-constants"></a>Constantes de direitos de acesso de arquivo

Os direitos de acesso válidos para arquivos e diretórios [](/windows/desktop/SecAuthZ/standard-access-rights)incluem os direitos de acesso padrão **DELETE,** READ **\_ CONTROL,** WRITE **\_ DAC,** WRITE **\_ OWNER** e **SYNCHRONIZE.** A tabela a seguir lista os direitos de acesso específicos para arquivos e diretórios.



| Constante/valor                                                                                                                                                                                                                                             | Descrição                                                                                                                                                                                                                                                                                                                                                                        |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="FILE_ADD_FILE"></span><span id="file_add_file"></span><dl> <dt>**ARQUIVO \_ ADICIONAR \_ ARQUIVO**</dt> <dt>2</dt> </dl>                                      | Para um diretório, o direito de criar um arquivo no diretório.<br/>                                                                                                                                                                                                                                                                                                           |
| <span id="FILE_ADD_SUBDIRECTORY"></span><span id="file_add_subdirectory"></span><dl> <dt>**ARQUIVO \_ ADICIONAR \_ SUBDIRETÓRIO**</dt> <dt>4</dt> </dl>              | Para um diretório, o direito de criar um subdiretório.<br/>                                                                                                                                                                                                                                                                                                                    |
| <span id="FILE_ALL_ACCESS"></span><span id="file_all_access"></span><dl> <dt>**ARQUIVO \_ TODO \_ O ACESSO**</dt><dt></dt> </dl>                                 | Todos os direitos de acesso possíveis para um arquivo.<br/>                                                                                                                                                                                                                                                                                                                                  |
| <span id="FILE_APPEND_DATA"></span><span id="file_append_data"></span><dl> <dt>**ARQUIVO \_ ANEXAR \_ DADOS**</dt> <dt>4</dt> </dl>                             | Para um objeto de arquivo, o direito de anexar dados ao arquivo. (Para arquivos locais, as operações de gravação não substituirão os dados existentes se esse sinalizador for especificado **sem FILE \_ WRITE \_ DATA**.) Para um objeto de diretório, o direito de criar um subdiretório (**FILE \_ ADD \_ SUBDIRECTORY**).<br/>                                                                                        |
| <span id="FILE_CREATE_PIPE_INSTANCE"></span><span id="file_create_pipe_instance"></span><dl> <dt>**ARQUIVO \_ CREATE \_ PIPE \_ INSTANCE**</dt> <dt>4</dt> </dl> | Para um pipe nomeado, o direito de criar um pipe.<br/>                                                                                                                                                                                                                                                                                                                           |
| <span id="FILE_DELETE_CHILD"></span><span id="file_delete_child"></span><dl> <dt>**ARQUIVO \_ DELETE \_ CHILD**</dt> <dt>64 (0x40)</dt> </dl>                  | Para um diretório, o direito de excluir um diretório e todos os arquivos que ele contém, incluindo arquivos somente leitura.<br/>                                                                                                                                                                                                                                                              |
| <span id="FILE_EXECUTE"></span><span id="file_execute"></span><dl> <dt>**ARQUIVO \_ EXECUTE**</dt> <dt>32 (0x20)</dt> </dl>                                  | Para um arquivo de código nativo, o direito de executar o arquivo. Esse direito de acesso dado aos scripts pode fazer com que o script seja executável, dependendo do interpretador de script.<br/>                                                                                                                                                                                                   |
| <span id="FILE_LIST_DIRECTORY"></span><span id="file_list_directory"></span><dl> <dt>**ARQUIVO \_ LIST \_ DIRECTORY**</dt> <dt>1</dt> </dl>                    | Para um diretório, o direito de listar o conteúdo do diretório.<br/>                                                                                                                                                                                                                                                                                                       |
| <span id="FILE_READ_ATTRIBUTES"></span><span id="file_read_attributes"></span><dl> <dt>**ARQUIVO \_ LER \_ ATRIBUTOS**</dt> <dt>128 (0x80)</dt> </dl>        | O direito de ler atributos de arquivo.<br/>                                                                                                                                                                                                                                                                                                                                      |
| <span id="FILE_READ_DATA"></span><span id="file_read_data"></span><dl> <dt>**ARQUIVO \_ LER \_ DADOS**</dt> <dt>1</dt> </dl>                                   | Para um objeto de arquivo, o direito de ler os dados de arquivo correspondentes. Para um objeto de diretório, o direito de ler os dados de diretório correspondentes.<br/>                                                                                                                                                                                                                           |
| <span id="FILE_READ_EA"></span><span id="file_read_ea"></span><dl> <dt>**ARQUIVO \_ LEIA \_ EA**</dt> <dt>8</dt> </dl>                                         | O direito de ler atributos de arquivo estendido.<br/>                                                                                                                                                                                                                                                                                                                             |
| <span id="FILE_TRAVERSE"></span><span id="file_traverse"></span><dl> <dt>**ARQUIVO \_ TRAVERSE**</dt> <dt>32 (0x20)</dt> </dl>                               | Para um diretório, o direito de percorrer o diretório. Por padrão, os usuários são atribuídos ao **privilégio BYPASS \_ TRAVERSE \_ CHECKING** [,](/windows/desktop/SecAuthZ/privileges)que ignora o direito **de acesso DE PERCORRER \_ ARQUIVO** . [](/windows/desktop/SecAuthZ/access-rights-and-access-masks) Consulte os comentários em Segurança [de Arquivo e Direitos de Acesso](file-security-and-access-rights.md) para obter mais informações.<br/> |
| <span id="FILE_WRITE_ATTRIBUTES"></span><span id="file_write_attributes"></span><dl> <dt>**ARQUIVO \_ GRAVAR \_ ATRIBUTOS**</dt> <dt>256 (0x100)</dt> </dl>    | O direito de gravar atributos de arquivo.<br/>                                                                                                                                                                                                                                                                                                                                     |
| <span id="FILE_WRITE_DATA"></span><span id="file_write_data"></span><dl> <dt>**ARQUIVO \_ GRAVAR \_ DADOS**</dt> <dt>2</dt> </dl>                                | Para um objeto de arquivo, o direito de gravar dados no arquivo. Para um objeto de diretório, o direito de criar um arquivo no diretório (**FILE \_ ADD \_ FILE**).<br/>                                                                                                                                                                                                                      |
| <span id="FILE_WRITE_EA"></span><span id="file_write_ea"></span><dl> <dt>**ARQUIVO \_ WRITE \_ EA**</dt> <dt>16 (0x10)</dt> </dl>                              | O direito de gravar atributos de arquivo estendidos.<br/>                                                                                                                                                                                                                                                                                                                            |
| <span id="STANDARD_RIGHTS_READ"></span><span id="standard_rights_read"></span><dl> <dt>**STANDARD \_ DIREITOS \_ LIDOS**</dt><dt></dt> </dl>                  | Inclui **READ \_ CONTROL**, que é o direito de ler as informações no descritor de segurança do objeto de arquivo ou diretório. Isso não inclui as informações na SACL.<br/>                                                                                                                                                                                        |
| <span id="STANDARD_RIGHTS_WRITE"></span><span id="standard_rights_write"></span><dl> <dt>**STANDARD \_ GRAVAÇÃO DE \_ DIREITOS**</dt><dt></dt> </dl>               | O mesmo que **o \_ STANDARD RIGHTS \_ READ**.<br/>                                                                                                                                                                                                                                                                                                                                     |
