# <a name="file-access-rights-constants"></a>Constantes de direitos de acesso a arquivos

Os direitos de acesso válidos para arquivos e diretórios incluem os direitos de acesso de **exclusão**, **\_ controle de leitura**, **gravação de \_ DAC**, **\_ proprietário de gravação** e **sincronização** [padrão](/windows/desktop/SecAuthZ/standard-access-rights). A tabela a seguir lista os direitos de acesso específicos para arquivos e diretórios.



| Constante/valor                                                                                                                                                                                                                                             | Descrição                                                                                                                                                                                                                                                                                                                                                                        |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="FILE_ADD_FILE"></span><span id="file_add_file"></span><dl> Do <dt>**arquivo \_ Adicionar \_ arquivo**</dt> <dt>2</dt> </dl>                                      | Para um diretório, o direito de criar um arquivo no diretório.<br/>                                                                                                                                                                                                                                                                                                           |
| <span id="FILE_ADD_SUBDIRECTORY"></span><span id="file_add_subdirectory"></span><dl> Do <dt>**arquivo \_ Adicionar \_ subdiretório**</dt> <dt>4</dt> </dl>              | Para um diretório, o direito de criar um subdiretório.<br/>                                                                                                                                                                                                                                                                                                                    |
| <span id="FILE_ALL_ACCESS"></span><span id="file_all_access"></span><dl> Do <dt>**arquivo \_ TODO o \_ acesso**</dt><dt></dt> </dl>                                 | Todos os direitos de acesso possíveis para um arquivo.<br/>                                                                                                                                                                                                                                                                                                                                  |
| <span id="FILE_APPEND_DATA"></span><span id="file_append_data"></span><dl> Do <dt>**arquivo \_ ACRESCENTAR \_ dados**</dt> <dt>4</dt> </dl>                             | Para um objeto de arquivo, o direito de acrescentar dados ao arquivo. (Para arquivos locais, as operações de gravação não substituirão os dados existentes se esse sinalizador for especificado sem **\_ \_ dados de gravação de arquivo**.) Para um objeto de diretório, o direito de criar um subdiretório (**arquivo \_ Adicionar \_ subdiretório**).<br/>                                                                                        |
| <span id="FILE_CREATE_PIPE_INSTANCE"></span><span id="file_create_pipe_instance"></span><dl> Do <dt>**arquivo \_ CRIAR \_ \_ instância de pipe**</dt> <dt>4</dt> </dl> | Para um pipe nomeado, o direito de criar um pipe.<br/>                                                                                                                                                                                                                                                                                                                           |
