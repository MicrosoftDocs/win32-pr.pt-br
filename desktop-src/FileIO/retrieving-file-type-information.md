# <a name="retrieving-file-type-information"></a>Recuperando informações de tipo de arquivo

A função [**Getfiletype**](/windows/desktop/api/FileAPI/nf-fileapi-getfiletype) recupera o tipo de um arquivo: disco, caractere (como um console), pipe ou desconhecido. A função [**Getbinarytype**](/windows/desktop/api/WinBase/nf-winbase-getbinarytypea) determina se um arquivo é executável e, nesse caso, o tipo de arquivo executável. Consulte **Getbinarytype** para obter uma lista dos tipos executáveis com suporte.
