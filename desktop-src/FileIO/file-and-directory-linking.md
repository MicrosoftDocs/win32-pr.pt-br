# <a name="file-and-directory-linking"></a>Vinculação de arquivo e diretório

O sistema de arquivos NTFS fornece a capacidade de criar uma representação do sistema de um arquivo ou diretório em um local na estrutura de diretório diferente do objeto de arquivo ou diretório ao qual está sendo vinculado. Esse processo é chamado de vinculação. Há dois tipos de links com suporte no sistema de arquivos NTFS: [links rígidos e junções](hard-links-and-junctions.md).

O sistema de arquivos NTFS também fornece o [serviço de](distributed-link-tracking-and-object-identifiers.md)acompanhamento de link distribuído , que rastreia automaticamente os links conforme eles são movidos.
