# <a name="file-and-directory-linking"></a>Vinculação de arquivos e diretórios

O sistema de arquivos NTFS fornece a capacidade de criar uma representação de sistema de um arquivo ou diretório em um local na estrutura de diretório que é diferente do objeto de arquivo ou diretório ao qual está sendo vinculado. Esse processo é chamado de vinculação. Há dois tipos de links com suporte no sistema de arquivos NTFS: [links físicos e junções](hard-links-and-junctions.md).

O sistema de arquivos NTFS também fornece o [serviço Distributed Link Tracking](distributed-link-tracking-and-object-identifiers.md), que rastreia automaticamente os links conforme eles são movidos.
