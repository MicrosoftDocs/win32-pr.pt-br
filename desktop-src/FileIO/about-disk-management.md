# <a name="about-disk-management"></a>Sobre o Gerenciamento de Disco

O sistema de arquivos fornece uma abstração das características físicas dos dispositivos de armazenamento para que os aplicativos possam simplesmente gravar e ler de arquivos. No entanto, o armazenamento em uma empresa depende muito do conceito de discos.
