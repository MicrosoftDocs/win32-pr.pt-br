# <a name="txf-control-codes"></a>Códigos de controle TxF

\[A Microsoft recomenda fortemente que os desenvolvedores utilizem meios alternativos para atingir as necessidades do aplicativo. Muitos cenários para os que o TxF foi desenvolvido podem ser obtidos por meio de técnicas mais simples e prontamente disponíveis. Além disso, o TxF pode não estar disponível em versões futuras do Microsoft Windows. Para obter mais informações e alternativas ao TxF, consulte Alternativas ao uso [de NTFS transacional.](deprecation-of-txf.md)\]

O NTFS transacional (TxF) fornece os seguintes códigos de controle.

## <a name="in-this-section"></a>Nesta seção



| Código de controle                                                                                                 | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                             |
|--------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FSCTL \_ TXFS \_ CREATE \_ MINIVERSION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_create_miniversion)<br/>                         | Cria uma [nova miniversão](glossary.md) para o arquivo especificado. <br/> Miniversions permitem que você consulte um instantâneo do arquivo durante uma transação. Miniversions são descartadas quando uma transação é comprometida ou revertida.<br/>                                                                                                                                                                      |
| [**FSCTL \_ TXFS \_ OBTER INFORMAÇÕES DE \_ \_ METADADOS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_metadata_info)<br/>                          | Recupera metadados TxF (Transacted NTFS) para um arquivo e o **GUID** da transação que travada o arquivo especificado (se o arquivo estiver bloqueado). <br/>                                                                                                                                                                                                                                                                         |
| [**FSCTL \_ TXFS \_ GET \_ TRANSACTED \_ VERSION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_transacted_version)<br/>                | Retorna uma [**estrutura GET \_ \_ TRANSACTED \_ VERSION do TXFS.**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_get_transacted_version) A estrutura identifica a versão mais recentemente comprometida do arquivo especificado, o número de versão do identificador. <br/>                                                                                                                                                                                                            |
| [**FSCTL \_ TXFS \_ LIST \_ TRANSACTION \_ LOCKED \_ FILES**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transaction_locked_files)<br/> | Retorna uma lista de todos os arquivos atualmente bloqueados pela transação especificada. Se o valor de retorno for **ERROR \_ MORE \_ DATA**, ele retornará o comprimento do buffer necessário para manter a lista completa de arquivos no momento dessa chamada.<br/>                                                                                                                                                                                           |
| [**FSCTL \_ TXFS \_ LIST \_ TRANSACTIONS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transactions)<br/>                           | Retorna uma lista de todas as transações atualmente envolvidas no gerenciador de recursos especificado.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**FSCTL \_ TXFS \_ MODIFY \_ RM**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_modify_rm)<br/>                                           | Define o modo de log e as informações de parâmetro de log para um RM (gerenciador de recursos secundário).<br/>                                                                                                                                                                                                                                                                                                                                       |
| [**FSCTL \_ TXFS \_ QUERY \_ RM \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_query_rm_information)<br/>                    | Recupera informações para um RM (gerenciador de recursos).<br/>                                                                                                                                                                                                                                                                                                                                                                           |
| [**FSCTL \_ TXFS \_ LER INFORMAÇÕES DE \_ BACKUP \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information)<br/>              | Retorna informações específicas de NTFS transacional (TxF) para o arquivo especificado.<br/>                                                                                                                                                                                                                                                                                                                                                |
| [**FSCTL \_ TXFS \_ SAVEPOINT \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information)<br/>                   | O [**código de controle FSCTL \_ TXFS \_ SAVEPOINT \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information) controla a configuração, a limpeza e a redução para o ponto de salvar especificado.<br/> Para executar essa operação, chame a [**função DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) com os parâmetros a seguir.<br/>                                                                                                                 |
| [**FSCTL \_ TXFS \_ TRANSACTION \_ ACTIVE**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_transaction_active)<br/>                         | Retorna um valor booliana que indica se havia transações ativas no volume associado quando o instantâneo foi tirado. Essa chamada só é válida para volumes de instantâneo somente leitura.<br/>                                                                                                                                                                                                                                   |
| [**FSCTL \_ TXFS \_ WRITE \_ BACKUP \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_write_backup_information)<br/>            | Grava informações específicas de NTFS transacionais (TxF) em um arquivo especificado. O **membro buffer** da estrutura [**TXFS \_ WRITE BACKUP \_ \_ INFORMATION**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_write_backup_information) deve ser o membro buffer da estrutura [**TXFS \_ READ BACKUP INFORMATION \_ \_ \_ OUT**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_read_backup_information_out) retornada por [**FSCTL \_ TXFS \_ READ BACKUP \_ \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information). <br/> |
