# <a name="encrypted-files-and-user-keys"></a>Arquivos criptografados e chaves de usuário

Para criar uma nova chave para um usuário, use a função [**SetUserFileEncryptionKey**](/windows/desktop/api/Winefs/nf-winefs-setuserfileencryptionkey) . Para adicionar chaves de usuário a um arquivo criptografado, use a função [**AddUsersToEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-adduserstoencryptedfile) . Para consultar as chaves de usuário de um arquivo criptografado, use a função [**QueryUsersOnEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-queryusersonencryptedfile) . Para remover as chaves de usuário de um arquivo criptografado, use a função [**RemoveUsersFromEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-removeusersfromencryptedfile) .
