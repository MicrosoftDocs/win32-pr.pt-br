# <a name="volume-management"></a>Gerenciamento de volume

O nível mais alto de organização no sistema de arquivos é o *volume*. Um sistema de arquivos reside em um volume. Um volume contém pelo menos uma *partição*, que é uma divisão lógica de um disco físico (para obter mais informações, consulte [partições e dispositivos de disco](disk-devices-and-partitions.md)). Um volume que contém dados que existem em uma partição é chamado de *volume simples* e um volume que contém dados que existem em mais de uma partição é chamado de *volume de multipartição*.
