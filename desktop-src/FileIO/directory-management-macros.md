# <a name="directory-management-macros"></a>Macros de gerenciamento de diretório

As macros a seguir são usadas com pontos de nova análise.

## <a name="in-this-section"></a>Nesta seção



| Macro                                                                     | Descrição                                                                                                                             |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**IsReparseTagMicrosoft**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagmicrosoft)<br/>         | Determina se uma marca de ponto de nova análise indica um ponto de nova análise da Microsoft.<br/>                                                  |
| [**IsReparseTagNameSurrogate**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagnamesurrogate)<br/> | Determina se o ponto de nova análise associado de uma marca é um substituto para outra entidade nomeada (por exemplo, uma pasta montada).<br/> |
