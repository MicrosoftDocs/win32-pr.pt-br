# <a name="txf-functions"></a>Funções TxF

O NTFS Transacional (TxF) fornece as funções a seguir.

## <a name="in-this-section"></a>Nesta seção



| Função                                                                      | Descrição                                                                                                                  |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [**TxfLogCreateFileReadContext**](/windows/desktop/api/TxfW32/nf-txfw32-txflogcreatefilereadcontext)<br/> | Cria um contexto a ser usado para ler registros de replicação.<br/>                                                         |
| [**TxfLogDestroyReadContext**](/windows/desktop/api/TxfW32/nf-txfw32-txflogdestroyreadcontext)<br/>       | Fecha um contexto de leitura criado pela função [**TxfLogCreateFileReadContext**](/windows/desktop/api/TxfW32/nf-txfw32-txflogcreatefilereadcontext) .<br/> |
| [**TxfLogReadRecords**](/windows/desktop/api/TxfW32/nf-txfw32-txflogreadrecords)<br/>                     | Lê os registros refazer do log.<br/>                                                                              |
