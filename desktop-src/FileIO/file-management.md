# <a name="file-management-local-file-systems"></a>Gerenciamento de arquivos (sistemas de arquivos locais)

Um *objeto File* fornece uma representação de um recurso (um dispositivo físico ou um recurso localizado em um dispositivo físico) que pode ser gerenciado pelo sistema de e/s. Assim como outros objetos, eles permitem o compartilhamento do recurso, eles têm nomes, eles são protegidos pela segurança baseada em objeto e oferecem suporte à sincronização. O sistema de e/s também permite a leitura ou gravação no recurso.
