# <a name="xaudio2-macros"></a>XAudio2 macros

Esta seção contém informações sobre macros fornecidas pela API do Microsoft XAudio2.



| Macro                          | Description                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAPOAlloc**](/windows/desktop/api/XAPO/nf-xapo-xapoalloc) | Macro de alocação de memória usada por métodos IXAPO que devem alocar estruturas de tamanho arbitrário que são retornadas posteriormente para o aplicativo.<br/> |
| [**XAPOFree**](/windows/desktop/api/XAPO/nf-xapo-xapofree)   | Macro usada para liberar memória alocada com a macro XAPOAlloc.<br/>                                                                                  |
