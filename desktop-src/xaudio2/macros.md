# <a name="xaudio2-macros"></a>Macros XAudio2

Esta seção contém informações sobre macros fornecidas pela API do Microsoft XAudio2.



| Macro                          | Descrição                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAPOAlloc**](/windows/desktop/api/XAPO/nf-xapo-xapoalloc) | Macro de alocação de memória usada por métodos IXAPO que devem alocar estruturas de tamanho arbitrárias que são retornadas posteriormente ao aplicativo.<br/> |
| [**XAPOFree**](/windows/desktop/api/XAPO/nf-xapo-xapofree)   | Macro usada para liberar memória alocada com a macro XAPOAlloc.<br/>                                                                                  |
