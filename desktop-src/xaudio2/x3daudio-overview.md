## <a name="listeners-and-emitters"></a><span data-ttu-id="317b2-106">Ouvintes e emissores</span><span class="sxs-lookup"><span data-stu-id="317b2-106">Listeners and Emitters</span></span>

-   <span data-ttu-id="317b2-109">Um ouvinte é definido como um ponto no espaço e uma orientação.</span><span class="sxs-lookup"><span data-stu-id="317b2-109">A listener is defined as a point in space and an orientation.</span></span> <span data-ttu-id="317b2-110">É a posição na qual o som é *ouvido*.</span><span class="sxs-lookup"><span data-stu-id="317b2-110">It is the position at which the sound is *heard*.</span></span> <span data-ttu-id="317b2-111">A posição e a orientação do ouvinte geralmente são as mesmas que a posição e a orientação da câmera.</span><span class="sxs-lookup"><span data-stu-id="317b2-111">The position and orientation of the listener generally is the same as the position and orientation of the camera.</span></span> <span data-ttu-id="317b2-112">Isso é verdadeiro se um título usa uma exibição de perspectiva da primeira pessoa ou da terceira pessoa.</span><span class="sxs-lookup"><span data-stu-id="317b2-112">This is true whether a title uses a first-person or third-person perspective view.</span></span> <span data-ttu-id="317b2-113">A posição do ouvinte é expressa em coordenadas mundiais.</span><span class="sxs-lookup"><span data-stu-id="317b2-113">The listener's position is expressed in world coordinates.</span></span> <span data-ttu-id="317b2-114">É importante observar que é a posição do ouvinte *relativa* a um emissor que determina como calcular os volumes de alto-falantes finais.</span><span class="sxs-lookup"><span data-stu-id="317b2-114">It is important to note that it is the listener's position *relative* to an emitter that determines how to calculate the final speaker volumes.</span></span>
-   <span data-ttu-id="317b2-115">Um emissor é definido como um (ou mais) pontos no espaço do qual um som é *originado*.</span><span class="sxs-lookup"><span data-stu-id="317b2-115">An emitter is defined as one (or more) points in space from which a sound *originates*.</span></span> <span data-ttu-id="317b2-116">A posição do emissor pode estar em qualquer lugar no espaço 3D.</span><span class="sxs-lookup"><span data-stu-id="317b2-116">The position of the emitter can be anywhere in 3D space.</span></span> <span data-ttu-id="317b2-117">Como um ouvinte, a posição de um emissor é expressa em coordenadas mundiais.</span><span class="sxs-lookup"><span data-stu-id="317b2-117">Like a listener, an emitter's position is expressed in world coordinates.</span></span> <span data-ttu-id="317b2-118">É a posição do emissor *relativa* ao ouvinte que determina como os volumes de alto-falante finais são calculados.</span><span class="sxs-lookup"><span data-stu-id="317b2-118">It is the emitter's position *relative* to the listener that determines how the final speaker volumes are calculated.</span></span>
-   <span data-ttu-id="317b2-119">X3DAudio usa coordenadas da mão esquerda.</span><span class="sxs-lookup"><span data-stu-id="317b2-119">X3DAudio uses left-handed coordinates.</span></span> <span data-ttu-id="317b2-120">Para usar com as coordenadas da mão direita, os desenvolvedores precisam negar o elemento. z dos membros OrientTop, OrientFront, position e Velocity de X3DAUDIO \_ Listener e X3DAUDIO \_ emissor.</span><span class="sxs-lookup"><span data-stu-id="317b2-120">To use with right-handed coordinates, developers need to negate the .z element of the OrientTop, OrientFront, Position, and Velocity members of X3DAUDIO\_LISTENER and X3DAUDIO\_EMITTER.</span></span>

<span data-ttu-id="317b2-121">Além da posição, os ouvintes e os emissores podem incluir a velocidade.</span><span class="sxs-lookup"><span data-stu-id="317b2-121">In addition to position, listeners and emitters can include velocity.</span></span> <span data-ttu-id="317b2-122">Ao contrário de um mecanismo de renderização 3D, o X3DAudio usa apenas a velocidade para calcular efeitos de Doppler (não é usado para calcular a posição).</span><span class="sxs-lookup"><span data-stu-id="317b2-122">Unlike a 3D rendering engine, X3DAudio only uses velocity to calculate Doppler effects (it is not used to calculate position).</span></span>

<span data-ttu-id="317b2-123">Para obter mais detalhes sobre ouvintes e emissores, consulte os tópicos de referência de estrutura do [**\_ emissor**](/windows/desktop/api/x3daudio/ns-x3daudio-x3daudio_emitter) do [**X3DAUDIO \_ Listener**](/windows/desktop/api/x3daudio/ns-x3daudio-x3daudio_listener) e X3DAUDIO.</span><span class="sxs-lookup"><span data-stu-id="317b2-123">For more details about listeners and emitters, see the [**X3DAUDIO\_LISTENER**](/windows/desktop/api/x3daudio/ns-x3daudio-x3daudio_listener) and [**X3DAUDIO\_EMITTER**](/windows/desktop/api/x3daudio/ns-x3daudio-x3daudio_emitter) structure reference topics.</span></span>
