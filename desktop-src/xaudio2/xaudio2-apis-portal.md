# <a name="xaudio2-apis"></a>APIs XAudio2

## <a name="purpose"></a>Finalidade

XAudio2 é uma API de áudio de baixo nível que fornece processamento de sinal e combinação de base para o desenvolvimento de mecanismos de áudio de alto desempenho para jogos.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs do XAudio2 são projetadas para serem usadas por desenvolvedores de jogos que desejam melhorar o desempenho em seus jogos.
