# <a name="proof-of-possession-cookie-api"></a>API do cookie de prova de posse

Permite criar cookies de prova de posse, que dão suporte ao logon único para aplicativos Web autenticados pelo protocolo de autenticação da Internet OAuth. Você pode usar as informações retornadas por essa API para implementar o formulário de logon único do Microsoft accounts.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                         | Descrição                                                      |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**IProofOfPossessionCookieInfoManager**](/windows/desktop/api/ProofOfPossessionCookieInfo/nn-proofofpossessioncookieinfo-iproofofpossessioncookieinfomanager)<br/> | Dá suporte à criação de cookies de prova de posse.<br/> |
