# <a name="proof-of-possession-cookie-api"></a>API de cookie de prova de posse

Permite que você crie cookies de prova de posse, que suportam o login único para aplicativos Web autenticados pelo protocolo de autenticação de Internet OAuth. Você pode usar as informações retornadas por essa API para implementar contas microsoft de formulário de login único.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                         | Descrição                                                      |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**IProofOfPossessionCookieInfoManager**](/windows/desktop/api/ProofOfPossessionCookieInfo/nn-proofofpossessioncookieinfo-iproofofpossessioncookieinfomanager)<br/> | Dá suporte à criação de cookies de prova de posse.<br/> |
