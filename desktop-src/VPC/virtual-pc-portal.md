# <a name="windows-virtual-pc"></a>Windows Virtual PC

\[Windows O Virtual PC não está mais disponível para uso a partir de Windows 8. Em vez disso, use o [provedor WMI do Hyper-V (v2)](../hyperv_v2/windows-virtualization-portal.md).\]

## <a name="purpose"></a>Finalidade

esta documentação fornece informações sobre a interface COM do Windows Virtual PC.

## <a name="where-applicable"></a>Quando aplicável

Windows O Virtual PC é a mais recente tecnologia de virtualização da Microsoft; ele permite que você execute vários aplicativos de produtividade em um ambiente de Windows virtual, com um único clique, diretamente do Windows 7. você pode criar máquinas virtuais separadas sobre sua área de trabalho do Windows 7. Cada máquina virtual emula um sistema de hardware completo — do processador para a placa de rede — em um ambiente de software isolado e independente, permitindo a operação simultânea de sistemas incompatíveis de outra forma.

## <a name="developer-audience"></a>Público de desenvolvedores

as interfaces COM do Windows Virtual PC são para os desenvolvedores que estão criando aplicativos cliente que automatizam a implantação e a operação de máquinas virtuais.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Windows o Virtual PC requer uma das seguintes Windows 7 edições-home Basic, home Premium, Professional, Ultimate ou Enterprise edition.

para obter os arquivos de cabeçalho e biblioteca relacionados, instale o SDK do Windows do Windows 7 no [centro de Download da Microsoft](https://www.microsoft.com/download/details.aspx?id=8279).
