# <a name="ivmvirtualmachinedvdromdrives-property"></a>IVMVirtualMachine: Propriedade VDROMDrives de:D

\[O Windows Virtual PC não está mais disponível para uso a partir do Windows 8. Em vez disso, use o [provedor WMI do Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Recupera uma coleção enumerável de unidades de CD e DVD conectadas à máquina virtual.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_DVDROMDrives(
  [out, retval] IVMDVDDriveCollection **dvdROMCollection
);
```



## <a name="property-value"></a>Valor da propriedade

Um objeto [**IVMDVDDriveCollection**](ivmdvddrivecollection.md) .
