# <a name="ivmvirtualmachineaccountant-property"></a>Propriedade IVMVirtualMachine::Desintente

\[Windows O PC virtual não está mais disponível para uso a partir Windows 8. Em vez disso, use o provedor WMI do [Hyper-V (V2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Recupera um contador para essa máquina virtual.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_Accountant(
  [out, retval] IVMAccountant **accountant
);
```



## <a name="property-value"></a>Valor da propriedade

O [**objeto IVMAccountant.**](ivmaccountant.md)
