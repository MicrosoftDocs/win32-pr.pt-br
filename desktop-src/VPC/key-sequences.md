<span data-ttu-id="7035d-107">Uma cadeia de caracteres de sequência de chave é um conjunto delimitado por vírgulas de identificadores de chave que são usados para simular o pressionamento de tecla e a sequência de versão de um teclado em estilo padrão dos EUA 101.</span><span class="sxs-lookup"><span data-stu-id="7035d-107">A key sequence string is a comma-delimited set of key identifiers which are used to simulate the key press and release sequence of a standard U.S. 101-key AT-style keyboard.</span></span>

<span data-ttu-id="7035d-108">Se um identificador de chave aparecer na cadeia de caracteres sem um modificador anterior, um código de chave pressionada será simulado, seguido imediatamente pelo seu código de chave liberada correspondente.</span><span class="sxs-lookup"><span data-stu-id="7035d-108">If a key identifier appears in the string without a preceding modifier, a key-pressed code is simulated, followed immediately by its corresponding key-released code.</span></span> <span data-ttu-id="7035d-109">Os modificadores de chave podem ser usados para alterar esse comportamento.</span><span class="sxs-lookup"><span data-stu-id="7035d-109">Key modifiers can be used to change this behavior.</span></span>

<span data-ttu-id="7035d-110">Por exemplo, o modificador **down** enviará o código de chave pressionada para o identificador de chave a seguir sem enviar o código de lançamento de chave.</span><span class="sxs-lookup"><span data-stu-id="7035d-110">For example, the **DOWN** modifier will send the key-pressed code for the following key identifier without sending the key-released code.</span></span> <span data-ttu-id="7035d-111">Isso é útil para simular as teclas CTRL, ALT e Shift quando elas são mantidas enquanto outras chaves estão sendo enviadas.</span><span class="sxs-lookup"><span data-stu-id="7035d-111">This is useful for simulating Ctrl, Alt, and Shift keys when they are held down while other keys are being sent.</span></span> <span data-ttu-id="7035d-112">Para liberar a chave, ela deve ser incluída na cadeia de caracteres de chave novamente junto com um modificador **up** anterior.</span><span class="sxs-lookup"><span data-stu-id="7035d-112">To release the key, it must be included in the key string again along with a preceding **UP** modifier.</span></span>
