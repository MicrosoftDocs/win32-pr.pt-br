# <a name="ivmvirtualmachineharddiskconnections-property"></a>Propriedade IVMVirtualMachine::HardDiskConnections

\[Windows O PC virtual não está mais disponível para uso a partir Windows 8. Em vez disso, use o provedor WMI do [Hyper-V (V2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Recupera uma coleção enumerável de conexões de disco rígido.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_HardDiskConnections(
  [out, retval] IVMHardDiskConnectionCollection **hardDiskConnectionCollection
);
```



## <a name="property-value"></a>Valor da propriedade

Um [**objeto IVMHardDiskConnectionCollection.**](ivmharddiskconnectioncollection.md)
