# <a name="pointer-device-input-stack"></a>Pilha de entrada do dispositivo ponteiro

## <a name="purpose"></a>Finalidade

Os tópicos nesta seção fornecem uma visão geral da pilha de entrada do dispositivo de ponteiro do Windows. As APIs de [pilha de entrada do dispositivo ponteiro](/windows/win32/api/_input_pointerdevice/) fornecem informações sobre os dispositivos de entrada conectados ao sistema.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|---|---|
| [Referência de pilha de entrada do dispositivo ponteiro](unified-input-stack-reference.md)<br/> | Os tópicos nesta seção fornecem as especificações de referência para a pilha de entrada do dispositivo ponteiro no Windows 8.<br/> |

## <a name="developer-audience"></a>Público de desenvolvedores

Os StackAPIs de entrada do dispositivo ponteiro são projetados para desenvolvedores que precisam de informações sobre o dispositivo ponteiro que está sendo usado para interagir com seus aplicativos.
