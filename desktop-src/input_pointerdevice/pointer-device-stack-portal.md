# <a name="pointer-device-input-stack"></a>Pilha de entrada do dispositivo de ponteiro

## <a name="purpose"></a>Finalidade

Os tópicos nesta seção fornecem uma visão geral da pilha de Windows de entrada do dispositivo de ponteiro. As APIs [da Pilha de](/windows/win32/api/_input_pointerdevice/) Entrada do Dispositivo de Ponteiro fornecem informações sobre os dispositivos de entrada conectados ao sistema.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|---|---|
| [Referência da pilha de entrada do dispositivo de ponteiro](unified-input-stack-reference.md)<br/> | Os tópicos nesta seção fornecem as especificações de referência para a Pilha de Entrada do Dispositivo de Ponteiro no Windows 8.<br/> |

## <a name="developer-audience"></a>Público de desenvolvedores

As StackAPIs de Entrada do Dispositivo de Ponteiro são projetadas para desenvolvedores que exigem informações sobre o dispositivo de ponteiro que está sendo usado para interagir com seu aplicativo.
