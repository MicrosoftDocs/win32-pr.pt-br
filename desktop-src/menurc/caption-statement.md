# <a name="caption-statement"></a>Instrução de legenda

Define o título de uma caixa de diálogo. O título aparece na barra de legenda da caixa (se tiver uma).

A legenda padrão está vazia.

``` syntax
CAPTION "captiontext"
```

<dl> <dt>

<span id="captiontext"></span><span id="CAPTIONTEXT"></span>*captiontext*
</dt> <dd>

Uma cadeia de caracteres entre aspas duplas (").

</dd> </dl>

## <a name="examples"></a>Exemplos

O exemplo a seguir demonstra o uso da instrução **Caption** :

``` syntax
CAPTION "Error!" 
```
