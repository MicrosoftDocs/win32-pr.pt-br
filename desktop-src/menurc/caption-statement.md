# <a name="caption-statement"></a>Instrução CAPTION

Define o título de uma caixa de diálogo. O título aparece na barra de legendas da caixa (se tiver um).

A legenda padrão está vazia.

``` syntax
CAPTION "captiontext"
```

<dl> <dt>

<span id="captiontext"></span><span id="CAPTIONTEXT"></span>*Captiontext*
</dt> <dd>

Uma cadeia de caracteres entre aspas duplas (").

</dd> </dl>

## <a name="examples"></a>Exemplos

O exemplo a seguir demonstra o uso da **instrução CAPTION:**

``` syntax
CAPTION "Error!" 
```
