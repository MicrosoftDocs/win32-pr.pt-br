# <a name="displaying-progress-messages"></a>Exibindo mensagens de progresso

Por padrão, o RC é compilado sem exibir as mensagens de progresso. No entanto, você pode especificar que o RC é para exibir as mensagens de progresso. Para fazer isso, use a opção **/v** .

O exemplo a seguir faz com que o RC relate seu progresso à medida que compila o arquivo de definição de recurso Sample. rc e cria o arquivo de recurso compilado Sample. res:

**RC/v Sample. rc**
