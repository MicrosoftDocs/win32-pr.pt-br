# <a name="displaying-progress-messages"></a>Exibindo mensagens de progresso

Por padrão, o RC é compilado sem exibir mensagens de progresso. No entanto, você pode especificar que o RC deve exibir mensagens de progresso. Para fazer isso, use a **opção /v.**

O exemplo a seguir faz com que o RC reporte seu progresso à medida que compila o arquivo de definição de recurso Sample.rc e cria o arquivo de recurso compilado Sample.res:

**rc /v sample.rc**
