# <a name="resource-compiler"></a>Compilador de Recurso

O compilador de recursos do Microsoft Windows (RC) é uma ferramenta usada na criação de aplicativos baseados no Windows. Esta visão geral descreve como criar um arquivo de definição de recurso (script), como compilar os recursos do aplicativo e como adicionar recursos compilados ao seu aplicativo.

Essa ferramenta está disponível no Visual Studio e no SDK (Software Development Kit) do Microsoft Windows.

Os tópicos a seguir descrevem a sintaxe do arquivo de recursos e a linha de comando RC:

-   [Sobre arquivos de recurso](about-resource-files.md)
-   [Usando o RC (a linha de comando do RC)](using-rc-the-rc-command-line-.md)
