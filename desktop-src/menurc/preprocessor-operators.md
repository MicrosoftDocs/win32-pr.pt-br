# <a name="preprocessor-operators"></a>Operadores de pré-processador

O RC dá suporte ao uso dos operadores de pré-processamento padrão do C nas definições de macro. Esses operadores são descritos na tabela a seguir.



| Operador | Descrição                                                 |
|----------|-------------------------------------------------------------|
| \#       | Inclui o argumento entre aspas.                            |
| \#@      | Inclui o argumento entre aspas simples.                     |
| \#\#     | Concatena tokens usados como argumentos para formar outros tokens. |
