# <a name="menus-and-other-resources"></a>Menus e outros recursos

Um *recurso* são dados binários que você pode adicionar ao arquivo executável de um Windows baseado em dados. Um recurso pode ser padrão ou definido. Os dados  em um recurso padrão descrevem um ícone, cursor, menu, caixa de diálogo, bitmap, metadados aprimorados, fonte, tabela de acelerador, entrada de tabela de mensagens, entrada de tabela de cadeia de caracteres ou informações de versão. Um *recurso definido pelo aplicativo*, também chamado de recurso *personalizado,* contém todos os dados exigidos por um aplicativo específico.
