# <a name="endif"></a>\#endif

A **\# diretiva endif** marca o final de um bloco de compilação condicional definido por uma **\# diretiva ifdef.** Um **\# endif** é necessário para cada **\# diretiva if**, **\# ifdef** ou **\# ifndef.**

``` syntax
#endif
```

Essa diretiva não tem parâmetros.
