# <a name="endif"></a>\#endif

A diretiva **\# endif** marca o final de um bloco de compilação condicional definido por uma diretiva **\# ifdef** . É necessário um **\# endif** para cada diretiva **\# If**, **\# ifdef** ou **\# ifndef** .

``` syntax
#endif
```

Essa diretiva não tem parâmetros.
