# <a name="package-resource-indexing-pri-reference"></a>Referência ao índice de recurso do pacote (PRI)

Um conjunto de APIs para trabalhar com um indexador de recursos. Um indexador de recursos é usado para gerar arquivos PRI (índice de recursos de pacote) para um aplicativo UWP. Para obter mais informações e passo a passo baseado em cenário de como usar essas APIs, consulte [APIs de PRI (indexação](/windows/uwp/app-resources/pri-apis-custom-build-systems)de recursos de pacote) e sistemas de build personalizados .
