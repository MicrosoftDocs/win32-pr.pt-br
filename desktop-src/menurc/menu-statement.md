# <a name="menu-statement"></a>Instrução de MENU

Define o menu para uma caixa de diálogo. Se nenhuma instrução for fornecida, a caixa de diálogo não terá nenhum menu.

``` syntax
MENU menuname
```

<dl> <dt>

<span id="menuname"></span><span id="MENUNAME"></span>*menuname*
</dt> <dd>

O nome ou o identificador do menu a ser usado.

</dd> </dl>

## <a name="examples"></a>Exemplos

O exemplo a seguir demonstra o uso da instrução de diálogo de **menu** :

``` syntax
MENU errmenu
```
