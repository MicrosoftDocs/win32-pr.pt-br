# <a name="i-menus-and-other-resources"></a>I (menus e outros recursos)

[A](a.md) [B](b.md) [C](c.md) D [E](e.md) [F G](f.md) H I J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) [U](u.md) [V](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**Acelerador inválido**
</dt> <dd>

Um *parâmetro* de evento na [**instrução ACCELERATORS**](accelerators-resource.md) não foi reconhecido ou tinha mais de dois caracteres.

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**Tipo de acelerador inválido (ASCII ou VIRTKEY)**
</dt> <dd>

O *parâmetro* type na [**instrução ACCELERATORS**](accelerators-resource.md) deve conter o **valor ASCII** ou **VIRTKEY.**

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**Caractere de controle inválido**
</dt> <dd>

Um caractere de controle na [**instrução ACCELERATORS**](accelerators-resource.md) é inválido. Um caractere de controle válido consiste em um a caret (^) seguido por uma única letra.

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**Tipo de controle inválido**
</dt> <dd>

A instrução control em uma instrução [**DIALOG**](dialog-resource.md) deve ser uma das seguintes: [**AUTO3STATE**](auto3state-control.md), [**AUTOCHECKBOX**](autocheckbox-control.md), [**AUTORADIOBUTTON**](autoradiobutton-control.md), [**CHECKBOX**](checkbox-control.md), [**COMBOBOX**](combobox-control.md), [**CONTROL**](control-control.md), [**CTEXT**](ctext-control.md), [**DEFPTEXTBUTTON**](defpushbutton-control.md), [**EDITTEXT**](edittext-control.md), [**GROUPBOX**](groupbox-control.md), [**ICON**](icon-control.md), [**LISTBOX**](listbox-control.md), [**LTEXT**](ltext-control.md), [**PUSHBOX**](pushbox-control.md), [**PUSHBUTTON**](pushbutton-control.md), [**RADIOBUTTON**](radiobutton-control.md), [**RTEXT**](rtext-control.md), [**SCROLLBAR**](scrollbar-control.md), [**STATE3**](state3-control.md).

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**Diretiva inválida no arquivo RC pré-processado**
</dt> <dd>

O nome de arquivo especificado tem um caractere de nova linha inserido.

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**Arquivo .EXE inválido**
</dt> <dd>

O arquivo executável (.exe) é inválido. Certifique-se de que o linker o criou corretamente e se o arquivo existe.

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**Opção inválida**
</dt> <dd>

Uma opção usada era inválida. Para uma lista das opções de linha de comando, use **rc /?** .

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**Comutadores inválidos, comutadores -D demais**
</dt> <dd>

Foram especificadas **muitas opções -d.** Para definir um grande número de símbolos, use a [**\# diretiva include**](-include.md) para incluir um arquivo de definições.

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**Tipo inválido**
</dt> <dd>

O tipo de recurso não estava entre os tipos definidos no arquivo de inclusão.

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**Uso inválido. Usar RC -? para ajuda**
</dt> <dd>

Certifique-se de que você tenha pelo menos um nome de arquivo com o qual trabalhar. Para uma lista das opções de linha de comando, use **rc -?** .

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**Arquivo de leitura de erro de E/S**
</dt> <dd>

Falha na leitura. Como essa é uma rotina genérica, nenhum nome de arquivo específico é fornecido.

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**Erro de E/S em busca de arquivo**
</dt> <dd>

Falha na busca no arquivo. Como essa é uma rotina genérica, nenhum nome de arquivo específico é fornecido.

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**Erro de E/S ao escrever arquivo**
</dt> <dd>

Falha na gravação. Como essa é uma rotina genérica, nenhum nome de arquivo específico é fornecido.

</dd> </dl>
