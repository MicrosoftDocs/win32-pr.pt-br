# <a name="i-menus-and-other-resources"></a>I (menus e outros recursos)

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**Acelerador inválido**
</dt> <dd>

Um parâmetro de *evento* na instrução [**Accelerators**](accelerators-resource.md) não foi reconhecido ou tinha mais de dois caracteres.

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**Tipo de acelerador inválido (ASCII ou VIRTKEY)**
</dt> <dd>

O parâmetro de *tipo* na instrução [**Accelerators**](accelerators-resource.md) deve conter o valor **ASCII** ou **VIRTKEY** .

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**Caractere de controle inválido**
</dt> <dd>

Um caractere de controle na instrução [**Accelerators**](accelerators-resource.md) é inválido. Um caractere de controle válido consiste em um acento circunflexo (^) seguido por uma única letra.

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**Tipo de controle inválido**
</dt> <dd>

A instrução de controle em uma instrução de [**caixa de diálogo**](dialog-resource.md) deve ser uma das seguintes: [**AUTO3STATE**](auto3state-control.md), [**autocaixa de seleção**](autocheckbox-control.md), [**AUTORADIOBUTTON**](autoradiobutton-control.md), [**CheckBox**](checkbox-control.md), [**ComboBox**](combobox-control.md), [**Control**](control-control.md), [**CTEXT**](ctext-control.md), [**DEFPUSHBUTTON**](defpushbutton-control.md), [**EDITTEXT**](edittext-control.md), [**GroupBox**](groupbox-control.md), [**Icon**](icon-control.md), [**ListBox**](listbox-control.md), [**LTEXT**](ltext-control.md), [**PUSHBOX**](pushbox-control.md), sutrato [**,**](pushbutton-control.md) [**RadioButton**](radiobutton-control.md), [**RTEXT**](rtext-control.md), [**ScrollBar**](scrollbar-control.md), [**STATE3**](state3-control.md).

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**Diretiva inválida no arquivo RC pré-processado**
</dt> <dd>

O nome de arquivo especificado tem um caractere de nova linha inserido.

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**Inválido. Arquivo EXE**
</dt> <dd>

O arquivo executável (. exe) é inválido. Certifique-se de que o vinculador o criou corretamente e que o arquivo existe.

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**Opção de opção inválida**
</dt> <dd>

Uma opção usada era inválida. Para obter uma lista das opções de linha de comando, use a versão **RC/?** .

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**Opção inválida, muitas opções-D**
</dt> <dd>

Foram especificadas opções **de-d em** excesso. Para definir um grande número de símbolos, use a diretiva [**\# include**](-include.md) para incluir um arquivo de definições.

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**Tipo inválido**
</dt> <dd>

O tipo de recurso não estava entre os tipos definidos no arquivo de inclusão.

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**Uso inválido. Usar RC-? para obter ajuda**
</dt> <dd>

Verifique se você tem pelo menos um nome de arquivo com o qual trabalhar. Para obter uma lista das opções de linha de comando, use o **RC-?** .

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**Erro de e/s ao ler arquivo**
</dt> <dd>

Falha na leitura. Como essa é uma rotina genérica, nenhum nome de arquivo específico é fornecido.

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**Erro de e/s ao procurar no arquivo**
</dt> <dd>

Falha ao procurar no arquivo. Como essa é uma rotina genérica, nenhum nome de arquivo específico é fornecido.

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**Erro de e/s ao gravar arquivo**
</dt> <dd>

Falha na gravação. Como essa é uma rotina genérica, nenhum nome de arquivo específico é fornecido.

</dd> </dl>
