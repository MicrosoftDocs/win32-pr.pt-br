# <a name="rc-diagnostic-messages"></a>Mensagens de diagnóstico RC

Esta seção descreve as mensagens de diagnóstico produzidas pelo RC. Muitas dessas mensagens aparecem quando o RC não é capaz de Compilar recursos corretamente. As descrições esclarecem a causa de cada mensagem.

Um V maiúsculo entre parênteses (V) no início de uma descrição da mensagem indica que a mensagem será exibida somente se o RC for executado com a opção **/v** (verbose). Essas mensagens são geralmente informativas e não necessariamente indicam erros.

As mensagens são listadas em ordem alfabética nos seguintes tópicos:

[A](a.md) [B](b.md) [C](c.md) [E](e.md) [F](f.md) [i](i.md) [N](n.md) [O](o.md) [R](r.md) [T](t.md) [U](u.md) [V](v.md) [W](w.md)
