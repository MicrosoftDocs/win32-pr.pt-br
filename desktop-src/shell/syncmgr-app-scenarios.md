# <a name="application-scenarios"></a>Cenários de aplicativo

Os aplicativos e serviços que podem usar o Gerenciador de sincronização incluem o seguinte:

-   Microsoft Office aplicativos que precisam preparar arquivos para uso offline.
-   Cache do lado do cliente que permite armazenar arquivos em cache localmente.
-   Navegadores que podem armazenar em cache páginas HTML localmente.
-   Programas de email que podem armazenar em cache mensagens de email localmente.
-   Bancos de dados que podem armazenar informações localmente.
