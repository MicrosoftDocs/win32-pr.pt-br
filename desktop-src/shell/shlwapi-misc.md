# <a name="shell-miscellaneous-utility-functions"></a>Funções de utilitário diversos do Shell

Esta seção descreve as funções do shell do Windows exportadas por Shlwapi.dll e definidas em Shlwapi. h e shlwapi. lib que não se enquadram em uma das outras categorias de função do utilitário.

| Tópico                                                                   | Descrição                                                                                                                                                                                                                                                                      |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| [**GetProcessReference**](/windows/desktop/api/Shlwapi/nf-shlwapi-getprocessreference)<br/>           | Recupera o objeto específico do processo fornecido pelo [**SetProcessReference**](/windows/desktop/api/Shlwapi/nf-shlwapi-setprocessreference), incrementando a contagem de referência para manter o processo ativo.<br/>                                                                                                     |

| [**IsOS**](/windows/desktop/api/Shlwapi/nf-shlwapi-isos)<br/>                                         | Verifica os sistemas operacionais especificados e os recursos do sistema operacional.<br/>                                                                                                                                                                                                 |

| [**Redefinição de IStream \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-istream_reset)<br/>                      | Move a posição de busca em um fluxo especificado para o início do fluxo.<br/>                                                                                                                                                                                         |
| [**Tamanho de IStream \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-istream_size)<br/>                        | Recupera o tamanho, em bytes, de um fluxo especificado.<br/>                                                                                                                                                                                                                  |
| [**Gravação de IStream \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-istream_write)<br/>                      | Grava dados de formato desconhecido de um buffer para um fluxo especificado.<br/>                                                                                                                                                                                                    |
| [**WritePidl de IStream \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-istream_writepidl)<br/>              | Grava um PIDL de um \_ objeto relativo PCUIDLIST em um objeto [**IStream**](/windows/win32/api/objidl/nn-objidl-istream) .<br/>                                                                                                                                                                            |
| [**WriteStr de IStream \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-istream_writestr)<br/>                | Lê de uma cadeia de caracteres e grava em um fluxo.<br/>                                                                                                                                                                                                                         |
| [**\_AtomicRelease IUnknown**](/windows/desktop/api/Shlwapi/nf-shlwapi-iunknown_atomicrelease)<br/>    | Libera um ponteiro de Component Object Model (COM) e o define como **nulo**.<br/>                                                                                                                                                                                              |
| [**GetSite IUnknown \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-iunknown_getsite)<br/>                | Chama o método [**IObjectWithSite:: GetSite**](/windows/win32/api/ocidl/nf-ocidl-iobjectwithsite-getsite) do objeto especificado.<br/>                                                                                                                                                                      |
| [**GetWindow IUnknown \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-iunknown_getwindow)<br/>            | Tenta recuperar um identificador de janela de um objeto COM consultando várias interfaces que têm um método **GetWindow** .<br/>                                                                                                                                           |
| [**QueryService de IUnknown \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-iunknown_queryservice)<br/>      | Recupera uma interface para um serviço de um objeto especificado.<br/>                                                                                                                                                                                                         |
| [**Conjunto de IUnknown \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-iunknown_set)<br/>                        | Altera o valor de um ponteiro de interface COM e libera a interface anterior.<br/>                                                                                                                                                                                     |
| [**SetSite IUnknown \_**](/windows/desktop/api/Shlwapi/nf-shlwapi-iunknown_setsite)<br/>                | Define o site do objeto especificado chamando seu método [**IObjectWithSite:: SetSite**](/windows/win32/api/ocidl/nf-ocidl-iobjectwithsite-setsite) .<br/>                                                                                                                                                   |

| [**QISearch**](/windows/desktop/api/Shlwapi/nf-shlwapi-qisearch)<br/>                                 | Uma implementação orientada por tabela do método [**IUnknown:: QueryInterface**](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) .<br/>                                                                                                                                                              |
| [**SetProcessReference**](/windows/desktop/api/Shlwapi/nf-shlwapi-setprocessreference)<br/>           | Fornece um objeto COM que permite que extensões de shell hospedadas e outros componentes impeçam que seu processo de host seja fechado prematuramente. O processo de host é normalmente o Windows Explorer ou o Internet Explorer, mas essa função também pode ser usada por outros aplicativos.<br/> |
| [**SHAutoComplete**](/windows/win32/api/shlwapi/nf-shlwapi-shautocomplete)<br/>                   | Instrui os controles de edição do sistema a usar o preenchimento automático para ajudar a concluir URLs ou caminhos do sistema de arquivos.<br/>                                                                                                                                                                        |
| [**SHCreateMemStream**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreatememstream)<br/>               | Cria um fluxo de memória usando um processo semelhante a [**CreateStreamOnHGlobal**](/windows/win32/api/combaseapi/nf-combaseapi-createstreamonhglobal).<br/>                                                                                                                                                            |
| [**SHCreateStreamOnFileEx**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreatestreamonfileex)<br/>     | Abre ou cria um arquivo e recupera um fluxo para ler ou gravar nesse arquivo.<br/>                                                                                                                                                                                         |
| [**SHCreateThread**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreatethread)<br/>                     | Cria um thread.<br/>                                                                                                                                                                                                                                                     |
| [**SHCreateThreadRef**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreatethreadref)<br/>               | Cria uma referência por thread a um objeto COM.<br/>                                                                                                                                                                                                                       |
| [**SHCreateThreadWithHandle**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreatethreadwithhandle)<br/> | Cria um novo thread e recupera seu identificador.<br/>                                                                                                                                                                                                                        |
| [**SHGetThreadRef**](/windows/desktop/api/Shlwapi/nf-shlwapi-shgetthreadref)<br/>                     | Recupera a referência de objeto por thread definida por [**SHSetThreadRef**](/windows/desktop/api/Shlwapi/nf-shlwapi-shsetthreadref).<br/>                                                                                                                                                                            |
| [**SHGlobalCounterDecrement**](/windows/desktop/api/Shlwapi/nf-shlwapi-shglobalcounterdecrement)<br/> | Decrementa um contador global.<br/>                                                                                                                                                                                                                                          |
| [**SHGlobalCounterGetValue**](/windows/desktop/api/Shlwapi/nf-shlwapi-shglobalcountergetvalue)<br/>   | Obtém o valor atual de um contador global.<br/>                                                                                                                                                                                                                           |
| [**SHGlobalCounterIncrement**](/windows/desktop/api/Shlwapi/nf-shlwapi-shglobalcounterincrement)<br/> | Incrementa um contador global.<br/>                                                                                                                                                                                                                                          |
| [**SHIsLowMemoryMachine**](/windows/desktop/api/shlwapi/nf-shlwapi-shislowmemorymachine)<br/>         |                                                                                                                                                                                                                                                                                  |
| [**SHReleaseThreadRef**](/windows/desktop/api/Shlwapi/nf-shlwapi-shreleasethreadref)<br/>             | Libera uma referência de thread antes que o procedimento de thread seja retornado.<br/>                                                                                                                                                                                                      |
| [**SHSetThreadRef**](/windows/desktop/api/Shlwapi/nf-shlwapi-shsetthreadref)<br/>                     | Armazena uma referência por thread a um objeto COM. Isso permite que o chamador controle o tempo de vida do thread para que ele possa garantir que o Windows não desligará o thread antes que o chamador esteja pronto.<br/>                                                                      |
