## <a name="remarks"></a>Comentários

Os valores de **FOLDERTYPEID** são GUIDs e são declarados em Shlguid. h.

Na prática, qualquer valor de **FOLDERTYPEID** pode ser aplicado a qualquer pasta, independentemente do conteúdo dessa pasta. O **FOLDERTYPEID** é apenas um modelo de exibição, fornecendo o layout apropriado e as colunas para o conteúdo de pasta esperado. Não há nenhuma verificação do conteúdo da pasta em relação ao valor **FOLDERTYPEID** para garantir a adequação.

Esses e outros tipos de pasta são definidos no registro na subchave **FolderTypes** , que está localizada como mostrado aqui:

```
HKEY_LOCAL_MACHINE
   SOFTWARE
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  FolderTypes
```
