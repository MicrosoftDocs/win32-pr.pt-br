# <a name="customizing-icons"></a>Personalizando ícones

O Windows fornece ícones padrão para cada item exibido na área de trabalho e no Windows Explorer. Dispositivos de armazenamento em massa, como unidades de disco, também são atribuídos a ícones padrão. No entanto, esses ícones geralmente fornecem pouca Insight ao usuário quanto ao conteúdo do arquivo ou a qual programa está associado. Você pode atribuir um ícone personalizado a uma pasta do sistema de arquivos criando um arquivo de [Desktop.ini](how-to-customize-folders-with-desktop-ini.md) . Consulte os documentos a seguir para saber como usar o registro para associar ícones personalizados a tipos de arquivo e letras de unidade.

-   [Atribuir um ícone personalizado a um tipo de arquivo](how-to-assign-a-custom-icon-to-a-file-type.md)
-   [Atribuir um ícone e um rótulo personalizados a uma letra da unidade](how-to-assign-a-custom-icon-and-label-to-a-drive-letter.md)
