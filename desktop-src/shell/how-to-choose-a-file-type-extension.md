# <a name="how-to-choose-a-file-type-extension"></a>Como escolher uma extensão de tipo de arquivo

Os tipos de arquivo são identificados por uma extensão de nome de arquivo. Antes de definir um novo tipo de arquivo, primeiro você deve escolher uma extensão de nome de arquivo.

## <a name="instructions"></a>Instruções

### <a name="step-1"></a>Etapa 1:

Escolha uma extensão de nome de arquivo longo que você deseja usar para identificar o tipo de arquivo. Para obter mais informações, consulte [manipuladores de tipo de arquivo](fa-file-extensions.md).

### <a name="step-2"></a>Etapa 2:

Verifique o banco de dados MIME da [IANA (Internet Assigned Numbers Authority)](https://www.iana.org/assignments/media-types) para determinar se a extensão de nome de arquivo já está associada a outro aplicativo.

### <a name="step-3"></a>Etapa 3:

Defina um tipo de MIME ao definir um novo tipo de arquivo público e registre-o com o IANA.
