# <a name="sfvm_backgroundenumdone-message"></a>\_Mensagem SFVM BACKGROUNDENUMDONE

Notifica o objeto de retorno de chamada que a enumeração em segundo plano está concluída. Usado por [**IShellFolderViewCB:: MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++

                SFVM_BACKGROUNDENUMDONE

            
```



## <a name="parameters"></a>Parâmetros

Esta mensagem não tem parâmetros.
