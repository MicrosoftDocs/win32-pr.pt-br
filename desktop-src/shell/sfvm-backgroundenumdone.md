# <a name="sfvm_backgroundenumdone-message"></a>Mensagem BACKGROUNDENUMDONE de SFVM \_

Notifica o objeto de retorno de chamada de que a enumeração em segundo plano está concluída. Usado por [**IShellFolderViewCB::MessageSFVCB.**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb)


```C++

                SFVM_BACKGROUNDENUMDONE

            
```



## <a name="parameters"></a>Parâmetros

Essa mensagem não tem parâmetros.
