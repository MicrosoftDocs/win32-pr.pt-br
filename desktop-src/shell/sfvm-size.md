# <a name="sfvm_size-message"></a>Mensagem de tamanho de SFVM \_

Notifica o objeto de retorno de chamada de que a exibição de pasta foi redimensionada. Usado por [**IShellFolderViewCB:: MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Parâmetros

Esta mensagem não tem parâmetros.
