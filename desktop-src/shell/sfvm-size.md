# <a name="sfvm_size-message"></a>Mensagem SFVM \_ SIZE

Notifica o objeto de retorno de chamada de que a exibição de pasta foi ressalvada. Usado por [**IShellFolderViewCB::MessageSFVCB.**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb)


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Parâmetros

Essa mensagem não tem parâmetros.
