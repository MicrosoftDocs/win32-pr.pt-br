# <a name="virtualdesktopmanager-class"></a>Classe VirtualDesktopManager

**VirtualDesktopManager** implementa a interface [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager) .

## <a name="remarks"></a>Comentários

Use essa interface para interagir com as áreas de trabalho virtuais e controlar onde as janelas de programa são iniciadas. Para obter mais informações, consulte [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager).
