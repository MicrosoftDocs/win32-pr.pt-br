# <a name="iaclcustommru-interface"></a>Interface IACLCustomMRU

Expõe métodos que são usados para inicializar uma lista de MRU (usada mais recentemente) para um objeto de conclusão automática.

## <a name="members"></a>Membros

A interface **IACLCustomMRU** herda da interface [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **IACLCustomMRU** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IACLCustomMRU** tem esses métodos.



| Método                                             | Descrição                                                             |
|:---------------------------------------------------|:------------------------------------------------------------------------|
| [**AddMRUString**](iaclcustommru-addmrustring.md) | Adiciona uma entrada à lista MRU.<br/>                               |
| [**Inicializar**](iaclcustommru-initialize.md)     | Carrega uma lista de cadeias de caracteres na lista MRU do registro.<br/> |
