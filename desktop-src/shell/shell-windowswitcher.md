# <a name="shellwindowswitcher-method"></a>Método Shell:: WindowSwitcher

Exibe as janelas abertas em uma pilha 3D que você pode percorrer.

## <a name="syntax"></a>Sintaxe


```JScript
.WindowSwitcher()
```


```VB

Sub WindowSwitcher()
```





## <a name="parameters"></a>Parâmetros
