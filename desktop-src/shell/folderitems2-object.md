# <a name="folderitems2-object"></a>Objeto FolderItems2

Estende o objeto [**FolderItems**](folderitems.md) . Ele dá suporte a um método adicional.

## <a name="members"></a>Membros

O objeto **FolderItems2** tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

O objeto **FolderItems2** tem esses métodos.



| Método                                            | Descrição                                                                                                                                                                                                                                         |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InvokeVerbEx**](folderitems2-invokeverbex.md) | Executa um verbo em uma coleção de objetos [**FolderItem**](folderitem.md) . Esse método é uma extensão do método [**InvokeVerb**](folderitem-invokeverb.md) , permitindo um controle adicional da operação por meio de um conjunto de sinalizadores.<br/> |
