# <a name="newwdevents-object"></a>Objeto NewWDEvents

Estende o objeto [**WebWizardHost**](webwizardhost.md) habilitando páginas do lado do servidor hospedadas em um assistente para verificar se o usuário foi autenticado por meio de um conta Microsoft.

## <a name="members"></a>Membros

O objeto **NewWDEvents** tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

O objeto **NewWDEvents** tem esses métodos.



| Método                                                            | Descrição                                                                                                                         |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------|
| [**PassportAuthenticate**](inewwdevents-passportauthenticate.md) | Permite que as páginas do lado do servidor hospedadas em um assistente verifiquem se o usuário foi autenticado por meio de um conta Microsoft.<br/> |
