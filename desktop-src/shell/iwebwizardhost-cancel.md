# <a name="webwizardhostcancel-method"></a>Método WebWizardHost. Cancel

Simula um clique no botão **Cancelar** .

## <a name="syntax"></a>Sintaxe


```JScript
iRetVal = WebWizardHost.Cancel()
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="remarks"></a>Comentários

O cliente é responsável por responder a esse método com o comportamento esperado fechando o assistente.
