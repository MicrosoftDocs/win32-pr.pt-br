# <a name="folderitems3-object"></a>Objeto FolderItems3

Estende o objeto [**FolderItems2**](folderitems2-object.md) . Este objeto dá suporte a um método e uma propriedade adicionais.

## <a name="members"></a>Membros

O objeto **FolderItems3** tem estes tipos de membros:

-   [Métodos](#methods)
-   [Propriedades](#properties)

### <a name="methods"></a>Métodos

O objeto **FolderItems3** tem esses métodos.



| Método                                | Descrição                                                       |
|:--------------------------------------|:------------------------------------------------------------------|
| [**Filtrar**](folderitems3-filter.md) | Define um filtro curinga a ser aplicado aos itens retornados.<br/> |



 

### <a name="properties"></a>Propriedades

O objeto **FolderItems3** tem essas propriedades.



| Propriedade                                       | Tipo de acesso          | Description                                                       |
|:-----------------------------------------------|:---------------------|:------------------------------------------------------------------|
| [**Verbos**](folderitems3-verbs.md)<br/> | Somente leitura<br/> | Obtém a lista de verbos comuns a todos os itens de pasta.<br/> |
