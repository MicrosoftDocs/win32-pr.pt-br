# <a name="folderitemverb-object"></a>Objeto FolderItemVerb

Representa um único verbo disponível para um item. Este objeto contém propriedades e métodos que permitem que você recupere informações sobre o verbo.

## <a name="members"></a>Membros

O objeto **FolderItemVerb** tem estes tipos de membros:

-   [Métodos](#methods)
-   [Propriedades](#properties)

### <a name="methods"></a>Métodos

O objeto **FolderItemVerb** tem esses métodos.



| Método                              | Descrição                                                                                  |
|:------------------------------------|:---------------------------------------------------------------------------------------------|
| [**DoIt**](folderitemverb-doit.md) | Executa um verbo no [**FolderItem**](folderitem.md) associado ao verbo.<br/> |



 

### <a name="properties"></a>Propriedades

O objeto **FolderItemVerb** tem essas propriedades.



| Propriedade                                                     | Tipo de acesso          | Descrição                          |
|:-------------------------------------------------------------|:---------------------|:-------------------------------------|
| [**Aplicativo**](folderitemverb-application.md)<br/> | Somente leitura<br/> | Não implementado.<br/>          |
| [**Nome**](folderitemverb-name.md)<br/>               | Somente leitura<br/> | Contém o nome do verbo.<br/> |
| [**Pai**](folderitemverb-parent.md)<br/>           | Somente leitura<br/> | Não implementado.<br/>          |
