# <a name="diskquotacontrolquotafileincomplete-property"></a>Propriedade DiskQuotaControl. QuotaFileIncomplete

Obtém um valor booliano que indica se o arquivo de cota do volume está concluído.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
bQuotaFileIncomplete = DiskQuotaControl.QuotaFileIncomplete
```



## <a name="property-value"></a>Valor da propriedade

Essa propriedade será definida como **true** se o arquivo de cota estiver incompleto ou **false** caso contrário.
