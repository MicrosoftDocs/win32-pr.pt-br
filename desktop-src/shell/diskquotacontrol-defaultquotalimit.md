# <a name="diskquotacontroldefaultquotalimit-property"></a>Propriedade DiskQuotaControl. DefaultQuotaLimit

Define ou Obtém o limite de cota padrão.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```JScript
iDefaultQuotaLimit = DiskQuotaControl.DefaultQuotaLimit
DiskQuotaControl.DefaultQuotaLimit = iDefaultQuotaLimit
```



## <a name="property-value"></a>Valor da propriedade

Um valor **inteiro** que especifica ou recebe o limite de cota padrão para novos usuários, em bytes.
