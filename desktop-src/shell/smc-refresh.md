# <a name="smc_refresh-message"></a>Mensagem de atualização do SMC \_

Envia a notificação de que os menus foram completamente atualizados e você pode redefinir seu estado.


```C++
SMC_REFRESH
            
```



## <a name="parameters"></a>Parâmetros

Esta mensagem não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retornar S \_ OK.

## <a name="remarks"></a>Comentários

Essa notificação é recebida pelo método [**IShellMenuCallback:: CallbackSM**](/windows/desktop/api/shobjidl_core/nf-shobjidl_core-ishellmenucallback-callbacksm) .
