# <a name="about-synchronization-manager"></a>Sobre o Gerenciador de sincronização

O Gerenciador de sincronização fornece uma tecnologia padrão centralizada para sincronizar arquivos para uso offline em um computador local. O usuário pode preparar um computador para uso offline atualizando os arquivos necessários do armazenamento de rede. Em seguida, quando o computador volta a ficar online, todas as alterações offline podem ser copiadas de volta para a rede. O SyncMgr fornece uma interface do usuário comum que todos os aplicativos podem compartilhar para sincronizar seus arquivos.

Os arquivos são transferidos independentemente do protocolo. Por exemplo, um programa de email pode transferir suas mensagens usando SMTP, NMTP ou POP3, enquanto um navegador pode usar HTTP e um banco de dados pode usar RPC. O protocolo é transparente para o SyncMgr.

O SyncMgr também é independente de armazenamento e pode transferir arquivos, mensagens de email, páginas HTML ou informações de banco de dados de forma transparente.
