# <a name="using-hardware-autoplay"></a>Usando a Reprodução Automática de Hardware

O AutoPlay é um novo recurso introduido no Windows XP que detecta conteúdo como imagens, música ou arquivos de vídeo em mídia removível e dispositivos removíveis. O AutoPlay inicia automaticamente os aplicativos para reproduzir ou exibir esse conteúdo. Isso simplifica o uso de dispositivos periféricos especializados, como MPEG-Layer 3 (MP3) e leitores de fotos digitais. Ele também facilita para usuários que não estão familiarizados com o software necessário para acessar vários tipos de conteúdo.

Uma seleção expandida de ícones, incluindo ícones de marca para grandes fabricantes, é incluída para oferecer uma diferenciação visual aprimorada de unidades e outros dispositivos anexados **no Meu Computador**. Ícones para dispositivos com mídia removível, com exceção de unidades de disquete, são fornecidos em versões prontas e não prontas para indicar o estado atual do dispositivo. Isso confirma que um dispositivo conectado ao sistema está pronto para uso e ajuda a evitar mensagens de "dispositivo não pronto" de dispositivos sem mídia inserida.

> [!Note]  
> A Reprodução Automática não deve ser confundida com a AutoRun. A AutoRun permite que um compact disc iniciar automaticamente uma função, como um instalador ou jogo, quando colocado na unidade CD-ROM. Isso é feito por meio do uso de um arquivo Autorun.inf no diretório raiz do compact disc. Para obter mais detalhes, consulte [Criando um aplicativo CD-ROM habilitado para AutoRun.](autoplay.md)
