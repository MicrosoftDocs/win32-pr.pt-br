# <a name="using-hardware-autoplay"></a>Usando a reprodução automática de hardware

A reprodução automática é um novo recurso aprender no Windows XP que detecta conteúdo como imagens, música ou arquivos de vídeo em mídia removível e dispositivos removíveis. Em seguida, a reprodução automática inicia automaticamente os aplicativos para reproduzir ou exibir esse conteúdo. Isso simplifica o uso de dispositivos periféricos especializados, como players de MPEG-Layer 3 (MP3) e leitores de fotos digitais. Ele também torna mais fácil para os usuários que não estão familiarizados com o software necessário para acessar vários tipos de conteúdo.

Uma seleção expandida de ícones, incluindo ícones com marca para os principais fabricantes, está incluída para oferecer uma melhor diferenciação visual de unidades e outros dispositivos anexados no **meu computador**. Os ícones para dispositivos com mídia removível, com exceção de unidades de disquete, são fornecidos em versões prontas e não prontas para indicar o estado atual do dispositivo. Isso confirma que um dispositivo conectado de forma ativa ao sistema está pronto para uso e ajuda a evitar mensagens de "dispositivo não pronto" de dispositivos sem mídia inserida.

> [!Note]  
> A reprodução automática não deve ser confundida com o AutoRun. O AutoRun permite que um CD inicie automaticamente uma função, como um instalador ou jogo, quando colocado na unidade de CD-ROM. Isso é feito por meio do uso de um arquivo autorun. inf no diretório raiz do CD. Para obter mais detalhes, consulte [criando um aplicativo de CD-ROM habilitado para autorun](autoplay.md).
