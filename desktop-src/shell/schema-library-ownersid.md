# <a name="ownersid-element-library-schema"></a>Elemento ownerSID (esquema de biblioteca)

O <ownerSID> elemento especifica o SID (identificador de segurança) do criador da biblioteca. Esse elemento é opcional e não tem atributos ou elementos filho.

## <a name="syntax"></a>Syntax

``` syntax
<!-- ownerSID -->
<xs:element name="ownerSID" minOccurs="0"/>
```
