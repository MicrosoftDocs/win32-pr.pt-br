# <a name="ownersid-element-library-schema"></a>Elemento OwnerId (esquema de biblioteca)

O <ownerSID> elemento Especifica o identificador de segurança (SID) do criador da biblioteca. Esse elemento é opcional e não tem atributos ou elementos filho.

## <a name="syntax"></a>Syntax

``` syntax
<!-- ownerSID -->
<xs:element name="ownerSID" minOccurs="0"/>
```
