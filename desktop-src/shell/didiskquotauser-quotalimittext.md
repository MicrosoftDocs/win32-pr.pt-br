# <a name="didiskquotauserquotalimittext-property"></a>Propriedade DIDiskQuotaUser.QuotaLimitText

Obtém o limite de cota atual [**do usuário como**](diskquotacontrol-object.md) uma cadeia de caracteres de texto.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```JScript
QuotaLimitText = DIDiskQuotaUser.QuotaLimitText
```



## <a name="property-value"></a>Valor da propriedade

Um valor de cadeia de caracteres que contém o limite de cota atual do usuário.
