# <a name="stackedby-argument-the-windows-shell"></a>Argumento STACKEDBY (o Shell do Windows)

O `stackedby` argumento especifica a coluna de propriedade à qual os resultados são empilhados. Você pode empilhar por qualquer propriedade válida do sistema de propriedades.

## <a name="example"></a>Exemplo


```
search:query=vacation&stackedby=System.Author
```



### <a name="argument-information"></a>Informações do argumento



|                          |                                         |
|--------------------------|-----------------------------------------|
| Sistema operacional mínimo | Windows Vista com Service Pack 1 (SP1) |
