# <a name="idwritetextformat2-interface"></a>Interface IDWriteTextFormat2

Descreve as propriedades de fonte e parágrafo usadas para formatar texto e descreve informações de localidade.

## <a name="members"></a>Membros

A interface **IDWriteTextFormat2** herda [**de IDWriteTextFormat1.**](idwritetextformat1.md) **IDWriteTextFormat2** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IDWriteTextFormat2** tem esses métodos.



| Método                                                      | Descrição                                                                      |
|:------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**GetLineSpacing**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritetextformat2-getlinespacing) | Obtém o conjunto de ajuste de espaçamento de linha para um parágrafo de texto multilinha. <br/> |
| [**SetLineSpacing**](idwritetextformat2-setlinespacing.md) | Definir o espaçamento de linha.<br/>                                                     |
