# <a name="idwritenumbersubstitution-interface"></a>Interface IDWriteNumberSubstitution

Mantém os dígitos apropriados e a pontuação numérica para uma localidade especificada.

## <a name="members"></a>Membros

A interface **IDWriteNumberSubstitution** herda da interface [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) , mas não tem membros adicionais.
