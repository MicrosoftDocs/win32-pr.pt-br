# <a name="win32-text-api-comparison"></a>Comparação de API de texto do Win32

Para os desenvolvedores que estão migrando seu código de aplicativo Win32, a tabela a seguir lista as APIs de texto do Win32 e o equivalente aproximado em DirectWrite.

O DirectWrite preenche a coleção de fontes do sistema e monitora automaticamente as atualizações para ela, de modo que nenhuma instalação nunca seja necessária.                                                                              |
