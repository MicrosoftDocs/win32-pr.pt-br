# <a name="federated-search"></a>Pesquisa federada

O Windows 7 oferece suporte à pesquisa de documentos além do próprio computador do usuário. Os desenvolvedores e profissionais de ti podem habilitar seus mecanismos de pesquisa, repositórios de documentos, aplicativos Web e repositórios de dados proprietários para serem pesquisados no Windows 7 sem a necessidade de escrever e implantar o código do cliente. Isso permite que os usuários finais pesquisem sua intranet corporativa ou a Web tão facilmente quanto podem pesquisar seus arquivos locais, tudo isso dentro da mesma interface familiar do Windows.

O suporte a *OpenSearch* compatível com o Windows 7 pode ser adicionado por desenvolvedores e profissionais de ti a qualquer aplicativo Web pesquisável existente, adicionando a saída *RSS* ou *Atom* , conforme definido no padrão [OpenSearch](http://www.opensearch.org/) público. Os arquivos de descrição de *OpenSearch* podem ser criados por desenvolvedores e usuários avançados para definir os parâmetros de conexão que permitem que o cliente de desktop crie um conector de pesquisa para o serviço.

O servidor de pesquisa do SharePoint também pode consultar esses serviços *OpenSearch* compatíveis. Isso também habilita o acesso ao serviço de pesquisa por meio da interface da Web do centro de pesquisa do *SharePoint* .
