# <a name="federated-search"></a>Pesquisa Federada

Windows 7 dá suporte à pesquisa de documentos além do próprio computador do usuário. Os desenvolvedores e profissionais de TI podem habilitar seus mecanismos de pesquisa, repositórios de documentos, aplicativos Web e armazenamentos de dados proprietários a serem pesquisados no Windows 7 sem a necessidade de escrever e implantar o código do cliente. Isso permite que os usuários finais pesquisem a intranet corporativa ou a Web tão facilmente quanto podem pesquisar seus arquivos locais, tudo de dentro da mesma interface Windows familiar.

Windows 7 suporte compatível com OpenSearch pode ser adicionado por desenvolvedores e profissionais de TI *a* qualquer aplicativo Web pesquisável existente adicionando a saída *RSS* ou *ATOM* conforme definido no padrão OpenSearch [público.](http://www.opensearch.org/) *OpenSearch* arquivos de descrição podem ser criado por desenvolvedores e usuários avançados para definir os parâmetros de conexão que permitem que o cliente da área de trabalho crie um Conector de Pesquisa para o serviço.

SharePoint O Servidor de Pesquisa também pode consultar esses serviços *OpenSearch* compatíveis. Isso habilita o acesso ao serviço de pesquisa *SharePoint* interface da Web do centro de pesquisa também.
