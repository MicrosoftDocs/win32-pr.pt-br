# <a name="federated-search"></a>Pesquisa federada

o Windows 7 dá suporte à pesquisa de documentos além do próprio computador do usuário. os desenvolvedores e profissionais de ti podem habilitar seus mecanismos de pesquisa, repositórios de documentos, aplicativos web e repositórios de dados proprietários para serem pesquisados no Windows 7 sem a necessidade de escrever e implantar o código do cliente. isso permite que os usuários finais pesquisem sua intranet corporativa ou a web tão facilmente quanto podem pesquisar seus arquivos locais, tudo isso dentro da mesma interface de Windows familiar.

o suporte a *OpenSearch* compatível com Windows 7 pode ser adicionado por desenvolvedores e profissionais de ti a qualquer aplicativo web pesquisável existente, adicionando a saída *RSS* ou *ATOM* , conforme definido no padrão de [OpenSearch](http://www.opensearch.org/) público. *OpenSearch* arquivos de descrição podem ser criados por desenvolvedores e usuários avançados para definir os parâmetros de conexão que permitem que o cliente de desktop crie um conector de pesquisa para o serviço.

SharePoint o servidor de pesquisa também pode consultar esses serviços de *OpenSearch* compatíveis. isso habilita o acesso ao serviço de pesquisa por meio da interface da web do centro de pesquisa *SharePoint* também.
