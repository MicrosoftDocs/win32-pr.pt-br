# <a name="the-best-of-windows-and-the-web"></a>O melhor do Windows e da Web

Com o Windows 7, é fácil conectar-se e permanecer conectado. A rede do Windows oferece opções de desenvolvedores para melhor cache e compartilhamento de dados para melhorar o desempenho da rede. O diagnóstico de rede no Windows 7 fornece aos desenvolvedores informações relevantes para o monitoramento de problemas de rede. Se o objetivo é conectar dispositivos, conectar-se a uma rede ou simplesmente entender o status da conexão, o Windows 7 fornece uma plataforma aprimorada.

-   [Pesquisa federada](federated-search.md)
-   [Plataforma de sensor](sensor-platform.md)
