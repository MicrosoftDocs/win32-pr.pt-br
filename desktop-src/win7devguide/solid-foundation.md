# <a name="solid-foundation"></a>Solid Foundation

Windows 7 fornece uma plataforma de desenvolvedor altamente produtiva e ferramentas que entregam conceitos básicos do sistema operacional. Windows 7 se baseia na plataforma Windows Vista, dando aos desenvolvedores o poder de criar aplicativos compatíveis com ambas as plataformas, limitando os problemas de compatibilidade do aplicativo. Windows 7 melhora drasticamente o desempenho e o gerenciamento de energia para que você possa criar aplicativos que otimizam a experiência móvel. Como resultado, seus aplicativos serão mais visualmente atraentes, mais fáceis de criar e dar suporte a uma ampla variedade de padrões internacionais. Também foram feitos avanços no suporte global, acessibilidade e implantação de aplicativos.
