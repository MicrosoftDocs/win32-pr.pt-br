# <a name="solid-foundation"></a>Base sólida

o Windows 7 fornece uma plataforma de desenvolvedor altamente produtiva e ferramentas que fornecem nos principais conceitos básicos do sistema operacional. o Windows 7 baseia-se na plataforma Windows Vista, oferecendo aos desenvolvedores a capacidade de criar aplicativos compatíveis com ambas as plataformas e, ao mesmo tempo, limitar os problemas de compatibilidade de aplicativos. o Windows 7 melhora drasticamente o desempenho e o gerenciamento de energia para que você possa criar aplicativos que otimizem a experiência móvel. Como resultado, seus aplicativos serão mais visualmente atraentes, fáceis de criar e oferecerão suporte a uma maior variedade de padrões internacionais. Os avanços também foram feitos no suporte global, na acessibilidade e na implantação de aplicativos.
