# <a name="richer-application-experiences"></a>Experiências de aplicativos mais ricas

O Windows 7 permite que os desenvolvedores criem aplicativos distintivos e intuitivos que melhoram significativamente a descoberta, a usabilidade e o enorme aproveitamento. Os novos métodos de integração da área de trabalho colocam a funcionalidade do aplicativo diretamente nas mãos do usuário, e o Windows Explorer e as *bibliotecas* fornecem acesso fácil a informações de alto valor. O controle de faixa de visão do Windows e a estrutura de animação do Windows facilitam a criação de interfaces de usuário atraentes e atrativas. Novas APIs de toque habilitam interações naturais por meio do multitoque e do movimento de dedos e APIs de manipulação e inércia permitem efeitos visuais impressionantes.

Avanços rápidos em tecnologia de hardware e software também estão impulsionando experiências de usuário de alta fidelidade. O Windows 7 traz esses avanços sob controle do desenvolvedor com APIs novas e flexíveis que aproveitam totalmente a tecnologia, ao mesmo tempo que facilita ainda mais o desenvolvimento de aplicativos atraentes.
