# <a name="richer-application-experiences"></a>Experiências de aplicativo mais ricas

Windows 7 permite que os desenvolvedores criem aplicativos distintos e intuitivos que melhorem significativamente a capacidade de descoberta, a usabilidade e o grande prazer. Novos métodos de integração da área de trabalho colocam a funcionalidade do aplicativo à mão do usuário e Windows Explorer e *Bibliotecas* fornecem acesso fácil a informações de alto valor. O Windows faixa de opções e Windows de Animação facilitam a complicação de interfaces do usuário interativas e atraentes. As novas APIs de toque permitem interações naturais por meio de panorâmico com vários toques e dedos e as APIs de manipulação e inércia permitem efeitos visuais impressionantes.

Os rápidos avanços na tecnologia de hardware e software também estão impulsionando experiências de usuário com maior fidelidade. Windows 7 traz esses avanços sob controle do desenvolvedor com APIs novas e flexíveis que aproveitam ao máximo a tecnologia, facilitando ainda mais o desenvolvimento de aplicativos atraentes.
