# <a name="introduction-windows-7-developer-guide"></a>introdução (guia do desenvolvedor do Windows 7)

O desenvolvimento de software nunca foi mais empolgante. O mundo nunca foi mais conectado, interativo e móvel. As experiências e o conteúdo do usuário nunca tinham mais fidelidade. Os avanços em conectividade, mobilidade, interfaces naturais, gráficos e mídia estão permitindo novos cenários e oportunidades para os desenvolvedores. Ao mesmo tempo, as expectativas nunca foram mais altas para desempenho, flexibilidade e interoperabilidade. Uma plataforma de desenvolvedor de nível mundial deve fornecer uma base completa e flexível para os desenvolvedores e suas soluções, capacitando-os a criar aplicativos que os configuram de forma diferente da competição, maximizando seus investimentos.

o sistema operacional Windows 7 é a plataforma essencial para os desenvolvedores. ouvindo nossos clientes, observando os cenários do amanhã e aproveitando os recursos do Windows Vista, a Microsoft oferece aos desenvolvedores uma ampla variedade de opções e recursos na plataforma Windows developer, enquanto capacita os desenvolvedores a fornecer soluções criativas que reconhecem dispositivos móveis, conectados, de alta fidelidade e fornecem uma experiência de usuário altamente intuitiva. o mais importante é que os conceitos básicos da plataforma de desenvolvedor, como segurança, desempenho e compatibilidade, são as principais prioridades do Windows 7.

Este guia resume os principais avanços para desenvolvedores em cada uma das três áreas a seguir:

[Base sólida](solid-foundation.md)

Ótima experiência do usuário começa com uma base sólida. o Windows 7 fornece uma plataforma de desenvolvimento sólida e ferramentas inovadoras que oferecem mais opções, maior compatibilidade de aplicativos, melhor desempenho e suporte a documentos sofisticados. Isso resulta em um ambiente simples e mais confiável para seus aplicativos e um conhecido e intuitivo para seus usuários.

Mantivemos a interface do usuário consistente, previsível e fácil de usar, enquanto adicionamos suporte multitoque, rolagem e outros recursos de interação intuitivos que aproveitam os mais recentes recursos de PC e dispositivo móvel. com o Windows 7, os desenvolvedores têm a capacidade de criar as experiências de usuário certas para seus aplicativos. (consulte [Windows diretrizes de interação da experiência do usuário](/windows/apps/desktop/).)

[Experiências de aplicativos mais ricas](richer-application-experiences.md)

o Windows 7 permite que os desenvolvedores criem aplicativos hoje que serão executados nos PCs do amanhã. Software e dispositivos habilitados para serviço, como telefones celulares, players de mídia portáteis e câmeras digitais, demandam conectividade constante e aplicativos avançados. o Windows 7 fornece uma plataforma para ambos, tornando mais fácil para os desenvolvedores aproveitarem a funcionalidade e os recursos do hardware da próxima geração, garantindo que os usuários estejam sempre ativados e sempre conectados. (consulte [Windows Central do desenvolvedor de Hardware](https://www.microsoft.com/whdc/default.mspx).)

[o melhor de Windows e da Web](the-best-of-windows-and-the-web.md)

com o Windows 7, é fácil conectar-se e permanecer conectado. Windows rede oferece aos desenvolvedores opções para melhor cache e compartilhamento de dados para melhorar o desempenho da rede. o diagnóstico de rede no Windows 7 fornece aos desenvolvedores informações relevantes para o monitoramento de problemas de rede. Windows O Internet Explorer 8 é a plataforma essencial para a criação de experiências da Web rápidas e avançadas. se o objetivo é conectar dispositivos, conectar-se a uma rede ou conectar-se à *Internet*, Windows 7 fornece uma plataforma aprimorada. (Consulte [desenvolvimento](https://msdn.microsoft.com/library/aa155073.aspx)para a Web.)
