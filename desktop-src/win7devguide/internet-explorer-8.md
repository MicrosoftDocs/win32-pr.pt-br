<span data-ttu-id="dd14b-109">Com as versões anteriores do Windows Internet Explorer, os desenvolvedores e designers às vezes observaram que o Internet Explorer teve sua própria interpretação de <span data-ttu-id="dd14b-115">Por fim, o Internet Explorer 8 permite experiências que vão além da página por meio de novos recursos de navegador que conectam facilmente os usuários aos serviços Web data-stu-id="dd14b-119">Internet Explorer 8's standards mode layout engine was also built with the Cascading Style Sheets, Level 2 Revision 1 (CSS2.1) spec in hand and full compliance in mind, helping developers to easily support the latest standards while having the option to fall back to Windows Internet Explorer 7- compatible (or earlier) behavior if necessary.</span></span> <span data-ttu-id="dd14b-120">Para ajudar os desenvolvedores a aproveitar ao máximo os elementos oferecidos pelo HTML 4, o Internet Explorer 8 fornece suporte atualizado para vários elementos de apresentação.</span><span class="sxs-lookup"><span data-stu-id="dd14b-120">To assist developers in taking full advantage of the elements offered by HTML 4, Internet Explorer 8 provides upgraded fornecer uma marcação HTML mais expressiva e acessível.</span><span class="sxs-lookup"><span data-stu-id="dd14b-122">Through improved support for these and other HTML elements, web developers can
