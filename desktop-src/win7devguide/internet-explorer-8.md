# <a name="internet-explorer-8"></a>Internet Explorer 8

A evolução da *Internet* introduziu novas fontes de informações avançadas e mais maneiras de acessá-las. Esse crescimento criou novas oportunidades, experiências, serviços online e padrões para a Web. Com essa intensidade e confiança, os desenvolvedores da Web enfrentam um conjunto crescente de necessidades, que são endereços do Windows Internet Explorer 8 de três maneiras principais.

Primeiro, o Internet Explorer 8 fornece interoperabilidade do mundo real com outros navegadores e compatibilidade para sites existentes. Ele inclui aprimoramentos na plataforma e arquitetura principais, oferecendo desempenho, segurança, confiabilidade e compatibilidade aprimorados. Com as versões anteriores do Windows Internet Explorer, os desenvolvedores e designers às vezes observaram que o Internet Explorer teve sua própria interpretação de padrões da Web e a maneira como o navegador lida com HTML, folhas de estilos em cascata (CSS), scripting e assim por diante. Em alguns casos, as interpretações foram decididas porque o Internet Explorer oferecia suporte a determinados recursos antes que os padrões correspondentes fossem finalizados. Se esses padrões mudarem à medida que forem finalizados, a implementação do Internet Explorer poderá variar do que o padrão especifica. Com o Internet Explorer 8, a decisão de dar suporte a comportamentos herdados versus padrões estritos será colocada nas mãos dos desenvolvedores, permitindo que você selecione o modo de renderização de cada página.

Segundo, o Internet Explorer 8 torna o desenvolvimento para a Web mais rápido e fácil com ferramentas de desenvolvedor internas. Com avanços no suporte a aplicativos JavaScript e XML (AJAX) assíncronos, bem como melhorias que simplificam o processo de criação de aplicativos entre navegadores, o Internet Explorer 8 permite que os desenvolvedores sejam mais produtivos ao criar os aplicativos Web mais robustos possíveis.

Por fim, o Internet Explorer 8 permite experiências que vão além da página por meio de novos recursos de navegador que conectam facilmente os usuários aos serviços Web inovadores. O Internet Explorer 8 apresenta um conjunto de ferramentas de desenvolvedor integradas que são um componente do navegador. Como resultado, o desempenho é melhorado e nenhuma memória é usada quando as ferramentas não estão em execução. As ferramentas de desenvolvedor integradas tornam mais rápido e fácil para os desenvolvedores desenvolver e solucionar problemas de sites de conteúdo avançados. O mecanismo de layout do modo de padrões do Internet Explorer 8 também foi criado com a especificação de folhas de estilos em cascata, nível 2 revisão 1 (CSS 2.1) e total conformidade em mente, ajudando os desenvolvedores a dar suporte fácil aos padrões mais recentes, ao mesmo tempo em que têm a opção de fazer fallback para o comportamento compatível com o Windows Internet Explorer 7 (ou anterior), se necessário. Para ajudar os desenvolvedores a aproveitar ao máximo os elementos oferecidos pelo HTML 4, o Internet Explorer 8 fornece suporte atualizado para vários elementos de apresentação. Por exemplo, o elemento Q representa uma cadeia de caracteres entre aspas embutida, e o elemento Object agora pode representar qualquer "Object", incluindo imagens. Por meio de suporte aprimorado para esses e outros elementos HTML, os desenvolvedores da Web podem fornecer uma marcação HTML mais expressiva e acessível.

Além dos aprimoramentos feitos na plataforma subjacente e na experiência do desenvolvedor, o Internet Explorer 8 apresenta novos recursos que oferecem novas oportunidades para que os desenvolvedores da Web ampliem sua experiência além da página da Web. Esses novos recursos são habilitados por meio de padrões da Comunidade em vez de criar código de cliente personalizado que não é interoperável entre navegadores.
