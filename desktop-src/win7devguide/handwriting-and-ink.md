# <a name="handwriting-and-ink"></a>Manuscrito e tinta

Com a proliferação de *Tablet PCs* no mercado, os recursos do *Tablet* estão se tornando parte da computação principal. No Windows 7, o toque e a escrita são experiências de usuário de primeira classe. O Windows 7 melhora a experiência com a caneta, fornecendo maior precisão e velocidade. A entrada de manuscrito é aprimorada e há suporte para mais idiomas. O painel de entrada de texto oferece texto de previsão para maior velocidade de entrada e correção. A precisão de manuscrito é aprimorada por meio de personalização em todos os idiomas, dicionários personalizados e inovações no reconhecimento de idioma do leste asiático. O modelo de interação aprimorado oferece uma melhor experiência de leitura nas telas pequenas e de alta resolução comuns em computadores portáteis. (Consulte [programação no painel de entrada de texto](../tablet/programming-the-text-input-panel.md).)

![painel de entrada do Tablet PC](images/windows7-4.jpg)

O painel de entrada de texto apresenta correção de texto fácil

## <a name="math-recognition"></a>Reconhecimento de matemática

O novo recurso de reconhecimento de matemática permite que os usuários insiram cálculos em aplicativos por meio de manuscrito — a maneira mais natural e eficiente de inserir expressões matemáticas. A funcionalidade é fornecida por dois componentes de interface do usuário. O painel de entrada matemática é um acessório autônomo do Windows que funciona com qualquer aplicativo com reconhecimento de matemática. O controle de entrada matemática é integrado aos aplicativos por meio de sua API.

Subjacentes aos componentes da interface do usuário está o reconhecedor de matemática. Esse mecanismo reconhece a expressão matemática manuscrita e traduz o resultado no formato *MathML* para que os aplicativos usem. A experiência de correção foi aprimorada para ajudar os usuários a fazer correções mais rapidamente. (Consulte [programação do controle de entrada de matemática](../tablet/programming-the-math-input-control.md).)

## <a name="handwriting-with-personalized-custom-dictionary"></a>Manuscrito com dicionário personalizado personalizados

Para muitos cenários, uma boa precisão de manuscrito requer um dicionário adaptado ao domínio de uso. O Windows 7 apresenta dicionários personalizados, que permitem um melhor reconhecimento de manuscrito para vocabulários especializados. Os desenvolvedores que estão escrevendo aplicativos verticais, por exemplo, um bloco de notas de receita médica, agora podem adicionar termos específicos ao seu aplicativo, como nomes de drogas. (Consulte [aprimoramentos do desenvolvedor para o Windows Touch e Tablet PC](https://www.microsoft.com/whdc/device/input/Touch_tab_enhance.mspx).)
