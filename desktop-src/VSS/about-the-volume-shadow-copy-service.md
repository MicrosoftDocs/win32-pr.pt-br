# <a name="about-the-volume-shadow-copy-service"></a>Sobre o Serviço de Cópias de Sombra de Volume

O Serviço de Cópias de Sombra de Volume (VSS) captura e copia imagens estáveis para backup em sistemas em execução, principalmente servidores, sem degradar indevidamente o desempenho e a estabilidade dos serviços que eles fornecem.

A solução VSS foi projetada para permitir que os desenvolvedores criem serviços ([*gravadores*](vssgloss-w.md)) que podem ser efetivamente submetidos a backup pelo aplicativo de backup de qualquer fornecedor usando o VSS ([*solicitantes*](vssgloss-r.md)).
