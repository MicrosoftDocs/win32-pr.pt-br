<dl> <dt>

<span id="base.vssgloss_thaw_event"></span><span id="BASE.VSSGLOSS_THAW_EVENT"></span>**Descongelar evento**
</dt> <dd>

Um evento VSS emitido pelo VSS indicando que um congelamento de cópia de sombra foi concluído. Ele é usado para remover o preparação de um gravador para um congelamento. Consulte também [*congelar*](vssgloss-f.md), [*cópia de sombra*](vssgloss-s.md).

</dd> <dt>

<span id="base.vssgloss_transportable_shadow_copy"></span><span id="BASE.VSSGLOSS_TRANSPORTABLE_SHADOW_COPY"></span>**cópia de sombra transportável**
</dt> <dd>

Uma cópia de sombra que pode ser movida de um sistema para outro. Normalmente, uma cópia de sombra transportável não é retonada localmente. Consulte também [*cópia de sombra em superfície*](vssgloss-s.md).

</dd> </dl>
