# <a name="vss-security-issues"></a>Problemas de segurança do VSS

A infraestrutura do VSS é internamente baseada em COM e, portanto, permitir que gravadores e solicitantes se comuniquem requer segurança baseada em COM. A segurança COM fornece mecanismos e diretrizes que um desenvolvedor VSS deve considerar:

-   [Considerações de segurança para gravadores](security-considerations-for-writers.md)
-   [Considerações de segurança para solicitantes](security-considerations-for-requestors.md)
