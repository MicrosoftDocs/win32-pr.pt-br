# <a name="volume-shadow-copy-glossary"></a>Glossário de cópia de sombra de volume

Muitos desses termos são familiares aos desenvolvedores, mas têm definições novas ou alteradas no ambiente da API do VSS. Examine as definições a seguir para entender a implementação da API do VSS desses termos.
