# <a name="working-with-the-writer-metadata-document"></a>Trabalhando com o documento de metadados do writer

Cada autor produz um Documento de Metadados do Writer. O aplicativo de backup usa este documento para obter informações sobre esse autor, os dados que ele possui e como restaurar esses dados. Depois que o autor o produz, o Documento de Metadados do Writer é um documento somente leitura para o aplicativo de backup.

Para usar o Documento de Metadados do Writer, é necessário que você entenda o seguinte:

-   [Ciclo de vida do documento de metadados de autor](writer-metadata-document-life-cycle.md)
-   [Conteúdo do documento de metadados do writer](writer-metadata-document-contents.md)
