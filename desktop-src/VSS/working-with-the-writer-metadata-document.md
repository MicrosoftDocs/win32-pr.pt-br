# <a name="working-with-the-writer-metadata-document"></a>Trabalhando com o documento de metadados do gravador

Cada gravador produz um documento de metadados do gravador. O aplicativo de backup usa este documento para obter informações sobre esse gravador, os dados que ele possui e como restaurá-los. Depois que o gravador o produz, o documento de metadados do gravador é um documento somente leitura para o aplicativo de backup.

Para usar o documento de metadados do gravador, é necessário que você compreenda o seguinte:

-   [Ciclo de vida do documento de metadados do gravador](writer-metadata-document-life-cycle.md)
-   [Conteúdo do documento de metadados do gravador](writer-metadata-document-contents.md)
