# <a name="working-with-alternate-locations-during-restore"></a><span data-ttu-id="1e21b-103">Trabalhando com locais alternativos durante a restauração</span><span class="sxs-lookup"><span data-stu-id="1e21b-103">Working with Alternate Locations during Restore</span></span>

<span data-ttu-id="1e21b-104">Há muitas razões pelas quais um solicitante não deve ou não ser capaz de restaurar arquivos de mídia de backup para seu local original.</span><span class="sxs-lookup"><span data-stu-id="1e21b-104">There are many reasons why a requester either should not, or may not, be able to restore files from backup media to their original location.</span></span> <span data-ttu-id="1e21b-105">Por exemplo, um método ou destino de restauração pode exigir tal restauração ou o local de restauração atual pode ser ocupado e não gravável.</span><span class="sxs-lookup"><span data-stu-id="1e21b-105">For example, a restore method or target may require such a restoration, or the current restoration location may be occupied and unwritable.</span></span>

<span data-ttu-id="1e21b-106">Para lidar com esses casos, um gravador pode ter definido um [*mapeamento de local alternativo*](vssgloss-a.md), um destino de restauração não padrão a ser usado para circunstâncias especiais.</span><span class="sxs-lookup"><span data-stu-id="1e21b-106">To handle such cases, a writer may have defined an [*alternate location mapping*](vssgloss-a.md), a nonstandard restore destination to be used for special circumstances.</span></span>

<span data-ttu-id="1e21b-107">O termo mapeamento de local alternativo, conforme usado com o VSS, não deve ser confundido com o termo [*caminho alternativo*](vssgloss-a.md).</span><span class="sxs-lookup"><span data-stu-id="1e21b-107">The term alternate location mapping, as used with VSS, should not be confused with the term [*alternate path*](vssgloss-a.md).</span></span> <span data-ttu-id="1e21b-108">Mapeamentos de local alternativos são usados somente durante operações de restauração e se referem a um destino alternativo para operações de restauração.</span><span class="sxs-lookup"><span data-stu-id="1e21b-108">Alternate location mappings are used only during restore operations, and refer to an alternate destination for restore operations.</span></span> <span data-ttu-id="1e21b-109">Os caminhos alternativos são usados somente durante as operações de backup e se referem a uma fonte alternativa da qual fazer backup.</span><span class="sxs-lookup"><span data-stu-id="1e21b-109">Alternate paths are used only during backup operations, and refer to an alternate source from which to back up.</span></span>

<span data-ttu-id="1e21b-110">Para usar mapeamentos de local alternativos durante a restauração, um solicitante faria o seguinte (normalmente seguindo a geração de um evento de [**Prerestore**](/windows/desktop/api/VsBackup/nf-vsbackup-ivssbackupcomponents-prerestore) ):</span><span class="sxs-lookup"><span data-stu-id="1e21b-110">To use alternate location mappings during restore, a requester would do the following (typically following the generation of a [**PreRestore**](/windows/desktop/api/VsBackup/nf-vsbackup-ivssbackupcomponents-prerestore) event):</span></span>

    > <span data-ttu-id="1e21b-112">O solicitante usa [**IVssExamineWriterMetadata:: GetAlternateLocationMapping**](/windows/desktop/api/VsBackup/nf-vsbackup-ivssexaminewritermetadata-getalternatelocationmapping), não [**IVssComponent:: GetAlternateLocationMapping**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscomponent-getalternatelocationmapping).</span><span class="sxs-lookup"><span data-stu-id="1e21b-112">The requester uses [**IVssExamineWriterMetadata::GetAlternateLocationMapping**](/windows/desktop/api/VsBackup/nf-vsbackup-ivssexaminewritermetadata-getalternatelocationmapping), not [**IVssComponent::GetAlternateLocationMapping**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscomponent-getalternatelocationmapping).</span></span> <span data-ttu-id="1e21b-113">O primeiro retorna os mapeamentos de local alternativos disponíveis para uso por um solicitante.</span><span class="sxs-lookup"><span data-stu-id="1e21b-113">The former returns those alternate location mappings available for use by a requester.</span></span> <span data-ttu-id="1e21b-114">O último é usado para indicar os mapeamentos de local alternativos realmente usados por um solicitante.</span><span class="sxs-lookup"><span data-stu-id="1e21b-114">The latter is used to indicate those alternate location mappings actually used by a requester.</span></span>
