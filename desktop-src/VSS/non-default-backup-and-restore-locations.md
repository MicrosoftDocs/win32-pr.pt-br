# <a name="non-default-backup-and-restore-locations"></a>Locais de backup e restauração não padrão

As operações de backup e restauração normalmente copiam arquivos de um determinado local padrão no disco para a mídia de backup e, em seguida, restauram dessa mídia para o mesmo local padrão no disco.

Há muitos motivos, especialmente ao lidar com processos em execução, para não seguir esse modelo simples. O VSS dá suporte ao uso de fontes não padrão para backup e destinos alternativos para restauração e inclui mecanismos para trabalhar com subconjunto de arquivos e para remapear arquivos.

-   [Trabalhando com caminhos alternativos durante o backup](working-with-alternate-paths-during-backup.md)
-   [Trabalhando com locais alternativos durante a restauração](working-with-alternate-locations-during-restore.md)
-   [Trabalhando com novos destinos durante a restauração](working-with-new-targets-during-restore.md)
-   [Trabalhando com arquivos parciais](working-with-partial-files.md)
-   [Trabalhando com destinos direcionados](working-with-directed-targets.md)
