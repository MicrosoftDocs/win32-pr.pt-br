# <a name="volume-shadow-copy-service-overview"></a>Visão geral de Serviço de Cópias de Sombra de Volume

O Serviço de Cópias de Sombra de Volume (VSS) é um conjunto de APIs COM que implementa uma estrutura para permitir que os backups de volume sejam executados enquanto os aplicativos em um sistema continuam a gravar nos volumes.

O VSS fornece uma interface consistente que permite a coordenação entre aplicativos de usuário que atualizam dados em disco ([*gravadores*](vssgloss-w.md)) e aqueles que fazem backup de aplicativos ([*solicitantes*](vssgloss-r.md)).
