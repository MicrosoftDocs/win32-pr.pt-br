# <a name="volume-shadow-copy-service-overview"></a>Serviço de Cópias de Sombra de Volume geral

O Serviço de Cópias de Sombra de Volume (VSS) é um conjunto de APIs COM que implementa uma estrutura para permitir que os backups de volume sejam executados enquanto os aplicativos em um sistema continuam a gravar nos volumes.

O VSS fornece uma interface consistente que permite a [](vssgloss-w.md)coordenação entre aplicativos de usuário que atualizam dados em disco (autores) e aqueles que fazem backup de aplicativos [*(solicitantes).*](vssgloss-r.md)
