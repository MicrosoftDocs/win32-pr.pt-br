# <a name="vss-backups-and-restores-of-the-cluster-database"></a>Backups e restaurações do VSS do banco de dados de cluster

O clustering de failover usa Serviço de Cópias de Sombra de Volume (VSS) para fazer o back-up e restaurar a configuração do cluster de failover. Para obter mais informações, consulte [Backing Up and Restoring the Failover Cluster Configuration Using VSS](/previous-versions/windows/desktop/mscs/backing-up-and-restoring-the-failover-cluster-configuration-using-vss).
