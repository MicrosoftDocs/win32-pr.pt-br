# <a name="window-styles"></a>Estilos de janela

A seguir estão os estilos de janela. Depois que a janela tiver sido criada, esses estilos não poderão ser modificados, exceto quando observado.



| Constante/valor                                                                                                                                                                                                                                                                                                                              | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WS_BORDER"></span><span id="ws_border"></span><dl> <dt>**WS \_**</dt> <dt>0X00800000L</dt> de borda </dl>                                                                                                                          | A janela tem uma borda de linha fina.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_CAPTION"></span><span id="ws_caption"></span><dl> <dt>**WS \_**</dt> <dt>0X00C00000L</dt> de legenda </dl>                                                                                                                       | A janela tem uma barra de título (inclui o estilo de **\_ borda WS** ).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_CHILD"></span><span id="ws_child"></span><dl> <dt>**WS \_**</dt> <dt>0x40000000L</dt> filho </dl>                                                                                                                             | A janela é uma janela filho. Uma janela com este estilo não pode ter uma barra de menus. Este estilo não pode ser usado com o estilo **WS \_ -Popup** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_CHILDWINDOW"></span><span id="ws_childwindow"></span><dl> <dt>**WS \_ CHILDWINDOW**</dt> <dt>0x40000000L</dt> </dl>                                                                                                           | O mesmo que o estilo **\_ filho WS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="WS_CLIPCHILDREN"></span><span id="ws_clipchildren"></span><dl> <dt>**WS \_ CLIPCHILDREN**</dt> <dt>0x02000000L</dt> </dl>                                                                                                        | Exclui a área ocupada por janelas filhas quando o desenho ocorre dentro da janela pai. Esse estilo é usado ao criar a janela pai.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="WS_CLIPSIBLINGS"></span><span id="ws_clipsiblings"></span><dl> <dt>**WS \_ CLIPSIBLINGS**</dt> <dt>0x04000000L</dt> </dl>                                                                                                        | Corta janelas filhas em relação umas às outras; ou seja, quando uma janela filho específica recebe uma mensagem do [**WM \_ Paint**](../gdi/wm-paint.md) , o estilo **WS \_ CLIPSIBLINGS** corta todas as outras janelas filho sobrepostas da região da janela filho a ser atualizada. Se **WS \_ CLIPSIBLINGS** não for especificado e a sobreposição de janelas filho for possível, ao desenhar dentro da área do cliente de uma janela filho, será possível desenhar dentro da área do cliente de uma janela filho vizinha.<br/>                                                                                                                                                                                |
| <span id="WS_DISABLED"></span><span id="ws_disabled"></span><dl> <dt>**WS \_ 0x08000000L DESABILITAdo**</dt> <dt></dt> </dl>                                                                                                                    | A janela está inicialmente desabilitada. Uma janela desabilitada não pode receber entrada do usuário. Para alterar isso após a criação de uma janela, use a função [**EnableWindow**](/windows/win32/api/winuser/nf-winuser-enablewindow) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="WS_DLGFRAME"></span><span id="ws_dlgframe"></span><dl> <dt>**WS \_ DLGFRAME**</dt> <dt>0x00400000L</dt> </dl>                                                                                                                    | A janela tem uma borda de um estilo normalmente usada com caixas de diálogo. Uma janela com este estilo não pode ter uma barra de título.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_GROUP"></span><span id="ws_group"></span><dl> <dt>**WS \_**</dt> <dt>0X00020000L</dt> de grupo </dl>                                                                                                                             | A janela é o primeiro controle de um grupo de controles. O grupo consiste nesse primeiro controle e todos os controles definidos depois dele, até o próximo controle com o estilo **de \_ grupo WS** . O primeiro controle em cada grupo geralmente tem o estilo **WS \_ TabStop** para que o usuário possa passar de um grupo para um grupo. O usuário pode posteriormente alterar o foco do teclado de um controle no grupo para o próximo controle no grupo usando as teclas de direção.<br/> Você pode ativar e desativar esse estilo para alterar a navegação da caixa de diálogo. Para alterar esse estilo após a criação de uma janela, use a função [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) .<br/> |
| <span id="WS_HSCROLL"></span><span id="ws_hscroll"></span><dl> <dt>**WS \_ HSCROLL**</dt> <dt>0x00100000L</dt> </dl>                                                                                                                       | A janela tem uma barra de rolagem horizontal.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_ICONIC"></span><span id="ws_iconic"></span><dl> <dt>**WS \_ ICÔNICO**</dt> <dt>0x20000000L</dt> </dl>                                                                                                                          | A janela é inicialmente minimizada. O mesmo que o estilo **WS \_ minimize** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_MAXIMIZE"></span><span id="ws_maximize"></span><dl> <dt>**WS \_ MAXIMIZAR**</dt> <dt>0x01000000L</dt> </dl>                                                                                                                    | A janela está inicialmente maximizada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_MAXIMIZEBOX"></span><span id="ws_maximizebox"></span><dl> <dt>**WS \_ MAXIMIZEBOX**</dt> <dt>0x00010000L</dt> </dl>                                                                                                           | A janela tem um botão Maximizar. Não pode ser combinado com o estilo **WS \_ ex \_ CONTEXTHELP** . O estilo **WS \_ SYSMENU** também deve ser especificado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_MINIMIZE"></span><span id="ws_minimize"></span><dl> <dt>**WS \_ MINIMIZAR**</dt> <dt>0x20000000L</dt> </dl>                                                                                                                    | A janela é inicialmente minimizada. O mesmo que o estilo **WS \_ icônico** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="WS_MINIMIZEBOX"></span><span id="ws_minimizebox"></span><dl> <dt>**WS \_ MINIMIZEBOX**</dt> <dt>0x00020000L</dt> </dl>                                                                                                           | A janela tem um botão minimizar. Não pode ser combinado com o estilo **WS \_ ex \_ CONTEXTHELP** . O estilo **WS \_ SYSMENU** também deve ser especificado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_OVERLAPPED"></span><span id="ws_overlapped"></span><dl> <dt>**WS \_ SOBREPOSTO**</dt> em <dt>0x00000000</dt> </dl>                                                                                                              | A janela é uma janela sobreposta. Uma janela sobreposta tem uma barra de título e uma borda. Igual ao estilo de **WS- \_ xadrez** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_OVERLAPPEDWINDOW"></span><span id="ws_overlappedwindow"></span><dl> <dt>**WS \_ OVERLAPPEDWINDOW**</dt> <dt>(WS \_ sobreposta \| WS \_ Caption \| WS \_ SYSMENU \| WS \_ THICKFRAME \| WS \_ MINIMIZEBOX \| WS \_ MAXIMIZEBOX)</dt> </dl> | A janela é uma janela sobreposta. | <span id="WS_POPUP"></span><span id="ws_popup"></span><dl> <dt>**WS \_ 0x80000000L pop-up**</dt> <dt></dt> </dl>                                                                                                                             | A janela é uma janela pop-up. Este estilo não pode ser usado com o estilo **\_ filho WS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_POPUPWINDOW"></span><span id="ws_popupwindow"></span><dl> <dt>**WS \_ POPUPWINDOW**</dt> <dt>(WS \_ pop-up WS \| \_ Border \| WS \_ SYSMENU)</dt> </dl>                                                                              | A janela é uma janela pop-up. Os estilos **WS \_ Caption** e **WS \_ POPUPWINDOW** devem ser combinados para tornar o menu janela visível.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="WS_SIZEBOX"></span><span id="ws_sizebox"></span><dl> <dt>**WS \_ SIZEBOX**</dt> <dt>0x00040000L</dt> </dl>                                                                                                                       | A janela tem uma borda de dimensionamento. O mesmo que o estilo **WS \_ THICKFRAME** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="WS_SYSMENU"></span><span id="ws_sysmenu"></span><dl> <dt>**WS \_ SYSMENU**</dt> <dt>0x00080000L</dt> </dl>                                                                                                                       | A janela tem um menu de janela em sua barra de título. O estilo de **\_ legenda WS** também deve ser especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="WS_TABSTOP"></span><span id="ws_tabstop"></span><dl> <dt>**WS \_**</dt> <dt>0X00010000L</dt> do TabStop </dl>                                                                                                                       | A janela é um controle que pode receber o foco do teclado quando o usuário pressiona a tecla TAB. Pressionar a tecla TAB altera o foco do teclado para o próximo controle com o estilo **WS \_ TabStop** .<br/> Você pode ativar e desativar esse estilo para alterar a navegação da caixa de diálogo. Para alterar esse estilo após a criação de uma janela, use a função [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) . Para caixas de diálogo Windows e sem janela restrita do usuário para trabalhar com paradas de tabulação, altere o loop de mensagem para chamar a função [**IsDialogMessage**](/windows/win32/api/winuser/nf-winuser-isdialogmessagea) .<br/>                                                                                           |

O mesmo que o estilo **WS \_ SIZEBOX** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
