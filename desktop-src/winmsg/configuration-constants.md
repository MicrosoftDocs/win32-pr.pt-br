# <a name="configuration-constants"></a>Constantes de configuração

Esta seção fornece as especificações de referência [**para constantes SystemParametersInfo**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) relacionadas aos [atributos do sistema](configuration.md) de configuração.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                             | Descrição                                                                                                                                                            |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Visualização de contato**](contact-visualization.md)<br/> | As constantes a seguir são usadas por aplicativos ou estruturas de interface do usuário para identificar como os comentários da interface do usuário são processados quando um contato de entrada é detectado.<br/>               |
| [**Visualização de gestos**](gesture-visualization.md)<br/> | As constantes a seguir são usadas por aplicativos ou estruturas de interface do usuário para identificar como os comentários da interface do usuário são processados quando um dos gestos listados é detectado.<br/>     |
| [**Visualização de caneta**](pen-visualization.md)<br/>         | As constantes a seguir são usadas por aplicativos ou estruturas de interface do usuário para identificar como os comentários da interface do usuário são processados quando um dos gestos de caneta listados é detectado.<br/> |
