# <a name="windows-and-messages"></a>Windows e mensagens

As seções a seguir descrevem os elementos de um aplicativo com uma interface gráfica do usuário Windows baseada em dados.

### <a name="in-this-section"></a>Nesta seção



| Nome                                                           | Descrição                                                                                                                                                                                                                  |
|----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Windows](windows.md)                                         | Discute janelas em geral.<br/>                                                                                                                                                                                     |
| [Classes de janela](window-classes.md)                           | Descreve os tipos de classes de janela, como o sistema as localiza e os elementos que definem o comportamento padrão das janelas que pertencem a elas.<br/>                                                             |
| [Procedimentos de janela](window-procedures.md)                     | Discute procedimentos de janela. Cada janela tem um procedimento de janela associado que processa todas as mensagens enviadas ou postadas em todas as janelas da classe.<br/>                                                              |
| [Mensagens e filas de mensagens](messages-and-message-queues.md) | Descreve mensagens e filas de mensagens e como usá-las em seus aplicativos.<br/>                                                                                                                                   |
| [Temporizadores](timers.md)                                           | Discute temporizadores. Um temporizador é uma rotina interna que mede repetidamente um intervalo especificado, em milissegundos.<br/>                                                                                                  |
| [Propriedades da janela](window-properties.md)                     | Discute as propriedades da janela. Uma propriedade de janela é qualquer dado atribuído a uma janela.<br/>                                                                                                                                  |
| [Configuration](configuration.md)                             | Descreve as funções que podem ser usadas para controlar a configuração de métricas do sistema e vários atributos do sistema, como tempo de clique duplo, tempo limite da economia de tela, largura da borda da janela e padrão de área de trabalho.<br/> |
| [Ganchos](hooks.md)                                             | Discute ganchos. Um gancho é um ponto no mecanismo de manipulação de mensagens do sistema em que um aplicativo pode instalar uma sub-rotina para monitorar o tráfego da mensagem.<br/>                                                         |
| [Interface de vários documentos](multiple-document-interface.md) | Discute a Interface de Vários Documentos, que é uma especificação que define uma interface do usuário para aplicativos que permitem que o usuário trabalhe com mais de um documento ao mesmo tempo.<br/>                      |
