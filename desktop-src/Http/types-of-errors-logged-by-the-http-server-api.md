# <a name="types-of-errors-logged-by-the-http-server-api"></a>Tipos de erros registrados pela API do servidor HTTP

A API do servidor HTTP registra respostas de erro para clientes, tempos limite de conexão, solicitações órfãs e conexões descartadas que são tratadas incorretamente.

A tabela a seguir lista os erros registrados por HTTP.



| Erro                                                                                                                                                  | Descrição                                                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="Responses_to_clients"></span><span id="responses_to_clients"></span><span id="RESPONSES_TO_CLIENTS"></span>"Respostas para clientes"<br/> | A API do servidor HTTP envia uma resposta de erro para um cliente, por exemplo, um erro 400 causado por um erro de análise na última solicitação recebida. Depois de enviar a resposta de erro, a API do servidor HTTP encerra a conexão.<br/> |
| <span id="Connection_time-outs"></span><span id="connection_time-outs"></span><span id="CONNECTION_TIME-OUTS"></span>"Tempos limite de conexão"<br/> | A API do servidor HTTP expira uma conexão. Se uma solicitação estiver pendente quando a conexão atingir o tempo limite, ela será usada para fornecer mais informações sobre a conexão no log de erros.<br/>                                         |
| <span id="Orphaned_requests"></span><span id="orphaned_requests"></span><span id="ORPHANED_REQUESTS"></span>"Solicitações órfãs"<br/>             | Um processo de modo de usuário é encerrado inesperadamente enquanto ainda há solicitações em fila roteadas para esse processo. A API do servidor HTTP registra as solicitações órfãs no log de erros.<br/>                                      |



 

Tipos de erro específicos são designados por cadeias de caracteres de frase de motivo que sempre aparecem como o último campo de cada linha de erro. A tabela a seguir lista as frases de motivo HTTP.



| Frase do motivo                                                                                                                                                                                       | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| <span id="AppPoolTimer"></span><span id="apppooltimer"></span><span id="APPPOOLTIMER"></span>AppPoolTimer<br/>                                                                                | Ocorreu um erro de serviço indisponível (erro HTTP 503). O serviço não está disponível porque o processo do pool de aplicativos está muito ocupado para lidar com a solicitação.<br/>                                                                                                                                                                                                                                                                                                                                  |
| <span id="AppShutdown"></span><span id="appshutdown"></span><span id="APPSHUTDOWN"></span>AppShutdown<br/>                                                                                    | Ocorreu um erro de serviço indisponível (erro HTTP 503). O serviço não está disponível porque o aplicativo é desligado automaticamente em resposta à política do administrador.<br/>                                                                                                                                                                                                                                                                                                                     |
| <span id="BadRequest"></span><span id="badrequest"></span><span id="BADREQUEST"></span>BadRequest<br/>                                                                                        | Foi encontrado um erro de análise ao processar uma solicitação.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                  |

| <span id="Connection_Dropped"></span><span id="connection_dropped"></span><span id="CONNECTION_DROPPED"></span>Conexão \_ cancelada<br/>                                                       | Falha do IIS ao retornar os dados de log dentro do limite de tempo especificado (1 minuto) devido a uma desconexão do cliente (ou redefinição) antes da resposta ser registrada. | <span id="ConnLimit"></span><span id="connlimit"></span><span id="CONNLIMIT"></span>ConnLimit<br/>                                                                                            | Ocorreu um erro de serviço indisponível (erro HTTP 503). | <span id="Disabled"></span><span id="disabled"></span><span id="DISABLED"></span>Desabilitado<br/>                                                                                                | Ocorreu um erro de serviço indisponível (erro HTTP 503). | <span id="FieldLength"></span><span id="fieldlength"></span><span id="FIELDLENGTH"></span>FieldLength<br/>                                                                                    | Um limite de tamanho de campo foi excedido.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="Forbidden"></span><span id="forbidden"></span><span id="FORBIDDEN"></span>Proibido<br/>                                                                                            | Um elemento ou sequência proibidas foi encontrado durante a análise.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="Header"></span><span id="header"></span><span id="HEADER"></span>Verga<br/>                                                                                                        | Um erro de análise foi encontrado em um cabeçalho.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                 |

| <span id="RequestLength"></span><span id="requestlength"></span><span id="REQUESTLENGTH"></span>RequestLength<br/>                                                                            | Um limite de tamanho de solicitação foi excedido.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
