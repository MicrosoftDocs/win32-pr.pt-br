# <a name="whats-new-for-windows-server-2003-sp1"></a>O que há de novo no Windows Server 2003 SP1

Os seguintes recursos estão disponíveis para o Windows Server 2003 com Service Pack 1 (SP1):

-   [Suporte a SSL no modo kernel.](kernel-mode-ssl.md)
    -   O Windows Server 2003 com SP1 adiciona a opção de executar operações SSL no modo kernel para melhorar o desempenho.
-   Suporte para a execução de aplicativos de 32 bits em WOW64 (sistema operacional de 64 bits).
-   [Aprimoramentos no log de erros HTTP.](error-logging-in-windows-server-2003-sp1.md)
    -   Suporte para cabeçalhos de estilo W3C.
    -   Campos de log adicionais disponíveis.
    -   Substituição do arquivo de log baseado em data e hora.
-   [Exceções de análise de HTTP.](parsing-exceptions.md)
    -   Dá suporte a exceções para HTTP/1.1 para compatibilidade com versões anteriores.
-   [Aprimoramentos do analisador HTTP.](parser-enhancements.md)
