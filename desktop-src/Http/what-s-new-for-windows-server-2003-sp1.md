# <a name="whats-new-for-windows-server-2003-sp1"></a>Novidades do Windows Server 2003 SP1

Os seguintes recursos estão disponíveis para Windows Server 2003 com Service Pack 1 (SP1):

-   [Suporte a SSL no modo kernel.](kernel-mode-ssl.md)
    -   Windows O Server 2003 com SP1 adiciona a opção de executar operações SSL no modo kernel para melhorar o desempenho.
-   Suporte para execução de aplicativos de 32 bits no WOW64 (sistema operacional de 64 bits).
-   [Aprimoramentos no log de erros HTTP.](error-logging-in-windows-server-2003-sp1.md)
    -   Suporte para os títulos de estilo W3C.
    -   Campos de log adicionais disponíveis.
    -   Sobrepondo arquivo de log baseado em data e hora.
-   [Exceções de análise HTTP.](parsing-exceptions.md)
    -   Dá suporte a exceções a HTTP/1.1 para compatibilidade com backward.
-   [Aprimoramentos do analisador HTTP.](parser-enhancements.md)
