# <a name="httpsys-manageability-in-windows-vista-and-windows-server-2008"></a>HTTP.sys capacidade de gerenciamento no Windows Vista e no Windows Server 2008

Com o Windows Vista e o Windows Server 2008, os profissionais de ti encontrarão uma funcionalidade de gerenciamento aprimorada para o componente de API do servidor HTTP, além do suporte a logs HTTP introduzido no Windows Server 2003. Para diagnosticar e configurar o componente de API do servidor HTTP, os profissionais de ti podem usar três ferramentas:

-   Nova funcionalidade de rastreamento de ETW
-   Contadores de desempenho
-   Comandos de configuração netsh

A API do servidor HTTP permite que os aplicativos se comuniquem via HTTP e inclui suporte a SSL para que os aplicativos possam trocar dados por conexões HTTP seguras. A API do servidor HTTP tem suporte nos sistemas operacionais Windows Server 2003 e no Windows XP SP2.
