O modelo de programação de API do servidor HTTP inclui cinco grupos de atividades:

-   [Configuração da instalação](setup-configuration.md)
-   [Configuração do runtime](run-time-configuration.md)
-   [Criando e ligando a uma fila de solicitações](creating-and-binding-to-a-request-queue.md)
-   [Solicitações de processamento](processing-requests.md)
-   [Desligamento e limpeza](shutdown-and-cleanup.md)

Para um aplicativo de exemplo que mostra como lidar com as ações de solicitação HTTP GET e POST e como enviar um erro 503 (**não \_ implementado**) se as ações estiverem presentes na solicitação que o aplicativo não manipula, consulte [aplicativo de exemplo do servidor http](http-server-sample-application.md).
