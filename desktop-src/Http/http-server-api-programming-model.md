-   [Configuração da instalação](setup-configuration.md)
-   [Configuração de tempo de execução](run-time-configuration.md)
-   [Criando e ligando a uma fila de solicitações](creating-and-binding-to-a-request-queue.md)
-   [Solicitações de processamento](processing-requests.md)
-   [Desligamento e limpeza](shutdown-and-cleanup.md)
