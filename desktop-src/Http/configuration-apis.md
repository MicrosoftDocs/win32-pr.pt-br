# <a name="configuration-apis"></a>APIs de configuração

As APIs do servidor HTTP fornecem funcionalidade para que os administradores do sistema definam, consultem e excluam parâmetros de configuração. As APIs de configuração controlam as configurações para todos os aplicativos que usam a API do servidor HTTP no sistema.

As seções listadas abaixo incluem informações sobre as definições de configuração.

> [!Note]  
> Todas essas são etapas opcionais de configuração a serem aplicadas com base nos requisitos do sistema e/ou aplicativo.

 

Lista de escuta de [IP](ip-listen-list.md)   — Descreve como configurar o HTTP para operar em interfaces IP específicas.

[Certificados SSL](ssl-certificates.md)   — detalha os procedimentos para configurar certificados com a API do servidor HTTP.

[Reservas de namespace, registros e roteamento](namespace-reservations-registrations-and-routing.md)   — Descreve o processo de reserva de partes do namespace para todos os aplicativos no computador usando a API do servidor HTTP. O processo para registrar e rotear solicitações para aplicativos específicos também é descrito.

[HttpCfg.exe](httpcfg-exe.md)   — um programa de exemplo que usa as APIs de configuração. Uma versão compilada dessa ferramenta também é fornecida com o sistema operacional entre as ferramentas de suporte, junto com um arquivo de ajuda que documenta como usá-la.
