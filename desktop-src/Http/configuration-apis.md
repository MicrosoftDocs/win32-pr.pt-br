# <a name="configuration-apis"></a>APIs de configuração

As APIs do servidor HTTP fornecem funcionalidade para os administradores do sistema definirem, consultarem e excluirem parâmetros de configuração. As APIs de configuração controlam as configurações de todos os aplicativos usando a API do servidor HTTP no sistema.

As seções listadas abaixo incluem informações sobre definições de configuração.

> [!Note]  
> Todas essas são etapas de configuração opcionais a serem aplicadas com base nos requisitos do sistema e/ou do aplicativo.

 

[Lista de escuta de IP](ip-listen-list.md) — descreve como configurar HTTP para operar em interfaces IP específicas.

[Certificados SSL](ssl-certificates.md) — detalha os procedimentos para configurar certificados com a API do servidor HTTP.

[Reservas, registros](namespace-reservations-registrations-and-routing.md) e roteamento do namespace — descreve o processo de reserva de partes do namespace para todos os aplicativos no computador usando a API do servidor HTTP. O processo para registrar e rotear solicitações para aplicativos específicos também é descrito.

[HttpCfg.exe](httpcfg-exe.md) — um programa de exemplo que usa as APIs de configuração. Uma versão compilada dessa ferramenta também é fornecidas com o sistema operacional entre as ferramentas de suporte, juntamente com um arquivo de ajuda que documenta como usá-lo.
