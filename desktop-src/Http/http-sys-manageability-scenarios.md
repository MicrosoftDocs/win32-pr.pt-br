# <a name="httpsys-manageability-scenarios"></a>Cenários de gerenciamento de HTTP.sys

Os cenários a seguir fornecem exemplos de como essas ferramentas podem ajudar os profissionais de ti a diagnosticar e resolver problemas relacionados ao componente de API do servidor HTTP (HTTP.sys). as ferramentas usadas nesses exemplos estão disponíveis no Windows Vista e no Windows Server 2008.

-   [Cenário 1: exemplo de tempo limite HTTP usando o rastreamento ETW e comandos netsh](scenario-1--http-timeout-example-using-etw-tracing-and-netsh-commands.md)
-   [Cenário 2: exemplo do analisador usando o rastreamento ETW](scenario-2--parser-example-using-etw-tracing.md)
-   [Cenário 3: contadores de desempenho](scenario-3--performance-counters.md)

Os cenários neste documento são exemplos de como usar as ferramentas de gerenciamento do componente de API do servidor HTTP. Esses exemplos não destinam-se a sugerir alterações de configuração específicas.
