# <a name="show-urlacl"></a>show urlacl

Lista as DACLs para a URL reservada especificada ou todas as URLs reservadas.

``` syntax
show urlacl [url=]string
 
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="_url__string"></span><span id="_URL__STRING"></span>**\[URL = \] * * * cadeia de caracteres*
</dt> <dd>

Especifica a URL totalmente qualificada. Se não for especificado, implica todas as URLs.

</dd> </dl>

## <a name="examples"></a>Exemplos

**Mostrar URL do urlacl =https://+:80/MyUrl**

**Mostrar URL do urlacl =https://www.contoso.com:80/MyUrl**
