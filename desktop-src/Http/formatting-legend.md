# <a name="formatting-legend"></a>Legenda de formatação

A seção usa as convenções de formatação a seguir.

| Formatar                                                                    | Significado                                                        |
|---------------------------------------------------------------------------|----------------------------------------------------------------|
| Itálico                                                                    | Informações que devem ser fornecidas pelo usuário                          |
| Negrito                                                                      | Elementos devem ser digitados pelo usuário exatamente como mostrados              |
| Reticências (...)                                                            | Parâmetro que pode ser repetido várias vezes em uma linha de comando |
| Entre colchetes ( \[ \] )                                                   | Itens opcionais                                                 |
| Entre chaves ( {} ); opções separadas por pipe ( \| ). Exemplo: {mesmo \| ímpar} | Conjunto de opções das quais o usuário deve escolher apenas uma        |
| Fonte Courier                                                              | Saída de código ou de programa                                         |
