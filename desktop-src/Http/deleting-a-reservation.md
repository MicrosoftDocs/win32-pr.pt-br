# <a name="deleting-a-reservation"></a>Excluindo uma reserva

O processo para remover uma reserva começa com uma verificação de acesso idêntica à descrita acima. A ID de usuário do processo de chamada deve ter direitos de acesso ao *nó* pai da entrada de reserva que está sendo excluída.
