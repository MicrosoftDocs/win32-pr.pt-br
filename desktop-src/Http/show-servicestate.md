# <a name="show-servicestate"></a>show servicestate

Mostra um instantâneo do serviço HTTP.

``` syntax
show servicestate [[view=]{session|requestq}] [[verbose=]{yes|no}]
 
```

## <a name="parameters"></a>Parâmetros

<dl> <dt>

<span id="__view___session_requestq__"></span><span id="__VIEW___SESSION_REQUESTQ__"></span>**\[\[View = \] {Session \| requestq}\]**
</dt> <dd>

Exiba o instantâneo do estado do serviço HTTP com base na sessão do servidor ou nas filas de solicitação.

</dd> <dt>

<span id="__verbose___yes_no__"></span><span id="__VERBOSE___YES_NO__"></span>**\[\[verbose = \] {Sim \| não}\]**
</dt> <dd>

Exiba informações detalhadas mostrando informações de propriedade também.

</dd> </dl>

## <a name="examples"></a>Exemplos

**Mostrar exibição de ServiceState = sessão**

**Mostrar exibição de ServiceState = requestq Verbose = Sim**
