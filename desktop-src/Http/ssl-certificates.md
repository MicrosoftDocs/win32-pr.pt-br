# <a name="ssl-certificates"></a><span data-ttu-id="647d4-104">Certificados SSL</span><span class="sxs-lookup"><span data-stu-id="647d4-104">SSL Certificates</span></span>

<span data-ttu-id="647d4-105">O protocolo SSL (SSL), também conhecido como TLS, tornou-se um padrão para proteger as conexões de Internet e é usado para impedir a espionagem na rede.</span><span class="sxs-lookup"><span data-stu-id="647d4-105">Secure Sockets Layer (SSL), also known as Transport Layer Security (TLS), has become a standard for securing Internet connections and is used to prevent eavesdropping on the network.</span></span> <span data-ttu-id="647d4-106">O protocolo SSL/TLS permite que um cliente e servidor autentiquem um ao outro e negociem algoritmos de criptografia.</span><span class="sxs-lookup"><span data-stu-id="647d4-106">The SSL/TLS protocol allows a client and server to authenticate each other and negotiate encryption algorithms.</span></span>

<span data-ttu-id="647d4-107">O SSL usa uma chave de criptografia e um algoritmo de criptografia para proteger a conexão HTTP.</span><span class="sxs-lookup"><span data-stu-id="647d4-107">SSL uses an encryption key and an encryption algorithm to secure the HTTP connection.</span></span> <span data-ttu-id="647d4-108">As chaves de criptografia estão contidas em certificados SSL usados pelo cliente e pelo servidor.</span><span class="sxs-lookup"><span data-stu-id="647d4-108">The encryption keys are contained in SSL certificates used by both the client and the server.</span></span> <span data-ttu-id="647d4-109">Normalmente, o certificado é um documento X. 509 ([RFC 2459](https://www.ietf.org/rfc/rfc2459.txt)).</span><span class="sxs-lookup"><span data-stu-id="647d4-109">The certificate is typically an X.509 ([RFC 2459](https://www.ietf.org/rfc/rfc2459.txt)) document.</span></span> <span data-ttu-id="647d4-110">O servidor fornece o certificado SSL para a sessão e envia o certificado para o cliente na fase de handshake.</span><span class="sxs-lookup"><span data-stu-id="647d4-110">The server provides the SSL certificate for the session and sends the certificate to the client in the handshake phase.</span></span> <span data-ttu-id="647d4-111">O cliente enviará seu certificado ao servidor somente se o servidor enviar uma solicitação ao cliente para um certificado.</span><span class="sxs-lookup"><span data-stu-id="647d4-111">The client sends its certificate to the server only if the server sends a request to the client for a certificate.</span></span> <span data-ttu-id="647d4-112">Assim, o cliente sempre autentica o servidor, mas o servidor tem a opção de autenticar ou não o cliente.</span><span class="sxs-lookup"><span data-stu-id="647d4-112">Thus the client always authenticates the server but the server has the option whether or not to authenticate the client.</span></span>

<span data-ttu-id="647d4-113">Os certificados de servidor devem ser armazenados no armazenamento persistente local da API do servidor HTTP, para uso cada vez que uma conexão segura é criada.</span><span class="sxs-lookup"><span data-stu-id="647d4-113">Server certificates must be stored in the HTTP Server API's local persistent storage, for use each time a secure connection is created.</span></span> <span data-ttu-id="647d4-114">Cada entrada de repositório de certificados também contém o endereço IP e a porta do servidor, o hash de certificado (usado para assinar as mensagens) e a ID do aplicativo.</span><span class="sxs-lookup"><span data-stu-id="647d4-114">Each certificate store entry also contains the IP Address and port of the server, the certificate hash (used for signing the messages) and the application ID.</span></span> <span data-ttu-id="647d4-115">A ID do aplicativo é usada para identificar o aplicativo que possui o certificado.</span><span class="sxs-lookup"><span data-stu-id="647d4-115">The application ID is used to identify the application that owns the certificate.</span></span>

<span data-ttu-id="647d4-116">Os administradores do sistema podem armazenar informações de certificado do servidor SSL com as APIs de configuração.</span><span class="sxs-lookup"><span data-stu-id="647d4-116">System administrators can store SSL server certificate information with the configuration APIs.</span></span> <span data-ttu-id="647d4-117">Uma ferramenta administrativa chama a função [**HttpSetServiceConfiguration**](/windows/desktop/api/Http/nf-http-httpsetserviceconfiguration) e especifica o valor de HttpServiceConfigSSLCertInfo para o parâmetro de configuração de serviço para definir informações para um certificado SSL.</span><span class="sxs-lookup"><span data-stu-id="647d4-117">An administrative tool calls the [**HttpSetServiceConfiguration**](/windows/desktop/api/Http/nf-http-httpsetserviceconfiguration) function and specifies HttpServiceConfigSSLCertInfo value for the service configuration parameter to set information for an SSL certificate.</span></span> <span data-ttu-id="647d4-118">Somente um certificado de servidor pode ser configurado para cada par de endereço IP e porta no computador.</span><span class="sxs-lookup"><span data-stu-id="647d4-118">Only one server certificate can be configured for each IP Address and port pair on the machine.</span></span> <span data-ttu-id="647d4-119">A API do servidor HTTP também fornece funções de [**consulta**](/windows/desktop/api/Http/nf-http-httpqueryserviceconfiguration) e [**exclusão**](/windows/desktop/api/Http/nf-http-httpdeleteserviceconfiguration) para acessar ou excluir certificados existentes.</span><span class="sxs-lookup"><span data-stu-id="647d4-119">The HTTP Server API also provides [**query**](/windows/desktop/api/Http/nf-http-httpqueryserviceconfiguration) and [**delete**](/windows/desktop/api/Http/nf-http-httpdeleteserviceconfiguration) functions to access or delete existing certificates.</span></span>
