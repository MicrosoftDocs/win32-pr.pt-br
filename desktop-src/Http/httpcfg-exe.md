# <a name="httpcfgexe"></a>HttpCfg.exe

Essa ferramenta ajuda você a configurar a API do Servidor HTTP, envolvendo as [APIs](configuration-apis.md) de configuração HTTP para os seguintes parâmetros:

-   Lista de escuta de IP
-   Certificados SSL
-   Reservas de namespace

HttpCfg.exe com o sistema operacional a partir do Windows Server 2003 como um executável binário, junto com um arquivo de ajuda que descreve como usá-lo. Ele pode ser instalado do disco de distribuição do sistema operacional como uma ferramenta de suporte.

HttpCfg.exe também está disponível como um download; consulte [Windows ferramentas de suporte do XP SP2](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38).

**Windows Vista e Windows Server 2008:** HttpCfg.exe está obsoleto e substituído por uma funcionalidade semelhante no netsh.exe.
