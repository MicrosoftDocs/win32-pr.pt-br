# <a name="httpcfgexe"></a>HttpCfg.exe

Essa ferramenta ajuda a configurar a API do servidor HTTP encapsulando as [APIs de configuração](configuration-apis.md) de http para os seguintes parâmetros:

-   Lista de escuta de IP
-   Certificados SSL
-   Reservas de namespace

O HttpCfg.exe é fornecido com o sistema operacional a partir do Windows Server 2003 como um executável binário, junto com um arquivo de ajuda que descreve como usá-lo. Ele pode ser instalado do disco de distribuição do sistema operacional como uma ferramenta de suporte.

O HttpCfg.exe também está disponível como um download; consulte [ferramentas de suporte do Windows XP SP2](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38).

**Windows Vista e Windows Server 2008:** o HttpCfg.exe é obsoleto e substituído por uma funcionalidade semelhante no netsh.exe.
