<span data-ttu-id="deb8d-104">Reserva e registro são as operações pelas quais a API do servidor HTTP dá acesso ao namespace da URL em um computador.</span><span class="sxs-lookup"><span data-stu-id="deb8d-104">Reservation and registration are the operations by which the HTTP Server API gives access to the URL namespace on a machine.</span></span> <span data-ttu-id="deb8d-105">Os aplicativos podem se registrar para uma parte do namespace da URL a fim de atender a solicitações de clientes HTTP.</span><span class="sxs-lookup"><span data-stu-id="deb8d-105">Applications can usando a função [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl) .</span><span class="sxs-lookup"><span data-stu-id="deb8d-106">The application registers a namespace with the HTTP Server API using the [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl) function.</span></span> <span data-ttu-id="deb8d-107">A API do servidor HTTP adiciona as URLs à fila de solicitações para o <span data-ttu-id="deb8d-114">Os namespaces reservados são identificados por cadeias de caracteres de prefixo de URL, formatados da mesma maneira que os prefixos de URL usados para os registros.</span><span class="sxs-lookup"><span data-stu-id="deb8d-114">Reserved namespaces are identified by URL prefix strings, formatted in the same fashion as URL prefixes used for registrations.</span></span> <span data-ttu-id="deb8d-115">Isso significa que todas as várias categorias de especificador de host também estão disponíveis para reservas.</span><span class="sxs-lookup"><span data-stu-id="deb8d-115">This means that all the various host specifier categories are also available for reservations.</span></span>

-   <span data-ttu-id="deb8d-118">Falta.</span><span class="sxs-lookup"><span data-stu-id="deb8d-118">REGISTRATION.</span></span> <span data-ttu-id="deb8d-119">O registro é a operação pela qual um aplicativo indica interesse no recebimento de solicitações para um UrlPrefix especificado.</span><span class="sxs-lookup"><span data-stu-id="deb8d-119">Registration is the operation by which an application indicates interest in receiving requests for a specified UrlPrefix.</span></span> <span data-ttu-id="deb8d-120">A API para o registro de URL é [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl).</span><span class="sxs-lookup"><span data-stu-id="deb8d-120">The API for URL registration is [**HttpAddUrl**](/windows/desktop/api/Http/nf-http-httpaddurl).</span></span> <span data-ttu-id="deb8d-121">Normalmente, o registro ocorre durante a inicialização do aplicativo e deve ser executado cada vez que o aplicativo é iniciado.</span><span class="sxs-lookup"><span data-stu-id="deb8d-121">Registration typically occurs during application startup and must be performed each time the application starts.</span></span>

-   <span data-ttu-id="deb8d-125">Contra.</span><span class="sxs-lookup"><span data-stu-id="deb8d-125">RESERVATION.</span></span> <span data-ttu-id="deb8d-126">A reserva aloca uma parte do namespace da URL para um ou mais usuários.</span><span class="sxs-lookup"><span data-stu-id="deb8d-126">Reservation allocates a portion of the URL namespace to one or more users.</span></span> <span data-ttu-id="deb8d-127">Esta operação fornece aos usuários o direito de se registrar para o namespace especificado.</span><span class="sxs-lookup"><span data-stu-id="deb8d-127">This operation gives users the right to register for the specified namespace.</span></span> <span data-ttu-id="deb8d-128">Um usuário para o qual um namespace é reservado é dito como "proprietário" dessa parte do namespace da URL.</span><span class="sxs-lookup"><span data-stu-id="deb8d-128">A user for whom a namespace is reserved is said to "own" that part of URL namespace.</span></span> <span data-ttu-id="deb8d-129">As reservas de namespace normalmente são executadas durante a instalação do aplicativo e são uma operação infrequente.</span><span class="sxs-lookup"><span data-stu-id="deb8d-129">Namespace reservations are typically performed during the installation of the application and are an infrequent operation.</span></span> <span data-ttu-id="deb8d-130">As reservas persistem entre reinicializações do computador e exigem privilégios de administrador no computador ou na propriedade com privilégios de delegação para criar ou excluir.</span><span class="sxs-lookup"><span data-stu-id="deb8d-130">Reservations persist across reboots of the machine and require administrator privileges on the machine or ownership with delegation privileges to create or delete.</span></span>
