# <a name="http-version-20-features"></a>Recursos de HTTP versão 2,0

O recurso a seguir tem suporte para a API do servidor HTTP versão 2,0.

-   Objetos de configuração de grupo de URL e de sessão de servidor
-   Propriedades de configuração expostas nos objetos de configuração
-   Gerenciamento e configuração avançados da fila de solicitações
-   SSL de modo kernel completo
-   Cache de resposta do modo kernel
-   Autenticação no lado do servidor
-   Controle de versão com escopo de objeto
-   Registro no lado do servidor
-   Desligamento normal da fila de solicitações
-   Início da demanda na fila de solicitações
-   Compartilhamento de porta
-   rastreamento de eventos para suporte a Windows
-   Contadores de desempenho
-   Suporte a IDN (nome de domínio internacionalizado)
-   Utilitário de configuração do NET Shell
