# <a name="using-http-server-api"></a>Usando a API do servidor HTTP

Os arquivos de origem do programa de aplicativo incluem o arquivo de cabeçalho http. h para acessar protótipos de função e definições de estrutura para a API do servidor HTTP. Os desenvolvedores podem usar o arquivo de biblioteca Httpapi. lib para criar aplicativos que usam a API do servidor HTTP. Em tempo de execução, os aplicativos são vinculados ao Httpapi.dll.
