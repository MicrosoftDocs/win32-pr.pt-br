# <a name="using-teredo"></a>Usando Teredo

Ao desenvolver um aplicativo que usa [Teredo](about-teredo.md), o objetivo é obter conectividade IPv6 e não se concentrar no uso do Teredo. Por exemplo, o espaço de reunião do Windows e a assistência remota são dois aplicativos que aproveitam a conectividade de ponta a ponta fornecida pelo IPv6. Esses aplicativos podem obter conectividade usando endereços IPv6, 6to4 ou Teredo nativos. Para esses aplicativos e quaisquer outros que possam ser desenvolvidos para IPv6, o Teredo é tratado como qualquer outro protocolo que permita conectividade.
