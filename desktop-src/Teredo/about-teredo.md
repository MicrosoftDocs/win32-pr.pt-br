# <a name="about-teredo"></a>Sobre o Teredo

Devido ao número limitado de endereços IPv4 disponíveis, o ISP (Provedores de Serviços de Internet) precisará iniciar a transição para o IPv6. Windows O Vista fornecerá suporte completo para IPv6, permitindo que os desenvolvedores verifiquem se seus aplicativos estão prontos para amanhã, habilitando o suporte a IPv6 hoje. Com o Teredo, Windows áreas de trabalho do Vista são capazes de conectividade IPv6, mesmo que as redes ISP sejam somente IPv4.

Tecnologias como Ponto a Ponto exigem conectividade de ponta a ponta para a experiência completa do usuário. Muitas residências têm dispositivos NAT (Conversão de Endereços de Rede) que limitam essas tecnologias e os desenvolvedores de aplicativos têm que se concentrar em encontrar maneiras eficientes de solucionar esses problemas. No entanto, o suporte ao Teredo com Windows Vista reduz a carga sobre os desenvolvedores de aplicativos fornecendo uma travessia nat perfeita.
