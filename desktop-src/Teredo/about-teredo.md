# <a name="about-teredo"></a>Sobre o Teredo

Devido ao número limitado de endereços IPv4 disponíveis, os provedores de serviços de Internet (ISP) precisarão iniciar a transição para o IPv6. O Windows Vista fornecerá suporte completo para IPv6, permitindo que os desenvolvedores garantam que seus aplicativos estejam prontos para o futuro, habilitando o suporte a IPv6 hoje mesmo. Com o Teredo, as áreas de trabalho do Windows Vista são capazes de conectividade IPv6, mesmo que as redes do ISP sejam somente IPv4.

Tecnologias como ponto a ponto exigem conectividade de ponta a ponta para a experiência completa do usuário. Muitas residências têm dispositivos NAT (conversão de endereços de rede) que limitam essas tecnologias e os desenvolvedores de aplicativos precisam se concentrar em encontrar maneiras eficientes de solucionar esses problemas. No entanto, o suporte a Teredo com o Windows Vista reduz a carga dos desenvolvedores de aplicativos fornecendo percurso NAT contínuo.
