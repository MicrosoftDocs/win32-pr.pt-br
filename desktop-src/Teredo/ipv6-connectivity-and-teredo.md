# <a name="ipv6-connectivity-and-teredo"></a>Conectividade IPv6 e Teredo

A maioria das redes é somente IPv4 e a conectividade IPv6 de ponta a ponta não é facilmente obtida. Além disso, a maioria dos dispositivos NAT até limita a conectividade IPv4 de ponta a ponta. Com Windows Vista, a conectividade de ponta a ponta IPv6 entre dois hosts é possível. Os desenvolvedores não têm mais que se preocupar com a conectividade de ponta a ponta, mas sim criar novos cenários de aplicativo que permitem que eles aproveitem essa conectividade.

A meta do Teredo, como uma tecnologia de transição IPv6, é tornar a conectividade IPv6 de ponta a ponta simples e automática. Em muitos casos, os desenvolvedores talvez nem precisem saber que tecnologias de transição como Teredo existem no Windows Vista. Windows O Vista tem a capacidade de usar o Teredo em segundo plano como um método contínuo de conectividade IPv6.
