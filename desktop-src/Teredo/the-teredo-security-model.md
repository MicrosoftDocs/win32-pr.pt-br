# <a name="the-teredo-security-model"></a>O modelo de segurança Teredo

A tecnologia [Teredo](about-teredo.md) , quando associada à central de segurança do Windows e ao [Firewall do Windows](/previous-versions/windows/desktop/ics/windows-firewall-start-page), fornece segurança aprimorada para o usuário final. Para o Firewall do Windows, o tráfego IPv6 enviado e recebido pela interface Teredo é processado da mesma forma que o tráfego IPv6 nativo. A interface Teredo é reconhecida como outra mini-porta no sistema.

Para obter detalhes sobre o uso, consulte [implementando o modelo de segurança Teredo](implementing-the-teredo-security-model.md).
