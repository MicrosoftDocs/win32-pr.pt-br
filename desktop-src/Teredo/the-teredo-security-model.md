# <a name="the-teredo-security-model"></a>O modelo de segurança teredo

A [tecnologia Teredo,](about-teredo.md) quando juntamente com o Segurança do Windows Center [e Windows Firewall](/previous-versions/windows/desktop/ics/windows-firewall-start-page), fornece segurança aprimorada para o usuário final. Para Windows Firewall, o tráfego IPv6 enviado e recebido pela interface Teredo é processado da mesma forma que o tráfego IPv6 nativo. A interface Teredo é reconhecida como outra miniporta no sistema.

Para obter detalhes sobre o uso, [consulte Implementando o modelo de segurança teredo](implementing-the-teredo-security-model.md).
