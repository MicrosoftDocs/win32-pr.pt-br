# <a name="the-private-data-object-type"></a>O tipo de objeto de dados privados

Você pode usar objetos de dados privados para armazenar cadeias de caracteres de dados em um formato criptografado. Um sistema pode conter vários objetos de dados privados. Os objetos de dados privados são fornecidos principalmente para dar suporte ao armazenamento de senhas de conta de servidor. As informações armazenadas em objetos de dados privados são criptografadas usando uma chave específica do sistema gerada.
