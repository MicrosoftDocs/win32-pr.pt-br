# <a name="the-private-data-object-type"></a>O tipo de objeto de dados privado

Você pode usar objetos dados privados para armazenar cadeias de caracteres de dados em um formulário criptografado. Um sistema pode conter vários objetos dados privados. Os objetos dados privados são fornecidos principalmente para dar suporte ao armazenamento de senhas de conta de servidor. As informações armazenadas em objetos de Dados Privados são criptografadas usando uma chave específica do sistema gerada.
