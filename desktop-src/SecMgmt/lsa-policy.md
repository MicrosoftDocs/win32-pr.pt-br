# <a name="lsa-policy"></a>Política de LSA

A [*autoridade de segurança local*](/windows/desktop/SecGloss/l-gly) (LSA) é um subsistema protegido do Windows que mantém informações sobre todos os aspectos da segurança local em um sistema, coletivamente conhecido como a [diretiva de segurança local](local-security-policy.md) do sistema. Além das informações de política de hospedagem, a LSA fornece serviços para conversão entre nomes e SIDs ( [*identificadores de segurança*](/windows/desktop/SecGloss/s-gly) ).
