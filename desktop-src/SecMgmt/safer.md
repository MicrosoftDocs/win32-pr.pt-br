# <a name="safer"></a>Mais seguro

As funções na API mais segura fornecem qualquer aplicativo que inicia programas de fontes externas a capacidade de consultar a política de segurança para aprovação antes de um executável ser iniciado. Essas funções podem ser chamadas antes do carregamento e execução de um executável ou conteúdo ativo. Os tipos de aplicativos em que a API mais segura é útil incluem aplicativos que manipulam anexos (como clientes de email e mensagens instantâneas que podem transferir arquivos) e interpretadores de script. O suporte para funções na API mais segura começa com o Windows XP. Para obter mais informações, consulte [funções mais seguras](management-functions.md).
