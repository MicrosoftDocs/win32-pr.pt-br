# <a name="using-service-security-attachments"></a>Usando anexos de segurança de serviço

Esta seção descreve como criar os componentes de anexo necessários para dar suporte a uma configuração específica do serviço nos snap-ins de configuração de segurança. Será necessário implementar e registrar dois componentes: um snap-in de extensão para exibir uma interface do usuário e um mecanismo de anexo para processar as solicitações de configuração e análise.

Para obter mais informações, consulte estes tópicos:

-   [Criando uma extensão de snap-in de anexo](creating-an-attachment-snap-in-extension.md)
-   [Registrando uma extensão de snap-in de anexo](registering-an-attachment-snap-in-extension.md)
-   [Criando um mecanismo de anexo](creating-an-attachment-engine.md)
-   [Registrando um mecanismo de anexo](registering-an-attachment-engine.md)
