# <a name="scestatus"></a>SCESTATUS

O tipo de dados **SCESTATUS** é usado pela API do conjunto de ferramentas de configuração de segurança para retornar informações sobre os resultados de uma chamada de função. Para obter mais informações sobre os valores de código de status definidos, consulte [valores de retorno de anexo](management-return-values.md).


```C++
typedef DWORD SCESTATUS;
```
