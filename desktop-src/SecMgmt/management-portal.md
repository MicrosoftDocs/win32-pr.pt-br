# <a name="security-management"></a>Gerenciamento de Segurança

## <a name="purpose"></a>Finalidade

As tecnologias de gerenciamento de segurança podem ser usadas [](/windows/desktop/SecGloss/p-gly) para gerenciar a política de LSA (Autoridade de Segurança [*Local)*](/windows/desktop/SecGloss/l-gly) e a política de filtro de senha, consultar a capacidade de programas de fontes externas e anexos de serviço que estendem a funcionalidade da ferramenta configuração de segurança.

As tecnologias de gerenciamento de segurança da Microsoft incluem a API de Política de LSA, a API de Filtro de Senha, a API mais Segura e a API de Anexos de Segurança de Serviço. Essas tecnologias permitem que os desenvolvedores de software criem aplicativos que gerenciam sistemas e aplicativos.

## <a name="developer-audience"></a>Público de desenvolvedores

A API de Política LSA, a API de Filtro de Senha, a API mais Segura e a API de Anexos de Segurança de Serviço destinam-se ao uso por desenvolvedores de aplicativos que permitem que os administradores gerenciem e protejam seus sistemas. Os desenvolvedores devem estar familiarizados com as linguagens de programação C e C++ e com Windows com base em programação.
