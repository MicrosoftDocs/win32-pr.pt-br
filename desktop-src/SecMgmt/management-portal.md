# <a name="security-management"></a>Gerenciamento de Segurança

## <a name="purpose"></a>Finalidade

As tecnologias de gerenciamento de segurança podem ser usadas para gerenciar política de [*autoridade de segurança local*](/windows/desktop/SecGloss/l-gly) (LSA) e política de [*filtro de senha*](/windows/desktop/SecGloss/p-gly) , consultar a capacidade de programas de fontes externas e anexos de serviço que estendem a funcionalidade da ferramenta de configuração de segurança.

As tecnologias de gerenciamento de segurança da Microsoft incluem a API de diretiva LSA, a API de filtro de senha, a API mais segura e a API de anexos de segurança de serviço. Essas tecnologias permitem que os desenvolvedores de software criem aplicativos que gerenciam sistemas e aplicativos.

## <a name="developer-audience"></a>Público de desenvolvedores

A API de diretiva LSA, a API de filtro de senha, a API mais segura e a API de anexos de segurança de serviço são destinadas a serem usadas por desenvolvedores de aplicativos que permitem que os administradores gerenciem e protejam seus sistemas. Os desenvolvedores devem estar familiarizados com as linguagens de programação C e C++ e com a programação baseada no Windows.
