# <a name="account-object"></a>Objeto de conta

O objeto **Account** é usado para atribuir [*privilégios*](/windows/desktop/SecGloss/p-gly), acesso ao sistema e cotas especiais a usuários individuais ou a membros de grupos ou grupos locais.

As informações armazenadas em um objeto de **conta** incluem:

-   O SID de uma conta de usuário, conta de grupo ou conta de grupo local.
-   Uma lista de privilégios a serem atribuídos ao usuário ou aos membros de um grupo. Se um usuário receber privilégios por vários objetos de **conta** , a lista incluirá todos os privilégios atribuídos.
