# <a name="modifying-configuration-information-in-the-database"></a>Modificando informações de configuração no banco de dados

A extensão do snap-in de anexos deve permitir que os usuários alterem informações de configuração por meio de seu nó de análise. Isso permite que os usuários façam alterações e apliquem novamente a configuração ao longo do tempo. A lógica para fazer isso deve ser idêntica à modificação de informações nos arquivos de configuração. As alterações feitas entrarão em vigor na configuração salva no banco de dados de segurança.
