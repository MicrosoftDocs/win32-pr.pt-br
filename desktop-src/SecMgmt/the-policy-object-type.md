# <a name="the-policy-object-type"></a>O tipo de objeto de política

O objeto de [**política**](policy-object.md) é um objeto de contêiner. Os seguintes tipos de objeto podem residir nesse contêiner:

-   Objetos de [**conta**](account-object.md)
-   Objeto [**TrustedDomain**](trusteddomain-object.md)
-   Objeto secreto

Há apenas uma única instância do objeto de [**política**](policy-object.md) . Ele não pode ser criado nem destruído.
