# <a name="the-policy-object-type"></a>O tipo de objeto Policy

O [**objeto Policy**](policy-object.md) é um objeto de contêiner. Os seguintes tipos de objeto podem residir dentro desse contêiner:

-   [**Objetos de**](account-object.md) conta
-   [**Objeto TrustedDomain**](trusteddomain-object.md)
-   Objeto de segredo

Há apenas uma única instância do [**objeto Policy.**](policy-object.md) Ele não pode ser criado nem destruído.
