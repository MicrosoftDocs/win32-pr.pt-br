# <a name="the-account-object-type"></a>O tipo de objeto account

[**Os**](account-object.md) objetos de conta são usados para significar privilégios ou direitos de acesso do sistema que devem ser [*concedidos*](/windows/desktop/SecGloss/p-gly)a um usuário ou membros de grupos ou grupos locais.

[**Os objetos**](account-object.md) de conta residem dentro [**do objeto Política.**](policy-object.md) Pode haver zero ou mais desses objetos.

O [**objeto Account**](account-object.md) não tem nenhum tipo de dados específico do objeto.
