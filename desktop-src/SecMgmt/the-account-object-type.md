# <a name="the-account-object-type"></a>O tipo de objeto da conta

Os objetos de [**conta**](account-object.md) são usados para significar [*privilégios*](/windows/desktop/SecGloss/p-gly)ou direitos de acesso do sistema que devem ser conferidasdos em um usuário ou em membros de grupos ou grupos locais.

Os objetos de [**conta**](account-object.md) residem no objeto de [**política**](policy-object.md) . Pode haver zero ou mais desses objetos.

O objeto de [**conta**](account-object.md) não tem tipos de dados específicos de objeto.
