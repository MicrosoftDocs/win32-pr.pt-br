# <a name="security-templates"></a>Modelos de segurança

Os Modelos de Segurança são uma ferramenta de snap-in autônoma que os usuários podem usar para definir configurações de segurança independentes de computador. Essas configurações são salvas como arquivos .inf baseados em texto.

Os Modelos de Segurança e a [Configuração de](security-configuration-and-analysis.md) Segurança e a Análise comem os snap-ins de Configuração de Segurança.
