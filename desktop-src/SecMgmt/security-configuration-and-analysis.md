# <a name="security-configuration-and-analysis"></a>Configuração e análise de segurança

A configuração e análise de segurança é uma ferramenta de snap-in autônoma que os usuários podem usar para importar uma ou mais configurações salvas para um banco de dados de segurança privada. A importação de configurações cria um banco de dados de segurança específico do computador que armazena uma configuração composta. Você pode aplicar essa configuração composta ao computador e analisar a configuração atual do sistema em relação à configuração composta armazenada armazenada no banco de dados.

Os [modelos de segurança](security-templates.md) e a configuração e análise de segurança compõem os snap-ins de configuração de segurança.
