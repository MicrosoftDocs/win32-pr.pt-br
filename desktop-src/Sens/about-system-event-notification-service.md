# <a name="about-system-event-notification-service"></a>Sobre o serviço de notificação de eventos do sistema

Os aplicativos projetados para uso por usuários móveis exigem um conjunto exclusivo de funções e notificações de conectividade, e nos aplicativos individuais anteriores eram necessários para implementar esses recursos internamente. O SENS (serviço de notificação de eventos do sistema) agora fornece esses recursos no sistema operacional, criando uma interface de notificação e conectividade uniforme para aplicativos. O SENS, o Gerenciador de sincronização e o cache do lado do cliente se combinam para fornecer a infraestrutura para oferecer suporte total à computação móvel.

Os tópicos desta seção oferecem uma visão geral sobre o SENS:

    Lista várias configurações de computadores móveis que podem se beneficiar do uso de aplicativos habilitados para SENS.

-   [Cenários de aplicativo para o SENS](application-scenarios-for-sens.md)

    Lista vários tipos de aplicativos que podem se beneficiar utilizando a funcionalidade SENS.

-   [Notificações](notifications.md)

    Lista os eventos do sistema que o SENS monitora e despacha.

-   [Arquitetura SENS](sens-architecture.md)

    Uma descrição da arquitetura interna do SENS.
