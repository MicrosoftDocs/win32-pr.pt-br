# <a name="mobile-computing-configurations-for-sens"></a>Configurações de computação móvel para o SENS

As funções de conectividade e as notificações são úteis para computadores configurados da seguinte maneira:

-   Um computador móvel usado em uma estação de encaixe em uma rede de alta largura de banda que pode, ocasionalmente, usar uma conexão de discagem.
-   Um computador móvel usando uma conexão de discagem exclusivamente.
-   Um computador desktop usando uma conexão dial-in exclusivamente.
-   Um computador desktop conectado a uma rede de largura de banda alta com problemas de latência.

Em cada uma dessas configurações, a largura de banda de conexão e as informações de latência podem ser usadas por um aplicativo para otimizar dinamicamente suas operações de disponibilidade de rede.
