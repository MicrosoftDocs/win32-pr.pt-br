# <a name="ivectorchangedeventargs-interface"></a>Interface IVectorChangedEventArgs

Fornece dados para um evento de [**vetorchanged**](/uwp/api/windows.foundation.collections.iobservablevector-1?view=winrt-19041) .

## <a name="members"></a>Membros

A interface **IVectorChangedEventArgs** herda de **IInspectable**. **IVectorChangedEventArgs** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IVectorChangedEventArgs** tem esses métodos.



| Método                                                                        | Descrição                                                           |
|:------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| [**obter \_ CollectionChange**](ivectorchangedeventargs-get-collectionchange.md) | Obtém o tipo de alteração que ocorreu no vetor.<br/>       |
| [**obter \_ índice**](ivectorchangedeventargs-get-index.md)                       | Obtém a posição no vetor em que a alteração ocorreu.<br/> |
