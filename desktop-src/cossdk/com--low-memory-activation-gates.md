# <a name="com-low-memory-activation-gates"></a>Grupos de ativação do COM+ Low-Memory

O serviço de entradas de ativação de baixa memória do COM+ verifica automaticamente a memória antes de um servidor ou objeto COM+ ser criado. Se a porcentagem de memória virtual disponível para o aplicativo ficar abaixo de um limite fixo, a ativação falhará antes de o objeto ser criado. Dessa forma, o serviço de Gates de ativação de baixa memória aprimora muito a confiabilidade do sistema.
