# <a name="com-queued-components"></a>Componentes em fila COM+

O recurso de componentes em fila COM+ permite que um cliente solicite um serviço de um aplicativo de servidor COM+, mesmo quando o aplicativo de servidor está temporariamente indisponível. A solicitação do cliente é armazenada em uma fila e processada quando o servidor se torna disponível.

> [!Note]  
> Para usar o serviço de componentes na fila COM+, você deve ter o [enfileiramento de mensagens](/previous-versions/windows/desktop/legacy/ms711472(v=vs.85)) instalado no computador.
