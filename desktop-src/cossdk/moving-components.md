# <a name="moving-components"></a>Movendo componentes

Você pode mover um componente de um aplicativo COM+ para outro. Na verdade, mover um componente remove o componente do aplicativo COM+ original e o instala no aplicativo COM+ de destino.

**Para mover um componente de um aplicativo COM+ para outro**

1.  No painel de detalhes da ferramenta administrativa serviços de componentes, clique com o botão direito do mouse no componente que você deseja mover e clique em **mover**.

2.  Na caixa de diálogo **mover componente (s)** , selecione o aplicativo com+ de destino para o qual você deseja que o componente seja movido.

3.  Clique em **OK**.
