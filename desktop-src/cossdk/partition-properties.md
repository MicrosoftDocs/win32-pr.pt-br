# <a name="partition-properties"></a>Propriedades da partição

As partições têm cinco propriedades disponíveis. Essas propriedades e suas descrições são as seguintes:

<dl> <dt>

<span id="Partition_Name"></span><span id="partition_name"></span><span id="PARTITION_NAME"></span>Nome da partição
</dt> <dd>

Uma cadeia de texto associada à partição.

</dd> <dt>

<span id="Partition_ID"></span><span id="partition_id"></span><span id="PARTITION_ID"></span>ID da partição
</dt> <dd>

Um GUID que o sistema usa para correlacionar um usuário a uma partição.

</dd> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>Ndescrição
</dt> <dd>

Uma descrição textual da partição.

</dd> <dt>

<span id="Disable_Deletion"></span><span id="disable_deletion"></span><span id="DISABLE_DELETION"></span>Desabilitar exclusão
</dt> <dd>

Um mecanismo para proibir um usuário de excluir uma partição.

</dd> <dt>

<span id="Disable_Changes"></span><span id="disable_changes"></span><span id="DISABLE_CHANGES"></span>Desabilitar alterações
</dt> <dd>

Um mecanismo para proibir um usuário de fazer alterações em uma partição.

</dd> </dl>

Essas propriedades são visíveis clicando com o botão direito do mouse em um nome de partição na ferramenta administrativa serviços de componentes e escolhendo **Propriedades**.

Para obter mais informações sobre a propriedade de ID de partição, consulte [registrando e ativando componentes em partições](registering-and-activating-components-in-partitions.md).
