# <a name="local-partitions"></a>Partições locais

Uma maneira de usar partições pode ser usada localmente em um servidor de aplicativos. Nesse cenário, um administrador pode criar uma partição usando a ferramenta administrativa serviços de componentes. As partições implementadas localmente e não dentro de Active Directory são acessíveis somente para usuários locais, não para usuários em outros computadores no domínio. Depois que uma partição é criada e um aplicativo COM+ é instalado nela, o administrador pode atribuir essa partição como uma partição padrão para um ou mais usuários nesse servidor. Para atribuir uma partição padrão a um usuário, o administrador pode usar a ferramenta administrativa serviços de componentes.
