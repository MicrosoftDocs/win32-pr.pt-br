# <a name="com-shared-property-manager-sample"></a>Exemplo de Gerenciador de Propriedades compartilhado do COM+

Para obter o código de exemplo sobre como usar o Gerenciador de Propriedades compartilhado do COM+, confira [**ISharedProperty::p \_ valor de UT**](/windows/desktop/api/ComSvcs/nf-comsvcs-isharedproperty-put_value). Este exemplo de C++ mostra como criar um grupo de propriedades compartilhado, criar uma propriedade compartilhada dentro desse grupo e, em seguida, definir o valor dessa propriedade.
