# <a name="com-resource-dispenser-interfaces"></a>Interfaces do dispensador de recursos COM+

Os componentes do aplicativo usam dispensadores de recursos para acessar informações compartilhadas. As interfaces descritas na tabela a seguir fornecem informações de referência detalhadas para desenvolvedores de dispensadores de recursos.



| Interfaces                                                | Descrição                                                                                                               |
|-----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**IDispenserDriver**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispenserdriver)<br/>   | Essa interface é chamada pelo titular do dispensador de recursos para criar, inscrever, avaliar e destruir um recurso.<br/> |
| [**IDispenserManager**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager)<br/> | Os dispensadores de recursos usam essa interface para se conectar ao Gerenciador do dispensador.<br/>                                    |
| [**IHolder**](/windows/desktop/api/ComSvcs/nn-comsvcs-iholder)<br/>                     | Essa interface é usada para preparar e manipular recursos.<br/>                                                        |
