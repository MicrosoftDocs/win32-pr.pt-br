# <a name="com-resource-dispenser-interfaces"></a>COM+ Resource Dispenser Interfaces

Os componentes do aplicativo usam os distribuidores de recursos para acessar informações compartilhadas. As interfaces descritas na tabela a seguir fornecem informações de referência detalhadas para desenvolvedores de distribuidores de recursos.



| Interfaces                                                | Descrição                                                                                                               |
|-----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**IDispenserDriver**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispenserdriver)<br/>   | Essa interface é chamada pelo titular do distribuidor de recursos para criar, inscrever, avaliar e destruir um recurso.<br/> |
| [**IDispenserManager**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager)<br/> | Os distribuidores de recursos usam essa interface para se conectar ao gerenciador do distribuidor.<br/>                                    |
| [**IHolder**](/windows/desktop/api/ComSvcs/nn-comsvcs-iholder)<br/>                     | Essa interface é usada para preparar e manipular recursos.<br/>                                                        |
