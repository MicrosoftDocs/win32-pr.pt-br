# <a name="managing-users-in-roles"></a>Gerenciando usuários em funções

A maioria das tarefas de administração para configurar a segurança de aplicativos COM+ envolve o preenchimento de funções predefinidas com usuários, tanto ao instalar aplicativos quanto em uma base contínua. Para obter informações introdutórias sobre como esses recursos de segurança funcionam, consulte [Configurando Role-Based segurança](configuring-role-based-security.md).

Nesta seção, são fornecidas instruções passo a passo para as seguintes tarefas:

-   [Habilitando a verificação de autorização de Role-Based](enabling-role-based-authorization-checking.md)
-   [Atribuindo uma conta de usuário ou grupo a uma função](assigning-a-user-account-or-group-to-a-role.md)
-   [Removendo uma conta de usuário ou grupo de uma função](removing-a-user-account-or-group-from-a-role.md)
-   [Movendo uma conta de usuário ou grupo para uma função diferente](moving-a-user-account-or-group-to-a-different-role.md)
