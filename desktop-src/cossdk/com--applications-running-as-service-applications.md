# <a name="com-applications-running-as-service-applications"></a>Aplicativos COM+ em execução como aplicativos de serviço

Os desenvolvedores de COM+ podem usar a ferramenta administrativa serviços de componentes para configurar e implementar um aplicativo de servidor COM+ como um aplicativo de serviço. Isso significa que o servidor poderá ser iniciado automaticamente ou reiniciado se seu aplicativo sempre precisar ser executado; que seu aplicativo COM+ possa ser executado como a conta do sistema local se precisar executar operações privilegiadas; e que os serviços dependentes de seu aplicativo agora podem ser iniciados automaticamente.

Os tópicos a seguir fornecem informações em segundo plano e tarefas sobre como configurar seus aplicativos para executar como aplicativos de serviço.

| [Conceitos do aplicativo de serviço COM+](com--service-application-concepts.md) | Fornece uma visão geral dos benefícios de executar um aplicativo de servidor do COM+ como um aplicativo de serviço. |
| [Tarefas do aplicativo de serviço COM+](com--service-application-tasks.md)       | Fornece instruções para configurar um aplicativo de servidor do COM+ como um serviço.                       |
