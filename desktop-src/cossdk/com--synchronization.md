# <a name="com-synchronization"></a>Sincronização COM+

O serviço de sincronização COM+ fornece sincronização que flui do componente para o componente e proíbe que mais de um chamador entre no componente a qualquer momento. A sincronização determina quando os threads podem expedir chamadas para um objeto .
