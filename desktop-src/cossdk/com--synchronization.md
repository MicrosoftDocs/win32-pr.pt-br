# <a name="com-synchronization"></a>Sincronização COM+

O serviço de sincronização COM+ fornece sincronização que flui de componente para componente e proíbe que mais de um chamador insira o componente em um determinado momento. A sincronização determina quando os threads podem distribuir chamadas para um objeto.
