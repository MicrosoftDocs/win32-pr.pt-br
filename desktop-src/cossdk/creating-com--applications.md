# <a name="creating-com-applications"></a>Criando aplicativos COM+

Esta seção descreve como criar um novo aplicativo COM+ (vazio) e, em seguida, adicionar componentes a esse aplicativo. Ele também descreve como adicionar componentes COM e removê-los de aplicativos COM+ existentes e como mover e copiar componentes de um aplicativo COM+ para outro.

| [Copiando componentes](copying-components.md)<br/>                                                     | Descreve como copiar um componente de um aplicativo COM+ para outro.<br/> |
| [Excluindo um aplicativo COM+](deleting-a-com--application.md)<br/>                                   | Descreve como excluir um aplicativo COM+.<br/>                             |
