# <a name="com-instrumentation-interfaces"></a>Interfaces de instrumentação COM+

As métricas de instrumentação COM+ são publicadas em um cliente usando [eventos com+](com--events.md), um sistema LCE (eventos menos acoplados) que armazena informações de eventos de editores diferentes em um repositório de eventos no catálogo com+. As métricas são agrupadas em categorias com base no tipo de objeto para o qual fornecem informações. Para cada categoria, há uma interface correspondente à qual o cliente pode assinar e o cliente de assinatura recebe todas as métricas nessa categoria. O cliente também pode assinar métodos específicos na interface para receber métricas específicas por meio de métricas de filtragem.

As interfaces de instrumentação COM+ descritas na tabela a seguir permitem que os clientes recebam notificações para uma variedade de métricas de instrumentação COM+.
