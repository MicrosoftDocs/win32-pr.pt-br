# <a name="com-soap-service"></a>Serviço SOAP COM+

Simplesmente selecionando a caixa de seleção **usa SOAP** na ferramenta administrativa serviços de componentes, você pode expor um aplicativo com+ como um serviço Web XML. Você também pode usar um Web Service XML de modo transparente, seja implantado usando COM+ ou não, como um componente COM. Isso significa que você pode criar facilmente novos serviços Web XML fora dos aplicativos COM+ existentes e incorporar facilmente os serviços Web XML em novos aplicativos COM+.

Os tópicos descritos na tabela a seguir fornecem informações de plano de fundo e tarefas sobre como usar o COM+ para expor os componentes COM configurados como serviços Web XML.
