# <a name="com-partitions-tasks"></a>Tarefas de partições COM+

Os tópicos a seguir nesta seção fornecem instruções passo a passo para usar partições COM+.

**Windows XP:** A capacidade de criar, configurar ou delegar partições COM+ não está disponível. A partição global é a única partição COM+ disponível.

* * Windows 2000: * * o serviço de partições COM+ não está disponível no Windows 2000.



| Tópico                                                                                                         | Descrição                                                                                    |
|---------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [Criando e configurando partições COM+](creating-and-configuring-com--partitions.md)<br/>           | Descreve como criar e configurar uma partição COM+.<br/>                             |
| [Gerenciando partições locais](managing-local-partitions.md)<br/>                                         | Descreve como gerenciar partições COM+ que não estão dentro de Active Directory.<br/>       |
| [Gerenciando partições no Active Directory](managing-partitions-within-active-directory.md)<br/>     | Descreve como gerenciar partições COM+ que são especificadas dentro de Active Directory.<br/> |
| [Configurando direitos administrativos para uma partição](setting-administrative-rights-for-a-partition.md)<br/> | Descreve como definir privilégios de segurança para uma partição COM+.<br/>                      |
| [Agrupando aplicativos em partições](grouping-applications-into-partitions.md)<br/>                 | Descreve como configurar aplicativos COM+ para pertencer a uma partição COM+. <br/>        |
| [Coletando métricas de partição](collecting-partition-metrics.md)<br/>                                   | Descreve como coletar dados sobre partições COM+.<br/>                                |
| [Configurando o cache de partição](configuring-the-partition-cache.md)<br/>                             | Descreve como configurar o cache de partição COM+.<br/>                                |



 

> [!Note]  
> O serviço de partições COM+ não está habilitado por padrão. Para usar o serviço de partições COM+, você deve habilitá-lo por meio da ferramenta de administração de serviços de componentes ou alterando a propriedade PartitionsEnabled na coleção [**LocalComputer**](localcomputer.md) para true.
