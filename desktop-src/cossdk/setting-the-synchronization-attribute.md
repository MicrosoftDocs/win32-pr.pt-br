# <a name="setting-the-synchronization-attribute"></a>Configurando o atributo de sincronização

Defina o atributo de sincronização usando a ferramenta administrativa serviços de componentes ou a biblioteca de administração COM+. (Consulte [automatizando a administração do com+](automating-com--administration.md).)

**Para definir o atributo usando a ferramenta administrativa serviços de componentes**

1.  Depois de instalar o componente, abra a caixa de diálogo **Propriedades** do componente.

2.  Clique na guia **simultaneidade** .

3.  Em **suporte à sincronização**, clique no botão de opção adjacente ao valor que você deseja selecionar para o componente.

4.  Clique em **OK**.
