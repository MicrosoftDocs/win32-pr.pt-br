# <a name="com-administration-classes"></a>Classes de administração COM+

A seguir estão as classes de administração do COM+.



| Classe                                                          | Descrição                                                                                                                                            |
|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**COMAdminCatalog**](comadmincatalog.md)                     | Acessa os dados armazenados no catálogo COM+.                                                                                                  |
| [**COMAdminCatalogCollection**](comadmincatalogcollection.md) | Representa qualquer coleção no catálogo COM+. Use-o para enumerar, adicionar, remover e recuperar itens em uma coleção e para acessar coleções relacionadas. |
| [**COMAdminCatalogObject**](comadmincatalogobject.md)         | Representa itens em coleções no catálogo COM+. Use-o para recuperar e modificar as propriedades expostas por um item em uma coleção.                      |
