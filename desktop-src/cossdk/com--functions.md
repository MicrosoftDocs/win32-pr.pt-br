# <a name="com-functions"></a>Funções COM+

A seguir estão as funções COM+.



| Função                                                                 | Descrição                                                                                                                             |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**CoCreateActivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-cocreateactivity)                             | Cria uma atividade para realizar trabalho em lote síncrono ou assíncrono e que possa usar serviços COM+ sem a necessidade de criar um componente COM+. |
| [**CoEnterServiceDomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coenterservicedomain)                     | Usado para inserir o código que pode usar os serviços COM+.                                                                                     |
| [**CoGetDefaultContext**](/windows/desktop/api/combaseapi/nf-combaseapi-cogetdefaultcontext)                       | Recupera uma referência para o contexto padrão do apartamento especificado.                                                                |
| [**CoLeaveServiceDomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coleaveservicedomain)                     | Usado para deixar o código que usa serviços COM+.                                                                                             |
| **ComPlusCompleteCbbSetup**               | Conclui uma migração de catálogo no computador de destino.                                                                              |
| **GetComPlusPackageInstallStatus** | Indica se o CLR (Common Language Runtime) de 64 bits está instalado.                                                                |
| [**GetDispenserManager**](/windows/desktop/api/MtxDM/nf-mtxdm-getdispensermanager)                       | Recupera a interface [**IDispenserManager**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager) do Gerenciador do dispensador.                                             |
| [**GetManagedExtensions**](/windows/desktop/api/ComSvcs/nf-comsvcs-getmanagedextensions)                     | Determina se a versão instalada do COM+ dá suporte a recursos especiais fornecidos para gerenciar componentes atendidos (objetos gerenciados).    |
| [**GetObjectContext**](/windows/desktop/api/ComSvcs/nf-comsvcs-getobjectcontext)                             | Recupera uma referência ao contexto associado ao objeto COM+ atual.                                                   |
| [**MTSCreateActivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-mtscreateactivity)                           | Cria uma atividade em um apartamento de thread único para fazer o trabalho em lote síncrono ou assíncrono.                                        |
| [**RecycleSurrogate**](/windows/desktop/api/ComSvcs/nf-comsvcs-recyclesurrogate)                             | Recicla o processo de chamada.                                                                                                           |
| [**SafeRef**](/windows/desktop/api/ComSvcs/nf-comsvcs-saferef)                                               | Substituí Não use.                                                                                                                   |
