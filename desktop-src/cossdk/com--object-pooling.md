# <a name="com-object-pooling"></a>Pool de objetos COM+

O serviço de pooling de objetos do COM+ permite que você configure um componente para que as instâncias dele sejam mantidas ativas em um pool, prontas para serem usadas por qualquer cliente que solicite o componente.

Os tópicos descritos na tabela a seguir fornecem informações de plano de fundo e tarefa sobre como usar o pool de objetos no COM+.

|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Conceitos de pool de objetos COM+](com--object-pooling-concepts.md)<br/> | Fornece informações sobre como usar o serviço de pooling de objetos COM+ para melhorar o desempenho do sistema.<br/> |
| [Tarefas de pooling de objetos COM+](com--object-pooling-tasks.md)<br/>       | Fornece instruções detalhadas para implementar e monitorar o pool de objetos COM+.<br/>             |
