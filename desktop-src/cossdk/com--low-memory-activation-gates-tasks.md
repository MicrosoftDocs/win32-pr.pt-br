# <a name="com-low-memory-activation-gates-tasks"></a>Tarefas de retransmissores de ativação Low-Memory do COM+

O recurso de Gates de ativação de baixa memória do COM+ é fornecido automaticamente pelo COM+. Esse recurso não é configurável, portanto, não há tarefas associadas a ele.
