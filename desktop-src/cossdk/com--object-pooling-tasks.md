# <a name="com-object-pooling-tasks"></a>Tarefas de pooling de objetos COM+

Você pode configurar um componente a ser agrupado usando a ferramenta administrativa serviços de componentes ou as funções administrativas do COM+. Antes de configurá-lo, você deve ter certeza de que o componente está gravado corretamente para dar suporte ao pooling.

Os tópicos a seguir nesta seção fornecem instruções passo a passo para usar o pool de objetos COM+.
