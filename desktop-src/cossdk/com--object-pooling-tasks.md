# <a name="com-object-pooling-tasks"></a>Tarefas de pool de objetos COM+

Você pode configurar um componente para ser em pool usando a ferramenta administrativa serviços de componentes ou as funções administrativas COM+. Antes de configurá-lo, você deve ter certeza de que o componente foi gravado corretamente para dar suporte ao pooling.

Os tópicos a seguir nesta seção fornecem instruções passo a passo para usar o pool de objetos COM+.

| [Monitorando estatísticas de objeto](monitoring-object-statistics.md)<br/>                 | Descreve como monitorar objetos para que você possa ajustar as características do pool de objetos para obter o desempenho que deseja. <br/> |
