# <a name="com-just-in-time-activation-tasks"></a>Tarefas de ativação Just-In-Time do COM+

Os tópicos a seguir fornecem instruções passo a passo para usar a ativação JIT (Just-In-Time) no COM+.



| Tópico                                                                                                       | Descrição                                                                                                                               |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [Habilitando a ativação JIT para um componente](enabling-jit-activation-for-a-component.md)<br/>           | Fornece um procedimento passo a passo para ativar o serviço de ativação JIT COM+ para seus componentes COM+.<br/>                     |
| [Definindo o bit feito](setting-the-done-bit.md)<br/>                                                 | Fornece informações sobre como definir o bit feito para seus componentes ativados por JIT.<br/>                                             |
| [Habilitando o auto-done para um método](enabling-auto-done-for-a-method.md)<br/>                           | Fornece um procedimento passo a passo para habilenciar o recurso de auto-done para seus métodos de componente COM+.<br/>                          |
| [Impondo a ativação no contexto do chamador](enforcing-activation-in-the-caller-s-context.md)<br/> | Fornece um procedimento passo a passo para impor a ativação de um objeto no contexto do chamador, em vez de em seu próprio contexto.<br/> |
| [Impondo a ativação no contexto padrão](enforcing-activation-in-the-default-context.md)<br/>   | Fornece um procedimento passo a passo para impor a ativação de um objeto no contexto do chamador, em vez de em seu próprio contexto.<br/> |
