# <a name="optimizing-interactions-with-the-com-business-logic-tier"></a>Otimizando interações com a camada de lógica de negócios do COM+

A arquitetura para um aplicativo distribuído geralmente divide o sistema em três (ou mais) camadas. As interações entre essas camadas podem afetar drasticamente o sucesso do seu aplicativo. Esta seção discute como otimizar as interações entre a camada de lógica de negócios e as camadas de apresentação e dados. Para obter mais informações, confira os tópicos a seguir:

-   [Otimização de interações entre a camada de lógica de negócios COM+ e a camada de apresentação](optimizing-interactions-between-the-com--business-logic-tier-and-the-presentation-tier.md)
-   [Otimização de interações entre a camada de lógica de negócios COM+ e a camada de dados](optimizing-interactions-between-the-com--business-logic-tier-and-the-data-tier.md)
