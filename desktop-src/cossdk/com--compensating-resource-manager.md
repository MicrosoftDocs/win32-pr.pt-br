# <a name="com-compensating-resource-manager"></a>Gerenciador de recursos de compensação COM+

O Gerenciador de recursos de compensação COM+ (CRM) fornece uma maneira rápida e fácil de integrar recursos de aplicativos com transações do Microsoft Coordenador de Transações Distribuídas (DTC). Usar o CRM é uma alternativa ao desenvolvimento de um Gerenciador de recursos completo do MTS (Microsoft Transaction Services).

Os tópicos a seguir fornecem informações em segundo plano e tarefas sobre como usar o CRM.

| [Conceitos do Gerenciador de recursos de compensação COM+](com--compensating-resource-manager-concepts.md) | Fornece informações sobre a arquitetura e a funcionalidade do CRM. |
| [Tarefas do Gerenciador de recursos de compensação COM+](com--compensating-resource-manager-tasks.md)       | Fornece instruções para criar e configurar o CRM.               |
