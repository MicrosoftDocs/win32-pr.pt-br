# <a name="com-application-pooling"></a>Pool de aplicativos COM+

O pool de aplicativos COM+ permite o dimensionamento de processos single-thread e também pode ajudá-lo a se recuperar de falhas em processos únicos, fornecendo outros processos em execução capazes de lidar com ativações.
