# <a name="packaging-deployment-and-query-apis"></a>APIs de empacotamento, implantação e consulta

Use essas APIs em seus aplicativos de área de trabalho para criar, consumir, implantar ou consultar pacotes de aplicativo para aplicativos do Windows:

-   [API de empacotamento](interfaces.md)
-   [API de implantação do pacote](package-deployment-api.md)
-   [API de consulta de pacote](functions.md)

Use essas APIs em seus aplicativos de área de trabalho para implantar ou consultar pacotes de aplicativos:

-   [**Windows. ApplicationModel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. ApplicationModel. PackageID**](/uwp/api/Windows.ApplicationModel.PackageId)
-   [**Windows.Management.Deployment.PackageManager**](/uwp/api/Windows.Management.Deployment.PackageManager)
-   [**Windows.Management.Deployment.PackageUserInformation**](/uwp/api/Windows.Management.Deployment.PackageUserInformation)

Use essas APIs em seus aplicativos UWP para consultar pacotes de aplicativos:

-   [**Windows. ApplicationModel. Package**](/uwp/api/Windows.ApplicationModel.Package)
-   [**Windows. ApplicationModel. PackageID**](/uwp/api/Windows.ApplicationModel.PackageId)
