# <a name="packaging-and-deployment-tools-and-powershell-cmdlets"></a>Ferramentas de empacotamento e implantação e cmdlets do PowerShell

Use essas ferramentas e cmdlets para criar, implantar e inventariar pacotes de aplicativo para aplicativos do Windows.

## <a name="app-packaging-tools-and-guidance"></a>Diretrizes e ferramentas de empacotamento de aplicativos

Saiba como empacotar um aplicativo, criar um certificado de teste e assinar seu aplicativo usando o prompt de comando e o PowerShell. Para saber mais, veja:

-   [Criar um pacote de aplicativo com a ferramenta MakeAppx.exe](/windows/msix/package/create-app-package-with-makeappx-tool)
-   [Criar um certificado para a assinatura de pacote](/windows/msix/package/create-certificate-package-signing)
-   [Assinar um pacote do aplicativo usando a SignTool](/windows/msix/package/sign-app-package-using-signtool)

Como alternativa, você pode usar o Visual Studio para criar e assinar um pacote de aplicativo. Para obter mais informações, consulte [empacotar um aplicativo UWP com o Visual Studio](/windows/msix/package/packaging-uwp-apps).

## <a name="powershell-cmdlets"></a>Cmdlets do PowerShell

Use o Windows PowerShell para instalar, inventariar e desinstalar aplicativos do Windows de linha de negócios (LOB) que não são assinados por armazenamento. Para obter mais informações, consulte: [cmdlets do módulo Appx](/powershell/module/appx/index?view=win10-ps).
