# <a name="flush-method-of-the-control-class"></a>Método Flush da classe Control

Limpa os buffers de encaminhamento do fluxo e grava os dados armazenados em buffer no arquivo de configuração.

## <a name="syntax"></a>Sintaxe


```mof
void Flush();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
