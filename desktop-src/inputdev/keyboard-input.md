### <a name="notifications"></a>Notificações



| Nome                                      | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WM \_ ACTIVATE**](wm-activate.md)       | Enviado para a janela que está sendo ativada e a janela que está sendo desativada. Se as janelas usarem a mesma fila de entrada, a mensagem será enviada de forma síncrona, primeiro para o procedimento de janela da janela de nível superior que está sendo desativada e, em seguida, para o procedimento de janela da janela de nível superior que está sendo ativada. Se as janelas usarem filas de entrada diferentes, a mensagem será enviada de forma assíncrona, portanto, a janela será ativada imediatamente. <br/>                                                                                                                               |
| [**WM \_ APPCOMMAND**](wm-appcommand.md)   | Notifica uma janela de que o usuário gerou um evento de comando de aplicativo, por exemplo, clicando em um botão de comando do aplicativo usando o mouse ou digitando uma tecla de comando do aplicativo no teclado.<br/>                                                                                                                                                                                                                                                                                                                                                       |
| [**WM \_ CHAR**](wm-char.md)               | Postado na janela com o foco do teclado quando uma mensagem [**WM \_ KEYDOWN**](wm-keydown.md) é convertida pela [**função TranslateMessage.**](/windows/desktop/api/winuser/nf-winuser-translatemessage) A [**mensagem WM \_ CHAR**](wm-char.md) contém o código de caractere da chave que foi pressionada. <br/>                                                                                                                                                                                                                                                                             |
| [**WM \_ DEADCHAR**](wm-deadchar.md)       | Postado na janela com o foco do teclado quando uma mensagem [**WM \_ KEYUP**](wm-keyup.md) é convertida pela [**função TranslateMessage.**](/windows/desktop/api/winuser/nf-winuser-translatemessage) [**WM \_ DEADCHAR**](wm-deadchar.md) especifica um código de caractere gerado por uma chave morta. Uma chave morta é uma chave que gera um caractere, como o umlaut (ponto duplo), que é combinado com outro caractere para formar um caractere composto. Por exemplo, o caractere umlaut-O ( ) é gerado digitando a chave morta para o caractere umlaut e digitando a tecla O. <br/> |
| [**WM \_ HOTKEY**](wm-hotkey.md)           | Postado quando o usuário pressiona uma tecla de acesso registrada pela [**função RegisterHotKey.**](/windows/win32/api/winuser/nf-winuser-registerhotkey) A mensagem é colocada na parte superior da fila de mensagens associada ao thread que registrou a tecla de hot. <br/>                                                                                                                                                                                                                                                                                                                                 |
| [**WM \_ KEYDOWN**](wm-keydown.md)         | Postado na janela com o foco do teclado quando uma tecla não sistema é pressionada. Uma chave não sistema é uma tecla pressionada quando a tecla ALT não é pressionada. <br/>                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**WM \_ KEYUP**](wm-keyup.md)             | Postado na janela com o foco do teclado quando uma tecla não sistema é liberada. Uma tecla não sistema é uma tecla pressionada quando a tecla ALT não é pressionada ou uma tecla de teclado pressionada quando uma janela tem o foco do teclado. <br/>                                                                                                                                                                                                                                                                                                                          |
| [**WM \_ KILLFOCUS**](wm-killfocus.md)     | Enviado para uma janela imediatamente antes de perder o foco do teclado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**WM \_ SETFOCUS**](wm-setfocus.md)       | Enviado para uma janela depois de ter obtido o foco do teclado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**WM \_ SYSDEADCHAR**](wm-sysdeadchar.md) | Enviado para a janela com o foco do teclado quando uma mensagem [**WM \_ SYSKEYDOWN**](wm-syskeydown.md) é convertida pela [**função TranslateMessage.**](/windows/desktop/api/winuser/nf-winuser-translatemessage) [**WM \_ SYSDEADCHAR**](wm-sysdeadchar.md) especifica o código de caractere de uma chave morta do sistema, ou seja, uma tecla morta pressionada enquanto mantém pressionada a tecla ALT. <br/>                                                                                                                                                                                                        |
| [**WM \_ SYSKEYDOWN**](wm-syskeydown.md)   | Postado na janela com o foco do teclado quando o usuário pressiona a tecla F10 (que ativa a barra de menus) ou mantém a tecla ALT pressionada e pressiona outra tecla. Isso também ocorre quando nenhuma janela tem o foco do teclado no momento; nesse caso, a mensagem [**WM \_ SYSKEYDOWN**](wm-syskeydown.md) é enviada para a janela ativa. A janela que recebe a mensagem pode distinguir entre esses dois contextos verificando o código de contexto no *parâmetro lParam.* <br/>                                                                             |
| [**WM \_ SYSKEYUP**](wm-syskeyup.md)       | Postado na janela com o foco do teclado quando o usuário libera uma tecla que foi pressionada enquanto a tecla ALT era pressionada. Isso também ocorre quando nenhuma janela tem o foco do teclado no momento; nesse caso, a mensagem [**WM \_ SYSKEYUP**](wm-syskeyup.md) é enviada para a janela ativa. A janela que recebe a mensagem pode distinguir entre esses dois contextos verificando o código de contexto no *parâmetro lParam.* <br/>                                                                                                                           |
| [**WM \_ UNICHAR**](wm-unichar.md)         | Postado na janela com o foco do teclado quando uma mensagem [**WM \_ KEYDOWN**](wm-keydown.md) é convertida pela [**função TranslateMessage.**](/windows/desktop/api/winuser/nf-winuser-translatemessage) A [**mensagem \_ WM UNICHAR**](wm-unichar.md) contém o código de caractere da chave que foi pressionada.<br/>                                                                                                                                                                                                                                                                        |
