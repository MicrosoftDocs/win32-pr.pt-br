# <a name="xinput-and-controller-subtypes"></a>Subtipos de controlador e XINPUT

Uma tabela de subtipos de controlador disponíveis no XInput.



| Subtype                               | Valor | Significado                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------|-------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| XINPUT \_ DEVSUBTYPE \_ UNKNOWN           | 0x00  | Desconhecido.<br/> O tipo de controlador é desconhecido.<br/>                                                                                                                                                                                                                                                                                                                                                                |
| XINPUT \_ DEVSUBTYPE \_ GAMEPAD           | 0x01  | Controlador do gamepad.<br/> Inclui os botões Left e Right, Left e Right Triggers, Directional Pad e todos os botões padrão (A, B, X, Y, START, BACK, LB, RB, LSB, RSB).<br/>                                                                                                                                                                                                                                     |
| RODA XINPUT \_ \_ DEVSUBTYPE             | 0x02  | Controlador de roda de corrida. <br/> O Stick X esquerdo relata a rotação da roda, Gatilho direito é o acelerador de aceleração e Gatilho Esquerdo é o acelerador de trava. Inclui Painel Direcional e a maioria dos botões padrão (A, B, X, Y, START, BACK, LB, RB). LSB e RSB são opcionais.<br/>                                                                                                                                        |
| XINPUT \_ DEVSUBTYPE \_ STICK DE STICK \_     | 0x03  | Controlador de bastião do bastão. <br/> Inclui um Stick Digital que relata como um DPAD (para cima, para baixo, para a esquerda, para a direita) e a maioria dos botões padrão (A, B, X, Y, START, BACK). Os gatilhos esquerdo e direito são implementados como botões digitais e relatam 0 ou 0xFF. LB, LSB, RB e RSB são opcionais.<br/>                                                                                                                  |
| XINPUT \_ DEVSUBTYPE \_ FLIGHT \_ STICK     | 0x04  | Controlador de stick de voo. <br/> Inclui um bastão de apresentação e rolagem que relata como o Bastão Esquerdo, um GUID Hat que se reporta como o Bastão direito, um lança-rolagem (alça ou rocha) que relata como Gatilho Esquerdo e um controle de aceleração como o Gatilho Direito. Inclui suporte para uma primária (A), uma falsa secundária (B) e outros botões padrão (X, Y, START, BACK). LB, LSB, RB e RSB são opcionais.<br/>  |
| XINPUT \_ DEVSUBTYPE \_ PAD DE \_ XINPUT        | 0x05  | Controlador de pad de música. <br/> Inclui o Painel Direcional e os botões padrão (A, B, X, Y) no painel, além de BACK e START.<br/>                                                                                                                                                                                                                                                                                  |
| XINPUT \_ DEVSUBTYPETYPE \_            | 0x06  | Controlador de controlador de controlador. <br/> A barra de strum é mapeada para o DPAD (para cima e para baixo) e as barras são atribuídas a A (verde), B (vermelho), Y (amarelo), X (azul) e LB (laranja). O Stick Y direito está associado a um sensor de orientação vertical; Right Stick X é a barra whammy. Inclui suporte para BACK, START, DPAD (esquerda, direita). Gatilho esquerdo (seletor de retirada), Gatilho Direito, RB, LSB (modificador de traste), RSB são opcionais.<br/> |
| XINPUT \_ DEVSUBTYPE \_ GUITAR \_ ALTERNATE | 0x07  | Controlador de controlador de controlador alternativo. <br/> Dá suporte a um intervalo maior de movimento para o sensor de orientação vertical.<br/>                                                                                                                                                                                                                                                                                                  |
| KIT DE \_ KIT XINPUT DEVSUBTYPE \_ \_         | 0x08  | Controlador de controlador de controlador.<br/> Os pads de carrinhos são atribuídos aos botões: A para verde (Tom de Chão), B para vermelho (Snare Yellow), X para azul (Tom Baixo), Y para amarelo (Tom Alto) e LB para oucártico (Paulo Paulo). Inclui Painel Direcional, BACK e START. RB, LSB e RSB são opcionais.<br/>                                                                                                                                     |
| XINPUT \_ DEVSUBTYPE– \_ \_ AVÚS.      | 0x0B  | Controlador de controlador de instrumento. <br/> Idêntico a Meio, com o subtipo distinto para simplificar a instalação.<br/>                                                                                                                                                                                                                                                                                                              |
| XINPUT \_ DEVSUBTYPE \_ PAD DE FLIP \_       | 0x13  | Controlador de painel de teclado. <br/> Inclui Painel Direcional e a maioria dos botões padrão (A, B, X, Y, START, BACK, LB, RB). Os gatilhos esquerdo e direito são implementados como botões digitais e relatam 0 ou 0xFF. O stick esquerdo, o stick direito, o LSB e o RSB são opcionais.<br/>                                                                                                                                           |
> [!Note]  
> A versão herdada do XINPUT no Windows Vista (XInput 9.1.0) sempre retorna um subtipo fixo **de GAMEPAD XINPUT \_ DEVSUBTYPE, \_** independentemente do dispositivo anexado.
