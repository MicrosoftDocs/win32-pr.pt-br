# <a name="xinput-and-controller-subtypes"></a>XINPUT e subtipos de controlador

Uma tabela de subtipos de controlador disponíveis em XInput.



| Subtype                                                                                           | Significado                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|---------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| XINPUT \_ DEVSUBTYPE \_ desconhecido                                                                       | Desconhecido.<br/> O tipo de controlador é desconhecido.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| XINPUT \_ DEVSUBTYPE \_ gamepad                                                                       | Controlador de Gamepad.<br/> Inclui os pentes esquerdo e direito, gatilhos esquerdo e direito, direcional e todos os botões padrão (A, B, X, Y, iniciar, voltar, LB, RB, LSB, RSB).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                |
| XINPUT \_ DEVSUBTYPE \_ Wheel                                                                         | Controlador de roda de corrida. <br/> O direcional analógico X relata a rotação da roda, o gatilho direito é o pedal de aceleração e o gatilho esquerdo é o pedal do freio. Inclui o painel direcional e a maioria dos botões padrão (A, B, X, Y, iniciar, voltar, LB, RB). LSB e RSB são opcionais.<br/>                                                                                                                                                                                                                                                                                                                                   |
| XINPUT \_ DEVSUBTYPE \_ \_                                                                 | Controlador de pente de vôo. <br/> Inclui um pente digital que é reportado como um DPAD (para cima, para baixo, à esquerda, à direita) e a maioria dos botões padrão (A, B, X, Y, iniciar, voltar). Os gatilhos esquerdo e direito são implementados como botões digitais e relatam 0 ou 0xFF. LB, LSB, RB e RSB são opcionais.<br/>                                                                                                                                                                                                                                                                                                             |
| XINPUT \_ DEVSUBTYPE \_ Flight \_ Stick                                                                 | Controlador de vôo. <br/> Inclui uma inclinação e um rolo que se relatam como o direcional, um POV Hat que relata como o pente direito, um leme (alças de rotação ou rock) que relata como gatilho esquerdo e um controle de restrição como o gatilho direito. Inclui suporte para uma arma primária (A), armas secundárias (B) e outros botões padrão (X, Y, iniciar, voltar). LB, LSB, RB e RSB são opcionais.<br/>                                                                                                                                                                                             |
| XINPUT \_ DEVSUBTYPE \_ Dance \_ pad                                                                    | Controlador do Dance Pad. <br/> Inclui o painel direcional e os botões padrão (A, B, X, Y) no painel, além de voltar e iniciar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| XINPUT \_ DEVSUBTYPE \_ guitarra XInput \_ DEVSUBTYPE \_ guitarra \_ Alternate XInput \_ DEVSUBTYPE \_ guitarra \_ Bass | Controlador de guitarra. <br/> A Barra Strum mapeia para DPAD (para cima e para baixo) e os Frets são atribuídos a um (verde), B (vermelho), Y (amarelo), X (azul) e LB (laranja). O direcional direito Y está associado a um sensor de orientação vertical; O botão direito X é a barra desvantagens. Inclui suporte para voltar, iniciar, DPAD (esquerda, direita). Gatilho esquerdo (seletor de retirada), gatilho direito, RB, LSB (modificador se preocupe), RSB são opcionais.<br/> A guitarra Bass é idêntica à guitarra, com o subtipo DISTINCT para simplificar a instalação.<br/> A guitarra Alt dá suporte a uma grande variedade de movimentos para o sensor de orientação vertical.<br/> |
| \_Kit de \_ tambor XInput DEVSUBTYPE \_                                                                     | Controlador do tambor.<br/> Os pads dos tambores são atribuídos aos botões: A para verde (piso), B para vermelho (tambor snare), X para azul (baixo Tom), Y para amarelo (alto tom) e LB para o pedal (tambor de graves). Inclui painel direcional, voltar e iniciar. RB, LSB e RSB são opcionais.<br/>                                                                                                                                                                                                                                                                                                                                |
| XINPUT \_ DEVSUBTYPE \_ \_                                                                   | Controlador de pad de lo. <br/> Inclui o painel direcional e a maioria dos botões padrão (A, B, X, Y, iniciar, voltar, LB, RB). Os gatilhos esquerdo e direito são implementados como botões digitais e relatam 0 ou 0xFF. O pente à esquerda, o direcional direito, o LSB e o RSB são opcionais.<br/>                                                                                                                                                                                                                                                                                                                                      |
> [!Note]  
> A versão herdada do XINPUT no Windows Vista (XInput 9.1.0) sempre retorna um subtipo fixo de **XInput \_ DEVSUBTYPE \_ gamepad**, independentemente do dispositivo anexado.
