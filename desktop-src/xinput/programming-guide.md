# <a name="programming-guide-xinput-game-controller-apis"></a>Guia de programação (APIs do XInput Game Controller)

Este guia contém informações sobre como usar a API XInput para interagir com o controlador Xbox quando ele está conectado a um computador Windows.

- [Introdução com XInput](getting-started-with-xinput.md) -esta seção contém uma descrição de XInput e descreve como começar a usar o XInput em seu aplicativo.
- [Versões do XInput](xinput-versions.md) -esta seção contém informações sobre as versões do XInput.
- [XInput e DirectInput](xinput-and-directinput.md) -esta seção contém informações sobre as diferenças entre XInput e DirectInput e como você pode usar ambos lado a lado.
- [Dispositivos DirectInput e XUSB](directinput-and-xusb-devices.md) -contém informações sobre os mapeamentos de dispositivo do XUSB (classe de controlador comum) do Xbox.
- [XInput e subtipos de controlador](xinput-and-controller-subtypes.md) – lista os subtipos de controlador disponíveis em XInput.

Para obter mais informações sobre métodos de API específicos, consulte a [referência de programação](programming-reference.md).
