# <a name="functions-programming-reference"></a>Funções (referência de programação)

Funções XInput disponíveis.

## <a name="in-this-section"></a>Nesta seção

| Tópico                                                                                 | Descrição                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XInputEnable**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | Define o estado de relatório de XInput.<br/>                                                                                                          |
| [**XInputGetAudioDeviceIds**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | Recupera a renderização de som e as IDs de dispositivo de áudio de captura de som associadas ao headset conectado ao controlador especificado.<br/> |
| [**XInputGetBatteryInformation**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | Recupera o tipo de bateria e o status de cobrança de um controlador sem fio.<br/>                                                                       |
| [**XInputGetCapabilities**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | Recupera os recursos e recursos de um controlador conectado.<br/>                                                                           |
| [**XInputGetDSoundAudioDeviceGuids**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | Obtém os GUIDs de dispositivo de renderização e captura de som que estão associados ao headset conectado ao controlador especificado.<br/>          |
| [**XInputGetKeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | Recupera um evento de entrada de Gamepad.<br/>                                                                                                             |
| [**XInputGetState**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | Recupera o estado atual do controlador especificado.<br/>                                                                                     |
| [**XInputSetState**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | Envia dados para um controlador conectado. Essa função é usada para ativar a função de vibração de um controlador.<br/>                              |
