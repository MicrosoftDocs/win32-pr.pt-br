# <a name="xinput-game-controller-apis"></a>APIs do XInput Game Controller

## <a name="purpose"></a>Finalidade

A API do controlador de jogo XInput permite que os aplicativos recebam entrada do controlador Xbox para Windows.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs do XInput Game Controller foram projetadas para uso pelos desenvolvedores que desejam usar o controlador Xbox para seus aplicativos do Windows.
