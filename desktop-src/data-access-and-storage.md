# <a name="data-access-and-storage"></a>Acesso a dados e armazenamento

O Windows tem APIs, componentes e serviços que dão suporte a seus aplicativos de desktop em acesso a dados e armazenamento. Elas fornecem:

-   Gerenciamento do sistema de arquivos e arquivos.
-   Acesso ao banco de dados.
-   Suporte para transferência, sincronização e replicação de dados.
-   Acesso ao XML, pacote e arquivos de log.
-   Mestre de imagem.
-   Suporte a backup.
