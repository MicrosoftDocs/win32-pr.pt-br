# <a name="data-access-and-storage"></a>Acesso a dados e armazenamento

O Windows tem APIs, componentes e serviços que suportam seus aplicativos da área de trabalho no acesso a dados e armazenamento. Elas fornecem:

-   Gerenciamento de sistema de arquivos e arquivos.
-   Acesso ao banco de dados.
-   Suporte para transferência, sincronização e replicação de dados.
-   Acesso a arquivos XML, pacote e log.
-   Controle de imagem.
-   Suporte a backup.
