# <a name="toolbar-buttons-microsoft-agent-character-editor"></a>Botões da barra de ferramentas (editor de caracteres do Microsoft Agent)

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

:::image type="icon" source="images/f9charnew.gif":::

<dl> <dt>

<span id="New_Custom_Character"></span><span id="new_custom_character"></span><span id="NEW_CUSTOM_CHARACTER"></span>**Novo caractere personalizado**
</dt> <dd>

Redefine o editor para criar uma nova definição de caractere personalizada. Se um caractere existente for carregado e tiver edições não salvas em um ou mais arquivos, o editor exibirá uma mensagem para determinar se deseja salvar ou descartar as alterações não salvas.

</dd> </dl>

:::image type="icon" source="images/f10charopen.gif":::

<dl> <dt>

<span id="Open_Character_Definition"></span><span id="open_character_definition"></span><span id="OPEN_CHARACTER_DEFINITION"></span>**Abrir definição de caractere**
</dt> <dd>

Exibe a caixa de diálogo abrir arquivo, permitindo que você abra um arquivo de definição de caractere existente para edição. Se um caractere existente for carregado e tiver edições não salvas em um ou mais arquivos, o editor exibirá uma mensagem para determinar se deseja salvar ou descartar as alterações não salvas.

</dd> </dl>

:::image type="icon" source="images/f11charsave.gif":::

<dl> <dt>

<span id="Save_Character_Definition"></span><span id="save_character_definition"></span><span id="SAVE_CHARACTER_DEFINITION"></span>**Salvar definição de caractere**
</dt> <dd>

Salva a definição de caractere. Se a definição de caractere não existir (não foi nomeada), o editor exibirá a caixa de diálogo Salvar como para a entrada do nome de arquivo.

</dd> </dl>

:::image type="icon" source="images/f12charbuild.gif":::

<dl> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Caractere de Build**
</dt> <dd>

Cria um caractere do Microsoft Agent a partir da definição de caractere.

</dd> </dl>

:::image type="icon" source="images/f13charprint.gif":::

<dl> <dt>

<span id="Print_Character_Definition"></span><span id="print_character_definition"></span><span id="PRINT_CHARACTER_DEFINITION"></span>**Imprimir definição de caractere**
</dt> <dd>

Imprime o arquivo de definição de caractere atual aberto no editor.

</dd> </dl>

:::image type="icon" source="images/f14charcut.gif":::

<dl> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Migração**

</dd> </dl>

:::image type="icon" source="images/f15charcopy.gif":::

<dl> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**CopiarObjeto**

</dd> </dl>

:::image type="icon" source="images/f16charpaste.gif":::

<dl> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Olar**

</dd> </dl>

:::image type="icon" source="images/f17chardel.gif":::

<dl> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Apagar**

</dd> </dl>
