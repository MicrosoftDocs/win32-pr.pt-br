# <a name="the-listening-tip"></a>A dica de escuta

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

A Dica de Escuta é outro serviço de entrada de fala fornecido pelo Microsoft Agent. Quando a entrada de fala é instalada, o Agent inclui uma janela de dica de ferramenta especial que aparece quando o usuário pressiona a tecla de atalho Escuta ou chama o método Listen. A Dica de Escuta aparece somente quando os serviços de fala estão disponíveis. Se nenhum cliente tiver sido autor de um comando de voz ou carregar com êxito um mecanismo de fala, a Dica de Escuta não será exibida. Além disso, tanto a entrada de fala quanto a opção Dicas exibição nas Opções avançadas de caractere devem ser habilitadas para que a dica seja exibida.

A Dica de Escuta faz o tempo final automaticamente depois de ser apresentada. Se o tempo de tempo de texto "Ouvido" for concluído enquanto o usuário ainda estiver mantendo a tecla de atalho, a dica será reverteda para o texto de "escuta", a menos que o servidor receba outro enunciado correspondente. Nesse caso, a dica exibe o novo texto "Escutado" e inicia o tempo-tempo para esse texto de gorjeta. Se o usuário liberar a tecla de acesso e o servidor estiver exibindo o texto "Ouvido", o tempo-final continuará e a janela Dica de Escuta será ocultada quando o intervalo de tempo-tempo for desajustado.

Se o servidor ainda não tiver tentado carregar um mecanismo de reconhecimento de fala, a Dica de Escuta não será exibida. Da mesma forma, se o usuário tiver desabilitado a exibição da dica de escuta ou da entrada de fala desabilitada em Opções avançadas de caractere, a Dica de Escuta não será exibida.

A Dica de Escuta não aparece quando o ponteiro está sobre o ícone da barra de tarefas do caractere. Em vez disso, a janela de dica de notificação padrão é exibida e exibe o nome do caractere.

Os aplicativos cliente não podem gravar diretamente na Dica de Escuta, mas você pode especificar texto alternativo que o servidor exibe no reconhecimento de um comando de voz correspondente. Para fazer isso, de definir [**a propriedade Confidence**](confidence-property.md) e a nova propriedade [**ConfidenceText**](confidencetext-property.md)para o comando. Se a entrada falada corresponder ao comando, mas a melhor combinação não exceder a configuração de confiança, o servidor usará o conjunto de texto na propriedade **ConfidenceText** na janela de gorjeta. Se o cliente não fornecer esse valor, o servidor exibirá o texto (gramática) que ele corresponder.

O texto Dica de Escuta aparece no idioma com base na configuração de ID de idioma de caractere do cliente ativo de entrada, independentemente de haver um mecanismo de reconhecimento de fala compatível com o idioma disponível.
