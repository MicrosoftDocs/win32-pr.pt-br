# <a name="introduction-to-microsoft-agent"></a>Introdução ao Microsoft Agent

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Microsoft Agent é um conjunto de serviços de software programáveis que oferece suporte à apresentação de caracteres animados interativos. Os desenvolvedores podem usar caracteres como assistentes interativos para introduzir, guiar, fazer entretenimento ou, de outra forma, aprimorar suas páginas da Web ou aplicativos, além do uso convencional do Windows, de menus e de controles.

O Microsoft Agent permite que desenvolvedores de software e autores da Web incorporem uma nova forma de interação do usuário, conhecida como interfaces de conversação, que aproveita aspectos naturais da comunicação social humana. Além da entrada do mouse e do teclado, o Microsoft Agent inclui suporte opcional para reconhecimento de fala para que os aplicativos possam responder a comandos de voz. Os caracteres podem responder usando fala sintetizada, áudio gravado ou texto em um balão de palavras de desenho.

A abordagem de interface de conversação facilitada pelos serviços do Microsoft Agent não substitui o design da GUI (interface gráfica do usuário) convencional. Em vez disso, a interação de caractere pode ser facilmente combinada com os componentes de interface convencionais, como Windows, menus e controles para estender e aprimorar a interface do seu aplicativo.

As interfaces de programação do Microsoft Agent facilitam a animação de um caractere para responder à entrada do usuário. Os caracteres animados aparecem em sua própria janela, fornecendo flexibilidade máxima para o local em que podem ser exibidos na tela. o Microsoft Agent inclui um controle ActiveX que torna seus serviços acessíveis a linguagens de programação que dão suporte a ActiveX, incluindo linguagens de script da Web, como o Visual Basic scripting Edition (VBScript). Isso significa que a interação de caractere pode ser programada até mesmo de páginas HTML usando a <OBJECT> marca.
