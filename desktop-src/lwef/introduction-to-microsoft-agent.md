# <a name="introduction-to-microsoft-agent"></a>Introdução ao Microsoft Agent

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Microsoft Agent é um conjunto de serviços de software programáveis que dá suporte à apresentação de caracteres animados interativos. Os desenvolvedores podem usar caracteres como assistentes interativos para introduzir, orientar, animar ou aprimorar suas páginas da Web ou aplicativos, além do uso convencional de janelas, menus e controles.

O Microsoft Agent permite que desenvolvedores de software e autores da Web incorporem uma nova forma de interação do usuário, conhecida como interfaces de conversa, que aproveita aspectos naturais da comunicação social humana. Além da entrada do mouse e do teclado, o Microsoft Agent inclui suporte opcional para reconhecimento de fala para que os aplicativos possam responder a comandos de voz. Os caracteres podem responder usando fala sintetizada, áudio gravado ou texto em um balão de palavra de desenho.

A abordagem de interface de conversa facilitada pelos serviços do Microsoft Agent não substitui o design de GUI (interface gráfica do usuário) convencional. Em vez disso, a interação de caracteres pode ser facilmente mesclada com os componentes de interface convencionais, como janelas, menus e controles, para estender e aprimorar a interface do aplicativo.

As interfaces de programação do Microsoft Agent facilitam a animação de um caractere para responder à entrada do usuário. Os caracteres animados aparecem em sua própria janela, fornecendo flexibilidade máxima para onde eles podem ser exibidos na tela. O Microsoft Agent inclui um controle ActiveX que torna seus serviços acessíveis a linguagens de programação que suportam ActiveX, incluindo linguagens de script Web, como Visual Basic VBScript (Scripting Edition). Isso significa que a interação de caracteres pode ser programada até mesmo de páginas HTML usando a &lt; marca &gt; OBJECT.
