# <a name="accessing-microsoft-agent-services-directly"></a>Acessando os serviços do Microsoft Agent diretamente

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Se você estiver usando C, C++ ou Java, poderá acessar o servidor do Microsoft Agent diretamente usando suas interfaces ActiveX (OLE). Para obter mais informações sobre essas interfaces, consulte [Programming The Microsoft Agent Server interface](programming-the-microsoft-agent-server-interface.md).
