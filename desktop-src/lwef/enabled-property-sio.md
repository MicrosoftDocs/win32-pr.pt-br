# <a name="enabled-property-speech-input-object"></a>Propriedade Enabled (objeto de entrada de fala)

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Retorna um valor booliano que indica se a entrada de fala está habilitada.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

*Agent * * *. SpeechInput. Enabled**



| Valor     | Descrição               |
|-----------|---------------------------|
| **Verdadeiro**  | A entrada de fala está habilitada.  |
| **Falso** | A entrada de fala está desabilitada. |



 

</dd> </dl>

## <a name="remarks"></a>Comentários

A propriedade [**Enabled**](enabled-property.md) reflete os caracteres de escuta da opção de entrada na página de entrada de fala da folha de propriedades do agente (opções avançadas de caractere). A configuração da propriedade afeta todos os caracteres do agente e é somente leitura; somente o usuário pode alterar essa propriedade.
