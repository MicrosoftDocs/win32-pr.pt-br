# <a name="vol-tag"></a>Vol Tag

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrição**
</dt> <dd>

Define o volume de fala de linha de base da saída de fala.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

**\\ Vol=**_number_*_\\_*



| Parte     | Descrição                                                         |
|----------|---------------------------------------------------------------------|
| *number* | Volume de fala de linha de base: 0 é silêncio e 65535 é o volume máximo. |



 

</dd> </dl>

### <a name="remarks"></a>Comentários

A configuração de volume afeta os canais esquerdo e direito. Não é possível definir o volume de cada canal separadamente. Essa marca só tem suporte para a saída gerada pelo TTS.
