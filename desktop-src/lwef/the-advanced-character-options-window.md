# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>Janela Opções avançadas de caractere (interface do usuário do Microsoft Agent)

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Microsoft Agent mantém determinadas configurações globais que permitem que um usuário controle a interação com todos os caracteres. A janela Opções de caractere avançadas exibe essas opções e suas configurações atuais e pode ser mostrada por qualquer aplicativo de hospedagem usando a interface de programação do Microsoft Agent. O Microsoft Agent lembra a última página exibida e exibe essa página quando a folha de propriedades é exibida.

-   [A página saída](the-output-page.md)
-   [A página de entrada de fala](the-speech-input-page.md)
-   [A página de direitos autorais](the-copyright-page.md)
