# <a name="audio-output-support"></a>Suporte de saída de áudio

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Microsoft Agent permite que você use arquivos de áudio para a saída falada de um caractere. Você pode gravar arquivos de áudio e usar o método [**Speak**](speak-method.md) para reproduzir esses dados. Os serviços de animação do Microsoft Agent dão suporte automaticamente ao Lip-sincronizando a boca por caractere usando as características de áudio do arquivo de áudio. O Microsoft Agent também dá suporte a um formato especial para arquivos de áudio, que inclui fonema adicional e informações de quebra de palavras para obter suporte mais avançado a Lip-Sync. Você pode gerar esse formato especial usando a ferramenta de edição de som de informações linguísticas da Microsoft.
