# <a name="width-property-propertysheet-object"></a>Propriedade Width (objeto folha)

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Retorna um valor inteiro especificando a largura atual, em pixels, da janela da folha de propriedades do Microsoft Agent.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintaxe** 
</dt> <dd>

</dd> </dl>

## <a name="remarks"></a>Comentários
