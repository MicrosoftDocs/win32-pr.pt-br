# <a name="animation-sound-effects"></a>Efeitos de som de animação

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Microsoft Agent também permite que você inclua efeitos sonoros como parte da animação de um caractere. Usando o editor de caracteres do Microsoft Agent, você pode especificar o nome de arquivo do som padrão do Windows (. WAV) para disparar em um determinado quadro. Observe que o Microsoft Agent não mistura efeitos sonoros e saída falada, portanto, a saída falada não começa até que um efeito de som seja concluído. Portanto, evite qualquer efeito de som longo ou em loop como parte da animação de um caractere.
