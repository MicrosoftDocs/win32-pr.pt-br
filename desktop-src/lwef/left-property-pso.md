# <a name="left-property-propertysheet-object"></a>Propriedade Left (objeto folha)

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Retorna um valor inteiro especificando a borda esquerda atual, em pixels, da janela da folha de propriedades do Microsoft Agent.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

*Agent * * *. Folha. Left**

</dd> </dl>

## <a name="remarks"></a>Comentários

O servidor exibe a janela com base no local definido pelo usuário.
