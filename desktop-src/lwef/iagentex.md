# <a name="iagentex"></a>IAgentEx

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

**IAgentEx** deriva da interface [**IAgent**](iagent.md) . Ele inclui todos os métodos **IAgent** , bem como fornece acesso a funções adicionais.

**Métodos em ordem vtable**



| Métodos IAgentEx                                                                   | Descrição                                              |
|------------------------------------------------------------------------------------|----------------------------------------------------------|
| [**ShowDefaultCharacterProperties**](iagentex--showdefaultcharacterproperties.md) | Exibe as propriedades de caractere padrão.               |
| [**Obter versão**](iagentex--getversion.md)                                         | Retorna o número de versão do Microsoft Agent (servidor). |
