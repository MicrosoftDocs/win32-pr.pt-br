# <a name="iagentnotifysinkex"></a>IAgentNotifySinkEx

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

**IAgentNotifySinkEx** é derivado da interface [**IAgentNotifySink**](https://www.bing.com/search?q=**IAgentNotifySink**) . Ele inclui todos os métodos **IAgentNotifySink** e fornece acesso a funções adicionais.

**Métodos em ordem vtable**



| Métodos IAgentNotifySinkEx                                                   | Descrição                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------|
| [**HelpComplete**](iagentnotifysinkex--helpcomplete.md)                     | Ocorre quando o usuário seleciona um menu ou o caractere no modo de ajuda. |
| [**Ouvible**](iagentnotifysinkex--listeningstate.md)                 | Ocorre quando o estado de escuta do caractere é alterado.               |
| [**DefaultCharacterChange**](iagentnotifysinkex--defaultcharacterchange.md) | Ocorre quando o usuário altera o caractere padrão.                |
| [**AgentPropertyChange**](iagentnotifysinkex--agentpropertychange.md)       | Ocorre quando o usuário altera uma configuração de Propriedade do agente.            |
| [**ActiveClientChange**](iagentnotifysinkex--activeclientchange.md)         | Ocorre quando o cliente ativo de um caractere é alterado.              |
