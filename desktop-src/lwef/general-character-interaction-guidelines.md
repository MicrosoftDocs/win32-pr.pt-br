# <a name="general-character-interaction-guidelines"></a>Diretrizes gerais de interação de caracteres

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Os princípios de um bom design de interface do usuário também se aplicam à criação de uma interface usando o Microsoft Agent. Você deve entender seus usuários e suas metas, bem como seguir um bom processo de design de interface do usuário. As diretrizes Windows interface para design de software fornece uma excelente visão geral dos princípios gerais de design e da metodologia.

-   [Ser não exclusivo](be-non-exclusive.md)
-   [Fornecer comentários apropriados](provide-appropriate-feedback.md)
-   [Usar variação natural](use-natural-variation.md)
