# <a name="extradata-property"></a>Propriedade ExtraData

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrição**
</dt> <dd>

Retorna uma cadeia de caracteres que especifica dados adicionais armazenados como parte do caractere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

*agente*. **Characters("**_CharacterID_*_"). ExtraData_*

</dd> </dl>

## <a name="remarks"></a>Comentários

O valor padrão para a **propriedade ExtraData** de um caractere é definido quando o caractere é compilado com o Editor de Caracteres do Microsoft Agent. Ele não pode ser alterado ou especificado em tempo de executar.

> [!Note]  
> A **configuração de** propriedade ExtraData é opcional e pode não ser fornecida para todos os caracteres.
