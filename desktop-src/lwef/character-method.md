# <a name="character-method"></a>Método Character

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrição**
</dt> <dd>

Retorna um [**objeto Character**](/windows/desktop/lwef/the-characters-object) em uma coleção [**De**](/windows/desktop/lwef/the-characters-object) caracteres.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

*agent***. Characters.Character "**_CharacterID_*_"_*

</dd> </dl>

## <a name="remarks"></a>Comentários

Você pode usar esse método para acessar os [**métodos**](/windows/desktop/lwef/the-characters-object) e as propriedades de um objeto Character.

> [!Note]  
> Esse método pode ser necessário para algumas linguagens de programação que não são suportadas por coleções. Ele não é necessário para VBScript ou Visual Basic. Para obter mais informações sobre como especificar [**métodos character,**](/windows/desktop/lwef/the-characters-object) consulte Métodos de objeto de caractere.
