# <a name="character-method"></a>Método de caractere

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Retorna um objeto [**Character**](/windows/desktop/lwef/the-characters-object) em uma coleção de [**caracteres**](/windows/desktop/lwef/the-characters-object) .

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

*agente do ***. Caracteres. caractere "*** characterid * * *"**

</dd> </dl>

## <a name="remarks"></a>Comentários

Você pode usar esse método para acessar os métodos e as propriedades de um objeto de [**caractere**](/windows/desktop/lwef/the-characters-object) .

> [!Note]  
> Esse método pode ser necessário para algumas linguagens de programação que não dão suporte a coleções. Ele não é necessário para o VBScript ou o Visual Basic. Para obter mais informações sobre como especificar métodos de [**caractere**](/windows/desktop/lwef/the-characters-object) , consulte métodos de objeto de caractere.
