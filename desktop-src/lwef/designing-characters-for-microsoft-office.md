# <a name="designing-characters-for-microsoft-office"></a>Criando caracteres para Microsoft Office

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Este documento fornece informações que podem ajudá-lo a projetar e desenvolver um caractere do Microsoft Agent para uso no Microsoft Office 2000 (os caracteres do Microsoft Agent não funcionam com Microsoft Office 97). Este documento pressupõe que você tenha um conhecimento prático do editor de caracteres do Microsoft Agent e saiba como criar caracteres do Microsoft Agent. Mais informações sobre o editor de caracteres do agente e o design e o desenvolvimento de caracteres são fornecidas no [usando o editor de caracteres do Microsoft Agent](using-the-microsoft-agent-character-editor.md). Mais informações sobre a criação de caracteres para o Microsoft Agent são fornecidas na [criação de caracteres para o Microsoft Agent](designing-characters-for-microsoft-agent.md).
