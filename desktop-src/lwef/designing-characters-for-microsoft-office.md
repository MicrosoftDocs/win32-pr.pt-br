# <a name="designing-characters-for-microsoft-office"></a>Projetando caracteres para Microsoft Office

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Este documento fornece informações que podem ajudá-lo a projetar e desenvolver um caractere do Microsoft Agent para uso no Microsoft Office 2000 (os caracteres do Microsoft Agent não funcionam com o Microsoft Office 97). Este documento pressu que você tenha um conhecimento prático do Editor de Caracteres do Microsoft Agent e saiba como criar caracteres do Microsoft Agent. Mais informações sobre o Editor de Caracteres do Agent e a criação e o desenvolvimento de caracteres são fornecidas em Usando o Editor de [Caracteres do Microsoft Agent](using-the-microsoft-agent-character-editor.md). Mais informações sobre como criar caracteres para o Microsoft Agent são fornecidas em [Criando caracteres para o Microsoft Agent.](designing-characters-for-microsoft-agent.md)
