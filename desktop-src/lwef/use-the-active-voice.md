# <a name="use-the-active-voice"></a>Usar o Active Voice

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Ao usar a saída de fala para fornecer informações de diretiva ou para extrair uma resposta do usuário, use a voz ativa e especifique claramente a ação esperada do usuário. O exemplo a seguir ilustra as diferenças:



|                                      |                               |
|--------------------------------------|-------------------------------|
| Vou repetir o número.           | Nenhuma ação do usuário                |
| O número será repetido.         | Voz passiva, sem ação do usuário |
| Ouça enquanto o número é repetido. | Voz passiva                 |
| Ouça a repetição.            | Melhor opção                   |



 

Além disso, construa sua saída para desdobrar as informações de chave no final da frase, conforme mostrado nos exemplos a seguir:



|             |                                      |                                            |
|-------------|--------------------------------------|--------------------------------------------|
| Em vez de: | "Tem três dígitos do próximo dígito?"           | Use: "é o próximo dígito três?"            |
| Em vez de: | "Clique em OK para começar."                 | Use: "para começar, clique em OK".                 |
| Em vez de: | "Diga ' done ' para concluir seu pedido." | Use: "para concluir seu pedido, digamos ' concluído. '" |
