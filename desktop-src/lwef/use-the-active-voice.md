# <a name="use-the-active-voice"></a>Usar a Voz Ativa

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Ao usar a saída de fala para fornecer informações de diretiva ou para emitir uma resposta do usuário, use a voz ativa e especifique claramente a ação esperada do usuário. O exemplo a seguir ilustra as diferenças:



| Diretiva                            |  Avaliação                   |
|--------------------------------------|-------------------------------|
| Deixe-me repetir seu número.           | Nenhuma ação do usuário                |
| O número será repetido.         | Voz passiva, nenhuma ação do usuário |
| Escutar enquanto o número é repetido. | Voz passiva                 |
| Escutar a repetição.            | Melhor opção                   |



 

Além disso, construa sua saída para revelar as informações principais no final da frase, conforme mostrado nos exemplos a seguir:



| Em vez disto...                                                 | Use isso                                           |
|--------------------------------------------------|--------------------------------------------|
| "Três são o próximo dígito?"           | "O próximo dígito é três?"            |
| "Clique em OK para começar."                 | "Para começar, clique em OK."                 |
| "Diga "Concluído" para concluir seu pedido." | "Para concluir seu pedido, diga "Concluído". |
