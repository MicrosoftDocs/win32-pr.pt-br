# <a name="using-the-linguistic-information-sound-editing-tool"></a>Usando a Ferramenta de Edição de Som de Informações Linguísticas

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

A Ferramenta de Edição de Som de Informações Linguísticas permite que você gere informações de phoneme e quebra de palavras para aprimorar Windows arquivos de som (.wav) para dar suporte à animação de caracteres de sincronização de caracteres de alta qualidade. As seções a seguir descrevem como usar a ferramenta.

-   [Instalando o Editor de Som](installing-the-sound-editor.md)
-   [Iniciando o Editor de Som](starting-the-sound-editor.md)
-   [Criando um novo arquivo de som](creating-a-new-sound-file.md)
-   [Carregando um arquivo de som existente](loading-an-existing-sound-file.md)
-   [Gerando informações linguísticas](generating-linguistic-information.md)
-   [Salvando um arquivo de som](saving-a-sound-file.md)
-   [Referência de comando](command-reference.md)
-   [Botões da Barra de ferramentas](toolbar-buttons.md)

Você pode usar arquivos de som linguisticamente aprimorados gerados com o editor de som para dar suporte à sincronização da saída de caracteres do Microsoft Agent. Para fazer isso, basta passar o arquivo como um parâmetro para o [**método Speak.**](speak-method.md) Para obter mais informações, consulte [Programando o controle do Microsoft Agent ou](programming-the-microsoft-agent-control.md) [Programando a interface do servidor do Microsoft Agent](programming-the-microsoft-agent-server-interface.md).
