# <a name="the-characters-object"></a>O objeto Characters

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Seu aplicativo cliente pode dar suporte a um ou mais caracteres. Além disso, você pode compartilhar um caractere entre vários aplicativos. O Microsoft Agent define o objeto [**Characters**](/windows/desktop/lwef/the-characters-object) como uma coleção de caracteres. Para acessar um caractere, carregue os dados do caractere na coleção de [**caracteres**](/windows/desktop/lwef/the-characters-object) e especifique que o item na coleção usa os métodos e propriedades com suporte para esse caractere.

-   [Métodos de objeto de caracteres](characters-object-methods.md)
-   [Métodos de objeto de caracteres](character-object-methods.md)
-   [Propriedades do objeto Characters](character-object-properties.md)
