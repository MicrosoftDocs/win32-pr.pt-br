# <a name="the-characters-object"></a>O objeto Characters

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Seu aplicativo cliente pode dar suporte a um ou mais caracteres. Além disso, você pode compartilhar um caractere entre vários aplicativos. O Microsoft Agent define o [**objeto Characters**](/windows/desktop/lwef/the-characters-object) como uma coleção de caracteres. Para acessar um caractere, carregue os dados do caractere na coleção [**Caracteres**](/windows/desktop/lwef/the-characters-object) e especifique que o item na coleção usa os métodos e as propriedades com suporte para esse caractere.

-   [Métodos de objeto Characters](characters-object-methods.md)
-   [Métodos de objeto Characters](character-object-methods.md)
-   [Propriedades do objeto Characters](character-object-properties.md)
