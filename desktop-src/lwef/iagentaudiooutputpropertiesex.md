# <a name="iagentaudiooutputpropertiesex"></a>IAgentAudioOutputPropertiesEx

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

IAgentAudioOutputPropertiesEx é derivado da interface IAgentAudioOutputPropertiesEx. Ele inclui todos os métodos IAgentAudioOutputPropertiesEx e fornece acesso a funções adicionais.

**Métodos em ordem vtable**



| Métodos IAgentAudioOutputPropertiesEx                         | Descrição                                     |
|---------------------------------------------------------------|-------------------------------------------------|
| [**GetStatus**](iagentaudiooutputpropertiesex--getstatus.md) | Retorna o status do canal de saída de áudio. |
