# <a name="hotkey-property"></a>Propriedade HotKey

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrição**
</dt> <dd>

Retorna uma cadeia de caracteres que especifica a configuração atual do usuário para a chave de escuta.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintaxe** 
</dt> <dd>

*agent***. SpeechInput.HotKey**

</dd> </dl>

## <a name="remarks"></a>Comentários

A **propriedade Hotkey** reflete a configuração atual da chave de escuta na página Entrada de Fala da folha de propriedades do Agent (Opções de Caractere Avançadas). A propriedade é somente leitura; somente o usuário pode alterar a configuração.
