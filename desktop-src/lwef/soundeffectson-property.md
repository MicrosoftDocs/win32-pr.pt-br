# <a name="soundeffectson-property"></a>Propriedade SoundEffectsOn

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Retorna ou define se os efeitos de som estão habilitados para seu caractere.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

*agente* do. **Caracteres ("***characterid***"). SoundEffectsOn** \[  =  *booliano*\]



| Parte      | Descrição                                                                                                                                                        |
|-----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *booleano* | Uma expressão booliana que especifica se os efeitos de som estão habilitados. **Verdadeiro** Os efeitos de som estão habilitados.<br/> **Falso** Os efeitos de som estão desabilitados.<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Comentários

Essa propriedade determina se os efeitos sonoros incluídos como parte das animações de um caractere serão reproduzidos quando uma animação for reproduzida. A configuração dessa propriedade se aplica a todos os clientes do caractere.
