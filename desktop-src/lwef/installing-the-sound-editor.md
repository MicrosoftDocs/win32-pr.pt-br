# <a name="installing-the-sound-editor"></a>Instalando o editor de som

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

A configuração de sistema recomendada para usar o editor de som é um PC com um Pentium 166, pelo menos 48 MB de RAM e uma placa de som compatível com o Windows. Se você quiser registrar entradas faladas com a ferramenta, também será necessário um microfone compatível.

Para instalar a ferramenta de edição de som linguístico da Microsoft, abra seu arquivo de instalação de extração automática. Isso instalará automaticamente os arquivos apropriados em seu sistema. Se você baixar o editor de som do site do Microsoft Agent, poderá optar por instalar o editor depois de baixá-lo ou salvá-lo em seu disco para ser aberto e instalado posteriormente. A ferramenta de instalação irá propor a instalação própria no subdiretório de ferramentas do Microsoft Agent. Recomendamos que você use esse local.

O comando da Microsoft e controlar o mecanismo de reconhecimento de fala (versão do. 4,0) também deve ser instalado antes que você possa usar o editor de som. Normalmente, isso é instalado com o editor de som, mas se ele foi subseqüentemente desinstalado, você pode reinstalá-lo no Microsoft Platform SDK ou no site do Microsoft Agent em <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> . O editor de som só pode gerar informações linguísticas com base no idioma suportado pelo mecanismo de fala. Para gerar informações para outros idiomas, é necessário instalar um mecanismo de reconhecimento de fala compatível para esse idioma. Entre em contato com seu fornecedor do mecanismo de fala para determinar se eles dão suporte à ferramenta de edição de som linguístico da Microsoft.
