# <a name="the-default-character-properties-window"></a>A janela Propriedades do caractere padrão

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Além de permitir que os aplicativos carreguem um caractere específico, os aplicativos podem carregar um caractere que é um recurso compartilhado para o usuário, conhecido como *caractere padrão*. O caractere padrão é acessível de qualquer aplicativo, mas o caractere só é selecionável pelo usuário. Para facilitar a seleção desse caractere, o Agent fornece uma janela que fornece acesso à seleção desse caractere, chamada de janela Propriedades de caractere padrão. Há suporte para o acesso a essa janela na API do agente.

![caixa de diálogo Propriedades do caractere gênio](images/f8dpwin.gif)

A janela Propriedades do caractere padrão não pode ser usada para fornecer seleção de caracteres diferente de para o caractere padrão.
