# <a name="dlpnotifypredragdrop-function"></a>Função DlpNotifyPreDragDrop

Fornece ao sistema informações sobre um documento antes de uma operação arrastar e soltar ser iniciada.

## <a name="syntax"></a>Sintaxe


```C++
void WINAPI DlpNotifyPreDragDrop(_In_ const PDLP_DOCUMENT_INFO DocumentInfo);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*DocumentInfo* \[ no\]
</dt> <dd>

Um ponteiro para uma estrutura de [PDLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md) que contém informações sobre o documento associado à operação de arrastar e soltar.

</dd> </dl>


## <a name="return-value"></a>Retornar valor

Retornar void.
