# <a name="visible-property-propertysheet-object"></a>Propriedade Visible (objeto folha)

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Retorna ou define se a janela da folha de propriedades do Microsoft Agent está visível (aberta).

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintaxe** 
</dt> <dd>

*Agent * * *.* *  \[  =  *Booliano* folha. Visible\]



| Parte      | Descrição                                                                                                                                                   |
|-----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *booleano* | Uma expressão booliana que especifica se a janela está visível. **Verdadeiro** A janela está visível.<br/> **Falso** A janela está oculta (fechada).<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Comentários

O servidor exibe a janela com base no local e no tamanho definidos pelo usuário.

> [!Note]  
> O usuário pode substituir essa propriedade.
