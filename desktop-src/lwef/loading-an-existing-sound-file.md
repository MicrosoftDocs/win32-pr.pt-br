# <a name="loading-an-existing-sound-file"></a>Carregando um arquivo de som existente

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Você também pode carregar um arquivo de som Windows existente (.wav) ou um arquivo de  som linguisticamente aprimorado (.lwv) escolhendo o comando Abrir no **menu** Arquivo ou na barra de ferramentas. Isso exibe a **caixa de diálogo** Abrir. Selecione um arquivo e clique **em Abrir** para carregar o arquivo no editor.
