# <a name="iagentballoonex"></a>IAgentBalloonEx

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

[**IAgentBalloonEx**](/windows/desktop/lwef/iagentballoon) é derivado da interface **IAgentBalloon** . Ele inclui todos os métodos **IAgentBalloon** e fornece acesso a funções adicionais.

**Métodos em ordem vtable**



| Métodos IAgentBalloonEx                                  | Description                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**GetStyle**](https://www.bing.com/search?q=**GetStyle**)                     | Retorna o estilo de saída do balão de palavras.                           |
| [**SetStyle**](https://www.bing.com/search?q=**SetStyle**)                     | Define o estilo de saída do balão de palavras.                              |
| [**SetNumLines**](https://www.bing.com/search?q=**SetNumLines**)               | Define a saída das linhas numéricas na palavra balão.                  |
| [**SetNumCharsPerLine**](https://www.bing.com/search?q=**SetNumCharsPerLine**) | Define o número de caracteres por saída de linha no balão de palavras. |
