# <a name="designing-characters-for-microsoft-agent"></a>Criando caracteres para o Microsoft Agent

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Este documento fornece informações que podem ajudá-lo a projetar e desenvolver um caractere para uso com o Microsoft Agent. Ele inclui informações conceituais e técnicas sobre o design de caracteres, imagens e animações; o tamanho, o uso de cor e os tipos de imagens que você precisa criar; animações sugeridas; animações de fala; e Estados de agente. Os princípios de animação efetivo também são descritos para seu uso na criação de caracteres animados visualmente convincentes.

As seções a seguir descrevem como criar caracteres para o Microsoft Agent:

-   [Characters](characters.md)
-   [Animações](animations.md)
-   [Estados do agente](agent-states.md)
-   [Princípios de animação](animation-principles.md)
