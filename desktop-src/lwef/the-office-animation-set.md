# <a name="the-office-animation-set"></a>o conjunto de animações Office

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

a tabela a seguir lista as animações definidas para os Microsoft Office caracteres 2000. se você pretende usar seu caractere no Microsoft Office, você deve dar suporte a todas as animações nesta tabela. além disso, você pode adicionar outras animações que viver, mas tenha em mente que Microsoft Office não as chamará. As animações com asteriscos ( \* ) devem ser de 100% em loop. Outras animações devem ser breves.

|-------------------------|----------------|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **Alerta**               | Nenhum           | Quando o caractere deseja alertar o usuário                                                           | O caractere procura o usuário.                                                                                                              |

| **EmptyTrash**          | Nenhum           | A lixeira foi esvaziada no Outlook                                                                          | As luzes de caractere Trash podem ser acionadas                                                                                                        |
| **Explicar**             | Nenhum           | Quando o caractere deseja explicar algo ao usuário                                            | Parece resumidamente, mas attentively no usuário e, em seguida, olha                                                                                     |
| **GestureDown**         | GesturingDown  | O caractere indica algo na tela                                                         | O caractere examina o usuário e, em seguida, aponta e examina a tela                                                                           |
