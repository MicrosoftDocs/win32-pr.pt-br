# <a name="using-the-editor-with-a-different-speech-engine"></a>Usando o editor com um mecanismo de fala diferente

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Enquanto o editor de som instala o mecanismo de reconhecimento de fala da Microsoft (4,0), ele pode ser usado com outro mecanismo de fala, se esse mecanismo der suporte às interfaces necessárias documentadas no documento requisitos do mecanismo de fala. Antes de tentar usar o editor com outro mecanismo, confirme com seu fornecedor se eles estão em conformidade com esses requisitos.

Para usar o editor de som com outro mecanismo de fala, escolha o comando mecanismo de fala no menu **Editar** . Isso exibe uma caixa de diálogo mostrando o mecanismo atual em uso. Para escolher outro mecanismo, exiba a lista de mecanismos e clique em **OK**. Se não houver outros mecanismos listados, você não terá nenhum outro mecanismo compatível incluído.
