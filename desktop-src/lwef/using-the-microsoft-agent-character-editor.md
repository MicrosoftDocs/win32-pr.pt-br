# <a name="using-the-microsoft-agent-character-editor"></a>Usando o editor de caracteres do Microsoft Agent

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O editor de caracteres do Microsoft Agent permite que você compile animações de caracteres para uso com o Microsoft Agent. Você pode definir animações importando imagens de bitmap do Windows, definindo sua duração e, opcionalmente, incluindo ramificação, efeitos sonoros e sobreposições de fala. Para obter informações sobre como criar animações de um caractere, consulte [criando caracteres para o Microsoft Agent](designing-characters-for-microsoft-agent.md).

Com o editor de caracteres do agente, você pode definir o nome e a descrição do caractere, bem como as opções de saída, incluindo conversão de texto em fala (TTS), saída de voz sintetizada, suporte de menu pop-up e design de balão de palavras do caractere.

Este documento descreve o editor de caracteres do Microsoft Agent para o Microsoft Agent do qual você pode baixar <https://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=c21be0ec-1712-427d-a43d-94c122ef8eb6> .
