# <a name="using-the-microsoft-agent-character-editor"></a>Usando o Editor de Caracteres do Microsoft Agent

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Editor de Caracteres do Microsoft Agent permite compilar animações de caracteres para uso com o Microsoft Agent. Você pode definir animações importando Windows de bitmap, definindo sua duração e, opcionalmente, incluindo ramificação, efeitos de som e sobreposições de fala. Para obter informações sobre como criar animações de um caractere, consulte [Projetando caracteres para o Microsoft Agent.](designing-characters-for-microsoft-agent.md)

Com o Editor de Caracteres do Agente, você pode definir o nome e a descrição do caractere, bem como opções de saída, incluindo TTS (texto em fala), saída de voz sintetizada, suporte a menu pop-up e design de balão de palavra do caractere.

Este documento descreve o Editor de Caracteres do Microsoft Agent para o Microsoft Agent que você pode baixar do <https://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=c21be0ec-1712-427d-a43d-94c122ef8eb6> .
