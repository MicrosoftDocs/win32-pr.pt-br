# <a name="the-character-taskbar-icon"></a>O ícone da barra de tarefas de caracteres

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Se um caractere tiver sido criado para incluir um ícone, o ícone será exibido na área de notificação da barra de tarefas quando o Microsoft Agent for executado. Esse ícone fornece acesso ao menu pop-up do caractere, que fornece acesso aos comandos globais do agente, como aqueles que ocultam e mostram o caractere.

![área de notificação com relógio e ícone](images/f1tbicon.gif)

Mover o ponteiro sobre o ícone da barra de tarefas exibe uma janela tip que reflete o nome do caractere (no idioma atual do sistema). Clicar uma vez no ícone da barra de tarefas do caractere exibe o caractere. A ação associada ao clique duplo no ícone depende do aplicativo atual que controla esse caractere.

Clicar com o botão direito do mouse no ícone exibe um menu pop-up. Quando o caractere estiver visível, o menu pop-up exibirá os mesmos comandos exibidos ao clicar com o botão direito do mouse no caractere. Se o caractere estiver oculto, somente a janela de comandos de voz aberta (ou fechar) e os comandos mostrar serão exibidos.
