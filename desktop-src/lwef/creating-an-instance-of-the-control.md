# <a name="creating-an-instance-of-the-control"></a>Criando uma instância do controle

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Acessar o controle do Microsoft Agent exige que você primeiro crie uma instância do controle. A maneira mais fácil de fazer isso é colocar uma instância do controle em um formulário. Talvez seja preciso adicionar o controle à caixa de ferramentas antes de adicioná-lo ao formulário. Dependendo da linguagem de programação, você também poderá criar o controle em runtime.
