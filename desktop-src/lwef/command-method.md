Você pode usar esse método para acessar as propriedades de um objeto de [**comando**](/windows/desktop/lwef/the-command-object) .

> Esse método pode ser necessário para algumas linguagens de programação. Ele não é necessário para o VBScript ou o Visual Basic. Para obter mais informações sobre como especificar métodos de **comando** , consulte Propriedades do objeto de comando.
