# <a name="using-speech-engines-with-microsoft-agent"></a>Usando mecanismos de fala com o Microsoft Agent

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

A Microsoft fornece um conjunto de mecanismos de entrada de fala (reconhecimento) e de saída de fala (TeX para fala ou TTS) que você pode usar com seus aplicativos ou páginas da Web habilitados para o Microsoft Agent. O uso desses mecanismos está sujeito ao contrato de licença do Microsoft Agent e ao adendo suplementar exibido quando você instala os mecanismos. Essa licença o autoriza a distribuir os mecanismos, mas somente ao usá-los por meio da API do Microsoft Agent e usando um caractere visível.
