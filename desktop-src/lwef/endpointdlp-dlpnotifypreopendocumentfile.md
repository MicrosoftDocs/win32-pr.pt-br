# <a name="dlpnotifypreopendocumentfile-function"></a>Função DlpNotifyPreOpenDocumentFile

Fornece ao sistema informações sobre um arquivo de documento antes de uma operação abrir ser iniciada.

## <a name="syntax"></a>Sintaxe


```C++
void WINAPI DlpNotifyPreOpenDocumentFile(_In_ const PDLP_DOCUMENT_INFO DocumentInfo);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*DocumentInfo* \[ no\]
</dt> <dd>

Um ponteiro para uma estrutura de [PDLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md) que contém informações sobre o documento a ser aberto.

</dd> </dl>


## <a name="return-value"></a>Retornar valor

Retornar void.
