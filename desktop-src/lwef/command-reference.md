# <a name="command-reference-linguistic-information-sound-editing-tool"></a>Referência de comando (ferramenta de edição de som de informações linguísticas)

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

### <a name="the-file-menu"></a>O menu arquivo

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Novo**
</dt> <dd>

Redefine o editor de som para criar um novo arquivo de som avançado. Se um arquivo de som existente for carregado e tiver edições não salvas, o editor de som exibirá uma mensagem para determinar se deseja salvar ou descartar as alterações não salvas.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Abrir**
</dt> <dd>

Exibe a caixa de diálogo abrir, permitindo que você abra um arquivo de som existente. Se um arquivo de som existente for carregado e tiver edições não salvas, o editor de som exibirá uma mensagem para determinar se deseja salvar ou descartar as alterações não salvas.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Guarde**
</dt> <dd>

Salva um arquivo de som. Se o arquivo de som não existir (não foi nomeado), o editor de som exibirá a caixa de diálogo Salvar como para a entrada do nome de arquivo.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Salvar como**
</dt> <dd>

Exibe a caixa de diálogo Salvar como, permitindo que você insira um novo nome para o arquivo de som.

</dd> <dt>

<span id="Save_Selection_As"></span><span id="save_selection_as"></span><span id="SAVE_SELECTION_AS"></span>**Salvar seleção como**
</dt> <dd>

Exibe a caixa de diálogo Salvar seleção como, permitindo que você insira um nome para a parte selecionada do arquivo de som.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Arquivos abertos mais recentemente**
</dt> <dd>

Controla os arquivos de definição de caracteres recentes que você abriu. A escolha de um arquivo abre automaticamente esse arquivo para edição. Se um caractere existente for carregado e tiver edições não salvas em um arquivo, o editor de som exibirá uma mensagem para determinar se deseja salvar ou descartar as alterações não salvas.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Sido**
</dt> <dd>

Encerra o editor de som. Se um arquivo existente for carregado e tiver edições não salvas, o editor de som exibirá uma mensagem para determinar se deseja salvar ou descartar as alterações não salvas.

</dd> </dl>

### <a name="the-edit-menu"></a>O menu Editar

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Operação**
</dt> <dd>

Remove uma alteração feita no editor de som.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Ações**
</dt> <dd>

Reverte uma ação de desfazer no editor de som.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Migração**
</dt> <dd>

Remove o texto selecionado e o coloca na área de transferência.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**CopiarObjeto**
</dt> <dd>

Copia o texto selecionado para a área de transferência.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Olar**
</dt> <dd>

Copia o texto na área de transferência para o ponto de inserção ou seleção na caixa de texto representação de texto.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Apagar**
</dt> <dd>

Remove o texto selecionado.

</dd> <dt>

<span id="Select_All"></span><span id="select_all"></span><span id="SELECT_ALL"></span>**Selecionar tudo**
</dt> <dd>

Seleciona o texto na caixa de texto representação de texto.

</dd> <dt>

<span id="Generate_Linguistic_Info"></span><span id="generate_linguistic_info"></span><span id="GENERATE_LINGUISTIC_INFO"></span>**Gerar informações linguísticas**
</dt> <dd>

Começa a gerar informações de quebra de palavra e fonema para um arquivo de som.

</dd> <dt>

<span id="Insert_Phoneme"></span><span id="insert_phoneme"></span><span id="INSERT_PHONEME"></span>**Inserir fonema**
</dt> <dd>

Exibe a caixa de diálogo Inserir fonema que permite inserir um rótulo de fonema selecionado.

</dd> <dt>

<span id="Replace_Phoneme"></span><span id="replace_phoneme"></span><span id="REPLACE_PHONEME"></span>**Substituir fonema**
</dt> <dd>

Exibe a caixa de diálogo Substituir fonema, que permite substituir o rótulo de fonema selecionado.

</dd> <dt>

<span id="Delete_Phoneme"></span><span id="delete_phoneme"></span><span id="DELETE_PHONEME"></span>**Excluir fonema**
</dt> <dd>

Exclui o rótulo de fonema selecionado.

</dd> <dt>

<span id="Insert_Word"></span><span id="insert_word"></span><span id="INSERT_WORD"></span>**Inserir palavra**
</dt> <dd>

Exibe a caixa de diálogo Inserir palavra que permite inserir um rótulo de palavra na representação de áudio.

</dd> <dt>

<span id="Replace_Word"></span><span id="replace_word"></span><span id="REPLACE_WORD"></span>**Substituir palavra**
</dt> <dd>

Exibe a caixa de diálogo Substituir palavra que permite substituir o rótulo da palavra selecionada na representação de áudio.

</dd> <dt>

<span id="Delete_Word"></span><span id="delete_word"></span><span id="DELETE_WORD"></span>**Excluir palavra**
</dt> <dd>

Exclui o rótulo de palavra selecionado na representação de áudio.

</dd> <dt>

<span id="Phoneme_Label_Display"></span><span id="phoneme_label_display"></span><span id="PHONEME_LABEL_DISPLAY"></span>**Exibição do rótulo de fonema**
</dt> <dd>

Altera a exibição do rótulo de fonema entre nomes descritivos e valores de byte IPA.

</dd> <dt>

<span id="Speech_Engine"></span><span id="speech_engine"></span><span id="SPEECH_ENGINE"></span>**Mecanismo de fala**
</dt> <dd>

Permite alterar o mecanismo de fala que você usa para gerar as informações de quebra de palavra e fonema.

</dd> </dl>

<dl> <dt>

</dd> <dt>

</dd> <dt>

<span id="Stop"></span><span id="stop"></span><span id="STOP"></span>**Deixar**

</dd> </dl>
