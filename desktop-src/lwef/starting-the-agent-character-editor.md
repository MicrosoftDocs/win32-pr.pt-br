# <a name="starting-the-agent-character-editor"></a>Iniciando o Editor de Caracteres do Agente

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Para executar o Editor de Caracteres do Agente, escolha a opção **Editor** de Caracteres do Agente no **menu** Iniciar da barra de tarefas do Windows ou clique duas vezes no ícone Editor de Caracteres do Microsoft **Agent** na área de trabalho. A janela do Editor será aberta, exibindo seus menus, uma barra de ferramentas com comandos usados com frequência, uma árvore listando os componentes que compoem a definição de um caractere e um conjunto de páginas com guias que mudam com base em sua seleção na árvore de componentes.

![Captura de tela que mostra a página 'Propriedades' do 'Caractere – Editor de Caracteres do Microsoft Agent'.](images/f1chared.gif)

Para acessar campos na janela do teclado, você pode usar a tecla TAB e SHIFT+TABpara navegar entre controles ou usar a tecla de acesso (alt+ letra sublinhada ) para passar para um controle específico. Depois que o Editor concluir a iniciação, você poderá começar a criar uma nova definição de caractere ou carregar uma definição de caractere existente.

A barra de status exibe informações sobre comandos ou botões de barra de ferramentas quando você move o ponteiro sobre eles. Ele também exibe informações resumidas sobre seus dados de animação de caracteres e informações de status ao criar um caractere.
