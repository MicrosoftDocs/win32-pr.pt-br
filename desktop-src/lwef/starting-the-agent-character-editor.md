# <a name="starting-the-agent-character-editor"></a>Iniciando o editor de caracteres do agente

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Para executar o editor de caracteres do agente, escolha a opção **Editor de caracteres do agente** no menu **Iniciar** da barra de tarefas do Windows ou clique duas vezes no ícone do editor de caracteres do Microsoft **Agent** na área de trabalho. A janela do editor será aberta, exibindo seus menus, uma barra de ferramentas com comandos usados com frequência, uma árvore listando os componentes que compõem a definição de um caractere e um conjunto de páginas com guias que são alteradas com base na sua seleção na árvore de componentes.

![Captura de tela que mostra a página "Propriedades" do "caractere-editor de caracteres do agente da Microsoft".](images/f1chared.gif)

Para acessar campos na janela do teclado, você pode usar a tecla TAB e SHIFT + TAB para navegar entre os controles ou usar a tecla de acesso (ALT +*letra sublinhada*) para mover para um controle específico. Depois que o editor concluir a inicialização, você poderá começar a criar uma nova definição de caractere ou carregar uma definição de caractere existente.

A barra de status exibe informações sobre comandos ou botões da barra de ferramentas quando você move o ponteiro sobre eles. Ele também exibe informações resumidas sobre seus dados de animação de caracteres e informações de status quando você cria um caractere.
