# <a name="the-word-balloon"></a>O balão do Word

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Além da saída de áudio falada, a interface do Microsoft Agent dá suporte à legenda textual na forma de saída de texto em balões de palavras no estilo de desenho. As palavras aparecem no balão à medida que são faladas. O balão oculta quando a saída falada é concluída. A janela Opções avançadas de caractere fornece opções para desabilitar a exibição do balão, bem como os atributos de controle relacionados à sua aparência.
