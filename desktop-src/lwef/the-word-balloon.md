# <a name="the-word-balloon"></a>O balão de palavras

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Além da saída de áudio falada, a interface do Microsoft Agent dá suporte à legenda textual na forma de saída de texto em balão de palavras no estilo desenho. As palavras aparecem no balão conforme elas são faladas. O balão oculta quando a saída falada é concluída. A janela Opções Avançadas de Caracteres fornece opções para desabilitar a exibição do balão, bem como atributos de controle relacionados à sua aparência.
