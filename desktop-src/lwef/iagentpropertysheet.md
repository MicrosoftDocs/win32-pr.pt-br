# <a name="iagentpropertysheet"></a>IAgentPropertySheet

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

**IAgentPropertySheet** define uma interface que permite que os aplicativos definam e consultem Propriedades para a folha de propriedades do Microsoft Agent (janela).

**Métodos em ordem vtable**



| Métodos IAgentPropertySheet                             | Descrição                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**GetVisible**](iagentpropertysheet--getvisible.md)   | Retorna se a folha de propriedades do Microsoft Agent está visível.                               |
| [**Não visível**](iagentpropertysheet--setvisible.md)   | Define a propriedade [**visível**](visible-property.md) da folha de propriedades do Microsoft Agent. |
| [**GetPosition**](iagentpropertysheet--getposition.md) | Retorna a posição da folha de propriedades do Microsoft Agent.                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | Retorna o tamanho da folha de propriedades do Microsoft Agent.                                      |
| [**GetPage**](iagentpropertysheet--getpage.md)         | Retorna a página atual da folha de propriedades do Microsoft Agent.                             |
| [**SetPage**](iagentpropertysheet--setpage.md)         | Define a página atual para a folha de propriedades do Microsoft Agent.                                |
