# <a name="programming-the-microsoft-agent-control"></a>Programando o controle do Microsoft Agent

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Embora os aplicativos possam gravar diretamente nos serviços do Microsoft Agent usando suas interfaces de servidor de automação, o Microsoft Agent também inclui um controle ActiveX (OLE). O controle dá suporte à programação fácil usando uma linguagem de script, como o Microsoft Visual Basic VBScript (Scripting Edition) ou outras linguagens que suportam a interface ActiveX controle.
