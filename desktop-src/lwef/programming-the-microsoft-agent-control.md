# <a name="programming-the-microsoft-agent-control"></a>Programando o Microsoft Agent Control

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Embora os aplicativos possam gravar diretamente nos serviços do Microsoft Agent usando suas interfaces de servidor de automação, o Microsoft Agent também inclui um controle ActiveX (OLE). O controle dá suporte à fácil programação usando uma linguagem de script, como o Microsoft Visual Basic Scripting Edition (VBScript) ou outras linguagens que dão suporte à interface do controle ActiveX.
