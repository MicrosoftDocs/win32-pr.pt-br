# <a name="requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool"></a>Requisitos para dar suporte à ferramenta de edição de som de informações linguísticas da Microsoft

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

A ferramenta de edição de som de informações linguísticas da Microsoft usa um mecanismo de reconhecimento de fala para produzir quebras de palavras e informações fonéticas para arquivos de som wave (. wav) padrão do Windows. A versão 2,0 agora dá suporte ao uso com outros mecanismos de fala. Os fornecedores que desejam dar suporte ao editor de som devem garantir que seus mecanismos tenham suporte total à especificação SAPI 4,0 para mecanismos de gramática sem contexto e os requisitos a seguir.

-   [ISRResGraphEx e IAttributes](isrresgraphex-and-iattributes.md)
