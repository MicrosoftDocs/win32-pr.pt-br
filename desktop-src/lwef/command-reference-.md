# <a name="command-reference-microsoft-agent-character-editor"></a>Referência de comando (Editor de Caracteres do Microsoft Agent)

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

## <a name="the-file-menu"></a>O menu Arquivo

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Novo**
</dt> <dd>

Redefine o Editor de Caracteres do Agente para criar uma nova definição de caractere. Se um caractere existente for carregado e tiver edições não salvas, o Editor exibirá uma mensagem para determinar se as alterações não salvas serão salvas ou descartadas.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Aberto**
</dt> <dd>

Exibe a caixa de diálogo Abrir Arquivo, permitindo que você abra um arquivo de definição de caractere existente para edição. Se um caractere existente for carregado e tiver edições não salvas em um arquivo, o Editor exibirá uma mensagem para determinar se as alterações não salvas serão salvas ou descartadas.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Salvar**
</dt> <dd>

Salva a definição de caractere. Se a definição de caractere não existir (não foi nomeada), o Editor exibirá a caixa de diálogo Salvar como para entrada do nome do arquivo.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Salvar como**
</dt> <dd>

Exibe a caixa de diálogo Salvar como, permitindo que você insira um novo nome para o arquivo de definição de caractere.

</dd> <dt>

<span id="Print"></span><span id="print"></span><span id="PRINT"></span>**Imprimir**
</dt> <dd>

Exibe a caixa de diálogo Imprimir, permitindo que você escolha uma opção de impressão e imprima o arquivo de definição de caractere.

</dd> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Caractere de build**
</dt> <dd>

Exibe a caixa de diálogo Caractere de Build, que inclui opções para definir como criar arquivos de animação e dados de um caractere para uso com o Microsoft Agent.

</dd> <dt>

<span id="Page_Setup"></span><span id="page_setup"></span><span id="PAGE_SETUP"></span>**Configuração de página**
</dt> <dd>

Exibe a caixa de diálogo Configuração de Página que permite definir as opções de impressão para o arquivo de definição de caractere.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Arquivos abertos mais recentemente**
</dt> <dd>

Mantém o controle dos arquivos de definição de caracteres recentes que você abriu. Escolher um arquivo abre automaticamente esse arquivo para edição. Se um caractere existente for carregado e tiver edições não salvas em um arquivo, o Editor exibirá uma mensagem para determinar se as alterações não salvas serão salvas ou descartadas.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Saída**
</dt> <dd>

Sai do Editor de Caracteres do Agente. Se um caractere existente for carregado e tiver edições não salvas em um arquivo, o Editor exibirá uma mensagem para determinar se as alterações não salvas serão salvas ou descartadas.

</dd> </dl>

## <a name="the-edit-menu"></a>O menu Editar

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Desfazer**
</dt> <dd>

Remove uma alteração feita no Editor.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Refazer**
</dt> <dd>

Inverte uma ação desfazer no Editor.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Cortar**
</dt> <dd>

Remove o item selecionado do Editor e o coloca na área Windows Área de Transferência.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Cópia**
</dt> <dd>

Copia o item selecionado no Editor para a área Windows Área de Transferência.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Colar**
</dt> <dd>

Copia dados da área de Windows atual para o local selecionado.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Excluir**
</dt> <dd>

Remove o item selecionado do Editor.

</dd> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Nova animação**
</dt> <dd>

Cria um novo objeto de animação no Editor.

</dd> <dt>

<span id="New_Frame"></span><span id="new_frame"></span><span id="NEW_FRAME"></span>**Novo quadro**
</dt> <dd>

Cria um novo quadro para uma animação.

</dd> <dt>

<span id="New_Frames_from_Files"></span><span id="new_frames_from_files"></span><span id="NEW_FRAMES_FROM_FILES"></span>**Novos quadros de arquivos**
</dt> <dd>

Exibe a caixa de diálogo Selecionar Arquivos de Imagem e cria quadros usando os arquivos selecionados.

</dd> <dt>

<span id="Open_Frame_Window"></span><span id="open_frame_window"></span><span id="OPEN_FRAME_WINDOW"></span>**Janela Abrir Quadro**
</dt> <dd>

Exibe o quadro atual e suas imagens em uma janela separada sem dimensionar as imagens carregadas no quadro.

</dd> <dt>

<span id="Preview___Stop_Preview"></span><span id="preview___stop_preview"></span><span id="PREVIEW___STOP_PREVIEW"></span>**Visualização \| parar visualização**
</dt> <dd>

Reproduz (ou para de tocar) uma animação, começando no quadro selecionado.

</dd> <dt>

<span id="Preview___Stop_Preview_Exit_Branching"></span><span id="preview___stop_preview_exit_branching"></span><span id="PREVIEW___STOP_PREVIEW_EXIT_BRANCHING"></span>**Visualizar \| parar a ramificação de saída da visualização**
</dt> <dd>

Reproduz (ou para de tocar) a ramificação de saída de uma animação, começando em seu quadro selecionado.

</dd> </dl>
