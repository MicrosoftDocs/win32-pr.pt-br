# <a name="the-propertysheet-object"></a>O objeto folha

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O objeto [**folha**](https://www.bing.com/search?q=**PropertySheet**) fornece várias propriedades que você pode usar se quiser manipular o caractere em relação à folha de propriedades do Microsoft Agent (também conhecida como a janela Opções de caractere avançado).

-   [**Altura**](height-property-pso.md)
-   [**Mantida**](left-property-pso.md)
-   [**Web**](page-property.md)
-   [**Início**](top-property-pso.md)
-   [**Visível**](visible-property.md)
-   [**Largura**](width-property-pso.md)

Se você consultar as propriedades [**altura**](height-property-pso.md), [**esquerda**](left-property-pso.md), [**superior**](top-property-pso.md)e [**largura**](width-property-pso.md) antes que a folha de propriedades tenha sido mostrada, seus valores retornarão como zero (0). Depois de mostrado, essas propriedades retornam a última posição e o tamanho da janela (em relação à resolução de tela atual).
