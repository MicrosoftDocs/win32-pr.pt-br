# <a name="visible-property-commandswindow-object"></a>Propriedade Visible (objeto CommandsWindow)

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descrição**
</dt> <dd>

Retorna ou define se a janela Comandos de Voz está visível (aberta).

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintaxe** 
</dt> <dd>

*agent***. CommandsWindow.Visible* *  \[  =  *booliana*\]



| Parte      | Descrição                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *booleano* | Uma expressão booliana que especifica se a Janela comandos de voz está visível.<br/> **True** A janela está visível.<br/> **False** A janela está oculta (fechada).<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Comentários

O servidor exibe a janela com base no local e no tamanho definidos pelo usuário.

> [!Note]  
> O usuário pode substituir essa propriedade.
