# <a name="microsoft-agent"></a>Microsoft Agent

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows. Para obter mais informações, [consulte Windows 7 e Windows Server 2008 R2 Application Quality Cookbook](../win7appqual/windows-7-application-quality-cookbook.md).\]

## <a name="purpose"></a>Finalidade

O Microsoft Agent versão 2.0 fornece tecnologia para criar interfaces de conversa inovadoras e novas para aplicativos e páginas da Web. Ele fornece capacidade de animação, interatividade e versatilidade poderosas, com incrível facilidade de desenvolvimento.

O Microsoft Agent é uma tecnologia que fornece uma base para maneiras mais naturais para as pessoas se comunicarem com seus computadores. É um conjunto de serviços de software que permite aos desenvolvedores incorporar caracteres animados interativos em seus aplicativos e páginas da Web. Esses caracteres podem falar, por meio de um mecanismo de texto em fala ou áudio gravado, e até mesmo aceitar comandos de voz falada. O Microsoft Agent capacita os desenvolvedores a estender a interface do usuário além das interações convencionais de mouse e teclado predominantes atualmente.

Aprimorar aplicativos e páginas da Web com uma personalidade interativa visível ampliará e humanizará a interação entre os usuários e seus computadores.

## <a name="where-applicable"></a>Quando aplicável

Há um número ilimitado de funções e funções que os desenvolvedores podem criar para essas personalidades executarem.

-   Um host de boas-vindas pode saudar novos usuários e fornecer um tour guiado na primeira vez que um computador é ligado, um aplicativo é executado ou um site é procurado.
-   Um tutor amigável pode conduzir alguém por meio de uma tarefa ou uma árvore de decisão com instruções passo a passo ao longo do caminho.
-   Um messenger poderia entregar uma notificação ou um alerta de que um novo email chegou e, em seguida, oferecer para lê-lo para você.
-   Um assistente pode executar tarefas para você, como procurar informações na Internet e, em seguida, lê-la em voz alta.

## <a name="developer-audience"></a>Público de desenvolvedores

O Microsoft Agent foi projetado principalmente para desenvolvedores que usam linguagens ou ambientes que suportam interfaces de controle COM ou ActiveX Microsoft. Elas incluem:

-   Microsoft Visual Studio (Visual C++, Visual Basic)

-   Microsoft Office (Visual Basic for Applications)

-   Microsoft Internet Explorer (Visual Basic Scripting Edition ou Microsoft JScript)

-   Host Windows Script do Microsoft ActiveX (ActiveX de Script)

-   Outros aplicativos e ambientes que suportam interfaces de ActiveX COM.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

**Necessário:**

-   Microsoft Windows 95, Windows 98, Windows Me, Windows NT 4.0 (x86) ou Windows 2000
-   Internet Explorer versão 3.02 ou posterior
-   Um PC Pentium de 100 MHz (ou mais rápido)
-   Pelo menos 16 MB (megabytes de RAM)
-   Pelo menos 1 MB de espaço livre em disco para os componentes principais
-   Um adicional de 2 a 4 MB para cada caractere que você instalar
-   Mais 32 KB para cada DLL (componente de linguagem)

**Recomendado:**

-   Um espaço em disco livre adicional de 1,6 MB se você planeja usar o Mecanismo de Texto em Fala lernout & Gotpie TruVoice para saída de fala
-   Mais 22 MB de espaço livre em disco se você planeja usar o Mecanismo de Reconhecimento de Fala da Microsoft para entrada de fala
-   Uma Windows de som compatível com o Windows
-   Um conjunto compatível de alto-falantes e microfone
