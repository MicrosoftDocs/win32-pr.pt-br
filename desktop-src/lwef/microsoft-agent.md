# <a name="microsoft-agent"></a>Agente da Microsoft

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows. Para obter mais informações, consulte [Windows 7 e Windows Server 2008 R2 Application Quality Cookbook](../win7appqual/windows-7-application-quality-cookbook.md).\]

## <a name="purpose"></a>Finalidade

A versão 2,0 do Microsoft Agent fornece tecnologia para criar interfaces de conversação inovadoras e novas para aplicativos e páginas da Web. Ele fornece recursos avançados de animação, interatividade e versatilidade, com uma incrível facilidade de desenvolvimento.

O Microsoft Agent é uma tecnologia que fornece uma base de maneiras mais naturais para as pessoas se comunicarem com seus computadores. Trata-se de um conjunto de serviços de software que permite aos desenvolvedores incorporar caracteres interativos e animados em seus aplicativos e páginas da Web. Esses caracteres podem falar, por meio de um mecanismo de conversão de texto em fala ou áudio gravado, e até mesmo aceitar comandos de voz falados. O Microsoft Agent capacita os desenvolvedores a estender a interface do usuário para além das interações convencionais de mouse e teclado predominantes atualmente.

Aprimorar aplicativos e páginas da Web com uma personalidade interativa visível ampliará e tornará a interação entre os usuários e seus computadores.

## <a name="where-applicable"></a>Quando aplicável

Há um número ilimitado de funções e funções que os desenvolvedores podem criar para a execução desses personalidades.

-   Um host de boas-vindas poderia receber novos usuários e fornecer um tour guiado na primeira vez em que um computador for ligado, um aplicativo é executado ou um site é procurado.
-   Um tutorial amigável pode levar alguém por uma tarefa ou uma árvore de decisão com instruções passo a passo ao longo do caminho.
-   Um Messenger pode entregar uma notificação ou alerta de que um novo email chegou e, em seguida, oferecer para lê-lo para você.
-   Um assistente pode executar tarefas para você como pesquisar informações na Internet e, em seguida, lê-las em voz alta.

## <a name="developer-audience"></a>Público de desenvolvedores

O Microsoft Agent foi projetado principalmente para desenvolvedores que usam linguagens ou ambientes que dão suporte a interfaces de controle de COM ou Microsoft ActiveX. Estão incluídos:

-   Microsoft Visual Studio (Visual C++, Visual Basic)

-   Microsoft Office (Visual Basic for Applications)

-   Microsoft Internet Explorer (Visual Basic Scripting Edition ou Microsoft JScript)

-   Microsoft Windows Script Host (linguagem de script do ActiveX)

-   Outros aplicativos e ambientes que dão suporte a interfaces de controle COM ou ActiveX.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

**Necessário:**

-   Microsoft Windows 95, Windows 98, Windows me, Windows NT 4,0 (x86) ou Windows 2000
-   Internet Explorer versão 3, 2 ou posterior
-   Um PC Pentium 100 MHz (ou mais rápido)
-   Pelo menos 16 megabytes (MB) de RAM
-   Pelo menos 1 MB de espaço livre em disco para os componentes principais
-   Um adicional de 2 a 4 MB para cada caractere que você instalar
-   Um adicional de 32 KB para cada componente de linguagem (DLL)

**Recomendado:**

-   Mais 1,6 MB de espaço livre em disco se você planeja usar o mecanismo de conversão de texto em Lernout & Hauspie TruVoice para saída de fala
-   Mais de 22 MB de espaço livre em disco se você planeja usar o mecanismo de reconhecimento de fala da Microsoft para entrada de fala
-   Uma placa de som compatível com o Windows
-   Um conjunto compatível de alto-falantes e microfone
