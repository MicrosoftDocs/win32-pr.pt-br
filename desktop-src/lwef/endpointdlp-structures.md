# <a name="endpoint-data-loss-prevention-structures"></a>Estruturas de prevenção contra perda de dados do ponto de extremidade

Estruturas usadas pelo recurso de prevenção de perda de dados do ponto de extremidade.



| Estruturas                                                       | Descrição                                                           |
|-------------------------------------------------------------------|-----------------------------------------------------------------------|
| [DLP_APP_OP_ENLIGHTENED_LEVEL](endpointdlp-dlp_app_op_enlightened_level.md)                       | Associa uma ação de DLP (prevenção de perda de dados) de ponto de extremidade com um nível de imposição.                                  |
| [DLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md)                       | Especifica informações sobre um documento associado a uma operação de DLP de ponto de extremidade.                                  |
| [DLP_POSTOP_STATUS](enpointdlp-dlp_postop_status.md)                         | Especifica informações de status sobre uma operação de DLP de ponto de extremidade.  |
| [DLP_PRINT_INFO](endpointdlp-dlp_print_info.md)                       | Especifica informações sobre um documento que está associado a uma operação de impressão de DLP de ponto de extremidade.                                  |
