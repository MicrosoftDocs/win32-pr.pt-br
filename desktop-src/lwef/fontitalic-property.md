# <a name="fontitalic-property"></a>Propriedade FontItalic

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Retorna o estilo de fonte exibido atualmente na janela de balão do texto do caractere especificado.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

*agente do ***. Caracteres ("*** characterid * * *"). Balloon. FontItalic**



| Valor     | Descrição                     |
|-----------|---------------------------------|
| **Verdadeiro**  | A fonte do balão é itálico.     |
| **Falso** | A fonte do balão não está em itálico. |



 

</dd> </dl>

## <a name="remarks"></a>Comentários

O valor padrão das configurações de fonte do balão de palavras de um caractere é definido no editor de caracteres do Microsoft Agent. Além disso, o usuário pode substituir as configurações de fonte para todos os caracteres na folha de propriedades do Microsoft Agent.
