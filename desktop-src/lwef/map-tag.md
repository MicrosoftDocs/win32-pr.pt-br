# <a name="map-tag"></a>Marca de mapa

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Ndescrição**
</dt> <dd>

Mapeia texto falado em texto exibido na palavra balão.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxe**
</dt> <dd>

\\**MAP**= "SpokenText" = "balloontext"\\



| Parte          | Descrição                                           |
|---------------|-------------------------------------------------------|
| *spokentext*  | Uma cadeia de caracteres que especifica o texto para a saída falada.       |
| *balloontext* | Uma cadeia de caracteres que especifica o texto para a saída do balão de palavras. |



 

</dd> </dl>

### <a name="remarks"></a>Comentários

Essa marca permite que você use texto falado diferente daquele exibido na palavra balão.
