# <a name="other-speech-engines-compatible-with-microsoft-agent"></a>Outros mecanismos de fala compatíveis com o Microsoft Agent

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Além dos mecanismos de fala da Microsoft fornecidos e que oferecem suporte ao Microsoft Agent, várias empresas também oferecem mecanismos de fala com suporte para o Microsoft Agent. Esta página fornece uma breve visão geral desses mecanismos que podem estar disponíveis em inglês dos EUA e em outros idiomas. Informações sobre como instalar e acessar, bem como licença e redistribuir os mecanismos podem ser encontradas em seus sites.

Ao usar qualquer mecanismo de fala de terceiros, você também precisa instalar binários de runtime do Microsoft SAPI 4.0a para que os mecanismos sejam enumerados corretamente. Na página da Web, você pode incluir a seguinte marca para disparar um carregamento automático do Speech API:

``` syntax
<OBJECT WIDTH=0 HEIGHT=0
  CLASSID="CLSID:0C7F3F20-8BAB-11d2-9432-00C04F8EF48F"
  CODEBASE="#VERSION=4,0,0,0">
</OBJECT>
```

Para obter mais informações sobre os binários de runtime do SAPI, consulte o site do [grupo de Fala da Microsoft](https://msdn.microsoft.com/library/ee705648.aspx). O Microsoft Agent também instala os binários de runtime do SAPI com o Mecanismo de Reconhecimento de Fala da Microsoft, o Painel de Controle de Fala e o Editor de Caracteres do Agente.

Observe que esses links apontam para servidores que não estão sob controle da Microsoft. Leia a instrução [oficial da](https://www.microsoft.com/isapi/gomscom.asp?TARGET=/Misc/NonMS.md) Microsoft sobre outros servidores. A Microsoft não endossa o conteúdo nem o software fornecido nesses sites. Todas as perguntas técnicas e de suporte também devem ser direcionadas aos fornecedores dos mecanismos e não à Microsoft ou à equipe do Microsoft Agent.

**Mecanismo de texto em fala de Digalo**

O Digalo é um mecanismo TTS acessível projetado para usuários finais e desenvolvedores. O mecanismo é oferecido em vários idiomas: francês, alemão, português (Brasil), espanhol, russo e inglês dos EUA e reino unido, com italiano, polonês e outros idiomas em breve. Informações do desenvolvedor, bem como detalhes sobre o registro do usuário final e programas afiliados, também estão disponíveis.

**Mecanismo de Fala Elan**

O Mecanismo de Fala elan usa a tecnologia text to speech da Elan e está disponível em sete idiomas: francês, alemão, português (Brasil), espanhol, russo, Reino Unido e inglês dos EUA.

**IBM ViaVoice Outloud**

A IBM desenvolveu vários caracteres do Microsoft Agent para uso com o ViaVoice Outloud. O IBM ViaVoice Outloud é uma tecnologia de texto em fala (síntese de fala) e está disponível em francês, alemão, italiano, espanhol, inglês do Reino Unido e inglês dos EUA. Quando combinado com o Microsoft Agent, você pode criar aplicativos vibrantes em muitas linguagens e estender suas oportunidades de vendas para novos mercados em todo o mundo.
