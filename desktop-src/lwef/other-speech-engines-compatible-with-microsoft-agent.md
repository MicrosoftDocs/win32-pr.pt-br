# <a name="other-speech-engines-compatible-with-microsoft-agent"></a>Outros mecanismos de fala compatíveis com o Microsoft Agent

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Além dos mecanismos de fala da Microsoft que são fornecidos com o e dão suporte ao Microsoft Agent, várias empresas também oferecem mecanismos de fala com suporte para o Microsoft Agent. Esta página fornece uma breve visão geral de tais mecanismos que podem estar disponíveis no inglês dos EUA e em outras linguagens. Informações sobre como instalar e acessar, bem como licenciar e redistribuir os mecanismos, podem ser encontradas em seus sites.

Ao usar qualquer mecanismo de fala de terceiros, você também precisa instalar os binários do Microsoft SAPI 4.0 em tempo de execução para que os mecanismos sejam enumerados corretamente. Na página da Web, você pode incluir a seguinte marcação para disparar um download automotivo do Speech API:

``` syntax
<OBJECT WIDTH=0 HEIGHT=0
  CLASSID="CLSID:0C7F3F20-8BAB-11d2-9432-00C04F8EF48F"
  CODEBASE="#VERSION=4,0,0,0">
</OBJECT>
```

Para obter mais informações sobre os binários de tempo de execução SAPI, consulte o [site do grupo de fala da Microsoft](https://msdn.microsoft.com/library/ee705648.aspx). O Microsoft Agent também instala os binários de tempo de execução SAPI com o mecanismo de reconhecimento de fala da Microsoft, o painel de controle de fala e o editor de caracteres do agente.

Observe que esses links apontam para servidores que não estão sob o controle da Microsoft. Leia a [declaração oficial](https://www.microsoft.com/isapi/gomscom.asp?TARGET=/Misc/NonMS.md) da Microsoft sobre outros servidores. A Microsoft não endossa o conteúdo nem o software fornecido nesses sites. Todas as perguntas técnicas e de suporte também devem ser direcionadas aos fornecedores dos mecanismos e não à Microsoft ou à equipe do Microsoft Agent.

**Mecanismo de conversão de texto em fala do Digalo**

Digalo é um mecanismo de TTS econômico projetado para usuários finais e desenvolvedores. O mecanismo é oferecido em várias linguagens: francês, alemão, Português (Brasil), espanhol, russo e Reino Unido e inglês americano, com italiano, polonês e outras linguagens em breve. As informações do desenvolvedor, bem como detalhes sobre o registro do usuário final e os programas afiliados, também estão disponíveis.

**Mecanismo de fala Elan**

O mecanismo de fala Elan usa a tecnologia de texto para fala da Elan e está disponível em sete idiomas: francês, alemão, Português (Brasil), espanhol, russo, Reino Unido e inglês americano.

**IBM ViaVoice muito alto**

A IBM desenvolveu vários caracteres do Microsoft Agent para uso com ViaVoice de baixo. O IBM ViaVoice com alta disponibilidade é uma tecnologia de texto para fala (síntese de fala) e está disponível em francês, alemão, italiano, espanhol, Reino Unido e inglês dos EUA. Quando combinado com o Microsoft Agent, você pode criar aplicativos vibrantes em várias linguagens e estender suas oportunidades de vendas para novos mercados mundiais.
