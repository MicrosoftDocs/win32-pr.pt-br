# <a name="programming-the-microsoft-agent-server-interface"></a>Programando a interface do servidor do Microsoft Agent

\[o Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Microsoft Agent fornece serviços que permitem que você programe caracteres animados de um aplicativo. Esses serviços são implementados como um servidor de automação OLE. A automação OLE permite que um aplicativo controle o objeto de outro aplicativo programaticamente. Este documento pressupõe uma compreensão do Component Object Model (COM) e OLE. Para obter uma introdução desses serviços, consulte [visão geral da interface de programação](microsoft-agent-programming-interface-overview.md).
