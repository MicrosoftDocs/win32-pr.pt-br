# <a name="speech-engine-support-requirements"></a>Requisitos de suporte do mecanismo de fala

\[O Microsoft Agent foi preterido a partir do Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

O Microsoft Agent usa a interface de programação de aplicativo (SAPI) do Microsoft Speech para dar suporte à entrada de fala (reconhecimento de fala ou SR) e à saída de fala (conversão de texto em fala ou TTS). Ao dar suporte a esse padrão, os serviços de fala do Microsoft Agent podem ter suporte de outros mecanismos de fala. Este documento descreve as interfaces SAPI necessárias usadas pelo Microsoft Agent. Para obter mais informações sobre o SAPI, consulte o site do grupo de [falas da Microsoft](https://msdn.microsoft.com/library/ee705648.aspx) .

-   [Requisitos para mecanismos de conversão de texto em fala](requirements-for-text-to-speech-engines.md)
-   [Requisitos para mecanismos de reconhecimento de fala](requirements-for-speech-recognition-engines.md)
-   [Requisitos para dar suporte à ferramenta de edição de som de informações linguísticas da Microsoft](requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool.md)
