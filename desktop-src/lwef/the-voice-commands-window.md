<span data-ttu-id="4dcf1-105">Se um mecanismo de fala compatível estiver instalado, o Microsoft Agent fornecerá uma janela especial chamada **janela de comandos de voz** que exibe os comandos que foram habilitados para voz para reconhecimento de fala.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-105">If a compatible speech engine is installed, Microsoft Agent supplies a special window called the **Voice Commands Window** that displays the commands that have been voice-enabled for speech recognition.</span></span> <span data-ttu-id="4dcf1-106">A **janela comandos de voz** serve como um prompt Visual para o que pode ser falado como entrada (comandos não podem ser selecionados com o mouse).</span><span class="sxs-lookup"><span data-stu-id="4dcf1-106">The **Voice Commands Window** serves as a visual prompt for what can be spoken as input (commands cannot be selected with the mouse).</span></span>

![caixa de diálogo comandos de voz](images/f2voice.gif)

<span data-ttu-id="4dcf1-108">A janela é exibida quando um usuário seleciona o comando **Abrir janela comandos de voz** , falando o comando ou clicando com o botão direito do mouse no caractere e escolhendo o comando no menu pop-up do caractere.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-108">The window appears when a user selects the **Open Voice Commands Window** command, either by speaking the command or right-clicking the character and choosing the command from the character's pop-up menu.</span></span> <span data-ttu-id="4dcf1-109">No entanto, se o usuário desabilitar a entrada de fala, a **janela de comandos de voz** não estará acessível.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-109">However, if the user disables speech input, the **Voice Commands Window** is not accessible.</span></span>

<span data-ttu-id="4dcf1-110">A **janela comandos de voz** exibe comandos habilitados para voz como uma árvore.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-110">The **Voice Commands Window** displays voice-enabled commands as a tree.</span></span> <span data-ttu-id="4dcf1-111">Se o aplicativo de hospedagem atual fornecer comandos de voz, eles aparecerão expandidos e na parte superior da janela.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-111">If the current hosting application supplies voice commands, they appear expanded and at the top of the window.</span></span> <span data-ttu-id="4dcf1-112">As entradas também aparecem para outros aplicativos que usam o caractere.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-112">Entries also appear for other applications using the character.</span></span> <span data-ttu-id="4dcf1-113">A janela também inclui os comandos de voz globais fornecidos pelo Microsoft Agent.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-113">The window also includes the global voice commands supplied by Microsoft Agent.</span></span> <span data-ttu-id="4dcf1-114">Se o aplicativo de hospedagem atual não tiver comandos de voz, os comandos de voz globais aparecerão expandidos e na parte superior da janela.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-114">If the current hosting application has no voice commands, the global voice commands appear expanded and at the top of the window.</span></span>

<span data-ttu-id="4dcf1-115">O usuário pode dimensionar e mover a **janela de comandos de voz**.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-115">The user can size and move the **Voice Commands Window**.</span></span> <span data-ttu-id="4dcf1-116">O Microsoft Agent lembra o último local da janela e o exibe nesse local se o usuário fechar e abrir novamente a janela.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-116">Microsoft Agent remembers the last location of the window and redisplays it at that location if the user closes and re-opens the window.</span></span> <span data-ttu-id="4dcf1-117">Se as entradas na janela excederem o tamanho de exibição atual da janela, as barras de rolagem serão exibidas.</span><span class="sxs-lookup"><span data-stu-id="4dcf1-117">If the entries in the window exceed the current display size of the window, scroll bars appear.</span></span>
