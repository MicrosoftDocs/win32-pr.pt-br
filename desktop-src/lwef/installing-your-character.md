# <a name="installing-your-character"></a>Instalando o caractere

\[O Microsoft Agent foi preterido a partir Windows 7 e pode não estar disponível nas versões subsequentes do Windows.\]

Para que seu caractere seja show na Galeria de Assistentes, instale-o na pasta Microsoft Office Actors dentro da pasta Dados do Aplicativo \\ \\ do usuário. A pasta Dados do Aplicativo pode ser encontrada no diretório Windows ou no diretório Perfis de Usuário dentro do diretório Windows aplicativo. Por exemplo, no Windows 2000, você pode instalar seu caractere em "C: Documentos e Configurações Dados do Aplicativo \\ \\ YourLoginName \\ \\ Microsoft Office \\ \\ Actors".
