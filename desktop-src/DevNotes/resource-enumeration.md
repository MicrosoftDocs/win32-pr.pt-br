# <a name="resource-enumeration"></a>Enumeração de recursos

Os desenvolvedores podem usar a enumeração de recursos como parte da instrumentação que fornece tipos específicos de informações sobre seus aplicativos.

A Microsoft desenvolve a instrumentação no sistema operacional Windows, mas esses recursos estão vinculados de forma tão próxima ao sistema operacional que eles podem ser de uso limitado para os desenvolvedores de aplicativos. A Microsoft está disponibilizando uma API (interface de programação de aplicativo) de enumeração de recursos que pode ser usada com os seguintes recursos de instrumentação: alocações de heap e informações de rastreamento de identificador.

Os tópicos a seguir descrevem como usar essa API em seus aplicativos:

-   [Sobre a enumeração de recursos](about-resource-enumeration.md)
-   [Referência de enumeração de recursos](resource-enumeration-reference.md)
