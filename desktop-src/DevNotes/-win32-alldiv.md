# <a name="_alldiv-routine"></a>\_Rotina alldiv

Divide dois inteiros **LONGLONG** . Por exemplo, para dividir dois valores Int64, o compilador pode gerar uma chamada para a \_ rotina alldiv.

## <a name="remarks"></a>Comentários

\_a rotina alldiv é uma rotina auxiliar para o compilador C. Se o compilador chama a \_ rotina alldiv é completamente dependente do conjunto de otimização.
