# <a name="cabinet-api-structures"></a>Estruturas de API de gabinete

Esta seção detalha as seguintes estruturas de API de Gabinete:



| Estrutura                                              | Descrição                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Contém as informações de versão para Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Contém informações de gabinete.<br/>                                                        |
| [**Erf**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Retorna informações de erro de FCI/FDI. O chamador não deve modificar essa estrutura.<br/> |
| [**FDICABINETINFO**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Contém detalhes sobre um arquivo de gabinete específico.<br/>                                    |
| [**FDINOTIFICATION**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Forneça informações para [**FNFDINOTIFY.**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify)<br/>                           |
| [**Sessão**](session.md)                             | Contém informações sobre a sessão atual.<br/>                                      |
