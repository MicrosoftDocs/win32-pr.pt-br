<span data-ttu-id="c7bae-104">A WSC (central de segurança do Windows) é uma ferramenta de relatório abrangente que ajuda os usuários a estabelecer e manter uma camada de segurança protetora em seus sistemas de computador.</span><span class="sxs-lookup"><span data-stu-id="c7bae-104">Windows Security Center (WSC) is a comprehensive reporting tool that helps users establish and maintain a protective security layer around their computer systems.</span></span> <span data-ttu-id="c7bae-105">Quando uma camada de segurança é estabelecida, a central de segurança do Windows é inevidente enquanto monitora o estado de integridade do computador.</span><span class="sxs-lookup"><span data-stu-id="c7bae-105">Once a security layer is established, Windows Security Center is inconspicuous as it monitors the computer’s health state.</span></span> <span data-ttu-id="c7bae-106">No entanto, se existirem vulnerabilidades, a WSC fornecerá alertas e orientação prescritiva para ajudar o usuário a alcançar um estado seguro que é exibido para o usuário final por meio da central de ações.</span><span class="sxs-lookup"><span data-stu-id="c7bae-106">However, if vulnerabilities exist, WSC provides alerts and prescriptive guidance to assist the user in achieving a secure state which is surfaced to the end user through the Action Center.</span></span>
