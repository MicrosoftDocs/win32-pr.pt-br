# <a name="windows-security-center"></a>Central de Segurança do Windows

A WSC (central de segurança do Windows) é uma ferramenta de relatório abrangente que ajuda os usuários a estabelecer e manter uma camada de segurança protetora em seus sistemas de computador. Quando uma camada de segurança é estabelecida, a central de segurança do Windows é inevidente enquanto monitora o estado de integridade do computador. No entanto, se existirem vulnerabilidades, a WSC fornecerá alertas e orientação prescritiva para ajudar o usuário a alcançar um estado seguro que é exibido para o usuário final por meio da central de ações.

Para que as soluções de segurança de terceiros (antivírus, Antimalware ou AntiSpyware) sejam compatíveis com o Windows e relatem com êxito o status para a central de ações, elas são necessárias para se registrarem com a central de segurança e relatar quaisquer alterações de status subsequentes usando APIs privadas para se comunicar com a WSC. A WSC, por sua vez, comunica essas atualizações para a central de ações, onde elas são finalmente exibidas para o usuário final.
