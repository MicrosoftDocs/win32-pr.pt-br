# <a name="windows-security-center"></a>Central de Segurança do Windows

Segurança do Windows O WSC (Center) é uma ferramenta de relatório abrangente que ajuda os usuários a estabelecer e manter uma camada de segurança de proteção em torno de seus sistemas de computador. Depois que uma camada de segurança é estabelecida, Segurança do Windows Center fica inconspicuo, pois monitora o estado de saúde do computador. No entanto, se existirem vulnerabilidades, o WSC fornece alertas e diretrizes prescritivas para ajudar o usuário a alcançar um estado seguro que é a superfície para o usuário final por meio da Central de Ações.

Para que soluções de segurança de terceiros (antivírus, antimalware ou antispyware) sejam compatíveis com o Windows e reportem com êxito o status à Central de Ações, elas são necessárias para se registrar na Central de Segurança e relatar quaisquer alterações de status subsequentes usando APIs privadas para se comunicar com o WSC. O WSC, por sua vez, comunica essas atualizações à Central de Ações, em que elas são finalmente exibidas para o usuário final.
