# <a name="windows-security-center-interfaces"></a>Segurança do Windows Interfaces do Centro

As interfaces a seguir estão disponíveis para uso com o Segurança do Windows Center.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                 | Descrição                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**IWscProduct**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Fornece métodos para obter informações sobre o produto para um provedor individual interagir com Segurança do Windows Center.<br/> |
| [**IWSCProductList**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Fornece métodos para coletar informações do produto para o tipo selecionado de provedores instalados no computador.<br/>         |
