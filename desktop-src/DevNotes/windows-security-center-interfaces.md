# <a name="windows-security-center-interfaces"></a>Interfaces da central de segurança do Windows

As interfaces a seguir estão disponíveis para uso com a central de segurança do Windows.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                 | Descrição                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**IWscProduct**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Fornece métodos para obter informações de produto para um provedor individual interagir com a central de segurança do Windows.<br/> |
| [**IWSCProductList**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Fornece métodos para coletar informações de produtos para o tipo selecionado de provedores instalados no computador.<br/>         |
