# <a name="tag-types"></a>Tipos de marca

Descreve o tipo de dados do valor associado a uma marca.



| Constante/valor                                                                                                                                                                                                                            | Descrição                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| <span id="TAG_TYPE_NULL"></span><span id="tag_type_null"></span><dl> <dt>**Marca \_ de Digite \_**</dt> o <dt>0x1000</dt> nulo </dl>                | Não há nenhum valor associado à marca.<br/> |
| <span id="TAG_TYPE_BYTE"></span><span id="tag_type_byte"></span><dl> <dt>**Marca \_ de 0x2000 de tipo \_ byte**</dt> <dt></dt> </dl>                | Um valor de **byte** .<br/>                          |
| <span id="TAG_TYPE_WORD"></span><span id="tag_type_word"></span><dl> <dt>**Marca \_ de Digite o \_ Word**</dt> <dt>0x3000</dt> </dl>                | Um valor de **palavra** .<br/>                          |
| <span id="TAG_TYPE_DWORD"></span><span id="tag_type_dword"></span><dl> <dt>**Marca \_ de Digite \_ DWORD**</dt> <dt>0x4000</dt> </dl>             | Um valor **DWORD** .<br/>                         |
| <span id="TAG_TYPE_QWORD"></span><span id="tag_type_qword"></span><dl> <dt>**Marca \_ de Digite \_ QWORD**</dt> <dt>0x5000</dt> </dl>             | Um valor de **ULONGLONG** .<br/>                     |
