# <a name="sndvol32exe"></a>SndVol32.exe

O programa SndVol32.exe controla as configurações de volume do seu sistema. Para exibir esse controle de volume, vá para o painel de controle e inicie sons e dispositivos de áudio. Vá para o **volume do dispositivo** e clique em **avançado**.

Para executar esse programa com argumentos de linha de comando, vá para o menu **Iniciar** , clique em **executar** e digite o comando a seguir.

**SndVol32.exe \[ -R \| -D** *n * * * \]**

<dl> <dt>

<span id="-R"></span><span id="-r"></span>-R
</dt> <dd>

Inicia o aplicativo no modo de registro.

</dd> <dt>

<span id="-D_n"></span><span id="-d_n"></span><span id="-D_N"></span>-D *n*
</dt> <dd>

Especifica um identificador de dispositivo.

</dd> </dl>

## <a name="remarks"></a>Comentários

Este programa é o volume de áudio nativo e o controle do mixer incluídos no Windows; Ele não está disponível para download da Microsoft. O programa é normalmente instalado no seguinte diretório: C: \\ Windows \\ System32.

Para localizar esse arquivo em seu sistema, abra o Windows Explorer, clique no botão **Pesquisar** e selecione **todos os arquivos e pastas**. Digite o nome do arquivo (SndVol32.exe) na primeira linha disponível e clique no botão **Pesquisar** .

Para obter mais informações sobre como usar esse controle, inicie o programa e selecione **ajuda**. Observe que não há como acessar programaticamente a funcionalidade deste programa.
