# <a name="termsrvappinstallmode-function"></a>Função TermsrvAppInstallMode

\[Essa função não tem suporte e não deve ser usada. Ele pode mudar ou desaparecer completamente sem aviso prévio.\]

Determina se o Terminal Server está no modo de instalação.

## <a name="syntax"></a>Sintaxe


```C++
BOOL TermsrvAppInstallMode(void);
```



## <a name="parameters"></a>Parâmetros

Essa função não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Essa função retornará **true** se o Terminal Server estiver no modo de instalação e **false** se estiver no modo de execução.
