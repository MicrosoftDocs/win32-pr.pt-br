# <a name="termsrvappinstallmode-function"></a>Função TermsrvAppInstallMode

\[Não há suporte para essa função e não deve ser usada. Ele pode mudar ou desaparecer completamente sem aviso prévio.\]

Determina se o Servidor de Terminal está no modo DE INSTALAÇÃO.

## <a name="syntax"></a>Sintaxe


```C++
BOOL TermsrvAppInstallMode(void);
```



## <a name="parameters"></a>Parâmetros

Essa função não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Essa função **retornará TRUE** se o Servidor de Terminal estiver no modo INSTALL e **FALSE** se estiver no modo EXECUTE.
