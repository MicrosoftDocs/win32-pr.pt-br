# <a name="windows-security-center-enumerations"></a>Enumerações da central de segurança do Windows

As enumerações a seguir são usadas com a central de segurança do Windows.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                | Descrição                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**\_estado do \_ produto de segurança da WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | Define o estado atual do produto de segurança que é disponibilizado para a central de segurança do Windows. <br/> |
| [**\_provedor de segurança da WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | Define todos os serviços monitorados pela WSC (central de segurança do Windows).<br/>                         |
| [**\_integridade do \_ provedor de segurança WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | Define os possíveis estados de qualquer serviço monitorado pela WSC (central de segurança do Windows).<br/>               |
| [**\_status da \_ assinatura de segurança WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | Relata o status da versão atual do produto de segurança para a central de segurança do Windows. <br/>               |
