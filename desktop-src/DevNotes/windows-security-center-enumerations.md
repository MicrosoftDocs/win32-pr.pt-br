# <a name="windows-security-center-enumerations"></a>Segurança do Windows Enumerações do Centro

As enumerações a seguir são usadas com Segurança do Windows Center.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                | Descrição                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**ESTADO DO PRODUTO \_ \_ DE SEGURANÇA WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | Define o estado atual do produto de segurança que é disponibilizado para Segurança do Windows Central. <br/> |
| [**PROVEDOR DE SEGURANÇA \_ WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | Define todos os serviços monitorados pelo WSC (Segurança do Windows Center).<br/>                         |
| [**WSC \_ SECURITY \_ PROVIDER \_ HEALTH**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | Define os estados possíveis para qualquer serviço monitorado pelo WSC (Segurança do Windows Center).<br/>               |
| [**STATUS DA \_ ASSINATURA \_ DE SEGURANÇA DO WSC \_**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | Relata o status da versão atual do produto de segurança para Segurança do Windows Central. <br/>               |
