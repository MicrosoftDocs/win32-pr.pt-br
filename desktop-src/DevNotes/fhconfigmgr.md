# <a name="fhconfigmgr-class"></a>Classe FhConfigMgr

Representa a configuração do histórico de arquivos do usuário sob o qual o objeto **FhConfigMgr** é criado. Todas as operações de configuração são executadas chamando os métodos da interface [**IFhConfigMgr**](/windows/desktop/api/Fhcfg/nn-fhcfg-ifhconfigmgr) .

## <a name="when-to-implement"></a>Quando implementar

A API de histórico de arquivo implementa essa classe. Para criar uma instância dessa classe, use a função [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) .
