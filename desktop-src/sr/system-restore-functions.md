# <a name="system-restore-functions"></a>Restauração do Sistema funções

As funções a seguir são usadas com Restauração do Sistema.



| Função                                             | Descrição                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Exclui o ponto de restauração especificado.                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Especifica o início e o término de um conjunto de alterações para que Restauração do Sistema possa criar um ponto de restauração. |
