# <a name="system-restore-functions"></a>Funções de restauração do sistema

As funções a seguir são usadas com a restauração do sistema.



| Função                                             | Descrição                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Exclui o ponto de restauração especificado.                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Especifica o início e o final de um conjunto de alterações para que a restauração do sistema possa criar um ponto de restauração. |
