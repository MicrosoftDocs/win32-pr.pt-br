# <a name="about-system-restore"></a>Sobre a restauração do sistema

A restauração do sistema monitora as alterações do sistema e salva o estado do sistema como um *ponto de restauração*. Se um problema do sistema for Desenvolvedo como resultado de uma alteração no sistema, o usuário poderá retornar o sistema para um estado anterior usando os dados de um ponto de restauração.

Os aplicativos e o sistema podem criar pontos de restauração quando ocorrem alterações no sistema. A restauração do sistema também pode ser configurada para criar pontos de restauração em intervalos regulares, e os usuários podem criar pontos de restauração a qualquer momento. a restauração do sistema no Windows 7 é verificada todos os dias e cria um ponto de restauração agendado se não forem criados outros pontos de restauração nos últimos 7 dias. a restauração do sistema no Windows Vista criará um ponto de restauração se nenhum outro ponto de restauração tiver sido criado nas últimas 24 horas. a restauração do sistema no Windows XP cria um ponto de verificação a cada 24 horas de tempo absoluto.

A restauração do sistema não restaura dados ou documentos do usuário, portanto, não fará com que os usuários percam seus arquivos, email, histórico de navegação ou favoritos. a restauração do sistema também é disponibilizada para os usuários no ambiente de recuperação de Windows ou no modo de segurança, facilitando a restauração de seus computadores para um estado anterior à ocorrência de problemas.
