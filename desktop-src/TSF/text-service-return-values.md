# <a name="text-service-return-values"></a>Valores de retorno do serviço de texto

Um serviço de texto produz um valor de retorno de 0x *HHHH* 0600. A tabela a seguir lista o valor de retorno do serviço de texto.



| Código/valor de retorno                                                                                                                                                                                                                          | Descrição                                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <span id="TF_E_NOCONVERSION"></span><span id="tf_e_noconversion"></span><dl> <dt>**TF \_ E \_ noConversion**</dt> <dt>0x80040600</dt> </dl> | Nenhuma conversão está disponível para o intervalo de texto selecionado.<br/> |
