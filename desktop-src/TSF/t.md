# <a name="t-text-services-framework"></a>T (estrutura de serviços de texto)

<dl> <dt>

<span id="_tsf_text_service_gly"></span><span id="_TSF_TEXT_SERVICE_GLY"></span>**serviço de texto**
</dt> <dd>

Um tipo de aplicativo que permite a entrada de texto por meio de fala, teclado multilíngue ou dispositivo de caneta. Um serviço de texto também pode habilitar ferramentas de edição de texto, como verificação ortográfica e gramatical.

Consulte também *processador de texto*.

</dd> <dt>

<span id="_tsf_text_services_framework_gly"></span><span id="_TSF_TEXT_SERVICES_FRAMEWORK_GLY"></span>**TSF (estrutura de serviços de texto)**
</dt> <dd>

Um sistema independente de dispositivo, neutro por idioma e extensível para entrada de texto e tecnologias de idioma natural.

</dd> </dl>
