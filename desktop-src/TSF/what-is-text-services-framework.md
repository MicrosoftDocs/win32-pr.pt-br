# <a name="what-is-text-services-framework"></a>O que é a estrutura de serviços de texto?

A TSF (estrutura de serviços de texto) é um sistema independente de dispositivo, com neutralidade de linguagem e de linguagem extensível para entrada de texto e tecnologias de idioma.

Os aplicativos habilitados para TSF podem receber entrada de texto de qualquer serviço de texto que ofereça suporte a TSF sem ter que estar ciente dos detalhes da fonte de texto.

Os serviços de texto fornecem texto aos aplicativos de uma maneira que não exige nenhum conhecimento detalhado dos aplicativos que produzem ou recebem o texto. Por exemplo, um serviço de texto pode fornecer entrada de texto de dispositivos de entrada de fala ou manuscrito.
