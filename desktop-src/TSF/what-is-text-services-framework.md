# <a name="what-is-text-services-framework"></a>O que é Estrutura de Serviços de Texto?

Estrutura de Serviços de Texto (TSF) é um sistema independente de dispositivo, neutro em idioma e extensível para tecnologias de entrada de texto e idioma.

Aplicativos habilitados para TSF podem receber entrada de texto de qualquer serviço de texto que dá suporte a TSF sem precisar estar ciente de nenhum detalhe da fonte de texto.

Os serviços de texto fornecem texto para aplicativos de uma maneira que não exige nenhum conhecimento detalhado dos aplicativos que produzem ou recebem o texto. Por exemplo, um serviço de texto pode fornecer entrada de texto de dispositivos de entrada de fala ou manuscrito.
