# <a name="tf_lbmenuf_-constants"></a>TF \_ LBMENUF \_ \* constantes

As constantes ** \_ TF \_ \* LBMENUF* _ são usadas no método [ITfMenu:: addMenuItem](/windows/desktop/api/Ctfutb/nf-ctfutb-itfmenu-addmenuitem) para especificar as características de um item de menu na barra de idiomas.



| Constante/valor                                                                                                                                                                                                                                         | Descrição                                     |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------|
| <span id="TF_LBMENUF_CHECKED"></span><span id="tf_lbmenuf_checked"></span><dl> <dt>_ * TF \_ LBMENUF \_ verificado * *</dt> <dt>0x01</dt> </dl>                | O item de menu está marcado.<br/>            |
| <span id="TF_LBMENUF_SUBMENU"></span><span id="tf_lbmenuf_submenu"></span><dl> <dt>**TF \_ \_Submenu LBMENUF**</dt> <dt>0x02</dt> </dl>                | O item de menu é um submenu.<br/>          |
| <span id="TF_LBMENUF_SEPARATOR"></span><span id="tf_lbmenuf_separator"></span><dl> <dt>**TF \_ \_Separador de LBMENUF**</dt> <dt>0x04</dt> </dl>          | O item de menu é um separador.<br/>        |
| <span id="TF_LBMENUF_RADIOCHECKED"></span><span id="tf_lbmenuf_radiochecked"></span><dl> <dt>**TF \_ LBMENUF \_ RADIOCHECKED**</dt> <dt>0x08</dt> </dl> | O item de menu é uma marca de seleção de rádio.<br/> |
| <span id="TF_LBMENUF_GRAYED"></span><span id="tf_lbmenuf_grayed"></span><dl> <dt>**TF \_ LBMENUF \_ acinzentado**</dt> , <dt>0x10</dt> </dl>                   | O item de menu está desabilitado.<br/>           |
