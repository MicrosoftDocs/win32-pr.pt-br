# <a name="about-text-services-framework"></a>Sobre a estrutura de serviços de texto

A Microsoft Windows Text Services Framework (TSF) é uma API (interface de programação de aplicativo) que permite entrada de texto avançada e independente de origem. O TSF expõe sua estrutura a aplicativos e serviços de texto por meio de um conjunto de interfaces COM.
