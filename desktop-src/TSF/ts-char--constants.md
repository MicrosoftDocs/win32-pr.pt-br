# <a name="ts_char_-constants"></a>\_Constantes de caractere TS \_ \*

As constantes de TS \_ Char \_ \* descrevem o tipo de caractere.



| Constante/valor                                                                                                                                                                                                                                   | Descrição                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_ CHAR \_ Embedded**</dt> <dt>(0xfffc)</dt> </dl>          | O caractere é um caractere de substituição de objeto Unicode 2,1.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_ \_Região Char**</dt> <dt>(0)</dt> </dl>                     | O caractere é um limite de região.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_ \_Substituição de caracteres**</dt> <dt>(0xFFFD)</dt> </dl> | O caractere é um caractere de espaço reservado de texto oculto ou um caractere de substituição Unicode.<br/> |
