# <a name="ts_char_-constants"></a>Constantes \_ CHAR \_ \* TS

As \_ constantes CHAR \_ \* TS descrevem o tipo de caractere.



| Constante/valor                                                                                                                                                                                                                                   | Descrição                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_ CHAR \_ EMBEDDED**</dt> <dt>( 0xfffc )</dt> </dl>          | O caractere é um caractere de substituição de objeto Unicode 2.1.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_ REGIÃO \_ CHAR**</dt> <dt>( 0 )</dt> </dl>                     | O caractere é um limite de região.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_ SUBSTITUIÇÃO \_ CHAR**</dt> <dt>( 0xfffd )</dt> </dl> | O caractere é um caractere de espaço reservado de texto oculto ou um caractere de substituição Unicode.<br/> |
