# <a name="text-services-framework-text-services-framework"></a>Estrutura de serviços de texto (estrutura de serviços de texto)

## <a name="purpose"></a>Finalidade

O Microsoft Windows Text Services Framework (TSF) é um serviço de sistema disponível como um pacote redistribuível para o Windows. O TSF fornece uma estrutura simples e escalonável para a entrega de tecnologias avançadas de entrada de texto e linguagem natural. O TSF pode ser habilitado em aplicativos ou como um serviço de texto do TSF. Um serviço de texto do TSF fornece suporte multilíngue e fornece serviços de texto, como processadores de teclado, reconhecimento de manuscrito e reconhecimento de fala.

## <a name="where-applicable"></a>Quando aplicável

A estrutura de serviços de texto é aplicável a computadores baseados no Windows usando serviços de texto e o Windows XP ou versões posteriores do sistema operacional.

## <a name="developer-audience"></a>Público de desenvolvedores

A estrutura de serviços de texto é projetada para uso por programadores de Component Object Model (COM) usando as linguagens de programação C/C++. Os programadores devem estar familiarizados com os serviços de texto para computadores baseados no Windows. É recomendável conhecer o reconhecimento de manuscrito, o reconhecimento de fala e a programação de suporte multilíngüe.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Para os pacotes redistribuíveis mais recentes, baixe o [SDK da plataforma Windows 10](https://developer.microsoft.com/windows/downloads/windows-10-sdk).
