# <a name="c-text-services-framework"></a>C (estrutura de serviços de texto)

<dl> <dt>

<span id="_tsf_candidates_gly"></span><span id="_TSF_CANDIDATES_GLY"></span>**Candidate**
</dt> <dd>

Um conjunto de caracteres alternativos ou ideogramas exibidos em uma janela candidata por um método de entrada. Os candidatos são obtidos de um dicionário de conversão e representam uma cadeia de caracteres fonética anteriormente inserida pelo usuário.

</dd> <dt>

<span id="_tsf_client_gly"></span><span id="_TSF_CLIENT_GLY"></span>**cliente**
</dt> <dd>

Na estrutura de serviços de texto, um serviço de texto ou aplicativo habilitado para TSF.

</dd> <dt>

</dd> </dl>
