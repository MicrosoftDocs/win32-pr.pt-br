# <a name="a-text-services-framework"></a>A (estrutura de serviços de texto)

<dl> <dt>

<span id="_tsf_anchor_gly"></span><span id="_TSF_ANCHOR_GLY"></span>**fundamenta**
</dt> <dd>

Um de um conjunto de elementos que marca o início e o término de um intervalo de texto. Consulte [âncoras](ranges.md).

</dd> <dt>

<span id="_tsf_application_character_position_gly"></span><span id="_TSF_APPLICATION_CHARACTER_POSITION_GLY"></span>**posição de caracteres do aplicativo (ACP)**
</dt> <dd>

Uma posição de caractere com base em zero, em um fluxo de texto contínuo, que é relativa ao início do fluxo. Um ACP é indicado com um valor inteiro longo. Confira [armazenamentos de texto](text-stores.md).

</dd> </dl>
