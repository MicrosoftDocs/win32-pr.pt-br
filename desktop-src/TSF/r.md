# <a name="r-text-services-framework"></a>R (estrutura de serviços de texto)

<dl> <dt>

<span id="_tsf_range_gly"></span><span id="_TSF_RANGE_GLY"></span>**amplitude**
</dt> <dd>

Um objeto que descreve um intervalo de texto que existe em um fluxo de texto. Um intervalo é usado para ler, gravar e manipular texto. Também conhecido como *intervalo de texto*.

Consulte também *região*.

</dd> <dt>

<span id="_tsf_reconversion_gly"></span><span id="_TSF_RECONVERSION_GLY"></span>**reconversão**
</dt> <dd>

O processo de permitir que o usuário selecione texto alternativo para o texto inserido.

</dd> <dt>

<span id="_tsf_region_gly"></span><span id="_TSF_REGION_GLY"></span>**regionais**
</dt> <dd>

Uma extensão de texto tratada como um intervalo de edição exclusivo, com base na formatação.

</dd> </dl>
