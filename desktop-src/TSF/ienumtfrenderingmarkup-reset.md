# <a name="ienumtfrenderingmarkupreset-method"></a>Método IEnumTfRenderingMarkup:: Reset

O método **IEnumTfRenderingMarkup:: Reset** redefine o objeto enumerador movendo a posição atual para o início da sequência de enumeração.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

| Valor                                                                                | Descrição                           |
|--------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**S \_ OK**</dt> </dl> | O método foi bem-sucedido.<br/> |
