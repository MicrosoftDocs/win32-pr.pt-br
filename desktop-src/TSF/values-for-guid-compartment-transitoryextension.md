# <a name="values-for-guid_compartment_transitoryextension"></a>Valores para o \_ compartimento de GUID \_ TRANSITORYEXTENSION

Os valores a seguir do compartimento de TRANSITORYEXTENSION do compartimento de GUID \_ \_ são usados para controlar o comportamento da extensão transitória.

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF \_ TRANSITORYEXTENSION \_ None**
</dt> <dd>

Esse valor interrompe a extensão transitória.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**TF \_ TRANSITORYEXTENSION \_ flutuante**
</dt> <dd>

Esse valor inicia a extensão transitória com a interface do usuário flutuante.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF \_ TRANSITORYEXTENSION \_ ATSELECTION**
</dt> <dd>

Esse valor inicia a extensão transitória com a interface do usuário pop-up no IP ou a seleção do Gerenciador de documentos pai.

</dd> </dl>
