# <a name="ts_ch_-constants"></a>\_Constantes TS ch \_ \*

As \_ constantes TS ch \_ \* são usadas internamente pelo TSF para indicar que as exclusões de texto ocorreram.



| Constante/valor                                                                                                                                                                                                                               | Descrição                                                        |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------|
| <span id="TS_CH_PRECEDING_DEL"></span><span id="ts_ch_preceding_del"></span><dl> <dt>**TS \_ CH \_ anterior \_ del**</dt> <dt>(1)</dt> </dl> | Texto que precede a posição do caractere foi excluído.<br/> |
| <span id="TS_CH_FOLLOWING_DEL"></span><span id="ts_ch_following_del"></span><dl> <dt>**TS \_ CH \_ após \_ del**</dt> <dt>(2)</dt> </dl> | O texto após a posição do caractere foi excluído.<br/> |
