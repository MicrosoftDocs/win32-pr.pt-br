# <a name="m-text-services-framework"></a>M (estrutura de serviços de texto)

<dl> <dt>

<span id="_tsf_modal_gly"></span><span id="_TSF_MODAL_GLY"></span>**janelas**
</dt> <dd>

Um termo que descreve uma janela secundária que restringe a interação do usuário com outras janelas. Uma janela modal, como uma caixa de diálogo, deve ser fechada pelo usuário para que o aplicativo possa continuar.

</dd> </dl>
