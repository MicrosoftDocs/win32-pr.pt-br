# <a name="d-text-services-framework"></a>D (estrutura de serviços de texto)

<dl> <dt>

<span id="_tsf_document_lock_gly"></span><span id="_TSF_DOCUMENT_LOCK_GLY"></span>**bloqueio de documento**
</dt> <dd>

Um estado de documento no qual o Gerenciador de TSF pode ler ou modificar o texto sem que o texto seja alterado por outros meios. Um bloqueio de documento pode ser somente leitura ou leitura/gravação.

</dd> </dl>
