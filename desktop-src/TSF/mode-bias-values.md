# <a name="mode-bias-values"></a>Valores de tendência do modo

<dl> <dt>

<span id="GUID_MODEBIAS_NONE_____________"></span><span id="guid_modebias_none_____________"></span>\_MODEBIAS GUID \_ None

</dd> <dt>

<span id="GUID_MODEBIAS_FILENAME_____________"></span><span id="guid_modebias_filename_____________"></span>\_nome do \_ arquivo GUID MODEBIAS

</dd> <dt>

<span id="GUID_MODEBIAS_NAME_____________"></span><span id="guid_modebias_name_____________"></span>\_nome do MODEBIAS de GUID \_

</dd> <dt>

<span id="GUID_MODEBIAS_READING_____________"></span><span id="guid_modebias_reading_____________"></span>\_MODEBIAS de \_ leitura de GUID

<span id="GUID_MODEBIAS_DATETIME_____________"></span><span id="guid_modebias_datetime_____________"></span>data e hora do GUID \_ MODEBIAS \_

<span id="GUID_MODEBIAS_CONVERSATION_____________"></span><span id="guid_modebias_conversation_____________"></span>\_MODEBIAS de \_ conversação GUID

<span id="GUID_MODEBIAS_CHINESE_____________"></span><span id="guid_modebias_chinese_____________"></span>GUID \_ MODEBIAS \_ chinês
