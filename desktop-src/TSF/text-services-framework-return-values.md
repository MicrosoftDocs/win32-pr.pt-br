# <a name="text-services-framework-return-values"></a>Valores de retorno da estrutura de serviços de texto

Os métodos de interface da estrutura de serviços de texto produzem valores de retorno no intervalo de 0x *HHHH* 0200 a 0x *HHHH* 0600, em que *HHHH* é um valor hexadecimal que depende do código de severidade e do código de recurso de um valor de retorno específico.

O grupo valores de retorno nas categorias a seguir.

-   [Valores de retorno do serviço de texto](text-service-return-values.md)
-   [Valores de retorno do gerente](manager-return-values.md)
-   [Valores de retorno do repositório de texto](text-store-return-values.md)
