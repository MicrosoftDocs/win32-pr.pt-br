# <a name="adding-source-nodes-with-topoedit"></a>Adicionando nós de origem com TopoEdit

Um nó de origem representa um fluxo em um arquivo de mídia. Para criar um nó de origem no TopoEdit, você especifica um arquivo de mídia. TopoEdit enumera os fluxos no arquivo e cria os nós de origem apropriados.

Para obter informações sobre como adicionar nós de origem programaticamente usando APIs Media Foundation, consulte [criando nós de origem](creating-source-nodes.md).

## <a name="to-add-a-source-node-to-a-topology"></a>Para adicionar um nó de origem a uma topologia

1.  No menu **topologia** , clique em **Adicionar origem**.

    A caixa de diálogo **selecionar origem da mídia** é aberta.

2.  Navegue até a pasta onde o arquivo de mídia está localizado.

3.  No campo **nome do arquivo:** , insira o nome do arquivo.

4.  Clique em **Abrir**.

TopoEdit cria nós de origem para os fluxos. O **painel topologia** mostra o nó de origem contido em uma caixa cinza que mostra o nome do arquivo de mídia. O nó de origem mostra o tipo de fluxo do nó.
