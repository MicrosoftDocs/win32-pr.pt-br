# <a name="adding-source-nodes-with-topoedit"></a>Adicionando nós de origem com TopEdit

Um nó de origem representa um fluxo em um arquivo de mídia. Para criar um nó de origem no TopoEdit, especifique um arquivo de mídia. TopoEdit enumera os fluxos no arquivo e cria os nós de origem apropriados.

Para obter informações sobre como adicionar nós de origem programaticamente usando APIs Media Foundation, consulte [Criando nós de origem](creating-source-nodes.md).

## <a name="to-add-a-source-node-to-a-topology"></a>Para adicionar um nó de origem a uma topologia

1.  No menu **Topologia,** clique em **Adicionar Origem.**

    A **caixa de diálogo Selecionar Fonte de** Mídia é aberta.

2.  Navegue até a pasta em que o arquivo de mídia está localizado.

3.  No campo **Nome do arquivo:** , insira o nome do arquivo.

4.  Clique em **Abrir**.

TopEdit cria nós de origem para os fluxos. O **Painel de Topologia mostra** o nó de origem contido em uma caixa cinza que mostra o nome do arquivo de mídia. O nó de origem mostra o tipo de fluxo do nó.
