# <a name="mf_mt_max_mastering_luminance-attribute"></a>Atributo de luminância de controle de mestre do MF \_ MT \_ Max \_ \_

Especifica a luminância máxima da exibição na qual o conteúdo foi criado, em nits. Esse atributo tem a mesma semântica que a **\_ \_ \_ luminância de mestre de exibição máxima** , conforme definido no padrão CEA-861,3.

## <a name="data-type"></a>Tipo de dados

**UINT32**
