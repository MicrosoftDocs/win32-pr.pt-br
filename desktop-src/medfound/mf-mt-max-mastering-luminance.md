# <a name="mf_mt_max_mastering_luminance-attribute"></a>Atributo \_ MF MT \_ MAX \_ MASTERING \_ LUMINANCE

Especifica a luminância máxima da exibição na qual o conteúdo foi autor, em Nits. Esse atributo tem a mesma semântica que **a \_ \_ \_ luminância** de masterização máxima de exibição, conforme definido no padrão CEA-861.3.

## <a name="data-type"></a>Tipo de dados

**UINT32**
