# <a name="codecapi_avencvideoselectlayer-property"></a>Propriedade CODECAPI \_ AVEncVideoSelectLayer

Define a camada de seleção de vídeo para um codificador de vídeo.

## <a name="data-type"></a>Tipo de dados

**ULONG (VT \_ UI4)**

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoSelectLayer**

## <a name="remarks"></a>Comentários

Essa propriedade também é usada com codificadores de câmera [UVC 1.5 H.264](camera-encoder-h264-uvc-1-5.md).
