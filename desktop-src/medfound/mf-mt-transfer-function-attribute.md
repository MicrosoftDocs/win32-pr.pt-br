# <a name="mf_mt_transfer_function-attribute"></a>\_Atributo de \_ função de transferência do MF MT \_

Especifica a função de conversão de RGB para R'G'B ' para um tipo de mídia de vídeo.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

O valor desse atributo é um membro da enumeração [**MFVideoTransferFunction**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideotransferfunction) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
