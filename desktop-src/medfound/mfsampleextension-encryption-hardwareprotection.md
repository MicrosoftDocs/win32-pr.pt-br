# <a name="mfsampleextension_encryption_hardwareprotection-attribute"></a>Atributo MFSampleExtension de \_ criptografia \_ HardwareProtection

Especifica se um exemplo de mídia está protegido por hardware.

## <a name="data-type"></a>Tipo de dados

**Bool** armazenado como **UINT32**

## <a name="remarks"></a>Comentários

Um valor de 0 indica que o exemplo não está protegido por hardware. Um valor diferente de zero indica que a amostra está protegida por hardware.
