# <a name="codecapi_videoencoderdisplaycontenttype-property"></a>\_Propriedade CODECAPI VideoEncoderDisplayContentType

Define se o conteúdo é um vídeo de tela inteira, em oposição ao conteúdo da tela que pode ter uma janela de vídeo menor ou sem nenhum vídeo.

## <a name="data-type"></a>Tipo de dados

**\_UI4 VT**

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ VideoEncoderDisplayContentType**

## <a name="property-value"></a>Valor da propriedade

Se o valor for diferente de zero, o conteúdo será vídeo em tela inteira. Se o valor for zero, o conteúdo não será um vídeo de tela inteira.

## <a name="remarks"></a>Comentários

No Windows 10, o valor dessa propriedade não é usado.
