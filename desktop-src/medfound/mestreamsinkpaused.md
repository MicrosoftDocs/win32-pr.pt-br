# <a name="mestreamsinkpaused-event"></a>Evento MEStreamSinkPaused

Gerado por um coletor de fluxo quando ele conclui a transição para o estado em pausa. A transição para em pausa ocorre quando o método [**IMFPresentationClock::P ause**](/windows/desktop/api/mfidl/nf-mfidl-imfpresentationclock-pause) é chamado no relógio de apresentação do coletor.

## <a name="event-values"></a>Valores de evento

Os valores possíveis recuperados de [**IMFMediaEvent:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) incluem o seguinte.



| VARTYPE              | Descrição                           |
|----------------------|---------------------------------------|
| VT \_ vazio<br/> | Nenhum dado do evento.<br/> <br/> |
