# <a name="mf_mt_video_renderer_extension_profile-attribute"></a>\_Atributo de \_ \_ perfil de extensão do processador de vídeo MF \_ MT \_

Contém uma cadeia de caracteres que corresponde a uma entrada na lista **VideoRendererExtensionProfiles** de um manifesto de aplicativo UWP para selecionar qual efeito será carregado.

## <a name="data-type"></a>Tipo de dados

**LPCWSTR**
