# <a name="table-of-contents-parser-interfaces"></a>Interfaces do analisador de Sumário

As interfaces a seguir provaram o acesso aos serviços do analisador de Sumário.



| Interface                                | Descrição                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IToc**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoc)                     | Representa um sumário individual.                                                                           |
| [**ITocCollection**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoccollection) | Representa uma coleção de sumários.                                                                        |
| [**ITocEntry**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentry)           | Representa uma entrada individual em um sumário.                                                                |
| [**ITocEntryList**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentrylist)   | Representa uma lista de entradas em um sumário.                                                                  |
| [**ITocParser**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocparser)         | Fornece métodos para inserir tabelas de conteúdo em um arquivo de vídeo e ler tabelas de conteúdo de um arquivo de vídeo. |
