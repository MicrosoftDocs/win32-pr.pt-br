# <a name="mfpkey_clsid-property"></a>Propriedade \_ CLSID MFPKEY

Contém o CLSID para uma transformação Media Foundation (MFT).



Tipo de dados

Tipo PROPVARIANT (vt)

Membro PROPVARIANT

**GUID** (**CLSID** \* )

\_CLSID da VT

**puuid**



## <a name="remarks"></a>Comentários

Essa propriedade é opcional e é apenas informacional. Para obter essa propriedade, consulte o MFT para a interface **IPropertyStore.**
