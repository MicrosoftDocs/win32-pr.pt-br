# <a name="mfpkey_clsid-property"></a>\_Propriedade CLSID MFPKEY

Contém o CLSID para uma Media Foundation transformação (MFT).



Tipo de dados

Tipo PROPVARIANT (VT)

Membro PROPVARIANT

**GUID** (**CLSID** \* )

CLSID do VT \_

**puuid**



## <a name="remarks"></a>Comentários

Essa propriedade é opcional e é apenas informativa. Para obter essa propriedade, consulte o MFT para a interface **IPropertyStore** .
