# <a name="imfdxgidevicemanagersource-interface"></a>Interface IMFDXGIDeviceManagerSource

Fornece a funcionalidade para obter o [**IMFDXGIDeviceManager**](/windows/desktop/api/mfobjects/nn-mfobjects-imfdxgidevicemanager) do coletor de renderização de vídeo Microsoft Media Foundation.

## <a name="members"></a>Membros

A interface **IMFDXGIDeviceManagerSource** herda da interface [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **IMFDXGIDeviceManagerSource** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IMFDXGIDeviceManagerSource** tem esses métodos.



| Método                                                      | Descrição                                                                                                              |
|:------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------|
| [**GetManager**](imfdxgidevicemanagersource-getmanager.md) | Obtém o [**IMFDXGIDeviceManager**](/windows/desktop/api/mfobjects/nn-mfobjects-imfdxgidevicemanager) do coletor de renderização de vídeo Media Foundation.<br/> |
