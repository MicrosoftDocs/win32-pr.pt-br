# <a name="mfpkey_wmresamp_lowpass_bandwidth-property"></a>\_Propriedade de \_ largura de banda MFPKEY WMRESAMP LOWPASS \_

Especifica a largura de banda de filtro de passagem baixa, como uma porcentagem da taxa de amostragem de destino.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

VT \_ R4

## <a name="applies-to"></a>Aplica-se A

-   [DSP de reamostragem de áudio](audioresampler.md)

## <a name="remarks"></a>Comentários

O intervalo válido dessa propriedade é de 0,0 a 1,0, inclusive
