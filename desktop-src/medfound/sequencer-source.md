# <a name="sequencer-source"></a>Origem do sequenciador

A origem do sequenciador permite que um aplicativo reproduza uma coleção de fontes de mídia sequencialmente, com transições diretas entre as fontes. Você pode usá-lo para criar listas de reprodução ou para reproduzir fluxos de várias fontes simultaneamente.
