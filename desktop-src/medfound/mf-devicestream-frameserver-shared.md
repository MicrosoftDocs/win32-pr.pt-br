# <a name="mf_devicestream_frameserver_shared-attribute"></a>\_ \_ Atributo compartilhado MF DEVICESTREAM FRAMESERVER \_

Esse atributo, quando definido em um fluxo, marca explicitamente o fluxo como compartilhado pelo servidor de quadro.

## <a name="data-type"></a>Tipo de dados

**UINT32**
