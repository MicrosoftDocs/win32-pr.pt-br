# <a name="direct3d-11-video-interfaces"></a>Interfaces de vídeo do Direct3D 11

Esta seção contém informações de referência para as interfaces de vídeo do Microsoft Direct3D 11.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                 | Descrição                                                                                              |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**ID3D11AuthenticatedChannel**](/windows/desktop/api/d3d11/nn-d3d11-id3d11authenticatedchannel)<br/>           | Fornece um canal de comunicação com o driver de gráficos ou o tempo de execução do Microsoft Direct3D. <br/> |
| [**ID3D11CryptoSession**](/windows/desktop/api/d3d11/nn-d3d11-id3d11cryptosession)<br/>                         | Representa uma sessão criptográfica. <br/>                                                          |
| [**ID3D11VideoContext**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videocontext)<br/>                           | Fornece a funcionalidade de vídeo de um dispositivo Direct3D 11. <br/>                                    |
| [**ID3D11VideoContext1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videocontext1)<br/>                         | Fornece a funcionalidade de vídeo de um dispositivo Direct3D 11.<br/>                                     |
| [**ID3D11VideoContext2**](/windows/win32/api/d3d11_4/nn-d3d11_4-id3d11videocontext2)<br/>                         | Fornece a funcionalidade de vídeo de um dispositivo Direct3D 11. <br/>                                    |
| [**ID3D11VideoDecoder**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoder)<br/>                           | Representa um decodificador de vídeo acelerado por hardware para o Direct3D 11.<br/>                              |
| [**ID3D11VideoDecoderOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoderoutputview)<br/>       | Identifica as superfícies de saída que podem ser acessadas durante a decodificação de vídeo.<br/>                    |
| [**ID3D11VideoDevice**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodevice)<br/>                             | Fornece a decodificação de vídeo e os recursos de processamento de vídeo de um dispositivo Direct3D 11. <br/>       |
| [**ID3D11VideoDevice1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videodevice1)<br/>                           | Fornece a decodificação de vídeo e os recursos de processamento de vídeo de um dispositivo Direct3D 11. <br/>       |
| [**ID3D11VideoProcessor**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessor)<br/>                       | Representa um processador de vídeo para o Direct3D 11.<br/>                                                 |
| [**ID3D11VideoProcessorEnumerator**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorenumerator)<br/>   | Enumera os recursos do processador de vídeo de um dispositivo Direct3D 11. <br/>                         |
| [**ID3D11VideoProcessorEnumerator1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videoprocessorenumerator1)<br/> | Enumera os recursos do processador de vídeo de um dispositivo Direct3D 11.<br/>                          |
| [**ID3D11VideoProcessorInputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorinputview)<br/>     | Identifica as superfícies de entrada que podem ser acessadas durante o processamento de vídeo.<br/>                   |
| [**ID3D11VideoProcessorOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessoroutputview)<br/>   | Identifica as superfícies de saída que podem ser acessadas durante o processamento de vídeo.<br/>                  |
