# <a name="mf_mt_min_mastering_luminance-attribute"></a>Atributo de luminância de controle de mestre de MF \_ MT \_ min \_ \_

Especifica a luminância máxima da exibição na qual o conteúdo foi criado, em nits. Esse atributo tem a mesma semântica que o **mínimo de tela de exibição para a \_ \_ \_ luminância** , conforme definido no padrão CEA-861,3.

## <a name="data-type"></a>Tipo de dados

**UINT32**
