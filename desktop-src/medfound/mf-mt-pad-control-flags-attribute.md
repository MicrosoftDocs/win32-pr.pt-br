# <a name="mf_mt_pad_control_flags-attribute"></a>\_Atributo de \_ sinalizador de controle de painel \_ \_ do MF MT

Especifica a taxa de proporção do retângulo de saída para um tipo de mídia de vídeo.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

O valor desse atributo é um membro da enumeração [**MFVideoPadFlags**](/windows/desktop/api/mfapi/ne-mfapi-mfvideopadflags) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
