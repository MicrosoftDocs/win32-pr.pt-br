# <a name="mf_devicestream_max_frame_buffers-attribute"></a>Atributo de buffer de quadro do MF \_ DEVICESTREAM \_ máximo \_ \_

Especifica o número máximo de quadros que a fonte de captura de vídeo armazenará em buffer para esse fluxo.

## <a name="data-type"></a>Tipo de dados

**UINT32**
