# <a name="about-media-foundation"></a>Sobre o Media Foundation

Microsoft Media Foundation é a plataforma multimídia de última geração para Windows que permite aos desenvolvedores, consumidores e provedores de conteúdo adotar a nova onda de conteúdo premium com robustez aprimorada, qualidade incomparável e interoperabilidade perfeita.

Media Foundation requer Windows Vista ou posterior. Ele usa o COM (modelo de objeto de componente) e requer C/C++. A Microsoft não fornece uma API gerenciada para Media Foundation.

As APIs Media Foundation são parte do [SDK Windows .](https://msdn.microsoft.com/windowsvista/bb980924.aspx) Para desenvolver um Media Foundation, instale a versão mais recente do SDK do Windows.

## <a name="audio-and-video-quality"></a>Qualidade de áudio e vídeo

Media Foundation foi projetado para atender aos desafios enfrentados pelo conteúdo de alta definição. Os aprimoramentos de qualidade de áudio e vídeo feitos em toda a plataforma agora possibilitam fornecer uma ótima experiência para conteúdo de alta definição de última geração.

-   A DXVA (Aceleração de Vídeo) 2.0 do DirectX oferece aceleração de vídeo mais eficiente, em comparação com o DXVA 1.0, com decodificação de vídeo mais robusta e simplificada e uso estendido de hardware no processamento de vídeo. Com o DXVA 2.0, Windows pode lidar com alguns dos conteúdos de alta definição mais exigentes com alta qualidade e maior resiliência de falhas.

-   As informações de espaço em cores são preservadas em todo o pipeline de vídeo. Os usuários podem aproveitar o conteúdo de vídeo com fidelidade total. As informações de cores e as imagens entrelaçadas agora são passadas para o hardware para composições de passagem única. Preservar informações de espaço em cores também reduz conversões desnecessárias de espaço em cores, o que libera mais ciclos para processar conteúdo HD exigente.
-   O EVR (renderização de vídeo) aprimorado oferece melhor suporte de tempo, processamento aprimorado de vídeo e resiliência de falhas aprimorada. O suporte à reprodução de tela inteira foi aprimorado e a remação de vídeo no modo em janelas foi minimizada.
-   Media Foundation usa o MMCSS (Serviço de Agendador de Classes Multimídia), um novo serviço de sistema no Windows Vista. O MMCSS permite que aplicativos multimídia assegurem que seu processamento sensível ao tempo receba acesso priorizado aos recursos de CPU.

## <a name="content-access"></a>Acesso ao conteúdo

À medida que o entretenimento digital passa para a era de alta definição e o conteúdo se torna mais portátil e onipresente, a proteção de conteúdo se tornará uma parte integrante dos produtos de mídia digital. A extensibilidade do Media Foundation garante que ele possa dar suporte a essas tendências.

Além disso, Media Foundation extensibilidade permite que diferentes sistemas de proteção de conteúdo operem juntos.
