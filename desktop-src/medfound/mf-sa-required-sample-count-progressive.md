# <a name="mf_sa_required_sample_count_progressive-attribute"></a>\_Exemplo de \_ \_ \_ atributo progressivo de contagem de amostra necessária \_ do MF

Indica o número de amostras que uma Microsoft Media Foundation transformação (MFT) exige para ser alocada para conteúdo progressivo.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse valor será usado se o próximo nó downstream tiver um [**IMFVideoSampleAllocator**](/windows/desktop/api/mfidl/nn-mfidl-imfvideosampleallocator).
