# <a name="mf_mt_dv_vaux_src_pack-attribute"></a>\_Atributo de \_ pacote de src MT DV \_ Vaux \_ \_

Pacote de origem de vídeo auxiliar (VAUX) em um tipo de mídia de vídeo digital (DV).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

esse atributo corresponde ao membro **dwDVVAuxSrc** da estrutura do DirectShow [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
