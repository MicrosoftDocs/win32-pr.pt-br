# <a name="imfmediasourceextensiongetreadystate-method"></a>Método IMFMediaSourceExtension:: getpreparestate

Obtém o estado pronto da origem da mídia.

## <a name="syntax"></a>Sintaxe


```C++
MF_MSE_READY GetReadyState();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

O estado pronto da origem da mídia.
