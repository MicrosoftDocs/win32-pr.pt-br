# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Decodificação para S/PDIF (Microsoft Media Foundation)

O identificador de classe para o objeto de decodificador que transforma o conteúdo de Professional do Windows Professional media Audio 9.1 para entrega pelo S/PDIF (Formato de Interconexão Digital) da Philips/Philips é CLSID \_ CWMAudioSpdTxDMO. Você usa esse objeto como faria com qualquer outro decodificador . Ele dá suporte às mesmas propriedades que o decodificador Windows Media Audio 9 . Os tipos de saída enumerados por esse decodificador são todos formatos S/PDIF.

Para dar suporte total à saída S/PDIF, o computador cliente deve ter um hardware especial instalado.
