# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Decodificando para S/PDIF (Microsoft Media Foundation)

O identificador de classe do objeto decodificador que transforma o conteúdo do Windows Media Audio 9,1 Professional para entrega no formato de interconexão digital (S/PDIF) Sony/Philips é CLSID \_ CWMAudioSpdTxDMO. Você usa esse objeto como faria com qualquer outro decodificador. Ele dá suporte às mesmas propriedades do decodificador Windows Media Audio 9. Os tipos de saída enumerados por esse decodificador são todos os formatos S/PDIF.

Para oferecer suporte total à saída de S/PDIF, o computador cliente deve ter um hardware especial instalado.
