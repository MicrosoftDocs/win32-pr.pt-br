# <a name="codec-objects"></a>Objetos codec

Esta seção descreve os codificadores e decodificadores com suporte no Microsoft Media Foundation. Esta seção contém os seguintes tópicos.




| Tópico | Descrição | 
|-------|-------------|
| <a href="aac-decoder.md"><strong>Decodificador AAC</strong></a> | Um decodificador de áudio que decodifica os seguintes perfis AAC (Advanced Audio Coding) e HE-AAC (Alta Eficiência:<ul><li>Perfil de LC (baixa complexidade) do MPEG-2 AAC (multicanal).</li><li>MPEG-4 HE-AAC v1 (multicanal) com núcleo AAC-LC.</li><li>MPEG-4 HE-AAC v2 (estéreo) com núcleo AAC-LC.</li></ul> | 
| <a href="aac-encoder.md"><strong>Codificador AAC</strong></a> | Um codificador de áudio que codifica o perfil LC (Codificação de Áudio Avançado) de baixa complexidade (LC). | 
| <a href="dv-video-decoder.md"><strong>Decodificador de vídeo DV</strong></a> | Um decodificador de vídeo que decodifica o vídeo DV. | 
| <a href="h-264-video-decoder.md"><strong>Decodificador de vídeo H.264</strong></a> | Um decodificador de vídeo que decodifica o vídeo H.264. Ele dá suporte à decodificação de perfis de Linha de Base, Principal e Alta, até o nível 5.1. | 
| <a href="h-264-video-encoder.md"><strong>Codificador de vídeo H.264</strong></a> | Um codificador de vídeo que codifica o vídeo H.264. Ele dá suporte aos seguintes perfis H.264:<ul><li>Perfil de Linha de Base</li><li>Perfil Principal</li></ul> | 
| <a href="h-264-video-decoder.md"><strong>Decodificador de vídeo H.265/HEVC</strong></a> | Um decodificador de vídeo que dá suporte à decodificação de conteúdo H.265/HEVC no formato Anexo B e pode ser usado na reprodução de arquivos mp4 e m2ts. Ele dá suporte aos seguintes perfis H.264:<ul><li>Perfil Principal</li><li>Perfil principal 10</li><li>Imagem principal ainda</li></ul> | 
| <a href="h-264-video-encoder.md"><strong>Codificador de vídeo H.265/HEVC</strong></a> | Um codificador de vídeo que codifica o formato H.265/HEVC. Ele dá suporte aos seguintes perfis H.265:<ul><li>Perfil Principal</li></ul> | 
| <a href="mpeg4part2videodecoder.md"><strong>Decodificador de vídeo MPEG4 Parte 2</strong></a> | Decodificador de vídeo MPEG-4 Parte 2. | 
| <a href="windowsmediaaudiodecoder.md"><strong>Windows Decodificador de áudio de mídia</strong></a> | Um decodificador que decodifica fluxos codificados pelo codificador Windows de Áudio de Mídia. | 
| <a href="windowsmediaaudioencoder.md">Windows Codificador de áudio de mídia</a> | Um codificador de áudio que dá suporte a três categorias de saída codificada:<ul><li>Standard</li><li>Professional</li><li>Lossless</li></ul>A categoria Standard é para codificação de áudio de uso geral. A Professional é para codificar áudio de vários canais ou de alta definição. A categoria Sem perda é para compactar o áudio sem perder nenhum dos dados originais. | 
| <a href="windowsmediaaudiovoicedecoder.md"><strong>Windows Decodificador de voz de áudio de mídia</strong></a> | Um decodificador que decodifica fluxos codificados pelo codificador Windows Media Audio Voice. | 
| <a href="windowsmediaaudiovoiceencoder.md">Windows Codificador de voz de áudio de mídia</a> | Um codificador para codificação de áudio que contém principalmente fala. | 
| <a href="windows-media-mp3-decoder.md"><strong>Windows Decodificador mp3 de mídia</strong></a> | Um decodificador de áudio que decodifica os seguintes formatos de áudio.<ul><li>ISO/IEC 11172-3 (ÁUDIO MPEG-1) Camada 3</li><li>ISO/IEC 13818-3 (ÁUDIO MPEG-2) Camada 3, extensão de baixa frequência de amostragem</li></ul> | 
| <a href="windowsmediampeg4decoder.md"><strong>Windows Decodificador MPEG4 V1/V2 de mídia</strong></a> | Um decodificador que implementa a decodificação de vídeo MPEG-4 V1/V2. | 
| <a href="windows-media-video-7-and-8-encoders.md"><strong>Windows Codificador de Vídeo de Mídia 7/8</strong></a> | Um codificador de vídeo que implementa versões anteriores do codificador Windows Media Video. | 
| <a href="windowsmediavideo9decoder.md">Windows Decodificador do Vídeo de Mídia 9</a> | Um decodificador de vídeo que decodifica fluxos codificados pelo codificador Windows Media Video 9. | 
| <a href="windowsmediavideo9encoder.md">Windows Codificador de Vídeo de Mídia 9</a> | Um codificador de vídeo que dá suporte a três categorias de saída endoded:<ul><li>Windows Vídeo de mídia 9</li><li>Windows Perfil Avançado do Vídeo de Mídia 9</li><li>Windows Imagem do Vídeo de Mídia 9.1</li></ul>A Windows Vídeo de Mídia 9 é para codificação de vídeo de uso geral. A categoria Perfil Avançado é para codificar vídeo ou vídeo de alta definição que esteja em conformidade com o padrão vc-1 Advanced Profile SMPTE. A categoria Imagem é para converter imagens de bitmap em vídeo dinâmico. | 
| <a href="windowsmediavideo9screendecoder.md"><strong>Windows Decodificador de tela do Vídeo de Mídia 9</strong></a> | Um decodificador que decodifica fluxos codificados pelo codificador de tela Windows Media Video 9. | 
| <a href="windowsmediavideo9screenencoder.md"><strong>Windows Codificador de tela do Vídeo de Mídia 9</strong></a> | Um codificador para codificar vídeo de aplicativo de computador (captura de tela) ou outro vídeo altamente estático. |
