# <a name="mfpkey_videowindow-property"></a>\_Propriedade MFPKEY VIDEOWINDOW

Especifica a quantidade de conteúdo, em milissegundos, que pode caber no buffer de modelo.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMVCVideoWIndow

## <a name="data-type"></a>Tipo de Dados

\_I4 VT

## <a name="default-value"></a>Valor padrão

3000

## <a name="remarks"></a>Comentários

A janela buffer é um dos parâmetros de modelo "Bucket de vazamentos" usados no controle de taxa de codec.
