# <a name="direct3d-9-video-interfaces"></a>Interfaces de vídeo do Direct3D 9

Descreve as interfaces de vídeo do Microsoft Direct3D 9.



| Interface                                                                | Descrição                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | Consulta a sobreposição de recursos de hardware de um dispositivo Direct3D.                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | Fornece um canal de comunicação para o driver de gráficos ou para o tempo de execução do Direct3D.                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | Representa uma sessão criptográfica.                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | Permite que um aplicativo use a proteção de conteúdo e os serviços de criptografia implementados por um driver de gráficos. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Representa um acelerador de hardware para o DXVA (DirectX Video Acceleration) 1,0.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Habilita a decodificação acelerada por hardware de um dispositivo Direct3D, usando o DXVA versão 1,0.                      |
