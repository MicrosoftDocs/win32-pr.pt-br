# <a name="direct3d-9-video-interfaces"></a>Interfaces de vídeo do Direct3D 9

Descreve as interfaces de vídeo do Microsoft Direct3D 9.



| Interface                                                                | Descrição                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | Consulta os recursos de hardware de sobreposição de um dispositivo Direct3D.                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | Fornece um canal de comunicação para o driver gráfico ou para o runtime do Direct3D.                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | Representa uma sessão criptográfica.                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | Permite que um aplicativo use serviços de criptografia e proteção de conteúdo implementados por um driver gráfico. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Representa um acelerador de hardware para a DXVA (Aceleração de Vídeo) 1.0 do DirectX.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Habilita a decodificação acelerada por hardware de um dispositivo Direct3D usando a versão 1.0 do DXVA.                      |
