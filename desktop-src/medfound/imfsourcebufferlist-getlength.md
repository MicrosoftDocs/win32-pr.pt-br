# <a name="imfsourcebufferlistgetlength-method"></a>Método IMFSourceBufferList:: GetLength

Obtém o número de objetos [**IMFSourceBuffer**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) na lista.

## <a name="syntax"></a>Sintaxe


```C++
DWORD GetLength();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

O número de buffers de origem na lista.
