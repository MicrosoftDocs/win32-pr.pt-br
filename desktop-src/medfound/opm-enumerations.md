# <a name="opm-enumerations"></a>Enumerações de OPM

O [Gerenciador de proteção de saída](output-protection-manager.md) (OPM) usa as seguintes constantes e tipos de enumeração.



| Enumeração                                                                    | Descrição                                                                                                                                                                             |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sinalizadores de tipo de barramento OPM**](opm-bus-type-flags.md)                               | Especifica o tipo de barramento de e/s usado pelo adaptador gráfico.                                                                                                                             |
| [**Sinalizadores de tipo de conector OPM**](opm-connector-type-flags.md)                   | Especifica o tipo de conector físico para uma saída de vídeo.                                                                                                                            |
| [**Sinalizadores de tipo de proteção OPM**](opm-protection-type-flags.md)                 | Especifica mecanismos de proteção de saída para o Gerenciador de proteção de saída (OPM).                                                                                                             |
| [**Sinalizadores de status OPM**](opm-status-flags.md)                                   | Especifica o status de uma sessão OPM.                                                                                                                                                 |
| [**nível de proteção do OPM \_ ACP \_ \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_acp_protection_level)                | Especifica o nível de proteção para a proteção de cópia analógica (ACP).                                                                                                                        |
| [**\_nível de \_ proteção OPM DPCP \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_dpcp_protection_level)              | Especifica o nível de proteção para Proteção de Conteúdo DisplayPort (DPCP).                                                                                                               |
| [**\_nível de \_ proteção de HDCP de OPM \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_hdcp_protection_level)              | Especifica o nível de proteção para o High-Bandwidth digital Proteção de Conteúdo (HDCP).                                                                                                    |
| [**\_Taxa de proporção de imagem OPM \_ \_ \_ EN300294**](/windows/desktop/api/opmapi/ne-opmapi-opm_image_aspect_ratio_en300294) | Especifica a taxa de proporção para o ETSI EN 300 294.                                                                                                                                         |
| [**\_semântica de \_ saída de vídeo OPM \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_video_output_semantics)            | Especifica se a interface [**IOPMVideoOutput**](/windows/desktop/api/opmapi/nn-opmapi-iopmvideooutput) usará a semântica do Gerenciador de proteção de saída (OPM) ou a semântica do protocolo de proteção de saída (Copp) certificada. |
