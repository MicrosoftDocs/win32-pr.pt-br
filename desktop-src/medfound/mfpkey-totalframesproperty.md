# <a name="mfpkey_totalframes-property"></a>\_Propriedade MFPKEY TOTALFRAMES

Especifica o número total de quadros de vídeo passados para o codificador durante o processo de endocing.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMVCTotalFrames

## <a name="data-type"></a>Tipo de Dados

\_I4 VT

## <a name="remarks"></a>Comentários

Você pode obter esse valor depois de terminar de passar amostras.
