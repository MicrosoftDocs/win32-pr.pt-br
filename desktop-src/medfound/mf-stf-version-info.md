# <a name="mf_stf_version_info-attribute"></a>\_Atributo de \_ informações de versão do MF STF \_

Esse atributo representa a fábrica de transformação do sensor de informações de versão. Esse é o único atributo necessário para transformações de sensor. O valor é definido pelo desenvolvedor de transformação de sensor e é tratado como opaco pelo pipeline de mídia.

## <a name="data-type"></a>Tipo de dados

**WCHAR\***
