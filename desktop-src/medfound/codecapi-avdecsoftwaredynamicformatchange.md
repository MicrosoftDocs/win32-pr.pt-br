# <a name="codecapi_avdecsoftwaredynamicformatchange-property"></a>\_Propriedade CODECAPI AVDecSoftwareDynamicFormatChange

Define se a alteração de formato dinâmico de software deve ser usada para o redimensionamento interno.

## <a name="data-type"></a>Tipo de dados

**BOOL**

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVDecSoftwareDynamicFormatChange**

## <a name="property-value"></a>Valor da propriedade

Se definido como **true**, a alteração do formato dinâmico do software será usada para o redimensionamento interno.
