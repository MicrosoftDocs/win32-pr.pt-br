# <a name="mfpkey_wmavoice_enc_bufferwindow-property"></a>\_Propriedade MFPKEY WMAVOICE \_ ENC \_ BufferWindow

Especifica a janela de buffer, em milissegundos, a ser usada para o codec de voz.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMACVoiceBuffer

## <a name="data-type"></a>Tipo de Dados

\_I4 VT

## <a name="default-value"></a>Valor padrão

1500 para 4, 5, 8, 12 e 16 kbps

2177 para 10 e 20 kbps
