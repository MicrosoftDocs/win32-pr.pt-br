# <a name="how-do-i-decompress-streams-from-an-avi-file-or-other-file-about-which-i-have-no-format-information"></a>Como fazer descompactar fluxos de um arquivo AVI ou outro arquivo sobre o qual não tenho informações de formato?

sem informações de formato, incluindo dados de formato estendidos, não é possível descompactar dados que foram codificados com um codec de mídia Windows. Os codecs foram projetados para criar fluxos de dados para uso com o contêiner de arquivo ASF (Advanced Systems Format). A seção de cabeçalho de um arquivo ASF armazena as informações de formato de todos os fluxos no arquivo. ao usar o conteúdo codificado usando um dos Windows codecs de áudio e vídeo de mídia fora de um arquivo ASF, você deve garantir que as informações de formato estejam disponíveis. A maneira de fazer isso varia de um contêiner para outro.
