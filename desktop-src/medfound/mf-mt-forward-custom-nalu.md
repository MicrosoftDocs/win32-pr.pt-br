# <a name="mf_mt_forward_custom_nalu-attribute"></a>\_ \_ \_ Atributo NALU personalizado de encaminhamento MF MT \_

Especifica que os tipos de unidade da camada de abstração de rede (NAL) devem ser encaminhados nos exemplos de saída pelo decodificador.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Se o decodificador analisar um NALU, ele não será encaminhado.
