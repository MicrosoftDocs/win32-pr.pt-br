# <a name="saving-and-loading-topologies"></a>Salvando e carregando topologias

Com o TopoEdit, você pode salvar uma topologia como um arquivo XML e carregá-la posteriormente. Quando uma topologia completa é carregada, ela deve ser resolvida novamente. Para obter informações sobre esse procedimento, consulte [resolvendo uma topologia com TopoEdit](resolving-a-topology-with-topoedit.md).

## <a name="to-save-a-topology"></a>Para salvar uma topologia

1.  No menu **Arquivo**, clique em **Salvar**.

    A caixa de diálogo **salvar arquivo** é aberta.

2.  Navegue até a pasta onde o arquivo de topologia deve ser salvo.

3.  No campo **nome do arquivo:** , insira o nome do arquivo.

4.  Clique em **Save** (Salvar).

O arquivo de topologia será salvo como um arquivo XML.

## <a name="to-load-a-saved-topology"></a>Para carregar uma topologia salva

1.  No menu **Arquivo**, clique em **Abrir**.

    A caixa de diálogo **Abrir arquivo** é aberta.

2.  Navegue até a pasta onde o arquivo XML de topologia está localizado.

3.  Especifique o nome do arquivo no campo **nome do arquivo:** ou selecione o arquivo da lista.

4.  Clique em **Abrir**.

O arquivo de topologia é carregado no **painel topologia**.
