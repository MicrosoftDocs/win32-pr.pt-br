# <a name="mf_mt_video_lighting-attribute"></a>\_Atributo de \_ iluminação de vídeo MF MT \_

Especifica as condições de iluminação ideais para um tipo de mídia de vídeo.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

O valor desse atributo é um membro da enumeração [**MFVideoLighting**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideolighting) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
