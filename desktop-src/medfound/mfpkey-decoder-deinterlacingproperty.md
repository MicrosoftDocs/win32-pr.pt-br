# <a name="mfpkey_decoder_deinterlacing-property"></a>\_Propriedade de DESentrelaçamento do decodificador MFPKEY \_

Especifica se o decodificador decodifica quadros de vídeo entrelaçados do fluxo compactado como quadros progressivos.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMVCDecoderDeinterlacing

## <a name="data-type"></a>Tipo de Dados

**BOOL do VT \_**

## <a name="default-value"></a>Valor padrão

**VARIANTE \_ falso**
