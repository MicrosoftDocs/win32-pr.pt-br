# <a name="media-foundation-pipeline"></a>Pipeline de Media Foundation

A camada de *pipeline* no Microsoft Media Foundation é a camada da arquitetura que gera ou processa dados de mídia diretamente.

A maioria dos aplicativos não chama métodos diretamente nos componentes do pipeline, embora seja possível fazer isso. Leia estes tópicos se você estiver escrevendo um componente de pipeline personalizado.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                     | Descrição                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Fontes de mídia](media-sources.md)<br/>                             | As fontes de mídia geram dados de mídia, normalmente de um arquivo, dispositivo de captura ou fluxo de rede. <br/>                              |
| [Transformações de Media Foundation](media-foundation-transforms.md)<br/> | Media Foundation transformações (MFTs) processar dados de mídia. Por exemplo, os codecs no Media Foundation são implementados como MFTs. <br/>   |
| [Coletores de mídia](media-sinks.md)<br/>                                 | Os coletores de mídia consomem dados de mídia. Por exemplo, um coletor de mídia pode gravar os dados em um arquivo ou enviar os dados por uma rede.<br/> |
