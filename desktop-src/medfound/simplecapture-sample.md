# <a name="simplecapture-sample"></a>Exemplo de SimpleCapture

\[Preterido. A API MFPlay pode ser removida de versões futuras do Windows. Os aplicativos devem usar o [leitor de origem](source-reader.md) para captura de vídeo.\]

Mostra como visualizar o vídeo de um dispositivo de captura de vídeo usando a API MFPlay.

## <a name="apis-demonstrated"></a>APIs demonstradas

Este exemplo demonstra as seguintes interfaces de Microsoft Media Foundation:

-   [**IMFMediaSource**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**IMFPMediaItem**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaitem)
-   [**IMFPMediaPlayer**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayer)
-   [**IMFPMediaPlayerCallback**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayercallback)
