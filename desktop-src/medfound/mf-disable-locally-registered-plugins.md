# <a name="mf_disable_locally_registered_plugins-attribute"></a>MF \_ desabilitar \_ o \_ \_ atributo plugins registrados localmente

Especifica se plugins registrados localmente estão desabilitados.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Defina esse atributo como **true** para impedir que o carregador de topologia MF insira transformações registradas localmente.
