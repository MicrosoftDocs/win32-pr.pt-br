# <a name="transcode-attributes"></a>Atributos de transcodificação

Os atributos a seguir são usados com a API de transcodificação.



| Atributo                                                                          | Descrição                                                                                  |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [\_perfil de ajuste de transcodificação MF \_ \_](mf-transcode-adjust-profile.md)                  | Sinalizadores de perfil que definem as configurações de fluxo para a topologia de transcodificação.                    |
| [\_contêinertype de transcodificação MF \_](mf-transcode-containertype.md)                     | Especifica o tipo de contêiner de um arquivo codificado.                                             |
| [\_ \_ \_ codificador de inserção do MF transcodificar não cercamento \_](mf-transcode-donot-insert-encoder.md)     | Especifica se um codificador deve ser incluído na topologia de transcodificação.                     |
| [\_ENCODINGPROFILE de transcodificação MF \_](mf-transcode-encodingprofile.md)                 | Especifica o perfil de conformidade do dispositivo para codificar arquivos ASF (Advanced Streaming Format). |
| [\_QUALITYVSSPEED de transcodificação MF \_](mf-transcode-qualityvsspeed.md)                   | Especifica a compensação entre a qualidade de codificação e a velocidade de codificação.                          |
| [\_transferência de \_ \_ metadados ignorar a transcodificação MF \_](mf-transcode-skip-metadata-transfer.md) | Especifica se os metadados são gravados no arquivo transcodificado.                                |
| [**\_topologymode de transcodificação MF \_**](/windows/desktop/api/mfidl/ne-mfidl-mf_transcode_topologymode_flags)             | Especifica se o carregador de topologia carregará transformações baseadas em hardware.                   |
