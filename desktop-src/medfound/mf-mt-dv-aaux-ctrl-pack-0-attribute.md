# <a name="mf_mt_dv_aaux_ctrl_pack_0-attribute"></a>Atributo \_ \_ \_ \_ CTRL \_ PACK \_ 0 do MF MT DV

Pacote de controle do código-fonte auxiliar de áudio (A DIGIT) para o primeiro bloco de áudio em um tipo de mídia dv (vídeo digital).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse atributo corresponde ao membro **dwDVAVpCtl** da estrutura DirectShow [**DVINFO.**](/windows/win32/api/strmif/ns-strmif-dvinfo)

A constante GUID para esse atributo é exportada de mfuuid.lib.
