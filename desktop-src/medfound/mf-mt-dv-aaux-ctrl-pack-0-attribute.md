# <a name="mf_mt_dv_aaux_ctrl_pack_0-attribute"></a>\_Atributo MF MT \_ DV \_ AAUX \_ Ctrl \_ Pack \_ 0

Pacote de controle do código-fonte auxiliar de áudio (AAUX) para o primeiro bloco de áudio em um tipo de mídia de vídeo digital (DV).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse atributo corresponde ao membro **dwDVAAuxCtl** da estrutura [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) do DirectShow.

A constante de GUID para esse atributo é exportada de mfuuid. lib.
