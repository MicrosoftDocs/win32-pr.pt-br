# <a name="mfsampleextension_discontinuity-attribute"></a>\_Atributo de descontinuidade MFSampleExtension

Especifica se um exemplo de mídia é o primeiro exemplo após uma lacuna no fluxo.

## <a name="data-type"></a>Tipo de dados

**Bool** armazenado como **UINT32**

## <a name="getset"></a>Obter/definir

Para obter esse atributo, chame [**IMFAttributes:: GetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Para definir esse atributo, chame [**IMFAttributes:: setuint32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="applies-to"></a>Aplica-se a

[**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample)

## <a name="remarks"></a>Comentários

Esse atributo se aplica a exemplos de mídia. Se esse atributo for **true**, significa que houve uma descontinuidade no fluxo e esse exemplo é o primeiro a aparecer após a lacuna.

Se esse atributo não for definido, o valor padrão será **false**.

A constante de GUID para esse atributo é exportada de mfuuid. lib.
