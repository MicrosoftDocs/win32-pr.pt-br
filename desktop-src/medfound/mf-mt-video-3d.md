# <a name="mf_mt_video_3d-attribute"></a>\_ \_ Atributo 3D de vídeo MF MT \_

Especifica se um fluxo de vídeo contém conteúdo 3D.

## <a name="data-type"></a>Tipo de dados

**Bool** armazenado como **UINT32**

## <a name="remarks"></a>Comentários

Esse atributo se aplica a tipos de mídia de vídeo. Se esse atributo for **true**, o fluxo de vídeo conterá conteúdo 3D. O valor padrão é **false**.
