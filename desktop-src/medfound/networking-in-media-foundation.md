# <a name="networking-in-media-foundation"></a>Rede em Media Foundation

Aplicativos baseados em Media Foundation podem reproduzir fontes de mídia em uma rede. Para o aplicativo, o processo é semelhante à reprodução de um arquivo local.



| Tópico                                                                                      | Descrição                                                                                                                             |
|--------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [Recursos de origem da rede](network-source-features.md)                                     | Descreve os recursos com suporte pela fonte de rede e as opções de configuração associadas.                                        |
| [Suporte de proxy para fontes de rede](proxy-support-for-network-sources.md)                 | Fornece informações sobre como usar o objeto localizador de proxy que determina o proxy a ser usado ao se conectar a um servidor.                 |
| [Autenticação de origem da rede](network-source-authentication.md)                         | Fornece informações sobre como usar os componentes de autenticação, como o cache de credenciais e o Gerenciador de credenciais.                 |
| [Como obter eventos da fonte de rede](how-to-get-events-from-the-network-source.md) | Fornece informações sobre como obter Media Foundation eventos durante a abertura de uma conexão e antes da criação da fonte de rede. |
| [Protocolos com suporte](supported-protocols.md)                                             | Descreve os protocolos que são suportados pela fonte de rede. Além disso, fornece informações sobre o procedimento de substituição de protocolo.     |
| [Log de cliente](client-logging.md)                                                       | Descreve os campos de log do cliente que o aplicativo pode configurar e como ele pode recuperar estatísticas de rede.                      |
