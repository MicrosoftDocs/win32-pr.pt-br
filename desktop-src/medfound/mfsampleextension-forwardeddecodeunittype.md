# <a name="mfsampleextension_forwardeddecodeunittype-attribute"></a>\_Atributo MFSampleExtension ForwardedDecodeUnitType

Especifica o tipo, NALU ou SEI, de uma unidade anexada a um [**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample) em uma coleção [MFSampleExtension \_ ForwardedDecodeUnits](mfsampleextension-forwardeddecodeunits.md) .

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

O valor desse atributo é um valor da enumeração de [**\_ tipo de \_ \_ unidade de \_ decodificação personalizada MF**](mf-custom-decode-unit-type-enum.md) .
