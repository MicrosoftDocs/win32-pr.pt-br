# <a name="mfpkey_fullframerate-property"></a>\_Propriedade MFPKEY FULLFRAMERATE

Especifica se o endoder tem permissão para descartar quadros.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

**g \_ wszWMVCFullFrameRate**

## <a name="data-type"></a>Tipo de Dados

**\_I4 VT**

## <a name="remarks"></a>Comentários

Se você definir essa propriedade como 1, o codificador não removerá os quadros. Se você definir essa propriedade como 0, o codificador poderá descartar quadros.
