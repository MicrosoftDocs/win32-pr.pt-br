# <a name="mfpkey_resize_minapx-property"></a>MFPKEY \_ redimensionar \_ MINAPX Propriedade

Especifica a coordenada x do canto superior esquerdo da abertura de tela mínima.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

\_I4 VT

## <a name="applies-to"></a>Aplica-se A

-   [DSP de redimensionador de vídeo](videoresizer.md)

## <a name="remarks"></a>Comentários

O valor é um número real de ponto fixo. A parte inteira do número é armazenada nos 2 bytes mais altos e a parte fracionária é armazenada nos dois bytes inferiores.
