# <a name="overview-of-table-of-contents-parser"></a><span data-ttu-id="82f06-103">Visão geral do analisador de Sumário</span><span class="sxs-lookup"><span data-stu-id="82f06-103">Overview of Table of Contents Parser</span></span>

<span data-ttu-id="82f06-104">O analisador de Sumário (analisador de Sumário) é um componente do Windows que permite armazenar informações de índice avançadas em arquivos de mídia.</span><span class="sxs-lookup"><span data-stu-id="82f06-104">Table of Contents Parser (TOC Parser) is a Windows component that enables you to store advanced index information in media files.</span></span> <span data-ttu-id="82f06-105">Exemplos de informações de índice avançado são sumários, índices semânticos e quadros de história.</span><span class="sxs-lookup"><span data-stu-id="82f06-105">Examples of advanced index information are tables of contents, semantic indices, and story boards.</span></span>

<span data-ttu-id="82f06-106">Com o TOC Parser, você pode escrever software que permite aos usuários encontrar cenas de interesse em um arquivo de vídeo.</span><span class="sxs-lookup"><span data-stu-id="82f06-106">With TOC Parser, you can write software that enables users to find scenes of interest in a video file.</span></span> <span data-ttu-id="82f06-107">Por exemplo, você pode fornecer ao usuário um sumário para que ele possa identificar as seções principais do vídeo e, em seguida, exibir uma seção específica.</span><span class="sxs-lookup"><span data-stu-id="82f06-107">For example, you can provide the user with a table of contents so that he or she can identify the primary sections of the video and then view a particular section.</span></span> <span data-ttu-id="82f06-108">Você também pode fornecer ao usuário um índice para que ele possa encontrar uma lista de cenas relacionadas a uma determinada palavra ou frase.</span><span class="sxs-lookup"><span data-stu-id="82f06-108">You can also provide the user with an index so that he or she can find a list of scenes that are related to a particular word or phrase.</span></span>
