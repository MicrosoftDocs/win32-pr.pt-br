# <a name="mft_enum_adapter_luid-attribute"></a>\_ \_ Atributo LUID do adaptador de enumeração de MFT \_

Especifica o identificador exclusivo de um adaptador de vídeo. Use esse atributo ao chamar [**MFTEnum2**](/windows/desktop/api/mfapi/nf-mfapi-mftenum2) para enumerar MFTs associados a um adaptador específico.

## <a name="data-type"></a>Tipo de dados

**[**LUID**](/windows/win32/api/devicetopology/ns-devicetopology-luid)**
