# <a name="mft_enum_adapter_luid-attribute"></a>Atributo \_ LUID do ADAPTADOR DE ENUM \_ \_ MFT

Especifica o identificador exclusivo para um adaptador de vídeo. Use esse atributo ao chamar [**MFTEnum2**](/windows/desktop/api/mfapi/nf-mfapi-mftenum2) para enumerar MFTs associados a um adaptador específico.

## <a name="data-type"></a>Tipo de dados

**[**Luid**](/windows/win32/api/devicetopology/ns-devicetopology-luid)**
