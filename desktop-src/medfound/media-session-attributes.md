# <a name="media-session-attributes"></a>Atributos de sessão de mídia

Os atributos a seguir são usados para configurar a sessão de mídia.



| Atributo                                                                                            | Descrição                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_Gerenciador de \_ proteção de conteúdo de sessão MF \_ \_**](mf-session-content-protection-manager-attribute.md) | Fornece uma interface de retorno de chamada para que o aplicativo receba um objeto de habilitador de conteúdo da sessão do PMP. |
| [**\_ \_ tempo global da sessão MF \_**](mf-session-global-time-attribute.md)                                | Especifica se as topologias terão uma hora de início e de parada global.                                        |
| [**\_Gerenciador de \_ qualidade de sessão MF \_**](mf-session-quality-manager-attribute.md)                        | Contém o CLSID de um Gerenciador de qualidade para a sessão de mídia.                                              |
| [**\_modo de \_ \_ origem remoto da sessão \_ MF**](mf-session-remote-source-mode-attribute.md)                 | Especifica que a origem da mídia está sendo executada em um processo remoto.                                             |
| [**\_ \_ contexto do servidor de sessão MF \_**](mf-session-server-context-attribute.md)                          | Permite que duas instâncias da sessão de mídia compartilhem o mesmo processo de PMP.                                   |
| [**\_TOPOLOADER da sessão MF \_**](mf-session-topoloader-attribute.md)                                   | Contém o CLSID de um carregador de topologia para a sessão de mídia.                                              |
