# <a name="media-session-attributes"></a>Atributos de sessão de mídia

Os atributos a seguir são usados para configurar a Sessão de Mídia.



| Atributo                                                                                            | Descrição                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**GERENCIADOR DE PROTEÇÃO DE \_ \_ CONTEÚDO DA \_ SESSÃO \_ MF**](mf-session-content-protection-manager-attribute.md) | Fornece uma interface de retorno de chamada para o aplicativo receber um objeto do habilitador de conteúdo da sessão PMP. |
| [**HORA GLOBAL \_ DA SESSÃO \_ MF \_**](mf-session-global-time-attribute.md)                                | Especifica se as topologias terão uma hora de início e de parada global.                                        |
| [**MF \_ SESSION \_ QUALITY \_ MANAGER**](mf-session-quality-manager-attribute.md)                        | Contém o CLSID de um gerenciador de qualidade para a Sessão de Mídia.                                              |
| [**MODO DE \_ \_ ORIGEM REMOTA DA SESSÃO \_ MF \_**](mf-session-remote-source-mode-attribute.md)                 | Especifica que a fonte de mídia está em execução em um processo remoto.                                             |
| [**CONTEXTO DO SERVIDOR \_ DE \_ SESSÃO \_ MF**](mf-session-server-context-attribute.md)                          | Permite que duas instâncias da Sessão de Mídia compartilhem o mesmo processo PMP.                                   |
| [**TOPLOADER DE SESSÃO MF \_ \_**](mf-session-topoloader-attribute.md)                                   | Contém o CLSID de um carregador de topologia para a Sessão de Mídia.                                              |
