# <a name="mfpkey_mediaprotectionsystemidmapping-property"></a>\_Propriedade MFPKEY MediaProtectionSystemIdMapping

Especifica um [**IPropertySet**](/uwp/api/Windows.Foundation.Collections.IPropertySet?view=winrt-19041) que define o mapeamento de uma ID do sistema de propriedade para uma ID de ativação do sistema de propriedades.



Tipo de dados

Tipo PROPVARIANT (VT)

Membro PROPVARIANT

**IUNKNOWN**

VT \_ desconhecido

**punkVal**



## <a name="remarks"></a>Comentários

Essa chave de propriedade é usada para passar um [**IPropertySet**](/uwp/api/Windows.Foundation.Collections.IPropertySet?view=winrt-19041) que contém mapeamentos entre GUIDs do sistema de proteção e cadeias de caracteres de classe de tempo de execução para uma fonte de mídia.
