# <a name="stream-subtype-guids"></a>GUIDs de subtipo de fluxo

Os GUIDs de subtipo a seguir são definidos no arquivo de header mfapi.h. Para especificar o subtipo, de definido o [**atributo \_ MF MT \_ SUBTYPE**](mf-mt-subtype-attribute.md) no tipo de mídia.

Quando esses subtipos são usados, de definido o atributo [ \_ MF MT \_ MAJOR \_ TYPE](mf-mt-major-type-attribute.md) como **MFMediaType \_ Stream**.



| GUID                               | Descrição              |
|------------------------------------|--------------------------|
| **MFStreamFormat \_ MPEG2Program**   | Fluxo do programa MPEG-2    |
| **MFStreamFormat \_ MPEG2Transport** | Fluxo de transporte MPEG-2. |
