# <a name="topoedit-menus"></a>TopoEdit menus

A tabela a seguir descreve os menus do TopoEdit:



| Menu                               | Descrição                                                                    |
|------------------------------------|--------------------------------------------------------------------------------|
| [Menu arquivo](file-menu.md)         | Abre uma topologia nova ou existente.                                              |
| [Menu de topologia](topology-menu.md) | Adiciona nós de origem, transformação e saída. Resolve e exclui nós de topologia. |
| [Menu controles](controls-menu.md) | Reproduz, pausa e interrompe uma topologia resolvida.                                  |
| [Menu ferramentas](tools-menu.md)       | Registra as atividades de um nó de topologia selecionado.                                   |
| [Menu ajuda](help-menu.md)         | Fornece um link para os tópicos da ajuda do TopoEdit.                                   |
