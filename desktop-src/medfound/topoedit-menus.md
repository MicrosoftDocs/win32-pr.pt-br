# <a name="topoedit-menus"></a>Menus TopEdit

A tabela a seguir descreve os menus TopoEdit:



| Menu                               | Descrição                                                                    |
|------------------------------------|--------------------------------------------------------------------------------|
| [Menu Arquivo](file-menu.md)         | Abre uma topologia nova ou existente.                                              |
| [Menu topologia](topology-menu.md) | Adiciona nós de origem, transformação e saída. Resolve e exclui nós de topologia. |
| [Menu Controles](controls-menu.md) | Reproduz, pausa e interrompe uma topologia resolvida.                                  |
| [Menu Ferramentas](tools-menu.md)       | Registra atividades de um nó de topologia selecionado.                                   |
| [Menu Ajuda](help-menu.md)         | Fornece um link para os tópicos de ajuda do TopoEdit.                                   |
