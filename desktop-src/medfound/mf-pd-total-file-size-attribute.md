# <a name="mf_pd_total_file_size-attribute"></a>Atributo de tamanho de arquivo do MF \_ PD \_ total \_ \_

Especifica o tamanho total do arquivo de origem, em bytes. Este atributo se aplica aos descritores de apresentação. Uma fonte de mídia pode, opcionalmente, definir esse atributo.

## <a name="data-type"></a>Tipo de dados

**UINT64**

## <a name="remarks"></a>Comentários

A constante de GUID para esse atributo é exportada de mfuuid. lib.
