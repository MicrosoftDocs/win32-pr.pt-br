# <a name="melicenseacquisitioncompleted-event"></a>Evento MELicenseAcquisitionCompleted

Gerado quando a aquisição de licença é concluída. Para obter mais informações, consulte [MELicenseAcquisitionStart](melicenseacquisitionstart.md).

## <a name="event-values"></a>Valores de evento

Os valores possíveis recuperados de [**IMFMediaEvent:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) incluem o seguinte.



| VARTYPE              | Descrição                           |
|----------------------|---------------------------------------|
| VT \_ vazio<br/> | Nenhum dado do evento.<br/> <br/> |
