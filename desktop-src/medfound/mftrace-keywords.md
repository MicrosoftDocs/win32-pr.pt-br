# <a name="mftrace-keywords"></a>Palavras-chave MFTrace

Usando o [MFTrace](mftrace.md), você pode filtrar os resultados do rastreamento especificando uma lista de palavras-chave. Na linha de comando, use o argumento de linha de comando **-k** . Como alternativa, especifique o elemento de [**palavra-chave**](keyword.md) no arquivo de configuração. Para obter mais informações, consulte [usando MFTrace](using-mftrace.md).

O MFTrace dá suporte às seguintes palavras-chave. A maioria se refere a interfaces específicas ou exportações de biblioteca.

-   "All"
-   Os
-   Desvios
-   "IFilterGraph"
-   "IGraphBuilder"
-   "IMediaControl"
-   "IMediaObject"
-   "IMemInputPin"
-   "IMFActivate"
-   "IMFAttributes"
-   IMFByteStream
-   "IMFByteStreamHandler"
-   "IMFClock"
-   "IMFMediaEventGenerator"
-   "IMFMediaSession"
-   "IMFMediaSink"
-   "IMFMediaSource"
-   "IMFMediaStream"
-   "IMFPMediaItem"
-   "IMFPMediaPlayer"
-   "IMFPMediaPlayerCallback"
-   "IMFPresentationClock"
-   "IMFQualityAdvise"
-   "IMFQualityAdvise2"
-   "IMFQualityManager"
-   "IMFReadWriteClassFactory"
-   "IMFSample"
-   "IMFSchemeHandler"
-   "IMFSinkWriter"
-   IMFSourceReader
-   "IMFSourceReaderCallback"
-   "IMFSourceResolver"
-   "IMFStreamSink"
-   "IMFTopoLoader"
-   "IMFTopology"
-   "IMFTopologyNode"
-   "IMFTransform"
-   "IWMReader"
-   "Kernel32Export"
-   "MFExport"
-   "MFPlatExport"
-   "MFPlayExport"
-   "MFPublic"
-   "MFReadWriteExport"
-   "Ole32Export"
-   "wmvCoreExport"
