# <a name="mf_sensorstream_required_capabilities-attribute"></a>\_Atributo de \_ \_ funcionalidades necessárias do MF SENSORSTREAM

Esse atributo contém uma lista delimitada por ponto e vírgula de cadeias de caracteres de funcionalidade que especifica os recursos necessários para um fluxo específico. Para obter a lista de cadeias de caracteres de funcionalidade que podem ser incluídas nesse atributo, consulte [DeviceCapability](/uwp/schemas/appxpackage/appxmanifestschema/element-devicecapability).

## <a name="data-type"></a>Tipo de dados

**WCHAR\***
