# <a name="mf_stf_version_date-attribute"></a>\_Atributo de data MF STF \_ version \_

Esse atributo armazena um [FILETIME](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) que representa o carimbo de data/hora de uma fábrica de transformação de sensor.

## <a name="data-type"></a>Tipo de dados

**UINT64**
