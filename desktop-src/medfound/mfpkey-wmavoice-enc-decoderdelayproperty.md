# <a name="mfpkey_wmavoice_enc_decoderdelay-property"></a>\_Propriedade MFPKEY WMAVOICE \_ ENC \_ DecoderDelay

Especifica a latência do codificador em unidades de pacote.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

\_I4 VT

## <a name="remarks"></a>Comentários

Defina o valor dessa propriedade como 3, 4 ou 5. À medida que o valor aumenta de 3 para 5, a qualidade e o atraso também aumentam.
