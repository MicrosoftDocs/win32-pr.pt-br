# <a name="codecapi_avencadaptivemode-property"></a>\_Propriedade CODECAPI AVEncAdaptiveMode

Especifica o modo de codificação adaptável para um codificador de vídeo.

## <a name="data-type"></a>Tipo de dados

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncAdaptiveMode**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um bit a bit **ou** de valores da enumeração [**eAVEncAdaptiveMode**](/windows/win32/api/codecapi/ne-codecapi-eavencadaptivemode) .
