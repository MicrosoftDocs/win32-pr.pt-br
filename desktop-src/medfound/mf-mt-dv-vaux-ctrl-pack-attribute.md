# <a name="mf_mt_dv_vaux_ctrl_pack-attribute"></a>Atributo \_ CTRL PACK do MF MT \_ DV \_ VRL \_ \_ PACK

Pacote de controle do código-fonte auxiliar de vídeo (VVP) em um tipo de mídia DV (vídeo digital).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse atributo corresponde ao **membro dwDVVVpCtl** da estrutura DirectShow [**DVINFO.**](/windows/win32/api/strmif/ns-strmif-dvinfo)

A constante GUID para esse atributo é exportada de mfuuid.lib.
