# <a name="mf_mt_dv_vaux_ctrl_pack-attribute"></a>\_Atributo MF MT \_ DV \_ Vaux \_ Ctrl \_ Pack

Pacote de controle do código-fonte do VAUX (vídeo Auxiliary) em um tipo de mídia de vídeo digital (DV).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

esse atributo corresponde ao membro **dwDVVAuxCtl** da estrutura do DirectShow [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
