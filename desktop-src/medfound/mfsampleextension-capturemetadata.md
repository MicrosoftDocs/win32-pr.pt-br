# <a name="mfsampleextension_capturemetadata-attribute"></a>\_Atributo MFSampleExtension CaptureMetadata

O repositório [**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes) para todos os metadados relacionados ao pipeline de captura.

## <a name="data-type"></a>Tipo de dados

**[**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes)** armazenado como **IUnknown** 

## <a name="remarks"></a>Comentários

Esse atributo pode estar presente em qualquer [**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample).

Este atributo é somente leitura.
