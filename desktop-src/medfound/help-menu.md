# <a name="help-menu"></a>Menu Ajuda

O menu **ajuda** fornece os seguintes comandos:



| Comando      | Descrição                                                                      |
|--------------|----------------------------------------------------------------------------------|
| **Ajuda...**  | Abre a ajuda do TopoEdit no site do MSDN.                                         |
| **Sobre...** | Abre a caixa de diálogo **sobre** que exibe informações de versão sobre TopoEdit. |
