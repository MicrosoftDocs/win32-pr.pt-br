# <a name="transcode-sample"></a>Exemplo de transcódigo

Mostra como recodificar um arquivo de mídia para Windows mídia.

## <a name="apis-demonstrated"></a>APIs demonstradas

Este exemplo demonstra as APIs a seguir.

-   [**IMFMediaSession**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasession)
-   [**IMFMediaSource**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**IMFTopology**](/windows/desktop/api/mfidl/nn-mfidl-imftopology)
-   [**IMFTranscodeProfile**](/windows/desktop/api/mfidl/nn-mfidl-imftranscodeprofile)

## <a name="requirements"></a>Requisitos



| Produto                                                        | Versão   |
|----------------------------------------------------------------|-----------|
| [SDK do Windows](https://msdn.microsoft.com/windowsvista/bb980924.aspx) | Windows 7 |
