# <a name="mfpkey_droppedframes-property"></a>\_Propriedade MFPKEY DROPPEDFRAMES

Especifica o número de quadros de vídeo removidos durante a codificação.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

\_I4 VT

## <a name="remarks"></a>Comentários

Os quadros de vídeo são, às vezes, ignorados ou descartados durante a codificação devido a restrições de buffer.
