# <a name="codecapi_avencvideocontenttype-property"></a>\_Propriedade CODECAPI AVEncVideoContentType

Indica o tipo de conteúdo de vídeo para um codificador de vídeo.

## <a name="data-type"></a>Tipo de dados

**ULONG**

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoContentType**

## <a name="property-value"></a>Valor da propriedade

O valor dessa propriedade é um membro da enumeração [**eAVEncVideoContentType**](/windows/win32/api/codecapi/ne-codecapi-eavencvideocontenttype) .

## <a name="remarks"></a>Comentários

Essa propriedade é uma dica ao codificador para otimizar suas configurações de codificação.
