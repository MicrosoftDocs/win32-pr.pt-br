# <a name="introduction-to-topoedit"></a>Introdução ao TopoEdit

No Media Foundation, os dados de mídia são processados por meio da camada de pipeline e a sessão de mídia controla o fluxo de dados e fornece controles de reprodução. Os dados em cada componente de pipeline — origem, transformação e coletor – são representados como um *nó de topologia*. O *objeto Topology* é uma coleção de nós de topologia e representa o caminho inteiro do fluxo de dados por meio do pipeline de sessão de mídia.

Para obter informações sobre topologias, consulte [about topologias](about-topologies.md).

TopoEdit é uma ferramenta Visual para compilar e testar topologias. Com o TopoEdit, você pode executar as seguintes tarefas:

-   Crie uma topologia adicionando vários nós de topologia, como origem, transformação e nós de saída.

-   Conexão e resolver a topologia.

-   Teste a funcionalidade da topologia por meio da reprodução.
