# <a name="media-foundation-primitives"></a>Media Foundation primitivos

Media Foundation define vários tipos de objeto básicos que são usados em todas as APIs Media Foundation dados.



| Tópico                                       | Descrição                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [Atributos](attributes-and-properties.md) | Atributos e propriedades são pares chave/valor armazenados em um objeto .                  |
| [Tipos de mídia](media-types.md)              | Um tipo de mídia descreve o formato de um fluxo de mídia digital.                        |
| [Buffers de mídia](media-buffers.md)          | Um buffer de mídia gerencia um bloco de memória para que ele possa ser compartilhado entre objetos. |
| [Exemplos de mídia](media-samples.md)          | Um exemplo de mídia é um objeto que contém uma lista de buffers de mídia.                  |
