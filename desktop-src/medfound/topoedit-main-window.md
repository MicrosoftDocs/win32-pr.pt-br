# <a name="topoedit-main-window"></a>Janela principal do TopoEdit

Depois de iniciar o TopoEdit, a janela mostrada na captura de tela a seguir é exibida.

![captura de tela mostrando a janela principal do topoedit](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

Os elementos da janela principal do TopoEdit são:

-   **Painel topologia**. Aparece à esquerda da janela. Essa área é usada para adicionar nós de topologia, exibir uma topologia e conectar os nós de topologia. Para obter informações sobre como usar esse painel, consulte [criando topologias usando o TopoEdit](building-topologies-by-using-topoedit.md).
-   **Painel atributos**. Aparece à direita da janela. Esse painel exibe os atributos do nó de topologia selecionado no momento. Para obter informações sobre os atributos mostrados neste painel, consulte [exibindo informações de topologia](viewing-topology-information.md).
-   Completos. Consulte os [menus do TopoEdit](topoedit-menus.md).
-   Barra. Consulte a [barra de ferramentas TopoEdit](topoedit-toolbar.md).
