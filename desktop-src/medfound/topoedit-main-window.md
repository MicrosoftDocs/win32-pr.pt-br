# <a name="topoedit-main-window"></a>Janela Principal do TopEdit

Depois de iniciar TopEdit, a janela mostrada na captura de tela a seguir será exibida.

![captura de tela mostrando a janela principal topedit](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

Os elementos da janela principal TopoEdit são:

-   **Painel de Topologia**. Aparece à esquerda da janela. Essa área é usada para adicionar nós de topologia, exibir uma topologia e conectar os nós de topologia. Para obter informações sobre como usar esse painel, consulte [Criando topologias usando TopEdit](building-topologies-by-using-topoedit.md).
-   **Painel Atributos**. Aparece à direita da janela. Esse painel exibe os atributos do nó de topologia selecionado no momento. Para obter informações sobre os atributos mostrados nesse painel, consulte [Exibindo informações de topologia](viewing-topology-information.md).
-   Menus. Consulte [TopEdit Menus](topoedit-menus.md).
-   Toolbar. Consulte [TopEdit Toolbar](topoedit-toolbar.md).
