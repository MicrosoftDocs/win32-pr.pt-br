# <a name="mf_local_plugin_control_policy-attribute"></a>Atributo MF \_ LOCAL \_ PLUGIN CONTROL \_ \_ POLICY

Especifica uma política de controle de plug-in local.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

De definir esse atributo como um dos valores [**de POLÍTICA de CONTROLE DE \_ \_ PLUG-IN \_ do MF.**](/windows/desktop/api/mfobjects/ne-mfobjects-mf_plugin_control_policy)

Esses atributos permitem que o aplicativo especifique uma política local mais restritiva do que a política em todo o processo configurada pelo [**IMFPluginControl.**](/windows/desktop/api/mfobjects/nn-mfobjects-imfplugincontrol)
