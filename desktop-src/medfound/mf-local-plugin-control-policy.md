# <a name="mf_local_plugin_control_policy-attribute"></a>\_Atributo de \_ política de controle de plug-in local MF \_ \_

Especifica uma política de controle de plug-in local.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Defina esse atributo como um dos valores [**da \_ \_ \_ política de controle do plug-in MF**](/windows/desktop/api/mfobjects/ne-mfobjects-mf_plugin_control_policy) .

Esses atributos permitem que o aplicativo especifique uma política local mais restritiva do que a política de todo o processo configurada pelo [**IMFPluginControl**](/windows/desktop/api/mfobjects/nn-mfobjects-imfplugincontrol).
