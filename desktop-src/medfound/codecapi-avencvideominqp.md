# <a name="codecapi_avencvideominqp-property"></a>\_Propriedade CODECAPI AVEncVideoMinQP

Especifica o parâmetro de quantificação mínimo (QP) para codificação de vídeo.

## <a name="data-type"></a>Tipo de dados

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncVideoMinQP**

## <a name="property-value"></a>Valor da propriedade

O valor é o máximo de QP que é usado na codificação de variável-QP.
