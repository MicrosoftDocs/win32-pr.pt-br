# <a name="mf_mt_audio_num_channels-attribute"></a>\_Atributo de \_ canais de número de áudio MF MT \_ \_

Número de canais de áudio em um tipo de mídia de áudio.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse atributo corresponde ao membro **nChannels** da estrutura [WAVEFORMATEX](mf-mt-audio-prefer-waveformatex-attribute.md) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
