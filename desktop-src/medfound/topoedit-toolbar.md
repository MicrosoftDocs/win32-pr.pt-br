# <a name="topoedit-toolbar"></a>Barra de ferramentas TopoEdit

Por padrão, a barra de ferramentas é exibida na parte superior da janela do aplicativo e abaixo da barra de menus. A barra de ferramentas fornece acesso rápido ao mouse aos itens do [Menu controles](controls-menu.md) . Os botões da barra de ferramentas são habilitados somente depois que uma topologia é resolvida com êxito.

A captura de tela a seguir mostra a barra de ferramentas TopoEdit.

![captura de tela mostrando a barra de ferramentas topoedit](images/410257c5-ae5c-454c-a9a8-ba05fc5235b5.jpg)
