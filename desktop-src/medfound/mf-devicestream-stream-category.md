# <a name="mf_devicestream_stream_category-attribute"></a>\_Atributo de \_ categoria de fluxo DEVICESTREAM MF \_

Representa a categoria do fluxo.

## <a name="data-type"></a>Tipo de dados

**GUID**

## <a name="remarks"></a>Comentários

Esse atributo representa o GUID que identifica a categoria de fluxo definida em ksmedia. h.
