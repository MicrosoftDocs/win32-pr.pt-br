# <a name="codecapi_avench264cabacenable-property"></a>\_Propriedade CODECAPI AVEncH264CABACEnable

Habilita ou desabilita CABAC (codificação aritmética binária de contexto) para codificação de entropia H. 264.

## <a name="data-type"></a>Tipo de dados

**Variante \_ BOOL** (VT \_ bool)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncH264CABACEnable**

## <a name="property-value"></a>Valor da propriedade

Se o valor for **Variant \_ true**, CABAC será habilitado.
