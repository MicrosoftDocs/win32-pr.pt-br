# <a name="mf_mt_dv_aaux_ctrl_pack_1-attribute"></a>\_Atributo MF MT \_ DV \_ AAUX \_ Ctrl \_ Pack \_ 1

Pacote de controle do código-fonte auxiliar de áudio (AAUX) para o segundo bloco de áudio em um tipo de mídia de vídeo digital (DV).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

esse atributo corresponde ao membro **dwDVAAuxCtl1** da estrutura do DirectShow [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
