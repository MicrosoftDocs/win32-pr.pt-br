# <a name="microsoft-media-foundation"></a>Microsoft Media Foundation

O Microsoft Media Foundation permite o desenvolvimento de aplicativos e componentes para usar mídia digital no Windows Vista e versões posteriores.

Media Foundation é a próxima geração de plataforma multimídia para Windows que permite que desenvolvedores, consumidores e provedores de conteúdo adotem a nova onda de conteúdo premium com robustez avançada, qualidade incomparável e interoperabilidade direta.
