# <a name="microsoft-media-foundation"></a>Microsoft Media Foundation

Microsoft Media Foundation permite o desenvolvimento de aplicativos e componentes para usar mídia digital no Windows Vista e posterior.

Media Foundation é a plataforma multimídia de última geração para Windows que permite aos desenvolvedores, consumidores e provedores de conteúdo adotar a nova onda de conteúdo premium com robustez aprimorada, qualidade incomparável e interoperabilidade perfeita.
