# <a name="mf_mt_timestamp_can_be_dts-attribute"></a>MF \_ MT \_ TIMESTAMP \_ PODE SER atributo \_ \_ DTS

Especifica se um decodificador pode usar DTS (carimbos de data/hora de decodificar) ao definir carimbos de data/hora.

## <a name="data-type"></a>Tipo de dados

**BOOL** armazenado como **UINT32**
