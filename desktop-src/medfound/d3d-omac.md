# <a name="d3d_omac-structure"></a>\_Estrutura OMAC do D3D

Contém um Message Authentication Code (MAC).

## <a name="syntax"></a>Sintaxe


```C++
typedef struct _D3D_OMAC {
  BYTE Omac[D3D_OMAC_SIZE];
} D3D_OMAC;
```



## <a name="members"></a>Membros

<dl> <dt>

**Omac**
</dt> <dd>

Uma matriz de bytes que contém o valor de MAC criptográfico da mensagem.

</dd> </dl>
