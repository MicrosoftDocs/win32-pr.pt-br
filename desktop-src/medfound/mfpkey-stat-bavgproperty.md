# <a name="mfpkey_stat_bavg-property"></a>\_Propriedade MFPKEY stat \_ BAVG

Especifica a janela de buffer médio, em milissegundos, de um fluxo codificado. Somente leitura.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

**g \_ wszWMVCBAvg**

## <a name="data-type"></a>Tipo de Dados

**\_I4 VT**

## <a name="remarks"></a>Comentários

Para determinar a janela de buffer médio de um fluxo codificado, leia esta propriedade no final da codificação.
