# <a name="mfpkey_stat_rmax-property"></a>\_Propriedade MFPKEY stat \_ RMAX

Especifica a taxa de bits máxima, em bits por segundo, de um fluxo codificado. Somente leitura.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

**g \_ wszWMVCMaxBitrate**

## <a name="data-type"></a>Tipo de Dados

**\_I4 VT**

## <a name="remarks"></a>Comentários

Para determinar a taxa máxima de bits de um fluxo codificado, leia essa propriedade no final da codificação.
