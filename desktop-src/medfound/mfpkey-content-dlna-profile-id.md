# <a name="mfpkey_content_dlna_profile_id-property"></a>\_ \_ \_ Propriedade ID do perfil \_ DLNA do conteúdo do MFPKEY

Contém o identificador de perfil da rede de vida digital (DLNA) para uma origem de mídia.



Tipo de dados

Tipo PROPVARIANT (VT)

Membro PROPVARIANT

**WCHAR \_ t \** _

LPWStr do VT \_

_ *pwszVal**



## <a name="remarks"></a>Comentários

Um identificador de perfil DLNA é uma cadeia de caracteres que identifica o perfil de codificação do conteúdo.

Os identificadores de perfil são descritos nas [diretrizes de interoperabilidade de dispositivo em rede DLNA](https://www.dlna.org/about/organization). (Esse recurso pode não estar disponível em alguns idiomas e países.)
