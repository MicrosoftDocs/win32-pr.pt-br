# <a name="mfpkey_conv_inputframerate-property"></a>Propriedade MFPKEY \_ CONV \_ INPUTFRAMERATE

Especifica a taxa de quadros de entrada como uma taxa.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível somente usando [**IPropertyStore.**](/windows/win32/api/propsys/nn-propsys-ipropertystore)

## <a name="data-type"></a>Tipo de Dados

VT \_ UI8

## <a name="applies-to"></a>Aplica-se A

-   [Conversor de taxa de quadros](framerateconverter.md)

## <a name="remarks"></a>Comentários

Armazene o numerador nos 4 bytes superiores e o denominador nos 4 bytes inferiores.
