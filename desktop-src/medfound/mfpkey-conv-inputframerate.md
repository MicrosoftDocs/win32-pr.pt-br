# <a name="mfpkey_conv_inputframerate-property"></a>\_Propriedade MFPKEY CONV \_ INPUTFRAMERATE

Especifica a taxa de quadros de entrada como uma taxa.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

\_UI8 VT

## <a name="applies-to"></a>Aplica-se A

-   [Conversor de taxa de quadros](framerateconverter.md)

## <a name="remarks"></a>Comentários

Armazene o numerador em mais de 4 bytes e o denominador nos 4 bytes inferiores.
