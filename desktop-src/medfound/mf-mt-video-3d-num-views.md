# <a name="mf_mt_video_3d_num_views-attribute"></a>\_Atributo MF MT \_ Video \_ 3D \_ num \_ views

O número de exibições em uma sequência de vídeo 3D.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Uma sequência de vídeo 3D típica contém duas exibições, à esquerda e à direita. Atualmente, o valor desse atributo deve ser 1 ou 2.
