# <a name="mfpkey_closedentrypoint-property"></a>\_Propriedade MFPKEY CLOSEDENTRYPOINT

Especifica o padrão de codificação para o codificador a ser usado no início de um grupo de imagens.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

**g \_ wszWMVCClosedEntryPoint**

## <a name="data-type"></a>Tipo de Dados

**BOOL do VT \_**

## <a name="default-value"></a>Valor padrão

**VARIANTE \_ falso**

## <a name="remarks"></a>Comentários

Se você definir essa propriedade como **Variant \_ true**, o padrão de codificação será BBIBBP. Se você definir essa propriedade como **Variant \_ false**, o padrão de codificação será IBBPBBP.
