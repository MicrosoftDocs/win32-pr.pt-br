# <a name="mf_mt_mpeg2_standard-attribute"></a>\_ \_ Atributo padrão MF MT MPEG2 \_

Para um tipo de mídia que descreve um fluxo de programa MPEG-2 (PS) ou um fluxo de transporte (TS), especifica o padrão usado para multiplexar o fluxo.

## <a name="data-type"></a>Tipo de dados

**UINT32**



| Valor                                                                                                | Significado                                                                    |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------|
| <span id="0"></span><dl> <dt>**0**</dt> </dl> | Standard MPEG-2 PS ou TS.<br/>                                       |
| <span id="1"></span><dl> <dt>**1**</dt> </dl> | Padrão ATSC (Comitê de sistemas de televisão avançado).<br/>          |
| <span id="2"></span><dl> <dt>**2**</dt> </dl> | Padrão de transmissão de vídeo digital (DVB).<br/>                      |
| <span id="3"></span><dl> <dt>**3**</dt> </dl> | Associação de ARIB (setores de rádio e empresas) padrão.<br/> |
