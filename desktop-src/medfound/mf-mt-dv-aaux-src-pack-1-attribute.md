# <a name="mf_mt_dv_aaux_src_pack_1-attribute"></a>Atributo \_ MF MT \_ DV \_ A CDC \_ SRC PACK \_ \_ 1

Pacote de origem auxiliar de áudio (A DIGIT) para o segundo bloco de áudio em um tipo de mídia dv (vídeo digital).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse atributo corresponde ao membro **dwDVARiaSrc1** da estrutura DirectShow [**DVINFO.**](/windows/win32/api/strmif/ns-strmif-dvinfo)

A constante GUID para esse atributo é exportada de mfuuid.lib.
