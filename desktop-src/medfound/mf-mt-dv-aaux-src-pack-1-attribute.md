# <a name="mf_mt_dv_aaux_src_pack_1-attribute"></a>\_Atributo MF MT \_ DV \_ AAUX \_ src \_ Pack \_ 1

Pacote de origem do AAUX (audio auxiliar) para o segundo bloco de áudio em um tipo de mídia de vídeo digital (DV).

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse atributo corresponde ao membro **dwDVAAuxSrc1** da estrutura [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) do DirectShow.

A constante de GUID para esse atributo é exportada de mfuuid. lib.
