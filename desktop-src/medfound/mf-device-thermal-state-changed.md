# <a name="mf_device_thermal_state_changed-attribute"></a>\_Atributo de \_ estado térmico do dispositivo MF \_ \_ alterado

Representa um evento que sinaliza uma alteração de estado térmico no dispositivo.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

O valor desse atributo especifica o valor térmico do dispositivo em porcentagem.
