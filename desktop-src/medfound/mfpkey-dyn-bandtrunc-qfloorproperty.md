# <a name="mfpkey_dyn_bandtrunc_qfloor-property"></a>\_Propriedade MFPKEY dyn \_ BANDTRUNC \_ QFLOOR

Especifica a qualidade na qual a largura de banda máxima codificada é permitida. Em qualidade melhor que ou igual ao valor dessa propriedade, a largura de banda de código máxima é permitida.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

**VT \_ R4**
