# <a name="mf_mt_forward_custom_sei-attribute"></a>\_ \_ Atributo sei de encaminhamento personalizado MF MT \_ \_

Especifica que os tipos de unidade de informações de aprimoramento complementar (SEI) devem ser encaminhados em amostras de saída pelo decodificador.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Se o decodificador analisar um SEI, ele não será encaminhado.
