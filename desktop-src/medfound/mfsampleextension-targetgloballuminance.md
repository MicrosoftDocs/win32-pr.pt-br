# <a name="mfsampleextension_targetgloballuminance-attribute"></a>Atributo MFSampleExtension \_ TargetGlobalLuminance

O valor em Nits que especifica a luminância de luz de fundo global direcionada para o quadro de vídeo associado.

## <a name="data-type"></a>Tipo de dados

**UINT32**
