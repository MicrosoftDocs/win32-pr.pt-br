# <a name="mfsampleextension_targetgloballuminance-attribute"></a>\_Atributo MFSampleExtension TargetGlobalLuminance

O valor em nits que especifica a luminância de luz global de destino para o quadro de vídeo associado.

## <a name="data-type"></a>Tipo de dados

**UINT32**
