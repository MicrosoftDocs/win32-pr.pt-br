# <a name="mf_mt_max_luminance_level-attribute"></a>Atributo de nível de luminescência do MF \_ MT \_ Max \_ \_

Especifica o nível máximo de luminância do conteúdo em nits. Esse atributo tem a mesma semântica que **MaxCLL** , conforme definido no padrão CEA-861,3.

## <a name="data-type"></a>Tipo de dados

**UINT32**
