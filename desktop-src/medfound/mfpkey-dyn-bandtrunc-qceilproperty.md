# <a name="mfpkey_dyn_bandtrunc_qceil-property"></a>\_Propriedade MFPKEY dyn \_ BANDTRUNC \_ QCEIL

Especifica a qualidade na qual a largura de banda mínima codificada é permitida. Em qualidade pior ou igual ao valor dessa propriedade, a largura de banda de código mínima é permitida.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

**VT \_ R4**
