# <a name="mf_capture_metadata_digitalwindow-attribute"></a>\_ \_ Atributo DIGITALWINDOW de metadados de captura MF \_

Um valor que representa as configurações de limites da janela digital para captura de vídeo.

## <a name="data-type"></a>Tipo de dados

**BLOB**

## <a name="remarks"></a>Comentários

O valor de **blob** dessa propriedade é uma estrutura [DigitalWindowSetting](/windows/win32/api/mfapi/ns-mfapi-digitalwindowsetting) .
