# <a name="codecapi_avencnumworkerthreads-property"></a>\_Propriedade CODECAPI AVEncNumWorkerThreads

Define o número de threads de trabalho usados por um codificador de vídeo.

## <a name="data-type"></a>Tipo de dados

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>GUID da propriedade

**CODECAPI \_ AVEncNumWorkerThreads**

## <a name="remarks"></a>Comentários

Se o valor for zero, o codificador selecionará o número de threads.
