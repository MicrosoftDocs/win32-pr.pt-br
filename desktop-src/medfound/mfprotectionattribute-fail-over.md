# <a name="mfprotectionattribute_fail_over-attribute"></a>\_Atributo de failover MFPROTECTIONATTRIBUTE \_

Indica se a proteção faz failover para isso se o melhor esforço falhar. Esse atributo pode ser usado com objetos [**IMFOutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) .

## <a name="data-type"></a>Tipo de dados

**Bool** armazenado como **UINT32**

## <a name="remarks"></a>Comentários

Esse atributo pode ser usado com objetos [**IMFOutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) .
