# <a name="mfprotectionattribute_fail_over-attribute"></a>Atributo FAIL OVER MFPROTECTIONATTRIBUTE \_ \_

Indica se a proteção falhará para isso se o melhor esforço falhar. Esse atributo pode ser usado com [**objetos IMFOutputSchema.**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema)

## <a name="data-type"></a>Tipo de dados

**BOOL** armazenado como **UINT32**

## <a name="remarks"></a>Comentários

Esse atributo pode ser usado com [**objetos IMFOutputSchema.**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema)
