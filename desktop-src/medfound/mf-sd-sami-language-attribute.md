# <a name="mf_sd_sami_language-attribute"></a>\_Atributo de \_ idioma de Sami SD MF \_

Contém o nome do idioma de intercâmbio de mídia acessível (SAMI) que é definido para o fluxo.

Esse atributo está presente nos descritores de fluxo retornados da fonte de mídia SAMI.

## <a name="data-type"></a>Tipo de dados

Cadeia de caracteres largos

## <a name="remarks"></a>Comentários

O nome do idioma SAMI é especificado no arquivo SAMI.

A constante de GUID para esse atributo é exportada de mfuuid. lib.
