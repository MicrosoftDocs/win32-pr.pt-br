# <a name="mf_policy_id-attribute"></a>Atributo de ID da \_ política MF \_

Um identificador que pode ser definido em um [IMFOutputPolicy](/windows/win32/api/mfidl/nn-mfidl-imfoutputpolicy).

## <a name="data-type"></a>Tipo de dados

**UINT32** (Tratado como **bool**)

## <a name="remarks"></a>Comentários

O valor pode ser recebido do evento de mídia [MEPolicySet](./mepolicyset.md) . Ele é armazenado como uma carga de tipo **VT_UI4** no evento **MEPolicySet** .
