# <a name="content-protection-commands"></a>Comandos de Proteção de Conteúdo

Lista os comandos para o método [**IDirect3DAuthenticatedChannel9:: Configure**](/windows/desktop/api/d3d9/nf-d3d9-idirect3dauthenticatedchannel9-configure) .



| Comando                                                                                                           | Descrição                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**D3DAUTHENTICATEDCONFIGURE \_ inicializar**](d3dauthenticatedconfigure-initialize.md)                             | Inicializa o canal autenticado.                                                                                |
| [**D3DAUTHENTICATEDCONFIGURE \_ CRYPTOSESSION**](d3dauthenticatedconfigure-cryptosession.md)                       | Associa uma sessão criptográfica a um dispositivo decodificador do DirectX Video Acceleration 2 (DXVA-2) e a um dispositivo Direct3D. |
| [**D3DAUTHENTICATEDCONFIGURE \_ ENCRYPTIONWHENACCESSIBLE**](d3dauthenticatedconfigure-encryptionwhenaccessible.md) | Define o nível de criptografia que é executado antes que o conteúdo protegido se torne acessível à CPU ou ao barramento.         |
| [**Proteção do D3DAUTHENTICATEDCONFIGURE \_**](d3dauthenticatedconfigure-protection.md)                             | Habilita ou desabilita a proteção para o dispositivo.                                                                        |
| [**D3DAUTHENTICATEDCONFIGURE \_ SHAREDRESOURCE**](d3dauthenticatedconfigure-sharedresource.md)                     | Permite que um processo Abra um recurso compartilhado ou desabilite um processo de abrir recursos compartilhados.                     |
