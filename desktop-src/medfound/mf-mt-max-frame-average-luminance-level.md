# <a name="mf_mt_max_frame_average_luminance_level-attribute"></a>\_Atributo de \_ \_ \_ nível médio de \_ luminância \_ MF MT Max frame

Especifica a média máxima de nível de luminância por quadro do conteúdo em nits. Esse atributo tem a mesma semântica que **MaxFALL** , conforme definido no padrão CEA-861,3.

## <a name="data-type"></a>Tipo de dados

**UINT32**
