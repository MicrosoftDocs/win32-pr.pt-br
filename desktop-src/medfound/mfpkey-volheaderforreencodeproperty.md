# <a name="mfpkey_volheaderforreencode-property"></a>\_Propriedade MFPKEY VOLHEADERFORREENCODE

Para a recodificação de segmento, especifica os dados privados do codec do arquivo que está sendo codificado novamente.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Disponível apenas usando [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de Dados

**\_blob VT**
