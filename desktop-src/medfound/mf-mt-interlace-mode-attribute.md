# <a name="mf_mt_interlace_mode-attribute"></a>\_Atributo do \_ modo de entrelaçamento MF MT \_

Descreve como os quadros em um tipo de mídia de vídeo são entrelaçados.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

O valor desse atributo é um membro da enumeração [**MFVideoInterlaceMode**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideointerlacemode) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
