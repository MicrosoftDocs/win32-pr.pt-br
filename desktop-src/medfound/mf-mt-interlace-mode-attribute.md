# <a name="mf_mt_interlace_mode-attribute"></a>Atributo \_ MF MT \_ INTER MULTIPLAYER \_ MODE

Descreve como os quadros em um tipo de mídia de vídeo são entrelaçados.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

O valor desse atributo é um membro da [**enumeração MFVideoIntermodMode.**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideointerlacemode)

A constante GUID para esse atributo é exportada de mfuuid.lib.
