# <a name="mf_mt_audio_avg_bytes_per_second-attribute"></a>\_Atributo de \_ média \_ de \_ bytes \_ por \_ segundo do áudio MF MT

Número médio de bytes por segundo em um tipo de mídia de áudio.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Esse atributo corresponde ao membro **nAvgBytesPerSec** da estrutura [**WAVEFORMATEX**](/previous-versions/dd757713(v=vs.85)) .

A constante de GUID para esse atributo é exportada de mfuuid. lib.
