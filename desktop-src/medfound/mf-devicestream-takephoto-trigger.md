# <a name="mf_devicestream_takephoto_trigger-attribute"></a>\_Atributo de gatilho MF DEVICESTREAM \_ TAKEPHOTO \_

Especifica se o disparador tirar foto é encapsulado na origem do dispositivo.

## <a name="data-type"></a>Tipo de dados

**UINT32**

## <a name="remarks"></a>Comentários

Defina esse atributo como diferente de zero para encapsular o gatilho tirar foto na origem do dispositivo.
