# <a name="mft_enum_video_renderer_extension_profile-attribute"></a>\_Atributo de \_ \_ perfil de extensão de processador de vídeo MFT \_ enum \_

Contém uma lista de todas as entradas **VideoRendererExtensionProfile** na marca **VideoRendererExtensionProfiles** de um arquivo de manifesto do aplicativo UWP. O [**MFTEnumEx**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex) armazena isso no repositório de atributos do objeto [**IMFActivate**](/windows/desktop/api/mfobjects/nn-mfobjects-imfactivate) que **MFTEnumEx** cria para MFTs que têm um manifesto UWP associado contendo a marca **VideoRendererExtensionProfiles** .
