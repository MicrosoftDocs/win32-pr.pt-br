# <a name="controls-menu"></a>Menu controles

O menu **controles** fornece comandos de controle de transporte para uma topologia que é definida na sessão de mídia. Esse menu é habilitado somente depois que uma topologia é resolvida com êxito. Para acessar esses comandos rapidamente, você pode usar a [barra de ferramentas TopoEdit](topoedit-toolbar.md).

O menu **controles** fornece os seguintes comandos:



| Comando   | Descrição                                                              |
|-----------|--------------------------------------------------------------------------|
| **Reproduzir**  | Inicia a reprodução da última topologia resolvida.                           |
| **Parar**  | Interrompe a reprodução da última topologia resolvida que está sendo executada no momento.  |
| **Pausar** | Pausa a reprodução da última topologia resolvida que está sendo executada no momento. |
