# <a name="mfnetsource_statistics-property"></a>\_Propriedade de estatísticas MFNETSOURCE

Contém estatísticas de rede para a origem da rede.

## <a name="data-type"></a>Tipo de Dados

Consoante consulte comentários.

## <a name="remarks"></a>Comentários

A constante MFNETSOURCE \_ Statistics define um GUID que é usado em conjunto com a enumeração de [**\_ \_ IDs de estatísticas MFNETSOURCE**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids) para definir um conjunto de chaves de propriedade. Cada chave de propriedade no conjunto tem **fmtid** igual a MFNETSOURCE \_ estatísticas e **pid** igual a um membro da enumeração. Para obter mais informações, [**consulte \_ \_ IDs de estatísticas do MFNETSOURCE**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids).
