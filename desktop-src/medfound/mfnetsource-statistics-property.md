# <a name="mfnetsource_statistics-property"></a>Propriedade STATISTICS MFNETSOURCE \_

Contém estatísticas de rede para a origem da rede.

## <a name="data-type"></a>Tipo de Dados

Varia; consulte Comentários.

## <a name="remarks"></a>Comentários

A constante MFNETSOURCE STATISTICS define um GUID que é usado em conjunto com \_ a enumeração [**MFNETSOURCE \_ STATISTICS \_ IDS**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids) para definir um conjunto de chaves de propriedade. Cada chave de propriedade no conjunto **tem fmtid** igual a MFNETSOURCE STATISTICS e \_ **pid** igual a um membro da enumeração. Para obter mais informações, [**consulte MFNETSOURCE \_ STATISTICS \_ IDS**](/windows/desktop/api/mfidl/ne-mfidl-mfnetsource_statistics_ids).
