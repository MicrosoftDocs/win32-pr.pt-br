# <a name="mfstreamextension_pinholecameraintrinsics-attribute"></a>\_Atributo MFStreamExtension PinholeCameraIntrinsics

Contém os intrínsecos da câmera pinhole para o fluxo.

## <a name="data-type"></a>Tipo de dados

Matriz de bytes

## <a name="getset"></a>Obter/definir

Para obter esse atributo, chame [**IMFMediaSourceEx:: Getstreamattributes**](/windows/desktop/api/mfidl/nf-mfidl-imfmediasourceex-getstreamattributes).

## <a name="remarks"></a>Comentários

O valor do atributo é um [**MFPinholeCameraIntrinsics**](/windows/desktop/api/mfapi/ns-mfapi-mfpinholecameraintrinsics).
