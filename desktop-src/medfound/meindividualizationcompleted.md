# <a name="meindividualizationcompleted-event"></a>Evento MEIndividualizationCompleted

Gerado pelo mecanismo de política quando a individualização é concluída. Para obter mais informações, consulte evento [MEIndividualizationStart](meindividualizationstart.md) .

## <a name="event-values"></a>Valores de evento

Os valores possíveis recuperados de [**IMFMediaEvent:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) incluem o seguinte.



| VARTYPE              | Descrição                           |
|----------------------|---------------------------------------|
| VT \_ vazio<br/> | Nenhum dado do evento.<br/> <br/> |
