# <a name="imftimedtexttrack-data-format-guids"></a>GUIDs de formato de dados IMFTimedTextTrack

As constantes a seguir identificam o formato de dados subjacente de um [**IMFTimedTextTrack**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imftimedtexttrack). Obtenha o formato de uma faixa de texto com tempo chamando [**IMFTimedTextTrack:: GetDataFormat**](/windows/desktop/api/mfmediaengine/nf-mfmediaengine-imftimedtexttrack-getdataformat).

<dl> <dt>

<span id="MFSubtitleFormat_TTML"></span><span id="mfsubtitleformat_ttml"></span><span id="MFSUBTITLEFORMAT_TTML"></span>**MFSubtitleFormat \_ TTML**
</dt> <dd> <dl> <dt>



A faixa usa o formato de linguagem de marcação de texto Timed (TTML).


</dt> </dl> </dd> <dt>

<span id="MFSubtitleFormat_ATSC"></span><span id="mfsubtitleformat_atsc"></span><span id="MFSUBTITLEFORMAT_ATSC"></span>**\_ATSC MFSubtitleFormat**
</dt> <dd> <dl> <dt>



A faixa usa o formato ATSC (Advanced televisão System Committee).


</dt> </dl> </dd> <dt>

<span id="MFSubtitleFormat_ATSC"></span><span id="mfsubtitleformat_atsc"></span><span id="MFSUBTITLEFORMAT_ATSC"></span>**\_ATSC MFSubtitleFormat**
</dt> <dd> <dl> <dt>



A faixa usa o formato de controle de texto de vídeo da Web (WebVTT).


</dt> </dl> </dd> </dl>
