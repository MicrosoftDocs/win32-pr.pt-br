# <a name="mf_capture_engine_enable_camera_streamstate_notification-attribute"></a>ATRIBUTO DE NOTIFICAÇÃO DE \_ \_ \_ \_ STREAMSTATE DO MECANISMO \_ \_ DE CAPTURA DE MF

Indica se as notificações de estado de fluxo devem ser habilitadas.

## <a name="data-type"></a>Tipo de dados

## <a name="remarks"></a>Comentários

Um valor não zero especifica que as notificações devem ser habilitadas. Se o atributo não estiver presente ou se o valor do atributo for 0, as notificações não serão habilitadas.
