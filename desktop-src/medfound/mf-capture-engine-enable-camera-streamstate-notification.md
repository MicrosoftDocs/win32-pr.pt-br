# <a name="mf_capture_engine_enable_camera_streamstate_notification-attribute"></a>\_Mecanismo de captura MF \_ \_ habilitar atributo de \_ \_ notificação do streamstate de câmera \_

Indica se as notificações de estado do fluxo devem ser habilitadas.

## <a name="data-type"></a>Tipo de dados

## <a name="remarks"></a>Comentários

Um valor diferente de zero Especifica que as notificações devem ser habilitadas. Se o atributo não estiver presente ou se o valor do atributo for 0, as notificações não serão habilitadas.
