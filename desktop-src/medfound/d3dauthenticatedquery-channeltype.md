# <a name="d3dauthenticatedquery_channeltype"></a>\_Canaltype D3DAUTHENTICATEDQUERY

Retorna o tipo de canal autenticado.



| Requisito | Valor |
|-------------|--------------------------------------------------------------------------------------------------------------|
| GUID de consulta  | **\_Canaltype D3DAUTHENTICATEDQUERY**                                                                       |
| Dados de entrada  | [**\_Entrada de consulta D3DAUTHENTICATEDCHANNEL \_**](d3dauthenticatedchannel-query-input.md)                         |
| Retornar dados | [**\_Saída D3DAUTHENTICATEDCHANNEL QUERYCHANNELTYPE \_**](d3dauthenticatedchannel-querychanneltype-output.md) |



 

## <a name="remarks"></a>Comentários

Esta consulta é válida para todos os tipos de canal.
