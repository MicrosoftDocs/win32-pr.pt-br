# <a name="mfpkey_stat_ravg-property"></a>\_Propriedade MFPKEY stat \_ RAVG

Especifica a taxa média de bits, em bits por segundo, de um fluxo codificado. Somente leitura.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

**g \_ wszWMVCAvgBitrate**

## <a name="data-type"></a>Tipo de Dados

**\_I4 VT**

## <a name="remarks"></a>Comentários

Para determinar a taxa média de bits de um fluxo codificado, leia essa propriedade no final da codificação.
