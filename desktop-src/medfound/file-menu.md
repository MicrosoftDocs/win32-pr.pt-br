# <a name="file-menu"></a>Menu Arquivo

O menu **arquivo** fornece os seguintes comandos:



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Comando</th>
<th>Descrição</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Novo...</strong></td>
<td>Limpa a topologia existente e remove os nós de topologia do <strong>painel topologia</strong>.<br/>
<ul>
<li>Se uma topologia estiver aberta no momento, o solicitará que o usuário salve a topologia. Para obter informações sobre como salvar a topologia, consulte <a href="saving-and-loading-topologies.md">salvando e carregando topologias</a>.<br/></li>
<li>Cria uma topologia vazia. O usuário pode adicionar nós de topologia à nova topologia. Para obter mais informações, consulte <a href="building-topologies-by-using-topoedit.md">criando topologias usando TopoEdit</a>.<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>Renderizar arquivo...</strong></td>
<td>Cria uma topologia de reprodução para um arquivo de mídia. O TopoEdit cria e conecta automaticamente os nós na topologia. Para obter mais informações, consulte <a href="building-a-playback-topology-with-topoedit.md">criando uma topologia de reprodução com TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Renderizar transcodificação...</strong></td>
<td>Cria uma topologia de transcodificação. Para obter mais informações, consulte <a href="building-a-transcode-topology-with-topoedit.md">criando uma topologia de transcodificação com TopoEdit</a>.<br/>
<blockquote>
[!Note]<br />
Esse recurso requer o Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Abrir...</strong></td>
<td>Carrega uma topologia existente salva como um arquivo XML. Consulte <a href="saving-and-loading-topologies.md">salvando e carregando topologias</a>.</td>
</tr>
<tr class="odd">
<td><strong>Salvar...</strong></td>
<td>Salva a topologia aberta no momento como um arquivo XML. Consulte <a href="saving-and-loading-topologies.md">salvando e carregando topologias</a>.</td>
</tr>
<tr class="even">
<td><strong>Sair</strong></td>
<td>Fecha o aplicativo TopoEdit.</td>
</tr>
</tbody>
</table>
