# <a name="file-menu"></a>Menu Arquivo

O  menu Arquivo fornece os seguintes comandos:




| Comando | Descrição | 
|---------|-------------|
| <strong>Novo...</strong> | Limpa a topologia existente e remove os nós de topologia do <strong>Painel de Topologia</strong>.<br /><ul><li>Se uma topologia estiver aberta no momento, solicitará que o usuário salve a topologia. Para obter informações sobre como salvar a topologia, consulte <a href="saving-and-loading-topologies.md">Salvando e carregando topologias</a>.<br /></li><li>Cria uma topologia vazia. O usuário pode adicionar nós de topologia à nova topologia. Para obter mais informações, consulte <a href="building-topologies-by-using-topoedit.md">Criando topologias usando TopEdit</a>.<br /></li></ul> | 
| <strong>Renderizar Arquivo...</strong> | Cria uma topologia de reprodução para um arquivo de mídia. TopEdit cria e conecta automaticamente os nós na topologia. Para obter mais informações, consulte <a href="building-a-playback-topology-with-topoedit.md">Criando uma topologia de reprodução com TopEdit</a>. | 
| <strong>Renderizar Transcode...</strong> | Cria uma topologia de transcodificação. Para obter mais informações, consulte <a href="building-a-transcode-topology-with-topoedit.md">Criando uma topologia transcódigo com TopEdit</a>.<br /><blockquote>[!Note]<br />Esse recurso requer Windows 7.</blockquote><br /> | 
| <strong>Aberto...</strong> | Carrega uma topologia existente salva como um arquivo XML. Consulte <a href="saving-and-loading-topologies.md">Salvando e carregando topologias</a>. | 
| <strong>Salvar...</strong> | Salva a topologia aberta no momento como um arquivo XML. Consulte <a href="saving-and-loading-topologies.md">Salvando e carregando topologias</a>. | 
| <strong>Sair</strong> | Fecha o aplicativo TopoEdit. |
