# <a name="dds"></a>DDS

O formato de arquivo de superfície do DirectDraw (. DDS) foi introduzido com o DirectX 7 para armazenar texturas descompactadas e compactadas (DXTn). O formato de arquivo dá suporte a mipmaps, mapas de cubo e mapas de volume. O formato de arquivo DDS tem suporte de [DirectXTex](https://github.com/Microsoft/DirectXTex), [DirectXTK](https://github.com/Microsoft/DirectXTK), D3DX herdado e outras ferramentas do DirectX. A partir do Direct3D 10, os arquivos DDS dão suporte a matrizes de textura.

> [!Note]  
> A biblioteca de utilitários D3DX (D3DX 9, D3DX 10 e [D3DX 11](/windows/desktop/direct3d11/d3d11-graphics-reference-d3dx11)) foi preterida para o Windows 8 e não tem suporte para aplicativos da Windows Store. Recomendamos que você faça uso de [DirectXTex](https://github.com/Microsoft/DirectXTex), [DirectXTK](https://github.com/Microsoft/DirectXTK)ou ambos.
