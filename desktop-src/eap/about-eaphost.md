# <a name="learn-about-the-eap-and-eaphost-relationship"></a>Saiba mais sobre o relacionamento de EAP e EAPHost

Este tópico descreve a relação entre o EAP (Extensible Authentication Protocol) e o host do protocolo de autenticação extensível. Para o novo desenvolvimento de método EAP, consulte [host de protocolo de autenticação extensível](../eaphost/portal.md).

## <a name="eaphost-framework"></a>Estrutura EAPHost

O EAPHost é considerado uma estrutura para EAP por dois motivos.

-   O EAPHost é um componente de infraestrutura do Windows que hospeda os plug-ins do EAP
-   O EAPHost implementa o protocolo EAP de acordo com a [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063). Uma autenticação EAP consiste no protocolo EAP de acordo com a [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063)e os aspectos específicos do método EAP do protocolo.
