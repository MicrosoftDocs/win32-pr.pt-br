# <a name="configuration-user-interface"></a>Configuração Interface do Usuário

Interfaces do usuário de configuração para protocolos de autenticação são implementadas de maneira diferente, dependendo se a interface do usuário é para configurar o protocolo de autenticação no cliente ou no servidor. Os tópicos a seguir descrevem o processo usado para implementar uma interface do usuário de configuração para o cliente e para o servidor:

-   [Configuração do lado do servidor Interface do Usuário](server-side-configuration-user-interface.md)
-   [Configuração do lado do cliente Interface do Usuário](client-side-configuration-user-interface.md)
