# <a name="configuration-user-interface"></a>Interface do usuário de configuração

As interfaces do usuário de configuração (UI) para protocolos de autenticação são implementadas de forma diferente, dependendo se a interface do usuário é para configurar o protocolo de autenticação no cliente ou no servidor. Os tópicos a seguir descrevem o processo usado para implementar uma interface do usuário de configuração para o cliente do e para o servidor do:

-   [Interface do usuário de configuração do lado do servidor](server-side-configuration-user-interface.md)
-   [Interface do usuário de configuração do lado do cliente](client-side-configuration-user-interface.md)
