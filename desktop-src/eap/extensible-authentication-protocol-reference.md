# <a name="extensible-authentication-protocol-reference"></a>Referência de protocolo de autenticação extensível

Esta seção descreve os elementos de referência usados para implementar o Protocolo de Autenticação Extensível (EAP). Entre esses elementos de referência estão funções usadas para implementar protocolos de autenticação, provedores de autenticação e provedores de contabilidade. Esta seção também inclui as estruturas e tipos enumerados que essas funções usam.

Esta documentação descreve os tópicos a seguir.

-   [Funções EAP](eap-functions.md)
-   [EAP Interfaces](eap-interfaces.md)
-   [Estruturas de EAP](eap-structures.md)
