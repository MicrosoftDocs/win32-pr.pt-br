# <a name="extensible-authentication-protocol-reference"></a>Referência de protocolo de autenticação extensível

Esta seção descreve os elementos de referência que são usados para implementar o protocolo EAP (Extensible Authentication Protocol). Entre esses elementos de referência estão as funções que são usadas para implementar protocolos de autenticação, provedores de autenticação e provedores de contabilidade. Esta seção também inclui as estruturas e os tipos enumerados que essas funções usam.

Esta documentação descreve os tópicos a seguir.

-   [Enumerações EAP](eap-enumerations.md)
-   [Funções EAP](eap-functions.md)
-   [Interfaces EAP](eap-interfaces.md)
-   [Estruturas EAP](eap-structures.md)
