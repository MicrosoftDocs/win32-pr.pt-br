# <a name="role-based-access-control"></a>Controle de acesso baseado em função

A API do Gerenciador de autorização e o snap-in do MMC fornecem aplicativos com uma estrutura de controle de acesso baseada em função. A API do Gerenciador de autorização, também conhecida como AzMan, fornece um modelo de desenvolvimento simplificado no qual gerenciar grupos flexíveis e regras de negócio e armazenar políticas de autorização. Para obter mais informações, consulte estes tópicos:

-   [Vantagens da autorização baseada em funções](advantages-of-role-based-authorization.md)
-   [Modelo do Gerenciador de autorização](authorization-manager-model.md)

**Windows XP:** a API do gerenciador de autorização e o snap-in do MMC estão disponíveis como um download (pacote de ferramentas de administração do Windows Server 2003) do [https://www.microsoft.com/downloads](https://www.microsoft.com/Downloads/) . Este download dá suporte à administração e ao desenvolvimento de políticas do Gerenciador de autorização; Ele não dá suporte à implantação de aplicativos que usam o Gerenciador de autorização para controlar o acesso.
