# <a name="permissions-property-page"></a>Página de propriedades de permissões

A página de propriedades **permissões** contém controles para a edição avançada das [*ACEs*](/windows/desktop/SecGloss/a-gly) na [*DACL*](/windows/desktop/SecGloss/d-gly)de um objeto. Esta página permite que o usuário exiba e edite todas as informações na DACL, incluindo informações que não estão disponíveis na página de [Propriedades de segurança básica](basic-security-property-page.md). As informações avançadas incluem [ACEs específicas de objeto](object-specific-aces.md) e informações sobre [herança de Ace](ace-inheritance.md).

A página de propriedades **permissões** sempre é incluída na [folha de propriedades segurança avançada](advanced-security-property-sheet.md) exibida quando o usuário clica no botão **avançado** na página de propriedades de segurança básica.
