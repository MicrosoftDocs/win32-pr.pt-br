# <a name="advantages-of-role-based-authorization"></a>Vantagens da autorização baseada em funções

A API do Gerenciador de autorização fornece um modelo de autorização que tem várias vantagens em relação à autorização de nível baixo (baseada em ACL):

-   Simplifica a administração do controle de acesso.
-   disponibiliza a autorização para scripts e aplicativos desenvolvidos usando o sistema de desenvolvimento Microsoft Visual Basic.
-   Fornece um mecanismo para aplicar a lógica de negócios de tempo de execução ao verificar o acesso.
