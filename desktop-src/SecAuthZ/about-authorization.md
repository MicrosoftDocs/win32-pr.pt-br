# <a name="about-authorization"></a>Sobre a autorização

As seções a seguir fornecem informações sobre autorização.



| Seção                                                                                             | Descrição                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Controle de acesso](access-control.md)<br/>                                                     | Recursos de segurança que controlam quem pode acessar recursos no sistema operacional. Os aplicativos chamam funções de controle de acesso para definir quem pode acessar recursos específicos ou controlar o acesso aos recursos fornecidos pelo aplicativo.<br/>     |
| [Editor de controle de acesso](access-control-editor.md)<br/>                                       | Crie e Use folhas de propriedades e páginas de propriedades que permitam ao usuário exibir e modificar os componentes do [*descritor de segurança*](/windows/desktop/SecGloss/s-gly)de um objeto.<br/> |
| [Controle de acesso de cliente/servidor](client-server-access-control.md)<br/>                         | Aplicativos de servidor que fornecem serviços aos clientes.<br/>                                                                                                                                                                                |
| [Controle de acesso para recursos do aplicativo](access-control-for-application-resources.md)<br/> | Controle de acesso baseado em função e ACL para recursos do aplicativo.<br/>                                                                                                                                                                   |
| [Controle de Integridade de Credenciais](mandatory-integrity-control.md)<br/>                           | Controle de acesso no nível do sistema para objetos protegíveis.<br/>                                                                                                                                                                                   |
| [Controle de conta de usuário](user-account-control.md)<br/>                                         | Recurso de segurança que permite aos usuários executar tarefas comuns como não administradores, chamados usuários padrão e como administradores sem precisar alternar usuários, fazer logoff ou usar **Executar como**.<br/>                                           |
