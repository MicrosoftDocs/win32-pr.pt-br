# <a name="using-authorization-in-script"></a>Usando a autorização no script

Os exemplos nesta seção são escritos usando Visual Basic Scripting Edition (VBScript) e páginas de Active Server (ASP).

Para obter informações sobre outras tarefas de autorização, consulte [tarefas de suporte no script](supporting-tasks-for-authorization-in-script.md).
