# <a name="access-control-for-application-resources"></a>Controle de acesso para recursos do aplicativo

A API do Gerenciador de autorização e o snap-in do MMC disponíveis no Windows Server 2003 fornecem aplicativos com uma estrutura de controle de acesso baseada em função flexível. Essa estrutura inclui um conjunto de objetos COM removidas para aplicativos Web e de linha de negócios.

As seções a seguir fornecem informações sobre como controlar o acesso aos recursos do aplicativo:

-   [Controle de acesso baseado em função](role-based-access-control.md)
-   [Controle de acesso baseado em ACL](acl-based-access-control.md)
