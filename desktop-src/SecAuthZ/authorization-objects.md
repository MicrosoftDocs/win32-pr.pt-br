# <a name="authorization-objects"></a>Objetos de autorização

Os objetos a seguir são usados com aplicativos de autorização.

ActiveX O instalador fornece o seguinte objeto.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                               | Descrição                                                                                                                        |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**CIeAxiInstallerService**](cieaxiinstallerservice.md)<br/> | Implementa as interfaces [**IAxiService**](ieaxiservice.md) e [**IeAxiServiceCallback**](ieaxiservicecallback.md) .<br/> |



 


O Gerenciador de autorização fornece os seguintes objetos.



| Objeto                                               | Descrição                                                                          |
|------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**AzAuthorizationStore**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore) | Define o contêiner que é a raiz do repositório de política de autorização.<br/> |
| [**AzBizRuleContext**](/windows/desktop/api/Azroles/nn-azroles-iazbizrulecontext)         | Contém informações sobre uma operação de regra de negócio (BizRule).<br/>           |
