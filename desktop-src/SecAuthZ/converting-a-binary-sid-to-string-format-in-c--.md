# <a name="converting-a-binary-sid-to-string-format-in-c"></a>Convertendo um SID binário em formato de cadeia de caracteres em C++

As funções [**ConvertSidToStringSid**](/windows/desktop/api/Sddl/nf-sddl-convertsidtostringsida) e [**ConvertStringSidToSid**](/windows/desktop/api/Sddl/nf-sddl-convertstringsidtosida) convertem um SID ( [*identificador de segurança*](/windows/desktop/SecGloss/s-gly) ) de e para o formato de cadeia de caracteres. Para obter uma descrição do formato de cadeia de caracteres de SID, consulte [componentes de Sid](sid-components.md).
