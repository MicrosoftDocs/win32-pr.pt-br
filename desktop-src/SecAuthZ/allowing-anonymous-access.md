# <a name="allowing-anonymous-access"></a>Permitindo acesso anônimo

A política de segurança padrão restringe o acesso local anônimo para não ter nenhum direito. Os administradores podem adicionar ou subtrair os direitos conforme eles veem.

Existe um grupo de acesso local para aplicativos com os mesmos direitos de acesso que todos. os administradores podem, então, aumentar ou diminuir adequadamente o número de usuários nesse grupo, chamado de **Windows grupo de acesso compatível com 2000**.

Para obter mais informações, consulte as páginas de referência para as funções listadas na tabela do tópico [funções de grupo local](/windows/desktop/NetMgmt/local-group-functions) .
