# <a name="primary-group-of-a-new-object"></a>Grupo primário de um novo objeto

O grupo primário de um novo objeto é o grupo primário do [*descritor de*](/windows/desktop/SecGloss/s-gly) segurança especificado pelo criador do objeto. Se o criador de um objeto não especificar um grupo primário, o grupo primário do objeto será o grupo primário padrão do token primário ou de [*representação do criador.*](/windows/desktop/SecGloss/i-gly)
