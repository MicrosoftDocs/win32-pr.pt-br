# <a name="collections-authorization"></a>Coleções (autorização)

A API do Gerenciador de autorização fornece interfaces que representam coleções de outros objetos. Por exemplo, a interface [**IAzRoles**](/windows/desktop/api/Azroles/nn-azroles-iazroles) representa uma coleção de objetos [**IAzRole**](/windows/desktop/api/Azroles/nn-azroles-iazrole) .

As outras interfaces que representam as coleções são [**IAzApplications**](/windows/desktop/api/Azroles/nn-azroles-iazapplications), [**IAzApplicationGroups**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroups), [**IAzOperations**](/windows/desktop/api/Azroles/nn-azroles-iazoperations), [**IAzScopes**](/windows/desktop/api/Azroles/nn-azroles-iazscopes)e [**IAzTasks**](/windows/desktop/api/Azroles/nn-azroles-iaztasks). Cada uma dessas interfaces fornece propriedades que facilitam a enumeração de objetos em uma coleção.
