# <a name="matrixtransformvectors-methods"></a>Métodos Matrix. TransformVectors

Este tópico lista os métodos TransformVectors da classe [**Matrix**](/windows/win32/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) . Para obter uma lista completa de métodos para a classe **Matrix** , consulte [métodos de matriz](-gdiplus-class-matrix-methods.md).

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                                                 | Descrição                                                                                                                                                                                                                                                                                                                                                                      |
|:-------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TransformVectors (ponto \* , int)**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformvectors(inoutpoint_inint))   | O método [**Matrix:: TransformVectors**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-transformvectors(inoutpoint_inint)) multiplica cada vetor em uma matriz por essa matriz. Os elementos de translação da matriz (terceira linha) são ignorados. Cada vetor é tratado como uma matriz de linhas. A multiplicação é executada com a matriz de linhas à esquerda e essa matriz à direita.<br/>  |
| [**TransformVectors (PointF \* , int)**](/previous-versions//ms535319(v=vs.85)) | O método [**Matrix:: TransformVectors**](/previous-versions//ms535319(v=vs.85)) multiplica cada vetor em uma matriz por essa matriz. Os elementos de translação da matriz (terceira linha) são ignorados. Cada vetor é tratado como uma matriz de linhas. A multiplicação é executada com a matriz de linhas à esquerda e essa matriz à direita.<br/> |
