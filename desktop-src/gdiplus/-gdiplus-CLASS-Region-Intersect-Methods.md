# <a name="regionintersect-methods"></a>Métodos Region. Intersect

Este tópico lista os métodos de interseção da classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obter uma lista completa de métodos para a classe **Region** , consulte [Region Methods](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                     | Descrição                                                                                                                                                                                     |
|:---------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Intersect (Rect&)**](/previous-versions//ms534804(v=vs.85))     | O método [**Region:: Intersect**](/previous-versions//ms534804(v=vs.85)) atualiza essa região para a parte de si mesma que cruza o interior do retângulo especificado.<br/>  |
| [**Intersect (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_))   | O método [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_)) atualiza essa região para a parte de si mesma que cruza o interior do retângulo especificado.<br/> |
| [**Intersect (região \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion))     | O método [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion)) atualiza essa região para a parte de si mesma que cruza outra região.<br/>                         |
| [**Intersect (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) | O método [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) atualiza essa região para a parte de si mesma que cruza o interior do caminho especificado.<br/>            |
