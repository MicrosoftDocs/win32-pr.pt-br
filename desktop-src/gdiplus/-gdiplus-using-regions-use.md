# <a name="using-regions-gdi"></a>Usando regiões (GDI+)

a classe de [**região**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-region) Windows GDI+ permite que você defina uma forma personalizada. A forma pode ser composta de linhas, polígonos e curvas.

Dois usos comuns para regiões são teste de clique e recorte. O teste de clique está determinando se o mouse foi clicado em uma determinada região da tela. O recorte está restringindo o desenho a uma determinada região.

Os tópicos a seguir abrangem regiões com mais detalhes:

-   [Teste de clique com uma região](-gdiplus-hit-testing-with-a-region-use.md)
-   [Recorte com uma região](-gdiplus-clipping-with-a-region-use.md)
