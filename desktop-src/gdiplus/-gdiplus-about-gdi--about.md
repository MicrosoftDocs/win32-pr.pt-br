# <a name="about-gdi"></a>Sobre GDI+

Windows GDI+ é a parte do sistema operacional Windows XP ou do sistema operacional Windows Server 2003 que fornece gráficos vetoriais bidimensionais, imagens e tipografia. GDI+ melhora o Windows Graphics Device Interface (GDI) (a interface do dispositivo gráfico incluída nas versões anteriores do Windows) adicionando novos recursos e otimizando os recursos existentes.
