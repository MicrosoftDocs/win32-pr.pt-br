# <a name="about-gdi"></a>Sobre o GDI+

O Windows GDI+ é a parte do sistema operacional Windows XP ou do Windows Server 2003 que fornece gráficos vetoriais bidimensionais, imagens e tipografia. O GDI+ melhora o Windows Graphics Device Interface (GDI) (a interface gráfica do dispositivo incluída nas versões anteriores do Windows) adicionando novos recursos e otimizando os recursos existentes.
