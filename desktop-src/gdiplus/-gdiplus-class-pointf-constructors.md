# <a name="pointfpointf-constructors"></a>Construtores PointF. PointF

Este tópico lista os construtores da classe [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) . Para obter uma listagem de classe completa, consulte **classe PointF**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                    | Descrição                                                                                                                                                                       |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PointF (SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstsizef_))   | Cria um objeto [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) usando um objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) para especificar os membros de dados **X** e **Y** .<br/> |
| [**PointF (PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstpointf_)) | Cria um novo objeto [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) e copia os dados de outro objeto **PointF** .<br/>                                                 |
| [**PointF (REAL, REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inreal_inreal)) | Cria um objeto [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) usando dois números reais para especificar os membros de dados **X** e **Y** .<br/>                                     |
| [**PointF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf)              | Cria um objeto [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) e inicializa os membros de dados **X** e **Y** como zero. Esse é o construtor padrão.<br/>              |
