# <a name="coordinate-systems-and-transformations"></a>Sistemas de coordenadas e transformações

O Windows GDI+ fornece uma transformação do mundo e uma transformação de página para que você possa transformar (girar, dimensionar, traduzir e assim por diante) os itens desenhados. As duas transformações também permitem que você trabalhe em uma variedade de sistemas de coordenadas.

-   [Tipos de sistemas de coordenadas](-gdiplus-types-of-coordinate-systems-about.md)
-   [Representação matricial de transformações](-gdiplus-matrix-representation-of-transformations-about.md)
-   [Transformações globais e locais](-gdiplus-global-and-local-transformations-about.md)
