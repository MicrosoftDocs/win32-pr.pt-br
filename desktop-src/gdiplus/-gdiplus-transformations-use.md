# <a name="transformations"></a>Transformações

As transformações afim incluem rotação, dimensionamento, reflexão, distorção e tradução. No Windows GDI+, a classe [**Matrix**](/windows/desktop/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) fornece a base para executar transformações afim em desenhos vetoriais, imagens e texto.

Os tópicos a seguir abordam as transformações em mais detalhes:

-   [Usando a transformação global](-gdiplus-using-the-world-transformation-use.md)
-   [Por que a ordem das transformações é importante](-gdiplus-why-transformation-order-is-significant-use.md)
