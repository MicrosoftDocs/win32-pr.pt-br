# <a name="rectrect-constructors"></a>Construtores Rect. Rect

Este tópico lista os construtores da classe [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) . Para obter uma listagem de classe completa, consulte **Rect Class**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                                 | Descrição                                                                                                                                                                                                                                                                                        |
|:----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Rect (ponto&, tamanho&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect(inconstpoint__inconstsize_))       | Cria um objeto [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) usando um objeto [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) para inicializar os membros de dados **X** e **Y** e um objeto de [**tamanho**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) para inicializar os membros de dados de **largura** e **altura** .<br/> |
| [**Rect (INT, INT, INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect(inint_inint_inint_inint)) | Cria um objeto [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) usando quatro inteiros para inicializar os membros de **dados X**, **Y**, **Width** e **Height** .<br/>                                                                                                                               |
| [**Rect ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rect-rect)                                 | Cria um objeto [**Rect**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rect) cuja coordenada x, coordenada y, largura e altura são todos zero. Esse é o construtor padrão. <br/>                                                                                                                          |
