# <a name="colorcolor-constructors"></a>Construtores Color.Color

Este tópico lista os construtores da [**classe Color.**](/windows/win32/api/gdipluscolor/nl-gdipluscolor-color) Para uma listagem de classe completa, consulte **Classe color**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                               | Descrição                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Color(ARGB)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb))                   | Cria um [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb)) usando um **valor ARGB.**<br/>                                                                                                    |
| [**Color(BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte))        | Cria um [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte)) usando valores especificados para os componentes vermelho, verde e azul. Esse construtor define o componente alfa como 255 (opaco).<br/> |
| [**Color(BYTE, BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) | Cria um [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) usando valores especificados para os componentes alfa, vermelho, verde e azul.<br/>                                                   |
| [**Color()**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color)                            | Cria um [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color) e o inicializa para preto opaco. Esse é o construtor padrão.<br/>                                                                |
