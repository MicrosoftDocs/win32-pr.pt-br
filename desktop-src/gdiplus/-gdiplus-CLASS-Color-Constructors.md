# <a name="colorcolor-constructors"></a>Construtores de cor. cor

Este tópico lista os construtores da classe [**Color**](/windows/win32/api/gdipluscolor/nl-gdipluscolor-color) . Para obter uma listagem de classe completa, consulte **classe Color**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                               | Descrição                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Cor (ARGB)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb))                   | Cria um objeto [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb)) usando um valor **ARGB** .<br/>                                                                                                    |
| [**Cor (BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte))        | Cria um objeto [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte)) usando valores especificados para os componentes vermelho, verde e azul. Esse construtor define o componente alfa como 255 (opaco).<br/> |
| [**Cor (BYTE, BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) | Cria um objeto [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) usando valores especificados para os componentes alfa, vermelho, verde e azul.<br/>                                                   |
| [**Cor ()**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color)                            | Cria um objeto [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color) e o inicializa em preto opaco. Esse é o construtor padrão.<br/>                                                                |
