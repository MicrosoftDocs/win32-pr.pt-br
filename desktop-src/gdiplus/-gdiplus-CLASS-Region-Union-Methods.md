# <a name="regionunion-methods"></a>Métodos Region. Union

Este tópico lista os métodos Union da classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obter uma lista completa de métodos para a classe **Region** , consulte [Region Methods](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                             | Descrição                                                                                                                                                                                                                     |
|:-------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Union (Rect&)**](/previous-versions//ms534789(v=vs.85))     | O método [**Region:: Union**](/previous-versions//ms534789(v=vs.85)) atualiza essa região para todas as partes (interseção e não interseção) de si mesma e todas as partes do interior do retângulo especificado.<br/>  |
| [**Union (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstrectf_))   | O método [**Region:: Union**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstrectf_)) atualiza essa região para todas as partes (interseção e não interseção) de si mesma e todas as partes do interior do retângulo especificado.<br/> |
| [**Union (região \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstregion))     | O método [**Region:: Union**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstregion)) atualiza essa região para todas as partes (interseção e não interseção) de si mesma e de todas as partes de outra região.<br/>                         |
| [**Union (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstgraphicspath)) | O método [**Region:: Union**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-union(inconstgraphicspath)) atualiza essa região para todas as partes (interseção e não interseção) de si mesma e todas as partes do interior do caminho especificado.<br/>            |
