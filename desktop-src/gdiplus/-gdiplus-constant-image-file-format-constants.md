# <a name="image-file-format-constants"></a>Constantes de formato de arquivo de imagem

O método [**Image:: GetRawFormat**](/windows/desktop/api/Gdiplusheaders/nf-gdiplusheaders-image-getrawformat) retorna um GUID que indica o formato de arquivo de uma imagem. As constantes a seguir, definidas em Gdiplusimaging. h, representam os GUIDs que identificam esses formatos de arquivo.



| Constante                                                                                                                                                                                                                 | Descrição                                                               |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------|
| <span id="ImageFormatBMP"></span><span id="imageformatbmp"></span><span id="IMAGEFORMATBMP"></span><dl> <dt>**ImageFormatBMP**</dt> </dl>                         | indica o formato Windows BMP.<br/>                             |
| <span id="ImageFormatEMF"></span><span id="imageformatemf"></span><span id="IMAGEFORMATEMF"></span><dl> <dt>**ImageFormatEMF**</dt> </dl>                         | Indica o formato EMF.<br/>                                      |
| <span id="ImageFormatEXIF"></span><span id="imageformatexif"></span><span id="IMAGEFORMATEXIF"></span><dl> <dt>**ImageFormatEXIF**</dt> </dl>                     | Indica o formato EXIF (arquivo de imagem intercambiável).<br/>           |
| <span id="ImageFormatGIF"></span><span id="imageformatgif"></span><span id="IMAGEFORMATGIF"></span><dl> <dt>**ImageFormatGIF**</dt> </dl>                         | Indica o formato GIF.<br/>                                      |
| <span id="ImageFormatHEIF"></span><span id="imageformatheif"></span><span id="IMAGEFORMATHEIF"></span><dl> <dt>**ImageFormatHEIF**</dt> </dl>                     | Indica o formato de HEIF (formato de imagem de alta eficiência).<br/>      |
| <span id="ImageFormatIcon"></span><span id="imageformaticon"></span><span id="IMAGEFORMATICON"></span><dl> <dt>**ImageFormatIcon**</dt> </dl>                     | Indica o formato do ícone.<br/>                                     |
| <span id="ImageFormatJPEG"></span><span id="imageformatjpeg"></span><span id="IMAGEFORMATJPEG"></span><dl> <dt>**ImageFormatJPEG**</dt> </dl>                     | Indica o formato JPEG.<br/>                                     |
| <span id="ImageFormatMemoryBMP"></span><span id="imageformatmemorybmp"></span><span id="IMAGEFORMATMEMORYBMP"></span><dl> <dt>**ImageFormatMemoryBMP**</dt> </dl> | Indica que a imagem foi construída a partir de um bitmap de memória.<br/> |
| <span id="ImageFormatPNG"></span><span id="imageformatpng"></span><span id="IMAGEFORMATPNG"></span><dl> <dt>**ImageFormatPNG**</dt> </dl>                         | Indica o formato PNG.<br/>                                      |
| <span id="ImageFormatTIFF"></span><span id="imageformattiff"></span><span id="IMAGEFORMATTIFF"></span><dl> <dt>**ImageFormatTIFF**</dt> </dl>                     | Indica o formato TIFF.<br/>                                     |
| <span id="ImageFormatUndefined"></span><span id="imageformatundefined"></span><span id="IMAGEFORMATUNDEFINED"></span><dl> <dt>**ImageFormatUndefined**</dt> </dl> | indica que GDI+ não pode determinar o formato.<br/>         |
| <span id="ImageFormatWEBP"></span><span id="imageformatwebp"></span><span id="ImageFormatWEBP"></span><dl> <dt>**ImageFormatWEBP**</dt> </dl>                     | Indica o formato WebP.<br/>                                     |
| <span id="ImageFormatWMF"></span><span id="imageformatwmf"></span><span id="IMAGEFORMATWMF"></span><dl> <dt>**ImageFormatWMF**</dt> </dl>                         | Indica o formato WMF.<br/>                                      |
