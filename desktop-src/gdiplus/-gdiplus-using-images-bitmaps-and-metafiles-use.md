# <a name="using-images-bitmaps-and-metafiles"></a>Usando imagens, bitmaps e metadados

Windows GDI+ fornece a [**classe Image**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) para trabalhar com imagens de raster (bitmaps) e imagens de vetor (metadados). A [**classe Bitmap**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-bitmap) e a [**classe Metafile**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-metafile) herdam da **classe Image.** A **classe Bitmap** expande os recursos da classe **Image** fornecendo métodos adicionais para carregar, salvar e manipular imagens de raster. A **classe Metafile** expande os recursos da classe **Image** fornecendo métodos adicionais para gravar e examinar imagens de vetor.
