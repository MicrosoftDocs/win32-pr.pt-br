# <a name="using-images-bitmaps-and-metafiles"></a>Usando imagens, bitmaps e metaarquivos

O Windows GDI+ fornece a classe de [**imagem**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) para trabalhar com imagens rasterizadas (bitmaps) e imagens de vetor (metaarquivos). A classe [**bitmap**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-bitmap) e a classe [**Metafile**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-metafile) herdam da classe **Image** . A classe **bitmap** expande os recursos da classe **Image** fornecendo métodos adicionais para carregar, salvar e manipular imagens rasterizadas. A classe **Metafile** expande os recursos da classe **Image** fornecendo métodos adicionais para gravar e examinar imagens vetoriais.
