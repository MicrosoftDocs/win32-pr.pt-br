# <a name="whats-new-in-gdi"></a>O que há de novo no GDI+?

Windows GDI+ é diferente de Windows Graphics Device Interface (GDI) de duas maneiras. primeiro, GDI+ expande os recursos do GDI fornecendo novos recursos, como pincéis de gradiente e mistura alfa. Em segundo lugar, o modelo de programação foi revisado para tornar a programação de gráficos mais fácil e flexível.

-   [Alterações no modelo de programação](-gdiplus-changes-in-the-programming-model-about.md)
