# <a name="pathgradientbrushsetcenterpoint-methods"></a>Métodos PathGradientBrush. SetCenterPoint

Este tópico lista os métodos SetCenterPoint da classe [**PathGradientBrush**](/windows/win32/api/gdipluspath/nl-gdipluspath-pathgradientbrush) . Para obter uma lista completa de métodos para a classe **PathGradientBrush** , consulte [métodos PathGradientBrush](-gdiplus-class-pathgradientbrush-methods.md).

| [**SetCenterPoint (ponto&)**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_))   | O método [**PathGradientBrush:: SetCenterPoint**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_)) define o ponto central deste pincel de gradiente do caminho.
