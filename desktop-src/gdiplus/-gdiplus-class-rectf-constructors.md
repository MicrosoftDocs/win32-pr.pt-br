# <a name="rectfrectf-constructors"></a>Construtores RectF. RectF

Este tópico lista os construtores da classe [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) . Para obter uma listagem de classe completa, consulte **classe RectF**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                                        | Descrição                                                                                                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RectF (PointF&, SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))         | Cria um objeto [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) usando um objeto [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) para inicializar os membros de dados **X** e **Y** e usa um objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) para inicializar os membros de dados de **largura** e **altura** desse retângulo.<br/> |
| [**RectF (REAL, REAL, REAL, REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inreal_inreal_inreal_inreal)) | Cria um objeto [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) usando quatro inteiros para inicializar os membros de **dados X**, **Y**, **Width** e **Height** .<br/>                                                                                                                                                          |
| [**RectF()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))                                     | Cria um objeto [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) e inicializa os membros de dados **X**, **Y**, **Width** e **Height** como zero. Esse é o construtor padrão.<br/>                                                                                                                                      |
