# <a name="pointpoint-constructors"></a>Construtores ponto. Point

Este tópico lista os construtores da classe [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) . Para obter uma listagem de classe completa, consulte **Point Class**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                | Descrição                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Ponto (tamanho&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstsize_))   | Cria um objeto [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) usando um objeto [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) para inicializar os membros de dados **X** e **Y** .<br/> |
| [**Point (& de ponto)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstpoint_)) | Cria um novo objeto [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) e copia os membros de dados de outro objeto **Point** .<br/>                                           |
| [**Ponto (INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inint_inint))  | Cria um objeto [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) usando dois inteiros para inicializar os membros de dados **X** e **Y** .<br/>                                       |
| [**Ponto ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point)             | Cria um objeto [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) e inicializa os membros de dados **X** e **Y** como zero. Esse é o construtor padrão.<br/>               |
