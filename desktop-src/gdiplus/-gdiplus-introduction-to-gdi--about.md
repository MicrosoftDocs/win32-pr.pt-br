# <a name="introduction-to-gdi"></a>Introdução ao GDI+

O Windows GDI+ é uma interface gráfica de dispositivo que permite aos programadores escrever aplicativos independentes de dispositivo. Os serviços do GDI+ são expostos por meio de um conjunto de classes C++.
