# <a name="filling-a-shape-with-a-color-gradient"></a>Preenchendo uma forma com um gradiente de cor

Você pode preencher uma forma com uma cor que muda gradualmente usando um pincel de gradiente. O Windows GDI+ fornece pincéis de gradiente linear e pincéis de gradiente de caminho. A ilustração a seguir mostra um retângulo preenchido com um pincel gradiente linear e uma elipse preenchida com um pincel gradiente de demarcador.

![ilustração mostrando um retângulo que é preenchido da parte superior esquerda para a inferior direita e uma elipse que preenche de centro para borda](images/gradient2.png)

Para obter mais informações sobre pincéis de gradiente, consulte [preenchendo formas com um pincel de gradiente](-gdiplus-filling-shapes-with-a-gradient-brush-use.md).
