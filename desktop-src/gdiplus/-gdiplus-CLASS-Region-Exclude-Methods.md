# <a name="regionexclude-methods"></a>Métodos Region. Exclude

Este tópico lista os métodos de exclusão da classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obter uma lista completa de métodos para a classe **Region** , consulte [Region Methods](-gdiplus-class-region-methods.md).

| Método                                                                 | Descrição                                                                                                                                                                                         |
|:-----------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Excluir (Rect&)**](/previous-versions//ms534825(v=vs.85))     | O método [**Region:: Exclude**](/previous-versions//ms534825(v=vs.85)) atualiza essa região para a parte de si mesma que não faz a interseção do interior do retângulo especificado.<br/>  |
| [**Exclude (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstrectf_))   | O método [**Region:: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstrectf_)) atualiza essa região para a parte de si mesma que não faz a interseção do interior do retângulo especificado.<br/> |
| [**Excluir (região \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion))     | O método [**Region:: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion)) atualiza essa região para a parte de si mesma que não faz interseção com outra região.<br/>                         |
| [**Excluir (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) | O método [**Region:: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) atualiza essa região para a parte de si mesma que não faz a interseção do interior do caminho especificado.<br/>            |
