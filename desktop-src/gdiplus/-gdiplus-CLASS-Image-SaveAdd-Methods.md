# <a name="imagesaveadd-methods"></a>Métodos Image. SaveAdd

|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
