# <a name="sizefsizef-constructors"></a>Construtores SizeF. SizeF

Este tópico lista os construtores da classe [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) . Para obter uma listagem de classe completa, consulte **classe SizeF**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                          | Descrição                                                                                                                                             |
|:---------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SizeF (SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inconstsizef_))            | Cria um objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) e inicializa seus membros copiando os membros de outro objeto **SizeF** .<br/> |
| [**SizeF (REAL, REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inreal_inreal)) | Cria um objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) e inicializa seus membros de dados de **largura** e **altura** .<br/>                      |
| [**SizeF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef)                       | Cria um objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) e inicializa os membros como zero. Esse é o construtor padrão.<br/>           |
