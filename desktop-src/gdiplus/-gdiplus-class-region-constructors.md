# <a name="regionregion-constructors"></a>Construtores Region. Region

Este tópico lista os construtores da classe [**Region**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obter uma listagem de classe completa, consulte **região classe**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                                 | Descrição                                                                                                                                                                                      |
|:----------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Região (HRGN)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inhrgn))                  | Cria uma região que é idêntica à região especificada por um identificador para uma região GDI.<br/>                                                                                       |
| [**Região (Rect&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstrect_))            | Cria uma região que é definida por um retângulo.<br/>                                                                                                                                      |
| [**Região (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstrectf_))          | Cria uma região que é definida por um retângulo.<br/>                                                                                                                                      |
| [**Região (BYTE \* , int)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstbyte_inint)) | Cria uma região que é definida por dados obtidos de outra região. <br/>                                                                                                               |
| [**Região (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstgraphicspath))        | Cria uma região que é definida por um caminho (um objeto [**GraphicsPath**](/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) ) e tem um modo de preenchimento que está contido no objeto **GraphicsPath** .<br/> |
| [**Região ()**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(constregion_))                           | Cria uma região infinita. Esse é o construtor padrão. <br/>                                                                                                                  |
