# <a name="sizesize-constructors"></a>Construtores Size.Size

Este tópico lista os construtores da [**classe**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) Size. Para uma listagem de classe completa, consulte **Classe Size**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                     | Descrição                                                                                                                                          |
|:----------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Size(Size&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inconstsize_))           | Cria um [**objeto Size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) e inicializa seus membros copiando os membros de outro **objeto** Size.<br/> |
| [**Size(INT,INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inint_inint)) | Cria um [**objeto Size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) e inicializa seus membros de **dados Width** e **Height.**<br/>                     |
| [**Size()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size)                     | Cria um novo [**objeto Size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) e inicializa os membros como zero. Esse é o construtor padrão.<br/>      |
