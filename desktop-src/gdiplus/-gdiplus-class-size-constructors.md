# <a name="sizesize-constructors"></a>Construtores de tamanho. tamanho

Este tópico lista os construtores da classe [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) . Para obter uma listagem de classe completa, consulte **tamanho classe**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                     | Descrição                                                                                                                                          |
|:----------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Tamanho (tamanho&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inconstsize_))           | Cria um objeto de [**tamanho**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) e inicializa seus membros copiando os membros de outro objeto de **tamanho** .<br/> |
| [**Tamanho (INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size(inint_inint)) | Cria um objeto de [**tamanho**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) e inicializa seus membros de dados de **largura** e **altura** .<br/>                     |
| [**Tamanho ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-size-size)                     | Cria um novo objeto de [**tamanho**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) e inicializa os membros como zero. Esse é o construtor padrão.<br/>      |
