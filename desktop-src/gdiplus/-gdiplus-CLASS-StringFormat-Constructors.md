# <a name="stringformatstringformat-constructors"></a>Construtores StringFormat. StringFormat

Este tópico lista os construtores da classe [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) . Para obter uma listagem de classe completa, consulte **classe StringFormat**.

### <a name="overload-list"></a>Lista de sobrecargas



| Construtor                                                                                        | Descrição                                                                                                                        |
|:---------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**StringFormat (INT, LANGid)**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(inint_inlangid)) | Cria um objeto [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) com base em sinalizadores de formato de cadeia de caracteres e em um idioma.<br/> |
| [**StringFormat (StringFormat \* )**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(conststringformat_))           | Cria um objeto [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) a partir de outro objeto **StringFormat** .<br/>        |
