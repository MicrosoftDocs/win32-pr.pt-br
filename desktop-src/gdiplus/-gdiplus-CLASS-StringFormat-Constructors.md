# <a name="stringformatstringformat-constructors"></a>Construtores StringFormat.StringFormat

### <a name="overload-list"></a>Lista de sobrecargas

|:---------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**StringFormat(INT,LANGID)**](/windows/win32/api/gdiplusstringformat/nf-gdiplusstringformat-stringformat-stringformat(inint_inlangid)) | Cria um objeto [**StringFormat**](/windows/win32/api/gdiplusstringformat/nl-gdiplusstringformat-stringformat) com base em sinalizadores de formato de cadeia de caracteres e em um idioma.<br/> |
