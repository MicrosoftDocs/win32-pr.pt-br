# <a name="image-effect-constants"></a>Constantes de efeito de imagem

A classe [**Effect**](/windows/desktop/api/gdipluseffects/nl-gdipluseffects-effect) tem vários descendentes que podem ser usados para aplicar um efeito de cor ou ajuste a um bitmap. Cada um dos descendentes é identificado por um GUID. As constantes a seguir, definidas em Gdipluseffects. h, representam GUIDs que identificam os vários efeitos.

-   BlurEffectGuid
-   BrightnessContrastEffectGuid
-   ColorBalanceEffectGuid
-   ColorCurveEffectGuid
-   ColorLUTEffectGuid
-   ColorMatrixEffectGuid
-   HueSaturationLightnessEffectGuid
-   LevelsEffectGuid
-   RedEyeCorrectionEffectGuid
-   SharpenEffectGuid
-   TintEffectGuid
