# <a name="regioncomplement-methods"></a>Métodos Region. complemento

Este tópico lista os métodos de complemento da classe [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obter uma lista completa de métodos para a classe **Region** , consulte [Region Methods](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecargas



| Método                                                                       | Descrição                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Complemento (Rect&)**](/previous-versions//ms534918(v=vs.85))     | O método [**Region:: complemento**](/previous-versions//ms534918(v=vs.85)) atualiza essa região para a parte do interior do retângulo especificado que não faz a interseção dessa região.<br/>  |
| [**Complemento (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_))   | O método [**Region:: complemento**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_)) atualiza essa região para a parte do interior do retângulo especificado que não faz a interseção dessa região.<br/> |
| [**Complemento (região \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion))     | O método [**Region:: complemento**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion)) atualiza essa região para a parte de outra região que não intercepta essa região.<br/>                         |
| [**Complemento (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) | O método [**Region:: complemento**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) atualiza essa região para a parte do interior do caminho especificado que não faz a interseção dessa região.<br/>            |
