# <a name="image-frame-dimension-constants"></a>Constantes de dimensão de quadro de imagem

Os formatos de arquivo de imagem Graphics Interchange Format (GIF) e TIFF (Formato de Arquivo de Imagem Marcada) permitem que você armazene vários quadros em um único arquivo de imagem. Vários quadros em um arquivo GIF são usados para animação, portanto, os quadros devem estar na dimensão de tempo. Vários quadros em um arquivo TIFF normalmente são usados como páginas separadas, portanto, os quadros são usados na dimensão da página.

As constantes a seguir, definidas em Gdiplusimaging.h, representam GUIDs que identificam as dimensões de página e hora.

-   FrameDimensionPage
-   FrameDimensionTime
