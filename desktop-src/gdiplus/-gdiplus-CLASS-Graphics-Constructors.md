# <a name="graphicsgraphics-constructors"></a>Construtores Graphics.Graphics

Este tópico lista os construtores da [**classe Graphics.**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) Para uma listagem de classe completa, consulte **Classe de gráficos**.

|:------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Graphics (HDC)**](/previous-versions//ms536160(v=vs.85))                | Cria um [**objeto Graphics::Graphics**](/previous-versions//ms536160(v=vs.85)) associado a um contexto de dispositivo especificado.<br/>                                |
| [**Graphics(Image \* )**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage))          | Cria um [**objeto Graphics::Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage)) associado a um [**objeto Image.**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-image)<br/>   |
| [**Graphics(HWND,BOOL)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool))     | Cria um [**objeto Graphics::Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool)) associado a uma janela especificada.<br/>                                   |
| [**Graphics(HDC,HANDLE)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) | Cria um [**objeto Graphics::Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) associado a um contexto de dispositivo especificado e a um dispositivo especificado.<br/> |
