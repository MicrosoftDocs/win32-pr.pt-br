# <a name="graphicsgraphics-constructors"></a>Construtores gráficos. gráficos

| Construtor                                                                   | Descrição                                                                                                                                                                        |
|:------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gráficos (HDC)**](/previous-versions//ms536160(v=vs.85))                | Cria um objeto [**gráfico:: Graphics**](/previous-versions//ms536160(v=vs.85)) que está associado a um contexto de dispositivo especificado.<br/>                                |
| [**Gráficos (imagem \* )**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage))          | Cria um objeto [**Graphics:: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage)) que está associado a um objeto [**Image**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-image) .<br/>   |
| [**Gráficos (HWND, BOOL)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool))     | Cria um objeto [**Graphics:: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool)) que está associado a uma janela especificada.<br/>                                   |
| [**Gráficos (HDC, HANDLE)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) | Cria um objeto [**gráfico:: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) que está associado a um contexto de dispositivo especificado e a um dispositivo especificado.<br/> |
