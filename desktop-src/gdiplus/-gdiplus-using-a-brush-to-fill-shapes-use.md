# <a name="using-a-brush-to-fill-shapes"></a>Usando um pincel para preencher formas

Um objeto de [**pincel**](/windows/desktop/api/gdiplusbrush/nl-gdiplusbrush-brush) do Windows GDI+ é usado para preencher o interior de uma forma fechada. O GDI+ define vários estilos de preenchimento: cor sólida, padrão de hachura, textura de imagem e gradiente de cor.

Os tópicos a seguir abordam o uso de pincéis com mais detalhes:

-   [Preenchendo uma forma com uma cor sólida](-gdiplus-filling-a-shape-with-a-solid-color-use.md)
-   [Preenchendo uma forma com um padrão de hachura](-gdiplus-filling-a-shape-with-a-hatch-pattern-use.md)
-   [Preenchendo uma forma com uma textura de imagem](-gdiplus-filling-a-shape-with-an-image-texture-use.md)
-   [Colocação em um formato de forma lado a imagem](-gdiplus-tiling-a-shape-with-an-image-use.md)
-   [Preenchendo uma forma com um gradiente de cor](-gdiplus-filling-a-shape-with-a-color-gradient-use.md)
