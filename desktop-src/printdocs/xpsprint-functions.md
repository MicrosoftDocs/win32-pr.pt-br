# <a name="xps-print-api-functions"></a>Funções da API de impressão XPS

\[As funções descritas nesta seção foram preteridas. Em vez disso, os aplicativos cliente devem usar a referência de [API de pacote de documento de impressão](./tailored-app-printing-api.md) .\]

\[StartXpsPrintJob não tem suporte e pode ser alterado ou indisponível no futuro. \]

\[StartXpsPrintJob1 não tem suporte e pode ser alterado ou indisponível no futuro. \]

## <a name="in-this-section"></a>Nesta seção



| Função                                                  | Descrição                                                                   |
|-----------------------------------------------------------|-------------------------------------------------------------------------------|
| [**StartXpsPrintJob**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob)<br/>   | Inicia a impressão de um fluxo de documentos XPS em uma impressora.<br/>               |
| [**StartXpsPrintJob1**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob1)<br/> | Cria um trabalho de impressão para enviar conteúdo de documento XPS para uma impressora.<br/> |
