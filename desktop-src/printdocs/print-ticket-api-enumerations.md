# <a name="print-ticket-api-enumerations"></a>Enumerações de API de tíquete de impressão

As enumerações a seguir são usadas pela API de tíquete de impressão.

## <a name="in-this-section"></a>Nesta seção



| Enumeração                                                   | Descrição                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EDefaultDevmodeType**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | Permite que os usuários especifiquem qual [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) usar como a fonte de valores padrão quando um tíquete de impressão não especificar todas as configurações possíveis.<br/> |
| [**EPrintTicketScope**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | Especifica o escopo de um tíquete de impressão.<br/>                                                                                                                       |
