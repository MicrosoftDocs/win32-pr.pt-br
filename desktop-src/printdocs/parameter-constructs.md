# <a name="parameter-constructs"></a>Construções de parâmetro

Este tópico não é atual. Para obter as informações mais atuais, consulte a [especificação do esquema de impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Os parâmetros usados no esquema de impressão descrevem de forma concisa um intervalo aceitável de valores, em vez de uma enumeração discreta de valores. A estrutura de esquema de impressão define três elementos específicos para parâmetros: os elementos ParameterDef descrevem o parâmetro e suas características, os elementos ParameterInit definem uma instância ou um valor específico para um parâmetro, e os elementos de ParameterRef referem-se a um elemento ParameterInit específico que é definido em outro lugar.
