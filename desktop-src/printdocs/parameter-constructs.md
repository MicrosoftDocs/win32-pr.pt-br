# <a name="parameter-constructs"></a>Constructos de parâmetro

Este tópico não é atual. Para obter as informações mais atuais, consulte a [Especificação de Esquema de Impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Os parâmetros usados no Esquema de Impressão descrevem concisamente um intervalo aceitável de valores, em vez de uma enumeração discreta de valores. A Estrutura de Esquema de Impressão define três elementos específicos para parâmetros: elementos ParameterDef descrevem o parâmetro e suas características, elementos ParameterInit definem uma instância ou valor específico para um parâmetro e elementos ParameterRef referem-se a um elemento ParameterInit específico que é definido em outro lugar.
