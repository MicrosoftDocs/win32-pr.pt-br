# <a name="print-spooler-api-enumerations"></a>Enumerações da API do spooler de impressão

Esta seção contém uma listagem alfabética das enumerações que são usadas pela API do spooler de impressão.

## <a name="in-this-section"></a>Nesta seção



| Enumeração                                                             | Descrição                                                                                                        |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**EPrintXPSJobOperation**](eprintxpsjoboperation.md)<br/>       | Especifica se um trabalho de impressão XPS está na fase de processamento de spool ou de renderização.<br/>                           |
| [**EPrintXPSJobProgress**](eprintxpsjobprogress.md)<br/>         | Especifica o que o spooler está fazendo atualmente enquanto processa um trabalho de impressão XPS.<br/>                         |
| [**\_contexto de execução de impressão \_**](print-execution-context.md)<br/> | Representa o contexto de execução quando [**GetPrintExecutionData**](getprintexecutiondata.md) é chamado.<br/> |
| [**\_sinalizadores de opção de impressora \_**](printer-option-flags.md)<br/>       | Especifica o cache de um identificador para uma impressora aberta com [**OpenPrinter2**](openprinter2.md).<br/>       |
