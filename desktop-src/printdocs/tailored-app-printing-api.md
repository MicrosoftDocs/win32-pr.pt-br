# <a name="print-document-package-api"></a>Imprimir API de pacote de documento

Fornece uma interface que permite que um aplicativo acesse e gerencie o pacote de documentos de impressão.

A API de impressão de aplicativos da Windows Store está disponível com o Windows 8 e versões posteriores do Windows desktop.
