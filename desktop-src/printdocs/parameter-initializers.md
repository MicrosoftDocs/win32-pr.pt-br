# <a name="parameter-initializers"></a>Inicializadores de parâmetro

Este tópico não é atual. Para obter as informações mais atuais, consulte a [Especificação de Esquema de Impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Para ver um resumo do tipo de elemento ParameterInit, consulte a [seção ParameterInit.](parameterinit.md)

Um tipo de elemento ParameterInit atribui um valor a um ParameterDef. O ParameterDef é, portanto, inicializado no PrintTicket. Para ver um exemplo de um tipo de elemento ParameterInit, consulte o Exemplo [printTicket](printticket-example.md)do esquema de impressão.

Para obter mais informações sobre a interação entre elementos ParameterDef e elementos ParameterInit, consulte a seção [ParameterDef e ParameterInit Elements.](./parameterdef-and-parameterinit-elements.md)
