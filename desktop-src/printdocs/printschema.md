## <a name="print-schema-overview"></a><span data-ttu-id="872a1-109">Visão geral do esquema de impressão</span><span class="sxs-lookup"><span data-stu-id="872a1-109">Print Schema Overview</span></span>

<span data-ttu-id="872a1-110">O esquema de impressão é um esquema baseado em XML, hierarquicamente estruturado, usado para organizar e descrever as propriedades de uma impressora ou de um trabalho de impressão.</span><span class="sxs-lookup"><span data-stu-id="872a1-110">The Print Schema is a hierarchically structured, XML-based schema that is used to organize and describe the properties of a printer or print job.</span></span> <span data-ttu-id="872a1-111">O esquema de impressão inclui dois componentes: as palavras-chave do esquema de impressão e a estrutura de esquema de impressão.</span><span class="sxs-lookup"><span data-stu-id="872a1-111">The Print Schema includes two components: the Print Schema Keywords and the Print Schema Framework.</span></span> <span data-ttu-id="872a1-112">As palavras-chave do esquema de impressão são um conjunto de instâncias de elemento que descrevem atributos de impressora e a intenção de formatação do trabalho de impressão.</span><span class="sxs-lookup"><span data-stu-id="872a1-112">The Print Schema Keywords are a set of element instances that describe printer attributes and print job formatting intent.</span></span> <span data-ttu-id="872a1-113">A estrutura de esquema de impressão define uma coleção hierarquicamente estruturada de tipos de elemento XML e como esses tipos de elementos podem ser usados juntos.</span><span class="sxs-lookup"><span data-stu-id="872a1-113">The Print Schema Framework defines a hierarchically structured collection of XML element types and how these element types can be used together.</span></span>

<span data-ttu-id="872a1-114">As tecnologias de esquema de impressão, chamadas de *PrintCapabilities* e *PrintTicket*, são criadas usando as palavras-chave do esquema de impressão, conforme especificado pela estrutura de esquema de impressão.</span><span class="sxs-lookup"><span data-stu-id="872a1-114">The Print Schema technologies, called *PrintCapabilities* and *PrintTicket*, are built using the Print Schema Keywords as specified by the Print Schema Framework.</span></span> <span data-ttu-id="872a1-115">A especificação de esquema de impressão dá suporte a extensões de esquema por terceiros, para que os usuários do esquema de impressão não sejam restritos às instâncias de propriedade, recurso, opção ou ParameterInit definidas pelas palavras-chave do esquema de impressão.</span><span class="sxs-lookup"><span data-stu-id="872a1-115">The Print Schema Specification supports schema extensions by third parties, so that Print Schema users are not restricted to the Property, Feature, Option, or ParameterInit instances that are defined by the Print Schema Keywords.</span></span> <span data-ttu-id="872a1-116">As instâncias de elemento de terceiros podem ser adicionadas às instâncias de elemento que são definidas pelas palavras-chave do esquema de impressão; no entanto, as instâncias de propriedade de terceiros privadas devem pertencer a um namespace que está claramente associado a terceiros que criou o namespace.</span><span class="sxs-lookup"><span data-stu-id="872a1-116">Third-party element instances can be added to element instances that are defined by the Print Schema Keywords; however, private, third-party Property instances must belong to a namespace that is clearly associated with the third party that created the namespace.</span></span>
