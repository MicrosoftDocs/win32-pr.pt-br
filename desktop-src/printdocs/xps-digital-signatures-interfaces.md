# <a name="xps-digital-signature-api-interfaces"></a>Interfaces da API de Assinatura Digital XPS

## <a name="contents"></a>Sumário

## <a name="in-this-section"></a>Nesta seção



| Interface                                                                           | Descrição                                                                                         |
|-------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**IXpsSignature**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignature)<br/>                                   | Representa uma única assinatura digital.<br/>                                                   |
| [**IXpsSignatureBlock**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignatureblock)<br/>                         | Representa um bloco de solicitações de assinatura que são armazenadas em uma parte SignatureDefinitions.<br/> |
| [**IXpsSignatureBlockCollection**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignatureblockcollection)<br/>     | Uma coleção de [**interfaces IXpsSignatureBlock.**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignatureblock)<br/>             |
| [**IXpsSignatureCollection**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturecollection)<br/>               | Uma coleção de ponteiros de interface [**IXpsSignature.**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignature)<br/>               |
| [**IXpsSignatureManager**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturemanager)<br/>                     | Gerencia as assinaturas digitais e as solicitações de assinatura digital de um documento XPS.<br/>        |
| [**IXpsSignatureRequest**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturerequest)<br/>                     | Acessa os componentes de uma solicitação de assinatura.<br/>                                          |
| [**IXpsSignatureRequestCollection**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturerequestcollection)<br/> | Uma coleção de [**interfaces IXpsSignatureRequest.**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssignaturerequest)<br/>         |
| [**IXpsSigningOptions**](/windows/desktop/api/xpsdigitalsignature/nn-xpsdigitalsignature-ixpssigningoptions)<br/>                         | Fornece acesso às opções de assinatura individuais que são usadas por novas assinaturas.<br/>       |
