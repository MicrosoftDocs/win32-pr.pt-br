# <a name="xps-om-color-management"></a>Gerenciamento de cores do XPS OM

Este tópico lista as interfaces de gerenciamento de cores da API de documento XPS em um OM XPS.

-   [**Interface IXpsOMColorProfileResource**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresource)
-   [**Interface IXpsOMColorProfileResourceCollection**](/windows/desktop/api/xpsobjectmodel/nn-xpsobjectmodel-ixpsomcolorprofileresourcecollection)

Ao usar uma estrutura de [**\_ cores XPS**](xps-color.md) , um recurso de perfil de cor é necessário quando o valor do membro **ColorType** é um **\_ contexto de \_ tipo \_ de cor XPS**.
