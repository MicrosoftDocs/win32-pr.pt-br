# <a name="xps-om-trunk-interfaces"></a>Interfaces de tronco do XPS OM

As interfaces de nível de tronco da API de documento XPS descrevem a estrutura de um documento XPS conforme o OM XPS o representa.

<dl>

[Interfaces de pacote do XPS OM](xps-object-model-package-interfaces.md) As interfaces de pacote descrevem os recursos e as operações que atuam no nível superior de um OM XPS.  
[Interfaces de documento do XPS OM](xps-object-model-document-interfaces.md) As interfaces de documento descrevem a organização e a sequência dos FixedDocuments do pacote e dos FixedPages dos documentos.  
[Interfaces de parte do XPS OM](working-with-xps-object-model-part-interfaces.md) As interfaces de parte descrevem os recursos que são usados pelo documento e seus componentes.  
</dl>
