# <a name="print-spooler-api"></a>API do spooler de impressão

A API spooler de impressão fornece uma interface para o spooler de impressão para que os aplicativos gerenciem impressoras e trabalhos de impressão.

A API do spooler de impressão é usada por um aplicativo como parte de sua programação e não diretamente pelos usuários finais.

Esta seção contém informações sobre os tópicos a seguir.



| Tópico                                                                                             | Descrição                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Referência da API do spooler de impressão](printing-and-print-spooler-reference.md)<br/>                | Informações detalhadas sobre as funções, estruturas e outros elementos da API do spooler de impressão.<br/>                                                                                                           |
| [Referência de notificação de impressão assíncrona](asynchronous-printing-notification.md)<br/> | Descrições das funções, interfaces e enumerações que são usadas na comunicação assíncrona entre aplicativos e componentes hospedados no spooler de impressão, como drivers de impressora e monitores de porta.<br/> |
| [Referência de instalação do driver de impressora](printer-driver-installation-reference.md)<br/>     | Descreve as funções que instalam e configuram drivers de impressora em um computador.<br/>                                                                                                                            |
