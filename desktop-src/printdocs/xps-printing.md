# <a name="xps-print-api"></a>API de impressão XPS

\[Não há suporte para a API de impressão XPS e elas podem ser alteradas ou não disponíveis no futuro. Em vez disso, os aplicativos cliente devem usar a [API de pacote de documentos de impressão](./tailored-app-printing-api.md) .\]

Fornece uma interface para o spooler de impressão. Os aplicativos podem usar essa API para enviar documentos XPS para uma impressora.

Esta seção contém informações sobre os tópicos a seguir.

<dl>

[Sobre a API de impressão do XPS](about-xps-print-api.md)  
[Usando a API de impressão XPS](using-the-xps-print-api.md)  
[Referência da API de impressão do XPS](xpsprint-interfaces.md)  
</dl>

Os aplicativos nativos do Windows que criam documentos XPS, como usando a [API de documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)), podem usar a API de impressão XPS para enviar conteúdo de documento XPS para uma impressora.

O diagrama a seguir mostra a relação da API de impressão XPS com as outras APIs de impressão que um aplicativo nativo do Windows pode usar.
