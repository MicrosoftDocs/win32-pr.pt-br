# <a name="user-configurable-elements"></a>Elementos configuráveis pelo usuário

Este tópico não é atual. Para obter as informações mais atuais, consulte a [Especificação de Esquema de Impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Esta seção descreve as palavras-chave públicas que podem aparecer em um documento PrintCapabilities ou PrintTicket.

Não há suporte para todas as palavras-chave e elementos de palavras-chave. Os implementadores de PrintTicket e PrintCapabilities só deverão dar suporte à funcionalidade se ela for aplicável ao dispositivo de destino e/ou driver.

## <a name="element-type-implementations-for-user-configurable-elements"></a>Implementações de tipo de elemento para elementos configuráveis pelo usuário

É importante prestar atenção aos elementos Pai e Filho permitidos no conteúdo estrutural de cada palavra-chave. Para obter mais informações, consulte a [seção Resumo dos tipos de](./summary-of-element-types.md) elemento.
