# <a name="printticket-specific-property-descriptions"></a>Descrições de propriedades específicas do PrintTicket

Este tópico não é atual. Para obter as informações mais atuais, consulte a [especificação do esquema de impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Esta seção descreve as palavras-chave públicas que podem aparecer em um PrintTicket.

Não é necessário ter suporte para todas as palavras-chave. Os implementadores de PrintTicket podem dar suporte à funcionalidade somente se aplicável a sua própria implementação específica.

## <a name="element-type-implementations-for-printticket-specific-property-descriptions"></a>Implementações de tipo de elemento para descrições de propriedades específicas do PrintTicket

É importante prestar atenção aos elementos pai e filho permitidos no conteúdo estrutural para cada palavra-chave. Para obter mais informações, consulte o [Resumo da seção tipos de elemento](./summary-of-element-types.md) .
