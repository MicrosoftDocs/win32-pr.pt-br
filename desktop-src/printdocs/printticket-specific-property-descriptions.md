# <a name="printticket-specific-property-descriptions"></a>Descrições de propriedade específicas do PrintTicket

Este tópico não é atual. Para obter as informações mais atuais, consulte a [Especificação de Esquema de Impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Esta seção descreve as palavras-chave públicas que podem aparecer em um PrintTicket.

Não é necessário ter suporte para todas as palavras-chave. Os implementadores do PrintTicket PODEM dar suporte à funcionalidade somente se ela for aplicável à sua própria implementação específica.

## <a name="element-type-implementations-for-printticket-specific-property-descriptions"></a>Implementações de tipo de elemento para descrições de propriedade específicas printTicket

É importante prestar atenção aos elementos Pai e Filho permitidos no conteúdo estrutural de cada palavra-chave. Para obter mais informações, consulte a [seção Resumo dos tipos de](./summary-of-element-types.md) elemento.
