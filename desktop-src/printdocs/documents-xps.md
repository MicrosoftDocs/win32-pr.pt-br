# <a name="xps-document-api"></a>API de documento XPS

a api de documento XPS é uma api Windows nativa que permite que os programas criem, leiam, editem, salvem e imprimam documentos XPS. a API de documento XPS tem suporte no Windows 7 e em versões posteriores do Windows e pode ser usada em programas de modo de usuário e drivers de impressora XPSDrv.

a API de documento xps suports xps e OpenXPS e foi projetada para programadores de C++ que estão escrevendo aplicativos Windows nativos. Para obter mais informações sobre a especificação XPS aberta, consulte [Open XML Paper Specification (OpenXPS)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).

### <a name="about-xps-document-api"></a>Sobre a API de documento XPS

Sobre o tópico da [API de documento XPS](about-xps-document-api.md) descreve o OM XPS, pois ele é implementado pela API de documento XPS.

### <a name="using-xps-document-api"></a>Usando a API de documento XPS

[Usando a API de documento XPS](using-xps-document-api.md) descreve como começar a usar a API de documento XPS em seus programas e fornece exemplos de como executar tarefas comuns de programação. Esta seção também inclui links para aplicativos de exemplo que usam diferentes recursos da API de documento XPS.

### <a name="xps-document-api-reference"></a>Referência da API do documento XPS

A [referência da API do documento XPS](xps-programming-reference.md) contém uma lista completa de interfaces, métodos, estruturas e enumeradores que são implementados pela API do documento XPS.
