# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>Processando documentos XPS em um filtro XPSDrv

O OM XPS pode ser usado para acessar os componentes de um documento XPS de dentro de um filtro de driver de impressora XPSDrv.

Para obter mais informações sobre os filtros XPS e drivers de impressora XPSDrv, consulte [filtros XPS](/windows-hardware/drivers/print/xps-filters) no kit de driver do Windows.
