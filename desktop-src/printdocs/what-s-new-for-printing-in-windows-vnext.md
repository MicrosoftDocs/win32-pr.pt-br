# <a name="whats-new-for-printing"></a>O que há de novo para impressão

Windows 8 dá suporte à [impressão para aplicativos da Windows store usando JavaScript e HTML](/previous-versions/windows/apps/hh465225(v=win.10)) e [impressão para aplicativos da Windows store usando C#/VB/C + + e XAML](/previous-versions/windows/apps/hh465196(v=win.10)).

o Windows 8 também inclui uma nova API baseada em com que fornece suporte completo para [impressão OpenXPS](app-support-for-openxps-printing.md) e acesso a partes dos novos drivers de impressora aos quais Windows 8 dá suporte. Para obter mais informações sobre a nova API, consulte [Imprimir API do pacote de documentos](tailored-app-printing-api.md).

o programa caixa de entrada do Driver de impressão Windows garante que Windows 8 inclua suporte para uma alta porcentagem das impressoras populares.
