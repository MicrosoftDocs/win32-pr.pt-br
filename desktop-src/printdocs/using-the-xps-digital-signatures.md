# <a name="xps-om-digital-signatures"></a>Assinaturas digitais do XPS OM

Este tópico lista as interfaces de assinatura digital da API de Documentos XPS.

O XPS OM dá suporte a assinaturas digitais; no entanto, as assinaturas digitais em um documento XPS também podem ser acessadas diretamente sem a necessidade de criar um OM XPS. Para obter mais informações sobre como acessar assinaturas digitais XPS sem um OM XPS, consulte [XPS Digital Signature API](xps-digital-signatures.md).
