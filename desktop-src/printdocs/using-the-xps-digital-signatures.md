# <a name="xps-om-digital-signatures"></a>Assinaturas digitais do XPS OM

Este tópico lista as interfaces de assinatura digital da API de documento XPS.

O OM XPS dá suporte a assinaturas digitais; no entanto, as assinaturas digitais em um documento XPS também podem ser acessadas diretamente sem a necessidade de criar um OM XPS. Para obter mais informações sobre como acessar as assinaturas digitais XPS sem um OM XPS, consulte [API de assinatura digital XPS](xps-digital-signatures.md).
