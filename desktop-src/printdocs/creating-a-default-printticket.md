# <a name="creating-a-default-printticket"></a>Criando um PrintTicket padrão

Este tópico não é atual. Para obter as informações mais atuais, consulte a [Especificação de Esquema de Impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

A Interface do Provedor PrintTicket contém um método que retorna um PrintTicket padrão. Além disso, a interface do provedor PrintCapabilities tem um método que retorna um documento PrintCapabilities padrão; ou seja, um instantâneo das PrintCapabilities criadas usando um PrintTicket padrão. Esse PrintTicket padrão deve ser gerado automaticamente pelo processo a seguir.
