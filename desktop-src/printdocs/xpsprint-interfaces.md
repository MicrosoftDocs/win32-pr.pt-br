# <a name="xps-print-api-interfaces"></a>Interfaces da API de impressão XPS

\[As interfaces descritas nesta seção foram preteridas. Em vez disso, os aplicativos cliente devem usar a [API de pacote de documentos de impressão](./tailored-app-printing-api.md) .\]

\[IXpsPrintJob não tem suporte e pode ser alterado ou indisponível no futuro. \]

\[IXpsPrintJobStream não tem suporte e pode ser alterado ou indisponível no futuro. \]

## <a name="in-this-section"></a>Nesta seção



| Interface                                                   | Descrição                                                                               |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IXpsPrintJob**](/windows/desktop/api/XpsPrint/nn-xpsprint-ixpsprintjob)<br/>             | Fornece acesso a um trabalho de impressão que está atualmente em andamento.<br/>                  |
| [**IXpsPrintJobStream**](/windows/desktop/api/XpsPrint/nn-xpsprint-ixpsprintjobstream)<br/> | Uma interface de fluxo somente gravação na qual um aplicativo grava dados de trabalho de impressão.<br/> |
