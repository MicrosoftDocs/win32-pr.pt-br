# <a name="asynchronous-printing-notification-enumerations"></a>Enumerações de notificação de impressão assíncrona

As enumerações a seguir são usadas na comunicação assíncrona entre aplicativos e componentes que são hospedados pelo spooler de impressão, como drivers de impressora e monitores de porta.

## <a name="in-this-section"></a>Nesta seção



| Enumeração                                                                               | Descrição                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PrintAsyncNotifyConversationStyle**](/windows/desktop/api/prnasnot/ne-prnasnot-printasyncnotifyconversationstyle)<br/> | Especifica se a comunicação é bidirecional ou unidirecional entre aplicativos e componentes hospedados no spooler de impressão, como drivers de impressora, processadores de impressão e monitores de porta.<br/>           |
| [**PrintAsyncNotifyError**](/windows/desktop/api/prnasnot/ne-prnasnot-printasyncnotifyerror)<br/>                         | Especifica a parte do código de erro do **HRESULT** retornado após uma falha de notificação assíncrona.<br/>                                                                                            |
| [**PrintAsyncNotifyUserFilter**](/windows/desktop/api/prnasnot/ne-prnasnot-printasyncnotifyuserfilter)<br/>               | Especifica se as notificações vão apenas para os aplicativos de escuta associados ao mesmo usuário que o remetente hospedado pelo spooler de impressão ou vão para um conjunto mais amplo de aplicativos de escuta.<br/> |
