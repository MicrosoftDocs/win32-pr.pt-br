# <a name="configuration-information-not-related-to-printcapabilities"></a>Informações de configuração não relacionadas a PrintCapabilities

Este tópico não é atual. Para obter as informações mais atuais, consulte a [especificação do esquema de impressão](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Além de fornecer seleções para os atributos de dispositivo definidos nos recursos de PrintCapabilities, o PrintTicket permite que informações adicionais sejam adicionadas pelos clientes na forma de elementos de propriedade. O esquema de impressão define um número de instâncias de propriedade padrão e o provedor de interface também é gratuito para introduzir instâncias de propriedade privada. Por exemplo, se os membros de uma organização enviam a maioria de seus trabalhos de impressão para um recurso de lote central, eles podem especificar para cada trabalho uma instância de propriedade privada que contenha um endereço de retorno. Essa propriedade pode facilitar a entrega do trabalho concluído pode ser entregue ao originador.
