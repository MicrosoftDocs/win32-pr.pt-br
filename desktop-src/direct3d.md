# <a name="direct3d"></a>Direct3D

O Direct3D é uma API de nível baixo para desenhar primitivos com o pipeline de renderização ou para executar operações paralelas com o sombreador de computação. Consulte o conteúdo abaixo para obter mais informações.

Para obter informações sobre como obter e instalar o Direct3D, consulte [configuração do ambiente de programação do Direct3D 12](./direct3d12/directx-12-programming-environment-set-up.md).

| [Elementos gráficos do Direct3D 11](./direct3d11/atoc-dx-graphics-direct3d-11.md) | Você pode usar gráficos do Microsoft Direct3D 11 para criar gráficos 3D para jogos e aplicativos de área de trabalho e científica. |

| [DDS](./direct3ddds/dx-graphics-dds.md) | O formato de arquivo de superfície do DirectDraw (DDS) dá suporte a texturas descompactadas e compactadas (DXTn), mipmaps, mapas de cubo e mapas de volume. Ele tem suporte de DirectXTex, DirectXTK, D3DX herdado e outras ferramentas do DirectX. |
