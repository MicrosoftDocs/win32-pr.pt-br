# <a name="installing-an-expert-to-network-monitor-21"></a>Instalando um especialista para Monitor de Rede 2.1

Para instalar uma DLL especializada, copie-a para o subdiretório Monitor de Rede Especialistas em Monitor de Rede (por exemplo, C: Arquivos de Programas \\ \\ Especialistas em \\ NetMon2). \\ Após a instalação, o especialista será reconhecido e estará disponível na próxima vez que Monitor de Rede iniciar.

A ilustração a seguir mostra a DLL myExpert, MyExpert criada com base no código de exemplo e copiada para o \\ subdiretório Especialistas.
