# <a name="installing-an-expert-to-network-monitor-21"></a>Instalando um especialista para Monitor de Rede 2,1

Para instalar uma DLL de especialista, copie-a para o \\ subdiretório de especialistas de monitor de rede (por exemplo, C: \\ arquivos de programas \\ NetMon2 \\ experts). Após a instalação, o especialista será reconhecido e estará disponível na próxima vez em que Monitor de Rede for iniciado.

A ilustração a seguir mostra a DLL MyExpert, o MyExpert criado a partir do código de exemplo e copiado para o \\ subdiretório experts.
