# <a name="network-monitor"></a>Monitor de Rede

## <a name="purpose"></a>Finalidade

Monitor de Rede captura o tráfego de rede para exibição e análise. Ele permite que você execute tarefas como analisar dados capturados anteriormente em métodos definidos pelo usuário e extrair dados de analisadores de protocolo definidos.

## <a name="developer-audience"></a>Público de desenvolvedores

Todos os conjuntos de API fornecidos Monitor de Rede podem ser acessados usando C/C++. Esses desenvolvedores também trabalham com [*provedores de pacotes de rede*](n.md) também devem estar familiarizados com o COM.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Para chamar da API do Monitor de Rede, você deve estar executando no Windows NT Server 4.0, no Windows 2000 Server ou no Windows Server 2003 ou ter o Microsoft Systems Management Server instalado.

O driver NPP e os recursos com suporte incluem todas as versões do Windows NT 4.0 e Windows 2000 Server.

> [!Note]  
> Monitor de Rede 2.1 e anteriores não têm suporte no Windows Vista e posterior.
