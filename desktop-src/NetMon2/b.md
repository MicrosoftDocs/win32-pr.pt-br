# <a name="b-network-monitor"></a>B (Monitor de Rede)

<dl> <dt>

<span id="_netmon_binary_large_object_gly"></span><span id="_NETMON_BINARY_LARGE_OBJECT_GLY"></span>**OBJETO binário grande (BLOB)**
</dt> <dd>

A estrutura de dados genéricos Monitor de Rede. Você pode usar um BLOB para se comunicar e armazenar informações de configuração e localização e armazenar informações específicas do aplicativo , sem afetar outros dados em um BLOB.

</dd> <dt>

<span id="_netmon_blob_gly"></span><span id="_NETMON_BLOB_GLY"></span>**Blob**
</dt> <dd>

Consulte objeto binário grande.

</dd> <dt>

<span id="_netmon_buffer_gly"></span><span id="_NETMON_BUFFER_GLY"></span>**Buffer**
</dt> <dd>

Uma parte reservada da memória em que os dados são mantidos temporariamente aguardando para concluir uma transferência de ou para um dispositivo de armazenamento ou outro local na memória.

</dd> </dl>
