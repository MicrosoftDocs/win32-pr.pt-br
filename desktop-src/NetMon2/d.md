# <a name="d-network-monitor"></a>D (Monitor de Rede)

<dl> <dt>

<span id="_netmon_delayed_capture_gly"></span><span id="_NETMON_DELAYED_CAPTURE_GLY"></span>**captura atrasada**
</dt> <dd>

O tráfego de rede coletado armazenado em um arquivo de captura. Um NPP (provedor de pacotes de rede) executa uma captura atrasada. Analisadores e especialistas usam capturas atrasadas. Consulte também [*captura*](c.md)e [*captura em tempo real*](/windows)

</dd> <dt>

<span id="_netmon_destination_address_gly"></span><span id="_NETMON_DESTINATION_ADDRESS_GLY"></span>**Endereço de destino**
</dt> <dd>

O endereço do computador para o qual os dados são enviados.

</dd> <dt>

<span id="_netmon_detail_pane_gly"></span><span id="_NETMON_DETAIL_PANE_GLY"></span>**painel de detalhes**
</dt> <dd>

O painel na janela Monitor de Rede quadro que exibe informações detalhadas sobre as propriedades do protocolo. Por padrão, o painel aparece no centro da janela.

</dd> <dt>

<span id="_netmon_display_filter_gly"></span><span id="_NETMON_DISPLAY_FILTER_GLY"></span>**filtro de exibição**
</dt> <dd>

Um filtro usado para limitar a quantidade de dados capturados que são exibidos na janela do Visualizador de quadros. Um filtro de exibição é uma pilha de sufixo de objetos que podem ser um operador, valor ou propriedade.

</dd> </dl>
