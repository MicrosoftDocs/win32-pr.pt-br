# <a name="installing-existing-erps-to-network-monitor-21"></a>Instalando ERPs existentes para Monitor de Rede 2.1

Depois de escrever e nomear uma página de referência de evento (ERP), você deve instalá-la.

**Para instalar uma página de referência de evento**

1.  Crie um diretório ERP. Estes exemplos demonstram diretórios para especialistas:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Expert\
    ```

2.  Crie um subdiretório para cada novo especialista que exibirá os ERPs. As seguintes listagem de subdiretório são para um especialista:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert
    ```

3.  Copie os arquivos ERP para o subdiretório associado. Os exemplos a seguir fornecem caminhos completos para ERPs especialistas instalados:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert\MyExpert1.htm
    ```
