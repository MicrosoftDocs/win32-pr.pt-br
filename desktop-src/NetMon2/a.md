# <a name="a-network-monitor"></a>A (Monitor de Rede)

<dl> <dt>

<span id="_netmon_address_gly"></span><span id="_NETMON_ADDRESS_GLY"></span>**corrigir**
</dt> <dd>

Um identificador de um nó de rede que diferencia um nó de outro em uma rede. Também conhecido como *endereço de rede* ou endereço *Mac*. No Microsoft Systems Management Server (SMS), um endereço é usado para conectar um site e um sistema de site. Um remetente usa um endereço específico para enviar instruções e dados para outros sites.

</dd> <dt>

<span id="_netmon_address_pair_gly"></span><span id="_NETMON_ADDRESS_PAIR_GLY"></span>**par de endereços**
</dt> <dd>

Endereços de origem e de destino de uma transmissão de dados. O destino pode ser um único endereço, multicast ou difusão. Em dados capturados, o endereço de origem é um endereço específico. Nas especificações de filtro, o endereço de origem pode ser um endereço específico, ou qualquer, mas não pode ser difundido ou multicast.

</dd> </dl>
