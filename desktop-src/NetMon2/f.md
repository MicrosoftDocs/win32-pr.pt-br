# <a name="f-network-monitor"></a>F (Monitor de Rede)

<dl> <dt>

<span id="_netmon_filter_blob_gly"></span><span id="_NETMON_FILTER_BLOB_GLY"></span>**BLOB de filtro**
</dt> <dd>

BLOB (objeto binário grande) usado para restringir a pesquisa de uma NIC (placa de interface de rede). Cada entrada no BLOB de filtro deve corresponder a uma entrada equivalente no BLOB do NPP (provedor de pacotes de rede) que representa a NIC. Por exemplo, um aplicativo pode solicitar somente placas Ethernet ou cartões que dão suporte a uma taxa de quadros específica. Você pode usar um BLOB de filtro para chamar as funções **GetNPPBlobFromUI** e **GetNPPBlobTable** .

</dd> <dt>

<span id="_netmon_follow_set_gly"></span><span id="_NETMON_FOLLOW_SET_GLY"></span>**Siga o conjunto**
</dt> <dd>

Um conjunto de protocolos que segue um protocolo. Monitor de Rede usará o seguinte conjunto de um protocolo se o analisador não puder detectar o próximo protocolo dos dados na captura. Confira também [ *conjunto de entrega*](h.md)

</dd> <dt>

<span id="_netmon_frame_gly"></span><span id="_NETMON_FRAME_GLY"></span>**quadro**
</dt> <dd>

Um pacote de dados que é transmitido como uma unidade em uma rede. Cada quadro segue a mesma organização básica e contém informações de controle, como sincronização de caracteres, endereços de estações, um valor de verificação de erros e uma quantidade variável de dados. Também chamado de pacote.

</dd> <dt>

<span id="_netmon_frame_viewer_window_gly"></span><span id="_NETMON_FRAME_VIEWER_WINDOW_GLY"></span>**janela Visualizador de quadros**
</dt> <dd>

Uma janela Monitor de Rede que exibe informações detalhadas sobre quadros individuais. Você pode abrir várias janelas do Visualizador de quadros ao mesmo tempo.

</dd> </dl>
