# <a name="testing-a-monitor"></a>Testando um monitor

Depois que um monitor é instalado, você pode testá-lo com dados dinâmicos de suas operações de rede ou pode testar um conjunto conhecido de eventos capturando os dados com Monitor de Rede, salvando o arquivo de captura e, em seguida, transmitindo os dados localmente. (Para obter detalhes sobre como transmitir um arquivo de captura em uma rede, consulte Monitor de Rede ajuda online.)
