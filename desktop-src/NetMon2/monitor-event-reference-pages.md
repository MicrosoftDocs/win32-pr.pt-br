# <a name="monitor-event-reference-pages"></a>Monitorar páginas de referência de evento

As páginas de referência de evento (ERPs) podem ser usadas para conectar o monitor à interface do usuário do Monitor de Rede. Eles podem ser usados para alertar os usuários finais sobre problemas e condições detectados, ou podem ser usados para fornecer correções sugeridas ou dados explicativos.
