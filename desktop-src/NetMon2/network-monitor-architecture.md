# <a name="network-monitor-architecture"></a>Arquitetura de Monitor de Rede

A arquitetura de Monitor de Rede pode ser separada em dois conjuntos de conceitos de programação, aqueles relacionados ao desenvolvimento de [*especialistas*](e.md) e [*analisadores*](p.md) e aos relacionados ao desenvolvimento de [*Monitor*](m.md) .

Embora ambos usem os mesmos componentes para capturar o tráfego de rede, cada um deles usa interfaces diferentes para capturar os dados e diferentes componentes para analisar o que foi capturado.
