# <a name="t-network-monitor"></a>T (Monitor de Rede)

<dl> <dt>

<span id="_netmon_total_statistics_gly"></span><span id="_NETMON_TOTAL_STATISTICS_GLY"></span>**Estatísticas totais**
</dt> <dd>

As estatísticas que os relatórios da NIC (placa de interface de rede) e Monitor de Rede geram. Por exemplo, as estatísticas para os quadros de rede incluem o número de quadros que a NIC relata, o número de quadros que a NIC manipula, o número de quadros que são passados pelo filtro de captura e o número de quadros armazenados no buffer ou no arquivo de captura. Para obter uma lista completa dessas estatísticas, consulte a estrutura de **estatísticas** .

</dd> </dl>
