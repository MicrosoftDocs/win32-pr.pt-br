# <a name="selecting-a-nic-from-a-supplied-npp-blob-table"></a>Selecionando uma NIC de uma tabela de BLOB NPP fornecida

Para selecionar uma NIC (placa de interface de rede) de uma tabela de BLOB NPP fornecida, chame a função [**SelectNPPBlobFromTable**](selectnppblobfromtable.md) . Essa função usa a interface do usuário do Monitor de Rede para exibir as NICs representadas na tabela de BLOB e retorna o BLOB NPP que representa a NIC selecionada.

Quando você chama a função [**SelectNPPBlobFromTable**](selectnppblobfromtable.md) , a caixa de diálogo **selecionar uma rede** é exibida. A partir dele, você pode exibir os detalhes do NPPs representado na tabela de BLOB NPP, selecionar uma NIC específica e exibir os detalhes de sua estrutura de BLOB NPP associada.

A ilustração a seguir mostra as NICs representadas em uma tabela de BLOB NPP fornecida.
