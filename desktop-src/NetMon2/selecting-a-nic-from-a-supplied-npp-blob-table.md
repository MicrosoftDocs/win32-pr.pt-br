# <a name="selecting-a-nic-from-a-supplied-npp-blob-table"></a>Selecionando uma NIC em uma tabela de BLOB NPP fornecida

Para selecionar uma NIC (placa de interface de rede) de uma tabela NPP BLOB fornecida, chame a [**função SelectNPPBlobFromTable.**](selectnppblobfromtable.md) Essa função usa a Monitor de Rede interface do usuário para exibir as NICs representadas na tabela BLOB e retorna o BLOB NPP que representa a NIC selecionada.

Quando você chama a [**função SelectNPPBlobFromTable,**](selectnppblobfromtable.md) **a caixa de diálogo** Selecionar uma rede é exibida. Nele, você pode exibir os detalhes dos NPPs representados na tabela NPP BLOB, selecionar uma NIC específica e exibir os detalhes de sua estrutura de BLOB NPP associada.

A ilustração a seguir mostra as NICs representadas em uma tabela NPP BLOB fornecida.
