# <a name="installing-network-monitor-files"></a>Instalando arquivos de Monitor de Rede

Para instalar e executar a parte Monitor de Rede do SDK (Software Development Kit) da plataforma, siga este procedimento.

**Para instalar e executar Monitor de Rede arquivos do SDK da plataforma**

1.  Para instalar os arquivos, durante a instalação do Platform SDK, selecione o Monitor de Rede documentos e o código de exemplo.

    Quando a instalação for concluída, o programa de instalação do SDK da plataforma criará um arquivo em lotes, Setenv.bat.

2.  Para inicializar as variáveis de ambiente necessárias, execute Setenv.bat.
3.  Para compilar os exemplos, execute **NMAKE**, que está incluído no Microsoft Visual Studio.
