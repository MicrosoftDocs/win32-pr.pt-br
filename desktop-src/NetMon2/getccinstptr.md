# <a name="getccinstptr-function"></a>Função GetCCInstPtr

A função **GetCCInstPtr** retorna o ponteiro para os dados da instância adicionados ao contexto de captura.

## <a name="syntax"></a>Sintaxe


```C++
LPVOID WINAPI GetCCInstPtr(void);
```



## <a name="parameters"></a>Parâmetros

Essa função não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Se a função for bem-sucedida, o valor de retorno será um ponteiro para os dados da instância de uma captura específica.

Se a função não for bem-sucedida, o valor de retorno será **nulo**.
