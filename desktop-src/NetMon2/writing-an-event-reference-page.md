# <a name="writing-an-event-reference-page"></a>Escrevendo uma página de referência de evento

Use páginas de referência de evento (ERPs) para descrever como resolver um problema detectado pelo especialista. Como ERPs consistem em texto HTML, você pode usar qualquer editor de HTML ou editor de texto para criar um.

O painel navegador de eventos Monitor de Rede que exibe o ERP dá suporte a toda a funcionalidade do Microsoft Internet Explorer 4, 1 e posterior. Insira qualquer controle com suporte ou linguagem com script em seu ERP para exibir informações ou iniciar outras funcionalidades de seu ERP. Por exemplo, você pode criar um controle que envia email para operações de rede.

Para obter mais informações sobre como criar um ERP, consulte os seguintes tópicos:

-   [Criando páginas de referência de evento expert e monitor](creating-expert-and-monitor-event-reference-pages.md)
-   [Nomeando uma página de referência de evento](naming-an-event-reference-page.md)
-   [Testando uma página de referência de evento](testing-an-event-reference-page.md)
