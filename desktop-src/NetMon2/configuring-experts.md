# <a name="configuring-experts"></a>Configurando especialistas

Para configurar especialistas, use as caixas de diálogo personalizadas e a função [**Configurar**](configure.md) . Ao escrever especialistas no monitor de rede, você tem uma opção. Você pode criar um conjunto de itens de configuração padrão que podem ser usados quando o especialista é executado inicialmente ou pode permitir que Monitor de Rede usuários configurem o especialista em tempo de execução. As configurações para cada especialista são armazenadas por Monitor de Rede. Como opção, os usuários podem redefinir a configuração de especialista de volta para as configurações padrão originais. Lembre-se de que você pode ignorar o uso de **Configurar** desenvolvendo um especialista que passa uma configuração fixa para monitor de rede.
