# <a name="network-monitor-introduction"></a>Monitor de Rede introdução

A Monitor de Rede do SDK (Kit de Desenvolvimento de Software de Plataforma) estende a funcionalidade do Monitor de Rede, ajudando você a criar especialistas personalizados, analisadores de protocolo e NPPs (provedores de pacotes de rede). Esta seção inclui a seguinte documentação:

-   Tópicos de programação necessários para criar especialistas e analisadores. Essas informações também abordam como escrever um filtro de captura e usar BLOBs e NPPs.
-   Informações de referência relacionadas à API sobre especialistas, analisadores, BLOBs e NPPs.
-   Um glossário de termos usados na Monitor de Rede documentação.

Os subtópicos a seguir são fornecidos para ajudar a obter e instalar Monitor de Rede.

-   [Requisitos do sistema](system-requirements.md)
-   [Obtendo Monitor de Rede 2.1](obtaining-network-monitor-2-1.md)
-   [Instalando arquivos Monitor de Rede dados](installing-network-monitor-files.md)
