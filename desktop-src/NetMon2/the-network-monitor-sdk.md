# <a name="network-monitor-introduction"></a>Introdução Monitor de Rede

A parte Monitor de Rede da plataforma SDK (Software Development Kit) estende a funcionalidade do Monitor de Rede, ajudando-o a criar especialistas personalizados, analisadores de protocolos e NPPs (provedores de pacotes de rede). Esta seção inclui a seguinte documentação:

-   Tópicos de programação necessários para criar especialistas e analisadores. Essas informações também abordam como escrever um filtro de captura e usar BLOBs e NPPs.
-   Informações de referência relacionadas à API sobre especialistas, analisadores, BLOBs e NPPs.
-   Um glossário de termos usados na documentação Monitor de Rede.

Os subtópicos a seguir são fornecidos para auxiliar na obtenção e instalação de Monitor de Rede.

-   [Requisitos do sistema](system-requirements.md)
-   [Obtendo Monitor de Rede 2,1](obtaining-network-monitor-2-1.md)
-   [Instalando arquivos de Monitor de Rede](installing-network-monitor-files.md)
