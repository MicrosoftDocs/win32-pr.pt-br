# <a name="installing-a-monitor-to-network-monitor"></a>Instalando um monitor para Monitor de Rede

A instalação de uma DLL de monitor é um processo simples. Primeiro, copie a DLL e o formulário de configuração HTML associado para o subdiretório de monitores do NPP (por exemplo, C: \\ Windows \\ System32 \\ NPP \\ monitores). Quando copiado para o diretório, o monitor será reconhecido e estará disponível na próxima vez que a ferramenta de controle de monitor ou o serviço de controle de monitor (MCSVC) for ativado.

Por exemplo, se a variável de ambiente,% SystemRoot% for C: \\ Windows, o subdiretório monitor será c: \\ Windows \\ System32 \\ NPP \\ monitores.
