# <a name="obtaining-network-monitor-21"></a>Obtendo Monitor de Rede 2,1

Monitor de Rede 2,1 está disponível como uma instalação separada no Microsoft Windows Systems Management Server (SMS) 2003 ou na mídia do Windows Server 2003.

> [!Note]  
> No momento, Monitor de Rede 2,1 não está disponível como parte do SDK do Windows; Nem está disponível no Windows Vista.

 

## <a name="microsoft-windows-systems-management-server-2003"></a>Microsoft Windows Systems Management Server 2003

Instale o Monitor de Rede 2,1 executando "NetmonSetup.exe" na pasta " \\ Netmon \\ i386" no CD do produto SMS 2003.

Se estiver atualizando do SMS 2,0, primeiro você deve desinstalar Monitor de Rede 2,0 usando **Adicionar ou remover programas** no painel de controle.

## <a name="microsoft-windows-server-2003"></a>Microsoft Windows Server 2003

Siga as instruções abaixo para instalar o Monitor de Rede 2,1 da mídia do Windows Server 2003.

-   Clique em **Iniciar**, aponte para **configurações** e selecione o item de menu do **painel de controle** .
-   Clique duas vezes em **Adicionar/remover programas**.
-   Clique em **Adicionar ou remover componentes do Windows**.
-   Clique em **ferramentas de gerenciamento e monitoramento** e, em seguida, clique em **detalhes**.
-   Clique para marcar a caixa de seleção **ferramentas de monitor de rede** e clique em **OK**.
-   Clique em **Próximo**.
