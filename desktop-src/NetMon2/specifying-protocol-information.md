# <a name="specifying-protocol-information"></a>Especificando informações de protocolo

O driver Monitor de Rede pode adicionar informações à tabela de quadros especificando qual protocolo de nível baixo (IP ou IPX) foi encontrado e o deslocamento para o início do protocolo. Para adicionar essas informações à tabela de quadros, defina qualquer filtro de captura.
