# <a name="h-network-monitor"></a>H (Monitor de Rede)

<dl> <dt>

<span id="_netmon_handoff_set_gly"></span><span id="_NETMON_HANDOFF_SET_GLY"></span>**conjunto de entrega**
</dt> <dd>

Um conjunto de protocolos que segue um protocolo. Um analisador usará o conjunto de entrega de um protocolo se o analisador puder detectar o próximo protocolo dos dados na captura. Consulte também [ *seguir o conjunto*](f.md)

</dd> <dt>

<span id="_netmon_handoff_table_gly"></span><span id="_NETMON_HANDOFF_TABLE_GLY"></span>**tabela de entrega**
</dt> <dd>

Uma tabela que contém o conjunto de entrega do analisador. Você pode usar uma tabela de entrega para acessar as informações do conjunto de entrega, sem acessar o arquivo INI definido pelo analisador. As informações do conjunto de entrega são armazenadas no arquivo INI de cada analisador. Confira também *conjunto de entrega*

</dd> <dt>

<span id="_netmon_hex_pane_gly"></span><span id="_NETMON_HEX_PANE_GLY"></span>**painel hex**
</dt> <dd>

O painel na janela Visualizador de quadros Monitor de Rede que mostra os bytes hexadecimais transmitidos em quadros que são incluídos em uma captura. Por padrão, o painel Hex é exibido na parte inferior da janela.

</dd> </dl>
