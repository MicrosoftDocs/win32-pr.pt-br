# <a name="blob-structures"></a>Estruturas de BLOB

A API do auxiliar de BLOB inclui as seguintes estruturas.



| Estrutura                                                        | Descrição                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**CORRIGIR**](address.md)                                       | Contém um único endereço com suporte.                                  |
| [**ADDRESSPAIR**](addresspair.md)                               | Constrói um filtro de captura.                                           |
| [**Endereço de endereçamento**](addresstable.md)                             | Especifica uma tabela de pares de endereços.                                    |
| [**ANDEXP**](andexp.md)                                         | Especifica uma matriz de correspondências de padrão.                                 |
| [**tabela de BLOBs \_**](blob-table.md)                                | Especifica uma matriz de BLOBs.                                           |
| [**valores de \_ cabeçalho de captura \_**](capturefile-header-values.md) | Define a estrutura do arquivo de cabeçalho de Monitor de Rede.              |
| [**CAPTUREFILTER**](capturefilter.md)                           | Especifica dados de filtro de captura.                                         |
| [**EXPRESSÃO**](expression.md)                                 | Especifica uma expressão de pares estruturados em instruções AND lógicas. |
| [**QUADRO**](frame.md)                                           | Especifica um quadro.                                                     |
| [**\_descritor de quadros**](frame-descriptor.md)                    | Especifica detalhes de quadros brutos.                                       |
| [**Estrutura de quadros**](frametable.md)                                 | Especifica um conjunto de quadros.                                             |
| [**\_porta genérica**](generic-port.md)                            | Especifica um número de porta usado em protocolos IP ou IPX.                   |
| [**\_Endereço IPX**](ipx-address.md)                              | Especifica um Endereço IPX.                                              |
| [**NETWORKINFO**](networkinfo.md)                               | Especifica uma NIC.                                                       |
| [**PATTERNMATCH**](patternmatch.md)                             | Especifica os elementos de padrão usados para avaliar os quadros.                    |
| [**TRIGGER**](trigger.md)                                       | Especifica que um evento ocorre em um momento específico.                      |
| [**ATUALIZAR \_ evento**](update-event.md)                            | Atualiza eventos.                                                        |
| [**Endereço IP de VINEs \_ \_**](vines-ip-address.md)                   | Especifica um endereço em uma rede de Vines.                               |
