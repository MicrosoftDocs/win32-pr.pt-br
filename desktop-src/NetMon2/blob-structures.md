# <a name="blob-structures"></a>Estruturas BLOB

A API auxiliar de BLOB inclui as estruturas a seguir.



| Estrutura                                                        | Descrição                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**Endereço**](address.md)                                       | Contém um único endereço com suporte.                                  |
| [**ADDRESSPAIR**](addresspair.md)                               | Constrói um filtro de captura.                                           |
| [**ADDRESSTABLE**](addresstable.md)                             | Especifica uma tabela de pares de endereços.                                    |
| [**ANDEXP**](andexp.md)                                         | Especifica uma matriz de corresponde ao padrão.                                 |
| [**TABELA DE \_ BLOB**](blob-table.md)                                | Especifica uma matriz de BLOBs.                                           |
| [**VALORES DE \_ HEADER CAPTUREFILE \_**](capturefile-header-values.md) | Define a estrutura do arquivo de Monitor de Rede de texto.              |
| [**CAPTUREFILTER**](capturefilter.md)                           | Especifica dados de filtro de captura.                                         |
| [**Expressão**](expression.md)                                 | Especifica uma expressão de pares estruturados em instruções AND lógicas. |
| [**Quadro**](frame.md)                                           | Especifica um quadro.                                                     |
| [**DESCRITOR \_ DE QUADRO**](frame-descriptor.md)                    | Especifica detalhes de quadros brutos.                                       |
| [**FRAMETABLE**](frametable.md)                                 | Especifica um conjunto de quadros.                                             |
| [**PORTA \_ GENÉRICA**](generic-port.md)                            | Especifica um número da porta usado em protocolos IP ou IPX.                   |
| [**ENDEREÇO \_ IPX**](ipx-address.md)                              | Especifica um endereço IPX.                                              |
| [**NETWORKINFO**](networkinfo.md)                               | Especifica uma NIC.                                                       |
| [**PATTERNMATCH**](patternmatch.md)                             | Especifica os elementos padrão usados para avaliar quadros.                    |
| [**TRIGGER**](trigger.md)                                       | Especifica que um evento ocorre em um momento específico.                      |
| [**EVENTO \_ UPDATE**](update-event.md)                            | Atualiza eventos.                                                        |
| [**ENDEREÇO \_ IP \_ VINES**](vines-ip-address.md)                   | Especifica um endereço em uma rede Vines.                               |
