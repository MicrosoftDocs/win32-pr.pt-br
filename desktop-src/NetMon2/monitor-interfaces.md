# <a name="monitor-interfaces"></a>Monitorar interfaces

As interfaces COM a seguir são usadas para desenvolver monitores.



| Interface                              | Descrição                                                                                                                                             |
|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [IMonitor](imonitor.md)               | Fornece métodos que controlam a operação do monitor. Essa interface deve ser implementada pelo monitor.                                                     |
| [IMonitorEventer](imonitoreventer.md) | Fornece métodos usados para enviar eventos. Essa interface é fornecida pelo MCSVC ( [*serviço de controle de monitor*](m.md) ). |
