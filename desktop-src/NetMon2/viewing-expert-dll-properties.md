# <a name="viewing-expert-dll-properties"></a>Exibindo Propriedades de DLL de especialistas

Use o procedimento a seguir para exibir as propriedades de uma DLL de especialista.

**Para exibir as propriedades de DLL**

1.  Abra o Windows Explorer e vá para o \\ subdiretório experts de monitor de rede (C: \\ Program Files \\ NetMon2 \\ experts, por exemplo).
2.  Selecione a DLL desejada e clique com o botão direito do mouse no arquivo.
3.  Clique em **Propriedades** e **versão**. O exemplo de janela de status de propriedades a seguir Myexpert.dll mostra as propriedades de DLL.
4.  Clique no **nome do item** desejado para exibir informações no painel **valor** .
5.  Clique em **OK** para fechar a caixa de diálogo **Propriedades**.
