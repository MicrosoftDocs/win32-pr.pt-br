# <a name="npp-structures"></a>Estruturas NPP

Esta seção descreve as estruturas NPP usadas pelos métodos NPP. Essas estruturas são usadas para recuperar estatísticas, fornecer status do sistema e informações estatísticas e indicar quais computadores estão em execução Monitor de Rede. As tabelas a seguir descrevem essas estruturas.



| Estruturas NPP                     | Descrição                                                                                                                      |
|------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [SESSIONSTATS](sessionstats.md)   | Fornece informações de sessão quando as estatísticas de conversa são recuperadas.                                                         |
| [STATIONSTATS](stationstats.md)   | Fornece estatísticas sobre uma [*estação*](s.md)específica.                                                     |
| [ESTATÍSTICA](statistics.md)       | Fornece estatísticas de rede quando as estatísticas totais são recuperadas e quando a captura atual foi pausada ou interrompida.              |
| [CONSULTA](querytable.md)       | Indica quais computadores estão usando Monitor de Rede.                                                                             |
| [STATIONQUERY](stationquery.md)   | Fornece informações sobre um computador que está usando Monitor de Rede. Essa estrutura é usada pela estrutura do NPP **QUERYTABLE** . |
| [NETWORKSTATUS](networkstatus.md) | Fornece o status da rede atual em termos de estado do sistema.                                                                        |
