# <a name="system-requirements"></a>Requisitos do Sistema

As configurações mínimas do computador diferem; para Monitor de Rede desenvolvimento, é recomendável que um desenvolvedor tenha um computador que inclua as seguintes especificações mínimas:

-   Processador Pentium
-   32 MB de RAM
-   30 MB de espaço em disco disponível
-   Suporte ao monitor de vídeo e à placa de vídeo para 256 cores em uma resolução 800 X 600
-   Conexão de rede
-   Microsoft Internet Explorer 4.01
-   Microsoft Visual C++ versão 5.0

Testar um aplicativo desenvolvido com Monitor de Rede requer um computador com as seguintes especificações mínimas:

-   Processador de 486 executando pelo menos 50 MHz
-   16 MB de RAM
-   20 MB de espaço em disco disponível
-   Conexão de rede
