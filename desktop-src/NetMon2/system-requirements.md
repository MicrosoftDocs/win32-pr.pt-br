# <a name="system-requirements"></a>Requisitos do Sistema

As configurações mínimas do computador diferem; para fins de desenvolvimento Monitor de Rede, é recomendável que um desenvolvedor tenha um computador que inclua as seguintes especificações mínimas:

-   Processador Pentium
-   32 MB DE RAM
-   30 MB de espaço disponível no disco rígido
-   Monitor de vídeo e suporte a placa de vídeo para 256 cores com resolução 800 X 600
-   Conexão de rede
-   Microsoft Internet Explorer 4, 1
-   Microsoft Visual C++ versão 5,0

O teste de um aplicativo desenvolvido com o Monitor de Rede requer um computador com as seguintes especificações mínimas:

-   486 processador executando pelo menos 50 MHz
-   16 MB DE RAM
-   20 MB de espaço disponível no disco rígido
-   Conexão de rede
