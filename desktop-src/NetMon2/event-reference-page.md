# <a name="event-reference-page"></a>Página referência de evento

Esta seção descreve a página de referência de eventos (ERP), um documento baseado em HTML que você pode escrever para exibir eventos que ocorrem durante a operação de um especialista em Monitor de Rede. Depois de definir os requisitos de criação de especialistas, você pode usar o ERPs para ajudar a Monitor de Rede resolver problemas de rede.
