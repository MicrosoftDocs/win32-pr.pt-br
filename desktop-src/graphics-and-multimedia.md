# <a name="graphics-and-gaming"></a>Elementos gráficos e jogos

O Windows fornece APIs e componentes que dão suporte a gráficos, jogos e imagens.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [Elementos gráficos e jogos do DirectX](./directx.md)<br/> | O DirectX Graphics fornece um conjunto de APIs que você pode usar para criar jogos e outros aplicativos multimídia de alto desempenho. <br/> |
| [DirectComposition](./directcomp/directcomposition-portal.md)<br/> | O DirectComposition permite a composição de bitmap de alto desempenho com transformações, efeitos e animações. Você pode usar a API DirectComposition para criar interfaces de usuário visualmente atraentes que apresentam transições animadas ricas e fluidas de um Visual para outro.<br/> |
| [Modo de Jogo](/previous-versions/windows/desktop/gamemode/game-mode-portal)<br/> | Com as APIs de Modo de Jogo para a UWP (Plataforma Universal do Windows), você pode produzir a melhor experiência de jogo, aproveitando o Modo de Jogo no Windows 10.<br/> |
| [Informações do dispositivo de jogo](/previous-versions/windows/desktop/gamingdvcinfo/gaming-device-information-portal)<br/> | As APIs de informações de dispositivo de jogos permitem que os desenvolvedores de jogos UWP determinem o tipo de console em que o jogo está sendo executado, a fim de fazer escolhas em tempo de execução sobre como usar melhor o hardware.<br/> |
| [Windows Imaging Component (WIC)](./wic/-wic-lh.md)<br/> | O Windows Imaging Component (WIC) é uma plataforma extensível que fornece uma API de nível baixo para imagens digitais. O WIC dá suporte aos formatos de imagem da Web padrão, alta imagens de intervalo dinâmico e dados brutos da câmera.<br/> |
| [Win2D (site externo)](https://github.com/Microsoft/Win2D)<br/> | Win2D é uma API de Windows Runtime fácil de usar para renderização de gráficos 2D de modo imediato com aceleração de GPU. Ele está disponível para \# desenvolvedores C e C++ que escrevem aplicativos do Windows para Windows 8.1, Windows Phone 8,1 e Windows 10. Ele utiliza o poder do Direct2D e se integra perfeitamente com XAML e CoreWindow.<br/> |
| [ÂNGULO da Windows Store (site externo)](https://github.com/microsoft/angle/wiki)<br/> | O ângulo da Windows Store é um projeto de software livre que permite aos desenvolvedores executar o conteúdo do OpenGL ES no Windows, convertendo chamadas de API do OpenGL ES em chamadas à API do DirectX 11. O ângulo da Windows Store dá suporte a Windows 8.1, Windows Phone 8,1 e Windows 10.<br/> |

> [!NOTE]
> A partir do Windows 10, versão 1809 (10.0; Build 17763), o TruePlay foi removido do Windows. A documentação do TruePlay não foi publicada.
