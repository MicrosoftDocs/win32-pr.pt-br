# <a name="winsat-mof-classes"></a>Classes MOF do WinSAT

\[As classes MOF do WinSAT podem ser alteradas ou indisponíveis para versões após a Windows 8.1.\]

o Windows WinSAT (ferramenta de avaliação do sistema) define a seguinte classe de Instrumentação de Gerenciamento do Windows (WMI) formato MOF (MOF), que existe no \\ \\ namespace raiz cimv2:

-   [**WinSAT do Win32 \_**](win32-winsat.md)

Você pode usar essa classe para recuperar informações de resumo sobre a avaliação formal mais recente.
