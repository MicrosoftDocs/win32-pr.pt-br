# <a name="windows-system-assessment-tool-provider"></a>Windows Provedor de Ferramentas de Avaliação do Sistema

\[O WinSAT pode ser alterado ou não disponível para versões após Windows 8.1.\]

## <a name="purpose"></a>Finalidade

O Windows WinSAT (System Assessment Tool) expõe várias classes que avaliam as características de desempenho e os recursos de um computador. Os desenvolvedores podem usar essa API para desenvolver softwares que podem acessar as informações de desempenho e capacidade de um computador para determinar as configurações de aplicativo ideais com base nos recursos de desempenho do computador.

Os usuários podem usar os resultados de uma avaliação formal para determinar quais cenários e aplicativos terão um bom desempenho em um computador. Por exemplo, se um pacote de software contiver uma classificação em seu empacotamento, um usuário poderá usar a classificação para determinar se o software será executado bem no computador.

## <a name="developer-audience"></a>Público de desenvolvedores

O WinSAT foi projetado para C, C++ e desenvolvedores Visual Basic e aqueles que escrevem scripts.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O WinSAT está disponível em computadores cliente a partir do sistema operacional Windows Vista.
