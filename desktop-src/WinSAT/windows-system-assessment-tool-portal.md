# <a name="windows-system-assessment-tool-provider"></a>Provedor de ferramentas de avaliação de sistema do Windows

\[O WinSAT pode ser alterado ou indisponível para versões após a Windows 8.1.\]

## <a name="purpose"></a>Finalidade

A ferramenta de avaliação de sistema do Windows (WinSAT) expõe várias classes que avaliam as características e os recursos de desempenho de um computador. Os desenvolvedores podem usar essa API para desenvolver software que possa acessar as informações de desempenho e capacidade de um computador para determinar as configurações de aplicativo ideais com base nos recursos de desempenho do computador.

Os usuários podem usar os resultados de uma avaliação formal para determinar quais cenários e aplicativos terão um bom desempenho em um computador. Por exemplo, se um pacote de software contiver uma classificação em seu empacotamento, um usuário poderá usar a classificação para determinar se o software será executado corretamente no computador.

## <a name="developer-audience"></a>Público de desenvolvedores

O WinSAT foi projetado para desenvolvedores de C, C++ e Visual Basic e aqueles que escrevem scripts.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O WinSAT está disponível em computadores cliente que começam com o sistema operacional Windows Vista.
