# <a name="methods-_imanipulationevents"></a>Métodos (_IManipulationEvents)

Esta seção contém os métodos para a interface [**\_ IManipulationEvents.**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents)



| Método                                                                      | Descrição                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted)     | Manipula o evento para quando a manipulação ou a inércia começa.    |
| [**ManipulationDelta**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta)         | Manipula eventos que ocorrem quando um objeto manipulado muda. |
| [**ManipulationCompleted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted) | Manipula o evento quando a manipulação ou a inércia termina.      |
