# <a name="touch-input"></a>Entrada por toque

## <a name="purpose"></a>Finalidade

O Windows dá suporte à entrada por toque por meio do modelo de interação exposto pela estrutura de interface do usuário e pela plataforma de ponta a ponta do Windows Touch.

## <a name="developer-audience"></a>Público de desenvolvedores

Os recursos do Windows Touch são projetados para uso por desenvolvedores experientes de C/C++ que estão familiarizados com Component Object Model (COM), conceitos de programação de interface do usuário e conceitos de interação de toque geral.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Windows 7
