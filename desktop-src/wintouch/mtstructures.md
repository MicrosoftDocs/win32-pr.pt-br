# <a name="structures-windows-touch-gestures"></a>estruturas (Windows gestos de toque)

esta seção contém estruturas para gestos de toque Windows.

As seguintes estruturas são especificadas nesta seção.



| Estrutura                                          | Descrição                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**GESTURECONFIG**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | É usado para definir a configuração de quais mensagens de gesto estão habilitadas ou desabilitadas.      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | Contém informações sobre um gesto.                                                    |
| [**GESTURENOTIFYSTRUCT**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | Contém informações sobre uma notificação do [**WM \_ GESTURENOTIFY**](wm-gesturenotify.md) . |
