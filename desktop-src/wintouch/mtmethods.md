# <a name="methods-imanipulationprocessor"></a>Métodos (IManipulationProcessor)

Esta seção contém os métodos para a interface [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) .

Os métodos a seguir são expostos pela interface [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) .



| Método                                                                      | Descrição                                                                              |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**CompleteManipulation**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-completemanipulation) | Esse método é chamado quando o desenvolvedor opta por encerrar a manipulação.                |
| [**GetAngularVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getangularvelocity)     | Calcula a velocidade rotacional que o objeto de destino está movendo.                  |
| [**GetExpansionVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getexpansionvelocity) | Calcula a taxa em que o objeto de destino está expandindo.                              |
| [**GetVelocityX**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityx)                 | Calcula e retorna a velocidade horizontal para o objeto de destino.                    |
| [**Getvelocityy**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityy)                 | Calcula e retorna a velocidade vertical.                                            |
| [**ProcessDown**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdown)                   | Alimenta dados para o processador de manipulação associado a um destino.                       |
| [**ProcessMove**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmove)                   | Alimenta dados para o processador de manipulação associado a um destino.                       |
| [**ProcessUp**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processup)                       | Alimenta dados para o processador de manipulação associado a um destino.                       |
| [**ProcessDownWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdownwithtime)   | Alimenta dados, incluindo um carimbo de data/hora para o processador de manipulação associado a um destino. |
| [**ProcessMoveWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmovewithtime)   | Alimenta dados, incluindo um carimbo de data/hora para o processador de manipulação associado a um destino. |
| [**ProcessUpWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processupwithtime)       | Alimenta dados, incluindo um carimbo de data/hora para o processador de manipulação associado a um destino. |
