# <a name="windows-touch-samples"></a>Exemplos de toque do Windows

Descreve o caminho para os exemplos de toque do Windows e fornece breves visões gerais dos exemplos.

Ao instalar os [exemplos do Windows 7 Touch](), você pode escolher onde os exemplos são instalados. Por exemplo, se você estiver instalando na unidade C, o SDK usará como padrão "C: \\ arquivos de programas \\ Microsoft SDKs \\ Windows \\ v 7.0 \\ Samples". Dentro do diretório de exemplos, está a pasta de toque. Esta pasta contém todos os exemplos de toque do Windows.

Os exemplos de toque do Windows a seguir são fornecidos com o [SDK do Windows 7]().

- [Manipulação e exemplo de inércia](manipulation-and-inertia-sample.md)
- [Exemplo de gesto de toque do Windows (MTGestures)](windows-touch-gesture-sample--mtgestures-.md)
- [Gestos do Windows Touch na amostra do C# (MTGesturesCS)](windows-touch-gestures-in-c--sample--mtgesturescs-.md)
- [Exemplo de manipulação de toque do Windows (MTManipulation)](windows-touch-manipulation-sample--mtmanipulation-.md)
- [Exemplo de bloco de rascunho do Windows (MTScratchpadWMTouch)](windows-touch-scratchpad-sample--mtscratchpadwmtouch-.md)
- [Exemplo de bloco de rascunho do Windows em C# (MTScratchpadWMTouchCS)](windows-touch-scratchpad-sample-in-c---mtscratchpadwmtouchcs-.md)
- [Bloco de rascunho de toque do Windows usando o exemplo de Stylus (MTScratchpadRTStylus) em tempo real](windows-touch-scratchpad-using-the-real-time-stylus-sample--mtscratchpadrtstylus-.md)
- [Bloco de rascunho de toque do Windows usando a caneta em tempo real no exemplo de C# (MTScratchpadRTStylusCS)](windows-touch-scratchpad-using-the-real-time-stylus-in-c--sample--mtscratchpadrtstyluscs-.md)
