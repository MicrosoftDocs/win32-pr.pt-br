# <a name="messages-touch-input"></a>Mensagens (entrada de toque)

Esta seção contém mensagens para entrada multitouch.

As mensagens a seguir são usadas para entrada multitouch.



| Mensagem                           | Descrição                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**WM \_ TOUCH**](wm-touchdown.md) | Indica que um ou mais pontos de toque, como um dedo ou uma caneta, tocaram em uma superfície do digitalizador sensível ao toque. |
