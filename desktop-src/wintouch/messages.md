# <a name="messages-touch-input"></a>Mensagens (entrada por toque)

Esta seção contém mensagens para entrada multitoque.

As mensagens a seguir são usadas para entrada multitoque.



| Mensagem                           | Descrição                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**WM \_ Touch**](wm-touchdown.md) | Indica que um ou mais pontos de toque, como um dedo ou uma caneta, têm tocado uma superfície digitalizadora sensível ao toque. |
