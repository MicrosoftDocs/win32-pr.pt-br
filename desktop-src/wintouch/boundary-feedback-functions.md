# <a name="functions-boundary-feedback"></a>Funções (comentários de limite)

Esta seção descreve as funções de comentários de limite.



| Função                                               | Descrição                                                                                        |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**BeginPanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-beginpanningfeedback)     | Essa função é invocada pelo usuário antes de usar comentários de limite.                               |
| [**EndPanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-endpanningfeedback)     | Essa função é chamada após a conclusão dos comentários de limite e, opcionalmente, dispara uma animação. |
| [**UpdatePanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-updatepanningfeedback) | Essa função é chamada durante os comentários de limite e, opcionalmente, dispara uma animação.             |
