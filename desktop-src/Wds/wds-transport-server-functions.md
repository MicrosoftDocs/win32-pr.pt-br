# <a name="wds-transport-server-functions"></a>Funções do servidor de transporte do WDS

Funções fornecidas pelo servidor de multicast que são interface com provedores de conteúdo.



| Função                                                                         | Descrição                                              |
|----------------------------------------------------------------------------------|----------------------------------------------------------|
| [**WdsTransportServerAllocateBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverallocatebuffer)     | Aloca um buffer na memória.                            |
| [**WdsTransportServerCompleteRead**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservercompleteread)         | Fornece o status da operação de leitura.                       |
| [**WdsTransportServerFreeBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverfreebuffer)             | Libera a memória usada por um buffer.                        |
| [**WdsTransportServerRegisterCallback**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverregistercallback) | Registra um retorno de chamada do provedor com o servidor multicast. |
| [**WdsTransportServerTrace**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertrace)                       | Envia uma mensagem de depuração.                               |
| [**WdsTransportServerTraceV**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertracev)                     | Envia uma mensagem de depuração.                               |
