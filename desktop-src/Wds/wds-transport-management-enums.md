# <a name="wds-transport-management-enumerations"></a>Enumerações de gerenciamento de transporte do WDS

A API de gerenciamento de transporte do WDS (serviços de implantação do Windows) usa as seguintes enumerações:

-   [**\_sinalizadores de \_ componente de diagnóstico do WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_diagnostics_component_flags)
-   [**\_tipo de desconexão de WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_disconnect_type)
-   [**\_sinalizadores de recursos do WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_feature_flags)
-   [**\_tipo de \_ origem do endereço IP \_ do WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_ip_address_source_type)
-   [**\_tipo de \_ endereço \_ IP WDSTRANSPORT**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_ip_address_type)
-   [**\_tipo de namespace WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_namespace_type)
-   [**\_tipo de \_ perfil de rede WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_network_profile_type)
-   [**\_sinalizadores de protocolo WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_protocol_flags)
-   [**\_notificação do serviço WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_service_notification)
-   [**\_tipo de \_ manipulação de cliente WDSTRANSPORT lento \_ \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_slow_client_handling_type)

A seguir está disponível a partir do Windows 8 e do Windows Server 2012.

-   [**\_recurso de TFTP do WDSTRANSPORT \_**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_tftp_capability)
-   [**\_política de \_ porta \_ UDP WDSTRANSPORT**](/windows/win32/api/wdstptmgmt/ne-wdstptmgmt-wdstransport_udp_port_policy)
