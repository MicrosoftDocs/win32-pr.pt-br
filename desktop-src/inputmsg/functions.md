# <a name="pointer-input-messages-and-notifications-functions"></a>Mensagens de entrada de ponteiro e funções de notificações

Os tópicos nesta seção fornecem as especificações de referência para [mensagens de entrada de ponteiro e](messages-and-notifications-portal.md) funções de notificações.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                 | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EnableMouseInPointer**](/previous-versions/windows/desktop/api)<br/>                       | Permite que o mouse atue como um dispositivo de entrada de ponteiro e envie [WM_POINTER](messages.md) mensagens.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**GetPointerCursorId**](/previous-versions/windows/desktop/api)<br/>                           | Recupera o identificador de cursor associado ao ponteiro especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**GetPointerFrameInfo**](/previous-versions/windows/desktop/api)<br/>                         | Obtém o quadro inteiro de informações para os ponteiros especificados associados à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**GetPointerFrameInfoHistory**](/previous-versions/windows/desktop/api)<br/>           | Obtém o quadro inteiro de informações (incluindo quadros de entrada Unidos) para os ponteiros especificados associados à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**GetPointerFramePenInfo**](/previous-versions/windows/desktop/api)<br/>                   | Obtém o quadro inteiro de informações baseadas em caneta para os ponteiros especificados (do tipo [**PT_PEN**](/previous-versions/windows/desktop/api)) associados à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**GetPointerFramePenInfoHistory**](/previous-versions/windows/desktop/api)<br/>     | Obtém o quadro inteiro de informações baseadas em caneta (incluindo quadros de entrada Unidos) para os ponteiros especificados (do tipo [**PT_PEN**](/previous-versions/windows/desktop/api)) associados à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetPointerFrameTouchInfo**](/previous-versions/windows/desktop/api)<br/>               | Obtém o quadro inteiro de informações baseadas em toque para os ponteiros especificados (do tipo [**PT_TOUCH**](/previous-versions/windows/desktop/api)) associados à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**GetPointerFrameTouchInfoHistory**](/previous-versions/windows/desktop/api)<br/> | Obtém o quadro inteiro de informações baseadas em toque (incluindo quadros de entrada Unidos) para os ponteiros especificados (do tipo [**PT_TOUCH**](/previous-versions/windows/desktop/api)) associados à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                    |
| [**GetPointerInfo**](/previous-versions/windows/desktop/api)<br/>                                   | Obtém as informações para o ponteiro especificado associado à mensagem atual.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetPointerInfoHistory**](/previous-versions/windows/desktop/api)<br/>                     | Obtém as informações associadas às entradas individuais, se houver, que foram unidas à mensagem atual para o ponteiro especificado. A entrada mais recente é incluída no histórico retornado e é igual à entrada mais recente retornada pela função [**GetPointerInfo**](/previous-versions/windows/desktop/api) .<br/>                                                                                                                                                                                                                                                                              |
| [**GetPointerInputTransform**](/previous-versions/windows/desktop/api)<br/>               | Obtém uma ou mais transformações para as coordenadas de informações do ponteiro associadas à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**GetPointerPenInfo**](/previous-versions/windows/desktop/api)<br/>                             | Obtém as informações baseadas em caneta para o ponteiro especificado (do tipo [**PT_PEN**](/previous-versions/windows/desktop/api)) associado à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetPointerPenInfoHistory**](/previous-versions/windows/desktop/api)<br/>               | Obtém as informações baseadas em caneta associadas às entradas individuais, se houver, que foram unidas à mensagem atual para o ponteiro especificado (do tipo [**PT_PEN**](/previous-versions/windows/desktop/api)). A entrada mais recente é incluída no histórico retornado e é igual à entrada mais recente retornada pela função [**GetPointerPenInfo**](/previous-versions/windows/desktop/api) .<br/>                                                                                                                                                                                                         |
| [**GetPointerTouchInfo**](/previous-versions/windows/desktop/api)<br/>                         | Obtém as informações baseadas em toque para o ponteiro especificado (do tipo [**PT_TOUCH**](/previous-versions/windows/desktop/api)) associado à mensagem atual. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetPointerTouchInfoHistory**](/previous-versions/windows/desktop/api)<br/>           | Obtém as informações baseadas em toque associadas às entradas individuais, se houver, que foram unidas à mensagem atual para o ponteiro especificado (do tipo [**PT_TOUCH**](/previous-versions/windows/desktop/api)). A entrada mais recente é incluída no histórico retornado e é igual à entrada mais recente retornada pela função [**GetPointerTouchInfo**](/previous-versions/windows/desktop/api) .<br/>                                                                                                                                                                                                 |
| [**Getponteirotype**](/previous-versions/windows/desktop/api)<br/>                                   | Recupera o tipo de ponteiro para um ponteiro especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetUnpredictedMessagePos**](/previous-versions/windows/desktop/api)<br/>               | Obtém dados de ponteiro antes de ter passado por meio do processamento de previsão de toque.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IsMouseInPointerEnabled**](/previous-versions/windows/desktop/api)<br/>                 | Indica se [**EnableMouseInPointer**](/previous-versions/windows/desktop/api) está definido para o mouse para atuar como um dispositivo de entrada de ponteiro e enviar [WM_POINTER](messages.md) mensagens.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**SkipPointerFrameMessages**](/previous-versions/windows/desktop/api)<br/>               | Determina qual quadro de entrada do ponteiro gerou a mensagem recuperada mais recentemente para o ponteiro especificado e descarta todas as mensagens de entrada de ponteiro na fila (não recuperadas) geradas do mesmo quadro de entrada do ponteiro. Se um aplicativo tiver recuperado informações para um quadro inteiro usando a função [**GetPointerFrameInfo**](/previous-versions/windows/desktop/api) , a função [**GetPointerFrameInfoHistory**](/previous-versions/windows/desktop/api) ou uma de suas variantes específicas de tipo, ele poderá usar essa função para evitar recuperar e descartar as mensagens restantes desse quadro um a um.<br/> |
