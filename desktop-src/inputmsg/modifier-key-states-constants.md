# <a name="modifier-key-state"></a>Estado da chave modificadora

Indica quais teclas modificadora de teclado foram pressionadas no momento em que a entrada estava sendo gerada.

<dl> <dt>

<span id="POINTER_MOD_SHIFT"></span><span id="pointer_mod_shift"></span>**POINTER_MOD_SHIFT**
</dt> <dd> <dl> <dt>

0x0004
</dt> <dt>



Uma tecla Shift foi pressionada.


</dt> </dl> </dd> <dt>

<span id="POINTER_MOD_CTRL"></span><span id="pointer_mod_ctrl"></span>**POINTER_MOD_CTRL**
</dt> <dd> <dl> <dt>

0x0008
</dt> <dt>



Uma tecla Ctrl foi pressionada.


</dt> </dl> </dd> </dl>
