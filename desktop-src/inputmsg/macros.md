# <a name="macros"></a>Macros

Os tópicos nesta seção fornecem as especificações de referência para as [macros](messages-and-notifications-portal.md) Mensagens de Entrada de Ponteiro e Notificações para recuperar informações de parâmetros de mensagem de entrada de ponteiro.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                  | Descrição                                                                                                                         |
|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**GET_POINTERID_WPARAM**](/previous-versions/windows/desktop/api)<br/>                      | Recupera a ID do ponteiro usando o valor especificado. <br/>                                                                     |
| [**HAS_POINTER_CONFIDENCE_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se a mensagem de ponteiro especificada é considerada intencional em vez de acidental.<br/>                           |
| [**IS_POINTER_CANCELED_WPARAM**](/previous-versions/windows/desktop/api)<br/>         | Verifica se a entrada de ponteiro especificada terminou de forma repentina ou inválida, indicando que a interação não foi concluída.<br/> |
| [**IS_POINTER_FIFTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se o ponteiro especificado fez a quinta ação. <br/>                                                                 |
| [**IS_POINTER_FIRSTBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se o ponteiro especificado teve a primeira ação.<br/>                                                                  |
| [**IS_POINTER_FLAG_SET_WPARAM**](/previous-versions/windows/desktop/api)<br/>        | Verifica se uma macro de ponteiro define o sinalizador especificado. <br/>                                                                 |
| [**IS_POINTER_FOURTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Verifica se o ponteiro especificado teve a quarta ação. <br/>                                                                |
| [**IS_POINTER_INCONTACT_WPARAM**](/previous-versions/windows/desktop/api)<br/>       | Verifica se o ponteiro especificado está em contato. <br/>                                                                     |
| [**IS_POINTER_INRANGE_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Verifica se o ponteiro especificado está no intervalo. <br/>                                                                       |
| [**IS_POINTER_NEW_WPARAM**](/previous-versions/windows/desktop/api)<br/>                   | Verifica se o ponteiro especificado é um novo ponteiro. <br/>                                                                  |
| [**IS_POINTER_PRIMARY_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Verifica se o ponteiro especificado é o ponteiro primário. <br/>                                                            |
| [**IS_POINTER_SECONDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Verifica se o ponteiro especificado teve a segunda ação. <br/>                                                                |
| [**IS_POINTER_THIRDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se o ponteiro especificado teve a terceira ação. <br/>                                                                 |
