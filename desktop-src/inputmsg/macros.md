# <a name="macros"></a>Macros

Os tópicos nesta seção fornecem as especificações de referência para [mensagens de entrada de ponteiro e](messages-and-notifications-portal.md) macros de notificações para recuperar informações de parâmetros de mensagem de entrada de ponteiro.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                  | Descrição                                                                                                                         |
|----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**GET_POINTERID_WPARAM**](/previous-versions/windows/desktop/api)<br/>                      | Recupera a ID do ponteiro usando o valor especificado. <br/>                                                                     |
| [**HAS_POINTER_CONFIDENCE_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se a mensagem de ponteiro especificada é considerada intencional em vez de acidental.<br/>                           |
| [**IS_POINTER_CANCELED_WPARAM**](/previous-versions/windows/desktop/api)<br/>         | Verifica se a entrada do ponteiro especificada foi encerrada abruptamente ou se era inválida, indicando que a interação não foi concluída.<br/> |
| [**IS_POINTER_FIFTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se o ponteiro especificado executou a quinta ação. <br/>                                                                 |
| [**IS_POINTER_FIRSTBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se o ponteiro especificado realizou a primeira ação.<br/>                                                                  |
| [**IS_POINTER_FLAG_SET_WPARAM**](/previous-versions/windows/desktop/api)<br/>        | Verifica se uma macro de ponteiro define o sinalizador especificado. <br/>                                                                 |
| [**IS_POINTER_FOURTHBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Verifica se o ponteiro especificado executou a quarta ação. <br/>                                                                |
| [**IS_POINTER_INCONTACT_WPARAM**](/previous-versions/windows/desktop/api)<br/>       | Verifica se o ponteiro especificado está em contato. <br/>                                                                     |
| [**IS_POINTER_INRANGE_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Verifica se o ponteiro especificado está no intervalo. <br/>                                                                       |
| [**IS_POINTER_NEW_WPARAM**](/previous-versions/windows/desktop/api)<br/>                   | Verifica se o ponteiro especificado é um novo ponteiro. <br/>                                                                  |
| [**IS_POINTER_PRIMARY_WPARAM**](/previous-versions/windows/desktop/api)<br/>           | Verifica se o ponteiro especificado é o ponteiro principal. <br/>                                                            |
| [**IS_POINTER_SECONDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/> | Verifica se o ponteiro especificado executou a segunda ação. <br/>                                                                |
| [**IS_POINTER_THIRDBUTTON_WPARAM**](/previous-versions/windows/desktop/api)<br/>   | Verifica se o ponteiro especificado executou a terceira ação. <br/>                                                                 |
