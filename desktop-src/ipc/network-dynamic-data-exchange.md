# <a name="network-dynamic-data-exchange"></a>troca dinâmica de dados de rede

\[Não há mais suporte para DDE de rede. Nddeapi.dll está presente no Windows Vista, mas todas as chamadas de função retornam NDDE \_ não \_ implementado.\]

As funções de troca dinâmica de dados de rede (DDE de rede) permitem que um processo estabeleça conversas com processos em execução em computadores diferentes em uma rede.
