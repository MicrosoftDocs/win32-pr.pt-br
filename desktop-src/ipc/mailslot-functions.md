# <a name="mailslot-functions"></a>Funções do processador de processadores

As funções a seguir podem ser usadas com processadores de enup.



| Função                                   | Descrição                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**CreateMailslot**](/windows/desktop/api/Winbase/nf-winbase-createmailslota)   | Cria um processador de processadores com o nome especificado.                                  |
| [**GetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-getmailslotinfo) | Recupera informações sobre o processador de processadores especificado.                          |
| [**SetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-setmailslotinfo) | Define o valor de tempo limite usado pelo processador de linhas especificado para uma operação de leitura. |
