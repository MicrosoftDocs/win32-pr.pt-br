# <a name="anonymous-pipes"></a>Pipes anônimos

Um *pipe anônimo* é um pipe não nomeado e unidirecional que normalmente transfere dados entre um processo pai e um processo filho. Os pipes anônimos são sempre locais; Eles não podem ser usados para comunicação em uma rede.

Para mais informações, consulte os seguintes tópicos:

-   [Operações de pipe anônimo](anonymous-pipe-operations.md)
-   [Herança de identificador de pipe](pipe-handle-inheritance.md)
-   [Segurança de pipe anônimo e direitos de acesso](anonymous-pipe-security-and-access-rights.md)
