# <a name="pipes-interprocess-communications"></a>Pipes (comunicações entre processos)

Um *pipe* é uma seção da memória compartilhada que processa o uso da comunicação. O processo que cria um pipe é o *servidor de pipe*. Um processo que se conecta a um pipe é um *cliente de pipe*. Um processo grava as informações no pipe e, em seguida, o outro processo lê as informações do pipe. Esta visão geral descreve como criar, gerenciar e usar pipes.
