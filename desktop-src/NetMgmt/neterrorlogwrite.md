# <a name="neterrorlogwrite"></a>NetErrorLogWrite

A **função NetErrorLogWrite** está obsoleta. Ele está incluído para compatibilidade com versões de 16 bits do Windows. Outros aplicativos devem usar o [log de eventos](/windows/desktop/EventLog/event-logging).
