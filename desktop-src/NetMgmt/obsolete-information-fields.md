# <a name="obsolete-information-fields"></a>Campos de informações obsoletos

Muitos dos campos de informações nas estruturas de informações principais ficarão obsoletos. Esses campos permanecerão na estrutura de informações para compatibilidade com versões de 16 bits do Windows e retornará um padrão inteligente em outros sistemas Windows.
