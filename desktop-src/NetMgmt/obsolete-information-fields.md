# <a name="obsolete-information-fields"></a>Campos de informações obsoletos

Muitos dos campos de informações nas estruturas de informações principais serão obsoletos. Esses campos permanecerão na estrutura de informações para compatibilidade com versões de 16 bits do Windows e retornarão um padrão inteligente em outros Windows sistemas.
