# <a name="network-management"></a>Gerenciamento de Rede

## <a name="purpose"></a>Finalidade

As funções de gerenciamento de rede fornecem a capacidade de gerenciar contas de usuário e recursos de rede. Muitos dos recursos fornecidos pelas funções de gerenciamento de rede não são fornecidos por outras funções de rede. No entanto, se os recursos são fornecidos por outro conjunto de funções, a documentação das funções de gerenciamento de rede fará referência a outras funções que você pode usar para a mesma tarefa.

## <a name="developer-audience"></a>Público de desenvolvedores

As funções de gerenciamento de rede são projetadas para uso por programadores C/C++. Os programadores também devem estar familiarizados com Windows de rede

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

As funções de gerenciamento de rede podem ser usadas em todas as Windows plataformas. Onde determinadas implementações ou funcionalidades de restrições de plataforma de gerenciamento de rede existem, elas são claramente notadas na documentação.
