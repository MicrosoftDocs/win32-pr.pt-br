# <a name="netconfigget"></a>NetConfigGet

A função **NetConfigGet** está obsoleta. Ele é incluído para compatibilidade com versões de 16 bits do Windows. Outros aplicativos devem usar o [registro](/windows/desktop/SysInfo/registry).
