# <a name="about-network-management"></a>Sobre o gerenciamento de rede

As funções de gerenciamento de rede permitem que você gerencie compartilhamentos de rede como o Windows Explorer e o comando net. Você também pode gerenciar contas de usuário como o Gerenciador de usuários faz.

Para mais informações, consulte os seguintes tópicos:

-   [Mapeando interfaces ADSI para as funções de gerenciamento de rede](mapping-adsi-interfaces-to-the-network-management-functions.md)
