# <a name="netserviceinstall"></a>NetServiceInstall

A função **NetServiceInstall** está obsoleta. Ele é incluído para compatibilidade com versões de 16 bits do Windows. Outros aplicativos devem usar as [funções de serviço](/windows/desktop/Services/service-functions).
