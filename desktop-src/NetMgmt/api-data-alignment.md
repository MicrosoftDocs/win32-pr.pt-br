# <a name="api-data-alignment"></a>Alinhamento de dados de API

Todas as estruturas especificadas para as funções de gerenciamento de rede devem ser alinhadas com o Word de 32 bits. O tamanho base de um elemento de estrutura é um **DWORD**.
