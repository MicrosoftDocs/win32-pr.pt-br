# <a name="netservicecontrol"></a>NetServiceControl

A **função NetServiceControl** está obsoleta. Ele está incluído para compatibilidade com versões de 16 bits do Windows. Outros aplicativos devem usar as [funções de serviço](/windows/desktop/Services/service-functions).
