# <a name="nls-support"></a>Suporte a NLS

As funções de gerenciamento de rede usam cadeias de caracteres Unicode como entrada e fornecem cadeias de caracteres Unicode na saída. Se seu aplicativo geralmente funciona com cadeias de caracteres ANSI, deve-se ter cuidado para converter de e para Unicode, quando apropriado.
