# <a name="wrp-functions"></a>Funções WRP

As funções a seguir permitem enumerar recursos protegidos.



| Função                                         | Descrição                                                 |
|--------------------------------------------------|-------------------------------------------------------------|
| [**SfcIsFileProtected**](/windows/desktop/api/Sfc/nf-sfc-sfcisfileprotected) | Determina se o arquivo especificado está protegido.         |
| [**SfcIsKeyProtected**](/windows/desktop/api/Sfc/nf-sfc-sfciskeyprotected)   | Determina se a chave do Registro especificada está protegida. |



 

Se estiver disponível para o sistema operacional, as funções nesta tabela deverão ser usadas em vez das funções preteridas: [**SfcGetNextProtectedFile**](/windows/desktop/api/Sfc/nf-sfc-sfcgetnextprotectedfile) e [**SfcGetFiles**](sfcgetfiles.md).
