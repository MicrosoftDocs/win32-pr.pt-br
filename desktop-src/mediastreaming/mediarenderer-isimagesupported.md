# <a name="mediarendererisimagesupported-property"></a>Propriedade MediaRenderer. IsImageSupported

Obtém um valor que indica se o DMR é capaz de exibir imagens.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_IsImageSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valor da propriedade

Um valor booliano que será **true** se o DMR for capaz de exibir imagens e **false** se não for.
