# <a name="basicdevicecanwakedevices-property"></a>Propriedade BasicDevice. CanWakeDevices

Obtém um valor que indica se o dispositivo pode ser ativado.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_CanWakeDevices(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valor da propriedade

Um ponteiro para um valor booliano que é **verdadeiro** se o dispositivo puder ser ativado.
