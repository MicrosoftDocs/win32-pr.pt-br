# <a name="getmuteoperation-class"></a>Classe GetMuteOperation

Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada por [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) é concluída e fornece um método que retorna os resultados da operação.

**GetMuteOperation** tem estes tipos de membros:

-   [Métodos](#methods)
-   [Propriedades](#properties)

### <a name="methods"></a>Métodos

A **classe GetMuteOperation** tem esses métodos.



| Método                                            | Descrição                                                                                                                   |
|:--------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getmuteoperation-getresults.md) | Retorna os resultados da operação assíncrona iniciada por [**GetMuteAsync.**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) <br/> |



 

### <a name="properties"></a>Propriedades

A **classe GetMuteOperation** tem essas propriedades.



| Propriedade                                                   | Tipo de acesso           | Descrição                                                                                                                                                            |
|:-----------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Concluído**](getmuteoperation-completed.md)<br/> | Leitura/gravação<br/> | Obtém ou define um manipulador de eventos que é invocado quando a operação assíncrona iniciada por [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) é concluída. <br/> |
