# <a name="createmediarendereroperationgetresults-method"></a>Método CreateMediaRendererOperation:: GetResults

Obtém os resultados da operação assíncrona.

## <a name="syntax"></a>Sintaxe


```C++
MediaRenderer* GetResults();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Os resultados da operação. Um ponteiro para o objeto [**MediaRenderer**](mediarenderer.md) recém-criado.
