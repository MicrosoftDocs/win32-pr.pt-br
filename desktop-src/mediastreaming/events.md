# <a name="events-media-streaming-api"></a>Eventos (API de streaming de mídia)

A [API de streaming de mídia](media-streaming-api-portal.md) gera os eventos a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                           | Descrição                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Evento ConnectionStatusChanged**](connectionstatuschanged.md)<br/>     | Ocorre quando o status da conexão de rede do dispositivo é alterado.<br/>                                                                                   |
| [**Evento DeviceArrival**](devicearrival.md)<br/>                         | Ocorre quando um novo dispositivo é adicionado à lista de dispositivos retornada pelo método [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/>     |
| [**Evento DeviceDeparture**](devicedeparture.md)<br/>                     | Ocorre quando um novo dispositivo é removido da lista de dispositivos retornada pelo método [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/> |
| [**Evento RenderingParametersUpdate**](renderingparametersupdate.md)<br/> | Ocorre sempre que qualquer um de um conjunto de parâmetros de controle de renderização é atualizado no DMR.<br/>                                                          |
| [**Evento TransportParametersUpdate**](transportparametersupdate.md)<br/> | Ocorre sempre que qualquer um de um conjunto de parâmetros de transporte é atualizado no DMR.<br/>                                                                  |
