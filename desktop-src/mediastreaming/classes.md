# <a name="classes-media-streaming-api"></a>Classes (API de streaming de mídia)

A [API de streaming de mídia](media-streaming-api-portal.md) fornece as seguintes classes.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                   | Descrição                                                                                                                                                                                                                                                                                                                                                                         |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActiveBasicDevice**](/previous-versions/windows/desktop/legacy/dn385755(v=vs.85))<br/>                               | Implementa a interface [**IActiveBasicDevice**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-iactivebasicdevice) que representa um dispositivo de rede de vida digital (DLNA) ativo.<br/>                                                                                                                                                                                                                       |
| [**BasicDevice**](/previous-versions/windows/desktop/legacy/hh828813(v=vs.85))<br/>                                           | Implementa a interface [**IBasicDevice**](ibasicdevice.md) que representa um dispositivo DLNA.<br/>                                                                                                                                                                                                                                                                             |
| [**CreateMediaRendererOperation**](createmediarendereroperation.md)<br/>         | Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada por [**CreateMediaRendererAsync**](imediarendererfactory-createmediarendererasync.md) ou [**CreateMediaRendererFromBasicDeviceAsync**](imediarendererfactory-createmediarendererfrombasicdeviceasync.md) é concluída e fornece um método que retorna os resultados da operação.<br/> |
| [**DeviceController**](/previous-versions/windows/desktop/legacy/hh828842(v=vs.85))<br/>                                 | Implementa a interface [**IDeviceController**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-idevicecontroller) que recupera uma lista de DMRs (renderizadores de mídia digital) e/ou servidores de mídia digital (DMSs) em cache, ou para localizar de forma assíncrona o DMRs e/ou o DMSs que estão atualmente na rede.<br/>                                                                                                            |
| [**DevicePair**](/previous-versions/windows/desktop/legacy/dn385771(v=vs.85))<br/>                                             | Implementa a interface [**IDevicePair**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-idevicepair) que representa um par de objetos [**ActiveBasicDevice**](/previous-versions/windows/desktop/legacy/dn385755(v=vs.85)) que é composto por um processador e um servidor.<br/>                                                                                                                                                                              |
| [**GetMuteOperation**](getmuteoperation.md)<br/>                                 | Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) é concluída e fornece um método que retorna os resultados da operação.<br/>                                                                                                                                                    |
| [**GetPositionInformationOperation**](getpositioninformationoperation.md)<br/>   | Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) é concluída e fornece um método que retorna os resultados da operação.<br/>                                                                                                                      |
| [**GetStreamPropertiesOperation**](getstreampropertiesoperation.md)<br/>         | Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) é concluída e fornece um método que retorna os resultados da operação.<br/>                                                                                                                            |
| [**GetTransportInformationOperation**](gettransportinformationoperation.md)<br/> | Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetTransportInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-gettransportinformationasync) é concluída e fornece um método que retorna os resultados da operação.<br/>                                                                                                                    |
| [**GetVolumeOperation**](getvolumeoperation.md)<br/>                             | Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetVolumeAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getvolumeasync) é concluída e fornece um método que retorna os resultados da operação.<br/>                                                                                                                                                |
| [**MediaRenderer**](mediarenderer.md)<br/>                                       | Implementa a interface [**IMediaRenderer**](imediarenderer.md) que representa um dispositivo DMR (processador de mídia digital DLNA).<br/>                                                                                                                                                                                                                                            |
| [**PlaybackOperation**](playbackoperation.md)<br/>                               | Registra um manipulador de eventos que é invocado quando uma operação assíncrona iniciada por um dos métodos de reprodução [**MediaRenderer**](mediarenderer.md) é concluída e fornece um método que retorna os resultados da operação.<br/>                                                                                                                                      |
| [**StreamSelectOperation**](streamselectoperation.md)<br/>                       | Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) é concluída e fornece um método que retorna os resultados da operação.<br/>                                                                                                                                                    |
| [**StreamSelector**](streamselector.md)<br/>                                     | Implementa a interface [**IStreamSelectorStatics**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-istreamselectorstatics) e habilita a seleção de um fluxo.<br/>                                                                                                                                                                                                                                                        |
