# <a name="getpositioninformationoperation-class"></a>Classe GetPositionInformationOperation

Registra um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) é concluída e fornece um método que retorna os resultados da operação.

**GetPositionInformationOperation** tem estes tipos de membros:

-   [Métodos](#methods)
-   [Propriedades](#properties)

### <a name="methods"></a>Métodos

A classe **GetPositionInformationOperation** tem esses métodos.



| Método                                                           | Descrição                                                                                                                                                 |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getpositioninformationoperation-getresults.md) | Retorna os resultados da operação assíncrona iniciada pelo [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync). <br/> |



 

### <a name="properties"></a>Propriedades

A classe **GetPositionInformationOperation** tem essas propriedades.



| Propriedade                                                                  | Tipo de acesso           | Descrição                                                                                                                                                                                          |
|:--------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Concluído**](getpositioninformationoperation-completed.md)<br/> | Leitura/gravação<br/> | Obtém ou define um manipulador de eventos que é invocado quando a operação assíncrona iniciada pelo [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) é concluída. <br/> |
