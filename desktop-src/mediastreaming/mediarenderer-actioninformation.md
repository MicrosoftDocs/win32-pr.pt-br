# <a name="mediarendereractioninformation-property"></a>Propriedade MediaRenderer.ActionInformation

Obtém informações sobre quais métodos podem ser invocados no momento na DMR.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_ActionInformation(
  [out] IMediaRendererActionInformation **value
);
```



## <a name="property-value"></a>Valor da propriedade

Uma referência a [**uma interface IMediaRendererActionInformation.**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-imediarendereractioninformation)
