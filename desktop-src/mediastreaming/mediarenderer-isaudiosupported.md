# <a name="mediarendererisaudiosupported-property"></a>Propriedade MediaRenderer. IsAudioSupported

Obtém um valor que indica se o DMR é capaz de reproduzir conteúdo de áudio.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_IsAudioSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valor da propriedade

Um valor booliano que será **true** se o DMR for capaz de reproduzir o conteúdo de áudio e **false** se não for.
