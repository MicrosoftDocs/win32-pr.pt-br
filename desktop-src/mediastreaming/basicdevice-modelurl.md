# <a name="basicdevicemodelurl-property"></a>Propriedade BasicDevice.ModelUrl

Obtém a URL do modelo do dispositivo.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_ModelUrl(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valor da propriedade

Um ponteiro para a URL do modelo do dispositivo.
