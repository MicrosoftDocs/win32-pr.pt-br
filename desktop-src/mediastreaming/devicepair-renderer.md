# <a name="devicepairrenderer-property"></a>Propriedade DevicePair:: Renderer

Obtém o renderizador para o par de dispositivos básico ativo.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_Renderer(
  [out] ActiveBasicDevice **value
);
```



## <a name="property-value"></a>Valor da propriedade

Recebe um objeto [**ActiveBasicDevice**](/previous-versions/windows/desktop/legacy/dn385755(v=vs.85)) que representa o dispositivo de processador.
