# <a name="basicdeviceremotestreamingurls-property"></a>Propriedade BasicDevice. RemoteStreamingUrls

Obtém um vetor de URLs de streaming remoto.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_RemoteStreamingUrls(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>Valor da propriedade

Uma coleção enumerável de ponteiros para URLs de streaming remota.
