# <a name="streamselector-class"></a>Classe StreamSelector

Implementa a interface [**IStreamSelectorStatics**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-istreamselectorstatics) e habilita a seleção de um fluxo.

**StreamSelector** tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A classe **StreamSelector** tem esses métodos.



| Método                                                                      | Descrição                                                  |
|:----------------------------------------------------------------------------|:-------------------------------------------------------------|
| [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) | Obtém as propriedades do fluxo de forma assíncrona.<br/> |
| [**SelectBestStreamAsync**](/previous-versions/windows/desktop/legacy/hh829002(v=vs.85))       | Consultas de forma assíncrona para o melhor fluxo.<br/>       |
