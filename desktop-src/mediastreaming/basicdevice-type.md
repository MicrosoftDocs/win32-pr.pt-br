# <a name="basicdevicetype-property"></a>Propriedade BasicDevice. Type

Obtém um valor de enumeração que indica o tipo de dispositivo do dispositivo DLNA.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_Type(
  [out] DeviceTypes *value
);
```



## <a name="property-value"></a>Valor da propriedade

Um ponteiro para um valor de [**DeviceTypes**](devicetypes.md) .
