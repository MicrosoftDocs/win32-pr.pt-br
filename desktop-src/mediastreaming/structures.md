# <a name="media-streaming-structures"></a>Estruturas de streaming de mídia

A [API de streaming de mídia](media-streaming-api-portal.md) fornece as seguintes estruturas.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                           | Descrição                                                                                                                                        |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PlaySpeed**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-playspeed)<br/>                       | Representa uma velocidade de reprodução como um número racional. <br/>                                                                                      |
| [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation)<br/>   | Contém os valores atuais das informações de posição de reprodução de mídia obtidas do DMR.<br/>                                               |
| [**Renderingparameters**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-renderingparameters)<br/>   | Contém os valores atuais dos parâmetros de renderização no DMR.<br/>                                                                         |
| [**TrackInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-trackinformation)<br/>         | Contém o número de faixa atual e a duração como parte do [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation) obtido do DMR.<br/> |
| [**TransportInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-transportinformation)<br/> | Contém os valores atuais das informações relacionadas ao transporte de reprodução de mídia obtidas do DMR.<br/>                                      |
