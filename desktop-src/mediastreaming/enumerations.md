# <a name="media-streaming-enumerations"></a>Enumerações de streaming de mídia

A [API de streaming de mídia](media-streaming-api-portal.md) fornece as seguintes enumerações.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                   | Descrição                                                                               |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**ConnectionStatus**](connectionstatus.md)<br/> | Representa o estado do dispositivo na rede como visto pela última vez.<br/>                |
| [**DeviceTypes**](devicetypes.md)<br/>           | Descreve os tipos de dispositivo DLNA que são suportados pela API de streaming de mídia.<br/> |
| [**Transportestate**](transportstate.md)<br/>     | Define os Estados de transporte disponíveis, conforme definido pelas diretrizes de UPnP.<br/>      |
| [**TransportStatus**](transportstatus.md)<br/>   | Define o status de transporte disponível conforme definido pelas diretrizes de UPnP.<br/>      |
