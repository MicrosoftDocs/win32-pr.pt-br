# <a name="media-streaming-enumerations"></a>Enumerações de Streaming de Mídia

A [API de Streaming de Mídia](media-streaming-api-portal.md) fornece as enumerações a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                   | Descrição                                                                               |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**ConnectionStatus**](connectionstatus.md)<br/> | Representa o estado do dispositivo na rede como visto pela última vez.<br/>                |
| [**DeviceTypes**](devicetypes.md)<br/>           | Descreve os tipos de dispositivo DLNA com suporte pela API de Streaming de Mídia.<br/> |
| [**TransportState**](transportstate.md)<br/>     | Define os estados de transporte disponíveis, conforme definido pelas Diretrizes de UPnP.<br/>      |
| [**TransportStatus**](transportstatus.md)<br/>   | Define o status de transporte disponível conforme definido pelas Diretrizes de UPnP.<br/>      |
