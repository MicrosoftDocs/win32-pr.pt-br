# <a name="basicdeviceuniquedevicename-property"></a>Propriedade BasicDevice. UniqueDeviceName

Obtém o nome de dispositivo exclusivo (UDN) do dispositivo.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_UniqueDeviceName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valor da propriedade

Um ponteiro para o modelo do dispositivo UDN.
