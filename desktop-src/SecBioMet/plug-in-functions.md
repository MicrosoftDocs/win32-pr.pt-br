## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                 | Descrição                                                                                                                                                                                                                                  |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Funções de adaptador de mecanismo](engine-adapter-functions.md)<br/>   | Um adaptador de mecanismo gera modelos biométricos de exemplos capturados, corresponde exemplos a modelos existentes e indexa modelos.<br/>                                                                                              |
| [Funções do adaptador de sensor](sensor-adapter-functions.md)<br/>   | Um adaptador de sensor envolve um dispositivo biométrico e fornece uma interface padrão que permite que o serviço biométrico Windows configure o sensor, capture exemplos e controle o fluxo de dados biométricos para o mecanismo de processamento.<br/> |
