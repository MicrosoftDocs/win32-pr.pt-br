# <a name="winbio_no_format-constants"></a>Constantes WINBIO \_ NO \_ FORMAT

Se os membros de uma estrutura [**WINBIO \_ REGISTERED \_ FORMAT**](winbio-registered-format.md) não puderem ser zero, você poderá usar as constantes a seguir para verificar se há erros.



| Constante/valor                                                                                                                                                                                                                                                                   | Descrição                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO \_ NENHUM \_ PROPRIETÁRIO DE FORMATO \_ \_ DISPONÍVEL**</dt> <dt>0</dt> </dl> | Nenhum valor de proprietário atribuído à IBIA (Associação Internacional do Setor Biométrico) está disponível.<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO \_ NENHUM \_ TIPO DE FORMATO \_ \_ DISPONÍVEL**</dt> <dt>0</dt> </dl>    | Nenhum tipo de formato atribuído ao proprietário está disponível.<br/>                                               |
