# <a name="winbio_no_format-constants"></a>WINBIO \_ nenhuma \_ constante de formato

Se os membros de uma estrutura de [**\_ \_ formato registrado WINBIO**](winbio-registered-format.md) não puderem ser zero, você poderá usar as constantes a seguir para verificar se há erros.



| Constante/valor                                                                                                                                                                                                                                                                   | Descrição                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO \_ NENHUM \_ proprietário de formato \_ \_ disponível**</dt> <dt>0</dt> </dl> | Não há um valor de proprietário atribuído à IBIA (Associação de indústria biométrica internacional) disponível.<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO \_ NENHUM \_ tipo de formato \_ \_ disponível**</dt> <dt>0</dt> </dl>    | Nenhum tipo de formato atribuído ao proprietário está disponível.<br/>                                               |
