# <a name="system-pool-behavior"></a><span data-ttu-id="97334-103">Comportamento do pool do sistema</span><span class="sxs-lookup"><span data-stu-id="97334-103">System Pool Behavior</span></span>

<span data-ttu-id="97334-104">A discussão a seguir realça as ações executadas pelo pool do sistema quando avisos de evento são enviados e quando nenhuma operação biométrica está pendente.</span><span class="sxs-lookup"><span data-stu-id="97334-104">The following discussion highlights the actions taken by the system pool when event notices are sent and when no biometric operations are pending.</span></span>

## <a name="event-dispatching"></a><span data-ttu-id="97334-105">Expedição de eventos</span><span class="sxs-lookup"><span data-stu-id="97334-105">Event Dispatching</span></span>

<span data-ttu-id="97334-106">Quando uma unidade biométrica gera um aviso de evento, o pool de sistema usa um filtro em cascata para enviar o aviso e atribuir a ele um dos seguintes níveis de prioridade:</span><span class="sxs-lookup"><span data-stu-id="97334-106">When a biometric unit generates an event notice, the system pool uses a cascading filter to dispatch the notice and assign it one of the following priority levels:</span></span>

-   <span data-ttu-id="97334-107">A prioridade alta é atribuída à correspondência explícita e às solicitações de registro geradas pelos clientes.</span><span class="sxs-lookup"><span data-stu-id="97334-107">High priority is assigned to explicit matching and enrollment requests generated by clients.</span></span>
-   <span data-ttu-id="97334-108">A prioridade média é atribuída a eventos de registro ou de correspondência inesperados ou não reivindicados.</span><span class="sxs-lookup"><span data-stu-id="97334-108">Medium priority is assigned to unexpected or unclaimed matching or enrollment events.</span></span>
-   <span data-ttu-id="97334-109">Baixa prioridade é atribuída a eventos de navegação.</span><span class="sxs-lookup"><span data-stu-id="97334-109">Low priority is assigned to navigation events.</span></span>

<span data-ttu-id="97334-110">Os eventos de captura são entregues na seguinte sequência:</span><span class="sxs-lookup"><span data-stu-id="97334-110">Capture events are delivered in the following sequence:</span></span>

-   <span data-ttu-id="97334-111">Se a janela de foco atual estiver aguardando uma operação de registro ou correspondência, o exemplo será processado e enviado ao cliente que possui a janela de foco atual.</span><span class="sxs-lookup"><span data-stu-id="97334-111">If the current focus window is waiting for a matching or an enrollment operation, the sample is processed and sent to the client that owns the current focus window.</span></span>
-   <span data-ttu-id="97334-112">Se o evento de captura não for reivindicado pela janela de foco atual e um manipulador de eventos não reivindicado tiver sido registrado com o serviço biométrico do Windows, o evento de captura será enviado para esse manipulador.</span><span class="sxs-lookup"><span data-stu-id="97334-112">If the capture event is unclaimed by the current focus window and an unclaimed event handler has been registered with the Windows Biometric service, the capture event is sent to this handler.</span></span>
-   <span data-ttu-id="97334-113">Se o evento permanecer não reivindicado, ele será Descartado.</span><span class="sxs-lookup"><span data-stu-id="97334-113">If the event remains unclaimed, it is discarded.</span></span>

<span data-ttu-id="97334-114">Se o evento for um evento de navegação e um manipulador de eventos de navegação tiver sido registrado com o serviço biométrico do Windows, o evento de captura será enviado para esse manipulador.</span><span class="sxs-lookup"><span data-stu-id="97334-114">If the event is a navigation event and a navigation event handler has been registered with the Windows Biometric service, the capture event is sent to this handler.</span></span> <span data-ttu-id="97334-115">Se não houver nenhum manipulador de eventos, o evento será Descartado.</span><span class="sxs-lookup"><span data-stu-id="97334-115">If there is no event handler, the event is discarded.</span></span>

## <a name="idle-mode"></a><span data-ttu-id="97334-116">Modo ocioso</span><span class="sxs-lookup"><span data-stu-id="97334-116">Idle Mode</span></span>

<span data-ttu-id="97334-117">Quando não há clientes aguardando que as solicitações de registro ou correspondência explícita sejam concluídas, o pool do sistema determina se as solicitações de captura repetidas devem ser geradas automaticamente e o aviso de evento resultante será enviado ao manipulador de eventos não reivindicado ou se os eventos de navegação serão enviados para o manipulador de eventos de navegação.</span><span class="sxs-lookup"><span data-stu-id="97334-117">When there are no clients waiting for explicit matching or enrollment requests to complete, the system pool determines whether to automatically generate repeated capture requests and send the resulting event notice to the unclaimed event handler or wait for navigation events and send them to the navigation event handler.</span></span>

<span data-ttu-id="97334-118">Se um manipulador de eventos não reivindicado tiver sido registrado com o serviço biométrico do Windows, o pool do sistema executará as seguintes ações:</span><span class="sxs-lookup"><span data-stu-id="97334-118">If an unclaimed event handler has been registered with the Windows Biometric service, the system pool performs the following actions:</span></span>

-   <span data-ttu-id="97334-119">O modo de navegação do sensor está desabilitado.</span><span class="sxs-lookup"><span data-stu-id="97334-119">The navigation mode for the sensor is disabled.</span></span>
-   <span data-ttu-id="97334-120">As operações não solicitadas são enviadas para o manipulador de eventos, independentemente do foco da janela.</span><span class="sxs-lookup"><span data-stu-id="97334-120">Unclaimed operations are sent to the event handler regardless of window focus.</span></span>
-   <span data-ttu-id="97334-121">Se não houver nenhuma solicitação pendente para uma operação biométrica, uma captura automática será executada.</span><span class="sxs-lookup"><span data-stu-id="97334-121">If there are no outstanding requests for a biometric operation, an automatic capture is performed.</span></span>

<span data-ttu-id="97334-122">Se um manipulador de navegação tiver sido registrado com o serviço biométrico do Windows, o pool do sistema fará o seguinte:</span><span class="sxs-lookup"><span data-stu-id="97334-122">If a navigation handler has been registered with the Windows Biometric service, the system pool does the following:</span></span>

-   <span data-ttu-id="97334-123">As unidades biométricas no pool do sistema serão colocadas em um estado de navegação se nenhuma operação biométrica estiver pendente.</span><span class="sxs-lookup"><span data-stu-id="97334-123">The biometric units in the system pool are placed into a navigation state if no biometric operations are pending.</span></span>
-   <span data-ttu-id="97334-124">Os eventos de navegação serão desabilitados se um aviso de evento de correspondência ou de registro for enviado por um cliente.</span><span class="sxs-lookup"><span data-stu-id="97334-124">Navigation events are disabled if a match or enrollment event notice is sent by a client.</span></span>
-   <span data-ttu-id="97334-125">Se um manipulador de eventos não reivindicado tiver sido registrado, os eventos de navegação serão desabilitados.</span><span class="sxs-lookup"><span data-stu-id="97334-125">If an unclaimed events handler has been registered, navigation events are disabled.</span></span>
