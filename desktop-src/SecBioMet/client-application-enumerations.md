# <a name="client-application-enumerations"></a>Enumerações de aplicativo cliente

Os seguintes tipos de enumeração são suportados pela API Windows Biometric Framework.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                      | Descrição                                                                                                                                       |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AÇÃO DE POLÍTICA \_ WINBIO ANTI \_ \_ SPOOF \_**](winbio-anti-spoof-policy-action.md)<br/>  | Especifica os tipos de ações que você toma para a política antispoofing de um usuário. <br/>                                                        |
| [**MÉTODO DE \_ NOTIFICAÇÃO ASSÍNCRONA WINBIO \_ \_**](/windows/desktop/api/Winbio/ne-winbio-winbio_async_notification_method)<br/> | Define constantes que especificam como as notificações de conclusão para operações assíncronas devem ser entregues ao aplicativo cliente.<br/> |
| [**FORMATO DE \_ CREDENCIAL WINBIO \_**](winbio-credential-format.md)<br/>                  | Define sinalizadores que podem ser usados para especificar o formato de credencial do usuário final.<br/>                                                              |
| [**ESTADO DA \_ CREDENCIAL \_ WINBIO**](winbio-credential-state.md)<br/>                    | Define valores que especificam se uma credencial foi associada aos dados biométricos de um usuário final.<br/>                          |
| [**TIPO DE \_ CREDENCIAL \_ WINBIO**](winbio-credential-type.md)<br/>                      | Define sinalizadores que podem ser usados para filtrar o tipo de credencial.<br/>                                                                       |
| [**FONTE DE POLÍTICA DO WINBIO \_ \_**](winbio-policy-source.md)<br/>                          | Lista as possíveis fontes de informações de política para a detecção de fraudes para fatores biométricos.<br/>                                  |
