# <a name="framework-security"></a>Segurança da estrutura

O Windows Biométrico permite que aplicativos cliente sem privilégios executem operações biométricas sem fornecer acesso direto a amostras biométricas ou hardware. O serviço cria um limite de segurança entre os clientes e os componentes biométricos, conforme mostrado na ilustração a seguir.

![limite de segurança entre os clientes e os componentes biométricos](images/bioframework-security.png)
