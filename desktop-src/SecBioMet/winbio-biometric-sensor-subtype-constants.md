# <a name="winbio_biometric_sensor_subtype-constants"></a>\_ \_ Constantes de subtipo de sensor BIOmétrico WINBIO \_

As constantes a seguir podem ser usadas como um bitmask para o parâmetro *Capabilities* da estrutura de [**\_ \_ esquema de unidade WINBIO**](winbio-unit-schema.md) . Eles se referem aos recursos do sensor integrado.



| Constante                                                                                                                                                                                                            | Descrição                                        |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| <span id="WINBIO_SENSOR_SUBTYPE_UNKNOWN"></span><span id="winbio_sensor_subtype_unknown"></span><dl> <dt>**subtipo de \_ sensor WINBIO \_ \_ desconhecido**</dt> </dl>     | Os subtipos de sensor não são conhecidos.<br/>      |
| <span id="WINBIO_FP_SENSOR_SUBTYPE_SWIPE"></span><span id="winbio_fp_sensor_subtype_swipe"></span><dl> <dt>**\_deslize o \_ \_ subtipo do \_ sensor de WINBIO FP**</dt> </dl> | O sensor dá suporte a passes de impressão digital.<br/> |
| <span id="WINBIO_FP_SENSOR_SUBTYPE_TOUCH"></span><span id="winbio_fp_sensor_subtype_touch"></span><dl> <dt>**\_toque de \_ \_ subtipo de \_ sensor de WINBIO FP**</dt> </dl> | O sensor dá suporte a toques de dedos.<br/>     |
