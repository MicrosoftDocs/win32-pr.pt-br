# <a name="winbio_ansi_385_face-constants"></a>Constantes de face do WINBIO \_ ANSI \_ 385 \_

As seguintes constantes são valores de **\_ \_ subtipo biométrico WINBIO** que podem ser usados para especificar os dois tipos de imagens de face frontais, conforme definido pelo ANSI INCITS 385-2004: "formato de reconhecimento facial para intercâmbio de dados": resolução completa e baixa resolução. Na prática, a estrutura biométrica usará apenas imagens de resolução completa para reconhecimento facial.



| Constante/valor                                                                                                                                                                                                                                                                          | Descrição                                                |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------|
| <span id="WINBIO_ANSI_385_FACE_TYPE_UNKNOWN"></span><span id="winbio_ansi_385_face_type_unknown"></span><dl> <dt>**WINBIO \_ \_Tipo de \_ FACE ANSI 385 \_ \_ desconhecido**</dt> <dt>0</dt> </dl>    | O tipo de imagem de face frontais é desconhecido.<br/>         |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_FULL"></span><span id="winbio_ansi_385_face_frontal_full"></span><dl> <dt>**WINBIO \_ ANSI \_ 385 \_ face \_ frontais \_ Full**</dt> <dt>1</dt> </dl>    | O tipo de imagem de face frontais é resolução completa.<br/> |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_TOKEN"></span><span id="winbio_ansi_385_face_frontal_token"></span><dl> <dt>**WINBIO \_ \_Símbolo de \_ \_ frontais de \_ face ANSI 385**</dt> <dt>2</dt> </dl> | O tipo de imagem de face frontais é de baixa resolução.<br/>  |
