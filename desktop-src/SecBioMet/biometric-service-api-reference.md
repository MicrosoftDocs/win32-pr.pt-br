Windows Biometric Framework páginas de referência de API contêm descrições detalhadas de funções, estruturas e recursos que podem ser usados para criar aplicativos cliente e plug-ins. As páginas de referência são divididas nos seguintes grupos:

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                       | Descrição                                                                                                                                         |
|-----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [Referência de aplicativo cliente](client-application-reference.md)<br/> | Funções, estruturas e outros tipos que você pode usar para criar aplicativos cliente que interagem com o Windows Biometric Framework.<br/> |
| [Referência de plug-in](plug-in-reference.md)<br/>                       | Funções de adaptador, funções de wrapper e estruturas para criar adaptadores de plug-ins personalizados de três tipos: mecanismo, sensor e armazenamento.<br/>   |
