# <a name="creating-a-private-sensor-pool"></a>Criando um pool de sensor privado

Um pool de sensor privado é uma coleção de unidades biométricas reservadas para uso exclusivo por um aplicativo cliente. Os pools privados suportam métodos de autenticação proprietários e permitem que um aplicativo cliente acesse uma unidade biométrica usando comandos de controle especificados pelo fornecedor.

Os tópicos a seguir contêm exemplos de código que mostram como criar um pool de sensor privado.
