# <a name="winbio_orientation-constants"></a>Constantes de orientação de WINBIO \_

As constantes a seguir especificam as possíveis orientações de câmera que o componente de sensor especifica como obrigatório.



| Constante/valor                                                                                                                                                                                                                                                           | Descrição                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------|
| <span id="WINBIO_ORIENTATION_UNSPECIFIED"></span><span id="winbio_orientation_unspecified"></span><dl> <dt>**WINBIO \_ ORIENTAÇÃO \_ não especificada**</dt> <dt>0</dt> </dl> | Não foi especificada uma orientação obrigatória para a câmera.<br/> |
| <span id="WINBIO_ORIENTATION_LANDSCAPE"></span><span id="winbio_orientation_landscape"></span><dl> <dt>**WINBIO \_ \_Paisagem de orientação**</dt> <dt>1</dt> </dl>       | A orientação paisagem é necessária para a câmera.<br/>    |
| <span id="WINBIO_ORIENTATION_PORTRAIT"></span><span id="winbio_orientation_portrait"></span><dl> <dt>**WINBIO \_ ORIENTAÇÃO \_ retrato**</dt> <dt>2</dt> </dl>          | A orientação retrato é necessária para a câmera.<br/>     |
| <span id="WINBIO_ORIENTATION_ANY"></span><span id="winbio_orientation_any"></span><dl> <dt>**WINBIO \_ ORIENTAÇÃO de \_ qualquer**</dt> <dt>3</dt> </dl>                         | Qualquer orientação é permitida para a câmera.<br/>             |
