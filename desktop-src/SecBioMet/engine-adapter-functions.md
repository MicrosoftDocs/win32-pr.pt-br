# <a name="engine-adapter-functions"></a>Funções do adaptador do mecanismo

Um adaptador de mecanismo gera modelos biométricos de amostras capturadas, corresponde a amostras a modelos existentes e modelos de índices. As funções a seguir devem ser implementadas pelo desenvolvedor do adaptador. Eles são chamados pelo serviço de biometria do Windows.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                       | Descrição                                                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EngineAdapterCreateKey**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_create_key_fn)<br/>                                         | Chamado pelo Windows Biometric Framework para enviar por push uma chave HMAC para o sensor. O identificador de chave retornado será passado de volta para a unidade biométrica quando a estrutura chamar [**EngineAdapterIdentifyFeatureSetSecure**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_identify_feature_set_secure_fn). <br/>           |
| [**EngineAdapterAcceptSampleData**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_accept_sample_data_fn)<br/>                           | Aceita um exemplo biométrico bruto e extrai um conjunto de recursos.<br/>                                                                                                                                                                                                                     |
| [**EngineAdapterActivate**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_activate_fn)<br/>                                           | Dá ao adaptador do mecanismo a chance de executar qualquer trabalho necessário para colocar o componente do sensor fora de um estado ocioso.<br/>                                                                                                                                                             |
| [**EngineAdapterAttach**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_attach_fn)<br/>                                               | Adiciona um adaptador de mecanismo ao pipeline de processamento da unidade biométrica.<br/>                                                                                                                                                                                                       |
| [**EngineAdapterCheckForDuplicate**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_check_for_duplicate_fn)<br/>                         | Determina se um novo modelo no pipeline duplica qualquer modelo já salvo no banco de dados, independentemente da identidade associada aos modelos.<br/>                                                                                                              |
| [**EngineAdapterClearContext**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_clear_context_fn)<br/>                                   | Prepara o pipeline de processamento da unidade biométrica para uma nova operação.<br/>                                                                                                                                                                                                    |
| [**EngineAdapterCommitEnrollment**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_commit_enrollment_fn)<br/>                           | Finaliza o objeto de registro, converte-o em um modelo e salva o modelo no banco de dados.<br/>                                                                                                                                                                            |
| [**EngineAdapterControlUnit**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_control_unit_fn)<br/>                                     | Executa uma operação de controle definida pelo fornecedor que não requer privilégios elevados.<br/>                                                                                                                                                                                          |
| [**EngineAdapterControlUnitPrivileged**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_control_unit_privileged_fn)<br/>                 | Executa uma operação de controle definida pelo fornecedor que requer privilégio elevado.<br/>                                                                                                                                                                                                  |
| [**EngineAdapterCreateEnrollment**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_create_enrollment_fn)<br/>                           | Inicializa o objeto de registro no pipeline de unidade biométrica.<br/>                                                                                                                                                                                                              |
| [**EngineAdapterDeactivate**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_deactivate_fn)<br/>                                       | Dá ao adaptador do mecanismo a chance de executar qualquer trabalho necessário para colocar o componente do sensor em um estado ocioso.<br/>                                                                                                                                                                 |
| [**EngineAdapterDetach**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_detach_fn)<br/>                                               | Libera recursos específicos do adaptador anexados ao pipeline.<br/>                                                                                                                                                                                                                  |
| [**EngineAdapterDiscardEnrollment**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_discard_enrollment_fn)<br/>                         | Exclui informações de estado de registro intermediário do pipeline.<br/>                                                                                                                                                                                                           |
| [**EngineAdapterExportEngineData**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_export_engine_data_fn)<br/>                           | Recupera uma cópia do conjunto de recursos ou modelo processados mais recentemente do mecanismo em um registro de informações biométricas padrão.<br/>                                                                                                                                            |
| [**EngineAdapterGetEnrollmentHash**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_get_enrollment_hash_fn)<br/>                         | Recupera o hash do modelo de registro concluído no pipeline.<br/>                                                                                                                                                                                                       |
| [**EngineAdapterGetEnrollmentStatus**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_get_enrollment_status_fn)<br/>                     | Determina se o objeto de registro está pronto para ser confirmado no pipeline.<br/>                                                                                                                                                                                             |
| [**EngineAdapterIdentifyAll**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_identify_all_fn)<br/>                                     | Determina as identidades de qualquer pessoa que esteja atualmente no quadro da câmera.<br/>                                                                                                                                                                                                     |
| [**EngineAdapterIdentifyFeatureSet**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_identify_feature_set_fn)<br/>                       | Cria um modelo a partir do conjunto de recursos atual e localiza um modelo correspondente no banco de dados.<br/>                                                                                                                                                                                |
| [**EngineAdapterIdentifyFeatureSetSecure**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_identify_feature_set_secure_fn)<br/>           | Chamado pelo Windows Biometric Framework para criar um modelo a partir do conjunto de recursos atual e localizar um modelo correspondente no banco de dados. Se uma correspondência puder ser encontrada, o adaptador do mecanismo deverá preencher os campos *identidade*, *subfator*, *autorização* e *AuthorizationSize* .<br/> |
| [*EngineAdapterNotifyPowerChange*](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_notify_power_change_fn)<br/>                           | Recebe uma notificação sobre uma alteração no estado de energia do computador e prepara o adaptador do mecanismo de acordo.<br/>                                                                                                                                                                  |
| [**EngineAdapterPipelineCleanup**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_pipeline_cleanup_fn)<br/>                             | Dá ao adaptador do mecanismo a chance de executar qualquer limpeza que exija ajuda do adaptador de armazenamento.<br/>                                                                                                                                                                        |
| [**EngineAdapterPipelineInit**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_pipeline_init_fn)<br/>                                   | Dá ao adaptador do mecanismo a chance de executar qualquer inicialização que permaneça incompleta.<br/>                                                                                                                                                                                     |
| [**EngineAdapterQueryCalibrationData**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_query_calibration_data_fn)<br/>                   | Obtém um conjunto de dados de calibragem de captura de post do adaptador do mecanismo.<br/>                                                                                                                                                                                                           |
| [**EngineAdapterQueryExtendedEnrollmentStatus**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_query_extended_enrollment_status_fn)<br/> | Consulta a propriedade de **\_ \_ \_ \_ status de registro estendido da propriedade WINBIO** .<br/>                                                                                                                                                                                                       |
| [**EngineAdapterQueryExtendedInfo**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_query_extended_info_fn)<br/>                         | Determina os recursos e as limitações do componente de mecanismo biométrico.<br/>                                                                                                                                                                                                 |
| [**EngineAdapterQueryHashAlgorithms**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_query_hash_algorithms_fn)<br/>                     | Recupera uma matriz de identificadores de objeto que representam os algoritmos de hash com suporte no adaptador do mecanismo.<br/>                                                                                                                                                                   |
| [**EngineAdapterQueryIndexVectorSize**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_query_index_vector_size_fn)<br/>                   | Recupera o tamanho do vetor de índice usado pelo adaptador do mecanismo.<br/>                                                                                                                                                                                                             |
| [**EngineAdapterQueryPreferredFormat**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_query_preferred_format_fn)<br/>                   | Determina o formato de dados de entrada preferencial pelo adaptador do mecanismo.<br/>                                                                                                                                                                                                              |
| [**EngineAdapterQuerySampleHint**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_query_sample_hint_fn)<br/>                             | Recupera o número de amostras corretas exigidas pelo adaptador do mecanismo para construir um modelo de registro. <br/>                                                                                                                                                                   |
| [**EngineAdapterRefreshCache**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_refresh_cache_fn)<br/>                                   | Notifica o adaptador do mecanismo de que ele deve descartar todos os modelos armazenados em cache que possam estar sendo mantidos na memória.<br/>                                                                                                                                                                      |
| [**EngineAdapterSelectCalibrationFormat**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_select_calibration_format_fn)<br/>             | Chamado pelo Windows Biometric Framework para determinar quais dos formatos de calibragem dos adaptadores do sensor o adaptador do mecanismo deseja usar.<br/>                                                                                                                                      |
| [**EngineAdapterSetAccountPolicy**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_set_account_policy_fn)<br/>                           | Define o padrão estendido e as políticas de antifalsificação por usuário usadas pelo adaptador do mecanismo.<br/>                                                                                                                                                                                       |
| [**EngineAdapterSetEnrollmentParameters**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_set_enrollment_parameters_fn)<br/>             | Fornece ao adaptador do mecanismo informações adicionais sobre uma operação de registro.<br/>                                                                                                                                                                                                 |
| [**EngineAdapterSetEnrollmentSelector**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_set_enrollment_selector_fn)<br/>                 | Informa ao adaptador do mecanismo qual pessoa rastrear para a operação de registro atual.<br/>                                                                                                                                                                                           |
| [**EngineAdapterSetHashAlgorithm**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_set_hash_algorithm_fn)<br/>                           | Seleciona um algoritmo de hash para uso em operações subsequentes.<br/>                                                                                                                                                                                                                     |
| [**EngineAdapterUpdateEnrollment**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_update_enrollment_fn)<br/>                           | Adiciona o conjunto de recursos atual ao objeto de registro.<br/>                                                                                                                                                                                                                         |
| [**EngineAdapterVerifyFeatureSet**](/windows/desktop/api/Winbio_adapter/nc-winbio_adapter-pibio_engine_verify_feature_set_fn)<br/>                           | Compara o modelo no conjunto de recursos atual com um modelo específico no banco de dados.<br/>                                                                                                                                                                                     |
| [**WbioQueryEngineInterface**](/windows/desktop/api/Winbio_adapter/nf-winbio_adapter-wbioqueryengineinterface)<br/>                                     | Recupera um ponteiro para a estrutura da [**\_ \_ interface do mecanismo WINBIO**](/windows/desktop/api/Winbio_adapter/ns-winbio_adapter-winbio_engine_interface) para o adaptador do mecanismo.<br/>                                                                                                                                                      |
