# <a name="windows-biometric-framework"></a>Windows Biometric Framework

## <a name="purpose"></a>Finalidade

Você pode usar a API Windows Biometric Framework para criar aplicativos cliente que capturam, salvam e comparam informações biométricas do usuário final com segurança.

## <a name="developer-audience"></a>Público de desenvolvedores

Os desenvolvedores que usam essa API devem estar familiarizados com as linguagens de programação C e C++ e com o ambiente de programação baseado no Windows.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

A API Windows Biometric Framework tem suporte a partir do Windows Server 2008 R2 e do Windows 7. Para obter informações sobre os requisitos de tempo de execução para um determinado elemento de programação, consulte a seção requisitos da página de referência para esse elemento.
