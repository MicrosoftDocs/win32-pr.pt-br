# <a name="windows-biometric-framework"></a>Windows Biometric Framework

## <a name="purpose"></a>Finalidade

Você pode usar a API Windows Biometric Framework para criar aplicativos cliente que capturam, salvam e comparam com segurança as informações biométricas do usuário final.

## <a name="developer-audience"></a>Público de desenvolvedores

Os desenvolvedores que usam essa API devem estar familiarizados com as linguagens de programação C e C++ e o Windows de programação baseada em programação.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O Windows API do Biometric Framework tem suporte a partir do Windows Server 2008 R2 e Windows 7. Para obter informações sobre os requisitos de tempo de executar para um elemento de programação específico, consulte a seção Requisitos da página de referência para esse elemento.
