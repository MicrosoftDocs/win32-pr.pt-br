# <a name="interaction-context-enumerations"></a>Enumerações de contexto de interação

Os tópicos nesta seção fornecem as especificações de referência para [enumerações de Contexto de](interaction-context-portal.md) Interação.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|---|---|
| [**SINALIZADORES \_ \_ DE SLIDES CRUZADOS**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_flags)<br/> | Especifica o estado da interação entre slides.<br/> |
| [**LIMITE \_ DE SLIDE \_ CRUZADO**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_threshold)<br/> | Especifica os limites de comportamento entre slides.<br/> |
| [**PARÂMETRO \_ INÉRCIA**](/windows/win32/api/interactioncontext/ne-interactioncontext-inertia_parameter)<br/> | Especifica os valores de inércia para uma manipulação (conversão, rotação, dimensionamento).<br/> |
| [**SINALIZADORES \_ DE \_ CONFIGURAÇÃO DE INTERAÇÃO**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_configuration_flags)<br/> | Especifica as interações a habilitar ao configurar um objeto [de Contexto de](interaction-context-portal.md) Interação.<br/> |
| [**PROPRIEDADE DE \_ CONTEXTO \_ DE INTERAÇÃO**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_context_property)<br/> | Especifica as propriedades do objeto [De contexto de](interaction-context-portal.md) interação. <br/> |
| [**SINALIZADORES \_ DE INTERAÇÃO**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_flags)<br/> | Especifica o estado de uma interação.<br/> |
| [**\_ID DE INTERAÇÃO**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_id)<br/> | Especifica os estados de interação usados para configurar um objeto [de Contexto de](interaction-context-portal.md) Interação. As interações podem ser estáticas (contato único sem manipulação, como toque, toque duplo, toque à direita, pressionar e segurar) ou dinâmica (um ou mais contatos com manipulação, como tradução, rotação ou dimensionamento).<br/> |
| [**ESTADO DE \_ INTERAÇÃO**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_state)<br/> | Especifica o estado do objeto [De contexto de](interaction-context-portal.md) interação.<br/> |
| [**ESTADO \_ DOS TRILHOS DE \_ MANIPULAÇÃO**](/windows/win32/api/interactioncontext/ne-interactioncontext-manipulation_rails_state)<br/> | Especifica os estados de trilho para uma interação.<br/> |
| [**PARÂMETRO DE \_ RODA DO \_ MOUSE**](/windows/win32/api/interactioncontext/ne-interactioncontext-mouse_wheel_parameter)<br/> | Especifica as manipulações que podem ser mapeadas para a rotação da roda do mouse.<br/> |
