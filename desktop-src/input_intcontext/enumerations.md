# <a name="interaction-context-enumerations"></a>Enumerações de contexto de interação

Os tópicos nesta seção fornecem as especificações de referência para enumerações de [contexto de interação](interaction-context-portal.md) .

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|---|---|
| [**\_sinalizadores de slide cruzado \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_flags)<br/> | Especifica o estado da interação entre slides.<br/> |
| [**\_limite de slide cruzado \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_threshold)<br/> | Especifica os limites de comportamento entre slides.<br/> |
| [**\_parâmetro inércia**](/windows/win32/api/interactioncontext/ne-interactioncontext-inertia_parameter)<br/> | Especifica os valores de inércia para uma manipulação (translação, rotação, dimensionamento).<br/> |
| [**\_sinalizadores de configuração de interação \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_configuration_flags)<br/> | Especifica as interações a serem habilitadas ao configurar um objeto de [contexto de interação](interaction-context-portal.md) .<br/> |
| [**\_propriedade de contexto de interação \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_context_property)<br/> | Especifica as propriedades do objeto de [contexto de interação](interaction-context-portal.md) . <br/> |
| [**sinalizadores de interação \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_flags)<br/> | Especifica o estado de uma interação.<br/> |
| [**ID de interação \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_id)<br/> | Especifica os Estados de interação usados para configurar um objeto de [contexto de interação](interaction-context-portal.md) . As interações podem ser estáticas (contato único sem manipulação, como toque, toque duplo, toque com o botão direito, pressionar e manter pressionado) ou dinâmico (um ou mais contatos com manipulação, como tradução, rotação ou dimensionamento).<br/> |
| [**Estado de interação \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_state)<br/> | Especifica o estado do objeto de [contexto de interação](interaction-context-portal.md) .<br/> |
| [**Estado dos trilhos de manipulação \_ \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-manipulation_rails_state)<br/> | Especifica os Estados do trilho para uma interação.<br/> |
| [**\_parâmetro de roda do mouse \_**](/windows/win32/api/interactioncontext/ne-interactioncontext-mouse_wheel_parameter)<br/> | Especifica as manipulações que podem ser mapeadas para a rotação da roda do mouse.<br/> |
