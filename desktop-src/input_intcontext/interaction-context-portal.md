# <a name="interaction-context"></a>Contexto de interação

## <a name="purpose"></a>Finalidade

Os tópicos nesta seção fornecem uma visão geral do suporte ao contexto de interação no Windows 8. O contexto de interação permite que os aplicativos do Windows 8 e as estruturas de interface do usuário ofereçam suporte a várias interações simultâneas fornecendo detecção de gestos e processamento de manipulação.

A coleção de tópicos fornecida aqui descreve como seu aplicativo pode usar o contexto de interação para otimizar a experiência do usuário para o cliente.

## <a name="developer-audience"></a>Público de desenvolvedores

As APIs de contexto de interação são projetadas para desenvolvedores que criam estruturas de interface do usuário que fornecem uma experiência de usuário consistente e otimizada para toque em aplicativos de área de trabalho.
