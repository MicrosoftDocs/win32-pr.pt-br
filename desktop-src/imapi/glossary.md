# <a name="glossary-imapi"></a>Glossário (IMAPi)

<dl> <dt>

<span id="imapi.imapi2_glossary_ANSI"></span><span id="imapi.imapi2_glossary_ansi"></span><span id="IMAPI.IMAPI2_GLOSSARY_ANSI"></span>**ANSI**
</dt> <dd>

O American National Standards Institute.

</dd> <dt>

<span id="imapi.imapi2_glossary_BurnEngine"></span><span id="imapi.imapi2_glossary_burnengine"></span><span id="IMAPI.IMAPI2_GLOSSARY_BURNENGINE"></span>**Mecanismo de gravação**
</dt> <dd>

Um componente que lê dados de um fluxo de dados fornecido (por exemplo, IStream ou IO. Stream) e grava esses dados em um dispositivo de CD/DVD. O componente deve considerar o buffer disponível na unidade, mas não contém nenhum conhecimento do formato dos dados, preparar a unidade para gravação ou informar a unidade que o processo de gravação concluiu.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDDVDDevice"></span><span id="imapi.imapi2_glossary_cddvddevice"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDDVDDEVICE"></span>**Dispositivo de CD/DVD**
</dt> <dd>

Um dispositivo que se reporta como um dispositivo "tipo 5". Isso inclui CD-ROM, DVD-ROM e a maioria das unidades de gravador de CD e DVD. Além disso, algumas unidades com mídia magnética (como o IOMEGA REV) se reportam como dispositivos "tipo 5" e aparecem como dispositivos de CD-ROM no Windows.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDG"></span><span id="imapi.imapi2_glossary_cdg"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDG"></span>**CD-G** 
</dt> <dd>

Um acrônimo para gráficos de CD, tecnologia exclusiva para CDs de áudio que permite que gráficos de baixa taxa de bits sejam adicionados a uma faixa de áudio. Essa tecnologia é usada com mais frequência em CDs usados para o karaokê.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_R"></span><span id="imapi.imapi2_glossary_cd_r"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_R"></span>**CD-R**
</dt> <dd>

Um CD de gravação única.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RandRW"></span><span id="imapi.imapi2_glossary_cd_randrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RANDRW"></span>**CD-R/RW**
</dt> <dd>

Um CD-R ou CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_ROM"></span><span id="imapi.imapi2_glossary_cd_rom"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_ROM"></span>**CD-ROM**
</dt> <dd>

Um CD somente leitura.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RW"></span><span id="imapi.imapi2_glossary_cd_rw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RW"></span>**CD-RW**
</dt> <dd>

Um CD regravável.

</dd> <dt>

<span id="imapi.imapi2_glossaryCDText"></span><span id="imapi.imapi2_glossarycdtext"></span><span id="IMAPI.IMAPI2_GLOSSARYCDTEXT"></span>**Texto do CD** 
</dt> <dd>

Uma tecnologia exclusiva para CDs de áudio que habilita informações de taxa de bits baixa, como nome do artista e nome da música a ser armazenado na mídia física. Aplicativos ou dispositivos que reproduzem o CD resultante podem então recuperar essas informações, se existir, para exibir para o usuário.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disc"></span><span id="imapi.imapi2_glossary_disc"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISC"></span>**Discos**
</dt> <dd>

Um termo genérico para mídia óptica redonda, como um CD-ROM ou um DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DiscAtOnce"></span><span id="imapi.imapi2_glossary_discatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISCATONCE"></span>**Disco em uma única vez** 
</dt> <dd>

Um método de gravação na mídia de CD em que todo o conteúdo da imagem de gravação é gravado no disco em um fluxo contínuo. Além disso, os dados não acessíveis, como o lead-in e o LEADOUT, são incluídos na imagem de gravação.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disk"></span><span id="imapi.imapi2_glossary_disk"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISK"></span>**Disco**
</dt> <dd>

Um termo genérico para mídia magnética redonda, como um disquete ou disco rígido.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dispatch"></span><span id="imapi.imapi2_glossary_dispatch"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISPATCH"></span>**Enviar** 
</dt> <dd>

Uma designação de interface COM em que uma determinada interface dá suporte ao método **IDispatch** de chamar uma função. Consulte o MSDN para obter mais informações sobre este tópico COM avançado.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dual"></span><span id="imapi.imapi2_glossary_dual"></span><span id="IMAPI.IMAPI2_GLOSSARY_DUAL"></span>**Simplifica** 
</dt> <dd>

Uma designação de interface COM em que uma determinada interface dá suporte aos métodos VTable e **IDispatch** de chamar uma função. Consulte o MSDN para obter mais informações sobre esses tópicos COM avançados.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVD"></span><span id="imapi.imapi2_glossary_dvd"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVD"></span>**DVD**
</dt> <dd>

Um disco versátil digital. originalmente, essa abreviação significava "disco de vídeo digital".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusR"></span><span id="imapi.imapi2_glossary_dvdplusr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSR"></span>**DVD + R**
</dt> <dd>

Um DVD de gravação única. O acrônimo é pronunciado "DVD Plus R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRDL"></span><span id="imapi.imapi2_glossary_dvdplusrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRDL"></span>**DVD + R DL**
</dt> <dd>

Um disco DVD + R de camada dupla. Esse tipo de disco também é conhecido como "DVD + R Double Layer".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRRW"></span><span id="imapi.imapi2_glossary_dvdplusrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRRW"></span>**DVD + R/RW**
</dt> <dd>

Um disco DVD + R e um disco DVD + RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRW"></span><span id="imapi.imapi2_glossary_dvdplusrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRW"></span>**DVD + RW**
</dt> <dd>

Um DVD regravável. O acrônimo é pronunciado "DVD Plus RW".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashR"></span><span id="imapi.imapi2_glossary_dvdplusdashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHR"></span>**DVD ± R**
</dt> <dd>

Um disco DVD + R e um disco DVD-R.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRRW"></span>**DVD ± R/RW**
</dt> <dd>

Um disco DVD + R, um disco DVD-R, um disco DVD + RW e um disco DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRW"></span>**DVD ± RW**
</dt> <dd>

Um disco DVD + RW e DVD + R Media

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashR"></span><span id="imapi.imapi2_glossary_dvddashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHR"></span>**DVD-R**
</dt> <dd>

Um DVD de gravação única. O acrônimo é pronunciado "DVD Dash R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRDL"></span><span id="imapi.imapi2_glossary_dvddashrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRDL"></span>**DVD-R DL**
</dt> <dd>

Uma mídia de DVD-R de camada dupla.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDRAM"></span><span id="imapi.imapi2_glossary_dvdram"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDRAM"></span>**DVD-RAM**
</dt> <dd>

Um DVD gravável aleatoriamente. Esse tipo de mídia dá suporte ao gerenciamento de defeitos de hardware que identifica e evita o uso de faixas danificadas. As unidades que dão suporte a esse formato são mais predominantes no Japão do que nos EUA.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDROM"></span><span id="imapi.imapi2_glossary_dvdrom"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDROM"></span>**DVD-ROM**
</dt> <dd>

Um DVD somente leitura.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRRW"></span><span id="imapi.imapi2_glossary_dvddashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRRW"></span>**DVD-R/RW**
</dt> <dd>

Um disco DVD-R e DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRW"></span><span id="imapi.imapi2_glossary_dvddashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRW"></span>**DVD-RW**
</dt> <dd>

Um DVD regravável. O acrônimo é pronunciado "DVD Dash RW".

</dd> <dt>

<span id="imapi.imapi2_glossary_ECMA"></span><span id="imapi.imapi2_glossary_ecma"></span><span id="IMAPI.IMAPI2_GLOSSARY_ECMA"></span>**ECMA** 
</dt> <dd>

A associação de fabricantes de computadores da Europa é uma associação do setor fundada em 1961 que é dedicada à padronização de informações e sistemas de comunicação. A ECMA edita padrões e relatórios técnicos. Todas as publicações do ECMA estão disponíveis gratuitamente.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPI"></span><span id="imapi.imapi2_glossary_imapi"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPI"></span>**IMAPi** 
</dt> <dd>

A API de Mestração de imagem usada para criar e gravar imagens mestras (em oposição a um Live FS) para dispositivos de CD/DVD. Esse termo se aplica genericamente a todas as versões da API de mestre de imagem.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv1"></span><span id="imapi.imapi2_glossary_imapiv1"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV1"></span>**IMAPIv1** 
</dt> <dd>

a versão do imapi primeiro exposta no Windows XP. O IMAPi 1,0 dá suporte a mídia CD-R e CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv2"></span><span id="imapi.imapi2_glossary_imapiv2"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV2"></span>**IMAPIv2** 
</dt> <dd>

a segunda versão do imapi, lançada com o Windows Vista.

</dd> <dt>

<span id="imapi.imapi2_glossary_INCITS"></span><span id="imapi.imapi2_glossary_incits"></span><span id="IMAPI.IMAPI2_GLOSSARY_INCITS"></span>**INCITS**
</dt> <dd>

O Comitê Internacional para padrões de tecnologia da informação, uma organização reconhecida e aprovada por ANSI.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISO9660"></span><span id="imapi.imapi2_glossary_iso9660"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISO9660"></span>**ISO9660**
</dt> <dd>

Um sistema de arquivos projetado para a primeira geração de mídia de CD. Este sistema de arquivos tem limitações de classificação e nome de arquivo significativas.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISOImage"></span><span id="imapi.imapi2_glossary_isoimage"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISOIMAGE"></span>**Imagem ISO**
</dt> <dd>

Um nome comum para uma imagem de setor por setor de toda a área de dados de um disco de CD ou DVD de dados. O nome cresceu do uso comum de ISO9660 como o sistema de arquivos para mídia de CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_Joliet"></span><span id="imapi.imapi2_glossary_joliet"></span><span id="IMAPI.IMAPI2_GLOSSARY_JOLIET"></span>**Joliet**
</dt> <dd>

Um conjunto de extensões padronizadas para ISO9660 que dá suporte a nomes de formato UNICODE de até 64 caracteres de comprimento. Consulte a especificação ECMA número 119 para obter detalhes.

</dd> <dt>

<span id="imapi.imapi2_glossary_MMC"></span><span id="imapi.imapi2_glossary_mmc"></span><span id="IMAPI.IMAPI2_GLOSSARY_MMC"></span>**CONSOLE**
</dt> <dd>

O conjunto de comandos de multimídia que é definido pelo grupo de trabalho T10 de INCITS. O MMC define comandos de pacote para dispositivos de CD e DVD. Os rascunhos de trabalho atuais do MMC podem ser encontrados no site da T10.

</dd> <dt>

<span id="imapi.imapi2_glossary_MtFuji"></span><span id="imapi.imapi2_glossary_mtfuji"></span><span id="IMAPI.IMAPI2_GLOSSARY_MTFUJI"></span>**Mt. Fuji**
</dt> <dd>

Um conjunto de comandos para controlar dispositivos multimídia, semelhante ao conjunto de comandos do MMC. Alguns itens (como recursos ou tipos de mídia) são descritos no Mt. Os documentos de São Paulo, enquanto outros são descritos pela primeira vez nos documentos do MMC. Os rascunhos de trabalho atuais do conjunto de comandos podem ser encontrados [em ftp://ftp.avc-pioneer.com](ftp://ftp.avc-pioneer.com/).

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSI"></span><span id="imapi.imapi2_glossary_scsi"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSI"></span>**Scsi**
</dt> <dd>

A interface do sistema de computador pequeno.

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSICommand"></span><span id="imapi.imapi2_glossary_scsicommand"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSICOMMAND"></span>**Comando SCSI**
</dt> <dd>

Qualquer um de vários comandos de pacote, como aqueles definidos no documento do MMC. Esses comandos podem ser enviados usando qualquer mecanismo de transporte apropriado, como SCSI, ATAPI, USB ou IEEE1394.

</dd> <dt>

<span id="imapi.imapi2_glossary_SessonAtOnce"></span><span id="imapi.imapi2_glossary_sessonatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_SESSONATONCE"></span>**Sessão ao mesmo tempo**
</dt> <dd>

Um método de criação de um disco CD Plus. Uma sessão de gravação registra faixas de áudio no disco e uma segunda sessão grava uma única faixa CD-XA no disco e fecha o disco.

</dd> <dt>

<span id="imapi.imapi2_glossary_TrackAtOnce"></span><span id="imapi.imapi2_glossary_trackatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_TRACKATONCE"></span>**Acompanhar de uma vez**
</dt> <dd>

Um método de gravação na mídia de CD em que os dados de uma única faixa são gravados ao mesmo tempo. Quando todas as faixas são escritas, a sessão é fechada como uma etapa final que envolve a escrita das áreas de lead-in e de saída do disco.

</dd> <dt>

<span id="imapi.imapi2_glossary_UDF"></span><span id="imapi.imapi2_glossary_udf"></span><span id="IMAPI.IMAPI2_GLOSSARY_UDF"></span>**Formato de disco universal (UDF)**
</dt> <dd>

Esse sistema de arquivos é definido pelo OSTA (Associação de Tecnologia Armazenamento Óptico) que dá suporte a discos de sessão única e de várias sessões.

</dd> <dt>

<span id="imapi.imapi2_glossary_VTable"></span><span id="imapi.imapi2_glossary_vtable"></span><span id="IMAPI.IMAPI2_GLOSSARY_VTABLE"></span>**Tabelav** 
</dt> <dd>

Uma designação de interface COM em que uma determinada interface dá suporte ao método vtable de chamar uma função. Consulte MSDN para obter mais informações sobre este tópico COM avançado.

</dd> </dl>
