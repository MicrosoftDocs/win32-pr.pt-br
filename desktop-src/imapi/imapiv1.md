# <a name="imapiv1-support"></a>Suporte do IMAPIv1

A API de mestre de imagem versão 1 (IMAPIv1) permite que um estágio de aplicativo e grave uma imagem simples de áudio ou de dados em dispositivos CD-R e CD-RW. A API dá suporte aos formatos Joliet e ISO 9660 para discos de áudio e dados Redbook.

> [!Note]  
> O IMAPi foi completamente reprojetado para a versão 2,0. A maior parte da funcionalidade da versão 1,0 está disponível na versão 2,0. Os aplicativos de escrita de mestre de imagem ou a execução de novos dispositivos e desenvolvimento de formato são incentivados a usar a [versão 2,0](using-imapi.md) em vez da versão 1,0. Se você executar um aplicativo da versão 1,0 no Windows Vista, as interfaces da versão 1,0 converterão essas chamadas na versão apropriada do 2,0 chamadas para obter funcionalidades idênticas.

 

Para obter mais informações sobre como usar a versão 1,0 no Windows XP, consulte os seguintes tópicos:

-   [Sobre a API de mestre de imagem](about-the-image-mastering-api.md)
-   [Usando a API de mestre de imagem](using-the-image-mastering-api.md)
-   [Referência de IMAPIv1](imapiv1-reference.md)
