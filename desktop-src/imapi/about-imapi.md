# <a name="about-imapi"></a>Sobre o IMAPi

O IMAPi permite a criação de mídia óptica usando uma variedade de sistemas de arquivos.

## <a name="features"></a>Recursos

O IMAPi fornece os seguintes recursos:

-   Cria e grava discos de sessão única e várias sessões.
-   Cria e grava discos inicializáveis.
-   Cria e grava imagens de disco de CD BRUTOs.
-   Fornece acesso de baixo nível ao mecanismo de gravação para o desenvolvimento de suporte para novos dispositivos.
-   Fornece acesso às propriedades do gravador estendido para o desenvolvimento de suporte para novos dispositivos.
-   Fornece opções de verificação de gravação.

O IMAPi oferece suporte aos seguintes formatos. Para obter mais detalhes, consulte [formatos de disco](disc-formats.md).

-   Formato de imagem ISO9660 (inclui áudio Redbook).
-   Formato Joliet.
-   UDF (formato de disco universal).

O IMAPi dá suporte aos seguintes dispositivos de mídia.

-   CD-R gravável. Anteriormente conhecido como CD Write Once.
-   CD regravável (CD-RW)
-   DVD gravável (DVD-R e DVD + R).
-   DVD regravável (DVD-RW e DVD + RW).
-   DVD-RAM de acesso aleatório.
-   DVD de camada dupla gravável (em formatos DVD-R e DVD + R).
-   BD gravável (BD-R)
-   BD regravável (BD-RE)
-   Grava em outra mídia semelhante a um disco.
