# <a name="msftstreamzero-object"></a>Objeto MsftStreamZero

Use esse objeto para criar um fluxo de dados somente leitura cujos dados são inicializados como zeros. Você deve chamar o método **SetSize** para definir o tamanho solicitado do fluxo.

## <a name="members"></a>Membros

O objeto **MsftStreamZero** não define nenhum membro.
