# <a name="bitlocker-drive-encryption-provider"></a>Fornecedor de Criptografia de Unidade de Disco BitLocker.

A interface do provedor WMI é para gerenciar e configurar o Criptografia de Unidade de Disco BitLocker (BDE) no Windows Server 2008 R2, no Windows Server 2008 e somente nas versões específicas do Windows 7, do Windows Vista Enterprise e do Windows Vista Ultimate. Ele deve ser lido por esses scripts de escrita, elementos de interface do usuário ou outras ferramentas administrativas para o BDE, bem como as pessoas que desejam entender mais sobre a funcionalidade do BDE.
