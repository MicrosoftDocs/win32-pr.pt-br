# <a name="firewall-dynamic-keywords-enumerations"></a>Enumerações de palavras-chave dinâmicas do firewall

As palavras-chave dinâmicas do firewall contêm as seguintes enumerações.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_ENUM_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_enum_flags) | Define constantes que especificam os tipos de objetos a serem incluídos em uma operação de enumeração. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_flags) | Define constantes que especificam como os endereços IP devem ser resolvidos. |
| [**FW_DYNAMIC_KEYWORD_ORIGIN_TYPE**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_origin_type) | Define constantes que especificam a origem do objeto de endereço de palavra-chave dinâmica em um [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0). |
