# <a name="firewall-dynamic-keywords-structures"></a>Estruturas de palavras-chave dinâmicas do firewall

As palavras-chave dinâmicas do firewall contêm as seguintes estruturas.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0) | Contém os dados retornados ao cliente quando as APIs **de Enumeração** são chamadas. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address0) | Permite que o cliente crie um endereço de palavra-chave dinâmico, que contém uma lista de endereços IP. |
