# <a name="multiple-threads"></a>Vários threads

Um *thread* é a entidade em um processo que pode ser agendada para execução. Todos os threads de um processo compartilham seu espaço de endereço virtual e os recursos do sistema. Cada processo é iniciado com um único thread, mas pode criar threads adicionais de qualquer um de seus threads.
