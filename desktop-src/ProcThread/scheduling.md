# <a name="scheduling"></a>Agendamento

O Agendador do sistema controla a multitarefa, determinando qual dos threads concorrentes recebe a próxima fatia de tempo do processador. O Agendador determina qual thread é executado em seguida usando prioridades de agendamento.
