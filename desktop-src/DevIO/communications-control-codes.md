# <a name="communications-control-codes"></a>Códigos de controle de comunicações

O código de controle a seguir é usado com entrada e saída do dispositivo.



| Código de controle                                                        | Operação                                                                           |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**IOCTL \_ SERIAL \_ LSRMST \_ INSERT**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_serial_lsrmst_insert) | Habilita ou desabilita o posicionamento de dados de status de linha e modem no fluxo de dados. |
