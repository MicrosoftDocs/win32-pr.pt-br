# <a name="about-communications-resources"></a>Sobre os recursos de comunicação

As funções de e/s (entrada e saída) de arquivo ( [**CreateFile**](/windows/desktop/api/fileapi/nf-fileapi-createfilea), [**CloseHandle**](/windows/desktop/api/handleapi/nf-handleapi-closehandle), [**ReadFile**](/windows/desktop/api/fileapi/nf-fileapi-readfile), [**ReadFileEx**](/windows/desktop/api/fileapi/nf-fileapi-readfileex), [**WriteFile**](/windows/desktop/api/fileapi/nf-fileapi-writefile)e [**WriteFileEx**](/windows/desktop/api/fileapi/nf-fileapi-writefileex)) fornecem a interface básica para abrir e fechar um identificador de recurso de comunicações e para executar operações de leitura e gravação. As funções de comunicação fornecem acesso aos recursos de comunicação. Esta visão geral descreve o uso de funções de e/s de arquivo e de comunicação.
