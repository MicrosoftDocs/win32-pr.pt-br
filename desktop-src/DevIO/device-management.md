# <a name="device-management"></a>Gerenciamento de dispositivo

O gerenciamento de dispositivos fornece uma maneira de notificar de forma uniforme todos os aplicativos e recursos do sistema de alterações que podem afetar a operação ou o acesso aos recursos. Os aplicativos e o sistema usam e processam eventos de dispositivo para aproveitar os novos recursos quando eles se tornam disponíveis e para evitar a perda de dados quando os recursos existentes se tornam indisponíveis.

-   [Usando o gerenciamento de dispositivos](using-device-management.md)
-   [Referência de gerenciamento de dispositivo](device-management-reference.md)
