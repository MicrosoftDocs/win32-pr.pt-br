# <a name="device-management"></a>Gerenciamento de dispositivo

O gerenciamento de dispositivos fornece uma maneira de notificar uniformemente todos os aplicativos e recursos do sistema de alterações que podem afetar sua operação ou o acesso aos recursos. Os aplicativos e o sistema usam e processam eventos de dispositivo para aproveitar novos recursos quando eles ficam disponíveis e para evitar a perda de dados quando os recursos existentes ficam indisponíveis.
