# <a name="about-device-management"></a>Sobre Gerenciamento de Dispositivos

O sistema usa mensagens para notificar aplicativos sobre alterações de dispositivo e alterações de energia. As mensagens do dispositivo notificam os aplicativos sobre eventos de alteração do dispositivo; as mensagens de energia notificam os aplicativos sobre eventos de gerenciamento de energia. Aplicativos e drivers também podem definir e usar mensagens personalizadas para habilitar a notificação de outros tipos de eventos. Esta visão geral fornece informações sobre os seguintes tópicos:

-   [Mensagens do dispositivo](device-messages.md)
-   [Tipos de evento de dispositivo](device-event-types.md)
-   [Notificações de dispositivo](device-notifications.md)
-   [Eventos de dispositivo](device-events.md)
-   [Controle de entrada e saída do dispositivo (IOCTL)](device-input-and-output-control-ioctl-.md)

Para obter mais informações sobre mensagens de energia, consulte [Power Management](/windows/desktop/Power/power-management-portal).
