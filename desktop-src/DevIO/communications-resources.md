# <a name="communications-resources"></a>Recursos de comunicação

Um *recurso de comunicação* é um dispositivo físico ou lógico que fornece um único fluxo de dados bidirecional e assíncrono. Portas seriais, portas paralelas, máquinas de fax e modems são exemplos de recursos de comunicação. Para cada recurso de comunicação, há um provedor de serviços, que consiste em uma biblioteca ou driver, que permite que os aplicativos acessem o recurso.

-   [Usando recursos de comunicação](using-communications-resources.md)
