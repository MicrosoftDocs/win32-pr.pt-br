# <a name="id3dx10meshbuffergetsize-method"></a>Método ID3DX10MeshBuffer:: GetSize

Obtenha o tamanho do buffer de malha, em bytes.

## <a name="syntax"></a>Sintaxe


```C++
SIZE_T GetSize();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **tamanho \_ T**](../winprog/windows-data-types.md)**

O tamanho do buffer de malha, em bytes.
