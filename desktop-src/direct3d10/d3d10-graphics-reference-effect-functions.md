# <a name="effect-functions-direct3d-10"></a>Funções de efeito (Direct3D 10)

Esta seção contém informações sobre as seguintes funções de efeito:



| Funções                                                                      | Descrição                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**D3D10CompileEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10compileeffectfrommemory)           | Compile um efeito.                                                  |
| [**D3D10CreateEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectfrommemory)             | Crie um efeito.                                                   |
| [**D3D10CreateEffectPoolFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectpoolfrommemory)     | Crie um pool de efeitos.                                              |
| [**D3D10CreateStateBlock**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createstateblock)                         | Crie um bloco de estado.                                               |
| [**D3D10DisassembleEffect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10disassembleeffect)                       | Desmonte um efeito compilado nas instruções do assembly do sombreador.    |
| [**D3D10StateBlockMaskDifference**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdifference)         | Combine duas máscaras de bloco de estado com um XOR bit a bit.                   |
| [**D3D10StateBlockMaskDisableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisableall)         | Desabilitar captura de estado.                                            |
| [**D3D10StateBlockMaskDisableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisablecapture) | Desabilite a captura de estado com uma máscara de bloco de estado.                    |
| [**D3D10StateBlockMaskEnableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenableall)           | Habilite uma máscara de bloco de estado para capturar e aplicar todas as variáveis de estado. |
| [**D3D10StateBlockMaskEnableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenablecapture)   | Habilite um intervalo de valores de estado em uma máscara de bloco de estado.               |
| [**D3D10StateBlockMaskGetSetting**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskgetsetting)         | Obter um elemento em uma máscara de bloco de estado.                               |
| [**D3D10StateBlockMaskIntersect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskintersect)           | Combine duas máscaras de bloco de estado com um AND bit a bit.                   |
| [**D3D10StateBlockMaskUnion**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskunion)                   | Combine duas máscaras de bloco de estado com um OR bit a bit.                    |
