# <a name="d3dx-enumerations-direct3d-10-graphics"></a>Enumerações D3DX (gráficos do Direct3D 10)

Esta seção contém informações sobre os seguintes tipos enumerados e sinalizadores usados com D3DX.:



| Enumeração                                                       | Descrição                                                                                                                                                                            |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Sinalizador de canal D3DX10 \_**](d3dx10-channel-flag.md)              | Esses sinalizadores são usados por funções que operam em um ou mais canais em uma textura.                                                                                                  |
| [**\_Otimização de CPU D3DX \_**](d3dx-cpu-optimization.md)          | Especifica o conjunto de instruções D3DX está atualmente otimizado para.                                                                                                                         |
| [**\_Erro D3DX10**](d3dx10-err.md)                                 | Os erros são representados por valores negativos e não podem ser combinados.                                                                                                                      |
| [**\_Sinalizador de filtro D3DX10 \_**](d3dx10-filter-flag.md)                | Sinalizadores de filtragem de textura.                                                                                                                                                               |
| [**\_Formato de \_ arquivo de imagem D3DX10 \_**](d3dx10-image-file-format.md)   | Descreve os formatos de arquivo de imagem com suporte.                                                                                                                                            |
| [**\_Malha D3DX10**](d3dx10-mesh.md)                               | Sinalizadores usados para especificar as opções de criação para uma malha.                                                                                                                                     |
| [**\_Sinalizadores de \_ descarte de malha D3DX10 \_**](d3dx10-mesh-discard-flags.md) | Especifica quais partes de dados de malha descartar do dispositivo. Usado com [**ID3DX10Mesh::D iscard**](id3dx10mesh-discard.md).                                                         |
| [**D3DX10 \_ MESHOPT**](d3dx10-meshopt.md)                         | Especifica o tipo de otimização de malha a ser executada.                                                                                                                               |
| [**\_Sinalizador D3DX10 NormalMap \_**](d3dx10-normalmap-flag.md)          | Esses sinalizadores são usados para controlar como o [**D3DX10ComputeNormalMap**](d3dx10computenormalmap.md) gera mapas normais. Qualquer número desses sinalizadores pode estar ou juntos em qualquer combinação. |
| [**Sinalizador de D3DX10 \_ Salvar \_ textura \_**](d3dx10-save-texture-flag.md)   | Opções de salvamento de textura.                                                                                                                                                                  |
