# <a name="effect-reference-direct3d-10-graphics"></a>Referência de efeito (gráficos do Direct3D 10)

A API do Direct3D define vários elementos de API para ajudá-lo a criar e gerenciar o sistema de efeitos.

-   [Interfaces](d3d10-graphics-reference-effect-interfaces.md)
-   [Funções](d3d10-graphics-reference-effect-functions.md)
-   [Estruturas](d3d10-graphics-reference-effect-structures.md)
-   [Constantes](d3d10-graphics-reference-effect-constants.md)

O sistema de efeitos encapsula as atribuições de estado em um efeito. Cada efeito é uma coleção de estado do pipeline que pode ser dividido em atribuições de estado usando blocos de estado, recursos e sombreadores. Dependendo do tamanho de um efeito, você pode optar por implementar um em uma cadeia de caracteres de texto ASCII ou um arquivo de efeito (. FX). Para organizar as informações em um efeito, consulte o [formato de efeito](d3d10-effect-format.md).

Tendo criado um efeito, use a API de efeito para definir o estado no dispositivo durante a renderização. Também, o sistema de efeitos dá suporte a uma série de APIs de reflexão para obter e definir dados de efeito. Para obter mais informações, consulte [interfaces do sistema de efeito (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
