# <a name="effect-reference-direct3d-10-graphics"></a>Referência de efeito (elementos gráficos do Direct3D 10)

A API do Direct3D define vários elementos de API para ajudá-lo a criar e gerenciar o sistema de efeito.

-   [Interfaces](d3d10-graphics-reference-effect-interfaces.md)
-   [Funções](d3d10-graphics-reference-effect-functions.md)
-   [Estruturas](d3d10-graphics-reference-effect-structures.md)
-   [Constantes](d3d10-graphics-reference-effect-constants.md)

O sistema de efeito encapsula as atribuições de estado em um efeito. Cada efeito é uma coleção de estado de pipeline que pode ser dividida em atribuições de estado usando blocos de estado, recursos e sombreadores. Dependendo do tamanho de um efeito, você pode optar por implementar um em uma cadeia de caracteres de texto ASCII ou em um arquivo de efeito (.fx). Para organizar informações em um efeito, consulte o formato [de efeito](d3d10-effect-format.md).

Depois de criar um efeito, use a API de efeito para definir o estado no dispositivo durante a renderização. Além disso, o sistema de efeito dá suporte a uma série de API de reflexão para obter e definir dados de efeito. Para obter mais informações, consulte Interfaces do sistema [de efeito (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
