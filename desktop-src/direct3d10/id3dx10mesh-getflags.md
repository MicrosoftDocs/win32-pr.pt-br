# <a name="id3dx10meshgetflags-method"></a>Método ID3DX10Mesh:: GetFlags

Acesse os sinalizadores de criação da malha.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetFlags();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Os sinalizadores de criação passados para o parâmetro *Options* de [**D3DX10CreateMesh**](d3d10-d3dx10createmesh.md) quando a malha foi criada. Consulte [**\_ malha D3DX10**](d3dx10-mesh.md).
