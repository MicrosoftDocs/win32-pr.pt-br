# <a name="shader-enumerations-direct3d-10-graphics"></a>Enumerações de sombreador (elementos gráficos do Direct3D 10)

Esta seção contém informações sobre as seguintes enumerações de sombreador:



| Enumeração                                                             | Descrição                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**\_Tipo de CBUFFER d3d10 \_**](/previous-versions/windows/desktop/legacy/bb204899(v=vs.85))                      | Esses sinalizadores identificam tipos de dados de buffer constante.                                          |
| [**\_Tipo de inclusão do D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_include_type)                      | Esses sinalizadores identificam o local de um arquivo de inclusão.                                     |
| [**Primitiva de D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_primitive)                             | Esses sinalizadores identificam tipos primitivos para um dispositivo Direct3D 10,1.                          |
| [**\_REGTYPE de depuração do sombreador d3d10 \_ \_**](/windows/win32/api/D3D10_1Shader/ne-d3d10_1shader-d3d10_shader_debug_regtype)     | Tipos de registro de sombreador.                                                                    |
| [**\_Escopo de depuração do sombreador d3d10 \_ \_**](/windows/win32/api/d3d10_1shader/ne-d3d10_1shader-d3d10_shader_debug_scopetype) | Tipos de escopo.                                                                              |
| [**Nome do D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_name)                                       | Esses sinalizadores identificam os parâmetros do sombreador que usam a semântica de valor do sistema.                   |
| [**\_Sinalizadores de CBUFFER do sombreador D3D \_ \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_cbuffer_flags)     | Esses sinalizadores identificam Propriedades de buffer de constantes.                                          |

| [**\_Tipo de entrada do sombreador D3D \_ \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_input_type)           | Esses sinalizadores identificam um tipo de recurso de sombreador.                                              |
| [**\_Classe de variável do sombreador D3D \_ \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_class)   | Esses sinalizadores identificam a classe Shader-Variable.                                           |
| [**\_Sinalizadores de variáveis do sombreador D3D \_ \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_flags)   | Esses sinalizadores identificam uma propriedade de variável de sombreador.                                          |
| [**\_Tipo de variável do sombreador D3D \_ \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_type)     | Esses sinalizadores identificam o tipo de variável de sombreador.                                            |
| [**\_Dimensão SRV do d3d10 \_**](/previous-versions/windows/desktop/legacy/bb172452(v=vs.85))                    | Esses sinalizadores identificam o tipo de recurso que será exibido para um dispositivo Direct3D 10,0. |
| [**D3D10 \_ SRV \_ DIMENSION1**](/previous-versions/windows/desktop/legacy/bb694535(v=vs.85))                  | Esses sinalizadores identificam o tipo de recurso que será exibido para um dispositivo Direct3D 10,1. |
