# <a name="id3dx10threadpumpgetworkitemcount-method"></a>Método ID3DX10ThreadPump:: GetWorkItemCount

Obter o número de itens de trabalho atualmente na bomba de thread.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetWorkItemCount();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

O número de itens de trabalho atualmente na fila na bomba do thread.
