# <a name="id3dx10dataloaderdestroy-method"></a>ID3DX10DataLoader: método estroy de:D

Usado por uma [**interface ID3DX10ThreadPump**](id3dx10threadpump.md) para destruir o carregador após a conclusão de um item de trabalho.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT Destroy();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

O valor de retorno é um dos valores listados nos [códigos de retorno do Direct3D 10](d3d10-graphics-reference-returnvalues.md).
