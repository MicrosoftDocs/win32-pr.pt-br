# <a name="id3dx10skininfogetmaxboneinfluences-method"></a>Método ID3DX10SkinInfo:: GetMaxBoneInfluences

Obter o número de vértices que um Bone pode influenciar de uma influência.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetMaxBoneInfluences();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

O número de vértices que um Bone pode influenciar no máximo.
