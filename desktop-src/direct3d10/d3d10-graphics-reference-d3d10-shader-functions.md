# <a name="shader-functions-direct3d-10-graphics"></a>Funções de sombreador (gráficos do Direct3D 10)

Esta seção contém informações sobre as seguintes funções de sombreador:



| Funções                                                                          | Descrição                                                                 |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**D3D10CompileShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10compileshader)                                   | Compilar um sombreador.                                                           |
| [**D3D10DisassembleShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10disassembleshader)                           | Preterido. Consulte [**D3DX10DisassembleShader**](d3dx10disassembleshader.md). |
| [**D3D10GetGeometryShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getgeometryshaderprofile)             | Obtenha o perfil Geometry-Shader mais adequado para um determinado dispositivo.              |
| [**D3D10GetInputAndOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputandoutputsignatureblob) | Obtenha um buffer que contém assinaturas de sombreador.                               |
| [**D3D10GetInputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputsignatureblob)                   | Obtenha um buffer que contém assinaturas de entrada de sombreador.                         |
| [**D3D10GetOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getoutputsignatureblob)                 | Obtenha um buffer que contém assinaturas de saída de sombreador.                        |
| [**D3D10GetPixelShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getpixelshaderprofile)                   | Obtenha o perfil do pixel shader mais adequado para um determinado dispositivo.                 |
| [**D3D10GetShaderDebugInfo**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getshaderdebuginfo)                         | Obter informações de depuração do sombreador de um sombreador compilado.                               |
| [**D3D10GetVertexShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getvertexshaderprofile)                 | Obtenha o perfil do sombreador de vértice mais adequado para um determinado dispositivo.                |
| [**D3D10PreprocessShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10preprocessshader)                             | Gere uma cadeia de caracteres de texto que contenha o fluxo do token do sombreador.               |
| [**D3D10ReflectShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10reflectshader)                                   | Preterido. Consulte [**D3DX10ReflectShader**](d3dx10reflectshader.md).         |
