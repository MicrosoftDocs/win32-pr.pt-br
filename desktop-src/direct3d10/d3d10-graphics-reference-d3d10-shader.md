# <a name="shader-reference-direct3d-10-graphics"></a>Referência de sombreador (gráficos do Direct3D 10)

A API do Direct3D define vários elementos de API para ajudá-lo a criar e gerenciar sombreadores programáveis.

-   [Interfaces do sombreador](d3d10-graphics-reference-d3d10-shader-interfaces.md)
-   [Funções de sombreador](d3d10-graphics-reference-d3d10-shader-functions.md)
-   [Estruturas do sombreador](d3d10-graphics-reference-d3d10-shader-structures.md)
-   [Enumerações do sombreador](d3d10-graphics-reference-d3d10-shader-enums.md)
-   [Constantes do sombreador](d3d10-graphics-reference-d3d10-shader-constants.md)

Sombreadores são programas executáveis que controlam os estágios do sombreador de vértice, do sombreador de geometria e do [pipeline](d3d10-graphics-programming-guide-pipeline-stages.md)do pixel shader. Os sombreadores são programados exclusivamente usando [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md).
