# <a name="direct3d-10-core-functions"></a>Funções principais do Direct3D 10

Esta seção contém informações sobre as seguintes funções principais:



| Funções                                                                | Descrição                                     |
|--------------------------------------------------------------------------|-------------------------------------------------|
| [**D3D10CreateBlob**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createblob)                               | Crie um buffer.                                |
| [**D3D10CreateDevice**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createdevice)                           | Crie um dispositivo Direct3D 10.                    |
| [**D3D10CreateDevice1**](/windows/desktop/api/D3D10_1/nf-d3d10_1-d3d10createdevice1)                         | Crie um dispositivo Direct3D 10,1.                  |
| [**D3D10CreateDeviceAndSwapChain**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createdeviceandswapchain)   | Crie um dispositivo Direct3D 10 e uma cadeia de permuta.   |
| [**D3D10CreateDeviceAndSwapChain1**](/windows/desktop/api/D3D10_1/nf-d3d10_1-d3d10createdeviceandswapchain1) | Crie um dispositivo Direct3D 10,1 e uma cadeia de permuta. |
