<span data-ttu-id="bd0a4-105">Um efeito DirectX é uma coleção de estado de pipeline, definida por expressões escritas [em HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md) e alguma sintaxe específica à estrutura de efeito.</span><span class="sxs-lookup"><span data-stu-id="bd0a4-105">A DirectX effect is a collection of pipeline state, set by expressions written in [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md) and some syntax that is specific to the effect framework.</span></span> <span data-ttu-id="bd0a4-106">Depois de compilar um efeito, use as APIs da estrutura de efeito para renderizar.</span><span class="sxs-lookup"><span data-stu-id="bd0a4-106">After compiling an effect, use the effect framework APIs to render.</span></span> <span data-ttu-id="bd0a4-107">A funcionalidade de efeito pode variar de algo tão simples quanto um sombreador de vértice que transforma geometria e um sombreador de pixel que saída de uma cor sólida, até uma técnica de renderização que requer várias passagens, usa cada estágio do pipeline gráfico e manipula o estado do sombreador, bem como o estado do pipeline não associado aos sombreadores programáveis.</span><span class="sxs-lookup"><span data-stu-id="bd0a4-107">Effect functionality can range from something as simple as a vertex shader that transforms geometry and a pixel shader that outputs a solid color, to a rendering technique that requires multiple passes, uses every stage of the graphics pipeline, and manipulates shader state as well as the pipeline state not associated with the programmable shaders.</span></span>
