# <a name="d3d12_gpu_virtual_address"></a>\_ \_ Endereço virtual da GPU D3D12 \_

Uma D3D12_GPU_VIRTUAL_ADDRESS representa um endereço virtual na memória da GPU. O tipo de dados é definido em d3d12. h da seguinte maneira:


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
