# <a name="interface-hierarchy"></a>Hierarquia de interface

O diagrama mostra a hierarquia de herança de interface.

A maioria dos aplicativos usará apenas as interfaces na parte superior da hierarquia e não usará diretamente as interfaces herdadas; [**ID3D12Pageable**](/windows/win32/api/d3d12/nn-d3d12-id3d12pageable), [**ID3D12DeviceChild**](/windows/desktop/api/d3d12/nn-d3d12-id3d12devicechild), [**ID3D12Object**](/windows/desktop/api/d3d12/nn-d3d12-id3d12object) e [**ID3D12CommandList**](/windows/desktop/api/d3d12/nn-d3d12-id3d12commandlist).

Novas interfaces são mostradas em uma caixa verde.
