# <a name="descriptor-tables"></a>Tabelas de descritores

Uma tabela de descritores é logicamente uma matriz de descritores.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                 | Descrição                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Visão geral das tabelas de descritores](descriptor-tables-overview.md)<br/>               | Cada tabela de descritores armazena os descritores de um ou mais tipos-SRVs, UAVe, CBVs e amostras. Uma tabela de descritores não é uma alocação de memória; é simplesmente um deslocamento e um comprimento em um heap de descritor.<br/> |
| [Usando tabelas de descritores](using-descriptor-tables.md)<br/>                     | As tabelas de descritores, cada uma identificando um intervalo em um heap de descritor, são associadas a Slots definidos pela assinatura raiz atual em uma lista de comandos. <br/>                                                               |
| [Uso avançado de tabelas de descritores](advanced-use-of-descriptor-tables.md)<br/> | As seções a seguir fornecem informações sobre o uso avançado de tabelas de descritores.<br/>                                                                                                                      |
