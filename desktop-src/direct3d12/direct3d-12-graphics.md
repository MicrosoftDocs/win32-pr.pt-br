# <a name="direct3d-12-graphics"></a>Gráficos do Direct3D 12

Este guia de programação contém informações sobre como usar o pipeline programável do Direct3D 12 para criar um mecanismo de gráficos personalizado.

Os cabeçalhos e as bibliotecas do Direct3D 12 fazem parte do SDK do Windows 10. Não há nenhum download ou instalação separado necessário para usar o Direct3D 12.
