# <a name="11on12-interfaces"></a>Interfaces 11on12
As seguintes interfaces são declaradas em d3d11on12. h.

## <a name="in-this-section"></a>Nesta seção
| Tópico | Descrição |
|-|-|
| [**ID3D11On12Device**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device) | Manipula a criação, o encapsulamento e a liberação de recursos de D3D11 para Direct3D 11on12. |
| [**ID3D11On12Device1**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1) | Permite uma melhor interoperabilidade com um componente que pode ser entregue a um dispositivo Direct3D 11, mas que deseja aproveitar o Direct3D 12 em vez disso. |
| [**ID3D11On12Device2**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device2) | Permite que você utilize recursos criados por meio das APIs do Direct3D 11 e use-os no Direct3D 12. Essa interface estende [ID3D11On12Device1](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1). |
