# <a name="call-parameter-structure"></a>Estrutura de parâmetros de chamada

Uma estrutura definida pelo usuário fornecida como um argumento Inout para uma chamada [**CallShader**](callshader-function.md) e como um parâmetro Inout para o [sombreador callable](callable-shader.md). O tipo de estrutura usado no sombreador callable deve corresponder à estrutura fornecida para a chamada **CallShader** correspondente.
