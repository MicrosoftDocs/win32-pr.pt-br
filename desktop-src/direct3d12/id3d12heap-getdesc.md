# <a name="id3d12heapgetdesc-method"></a>Método ID3D12Heap:: GetDesc

Obtém a descrição do heap.

## <a name="syntax"></a>Sintaxe


```C++
D3D12_HEAP_DESC GetDesc();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **D3D12 \_ heap \_ desc**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc)**

Retorna a [**estrutura \_ \_ desc de heap D3D12**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc) que descreve o heap.
