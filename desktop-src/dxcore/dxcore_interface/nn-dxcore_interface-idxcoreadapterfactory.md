# <a name="idxcoreadapterfactory-interface"></a>Interface IDXCoreAdapterFactory

A interface **IDXCoreAdapterFactory**   implementa métodos para gerar objetos de enumeração do adaptador DXCore e para recuperar seus detalhes. **IDXCoreAdapterFactory** herda da interface [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) . Para obter diretrizes de programação e exemplos de código, consulte [usando DXCore para enumerar adaptadores](../dxcore-enum-adapters.md).
