# <a name="dxcore-enumerations"></a>Enumerações de DXCore

As enumerações a seguir são declaradas em dxcore_interface. h.

## <a name="in-this-section"></a>Nesta seção

| Tópico e descrição |
|-|
| [**DXCoreAdapterPreference**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterpreference.md). Define constantes que especificam as preferências do adaptador DXCore a serem usadas como critérios de classificação de lista. |
| [**DXCoreAdapterProperty**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterproperty.md). Define constantes que especificam as propriedades do adaptador DXCore. |
| [**DXCoreAdapterState**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterstate.md). Define constantes que especificam tipos de Estados de adaptador DXCore. |
| [**DXCoreNotificationType**](./dxcore_interface/ne-dxcore_interface-dxcorenotificationtype.md). Define constantes que especificam tipos de notificações geradas por objetos [IDXCoreAdapter](./dxcore_interface/nn-dxcore_interface-idxcoreadapter.md) ou [IDXCoreAdapterList](./dxcore_interface/nn-dxcore_interface-idxcoreadapterlist.md) . |
| [**DXCoreSegmentGroup**](./dxcore_interface/ne-dxcore_interface-dxcoresegmentgroup.md). Define constantes que especificam o agrupamento de segmentos de memória de um adaptador. |
