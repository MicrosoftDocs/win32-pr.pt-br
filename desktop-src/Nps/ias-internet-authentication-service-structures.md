# <a name="nps-extensions-structures"></a>Estruturas de extensões do NPS

> [!Note]  
> O IAS (serviço de autenticação da Internet) foi renomeado como NPS (servidor de políticas de rede) a partir do Windows Server 2008. O conteúdo deste tópico aplica-se ao IAS e ao NPS.

 

Use as estruturas de [**\_ atributo RADIUS**](/windows/desktop/api/authif/ns-authif-radius_attribute) para representar um atributo RADIUS ou um atributo estendido ao desenvolver DLLs de extensão NPS.

-   [**\_atributo RADIUS**](/windows/desktop/api/authif/ns-authif-radius_attribute)
-   [**\_matriz de atributo RADIUS \_**](/windows/desktop/api/authif/ns-authif-radius_attribute_array)
-   [**\_bloco de \_ controle de extensão RADIUS \_**](/windows/desktop/api/authif/ns-authif-radius_extension_control_block)
-   [**\_formato de VSA RADIUS \_**](/windows/desktop/api/authif/ns-authif-radius_vsa_format)
