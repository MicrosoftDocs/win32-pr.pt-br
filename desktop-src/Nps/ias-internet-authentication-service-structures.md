# <a name="nps-extensions-structures"></a>Estruturas de extensões NPS

> [!Note]  
> O IAS (Serviço de Autenticação da Internet) foi renomeado como NPS (Servidor de Políticas de Rede) a partir do Windows Server 2008. O conteúdo deste tópico se aplica a IAS e NPS.

 

Use as [**estruturas \_ RADIUS ATTRIBUTE**](/windows/desktop/api/authif/ns-authif-radius_attribute) para representar um atributo RADIUS ou um atributo estendido ao desenvolver DLLs de extensão NPS.

-   [**ATRIBUTO \_ RADIUS**](/windows/desktop/api/authif/ns-authif-radius_attribute)
-   [**MATRIZ \_ DE ATRIBUTO \_ RADIUS**](/windows/desktop/api/authif/ns-authif-radius_attribute_array)
-   [**BLOCO DE \_ CONTROLE DE \_ EXTENSÃO \_ RADIUS**](/windows/desktop/api/authif/ns-authif-radius_extension_control_block)
-   [**FORMATO \_ RADIUS VSA \_**](/windows/desktop/api/authif/ns-authif-radius_vsa_format)
