# <a name="manipulation-target"></a>Manipulation-Target

> [!Note]  
> O IAS (serviço de autenticação da Internet) foi renomeado como NPS (servidor de políticas de rede) a partir do Windows Server 2008.

 

Os dados de destino aos quais uma regra de manipulação de atributo é aplicada. A manipulação de atributos era conhecida anteriormente como "processamento de territórios". Para obter mais informações, consulte [processamento de territórios e manipulação de atributos](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**Nome LDAP:** msManipulationTarget

**ID:** 4113 = [ **\_ destino de \_ manipulação \_ de atributo do IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxe:** Enumera

**Valores:  **

Definido pela IANA (Internet Assigned Numbers Authority) no documento de [tipos RADIUS](https://www.iana.org/assignments/radius-types) com base na [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   User-Name (1)
-   Called-Station-ID (30)
-   ID de estação de chamada (31)
