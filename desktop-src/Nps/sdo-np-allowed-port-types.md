# <a name="np-allowed-port-types"></a>NP-Allowed-Port-Types

Tipos de porta permitidos para uma conexão.

**Nome LDAP:** msNPAllowedPortTypes

**ID:** 4104 = [ **TIPOS DE PORTA PERMITIDAS DO ATRIBUTO IAS \_ \_ \_ \_ \_ NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxe:** Enumerador

**Valores: **

Definido pela IANA (Internet Assigned Numbers Authority) no documento [RADIUS TYPES](https://www.iana.org/assignments/radius-types) com base [no RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Assíncrono (Modem) (0)
-   Sincronização (linha T1) (1)
-   Sincronização ISDN (2)
-   ISDN Async V.120 (3)
-   ISDN Async V.110 (4)
-   Virtual (VPN) (5)
-   PIAFS (6)
-   HdLC Clear Channel (7)
-   X.25 (8)
-   X.75 (9)
-   Fax G.3 (10)
-   SDSL – DSL simétrica (11)
-   ADSL-CAP – Modularidade assimétrica da fase de amplitude sem operadora DSL (12)
-   ADSL-DMT – Multi-Tom Discreto Assimétrico de DSL (13)
-   IDSL – Linha do Assinante Digital ISDN (14)
-   Ethernet (15)
-   xDSL – Linha do Assinante Digital do tipo desconhecido (16)
-   Cabo (17)
-   Sem fio – Outros (18)
-   Sem fio – IEEE 802.11 (19)
-   Anel de token (20)
-   FDDI (21)
