# <a name="np-allowed-port-types"></a>NP-permitidos-tipos de porta

Tipos de porta permitidos para uma conexão.

**Nome LDAP:** msNPAllowedPortTypes

**ID:** 4104 = [ **\_ tipos de \_ \_ \_ porta \_ permitidos do atributo ias NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxe:** Enumera

**Valores:  **

Definido pela IANA (Internet Assigned Numbers Authority) no documento de [tipos RADIUS](https://www.iana.org/assignments/radius-types) com base na [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Async (modem) (0)
-   Sincronização (linha T1) (1)
-   Sincronização de ISDN (2)
-   ISDN assíncrono V. 120 (3)
-   ISDN Async V. 110 (4)
-   Virtual (VPN) (5)
-   PIAFS (6)
-   Canal de limpeza de HDLC (7)
-   X. 25 (8)
-   X. 75 (9)
-   G. 3 fax (10)
-   SDSL-DSL simétrico (11)
-   Modulação da fase de amplitude da concessionária de DSL de CAP assimétrico (12)
-   ADSL-DMT-multitom discreto DSL (13)
-   IDSL-linha de assinante digital de ISDN (14)
-   Ethernet (15)
-   xDSL-linha de assinante digital de tipo desconhecido (16)
-   Cabo (17)
-   Sem fio-outros (18)
-   Sem fio-IEEE 802,11 (19)
-   Token ring (20)
-   FDDI (21)
