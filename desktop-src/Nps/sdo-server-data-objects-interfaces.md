# <a name="server-data-objects-interfaces"></a>Interfaces de objetos de dados do servidor

A API de SDO (objetos de dados do servidor) expõe as seguintes interfaces.




| Interface | Finalidade | 
|-----------|---------|
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdo"><strong>ISdo</strong></a> | Manipule um objeto SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdocollection"><strong>ISdoCollection</strong></a> | Manipule uma coleção de objetos SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdodictionaryold"><strong>ISdoDictionaryOld</strong></a> | Manipule o dicionário de atributos.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdomachine"><strong>ISdoMachine</strong></a> | Gerencie o computador SDO e recupere objetos SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdoservicecontrol"><strong>ISdoServiceControl</strong></a> | Gerencie o serviço que está sendo administrado por meio de SDO, ou seja, o IAS (serviço de autenticação da Internet) ou o RAS (servidor de acesso remoto).<br /><blockquote>[!Note]<br />o IAS (serviço de autenticação da Internet) foi renomeado como NPS (servidor de políticas de rede) a partir do Windows Server 2008.</blockquote><br /> |
