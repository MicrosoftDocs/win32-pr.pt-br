# <a name="server-data-objects-interfaces"></a>Interfaces de objetos de dados do servidor

A API de SDO (objetos de dados do servidor) expõe as seguintes interfaces.



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Interface</th>
<th>Finalidade</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><a href="/windows/desktop/api/sdoias/nn-sdoias-isdo"><strong>ISdo</strong></a></td>
<td>Manipule um objeto SDO.<br/></td>
</tr>
<tr class="even">
<td><a href="/windows/desktop/api/sdoias/nn-sdoias-isdocollection"><strong>ISdoCollection</strong></a></td>
<td>Manipule uma coleção de objetos SDO.<br/></td>
</tr>
<tr class="odd">
<td><a href="/windows/desktop/api/sdoias/nn-sdoias-isdodictionaryold"><strong>ISdoDictionaryOld</strong></a></td>
<td>Manipule o dicionário de atributos.<br/></td>
</tr>
<tr class="even">
<td><a href="/windows/desktop/api/sdoias/nn-sdoias-isdomachine"><strong>ISdoMachine</strong></a></td>
<td>Gerencie o computador SDO e recupere objetos SDO.<br/></td>
</tr>
<tr class="odd">
<td><a href="/windows/desktop/api/sdoias/nn-sdoias-isdoservicecontrol"><strong>ISdoServiceControl</strong></a></td>
<td>Gerencie o serviço que está sendo administrado por meio de SDO, ou seja, o IAS (serviço de autenticação da Internet) ou o RAS (servidor de acesso remoto).<br/>
<blockquote>
[!Note]<br />
O IAS (serviço de autenticação da Internet) foi renomeado como NPS (servidor de políticas de rede) a partir do Windows Server 2008.
</blockquote>
<br/></td>
</tr>
</tbody>
</table>
