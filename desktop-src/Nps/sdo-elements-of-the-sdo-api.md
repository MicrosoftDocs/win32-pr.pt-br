# <a name="elements-of-the-sdo-api"></a>Elementos da API de SDO

A API de SDO é projetada em um modelo de objeto que consiste em objetos e coleções de objetos. No modelo de objeto SDO, os objetos e as coleções são organizados em uma hierarquia, ou seja, objetos e coleções contêm outros objetos e coleções.

Os objetos são manipulados pela recuperação e definição de propriedades de objeto.

As seções a seguir descrevem como a API de SDO usa objetos, propriedades de objeto e coleções:

-   [Objetos e propriedades](/windows/desktop/Nps/sdo-objects-and-properties)
-   [Hierarquia de modelo de objeto](/windows/desktop/Nps/sdo-object-model-hierarchy)
-   [Atributos](/windows/desktop/Nps/sdo-sdo-supported-attributes)
