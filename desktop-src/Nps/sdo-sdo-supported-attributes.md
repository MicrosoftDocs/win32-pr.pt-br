# <a name="sdo-supported-attributes"></a>Atributos com suporte a SDO

> [!Note]  
> o IAS (serviço de autenticação da Internet) foi renomeado como NPS (servidor de políticas de rede) a partir do Windows Server 2008.

 

Um objeto de dados do servidor NPS tem uma propriedade [**Profiles**](/windows/desktop/api/sdoias/ne-sdoias-profileproperties) que fornece acesso a uma coleção de atributos. Os atributos com suporte são os seguintes.

-   [ACCT-Provider-Name](/windows/desktop/Nps/sdo-acct-provider-name)
-   [Tipo de provedor de acct](/windows/desktop/Nps/sdo-acct-provider-type)
-   [Nome do provedor de autenticação](/windows/desktop/Nps/sdo-auth-provider-name)
-   [Tipo de provedor de autenticação](/windows/desktop/Nps/sdo-auth-provider-type)
-   [Ignore-User-Diale-Properties](/windows/desktop/Nps/sdo-ignore-user-dialin-properties)
-   [Manipulação-regra](/windows/desktop/Nps/sdo-manipulation-rule)
-   [Manipulação-destino](/windows/desktop/Nps/sdo-manipulation-target)
-   [NP-permitido-tipo de EAP](/windows/desktop/Nps/sdo-np-allowed-eap-type)
-   [NP-permitidos-tipos de porta](/windows/desktop/Nps/sdo-np-allowed-port-types)
-   [NP-autenticação-tipo](/windows/desktop/Nps/sdo-np-authentication-type)
-   [NP-ID de estação chamada](/windows/desktop/Nps/sdo-np-called-station-id)
