# <a name="manipulation-rule"></a>Manipulation-Rule

> [!Note]  
> O IAS (serviço de autenticação da Internet) foi renomeado como NPS (servidor de políticas de rede) a partir do Windows Server 2008.

 

A regra de manipulação a ser aplicada aos dados especificados pelo atributo de [destino Manipulation](/windows/desktop/Nps/sdo-manipulation-target) . Para obter mais informações, consulte [processamento de territórios e manipulação de atributos](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**Nome LDAP:** msManipulationRule

**ID:** 4114 = [ **\_ regra de \_ manipulação \_ de atributo do IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxe:** Strings
