# <a name="manipulation-rule"></a>Manipulation-Rule

> [!Note]  
> O IAS (Serviço de Autenticação da Internet) foi renomeado como NPS (Servidor de Políticas de Rede) a partir do Windows Server 2008.

 

A regra de manipulação a ser aplicada aos dados especificados pelo [atributo Manipulation-Target.](/windows/desktop/Nps/sdo-manipulation-target) Para obter mais informações, consulte [Processamento de realms e manipulação de atributo](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**Nome LDAP:** msManipulationRule

**ID:** 4114 = [ **REGRA DE MANIPULAÇÃO \_ \_ DE \_ ATRIBUTO IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxe:** String
