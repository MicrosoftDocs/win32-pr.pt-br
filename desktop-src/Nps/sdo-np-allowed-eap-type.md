# <a name="np-allowed-eap-type"></a>NP-permitido-tipo de EAP

Modos de criptografia EAP permitidos para uma conexão.

**Nome LDAP:** msNPAllowedEapType

**ID:** 4106 = [ **tipo de EAP de NP de atributo do IAS \_ \_ \_ permitido \_ \_**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxe:** Valores

**Valores:  **

Definido pela IANA (Internet Assigned Numbers Authority) no documento [do Registro EAP (Extensible Authentication Protocol)](https://www.iana.org/assignments/eap-numbers) baseado no [RFC 3748](https://www.ietf.org/rfc/rfc3748.txt).

-   **MD5-desafio** (4)
-   **EAP \_ TLS** (13)
-   **PEAP** (25)
-   **MS-EAP-autenticação** (26)
