# <a name="legacy-accessibility-and-automation-technology---msaa-to-ui-automation"></a>Tecnologia de acessibilidade e automação herdada – MSAA para automação da interface do usuário

A acessibilidade e a automação do Windows consistem em duas tecnologias: Microsoft Acessibilidade Ativa (MSAA) e automação da interface do usuário da Microsoft. O MSAA é uma tecnologia herdada introduzida no Windows 95, enquanto a automação da interface do usuário é uma tecnologia mais nova e mais compatível que aborda as limitações do MSAA.

Esta seção descreve as principais diferenças entre o MSAA e a automação da interface do usuário (e como fazer a transição para a automação da interface do usuário).
