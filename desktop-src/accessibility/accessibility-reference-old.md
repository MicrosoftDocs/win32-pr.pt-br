# <a name="windows-accessibility-and-automation-reference"></a>Windows referência de automação e acessibilidade

Windows A acessibilidade dá suporte a duas categorias de recursos para ajudar os desenvolvedores Windows a criar aplicativos acessíveis, os desenvolvedores de tecnologia adaptativa criam ferramentas como leitores de tela e lupas e engenheiros de teste de software criam scripts automatizados para testar aplicativos Windows: Configurações do usuário e um conjunto abrangente de APIs do Win32.

Para obter mais informações, [consulte Windows recursos de acessibilidade.](../WinAuto/about-windows-accessibility-features.md)
