# <a name="developing-assistive-technology-for-windows"></a>Desenvolvimento de tecnologias adaptativas para Windows

Produtos de tecnologia adaptativa de terceiros, como leitores de tela, lupas e hardware de especialização, são ferramentas essenciais para muitos usuários do Windows (e outros produtos da Microsoft).

Esta visão geral descreve como criar ferramentas de tecnologia adaptativa compatíveis com produtos da Microsoft para pessoas com deficiências ou limitações de visão, aprendizado, mobilidade/linguagem/comunicação.
