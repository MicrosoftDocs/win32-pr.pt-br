# <a name="developing-assistive-technology-for-windows"></a>Desenvolvimento de tecnologias adaptativas para Windows

Produtos de tecnologia assistencial de terceiros, como leitores de tela, ampliadores e hardware de especialização, são ferramentas essenciais para muitos usuários do Windows (e outros produtos da Microsoft).

Esta visão geral descreve como criar ferramentas de tecnologia assistencial que são compatíveis com os produtos da Microsoft para pessoas com visão, aprendizado, destreza/mobilidade e deficiências ou limitações de linguagem/comunicação.
