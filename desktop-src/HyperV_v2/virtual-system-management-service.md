# <a name="virtual-system-management-service"></a>Serviço de gerenciamento do sistema virtual

O perfil do sistema virtual descreve os objetos que compõem uma máquina virtual. Esses objetos incluem o sistema base, os dispositivos que compõem o sistema, as configurações do sistema e seus dispositivos e o serviço de gerenciamento que executa operações no sistema.
