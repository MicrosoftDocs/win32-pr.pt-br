# <a name="reset-method-of-the-msvm_externalethernetport-class"></a>Método reset da classe Msvm \_ ExternalEthernetPort

Solicita uma redefinição.

## <a name="syntax"></a>Sintaxe


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parâmetros
