# <a name="cim_cdromdrive-class-hyper-v-management"></a>Classe CIM_CDROMDrive (gerenciamento do Hyper-V)

Representa os recursos e o gerenciamento de uma unidade de CD ROM.

## <a name="syntax"></a>Sintaxe

``` syntax
[Abstract, Version("2.6.0"), UMLPackagePath("CIM::Device::StorageDevices")]
class CIM_CDROMDrive : CIM_MediaAccessDevice
{
};
```
