# <a name="commitfailover-method-of-the-msvm_replicationservice-class"></a>Método CommitFailover da \_ classe ReplicationService Msvm

Confirma o instantâneo de recuperação que o método [**InitiateFailover**](initiatefailover-msvm-replicationservice.md) usou para um failover.

Esse método mescla os pontos de recuperação na máquina virtual de recuperação aplicando o instantâneo de recuperação. Depois que isso for feito, a operação de failover não poderá ser cancelada. Normalmente, os usuários executarão isso quando o ponto de replicação usado para failover for satisfatório e a máquina virtual de recuperação puder alterar a função para primária.

## <a name="syntax"></a>Sintaxe


```mof
uint32 CommitFailover(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*ComputerSystem* \[ no\]
</dt> <dd>

Uma referência a uma instância de [**\_ sistema**](/windows/desktop/CIMWin32Prov/cim-computersystem) de computador CIM que representa a máquina virtual para a qual confirmar o failover.

</dd> <dt>

*Trabalho* \[ do fora\]
</dt> <dd>

Se a operação for executada de forma assíncrona, esse método retornará 4096, e esse parâmetro conterá uma referência a um objeto derivado de [**CIM \_ ConcreteJob**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
