# <a name="cim-classes-hyper-v-management"></a>Classes CIM (gerenciamento do Hyper-V)

As classes CIM são classes base das que derivam todos os objetos WMI. Cada provedor potencialmente pode usar uma versão ligeiramente diferente das classes CIM; portanto, a maioria dos provedores WMI v2 contém sua própria documentação de classe CIM.

As classes CIM a seguir são encontradas no \\ \\ namespace v2 de virtualização raiz.
