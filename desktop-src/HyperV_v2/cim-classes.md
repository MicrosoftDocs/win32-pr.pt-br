# <a name="cim-classes-hyper-v-management"></a>Classes CIM (gerenciamento do Hyper-V)

As classes CIM são classes base das quais todos os objetos WMI derivam. Cada provedor potencialmente pode usar versões um pouco diferentes das classes CIM; Portanto, a maioria dos provedores WMI de v2 contém sua própria documentação de classe CIM.

As classes CIM a seguir são encontradas no \\ namespace root Virtualization \\ v2.
