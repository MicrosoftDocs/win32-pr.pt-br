# <a name="msvm_virtualsystemmigrationnetworksettingdata-class"></a>\_Classe Msvm VirtualSystemMigrationNetworkSettingData

Representa a rede na qual o serviço de migração de sistema virtual está escutando a migração de sistema virtual de entrada.

A sintaxe a seguir é simplificada formato MOF código (MOF) e inclui todas as propriedades herdadas.

## <a name="syntax"></a>Sintaxe

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_VirtualSystemMigrationNetworkSettingData : CIM_SettingData
{
  string InstanceID;
  string Caption;
  string Description;
  string ElementName;
  string SubnetNumber;
  uint8  PrefixLength;
  uint32 Metric;
  string Tags[];
};
```
