# <a name="reset-method-of-the-msvm_serialport-class"></a>Método Reset da \_ classe SerialPort Msvm

Solicita uma redefinição.

## <a name="syntax"></a>Sintaxe


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

<dl> <dt>
