# <a name="remote-desktop-classes"></a>Área de Trabalho Remota classes

As Área de Trabalho Remota WMI de virtualização são usadas para gerenciar Área de Trabalho Remota e RemoteFX e máquinas virtuais.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                            | Descrição                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ InteractiveSessionACE**](msvm-interactivesessionace.md)<br/>                                     | Representa uma *ACE (entrada de* controle de acesso) que determina o acesso à sessão interativa de uma máquina virtual.<br/>                                |
| [**Msvm \_ Physical3dGraphicsProcessor**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Descreve a GPU (unidade de processamento de gráficos) 3D física.<br/>                                                                                        |
| [**Msvm \_ PhysicalGPUInfo**](msvm-physicalgpuinfo.md)<br/>                                                 | Contém informações sobre uma RemoteFX GPU (unidade de processamento de gráficos físicos).<br/>                                                                    |
| [**Msvm \_ RdvComponent**](msvm-rdvcomponent.md)<br/>                                                       | Representa o estado do componente RDV, que é responsável por fornecer um transporte para o pai para o convidado para fins de configuração.<br/> |
| [**Msvm \_ RdvComponentSettingData**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Representa o estado configurado do componente Área de Trabalho Remota virtualização (RDV). O estado padrão é Habilitado.<br/>                               |
| [**Msvm \_ Synth3dVideoPool**](msvm-synth3dvideopool.md)<br/>                                               | Contém informações sobre as GPUs (unidades de processamento de gráficos) de vídeo 3D sintéticas disponíveis no sistema host.<br/>                                 |
| [**Msvm \_ Synthetic3DDisplayController**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Representa o controlador de exibição 3D sintético atribuído a uma máquina virtual.<br/>                                                            |
| [**Msvm \_ Synthetic3DDisplayControllerSettingData**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Representa configurações para um controlador de exibição 3D sintético para uma máquina virtual.<br/>                                                                 |
| [**Msvm \_ Synthetic3DService**](msvm-synthetic3dservice.md)<br/>                                           | Descreve o serviço de GPU 3D sintético.<br/>                                                                                                          |
| [**Msvm \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Representa as configurações para o serviço 3D sintético presente em um único sistema de host.<br/>                                                            |
| [**Msvm \_ TerminalService**](msvm-terminalservice.md)<br/>                                                 | Gerencia todas as conexões de terminal remoto para um host específico.<br/>                                                                                     |
| [**Msvm \_ TerminalServiceSettingData**](msvm-terminalservicesettingdata.md)<br/>                           | Representa as configurações para os serviços de terminal do computador virtual em um host.<br/>                                                                     |
