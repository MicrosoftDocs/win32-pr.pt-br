# <a name="remote-desktop-classes"></a>Classes de Área de Trabalho Remota

As classes WMI de virtualização Área de Trabalho Remota são usadas para gerenciar Área de Trabalho Remota e recursos do RemoteFX e máquinas virtuais.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                            | Descrição                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ InteractiveSessionACE**](msvm-interactivesessionace.md)<br/>                                     | Representa uma ACE ( *entrada de controle de acesso* ) que determina o acesso à sessão interativa de uma máquina virtual.<br/>                                |
| [**Msvm \_ Physical3dGraphicsProcessor**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Descreve a GPU (unidade de processamento gráfico) 3-D física.<br/>                                                                                        |
| [**Msvm \_ PhysicalGPUInfo**](msvm-physicalgpuinfo.md)<br/>                                                 | Contém informações sobre uma GPU (unidade de processamento gráfico físico) do RemoteFX.<br/>                                                                    |
| [**Msvm \_ RdvComponent**](msvm-rdvcomponent.md)<br/>                                                       | Representa o estado do componente RDV, que é responsável por fornecer um transporte para o pai para o convidado para fins de configuração.<br/> |
| [**Msvm \_ RdvComponentSettingData**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Representa o estado configurado do componente de RDV (virtualização de Área de Trabalho Remota). O estado padrão é habilitado.<br/>                               |
| [**Msvm \_ Synth3dVideoPool**](msvm-synth3dvideopool.md)<br/>                                               | Contém informações sobre as GPUs (unidades de processamento de gráficos de vídeo) 3D sintéticas disponíveis no sistema host.<br/>                                 |
| [**Msvm \_ Synthetic3DDisplayController**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Representa o controlador de vídeo sintético 3D atribuído a uma máquina virtual.<br/>                                                            |
| [**Msvm \_ Synthetic3DDisplayControllerSettingData**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Representa as configurações de um controlador de vídeo sintético 3D para uma máquina virtual.<br/>                                                                 |
| [**Msvm \_ Synthetic3DService**](msvm-synthetic3dservice.md)<br/>                                           | Descreve o serviço de GPU 3D sintético.<br/>                                                                                                          |
| [**Msvm \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Representa as configurações do serviço 3D sintético presente em um único sistema de host.<br/>                                                            |
| [**Msvm \_ TerminalService**](msvm-terminalservice.md)<br/>                                                 | Gerencia todas as conexões de terminal remotas para um host específico.<br/>                                                                                     |
| [**Msvm \_ TerminalServiceSettingData**](msvm-terminalservicesettingdata.md)<br/>                           | Representa as configurações dos serviços de terminal do computador virtual em um host.<br/>                                                                     |
