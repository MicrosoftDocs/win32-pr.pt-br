# <a name="modifyservicesettings-method-of-the-msvm_synthetic3dservice-class"></a>Método ModifyServiceSettings da classe Msvm \_ Synthetic3DService

Modifica os dados de configuração para o serviço 3D sintético.

## <a name="syntax"></a>Sintaxe


```mof
uint32 ModifyServiceSettings(
  [in]  string              SettingData,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*SettingData* \[ Em\]
</dt> <dd>

Uma representação de cadeia de caracteres da [**classe Msvm \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md) que contém os dados de configuração modificados para o serviço.

</dd> <dt>

*Trabalho* \[ out\]
</dt> <dd>

Se a operação for executada de forma assíncrona, esse método retornará 4096 e esse parâmetro conterá uma referência a um objeto derivado de [**CIM \_ ConcreteJob.**](/previous-versions//cc136808(v=vs.85))

</dd> </dl>
