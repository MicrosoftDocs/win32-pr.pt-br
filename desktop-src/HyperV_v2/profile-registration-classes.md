# <a name="profile-registration-classes"></a>Classes de registro de perfil

Veja a seguir as classes WMI de virtualização relacionadas ao registro de perfil.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                    | Descrição                                                                                                                                                                      |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ ElementConformsToProfile**](msvm-elementconformstoprofile.md)<br/>                       | Define os perfis registrados para os quais o sistema referenciado está em conformidade.<br/>                                                                                              |
| [**Msvm \_ HostedResourcePool**](msvm-hostedresourcepool.md)<br/>                                   | Representa uma especialização da Associação de componente do sistema que estabelece que o pool de recursos é definido no contexto do sistema.<br/>                      |
| [**Msvm \_ ReferencedProfile**](msvm-referencedprofile.md)<br/>                                     | Descreve um perfil que é referenciado por outro perfil registrado.<br/>                                                                                                 |
| [**Msvm \_ RegisteredProfile**](msvm-registeredprofile.md)<br/>                                     | Descreve um conjunto de classes com propriedades e métodos necessários, necessários para gerenciar uma entidade do mundo real ou para dar suporte a um cenário de uso de maneira interoperável.<br/> |
| [**Msvm \_ ResourcePoolComponent**](msvm-resourcepoolcomponent.md)<br/>                             | Representa um elemento do pool de recursos da plataforma Microsoft Windows Hyper-V.<br/>                                                                                         |
| [**Msvm \_ ResourcePoolRegistration**](msvm-resourcepoolregistration.md)<br/>                       | Registra um serviço que fornece objetos relacionados ao pool de recursos globais.<br/>                                                                                               |
| [**Msvm \_ ResourceTypeDefinition**](msvm-resourcetypedefinition.md)<br/>                           | Define um mapeamento de um tipo de recurso para suas classes de implementação.<br/>                                                                                                   |
| [**Msvm \_ VirtualizationComponent**](msvm-virtualizationcomponent.md)<br/>                         | Representa um serviço da plataforma Microsoft Windows Hyper-V.<br/>                                                                                                       |
| [**Msvm \_ VirtualizationComponentRegistration**](msvm-virtualizationcomponentregistration.md)<br/> | Representa o registro de um serviço na plataforma Microsoft Hyper-V.<br/>                                                                                           |
| [**Msvm \_ VirtualSystemResourceComponent**](msvm-virtualsystemresourcecomponent.md)<br/>           | Representa um serviço de dispositivo virtual da plataforma Microsoft Windows Hyper-V.<br/>                                                                                        |
| [**Msvm \_ VirtualSystemResourceRegistration**](msvm-virtualsystemresourceregistration.md)<br/>     | Registra um serviço que fornece objetos relacionados a recursos específicos da máquina virtual.<br/>                                                                                  |
