# <a name="memory-classes"></a>Classes de memória

As classes de memória modelam a memória para uma máquina virtual.

Veja a seguir as classes WMI de virtualização relacionadas à memória.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                      | Descrição                                                                                                         |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ ElementAllocatedFromNumaNode**](msvm-elementallocatedfromnumanode.md)<br/> | Associa uma instância de um recurso alocado com o nó NUMA físico do qual ele foi alocado.<br/> |
| [**\_Memória Msvm**](msvm-memory.md)<br/>                                             | Representa a memória atualmente alocada para uma máquina virtual.<br/>                                          |
| [**Msvm \_ MemorySettingData**](msvm-memorysettingdata.md)<br/>                       | Representa o estado configurado da memória para uma máquina virtual.<br/>                                     |
| [**Msvm \_ NumaNode**](msvm-numanode.md)<br/>                                         | Representa um nó NUMA (acesso não uniforme à memória) de um sistema virtual.<br/>                                  |
