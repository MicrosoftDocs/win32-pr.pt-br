# <a name="virtual-system-classes"></a>Classes do sistema virtual

A seguir estão as classes WMI de virtualização relacionadas à máquina virtual.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                            |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ BootSourceComponent**](msvm-bootsourcecomponent.md)<br/>                             | Associa o [**Msvm \_ BootSourceSettingData**](msvm-bootsourcesettingdata.md) ao [**Msvm \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md)geral. <br/>                                                                                                                                                                                                                                                   |
| [**Msvm \_ BootSourceSettingData**](msvm-bootsourcesettingdata.md)<br/>                         | Representa os parâmetros para definir a origem de inicialização de uma máquina virtual. <br/>                                                                                                                                                                                                                                                                                                                                                     |
| [**Msvm \_ ComputerSystem**](msvm-computersystem.md)<br/>                                       | Representa um sistema de computador físico ou uma máquina virtual.<br/>                                                                                                                                                                                                                                                                                                                                                                   |
| [**Msvm \_ ConcreteComponent**](msvm-concretecomponent.md)<br/>                                 | Uma associação genérica usada para estabelecer relações 'parte de' entre elementos gerenciados.<br/>                                                                                                                                                                                                                                                                                                                                   |
| [**Msvm \_ LogicalIdentity**](msvm-logicalidentity.md)<br/>                                     | Associa dois elementos gerenciados que representam diferentes aspectos da mesma entidade subjacente. <br/>                                                                                                                                                                                                                                                                                                                            |
| [**Msvm \_ MostCurrentSnapshotInBranch**](msvm-mostcurrentsnapshotinbranch.md)<br/>             | Associa uma instância da classe [**Msvm \_ ComputerSystem**](msvm-computersystem.md) ou [**Msvm \_ PlannedComputerSystem**](msvm-plannedcomputersystem.md) que representa um sistema virtual, a uma instância da [**classe Msvm \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md) que representa o instantâneo do sistema virtual que é o instantâneo mais atual em um branch de instantâneos dependentes.<br/>            |
| [**Msvm \_ ParentChildSettingData**](msvm-parentchildsettingdata.md)<br/>                       | Uma associação entre uma instância do [**Msvm \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md) e a instância **do Msvm \_ VirtualSystemSettingData** que representa o instantâneo mais recente no qual esse objeto se baseia.<br/>                                                                                                                                                                                 |
| [**Msvm \_ PlannedComputerSystem**](msvm-plannedcomputersystem.md)<br/>                         | Representa uma máquina virtual planejada.<br/>                                                                                                                                                                                                                                                                                                                                                                                       |
| [**Msvm \_ SnapshotOfVirtualSystem**](msvm-snapshotofvirtualsystem.md)<br/>                     | Associa um sistema virtual a um instantâneo que foi capturado do sistema virtual.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**Msvm \_ SummaryInformation**](msvm-summaryinformation.md)<br/>                               | Usado nos métodos [**GetSummaryInformation**](getsummaryinformation-msvm-virtualsystemmanagementservice.md) e [**GetDefinitionFileSummaryInformation**](getdefinitionfilesummaryinformation-msvm-virtualsystemmanagementservice.md) na classe [**\_ VirtualSystemManagementService do Msvm**](msvm-virtualsystemmanagementservice.md) para recuperar rapidamente informações comuns relacionadas a uma máquina virtual ou instantâneo.<br/> |
| [**Msvm \_ SystemComponent**](msvm-systemcomponent.md)<br/>                                     | Estabelece uma relação de "parte de" entre um sistema e qualquer elemento do sistema gerenciado do qual ele é composto.<br/>                                                                                                                                                                                                                                                                                                               |
| [**SystemDevice do \_ Msvm**](msvm-systemdevice.md)<br/>                                           | Associa um dispositivo lógico ao sistema pai.<br/>                                                                                                                                                                                                                                                                                                                                                                         |
| [**Msvm \_ VirtualSystemCapabilities**](msvm-virtualsystemcapabilities.md)<br/>                 | Descreve os recursos do [**Msvm \_ ComputerSystem associado.**](msvm-computersystem.md)<br/>                                                                                                                                                                                                                                                                                                                           |
| [**Msvm \_ VirtualSystemSettingData**](msvm-virtualsystemsettingdata.md)<br/>                   | Representa as configurações específicas de virtualização para uma máquina virtual.<br/>                                                                                                                                                                                                                                                                                                                                                      |
| [**Msvm \_ VirtualSystemSettingDataComponent**](msvm-virtualsystemsettingdatacomponent.md)<br/> | Uma associação genérica usada para estabelecer relações 'parte de' entre uma instância do [**CIM \_ VirtualSystemSettingData**](/previous-versions//cc136954(v=vs.85)) e uma ou mais instâncias de [**\_ ResourceAllocationSettingData do CIM.**](/previous-versions/windows/desktop/clushyperv/cim-resourceallocationsettingdata)<br/>                                                                                                                                                 |
