# <a name="virtual-system-management-classes"></a>Classes de gerenciamento do sistema virtual

Veja a seguir as classes WMI de virtualização relacionadas ao gerenciamento de máquinas virtuais.

| Tópico                                                                                                                | Descrição                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AffectedJobElement**](msvm-affectedjobelement.md)<br/>                                               | Representa uma associação entre um trabalho e o elemento gerenciado que pode ser afetado pela sua execução.<br/>                                                                                                                                                    |
| [**Msvm \_ ConcreteJob**](msvm-concretejob.md)<br/>                                                             | Representa uma unidade de trabalho e é usado para acompanhar o progresso de operações assíncronas.<br/>                                                                                                                                                                   |
| [**Msvm \_ DiskMergeSettingData**](msvm-diskmergesettingdata.md)<br/>                                           | Representa o estado de configuração das configurações de mesclagem de disco para uma máquina virtual.<br/>                                                                                                                                                                      |
| [**Msvm \_ ElementSettingData**](msvm-elementsettingdata.md)<br/>                                               | Associa um elemento gerenciado a seus dados de configuração.<br/>                                                                                                                                                                                                 |
| [**Erro de Msvm \_**](msvm-error.md)<br/>                                                                         | Contém informações sobre a gravidade, a causa, as ações recomendadas e outros dados relacionados à falha de uma operação CIM.<br/>                                                                                                                            |
| [**Msvm \_ HostedDependency**](msvm-hosteddependency.md)<br/>                                                   | Associa uma instância de máquina virtual ao objeto do sistema de computador que representa o sistema físico, de hospedagem.<br/>                                                                                                                                       |
| [**Msvm \_ HostedService**](msvm-hostedservice.md)<br/>                                                         | Associa um serviço ao seu sistema de computador de hospedagem.<br/>                                                                                                                                                                                                    |
| [**Msvm \_ LastAppliedSnapshot**](msvm-lastappliedsnapshot.md)<br/>                                             | Representa uma associação entre um sistema virtual e os dados de configuração do instantâneo que foi aplicado mais recentemente ao sistema virtual.<br/>                                                                                                             |
| [**Msvm \_ OwningJobElement**](msvm-owningjobelement.md)<br/>                                                   | Representa uma associação entre um trabalho e o elemento gerenciado responsável pela criação do trabalho.<br/>                                                                                                                                                  |
| [**Msvm não \_ afetaelement**](msvm-serviceaffectselement.md)<br/>                                         | Associa uma instância de máquina virtual ao serviço de gerenciamento que controla seu estado.<br/>                                                                                                                                                                |
| [**Msvm \_ ServicingSettings**](msvm-servicingsettings.md)<br/>                                                 | Contém as configurações usadas durante as operações de manutenção.<br/>                                                                                                                                                                                                       |
| [**Msvm \_ SettingsDefineState**](msvm-settingsdefinestate.md)<br/>                                             | Associa uma máquina virtual e seus dispositivos a instâncias do [**CIM \_ SettingData**](/previous-versions//cc136911(v=vs.85)) que representam as configurações atuais que se aplicam a esses objetos.<br/>                                                                               |
| [**Msvm \_ SystemExportSettingData**](msvm-systemexportsettingdata.md)<br/>                                     | Associa uma máquina virtual e seus dados de configuração de exportação.<br/>                                                                                                                                                                                                 |
| [**Msvm \_ VirtualSystemExportSettingData**](msvm-virtualsystemexportsettingdata.md)<br/>                       | Fornece informações adicionais a serem usadas com o método [**ExportSystemDefinition**](exportsystemdefinition-msvm-virtualsystemmanagementservice.md) da classe [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md) .<br/> |
| [**Msvm \_ VirtualSystemManagementCapabilities**](msvm-virtualsystemmanagementcapabilities.md)<br/>             | Descreve os recursos do [**\_ VirtualSystemManagementService Msvm**](msvm-virtualsystemmanagementservice.md)associado.<br/>                                                                                                                         |
| [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md)<br/>                       | Representa o serviço de virtualização presente em um único sistema de host.<br/>                                                                                                                                                                                    |
| [**Msvm \_ VirtualSystemManagementServiceSettingData**](msvm-virtualsystemmanagementservicesettingdata.md)<br/> | Representa as configurações do serviço de virtualização presente em um único sistema de host.<br/>                                                                                                                                                                   |
| [**Msvm \_ VirtualSystemSnapshotService**](msvm-virtualsystemsnapshotservice.md)<br/>                           | Serviço para criar, aplicar e destruir instantâneos de máquinas virtuais.<br/>                                                                                                                                                                                      |
