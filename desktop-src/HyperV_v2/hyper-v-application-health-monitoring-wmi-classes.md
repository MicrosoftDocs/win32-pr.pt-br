# <a name="hyper-v-application-health-monitoring-wmi-classes"></a>Classes WMI de monitoramento de integridade do aplicativo Hyper-V

A API de monitoramento de integridade do aplicativo Hyper-V define as classes WMI a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                        | Descrição                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ HeartbeatComponent**](msvm-heartbeatcomponent.md)<br/>                       | Representa o estado do serviço de pulsação, que é responsável por monitorar o estado de uma máquina virtual, relatando uma pulsação em intervalos regulares.<br/> |
| [**Msvm \_ HeartbeatComponentSettingData**](msvm-heartbeatcomponentsettingdata.md)<br/> | Representa o estado configurado do serviço de pulsação.<br/>                                                                                                        |
