# <a name="msvm_ethernetswitchportfeaturesettingdata-class"></a>\_Classe Msvm EthernetSwitchPortFeatureSettingData

Classe base abstrata para classes que representam configurações para um recurso de porta de comutador Ethernet.

A sintaxe a seguir é simplificada formato MOF código (MOF) e inclui todas as propriedades herdadas.

## <a name="syntax"></a>Sintaxe

``` syntax
[Abstract, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_EthernetSwitchPortFeatureSettingData : Msvm_FeatureSettingData
{
  string InstanceID;
  string Caption;
  string Description;
  string ElementName;
};
```
