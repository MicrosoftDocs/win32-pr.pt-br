# <a name="cim_processindication-class"></a>\_Classe CIM ProcessIndication

**CIM \_ ProcessIndication** é uma superclasse abstrata para classes de indicação especializadas, que abordam alterações e alertas específicos publicados por provedores e instrumentação.

## <a name="syntax"></a>Sintaxe

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>Membros

A classe **CIM \_ ProcessIndication** não define nenhum membro.
