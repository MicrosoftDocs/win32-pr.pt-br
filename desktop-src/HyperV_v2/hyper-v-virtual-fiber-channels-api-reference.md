# <a name="hyper-v-virtual-fibre-channel-api-reference"></a>Referência da API de Fibre Channel virtual do Hyper-V

A API de Fibre Channel virtual do Hyper-V define os seguintes elementos de programação.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                    | Descrição                                                                                                    |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ ExternalFcPort**](msvm-externalfcport.md)<br/>                           | Representa uma porta de Fibre Channel externa.<br/>                                                          |
| [**Msvm \_ FcActiveConnection**](msvm-fcactiveconnection.md)<br/>                   | Conecta uma porta de comutador ao ponto de extremidade de Fibre Channel ao qual a porta está conectada.<br/>                |
| [**Msvm \_ FcDeviceSAPImplementation**](msvm-fcdevicesapimplementation.md)<br/>     | Representa uma associação entre um ponto de acesso de serviço e o dispositivo lógico que o implementa.<br/> |
| [**Msvm \_ FcEndpoint**](msvm-fcendpoint.md)<br/>                                   | Representa o ponto de conexão lógica para uma porta Fibre Channel.<br/>                                   |
| [**Msvm \_ FcPortAllocationSettingData**](msvm-fcportallocationsettingdata.md)<br/> | Representa o estado configurado de uma porta de Fibre Channel sintética ou uma porta de comutador Fibre Channel.<br/>   |
| [**Msvm \_ FcSwitchPort**](msvm-fcswitchport.md)<br/>                               | Representa uma porta no comutador de Fibre Channel virtual.<br/>                                              |
| [**Msvm \_ SyntheticFcPort**](msvm-syntheticfcport.md)<br/>                         | Representa uma porta de Fibre Channel sintética.<br/>                                                          |
| [**Msvm \_ SyntheticFcPortSettingData**](msvm-syntheticfcportsettingdata.md)<br/>   | Representa o estado configurado de uma porta de Fibre Channel sintética.<br/>                                  |
| [**Msvm \_ VirtualFcSwitch**](msvm-virtualfcswitch.md)<br/>                         | Representa um comutador Fibre Channel virtual.<br/>                                                          |
| [**Msvm \_ VirtualFcSwitchSettingData**](msvm-virtualfcswitchsettingdata.md)<br/>   | Representa a configuração de um comutador Fibre Channel virtual.<br/>                                     |
