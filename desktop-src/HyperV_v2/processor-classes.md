# <a name="processor-classes"></a>Classes de processador

As classes de processador modelam um processador para uma máquina virtual.

A seguir estão as classes WMI de virtualização relacionadas ao processador.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                      | Descrição                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Processador Msvm \_**](msvm-processor.md)<br/>                       | Representa o processador virtual em uma máquina virtual.<br/>                         |
| [**Msvm \_ ProcessorPool**](msvm-processorpool.md)<br/>               | Agrega os recursos do processador que podem ser alocados a uma máquina virtual.<br/> |
| [**Msvm \_ ProcessorSettingData**](msvm-processorsettingdata.md)<br/> | Representa as configurações do processador virtual para uma máquina virtual.<br/>               |
