# <a name="startservice-method-of-the-msvm_metricservice-class"></a>Método StartService da classe Msvm \_ MetricService

Inicia o serviço.

## <a name="syntax"></a>Sintaxe


```mof
uint32 StartService();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.
