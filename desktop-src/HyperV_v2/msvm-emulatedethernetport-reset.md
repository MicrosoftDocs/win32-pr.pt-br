# <a name="reset-method-of-the-msvm_emulatedethernetport-class"></a>Método Reset da classe Msvm \_ EmulatedEthernetPort

Redefine o dispositivo virtual.

## <a name="syntax"></a>Sintaxe


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

Esse método retorna um dos seguintes valores:

<dl> <dt>
