# <a name="msvm_persistentmemorycontroller-class"></a>\_Classe Msvm PersistentMemoryController

Representa um controlador de memória persistente virtual.

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas.

## <a name="syntax"></a>Sintaxe

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_PersistentMemoryController : CIM_Controller
{
};
```

## <a name="members"></a>Membros

A classe **Msvm \_ PersistentMemoryController** não define nenhum membro.
