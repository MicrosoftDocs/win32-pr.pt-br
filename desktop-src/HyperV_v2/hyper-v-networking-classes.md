# <a name="hyper-v-networking-api-reference"></a>Referência da API de rede do Hyper-V

A arquitetura de rede para virtualização modela a arquitetura de rede física. Ele usa objetos de rede padrão, como comutadores, portas de comutadores e adaptadores de rede.

A seguir estão as classes WMI de virtualização relacionadas à rede.
