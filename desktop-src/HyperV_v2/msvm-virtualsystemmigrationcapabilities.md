# <a name="msvm_virtualsystemmigrationcapabilities-class"></a>\_Classe Msvm VirtualSystemMigrationCapabilities

Define o meio pelo qual um cliente pode descobrir os métodos fornecidos pelo serviço de migração e o intervalo válido de dados de configuração de migração do sistema virtual. O objeto **Msvm \_ VirtualSystemMigrationCapabilities** é associado a objetos [**Msvm \_ VirtualSystemMigrationSettingData**](msvm-virtualsystemmigrationsettingdata.md) . Essas associações definem o intervalo válido de serviços de migração disponíveis.

A sintaxe a seguir é simplificada formato MOF código (MOF) e inclui todas as propriedades herdadas.

## <a name="syntax"></a>Sintaxe

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_VirtualSystemMigrationCapabilities : CIM_VirtualSystemMigrationCapabilities
{
  string InstanceID;
  string Caption = "Migration Capabilities";
  string Description = "Virtual System Migration Capabilities";
  string ElementName;
  uint16 SynchronousMethodsSupported[];
  uint16 AsynchronousMethodsSupported[];
  uint16 DestinationHostFormatsSupported[];
};
```
