# <a name="geterror-method-of-the-msvm_virtualsystemreferencepointexportjob-class"></a>Método GetError da \_ classe Msvm VirtualSystemReferencePointExportJob

Recupera o erro.

## <a name="syntax"></a>Sintaxe


```mof
uint32 GetError(
  [out] string Error
);
```
