# <a name="startservice-method-of-the-msvm_imagemanagementservice-class"></a>Método StartService da classe Msvm \_ ImageManagementService

Inicia o serviço.

## <a name="syntax"></a>Sintaxe


```mof
uint32 StartService();
```



## <a name="parameters"></a>Parâmetros
