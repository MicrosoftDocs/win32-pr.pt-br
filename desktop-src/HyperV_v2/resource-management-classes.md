# <a name="resource-management-classes"></a>Classes de gerenciamento de recursos

A seguir estão as classes WMI de virtualização relacionadas ao gerenciamento de recursos.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                        | Descrição                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AbstractResourcePoolSettingData**](msvm-abstractresourcepoolsettingdata.md)<br/>             | Representa as configurações de uma [**instância do Msvm \_ ResourcePool**](msvm-resourcepool.md) que não estão relacionadas à alocação.<br/>                                |
| [**Msvm \_ AllocationCapabilities**](msvm-allocationcapabilities.md)<br/>                               | Define os meios pelos quais um cliente pode descobrir o intervalo válido de configurações padrão para um recurso virtual.<br/>                                          |
| [**Elemento \_ MsvmAllocatedFromPool**](msvm-elementallocatedfrompool.md)<br/>                           | Associa uma instância de um recurso alocado ao pool de recursos do qual ele foi alocado.<br/>                                                   |
| [**Msvm \_ ElementCapabilities**](msvm-elementcapabilities.md)<br/>                                     | Representa a associação entre elementos gerenciados e seus recursos.<br/>                                                                           |
| [**Msvm \_ FeatureSettingData**](msvm-featuresettingdata.md)<br/>                                       | Uma classe abstrata que representa as configurações de um recurso específico de um sistema ou componente.<br/>                                                           |
| [**Msvm \_ FeatureSettingsDefineCapabilities**](msvm-featuresettingsdefinecapabilities.md)<br/>         | Fornece um link entre a instância de recursos de recurso do com opção Ethernet e as configurações mínimas, máximas, incrementais e padrão para um recurso.<br/> |
| [**Msvm \_ ResourceAllocationFromPool**](msvm-resourceallocationfrompool.md)<br/>                       | Associa uma instância de uma alocação de recursos ao pool de recursos do qual ela é alocada.<br/>                                                    |
| [**Msvm \_ ResourceAllocationSettingData**](msvm-resourceallocationsettingdata.md)<br/>                 | Representa os estados de alocação atuais e registrados de um recurso virtual.<br/>                                                                          |
| [**Msvm \_ ResourcePool**](msvm-resourcepool.md)<br/>                                                   | Descreve um tipo de recurso virtual disponível para uso em máquinas virtuais.<br/>                                                                           |
| [**Msvm \_ ResourcePoolConfigurationCapabilities**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | Descreve os recursos da classe [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md) associada.<br/>           |
| [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md)<br/>           | Fornece o gerenciamento ativo de pools de recursos.<br/>                                                                                                     |
| [**Msvm \_ ResourcePoolSettingData**](msvm-resourcepoolsettingdata.md)<br/>                             | Representa as configurações de uma [**instância do Msvm \_ ResourcePool**](msvm-resourcepool.md) que não estão relacionadas à alocação.<br/>                                |
| [**Configurações do \_ MsvmDefineCapabilities**](msvm-settingsdefinecapabilities.md)<br/>                       | Fornece um link entre a instância de funcionalidades e as configurações mínimas, máximas, incrementais e padrão para um recurso.<br/>                         |
