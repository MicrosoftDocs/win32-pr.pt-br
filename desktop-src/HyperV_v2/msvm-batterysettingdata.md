# <a name="msvm_batterysettingdata-class"></a>Classe Msvm \_ BatterySettingData

Representa as configurações de um dispositivo de bateria virtual.

A sintaxe a seguir é simplificada do código MOF (Managed Object Format) e inclui todas as propriedades herdadas.

## <a name="syntax"></a>Sintaxe

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_BatterySettingData : CIM_ResourceAllocationSettingData
{
};
```

## <a name="members"></a>Membros

A **classe Msvm \_ BatterySettingData** não define nenhum membro.
