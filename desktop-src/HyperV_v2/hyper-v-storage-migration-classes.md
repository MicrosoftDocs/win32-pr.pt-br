# <a name="hyper-v-migration-api-reference"></a>Referência da API de migração do Hyper-V

A API de migração do Hyper-V define os seguintes elementos de programação.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                                                | Descrição                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ MigrationJob**](msvm-migrationjob.md)<br/>                                                                           | Essa classe representa um trabalho de operação de migração criado para armazenamento ou migração de sistema virtual pelo serviço de migração de sistema virtual.<br/>                                                 |
| [**Msvm \_ VirtualSystemMigrationCapabilities**](msvm-virtualsystemmigrationcapabilities.md)<br/>                               | Define o meio pelo qual um cliente pode descobrir os métodos fornecidos pelo serviço de migração e o intervalo válido de dados de configuração de migração do sistema virtual.<br/>                                |
| [**Msvm \_ VirtualSystemMigrationNetworkSettingData**](msvm-virtualsystemmigrationnetworksettingdata.md)<br/>                   | Representa a rede na qual o serviço de migração de sistema virtual está escutando a migração de sistema virtual de entrada.<br/>                                                                 |
| [**Msvm \_ VirtualSystemMigrationService**](msvm-virtualsystemmigrationservice.md)<br/>                                         | Representa o serviço de migração do sistema virtual. Ele é usado para migrar um sistema virtual ou para migrar o armazenamento de um sistema virtual de uma plataforma de virtualização para outra.<br/> |
| [**Msvm \_ VirtualSystemMigrationServiceSettingData**](msvm-virtualsystemmigrationservicesettingdata.md)<br/>                   | Representa as configurações do serviço de migração de sistema virtual em um host.<br/>                                                                                                              |
| [**Msvm \_ VirtualSystemMigrationServiceSettingDataComponent**](msvm-virtualsystemmigrationservicesettingdatacomponent.md)<br/> | Uma associação usada para representar as configurações de rede de migração do sistema virtual do serviço de migração do sistema virtual.<br/>                                                                      |
| [**Msvm \_ VirtualSystemMigrationSettingData**](msvm-virtualsystemmigrationsettingdata.md)<br/>                                 | Representa as configurações de migração para migrar um sistema virtual e o armazenamento anexado a um sistema virtual.<br/>                                                                           |
