# <a name="remotefx-media-redirection-api-functions"></a>Funções da API de redirecionamento de mídia do RemoteFX

A API de redirecionamento de mídia do RemoteFX dá suporte às funções a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**WTSSetRenderHint**](/windows/desktop/api/WtsHintApi/nf-wtshintapi-wtssetrenderhint)
</dt> <dd>

Usado por um aplicativo que está exibindo conteúdo que pode ser otimizado para ser exibido em uma sessão remota para identificar a região de uma janela que é o conteúdo real.

</dd> </dl>
