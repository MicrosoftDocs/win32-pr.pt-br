# <a name="remotefx-media-redirection-api-functions"></a>RemoteFX Funções da API de Redirecionamento de Mídia

A API RemoteFX redirecionamento de mídia oferece suporte às funções a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**WTSSetRenderHint**](/windows/desktop/api/WtsHintApi/nf-wtshintapi-wtssetrenderhint)
</dt> <dd>

Usado por um aplicativo que está exibindo conteúdo que pode ser otimizado para exibição em uma sessão remota para identificar a região de uma janela que é o conteúdo real.

</dd> </dl>
