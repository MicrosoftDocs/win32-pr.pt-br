# <a name="iremotedesktopclienttouchpointer-properties"></a>Propriedades de IRemoteDesktopClientTouchPointer

A interface [**IRemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) expõe as propriedades a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Propriedade Enabled**](iremotedesktopclienttouchpointer-enabled.md)
</dt> <dd>

Se o recurso de ponteiro de toque está habilitado no controle de cliente do contêiner de aplicativo RDP.

</dd> <dt>

[**Propriedade EventsEnabled**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled)
</dt> <dd>

Se as notificações de evento de ponteiro de toque estão habilitadas para o controle de cliente do contêiner de aplicativo RDP.

</dd> <dt>

[**Propriedade PointerSpeed**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_pointerspeed)
</dt> <dd>

Quão rápido o cursor do ponteiro de toque se moverá na área de trabalho virtual em relação à velocidade do gesto no cliente.

</dd> </dl>
