# <a name="imsrdpclientadvancedsettings-interface"></a>Interface IMsRdpClientAdvancedSettings

Gerencia configurações avançadas do cliente. Deriva da interface [**IMsTscAdvancedSettings**](imstscadvancedsettings-interface.md) . Essa interface inclui métodos para recuperar e definir propriedades avançadas (opcionais) para o controle ActiveX Área de Trabalho Remota.

Para obter uma instância dessa interface, use a propriedade [**IMsTscAx:: AdvancedSettings**](imstscax-advancedsettings.md) para obter um ponteiro de interface [**IMsTscAdvancedSettings**](imstscadvancedsettings-interface.md) . Em seguida, chame [**QueryInterface**](/windows/desktop/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) no ponteiro **IMsTscAdvancedSettings** e passe **\_ IMsRdpClientAdvancedSettings de IID** para **QueryInterface**.

-   [Propriedades](#properties)
