# <a name="win32_tssessiondirectory-methods"></a>\_Métodos TSSessionDirectory do Win32

A classe [**Win32 \_ TSSessionDirectory**](win32-tssessiondirectory.md) expõe os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método CreateUserDiskTemplate**](createuserdisktemplate-win32-tssessiondirectory.md)
</dt> <dd>

Cria um modelo de disco de usuário.

</dd> <dt>

[**Método DisableUserVhd**](disableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Desabilita um VHD de perfil de usuário.

</dd> <dt>

[**Método EnableUserVhd**](enableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Habilita um VHD de perfil de usuário em um servidor RDSH.

</dd> <dt>

[**Método GetCurrentRedirectableAddresses**](getcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Obtém a lista atualmente configurada de endereços DNS qualificados que podem ser usados para redirecionamento.

</dd> <dt>

[**Método GetRedirectableAddresses**](getredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Obtém a lista completa de endereços do DNS qualificados.

</dd> <dt>

[**Método PingSessionDirectory**](pingsessiondirectory-win32-tssessiondirectory.md)
</dt> <dd>

Verifica se o servidor do agente de Conexão de Área de Trabalho Remota (agente de conexão RD) está disponível.

</dd> <dt>

[**Método SetCurrentRedirectableAddresses**](setcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Define a lista configurada de endereços DNS qualificados que podem ser usados para redirecionamento.

</dd> <dt>

[**Método SetLoadBalancingState**](setloadbalancingstate-win32-tssessiondirectory.md)
</dt> <dd>

Define o valor para indicar se o servidor participará do balanceamento de carga do agente de conexão RD.

</dd> <dt>

[**Método SetServerWeight**](setserverweight-win32-tssessiondirectory.md)
</dt> <dd>

Define o valor de peso do servidor para o balanceamento de carga do agente de conexão RD.

</dd> <dt>

[**Método SetSessionDirectoryActive**](win32-tssessiondirectory-setsessiondirectoryactive.md)
</dt> <dd>

O método **SetSessionDirectoryActive** define a propriedade **SessionDirectoryActive** .

</dd> <dt>

[**Método SetSessionDirectoryExposeServerIP**](win32-tssessiondirectory-setsessiondirectoryexposeserverip.md)
</dt> <dd>

Define a propriedade **SessionDirectoryExposeServerIP** .

</dd> <dt>

[**Método SetSessionDirectoryProperty**](win32-tssessiondirectory-setsessiondirectoryproperty.md)
</dt> <dd>

Define a propriedade **SessionDirectoryLocation** ou a propriedade **SessionDirectoryClusterName** para a classe.

</dd> <dt>

[**Método SetTSRedirectorMode**](settsredirectormode-win32-tssessiondirectory.md)
</dt> <dd>

Define o valor para indicar se o servidor atuará como um redirecionador Serviços de Área de Trabalho Remota.

</dd> </dl>
