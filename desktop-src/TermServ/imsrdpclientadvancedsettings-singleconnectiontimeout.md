# <a name="imsrdpclientadvancedsettingssingleconnectiontimeout-property"></a>Propriedade IMsRdpClientAdvancedSettings:: singleConnectionTimeout

Especifica o período máximo de tempo, em segundos, que o controle de cliente espera por uma conexão com um endereço IP.

Durante a conexão, o controle pode tentar se conectar a vários endereços IP.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_singleConnectionTimeout(
  [in]  LONG singleConnectionTimeout
);

HRESULT get_singleConnectionTimeout(
  [out] LONG *psingleConnectionTimeout
);
```



## <a name="property-value"></a>Valor da propriedade

O novo horário, em segundos. O valor máximo é 600.

## <a name="error-codes"></a>Códigos do Erro

Retornará **S \_ OK** se for bem-sucedido.

## <a name="remarks"></a>Comentários

Para obter mais informações sobre Conexão Web de Área de Trabalho Remota, consulte [Requirements for conexão Web de área de trabalho remota](requirements-for-remote-desktop-web-connection.md).
