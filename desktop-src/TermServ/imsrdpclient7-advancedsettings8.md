# <a name="imsrdpclient7advancedsettings8-property"></a>Propriedade IMsRdpClient7:: AdvancedSettings8

Recupera um objeto que dá suporte à interface [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md) .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_AdvancedSettings8(
  [out, retval] IMsRdpClientAdvancedSettings7 **ppAdvSettings
);
```



## <a name="property-value"></a>Valor da propriedade

O endereço de um ponteiro de interface [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md) que recebe o objeto.
