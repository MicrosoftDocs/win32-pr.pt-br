# <a name="itssbtaskpluginnotifysink-methods"></a>Métodos ITsSbTaskPluginNotifySink

As interfaces [**ITsSbTaskPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink) expõem os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método OnDeleteTaskTime**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime)
</dt> <dd>

Notifica o agente de Conexão de Área de Trabalho Remota (agente de conexão RD) que uma tarefa foi removida da fila.

</dd> <dt>

[**Método OnReportTasks**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks)
</dt> <dd>

Notifica o agente de conexão RD de um novo relatório de tarefa.

</dd> <dt>

[**Método OnSetTaskTime**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime)
</dt> <dd>

Notifica o agente de conexão de área de trabalho remota que uma tarefa foi agendada.

</dd> <dt>

[**Método OnUpdateTaskStatus**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus)
</dt> <dd>

Notifica o agente de conexão RD que o status de uma tarefa foi alterado.

</dd> </dl>
