# <a name="remote-desktop-services-api-data-types"></a>Tipos de dados de API Serviços de Área de Trabalho Remota

Serviços de Área de Trabalho Remota define os tipos de dados a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**identificador de TLS \_**](tls-handle.md)
</dt> <dd>

Representa um identificador para um servidor de licença Área de Trabalho Remota.

</dd> <dt>

[**WTSLISTENERNAME**](wtslistenername.md)
</dt> <dd>

Representa o nome de um ouvinte de Serviços de Área de Trabalho Remota em um servidor de Host da Sessão da Área de Trabalho Remota (Host da Sessão RD).

</dd> </dl>
