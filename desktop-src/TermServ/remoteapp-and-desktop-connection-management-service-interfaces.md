# <a name="remoteapp-and-desktop-connection-management-service-interfaces"></a>Interfaces de Serviço Gerenciamento de Conexão de RemoteApp e Área de Trabalho

A API do serviço de Gerenciamento de Conexão de RemoteApp e Área de Trabalho fornece as seguintes interfaces.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**ItsPubPlugin**](/windows/desktop/api/tspubplugincom/nn-tspubplugincom-itspubplugin)
</dt> <dd>

Expõe propriedades e métodos que fornecem informações sobre recursos disponíveis para usuários de conexões de RemoteApp e área de trabalho.

</dd> <dt>

[**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2)
</dt> <dd>

Especifica métodos que fornecem informações sobre recursos disponíveis para usuários de conexões de RemoteApp e área de trabalho.

</dd> </dl>
