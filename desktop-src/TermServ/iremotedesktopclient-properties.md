# <a name="iremotedesktopclient-properties"></a>Propriedades de IRemoteDesktopClient

A interface [**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) expõe as propriedades a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Propriedade Actions**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_actions)
</dt> <dd>

Recupera o objeto Actions para o cliente de contêiner de aplicativo protocolo RDP (RDP).

</dd> <dt>

[**Propriedade de configurações**](iremotedesktopclient-settings.md)
</dt> <dd>

Recupera o objeto de configurações para o cliente de contêiner de aplicativo protocolo RDP (RDP).

</dd> <dt>

[**Propriedade TouchPointer**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_touchpointer)
</dt> <dd>

Contém o objeto [**RemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) para o cliente de contêiner de aplicativo protocolo RDP (RDP).

</dd> </dl>
