# <a name="iremotedesktopclientactions-methods"></a>Métodos IRemoteDesktopClientActions

A interface [**IRemoteDesktopClientActions**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientactions) expõe os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método ExecuteRemoteAction**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-executeremoteaction)
</dt> <dd>

Faz com que uma ação seja executada na sessão remota.

</dd> <dt>

[**Método GetSnapshot**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-getsnapshot)
</dt> <dd>

Faz com que um instantâneo da área de trabalho na sessão do cliente do contêiner de aplicativo protocolo RDP (RDP) seja usado.

</dd> <dt>

[**Método ResumeScreenUpdates**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-resumescreenupdates)
</dt> <dd>

Retoma a envio de atualizações de tela ao cliente.

</dd> <dt>

[**Método SuspendScreenUpdates**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-suspendscreenupdates)
</dt> <dd>

Suspende as atualizações de tela que estão sendo enviadas ao cliente.

</dd> </dl>
