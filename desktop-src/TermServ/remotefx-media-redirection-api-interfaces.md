# <a name="remotefx-media-redirection-api-interfaces"></a>RemoteFX Interfaces da API de Redirecionamento de Mídia

A API RemoteFX redirecionamento de mídia oferece suporte às interfaces a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**IWTSBitmapRenderer**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderer)
</dt> <dd>

Usado por um plug-in de canal virtual dinâmico para renderizar bitmaps.

</dd> <dt>

[**IWTSBitmapRendererCallback**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderercallback)
</dt> <dd>

Um plug-in de canal virtual dinâmico implementa essa interface para ser notificado quando o tamanho da área de renderização muda.

</dd> <dt>

[**IWTSBitmapRenderService**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtsbitmaprenderservice)
</dt> <dd>

Esse serviço é usado para criar um mapeamento visual no cliente correspondente a uma janela mapeada no servidor.

</dd> <dt>

[**IWTSPluginServiceProvider**](/windows/desktop/api/tsvirtualchannels/nn-tsvirtualchannels-iwtspluginserviceprovider)
</dt> <dd>

Fornece uma maneira de os plug-ins de Canal Virtual Dinâmico consultarem vários Área de Trabalho Remota cliente.

</dd> </dl>
