# <a name="imsrdpclientadvancedsettingsbitmapcachesize-property"></a>Propriedade IMsRdpClientAdvancedSettings:: BitmapCacheSize

O tamanho, em kilobytes, do arquivo de cache de bitmap usado para bitmaps de 8 bits por pixel.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_BitmapCacheSize(
  [in]  LONG bitmapCacheSize
);

HRESULT get_BitmapCacheSize(
  [out] LONG *pbitmapCacheSize
);
```



## <a name="property-value"></a>Valor da propriedade

O novo tamanho do cache, em kilobytes. Os valores numéricos válidos são de 1 a 32, inclusive.

## <a name="error-codes"></a>Códigos do Erro

Retornará **S \_ OK** se for bem-sucedido.

## <a name="remarks"></a>Comentários

Para obter mais informações sobre Conexão Web de Área de Trabalho Remota, consulte [Requirements for conexão Web de área de trabalho remota](requirements-for-remote-desktop-web-connection.md).
