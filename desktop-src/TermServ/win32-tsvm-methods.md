# <a name="win32_tsvm-methods"></a>\_Métodos TSVm do Win32

A classe de [**\_ métodos TSVm do Win32**](win32-tsvm.md) expõe os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método QueryOfflineInformation**](queryofflineinformation-win32-tsvm.md)
</dt> <dd>

Recupera as propriedades do VDS (servidor de área de trabalho virtual) atual, mas somente quando o servidor estiver offline.

</dd> <dt>

[**Método DumpVmInfo**](dumpvminfo-win32-tsvm.md)
</dt> <dd>

Esse membro é para fins de teste interno e não deve ser usado em seu código.

</dd> <dt>

[**Método de exportação**](export-win32-tsvm.md)
</dt> <dd>

Recupera as informações de monitoramento da sessão de máquina virtual.

</dd> </dl>
