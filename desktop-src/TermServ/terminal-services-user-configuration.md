# <a name="remote-desktop-services-user-configuration"></a>Serviços de Área de Trabalho Remota configuração do usuário

As funções [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) e [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) permitem que você defina e recupere informações de configuração para uma conta de usuário especificada. O tipo de enumeração da [**\_ \_ classe WTS config**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) define os parâmetros de configuração do usuário.
