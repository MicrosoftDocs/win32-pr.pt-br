# <a name="remote-desktop-services-user-configuration"></a>Serviços de Área de Trabalho Remota configuração do usuário

As [**funções WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) e [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) permitem definir e recuperar informações de configuração para uma conta de usuário especificada. O [**WTS \_ enumeração CONFIG \_ CLASS**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) define os parâmetros de configuração do usuário.
