# <a name="remote-desktop-virtualization-enumerations"></a>Enumerações de virtualização Área de Trabalho Remota

A API de virtualização Área de Trabalho Remota dá suporte aos tipos de enumeração a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_tipo de alteração de destino \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_change_type)
</dt> <dd>

Especifica o tipo de alteração que ocorreu em um destino.

</dd> <dt>

[**Estado de destino \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_state)
</dt> <dd>

Indica o estado de um destino.

</dd> <dt>

[**tipo de destino \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_type)
</dt> <dd>

Indica se um destino pertence a um pool ou farm.

</dd> <dt>

[**\_estado TSSESSION**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssession_state)
</dt> <dd>

Indica o estado de uma sessão.

</dd> <dt>

[**TSSD \_ AddrV46Type**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssd_addrv46type)
</dt> <dd>

Indica o tipo de endereço IP.

</dd> <dt>

[**\_ \_ classificação de TS SB \_ por**](/windows/desktop/api/sbtsv/ne-sbtsv-ts_sb_sort_by)
</dt> <dd>

Especifica a ordem de classificação. Ele é usado como um parâmetro no método [**EnumerateTargets**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets) .

</dd> </dl>
