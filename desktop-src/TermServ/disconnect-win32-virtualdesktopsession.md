# <a name="disconnect-method-of-the-win32_virtualdesktopsession-class"></a>Método Disconnect da classe Win32 \_ VirtualDesktopSession

Desconecta a sessão de área de trabalho virtual.

## <a name="syntax"></a>Sintaxe


```mof
uint32 Disconnect();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna 0 em caso de êxito; caso contrário, retorna um código de erro WMI.
