# <a name="adsi-extension-for-remote-desktop-services-user-configuration"></a>Extensão ADSI para Serviços de Área de Trabalho Remota configuração de usuário

Uma extensão ADSI (interface do serviço de Active Directory) é um objeto COM especial que permite a extensão de um objeto COM. As extensões ADSI podem adicionar funcionalidade a um objeto de diretório e simplificar as tarefas de administração.

A extensão ADSI para Serviços de Área de Trabalho Remota configuração de usuário estende um objeto ADSI que permite que os administradores de domínio gravem scripts que automatizarão a manutenção de propriedades de usuário específicas Serviços de Área de Trabalho Remota.

A extensão ADSI para Serviços de Área de Trabalho Remota configuração do usuário está incluída no Windows Server.

Para mais informações, consulte os seguintes tópicos:

-   [Usando a extensão ADSI para Serviços de Área de Trabalho Remota configuração de usuário](using-the-adsi-extension-for-terminal-services-user-configuration.md)
-   [Referência para a extensão ADSI para Serviços de Área de Trabalho Remota configuração de usuário](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)

Para obter informações gerais sobre ADSI, consulte [extensões ADSI](/windows/desktop/ADSI/adsi-extensions), [o cache de atributo ADSI](/windows/desktop/ADSI/the-adsi-attribute-cache)e [Active Directory interfaces de serviço](/windows/desktop/ADSI/active-directory-service-interfaces-adsi).
