# <a name="iremotedesktopclienteventsonconnected-method"></a>Método IRemoteDesktopClientEvents:: onconnected

Chamado quando o controle de cliente se conectou a uma sessão remota.

## <a name="syntax"></a>Sintaxe


```C++
void OnConnected();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
