# <a name="imsrdpdevicecollection2-interface"></a>Interface IMsRdpDeviceCollection2

Representa uma coleção de objetos de dispositivo. Esse é um aprimoramento da interface [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md) .

## <a name="members"></a>Membros

A interface **IMsRdpDeviceCollection2** herda de [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md). **IMsRdpDeviceCollection2** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **IMsRdpDeviceCollection2** tem esses métodos.



| Método                                                                         | Descrição                                                                                                 |
|:-------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**AddDeviceByInstanceId**](imsrdpdevicecollection2-adddevicebyinstanceid.md) | Adiciona um dispositivo não listado à coleção de dispositivos.<br/>                                                |
| [**RedirectNow**](imsrdpdevicecollection2-redirectnow.md)                     | Força os dispositivos que foram selecionados ou desmarcados da coleção a serem redirecionados ou removidos.<br/> |
