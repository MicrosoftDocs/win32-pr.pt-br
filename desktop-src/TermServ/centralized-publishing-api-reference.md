# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>referência Gerenciamento de Conexão de RemoteApp e Área de Trabalho API de Serviço do Gerenciamento de Conexão de RemoteApp e Área de Trabalho

A API Gerenciamento de Conexão de RemoteApp e Área de Trabalho Serviço do Gerenciamento de Conexão de RemoteApp e Área de Trabalho dá suporte a uma interface e a uma estrutura que fornece informações sobre os recursos disponíveis para usuários de Conexões de Área de Trabalho e RemoteApp. Sua implementação dessa interface pode executar a filtragem personalizada de recursos e fornecer suporte para tipos de arquivo que não têm suporte no momento. Para obter mais informações, consulte estes tópicos.
