# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>Referência da API do Serviço Gerenciamento de Conexão de RemoteApp e Área de Trabalho

A API do serviço de Gerenciamento de Conexão de RemoteApp e Área de Trabalho dá suporte a uma interface e a uma estrutura que fornece informações sobre recursos disponíveis para usuários de conexões de RemoteApp e área de trabalho. Sua implementação dessa interface pode executar a filtragem personalizada de recursos e fornecer suporte para tipos de arquivos que não têm suporte no momento. Para obter mais informações, consulte os tópicos a seguir.
