# <a name="imsrdpclientnonscriptable3devicecollection-property"></a>IMsRdpClientNonScriptable3: Propriedade eviceCollection de:D

Recupera a coleção de objetos de dispositivo Plug and Play (PnP) a serem redirecionados.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_DeviceCollection(
  [out] IMSRdpDeviceCollection **ppDeviceCollection
);
```



## <a name="property-value"></a>Valor da propriedade

Recupera a coleção de objetos de dispositivo PnP do tipo [**IMSRdpDevice**](imsrdpdevice.md).
