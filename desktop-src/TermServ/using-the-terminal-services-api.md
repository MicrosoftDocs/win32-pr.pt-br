# <a name="using-the-remote-desktop-services-api"></a>Usando a API de Serviços de Área de Trabalho Remota

Esta seção descreve como você pode usar a API de Serviços de Área de Trabalho Remota para permitir que os aplicativos executem as seguintes tarefas em um ambiente de Serviços de Área de Trabalho Remota:

-   Executar [serviços de área de trabalho remota](terminal-services-administration.md) tarefas de administração, como enumerar e gerenciar servidores Host da Sessão da Área de Trabalho Remota (host da Sessão RD) em um domínio ou enumerar e gerenciar sessões e processos em um servidor host da Sessão RD.
-   Aprimorando aplicativos cliente/servidor em um ambiente Serviços de Área de Trabalho Remota.
-   [Usar serviços de área de trabalho remota canais virtuais](using-terminal-services-virtual-channels.md) para se comunicar entre módulos de cliente e de servidor de um aplicativo.
-   Configuração e recuperação de [serviços de área de trabalho remota](terminal-services-user-configuration.md) informações de configuração do usuário.

Se seu aplicativo for executado em um ambiente que não seja um ambiente Serviços de Área de Trabalho Remota, mas você quiser usar a API Serviços de Área de Trabalho Remota para fornecer funcionalidade adicional em um ambiente de Serviços de Área de Trabalho Remota, consulte [vinculação de tempo de execução para Wtsapi32.dll](run-time-linking-to-wtsapi32-dll.md).
