# <a name="itspubplugin2-methods"></a>Métodos ItsPubPlugin2

A interface [**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2) expõe os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método DeletePersonalDesktopAssignment**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

Chamado para excluir um mapeamento entre o usuário especificado e uma máquina virtual em uma coleção de área de trabalho virtual pessoal.

</dd> <dt>

[**Método GetResource2**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

Esse método é reservado e sempre deve retornar **E \_ NOTIMPL.**

</dd> <dt>

[**Método GetResource2List**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

Recupera uma lista de recursos atribuídos ao usuário especificado.

</dd> <dt>

[**Método ResolvePersonalDesktop**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

Chamado para resolver um mapeamento entre o usuário especificado e uma máquina virtual em uma coleção de área de trabalho virtual pessoal.

</dd> </dl>
