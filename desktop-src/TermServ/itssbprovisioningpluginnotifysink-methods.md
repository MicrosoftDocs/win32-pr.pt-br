# <a name="itssbprovisioningpluginnotifysink-methods"></a>Métodos ITsSbProvisioningPluginNotifySink

As interfaces [**ITsSbProvisioningPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioningpluginnotifysink) expõem os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método LockVirtualMachine**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-lockvirtualmachine)
</dt> <dd>

Notifica Conexão de Área de Trabalho Remota Broker (Agente de Conexão de RD) de que a máquina virtual está bloqueada.

</dd> <dt>

[**Método OnJobCancelled**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcancelled)
</dt> <dd>

Notifica o Agente de Conexão de RD de que o trabalho foi cancelado.

</dd> <dt>

[**Método OnJobCompleted**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcompleted)
</dt> <dd>

Notifica o Agente de Conexão de RD de que o trabalho foi concluído.

</dd> <dt>

[**Método OnJobCreated**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcreated)
</dt> <dd>

Notifica o Agente de Conexão de RD de que um trabalho de provisionamento é criado.

</dd> <dt>

[**Método OnVirtualMachineHostStatusChanged**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged)
</dt> <dd>

Notifica o Agente de Conexão de RD de que o status do host de uma máquina virtual foi alterado.

</dd> <dt>

[**Método OnVirtualMachineStatusChanged**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged)
</dt> <dd>

Notifica o Agente de Conexão de RD de que o status de uma máquina virtual foi alterado.

</dd> </dl>
