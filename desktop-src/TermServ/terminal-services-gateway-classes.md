# <a name="remote-desktop-gateway-classes"></a>Classes de gateway de Área de Trabalho Remota

O provedor WMI do gateway de Área de Trabalho Remota (Gateway RD) fornece as seguintes classes.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_TSGatewayConnection Win32**](win32-tsgatewayconnection.md)
</dt> <dd>

Representa uma conexão de um computador cliente com um servidor de gateway de área de trabalho remota.

</dd> <dt>

[**\_TSGatewayConnectionAuthorizationPolicy Win32**](win32-tsgatewayconnectionauthorizationpolicy.md)
</dt> <dd>

Descreve um Área de Trabalho Remota RD CAP (política de autorização de conexão). As RD CAPs são usadas para determinar se um usuário tem permissão para se conectar ao servidor de gateway de área de trabalho remota.

</dd> <dt>

[**\_TSGatewayLoadBalancer Win32**](win32-tsgatewayloadbalancer.md)
</dt> <dd>

Descreve um conjunto de servidores de balanceamento de carga do gateway de área de trabalho remota. Eles são usados para balancear a carga de conexões de gateway de área de trabalho remota em vários servidores.

</dd> <dt>

[**\_TSGatewayRADIUSServer Win32**](win32-tsgatewayradiusserver.md)
</dt> <dd>

Descreve um servidor serviço RADIUS (RADIUS), que tem um conjunto de Serviços de Área de Trabalho Remota RD CAPs (políticas de autorização de conexão).

</dd> <dt>

[**\_TSGatewayResourceAuthorizationPolicy Win32**](win32-tsgatewayresourceauthorizationpolicy.md)
</dt> <dd>

Descreve um Área de Trabalho Remota RD RAP (política de autorização de recursos). Uma RD RAP é usada para decidir se um usuário está autorizado a se conectar a um recurso especificado por meio do gateway de área de trabalho remota.

</dd> <dt>

[**\_TSGatewayResourceGroup Win32**](win32-tsgatewayresourcegroup.md)
</dt> <dd>

Descreve um grupo de recursos, que mapeia um conjunto de recursos (nomes de computador) para uma única entidade.

</dd> <dt>

[**\_TSGatewayServer Win32**](win32-tsgatewayserver.md)
</dt> <dd>

Usado para operações específicas do servidor de gateway de área de trabalho remota.

</dd> <dt>

[**\_TSGatewayServerSettings Win32**](win32-tsgatewayserversettings.md)
</dt> <dd>

Fornece métodos e propriedades para exibir e definir configurações do servidor de gateway de área de trabalho remota.

</dd> </dl>
