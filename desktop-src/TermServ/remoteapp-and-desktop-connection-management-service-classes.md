# <a name="remoteapp-and-desktop-connection-management-service-classes"></a>Classes de Serviço Gerenciamento de Conexão de RemoteApp e Área de Trabalho

O provedor WMI do serviço de Gerenciamento de Conexão de RemoteApp e Área de Trabalho fornece as seguintes classes.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_CentralPublishingChangeEvent Win32**](win32-centralpublishingchangeevent.md)
</dt> <dd>

Um evento que representa uma alteração nas configurações do RDV central.

</dd> <dt>

[**\_RDCentralPublishedDeploymentSettings Win32**](win32-rdcentralpublisheddeploymentsettings.md)
</dt> <dd>

Contém as configurações de implantação usadas para gerar arquivos RDP para recursos publicados de um farm.

</dd> <dt>

[**\_RDCentralPublishedFarm Win32**](win32-rdcentralpublishedfarm.md)
</dt> <dd>

A lista de farms da qual os desktops ou aplicativos foram publicados.

</dd> <dt>

[**\_RDCentralPublishedFileAssociation Win32**](win32-rdcentralpublishedfileassociation.md)
</dt> <dd>

Informações para uma extensão de arquivo associada a um aplicativo

</dd> <dt>

[**\_RDCentralPublishedRemoteApplication Win32**](win32-rdcentralpublishedremoteapplication.md)
</dt> <dd>

Descreve um aplicativo publicado em outro computador, para uso remoto por meio dos serviços de terminal.

</dd> <dt>

[**\_RDCentralPublishedRemoteDesktop Win32**](win32-rdcentralpublishedremotedesktop.md)
</dt> <dd>

Área de trabalho publicada em outro computador, para uso remoto por meio dos serviços de terminal

</dd> <dt>

[**\_RDPersonalDesktopAssignment Win32**](win32-rdpersonaldesktopassignment.md)
</dt> <dd>

A lista de atribuições de área de trabalho pessoal.

</dd> <dt>

[**\_TSCPubPlugin Win32**](win32-tscpubplugin.md)
</dt> <dd>

Representa um plug-in configurado para ser usado por meio do serviço de Gerenciamento de Conexão de RemoteApp e Área de Trabalho.

</dd> <dt>

[**\_TSLegacyPlugin Win32**](win32-tslegacyplugin.md)
</dt> <dd>

Representa um servidor Área de Trabalho Remota que os plug-ins de Serviço Gerenciamento de Conexão de RemoteApp e Área de Trabalho internos consultarão os programas do RemoteApp.

</dd> <dt>

[**\_Espaço de trabalho Win32**](win32-workspace.md)
</dt> <dd>

Especifica Serviços de Área de Trabalho Remota informações de configuração do espaço de trabalho.

</dd> </dl>
