# <a name="imsrdpclientadvancedsettingsenablewindowskey-property"></a>Propriedade IMsRdpClientAdvancedSettings::EnableWindowsKey

Especifica se a Windows chave pode ser usada na sessão remota.

Essa propriedade é leitura/gravação.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT put_EnableWindowsKey(
  [in]  LONG enableWindowsKey
);

HRESULT get_EnableWindowsKey(
  [out] LONG *penableWindowsKey
);
```



## <a name="property-value"></a>Valor da propriedade

De definir esse parâmetro como 0 para desabilitar o recurso ou um valor não zero para habilitar o recurso.

## <a name="error-codes"></a>Códigos do Erro

Retornará **S \_ OK** se for bem-sucedido.

## <a name="remarks"></a>Comentários

Para obter mais informações sobre Conexão Web de Área de Trabalho Remota, consulte [Requirements for Conexão Web de Área de Trabalho Remota](requirements-for-remote-desktop-web-connection.md).
