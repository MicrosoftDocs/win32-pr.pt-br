# <a name="iremotedesktopclient-methods"></a>Métodos IRemoteDesktopClient

A interface [**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) expõe os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método attachEvent**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-attachevent)
</dt> <dd>

Anexa um manipulador de eventos a um evento.

</dd> <dt>

[**Conexão método**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-connect)
</dt> <dd>

Inicia uma conexão usando as propriedades atualmente definidas no controle de cliente do contêiner de aplicativo protocolo RDP (RDP).

</dd> <dt>

[**Método DeleteSavedCredentials**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-deletesavedcredentials)
</dt> <dd>

Exclui as credenciais salvas para o computador remoto especificado.

</dd> <dt>

[**Método detachEvent**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-detachevent)
</dt> <dd>

Desconecta um manipulador de eventos de um evento.

</dd> <dt>

[**Método Disconnect**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-disconnect)
</dt> <dd>

Desconecta a conexão ativa.

</dd> <dt>

[**Método Reconectar**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-reconnect)
</dt> <dd>

Inicia uma reconexão automática do controle de cliente de contêiner de aplicativo protocolo RDP (RDP) para ajustar a sessão à nova largura e altura.

</dd> <dt>

[**Método UpdateSessionDisplaySettings**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-updatesessiondisplaysettings)
</dt> <dd>

Atualiza as configurações de largura e altura do controle de cliente protocolo RDP contêiner de aplicativo (RDP).

</dd> </dl>
