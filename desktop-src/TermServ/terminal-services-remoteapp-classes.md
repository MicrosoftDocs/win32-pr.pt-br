# <a name="remoteapp-classes"></a>Classes do RemoteApp

O provedor WMI do RemoteApp fornece as seguintes classes.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_RDAllowListFileAssociation Win32**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Descreve uma associação de tipo de arquivo publicado com um RemoteApp.

</dd> <dt>

[**\_RDFileAssociation Win32**](win32-rdfileassociation.md)
</dt> <dd>

Representa uma associação de tipo de arquivo para um RemoteApp.

</dd> <dt>

[**\_RemoteAppChangeEvent Win32**](win32-remoteappchangeevent.md)
</dt> <dd>

Representa uma alteração nas configurações do RemoteApp do Windows Server 2008 R2 no servidor de Host da Sessão da Área de Trabalho Remota (Host da Sessão RD).

</dd> <dt>

[**\_TSApplicationFileExtensions Win32**](win32-tsapplicationfileextensions.md)
</dt> <dd>

as extensões de nome de arquivo que são manipuladas por um aplicativo em um servidor de Host da Sessão RD.

</dd> <dt>

[**\_TSDeploymentLicensing Win32**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Configurações de licenciamento para a implantação de RDV (virtualização de Área de Trabalho Remota).

</dd> <dt>

[**\_TSDeploymentSettings Win32**](win32-tsdeploymentsettings.md)
</dt> <dd>

Define as configurações padrão que o Gerenciador RemoteApp usa ao criar arquivos de protocolo RDP (RDP).

</dd> <dt>

[**\_TSExpandEnvironmentVariables Win32**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Expande variáveis de ambiente definidas pelo sistema.

</dd> <dt>

[**\_TSGetIcon Win32**](win32-tsgeticon.md)
</dt> <dd>

Retorna o conteúdo do ícone especificado pelo caminho e o índice do arquivo

</dd> <dt>

[**\_TSPublishedApplication Win32**](win32-tspublishedapplication.md)
</dt> <dd>

Define os aplicativos que são disponibilizados para uso remoto por meio do Windows Server 2008 R2 RemoteApp.

</dd> <dt>

[**\_TSPublishedApplicationList Win32**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Representa a lista de programas que estão na lista de programas do RemoteApp em um servidor de Host da Sessão RD.

</dd> <dt>

[**\_TSRemoteDesktop Win32**](win32-tsremotedesktop.md)
</dt> <dd>

Descreve uma conexão de área de trabalho remota que está disponível por meio de Acesso via Web à Área de Trabalho Remota (RD Acesso via Web).

</dd> <dt>

[**\_TSStartMenuApplication Win32**](win32-tsstartmenuapplication.md)
</dt> <dd>

Descreve os aplicativos que estão no menu iniciar de um servidor Host da Sessão RD.

</dd> <dt>

[**\_TSSystemInfo Win32**](win32-tssysteminfo.md)
</dt> <dd>

Representa o serviço de informações do servidor de publicação centralizado

</dd> </dl>
