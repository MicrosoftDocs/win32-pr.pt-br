# <a name="itsremoteprogram3-interface"></a>Interface ITSRemoteProgram3

Define propriedades e métodos para uso com um RemoteApp.

## <a name="members"></a>Membros

A interface **ITSRemoteProgram3** herda de **ITSRemoteProgram2**. **ITSRemoteProgram3** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **ITSRemoteProgram3** tem esses métodos.



| Método                                                     | Descrição                                                  |
|:-----------------------------------------------------------|:-------------------------------------------------------------|
| [**ServerStartApp**](itsremoteprogram3-serverstartapp.md) | Especifica um aplicativo para iniciar na sessão remota. <br/> |
