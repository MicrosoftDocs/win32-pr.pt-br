# <a name="remote-desktop-services-api-enumeration-types"></a>Serviços de Área de Trabalho Remota enumeração de API do Serviços de Área de Trabalho Remota

Os seguintes tipos de enumeração são usados com a API Serviços de Área de Trabalho Remota dados.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_WTS CLASSE \_ CONFIG**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class)
</dt> <dd>

Contém valores que indicam o tipo de informações de configuração do usuário a ser definido ou recuperado em uma chamada para as funções [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) e [**WTSSetUserConfig.**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga)

</dd> <dt>

[**\_WTS ORIGEM DA \_ CONFIGURAÇÃO**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_source)
</dt> <dd>

Especifica a origem das informações de configuração retornadas pela [**função WTSQueryUserConfig.**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga)

</dd> <dt>

[**\_WTS CLASSE \_ CONNECTSTATE**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_connectstate_class)
</dt> <dd>

Especifica o estado de conexão de uma Serviços de Área de Trabalho Remota sessão.

</dd> <dt>

[**\_WTS CLASSE \_ INFO**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_info_class)
</dt> <dd>

Contém valores que indicam o tipo de informações de sessão a ser recuperado em uma chamada para a [**função WTSQuerySessionInformation.**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsquerysessioninformationa)

</dd> <dt>

[**\_WTS CLASSE \_ TYPE**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_type_class)
</dt> <dd>

Especifica o tipo de estrutura que uma função Serviços de Área de Trabalho Remota retornou em um buffer.

</dd> <dt>

[**\_WTS CLASSE \_ VIRTUAL**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_virtual_class)
</dt> <dd>

Contém valores que indicam o tipo de informações de canal virtual a ser recuperado.

</dd> <dt>

[**FAMÍLIA DE ENDEREÇOS WTSSBX \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_address_family)
</dt> <dd>

Contém valores que indicam a família de endereços de um endereço de rede que está sendo usado para redirecionamento.

</dd> <dt>

[**WTSSBX \_ MACHINE \_ DRAIN**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_drain)
</dt> <dd>

Contém valores que indicam o estado de descarregamento de um Host da Sessão da Área de Trabalho Remota (Host da Sessão RD) .

</dd> <dt>

[**MODO DE SESSÃO DO COMPUTADOR WTSSBX \_ \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_session_mode)
</dt> <dd>

Contém valores que indicam o modo de sessão de um Host da Sessão RD servidor.

</dd> <dt>

[**ESTADO DO COMPUTADOR WTSSBX \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_state)
</dt> <dd>

Contém valores que indicam o estado atual de um servidor.

</dd> <dt>

[**TIPO DE NOTIFICAÇÃO WTSSBX \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_notification_type)
</dt> <dd>

Contém valores que indicam o tipo de alteração de status que ocorreu em um servidor Host da Sessão RD ou em uma sessão de usuário.

</dd> <dt>

[**ESTADO DE SESSÃO WTSSBX \_ \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_session_state)
</dt> <dd>

Contém valores que indicam o estado de conexão de uma sessão de usuário.

</dd> </dl>
