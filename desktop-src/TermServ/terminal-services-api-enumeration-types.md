# <a name="remote-desktop-services-api-enumeration-types"></a>Serviços de Área de Trabalho Remota tipos de enumeração de API

Os tipos de enumeração a seguir são usados com a API Serviços de Área de Trabalho Remota.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_classe de configuração WTS \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class)
</dt> <dd>

Contém valores que indicam o tipo de informações de configuração do usuário a serem definidas ou recuperadas em uma chamada para as funções [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) e [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) .

</dd> <dt>

[**\_origem de configuração do WTS \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_source)
</dt> <dd>

Especifica a fonte de informações de configuração retornada pela função [**WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) .

</dd> <dt>

[**\_classe WTS connectstate \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_connectstate_class)
</dt> <dd>

Especifica o estado de conexão de uma sessão de Serviços de Área de Trabalho Remota.

</dd> <dt>

[**\_classe de informações de WTS \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_info_class)
</dt> <dd>

Contém valores que indicam o tipo de informações de sessão a serem recuperadas em uma chamada para a função [**WTSQuerySessionInformation**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsquerysessioninformationa) .

</dd> <dt>

[**\_classe de tipo WTS \_**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_type_class)
</dt> <dd>

Especifica o tipo de estrutura que uma função de Serviços de Área de Trabalho Remota retornou em um buffer.

</dd> <dt>

[**\_classe virtual \_ WTS**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_virtual_class)
</dt> <dd>

Contém valores que indicam o tipo de informações de canal virtual a serem recuperadas.

</dd> <dt>

[**\_família de endereços WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_address_family)
</dt> <dd>

Contém valores que indicam a família de endereços de um endereço de rede que está sendo usado para redirecionamento.

</dd> <dt>

[**\_drenagem de máquina WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_drain)
</dt> <dd>

Contém valores que indicam o estado de drenagem de um servidor de Host da Sessão da Área de Trabalho Remota (Host da Sessão RD).

</dd> <dt>

[**\_modo de \_ sessão do computador WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_session_mode)
</dt> <dd>

Contém valores que indicam o modo de sessão de um servidor de Host da Sessão RD.

</dd> <dt>

[**\_estado da máquina WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_machine_state)
</dt> <dd>

Contém valores que indicam o estado atual de um servidor.

</dd> <dt>

[**\_tipo de notificação WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_notification_type)
</dt> <dd>

Contém valores que indicam o tipo de alteração de status que ocorreu em um servidor de Host da Sessão RD ou uma sessão de usuário.

</dd> <dt>

[**\_estado de sessão WTSSBX \_**](/windows/win32/api/tssbx/ne-tssbx-wtssbx_session_state)
</dt> <dd>

Contém valores que indicam o estado de conexão de uma sessão de usuário.

</dd> </dl>
