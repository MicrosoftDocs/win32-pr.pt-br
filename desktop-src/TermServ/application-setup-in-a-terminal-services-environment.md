# <a name="application-setup"></a><span data-ttu-id="234ff-103">Instalação do aplicativo</span><span class="sxs-lookup"><span data-stu-id="234ff-103">Application setup</span></span>

<span data-ttu-id="234ff-104">O procedimento de configuração automatizada para muitos aplicativos existentes pressupõe que o aplicativo está sendo instalado para um único usuário.</span><span class="sxs-lookup"><span data-stu-id="234ff-104">The automated setup procedure for many existing applications assumes that the application is being installed for a single user.</span></span> <span data-ttu-id="234ff-105">Em um ambiente de Serviços de Área de Trabalho Remota multiusuário, essa suposição pode criar os seguintes problemas:</span><span class="sxs-lookup"><span data-stu-id="234ff-105">In a multiuser Remote Desktop Services environment, this assumption can create the following problems:</span></span>

-   <span data-ttu-id="234ff-106">Se o procedimento de instalação atualizar o ambiente do registro e da área de trabalho para apenas um usuário, os usuários adicionais deverão reinstalar o pacote inteiro ou o administrador deverá copiar manualmente as informações do registro e da área de trabalho de um usuário para os outros usuários.</span><span class="sxs-lookup"><span data-stu-id="234ff-106">If the setup procedure updates the registry and desktop environment for just one user, additional users must reinstall the entire package or an administrator must manually copy information from the registry and desktop of one user to the other users.</span></span>
-   <span data-ttu-id="234ff-107">Com alguns procedimentos de instalação, você pode personalizar o aplicativo no momento da instalação excluindo recursos.</span><span class="sxs-lookup"><span data-stu-id="234ff-107">With some setup procedures, you can customize the application at installation time by excluding features.</span></span> <span data-ttu-id="234ff-108">Se o instalador inicial excluir parte do aplicativo, os usuários adicionais deverão reinstalar o aplicativo para obter os recursos excluídos.</span><span class="sxs-lookup"><span data-stu-id="234ff-108">If the initial installer excludes part of the application, additional users must reinstall the application to get the excluded features.</span></span>

<span data-ttu-id="234ff-109">Para evitar esses problemas, os procedimentos de instalação devem usar as seguintes diretrizes ao instalar um aplicativo em um servidor de Host da Sessão da Área de Trabalho Remota (Host da Sessão RD):</span><span class="sxs-lookup"><span data-stu-id="234ff-109">To avoid these problems, setup procedures should use the following guidelines when installing an application on a Remote Desktop Session Host (RD Session Host) server:</span></span>

-   <span data-ttu-id="234ff-110">Instale aplicativos no ambiente de usuário padrão comum a todos os usuários.</span><span class="sxs-lookup"><span data-stu-id="234ff-110">Install applications into the default user environment common to all users.</span></span> <span data-ttu-id="234ff-111">Antes de instalar o aplicativo, execute o comando **Change User/install** console e, após a conclusão da instalação, execute o comando **Change User/execute** console.</span><span class="sxs-lookup"><span data-stu-id="234ff-111">Before installing the application, execute the **change user /install** console command, and after installation is complete, execute the **change user /execute** console command.</span></span> <span data-ttu-id="234ff-112">Use um script de compatibilidade Serviços de Área de Trabalho Remota para a instalação.</span><span class="sxs-lookup"><span data-stu-id="234ff-112">Use a Remote Desktop Services compatibility script for the installation.</span></span>
-   <span data-ttu-id="234ff-113">Dar suporte à personalização específica do usuário por meio do uso de perfis de usuário.</span><span class="sxs-lookup"><span data-stu-id="234ff-113">Support user-specific customization through the use of user profiles.</span></span> <span data-ttu-id="234ff-114">Para fazer isso, crie um [formato de arquivo de modelo administrativo](/previous-versions/windows/desktop/Policy/administrative-template-file-format) para que um administrador possa configurar o registro para indicar os recursos disponíveis para cada usuário.</span><span class="sxs-lookup"><span data-stu-id="234ff-114">To do this, create an [Administrative Template File Format](/previous-versions/windows/desktop/Policy/administrative-template-file-format) so an administrator can configure the registry to indicate the features available to each user.</span></span> <span data-ttu-id="234ff-115">Em seguida, em tempo de execução, o aplicativo pode habilitar ou desabilitar recursos dependendo das configurações nas configurações do registro do usuário atual.</span><span class="sxs-lookup"><span data-stu-id="234ff-115">Then, at run time, the application can enable or disable features depending on the settings in the current user's registry settings.</span></span> <span data-ttu-id="234ff-116">O aplicativo pode armazenar a configuração por usuário no hive **HKEY Current User** Registry e permitir que todos os usuários configurem o aplicativo de acordo com suas preferências.</span><span class="sxs-lookup"><span data-stu-id="234ff-116">The application can store the per user configuration in the **HKEY CURRENT USER** registry hive and let every user configure the application according to their preferences.</span></span>
