# <a name="imstscaxeventsondevicesbuttonpressed-method"></a>Método IMsTscAxEvents:: OnDevicesButtonPressed

Chamado quando o botão dispositivos na barra de conexão é pressionado.

## <a name="syntax"></a>Sintaxe


```C++
void OnDevicesButtonPressed();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.

## <a name="remarks"></a>Comentários

O botão dispositivos é exibido na barra de conexão quando o redirecionamento de USB é habilitado no cliente e no servidor.
