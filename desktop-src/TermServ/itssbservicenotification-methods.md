# <a name="itssbservicenotification-methods"></a>Métodos ITsSbServiceNotification

As interfaces [**ITsSbServiceNotification**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbservicenotification) expõem os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método NotifyServiceFailure**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure)
</dt> <dd>

Notifica os plug-ins registrados de que o serviço Conexão de Área de Trabalho Remota Broker (Agente de Conexão RD) foi interrompido.

</dd> <dt>

[**Método NotifyServiceSuccess**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess)
</dt> <dd>

Notifica os plug-ins registrados de que o serviço agente de conexão de área de trabalho trabalho foi iniciado.

</dd> </dl>
