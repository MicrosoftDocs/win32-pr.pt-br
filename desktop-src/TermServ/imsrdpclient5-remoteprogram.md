# <a name="imsrdpclient5remoteprogram-property"></a>Propriedade IMsRdpClient5:: RemoteProgram

Recupera um objeto que dá suporte à interface [**ITSRemoteProgram**](itsremoteprogram.md) .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_RemoteProgram(
  [out] ITSRemoteProgram **ppRemoteProgram
);
```



## <a name="property-value"></a>Valor da propriedade

Um ponteiro de interface [**ITSRemoteProgram**](itsremoteprogram.md) .
