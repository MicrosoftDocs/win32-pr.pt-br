# <a name="remote-desktop-protocol-provider-unions"></a>Uniões de provedor de protocolo RDP

A API de protocolo remoto personalizada dá suporte às seguintes uniões.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_configuração de conexão do WRDS \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_setting)
</dt> <dd>

Contém diferentes níveis de configurações para uma conexão de área de trabalho remota.

</dd> <dt>

[**\_configuração do ouvinte WRDS \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_setting)
</dt> <dd>

Contém diferentes níveis de configurações de ouvinte para uma conexão de área de trabalho remota.

</dd> <dt>

[**configuração de WRDS \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_setting)
</dt> <dd>

Contém diferentes níveis de configurações relacionadas à política para uma conexão de área de trabalho remota.

</dd> <dt>

[**WTS \_ \_desfazer estatísticas do cache \_**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wts_cache_stats_un)
</dt> <dd>

Contém estatísticas de cache.

</dd> </dl>
