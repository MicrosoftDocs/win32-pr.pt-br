# <a name="remote-desktop-activex-control-enumerations"></a>Área de Trabalho Remota ActiveX enumerações de controle

As enumerações a seguir são usadas com o Área de Trabalho Remota ActiveX controle .

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**ControlCloseStatus**](controlclosestatus.md)
</dt> <dd>

Indica se o aplicativo que contém o controle pode fechar o controle imediatamente.

</dd> <dt>

[**ControlReconnectStatus**](controlreconnectstatus.md)
</dt> <dd>

Especifica o resultado do método [**IMsRdpClient8::Reconnect.**](imsrdpclient8-reconnect.md)

</dd> <dt>

[**ClientSpec**](clientspec.md)
</dt> <dd>

Usado com a [**propriedade ClientProtocolSpec**](imsrdpclientadvancedsettings8-clientprotocolspec.md) para especificar o protocolo de área de trabalho remota usado entre o cliente e o servidor.

</dd> <dt>

[**ExtendedDisconnectReasonCode**](extendeddisconnectreasoncode.md)
</dt> <dd>

Define informações estendidas sobre o motivo da desconexão do controle.

</dd> <dt>

[**RedirectDeviceType**](redirectdevicetype.md)
</dt> <dd>

Usado para especificar o tipo de um dispositivo.

</dd> <dt>

[**RemoteActionType**](/windows/win32/api/rdpappcontainerclient/ne-rdpappcontainerclient-remoteactiontype)
</dt> <dd>

A ação a ser enviado para a sessão remota.

</dd> <dt>

[**RemoteSessionActionType**](remotesessionactiontype.md)
</dt> <dd>

Usado para especificar o tipo de ação remota.

</dd> <dt>

[**RemoteWindowDisplayedAttribute**](remotewindowdisplayedattribute.md)
</dt> <dd>

Usado com o método para especificar informações sobre o evento.

</dd> <dt>

[**SnapshotEncodingType**](/windows/win32/api/rdpappcontainerclient/ne-rdpappcontainerclient-snapshotencodingtype)
</dt> <dd>

O tipo de codificação usado para um instantâneo de cliente de contêiner de aplicativo protocolo RDP (RDP).

</dd> <dt>

[**SnapshotFormatType**](/windows/win32/api/rdpappcontainerclient/ne-rdpappcontainerclient-snapshotformattype)
</dt> <dd>

O formato de dados usado para um instantâneo de cliente de contêiner protocolo RDP aplicativo (RDP).

</dd> </dl>
