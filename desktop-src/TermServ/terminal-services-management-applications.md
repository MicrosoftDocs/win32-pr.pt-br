# <a name="remote-desktop-services-management-applications"></a>Serviços de Área de Trabalho Remota aplicativos de gerenciamento

O Serviços de Área de Trabalho Remota dá suporte aos seguintes aplicativos de gerenciamento.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[Extensão ADSI para Serviços de Área de Trabalho Remota configuração de usuário](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Uma extensão ADSI (interface do serviço de Active Directory) é um objeto COM especial que permite a extensão de um objeto COM.

</dd> <dt>

[Serviços de Área de Trabalho Remota permissões](terminal-services-permissions.md)
</dt> <dd>

Você pode usar as permissões fornecidas para Serviços de Área de Trabalho Remota para controlar como os usuários e grupos acessam o servidor.

</dd> </dl>

O [provedor WMI de serviços de área de trabalho remota](terminal-services-wmi-provider.md) fornece acesso de WMI a informações que os administradores podem configurar usando o snap-in do MMC (console de gerenciamento Microsoft) serviços de área de trabalho remota Configuration/Connections.

A [extensão ADSI para serviços de área de trabalho remota configuração de usuário](adsi-extension-for-terminal-services-user-configuration.md) estende um objeto ADSI que permite que os administradores de domínio gravem scripts que automatizarão a manutenção de propriedades de usuário específicas do serviços de área de trabalho remota.

Para obter uma lista das permissões que você pode definir e as tarefas que as permissões permitem, [serviços de área de trabalho remota permissões](terminal-services-permissions.md).

Para obter mais informações, consulte a [referência do provedor WMI serviços de área de trabalho remota](terminal-services-wmi-provider-reference.md) e [usando a extensão ADSI para serviços de área de trabalho remota configuração do usuário](using-the-adsi-extension-for-terminal-services-user-configuration.md).
