# <a name="remote-desktop-services-audioendpoint-enumeration-types"></a>Serviços de Área de Trabalho Remota tipos de enumeração AudioEndpoint

Os tipos de enumeração a seguir são usados com a API AudioEndpoint.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_sinalizadores de posição da AE \_**](/windows/desktop/api/Audioengineendpoint/ne-audioengineendpoint-ae_position_flags)
</dt> <dd>

Define constantes para a estrutura de [**\_ \_ posição atual da AE**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position) . Essas constantes descrevem o grau de validade da posição atual.

</dd> <dt>

[**\_sinalizadores de buffer APO \_**](/windows/desktop/api/Audioapotypes/ne-audioapotypes-apo_buffer_flags)
</dt> <dd>

Define os sinalizadores de validação de buffer para a estrutura de [**\_ \_ propriedade de conexão APO**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property) associada a cada conexão APO.

</dd> </dl>

## <a name="remarks"></a>Comentários

O Serviços de Área de Trabalho Remota API AudioEndpoint é para uso em cenários de Área de Trabalho Remota; Não é para aplicativos cliente.
