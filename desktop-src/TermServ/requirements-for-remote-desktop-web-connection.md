# <a name="requirements-for-remote-desktop-web-connection"></a>Requisitos para Conexão Web de Área de Trabalho Remota

Conexão Web de Área de Trabalho Remota requer dois sistemas configurados da seguinte maneira:

-   Um servidor host que executa o Windows e o IIS 4,0 ou posterior.
-   Um cliente que pode baixar e executar o Área de Trabalho Remota controle ActiveX. Qualquer cliente que esteja executando o Microsoft Internet Explorer 4, 1 (preferivelmente Service Pack 2) ou posterior pode executar o controle ActiveX Área de Trabalho Remota.

O usuário que está instalando o software deve ter privilégios suficientes para instalar o software. Por exemplo, no Windows XP, o usuário deve ser membro dos grupos usuários avançados ou administradores.
