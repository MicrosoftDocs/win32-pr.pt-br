# <a name="reference-for-the-adsi-extension-for-remote-desktop-services-user-configuration"></a>Referência da Extensão ADSI para configuração Serviços de Área de Trabalho Remota usuário

A Extensão ADSI para Serviços de Área de Trabalho Remota configuração do usuário permite que os administradores de domínio escrevam scripts para automatizar a manutenção Serviços de Área de Trabalho Remota propriedades específicas do usuário. Ele inclui a interface dupla a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**IADsTSUserEx**](/windows/desktop/api/Tsuserex/nn-tsuserex-iadstsuserex)
</dt> <dd>

Usado para examinar e configurar Serviços de Área de Trabalho Remota propriedades do usuário.

</dd> </dl>
