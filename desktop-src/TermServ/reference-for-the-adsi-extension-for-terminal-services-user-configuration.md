# <a name="reference-for-the-adsi-extension-for-remote-desktop-services-user-configuration"></a>Referência para a extensão ADSI para Serviços de Área de Trabalho Remota configuração de usuário

A extensão ADSI para Serviços de Área de Trabalho Remota configuração de usuário permite que os administradores de domínio gravem scripts para automatizar a manutenção de propriedades de usuário específicas de Serviços de Área de Trabalho Remota. Ele inclui a interface dupla a seguir.
