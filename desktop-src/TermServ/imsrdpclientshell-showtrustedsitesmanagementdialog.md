# <a name="imsrdpclientshellshowtrustedsitesmanagementdialog-method"></a>Método IMsRdpClientShell:: ShowTrustedSitesManagementDialog

Mostra a caixa de diálogo lista de sites confiáveis.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT ShowTrustedSitesManagementDialog();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Se esse método for bem sucedido, ele retornará **S \_ OK**. Caso contrário, ele retorna um código de erro **HRESULT** .
