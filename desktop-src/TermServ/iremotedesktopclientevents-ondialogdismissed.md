# <a name="iremotedesktopclienteventsondialogdismissed-method"></a>Método IRemoteDesktopClientEvents:: OnDialogDismissed

Chamado depois que uma caixa de diálogo exibida pelo controle de cliente é ignorada.

## <a name="syntax"></a>Sintaxe


```C++
void OnDialogDismissed();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
