# <a name="imsrdpclient10remoteprogram3-property"></a>Propriedade IMsRdpClient10:: RemoteProgram3

Um objeto que dá suporte à interface [**ITSRemoteProgram3**](itsremoteprogram3.md) .

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_RemoteProgram3(
  [out, retval] ITSRemoteProgram3 **ppRemoteProgram
);
```



## <a name="property-value"></a>Valor da propriedade

Retorna um ponteiro de interface [**ITSRemoteProgram3**](itsremoteprogram3.md) .
