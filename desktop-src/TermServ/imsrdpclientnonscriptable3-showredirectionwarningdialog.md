# <a name="imsrdpclientnonscriptable3showredirectionwarningdialog-property"></a>Propriedade IMsRdpClientNonScriptable3:: ShowRedirectionWarningDialog

Especifica ou recupera se a caixa de diálogo de aviso de redirecionamento deve ser mostrada.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_ShowRedirectionWarningDialog(
  [in]  VARIANT_BOOL fShowRdrDlg
);

HRESULT get_ShowRedirectionWarningDialog(
  [out] VARIANT_BOOL *pfShowRdrDlg
);
```



## <a name="property-value"></a>Valor da propriedade

Especifica se a caixa de diálogo de aviso de redirecionamento deve ser mostrada.
