# <a name="remote-desktop-connection-broker-client-api"></a>API de cliente do agente de Conexão de Área de Trabalho Remota

O agente de Conexão de Área de Trabalho Remota é um componente da implantação do Virtual Desktop Infrastructure (VDI), que, para uma conexão remota, encontrará um destino adequado de um pool de computadores em que a conexão pode ser redirecionada e retorna o endereço IP do destino. A API do cliente do agente de conexão permite que fornecedores de protocolo de terceiros aproveitem o agente de conexão para agilizar a manipulação de conexões usando seu protocolo para máquinas virtuais ou servidores Área de Trabalho Remota em um farm.
