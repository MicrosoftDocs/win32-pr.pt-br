# <a name="remote-desktop-connection-broker-client-api"></a>API de cliente Conexão de Área de Trabalho Remota Broker

O Conexão de Área de Trabalho Remota Broker é um componente da implantação do Virtual Desktop Infrastructure (VDI), que, para uma conexão remota, encontrará um destino adequado de um pool de computadores em que a conexão pode ser redirecionada e retorna o endereço IP do destino. A API de cliente do Agente de Conexão permite que fornecedores de protocolo de terceiros aproveitem o Agente de Conexão para agilizar o tratamento de conexões usando seu protocolo para máquinas virtuais ou Área de Trabalho Remota servidores em um farm.
