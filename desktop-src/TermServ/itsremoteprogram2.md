# <a name="itsremoteprogram2-interface"></a>Interface ITSRemoteProgram2

Define as propriedades para uso com um RemoteApp.

## <a name="members"></a>Membros

A interface **ITSRemoteProgram2** herda de [**ITSRemoteProgram**](itsremoteprogram.md). **ITSRemoteProgram2** também tem estes tipos de membros:

-   [Propriedades](#properties)

### <a name="properties"></a>Propriedades

A interface **ITSRemoteProgram2** tem essas propriedades.



| Propriedade                                                                                  | Tipo de acesso           | Descrição                                                 |
|:------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------|
| [**RemoteApplicationArgs**](itsremoteprogram2-remoteapplicationargs.md)<br/>       | Somente gravação<br/> | Os argumentos de linha de comando para o RemoteApp.<br/>    |
| [**RemoteApplicationName**](itsremoteprogram2-remoteapplicationname.md)<br/>       | Somente gravação<br/> | O nome do RemoteApp.<br/>                       |
| [**RemoteApplicationProgram**](itsremoteprogram2-remoteapplicationprogram.md)<br/> | Somente gravação<br/> | O caminho e o nome do arquivo do programa RemoteApp.<br/> |
