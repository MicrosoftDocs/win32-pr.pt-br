# <a name="iremotedesktopclientevents-methods"></a>Métodos IRemoteDesktopClientEvents

A interface [**IRemoteDesktopClientEvents**](iremotedesktopclientevents.md) expõe os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método onconnecting**](iremotedesktopclientevents-onconnecting.md)
</dt> <dd>

Chamado quando o controle de cliente tenta estabelecer uma conexão com uma sessão remota.

</dd> <dt>

[**Método onconnected**](iremotedesktopclientevents-onconnected.md)
</dt> <dd>

Chamado quando o controle de cliente se conectou a uma sessão remota.

</dd> <dt>

[**Método OnLoginCompleted**](iremotedesktopclientevents-onlogincompleted.md)
</dt> <dd>

Chamado quando o controle de cliente faz logon com êxito em uma sessão remota.

</dd> <dt>

[**Método OnDisconnect**](iremotedesktopclientevents-ondisconnected.md)
</dt> <dd>

Chamado quando o controle de cliente foi desconectado de uma sessão remota.

</dd> <dt>

[**Método onstatuschanged**](iremotedesktopclientevents-onstatuschanged.md)
</dt> <dd>

Chamado quando o controle de cliente atualizou seu status.

</dd> <dt>

[**Método OnAutoReconnecting**](iremotedesktopclientevents-onautoreconnecting.md)
</dt> <dd>

Chamado quando o controle de cliente tenta restabelecer automaticamente uma conexão com uma sessão remota.

</dd> <dt>

[**Método OnAutoReconnected**](iremotedesktopclientevents-onautoreconnected.md)
</dt> <dd>

Chamado quando o controle de cliente é reconectado automaticamente a uma sessão remota.

</dd> <dt>

[**Método OnDialogDisplaying**](iremotedesktopclientevents-ondialogdisplaying.md)
</dt> <dd>

Chamado antes de o controle exibir uma caixa de diálogo.

</dd> <dt>

[**Método OnDialogDismissed**](iremotedesktopclientevents-ondialogdismissed.md)
</dt> <dd>

Chamado depois que uma caixa de diálogo exibida pelo controle de cliente é ignorada.

</dd> <dt>

[**Método OnNetworkStatusChanged**](iremotedesktopclientevents-onnetworkstatuschanged.md)
</dt> <dd>

Chamado quando o status da rede é alterado.

</dd> <dt>

[**Método OnAdminMessageReceived**](iremotedesktopclientevents-onadminmessagereceived.md)
</dt> <dd>

Chamado quando o controle de cliente recebe uma mensagem administrativa.

</dd> <dt>

[**Método OnKeyCombinationPressed**](iremotedesktopclientevents-onkeycombinationpressed.md)
</dt> <dd>

Chamado quando as combinações de teclas especiais são pressionadas na sessão remota.

</dd> <dt>

[**Método OnRemoteDesktopSizeChanged**](iremotedesktopclientevents-onremotedesktopsizechanged.md)
</dt> <dd>

Chamado quando o tamanho da área de trabalho remota é alterado.

</dd> <dt>

[**Método OnTouchPointerCursorMoved**](iremotedesktopclientevents-ontouchpointercursormoved.md)
</dt> <dd>

Chamado quando o cursor do ponteiro de toque foi movido e a propriedade [**EventsEnabled**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled) está definida como true.

</dd> </dl>
