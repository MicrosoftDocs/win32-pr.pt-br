# <a name="imsrdpclientadvancedsettings3connectionbarshowminimizebutton-property"></a>Propriedade IMsRdpClientAdvancedSettings3:: ConnectionBarShowMinimizeButton

Especifica se o botão de **minimização** deve ser exibido na barra de conexão.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_ConnectionBarShowMinimizeButton(
  [in]  VARIANT_BOOL fShowMinimize
);

HRESULT get_ConnectionBarShowMinimizeButton(
  [out] VARIANT_BOOL *pfShowMinimize
);
```



## <a name="property-value"></a>Valor da propriedade

Defina esse parâmetro como **Variant \_ true** para exibir o botão **minimizar** e como **Variant \_ false** para desabilitar a exibição do botão **minimizar** .

## <a name="error-codes"></a>Códigos do Erro

Retornará **S \_ OK** se for bem-sucedido.

## <a name="remarks"></a>Comentários

Esta propriedade não pode ser definida enquanto o controle está conectado.

Para obter mais informações sobre Conexão Web de Área de Trabalho Remota, consulte [Requirements for conexão Web de área de trabalho remota](requirements-for-remote-desktop-web-connection.md).
