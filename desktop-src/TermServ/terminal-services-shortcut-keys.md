# <a name="remote-desktop-services-shortcut-keys"></a>Teclas de atalho Serviços de Área de Trabalho Remota

A seguir está uma lista de teclas de atalho Serviços de Área de Trabalho Remota.

Uma observação sobre chaves ausentes: muitos teclados compacto não contêm algumas chaves. Por exemplo, muitos laptops não têm uma chave de quebra dedicada. No entanto, normalmente têm atalhos de teclado que substituem chaves dedicadas. Essas substituições de chave são especificadas pelo fabricante do teclado, portanto, talvez seja necessário pesquisar as substituições de chave na documentação fornecida pelo seu fabricante de teclado ou laptop.

Há duas combinações possíveis de teclas de atalho que podem ser usadas em uma conexão de área de trabalho remota: as teclas de atalho padrão do Windows ou as teclas de atalho originalmente projetadas para a área de trabalho remota. Você pode definir quais teclas de atalho usar no computador local e remoto por meio do cliente Conexão de Área de Trabalho Remota (ou seja, a caixa de diálogo que aparece quando você clica no ícone de **conexão de área de trabalho remota** ). A partir daí, clique em **Mostrar opções** (se você não puder ver as opções) e, em seguida, clique na guia **recursos locais** . Na lista suspensa **aplicar combinações de chaves do Windows** , você tem três opções:

<dl> <dt>

<span id="On_this_computer"></span><span id="on_this_computer"></span><span id="ON_THIS_COMPUTER"></span>Neste computador
</dt> <dd>

as combinações de teclas padrão funcionarão somente no computador local. Você deve usar as combinações alternativas na área de trabalho remota.

</dd> <dt>

<span id="On_the_remote_computer"></span><span id="on_the_remote_computer"></span><span id="ON_THE_REMOTE_COMPUTER"></span>No computador remoto
</dt> <dd>

As combinações de teclas padrão só funcionarão na área de trabalho remota. Você deve usar as combinações alternativas no computador local. Observe que, depois de fechar a Conexão de Área de Trabalho Remota, seu computador local usará novamente os atalhos padrão do Windows.

</dd> <dt>

<span id="Only_when_using_the_full_screen"></span><span id="only_when_using_the_full_screen"></span><span id="ONLY_WHEN_USING_THE_FULL_SCREEN"></span>Somente ao usar a tela inteira
</dt> <dd>

As combinações de teclas padrão funcionarão em qualquer computador que tenha a área de trabalho completa; funcionalmente, isso significa que as combinações de teclas padrão funcionam para o computador local, a menos que você tenha a janela de Conexão de Área de Trabalho Remota no modo de tela inteira.

</dd> </dl>

Para obter mais informações sobre o usuário sobre Área de Trabalho Remota conexão, consulte [conexão de área de trabalho remota: perguntas](https://windows.microsoft.com/windows/remote-desktop-connection-faq#1TC=windows-8)frequentes.
