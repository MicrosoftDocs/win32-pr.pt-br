# <a name="whats-new-in-windows-server-2008-r2"></a>Novidades no Windows Server 2008 R2

Windows O Server 2008 R2 apresenta os novos elementos de programação a seguir para Serviços de Área de Trabalho Remota.




| | | Os serviços de terminal agora estão Serviços de Área de Trabalho Remota<br /> | Os Serviços de Terminal foram renomeados Serviços de Área de Trabalho Remota. Um servidor que tem o Host da Sessão da Área de Trabalho Remota (Host da Sessão RD) instalado agora é chamado de servidor Host da Sessão RD, em vez de um servidor de terminal.<br /><ul><li><a href="terminal-services-is-now-remote-desktop-services.md">Os serviços de terminal agora estão Serviços de Área de Trabalho Remota</a></li></ul> | | API Área de Trabalho Remota Virtualização do Área de Trabalho Remota<br /> | A API de Virtualização do Área de Trabalho Remota fornece enumerações, interfaces e estruturas que você pode usar para criar plug-ins personalizados que substituem a lógica de redirecionamento padrão do Conexão de Área de Trabalho Remota Broker (Agente de Conexão RD). O Agente de Conexão de Área de Trabalho Trabalho (anteriormente conhecido como Agente de Sessão TS) foi expandido para dar suporte a conexões com máquinas virtuais.<br /><ul><li><a href="using-the-remote-desktop-virtualization-api.md">Usando a API Área de Trabalho Remota Virtualização</a></li><li><a href="terminal-services-virtualization-api-reference.md">Referência Área de Trabalho Remota API de Virtualização do Área de Trabalho Remota</a></li></ul> | | provedor protocolo RDP de dados<br /> | A API protocolo RDP provedor pode ser usada para criar um protocolo que personalize a interação entre o serviço Serviços de Área de Trabalho Remota e os clientes da área de trabalho.<br /><ul><li><a href="creating-a-custom-remote-protocol.md">Criando um provedor protocolo RDP dados</a></li><li><a href="custom-remote-protocol-reference.md">referência protocolo RDP provedor de dados</a></li></ul> | | API Serviços de Área de Trabalho Remota AudioEndpoint<br /> | A API Serviços de Área de Trabalho Remota AudioEndpoint dá suporte à enumeração, interfaces e estruturas para registro de ponto de extremidade de áudio e transporte de dados.<br /><ul><li><a href="terminal-services-audioendpoint-api-reference.md">referência Serviços de Área de Trabalho Remota API do AudioEndpoint</a></li></ul> | | API Gerenciamento de Conexão de RemoteApp e Área de Trabalho Serviço do Gerenciamento de Conexão de RemoteApp e Área de Trabalho<br /> | A API de Serviço Gerenciamento de Conexão de RemoteApp e Área de Trabalho dá suporte a interfaces e estruturas que você pode usar para executar a filtragem personalizada de recursos e dar suporte a tipos de arquivo que não têm suporte nativo no Windows Server 2008 R2.<br /><ul><li><a href="centralized-publishing-api-reference.md">Referência Gerenciamento de Conexão de RemoteApp e Área de Trabalho API de Serviço do Gerenciamento de Conexão de RemoteApp e Área de Trabalho</a></li></ul> |
