# <a name="dynamic-virtual-channels-reference"></a>Referência de canais virtuais dinâmicos

As interfaces de cliente do DVC (canal virtual dinâmico) a seguir têm suporte pelo Serviços de Área de Trabalho Remota (anteriormente conhecido como serviços de terminal).

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[APIs de cliente do DVC](dvc-client-apis.md)
</dt> <dd>

As APIs de cliente de canal virtual dinâmico (DVC) são implementadas especificamente para o cliente de Conexão de Área de Trabalho Remota (RDC) da conexão.

</dd> <dt>

[APIs de servidor DVC](dvc-server-apis.md)
</dt> <dd>

As APIs de servidor de canal virtual dinâmico (DVC) são implementadas pelo servidor de Conexão de Área de Trabalho Remota (RDC) da conexão.

</dd> </dl>
