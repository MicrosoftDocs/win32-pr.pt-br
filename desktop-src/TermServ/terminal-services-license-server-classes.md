# <a name="remote-desktop-license-server-classes"></a>Área de Trabalho Remota classes de servidor de licença

O provedor WMI de servidor de licença Área de Trabalho Remota fornece as seguintes classes.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_TSIssuedLicense Win32**](win32-tsissuedlicense.md)
</dt> <dd>

Descreve instâncias de Serviços de Área de Trabalho Remota licenças de acesso para cliente por dispositivo (RDS CALs por dispositivo) emitidas de um servidor de licença Área de Trabalho Remota.

</dd> <dt>

[**\_TSLicenseKeyPack Win32**](win32-tslicensekeypack.md)
</dt> <dd>

Fornece métodos e propriedades para exibir e instalar Serviços de Área de Trabalho Remota pacotes de chaves de licença.

</dd> <dt>

[**\_TSLicenseReport Win32**](win32-tslicensereport.md)
</dt> <dd>

Fornece instâncias de Serviços de Área de Trabalho Remota relatórios de uso de licença de acesso para cliente por usuário (RDS por CAL) que são geradas no servidor de licença Área de Trabalho Remota e métodos para operações de geração de relatório de licença, busca e exclusão.

</dd> <dt>

[**\_TSLicenseReportEntry Win32**](win32-tslicensereportentry.md)
</dt> <dd>

Fornece detalhes do Serviços de Área de Trabalho Remota emitido por licença de acesso para cliente por usuário (RDS CAL por usuário).

</dd> <dt>

[**\_TSLicenseReportFailedPerUserEntry Win32**](win32-tslicensereportfailedperuserentry.md)
</dt> <dd>

Fornece detalhes sobre a licença de acesso para cliente de Serviços de Área de Trabalho Remota por usuário com falha (RDS CAL por usuário).

</dd> <dt>

[**\_TSLicenseReportFailedPerUserSummaryEntry Win32**](win32-tslicensereportfailedperusersummaryentry.md)
</dt> <dd>

Fornece detalhes dos domínios para os quais a emissão por usuário falhou.

</dd> <dt>

[**\_TSLicenseReportPerDeviceEntry Win32**](win32-tslicensereportperdeviceentry.md)
</dt> <dd>

Fornece detalhes sobre o Serviços de Área de Trabalho Remota com falha por licença de acesso para cliente por dispositivo (RDS CAL por dispositivo).

</dd> <dt>

[**\_TSLicenseReportSummaryEntry Win32**](win32-tslicensereportsummaryentry.md)
</dt> <dd>

Fornece um resumo das licenças de acesso para cliente de Serviços de Área de Trabalho Remota instaladas e emitidas por usuário (RDS CALs por usuário).

</dd> <dt>

[**\_TSLicenseServer Win32**](win32-tslicenseserver.md)
</dt> <dd>

Fornece métodos e propriedades para exibir e configurar o licenciamento de Área de Trabalho Remota (Licenciamento RD) em um servidor de licença Área de Trabalho Remota.

</dd> </dl>
