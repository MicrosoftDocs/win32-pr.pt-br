# <a name="itssbresourcenotificationex-methods"></a>Métodos ITsSbResourceNotificationEx

As interfaces [**ITsSbResourceNotificationEx**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbresourcenotificationex) expõem os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método NotifyClientConnectionStateChangeEx**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcenotificationex-notifyclientconnectionstatechangeex)
</dt> <dd>

Notifica os plug-ins registrados sobre alterações de estado em uma conexão de cliente.

</dd> <dt>

[**Método NotifySessionChangeEx**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcenotificationex-notifysessionchangeex)
</dt> <dd>

Notifica os plug-ins registrados sobre alterações de estado em um objeto de sessão.

</dd> <dt>

[**Método NotifyTargetChangeEx**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcenotificationex-notifytargetchangeex)
</dt> <dd>

Notifica os plug-ins registrados sobre alterações de estado em um objeto de destino.

</dd> </dl>
