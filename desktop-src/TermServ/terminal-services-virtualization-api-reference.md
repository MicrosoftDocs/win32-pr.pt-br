# <a name="remote-desktop-virtualization-api-reference"></a>Referência de API de virtualização Área de Trabalho Remota

A API de virtualização Área de Trabalho Remota dá suporte a interfaces e estruturas que você pode usar para criar plug-ins personalizados que substituem a lógica de redirecionamento padrão do Conexão de Área de Trabalho Remota Broker (agente de conexão RD). Os plug-ins personalizados podem redirecionar conexões para servidores Host da Sessão da Área de Trabalho Remota (Host da Sessão RD), bem como máquinas virtuais, áreas de trabalho virtuais, servidores de folha e áreas de trabalho físicas.
