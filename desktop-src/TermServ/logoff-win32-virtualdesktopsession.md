# <a name="logoff-method-of-the-win32_virtualdesktopsession-class"></a>Método logoff da classe Win32 \_ VirtualDesktopSession

Faz logoff do usuário da sessão de área de trabalho virtual.

## <a name="syntax"></a>Sintaxe


```mof
uint32 Logoff();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna 0 em caso de êxito; caso contrário, retorna um código de erro WMI.
