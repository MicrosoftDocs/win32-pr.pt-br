# <a name="imsrdpclientadvancedsettings8-interface"></a>Interface IMsRdpClientAdvancedSettings8

Expõe métodos e propriedades que gerenciam as configurações avançadas do controle ActiveX Área de Trabalho Remota.

Para obter uma instância dessa interface, use a propriedade [**IMsRdpClient8:: AdvancedSettings9**](imsrdpclient8-advancedsettings9.md) .

## <a name="members"></a>Membros

A interface **IMsRdpClientAdvancedSettings8** herda de [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md). **IMsRdpClientAdvancedSettings8** também tem estes tipos de membros:

-   [Propriedades](#properties)

### <a name="properties"></a>Propriedades

A interface **IMsRdpClientAdvancedSettings8** tem essas propriedades.



| Propriedade                                                                                  | Tipo de acesso           | Descrição                                                           |
|:------------------------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------|
| [**BandwidthDetection**](imsrdpclientadvancedsettings8-bandwidthdetection.md)<br/> | Leitura/gravação<br/> | Especifica se as alterações de largura de banda são detectadas automaticamente.<br/> |
