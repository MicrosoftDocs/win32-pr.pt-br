# <a name="imsrdpclientnonscriptable3drivecollection-property"></a>IMsRdpClientNonScriptable3::D Propriedade rivescollection

Recupera a coleção de objetos de unidade a ser redirecionada.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT get_DriveCollection(
  [out] IMsRdpDriveCollection **ppDriveCollection
);
```



## <a name="property-value"></a>Valor da propriedade

Recupera a coleção de objetos de unidade do tipo [**IMSRdpDrive**](imsrdpdrive.md).
