# <a name="imsrdpclientadvancedsettingsscalebitmapcachesbybpp-property"></a>Propriedade IMsRdpClientAdvancedSettings:: ScaleBitmapCachesByBPP

Não há suporte a esta propriedade.

Esta propriedade é de leitura/gravação.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_ScaleBitmapCachesByBPP(
  [in]  LONG bScale
);

HRESULT get_ScaleBitmapCachesByBPP(
  [out] LONG *pbScale
);
```



## <a name="property-value"></a>Valor da propriedade

Defina esse parâmetro como 0 para desabilitar o recurso ou um valor diferente de zero para habilitar o recurso.
