A interface **IMsRdpDriveV2** herda de [**IMsRdpDrive**](imsrdpdrive.md). **IMsRdpDriveV2** também tem estes tipos de membros:

-   [Propriedades](#properties)

A interface **IMsRdpDriveV2** tem essas propriedades.
