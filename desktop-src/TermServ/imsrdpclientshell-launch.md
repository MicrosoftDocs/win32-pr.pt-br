# <a name="imsrdpclientshelllaunch-method"></a>Método IMsRdpClientShell:: Launch

Inicia o conteúdo do arquivo remoto de Acesso via Web à Área de Trabalho Remota (RD Acesso via Web) ou de algum outro portal da Web.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT Launch();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Se esse método for bem sucedido, ele retornará **S \_ OK**. Caso contrário, ele retorna um código de erro **HRESULT** .
