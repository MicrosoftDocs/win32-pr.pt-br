# <a name="remoteapp-and-desktop-connection-management-service-structures"></a>Estruturas de Serviço Gerenciamento de Conexão de RemoteApp e Área de Trabalho

A API do serviço de Gerenciamento de Conexão de RemoteApp e Área de Trabalho fornece as estruturas a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**pluginResource**](/windows/win32/api/tspubplugincom/ns-tspubplugincom-pluginresource)
</dt> <dd>

Contém informações sobre um recurso que pode ser atribuído a usuários no Conexão de RemoteApp e Área de Trabalho.

</dd> <dt>

[**pluginResource2**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2)
</dt> <dd>

Contém informações adicionais sobre um recurso que pode ser atribuído a usuários no Conexão de RemoteApp e Área de Trabalho.

</dd> <dt>

[**pluginResource2FileAssociation**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2fileassociation)
</dt> <dd>

Contém informações sobre uma associação de arquivo no Conexão de RemoteApp e Área de Trabalho.

</dd> </dl>
