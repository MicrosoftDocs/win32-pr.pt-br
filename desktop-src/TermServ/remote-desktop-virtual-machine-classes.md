# <a name="remote-desktop-virtual-machine-classes"></a>Área de Trabalho Remota de Máquina Virtual

O Área de Trabalho Remota WMI de Máquina Virtual fornece as classes a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Win32 \_ TSVirtualDesktopServerSettings**](win32-tsvirtualdesktopserversettings.md)
</dt> <dd>

Contém informações de configuração para um Área de Trabalho Remota Host de Virtualização (Host de Virtualização de RD).

</dd> <dt>

[**Win32 \_ TSVm**](win32-tsvm.md)
</dt> <dd>

Representa uma Área de Trabalho Remota virtual.

</dd> <dt>

[**Win32 \_ TSVmMetadataAssociation**](win32-tsvmmetadataassociation.md)
</dt> <dd>

Representa uma associação entre uma Área de Trabalho Remota virtual e suas propriedades de metadados.

</dd> <dt>

[**Win32 \_ TSVmMetadataItem**](win32-tsvmmetadataitem.md)
</dt> <dd>

Representa um item de metadados para uma Área de Trabalho Remota virtual.

</dd> <dt>

[**Win32 \_ RdvhManagement**](win32-rdvhmanagement.md)
</dt> <dd>

Descreve um serviço Área de Trabalho Remota RDVH (Host Virtual).

</dd> </dl>
