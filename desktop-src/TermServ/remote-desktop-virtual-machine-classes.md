# <a name="remote-desktop-virtual-machine-classes"></a>Área de Trabalho Remota classes de máquina virtual

O provedor WMI de máquina virtual Área de Trabalho Remota fornece as seguintes classes.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_TSVirtualDesktopServerSettings Win32**](win32-tsvirtualdesktopserversettings.md)
</dt> <dd>

Contém informações de configuração para um servidor de Host de Virtualização de Área de Trabalho Remota (host de Virtualização RD).

</dd> <dt>

[**\_TSVm Win32**](win32-tsvm.md)
</dt> <dd>

Representa um Área de Trabalho Remota máquina virtual.

</dd> <dt>

[**\_TSVmMetadataAssociation Win32**](win32-tsvmmetadataassociation.md)
</dt> <dd>

Representa uma associação entre uma máquina virtual Área de Trabalho Remota e suas propriedades de metadados.

</dd> <dt>

[**\_TSVmMetadataItem Win32**](win32-tsvmmetadataitem.md)
</dt> <dd>

Representa um item de metadados para uma máquina virtual Área de Trabalho Remota.

</dd> <dt>

[**\_RdvhManagement Win32**](win32-rdvhmanagement.md)
</dt> <dd>

Descreve um serviço de gerenciamento de Área de Trabalho Remota host virtual (RDVH).

</dd> </dl>
