# <a name="remote-desktop-gateway-enumeration-types"></a>Tipos de enumeração de gateway Área de Trabalho Remota

A API do gateway de Área de Trabalho Remota (Gateway RD) dá suporte aos tipos de enumeração a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**AAAccountingDataType**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaaccountingdatatype)
</dt> <dd>

Especifica o tipo de evento do qual o método [**ITSGAccountingEngine::D oaccounting**](/windows/desktop/api/TSGPolicyEngine/nf-tsgpolicyengine-itsgaccountingengine-doaccounting) está sendo notificado.

</dd> <dt>

[**AAAuthSchemes**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-aaauthschemes)
</dt> <dd>

Especifica o tipo de autenticação usado para se conectar ao Gateway RD.

</dd> <dt>

[**PolicyAttributeType**](/windows/win32/api/tsgpolicyengine/ne-tsgpolicyengine-policyattributetype)
</dt> <dd>

Especifica as configurações de redirecionamento associadas a uma conexão.

</dd> </dl>
