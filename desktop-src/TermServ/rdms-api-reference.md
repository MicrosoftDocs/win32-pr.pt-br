# <a name="remote-desktop-management-services-provider"></a>Provedor de serviços de gerenciamento de Área de Trabalho Remota

O provedor de serviços de gerenciamento de Área de Trabalho Remota (RDMS) gerencia ambientes de VDI (Virtual Desktop Infrastructure).

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**\_RDMSCollectionProperties Win32**](win32-rdmscollectionproperties.md)
</dt> <dd>

Gerencia as propriedades de uma coleção de áreas de trabalho virtuais.

</dd> <dt>

[**\_RDMSDeploymentSettings Win32**](win32-rdmsdeploymentsettings.md)
</dt> <dd>

Gerencia configurações de implantação para uma coleção de áreas de trabalho virtuais.

</dd> <dt>

[**\_RDMSDesktopAssignment Win32**](win32-rdmsdesktopassignment.md)
</dt> <dd>

Descreve a atribuição da área de trabalho do usuário da coleção RD.

</dd> <dt>

[**\_RDMSEnvironment Win32**](win32-rdmsenvironment.md)
</dt> <dd>

Gerencia um ambiente de serviços de gerenciamento de Área de Trabalho Remota (RDMS).

</dd> <dt>

[**\_RDMSFileTypeAssociation Win32**](win32-rdmsfiletypeassociation.md)
</dt> <dd>

Gerencia uma associação de tipo de arquivo para um aplicativo publicado.

</dd> <dt>

[**\_RDMSJoinedNode Win32**](win32-rdmsjoinednode.md)
</dt> <dd>

Representa um nó de servidor gerenciado por serviços de gerenciamento de Área de Trabalho Remota (RDMS).

</dd> <dt>

[**\_RDMSManagementData Win32**](win32-rdmsmanagementdata.md)
</dt> <dd>

Sincroniza dados de publicação para serviços de gerenciamento de Área de Trabalho Remota (RDMS).

</dd> <dt>

[**\_RDMSPublishedApplication Win32**](win32-rdmspublishedapplication.md)
</dt> <dd>

Gerencia um aplicativo publicado.

</dd> <dt>

[**\_RDMSVirtualDesktop Win32**](win32-rdmsvirtualdesktop.md)
</dt> <dd>

Representa uma área de trabalho virtual.

</dd> <dt>

[**\_RDMSVirtualDesktopCollection Win32**](win32-rdmsvirtualdesktopcollection.md)
</dt> <dd>

Cria e gerencia uma coleção de áreas de trabalho virtuais.

</dd> <dt>

[**\_RDSHCollection Win32**](win32-rdshcollection.md)
</dt> <dd>

Gerencia uma coleção de hosts de sessão de Área de Trabalho Remota.

</dd> <dt>

[**\_RDSHServer Win32**](win32-rdshserver.md)
</dt> <dd>

Gerencia um servidor de Host da Sessão da Área de Trabalho Remota (RDSH).

</dd> <dt>

[**\_VirtualDesktopSession Win32**](win32-virtualdesktopsession.md)
</dt> <dd>

Gerencia uma sessão de área de trabalho virtual.

</dd> </dl>
