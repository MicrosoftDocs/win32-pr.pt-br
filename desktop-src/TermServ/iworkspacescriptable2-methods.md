# <a name="iworkspacescriptable2-methods"></a>Métodos IWorkspaceScriptable2

A interface [**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2) dá suporte aos métodos a seguir.

## <a name="in-this-section"></a>Nesta seção

<dl> <dt>

[**Método StartWorkspaceEx**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-startworkspaceex)
</dt> <dd>

Associa credenciais de usuário e certificados a uma ID de conexão; também contém elementos de interface do usuário e segurança adicionais.

</dd> <dt>

[**Método ResourceDismissed**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-resourcedismissed)
</dt> <dd>

Alerta o usuário de que um recurso foi desabilitado ou ignorado de outra forma.

</dd> </dl>
