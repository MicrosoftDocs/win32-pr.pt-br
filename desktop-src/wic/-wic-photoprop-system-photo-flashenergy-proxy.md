# <a name="systemphotoflashenergy-proxy"></a>System.Photo.FlashEnergy Proxy

O proxy de propriedade para a política de metadados de foto System.Photo.FlashEnergy.

### <a name="description"></a>Descrição

O proxy de propriedade retorna um par racional a por meio [do membro propvariANT](/windows/win32/api/propidlbase/ns-propidlbase-propvariant) uhVal.

### <a name="containers"></a>Contêineres

JPEG, TIFF

### <a name="read-only"></a>Somente leitura

Sim

### <a name="output-propvariant-type"></a>Tipo PROPVARIANT de saída

VT \_ UI8

### <a name="denominator"></a>Denominador

HighPart do uhVal.

### <a name="numerator"></a>Numerador

LowPart do uhVal.
