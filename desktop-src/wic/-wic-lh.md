# <a name="windows-imaging-component"></a>Windows Imaging Component

## <a name="purpose"></a>Finalidade

O Windows Imaging Component (WIC) é uma plataforma extensível que fornece uma API de nível baixo para imagens digitais.  O WIC dá suporte aos formatos de imagem da Web padrão, alta imagens de intervalo dinâmico e dados brutos da câmera.  O WIC também oferece suporte a recursos adicionais, como:

-   Suporte interno para formatos de metadados padrão
-   Estrutura extensível para codecs de imagem, formatos de pixel e formatos de metadados.
-   Ampla gama de suporte a formato de pixel.
-   Suporte de alta cor; incluindo um intervalo estendido de 30 bits, alta precisão de 30 bits e formatos de pixel de grande precisão de 48 bits e de gama ampla.
-   Decodificação de imagem progressiva.

## <a name="developer-audience"></a>Público do desenvolvedor

O WIC foi projetado para atender às necessidades das seguintes classes de desenvolvedores:

-   Desenvolvedores que lêem e gravam dados de imagem em um aplicativo.
-   Desenvolvedores que lêem e gravam metadados de imagem em um aplicativo.
-   Desenvolvedores que desejam a descoberta de codecs de tempo de execução para codecs de imagem criados recentemente ou implementados.
-   Desenvolvedores que projetam ou implementam novos codecs de imagem e formatos de metadados.
