# <a name="systemphotosubjectdistance-proxy"></a>Proxy System. Photo. SubjectDistance

O proxy de propriedade para a política de metadados de foto System. Photo. SubjectDistance.

### <a name="read-only"></a>Somente leitura

Sim

\_UI8 VT

### <a name="denominator"></a>Denominador

### <a name="numerator"></a>Numera
