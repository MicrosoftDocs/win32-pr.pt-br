# <a name="wic-samples"></a>Exemplos de WIC

Esta seção contém os exemplos Windows WIC (componente de imagens de dados).

O WIC fornece dois tipos de exemplos:

-   Exemplos: segmentos de código que usam vários recursos do WIC. O código-fonte do exemplo é entregue no SDK Windows dados.
-   Exemplos de código: exemplos mais curtos que demonstram um recurso específico. O código-fonte do exemplo não é fornecido com o SDK.
