# <a name="wic-samples"></a>Exemplos de WIC

Esta seção contém os exemplos do Windows Imaging Component (WIC).

O WIC fornece dois tipos de exemplos:

-   Exemplos: segmentos de código que usam vários recursos de WIC. O código-fonte do exemplo é entregue na SDK do Windows.
-   Exemplos de código: amostras mais curtas que demonstram um recurso específico. O código-fonte do exemplo não é fornecido com o SDK.
