# <a name="wic-structures"></a>Estruturas de WIC

esta seção contém informações sobre as estruturas do componente de Windows Imaging (WIC).

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                          | Descrição                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmappattern)<br/>             | Contém membros que identificam um padrão dentro de um arquivo de imagem que pode ser usado para identificar um formato específico.<br/>   |
| [**WICBitmapPlane**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplane)<br/>                            | Especifica o formato de pixel, o buffer, o Stride e o tamanho de um plano de componente para um formato de pixel planar.<br/>                |
| [**WICBitmapPlaneDescription**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplanedescription)<br/>      | Especifica o formato de pixel e o tamanho de um plano de componente.<br/>                                                          |
| [**WICDdsFormatInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsformatinfo)<br/>                        | Especifica o [**\_ formato dxgi**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) e as informações de bloco de um formato DDS.<br/>                  |
| [**WICDdsParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsparameters)<br/>                        | Especifica a dimensão de imagem DDS, o [**\_ formato dxgi**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) e o modo alfa dos dados contidos.<br/>  |
| [**WICImageParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicimageparameters)<br/>                    | Isso define os parâmetros que você pode usar para substituir os parâmetros padrão normalmente usados ao codificar uma imagem. <br/> |
| [**WICJpegFrameHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegframeheader)<br/>                    | Representa um cabeçalho de quadro JPEG.<br/>                                                                                    |
| [**WICJpegScanHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegscanheader)<br/>                      | Representa um cabeçalho de quadro JPEG.<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadataheader)<br/>           | Representa o cabeçalho de metadados.<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern)<br/>         | Representa um padrão de metadados.<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawcapabilitiesinfo)<br/> | Define os recursos de codec brutos.<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurve)<br/>               | Representa uma curva de Tom de imagem bruta.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurvepoint)<br/>     | Representa um ponto de curva de Tom de imagem bruto.<br/>                                                                           |
| [**WICRect**](/windows/desktop/api/Wincodec/ns-wincodec-wicrect)<br/>                               | Representa um retângulo para a API do WIC.<br/>                                                                                |
