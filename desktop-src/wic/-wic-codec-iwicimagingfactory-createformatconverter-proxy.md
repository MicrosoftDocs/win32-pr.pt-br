# <a name="iwicimagingfactory_createformatconverter_proxy-function"></a>Função proxy IWICImagingFactory \_ CreateFormatConverter \_

Função proxy para o [**método CreateFormatConverter.**](/windows/desktop/api/Wincodec/nf-wincodec-iwicimagingfactory-createformatconverter)

## <a name="syntax"></a>Sintaxe


```C++
HRESULT IWICImagingFactory_CreateFormatConverter_Proxy(
  _In_  IWICImagingFactory  *pFactory,
  _Out_ IWICFormatConverter **ppIFormatConverter
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

Tipo: **[ **IWICImagingFactory**](/windows/desktop/api/Wincodec/nn-wincodec-iwicimagingfactory)\***
