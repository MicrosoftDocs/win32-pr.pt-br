# <a name="id3dxprtbuffergetnumcoeffs-method"></a>Método ID3DXPRTBuffer:: GetNumCoeffs

Recupera o número de escalares por canal de cor usado na memória para armazenar amostras.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumCoeffs();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Retorna o número de escalares por canal de cor usado na memória para armazenar amostras.
