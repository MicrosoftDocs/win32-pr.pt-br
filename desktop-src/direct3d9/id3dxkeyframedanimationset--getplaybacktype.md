# <a name="id3dxkeyframedanimationsetgetplaybacktype-method"></a>Método ID3DXKeyframedAnimationSet:: getreproduzirtype

Obtém o tipo do loop de reprodução do conjunto de animação.

## <a name="syntax"></a>Sintaxe


```C++
D3DXPLAYBACK_TYPE GetPlaybackType();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **\_ tipo de D3DXPLAYBACK**](./d3dxplayback-type.md)**

Tipo do loop de reprodução do conjunto de animações. Consulte [**\_ tipo de D3DXPLAYBACK**](./d3dxplayback-type.md).
