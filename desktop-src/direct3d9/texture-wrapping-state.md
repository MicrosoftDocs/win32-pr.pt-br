# <a name="texture-wrapping-state-direct3d-9"></a>Estado de disposição da textura (Direct3D 9)

Os \_ Estados de RENDERIZAÇÃO D3DRS WRAP0 a D3DRS \_ WRAP7 habilitam e desabilitam o encapsulamento u-e-v para várias texturas na cascata multitextura do dispositivo. Você pode definir esses Estados de renderização como uma combinação dos \_ sinalizadores D3DWRAPCOORD 0, D3DWRAPCOORD \_ 1, D3DWRAPCOORD \_ 2 e D3DWRAPCOORD \_ 3 para habilitar o encapsulamento nas primeiras, segunda, terceira e quarta direções da textura. Use um valor de zero para desabilitar totalmente o encapsulamento. O encapsulamento de textura é desabilitado em todas as direções para todos os estágios de textura por padrão.

Para obter uma visão geral conceitual, consulte [disposição de textura (Direct3D 9)](texture-wrapping.md).
