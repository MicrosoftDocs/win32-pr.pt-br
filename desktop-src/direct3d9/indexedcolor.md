# <a name="indexedcolor"></a>IndexedColor

Consiste em um parâmetro de índice e uma cor RGBA e é usado para definir cores de vértice de malha. O índice define o vértice ao qual a cor é aplicada.

``` syntax
template IndexedColor
{
    < 1630B820-7842-11cf-8F52-0040333594A3 >
    DWORD index;
    ColorRGBA indexColor;
} 
```

Em que:

-   índice-um DWORD. Consulte a descrição acima.
-   indexColor-um modelo de ColorRGBA. Consulte [**ColorRGBA**](colorrgba.md).
