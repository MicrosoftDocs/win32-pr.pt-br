# <a name="id3dxpatchmeshgetcontrolverticesperpatch-method"></a>Método ID3DXPatchMesh:: GetControlVerticesPerPatch

Obtém o número de vértices de controle por patch.

## <a name="syntax"></a>Sintaxe


```C++
DWORD GetControlVerticesPerPatch();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **DWORD**](../winprog/windows-data-types.md)**

O número de vértices de controle por patch.
