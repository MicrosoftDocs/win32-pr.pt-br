# <a name="id3dxanimationcontrollergetpriorityblend-method"></a>Método ID3DXAnimationController:: GetPriorityBlend

Obtém o peso de mesclagem de prioridade atual usado pelo controlador de animação.

## <a name="syntax"></a>Sintaxe


```C++
FLOAT GetPriorityBlend();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **float**](../winprog/windows-data-types.md)**

Retorna o peso de mesclagem de prioridade atual.

## <a name="remarks"></a>Comentários

O peso de mesclagem de prioridade é usado para misturar faixas de prioridade alta e baixa.
