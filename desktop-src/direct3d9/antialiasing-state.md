# <a name="antialiasing-state-direct3d-9"></a>Estado de anti-aliasing (Direct3D 9)

A [anti-aliasing (Direct3D 9)](antialiasing.md) é um método de fazer com que as linhas e bordas pareçam mais suaves na tela. Por padrão, o Direct3D não executa a suavização. Para habilitar o anti-aliasing em toda a cena, defina o \_ estado de RENDERIZAÇÃO D3DRS MULTISAMPLEANTIALIAS como **true**. Para desabilitá-lo, defina D3DRS \_ MULTISAMPLEANTIALIAS como **false**.
