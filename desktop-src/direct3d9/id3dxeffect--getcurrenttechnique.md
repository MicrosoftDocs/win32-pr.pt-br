# <a name="id3dxeffectgetcurrenttechnique-method"></a>Método ID3DXEffect:: GetCurrentTechnique

Obtém a técnica atual.

## <a name="syntax"></a>Sintaxe


```C++
D3DXHANDLE GetCurrentTechnique();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[D3DXHANDLE](dx9-graphics-reference-effects-constants.md)**

Um identificador exclusivo para a técnica atual. Consulte [Handles (Direct3D 9)](handles.md).
