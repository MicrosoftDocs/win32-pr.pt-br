# <a name="id3dxkeyframedanimationsetgetnumrotationkeys-method"></a>Método ID3DXKeyframedAnimationSet:: GetNumRotationKeys

Obtém o número de chaves de rotação na animação do quadro chave especificado.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumRotationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Animação* \[ no\]
</dt> <dd>

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Índice de animação.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Número de chaves de rotação na animação do quadro chave especificado.
