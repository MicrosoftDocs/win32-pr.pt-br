# <a name="d3dcomposerects_maxnumrects"></a>D3DCOMPOSERECTS \_ MAXNUMRECTS

Especifica o número máximo de retângulos usados na composição de glifos em [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects).



|                              |                                                                                                                                      |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| \#definir                     | Descrição                                                                                                                          |
| D3DCOMPOSERECTS \_ MAXNUMRECTS | Número máximo de glifos de retângulo para compor em uma cadeia de texto. Consulte [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects). |
