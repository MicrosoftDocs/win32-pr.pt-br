# <a name="programming-guide-for-direct3d-9"></a>Guia de programação do Direct3D 9

Este guia contém uma descrição do pipeline de gráficos implementado pelo Microsoft Direct3D. Trata-se de um guia para os desenvolvedores que implementam a funcionalidade de gráficos do Direct3D em seus aplicativos. O guia contém descrições de arquitetura, diagramas de bloco funcionais e descrições dos blocos de construção no pipeline, bem como trechos de código e aplicativos de exemplo. As informações são divididas nas seguintes seções:

-   [Introdução](getting-started.md) -esta seção contém uma visão geral do pipeline e dos tutoriais que podem ajudá-lo a obter um aplicativo de gráficos simples em execução em alguns minutos.
-   [Efeitos](effects.md) – esta seção aborda os efeitos e os arquivos de efeito para a criação de aplicativos que podem ser executados em uma variedade de plataformas de hardware.
-   [Tópicos avançados](advanced-topics.md) – esta seção contém exemplos de diferentes tipos de efeitos especiais que podem ser implementados. Os tópicos como ambiente e mapeamento de relevo, anti-aliasing, mesclagem de vértice, interpolação, iluminação de alto intervalo dinâmico (HDR) e transferência de radiante (PRT) precomputado mostram como aplicar efeitos especiais de ponta para o seu aplicativo.
-   [Dicas de programação](programming-tips.md) -esta seção contém informações para ajudá-lo a desenvolver aplicativos gráficos do Direct3D com eficiência.

Para obter mais informações sobre métodos de API específicos, consulte as páginas de [referência](dx9-graphics-reference.md) .
