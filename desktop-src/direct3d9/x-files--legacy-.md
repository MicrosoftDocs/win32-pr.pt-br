# <a name="x-files-legacy-direct3d-9"></a>Arquivos X (herdado) (Direct3D 9)

O formato de arquivo X refere-se a arquivos com a extensão de nome de arquivo .x. Arquivos X foram introduzidos com o DirectX 2.0. Uma versão binária desse formato foi lançada posteriormente com o DirectX 3.0, que também é descrito nesta documentação. O DirectX 6.0 introduziu interfaces e métodos que permitem ler e escrever em arquivos .x.

Os arquivos X fornecem um formato baseado em modelo que permite o armazenamento de malhas, texturas, animações e objetos definíveis pelo usuário. O suporte para conjuntos de animação permite armazenar caminhos predefinidos para reprodução em tempo real. Também há suporte para instanciamento e hierarquias. A instanciamento permite várias referências a um objeto, como uma malha, ao armazenar seus dados apenas uma vez por arquivo. Hierarquias são usadas para expressar relações entre registros de dados.

O formato de arquivo .x fornece primitivos de dados de baixo nível nos quais os aplicativos definem primitivos de nível superior por meio de modelos.

Modelos tridimensionais criados nos aplicativos Maya do 3ds max ou Alias Wavefront podem ser convertidos em arquivos .x com as Extensões directX para \| Alias Maya.
