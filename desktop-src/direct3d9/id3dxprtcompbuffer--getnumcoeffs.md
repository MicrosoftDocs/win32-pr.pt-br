# <a name="id3dxprtcompbuffergetnumcoeffs-method"></a>Método ID3DXPRTCompBuffer:: GetNumCoeffs

Recupera o número de escalares por canal de cor usado na memória para armazenar amostras.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumCoeffs();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Retorna o número de escalares por canal de cor usado na memória para armazenar amostras.
