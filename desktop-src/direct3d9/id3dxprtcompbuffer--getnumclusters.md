# <a name="id3dxprtcompbuffergetnumclusters-method"></a>Método ID3DXPRTCompBuffer:: GetNumClusters

Recupera o número de clusters a serem usados para compactação.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumClusters();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Retorna o número de clusters a ser usado para compactação.
