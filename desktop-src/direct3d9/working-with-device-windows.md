# <a name="working-with-device-windows-direct3d-9"></a>Trabalhando com dispositivos Windows (Direct3D 9)

Esta seção lista um problema que você pode encontrar ao trabalhar com janelas de dispositivos em aplicativos Direct3D.

-   O Direct3D só conecta as janelas de foco em vez da janela do dispositivo com a função de processamento de mensagens do Direct3D e processa apenas as mensagens da janela de foco. Portanto, a janela de foco deve ser o pai de qualquer janela do dispositivo.
