# <a name="id3dxanimationcontrollergetmaxnumanimationoutputs-method"></a>Método ID3DXAnimationController:: GetMaxNumAnimationOutputs

Obter o número máximo de saídas de animação para as quais o controlador de animação pode dar suporte.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetMaxNumAnimationOutputs();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

O número máximo de saídas de animação que o controlador pode gerenciar.
