# <a name="effectparamdword"></a>EffectParamDWord

Define um efeito DWORD. Esse modelo substituiu o [**modelo EffectDWord.**](effectdword.md)

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Em que:

-   ParamName – Nome do parâmetro.
-   Valor – Valor do parâmetro.
