# <a name="effectparamdword"></a>EffectParamDWord

Define um efeito DWORD. Este modelo substituiu o modelo [**EffectDWord**](effectdword.md) .

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Em que:

-   ParamName-nome do parâmetro.
-   Valor de parâmetro de valor.
