# <a name="effect-reference-direct3d-9-graphics"></a>Referência de efeito (gráficos do Direct3D 9)

Os efeitos são uma abstração de nível superior que aproveita os sombreadores de vértice e pixel para encapsular os efeitos de renderização. Os efeitos podem ser escritos com a linguagem de alto nível ou com a linguagem do assembly do sombreador.

Aqui estão os tópicos da API.

-   [Interfaces de efeito](dx9-graphics-reference-effects-interfaces.md)
-   [Funções de efeito](dx9-graphics-reference-effects-functions.md)
-   [Estruturas de efeito](dx9-graphics-reference-effects-structures.md)
-   [Constantes de efeito](dx9-graphics-reference-effects-constants.md)
-   [Formato de efeito (Direct3D 9)](dx9-graphics-reference-effects-file-format.md)
-   [Referência de semântica e anotações padrão do DirectX](dx9-graphics-reference-effects-dxsas.md)

Como alternativa ao uso da API para criar efeitos, você também pode usar uma ferramenta de compilação de linha de comando. Confira [ferramenta de compilador de efeito](../direct3dtools/fxc.md).
