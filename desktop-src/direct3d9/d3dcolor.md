# <a name="d3dcolor"></a>D3DCOLOR

Define o tipo de cor do Direct3D fundamental.


```C++
typedef DWORD D3DCOLOR;
```



## <a name="remarks"></a>Comentários

Um valor de quatro bytes que normalmente representa os componentes alfa, vermelho, verde e azul de uma cor. Ele também pode representar a luminância e o brilho.

Você pode definir o tipo **D3DCOLOR** usando uma das macros a seguir.

-   [**\_ARGB D3DCOLOR**](d3dcolor-argb.md)
-   [**D3DCOLOR \_ AYUV**](d3dcolor-ayuv.md)
-   [**D3DCOLOR \_ ColorValue**](d3dcolor-colorvalue.md)
-   [**\_RGBA D3DCOLOR**](d3dcolor-rgba.md)
-   [**D3DCOLOR \_ XRGB**](d3dcolor-xrgb.md)
-   [**D3DCOLOR \_ XYUV**](d3dcolor-xyuv.md)
