# <a name="math-function-support-in-d3dx-direct3d-9"></a>Suporte à função matemática no D3DX (Direct3D 9)

D3DX é uma biblioteca de utilitários que fornece serviços auxiliares. É uma camada acima do componente do Direct3D.

## <a name="math"></a>Matemática

O suporte a matemática, contido em um conjunto de funções, é fornecido para:

-   Cálculos de cores
-   Aviões
-   Manipulação de matriz
-   Os quatérnios
-   vetores 2D
-   vetores 3D
-   vetores 4D

Observe que, quando combinado com as sobrecargas de C++, o suporte para tipos de matemática 3D básico é extensivo.

Para obter mais informações sobre essas funções, consulte [D3DX Functions](dx9-graphics-reference-d3dx-functions.md). Para ajudar a localizar a função de que você precisa, elas são divididas em várias pastas.

## <a name="float16"></a>FLOAT16

Ao usar o tipo de dados FLOAT16, certifique-se de limitar os valores para um máximo de D3DX \_ 16F \_ Max. Qualquer valor de FLOAT16 que exceda isso resultará em um comportamento indefinido no pipeline. Consulte [outras constantes D3DX](other-d3dx-constants.md).
