# <a name="math-function-support-in-d3dx-direct3d-9"></a>Suporte à função matemática no D3DX (Direct3D 9)

D3DX é uma biblioteca de utilitários que fornece serviços auxiliares. É uma camada acima do componente Direct3D.

## <a name="math"></a>Matemática

O suporte matemático, contido em um conjunto de funções, é fornecido para:

-   Cálculos de cores
-   Aviões
-   Manipulação de matriz
-   Quaternions
-   Vetores 2D
-   Vetores 3D
-   Vetores 4D

Observe que, quando juntamente com as sobrecargas do C++, o suporte para tipos de matemática 3D básicos é extenso.

Para obter mais informações sobre essas funções, consulte [Funções D3DX](dx9-graphics-reference-d3dx-functions.md). Para ajudar a encontrar a função de que você precisa, elas são divididas em várias pastas.

## <a name="float16"></a>FLOAT16

Ao usar o tipo de dados FLOAT16, certifique-se de limitar os valores a um máximo de D3DX \_ 16F \_ MAX. Qualquer valor FLOAT16 que exceder isso resultará em comportamento indefinido no pipeline. Consulte [Outras constantes D3DX.](other-d3dx-constants.md)
