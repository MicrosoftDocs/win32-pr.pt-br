# <a name="colorrgba"></a>ColorRGBA

Define um objeto de cor com um componente alfa. Isso é usado para a cor da face na definição do modelo de material.

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
