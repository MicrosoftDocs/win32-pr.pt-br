# <a name="dxfile-constants"></a>Constantes DXFILE

Os sinalizadores a seguir são usados para especificar em quais canais em uma textura operar. Preterido.

## <a name="dxfileformat"></a>DXFILEFORMAT



| \#Definir                 | Valor | Descrição      |
|--------------------------|-------|------------------|
| BINÁRIO DXFILEFORMAT \_     | 0     | Arquivo binário.     |
| DXFILEFORMAT \_ TEXT       | 1     | Arquivo de texto.       |
| DXFILEFORMAT \_ COMPACTADO | 2     | Arquivo compactado. |



 

Essas \# definições são declaradas em Dxfile.h.

## <a name="dxfileloadoptions"></a>DXFILELOADOPTIONS



| \#Definir                 | Valor | Descrição                  |
|--------------------------|-------|------------------------------|
| DXFILELOAD \_ FROMFILE     | 0x00L | Carregar um arquivo de um arquivo.     |
| DXFILELOAD \_ FROMRESOURCE | 0x01L | Carregar um arquivo de um recurso. |
| DXFILELOAD \_ FROMMEMORY   | 0x02L | Carregar um arquivo da memória.     |
| DXFILELOAD \_ FROMSTREAM   | 0x04L | Carregar um arquivo de um fluxo.   |
| DXFILELOAD \_ FROMURL      | 0x08L | Carregue um arquivo de uma URL.      |



 

Essas \# definições são declaradas em Dxfile.h.
