# <a name="d3dxvec2lengthsq-function"></a>Função D3DXVec2LengthSq

Retorna o quadrado do comprimento de um vetor 2D.

## <a name="syntax"></a>Sintaxe


```C++
FLOAT D3DXVec2LengthSq(
  _In_ const D3DXVECTOR2 *pV
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*VP* \[ no\]
</dt> <dd>

Tipo: **const [**D3DXVECTOR2**](d3dxvector2.md) \***

Ponteiro para a estrutura de [**D3DXVECTOR2**](d3dxvector2.md) de origem.

</dd> </dl>

## <a name="return-value"></a>Retornar valor

Tipo: **[ **float**](../winprog/windows-data-types.md)**

O comprimento quadrado do vetor.
