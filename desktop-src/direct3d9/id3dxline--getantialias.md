# <a name="id3dxlinegetantialias-method"></a>Método ID3DXLine:: GetAntialias

Obtém o estado de anti-aliasing da linha.

## <a name="syntax"></a>Sintaxe


```C++
BOOL GetAntialias();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **bool**](../winprog/windows-data-types.md)**

Retorna o valor da opção de anti-aliasing. **Verdadeiro** significa que a suavização está ativada e **false** significa que a suavização está desativada.
