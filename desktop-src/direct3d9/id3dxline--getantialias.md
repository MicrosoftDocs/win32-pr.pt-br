# <a name="id3dxlinegetantialias-method"></a>Método ID3DXLine::GetAntialias

Obtém o estado de aninhamento de linha.

## <a name="syntax"></a>Sintaxe


```C++
BOOL GetAntialias();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **BOOL**](../winprog/windows-data-types.md)**

Retorna o valor da opção de analiação. **TRUE** significa que a antialiasing está em e **FALSE** significa que a antialiação está desligada.
