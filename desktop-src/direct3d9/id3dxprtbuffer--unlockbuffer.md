# <a name="id3dxprtbufferunlockbuffer-method"></a>Método ID3DXPRTBuffer:: UnlockBuffer

Termina o ciclo de vida do ponteiro ppData retornado por [**ID3DXPRTBuffer:: LockBuffer**](id3dxprtbuffer--lockbuffer.md).

## <a name="syntax"></a>Sintaxe


```C++
HRESULT UnlockBuffer();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

Se o método for bem sucedido, o valor de retorno será D3D \_ OK.
