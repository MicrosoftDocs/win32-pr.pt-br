# <a name="id3dxprtcompbuffergetnumpca-method"></a>Método ID3DXPRTCompBuffer:: GetNumPCA

Recupera o número de vetores de base do PCA (análise de componente principal) a serem usados em cada cluster.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumPCA();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Retorna o número de vetores base do PCA (análise de componente principal) a serem usados em cada cluster.
