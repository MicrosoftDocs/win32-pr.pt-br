# <a name="d3dsincosconst2"></a>D3DSINCOSCONST2

Cria a segunda constante (no registro src3) para a instrução de sombreador de pixel [Sincos-PS](../direct3dhlsl/sincos---ps.md) .

``` syntax
#define D3DSINCOSCONST2 -0.020833334f, -0.12500000f, 1.0f, 0.50000000f
```

## <a name="return-value"></a>Valor Retornado

Retorna a segunda constante necessária para [Sincos-PS](../direct3dhlsl/sincos---ps.md).
