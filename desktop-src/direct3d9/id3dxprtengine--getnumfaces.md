# <a name="id3dxprtenginegetnumfaces-method"></a>Método ID3DXPRTEngine:: GetNumFaces

Recupera o número de faces na malha, incluindo quaisquer faces novas adicionadas como resultado da amostragem espacial adaptável.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumFaces();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Número de rostos na malha, incluindo quaisquer faces novas adicionadas como resultado da amostragem adaptável.
