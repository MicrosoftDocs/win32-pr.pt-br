# <a name="d3dx_channel"></a>\_Canal D3DX

Os sinalizadores a seguir são usados para especificar em quais canais em uma textura operar.



| \#definir                 | Descrição                                                                   |
|--------------------------|-------------------------------------------------------------------------------|
| D3DX do \_ canal \_ vermelho       | Indica que o canal vermelho deve ser usado.                                     |
| D3DX de \_ canal \_ azul      | Indica que o canal azul deve ser usado.                                    |
| D3DX de \_ canal \_ verde     | Indica que o canal verde deve ser usado.                                   |
| D3DX \_ canal \_ alfa     | Indica que o canal alfa deve ser usado.                                   |
| \_Luminância de canal D3DX \_ | Indica que as luminâncias dos canais vermelho, verde e azul devem ser usadas. |
