# <a name="d3dx_channel"></a>CANAL \_ D3DX

Os sinalizadores a seguir são usados para especificar em quais canais em uma textura operar.



| \#Definir                 | Descrição                                                                   |
|--------------------------|-------------------------------------------------------------------------------|
| CANAL D3DX \_ \_ VERMELHO       | Indica que o canal vermelho deve ser usado.                                     |
| CANAL D3DX \_ \_ AZUL      | Indica que o canal azul deve ser usado.                                    |
| CANAL D3DX \_ \_ VERDE     | Indica que o canal verde deve ser usado.                                   |
| CANAL D3DX \_ \_ ALFA     | Indica que o canal alfa deve ser usado.                                   |
| LUMINÂNCIA DO CANAL D3DX \_ \_ | Indica que as luminâncias dos canais vermelho, verde e azul devem ser usadas. |
