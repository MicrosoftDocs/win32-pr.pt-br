# <a name="other-d3dx-constants"></a>Outras constantes D3DX

Outras constantes D3DX incluem o seguinte:

## <a name="16-bit-floating-point-numbers"></a>Números de ponto flutuante de 16 bits



|                         |                                         |
|-------------------------|-----------------------------------------|
| \#definir                | Descrição                             |
| D3DX \_ 16F \_          | Número de dígitos decimais de precisão   |
| D3DX \_ 16F \_ Épsilon      | O menor é que 1,0 + Épsilon! = 1,0 |
| D3DX \_ 16F \_ Mant \_    | Número de bits da mantissa              |
| D3DX \_ 16F \_ Max          | Valor máximo                           |
| D3DX \_ 16F \_ Max \_ 10 \_ exp | Expoente decimal máximo                |
| \_Exp. \_ Max D3DX 16F \_     | Expoente binário máximo                 |
| D3DX \_ 16F \_ min          | Valor positivo mínimo                  |
| D3DX \_ 16F \_ min \_ 10 \_ exp | Expoente decimal mínimo                |
| D3DX \_ 16F \_ min \_ exp     | Expoente binário mínimo                 |
| D3DX \_ 16F \_ Radix        | Expoente base                          |
| D3DX \_ 16F \_ arredonda       | Adição de arredondamento: próximo                 |
| D3DX \_ 1BYPI             | 1/pi                                    |
| D3DX \_ PI                | pi                                      |
| D3DX \_ padrão \_ flutuante    | Valor float máximo                     |



 

Essas \# definições são declaradas em d3dx9. h e d3dx9math. h.

## <a name="prt-constants"></a>Constantes de PRT



|                  |                                            |
|------------------|--------------------------------------------|
| \#definir         | Descrição                                |
| D3DXSH \_ MINORDER | Ordem mais baixa permitida da simulação.  |
| D3DXSH \_ MAXORDER | Ordem mais alta permitida da simulação. |



 

Essas \# definições são declaradas em d3dx9math. h. Para obter mais informações sobre o PRT, consulte [transferência radiante de computação (Direct3D 9)](precomputed-radiance-transfer.md).

## <a name="texture-constants"></a>Constantes de textura



|                        |                                                                    |
|------------------------|--------------------------------------------------------------------|
| \#definir               | Descrição                                                        |
| D3DFMT \_ do \_ arquivo     | Pegue o formato exatamente de um arquivo.                               |
| D3DX \_ padrão          | Um valor padrão.                                                   |
| D3DX \_ padrão \_ NONPOW2 | Não arredondar números, como largura ou altura, para uma potência de dois. |
| D3DX \_ do \_ arquivo       | Pegue as dimensões de textura exatamente de um arquivo.                   |



 

Essas \# definições são declaradas em d3dx9. h.

## <a name="other-d3dx-constants"></a>Outras constantes D3DX

As constantes de versão UNUSED16 e D3DX \_ definidas em D3dx9mesh. h e D3dx9core. h são usadas internamente. Não use essas constantes.
