# <a name="id3dxprtenginefreebouncedata-method"></a>Método ID3DXPRTEngine:: FreeBounceData

Libera a memória usada para dados temporários de simulação de luz devolvida.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT FreeBounceData();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

O valor de retorno é S \_ OK.
