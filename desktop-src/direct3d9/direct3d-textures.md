# <a name="direct3d-textures-direct3d-9"></a>Texturas do Direct3D (Direct3D 9)

Texturas são uma ferramenta poderosa para criar realismo em imagens 3D geradas por computador. O Direct3D dá suporte a um amplo conjunto de recursos de texturização, fornecendo aos desenvolvedores acesso fácil a técnicas avançadas de texturização.

Esta seção o ajudará a começar a usar texturas.

-   [Conceitos básicos de texturing (Direct3D 9)](basic-texturing-concepts.md)
-   [Coordenadas de textura (Direct3D 9)](texture-coordinates.md)
-   [Filtragem de textura (Direct3D 9)](texture-filtering.md)
-   [Recursos de textura (Direct3D 9)](texture-resources.md)
-   [Disposição de textura (Direct3D 9)](texture-wrapping.md)
-   [Mesclagem de textura (Direct3D 9)](texture-blending.md)

Esses tópicos entrarão em mais detalhes sobre a funcionalidade adicional do texturing.

-   [Geração automática de mipmaps (Direct3D 9)](automatic-generation-of-mipmaps.md)
-   [Gerenciamento automático de textura (Direct3D 9)](automatic-texture-management.md)
-   [Recursos de textura compactados (Direct3D 9)](compressed-texture-resources.md)
-   [Considerações de hardware para texturing (Direct3D 9)](hardware-considerations-for-texturing.md)
-   [Recursos de textura de volume (Direct3D 9)](volume-texture-resources.md)

Para melhorar o desempenho, é recomendável usar texturas dinâmicas. Uma textura dinâmica pode ser bloqueada, gravada e desbloqueada em cada quadro. Consulte [usando texturas dinâmicas](performance-optimizations.md).
