# <a name="animationset"></a>Animaçãoset

Contém um ou mais objetos de [**animação**](animation.md) . Cada animação em um conjunto de animações tem a mesma hora em qualquer ponto determinado. Aumentar o tempo do conjunto de animações aumenta o tempo para todas as animações que ele contém.

``` syntax
template AnimationSet
{
    < 3D82AB50-62DA-11cf-AB39-0020AF71E433 >
    [ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > ]
} 
```

Em que:

-   \[ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > \] -modelo de animação opcional.
