# <a name="animationset"></a>AnimationSet

Contém um ou mais [**objetos Animation.**](animation.md) Cada animação dentro de um conjunto de animação tem o mesmo tempo em qualquer ponto determinado. Aumentar o tempo do conjunto de animação aumenta o tempo para todas as animações que ele contém.

``` syntax
template AnimationSet
{
    < 3D82AB50-62DA-11cf-AB39-0020AF71E433 >
    [ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > ]
} 
```

Em que:

-   \[ Animação < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > – modelo de animação \] opcional.
