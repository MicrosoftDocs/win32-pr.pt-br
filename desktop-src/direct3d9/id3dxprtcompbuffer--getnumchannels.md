# <a name="id3dxprtcompbuffergetnumchannels-method"></a>Método ID3DXPRTCompBuffer:: GetNumChannels

Recupera o número de canais de cores usados na memória para armazenar amostras.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumChannels();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Retorna o número de canais de cores usados na memória para armazenar amostras. O valor geralmente será 1 para representar valores de luminância ou 3 para representar valores RGB.
