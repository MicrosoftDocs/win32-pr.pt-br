# <a name="floatkeys"></a>FloatKeys

Define uma matriz de números de ponto flutuante (floats) e o número de floats nessa matriz. Isso é usado para definir conjuntos de chaves de animação.

``` syntax
template FloatKeys
{
    < 10DD46A9-775B-11cf-8F52-0040333594A3 >
    DWORD nValues;
    array float values[nValues];
} 
```

-   nValores-número de floats.
-   valores \[ nValores \] -matriz de valores float.
