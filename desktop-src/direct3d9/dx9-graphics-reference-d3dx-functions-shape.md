# <a name="shape-drawing-functions"></a>Funções de desenho de forma

O D3DX fornece as seguintes funções de desenho de forma.

-   [**D3DXCreateBox**](d3dxcreatebox.md)
-   [**D3DXCreateCylinder**](d3dxcreatecylinder.md)
-   [**D3DXCreatePolygon**](d3dxcreatepolygon.md)
-   [**D3DXCreateSphere**](d3dxcreatesphere.md)
-   [**D3DXCreateTeapot**](d3dxcreateteapot.md)
-   [**D3DXCreateText**](d3dxcreatetext.md)
-   [**D3DXCreateTorus**](d3dxcreatetorus.md)

Todas as formas são retornadas como objetos de malha.

Essas funções podem ser usadas em um sistema de coordenadas da mão esquerda ou da mão direita.
