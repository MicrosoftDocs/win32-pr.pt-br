# <a name="frametransformmatrix"></a>FrameTransformMatrix

Define uma transformação local para um quadro (e todos os seus objetos filho).

``` syntax
template FrameTransformMatrix
{
    < F6F23F41-7686-11cf-8F52-0040333594A3 >
    Matrix4x4 frameMatrix;
} 
```

Em que:

-   frameMatrix – um modelo Matrix4x4. Consulte [**Matrix4x4**](matrix4x4.md).
