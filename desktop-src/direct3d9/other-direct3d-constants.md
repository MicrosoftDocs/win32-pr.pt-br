## <a name="sdk-version"></a>Versão do SDK



|                     |
|---------------------|
| \#definir            |
| \_Versão do SDK do D3D \_   |
| \_Versão do SDK do D3D9b \_ |



 

Essas \# definições são declaradas em d3d9. h.

## <a name="other-constants"></a>Outras constantes

A tabela a seguir lista as constantes que são usadas internamente:



| \#definir                              | Valor                                             | Descrição                                                        |
|---------------------------------------|---------------------------------------------------|--------------------------------------------------------------------|
| \_ \_ RENDERTARGETS simultâneo máximo do D3D \_ | 4                                                 | O número máximo de rendertargets.                               |
| D3DDMAPSAMPLER                        | 256                                               | O número máximo de amostras de mapa de deslocamento.                    |
| D3DDP \_ MAXTEXCOORD                    | 8                                                 | O número máximo de coordenadas de textura.                         |
| MAXD3DDECLLENGTH                      | 64 (não inclui o elemento de vértice do marcador "End") | Número máximo de elementos em uma declaração de vértice.                |
| MAXD3DDECLUSAGEINDEX                  | 15                                                | O índice máximo (0-15) que pode ser usado em uma declaração de vértice. |



 

Essas \# definições são declaradas em d3d9types. h.
