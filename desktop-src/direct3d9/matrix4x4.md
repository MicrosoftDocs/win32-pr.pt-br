# <a name="matrix4x4"></a>Matrix4x4

Define uma matriz de 4 x 4. Isso é usado como uma matriz de transformação de quadro.

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

Em que:

-   matriz float \[ de array 16 \] -matriz de 16 floats.
