# <a name="shadowing-direct3d-9"></a>Sombreamento (Direct3D 9)

Os buffers de estêncil são uma técnica de buffer de profundidade que pode ser atualizada conforme a geometria é renderizada e usada novamente como uma máscara para desenhar mais geometria. Os efeitos comuns incluem espelhos, sombras (uma técnica avançada), elimina e assim por diante. O [exemplo ShadowVolume](https://msdn.microsoft.com/library/Ee418792(v=VS.85).aspx) usa buffers de estêncil para implementar sombras em tempo real. Informações adicionais estão contidas no [estêncil de duas laterais (Direct3D 9)](two-sided-stencil.md).
