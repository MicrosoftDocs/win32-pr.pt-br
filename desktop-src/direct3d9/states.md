# <a name="states-direct3d-9"></a>Estados (Direct3D 9)

Um dispositivo Direct3D usa Estados para configurar todo o processamento de pipeline. Os Estados de amostra controlam a configuração dos classificadores de textura definindo o endereçamento de textura e os modos de filtro, por exemplo. Estados de textura configuram o misturador de várias texturas que controla como as texturas são combinadas e mescladas umas com as outras. Os Estados de renderização controlam a maior parte do estado do pipeline.

Os blocos de estado são um mecanismo conveniente para salvar e restaurar o estado. Consulte [estado de salvamento e restauração de blocos de estado (Direct3D 9)](state-blocks-save-and-restore-state.md)
