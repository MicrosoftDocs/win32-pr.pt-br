# <a name="d3dissue_begin"></a>D3DISSUE \_ BEGIN

Essa macro cria um valor usado por [**Issue**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-issue) para emitir um início de consulta.

``` syntax
#define D3DISSUE_BEGIN (1 << 1)
```

## <a name="remarks"></a>Comentários

D3DISSUE \_ BEGIN é válido para o tipo de consulta a seguir.
