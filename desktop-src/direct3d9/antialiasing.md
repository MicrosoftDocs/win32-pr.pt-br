# <a name="antialiasing-direct3d-9"></a>Anti-aliasing (Direct3D 9)

A suavização é uma técnica que você pode usar para reduzir a aparência dos pixels da etapa de escada ao desenhar qualquer linha que não seja exatamente horizontal ou vertical. Em cenas tridimensionais, esse artefato é mais perceptível nos limites entre polígonos de cores diferentes. A suavização mistura efetivamente os pixels nesses limites para produzir uma aparência mais natural para a cena.
