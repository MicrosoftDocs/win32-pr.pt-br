# <a name="resource-relationships-direct3d-9"></a>Relações de recursos (Direct3D 9)

O diagrama a seguir mostra as operações que são sintaticamente possíveis em recursos específicos e seu conteúdo. Para obter mais informações sobre como usar recursos, consulte [manipulando recursos (Direct3D 9)](manipulating-resources.md).

![diagrama de relações de recursos](images/resourcerelationship.png)

As setas em execução da esquerda para a direita indicam que os tipos de destino são criados pelos métodos fornecidos, enquanto as setas em execução da direita para a esquerda indicam que esses tipos de recursos podem ser passados como argumentos para os métodos fornecidos.
