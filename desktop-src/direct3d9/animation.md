# <a name="animation-direct3d-9-graphics"></a>Animação (gráficos do Direct3D 9)

Contém animações que fazem referência a um quadro anterior. Ele deve conter uma referência a um quadro e ao menos um conjunto de modelos de [**AnimationKey**](animationkey.md) . Ele também pode conter um objeto de dados [**animationoptions**](animationoptions.md) .

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Em que:

-   \[ ... \] -Qualquer modelo de arquivo. x pode ser usado aqui. Isso torna a arquitetura extensível.
