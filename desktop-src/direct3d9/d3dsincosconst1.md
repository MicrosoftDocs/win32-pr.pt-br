# <a name="d3dsincosconst1"></a>D3DSINCOSCONST1

Cria a primeira constante (em registrar src2) para a instrução de sombreador de pixel [Sincos-PS](../direct3dhlsl/sincos---ps.md) .

``` syntax
#define D3DSINCOSCONST1  -1.5500992e-006f, -2.1701389e-005f,  0.0026041667f, 0.00026041668f
```

## <a name="return-value"></a>Valor Retornado

Retorna a primeira constante necessária para [Sincos-PS](../direct3dhlsl/sincos---ps.md).
