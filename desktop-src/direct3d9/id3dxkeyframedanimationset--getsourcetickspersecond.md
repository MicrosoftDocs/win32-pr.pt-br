# <a name="id3dxkeyframedanimationsetgetsourcetickspersecond-method"></a>Método ID3DXKeyframedAnimationSet:: GetSourceTicksPerSecond

Obtém o número de tiques de quadros-chave de animação que ocorrem por segundo.

## <a name="syntax"></a>Sintaxe


```C++
DOUBLE GetSourceTicksPerSecond();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **duplo**](../winprog/windows-data-types.md)**

Número de tiques de quadros-chave de animação que ocorrem por segundo.
