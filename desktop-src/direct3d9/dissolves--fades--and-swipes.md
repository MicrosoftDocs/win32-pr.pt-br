# <a name="dissolves-fades-and-swipes-direct3d-9"></a><span data-ttu-id="3ecde-103">Dessolucionar, esmaecer e passar o dedo (Direct3D 9)</span><span class="sxs-lookup"><span data-stu-id="3ecde-103">Dissolves, Fades, and Swipes (Direct3D 9)</span></span>

<span data-ttu-id="3ecde-104">Cada vez mais, aplicativos empregam efeitos especiais que são comumente usados em filmes e vídeo, como desintegração, passagem de dedo e desvanecimento.</span><span class="sxs-lookup"><span data-stu-id="3ecde-104">Increasingly, applications employ special effects that are commonly used in movies and video, such as dissolves, swipes, and fades.</span></span>

<span data-ttu-id="3ecde-105">Em uma desintegração, uma imagem é gradualmente substituída por outra em uma sequência de quadros suavizada.</span><span class="sxs-lookup"><span data-stu-id="3ecde-105">In a dissolve, one image is gradually replaced by another in a smooth sequence of frames.</span></span> <span data-ttu-id="3ecde-106">Embora o Direct3D forneça métodos de utilizar várias mesclagens de textura para conseguir o mesmo efeito, os aplicativos que utilizam o buffer de estêncil para desintegrações podem usar recursos de mesclagem de textura para outros efeitos enquanto eles fazem uma desintegração.</span><span class="sxs-lookup"><span data-stu-id="3ecde-106">Although Direct3D provides methods of using multiple texture blending to achieve the same effect, applications that use the stencil buffer for dissolves can use texture-blending capabilities for other effects while they do a dissolve.</span></span>

<span data-ttu-id="3ecde-107">Quando seu aplicativo executa uma desintegração, ele deve renderizar duas imagens diferentes.</span><span class="sxs-lookup"><span data-stu-id="3ecde-107">When your application performs a dissolve, it must render two different images.</span></span> <span data-ttu-id="3ecde-108">Ele usa o buffer de estêncil para controlar quais pixels de cada imagem são desenhadas na superfície de destino de renderização.</span><span class="sxs-lookup"><span data-stu-id="3ecde-108">It uses the stencil buffer to control which pixels from each image are drawn to the rendering target surface.</span></span> <span data-ttu-id="3ecde-109">Você pode definir uma série de máscaras de estêncil e copiá-las para o buffer de estêncil em quadros sucessivos.</span><span class="sxs-lookup"><span data-stu-id="3ecde-109">You can define a series of stencil masks and copy them into the stencil buffer on successive frames.</span></span> <span data-ttu-id="3ecde-110">Como alternativa, você pode definir uma máscara de estêncil de base para o primeiro quadro e alterá-la de forma incremental.</span><span class="sxs-lookup"><span data-stu-id="3ecde-110">Alternately, you can define a base stencil mask for the first frame and alter it incrementally.</span></span>

<span data-ttu-id="3ecde-111">No início da desintegração, seu aplicativo define a função de estêncil e máscara de estêncil para que a maioria dos pixels da imagem inicial passem no teste de estêncil.</span><span class="sxs-lookup"><span data-stu-id="3ecde-111">At the beginning of the dissolve, your application sets the stencil function and stencil mask so that most of the pixels from the starting image pass the stencil test.</span></span> <span data-ttu-id="3ecde-112">A maioria dos pixels da imagem final deve falhar no teste de estêncil.</span><span class="sxs-lookup"><span data-stu-id="3ecde-112">Most of the pixels from the ending image should fail the stencil test.</span></span> <span data-ttu-id="3ecde-113">Em quadros sucessivos, a máscara de estêncil é atualizada para que cada vez menos pixels da imagem inicial passem no teste.</span><span class="sxs-lookup"><span data-stu-id="3ecde-113">On successive frames, the stencil mask is updated so that fewer and fewer of the pixels in the starting image pass the test.</span></span> <span data-ttu-id="3ecde-114">À medida que os quadros progridem, cada vez menos pixels da imagem final falham no teste.</span><span class="sxs-lookup"><span data-stu-id="3ecde-114">As the frames progress, fewer and fewer of the pixels in the ending image fail the test.</span></span> <span data-ttu-id="3ecde-115">Dessa forma, seu aplicativo pode executar uma desintegração usando qualquer padrão arbitrário de desintegração.</span><span class="sxs-lookup"><span data-stu-id="3ecde-115">In this manner, your application can perform a dissolve using any arbitrary dissolve pattern.</span></span>

<span data-ttu-id="3ecde-116">O desvanecimento para dentro ou para fora é um caso especial de desintegração.</span><span class="sxs-lookup"><span data-stu-id="3ecde-116">Fading in or fading out is a special case of dissolving.</span></span> <span data-ttu-id="3ecde-117">Ao desvanecer para dentro, o buffer de estêncil é usado para desintegrar a partir de uma imagem preta ou branca para uma renderização de uma cena 3D.</span><span class="sxs-lookup"><span data-stu-id="3ecde-117">When fading in, the stencil buffer is used to dissolve from a black or white image to a rendering of a 3D scene.</span></span> <span data-ttu-id="3ecde-118">O desvanecimento para fora é o oposto, seu aplicativo é iniciado com uma renderização de uma cena 3D e se desintegra em preto e branco.</span><span class="sxs-lookup"><span data-stu-id="3ecde-118">Fading out is the opposite, your application starts with a rendering of a 3D scene and dissolves to black or white.</span></span> <span data-ttu-id="3ecde-119">O desvanecimento pode ser feito usando qualquer padrão arbitrário que você deseja empregar.</span><span class="sxs-lookup"><span data-stu-id="3ecde-119">The fade can be done using any arbitrary pattern you want to employ.</span></span>

<span data-ttu-id="3ecde-120">Os aplicativos Direct3D usam uma técnica semelhante para as passagens de dedo.</span><span class="sxs-lookup"><span data-stu-id="3ecde-120">Direct3D applications use a similar technique for swipes.</span></span> <span data-ttu-id="3ecde-121">Por exemplo, quando um aplicativo realiza uma passagem de dedo da esquerda para a direita, a imagem final parece deslizar gradualmente na parte superior da imagem inicial da esquerda para a direita.</span><span class="sxs-lookup"><span data-stu-id="3ecde-121">For example, when an application performs a left-to-right swipe, the ending image appears to slide gradually on top of the starting image from left to right.</span></span> <span data-ttu-id="3ecde-122">Assim como em uma desintegração, você deve definir uma série de máscaras de estêncil que são carregadas no buffer de estêncil em quadros sucessivos, ou modificar sucessivamente a máscara de estêncil inicial.</span><span class="sxs-lookup"><span data-stu-id="3ecde-122">As in a dissolve, you must define a series of stencil masks that are loaded into the stencil buffer on successive frames, or successively modify the starting stencil mask.</span></span> <span data-ttu-id="3ecde-123">As máscaras de estêncil são usadas para desativar a gravação de pixels da imagem inicial e para habilitar a gravação de pixels da imagem final.</span><span class="sxs-lookup"><span data-stu-id="3ecde-123">The stencil masks are used to disable the writing of pixels from the starting image and to enable the writing of pixels from the ending image.</span></span>

<span data-ttu-id="3ecde-124">Uma passagem de dedo é um pouco mais complexa do que uma desintegração em que seu aplicativo deve ler os pixels da imagem final na ordem inversa da passagem do dedo.</span><span class="sxs-lookup"><span data-stu-id="3ecde-124">A swipe is somewhat more complex than a dissolve in that your application must read pixels from the ending image in the reverse order of the swipe.</span></span> <span data-ttu-id="3ecde-125">Ou seja, se a passagem do dedo acontecer da esquerda para a direita, seu aplicativo deverá ler pixels da imagem final da direita para a esquerda.</span><span class="sxs-lookup"><span data-stu-id="3ecde-125">That is, if the swipe is moving from left to right, your application must read pixels from the ending image from right to left.</span></span>
