# <a name="id3dxkeyframedanimationsetgetnumscalekeys-method"></a>Método ID3DXKeyframedAnimationSet:: GetNumScaleKeys

Obtém o número de chaves de escala na animação do quadro chave especificado.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumScaleKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*Animação* \[ no\]
</dt> <dd>

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Índice de animação.

</dd> </dl>

## <a name="return-value"></a>Valor retornado

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Número de chaves de escala na animação do quadro chave especificado.
