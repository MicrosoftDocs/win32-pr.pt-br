# <a name="boolean2d"></a>Boolean2d

Define um conjunto de dois valores boolianas usados no modelo [**MeshFaceWraps**](meshfacewraps.md) para definir a topologia de textura de um rosto individual.

``` syntax
template Boolean2d
{
    < 4885AE63-78E8-11cf-8F52-0040333594A3 >
    Boolean u;
    Boolean v;
} 
```

Em que:

-   u - valor booliana. Consulte [**Boolean**](boolean.md).
-   v – valor booliana. Consulte [**Boolean**](boolean.md).
