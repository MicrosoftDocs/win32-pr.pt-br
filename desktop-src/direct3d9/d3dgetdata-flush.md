# <a name="d3dgetdata_flush"></a>D3DGETDATA \_ liberar

Essa macro cria o valor passado para [**GetData**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-getdata) para liberar dados de consulta.

``` syntax
#define D3DGETDATA_FLUSH (1 << 0)
```

## <a name="remarks"></a>Comentários

Para obter mais informações sobre como usar consultas, consulte [consultas (Direct3D 9)](queries.md).
