# <a name="id3dxanimationsetgetnumanimations-method"></a>Método ID3DXAnimationSet::GetNumAnimations

Obtém o número de animações no conjunto de animação.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetNumAnimations();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Número de animações no conjunto de animação.
