# <a name="d3dpresent_back_buffers"></a>BUFFERS DE BACK D3DPRESENT \_ \_

Especifica o número de buffers de retorno que podem ser usados pelo dispositivo.



| \#Definir                           | Valor | Descrição                                               |
|------------------------------------|-------|-----------------------------------------------------------|
| D3DPRESENT \_ BACK \_ BUFFERS \_ MAX     | 3     | Número máximo de buffers back com suporte no Direct3D 9.   |
| D3DPRESENT \_ BACK \_ BUFFERS \_ MAX \_ EX | 30    | Número máximo de buffers back com suporte no Direct3D 9Ex. |



 

## <a name="constant-information"></a>Informações constantes



|  Requisito                        | Valor           |
|--------------------------|------------|
| parâmetro                   | d3d9.h     |
| Sistema operacional mínimo | Windows 98 |
