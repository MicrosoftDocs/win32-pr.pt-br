# <a name="d3dpresent_back_buffers"></a>\_Buffers de fundo de D3DPRESENT \_

Especifica o número de buffers de fundo que podem ser usados pelo dispositivo.



|                                    |       |                                                           |
|------------------------------------|-------|-----------------------------------------------------------|
| \#definir                           | Valor | Descrição                                               |
| \_Máximo de \_ buffers de fundo de D3DPRESENT \_     | 3     | Número máximo de buffers de fundo com suporte no Direct3D 9.   |
| Máximo de buffers de D3DPRESENT de \_ fundo \_ \_ \_ ex | 30    | Número máximo de buffers de fundo com suporte no Direct3D 9Ex. |
