# <a name="effectparamstring"></a>EffectParamString

Atribui um valor a um parâmetro em um arquivo de efeito.

``` syntax
template EffectParamString
{
    < 1DBC4C88-94C1-46ee-9076-2C28818C9481 >
    STRING ParamName;
    STRING Value;
} 
```

-   ParamName-nome do parâmetro no arquivo de efeitos.
-   Valor-valor atribuído ao parâmetro.

Esse modelo substitui o modelo [**EffectProperty**](effectstring.md) anterior.
