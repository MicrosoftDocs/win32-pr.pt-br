# <a name="rendering-from-user-memory-pointers-direct3d-9"></a>Renderização de ponteiros de memória do usuário (Direct3D 9)

Um conjunto secundário de interfaces de renderização oferece suporte à passagem de dados de vértices e de índices diretamente de ponteiros de memória do usuário. Essas interfaces dão suporte apenas a um único fluxo de dados de vértice. Para obter mais informações, consulte os tópicos de referência a seguir.

-   [**IDirect3DDevice9::DrawPrimitiveUP**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-drawprimitiveup)
-   [**IDirect3DDevice9::DrawIndexedPrimitiveUP**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-drawindexedprimitiveup)

Esses métodos são renderizados com os dados especificados pelos ponteiros de memória do usuário, em vez de vértices e buffers de índice.
