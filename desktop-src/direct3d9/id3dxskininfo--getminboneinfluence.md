# <a name="id3dxskininfogetminboneinfluence-method"></a>Método ID3DXSkinInfo:: GetMinBoneInfluence

Obtém a influência mínima do Bone. Influenciar valores menores do que isso é ignorado.

## <a name="syntax"></a>Sintaxe


```C++
FLOAT GetMinBoneInfluence();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **float**](../winprog/windows-data-types.md)**

Retorna o valor de influência do Bone mínimo.
