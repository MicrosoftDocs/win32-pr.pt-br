# <a name="manipulating-the-depth-buffer-direct3d-9"></a>Manipulando o buffer de profundidade (Direct3D 9)

Os buffers de profundidade estão associados ao dispositivo. Os aplicativos são necessários para mover os buffers de profundidade quando definem destinos de renderização. Os métodos [**IDirect3DDevice9:: GetDepthStencilSurface**](/windows/desktop/api) e [**IDirect3DDevice9:: SetDepthStencilSurface**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setdepthstencilsurface) são usados para manipular buffers de profundidade.
