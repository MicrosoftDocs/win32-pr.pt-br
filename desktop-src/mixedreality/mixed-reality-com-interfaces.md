# <a name="windows-mixed-reality-com-interfaces"></a>Interfaces COM de realidade mista do Windows

Os tópicos contidos nesta seção fornecem as especificações de referência para as interfaces COM do Windows Mixed Reality.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Permite a interoperabilidade com Plataforma Universal do Windows um objeto [SpatialInteractionManager](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) (UWP) e fornece acesso a membros do **SpatialInteractionManager** para acessar a entrada do usuário de mãos, controladores de movimento e comandos de voz do sistema. |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Permite a interoperabilidade com Plataforma Universal do Windows um objeto [HolographicSpace](/uwp/api/windows.graphics.holographic.holographicspace) (UWP) e fornece acesso aos membros do **HolographicSpace** para representar uma cena Holographic. |
