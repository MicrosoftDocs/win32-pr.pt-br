# <a name="locale_stime-constants"></a>\_Constantes STIME de localidade \*

Este tópico define as constantes de STIME de localidade \_ \* usadas pelo NLS.



| Valor               | Significado                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| STIME de localidade \_       | Caractere (s) para o separador de tempo. O número máximo de caracteres permitido para essa cadeia de caracteres é quatro, incluindo um caractere nulo de terminação. <br/> **Windows Vista e posterior:** Essa constante foi preterida. \_Em vez disso, use a localidade STIMEFORMAT. Uma localidade personalizada pode não ter um único caractere separador uniforme. Por exemplo, um formato como "03:56" 23 "é válido.<br/> |
| STIMEFORMAT de localidade \_ | Cadeias de caracteres de formatação de tempo para a localidade. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. A cadeia de caracteres pode consistir em uma combinação de [imagens de formato de hora, minuto e segundo](hour--minute--and-second-format-pictures.md).                                                                                                      |
