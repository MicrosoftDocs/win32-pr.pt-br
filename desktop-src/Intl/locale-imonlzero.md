# <a name="locale_imonlzero"></a>IMONLZERO de localidade \_

Especificador para os zeros à esquerda em campos de mês apenas para uma data abreviada. É preferível que seu aplicativo use a [constante \_ SSHORTDATE de localidade](locale-sshortdate.md) em vez de IMONLZERO de localidade \_ .



| Valor | Significado                     |
|-------|-----------------------------|
| 0     | Sem zeros à esquerda para meses |
| 1     | Zeros à esquerda para meses    |
