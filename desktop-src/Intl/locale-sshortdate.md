# <a name="locale_sshortdate"></a>SSHORTDATE de localidade \_

Cadeia de caracteres de formatação de data abreviada para a localidade. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. A cadeia de caracteres pode consistir em uma combinação de [imagens de formato de dia, mês, ano e era](day--month--year--and-era-format-pictures.md). Por exemplo, "M/d/aaaa" indica que 3 de setembro de 2004 é gravado 9/3/2004.

As localidades podem definir vários formatos de datas curtas. Para obter todos os formatos de data abreviada dessa localidade, use [EnumDateFormats](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsa), [EnumDateFormatsEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexa)ou [EnumDateFormatsExEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexex).
