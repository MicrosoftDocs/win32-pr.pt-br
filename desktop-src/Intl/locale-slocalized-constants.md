# <a name="locale_slocalized-constants"></a>\_Constantes SLOCALIZED de localidade \*

Este tópico define as constantes de SLOCALIZED de localidade \_ \* usadas pelo NLS.



| Valor                          | Significado                                                                                                                                                                                                                                                                                                                                     |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SLOCALIZEDCOUNTRYNAME de localidade \_  | **Windows 7 e posterior:** Nome localizado completo do país/região, por exemplo, Deutschland para a Alemanha. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. Como esse nome se baseia na localização do produto, ele é alterado para cada versão localizada. <br/>           |
| SLOCALIZEDDISPLAYNAME de localidade \_  | **Windows 7 e posterior:** Nome localizado completo da localidade para o idioma da interface do usuário, por exemplo, Deutsch (Deutschland) para alemão (Alemanha) "não há limite para o número de caracteres permitidos para essa cadeia de caracteres. Como esse nome se baseia na localização do produto, ele é alterado para cada versão localizada. <br/> |
| SLOCALIZEDLANGUAGENAME de localidade \_ | **Windows Vista:** Nome primário localizado completo do idioma da interface do usuário incluído em um nome de exibição localizado, por exemplo, Deutsch que representa o alemão. Como esse nome se baseia na localização do produto, ele é alterado para cada versão localizada. <br/>                                                               |
