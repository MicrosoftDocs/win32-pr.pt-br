# <a name="locale_slocalized-constants"></a>Constantes \_ LOCALE SLOCALIZED \*

Este tópico define as \_ constantes LOCALE SLOCALIZED \* usadas pelo NLS.



| Valor                          | Significado                                                                                                                                                                                                                                                                                                                                     |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SLOCALIZEDCOUNTRYNAME  | **Windows 7 e posterior:** Nome localizado completo do país/região, por exemplo, Deutschland para Alemanha. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. Como esse nome é baseado na localização do produto, ele muda para cada versão localizada. <br/>           |
| LOCALE \_ SLOCALIZEDDISPLAYNAME  | **Windows 7 e posterior:** Nome localizado completo da localidade para a linguagem de interface do usuário, por exemplo, Deutsch (Deutschland) para alemão (Alemanha)" Não há limite para o número de caracteres permitido para essa cadeia de caracteres. Como esse nome é baseado na localização do produto, ele muda para cada versão localizada. <br/> |
| LOCALE \_ SLOCALIZEDLANGUAGENAME | **Windows Vista:** Nome primário localizado completo da linguagem de interface do usuário incluída em um nome de exibição localizado, por exemplo, Deutsch que representa o alemão. Como esse nome é baseado na localização do produto, ele muda para cada versão localizada. <br/>                                                               |
