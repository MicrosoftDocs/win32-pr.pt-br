# <a name="locale_itime"></a>ITIME de localidade \_

Especificação de formato de hora. A especificação é um dos valores a seguir. É preferível que seu aplicativo use a [constante \_ STIMEFORMAT de localidade](locale-stime-constants.md) em vez de ITIME de localidade \_ .



| Valor | Descrição          |
|-------|----------------------|
| 0     | Formato de 12 horas AM/PM |
| 1     | formato de 24 horas       |
