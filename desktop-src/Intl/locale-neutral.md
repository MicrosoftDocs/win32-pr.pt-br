# <a name="locale_neutral"></a>neutro de localidade \_

A localidade neutra. Essa constante geralmente não é usada ao chamar APIs NLS. Em vez disso, use o padrão do sistema de localidade ou do [ \_ usuário \_ de localidade](locale-user-default.md). [ \_ \_ ](locale-system-default.md) O valor de localidade \_ neutra é 0x0000.
