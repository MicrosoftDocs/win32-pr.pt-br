# <a name="locale_neutral"></a>LOCALIDADE \_ NEUTRA

A localidade neutra. Essa constante geralmente não é usada ao chamar APIs NLS. Em vez disso, use [LOCALE \_ SYSTEM \_ DEFAULT](locale-system-default.md) ou [LOCALE \_ USER \_ DEFAULT.](locale-user-default.md) O valor de LOCALE \_ NEUTRAL é 0x0000.
