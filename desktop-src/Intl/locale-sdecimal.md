# <a name="locale_sdecimal"></a>SDECIMAL de localidade \_

Caracteres usados para o separador decimal, por exemplo, "." em "3,14" ou "," em "3, 14". O número máximo de caracteres permitido para essa cadeia de caracteres é quatro, incluindo um caractere nulo de terminação.
