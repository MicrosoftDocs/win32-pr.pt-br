# <a name="locale_seng-constants"></a>\_Constantes Seng de localidade \*

Este tópico define as constantes de Seng de localidade \_ \* usadas pelo NLS.



| Valor                | Significado                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SENGCOUNTRY de localidade \_  | preterido para o Windows 7 e posterior. Nome completo em inglês do país/região. Consulte [localidade \_ SENGLISHCOUNTRYNAME](locale-senglish-constants.md).                            |
| SENGCURRNAME de localidade \_ | **Windows Me/98, Windows 2000:** O nome completo em inglês da moeda associada à localidade. Não há nenhum limite para o número de caracteres permitido para essa cadeia. |
| SENGLANGUAGE de localidade \_ | preterido para o Windows 7 e posterior. Nome completo em inglês do idioma do ISO Standard 639. Consulte [localidade \_ SENGLISHLANGUAGENAME](locale-senglish-constants.md).           |
