# <a name="extended-linguistic-services"></a>Serviços lingüísticos estendidos

Os ELS (serviços lingüísticos estendidos) oferecem suporte a serviços baseados em texto, incluindo detecção de idioma, detecção de script e serviços de transliteração. A documentação do ELS inclui as seguintes seções:

-   [Sobre os serviços lingüísticos estendidos](about-extended-linguistic-services.md)
-   [Usando serviços lingüísticos estendidos](using-extended-linguistic-services.md)
-   [Referência de serviços linguísticos estendidos](extended-linguistic-services-reference.md)
