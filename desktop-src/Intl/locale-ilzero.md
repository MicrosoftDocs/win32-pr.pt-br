# <a name="locale_ilzero"></a>ILZERO de localidade \_

Especificador para zeros à esquerda em campos decimais.



| Valor | Significado          |
|-------|------------------|
| 0     | Sem zeros à esquerda |
| 1     | Zeros à esquerda    |
