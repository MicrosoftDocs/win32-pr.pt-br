# <a name="input-method-manager"></a>Gerenciador de Métodos de Entrada

O IMM (Input Method Manager) é uma tecnologia usada por um aplicativo para se comunicar com um IME (editor de método de entrada), que é executado como um serviço. O IME permite que os usuários de computador insiram caracteres complexos e símbolos, como caracteres kanji japonês, usando um teclado padrão.

Esta seção descreve a API do IMM e explica como usar a funcionalidade para criar e gerenciar janelas IME. Ele inclui as seguintes seções:

-   [Sobre o Gerenciador de Métodos de Entrada](about-input-method-manager.md)
-   [Usando o Gerenciador de Métodos de Entrada](using-input-method-manager.md)
-   [Referência do Gerenciador de Métodos de Entrada](input-method-manager-reference.md)
