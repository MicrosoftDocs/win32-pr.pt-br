# <a name="input-method-manager"></a>Gerenciador de métodos de entrada

O IMM (Input Method Manager) é uma tecnologia usada por um aplicativo para se comunicar com um IME (editor de método de entrada), que é executado como um serviço. O IME permite que os usuários de computador insiram caracteres e símbolos complexos, como caracteres kanji japoneses, usando um teclado padrão.

Esta seção descreve a API do IMM e explica como usar a funcionalidade para criar e gerenciar janelas do IME. Ele inclui as seguintes seções:

-   [Sobre o Gerenciador de métodos de entrada](about-input-method-manager.md)
-   [Usando o Gerenciador de métodos de entrada](using-input-method-manager.md)
-   [Referência do Gerenciador de método de entrada](input-method-manager-reference.md)
