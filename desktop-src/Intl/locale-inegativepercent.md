# <a name="locale_inegativepercent"></a>INEGATIVEPERCENT de localidade \_

**Windows 7 e posterior:** Padrão de formatação de porcentagem negativa para a localidade. Apenas um padrão pode ser indicado. Se mais de um formato for usado para a localidade, escolha a opção preferencial. Por exemplo, se um percentual negativo for exibido "-9%" para "um percentual de nove%", a opção apropriada para essa constante será 0.



| Valor | Formatar                                                    |
|-------|-----------------------------------------------------------|
| 0     | Sinal negativo, número, espaço, porcentagem; por exemplo,-\# % |
| 1     | Sinal negativo, número, porcentagem; por exemplo,-\#%         |
| 2     | Sinal negativo, porcentagem, número; por exemplo,-%\#         |
| 3     | Porcentagem, sinal negativo, número; por exemplo,%-\#         |
| 4     | Porcentagem, número, sinal negativo; por exemplo,%\#-         |
| 5     | Número, sinal negativo, porcentagem; por exemplo, \#-%         |
| 6     | Número, porcentagem, sinal negativo; por exemplo, \#%-         |
| 7     | Sinal negativo, porcentagem, espaço, número; por exemplo,-% \# |
| 8     | Número, espaço, porcentagem, sinal negativo; por exemplo, \# %- |
| 9     | Porcentagem, espaço, número, sinal negativo; por exemplo,% \#- |
| 10    | Porcentagem, espaço, sinal negativo, número; por exemplo,%-\# |
| 11    | Número, sinal negativo, espaço, porcentagem; por exemplo, \# -% |
