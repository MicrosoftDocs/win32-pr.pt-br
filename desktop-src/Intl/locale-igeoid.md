# <a name="locale_igeoid"></a>IGEOID de localidade \_

Um número assinado de 32 bits que identifica exclusivamente uma [localização geográfica](table-of-geographical-locations.md). O aplicativo usa essa constante para fornecer serviços específicos de localidade aos clientes. Por exemplo, ele pode ser usado como uma chave para acessar um registro de banco de dados que contém informações específicas sobre um país/região.
