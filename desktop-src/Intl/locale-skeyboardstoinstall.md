# <a name="locale_skeyboardstoinstall"></a>SKEYBOARDSTOINSTALL de localidade \_

**Windows Vista e posterior:** Uma lista delimitada por ponto-e-vírgula de teclados para ser possível instalar para a localidade e ser usada internamente pelo Windows. Não há nenhum limite para o número de caracteres permitido para essa cadeia. Para recuperar o nome do identificador de localidade de entrada ativo (anteriormente chamado de layout de teclado), seu aplicativo pode chamar a função [**GetKeyboardLayoutName**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutnamea) .
