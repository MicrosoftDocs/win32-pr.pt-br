# <a name="locale_smonthname-constants"></a>Constantes \_ LOCALE SMONTHNAME \*

Este tópico define as \_ constantes LOCALE SMONTHNAME \* usadas pelo NLS.




| Valor | Significado |

| LOCALE_SMONTHNAME1 | Nome longo nativo para janeiro. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.<blockquote>[!Note]<br />Chamar a <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa"><strong>função GetLocaleInfo</strong></a> ou <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoex"><strong>GetLocaleInfoEx</strong></a> com uma constante LOCALE_SMONTHNAME* retorna a forma autônoma ou nominativa do nome do mês. Para obter a forma genitiva do nome do mês, o aplicativo chama <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata">GetDateFormat</a> ou <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformatex">GetDateFormatEx</a> com uma imagem de data do ddMMMM e remove os dois dígitos do início da cadeia de caracteres recuperada.</blockquote><br /> |
