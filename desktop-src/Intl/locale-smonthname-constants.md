<blockquote>
[!Note]<br />
Chamar a função <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa"><strong>GetLocaleInfo</strong></a> ou <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoex"><strong>GetLocaleInfoEx</strong></a> com uma constante LOCALE_SMONTHNAME * retorna a forma autônoma ou nomeada do nome do mês. Para obter a forma genitive do nome do mês, o aplicativo chama <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata">GetDateFormat</a> ou <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformatex">GetDateFormatEx</a> com uma imagem de data de ddMMMM e remove os dois dígitos do início da cadeia de caracteres recuperada.
