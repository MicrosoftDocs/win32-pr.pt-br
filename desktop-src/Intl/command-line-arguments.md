# <a name="command-line-arguments"></a>Argumentos de linha de comando

Um aplicativo pode usar a função [GetCommandLine](/windows/win32/api/processenv/nf-processenv-getcommandlinea) do Win32 para recuperar argumentos de linha de comando Unicode. O aplicativo deve chamar a função do Win32 como uma função Unicode.
