# <a name="locale_smonthday"></a>SMONTHDAY de localidade \_

**Windows 7 e posterior:** Cadeia de caracteres de formato para exibir apenas o mês e o dia. A formatação é semelhante àquela definida para [ \_ SLONGDATE de localidade](locale-slongdate.md). Por exemplo, se o padrão mês/dia for o nome completo do mês seguido do número do dia com zeros à esquerda, como em "setembro de 03", a cadeia de caracteres de formato será "MMMM dd". A cadeia de caracteres pode consistir em uma combinação de imagens de formato de dia e mês e qualquer cadeia de caracteres entre aspas simples. Os caracteres entre aspas simples permanecem conforme especificado, por exemplo, ' de ' para espanhol (Espanha). Uma localidade pode especificar apenas um formato de mês/dia.
