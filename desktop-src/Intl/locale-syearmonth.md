# <a name="locale_syearmonth"></a>LOCALE \_ SYEARMONTH

**Windows Me/98, Windows 2000:** A cadeia de caracteres de formatação de mês de ano para a localidade. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. Essa cadeia de caracteres mostra o formato adequado para uma cadeia de caracteres de data que contém apenas o ano e o mês, usando imagens de formato conforme definido em Imagens de Formato de [Dia, Mês, Ano e Era.](day--month--year--and-era-format-pictures.md)
