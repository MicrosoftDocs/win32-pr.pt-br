# <a name="locale_sshortestam"></a>SSHORTESTAM de localidade \_

Cadeia de caracteres do indicador AM mais curto possível para uma localidade. Para inglês que pode ser tão curto quanto "a"; no entanto, outras linguagens poderiam ter mais íntegra.
