# <a name="locale_iintlcurrdigits"></a>IINTLCURRDIGITS de localidade \_

\[\_A localidade IINTLCURRDIGITS não tem suporte e pode ser alterada ou não estar disponível no futuro. Em vez disso, use a [localidade \_ ICURRDIGITS](locale-icurrdigits.md).\]

Número de dígitos fracionários para o formato Monetário Internacional. O número máximo de caracteres permitido para essa cadeia de caracteres é três, incluindo um caractere nulo de terminação. Por exemplo, o número de dígitos fracionários para o formato Monetário Internacional de dólares dos EUA é 2, como em $345.25.
