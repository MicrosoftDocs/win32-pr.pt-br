# <a name="locale_windows"></a>janelas de localidade \_

**Windows Vista e posterior:** Enumere todas as localidades fornecidas com o sistema operacional, incluindo localidades de substituição, mas excluindo classificações alternativas. Para obter mais informações, consulte [localidades personalizadas](custom-locales.md).
