# <a name="locale_iuseutf8legacyacp"></a>IUSEUTF8LEGACYACP de localidade \_

**A partir da próxima versão do Windows:** É recomendável que os aplicativos usem Unicode em vez disso. Em computadores que o usuário configurou para usar uma página de código UTF-8, LOCALE_IUSEUTF8LEGACYACP é a página de código herdada do Windows que teria sido usada em um computador Windows herdado não UTF-8.
