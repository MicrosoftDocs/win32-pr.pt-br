# <a name="locale_iuseutf8legacyacp"></a>LOCALE \_ IUSEUTF8LEGACYACP

**Começando com a próxima versão do Windows:** Em vez disso, é recomendável que os aplicativos usem Unicode. Em máquinas que o usuário configurou para usar uma página de código UTF-8, o LOCALE_IUSEUTF8LEGACYACP é a página de código Windows herdada que teria sido usada em um computador de Windows herdado não UTF-8.
