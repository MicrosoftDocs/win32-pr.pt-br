# <a name="locale_sortname"></a>LOCALE \_ SORTNAME

**Windows 7 e posterior:** Nome da localidade a ser usado para classificação, se for diferente da localidade especificada por [LOCALE \_ SNAME.](locale-sname.md) Essa constante é usada por localidades personalizadas. Consulte também [LOCALE \_ SSORTNAME](locale-ssort-constants.md).
