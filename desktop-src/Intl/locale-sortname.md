# <a name="locale_sortname"></a>tipo de classificação de localidade \_

**Windows 7 e posterior:** Nome da localidade a ser usado para classificação, se for diferente da localidade especificada pela [localidade \_ SNAME](locale-sname.md). Essa constante é usada por localidades personalizadas. Consulte também [ \_ SSORTNAME de localidade](locale-ssort-constants.md).
