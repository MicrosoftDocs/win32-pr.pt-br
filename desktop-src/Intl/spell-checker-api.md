# <a name="spell-checking-api"></a>API de verificação ortográfica

A API de verificação ortográfica permite que os desenvolvedores consumam a capacidade do verificador ortográfico para verificar o texto, obter sugestões e manter configurações e dicionários do usuário. Os desenvolvedores também podem criar verificadores ortográficos integrados à estrutura de verificação ortográfica como provedores.

-   [Sobre a API de verificação ortográfica](about-the-spell-checker-api.md)
-   [Referência de API de verificação ortográfica](spell-checker-api-reference.md)
