# <a name="glossary-for-understanding-mui"></a>Glossário para entender o MUI

-   [Globalização](#globalization)
-   [Possibilidade de localização](#localizability)
-   [Localização](#localization)
-   [Internacionalização](#internationalization)

## <a name="globalization"></a>Globalização

O termo *globalização* indica a necessidade da plataforma para trabalhar em qualquer lugar do box. Na prática, essa visão é realizada no Windows com um único binário mundial para o sistema operacional Windows, começando com a versão do Windows Vista, dando suporte à implantação de imagem única em um contexto multilíngue e fornecendo APIs NLS para recuperar dados específicos de localidade.

## <a name="localizability"></a>Possibilidade de localização

A *possibilidade* de localização de termos significa tornar o produto mais fácil de localizar, o que, por sua vez, impulsiona os custos de localização. Isso pode ser feito com o uso eficiente de APIs de recurso, controlando a extensão e o número de recursos adicionados ao sistema e melhorando as ferramentas usadas para trabalhar com esses recursos.

## <a name="localization"></a>Localização

A *localização* é o processo de adaptação de um programa para um mercado local específico, que inclui a tradução da interface do usuário, as caixas de diálogo de redimensionamento, a personalização de recursos (se necessário) e os resultados do teste para garantir que o programa ainda funcione. No contexto mais comumente usado, a localização se concentra na tradução da interface do usuário.

## <a name="internationalization"></a>Internacionalização

Usado como um sinônimo de globalização em alguns lugares, ou como globalização + localização em outros lugares, a *internacionalização* é melhor usada para descrever todas as atividades que melhoram um produto no contexto mundial.
