# <a name="locale_use_cp_acp"></a>LOCALIDADE \_ use \_ CP \_ ACP

**Windows me/98, windows 2000:** Página de código ANSI padrão do sistema (ACP), em vez da página de código de localidade usada para conversão de cadeia de caracteres. Consulte [identificadores de página de código](code-page-identifiers.md) para obter uma lista de ANSI e outras páginas de código.
