# <a name="locale_use_cp_acp"></a>LOCALIDADE \_ use \_ CP \_ ACP

**Windows Me/98, Windows 2000:** o padrão do sistema Windows página de código ANSI (ACP) em vez da página de código de localidade usada para conversão de cadeia de caracteres. Consulte [identificadores de página de código](code-page-identifiers.md) para obter uma lista de ANSI e outras páginas de código.
