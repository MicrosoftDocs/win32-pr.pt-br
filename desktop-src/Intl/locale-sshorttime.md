# <a name="locale_sshorttime"></a>SSHORTTIME de localidade \_

**Windows 7 e posterior:** Cadeia de caracteres de formatação de hora curta para a localidade. Normalmente, os padrões são derivados removendo o valor "SS" (segundos) do padrão de formato de hora longa. Por exemplo, se o formato de hora longa for "h:mm: SS TT", o formato de hora abreviada provavelmente será "h:mm tt". Essa constante pode especificar vários formatos em uma lista delimitada por ponto-e-vírgula. No entanto, o formato de hora curto preferencial deve ser o primeiro valor listado.
