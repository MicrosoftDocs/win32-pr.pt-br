# <a name="locale_sshorttime"></a>LOCALE \_ SSHORTTIME

**Windows 7 e posterior:** Cadeia de caracteres de formatação de tempo curto para a localidade. Os padrões normalmente são derivados pela remoção do valor "ss" (segundos) do padrão de formato de longo prazo. Por exemplo, se o formato de tempo longo for "h:mm:ss tt", o formato de tempo curto provavelmente será "h:mm tt". Essa constante pode especificar vários formatos em uma lista delimitada por ponto e vírgula. No entanto, o formato de tempo curto preferencial deve ser o primeiro valor listado.
