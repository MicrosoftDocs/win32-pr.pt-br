# <a name="locale_ipositivepercent"></a>LOCALE \_ IPOSITIVEPERCENT

**Windows 7 e posterior:** Padrão de formatação de percentual positivo para a localidade. Somente um padrão pode ser indicado.



| Valor | Formatar                                    |
|-------|-------------------------------------------|
| 0     | Número, espaço, porcentagem; por exemplo, \# % |
| 1     | Número, porcentagem; por exemplo, \#%         |
| 2     | Percentual, número; por exemplo, %\#         |
| 3     | Porcentagem, espaço, número; por exemplo, % \# |
