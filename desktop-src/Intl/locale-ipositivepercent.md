# <a name="locale_ipositivepercent"></a>IPOSITIVEPERCENT de localidade \_

**Windows 7 e posterior:** Padrão de formatação de porcentagem positiva para a localidade. Apenas um padrão pode ser indicado.



| Valor | Formatar                                    |
|-------|-------------------------------------------|
| 0     | Número, espaço, porcentagem; por exemplo, \# % |
| 1     | Número, porcentagem; por exemplo, \#%         |
| 2     | Porcentagem, número; por exemplo,%\#         |
| 3     | Porcentagem, espaço, número; por exemplo,% \# |
