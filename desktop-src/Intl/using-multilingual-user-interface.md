# <a name="using-multilingual-user-interface"></a>Usando a interface do usuário multilíngue

Esta seção descreve como usar a funcionalidade MUI (Multilingual User interface) para projetar e implementar seus aplicativos. Veja a seguir os aspectos da tecnologia MUI que entrarão em cena na maioria dos estágios do ciclo de vida do aplicativo.

-   Desenvolvimento de aplicativos, incluindo preparação de recursos, configuração de preferências de idioma do aplicativo, localização e carregamento de recursos
-   Compilando e localizando o aplicativo
-   Implantação do aplicativo

Aqui estão algumas perguntas a serem consideradas ao projetar e implementar um aplicativo MUI.

-   Quais são as versões do Windows para as quais o aplicativo dará suporte?
-   Em quais idiomas o aplicativo será localizado?
-   As configurações de idioma do aplicativo devem sempre seguir as configurações de idioma do sistema operacional ou o usuário do aplicativo poderá definir idiomas específicos?
-   Que tipo de recursos de interface do usuário o aplicativo usa e onde eles são armazenados?

Esta seção inclui os seguintes tópicos. As descrições pressupõem um aplicativo MUI direcionado ao Windows Vista e posterior. Os recursos para esse aplicativo são recursos do Win32, com recursos específicos de idioma e código executável que residem em arquivos do Win32 PE separados. Considerações especiais para o suporte de sistemas operacionais anteriores ao Windows Vista ou para diferentes tipos de recursos são adicionadas conforme apropriado.
