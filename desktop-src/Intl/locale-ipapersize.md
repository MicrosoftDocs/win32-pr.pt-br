# <a name="locale_ipapersize"></a>IPAPERSIZE de localidade \_

**Windows 2000:** Tamanho de papel padrão associado à localidade. O tamanho normalmente tem um dos seguintes valores, embora possa ser definido como qualquer um dos tamanhos de papel definidos que são compreendidos pelo spooler.



| Valor | Significado   |
|-------|-----------|
| 1     | Carta dos EUA |
| 5     | Ofício dos EUA  |
| 8     | A3        |
| 9     | A4        |
