# <a name="locale_ildate"></a>ILDATE de localidade \_

Formato de data por extenso – especificador de ordenação. É preferível que seu aplicativo use a [constante \_ SLONGDATE de localidade](locale-slongdate.md) em vez de ILDATE de localidade \_ .



| Valor | Significado        |
|-------|----------------|
| 0     | Mês-dia-ano |
| 1     | Dia-mês-ano |
| 2     | Ano-mês-dia |
