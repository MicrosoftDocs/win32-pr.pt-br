# <a name="locale_ildate"></a>LOCALE \_ ILDATE

Especificador de ordenação de formato de data longa. É preferível que seu aplicativo use a constante [ \_ LOCALE SLONGDATE](locale-slongdate.md) em vez de LOCALE \_ ILDATE.



| Valor | Significado        |
|-------|----------------|
| 0     | Mês-Dia-Ano |
| 1     | Day-Month-Year |
| 2     | Year-Month-Day |
