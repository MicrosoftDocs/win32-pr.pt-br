# <a name="time-and-date"></a>Data e hora

O NLS dá suporte à recuperação de cadeias de caracteres para qualquer hora ou data em um formato apropriado para a localidade atual. No Windows Vista e posterior, o NLS também dá suporte à recuperação de cadeias de caracteres de duração para quaisquer localidades, incluindo localidades personalizadas. Para obter instruções de desenvolvimento, consulte [recuperando informações de data e hora](retrieving-time-and-date-information.md).
