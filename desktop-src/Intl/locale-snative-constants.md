# <a name="locale_snative-constants"></a>Constantes \_ SNATIVE \* LOCALE

Este tópico define as \_ constantes LOCALE SNATIVE \* usadas pelo NLS para representar nomes de idiomas nativos.



| Valor                       | Significado                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SNATIVECOUNTRYNAME  | **Windows 7 e posteriores:** Nome nativo do país/região, por exemplo, Esparca para Espanha. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.                                                                 |
| LOCALE \_ SNATIVECTRYNAME     | Preterido para o Windows 7 e posterior. Nome nativo do país/região. Consulte LOCALE \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| LOCALE \_ SNATIVECURRNAME     | **Windows Me/98, Windows 2000:** O nome nativo da moeda associada à localidade, no idioma nativo da localidade. Não há limite para o número de caracteres permitido para essa cadeia de caracteres.                                                          |
| LOCALE \_ SNATIVEDIGITS       | Equivalentes nativos de ASCII 0 a 9. O número máximo de caracteres permitidos para essa cadeia de caracteres é de 11, incluindo um caractere nulo de terminação. Por exemplo, o árabe usa "". Consulte também [LOCALE \_ IDIGITSUBSTITUTION.](locale-idigitsubstitution.md) |
| LOCALE \_ SNATIVEDISPLAYNAME  | **Windows 7 e posteriores:** Nome de exibição da localidade em seu idioma nativo, por exemplo, Deutsch (Deutschland) para a localidade alemã (Alemanha). <br/>                                                                                                        |
| LOCALE \_ SNATIVELANGNAME     | Preterido para o Windows 7 e posterior. Nome nativo do idioma. Consulte LOCALE \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| LOCALE \_ SNATIVELANGUAGENAME | **Windows 7 e posteriores:** Nome nativo da linguagem, por exemplo, ª para Armenia (Armenia). O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.                                                         |
