# <a name="locale_snative-constants"></a>\_Constantes SNATIVE de localidade \*

Este tópico define as constantes de SNATIVE de localidade \_ \* usadas pelo NLS para representar nomes de idiomas nativos.



| Valor                       | Significado                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SNATIVECOUNTRYNAME de localidade \_  | **Windows 7 e posterior:** Nome nativo do país/região, por exemplo, España para a Espanha. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.                                                                 |
| SNATIVECTRYNAME de localidade \_     | Preterido para o Windows 7 e posterior. Nome nativo do país/região. Consulte localidade \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| SNATIVECURRNAME de localidade \_     | **Windows me/98, windows 2000:** O nome nativo da moeda associada à localidade, no idioma nativo da localidade. Não há nenhum limite para o número de caracteres permitido para essa cadeia.                                                          |
| SNATIVEDIGITS de localidade \_       | Equivalentes nativos de ASCII 0 a 9. O número máximo de caracteres permitido para essa cadeia de caracteres é onze, incluindo um caractere nulo de terminação. Por exemplo, o árabe usa "012345 6789". Consulte também [ \_ IDIGITSUBSTITUTION de localidade](locale-idigitsubstitution.md). |
| SNATIVEDISPLAYNAME de localidade \_  | **Windows 7 e posterior:** Nome de exibição da localidade em seu idioma nativo, por exemplo, Deutsch (Deutschland) para a localidade alemão (Alemanha). <br/>                                                                                                        |
| SNATIVELANGNAME de localidade \_     | Preterido para o Windows 7 e posterior. Nome nativo do idioma. Consulte localidade \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| SNATIVELANGUAGENAME de localidade \_ | **Windows 7 e posterior:** Nome nativo do idioma, por exemplo, Հայերեն para armênio (Armênia). O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.                                                         |
