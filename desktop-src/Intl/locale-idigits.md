# <a name="locale_idigits"></a>LOCALE \_ IDIGITS

Número de dígitos fracionados colocados após o separador decimal. O número máximo de caracteres permitidos para essa cadeia de caracteres é dois, incluindo um caractere nulo de terminação. Por exemplo, 2 para 5,00, 1 para 5,0.
