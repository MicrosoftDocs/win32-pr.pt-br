# <a name="locale_idigits"></a>IDIGITS de localidade \_

Número de dígitos fracionários colocados após o separador decimal. O número máximo de caracteres permitido para essa cadeia de caracteres é dois, incluindo um caractere nulo de terminação. Por exemplo, 2 para 5, 0, 1 para 5,0.
