# <a name="locale_snan"></a>SNAN de localidade \_

**Windows Vista e posterior:** Valor da cadeia de caracteres de "não é um número", por exemplo, "Nan" para a localidade em inglês (Estados Unidos). Não há nenhum limite para o número de caracteres permitido para essa cadeia.
