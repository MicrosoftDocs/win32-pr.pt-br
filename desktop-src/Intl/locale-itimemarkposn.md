# <a name="locale_itimemarkposn"></a>ITIMEMARKPOSN de localidade \_

Especificador que indica se a cadeia de caracteres do marcador de tempo (AM ou PM) precede ou segue a cadeia de caracteres de tempo. O valor do registro é iTimePrefix para compatibilidade com versões asiáticas anteriores do Windows. O especificador deve ter um dos valores a seguir. Nenhum valor especificado pelo usuário é permitido. É preferível que seu aplicativo use a [constante \_ STIMEFORMAT de localidade](locale-stime-constants.md) em vez de ITIMEMARKPOSN de localidade \_ .



| Valor | Significado        |
|-------|----------------|
| 0     | Usar como sufixo. |
| 1     | Use como prefixo. |
