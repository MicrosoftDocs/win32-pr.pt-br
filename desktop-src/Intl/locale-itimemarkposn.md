# <a name="locale_itimemarkposn"></a>LOCALE \_ ITIMEMARKPOSN

Especificador que indica se a cadeia de caracteres de marcador de hora (AM ou PM) precede ou segue a cadeia de caracteres de tempo. O valor do Registro é iTimePrefix para compatibilidade com versões anteriores da Ásia Windows. O especificador deve ter um dos valores a seguir. Nenhum valor especificado pelo usuário é permitido. É preferível que seu aplicativo use a constante [LOCALE \_ STIMEFORMAT](locale-stime-constants.md) em vez de LOCALE \_ ITIMEMARKPOSN.



| Valor | Significado        |
|-------|----------------|
| 0     | Use como sufixo. |
| 1     | Use como prefixo. |
