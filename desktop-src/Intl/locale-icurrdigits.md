# <a name="locale_icurrdigits"></a>ICURRDIGITS de localidade \_

Número de dígitos fracionários para o formato monetário local. O número máximo de caracteres permitido para essa cadeia de caracteres é dois, incluindo os valores 0-9 e um caractere nulo de terminação. Por exemplo, 200, 0 tem um valor de 2 porque há dois dígitos fracionários, enquanto 200 tem um valor de 0 porque não há dígitos fracionários exibidos.
