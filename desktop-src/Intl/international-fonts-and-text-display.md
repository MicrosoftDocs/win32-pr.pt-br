# <a name="international-fonts-and-text-display"></a>Fontes internacionais e exibição de texto

As fontes internacionais e a exibição de texto fornecem suporte de plataforma para fontes internacionais, texto internacional e tipografia fina na plataforma Windows.

Fontes e textos internacionais baseiam-se na tipografia básica, no texto e nas tecnologias de leitura avançadas do Windows para fornecer fontes e texto para usuários de toda a plataforma do Windows. A tecnologia aprimora os cenários de exibição de texto e fonte de ponta a ponta.

As seguintes tecnologias estão incluídas:

-   [Gerenciamento de fontes internacionais](about-international-fonts-and-text.md). Apresenta a funcionalidade básica de fontes internacionais.
-   [Usando MS Shell Dlg e MS Shell Dlg 2](using-ms-shell-dlg-and-ms-shell-dlg-2.md). Visão geral das fontes MS Shell Dlg e MS Shell Dlg 2 e mapeamento de fonte do MS Shell Dlg e MS Shell Dlg 2 em diferentes versões do Windows.
-   [Seleção e enumeração de fontes internacionais](using-international-fonts-and-text.md). Explica possíveis maneiras de enumerar fontes internacionais para permitir que os usuários selecionem a fonte que desejam.
-   [Exibição de texto internacional](creating-your-own-format-selection-user-interface.md). Explica possíveis maneiras de exibir texto internacional.
-   [Uniscribe](uniscribe.md). Permite um alto grau de controle para a tipografia fina e para o processamento de scripts complexos.
