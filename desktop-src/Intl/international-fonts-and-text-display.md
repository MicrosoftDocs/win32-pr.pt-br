# <a name="international-fonts-and-text-display"></a>Exibição de texto e fontes internacionais

Fontes Internacionais e Exibição de Texto fornece suporte à plataforma para fontes internacionais, texto internacional e tipografia fina na Windows plataforma.

Fontes e texto internacionais se baseia nas tecnologias básicas de tipografia, texto e leitura avançadas do Windows para fornecer fontes e texto para usuários de todo o mundo da plataforma Windows. A tecnologia aprimora cenários de exibição de texto e fonte de ponta a ponta.

As seguintes tecnologias estão incluídas:

-   [Gerenciamento internacional de fontes.](about-international-fonts-and-text.md) Apresenta a funcionalidade básica de fontes internacionais.
-   [Usando o MS Shell Dlg e o MS Shell Dlg 2.](using-ms-shell-dlg-and-ms-shell-dlg-2.md) Visão geral das fontes MS Shell Dlg e MS Shell Dlg 2 e mapeamento de fonte do MS Shell Dlg e MS Shell Dlg 2 em versões Windows diferentes.
-   [Enumeração e seleção de fontes internacionais.](using-international-fonts-and-text.md) Explica possíveis maneiras de enumerar fontes internacionais para permitir que os usuários selecionem a fonte que eles querem.
-   [Exibição de texto internacional.](creating-your-own-format-selection-user-interface.md) Explica possíveis maneiras de exibir texto internacional.
-   [Uniscribe](uniscribe.md). Permite um alto grau de controle para tipografia fina e para processamento de scripts complexos.
