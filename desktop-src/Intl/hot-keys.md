# <a name="hot-keys"></a>Teclas de Atalho

As teclas de acesso dão ao usuário uma maneira de alterar rapidamente o modo de entrada de um IME ou alternar para outro IME. Embora os aplicativos não possam adicionar teclas de acesso ao sistema operacional, eles podem iniciar a mesma ação que uma tecla de atalho usando a função [**ImmSimulateHotKey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) .

O IME HexToUnicode também permite a conversão entre caracteres hexadecimais e Unicode. Para obter mais informações, consulte [HEXTOUNICODE IME](hextounicode-ime.md).
