# <a name="locale_spositivesign"></a>SPOSITIVESIGN de localidade \_

Valor de cadeia de caracteres localizado para o sinal positivo para a localidade. Por exemplo, a **localidade \_ SPOSITIVESIGN** é considerada "+" para a localidade inglês (Estados Unidos). Observe que a presença de dados para essa cadeia de caracteres não significa que um sinal positivo é necessariamente usado ao Formatar um número. Se esses dados estiverem em branco/vazios, um "+" será considerado para ser usado. O número máximo de caracteres permitido para essa cadeia de caracteres é cinco, incluindo um caractere nulo de terminação.
