# <a name="about-multilingual-user-interface"></a>Sobre Interface de Usuário Multilíngue

Interface de Usuário Multilíngue (MUI) é uma tecnologia que fornece aos usuários uma interface do usuário localizada para aplicativos globalizados e gerenciamento de recursos de linguagem de interface do usuário Windows sistema operacional. O suporte é fornecido para adicionar a funcionalidade de MUI a aplicativos globalizados para executar no Windows Vista e posterior, bem como muitos sistemas operacionais Windows Vista pré-teste. Os modelos de gerenciamento de recursos e localização da MUI aprimoram o desenvolvimento, o teste e o suporte para software pronto para o mundo.

Os recursos da tecnologia MUI incluem:

-   Gerenciamento de linguagem simplificado para localização de software. A MUI permite que as configurações de idioma do sistema operacional sejam alteradas para um idioma com suporte de acordo com as preferências do usuário.
-   Uma tecnologia de recursos inovadora com base na divisão do binário do código do aplicativo de arquivos de recursos de linguagem. Você pode adicionar recursos para um idioma adicional sem precisar recompilar ou relink seu aplicativo. Recursos localizados adicionais se tornam complementos opcionais.
-   Api (interface de programação de aplicativo) completa. Windows O Vista e posterior expõem a API de MUI para você usar na adição de funcionalidade de MUI a aplicativos globalizados. As funções de gerenciamento de linguagem permitem que seus aplicativos deem suporte a uma ampla variedade de linguagens de interface do usuário. A API também inclui funções de carregamento de recursos que permitem que o carregador de recursos carregue recursos no Windows Vista e posterior, bem como pré-Windows sistemas operacionais Vista.
-   Um conjunto de ferramentas de gerenciamento de pacotes de idiomas que fornecem flexibilidade no gerenciamento de imagens de implantação internacional. Essas ferramentas não são direcionadas a desenvolvedores e, portanto, não são abordadas no SDK. Mais informações podem ser encontradas no Windows Kit de Instalação Automatizada disponível no Technet.

O benefício mais visível da MUI é que vários usuários podem compartilhar a mesma estação de trabalho e exibir a interface do usuário em linguagens diferentes. As empresas e os OEMs se beneficiarão da capacidade que precisam para lançar, dar suporte e manter imagens multilíngues com uma única instalação. Mas talvez o principal benefício da MUI venha nas eficiências obtidas ao desenvolver, criar e atender seu aplicativo. Você pode enviar um binário de funcionalidade principal aplicável a todas as plataformas, independentemente da linguagem de interface do usuário, o que reduz significativamente os esforços de desenvolvimento e teste. Se você tiver que emitir uma atualização ou uma service pack, ela se aplicará a todos os idiomas com suporte sem nenhum esforço de engenharia adicional. O suporte posterior para idiomas adicionais se torna um projeto de localização em vez de um projeto completo de desenvolvimento de software.
