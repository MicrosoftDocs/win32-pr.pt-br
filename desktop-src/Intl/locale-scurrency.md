# <a name="locale_scurrency"></a>SCURRENCY de localidade \_

Cadeia de caracteres usada como o símbolo monetário local. O número máximo de caracteres permitido para essa cadeia de caracteres é 13, incluindo um caractere nulo de terminação. Por exemplo, na Estados Unidos, esse símbolo é "$".
