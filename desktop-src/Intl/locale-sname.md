# <a name="locale_sname"></a>LOCALE \_ SNAME

**Windows Vista e posterior:** Nome da localidade, uma marca de várias partes para identificar exclusivamente a localidade. O número máximo de caracteres permitidos para essa cadeia de caracteres é 85, incluindo um caractere nulo de terminação. A marca é baseada nas convenções de marcação de idioma do RFC 4646. O padrão a ser usado é descrito em [Nomes de localidade.](locale-names.md)
