# <a name="about-unicode-and-character-sets"></a>Sobre Unicode e conjuntos de caracteres

Os dados baseados em caracteres do mundo foram desenvolvidos usando conjuntos de caracteres [Unicode](unicode.md) e tradicionais. Por isso, o Windows fornece funções de conjunto de caracteres que ajudam os aplicativos a converter os dados baseados em caracteres de seu conjunto de caracteres original definido para Unicode ou outro conjunto de caracteres tradicional. Essas funções de conjunto de caracteres também ajudam os aplicativos a criar dados baseados em caracteres que podem ser transferidos e usados em qualquer sistema operacional, incluindo aqueles que não dão suporte a Unicode.

Para obter detalhes sobre o Unicode além do escopo desta visão geral, consulte [o padrão Unicode](https://www.unicode.org/standard/standard.html).
