# <a name="about-unicode-and-character-sets"></a>Sobre unicode e conjuntos de caracteres

Os dados baseados em caractere do mundo foram desenvolvidos usando [conjuntos de caracteres Unicode](unicode.md) e tradicionais. Por isso, o Windows fornece funções de conjunto de caracteres que ajudam os aplicativos a converter os dados baseados em caractere de seu conjunto de caracteres original para Unicode ou outro conjunto de caracteres tradicional. Essas funções de conjunto de caracteres também ajudam os aplicativos a criar dados baseados em caracteres que podem ser transferidos e usados em qualquer sistema operacional, incluindo aqueles que não têm suporte a Unicode.

Para obter detalhes sobre Unicode além do escopo desta visão geral, consulte [The Unicode Standard](https://www.unicode.org/standard/standard.html).
