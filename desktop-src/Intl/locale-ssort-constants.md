# <a name="locale_ssort_constants"></a>\_constantes SSORT de localidade \_

Este tópico define as constantes de SSORT de localidade \_ \* usadas pelo NLS.



| Valor               | Significado                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SSORTLOCALE de localidade \_ | **Windows 7 e posterior:** Nome da localidade a ser usada para a classificação ou o comportamento de maiúsculas e minúsculas.                                                                                                                                |
| SSORTNAME de localidade \_   | **Windows me/98, windows 2000:** O nome localizado completo da classificação para o identificador de localidade especificado depende do idioma do Shell. Essa constante é usada para determinar o comportamento de maiúsculas e minúsculas e classificação. |
