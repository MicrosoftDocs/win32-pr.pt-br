# <a name="locale_sdayname-constants"></a>Constantes \_ LOCALE SDAYNAME \*

Este tópico define as \_ constantes LOCALE SDAYNAME \* usadas pelo NLS.



| Valor             | Significado                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SDAYNAME1 | Nome longo nativo para segunda-feira. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.    |

| LOCALE \_ SDAYNAME5 | Nome longo nativo para sexta-feira. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.    |
| LOCALE \_ SDAYNAME6 | Nome longo nativo para sábado. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.  |
| LOCALE \_ SDAYNAME7 | Nome longo nativo para domingo. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.    |
