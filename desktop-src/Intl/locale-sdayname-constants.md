# <a name="locale_sdayname-constants"></a>\_Constantes SDAYNAME de localidade \*

Este tópico define as constantes de SDAYNAME de localidade \_ \* usadas pelo NLS.



| Valor             | Significado                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| SDAYNAME1 de localidade \_ | Nome longo nativo para segunda-feira. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.    |
| SDAYNAME2 de localidade \_ | Nome longo nativo para terça-feira. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.   |
| SDAYNAME3 de localidade \_ | Nome longo nativo para quarta-feira. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. |
| SDAYNAME4 de localidade \_ | Nome longo nativo da quinta-feira. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.  |
| SDAYNAME5 de localidade \_ | Nome longo nativo para sexta-feira. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.    |
| SDAYNAME6 de localidade \_ | Nome longo nativo para sábado. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.  |
| SDAYNAME7 de localidade \_ | Nome longo nativo para domingo. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação.    |
