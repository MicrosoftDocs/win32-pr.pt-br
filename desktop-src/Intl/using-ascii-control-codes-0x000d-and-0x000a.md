# <a name="using-ascii-control-codes-0x000d-and-0x000a"></a>Usando códigos de controle ASCII 0x000D e 0x000A

O padrão Unicode não prescreve significativos específicos para os códigos de controle 0x000D (retorno de carro) e 0x000A (avanço de alimentação). Esses códigos não precisam ser usados em combinação. Se usado individualmente, qualquer código pode representar apenas ou ambos os códigos juntos. O aplicativo sempre deve determinar o que esses códigos representam.
