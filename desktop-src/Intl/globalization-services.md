# <a name="globalization-services"></a>Serviços de Globalização

Os Serviços de Globalização expõem o suporte de globalização principal para idiomas e regiões para habilitar a criação de aplicativos globalizados que se aplicam a uma ampla variedade de clientes internacionais. Essa tecnologia fornece a infraestrutura de globalização para dar suporte a uma Windows de usuário consistente e linguística e culturalmente apropriada. Os Serviços de Globalização incluem as seguintes tecnologias:

-   [Serviços Linguísticos Estendidos.](extended-linguistic-services.md) Fornece uma variedade de serviços baseados em texto para desenvolvedores do Windows (incluindo detecção de idioma, transliteração e outras funcionalidades linguísticas principais) e a plataforma e interfaces para aplicativos consumirem serviços facilmente, bem como interfaces para permitir que os desenvolvedores adicionem serviços próprios.
-   [Componentes internacionais para Unicode (ICU).](international-components-for-unicode--icu-.md) Fornece um conjunto maduro e amplamente usado de APIs de globalização de código aberto. A ICU utiliza o amplo CLDR (Common Locale Data Repository) da Unicode como sua biblioteca de dados, fornecendo suporte à globalização para aplicativos de software. A ICU é amplamente portátil e fornece aos aplicativos os mesmos resultados em todas as plataformas.
-   [Suporte a idiomas nacionais.](national-language-support.md) Fornece dados internacionais e APIs para dar suporte à formatação de data, hora e calendário, layouts de teclado, comportamento de classificação, aspectos da habilitação de texto e outras informações de localidade principais.
-   [API de Verificação Ort spell.](spell-checker-api.md) Fornece uma API para os desenvolvedores consumirem a capacidade de verificação ortária para verificar texto, obter sugestões e manter configurações e dicionários de usuário. Os desenvolvedores também podem criar verificadores ort spell que são integrados à estrutura de verificação ort ortús ortagem como provedores.
