# <a name="globalization-services"></a>Serviços de globalização

Os serviços de globalização expõem o suporte de globalização de núcleo para linguagens e regiões para permitir a criação de aplicativos globalizados que se aplicam a uma ampla variedade de clientes internacionais. Essa tecnologia fornece a infraestrutura de globalização para dar suporte a uma experiência de usuário do Windows consistente e, de forma lingüística e cultural, apropriada. Os serviços de globalização incluem as seguintes tecnologias:

-   [Serviços lingüísticos estendidos](extended-linguistic-services.md). Fornece uma variedade de serviços baseados em texto para desenvolvedores do Windows (incluindo detecção de idioma, transliteração e outras funcionalidades linguísticas principais) e a plataforma e as interfaces para que os aplicativos consumam serviços facilmente, bem como interfaces para permitir que os desenvolvedores adicionem seus próprios serviços.
-   [ICU (componentes internacionais para Unicode)](international-components-for-unicode--icu-.md). Fornece um conjunto maduro e amplamente utilizado de APIs de globalização de código aberto. O ICU utiliza o CLDR (repositório de dados de localidade comum) de Unicode como sua biblioteca de dados, fornecendo suporte à globalização para aplicativos de software. O ICU é amplamente portátil e dá aos aplicativos os mesmos resultados em todas as plataformas.
-   [Suporte ao idioma nacional](national-language-support.md). Fornece dados internacionais e APIs para dar suporte à formatação de data, hora e calendário, layouts de teclado, comportamento de classificação, aspectos de habilitação de texto e outras informações de localidade principal.
-   [API de verificação ortográfica](spell-checker-api.md). Fornece uma API para que os desenvolvedores consumam a capacidade de verificação ortográfica para verificar o texto, obter sugestões e manter configurações e dicionários do usuário. Os desenvolvedores também podem criar verificadores ortográficos integrados à estrutura de verificação ortográfica como provedores.
