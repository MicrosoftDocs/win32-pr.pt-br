# <a name="locale_idaylzero"></a>LOCALE \_ IDAYLZERO

Especificador para zeros à esquerda nos campos de dia apenas para uma data curta. O número máximo de caracteres permitidos para essa cadeia de caracteres é dois, incluindo um caractere nulo de terminação. É preferível que seu aplicativo use a constante [LOCALE \_ SSHORTDATE](locale-sshortdate.md) em vez de LOCALE \_ IDAYLZERO.



| Valor | Significado                   |
|-------|---------------------------|
| 0     | Nenhum zero à esquerda por dias |
| 1     | Zeros à esquerda por dias    |
