# <a name="locale_idaylzero"></a>IDAYLZERO de localidade \_

Especificador para os zeros à esquerda em campos de dia apenas para uma data abreviada. O número máximo de caracteres permitido para essa cadeia de caracteres é dois, incluindo um caractere nulo de terminação. É preferível que seu aplicativo use a [constante \_ SSHORTDATE de localidade](locale-sshortdate.md) em vez de IDAYLZERO de localidade \_ .



| Valor | Significado                   |
|-------|---------------------------|
| 0     | Sem zeros à esquerda por dias |
| 1     | Zeros à esquerda por dias    |
