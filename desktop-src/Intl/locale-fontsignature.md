# <a name="locale_fontsignature"></a>fontSignature de localidade \_

**Windows me/98/95, Windows NT 4,0:** Um padrão de bit específico que determina a relação entre a cobertura de caracteres necessária para dar suporte à localidade e ao conteúdo da fonte. Observe que \_ os dados de fontSignature de localidade assumem um formulário diferente de todas as outras informações de localidade. Todas as outras informações de localidade podem ser expressas em um formato de cadeia de caracteres ou como um número. \_Os dados de fontSignature de localidade são recuperados em uma estrutura [LOCALESIGNATURE](/windows/win32/api/wingdi/ns-wingdi-localesignature) .
