# <a name="about-national-language-support"></a>Sobre o suporte ao idioma nacional

As funções NLS (suporte ao idioma nacional) permitem que os aplicativos:

-   Definir a localidade do usuário
-   Identificar o idioma no qual o usuário trabalha
-   Recuperar cadeias de caracteres representando horas, datas e outras informações formatadas corretamente para o idioma e a localidade especificados

O NLS também inclui suporte para layouts de teclado e fontes específicas de idioma. Para obter mais informações, consulte [Gerenciador de métodos de entrada](input-method-manager.md) e [fontes internacionais e exibição de texto](international-fonts-and-text-display.md).
