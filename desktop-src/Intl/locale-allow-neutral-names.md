# <a name="locale_allow_neutral_names"></a>LOCALIDADE \_ permitir \_ nomes neutros \_

**Windows 7 e posterior:** Permitir o retorno de nomes neutros ou LCIDs ao converter entre os nomes de localidade e os identificadores de localidade.
