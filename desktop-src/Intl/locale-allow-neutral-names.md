# <a name="locale_allow_neutral_names"></a>LOCALE \_ PERMITIR \_ NOMES \_ NEUTROS

**Windows 7 e posterior:** Permita o retorno de nomes neutros ou LCIDs ao converter entre nomes de localidade e identificadores de localidade.
