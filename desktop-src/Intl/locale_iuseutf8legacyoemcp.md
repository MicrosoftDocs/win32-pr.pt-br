# <a name="locale_iuseutf8legacyoemcp"></a>IUSEUTF8LEGACYOEMCP de localidade \_

**A partir da próxima versão do Windows:** É recomendável que os aplicativos usem Unicode em vez disso. em computadores que o usuário configurou para usar uma página de código UTF-8, LOCALE_IUSEUTF8LEGACYOEMCP é a página de código OEM herdada que teria sido usada em um computador Windows herdado não UTF-8.
