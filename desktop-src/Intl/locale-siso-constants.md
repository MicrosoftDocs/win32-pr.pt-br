# <a name="locale_siso-constants"></a>\_Constantes siso de localidade \*

Este tópico define as constantes de siso de localidade \_ \* usadas pelo NLS.



| Valor                     | Significado                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SISO3166CTRYNAME de localidade \_  | **Windows me/98, Windows NT 4,0:** Nome do país/região, com base no ISO padrão 3166, como "US" para o Estados Unidos. Isso também pode retornar um número, como "029" para o Caribe. O número máximo de caracteres permitido para essa cadeia de caracteres é nove, incluindo um caractere nulo de terminação.                                                                                        |
| SISO3166CTRYNAME2 de localidade \_ | **Windows Vista e posterior:** Nome da região ISO de três letras (código de letra ISO 3166 3 para o país/região), como "EUA" para o Estados Unidos. Isso também pode retornar um número, como "029" para o Caribe. O número máximo de caracteres permitido para essa cadeia de caracteres é nove, incluindo um caractere nulo de terminação.                                                            |
| SISO639LANGNAME de localidade \_   | **Windows me/98, Windows NT 4,0:** O nome abreviado da linguagem com base inteiramente nos valores ISO Standard 639, em formato minúsculo, como "en" para inglês. Esse pode ser um código de três letras para idiomas que não têm um código de duas letras, como "Haw" para o havaiano. O número máximo de caracteres permitido para essa cadeia de caracteres é nove, incluindo um caractere nulo de terminação. |
| SISO639LANGNAME2 de localidade \_  | **Windows Vista e posterior:** Nome de idioma ISO de três letras, em formato minúsculo (código de três letras ISO 639-2 para a linguagem), como "ENG" para inglês. O número máximo de caracteres permitido para essa cadeia de caracteres é nove, incluindo um caractere nulo de terminação.                                                                                                                  |
