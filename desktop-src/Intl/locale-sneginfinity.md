# <a name="locale_sneginfinity"></a>SNEGINFINITY de localidade \_

**Windows Vista e posterior:** Valor da cadeia de caracteres de "infinito negativo", por exemplo, "-Infinity" para a localidade inglês (Estados Unidos). Não há nenhum limite para o número de caracteres permitido para essa cadeia.
