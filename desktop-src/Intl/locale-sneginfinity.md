# <a name="locale_sneginfinity"></a>LOCALE \_ SNEGINFINITY

**Windows Vista e posterior:** Valor da cadeia de caracteres para "infinito negativo", por exemplo, "-Infinity" para a localidade inglês (Estados Unidos). Não há limite para o número de caracteres permitido para essa cadeia de caracteres.
