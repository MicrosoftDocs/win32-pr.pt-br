# <a name="uniscribe"></a>Uniscribe

Uniscribe é um conjunto de APIs que permitem um alto grau de controle para tipografia fina e para processamento de scripts complexos. Scripts complexos e scripts simples com efeitos de tipografia finos exigem processamento especial para exibição e edição porque os caracteres ("glifos") não são apresentados de maneira simples. Para scripts complexos, as regras que regem a formatação e o posicionamento de glifos são especificadas e catalogadas [no Padrão Unicode.](https://www.unicode.org/)

Este tópico aborda vários aspectos da tipografia fina e o processamento de scripts complexos e é apresentado da seguinte forma:

-   [Sobre o Uniscribe](about-uniscribe.md)
-   [Usando o Uniscribe](using-uniscribe.md)
-   [Referência do Uniscribe](uniscribe-reference.md)
