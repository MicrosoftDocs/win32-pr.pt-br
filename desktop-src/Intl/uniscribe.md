# <a name="uniscribe"></a>Cancelar

O Uniscribe é um conjunto de APIs que permite um alto grau de controle para a tipografia fina e para o processamento de scripts complexos. Tanto scripts complexos quanto scripts simples com efeitos de tipografia finas exigem processamento especial para exibir e editar porque os caracteres ("glifos") não são dispostos de forma simples. Para scripts complexos, as regras que regem a formatação e o posicionamento de glifos são especificadas e catalogadas no [padrão Unicode](https://www.unicode.org/).

Este tópico aborda vários aspectos da tipografia fina e o processamento de scripts complexos, e é apresentado da seguinte maneira:

-   [Sobre o Uniscribe](about-uniscribe.md)
-   [Usando o Uniscribe](using-uniscribe.md)
-   [Referência do Uniscribe](uniscribe-reference.md)
