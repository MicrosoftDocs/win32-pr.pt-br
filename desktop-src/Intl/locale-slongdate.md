# <a name="locale_slongdate"></a>LOCALE \_ SLONGDATE

Cadeia de caracteres de formatação de data longa para a localidade. O número máximo de caracteres permitidos para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. A cadeia de caracteres pode consistir em uma combinação de imagens de formato de [dia, mês,](day--month--year--and-era-format-pictures.md) ano e era e qualquer cadeia de caracteres entre aspas simples. Os caracteres entre aspas simples permanecem conforme especificado. Por exemplo, a data longa do espanhol (Espanha) é "dddd, dd" de 'MMMM' de 'aaaa'. As localidades podem definir vários formatos de data longos.

Para obter todos os formatos de data longa para uma localidade, use [EnumDateFormats,](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsa) [EnumDateFormatsEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexa)ou [EnumDateFormatsExEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexex).
