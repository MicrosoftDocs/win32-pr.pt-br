# <a name="locale_slongdate"></a>SLONGDATE de localidade \_

Cadeia de caracteres de formatação de data por extenso para a localidade. O número máximo de caracteres permitido para essa cadeia de caracteres é 80, incluindo um caractere nulo de terminação. A cadeia de caracteres pode consistir em uma combinação de [imagens de formato de dia, mês, ano e era](day--month--year--and-era-format-pictures.md) e qualquer cadeia de caracteres entre aspas simples. Os caracteres entre aspas simples permanecem conforme especificado. Por exemplo, a data por extenso espanhola (Espanha) é "dddd, dd ' de ' MMMM ' de ' yyyy". As localidades podem definir vários formatos de datas longas.

Para obter todos os formatos de datas longas para uma localidade, use [EnumDateFormats](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsa), [EnumDateFormatsEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexa)ou [EnumDateFormatsExEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexex).
