# <a name="candidate-lists"></a>Listas de candidatos

Uma lista de candidatos é representada por uma estrutura [**CANDIDATELIST**](/windows/win32/api/imm/ns-imm-candidatelist) que contém uma matriz de cadeias de caracteres que especificam o caractere ou as cadeias de caracteres das quais o usuário pode escolher. Seu aplicativo com reconhecimento de IME pode recuperar informações de lista de candidatos usando as funções [**ImmGetCandidateListCount**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelistcounta) e [**ImmGetCandidateList**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelista) .
