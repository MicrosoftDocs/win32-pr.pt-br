# <a name="locale_sintlsymbol"></a>LOCALE \_ SINTLSYMBOL

Três caracteres do símbolo monetário internacional especificado em ISO 4217, seguidos pelo caractere que separa essa cadeia de caracteres da quantidade. O número máximo de caracteres permitidos para essa cadeia de caracteres é de nove, incluindo um caractere nulo de terminação.
