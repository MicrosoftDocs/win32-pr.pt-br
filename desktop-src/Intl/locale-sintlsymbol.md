# <a name="locale_sintlsymbol"></a>SINTLSYMBOL de localidade \_

Três caracteres do símbolo monetário internacional especificado em ISO 4217, seguido pelo caractere que separa essa cadeia de caracteres do valor. O número máximo de caracteres permitido para essa cadeia de caracteres é nove, incluindo um caractere nulo de terminação.
