# <a name="locale_idialingcode-and-locale_icountry"></a>LOCALIDADE \_ IDIALINGCODE e localidade \_ ICOUNTRY

Código de país/região, com base em códigos de telefone internacionais, também chamados de códigos de país/região da IBM. O número máximo de caracteres permitido para essa cadeia de caracteres é seis, incluindo um caractere nulo de terminação.

O Windows 10 adicionou \_ a localidade IDIALINGCODE como um sinônimo com nome mais preciso para a localidade \_ ICOUNTRY. Incentivamos você a usar o novo nome para melhorar a legibilidade do código.

> [!Note]  
> Os códigos de discagem internacionais não são exclusivos para um país específico. Por exemplo, vários países, incluindo o Estados Unidos e o Canadá, usam "1" como o código de discagem internacional.
