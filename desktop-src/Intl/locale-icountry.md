# <a name="locale_idialingcode-and-locale_icountry"></a>LOCALE \_ IDIALINGCODE E \_ LOCALE ICOUNTRY

Código de país/região, com base em códigos de telefone internacionais, também conhecido como códigos de país/região da IBM. O número máximo de caracteres permitidos para essa cadeia de caracteres é de seis, incluindo um caractere nulo de terminação.

Windows 10 localE IDIALINGCODE como um sinônimo nomeado com mais \_ precisão para LOCALE \_ ICOUNTRY. Incentivamos você a usar o novo nome para melhorar a facilitar a leitura do código.

> [!Note]  
> Códigos de discagem internacionais não são exclusivos para um determinado país. Por exemplo, vários países, incluindo o Estados Unidos e o Canadá, usam "1" como o código de discagem internacional.
