# <a name="locale_ireadinglayout"></a>LOCALE \_ IREADINGLAYOUT

**Windows 7 e posteriores:** O layout de leitura para texto. Os valores possíveis são definidos na tabela a seguir.



| Valor | Significado                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Leia da esquerda para a direita, como para a localidade Estados Unidos inglês.                                                                                                 |
| 1     | Leia da direita para a esquerda, como para localidades árabe.                                                                                                                     |
| 2     | Leia verticalmente de cima para baixo com colunas indo da direita para a esquerda ou leia em linhas horizontais da esquerda para a direita, como para a localidade japonês (Japão). |
| 3     | Leia verticalmente de cima para baixo com colunas da esquerda para a direita, como para a localidade Deliano (Turco).                                                  |
