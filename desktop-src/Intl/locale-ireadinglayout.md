# <a name="locale_ireadinglayout"></a>IREADINGLAYOUT de localidade \_

**Windows 7 e posterior:** O layout de leitura do texto. Os valores possíveis são definidos na tabela a seguir.



| Valor | Significado                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Leia da esquerda para a direita, como para a localidade em inglês (Estados Unidos).                                                                                                 |
| 1     | Leia da direita para a esquerda, como para as localidades árabes.                                                                                                                     |
| 2     | Leia verticalmente de cima para baixo com colunas indo da direita para a esquerda ou leia em linhas horizontais da esquerda para a direita, como para a localidade japonesa (Japão). |
| 3     | Leia verticalmente de cima para baixo com colunas que vão da esquerda para a direita, como para a localidade mongol (mongol).                                                  |
