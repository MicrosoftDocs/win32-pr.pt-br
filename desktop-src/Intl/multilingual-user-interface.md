# <a name="multilingual-user-interface"></a>Interface do Usuário Multilíngue

Interface de Usuário Multilíngue (MUI) permite a localização de interfaces do usuário para aplicativos globalizados. A MUI também dá suporte à criação de recursos para qualquer número de linguagens de interface do usuário.

Esta documentação descreve a MUI e informa como adicionar a funcionalidade de MUI ao seu aplicativo totalmente globalizado. No entanto, ele não tenta descrever todos os detalhes do desenvolvimento globalizado de aplicativos Windows. Para obter detalhes desse tipo, consulte o Centro de [DesenvolvedorEs Globais do](https://msdn.microsoft.com/goglobal)Microsoft Go.

A documentação da MUI inclui as seguintes seções:

-   [Sobre Interface de Usuário Multilíngue](about-multilingual-user-interface.md)
-   [Usando Interface de Usuário Multilíngue](using-multilingual-user-interface.md)
-   [Interface de Usuário Multilíngue Referência](multilingual-user-interface-reference.md)
