# <a name="multilingual-user-interface"></a>Interface do Usuário Multilíngue

A MUI (Multilingual User interface) permite a localização de interfaces do usuário para aplicativos globalizados. O MUI também dá suporte à criação de recursos para qualquer número de idiomas da interface do usuário.

Esta documentação descreve o MUI e explica como adicionar a funcionalidade do MUI ao aplicativo totalmente globalizado. No entanto, ele não tenta descrever todos os detalhes do desenvolvimento de aplicativos globalizados no Windows. Para obter detalhes desse tipo, consulte o centro de desenvolvedores do Microsoft [Go Global](https://msdn.microsoft.com/goglobal).

A documentação do MUI inclui as seguintes seções:

-   [Sobre a interface do usuário multilíngue](about-multilingual-user-interface.md)
-   [Usando a interface do usuário multilíngue](using-multilingual-user-interface.md)
-   [Referência de interface do usuário multilíngue](multilingual-user-interface-reference.md)
