# <a name="locale_snegativesign"></a>SNEGATIVESIGN de localidade \_

Valor da cadeia de caracteres para o sinal negativo, por exemplo, "-" para a localidade em inglês (Estados Unidos). O número máximo de caracteres permitido para essa cadeia de caracteres é cinco, incluindo um caractere nulo de terminação.
