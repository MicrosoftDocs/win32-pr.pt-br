# <a name="locale_senglish-constants"></a>Constantes \_ LOCALE SENGLISH \*

Este tópico define as \_ constantes LOCALE SENGLISH \* usadas pelo NLS.



| Valor                        | Significado                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SENGLISHCOUNTRYNAME  | **Windows 7 e posterior:** Nome em inglês do país/região, por exemplo, Alemanha para Deutschland. Esse nome sempre é restrito a caracteres que podem ser mapeados para o subconjunto ASCII de 127 caracteres. <br/>                                      |
| LOCALE \_ SENGLISHDISPLAYNAME  | **Windows 7 e posterior:** Nome de exibição da localidade em inglês. Normalmente, o nome de exibição consiste no idioma e no país/região, por exemplo, alemão (Alemanha) para Deutsch (Deutschland). <br/>                                           |
| LOCALE \_ SENGLISHLANGUAGENAME | **Windows 7 e posterior:** Nome em inglês do idioma em inglês, por exemplo, alemão para Deutsch, do ISO Standard Internacional 639. Esse nome sempre é restrito a caracteres que podem ser mapeados para o subconjunto ASCII de 127 caracteres. <br/> |
