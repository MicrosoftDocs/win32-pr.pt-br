# <a name="locale_senglish-constants"></a>\_Constantes SENGLISH de localidade \*

Este tópico define as constantes de SENGLISH de localidade \_ \* usadas pelo NLS.



| Valor                        | Significado                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SENGLISHCOUNTRYNAME de localidade \_  | **Windows 7 e posterior:** Nome em inglês do país/região, por exemplo, Alemanha para Deutschland. Esse nome é sempre restrito a caracteres que podem ser mapeados para o subconjunto de caracteres ASCII 127. <br/>                                      |
| SENGLISHDISPLAYNAME de localidade \_  | **Windows 7 e posterior:** Nome de exibição da localidade em inglês. Normalmente, o nome de exibição consiste no idioma e no país/região, por exemplo, alemão (Alemanha) para Deutsch (Deutschland). <br/>                                           |
| SENGLISHLANGUAGENAME de localidade \_ | **Windows 7 e posterior:** O nome em inglês do idioma em inglês, por exemplo, alemão para Deutsch, do International ISO Standard 639. Esse nome é sempre restrito a caracteres que podem ser mapeados para o subconjunto de caracteres ASCII 127. <br/> |
