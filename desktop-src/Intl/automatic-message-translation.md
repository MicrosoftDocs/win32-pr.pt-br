# <a name="automatic-message-translation"></a>Tradução automática de mensagens

Os aplicativos que usam as funções Unicode e API de conjunto de caracteres geralmente usam a mesma classe de janela. O sistema operacional converte mensagens de forma transparente entre janelas de classes diferentes.

Uma função de janela é implementada para receber mensagens em formato Unicode ou de página de código do Windows. A função Window pode enviar mensagens ou chamar funções de qualquer um dos tipos.

As mensagens a seguir têm argumentos de texto e estão sujeitas à tradução automática de texto. Para obter informações sobre a tradução automática, consulte [subclasse e conversão automática de mensagens](subclassing-and-automatic-message-translation.md).
