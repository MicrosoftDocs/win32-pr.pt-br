# <a name="locale_sthousand"></a>STHOUSAND de localidade \_

Caracteres que são usados para separar grupos de dígitos à esquerda do decimal. O número máximo de caracteres permitido para essa cadeia de caracteres é quatro, incluindo um caractere nulo de terminação. Normalmente, esses grupos representam milhares. No entanto, dependendo do valor especificado para [ \_ SGROUPING de localidade](locale-sgrouping.md), eles podem representar outra coisa.
