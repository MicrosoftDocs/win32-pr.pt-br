# <a name="locale_sthousand"></a>LOCALE \_ STHOUSAND

Caracteres usados para separar grupos de dígitos à esquerda do decimal. O número máximo de caracteres permitidos para essa cadeia de caracteres é quatro, incluindo um caractere nulo de terminação. Normalmente, esses grupos representam milhares. No entanto, dependendo do valor especificado para [LOCALE \_ SGROUPING](locale-sgrouping.md), eles podem representar outra coisa.
