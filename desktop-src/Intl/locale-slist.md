# <a name="locale_slist"></a>SLIST da localidade \_

Os caracteres usados para separar itens de lista, por exemplo, uma vírgula são usados em muitas localidades. O número máximo de caracteres permitido para essa cadeia de caracteres é quatro, incluindo um caractere nulo de terminação.
