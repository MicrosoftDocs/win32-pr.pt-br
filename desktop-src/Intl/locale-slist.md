# <a name="locale_slist"></a>LOCALE \_ SLIST

Caracteres usados para separar itens de lista, por exemplo, uma vírgula é usada em muitas localidades. O número máximo de caracteres permitidos para essa cadeia de caracteres é quatro, incluindo um caractere nulo de terminação.
