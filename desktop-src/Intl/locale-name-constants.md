# <a name="locale_name-constants"></a>\_Constantes de nome de localidade \*

Este tópico define as constantes de nome de localidade \_ \* usadas pelo NLS.




| Valor | Significado | 
|-------|---------|
| LOCALE_NAME_INVARIANT | Nome de uma localidade invariável que fornece dados de localidade e calendário estáveis. | 
| LOCALE_NAME_MAX_LENGTH | Comprimento máximo de um nome de localidade. O número máximo de caracteres permitido para essa cadeia de caracteres é 85, incluindo um caractere nulo de terminação.<blockquote>[!Note]<br />Seu aplicativo deve usar a constante para o comprimento máximo do nome de localidade, em vez de embutir em código o valor "85".</blockquote><br /> | 
| LOCALE_NAME_SYSTEM_DEFAULT | Nome da localidade do sistema operacional atual. | 
| LOCALE_NAME_USER_DEFAULT | Nome da localidade do usuário atual, correspondente à preferência definida na parte das opções regionais e de idioma do painel de controle. Essa localidade pode ser diferente da localidade para o idioma atual da interface do usuário. |
