# <a name="locale_name-constants"></a>\_Constantes de nome de localidade \*

Este tópico define as constantes de nome de localidade \_ \* usadas pelo NLS.



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Valor</th>
<th>Significado</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td>LOCALE_NAME_INVARIANT</td>
<td>Nome de uma localidade invariável que fornece dados de localidade e calendário estáveis.</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_MAX_LENGTH</td>
<td>Comprimento máximo de um nome de localidade. O número máximo de caracteres permitido para essa cadeia de caracteres é 85, incluindo um caractere nulo de terminação.
<blockquote>
[!Note]<br />
Seu aplicativo deve usar a constante para o comprimento máximo do nome de localidade, em vez de embutir em código o valor &quot; 85 &quot; .
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td>LOCALE_NAME_SYSTEM_DEFAULT</td>
<td>Nome da localidade do sistema operacional atual.</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_USER_DEFAULT</td>
<td>Nome da localidade do usuário atual, correspondente à preferência definida na parte das opções regionais e de idioma do painel de controle. Essa localidade pode ser diferente da localidade para o idioma atual da interface do usuário.</td>
</tr>
</tbody>
</table>
