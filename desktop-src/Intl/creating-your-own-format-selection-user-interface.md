# <a name="international-text-display"></a>Exibição de texto internacional

Há quatro opções possíveis para exibir texto internacional, que também envolve a saída de scripts complexos:

-   Chamando APIs de texto do Windows
-   Instanciando controles de edição padrão do Windows
-   Criando uma instância de controles de edição avançados
-   Chamando o Uniscribe

Consulte [as opções para exibir o texto](https://msdn.microsoft.com/globalization/mt662335) para obter a explicação e as vantagens de cada uma dessas opções. Cabe a você decidir qual é a melhor opção para seu aplicativo, com base na complexidade do aplicativo e em seus recursos de design.
