# <a name="locale_imeasure"></a>IMEASURE de localidade \_

Sistema de medição. O número máximo de caracteres permitido para essa cadeia de caracteres é dois, incluindo um caractere nulo de terminação. Esse valor será 0 se o sistema de métrica (Systéme International d'Units ou S.I.) for usado e 1 se o sistema de Estados Unidos for usado.
