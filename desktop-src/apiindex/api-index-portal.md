# <a name="api-index-for-desktop-windows-applications"></a>Índice de API para aplicativos Windows área de trabalho

Este artigo fornece links para documentação de referência para APIs que podem ser usadas em aplicativos Windows área de trabalho.

## <a name="win32-windows-api"></a>Win32 (API do Windows)

A API do Win32 (também chamada de API Windows) é a plataforma nativa para Windows aplicativos. Essa API é melhor para aplicativos da área de trabalho que exigem acesso direto aos recursos e ao hardware do sistema. A API Windows pode ser usada em todos os aplicativos da área de trabalho e as mesmas funções geralmente têm suporte em 32 bits e 64 bits Windows.

* [Referência da API do Win32 por recurso](windows-api-list.md)
* [Referência da API do Win32 por header](/windows/win32/api/)
* [APIs COM e Win32 para aplicativos UWP](/uwp/win32-and-com/win32-and-com-for-uwp-apps)
* [Bibliotecas de abrangência do Windows](windows-umbrella-libraries.md)
* [Conjuntos de API do Windows](windows-apisets.md)

## <a name="windows-runtime-winrt"></a>Windows Runtime (WinRT)

O WinRT é a plataforma de borda líder para Windows 10 aplicativos e jogos, incluindo aplicativos da área de trabalho. A API do WinRT é adequada para aplicativos nativos do C++ e da área de trabalho gerenciada que exigem uma interface do usuário sofisticada, personalização de estilos e cenários com uso intensivo de elementos gráficos.

* [Referência da API WinRT](/uwp/api/)
* [APIs do WinRT que podem ser chamadas de um aplicativo para desktop](uwp-apis-callable-from-a-classic-desktop-app.md)

## <a name="net"></a>.NET

As bibliotecas de classes do .NET fornecem acesso Windows recursos do sistema e da interface do usuário para aplicativos de área de trabalho gerenciados, incluindo aplicativos WPF e Windows Forms.

* [API do .NET](/dotnet/api/index)
