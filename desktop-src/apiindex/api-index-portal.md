# <a name="api-index-for-desktop-windows-applications"></a>Índice de API para aplicativos do Windows para desktop

Este artigo fornece links para a documentação de referência para APIs que podem ser usadas em aplicativos de área de trabalho do Windows.

## <a name="win32-windows-api"></a>Win32 (API do Windows)

A API do Win32 (também chamada de API do Windows) é a plataforma nativa para aplicativos do Windows. Essa API é melhor para aplicativos de área de trabalho que exigem acesso direto aos recursos do sistema e ao hardware. A API do Windows pode ser usada em todos os aplicativos da área de trabalho e as mesmas funções geralmente têm suporte em Windows de 32 bits e 64 bits.

* [Referência de API do Win32 por recurso](windows-api-list.md)
* [Referência da API do Win32 por cabeçalho](/windows/win32/api/)
* [APIs COM e Win32 para aplicativos UWP](/uwp/win32-and-com/win32-and-com-for-uwp-apps)
* [Bibliotecas de abrangência do Windows](windows-umbrella-libraries.md)
* [Conjuntos de API do Windows](windows-apisets.md)

## <a name="windows-runtime-winrt"></a>Windows Runtime (WinRT)

O WinRT é a principal plataforma de ponta para aplicativos e jogos do Windows 10, incluindo aplicativos da área de trabalho. A API do WinRT é adequada para C++ nativo e aplicativos de área de trabalho gerenciados que exigem uma interface do usuário sofisticada, personalização de estilos e cenários com uso intensivo de gráficos.

* [Referência da API WinRT](/uwp/api/)
* [APIs do WinRT que podem ser chamadas de aplicativos do Win32, do WPF e do Windows Forms desktop](uwp-apis-callable-from-a-classic-desktop-app.md)

## <a name="net"></a>.NET

As bibliotecas de classes do .NET fornecem acesso aos recursos do sistema Windows e da interface do usuário para aplicativos de área de trabalho gerenciado, incluindo WPF e Windows Forms aplicativos.

* [API do .NET](/dotnet/api/index)
