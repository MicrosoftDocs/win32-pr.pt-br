# <a name="windows-event-collector-enumerations"></a>Enumerações do coletor de eventos do Windows

A tabela a seguir lista as enumerações do SDK do coletor de eventos do Windows.



| Enumeração                                                                                               | Descrição                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_modo de \_ configuração de assinatura do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Especifica modos de configuração diferentes que alteram as configurações padrão de uma assinatura.                                            |
| [**\_formato de \_ conteúdo de assinatura do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Especifica como os eventos serão processados no computador que envia os eventos antes que os eventos sejam enviados para o computador do coletor de eventos. |
| [**\_tipo de \_ credenciais de assinatura do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Especifica o tipo de credenciais a ser usado ao se comunicar com origens de evento.                                                         |
| [**\_modo de \_ entrega de assinatura do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Especifica como os eventos são entregues por meio de uma assinatura de evento (usando um modelo de Push ou pull).                                          |
| [**\_ID da \_ propriedade da assinatura do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Define valores para identificar as propriedades de assinatura de evento usadas para configuração de assinatura.                                           |
| [**\_ \_ \_ status ativo do status de tempo de execução da \_ assinatura do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Especifica o status de uma assinatura ou uma origem do evento em relação a uma assinatura.                                               |
| [**\_ID das \_ informações de status do tempo de execução da assinatura do \_ \_ \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Especifica um valor que identifica uma propriedade do status de tempo de execução de uma origem de evento ou uma assinatura.                                |
| [**\_tipo de assinatura do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Especifica o tipo de assinatura a ser usado (uma assinatura iniciada por uma fonte ou iniciada pelo coletor).                                     |
| [**\_tipo de variante do EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Define os valores que especificam os tipos de dados que são usados nas funções do coletor de eventos do Windows.                                  |
