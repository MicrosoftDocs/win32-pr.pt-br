# <a name="windows-event-collector-enumerations"></a>Windows Enumerações do Coletor de Eventos

A tabela a seguir lista as enumerações do SDK Windows Coletor de Eventos.



| Enumeração                                                                                               | Descrição                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**MODO DE \_ CONFIGURAÇÃO \_ DE ASSINATURA \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Especifica diferentes modos de configuração que alteram as configurações padrão de uma assinatura.                                            |
| [**FORMATO DE \_ CONTEÚDO \_ DA ASSINATURA \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Especifica como os eventos serão renderizados no computador que envia os eventos antes que os eventos sejam enviados ao computador do coletor de eventos. |
| [**TIPO \_ DE CREDENCIAIS DE ASSINATURA \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Especifica o tipo de credenciais a ser usado ao se comunicar com fontes de eventos.                                                         |
| [**MODO DE \_ ENTREGA \_ DE ASSINATURA \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Especifica como os eventos são entregues por meio de uma assinatura de evento (usando um modelo de push ou pull).                                          |
| [**\_ID DA \_ PROPRIEDADE DA ASSINATURA \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Define valores para identificar as propriedades de assinatura de evento usadas para a configuração da assinatura.                                           |
| [**STATUS \_ ATIVO DO STATUS DO \_ RUNTIME DA ASSINATURA \_ \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Especifica o status de uma assinatura ou uma origem de evento em relação a uma assinatura.                                               |
| [**\_ID DE INFORMAÇÕES \_ DE STATUS DO \_ RUNTIME DA ASSINATURA \_ \_ DO EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Especifica um valor que identifica uma propriedade do status de runtime de uma origem do evento ou uma assinatura.                                |
| [**TIPO \_ DE ASSINATURA \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Especifica o tipo de assinatura a ser usado (uma assinatura iniciada por origem ou iniciada pelo coletor).                                     |
| [**TIPO \_ DE VARIANTE DE \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Define os valores que especificam os tipos de dados usados nas funções Windows Coletor de Eventos.                                  |
