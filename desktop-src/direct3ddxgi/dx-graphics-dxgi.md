# <a name="dxgi"></a><span data-ttu-id="e2d33-103">DXGI</span><span class="sxs-lookup"><span data-stu-id="e2d33-103">DXGI</span></span>

<span data-ttu-id="e2d33-104">O DXGI (Microsoft DirectX Graphic Infrastructure) lida com a enumeração de adaptadores gráficos, a enumeração de modos de exibição, a seleção de formatos de buffer, o compartilhamento de recursos entre processos (por exemplo, entre aplicativos e o DWM (Gerenciador de Janelas da Área de Trabalho) e a apresentação de quadros renderizados em uma janela ou monitor para exibição.</span><span class="sxs-lookup"><span data-stu-id="e2d33-104">Microsoft DirectX Graphics Infrastructure (DXGI) handles enumerating graphics adapters, enumerating display modes, selecting buffer formats, sharing resources between processes (such as, between applications and the Desktop Window Manager (DWM)), and presenting rendered frames to a window or monitor for display.</span></span>

<span data-ttu-id="e2d33-105">O DXGI é usado pelo Direct3D 10, Direct3D 11 e Direct3D 12.</span><span class="sxs-lookup"><span data-stu-id="e2d33-105">DXGI is used by Direct3D 10, Direct3D 11 and Direct3D 12.</span></span>

<span data-ttu-id="e2d33-106">Embora a maior parte da programação gráfica seja feita usando o Direct3D, você pode usar o DXGI para apresentar quadros a uma janela, monitor ou outro componente gráfico para eventual composição e exibição.</span><span class="sxs-lookup"><span data-stu-id="e2d33-106">Though most graphics programming is done using Direct3D, you can use DXGI to present frames to a window, monitor, or other graphics component for eventual composition and display.</span></span> <span data-ttu-id="e2d33-107">Você também pode usar o DXGI para ler o conteúdo em um monitor.</span><span class="sxs-lookup"><span data-stu-id="e2d33-107">You can also use DXGI to read the contents on a monitor.</span></span>
