# <a name="mru-source-list"></a>Lista de origem MRU

A API de instalação do fornece funções que armazenam os diretórios de origem usados mais recentemente (MRU). Essas informações são armazenadas no sistema do usuário e podem ser acessadas por instalações subsequentes.

Os tópicos a seguir descrevem a lista de origem MRU e como usar as funções de instalação para abrir, modificar, verificar e fechá-la:

-   [Sobre a lista de origem do MRU](about-the-mru-source-list.md)
-   [Usando a lista de origem MRU](using-the-mru-source-list.md)
-   [Referência da lista de origem MRU](mru-source-list-reference.md)
