# <a name="disk-prompting-and-error-handling-functions"></a>Funções de alerta de disco e tratamento de erros

As funções a seguir fornecem solicitação de disco e uma interface do usuário de tratamento de erros.



| Função                                         | Descrição                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora)         | Gera uma caixa de diálogo que informa ao usuário um erro de cópia.                          |
| [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)     | Gera uma caixa de diálogo que informa ao usuário um erro de exclusão.                      |
| [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska) | Gera uma caixa de diálogo que solicita ao usuário um disco de mídia ou um local de arquivo de origem. |
| [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora)     | Gera uma caixa de diálogo que informa ao usuário um erro de renomeação.                        |
