# <a name="cabinet-file-notifications"></a>Notificações de arquivo de gabinete

As notificações a seguir são enviadas pelo [**SetupIterateCabinet**](/windows/desktop/api/Setupapi/nf-setupapi-setupiteratecabineta). Para obter mais informações sobre o formato e o uso de notificações, consulte [notificações](notifications.md).



| Notificação                                                        | Descrição                                                                    |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**SPFILENOTIFY \_ extraied**](spfilenotify-fileextracted.md)   | O arquivo foi extraído do gabinete.                                  |
| [**SPFILENOTIFY \_ FILEINCABINET**](spfilenotify-fileincabinet.md)   | Um arquivo é encontrado no gabinete, o aplicativo deseja extraí-lo? |
| [**SPFILENOTIFY \_ NEEDNEWCABINET**](spfilenotify-neednewcabinet.md) | O arquivo atual continua no próximo gabinete.                             |
