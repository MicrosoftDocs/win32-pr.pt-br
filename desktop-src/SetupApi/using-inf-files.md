# <a name="using-inf-files"></a>Usando arquivos INF

Depois de criar um arquivo INF para o processo de instalação, as funções de instalação podem acessar essas informações para usar ao copiar, excluir ou renomear arquivos e atualizar INIs e entradas de registro.

Os tópicos a seguir descrevem como usar as funções de instalação para recuperar informações de arquivos INF e como executar operações de instalação referenciando um arquivo INF:

-   [Criando um arquivo INF](authoring-an-inf-file.md)
-   [Abrindo e fechando um arquivo INF](opening-and-closing-an-inf-file.md)
-   [Recuperando informações de um arquivo INF](retrieving-information-from-an-inf-file.md)
-   [Instalando a partir de um arquivo INF](installing-from-an-inf-file.md)
-   [Especificando um descritor de segurança de um arquivo INF](specifying-a-security-descriptor-from-an-inf-file.md)
