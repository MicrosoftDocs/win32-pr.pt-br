# <a name="about-the-file-log"></a>Sobre o log de arquivo

Um log de arquivo registra informações sobre os arquivos copiados para um sistema durante uma instalação. Ele pode ser usado para reparo de emergência ou como ferramenta de diagnóstico.

O arquivo usado para as instalações de arquivos de log pode ser o log do sistema ou qualquer outro log de arquivo que você especificar.
