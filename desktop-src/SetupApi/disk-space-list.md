# <a name="disk-space-list"></a>Lista de Disk-Space

A API de instalação fornece a funcionalidade de lista de espaço em disco. Uma lista de espaço em disco é uma lista de operações de arquivo que você pode consultar para determinar a quantidade de espaço em disco necessária para essas operações.

Os tópicos a seguir descrevem as listas de espaço em disco em detalhes, como calculam o espaço necessário e como usá-las em um aplicativo de instalação:

-   [Sobre a lista de Disk-Space](about-the-disk-space-list.md)
-   [Usando a lista de Disk-Space](using-the-disk-space-list.md)
-   [Referência de lista de espaço em disco](disk-space-list-reference.md)
