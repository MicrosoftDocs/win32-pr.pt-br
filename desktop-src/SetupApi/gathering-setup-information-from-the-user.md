# <a name="gathering-setup-information-from-the-user"></a>Coletando informações de instalação do usuário

É altamente recomendável que o aplicativo de instalação inclua uma interface do usuário que adquire todas as informações necessárias do usuário. Essa funcionalidade não é fornecida pelas funções de instalação e deve ser implementada pelo aplicativo de instalação, geralmente na forma de um assistente de instalação.
