# <a name="mru-source-functions"></a>Funções de origem MRU

As funções a seguir são usadas com listas de arquivos de origem MRU.



| Função                                                                 | Descrição                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**SetupAddToSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupaddtosourcelista)                     | Adiciona uma entrada a uma lista de origem.                                                                   |
| [**SetupCancelTemporarySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupcanceltemporarysourcelist) | Cancela qualquer lista temporária ou comportamento de não navegação e restabelece o comportamento da lista padrão.        |
| [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista)                       | Libera recursos alocados para uma lista de origem.                                                       |
| [**SetupQuerySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupquerysourcelista)                     | Consulta a lista atual de fontes de instalação.                                                 |
| [**SetupRemoveFromSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefromsourcelista)           | Remove uma entrada de uma lista de origem de instalação.                                                |
| [**SetupSetSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupsetsourcelista)                         | Define a lista de origem da instalação para a lista MRU do sistema, a lista MRU do usuário ou uma lista temporária. |
