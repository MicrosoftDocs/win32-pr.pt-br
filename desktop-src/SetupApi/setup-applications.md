# <a name="setup-applications"></a>Aplicativos de instalação

Normalmente, você cria um *arquivo inf* antes de criar um aplicativo de instalação. Um arquivo INF é um arquivo de texto que contém informações usadas pelo aplicativo de instalação durante uma instalação. Exemplos das informações que podem ser armazenadas em um arquivo INF incluem alterações de INI e registro, nomes de arquivo e locais dos arquivos de origem na mídia de origem.

Depois de criar um arquivo INF, você escreve o código-fonte do seu aplicativo de instalação. Você pode usar as funções de instalação em seu aplicativo de instalação para acessar as informações no arquivo INF e executar operações de instalação.

As seções a seguir descrevem o uso de arquivos INF e as funções de processamento de instalação:

-   [Sobre arquivos INF](about-inf-files.md)
-   [Usando arquivos INF](using-inf-files.md)
-   [Funções de configuração de arquivo INF](inf-file-setup-functions.md)

Para obter mais informações sobre como usar as funções de instalação em um aplicativo, consulte [criando aplicativos de instalação](creating-setup-applications.md).
