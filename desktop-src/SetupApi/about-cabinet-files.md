# <a name="about-cabinet-files"></a>Sobre arquivos de gabinete

Um gabinete é um único arquivo, geralmente sufixo com .CAB, que armazena arquivos compactados em uma biblioteca de arquivos. Um arquivo compactado pode ser distribuído em vários arquivos de gabinete. Durante a instalação, o aplicativo de instalação descompacta os arquivos armazenados em um gabinete e os copia para o sistema do usuário.

Se arquivos de gabinete são usados, as funções de instalação usam o nome do gabinete como o arquivo de marca para o disco de origem em que ele reside.
