# <a name="about-cabinet-files"></a>Sobre arquivos de gabinete

Um gabinete é um arquivo único, geralmente com sufixo. CAB, que armazena arquivos compactados em uma biblioteca de arquivos. Um arquivo compactado pode ser distribuído por vários arquivos de gabinete. Durante a instalação, o aplicativo de instalação descompacta os arquivos armazenados em um gabinete e os copia para o sistema do usuário.

Se os arquivos de gabinete forem usados, as funções de instalação usarão o nome do gabinete como o arquivo de marca para o disco de origem em que reside.
