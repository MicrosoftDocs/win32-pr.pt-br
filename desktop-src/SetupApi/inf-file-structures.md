# <a name="inf-file-structures"></a>Estruturas de arquivo INF

As estruturas a seguir são usadas com arquivos INF.



| Estrutura                                              | Descrição                        |
|--------------------------------------------------------|------------------------------------|
| [**INFCONTEXT**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | Contexto posicional em um arquivo INF. |
| [**\_informações de inf do SP \_**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | Informações sobre o arquivo INF.    |



 

> [!Note]  
> A estrutura INFCONTEXT é usada internamente pelas funções de instalação e não deve ser alterada ou referenciada pelos aplicativos de instalação. Ele está incluído aqui apenas para fins informativos.
