# <a name="inf-file-structures"></a>Estruturas de arquivo INF

As estruturas a seguir são usadas com arquivos INF.



| Estrutura                                              | Descrição                        |
|--------------------------------------------------------|------------------------------------|
| [**INFCONTEXT**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | Contexto posicional em um arquivo INF. |
| [**INFORMAÇÕES \_ DO SP \_ INF**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | Informações sobre o arquivo INF.    |



 

> [!Note]  
> A estrutura INFCONTEXT é usada internamente pelas funções de instalação e não deve ser alterada ou referenciada por aplicativos de instalação. Ele está incluído aqui apenas para fins informacionais.
