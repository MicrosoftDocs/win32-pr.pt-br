# <a name="file-log-functions"></a>Funções de log de arquivo

As funções a seguir são usadas com logs de arquivo.



| Função                                                   | Descrição                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**SetupInitializeFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitializefileloga)   | Inicializa um arquivo de log para uso.             |
| [**SetupLogFile**](/windows/desktop/api/Setupapi/nf-setupapi-setuplogfilea)                       | Adiciona uma entrada ao arquivo de log.              |
| [**SetupQueryFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupqueryfileloga)             | Recupera informações de um arquivo de log.      |
| [**SetupRemoveFileLogEntry**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefilelogentrya) | Remove uma entrada de um arquivo de log.           |
| [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)     | Libera recursos alocados para um arquivo de log. |
