# <a name="releasing-other-system-resources"></a>Liberando outros recursos do sistema

Se uma lista de origem ou arquivo de log tiver sido usado, os recursos deverão ser liberados chamando [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) ou [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog).
