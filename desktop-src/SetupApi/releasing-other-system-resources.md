# <a name="releasing-other-system-resources"></a>Liberando outros recursos do sistema

Se uma lista de origem ou um arquivo de log foi usado, os recursos devem ser liberados chamando [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) ou [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog).
