# <a name="default-queue-callback-routine-functions"></a>Funções de rotina de retorno de chamada de fila padrão

As funções a seguir são usadas com a rotina de retorno de chamada de fila padrão.



| Função                                                                   | Descrição                                                                                                                                                                            |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SetupDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka)             | A rotina de retorno de chamada de fila padrão.                                                                                                                                                    |
| [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback)     | Inicializa as informações de contexto necessárias para a rotina de retorno de chamada de fila padrão.                                                                                                          |
| [**SetupInitDefaultQueueCallbackEx**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex) | Inicializa as informações de contexto necessárias para a rotina de retorno de chamada de fila padrão e especifica uma janela alternativa para exibir as mensagens de progresso.                                           |
| [**SetupTermDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setuptermdefaultqueuecallback)     | Libera recursos alocados por [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback) e [**SetupInitDefaultQueueCallbackEx**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex). |
