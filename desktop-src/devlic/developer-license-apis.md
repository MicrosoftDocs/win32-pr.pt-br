# <a name="developer-license-apis"></a>APIs de licença de desenvolvedor

Use essas APIs para verificar, adquirir e remover uma licença de desenvolvedor.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                 | Descrição                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**CheckDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | Verifica se existe uma licença de desenvolvedor.<br/> |
| [**AcquireDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | Adquire uma licença de desenvolvedor.<br/>                |
| [**RemoveDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | Remove uma licença de desenvolvedor.<br/>                 |
