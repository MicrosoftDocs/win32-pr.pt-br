# <a name="radial-controller-interfaces"></a>Interfaces do controlador radial

Os tópicos contidos nesta seção fornecem as especificações de referência para interfaces de [entrada do controlador radial](radialcontroller-portal.md) .

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|---|---|
| [**IRadialControllerConfigurationInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerconfigurationinterop)<br/> | Permite a interoperabilidade com Plataforma Universal do Windows um objeto [**RadialControllerConfiguration**](/uwp/api/Windows.UI.Input.RadialControllerConfiguration) (UWP) e fornece acesso a membros do **RadialControllerConfiguration** para personalizar um menu do [**RadialController**](/uwp/api/Windows.UI.Input.RadialController) .<br/> |
| [**IRadialControllerInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerinterop)<br/> | Permite a interoperabilidade com um objeto [**RADIALCONTROLLER**](/uwp/api/Windows.UI.Input.RadialController) UWP e fornece acesso a membros do **RadialController** para personalizar a experiência de interação.<br/> |
