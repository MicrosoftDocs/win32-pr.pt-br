# <a name="getdistinguishedname-method-of-the-microsoftdns_cache-class"></a>Método getdistinguiname da classe de \_ cache MicrosoftDNS

O método **Getdistinguiname** recupera o nome distinto da zona.

## <a name="syntax"></a>Sintaxe


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna o nome da zona.
