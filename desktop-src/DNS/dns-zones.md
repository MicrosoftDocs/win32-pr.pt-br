# <a name="dns-zones"></a>Zonas DNS

Uma zona DNS é um conjunto de arquivos ou registros (mais precisamente, um banco de dados de entradas de registro de recurso) que corresponde a parte do namespace hierárquico dns. As zonas DNS são usadas para delinear quais Servidores DNS são responsáveis (autoritativos) para resolver consultas de resolução de nomes para uma determinada seção da hierarquia DNS. As zonas DNS diferem da estrutura de domínio da seguinte maneira: as zonas podem ser compostas por um ou mais domínios DNS. Uma zona na árvore gadgets.widgets.microsoft.com domínio pode ser autoritativa para os domínios de widgets e widgets. Em outras palavras, não há um requisito para que as zonas DNS tenham uma relação um-para-um com domínios DNS.
