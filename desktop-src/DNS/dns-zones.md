# <a name="dns-zones"></a>Zonas DNS

Uma zona DNS é um conjunto de arquivos ou registros (mais precisamente, um banco de dados de entradas de registro de recurso) que corresponde a parte do namespace hierárquico do DNS. As zonas DNS são usadas para delinear quais servidores DNS são responsáveis (autoritativos) para resolver consultas de resolução de nomes para uma determinada seção da hierarquia de DNS. As zonas DNS diferem da estrutura de domínio do seguinte modo: as zonas podem ser compostas de um ou mais domínios DNS. Uma zona na árvore de domínio gadgets.widgets.microsoft.com pode ser autoritativa para os domínios de gadgets e widgets. Em outras palavras, não há um requisito para que as zonas DNS tenham uma relação um-para-um com domínios DNS.
