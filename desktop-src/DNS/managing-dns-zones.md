# <a name="managing-dns-zones"></a>Gerenciando Zonas DNS

Uma zona DNS é um banco de dados de entradas de RR que corresponde a parte do espaço de nome hierárquico DNS. As zonas DNS são usadas para delinear quais servidores DNS são autoritativos para resolver consultas de resolução de nomes para uma determinada seção da hierarquia de DNS.

## <a name="administration-steps"></a>Etapas de administração

O provedor WMI de DNS permite a administração de zonas DNS do próprio servidor DNS autoritativo ou de hosts remotos. As etapas gerais necessárias para administrar uma zona usando o provedor WMI de DNS são:

-   Conectando-se ao provedor WMI de DNS
-   Obtendo uma instância de zona
-   Listando ou modificando uma propriedade de zona ou usando um método
