# <a name="dns-concepts"></a>Conceitos de DNS

Há alguns conceitos básicos sobre os quais a estrutura funcional do DNS Opera. Esses conceitos básicos variam de definições de espaço de nome para demarcação de limites de responsabilidade de resolução de nomes. Entender esses conceitos é uma longa maneira de obter um sólido conhecimento funcional do DNS. Os seguintes conceitos de DNS são definidos e explicados nesta seção:

-   [Espaço de nome](name-space.md)
-   [Domínios DNS](dns-domains.md)
-   [Zonas DNS](dns-zones.md)
-   [Básica](root.md)
-   [Resolução de nomes](name-resolution.md)
-   [Servidores DNS](dns-servers.md)
-   [Resolvedores de DNS](dns-resolvers.md)
-   [Registros de recursos](resource-records.md)
