-   [Zonas DNS](dns-zones.md)
-   [Raiz](root.md)
-   [Resolução de nomes](name-resolution.md)
-   [Servidores DNS](dns-servers.md)
-   [Resolvedores dns](dns-resolvers.md)
-   [Registros de recursos](resource-records.md)
