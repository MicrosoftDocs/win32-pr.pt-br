# <a name="clearcache-method-of-the-microsoftdns_cache-class"></a>Método ClearCache da classe de \_ cache MicrosoftDNS

O método **ClearCache** limpa o cache do servidor DNS dos registros de recursos.

## <a name="syntax"></a>Sintaxe


```mof
void ClearCache();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
