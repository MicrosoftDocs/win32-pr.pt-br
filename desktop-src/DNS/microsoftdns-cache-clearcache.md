# <a name="clearcache-method-of-the-microsoftdns_cache-class"></a>Método ClearCache da classe Cache \_ MicrosoftDNS

O **método ClearCache** limpa o cache do Servidor DNS dos registros de recursos.

## <a name="syntax"></a>Sintaxe


```mof
void ClearCache();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
