# <a name="resumezone-method-of-the-microsoftdns_zone-class"></a>Método ResumeZone da classe de \_ zona MicrosoftDNS

O método **ResumeZone** retoma a zona DNS.

## <a name="syntax"></a>Sintaxe


```mof
void ResumeZone();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
