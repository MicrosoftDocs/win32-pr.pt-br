# <a name="root"></a>Root

O domínio superior em um espaço de nome hierárquico, como o espaço de nome DNS, é chamado de [*raiz*](r-gly.md). A raiz, que é definida como "" (Nothing) ou mais precisamente "." (um ponto), é a base na qual qualquer pesquisa de árvore de domínio inteira deve ser iniciada.
