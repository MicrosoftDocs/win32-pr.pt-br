## <a name="fully-qualified-domain-name"></a><span data-ttu-id="45e33-126">Nome do Domínio Totalmente Qualificado</span><span class="sxs-lookup"><span data-stu-id="45e33-126">Fully Qualified Domain Name</span></span>

<span data-ttu-id="45e33-127">O DNS é uma solução melhor; com seu espaço de nome hierárquico, a necessidade de nomes de computador exclusivos é isolada para um determinado domínio, permitindo que um nome de computador, como *Server1* , exista em diferentes locais de domínio na mesma hierarquia.</span><span class="sxs-lookup"><span data-stu-id="45e33-127">DNS is a better solution; with its hierarchical name space, the need for unique computer names is isolated to a given domain, enabling a computer name such as *server1* to exist in different domain locations in the same hierarchy.</span></span> <span data-ttu-id="45e33-128">Com a capacidade de ter o mesmo nome de host em domínios diferentes, a necessidade surgiu para um nome que resolveu corretamente a hierarquia de DNS.</span><span class="sxs-lookup"><span data-stu-id="45e33-128">With the capability to have the same host name in different domains, the necessity arose for a name that properly addressed the DNS hierarchy.</span></span> <span data-ttu-id="45e33-129">O nome tinha que incluir não apenas o nome do computador ou o nome do host, mas também um nome que poderia identificar, ou qualificar totalmente, esse computador dentro de toda a hierarquia DNS.</span><span class="sxs-lookup"><span data-stu-id="45e33-129">The name had to include not only the computer name or host name, but also a name that could unambiguously identify, or fully qualify, that computer within the entire DNS hierarchy.</span></span> <span data-ttu-id="45e33-130">Esse nome é o FQDN ( [*nome de domínio totalmente qualificado*](f-gly.md) ) — por exemplo, server1.widgets.Microsoft.com.</span><span class="sxs-lookup"><span data-stu-id="45e33-130">That name is the [*fully qualified domain name*](f-gly.md) (FQDN) — for example, server1.widgets.microsoft.com.</span></span>
