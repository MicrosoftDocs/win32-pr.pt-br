# <a name="about-the-dns-wmi-provider"></a>Sobre o provedor WMI de DNS

Os servidores DNS mapeiam endereços IP para nomes de computador e nomes de computador para endereços IP. Ao fazer isso, eles fornecem o mecanismo para localizar recursos de rede. O provedor WMI de DNS permite que os aplicativos interajam com servidores DNS por meio da estrutura de gerenciamento unificada do Instrumentação de Gerenciamento do Windows (WMI).
