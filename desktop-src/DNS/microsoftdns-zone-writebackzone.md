# <a name="writebackzone-method-of-the-microsoftdns_zone-class"></a>Método WriteBackZone da classe de \_ zona MicrosoftDNS

O método **WriteBackZone** salva dados de zona em seu arquivo de zona.

## <a name="syntax"></a>Sintaxe


```mof
void WriteBackZone();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Esse método não retorna um valor.
