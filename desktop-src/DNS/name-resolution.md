# <a name="name-resolution"></a>Resolução de nomes

A finalidade do DNS é fornecer resolução de nomes: o processo de comparar um nome de host (como um nome de computador) a uma lista de RRs e identificar o endereço IP correspondente. A resolução inversa de nomes é o processo de comparar um endereço IP com uma lista de RRs e identificar o nome do host correspondente. Esses processos de resolução são iniciados quando um servidor DNS recebe uma consulta.
