# <a name="name-resolution"></a>Resolução de nomes

A finalidade do DNS é fornecer a resolução de nomes: o processo de comparação de um nome de host (como um nome de computador) com uma lista de RRs e identificação do endereço IP correspondente. A resolução de nomes inversa é o processo de comparação de um endereço IP com uma lista de RRs e identificação do nome de host correspondente. Esses processos de resolução são iniciados quando um servidor DNS recebe uma consulta.
