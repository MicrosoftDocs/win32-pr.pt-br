# <a name="writebackroothintdatafile-method-of-the-microsoftdns_roothints-class"></a>Método WriteBackRootHintDatafile da classe RootHints do MicrosoftDNS \_

O **método WriteBackRootHintDatafile** grava o RootHints de volta no arquivo de Cache DNS.

## <a name="syntax"></a>Sintaxe


```mof
void WriteBackRootHintDatafile();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Esse método não retorna um valor.
