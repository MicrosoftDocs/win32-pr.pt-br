# <a name="dns-wmi-provider-scripting-examples"></a>Exemplos de script do provedor WMI de DNS

Esta seção fornece exemplos de script que demonstram como criar scripts de tarefas administrativas usando o provedor WMI de DNS.

Os seguintes tipos de exemplos de script são fornecidos.



| Seção                                                                                     | Exemplos fornecidos                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Gerenciando um servidor DNS](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Fornece exemplos que permitem o gerenciamento de um servidor DNS, incluindo parar, iniciar ou reiniciar um servidor, listar zonas, adicionar ou excluir endereços IP e outros.                |
| [Gerenciando registros de recursos de DNS](dns-wmi-provider-samples-managing-dns-resource-records.md) | Demonstra como se conectar ao provedor WMI de DNS e como executar várias tarefas associadas ao gerenciamento de registros de recursos — adicionar, excluir, modificar ou listar registros de recursos. |
| [Gerenciando Zonas DNS](dns-wmi-provider-samples-managing-dns-zones.md)                       | Os exemplos aqui incluem criar, modificar e excluir uma zona, bem como pausar, retomar, atualizar e outras tarefas de zona, que são causadas por administradores de DNS.                                  |
