Esta seção fornece exemplos de script que demonstram como script de tarefas administrativas usando o Provedor WMI dns.

Os seguintes tipos de exemplos de script são fornecidos.



| Seção                                                                                     | Exemplos fornecidos                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Gerenciando um servidor DNS](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Fornece exemplos que permitem o gerenciamento de um servidor DNS, incluindo parar, iniciar ou reiniciar um servidor, listar zonas, adicionar ou excluir endereços IP e outros.                |
| [Gerenciando registros de recursos DNS](dns-wmi-provider-samples-managing-dns-resource-records.md) | Demonstra como se conectar ao Provedor WMI dns e como executar várias tarefas associadas ao gerenciamento de registros de recursos – adicionando, excluindo, modificando ou listando registros de recursos. |
| [Gerenciando zonas DNS](dns-wmi-provider-samples-managing-dns-zones.md)                       | Os exemplos aqui incluem criar, modificar e excluir uma zona, bem como pausar, retomar, atualizar e outras tarefas de zona colocadas aos administradores de DNS.                                  |
