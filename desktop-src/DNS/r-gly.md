# <a name="r-dns"></a>R (DNS)

<dl> <dt>

<span id="_dns_relative_distinguished_name_gly"></span><span id="_DNS_RELATIVE_DISTINGUISHED_NAME_GLY"></span>**nome distinto relativo**
</dt> <dd>

Nome que identifica exclusivamente um host em seu próprio domínio, mas não em toda a hierarquia DNS. Por exemplo, server1 é um nome distinto relativo, enquanto server1.products.Microsoft.com é um FQDN (nome de domínio totalmente qualificado).

</dd> <dt>

<span id="_dns_resolver_gly"></span><span id="_DNS_RESOLVER_GLY"></span>**soluciona**
</dt> <dd>

Um componente de software incluído na pilha TCP/IP (protocolo de controle de transmissão) do Windows que se comunica com servidores DNS para criar e resolver consultas de resolução de nomes.

</dd> <dt>

<span id="_dns_resource_record_rr__gly"></span><span id="_DNS_RESOURCE_RECORD_RR__GLY"></span>**registro de recurso (RR)**
</dt> <dd>

A unidade de informações no DNS. Há muitos tipos de registros de recursos.

</dd> <dt>

<span id="_dns_root_gly"></span><span id="_DNS_ROOT_GLY"></span>**básica**
</dt> <dd>

O nível superior em uma determinada hierarquia.

</dd> </dl>
