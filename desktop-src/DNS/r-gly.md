# <a name="r-dns"></a>R (DNS)

A B [C](c-gly.md) D E [F G](f-gly.md) [H](h-gly.md) I J K [L M](l-gly.md) N O [P](p-gly.md) Q R [S](s-gly.md) T U V W X Y [Z](z-gly.md)

<dl> <dt>

<span id="_dns_relative_distinguished_name_gly"></span><span id="_DNS_RELATIVE_DISTINGUISHED_NAME_GLY"></span>**nome diferenciado relativo**
</dt> <dd>

Nome que identifica exclusivamente um host dentro de seu próprio domínio, mas não em toda a hierarquia DNS. Por exemplo, server1 é um nome diferenciado relativo, enquanto server1.products.Microsoft.com é um FQDN (nome de domínio totalmente qualificado).

</dd> <dt>

<span id="_dns_resolver_gly"></span><span id="_DNS_RESOLVER_GLY"></span>**Resolvedor**
</dt> <dd>

Um componente de software agrupado na pilha TCP/IP (protocolo TCP/IP) do Windows que se comunica com servidores DNS para criar e resolver consultas de resolução de nomes.

</dd> <dt>

<span id="_dns_resource_record_rr__gly"></span><span id="_DNS_RESOURCE_RECORD_RR__GLY"></span>**registro de recurso (RR)**
</dt> <dd>

A unidade de informações no DNS. Há muitos tipos de registros de recursos.

</dd> <dt>

<span id="_dns_root_gly"></span><span id="_DNS_ROOT_GLY"></span>**Raiz**
</dt> <dd>

O nível mais alto em uma determinada hierarquia.

</dd> </dl>
