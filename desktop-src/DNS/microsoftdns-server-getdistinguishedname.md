# <a name="getdistinguishedname-method-of-the-microsoftdns_server-class"></a>Método getdistinguiname da classe de \_ servidor MicrosoftDNS

O método **Getdistinguiname** recupera o nome distinto DS para a zona.

## <a name="syntax"></a>Sintaxe


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Retorna o nome distinto do DS para a zona.
