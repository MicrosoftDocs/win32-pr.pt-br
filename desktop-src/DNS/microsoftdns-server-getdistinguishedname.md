# <a name="getdistinguishedname-method-of-the-microsoftdns_server-class"></a>Método GetDistinguishedName da classe MicrosoftDNS \_ Server

O **método GetDistinguishedName** recupera o nome diferenciado de DS para a zona.

## <a name="syntax"></a>Sintaxe


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Retorna o nome diferenciado de DS para a zona.
