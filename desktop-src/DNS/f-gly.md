# <a name="f-dns"></a>F (DNS)

A B [C](c-gly.md) D E F G [H](h-gly.md) i J K [L](l-gly.md) M N O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T U V W X Y [Z](z-gly.md)

<dl> <dt>

<span id="_dns_fault_tolerance_gly"></span><span id="_DNS_FAULT_TOLERANCE_GLY"></span>**tolerância a falhas**
</dt> <dd>

Método de garantir que um sistema de rede ou de computador seja resistente a falhas de software ou hardware.

</dd> <dt>

<span id="_dns_flat_name_space_gly"></span><span id="_DNS_FLAT_NAME_SPACE_GLY"></span>**espaço de nome simples**
</dt> <dd>

Um espaço de nome não particionado, no qual cada entrada deve ser exclusiva. Os espaços de nome simples não são bem dimensionados, devido ao requisito de que cada entrada seja exclusiva.

</dd> <dt>

<span id="_dns_forwarder_gly"></span><span id="_DNS_FORWARDER_GLY"></span>**encaminhador**
</dt> <dd>

Um servidor DNS que foi designado para lidar com a comunicação com servidores DNS externos.

</dd> <dt>

<span id="_dns_fully_qualified_domain_name_fqdn__gly"></span><span id="_DNS_FULLY_QUALIFIED_DOMAIN_NAME_FQDN__GLY"></span>**FQDN (nome de domínio totalmente qualificado)**
</dt> <dd>

Um nome que identifica exclusivamente um host na hierarquia de DNS, de modo que um host chamado server1 na hierarquia de produtos da Microsoft possa ter um FQDN de server1.products.Microsoft.com.

</dd> </dl>
