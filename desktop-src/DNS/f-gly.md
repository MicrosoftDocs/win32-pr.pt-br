# <a name="f-dns"></a>F (DNS)

A B [C](c-gly.md) D E F G [H](h-gly.md) I J K [L M](l-gly.md) N O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T U V W X Y [Z](z-gly.md)

<dl> <dt>

<span id="_dns_fault_tolerance_gly"></span><span id="_DNS_FAULT_TOLERANCE_GLY"></span>**tolerância a falhas**
</dt> <dd>

Método de garantir que um sistema de rede ou computador seja resistente a falhas de software ou hardware.

</dd> <dt>

<span id="_dns_flat_name_space_gly"></span><span id="_DNS_FLAT_NAME_SPACE_GLY"></span>**espaço de nome simples**
</dt> <dd>

Um espaço de nome não partição, no qual cada entrada deve ser exclusiva. Os espaços de nome simples não são bem escalados, devido ao requisito de que cada entrada seja exclusiva.

</dd> <dt>

<span id="_dns_forwarder_gly"></span><span id="_DNS_FORWARDER_GLY"></span>**Encaminhador**
</dt> <dd>

Um servidor DNS que foi designado para lidar com a comunicação com servidores DNS externos.

</dd> <dt>

<span id="_dns_fully_qualified_domain_name_fqdn__gly"></span><span id="_DNS_FULLY_QUALIFIED_DOMAIN_NAME_FQDN__GLY"></span>**FQDN (Nome de Domínio Totalmente Qualificado)**
</dt> <dd>

Um nome que identifica exclusivamente um host na hierarquia DNS, de forma que um host chamado server1 na hierarquia de produtos da Microsoft pode ter um FQDN de server1.products.Microsoft.com.

</dd> </dl>
