# <a name="using-the-dns-wmi-provider"></a>Usando o provedor WMI de DNS

O provedor WMI de DNS define as [*classes CIM*](c-gly.md) do WMI que permitem que seu aplicativo controle um servidor DNS. As seções a seguir fornecem informações de visão geral que podem ajudá-lo a começar a usar o provedor WMI de DNS. São fornecidos links para scripts de exemplo que demonstram como fazer uso do provedor WMI de DNS.

Para obter mais informações sobre as operações básicas de WMI, consulte Instrumentação de Gerenciamento do Windows (WMI) no SDK (Software Development Kit) da plataforma.
