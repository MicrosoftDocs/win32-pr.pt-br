# <a name="dns-standards-documents"></a>Documentos de padrões DNS

O DNS (Sistema de Nomes de Domínio) é um protocolo aberto. Dessa forma, os esforços colaborativos do setor como um todo garantem que sua implementação em vários sistemas não limite a interoperabilidade. O corpo de padrões que supervisiona essas recomendações é o IETF (Internet Engineering Task Force). Veja a seguir solicitações de IETF para comentários (RFC). Para obter mais informações sobre qualquer um desses documentos, [visite www.ietf.org](https://www.ietf.org/).
