# <a name="dns-standards-documents"></a>Documentos de padrões de DNS

O DNS (sistema de nomes de domínio) é um protocolo aberto. Como tal, os esforços colaborativos do setor como um todo garantem que sua implementação em vários sistemas não limite a interoperabilidade. O corpo dos padrões que supervisiona essas recomendações é a IETF (Internet Engineering Task Force). Veja a seguir as solicitações IETF para comentários (RFC). Para obter mais informações sobre qualquer um desses documentos, visite [www.ietf.org](https://www.ietf.org/).

## <a name="dns-rfcs"></a>RFCs do DNS

-   [RFC 1034](https://www.ietf.org/rfc/rfc1034.txt): nomes de domínio – conceitos e instalações
-   [RFC 1035](https://www.ietf.org/rfc/rfc1035.txt): nomes de domínio – implementação e especificação
-   [RFC 1122](https://www.ietf.org/rfc/rfc1122.txt): requisitos para hosts de Internet-camadas de comunicação
-   [RFC 1123](https://www.ietf.org/rfc/rfc1123.txt): requisitos para hosts de Internet-aplicativo e suporte
-   [RFC 1876](https://www.ietf.org/rfc/rfc1876.txt): um meio para expressar informações de local no sistema de nomes de domínio
-   [RFC 1995](https://www.ietf.org/rfc/rfc1995.txt): transferência de zona incremental no DNS
-   [RFC 1996](https://www.ietf.org/rfc/rfc1996.txt): um mecanismo para notificação de prompt de alterações de zona (notificação DNS)
-   [RFC 2136](https://www.ietf.org/rfc/rfc2136.txt): atualizações dinâmicas no sistema de nomes de domínio (atualização de DNS)
-   [RFC 2181](https://www.ietf.org/rfc/rfc2181.txt): esclarecimentos para a especificação de DNS
-   [RFC 2308](https://www.ietf.org/rfc/rfc2308.txt): cache negativo de consultas DNS (DNS NCache)
-   [RFC 2535](https://www.ietf.org/rfc/rfc2535.txt): extensões de segurança do sistema de nome de domínio
-   [RFC 2782](https://www.ietf.org/rfc/rfc2782.txt): um RR DNS para especificar o local dos serviços (DNS SRV)
-   [RFC 2845](https://www.ietf.org/rfc/rfc2845.txt): autenticação de transação de chave secreta para DNS (TSIG)
-   [RFC 2915](https://www.ietf.org/rfc/rfc2915.txt): o registro de recurso DNS NAPTR (ponteiro de autoridade de nomenclatura)
-   [RFC 2930](https://www.ietf.org/rfc/rfc2930.txt): estabelecimento de chave secreta para DNS (TKey RR)
-   [RFC 2931](https://www.ietf.org/rfc/rfc2931.txt): assinaturas de solicitação e de transação DNS (SIG (0) s)
-   [RFC 3110](https://www.ietf.org/rfc/rfc3110.txt): RSA/SHA-1 Sigs e chaves RSA no DNS (sistema de nomes de domínio)
-   [RFC 3445](https://www.ietf.org/rfc/rfc3445.txt): limitando o escopo do registro de recurso de chave (RR)
-   [RFC 3596](https://www.ietf.org/rfc/rfc3596.txt): extensões DNS para dar suporte ao IP versão 6
-   [RFC 3645](https://www.ietf.org/rfc/rfc3645.txt): algoritmo de serviço de segurança genérico para autenticação de transação de chave secreta para DNS (GSS-TSIG)
