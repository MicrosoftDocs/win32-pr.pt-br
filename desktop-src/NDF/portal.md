# <a name="network-diagnostics-framework"></a>Estrutura de diagnóstico de rede

## <a name="purpose"></a>Finalidade

O NDF (estrutura de diagnóstico de rede) fornece uma maneira para que os desenvolvedores de componentes e aplicativos simplifiquem a solução de problemas de rede para os usuários. Os usuários podem tentar diagnosticar e reparar um problema de rede usando uma única ferramenta de solução de problemas.

A Microsoft fornece classes auxiliares NDF, algumas das quais são extensíveis para que os desenvolvedores possam criar unidades de solução de problemas chamadas de extensões de classe auxiliar para fornecer diagnósticos mais detalhados específicos para determinados componentes de software ou hardware.

## <a name="where-applicable"></a>Quando aplicável

O NDF pode ser usado por qualquer software de fornecedor que dependa da conectividade de rede.

## <a name="developer-audience"></a>Público de desenvolvedores

A API NDF foi projetada para desenvolvedores de C/C++.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

O NDF está disponível para computadores que executam o Windows Vista, o Windows Server 2008 ou posterior.
