# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>Ferramentas para solução de problemas usando o rastreamento de rede no Windows 7

Há várias ferramentas que podem ser usadas para ajudar a habilitar e gerenciar o rastreamento de rede e exibir os dados resultantes. Algumas das ferramentas que podem ser úteis são Netsh.exe, Monitor de Rede, Visualizador de Eventos, **netsh trace Convert**, Tracerpt.exe e Xperf.exe.

Nesta seção, veremos como usar Netsh.exe e Monitor de Rede para ajudar na solução de problemas de rastreamento de rede.
