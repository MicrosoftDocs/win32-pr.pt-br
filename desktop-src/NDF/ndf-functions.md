# <a name="ndf-functions"></a>Funções NDF

As funções a seguir permitem que os desenvolvedores de software usem a funcionalidade fornecida pela estrutura de diagnóstico de rede (NDF). Quando ocorrem problemas de rede, o NDF pode diagnosticar a causa raiz e lidar com a resolução necessária normalmente, às vezes até mesmo executando os reparos necessários.

> [!Note]  
> Essas funções são para os desenvolvedores que implementam a funcionalidade NDF básica em seus aplicativos.
