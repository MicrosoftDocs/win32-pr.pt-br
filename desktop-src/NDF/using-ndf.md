# <a name="using-ndf"></a>Usando NDF

O NDF (estrutura de diagnóstico de rede) é fornecido como parte do Windows Vista, do Windows Server 2008 e posterior. O NDF pode ser usado por desenvolvedores de software para diagnosticar e reparar problemas de rede que podem estar causando falhas nos aplicativos. Os desenvolvedores podem usar APIs fornecidas pela Microsoft que conectam seus aplicativos ao NDF. Em alguns casos, os desenvolvedores também podem criar extensões de classe auxiliar NDF para ajudar a diagnosticar problemas específicos de seu software de rede.
