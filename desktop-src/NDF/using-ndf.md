# <a name="using-ndf"></a>Usando NDF

O Estrutura de Diagnóstico de Rede (NDF) é Windows vista, Windows Server 2008 e posterior. O NDF pode ser usado por desenvolvedores de software para diagnosticar e reparar problemas de rede que podem estar fazendo com que seus aplicativos falhem. Os desenvolvedores podem usar APIs fornecidas pela Microsoft que conectam seus aplicativos ao NDF. Em alguns casos, os desenvolvedores também podem criar extensões de classe auxiliar do NDF para ajudar a diagnosticar problemas específicos de seu software de rede.
