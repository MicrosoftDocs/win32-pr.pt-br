# <a name="ndf-helper-class-extension-examples"></a>Exemplos de extensão de classe auxiliar do NDF

Esses exemplos se aplicam somente quando os desenvolvedores acreditarem que é necessário estender a funcionalidade de uma Classe Auxiliar NDF da Microsoft que é declarada como extensível. Isso geralmente não é necessário, mas, em alguns casos, oportunidades de diagnóstico exclusivas se apresentam devido aos requisitos específicos de hardware ou software do componente.

Os dois exemplos a seguir estão relacionados e o primeiro deve ser compreendido antes de analisar o segundo. O primeiro fornece uma implementação de uma Extensão [de Classe Auxiliar NDF](ndf-helper-class-example.md) usando uma classe chamada SimpleHelperFileClass.

A segunda, Extensão de Classe Auxiliar [NDF](ndf-helper-class-example-with-handoff.md)com Entrega , fornece um exemplo de uma extensão que não executa tarefas de diagnóstico específicas, mas gera apenas uma hipótese de baixa saúde para SimpleHelperFileClass no primeiro exemplo.
