# <a name="process-status-api"></a>API de status do processo

A interface de programação de aplicativo de status do processo (PSAPI) é uma biblioteca auxiliar que torna mais fácil obter informações sobre processos e drivers de dispositivo. Para mais informações, consulte os seguintes tópicos:

-   [Sobre o PSAPI](about-psapi.md)
-   [Usando PSAPI](using-psapi.md)
-   [Referência de PSAPI](psapi-reference.md)

Essas funções estão disponíveis em Psapi.dll.

As mesmas informações estão geralmente disponíveis por meio dos dados de desempenho no registro. Para obter mais informações, consulte [Performance Counters](/windows/desktop/PerfCtrs/performance-counters-portal).
