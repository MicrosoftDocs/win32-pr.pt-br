# <a name="flagname-debugdatatype-element"></a>Elemento flagName (DebugDataType)

O valor do sinalizador passado ao provedor quando ele foi habilitado.

``` syntax
<xs:element name="FlagName"
    type="string"
 />
```

O elemento **flagName** é definido pelo tipo complexo [**DebugDataType**](eventschema-debugdatatype-complextype.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do vista\]<br/>       |
| Servidor mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do servidor 2008\]<br/> |
