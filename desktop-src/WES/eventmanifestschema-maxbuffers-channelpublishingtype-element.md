# <a name="maxbuffers-channelpublishingtype-element"></a>Elemento maxBuffers (ChannelPublishingType)

O número máximo de buffers a serem alocados para a sessão.

``` syntax
<xs:element name="maxBuffers"
    type="unsignedInt"
 />
```

O elemento **maxBuffers** é definido pelo tipo complexo [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
