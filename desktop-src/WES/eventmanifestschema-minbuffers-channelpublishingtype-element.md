O número mínimo de buffers a serem alocados para a sessão.

``` syntax

    type="unsignedInt"
 />
```

O elemento **minBuffers** é definido pelo tipo complexo [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
