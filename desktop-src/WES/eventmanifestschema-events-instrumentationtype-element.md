# <a name="events-instrumentationtype-element"></a>Elemento Events (InstrumentationType)

Contém uma lista de provedores que são definidos no manifesto.

``` syntax
<xs:element name="events"
    type="EventsType"
 />
```

O elemento **Events** é definido pelo tipo complexo [**InstrumentationType**](eventmanifestschema-instrumentationtype-complextype.md) .
