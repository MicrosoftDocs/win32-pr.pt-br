# <a name="controlguid-channelpublishingtype-element"></a>Elemento controlGuid (ChannelPublishingType)

Identifica o GUID de sessão para uma sessão de ETW que contém eventos de WPP.

``` syntax
<xs:element name="controlGuid"
    type="GUIDType"
 />
```

O elemento **controlGuid** é definido pelo tipo complexo [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
