# <a name="xmltypes-typelisttype-element"></a>Elemento xmlTypes (TypeListType)

Define uma lista de tipos de saída que o serviço usa para determinar como renderizar um tipo de dados de entrada.

``` syntax
<xs:element name="xmlTypes"
    type="XmlTypeListType"
 />
```

O **elemento xmlTypes** é definido pelo tipo complexo [**TypeListType.**](eventmanifestschema-typelisttype-complextype.md)
