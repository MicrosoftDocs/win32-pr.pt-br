# <a name="localization-instrumentationmanifest-element"></a>Elemento de localização (instrumentationManifest)

Define um grupo de recursos localizados que você referencia em seu manifesto.

``` syntax
<xs:element name="localization"
    type="LocalizationType"
 />
```

O elemento de **localização** é definido pelo elemento [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md) .
