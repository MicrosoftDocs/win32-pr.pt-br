# <a name="keywords-providertype-element"></a>Elemento Keywords (ProviderType)

Define uma lista de palavras-chave que categorizam eventos.

``` syntax
<xs:element name="keywords"
    type="KeywordListType"
 />
```

O elemento **Keywords** é definido pelo tipo complexo [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
