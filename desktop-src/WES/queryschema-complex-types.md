# <a name="query-schema-complex-types"></a>Tipos complexos de esquema de consulta

A seguir estão os tipos complexos que o esquema de consulta define.



| Tipo complexo                                                   | Descrição                                                                                                                                                         |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**QueryListType**](queryschema-querylisttype-complextype.md) | Define uma lista de consultas que podem conter um conjunto de seletores e supressers que são usados para incluir eventos em ou excluir eventos do conjunto de resultados. <br/> |
| [**QueryType**](queryschema-querytype-complextype.md)         | Define um conjunto de consultas de seletor e supressor que são usadas para incluir eventos no ou excluir eventos do conjunto de resultados.<br/>                               |
