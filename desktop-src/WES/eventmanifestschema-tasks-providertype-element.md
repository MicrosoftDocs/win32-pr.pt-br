# <a name="tasks-providertype-element"></a>Elemento Tasks (ProviderType)

Define uma lista de tarefas que um provedor pode usar para agrupar eventos.

``` syntax
<xs:element name="tasks"
    type="TaskListType"
 />
```

O elemento **Tasks** é definido pelo tipo complexo [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
