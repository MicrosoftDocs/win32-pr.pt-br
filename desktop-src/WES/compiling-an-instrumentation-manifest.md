# <a name="compiling-an-instrumentation-manifest"></a>Compilando um manifesto de instrumentação

Depois de [gravar o manifesto](writing-an-instrumentation-manifest.md), use o compilador de mensagem para validar o manifesto e gerar os arquivos de recurso e de cabeçalho que você inclui em seu provedor. Para obter detalhes sobre como usar o compilador, consulte [**compilador de mensagem**](message-compiler--mc-exe-.md). Inclua os arquivos de cabeçalho e de recurso que o compilador gera no projeto que contém o restante do código-fonte do provedor. Para obter informações sobre como gravar os eventos que são definidos em seu manifesto, consulte [desenvolvendo um provedor](developing-a-provider.md).
