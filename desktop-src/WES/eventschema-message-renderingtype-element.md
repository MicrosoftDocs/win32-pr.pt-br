# <a name="message-renderinginfotype-element"></a>Elemento Message (RenderingInfoType)

Contém a mensagem de evento que é renderizada para o evento.

``` syntax
<xs:element name="Message"
    type="string"
 />
```

O elemento **Message** é definido pelo tipo complexo [**RenderingInfoType**](eventschema-renderingtype-complextype.md) .
