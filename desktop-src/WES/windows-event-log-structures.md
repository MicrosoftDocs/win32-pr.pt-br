# <a name="windows-event-log-structures"></a>Windows Estruturas de log de eventos

Windows O log de eventos define as estruturas a seguir.



| Estrutura                                | Descrição                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**\_logon de RPC EVT \_**](/windows/desktop/api/WinEvt/ns-winevt-evt_rpc_login) | Contém as informações usadas para se conectar a um computador remoto. |
| [**variante de EVT \_**](/windows/desktop/api/WinEvt/ns-winevt-evt_variant)      | Contém dados de evento ou valores de propriedade.                        |
