# <a name="templates-providertype-element"></a>Elemento templates (ProviderType)

Define uma lista de modelos que especificam os dados a serem incluídos com os eventos.

``` syntax
<xs:element name="templates"
    type="TemplateListType"
 />
```

O elemento **templates** é definido pelo tipo complexo [**ProviderType**](eventmanifestschema-providertype-complextype.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Vista\]<br/>       |
| Servidor mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Server 2008\]<br/> |
