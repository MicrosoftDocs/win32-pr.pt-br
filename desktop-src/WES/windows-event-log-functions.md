# <a name="windows-event-log-functions"></a>Funções de log de eventos do Windows

O log de eventos do Windows define as seguintes funções que você pode usar para obter eventos de um canal ou log de eventos e obter os metadados para um provedor e os eventos que ele gera.



| Função                                                                   | Descrição                                                                                                                                                                                                       |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**retorno de chamada de \_ assinatura EVT \_**](/windows/win32/api/winevt/nc-winevt-evt_subscribe_callback)                 | Implemente esse retorno de chamada se você chamar a função [**EvtSubscribe**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe) para receber eventos que correspondam à sua consulta.                                                                                    |
| [**EvtArchiveExportedLog**](/windows/desktop/api/WinEvt/nf-winevt-evtarchiveexportedlog)                     | Adiciona cadeias de caracteres localizadas aos eventos no arquivo de log especificado.                                                                                                                                                   |
| [**EvtCancel**](/windows/desktop/api/WinEvt/nf-winevt-evtcancel)                                             | Cancela todas as operações pendentes em um identificador.                                                                                                                                                                       |
| [**EvtClearLog**](/windows/desktop/api/WinEvt/nf-winevt-evtclearlog)                                         | Remove todos os eventos do canal especificado e os grava no arquivo de log de destino.                                                                                                                             |
| [**EvtClose**](/windows/desktop/api/WinEvt/nf-winevt-evtclose)                                               | Fecha um identificador aberto.                                                                                                                                                                                            |
| [**EvtCreateBookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtcreatebookmark)                             | Cria um indicador que identifica um evento em um canal.                                                                                                                                                         |
| [**EvtCreateRenderContext**](/windows/desktop/api/WinEvt/nf-winevt-evtcreaterendercontext)                   | Cria um contexto que especifica as informações no evento que você deseja renderizar.                                                                                                                            |
| [**EvtExportLog**](/windows/desktop/api/WinEvt/nf-winevt-evtexportlog)                                       | Copia eventos do canal ou arquivo de log especificado e os grava no arquivo de log de destino.                                                                                                                      |
| [**EvtFormatMessage**](/windows/desktop/api/WinEvt/nf-winevt-evtformatmessage)                               | Formata uma cadeia de caracteres de mensagem.                                                                                                                                                                                         |
| [**EvtGetChannelConfigProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetchannelconfigproperty)         | Obtém a propriedade de configuração do canal especificado.                                                                                                                                                                |
| [**EvtGetEventInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventinfo)                                 | Obtém informações que identificam a consulta XML estruturada que selecionou o evento e o canal ou arquivo de log do qual ele veio.                                                                                 |
| [**EvtGetEventMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventmetadataproperty)         | Obtém a propriedade de metadados de evento especificada.                                                                                                                                                                       |
| [**EvtGetExtendedStatus**](/windows/desktop/api/WinEvt/nf-winevt-evtgetextendedstatus)                       | Obtém uma mensagem de texto que contém as informações de erro estendidas para o erro atual.                                                                                                                           |
| [**EvtGetLogInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetloginfo)                                     | Obtém informações sobre um canal ou arquivo de log.                                                                                                                                                                     |
| [**EvtGetObjectArrayProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarrayproperty)             | Obtém uma propriedade de metadados de provedor do objeto especificado na matriz.                                                                                                                                         |
| [**EvtGetObjectArraySize**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarraysize)                     | Obtém o número de elementos na matriz de objetos.                                                                                                                                                              |
| [**EvtGetPublisherMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetpublishermetadataproperty) | Obtém a propriedade de metadados do provedor especificado.                                                                                                                                                                    |
| [**EvtGetQueryInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetqueryinfo)                                 | Obtém informações sobre uma consulta que você executou que identifica a lista de canais ou arquivos de log que a consulta tentou acessar e uma lista de códigos de retorno que indicam o êxito ou a falha de cada acesso. |
| [**EvtNext**](/windows/desktop/api/WinEvt/nf-winevt-evtnext)                                                 | Obtém o próximo evento dos resultados da consulta ou da assinatura.                                                                                                                                                       |
| [**EvtNextChannelPath**](/windows/desktop/api/WinEvt/nf-winevt-evtnextchannelpath)                           | Obtém um nome de canal do enumerador.                                                                                                                                                                          |
| [**EvtNextEventMetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtnexteventmetadata)                       | Obtém uma definição de evento do enumerador.                                                                                                                                                                     |
| [**EvtNextPublisherId**](/windows/desktop/api/WinEvt/nf-winevt-evtnextpublisherid)                           | Obtém o identificador de um provedor do enumerador.                                                                                                                                                            |
| [**EvtOpenChannelConfig**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelconfig)                       | Obtém um identificador que você usa para ler ou modificar a propriedade de configuração de um canal.                                                                                                                                  |
| [**EvtOpenChannelEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelenum)                           | Obtém um identificador que você usa para enumerar a lista de canais que estão registrados no computador.                                                                                                                 |
| [**EvtOpenEventMetadataEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopeneventmetadataenum)               | Obtém um identificador que você usa para enumerar a lista de eventos que o provedor define.                                                                                                                             |
| [**EvtOpenLog**](/windows/desktop/api/WinEvt/nf-winevt-evtopenlog)                                           | Obtém um identificador para um canal ou arquivo de log que você pode usar para obter informações sobre o canal ou arquivo de log.                                                                                                    |
| [**EvtOpenPublisherEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublisherenum)                       | Obtém um identificador que você usa para enumerar a lista de provedores registrados no computador.                                                                                                                         |
| [**EvtOpenPublisherMetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublishermetadata)               | Obtém um identificador que você usa para ler os metadados do provedor especificado.                                                                                                                                             |
| [**EvtOpenSession**](/windows/desktop/api/WinEvt/nf-winevt-evtopensession)                                   | Estabelece uma conexão com um computador remoto que você pode usar ao chamar as outras funções de log de eventos do Windows.                                                                                                |
| [**EvtQuery**](/windows/desktop/api/WinEvt/nf-winevt-evtquery)                                               | Executa uma consulta para recuperar eventos de um canal ou arquivo de log que correspondem aos critérios de consulta especificados.                                                                                                               |
| [**EvtRender**](/windows/desktop/api/WinEvt/nf-winevt-evtrender)                                             | Renderiza um fragmento XML com base no contexto de renderização que você especificar.                                                                                                                                          |
| [**EvtSaveChannelConfig**](/windows/desktop/api/WinEvt/nf-winevt-evtsavechannelconfig)                       | Salva as alterações feitas na configuração de um canal.                                                                                                                                                              |
| [**EvtSeek**](/windows/desktop/api/WinEvt/nf-winevt-evtseek)                                                 | Busca um evento específico em um conjunto de resultados da consulta.                                                                                                                                                                  |
| [**EvtSetChannelConfigProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtsetchannelconfigproperty)         | Define a propriedade de configuração especificada de um canal.                                                                                                                                                           |
| [**EvtSubscribe**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe)                                       | Cria uma assinatura que receberá eventos atuais e futuros de um canal ou arquivo de log que correspondam aos critérios de consulta especificados.                                                                            |
| [**EvtUpdateBookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtupdatebookmark)                             | Atualiza o indicador com informações que identificam o evento especificado.                                                                                                                                        |
