# <a name="keyword-keywordlisttype-element"></a>elemento Keyword ()

Define uma palavra-chave que identifica uma categoria de eventos. Uma palavra-chave é uma marca que você anexa a um evento para agrupar eventos com base em seu uso.

``` syntax
<xs:element name="keyword"
    type="KeywordType"
 />
```

O elemento **keyword** é definido pelo tipo [**complexo**](eventmanifestschema-keywordlisttype-complextype.md) .
