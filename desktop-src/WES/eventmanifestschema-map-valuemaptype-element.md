# <a name="map-valuemaptype-element"></a>Elemento Map (ValueMapType)

Define o mapeamento entre um valor inteiro e um valor de cadeia de caracteres.

``` syntax
<xs:element name="map"
    type="ValueMapValueType"
 />
```

O elemento **MAP** é definido pelo tipo complexo [**ValueMapType**](eventmanifestschema-valuemaptype-complextype.md) .
