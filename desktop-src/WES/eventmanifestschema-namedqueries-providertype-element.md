# <a name="namedqueries-providertype-element"></a>Elemento namedQueries (ProviderType)

Não usado. Define uma lista de consultas nomeadas que consultam a cadeia de caracteres de mensagem de evento para um valor e executam uma ação especificada, se encontradas.

``` syntax
<xs:element name="namedQueries"
    type="NamedQueryType"
 />
```

O elemento **namedQueries** é definido pelo tipo complexo [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
