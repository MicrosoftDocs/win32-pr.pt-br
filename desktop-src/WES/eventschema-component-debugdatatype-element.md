# <a name="component-debugdatatype-element"></a>Elemento Component (DebugDataType)

O nome do componente que registrou a mensagem de rastreamento.

``` syntax
<xs:element name="Component"
    type="string"
 />
```

O elemento **Component** é definido pelo tipo complexo [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
