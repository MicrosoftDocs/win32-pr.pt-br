# <a name="opcodes-metadatatype-element"></a>Elemento opcodes (MetadataType)

Define uma lista de opcodes que você pode usar para agrupar eventos dentro de uma tarefa.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

O elemento **opcodes** é definido pelo tipo complexo [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) .
