# <a name="levels-providertype-element"></a>Elemento Levels (ProviderType)

Define uma lista de níveis que especificam a severidade de um evento.

``` syntax
<xs:element name="levels"
    type="LevelListType"
 />
```

O elemento **Levels** é definido pelo tipo complexo [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
