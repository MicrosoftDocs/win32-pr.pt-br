# <a name="map-patternmaptype-element"></a>Elemento Map (PatternMapType)

Define as expressões regulares usadas para localizar uma cadeia de caracteres correspondente na cadeia de caracteres da mensagem e alterá-la.

``` syntax
<xs:element name="map"
    type="PatternMapValueType"
 />
```

O elemento **MAP** é definido pelo tipo complexo [**PatternMapType**](eventmanifestschema-patternmaptype-complextype.md) .
