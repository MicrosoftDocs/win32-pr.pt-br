# <a name="namedqueries-metadatatype-element"></a>Elemento namedQueries (MetadataType)

Não usado. Define uma lista de consultas nomeadas que consultam a cadeia de caracteres de mensagem de evento para um valor e executam uma ação especificada, se encontradas.

``` syntax
<xs:element name="namedQueries"
    type="NamedQueryType"
 />
```

O elemento **namedQueries** é definido pelo tipo complexo [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) .
