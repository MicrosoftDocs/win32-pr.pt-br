# <a name="sequencenumber-debugdatatype-element"></a>Elemento SequenceNumber (DebugDataType)

O número de sequência local ou global da mensagem de rastreamento.

``` syntax
<xs:element name="SequenceNumber"
    type="unsignedInt"
    minOccurs="0"
 />
```

O elemento **SequenceNumber** é definido pelo tipo complexo [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
