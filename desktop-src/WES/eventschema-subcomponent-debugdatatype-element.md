# <a name="subcomponent-debugdatatype-element"></a>Elemento subcomponente (DebugDataType)

O nome do subcomponente que registrou a mensagem de rastreamento.

``` syntax
<xs:element name="SubComponent"
    type="string"
 />
```

O elemento de **subcomponente** é definido pelo tipo complexo [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
