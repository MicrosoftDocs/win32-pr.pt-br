# <a name="channels-metadatatype-element"></a>Elemento channels (MetadataType)

Define uma lista de canais para os quais os provedores podem registrar eventos.

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

O **elemento** channels é definido pelo tipo complexo [**MetadataType.**](eventmanifestschema-metadatatype-complextype.md)
