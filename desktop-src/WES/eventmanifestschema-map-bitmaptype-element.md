# <a name="map-bitmaptype-element"></a>Elemento Map (BitMaptype)

Define o mapeamento entre um valor de bit e um valor de cadeia de caracteres.

``` syntax
<xs:element name="map"
    type="BitMapValueType"
 />
```

O elemento **MAP** é definido pelo tipo complexo [**bitmaptype**](eventmanifestschema-bitmaptype-complextype.md) .
