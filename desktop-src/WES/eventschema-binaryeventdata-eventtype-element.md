# <a name="binaryeventdata-eventtype-element"></a>Elemento BinaryEventData (EventType)

Contém os dados de evento como um blob binário.

``` syntax
<xs:element name="BinaryEventData"
    type="hexBinary"
 />
```

O elemento **BinaryEventData** é definido pelo tipo complexo [**EventType**](eventschema-eventtype-complextype.md) .
