# <a name="stringtable-localizationtype-element"></a>Elemento stringTable (corlocalizatype)

Define uma lista de cadeias de caracteres localizadas que você pode referenciar em seu manifesto.

``` syntax
<xs:element name="stringTable"
    type="StringTableType"
 />
```

O elemento **STRINGTABLE** é definido pelo tipo complexo de [**corlocalizatype**](eventmanifestschema-localizationtype-complextype.md) .
