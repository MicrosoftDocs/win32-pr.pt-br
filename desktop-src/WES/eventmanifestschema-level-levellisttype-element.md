# <a name="level-levellisttype-element"></a>Elemento Level (LevelListType)

Define um valor de severidade que determina o detalhamento de eventos durante o registro em log.

``` syntax
<xs:element name="level"
    type="LevelType"
 />
```

O elemento **Level** é definido pelo tipo complexo [**LevelListType**](eventmanifestschema-levellisttype-complextype.md) .
