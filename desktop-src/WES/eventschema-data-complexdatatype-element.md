# <a name="data-complexdatatype-element"></a>Elemento data (ComplexDataType)

A lista de itens de dados na estrutura. A lista de itens está na mesma ordem definida no modelo.

``` syntax
<xs:element name="Data"
    type="DataType"
 />
```

O elemento de **dados** é definido pelo tipo complexo [**ComplexDataType**](eventschema-complexdatatype-complextype.md) .
