# <a name="query-schema-elements"></a>Elementos do esquema de consulta

A seguir estão os elementos que o esquema de consulta define. Esta seção contém os nomes dos elementos que você usaria em sua consulta; no entanto, para obter os detalhes de cada elemento, consulte o tipo complexo que contém o elemento.



| Elemento                                                                  | Descrição                                                                                                                     |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**Consulta**](queryschema-querylist-element.md)                       | O nó raiz da consulta.<br/>                                                                                          |
| [**Consulta (QueryListType)**](queryschema-query-querylisttype-element.md) | Define um conjunto de seletores e supressers que são usados para incluir eventos no ou excluir eventos do conjunto de resultados.<br/> |
| [**Selecionar (QueryType)**](queryschema-select-querytype-element.md)       | Uma consulta XPath que identifica os eventos a serem incluídos no conjunto de resultados da consulta.<br/>                                        |
| [**Suprimir (QueryType)**](queryschema-suppress-querytype-element.md)   | Uma consulta XPath que identifica os eventos a serem excluídos do conjunto de resultados da consulta.<br/>                                      |
