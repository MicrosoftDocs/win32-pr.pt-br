# <a name="maxsize-channelloggingtype-element"></a>Elemento maxSize (ChannelLoggingType)

O tamanho máximo, em bytes, do arquivo de log.

``` syntax
<xs:element name="maxSize"
    type="long"
 />
```

O elemento **MaxSize** é definido pelo tipo complexo [**ChannelLoggingType**](eventmanifestschema-channelloggingtype-complextype.md) .
