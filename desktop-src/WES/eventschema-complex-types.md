# <a name="event-schema-complex-types"></a>Tipos complexos de esquema de evento

A seguir estão os tipos complexos que o esquema de evento define.



| Tipo complexo                                                                       | Descrição                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ComplexDataType**](eventschema-complexdatatype-complextype.md)                 | Define uma estrutura que contém um ou mais itens de dados.<br/>                                                                                                                                      |
| [**Tipo de dados**](eventschema-datafieldtype-complextype.md)                          | Define um item de dados.<br/>                                                                                                                                                                           |
| [**DebugDataType**](eventschema-debugdatatype-complextype.md)                     | Define os dados que podem ser registrados para eventos de pré-processamento (pré-processador de rastreamento de software) do Windows.<br/>                                                                                                      |
| [**EventDataType**](eventschema-eventdatatype-complextype.md)                     | Define os itens de dados do evento e as estruturas que contêm os dados do evento.<br/>                                                                                                                      |
| [**EventType**](eventschema-eventtype-complextype.md)                             | Define o nó raiz do esquema de evento.<br/>                                                                                                                                                     |
| [**ProcessingErrorDataType**](eventschema-processingerrordatatype-complextype.md) | Define um erro que ocorreu ao renderizar os dados de evento para o evento.<br/>                                                                                                                   |
| [**RenderingInfoType**](eventschema-renderingtype-complextype.md)                 | Define as mensagens renderizadas para o evento.<br/>                                                                                                                                                   |
| [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md)       | Define as informações que identificam o provedor e como elas foram habilitadas, o evento, o canal no qual o evento foi gravado e informações do sistema, como as IDs de processo e thread.<br/> |
| [**UserDataType**](eventschema-userdatatype-complextype.md)                       | Define um fragmento XML que especifica o layout dos dados do evento.<br/>                                                                                                                           |
