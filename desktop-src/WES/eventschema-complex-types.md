# <a name="event-schema-complex-types"></a>Tipos complexos de esquema de evento

A seguir estão os tipos complexos que o esquema de evento define.



| Tipo complexo                                                                       | Descrição                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ComplexDataType**](eventschema-complexdatatype-complextype.md)                 | Define uma estrutura que contém um ou mais itens de dados.<br/>                                                                                                                                      |
| [**DataType**](eventschema-datafieldtype-complextype.md)                          | Define um item de dados.<br/>                                                                                                                                                                           |
| [**DebugDataType**](eventschema-debugdatatype-complextype.md)                     | Define os dados que podem ser registrados em Windows eventos WPP (pré-processador de rastreamento de software).<br/>                                                                                                      |
| [**EventDataType**](eventschema-eventdatatype-complextype.md)                     | Define os itens de dados de evento e estruturas que contêm os dados do evento.<br/>                                                                                                                      |
| [**EventType**](eventschema-eventtype-complextype.md)                             | Define o nó raiz do esquema de evento.<br/>                                                                                                                                                     |
| [**ProcessingErrorDataType**](eventschema-processingerrordatatype-complextype.md) | Define um erro que ocorreu ao renderizar os dados do evento.<br/>                                                                                                                   |
| [**RenderingInfoType**](eventschema-renderingtype-complextype.md)                 | Define as mensagens renderizadas para o evento.<br/>                                                                                                                                                   |
| [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md)       | Define as informações que identificam o provedor e como ele foi habilitado, o evento, o canal no qual o evento foi gravado e informações do sistema, como as IDs do processo e do thread.<br/> |
| [**UserDataType**](eventschema-userdatatype-complextype.md)                       | Define um fragmento XML que especifica o layout dos dados do evento.<br/>                                                                                                                           |
