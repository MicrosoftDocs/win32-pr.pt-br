# <a name="event-definitiontype-element"></a>Elemento de evento (Definitiontype)

Define um evento que seu provedor pode registrar.

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

O elemento **Event** é definido pelo tipo complexo [**definitiontype**](eventmanifestschema-definitiontype-complextype.md) .
