# <a name="opcodes-providertype-element"></a>o elemento opcodes (ProviderType)

Define uma lista de opcodes que você pode usar para agrupar eventos dentro de uma tarefa.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

O elemento **opcodes** é definido pelo tipo complexo [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
