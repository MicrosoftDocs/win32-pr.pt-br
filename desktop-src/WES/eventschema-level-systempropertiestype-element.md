# <a name="level-systempropertiestype-element"></a>Elemento Level (SystemPropertiesType)

Contém o nível de severidade do evento.

``` syntax
<xs:element name="Level"
    type="unsignedByte"
 />
```

O elemento **Level** é definido pelo tipo complexo [**SystemPropertiesType**](eventschema-systempropertiestype-complextype.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do vista\]<br/>       |
| Servidor mínimo com suporte<br/> | Windows \[Somente aplicativos da área de trabalho do servidor 2008\]<br/> |
