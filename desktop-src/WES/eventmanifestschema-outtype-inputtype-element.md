# <a name="outtype-inputtype-element"></a>Elemento outtype (InputType)

Uma lista de tipos de dados de saída que determinam como os dados são renderizados.

``` syntax
<xs:element name="outType"
    type="OutputType"
 />
```

O elemento **outtype** é definido pelo tipo complexo [**InputType**](eventmanifestschema-inputtype-complextype.md) .
