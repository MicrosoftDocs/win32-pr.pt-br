# <a name="userdata-templateitemtype-element"></a>Elemento UserData (TemplateItemType)

Um fragmento XML que é usado para renderizar os dados do evento.

``` syntax
<xs:element name="UserData"
    type="XmlType"
 />
```

O elemento **UserData** é definido pelo tipo complexo [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md) .
