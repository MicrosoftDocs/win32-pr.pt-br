# <a name="keywords-channelpublishingtype-element"></a>Elemento Keywords (ChannelPublishingType)

Um bitmask que determina a categoria de eventos que são gravados no canal.

``` syntax
<xs:element name="keywords"
    type="UInt64Type"
 />
```

O elemento **Keywords** é definido pelo tipo complexo [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
