# <a name="intype-inputtypelisttype-element"></a>Elemento inType (InputTypeListType)

``` syntax

 />
```

O **elemento inType** é definido pelo tipo complexo [**InputTypeListType.**](eventmanifestschema-inputtypelisttype-complextype.md)
