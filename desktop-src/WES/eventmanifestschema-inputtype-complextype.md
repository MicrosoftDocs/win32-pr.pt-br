# <a name="inputtype-complex-type"></a>Tipo complexo InputType

Define um tipo de dados de entrada.

``` syntax
<xs:complexType name="InputType">
    <xs:sequence>
        <xs:element name="description"
            type="string"
         />
        <xs:element name="outType"
            type="OutputType"
            minOccurs="0"
            maxOccurs="unbounded"
         />
        <xs:any
            processContents="lax"
            namespace="##other"
         />
    </xs:sequence>
    <xs:attribute name="name"
        type="QName"
        use="required"
     />
    <xs:attribute name="value"
        type="string"
        use="required"
     />
    <xs:attribute name="symbol"
        type="CSymbolType"
        use="required"
     />
</xs:complexType>
```

## <a name="child-elements"></a>Elementos filho



| Elemento                                                                  | Type                                                             | Descrição                                                                                                                                                                                                                                                                                                                                                                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ndescrição**](eventmanifestschema-description-inputtype-element.md) | string                                                           | Uma descrição do tipo de entrada.<br/>                                                                                                                                                                                                                                                                                                                                                                   |
| [**outtipo**](eventmanifestschema-outtype-inputtype-element.md)         | [**OutputType**](eventmanifestschema-outputtype-complextype.md) | Uma lista de tipos de dados de saída que determinam como os dados são renderizados. Você deve especificar pelo menos um tipo de saída. Especifique uma lista se o tipo de entrada puder ser renderizado como tipos de saída diferentes. Por exemplo, o tipo de entrada Win: UInt16 pode ser renderizado como um número hexadecimal, um número de porta ou um números hexadecimais não assinados. Um dos tipos de saída especificados deve ter o atributo padrão definido como true.<br/> |



## <a name="attributes"></a>Atributos



| Nome   | Tipo                                                              | Descrição                                                                                                                                                                                                                                              |
|--------|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| símbolo | [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | O símbolo a ser usado para fazer referência ao tipo de entrada em seu aplicativo. O [**compilador de mensagem (MC.exe)**](message-compiler--mc-exe-.md) usa o símbolo para criar uma constante para o tipo de entrada no arquivo de cabeçalho que o compilador gera.<br/> |
| value  | string                                                            | Um identificador numérico que identifica exclusivamente o tipo de entrada em uma lista de tipos de entrada que você define.<br/>                                                                                                                                        |
