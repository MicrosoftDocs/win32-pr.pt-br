# <a name="processingerrordata-eventtype-element"></a>Elemento ProcessingErrorData (EventType)

Contém detalhes do erro que ocorreu ao tentar renderizar o evento.

``` syntax
<xs:element name="ProcessingErrorData"
    type="ProcessingErrorDataType"
 />
```

O elemento **ProcessingErrorData** é definido pelo tipo complexo [**EventType**](eventschema-eventtype-complextype.md) .
