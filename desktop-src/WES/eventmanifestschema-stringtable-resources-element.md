# <a name="stringtable-resources-element"></a>Elemento stringTable (Resources)

Define uma lista de cadeias de caracteres localizadas que você pode referenciar em seu manifesto.

``` syntax
<xs:element name="stringTable"
    type="StringTableType"
 />
```

O elemento **STRINGTABLE** é definido pelo elemento [**Resources**](eventmanifestschema-resources-localizationtype-element.md) .
