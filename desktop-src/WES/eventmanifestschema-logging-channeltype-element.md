# <a name="logging-channeltype-element"></a>Elemento Logging (channelType)

Define as propriedades do arquivo de log que faz o backup do canal, como sua capacidade e se ele é sequencial ou circular.

``` syntax
<xs:element name="logging"
    type="ChannelLoggingType"
 />
```

O elemento **Logging** é definido pelo tipo complexo [**channelType**](eventmanifestschema-channeltype-complextype.md) .
