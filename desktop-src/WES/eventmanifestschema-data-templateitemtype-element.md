# <a name="data-templateitemtype-element"></a>Elemento data (TemplateItemType)

Define um item de dados que você deseja incluir com o evento.

``` syntax
<xs:element name="data"
    type="DataDefinitionType"
 />
```

O elemento de **dados** é definido pelo tipo complexo [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md) .
