# <a name="latency-channelpublishingtype-element"></a>Elemento latência (ChannelPublishingType)

O tempo de espera antes de liberar os buffers, em milissegundos.

``` syntax
<xs:element name="latency"
    type="unsignedInt"
 />
```

O elemento de **latência** é definido pelo tipo complexo [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .
