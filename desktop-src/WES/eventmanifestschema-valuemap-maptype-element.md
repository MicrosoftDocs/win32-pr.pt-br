# <a name="valuemap-maptype-element"></a>Elemento valueMap (MapType)

Define uma lista de pares de nome/valor que mapeia valores inteiros e valores de cadeia de caracteres.

``` syntax
<xs:element name="valueMap"
    type="ValueMapType"
 />
```

O elemento **valueMap** é definido pelo tipo complexo [**MapType**](eventmanifestschema-maptype-complextype.md) .
