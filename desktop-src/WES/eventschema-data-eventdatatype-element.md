# <a name="data-eventdatatype-element"></a>Elemento data (EventDataType)

Um item de dados de nível superior que é definido no modelo para o evento.

``` syntax
<xs:element name="Data"
    type="DataFieldType"
 />
```

O elemento de **dados** é definido pelo tipo complexo [**EventDataType**](eventschema-eventdatatype-complextype.md) .
