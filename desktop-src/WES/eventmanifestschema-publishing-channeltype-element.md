# <a name="publishing-channeltype-element"></a>Elemento de publicação (channelType)

Define as propriedades de log para a sessão que o canal usa.

``` syntax
<xs:element name="publishing"
    type="ChannelPublishingType"
 />
```

O elemento de **publicação** é definido pelo tipo complexo [**channelType**](eventmanifestschema-channeltype-complextype.md) .
