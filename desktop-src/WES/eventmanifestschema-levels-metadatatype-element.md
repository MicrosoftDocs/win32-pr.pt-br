# <a name="levels-metadatatype-element"></a>Elemento Levels (MetadataType)

Define uma lista de níveis que especificam a severidade de um evento.

``` syntax
<xs:element name="levels"
    type="LevelListType"
 />
```

O elemento **Levels** é definido pelo tipo complexo [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) .
