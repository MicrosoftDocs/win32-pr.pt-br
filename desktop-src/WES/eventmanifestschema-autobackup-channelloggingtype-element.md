# <a name="autobackup-channelloggingtype-element"></a>Elemento backup (ChannelLoggingType)

Determina se um novo arquivo de log deve ser criado quando o arquivo de log atual atinge seu tamanho máximo.

``` syntax
<xs:element name="autoBackup"
    type="boolean"
 />
```

O elemento **AutoBackup** é definido pelo tipo complexo [**ChannelLoggingType**](eventmanifestschema-channelloggingtype-complextype.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Vista\]<br/>       |
| Servidor mínimo com suporte<br/> | \[Somente aplicativos da área de trabalho do Windows Server 2008\]<br/> |
