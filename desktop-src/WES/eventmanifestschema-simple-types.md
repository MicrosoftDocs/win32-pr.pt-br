# <a name="eventmanifest-schema-simple-types"></a>Tipos simples de esquema EventManifest

A seguir estão os tipos simples que o esquema EventManifest define.



| Tipo simples                                                       | Descrição                                                                                                                                |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**CountType**](eventmanifestschema-counttype-simpletype.md)     | Define um tipo de contagem usado para especificar o número de elementos em uma matriz.<br/>                                                |
| [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | Define um nome de símbolo C/C++ válido.<br/>                                                                                              |
| [**Filepath**](eventmanifestschema-filepath-simpletype.md)       | Define uma cadeia de caracteres que contém um caminho totalmente qualificado para um arquivo.<br/>                                                                |
| [**GUIDType**](eventmanifestschema-guidtype-simpletype.md)       | Define um tipo de identificador global exclusivo, no formato registro.<br/>                                                                  |
| [**HexInt8Type**](eventmanifestschema-hex8type-simpletype.md)    | Define um tipo hexadecimal de 1 byte.<br/>                                                                                              |
| [**HexInt16Type**](eventmanifestschema-hex16type-simpletype.md)  | Define um tipo hexadecimal de 2 byte.<br/>                                                                                              |
| [**HexInt32Type**](eventmanifestschema-hex32type-simpletype.md)  | Define um tipo hexadecimal de 4 byte.<br/>                                                                                              |
| [**HexInt64Type**](eventmanifestschema-hex64type-simpletype.md)  | Define um tipo hexadecimal de 8 byte.<br/>                                                                                             |
| [**UInt8Type**](eventmanifestschema-hexint8type-simpletype.md)   | Define um tipo de byte não assinado.<br/>                                                                                                  |
| [**UInt16Type**](eventmanifestschema-hexint16type-simpletype.md) | Define um tipo curto sem assinatura.<br/>                                                                                                 |
| [**UInt32Type**](eventmanifestschema-hexint32type-simpletype.md) | Define um tipo inteiro sem sinal.<br/>                                                                                               |
| [**UInt64Type**](eventmanifestschema-hexint64type-simpletype.md) | Define um tipo longo sem assinatura.<br/>                                                                                                  |
| [**LengthType**](eventmanifestschema-lengthtype-simpletype.md)   | Define um tipo de comprimento usado para especificar o número de bytes ou caracteres em um item de dados de comprimento variável.<br/>                 |
| [**QNameList**](eventmanifestschema-qnamelist-simpletype.md)     | Define uma lista de nomes qualificados.<br/>                                                                                              |
| [**strTableRef**](eventmanifestschema-strtableref-simpletype.md) | Define uma cadeia de caracteres que faz referência a uma cadeia de caracteres de mensagem definida em uma tabela de cadeia de caracteres no manifesto ou em um arquivo de mensagem (.mc).<br/> |
