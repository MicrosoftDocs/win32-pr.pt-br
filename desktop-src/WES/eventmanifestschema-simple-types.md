# <a name="eventmanifest-schema-simple-types"></a>Tipos simples de esquema EventManifest

A seguir estão os tipos simples que o esquema EventManifest define.



| Tipo simples                                                       | Descrição                                                                                                                                |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**Número de contagem**](eventmanifestschema-counttype-simpletype.md)     | Define um tipo de contagem que é usado para especificar o número de elementos em uma matriz.<br/>                                                |
| [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | Define um nome de símbolo do C/C++ válido.<br/>                                                                                              |
| [**filePath**](eventmanifestschema-filepath-simpletype.md)       | Define uma cadeia de caracteres que contém um caminho totalmente qualificado para um arquivo.<br/>                                                                |
| [**GUIDtype**](eventmanifestschema-guidtype-simpletype.md)       | Define um tipo de identificador global exclusivo, no formato de registro.<br/>                                                                  |
| [**HexInt8Type**](eventmanifestschema-hex8type-simpletype.md)    | Define um tipo hexadecimal de 1 byte.<br/>                                                                                              |
| [**HexInt16Type**](eventmanifestschema-hex16type-simpletype.md)  | Define um tipo hexadecimal de 2 bytes.<br/>                                                                                              |
| [**HexInt32Type**](eventmanifestschema-hex32type-simpletype.md)  | Define um tipo hexadecimal de 4 bytes.<br/>                                                                                              |
| [**HexInt64Type**](eventmanifestschema-hex64type-simpletype.md)  | Define um tipo hexadecimal de 8 bytes.<br/>                                                                                             |
| [**UInt8Type**](eventmanifestschema-hexint8type-simpletype.md)   | Define um tipo de byte não assinado.<br/>                                                                                                  |
| [**UInt16type**](eventmanifestschema-hexint16type-simpletype.md) | Define um tipo curto sem sinal.<br/>                                                                                                 |
| [**UInt32type**](eventmanifestschema-hexint32type-simpletype.md) | Define um tipo inteiro sem sinal.<br/>                                                                                               |
| [**UInt64type**](eventmanifestschema-hexint64type-simpletype.md) | Define um tipo longo sem sinal.<br/>                                                                                                  |
| [**Comprimentotype**](eventmanifestschema-lengthtype-simpletype.md)   | Define um tipo de comprimento que é usado para especificar o número de bytes ou caracteres em um item de dados de comprimento variável.<br/>                 |
| [**QNamelist**](eventmanifestschema-qnamelist-simpletype.md)     | Define uma lista de nomes qualificados.<br/>                                                                                              |
| [**strTableRef**](eventmanifestschema-strtableref-simpletype.md) | Define uma cadeia de caracteres que faz referência a uma cadeia de caracteres de mensagem que é definida em uma tabela de cadeia de caracteres no manifesto ou em um arquivo de mensagem (. MC).<br/> |
