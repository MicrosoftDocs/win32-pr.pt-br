# <a name="webdav-api-structures"></a>Estruturas da API do WebDAV

As estruturas a seguir são usadas na API do WebDAV.



| Estrutura                                                   | Descrição                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BLOB de autenticação de retorno de \_ chamada DAV \_ \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Essa estrutura é usada para armazenar um [*blob*](/windows/desktop/SecGloss/b-gly) de autenticação que foi recuperado pela função de retorno de chamada [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) . |
| [**UNP de autenticação de retorno de \_ chamada DAV \_ \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Essa estrutura é usada para armazenar informações de nome de usuário e senha que foram recuperadas pela função de retorno de chamada [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) .                                               |
| [**\_credenciais de retorno de chamada DAV \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | A função de retorno de chamada [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) usa essa estrutura para armazenar informações de credenciais de usuário.                                                                               |
