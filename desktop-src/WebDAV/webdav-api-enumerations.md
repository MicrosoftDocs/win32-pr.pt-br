# <a name="webdav-api-enumerations"></a>Enumerações da API WebDAV

A enumeração a seguir é usada na API WebDAV.



| Enumeração                          | Descrição                                                                                                                                          |
|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AUTHNEXTSTEP**](/windows/desktop/api/davclnt/ne-davclnt-authnextstep) | Especifica a próxima ação que o cliente WebDAV deve tomar após uma chamada bem-sucedida para a função de retorno de chamada [*DavAuthCallback.*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) |
