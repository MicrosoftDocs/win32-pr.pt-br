# <a name="webdav"></a>WebDAV

## <a name="purpose"></a>Finalidade

WebDAV (Criação e Versão Distribuída da Web) é uma extensão para o PROTOCOLO HTTP que define como as funções de arquivo básicas, como copiar, mover, excluir e criar, são executadas usando HTTP. A API WebDAV é um conjunto de funções Win32 para criar e gerenciar conexões com servidores WebDAV e executar operações de E/S de arquivo em arquivos remotos em servidores WebDAV. As funções da API WebDAV são projetadas para serem usadas junto com outras funções de API do Win32, como funções de rede [Windows](/windows/desktop/WNet/windows-networking-functions) funções de gerenciamento [de arquivos](/windows/desktop/FileIO/file-management-functions).

## <a name="developer-audience"></a>Público de desenvolvedores

A API WebDAV foi projetada para uso por desenvolvedores C e C++.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

A API WebDAV está no Windows Vista e posterior. Para obter informações sobre quais versões do sistema operacional são necessárias para usar um elemento de programação específico, consulte a seção Requisitos da página de referência para esse elemento.
