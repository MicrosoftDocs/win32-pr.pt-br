# <a name="webdav"></a>WebDAV

## <a name="purpose"></a>Finalidade

O WebDAV (criação e controle de versão distribuída na Web) é uma extensão do protocolo HTTP que define como as funções básicas de arquivo, como copiar, mover, excluir e criar, são executadas usando HTTP. A API do WebDAV é um conjunto de funções do Win32 para criar e gerenciar conexões com servidores WebDAV e executar operações de e/s de arquivo em arquivos remotos em servidores WebDAV. As funções da API do WebDAV são projetadas para serem usadas junto com outras funções da API do Win32, como funções de [rede do Windows](/windows/desktop/WNet/windows-networking-functions) e funções de [Gerenciamento de arquivos](/windows/desktop/FileIO/file-management-functions).

## <a name="developer-audience"></a>Público de desenvolvedores

A API do WebDAV foi projetada para ser usada por desenvolvedores de C e C++.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

A API do WebDAV está no Windows Vista e versões posteriores. Para obter informações sobre quais versões do sistema operacional são necessárias para usar um elemento de programação específico, consulte a seção requisitos da página de referência para esse elemento.
