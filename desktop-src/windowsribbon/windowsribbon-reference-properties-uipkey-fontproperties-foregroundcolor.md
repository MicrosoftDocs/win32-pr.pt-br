# <a name="ui_pkey_fontproperties_foregroundcolor"></a>\_ \_ FontProperties \_ ForegroundColor da interface do usuário PKEY

Identifica a propriedade \_ \_ FontProperties \_ ForegroundColor da IU PKEY.

```
propertyDescription
   name = UI_PKEY_FontProperties_ForegroundColor
   shellPKey = UI_PKEY_FontProperties_ForegroundColor
   formatID = 00000308-7363-696e-8441798acf5aebb7
   propID = 308
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>Comentários

A interface do usuário PKEY FontProperties ForegroundColor é usada por um aplicativo, em conjunto com a interface do usuário \_ \_ \_ [ \_ PKEY \_ FontProperties \_ ForegroundColorType](windowsribbon-reference-properties-uipkey-fontproperties-foregroundcolortype.md), para consultar as **configurações** da galeria de cores do texto.

O valor padrão é `0x00000000`.
