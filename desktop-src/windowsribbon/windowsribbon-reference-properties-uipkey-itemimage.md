# <a name="ui_pkey_itemimage"></a>PKEY da interface do usuário \_ \_

Identifica a \_ Propriedade PKEY ItemProperty da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ é usada por um aplicativo para consultar a imagem associada a um item ou comando em um controle de galeria.

O valor da propriedade é um objeto [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
