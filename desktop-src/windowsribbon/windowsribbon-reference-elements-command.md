# <a name="commands-and-resources"></a>Comandos e recursos

Documentação de referência para elementos de marcação de comando e recurso na estrutura da faixa de referência do Windows.



| Tópico                                                                                            | Sumário                                                                                                                |
|--------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Comando**](windowsribbon-element-command.md)                                                 | Representa uma definição de comando.<br/>                                                                             |
| [**Comando. Comment**](windowsribbon-element-command-comment.md)                                 | Representa um comentário ou uma anotação para um comando.<br/>                                                          |
| [**Command.Id**](windowsribbon-element-command-id.md)                                           | Representa uma ID exclusiva para um comando.<br/>                                                                        |
| [**Comando. KeyTip**](windowsribbon-element-command-keytip.md)                                   | Representa o KeyTip de um controle.<br/>                                                                         |
| [**Comando. LabelDescription**](windowsribbon-element-command-labeldescription.md)               | Representa uma descrição do rótulo.<br/>                                                                              |
| [**Comando. LabelTitle**](windowsribbon-element-command-labeltitle.md)                           | Representa um título de rótulo.<br/>                                                                                    |
| [**Comando. LargeHighContrastImages**](windowsribbon-element-command-largehighcontrastimages.md) | Representa um contêiner de imagens; Nesse caso, imagens grandes para uso com configurações do sistema de alto contraste.<br/>     |
| [**Comando. LargeImages**](windowsribbon-element-command-largeimages.md)                         | Representa um contêiner de imagens; Nesse caso, imagens grandes.<br/>                                                |
| [**Command.Name**](windowsribbon-element-command-name.md)                                       | Representa o nome de um comando.<br/>                                                                            |
| [**Comando. SmallHighContrastImages**](windowsribbon-element-command-smallhighcontrastimages.md) | Representa um contêiner de imagens; Nesse caso, imagens pequenas para uso com configurações do sistema de alto contraste.<br/>     |
| [**Comando. SmallImages**](windowsribbon-element-command-smallimages.md)                         | Representa um contêiner de imagens; Nesse caso, imagens pequenas.<br/>                                                |
| [**Comando. Symbol**](windowsribbon-element-command-symbol.md)                                   | Representa o nome de um [**comando**](windowsribbon-element-command.md) que pode ser referenciado externamente.<br/> |
| [**Comando. TooltipDescription**](windowsribbon-element-command-tooltipdescription.md)           | Representa uma descrição da dica de ferramenta.<br/>                                                                            |
| [**Comando. TooltipTitle**](windowsribbon-element-command-tooltiptitle.md)                       | Representa um título de dica de ferramenta.<br/>                                                                                  |
| [**Imagem**](windowsribbon-element-image.md)                                                     | Representa uma imagem.<br/>                                                                                         |
| [**Imagem. origem**](windowsribbon-element-image-source.md)                                       | Representa o caminho de diretório de uma imagem.<br/>                                                                   |
| [**String**](windowsribbon-element-string.md)                                                   | Representa um recurso de cadeia de caracteres.<br/>                                                                                |
| [**String. Content**](windowsribbon-element-string-content.md)                                   | Representa o conteúdo de um recurso de cadeia de caracteres.<br/>                                                                 |
| [**String.Id**](windowsribbon-element-string-id.md)                                             | Representa a ID exclusiva de um recurso de cadeia de caracteres.<br/>                                                               |
| [**String. Symbol**](windowsribbon-element-string-symbol.md)                                     | Representa o nome de um recurso de cadeia de caracteres.<br/>                                                                    |
