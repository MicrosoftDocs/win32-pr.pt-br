# <a name="ui_pkey_globalbackgroundcolor"></a>\_GlobalBackgroundColor PKEY \_ UI

Identifica a \_ Propriedade PKEY GlobalBackgroundColor da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ GlobalBackgroundColor é usada por um aplicativo para consultar a cor global do plano de fundo de uma faixa de faixas personalizada.

**Introduzido no Windows 8:** a  [interface do usuário \_ PKEY \_ GlobalHighlightColor](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) não pode ser definida independentemente da interface do usuário \_ PKEY \_ GlobalBackgroundColor.
