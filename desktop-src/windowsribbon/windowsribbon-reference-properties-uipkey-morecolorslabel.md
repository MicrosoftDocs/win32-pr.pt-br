# <a name="ui_pkey_morecolorslabel"></a>\_MoreColorsLabel PKEY \_ UI

Identifica a \_ Propriedade PKEY MoreColorsLabel da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ MoreColorsLabel é usada por um aplicativo para consultar o valor do rótulo associado ao botão **mais cores** de um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

A interface do usuário \_ PKEY \_ MoreColorsLabel é válida somente para um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) onde `ThemeColors` ou `StandardColors` é especificado como o valor do atributo **colortemplate** .
