# <a name="ui_pkey_morecolorslabel"></a>Interface do \_ usuário PKEY \_ MoreColorsLabel

Identifica a propriedade \_ UI PKEY \_ MoreColorsLabel.

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

A interface do usuário PKEY MoreColorsLabel é usada por um aplicativo para consultar o valor do rótulo associado ao botão Mais cores de \_ \_ um [**DropDownColorPicker.**](windowsribbon-element-dropdowncolorpicker.md) 

A \_ interface do usuário PKEY MoreColorsLabel só é válida para \_ um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) em que ou é especificado como o valor `ThemeColors` do atributo `StandardColors` **ColorTemplate.**
