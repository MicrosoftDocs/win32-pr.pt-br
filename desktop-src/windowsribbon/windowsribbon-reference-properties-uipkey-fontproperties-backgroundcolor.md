# <a name="ui_pkey_fontproperties_backgroundcolor"></a>\_ \_ FontProperties BackgroundColor da interface do usuário PKEY \_

Identifica a propriedade \_ BackgroundColor da interface do usuário PKEY \_ FontProperties. \_

```
propertyDescription
   name = UI_PKEY_FontProperties_BackgroundColor
   shellPKey = UI_PKEY_FontProperties_BackgroundColor
   formatID = 00000309-7363-696e-8441798acf5aebb7
   propID = 309
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>Comentários

A Interface do usuário PKEY FontProperties BackgroundColor é usada por um aplicativo, em conjunto com a interface do usuário \_ \_ \_ [ \_ \_ PKEY FontProperties \_ BackgroundColorType](windowsribbon-reference-properties-uipkey-fontproperties-backgroundcolortype.md), para consultar as **configurações** da galeria de cores de realçadas de texto.

O valor padrão é `0x00000000`.
