# <a name="ui_pkey_tooltipdescription"></a>\_TooltipDescription PKEY \_ UI

Identifica a \_ Propriedade PKEY TooltipDescription da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_TooltipDescription
   shellPKey = UI_PKEY_TooltipDescription
   formatID = 00000005-7363-696e-8441798acf5aebb7
   propID = 5
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ TooltipDescription é usada por um aplicativo para consultar a descrição associada a uma [interface do usuário \_ PKEY \_ TooltipTitle](windowsribbon-reference-properties-uipkey-tooltiptitle.md).

O valor da propriedade é uma cadeia de caracteres restrita a qualquer sequência de caracteres, incluindo espaços em branco e caracteres de quebra de linha.

> [!Note]  
> Use a referência de caractere XML UCS (conjunto de caracteres universal) `&#xA;` para especificar uma quebra de linha.

 

O comprimento máximo da interface do usuário \_ PKEY \_ TooltipDescription é não associado.

Não há suporte para alinhamento à direita.
