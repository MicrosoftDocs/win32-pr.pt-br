# <a name="ui_pkey_contextavailable"></a>Contexto \_ PKEY da \_ interface do usuárioAvailable

Identifica a propriedade \_ ContextAvailable de PKEY \_ da interface do usuário.

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Comentários

Contexto PKEY da interface do usuárioAvailable é usado por um aplicativo para consultar a disponibilidade de um controle Grupo de Guias \_ com base no contexto do aplicativo \_ atual. [](windowsribbon-controls-tabgroup.md)
