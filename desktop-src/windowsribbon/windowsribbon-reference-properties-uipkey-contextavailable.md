# <a name="ui_pkey_contextavailable"></a>\_ContextAvailable PKEY \_ UI

Identifica a \_ Propriedade PKEY ContextAvailable da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ ContextAvailable é usada por um aplicativo para consultar a disponibilidade de um controle de [grupo de guias](windowsribbon-controls-tabgroup.md) com base no contexto do aplicativo atual.
