# <a name="ui_pkey_recentcolorscategorylabel"></a>Interface \_ do usuário PKEY \_ RecentColorsCategoryLabel

Identifica a propriedade \_ PKEY \_ RecentColorsCategoryLabel da interface do usuário.

```
propertyDescription
   name = UI_PKEY_RecentColorsCategoryLabel
   shellPKey = UI_PKEY_RecentColorsCategoryLabel
   formatID = 00000405-7363-696e-8441798acf5aebb7
   propID = 405
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

A \_ interface do usuário PKEY \_ RecentColorsCategoryLabel é usada por um aplicativo  para consultar o valor do rótulo associado à categoria Cores Recentes de [**um DropDownColorPicker.**](windowsribbon-element-dropdowncolorpicker.md)

A interface do usuário \_ PKEY RecentColorsCategoryLabel é válida somente para \_ um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) em que é especificado como o valor do atributo `ThemeColors` **ColorTemplate** (esse é o único modelo que contém categorias rotuladas).

A captura de tela a seguir mostra `ThemeColors` [**um DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).
