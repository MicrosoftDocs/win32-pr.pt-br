# <a name="ui_all_commands"></a>\_todos os comandos da interface do usuário \_

Especifica uma constante que identifica a coleção de comandos declarados no arquivo de recurso de marcação.

## <a name="remarks"></a>Comentários

**Interface do usuário \_ Todos os \_ comandos** são úteis quando é necessário acessar propriedades semelhantes em todos os comandos. Por exemplo, **interface do usuário \_ todos os \_ comandos** podem ser passados para [**IUIFramework:: InvalidateUICommand**](/windows/desktop/api/uiribbon/nf-uiribbon-iuiframework-invalidateuicommand) para invalidar e atualizar todos os comandos da faixa de faixas consultando o aplicativo host para obter novos valores de propriedade.
