# <a name="ui_all_commands"></a>TODOS OS \_ COMANDOS DA INTERFACE DO \_ USUÁRIO

Especifica uma constante que identifica a coleção de Comandos declarados no arquivo de recurso de Marcação.

## <a name="remarks"></a>Comentários

**Interface do usuário \_ ALL \_ COMMANDS** é útil quando é necessário acessar propriedades semelhantes em todos os Comandos. Por exemplo, **todos \_ \_** os COMANDOS da interface do usuário podem ser passados para [**IUIFramework::InvalidateUICommand**](/windows/desktop/api/uiribbon/nf-uiribbon-iuiframework-invalidateuicommand) para invalidar e atualizar todos os Comandos da Faixa de Opções consultando o aplicativo host para novos valores de propriedade.
