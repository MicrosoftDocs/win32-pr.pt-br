# <a name="ui_pkey_smallhighcontrastimage"></a>PKEY \_ da interface do usuário \_ SmallHighContrastImage

Identifica a propriedade \_ PKEY \_ SmallHighContrastImage da interface do usuário.

```
propertyDescription
   name = UI_PKEY_SmallHighContrastImage
   shellPKey = UI_PKEY_SmallHighContrastImage
   formatID = 00000010-7363-696e-8441798acf5aebb7
   propID = 10
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Comentários

A \_ interface do usuário PKEY SmallHighContrastImage é usada por um aplicativo para consultar a imagem pequena e de alto contraste associada a um controle de faixa \_ de opções.

O valor da propriedade é um [**objeto IUIImage.**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage)
