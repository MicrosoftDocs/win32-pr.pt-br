# <a name="constants-and-enumerations"></a>Constantes e enumerações

Documentação de referência para as Windows e enumerações da estrutura da Faixa de Opções.

### <a name="constants-and-enumerations"></a>Constantes e enumerações




| Constante | Descrição | 
|----------|-------------|
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-all-commands"><strong>UI_ALL_COMMANDS</strong></a><br /> | Especifica uma constante que identifica a coleção de Comandos declarados no arquivo de recurso de Marcação.<br /> | 
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-collection-invalidindex"><strong>UI_COLLECTION_INVALIDINDEX</strong></a><br /> | Especifica uma constante que identifica um índice inválido em uma coleção.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_collectionchange"><strong>UI_COLLECTIONCHANGE</strong></a><br /> | Especifica valores que identificam os tipos de alterações que podem ser feitas em uma coleção.<br /> | 
| <a href="/previous-versions/windows/desktop/legacy/hh448736(v=vs.85)"><strong>UI_COMMANDCATEGORY</strong></a><br /> | Especifica valores que indicam se um Comando foi definido em tempo de compilação ou em tempo de run.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype"><strong>UI_COMMANDTYPE</strong></a><br /> | Especifica valores que identificam o tipo de Comando associado a um controle faixa de opções.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_contextavailability"><strong>UI_CONTEXTAVAILABILITY</strong></a><br /> | Especifica valores que identificam a disponibilidade de uma <a href="windowsribbon-element-ribbon-contextualtabs.md"><strong>guia contextual</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_controldock"><strong>UI_CONTROLDOCK</strong></a><br /> | Especifica valores que identificam o estado de encaixe da QAT (Barra de Ferramentas de Acesso Rápido). <br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventlocation"><strong>UI_EVENTLOCATION</strong></a><br /> | Identifica os locais em que os eventos associados a um controle faixa de opções podem ser originados.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventtype"><strong>UI_EVENTTYPE</strong></a><br /> | Identifica os tipos de eventos associados a uma Faixa de <a href="windowsribbon-element-ribbon.md"><strong>Opções</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_executionverb"><strong>UI_EXECUTIONVERB</strong></a><br /> | Especifica valores que identificam as IDs de execução que mapeiam para ações que um usuário pode iniciar em um <a href="windowsribbon-element-command.md"><strong>Comando</strong></a>. <br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontdeltasize"><strong>UI_FONTDELTASIZE</strong></a><br /> | Especifica valores que identificam se o tamanho da fonte de uma run de texto realçada deve ser incrementado ou decrementado.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontproperties"><strong>UI_FONTPROPERTIES</strong></a><br /> | Especifica valores que identificam o estado da propriedade de fonte de <a href="windowsribbon-element-fontcontrol.md"><strong>um FontControl,</strong></a>como <strong>tachado.</strong><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontunderline"><strong>UI_FONTUNDERLINE</strong></a><br /> | Especifica valores que identificam o estado sublinhado de <a href="windowsribbon-element-fontcontrol.md"><strong>um FontControl.</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontverticalposition"><strong>UI_FONTVERTICALPOSITION</strong></a><br /> | Especifica valores que identificam o estado de alinhamento vertical de <a href="windowsribbon-element-fontcontrol.md"><strong>um FontControl.</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_invalidations"><strong>UI_INVALIDATIONS</strong></a><br /> | Especifica valores que identificam o aspecto de um Comando a ser invalidado.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_ownership"><strong>UI_OWNERSHIP</strong></a><br /> | Especifica valores que identificam as condições de propriedade nas quais uma imagem é criada pela estrutura da Faixa de Opções.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode"><strong>UI_SWATCHCOLORMODE</strong></a><br /> | Especifica se um swatch tem o modo normal ou monocromático.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_swatchcolortype"><strong>UI_SWATCHCOLORTYPE</strong></a><br /> | Especifica os valores que identificam como um swatch de cor em <a href="windowsribbon-element-dropdowncolorpicker.md"><strong>um DropDownColorPicker</strong></a> ou um <a href="windowsribbon-element-fontcontrol.md"><strong>selador</strong></a> de cores FontControl<strong>(</strong> Cor do texto ou Realçamento <strong>de</strong>texto ) é preenchido.<br /><blockquote>[!Note]<br />Essas são apenas recomendações. O aplicativo pode associar qualquer tipo de preenchimento a esses valores.</blockquote><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewtype"><strong>UI_VIEWTYPE</strong></a><br /> | Especifica valores que identificam a Exibição da estrutura da Faixa de Opções.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewverb"><strong>UI_VIEWVERB</strong></a><br /> | Especifica valores que identificam o tipo de ação a ser concluída em uma Exibição de estrutura da Faixa de Opções.<br /> |
