# <a name="ui_pkey_decimalplaces"></a>\_PKEY \_ CasasDecimais da interface do usuário

Identifica a \_ Propriedade PKEY CasasDecimais da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Comentários

O pkey CasasDecimais da interface do usuário \_ \_ é usado por um aplicativo para consultar o número de dígitos mostrados após o ponto decimal no campo de edição do controle [Spinner](windowsribbon-controls-spinner.md) .
