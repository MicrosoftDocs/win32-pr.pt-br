# <a name="ui_pkey_decimalplaces"></a>Casas deCimais \_ PKEY \_ da interface do usuário

Identifica a propriedade \_ DecimalPlaces PKEY \_ da interface do usuário.

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Comentários

A \_ interface do usuário PKEY DecimalPlaces é usada por um aplicativo para consultar o número de dígitos mostrados após o ponto decimal no campo de edição do \_ controle [Spinner.](windowsribbon-controls-spinner.md)
