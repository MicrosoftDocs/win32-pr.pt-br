# <a name="ui_pkey_themecolorscategorylabel"></a>\_ThemeColorsCategoryLabel PKEY \_ UI

Identifica a \_ Propriedade PKEY ThemeColorsCategoryLabel da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_ThemeColorsCategoryLabel
   shellPKey = UI_PKEY_ThemeColorsCategoryLabel
   formatID = 00000403-7363-696e-8441798acf5aebb7
   propID = 403
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ ThemeColorsCategoryLabel é usada por um aplicativo para consultar o valor do rótulo associado à categoria **cores do tema** de um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

A interface do usuário \_ PKEY \_ ThemeColorsCategoryLabel é válida somente para um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) em que `ThemeColors` é especificado como o valor do atributo **colortemplate** (esse é o único modelo que contém categorias rotuladas).

A captura de tela a seguir mostra um `ThemeColors` [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).
