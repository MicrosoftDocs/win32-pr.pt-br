# <a name="ui_pkey_color"></a>\_Cor de PKEY da interface do usuário \_

Identifica a propriedade de cor da interface do usuário \_ PKEY \_ .

```
propertyDescription
   name = UI_PKEY_Color
   shellPKey = UI_PKEY_Color
   formatID = 00000400-7363-696e-8441798acf5aebb7
   propID = 400
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Comentários

A \_ cor de PKEY da interface do usuário \_ é usada por um aplicativo para consultar o valor de cor do controle [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) .

O valor da propriedade é um valor de [COLORREF](../gdi/colorref.md) .

O byte de ordem superior do valor de [COLORREF](../gdi/colorref.md) é ignorado.
