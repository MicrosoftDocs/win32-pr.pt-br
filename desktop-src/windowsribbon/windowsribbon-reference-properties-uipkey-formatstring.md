# <a name="ui_pkey_formatstring"></a>\_FormatString PKEY \_ da interface do usuário

Identifica a propriedade \_ FormatString PKEY da interface do \_ usuário.

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

FormatString PKEY da interface do usuário é usado por um aplicativo para consultar o formato (ou unidade de medida) para o valor \_ \_ do controle [Spinner.](windowsribbon-controls-spinner.md)
