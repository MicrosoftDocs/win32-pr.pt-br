# <a name="ui_pkey_automaticcolorlabel"></a>\_AutomaticColorLabel PKEY \_ UI

Identifica a \_ Propriedade PKEY AutomaticColorLabel da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_AutomaticColorLabel
   shellPKey = UI_PKEY_AutomaticColorLabel
   formatID = 00000406-7363-696e-8441798acf5aebb7
   propID = 406
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ AutomaticColorLabel é usada por um aplicativo para consultar o valor do rótulo associado ao botão **automático** de um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md). Esse botão é usado pelo aplicativo para especificar uma cor padrão, conforme necessário.

A interface do usuário \_ PKEY \_ AutomaticColorLabel é válida somente para um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) onde `ThemeColors` ou `StandardColors` é especificado como o valor do atributo **colortemplate** .
