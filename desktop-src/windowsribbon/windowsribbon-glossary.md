# <a name="windows-ribbon-framework-glossary"></a>Glossário do Windows Ribbon Framework

## <a name="g"></a>G

<dl> <dt>

**galeria**
</dt> <dd>

Caixas de listagem ricas e personalizáveis que podem ser usadas para organizar itens por categoria, exibi-los em layouts baseados em coluna e em linha flexíveis e representá-los com imagens e texto. Dependendo do tipo de galeria, a visualização dinâmica também tem suporte.

</dd> </dl>

## <a name="m"></a>M

<dl> <dt>

**Minibarra**
</dt> <dd>

Um conjunto de comandos da faixa de, com escopo para o contexto atual e apresentado em um contêiner retangular que aparece próximo ao ponteiro.

</dd> </dl>

## <a name="q"></a>Q

<dl> <dt>

**Barra de ferramentas de acesso rápido (QAT)**
</dt> <dd>

Uma pequena barra de ferramentas personalizável na interface do usuário da faixa de bits que fornece acesso com um clique aos comandos selecionados pelo usuário.

</dd> </dl>

## <a name="r"></a>R

<dl> <dt>

**Marcação da faixa de medida**
</dt> <dd>

Marcação declarativa baseada em XML padrão que especifica os comandos, recursos visuais, layout de controle e comportamento de dimensionamento da interface do usuário da faixa de opção.

</dd> </dl>

## <a name="t"></a>T

<dl> <dt>

**modelos**
</dt> <dd>

Um conjunto de dicas de layout que especificam o tamanho e a organização dos controles da faixa de opção com base no tamanho da faixa de uma.

</dd> </dl>

## <a name="v"></a>V

<dl> <dt>

**Exibir**
</dt> <dd>

Um contêiner de nível superior que hospeda os controles da interface do usuário do Windows.

</dd> </dl>

## <a name="w"></a>W

<dl> <dt>

**Estrutura de faixa de medida do Windows (faixa de faixas)**
</dt> <dd>

Um sistema avançado de apresentação de comandos que expõe os principais recursos de um aplicativo por meio de uma barra de comandos na parte superior de uma janela de aplicativo.

</dd> </dl>
