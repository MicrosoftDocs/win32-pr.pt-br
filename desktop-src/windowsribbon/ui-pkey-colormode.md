# <a name="ui_pkey_colormode"></a>Interface do usuário \_ PKEY \_ ColorMode

Identifica a \_ Propriedade ColorMode da interface do usuário PKEY \_ .

```
propertyDescription
   name = UI_PKEY_ColorMode
   shellPKey = UI_PKEY_ColorMode
   formatID = 00000402-7363-696e-8441798acf5aebb7
   propID = 402
   typeInfo
      type = UI_SWATCHCOLORMODE
```

## <a name="remarks"></a>Comentários

O valor da propriedade é da [**enumeração \_ SWATCHCOLORMODE da interface do usuário**](/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode) .
