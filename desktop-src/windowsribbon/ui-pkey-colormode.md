# <a name="ui_pkey_colormode"></a>ColorMode \_ PKEY \_ da interface do usuário

Identifica a propriedade \_ ColorMode PKEY \_ da interface do usuário.

```
propertyDescription
   name = UI_PKEY_ColorMode
   shellPKey = UI_PKEY_ColorMode
   formatID = 00000402-7363-696e-8441798acf5aebb7
   propID = 402
   typeInfo
      type = UI_SWATCHCOLORMODE
```

## <a name="remarks"></a>Comentários

O valor da propriedade é da [**enumeração \_ SWATCHCOLORMODE da**](/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode) interface do usuário.
