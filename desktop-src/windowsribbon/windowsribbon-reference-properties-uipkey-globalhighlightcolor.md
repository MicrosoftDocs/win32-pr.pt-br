# <a name="ui_pkey_globalhighlightcolor"></a>\_GlobalHighlightColor PKEY \_ UI

Identifica a \_ Propriedade PKEY GlobalHighlightColor da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_GlobalHighlightColor
   shellPKey = UI_PKEY_GlobalHighlightColor
   formatID = 00002001-7363-696e-8441798acf5aebb7
   propID = 2001
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ GlobalHighlightColor é usada por um aplicativo para consultar a cor de realce global de uma faixa de faixas personalizada.

**Introduzido no Windows 8:**   A interface do usuário \_ PKEY \_ GlobalHighlightColor não pode ser definida independentemente da [interface do usuário \_ PKEY \_ GlobalBackgroundColor](windowsribbon-reference-properties-uipkey-globalbackgroundcolor.md).
