# <a name="ui_pkey_maxvalue"></a>Interface do usuário \_ PKEY \_ MaxValue

Identifica a \_ Propriedade PKEY MaxValue da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ MaxValue é usada por um aplicativo para consultar o valor máximo do controle [Spinner](windowsribbon-controls-spinner.md) .
