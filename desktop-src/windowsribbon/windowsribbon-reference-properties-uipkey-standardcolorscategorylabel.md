# <a name="ui_pkey_standardcolorscategorylabel"></a>UI \_ PKEY \_ StandardColorsCategoryLabel

Identifica a propriedade \_ PKEY \_ StandardColorsCategoryLabel da interface do usuário.

```
propertyDescription
   name = UI_PKEY_StandardColorsCategoryLabel
   shellPKey = UI_PKEY_StandardColorsCategoryLabel
   formatID = 00000404-7363-696e-8441798acf5aebb7
   propID = 404
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentários

UI \_ PKEY \_ StandardColorsCategoryLabel é usado por um aplicativo para consultar  o valor do rótulo associado à categoria Cores Padrão de [**um DropDownColorPicker.**](windowsribbon-element-dropdowncolorpicker.md)

A interface do usuário \_ PKEY StandardColorsCategoryLabel é válida somente para \_ um [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) em que é especificado como o valor do atributo `ThemeColors` **ColorTemplate** (esse é o único modelo que contém categorias rotuladas).

A captura de tela a seguir mostra `ThemeColors` [**um DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).
