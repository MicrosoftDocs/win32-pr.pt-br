# <a name="ui_pkey_globaltextcolor"></a>\_GlobalTextColor PKEY \_ UI

Identifica a \_ Propriedade PKey GlobalTextColor da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_GlobalTextColor
   shellPKey = UI_PKEY_GlobalTextColor
   formatID = 00002002-7363-696e-8441798acf5aebb7
   propID = 2002
   typeInfo
      type = VT_UINT
```

## <a name="remarks"></a>Comentários

A interface do usuário \_ PKEY \_ GlobalTextColor é usada por um aplicativo para consultar a cor do texto global de uma faixa de faixas personalizada.

Introduzido no Windows 8 **:** as alterações no valor padrão da [interface do usuário \_ PKEY \_ GlobalBackgroundColor](windowsribbon-reference-properties-uipkey-globalbackgroundcolor.md) no Windows 8 podem exigir um ajuste para a interface do usuário do \_ PKEY \_ GlobalTextColor em aplicativos da faixa de forma projetados para o Windows 7.
