# <a name="ui_pkey_fontproperties"></a>\_FontProperties PKEY \_ da interface do usuário

Identifica a propriedade \_ FontProperties PKEY da interface do \_ usuário.

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Comentários

FontProperties PKEY da interface do usuário é um armazenamento de propriedades usado por um aplicativo para \_ consultar todas as propriedades da \_ fonte.
