# <a name="ui_pkey_fontproperties"></a>Interface do usuário \_ PKEY \_ fontproperties

Identifica a \_ Propriedade fontproperties da interface do usuário PKEY \_ .

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Comentários

A interface de usuário \_ PKEY \_ fontproperties é um repositório de propriedades usado por um aplicativo para consultar todas as propriedades de fonte.
