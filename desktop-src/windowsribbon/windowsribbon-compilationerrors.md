# <a name="understanding-markup-compiler-messages"></a>Compreendendo as mensagens do compilador de marcação

O compilador de marcação do Windows Ribbon Framework (faixa de opções), o compilador de comando da interface do usuário (UICC.exe), valida a marcação da faixa de opções em relação ao esquema da faixa e um conjunto adicional de regras definidas pela estrutura da faixa de opções.

-   [Mensagens de validação](#validation-messages)
    -   [Erros de validação](#understanding-markup-compiler-messages)
    -   [Avisos de validação](#validation-warnings)
    -   [Mensagens de informações](#information-messages)
-   [Tópicos relacionados](#related-topics)

## <a name="validation-messages"></a>Mensagens de validação

As seguintes mensagens de validação são geradas durante a compilação. Consulte [compilando marcação da faixa](windowsribbon-intentcl.md)de medida para obter mais informações.

> [!Note]  
> A `/W{0\1\2}` opção de linha de comando pode ser especificada para filtrar mensagens de evento com base na severidade.

 

### <a name="validation-errors"></a>Erros de validação

Mensagens de evento de **erro** indicam problemas na marcação da faixa de faixas grave o suficiente para que o compilador descontinue o processo de compilação. Esses problemas devem ser resolvidos para que a compilação tenha sucesso.
