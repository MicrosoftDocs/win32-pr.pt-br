# <a name="ui_pkey_booleanvalue"></a>\_Valor booliano da interface do usuário PKEY \_

Identifica a \_ Propriedade boolianvalue da interface do usuário PKEY \_ .

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Comentários

O boolianvalue da interface do usuário \_ PKEY \_ é usado por um aplicativo para consultar o estado de alternância de uma [caixa de seleção](windowsribbon-controls-checkbox.md), o [botão de alternância](windowsribbon-controls-togglebutton.md)e o controle de botão de um [**SplitButtonGallery**](windowsribbon-element-splitbuttongallery.md).
