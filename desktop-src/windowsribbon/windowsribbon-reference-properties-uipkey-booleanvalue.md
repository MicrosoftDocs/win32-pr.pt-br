# <a name="ui_pkey_booleanvalue"></a>BooleanValue da interface do usuário \_ \_ PKEY

Identifica a propriedade \_ BooleanValue da IU \_ PKEY.

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Comentários

A interface do usuário PKEY BooleanValue é usada por um aplicativo para consultar o estado de alternância de uma caixa de seleção, botão de alternância e o controle de botão de \_ \_ um [](windowsribbon-controls-checkbox.md) [](windowsribbon-controls-togglebutton.md) [**SplitButtonGallery.**](windowsribbon-element-splitbuttongallery.md)
