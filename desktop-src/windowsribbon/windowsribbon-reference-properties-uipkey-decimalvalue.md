# <a name="ui_pkey_decimalvalue"></a>\_PKEY DecimalValue da interface do usuário \_

Identifica a \_ Propriedade PKEY DecimalValue da interface do usuário \_ .

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Comentários

\_O pkey DecimalValue da interface do usuário \_ é usado por um aplicativo para consultar o valor no campo de edição do controle [Spinner](windowsribbon-controls-spinner.md) .
