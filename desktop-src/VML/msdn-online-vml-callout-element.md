# <a name="vml-callout-element"></a>Elemento de saída de chamada VML

Este tópico descreve o VML, um recurso que foi preterido a partir Windows Internet Explorer 9. As páginas da Web e os aplicativos que dependem do VML devem ser migrados para o SVG ou outros padrões amplamente suportados.

> [!Note]  
> A partir de dezembro de 2011, este tópico foi arquivado. Como resultado, ele não é mais mantido ativamente. Para obter mais informações, consulte [Conteúdo arquivado.](/previous-versions/windows/internet-explorer/ie-developer/) Para obter informações, recomendações e diretrizes sobre a versão atual do Windows Internet Explorer, consulte [Internet Explorer Developer Center.](https://msdn.microsoft.com/ie/)

 

Define um callout para uma forma.

Os atributos a seguir modificam um texto explicado.



| Atributo                                                        | Descrição                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [AccentBar](msdn-online-vml-accentbar-attribute.md)             | Determina se uma barra de acentos será usada com o destaque.                         |
| [Ângulo](angle-attribute--callout--vml.md)                       | Define o ângulo que o texto explicação faz em relação à caixa delimitada da forma. |
| [Distância](msdn-online-vml-distance-attribute.md)               | Define a distância de soltar de um final de chamada.                                                 |
| [Gota](msdn-online-vml-drop-attribute.md)                       | Determina onde a queda de um callout será colocada.                                  |
| [DropAuto](msdn-online-vml-dropauto-attribute.md)               | Determina se o callout terá uma queda automática.                             |
| [Ext](ext-attribute--callout--vml.md)                           | Define como um callout é processado.                                                     |
| [Lacuna](msdn-online-vml-gap-attribute.md)                         | Define a distância do chamador do retângulo delimitado.                        |
| [Comprimento](msdn-online-vml-length-attribute.md)                   | Define o comprimento do texto explicado.                                                      |
| [LengthSpecified](msdn-online-vml-lengthspecified-attribute.md) | Determina se o **atributo Length** será usado para o texto explicado.               |
| [MinusX](msdn-online-vml-minusx-attribute.md)                   | Determina se o callout será in flip ao longo do eixo x.                              |
| [Subtração](msdn-online-vml-minusy-attribute.md)                   | Determina se o callout será in flip ao longo do eixo y.                              |
| [Ativado](on-attribute--callout--vml.md)                             | Determina se uma forma é um callout.                                                |
| [TextBorder](msdn-online-vml-textborder-attribute.md)           | Determina se um texto explicado terá uma borda de texto.                                   |
| [Tipo](type-attribute--callout--vml.md)                         | Define o tipo de texto explicado.                                                            |



 

**Comentários**

Esse elemento deve ser definido dentro de um [elemento Shape.](shape-element--vml.md)

O **elemento Callout** é uma extensão Microsoft Office para VML.
