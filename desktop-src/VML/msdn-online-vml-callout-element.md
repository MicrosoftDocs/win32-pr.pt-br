# <a name="vml-callout-element"></a>Elemento de texto explicativo do VML

Este tópico descreve a VML, um recurso que foi preterido a partir do Windows Internet Explorer 9. As páginas da Web e os aplicativos que dependem de VML devem ser migrados para o SVG ou outros padrões amplamente suportados.

> [!Note]  
> A partir de dezembro de 2011, este tópico foi arquivado. Como resultado, ele não é mais mantido ativamente. Para obter mais informações, consulte [conteúdo arquivado](/previous-versions/windows/internet-explorer/ie-developer/). Para obter informações, recomendações e orientações sobre a versão atual do Windows Internet Explorer, consulte [Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Define um texto explicativo para uma forma.

Os atributos a seguir modificam um texto explicativo.



| Atributo                                                        | Descrição                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [AccentBar](msdn-online-vml-accentbar-attribute.md)             | Determina se uma barra de ênfase será usada com o texto explicativo.                         |
| [Firmeza](angle-attribute--callout--vml.md)                       | Define o ângulo que o texto explicativo faz em relação à caixa delimitadora da forma. |
| [Distância](msdn-online-vml-distance-attribute.md)               | Define a distância de posicionamento de um texto explicativo.                                                 |
| [Suspensa](msdn-online-vml-drop-attribute.md)                       | Determina onde o descarte de um texto explicativo será colocado.                                  |
| [DropAuto](msdn-online-vml-dropauto-attribute.md)               | Determina se o texto explicativo terá um descarte automático.                             |
| [Externa](ext-attribute--callout--vml.md)                           | Define como um texto explicativo é processado.                                                     |
| [Defasa](msdn-online-vml-gap-attribute.md)                         | Define a distância do texto explicativo do seu retângulo delimitador.                        |
| [Comprimento](msdn-online-vml-length-attribute.md)                   | Define o comprimento do texto explicativo.                                                      |
| [LengthSpecified](msdn-online-vml-lengthspecified-attribute.md) | Determina se o atributo de **comprimento** será usado para o texto explicativo.               |
| [MinusX](msdn-online-vml-minusx-attribute.md)                   | Determina se o texto explicativo será alternado ao longo do eixo x.                              |
| [Subtração](msdn-online-vml-minusy-attribute.md)                   | Determina se o texto explicativo será invertido ao longo do eixo y.                              |
| [Ativado](on-attribute--callout--vml.md)                             | Determina se uma forma é um texto explicativo.                                                |
| [Textborder](msdn-online-vml-textborder-attribute.md)           | Determina se um texto explicativo terá uma borda de texto.                                   |
| [Tipo](type-attribute--callout--vml.md)                         | Define o tipo de texto explicativo.                                                            |



 

**Comentários**

Esse elemento deve ser definido dentro de um elemento [Shape](shape-element--vml.md) .

O elemento de **texto explicativo** é uma extensão Microsoft Office para VML.
