# <a name="vml-hralign-attribute"></a>Atributo HRAlign do VML

Este tópico descreve o VML, um recurso que foi preterido a partir Windows Internet Explorer 9. As páginas da Web e os aplicativos que dependem do VML devem ser migrados para o SVG ou outros padrões amplamente suportados.

> [!Note]  
> A partir de dezembro de 2011, este tópico foi arquivado. Como resultado, ele não é mais mantido ativamente. Para obter mais informações, consulte [Conteúdo arquivado.](/previous-versions/windows/internet-explorer/ie-developer/) Para obter informações, recomendações e diretrizes sobre a versão atual do Windows Internet Explorer, consulte [Internet Explorer Developer Center.](https://msdn.microsoft.com/ie/)

 

Define o alinhamento de uma regra horizontal. Leitura/gravação. **Cadeia de caracteres**.

**Aplica-se a**

[Forma](shape-element--vml.md)

**Sintaxe de marca**

<v: *elemento* o:hralign=" *expressão* ">

**Comentários**

Os valores são:



| Valor  | Descrição                                            |
|--------|--------------------------------------------------------|
| Esquerda   | A regra é liberada com a margem esquerda. Valor padrão. |
| Centro | A regra é centralizada na página.                      |
| Direita  | A regra é liberada com a margem direita.               |
