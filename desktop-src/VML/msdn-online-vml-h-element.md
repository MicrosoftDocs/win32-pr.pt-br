# <a name="vml-h-element"></a>Elemento VML H

Este tópico descreve a VML, um recurso que foi preterido a partir do Windows Internet Explorer 9. As páginas da Web e os aplicativos que dependem de VML devem ser migrados para o SVG ou outros padrões amplamente suportados.

> [!Note]  
> A partir de dezembro de 2011, este tópico foi arquivado. Como resultado, ele não é mais mantido ativamente. Para obter mais informações, consulte [conteúdo arquivado](/previous-versions/windows/internet-explorer/ie-developer/). Para obter informações, recomendações e orientações sobre a versão atual do Windows Internet Explorer, consulte [Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Define um identificador para uma forma.

Os atributos a seguir modificam um identificador.



| Atributo                                                | Descrição                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [InvX](msdn-online-vml-invx-attribute.md)               | Determina se a posição *x* do identificador é invertida. |
| [InvY](msdn-online-vml-invy-attribute.md)               | Determina se a posição *y* do identificador é invertida. |
| [Map](msdn-online-vml-map-attribute.md)                 | Define o intervalo de mapeamento de um identificador.                         |
| [Devidamente](msdn-online-vml-polar-attribute.md)             | Define a posição central para identificadores polares.                 |
| [Posição](position-attribute--h--vml.md)               | Especifica os valores *x* e *y* do identificador.                |
| [RadiusRange](msdn-online-vml-radiusrange-attribute.md) | Define o intervalo de um identificador polar.                           |
| [Switch](msdn-online-vml-switch-attribute.md)           | Determina se as direções do identificador são trocadas.          |
| [XRange](msdn-online-vml-xrange-attribute.md)           | Define o intervalo *x* de um identificador.                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | Define o intervalo *y* de um identificador.                             |



 

**Comentários**

Esse elemento deve ser definido dentro de um elemento [Shape](shape-element--vml.md) .
