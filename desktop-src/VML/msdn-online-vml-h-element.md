# <a name="vml-h-element"></a>Elemento H do VML

Este tópico descreve o VML, um recurso que foi preterido a partir Windows Internet Explorer 9. As páginas da Web e os aplicativos que dependem do VML devem ser migrados para o SVG ou outros padrões amplamente suportados.

> [!Note]  
> A partir de dezembro de 2011, este tópico foi arquivado. Como resultado, ele não é mais mantido ativamente. Para obter mais informações, consulte [Conteúdo arquivado.](/previous-versions/windows/internet-explorer/ie-developer/) Para obter informações, recomendações e diretrizes sobre a versão atual do Windows Internet Explorer, consulte [Internet Explorer Developer Center.](https://msdn.microsoft.com/ie/)

 

Define um alça para uma forma.

Os atributos a seguir modificam um handle.



| Atributo                                                | Descrição                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [InvX](msdn-online-vml-invx-attribute.md)               | Determina se a *posição x* do alça é invertida. |
| [InvY](msdn-online-vml-invy-attribute.md)               | Determina se a posição *y* do alça é invertida. |
| [Map](msdn-online-vml-map-attribute.md)                 | Define o intervalo de mapeamento de um alça.                         |
| [Polar](msdn-online-vml-polar-attribute.md)             | Define a posição central para alças polares.                 |
| [Posição](position-attribute--h--vml.md)               | Especifica os *valores x* e *y* do handle.                |
| [RadiusRange](msdn-online-vml-radiusrange-attribute.md) | Define o intervalo de uma alça polar.                           |
| [Switch](msdn-online-vml-switch-attribute.md)           | Determina se as instruções de alça são trocadas.          |
| [XRange](msdn-online-vml-xrange-attribute.md)           | Define o *intervalo x* de um handle.                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | Define o *intervalo y* de um handle.                             |



 

**Comentários**

Esse elemento deve ser definido dentro de um [elemento Shape.](shape-element--vml.md)
