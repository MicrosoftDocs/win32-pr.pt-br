# <a name="allowincell-attribute"></a>Atributo AllowInCell

Determina se uma forma pode ser colocada em uma tabela. Leitura/gravação. **VgTriState**.

**Aplica-se a**

[Forma](shape-element--vml.md)

**Sintaxe de marca**

v:*Element* o:allowincell = "*expressão*" >

**Comentários**

Usado pelo Microsoft Word. O valor padrão é **Falso**.

*Atributo de extensões de Microsoft Office*
