# <a name="journal-reader-namespace-elements"></a>Elementos do namespace do leitor de diário

Esta seção documenta os elementos de namespace XML usados pelo componente de leitor de diário.

## <a name="in-this-section"></a>Nesta seção



| Elemento                                                                   | Descrição                                                                                                                                                                                                                                                                                                                |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Elemento alternativo](alternate-element.md)                                | Contém uma alternativa de reconhecimento para um [elemento InkWord](inkword-element.md).<br/>                                                                                                                                                                                                                                 |
| [Elemento alternativolist](alternatelist-element.md)                        | Contém o conjunto de [elementos alternativos](alternate-element.md)que contêm as alternativas retornadas pelo reconhecedor para o elemento pai [InkWord](inkword-element.md).<br/>                                                                                                                                      |
| [Elemento background](background-element.md)                              | Contém o plano de fundo de um elemento [JournalDocument](journaldocument-element.md) ou de um [elemento JournalPage](journalpage-element.md).<br/>                                                                                                                                                                         |
| [Elemento de linha de base](baseline-element.md)                                  | Fornece informações (x, y) para os pontos inicial e final da linha de base de um parágrafo no documento do diário. O espaço de coordenadas usado para esses valores é HIMETRIC.<br/>                                                                                                                                  |
| [Elemento bitmap](bitmap-element.md)                                      | Contém os dados codificados em base64 para um bitmap contido no arquivo de diário como uma imagem de plano de fundo.<br/>                                                                                                                                                                                                              |
| [Elemento CanReClassify](canreclassify-element.md)                        | Quando **true**, indica que a tinta pode ser analisada novamente. Esse elemento será **false** se o usuário tiver definido explicitamente o tipo de tinta usando converter manuscrito em texto ou alterar forma para recursos do diário.<br/>                                                                                     |
| [Elemento de confiança](confidence-element.md)                              | Contém a classificação de confiança retornada pelo analisador de tinta do diário para o [elemento InkWord](inkword-element.md).<br/>                                                                                                                                                                                             |
| [Leitor de diário de elementos de conteúdo \[\]](content-element--journal-reader.md) | Contém o conteúdo de uma página de diário.<br/>                                                                                                                                                                                                                                                                        |
| [Elemento Date](date-element.md)                                          | Contém a data em que a nota do diário foi criada.<br/>                                                                                                                                                                                                                                                                 |
| [Elemento DocImage](docimage-element.md)                                  | Se a nota do diário que está sendo lida tiver sido criada usando o driver de impressora do gravador de notas do diário, a imagem do documento em segundo plano será mantida nesse elemento.<br/>                                                                                                                                                         |
| [Elemento Drawing](drawing-element.md)                                    | Contém o conteúdo que foi classificado pelo analisador ou pelo usuário como um desenho.<br/>                                                                                                                                                                                                                             |
| [Elemento Flag](flag-element.md)                                          | Contém um bitmap codificado em Base64 de um sinalizador associado à seção de uma nota do diário.<br/>                                                                                                                                                                                                                           |
| [Elemento GroupNode](groupnode-element.md)                                | Contém um conjunto de elementos-[elemento Paragraph](paragraph-element.md), [elemento InkWord](inkword-element.md), [elemento de desenho](drawing-element.md), [elemento de texto](text-element.md), elemento [Image](image-element.md)ou [elemento Flag](flag-element.md), que são agrupados na nota do diário.<br/> |
| [Elemento horizontal](horizontal-element.md)                              | Contém informações de formatação para as linhas horizontais usadas para o papel de carta em uma nota do diário.<br/>                                                                                                                                                                                                             |
| [Elemento Image](image-element.md)                                        | Contém dados binários codificados para uma imagem no documento do diário, se presente.<br/>                                                                                                                                                                                                                                  |
| [Elemento InkClass](inkclass-element.md)                                  | Representa os traços de tinta coletados em um espaço de tinta.<br/>                                                                                                                                                                                                                                                    |
| [Elemento InkObject](inkobject-element.md)                                | Contém os dados binários codificados em base64 do objeto de [**tinta**](inkdisp-class.md) para um [elemento InkWord](inkword-element.md) ou [elemento de desenho](drawing-element.md).<br/>                                                                                                                                     |
| [Elemento InkWord](inkword-element.md)                                    | Contém informações sobre uma determinada palavra de tinta na nota do diário, incluindo posição, alternativas e dados reais de tinta.<br/>                                                                                                                                                                                       |
| [Elemento JournalDocument](journaldocument-element.md)                    | O elemento de nível superior em um arquivo XML de nota do diário.<br/>                                                                                                                                                                                                                                                               |
| [Elemento JournalPage](journalpage-element.md)                            | Contém os detalhes sobre uma página individual em uma nota do diário.<br/>                                                                                                                                                                                                                                                |
| [Elemento de linha](line-element.md)                                          | Contém uma linha dentro de um [elemento de parágrafo](paragraph-element.md).<br/>                                                                                                                                                                                                                                            |
| [Elemento LineLayout](linelayout-element.md)                              | Contém informações sobre o layout das linhas usadas no papel de carta para a nota do diário.<br/>                                                                                                                                                                                                                     |
| [Elemento Margin](margin-element.md)                                      | Define as linhas de margem desenhadas na página.<br/>                                                                                                                                                                                                                                                                     |
| [Elemento Paragraph](paragraph-element.md)                                | Contém um parágrafo.<br/>                                                                                                                                                                                                                                                                                           |
| [Elemento ParagraphLineMetrics](paragraphlinemetrics-element.md)          | Contém informações sobre as linhas de um parágrafo.<br/>                                                                                                                                                                                                                                                            |
| [Elemento Path](path-element.md)                                          | O caminho para a imagem usada para o plano de fundo de uma nota do diário.<br/>                                                                                                                                                                                                                                                |
| [Elemento de refluxo](reflow-element.md)                                      | Indica que o grupo foi refluído.<br/>                                                                                                                                                                                                                                                                          |
| [Elemento ScalarTransform](scalartransform-element.md)                    | Transformação escalar usada pelo desenho ou InkWord.<br/>                                                                                                                                                                                                                                                                |
| [Elemento de papel de carta](stationery-element.md)                              | Contém informações sobre o papel de carta usado na nota do diário, incluindo a cor ou a imagem do plano de fundo, as informações de título e as informações de layout da linha, como regras estreitas e regras de margem.<br/>                                                                                                     |
| [Elemento de texto](text-element.md)                                          | Contém informações de texto da nota do diário.<br/>                                                                                                                                                                                                                                                                |
| [Elemento title](title-element.md)                                        | Contém informações de título sobre a nota do diário.<br/>                                                                                                                                                                                                                                                              |
| [Elemento TitleArea](titlearea-element.md)                                | Contém informações de local e delimitador para o [elemento de título](title-element.md)da nota, se presente.<br/>                                                                                                                                                                                                         |
| [Elemento TitleInfo](titleinfo-element.md)                                | Contém o título da observação e a data no título, se presente.<br/>                                                                                                                                                                                                                                           |
| [Elemento vertical](vertical-element.md)                                  | Contém as informações que descrevem o componente vertical das linhas no papel de carta usado pela anotação do diário.<br/>                                                                                                                                                                                         |
