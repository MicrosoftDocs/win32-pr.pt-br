# <a name="journal-reader-namespace-elements"></a>Elementos de namespace do leitor de diário

Esta seção documenta os elementos de namespace XML usados pelo componente Leitor de Diário.

## <a name="in-this-section"></a>Nesta seção



| Elemento                                                                   | Descrição                                                                                                                                                                                                                                                                                                                |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Elemento alternativo](alternate-element.md)                                | Contém uma alternativa de reconhecimento para um [elemento InkWord.](inkword-element.md)<br/>                                                                                                                                                                                                                                 |
| [Elemento AlternateList](alternatelist-element.md)                        | Contém o conjunto de [elementos alternativos](alternate-element.md)que contêm as alternativas retornadas pelo reconhecedor para o [elemento pai InkWord.](inkword-element.md)<br/>                                                                                                                                      |
| [Elemento Background](background-element.md)                              | Contém a plano de fundo de [um elemento JournalDocument ou](journaldocument-element.md) elemento [JournalPage](journalpage-element.md).<br/>                                                                                                                                                                         |
| [Elemento Baseline](baseline-element.md)                                  | Fornece (x, y) informações para os pontos inicial e final da linha de base de um parágrafo no documento do Diário. O espaço de coordenadas usado para esses valores é HIMETRIC.<br/>                                                                                                                                  |
| [Elemento Bitmap](bitmap-element.md)                                      | Contém os dados codificados em Base64 para um bitmap contido no arquivo De diário como uma imagem em segundo plano.<br/>                                                                                                                                                                                                              |
| [Elemento CanReClassify](canreclassify-element.md)                        | Quando **true**, indica que a tinta pode ser re analisada. Esse elemento será **false** se o usuário tiver definido explicitamente o tipo da tinta usando os recursos Converter Manuscrito em Texto ou Alterar Forma para de Diário.<br/>                                                                                     |
| [Elemento Confidence](confidence-element.md)                              | Contém a classificação de confiança retornada pelo analisador de tinta do Diário para o [elemento InkWord](inkword-element.md).<br/>                                                                                                                                                                                             |
| [Leitor de \[ Diário do elemento de conteúdo\]](content-element--journal-reader.md) | Contém o conteúdo de uma página De diário.<br/>                                                                                                                                                                                                                                                                        |
| [Elemento Date](date-element.md)                                          | Contém a data em que a nota Do diário foi criada.<br/>                                                                                                                                                                                                                                                                 |
| [Elemento DocImage](docimage-element.md)                                  | Se a nota de diário que está sendo lida tiver sido criada usando o driver de impressora do Journal Note Writer, a imagem do documento em segundo plano será mantida nesse elemento.<br/>                                                                                                                                                         |
| [Elemento Drawing](drawing-element.md)                                    | Contém o conteúdo que foi classificado pelo analisador ou pelo usuário como um desenho.<br/>                                                                                                                                                                                                                             |
| [Elemento Flag](flag-element.md)                                          | Contém um bitmap codificado em base64 de um sinalizador associado à seção de uma nota de diário.<br/>                                                                                                                                                                                                                           |
| [Elemento GroupNode](groupnode-element.md)                                | Contém um conjunto de elementos - elemento[Paragraph](paragraph-element.md), elemento [InkWord](inkword-element.md) [,](text-element.md)elemento [Drawing](drawing-element.md), elemento Text , [elemento Image](image-element.md)ou elemento [Flag](flag-element.md)- que são agrupados na nota Journal.<br/> |
| [Elemento horizontal](horizontal-element.md)                              | Contém informações de formatação para as linhas horizontais usadas para a papelaria em uma nota de Diário.<br/>                                                                                                                                                                                                             |
| [Elemento Image](image-element.md)                                        | Contém dados binários codificados para uma imagem no documento Diário, se presente.<br/>                                                                                                                                                                                                                                  |
| [Elemento InkClass](inkclass-element.md)                                  | Representa os traços de tinta coletados dentro de um espaço de tinta.<br/>                                                                                                                                                                                                                                                    |
| [Elemento InkObject](inkobject-element.md)                                | Contém os dados binários codificados em Base64 do objeto [**Ink**](inkdisp-class.md) para um [elemento InkWord ou](inkword-element.md) elemento [Drawing](drawing-element.md).<br/>                                                                                                                                     |
| [Elemento InkWord](inkword-element.md)                                    | Contém informações sobre uma determinada palavra de tinta na nota Do diário, incluindo posição, alternativas e os dados reais de tinta.<br/>                                                                                                                                                                                       |
| [Elemento JournalDocument](journaldocument-element.md)                    | O elemento de nível superior em um arquivo XML de observação de diário.<br/>                                                                                                                                                                                                                                                               |
| [Elemento JournalPage](journalpage-element.md)                            | Contém os detalhes sobre uma página individual em uma nota de Diário.<br/>                                                                                                                                                                                                                                                |
| [Elemento Line](line-element.md)                                          | Contém uma linha dentro de um [elemento Paragraph](paragraph-element.md).<br/>                                                                                                                                                                                                                                            |
| [Elemento LineLayout](linelayout-element.md)                              | Contém informações sobre o layout das linhas usadas na papelaria para a nota Do diário.<br/>                                                                                                                                                                                                                     |
| [Elemento Margin](margin-element.md)                                      | Define as linhas de margem desenhadas na página.<br/>                                                                                                                                                                                                                                                                     |
| [Elemento Paragraph](paragraph-element.md)                                | Contém um parágrafo.<br/>                                                                                                                                                                                                                                                                                           |
| [Elemento ParagraphLineMetrics](paragraphlinemetrics-element.md)          | Contém informações sobre as linhas de um parágrafo.<br/>                                                                                                                                                                                                                                                            |
| [Elemento Path](path-element.md)                                          | O caminho para a imagem usada para a plano de fundo de uma nota de Diário.<br/>                                                                                                                                                                                                                                                |
| [Elemento Reflow](reflow-element.md)                                      | Indica que o grupo foi refluido.<br/>                                                                                                                                                                                                                                                                          |
| [Elemento ScalarTransform](scalartransform-element.md)                    | Transformação escalar usada pelo desenho ou inkWord.<br/>                                                                                                                                                                                                                                                                |
| [Elemento Stationery](stationery-element.md)                              | Contém informações sobre a papelaria usada na observação do diário, incluindo a cor da tela de fundo ou a imagem, as informações de título e as informações de layout de linha, como regras estreitas e regras de margem.<br/>                                                                                                     |
| [Elemento Text](text-element.md)                                          | Contém informações de texto da nota Diário.<br/>                                                                                                                                                                                                                                                                |
| [Elemento Title](title-element.md)                                        | Contém informações de título sobre a nota Do diário.<br/>                                                                                                                                                                                                                                                              |
| [Elemento TitleArea](titlearea-element.md)                                | Contém informações de localização e delimitadores para o elemento [Title da nota](title-element.md), se presente.<br/>                                                                                                                                                                                                         |
| [Elemento TitleInfo](titleinfo-element.md)                                | Contém o título da nota e a data no título, se presente.<br/>                                                                                                                                                                                                                                           |
| [Elemento vertical](vertical-element.md)                                  | Contém as informações que descrevem o componente vertical das linhas na papelaria usada pela observação Do diário.<br/>                                                                                                                                                                                         |
