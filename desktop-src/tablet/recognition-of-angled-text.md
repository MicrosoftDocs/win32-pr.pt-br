# <a name="recognition-of-angled-text"></a>Reconhecimento de texto angular

Os reconhecedores da Microsoft lidam com a escrita em um ângulo de +/-20 graus, mas a precisão é prejudicada à medida que a gravação se desvia mais da horizontal ideal. Para escrita de tinta em mais de vinte graus do eixo horizontal, é recomendável que o desenvolvedor gire a tinta de volta para horizontal antes de passar para o reconhecedor.
