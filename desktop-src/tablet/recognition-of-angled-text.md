# <a name="recognition-of-angled-text"></a>Reconhecimento de texto emaranhado

Os reconhecedores da Microsoft lidam com a escrita em um ângulo de +/- 20 graus, mas a precisão diminui à medida que a escrita se desvia mais do ideal horizontal. Para tinta escrita a mais de 20 graus do eixo horizontal, é recomendável que o desenvolvedor gire a tinta de volta para horizontal antes de passar para o reconhecedor.
