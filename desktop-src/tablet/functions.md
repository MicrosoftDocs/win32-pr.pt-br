# <a name="core-tablet-pc-functions"></a>Funções principais do tablet pc

Esta seção contém as funções principais para Tablet PC.

## <a name="in-this-section"></a>Nesta seção



| Função                                                         | Descrição                                                                                                                                                                                         |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddOneStrke**](addonestroke.md)                             | Adiciona um novo [**objeto IInkStrkeDisp**](/windows/desktop/api/msinkaut/nn-msinkaut-iinkstrokedisp) [**ao objeto InkDivider**](inkdivider-class.md) que foi passado.                                                                 |
| [**CallDivide**](calldivide.md)                                 | Recupera informações de análise do [**objeto InkDivider.**](inkdivider-class.md)                                                                                                              |
| [**CallDivideResults**](calldivideresults.md)                   | Retorna resultados de análise do [**objeto InkDivider.**](inkdivider-class.md)                                                                                                                    |
| [**CallDivideResultsRogkeIds**](calldivideresultsstrokeids.md) | Recupera as [**propriedades de ID**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkstrokedisp-get_id) para os [**objetos IInkStrkeDisp**](/windows/desktop/api/msinkaut/nn-msinkaut-iinkstrokedisp) da palavra, linha, parágrafo ou desenho correspondentes determinados pela análise de tinta. |
| [**CreateInkDivider**](createinkdivider.md)                     | Instalita uma implementação da interface [**InkDivider**](inkdivider-class.md) e retorna seu handle.                                                                                      |
| [**DeleteInkDivider**](deleteinkdivider.md)                     | Exclui um [**objeto InkDivider**](inkdivider-class.md) e libera recursos associados.                                                                                                         |
| [**InvokeIDispatch**](invokeidispatch.md)                       | Invoca a funcionalidade auxiliar para a interface IDispatch.                                                                                                                                           |
| [**RecognizerContextSet**](recognizercontextset.md)             | Testa se o [**objeto InkDivider**](inkdivider-class.md) pode usar a [**classe InkRecognizerContext**](inkrecognizercontext-class.md) para analisar palavras.                                      |
| [**RemoveStrkes**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkstrokes-removestrokes)                | Remove [**objetos IInkStrkeDisp**](/windows/desktop/api/msinkaut/nn-msinkaut-iinkstrokedisp) do [**InkDivider.**](inkdivider-class.md)                                                                                           |
| [**SetLineHeight**](setlineheight.md)                           | Define a [**propriedade LineHeight**](/windows/win32/api/msinkaut15/nf-msinkaut15-iinkdivider-get_lineheight) no [**objeto InkDivider.**](inkdivider-class.md)                                                                                 |
| [**SetLineRecoCallback**](setlinerecocallback.md)               | Define uma função de retorno de chamada a ser usada durante o reconhecimento de linha.                                                                                                                                        |
