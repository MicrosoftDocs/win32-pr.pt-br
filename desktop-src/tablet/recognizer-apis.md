# <a name="recognizer-reference"></a>Referência do reconhecedor

O reconhecedor manipula:

-   Determine quais reconhecedores estão disponíveis no computador.
-   Identifique o reconhecedor a ser usado.
-   Crie um contexto de reconhecedor.
-   Recuperar resultados e alternativas do reconhecedor.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                      | Descrição                                                                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Identificador de HRECOALT](hrecoalt-handle.md)                     | Recupera a cadeia de caracteres alternativa e os valores de propriedade, como o nível de confiança e as métricas.<br/>                                                                  |
| [Identificador de HRECOCONTEXT](hrecocontext-handle.md)             | Adiciona tinta ao contexto, executa o reconhecimento de tinta e recupera o resultado de reconhecimento e as alternativas.<br/>                                                    |
| [Identificador de HRECOGNIZER](hrecognizer-handle.md)               | Cria um contexto de reconhecedor, recupera seus atributos e propriedades e determina quais propriedades de pacote o reconhecedor precisa para executar o reconhecimento.<br/> |
| [Identificador de HRECOWORDLIST](hrecowordlist-handle.md)           | Gerencia uma lista de palavras anexada a um contexto de reconhecedor. Essa lista melhora os resultados de reconhecimento.<br/>                                                   |
| [Enumerações do reconhecedor](recognizer-api-enumerations.md) | Descreve os tipos de enumeração de reconhecedor.<br/>                                                                                                                |
| [Estruturas do reconhecedor](recognizer-api-structures.md)     | Descreve as estruturas do reconhecedor.<br/>                                                                                                                       |
| [GUIDs do reconhecedor](recognizer-guids.md)                   | Especifica o tipo de propriedade para propriedades do pacote e propriedades do reconhecedor.<br/>                                                                            |
