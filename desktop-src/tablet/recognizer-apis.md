# <a name="recognizer-reference"></a>Referência do reconhecedor

O reconhecedor lida com:

-   Determine quais reconhecedores estão disponíveis no computador.
-   Identificar qual reconhecedor usar.
-   Crie um contexto de reconhecedor.
-   Recuperar resultados e alternativas do reconhecedor.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                      | Descrição                                                                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Alça HRECOALT](hrecoalt-handle.md)                     | Recupera a cadeia de caracteres e os valores de propriedade alternativos, como nível de confiança e métricas.<br/>                                                                  |
| [Alça HRECOCONTEXT](hrecocontext-handle.md)             | Adiciona tinta ao contexto, executa o reconhecimento de tinta e recupera o resultado do reconhecimento e as alternativas.<br/>                                                    |
| [Alça HRECOGNIZER](hrecognizer-handle.md)               | Cria um contexto de reconhecedor, recupera seus atributos e propriedades e determina quais propriedades de pacote o reconhecedor precisa para executar o reconhecimento.<br/> |
| [Alça HRECOWORDLIST](hrecowordlist-handle.md)           | Gerencia uma lista de palavras que você anexa a um contexto de reconhecedor. Essa lista melhora os resultados do reconhecimento.<br/>                                                   |
| [Enumerações do reconhecedor](recognizer-api-enumerations.md) | Descreve os tipos de enumeração do reconhecedor.<br/>                                                                                                                |
| [Estruturas do reconhecedor](recognizer-api-structures.md)     | Descreve as estruturas do reconhecedor.<br/>                                                                                                                       |
| [GUIDs do Reconhecedor](recognizer-guids.md)                   | Especifica o tipo de propriedade para propriedades de pacote e propriedades do reconhecedor.<br/>                                                                            |
