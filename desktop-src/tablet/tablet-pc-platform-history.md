# <a name="tablet-pc-platform-history"></a>Histórico da plataforma de tablet pc

Esta seção descreve o histórico de versão da Tecnologia de Tablet PC.

## <a name="platform-description"></a>Descrição da plataforma

Os componentes da Tecnologia de Tablet PC permitem que você desenvolva aplicativos projetados para hardware de Tablet PC.

Esses componentes podem ser usados para criar aplicativos executados no Windows XP Tablet PC Edition 2005, Windows Vista, Windows 7, Windows Server 2008 R2 e alguns outros sistemas operacionais anteriores.

O Windows Presentation Foundation também dá suporte ao desenvolvimento de aplicativos de Tablet PC.

## <a name="version-history"></a>Histórico de versão

### <a name="tablet-platform-in-windows-7-and-windows-server-2008-r2"></a>Plataforma de tablet no Windows 7 e Windows Server 2008 R2

Windows 7 apresenta suporte a idiomas adicionais, o controle de entrada matemática e dicionários personalizados. [Windows recursos do Touch](../wintouch/windows-touch-portal.md) também foram adicionados aos Windows operacionais.

Windows O Server 2008 R2 introduz suporte para idiomas adicionais, dicionários personalizados e reconhecimento do lado do servidor.

Os recursos a seguir aprimoram a funcionalidade do Tablet e permitem que os desenvolvedores entreguem novos aplicativos que oferecem suporte a cenários práticos para usuários finais.

-   O controle de entrada matemática permite a entrada de fórmulas matemáticas e funções de texto manuscrito Windows 7.
-   Em um esforço para melhorar o reconhecimento de texto, Windows 7 e Windows Server 2008 R2 suportam dicionários personalizados para que os administradores possam habilitar diretamente o suporte para listas de palavras.
-   Windows O Toque dá suporte à entrada de várias fontes de toque por meio de novas mensagens de janela, além de uma API de reconhecimento de gestos que dá suporte a panorâmico, zoom e rotação.
-   Windows O Server 2008 R2 dá suporte ao reconhecimento do lado do servidor da entrada de formulário e também dá suporte a dicionários personalizados para reconhecimento do lado do servidor. Com a adição desses recursos, os desenvolvedores e administradores podem criar e personalizar aplicativos avançados que suportam o reconhecimento de manuscrito do lado do servidor.

### <a name="tablet-platform-in-windows-vista"></a>Plataforma de tablet no Windows Vista

Os binários da Tecnologia de Tablet PC foram atualizados no Windows Vista. A Tecnologia de Tablet PC atualizada inclui novas APIs de análise de tinta e uma versão COM das APIs de Entrada da Caneta. Aplicativos escritos em versões anteriores da Tecnologia de Tablet PC são executados Windows Vista sem modificação.

Os componentes de Tecnologia de Tablet PC estão disponíveis apenas como parte do SDK do Windows a partir do lançamento do Windows Vista. Para obter mais informações, consulte [Novidades no desenvolvimento de tablets.](what-s-new-in-tablet-pc-development.md)

### <a name="version-17"></a>Versão 1.7

O Tablet PC Development Kit 1.7 estendeu a funcionalidade de desenvolvimento além da disponível na versão 1.5, adicionando as APIs de Entrada da Caneta e o suporte para tinta na Web.

No passado, a funcionalidade da versão 1.5 estava em um binário separado, mas no Tablet PC Development Kit 1.7, todas as funcionalidades foram colocadas em um único binário.

### <a name="version-15"></a>Versão 1.5

A versão 1.5 do Kit de Desenvolvimento foi superada na versão 1.1. Ele forneceu APIs de painel de entrada de caneta e o objeto Divisor.

> [!Note]  
> Se você instalar o Tablet PC Development Kit versão 1.5, após a instalação, deverá restabelecer referências ao assembly de tinta da Microsoft em aplicativos escritos em versões anteriores do SDK do Tablet PC antes de compilar e executar.

 

### <a name="version-11"></a>Versão 1.1

A versão 1.1 do Kit de Desenvolvimento foi superada na versão 1.0. A versão 1.1 consistia apenas em atualizações para a documentação; os binários da plataforma não foram alterados entre a versão 1.1 do Kit de Desenvolvimento e a versão enviada do Windows XP Tablet PC Edition. Portanto, os aplicativos desenvolvidos no Kit de Desenvolvimento da versão 1.1 não precisam redistribuir nenhum componente para usar recursos de plataforma quando instalados em PCs tablet.

> [!Note]  
> Os aplicativos que estão sendo distribuídos para sistemas operacionais não Tablet PC podem usar um subconjunto de recursos de plataforma de tablet pc. Esses aplicativos devem incluir o módulo de mesclagem redistribuído enviado no Kit de Desenvolvimento versão 1.1 com sua configuração.

 

### <a name="version-10"></a>Versão 1.0

A versão 1.0 do Kit de Desenvolvimento foi superada pela versão 1.1.
