# <a name="exposing-the-contents-of-a-window"></a>Expondo o conteúdo de uma janela

Um aplicativo deve expor o conteúdo de classes de janela personalizadas que usam o Microsoft Acessibilidade Ativa. A janela inteira deve ser tratada como um controle personalizado, que é um contêiner para outros controles padrão, personalizados ou personalizados.

Para obter mais informações sobre como expor o conteúdo de janelas personalizadas usando Acessibilidade Ativa, consulte a seção [acessibilidade](/windows/desktop/accessibility) .
