# <a name="ink-interoperability"></a>Interoperabilidade de tinta

A persistência e a interoperabilidade são dois recursos extremamente importantes, especialmente para aplicativos que usam tinta. Para habilitar a persistência e a interoperabilidade de tinta, um aplicativo deve permitir que os usuários:

-   Salve e abra seus documentos com fidelidade total de dados de tinta.
-   Mover dados de e para outros aplicativos baseados no Microsoft Windows usando a área de transferência.

As seções a seguir detalham a interoperabilidade de tinta.

-   [Formatos de dados de tinta](ink-data-formats.md)
-   [Cenários importantes de interoperabilidade](important-interoperability-scenarios.md)
-   [Formatos de persistência](persistence-formats.md)
-   [Escolhendo quais formatos dar suporte](choosing-which-formats-to-support.md)
-   [Blocos de construção](building-blocks.md)
-   [Armazenando tinta em HTML](storing-ink-in-html.md)

Este documento pressupõe uma compreensão detalhada da área de transferência. Para obter mais informações sobre a área de transferência, consulte a [documentação da área de transferência](/windows/desktop/dataxchg/clipboard).
