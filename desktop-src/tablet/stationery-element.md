# <a name="stationery-element"></a>Elemento de papel de carta

Contém informações sobre o papel de carta usado na nota do diário, incluindo a cor ou a imagem do plano de fundo, as informações de título e as informações de layout da linha, como regras estreitas e regras de margem.

## <a name="definition"></a>Definição

``` syntax
<xs:element name="Stationery" type="StationeryType" minOccurs="0" maxOccurs="unbounded" />
```

## <a name="parent-elements"></a>Elementos pai

[**JournalDocument**](journaldocument-element.md)

## <a name="child-elements"></a>Elementos filho

[**Segundo plano**](background-element.md)

[**Título**](title-element.md)

[**LineLayout**](linelayout-element.md)

## <a name="attributes"></a>Atributos

Nenhum.

## <a name="element-information"></a>Informações do elemento



|  Elemento     | Valor                                                             |
|--------------|-------------------------------------------------------------------|
| Tipo de elemento | ComplexType [**StationeryType**](stationerytype-complex-type.md) |
| Namespace    | urn: esquemas-Microsoft-com: Tablet: RichInk                        |
| Nome do esquema  | Leitor de diário                                                    |
