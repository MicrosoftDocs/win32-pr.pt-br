# <a name="exposing-text-programmatically"></a>Expondo texto programaticamente

Expor o texto programaticamente fornece um excelente benefício para as ajudas de acessibilidade, fornecendo uma interpretação muito mais confiável e, em alguns casos, de mais sofisticada das informações.

Há vários métodos básicos de expor objetos para expor texto programaticamente. No entanto, nenhum método é eficaz com todos os auxílios de acessibilidade, portanto, também é necessário desenhar texto na tela para que possa ser reconhecido. Para obter informações sobre como expor texto de grandes quantidades, consulte [usando acessibilidade ativa para expor texto](using-active-accessibility-to-expose-text.md).
