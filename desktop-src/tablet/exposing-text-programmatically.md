# <a name="exposing-text-programmatically"></a>Expondo texto programaticamente

Expor texto programaticamente fornece um ótimo benefício para os auxílios de acessibilidade, fornecendo uma interpretação muito mais confiável e, em alguns casos, mais rica das informações.

Há vários métodos básicos de expor objetos para expor o texto programaticamente. No entanto, nenhum método é eficaz com todos os auxílios de acessibilidade, portanto, também é necessário desenhar texto para a tela para que ele possa ser reconhecido. Para obter informações sobre como expor texto de grandes quantidades, consulte [Using Acessibilidade Ativa to Expose Text](using-active-accessibility-to-expose-text.md).
