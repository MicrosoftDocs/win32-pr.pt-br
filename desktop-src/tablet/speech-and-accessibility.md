# <a name="speech-and-accessibility"></a>Fala e acessibilidade

Para que os aplicativos sejam acessíveis, as informações sobre o conteúdo da tela devem ser expostas usando os utilitários de reconhecimento de voz. Esses utilitários precisam identificar e nomear controles específicos para que possam reconhecer um nome de controle que é falado e manipular esse controle em resposta a comandos falados.

Por exemplo, ao usar um utilitário de entrada de voz, um usuário pode dizer "arquivo" e, em seguida, dizer "configuração de página" para escolher o item de menu de comando correspondente e, em seguida, dizer "paisagem" para marcar a caixa de seleção correspondente. Você deve expor as informações sobre esses elementos para realizar a meta desejada.

-   [Expondo uma janela](exposing-a-window.md)
-   [Expondo texto programaticamente](exposing-text-programmatically.md)
