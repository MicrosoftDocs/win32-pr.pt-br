# <a name="speech-and-accessibility"></a>Fala e acessibilidade

Para que os aplicativos sejam acessíveis, as informações sobre o conteúdo da tela devem ser expostas usando os utilitários de reconhecimento de voz. Esses utilitários precisam identificar e nomear controles específicos para que possam reconhecer um nome de controle falado e manipular esse controle em resposta a comandos falados.

Por exemplo, ao usar um utilitário de entrada de voz, um usuário pode dizer "Arquivo", em seguida, dizer "Configuração de Página" para escolher o item de menu de comando correspondente e, em seguida, dizer "Paisagem" para selecionar a caixa de seleção correspondente. Você deve expor as informações sobre esses elementos para atingir a meta desejada.
