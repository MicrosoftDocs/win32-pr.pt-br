# <a name="system-gestures"></a>Gestos do sistema

Os gestos do sistema são mapeados para mensagens tradicionais do mouse. O Tablet PCs dá suporte a gestos de entrada do sistema de caneta de clique com o botão esquerdo, clique duas vezes, clique com o botão direito do mouse, arraste para a esquerda, arraste e focalize as mensagens do mouse. Os aplicativos desenvolvidos para um Tablet PC devem dar suporte a esses gestos da mesma forma que dão suporte a ações do mouse. Para aplicativos tradicionais, muitos dos eventos disparados por gestos são interpretados como mensagens de mouse tradicionais.

As seções a seguir discutem eventos de caneta e de sistema.

-   [Ouvindo eventos do sistema](listening-to-system-events.md)
-   [Linha do tempo de mensagens do mouse e eventos do sistema](timeline-of-mouse-messages-and-system-events.md)
