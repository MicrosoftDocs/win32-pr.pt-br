# <a name="recognizer-context"></a>Contexto do Reconhecedor

O [**objeto RecognizerContext**](inkrecognizercontext-class.md) define as configurações para um único contexto de reconhecedor. Ele recebe a tinta a ser reconhecida, define as restrições na entrada de tinta e na saída de reconhecimento e pode ser consultada para resultados de reconhecimento.

Para obter mais informações sobre dados de tinta, consulte [Dados de tinta.](ink-data.md) Para obter mais informações sobre a coleção de tinta, consulte [Coleção de tintas.](ink-collection.md)

As restrições que você pode definir na entrada de tinta e na saída de reconhecimento [**do objeto RecognizerContext**](inkrecognizercontext-class.md) incluem:

-   O idioma que é reconhecido ou retornado
-   O dicionário que foi usado
-   Factóides

Para obter mais informações sobre restrições que podem ser definidas no objeto [**RecognizerContext,**](inkrecognizercontext-class.md) consulte [Dictionaries e Factoids](dictionaries-and-factoids.md).
