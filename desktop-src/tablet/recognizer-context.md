# <a name="recognizer-context"></a>Contexto do reconhecedor

O objeto [**RecognizerContext**](inkrecognizercontext-class.md) define as configurações para um único contexto de reconhecedor. Ele recebe a tinta para reconhecer, define as restrições na entrada de tinta e a saída de reconhecimento e pode ser consultado quanto aos resultados de reconhecimento.

Para obter mais informações sobre dados de tinta, consulte [dados de tinta](ink-data.md). Para obter mais informações sobre a coleta de tinta, consulte [coleta de tinta](ink-collection.md).

As restrições que você pode definir na entrada de tinta e saída de reconhecimento do objeto [**RecognizerContext**](inkrecognizercontext-class.md) incluem:

-   O idioma reconhecido ou retornado
-   O dicionário que foi usado
-   Factóides

Para obter mais informações sobre restrições que podem ser definidas no objeto [**RecognizerContext**](inkrecognizercontext-class.md) , consulte [Dictionaries e factos](dictionaries-and-factoids.md).
