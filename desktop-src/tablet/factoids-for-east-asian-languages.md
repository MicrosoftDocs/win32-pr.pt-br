# <a name="factoids-for-east-asian-languages"></a>Factoids para idiomas do Leste da Ásia

Os idiomas do Leste da Ásia são definidos como japonês, chinês (simplificado), chinês (tradicional) e coreano. Os formatos dentro dos factoids na tabela a seguir são específicos para o reconhecedor de cada idioma.

Por exemplo, o factoid **De** telefone é diferente em cada idioma. Além disso, cada factoid é específico para um reconhecedor específico. Por exemplo, somente o factoid **de Telefone** japonês pode ser usado com o reconhecedor japonês. Além dos factoids na tabela a seguir, todos os idiomas usam os [factoids listados em Factoids Common Across Languages](factoids-common-across-languages.md).

> [!Note]  
> Os factoids para idiomas do Leste da Ásia são implementados especificando uma lista de caracteres Unicode aceitáveis. Os factoids para linguagens ocidental são implementados usando expressões regulares que descrevem a entrada esperada. Isso porque as linguagens ocidental são compostas por letras combinadas para fazer palavras, enquanto as linguagens do Leste da Ásia são baseadas em caracteres.

 

Os reconhecedores de caracteres do Leste da Ásia suportam combinar até dez factoids juntos. Essas combinações factoid empregam um operador OR lógico; portanto, a entrada pode corresponder a qualquer um dos factoids na expressão.
