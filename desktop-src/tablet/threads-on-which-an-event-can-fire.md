# <a name="threads-on-which-an-event-can-fire"></a>Threads nos quais um evento pode ser acionado

Descrição dos threads nos quais um evento pode ser acionado.

Os eventos da API (interface de programação de aplicativo) da plataforma do Tablet PC são gerados no thread de tinta, no thread de reconhecimento em segundo plano ou em um thread de aplicativo. As tabelas a seguir descrevem os threads nos quais os eventos para cada objeto ou controle podem ser acionados.

-   [Eventos do objeto InkCollector](inkcollector-object-events.md)
-   [Eventos de objeto InkOverlay](inkoverlay-object-events.md)
-   [Eventos de objeto RecognizerContext](recognizercontext-object-events.md)
-   [Eventos de coleção Strokes](strokes-collection-events.md)
-   [Eventos de controle InkEdit](inkedit-control-events.md)
-   [Eventos de controle InkPicture](inkpicture-control-events.md)
-   [Eventos de objeto PenInputPanel](peninputpanel-object-events.md)
