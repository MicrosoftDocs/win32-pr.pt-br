# <a name="alternatelist-element"></a>Elemento alternativolist

Contém o conjunto de elementos [**alternativos**](alternate-element.md) que contêm as alternativas retornadas pelo reconhecedor para o elemento pai [**InkWord**](inkword-element.md) . As alternativas são ordenadas pela confiança do reconhecedor na alternativa, com o mais alto primeiro.

## <a name="definition"></a>Definição

``` syntax
<xs:element name="AlternateList" type="AlternatesListType" />
```

## <a name="parent-elements"></a>Elementos pai

[**InkWord**](inkword-element.md)

## <a name="child-elements"></a>Elementos filho

[**Alternativo**](alternate-element.md)

## <a name="attributes"></a>Atributos

Nenhum

## <a name="element-information"></a>Informações do elemento



|                 | Valor                                                                     |
|-----------------|---------------------------------------------------------------------------|
| **Tipo de elemento**| ComplexType [**AlternatesListType**](alternateslisttype-complex-type.md) |
| **Namespace**   | urn: esquemas-Microsoft-com: Tablet: RichInk                                |
| **Nome do esquema** | Leitor de diário                                                            |
