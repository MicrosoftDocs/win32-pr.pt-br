# <a name="text-ink-reference"></a>Referência de tinta de texto

Além de dar suporte às interfaces OLE padrão, o objeto de tinta de texto (tInk) também dá suporte ao [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo).

Para obter informações sobre como especificar propriedades de exibição para um objeto de tinta de texto, consulte estrutura [**INKMETRIC**](/windows/win32/api/msinkaut/ns-msinkaut-inkmetric) .
