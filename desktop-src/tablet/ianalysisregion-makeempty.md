# <a name="ianalysisregionmakeempty-method"></a>Método IAnalysisRegion::MakeEmpty

## <a name="syntax"></a>Sintaxe


```C++
HRESULT MakeEmpty();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.
