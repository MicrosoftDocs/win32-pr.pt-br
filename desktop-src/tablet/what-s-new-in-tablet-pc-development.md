# <a name="whats-new-in-tablet-pc-development"></a>Novidades no desenvolvimento de tablets

Esta seção descreve as novidades no desenvolvimento da Tecnologia de Tablet PC.

## <a name="tablet-pc-development-kit-now-in-windows-sdk"></a>Kit de Desenvolvimento de Tablet pc agora Windows SDK

A partir do início de 2006 e no futuro, o SDK do Windows é o único mecanismo de distribuição para a tecnologia de tablet mais recente. Não há mais um SDK de tablet autônomo. Os binários, os headers, as bibliotecas, a documentação e o código de exemplo da plataforma estão disponíveis no SDK do Windows. Essa distribuição inclui os binários da plataforma 1.7 tablet PC e os módulos de mesclagem redistribuíveis para esses binários, bem como a documentação para os binários da versão 1.7 e os novos binários da plataforma de tecnologia de PC tablet do Vista do Windows (versão 6).

Consulte [O Ambiente de Desenvolvimento para](the-development-environment.md) obter detalhes sobre como configurar seu computador para desenvolvimento de tablets.

## <a name="windows-server-2008-r2-now-supports-handwriting-recognition-and-custom-dictionaries"></a>Windows O Server 2008 R2 agora dá suporte ao reconhecimento de manuscrito e dicionários personalizados

Windows O Servidor 2008 R2 agora pode executar o reconhecimento de tinta e os recursos de dicionário personalizado foram adicionados para dar suporte a novos cenários de reconhecimento.

Consulte [Reconhecimento de manuscrito no Windows Server 2008 R2](handwriting-recognition-in-windows-server-2008-r2.md) e Criando dicionários [personalizados](creating-custom-dictionaries-for-handwriting-recognition-in-windows-7-and-windows-server-2008-r2.md) para reconhecimento de manuscrito para obter mais informações.

## <a name="math-input"></a>Entrada matemática

A API de Entrada matemática agora está disponível em uma versão COM. Consulte [Programando a Controle de Entrada de Expressões Matemáticas](programming-the-math-input-control.md) e a referência Controle de Entrada de Expressões Matemáticas dados [para](math-input-control-reference.md) obter mais informações.

## <a name="ink-analysis"></a>Análise do Ink

Novos recursos de análise de tinta foram criados para a plataforma de Tecnologia de Tablet PC. Consulte [Visão geral da análise de tinta](ink-analysis-overview.md) para obter detalhes conceituais.

## <a name="stylus-input-api-com-version"></a>Versão COM da API de Entrada da Caneta

A API de Entrada da Caneta agora está disponível em uma versão COM. Consulte [Referência de RealTimeStylus](realtimestylus-reference.md) para a versão COM dos tópicos de referência de API. Consulte [Acessando e manipulando a entrada de caneta para](accessing-and-manipulating-stylus-input.md) informações conceituais da API de Entrada de Caneta.

## <a name="textinputpanel-class"></a>Classe TextInputPanel

A [classe TextInputPanel](/previous-versions/ms573640(v=vs.100)) está disponível como um objeto anexável, permitindo que você adicione a funcionalidade Painel de Entrada do Tablet PC aos controles existentes. A interface do usuário é amplamente ordenada pela linguagem de entrada atual. Você tem a opção de escolher manuscrito ou teclado como o método de entrada padrão para o objeto TextInputPanel. O usuário final pode alternar entre métodos de entrada usando botões na interface do usuário.

## <a name="journal-reader-component"></a>Componente leitor de diário

O componente Leitor de Diário está disponível. Consulte [Usando o componente Leitor de Diário para](using-the-journal-reader-component.md) ver o material de visão geral. Consulte a [classe JournalReader](/previous-versions/ms583815(v=vs.100)) para as APIs gerenciadas. Consulte [Referência de componente de leitor de](journal-reader-component-reference.md) diário para a versão COM das APIs.
