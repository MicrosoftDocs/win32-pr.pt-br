# <a name="inkpicture-methods"></a>Métodos InkPicture

Esta seção contém métodos para o controle InkPicture.



| Método                                                                                   | Descrição                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Método GetEventInterest**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-geteventinterest)                           | Recupera um valor que indica se o controle [InkPicture](inkpicture-control-reference.md) tem interesse em um evento específico.<br/>                               |
| [**GetGestureStatus**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-getgesturestatus)                                  | Recupera um valor que indica se o controle [InkPicture](inkpicture-control-reference.md) tem interesse em um gesto de aplicativo específico.<br/>                 |
| [**Método GetWindowInputRectangle**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-getwindowinputrectangle)             | Retorna o retângulo da janela, em pixels, no qual a tinta é desenhada.<br/>                                                                                                   |
| [**HitTestSelection**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-hittestselection)                                  | Recupera um membro da enumeração [**SelectionHitResult**](/windows/desktop/api/msinkaut/ne-msinkaut-selectionhitresult) , que especifica qual parte de uma seleção, se houver, foi atingida durante um teste de clique.<br/> |
| [**Método SetAllTabletsMode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setalltabletsmode)                         | Permite que o controle [InkPicture](inkpicture-control-reference.md) colete tinta de qualquer Tablet conectado ao Tablet PC.<br/>                                          |
| [**Método SetEventInterest**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-seteventinterest)                           | Modifica um valor que indica se um controle [InkPicture](inkpicture-control-reference.md) tem interesse em um evento especificado.<br/>                                  |
| [**Método SetGestureStatus**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setgesturestatus)                           | Modifica o interesse do objeto [InkPicture](inkpicture-control-reference.md) em um gesto de aplicativo especificado.<br/>                                                |
| [**Método SetSingleTabletIntegratedMode**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setsingletabletintegratedmode) | Modifica o controle [InkPicture](inkpicture-control-reference.md) para coletar tinta de apenas um Tablet anexado ao Tablet PC. A tinta de outros tablets é ignorada.<br/> |
| [**Método SetWindowInputRectangle**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkpicture-setwindowinputrectangle)             | Especifica o retângulo da janela a ser definido, em coordenadas da janela, dentro da qual a tinta é desenhada.<br/>                                                                              |
