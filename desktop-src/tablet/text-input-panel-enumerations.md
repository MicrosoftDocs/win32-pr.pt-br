# <a name="text-input-panel-enumerations"></a>Enumerações do painel de entrada de texto

Esta seção contém as enumerações usadas pela [**interface ITextInputPanel**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel).

## <a name="in-this-section"></a>Nesta seção



| Enumeração                                                  | Descrição                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Enumeração do CorrectionMode**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionmode)         | Define os valores que representam os modos de correção do painel de entrada do Tablet PC.<br/>                                               |
| [**Enumeração CorrectionPosition**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionposition) | Define os valores que representam as posições de correção de inserção de postagem do painel de entrada.<br/>                                      |
| [**EventMask**](/windows/win32/api/peninputpanel/ne-peninputpanel-eventmask)                               | Especifica os eventos para os quais você pode definir a atenção.                                                                                      |
| [**Enumeração InPlaceDirection**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacedirection)     | Define os valores que representam as posições em que o painel de entrada pode ser posicionado em relação ao ponto de inserção de texto atual.<br/> |
| [**Enumeração de in-Placestate**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacestate)             | Define valores que representam como o painel de entrada é exibido na tela.<br/>                                                       |
| [**Enumeração entre ações**](/windows/win32/api/peninputpanel/ne-peninputpanel-interactionmode)       | Define os valores que determinam o posicionamento do painel de entrada.<br/>                                                                |
| [**Enumeração PanelInputArea**](/windows/win32/api/peninputpanel/ne-peninputpanel-panelinputarea)         | Define os valores que representam as áreas de entrada do painel de entrada.<br/>                                                              |
