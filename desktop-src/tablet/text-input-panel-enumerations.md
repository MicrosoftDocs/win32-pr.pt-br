# <a name="text-input-panel-enumerations"></a>Enumerações do painel de entrada de texto

Esta seção contém as enumerações usadas pela [**interface ITextInputPanel**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel).

## <a name="in-this-section"></a>Nesta seção



| Enumeração                                                  | Descrição                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Enumeração CorrectionMode**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionmode)         | Define valores que representam os modos de correção do Painel de Entrada do Tablet PC.<br/>                                               |
| [**Enumeração CorrectionPosition**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionposition) | Define valores que representam as posições de correção pós-inserção do Painel de Entrada.<br/>                                      |
| [**EventMask**](/windows/win32/api/peninputpanel/ne-peninputpanel-eventmask)                               | Especifica os eventos para os que você pode definir atenção.                                                                                      |
| [**Enumeração InPlaceDirection**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacedirection)     | Define valores que representam as posições em que o Painel de Entrada pode ser colocado, em relação ao ponto de inserção de texto atual.<br/> |
| [**Enumeração InPlaceState**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacestate)             | Define valores que representam como o Painel de Entrada é exibido na tela.<br/>                                                       |
| [**Enumeração InteractionMode**](/windows/win32/api/peninputpanel/ne-peninputpanel-interactionmode)       | Define valores que determinam o posicionamento do Painel de Entrada.<br/>                                                                |
| [**Enumeração PanelInputArea**](/windows/win32/api/peninputpanel/ne-peninputpanel-panelinputarea)         | Define valores que representam as áreas de entrada do Painel de Entrada.<br/>                                                              |
