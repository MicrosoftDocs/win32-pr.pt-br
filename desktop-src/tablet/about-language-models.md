# <a name="about-language-models"></a>Sobre modelos de linguagem

Os reconhecedores de manuscrito usam um conjunto definido de regras para um determinado idioma como o contexto principal para reconhecer palavras. Essas regras são chamadas de modelos de linguagem. o modelo de linguagem é implementado pelo reconhecedor e não pode ser alterado diretamente por um desenvolvedor usando o SDK (Software Development Kit) do Microsoft Windows XP Tablet PC Edition. No entanto, há maneiras de refinar o contexto para campos específicos em um aplicativo. Essas técnicas incluem:

-   Escopos de entrada
-   Factóides
-   Dicionários
-   Listas de frase
