# <a name="documenting-the-gestures-used-by-your-applications"></a>Documentando os gestos usados por seus aplicativos

Se você estiver criando uma ajuda online ou outra documentação para seu aplicativo, a Microsoft recomenda que você use ilustrações de glifos e descrições semelhantes às dos [gestos do sistema](system-gestures.md) e dos [gestos do aplicativo](application-gestures.md).
