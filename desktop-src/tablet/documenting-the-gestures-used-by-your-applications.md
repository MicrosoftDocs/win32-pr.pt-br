# <a name="documenting-the-gestures-used-by-your-applications"></a>Documentando os gestos usados por seus aplicativos

Se você estiver criando ajuda online ou outra documentação para seu aplicativo, a Microsoft recomenda que você use ilustrações de glifo e descrições semelhantes às do [Gestos](system-gestures.md) do Sistema e [Gestos do Aplicativo](application-gestures.md).
