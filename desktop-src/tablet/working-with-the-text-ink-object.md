<span data-ttu-id="40479-104">Para auxiliar o suporte à tinta em aplicativos, há dois objetos, os quais podem ser incorporados e têm suporte de qualquer contêiner OLE, do objeto de tinta de texto (tInk) e do objeto de tinta do esboço (coletor).</span><span class="sxs-lookup"><span data-stu-id="40479-104">To aid the support of ink in applications, there are two objects, both of which can be embedded and are supported by any OLE container, the text ink object (tInk) and the sketch ink object (sInk).</span></span>

<span data-ttu-id="40479-105">O objeto de tinta de texto é um objeto OLE que representa a tinta que deve formar palavras.</span><span class="sxs-lookup"><span data-stu-id="40479-105">The text ink object is an OLE object representing ink that is expected to form words.</span></span> <span data-ttu-id="40479-106">Um objeto de tinta de texto permite que a tinta manuscrita seja convertida em texto, escolhendo em uma lista de alternativas.</span><span class="sxs-lookup"><span data-stu-id="40479-106">A text ink object enables the handwritten ink to be converted to text by choosing from a list of alternates.</span></span> <span data-ttu-id="40479-107">A cor e o tamanho do objeto de tinta de texto podem ser definidos programaticamente e podem ser baseados nos atributos do texto em volta do objeto.</span><span class="sxs-lookup"><span data-stu-id="40479-107">The color and size of the text ink object can be set programmatically and can be based on the attributes of the text around the object.</span></span> <span data-ttu-id="40479-108">O objeto de tinta de texto deve conter uma única palavra.</span><span class="sxs-lookup"><span data-stu-id="40479-108">The text ink object is intended to contain a single word.</span></span>

<span data-ttu-id="40479-109">O objeto de tinta de texto dá suporte ao conjunto padrão de interfaces OLE necessárias para inserção e suporte à área de transferência.</span><span class="sxs-lookup"><span data-stu-id="40479-109">The text ink object supports the standard set of OLE interfaces required for embedding and clipboard support.</span></span> <span data-ttu-id="40479-110">A interface [**IPersistStream**](/windows/win32/api/objidl/nn-objidl-ipersiststream) lê e grava em um fluxo no formato serializado da tinta (ISF).</span><span class="sxs-lookup"><span data-stu-id="40479-110">The [**IPersistStream**](/windows/win32/api/objidl/nn-objidl-ipersiststream) interface reads from and writes to a stream in ink serialized format (ISF).</span></span> <span data-ttu-id="40479-111">O objeto de tinta de texto fornece a interface [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo) para acessar suas propriedades de exibição e a lista de resultados de reconhecimento.</span><span class="sxs-lookup"><span data-stu-id="40479-111">The text ink object provides the [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo) interface to access its display properties, and recognition result list.</span></span>

<span data-ttu-id="40479-112">O objeto de tinta de texto pode ser usado para interoperabilidade entre aplicativos, colocando-o no slot do objeto OLE na área de transferência, inserindo-o em RTF ou mantendo-o em um fluxo ISF.</span><span class="sxs-lookup"><span data-stu-id="40479-112">The text ink object can be used for interoperability between applications either by placing it in the OLE object slot on the Clipboard, by embedding it in RTF, or by persisting it in an ISF stream.</span></span>

<span data-ttu-id="40479-113">Um objeto de tinta de texto pode ser gerado das seguintes maneiras.</span><span class="sxs-lookup"><span data-stu-id="40479-113">A text ink object can be generated in the following ways.</span></span>

-   <span data-ttu-id="40479-117">Quando um aplicativo copia um objeto [InkStrokes](/previous-versions/windows/desktop/legacy/ms703293(v=vs.85)) ou [InkEdit](inkedit-control-reference.md) na área de transferência e o formato de [**Enumeração InkClipboardFormats**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardformats) é definido, o slot de área de transferência do objeto OLE contém um objeto OLE de tinta de texto.</span><span class="sxs-lookup"><span data-stu-id="40479-117">When an application copies an [InkStrokes](/previous-versions/windows/desktop/legacy/ms703293(v=vs.85)) or an [InkEdit](inkedit-control-reference.md) object onto the Clipboard and the [**InkClipboardFormats Enumeration**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardformats) format is set, the OLE object Clipboard slot contains a text ink OLE object.</span></span>
