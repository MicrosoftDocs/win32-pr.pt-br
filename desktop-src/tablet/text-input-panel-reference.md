# <a name="text-input-panel-reference"></a>Referência do painel de entrada de texto

Fornece acesso aos recursos programáveis do painel de entrada do Tablet PC.

## <a name="in-this-section"></a>Nesta seção

-   [Mensagens do painel de entrada de texto](tip-messages.md)
-   [Enumerações do painel de entrada de texto](text-input-panel-enumerations.md)
-   [Interfaces do painel de entrada de texto](text-input-panel-interfaces.md)

## <a name="com-implementation"></a>Implementação COM

O tópico inclui referências para interfaces COM do painel de entrada de texto e enumerações.

Essas interfaces podem ser instanciadas chamando o método [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) em C++.

## <a name="remarks"></a>Comentários

As interfaces e enumerações do painel de entrada de texto são usadas para controlar a aparência e o comportamento do painel de entrada.

Esses elementos são declarados em PenInputPanel. h.
