# <a name="flicks-enumerations"></a>Enumerações de movimento

Esta seção contém as enumerações usadas na mensagem [**WM TABLET WM NOTEBOOK \_ \_ .**](wm-tablet-flick-message.md)



| Enumeração                                                 | Descrição                                                                                                                                                   |
|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-flickdirection)                    | Representa a direção do movimento.<br/>                                                                                                             |
| [**MODEMODE**](/windows/desktop/api/tabflicks/ne-tabflicks-flickmode)                              | Especifica onde os gestos de movimento estão visíveis.<br/>                                                                                                        |
| [**CÓDIGO DE \_ COMANDO DECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-flickaction_commandcode) | Especifica o código de comando arquivado na Estrutura [**DE DADOS DE MOVIMENTO \_**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_data) e representa que tipo de ação de movimento deve ser executada.<br/> |
| [**SCROLLDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-scrolldirection)                  | Especifica qual direção a caixa de rolagem deve mover.<br/>                                                                                               |
| [**KEYMODIFIER**](/windows/desktop/api/tabflicks/ne-tabflicks-keymodifier)                          | Especifica quais teclas modificadora, se alguma, foram pressionadas quando o gesto de movimento ocorreu.<br/>                                                               |
