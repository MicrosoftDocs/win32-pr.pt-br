# <a name="flicks-enumerations"></a>As enumerações de movimentos

Esta seção contém as enumerações usadas na [**mensagem de \_ \_ movimento do Tablet do WM**](wm-tablet-flick-message.md).



| Enumeração                                                 | Descrição                                                                                                                                                   |
|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FLICKDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-flickdirection)                    | Representa a direção do movimento.<br/>                                                                                                             |
| [**Movimento de movimentos**](/windows/desktop/api/tabflicks/ne-tabflicks-flickmode)                              | Especifica onde os gestos de movimento estão visíveis.<br/>                                                                                                        |
| [**COMMANDCODE de movimentos \_**](/windows/desktop/api/tabflicks/ne-tabflicks-flickaction_commandcode) | Especifica o código de comando arquivado [**na \_ estrutura de dados de movimento**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_data) e representa que tipo de ação de movimento deve ser executada.<br/> |
| [**SCROLLDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-scrolldirection)                  | Especifica a direção que a caixa de rolagem deve mover.<br/>                                                                                               |
| [**Keymodificador**](/windows/desktop/api/tabflicks/ne-tabflicks-keymodifier)                          | Especifica que, se houver, as teclas modificadoras foram pressionadas quando o gesto de movimento ocorreu.<br/>                                                               |
