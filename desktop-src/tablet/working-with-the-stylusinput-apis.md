<span data-ttu-id="94b82-105">A classe [**RealTimeStylus**](realtimestylus-class.md) permite que você interaja com o fluxo de dados da caneta eletrônica.</span><span class="sxs-lookup"><span data-stu-id="94b82-105">The [**RealTimeStylus**](realtimestylus-class.md) class enables you to interact with the data stream from the tablet pen.</span></span> <span data-ttu-id="94b82-106">Para interagir com o fluxo de dados, adicione um objeto **RealTimeStylus** ao seu aplicativo e adicione plug-ins ao objeto **RealTimeStylus** .</span><span class="sxs-lookup"><span data-stu-id="94b82-106">To interact with the data stream, add a **RealTimeStylus** object to your application, and add plug-ins to the **RealTimeStylus** object.</span></span>

<span data-ttu-id="94b82-107">Os plug-ins podem modificar os dados associados aos pacotes no ar, à caneta, aos pacotes e aos métodos de notificação de caneta.</span><span class="sxs-lookup"><span data-stu-id="94b82-107">The plug-ins can modify the data associated with in-air packets, stylus down, packets, and stylus up notification methods.</span></span> <span data-ttu-id="94b82-108">Os plug-ins podem cancelar os pacotes em ar e os métodos de notificação de pacotes.</span><span class="sxs-lookup"><span data-stu-id="94b82-108">The plug-ins can cancel the in-air packets and packets notification methods.</span></span> <span data-ttu-id="94b82-109">Os plug-ins também podem adicionar dados de aplicativo ao fluxo na forma de objetos [CustomStylusData](/previous-versions/ms575208(v=vs.100)) .</span><span class="sxs-lookup"><span data-stu-id="94b82-109">The plug-ins can also add application data to the stream in the form of [CustomStylusData](/previous-versions/ms575208(v=vs.100)) objects.</span></span> <span data-ttu-id="94b82-110">A lista a seguir oferece ideias para categorias comuns de plug-ins que talvez você queira usar ou criar.</span><span class="sxs-lookup"><span data-stu-id="94b82-110">The following list offers ideas for common categories of plug-ins that you may want to use or create.</span></span>

-   <span data-ttu-id="94b82-111">Plug-in de filtro: um objeto que remove ou cancela seletivamente dados no fluxo de dados da caneta eletrônica.</span><span class="sxs-lookup"><span data-stu-id="94b82-111">Filter plug-in: An object that selectively removes or cancels data in the tablet pen data stream.</span></span>
-   <span data-ttu-id="94b82-112">Plug-in do modificador: um objeto que modifica dados seletivamente no fluxo de dados da caneta eletrônica.</span><span class="sxs-lookup"><span data-stu-id="94b82-112">Modifier plug-in: An object that selectively modifies data in the tablet pen data stream.</span></span>
-   <span data-ttu-id="94b82-113">Plug-in de processador dinâmico: um objeto que exibe os dados da caneta do Tablet em tempo real conforme ele está sendo manipulado pelo objeto [**RealTimeStylus**](realtimestylus-class.md) .</span><span class="sxs-lookup"><span data-stu-id="94b82-113">Dynamic-renderer plug-in: An object that displays the tablet pen data in real-time as it is being handled by the [**RealTimeStylus**](realtimestylus-class.md) object.</span></span> <span data-ttu-id="94b82-114">Posteriormente, para eventos como uma atualização de formulário, o plug-in de processamento dinâmico ou um plug-in de coleção de tinta pode redesenhar a tinta.</span><span class="sxs-lookup"><span data-stu-id="94b82-114">Later, for events such as a form refresh, the dynamic renderer plug-in or an ink collection plug-in might redraw the ink.</span></span>
-   <span data-ttu-id="94b82-115">Plug-in do reconhecedor: um objeto que examina a movimentação da caneta do Tablet para gestos, manuscritos ou outros glifos.</span><span class="sxs-lookup"><span data-stu-id="94b82-115">Recognizer plug-in: An object that scans the movement of the tablet pen for gestures, handwriting, or other glyphs.</span></span>
-   <span data-ttu-id="94b82-116">Plug-in do coletor de tinta: um objeto do fluxo de dados da caneta do Tablet cria e armazena a tinta.</span><span class="sxs-lookup"><span data-stu-id="94b82-116">Ink-collector plug-in: An object that from the tablet pen data stream creates and stores ink.</span></span>
-   <span data-ttu-id="94b82-117">Plug-in de wrapper: um plug-in que atua como uma interface entre o objeto [**RealTimeStylus**](realtimestylus-class.md) e outro plug-in ou objeto como uma maneira de modificar o comportamento do objeto encapsulado.</span><span class="sxs-lookup"><span data-stu-id="94b82-117">Wrapper plug-in: A plug-in that acts as an interface between the [**RealTimeStylus**](realtimestylus-class.md) object and another plug-in or object as a way of modifying the behavior of the wrapped object.</span></span>

<span data-ttu-id="94b82-118">Os plug-ins de renderização dinâmica e de coleção de tinta podem ser criados para serem renderizados em vários contextos — como em um arquivo, um fluxo ou em um dispositivo de vídeo.</span><span class="sxs-lookup"><span data-stu-id="94b82-118">Both dynamic-renderer and ink-collection plug-ins can be created to render to various contexts—such as to a file, a stream, or to a display device.</span></span> <span data-ttu-id="94b82-119">A tinta também pode ser armazenada em vários formatos, como um objeto de [tinta](/previous-versions/aa515768(v=msdn.10)) , um arquivo de Graphics Interchange Format reforçada (GIF), um arquivo ISF (formato serializado de tinta) ou outros formatos.</span><span class="sxs-lookup"><span data-stu-id="94b82-119">Ink can also be stored in various formats, such as an [Ink](/previous-versions/aa515768(v=msdn.10)) object, a fortified Graphics Interchange Format (GIF) file, an Ink Serialized Format (ISF) file, or other formats.</span></span>

<span data-ttu-id="94b82-120">Dois plug-ins são fornecidos com as APIs StylusInput: a classe [**DynamicRenderer**](/previous-versions/windows/desktop/legacy/ms701168(v=vs.85)) e a classe [**GestureRecognizer**](gesturerecognizer-class.md) .</span><span class="sxs-lookup"><span data-stu-id="94b82-120">Two plug-ins are provided with the StylusInput APIs: the [**DynamicRenderer**](/previous-versions/windows/desktop/legacy/ms701168(v=vs.85)) class and the [**GestureRecognizer**](gesturerecognizer-class.md) class.</span></span> <span data-ttu-id="94b82-121">A classe **DynamicRenderer** fornece renderização básica dos dados de tinta em tempo real e é simplificada para ter um impacto mínimo no desempenho.</span><span class="sxs-lookup"><span data-stu-id="94b82-121">The **DynamicRenderer** class provides basic rendering of the ink data in real time and is streamlined to have a minimal performance impact.</span></span> <span data-ttu-id="94b82-122">A classe **GestureRecognizer** fornece reconhecimento de gesto para a classe [**RealTimeStylus**](realtimestylus-class.md) .</span><span class="sxs-lookup"><span data-stu-id="94b82-122">The **GestureRecognizer** class provides gesture recognition for the [**RealTimeStylus**](realtimestylus-class.md) class.</span></span>
