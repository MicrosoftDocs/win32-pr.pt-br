# <a name="recognizer-dll-sample-template"></a>Modelo de exemplo de DLL do Reconhecedor

O exemplo de DLL do Reconhecedor contém um modelo para criar uma DLL do reconhecedor. O modelo contém funções para registrar e não registrar o servidor, bem como esqueletos para funções de reconhecedor primário.

Este modelo mostra como registrar uma CLSID no Registro. Você também pode fazer isso ao instalar o reconhecedor usando seu programa de instalação. Certifique-se de criar seu próprio CLSID exclusivo para seus idiomas e não reutilizar o CLSID mostrado no exemplo.

Por padrão, os exemplos são instalados no diretório Exemplos no diretório no qual você instalou o SDK.

Para ver um exemplo de como usar reconhecedores da API da Plataforma de Tablet PC, consulte o [Exemplo de Reconhecimento Avançado](advanced-recognition-sample.md).
