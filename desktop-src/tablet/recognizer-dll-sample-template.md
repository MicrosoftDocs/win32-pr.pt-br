# <a name="recognizer-dll-sample-template"></a>Modelo de exemplo de DLL do Recognizer

O exemplo de DLL do reconhecedor contém um modelo para criar uma DLL de reconhecedor. O modelo contém funções para registrar e cancelar o registro do servidor, bem como esqueletos para funções de reconhecedor primário.

Este modelo mostra como registrar um CLSID no registro. Você também pode fazer isso ao instalar o reconhecedor usando o programa de instalação. Certifique-se de criar seu próprio CLSID exclusivo para seus idiomas e não reutilize o CLSID mostrado no exemplo.

Por padrão, os exemplos são instalados no diretório de exemplos no diretório no qual você instalou o SDK.

Para obter um exemplo de como usar os reconhecedores da API da plataforma do Tablet PC, consulte o [exemplo de reconhecimento avançado](advanced-recognition-sample.md).
