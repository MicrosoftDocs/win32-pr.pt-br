# <a name="inkedit-enumerations"></a>Enumerações InkEdit

Esta seção contém enumerações que pertencem ao controle InkEdit.



| Enumeração                                            | Descrição                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AppearanceConstants**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | Especifica se o controle parece simples ou 3D.<br/>                                                                                            |
| [**BorderStyleConstants**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | Especifica se o controle tem uma borda.<br/>                                                                                                   |
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | Especifica o interesse em um conjunto de gestos específicos do aplicativo.<br/>                                                                             |
| [**InkDisplayMode**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | Especifica se uma seleção aparece como tinta ou texto.<br/>                                                                                         |
| [**InkEditStatus**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | Especifica se o controle [InkEdit](inkedit-control-reference.md) está ocioso, coletando tinta ou reconhecendo a tinta.<br/>                           |
| [**InkInsertMode**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | Especifica como a tinta é inserida no controle [InkEdit](inkedit-control-reference.md) .<br/>                                                      |
| [**InkMode**](/windows/desktop/api/inked/ne-inked-inkmode)                             | Especifica as configurações do modo de coleta para tinta desenhada-se a coleta de tinta está desabilitada, a tinta é coletada ou a tinta e os gestos são coletados.<br/> |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | Especifica qual botão do mouse foi pressionado.<br/>                                                                                                     |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | Especifica o tipo de ponteiro do mouse que aparece.<br/>                                                                                             |
| [**MouseButton**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | Especifica qual botão do mouse foi pressionado.<br/>                                                                                                     |
| [**ScrollBarsConstants**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | Especifica como as barras de rolagem de um controle [InkEdit](inkedit-control-reference.md) aparecem na tela.<br/>                                    |
| [**SelAlignmentConstants**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | Especifica o alinhamento do parágrafo em relação às margens do controle [InkEdit](inkedit-control-reference.md) .<br/>                     |
