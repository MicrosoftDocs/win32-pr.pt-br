# <a name="adding-ink-controls-to-a-project"></a>Adicionando controles de tinta a um projeto

Para adicionar um controle [InkEdit](inkedit-control-reference.md) ou [InkPicture](inkpicture-control-reference.md) a um projeto do Microsoft Visual Basic .net ou Microsoft Visual C \# :

1.  Clique com o botão direito do mouse na caixa de ferramentas e clique em **Personalizar caixa de ferramentas**.
2.  Na caixa de diálogo **Personalizar Toolbox** , clique na guia **componentes .NET Framework** .
3.  Na caixa de listagem, selecione os controles InkEdit e InkPicture e clique em **OK**. Os controles são adicionados à caixa de ferramentas na guia **componentes** .
4.  Selecione o controle desejado na caixa de ferramentas e arraste o controle para o local no formulário onde você deseja que ele apareça.

> [!Note]  
> A biblioteca gerenciada do Tablet PC pode dar suporte a controles da Web em ambientes parcialmente confiáveis, ou seja, a execução de assemblies gerenciados de páginas da Web. No entanto, não há suporte para controles ActiveX habilitados para tinta em páginas da Web.
