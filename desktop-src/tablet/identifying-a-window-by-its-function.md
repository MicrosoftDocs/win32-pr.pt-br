# <a name="identifying-a-window-by-its-function"></a>Identificando uma janela por sua função

Identifique cada tipo de janela atribuindo a ela uma classe de janela exclusiva. Evite ter uma única classe de janela que seja usada para uma ampla variedade de funções.

Os auxílios de acessibilidade devem ter essa identificação para lidar com janelas diferentes no mesmo aplicativo. Pode haver instruções separadas para lidar com essas janelas, como anunciar áreas específicas quando o conteúdo é alterado.

Se você não puder usar classes de janela separadas, poderá expor as diferenças por meio do Microsoft <entity type="reg"/> acessibilidade ativa <entity type="reg"/> ou, se necessário, por uma mensagem de janela particular que você documenta em seu site.

Para obter mais informações sobre como expor classes de janela usando Acessibilidade Ativa, consulte a seção [acessibilidade](/windows/desktop/accessibility) .
