Esta seção descreve as enumerações do reconhecedor.

## <a name="in-this-section"></a>Nesta seção



| Enumeração                                   | Descrição                                                                                |
|-----------------------------------------------|--------------------------------------------------------------------------------------------|
| [**interrupções de ALT \_**](/windows/win32/api/rectypes/ne-rectypes-alt_breaks)             | Especifica como criar alternativas de uma melhor cadeia de caracteres de resultado.<br/>                   |
