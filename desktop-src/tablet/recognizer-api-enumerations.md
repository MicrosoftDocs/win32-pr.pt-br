Esta seção descreve as enumerações do reconhecedor.

## <a name="in-this-section"></a>Nesta seção



| Enumeração                                   | Descrição                                                                                |
|-----------------------------------------------|--------------------------------------------------------------------------------------------|
| [**interrupções de ALT \_**](/windows/win32/api/rectypes/ne-rectypes-alt_breaks)             | Especifica como criar alternativas de uma melhor cadeia de caracteres de resultado.<br/>                   |
| [**nível de confiança \_**](/windows/win32/api/rectypes/ne-rectypes-confidence_level) | Indica o nível de confiança que o reconhecedor tem no resultado do reconhecimento.<br/> |
| [**\_métricas de linha**](/windows/win32/api/rectypes/ne-rectypes-line_metrics)         | Representa as linhas encontradas em um segmento.<br/>                                        |
