# <a name="factoid-support"></a>Suporte ao factor

Os *factos* melhoram a precisão do reconhecimento fornecendo ao reconhecedor uma definição e a ordem das palavras, números e marcas de Pontuação esperados.

Para obter mais informações sobre os factos, consulte [usando o contexto para melhorar a precisão](using-context-to-improve-accuracy.md).
