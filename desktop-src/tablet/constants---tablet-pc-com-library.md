# <a name="core-tablet-pc-constants"></a>Principais constantes do Tablet PC

Esta seção contém as principais constantes para o Tablet PC.

## <a name="in-this-section"></a>Nesta seção



| Constante                                                 | Descrição                                                                                                                                                                                                                                               |
|----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Facto](factoid-constants.md)                         | Define valores de cadeia de caracteres constantes que são usados para aumentar a precisão do reconhecimento fornecendo informações contextuais para o reconhecedor.<br/>                                                                                                           |
| [PacketPropertyGuids](packetpropertyguids-constants.md) | Define os valores que especificam as propriedades do pacote. A API (interface de programação de aplicativo) do Tablet PC usa identificadores globais exclusivos (GUIDs) para identificar Propriedades de pacote, que na automação são cadeias de caracteres constantes.<br/>                     |
| [Reconhecimentoproperty](recognitionproperty-constants.md) | Define os valores que especificam as propriedades de uma alternativa de reconhecimento. A API (interface de programação de aplicativo) do Tablet PC usa identificadores globais exclusivos (GUIDs) para identificar Propriedades de pacote, que na automação são cadeias de caracteres constantes.<br/> |
