# <a name="creating-a-recognizer"></a>Criando um reconhecedor

Se você optar por fazer isso, seu aplicativo poderá fornecer suas próprias implementações de reconhecedor. Esta seção descreve os requisitos para criar um reconhecedor de aplicativo personalizado para seu aplicativo que você pode usar com a API da Plataforma de Tablet PC.

> [!Note]  
> Os reconhecedores de aplicativos personalizados não são usados pelo Painel de Entrada do tablet ou pelo Microsoft Windows Journal. Esses acessórios usam somente reconhecedores com os quais foram testados.
