# <a name="creating-a-recognizer"></a>Criando um reconhecedor

Se você optar por fazer isso, seu aplicativo poderá fornecer suas próprias implementações de reconhecedor. Esta seção descreve os requisitos para criar um reconhecedor de aplicativo personalizado para seu aplicativo que você pode usar com a API da plataforma Tablet PC.

> [!Note]  
> Os reconhecedores de aplicativos personalizados não são usados pelo painel de entrada do Tablet PC ou pelo diário do Microsoft Windows. Esses acessórios usam apenas os reconhecedores com os quais foram testados.
