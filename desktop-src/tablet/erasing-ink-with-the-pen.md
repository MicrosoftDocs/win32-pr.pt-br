# <a name="erasing-ink-with-the-pen"></a>Apagando tinta com a caneta

Além de habilitar o manuscrito e o desenho, você pode criar seu aplicativo para dar suporte ao uso da caneta do Tablet para apagar a tinta. Use o objeto [**InkOverlay**](inkoverlay-class.md) para fazer isso.

As seções a seguir descrevem como apagar a tinta com a caneta eletrônica.

-   [Apagando usando o objeto InkOverlay](erasing-by-using-the-inkoverlay-object.md)
-   [Apagando usando a caneta](erasing-by-using-the-pen.md)
