# <a name="application-gestures"></a>Gestos de aplicativo

Gestos de aplicativo são gestos que você escolhe para que o seu aplicativo dê suporte. Eles são mais comuns em aplicativos projetados especificamente para trabalhar com uma caneta eletrônica. Um aplicativo herdado pode usar gestos, mas modelos de seleção existentes dentro do aplicativo podem complicar o suporte para gestos. Somente os gestos com suporte no painel de entrada do Tablet PC estão disponíveis para aplicativos herdados.

As seções a seguir descrevem os gestos de aplicativo e seu uso.

-   [Gestos de aplicativo e comportamento semântico](application-gestures-and-semantic-behavior.md)
-   [Usando apenas o reconhecedor de gestos da Microsoft](using-the-microsoft-gesture-recognizer-only.md)
-   [Usando um reconhecedor de gestor personalizado](using-a-custom-gesture-recognizer.md)
-   [Possíveis conflitos com gestos de aplicativo](potential-conflicts-with-application-gestures.md)
-   [Documentando os gestos usados por seus aplicativos](documenting-the-gestures-used-by-your-applications.md)
-   [Glifos não implementados](unimplemented-glyphs.md)
