# <a name="application-gestures"></a>Gestos de aplicativo

Gestos de aplicativo são gestos que você escolhe para ter suporte ao aplicativo. Eles são mais comuns em aplicativos projetados especificamente para trabalhar com uma caneta de tablet. Um aplicativo herdado pode usar gestos, mas os modelos de seleção existentes dentro do aplicativo podem complicar o suporte para gestos. Somente esses gestos com suporte no Painel de Entrada do Tablet PC estão disponíveis para aplicativos herdados.

As seções a seguir descrevem os gestos do aplicativo e seu uso.

-   [Gestos de aplicativo e comportamento semântico](application-gestures-and-semantic-behavior.md)
-   [Usando somente o Reconhecedor de Gestos da Microsoft](using-the-microsoft-gesture-recognizer-only.md)
-   [Usando um reconhecedor de gesto personalizado](using-a-custom-gesture-recognizer.md)
-   [Possíveis conflitos com gestos de aplicativo](potential-conflicts-with-application-gestures.md)
-   [Documentando os gestos usados por seus aplicativos](documenting-the-gestures-used-by-your-applications.md)
-   [Glifos não simplificados](unimplemented-glyphs.md)
