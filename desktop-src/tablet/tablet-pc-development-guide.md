# <a name="tablet-pc"></a>Tablet

O Tablet PC é um computador pessoal totalmente funcional voltado para aplicativos habilitados para caneta, habilitados para manuscrito e habilitados para fala. A combinação de software e hardware em um tablet pc permite esses métodos de interação do usuário e permite uma experiência de computação rica, interativa e produtiva para os usuários.

A tecnologia de tablet pc Windows permite a entrada e a saída de dados de manuscrito e fala em um tablet pc, bem como o intercâmbio desses dados com outros computadores.

Windows O toque, introduzido no Windows 7, permite gestos multitouch e uma experiência de toque aprimorada para os usuários.
