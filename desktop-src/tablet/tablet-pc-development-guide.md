# <a name="tablet-pc"></a>Tablet

O Tablet PC é um computador pessoal totalmente funcional voltado para aplicativos habilitados para caneta, com manuscrito e habilitado para fala. A combinação de software e hardware em um Tablet PC habilita esses métodos de interação do usuário e permite uma experiência de computação avançada, interativa e produtiva para os usuários.

A tecnologia do Tablet PC no Windows permite a entrada e a saída de dados manuscritos e de fala em um Tablet PC, bem como o intercâmbio desses dados com outros computadores.

O Windows Touch, introduzido no Windows 7, permite gestos de multitoque e uma experiência de toque aprimorada para os usuários.
