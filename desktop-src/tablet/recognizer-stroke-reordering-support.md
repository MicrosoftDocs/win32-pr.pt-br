# <a name="recognizer-stroke-reordering-support"></a>Suporte à reordenação de Stroke do Recognizer

Os reconhecedores da Microsoft do script latino dão suporte à reordenação do traço.

Os reconhecedores de manuscrito da Microsoft do script Latin não dependem dos traços em uma linha que está sendo gravada em uma ordem específica. Em vez disso, os reconhecedores ordenam os traços internamente com base nas relações espaciais entre os traços. Por exemplo, os reconhecedores do script Latin dão suporte ao cenário comum no qual alguém retorna uma linha para gravar um parêntese esquerdo, "(" ou para adicionar aspas.
