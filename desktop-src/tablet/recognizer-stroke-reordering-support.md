# <a name="recognizer-stroke-reordering-support"></a>Suporte à reordenação de traço do reconhecedor

Os reconhecedores da Microsoft do script latino suportam a reordenação de traço.

Os reconhecedores de manuscrito da Microsoft do script latino não dependem dos traços em uma linha que está sendo escrita em uma ordem específica. Em vez disso, os reconhecedores ordenam os traços internamente com base em relações espaciais entre os traços. Por exemplo, os reconhecedores de script latino suportam o cenário comum em que alguém volta em uma linha para escrever um parêntese esquerdo, "(", ou para adicionar aspas.
