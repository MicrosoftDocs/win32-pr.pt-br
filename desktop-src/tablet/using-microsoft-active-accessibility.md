# <a name="using-microsoft-active-accessibility"></a>Usando o Microsoft Acessibilidade Ativa

O mecanismo de reconhecimento de fala no Tablet PC usa Acessibilidade Ativa para recuperar informações diretamente do objeto e ser notificado quando os objetos forem alterados. Essa interface é suportada pelos controles padrão do Microsoft Windows e também pode ser usada por controles personalizados e classes de janela personalizadas.
