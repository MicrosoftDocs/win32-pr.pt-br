# <a name="getting-started-tablet-pc"></a>Introdução (Tablet PC)

O Tablet PC é um computador pessoal totalmente funcional da plataforma Microsoft Windows e projetado para aplicativos habilitados para tinta, habilitado para caneta e habilitado para fala. A combinação de software e hardware em um Tablet PC permite que esses métodos de interação do usuário criem uma experiência de computação avançada, interativa e produtiva para os usuários.

A plataforma do Tablet PC permite a entrada e a saída de dados manuscritos e de fala em um Tablet PC, bem como o intercâmbio desses dados com outros computadores. Os componentes da plataforma Tablet PC instalados pelo SDK do Windows 7 permitem que você crie aplicativos habilitados para a tinta e habilitados para caneta para o Tablet PC.
