# <a name="getting-started-tablet-pc"></a>Ponto de Partida (Tablet PC)

O Tablet PC é um computador pessoal totalmente funcional da plataforma Microsoft Windows e projetado para aplicativos habilitados para tinta, habilitados para caneta e habilitados para fala. A combinação de software e hardware em um tablet pc permite que esses métodos de interação do usuário criam uma experiência de computação rica, interativa e produtiva para os usuários.

A plataforma Tablet PC habilita a entrada e a saída de dados de manuscrito e fala em um tablet pc, bem como o intercâmbio desses dados com outros computadores. Os componentes da plataforma Tablet PC instalados pelo SDK do Windows 7 permitem que você crie aplicativos habilitados para tinta e habilitados para caneta para o Tablet PC.
