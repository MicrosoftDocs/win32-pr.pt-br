# <a name="iinkanalysisrecognizersetashighestpriorityinkanalysisrecognizer-method"></a>Método IInkAnalysisRecognizer:: SetAsHighestPriorityInkAnalysisRecognizer

Modifica esse [**IInkAnalysisRecognizer**](iinkanalysisrecognizer.md) para a prioridade mais alta na lista ordenada interna de [**IInkAnalyzer**](iinkanalyzer.md) de reconhecedores.

## <a name="syntax"></a>Sintaxe


```C++
HRESULT SetAsHighestPriorityInkAnalysisRecognizer();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Valor retornado

Para obter uma descrição dos valores de retorno, consulte [classes e interfaces – análise de tinta](classes-and-interfaces---ink-analysis.md).
