# <a name="conflicts-between-application-gestures"></a>Conflitos entre gestos de aplicativo

Alguns gestos de aplicativo são semelhantes o suficiente entre si para resultar em possíveis conflitos durante o reconhecimento. Não use simultaneamente gestos de aplicativo potencialmente conflitantes em seu aplicativo.
