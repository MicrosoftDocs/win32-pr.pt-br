# <a name="windows-hello-world-sample"></a>Windows Hello Exemplo de mundo

Este aplicativo de exemplo mostra como criar um programa de Windows mínimo.

## <a name="description"></a>Descrição

O Windows Hello aplicativo de exemplo World cria e mostra uma janela vazia, conforme mostrado na captura de tela a seguir. Este exemplo é discutido no [Módulo 1. Seu primeiro programa Windows .](your-first-windows-program.md)

![captura de tela do programa de exemplo.](images/window01.png)

## <a name="downloading-the-sample"></a>Baixando o exemplo

Este exemplo está disponível [aqui.](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/begin/LearnWin32/HelloWorld)

Para baixá-lo, vá para a raiz do repo de exemplo no GitHub ([microsoft/Windows-classic-samples)](https://github.com/microsoft/Windows-classic-samples/)e clique no botão **Clonar** ou baixar para baixar o arquivo zip de todos os exemplos no computador. Em seguida, descompcomple a pasta.

Para abrir o exemplo no Visual Studio, selecione **Arquivo/Abrir/Project/Solução** e navegue até o local em que você desemparou a pasta e **Windows-classic-samples-master/Samples/Win7Samples/begin/LearnWin32 /HelloWorld/cpp**. Abra o arquivo **HelloWorld.sln**.

Depois que o exemplo tiver sido carregado, você precisará atualizá-lo para trabalhar com Windows 10. No menu **Project** no Visual Studio, selecione **Propriedades**. Atualize **Windows versão do SDK** do Windows 10 para um SDK do Windows 10, como 10.0.17763.0 ou melhor. Em seguida, **altere o Platform Toolset** para Visual Studio 2017 ou melhor. Agora você pode executar o exemplo pressionando F5!
