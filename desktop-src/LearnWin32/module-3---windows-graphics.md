# <a name="module-3-windows-graphics"></a>Módulo 3. Gráficos do Windows

O [módulo 1](your-first-windows-program.md) desta série mostrou como criar uma janela em branco. O [módulo 2](module-2--using-com-in-your-windows-program.md) tirou um ligeiro desvio por meio de Component Object Model (com), que é a base para muitas das APIs modernas do Windows. Agora é hora de adicionar gráficos à janela em branco que criamos no módulo 1.

Este módulo começa com uma visão geral de alto nível da arquitetura de gráficos do Windows. Em seguida, examinamos o Direct2D, uma poderosa API de gráficos que foi introduzida no Windows 7.
