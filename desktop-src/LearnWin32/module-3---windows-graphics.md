# <a name="module-3-windows-graphics"></a>Módulo 3. Windows Gráficos

[O módulo 1](your-first-windows-program.md) desta série mostrou como criar uma janela em branco. [O Módulo 2](module-2--using-com-in-your-windows-program.md) fez um pequeno desvio pelo COM (Component Object Model), que é a base para muitas das APIs Windows modernas. Agora é hora de adicionar elementos gráficos à janela em branco que criamos no Módulo 1.

Este módulo começa com uma visão geral de alto nível da arquitetura Windows elementos gráficos. Em seguida, vamos Direct2D, uma API de gráficos poderosa que foi introduzida no Windows 7.
