# <a name="open-dialog-box-sample"></a>Exemplo de caixa de diálogo aberta

Este aplicativo de exemplo mostra como inicializar a biblioteca do Component Object Model (COM) e usar uma API baseada em COM em um programa do Windows.

## <a name="description"></a>Descrição

A caixa de diálogo abrir aplicativo de exemplo exibe a caixa de diálogo **abrir** , conforme mostrado na captura de tela a seguir. O exemplo demonstra como chamar um objeto COM em um programa do Windows. Este exemplo é discutido no [módulo 2: usando com em seu programa do Windows](module-2--using-com-in-your-windows-program.md).
