# <a name="animation-timer-interfaces"></a>Interfaces do temporizador de animação

Esta seção contém as especificações de referência para as interfaces do Gerenciador de animação do Windows que dão suporte a temporizadores de animação.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                         | Descrição                                                                                   |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IUIAnimationTimer**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimer)<br/>                                     | Define um temporizador de animação, que fornece serviços para gerenciar o tempo de animação.<br/> |
| [**IUIAnimationTimerClientEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerclienteventhandler)<br/> | Define um método para manipular eventos relacionados a alterações no status do cliente do temporizador.<br/>    |
| [**IUIAnimationTimerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimereventhandler)<br/>             | Define métodos para manipular eventos de tempo.<br/>                                        |
| [**IUIAnimationTimerUpdateHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerupdatehandler)<br/>           | Define métodos para manipular eventos de atualização de tempo.<br/>                                 |
