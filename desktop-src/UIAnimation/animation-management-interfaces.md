# <a name="animation-management-interfaces"></a>Interfaces de gerenciamento de animação

esta seção contém as especificações de referência para as Windows interfaces do gerenciador de animação que dão suporte ao gerenciamento de animação.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                   | Descrição                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationManager**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager)<br/>                           | Define o Gerenciador de animação, que fornece uma interface central para criar e gerenciar animações.<br/>                                                       |
| [**IUIAnimationManager2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2)<br/>                         | Define um [**Gerenciador de animação**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2), que fornece uma interface central para criar e gerenciar animações em várias dimensões.<br/> |
| [**IUIAnimationManagerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler)<br/>   | Define um método para lidar com atualizações de status para um Gerenciador de animação.<br/>                                                                                         |
| [**IUIAnimationManagerEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler2)<br/> | Define um método para lidar com atualizações para um [**Gerenciador de animação**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2).<br/>                                                                |
