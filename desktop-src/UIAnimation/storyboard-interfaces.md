# <a name="storyboard-interfaces"></a>Interfaces de storyboard

Esta seção contém as especificações de referência para as interfaces do Gerenciador de animação do Windows que dão suporte a storyboards.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                 | Descrição                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationStoryboard**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboard)<br/>                                   | Define um storyboard, que contém um grupo de transições que são sincronizadas em relação umas às outras.<br/> |
| [**IUIAnimationStoryboard2**](/windows/win32/api/uianimation/nn-uianimation-iuianimationstoryboard2)<br/>                                 | Define um storyboard, que contém um grupo de transições que são sincronizadas em relação umas às outras.<br/> |
| [**IUIAnimationStoryboardEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler)<br/>           | Define métodos para lidar com status e eventos de atualização para um Storyboard.<br/>                                    |
| [**IUIAnimationStoryboardEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler2)<br/>         | Define métodos para manipular eventos de storyboard. <br/>                                                           |
| [**IUIAnimationLoopIterationChangeHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationloopiterationchangehandler2)<br/> | Define um método para manipular eventos de iteração de loop Storyboard.<br/>                                            |
| [**IUIAnimationPriorityComparison**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison)<br/>                   | Define um método para comparação de prioridade que o Gerenciador de animação usa para resolver conflitos de agendamento.<br/>  |
| [**IUIAnimationPriorityComparison2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison2)<br/>                 | Define um método que resolve conflitos de agendamento por meio de comparação de prioridade.<br/>                           |
