# <a name="id3dx11effectstringvariable-interface"></a>Interface ID3DX11EffectStringVariable

Uma interface de variável de cadeia de caracteres acessa uma variável de cadeia de caracteres.

## <a name="members"></a>Membros

A interface **ID3DX11EffectStringVariable** herda de [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectStringVariable** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **ID3DX11EffectStringVariable** tem esses métodos.



| Método                                                               | Descrição                         |
|:---------------------------------------------------------------------|:------------------------------------|
| [**GetString**](id3dx11effectstringvariable-getstring.md)           | Obter a cadeia de caracteres.<br/>          |
| [**GetStringArray**](id3dx11effectstringvariable-getstringarray.md) | Obter uma matriz de cadeias de caracteres.<br/> |



 

## <a name="remarks"></a>Comentários

> [!Note]  
> O SDK do DirectX não fornece nenhum binário compilado para efeitos. Você deve usar a fonte Effects 11 para criar seu aplicativo de tipo de efeitos. Para obter mais informações sobre como usar a fonte Effects 11, consulte [diferenças entre os efeitos 10 e os efeitos 11](d3d11-graphics-programming-guide-effects-differences.md).
