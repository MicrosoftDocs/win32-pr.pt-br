# <a name="cd3d11-helper-structures"></a>Estruturas auxiliares CD3D11

O Direct3D 11 define várias estruturas auxiliares que você pode usar para criar estruturas de Direct3D. Essas estruturas auxiliares se comportam como classes C++.

| Tópico                                                                                         | Descrição                                                                                                                         |
|-----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**CD3D11 \_ Rect**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_rect)<br/>                                                | Representa um retângulo e fornece métodos de conveniência para a criação de retângulos.<br/>                                         |
| [**Caixa de CD3D11 \_**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_box)<br/>                                                  | Representa uma caixa e fornece métodos de conveniência para a criação de caixas.<br/>                                                    |
| [**Desc. de estêncil de profundidade de CD3D11 \_ \_ \_**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_depth_stencil_desc)<br/>                  | Representa uma estrutura de estado de estêncil de profundidade e fornece métodos de conveniência para criar estruturas de estado de estêncil de profundidade.<br/> |
| [**CD3D11 de \_ combinação \_ desc**](/windows/desktop/api/D3D11/ns-d3d11-cd3d11_blend_desc)<br/>                                   | Representa uma estrutura de estado de mistura e fornece métodos de conveniência para criar estruturas de estado de mesclagem.<br/>                 |

| [**CD3D11 \_ buffer \_ desc**](/windows/desktop/api/D3D11/ns-d3d11-cd3d11_buffer_desc)<br/>                                 | Representa um buffer e fornece métodos de conveniência para a criação de buffers.<br/>                                               |
