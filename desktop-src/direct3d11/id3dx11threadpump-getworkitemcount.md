# <a name="id3dx11threadpumpgetworkitemcount-method"></a>Método ID3DX11ThreadPump:: GetWorkItemCount

> [!Note]  
> A biblioteca de utilitários D3DX (D3DX 9, D3DX 10 e D3DX 11) foi preterida para o Windows 8 e não tem suporte para aplicativos da Windows Store.

 

Obtém o número de itens de trabalho na bomba do thread.

## <a name="syntax"></a>Sintaxe


```C++
UINT GetWorkItemCount();
```



## <a name="parameters"></a>Parâmetros

Esse método não tem parâmetros.

## <a name="return-value"></a>Retornar valor

Tipo: **[ **uint**](/windows/desktop/WinProg/windows-data-types)**

O número de itens de trabalho na fila na bomba do thread.
