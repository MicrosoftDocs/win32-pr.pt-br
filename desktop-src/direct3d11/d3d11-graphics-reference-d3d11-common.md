# <a name="common-version-reference"></a>Referência de versão comum

A API do Direct3D define vários elementos de API que são comuns ao Direct3D 12, Direct3D 11, Direct3D 10 e Direct3D 10,1. Você pode usar esses elementos de API em seu código para qualquer uma dessas versões do Direct3D. Esses elementos de API são conhecidos como versão neutra.
