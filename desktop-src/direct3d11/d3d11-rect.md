# <a name="d3d11_rect"></a>D3D11 \_ Rect

D3D11 \_ RECT é declarado da seguinte maneira:

``` syntax
typedef RECT D3D11_RECT;
```

Para obter mais informações sobre essa estrutura de retângulo GDI, consulte [**Rect**](/previous-versions//dd162897(v=vs.85)).

## <a name="remarks"></a>Comentários

Essa estrutura é usada para retângulos de recorte por [**ID3D11DeviceContext:: RSGetScissorRects**](/windows/desktop/api/D3D11/nf-d3d11-id3d11devicecontext-rsgetscissorrects) e [**ID3D11DeviceContext:: RSSetScissorRects**](/windows/desktop/api/D3D11/nf-d3d11-id3d11devicecontext-rssetscissorrects).
