# <a name="id3dx11effectinterfacevariable-interface"></a>Interface ID3DX11EffectInterfaceVariable

Acessa uma variável de interface.

## <a name="members"></a>Membros

A interface **ID3DX11EffectInterfaceVariable** herda de [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectInterfaceVariable** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **ID3DX11EffectInterfaceVariable** tem esses métodos.



| Método                                                                      | Descrição                       |
|:----------------------------------------------------------------------------|:----------------------------------|
| [**GetClassInstance**](id3dx11effectinterfacevariable-getclassinstance.md) | Obter uma instância de classe.<br/>  |
| [**SetClassInstance**](id3dx11effectinterfacevariable-setclassinstance.md) | Define uma instância de classe.<br/> |



 

## <a name="remarks"></a>Comentários

> [!Note]  
> O SDK do DirectX não fornece nenhum binário compilado para efeitos. Você deve usar a fonte Effects 11 para criar seu aplicativo de tipo de efeitos. Para obter mais informações sobre como usar a fonte Effects 11, consulte [diferenças entre os efeitos 10 e os efeitos 11](d3d11-graphics-programming-guide-effects-differences.md).
