# <a name="d3dx-11-reference"></a>Referência do D3DX 11

A API D3DX 11 é descrita nesta seção.

> [!NOTE]  
> A biblioteca de utilitários D3DX (D3DX 9, D3DX 10 e D3DX 11) foi preterida para Windows 8 e não tem suporte para aplicativos Microsoft Store. Para obter detalhes sobre as substituições recomendadas para D3DX11, consulte [Morando sem D3DX.](https://walbourn.github.io/living-without-d3dx/) Para projetos mais antigos, você pode usar o pacote de NuGet [Microsoft.DXSDK.D3DX](https://www.nuget.org/packages/Microsoft.DXSDK.D3DX) em vez de depender do SDK do DirectX herdado ou do DirectSetup.
