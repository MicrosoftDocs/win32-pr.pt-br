# <a name="resource-reference-direct3d-11-graphics"></a>Referência de recurso (gráficos do Direct3D 11)

A API do Direct3D define vários elementos de API para ajudá-lo a criar e gerenciar recursos.


## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                              | Descrição                                                                                                                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaces de recurso](d3d11-graphics-reference-resource-interfaces.md)<br/> | Há várias interfaces para os dois tipos básicos de recursos: buffers e texturas. Também há interfaces para exibições de recursos. <br/> Um aplicativo usa uma exibição para associar um recurso a um estágio de pipeline. A exibição define como o recurso pode ser acessado durante a renderização. Esta seção descreve as interfaces de exibição. <br/> |
| [Funções de recurso](d3d11-graphics-reference-resource-functions.md)<br/>   | Esta seção contém informações sobre as funções de recurso.<br/>                                                                                                                                                                                                                                                                                |
