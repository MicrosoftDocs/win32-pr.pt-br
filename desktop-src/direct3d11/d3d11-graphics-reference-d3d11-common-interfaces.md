# <a name="common-version-interfaces"></a>Interfaces de versão comuns

Esta seção contém informações sobre as interfaces de versão comuns.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**ID3D10Blob**](/windows/win32/api/d3dcommon/nn-d3dcommon-id3d10blob) | Essa interface é usada para retornar dados de comprimento arbitrário. |
| [**ID3DBlob**](/previous-versions/windows/desktop/legacy/ff728743(v=vs.85)) | Essa interface é usada para retornar dados de comprimento arbitrário. |
| [**ID3DDestructionNotifier**](/windows/win32/api/d3dcommon/nn-d3dcommon-id3ddestructionotifier) | Usado para se registrar para retornos de chamada quando um objeto nano 3D direto for destruído. |
| [**ID3DInclude**](/windows/desktop/api/D3Dcommon/nn-d3dcommon-id3dinclude) | [**ID3DInclude**](/windows/desktop/api/D3Dcommon/nn-d3dcommon-id3dinclude) é uma interface de inclusão que o usuário implementa para permitir que um aplicativo chame métodos substituíveis pelo usuário para abrir e fechar arquivos de inclusão de sombreador \# . |
| [**ID3DUserDefinedAnnotation**](/windows/desktop/api/D3D11_1/nn-d3d11_1-id3duserdefinedannotation) | A interface [**ID3DUserDefinedAnnotation**](/windows/desktop/api/D3D11_1/nn-d3d11_1-id3duserdefinedannotation) permite que um aplicativo Descreva seções conceituais e marcadores dentro do fluxo de código do aplicativo. Uma ferramenta habilitada adequadamente, como o Microsoft Visual Studio Ultimate 2012, pode exibir essas seções e marcadores visualmente ao longo da linha de tempo do Microsoft Direct3D da ferramenta, enquanto a ferramenta desbugs o aplicativo. Essas observações visuais permitem que os usuários de tal ferramenta naveguem para partes da linha de tempo que são de interesse ou para entender qual conjunto de chamadas Direct3D são produzidos por determinadas seções do código do aplicativo. |
