# <a name="types-of-resources"></a>Tipos de recursos

Os recursos são áreas na memória que podem ser acessadas pelo pipeline do Direct3D, são os blocos de construção da sua cena. Os recursos contêm muitos tipos de dados, como geometria, texturas ou dados de sombreador que o Direct3D usa para popular e renderizar sua cena. Este tópico descreve os tipos de recursos do Direct3D 10, bem como os novos tipos no Direct3D 11, incluindo buffers estruturados e texturas e buffers graváveis.
