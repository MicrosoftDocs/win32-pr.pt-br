# <a name="direct3d-11-graphics"></a>Elementos gráficos do Direct3D 11

Você pode usar gráficos do Microsoft Direct3D 11 para criar gráficos 3D para jogos e aplicativos científicos e de área de trabalho.

Esta seção contém informações sobre programação com elementos gráficos do Direct3D 11.

Para obter mais informações, consulte [Recursos do Direct3D 11](direct3d-11-features.md).



|                                   | Com suporte                                                                                                  |
|-----------------------------------|---------------------------------------------------------------------------------------------------|
| **Ambientes de runtime com suporte**    | Windows/C++            |
| **Linguagens de programação recomendadas** | C/C++                                                                                             |
| **Cliente mínimo com suporte**          | Windows 7             |
| **Servidor mínimo com suporte**          | Windows Server 2008 R2 |

| [Novidades no Direct3D 11](dx-graphics-overviews-introduction.md)<br/> | Esta seção descreve os recursos adicionados no Direct3D 11, Direct3D 11.1 e Direct3D 11.2.<br/>                                                                                                    |
| [Guia de programação para Direct3D 11](dx-graphics-overviews.md)<br/>      | O guia de programação contém informações sobre como usar o pipeline programável do Direct3D 11 para criar gráficos 3D em tempo real para jogos, bem como aplicativos científicos e de área de trabalho.<br/> |

Além do Direct3D 11 ser suportado pelo Windows 7 e posterior e pelo Windows Server 2008 R2 e posterior, o Direct3D 11 está disponível no nível inferior para o Windows Vista com Service Pack 2 (SP2) e Windows Server 2008 baixando [KB 971644](https://support.microsoft.com/kb/971644) e [KB 971512](https://support.microsoft.com/kb/971512/).

Para obter informações sobre os novos recursos do Direct3D 11.1 que estão incluídos no Windows 8, no Windows Server 2012 e estão parcialmente disponíveis no Windows 7 e no Windows Server 2008 R2 por meio da Atualização da Plataforma para [o Windows 7,](https://support.microsoft.com/kb/2670838)consulte Recursos do [Direct3D 11.1](direct3d-11-1-features.md) e Atualização da plataforma [para Windows 7](/windows/desktop/direct3darticles/platform-update-for-windows-7).
