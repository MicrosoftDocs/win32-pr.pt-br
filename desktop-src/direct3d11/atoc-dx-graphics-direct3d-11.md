# <a name="direct3d-11-graphics"></a>Elementos gráficos do Direct3D 11

Você pode usar gráficos do Microsoft Direct3D 11 para criar gráficos 3D para jogos e aplicativos de área de trabalho e científica.

Esta seção contém informações sobre a programação com gráficos do Direct3D 11.

Para obter mais informações, consulte [recursos do Direct3D 11](direct3d-11-features.md).

Além do Direct3D 11 ter suporte do Windows 7 e posterior e do Windows Server 2008 R2 e posterior, o Direct3D 11 está disponível de nível inferior para o Windows Vista com Service Pack 2 (SP2) e o Windows Server 2008 baixando [kb 971644](https://support.microsoft.com/kb/971644) e [KB 971512](https://support.microsoft.com/kb/971512/).

Para obter informações sobre os novos recursos do Direct3D 11,1 que estão incluídos no Windows 8, no Windows Server 2012 e estão parcialmente disponíveis no Windows 7 e no Windows Server 2008 R2 por meio da [atualização da plataforma para o Windows 7](https://support.microsoft.com/kb/2670838), consulte [recursos do Direct3D 11,1](direct3d-11-1-features.md) e a [atualização da plataforma para o Windows 7](/windows/desktop/direct3darticles/platform-update-for-windows-7).
