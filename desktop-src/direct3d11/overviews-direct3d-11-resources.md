# <a name="resources"></a>Recursos

Os recursos fornecem dados para o pipeline e definem o que é renderizado durante a cena. Os recursos podem ser carregados a partir da mídia de jogo ou criados dinamicamente no momento de execução. Normalmente, os recursos incluem dados de textura, dados de vértice e dados de sombreador. A maioria dos aplicativos Direct3D cria e destrói recursos extensivamente ao longo do tempo de vida. Esta seção descreve os aspectos dos recursos do Direct3D 11.
