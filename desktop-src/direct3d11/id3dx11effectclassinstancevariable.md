# <a name="id3dx11effectclassinstancevariable-interface"></a>Interface ID3DX11EffectClassInstanceVariable

Acessa uma instância de classe.

## <a name="members"></a>Membros

A interface **ID3DX11EffectClassInstanceVariable** herda de [**ID3DX11EffectVariable.**](id3dx11effectvariable.md) **ID3DX11EffectClassInstanceVariable** também tem estes tipos de membros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

A interface **ID3DX11EffectClassInstanceVariable** tem esses métodos.



| Método                                                                          | Descrição                       |
|:--------------------------------------------------------------------------------|:----------------------------------|
| [**GetClassInstance**](id3dx11effectclassinstancevariable-getclassinstance.md) | Obtém uma instância de classe.<br/> |



 

## <a name="remarks"></a>Comentários

> [!Note]  
> O SDK do DirectX não fornece binários compilados para efeitos. Você deve usar a origem efeitos 11 para criar seu aplicativo do tipo efeitos. Para obter mais informações sobre como usar a origem dos Efeitos 11, consulte [Diferenças entre efeitos 10 e efeitos 11.](d3d11-graphics-programming-guide-effects-differences.md)
