# <a name="whats-new-in-direct3d-11"></a>Novidades no Direct3D 11

Esta seção descreve os recursos adicionados no Direct3D 11, Direct3D 11.1 e Direct3D 11.2.

O Microsoft Direct3D 11 é uma extensão da API de renderização do Microsoft Direct3D 10/Microsoft Direct3D 10.1. Para obter mais material introdutório sobre como usar o Direct3D 11, consulte o Guia de Programação [para Direct3D 10](/windows/desktop/direct3d10/d3d10-graphics-programming-guide).
