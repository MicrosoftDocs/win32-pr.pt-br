-   <span data-ttu-id="7d52c-116">Restrições de alinhamento para formas de bloco padrão: os mipmaps que preenchem pelo menos um bloco padrão em todas as dimensões usam garantidamente o agrupamento lado a lado padrão, com o restante considerado compactado como uma **unidade** em blocos N (N relatado ao aplicativo).</span><span class="sxs-lookup"><span data-stu-id="7d52c-116">Alignment constraints for standard tile shapes: Mipmaps that fill at least one standard tile in all dimensions are guaranteed to use the standard tiling, with the remainder considered packed as a **unit** into N tiles (N reported to the application).</span></span> <span data-ttu-id="7d52c-117">O aplicativo pode mapear os blocos N para locais arbitrariamente separados em um pool de blocos, mas deve mapear todos ou nenhum dos blocos compactados.</span><span class="sxs-lookup"><span data-stu-id="7d52c-117">The application can map the N tiles into arbitrarily disjoint locations in a tile pool, but must either map all or none of the packed tiles.</span></span> <span data-ttu-id="7d52c-118">A compactação MIP é um conjunto exclusivo de blocos compactados por fatia de matriz.</span><span class="sxs-lookup"><span data-stu-id="7d52c-118">The mip packing is a unique set of packed tiles per array slice.</span></span>
-   <span data-ttu-id="7d52c-119">Há suporte para a filtragem de redução mínima/máxima.</span><span class="sxs-lookup"><span data-stu-id="7d52c-119">Min/Max reduction filtering is supported.</span></span> <span data-ttu-id="7d52c-120">Para obter informações sobre a filtragem de redução mín./máx., consulte recursos de [amostragem de textura de recursos de lado](tiled-resources-texture-sampling-features.md).</span><span class="sxs-lookup"><span data-stu-id="7d52c-120">For info about Min/Max reduction filtering, see [Tiled resources texture sampling features](tiled-resources-texture-sampling-features.md).</span></span>

-   <span data-ttu-id="7d52c-122">Limitações de como os blocos podem ser acessados quando há mapeamentos duplicados, descritos em [limitações de acesso de bloco com mapeamentos duplicados](tile-access-limitations-with-duplicate-mappings-.md), continuar a aplicar.</span><span class="sxs-lookup"><span data-stu-id="7d52c-122">Limitations on how tiles can be accessed when there are duplicate mappings, described in [Tile access limitations with duplicate mappings](tile-access-limitations-with-duplicate-mappings-.md), continue to apply.</span></span>
