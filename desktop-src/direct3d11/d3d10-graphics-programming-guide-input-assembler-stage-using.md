<span data-ttu-id="acde8-128">O estágio IA adicionará uma ID primitiva a cada primitiva para uso pelo sombreador geometry ou pelo estágio pixel shader (o que for o primeiro estágio ativo após o estágio IA).</span><span class="sxs-lookup"><span data-stu-id="acde8-128">The IA stage will add a primitive id to each primitive for use by the geometry shader or the pixel shader stage (whichever is the first stage active after the IA stage).</span></span> <span data-ttu-id="acde8-129">Para cada chamada de desenho indexada, a ID primitiva é incrementada em 1, no entanto, a ID primitiva é redefinida para 0 sempre que uma nova instância é iniciada.</span><span class="sxs-lookup"><span data-stu-id="acde8-129">For each indexed draw call, the primitive id is incremented by 1, however, the primitive id resets to 0 whenever a new instance begins.</span></span> <span data-ttu-id="acde8-130">Todas as outras chamadas de desenho não alteram o valor da ID da instância.
