# <a name="shader-reference-direct3d-11-graphics"></a>Referência de sombreador (gráficos do Direct3D 11)

A API do Direct3D define vários elementos de API para ajudá-lo a criar e gerenciar sombreadores programáveis. Os sombreadores são programas executáveis que são programados exclusivamente usando HLSL.
