# <a name="palette-awareness"></a>Reconhecimento de paleta

Seu aplicativo deve responder às [mensagens WM \_ PALETTECHANGED,](/windows/desktop/gdi/wm-palettechanged) [WM \_ QUERYNEWPALETTE](/windows/desktop/gdi/wm-querynewpalette)e [WM \_ ACTIVATE](../inputdev/wm-activate.md) para estar ciente e usar paletas. Projete seu aplicativo para selecionar e realizar paletas em resposta a essas mensagens.

Para obter mais informações sobre paletas e reconhecimento de paletas, consulte os artigos "Reconhecimento de paleta" e "O Gerenciador de Paletas: Como e Por que" nos discos compactos da Biblioteca de Desenvolvimento de Rede Desenvolvedor Microsoft.
