# <a name="depth-buffer-test"></a>Teste de buffer de profundidade

O teste de buffer de profundidade descartará um fragmento se uma comparação de profundidade falhar; [**glDepthFunc**](gldepthfunc.md) especifica a função de comparação. Se a tensão estiver habilitada, o resultado da comparação de profundidade também afetará o valor de atualização do buffer de estêncil.
