# <a name="appendix-opengl"></a>Apêndice (OpenGL)

O OpenGL, originalmente desenvolvido pelo Silicon Graphics Incorporated (SGI) para suas estações de trabalho gráficas, permite que os aplicativos criem imagens de cores de alta qualidade independentes de sistemas de janelas, sistemas operacionais e hardware.

A ARB (placa de análise da arquitetura do OpenGL), um consórcio do setor, é responsável atualmente pela definição do OpenGL. Os membros do ARB incluem o silício gráfico Incorporated, a Microsoft Corporation, a Intel, a IBM e a Digital Equipment Corporation.

O documento de referência oficial para OpenGL versão 1, é o *manual de referência do OpenGL*, pela placa de revisão da arquitetura do OpenGL (ISBN 0-201-63276-4). O guia oficial para aprender o OpenGL versão 1 é o *Guia de programação OpenGL*, por Jackie neider, Tom Davis e Mason Woo (ISBN 0-201-63274-8). Ambos os livros são publicados por Addison-Wesley.
