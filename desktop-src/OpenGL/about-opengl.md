# <a name="appendix-opengl"></a>Apêndice (OpenGL)

O OpenGL, originalmente desenvolvido pela SGI (Silicon Graphics Incorporated) para suas estações de trabalho gráficas, permite que os aplicativos criem imagens de cores de alta qualidade, independentemente de sistemas de janelas, sistemas operacionais e hardware.

Atualmente, o ARB (OpenGL Architecture Review Board), um consórcio do setor, é responsável por definir o OpenGL. Os membros do ARB incluem Silicon Graphics Incorporated, Microsoft Corporation, Intel, IBM e Digital Equipment Corporation.

O documento de referência oficial do OpenGL versão 1 é o Manual de Referência do *OpenGL,* da Placa de Revisão de Arquitetura openGL (ISBN 0-201-63276-4). O guia oficial para aprender o OpenGL versão 1 é o Guia de Programação *OpenGL,* de Nós Neider, Tom Brown e Woo Woo (ISBN 0-201-63274-8). Ambos os livros são publicados por Ltdn-Bailey.
