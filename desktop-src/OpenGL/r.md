# <a name="r-opengl"></a>R (OpenGL)

<dl> <dt>

<span id="opengl_rasterize"></span><span id="OPENGL_RASTERIZE"></span>**Rasterize**
</dt> <dd>

Para converter um ponto projetado, linha ou polígono ou os pixels de um bitmap ou imagem, em fragmentos, cada um correspondendo a um pixel no framebuffer. Observe que todos os primitivos são rasterizados, não apenas pontos, linhas e polígonos

</dd> <dt>

<span id="opengl_rectangle"></span><span id="OPENGL_RECTANGLE"></span>**Retângulo**
</dt> <dd>

Um lado a lado cujas bordas alternativas são paralelas umas às outras em coordenadas de objeto. Polígonos especificados com glRect ( ) são \* sempre retângulos; outros retângulos podem ser retângulos.

</dd> <dt>

<span id="opengl_rendering"></span><span id="OPENGL_RENDERING"></span>**Renderização**
</dt> <dd>

Conversão de primitivos especificados em coordenadas de objeto em uma imagem no framebuffer. A renderização é a operação principal do OpenGL.

</dd> <dt>

<span id="opengl_rgba"></span><span id="OPENGL_RGBA"></span>**Rgba**
</dt> <dd>

Os componentes de cor vermelho, verde, azul e alfa do modo RGBA.

</dd> <dt>

<span id="opengl_rgba_mode"></span><span id="OPENGL_RGBA_MODE"></span>**Modo RGBA**
</dt> <dd>

Um contexto OpenGL no qual seus buffers de cores armazenam componentes de cor vermelho, verde, azul e alfa, em vez de índices de cores.

</dd> </dl>
