# <a name="r-opengl"></a>R (OpenGL)

<dl> <dt>

<span id="opengl_rasterize"></span><span id="OPENGL_RASTERIZE"></span>**rasterizar**
</dt> <dd>

Para converter um ponto, uma linha ou um polígono projetado, ou os pixels de um bitmap ou imagem, para fragmentos, cada um correspondendo a um pixel no framebuffer. Observe que todos os primitivos são rasterizados, não apenas pontos, linhas e polígonos

</dd> <dt>

<span id="opengl_rectangle"></span><span id="OPENGL_RECTANGLE"></span>**nele**
</dt> <dd>

Um diamante cujas bordas alternativas são paralelas entre si em coordenadas de objeto. Os polígonos especificados com glRect \* () são sempre retângulos; outros quadrilaterals podem ser retângulos.

</dd> <dt>

<span id="opengl_rendering"></span><span id="OPENGL_RENDERING"></span>**tratamento**
</dt> <dd>

Conversão de primitivos especificados em coordenadas de objeto em uma imagem no framebuffer. A renderização é a principal operação do OpenGL.

</dd> <dt>

<span id="opengl_rgba"></span><span id="OPENGL_RGBA"></span>**RGBA**
</dt> <dd>

Os componentes de cor vermelha, verde, azul e alfa do modo RGBA.

</dd> <dt>

<span id="opengl_rgba_mode"></span><span id="OPENGL_RGBA_MODE"></span>**Modo RGBA**
</dt> <dd>

Um contexto OpenGL no qual seus buffers de cor armazenam componentes de cor vermelho, verde, azul e alfa, em vez de índices de cores.

</dd> </dl>
