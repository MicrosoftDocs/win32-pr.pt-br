# <a name="opengl-utility-library"></a>Biblioteca do Utilitário OpenGL

A biblioteca GLU (Utilitário OpenGL) contém vários grupos de funções que complementam a interface principal do OpenGL, fornecendo suporte para recursos auxiliares. Essas funções de utilitário usam as funções principais do OpenGL, portanto, qualquer implementação do OpenGL tem a garantia de dar suporte às funções do utilitário.

> [!Note]  
> O prefixo para funções de biblioteca do Utilitário é "glu" em vez de "gl".

 

Muitas dessas funções são descritas nas seções anteriores da documenation à medida que seus tópicos surgem. Essas funções são listadas aqui para a conclusão. As funções GLU que não são discutidas nas seções anteriores são descritas aqui. Para obter descrições mais detalhadas dessas funções, consulte o *Manual de Referência do OpenGL.* Esta seção grupos de funções GLU relacionadas, da seguinte forma:

-   [Polígonos de mosaico](tessellating-polygons.md)
-   [Usando funções de retorno de chamada](using-callback-functions.md)
-   [Usando objetos de mosaico](using-tessellation-objects.md)
-   [Especificando retornos de chamada](specifying-callbacks.md)
