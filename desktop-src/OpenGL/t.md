# <a name="t-opengl"></a>T (OpenGL)

[A](a.md) [B](b.md) [C](c.md) [D](d.md) [E](e.md) [F](f.md) [G](g.md) [H](h.md) [i](i.md) [J K](jk.md) [L](l.md) [M](m.md) [N](n.md) [O](o.md) [P](p.md) [Q](q.md) [R](r.md) [S](s.md) T [U V](u-v.md) [W](w.md) [X Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_tessellation"></span><span id="OPENGL_TESSELLATION"></span>**mosaico**
</dt> <dd>

Redução de uma parte de uma superfície analítica para uma malha de polígonos ou de uma parte de uma curva analítica para uma sequência de linhas.

</dd> <dt>

<span id="opengl_texel"></span><span id="OPENGL_TEXEL"></span>**Texel**
</dt> <dd>

Um elemento Texture. Um Texel é obtido da memória de textura e representa a cor da textura a ser aplicada a um fragmento correspondente. Consulte também [fragmento](f.md).

</dd> <dt>

<span id="opengl_texture"></span><span id="OPENGL_TEXTURE"></span>**textura**
</dt> <dd>

Uma imagem de uma ou duas dimensões usada para modificar a cor dos fragmentos produzidos pela rasterização. Consulte também [rasterizar](r.md).

</dd> <dt>

<span id="opengl_texture_mapping"></span><span id="OPENGL_TEXTURE_MAPPING"></span>**mapeamento de textura**
</dt> <dd>

O processo de aplicação de uma imagem (a textura) a um primitivo. O mapeamento de textura geralmente é usado para adicionar Realm a uma cena. Por exemplo, você pode aplicar uma imagem de uma fachada de construção a um polígono que representa uma parede. Consulte também textura.

</dd> <dt>

<span id="opengl_texture_matrix"></span><span id="OPENGL_TEXTURE_MATRIX"></span>**matriz de textura**
</dt> <dd>

A matriz 4x4 que transforma as coordenadas de textura das coordenadas nas quais elas são especificadas nas coordenadas que são usadas para a interpolação e pesquisa de textura.

</dd> <dt>

<span id="opengl_transformation"></span><span id="OPENGL_TRANSFORMATION"></span>**adaptação**
</dt> <dd>

Uma detorção de espaço. No OpenGL, as transformações são limitadas a transformações projetadas que incluem qualquer coisa que possa ser representada por uma matriz 4x4. Essas transformações incluem rotações, conversões, (não uniformes) dimensionamentos ao longo dos eixos de coordenadas, transformações de perspectiva e combinações desses.

</dd> <dt>

<span id="opengl_triangle"></span><span id="OPENGL_TRIANGLE"></span>**ângulo**
</dt> <dd>

Um polígono com três bordas. Triângulos são sempre convexa.

</dd> </dl>
