# <a name="g-opengl"></a>G (OpenGL)

[A](a.md) [B](b.md) [C](c.md) [D](d.md) [E](e.md) [F](f.md) G [H](h.md) [i](i.md) [J K](jk.md) [L](l.md) [M](m.md) [N](n.md) [O](o.md) [P](p.md) [Q](q.md) [R](r.md) [S](s.md) [T](t.md) [U V](u-v.md) [W](w.md) [X Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_gamma_correction"></span><span id="OPENGL_GAMMA_CORRECTION"></span>**correção gama**
</dt> <dd>

Uma função aplicada às cores armazenadas no buffer de quadro para corrigir a resposta não linear do olho (e, às vezes, do monitor) a alterações lineares em valores de intensidade de cor.

</dd> <dt>

<span id="opengl_geometric_model"></span><span id="OPENGL_GEOMETRIC_MODEL"></span>**modelo geométrico**
</dt> <dd>

Os vértices de coordenadas de objeto e os parâmetros que descrevem um objeto. Observe que o OpenGL não define uma sintaxe para modelos geométricos, mas sim uma sintaxe e uma semântica para a renderização de modelos geométricos.

</dd> <dt>

<span id="opengl_geometric_object"></span><span id="OPENGL_GEOMETRIC_OBJECT"></span>**objeto geométrico**
</dt> <dd>

Um modelo geométrico.

</dd> <dt>

<span id="opengl_geometric_primitive"></span><span id="OPENGL_GEOMETRIC_PRIMITIVE"></span>**primitivo Geométrico**
</dt> <dd>

Um ponto, uma linha ou um polígono.

</dd> <dt>

<span id="opengl_gouraud_shading"></span><span id="OPENGL_GOURAUD_SHADING"></span>**Sombreamento Gouraud**
</dt> <dd>

Ajuste de interpolação suave de cores em um polígono ou segmento de linha. As cores são atribuídas em vértices e interpoladas linearmente por meio do primitivo para produzir uma variação relativamente suave em cores. Também chamado de sombreamento suave.

</dd> <dt>

<span id="opengl_group"></span><span id="OPENGL_GROUP"></span>**Group**
</dt> <dd>

Um grupo de um, dois, três ou quatro elementos que representa cada pixel de uma imagem na memória do cliente. Portanto, no contexto de uma imagem de memória do cliente, um grupo e um pixel são a mesma coisa.

</dd> </dl>
