# <a name="porting-to-opengl"></a>Portando para OpenGL

O OpenGL foi projetado para compatibilidade entre hardware e sistemas operacionais. Esse design facilita para os programadores a portabilidade de programas OpenGL de um sistema para outro. Embora cada sistema operacional tenha requisitos exclusivos, grande parte do código OpenGL em seus programas atuais pode ser usada como está. Para por o aplicativo OpenGL para o Microsoft Windows, você terá que modificar seus programas para trabalhar com os sistemas de Windows janelas.

Em geral, os aplicativos são portados para o OpenGL Windows de uma das duas plataformas:

-   Aplicativos OpenGL desenvolvidos para o Sistema de JanelaS X e a biblioteca X (Xlib)
-   Aplicativos IRIS GL
