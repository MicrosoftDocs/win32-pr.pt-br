# <a name="porting-to-opengl"></a>Portando para OpenGL

O OpenGL foi projetado para compatibilidade entre hardware e sistemas operacionais. Esse design torna mais fácil para os programadores portarem programas OpenGL de um sistema para outro. Embora cada sistema operacional tenha requisitos exclusivos, grande parte do código OpenGL em seus programas atuais pode ser usada como está. Para portar seu aplicativo OpenGL para o Microsoft Windows, você precisará modificar seus programas para trabalhar com os sistemas de janelas do Windows.

Em geral, os aplicativos são portados para o OpenGL para Windows de uma das duas plataformas:

-   Aplicativos OpenGL desenvolvidos para o sistema de janelas X e a biblioteca X (Xlib)
-   Aplicativos da íris GL
