# <a name="assembling-primitives"></a>Montando primitivos

Depois que todos os cálculos necessários forem realizados, os vértices serão montados em pontos [primitivos](primitives.md) , segmentos de linha ou polígonos junto com as informações relevantes de sinalizador de borda, cor e textura para cada vértice.
