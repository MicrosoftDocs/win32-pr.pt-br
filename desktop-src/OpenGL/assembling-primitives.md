# <a name="assembling-primitives"></a>Assembling Primitives

Depois que todos os cálculos necessários foram executados, os vértices são montados em pontos [primitivos, segmentos](primitives.md) de linha ou polígonos junto com o sinalizador de borda, a cor e as informações de textura relevantes para cada vértice.
