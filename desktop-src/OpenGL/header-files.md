# <a name="header-files"></a>Arquivos de cabeçalho

Aplicativos que usam:

-   As funções principais do OpenGL devem incluir o arquivo de cabeçalho <GL \\ GL. h>.
-   A biblioteca do utilitário OpenGL deve incluir o arquivo de cabeçalho <GL \\ Glu. h>.
-   A biblioteca auxiliar do guia de programação OpenGL deve incluir o arquivo de cabeçalho <GL \\ Glaux. h>.
-   As funções WGL devem incluir o arquivo de cabeçalho Windows. h.
-   As funções do Windows que dão suporte à implementação do OpenGL no Windows da Microsoft devem incluir o arquivo de cabeçalho Windows. h.
