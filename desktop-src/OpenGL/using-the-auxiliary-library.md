# <a name="using-the-auxiliary-library"></a>Usando a biblioteca auxiliar

Usando o OpenGL, o Silicon Graphics (SGI) criou a biblioteca auxiliar para escrever programas de exemplo simples para o *Guia de programação OpenGL*. O código-fonte da biblioteca auxiliar é fornecido com o SDK da plataforma Microsoft, juntamente com os exemplos de OpenGL. Para entender como a biblioteca auxiliar foi desenvolvida a partir de funções e rotinas OpenGL, examine o código-fonte. Você pode usar funções de biblioteca auxiliar em seus próprios programas. Para obter uma descrição da biblioteca auxiliar, consulte o *Guia de programação OpenGL*.
