# <a name="f-opengl"></a>F (OpenGL)

[A](a.md) [B](b.md) [C](c.md) [D](d.md) [E](e.md) F [G](g.md) [H](h.md) [i](i.md) [J K](jk.md) [L](l.md) [M](m.md) [N](n.md) [O](o.md) [P](p.md) [Q](q.md) [R](r.md) [S](s.md) [T](t.md) [U V](u-v.md) [W](w.md) [X Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_face"></span><span id="OPENGL_FACE"></span>**sorridente**
</dt> <dd>

Um lado de um polígono. Cada polígono tem duas faces: uma face frontal e uma face de fundo. Apenas uma face está visível na janela. Se a face traseira ou frontal está visível é efetivamente determinada depois que o polígono é projetado na janela. Após essa projeção, se as bordas do polígono forem direcionadas para o sentido horário, uma das faces será visível; Se for direcionado no sentido anti-horário, a outra face estará visível. Se o sentido horário corresponde à frente ou ao verso (e o sentido anti-horário corresponde à parte traseira ou frontal) é determinado pelo programador OpenGL.

</dd> <dt>

<span id="opengl_flat_shading"></span><span id="OPENGL_FLAT_SHADING"></span>**sombreamento simples**
</dt> <dd>

Refere-se à coloração de uma primitiva com uma única cor constante em sua extensão, em vez de interpolar cores suavemente em todo o primitivo. Consulte [sombreamento Gouraud](g.md).

</dd> <dt>

<span id="opengl_fog"></span><span id="OPENGL_FOG"></span>**neblina**
</dt> <dd>

Uma técnica de renderização que pode ser usada para simular efeitos atmosféricas, como criações, neblina e smog, esmaecidas cores de objetos para uma cor de plano de fundo com base na distância do visualizador. A neblina também ajuda na percepção da distância do visualizador, dando uma indicação de profundidade. Consulte também [Depth-advertência](d.md).

</dd> <dt>

<span id="opengl_font"></span><span id="OPENGL_FONT"></span>**la**
</dt> <dd>

Um grupo de representações de caractere gráfico geralmente usado para exibir cadeias de caracteres de texto. Os caracteres podem ser letras romanos, símbolos matemáticos, ideograms asiático, egípcio hieroglyphs e assim por diante.

</dd> <dt>

<span id="opengl_fragment"></span><span id="OPENGL_FRAGMENT"></span>**fragmento**
</dt> <dd>

Dados gráficos gerados pela rasterização de primitivos. Cada fragmento corresponde a um único pixel e inclui valores de coordenadas de cor, profundidade e, às vezes, de textura.

</dd> <dt>

<span id="opengl_framebuffer"></span><span id="OPENGL_FRAMEBUFFER"></span>**framebuffer**
</dt> <dd>

Uma pilha de bitplanes. Todos os buffers de uma determinada janela ou contexto. Às vezes inclui toda a memória de pixel do acelerador de hardware de gráficos. Consulte também [Bitplane](b.md).

</dd> <dt>

<span id="opengl_front_face"></span><span id="OPENGL_FRONT_FACE"></span>**face frontal**
</dt> <dd>

Consulte face.

</dd> <dt>

<span id="opengl_frustum"></span><span id="OPENGL_FRUSTUM"></span>**frustum**
</dt> <dd>

O volume de exibição distorcido por divisão em perspectiva.

</dd> </dl>
