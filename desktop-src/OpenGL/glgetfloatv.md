# <a name="glgetfloatv-function"></a>função glGetFloatv

A função **glGetFloatv** retorna o valor ou os valores de um parâmetro selecionado.

```C++
void WINAPI glGetFloatv(
   GLenum  pname,
   GLfloat *params
```

## <a name="remarks"></a>Comentários

Essa função retorna valores para variáveis de estado simples no OpenGL. O parâmetro *pname* é uma constante simbólica que indica a variável de estado a ser retornada e *params* é um ponteiro para uma matriz do tipo indicado no qual os dados retornados serão colocados.

A conversão de tipo será executada se *params* tiver um tipo diferente do valor da variável de estado que está sendo solicitado. Se você chamar [**glGetBooleanv**](glgetbooleanv.md), um ponto flutuante ou um valor inteiro será convertido em GL \_ false se e somente se for zero. Caso contrário, ele será convertido em GL \_ true.

Se você chamar [**glGetIntegerv**](glgetintegerv.md), valores Boolianos serão retornados como GL \_ true ou GL \_ false e a maioria dos valores de ponto flutuante serão arredondados para o valor inteiro mais próximo. As cores de ponto flutuante e normais, no entanto, são retornadas com um mapeamento linear que mapeia 1,0 para o valor inteiro representável mais positivo e 1,0 para o valor inteiro reapresentável mais negativo.

Se você chamar **glGetFloatv** ou [**glGetDoublev**](glgetdoublev.md), os valores Boolianos serão retornados como GL \_ true ou GL \_ false, e os valores inteiros serão convertidos em valores de ponto flutuante.

Você pode consultar muitos dos parâmetros boolianos mais facilmente com [**glIsEnabled**](glisenabled.md).
