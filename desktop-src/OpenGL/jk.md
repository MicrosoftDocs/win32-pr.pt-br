<dl> <dt>

<span id="opengl_jaggies"></span><span id="OPENGL_JAGGIES"></span>**Jaggies**
</dt> <dd>

Artifacts de renderização com alias. As bordas de primitivos renderizados com alias são irregulares em vez de suaves. Uma linha com alias quase horizontal, por exemplo, é renderizada como um conjunto de linhas horizontais em linhas de pixel adjacentes, em vez de como uma linha suave e contínua.

</dd> </dl>
