<dl> <dt>

<span id="opengl_back_face"></span><span id="OPENGL_BACK_FACE"></span>**verso**
</dt> <dd>

Consulte [face](f.md).

</dd> <dt>

<span id="opengl_bit"></span><span id="OPENGL_BIT"></span>**parte**
</dt> <dd>

Dígito binário. Uma variável de estado que tem apenas dois valores possíveis: 0 ou 1. Os números binários são construções de um ou mais bits.

</dd> <dt>

<span id="opengl_bitmap"></span><span id="OPENGL_BITMAP"></span>**bitmap**
</dt> <dd>

Uma matriz retangular de bits. Além disso, o primitivo renderizado pelo comando [**glBitmap**](glbitmap.md) , que usa seu parâmetro bitmap como uma máscara.

</dd> <dt>

<span id="opengl_bitplane"></span><span id="OPENGL_BITPLANE"></span>**bitplane**
</dt> <dd>

Uma matriz retangular de bits mapeada um-para-um com pixels.

</dd> <dt>

<span id="opengl_blending"></span><span id="OPENGL_BLENDING"></span>**mesclagem**
</dt> <dd>

Reduzir dois componentes de cor para um componente, geralmente como uma interpolação linear entre os dois componentes.

</dd> <dt>

<span id="opengl_buffer"></span><span id="OPENGL_BUFFER"></span>**completo**
</dt> <dd>

Um grupo de bitplanes que armazena um único componente (como profundidade ou verde) ou um único índice (como o índice de cor ou o índice de estêncil). Às vezes, os buffers vermelho, verde, azul e alfa são referidos coletivamente como o buffer de cores (em vez de como os buffers de cores).

</dd> </dl>
