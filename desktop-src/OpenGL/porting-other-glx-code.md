# <a name="porting-other-glx-code"></a>Portando outros códigos de GLX

Além das funções Xlib e GLX descritas nas seções anteriores, o programa provavelmente contém algumas das outras funções GLX ou Xlib listadas na [tradução da biblioteca GLX](translating-the-glx-library.md). Reescreva o código do sistema Window X para o Windows, substituindo as funções apropriadas.
