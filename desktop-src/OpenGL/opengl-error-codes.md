# <a name="opengl-error-codes"></a>Códigos de erro do OpenGL

O OpenGL inclui os seguintes códigos de erro.



| Código do erro             | Descrição                                 |
|------------------------|---------------------------------------------|
| GL \_ INVALID \_ ENUM      | **Argumento GLenum** fora do intervalo.           |
| VALOR INVÁLIDO DE GL \_ \_     | Argumento numérico fora do intervalo.              |
| OPERAÇÃO \_ GL \_ INVÁLIDA | Operação ilegal no estado atual.         |
| GL \_ STACK \_ OVERFLOW    | A função causaria um estouro de pilha.      |
| GL \_ STACK \_ UNDERFLOW   | A função causaria um subfluxo de pilha.     |
| GL \_ SEM \_ \_ MEMÓRIA    | Não há memória suficiente para executar a função. |
