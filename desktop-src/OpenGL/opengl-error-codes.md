# <a name="opengl-error-codes"></a>Códigos de erro OpenGL

O OpenGL inclui os seguintes códigos de erro.



| Código do erro             | Descrição                                 |
|------------------------|---------------------------------------------|
| GL \_ inválido de \_ enumeração      | Argumento **GLenum** fora do intervalo.           |
| \_valor inválido do GL \_     | Argumento numérico fora do intervalo.              |
| GL \_ operação inválida \_ | Operação inválida no estado atual.         |
| \_estouro de pilha GL \_    | A função causaria um estouro de pilha.      |
| \_estouro negativo de pilha GL \_   | A função causaria um estouro negativo de pilha.     |
| GL \_ sem \_ \_ memória    | Não há memória suficiente para executar a função. |
