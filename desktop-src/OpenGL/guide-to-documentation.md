# <a name="guide-to-documentation"></a>Guia de documentação

A documentação definida para OpenGL no Windows inclui cinco elementos.

-   O *manual de referência do OpenGL* inclui uma visão geral de como o OpenGL funciona e um conjunto de páginas de referência detalhadas. As páginas de referência abrangem todas as 115 funções OpenGL distintas, bem como as funções 43 na biblioteca do utilitário OpenGL (GLU).

-   O *Guia de programação OpenGL* explica como criar programas gráficos usando OpenGL. Ele inclui discussões dos seguintes tópicos principais:

    -   Desenho de formas geométricas
    -   Pixels, bitmaps, fontes e imagens
    -   Exibindo e transformações de matriz
    -   Mapeamento de textura
    -   Exibir listas
    -   Técnicas compostas avançadas
    -   Cor
    -   Avaliadores e NURBS
    -   Iluminação
    -   Seleção e comentários
    -   Mesclagem, suavização e neblina
    -   Técnicas avançadas

    Além disso, o *Guia de programação OpenGL* contém apêndices que abordam a biblioteca do utilitário OpenGL e a biblioteca auxiliar do guia de programação OpenGL.

-   O terceiro elemento de documentação é esta visão geral. Ele descreve a implementação do OpenGL do Windows e fornece uma visão geral de seus componentes. Ele aborda os conceitos de contextos de renderização, formatos de pixel e buffers; as funções WGL que conectam OpenGL aos sistemas de janelas do Windows; e as funções do Windows que dão suporte a formatos de pixel por janela e buffer duplo do Windows para janelas de gráficos OpenGL. As funções WGL e as funções são específicas para a implementação do OpenGL do Windows.

-   O quarto elemento de documentação é o conjunto de páginas de referência para as funções WGL, as funções do Windows que acabamos de mencionar e a estrutura de dados [**PIXELFORMATDESCRIPTOR**](/windows/win32/api/wingdi/ns-wingdi-pixelformatdescriptor) .

-   O quinto elemento de documentação é o guia de portabilidade. Ele aborda a movimentação do código OpenGL existente de outros ambientes para o Windows.

Os dois primeiros elementos são os manuais OpenGL oficiais; Eles não são incluídos no SDK do Microsoft Windows.
