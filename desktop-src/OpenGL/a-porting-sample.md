é mais fácil entender como modificar o programa OpenGL do sistema de janelas X para Windows se você puder comparar os exemplos antes e depois, e pode ver melhor como o código traduzido é usado no contexto apropriado. Esta seção apresenta um exemplo de um programa OpenGL do sistema de janelas X e mostra como o programa é exibido depois de ser transportado para Windows. Observe que o código OpenGL é o mesmo em ambos os programas.
