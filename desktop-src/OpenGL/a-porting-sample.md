# <a name="a-porting-sample"></a>Um exemplo de portabilidade

É mais fácil entender como modificar o programa OpenGL do sistema de janelas X para Windows se você puder comparar os exemplos antes e depois, e pode ver melhor como o código traduzido é usado no contexto apropriado. Esta seção apresenta um exemplo de um programa OpenGL do sistema de janelas X e, em seguida, mostra como o programa se parece depois de ser portado para o Windows. Observe que o código OpenGL é o mesmo em ambos os programas.

-   [Um programa OpenGL do sistema de janelas X](an-x-window-system-opengl-program.md)
-   [O programa portado para o Windows](the-program-ported-to-win32.md)
