# <a name="alpha-test"></a>Teste alfa

O teste alfa (executado somente no modo RGBA) descarta um fragmento, dependendo do resultado de uma comparação entre o valor alfa do fragmento e um valor de referência constante. Especifique a função de comparação e o valor de referência [**com glAlphaFunc**](glalphafunc.md).
