# <a name="logical-operations"></a>Operações lógicas

Uma operação lógica pode ser aplicada entre o fragmento e o valor armazenado no local correspondente no framebuffer; o resultado substitui o valor de framebuffer atual. Você escolhe a operação lógica desejada com [**glLogicOp.**](gllogicop.md) As operações lógicas são executadas somente em índices de cores, nunca em valores RGBA.
