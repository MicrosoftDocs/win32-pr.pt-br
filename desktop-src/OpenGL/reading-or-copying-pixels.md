# <a name="reading-or-copying-pixels"></a>Lendo ou copiando pixels

Você pode ler os pixels do framebuffer na memória, codificá-los de várias maneiras e armazenar o resultado codificado na memória com [**glReadPixels**](glreadpixels.md). Além disso, você pode copiar um retângulo de valores de pixel de uma região do framebuffer para outra com [**glCopyPixels**](glcopypixels.md). A função [**glReadBuffer**](glreadbuffer.md) controla em qual buffer de cores os pixels são lidos ou copiados.
