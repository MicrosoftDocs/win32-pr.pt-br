# <a name="glislist-function"></a>função glIsList

A função **gllsList** testa a existência da lista de exibição.

## <a name="syntax"></a>Sintaxe


```C++
GLboolean WINAPI glIsList(
   GLuint list
);
```



## <a name="parameters"></a>Parâmetros

<dl> <dt>

*list* 
</dt> <dd>

Um nome de lista de exibição potencial.

</dd> </dl>

## <a name="error-codes"></a>Códigos do Erro

O código de erro a seguir pode ser recuperado pela função [**glGetError**](glgeterror.md) .



| Nome                                                                                                  | Significado                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**GL \_ operação inválida \_**</dt> </dl> | A função foi chamada entre uma chamada para [**glBegin**](glbegin.md) e a chamada correspondente para [**glEnd**](glend.md).<br/> |



## <a name="remarks"></a>Comentários

A função **gllsList** retornará GL \_ true se *list* for o nome de uma lista de exibição e retornará GL \_ false caso contrário.
