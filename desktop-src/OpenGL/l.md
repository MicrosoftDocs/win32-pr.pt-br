# <a name="l-opengl"></a>L (OpenGL)

<dl> <dt>

<span id="opengl_lighting"></span><span id="OPENGL_LIGHTING"></span>**ilumina**
</dt> <dd>

O processo de computar a cor de um vértice com base nas luzes atuais, nas propriedades do material e nos modos de modelo de iluminação.

</dd> <dt>

<span id="opengl_line"></span><span id="OPENGL_LINE"></span>**descritos**
</dt> <dd>

Uma região reta de largura finita entre dois vértices. (Ao contrário das linhas matemáticas, as linhas OpenGL têm largura e comprimento finitos.) Cada segmento de uma faixa de linhas é uma linha.

</dd> <dt>

<span id="opengl_luminance"></span><span id="OPENGL_LUMINANCE"></span>**luminância**
</dt> <dd>

O brilho percebido de uma superfície. Geralmente refere-se a uma média ponderada de valores de cor vermelho, verde e azul que fornece o brilho percebido da combinação.

</dd> </dl>
