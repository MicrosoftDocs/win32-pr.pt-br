# <a name="initializing"></a>Inicializando

Com o GLU versão 1,1 ou posterior, o [**gluGetString**](glugetstring.md) retorna o número de versão da biblioteca Glu ou o número de versão e as chamadas de extensão Glu específicas do fornecedor.

O protótipo de **gluGetString** é:

const **GLubyte** \* **gluGetString**( *nome* do GLenum);
