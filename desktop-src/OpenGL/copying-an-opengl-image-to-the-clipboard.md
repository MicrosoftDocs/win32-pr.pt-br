# <a name="copying-an-opengl-image-to-the-clipboard"></a>Copiando uma imagem OpenGL para a área de transferência

Embora a versão atual da implementação da Microsoft do OpenGL no Windows não ofereça suporte diretamente à área de transferência, você pode copiar uma imagem do Windows OpenGL para a área de transferência.

**Para copiar uma imagem OpenGL para a área de transferência**

1.  Desenhe a imagem para um bitmap de memória ou um metarquivo avançado.
2.  Copie esse bitmap para a área de transferência.
