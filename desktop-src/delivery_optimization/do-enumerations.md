# <a name="do-enumerations"></a>FAZER enumerações

As [interfaces de otimização de entrega (do)](do-interfaces.md) usam as seguintes enumerações.

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | A enumeração **BG_ERROR_CONTEXT** define os valores constantes que especificam o contexto no qual o erro ocorreu. |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | A enumeração **BG_JOB_PRIORITY** define os valores constantes que especificam o nível de prioridade de um trabalho.  |
| [**BG_JOB_STATE**](bg-job-state-.md) | A enumeração **BG_JOB_STATE** define valores constantes para os diferentes Estados de um trabalho. |
| [**BG_JOB_TYPE**](bg-job-type.md) | A enumeração **BG_JOB_TYPE** define valores constantes que especificam o tipo de trabalho de transferência, como download. |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | A enumeração **BITS_FILE_PROPERTY_ID** especifica valores que definem valores de ID correspondentes às propriedades **BackgroundCopyFile** . |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | A enumeração **BITS_JOB_PROPERTY_ID** especifica a ID da propriedade para o trabalho do. Essa enumeração é usada na União de [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) para determinar o tipo de valor contido na União. |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | A enumeração **BITS_JOB_TRANSFER_POLICY** define os valores de ID correspondentes às propriedades do. |
| [**DeliveryOptimizationFileProperty**](deliveryoptimizationfileproperty.md) | Especifica a ID de uma propriedade opcional para o arquivo do. |
| [**DODownloadCostPolicy**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | Especifica a ID das opções de políticas de custo associadas à propriedade **DODownloadProperty_CostPolicy** . |
| [**Dobaixeproperty**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | Especifica a ID das propriedades para a operação de download. |
| [**DODownloadPropertyEx**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | Especifica a ID das propriedades estendidas para a operação de download. |
| [**Dodownloadstate**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | Especifica a ID do estado de download atual, que faz parte da estrutura de **DO_DOWNLOAD_STATUS** . |
| [**DownloadMode**](downloadmode.md) | Define os diferentes modos de download que a otimização de entrega usa.  |
| [**SwarmStatus**](swarmstatus.md) | Define o status de um arquivo dentro do cliente de otimização de entrega. |
