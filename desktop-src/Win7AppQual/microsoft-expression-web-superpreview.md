# <a name="microsoft-expression-web-superpreview"></a>Microsoft Expression Web SuperPreview

O Microsoft Expression Web expreview para Internet Explorer é uma ferramenta de depuração Visual autônoma que ajuda você a depurar seus sites para compatibilidade entre navegadores. a supervisualização da Web garante que seus sites funcionem corretamente no Windows Internet Explorer 8 e mantém a compatibilidade com versões anteriores do Windows Internet explorer.

a supervisualização da Web mostra como as suas páginas da websão renderizadas no Microsoft Internet explorer 6 e Windows internet explorer 7 ou internet explorer 8, dependendo da versão instalada no computador. Você pode exibir as páginas lado a lado ou como uma sobreposição. Você pode usar réguas, guias e ferramentas de zoom e panorâmica para identificar precisamente as diferenças no layout. Você pode até mesmo comparar sua página com o modo como os navegadores de destino renderizam a página.

A supervisualização da Web também identifica a marca, o tamanho, a posição, os estilos aplicados e a localização de um elemento na árvore Modelo de Objeto do Documento (DOM) para que você possa corrigir o erro rapidamente.

> [!Note]  
> A supervisualização da Web é um aplicativo gratuito que não tem expiração e nenhum suporte técnico da Microsoft.

 

Com a ferramenta de supervisualização da Web, você pode:

-   Depurar páginas em seu computador de desenvolvimento. a supervisualização da Web funciona diretamente com os navegadores do Microsoft Windows instalados e inclui o mecanismo de renderização do Internet Explorer 6. Você pode usar esse mecanismo lado a lado junto com outras versões do Internet Explorer ou do Mozilla Firefox.
-   Depurar mais rapidamente usando informações de árvore de elementos e DOM. A supervisualização da Web também combina a renderização de imagem com dados DOM, para que você possa realçar elementos DOM específicos dentro da imagem e mostrar as diferenças entre navegadores. A ferramenta fornece informações sobre o elemento, como posição, dimensões e ID, para ajudá-lo a encontrar o elemento que está causando o problema de compatibilidade.
-   dê suporte a navegadores Windows e macintosh, incluindo o Internet Explorer 6 e versões posteriores, o Firefox 3,6 e o Macintosh Safari.

Para obter mais informações sobre essa ferramenta e outras ferramentas de desenvolvedores, consulte a [visão geral da Web do Expression Studio 4](https://www.microsoft.com/expression/products/StudioWebPro_Overview.aspx), [demonstração da](https://msdn.microsoft.com/expression/ff723803.aspx)Web do Expression e [fóruns de expressões](https://social.msdn.microsoft.com/forums/announce/threads).
