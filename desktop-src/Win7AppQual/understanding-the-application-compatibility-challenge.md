# <a name="understanding-the-application-compatibility-challenge"></a>Noções básicas sobre o desafio da compatibilidade de aplicativos

O Windows Internet Explorer 8 é o navegador padrão no sistema operacional Windows 7. O Internet Explorer 8 melhora muitos recursos, incluindo uma arquitetura de plataforma aprimorada que foi projetada para tornar a Web mais segura e mais rápida para os usuários e facilitar a configuração e o gerenciamento para profissionais de ti, bem como para criar conteúdo inovador para os desenvolvedores da Web.

Alguns desses recursos usam novos mecanismos de geração de scripts e renderização, que são os principais elementos que o Windows Internet Explorer usa para exibir conteúdo HTML e CSS e executar scripts. As alterações de arquitetura e os aprimoramentos de plataforma introduzem problemas de compatibilidade de aplicativos que podem ocorrer ao migrar aplicativos para o Internet Explorer 8 de um navegador anterior.

O Internet Explorer 8 inclui alguma compatibilidade com versões anteriores, mas há algumas alterações (incluindo suporte para suporte a padrões da Web e recursos de segurança) que podem exigir a atualização de aplicativos Web criados para o Microsoft Internet Explorer 6 ou para versões anteriores.

Para obter mais informações sobre como testar e solucionar problemas de compatibilidade do Internet Explorer 8, consulte [lidando com a compatibilidade de aplicativos ao migrar para o Internet Explorer 8 – informações para profissionais de tecnologia da informação](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10)).
