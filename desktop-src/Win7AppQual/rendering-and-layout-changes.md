# <a name="rendering-and-layout-changes"></a>Alterações de layout e renderização

Windows O Internet Explorer 8 interpreta mais estritamente a marcação HTML do que as versões anteriores, e essa alteração pode fazer com que as páginas da Web sejam exibidas incorretamente. por exemplo, quando um site exibe uma lista ou tabela corretamente no Windows Internet explorer 7, ele pode parecer quebrado no internet explorer 8.

Você pode resolver esses tipos de problemas reduzindo a página da Web para localizar o HTML malformado. se você corrigir a maioria dos problemas de exibição, as alterações não criam problemas em versões anteriores do Windows Internet Explorer porque as versões anteriores simplesmente ignoram o HTML estranho. Para obter mais informações sobre essas alterações, consulte a postagem no blog [compatibilidade do site e o IE8](/archive/blogs/ie/site-compatibility-and-ie8) .
