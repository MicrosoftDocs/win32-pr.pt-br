<span data-ttu-id="8e88a-104">A *exibição de compatibilidade* é um recurso do Windows Internet Explorer 8 que permite que o navegador processe uma página da Web quase idênticamente à forma como o Windows Internet Explorer 7 a processaria.</span><span class="sxs-lookup"><span data-stu-id="8e88a-104">*Compatibility View* is a feature of Windows Internet Explorer 8 that enables the browser to render a webpage nearly identically to the way that Windows Internet Explorer 7 would render it.</span></span>
