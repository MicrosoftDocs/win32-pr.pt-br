# <a name="internet-explorer-8-developer-tools"></a>Ferramentas de desenvolvimento do Internet Explorer 8

O Windows Internet Explorer 8 inclui o recurso Ferramentas para Desenvolvedores. Esse recurso permite que você depure o Microsoft JScript rapidamente, investigue um comportamento específico do Windows Internet Explorer ou itere rapidamente para criar um protótipo ou testar uma nova solução de design.

As seções a seguir descrevem as características importantes do Ferramentas para Desenvolvedores no Internet Explorer 8.

## <a name="integrated-and-simple-to-use"></a>Integrado e simples de usar

Os Ferramentas para Desenvolvedores estão incluídos em todas as instalações do Internet Explorer 8, para que você possa depurar problemas em qualquer computador que esteja executando o Internet Explorer 8. Além disso, as ferramentas são carregadas somente quando você precisa delas para limitar como elas afetam o desempenho do navegador. Usando o Ferramentas para Desenvolvedores, você pode depurar rapidamente apenas o processo atual do Internet Explorer, em vez de Depurar para todo o Internet Explorer.

## <a name="provide-a-visual-interface-to-the-platform"></a>Fornecer uma interface visual para a plataforma

Em vez de fazer a engenharia reversa de como seu site funciona ou modificar seu site para gerar informações de depuração, o Ferramentas para Desenvolvedores permite que você examine o Internet Explorer para exibir sua representação do seu site. Essa exibição reduz o tempo gasto na depuração de sites dinâmicos, em que a inspeção de origem não é útil ou investiga um comportamento específico do Internet Explorer, no qual uma ferramenta de criação genérica não pode ajudar.

## <a name="enable-fast-experimentation"></a>Habilitar experimentação rápida

Ao criar um protótipo de um novo design ou correções de teste nas versões anteriores do Internet Explorer, é provável que você edite sua origem, salve-a, atualize sua página no navegador e repita. O Ferramentas para Desenvolvedores simplificar esse cenário permitindo que você edite seu site no navegador e veja as alterações imediatamente.

## <a name="optimize-application-performance"></a>Otimizar o desempenho do aplicativo

Para identificar e corrigir problemas de desempenho, é provável que você itere concentrando-se em um cenário por vez. Usando o criador de perfil de script no Ferramentas para Desenvolvedores, você pode coletar estatísticas, incluindo o tempo de execução e o número de vezes que uma função JavaScript é chamada. Você pode usar essas informações ao testar seu aplicativo e usar o relatório de perfil para identificar e corrigir gargalos de desempenho rapidamente.

Para acessar o Ferramentas para Desenvolvedores, pressione F12 enquanto estiver exibindo uma página da Web ou clique em **ferramentas para desenvolvedores** no menu **ferramentas** .
