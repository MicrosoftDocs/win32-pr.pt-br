# <a name="internet-explorer-8-developer-tools"></a>Ferramentas de desenvolvimento do Internet Explorer 8

Windows Internet Explorer 8 inclui o recurso Ferramentas para Desenvolvedores dados. Esse recurso permite que você depure o Microsoft JScript rapidamente, investigue um comportamento específico do Windows Internet Explorer ou itere rapidamente para criar protótipos ou testar uma nova solução de design.

As seções a seguir descrevem as características importantes do Ferramentas para Desenvolvedores no Internet Explorer 8.

## <a name="integrated-and-simple-to-use"></a>Integrado e simples de usar

Os Ferramentas para Desenvolvedores estão incluídos em cada instalação do Internet Explorer 8, para que você possa depurar problemas em qualquer computador que executa Internet Explorer 8. Além disso, as ferramentas são carregadas somente quando você precisa delas para limitar como elas impactam o desempenho do navegador. Usando o Ferramentas para Desenvolvedores, você pode depurar rapidamente apenas o processo Internet Explorer atual, em vez de depurar para todas as Internet Explorer.

## <a name="provide-a-visual-interface-to-the-platform"></a>Fornecer uma interface visual para a plataforma

Em vez de engenharia reversa como seu site funciona ou modificando seu site para exibir informações de depuração, o Ferramentas para Desenvolvedores permite que você veja Internet Explorer exibir sua representação do site. Essa exibição reduz o tempo gasto com a depuração de sites dinâmicos, em que a inspeção de origem não é útil ou investiga um comportamento específico Internet Explorer, em que uma ferramenta de autor genérica não pode ajudar.

## <a name="enable-fast-experimentation"></a>Habilitar Experimentação Rápida

Ao criar protótipos de um novo design ou correções de teste nas versões anteriores do Internet Explorer, você provavelmente editará sua origem, a salvará, atualizará sua página no navegador e repetirá. O Ferramentas para Desenvolvedores simplificar esse cenário, permitindo que você edite seu site no navegador e veja as alterações imediatamente.

## <a name="optimize-application-performance"></a>Otimizar o desempenho do aplicativo

Para identificar e corrigir problemas de desempenho, você provavelmente itera concentrando-se em um cenário de cada vez. Usando o profiler de script no Ferramentas para Desenvolvedores, você pode coletar estatísticas, incluindo o tempo de execução e o número de vezes que uma função JavaScript é chamada. Você pode usar essas informações ao testar seu aplicativo e usar o relatório de perfil para identificar e corrigir rapidamente gargalos de desempenho.

Para acessar o Ferramentas para Desenvolvedores, pressione F12 enquanto estiver exibindo uma página da Web ou clique em Ferramentas para Desenvolvedores **no** **menu** Ferramentas.
