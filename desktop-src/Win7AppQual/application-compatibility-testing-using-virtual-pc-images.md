# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Teste de compatibilidade de aplicativos usando imagens Virtual PC

Para começar a testar a compatibilidade de aplicativos Web, você precisa estabelecer um ambiente de teste. O processo para criar e configurar computadores para essa tarefa é demorado e geralmente dispendioso.

Para facilitar o teste de compatibilidade, você pode usar as imagens de Virtual PC (VPC) de compatibilidade de aplicativos predefinidas. As imagens do VPC são imagens de unidade de disco rígido pré-configuradas que você pode carregar em computadores existentes que executam o [software do Virtual PC](https://support.microsoft.com/help/14223/windows-xp-end-of-support). Essas imagens permitem que você teste exaustivamente várias versões do Microsoft Windows e do Windows Internet Explorer com aplicativos existentes em um curto período de tempo.

As imagens do VPC estão disponíveis para todas as versões do Windows atualmente com suporte e contêm muitos aplicativos e recursos de teste de compatibilidade. Para localizar as imagens de teste, consulte [imagem do VPC de compatibilidade de aplicativos do Internet Explorer](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en).
