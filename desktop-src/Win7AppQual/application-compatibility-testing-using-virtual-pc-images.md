# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Teste de compatibilidade de aplicativos usando imagens Virtual PC

Para começar a testar a compatibilidade do aplicativo Web, você precisa estabelecer um ambiente de teste. O processo para criar e configurar computadores para essa tarefa é demorado e, muitas vezes, caro.

Para facilitar o teste de compatibilidade, você pode usar as imagens de VPC (Virtual PC) de compatibilidade de aplicativos gratuitas e pré-construídas. As imagens de VPC são imagens de unidade de disco rígido pré-configuradas que você pode carregar em computadores existentes que executam o [software de COMPUTADOR virtual](https://support.microsoft.com/help/14223/windows-xp-end-of-support). Essas imagens permitem que você teste completamente várias versões do Microsoft Windows e Windows Internet Explorer com aplicativos existentes em um curto período de tempo.

As imagens VPC estão disponíveis para todas as versões com suporte no momento do Windows e contêm muitos recursos e aplicativos de teste de compatibilidade. Para encontrar as imagens de teste, consulte [Internet Explorer VPC de Compatibilidade do Aplicativo](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en).
