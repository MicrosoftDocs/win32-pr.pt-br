# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Ferramentas para depurar aplicativos Web e Add-Ons

Em vez de [corrigir problemas de compatibilidade em aplicativos Web e](remediating-web-applications-and-add-ons.md)Complementos, você pode fazer com que o aplicativo Web ou site funcione nativamente no modo de padrões do Windows Internet Explorer 8 escrevendo seu código para os padrões da Web mais recentes.

Para avaliar totalmente quaisquer problemas em seu aplicativo ou site, você deve pesquisar a causa raiz do problema. Por exemplo, a equipe do Internet Explorer determinou as causas raiz usando o Ferramentas para Desenvolvedores e reduzindo logicamente possíveis problemas. As seções a seguir descrevem várias ferramentas para ajudar a simplificar o processo de depuração.



| Seção                                                                                                                    | Descrição                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Uma ferramenta de depuração Visual autônoma que torna mais rápida e fácil migrar seus sites do Microsoft Internet Explorer 6 para o Windows Internet Explorer 7 ou Internet Explorer 8. |
| [IECTT (Internet Explorer Compatibility Test Tool)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Um componente do [Kit de ferramentas de compatibilidade de aplicativos da Microsoft](/windows-hardware/get-started/adk-install) que pode ajudá-lo a diagnosticar problemas em seus aplicativos Web.       |
| [Ferramentas de desenvolvimento do Internet Explorer 8](internet-explorer-8-developer-tools.md)                                             | As ferramentas de desenvolvedor incluídas no Internet Explorer 8 para depuração.                                                                                                         |
| [Assistente de compatibilidade do Internet Explorer 8 (aggiorno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Uma ferramenta de terceiros que atualiza automaticamente os sites para renderização correta no Internet Explorer 8 sem interromper a compatibilidade em navegadores mais antigos.                                |
| [Ferramenta de depurador da Web do Fiddler](fiddler-web-debugger-tool.md)                                                                 | Uma ferramenta de terceiros para capturar o tráfego de rede entre a Internet e os computadores de teste.                                                                                           |
| [Teste de compatibilidade de aplicativos usando imagens Virtual PC](application-compatibility-testing-using-virtual-pc-images.md) | Imagens de Virtual PC que simplificam o teste de compatibilidade.                                                                                                                              |
