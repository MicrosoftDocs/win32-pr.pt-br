# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Ferramentas para depurar aplicativos Web e Add-Ons

Em vez de corrigir problemas de compatibilidade em aplicativos Web e [complementos,](remediating-web-applications-and-add-ons.md)você pode fazer com que o aplicativo Web ou o site funcione na na verdade no modo de padrões do Windows Internet Explorer 8 escrevendo seu código para os padrões web mais recentes.

Para avaliar totalmente os problemas em seu aplicativo ou site, você deve pesquisar a causa raiz do problema. Por exemplo, a equipe Internet Explorer de dados determinou as causas raiz usando o Ferramentas para Desenvolvedores e reduzindo logicamente possíveis problemas. As seções a seguir descrevem várias ferramentas para ajudar a simplificar o processo de depuração.



| Seção                                                                                                                    | Descrição                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Uma ferramenta de depuração visual autônomo que torna mais rápido e mais fácil migrar seus sites do Microsoft Internet Explorer 6 para Windows Internet Explorer 7 ou Internet Explorer 8. |
| [IECTT (Internet Explorer Compatibility Test Tool)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Um componente do Microsoft [Application Compatibility Toolkit](/windows-hardware/get-started/adk-install) que pode ajudá-lo a diagnosticar problemas em seus aplicativos Web.       |
| [Ferramentas de desenvolvimento do Internet Explorer 8](internet-explorer-8-developer-tools.md)                                             | As ferramentas para desenvolvedores incluídas no Internet Explorer 8 para depuração.                                                                                                         |
| [Internet Explorer Assistente de Compatibilidade do Internet Explorer 8 (Ano)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Uma ferramenta de terceiros que atualiza automaticamente sites para renderizar corretamente no Internet Explorer 8 sem compatibilidade com a quebra de compatibilidade em navegadores mais antigos.                                |
| [Ferramenta de depurador da Web do Fiddler](fiddler-web-debugger-tool.md)                                                                 | Uma ferramenta de terceiros para capturar o tráfego de rede entre a Internet e os computadores de teste.                                                                                           |
| [Teste de compatibilidade de aplicativos usando imagens Virtual PC](application-compatibility-testing-using-virtual-pc-images.md) | Imagens de PC virtual que simplificam o teste de compatibilidade.                                                                                                                              |
