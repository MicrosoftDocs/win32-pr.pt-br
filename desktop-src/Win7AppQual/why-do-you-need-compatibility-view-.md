<span data-ttu-id="b0cb1-104">O recurso de modo de exibição de compatibilidade segue um conjunto de regras para exibir o conteúdo corretamente, sem a necessidade de qualquer ação de usuário (ou administrador de ti) sempre que possível.</span><span class="sxs-lookup"><span data-stu-id="b0cb1-104">The Compatibility View feature follows a set of rules to display content correctly, without any user (or IT administrator) action needed whenever possible.</span></span> <span data-ttu-id="b0cb1-105">A exibição de compatibilidade permite que os desenvolvedores instruam o navegador que o mecanismo de renderização a usar e permite que os usuários substituam os modos de opção e comutador do navegador.</span><span class="sxs-lookup"><span data-stu-id="b0cb1-105">Compatibility View enables developers to instruct the browser which rendering engine to use and enables users to override the browser’s choice and switch modes.</span></span> <span data-ttu-id="b0cb1-106">Se o desenvolvedor e o profissional de ti não especificarem o modo de renderização a ser usado, o usuário verá o ícone "página quebrada" na extremidade direita da barra de endereços, como mostra a ilustração a seguir.</span><span class="sxs-lookup"><span data-stu-id="b0cb1-106">If the developer and IT professional do not specify which rendering mode to use, user sees the "broken page" icon at the right end of the address bar, as the following illustration shows.</span></span>
