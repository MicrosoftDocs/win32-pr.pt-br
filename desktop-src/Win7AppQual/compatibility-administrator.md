# <a name="compatibility-administrator"></a>Administrador de compatibilidade

## <a name="affected-platforms"></a>Plataformas afetadas

 **Clientes:** Windows 2000, Windows XP, Windows Vista, Windows 7  
**Servidores:** Windows Server 2003, Windows Server 2008, Windows Server 2008 R2  


## <a name="description"></a>Descrição

A ferramenta Compatibility Administrator, fornecida pelo ACT (Kit de ferramentas de compatibilidade de aplicativos), permite que você resolva muitos dos problemas potenciais de compatibilidade de aplicativos, antes de implantar uma nova versão do Windows em sua organização, por:

-   Fornecendo correções de compatibilidade individuais, modos de compatibilidade e mensagens AppHelp que você pode usar para resolver problemas de compatibilidade específicos
-   Permitindo que você crie correções de compatibilidade personalizadas, modos de compatibilidade, mensagens AppHelp e bancos de dados de compatibilidade
-   Fornecendo uma ferramenta de consulta que permite pesquisar correções instaladas em seus computadores locais

## <a name="usage"></a>Uso

O fluxograma a seguir ilustra as etapas necessárias para usar o administrador de compatibilidade para criar suas correções de compatibilidade, modos de compatibilidade e suas mensagens AppHelp.



|                                            |          |                                                                                            |          |                                                     |          |                                                                             |
|--------------------------------------------|----------|--------------------------------------------------------------------------------------------|----------|-----------------------------------------------------|----------|-----------------------------------------------------------------------------|
| Criar um novo banco de dados de compatibilidade (. sdb) | **>** | Selecione o aplicativo e, em seguida, selecione as correções de compatibilidade a serem aplicadas ao aplicativo | **>** | Testar o aplicativo com a nova correção de compatibilidade | **>** | Salve o banco de dados de compatibilidade e, em seguida, implante a correção em sua empresa |
