# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Assistente de compatibilidade do Internet Explorer 8 (aggiorno)

O assistente de compatibilidade do Windows Internet Explorer 8 atualiza automaticamente os sites para renderização correta no Internet Explorer 8, sem interromper a compatibilidade em navegadores mais antigos. O assistente automatiza a tarefa de direcionamento da versão do navegador, mantendo a aparência e a funcionalidade das páginas. Para obter mais informações sobre esse assistente, consulte o [Assistente de compatibilidade do Internet Explorer 8 início rápido](https://www.mobilize.net/).
