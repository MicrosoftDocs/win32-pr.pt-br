# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Internet Explorer Assistente de Compatibilidade do Internet Explorer 8 (Ano)

O Assistente de Compatibilidade no Windows Internet Explorer 8 atualiza automaticamente os sites para renderizar corretamente no Internet Explorer 8 sem quebrar sua compatibilidade em navegadores mais antigos. O assistente automatiza a tarefa de direcionamento de versão do navegador, mantendo a aparência e a funcionalidade das páginas. Para obter mais informações sobre esse assistente, consulte [Internet Explorer assistente de compatibilidade 8 Início Rápido](https://www.mobilize.net/).
