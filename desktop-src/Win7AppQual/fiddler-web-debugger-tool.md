# <a name="fiddler-web-debugger-tool"></a>Ferramenta de depurador da Web do Fiddler

A ferramenta Fiddler ajuda a depurar aplicativos Web capturando o tráfego de rede entre a Internet e os computadores de teste. A ferramenta permite que você inspecione os dados de entrada e saída para monitorar e modificar solicitações e respostas antes que o navegador as receba. O Fiddler também inclui um subsistema de script baseado em evento poderoso, que você pode estender usando qualquer linguagem .NET Framework aplicativo.

O Fiddler e as opções de reprodução HTTP podem ajudá-lo a solucionar problemas do lado do cliente com aplicativos Web fazendo uma cópia offline do site de teste. Com essas ferramentas, você pode criar imagens offline da experiência de navegação e, em seguida, empacotar e analisar os resultados para obter informações de depuração mais detalhadas.

Para baixar o complemento Fiddler, acesse a [página Internet Explorer complementos.](https://www.microsoft.com/iegallery) Para obter mais informações sobre como solucionar problemas usando o Fiddler e ferramentas relacionadas, consulte a postagem no blog Como usar [o Fiddler](https://blogs.msdn.com/b/ricardo/archive/2010/06/22/how-to-user-fiddler-and-http-replay-to-have-an-offline-copy-of-your-site.aspx) e HTTP para ter uma cópia offline do seu site e a postagem no blog Solucionando problemas de autenticação com [o Fiddler.](https://blogs.msdn.com/b/ieinternals/archive/2009/11/22/clipboard-paste-bug-in-windows-7-password-prompt.aspx)
