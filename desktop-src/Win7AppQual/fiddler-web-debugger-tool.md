# <a name="fiddler-web-debugger-tool"></a>Ferramenta de depurador da Web do Fiddler

A ferramenta Fiddler ajuda você a depurar aplicativos Web capturando o tráfego de rede entre a Internet e os computadores de teste. A ferramenta permite que você inspecione os dados de entrada e saída para monitorar e modificar solicitações e respostas antes que o navegador as receba. O Fiddler também inclui um poderoso subsistema de scripts baseado em eventos, que pode ser estendido usando qualquer linguagem de .NET Framework.

O Fiddler e as opções de reprodução de HTTP podem ajudá-lo a solucionar problemas do lado do cliente com aplicativos Web, fazendo uma cópia offline do site de teste. Com essas ferramentas, você pode criar imagens offline da experiência de navegação e, em seguida, empacotar e analisar os resultados para obter informações de depuração mais detalhadas.

Para baixar o complemento Fiddler, vá para a página [Complementos do Internet Explorer](https://www.microsoft.com/iegallery) . Para obter mais informações sobre como solucionar problemas usando o Fiddler e as ferramentas relacionadas, consulte a postagem de blog [como usar o Fiddler e o http replay para ter uma cópia offline do seu site](https://blogs.msdn.com/b/ricardo/archive/2010/06/22/how-to-user-fiddler-and-http-replay-to-have-an-offline-copy-of-your-site.aspx) e a postagem de [solução de problemas de autenticação com o Fiddler no](https://blogs.msdn.com/b/ieinternals/archive/2009/11/22/clipboard-paste-bug-in-windows-7-password-prompt.aspx) blog.
