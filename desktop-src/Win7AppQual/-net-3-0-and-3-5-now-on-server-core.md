# <a name="net-30-and-35-now-on-server-core"></a>.NET 3,0 e 3,5 agora no Server Core

## <a name="platform"></a>Plataforma

**Servidores** -Windows Server 2008 R2  



## <a name="feature-impact"></a>Impacto do recurso

 **Severidade** -baixa  
**Frequência** -baixa  





## <a name="description"></a>Descrição

A opção de instalação Server Core para o Windows Server 2008 R2 agora inclui um subconjunto do .NET Framework 3,0 e 3,5, especificamente WCF, WF, LINQ.

Não havia suporte ao .NET na opção de instalação do Server Core para o Windows Server 2008.

## <a name="manifestation-of-impact"></a>Manifestação do impacto

Isso permite que alguns aplicativos baseados no WCF e no WF, bem como aplicativos que usam o LINQ, sejam executados no Server Core do Windows Server 2008 R2.

## <a name="testing"></a>Teste

Testar aplicativos que incorporam o WCF, o WF e o código do LINQ no Server Core.
