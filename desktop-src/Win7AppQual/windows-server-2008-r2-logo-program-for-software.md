# <a name="windows-server-2008-r2-logo-program-for-software"></a>Windows Programa de logotipo do Server 2008 R2 para software

## <a name="affected-platforms"></a>Plataformas afetadas

 **servidores-** Windows Server 2008 R2  

### <a name="description"></a>Descrição

o programa de logotipo do Windows Server 2008 R2 foi desativado.

Windows o servidor 2008 R2 2012 envios de teste não são mais aceitos para competências de Microsoft Partner Network (MPN) ou o logotipo certificado para Windows Server.

Windows Server 2012 o R2 é atualmente a plataforma preferencial para o logotipo certificado para Windows Server. consulte o [programa de certificação do logotipo do aplicativo de nuvem Enterprise](https://azure.microsoft.com/overview/commercial-marketplace/) para obter mais informações.
