# <a name="windows-server-2008-r2-logo-program-for-software"></a>Programa de logotipo do Windows Server 2008 R2 para software

## <a name="affected-platforms"></a>Plataformas afetadas

 **Servidores-** Windows Server 2008 R2  

### <a name="description"></a>Descrição

O programa de logotipo do Windows Server 2008 R2 foi desativado.

Os envios de teste do Windows Server 2008 R2 2012 não são mais aceitos para competências de Microsoft Partner Network (MPN) ou para o logotipo certificado para Windows Server.

O Windows Server 2012 R2 é atualmente a plataforma preferencial para o logotipo certificado para Windows Server. Consulte o [programa de certificação do logotipo do aplicativo Enterprise Cloud](https://azure.microsoft.com/overview/commercial-marketplace/) para obter mais informações.
