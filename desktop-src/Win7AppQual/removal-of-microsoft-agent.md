# <a name="removal-of-microsoft-agent"></a>Remoção do Microsoft Agent

## <a name="platform"></a>Plataforma

**Clientes** – Windows 7  
**Servidores** – Windows Server 2008 R2  









## <a name="feature-impact"></a>Impacto do recurso

 **Gravidade –** Média  
**Frequência** – Baixa  


## <a name="description"></a>Descrição

A Microsoft decidiu descontinuar o desenvolvimento de tecnologias do Microsoft Agent. A partir Windows 7, o Microsoft Agent não será incluído nem terá suporte em versões futuras do sistema operacional Microsoft Windows. Incentivamos os desenvolvedores e os redistributores de aplicativos do Microsoft Agent a avaliar suas atividades à luz dessa decisão.

## <a name="manifestation"></a>Manifestação

-   A Microsoft não oferecerá novas licenças ou renovações
-   A Microsoft não atualizará os componentes do Microsoft Agent, incluindo:
    -   O runtime "msagent.exe" e as bibliotecas
    -   Os caracteres Queliano, Anna,Dy e Ltdo
    -   O Editor de Caracteres do Agente
    -   Todos os softwares, ferramentas e documentação relacionados do Microsoft Agent
-   Windows 7 e versões posteriores não darão suporte ao desenvolvimento de caracteres ou aplicativos do Microsoft Agent
