# <a name="removal-of-microsoft-agent"></a>Remoção do Microsoft Agent

## <a name="platform"></a>Plataforma

**Clientes** -Windows 7  
**Servidores** -Windows Server 2008 R2  









## <a name="feature-impact"></a>Impacto do recurso

 **Severidade** -média  
**Frequência** -baixa  


## <a name="description"></a>Descrição

A Microsoft decidiu interromper o desenvolvimento de tecnologias do Microsoft Agent. A partir do Windows 7, o Microsoft Agent não será incluído ou terá suporte em versões futuras do sistema operacional Microsoft Windows. Incentivamos os desenvolvedores e redistribuidores de aplicativos do Microsoft Agent a avaliar suas atividades com essa decisão.

## <a name="manifestation"></a>Manifestação

-   A Microsoft não oferecerá novas licenças ou renovações
-   A Microsoft não atualizará os componentes do Microsoft Agent, incluindo:
    -   O tempo de execução "msagent.exe" e bibliotecas
    -   Os caracteres Merlin, gênio, Peedyy e Robbo
    -   O editor de caracteres do agente
    -   Todos os softwares, ferramentas e documentação relacionados ao Microsoft Agent
-   O Windows 7 e versões posteriores não terão suporte para o desenvolvimento de caracteres ou aplicativos do Microsoft Agent

## <a name="mitigation"></a>Atenuação

Os desenvolvedores ainda podem usar os seguintes recursos para versões anteriores do Windows:

-   [Componentes principais do Microsoft Agent (para uso somente no Microsoft Windows 95, Windows 98 e Windows NT 4,0)](https://www.microsoft.com/downloads/details.aspx?familyid=E11BF712-7862-45BA-826D-44AE3A11836F)
-   [Arquivo de caracteres Merlin](https://www.microsoft.com/downloads/details.aspx?FamilyID=fee1dadd-2f23-41d0-8a81-2affd74c0aa5)
-   [Arquivo de caracteres de Peedy](https://www.microsoft.com/downloads/details.aspx?FamilyID=bd3c4655-79e4-4791-ab9d-abc7bbd133ef)
-   [Arquivo de caractere gênio](https://www.microsoft.com/downloads/details.aspx?FamilyID=da86ba4e-bc2d-4c1d-b5a0-3183fe206414)
-   [Arquivo de caractere Robby](https://www.microsoft.com/downloads/details.aspx?FamilyID=fa36d1d5-d828-494a-ad0a-7b571db5bd2e)
