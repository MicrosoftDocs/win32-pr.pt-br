# <a name="versioning-windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Controle de versão (Windows 7 e Windows Server 2008 R2 Application Quality Cookbook)

Um dos problemas mais comuns de compatibilidade de aplicativos Windows Internet Explorer cadeias de caracteres de agente do usuário (UA) e vetores de versão. Windows Internet Explorer 8 apresenta uma nova Cadeia de Caracteres de UA para ajudar os aplicativos a detectar qual versão os usuários estão executando. Além de simplesmente aumentar o valor MSIE associado ao Internet Explorer na Cadeia de Caracteres da UA, o Internet Explorer 8 adiciona um valor Trident/4.0 para ajudá-lo a detectar se o Internet Explorer 8 está em execução no modo Modo de Exibição de Compatibilidade. Essas alterações, além de verificações de versão em código, podem potencialmente introduzir problemas de compatibilidade entre navegadores.
