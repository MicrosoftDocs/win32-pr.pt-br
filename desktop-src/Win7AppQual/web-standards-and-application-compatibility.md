# <a name="web-standards-and-application-compatibility"></a>Compatibilidade de aplicativos e padrões da Web

Os padrões da Web fornecem uma linha de base comum de tecnologias que permitem que os usuários selecionem um navegador da Web sem se preocuparem com o uso dos sites que eles querem e que permitem que designers e desenvolvedores da Web criem um único site que todos possam usar.

Os padrões do navegador da Web estão em constante evolução e os navegadores da Web estão continuamente sendo atualizados para adotar esses padrões. Consequentemente, os profissionais de TI e os desenvolvedores devem garantir que os aplicativos Web sejam compatíveis com os padrões e navegadores da Web atuais e emergentes. Essa manutenção garante que os aplicativos Web e sites sejam compatíveis com novas versões de navegadores e ajuda a evitar muitos problemas de compatibilidade que podem ocorrer.
