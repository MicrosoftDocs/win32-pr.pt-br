# <a name="web-standards-and-application-compatibility"></a>Compatibilidade de aplicativos e padrões da Web

Os padrões da Web fornecem uma linha de base comum de tecnologias que permitem aos usuários selecionar um navegador da Web sem se preocupar com o uso dos sites que desejam e que permitem que designers da Web e desenvolvedores criem um único site que pode ser usado por todos.

Os padrões para o navegador da Web estão constantemente evoluindo e os navegadores da Web estão sendo atualizados continuamente para adotar esses padrões. Consequentemente, os profissionais de ti e os desenvolvedores devem garantir que os aplicativos Web sejam compatíveis com os navegadores e padrões da Web atuais e emergentes. Essa manutenção garante que os aplicativos e sites da Web sejam compatíveis com novas versões de navegadores e ajuda a evitar muitos problemas de compatibilidade que podem ocorrer.
