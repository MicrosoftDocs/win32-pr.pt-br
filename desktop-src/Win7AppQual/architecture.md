# <a name="architecture-windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>arquitetura (Windows 7 e Windows Server 2008 R2 Application Quality Cookbook)

O LCIE ( *Internet Explorer com acoplamento flexível* ) melhora a confiabilidade do navegador, separando seus componentes e soltando sua interdependência.

em particular, o LCIE tenta isolar o Windows quadro do Internet Explorer e suas guias em processos separados. no Windows Internet Explorer 8, esse isolamento melhora o desempenho e a escalabilidade. essa alteração de arquitetura pode afetar a compatibilidade de extensões e complementos, incluindo controles ActiveX, objetos auxiliares de navegador (BHOs) e componentes da barra de ferramentas da interface do usuário que são criados usando técnicas de programação mais antigas.
