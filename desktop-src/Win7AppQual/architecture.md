# <a name="architecture-windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Arquitetura (Windows 7 e Windows Server 2008 R2 Application Quality Cookbook)

O LCIE ( *Internet Explorer com acoplamento flexível* ) melhora a confiabilidade do navegador, separando seus componentes e soltando sua interdependência.

Em particular, o LCIE tenta isolar o quadro do Windows Internet Explorer e suas guias em processos separados. No Windows Internet Explorer 8, esse isolamento melhora o desempenho e a escalabilidade. Essa alteração de arquitetura pode afetar a compatibilidade de extensões e Complementos, incluindo controles ActiveX, BHOs (navegador Helper Objects) e componentes da barra de ferramentas da interface do usuário criados usando técnicas de programação mais antigas.
