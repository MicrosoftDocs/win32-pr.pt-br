# <a name="tbs-structures"></a>Estruturas TBS

O TBS dá suporte às seguintes estruturas.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                           | Descrição                                                                                                                                          |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_parâmetros de contexto TBS \_**](/windows/win32/api/tbs/ns-tbs-tbs_context_params)<br/>   | Especifica a versão da implementação do contexto TBS.<br/>                                                                                  |
| [**\_PARAMS2 de contexto TBS \_**](/windows/win32/api/tbs/ns-tbs-tbs_context_params2)<br/> | Especifica a versão da implementação do contexto TBS. Você deve usar essa estrutura se seu aplicativo funcionar com ambas as versões do TPM.<br/> |
| [**\_informações do dispositivo TPM \_**](/windows/desktop/api/Tbs/ns-tbs-tpm_device_info)<br/>         | Fornece informações sobre a versão do TPM.<br/>                                                                                        |
