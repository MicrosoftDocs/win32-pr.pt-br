# <a name="tbs-data-types"></a>Tipos de dados TBS

O TBS define os seguintes tipos de dados.



| Tipo de dados                                                                                                | Descrição                                                                                                                                                            |
|----------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TBS_HCONTEXT"></span><span id="tbs_hcontext"></span>**\_HCONTEXT TBS**                          | Identificador de objeto de contexto.<br/>                                                                                                                                      |
| <span id="TBS_COMMAND_PRIORITY"></span><span id="tbs_command_priority"></span>**\_prioridade de comando TBS \_** | Prioridade associada aos comandos.<br/>                                                                                                                          |
| <span id="TBS_COMMAND_LOCALITY"></span><span id="tbs_command_locality"></span>**\_localidade de comando TBS \_** | A localidade da qual o comando é enviado.<br/>                                                                                                            |
| <span id="TBS_OWNERAUTH_TYPE"></span><span id="tbs_ownerauth_type"></span>**\_tipo de OWNERAUTH TBS \_**       | O tipo de autenticação de proprietário.<br/> **Windows server 2008 R2, Windows 7, Windows server 2008 e Windows Vista:** Este tipo de dados não está disponível.<br/> |
