# <a name="boolean-certificate-enrollment-api"></a>BOOLEAN (API de Registro de Certificado)

Um valor booliana pode ser **TRUE** ou **FALSE.** **FALSE** é codificado como um tríplice TLV no qual **o campo** Valor é zero (0). **TRUE** é codificado como um valor não zero. A **marca** para o **tipo de dados BOOLEAN** é 0x01.
