# <a name="boolean-certificate-enrollment-api"></a>BOOLIANo (API de registro de certificado)

Um valor booliano pode ser **true** ou **false**. **False** é codificado como um TLV terceto no qual o campo de **valor** é zero (0). **True** é codificado como um valor diferente de zero. A **marca** para o tipo de dados **booliano** é 0x01.
