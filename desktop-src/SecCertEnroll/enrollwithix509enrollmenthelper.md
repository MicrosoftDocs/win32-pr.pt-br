# <a name="enrollwithix509enrollmenthelper"></a>enrollWithIX509EnrollmentHelper

O exemplo enrollWithIX509EnrollmentHelper demonstra como usar o protocolo HTTP do Windows 7 e C# para registrar um certificado em uma [*autoridade de certificação*](/windows/desktop/SecGloss/c-gly) (CA) corporativa. O exemplo é instalado por padrão na pasta *% ProgramFiles%* \\ Microsoft SDKs \\ Windows \\ v 7.0 \\ Samples \\ Security \\ X509 Certificate registro \\ Csharp \\ enrollWithIX509EnrollmentHelper.
