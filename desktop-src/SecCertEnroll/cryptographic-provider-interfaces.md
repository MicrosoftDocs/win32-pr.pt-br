# <a name="cryptographic-provider-interfaces"></a>Interfaces do provedor criptográfico

As interfaces a seguir podem ser usadas para recuperar informações sobre provedores criptográficos.



| Interface                                    | Descrição                                                                  |
|----------------------------------------------|------------------------------------------------------------------------------|
| [**ICspAlgorithm**](/windows/desktop/api/CertEnroll/nn-certenroll-icspalgorithm)       | Representa um algoritmo implementado por um provedor criptográfico.             |
| [**ICspAlgorithms**](/windows/desktop/api/CertEnroll/nn-certenroll-icspalgorithms)     | Gerencia uma coleção de [**objetos ICspAlgorithm.**](/windows/desktop/api/CertEnroll/nn-certenroll-ix509namevaluepair) |
| [**ICspInformation**](/windows/desktop/api/CertEnroll/nn-certenroll-icspinformation)   | Fornece acesso a informações gerais sobre um provedor criptográfico.       |
| [**ICspInformations**](/windows/desktop/api/CertEnroll/nn-certenroll-icspinformations) | Gerencia uma coleção de [**objetos ICspInformation.**](/windows/desktop/api/CertEnroll/nn-certenroll-icspinformation)  |
| [**ICspStatus**](/windows/desktop/api/CertEnroll/nn-certenroll-icspstatus)             | Contém informações sobre um par de provedor/algoritmo criptográfico.          |
| [**ICspStatuses**](/windows/desktop/api/CertEnroll/nn-certenroll-icspstatuses)         | Gerencia uma coleção de [**objetos ICspStatus.**](/windows/desktop/api/CertEnroll/nn-certenroll-icspstatus)            |
