# <a name="pki-elements"></a>Elementos PKI

Uma PKI (infraestrutura de chave pública) típica da Microsoft inclui os elementos a seguir.

| Elemento                                                                     | Descrição                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Autoridades de certificação](about-certification-authorities.md)<br/> | Forneça serviços que autenticam a identidade de indivíduos, computadores e outras entidades em uma rede. Isso abrange autoridades de certificação raiz e autoridades subordinadas.<br/> |
| [Certificar o diretório](about-certificate-directory.md)<br/>         | Salva solicitações de certificado e emitiu e revoga certificados e solicitações de certificado.<br/>                                                                                                      |
| [Servidor de Recuperação de Chave](about-key-recovery-server.md)<br/>             | Salva chaves privadas criptografadas no banco de dados de certificado para recuperação após a perda.<br/>                                                                                                             |
