# <a name="pki-elements"></a>Elementos PKI

Uma infraestrutura típica de chave pública da Microsoft (PKI) inclui os elementos a seguir.

| Elemento                                                                     | Descrição                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Autoridades de certificação](about-certification-authorities.md)<br/> | Forneça serviços que autentiquem a identidade de indivíduos, computadores e outras entidades em uma rede. Isso abrange tanto as autoridades de certificação raiz quanto as autoridades subordinadas.<br/> |
| [Certificar o diretório](about-certificate-directory.md)<br/>         | Salva solicitações de certificado e certificados emitidos e revogados e solicitações de certificado.<br/>                                                                                                      |
| [Servidor de recuperação de chave](about-key-recovery-server.md)<br/>             | Salva as chaves privadas criptografadas no banco de dados de certificado para recuperação após a perda.<br/>                                                                                                             |
