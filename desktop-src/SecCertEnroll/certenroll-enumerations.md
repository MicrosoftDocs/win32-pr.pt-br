# <a name="certenroll-enumerations"></a>Enumerações CertEnroll

As seguintes enumerações são suportadas pela API de registro de certificado.

| Enumeração                                                                              | Descrição                                                                                                                                                                                                                                                                                                              |
|------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AlgorithmFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmflags)                                            | Contém sinalizadores que podem ser usados para refinar a pesquisa de um algoritmo criptográfico.                                                                                                                                                                                                                                      |
| [**AlgorithmOperationFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmoperationflags)                          | Especifica as operações que um algoritmo pode executar. Isso é usado para recuperar os recursos operacionais de um algoritmo CSP ou informações de status do CSP com base nesses recursos.                                                                                                                                  |
| [**AlgorithmType**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmtype)                                              | Especifica a finalidade pretendida de um algoritmo criptográfico.                                                                                                                                                                                                                                                             |
| [**AlternativeNameType**](/windows/desktop/api/CertEnroll/ne-certenroll-alternativenametype)                                  | Especifica os tipos de nome alternativos que podem ser especificados ao criar uma extensão **alternativoname** .                                                                                                                                                                                                               |
| [**\_OBJECTID CERTENROLL**](/windows/desktop/api/CertEnroll/ne-certenroll-certenroll_objectid)                                 | Contém os [*identificadores de objeto*](/windows/desktop/SecGloss/o-gly) predefinidos (OIDs) com suporte para a API de registro de certificado.                                                                                                                                          |
| [**\_Propriedade CERTENROLL**](/windows/desktop/api/CertEnroll/ne-certenroll-certenroll_propertyid)                             | Contém OIDs predefinidos para propriedades externas com suporte pela API de registro de certificado.                                                                                                                                                                                                                            |
| [**CommitTemplateFlags**](/windows/desktop/api/Certenroll/ne-certenroll-committemplateflags)                                       | Especifica opções para salvar e excluir modelos.                                                                                                                                                                                                                                                                     |
| [**EncodingType**](/windows/desktop/api/CertEnroll/ne-certenroll-encodingtype)                                                | Especifica o tipo de codificação aplicado a uma matriz de bytes para fins de exibição.                                                                                                                                                                                                                                             |
| [**EnrollmentCAProperty**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentcaproperty)                                     | Especifica os valores de propriedade da autoridade de certificação.                                                                                                                                                                                                                                                                       |
| [**EnrollmentDisplayStatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentdisplaystatus)                          | Especifica se as informações de status do registro devem ser exibidas em uma interface do usuário.                                                                                                                                                                                                                                          |
| [**EnrollmentEnrollStatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentenrollstatus)                            | Especifica o status de registro de uma solicitação de certificado.                                                                                                                                                                                                                                                                |
| [**EnrollmentPolicyFlags**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentpolicyflags)                                   | Especifica sinalizadores de política de grupo.                                                                                                                                                                                                                                                                                            |
| [**EnrollmentPolicyServerPropertyFlags**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentpolicyserverpropertyflags)       | Especifica o servidor de políticas padrão.                                                                                                                                                                                                                                                                                     |
| [**EnrollmentSelectionStatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentselectionstatus)                      | Especifica se o status de registro de um objeto será monitorado durante o processo de registro.                                                                                                                                                                                                                    |
| [**EnrollmentTemplateProperty**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmenttemplateproperty)                         | Contém valores de propriedade para um determinado modelo.                                                                                                                                                                                                                                                                           |
| [**InnerRequestLevel**](/windows/desktop/api/CertEnroll/ne-certenroll-innerrequestlevel)                                      | Especifica o nível de confinamento de uma solicitação de certificado em uma \# solicitação PKCS 7 ou CMC.                                                                                                                                                                                                                               |
| [**InstallResponseRestrictionFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-installresponserestrictionflags)          | Contém sinalizadores que identificam as restrições colocadas na instalação local de uma cadeia de certificados.                                                                                                                                                                                                                   |
| [**KeyIdentifierHashAlgorithm**](/windows/desktop/api/CertEnroll/ne-certenroll-keyidentifierhashalgorithm)                    | Especifica o algoritmo usado para o hash da chave pública em uma solicitação de certificado.                                                                                                                                                                                                                                            |
| [**ObjectIdGroupId**](/windows/desktop/api/CertEnroll/ne-certenroll-objectidgroupid)                                          | Especifica a categoria ou o grupo ao qual um OID pertence.                                                                                                                                                                                                                                                                 |
| [**ObjectIdPublicKeyFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-objectidpublickeyflags)                            | Especifica se um algoritmo de chave pública é usado para assinatura ou criptografia.                                                                                                                                                                                                                                          |
| [**PFXExportOptions**](/windows/desktop/api/CertEnroll/ne-certenroll-pfxexportoptions)                                        | Especifica a quantidade de uma cadeia de certificados incluída ao criar uma mensagem de troca de informações pessoais (PFX).                                                                                                                                                                                                       |
| [**Pkcs10AllowedSignatureTypes**](/windows/desktop/api/CertEnroll/ne-certenroll-pkcs10allowedsignaturetypes)                  | Especifica o tipo de assinatura permitida ao assinar uma solicitação de certificado.                                                                                                                                                                                                                                            |
| [**PolicyQualifierType**](/windows/desktop/api/CertEnroll/ne-certenroll-policyqualifiertype)                                  | Especifica o tipo de qualificador aplicado a uma política de certificado.                                                                                                                                                                                                                                                         |
| [**PolicyServerUrlFlags**](/windows/desktop/api/Certenroll/ne-certenroll-policyserverurlflags)                                     | Contém sinalizadores de servidor de política de registro de certificado (CEP).                                                                                                                                                                                                                                                               |
| [**PolicyServerUrlPropertyID**](/windows/desktop/api/Certenroll/ne-certenroll-policyserverurlpropertyid)                           | Contém valores que especificam o tipo de valor da propriedade a ser retornado pelo método [**Getstringproperty**](/windows/desktop/api/Certenroll/nf-certenroll-ix509policyserverurl-getstringproperty) ou definido pelo método [**setstringproperty**](/windows/desktop/api/Certenroll/nf-certenroll-ix509policyserverurl-setstringproperty) na interface [**IX509PolicyServerUrl**](/windows/desktop/api/Certenroll/nn-certenroll-ix509policyserverurl) . |
| [**RequestClientInfoClientId**](/windows/desktop/api/CertEnroll/ne-certenroll-requestclientinfoclientid)                      | Especifica o tipo de aplicativo que criou uma solicitação de certificado.                                                                                                                                                                                                                                                    |
| [**WebEnrollmentFlags**](/windows/desktop/api/Certenroll/ne-certenroll-webenrollmentflags)                                         | Especifica o comportamento do registro na Web.                                                                                                                                                                                                                                                                                       |
| [**WebSecurityLevel**](/windows/desktop/api/CertEnroll/ne-certenroll-websecuritylevel)                                        | Especifica se um método ou propriedade habilitado para Web é seguro para scripts.                                                                                                                                                                                                                                                |
| [**X500NameFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x500nameflags)                                              | Especifica as características de exibição e codificação de um nome diferenciado ou nome distinto relativo.                                                                                                                                                                                                               |
| [**X509CertificateEnrollmentContext**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificateenrollmentcontext)        | Especifica a natureza da entidade final para a qual o certificado se destina.                                                                                                                                                                                                                                            |
| [**X509CertificateTemplateEnrollmentFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplateenrollmentflag)   | Contém valores que especificam ações de servidor e cliente durante o registro.                                                                                                                                                                                                                                                |
| [**X509CertificateTemplateGeneralFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplategeneralflag)         | Contém informações de uso e modificação sobre modelos e certificados associados.                                                                                                                                                                                                                                   |
| [**X509CertificateTemplatePrivateKeyFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplateprivatekeyflag)   | Contém valores que especificam ações de cliente em relação a uma chave privada.                                                                                                                                                                                                                                                     |
| [**X509CertificateTemplateSubjectNameFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplatesubjectnameflag) | Contém valores que especificam ações de servidor e cliente relacionadas a nomes de entidades.                                                                                                                                                                                                                                         |
| [**X509EnrollmentPolicyExportFlags**](/windows/desktop/api/Certenroll/ne-certenroll-x509enrollmentpolicyexportflags)               | Especifica quais itens exportar do servidor de políticas.                                                                                                                                                                                                                                                                   |
| [**X509EnrollmentPolicyLoadOption**](/windows/desktop/api/Certenroll/ne-certenroll-x509enrollmentpolicyloadoption)                 | Especifica como recuperar a política do servidor de políticas.                                                                                                                                                                                                                                                                 |
| [**X509KeySpec**](/windows/desktop/api/CertEnroll/ne-certenroll-x509keyspec)                                                  | Especifica o uso pretendido de uma chave para um CSP herdado.                                                                                                                                                                                                                                                                    |
| [**X509KeyUsageFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509keyusageflags)                                      | Especifica a finalidade de uma chave contida em um certificado.                                                                                                                                                                                                                                                               |
| [**X509PrivateKeyExportFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyexportflags)                           | Especifica a política de exportação para uma chave privada.                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyProtection**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyprotection)                             | Especifica o nível de proteção de chave privada com suporte por um KSP ou CSP.                                                                                                                                                                                                                                                 |
| [**X509PrivateKeyUsageFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyusageflags)                             | Especifica os usos permitidos de uma chave privada.                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyVerify**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyverify)                                     | Especifica se uma interface do usuário é exibida durante a verificação de chave privada e se a verificação pode continuar se o provedor ou o KSP representar um cartão inteligente.                                                                                                                                                           |
| [**X509ProviderType**](/windows/desktop/api/CertEnroll/ne-certenroll-x509providertype)                                        | Especifica o tipo de um provedor de serviços de criptografia ou provedor de serviços de chave.                                                                                                                                                                                                                                          |
| [**X509RequestInheritOptions**](/windows/desktop/api/CertEnroll/ne-certenroll-x509requestinheritoptions)                      | Especifica como as chaves, os valores de extensão e as propriedades externas são herdadas quando uma nova solicitação é criada a partir de um certificado existente.                                                                                                                                                                                  |
| [**X509RequestType**](/windows/desktop/api/CertEnroll/ne-certenroll-x509requesttype)                                          | Especifica o tipo de solicitação de certificado.                                                                                                                                                                                                                                                                                  |
| [**X509SCEPDisposition**](/windows/desktop/api/certpol/ne-certpol-x509scepdisposition)                                       | Descreve a disposição resultante de uma solicitação para processar uma mensagem de resposta.                                                                                                                                                                                                                                          |
| [**X509SCEPFailInfo**](/windows/desktop/api/certpol/ne-certpol-x509scepfailinfo)                                             | Descreve a natureza de uma falha de registro de certificado SCEP.                                                                                                                                                                                                                                                          |
