# <a name="iobjectids-methods"></a>Métodos IObjectIds

A interface [**IObjectIds**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectids) expõe os métodos a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                            | Descrição                                                                                     |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Adicionar método**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-add)<br/>           | Adiciona um objeto [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) à coleção.<br/>                     |
| [**Método AddRange**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-addrange)<br/> | Adiciona um intervalo de objetos [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) à coleção.<br/>            |
| [**Método Clear**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-clear)<br/>       | Remove todos os objetos [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) da coleção.<br/>              |
| [**Remover método**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-remove)<br/>     | Remove um objeto [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) da coleção por valor de índice.<br/> |
