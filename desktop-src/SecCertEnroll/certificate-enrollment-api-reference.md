# <a name="certificate-enrollment-api-reference"></a>Referência da API de Registro de Certificado

As páginas de referência da API de Registro de Certificado contêm descrições detalhadas de interfaces, enumerações e outros elementos de programação que podem ser usados para registrar um usuário ou computador em uma hierarquia de certificados. As páginas de referência são divididas nos seguintes grupos:

-   [CertEnroll Interfaces](certenroll-interfaces.md)
-   [Enumerações CertEnroll](certenroll-enumerations.md)
