# <a name="icertificateattestationchallenge-methods"></a>Métodos ICertificateAttestationChallenge

A interface [**ICertificateAttestationChallenge**](/windows/desktop/api/Certenroll/nn-certenroll-icertificateattestationchallenge) dá suporte aos métodos a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                           | Descrição                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Método DecryptChallenge**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-decryptchallenge)<br/> | Descriptografa o desafio da resposta do CMC ( [Gerenciamento de certificados sobre o CMS](/windows/desktop/SecGloss/c-gly) ) e cria uma resposta criptografada novamente para enviar para a autoridade de certificação.<br/> |
| [**Método Initialize**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-initialize)<br/>             | Inicialize usando a resposta do CMC ( [Gerenciamento de certificado completo sobre o CMS](/windows/desktop/SecGloss/c-gly) ) retornada da autoridade de certificação.<br/>                                        |
