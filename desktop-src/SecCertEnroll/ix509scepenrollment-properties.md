# <a name="ix509scepenrollment-properties"></a>Propriedades de IX509SCEPEnrollment

A interface [**IX509SCEPEnrollment**](/windows/desktop/api/Certenroll/nn-certenroll-ix509scepenrollment) expõe as propriedades a seguir.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                              | Descrição                                                                                                                                            |
|----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Propriedade de certificado**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificate)<br/>                         | Obtém o certificado para a solicitação.<br/>                                                                                                       |
| [**Propriedade CertificateFriendlyName**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificatefriendlyname)<br/> | Obtém ou define o nome amigável do certificado.<br/>                                                                                         |
| [**Propriedade FailInfo**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_failinfo)<br/>                               | Obtém informações quando o método [**ProcessResponseMessage**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-processresponsemessage) detecta um ambiente com falha.<br/> |
| [**Propriedade OldCertificate**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_oldcertificate)<br/>                   | Obtém ou define um certificado antigo que uma solicitação deve substituir.<br/>                                                                      |
| [**Propriedade de solicitação**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_request)<br/>                                 | Obtém a solicitação PKCS10 interna.<br/>                                                                                                              |
| [**Propriedade ServerCapabilities**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_servercapabilities)<br/>           | Define o hash preferencial e os algoritmos de criptografia para a solicitação.<br/>                                                                          |
| [**Propriedade SignerCertificate**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_signercertificate)<br/>             | Obtém ou define o certificado do signatário para a solicitação.<br/>                                                                                        |
| [**Propriedade silenciosa**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_silent)<br/>                                   | Obtém ou define se a interface do usuário deve ser permitida durante a solicitação.<br/>                                                                                        |
| [**Propriedade de status**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_status)<br/>                                   | Obtém o status da solicitação.<br/>                                                                                                             |
| [**Propriedade TransactionId**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_transactionid)<br/>                     | Obtém ou define a ID da transação para a solicitação.<br/>                                                                                            |
