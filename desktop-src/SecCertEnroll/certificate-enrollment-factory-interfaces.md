# <a name="certificate-enrollment-factory-interfaces"></a>Interfaces de fábrica de registro de certificado

As interfaces a seguir podem ser usadas para criar objetos de API de registro de certificado em uma página da Web.



| Interface                                                                | Descrição                                                                            |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**IX509EnrollmentWebClassFactory**](/windows/desktop/api/CertEnroll/nn-certenroll-ix509enrollmentwebclassfactory) | Cria um objeto no contexto de usuário em uma página da Web.                                    |
| [**IX509MachineEnrollmentFactory**](/windows/desktop/api/Certenroll/nn-certenroll-ix509machineenrollmentfactory)   | Cria um objeto [**IX509EnrollmentHelper**](/windows/desktop/api/Certenroll/nn-certenroll-ix509enrollmenthelper) em uma página da Web. |
