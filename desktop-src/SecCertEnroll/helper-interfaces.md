# <a name="helper-interfaces"></a>Interfaces auxiliares

As interfaces de uso gerais a seguir têm suporte da API de registro de certificado.



| Interface                                                                    | Descrição                                                                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBinaryConverter**](/windows/desktop/api/CertEnroll/nn-certenroll-ibinaryconverter)                                 | Cria uma cadeia de caracteres codificada em Unicode de uma matriz de bytes, cria uma matriz de bytes a partir de uma cadeia de caracteres codificada em Unicode e modifica o tipo de codificação Unicode aplicado a uma cadeia de caracteres. |
| [**ICertificateAttestationChallenge**](/windows/desktop/api/Certenroll/nn-certenroll-icertificateattestationchallenge) | Dá suporte a desafios de atestado de certificado.                                                                                                                           |
| [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid)                                               | Representa um identificador de objeto.                                                                                                                                       |
| [**IObjectIds**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectids)                                             | Gerencia uma coleção de objetos [**IObjectId**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) .                                                                                                        |
| [**IX500DistinguishedName**](/windows/desktop/api/CertEnroll/nn-certenroll-ix500distinguishedname)                     | Representa um nome diferenciado X. 500.                                                                                                                                |
| [**IX509EndorsementKey**](/windows/desktop/api/Certenroll/nn-certenroll-ix509endorsementkey)                           | Interface de chave de endosso X. 509                                                                                                                                        |
