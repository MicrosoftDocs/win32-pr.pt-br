# <a name="about-the-certificate-enrollment-api"></a>Sobre a API de registro de certificado

A API de registro de certificado permite que você crie aplicativos cliente que podem ser usados para solicitar e instalar certificados. As solicitações de certificado são discutidas nas seções a seguir. Para obter as informações mais recentes, consulte a biblioteca do Microsoft Developer Network (MSDN) ( [https://msdn.microsoft.com/library/default.asp](/documentation/) ).

-   [Infraestrutura de chave pública](public-key-infrastructure.md)
-   [Codificação de solicitação de certificado](about-certificate-request-encoding.md)
-   [Solicitações de certificado](certificate-requests.md)
-   [Solicitações de amostra](sample-requests.md)
