# <a name="about-the-certificate-enrollment-api"></a>Sobre a API de Registro de Certificado

A API de Registro de Certificado permite que você crie aplicativos cliente que podem ser usados para solicitar e instalar certificados. As solicitações de certificado são discutidas nas seções a seguir. Para obter as informações mais recentes, consulte a biblioteca Desenvolvedor Microsoft (MSDN) do Desenvolvedor Microsoft ( [https://msdn.microsoft.com/library/default.asp](/documentation/) ).

-   [Infraestrutura de chave pública](public-key-infrastructure.md)
-   [Codificação de solicitação de certificado](about-certificate-request-encoding.md)
-   [Solicitações de certificado](certificate-requests.md)
-   [Solicitações de exemplo](sample-requests.md)
