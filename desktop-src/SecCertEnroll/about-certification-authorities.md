<span data-ttu-id="92922-104">Uma [*autoridade de certificação*](/windows/desktop/SecGloss/c-gly) (CA) é responsável por atestar a identidade de usuários, computadores e organizações.</span><span class="sxs-lookup"><span data-stu-id="92922-104">A [*certification authority*](/windows/desktop/SecGloss/c-gly) (CA) is responsible for attesting to the identity of users, computers, and organizations.</span></span> <span data-ttu-id="92922-105">A AC autentica uma entidade e garante-a emitindo um certificado assinado digitalmente.</span><span class="sxs-lookup"><span data-stu-id="92922-105">The CA authenticates an entity and vouches for that identity by issuing a digitally signed certificate.</span></span> <span data-ttu-id="92922-106">A AC também pode gerenciar, revogar e renovar certificados.</span><span class="sxs-lookup"><span data-stu-id="92922-106">The CA can also manage, revoke, and renew certificates.</span></span>

<span data-ttu-id="92922-107">Uma CA pode ser pública ou privada.</span><span class="sxs-lookup"><span data-stu-id="92922-107">A CA can be public or private.</span></span> <span data-ttu-id="92922-108">Uma CA pública fornece serviços de certificação, normalmente por uma taxa, para o público pela Internet.</span><span class="sxs-lookup"><span data-stu-id="92922-108">A public CA provides certification services, typically for a fee, to the public over the Internet.</span></span> <span data-ttu-id="92922-109">Uma AC privada fornece esse serviço para os membros de uma população delimitada, como os funcionários de uma empresa ou membros de algum outro grupo privado.</span><span class="sxs-lookup"><span data-stu-id="92922-109">A private CA provides this service to the members of a delimited population such as the employees of a business or members of some other private group.</span></span>

<span data-ttu-id="92922-110">O meio pelo qual uma autoridade de certificação autentica um usuário final é variado e está além do escopo desta documentação.</span><span class="sxs-lookup"><span data-stu-id="92922-110">The means by which a CA authenticates an end user are varied and beyond the scope of this documentation.</span></span> <span data-ttu-id="92922-111">No entanto, claramente, os métodos de autenticação variam de acordo com o tipo de provedor.</span><span class="sxs-lookup"><span data-stu-id="92922-111">Clearly, however, methods of authentication vary by type of provider.</span></span> <span data-ttu-id="92922-112">Por exemplo, uma autoridade de certificação privada pode estabelecer a identidade dos usuários finais referindo-se a uma lista de grupos, como um banco de dados de funcionários ou Active Directory.</span><span class="sxs-lookup"><span data-stu-id="92922-112">For example, a private CA can establish the identity of end users by referring to a group roster such as an employee database or Active Directory.</span></span> <span data-ttu-id="92922-113">Os métodos de autenticação executados por uma CA pública geralmente são mais complexos e dependem parcialmente do nível de garantia sendo prometido pelo certificado.</span><span class="sxs-lookup"><span data-stu-id="92922-113">The authentication methods performed by a public CA are generally more complex and depend partly on the level of assurance being promised by the certificate.</span></span>

<span data-ttu-id="92922-114">À medida que a população de uma PKI (infraestrutura de chave pública) cresce, pode ser difícil para uma única AC gerenciar efetivamente todos os certificados <span data-ttu-id="92922-121">O termo autoridade de certificação pode se referir à organização que comprovará a identidade de um usuário final e o servidor usado pela organização para emitir e gerenciar certificados.</span><span class="sxs-lookup"><span data-stu-id="92922-121">The term certification authority can refer to both the organization that vouches for the identity of an end user and the server used by the organization to issue and manage certificates.</span></span> <span data-ttu-id="92922-122">Um Windows Server pode ser configurado para atuar como um servidor de CA, e esta documentação geralmente se refere ao servidor ao usar o termo CA.</span><span class="sxs-lookup"><span data-stu-id="92922-122">A Windows server can be configured to act as a CA server, and this documentation usually refers to the server when using the term CA.</span></span>
