# <a name="rgb-color-spaces"></a>Espaços de cores RGB

Um [espaço de cores](c.md) RGB é criado mapeando as cores vermelho, verde e azul em um sistema de coordenadas Cartesianas 3D. Esses resultados são um cubo 3D como aqueles mostrados na figura a seguir. Esta figura exibe o mesmo cubo RGB de dois ângulos diferentes. Observe que a origem do sistema de coordenadas é preta. É aí que os componentes de cores vermelho, verde e azul (RGB) são todos 0,0. O canto diagonal oposto do cubo é branco, onde os componentes de cor RGB estão em seu valor máximo.

![cubo de espaço de cores RGB em valores máximos](images/rgbclrs1.png)

![cubo de espaço de cores RGB em valores mínimos](images/rgbclrs2.png)

Como a maioria dos [espaços de cores](c.md), o espaço de cores RGB é normalizado. Ou seja, todos os valores de cor são restritos ao intervalo de zero a um inclusivo. Portanto, o preto é (0,0, 0,0, 0,0) e o branco é (1,0, 1,0, 1,0).

No espaço de cores RGB, as [cores primárias](p.md) são vermelho, verde e azul. As cores secundárias são ciano, amarelo e magenta.

Os espaços de cores RGB podem ser dependentes de dispositivo ou independentes de dispositivo.
