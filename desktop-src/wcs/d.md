# <a name="d"></a>D

## <a name="dac"></a>DAC

Conversor digital para analógico.

## <a name="device-link-profile"></a>Perfil de link do dispositivo

Um arquivo que contém uma concatenação de conversões de cor que são usadas com frequência.

## <a name="device-profile"></a>perfil do dispositivo

Um arquivo que contém informações sobre como caracterizar os recursos de cores de um dispositivo. Os perfis de dispositivo ICC convertem cores da gama de um dispositivo nos PCS. Ele também contém as informações necessárias para a conversão de cores dos PCS na gama do dispositivo. Os perfis de dispositivo XML WCS fornecem os dados de medição e as referências de algoritmos para converter entre o espaço de dispositivo nativo e o CIEXYZ.

## <a name="dmp"></a>DMP

O formato de perfil de modelo de dispositivo de cores é um formato de arquivo baseado em XML que contém medidas necessárias para descrever a relação entre valores de espaço de cores do dispositivo nativo e valores de CIEXYZ.
