# <a name="obsolete-wcs-functions"></a>Funções de WCS obsoletas

Esta seção delineia funções obsoletas no WCS 1.0. Atualmente, eles são mantidos para compatibilidade com backward. No entanto, eles serão removidos da API em versões posteriores do WCS.



|     Função                               |      Descrição                             |
|--------------------------------------------|----------------------------------------------|
| [**UpdateICMRegKey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Obsoleto:* gerencia perfis de cores e CMMs. |
