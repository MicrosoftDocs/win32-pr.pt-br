# <a name="obsolete-wcs-functions"></a>Funções de WCS obsoletas

Esta seção delineia as funções que estão obsoletas no WCS 1,0. Atualmente, eles são mantidos para compatibilidade com versões anteriores. No entanto, eles serão removidos da API em versões posteriores do WCS.



|                                            |                                              |
|--------------------------------------------|----------------------------------------------|
| [**UpdateICMRegKey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Obsoleto*: gerencia perfis de cor e CMMS. |
