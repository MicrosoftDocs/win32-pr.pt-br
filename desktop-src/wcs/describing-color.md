# <a name="describing-color"></a>Descrever a cor

A cor pode ser descrita em termos de componentes. Os componentes usados com mais frequência por profissionais de geração de imagens são *matiz*, *croma* e *claridade*. Um matiz é o que normalmente é considerado uma cor. O matiz de uma cor corrige seu lugar no espectro visível de luz. O croma de uma cor é como "puro" ou "forte" é uma cor. Um cinza neutro é considerado com zero saturação. A claridade de uma cor refere-se à intensidade de luz que é refletida ou transmitida por uma imagem.

Os termos *tonalidade*, *Tom* e *sombreamento* também são bem usados em literatura de imagens coloridas. Um tom de uma cor é obtido com a mistura de seu matiz com branco. Um tom de uma cor é criado com a combinação de um matiz com cinza. Uma tonalidade de cor é feita adicionando preto ao seu matiz.
