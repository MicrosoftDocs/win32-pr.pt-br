# <a name="hls-color-spaces"></a>Espaços de Cores HLS

Os espaços de [cores HLS](c.md) também são amplamente usados por artista. Seus componentes de cor são matiz, leveza e saturação (chroma).

O Hue tem o mesmo significado que o modelo HSV, exceto que um ângulo de matiz de 0 corresponde ao azul neste modelo. O magenta está em 60, vermelho está em 120. Assim como no modelo HSV, as cores complementares têm 180 de diferença.

A luz é a quantidade de preto ou branco em uma cor. Aumentar a leveidade adiciona branco ao matiz. Diminuir a luz adiciona preto ao matiz.

[A saturação](s.md) no modelo HLS é uma medida da " puridade" de um matiz. À medida que a saturação é reduzida, o matiz fica mais cinza. Um valor de saturação de zero resulta em um valor de escala de cinza.

A figura a seguir é um desenho de linha do espaço HLS, que é um hexacone duplo. Qualquer seção cruzada horizontal do espaço de cores HLS é um hexagono. O HLS é um espaço de cores normalizado. Ou seja, os valores de leveza e saturação variam de 0,0 a 1,0, inclusive. O matiz varia de 0 a 360, inclusive.

![hls color space](images/hlsline.png)

Espaços de cores HLS podem ser dependentes do dispositivo ou independentes do dispositivo.
