# <a name="hls-color-spaces"></a>Espaços de Cores HLS

Os [espaços de cores](c.md) HLS também são amplamente usados por artistas. Seus componentes de cor são matiz, claridade e saturação (croma).

O matiz tem o mesmo significado que o modelo HSV, exceto que um ângulo de matiz de 0 corresponde a azul nesse modelo. O magenta está em 60, o vermelho está em 120. Assim como acontece com o modelo HSV, as cores complementares são de diferença de 180.

A claridade é a quantidade de preto ou branco em uma cor. Aumentar a claridade adiciona branco ao matiz. Diminuir a claridade adiciona preto ao matiz.

A [saturação](s.md) no modelo HLS é uma medida da "pureza" de um matiz. À medida que a saturação é diminuída, o matiz se torna mais cinza. Um valor de saturação igual a zero resulta em um valor de escala cinza.

A figura a seguir é um desenho de linha de espaço HLS, que é um hexcone duplo. Qualquer seção cruzada horizontal do espaço de cores HLS é um hexágono. HLS é um espaço de cores normalizado. Ou seja, os valores de luminosidade e saturação variam de 0,0 a 1,0, inclusive. O matiz varia de 0 a 360, inclusive.

![espaço de cores HLS](images/hlsline.png)

Os espaços de cores HLS podem ser dependentes de dispositivo ou independentes de dispositivo.
