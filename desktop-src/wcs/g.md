# <a name="g"></a>G

## <a name="gamma-correction"></a>correção gama

As características do Tom físico nativo de um dispositivo geralmente são diferentes em tipos diferentes de dispositivos. O fator de correção gama compensa essas diferenças.

## <a name="gamma-curve"></a>curva gama

Uma coleção de correções de gama em todo o intervalo de cores. Quando plotado em um grafo, eles formam uma curva.

## <a name="gamma-ramp"></a>rampa gama

Consulte curva gama.

## <a name="gamut"></a>gama

O conjunto de cores de um espaço de cores que um dispositivo pode produzir.

## <a name="gmmp"></a>GMMP

O formato do perfil do modelo de mapa de cores é um formato de arquivo baseado em XML que controla o mapeamento de cores entre as gamas do dispositivo de origem e de destino.
