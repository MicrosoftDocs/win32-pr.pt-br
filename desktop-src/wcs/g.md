# <a name="g"></a>G

## <a name="gamma-correction"></a>correção gama

As características de tom físico nativo de um dispositivo geralmente são diferentes em diferentes tipos de dispositivos. O fator de correção gama compensa essas diferenças.

## <a name="gamma-curve"></a>curva gama

Uma coleção de correções gama em todo o intervalo de cores. Quando plotados em um grafo, eles formam uma curva.

## <a name="gamma-ramp"></a>rampa gama

Consulte curva gama.

## <a name="gamut"></a>Gama

O conjunto de cores de um espaço de cores que um dispositivo pode produzir.

## <a name="gmmp"></a>GMMP

O Formato de Perfil do Modelo de Mapa de Jogos de Cores é um formato de arquivo baseado em XML que controla o mapeamento de cores entre os jogos do dispositivo de origem e de destino.
