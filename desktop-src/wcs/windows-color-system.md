# <a name="windows-color-system"></a>Sistema de cores do Windows

## <a name="purpose"></a>Finalidade

Os esquemas de gerenciamento de cores são implementados no Microsoft Windows sistemas operacionais para melhorar a renderização do conteúdo de cores em todas as formas de comunicação digital.

O esquema de gerenciamento de cores usado a partir do Windows 2000, Windows XP e Windows Server 2003 é chamado de Gerenciamento de Cores de Imagem (ICM) 2.0. O esquema de gerenciamento de cores usado a partir do Windows Vista é chamado Windows Sistema de Cores (WCS) 1.0. O esquema de gerenciamento de cores do WCS 1.0 é um superconjunto de ICM APIs e funcionalidade 2.0.

## <a name="where-applicable"></a>Quando aplicável

ICM pode ser usado em todos os aplicativos Windows sistemas operacionais 2000 e posteriores. O WCS pode ser usado em todos os aplicativos Windows Vista e sistemas operacionais posteriores.

## <a name="developer-audience"></a>Público de desenvolvedores

A API do WCS foi projetada para uso por programadores C/C++. Familiaridade com a Windows gráfica do usuário, a arquitetura controlada por mensagens e um conhecimento prático dos conceitos de gerenciamento de cores são necessários.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

Aplicativos que usam a API ICM aplicativos exigem Windows 2000 ou posterior. Aplicativos que usam a API do WCS exigem Windows Vista ou posterior. Para obter informações específicas de tempo de executar em uma função, consulte a seção Requisitos da página de referência para essa função.
