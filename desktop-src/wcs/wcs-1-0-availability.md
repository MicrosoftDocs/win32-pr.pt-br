# <a name="wcs-10-availability"></a>Disponibilidade do WCS 1.0

O ICM1 está disponível para o Microsoft Windows 95. Ele não está disponível para MS DOS, Windows 3. *x* ou Windows NT 4,0 ou anterior.

O ICM2 está disponível para o Microsoft Windows 98, o Windows Millennium Edition (me), o Windows 2000 e o Windows XP. Ele não está disponível para MS DOS, Windows 3. *x*, Windows 95 ou Windows NT 4,0 ou anterior.

O WCS 1,0 está disponível para o Microsoft Windows Vista. Ele não está disponível para o Windows XP ou anterior.
