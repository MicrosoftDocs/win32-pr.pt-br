# <a name="wcs-10-availability"></a>Disponibilidade do WCS 1.0

o ICM1 está disponível para o Microsoft Windows 95. ele não está disponível para o MS DOS, Windows 3. *x* ou Windows NT 4,0 ou anterior.

o ICM2 está disponível para o Microsoft Windows 98, Windows Millennium Edition (Me), Windows 2000 e Windows XP. ele não está disponível para o MS DOS, Windows 3. *x*, Windows 95 ou Windows NT 4,0 ou anterior.

o WCS 1,0 está disponível para o Microsoft Windows Vista. ele não está disponível para o Windows XP ou anterior.
