# <a name="t"></a>T

## <a name="tint"></a>Nuance

Um tom de uma cor é criado com a combinação de um matiz com branco.

## <a name="tone"></a>tonal

Um tom de uma cor é obtido com a mistura de seu matiz com cinza.
