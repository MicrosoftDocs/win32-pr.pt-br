# <a name="additive-primary-colors"></a>Cores primárias aditivas

O método aditivo de combinação de cores se baseia na suposição de que você começa com preto. Ou seja, se não houver nenhuma outra cor presente (a imagem é preta) e você adicionar vermelho, a imagem aparecerá vermelha. Se você adicionar azul, a imagem aparecerá em magenta.

As três [cores primárias](p.md) aditivas são vermelho, verde e azul. A figura a seguir ilustra a combinação de cores primárias aditiva.
