# <a name="additive-primary-colors"></a>Cores primárias aditivas

O método aditivo de combinação de cores baseia-se na suposição de que você começa com preto. Ou seja, se não houver outras cores presentes (a imagem é preta) e você adicionar vermelho, a imagem aparecerá vermelha. Se você adicionar azul, a imagem será exibida em magenta.

As três cores [primárias aditivas](p.md) são vermelho, verde e azul. A figura a seguir ilustra a combinação de cores primárias aditivas.
