# <a name="basic-color-management-concepts"></a>Conceitos básicos de gerenciamento de cores

Para entender como o WCS (Windows Color System) versão 1,0 é usado, é necessário um conhecimento prático dos conceitos de gerenciamento de cores. Esta visão geral apresenta o plano de fundo de gerenciamento de cores suficiente para fornecer uma definição dos termos e conceitos de gerenciamento de cores que aparecem na referência do programador do WCS na documentação do SDK (Software Development Kit) da plataforma. Não é exaustiva em sua cobertura. Os tópicos desta seção abrangem as principais áreas de gerenciamento de cores, como:

-   [Cor na geração de imagens](color-in-imaging.md)
-   [Espaços de cores](color-spaces.md)
-   [Conversão de cores e correspondência de cores](color-conversion-and-color-matching.md)
-   [Algoritmos e esquemas do sistema de cores do Windows](windows-color-system-schemas-and-algorithms.md)

-   [Mais informações](further-information.md)
