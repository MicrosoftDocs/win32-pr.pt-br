# <a name="h"></a>H

## <a name="hard-proofing"></a>verificação de hardware

Visualização ou prova de uma imagem imprimindo uma cópia impressa dela.

## <a name="hls"></a>HLS

Um espaço de cores geralmente usado por artistas. Seus componentes são matiz, claridade e saturação (croma).

## <a name="hsv"></a>HSV

Um espaço de cores geralmente usado por artistas. Seus componentes são matiz, saturação (croma) e valor (claridade).

## <a name="hue"></a>Hue

Uma cor pura. Um matiz é uma posição específica na parte visível do espectro eletromagnética.
