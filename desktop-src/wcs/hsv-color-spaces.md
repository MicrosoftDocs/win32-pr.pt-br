# <a name="hsv-color-spaces"></a>Espaços de cores HSV

Os [espaços de cores](c.md) de matiz, saturação e valor (HSV) geralmente são usados por artistas. "Matiz" é o que normalmente pensamos como cor. É o atributo de uma cor pela qual damos um nome como "vermelho" ou "azul". "Value" é outra palavra para "claridade", o atributo de uma cor que faz com que pareça equivalente a algum tom de cinza entre preto e branco. Saturação é uma medida de quão diferente uma cor aparece de um cinza da mesma [claridade](b.md). Saturação zero indica que não há matiz, apenas a escala de cinza. O espaço de cores HSV é normalizado.

![espaço de cores HSV](images/hsvline.png)

A figura anterior mostra um desenho de linha de espaço HSV na forma de um hexcone. Cada uma de suas seções cruzadas é um hexágono. Nos vértices de cada seção cruzada estão as cores vermelho, amarelo, verde, ciano, azul e magenta. Uma cor no espaço HSV é especificada informando um ângulo de matiz, o nível de croma e o nível de claridade. Um ângulo de matiz de zero é vermelho. O ângulo de matiz aumenta em uma direção no sentido anti-horário. As cores complementares são de diferença de 180.

Os espaços de cores HSV podem ser dependentes de dispositivo ou independentes de dispositivo.
