# <a name="hsv-color-spaces"></a>Espaços de cores HSV

Espaços de cores matiz, saturação [](c.md) e valor (HSV) geralmente são usados por artista. "Matiz" é o que normalmente vemos como cor. É o atributo de uma cor pela qual damos a ela um nome como "vermelho" ou "azul". "Valor" é outra palavra para "leve", o atributo de uma cor que a torna equivalente a algum tom de cinza entre preto e branco. Saturação é uma medida de como uma cor diferente aparece de um cinza da mesma [luz.](b.md) Saturação zero não indica matiz, apenas escala cinza. O espaço de cores HSV é normalizado.

![hsv color space](images/hsvline.png)

A figura anterior mostra um desenho de linha do espaço HSV na forma de um hexacone. Cada uma de suas seções cruzadas é um hexágono. Nos vértices de cada seção cruzada estão as cores vermelho, amarelo, verde, ciano, azul e magenta. Uma cor no espaço HSV é especificada informando um ângulo de matiz, o nível de chroma e o nível de luz. Um ângulo de matiz de zero é vermelho. O ângulo de matiz aumenta em uma direção no sentido anti-horário. As cores complementares têm 180 de diferença.

Espaços de cores HSV podem ser dependentes do dispositivo ou independentes do dispositivo.
