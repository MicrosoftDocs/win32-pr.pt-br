# <a name="r"></a>R

## <a name="rendering-intent"></a>tentativa de renderização

Abordagens para renderizar a cor de um espaço de cor para outro. As tentativas de renderização usadas em perfis ICC são definidas pelo consórcio de cor internacional (ICC).

## <a name="rgb"></a>RGB

O espaço de cores vermelho, verde e azul.

## <a name="rgb-triple"></a>Triplo RGB

Uma quantidade que contém valores de espaço de cores RGB.
