# <a name="using-color-management-on-the-internet"></a>Uso de gerenciamento de cores na Internet

Imagens coloridas e gráficos são uma maneira cada vez mais importante de comunicar informações para usuários da Internet. As informações de conversão de cores podem ser inseridas em alguns formatos de arquivo. As técnicas de incorporação usadas atualmente não são compactos. As considerações sobre largura de banda da Internet geralmente tornam-as impraticáveis. As considerações da Internet incluem os seguintes tópicos:

-   [sRGB: um espaço de cores padrão](srgb--a-standard-color-space.md)
-   [Padrões WCS 1,0 para o espaço de cor de entrada e o perfil de saída](srgb--a-standard-color-space.md)
-   [sRGB e perfis inseridos](srgb--a-standard-color-space.md)
