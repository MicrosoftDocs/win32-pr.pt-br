# <a name="p"></a>P

## <a name="primary-colors"></a>cores primárias

As três cores essenciais em um espaço de cores que podem ser misturadas para produzir todas as outras cores.

## <a name="profile-connection-space-pcs"></a>Espaço de Conexão de Perfil (PCS)

Um espaço de cores independente do dispositivo usado para conversões de cores e correspondência de cores.
