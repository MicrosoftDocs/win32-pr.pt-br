# <a name="p"></a>P

## <a name="primary-colors"></a>cores primárias

As três cores essenciais em um espaço de cores que podem ser misturadas para produzir todas as outras cores.

## <a name="profile-connection-space-pcs"></a>Espaço de conexão do perfil (PCS)

Um espaço de cores independente de dispositivo usado para conversões de cor e correspondência de cores.
