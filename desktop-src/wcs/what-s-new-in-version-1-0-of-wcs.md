# <a name="whats-new-in-version-10-of-wcs"></a>Novidades da versão 1.0 do WCS

A versão 1.0 do ICM (Gerenciamento de Cores de Imagem) foi entregue no Microsoft Windows 95 e fornece recursos básicos de gerenciamento de cores Windows contextos de dispositivo.

ICM versão 2.0 está incluído no Windows 98, no Windows Edition daNium, no Windows 2000 e no Windows XP e inclui uma variedade de novas funções que implementam o gerenciamento de cores fora dos contextos do dispositivo. Essas novas funções são adequadas para requisitos de gerenciamento de cores mais exigentes e dão aos aplicativos maior controle sobre o processo de gerenciamento de cores.

A funcionalidade adicional está incluída na versão Windows Vista do Microsoft Windows.

-   escolha do fluxo de trabalho de CMM/Perfil estático/monolítico e dinâmico/modular
    -   estático fornecido por HeidelbergerDruckmaschinen no Win98 ICM2, dinâmico fornecido pela Canon Inc. usando a tecnologia Kyuianos da Canon
    -   dispositivo de linha de base, aparência de cor e perfis de modelo de mapeamento de jogos,
    -   suporte à geração de preto,
    -   suporte à preservação de preto,
    -   Suporte a intervalo dinâmico alto,
    -   ponto flutuante/maior que 16 bits por precisão de canal,
    -   suporte ao perfil de espaço de trabalho de gama larga padrão,
    -   flexibilidade para controlar as características internas do dispositivo de objetivo separadas das características entre dispositivos,
    -   capacidade de controlar o algoritmo de mapeamento de jogo único entre dois dispositivos,
    -   suporte para mapeamento de primários de dispositivo de origem para primários de destino em um fluxo de trabalho gerenciado por cores,
-   escolha de formatos de perfil de cor de texto e binário
    -   binário em conformidade com o formato de perfil baseado em TIFF ICC e texto usando esquemas XML
-   Suporte ao ICC versão 4,
-   painel de controle de gerenciamento de cores central,
-   compatibilidade herdada mantida para aplicativos e dispositivos,
-   extensibilidade por meio de modelos de dispositivo de plug-in de terceiros e modelos de mapeamento de jogos,
-   suporte para mapeamento avançado de dispositivo/gamut habilitando a comunicação direta entre plug-ins de DM e GMM,
-   O CMM estático (ICM2) agora assume como padrão a intenção Colorimetric Relativa em vez da intenção Colorimétrica Absoluta.
