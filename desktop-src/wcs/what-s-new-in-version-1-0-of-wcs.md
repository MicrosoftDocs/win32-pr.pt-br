# <a name="whats-new-in-version-10-of-wcs"></a>Novidades da versão 1.0 do WCS

A versão 1,0 do ICM (Image Color Management) foi fornecida no Microsoft Windows 95 e fornece recursos básicos de gerenciamento de cores em contextos de dispositivo do Windows.

O ICM versão 2,0 está incluído no Windows 98, no Windows Millennium Edition, no Windows 2000 e no Windows XP e inclui uma variedade de novas funções que implementam o gerenciamento de cores fora dos contextos de dispositivo. Essas novas funções são adequadas para requisitos de gerenciamento de cores mais exigentes e oferecem aos aplicativos mais controle sobre o processo de gerenciamento de cores.

A funcionalidade adicional está incluída na versão Windows Vista do Microsoft Windows.

-   opção de fluxo de trabalho estático/monolítico e dinâmico/modular CMM/Profile
    -   estático fornecido por HeidelbergerDruckmaschinen no Win98 ICM2, dinâmico fornecido pela Canon Inc. usando a tecnologia Kyuanos da Canon
    -   dispositivos de linha de base, aparência de cores e perfis de modelo de mapeamento de gamut,
    -   suporte à geração de preto,
    -   suporte à preservação de preto,
    -   Alto suporte a intervalo dinâmico,
    -   ponto flutuante/maior que 16 bits por precisão de canal,
    -   suporte a perfil de espaço de trabalho de gamut largo padrão,
    -   flexibilidade para controlar as características do objetivo intra-Device, separadas das características entre dispositivos,
    -   capacidade de controlar o algoritmo de mapeamento de gamut único entre dois dispositivos,
    -   suporte para mapeamento de primários de dispositivo de origem para primários de destino em um fluxo de trabalho gerenciado por cores,
-   opções de formatos de perfil de cor de texto e binário
    -   a conformidade binária com o formato de perfil baseado em TIFF ICC e o texto usando esquemas XML
-   Suporte do ICC versão 4,
-   painel de controle do gerenciamento de cores central,
-   compatibilidade herdada mantida para aplicativos e dispositivos,
-   extensibilidade por meio de modelos de dispositivo de plug-in de terceiros e modelos de mapeamento de gamut,
-   suporte para mapeamento avançado de dispositivo/gama habilitando a comunicação direta entre plug-ins DM e GMM,
-   o estático (ICM2) CMM agora usa como padrão a intenção colorimétrico relativa em vez de uma intenção colorimétrico absoluta.
