# <a name="c"></a>C

## <a name="camp"></a>CAMP

O formato de perfil do modelo de aparência de cor é um formato de arquivo baseado em XML que contém as condições de exibição necessárias para descrever a relação entre os valores de CIEXYZ do DMP e os valores de CIEJab.

## <a name="cmyk"></a>CMYK

O espaço de cores ciano, magenta, amarelo e preto. Geralmente, ele é implementado em impressoras.

## <a name="colorant"></a>colorant

Algo, especialmente um Pigment ou uma tinta, que colore ou modifica o matiz de outra coisa.

## <a name="color-channel"></a>canal de cores

Um componente de um espaço de cores. Por exemplo, um espaço de cores RGB tem canais de cor vermelho, verde e azul.

## <a name="color-conversion"></a>conversão de cores

O processo de converter cores de um espaço de cor para outro.

## <a name="color-management-module-cmm"></a>Módulo de gerenciamento de cores (CMM)

Um módulo de código que usa perfis de dispositivo para executar a conversão de cores e o mapeamento de cores.

## <a name="color-mapping"></a>mapeamento de cores

Consulte correspondência de cores.

## <a name="color-matching"></a>correspondência de cores

Correspondência de uma cor convertida com sua cor mais próxima no espaço de cores de destino.

## <a name="color-model"></a>modelo de cor

Consulte espaço de cores.

## <a name="color-space"></a>espaço de cores

Um mapeamento de componentes de cor em um sistema de coordenadas geométrico em três dimensões.
