# <a name="s"></a>S

## <a name="saturation-chroma"></a>Saturação (croma)

Geralmente croma é a quantidade de cinza que é misturada em um matiz. Zero croma produz um tom de cinza. O croma completo produz um matiz puro.

## <a name="shade"></a>tonalidade

Uma tonalidade de cor é feita adicionando preto ao seu matiz.

## <a name="soft-proofing"></a>prova de software

Visualização ou prova de uma imagem em uma tela de vídeo.

## <a name="srgb"></a>sRGB

Um espaço de cores RGB padrão internacional proposto pela Microsoft e Hewlett-Packard que é independente de dispositivo. Esse padrão foi aprovado como um padrão internacional formal dos sistemas de multimídia e da medição EquipmentColour e ManagementPart 2-1: cor ManagementDefault o espaço de cores RGB. Ele está disponível diretamente do IEC em https://www.iec.ch/ .

## <a name="subtractive-primary-colors"></a>cores primárias subtraíis

Cores que podem ser subtraídas de branco para misturar uma nova cor. As cores primárias subtraíis são ciano, amarelo e magenta.
