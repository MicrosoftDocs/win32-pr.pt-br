# <a name="color-space-constants"></a>Constantes de espaço de cores

Um valor de assinatura que indica o espaço de cores no qual os dados do perfil são definidos. Pode ser qualquer um dos valores a seguir.



| Classe de perfil     | Assinatura         |
|-------------------|-------------------|
| XYZ               | ESPAÇO \_ XYZ        |
| Laboratório               | Laboratório de espaço \_        |
| Luv               | Luv de espaço \_        |
| YCbCr             | YCbCr de espaço \_      |
| Yxy               | YXY de espaço \_        |
| RGB               | ESPAÇO \_ RGB        |
| Escala de cinza        | ESPAÇO \_ cinza       |
| HSV               | HSV de espaço \_        |
| HLS               | HLS de espaço \_        |
| CMYK              | ESPAÇO \_ CMYK       |
| CMY               | CMY de espaço \_        |
| Canal genérico 2 | Canal de espaço \_ 2 \_ |
| Canal genérico 3 | Canal de espaço \_ 3 \_ |
| 4 canais genéricos | ESPAÇO \_ 4 \_ canal |
| Canal genérico 5 | Canal de espaço \_ 5 \_ |
| Canal genérico 6 | ESPAÇO \_ 6 \_ canal |
| Canal genérico 7 | \_Canal 7 de espaço \_ |
| Canal genérico 8 | \_Canal 8 de espaço \_ |
