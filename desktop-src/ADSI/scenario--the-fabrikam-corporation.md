# <a name="scenario-the-fabrikam-corporation"></a>Cenário: a Fabrikam Corporation

A empresa da Fabrikam atualizou seu domínio do Windows NT 4,0 para o Windows 2000. Durante a instalação, Joe Worden, o administrador de rede, foi solicitado a fornecer um nome DNS de domínio. Joe digitou "fabrikam.com". Os usuários, grupos e computadores existentes são migrados para Active Directory no novo domínio: fabrikam.com.
