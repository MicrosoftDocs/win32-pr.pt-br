# <a name="winnt-schema"></a>Esquema WinNT

O esquema WinNT define o conjunto de objetos ADSI e sua hierarquia de classes para uma rede Windows NT 4.0. Os [objetos ADSI do WinNT](adsi-objects-of-winnt.md) incluem Usuário, Computador, Domínio, Grupo, Recurso, Sessão, Serviço e outros. A seção Hierarquia de Classes de Objeto do [WinNT](winnt-object-class-hierarchy.md) a seguir discute as relações hierárquicas entre esses objetos.

Cada objeto implementa um conjunto de interfaces ADSI. No entanto, de uma interface implementada, nem todas as propriedades e métodos têm suporte. [As Propriedades Obrigatórias e](winnt-schemaampaposs-mandatory-and-optional-properties.md) Opcionais do Esquema WinNT listam as propriedades com suporte nas classes de objeto WinNT.
