# <a name="winnt-schema"></a>Esquema WinNT

O esquema WinNT define o conjunto de objetos ADSI e sua hierarquia de classes para uma rede do Windows NT 4,0. Os [objetos ADSI do Winnt](adsi-objects-of-winnt.md) incluem usuário, computador, domínio, grupo, recurso, sessão, serviço e outros. A seção [hierarquia da classe de objeto WinNT](winnt-object-class-hierarchy.md) a seguir discute as relações hierárquicas entre esses objetos.

Cada objeto implementa um conjunto de interfaces ADSI. No entanto, de uma interface implementada, nem todas as propriedades e métodos têm suporte. [As propriedades obrigatórias e opcionais do esquema WinNT](winnt-schemaampaposs-mandatory-and-optional-properties.md) listam as propriedades com suporte nas classes de objeto do Winnt.
