# <a name="example-code-for-using-idirectorysearch"></a>Código de exemplo para usar IDirectorySearch

O SDK (Kit de Desenvolvimento de Software de Plataforma) inclui um tópico de exemplo de código, DsSrch, que fornece um aplicativo de prompt de comando simples que usa a interface [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) para consultar um serviço de diretório. O exemplo de código completo está incluído no SDK da Plataforma no diretório \\ \\ DsSrch Geral de Exemplos de ADSI de \\ \\ \\ Exemplos do NetDs.
