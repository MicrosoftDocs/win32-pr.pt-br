# <a name="example-code-for-using-idirectorysearch"></a>Exemplo de código para usar IDirectorySearch

O SDK (Software Development Kit) da plataforma inclui um tópico de exemplo de código, DsSrch, que fornece um aplicativo simples de prompt de comando que usa a interface [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) para consultar um serviço de diretório. O exemplo de código completo está incluído no SDK da plataforma nos exemplos \\ NetDs \\ ADSI \\ Samples \\ geral \\ DsSrch diretório.
