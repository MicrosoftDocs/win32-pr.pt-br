# <a name="active-directory-service-interfaces-scripting"></a>Script de interfaces de serviço Active Directory

O Microsoft Active Directory Service Interfaces (ADSI), quando combinado com uma linguagem de script, como o Microsoft Visual Basic Scripting Edition ou o software de desenvolvimento Microsoft JScript, fornecem um administrador de sistema com ferramentas poderosas para gerenciar usuários, serviços, filas de impressão e outras partes de uma rede de computadores que executam sistemas operacionais Microsoft Windows NT ou Windows 2000. Os scripts que usam o modelo de objeto ADSI podem substituir os esforços manuais repetitivos que normalmente seriam realizados usando ferramentas administrativas ou o console de gerenciamento Microsoft. O script ADSI também pode automatizar tarefas regulares de manutenção do sistema para uma operação sem intervenção e fornece uma interface fácil para escrever ferramentas de gerenciamento de sistemas baseadas na Web usando Active Server páginas.

Para obter mais informações sobre scripts com ADSI, com tutoriais de script, exemplos e ferramentas, consulte:

-   [Instruções elementares sobre script ADSI](https://www.microsoft.com/resources/documentation/windows/2000/server/scriptguide/sas_ads_overview.mspx) no site do Microsoft TechNet
-   [Script Center](https://www.microsoft.com/technet/community/scriptcenter/default.mspx) no site do Microsoft TechNet
-   [Erros e interceptação de erros](errors-and-error-trapping.md)
