# <a name="adsi-extensions"></a>Extensões ADSI

Uma extensão ADSI é um objeto COM especial que permite que um desenvolvedor estenda um objeto ADSI. Cada extensão é associada a uma classe especificada no diretório. Com esse modelo de extensão, os desenvolvedores podem adicionar métodos para fornecer um significado mais dinâmico a um objeto. Em um modelo de programação de diretório tradicional, um objeto é acessado obtendo e configurando os atributos de objeto. Usando extensões ADSI, você pode adicionar mais recursos a um objeto de diretório.
