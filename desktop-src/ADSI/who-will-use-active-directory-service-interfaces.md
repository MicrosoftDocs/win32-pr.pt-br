# <a name="who-will-use-active-directory-service-interfaces"></a>Who Usará Active Directory interfaces de serviço?

Os administradores de rede usarão a ADSI para automatizar tarefas administrativas comuns, como adicionar usuários e grupos, gerenciar impressoras e definir permissões em recursos de rede.

ISVs (fornecedores independentes de software) e desenvolvedores de usuários finais usarão o ADSI para "diretório habilitado" em seus produtos e aplicativos. Os serviços podem se publicar em um diretório, os clientes podem usar o diretório para localizar os serviços, e ambos podem usar o diretório para localizar e manipular outros objetos de interesse. Como Active Directory interfaces de serviço são independentes dos serviços de diretório subjacentes, os produtos e aplicativos habilitados para diretório funcionarão com êxito em vários ambientes de rede e de diretório.
