# <a name="propertycpp"></a>Propriedade. Cpp

No componente do provedor de exemplo, exemplos de código para armazenar tipos de dados diferentes em tipos de dados **VARIANT** para que possam ser salvos como propriedades de um objeto estão em property.cpp. As rotinas com suporte são listadas na tabela a seguir.



| Rotina                                                           | Descrição                  |
|-------------------------------------------------------------------|------------------------------|
| **get \_ Propriedade \_ BSTR** / **put \_ \_ BSTR Property**                   | Para propriedades de cadeia de caracteres.       |
| **get \_ Propriedade \_ LONG** / **put LONG \_ \_ Property**                   | Para propriedades de inteiro longo. |
| **get \_ Propriedade \_ DATE** / **put DATE \_ \_ Property**                   | Para propriedades de formato de data.  |
| **get \_ Propriedade \_ VARIANT \_ BOOL** / **put VARIANT \_ \_ BOOL \_ Property** | Para propriedades boolianas.      |
| **get \_ Propriedade \_ VARIANT** / **put VARIANT \_ \_ Property**             | Para **propriedades VARIANT.**  |
