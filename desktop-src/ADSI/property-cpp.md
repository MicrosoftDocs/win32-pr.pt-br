# <a name="propertycpp"></a>Propriedade. CPP

No exemplo de componente do provedor, exemplos de código para armazenar diferentes tipos de dados em tipos de dados **Variant** para que eles possam ser salvos como propriedades de um objeto estão em Property. cpp. As rotinas com suporte são listadas na tabela a seguir.



| Rotina                                                           | Descrição                  |
|-------------------------------------------------------------------|------------------------------|
| **obter \_ Propriedade \_ BSTR** / **Put \_ \_ Propriedade BSTR**                   | Para propriedades de cadeia de caracteres.       |
| **obter \_ \_** / **Propriedade longa Put \_ longa \_** de propriedade                   | Para propriedades de inteiro longo. |
| **obter \_ \_** / **\_ \_ Propriedade** data de Put da Propriedade Date                   | Para propriedades de formato de data.  |
| **obter \_ \_ \_ Propriedade bool variante** / **Put \_ variante \_ booliana \_** Propriedade | Para propriedades booleanas.      |
| **obter \_ Propriedade \_ Variant** / **Put \_ Variant \_ Property**             | Para propriedades de **variante** .  |
