# <a name="cschobjcpp"></a>CSCHOBJ. CPP

No exemplo de componente do provedor, um exemplo de código, que gerencia o tempo de vida dos objetos de esquema, está em cschobj. cpp. Os métodos com suporte são listados na tabela a seguir.



| Método                                                   | Descrição                                    |
|----------------------------------------------------------|------------------------------------------------|
| **CSampleDSSchema:: CreateSchema**                        | Construtor padrão.                          |
| **CSampleDSSchema:: ~ CSampleDSSchema**                    | Destruidor padrão.                           |
| Métodos [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) padrão.                   |                                                |
| Métodos do [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) padrão. |                                                |
| **CSampleDSSchema:: QueryInterface**                      | Verifique a ID de interface fornecida neste objeto.  |
| **CSampleDSSchema::AllocateSchema**                      | Crie um objeto de esquema e carregue seus dados de tipo. |
