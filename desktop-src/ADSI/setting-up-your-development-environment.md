# <a name="setting-up-your-development-environment"></a>Configurando seu ambiente de desenvolvimento

Você pode acessar o ADSI de várias plataformas de desenvolvimento diferentes, como scripts, ASP, os sistemas de desenvolvimento Microsoft Visual Basic ou Visual C++, ou qualquer outra linguagem compatível com a automação.

Para obter mais informações sobre como configurar um ambiente de desenvolvimento e criar aplicativos básicos da ADSI, consulte:

-   [Introdução com scripts para ADSI](getting-started-with-scripting-for-adsi.md)
-   [Introdução com ASP para ADSI](getting-started-with-asp-for-adsi.md)
-   [Configurando Visual Basic 6,0 para o desenvolvimento de ADSI](setting-up-visual-basic-for-adsi-development.md)
-   [Configurando Visual C++ 6,0 para o desenvolvimento de ADSI](setting-up-c---for-adsi-development.md)
