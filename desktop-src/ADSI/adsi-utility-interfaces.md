# <a name="adsi-utility-interfaces"></a>Interfaces do Utilitário ADSI

A ADSI fornece várias interfaces de utilitário para objetos ADSI que podem ajudar a acessar e manter o Active Directory. Essas interfaces são:

-   [**IADsDeleteOps**](iadsdeleteops-interface.md)
-   [**IADsObjectOptions**](iadsobjectoptions-interface.md)
-   [**IADsPathname**](iadspathname-interface.md)
-   [**IADsNameTranslate**](iadsnametranslate-interface.md)
