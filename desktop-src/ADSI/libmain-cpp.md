# <a name="libmaincpp"></a>LIBMAIN. CPP

No componente do provedor de exemplo, um exemplo de código do ponto de entrada padrão da DLL para Adssmp.dll está em LibMain. cpp. As rotinas com suporte são listadas na tabela a seguir.



| Item                                  | Descrição                                                              |
|---------------------------------------|--------------------------------------------------------------------------|
| **DllGetClassObject**<br/>      | Ponto de entrada de DLL padrão para localizar fábricas de classe.<br/>        |
| **DllCanUnloadNow**<br/>        | Ponto de entrada de DLL padrão para determinar se a DLL pode ser descarregada.<br/> |
| **LibMain**<br/>                | Ponto de entrada de inicialização de DLL padrão.<br/>                      |
| **IsCompatibleOleVersion**<br/> | Verifique a compatibilidade com o tempo de execução do OLE.<br/>                   |
| **DllMain**<br/>                | Ponto de entrada para a maioria das versões do Windows 2000 ou do Windows NT.<br/>     |
