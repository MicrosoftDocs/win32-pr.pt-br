Dim usr As IADsUser

<span data-ttu-id="02fe3-366">O exemplo de código a seguir mostra como associar a um objeto de conta de usuário e recuperar o nome completo do usuário.</span><span class="sxs-lookup"><span data-stu-id="02fe3-366">The following code example shows how to bind to a user account object and retrieve the full name of the user.</span></span>


```C++
IADsUser *GetUserObject(LPWSTR uPath)
{
    IADsUser *pUser;
    HRESULT hr = ADsGetObject(uPath,IID_IADsUser,(void**)&pUser);
    if (FAILED(hr)) {return NULL;}
    BSTR bstr;
    hr = pUser->get_FullName(&bstr);
    printf("User: %S\n", bstr);
    SysFreeString(bstr);
    return pUser;
}
```
