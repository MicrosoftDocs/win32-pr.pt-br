# <a name="interface-implementation-of-system-providers"></a>Implementação de interface de provedores de sistema

A ADSI incorpora vários provedores de serviços de diretório, ou seja, LDAP e WinNT. Provedores diferentes podem optar por implementar ou não implementar uma interface. Provedores de serviço de diretório diferentes dão suporte a propriedades e métodos diferentes. Para obter mais informações sobre o suporte ao provedor de serviços de diretório específico, consulte [ADSI System Providers](adsi-system-providers.md).
