# <a name="errors-and-error-trapping"></a>Erros e interceptação de erros

A ADSI gera uma ampla variedade de erros, do ADSI específico ao COM, relacionados a erros Win32 padrão. Esta seção do tutorial fornece estratégias para lidar com erros em scripts ADSI e ajuda você a tomar algum sentido dos diferentes códigos de erro. Os seguintes tópicos são abordados:

-   [Como interceptar erros ADSI](how-to-trap-adsi-errors.md)
-   [Erros comuns](common-errors.md)
