A ADSI lança uma ampla variedade de erros, desde específicos da ADSI até os erros padrão do Win32 relacionados ao COMI. Esta seção do tutorial fornece estratégias para lidar com erros em scripts ADSI e ajuda você a entender os diferentes códigos de erro. Os seguintes tópicos são abordados:

-   [Como interceptar erros ADSI](how-to-trap-adsi-errors.md)
-   [Erros comuns](common-errors.md)
