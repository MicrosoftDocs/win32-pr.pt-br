# <a name="packcpp"></a>Pacotes. CPP

No exemplo de componente do provedor, um exemplo de código de empacotamento e desempacotamento de tipos de dados em VARIAntes é encontrado em Pack. cpp. As funções a seguir são usadas quando as propriedades são carregadas e removidas do cache de propriedades:

-   **PackStringinVariant**
-   **UnpackStringinVariant**
-   **PackLONGinVariant**
-   **UnpackLONGfromVariant**
-   **PackDATEinVariant**
-   **UnpackDATEinVariant**
-   **PackVARIANT \_ BOOLinVariant**
-   **UnpackVARIANT \_ BOOLfromVariant**
-   **PackVARIANTinVariant**
-   **UnpackVARIANTfromVariant**
