# <a name="active-directory-operational-attributes"></a>Active Directory atributos operacionais

Os atributos operacionais são propriedades que são armazenadas e usadas pelo diretório para fins administrativos. Por exemplo, você pode manter um atributo operacional para registrar a data e a hora quando outro atributo é modificado. Atributos operacionais devem ser recuperados explicitamente com uma chamada para [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex).
