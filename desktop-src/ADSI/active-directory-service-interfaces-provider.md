# <a name="active-directory-service-interfaces-provider"></a>Provedor de interfaces de serviço Active Directory

Um provedor ADSI contém a implementação de objetos ADSI e objetos dependentes para um namespace específico. A ilustração a seguir mostra que os clientes estão preocupados apenas com a obtenção e o uso de interfaces em um objeto, e não com os detalhes de onde e como o software de um objeto é implementado.
