# <a name="commoncpp"></a>Common. CPP

No exemplo de componente do provedor, um exemplo de código comum a todos os objetos genéricos é encontrado em Common. cpp. Os objetos herdam esses métodos e propriedades por meio de regras de herança do C++. Esses métodos incluem a criação de cadeias de caracteres ADsPath, métodos de dados de tipo e outras rotinas de manutenção.
