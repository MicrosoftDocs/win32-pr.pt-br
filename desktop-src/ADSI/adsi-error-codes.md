# <a name="adsi-error-codes"></a>Códigos de erro ADSI

Todos os códigos de erro ADSI são retornados como valores de **HRESULT** com. Eles podem ser agrupados nos quatro tipos a seguir:

-   [Códigos de erro COM genéricos](generic-com-error-codes.md)
-   [Códigos de erro ADSI genéricos](generic-adsi-error-codes.md)
-   [Códigos de erro do Win32 para ADSI](win32-error-codes-for-adsi.md)
-   [Códigos de erro LDAP para ADSI](ldap-error-codes-for-adsi.md)

Além disso, determinadas interfaces fornecem informações de erro adicionais, conhecidas como mensagens de erro estendidas da ADSI, que podem ser obtidas usando a função [**ADsGetLastError**](/windows/desktop/api/Adshlp/nf-adshlp-adsgetlasterror) .

Por fim, o [código de exemplo](code-example-for-working-with-adsi-error-messages.md) C++ mostrado nesta seção mostra como trabalhar com mensagens de erro ADSI.
