# <a name="provider-minimum-requirements"></a>Requisitos mínimos do provedor

Para escrever um provedor ADSI, implemente as interfaces de serviço Active Directory conforme descrito nas seções a seguir.

-   A [implementação básica](core-implementation.md) lista brevemente os elementos de software que um provedor ADSI deve dar suporte.
-   A [implementação opcional](optional-implementation.md) lista brevemente os elementos de software para os quais um provedor ADSI é incentivado a dar suporte.
-   [As propriedades personalizadas](custom-properties.md) são as propriedades e interfaces expostas por cada provedor para fornecer programação dos recursos nativos do serviço de diretório.
-   As [interfaces duplas](dual-interfaces.md) descrevem a Convenção ADSI para fornecer interfaces que podem ser acessadas por controladores de automação e não compatíveis com a automação.
