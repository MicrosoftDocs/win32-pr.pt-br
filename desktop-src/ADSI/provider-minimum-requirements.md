# <a name="provider-minimum-requirements"></a>Requisitos mínimos do provedor

Para escrever um provedor ADSI, implemente as Interfaces de Serviço do Active Directory, conforme descrito nas seções a seguir.

-   [A Implementação principal](core-implementation.md) lista brevemente os elementos de software que um provedor ADSI deve dar suporte.
-   [A Implementação Opcional](optional-implementation.md) lista brevemente os elementos de software aos que um provedor ADSI é fortemente incentivado a dar suporte.
-   [Propriedades personalizadas](custom-properties.md) são as propriedades e interfaces expostas por cada provedor para fornecer programação dos recursos nativos do serviço de diretório.
-   [Interfaces duplas](dual-interfaces.md) descrevem a convenção ADSI para fornecer interfaces acessíveis tanto para a Automação quanto para controladores em conformidade com a Automação.
