# <a name="using-adsi-with-exchange"></a>Usando ADSI com Exchange

Por Exchange 2000, as informações para usar a ADSI com Exchange estão contidas no SDK Exchange 2000. Para obter mais informações, consulte [Tarefas de gerenciamento para ADSI](/previous-versions/office/developer/exchange-server-2003/aa125368(v=exchg.65)).

As seguintes tarefas podem ser encontradas nesta seção:

-   Criando uma URL de usuário
-   Criando caminhos do Active Directory
-   Enumerando Exchange 2.000 servidores com ADSI
-   Manipulando atributos de extensão com ADSI
-   Adicionando/removendo um objeto de gerente com ADSI
-   Enumerando Exchange propriedades de objeto com ADSI/ADO
-   Recuperando um nome herdado Exchange diferenciado com ADSI
-   Localizar o nome Exchange organização usando a ADSI
-   Localizar Exchange de endereços com ADSI
-   Criando uma lista de distribuição usando CDOEXM e ADSI
-   Definindo a restrição de mensagem em um servidor virtual SMTP usando ADSI

Por Exchange 5.5, a referência adsi e o uso de informações podem ser encontrados no guia de Exchange [ADSI.](/previous-versions/office/developer/exchange-server-2007/aa579394(v=exchg.80))

As seguintes tarefas podem ser encontradas nesta seção:

-   Exibindo e modificando o Exchange Server esquema
-   Exibindo as propriedades brutas de um Exchange objeto
-   Criando uma caixa de Exchange de correio
-   Definindo o descritor de segurança em uma caixa de Exchange de correio
-   Manipulação de sids e descritores de segurança
-   Excluindo um objeto de caixa de correio
-   Criando um destinatário personalizado
-   Criando um contêiner de destinatários
-   Obter o nome da organização e do site de um servidor
-   Listando a Exchange do servidor de todos os servidores na organização
-   Localizar o servidor principal de uma caixa de correio
-   Recuperando endereços de email
-   Acessando entradas ocultas ou excluídas
-   Recuperando as alterações feitas no serviço de diretório
-   Criando uma lista de distribuição com os resultados de uma consulta
-   Obter ou modificar o tamanho da mensagem
-   Usando códigos de erro LDAP para solucionar problemas
