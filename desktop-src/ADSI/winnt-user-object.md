# <a name="winnt-user-object"></a>Objeto de usuário do WinNT

o objeto de usuário do WinNT representa uma conta de usuário em um domínio Windows NT 4,0. O objeto exibe recursos especiais. Em uma instância, ele não dá suporte a todos os métodos de propriedade da interface [**IADsUser**](/windows/desktop/api/Iads/nn-iads-iadsuser) . Em uma segunda instância, ele dá suporte a algumas propriedades personalizadas que podem ser acessadas somente com o método [**IADs. Get**](/windows/desktop/api/Iads/nf-iads-iads-get) ou [**IADs. put**](/windows/desktop/api/Iads/nf-iads-iads-put) .

Para obter mais informações sobre objetos de usuário do WinNT, consulte:

-   [Propriedades IADsUser sem suporte](unsupported-iadsuser-properties.md)
-   [Propriedades do usuário personalizado do WinNT](winnt-custom-user-properties.md)
-   [Exemplos de gerenciamento de conta de usuário do WinNT](winnt-user-account-management-examples.md)
