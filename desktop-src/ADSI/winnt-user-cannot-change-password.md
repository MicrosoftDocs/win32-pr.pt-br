# <a name="user-cannot-change-password-winnt-provider"></a>O usuário não pode alterar a senha (provedor WinNT)

A capacidade de um usuário alterar sua senha é uma permissão que pode ser concedida ou negada. Para obter mais informações sobre como ler e modificar essa permissão programaticamente usando o provedor WinNT, consulte:

-   [A leitura do usuário não pode alterar a senha (provedor WinNT)](reading-user-cannot-change-password-winnt-provider.md)
-   [Modificar o usuário não pode alterar a senha (provedor WinNT)](modifying-user-cannot-change-password-winnt-provider.md)

O atributo **UserFlags** do objeto user está disponível somente ao usar o provedor Winnt.

Também é possível ler e modificar essa permissão usando o provedor LDAP. Para obter mais informações, consulte o [usuário não pode alterar a senha (provedor LDAP)](user-cannot-change-password.md).
