# <a name="winrm-client-shell-api"></a>API do shell do cliente WinRM

A API (interface de programação de aplicativos) do WinRM (Gerenciamento Remoto) do Windows fornece funcionalidade para criar e gerenciar shells e operações de shell, comandos e fluxos de dados em computadores remotos.

A seção referência da API do Shell do Cliente WinRM contém informações detalhadas sobre os seguintes elementos de API:

-   [Enumerações de API do Shell do Cliente](client-shell-api-enumerations.md)
