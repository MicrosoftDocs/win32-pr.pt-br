# <a name="winrm-client-shell-api"></a>API do shell do cliente WinRM

A API (interface de programação de aplicativo) do Shell de cliente do Gerenciamento Remoto do Windows (WinRM) fornece funcionalidade para criar e gerenciar shells e operações de Shell, comandos e fluxos de dados em computadores remotos.

A seção de referência da API do shell do cliente WinRM contém informações detalhadas sobre os seguintes elementos da API:

-   [Funções adicionais da API do shell do cliente](client-shell-api-additional-functions.md)
-   [Enumerações da API do shell do cliente](client-shell-api-enumerations.md)
