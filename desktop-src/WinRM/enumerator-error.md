# <a name="enumeratorerror-property"></a>Propriedade Enumerator. Error

Obtém uma representação XML de informações de erro adicionais.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
Enumerator.Error As String
```



## <a name="property-value"></a>Valor da propriedade

A representação XML de informações de erro adicionais.
