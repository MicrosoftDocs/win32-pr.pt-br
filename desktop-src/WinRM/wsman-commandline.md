# <a name="wsmancommandline-property"></a>Propriedade WSMan. CommandLine

Obtém a linha de comando do processo que carrega o componente de automação.

Esta propriedade é somente para leitura.

## <a name="syntax"></a>Sintaxe


```VB
WSMan.CommandLine As BSTR
```



## <a name="property-value"></a>Valor da propriedade

Cadeia de caracteres que contém a linha de comando.
