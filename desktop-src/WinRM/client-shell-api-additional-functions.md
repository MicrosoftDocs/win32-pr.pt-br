# <a name="client-shell-api-additional-functions"></a>Funções adicionais da API do shell do cliente

A tabela a seguir fornece uma visão geral das funções não principais para a API (interface de programação de aplicativo) do Shell de cliente do Gerenciamento Remoto do Windows (WinRM).



| Funções adicionais                                                   | Description                       |
|------------------------------------------------------------------------|-----------------------------------|
| [**WSManGetErrorMessage**](/windows/desktop/api/Wsman/nf-wsman-wsmangeterrormessage)                   | Obtém mensagens de erro.              |
| [**WSManGetSessionOptionAsDword**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasdword)   | Obtém as opções de sessão como um DWORD.  |
| [**WSManGetSessionOptionAsString**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasstring) | Obtém as opções de sessão como uma cadeia de caracteres. |
| [**WSManSetSessionOption**](/windows/desktop/api/Wsman/nf-wsman-wsmansetsessionoption)                 | Define opções de sessão.             |
