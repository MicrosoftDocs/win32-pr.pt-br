# <a name="client-shell-api-additional-functions"></a>Funções adicionais da API do Shell do Cliente

A tabela a seguir fornece uma visão geral das funções não principais para a API (interface de programação de aplicativos) do WinRM (Gerenciamento Remoto) Windows .



| Funções adicionais                                                   | Descrição                       |
|------------------------------------------------------------------------|-----------------------------------|
| [**WSManGetErrorMessage**](/windows/desktop/api/Wsman/nf-wsman-wsmangeterrormessage)                   | Obtém mensagens de erro.              |
| [**WSManGetSessionOptionAsDword**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasdword)   | Obtém opções de sessão como um DWORD.  |
| [**WSManGetSessionOptionAsString**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasstring) | Obtém opções de sessão como uma cadeia de caracteres. |
| [**WSManSetSessionOption**](/windows/desktop/api/Wsman/nf-wsman-wsmansetsessionoption)                 | Define opções de sessão.             |
