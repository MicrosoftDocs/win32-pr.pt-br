# <a name="underlying-protocols-for-ipv6-winsock-applications"></a>Protocolos subjacentes para aplicativos Winsock IPv6

Alguns protocolos dos quais seu aplicativo depende para determinados serviços, como RPC (chamada de procedimento remoto), podem ter funções e estruturas dependentes da versão de IP que exigem que você modifique seu aplicativo em termos de uso.

Parte do processo para modificar seu código para dar suporte a IPv6 deve incluir determinar se o uso dos protocolos subjacentes (da perspectiva da pilha, esses protocolos são considerados protocolos de camada mais alta) requer modificação para usar corretamente o IPv6.
