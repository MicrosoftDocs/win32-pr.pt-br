# <a name="winsock-enumerations"></a>Enumerações do Winsock

A lista a seguir fornece descrições concisas de cada enumeração do Winsock. Para obter informações adicionais sobre qualquer enumeração, clique no nome da enumeração.

| Enumeração | Descrição |
|-|-|
| [**\_status do \_ gatilho do canal de controle \_**](/windows/win32/api/mstcpip/ne-mstcpip-control_channel_trigger_status) | Especifica o status de uma consulta para a configuração de transporte de **capacidade de \_ notificação em tempo \_ \_ real** para um soquete TCP que é usado com [**ControlChannelTrigger**](/uwp/api/Windows.Networking.Sockets.ControlChannelTrigger) para receber notificações de rede de segundo plano em um aplicativo Windows Runtime. |
| [**\_método de avanço eWINDOW \_**](/windows/win32/api/wsrm/ne-wsrm-ewindow_advance_method) | Especifica o modo de avanço de janela usado para multicast confiável. |
| [**GARANTIA**](./guarantee-2.md) | Não se usa mais. |
| [**\_tipo de modo multicast \_**](/windows/win32/api/ws2ipdef/ne-ws2ipdef-multicast_mode_type) | Especifica o modo de filtro para endereços de grupo multicast. |
| [**\_nível do provedor nStatus \_**](/windows/win32/api/nsemail/ne-nsemail-napi_provider_level) | Especifica o nível de autoridade de provedor de um \_ provedor de namespace de email NS para um determinado domínio. |
| [**\_tipo de provedor nStatus \_**](/windows/win32/api/nsemail/ne-nsemail-napi_provider_type) | Especifica o tipo de hospedagem esperado para um provedor de namespace. |
| [**\_tipo de \_ conclusão de notificação rio \_**](/windows/win32/api/mswsock/ne-mswsock-rio_notification_completion_type) | Especifica o tipo de notificações de fila de conclusão a ser usado com a função [**RIONotify**](/windows/win32/api/mswsock/nc-mswsock-lpfn_rionotify) ao enviar ou receber dados usando as extensões de e/s registradas pelo Winsock. |
| [**\_protocolo de segurança de soquete \_**](/windows/win32/api/mstcpip/ne-mstcpip-socket_security_protocol) | Indica o tipo de protocolo de segurança a ser usado em um soquete. |
| [**\_tipo de uso de soquete \_**](/windows/win32/api/Mstcpip/ne-mstcpip-socket_usage_type) | Usado para especificar o tipo de uso para o soquete. |
| [**TCPSTATE**](/windows/win32/api/mstcpip/ne-mstcpip-tcpstate) | Indica os possíveis estados de uma conexão de protocolo TCP. |
| [**WSAECOMPARATOR**](/windows/win32/api/Winsock2/ne-winsock2-wsaecomparator) | Usado para semântica de comparação de versão. |
| [**\_tipo de \_ informações do provedor de WSC \_**](/windows/win32/api/ws2spi/ne-ws2spi-wsc_provider_info_type) | Especifica a classe Information de um LSP (protocolo de serviço em camadas). |
