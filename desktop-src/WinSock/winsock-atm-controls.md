# <a name="winsock-atm-controls"></a>Controles do WINSOCK ATM

A instalação e a redução de conexão ponto a ponto e ponto a multiponto do ATM têm suporte nativo pela especificação Windows Sockets 2. Na verdade, Windows especificação QoS 2 e mecanismos multicast/multicast independentes de protocolo foram projetados com o ATM em mente, juntamente com outros protocolos. Consulte a seção 2.7 e o Apêndice D da especificação da Windows API Windows Sockets 2 para soquetes 2, qualidade de serviço e suporte a vários pontos, respectivamente. Portanto, nenhum Ioctl específico do ATM precisa ser introduzido neste documento.
