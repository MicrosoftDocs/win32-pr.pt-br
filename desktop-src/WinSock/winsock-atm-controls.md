# <a name="winsock-atm-controls"></a>Controles Winsock ATM

A configuração de conexão ponto a ponto do ATM e de ponto a multiponto e a desmontagem são compatíveis nativamente com a especificação do Windows Sockets 2. Na verdade, a especificação de QoS do Windows Sockets 2 e os mecanismos de MultiPoint/multicast independentes de protocolo foram projetados com o ATM em mente, juntamente com outros protocolos. Consulte a seção 2,7 e o Apêndice D da especificação da API do Windows Sockets 2 para Windows Sockets 2, qualidade de serviço e suporte do MultiPoint, respectivamente. Portanto, nenhum IOCTL específico do ATM precisa ser apresentado neste documento.
