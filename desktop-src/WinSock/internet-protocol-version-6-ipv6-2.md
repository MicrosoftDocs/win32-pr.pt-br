# <a name="internet-protocol-version-6-ipv6"></a>IPv6 (Protocolo IP Versão 6)

É importante garantir que novos aplicativos Winsock, bem como aplicativos existentes, sejam totalmente compatíveis com o IPv6. A disponibilidade do espaço de endereço IPv4 para novas alocações de endereço IPv4 foi esgotada para a Ásia e o Pacífico em 2011. Espera-se que outras partes do mundo sejam esgotadas em alguns anos.

Um percentual crescente de novos sites e serviços está disponível usando endereços IPv6. Muitos usuários da Internet em mercados emergentes dependem do IPv6 para acesso à Internet.

A Microsoft tem um longo compromisso de dar suporte ao IPv6. O suporte completo a IPv6 é fornecido no Windows XP com Service Pack 1 (SP1) e posterior.

Este documento fornece informações sobre o suporte do Winsock para IPv6:

-   [Usando IPv6](using-ipv6-2.md)
-   [Usando o Internet Explorer para acessar sites IPv6](using-internet-explorer-to-access-ipv6-web-sites-2.md)
-   [Configurações recomendadas para IPv6](recommended-configurations-2.md)
-   [Tópicos adicionais sobre IPv6](additional-topics-2.md)

Para obter informações adicionais sobre como adicionar funcionalidades do IPv6 aos seus aplicativos do Windows Sockets, consulte [Guia do IPv6 para aplicativos do Windows Sockets](ipv6-guide-for-windows-sockets-applications-2.md).

Uma introdução ao protocolo IPv6 juntamente com visões gerais sobre a implantação e tecnologias de transição IPv6 está disponível no TechNet no [Microsoft Internet Protocol versão 6 (IPv6)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/dd379473(v=ws.10)).
