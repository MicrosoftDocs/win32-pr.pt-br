# <a name="directed-broadcast"></a>Difusão direcionada

Geralmente considerado mais amigável de rede do que uma difusão de todas as rotas, uma difusão direcionada limita-se à rede local que você especificar. Preencha o **SA \_ netnum** com o número de rede de destino e o **\_ nodenum SA** com binários. Os roteadores encaminham essa solicitação para a rede de destino onde se torna uma difusão local. Redes intermediárias não veem este pacote como uma difusão.
