# <a name="directed-broadcast"></a>Difusão direcionada

Geralmente considerado mais amigável à rede do que uma difusão de todas as rotas, uma difusão direcionada se limita à rede local especificada. Preencha **sa \_ netnum** com o número de rede de destino e **o \_ nodenum sa** com binários. Os roteadores encaminham essa solicitação para a rede de destino, em que ela se torna uma difusão local. As redes intermediárias não veem esse pacote como uma difusão.
