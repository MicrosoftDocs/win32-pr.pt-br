# <a name="pointers-windows-sockets-2"></a>Ponteiros (Windows Sockets 2)

Todos os ponteiros usados por aplicativos com Windows Sockets devem estar longe, embora isso seja relevante apenas para aplicativos de 16 bits e sem sentido em um 32 bits. Para facilitar isso, são fornecidas definições de tipo de dados, como **LPHOSTENT** .
