# <a name="usage-model"></a>Modelo de uso

As diretrizes de uso e implementação para aplicativos e serviços habilitados para QoS são fornecidas em uma seção separada no SDK (Software Development Kit) do Microsoft Windows dedicado à QoS. Consulte a seção [qualidade de serviço (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) do SDK do Windows para obter informações mais detalhadas sobre o modelo de uso de QoS.
