# <a name="getprotobyname-and-getprotobynumber-functions-in-the-api"></a>Funções getprotobyname e getprotobynumber na API

As [**funções getprotobyname**](/windows/desktop/api/winsock/nf-winsock-getprotobyname) e [**getprotobynumber**](/windows/desktop/api/winsock/nf-winsock-getprotobynumber) são implementadas no Ws232.dll consultando um banco de dados \_ de protocolos locais. Eles não resultam em nenhuma consulta de resolução de nomes.
