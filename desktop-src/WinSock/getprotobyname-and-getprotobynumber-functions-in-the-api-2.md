# <a name="getprotobyname-and-getprotobynumber-functions-in-the-api"></a>Funções getprotobyname e getprotobynumber na API

As funções [**getprotobyname**](/windows/desktop/api/winsock/nf-winsock-getprotobyname) e [**getprotobynumber**](/windows/desktop/api/winsock/nf-winsock-getprotobynumber) são implementadas dentro do ws232.dll consultando \_ um banco de dados de protocolos locais. Eles não resultam em nenhuma consulta de resolução de nome.
