# <a name="secure-winsock-programming"></a>Programação Secure Winsock

Este é um guia para proteger a programação Windows Sockets. Ele foi projetado para fornecer uma compreensão da segurança do Winsock e das opções disponíveis para o desenvolvedor de aplicativos de rede seguro.

-   [Usando SO \_ REUSEADDR e SO \_ EXCLUSIVEADDRUSE](using-so-reuseaddr-and-so-exclusiveaddruse.md)
-   [Extensões de soquete seguro winsock](winsock-secure-socket-extensions.md)

As comunicações que usam soquetes também podem ser criptografadas usando os padrões SSL/TLS usando o Canal Seguro, também conhecido como tecnologia Schannel. O Schannel é um SSP (provedor de suporte de segurança) que contém um conjunto de protocolos de segurança que fornecem autenticação de identidade e comunicação privada segura por meio da criptografia. O Schannel é usado principalmente para aplicativos da Internet que exigem comunicações HTTP (Protocolo de Transferência de Hipertexto) seguras. Para obter mais informações, consulte [Canal Seguro.](../secauthn/secure-channel.md)
