# <a name="sio_multipoint_loopback-command-code-for-wsaioctl"></a>Código de comando SIO \_ MULTIPOINT \_ LOOPBACK para WSAIoctl

Quando os soquetes folha d são usados em um plano de dados não rooted, é desejável que o tráfego enviado seja recebido novamente \_ no mesmo soquete. O código de comando SIO \_ MULTIPOINT \_ LOOPBACK para [**WSAIoctl**](/windows/desktop/api/Winsock2/nf-winsock2-wsaioctl) é usado para habilitar ou desabilitar o loopback do tráfego de vários pontos.
