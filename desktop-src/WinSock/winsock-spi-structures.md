# <a name="winsock-spi-structures"></a>Estruturas SPI do Winsock

A lista a seguir fornece descrições concisas de cada estrutura SPI do Winsock. Para obter informações adicionais sobre qualquer estrutura de SPI do Winsock, clique no nome da estrutura.



| Estrutura de SPI do Winsock                                         | Descrição                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_rotina NSP**](/windows/desktop/api/Ws2spi/ns-ws2spi-nsp_routine)                           | Contém informações sobre as funções implementadas por um provedor de provedor de serviços de namespace versão 1 (NSPv1). |
| [**\_Rotina NSPV2**](/windows/desktop/api/Ws2spi/ns-ws2spi-nspv2_routine)                       | Contém informações sobre as funções implementadas por um provedor de provedor de serviços de namespace versão 2 (NSPv2). |
| [**WSATHREADID**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsathreadid)                          | Permite que um provedor identifique um thread no qual as chamadas de procedimento assíncrono (APCs) podem ser enfileiradas.           |
| [**\_informações de \_ auditoria do provedor de WSC \_**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsc_provider_audit_info) | Contém informações de auditoria para uma entrada de LSP (provedor de serviço em camadas).                                        |
| [**WSPDATA**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspdata)                                  | Contém informações do provedor de serviços.                                                                        |
| [**\_tabela WSPPROC**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspproc_table)                       | Contém uma tabela de ponteiros para funções de provedor de serviço.                                                   |
| [**WSPUPCALLTABLE**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspupcalltable)                      | Contém uma tabela de ponteiros para funções de chamada de provedor de serviço.                                            |
