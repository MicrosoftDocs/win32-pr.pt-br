# <a name="summary-of-namespace-provider-functions"></a>Resumo das funções do provedor de namespace

As funções do provedor de serviço de namespace podem ser agrupadas em cinco categorias:

-   Configuração do provedor de namespace (e instalação)
-   Inicialização do provedor
-   Instalação do serviço
-   Consultas de cliente
-   Funções auxiliares (e macros)

As seções a seguir identificam as funções em cada categoria e descrevem brevemente o uso pretendido. As estruturas de dados-chave também são descritas.

-   [Configuração e instalação do provedor de namespace](name-space-provider-configuration-and-installation-2.md)
-   [Inicialização e limpeza do provedor de namespace](name-space-provider-initialization-and-cleanup-2.md)
-   [Instalação de serviço no SPI do Windows Sockets 2](service-installation-in-the-windows-sockets-2-spi-2.md)
-   [Consulta de serviço](service-query-2.md)
-   [Funções auxiliares no SPI](helper-functions-in-the-spi-2.md)
