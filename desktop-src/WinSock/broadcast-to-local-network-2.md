# <a name="broadcast-to-local-network"></a>Difusão para a rede local

Uma difusão pode ser feita na rede conectada localmente definindo **SA \_ netnum** como zeros binários (0) e **SA \_ nodenum** para Binary os (1). Essa difusão pode ser enviada para a rede principal para o dispositivo ou para todas as redes conectadas localmente na opção do provedor de serviços. Difusões para a rede local não são propagadas por roteadores.
