# <a name="multipoint-and-multicast-semantics"></a>Semântica de multicast e multiponto

Ao considerar como dar suporte à semântica multiponto e multicast no Windows Sockets 2 (Winsock), vários esquemas existentes e propostos (incluindo multicast de IP, conexão ponto a multiponto ATM, ST-II, T. 120, H. 320-MCU) foram examinados. Para habilitar uma discussão coerente dos vários esquemas, é importante criar primeiro uma taxonomia que caracteriza os atributos essenciais de cada um. Neste documento, o termo *MultiPoint* representa o MultiPoint e o multicast.
