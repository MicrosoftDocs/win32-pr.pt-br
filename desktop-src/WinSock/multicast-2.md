# <a name="multicast"></a>Multicast

As funções genéricas de multipoint Winsock são suportadas por multicast de IP. No entanto, os provedores de transporte TCP/IP que dão suporte a multicast também devem fornecer suporte ao multicast de estilo da Berkeley Software Distribution (BSD), dando suporte às opções de multicast correspondentes. Isso simplifica a portação de aplicativos multicast existentes para Windows Sockets 2.
