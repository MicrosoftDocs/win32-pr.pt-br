# <a name="multicast"></a>Multicast

As funções do MultiPoint do Winsock genérico dão suporte a multicast IP. No entanto, os provedores de transporte TCP/IP que oferecem suporte a multicast também devem fornecer o estilo BSD (Berkeley Software Distribution) – suporte multicast ao dar suporte às opções de multicast correspondentes. Isso simplifica a portabilidade de aplicativos multicast existentes para o Windows Sockets 2.
