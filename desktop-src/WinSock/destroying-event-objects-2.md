# <a name="destroying-event-objects"></a>Destruindo objetos de evento

A entidade que cria um objeto de evento (aplicativo ou provedor de serviços) é responsável por destruí-lo quando ele não for mais necessário. Os provedores de serviço fazem isso por meio de **WPUCloseEvent**.
