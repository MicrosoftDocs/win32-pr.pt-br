# <a name="default-values-in-the-spi"></a>Valores padrão no SPI

Os detalhes de implementação de QoS (qualidade de serviço) são fornecidos em uma seção de QoS separada e dedicada no SDK (Software Development Kit) do Microsoft Windows. Consulte a seção [qualidade de serviço](/previous-versions/windows/desktop/qos/qos-start-page) da SDK do Windows para obter informações sobre as diretrizes de uso de QoS em relação aos valores padrão.
