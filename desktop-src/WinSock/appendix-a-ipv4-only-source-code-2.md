# <a name="appendix-a-ipv4-only-source-code"></a>Apêndice A: código-fonte somente IPv4

Este apêndice fornece o código-fonte para os exemplos do SDK (Software Development Kit) da plataforma Simplec. c e simples. c, que são usados em todo este guia IPv6 como exemplos:

-   [Código de cliente somente IPv4](ipv4-only-client-code-2.md)
-   [Código de servidor somente IPv4](ipv4-only-server-code-2.md)

> [!WARNING]
>
> Esse código está incluído neste guia para comparar o código independente da versão IP fornecido no apêndice B. Este código de exemplo é fornecido para fins de comparação e para fornecer uma referência clara e prontamente disponível ao ler este guia IPv6. Uma versão de IP independente do código-fonte é fornecida no [Apêndice B: código-fonte independente da versão IP](appendix-b-ip-version-agnostic-source-code-2.md).

 

> [!Note]  
> Ao comparar o código-fonte no apêndice A (somente IPv4) e no apêndice B (versão do IP independente), você tem uma noção das alterações necessárias para modificar o aplicativo Windows Sockets para adicionar suporte para IPv6.
