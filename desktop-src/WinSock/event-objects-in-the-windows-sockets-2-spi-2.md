# <a name="event-objects-in-the-windows-sockets-2-spi"></a>objetos de evento no SPI do Windows sockets 2

os objetos de evento são introduzidos no Windows sockets 2 como um mecanismo de sincronização geral entre os provedores de serviço do Winsock e os aplicativos. Eles são usados para uma série de finalidades que incluem indicar a conclusão de operações sobrepostas e a ocorrência de um ou mais eventos de rede.
