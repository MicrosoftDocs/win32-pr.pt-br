# <a name="using-sendto-while-connected"></a>Usando SendTo enquanto conectado

O [**WSPSendTo**](/previous-versions/windows/desktop/legacy/ms742291(v=vs.85)) sempre fornecerá os dados para o endereço especificado, embora um par designado para o soquete de envio tenha sido estabelecido em [**WSPConnect**](/previous-versions/windows/hardware/network/ff566275(v=vs.85)).
