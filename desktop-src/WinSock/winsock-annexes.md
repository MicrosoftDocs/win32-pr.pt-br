# <a name="winsock-annexes"></a>Anexos do Winsock

Os anexos do Winsock fornecem informações de implementação para pacotes de protocolo Winsock comuns e discutem como usar cada protocolo com o Winsock.

Cada protocolo nesse anexo tem convenções, comportamentos ou recursos especiais que não se prestam à implementação genérica. Esta seção documenta os detalhes que os desenvolvedores devem considerar ao implementar ou usar os protocolos descritos.

Os anexos do Winsock incluem as seguintes seções:

-   [Anexo ATM do Winsock](winsock-atm-annex.md)
-   [Anexo de IPX/SPX do Winsock](winsock-ipx-spx-annex.md)
-   [Anexo TCP/IP do Winsock](winsock-tcp-ip-annex.md)
