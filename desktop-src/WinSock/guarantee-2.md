# <a name="guarantee"></a>GARANTIA

A enumeração de **garantia** do Windows Sockets não é mais usada. Para obter informações sobre a implementação de qualidade de serviço do Windows e a API associada, consulte [QoS (qualidade de serviço)](/previous-versions/windows/desktop/qos/qos-start-page) no SDK (Software Development Kit) da plataforma.
