# <a name="compiling-the-sample-service-provider-using-nmake"></a>Compilando o provedor de serviços de exemplo usando o NMAKE

Para compilar o provedor de serviços de exemplo usando o NMAKE, abra uma janela de comando do Visual Studio, navegue até <*caminho de instalação do SDK* > \\ WMFSDK95 \\ WMDM \\ MDSP e digite `nmake` no prompt.
