# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Política de privacidade (SDK do Windows Media Gerenciador de Dispositivos 11)

Os desenvolvedores que criam aplicativos do Windows Media Gerenciador de Dispositivos devem se comunicar claramente com seus usuários como sua privacidade pode ser afetada. Para ajudá-lo a criar uma declaração de privacidade do usuário final, a Microsoft postou diretrizes de privacidade na [página da Web de privacidade do Windows Media](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx).
