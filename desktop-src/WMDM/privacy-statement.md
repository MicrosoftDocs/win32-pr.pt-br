# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Política de privacidade (Windows mídia Gerenciador de Dispositivos 11 SDK)

Os desenvolvedores que Windows aplicativos Gerenciador de Dispositivos mídia devem se comunicar claramente com seus usuários sobre como sua privacidade pode ser afetada. Para ajudá-lo a criar uma política de privacidade do usuário final, a Microsoft publicou diretrizes de privacidade na [Windows Web de Privacidade de Mídia.](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx)
