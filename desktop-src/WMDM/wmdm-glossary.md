# <a name="wmdm-glossary"></a>Glossário do WMDM

Para localizar um termo no Glossário, clique na letra do alfabeto que é a primeira letra do termo que você deseja pesquisar.

Você também pode ler os termos do glossário dentro do texto da ajuda clicando nos links de termo do glossário sublinhado. Depois de clicar em um link do glossário, o termo e a definição do glossário aparecerão em uma janela pop-up. Para fechar a janela, clique em qualquer lugar na tela.

<dl> <dt>

<span id="projectid114.active_template_library__atl_"></span><span id="PROJECTID114.ACTIVE_TEMPLATE_LIBRARY__ATL_"></span>**Active Template Library (ATL)**
</dt> <dd>

Uma coleção de classes de modelo para Microsoft Visual C++. Ele é usado para criar servidores COM ao gravar um provedor de serviços para Windows Media Gerenciador de Dispositivos.

</dd> <dt>

<span id="projectid114.atl"></span><span id="PROJECTID114.ATL"></span>**ATL**
</dt> <dd>

Consulte a definição para: [Active Template Library (ATL)](/windows)

</dd> <dt>

<span id="projectid114.gls_authentication"></span><span id="PROJECTID114.GLS_AUTHENTICATION"></span>**autenticação** do
</dt> <dd>

O processo de verificar a identidade de um usuário, computador, processo ou outra entidade validando as credenciais fornecidas pela entidade. Formas comuns de credenciais são assinaturas digitais, cartões inteligentes, dados biométricos e uma combinação de nomes e senhas de usuários.

</dd> <dt>

<span id="projectid114.authorization"></span><span id="PROJECTID114.AUTHORIZATION"></span>**autorização** do
</dt> <dd>

Para conceder acesso a um recurso em um sistema de computador ou rede, geralmente durante a autenticação de uma entidade. Uma entidade autenticada pode ser autorizada para diferentes tipos de acesso ou atividade.

</dd> <dt>

<span id="projectid114.bit_rate"></span><span id="PROJECTID114.BIT_RATE"></span>**taxa de bits**
</dt> <dd>

O número de bits transferidos por unidade de tempo, normalmente expressos em bits por segundo.

</dd> <dt>

<span id="projectid114.buffer"></span><span id="PROJECTID114.BUFFER"></span>**buffer**
</dt> <dd>

Uma área de memória reservada para manter dados temporariamente antes que esses dados sejam usados por um dispositivo ou aplicativo receptor. O armazenamento em buffer protege contra a interrupção do fluxo de dados.

</dd> <dt>

<span id="projectid114.challenge"></span><span id="PROJECTID114.CHALLENGE"></span>**desafio**
</dt> <dd>

Uma solicitação de uma licença. Um desafio contém informações sobre o computador do consumidor, uma lista de direitos solicitados e outras informações sobre o conteúdo, incluindo o cabeçalho do conteúdo e a ID da chave. Esse desafio também pode conter um desafio da versão 1 caso uma licença da versão 1 seja necessária.

</dd> <dt>

<span id="projectid114.clear_content"></span><span id="PROJECTID114.CLEAR_CONTENT"></span>**Limpar conteúdo**
</dt> <dd>

Um arquivo de mídia do Windows que não está criptografado.

</dd> <dt>

<span id="projectid114.client"></span><span id="PROJECTID114.CLIENT"></span>**cliente** do
</dt> <dd>

Qualquer computador ou programa que se conecte ou solicite os serviços do, outro computador ou programa. O cliente também pode se referir ao software que permite que o computador ou programa estabeleça a conexão.

</dd> <dt>

<span id="projectid114.content"></span><span id="PROJECTID114.CONTENT"></span>**conteúdo** do
</dt> <dd>

Áudio, vídeo, imagens, texto ou qualquer outra informação contida em um fluxo ou arquivo de mídia digital.

</dd> <dt>

<span id="projectid114.decrypt"></span><span id="PROJECTID114.DECRYPT"></span>**descriptografar**
</dt> <dd>

Para converter o conteúdo criptografado de volta em seu formato original.

</dd> <dt>

<span id="projectid114.digital_rights_management__drm_"></span><span id="PROJECTID114.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**DRM (gerenciamento de direitos digitais)**
</dt> <dd>

Qualquer tecnologia usada para proteger os interesses dos proprietários de serviços e conteúdo digital (como os proprietários de direitos autorais). Normalmente, os destinatários autorizados ou os usuários devem adquirir uma licença para usar o conteúdo.

</dd> <dt>

<span id="projectid114.drm"></span><span id="PROJECTID114.DRM"></span>**DRM**
</dt> <dd>

Consulte a definição para: [DRM (gerenciamento de direitos digitais)](/windows)

</dd> <dt>

<span id="projectid114.encrypt"></span><span id="PROJECTID114.ENCRYPT"></span>**criptografar**
</dt> <dd>

Para disfarçar o conteúdo de forma programática para ocultar sua substância.

</dd> <dt>

<span id="projectid114.firewall"></span><span id="PROJECTID114.FIREWALL"></span>**Firewall** do
</dt> <dd>

Uma solução de segurança que separa uma parte de uma rede de outra parte, permitindo que apenas o tráfego de rede autorizado passe de acordo com as regras de filtragem de tráfego.

</dd> <dt>

<span id="projectid114.key"></span><span id="PROJECTID114.KEY"></span>**chave**
</dt> <dd>

Um dado que é necessário para desbloquear um arquivo de mídia do Windows empacotado. Essa chave é incluída em uma licença separada.

</dd> <dt>

<span id="projectid114.license"></span><span id="PROJECTID114.LICENSE"></span>**licença** do
</dt> <dd>

Dados anexados a conteúdo protegido que descreve como o conteúdo pode ser usado.

</dd> <dt>

<span id="projectid114.microsoft_secure_content_provider__scp_"></span><span id="PROJECTID114.MICROSOFT_SECURE_CONTENT_PROVIDER__SCP_"></span>**Provedor de conteúdo seguro da Microsoft (SCP)**
</dt> <dd>

Um plug-in para Windows Media Gerenciador de Dispositivos que fornece serviços DRM (gerenciamento de direitos digitais).

</dd> <dt>

<span id="projectid114.microsoft_windows_media_player"></span><span id="PROJECTID114.MICROSOFT_WINDOWS_MEDIA_PLAYER"></span>**Microsoft Windows Media Player**
</dt> <dd>

Um cliente/controle que recebe um fluxo de um servidor do Windows Media ou conteúdo local para reprodução. Ele pode ser executado como um programa executável de cliente autônomo ou ser inserido em uma página da Web, em um programa C++ ou em um programa de Visual Basic da Microsoft que usa o controle ActiveX do cliente.

</dd> <dt>

<span id="projectid114.object"></span><span id="PROJECTID114.OBJECT"></span>**objeto** do
</dt> <dd>

Uma coleção de propriedades e métodos que atendem a uma finalidade específica e são tratados como uma unidade.

</dd> <dt>

<span id="projectid114.player"></span><span id="PROJECTID114.PLAYER"></span>**Player** do
</dt> <dd>

Um programa cliente ou controle que recebe o conteúdo de mídia digital transmitido de um servidor ou executado de arquivos locais. O Windows Media Player é um exemplo de um player.

</dd> <dt>

<span id="projectid114.plug_in"></span><span id="PROJECTID114.PLUG_IN"></span>**plug-in**
</dt> <dd>

Um componente de software auxiliar que estende ou aprimora os recursos de outro software.

</dd> <dt>

<span id="projectid114.portable_device"></span><span id="PROJECTID114.PORTABLE_DEVICE"></span>**dispositivo portátil**
</dt> <dd>

Um dispositivo eletrônico móvel que pode trocar arquivos ou outros dados com um computador ou dispositivo. Exemplos de dispositivos portáteis incluem Pocket PCs, players de música digital portáteis e smartphones.

</dd> <dt>

<span id="projectid114.portable_media_identifier"></span><span id="PROJECTID114.PORTABLE_MEDIA_IDENTIFIER"></span>**identificador de mídia portátil**
</dt> <dd>

Um identificador exclusivo para mídia de armazenamento removível, como cartões CompactFlash. O identificador é embutido em código na mídia de armazenamento e não é alterável.

</dd> <dt>

<span id="projectid114.protect"></span><span id="PROJECTID114.PROTECT"></span>**proteger**
</dt> <dd>

Para criptografar arquivos com uma chave e adicionar informações como a URL de aquisição de licença.

</dd> <dt>

<span id="projectid114.sdmi"></span><span id="PROJECTID114.SDMI"></span>**SDMI**
</dt> <dd>

Consulte a definição de: [SDMI (iniciativa de música digital segura)](/windows)

</dd> <dt>

<span id="projectid114.secure_digital_music_initiative__sdmi_"></span><span id="PROJECTID114.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**SDMI (iniciativa de música digital) segura**
</dt> <dd>

Uma organização que define padrões para música digital segura. Um dos principais objetivos do SDMI é criar uma estrutura para a reprodução, o armazenamento e a distribuição seguros de música digital.

</dd> <dt>

<span id="projectid114.windows_media_encoder"></span><span id="PROJECTID114.WINDOWS_MEDIA_ENCODER"></span>**Codificador de mídia do Windows**
</dt> <dd>

Uma tecnologia que compacta áudio e vídeo dinâmicos ou pré-gravados em um fluxo de mídia do Windows, que pode ser distribuído imediatamente ou salvo como um arquivo de mídia do Windows para distribuição posterior.

</dd> <dt>

<span id="projectid114.windows_media_file"></span><span id="PROJECTID114.WINDOWS_MEDIA_FILE"></span>**Arquivo de mídia do Windows**
</dt> <dd>

Um arquivo que contém dados de áudio, vídeo ou script. O conteúdo do arquivo é codificado com um dos codecs de mídia do Windows.

</dd> <dt>

<span id="projectid114.windows_media_format"></span><span id="PROJECTID114.WINDOWS_MEDIA_FORMAT"></span>**Formato de mídia do Windows**
</dt> <dd>

O formato de um arquivo de mídia digital ou fluxo que foi codificado com os codecs de mídia do Windows.

</dd> <dt>

<span id="projectid114.windows_media_services"></span><span id="PROJECTID114.WINDOWS_MEDIA_SERVICES"></span>**Serviços de mídia do Windows**
</dt> <dd>

Um serviço que é usado para transmitir áudio digital e vídeo pela Internet ou por uma intranet.

</dd> </dl>
