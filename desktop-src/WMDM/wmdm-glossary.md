# <a name="wmdm-glossary"></a>Glossário do WMDM

Para encontrar um termo no glossário, clique na letra do alfabeto que é a primeira letra no termo que você deseja procurar.

Você também pode ler os termos do glossário dentro do texto da Ajuda clicando nos links de termo do glossário sublinhado. Depois de clicar em um link de glossário, o termo e a definição do glossário aparecem em uma janela pop-up. Para fechar a janela, clique em qualquer lugar na tela.

<dl> <dt>

<span id="projectid114.active_template_library__atl_"></span><span id="PROJECTID114.ACTIVE_TEMPLATE_LIBRARY__ATL_"></span>**Active Template Library (ATL)**
</dt> <dd>

Uma coleção de classes de modelo para Microsoft Visual C++. Ele é usado para criar servidores COM ao escrever um provedor de serviços para Windows Mídia Gerenciador de Dispositivos.

</dd> <dt>

<span id="projectid114.atl"></span><span id="PROJECTID114.ATL"></span>**ATL**
</dt> <dd>

Consulte a definição de: [Active Template Library (ATL)](/windows)

</dd> <dt>

<span id="projectid114.gls_authentication"></span><span id="PROJECTID114.GLS_AUTHENTICATION"></span>**autenticação**
</dt> <dd>

O processo de verificar a identidade de um usuário, computador, processo ou outra entidade validando as credenciais fornecidas pela entidade. Formas comuns de credenciais são assinaturas digitais, cartões inteligentes, dados biométricos e uma combinação de nomes e senhas de usuários.

</dd> <dt>

<span id="projectid114.authorization"></span><span id="PROJECTID114.AUTHORIZATION"></span>**autorização**
</dt> <dd>

Para conceder acesso a um recurso em um sistema de computador ou rede, geralmente durante a autenticação de uma entidade. Uma entidade autenticada pode ser autorizada para diferentes tipos de acesso ou atividade.

</dd> <dt>

<span id="projectid114.bit_rate"></span><span id="PROJECTID114.BIT_RATE"></span>**taxa de bits**
</dt> <dd>

O número de bits transferidos por unidade de tempo, normalmente expressos em bits por segundo.

</dd> <dt>

<span id="projectid114.buffer"></span><span id="PROJECTID114.BUFFER"></span>**buffer**
</dt> <dd>

Uma área de memória reservada para manter temporariamente os dados antes que esses dados são usados por um dispositivo ou aplicativo de recebimento. O buffer protege contra a interrupção do fluxo de dados.

</dd> <dt>

<span id="projectid114.challenge"></span><span id="PROJECTID114.CHALLENGE"></span>**desafio**
</dt> <dd>

Uma solicitação para uma licença. Um desafio contém informações sobre o computador do consumidor, uma lista de direitos solicitados e outras informações sobre o conteúdo, incluindo o título de conteúdo e a ID da chave. Esse desafio também pode conter um desafio de versão 1 caso uma licença da versão 1 seja necessária.

</dd> <dt>

<span id="projectid114.clear_content"></span><span id="PROJECTID114.CLEAR_CONTENT"></span>**limpar conteúdo**
</dt> <dd>

Um Windows de mídia que não está criptografado.

</dd> <dt>

<span id="projectid114.client"></span><span id="PROJECTID114.CLIENT"></span>**cliente**
</dt> <dd>

Qualquer computador ou programa que se conecte ou solicite os serviços de outro computador ou programa. O cliente também pode se referir ao software que permite que o computador ou o programa estabeleça a conexão.

</dd> <dt>

<span id="projectid114.content"></span><span id="PROJECTID114.CONTENT"></span>**conteúdo**
</dt> <dd>

Áudio, vídeo, imagens, texto ou qualquer outra informação contida em um arquivo ou fluxo de mídia digital.

</dd> <dt>

<span id="projectid114.decrypt"></span><span id="PROJECTID114.DECRYPT"></span>**descriptografar**
</dt> <dd>

Para converter conteúdo criptografado de volta em seu formulário original.

</dd> <dt>

<span id="projectid114.digital_rights_management__drm_"></span><span id="PROJECTID114.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**DRM (gerenciamento** de direitos digitais)
</dt> <dd>

Qualquer tecnologia usada para proteger os interesses dos proprietários de conteúdo digital e serviços (como proprietários de direitos autorais). Normalmente, os usuários ou destinatários autorizados devem adquirir uma licença para usar o conteúdo.

</dd> <dt>

<span id="projectid114.drm"></span><span id="PROJECTID114.DRM"></span>**DRM**
</dt> <dd>

Consulte a definição de: [DRM (gerenciamento de direitos digitais)](/windows)

</dd> <dt>

<span id="projectid114.encrypt"></span><span id="PROJECTID114.ENCRYPT"></span>**criptografar**
</dt> <dd>

Para desalocar programaticamente o conteúdo para ocultar sua essência.

</dd> <dt>

<span id="projectid114.firewall"></span><span id="PROJECTID114.FIREWALL"></span>**firewall**
</dt> <dd>

Uma solução de segurança que separa uma parte de uma rede de outra parte, permitindo que apenas o tráfego de rede autorizado passe de acordo com as regras de filtragem de tráfego.

</dd> <dt>

<span id="projectid114.key"></span><span id="PROJECTID114.KEY"></span>**chave**
</dt> <dd>

Uma parte dos dados necessários para desbloquear um arquivo Windows Mídia empacotado. Essa chave está incluída em uma licença separada.

</dd> <dt>

<span id="projectid114.license"></span><span id="PROJECTID114.LICENSE"></span>**licença**
</dt> <dd>

Dados anexados ao conteúdo protegido que descrevem como o conteúdo pode ser usado.

</dd> <dt>

<span id="projectid114.microsoft_secure_content_provider__scp_"></span><span id="PROJECTID114.MICROSOFT_SECURE_CONTENT_PROVIDER__SCP_"></span>**Provedor de Conteúdo Seguro da Microsoft (SCP)**
</dt> <dd>

Um plug-in para Windows media Gerenciador de Dispositivos que fornece serviços drm (gerenciamento de direitos digitais).

</dd> <dt>

<span id="projectid114.microsoft_windows_media_player"></span><span id="PROJECTID114.MICROSOFT_WINDOWS_MEDIA_PLAYER"></span>**Microsoft Windows Media Player**
</dt> <dd>

Um cliente/controle que recebe um fluxo de um servidor Windows mídia ou conteúdo local para reprodução. Ele pode ser executado como um programa executável de cliente autônomo ou ser inserido em uma página da Web, programa C++ ou um programa microsoft Visual Basic que usa o controle ActiveX cliente.

</dd> <dt>

<span id="projectid114.object"></span><span id="PROJECTID114.OBJECT"></span>**objeto**
</dt> <dd>

Uma coleção de propriedades e métodos que atendem a uma finalidade específica e são tratados como uma unidade.

</dd> <dt>

<span id="projectid114.player"></span><span id="PROJECTID114.PLAYER"></span>**player**
</dt> <dd>

Um programa cliente ou controle que recebe conteúdo de mídia digital transmitido de um servidor ou que é interpretado de arquivos locais. Windows Media Player é um exemplo de um jogador.

</dd> <dt>

<span id="projectid114.plug_in"></span><span id="PROJECTID114.PLUG_IN"></span>**plug-in**
</dt> <dd>

Um componente de software auxiliar que estende ou aprimora os recursos de outro software.

</dd> <dt>

<span id="projectid114.portable_device"></span><span id="PROJECTID114.PORTABLE_DEVICE"></span>**dispositivo portátil**
</dt> <dd>

Um dispositivo eletrônico móvel que pode trocar arquivos ou outros dados com um computador ou dispositivo. Exemplos de dispositivos portáteis incluem PCs de pocket, players de música digital portáteis e Smartphones.

</dd> <dt>

<span id="projectid114.portable_media_identifier"></span><span id="PROJECTID114.PORTABLE_MEDIA_IDENTIFIER"></span>**identificador de mídia portátil**
</dt> <dd>

Um identificador exclusivo para mídia de armazenamento removível, como cartões CompactFlash. O identificador é codificado na mídia de armazenamento e não pode ser alterável.

</dd> <dt>

<span id="projectid114.protect"></span><span id="PROJECTID114.PROTECT"></span>**proteger**
</dt> <dd>

Para criptografar arquivos com uma chave e adicionar informações como a URL de aquisição de licença.

</dd> <dt>

<span id="projectid114.sdmi"></span><span id="PROJECTID114.SDMI"></span>**SDMI**
</dt> <dd>

Consulte a definição de: [SDMI (Iniciativa de Música Digital Segura)](/windows)

</dd> <dt>

<span id="projectid114.secure_digital_music_initiative__sdmi_"></span><span id="PROJECTID114.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**SDMI (Iniciativa** de Música Digital Segura)
</dt> <dd>

Uma organização que define padrões para música digital segura. Uma das principais metas do SDMI é criar uma estrutura para reprodução segura, armazenamento e distribuição de música digital.

</dd> <dt>

<span id="projectid114.windows_media_encoder"></span><span id="PROJECTID114.WINDOWS_MEDIA_ENCODER"></span>**Windows codificador de mídia**
</dt> <dd>

Uma tecnologia que compacta áudio e vídeo ao vivo ou pré-gravado em um fluxo de mídia do Windows, que pode ser distribuído imediatamente ou salvo como um arquivo de mídia Windows para distribuição posterior.

</dd> <dt>

<span id="projectid114.windows_media_file"></span><span id="PROJECTID114.WINDOWS_MEDIA_FILE"></span>**Windows de mídia**
</dt> <dd>

Um arquivo que contém dados de áudio, vídeo ou script. O conteúdo do arquivo é codificado com um dos codecs Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_format"></span><span id="PROJECTID114.WINDOWS_MEDIA_FORMAT"></span>**Windows de mídia**
</dt> <dd>

O formato de um arquivo de mídia digital ou fluxo codificado com Windows de mídia.

</dd> <dt>

<span id="projectid114.windows_media_services"></span><span id="PROJECTID114.WINDOWS_MEDIA_SERVICES"></span>**Windows Media Services**
</dt> <dd>

Um serviço usado para transmitir áudio digital e vídeo pela Internet ou por uma intranet.

</dd> </dl>
