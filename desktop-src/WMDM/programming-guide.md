# <a name="wmdm-programming-guide"></a>Guia de Programação do WMDM

Esta seção descreve as ações que um aplicativo ou provedor de serviços criado no Windows Media Gerenciador de Dispositivos deve executar.

-   [Tarefas comuns a aplicativos](tasks-common-to-applications-and-service-providers.md) e provedores de serviços descreve tarefas que os aplicativos e provedores de serviços executarão, como registro em log, criptografia e descriptografia e autenticação.
-   [Criar um aplicativo de Gerenciador de Dispositivos De](creating-a-windows-media-device-manager-application.md) mídia do Windows descreve todas as ações que um aplicativo normalmente deve executar.
-   [A criação de um Provedor de](creating-a-service-provider.md) Serviços descreve todas as ações que um provedor de serviços normalmente deve executar.
