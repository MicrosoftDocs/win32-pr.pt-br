# <a name="windows-media-drm-implemented-interfaces"></a>Windows Interfaces DRM-Implemented mídia

As interfaces a seguir não se destinam a serem implementadas por provedores de serviços de terceiros, mas são implementadas e chamadas internamente pelo DRM de Mídia do Windows e Windows Media Gerenciador de Dispositivos. Implementar essas interfaces impedirá que o conteúdo protegido por DRM seja processado. Eles são documentados apenas para a conclusão.

-   [**IWMDRMDevice Interface**](iwmdrmdevice.md)
-   [**IWMDRMDevice2 Interface**](iwmdrmdevice2.md)
