# <a name="windows-media-drm-implemented-interfaces"></a>Interfaces de DRM-Implemented de mídia do Windows

As seguintes interfaces não se destinam a ser implementadas por provedores de serviços de terceiros, mas são implementadas e chamadas internamente pelo Windows Media DRM e pela Gerenciador de Dispositivos de mídia. Implementar essas interfaces impedirá que o conteúdo protegido por DRM seja processado. Eles são documentados simplesmente para fins de integridade.

-   [**Interface IWMDRMDevice**](iwmdrmdevice.md)
-   [**Interface IWMDRMDevice2**](iwmdrmdevice2.md)
