# <a name="interfaces-for-service-providers-and-applications"></a>Interfaces para provedores de serviços e aplicativos

as interfaces a seguir podem ser usadas por provedores de serviços e aplicativos que usam o SDK do Windows Media Gerenciador de Dispositivos.



| Interface                                                | Descrição                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Registra as entradas em um arquivo de log comum.                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | fornece comunicação segura e criptografada entre módulos de Gerenciador de Dispositivos de mídia Windows. |
