# <a name="wmdm-reference"></a>Referência de WMDM

O SDK do Windows Media Gerenciador de Dispositivos consiste em uma coleção de interfaces, estruturas, enumerações e constantes. A seção de referência divide as interfaces em grupos de acordo com o tipo de objeto que as utiliza.



| Seção                                                                                                    | Descrição                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaces para aplicativos](interfaces-for-applications.md)                                             | Descreve as interfaces que são usadas ou implementadas somente por aplicativos de área de trabalho, plug-ins do Windows Media Player ou objetos COM que precisam de acesso de alto nível a um dispositivo portátil.                     |
| [Interfaces para provedores de serviços](interfaces-for-service-providers.md)                                   | Descreve as interfaces que são usadas ou implementadas somente por provedores de serviço, que lidam com a comunicação de baixo nível real com um dispositivo portátil.                                                |
| [Interfaces de DRM-Implemented de mídia do Windows](windows-media-drm-implemented-interfaces.md)                   | Descreve as interfaces que não devem ser implementadas por provedores de serviço de terceiros, mas são implementadas e chamadas internamente pela Windows Media DRM e Windows Media Gerenciador de Dispositivos. |
| [Interfaces para provedores de serviços e aplicativos](interfaces-for-service-providers-and-applications.md) | Descreve as interfaces que podem ser usadas por aplicativos ou por provedores de serviço.                                                                                                               |
| [Interfaces para provedores de conteúdo seguro](interfaces-for-secure-content-providers.md)                     | Descreve as interfaces que devem ser implementadas por um dispositivo ou aplicativo que usará conteúdo protegido por DRM.                                                                                       |
| [Estruturas](structures.md)                                                                               | Descreve estruturas usadas para os métodos de Gerenciador de Dispositivos de mídia do Windows.                                                                                                                                 |
| [Tipos de enumeração](enumeration-types.md)                                                                 | Descreve as enumerações usadas para os métodos de Gerenciador de Dispositivos de mídia do Windows.                                                                                                                               |
| [Constantes de metadados](metadata-constants.md)                                                               | Descreve as constantes definidas pelo SDK do Windows Media Gerenciador de Dispositivos para configurar ou recuperar várias propriedades de metadados para um dispositivo ou objeto no dispositivo.                                 |
| [Códigos de Erro](error-codes.md)                                                                             | Descreve os códigos de erro que podem ser retornados pelos métodos do SDK do Windows Media Gerenciador de Dispositivos.                                                                                                         |
