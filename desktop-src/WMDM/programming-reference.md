# <a name="wmdm-reference"></a>Referência de WMDM

o SDK de Gerenciador de Dispositivos de mídia Windows consiste em uma coleção de interfaces, estruturas, enumerações e constantes. A seção de referência divide as interfaces em grupos de acordo com o tipo de objeto que as utiliza.



| Seção                                                                                                    | Descrição                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaces para aplicativos](interfaces-for-applications.md)                                             | descreve as interfaces que são usadas ou implementadas somente por aplicativos de área de trabalho, Windows Media Player plug-ins ou objetos COM que precisam de acesso de alto nível a um dispositivo portátil.                     |
| [Interfaces para provedores de serviços](interfaces-for-service-providers.md)                                   | Descreve as interfaces que são usadas ou implementadas somente por provedores de serviço, que lidam com a comunicação de baixo nível real com um dispositivo portátil.                                                |
| [Windows Interfaces de DRM-Implemented de mídia](windows-media-drm-implemented-interfaces.md)                   | descreve as interfaces que não devem ser implementadas por provedores de serviço de terceiros, mas são implementadas e chamadas internamente pelo Windows media DRM e Windows media Gerenciador de Dispositivos. |
| [Interfaces para provedores de serviços e aplicativos](interfaces-for-service-providers-and-applications.md) | Descreve as interfaces que podem ser usadas por aplicativos ou por provedores de serviço.                                                                                                               |
| [Interfaces para provedores de conteúdo seguro](interfaces-for-secure-content-providers.md)                     | Descreve as interfaces que devem ser implementadas por um dispositivo ou aplicativo que usará conteúdo protegido por DRM.                                                                                       |
| [Estruturas](structures.md)                                                                               | descreve estruturas usadas para Windows métodos de Gerenciador de Dispositivos de mídia.                                                                                                                                 |
| [Tipos de enumeração](enumeration-types.md)                                                                 | descreve as enumerações usadas para Windows métodos de Gerenciador de Dispositivos de mídia.                                                                                                                               |
| [Constantes de metadados](metadata-constants.md)                                                               | descreve as constantes definidas pelo Windows Media Gerenciador de Dispositivos SDK para configuração ou recuperação de várias propriedades de metadados para um dispositivo ou objeto no dispositivo.                                 |
| [Códigos de Erro](error-codes.md)                                                                             | descreve os códigos de erro que podem ser retornados pelo Windows Media Gerenciador de Dispositivos métodos do SDK.                                                                                                         |
