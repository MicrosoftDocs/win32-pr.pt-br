# <a name="wmdm-enumerations"></a>Enumerações do WMDM

Este tópico documenta um recurso do SDK do Windows Media Gerenciador de Dispositivos. Recomendamos que você migre seu aplicativo para usar a API de dispositivos portáteis do Windows. Para obter mais informações, consulte o SDK de dispositivos portáteis do Windows.

O Windows Media Gerenciador de Dispositivos define as seguintes enumerações.



| Tipo de enumeração                                                                  | Descrição                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**\_formulário de \_ \_ valores válidos \_ de prop Enumeração WMDM \_**](wmdm-enum-prop-valid-values-form.md) | Descreve possíveis formas de valores válidos para uma propriedade.                                        |
| [**\_escopo de localização do WMDM \_**](wmdm-find-scope.md)                                      | Define o escopo do objeto de armazenamento.                                                        |
| [**WMDM \_ FORMATCODE**](wmdm-formatcode.md)                                       | Define uma lista de códigos de formato que descrevem os tipos de conteúdo transferidos de e para um dispositivo. |
| [**\_tipo de sessão WMDM \_**](wmdm-session-type.md)                                  | Define o tipo de sessão.                                                                       |
| [**\_modo de \_ enumeração de armazenamento WMDM \_**](wmdm-storage-enum-mode.md)                       | Define como o conteúdo no armazenamento deve ser enumerado.                                     |
| [**tipo de dados de \_ marca WMDM \_**](wmdm-tag-datatype.md)                                  | Define um tipo de dados.                                                                            |
| [**WMDMMessage**](wmdmmessage.md)                                                | Define tipos e Estados de mensagens.                                                               |
