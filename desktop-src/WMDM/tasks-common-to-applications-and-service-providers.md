# <a name="tasks-common-to-applications-and-service-providers"></a>Tarefas comuns a aplicativos e provedores de serviços

Esta seção descreve as seguintes tarefas que se aplicam a aplicativos e provedores de serviços criados no Windows Media Gerenciador de Dispositivos.

-   [A compilação dos arquivos IDL fornecidos com o SDK](compiling-the-idl-files-supplied-with-the-sdk.md) descreve quais arquivos IDL são usados para criar quais arquivos de cabeçalho, se você optar por não usar os arquivos de cabeçalho fornecidos com o SDK.
-   [Habilitar o log](enabling-logging.md) descreve um objeto de log fornecido pelo Windows Media Gerenciador de dispositivos para registrar dados de um objeto em execução.
-   O [uso de canais autenticados seguros](using-secure-authenticated-channels.md) descreve os canais autenticados seguros, que são usados para criptografia, descriptografia e autenticação de componentes e mensagens. Qualquer coisa criada no Windows Media Gerenciador de Dispositivos deve, pelo menos, ser capaz de executar a autenticação de componentes.
-   O [tratamento de conteúdo protegido](handling-protected-content.md) descreve como seu aplicativo pode lidar com conteúdo protegido pelo DRM (gerenciamento de direitos digitais) do Windows Media.
-   [Obter e definir metadados e atributos](getting-and-setting-metadata-and-attributes.md) descreve como um aplicativo pode recuperar e definir atributos e metadados para um armazenamento ou dispositivo.
