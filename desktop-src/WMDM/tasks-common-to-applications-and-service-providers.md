# <a name="tasks-common-to-applications-and-service-providers"></a>Tarefas comuns a aplicativos e provedores de serviços

Esta seção descreve as seguintes tarefas que se aplicam a aplicativos e provedores de serviços Windows mídia Gerenciador de Dispositivos.

-   [Compilar os arquivos IDL](compiling-the-idl-files-supplied-with-the-sdk.md) fornecidos com o SDK descreve quais arquivos IDL são usados para criar quais arquivos de header, se você optar por não usar os arquivos de header fornecidos com o SDK.
-   [A habilitação do log](enabling-logging.md) descreve um objeto de log fornecido pelo Windows Media Gerenciador de Dispositivos para registrar dados de um objeto em execução.
-   [O uso de Canais Autenticados Seguros](using-secure-authenticated-channels.md) descreve os Canais Autenticados Seguros, que são usados para criptografia, descriptografia e autenticação de componentes e mensagens. Qualquer coisa criada no Windows media Gerenciador de Dispositivos deve ser, pelo menos, capaz de executar a autenticação de componente.
-   [Manipulação de conteúdo protegido](handling-protected-content.md) descreve como seu aplicativo pode lidar com o conteúdo protegido Windows DRM (Gerenciamento de Direitos Digitais de Mídia).
-   [Obter e definir metadados e atributos](getting-and-setting-metadata-and-attributes.md) descreve como um aplicativo pode recuperar e definir atributos e metadados para um armazenamento ou dispositivo.
