# <a name="wmdm-samples"></a>Exemplos de WMDM

O SDK do Windows Media Gerenciador de Dispositivos inclui dois exemplos compiláveis: um aplicativo de área de trabalho gráfica de exemplo que permite procurar dispositivos conectados, criar playlists e arrastar arquivos de e para um dispositivo; e um provedor de serviços de exemplo que se comunica com "dispositivos" virtuais no computador (que são, na verdade, os discos rígidos no computador). A documentação explica como compilar e executar esses projetos e também fornece informações sobre como eles funcionam.

Esses dois projetos estão documentados nas seções a seguir:

-   [Aplicativo de área de trabalho de exemplo](sample-desktop-application.md)
-   [Provedor de Serviços de Exemplo](sample-service-provider.md)
