# <a name="typeuri-element"></a>elemento typeUri

Especifica um tipo a ser incluído de um arquivo XSD.

## <a name="usage"></a>Uso

``` syntax
<typeUri
  type = "character_string"
  uri = "character_string"/>
```

## <a name="attributes"></a>Atributos



| Atributo           | Type                         | Obrigatório       | Descrição                                                            |
|---------------------|------------------------------|----------------|------------------------------------------------------------------------|
| **tipo**<br/> | Cadeia de caracteres \_<br/> | Sim<br/> | O nome do tipo.<br/> <br/>                           |
| **uri**<br/>  | Cadeia de caracteres \_<br/> | Sim<br/> | O namespace do tipo. Deve ser um URI válido.<br/> <br/> |



## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                       | Descrição                                                                       |
|-------------------------------|-----------------------------------------------------------------------------------|
| [**XSD**](xsd.md)<br/> | Especifica um arquivo XSD para processar informações de contrato.<br/> <br/> |
