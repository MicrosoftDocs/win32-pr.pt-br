# <a name="typeuri-element"></a>Elemento typeUri

Especifica um tipo a ser incluído de um arquivo XSD.

## <a name="usage"></a>Uso

``` syntax
<typeUri
  type = "character_string"
  uri = "character_string"/>
```

## <a name="attributes"></a>Atributos



| Atributo           | Type                         | Obrigatório       | Descrição                                                            |
|---------------------|------------------------------|----------------|------------------------------------------------------------------------|
| **tipo**<br/> | cadeia de \_ caracteres<br/> | Sim<br/> | O nome do tipo.<br/> <br/>                           |
| **uri**<br/>  | cadeia de \_ caracteres<br/> | Sim<br/> | O namespace do tipo. Deve ser um URI válido.<br/> <br/> |



## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                       | Descrição                                                                       |
|-------------------------------|-----------------------------------------------------------------------------------|
| [**Xsd**](xsd.md)<br/> | Especifica um arquivo XSD a ser processado para obter informações de contrato.<br/> <br/> |
