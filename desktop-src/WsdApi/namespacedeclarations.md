# <a name="namespacedeclarations-element"></a>elemento namespaceDeclarations

Gera declarações de C para tabelas de namespace.

## <a name="usage"></a>Uso

``` syntax
<namespaceDeclarations/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Grupo**](file.md)<br/> | Gera um arquivo do gerador de código.<br/> <br/> |



## <a name="remarks"></a>Comentários

As tabelas de namespace são referenciadas por tabelas de tipo e outros códigos gerados, portanto, esse elemento é usado para gerar arquivos de inclusão.
