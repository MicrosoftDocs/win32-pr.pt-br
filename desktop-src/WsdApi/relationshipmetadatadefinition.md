# <a name="relationshipmetadatadefinition-element"></a>Elemento relationshipMetadataDefinition

Gera uma definição de constante C para os metadados de hospedagem especificados no [**elemento hostMetadata.**](hostmetadata.md) Essa definição é usada em um arquivo de origem.

## <a name="usage"></a>Uso

``` syntax
<relationshipMetadataDefinition/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Arquivo**](file.md)<br/> | Saída de um arquivo do gerador de código.<br/> <br/> |



## <a name="remarks"></a>Comentários

Esse elemento geralmente é usado em arquivos de origem C para fornecer os metadados do host de serviço declarados por **relationshipMetadataDeclaration**.

## <a name="element-information"></a>Informações do elemento



| Rótulo | Valor |
|-------------------------------------|---------------|
| Sistema mínimo com suporte<br/> | Windows Vista |
| Pode estar vazio                        | Sim           |
