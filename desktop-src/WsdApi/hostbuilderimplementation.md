# <a name="hostbuilderimplementation-element"></a>elemento hostBuilderImplementation

Gera uma função que cria um host digitado.

## <a name="usage"></a>Uso

``` syntax
<hostBuilderImplementation>
  child elements
</hostBuilderImplementation>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho



| Elemento                                           | Descrição                                                      |
|---------------------------------------------------|------------------------------------------------------------------|
| [**hostedService**](hostedservice.md)<br/> | O serviço a ser incluído para o host. <br/> <br/> |



### <a name="child-element-sequence"></a>Sequência de elementos filho

``` syntax
hostedService+
```

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Arquivo**](file.md)<br/> | Saída de um arquivo do gerador de código.<br/> <br/> |



## <a name="element-information"></a>Informações do elemento



| Rótulo | Valor |
|-------------------------------------|---------------|
| Sistema mínimo com suporte<br/> | Windows Vista |
| Pode estar vazio                        | Não            |
