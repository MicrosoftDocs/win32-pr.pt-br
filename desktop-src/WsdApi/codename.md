# <a name="codename-element"></a>elemento de codinome

Define o nome a ser usado para identificar o namespace no código gerado.

## <a name="usage"></a>Uso

``` syntax
<codeName/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                                                         | Descrição                                                              |
|-----------------------------------------------------------------|--------------------------------------------------------------------------|
| [**nameSpace**](namespace.md)<br/>                       | Um namespace a ser usado para geração de código.<br/> <br/>       |
| [**portTypeDeclarations**](porttypedeclarations.md)<br/> | Gera declarações de constante C para tipos de porta.<br/> <br/> |
| [**portTypeDefinitions**](porttypedefinitions.md)<br/>   | Gera constantes C para tipos de porta.<br/> <br/>             |



## <a name="remarks"></a>Comentários

Esse elemento substitui o nome de código padrão usado para o código gerado. Por padrão, o código gerado cria um nome de código a partir do URI ou nome qualificado.
