# <a name="path-element"></a>elemento path

Especifica o local e o nome de arquivo de um arquivo WSDL. O caminho pode ser um caminho absoluto ou relativo para o arquivo, mas não deve ser uma URL.

## <a name="usage"></a>Uso

``` syntax
<path/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                             |
|---------------------------------|-------------------------------------------------------------------------|
| [**Wsdl**](wsdl.md)<br/> | Um arquivo WSDL a ser processado para obter informações de contrato.<br/> <br/> |



## <a name="examples"></a>Exemplos

O XML a seguir mostra um elemento **de caminho** válido.

``` syntax
<path>"..\wsdls\example.wsdl"</path>
```

## <a name="element-information"></a>Informações do elemento



| Rótulo | Valor |
|-------------------------------------|---------------|
| Sistema mínimo com suporte<br/> | Windows Vista |
| Pode estar vazio                        | Sim           |
