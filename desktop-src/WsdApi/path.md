# <a name="path-element"></a>elemento Path

Especifica o local e o nome do arquivo WSDL. O caminho pode ser um caminho absoluto ou relativo para o arquivo, mas não deve ser uma URL.

## <a name="usage"></a>Uso

``` syntax
<path/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                             |
|---------------------------------|-------------------------------------------------------------------------|
| [**WSDL**](wsdl.md)<br/> | Um arquivo WSDL para processar informações de contrato.<br/> <br/> |



## <a name="examples"></a>Exemplos

O XML a seguir mostra um elemento **path** válido.

``` syntax
<path>"..\wsdls\example.wsdl"</path>
```
