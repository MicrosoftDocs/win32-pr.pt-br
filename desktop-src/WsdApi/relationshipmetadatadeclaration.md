# <a name="relationshipmetadatadeclaration-element"></a>elemento relationshipMetadataDeclaration

Gera uma declaração de encaminhamento para os metadados de hospedagem especificados no elemento [**hostMetadata**](hostmetadata.md) . Essa declaração é usada em um arquivo de cabeçalho.

## <a name="usage"></a>Uso

``` syntax
<relationshipMetadataDeclaration/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Grupo**](file.md)<br/> | Gera um arquivo do gerador de código.<br/> <br/> |



## <a name="remarks"></a>Comentários

Os metadados de hospedagem são referenciados pelo aplicativo quando ele inicializa o host. Esse elemento é usado para gerar arquivos de inclusão que são incluídos pelo código-fonte do aplicativo.
