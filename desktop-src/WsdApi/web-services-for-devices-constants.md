# <a name="web-services-on-devices-constants"></a>Constantes de serviços Web em dispositivos

As constantes a seguir são usadas pelos Serviços Web em Dispositivos.



| Constante                                                                                                                                                                                                                        | Descrição                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**ENDEREÇO DE HOSPEDAGEM \_ PADRÃO DO WSD \_ \_**</dt> </dl>                       | O endereço padrão (no [formato UrlPrefix)](/windows/desktop/Http/urlprefix-strings) que um host WSDAPI usará para escutar solicitações na porta 5357. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**ENDEREÇO DE HOSPEDAGEM SEGURO PADRÃO DO WSD \_ \_ \_ \_**</dt> </dl> | O endereço seguro padrão (no formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) que um host WSDAPI usará para escutar solicitações na porta 5358. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ ENDEREÇO \_ DE EVENTO \_ PADRÃO**</dt> </dl>               | O endereço padrão (no formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) que um host WSDAPI usará para escutar eventos na porta 5358 para hosts seguros e na porta 5357 para outros hosts. <br/> |
