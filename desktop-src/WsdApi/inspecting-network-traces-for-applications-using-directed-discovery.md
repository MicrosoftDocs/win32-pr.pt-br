-   <span data-ttu-id="326d3-132">Se uma mensagem de [resolução](resolve-message.md) for enviada, uma mensagem [ResolveMatches](resolvematches-message.md) deverá ser enviada para a porta http ou HTTPS da qual a mensagem de resolução foi enviada.</span><span class="sxs-lookup"><span data-stu-id="326d3-132">If a [Resolve](resolve-message.md) message is sent, then a [ResolveMatches](resolvematches-message.md) message must be sent to the HTTP or HTTPS port from which the Resolve message was sent.</span></span>
-   <span data-ttu-id="326d3-133">Uma mensagem [ResolveMatches](resolvematches-message.md) deve ser enviada dentro de 4 segundos da mensagem de [resolução](resolve-message.md) correspondente.</span><span class="sxs-lookup"><span data-stu-id="326d3-133">A [ResolveMatches](resolvematches-message.md) message must be sent within 4 seconds of the corresponding [Resolve](resolve-message.md) message.</span></span> <span data-ttu-id="326d3-134">O Firewall do Windows pode cancelar um ResolveMatchesmessage enviado mais de 4 segundos após uma mensagem de resolução.</span><span class="sxs-lookup"><span data-stu-id="326d3-134">The Windows Firewall may drop a ResolveMatchesmessage sent more than 4 seconds after a Resolve message.</span></span>
