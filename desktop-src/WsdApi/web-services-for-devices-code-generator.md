# <a name="web-services-on-devices-code-generator"></a>Serviços Web no gerador de código de dispositivos

Os serviços Web no gerador de código de dispositivos, WsdCodeGen, é um utilitário de linha de comando que gera código de programa a partir de uma descrição de serviço. A descrição do serviço é armazenada em arquivos WSDL e/ou XSD. WsdCodeGen cria arquivos C++ e IDL (linguagem de definição de interface). Os desenvolvedores podem usar o WsdCodeGen para criar aplicativos WSDAPI sem se preocupar com a forma como os dados são empacotados e representados na conexão.

WsdCodeGen pode ser usado para gerar código para um serviço, um cliente ou ambos. O utilitário gera código stub para serviços, código proxy para clientes e interface e código de tipo para ambos.

Essa ferramenta está disponível no Microsoft Windows Software Development Kit (SDK) e no Windows Driver Kit (WDK). As ferramentas do SDK estão localizadas no seguinte diretório: <Windows SDK Install Folder> \\ bin.

O SDK do Windows inclui código de exemplo gerado por WsdCodeGen. Para obter mais informações, consulte [exemplos de WSDAPI](wsdapi-samples.md).
