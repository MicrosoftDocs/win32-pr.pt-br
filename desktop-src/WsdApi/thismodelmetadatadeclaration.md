# <a name="thismodelmetadatadeclaration-element"></a>elemento thisModelMetadataDeclaration

Gera uma declaração de encaminhamento para a constante C para os metadados do fabricante especificados no elemento [**thisModelMetadata**](thismodelmetadata.md) . Essa declaração é usada em um arquivo de cabeçalho.

## <a name="usage"></a>Uso

``` syntax
<thisModelMetadataDefinition/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Grupo**](file.md)<br/> | Gera um arquivo do gerador de código.<br/> <br/> |
