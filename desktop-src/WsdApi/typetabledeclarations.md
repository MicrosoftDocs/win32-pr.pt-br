# <a name="typetabledeclarations-element"></a>elemento typeTableDeclarations

Gera declarações de constante C para tabelas de esquema XML para tipos conhecidos.

## <a name="usage"></a>Uso

``` syntax
<typeTableDeclarations/>
```

## <a name="attributes"></a>Atributos

Não há atributos.

## <a name="child-elements"></a>Elementos filho

Não há elementos filho.

## <a name="parent-elements"></a>Elementos pai



| Elemento                         | Descrição                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Grupo**](file.md)<br/> | Gera um arquivo do gerador de código.<br/> <br/> |



## <a name="remarks"></a>Comentários

As tabelas de esquema XML são referenciadas por grande parte do código gerado. Esse elemento é usado para gerar arquivos de inclusão.

## <a name="element-information"></a>Informações do elemento



| Rótulo | Valor |
|-------------------------------------|---------------|
| Sistema mínimo com suporte<br/> | Windows Vista |
| Pode estar vazio                        | Sim           |
