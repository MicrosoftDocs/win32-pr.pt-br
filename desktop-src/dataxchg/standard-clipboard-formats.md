# <a name="standard-clipboard-formats"></a>Formatos de área de transferência padrão

Os formatos de área de transferência definidos pelo sistema são chamados de *formatos de área de transferência padrão*. Esses formatos de área de transferência são descritos na tabela a seguir.

## <a name="example"></a>Exemplo

```cpp
  case CF_BITMAP:
        hdcMem = CreateCompatibleDC(hdc);
        if (hdcMem != NULL)
        {
            if (OpenClipboard(hwnd))
            {
                hbm = (HBITMAP) 
                     GetClipboardData(uFormat);
                SelectObject(hdcMem, hbm);
                GetClientRect(hwnd, &rc);
 
                BitBlt(hdc, 0, 0, rc.right, rc.bottom,
                    hdcMem, 0, 0, SRCCOPY);
                CloseClipboard();
             }
             DeleteDC(hdcMem);
        }
    break;
```

O exemplo completo é sobre o [uso de áreas de transferência](using-the-clipboard.md).
