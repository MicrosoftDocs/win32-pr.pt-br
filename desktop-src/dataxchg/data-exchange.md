# <a name="data-exchange"></a>Troca de Dados

As seções a seguir descrevem os métodos básicos de troca de dados.

### <a name="data-exchange"></a>Troca de Dados



| Nome                                                                                     | Descrição                                                                                                                                               |
|------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Área de transferência](clipboard.md)                                                               | Discute a área de transferência. A área de transferência é um conjunto de funções e mensagens que permitem que os aplicativos transfiram dados.<br/>                           |
| [Cópia de dados](data-copy.md)                                                               | Discute como usar a mensagem de cópia de dados para transferir dados de um aplicativo para outro. <br/>                                                  |
| [troca dinâmica de dados](dynamic-data-exchange.md)                                       | Discute a implementação de intercâmbio dinâmico de dados para aplicativos que não podem usar a biblioteca de gerenciamento de troca dinâmica de dados (DDEML).<br/>            |
| [troca dinâmica de dados biblioteca de gerenciamento](dynamic-data-exchange-management-library.md) | Discute a troca dinâmica de dados, que é uma forma de comunicação entre processos que usa memória compartilhada para trocar dados entre aplicativos. <br/> |
