# <a name="windowsnumericsh-apis"></a>APIs windowsnumerics. h

O arquivo de cabeçalho windowsnumerics. h define os tipos de vetor e matriz do C++ no namespace [**Windows. Foundation. Numerics**](/uwp/api/Windows.Foundation.Numerics) . Ele estende as estruturas de **Windows. Foundation. Numerics** com um intervalo de funções e operadores matemáticos.

Esse namespace está disponível apenas em C++. Seu equivalente em .NET é [System. Numerics](/dotnet/api/system.numerics?view=netframework-4.8).

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|-|-|
| [**estrutura float2**](float2-structure.md) | Um vetor com dois componentes. |
| [**estrutura float3**](float3-structure.md) | Um vetor com três componentes. |
| [**estrutura float3x2**](float3x2-structure.md) | Uma matriz 3x2, usada para transformações 2D. |
| [**estrutura FLOAT4**](float4-structure.md) | Um vetor com quatro componentes. |
| [**estrutura float4x4**](float4x4-structure.md) | Uma matriz 4x4, usada para transformações 3D. |
| [**estrutura do plano**](plane-structure.md) | Essa estrutura representa um plano usando um vetor 3D normal e um valor de distância. |
| [**estrutura de Quaternion**](quaternion-structure.md) | Um vetor dimensional de quatro dimensões, usado para representar uma rotação. |
| [**APIs de interoperabilidade DirectXMath e numéricas do Windows**](windows-numerics-and-directxmath-interop-apis.md) | Essas funções convertem os tipos [**Windows. Foundation. Numerics**](/uwp/api/Windows.Foundation.Numerics) de e para os tipos DirectXMath SIMD [XMVECTOR](../dxmath/xmvector-data-type.md) e [XMMATRIX](/windows/win32/api/directxmath/ns-directxmath-xmmatrix). |
