# <a name="common-dialog-box-types"></a>Tipos comuns de caixa de diálogo

A Biblioteca de Caixas de Diálogo Comuns fornece uma função de criação e uma estrutura para cada tipo de caixa de diálogo comum. Para usar uma caixa de diálogo comum em sua forma mais simples, chame sua função de criação e especifique um ponteiro para uma estrutura que contém valores iniciais e sinalizadores de opção. Depois de inicializar a caixa de diálogo, o procedimento da caixa de diálogo usa a estrutura para retornar informações sobre a entrada do usuário. Você também pode personalizar uma caixa de diálogo comum para atender às necessidades do seu aplicativo.

A tabela a seguir fornece uma breve descrição dos diferentes tipos de caixas de diálogo comuns e mostra a função e a estrutura usadas com cada tipo.



| Caixa de diálogo                | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Cor**<br/>      | Exibe as cores disponíveis e, opcionalmente, permite que o usuário crie cores personalizadas. O usuário pode selecionar uma cor básica ou personalizada. Use a [**função ChooseColor**](/previous-versions/windows/desktop/legacy/ms646912(v=vs.85)) e [**a estrutura CHOOSECOLOR.**](/windows/win32/api/commdlg/ns-commdlg-choosecolora-r1) Para obter mais informações, consulte [Caixa de diálogo Cor](color-dialog-box.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Localizar**<br/>       | Exibe uma caixa de diálogo na qual o usuário pode digitar a cadeia de caracteres a ser achada. O usuário também pode especificar opções de pesquisa, como a direção da pesquisa e se a pesquisa é sensível a minúsculas. Use a [**função FindText**](/windows/desktop/api/Commdlg/nf-commdlg-findtexta) e [**a estrutura FINDREPLACE.**](/windows/win32/api/commdlg/ns-commdlg-findreplacea) Para obter mais informações, consulte [Encontrar e substituir caixas de diálogo](find-and-replace-dialog-boxes.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **Fonte**<br/>       | Exibe uma caixa de diálogo na qual o usuário pode selecionar uma família de fontes e o estilo de fonte associado, tamanhos de ponto e outros atributos de fonte, como cor da fonte, sublinhado ou tachado. Use a [**função ChooseFont**](/windows/win32/api/commdlg/ns-commdlg-choosefonta) e [**a estrutura CHOOSEFONT.**](/windows/win32/api/commdlg/ns-commdlg-choosefonta) Para obter mais informações, consulte [Caixa de diálogo Fonte](font-dialog-box.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Abrir**<br/>       | Exibe uma caixa de diálogo na qual o usuário pode digitar ou selecionar o nome de um arquivo ou objeto de espaço de nome do shell a ser aberto. A caixa de diálogo inclui listas de unidades, diretórios e extensões de name-space do shell que permitem que o usuário procure o espaço de nome do shell. Ele também inclui uma lista de extensões de nome de arquivo que permite que o usuário filtre os nomes de arquivo exibidos. Use a [**função GetOpenFileName**](/windows/desktop/api/Commdlg/nf-commdlg-getopenfilenamea) e a [**estrutura OPENFILENAME.**](/windows/win32/api/commdlg/ns-commdlg-openfilenamea) Para obter mais informações, consulte [Caixas de diálogo Abrir e Salvar como](open-and-save-as-dialog-boxes.md).<br/>                                                                                                                                                                                                                                                                                     |
| **Configuração de página**<br/> | Exibe a configuração da página atual. O usuário pode selecionar opções de configuração de página, como orientação de papel, tamanho, origem e margens. Use a [**função PageSetupDlg**](/previous-versions/windows/desktop/legacy/ms646937(v=vs.85)) e a [**estrutura PAGESETUPDLG.**](/windows/win32/api/commdlg/ns-commdlg-pagesetupdlga) Para obter mais informações, consulte [Caixa de diálogo Configuração de Página](page-setup-dialog-box.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Imprimir**<br/>      | Exibe informações sobre a impressora instalada e sua configuração. O usuário pode selecionar opções de trabalho de impressão, como o intervalo de páginas a imprimir e o número de cópias e iniciar o processo de impressão. Use a [**função PrintDlg**](/previous-versions/windows/desktop/legacy/ms646940(v=vs.85)) e a [**estrutura PRINTDLG.**](/windows/win32/api/commdlg/ns-commdlg-printdlga) Para obter mais informações, consulte [Imprimir caixa de diálogo](print-dialog-box.md).<br/> Para exibir uma **folha de** propriedades Imprimir em vez de uma caixa de diálogo Imprimir, use **a** função [**PrintDlgEx**](/previous-versions/windows/desktop/legacy/ms646942(v=vs.85)) com a [**estrutura PRINTDLGEX.**](/windows/win32/api/commdlg/ns-commdlg-printdlgexa) A **página** Geral da folha de propriedades é semelhante à caixa **de diálogo** Imprimir. A folha de propriedades pode ter páginas de propriedades adicionais específicas do aplicativo e específicas do driver seguindo a **página** Geral. Para obter mais informações, consulte [Imprimir folha de propriedades](print-property-sheet.md).<br/> |
| **Substituir**<br/>    | Exibe uma caixa de diálogo na qual o usuário pode digitar a cadeia de caracteres para encontrar e a cadeia de caracteres de substituição. O usuário pode especificar opções de pesquisa, como se a pesquisa é sensível a minúsculas e opções de substituição, como o escopo de substituição. Use a [**função ReplaceText**](/windows/desktop/api/Commdlg/nf-commdlg-replacetexta) e [**a estrutura FINDREPLACE.**](/windows/win32/api/commdlg/ns-commdlg-findreplacea) Para obter mais informações, consulte [Encontrar e substituir caixas de diálogo](find-and-replace-dialog-boxes.md).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **Salvar como**<br/>    | Exibe uma caixa de diálogo na qual o usuário pode digitar ou selecionar o nome com o qual salvar um arquivo ou objeto de espaço de nome do shell. A caixa de diálogo inclui listas de unidades, diretórios e extensões de name-space do shell que permitem que o usuário procure o espaço de nome do shell. Ele também inclui uma lista de extensões de nome de arquivo que permite que o usuário filtre os nomes de arquivo exibidos. Use a [**função GetSaveFileName**](/windows/desktop/api/Commdlg/nf-commdlg-getsavefilenamea) e a [**estrutura OPENFILENAME.**](/windows/win32/api/commdlg/ns-commdlg-openfilenamea) Para obter mais informações, consulte [Caixas de diálogo Abrir e Salvar como](open-and-save-as-dialog-boxes.md).<br/>                                                                                                                                                                                                                                                                             |



 

Embora uma **caixa de diálogo** Instalação de Impressão seja disponível, ela foi superada pela caixa de diálogo **Configuração** de Página . Os aplicativos devem usar a **caixa de diálogo Configuração** de Página em vez da **caixa de diálogo Instalação** de Impressão .

Todas as caixas de diálogo comuns são modais, exceto as **caixas** de diálogo Encontrar **e** Substituir. As caixas de diálogo modais devem ser fechadas pelo usuário antes que a função usada para criar a caixa de diálogo possa retornar. As **caixas** de **diálogo** Encontrar e Substituir não têm modo; a função retorna antes que a caixa de diálogo feche. Se você  usar  as caixas de diálogo Encontrar e Substituir, também deverá usar a função [**IsDialogMessage**](/windows/desktop/api/Winuser/nf-winuser-isdialogmessagea) no loop de mensagem principal do aplicativo para garantir que essas caixas de diálogo processem corretamente a entrada do teclado, como as teclas TAB e ESC.
