# <a name="about-window-stations-and-desktops"></a>Sobre estações de janela e áreas de trabalho

Uma *estação de janela* é um objeto protegível associado a um processo e contém uma área de transferência, uma tabela Atom e um ou mais objetos da área de trabalho.

Uma *área de trabalho* é um objeto protegível contido em uma estação de janela. Uma área de trabalho tem uma superfície de exibição lógica e contém objetos de interface do usuário, como Windows, menus e ganchos.
