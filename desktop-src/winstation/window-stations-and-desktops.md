# <a name="window-stations-and-desktops"></a>Estações e desktops da janela

Windows fornece três categorias principais de objetos: interface do usuário, interface gráfica do dispositivo (GDI) e kernel. [Objetos de kernel](/windows/desktop/SysInfo/kernel-objects) são protegíveis, enquanto [objetos de usuário](/windows/desktop/SysInfo/user-objects) e [objetos GDI](/windows/desktop/SysInfo/gdi-objects) não são. Portanto, para fornecer segurança adicional, os objetos da interface do usuário são gerenciados usando estações de janela e áreas de trabalho, que são objetos protegíveis.

Para obter mais informações, consulte estes tópicos:

-   [Sobre estações de janela e áreas de trabalho](about-window-stations-and-desktops.md)
-   [Estação de janela e referência de área de trabalho](window-station-and-desktop-reference.md)
