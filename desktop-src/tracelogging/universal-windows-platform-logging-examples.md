# <a name="universal-windows-platform-logging-example"></a>Exemplo de registro em log Windows plataforma universal

Este exemplo mostra como usar as APIs de log no Windows. Namespace Foundation.Diagnostics, incluindo LoggingChannel, LoggingActivity, LoggingSession e FileLoggingSession. Essas classes são projetadas para o log de diagnóstico em um Windows aplicativo. Essas APIs foram adicionadas Windows 8.1.

As APIs LoggingChannel e LoggingActivity foram estendidas em Windows 10 para dar suporte à escrita de eventos complexos usando a codificação de eventos TraceLogging.

O exemplo de log Windows Plataforma Universal pode ser baixado do [GitHub](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/Logging).
