<dl> <dt>

<span id="_Windows_Interface_Guidelines_for_Software_Design"></span><span id="_windows_interface_guidelines_for_software_design"></span><span id="_WINDOWS_INTERFACE_GUIDELINES_FOR_SOFTWARE_DESIGN"></span>*diretrizes Windows interface para design de software*
</dt> <dd>

(Microsoft Press, 1995; ISBN 1-55615-679-0)

Este livro explica todas as principais regras de interface do usuário a serem seguidas, com muitos exemplos de como estabelecer caixas de diálogo, assistentes, barras de ferramentas e menus de contexto.

</dt> <dd>

(AP Professional, 1994; ISBN 0125184069)

Este livro se concentra nas técnicas para medir efetivamente a usabilidade de uma interface. Ele explica muitas técnicas de teste diferentes para comparar ideias de design diferentes, incluindo sugestões para manter os custos de teste de usabilidade ino mesmo. Este livro aborda técnicas de análise que podem ser aplicadas a sites, páginas da Web ou qualquer tipo de software. É ótimo para as fases iniciais de aprendizado sobre os usuários e as fases intermediárias do teste de iteração e usabilidade.

Este livro discute os princípios éticos no trabalho que torna as interfaces tão difíceis de usar. Recomendado para qualquer pessoa interessada em interfaces do usuário e que queira ter uma ampla compreensão dos problemas envolvidos.
