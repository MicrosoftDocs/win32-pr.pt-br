<span data-ttu-id="8bfb1-180">Em casos raros, a consistência não fornece nenhum benefício ao usuário e pode até mesmo prejudicar a experiência do usuário.</span><span class="sxs-lookup"><span data-stu-id="8bfb1-180">In rare cases, consistency provides no benefit to the user and can even degrade the user experience.</span></span> <span data-ttu-id="8bfb1-181">Não torne a interface do usuário consistente se essa consistência prejudica a capacidade de realizar uma tarefa.</span><span class="sxs-lookup"><span data-stu-id="8bfb1-181">Do not make the UI consistent if that consistency impairs the ability to accomplish a task.</span></span> <span data-ttu-id="8bfb1-182">A consistência em si não garante a usabilidade.</span><span class="sxs-lookup"><span data-stu-id="8bfb1-182">Consistency in itself does not guarantee usability.</span></span> <span data-ttu-id="8bfb1-183">É um erro imaginar que a consistência na interface resultará em um bom design.</span><span class="sxs-lookup"><span data-stu-id="8bfb1-183">It is a mistake to think that consistency in the interface will lead to good design.</span></span>

<span data-ttu-id="8bfb1-184">Por exemplo, a interface do usuário do jogo de vídeo normalmente é muito específica para o tipo de jogo.</span><span class="sxs-lookup"><span data-stu-id="8bfb1-184">For example, video game UI is typically very specific to the kind of game.</span></span> <span data-ttu-id="8bfb1-185">A tentativa de criar uma interface de usuário genérica que funciona bem para dois jogos especializados, um que exija uma roda de direcionamento e outro que funcione melhor com um joystick e botões, provavelmente não será bem-sucedido para qualquer jogo.</span><span class="sxs-lookup"><span data-stu-id="8bfb1-185">Trying to design a generic UI that works well for two specialized games, one that requires a steering wheel and another that works best with a joystick and buttons, is likely not going to be successful for either game.</span></span> <span data-ttu-id="8bfb1-186">Na melhor das hipóteses, é provável que o primeiro seja o que não é bom para ambos.</span><span class="sxs-lookup"><span data-stu-id="8bfb1-186">At best a middle ground is likely to be achieved that is not good for either.</span></span>
