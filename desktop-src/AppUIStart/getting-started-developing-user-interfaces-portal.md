# <a name="getting-started-developing-user-interfaces-for-windows-applications"></a>Introdução desenvolver Interfaces do usuário para aplicativos Windows

## <a name="purpose"></a>Finalidade

as seções a seguir oferecem orientação geral aos desenvolvedores que estão projetando, implementando e testando a interface do usuário de um aplicativo Windows.

Além dos princípios básicos de design da interface do usuário, várias recomendações e sugestões são fornecidas para ajudar os desenvolvedores a fornecer uma experiência do usuário que seja tão simples, eficiente e agradável quanto possível.

> [!Note]  
> Essas diretrizes não devem ser abrangentes e estão sujeitas ao escopo e à funcionalidade específicos de um aplicativo. para obter diretrizes mais abrangentes, consulte as [diretrizes de interação da experiência do usuário Windows](../uxguide/guidelines.md).
