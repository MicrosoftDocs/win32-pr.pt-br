# <a name="directdraw"></a>Acelera

Esta seção contém informações sobre a programação com o componente do DirectDraw da API (interface de programação de aplicativo) do DirectX.

O DirectDraw não é mais recomendado para uso. Com o lançamento do Direct3D 9,0, todas as funcionalidades bidimensionais estão contidas no Direct3D, suas funções auxiliares associadas no D3DX e o DirectX 11 Technology Direct2D. No entanto, a documentação de referência do DirectDraw ainda está disponível nesta seção.

## <a name="in-this-section"></a>Nesta seção

Esta seção contém a documentação de [referência do DirectDraw](directdraw-reference.md) .
