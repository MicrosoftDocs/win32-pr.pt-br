## <a name="establishing-the-order-in-logical-navigation"></a><span data-ttu-id="14d2f-125">Estabelecendo a ordem na navegação lógica</span><span class="sxs-lookup"><span data-stu-id="14d2f-125">Establishing the Order in Logical Navigation</span></span>

<span data-ttu-id="14d2f-126">Na navegação lógica, os desenvolvedores que criam os objetos estabelecem as relações entre eles.</span><span class="sxs-lookup"><span data-stu-id="14d2f-126">In logical navigation, the developers who design the objects establish the relationships between them.</span></span> <span data-ttu-id="14d2f-127">A navegação lógica é mais subjetiva do que a navegação espacial.</span><span class="sxs-lookup"><span data-stu-id="14d2f-127">Logical navigation is more subjective than spatial navigation.</span></span> <span data-ttu-id="14d2f-128">Além disso, a ordem na navegação lógica não é a mesma que a ordem usada com as IDs filho.</span><span class="sxs-lookup"><span data-stu-id="14d2f-128">Also, the order in logical navigation is not the same as the order used with child IDs.</span></span>

<span data-ttu-id="14d2f-129">Para objetos que têm locais de tela, os desenvolvedores de servidor devem estabelecer a ordem de navegação da maneira que a maioria dos usuários consideraria lógica.</span><span class="sxs-lookup"><span data-stu-id="14d2f-129">For objects that have screen locations, server developers should establish the navigation order in the way that most users would consider logical.</span></span> <span data-ttu-id="14d2f-130">Em países/regiões de língua inglesa, por exemplo, isso significa uma ordenação da esquerda para a direita, de cima para baixo.</span><span class="sxs-lookup"><span data-stu-id="14d2f-130">In English-speaking countries/regions, for instance, this means a left-to-right, top-to-bottom ordering.</span></span>
