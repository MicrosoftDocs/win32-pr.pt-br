# <a name="built-in-accessibility-features"></a>Recursos de acessibilidade internos

Os recursos de acessibilidade internos são extensões para o sistema operacional. Esses recursos afetam o comportamento do sistema, independentemente de quais aplicativos estão em execução.

O Win32 dá suporte aos seguintes recursos internos de acessibilidade.



| Recurso       | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| AccessTimeout | Permite que um usuário especifique um intervalo de tempo limite após o qual os recursos de acessibilidade de todo o sistema são automaticamente desabilitados. O recurso AccessTimeout destina-se a computadores compartilhados por vários usuários com preferências diferentes. Cada pessoa pode usar teclas de acesso ou a central de facilidade de Access no painel de controle para habilitar os recursos preferenciais. O tempo limite de acessibilidade é o período de tempo que deve passar sem entrada de teclado e mouse antes que o sistema operacional desative automaticamente os recursos de acessibilidade. Os recursos de acessibilidade afetados pelo tempo limite são as teclas de filtragem, as teclas de aderência, os superativações e as alternâncias e o [alto contraste](high-contrast-parameter.md). A estrutura [**ACCESSTIMEOUT**](/windows/win32/api/winuser/ns-winuser-accesstimeout) define os parâmetros para esse recurso. |
| Ativar    | Habilita o controle das propriedades do teclado, como a quantidade de tempo antes que um pressionamento de tecla seja aceito como entrada e a quantidade de tempo antes que um pressionamento de tecla comece a ser repetido. O recurso de teclas de filtragem também fornece comentários sonoros quando uma tecla é pressionada e quando é aceita como entrada. A estrutura de [**teclas de filtragem**](/windows/win32/api/winuser/ns-winuser-filterkeys) define os parâmetros para esse recurso.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| Las     | Permite que o usuário controle o ponteiro do mouse usando o teclado numérico. A estrutura de [**teclas para mouse**](/windows/win32/api/winuser/ns-winuser-mousekeys) define os parâmetros para esse recurso.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Seria    | Permite que o usuário forneça entrada usando um dispositivo de entrada alternativo conectado à porta serial do computador. Esse recurso foi projetado para usuários que não conseguem usar o teclado e o mouse padrão do computador. A estrutura de [**dispositivos seriais**](/windows/win32/api/winuser/ns-winuser-serialkeysa) define os parâmetros para esse recurso.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| Som   | Exibe um sinal visual quando um som é gerado por um aplicativo baseado no Microsoft Win32 ou um aplicativo do Microsoft MS-DOS em execução em uma janela. A estrutura do [**sinalizador de som**](/windows/win32/api/winuser/ns-winuser-soundsentrya) define os parâmetros para esse recurso, incluindo o endereço da função de retorno de chamada [*SoundSentryProc*](/previous-versions/windows/desktop/legacy/dd373647(v=vs.85)) .                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Aderência    | Permite que o usuário digite combinações de teclas, como CTRL + ALT + DEL, em sequência, em vez de ao mesmo tempo. A estrutura de [**teclas de aderência**](/windows/win32/api/winuser/ns-winuser-stickykeys) define os parâmetros para esse recurso.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Ativar    | Fornece comentários sonoros quando o usuário ativa ou desativa as chaves CAPS LOCK, NUM LOCK ou SCROLL LOCK. A estrutura de [**alternâncias**](/windows/win32/api/winuser/ns-winuser-togglekeys) define os parâmetros para esse recurso.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |



 

A função [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) fornece a interface de programação para os recursos de acessibilidade internos. Essa função é útil em aplicativos do painel de controle ou outros programas que permitem que o usuário personalize o ambiente. A função **SystemParametersInfo** pode executar as seguintes tarefas:

-   Consulte o sistema para determinar a disponibilidade e o estado atual de um recurso de acessibilidade interno especificado.
-   Habilite um recurso e especifique parâmetros que controlam seu comportamento.
-   Desabilitar um recurso.
-   Habilite uma tecla de acesso que permita ao usuário habilitar ou desabilitar um recurso.

Para cada um dos recursos internos, o Win32 define uma estrutura que os aplicativos usam com o [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) para consultar ou definir os parâmetros do recurso.

Um aplicativo pode usar o [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) para habilitar ou desabilitar um recurso diretamente. Para alguns dos recursos, o **SystemParametersInfo** pode habilitar uma tecla de atalho ou uma combinação de teclas para que o usuário possa alternar o estado de um recurso. Não há teclas de acesso para os recursos de dispositivos seriais ou sinalizador de som ou para o parâmetro de mostrar sons. Os recursos de acessibilidade a seguir têm teclas de acesso.



| Recurso                                           | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Ativar                                        | Mantenha pressionada a tecla SHIFT direita por oito segundos para ativar ou desativar o recurso de teclas de filtragem. Manter pressionada a tecla SHIFT direita por 12 segundos permite que as teclas de filtragem usem os valores mais conservadores para o membro **iBounceMSec** da estrutura de [**teclas de filtragem**](/windows/win32/api/winuser/ns-winuser-filterkeys) . Manter pressionada a tecla SHIFT direita por 16 segundos permite que as teclas de filtragem usem os valores mais conservadores para o membro **iWaitMSec** da estrutura de **teclas de filtragem** . |
| Las                                         | Pressione ALT esquerda + SHIFT esquerda + NUM LOCK.                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Aderência                                        | Pressione a tecla SHIFT 5 vezes.                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Ativar                                        | Mantenha pressionada a tecla NUM LOCK por 5 segundos.                                                                                                                                                                                                                                                                                                                                                                                                                |
| [Modo de alto contraste](high-contrast-parameter.md) | Pressione ALT esquerda + SHIFT esquerda + tela de impressão.                                                                                                                                                                                                                                                                                                                                                                                                              |
