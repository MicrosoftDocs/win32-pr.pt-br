# <a name="dynamic-annotation"></a>Anotação dinâmica

A anotação dinâmica estava disponível anteriormente apenas para servidores e clientes Microsoft Acessibilidade Ativa. Agora a técnica está disponível para todos os clientes de automação da interface do usuário, bem como algumas implementações do servidor MSAA e do provedor de automação da interface do usuário. Além disso, as anotações feitas aos controles personalizados usando a anotação dinâmica (todas as três técnicas) são acessíveis aos clientes de automação da interface do usuário.

Para obter mais informações, consulte [API de anotação dinâmica](dynamic-annotation-api.md).
