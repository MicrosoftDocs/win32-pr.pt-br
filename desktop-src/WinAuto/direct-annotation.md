# <a name="direct-annotation"></a>Anotação direta

Esta seção fornece informações sobre como usar a anotação direta.

## <a name="summary"></a>Resumo

A anotação direta é a forma mais simples de anotação dinâmica. Ele é usado para substituir o valor de um ou um número limitado de propriedades de um elemento acessível. Ele é mais útil para elementos acessíveis cujas propriedades anotadas não dependem do estado do controle e não exigem o suporte adicional fornecido pela [anotação de mapa de valor](value-map-annotation.md) e anotação de [servidor](server-annotation.md).

## <a name="when-to-use-this-technique"></a>Quando usar essa técnica

Use essa técnica quando desejar substituir o valor de um ou um número limitado de propriedades para um elemento acessível.

Para obter mais informações, consulte [usando anotação direta](using-direct-annotation.md).
