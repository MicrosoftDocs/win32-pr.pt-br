# <a name="active-accessibility-objects"></a>Objetos Acessibilidade Ativa

Na terminologia do Microsoft Acessibilidade Ativa, há *objetos acessíveis* e *elementos simples*. Embora a maioria dos aplicativos contenham ambos, os objetos acessíveis são mais comuns do que os elementos simples. Esta seção define e discute objetos acessíveis e elementos simples e fornece exemplos de interface do usuário apropriados.
