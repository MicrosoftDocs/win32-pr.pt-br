# <a name="active-accessibility-objects"></a>Acessibilidade Ativa objetos

Na Microsoft Active Accessibility terminologia, há objetos *acessíveis e* *elementos simples.* Embora a maioria dos aplicativos contenha ambos, os objetos acessíveis são mais comuns do que elementos simples. Esta seção define e discute objetos acessíveis e elementos simples e fornece exemplos de interface do usuário apropriados.
