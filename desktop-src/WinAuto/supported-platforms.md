# <a name="supported-platforms"></a>Plataformas compatíveis

o Microsoft Acessibilidade Ativa dá suporte à maioria das plataformas da Microsoft e está incluído como parte do sistema operacional Windows a partir do Windows XP. Se você não souber qual versão do Microsoft Acessibilidade Ativa está executando, consulte [qual versão do acessibilidade ativa está instalada no momento?](which-version-of-active-accessibility-is-currently-installed.md)

Se você não tiver a versão mais recente do Microsoft Acessibilidade Ativa, use o [microsoft Acessibilidade Ativa 2,0 redistribuível Kit (RDK)](https://www.microsoft.com/downloads/details.aspx?familyid=9b14f6e1-888a-4f1d-b1a1-da08ee4077df&displaylang=en) para atualizar. Este RDK contém todos os principais arquivos de sistema necessários para incorporar a tecnologia de Acessibilidade Ativa da Microsoft em aplicativos cliente e servidor. ele dá suporte a Windows 98, Windows NT 4,0 com Service Pack 6 (SP6) e posterior, Windows 2000, Windows Millennium Edition (Windows Me), Windows XP e Windows Server 2003.

Para instalar o Microsoft Acessibilidade Ativa RDK, consulte o [centro de desenvolvedores de acessibilidade da Microsoft no MSDN](/windows/apps/accessibility).
