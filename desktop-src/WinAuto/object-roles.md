# <a name="object-roles"></a>Funções de objeto

Este tópico descreve os valores constantes usados para descrever as funções de vários objetos de interface do usuário em um aplicativo. As constantes de função são definidas no arquivo OleAcc. h.

Você deve usar apenas as constantes de função listadas listadas aqui; Não adicione funções ou funções personalizadas que não são predefinidas.

Antes de usar essas funções de objeto, os desenvolvedores de aplicativos cliente devem usar a ferramenta [inspecionar](inspect-objects.md) para verificar se as funções de objeto estão sendo usadas pelos elementos da interface do usuário.

Para recuperar a função de um objeto, os clientes chamam o método [**IAccessible:: get \_ accRole**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accrole) , que deve retornar um dos valores descritos na tabela. Para recuperar uma cadeia de caracteres localizada que descreve a função do objeto, os clientes chamam a função [**GetRoleText**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta) com o valor da função. As cadeias de caracteres localizadas para os valores de função estão no arquivo oleaccrc.dll.
