# <a name="microsoft-active-accessibility"></a>Acessibilidade Ativa da Microsoft

## <a name="purpose"></a>Finalidade

O Microsoft Acessibilidade Ativa é uma tecnologia baseada em Component Object Model (COM) que melhora a maneira como os recursos de acessibilidade funcionam com aplicativos executados no Microsoft Windows. Ele fornece bibliotecas de vínculo dinâmico que são incorporadas ao sistema operacional, bem como uma interface COM e elementos de API que fornecem métodos confiáveis para expor informações sobre elementos de interface do usuário.

## <a name="where-applicable"></a>Quando aplicável

ao usar o Microsoft Acessibilidade Ativa e as práticas de design acessíveis a seguir, os desenvolvedores podem fazer com que os aplicativos em execução Windows mais acessíveis a muitas pessoas com deficiências visuais, auditivas ou motoras.

## <a name="developer-audience"></a>Público de desenvolvedores

o microsoft Acessibilidade Ativa destina-se principalmente a desenvolvedores de Visual Basic C, C++ e Microsoft. Em geral, os desenvolvedores precisam de um nível moderado de compreensão sobre objetos e interfaces COM, bem como sobre Unicode.

## <a name="run-time-requirements"></a>Requisitos de tempo de execução

o suporte completo para o Microsoft Acessibilidade Ativa é integrado ao Windows XP e ao Windows Server 2003. o Microsoft Acessibilidade Ativa também tem suporte no Windows NT 4,0 com Service Pack 6 (SP6) e posterior e Windows 98.
