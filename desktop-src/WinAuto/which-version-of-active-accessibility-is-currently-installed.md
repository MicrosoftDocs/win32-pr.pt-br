# <a name="determine-the-installed-version-of-active-accessibility"></a>Determinar a versão instalada do Acessibilidade Ativa

Para descobrir qual versão do Microsoft Active Accessibility (MSAA) está instalada em seu sistema, verifique a versão do arquivo Oleacc.dll na pasta system32 no disco rígido. A tabela a seguir mapeia a Microsoft Active Accessibility para a versão Oleacc.dll dados.

| Acessibilidade Ativa versão     | Oleacc.dll versão                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1.0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1,3                              | 4.02.3100.0                                                                              |
| 2,0                              | 5.0.0.0                                                                                  |
| 3.0 (API Windows Automação 3.0) | 7.0.x.y, em *que x* é o número de build e *y* é o número QFE (Correção Rápida Engineering) |
