# <a name="determine-the-installed-version-of-active-accessibility"></a>Determinar a versão instalada do Acessibilidade Ativa

Para descobrir qual versão do Microsoft Acessibilidade Ativa (MSAA) está instalada no seu sistema, você deve verificar a versão do arquivo Oleacc.dll na pasta system32 do disco rígido. A tabela a seguir mapeia a versão do Microsoft Acessibilidade Ativa para a versão Oleacc.dll.

| Versão do Acessibilidade Ativa     | Versão do Oleacc.dll                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1,0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1,3                              | 4.02.3100.0                                                                              |
| 2,0                              | 5.0.0.0                                                                                  |
| 3,0 (API de automação do Windows 3,0) | 7.0. x. y, em que *x* é o número de Build e *y* é o número QFE (Quick Fix Engineering) |
