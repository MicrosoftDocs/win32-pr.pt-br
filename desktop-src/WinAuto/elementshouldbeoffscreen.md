# <a name="elementshouldbeoffscreen"></a>ElementShouldBeOffScreen

## <a name="text"></a>Texto

O elemento deve estar fora da tela

## <a name="type"></a>Type

Aviso

## <a name="description"></a>Descrição

O retângulo delimitador do elemento é recortado pelo retângulo delimitador de seu elemento pai. O elemento deve ser ignorado e considerado fora dos limites da área de trabalho.
