# <a name="check-box-msaa-ui-element-reference"></a>Caixa de seleção (Referência de elemento de interface do usuário MSAA)

> [!Note]  
> Este tópico descreve objetos **check-box** para fins de referência de elemento de interface do usuário da MSAA. Como criar objetos **de Caixa** de Seleção em várias estruturas de interface do usuário não é descrito aqui. Consulte a documentação de referência de API para a estrutura de interface do usuário que você está usando.

 

Caixas de seleção são usadas para habilitar ou desabilitar um ou mais recursos ou opções de um conjunto, geralmente dentro de uma caixa de diálogo. Normalmente, uma caixa de seleção contém uma caixa pequena com texto ao lado. Quando uma opção é selecionada, uma marca de seleção é exibida na caixa.

O nome da classe de janela para uma caixa de seleção é "BUTTON".
