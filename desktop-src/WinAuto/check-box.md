# <a name="check-box-msaa-ui-element-reference"></a>Caixa de seleção (referência de elemento de interface do usuário do MSAA)

> [!Note]  
> Este tópico descreve os objetos da **caixa de seleção** para fins de referência de elemento da interface do usuário do MSAA. A criação de objetos de **caixa de seleção** em várias estruturas de interface do usuário não é descrita aqui. Consulte a documentação de referência da API para a estrutura de interface do usuário que você está usando.

 

As caixas de seleção são usadas para habilitar ou desabilitar um ou mais recursos ou opções de um conjunto, geralmente dentro de uma caixa de diálogo. Normalmente, uma caixa de seleção contém uma pequena caixa com texto adjacente. Quando uma opção é selecionada, uma marca de seleção é exibida na caixa.

O nome da classe de janela para uma caixa de seleção é "BUTTON".
