# <a name="structures-and-data-types"></a>Estruturas e tipos de dados

Esta seção contém informações sobre as estruturas e os tipos de dados usados com o Microsoft Acessibilidade Ativa.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                             | Descrição                                                                                                                                                              |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSAAMENUINFO**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | Usado por desenvolvedores de servidor para expor os nomes dos itens de menu extraídos pelo proprietário.<br/>                                                                                      |
| [**HWINEVENTHOOK**](hwineventhook.md)<br/> | Usado para declarar uma função de gancho de evento de janela.<br/>                                                                                                                 |
| [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85))<br/>   | Usado para declarar um ponteiro para uma função de retorno de chamada definida pelo aplicativo que o sistema chama em resposta a eventos gerados por um objeto acessível.<br/> |
