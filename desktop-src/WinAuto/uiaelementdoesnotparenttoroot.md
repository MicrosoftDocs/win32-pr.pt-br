# <a name="uiaelementdoesnotparenttoroot"></a>UiaElementDoesNotParentToRoot

## <a name="text"></a>Texto

A cadeia pai do elemento não vai para o Elemento Raiz

## <a name="type"></a>Digite

Aviso

## <a name="description"></a>Descrição

A navegação do elemento determinado para a raiz da árvore da interface do usuário é interrompida.

Esse problema pode causar resultados imprevisíveis para um aplicativo que tenta navegar pela árvore de interface do usuário.

## <a name="possible-causes"></a>Possíveis causas

Implementação incorreta da hierarquia filho pai da árvore de interface do usuário.
