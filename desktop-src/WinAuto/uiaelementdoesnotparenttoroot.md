# <a name="uiaelementdoesnotparenttoroot"></a>UiaElementDoesNotParentToRoot

## <a name="text"></a>Texto

A cadeia pai do elemento não vai para o elemento raiz

## <a name="type"></a>Type

Aviso

## <a name="description"></a>Descrição

A navegação do elemento fornecido para a raiz da árvore de interface do usuário está quebrada.

Esse problema pode causar resultados imprevisíveis para um aplicativo que tenta navegar na árvore de interface do usuário.

## <a name="possible-causes"></a>Possíveis causas

Implementação incorreta da hierarquia pai filho da árvore de interface do usuário.
