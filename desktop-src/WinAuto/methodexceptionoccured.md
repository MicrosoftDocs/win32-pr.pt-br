# <a name="methodexceptionoccured"></a>MethodExceptionOccured

## <a name="text"></a>Texto

Ocorreu uma exceção \\ " {0} \\ "

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

Uma exceção inesperada ou HRESULT foi encontrado. Algumas exceções/HRESULTs são válidos, mas, nesse caso, os resultados não são esperados. Esse erro deve ser corrigido.
