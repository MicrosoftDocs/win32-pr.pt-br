# <a name="cc-developers-guide"></a>Guia do desenvolvedor do C/C++

Esta seção fornece informações sobre como Microsoft Active Accessibility funciona da perspectiva de um desenvolvedor de aplicativos C ou C++. Ele aborda problemas específicos para desenvolvedores de cliente e servidor.

Observe que um *cliente é* um auxílio de acessibilidade ou outro aplicativo que usa Microsoft Active Accessibility para obter informações sobre elementos de interface do usuário. Um *servidor* é um aplicativo que usa Microsoft Active Accessibility para fornecer informações sobre elementos de interface do usuário.
