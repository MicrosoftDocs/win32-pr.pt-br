# <a name="cc-developers-guide"></a>Guia do desenvolvedor de C/C++

Esta seção fornece informações sobre como a Microsoft Acessibilidade Ativa trabalha da perspectiva de um desenvolvedor de aplicativos C ou C++. Ele aborda problemas específicos de desenvolvedores de cliente e servidor.

Observe que um *cliente* é um auxílio de acessibilidade ou outro aplicativo que usa o Microsoft acessibilidade ativa para obter informações sobre elementos da interface do usuário. Um *servidor* é um aplicativo que usa o Microsoft acessibilidade ativa para fornecer informações sobre elementos da interface do usuário.
