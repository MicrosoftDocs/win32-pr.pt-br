# <a name="labeling-owner-drawn-controls"></a><span data-ttu-id="88f02-103">Rotulando controles de Owner-Drawn</span><span class="sxs-lookup"><span data-stu-id="88f02-103">Labeling Owner-Drawn Controls</span></span>

<span data-ttu-id="88f02-104">Os desenvolvedores de aplicativos devem definir um rótulo, também conhecido como texto de legenda, para todos os controles desenhados pelo proprietário, mesmo se o rótulo não estiver visível na tela.</span><span class="sxs-lookup"><span data-stu-id="88f02-104">Application developers should define a label, also known as caption text, for all owner-drawn controls, even if the label is not visible on the screen.</span></span> <span data-ttu-id="88f02-105">O rótulo deve vir imediatamente antes do controle que está rotulando e deve estar no próximo item na ordem de tabulação.</span><span class="sxs-lookup"><span data-stu-id="88f02-105">The label must come immediately before the control that it is labeling, and must be in the next item in the tab order.</span></span>
