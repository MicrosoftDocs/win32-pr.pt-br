# <a name="labeling-owner-drawn-controls"></a>Rotulando controles de Owner-Drawn

Os desenvolvedores de aplicativos devem definir um rótulo, também conhecido como texto de legenda, para todos os controles desenhados pelo proprietário, mesmo se o rótulo não estiver visível na tela. O rótulo deve vir imediatamente antes do controle que está rotulando e deve estar no próximo item na ordem de tabulação.

Por exemplo, se você definir um rótulo ao criar um botão de desenho de proprietário no qual um gráfico é exibido em vez do texto de legenda normal, os aplicativos cliente obterão o rótulo e o usarão para identificar o controle.
