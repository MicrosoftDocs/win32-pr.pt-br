# <a name="unexpectedtesterror"></a>UnexpectedTestError

## <a name="text"></a>Texto

Exceção:( {0} )

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

Uma exceção não relacionada às verificações foi gerada por AccChecker. Para relatar o problema, consulte a caixa de diálogo AccChecker **sobre** para obter informações de contato.
