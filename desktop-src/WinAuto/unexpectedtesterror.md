# <a name="unexpectedtesterror"></a>UnexpectedTestError

## <a name="text"></a>Texto

Exceção :( {0} )

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

Uma exceção não relacionada às verificações foi apurada pelo AccChecker. Para relatar o problema, consulte a caixa de diálogo AccChecker **Sobre** para obter informações de contato.
