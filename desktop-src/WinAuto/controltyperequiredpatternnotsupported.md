# <a name="controltyperequiredpatternnotsupported"></a>ControlTypeRequiredPatternNotSupported

## <a name="text"></a>Texto

Elemento com um ControlType de {0} deve dar {1} suporte, mas esse elemento não

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

O suporte de Automação da Interface do Usuário elemento não está em conformidade com a especificação Automação da Interface do Usuário porque o elemento deve dar suporte à lista fornecida de padrões de controle, mas não o faz. Como resultado, esse elemento pode não ser exposto corretamente a tecnologias adaptativas, o que pode ter um impacto sério na experiência do usuário.

## <a name="possible-causes"></a>Possíveis causas

-   Implementação Automação da Interface do Usuário incompleta.
-   A propriedade ControlType não está definida corretamente.
