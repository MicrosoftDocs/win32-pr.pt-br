# <a name="tabbedtoinfo"></a>TabbedToInfo

## <a name="text"></a>Texto

**Mensagem 1**

Descartando evento de foco extra

**Mensagem 2**

Voltando ao início

## <a name="type"></a>Tipo

Informativa

## <a name="description"></a>Descrição

**Mensagem 1**

Eventos de vários focos foram acionados por um único elemento ao usar a navegação de teclado padrão (TAB ou Shift + Tab). Os eventos falsos são descartados.

**Mensagem 2**

Foco deslocado de volta para o elemento inicial na ordem de tabulação. A Tabulação foi finalizada neste elemento porque a ordem de tabulação aparece circular.
