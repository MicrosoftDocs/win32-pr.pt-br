# <a name="getting-started-microsoft-active-accessibility"></a>Introdução (Microsoft Acessibilidade Ativa)

O Microsoft Acessibilidade Ativa é um conjunto de interfaces de Component Object Model (COM) e elementos de API que fornecem uma maneira confiável de expor e coletar informações sobre elementos de interface do usuário baseados no Microsoft Windows e conteúdo da Web. Usando essas informações, os fornecedores de tecnologia assistencial podem representar a interface do usuário em formatos alternativos, como fala ou Braille, e os aplicativos de controle e comando de voz podem manipular remotamente a interface. O Microsoft Acessibilidade Ativa conta com a tecnologia Windows e pode ser usado em conjunto somente com controles baseados no Windows e outros aplicativos do Windows.
