# <a name="getting-started-microsoft-active-accessibility"></a>Ponto de Partida (Microsoft Active Accessibility)

Microsoft Active Accessibility é um conjunto de interfaces Component Object Model (COM) e elementos de API que fornecem uma maneira confiável de expor e coletar informações sobre elementos de interface do usuário baseados no Microsoft Windows e conteúdo da Web. Usando essas informações, os fornecedores de tecnologia adaptativa podem representar a interface do usuário em formatos alternativos, como fala ou Voicele, e os aplicativos de comando e controle de voz podem manipular remotamente a interface. Microsoft Active Accessibility se baseia na Windows e pode ser usado em conjunto apenas com controles baseados em Windows e outros Windows aplicativos.
