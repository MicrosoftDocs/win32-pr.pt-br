# <a name="duplicateaccesskey"></a>DuplicateAccessKey

## <a name="text"></a>Texto

Há um elemento de automação irmão que tem a mesma chave de acesso/chave de aceleração

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

A chave de acesso/aceleração atribuída ao elemento também é atribuída a outro elemento no destino de verificação.

Esse problema causa problemas para as pessoas que dependem do teclado para navegação porque vários elementos podem ser invocados ou o foco do ciclo usando o mesmo atalho ou tecla de aceleração.
