# <a name="duplicateaccesskey"></a>DuplicateAccessKey

## <a name="text"></a>Texto

Há um elemento de automação irmão que tem a mesma chave de acesso/tecla de acelerador

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

A chave de acesso/acelerador atribuída ao elemento também é atribuída a outro elemento no destino de verificação.

Esse problema causa problemas para pessoas que dependem do teclado para navegação porque vários elementos podem ser invocados ou o foco do ciclo usando a mesma tecla de atalho ou acelerador.
