# <a name="windows-accessibility-api-overview"></a>Windows Visão geral da API de acessibilidade

esta seção fornece visões gerais de alto nível e referência de API detalhada para a api de automação do microsoft Windows 3,0, que inclui a implementação de Windows da [especificação de automação da interface do usuário](./uiauto-specandcommunitypromise.md)da microsoft e o microsoft Acessibilidade Ativa (recurso herdado introduzido no Windows 95 como um suplemento de plataforma).

Descrevemos as semelhanças e diferenças entre o Microsoft Acessibilidade Ativa e a automação da interface do usuário, os componentes e recursos que permitem que as duas tecnologias trabalhem em conjunto e fornecem diretrizes para escolher qual tecnologia implementar para seu cenário específico.
