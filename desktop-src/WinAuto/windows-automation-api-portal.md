# <a name="windows-accessibility-api-overview"></a>Visão geral da API de acessibilidade do Windows

Esta seção fornece visões gerais de alto nível e referência de API detalhada para a API de automação do Microsoft Windows 3,0, que inclui a implementação do Windows da [especificação de automação da interface do usuário](./uiauto-specandcommunitypromise.md)da Microsoft e o Microsoft acessibilidade ativa (recurso herdado introduzido no Windows 95 como um suplemento de plataforma).

Descrevemos as semelhanças e diferenças entre o Microsoft Acessibilidade Ativa e a automação da interface do usuário, os componentes e recursos que permitem que as duas tecnologias trabalhem em conjunto e fornecem diretrizes para escolher qual tecnologia implementar para seu cenário específico.
