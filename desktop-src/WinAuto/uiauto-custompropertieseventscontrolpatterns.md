# <a name="custom-properties-events-and-control-patterns"></a>Propriedades personalizadas, eventos e padrões de controle

A API de automação da interface do usuário da Microsoft especifica um conjunto principal predefinido de propriedades, padrões de controle e eventos. No entanto, os aplicativos não estão limitados ao uso dessas especificações predefinidas. Os recursos de extensibilidade de automação da interface do usuário permitem que terceiros introduzam Propriedades, eventos e padrões de controle acordados, personalizados e mutuamente, para dar suporte a novos elementos de interface do usuário e cenários de aplicativos. Os clientes e provedores de automação da interface do usuário podem começar a usar propriedades personalizadas, eventos e padrões de controle imediatamente sem exigir que a estrutura de automação da interface do usuário principal seja atualizada.
