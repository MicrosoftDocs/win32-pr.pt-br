# <a name="header-and-lib-files"></a>Arquivos de cabeçalho e lib

Os arquivos de cabeçalho e biblioteca a seguir dão suporte ao desenvolvimento de aplicativos do Microsoft Acessibilidade Ativa.



| Arquivo         | Descrição                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| WinUser. h    | Arquivo de cabeçalho C e C++ para usar controles do Windows                                                                                                                             |
| OleAcc. h     | Arquivo de cabeçalho C e C++ contendo definições necessárias para clientes e servidores do Microsoft Acessibilidade Ativa                                                                    |
| Winver. h     | Arquivo de cabeçalho C e C++ para informações de versão                                                                                                                                |
| OleAcc. lib   | Importar biblioteca para Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Biblioteca de vínculo dinâmico que fornece o tempo de execução da Microsoft Acessibilidade Ativa, gerencia solicitações de clientes do Microsoft Acessibilidade Ativa e proxies de elementos comuns da interface do usuário |
| Oleaccrc.dll | Biblioteca de vínculo dinâmico de recursos multilíngues usada pelo Oleacc.dll                                                                                                                |
