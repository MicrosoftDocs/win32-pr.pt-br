# <a name="header-and-lib-files"></a>Arquivos de header e lib

Os arquivos de biblioteca e de header a seguir suportam o desenvolvimento de Microsoft Active Accessibility aplicativos.



| Arquivo         | Descrição                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser.h    | Arquivo de header C e C++ para usar controles Windows C                                                                                                                             |
| Oleacc.h     | Arquivo de header C e C++ contendo definições necessárias para Microsoft Active Accessibility clientes e servidores                                                                    |
| Winver.h     | Arquivo de header C e C++ para obter informações de versão                                                                                                                                |
| Oleacc.lib   | Biblioteca de importação para Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Biblioteca de vínculo dinâmico que fornece Microsoft Active Accessibility tempo de Microsoft Active Accessibility, gerencia solicitações de clientes Microsoft Active Accessibility e proxies elementos comuns da interface do usuário |
| Oleaccrc.dll | Biblioteca de vínculo dinâmico de recurso multilíngue usada pelo Oleacc.dll                                                                                                                |
