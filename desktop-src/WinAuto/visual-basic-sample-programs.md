# <a name="visual-basic-sample-programs"></a>Visual Basic programas de exemplo

O livro *avançado Microsoft Visual Basic 6,0, 2ª edição* (Mandelbrot set International, Ltd.) inclui um capítulo que aborda os acessibilidade ativa da Microsoft. O capítulo fornece um código de exemplo que demonstra como usar os principais elementos da API do Microsoft Acessibilidade Ativa. (Microsoft Press, ISBN 1-57231-893-7) (Este livro pode não estar disponível em alguns idiomas e países.)
