# <a name="ui-automation-support-for-textual-content"></a>Suporte à automação da interface do usuário para conteúdo textual

Esta seção descreve como usar a automação da interface do usuário da Microsoft para expor o conteúdo textual, incluindo atributos de formato e estilo, dos controles de texto nas plataformas com suporte da automação da interface do usuário. Os controles de texto incluem, mas não se limitam a, documentos, dicas de ferramentas, controles de edição e controles de edição avançados.
