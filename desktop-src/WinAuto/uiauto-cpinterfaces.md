# <a name="control-pattern-interfaces-for-providers"></a>Interfaces de padrão de controle para provedores

Esta seção descreve as interfaces de *padrão de controle* implementadas pelos provedores de automação da interface do usuário para expor informações sobre controles em aplicativos do Windows não gerenciados.

## <a name="in-this-section"></a>Nesta seção



| Interface                                                                          | Descrição                                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAnnotationProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iannotationprovider)<br/>           | Expõe as propriedades de uma anotação em um documento.<br/>                                                                                                                                                                                                                                                   |
| [**IDockProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idockprovider)<br/>                           | Fornece acesso a um elemento em um contêiner de encaixe. <br/>                                                                                                                                                                                                                                                   |
| [**IDragProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idragprovider)<br/>                           | Permite que um elemento de automação da interface do usuário da Microsoft se descreva como um elemento que pode ser arrastado como parte de uma operação de arrastar e soltar.<br/>                                                                                                                                                                     |
| [**IDropTargetProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idroptargetprovider)<br/>               | Permite que um elemento de automação da interface do usuário se descreva como um elemento que pode receber uma queda de um elemento arrastado como parte de uma operação de arrastar e soltar da automação da interface do usuário.<br/>                                                                                                                                        |
| [**IExpandCollapseProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iexpandcollapseprovider)<br/>       | Fornece acesso a um controle que expande visualmente para exibir conteúdo e recolhe para ocultar o conteúdo. <br/>                                                                                                                                                                                                   |
| [**IGridItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igriditemprovider)<br/>                   | Fornece acesso a controles filho individuais de contêineres que implementam [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider).<br/>                                                                                                                                                                                 |
| [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)<br/>                           | Fornece acesso a controles que atuam como contêineres para uma coleção de elementos filho organizados em um sistema de coordenadas lógico bidimensional que pode ser percorrido (ou seja, um cliente de automação da interface do usuário pode ser movido para controles adjacentes) usando o teclado.<br/>                                              |
| [**IInvokeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iinvokeprovider)<br/>                       | Fornece acesso a controles que iniciam ou executam uma única ação não ambígua e não mantêm o estado quando ativados.<br/>                                                                                                                                                                              |
| [**IItemContainerProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iitemcontainerprovider)<br/>         | Fornece acesso a controles que atuam como contêineres de outros controles, como uma exibição de lista virtual. <br/>                                                                                                                                                                                                      |
| [**ILegacyIAccessibleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ilegacyiaccessibleprovider)<br/> | Permite que clientes de automação da interface do usuário acessem a implementação base do [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) dos elementos do Microsoft acessibilidade ativa.<br/>                                                                                                                                                     |
| [**IMultipleViewProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-imultipleviewprovider)<br/>           | Fornece acesso a controles que fornecem e são capazes de alternar entre várias representações do mesmo conjunto de informações ou controles filho.<br/>                                                                                                                                                     |
| [**IObjectModelProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iobjectmodelprovider)<br/>         | Fornece acesso ao modelo de objeto subjacente implementado por um controle ou aplicativo. <br/>                                                                                                                                                                                                                 |
| [**IRangeValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irangevalueprovider)<br/>               | Fornece acesso a controles que podem ser definidos para um valor dentro de um intervalo. <br/>                                                                                                                                                                                                                                  |
| [**IScrollItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollitemprovider)<br/>               | Fornece acesso a controles filho individuais de contêineres que implementam [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider). <br/>                                                                                                                                                                            |
| [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)<br/>                       | Fornece acesso a controles que atuam como contêineres roláveis para uma coleção de objetos filho.<br/>                                                                                                                                                                                                         |
| [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider)<br/>         | Fornece acesso a controles filho selecionáveis individuais de contêineres que implementam [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider). <br/>                                                                                                                                                          |
| [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider)<br/>                 | Fornece acesso a controles que atuam como contêineres para uma coleção de itens filho individuais selecionáveis.<br/>                                                                                                                                                                                               |
| [**ISelectionProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider2)<br/>               | Estende a interface [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider) para fornecer informações sobre os itens selecionados.<br/>                                                                                                                                                                       |
| [**ISpreadsheetItemProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetitemprovider)<br/> | Fornece acesso a informações sobre um item (célula) em uma planilha. <br/>                                                                                                                                                                                                                                   |
| [**ISpreadsheetProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetprovider)<br/>         | Fornece acesso a itens (células) em uma planilha.<br/>                                                                                                                                                                                                                                                       |
| [**IStylesProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-istylesprovider)<br/>                   | Fornece acesso aos estilos visuais associados ao conteúdo de um documento.<br/>                                                                                                                                                                                                                          |
| [**ISynchronizedInputProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-isynchronizedinputprovider)<br/> | Permite que aplicativos cliente de automação da interface do usuário direcionem a entrada do mouse ou do teclado para um elemento específico da interface do usuário. <br/>                                                                                                                                                                                               |
| [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider)<br/>                 | Fornece acesso a controles filho de contêineres que implementam [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider). <br/>                                                                                                                                                                                         |
| [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider)<br/>                         | Fornece acesso a controles que atuam como contêineres para uma coleção de elementos filho. Os filhos desse elemento devem implementar [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider) e ser organizados em um sistema de coordenadas lógico bidimensional que pode ser percorrido usando o teclado. <br/> |
| [**ITextChildProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextchildprovider)<br/>            | Fornece acesso a um controle baseado em texto (ou um objeto inserido em texto) que é um filho ou descendente de outro controle baseado em texto. <br/>                                                                                                                                                                    |
| [**ITextEditProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itexteditprovider)<br/>               | Estende a interface [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) para permitir que os provedores de automação da interface do usuário exponham ações de edição de texto programáticas.<br/>                                                                                                                                                          |
| [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider)<br/>                           | Fornece acesso a controles que contêm texto. <br/>                                                                                                                                                                                                                                                          |
| [**ITextProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextprovider2)<br/>                     | Estende a interface [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) para permitir que os provedores de automação da interface do usuário exponham o conteúdo textual que é o destino de uma anotação e informações sobre um cursor que pertence ao provedor.<br/>                                                                         |
| [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider)<br/>                 | Fornece acesso a um intervalo de texto contínuo em um contêiner de texto que implementa [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) ou [**ITextProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider2).<br/>                                                                                                                       |
| [**ITextRangeProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider2)<br/>               | Estende a interface [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider) para permitir que os provedores de automação da interface do usuário invoquem menus de contexto.<br/>                                                                                                                                                                 |
| [**IToggleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itoggleprovider)<br/>                       | Fornece acesso a controles que podem percorrer um conjunto de Estados e manter um estado depois que ele é definido.<br/>                                                                                                                                                                                                 |
| [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider)<br/>                 | Fornece acesso a controles que podem ser movidos, redimensionados e/ou girados dentro de um espaço bidimensional. <br/>                                                                                                                                                                                                  |
| [**ITransformProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itransformprovider2)<br/>           | Estende a interface [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider) para permitir que os provedores de automação da interface do usuário exponham propriedades para dar suporte à funcionalidade de zoom do visor de um controle. <br/>                                                                                                        |
| [**IValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivalueprovider)<br/>                         | Fornece acesso a controles que têm um valor intrínseco que não abrange um intervalo e que podem ser representados como uma cadeia de caracteres. <br/>                                                                                                                                                                           |
| [**IVirtualizedItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivirtualizeditemprovider)<br/>     | Fornece acesso a itens virtualizados, que são itens representados por elementos de automação de espaço reservado na árvore de automação da interface do usuário.<br/>                                                                                                                                                                 |
| [**IWindowProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iwindowprovider)<br/>                       | Fornece acesso à funcionalidade básica baseada em janela de um controle.<br/>                                                                                                                                                                                                                              |
