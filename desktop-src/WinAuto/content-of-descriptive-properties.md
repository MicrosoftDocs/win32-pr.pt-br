# <a name="content-of-descriptive-properties"></a>Conteúdo de propriedades descritivas

A interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) fornece propriedades descritivas, que descrevem vários aspectos de um objeto. Algumas dessas propriedades são específicas de conteúdo; outras propriedades têm conteúdo que consiste em texto descritivo fornecido pelo servidor. O tipo de informações para cada propriedade varia dependendo do objeto.

Os tópicos a seguir descrevem as informações que os clientes obtêm dessas propriedades. Eles também fornecem aos servidores sugestões para escolher o conteúdo.

-   [Propriedade Name](name-property.md)
-   [Propriedade de função](role-property.md)
-   [Propriedade State](state-property.md)
-   [Propriedade Value](value-property.md)
-   [Propriedade Description](description-property.md)
-   [Propriedade da ajuda](help-property.md)
-   [Propriedade HelpTopic](helptopic-property.md)
-   [Propriedade KeyboardShortcut](keyboardshortcut-property.md)
-   [Propriedade DefaultAction](defaultaction-property.md)

Ao criar objetos acessíveis, os desenvolvedores de servidor também devem consultar os seguintes tópicos:

-   [Escolhendo quais propriedades para dar suporte](choosing-which-properties-to-support.md)
-   [Escolhendo o conteúdo para propriedades descritivas](choosing-the-content-for-descriptive-properties.md)

Para obter informações sobre os parâmetros e valores de retorno dessas propriedades, consulte a seção [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) da referência do Microsoft acessibilidade ativa [C/C++](c-c---reference.md).
