# <a name="landmark-type-identifiers"></a>Identificadores de tipo de ponto de referência

Este tópico descreve as constantes nomeadas usadas para identificar tipos de pontos de referência de automação de interface do usuário da Microsoft.

<dl> <dt>

<span id="UIA_CustomLandmarkTypeId"></span><span id="uia_customlandmarktypeid"></span><span id="UIA_CUSTOMLANDMARKTYPEID"></span>**UIA \_ CustomLandmarkTypeId**
</dt> <dd> <dl> <dt>

80000
</dt> <dt>



Indica que uma cadeia de caracteres personalizada deve ser usada para descrever o ponto de referência usando o valor de [**UIA \_ LocalizedLandmarkTypePropertyId**](uiauto-automation-element-propids.md).


</dt> </dl> </dd> <dt>

<span id="UIA_FormLandmarkTypeId"></span><span id="uia_formlandmarktypeid"></span><span id="UIA_FORMLANDMARKTYPEID"></span>**UIA \_ FormLandmarkTypeId**
</dt> <dd> <dl> <dt>

80001
</dt> <dt>



Indica que o ponto de referência está relacionado aos elementos de tipo de formulário normalmente usados para entrada de dados.


</dt> </dl> </dd> <dt>

<span id="UIA_MainLandmarkTypeId"></span><span id="uia_mainlandmarktypeid"></span><span id="UIA_MAINLANDMARKTYPEID"></span>**UIA \_ MainLandmarkTypeId**
</dt> <dd> <dl> <dt>

80002
</dt> <dt>



Indica que o ponto de referência é para o conteúdo primário e não secundário ou menor prioridade.


</dt> </dl> </dd> <dt>

<span id="UIA_NavigationLandmarkTypeId"></span><span id="uia_navigationlandmarktypeid"></span><span id="UIA_NAVIGATIONLANDMARKTYPEID"></span>**UIA \_ NavigationLandmarkTypeId**
</dt> <dd> <dl> <dt>

80003
</dt> <dt>



Indica que o ponto de referência está relacionado aos elementos do tipo de navegação.


</dt> </dl> </dd> <dt>

<span id="UIA_SearchLandmarkTypeId"></span><span id="uia_searchlandmarktypeid"></span><span id="UIA_SEARCHLANDMARKTYPEID"></span>**UIA \_ SearchLandmarkTypeId**
</dt> <dd> <dl> <dt>

80004
</dt> <dt>



Indica que o ponto de referência está relacionado aos elementos do tipo de pesquisa.


</dt> </dl> </dd> </dl>
