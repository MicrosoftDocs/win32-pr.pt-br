# <a name="methodreturnedunexpectedhresult"></a>MethodReturnedUnexpectedHResult

## <a name="text"></a>Texto

{0} retornado {1} , mas o valor de retorno esperado deveria ter sido {2}

## <a name="type"></a>Type

Aviso

## <a name="description"></a>Descrição

Um HRESULT inesperado foi retornado ao tentar recuperar todos os filhos do elemento.

## <a name="possible-causes"></a>Possíveis causas

Uma implementação de MSAA incorreta ou inválida.
