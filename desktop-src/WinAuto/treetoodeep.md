# <a name="treetoodeep"></a>TreeTooDeep

## <a name="text"></a>Texto

A árvore tem {0} níveis de profundidade.

## <a name="type"></a>Tipo

Aviso

## <a name="description"></a>Descrição

A árvore de elementos pode ser muito profunda.

Esse problema causa problemas para as pessoas que contam com um leitor de tela e um teclado para navegação porque a passagem de elementos parecerá ser demorada e pode parecer cíclica.

## <a name="possible-causes"></a>Possíveis causas

-   Vários elementos ou seus pais são controles personalizados que não implementam a Tabulação corretamente. Por exemplo, a propriedade de estado MSAA não é atualizada corretamente.
-   A interface do usuário do aplicativo é excessivamente complexa.
