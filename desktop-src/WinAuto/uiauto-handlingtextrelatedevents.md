# <a name="handling-text-related-events"></a>Manipulando eventos relacionados a texto

Este tópico descreve os eventos gerados por controles baseados em texto quando o conteúdo é alterado, selecionado ou desmarcado.


Controles baseados em texto geram um evento de [**\_ \_ TextChangedEventId de texto UIA**](uiauto-event-ids.md) sempre que o conteúdo textual é alterado. Eles geram o evento [**UIA \_ Text \_ TextSelectionChangedEventId**](uiauto-event-ids.md) sempre que o conteúdo textual é selecionado ou desmarcado. Para obter mais informações, consulte [visão geral dos eventos de automação da interface do usuário](uiauto-eventsoverview.md) e [assinatura de eventos de automação da interface do usuário](uiauto-eventsforclients.md).
