# <a name="handling-text-related-events"></a>Manipulando eventos relacionados a texto

Este tópico descreve os eventos gerados por controles baseados em texto quando o conteúdo é alterado, selecionado ou deseletório.


Controles baseados em texto adquiram um evento [**\_ \_ TextChangedEventId**](uiauto-event-ids.md) da UIA sempre que o conteúdo textual é alterado. Eles acionam [**o evento \_ \_ TextSelectionChangedEventId**](uiauto-event-ids.md) da UIA sempre que o conteúdo textual é selecionado ou desmarcado. Para obter mais informações, [consulte Visão geral Automação da Interface do Usuário](uiauto-eventsoverview.md) eventos e Assinatura para Automação da Interface do Usuário [eventos](uiauto-eventsforclients.md).
