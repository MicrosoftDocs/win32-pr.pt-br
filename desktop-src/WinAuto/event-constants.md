# <a name="event-constants-winuserh"></a>Constantes de evento (WinUser. h)

Este tópico descreve os eventos que são gerados pelo sistema operacional e por aplicativos de servidor. As constantes são listadas em ordem alfabética.

Antes de usar esses eventos, os aplicativos cliente devem usar o [observador de eventos acessível](accessible-event-watcher.md) para verificar se esses eventos são usados por elementos da interface do usuário.

Para obter mais informações sobre eventos em geral, consulte [o que são WinEvents?](what-are-winevents.md) e eventos de nível [de sistema e objeto](system-level-and-object-level-events.md). Para obter mais informações sobre os eventos enviados pelo sistema, consulte o [Apêndice A: referência de elementos da interface do usuário com suporte](appendix-a--supported-user-interface-elements-reference.md).
