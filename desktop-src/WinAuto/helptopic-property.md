# <a name="helptopic-property"></a>Propriedade HelpTopic

A propriedade **HelpTopic** Obtém detalhes usados com a função [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) do Microsoft Win32 para exibir um tópico da ajuda associado ao objeto. Nem todos os objetos dão suporte a essa propriedade.

A propriedade **HelpTopic** é recuperada chamando o [**IAccessible:: get \_ accHelpTopic**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic).
