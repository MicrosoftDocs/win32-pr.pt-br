# <a name="boundingrectisincorrect"></a>BoundingRectIsIncorrect

## <a name="text"></a>Texto

A {0} borda do retângulo relatado é inconsistente com o teste de acerto da {1} rect no ponto {2} .

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

O retângulo delimitativo do elemento não é consistente com os resultados esperados do teste de acerto. Por exemplo, testar um ponto fora do retângulo delimitativo do elemento não retorna um elemento diferente.
