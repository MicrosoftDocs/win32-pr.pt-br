# <a name="boundingrectisincorrect"></a>BoundingRectIsIncorrect

## <a name="text"></a>Texto

A {0} borda do retângulo relatado é inconsistente com o teste {1} de clique no Rect no ponto {2} .

## <a name="type"></a>Type

Erro

## <a name="description"></a>Descrição

O retângulo delimitador do elemento não é consistente com os resultados de teste de clique esperados. Por exemplo, o teste de clique de um ponto fora do retângulo delimitador do elemento não retorna um elemento diferente.
