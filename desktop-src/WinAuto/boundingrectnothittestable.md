# <a name="boundingrectnothittestable"></a>BoundingRectNotHitTestable

## <a name="text"></a>Texto

O retângulo relatado não parece conter um ponto onde esse elemento pode ser recuperado com o teste de clique.

## <a name="type"></a>Tipo

Aviso

## <a name="description"></a>Descrição

O retângulo delimitador do elemento não pode ser acessado por testes de clique. Por exemplo, ele está fora das coordenadas da tela física (ou da região visível) da área de trabalho.
