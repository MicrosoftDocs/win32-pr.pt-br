# <a name="boundingrectnothittestable"></a>BoundingRectNotHitTestable

## <a name="text"></a>Texto

O retângulo relatado não parece conter um ponto em que esse elemento pode ser recuperado com testes de acerto.

## <a name="type"></a>Tipo

Aviso

## <a name="description"></a>Descrição

O retângulo delimitado do elemento não pode ser acessível por testes de acerto. Por exemplo, ele está fora das coordenadas de tela física (ou região exibivel) da área de trabalho.
