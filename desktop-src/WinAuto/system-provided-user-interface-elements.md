# <a name="system-provided-user-interface-elements"></a>System-Provided Interface do Usuário elementos

Microsoft Active Accessibility dá suporte à maioria dos controles predefinidos e comuns. A ilustração a seguir mostra uma janela típica e alguns dos elementos de interface do usuário fornecidos pelo sistema que o Microsoft Active Accessibility expõe, como barras de título, menus, caixas de combinação, controles de barra de ferramentas, controles de exibição de árvore, barras de status, barras de tamanho, controles de exibição de lista e barras de rolagem.

![captura de tela mostrando vários elementos diferentes da interface do usuário](images/explorer.gif)

Microsoft Active Accessibility expõe elementos de interface do usuário fornecidos pelo sistema para aplicativos de servidor sem exigir que o desenvolvedor do servidor implemente a interface [**IAccessible.**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) Qualquer aplicativo que contém esses elementos herda automaticamente sua acessibilidade.

Para ver uma lista dos controles e outros elementos de interface do usuário fornecidos pelo sistema que Microsoft Active Accessibility suporte, consulte [Apêndice A: Referência](appendix-a--supported-user-interface-elements-reference.md)de elementos Interface do Usuário suporte.
