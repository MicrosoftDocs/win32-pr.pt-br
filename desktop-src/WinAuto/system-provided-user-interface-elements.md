# <a name="system-provided-user-interface-elements"></a>System-Provided elementos da interface do usuário

O Microsoft Acessibilidade Ativa fornece suporte para a maioria dos controles predefinidos e comuns. A ilustração a seguir mostra uma janela típica e alguns dos elementos da interface do usuário fornecidos pelo sistema que a Microsoft Acessibilidade Ativa expõe, como barras de título, menus, caixas de combinação, controles da barra de ferramentas, controles de exibição de árvore, barras de status, alças de tamanho, controles de exibição de lista e barras de rolagem.

![captura de tela mostrando vários elementos diferentes da interface do usuário](images/explorer.gif)

O Microsoft Acessibilidade Ativa expõe elementos de interface do usuário fornecidos pelo sistema a aplicativos de servidor sem exigir que o desenvolvedor do servidor implemente a interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) . Qualquer aplicativo que contenha esses elementos herda automaticamente sua acessibilidade.

Para obter uma lista dos controles e outros elementos da interface do usuário fornecidos pelo sistema que o Microsoft Acessibilidade Ativa suporta, consulte [o apêndice a: referência de elementos da interface do usuário com suporte](appendix-a--supported-user-interface-elements-reference.md).
