# <a name="shortcuts-for-exposing-custom-user-interface-elements"></a>Atalhos para expor elementos personalizados da interface do usuário

Esta seção descreve os atalhos que os desenvolvedores de aplicativos podem usar para expor determinados tipos de elementos de interface do usuário personalizados. As técnicas descritas nesta seção simplificam a implementação da interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) ou fornecem uma alternativa para expor informações sem implementar o **IAccessible**.

> [!Note]  
> Você deve considerar o uso de alguma forma de anotação dinâmica (direta, mapa de valor ou servidor) antes de tentar as técnicas descritas nesta seção. Para obter mais informações, consulte [API de anotação dinâmica](dynamic-annotation-api.md).
