# <a name="client-functions"></a>Funções de cliente

Esta seção contém informações sobre as funções de cliente usadas com o Microsoft Acessibilidade Ativa.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                             | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AccessibleChildren**](/windows/desktop/api/Oleacc/nf-oleacc-accessiblechildren)<br/>                       | Recupera a ID filho ou [**IDispatch**](idispatch-interface.md) de cada filho dentro de um objeto de contêiner acessível.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**AccessibleObjectFromEvent**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromevent)<br/>         | Recupera o endereço da interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) para o objeto que gerou o evento que está sendo processado atualmente pela função de gancho de evento do cliente.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [**AccessibleObjectFromPoint**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfrompoint)<br/>         | Recupera o endereço do ponteiro da interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) para o objeto exibido em um ponto especificado na tela.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**AccessibleObjectFromWindow**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromwindow)<br/>       | Recupera o endereço da interface especificada para o objeto associado à janela especificada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**GetOleaccVersionInfo**](/windows/desktop/api/Oleacc/nf-oleacc-getoleaccversioninfo)<br/>                   | Recupera o número de versão e o número de Build do arquivo Microsoft Acessibilidade Ativa Oleacc.dll.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**GetProcessHandleFromHwnd**](getprocesshandlefromhwnd.md)<br/>           | Recupera um identificador de processo de um identificador de janela.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**GetRoleText**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta)<br/>                                     | Recupera a cadeia de caracteres localizada que descreve a função do objeto para o valor de função especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetStateText**](/windows/desktop/api/Oleacc/nf-oleacc-getstatetexta)<br/>                                   | Recupera uma cadeia de caracteres localizada que descreve o estado de um objeto para um único sinalizador de bit de estado predefinido. Como os valores de estado são uma combinação de um ou mais sinalizadores de bits, os clientes chamam essa função mais de uma vez para recuperar todas as cadeias de caracteres de estado.<br/>                                                                                                                                                                                                                                                                                                                      |
| [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)<br/>                             | Define uma função de gancho de evento para um intervalo de eventos.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**UnhookWinEvent**](/windows/desktop/api/Winuser/nf-winuser-unhookwinevent)<br/>                               | Remove uma função de gancho de evento criada por uma chamada anterior para [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**WindowFromAccessibleObject**](/windows/desktop/api/Oleacc/nf-oleacc-windowfromaccessibleobject)<br/>       | Recupera o identificador de janela que corresponde a uma instância específica de uma interface [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [*Função de retorno de chamada WinEventProc*](/windows/desktop/api/Winuser/nc-winuser-wineventproc)<br/> | Uma função de retorno de chamada (ou Hook) definida pelo aplicativo que o sistema chama em resposta a eventos gerados por um objeto acessível. A função Hook processa as notificações de eventos conforme necessário. Os clientes instalam a função Hook e solicitam tipos específicos de notificações de eventos chamando [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/> O tipo [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85)) define um ponteiro para essa função de retorno de chamada. [*WinEventProc*](/windows/desktop/api/winuser/nc-winuser-wineventproc) é um espaço reservado para o nome da função definida pelo aplicativo.<br/> |
