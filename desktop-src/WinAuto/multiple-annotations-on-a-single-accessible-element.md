# <a name="multiple-annotations-on-a-single-accessible-element"></a>Várias anotações em um único elemento acessível

Atualmente, um elemento acessível dá suporte a apenas uma anotação por propriedade. Anotações sucessivas em uma propriedade substituem e substituem as anotações anteriores dessa propriedade.
