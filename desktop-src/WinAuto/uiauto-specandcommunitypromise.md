# <a name="ui-automation-specification-and-community-promise"></a>Especificação da automação de interface do usuário e promessa da comunidade

A Automação da Interface do Usuário especificação fornece informações sobre estruturas de acessibilidade da Microsoft, incluindo Microsoft Active Accessibility e Microsoft Automação da Interface do Usuário. Ele discute os principais componentes de cada estrutura, fornece considerações de design e exemplos e descreve os requisitos de implementação. Para obter mais informações, consulte [Automação da Interface do Usuário Especificação .](ui-automation-specification.md)

O Automação da Interface do Usuário Community Promise permite que qualquer pessoa implemente livremente a especificação Automação da Interface do Usuário sem a necessidade de assinar contratos com a Microsoft. Além disso, não é necessário fazer nenhuma menção ou qualquer referência à Microsoft. Para obter mais informações, [consulte Automação da Interface do Usuário Community Promise](uiauto-specandcommunitypromise.md).
