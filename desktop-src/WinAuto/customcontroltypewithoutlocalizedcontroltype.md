# <a name="customcontroltypewithoutlocalizedcontroltype"></a>CustomControlTypeWithoutLocalizedControlType

## <a name="text"></a>Texto

O elemento não tem a propriedade LocalizedControlType definida

## <a name="type"></a>Tipo

Erro

## <a name="description"></a>Descrição

A propriedade [**LocalizedControlType**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationelement-get_currentlocalizedcontroltype) do elemento não está definida.

A maioria dos leitores de tela lê o tipo de controle localizado para descrever a natureza do elemento. Se o tipo de controle localizado não for definido, um leitor de tela voltará a ler o tipo de controle principal, que está sempre em inglês e pode não fazer sentido para todos os usuários.
