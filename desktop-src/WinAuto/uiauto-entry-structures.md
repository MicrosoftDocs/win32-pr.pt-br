# <a name="ui-automation-structures"></a>Estruturas de automação da interface do usuário

Esta seção descreve as estruturas usadas pelos aplicativos cliente de automação da interface do usuário e pelo provedor de automação da interface do usuário do Microsoft Win32.

## <a name="in-this-section"></a>Nesta seção



| Estrutura                                                                            | Descrição                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](/windows/desktop/api/UIAutomationClient/ns-uiautomationclient-extendedproperty)<br/>                       | Contém informações sobre uma propriedade estendida. <br/>                                  |
| [**UIAutomationEventInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationeventinfo)<br/>       | Contém informações sobre um evento personalizado.<br/>                                         |
| [**UIAutomationMethodInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationmethodinfo)<br/>     | Contém informações sobre um método que é suportado por um padrão de controle personalizado.<br/> |
| [**UIAutomationParameter**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationparameter)<br/>       | Contém informações sobre um parâmetro de um padrão de controle personalizado.<br/>                |
| [**UIAutomationPatternInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpatterninfo)<br/>   | Contém informações sobre um padrão de controle personalizado.<br/>                               |
| [**UIAutomationPropertyInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpropertyinfo)<br/> | Contém informações sobre uma propriedade personalizada.<br/>                                      |
| [**UiaChangeInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiachangeinfo)<br/>                                    | Contém dados sobre uma alteração de automação de interface do usuário que ocorreu.<br/>                          |
| [**UiaPoint**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiapoint)<br/>                                 | Contém as coordenadas de um ponto. <br/>                                              |
| [**UiaRect**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiarect)<br/>                                   | Contém a posição e o tamanho de um retângulo, em coordenadas da tela.<br/>              |
