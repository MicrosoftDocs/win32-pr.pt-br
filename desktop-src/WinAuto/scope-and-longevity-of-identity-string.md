# <a name="scope-and-longevity-of-identity-string"></a>Escopo e longevidade da cadeia de caracteres de identidade

Uma cadeia de caracteres de identidade identifica uma instância específica de um elemento de interface do usuário. Várias instâncias de um elemento acessível terão cadeias de caracteres de identidade diferentes; Portanto, normalmente não é útil manter uma cadeia de caracteres de identidade.
