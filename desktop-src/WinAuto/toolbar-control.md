# <a name="toolbar-control-msaa-ui-element-reference"></a>Controle Toolbar (referência de elemento da interface do usuário do MSAA)

> [!Note]  
> Este tópico descreve objetos de **controle da barra de ferramentas** para fins de referência de elemento da interface do usuário do MSAA. Como criar objetos de **controle da barra de ferramentas** em várias estruturas de interface do usuário não é descrito aqui. Consulte a documentação de referência da API para a estrutura de interface do usuário que você está usando.

 

Um controle Toolbar contém botões que executam comandos de menu e geralmente estão contidos em uma janela abaixo da barra de menus.

O nome da classe de janela para um controle ToolBar é TOOLBARCLASSNAME, que é definido como "ToolbarWindow32" em commctrl. h.
