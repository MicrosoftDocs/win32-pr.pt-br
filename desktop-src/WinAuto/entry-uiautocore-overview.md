# <a name="ui-automation-fundamentals"></a>Fundamentos de automação da interface do usuário

A automação da interface do usuário da Microsoft permite que aplicativos de tecnologia assistencial e ferramentas de teste automatizada interajam com os controles da interface do usuário de outros aplicativos. Esta seção explica os conceitos fundamentais em que a automação da interface do usuário se baseia.

A API de automação da interface do usuário está em duas partes. Uma parte é usada por aplicativos de *provedor de automação de interface do usuário* e a outra é usada por aplicativos *cliente de automação da interface do usuário* . A API do provedor permite que os desenvolvedores de controle personalizado do Microsoft Win32 e outras estruturas de controle exponham esses controles à automação da interface do usuário e os tornem visíveis para os aplicativos cliente. A API do cliente permite que os aplicativos interajam com controles em outros aplicativos e recuperem informações sobre eles.
