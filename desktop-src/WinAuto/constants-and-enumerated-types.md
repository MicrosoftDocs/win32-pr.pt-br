# <a name="constants-active-accessibility-user-interface-services"></a>Constantes (Acessibilidade Ativa serviços de interface do usuário)

Esta seção contém informações sobre as constantes usadas com o Microsoft Acessibilidade Ativa.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                               | Descrição                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Constantes de evento**](event-constants.md)<br/>               | Este tópico descreve os eventos que são gerados pelo sistema operacional e por aplicativos de servidor.<br/>                                                                                                                                                                                                                                                                     |
| [**Constantes de navegação**](navigation-constants.md)<br/>     | Este tópico descreve os valores constantes, definidos em OleAcc. h, que indicam a direção *espacial* (acima, abaixo, esquerda e direita) ou *lógica* (primeiro filho, última, próxima e anterior) observada quando os clientes usam [**IAccessible:: accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate) para navegar de um elemento de interface do usuário para outro dentro do mesmo contêiner.<br/> |
| [**Identificadores de objeto**](object-identifiers.md)<br/>         | Este tópico descreve os identificadores de objeto do Microsoft Acessibilidade Ativa, os valores de 32 bits que identificam *categorias* de objetos acessíveis em uma janela.<br/>                                                                                                                                                                                                                |
| [**Funções de objeto**](object-roles.md)<br/>                     | Este tópico descreve os valores constantes usados para descrever as funções de vários objetos de interface do usuário em um aplicativo.<br/>                                                                                                                                                                                                                                                               |
| [**Constantes de estado do objeto**](object-state-constants.md)<br/> | Este tópico descreve os valores constantes usados para descrever o estado dos objetos em uma interface do usuário do aplicativo.<br/>                                                                                                                                                                                                                                                                       |
| [**Constantes SELFLAG**](selflag.md)<br/>                     | Este tópico descreve os valores constantes usados para especificar como um objeto acessível é selecionado ou leva em foco.<br/>                                                                                                                                                                                                                                                     |
