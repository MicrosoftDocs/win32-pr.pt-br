# <a name="touch-injection-functions"></a>Funções de injeção de toque

Os tópicos nesta seção fornecem as especificações de referência para funções de [injeção de toque](touch-injection-portal.md) .

## <a name="in-this-section"></a>Nesta seção

| Tópico | Descrição |
|---|---|
| [**InitializeTouchInjection**](/windows/win32/api/winuser/nf-winuser-initializetouchinjection)<br/> | Configura o contexto de injeção de toque para o aplicativo de chamada e inicializa o número máximo de contatos simultâneos que o aplicativo pode injetar.<br/> |
| [**InjectTouchInput**](/windows/win32/api/winuser/nf-winuser-injecttouchinput)<br/>                 | Simula a entrada por toque.<br/> |
