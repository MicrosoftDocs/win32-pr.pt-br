# <a name="audio-and-video"></a>Áudio e vídeo

a Microsoft fornece componentes projetados para permitir que desenvolvedores de aplicativos, desenvolvedores da Web e administradores de sistemas desenvolvam programas de áudio e vídeo e criem Windows aplicativos e sites compatíveis com mídia.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                              | Descrição                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [APIs de áudio de núcleo](./coreaudio/core-audio-apis-in-windows-vista.md)<br/>                                           | Uma API de nível baixo para captura de áudio e renderização de áudio, que pode ser usada para atingir a latência mínima ou para implementar recursos que podem não ser totalmente suportados por APIs de mídia de nível superior.<br/>                           |
| [DirectShow](./directshow/directshow.md)<br/>                                                                          | Um pipeline de mídia de ponta a ponta, que dá suporte à reprodução, captura de áudio/vídeo, codificação, navegação e reprodução de DVD, televisão analógica e MPEG-2.<br/>                                                                       |
| [Microsoft Media Foundation](./medfound/microsoft-media-foundation-sdk.md)<br/>                                         | Um pipeline de mídia de ponta a ponta, que dá suporte à reprodução de p, captura de áudio/vídeo e codificação (sucessor para DirectShow).<br/>                                                                                                   |
| [Tecnologias de TV da Microsoft](/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal)<br/>                                      | Dá suporte a televisão digital, arquivos gravados de TV (. wtv) e dispositivos OCUR (receptor de cabo unidirecional) OpenCable.<br/>                                                                                                      |
| [Windows Kit de desenvolvimento de software do Media Center](/previous-versions/aa286546(v=msdn.10))<br/>                                          | usando o SDK (Software Development Kit) do Windows Media Center, os desenvolvedores podem criar aplicativos e serviços de mídia avançados para uso com o mouse, o teclado e o controle remoto em um PC com o Media Center.<br/>                        |
| [Windows SDK do formato de mídia 11](./wmformat/windows-media-format-11-sdk.md)<br/>                                     | dá suporte à leitura e gravação de arquivos ASF, decodificação e codificação Windows áudio e vídeo de mídia e streaming de asf em uma rede.<br/>                                                                                          |
| [Windows Serviços de compartilhamento de biblioteca de mídia](/previous-versions/windows/desktop/wmlss/windowsmedialibrarysharingservicesportal)<br/>                | Permite que os aplicativos descubram dispositivos de mídia na rede doméstica e compartilhem bibliotecas de mídia na rede doméstica e na Internet.<br/>                                                                                     |
| [Windows Media Player SDK](./wmp/windows-media-player-sdk.md)<br/>                                                | estende os recursos de Windows Media Player e Windows Media Player Mobile.<br/>                                                                                                                                       |
| [Windows SDK 10.1.2 do Media Rights Manager](/previous-versions//bb649422(v=vs.85))<br/> | dá suporte ao DRM (gerenciamento de direitos digitais) para proteger Windows arquivos de mídia.<br/>                                                                                                                                            |
| [série 9 Windows Media Services](/previous-versions/windows/desktop/dd873819(v=vs.85))<br/>                                 | fornece uma API baseada em automação para gerenciar Windows Media Services no Windows Server.<br/>                                                                                                                                 |
| [Windows API de streaming de mídia](./mediastreaming/media-streaming-api-portal.md)<br/>                                | Modela os dispositivos DMR (Digital Media Renderer) e DMS (Digital Media Server), conforme definido pelas diretrizes de DLNA, para um uso programático fácil.<br/>                                                                           |
| [Windows Controles de transporte de mídia](/previous-versions/windows/desktop/mediatransport/media-transport-controls-portal)<br/>                      | Fornece uma interface interna que mostra as informações do usuário sobre a mídia em execução no momento, como o título de uma música ou vídeo. O também fornece ao usuário uma maneira comum e familiar de controlar a reprodução de mídia.<br/> |
| [Windows SDK do Movie maker 6,0 e do Windows DVD maker 1,0](/previous-versions/windows/desktop/wmmdvdm/windows-movie-maker-6-0-and-windows-dvd-maker-1-0-sdk)<br/>     | habilita a personalização do Windows Movie maker 6,0 e do Microsoft Windows DVD maker 1,0.<br/>                                                                                                                                   |
| [Áudio e vídeo herdados](/windows-hardware/drivers/audio/legacy-audio-interfaces)<br/>                                                    | Tecnologias obsoletas e não devem ser usadas em novos aplicativos.<br/>                                                                                                                                              |
