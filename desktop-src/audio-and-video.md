# <a name="audio-and-video"></a>Áudio e vídeo

A Microsoft fornece componentes projetados para permitir que desenvolvedores de aplicativos, desenvolvedores web e administradores de sistemas desenvolvam programas de áudio e vídeo e criem aplicativos e sites compatíveis com Windows mídia.

## <a name="in-this-section"></a>Nesta seção



| Tópico                                                                                                              | Descrição                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [APIs de áudio principais](./coreaudio/core-audio-apis-in-windows-vista.md)<br/>                                           | Uma API de baixo nível para captura de áudio e renderização de áudio, que pode ser usada para obter latência mínima ou implementar recursos que talvez não sejam totalmente suportados por APIs de mídia de nível superior.<br/>                           |
| [DirectShow](./directshow/directshow.md)<br/>                                                                          | Um pipeline de mídia de ponta a ponta, que dá suporte à reprodução, captura de áudio/vídeo, codificação, navegação e reprodução de DVD, tv analoga e MPEG-2.<br/>                                                                       |
| [Microsoft Media Foundation](./medfound/microsoft-media-foundation-sdk.md)<br/>                                         | Um pipeline de mídia de ponta a ponta, que dá suporte à reprodução p, captura de áudio/vídeo e codificação (sucessor de DirectShow).<br/>                                                                                                   |
| [Tecnologias de TV da Microsoft](/previous-versions/windows/desktop/mstv/microsoft-tv-technologies-portal)<br/>                                      | Dá suporte à tv digital, arquivos de TV gravados (.wtv) e dispositivos OCUR (Receptor de Cabo Unidirecional OpenCable).<br/>                                                                                                      |
| [Windows Kit de Desenvolvimento de Software do Media Center](/previous-versions/aa286546(v=msdn.10))<br/>                                          | Usando o SDK (Kit de Desenvolvimento de Software) do Windows Media Center, os desenvolvedores podem criar aplicativos de mídia e serviços de mídia rich para uso com mouse, teclado e controle remoto em um computador do Media Center.<br/>                        |
| [Windows SDK do Formato de Mídia 11](./wmformat/windows-media-format-11-sdk.md)<br/>                                     | Dá suporte à leitura e gravação de arquivos ASF, decodificação e codificação Windows mídia de áudio e vídeo e streaming do ASF em uma rede.<br/>                                                                                          |
| [Windows Serviços de compartilhamento de biblioteca de mídia](/previous-versions/windows/desktop/wmlss/windowsmedialibrarysharingservicesportal)<br/>                | Permite que os aplicativos descubram dispositivos de mídia na rede base e compartilhem bibliotecas de mídia na rede base e na Internet.<br/>                                                                                     |
| [Windows Media Player Sdk](./wmp/windows-media-player-sdk.md)<br/>                                                | Estende os recursos do Windows Media Player e Windows Media Player Mobile.<br/>                                                                                                                                       |
| [Windows Media Rights Manager 10.1.2 SDK](/previous-versions//bb649422(v=vs.85))<br/> | Dá suporte ao DRM (gerenciamento de direitos digitais) para proteger Windows de mídia.<br/>                                                                                                                                            |
| [Windows Media Services série 9](/previous-versions/windows/desktop/dd873819(v=vs.85))<br/>                                 | Fornece uma API baseada em automação para gerenciar Windows Media Services no Windows Server.<br/>                                                                                                                                 |
| [Windows API de Streaming de Mídia](./mediastreaming/media-streaming-api-portal.md)<br/>                                | Modela os dispositivos DMR (Digital Media Renderer) e DMS (Servidor de Mídia Digital), conforme definido pelas diretrizes da DLNA, para facilitar o uso programático.<br/>                                                                           |
| [Windows Controles de transporte de mídia](/previous-versions/windows/desktop/mediatransport/media-transport-controls-portal)<br/>                      | Fornece uma interface interna que mostra as informações do usuário sobre a mídia em reprodução no momento, como o título de uma música ou vídeo. Também fornece ao usuário uma maneira comum e familiar de controlar a reprodução de mídia.<br/> |
| [Windows Movie Maker SDK 6.0 e Windows DVD Maker 1.0](/previous-versions/windows/desktop/wmmdvdm/windows-movie-maker-6-0-and-windows-dvd-maker-1-0-sdk)<br/>     | Habilita a personalização do Windows Movie Maker 6.0 e do Microsoft Windows DVD Maker 1.0.<br/>                                                                                                                                   |
| [Áudio e vídeo herdado](/windows-hardware/drivers/audio/legacy-audio-interfaces)<br/>                                                    | Tecnologias obsoletas e que não devem ser usadas em novos aplicativos.<br/>                                                                                                                                              |
